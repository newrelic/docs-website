이 문서에서는 Python 에이전트를 Tornado 6을 사용하는 앱과 통합하기 위한 요구 사항 및 팁을 설명합니다. 일반 설치 지침으로 돌아가려면 [Python 에이전트 설치](/docs/agents/python-agent/installation-configuration/python-agent-installation) 로 이동하십시오.

<Callout variant="important">
  Python 에이전트는 6.0 미만의 Tornado 버전을 지원하지 않습니다.
</Callout>

## 관리 스크립트로 자동 초기화 [#wrapper-script]

`python app.py` 으로 앱을 시작하고 Tornado 비동기 인터페이스를 사용하는 경우 권장되는 [관리 스크립트 통합 방법](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) 을 사용할 수 있습니다. 다음은 관리 스크립트 방법을 사용하는 예입니다.

```
NEW_RELIC_CONFIG_FILE=<var>path/to/</var>newrelic.ini newrelic-admin run-python app.py
```

앱 코드에서 수동 통합은 허용되지만 권장되지는 않습니다. 자세한 내용은 [수동 통합 지침](/docs/agents/python-agent/installation-configuration/python-agent-integration#manual-integration) 을 참조하세요.

## 이벤트 루프 진단 지원 [#event-loops]

Python 에이전트는 asyncio 이벤트 루프 진단을 지원합니다. 자세한 내용은 [Python 이벤트 루프 진단](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics) 을 참조하세요.
