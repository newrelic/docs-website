일부 New Relic 절차에서는 계정 ID를 사용해야 합니다(예: 일부 API 호출). 귀하의 계정 ID는 당사가 New Relic 계정에 할당한 ID 번호입니다. [New Relic 조직에는 여러 계정이 포함될 수 있습니다](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure) .

New Relic 계정의 ID를 찾는 몇 가지 방법:

* 조직에 여러 계정이 있는 경우: [one.newrelic.com](https://one.newrelic.com) 에서 페이지 상단 근처의 계정 선택기 드롭다운을 사용하여 계정 간에 전환하고 해당 ID를 확인합니다.

* 사용 중인 [사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 에 따라 다른 옵션이 있습니다.

  * 최신 사용자 모델: [계정 드롭다운](/docs/using-new-relic/welcome-new-relic/get-started/glossary#account-dropdown) 을 클릭한 다음 **관리 > 액세스 관리 > 계정** 으로 이동하여 계정 ID를 확인합니다.
  * 원래 사용자 모델: [계정 드롭다운](/docs/using-new-relic/welcome-new-relic/get-started/glossary#account-dropdown) , **계정 설정** , **API 키** 를 차례로 클릭합니다.계정 ID가 표시됩니다.

계정 액세스 작동 방식에 대한 자세한 내용은 액세스 [에 영향을 미치는 요소](/docs/accounts/accounts-billing/account-structure/factors-affecting-access-features-data) 를 참조하십시오.
