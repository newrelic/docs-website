title: 차트 새로 고침 빈도
metaDescription: New Relic chart refresh rates.
translationType: machine
---

New Relic 차트의 새로 고침 빈도는 쿼리 기간을 60으로 나눈 값입니다. 쿼리를 새로 고칠 수 있는 최대 빈도는 5초마다입니다.

예를 들어:

* 시간 범위가 7일인 쿼리의 새로 고침 빈도는 2.8시간마다 한 번입니다.
* 시간 범위가 1시간인 쿼리의 새로 고침 빈도는 1분에 한 번입니다.

`TIMESERIES` 을 사용하는 쿼리의 경우 `TIMESERIES` 버킷의 크기가 새로 고침 간격으로 사용됩니다. 예를 들어:

* `SELECT count(*) FROM PageView TIMESERIES 5 minutes` 5분마다 새로 고침됩니다.
* `SELECT count(*) FROM PageView TIMESERIES 1 hour SINCE 1 day ago` 매시간 새로고침합니다.
