## 문제

`x509: certificate signed by unknown authority` 메시지가 표시됩니다.

## 해결책

서명된 인증서가 올바르게 구성되었는지 확인해야 합니다.이렇게 하려면 에이전트 YML 컨테이너에 다음 환경 변수를 추가할 수 있습니다.

```yml
- name: NRIA_CA_BUNDLE_DIR 
  value: YOUR_CA_BUNDLE_DIR
- name: NRIA_CA_BUNDLE_FILE  
  value: YOUR_CA_BUNDLE_NAME
```
