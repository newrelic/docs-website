New Relic now empowers your team to monitor your Microsoft SQL Server database performance using OpenTelemetry (OTEL) with New Relic's database monitoring capabilities. This integration provides comprehensive insights into database metrics, query performance, and system health using the OpenTelemetry Collector.

This OTEL-based approach complements our existing On-Host Integration (OHI) by leveraging OpenTelemetry standards for database monitoring, making it easier to integrate with your existing observability stack.

## Prerequisites [#prerequisites]

Before you begin, ensure you have the following:

* Valid New Relic [license key](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key)
* Supported database versions: SQL Server 2017 or later
* SQL Server database setup:
  * Administrative access to your SQL Server instance (`sysadmin` role or equivalent)
  * Microsoft SQL Server that supports on-premises managed instances
  * Network connectivity between the OpenTelemetry Collector and your SQL Server
  * SQL Server Management Studio (SSMS) or `sqlcmd` utility
  * Server hostname or IP address
  * Port number (usually 1433) or custom port
  * Windows domain or SQL Server authentication
  * SSL preference

* System requirements:
  * Windows or Linux system to run the OpenTelemetry Collector
  * Network connectivity to New Relic's OTLP endpoint

### Configure database user [#configure-user]

Run the following script as a `root user/sysadmin` to create the `newrelic` monitoring user and grant global read permissions.

  <CollapserGroup>

    <Collapser
      title="For Windows (SSMS)"
      id="windows-user"
    >

    * To create a `newrelic` monitoring user, run the following script using SQL Server Management Studio (SSMS):

      ```sql
      USE [master];
      GO
      CREATE LOGIN [newrelic] WITH PASSWORD = 'YourSecurePassword';
      GO
      -- Instance-level permissions
      GRANT VIEW SERVER STATE TO [newrelic];
      GRANT VIEW ANY DEFINITION TO [newrelic];
      GRANT VIEW ANY DATABASE TO [newrelic];
      GO
      -- Grant read access privileges to all user databases
      DECLARE @name SYSNAME;
      DECLARE db_cursor CURSOR READ_ONLY FORWARD_ONLY FOR
      SELECT [name]
      FROM [master].[sys].[databases]
      WHERE [name] NOT IN ('master', 'msdb', 'tempdb', 'model', 'rdsadmin', 'distribution')
      AND [state] = 0; -- Only online databases
      OPEN db_cursor;
      FETCH NEXT FROM db_cursor INTO @name;
      WHILE @@FETCH_STATUS = 0
      BEGIN
        BEGIN TRY
          PRINT 'Processing database: ' + @name;
          EXEC('USE [' + @name + '];
          IF NOT EXISTS (SELECT 1 FROM sys.database_principals WHERE name = ''newrelic'')
          BEGIN
            CREATE USER [newrelic] FOR LOGIN [newrelic];
          END;
          GRANT VIEW DATABASE STATE TO [newrelic];
          GRANT VIEW DEFINITION TO [newrelic];
          ALTER ROLE db_datareader ADD MEMBER [newrelic];');
          PRINT 'Success: ' + @name;
        END TRY
        BEGIN CATCH
          PRINT 'Error on ' + @name + ': ' + ERROR_MESSAGE();
        END CATCH
        FETCH NEXT FROM db_cursor INTO @name;
      END
      CLOSE db_cursor;
      DEALLOCATE db_cursor;
      GO
      ```

    </Collapser>

### Configure NRDOT Collector [#configure-collector]

Replace your `config.yaml` with the below content:

* **Windows:** `C:\Program Files (x86)\NRDOT Collector Host\config.yaml`
* **Linux:** `/etc/nrdot-collector-host/config.yaml`

    <Callout variant="important">

      If you're an existing NRDOT Collector user, ensure to merge the above configuration with your existing `config.yaml` file. This includes adding the new receivers, processors, and exporters for MSSQL Database monitoring while retaining your current configurations.
    </Callout> 

```yaml

# SQL Server On-Premises/VM Configuration
extensions:
  health_check:

  newrelicsqlserver:
    hostname: "<YOUR_DB_HOST>"
    port: "<YOUR_DB_PORT>"
    username: "<USERNAME>"
    password: "<PASSWORD>"
    monitored_databases: []
    timeout: 600s
    collection_interval: 30s
    query_monitoring_fetch_interval: 30
    enable_query_monitoring: true
    query_monitoring_response_time_threshold: 500
    query_monitoring_count_threshold: 30
    enable_interval_based_averaging: true
    interval_calculator_cache_ttl_minutes: 10
    # Metric Category Toggles - Enable/disable entire categories of metrics
    enable_instance_metrics: false
    enable_database_metrics: false
    enable_user_connection_metrics: false
    enable_wait_time_metrics: false
    enable_failover_cluster_metrics: false
    enable_database_principals_metrics: false
    enable_database_role_membership_metrics: false
    enable_security_metrics: false
    enable_lock_metrics: false

    check_interval: 1s

  batch:
  # Transform to clear description and unit fields
  transform/clear_metadata:

      - context: metric

          - set(metric.description, "")
          - set(metric.unit, "")
  # Filter to only include execution plan and event metrics (for logs conversion)
  filter/exec_plan_include:
    metrics:
      include:
        match_type: strict
        metric_names:
          - sqlserver.slowquery.query_details
          - sqlserver.execution.plan
          - sqlserver.blocking_query.details

        - sqlserver.instance.forced_parameterizations_per_sec
        - sqlserver.instance.full_scans_rate

        # Database log (sys.dm_os_performance_counters)
        - sqlserver.database.log.flushes_per_sec
        - sqlserver.database.log.bytes_flushed_per_sec
        - sqlserver.database.log.flush_waits_per_sec
        # Failover cluster
        - sqlserver.failover_cluster.log_bytes_received_per_sec
        # User connections
        - sqlserver.user_connections.authentication.logins_per_sec
  deltatorate:
    metrics:
      - sqlserver.wait_stats.latch.wait_time_ms
      - sqlserver.wait_stats.latch.waiting_tasks_count
      - sqlserver.wait_stats.wait_time_ms
      - sqlserver.wait_stats.waiting_tasks_count
      - sqlserver.stats.sql_compilations_per_sec
      - sqlserver.stats.sql_recompilations_per_sec
      - sqlserver.stats.lock_waits_per_sec
      - sqlserver.stats.deadlocks_per_sec
      - sqlserver.stats.user_errors_per_sec
      - sqlserver.stats.kill_connection_errors_per_sec
      - sqlserver.access.page_splits_per_sec
      - sqlserver.buffer.checkpoint_pages_per_sec
      - sqlserver.bufferpool.batch_requests_per_sec
      - sqlserver.instance.transactions_per_sec
      - sqlserver.instance.forced_parameterizations_per_sec
      - sqlserver.instance.full_scans_rate
      - sqlserver.instance.lock_timeouts_rate
      - sqlserver.database.log.flushes_per_sec
      - sqlserver.database.log.bytes_flushed_per_sec
      - sqlserver.database.log.flush_waits_per_sec
      - sqlserver.failover_cluster.log_bytes_received_per_sec
      - sqlserver.user_connections.authentication.logins_per_sec
  resourcedetection/db_safe:
    detectors: ["system"]
    override: false
    system:
      hostname_sources: ["os"]
      resource_attributes:
        host.id:
          enabled: true
  resourcedetection/cloud:
    detectors: ["gcp", "ec2", "azure"]
    timeout: 2s
    override: true
  resourcedetection/env:
    detectors: ["env"]
    timeout: 2s
    override: true

connectors:
  metricsaslogs:
    include_resource_attributes: true
    include_scope_info: true

exporters:
  debug:
    verbosity: detailed
  otlphttp:
    endpoint: "<YOUR_NEWRELIC_OTLP_ENDPOINT>"
    headers:
      api-key: "<YOUR_NEWRELIC_API_KEY>"
    compression: gzip

service:
  pipelines:
    metrics:
      receivers: [newrelicsqlserver]
      processors: [memory_limiter, resourcedetection/db_safe, resourcedetection/cloud, resourcedetection/env, filter/exec_plan_exclude, batch]
      exporters: [otlphttp]
    logs:
      receivers: [metricsaslogs]
      processors: [memory_limiter, resourcedetection/db_safe, resourcedetection/cloud, resourcedetection/env, batch]
      exporters: [otlphttp, debug]
    metrics/exec_plan_to_logs:
      receivers: [newrelicsqlserver]
      processors: [memory_limiter, resourcedetection/db_safe, resourcedetection/cloud, resourcedetection/env, filter/exec_plan_include, batch]
      exporters: [metricsaslogs]
  extensions: [health_check]
```

        Write-Host "Running as: $env:USERNAME" -ForegroundColor Gray

        $scriptErrors = @()

                    $testRegistryPath = "HKLM:\SOFTWARE\Microsoft\Microsoft SQL Server\MSSQL$versionId.$SqlInstanceName\MSSQLServer"

                    } else {

                    }

            } catch {
                # Namespace doesn't exist, continue to next version
                continue

        if (-not $detectedVersionId) {
            Write-Host "ERROR: Could not detect SQL Server version for instance '$SqlInstanceName'" -ForegroundColor Red
            Write-Host "Please verify:" -ForegroundColor Yellow
            Write-Host "  1. The instance name is correct" -ForegroundColor Yellow
            Write-Host "  2. SQL Server is installed" -ForegroundColor Yellow

        $SqlVersion = $detectedVersionId

        try {

            Write-Host "Path: $ConfigPath" -ForegroundColor Yellow

        } catch {

            Write-Host "Please ensure you have write permissions to: $resolvedConfigPath" -ForegroundColor Yellow

        try {
            $configDir = Split-Path -Parent $resolvedConfigPath

            $systemAccess = $acl.Access | Where-Object { $_.IdentityReference -eq "NT AUTHORITY\SYSTEM" }
            if (-not $systemAccess) {

                icacls "$resolvedConfigPath" /grant "NT AUTHORITY\SYSTEM:R" | Out-Null
                icacls "$configDir" /grant "NT AUTHORITY\SYSTEM:(OI)(CI)RX" | Out-Null
                Write-Host "SUCCESS: SYSTEM read permissions granted." -ForegroundColor Green
            }
        } catch {
            Write-Host "ERROR: Could not verify or set config file permissions: $($_.Exception.Message)" -ForegroundColor Red
            Write-Host "The service may fail to start without SYSTEM read access to the config file." -ForegroundColor Yellow
            Stop-Transcript
            exit 1
        }

        Write-Host "`n============================================" -ForegroundColor Cyan
        Write-Host "SQL Server & New Relic Collector Setup" -ForegroundColor Cyan
        Write-Host "============================================" -ForegroundColor Cyan

        Write-Host "`n--- PART 1: SQL Server Configuration ---" -ForegroundColor Magenta

        # Use provided SqlVersion and SqlInstanceName for SQL Server configuration
        $v = $SqlVersion
        $instanceName = $SqlInstanceName
        $configuredInstanceName = $instanceName  # Store for collector logs
        $needsRestart = $false

        Write-Host "`n========================================" -ForegroundColor Cyan
        Write-Host "Instance: SQL Server $originalSqlVersion (Version ID: $v) - $instanceName" -ForegroundColor Cyan
        Write-Host "========================================" -ForegroundColor Cyan

        Write-Host "`nChecking current configuration..." -ForegroundColor Yellow
        $willNeedRestart = $false

        Write-Host "`nConfiguring SQL Server $originalSqlVersion (Version ID: $v) - $instanceName..." -ForegroundColor Yellow

                # --- ENABLE TCP/IP ---
                try {
                    $namespace = "root\Microsoft\SqlServer\ComputerManagement$v"
                    $tcpProtocol = Get-WmiObject -Namespace $namespace -Class "ServerNetworkProtocol" -Filter "ProtocolName = 'Tcp' and InstanceName = '$instanceName'" -ErrorAction Stop

                    if (-not $tcpProtocol) {
                        throw "TCP protocol settings not found for instance '$instanceName'. Verify version ID and instance name."
                    }
                    
                    if ($tcpProtocol.Enabled -eq $false) {
        # Check if restart is allowed before making changes
                        if (-not $allowSqlServerRestart) {

                            Write-Host "Instance: $instanceName" -ForegroundColor Yellow

                            Write-Host "  2. Manually enable TCP/IP Protocol from SQL Server Configuration Manager" -ForegroundColor Cyan
                            Write-Host "`nScript halted. No changes have been made." -ForegroundColor Red
                            Stop-Transcript
                            exit 1

                    $errorMsg = "ERROR: Failed to configure TCP/IP for $instanceName - $($_.Exception.Message)"

                    Write-Host "`nThis is a critical error. The script cannot continue without TCP/IP enabled." -ForegroundColor Yellow
                    Write-Host "Please check the error message above and verify the SQL Server installation." -ForegroundColor Yellow
                    $scriptErrors += $errorMsg

                try {
                    $registryPath = "HKLM:\SOFTWARE\Microsoft\Microsoft SQL Server\MSSQL$v.$instanceName\MSSQLServer"
                    
                    if (-not (Test-Path $registryPath)) {

                            Write-Host "Instance: $instanceName" -ForegroundColor Yellow
                            Write-Host "`nEnabling Mixed Mode Authentication requires a SQL Server restart." -ForegroundColor Yellow
                            Write-Host "`nTo proceed, choose one of the following options:" -ForegroundColor Yellow
                            Write-Host "  1. Set -allowSqlServerRestart `$true to allow automatic SQL Server restart" -ForegroundColor Cyan
                            Write-Host "  2. Manually enable Mixed Mode Authentication from SQL Server properties" -ForegroundColor Cyan
                            Write-Host "`nScript halted. No changes have been made." -ForegroundColor Red

                    Write-Host $errorMsg -ForegroundColor Red
                    Write-Host "`nThis is a critical error. The script cannot continue without Mixed Mode Authentication." -ForegroundColor Yellow
                    Write-Host "Please check the error message above and verify the SQL Server installation." -ForegroundColor Yellow

                    if ($username) {

                      GRANT VIEW DATABASE STATE TO [$username];

                            Write-Host "  - VIEW ANY DEFINITION (instance-level)" -ForegroundColor Gray
                            Write-Host "  - VIEW ANY DATABASE (instance-level)" -ForegroundColor Gray
                            Write-Host "  - db_datareader role (all user databases)" -ForegroundColor Gray
                            Write-Host "  - VIEW DATABASE STATE (all user databases)" -ForegroundColor Gray
                            Write-Host "  - VIEW DEFINITION (all user databases)" -ForegroundColor Gray
                        }
                    }
                } catch {
                    $errorMsg = "ERROR: Failed to create database user for $instanceName - $($_.Exception.Message)"
                    Write-Host $errorMsg -ForegroundColor Red
                    $scriptErrors += $errorMsg
                    Stop-Transcript
                    exit 1
                }

                Write-Host "`n--- Firewall Configuration ---" -ForegroundColor Cyan

                        Write-Host "  - Rule Name: $($rule.DisplayName)" -ForegroundColor Gray

                    Write-Host "  - Rule Name: $ruleName" -ForegroundColor Gray
                }
            } catch {

                Write-Host $errorMsg -ForegroundColor Red
                $scriptErrors += $errorMsg
                Stop-Transcript
                exit 1

            try {
                Write-Host "`n--- SQL Server Browser Service ---" -ForegroundColor Cyan
                $browser = Get-Service -Name "SQLBrowser" -ErrorAction Stop
                
                if ($browser.StartType -ne "Automatic" -or $browser.Status -ne "Running") {
                    if ($browser.StartType -ne "Automatic") {
                        Set-Service -Name "SQLBrowser" -StartupType Automatic -ErrorAction Stop
                    }
                    
                    if ($browser.Status -ne "Running") {
                        Start-Service -Name "SQLBrowser" -ErrorAction Stop
                        $browser = Get-Service -Name "SQLBrowser"
                        if ($browser.Status -ne "Running") {
                            throw "Service did not start successfully"
                        }
                    }
                    Write-Host "SUCCESS: SQL Browser Service set to Automatic and Started." -ForegroundColor Green
                } else {
                    Write-Host "CHECK: SQL Browser Service is already Automatic and Running." -ForegroundColor Gray
                }
            } catch {
                $errorMsg = "ERROR: Failed to configure SQL Browser Service - $($_.Exception.Message)"
                Write-Host $errorMsg -ForegroundColor Red
                $scriptErrors += $errorMsg
                Stop-Transcript
                exit 1
            }

        # ============================================
        # PART 2: NEW RELIC NRDOT COLLECTOR INSTALLATION
        # ============================================

        Write-Host "`n--- PART 2: New Relic NRDOT Collector Installation ---" -ForegroundColor Magenta

        # Collector configuration
        $url = "https://download.newrelic.com/database-monitoring/nrdot-collector/mssql/nrdot-collector-host-windows-amd64-1.0.0.exe"
            $nssmUrl = "https://nssm.cc/release/nssm-2.24.zip"
            $destDir = "C:\Program Files\New Relic\nrdot-collector"
            $exePath = "$destDir\nrdot-collector-host-windows-amd64-1.0.0.exe"
            $nssmDir = "$destDir\nssm"
            $nssmExe = "$nssmDir\nssm.exe"
            
            # Make service name unique per SQL instance
            if ($configuredInstanceName) {
                $serviceName = "nrdot-collector-$configuredInstanceName"
                $displayName = "New Relic NRDOT Collector ($configuredInstanceName)"
                $logsDir = "$destDir\logs\$configuredInstanceName"
                Write-Host "`nUsing instance-specific configuration:" -ForegroundColor Gray
                Write-Host "  Service Name: $serviceName" -ForegroundColor Gray
                Write-Host "  Log Directory: $logsDir" -ForegroundColor Gray
            } else {
                $serviceName = "nrdot-collector"
                $displayName = "New Relic NRDOT Collector"
                $logsDir = "$destDir\logs"
                Write-Host "`nUsing default configuration:" -ForegroundColor Gray
                Write-Host "  Service Name: $serviceName" -ForegroundColor Gray
                Write-Host "  Log Directory: $logsDir" -ForegroundColor Gray
            }
            $serviceLogFile = "$logsDir\otel-collector.log"

            # 1. DOWNLOAD THE EXE
            if (-not (Test-Path "$destDir")) { New-Item -ItemType Directory -Path "$destDir" -Force | Out-Null }
            if (-not (Test-Path "$logsDir")) { New-Item -ItemType Directory -Path "$logsDir" -Force | Out-Null }

            if (-not (Test-Path "$exePath")) {
                Write-Host "`nDownloading collector executable..." -ForegroundColor Yellow
                try {
                    curl.exe -L -o "$exePath" "$url"
                    if ($LASTEXITCODE -ne 0) {
                        Write-Host "ERROR: Download failed with exit code $LASTEXITCODE" -ForegroundColor Red
                        Stop-Transcript
                        exit 1
                    }
                    Write-Host "Download complete." -ForegroundColor Green
                } catch {
                    Write-Host "ERROR: Failed to download collector: $($_.Exception.Message)" -ForegroundColor Red
                    Stop-Transcript
                    exit 1
                }
            } else {
                Write-Host "`nCollector executable already exists." -ForegroundColor Gray
            }

            # 2. DOWNLOAD AND EXTRACT NSSM
            if (-not (Test-Path "$nssmExe")) {
                Write-Host "Downloading NSSM (Non-Sucking Service Manager)..." -ForegroundColor Yellow
                $nssmZip = "$env:TEMP\nssm.zip"

                try {
                    # Remove any existing corrupted zip file
                    if (Test-Path "$nssmZip") {
                        Remove-Item "$nssmZip" -Force -ErrorAction SilentlyContinue
                    }

                    # Use Invoke-WebRequest for better redirect handling
                    [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12
                    Invoke-WebRequest -Uri "$nssmUrl" -OutFile "$nssmZip" -UseBasicParsing -ErrorAction Stop

                    # Wait for file to be fully written and verify it exists
                    Start-Sleep -Seconds 3
                    if (-not (Test-Path "$nssmZip")) {
                        throw "Downloaded zip file not found at $nssmZip"
                    }
                    $zipSize = (Get-Item "$nssmZip").Length
                    if ($zipSize -lt 100KB) {
                        throw "Downloaded zip file is too small ($zipSize bytes), may be corrupted. Actual size: $zipSize bytes"
                    }
                    Write-Host "NSSM zip downloaded successfully ($zipSize bytes)." -ForegroundColor Green

                    Write-Host "Extracting NSSM..." -ForegroundColor Yellow
                    Expand-Archive -Path "$nssmZip" -DestinationPath "$env:TEMP\nssm-extract" -Force

                    $arch = if ([Environment]::Is64BitOperatingSystem) { "win64" } else { "win32" }
                    New-Item -ItemType Directory -Path "$nssmDir" -Force | Out-Null
                    Copy-Item "$env:TEMP\nssm-extract\nssm-2.24\$arch\nssm.exe" -Destination "$nssmExe" -Force

                    Write-Host "NSSM installed successfully." -ForegroundColor Green
                } catch {
                    Write-Host "ERROR: Failed to download or extract NSSM: $($_.Exception.Message)" -ForegroundColor Red
                    Write-Host "`nPlease check:" -ForegroundColor Yellow
                    Write-Host "  1. Internet connectivity to $nssmUrl" -ForegroundColor Yellow
                    Write-Host "  2. Write permissions to $env:TEMP" -ForegroundColor Yellow
                    Write-Host "  3. Available disk space" -ForegroundColor Yellow
                    Write-Host "`nAlternatively, manually download and install NSSM:" -ForegroundColor Cyan
                    Write-Host "  1. Download NSSM from: $nssmUrl" -ForegroundColor White
                    Write-Host "  2. Extract the zip file" -ForegroundColor White
                    Write-Host "  3. Copy nssm.exe (from win64 or win32 folder) to:" -ForegroundColor White
                    Write-Host "     $nssmExe" -ForegroundColor Green
                    Write-Host "  4. Re-run this script" -ForegroundColor White
                    Stop-Transcript
                    exit 1
                } finally {
                    Remove-Item "$nssmZip" -Force -ErrorAction SilentlyContinue
                    Remove-Item "$env:TEMP\nssm-extract" -Recurse -Force -ErrorAction SilentlyContinue
                }
            } else {
                Write-Host "NSSM already installed." -ForegroundColor Gray
            }

            # 3. CLEAN UP OLD SERVICE
            if (Get-Service -Name $serviceName -ErrorAction SilentlyContinue) {
                try {
                    Write-Host "`nWARNING: NRDOT Collector service is already detected in the system!" -ForegroundColor Yellow
                    Write-Host "Service Name: $serviceName" -ForegroundColor Gray

                    $currentService = Get-Service -Name $serviceName
                    Write-Host "Current Status: $($currentService.Status)" -ForegroundColor Gray

                    if (-not $ForceServiceRemoval) {
                        Write-Host "`nERROR: Service already exists. Set -ForceServiceRemoval `$true to remove existing service." -ForegroundColor Red
                        Write-Host "Existing service name: $serviceName" -ForegroundColor Yellow
                        Write-Host "Current status: $($currentService.Status)" -ForegroundColor Yellow
                        Stop-Transcript
                        exit 1
                    }
                    
                    Write-Host "`nThe existing service will be stopped and removed." -ForegroundColor Yellow
                    Write-Host "`nStopping and removing existing service..." -ForegroundColor Yellow
                    & "$nssmExe" stop $serviceName confirm
                    Start-Sleep -Seconds 2
                    & "$nssmExe" remove $serviceName confirm
                    Start-Sleep -Seconds 2
                    Write-Host "Old service removed." -ForegroundColor Green

                    if (Test-Path "$logsDir") {
                        $oldLogs = Get-ChildItem -Path "$logsDir" -Filter "otel-collector*" -ErrorAction SilentlyContinue
                        if ($oldLogs) {
                            $oldLogs | Remove-Item -Force -ErrorAction SilentlyContinue
                            Write-Host "Old otel-collector log files removed." -ForegroundColor Green
                        }

                        # Also remove cleanup script log file
                        $cleanupLog = "$logsDir\cleanup-log.log"
                        if (Test-Path $cleanupLog) {
                            Remove-Item $cleanupLog -Force -ErrorAction SilentlyContinue
                            Write-Host "Cleanup script log file removed." -ForegroundColor Green
                        }
                    }
                } catch {
                    Write-Host "ERROR: Failed to clean up old service: $($_.Exception.Message)" -ForegroundColor Red
                    Write-Host "You may need to manually remove the service using: sc.exe delete $serviceName" -ForegroundColor Yellow
                    Stop-Transcript
                    exit 1
                }
            }

            # 4. USE PROVIDED CONFIG FILE
            $selectedConfig = $ConfigPath
            Write-Host "`nUsing config file: $selectedConfig" -ForegroundColor Green

            # 5. CONFIGURE CONFIG FILE
            Write-Host "`n--- Configuring Config File ---" -ForegroundColor Cyan

            try {
                $configContent = Get-Content -Path $selectedConfig -Raw -ErrorAction Stop

                $placeholdersFound = @()
                if ($configContent -match '<YOUR_DB_HOST>') { $placeholdersFound += 'YOUR_DB_HOST' }
                if ($configContent -match '<YOUR_DB_USERNAME>') { $placeholdersFound += 'YOUR_DB_USERNAME' }
                if ($configContent -match '<YOUR_DB_PASSWORD>') { $placeholdersFound += 'YOUR_DB_PASSWORD' }
                if ($configContent -match '<YOUR_NEWRELIC_OTLP_ENDPOINT>') { $placeholdersFound += 'YOUR_NEWRELIC_OTLP_ENDPOINT' }
                if ($configContent -match '<YOUR_NEWRELIC_LICENSE_KEY>') { $placeholdersFound += 'YOUR_NEWRELIC_LICENSE_KEY' }
                if ($configContent -match '<YOUR_DB_PORT>') { $placeholdersFound += 'YOUR_DB_PORT' }
            } catch {
                Write-Host "ERROR: Failed to read config file: $($_.Exception.Message)" -ForegroundColor Red
                Stop-Transcript
                exit 1
            }

            if ($placeholdersFound.Count -eq 0) {
                Write-Host "No placeholders found in config file. Skipping configuration." -ForegroundColor Green
                $needsConfig = $false
            } else {
                Write-Host "Placeholders found: $($placeholdersFound -join ', ')" -ForegroundColor Green
                $needsConfig = $true
            }

            if ($needsConfig) {
                Write-Host "`n--- SQL Server Configuration ---" -ForegroundColor Cyan
                
                $hostname = $DBHostname
                Write-Host "Using hostname: $hostname" -ForegroundColor Green
                
                # Use the database credentials provided via command line parameters
                $dbUsername = $DBUsername
                $dbPassword = $DBPassword
                Write-Host "Using database credentials from command line parameters" -ForegroundColor Green
                Write-Host "  Username: $dbUsername" -ForegroundColor Gray
                
                # Use New Relic configuration from command line parameters
                $otlpEndpoint = $OtlpEndpoint
                $licenseKey = $LicenseKey
                Write-Host "Using New Relic configuration from command line parameters" -ForegroundColor Green
                Write-Host "  OTLP Endpoint: $otlpEndpoint" -ForegroundColor Gray
                Write-Host "  License Key: $($licenseKey.Substring(0, [Math]::Min(8, $licenseKey.Length)))..." -ForegroundColor Gray
                
                if ($needsConfig) {
                    Write-Host "`n--- Configuration Summary ---" -ForegroundColor Green
                    Write-Host "  Hostname:      $hostname" -ForegroundColor White
                    Write-Host "  Username:      $dbUsername" -ForegroundColor White
                    Write-Host "  Password:      $('*' * $dbPassword.Length)" -ForegroundColor White
                    Write-Host "  Port:          $Port" -ForegroundColor White
                    Write-Host "  OTLP Endpoint: $otlpEndpoint" -ForegroundColor White
                    Write-Host "  License Key:   $($licenseKey.Substring(0, [Math]::Min(8, $licenseKey.Length)))..." -ForegroundColor White
                    Write-Host "`nProceeding with configuration..." -ForegroundColor Green
                }
                
                if ($needsConfig) {
                    try {
                        $backupFile = "$selectedConfig.backup_$(Get-Date -Format 'yyyyMMdd_HHmmss')"
                        Copy-Item -Path $selectedConfig -Destination $backupFile -Force -ErrorAction Stop
                        Write-Host "`nBackup created: $backupFile" -ForegroundColor Gray
                    } catch {
                        Write-Host "WARNING: Failed to create backup: $($_.Exception.Message)" -ForegroundColor Yellow
                    }
                    
                    Write-Host "Updating config file..." -ForegroundColor Green
                    
                    $configContent = $configContent -replace '<YOUR_DB_HOST>', $hostname
                    $configContent = $configContent -replace '<YOUR_DB_USERNAME>', $dbUsername
                    $configContent = $configContent -replace '<YOUR_DB_PASSWORD>', $dbPassword
                    $configContent = $configContent -replace '<YOUR_DB_PORT>', $Port
                    $configContent = $configContent -replace '<YOUR_NEWRELIC_OTLP_ENDPOINT>', $otlpEndpoint
                    $configContent = $configContent -replace '<YOUR_NEWRELIC_LICENSE_KEY>', $licenseKey
                    
                    try {
                        Set-Content -Path $selectedConfig -Value $configContent -Force -ErrorAction Stop
                        Write-Host "Config file updated successfully!" -ForegroundColor Green
                    } catch {
                        Write-Host "`nERROR: Failed to update config file." -ForegroundColor Red
                        Write-Host $_.Exception.Message -ForegroundColor Red
                        Write-Host "`n--- Manual Configuration Required ---" -ForegroundColor Yellow
                        Write-Host "The config file could not be updated automatically." -ForegroundColor Yellow
                        Write-Host "Please manually edit the config file $selectedConfig" -ForegroundColor Yellow
                        Write-Host "`nFor detailed instructions, refer to Section 3 at:" -ForegroundColor Cyan
                        Write-Host "https://docs-preview.newrelic.com/docs/otel-mssql" -ForegroundColor Cyan
                        Write-Host "`nAfter manual editing, continue..." -ForegroundColor Yellow 
                        Stop-Transcript
                        exit 1
                    }
                }
            }

            # 5. TEST EXECUTABLE
            Write-Host "`nTesting executable..." -ForegroundColor Yellow

            try {
                $testLogFile = "$destDir\test-run.log"
                
                if (Test-Path "$testLogFile") {
                    Remove-Item "$testLogFile" -Force -ErrorAction SilentlyContinue
                }
                if (Test-Path "$testLogFile.err") {
                    Remove-Item "$testLogFile.err" -Force -ErrorAction SilentlyContinue
                }
                
                $testProcess = Start-Process -FilePath "$exePath" `
                                            -ArgumentList "--config=`"$selectedConfig`"" `
                                            -WorkingDirectory "$destDir" `
                                            -RedirectStandardOutput "$testLogFile" `
                                            -RedirectStandardError "$testLogFile.err" `
                                            -NoNewWindow `
                                            -PassThru `
                                            -ErrorAction Stop

                Start-Sleep -Seconds 3

                if ($testProcess.HasExited) {
                    Write-Host "ERROR: Executable exited immediately with code: $($testProcess.ExitCode)" -ForegroundColor Red
                    Write-Host "Check test logs:" -ForegroundColor Yellow
                    Write-Host "  Get-Content '$testLogFile'" -ForegroundColor White
                    Write-Host "  Get-Content '$testLogFile.err'" -ForegroundColor White
                    Stop-Transcript
                    exit 1
                } else {
                    Write-Host "Good! Executable is running. Stopping test process..." -ForegroundColor Green
                    try {
                        Stop-Process -Id $testProcess.Id -Force -ErrorAction Stop
                    } catch {
                        Get-Process -Name "nrdot-collector-host-windows-amd64-1.0.0" -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
                    }
                }
            } catch {
                Write-Host "ERROR: Failed to test executable: $($_.Exception.Message)" -ForegroundColor Red
                if ($testProcess -and -not $testProcess.HasExited) {
                    Stop-Process -Id $testProcess.Id -Force -ErrorAction SilentlyContinue
                }
                Stop-Transcript
                exit 1
            }

            # 6. INSTALL SERVICE USING NSSM
            Write-Host "`nInstalling Collector service..." -ForegroundColor Green

            try {
                Write-Host "Setting permissions..." -ForegroundColor Yellow
                icacls "$destDir" /grant "NT AUTHORITY\SYSTEM:(OI)(CI)F" /T | Out-Null
                icacls "$logsDir" /grant "NT AUTHORITY\SYSTEM:(OI)(CI)F" /T | Out-Null

                Write-Host "Installing service with NSSM..." -ForegroundColor Yellow
                & "$nssmExe" install $serviceName "$exePath" --config="$selectedConfig"

                if ($LASTEXITCODE -ne 0) {
                    Write-Host "ERROR: Failed to install service. Exit code: $LASTEXITCODE" -ForegroundColor Red
                    Stop-Transcript
                    exit 1
                }

                if (-not (Get-Service -Name $serviceName -ErrorAction SilentlyContinue)) {
                    Write-Host "ERROR: Service was not created!" -ForegroundColor Red
                    Stop-Transcript
                    exit 1
                }

                Write-Host "Service installed successfully!" -ForegroundColor Green

                Write-Host "Configuring log redirection..." -ForegroundColor Yellow
                & "$nssmExe" set $serviceName AppStdout "$serviceLogFile"
                & "$nssmExe" set $serviceName AppStderr "$serviceLogFile"

                & "$nssmExe" set $serviceName AppRotateFiles 1
                & "$nssmExe" set $serviceName AppRotateOnline 1
                & "$nssmExe" set $serviceName AppRotateBytes 52428800

                Write-Host "Configuring restart behavior..." -ForegroundColor Yellow
                & "$nssmExe" set $serviceName AppExit Default Exit
                & "$nssmExe" set $serviceName AppThrottle 1500
                & "$nssmExe" set $serviceName AppRestartDelay 1000

                & "$nssmExe" set $serviceName AppDirectory "$destDir"

                & "$nssmExe" set $serviceName Description "New Relic Database Monitoring Collector for SQL Server"
                & "$nssmExe" set $serviceName DisplayName "$displayName"
                & "$nssmExe" set $serviceName Start SERVICE_AUTO_START

                Write-Host "NSSM configuration complete." -ForegroundColor Green
            } catch {
                Write-Host "ERROR: Failed to install service: $($_.Exception.Message)" -ForegroundColor Red
                Stop-Transcript
                exit 1
            }

            # 7. START AND VERIFY
            try {
                Write-Host "`nStarting service..." -ForegroundColor Yellow
                & "$nssmExe" start $serviceName

                Start-Sleep -Seconds 5

                Write-Host "`nStatus Update:" -ForegroundColor Cyan
                $svcStatus = Get-Service -Name $serviceName -ErrorAction SilentlyContinue
                if ($svcStatus) {
                    $svcStatus | Select-Object Name, Status, StartType | Format-Table -AutoSize

                    if ($svcStatus.Status -eq "Running") {
                        Write-Host "Service is RUNNING!" -ForegroundColor Green
                    } else {
                        Write-Host "ERROR: Service is not running. Status: $($svcStatus.Status)" -ForegroundColor Red
                        Write-Host "`nCheck these logs:" -ForegroundColor Yellow
                        Write-Host "  Get-Content '$destDir\test-run.log'" -ForegroundColor White
                        Write-Host "  Get-Content '$serviceLogFile'" -ForegroundColor White
                        Stop-Transcript
                        exit 1
                    }
                } else {
                    Write-Host "ERROR: Service not found!" -ForegroundColor Red
                    Stop-Transcript
                    exit 1
                }
            } catch {
                Write-Host "ERROR: Failed to start service: $($_.Exception.Message)" -ForegroundColor Red
                Stop-Transcript
                exit 1
            }

        # ============================================
        # LOG CLEANUP SCHEDULED TASK
        # ============================================

        Write-Host "`n--- Configuring Log Cleanup ---" -ForegroundColor Magenta

        # Check if service is running before setting up cleanup
        $service = Get-Service -Name $serviceName -ErrorAction SilentlyContinue
        if (-not $service) {
            Write-Host "WARNING: Service '$serviceName' not found. Skipping log cleanup setup." -ForegroundColor Yellow
        } elseif ($service.Status -ne "Running") {
            Write-Host "WARNING: Service '$serviceName' is not running (Status: $($service.Status)). Skipping log cleanup setup." -ForegroundColor Yellow
            Write-Host "You can set up log cleanup manually after the service is running." -ForegroundColor Gray
        } else {
            Write-Host "Service is running. Proceeding with log cleanup setup..." -ForegroundColor Green

            $cleanupScriptPath = "$logsDir\cleanup-logs-script.ps1"
            $taskName = if ($configuredInstanceName) { 
                "Cleanup NRDOT Collector Logs ($configuredInstanceName)" 
            } else { 
                "Cleanup NRDOT Collector Logs" 
            }

            # Use log cleanup configuration from parameters
            $keepCount = $KeepLogCount
            $intervalMinutes = $CleanupIntervalMinutes
            
            Write-Host "`nLog Cleanup Configuration:" -ForegroundColor Yellow
            Write-Host "  Maximum log files to keep: $keepCount" -ForegroundColor Gray
            Write-Host "  Cleanup interval: Every $intervalMinutes minutes" -ForegroundColor Gray

        Write-Host "`nCreating cleanup script..." -ForegroundColor Yellow

        # Track whether cleanup script was created successfully
        $cleanupScriptCreated = $false

        $cleanupScriptContent = @"
        # NRDOT Collector Log Cleanup Script
        # Auto-generated - keeps only last $keepCount log files

        param(
            [string]`$serviceName = "$serviceName"
        )

        `$logsDir = "$logsDir"
        `$logPattern = "otel-collector-*"
        `$keepCount = $keepCount  # Keep last $keepCount log files total
        `$scriptLogFile = "`$logsDir\cleanup-log.log"

        function Write-Log {
            param([string]`$Message, [string]`$Level = "INFO")
            `$timestamp = Get-Date -Format "yyyy-MM-dd HH:mm:ss"
            `$logEntry = "[`$timestamp] [`$Level] `$Message"
            try {
                Add-Content -Path `$scriptLogFile -Value `$logEntry -ErrorAction SilentlyContinue
            } catch {}
        }

        Write-Log "Starting log cleanup..." "INFO"

        # Check if service is running before proceeding
        try {
            `$service = Get-Service -Name `$serviceName -ErrorAction Stop
            if (`$service.Status -ne "Running") {
                Write-Log "Service '`$serviceName' is not running (Status: `$(`$service.Status)). Skipping cleanup." "WARNING"
                exit 0
            }
            Write-Log "Service '`$serviceName' is running. Proceeding with cleanup." "INFO"
        } catch {
            Write-Log "Service '`$serviceName' not found. Skipping cleanup." "WARNING"
            exit 0
        }

        try {
            # Self-cleanup: Keep only last 100 lines of cleanup-log.log
            if (Test-Path `$scriptLogFile) {
                `$logSize = (Get-Item `$scriptLogFile).Length
                if (`$logSize -gt 100KB) {
                    `$lastLines = Get-Content `$scriptLogFile -Tail 100
                    Set-Content `$scriptLogFile -Value `$lastLines
                    Write-Log "Trimmed cleanup log to last 100 lines" "INFO"
                }
            }

            if (-not (Test-Path `$logsDir)) {
                Write-Log "Log directory not found: `$logsDir" "WARNING"
                exit 0
            }

            # Find all log files matching pattern (timestamp-based names with .log extension)
            `$allLogs = Get-ChildItem -Path `$logsDir -Filter `$logPattern -File -ErrorAction SilentlyContinue | 
                      Where-Object { `$_.Name -match '^otel-collector-\d{8}T\d+\.\d+\.log$' } |
                      Sort-Object LastWriteTime -Descending

            if (`$allLogs.Count -eq 0) {
                Write-Log "No log files found." "INFO"
                exit 0
            }

            Write-Log "Found `$(`$allLogs.Count) log file(s)" "INFO"

            if (`$allLogs.Count -le `$keepCount) {
                Write-Log "Within retention limit (`$keepCount files). No cleanup needed." "INFO"
                exit 0
            }

            # Delete files beyond the keep count (keeping the most recent based on LastWriteTime)
            `$logsToDelete = `$allLogs | Select-Object -Skip `$keepCount
            `$deletedCount = 0
            `$deletedSizeMB = 0

            foreach (`$log in `$logsToDelete) {
                try {
                    `$sizeMB = [math]::Round(`$log.Length / 1MB, 2)
                    Remove-Item -Path `$log.FullName -Force -ErrorAction Stop
                    Write-Log "Deleted: `$(`$log.Name) (`$sizeMB MB)" "INFO"
                    `$deletedCount++
                    `$deletedSizeMB += `$sizeMB
                } catch {
                    Write-Log "Failed to delete: `$(`$log.Name) - `$(`$_.Exception.Message)" "ERROR"
                }
            }

            Write-Log "Cleanup complete. Deleted `$deletedCount file(s), freed `$([math]::Round(`$deletedSizeMB, 2)) MB" "INFO"

        } catch {
            Write-Log "ERROR: `$(`$_.Exception.Message)" "ERROR"
            exit 1
        }

        exit 0
        "@

        try {
            Set-Content -Path $cleanupScriptPath -Value $cleanupScriptContent -Force
            Write-Host "Cleanup script created: $cleanupScriptPath" -ForegroundColor Green
            $cleanupScriptCreated = $true
        } catch {
            Write-Host "WARNING: Failed to create cleanup script: $($_.Exception.Message)" -ForegroundColor Yellow
            Write-Host "Skipping scheduled task creation." -ForegroundColor Yellow
            $scriptErrors += "Cleanup script creation failed..please try re-running or configure log cleanup manually."
        }

        # Only create scheduled task if cleanup script was created successfully
        if ($cleanupScriptCreated) {
            Write-Host "Registering scheduled task..." -ForegroundColor Yellow

            try {
            # Remove existing task if present
            $existingTask = Get-ScheduledTask -TaskName $taskName -ErrorAction SilentlyContinue
            if ($existingTask) {
                Unregister-ScheduledTask -TaskName $taskName -Confirm:$false
                Write-Host "Removed existing task." -ForegroundColor Gray
            }

            # Create new task
            $action = New-ScheduledTaskAction -Execute "PowerShell.exe" `
                -Argument "-NoProfile -ExecutionPolicy Bypass -WindowStyle Hidden -File `"$cleanupScriptPath`" -serviceName `"$serviceName`""

            $trigger = New-ScheduledTaskTrigger -Once -At (Get-Date) -RepetitionInterval (New-TimeSpan -Minutes $intervalMinutes)

            $principal = New-ScheduledTaskPrincipal -UserId "SYSTEM" -RunLevel Highest

            $settings = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable

            Register-ScheduledTask -TaskName $taskName `
                -Action $action `
                -Trigger $trigger `
                -Principal $principal `
                -Settings $settings `
                -Description "Automatically cleans up old NRDOT Collector log files every $intervalMinutes minutes, keeping only last $keepCount" | Out-Null

            Write-Host "Scheduled task registered successfully!" -ForegroundColor Green
            Write-Host "  Task Name: $taskName" -ForegroundColor Gray
            Write-Host "  Schedule: Every $intervalMinutes minutes" -ForegroundColor Gray
            Write-Host "  Retention: $keepCount files maximum" -ForegroundColor Gray

            } catch {
                Write-Host "WARNING: Failed to register scheduled task: $($_.Exception.Message)" -ForegroundColor Yellow
                Write-Host "You may need to manually create a scheduled task to run $cleanupScriptPath every $intervalMinutes minutes." -ForegroundColor Yellow
                $scriptErrors += "Scheduled task registration failed..please create a task to run $cleanupScriptPath every $intervalMinutes minutes manually."
            }
        } else {
            Write-Host "Skipping scheduled task registration because cleanup script creation failed." -ForegroundColor Yellow
        }
        }  # End of service status check for log cleanup

        # ============================================
        # FINAL SUMMARY
        # ============================================

        Write-Host "`n============================================" -ForegroundColor Cyan
        Write-Host "SETUP COMPLETE" -ForegroundColor Cyan
        Write-Host "Service Name:    $serviceName" -ForegroundColor Cyan
        Write-Host "Display Name:    $displayName" -ForegroundColor Cyan
