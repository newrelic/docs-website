## Configuration examples [#examples]

For detailed configuration examples and customization options, refer to the [internal telemetry configuration](https://github.com/newrelic/nrdot-collector-releases/blob/main/examples/internal-telemetry-config.yaml) in the NRDOT Collector releases repository.

The configuration exposes environment variables of the form `INTERNAL_TELEMETRY_...` to customize options such as detail levels and sampling. See the [setup documentation](/docs/opentelemetry/collector-observability/collector-observability-setup#customize) for more details.
