Spinners are used to indicate to your users that content is loading on the screen.

### Usage

```js
import { Spinner } from 'nr1'
```

### Examples

#### Basic

```js
<Spinner />
```

#### Dot

```js
<Spinner type={Spinner.TYPE.DOT} />
```

#### Inline

```js
<Spinner inline />
```
