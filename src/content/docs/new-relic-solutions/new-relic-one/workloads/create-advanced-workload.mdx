<Callout variant="important">
  Advanced workloads are in **Public Preview**.
</Callout>

Convert Transaction 360 views into monitored workloads to track transaction health continuously and configure alerts.

## Before you begin [#prereqs]

* [Distributed tracing enabled](/docs/distributed-tracing/enable-configure/quick-start)
* Access to Transaction 360
* Permissions to create workloads

## Create from Transaction 360 [#create]

1. Go to <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & Services**</DNT>
2. Select a service, then navigate to a transaction
3. Click <DNT>**Save as advanced workload**</DNT>
4. Enter a name and optional description
5. Click <DNT>**Save**</DNT>

You can also save from the Transactions page by selecting a transaction and clicking <DNT>**Save as advanced workload**</DNT>.

## View existing workloads [#view]

1. Go to <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Workloads**</DNT>
2. Filter by **Type: Advanced**
3. Click a workload to see:
   * Transaction 360 view (Summary)
   * Entity details
   * Health status
   * Activity stream

When viewing a saved Transaction 360, click <DNT>**View advanced workload**</DNT> to open it.

## Manage workloads [#manage]

### Configure health

1. Open your workload
2. Go to <DNT>**Health status > Configuration**</DNT>
3. Choose health roll-up method
4. Save

See [workload status configuration](/docs/new-relic-solutions/new-relic-one/workloads/workload-status-configuration).

### Create alerts

1. Open your workload
2. Click <DNT>**Create alert condition**</DNT>
3. Configure alerts based on:
   * Workload health status
   * Entity metrics
   * Custom NRQL queries

### Edit or delete

In the workloads list, click <Icon name="fe-more-horizontal"/> next to the workload name, then select <DNT>**Edit**</DNT> or <DNT>**Delete**</DNT>.

## How membership updates [#updates]

* **Frequency**: Every 5 minutes
* **Method**: Analyzes recent distributed traces
* **Automatic**: New entities added, removed entities dropped

## Troubleshooting [#troubleshooting]

### Missing entities

If entities are missing:

1. Verify distributed tracing is enabled
2. Check for recent trace data
3. Wait 5 minutes for next update
4. Verify entity access permissions

### Stale membership

If membership isn't updating:

1. Confirm transaction has traffic
2. Check trace data collection
3. Contact support if issue persists after 15 minutes
