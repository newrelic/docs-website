Some [New Relic integrations](https://newrelic.com/instant-observability/) require manual installation of an agent. How the behavior of those agents are configured depends on the specific agent.

## Configure APM agents [#apm-config]

The only required settings for the <InlinePopover type="apm"/> agents are your [license key](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) and [app name](/docs/apm/new-relic-apm/installation-configuration/name-your-application), which are configured when you [install the agent](/docs/agents/manage-apm-agents/installation/install-agent).

For additional configuration options, see:

* [Go](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [Java](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.NET](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## Configure browser agent [#browser-config]

See our documentation for [<InlinePopover type="browser"/> configuration](/docs/browser/new-relic-browser/configuration).

## Configure infrastructure agent [#infra-config]

The only required setting for our infrastructure agent is your <InlinePopover type="licenseKey"/>. This is configured when you [install the agent](/docs/agents/manage-apm-agents/installation/install-agent). You can also use [additional configuration options](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent) to monitor your infrastructure.

## Configure mobile agent [#mobile-config]

The only required setting for the mobile monitoring agents is the [application token](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token). This is configured when you install the mobile monitoring agent for your [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android), [iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) apps.

For additional configuration options, see:

* [Android configuration](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [iOS/tvOS configuration](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)

## Configure other New Relic tools [#other]

For configuring other tools not listed here, see the [documentation for a specific solution](https://newrelic.com/instant-observability/).
