This is the first step in diagnosing your New Relic session replay problem yourself.

If you don't find your problem listed here, you can always reach out to [New Relic Support](https://support.newrelic.com).

## Some events have aged out and may no longer appear in the timeline or event trail [#aged-out-events]

Session replay has a longer data retention than handled exception events. You may find a replay with other mobile events but no handled exception events in the event trail or represented as an error marker. This is because those events are past retention so they won't show up in a replay.
