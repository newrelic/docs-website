This guide shows you how to monitor the data collection process for New Relic Monitoring for SAP Solutions. You can use multiple methods to verify that data collection is working correctly.

## Prerequisites

- Access to <DNT>**SAP system**</DNT> with appropriate transaction permissions
- <DNT>**New Relic Monitoring for SAP Solutions**</DNT> installed and configured

## Runtime status and transmission log

Open the <DNT>**Data Provider Registry**</DNT> (transaction `/NEWR/REGISTRY`) to check runtime status and transmission details for each data provider. Click <DNT>**RuntimeStatus**</DNT> to view the last execution time and current status. Click <DNT>**Transmission Log**</DNT> to view the transmission time, HTTP response code, and transmission state for each record sent to New Relic.

## Check job status and spool

1. Use transaction `SM37` to check the status of scheduled jobs. Enter `/NEWR/*` in the <DNT>**ABAP Program Name**</DNT> field and click <DNT>**Execute**</DNT>.
2. Review the execution duration. An abnormally long execution time indicates a potential problem.
3. Check for overlapping jobs and adjust the schedule as needed.
4. Open the job spool from the job details. The spool shows the collection timestamp, transmission state, and outcome for each data provider.
5. Use the job number to correlate spool entries with the transmission log in the <DNT>**Data Provider Registry**</DNT>.

## Check application log

Data collection processes are logged in the application log (transaction `SLG1`). Query with the <DNT>**Application Log Object**</DNT> `/NEWR/DP`.

For asynchronous data providers (those with the `_COMPLETE` suffix), the job spool does not include their records. Check the application log using sub-category <DNT>**`/NEWR/DP_AGENT`**</DNT> to view their collection and transmission details.

## SAP data collection monitoring dashboard

Data collection metrics are also sent to New Relic for self-monitoring. Use the out-of-the-box <DNT>**SAP Data Collection Monitoring**</DNT> dashboard to monitor the data collection processes for each data provider. For installation instructions, see [Install monitoring components](/docs/sap-solutions/install-and-configure/install-out-of-the-box).
