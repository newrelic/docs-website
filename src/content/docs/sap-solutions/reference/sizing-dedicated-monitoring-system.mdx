- A dedicated server isn't required when the total monitored <DNT>**SIDs**</DNT> are fewer than 10 and the instances don't exceed 50. When the <DNT>**SIDs**</DNT> exceed 10 or the instances exceed 100, we recommend a dedicated server.
- Use no more than a dedicated server with two <DNT>**App Server**</DNT> instances, each with 96G of memory and 12 Core CPUs. If the monitoring system runs on a <DNT>**HANA database**</DNT>, we recommend separating the database server instances from the application server instances.
