This reference lists SAP tables retrieved for monitoring purposes by New Relic Monitoring for SAP Solutions. These tables provide data for various monitoring areas including AIF, SLT, IDOC, jobs, email, transports, RFC, business processes, and change documents.

## Authorization and table access

New Relic Monitoring for SAP authorization role provides broad table display access for the RFC user in the monitored system when setting up the RFC connections. To facilitate additional customization when retrieving additional SAP tables, including custom Z-tables, as well as retrieving IDOC or message payload data, we recommend keeping the access as broad. However, if you wish to restrict the table access, use this list to do so.

<Callout variant="important">
If any custom data collector retrieves data directly from a table, the security role may require updating for the new data provider to function.
</Callout>

## Who should use this reference

- SAP Basis administrators reviewing data collection scope
- Security teams auditing table access requirements
- Database administrators monitoring table usage
