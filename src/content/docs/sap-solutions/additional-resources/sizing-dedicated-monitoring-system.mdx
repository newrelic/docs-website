- If the total monitored SIDs are less than 10 and the instances do not exceed 50, a dedicated server is not necessary. If the SIDs are over 10 or instances are over 100, a dedicated server is recommended.
- At most, a dedicated server with two App Server instances each has 96G of memory and 12 Core CPUs should be used. If the monitoring system uses a HANA database, we recommend separating database server instances from the application server instances.
