## Verify your New Relic account
    - Do you already have a New Relic Account for SAP systems monitoring
        - **Yes**: Specify the New Relic data center your account is connected to (either US or EU).
        - **No**: Create a new account connected to one of the data centers.

## Prepare your list of SAP systems
    - For each SAP system you plan to monitor, provide the following information:
        - **System ID and clients**: Identify the system and its associated clients.
        - **Database information**: Specify the database vendor (e.g., HANA, Oracle) and its version.
        - **Software component details**: Include the SAP release and patch level for the following components:
            - **SAP_BASIS**
            - **Kernel**

## Designate a central SAP system
    - Choose one SAP system to serve as the central system for NRM4SAP installation.
        - **Installation details**: Provide the System ID and client where the ABAP component and configuration will be installed.
        - **Proxy server information**: If a proxy server is used, specify its host name and port. Include any relevant proxy server policies for accessing the New Relic One platform.
