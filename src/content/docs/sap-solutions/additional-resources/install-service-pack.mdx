title: "Install a service pack"
metaDescription: "Instructions for installing a service pack for SAP systems."
freshnessValidatedDate: never
---

Service packs are delivered as `.SAR` files with the naming convention `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`.

<Steps>
  <Step>
    Install the service pack using transaction `SPAM`.
  </Step>
  <Step>
    Load the new configuration template from client 000.
  </Step>
  <Step>
    Navigate to **Install New Release and Patch** > **Copy Template Tables** from Client 000.
  </Step>
  <Step>
    Generate the new data provider configurations by following the steps in **Install New Release and Patch** > **Generate Configuration for New Data Providers**.
  </Step>
  <Step>
    Click **Install New Release and Patch** > **Verify Installation Completeness** to verify the installation.
  </Step>
</Steps>
