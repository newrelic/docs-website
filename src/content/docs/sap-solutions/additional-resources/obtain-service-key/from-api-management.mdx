---
title: "From BTP - API management, API portal"
metaDescription: "Obtain service key from BTP - API management, API portal"
freshnessValidatedDate: never
---

<Steps>
  <Step>
      Log in to the **BTP subaccount**.
  </Step>
  <Step>
      In the **BTP cockpit**, click **Instances and Subscriptions**.
  </Step>
  <Step>
      Click **Create** at the top right.
  </Step>
  <Step>
      In the pop-up screen, select:
        - **Service**: API Management, API Portal
        - **Plan**: apiportal-apiaccess
        - **Runtime Environment**: Cloud Foundry
        - **Space**: Check with your BTP administrator
        - **Instance Name**: Follow your organization naming convention
  </Step>
  <Step>
      Check the cost disclaimer.
  </Step>
  <Step>
      Click **Next**, then select **Roles**: `APIPortal.Administrator`.
  </Step>
  <Step>
      Click **Next** to review the details.
  </Step>
  <Step>
      Click **Create**.
  </Step>
  <Step>
      Once created, locate the service under the **Instances** tab.
  </Step>
  <Step>
      Click the ellipsis (`...`) next to your instance, then click **Create Service Key**.
  </Step>
  <Step>
      Enter a Service Key Name and **Credential Type**: binding-secret in JSON format.
  </Step>
  <Step>
      Click **Create**, then download by clicking the ellipsis (`...`) next to your service key.
  </Step>
</Steps>
