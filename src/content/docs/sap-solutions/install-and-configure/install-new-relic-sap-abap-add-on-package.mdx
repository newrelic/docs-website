The ABAP component of <DNT>**NRM4SAP**</DNT> is packaged as a SAP add-on and can be installed using transaction <DNT>**SAINT**</DNT> on <DNT>**client 000**</DNT>.

## Add-on installation file

- **Initial installation**: Download the add-on file `K-xxxAOIINNEWR.SAR` to a folder on your local PC
- **Upgrade from previous version**: Use the file named `K-xxxAOUINNEWR.SAP`

## Install SAP add-on file

1. Log in to <DNT>**client 000**</DNT>
2. Type `SAINT` in the <DNT>**SAP GUI**</DNT> command field
3. Navigate to <DNT>**Installation Package**</DNT> > <DNT>**Load Packages**</DNT>
4. Choose the appropriate method based on where your installation file is located:

**Which method should you use?**
- **SAR archive from frontend** (Most common): Installation file is on your local computer
- **SAR archive from application server**: SAR file has been pre-uploaded to `/usr/sap/trans/download/`
- **EPS files from application server**: You need to extract the SAR file into PAT files on the server first

## Complete the installation

1. Click <DNT>**Start**</DNT>, select <DNT>**Installable Add-on packages for your system**</DNT>, and click <DNT>**Continue**</DNT>

2. When asked <DNT>**Do you want to add Modification Adjustment Transports to the queue?**</DNT>:
   - Click <DNT>**No**</DNT> for standard installation (default)
   - Click <DNT>**Yes**</DNT> only if your package includes adjustment transports with specific instructions

   <img
     title="Do you want to add Modification Adjustment Transports to the queue?"
     alt="A screenshot displaying question about Modification Adjustment Transports"
     src="/images/modify-adjustement-transports.webp"
     width="80%"
   />

3. Click the green check button to proceed

   <Callout variant="important">
     If the <DNT>**Errors during Disassembling OCS Packages**</DNT> page appears, click <DNT>**Ignore**</DNT>. This error is expected for certain package types and will not affect installation.
   </Callout>

4. Once the package is installed, click <DNT>**Finish**</DNT>

5. The <DNT>**Sending the runtime analysis**</DNT> popup appears. Choose one of the following:
   - <DNT>**Send**</DNT>: Opens a feedback form with the runtime analysis data in a local browser window
   - <DNT>**Display**</DNT>: Displays the runtime analysis first, then opens the feedback form
   - <DNT>**Do not send**</DNT>: Closes the dialog without sending runtime analysis data

6. Verify the installation in the <DNT>**Installed Components**</DNT> tab. The New Relic add-on should be listed with a status indicating successful installation

7. Log out from <DNT>**Client 000**</DNT>
