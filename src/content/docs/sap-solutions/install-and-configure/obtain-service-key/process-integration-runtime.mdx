This guide shows you how to obtain a service key from the SAP Business Technology Platform (BTP) for the Process Integration Runtime instance. You'll need this service key to configure monitoring for SAP Process Integration.

## Prerequisites

- Access to <DNT>**BTP subaccount**</DNT> with appropriate permissions
- Confirmation from your <DNT>**BTP administrator**</DNT> regarding <DNT>**Space**</DNT> and <DNT>**Grant-types**</DNT>

## Create the Process Integration Runtime instance

1. Log in to the <DNT>**BTP (Business Technology Platform) subaccount**</DNT>.
2. In the <DNT>**BTP cockpit**</DNT>, click <DNT>**Instances and Subscriptions**</DNT>.
3. Click <DNT>**Create**</DNT> at the top right.
4. In the pop-up screen, select:
   - <DNT>**Service**</DNT>: <DNT>**SAP Process Integration Runtime**</DNT>
   - <DNT>**Plan**</DNT>: <DNT>**API**</DNT>
   - <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>
   - <DNT>**Space**</DNT>: Check with your <DNT>**BTP administrator**</DNT>
   - <DNT>**Instance Name**</DNT>: Follow your organization naming convention

      <img
         title="Process Integration Instance"
         alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields"
         src="/images/process-integration-instance.webp"
         width="600"
      />

5. Click <DNT>**Next**</DNT>, then select:
   - <DNT>**Roles**</DNT>: `MonitoringDataRead`
   - <DNT>**Grant-types**</DNT>: <DNT>**Default Client Credentials**</DNT> (confirm with your <DNT>**BTP administrator**</DNT> and security team if other types are used)

      <img
         title="BTP Roles and Grant Types"
         alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected"
         src="/images/btp-roles-grant-types.webp"
         width="600"
      />

6. Click <DNT>**Next**</DNT> to review the details.
7. Click <DNT>**Create**</DNT>.

## Generate and download the service key

1. Once created, locate the service under the <DNT>**Instances**</DNT> tab.

      <img
         title="BTP Instance"
         alt="The Instances tab showing the created Process Integration Runtime instance"
         src="/images/btp-instance.webp"
         width="600"
      />

2. Click the ellipsis (`...`) in the row of your newly created instance, then select <DNT>**Create Service Key**</DNT>.
3. Enter a <DNT>**Service Key Name**</DNT> and select <DNT>**Key Type**</DNT>: `ClientId/Secret`.

      <img
         title="BTP New Service Key"
         alt="The Create Service Key dialog showing Service Key Name and Key Type fields"
         src="/images/btp-new-service-key.webp"
         width="600"
      />
4. Click <DNT>**Create**</DNT>.
5. Click the ellipsis (`...`) in the row of your <DNT>**service key**</DNT>, then select <DNT>**Download**</DNT>.

      <img
         title="BTP Download Service Key"
         alt="The service key row showing the ellipsis menu with the Download option"
         src="/images/btp-download.webp"
         width="600"
      />

The <DNT>**service key file**</DNT> downloads as a <DNT>**JSON file**</DNT>. Store it securely - you'll need the credentials from this file to configure your monitoring setup.
