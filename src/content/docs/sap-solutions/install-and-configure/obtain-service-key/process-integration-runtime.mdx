This guide shows you how to obtain a service key from the SAP Business Technology Platform (BTP) for the Process Integration Runtime instance. You'll need this service key to configure monitoring for SAP Process Integration.

## Prerequisites

- Access to BTP subaccount with appropriate permissions
- Confirmation from your BTP administrator regarding Space and Grant-types

## Create the Process Integration Runtime instance

1. Log in to the **BTP (Business Technology Platform) subaccount**.
2. In the **BTP cockpit**, click **Instances and Subscriptions**.
3. Click **Create** at the top right.
4. In the pop-up screen, select:
   - **Service**: SAP Process Integration Runtime
   - **Plan**: API
   - **Runtime Environment**: Cloud Foundry
   - **Space**: Check with your BTP administrator
   - **Instance Name**: Follow your organization naming convention

      <img
         title="Process Integration Instance"
         alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields"
         src="/images/process-integration-instance.webp"
         width="600"
      />

5. Click **Next**, then select:
   - **Roles**: `MonitoringDataRead`
   - **Grant-types**: Default Client Credentials (confirm with your BTP administrator and security team if other types are used)

      <img
         title="BTP Roles and Grant Types"
         alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected"
         src="/images/btp-roles-grant-types.webp"
         width="600"
      />

6. Click **Next** to review the details.
7. Click **Create**.

## Generate and download the service key

1. Once created, locate the service under the **Instances** tab.

      <img
         title="BTP Instance"
         alt="The Instances tab showing the created Process Integration Runtime instance"
         src="/images/btp-instance.webp"
         width="600"
      />

2. Click the ellipsis (`...`) in the row of your newly created instance, then select **Create Service Key**.
3. Enter a **Service Key Name** and select **Key Type**: `ClientId/Secret`.

      <img
         title="BTP New Service Key"
         alt="The Create Service Key dialog showing Service Key Name and Key Type fields"
         src="/images/btp-new-service-key.webp"
         width="600"
      />
4. Click **Create**.
5. Click the ellipsis (`...`) in the row of your service key, then select **Download**.

      <img
         title="BTP Download Service Key"
         alt="The service key row showing the ellipsis menu with the Download option"
         src="/images/btp-download.webp"
         width="600"
      />

The service key file downloads as a JSON file. Store it securely - you'll need the credentials from this file to configure your monitoring setup.
