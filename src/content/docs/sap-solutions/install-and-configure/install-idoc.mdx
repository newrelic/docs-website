freshnessValidatedDate: never
---

<Steps>
  <Step>
    Go to **[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**.
  </Step>

  <Step>
    Select **apps & visualizations**, and search for **IDOC Explorer**.
  </Step>

  <Step>
    Select **Manage access**. Choose the account for access to the app.

    <img
      title="IDOC Explorer"
      alt="A screenshot displaying the manage access settings for IDOC Explorer"
      src="/images/idoc-explorer.webp"
    />
  </Step>

  <Step>
    Click **Update**. The IDOC Explorer will be visible when viewing IDOC entities.
  </Step>

</Steps>
