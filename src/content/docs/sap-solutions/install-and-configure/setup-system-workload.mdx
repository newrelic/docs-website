---
title: "Setup system workload"
metaDescription: "Information on how to setup the system workload"
freshnessValidatedDate: never
---

<Steps>
  <Step>
    Navigate to **All entities > Workloads > + Create a workload**. See [Create and define workloads](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/) to know more on how to create a workload.
  </Step>

  <Step>
    Enter the name for the workload, and select the required account from the drop down list box.
  </Step>

  <Step>
    Click the **+** icon next to the **Entity Type = All** option and enter the condition `sap.system = {SAP System SID}` on the filter bar.

    <img
      title="SAP System"
      alt="A screenshot displaying the SAP System filter"
      src="/images/sap-system.webp"
    />
  </Step>

  <Step>
    Click **+ Add this query** to add the query to the workload definition.
  </Step>

  <Step>
    Click **Create a workload** to create the workload.
  </Step>

  <Step>
    The workload will be visible on the **All entities** UI and includes all the entities tagged with the SAP System.
  </Step>
</Steps>
