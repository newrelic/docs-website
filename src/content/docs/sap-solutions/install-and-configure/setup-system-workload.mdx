## Set up system workload

1. Go to **All entities > Workloads > + Create a workload**. For more information, see [Create and define workloads](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/).
2. Enter a name for your workload and select an account from the dropdown list.
3. Next to **Entity Type = All**, select the **+** icon and enter `sap.system = {SAP System SID}` in the filter bar.

    <img
      title="SAP System"
      alt="A screenshot displaying the SAP System filter"
      src="/images/sap-system.webp"
    />

4. Select **+ Add this query** to add the query to your workload definition.
5. Select **Create a workload**.

Your new workload appears in **All entities** and includes all entities tagged with your SAP system.

## Install service pack

Service packs are delivered as `.SAR` files with the naming convention `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`.

1. Install the service pack using transaction `SPAM`.
2. Load the new configuration template from client 000.
3. Navigate to **Install New Release and Patch** > **Copy Template Tables** from Client 000.
4. Generate the new data provider configurations by following the steps in **Install New Release and Patch** > **Generate Configuration for New Data Providers**.
5. Select **Install New Release and Patch** > **Verify Installation Completeness** to verify the installation.
