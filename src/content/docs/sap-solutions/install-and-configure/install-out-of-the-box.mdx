New Relic monitoring for SAP includes a list of out-of-the-box monitoring components including dashboards, alert policies, and the IDOC Explorer. Follow the instructions below to locate and install them.

## Before you begin

To perform these tasks, you must have New Relic Admin privileges on the account.

<Callout variant="important">
For the **SAP Cockpit** dashboard, you must [create a lookup table](#create-lookup-table) named `NR_LKUP_SYSTEM` before installation.
</Callout>

## Install dashboards

### Create lookup table [#create-lookup-table]

<Callout variant="tip">
Only required for the **SAP Cockpit** dashboard.
</Callout>

The **SAP Cockpit** dashboard uses a lookup table for SAP systems and their environments.

1. Create the lookup table with two columns: `SYS_ID` and `Env`, and save it as a CSV file. Here is a sample of the lookup file:

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. Go to **Log > Lookup Tables**.
3. Click **Add a table**.
4. In the **Add a lookup table** sidebar, name the table to **NR_LKUP_SYSTEM**, and upload the CSV file. See [Lookup tables](/docs/logs/ui-data/lookup-tables-ui/) for more information.
5. To verify the table content, use the query builder with `FROM lookup(NR_LKUP_SYSTEM) SELECT *`, then click **Run**.

### Install the dashboards [#install-dashboards]

1. Go to **[one.newrelic.com](https://one.newrelic.com)** > **Integrations & Agents**, and search for **New Relic Monitoring for SAP Solutions** to automatically download all dashboards
2. Alternatively, you can go to **Dashboards > Recommended Dashboards**, click **View all**, and search for SAP. See [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) for more information.
3. Install these dashboards:
   - **SAP Dashboards**
   - **SAP Cockpit**
   - **SAP Data Collection Monitoring**
   - **SAP Data Ingestion Estimate**

### Add dashboard filter [#add-dashboard-filter]

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Dashboards**.
2. Enable facet linking for filtering:
   - Click the ellipsis (**…**) on the widget's upper right corner
   - Select **Edit** to open the edit screen
   - Toggle the **Filter the current dashboard** switch
3. Facet linking can be enabled for tables, pie charts, and bar chart widgets to filter by SAP system, instance, IDOC status, jobs, etc.

## Install alert policies

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Alerts > Alert Policies**.
2. Search and click **New Relic Monitoring for SAP Solutions alert policy**.

      <img
        title="New Relic Monitoring for SAP Solutions alert policy"
        alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy"
        src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp"
        width=""
        />

3. In the **Alert condition** tab, select the account and toggle the alert policy to **Enable**. See [Manage your policies](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) to know how to manage the policies.
4. Go back and search for **Nr4sap default alert policy**. Toggle on the alert condition you'd like to enable.

## Install IDOC Explorer

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**.
2. Select **Apps & Visualizations**, and search for **IDOC Explorer**.
3. Select **Manage access**. Choose the account for access to the app.

   <img
     title="IDOC Explorer"
     alt="A screenshot displaying the manage access settings for IDOC Explorer"
     src="/images/idoc-explorer.webp"
   />

4. Click **Update**. The IDOC Explorer will be visible when viewing IDOC entities.
