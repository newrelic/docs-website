New Relic monitoring for SAP includes a list of out-of-the-box monitoring components including dashboards, alert policies, and the IDOC Explorer. Follow the instructions below to locate and install them.

## Before you begin

To perform these tasks, you must have New Relic Admin privileges on the account.

<Callout variant="important">
For the <DNT>**SAP Cockpit**</DNT> dashboard, you must [create a lookup table](#create-lookup-table) named `NR_LKUP_SYSTEM` before installation.
</Callout>

## Install dashboards

### Create lookup table [#create-lookup-table]

<Callout variant="tip">
Only required for the <DNT>**SAP Cockpit**</DNT> dashboard.
</Callout>

The <DNT>**SAP Cockpit**</DNT> dashboard uses a <DNT>**lookup table**</DNT> for <DNT>**SAP systems**</DNT> and their environments.

1. Create the <DNT>**lookup table**</DNT> with two columns: `SYS_ID` and `Env`, and save it as a <DNT>**CSV file**</DNT>. Here is a sample of the lookup file:

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. Go to <DNT>**Log > Lookup Tables**</DNT>.
3. Click <DNT>**Add a table**</DNT>.
4. In the <DNT>**Add a lookup table**</DNT> sidebar, name the table to <DNT>**NR_LKUP_SYSTEM**</DNT>, and upload the <DNT>**CSV file**</DNT>. See [Lookup tables](/docs/logs/ui-data/lookup-tables-ui/) for more information.
5. To verify the table content, use the <DNT>**query builder**</DNT> with `FROM lookup(NR_LKUP_SYSTEM) SELECT *`, then click <DNT>**Run**</DNT>.

### Install the dashboards [#install-dashboards]

1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> > <DNT>**Integrations & Agents**</DNT>, and search for <DNT>**New Relic Monitoring for SAP Solutions**</DNT> to automatically download all dashboards
2. Alternatively, you can go to <DNT>**Dashboards > Recommended Dashboards**</DNT>, click <DNT>**View all**</DNT>, and search for SAP. See [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) for more information.
3. Install these dashboards:
   - <DNT>**SAP Dashboards**</DNT>
   - <DNT>**SAP Cockpit**</DNT>
   - <DNT>**SAP Data Collection Monitoring**</DNT>
   - <DNT>**SAP Data Ingestion Estimate**</DNT>

### Add dashboard filter [#add-dashboard-filter]

1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Dashboards**</DNT>.
2. Enable facet linking for filtering:
   - Click the ellipsis (<DNT>**…**</DNT>) on the widget's upper right corner
   - Select <DNT>**Edit**</DNT> to open the edit screen
   - Toggle the <DNT>**Filter the current dashboard**</DNT> switch
3. <DNT>**Facet linking**</DNT> can be enabled for tables, pie charts, and bar chart widgets to filter by <DNT>**SAP system**</DNT>, <DNT>**instance**</DNT>, <DNT>**IDOC status**</DNT>, <DNT>**jobs**</DNT>, etc.

## Install alert policies

1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Alerts > Alert Policies**</DNT>.
2. Search and click <DNT>**New Relic Monitoring for SAP Solutions alert policy**</DNT>.

      <img
        title="New Relic Monitoring for SAP Solutions alert policy"
        alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy"
        src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp"
        width=""
        />

3. In the <DNT>**Alert condition**</DNT> tab, select the account and toggle the <DNT>**alert policy**</DNT> to <DNT>**Enable**</DNT>. See [Manage your policies](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) to know how to manage the policies.
4. Go back and search for <DNT>**Nr4sap default alert policy**</DNT>. Toggle on the <DNT>**alert condition**</DNT> you'd like to enable.

## Install IDOC Explorer

1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT>.
2. Select <DNT>**Apps & Visualizations**</DNT>, and search for <DNT>**IDOC Explorer**</DNT>.
3. Select <DNT>**Manage access**</DNT>. Choose the account for access to the app.

   <img
     title="IDOC Explorer"
     alt="A screenshot displaying the manage access settings for IDOC Explorer"
     src="/images/idoc-explorer.webp"
   />

4. Click <DNT>**Update**</DNT>. The <DNT>**IDOC Explorer**</DNT> will be visible when viewing <DNT>**IDOC entities**</DNT>.
