### Occasionally we use h3s [#h3-1]

0-level **spell** ability damage action caster level charm **subschool** copper piece dispel check flank helpless lava effects masterwork natural `weapon negate planning` domain prerequisite reflex save retribution domain scry splash weapon stack swift action trade domain.


## Examples

There's a few ways we handle examples.

Usually it's a box:

One style is a static box

```
concealment critical roll darkvision destruction domain extraplanar subtype giant type gnome domain grappling healing subschool humanoid type incorporeal subtype
```

Sometimes it's in a collapser:

<CollapserGroup>
  <Collapser

    title="This is a collapsible example"
  >
    It contains stuff
  </Collapser>

  <Collapser
    id="examplecollapser-2"
    title="This is another collapsible example"

  </Collapser>
</CollapserGroup>
