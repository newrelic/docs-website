In addition to the [required and optional environment variables for the .NET agent](/docs/apm/agents/net-agent/configuration/net-agent-configuration/#environment-variables), the following environment variables specific to Azure function monitoring can be set:

<table>
    <thead>
        <tr>
            <th>Environment variable</th>
            <th>Default value</th>
            <th>Options</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>`NEW_RELIC_AZURE_FUNCTION_MODE_ENABLED`</td>
            <td>`1`</td>
            <td>`1`, `0`</td>
            <td>Azure Function mode is enabled by default. You can disable Azure Function instrumentation by setting the value to `0`.</td>
        </tr>
    </tbody>
</table>
