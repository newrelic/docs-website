Use `${{ .workflowInputs.variableName }}` to pass dynamic values at runtime in endpoint URLs and other input fields.

## Example

  ```json
    {
      "inputs": [
        {
          "key": "urlParams",
          "value": "{\"filter\": \"active\"}"
        },
        {
          "key": "headers",
          "value": "{\"Api-Key\": \"your-api-key\"}"
        }
      ]
    }
  ```
