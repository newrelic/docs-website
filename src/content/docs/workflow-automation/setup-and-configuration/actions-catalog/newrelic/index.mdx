This page provides an overview of New Relic actions available in the workflow automation actions catalog. These actions enable you to integrate New Relic platform capabilities into your workflow definitions. Select a category below to view detailed documentation for each action type.

## Prerequisites

Before using New Relic actions in workflow automation, ensure you have:

  * A New Relic account with appropriate permissions.
  * A New Relic license key (if sending data to a different account).
  * The necessary permissions for the specific New Relic services you plan to use.

See [License Key](/docs/apis/intro-apis/new-relic-api-keys/#license-key) for information on how to create and manage your New Relic Account License Key.

## Available action categories

### Data Ingest actions

Send custom events and logs to New Relic from your workflows. These actions allow you to programmatically ingest telemetry data into your New Relic account.

**Available actions:**
- `newrelic.ingest.sendEvents` - Send custom events to New Relic
- `newrelic.ingest.sendLogs` - Send logs to New Relic

[View Data Ingest actions documentation](/docs/workflow-automation/setup-and-configuration/actions-catalog/newrelic/ingest)

### NerdGraph actions

Execute GraphQL queries and mutations against the New Relic NerdGraph API. NerdGraph provides programmatic access to all New Relic data and configuration capabilities.

**Available actions:**
- `newrelic.nerdgraph.execute` - Execute a GraphQL query or mutation against New Relic NerdGraph API

[View NerdGraph actions documentation](/docs/workflow-automation/setup-and-configuration/actions-catalog/newrelic/nerdgraph)

### NRDB Query actions

Execute NRQL queries against the New Relic Database (NRDB) to retrieve and analyze your telemetry data. These actions support cross-account queries for consolidated data analysis.

**Available actions:**
- `newrelic.nrdb.query` - Run a NRQL query against NRDB

[View NRDB Query actions documentation](/docs/workflow-automation/setup-and-configuration/actions-catalog/newrelic/nrdb)

### Notification actions

Send notifications through various channels using New Relic destinations. These actions integrate with platforms like Slack, Microsoft Teams, and Email to deliver workflow notifications.

**Available actions:**
- `newrelic.notification.send` - Send notification to a New Relic destination (e.g., Slack)
- `newrelic.notification.sendMicrosoftTeams` - Send notification to Microsoft Teams
- `newrelic.notification.sendEmail` - Send email notification with optional attachments

[View Notification actions documentation](/docs/workflow-automation/setup-and-configuration/actions-catalog/newrelic/notification)
