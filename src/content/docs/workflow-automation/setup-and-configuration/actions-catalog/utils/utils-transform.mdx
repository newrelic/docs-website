## Transform an input data value to a CSV format [#utils-transform-toCSV]


This is used to transform various type of input (JSON, map) to a CSV format.

  ```yaml
      name: nrqltocsv

steps:
  - name: queryForLog
    type: action
    action: newrelic.nrql.query
    version: 1
    inputs:
      accountIds: ['${{ .workflowInputs.accountId }}']
      query: ${{ .workflowInputs.nrql }}

  - name: csv1
    type: action
    action: utils.transform.toCSV
    version: 1
    inputs:
      data: ${{ .steps.queryForLog.outputs.results | tostring }}
      selectors:
        - name: csv
          expression: ".csv"
  ```
