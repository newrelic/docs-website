This page provides an overview of utility actions available in the workflow automation actions catalog. These actions enable you to perform common data transformation and utility operations in your workflow definitions.

## Available utility actions

The utility actions are organized into the following categories:

### DateTime actions

Transform and manipulate date and time values:

* **[utils.datetime.fromEpoch](/docs/workflow-automation/setup-and-configuration/actions-catalog/utils/datetime)** - Convert epoch timestamps to formatted date/time strings with timezone support

### Transform actions

Convert data between different formats:

* **[utils.transform.toCSV](/docs/workflow-automation/setup-and-configuration/actions-catalog/utils/transform)** - Transform JSON or map data into CSV format

### UUID actions

Generate unique identifiers:

* **[utils.uuid.generate](/docs/workflow-automation/setup-and-configuration/actions-catalog/utils/uuid)** - Generate RFC-compliant V4 UUIDs

## Getting started

To use these utility actions in your workflows:

1. Identify the utility action you need from the categories above
2. Click on the action link to view detailed documentation including:
   - Input parameters and their types
   - Output format and structure
   - Example workflow definitions
3. Incorporate the action into your workflow definition

## Common use cases

* **Data transformation**: Convert NRQL query results to CSV for reporting
* **Time handling**: Transform epoch timestamps from alerts into human-readable dates
* **Unique identifiers**: Generate correlation IDs for tracking workflow executions
