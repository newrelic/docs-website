New Relic actions catalog provides actions that can be performed against your infrastructure and integrations. You can orchestrate and automate your end-to-end processes by linking together actions that perform tasks in your cloud providers, and New Relic accounts.

Following lists are all the available actions that can be used in a data transformation:

## AWS

Begin by setting up AWS credentials, which includes creating IAM users and roles, and configuring static and session AWS credentials for integration with workflow automation AWS actions.
