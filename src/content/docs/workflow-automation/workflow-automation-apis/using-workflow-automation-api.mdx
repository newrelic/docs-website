Start by using the [NerdGraph API explorer](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) to create and edit workflow automation information. Definitions for each field are available in NerdGraph. For help with the GraphiQL IDE, see the [NerdGraph API explorer tutorial](/docs.newrelic.com/docs/apis/nerdgraph/get-started/nerdgraph-explorer). For authentication details related to secret management, refer to [Our GraphQL API](/docs.newrelic.com/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph).

This document describes some of the available Workflow Automation API calls. For a description of all our available APIs, see [Introduction to APIs](/docs/apis/intro-apis/introduction-new-relic-apis).
