tags:

metaDescription: "Returns all action definitions stored in the Workflow Service. This operation is behind NerdGraph, under account stitch point."
freshnessValidatedDate: never
---

## Request parameters [#requestparameters]

  None

## Response elements [#responseparameters]

  If the action is successful, the service sends back the following data is returned:

  - List of `ActionDefinition` objects

## Example [#example]

```graphql
    {
      actor {
        account(id: 12345678) {
          workflowAutomation {
            actionDefinitions {
              results {
                description
                inputs {
                  name
                  required
                  secretAllowed
                }
                name
                outputs {
                  name
                }
                timeout {
                  action
                }
                version
              }
            }
          }
        }
      }
    }
```
