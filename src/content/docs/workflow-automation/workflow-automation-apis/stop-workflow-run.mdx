This API stops a running workflow.

## Input parameters

<table>
  <thead>
    <tr>
      <th style={{ width: "200px" }}>Parameter</th>
      <th>Type</th>
      <th>Required</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`accountId`</td>
      <td>Number</td>
      <td>Yes</td>
      <td>Your account ID.</td>
    </tr>
    <tr>
      <td>`runId`</td>
      <td>String</td>
      <td>Yes</td>
      <td>The unique identifier of the workflow run to stop. Obtained from `StartWorkflowRun` API.</td>
    </tr>
  </tbody>
</table>

## Sample request

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```

## Sample response

The mutation returns:

- `runId`: The unique identifier of the stopped workflow run. Returns an empty `runId` if the workflow is not found.
