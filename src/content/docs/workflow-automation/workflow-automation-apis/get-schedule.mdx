This API gets and returns a schedule.

## Parameters

<table>
  <thead>
    <tr>
      <th style={{ width: "200px" }}>Parameter</th>
      <th>Type</th>
      <th>Required</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`account.id`</td>
      <td>Number</td>
      <td>Yes</td>
      <td>Your account ID.</td>
    </tr>
    <tr>
      <td>`schedule.id`</td>
      <td>String</td>
      <td>Yes</td>
      <td>The unique identifier of the schedule to retrieve.</td>
    </tr>
  </tbody>
</table>

## Example [#example]

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## Response

The query returns:

- `schedule.id`: The unique identifier of the schedule.
