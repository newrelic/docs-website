## Request parameters [#requestparameters]

  NA

## Response elements [#responseparameters]

  If the action is successful, the service sends back the following data:

  - List of `SchedulesResult` objects. The objects in the `SchedulesResult` objects are meant to be minimal outlines of the schedule and workflow, query the dedicated endpoints to get additional details for any specific schedule or workflow.

    - ScheduleOutline
      - id
        - `String`
    - WorkflowDefinitionOutline
      - name
        - `String`
      - scope
        - `Scope`
          - id
            - `String`
          - type
            - `ScopeType`

## Example [#example]

```graphql
    {
      actor {
        account(id: 12345678) {
          workflowAutomation {
            schedules {
              results {
                definition {
                  name
                  scope {
                    id
                    type
                  }
                  version
                }
                schedule {
                  id
                }
              }
            }
          }
        }
      }
    }
```
