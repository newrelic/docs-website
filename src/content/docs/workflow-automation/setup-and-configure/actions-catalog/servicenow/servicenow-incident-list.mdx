Gets a list of incidents from ServiceNow using the Table API.

For authentication setup, see [ServiceNow Incident actions](/docs/workflow-automation/setup-and-configure/actions-catalog/servicenow/servicenow-incident).

  <TabsPageItem id="servicenow-incident-list-example">
  <table>
    <thead>
        <tr>
            <th>Workflow example</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>
```yaml
name: get-servicenow-incident-list
description: to get list of incident from ServiceNow
steps:
  - type: action
    name: listIncidents
    action: serviceNow.incident.list
    version: 1
    inputs:
      domain: "your-domain"
      apiKey: "${{ :secrets:serviceNowApiKey }}"
      parameters:
        sysparm_fields: "number,short_description,description"
        sysparm_limit: "5"
      selectors:
        - name: incidents
          expression: .responseBody.result
    next: end
```
            </td>
