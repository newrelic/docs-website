Follow these quick steps to get started with New Relic and CodeStream.

<Callout title="Preview release">
CodeStream's integration with New Relic One is a preview release limited to New Relic One accounts on our US data center, and your use is subject to the pre-release policy. (This does not apply to all other CodeStream functionality.)
</Callout>

## Start with a New Relic account [#account]

If you don't have a New Relic account, sign up at [newrelic.com/signup](https://newrelic.com/signup). It's free, forever!

## Install CodeStream via New Relic [#quickstart]

Use our [CodeStream quickstart](https://developer.newrelic.com/instant-observability/codestream/29bd9a4a-1c19-4219-9694-0942f6411ce7) to install it for your IDE.

### Install CodeStream in your IDE [#install-ide]

You can also install CodeStream via your Visual Studio Code and Visual Studio extensions menu or JetBrains plugins menu.

## Install New Relic [#install-new-relic]

Follow the steps in our [Add your data](https://one.newrelic.com/launcher/nr1-core.settings?pane=eyJuZXJkbGV0SWQiOiJ0dWNzb24ucGxnLWluc3RydW1lbnQtZXZlcnl0aGluZyJ9) UI page to get data flowing in.
