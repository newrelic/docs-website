You can install the CodeStream extension for your specific IDE.

<CollapserGroup>
  <Collapser
    className="freq-link"
    id="vscode"
    title="Visual Studio Code"
  >
    Download and install CodeStream for [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream). You can also install it directly in Visual Studio Code via the extensions marketplace.
  </Collapser>

  <Collapser
    className="freq-link"
    id="visual-studio"
    title="Visual Studio"
  >
    Download and install CodeStream for [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22). You can also install it directly in Visual Studio via the extensions marketplace.
  </Collapser>

  <Collapser
    className="freq-link"
    id="jetbrains"
    title="JetBrains"
  >
    Download and install CodeStream for [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream). You can also install it from the JetBrains plugins menu.
  </Collapser>
</CollapserGroup>

If you're using VS Code, some CodeStream features require a language support extension for [Java](https://marketplace.visualstudio.com/items?itemName=redhat.java), [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp), [PHP](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client), [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python), [Go](https://marketplace.visualstudio.com/items?itemName=golang.go), or [Ruby](https://marketplace.visualstudio.com/items?itemName=Shopify.ruby-lsp).

## Sign in [#signin]

Once you have the CodeStream extension installed in your IDE, sign in with your New Relic credentials.
