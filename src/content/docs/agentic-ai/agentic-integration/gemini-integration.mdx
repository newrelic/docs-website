The New Relic agent for Google Gemini Code Assist extension provides intelligent insights and guidance tailored to optimize application performance, monitor infrastructure, and manage deployments—all seamlessly integrated within Gemini.

This guide will walk you through the process of installing and using the New Relic agent within the Gemini Code Assist.

## Install the New Relic agent in Gemini Code Assist [#install-new-relic-agent]

### Prerequisites [#prerequisites]

Before you begin, make sure that the following prerequisites are met to ensure a smooth setup process:

- An IDE is installed on your system.
- You have access to Google Gemini Code Assist.
- The Gemini Code Assist is installed in your IDE. For more information, refer [Google Gemini Code Assist documentation](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="tip">
If you want to restrict access to Gemini Code Assist for certain roles, configure the settings on Gemini Code Assist. To restrict access to New Relic AI for specific roles or employees, you can use New Relic's Role-Based Access Control (RBAC). For more information, refer [New Relic user management](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

**To enable New Relic agent in Gemini Code Assist:**

1. On your <DNT>Google Cloud Console</DNT>, search for <DNT>**NewRelic**</DNT>, and enable the extension. A sign-in dialog window opens.
2. Provide your New Relic credential to login. The **New Relic successfully linked.** message displays.

    Now, you can use New Relic agent within the Gemini Code Assist chat extension in your IDE. For more information, refer [using the New Relic agent in Gemini Code Assist](#communicate-with-new-relic-agent-in-gemini-code-assist).


## Communicate with New Relic agent in Gemini Code Assist [#communicate-with-new-relic-agent-in-gemini-code-assist]

The New Relic agent within Gemini Code Assist can assist with a variety of tasks and offer valuable insights. To interact with the New Relic agent, in the chat input box, type `@newrelic`. 


### Chat message structure [#chat-message-structure]

To interact with the New Relic agent, enter your prompt as follows:

``` @newrelic <your message> ```

Where:

- `@newrelic`: Interaction with the New Relic agent.
- `<your message>`: Action to submit to the agent for processing.
