Through the New Relic UI, you can access a collection of popular pre-built dashboards, to help you instantly visualize your data for common use cases.

Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Dashboards** to open the dashboards page.

## How to install pre-built dashboards [#use]

To install a pre-built dashboard:

1. From the dashboards page, search for the pre-built dashboard you want to customize.
2. Click the pre-built dashboard to select it.
3. Select the account in which to add the data and dashboard.
4. Follow the instructions in the **Setup data source** section. If you've already instrumented the data source, you can skip this step.
5. Click **View dashboard** to deploy the dashboard and start visualizing your data. Note that you can customize your new pre-built dashboard (see [Manage your dashboard](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/)). You can also change the name and settings, add new content and new widgets, refine the charts, etc.


## How to view more templates [#more-templates]

Browse the entire dashboard catalog to find dashboard templates for all the available integrations. Click the **View All** link next to the popular pre-built dashboards title to see and search for all the available pre-built dashboards. You can use the <Icon name="fe-chevron-left"/> and <Icon name="fe-chevron-right"/> buttons to browse more pre-built dashboards.
