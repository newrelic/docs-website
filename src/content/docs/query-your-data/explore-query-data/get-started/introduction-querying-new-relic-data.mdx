You can query your data in New Relic in several ways, including [in the UI](#query-ui) or [via API](#query-apis).

## Explore data [#explore]

Before querying data, consider using our [data explorer](/docs/query-your-data/explore-query-data/explore-data/introduction-data-explorer/). The data explorer in the New Relic UI requires no querying knowledge.

To better understand your data stored in New Relic, see [Data types](/docs/using-new-relic/data/understand-data/new-relic-data-types).

## Query data in the UI [#query-ui]

Reasons to query your data from the New Relic UI:

* To answer a specific question
* To create a custom chart or dashboard
* To access and navigate your data in a quick, visual way

To query your data in the New Relic UI, you can use [query languages](#use-nrql), including our [New Relic query language](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) or our [PromQL-style](https://prometheus.io/docs/prometheus/latest/querying/basics/) query language). You can also use [simpler query interfaces](#nrqlless-query) that don't require knowledge of how to write a query.

<Callout variant="tip">
  Access your data easily in [one.newrelic.com](https://one.newrelic.com): Click the **Browse data** dropdown, then select the data type (metrics, events, logs, and traces) you want to explore.
</Callout>

### Use NRQL or PromQL-style queries in the UI [#use-nrql]

There are two ways to write your own queries to retrieve data and build charts:

* [Query builder in NRQL mode](/docs/chart-builder/use-chart-builder/choose-data/use-advanced-nrql-mode-specify-data): query using [New Relic query language](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language) (NRQL), the same language we use to build most of our UI experiences.
* [Query builder in PromQL-style mode](/docs/query-your-data/explore-query-data/chart-builder/use-advanced-promql-mode-specify-data): write basic queries using a PromQL-style query.

### Query using simple UI interfaces [#nrqlless-query]

[New Relic One](/docs/new-relic-one/use-new-relic-one/get-started/introduction-new-relic-one) offers several experiences that don't require knowledge of NRQL:

* [Data explorer](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer): an intuitive data navigator to create visualizations for [events](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data) and [metrics](/docs/using-new-relic/data/understand-data/new-relic-data-types#metrics).
* [Query builder (basic mode)](/docs/chart-builder/use-chart-builder/choose-data/use-basic-mode-specify-data): a simple, query-less, chart-building experience.
* [Distributed tracing query](/docs/understand-dependencies/distributed-tracing/ui-data/additional-distributed-tracing-features-new-relic-one): a specialized UI for querying traces.
* [Logs query](/docs/logs/new-relic-logs/ui-data/query-syntax-logs): a specialized UI for querying [New Relic Logs](/docs/logs/new-relic-logs/get-started/introduction-new-relic-logs) data.

## Query data via API [#query-apis]

You can use APIs to retrieve and query your data in New Relic. For example, you can run [NRQL](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language) (our query language) queries with [NerdGraph](/docs/apis/graphql-api/get-started/introduction-new-relic-nerdgraph) (our GraphQL API).

For more information, see the [introduction to New Relic APIs](/docs/apis/get-started/intro-apis/introduction-new-relic-apis).
