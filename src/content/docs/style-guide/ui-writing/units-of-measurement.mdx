## Basics

-   Separate units of measurement from their value with a single space.
-   For measurements over time, insert a forward slash between the unit of measurement and the time unit.

| Unit of measurement | Do        | Don't    |
| --------- | --------- | -------- |
| requests per minute  |29.5K rpm | 29.5Krpm |
| requests per second  |14 rps  | 14rps  |
| pages per minute     |6.33 ppm  | 6.33ppm  |
| kilobytes per second |16 KB/s   | 16 KB s  |
