This is the template for Observability maturity implementation guides. To help you along, refer to:

* The [style guide introduction](/docs/style-guide/writing-strategies/introduction-style-guide)
* Other observability maturity [implementation guides](/docs/new-relic-solutions/observability-maturity/introduction)

To get your implementation guide to show up in the left nav as it should, edit `~src/nav/new-relic-solutions.yml`. Don't forget to add a link to it to the [Observability Maturity introduction page](/docs/new-relic-solutions/observability-maturity/introduction).
