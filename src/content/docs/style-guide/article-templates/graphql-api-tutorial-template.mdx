**This is a template.** Feel free to copy the text in this template to create your own GraphQL API tutorial, then fill in details that will help users learn how to use GraphQL in practical ways.

Ideally the final version of your GraphQL doc will include:
* An introduction paragraph describing what problem or use case the users want to solve, or why the API can help them query to get this data
* Three to five query examples that show interesting data they can get from GraphQL

**Sample intro paragraph:** You can use the [New Relic GraphQL API](/docs/introduction-new-relic-graphql) to <var>\[add key use cases here]</var>. To construct these queries and see responses, you can use the [New Relic GraphQL Explorer](https://api.newrelic.com/graphiql). This document explains some of the available functions to query <var>\[data to be queried]</var>.

## Example query #1 with GraphQL [#basic-queries]

Add a description here for your example query. Tell users why they want to query this kind of data. Fill out the first block with your code, and the second with the data returned by the query:

```
{
   actor {
      account(id: <var>YOUR_ACCOUNT_ID)</var> {
         nrql(query: "<a href="/docs/insights/nrql-new-relic-query-language/nrql-resources/nrql-syntax-components-functions#state-select">SELECT</a> count(*) <a href="/docs/insights/nrql-new-relic-query-language/nrql-resources/nrql-syntax-components-functions#sel-from">FROM</a> <a href="https://docs.newrelic.com/docs/insights/insights-data-sources/default-data/apm-default-event-attributes-insights#transaction-event">Transaction</a> <a href="/docs/insights/nrql-new-relic-query-language/nrql-resources/nrql-syntax-components-functions#sel-since">SINCE</a> 1 HOUR AGO") {
            results
         }
      }
   }
}
```

This query returns the following:

```
{
   "data": {
      "actor": {
         "account": {
            "nrql": {
               "results": [
                 {
                  "count": <mark>1000</mark>
                 }
                ]
            }
         }
      }
   }
}
```

The actual value of the count varies depending on your data. Use the [New Relic GraphQL Explorer](https://api.newrelic.com/graphiql) to experiment with queries.
