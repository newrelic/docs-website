If you're a [full platform user](/docs/accounts/accounts-billing/new-relic-one-pricing-users/users-roles#user-type), you theoretically have full access to all New Relic features, but whether you can use all those features depends on what role you're assigned within your New Relic organization.

For details on how New Relic permissions work, see [Important user and access management concepts](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts). To see the granular synthetics-related capabilities, see [Capabilities](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/#synthetics).
