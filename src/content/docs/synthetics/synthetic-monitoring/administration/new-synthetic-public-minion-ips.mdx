An updated runtime will be launched to support newer versions of Chrome. As a result, you'll need to reconfigure your firewall's allow list, bot detection software, and other related applications to accept network requests from new IP address ranges. 

Monitors created after the new runtime launch will use the new IP address space. Existing monitors will be moved to the new IP address space in the future. Check out [Synthetic monitor public minion IPs](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/) for more information about IP addresses and synthetic monitoring.

## New IP address ranges for public minions [#us_new_ip]

These ranges will cover both US and EU data centers.
