Synthetic monitors returns a number of additional response codes beyond standard HTTP response codes, visible in the [**Resources** page](/docs/synthetics/new-relic-synthetics/pages/synthetics-resources-understand-load-times). These response codes are based on [Chromium's response codes](https://code.google.com/p/chromium/codesearch#chromium/src/net/base/net_error_list.h&sq=package:chromium) and give you more detail on the behavior of your monitors.
