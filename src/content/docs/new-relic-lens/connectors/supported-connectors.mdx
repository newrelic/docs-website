Lens supports connecting to various external data sources. This page describes the configuration requirements for each supported connector type.

<Callout variant="important">
Only Org managers or users with custom roles containing connector creation capabilities can create data connectors.
</Callout>

## Create a connector [#create-connector]

To set up a data connector:

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Administration > Connectors**.
2. Click **Create a connector**.
3. Select your connector type from the dropdown menu.
4. Enter the required connection information for your selected connector type.
5. Optionally, grant access to users or user groups in the access control section.
6. Click **Create**.

After creating a connector, you can view its schema or delete it from the <Icon name="fe-more-horizontal"/> menu.

## Snowflake [#snowflake]

Connect to Snowflake data warehouses for querying cloud data.

<table>
  <thead>
    <tr>
      <th style={{ width: "250px" }}>Field</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>**Name**</td>
      <td>A unique identifier for this connector.</td>
    </tr>
    <tr>
      <td>**Connection URL**</td>
      <td>JDBC connection string. For example, `jdbc:snowflake://ACCOUNT.snowflakecomputing.com/`</td>
    </tr>
    <tr>
      <td>**Username**</td>
      <td>Snowflake user account.</td>
    </tr>
    <tr>
      <td>**Password**</td>
      <td>Snowflake user password.</td>
    </tr>
    <tr>
      <td>**Account identifier**</td>
      <td>Your Snowflake account identifier.</td>
    </tr>
    <tr>
      <td>**Database name**</td>
      <td>Target database.</td>
    </tr>
    <tr>
      <td>**Role**</td>
      <td>Snowflake role with appropriate permissions.</td>
    </tr>
    <tr>
      <td>**Warehouse name**</td>
      <td>Compute warehouse to use.</td>
    </tr>
    <tr>
      <td>**Auth domain**</td>
      <td>Select the authentication domain containing the users or groups you want to grant access to.</td>
    </tr>
    <tr>
      <td>**User or group**</td>
      <td>Search and select specific users or user groups to grant connector access.</td>
    </tr>
    <tr>
      <td>**Role**</td>
      <td>Select the Lens role to assign. Currently, only **Viewer** role is available, which allows users to query the connector.</td>
    </tr>
  </tbody>
</table>

Ensure your Snowflake account identifier is correct and the user has appropriate role and warehouse access.
