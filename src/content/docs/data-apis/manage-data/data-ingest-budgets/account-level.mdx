For organizations with multiple New Relic accounts, each with distinct data ingestion requirements, data ingest budgets provide per-account monitoring, visibility, and alerts to ensure accounts stay within their defined usage limits.

## Budget types by customer structure

The type of budgets you can create depends on your customer structure:

### For single-tenant customers

You can create <DNT>**account level budgets**</DNT> for individual accounts or groups of up to 5 accounts within your organization.

### For multi-tenant customers

<DNT>**Managing organizations**</DNT> can create <DNT>**account level budgets**</DNT> for accounts within the managing organization (same functionality as single-tenant customers).

<DNT>**Managed organizations**</DNT> can't create budgets.

## Set up account-level budgets

<Steps>
  <Step>
    ### Open Data Budgets

    1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com/)**</DNT>.
    2. In the <DNT>[user menu](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)</DNT>, select <DNT>**Administration** > **Data Management** > **Data Budgets**</DNT>.
  </Step>

  <Step>
    ### Create a budget

    In the <DNT>**Functional budgets**</DNT> section, select <DNT>**+ Add functional budget**</DNT>.

    <Callout variant="important">
      <DNT>**For multi-tenant customers**</DNT>, this section displays:
      - <DNT>**Organization level budgets**</DNT> for managed organizations
      - <DNT>**Account level budgets**</DNT> for the managing organization

      <DNT>**Managed organizations**</DNT> can't create account level budgets.
    </Callout>
  </Step>

  <Step>
    ### Assign your budget

    1. Enter a name for your budget.
    2. From the dropdown list, select and assign up to 5 accounts to this budget.

    <Callout variant="tip">
      Accounts that already have budgets assigned will appear disabled in the selection list.
    </Callout>

    3. Select <DNT>**Next**</DNT>.
  </Step>

  <Step>
    ### Set monthly budget limits

    1. In the <DNT>**Set monthly budget limits**</DNT> section, you'll see how many GB are remaining in your organization budget.
    2. Enter the amount of GB you want to allocate to this account-level budget.

    <Callout variant="tip">
      You can allocate more than your organization budget. This allows flexible planning but may result in overages.
    </Callout>

    3. Select <DNT>**Next**</DNT>.
  </Step>

  <Step>
    ### Set up notifications

    1. To receive a warning when you've used a certain percentage of the GB assigned to this budget for a specific month, in the <DNT>**Thresholds**</DNT> section, select <DNT>**+ Add a threshold**</DNT> and enter the percentage at which you want to receive notifications (for example, 75%, 90%, or 95%).
    2. To ensure notifications are sent when the budget exceeds the threshold, in the <DNT>**Destinations**</DNT> section, add at least one destination to be notified. You can choose from the following options:
       - <DNT>**Email**</DNT>: Enter the email addresses of New Relic users to notify about the budget.
       - <DNT>**Webhook**</DNT>: Enter the webhook URL to send notifications to a specific endpoint.
    3. Select <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    ### Review budget details

    After you create the budget, the <DNT>**Data Budgets**</DNT> page displays the following information for each budget:

    <table>
      <thead>
        <tr>
          <th>
            Field name
          </th>
          <th>
            Description
          </th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>
            <DNT>**Budget name**</DNT>
          </td>
          <td>
            The name you assigned to the budget.
          </td>
        </tr>
        <tr>
          <td>
            <DNT>**Status**</DNT>
          </td>
          <td>
            Current status of the budget.
          </td>
        </tr>
        <tr>
          <td>
            <DNT>**GB allocation**</DNT>
          </td>
          <td>
            How much data (GB) has been allocated to this budget.
          </td>
        </tr>
        <tr>
          <td>
            <DNT>**GB utilized**</DNT>
          </td>
          <td>
            How much of the allocated data has been used.
          </td>
        </tr>
        <tr>
          <td>
            <DNT>**Utilization percentage**</DNT>
          </td>
          <td>
            Percentage of the budget that has been used.
          </td>
        </tr>
      </tbody>
    </table>
  </Step>
</Steps>

## Manage existing budgets

To manage an existing budget:

1. Locate the budget in the list and select the three-dot menu.
2. Choose from the following options:
   - <DNT>**View details**</DNT>: See detailed information about the budget, including utilization graphs, thresholds, notification settings, associated accounts, and data sources.
   - <DNT>**Edit**</DNT>: Modify the budget allocation, thresholds, or notification settings.
   - <DNT>**Delete**</DNT>: Remove the budget.

### View budget details

Select a budget name or choose <DNT>**View details**</DNT> from the three-dot menu to see:

- <DNT>**Total allocation**</DNT>: The total GB allocated to this budget
- <DNT>**Utilization**</DNT>: Visual graph showing usage over time with day-over-day breakdown
- <DNT>**Thresholds**</DNT>: The notification thresholds you've configured
- <DNT>**Notification destinations**</DNT>: Where alerts are sent when thresholds are exceeded
- <DNT>**Associated accounts**</DNT>: The accounts included in this budget
- <DNT>**Data sources**</DNT>: Information about data sources contributing to budget usage, categorized by source type
- <DNT>**Budget time range**</DNT>: The time period covered by this budget
