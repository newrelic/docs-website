[New Relic Insights](/docs/insights/new-relic-insights/understanding-insights/new-relic-insights) is a software analytics resource to gather and visualize data about your software and your business. The New Relic Insights mobile app provides easy access to your Insights data from iPhone and Android devices. New Relic also offers an [Insights app for Apple TV](/docs/mobile-monitoring/new-relic-mobile-apps/insights-app/insights-app-apple-tv).

## Compatibility and requirements [#requirements]

The New Relic Insights mobile app allows you to:

* View your Insights [dashboards](/docs/insights/viewing-dashboards) and [widgets](/docs/insights/using-widgets).
* Use Insights to view the [NRQL query](/docs/insights/using-nrql) and [notes](/docs/insights/using-widgets#notes) attached to each dashboard widget.
* Select favorite dashboards for easy access.

<CollapserGroup>
  <Collapser
    id="iphone-reqs"
    title="iPhone requirements"
  >
    * iOS 7 or higher
    * iPhone 4S or higher

    You can also use the Insights app on your iPad and iPod Touch.
  </Collapser>

  <Collapser
    id="android-reqs"
    title="Android requirements"
  >
    * 4.0.3 Ice Cream Sandwich or higher
    * All Android phones and tablets are supported
  </Collapser>
</CollapserGroup>

## Install the Insights app [#install]

To install the Insights app on your iPhone or Android device, follow these instructions and any additional requirements.

<CollapserGroup>
  <Collapser

  >
    You can install Insights for iPhone from the [App Store](https://itunes.apple.com/us/app/new-relic-insights/id891431652?ls=1&mt=8 "Link opens in a new window"). Follow standard procedures to install any iPhone app, then [sign in](/docs/insights-ios-app/insights-iphone-authentication).
  </Collapser>

  <Collapser
    id="android-install"

</CollapserGroup>

## App features [#features]

<CollapserGroup>
  <Collapser
    id="view-dashboards"
    title="View dashboards"
  >
    The Insights companion app for iOS and Android devices allows you to view dashboards and widgets created via the Insights website. Information updates in real time, so you always have access to the latest metrics.

    To view a dashboard, locate it from the list on the app's main menu. When you select a dashboard, you can access all the widgets created on the Insights website, and [view details for each widget](#viewing-widgets).

    Easily access your dashboards using the search icon on the main menu, or with favorites. To favorite a dashboard, select the star icon on each dashboard. Access your favorites from the **Favorite Dashboards** list in the Insights UI.
  </Collapser>

  <Collapser
    id="view-widget"
    title="View widget details"
  >
    You can select any [widget](/docs/insights/using-widgets) to view details. The details available vary by widget:

    * **[Billboard](/docs/insights/using-widgets#widget-billboard)**: Billboards that have exceeded their alert thresholds will be yellow (Warning) or red (Critical).
    * **[Table](/docs/insights/using-widgets#widget-table)**, **[Event](/docs/insights/using-widgets#widget-event)**, or **[Attribute sheet](/docs/insights/using-widgets#widget-attributes)**: Swipe to view all rows or columns.
    * **[Line Chart](/docs/insights/using-widgets#widget-attributes)**, **[Area Chart](/docs/insights/using-widgets#widget-areachart)**, or **[Histogram](/docs/insights/using-widgets#widget-histogram)**: Tap and hold to view exact values for each column or [timeslice](/docs/site/glossary#timeslice).
    * **[Pie Chart](/docs/insights/using-widgets#widget-piechart)**: Tap a segment or item in the list to view exact values.

    You can also view the underlying [NRQL](/docs/insights/using-nrql) for each widget on your dashboard, and access any [notes](/docs/insights/using-widgets#notes) attached to that widget:

    1. From the Insights UI, select a dashboard containing the widget you want to view.
    2. Select the info <Icon style={{color: '#FFC753'}} name="fe-info"/>
       icon, then select **NRQL** or **Notes**.
  </Collapser>

  <Collapser
    id="swicth-accounts"
    title="Switch accounts"
  >
    To switch to any account associated with your user name:

    1. From the app's main menu, select your user name.
    2. Select an account from the list.
  </Collapser>

  <Collapser
    id="switch-users"
    title="Switch users"
  >
    All Insights accounts and dashboards associated with your user name appear automatically.

    * To log out from the device: From the main menu, select your user name, then select **Logout**.
    * To switch users: Log out, then log in with a new account.
  </Collapser>
</CollapserGroup>

## Authentication with partners [#partner-auth]

To authenticate using an account provided through a New Relic partner:

1. From the New Relic Insights app, type your email address associated with the partner account.
2. Select **Next**, then select **No password**.
3. From your iPhone or Android device, retrieve the authentication email within 20 minutes.
4. Select the **Tap here to log in** link in the email to be redirected to the app and authenticated.

If you **miss** the 20-minute deadline, sign in to Insights app again, and select the link to resend the authentication email. If you cannot find the authentication message from New Relic in your inbox, check your **Spam** folder.

## Data privacy [#privacy]

To make troubleshooting easier, New Relic's mobile apps only record the following information about you:

* User's email address associated with your New Relic account, including first and last name (for authentication purposes only)
* IP address
* Device ID

For more information, see our [Mobile data privacy and security documentation](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps).
