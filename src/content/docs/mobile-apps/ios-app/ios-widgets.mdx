Here's an example of adding a widget to the home screen:

1. Touch and hold an empty area or widget until the icons jiggle.
2. Tap the **+** icon in the upper-left corner.

    <img
      style={{ maxWidth: '290px' }}
      title="Screenshot showing the Add (+) button"
      alt="Screenshot showing the Add (+) button"
      src={iosWidgets1}
    />

3. Select the New Relic widget.

      title="Screenshot showing how to select the New Relic widget"
      alt="Screenshot showing how to select the New Relic widget"
      src={iosWidgets2}
    />

4. Swipe left or right to select from three widget sizes, and then tap **Add Widget**.

## Add a chart to a widget [#add-chart]

If you haven't already selected the chart you want to show from the New Relic app, then follow these steps:

1. Navigate to whatever chart you would like to see in the widget.
2. Tap on the chart to show the detail view.
3. Tap the three dots in the top right corner to show more options.

    <img
      style={{ maxWidth: '290px' }}
      title="Screenshot showing the dropdown options in the three dots"
      alt="Screenshot showing the dropdown options in the three dots menu"
      src={iosWidgetsInApp1}
    />

4. Select **Add to widgets**.
5. You can also remove and rearrange widgets with **Manage widgets**.

    <Video
      type="wistia"
      id="7dd2al2g8w"
      vertical
      width='290px'
    />

## Designate charts for widgets [#designate-charts]

To select which chart is shown for each of the iOS widgets:

1. Press and hold the widget until it jiggles.
2. Tap **Select chart**, or the chart name.

    <img
      style={{ maxWidth: '290px' }}
      title="Screenshot showing the option to select a chart"
      alt="Screen showing the option to select a chart"
      src={iosWidgets5}
    />

3. If you prefer, you can hide the title and/or legend from this screen.
