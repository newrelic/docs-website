<Callout variant="important">
  Metrics in this document have been deprecated and are no longer supported by Microsoft Azure. Refer to the [Cosmos DB](/docs/integrations/microsoft-azure-integrations/azure-integrations-list/azure-cosmos-db-document-db-monitoring-integration) metrics currently supported by Microsoft. For current users, both the deprecated and currently supported events, metrics, and accompanying metadata are included.

  Use this page as a reference to migrate your alert conditions and custom dashboards from the deprecated metrics to their supported counterparts, as the deprecated metrics may stop working without notice should Microsoft stop publishing them.
</Callout>
