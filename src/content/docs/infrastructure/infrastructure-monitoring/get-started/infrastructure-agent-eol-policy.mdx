## Operating systems and runtime platforms [#infra-os-runtime]

The agent is supported in a wide variety of environments. See [Requirements for the infrastructure agent](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/) for an up-to-date list of supported operating systems, architectures, and runtime environments.

The infrastructure monitoring agent supports these operating systems up to their manufacturer's end-of-life. The agent might be compatible and packages might be available for EOL systems, but support is not guaranteed.
