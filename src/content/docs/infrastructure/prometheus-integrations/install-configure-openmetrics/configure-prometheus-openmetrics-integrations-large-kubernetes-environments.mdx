<Callout variant="important">
The Prometheus Open Metrics integration for Kubernetes has been discontinued and will only be supported until the end of June 2023.
</Callout>

See how to install the [Prometheus agent](/docs/infrastructure/prometheus-integrations/install-configure-prometheus-agent/install-prometheus-agent) to understand its benefits and get a full visiblity of your Prometheus workloads running in a Kubernetes cluster.

In case you need to migrate from the Prometheus Open Metrics integration to Open Metrics check the following [migration guide](/docs/infrastructure/prometheus-integrations/install-configure-prometheus-agent/migration-guide).
