You can use the Grafana to New Relic Dashboard Translation REST API to convert your existing Grafana dashboards into New Relic native dashboards. This API can aid in the migration from Grafana to New Relic for customers who are currently using the Grafana plugin to visualize their New Relic data. Whether you're looking to consolidate your observability tools, migrate your entire monitoring setup to New Relic, or simply test out New Relic dashboards alongside your existing Grafana setup, this API makes it easier to get started.

This API uses New Relic's PromQL translation logic to translate your queries. To see a list of supported PromQL features and functions, see [PromQL features support](/docs/infrastructure/prometheus-integrations/view-query-data/supported-promql-features).

## Prerequisites [#prerequisites]

Before using this API, you'll need:

  * A [New Relic user API key](/docs/apis/get-started/intro-apis/types-new-relic-api-keys#user-api-key)
  * Your New Relic account ID (if you want to override the default account)

---

## API endpoints [#api-endpoints]

**Translation endpoints** — send a `POST` request with the dashboard JSON in the request body:

* **US:** `https://prometheus-api.newrelic.com/api/v1/translate/dashboard/grafana`
* **EU:** `https://prometheus-api.eu.newrelic.com/api/v1/translate/dashboard/grafana`

**NerdGraph endpoints** — use these to upload the translated dashboard to your New Relic account (see upload example below):

* **US:** `https://api.newrelic.com/graphql`
* **EU:** `https://api.eu.newrelic.com/graphql`

## API reference [#api-reference]

### Authentication [#authentication]

All API calls require a [user API key](/docs/apis/get-started/intro-apis/types-new-relic-api-keys#user-api-key). Include it in the `x-api-key` header of your request.

### Request body [#request-body]

The request body must be a JSON object with the following format:

```json
{
  "accountId": int,
  "dashboard": object
}
```

Where `dashboard` is the Grafana dashboard JSON object you want to translate. The `accountId` field is optional — if omitted, the translated dashboard will query data from the account of the user making the request.

### Response format [#response-format]

The API returns a JSON object containing the translated New Relic dashboard:

```json
{
  "dashboard": {
    "name": "Translated Dashboard",
    "permissions": "PUBLIC_READ_WRITE",
    "pages": [...]
  }
}
```

### Examples [#api-examples]

<CollapserGroup>
  <Collapser
    id="dashboard-translation-basic"
    title="Example: Basic API call to translate a Grafana dashboard"
  >
    ```bash
    export NR_USER_KEY="your_user_key"
    export JSON_FILE="my_grafana_dashboard.json"
    export TRANSLATOR_ENDPOINT="<your-region-endpoint>" # (see API endpoints section above)

    curl -s ${TRANSLATOR_ENDPOINT} \
    -H "Content-Type: application/json" \
    -H "x-api-key:${NR_USER_KEY}" \
    -d "$(jq -c -n --slurpfile dashboard "${JSON_FILE}" '{dashboard: $dashboard[0]}')" \
    | jq '.dashboard'
    ```

    curl -s ${TRANSLATOR_ENDPOINT} \
    -H "Content-Type: application/json" \
    -H "x-api-key:${NR_USER_KEY}" \
    -d "$(jq -c -n --arg accountId "$ACCOUNT_ID" --slurpfile dashboard "$JSON_FILE" '{accountId: $accountId, dashboard: $dashboard[0]}')" \
    | jq '.dashboard'
    ```

    <Callout variant="tip">

    export NR_USER_KEY="your_user_key"
    export ACCOUNT_ID="your_staging_account_id"
    export JSON_FILE="my_grafana_dashboard.json"

    export NERDGRAPH_ENDPOINT="US/EU Regional Nerdgraph endpoint"

    NR_DASHBOARD=$(curl -s ${TRANSLATOR_ENDPOINT} \
    -H "Content-Type: application/json" \
    -H "x-api-key:${NR_USER_KEY}" \
    -d "$(jq -c -n --arg accountId "$ACCOUNT_ID" --slurpfile dashboard "$JSON_FILE" '{accountId: $accountId, dashboard: $dashboard[0]}')" \

    curl -X POST ${NERDGRAPH_ENDPOINT} \
    -H "Content-Type: application/json" \
    -H "API-Key: ${NR_USER_KEY}" \
    -d '{
    "query": "mutation($accountId: Int!, $dashboard: DashboardInput!) { dashboardCreate(accountId: $accountId, dashboard: $dashboard) { entityResult { guid name } errors { description } } }",
    "variables": {
    "accountId": '${ACCOUNT_ID}',
    "dashboard": '${NR_DASHBOARD}'
    }
    }'
    ```

## Scripts [#scripts]

Both scripts **upload translated dashboards to New Relic automatically by default** via the NerdGraph API, and also save the translated JSON to a local file. Set `UPLOAD=false` to skip the upload and only save the translated JSON locally without creating it in your New Relic account.

The scripts default to the US platform endpoints. If you're on the EU platform, replace the endpoint values in the script with the EU URLs listed in the [API endpoints](#api-endpoints) section.

<CollapserGroup>
  <Collapser
    id="script-single"
    title="Script 1: Translate a single dashboard"
  >
    Translates one Grafana dashboard JSON file and uploads it to New Relic by default.

    **Required:** `NR_USER_KEY` always. `ACCOUNT_ID` is required when `UPLOAD=true` (the default).

    ```bash
    # Translate and upload to New Relic (default)
    NR_USER_KEY=your_key ACCOUNT_ID=your_account_id ./translate_dashboard.sh path/to/dashboard.json
    ```

    **Script (`translate_dashboard.sh`):**

    ```bash
    #!/usr/bin/env bash
    # Required env vars:
    #   NR_USER_KEY  — Your New Relic user API key
    #   ACCOUNT_ID   — Required when UPLOAD=true (the default)
    # Optional:
    #   UPLOAD       — Set to "false" to save locally only

    set -euo pipefail

    NR_USER_KEY="${NR_USER_KEY:?Please set NR_USER_KEY}"
    JSON_FILE="${1:?Usage: $0 <path/to/dashboard.json>}"
    ACCOUNT_ID="${ACCOUNT_ID:-}"
    UPLOAD="${UPLOAD:-true}"
    TRANSLATOR_ENDPOINT="https://prometheus-api.newrelic.com/api/v1/translate/dashboard/grafana"
    NERDGRAPH_ENDPOINT="https://api.newrelic.com/graphql"

    if [[ -n "${ACCOUNT_ID}" ]]; then
      PAYLOAD=$(jq -cn --arg a "${ACCOUNT_ID}" --slurpfile d "${JSON_FILE}" \
        '{accountId: ($a|tonumber), dashboard: $d[0]}')
    else
      PAYLOAD=$(jq -cn --slurpfile d "${JSON_FILE}" '{dashboard: $d[0]}')
    fi

    echo "Translating: ${JSON_FILE} ..."
    NR_DASHBOARD=$(curl -sf "${TRANSLATOR_ENDPOINT}" \
      -H "Content-Type: application/json" \
      -H "x-api-key: ${NR_USER_KEY}" \
      -d "${PAYLOAD}" | jq '.dashboard')

    OUTPUT="${JSON_FILE%.json}_translated.json"
    echo "${NR_DASHBOARD}" > "${OUTPUT}"
    echo "Saved: ${OUTPUT}"

    if [[ "${UPLOAD}" == "true" ]]; then
      if [[ -z "${ACCOUNT_ID}" ]]; then
        echo "WARNING: ACCOUNT_ID not set — skipping upload."
      else
        echo "Uploading to New Relic (account: ${ACCOUNT_ID}) ..."
        curl -sf -X POST "${NERDGRAPH_ENDPOINT}" \
          -H "Content-Type: application/json" \
          -H "API-Key: ${NR_USER_KEY}" \
          -d "$(jq -cn \
            --arg q 'mutation($accountId: Int!, $dashboard: DashboardInput!) { dashboardCreate(accountId: $accountId, dashboard: $dashboard) { entityResult { guid name } errors { description } } }' \
            --argjson v "{\"accountId\": ${ACCOUNT_ID}, \"dashboard\": ${NR_DASHBOARD}}" \
            '{query: $q, variables: $v}')" | jq '.data.dashboardCreate'
      fi
    else
      echo "UPLOAD=false — saved locally only."
    fi
    ```
  </Collapser>

  <Collapser
    id="script-batch"
    title="Script 2: Translate a set of dashboards"
  >
    Translates all Grafana dashboard JSON files in a directory and uploads each to New Relic by default.

    **Required:** `NR_USER_KEY` always. `ACCOUNT_ID` is required when `UPLOAD=true` (the default).

    ```bash
    # Translate and upload all dashboards to New Relic (default)
    NR_USER_KEY=your_key ACCOUNT_ID=your_account_id ./translate_dashboards.sh path/to/dashboards/
    ```

    **Script (`translate_dashboards.sh`):**

    ```bash
    #!/usr/bin/env bash
    # Required env vars:
    #   NR_USER_KEY  — Your New Relic user API key
    #   ACCOUNT_ID   — Required when UPLOAD=true (the default)
    # Optional:
    #   OUTPUT_DIR   — Directory to save translated files (default: <input_dir>/translated)
    #   UPLOAD       — Set to "false" to save locally only

    set -euo pipefail

    NR_USER_KEY="${NR_USER_KEY:?Please set NR_USER_KEY}"
    INPUT_DIR="${1:?Usage: $0 <path/to/dashboards/>}"
    ACCOUNT_ID="${ACCOUNT_ID:-}"
    UPLOAD="${UPLOAD:-true}"
    OUTPUT_DIR="${OUTPUT_DIR:-${INPUT_DIR}/translated}"
    TRANSLATOR_ENDPOINT="https://prometheus-api.newrelic.com/api/v1/translate/dashboard/grafana"
    NERDGRAPH_ENDPOINT="https://api.newrelic.com/graphql"

    mkdir -p "${OUTPUT_DIR}"

    if [[ "${UPLOAD}" == "true" && -z "${ACCOUNT_ID}" ]]; then
      echo "WARNING: ACCOUNT_ID not set — upload disabled."
      UPLOAD="false"
    fi

    shopt -s nullglob
    JSON_FILES=("${INPUT_DIR}"/*.json)
    shopt -u nullglob

    [[ ${#JSON_FILES[@]} -eq 0 ]] && { echo "No JSON files found in: ${INPUT_DIR}"; exit 1; }

    echo "Found ${#JSON_FILES[@]} dashboard(s). Output: ${OUTPUT_DIR}"
    SUCCESS=0; FAIL=0

    for JSON_FILE in "${JSON_FILES[@]}"; do
      BASENAME=$(basename "${JSON_FILE}" .json)
      OUTPUT="${OUTPUT_DIR}/${BASENAME}_translated.json"

      echo "Translating: ${BASENAME}.json ..."

      if [[ -n "${ACCOUNT_ID}" ]]; then
        PAYLOAD=$(jq -cn --arg a "${ACCOUNT_ID}" --slurpfile d "${JSON_FILE}" \
          '{accountId: ($a|tonumber), dashboard: $d[0]}')
      else
        PAYLOAD=$(jq -cn --slurpfile d "${JSON_FILE}" '{dashboard: $d[0]}')
      fi

      if NR_DASHBOARD=$(curl -sf "${TRANSLATOR_ENDPOINT}" \
        -H "Content-Type: application/json" \
        -H "x-api-key: ${NR_USER_KEY}" \
        -d "${PAYLOAD}" | jq '.dashboard'); then

        echo "${NR_DASHBOARD}" > "${OUTPUT}"
        echo "  Saved → ${OUTPUT}"

        if [[ "${UPLOAD}" == "true" ]]; then
          RESULT=$(curl -sf -X POST "${NERDGRAPH_ENDPOINT}" \
            -H "Content-Type: application/json" \
            -H "API-Key: ${NR_USER_KEY}" \
            -d "$(jq -cn \
              --arg q 'mutation($accountId: Int!, $dashboard: DashboardInput!) { dashboardCreate(accountId: $accountId, dashboard: $dashboard) { entityResult { guid name } errors { description } } }' \
              --argjson v "{\"accountId\": ${ACCOUNT_ID}, \"dashboard\": ${NR_DASHBOARD}}" \
              '{query: $q, variables: $v}')" \
            | jq -r '.data.dashboardCreate.entityResult.name // "upload failed"')
          echo "  Uploaded → ${RESULT}"
        fi
        ((SUCCESS++))
      else
        echo "  ERROR: Translation failed for ${BASENAME}.json"
        ((FAIL++))
      fi
    done

    echo ""
    echo "Done. Translated: ${SUCCESS} | Failed: ${FAIL}"
    ```
  </Collapser>
</CollapserGroup>
