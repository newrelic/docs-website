This document explains the files used for our [infrastructure agent on-host integrations](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations).

## Integration files [#overview]

An infrastructure agent on-host integration contains at least these files:

* An [executable file](/docs/infrastructure/infrastructure-integrations/custom-integrations/create-integration-executable-file), written in any language, that exports JSON data in a format expected by the infrastructure agent
* A [configuration file](/docs/create-integrations/infrastructure-integrations-sdk/specifications/host-integrations-standard-configuration-format)
