Our Port monitoring integration monitors status for networking ports, such as TCP, UDP, etc., and reports data to New Relic.

To get started, see the [Port monitoring on-host integration README](https://github.com/newrelic/nri-port-monitor) on GitHub.

<Callout variant="tip">
  This integration is released as [open source on Github](https://github.com/newrelic/nri-port-monitor). A change log is also available there for the latest updates. Support for this integration is available with an [Expert Services subscription](https://www.newrelic.com/expertservices).
</Callout>
