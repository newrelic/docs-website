The Collectd [integration](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) monitors and reports metrics from [Collectd](https://collectd.org/wiki/index.php/Main_Page)-configured devices to New Relic.

To get started, see the [Collectd On-Host Integration README](https://github.com/newrelic/nri-collectd) on GitHub.

<Callout variant="tip">
  This integration is released as [open source on Github](https://github.com/newrelic/nri-collectd). A change log is also available there for the latest updates. Support for this integration is available with an [Expert Services subscription](https://www.newrelic.com/expertservices).
</Callout>
