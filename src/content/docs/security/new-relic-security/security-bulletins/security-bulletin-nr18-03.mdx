## Vulnerability information [#vuln-info]

New Relic Synthetics Private Location images have been updated with patches to protect against the Meltdown CPU vulnerability. Patches to address Spectre have yet to be released by Ubuntu.

Additionally, customers can run in-place upgrades by accessing their private location and running:

```
sudo apt-get update
sudo apt-get dist-upgrade
```

and subsequently rebooting their machine.

Additional Information: [Ubuntu Advisory](https://wiki.ubuntu.com/SecurityTeam/KnowledgeBase/SpectreAndMeltdown)
