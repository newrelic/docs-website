When deploying the Kubernetes integration for the first time, we deploy a default set of [alert conditions](https://docs.newrelic.com/docs/alerts/new-relic-alerts/defining-conditions/define-alert-conditions) to your account, the [predefined alert policy](/docs/integrations/kubernetes-integration/kubernetes-events/kubernetes-integration-predefined-alert-policy), a basis for alert conditions on your Kubernetes cluster.

While we try to tackle the most common use cases across all the environments, there's a number of additional alerts you can set up to extend the default policy. What follow are some recommendations.
