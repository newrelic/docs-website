New Relic's Japan data center provides comprehensive observability capabilities for organizations with data residency requirements in Japan. However, some features and legacy APIs are not available in this region.

<Callout variant="important">
  If you require any of these capabilities, consider using the alternative New Relic features listed below or contact your account representative to discuss options.
</Callout>
