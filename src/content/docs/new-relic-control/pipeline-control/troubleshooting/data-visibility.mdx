## Problem: Account dropdown issues
You don't have **All accounts** or the account associated with the gateway license key selected in the Pipeline Control UI.
## Solution
- Check the account dropdown in the Pipeline Control UI to ensure the correct account is selected.
- Ensure you're viewing **All accounts** or the specific account associated with the gateway license key to access the relevant MELT data.
