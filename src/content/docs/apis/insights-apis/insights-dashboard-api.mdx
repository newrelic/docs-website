Do not use the Insights Dashboards API. Instead, use the [New Relic One Dashboards API](/docs/query-your-data/explore-query-data/dashboards/dashboards-api/) with NerdGraph, our GraphQL API. To learn how to easily make this transition, watch this short YouTube video (approx. 6:30 minutes).

<Video id="oONRQSZpMgs" type="youtube" />

## End of life notice [#eol]

The Insights Dashboard API reached end of life in 2021.

* As of July 28, 2021, the `CREATE` and `UPDATE` endpoints are not available.
* As of August 30, 2021, the `GET` and `DELETE` endpoints are not available.

To make the transition from the Insights Dashboard API to the New Relic One Dashboards API, see our [migration guide](/docs/new-relic-one/use-new-relic-one/core-concepts/dashboards-api-migration-insights-api-nerdgraph/). For more information, see the [NerdGraph dashboards tutorial](/docs/apis/nerdgraph/examples/create-widgets-dashboards-api/) and [Explorers Hub post](https://discuss.newrelic.com/t/important-insights-dashboard-api-end-of-life/149357).
