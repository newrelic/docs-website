## Syntax

```
add_user_attribute(key, value)
```

(Deprecated.) Adds a custom attribute to a transaction.

## Description

<Callout variant="important">
  This call is deprecated, and aliases to `add_custom_parameter()`. Use [`add_custom_parameter()`](/docs/agents/python-agent/python-agent-api/add_custom_parameter) instead.
</Callout>

This call records an [attribute](https://docs.newrelic.com/docs/accounts-partnerships/education/getting-started-new-relic/glossary#attribute) (a key/value pair attached to a [transaction](https://docs.newrelic.com/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction)). The key must be a string. The value must be either a string, integer, float, or boolean.
