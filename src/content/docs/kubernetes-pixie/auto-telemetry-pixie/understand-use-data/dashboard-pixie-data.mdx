New Relic persists select Pixie telemetry data for long-term storage. Find out more about the hybrid storage model for the New Relic Pixie integration [here](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/overview). After install, you can view your Pixie data with our <InlinePopover type="dashboards"/> feature.

## Pre-built Pixie dashboards [#pixie-dashboards]

To install a pre-built dashboard for Pixie data:

1. Navigate to the New Relic [Dashboards](https://one.newrelic.com/dashboards) page.
2. Select the <DNT>**Browse pre-built dashboards**</DNT> button.
3. Search for “Pixie” and pick one of the available options.
4. Select the <DNT>**Install this quickstart**</DNT> button.

## Custom Pixie dashboards [#custom-dashboards]

The visualization widgets in the pre-built Pixie dashboards are populated with data from NRQL queries. Select any dashboard widget’s three-dot menu to view the query providing the data. To learn how to write your own queries for persisted Pixie data, see the [Querying Pixie data](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/query-pixie-data) page.

Find out more information about how to customize a pre-built dashboard or build your own custom dashboard [here](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).
