The Browser and Single Page Application (SPA) APIs allow you to customize and extend your browser monitoring. To begin using the APIs, ensure the [Browser JavaScript snippet is present on every page you want to monitor](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). This document presents use cases for each Browser and SPA API method.

For an overview of the Browser agent and SPA APIs and their methods, see [Browser Agent and SPA API](/docs/browser/new-relic-browser/browser-agent-spa-api).

## Traces and events [#api-traces]

To monitor traces and events, use these methods:

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        If you want to...
      </th>

      <th>
        Use this
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Track an action that New Relic doesn't automatically track
      </td>

      <td>
        Use `newrelic.addPageAction()`
      </td>
    </tr>

    <tr>
      <td>
        Add an object to an in-progress [session trace](/docs/browser/new-relic-browser/browser-pro-features/session-traces-exploring-webpages-life-cycle)
      </td>

      <td>
        Use `newrelic.addToTrace()`
      </td>
    </tr>

    <tr>
      <td>
        Record a time point when your page is finished according to custom criteria
      </td>

      <td>
        Use `newrelic.finished()`
      </td>
    </tr>

    <tr>
      <td>
        Add an attribute name and value to subsequent events on the page
      </td>

      <td>
        Use `newrelic.setCustomAttribute()`
      </td>
    </tr>

    <tr>
      <td>
        Group [page views](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8302) by setting a page name
      </td>

      <td>
        Use `newrelic.setPageViewName()`
      </td>
    </tr>
  </tbody>
</table>

## Track single page apps (SPA API) [#spa-api]

To track single page applications, use these SPA API methods:

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        If you want to...
      </th>

      <th>
        Do this
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Set the [`actionText`](/attribute-dictionary/spa-browserinteraction/actiontext), or text of the HTML element clicked, of a SPA [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312)
      </td>

      <td>
        Use `newrelic.interaction().actionText()`
      </td>
    </tr>

    <tr>
      <td>
        Trace sub-components of a SPA [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312) separately
      </td>

      <td>
        Use `newrelic.interaction().createTracer()`
      </td>
    </tr>

    <tr>
      <td>
        End a SPA [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312)
      </td>

      <td>
        Use `newrelic.interaction().end()`
      </td>
    </tr>

    <tr>
      <td>
        Aggregate data associated with the current [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312)
      </td>

      <td>
        Use `newrelic.interaction().getContext()`
      </td>
    </tr>

    <tr>
      <td>
        Ignore an [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312)
      </td>

      <td>
        Use `newrelic.interaction().ignore()`
      </td>
    </tr>

    <tr>
      <td>
        Return a new API object bound to the current [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312)
      </td>

      <td>
        Use `newrelic.interaction()`
      </td>
    </tr>

    <tr>
      <td>
        Adjust attributes, such as name and ignored attributes, before the [interaction](//attribute-dictionary?attribute_name=&events_tids%5B%5D=8312) is recorded
      </td>

      <td>
        Use `newrelic.interaction().onEnd()`
      </td>
    </tr>

    <tr>
      <td>
        Ensure a browser [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312) will be saved when it ends
      </td>

      <td>
        Use `newrelic.interaction().save()`
      </td>
    </tr>

    <tr>
      <td>
        Add a custom attribute to the current [interaction](/attribute-dictionary?attribute_name=&events_tids%5B%5D=8312)
      </td>

      <td>
        Use `newrelic.interaction().setAttribute()`
      </td>
    </tr>

    <tr>
      <td>
        Give SPA routes more accurate names than default names
      </td>

      <td>
        Use `newrelic.interaction().setCurrentRouteName`
      </td>
    </tr>
  </tbody>
</table>
