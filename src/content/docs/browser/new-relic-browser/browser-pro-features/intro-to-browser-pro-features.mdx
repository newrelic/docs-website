There are three [types of New Relic browser monitoring agents](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#agent-types): Lite, Pro, and Pro+SPA. When browser monitoring is enabled, the Pro+SPA agent is installed by default. 

These are the available [Browser Pro feature docs](/docs/browser/new-relic-browser/browser-pro-features), which explain the features available for the Pro and Pro+SPA agents:

- [Single Page App (SPA) monitoring](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring/)
- [AJAX page](/docs/browser/browser-monitoring/browser-pro-features/ajax-page-identify-time-consuming-calls/)
- [JavaScript errors page](/docs/browser/new-relic-browser/browser-pro-features/javascript-errors-page-detect-analyze-errors)
- [Errors page](/docs/browser/new-relic-browser/browser-pro-features/browser-error-profiles-find-error-causes)
- [Session traces page](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle)
- [Filterable geography page](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)
- [Browser data in distributed tracing](/docs/browser/new-relic-browser/browser-pro-features/browser-data-distributed-tracing)
- [Upload source maps to un-minify JS errors](/docs/browser/browser-monitoring/browser-pro-features/upload-source-maps-un-minify-js-errors)
- [Upload source maps via API](/docs/browser/new-relic-browser/browser-pro-features/upload-source-maps-api)
