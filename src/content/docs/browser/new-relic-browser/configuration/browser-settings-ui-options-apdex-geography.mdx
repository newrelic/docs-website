As part of the browser [setup requirements](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements), you must [install the browser agent](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). Then you can use the your app's **Applications settings** page to update browser monitoring options for:

* Browser subscription levels
* Browser [Apdex T values](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)
* Cities, regions, or countries to be highlighted on the [**Filterable geography** page](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)

## Select Apdex and geography settings [#apdex-settings]

For apps that are also monitored by APM, you can set separate [Apdex T values](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) for browser (end users) and for [APM](/docs/apm/new-relic-apm/apdex/changing-your-apdex-settings). You can also select cities, regions, or countries to be highlighted on the **Filterable geography** page.

1. Go to **[one.newrelic.com](http://one.newrelic.com) > Browser > (select an app) > Settings > Application settings**.
2. From **Application settings**, select an Apdex T value specifically for browser monitoring, or leave the default Apdex value.
3. Select up to five countries or regions for browser monitoring to highlight on the **Filterable geography** page. If you are reporting data to [multiple app names](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), select the same countries in both the primary and secondary apps.
4. Select **Save application settings**.

Whenever you change Apdex settings, a black vertical bar appears on the **Page load timing** chart on your browser app's **[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)** page.

## View end-user Apdex [#view-browser-apdexs]

To view your Apdex score:

* Go to **[one.newrelic.com](http://one.newrelic.com) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**.

  OR
* Go to **[one.newrelic.com](http://one.newrelic.com) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**.
