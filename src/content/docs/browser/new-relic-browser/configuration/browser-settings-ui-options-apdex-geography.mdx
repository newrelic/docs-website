As part of the <InlinePopover type="browser" /> [setup requirements](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements), you must [install the browser agent](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). Then you can use the your app's <DoNotTranslate>**Applications settings**</DoNotTranslate> page to update browser monitoring options for:

* Browser subscription levels
* Browser [Apdex T values](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)
* Cities, regions, or countries to be highlighted on the [<DoNotTranslate>**Geography**</DoNotTranslate> page](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)

## Select Apdex and geography settings [#apdex-settings]

For apps that are also monitored by APM, you can set separate [Apdex T values](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) for browser (end users) and for [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex). You can also select cities, regions, or countries to be highlighted on the <DoNotTranslate>**Geography**</DoNotTranslate> page.

1. Go to <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >**</DoNotTranslate> (select an app) <DoNotTranslate>**> Settings > Application settings**</DoNotTranslate>.
2. From <DoNotTranslate>**Application settings**</DoNotTranslate>, select an Apdex T value specifically for browser monitoring, or leave the default Apdex value.
3. Select up to five countries or regions for browser monitoring to highlight on the <DoNotTranslate>**Geography**</DoNotTranslate> page. If you are reporting data to [multiple app names](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), select the same countries in both the primary and secondary apps.
4. Select <DoNotTranslate>**Save application settings**</DoNotTranslate>.

Whenever you change Apdex settings, a black vertical bar appears on the <DoNotTranslate>**Page load timing**</DoNotTranslate> chart on your browser app's <DoNotTranslate>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DoNotTranslate> page.

## View end-user Apdex [#view-browser-apdexs]

To view your Apdex score:

* Go to <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DoNotTranslate>.

  OR
* Go to <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DoNotTranslate>.
