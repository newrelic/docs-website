<DNT>Cloud Cost Intelligence</DNT> is an Advanced Compute product. Pricing is based on the volume of billing export data that your cloud provider publishes and CCI processes, referred to as billing file ingest.

<Callout variant="tip">
  Contact your New Relic account team for the latest pricing details or to discuss volume-based options.
</Callout>

## Billing file ingest rate

The list price for <DNT>Cloud Cost Intelligence</DNT> is **23 aCCUs per GB** of billing file ingest.

## What is billing file ingest?

Billing file ingest refers to the size of the billing export files that your cloud provider (AWS, Azure, or GCP) publishes to your designated storage container. <DNT>Cloud Cost Intelligence</DNT> imports these files to build cost analysis and insights.

<Callout variant="important">
  Billing file ingest is different from New Relic's standard [data ingest](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/data-ingest-billing/), which applies to streaming telemetry data such as metrics, logs, and traces. The two are metered and billed separately.
</Callout>

For an overview of how New Relic pricing works more broadly, see [New Relic pricing](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/).
