A budget establishes a spending limit for an account over a flexible timeframe ranging from one to twelve months. It helps you control cloud expenditures by monitoring them against this predefined limit.

## Set up Budgets

<Steps>

  <Step>
    ## Create budgets

      1. On the **Budget Summary** page, click **Create a budget**.
      2. In the **Create a budget** pop-up dialog box, enter the following information:
        - **Budget name:** A descriptive name for your budget.
        - Select the required setup type:
