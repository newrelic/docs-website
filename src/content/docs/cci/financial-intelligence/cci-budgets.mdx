A budget establishes a spending limit for an account over a flexible timeframe ranging from one to twelve months. It helps you control cloud expenditures by monitoring them against this predefined limit.

<img title="Cost types highlighted on the Cloud Cost Intelligence budgets page" alt="Cost types highlighted on the Cloud Cost Intelligence budgets page" src="/images/cci_budgets.webp"/>

## Set up Budgets

<Steps>

  <Step>
    ## Navigate to Cloud Cost Intelligence
      Navigate to **All Capabilities** > **Cloud Cost Intelligence**.
  </Step>

  <Step>
    ## Access Budgets
      From the left-hand navigation menu, select the **Budgets** tab.
  </Step>

  <Step>
    ## Create budgets

      1. On the **Budgets** page, click **Create a budget**.
      2. In the **Create a budget** pop-up dialog box, enter the following information:
        - **Budget name:** A descriptive name for your budget.
        - Select the required setup type:
