View and manage your cloud provider accounts. Pause or delete integrations for specific cloud provider accounts.

1. Go to <DNT>[**one.newrelic.com**](https://one.newrelic.com) > **All Capabilities** > **Cloud Cost Intelligence**</DNT>.
2. Select **Integrations** tab to view all cloud cost integrations.
3. Click a cloud provider account under **Name** column for details.

  <Callout variant="important">
  Currently, only AWS cloud costs are supported.
  </Callout>

4. Click the ellipsis icon next to an account and choose:
    - **Pause** - Temporarily stop data collection
    - **Delete** - Permanently remove integration
