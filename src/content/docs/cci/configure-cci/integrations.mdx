<Steps>

  <Step>
    ## Navigate to Integrations
      1. Navigate to <DNT>[**one.newrelic.com**](https://one.newrelic.com) > **All Capabilities** > **Cloud Cost Intelligence**</DNT>.
      2. Select the **Integrations** tab from the left-hand navigation menu.
  </Step>

  <Step>
    ## View and manage accounts

    - In the connections list, you can view:

      <table>
        <tr>
          <th>Field name</th>
          <th>Description</th>
        </tr>
        <tr>
          <td>Name</td>
          <td>Displays the name of the integration</td>
        </tr>
        <tr>
          <td>Status</td>
          <td>Indicates whether the integration is **Running** or **Paused**.</td>
        </tr>
        <tr>
          <td>Cloud Provider</td>
          <td>
          Displays the name of the cloud provider.

          <Callout title="important">
            Currently, this functionality supports AWS accounts.
          </Callout>

          </td>
        </tr>
        <tr>
          <td>Created on</td>
          <td>Displays the date when the integration was created.</td>
        </tr>
      </table>

    - Click the **ellipsis** icon next to the integration name to view the following options:
      - **Pause**: Temporarily stop the integration from collecting data.
      - **Delete**: Permanently remove the integration from your account.

      <img
        title="Pause and delete integration"
        alt="A screenshot displaying the pause and delete integration options"
        src="/images/pause-delete-cci.webp"
      />
  </Step>

</Steps>
