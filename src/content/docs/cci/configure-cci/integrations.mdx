The Cloud Cost Intelligence Integrations feature allows you to view and manage your cloud provider accounts. Use this feature to pause or delete the integration for a specific cloud provider account.

1. Navigate to <DNT>[**one.newrelic.com**](https://one.newrelic.com) > **All Capabilities** > **Cloud Cost Intelligence**</DNT>.
2. From the left-hand navigation menu, select the **Integrations** tab to display a list of all cloud cost integrations.
3. To view additional details, under the **Name** column, click the required cloud provider account.

  <Callout title="important">
  Currently, only cloud costs from AWS are supported.
  </Callout>

4. Next to the required account, click the ellipsis icon, and choose one of the following:
    - **Pause**: Temporarily stop the integration from collecting data.
    - **Delete**: Permanently remove the integration from your account.
