View and manage your cloud provider accounts, and pause or delete integrations for specific accounts.

<Steps>
  <Step>
    ### Access Cloud Cost Intelligence

    Go to <DNT>[**one.newrelic.com**](https://one.newrelic.com) > **All Capabilities** > **Cloud Cost Intelligence**</DNT>.
  </Step>

  <Step>
    ### View integrations

    Select **Integrations** tab to view all cloud cost integrations.
  </Step>

  <Step>
    ### Select an account

    Click a cloud provider account under **Name** column for details.

    <Callout variant="important">
    Currently, only AWS cloud costs are supported.
    </Callout>
  </Step>

  <Step>
    ### Manage the integration

    Click the ellipsis icon next to an account and choose:
    - **Pause** - Temporarily stop data collection
    - **Delete** - Permanently remove integration

    <img
      title="Pause and delete integration"
      alt="A screenshot displaying the pause and delete integration options"
      src="/images/pause-delete-cci.webp"
    />
  </Step>
</Steps>
