## New Relic integrations for third-party telemetry tools [#third-party-integrations]

If you are collecting data with these telemetry tools, you can send your data to New Relic:

* [OpenTelemetry](/docs/integrations/open-source-telemetry-integrations/opentelemetry/opentelemetry-quick-start/)
* [Istio](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/new-relics-istio-integration)
* [Kamon](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/kamon-reporter)
* [OpenCensus](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/new-relics-opencensus-integration)
* [AWS X-Ray](/docs/integrations/amazon-integrations/aws-integrations-list/aws-x-ray-monitoring-integration)
