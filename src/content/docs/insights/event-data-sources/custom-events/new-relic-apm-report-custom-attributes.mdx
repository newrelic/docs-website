Using New Relic APM, you can add your own custom [attributes](/docs/using-new-relic/welcome-new-relic/get-started/glossary#attribute), which you can then use for [querying and charting](https://docs.newrelic.com/docs/using-new-relic/data/understand-data/query-new-relic-data). This is one of several ways to [report custom data](/docs/insights/insights-data-sources/custom-data/send-custom-data-insights).

## Add custom attributes to APM events

To add custom [attributes](/docs/using-new-relic/welcome-new-relic/get-started/glossary#attribute) to APM events, you must first enable them for your New Relic APM agent, and then make an API call to record the attribute. Follow the agent-specific custom attribute procedures:

* [C SDK](/docs/agents/manage-apm-agents/agent-data/collect-custom-attributes#c-sdk-att)
* [Go](/docs/agents/manage-apm-agents/agent-data/collect-custom-attributes#go-att)
* [Java](/docs/apm/other-features/attributes/collecting-custom-attributes#java-att)
* [.NET](/docs/agents/manage-apm-agents/agent-data/collect-custom-attributes#net-att)
* [Node.js](/docs/apm/other-features/attributes/collecting-custom-attributes#nodejs-att)
* [PHP](/docs/apm/other-features/attributes/collecting-custom-attributes#php-att)
* [Python](/docs/apm/other-features/attributes/collecting-custom-attributes#python-att)
* [Ruby](/docs/apm/other-features/attributes/collecting-custom-attributes#ruby-att)

## Requirements for custom attributes

When creating your own custom attributes, follow data requirements for:

* [Size limits](/docs/insights/explore-data/custom-attributes/requirements-custom-attributes)
* [Attribute types](/docs/insights/explore-data/custom-attributes/requirements-custom-attributes)
* [Reserved words](/docs/insights/nrql-new-relic-query-language/nrql-resources/nrql-reserved-words)
