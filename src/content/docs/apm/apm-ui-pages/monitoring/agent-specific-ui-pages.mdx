[APM](/docs/apm/new-relic-apm/getting-started/welcome-new-relic-apm) has language-specific agents. All the APM agents have the same general functionality, but there are some agent-specific differences in UI pages and features.

## Custom UI pages for APM agents [#agents]

Some agents also have custom UI pages for certain features:

* C SDK: no C-specific UI pages
* Go: [**Go runtime**](/docs/agents/go-agent/features/go-runtime-ui-page) page
* Java: [**JVMs**](/docs/agents/java-agent/features/jvm-metrics-page) page and [**Instrumentation editor**](/docs/agents/java-agent/custom-instrumentation/custom-instrumentation-editor-quickly-customize-your-java-instrumentation) feature
* .NET: no .NET-specific UI pages
* Node.js: [**Node VMs**](/docs/agents/nodejs-agent/supported-features/node-vms-statistics-page) page
* PHP: [**WordPress**](/docs/agents/php-agent/frameworks-libraries/wordpress-specific-functionality) page and [**Drupal**](/docs/agents/php-agent/frameworks-libraries/drupal-specific-functionality) page
* Python: no Python-specific UI pages
* Ruby: [**Ruby VMs**](/docs/agents/ruby-agent/features/ruby-vm-measurements) page
