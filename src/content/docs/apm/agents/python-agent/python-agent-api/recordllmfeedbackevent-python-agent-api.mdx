## Syntax

```py
newrelic.agent.record_llm_feedback_event(trace_id, rating, category=None, message=None, metadata=None)
```

Records custom feedback events for AI Large Language Model applications.

## Requirements

Python agent version 9.8.0 or higher.

## Description

This API records a feedback event of type `LlmFeedbackMessage` that can be viewed and queried in the New Relic UI. Feedback events correlate trace IDs between an AI-generated message and the feedback an end user submitted on it.
To correlate messages with feedback, you can obtain the trace ID of the active transaction via a call to [`current_trace_id`](/docs/apm/agents/python-agent/python-agent-api/currenttraceid-python-agent-api/) right after the call that generates the AI message and pass it to the feedback call later on when a user provides feedback.

In many cases, the endpoint in which the AI message is generated is not the endpoint in which the feedback is recorded. These may happen during different transactions. For this reason, it's important to make sure that the trace id is captured inside of the endpoint that generates the AI message and that feedback is recorded and passed that trace id inside the endpoint that records the feedback.

## Return values

None.

## Examples

### Obtain trace ID and record feedback

Example of recording a feedback event:

```py
import newrelic.agent

def get_message(request):
    trace_id = newrelic.agent.current_trace_id()

def post_feedback(request):
    newrelic.agent.record_llm_feedback_event(trace_id=request.trace_id, rating=request.rating, metadata= {"my_key": "my_val"})
```
