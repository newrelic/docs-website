Read about the requirements and tips for integrating our Python agent with an app that uses the [FastAPI](https://fastapi.tiangolo.com/) web framework. To return to the general install instructions, go to [Install the Python agent](/docs/agents/python-agent/installation-configuration/python-agent-installation).

## Automatic initialization with admin script [#wrapper-script]

If you start your app with `python app.py` and use our Python agent **version 5.22.0.151** or higher, you can use the recommended [admin script integration method](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration).

Here's an example using the admin script method:

```
NEW_RELIC_CONFIG_FILE=<var>path/to/</var>newrelic.ini newrelic-admin run-program [web server] path_to_app
```

## Python agent API [#api]

You can use the [Python agent API](/docs/agents/python-agent/python-agent-api) to monitor any ASGI server or framework. To do this, mark the [ASGI application entry point](/docs/agents/python-agent/python-agent-api/asgiapplication-python-agent-api) and set the [transaction names](/docs/agents/python-agent/python-agent-api/settransactionname-python-agent-api).

## Event loop diagnostic support [#event-loops]

Our Python agent supports `asyncio` event loop diagnostics. For more information, see [Python event loop diagnostics](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics).
