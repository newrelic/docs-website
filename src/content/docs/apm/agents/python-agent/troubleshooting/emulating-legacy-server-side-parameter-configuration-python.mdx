## Problem

You want to emulate the legacy behavior of enabling the <DNT>**Capture attributes**</DNT> or <DNT>**Capture parameters**</DNT> options in the New Relic UI's [server-side configuration](/docs/agents/manage-apm-agents/configuration/server-side-agent-configuration) settings.

## Solution

You can emulate this legacy behavior by customizing your Python agent configuration file to capture request parameters. For details, see the [Capture request parameters example](/docs/agents/python-agent/attributes/python-attribute-examples#ex_req_params) in the Python attribute documentation.
