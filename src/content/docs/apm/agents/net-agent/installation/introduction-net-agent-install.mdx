Our .NET agent auto-instruments your code so you can start monitoring applications. You can use our launcher, or follow the instructions in this document to complete a basic .NET agent installation. To learn more about what APM for .NET does, see [Introduction to New Relic for .NET](/docs/agents/net-agent/getting-started/introduction-new-relic-net).

If you do not have one already, [create a New Relic account](https://newrelic.com/signup). It's free, forever.
