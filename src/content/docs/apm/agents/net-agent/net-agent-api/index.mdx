The .NET agent API allows you to extend the functionality of the .NET agent. To use the library, add a reference to `NewRelic.Api.Agent.dll` to your project. You can also view the package in the [NuGet Gallery](https://www.nuget.org/packages/NewRelic.Agent.Api/ 'Link opens in a new window.') . If you've disabled or uninstalled the agent, invoking these API calls has no effect.

Among other things, you can use the API to [customize your app name](/docs/agents/net-agent/net-agent-api/setapplicationname-net-agent); [collect errors](/docs/agents/net-agent/net-agent-api/noticeerror-net-agent); record [custom attributes](/docs/agents/net-agent/net-agent-api/addcustomparameter-net-agent), [custom events](/docs/agents/net-agent/net-agent-api/recordcustomevent-net-agent), and [custom metrics](/docs/agents/net-agent/net-agent-api/recordmetric-net-agent); and [enable](/docs/agents/net-agent/net-agent-api/getbrowsertimingheader-net-agent) or [disable](/docs/agents/net-agent/net-agent-api/disablebrowsermonitoring-net-agent) browser monitoring.

For a task-based overview of the API, see [Guide to using the .NET agent API](/docs/agents/net-agent/api-guides/guide-using-net-agent-api).
