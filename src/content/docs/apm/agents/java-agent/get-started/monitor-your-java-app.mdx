Get your Java application monitored by New Relic in just a few steps. This quick start guide walks you through the essentials.

## Before you begin [#prereqs]

Make sure your Java application meets our [compatibility requirements](/docs/apm/agents/java-agent/get-started/compatibility-requirements-java-agent).

You'll need:
* A [New Relic account](https://newrelic.com/signup) (free, no credit card required)
* Java 8 or higher
* Your New Relic [license key](/docs/apis/intro-apis/new-relic-api-keys/#license-key)

## Step 1: Download the Java agent [#download]

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Add data > Application monitoring > Java**
2. Download the `newrelic-java.zip` file
3. Unzip it to a location on your system

The archive contains:
* `newrelic.jar` - The agent
* `newrelic.yml` - Configuration file
* `README.txt` - Documentation

## Step 2: Configure the agent [#configure]

Open `newrelic.yml` and configure two required settings:

1. **License key** - Set your New Relic license key:
   ```yaml
   license_key: YOUR_LICENSE_KEY_HERE
   ```

2. **Application name** - Choose a name for your app in New Relic:
   ```yaml
   app_name: My Java Application
   ```

<Callout variant="tip">
  You can set these via [environment variables](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file#Environment_Variables) or [system properties](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file#System_Properties) instead of editing the YAML file.
</Callout>

## Step 3: Attach the agent to your JVM [#attach]

Add the `-javaagent` flag to your JVM startup command:

```bash
java -javaagent:/full/path/to/newrelic.jar -jar your-application.jar
```

<Callout variant="important">
  The `-javaagent` flag must come **before** the `-jar` flag.
</Callout>

## Step 4: Restart and verify [#verify]

1. Restart your Java application
2. Wait 2-5 minutes for data to appear
3. Go to **[one.newrelic.com](https://one.newrelic.com) > All capabilities > APM & services**
4. Find your application and click on it

You should see:
* Transaction throughput and response times
* Error rates
* JVM metrics (heap memory, garbage collection, thread count)
* External service calls

## Troubleshooting [#troubleshooting]

If you don't see data after 5-10 minutes:

1. **Check the agent log** - Look for `newrelic_agent.log` in your application's logs directory
2. **Verify the -javaagent flag** - Make sure it's in the correct position (before -jar)
3. **Check your license key** - Ensure it's valid and correctly set
4. **Review firewall rules** - The agent needs outbound HTTPS access to New Relic

See our [troubleshooting guide](/docs/apm/agents/java-agent/troubleshooting/no-data-appears-java) for more help.
