Starting with PHP agent release 10.10.0, the tar archive includes `aarch64` binaries for PHP 8.0 and higher.

## Requirements [#requirements]

* PHP 8.0 or higher
* aarch64 (ARM64) CPU architecture
* Root or sudo access on the host

## Install [#install]

ARM64 installation uses the standard tar archive method. Follow the [tar file installation instructions](/docs/apm/agents/php-agent/installation/php-agent-installation-tar-file), selecting the `aarch64` archive when prompted (filename contains `aarch64` or `musl-aarch64`).

<Callout variant="tip">
  If you're on Ubuntu/Debian or RedHat/CentOS running ARM64, the tar archive method is the correct path — package manager (apt/yum) repos only carry x86_64 builds.
</Callout>
