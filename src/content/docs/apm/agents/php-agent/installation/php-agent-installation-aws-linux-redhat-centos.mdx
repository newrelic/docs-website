<Callout variant="important">
    The New Relic PHP Agent RPM signing key will be rotated in March. To continue using the agent uninterrupted, the new key must be imported on all systems installing via rpm/yum with the following command. For more information, [read here](/docs/apm/agents/php-agent/installation/php-agent-installation-rpm-key-rotation).

```rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public```
</Callout>

Our PHP agent auto-instruments your code so you can start monitoring applications. Follow this procedure to install New Relic's PHP agent for APM using Amazon Linux 2, RedHat, or CentOS on x86_64 architecture. For installation on aarch64 architecture, please use [tar file](/docs/php/php-agent-installation-tar-files) method. <DNT>**Exception:**</DNT> If you have an earlier version installed, [upgrade the agent](/docs/agents/php-agent/installation/upgrading-php-agent) instead.

## Install the agent [#install]

Even though the package name for New Relic's PHP agent refers to PHP 5, the package works for all [supported PHP versions](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#php-release).

1. Make sure you have your <InlinePopover type="licenseKey"/> accessible.
2. Add the new RPM key
   ```bash
   rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public
   ```
3. Use either of the following ways to obtain the installation package:

   <CollapserGroup>
     <Collapser
       id="tell-rpm"
       title={<>Tell the package manager (<InlineCode>rpm</InlineCode>) about the New Relic repository</>}
     >
       ```bash
       sudo rpm -Uvh http://yum.newrelic.com/pub/newrelic/el5/x86_64/newrelic-repo-5-3.noarch.rpm
       ```
     </Collapser>

     <Collapser
       id="download-rpm"
       title="Download the rpm file from New Relic"
     >
       * ```
         newrelic-php5-common-X.X.X.X-1.noarch.rpm
         ```

       * ```
         newrelic-daemon-X.X.X.X-1.x86_64.rpm
         ```

       * ```
         newrelic-php5-X.X.X.X-1.x86_64.rpm
         ```
     </Collapser>
   </CollapserGroup>
4. Install the agent and daemon using your preferred package manager:

   <CollapserGroup>
     <Collapser
       id="manager-yum"
       title="yum"
     >
       ```bash
       sudo yum install newrelic-php5
       ```

       The first time you install New Relic for PHP, yum prompts you to accept the New Relic public key. New Relic's key ID is `548C16BF`.
     </Collapser>

     <Collapser
       id="manager-rpm"
       title="rpm"
     >
       Replace `X.X.X.X` with the most recent [PHP agent version number](/docs/release-notes/agent-release-notes/php-release-notes) when you run this command:

       ```bash
       rpm -i newrelic-php5-common-X.X.X.X-1.noarch.rpm newrelic-daemon-X.X.X.X-1.x86_64.rpm newrelic-php5-X.X.X.X-1.x86_64.rpm
       ```
     </Collapser>

     <Collapser
       id="manager-tarball"
       title="tarball"
     >
       If <DNT>**yum**</DNT> and <DNT>**rpm**</DNT> do not work with your host config, [install from the binary tarball](/docs/php/php-agent-installation-tar-files).
     </Collapser>
   </CollapserGroup>
5. Run the `newrelic-install` script and follow the instructions.

   ```bash
   sudo newrelic-install install
   ```
6. Restart your web server (Apache, NGINX, PHP-FPM, etc.).
7. Generate traffic to your application, and wait a few minutes for it to send data to New Relic. Then, [check your app's performance in the UI](/docs/apm/applications-menu/monitoring/apm-overview-page).
