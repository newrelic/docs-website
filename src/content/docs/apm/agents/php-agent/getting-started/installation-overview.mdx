Install the New Relic PHP agent to start monitoring your PHP applications. This page helps you choose the right installation method for your environment.

## Before you begin

1. Verify your system meets the [compatibility requirements](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements)
2. [Sign up for a New Relic account](https://newrelic.com/signup) - it's free, forever!
3. Get your [license key](/docs/apis/intro-apis/new-relic-api-keys/#license-key)

      ## Linux manual installation

      Choose your Linux distribution:

      ### RedHat, CentOS, AWS Linux
      Uses `yum` or `rpm` package manager.

      - [Installation guide](/docs/apm/agents/php-agent/installation/php-agent-installation-aws-linux-redhat-centos)
      - **Time:** 10-15 minutes
      - **Requires:** Root or sudo access

      ### Ubuntu, Debian
      Uses `apt-get` or `dpkg` package manager.

      - [Installation guide](/docs/apm/agents/php-agent/installation/php-agent-installation-ubuntu-debian)
      - **Time:** 10-15 minutes
      - **Requires:** Root or sudo access

      ### ARM64 systems
      For ARM-based processors (like AWS Graviton).

      - [ARM64 installation guide](/docs/apm/agents/php-agent/installation/php-agent-installation-arm64)
      - **Time:** 15-20 minutes
      - **Requires:** Root or sudo access

      ### Other Unix systems (tar archive)
      Universal installation method for any Unix-like OS.

      - [Tar archive installation](/docs/apm/agents/php-agent/installation/php-agent-installation-tar-file)
      - **Time:** 20-30 minutes
      - **Best for:** Non-standard systems or custom setups

      ### Advanced installation options

      - [Use the newrelic-install script](/docs/apm/agents/php-agent/advanced-installation/use-newrelic-install-script-php)
      - [Silent mode installation](/docs/apm/agents/php-agent/advanced-installation/silent-mode-install-script-advanced)
      - [Non-standard PHP installations](/docs/apm/agents/php-agent/advanced-installation/php-agent-installation-non-standard-php-advanced)
      - [Start the daemon manually](/docs/apm/agents/php-agent/advanced-installation/starting-php-daemon-advanced)

      ## Docker and containers

      Install the PHP agent in containerized environments.

      ### Same container (recommended for most users)
      Both agent and daemon run in the same container.

      - [Docker installation guide](/docs/apm/agents/php-agent/advanced-installation/docker-other-container-environments-install-php-agent#install-same-container)
      - **Pros:** Simpler setup, easier debugging
      - **Cons:** Daemon resources per container

      ### Separate containers (advanced)
      Agent in application container, daemon in separate container.

      - [Separate container guide](/docs/apm/agents/php-agent/advanced-installation/docker-other-container-environments-install-php-agent#daemon-container)
      - **Pros:** Shared daemon, lower resource usage
      - **Cons:** More complex networking

      **Tips for Docker:**
      - Use the [tar archive](/docs/apm/agents/php-agent/installation/php-agent-installation-tar-file) for installation
      - Set environment variables in your Dockerfile
      - Ensure proper daemon connectivity
    </TabsPageItem>

      ## Cloud platforms and hosting providers

      ### Heroku
      Deploy the PHP agent on Heroku.

      - [Heroku installation guide](/docs/apm/agents/php-agent/advanced-installation/php-agent-heroku)

      - **AWS Elastic Beanstalk:** Use Linux installation in `.ebextensions`

      - **Google Cloud Compute Engine:** Use Linux installation method

## Installation components

The PHP agent consists of two components that work together:

### PHP extension (`newrelic.so`)
- Instruments your PHP code
- Collects performance data
- Loaded by PHP when your application runs

### Proxy daemon (`newrelic-daemon`)
- Acts as a proxy between the agent and New Relic
- Aggregates and transmits data
- Can serve up to 500 applications per daemon

<Callout variant="tip">
For more details on how these components work together, see [New Relic daemon processes](/docs/apm/agents/php-agent/getting-started/new-relic-daemon-processes).
</Callout>
