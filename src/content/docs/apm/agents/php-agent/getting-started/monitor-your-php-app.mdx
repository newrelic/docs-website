| Environment | Install path |
|---|---|

| Heroku | [PHP agent on Heroku](/docs/apm/agents/php-agent/advanced-installation/php-agent-heroku/) |

<Callout variant="tip">
  Not sure whether to use guided or manual install? Use **guided install** if you want a UI walkthrough. Use **manual install** if you prefer full control from the command line or are automating deployments.
</Callout>

## What's next? [#next]

Once your app is reporting data, you can:

* [View your PHP app in APM](/docs/apm/agents/php-agent/getting-started/php-apm-summary) to understand what normal looks like and where to look first when something changes
* [Configure the PHP agent](/docs/apm/agents/php-agent/configuration/php-agent-configuration) to set your license key, app name, and other options
* [Connect logs and APM data](/docs/logs/logs-context/configure-logs-context-php) with PHP logs in context
* Check [compatibility and requirements](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements) if you're unsure whether your framework or PHP version is supported
