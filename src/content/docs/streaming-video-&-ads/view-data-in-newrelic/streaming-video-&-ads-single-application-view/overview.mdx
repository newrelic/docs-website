Checkout our key features for the New Relic Streaming Video & Ads individual platform view.

* [Summary](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/media-summary-page): View the summary from a specific application/platform where the video agent has been installed.
* [Ad Overview](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/ad-overview): View ad overview from a specific application/platform where the video agent has been installed.
* [User Sessions](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/user-sessions/overview): View User Sessions from a specific application/platform where the video agent has been installed.
* [Service Maps](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/service-maps): View service maps for streaming media and ads to understand dependencies.
* [Alert Conditions](/docs/tutorial-create-alerts/create-an-alert): View alert conditions for a specific application/platform where the video agent has been installed.
