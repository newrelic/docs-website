## Overview

The Individual Platform view provides detailed monitoring and analysis capabilities for a specific streaming application. This view helps you deep dive into application-specific performance, troubleshoot issues isolated to a single platform, and monitor the complete streaming experience for that application.

### What you can do with Individual Platform view

Use this view to:
- Monitor video and ad performance metrics for a specific application where the video agent is installed
- Analyze application-specific video quality and engagement metrics
- View detailed ad performance for the specific application
- Track user sessions within the application
- Troubleshoot issues isolated to a single platform or application
- Monitor service dependencies and relationships for the application
- Create application-specific alert conditions
- Access the Summary page for a consolidated view of the application's performance

### Who uses this view

This view is useful for application developers, product managers, and support teams who need to monitor and troubleshoot a specific streaming application, analyze performance issues within that platform, and ensure the application meets quality standards.

### When to use Individual Platform view vs All Platform view

Use the **Individual Platform view** when you need to:
- Deep dive into a specific application's performance
- Troubleshoot issues isolated to a single platform
- Monitor application-specific metrics and configurations
- Analyze user sessions within a specific application
- View service maps and dependencies for the application
- Create application-specific alerts

Use the **All Platform view** when you need to:
- Compare performance across multiple applications
- Identify trends affecting your entire streaming infrastructure
- Monitor overall streaming health across all platforms
- Investigate issues that may be affecting multiple applications
