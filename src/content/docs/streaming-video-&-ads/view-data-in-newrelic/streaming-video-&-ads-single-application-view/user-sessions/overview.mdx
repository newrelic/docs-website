## Overview

The User Sessions page provides a detailed view of individual viewing sessions for a specific monitored application. This page helps you track user viewing behavior, identify sessions with errors, and analyze playback patterns within a single streaming application or platform.

### What you can do with User Sessions

Use this page to:
- View all user viewing sessions for a specific application where the video agent is installed
- Track session-level details including View ID, Session ID, and User ID
- Monitor content being viewed by title
- Identify sessions with errors for troubleshooting
- View session start and end times to analyze viewing duration
- Access detailed session information by clicking on View ID or Session ID
- Filter sessions by time range to focus on specific periods
- Analyze viewing patterns for the specific device and platform
- Review content details for each session

### Who uses this page

This page is useful for application developers, customer support teams, and quality assurance engineers who need to troubleshoot user-reported issues for a specific application, analyze individual viewing sessions, and understand user behavior patterns within that platform.

## User Sessions page

The **Single platform > User Sessions** for Streaming Video & Ads lists all the sessions across a specific application.
