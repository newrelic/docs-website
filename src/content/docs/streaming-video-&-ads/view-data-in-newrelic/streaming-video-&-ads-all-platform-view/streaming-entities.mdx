## Overview

The Streaming Entities page provides a centralized view of all applications with the New Relic video agent installed. This page helps you monitor and compare video and ad playback performance across all your streaming applications and platforms.

### What you can do with Streaming Entities

Use this page to:
- View all applications where the video agent has been instrumented in a single table
- Compare video and ad KPIs across different applications and platforms
- Identify applications experiencing high rebuffering, start failures, or playback failures
- Assess video quality metrics like average bitrate and start time across all entities
- Click on any application to examine detailed video and ad playback data
- Create new streaming entities directly from this page

### Who uses this page

This page is useful for teams managing multiple streaming applications across different platforms who need to quickly assess performance across their entire streaming infrastructure and identify which applications require attention.

## Streaming entities page

The Streaming Entities page displays all applications with the New Relic video agent installed. In the table view, you'll find a summary of the video and ad KPIs. You can click on any application to examine and analyze its video and ad playback data.

## Streaming entities overview page UI

To view the **Streaming Entities** across all platforms, go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > All Capabilities > Streaming Video & Ads > Streaming Entity**</DNT>.

## Add Streaming entity

If you wish to create a new streaming entity:

1. Click **+ Add streaming entity**, top right corner of the page.
2. Select the type of app you want to monitor. See [Install the video agent](/docs/streaming-video-&-ads/installation/install-the-streaming-video-&-ads-agent/#installation)
