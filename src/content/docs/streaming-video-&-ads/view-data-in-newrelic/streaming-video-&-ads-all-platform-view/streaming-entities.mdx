The Streaming Entities page displays all applications with the New Relic video agent installed. In the table view, you'll find a summary of the video and ad KPIs. You can click on any application to examine and analyze its video and ad playback data.

## Streaming entities overview page UI

To view the **Streaming Entities** across all platforms, go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > All Capabilities > Streaming Video & Ads > Streaming Entity**</DNT>.

## Add Streaming entity

If you wish to create a new streaming entity:

1. Click **+ Add streaming entity**, top right corner of the page.
2. Select the type of app you want to monitor. See [Install the video agent](/docs/streaming-video-&-ads/installation/install-the-streaming-video-&-ads-agent/#installation)
