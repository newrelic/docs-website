## Overview

The User Sessions page provides a detailed view of individual viewing sessions across all your monitored streaming applications and platforms. This page helps you track user viewing behavior, identify sessions with errors, and analyze playback patterns across your entire streaming infrastructure.

### What you can do with User Sessions

Use this page to:
- View all user viewing sessions across all applications where the video agent is installed
- Track session-level details including View ID, Session ID, User ID, and Application Name
- Monitor content being viewed by title and platform
- Identify sessions with errors for troubleshooting
- View session start and end times to analyze viewing duration
- Access detailed session information by clicking on View ID or Session ID
- Filter sessions by time range to focus on specific periods
- Analyze viewing patterns across different devices and platforms
- Review content details for each session

### Who uses this page

This page is useful for customer support teams, quality assurance engineers, and operations teams who need to troubleshoot user-reported issues, analyze individual viewing sessions across platforms, and understand user behavior patterns across all streaming applications.

## User Sessions page

The **All Platforms > User Sessions** for Streaming Video & Ads lists all the sessions across all the applications.
