## Overview

The All Platform view provides a centralized location to monitor video and ad playback performance across all your streaming applications and platforms. This view helps you compare performance across different applications, identify trends across your streaming infrastructure, and quickly spot issues affecting multiple platforms.

### What you can do with All Platform view

Use this view to:
- Monitor streaming performance across all applications where the video agent has been instrumented
- Compare video and ad KPIs across different applications and platforms
- Identify platform-wide issues affecting multiple applications
- Analyze performance trends across your entire streaming infrastructure
- View all active streaming entities in one place
- Monitor user sessions across all applications and platforms
- Access video and ad overview metrics aggregated across all platforms

### Who uses this view

This view is useful for streaming operations teams, engineering managers, and executives who need a high-level view of streaming performance across the entire organization, can compare performance between different applications or platforms, and need to identify infrastructure-wide issues.

### When to use All Platform view vs Individual Platform view

Use the **All Platform view** when you need to:
- Compare performance across multiple applications
- Identify trends affecting your entire streaming infrastructure
- Monitor overall streaming health across all platforms
- Investigate issues that may be affecting multiple applications

Use the **Individual Platform view** when you need to:
- Deep dive into a specific application's performance
- Troubleshoot issues isolated to a single platform
- Monitor application-specific metrics and configurations
- Analyze user sessions within a specific application
