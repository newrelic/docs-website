Video agent for Android monitors the video player in your mobile app, giving you a comprehensive view of your video and ad playback performance. It works for Android apps written using Java or Kotlin.

## Prerequisites

Before you install the Android video agent, ensure you have:

- A New Relic [account ID](/docs/accounts/accounts-billing/account-setup/account-id)
- Reviewed the [compatibility and requirements](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements)
- Reviewed the [compatibility and requirements](/docs/streaming-video-&-ads/installation/compatibility-requirements) for the Streaming Video & Ads agent

## Before you begin

Determine which installation path applies to you:

- **New mobile application**: You're adding New Relic to an app for the first time. You'll need to install the [New Relic Android agent](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android) first.
- **Existing monitored application**: Your app already has the New Relic Android agent installed and is sending data to New Relic.

## Install the agent

<Steps>

<Step>
### Access New Relic

Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT>.
</Step>

<Step>
### Select Streaming Video & Ads

Search and select the <DNT>**Streaming Video & Ads**</DNT> tile.
</Step>

<Step>
### Select Mobile platform

Select <DNT>**Mobile**</DNT> and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Select Android

Select <DNT>**Android**</DNT> to monitor video players in your Android applications and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Name your application

Give a name for your application and click <DNT>**Create**</DNT>.

Once the application name is created, click <DNT>**Save and continue**</DNT>.
</Step>

<Step>
### Choose your instrumentation method

Select the instrumentation method that applies to your application:

<DocTiles>
  <DocTile title='Install automatically using jitpack' path="/docs/streaming-video-&-ads/installation/android/install-automatically-using-jitpack">Automatically install the video agent using jitpack</DocTile>
  <DocTile title='Install manually using AAR Files' path="/docs/streaming-video-&-ads/installation/android/install-manually-using-aar-files">Manually install the video agent using AAR files</DocTile>
  <DocTile title='Install manually using source code' path="/docs/streaming-video-&-ads/installation/android/install-manually-using-source-code">Manually install the video agent using source code</DocTile>
</DocTiles>
</Step>

</Steps>
