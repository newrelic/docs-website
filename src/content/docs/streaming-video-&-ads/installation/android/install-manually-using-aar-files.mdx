This page provides an overview of the installation steps when using AAR files to manually install the Streaming Video & Ads agent. For the complete installation with code examples, use the [guided install](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clone the [video-agent-android](https://github.com/newrelic/video-agent-android) repository.
  </Step>

  <Step>
    Open it with Android Studio.
  </Step>

  <Step>
    Go to <DNT>**View > Tool Windows > Gradle**</DNT> to open the Gradle tool window.
  </Step>

  <Step>
    In there, unfold <DNT>**NRVideoProject > Tasks > build**</DNT> and double-click <DNT>**assemble**</DNT>. This will generate the AAR libraries inside the module's folder <DNT>**build > outputs > aar**</DNT>.
  </Step>

  <Step>
    In your project, go to <DNT>**File > New > New Module**</DNT>, import `.JAR/.AAR` package and click <DNT>**Next**</DNT>.
  </Step>

  <Step>
    Select the AAR files you generated in the previous step and click <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    Repeat steps 5 and 6 with all the AAR files you want to include.
  </Step>

  <Step>
    In your app module's `build.gradle` file, add the following:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    Click <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Set up video player tracking

    Select the agent type that matches your implementation:
    - **Video agent with ExoPlayer tracker only**: For video playback tracking
    - **Video agent with ExoPlayer and IMA trackers**: For video playback and ad tracking

        <img
          title="Install automatically using jitpack"
          alt="A gif depicting how to install the video agent using jitpack"
          src="/images/install-automatically-using-jitpack.gif"
        />
  </Step>

  <Step>
    Click <DNT>**See your data**</DNT> to deploy your updated code and see the data.
  </Step>
</Steps>
