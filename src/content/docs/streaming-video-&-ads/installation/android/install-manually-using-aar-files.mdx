This page provides an overview of the installation steps when using AAR files to manually install the Streaming Video & Ads agent. For the complete installation with code examples, use the [guided install](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clone the [video-agent-android](https://github.com/newrelic/video-agent-android) repository.
  </Step>

  <Step>
    Open it with Android Studio.
  </Step>

  <Step>
    Go to <DNT>**View > Tool Windows > Gradle**</DNT> to open the Gradle tool window.
  </Step>

  <Step>
    In the Gradle tool window, unfold <DNT>**NRVideoProject > Tasks > build**</DNT> and double-click <DNT>**assemble**</DNT>. This will build the project and generate the AAR library files inside each module's folder <DNT>**build > outputs > aar**</DNT>.
  </Step>

  <Step>
    In your project, go to <DNT>**File > New > New Module**</DNT>, import `.JAR/.AAR` package and click <DNT>**Next**</DNT>.
  </Step>

  <Step>
    Select the AAR files you generated in the previous step and click <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    Repeat steps 5 and 6 for each AAR file you need:
    - **NewRelicVideoCore.aar** (Required): Core video agent functionality
    - **NRExoPlayerTracker.aar** (Required): ExoPlayer tracking
    - **NRIMATracker.aar** (Optional): Ad tracking with IMA SDK
  </Step>

  <Step>
    In your app module's `build.gradle` file, add the following dependencies:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")   // Required: Core video agent
      implementation project(":NRExoPlayerTracker")  // Required: ExoPlayer tracking
      implementation project(':NRIMATracker')        // Optional: Ad tracking with IMA
    }
    ```

    Note: Only include `NRIMATracker` if you need ad tracking functionality.
  </Step>

  <Step>
    Sync your project with Gradle files by clicking <DNT>**Sync Now**</DNT> in the notification bar, or go to <DNT>**File > Sync Project with Gradle Files**</DNT>.
  </Step>

  <Step>
    ### Initialize the video agent

    In your application's `onCreate()` method or activity, initialize the New Relic video agent. For detailed code examples and implementation instructions, refer to the [guided install](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    ### Set up video player tracking

    Configure the tracker based on your needs:
    - **ExoPlayer tracker only**: Set up ExoPlayer tracking for video playback monitoring
    - **ExoPlayer and IMA trackers**: Set up both ExoPlayer and IMA tracking for video playback and ad tracking

    <img
      title="Install automatically using jitpack"
      alt="A gif depicting how to install the video agent using jitpack"
      src="/images/install-automatically-using-jitpack.gif"
    />

    For complete implementation code examples, see the [guided install](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    Build and deploy your application to see the video tracking data in New Relic.
  </Step>
</Steps>
