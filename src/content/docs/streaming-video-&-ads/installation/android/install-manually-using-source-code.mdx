This page provides an overview of the installation steps when using source code to manually install the Streaming Video & Ads agent. For the complete installation with code examples, use the [guided install](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clone the [video-agent-android](https://github.com/newrelic/video-agent-android) repository.
  </Step>

  <Step>
    In your Android project, go to <DNT>**File > New > Import Module**</DNT>.
  </Step>

  <Step>
    Enter the location of the library module directory and click <DNT>**Finish**</DNT>.

    <Callout variant="important">
      Library module directory is located in the repo you just cloned. Example modules: `NewRelicVideoCore`, `NRExoPlayerTracker`, `NRIMATracker`.
    </Callout>
  </Step>

  <Step>
    Repeat steps 2 and 3 with all the modules you want to include.
  </Step>

  <Step>
    In your app module's `build.gradle` file, add the following:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    Click <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Set up video player tracking

    Select the agent type that matches your implementation:
    - **Video agent with ExoPlayer tracker only**: For video playback tracking
    - **Video agent with ExoPlayer and IMA trackers**: For video playback and ad tracking

        <img
          title="Install automatically using jitpack"
          alt="A gif depicting how to install the video agent using jitpack"
          src="/images/install-automatically-using-jitpack.gif"
        />
  </Step>

  <Step>
    Click <DNT>**See your data**</DNT> to deploy your updated code and see the data.
  </Step>
</Steps>
