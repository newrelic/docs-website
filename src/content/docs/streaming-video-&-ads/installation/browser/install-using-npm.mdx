Install the video agent using npm package manager for JavaScript applications.

## Before you begin

Complete the [common installation steps](/docs/streaming-video-&-ads/installation/browser/get-started-with-browser#install) first.

## Install using npm [#install-npm]

<Steps>

<Step>
### Name your application

Provide a name to your application and click <DNT>**Save and Continue**</DNT>.
</Step>

<Step>
### Install the agent

Install the latest release from New Relic by copying the code snippet and running it in your terminal (Mac) or command prompt (Windows).

<Callout variant="important">
  Select **npm** or **Yarn** to install the video agent.
</Callout>
</Step>

<Step>
### Add code to your application

After the script is downloaded, include the code snippet displayed on the platform in your application to start monitoring your video player.
</Step>

<Step>
### Generate application ID

Select <DNT>**Get Application Info**</DNT>. This will generate the application id for the browser entity generation.
</Step>

<Step>
### View your data

Click <DNT>**See your data**</DNT>.
</Step>

</Steps>

<Callout variant="important">
    It may take a few minutes to interact with your application and generate data.
</Callout>
