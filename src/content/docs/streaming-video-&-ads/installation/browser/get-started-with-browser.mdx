To monitor video player performance in your browser application, install the New Relic video agent using npm or a JavaScript snippet.

## Prerequisites [#before-start]

Before installing the Streaming Video & Ads agent for Browser, verify your application meets these requirements:
- [Browser monitoring compatibility requirements](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring)
- [Streaming Video & Ads agent compatibility requirements](/docs/streaming-video-&-ads/installation/compatibility-requirements)

## Install the agent [#install]

<Steps>

<Step>
### Access New Relic

Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT> and navigate to the integrations marketplace.
</Step>

<Step>
### Select Streaming Video & Ads

Search and select the <DNT>**Streaming Video & Ads**</DNT> tile.
</Step>

<Step>
### Select account

Select an <DNT>**Account**</DNT> from the dropdown and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Select platform

Select <DNT>**Browser**</DNT> and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Select video player

Select your video player and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Choose installation method

<DocTiles>
  <DocTile title='Install using npm' path="/docs/streaming-video-&-ads/installation/browser/install-using-npm">Install the video agent package manager for JavaScript applications.</DocTile>
  <DocTile title='Place a JavaScript snippet in frontend code' path="/docs/streaming-video-&-ads/installation/browser/install-using-snippet">Install the video agent by including a JavaScript code snippet in your application.</DocTile>
</DocTiles>
</Step>

</Steps>

## What's next?

After installing the agent:

1. Verify data is flowing in New Relic by checking for your application in the Streaming Video & Ads dashboard
2. [View your data](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-features): Explore monitoring features and dashboards
3. [Configure Quality of Experience thresholds](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): Customize performance metrics for your application
