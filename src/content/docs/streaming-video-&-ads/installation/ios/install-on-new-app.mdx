If you're adding New Relic to your app for the first time, follow these steps:

1. [Install the iOS agent](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) on an application you're not yet monitoring with New Relic.

2. Once you've completed all the steps, test the connection to ensure everything is working properly and view your data.

  <img
      width="80%;"
      title="UI image of iOS test connection"
      alt="UI image of iOS test connection"
      src="/images/ios-test-connection.webp"
  />

3. After the iOS agent is installed, you will be redirected to the <DNT>**Install media streaming agent**</DNT> page.

4. Click <DNT>**Continue**</DNT>.

5. Choose your instrumentation method from the following options:

<Tabs>
    <TabsBar>
        <TabsBarItem id="cocoapods-new">
            Install automatically using CocoaPods
        </TabsBarItem>
        <TabsBarItem id="manual-new">
            Install manually
        </TabsBarItem>
    </TabsBar>

    <TabsPages>
        <TabsPageItem id="cocoapods-new">
            1. Add the following lines to your Podfile:

            ```json
            pod 'NewRelicVideoCore', :git => 'https://github.com/newrelic/video-agent-iOS'
            pod 'NRAVPlayerTracker', :git => 'https://github.com/newrelic/video-agent-iOS'
            pod 'NRIMATracker', :git => 'https://github.com/newrelic/video-agent-iOS'
            ```

            2. Run `pod install` in your terminal.

            3. Open your project in Xcode and build it.

            4. Click <DNT>**Continue**</DNT>.

            5. Select an agent to deploy:

            <Tabs>
                <TabsBar>
                    <TabsBarItem id="avplayer-new-cocoapods">
                        Video agent with AVPlayer tracker only
                    </TabsBarItem>
                    <TabsBarItem id="avplayer-ima-new-cocoapods">
                        Video agent with AVPlayer and IMA trackers
                    </TabsBarItem>
                </TabsBar>

                <TabsPages>
                    <TabsPageItem id="avplayer-new-cocoapods">

                        ### Objective-C

                        ```Objective-C
                        NSNumber *trackerId = [[NewRelicVideoAgent sharedInstance] startWithContentTracker:[[NRTrackerAVPlayer alloc] initWithAVPlayer:player]];
                        ```

                        ### Swift

                        ```Swift
                        let trackerId = NewRelicVideoAgent.sharedInstance().start(withContentTracker: NRTrackerAVPlayer.init(avPlayer: player))
                        ```
                    </TabsPageItem>

                    <TabsPageItem id="avplayer-ima-new-cocoapods">

                        ### Objective-C

                        ```Objective-C
                        NSNumber *trackerId = [[NewRelicVideoAgent sharedInstance] startWithContentTracker:[[NRTrackerAVPlayer alloc] initWithAVPlayer:player] adTracker:[[NRTrackerIMA alloc] init]];
                        ```

                        ### Swift

                        ```Swift
                        let trackerId = NewRelicVideoAgent.sharedInstance().start(withContentTracker: NRTrackerAVPlayer.init(avPlayer: player), adTracker: NRTrackerIMA.init())
                        ```
                    </TabsPageItem>
                </TabsPages>
            </Tabs>

            6. Click <DNT>**See your data**</DNT> to deploy your updated code and view the data.
        </TabsPageItem>

        <TabsPageItem id="manual-new">
        1. Clone the [video-agent-iOS](https://github.com/newrelic/video-agent-ios) repository.

        2. Open each of the `.xcodeproj` files with Xcode.

        3. Select the appropriate scheme.

        4. Build (CMD+B).

        5. Include the generated `.framework` in your project and click <DNT>**Continue**</DNT>.

        6. Select an agent to deploy:

            <Tabs>
                <TabsBar>
                    <TabsBarItem id="avplayer-new-manual">
                        Video agent with AVPlayer tracker only
                    </TabsBarItem>
                    <TabsBarItem id="avplayer-ima-new-manual">
                        Video agent with AVPlayer and IMA trackers
                    </TabsBarItem>
                </TabsBar>

                <TabsPages>
                    <TabsPageItem id="avplayer-new-manual">

                        ### Objective-C

                        ```Objective-C
                        NSNumber *trackerId = [[NewRelicVideoAgent sharedInstance] startWithContentTracker:[[NRTrackerAVPlayer alloc] initWithAVPlayer:player]];
                        ```

                        ### Swift

                        ```Swift
                        let trackerId = NewRelicVideoAgent.sharedInstance().start(withContentTracker: NRTrackerAVPlayer.init(avPlayer: player))
                        ```
                    </TabsPageItem>

                    <TabsPageItem id="avplayer-ima-new-manual">

                        ### Objective-C

                        ```Objective-C
                        NSNumber *trackerId = [[NewRelicVideoAgent sharedInstance] startWithContentTracker:[[NRTrackerAVPlayer alloc] initWithAVPlayer:player] adTracker:[[NRTrackerIMA alloc] init]];
                        ```

                        ### Swift

                        ```Swift
                        let trackerId = NewRelicVideoAgent.sharedInstance().start(withContentTracker: NRTrackerAVPlayer.init(avPlayer: player), adTracker: NRTrackerIMA.init())
                        ```
                    </TabsPageItem>
                </TabsPages>
            </Tabs>

        7. Click <DNT>**See your data**</DNT> to deploy your updated code and view the data.
        </TabsPageItem>
    </TabsPages>
</Tabs>
