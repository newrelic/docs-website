The video agent for iOS gives you a comprehensive view of your video playback performance. The agent monitors iOS apps written in Objective-C, Swift, or both.

## Prerequisites

Before you install the iOS video agent, ensure you have:

- A New Relic [account ID](/docs/accounts/accounts-billing/account-setup/account-id)
- Reviewed the [compatibility and requirements](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements)

## Before you begin

Determine which installation path applies to you:

- **New mobile application**: You're adding New Relic to an app for the first time. You'll need to install the [New Relic iOS agent](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) first.
- **Existing monitored application**: Your app already has the New Relic iOS agent installed and is sending data to New Relic.

## Install the agent

<Steps>

<Step>
### Access New Relic

Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT>.
</Step>

<Step>
### Select Streaming Video & Ads

Search and select the <DNT>**Streaming Video & Ads**</DNT> tile.
</Step>

<Step>
### Name your application

Give a name for your application and click <DNT>**Create**</DNT>.

Once the application name is created, click <DNT>**Save and Continue**</DNT>.
</Step>

<Step>
### Select Mobile platform

Select <DNT>**Mobile**</DNT> and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Select iOS

Select <DNT>**iOS**</DNT> to monitor video players in your applications and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Choose your installation path

Select the installation path that applies to your application:

<DocTiles>
  <DocTile title='New mobile application' path="/docs/streaming-video-&-ads/installation/ios/install-on-new-app">Install the video agent for a new iOS application.</DocTile>
  <DocTile title='Existing monitored application' path="/docs/streaming-video-&-ads/installation/ios/install-on-existing-app">Install the video agent on an existing monitored iOS application.</DocTile>
</DocTiles>
</Step>

</Steps>
