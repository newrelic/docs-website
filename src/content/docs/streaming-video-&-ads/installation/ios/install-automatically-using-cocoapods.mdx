This page provides an overview of the installation steps when using CocoaPods to install the Streaming Video & Ads agent. For the complete installation with code examples, use the [guided install](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios).

<Steps>
  <Step>
    ### Add dependencies

    Add the video agent pods to your Podfile.
  </Step>

  <Step>
    ### Install pods

    Run `pod install` in your terminal.
  </Step>

  <Step>
    ### Build project

    Open your project in **Xcode** and build it.
  </Step>

  <Step>
    Click <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Initialize video tracker

    Select the agent type that matches your implementation:
    - **Video agent with AVPlayer tracker only**: For video playback tracking
    - **Video agent with AVPlayer and IMA trackers**: For video playback and ad tracking
  </Step>

  <Step>
    Click <DNT>**See your data**</DNT> to view your streaming data in New Relic.
  </Step>
</Steps>
