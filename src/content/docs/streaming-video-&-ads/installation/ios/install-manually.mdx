This page provides an overview of the installation steps when manually installing the Streaming Video & Ads agent. For the complete installation with code examples, use the [guided install](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios).

<Steps>
  <Step>
    Clone the [video-agent-iOS](https://github.com/newrelic/video-agent-ios) repository.
  </Step>

  <Step>
    Open each of the `.xcodeproj` files with Xcode.
  </Step>

  <Step>
    Select the appropriate scheme.
  </Step>

  <Step>
    Build (CMD+B).
  </Step>

  <Step>
    Include the generated `.framework` files in your project.
  </Step>

  <Step>
    Click <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Initialize video tracker

    Select the agent type that matches your implementation:
    - **Video agent with AVPlayer tracker only**: For video playback tracking
    - **Video agent with AVPlayer and IMA trackers**: For video playback and ad tracking
  </Step>

  <Step>
    Click <DNT>**See your data**</DNT> to view your streaming data in New Relic.
  </Step>
</Steps>
