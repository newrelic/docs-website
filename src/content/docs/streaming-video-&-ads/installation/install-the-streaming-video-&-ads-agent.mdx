Follow these steps to install the Streaming Video & Ads agent.

## Prerequisites [#requirements]

Before installing the video agent:

* Ensure you have a New Relic [account ID](/docs/accounts/accounts-billing/account-setup/account-id)
* Check [compatibility requirements](/docs/streaming-video-&-ads/installation/compatibility-requirements) for supported platforms and versions

## Install the video agent [#install]

<Steps>

<Step>
### Access New Relic

Navigate to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents > Streaming Video & Ads**</DNT>.
</Step>

<Step>
### Select account

Select your account from the dropdown and click <DNT>**Continue**</DNT>.
</Step>

<Step>
### Select platform

Choose your platform to continue with platform-specific installation:
