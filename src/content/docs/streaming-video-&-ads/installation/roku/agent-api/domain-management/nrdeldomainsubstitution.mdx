## Syntax

```js
  nrDelDomainSubstitution(nr as object, pattern as String) as Void
```

## Description

  Delete a matching pattern created with `nrAddDomainSubstitution`.

## Example

```js
  	nrDelDomainSubstitution(nr, "^.+\.my\.domain\.com$")
```
