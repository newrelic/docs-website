## Harvest configuration methods

<table>
  <thead>
    <tr>
      <th width="30%">
        Method
      </th>
      <th>
        Description
      </th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        [nrSetHarvestTime](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttime)
      </td>
      <td>
        Set harvest time, the time the samples are buffered before being sent to New Relic for both Events and Logs. Min value is 60.
      </td>
    </tr>
    <tr>
      <td>
        [nrSetHarvestTimeEvents](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimeevents)
      </td>
      <td>
        Set harvest time for Events, the time the events are buffered before being sent to New Relic. Min value is 60.
      </td>
    </tr>
    <tr>
      <td>
        [nrSetHarvestTimeLogs](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimelogs)
      </td>
      <td>
        Set harvest time for Logs, the time the logs are buffered before being sent to New Relic. Min value is 60.
      </td>
    </tr>
    <tr>
      <td>
        [nrSetHarvestTimeMetrics](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimemetrics)
      </td>
      <td>
        Set harvest time for Logs, the time the logs are buffered before being sent to New Relic. Min value is 60.
      </td>
    </tr>
    <tr>
      <td>
        [nrForceHarvest](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvest)
      </td>
      <td>
        Do harvest events and logs immediately. It doesn't reset the harvest timer.
      </td>
    </tr>
    <tr>
      <td>
        [nrForceHarvestEvents](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvestevents)
      </td>
      <td>
        Do harvest events immediately. It doesn't reset the harvest timer.
      </td>
    </tr>
    <tr>
      <td>
        [nrForceHarvestLogs](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvestlogs)
      </td>
      <td>
        Do harvest logs immediately. It doesn't reset the harvest timer.
      </td>
    </tr>
  </tbody>
</table>

## Examples

### nrSetHarvestTime

```js
  	nrSetHarvestTime(m.nr, 60)
```

### nrSetHarvestTimeEvents

```js
  	nrSetHarvestTimeEvents(m.nr, 60)
```

### nrSetHarvestTimeLogs

```js
  	nrSetHarvestTimeLogs(m.nr, 60)
```

### nrSetHarvestTimeMetrics

```js
  	nrSetHarvestTimeMetrics(m.nr, 60)
```

### nrForceHarvest

```js
  	nrForceHarvest(m.nr)
```

### nrForceHarvestEvents

```js
  	nrForceHarvestEvents(m.nr)
```

### nrForceHarvestLogs

```js
  	nrForceHarvestLogs(m.nr)
```
