## Syntax

```js
  nrAppStarted(nr as Object, obj as Object) as Void
```

## Description

Send an APP_STARTED event of type ConnectedDeviceSystem.

  <thead>

      </td>
    </tr>

    <tr>
      <td>
        `obj`
      </td>

      <td>

      </td>
    </tr>
  </tbody>
</table>
	
## Return

Nothing.
	
## Example

```js
  	sub Main(aa as Object)
		        ...
		        nrAppStarted(m.nr, aa)
```
