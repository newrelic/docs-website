## Syntax

```js
  nrSendLog(nr as Object, message as String, logtype as String, fields = invalid as Object) as Void
```

## Description

  Record a log using the New Relic Log API.

## Example

```js
  	nrSendLog(m.nr, "This is a log", "console", {"key": "value"})
```
