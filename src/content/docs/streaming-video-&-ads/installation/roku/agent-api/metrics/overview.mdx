## Metrics methods

<table>
  <thead>

        [nrSendMetric](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendmetric)

      <td>
        [nrSendCountMetric](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendcountmetric)

      </td>
    </tr>

## Examples

### nrSendMetric

```js
  	nrSendMetric(m.nr, "test", 11.1, {"one": 1})
```

### nrSendCountMetric

```js
  	nrSendCountMetric(m.nr, "test", 250, 1500, {"one": 1})
```

### nrSendSummaryMetric

```js
  	nrSendSummaryMetric(m.nr, "test", 2000, 5, 1000, 100, 200)
```
