## Syntax

```js
  nrSendCustomEvent(nr as Object, eventType as String, actionName as String, attr = invalid as Object) as Void
```

## Description

Send a custom event.

## Example

```js
  	nrSendCustomEvent(m.nr, "MyEvent", "MY_ACTION")
	  attr = {"key0":"val0", "key1":"val1"}
```
