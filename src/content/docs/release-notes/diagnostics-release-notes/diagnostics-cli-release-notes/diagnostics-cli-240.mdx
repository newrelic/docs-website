## Changes

### Attach
When uploading a Diagnostics CLI run to your account using the `-a` flag, you will now see a permalink to the run in the [Diagnostics CLI Output app](https://one.newrelic.com/diagnostics/cli-output).  This will only work if you have a valid license key.
