## Changes
### New Feature
`-api-key` command line flag
  * Use your <InlinePopover type="userKey" /> to automatically attach a Diagnostics CLI run to your account. This bypasses the requirement for a validated license key to attach a run.
  * Example usage: `./nrdiag -api-key NRAK-123...............`
  * [Read more here](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/run-diagnostics-cli-nrdiag/#automatic-api)

### Python agent
- **Python/Env/Version**: Add check for output of `python3 –version`
- **Python/Env/Dependencies**: Add check for output of `pip3 –version`
- **Python/Requirements/PythonVersion:** Update New Relic Agent Supportability for Python 3.6, 3.9, 3.10, and 3.11.
