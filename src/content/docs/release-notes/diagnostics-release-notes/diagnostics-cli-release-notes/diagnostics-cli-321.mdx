## Task updates

* Added new task to check if the user executing the CLI has root permissions. ([#195](https://github.com/newrelic/newrelic-diagnostics-cli/pull/195))
* Updated `Infra/Log/Collect` to include relevant environment variables and default log locations. ([#195](https://github.com/newrelic/newrelic-diagnostics-cli/pull/195))
* Updated `Infra/Agent/Debug` to make log collection more reliable and to allow log collection to be interrupted gracefully. ([#195](https://github.com/newrelic/newrelic-diagnostics-cli/pull/195))
* Fixed an issue with `Base/Log/Copy` that resulted in a blank entry being added to the list of logs to copy. ([#195](https://github.com/newrelic/newrelic-diagnostics-cli/pull/195))
* Fixed the Success summary message for `Dotnet/Config/Agent`.  ([#196](https://github.com/newrelic/newrelic-diagnostics-cli/pull/196))
* Added Ruby agent environment variables `NEW_RELIC_LOG_FILE_PATH` and `NEW_RELIC_LOG_FILE_NAME` to the list of environment variables to check when collecting logs. ([#196](https://github.com/newrelic/newrelic-diagnostics-cli/pull/196))
* Updated `Base/Collector/ConnectUS` and `Base/Collector/ConnectEU` to use new endpoints, [https://collector.newrelic.com/status/mongrel](https://collector.newrelic.com/status/mongrel) for the US and [https://collector.eu.newrelic.com/status/mongrel](https://collector.eu.newrelic.com/status/mongrel) for the EU. ([#197](https://github.com/newrelic/newrelic-diagnostics-cli/pull/197))
* Removed `Base/Collector/ConnectTLS`. This task was redundant for `Base/Collector/ConnectUS` and `Base/Collector/ConnectEU`. ([#197](https://github.com/newrelic/newrelic-diagnostics-cli/pull/197))
