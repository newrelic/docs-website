### New docs

* Added [Security RX Jira workflows](/docs/vulnerability-management/workflows/jira) to provide guidance on automatically creating and syncing Jira tickets for detected application vulnerabilities.
* Added [Stop an Azure virtual machine](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-vm-stop) to the actions catalog to provide guidance on managing the Azure VM lifecycle directly from workflows.
* Added [Save and manage dashboard preferences](/docs/streaming-video-&-ads/view-data-in-newrelic/user-preferences) to explain how New Relic saves and restores Video Overview and Ad Overview dashboard configurations across sessions.
* Added obfuscation rules documentation for the Streaming Video & Ads agent on [Android](/docs/streaming-video-&-ads/installation/android/obfuscation-rules), [browser](/docs/streaming-video-&-ads/installation/browser/obfuscation-rules), [iOS](/docs/streaming-video-&-ads/installation/ios/obfuscation-rules), and [Roku](/docs/streaming-video-&-ads/installation/roku/obfuscation-rules) to provide guidance on masking sensitive data with regex-based rules before transmission.

### Major changes

* Updated [Vulnerability management integrations overview](/docs/vulnerability-management/getting-started/integrations/overview) and [applications overview](/docs/vulnerability-management/applications/overview) to set 10.51.1 as the base version for the Security RX .NET integration.

### Release notes

* Stay up-to-date with our latest releases:

  * [Node.js agent v14.1.2](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-14-1-2):
    * Fixed Azure Functions crashing when they include response headers.
    * Updated the compatibility report.

  * [Node.js agent v14.1.1](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-14-1-1):
    * Updated the minimum version of `@opentelemetry` `0.x` packages to `^0.219.0`.
    * Added proxy configuration support and supportability metrics to OpenTelemetry metrics export.

  * [Ruby agent v10.6.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-10-6-0):
    * Added Hybrid agent support for SpanLink events, SpanEvent events, and span kind on all spans.
    * Added support for the `OpenTelemetry::Tracer#start_root_span` API.
    * Fixed `instrumentation.rails_event_logger: false` not disabling the instrumentation.

  * [.NET agent v10.52.0](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-52-0):
    * Added Kafka internal metrics and SQL metadata comments for APM to DB Query Performance Monitoring correlation.
    * Added alternative environment variables for expected and ignored error codes.
    * Fixed distributed tracing for HttpWebRequest body and async requests.

  * [Go agent v3.44.1](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-3-44-1):
    * Corrected the tag from the previous release.
    * Applied Dependabot security updates for nrgochi, nrmongo, nrmongo-v2, and nrnats.

  * [NRDOT v1.18.0](/docs/release-notes/nrdot-release-notes/nrdot-2026-06-22):
    * Added forked database components and additional providers for the database use case.
    * Added OHI components for AWS ECS container metrics, HAProxy, Kafka, and Redis.

  * [Infrastructure agent v1.77.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1770):
    * Added support for Ubuntu 26.04 LTS (Resolute Raccoon).
    * Updated the nri-flex and nri-prometheus dependencies.

  * [Synthetics Job Manager Release 568](/docs/release-notes/synthetics-release-notes/job-manager-release-notes/job-manager-release-568):
    * Fixed underlying OS vulnerabilities in the Synthetics job manager.

  * [Media agent for Video JS v4.2.0](/docs/release-notes/streaming-browser-release-notes/streaming-browser-agent-videojs-26-06-19):
    * Added explicit opt-in for AWS MediaTailor with full support for custom CDN domains.
    * Added a `config.ad.type` API to choose between CSAI and SSAI ad tracking.
    * Added end-to-end logging through `nrvideo.Log` with active detection mode logged at startup.

  * [React Native agent v1.8.5](/docs/release-notes/mobile-release-notes/react-native-release-notes/react-native-agent-185):
    * Updated the native Android agent to v7.7.6 and the native iOS agent to v7.7.2.

  * [Flutter agent v1.2.7](/docs/release-notes/mobile-release-notes/flutter-release-notes/flutter-agent-127), [Capacitor agent v1.6.7](/docs/release-notes/mobile-release-notes/capacitor-release-notes/capacitor-agent-167), [Unity agent v1.6.1](/docs/release-notes/mobile-release-notes/unity-release-notes/unity-agent-161), [Cordova agent v7.1.6](/docs/release-notes/mobile-release-notes/cordova-release-notes/cordova-agent-716), and [.NET MAUI agent v1.2.4](/docs/release-notes/mobile-release-notes/net-maui-release-notes/net-maui-agent-124):
    * Updated the native Android agent to v7.7.6.
