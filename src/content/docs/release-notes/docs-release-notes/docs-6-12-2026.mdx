### New docs

* Added [Configure a proxy for the eBPF agent](/docs/ebpf/proxy-configuration) to provide guidance for routing eBPF agent telemetry through corporate proxies in restricted-network environments.
* Added [Lambda Extension (Rust) licenses](/docs/licenses/product-or-service-licenses/new-relic-serverless/lambda-extension-rust-licenses) to document third-party licenses used in the New Relic Lambda Extension (Rust).

### Major changes

* Updated [eBPF agent requirements](/docs/ebpf/requirements) with an upper kernel version bound, specifying support for Linux kernel versions v5.8 to v6.14 on both x86-64 and arm64 architectures.
* Updated [eBPF Kubernetes installation](/docs/ebpf/k8s-installation) and [Linux installation](/docs/ebpf/linux-installation) with comprehensive proxy support documentation, clarifying that HTTP CONNECT proxies using the `http://` scheme are supported and that telemetry data remains TLS-encrypted end-to-end.
* Updated [Node.js agent compatibility requirements](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) with a new Hybrid Agent Instrumentations section, documenting Next.js v16+ support via native OpenTelemetry instrumentation starting with agent v14.1.0.
* Updated [SRE Agent setup](/docs/agentic-ai/sre-agent/sre-agent-setup) to remove the Advanced Compute SKU requirement, clarifying prerequisites for setting up the SRE Agent.
* Updated [Synthetics NerdGraph API](/docs/apis/nerdgraph/examples/synthetics-api/overview) reference pages for all monitor types.

### Minor changes

* Updated [AI monitoring compatibility requirements](/docs/ai-monitoring/compatibility-requirements-ai-monitoring) to direct readers to the [Java agent compatibility page](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent/#ai-monitoring) for the full list of supported AI monitoring libraries.
* Updated [Monitor your .NET MAUI application](/docs/mobile-monitoring/new-relic-mobile-maui-dotnet/monitor-your-net-maui-application) with minor formatting corrections.
* Updated [Downgrade or cancel your account](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) with an important callout warning about parent account cancellation cascade effects on sub-accounts.
* Updated [Migrate from API polling to CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/aws-integration-for-metrics/migrate-from-api-polling-to-cwms) to clarify the data validation section heading.
* Added [SRE Agent release notes](/docs/release-notes/sre-agent-release-notes) tile to the release notes index page.
* Added previously missing release history for streaming Video & Ads agents across [browser](/docs/release-notes/streaming-browser-release-notes), [mobile](/docs/release-notes/streaming-mobile-release-notes), and [other platforms](/docs/release-notes/streaming-others-release-notes).
