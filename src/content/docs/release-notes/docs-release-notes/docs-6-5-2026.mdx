### New docs

* Added comprehensive [Elasticsearch OpenTelemetry integration docs](/docs/opentelemetry/integrations/elasticsearch/) covering installation for standalone and Kubernetes environments with NR Dotty, OTel, and Prometheus.
* Added new [OTel Kafka integration docs for Kubernetes self-managed environments](/docs/opentelemetry/integrations/kafka/kubernetes-self-managed/) alongside expanded docs for self-hosted Kafka monitoring with Prometheus.
* Added [Pipeline Control Gateway v2.4.1 release notes](/docs/release-notes/pipeline-control-gateway-release-notes/pipeline-control-gateway-26-05-29/) documenting configuration-only mode (without Flux) and DaemonSet deployment mode.

### Major changes

* Updated [Service Architecture Intelligence (SAI) permissions docs](/docs/service-architecture-intelligence/getting-started/) — Catalogs and Maps no longer require extra roles; updated catalog and maps pages to reflect simplified permissions model.
* Updated [AWS Lambda compatibility and permission requirements](/docs/infrastructure/amazon-integrations/aws-integrations-list/compatibility-requirement-lambda-monitoring/) with significantly expanded permissions documentation (71 line additions).
* Updated [.NET agent Lambda and SQS instrumentation docs](/docs/apm/agents/net-agent/getting-started/net-agent-approaches-lambda/) to correct distributed tracing limitations and .NET Lambda instrumentation behavior.
* Updated [NerdGraph change tracking docs](/docs/change-tracking/config/nerdgraph/) with tutorial videos from the product team; updated the change tracking overview page.
* Added Ruby 3.4 as a supported runtime to the [AWS Lambda layered instrumentation docs](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/).
* Fixed orphaned navigation for Batch 2 docs — restored missing entries in APM, errors inbox, distributed tracing, infrastructure, Kubernetes, and logs nav files.
* Updated [New Relic Lens AWS connector docs](/docs/new-relic-lens/set-up-connectors/) to document the required `nrlens-` prefix for connector naming.
* Updated [user management UI and tasks](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/) to replace example domain names with compliant placeholder values.

### Minor changes

* Updated [.NET agent compatibility requirements](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) with latest Dotty package versions.
* Updated [Node.js agent AI monitoring library support page](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/) redirect to the more up-to-date AIM library support page.
* Updated [GCP Pipeline Control docs](/docs/new-relic-control/pipeline-control/gcp-cci/) with CCI Azure redirects and title fixes.
* Added [FedRAMP page](/docs/security/security-privacy/compliance/certificates-standards-regulations/fedramp/) updates.
* Updated delayed data section in monitoring docs with concrete examples and clearer explanations.
* Minor enhancement to [migrate from API polling to CWMS](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/migrate-from-api-polling-to-cloudwatch-metric-streams/) doc.

### Release notes

Stay up-to-date with our latest releases:

* [Node.js agent v14.1.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-14-1-0/):
    * Added support for native Next.js OTEL instrumentation via the Hybrid Agent.
    * Fixed label parser to handle empty values in an object key.
    * Refactored `@hapi/hapi`, `@hapi/vision`, and `koa` instrumentation to use subscriber-based events.
    * Deprecated `instrumentLoadedModule` API.

* [Python agent v13.1.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-130100/):
    * Added instrumentation for new Redis v8.8.0 commands.
    * Fixed crash in LangGraph instrumentation due to context propagation issue.
    * Improved handling and logging of harvest limit settings.

* [Python agent v13.1.1](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-130101/):
    * Added `request.model` and `response.model` attributes to LangChain Chain LLM events.
    * Fixed TypeError crash in LangGraph streaming instrumentation.

* [.NET agent v10.51.1](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-51-1/):
    * Fixed `RecordDatastoreSegment()` API to correctly use caller-supplied vendor name (note: metric names will change from `Datastore/*/Other/*` to `Datastore/*/<VendorName>/*`).
    * Fixed StackExchange.Redis segments not being added to transactions before harvest.
    * Updated MySql.Data instrumentation to support v9.7.0 signature changes.

* [Browser agent v1.316.0](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.316.0/):
    * Session info is now namespaced per browser app using `licenseKey` and `applicationID`, preventing session sharing across apps on the same origin.

* [Infrastructure agent v1.76.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1761/):
    * Fixed `set-service-account` failure for local users.
    * Security: bumped containerd to v1.7.32 (CVE-2026-46680) and nri-flex to v1.18.4 (CVE-2026-45570).

* [Kubernetes integration v4.2.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-4-2-0/):
    * Included in `newrelic-infrastructure-4.2.0` and `nri-bundle-7.0.14` chart versions.

* [Android agent v7.7.6](/docs/release-notes/mobile-release-notes/android-release-notes/android-776/):
    * Fixed crash in `newrelicTransformClasses` with `IllegalAccessError` after upgrading to AGP 9.1.0.
    * Fixed NullPointerException in session replay textview section.
    * Fixed `onTrimMemory()` call in hybrid applications.

* [Android NDK agent v1.1.4](/docs/release-notes/mobile-release-notes/android-release-notes/android-ndk-114/):
    * Fixed ANRMonitor blocking the main thread on its own lock.

* [Unity agent v1.6.0](/docs/release-notes/mobile-release-notes/unity-release-notes/unity-agent-160/):
    * iOS dependencies can now be resolved via Swift Package Manager.
    * Bundled External Dependency Manager for Unity (EDM4U) upgraded.

* [.NET MAUI agent v1.2.3](/docs/release-notes/mobile-release-notes/net-maui-release-notes/net-maui-agent-123/):
    * Native Android agent updated to v7.7.5.
    * Native iOS agent updated to v7.7.2.

* [Capacitor agent v1.6.6](/docs/release-notes/mobile-release-notes/capacitor-release-notes/capacitor-agent-166/):
    * Updated Native iOS agent to v7.7.2.

* [Synthetics Ping Runtime v1.70.0](/docs/release-notes/synthetics-release-notes/ping-runtime-release-notes/ping-runtime-1.70.0/):
    * Security: fixed CVE-2026-41989 and CVE-2026-34743.

* [Synthetics Node API Runtime rc1.23](/docs/release-notes/synthetics-release-notes/node-api-runtime-release-notes/node-api-runtime-rc1.23/):
    * Security: fixed multiple CVEs including CVE-2026-43284, CVE-2026-43500, CVE-2026-46300, and others.

* [Synthetics Node Browser Runtime rc1.23](/docs/release-notes/synthetics-release-notes/node-browser-runtime-release-notes/node-browser-runtime-rc1.23/):

* [Pipeline Control Gateway v2.4.1](/docs/release-notes/pipeline-control-gateway-release-notes/pipeline-control-gateway-26-05-29/):
    * Added configuration-only mode (without Flux) with namespace-scoped permissions and built-in config-watcher sidecar.
