### Major changes

* Updated [Install New Relic for Go](/docs/apm/agents/go-agent/installation/install-new-relic-go) with comprehensive step-by-step installation guide and examples for common Go application patterns.
* Added [Setup and Install Agent Control](/docs/new-relic-control/agent-control/setup) to provide complete installation and configuration guidance for Kubernetes clusters and Linux hosts.
* Enhanced [setUserId](/docs/browser/new-relic-browser/browser-apis/setuserid) API documentation with session reset functionality and updated session tracking behavior.

### Minor changes

* Added PHP agent compatibility information for [PHP agent compatibility requirements](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements).
* Updated [SPM installation](/docs/mobile-monitoring/new-relic-mobile-ios/installation/spm-installation) documentation with enhanced guidance.
* Enhanced [CocoaPods installation](/docs/mobile-monitoring/new-relic-mobile-ios/installation/cocoapods-installation) documentation with improved instructions.
* Updated [Filter AJAX request events](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events) documentation with wildcard support for deny list.
* Updated [Upgrade to APM experience](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/upgrade-to-apm-experience) for Lambda function monitoring.
