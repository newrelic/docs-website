### Major changes 

* Updated the [Microsoft SQL Server integration](/install/microsoft-sql) documentation to add the Query Performance Monitoring (QPM) for following authentication methods:
  * Windows Domain
  * Windows group Managed Service Account (gMSA)
* Updated the [Elasticsearch monitoring with OpenTelemetry](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) documentation for clarity and added comprehensive Kubernetes installation documentation.
* Updated [Dashboards with NerdGraph API](/docs/apis/nerdgraph/examples/nerdgraph-dashboards) with template variable visibility features.
* Updated [Workflow automation definition schema](/docs/workflow-automation/workflow-automation-apis/definition-schema) with extensive schema updates and comprehensive examples.

### Minor changes

* Updated [Consumption management](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/consumption-management) with CCU usage forecasting features and enhanced dashboard capabilities.
* Updated [View ping monitor results](/docs/synthetics/synthetic-monitoring/using-monitors/view-ping-monitor-results) with improved guidance and examples.
* Updated attribute dictionary with additional attribute definitions and descriptions for Browser.
