### New docs

* Added [Browser consent mode](/docs/browser/new-relic-browser/configuration/consent-mode) to document consent management implementation for browser monitoring compliance with privacy regulations.
* Added [Java extensions in Kubernetes auto-attach](/docs/kubernetes-pixie/kubernetes-integration/java-extensions-k8s-auto-attach) to provide configuration guidance for Java APM instrumentation in Kubernetes environments.
* Added [Bring your own cache configuration](/docs/distributed-tracing/bring-your-own-cache) to document OpenTelemetry cache configuration options for distributed tracing.

### Major changes

* Enhanced [Response Intelligence AI](/docs/alerts/response-intelligence-ai) with AI log summarization capabilities for analyzing log-based alerts.
* Renamed and restructured Compute Management documentation to [Consumption Management](/docs/accounts/accounts-billing/consumption-management).
* Updated [Java agent configuration](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file) with Spring transaction naming options and SQL parser configuration.
* Updated [GCP integrations custom roles](/docs/infrastructure/google-cloud-platform-integrations/get-started/integrations-custom-roles) with revised permissions for 4 Google Cloud services.
* Enhanced [MCP troubleshooting](/docs/agentic-ai/mcp/troubleshoot) documentation with Model Context Protocol setup scenarios and solutions.
* Enhanced [Ruby agent configuration](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration) with new Sidekiq retry error configuration options.
* Updated [PHP agent installation](/docs/apm/agents/php-agent/installation/php-agent-installation-ubuntu-debian) with support for Debian Trixie.
