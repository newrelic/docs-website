### Major changes


* Revamped and added new sections to the [AWS Lambda monitoring](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/introduction-lambda/) documentation for better readability and understanding.
* Added a [Golden metrics](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/#golden-metrics) section and a callout for the change in calculation method in the [OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/) document.
* Added a callout to update the webhook URLs for Microsoft Office 365 webhook-based connectors in Teams in the [destination for alerts](/docs/alerts/get-notified/destinations/) section.
* Added a new section to configure [resource attributes and tags](/docs/infrastructure/other-infrastructure-integrations/monitoring-jenkins-ot/#attributes) in the [Jenkins with OpenTelemetry](/docs/infrastructure/other-infrastructure-integrations/monitoring-jenkins-ot/) document.



### Minor changes


* Fixed a broken link in the [Connect AWS polling integration](/docs/infrastructure/amazon-integrations/connect/set-up-aws-api-polling) document.
* Updated the platform support versions for [New Relic Nozzle](/docs/infrastructure/other-infrastructure-integrations/cloudfoundry-integrations/vmware-tanzu-firehose-nozzle-integration/), [New Relic Service Broker](/docs/infrastructure/other-infrastructure-integrations/cloudfoundry-integrations/vmware-tanzu-service-broker-integration/), and [NGINX integration buildpack](/docs/infrastructure/other-infrastructure-integrations/cloudfoundry-integrations/vmware-tanzu-nginx-buildpack-integration/) for VMware Tanzu Application Service.
* Fixed a broken link for [Amazon CloudWatch metric stream](/install/aws-cloudwatch).
* Fixed the capitalization of the  NRQL queries in the [FOSSA](/docs/vulnerability-management/integrations/fossa/) and [Snyk](/docs/vulnerability-management/integrations/snyk/) integrations for vulnerability management.
* Fixed a broken link in the [Alerts overview](/docs/alerts/overview) page.
