### New docs

* Added [Model Context Protocol (MCP)](/docs/agentic-ai/mcp/overview) documentation to provide comprehensive guidance for integrating New Relic with AI agents, including setup instructions, tool reference, and troubleshooting.
* Added [Workflow automation](/docs/workflow-automation/introduction-to-workflow) documentation to provide comprehensive guidance for automating workflows, including templates, actions catalog, destinations, and API references.
* Added [Schedule NRQL searches](/docs/nrql/using-nrql/schedule-nrql-searches) to provide guidance for scheduling log searches and NRQL queries.
* Added [ISO 42001](/docs/security/security-privacy/compliance/certificates-standards-regulations/iso-42001) certification documentation.
* Added [SOC1](/docs/security/security-privacy/compliance/certificates-standards-regulations/soc1) compliance documentation.

* Updated [Transaction workloads](/docs/apm/transactions/workload-performance-monitoring/transaction-workloads) with extensive revisions for database and serverless metrics in Transaction 360.
