### New docs

* Added a dedicated troubleshooting page for [Windows MSI installer fails with error 1603](/docs/infrastructure/infrastructure-troubleshooting/troubleshoot-infrastructure/windows-msi-install-fails-error-1603), so you can resolve infrastructure agent installation failures on Windows.

### Major changes

* Restructured the OpenTelemetry [Microsoft SQL Server monitoring](/docs/opentelemetry/database/mssql/introduction) docs into a dedicated multi-page section, from [guided setup](/docs/opentelemetry/database/mssql/guided) through host and Amazon RDS instructions, metrics, compatibility, and troubleshooting.
* Updated [ECS on EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) and [ECS on Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) monitoring with a NRDOT Collector option alongside the OpenTelemetry Collector, so you can run the sidecar with New Relic's supported distribution.

### Minor changes

* Updated the OpenTelemetry [Oracle Database](/docs/opentelemetry/database/otel-oracledb) and [database service identification](/docs/opentelemetry/database/database-service-identification) docs with corrected parameters.
* Documented how to [export dashboards and charts through the NerdGraph API](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data/#export-api) as PDF or PNG, with custom dimensions, template variable overrides, and entity filters, and updated the UI export steps to cover PNG.
* Documented how to [query specific knowledge connectors](/docs/agentic-ai/knowledge-integration/overview/#query-specific-connectors) in New Relic AI chat, using `/{connector-name}` to target a single source or a broad search across all connectors.
* Added the gauge chart type to the [chart types](/docs/query-your-data/explore-query-data/use-charts/chart-types) documentation.
* Updated the [Node.js agent compatibility requirements](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) with the latest compatibility report.
* Updated the [.NET agent compatibility requirements](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements) with the latest supported library versions.
* Updated the [Ruby agent hybrid support](/docs/apm/agents/manage-apm-agents/opentelemetry-api-support) details, including raising the minimum supported Ruby agent version to 10.6.0, and the [AWS integrations](/docs/infrastructure/amazon-integrations/get-started/introduction-aws-integrations) setup instructions.
* Clarified in the [Data ingestion UI](/docs/data-apis/manage-data/manage-data-coming-new-relic) docs that the Auto view doesn't support the time picker and shows data in 5-minute intervals for up to one day, so use the Daily, Weekly, or Monthly views for longer ranges.
* Removed the overlay APIs from the [SDK navigation](/docs/new-relic-solutions/build-nr-ui/sdk-component/platform-apis/navigation) documentation.

### What's new posts

* Added [Enhanced dashboard exports and a new chart image snapshot mutation in NerdGraph](/whats-new/2026/06/whats-new-0626-dashboards) to announce two new NerdGraph mutations for generating PDF snapshots of full dashboard pages and standalone chart images.

### Release notes

* Stay up-to-date with our latest releases:

  * [NRDOT v1.19.0](/docs/release-notes/nrdot-release-notes/nrdot-2026-06-29):
    * Added the `nroracledbreceiver`, `nrsqlserverreceiver`, `secretsmanagerprovider`, and `aesprovider` components for database monitoring.

  * [Infrastructure agent v1.77.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1771):
    * Updated bundled dependencies, including `golang.org/x/net`, the Alpine base image (v3.24), and containerd (v1.7.33).

  * [PHP agent v12.8.0.37](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-12-8-0-37):
    * Added Redis Cluster instrumentation.
    * Upgraded golang to 1.26.4 and updated outdated dependencies.
    * Fixed a realpath buffer overrun in install-util and a stack overflow.

  * [Android agent v7.7.7](/docs/release-notes/mobile-release-notes/android-release-notes/android-777):
    * Moved Session Replay image compression off the main thread to reduce capture overhead.
    * Fixed stale offline payloads by enforcing a time-to-live that discards data queued too long.
    * Ensured buffered events are sent in the final harvest during app shutdown, and fixed session attributes that could be lost during the shutdown harvest.

  * [iOS agent v7.7.3](/docs/release-notes/mobile-release-notes/ios-release-notes/ios-agent-773):
    * Added offline storage functionality for Mobile Session Replay.
    * Changed the default session replay masking mode to disallow programmatic unmasking.
    * Fixed an infinite loop in EventDeserializer that froze the agent on launch.

  * [Kubernetes integration v4.3.6](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-4-3-6):
    * Released as part of the newrelic-infrastructure-4.3.6 and nri-bundle-8.0.0 chart versions.

  * [Mobile app for Android v5.43.2](/docs/release-notes/mobile-apps-release-notes/new-relic-android-release-notes/new-relic-android-5432):
    * Bug fixes and UI improvements.

  * [Mobile app for iOS v6.9.30](/docs/release-notes/mobile-apps-release-notes/new-relic-ios-release-notes/new-relic-ios-6110):
    * Added an in-app log detail view to inspect full log attributes, filter by attribute, and copy values.
    * Added configurable in-app surveys for gathering user feedback.
    * Improved login reliability along with performance and stability.

  * [React Native agent v1.8.6](/docs/release-notes/mobile-release-notes/react-native-release-notes/react-native-agent-186), [Flutter agent v1.2.8](/docs/release-notes/mobile-release-notes/flutter-release-notes/flutter-agent-128), [Capacitor agent v1.6.8](/docs/release-notes/mobile-release-notes/capacitor-release-notes/capacitor-agent-168), [Unity agent v1.6.2](/docs/release-notes/mobile-release-notes/unity-release-notes/unity-agent-162), [Cordova agent v7.1.7](/docs/release-notes/mobile-release-notes/cordova-release-notes/cordova-agent-717), and [.NET MAUI agent v1.2.5](/docs/release-notes/mobile-release-notes/net-maui-release-notes/net-maui-agent-125):
    * Updated the native Android agent to v7.7.7 and the native iOS agent to v7.7.3.

  * [Media agent for HTML5 v4.1.1](/docs/release-notes/streaming-browser-release-notes/streaming-browser-agent-html5-26-07-02):
    * Added custom-attribute validation and an enforced per-event limit on the number of custom attributes.

  * Synthetics runtimes — [Job Manager v574](/docs/release-notes/synthetics-release-notes/job-manager-release-notes/job-manager-release-574), [Node API runtime rc1.27](/docs/release-notes/synthetics-release-notes/node-api-runtime-release-notes/node-api-runtime-rc1.27), [Node browser runtime rc1.27](/docs/release-notes/synthetics-release-notes/node-browser-runtime-release-notes/node-browser-runtime-rc1.27), and [Ping runtime v1.73.0](/docs/release-notes/synthetics-release-notes/ping-runtime-release-notes/ping-runtime-1.73.0):
    * Patched underlying OS and dependency CVEs across all synthetics images.
    * For the Node browser runtime, removed performance logs, HAR files, and screenshots from the container logs.
