### New docs

* Added [Monitor GenAI with OpenTelemetry](/docs/ai-monitoring/opentelemetry-ai-monitoring) to provide step-by-step guidance for capturing GenAI traces and viewing them in AI monitoring.
* Added [Windows MSI installer fails with error 1603](/docs/infrastructure/infrastructure-troubleshooting/troubleshoot-infrastructure/windows-msi-install-fails-error-1603) as a dedicated troubleshooting page so you can resolve infrastructure agent installation failures on Windows.

### Major changes

* Restructured the OpenTelemetry [Microsoft SQL Server monitoring](/docs/opentelemetry/database/mssql/introduction) documentation into a dedicated multi-page section, with separate [guided setup](/docs/opentelemetry/database/mssql/guided), [Linux](/docs/opentelemetry/database/mssql/linux-hosted) and [Windows](/docs/opentelemetry/database/mssql/windows-hosted) host instructions, [Amazon RDS](/docs/opentelemetry/database/mssql/linux-rds) guidance, a [metrics reference](/docs/opentelemetry/database/mssql/metrics-reference), [compatibility](/docs/opentelemetry/database/mssql/compatibility), and [troubleshooting](/docs/opentelemetry/database/mssql/troubleshooting).
* Updated [ECS on EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) and [ECS on Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) monitoring with a NRDOT Collector option alongside the OpenTelemetry Collector, so you can run the sidecar with New Relic's supported distribution.

### Release notes

* Stay up-to-date with our latest releases:

  * [NRDOT v1.19.0](/docs/release-notes/nrdot-release-notes/nrdot-2026-06-29):
    * Added the `nroracledbreceiver`, `nrsqlserverreceiver`, `secretsmanagerprovider`, and `aesprovider` components for database monitoring.

  * [PHP agent v12.8.0.37](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-12-8-0-37):
    * Added Redis Cluster instrumentation.
    * Upgraded golang to 1.26.4 and updated outdated dependencies.
    * Fixed a realpath buffer overrun in install-util and a stack overflow.

  * [Android agent v7.7.7](/docs/release-notes/mobile-release-notes/android-release-notes/android-777):
    * Moved Session Replay image compression off the main thread to reduce capture overhead.
    * Fixed stale offline payloads by enforcing a time-to-live that discards data queued too long.
    * Ensured buffered events and session attributes are sent in the final harvest during app shutdown.

  * [iOS agent v7.7.3](/docs/release-notes/mobile-release-notes/ios-release-notes/ios-agent-773):
    * Added offline storage functionality for Mobile Session Replay.
    * Changed the default session replay masking mode to disallow programmatic unmasking.
    * Fixed an infinite loop in EventDeserializer that froze the agent on launch.

  * [Kubernetes integration v4.3.6](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-4-3-6):
    * Released as part of the newrelic-infrastructure-4.3.6 and nri-bundle-8.0.0 chart versions.

  * [Mobile app for iOS v6.9.30](/docs/release-notes/mobile-apps-release-notes/new-relic-ios-release-notes/new-relic-ios-6110):
    * Added an in-app log detail view to inspect full log attributes, filter by attribute, and copy values.
    * Added configurable in-app surveys for gathering user feedback.
    * Improved login reliability along with performance and stability.

  * [React Native agent v1.8.6](/docs/release-notes/mobile-release-notes/react-native-release-notes/react-native-agent-186), [Flutter agent v1.2.8](/docs/release-notes/mobile-release-notes/flutter-release-notes/flutter-agent-128), [Capacitor agent v1.6.8](/docs/release-notes/mobile-release-notes/capacitor-release-notes/capacitor-agent-168), [Unity agent v1.6.2](/docs/release-notes/mobile-release-notes/unity-release-notes/unity-agent-162), [Cordova agent v7.1.7](/docs/release-notes/mobile-release-notes/cordova-release-notes/cordova-agent-717), and [.NET MAUI agent v1.2.5](/docs/release-notes/mobile-release-notes/net-maui-release-notes/net-maui-agent-125):
    * Updated the native Android agent to v7.7.7 and the native iOS agent to v7.7.3.
