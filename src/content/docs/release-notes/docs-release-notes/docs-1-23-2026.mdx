### Major changes

* Updated the eBPF documentation with [new support matrix](/docs/ebpf/requirements), [eBPF agent Linux installation](/docs/ebpf/linux-installation) and [Kubernetes installation](/docs/ebpf/k8s-installation) with configuration changes for client removal.
* Updated [fine-grained access control documentation for secrets management](/docs/apis/nerdgraph/examples/nerdgraph-api-secret-management-service) to improve API access controls procedure.
* Updated [Ruby agent requirements and supported frameworks](/docs/apm/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) with 10.x version changes.
* Added [Nuxt.js integration documentation](/docs/browser/browser-integrations/nuxt-integration/) with updated Nuxt 3+ support and improved formatting.


### Minor changes

* Enhanced [Python agent notice_error API](/docs/apm/agents/python-agent/python-agent-api/noticeerror-python-agent-api) documentation with exception instance support.
* Updated [browser logs getting started guide](/docs/browser/browser-monitoring/browser-pro-features/browser-logs/get-started/) with logging documentation improvements.
* Fixed session trace documentation for [webpage life cycle exploration](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle).
* Updated [Node.js agent compatibility requirements](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) with latest compatibility report.
* Updated [NGINX OpenTelemetry configuration](/docs/opentelemetry/nginx/nginx-otel-host) with improved stub_status configuration.

### What's new

* [Dashboard operations for read-only users](/whats-new/2026/01/whats-new-01-19-read-only-dashboards) - Standard read-only role permissions for dashboard management will be limited to view-only access effective April 30, 2026.

* Stay up-to-date with our latest releases:
