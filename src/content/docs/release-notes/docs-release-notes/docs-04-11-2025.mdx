### New Document

- Added a new [document to monitor Lambda response monitoring](/docs/serverless-function-monitoring/aws-lambda-monitoring/response-stream) functions in Node.js applications to gain real-time insights into the performance and behavior of each invocation.

### Major Updates

- Extended the deadline to May 14, 2025, for updating the Synthetic IP ranges of your [Synthetic monitors](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/).
- Added new IP ranges for Synthetic monitoring in [Synthetic monitoring](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/).
- Updated the [Query account document](/docs/accounts/accounts/account-maintenance/query-account-audit-logs-nrauditevent) to include queries for logged users.
- Updated the feature flags for [Android monitoring](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings).
- Updated the [Anomaly detection document](/docs/alerts/create-alert/set-thresholds/anomaly-detection) to include information on a new field for anomaly NRQL alert conditions, allowing users to configure seasonality.

### Minor Updates

- Updated the [email account settings document](/docs/accounts/accounts/account-maintenance/account-email-settings/#change-email) to include information on special character support.
- Updated the [Agent Control document](/docs/new-relic-control/agent-control/overview/) to clarify that multiple installations of Agent Control on the same cluster are not supported.
- Updated the [Node.js agent configuration document](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration) to include batching and compression settings for infinite tracing.
- Updated the [use TCP endpoint document](/docs/logs/log-api/use-tcp-endpoint-forward-logs-new-relic) to include information on finding logs by searching for the attribute `plugin.type:"syslog-newrelic"`.
- Added a new access and data management section in the [Agentic AI document](/docs/agentic-ai/agentic-integration/introduction-agentic-ai).
- Added limitations of log APIs in the [use TCP endpoint document](/docs/logs/log-api/use-tcp-endpoint-forward-logs-new-relic).
- Updated the [Kubernetes document](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-apm-applications-kubernetes) to highlight the importance of restarting pods after deployment.
- Updated the environmental variables for [Azure functions](/docs/serverless-function-monitoring/azure-function-monitoring/install-serverless-azure-monitoring).
- Updated the [New Relic OTLP attribute processing document](/docs/opentelemetry/best-practices/opentelemetry-otlp-attributes-processing-update) for added clarity.
