### New docs

* Added [OpenTelemetry Collector Observability](/docs/opentelemetry/collector-observability/collector-observability) to provide comprehensive monitoring guidance for collector performance and metrics.
* Added [Intelligent Workloads](/docs/new-relic-solutions/new-relic-one/workloads/workloads-overview) documentation suite to provide guidance for AI-powered workload creation and KPI management, including [creating intelligent workloads](/docs/new-relic-solutions/new-relic-one/workloads/create-intelligent-workload), [AI insights](/docs/new-relic-solutions/new-relic-one/workloads/advanced-workload-ai-insights), [KPI management](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-kpis), and [UI features](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-ui).
* Added [OpenTelemetry API Support](/docs/apm/agents/manage-apm-agents/opentelemetry-api-support) to provide comprehensive guidance for vendor-neutral instrumentation across all New Relic APM agents.

### Major changes

* [Workflow Automation](/docs/workflow-automation/introduction-to-workflow) is now generally available.
* [Outlier Detection](/docs/alerts/create-alert/set-thresholds/outlier-detection) is now generally available.
* Added comprehensive [Windows support documentation](/docs/new-relic-control/agent-control/overview) for Agent Control with setup, configuration, monitoring, and troubleshooting guidance.

* Enhanced [OpenTelemetry Kafka monitoring](/docs/opentelemetry/integrations/kafka/self-hosted) documentation with architecture diagrams and expanded troubleshooting guidance.

### Release notes

* Stay up-to-date with our latest releases:

  * [Ruby agent v10.2.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-10-2-0):
    * Introduced Hybrid Agent for OpenTelemetry Tracing support with vendor-neutral API compatibility.
    * Added error collector backtrace truncation configuration and Logging gem instrumentation.
    * Enhanced cloud vendor detection with parallel processing option.

  * [Java agent v9.1.0](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-910):
    * Added comprehensive Hybrid Agent implementation with OpenTelemetry API compatibility for tracing, metrics, and logs.
    * Enhanced Kotlin Coroutines support and fixed W3C trace propagation issues.
    * Added agent metadata actions and multihost configuration fixes.

  * [Node.js agent v13.14.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-14-0):
    * Added automatic log ingestion from Azure Functions Apps.
    * Implemented error causes support and profiler delay/duration controls.
    * Updated Google Gen AI integration for `embedContentInternal` method compatibility.

  * [Android agent v7.7.0](/docs/release-notes/mobile-release-notes/android-release-notes/android-770):
    * Added support for Android Gradle Plugin 9.
    * Enhanced mobile session replay with error sampling and new `pauseReplay`/`recordReplay` APIs.
    * Implemented multiple performance improvements for session replay functionality.

  * [Infrastructure agent v1.72.6](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1726):
    * Updated fluent-bit version to 4.2.2 for Windows environments.

  * [Browser agent v1.310.1](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.310.1):
    * Enhanced LCP entry with fallback attribution values for better Core Web Vitals reporting.
