### Release notes

* Stay up-to-date with our latest releases:

  * [Python agent v12.0.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-120000):

    * Added Free-Threaded Python support for Python 3.13+ with GIL disabled.
