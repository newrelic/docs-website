### Major changes

* Reviewed and rewrote on-host integration docs for [RabbitMQ](/install/rabbitmq/), [JMX](/install/jmx/), [Apache Flink](/install/apache-flink/), and [HAProxy](/install/haproxy/) to make it easier to install and ensure the docs are accurate and up to date.
* For [AWS Lambda Serverless monitoring](/docs/serverless-function-monitoring/aws-lambda-monitoring/get-started/compatibility-requirements-aws-lambda-monitoring/), updated information about supported runtime versions, dependencies, and the deprecated .NET NuGet package SDK.
* Updated our [FedRAMP](/docs/security/security-privacy/compliance/certificates-standards-regulations/fedramp/), [HIPAA](/docs/security/security-privacy/compliance/certificates-standards-regulations/hipaa/), [ISO-27001](/docs/security/security-privacy/compliance/certificates-standards-regulations/iso-27001/), [SOC2](/docs/security/security-privacy/compliance/certificates-standards-regulations/soc2/), [TISAX](/docs/security/security-privacy/compliance/certificates-standards-regulations/tisax/), [data encryption](/docs/security/security-privacy/compliance/data-encryption/), and [HITRUST CSF](/docs/security/security-privacy/compliance/hitrust/) compliance docs to reflect New Relic's 2024 compliance status.
* Continued overhaul and reorganization of our [NRQL docs](/docs/nrql/get-started/introduction-nrql-new-relics-query-language) to improve readability and quality.
* Clarified that Android [mobile monitoring offline storage](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings/#android-offline-storage) is disabled by default and updated the [offline storage configuration](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/set-max-offline-storage/#android) doc.
* Added [Unity mobile monitoring settings](https://docs.newrelic.com/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings/#unity) info.
* Added synthetics monitoring endpoint for [AWS PrivateLink](/docs/data-apis/custom-data/aws-privatelink).
* Added troubleshooting section to [Statsd integration](/docs/more-integrations/open-source-telemetry-integrations/statsd/statsd-monitoring-integration/#troubleshooting) doc.
* Added new [rules element instrumentation](/docs/apm/agents/net-agent/configuration/net-agent-configuration/#instrumentation-rules) .NET config option section.
* Updated synthetic monitoring [job manager transition version requirements](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide).

### Smaller updates

* Added synthetic monitoring [Kubernetes `vsePassphrase`](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-configuration/#kubernetes-env-config) environment variables.
* Fixed some broken internal links.
* Updated [errors inbox error status](/docs/errors-inbox/errors-inbox) section with a new image and more detail about the UI.
* Updated [synthetic monitoring runtime module](/docs/synthetics/synthetic-monitoring/using-monitors/manage-monitor-runtimes) versions.
* Added information about how to use [multiple log sources with the Fluent Bit plugin](/docs/logs/forward-logs/fluent-bit-plugin-log-forwarding).
* Added how to [query logs live archives via the NerdGraph API](/docs/logs/get-started/live-archives).
* Updated and fixed various minor style and image issues.
* Added NerdGraph applied intelligence [cursor pagination example](/docs/apis/nerdgraph/examples/nerdgraph-api-notifications-channels/#paginate).
* Fixed broken [NVIDIA GPU integration](/docs/infrastructure/host-integrations/host-integrations-list/nvidia-gpu-integration) configuration.
* Added section about Android mobile monitoring [mapping upload host](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/android-agent-crash-reporting) configuration.
* Added configuration details for the [Temporal monitoring integration](/docs/infrastructure/host-integrations/host-integrations-list/temporal-monitoring-integration) for Java.
* Added missing permission for [GCP Cloud Functions](/docs/infrastructure/google-cloud-platform-integrations/get-started/integrations-custom-roles).
* Updated docs link related to [reporting security vulnerabilities](/docs/security/security-privacy/information-security/security-bulletins/#report).
* Updated security doc to remove mention of HackerOne, which we no longer use for our [coordinated disclosure process](/docs/security/security-privacy/information-security/software-development-lifecycle).
* Added information about the legal implications of using [browser monitoring proxy methods](/docs/browser/new-relic-browser/configuration/proxy-agent-requests).
* Clarified that the [`coreCount`](https://docs.newrelic.com/attribute-dictionary/?event=SystemSample&attribute=coreCount) attribute for the `SystemSample` event on Windows counts the number of processors for the current instance of the processor.
* Updated [network performance monitoring configuration](/docs/network-performance-monitoring/advanced/advanced-config) docs to describe changes related to Meraki API and network device syslogs.
* Removed [Prometheus OpenMetrics integration in Kubernetes](/docs/infrastructure/prometheus-integrations/install-configure-prometheus-agent/migration-guide) EOL date.
* In the left nav, renamed Serverless legacy manual instrumentation to [layerless manual instrumentation](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-layerless/).
* Assorted minor copyedits for clarity and improved consistency with our style guide.
