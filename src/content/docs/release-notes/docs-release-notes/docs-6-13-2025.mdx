### New docs
* Added [Change Tracking Events](/docs/change-tracking/change-tracking-events/) to track changes across your applications and infrastructure using New Relic's Change Tracking capability. Capture, visualize, and analyze changes to understand their impact on your systems.
* Added the [procedure](/docs/infrastructure/prometheus-integrations/migrations/v2-upgrade/) to upgrade the Prometheus agent to version 2.
* Added a [document](/docs/query-your-data/explore-query-data/dashboards/nested-variables/) that explains what is nested variables and how to use them on dashboards.

### Major changes

* Added [Application hangs on startup](/docs/apm/agents/net-agent/troubleshooting/app-hangs-on-startup/) troubleshooting guide for .NET agent.
* Added examples of [Run on-host integrations manually (dry-run)](/docs/infrastructure/host-integrations/troubleshooting/run-integrations-manually/) about Windows - Microsoft SQL Server (from an Administrator powershell).
* Added [Measure API for Browser API](/docs/browser/new-relic-browser/browser-apis/measure/).
* Added [Java agent config file template](/docs/apm/agents/java-agent/configuration/java-agent-config-file-template/) for 8.21.0 Java agent release.
* Updated user types required for sharing dashboards and charts in the [security and access document](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/).
* New Relic AI agent integrations for Github Copilot and Gemini are now in general availability. Updated the documents as follows:
    * [New Relic AI agent integration with GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration/)
    * [New Relic AI agent integration with Gemini](/docs/agentic-ai/agentic-integration/gemini-integration/)
* Updated the [View Session Replays in New Relic](/docs/browser/browser-monitoring/browser-pro-features/session-replay/view-session-replays/) document to reflect server-side configuration changes for session replay settings in browser agent version 1.259.0+.
* Added information to [Data Privacy with New Relic](https://docs.newrelic.com/docs/security/security-privacy/data-privacy/data-privacy-new-relic/#account-security) and [Security policy](/docs/licenses/license-information/referenced-policies/security-policy/) documents about periodic audits and deactivation of accounts for terminated employees.
* Added the updated configuration to the document that explains how to [install media video agent for browser application](/install/browser-media/).
