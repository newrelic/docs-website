## New docs

* Added [Grafana dashboard translation API](/docs/infrastructure/prometheus-integrations/view-query-data/grafana) functionality to support dashboard migration workflows.
* Released **New Relic Japan data region** and updated various documentation pages to reflect the new region and its capabilities across all products.

### Major changes

* **Rebranded SRE Agent to New Relic Autopilot** across all documentation with updated terminology and references throughout the platform.
* Updated [Vulnerability management](/docs/vulnerability-management/advanced/query-examples) documentation to remove invalid NRQL functions and fix query patterns for better accuracy.
* **Removed vulnerability management add-on documentation** as the feature has been integrated into core platform capabilities.
* Updated [Scorecards templates library](/docs/service-architecture-intelligence/scorecards/templates) with comprehensive coverage of all available templates and improved organization structure.
* Updated the Node.js [compatibility matrix](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) to include the latest supported versions and clarify compatibility with various Node.js releases.
* Updated [New Relic MCP server](/docs/agentic-ai/mcp/overview) document to make it compliant with HIPAA compliance.

### Minor changes


* Added [scale an AKS agent pool](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-aks-scalepool) actions to manage your AKS clusters directly from your workflows.
* Updated [Terraform integration setup](/docs/infrastructure/amazon-integrations/aws-integration-for-metrics/terraform).
* Enhanced [Microsoft SQL Server integration](/docs/opentelemetry/database/mssql/introduction) with updated configuration links and improved setup guidance.
* Fixed [HAProxy integration endpoint](/docs/infrastructure/host-integrations/host-integrations-list/haproxy-monitoring-integration) documentation for improved accuracy and clarity.
* Added comprehensive [custom certificate support for private locations](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-configuration/#custom-ca-certificates) with detailed setup instructions and configuration options for secure monitoring.
* Updated [student program documentation](/docs/accounts/accounts-billing/account-setup/student-program) with program details.
