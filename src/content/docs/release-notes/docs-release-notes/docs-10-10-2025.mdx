### New docs

* Added [Collect marks and measures with the browser agent](/whats-new/2025/09/whats-new-10-02-browser-marks-and-measures) to provide guidance on capturing performance marks and measures for monitoring task durations in web applications.
* Added [No data with Kubernetes APM auto-attach](/docs/apm/agents/ruby-agent/troubleshooting/no-data-kubernetes-auto-attach) to provide troubleshooting guidance for Ruby applications using Kubernetes APM auto-attach with non-Rails frameworks.

### Major changes

* Updated [Attribute dictionary](/attribute-dictionary) with deprecation notices for `BrowserTiming` events and enhanced browser agent functionality descriptions for improved technical reference accuracy.
* Enhanced [Pipeline Cloud Rules API](/docs/new-relic-control/pipeline-control/cloud-rules-api) with comprehensive guidance on enabling, disabling, and re-enabling cloud rules for improved operational control.

### Minor changes

* Updated pricing terminology in [Live archives billing](/docs/logs/get-started/live-archives-billing) from "Compute Capacity Units" to "Advanced Compute Capacity Units" for improved accuracy.
* Improved organization of [Agent Control setup](/docs/new-relic-control/agent-control/setup) instructions by relocating tip callout and removing duplicate content for streamlined installation guidance.
* Updated workflow automation documentation with latest changes to [actions catalog](/docs/workflow-automation/setup-and-configuration/actions-catalog), [scheduling APIs](/docs/workflow-automation/create-a-workflow-automation/start-schedule), and template configurations for improved platform synchronization.
* Enhanced [New Relic Control announcement](/whats-new/2025/09/whats-new-30-09-new-relic-control) with refined messaging and clearer feature descriptions for the Agent Fleet Control GA release.

### Release notes

* Stay up-to-date with our latest releases:

  * [Ruby agent v9.22.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-9-22-0):
    * Added one-step instrumentation for Kubernetes (now generally available).
    * Added deprecation notices for Ruby 2.4 and 2.5 support.
    * Added deprecation notice for the `newrelic deployments` command.
    * Ensured compatibility with Ruby 3.5 changes to Method#source_location.

  * [.NET agent v10.45.0](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-45-0):
    * Added new Distributed Tracing sampler, the TraceId Ratio-Based Sampler.
    * Fixed MSSQL connection string parsing exceptions that could disable Datastore instrumentation.
    * Resolved issues with Kafka "Consume" instrumentation compatibility.

  * [Python agent v11.0.1](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-110001):
    * Prevented early activation in Django instrumentation.
    * Fixed server configuration to local configuration sync issues.
    * Added guards against crashes from trace_cache.pop().

  * [Infrastructure agent v1.69.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1690):
    * Upgraded AWS SDK from V1 to V2.
    * Upgraded Docker API library from v26 to v28.
    * Fixed Windows CPU handling for instances with multiple CPU groups.

  * [iOS agent v7.5.11](/docs/release-notes/mobile-release-notes/ios-release-notes/ios-agent-7511):
    * Fixed performance issues that could occur when using Interaction Tracing.
    * Fixed a log related to Distributed Tracing that could be incorrectly shown.

  * [Node browser runtime v3.0.55](/docs/release-notes/synthetics-release-notes/node-browser-runtime-release-notes/node-browser-runtime-3.0.55):
    * Fixed underlying Ubuntu vulnerabilities addressing multiple security CVEs.
