### New docs

* Added [PHP agent support for Laravel](/docs/apm/agents/php-agent/frameworks-libraries/laravel) to provide comprehensive guidance for Laravel web transactions, Laravel Queue Workers, and Laravel Horizon Workers.
* Added [End of Life announcement for Pre-insights APM legacy embedded charts](/eol/2025/11/pre-insights/) to notify users about the deprecation of legacy APM embedded charts.
* Added [PHPUnit runs out of memory](/docs/apm/agents/php-agent/troubleshooting/phpunit-incompatibility) troubleshooting guide for PHPUnit 11+ compatibility issues with the New Relic PHP agent.

### Major changes

* Updated [User management concepts](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts) and related RBAC documentation with new grant and role creation experience introducing three role types (Organization-based roles, Account-based roles, and Entity-based roles) consolidated under the <DNT>**Roles**</DNT> tab.
* Updated [Alert channels](/docs/apis/nerdgraph/examples/nerdgraph-api-notifications-channels) with Microsoft Teams integration API methods and examples.
* Updated [Alert destinations](/docs/apis/nerdgraph/examples/nerdgraph-api-notifications-destinations) with Microsoft Teams integration support.

### Minor changes

* Updated [New Relic One pricing and billing](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing) to bring clarity to pricing models and capabilities offered with it.
* Updated [.NET agent configuration](/docs/apm/agents/net-agent/configuration/net-agent-configuration) with browser instrumentation configuration options.
* Updated [Browser monitoring with .NET agent](/docs/apm/agents/net-agent/other-features/browser-monitoring-net-agent) with simplified configuration guidance.
* Updated [PHP agent compatibility requirements](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements) with Laravel Horizon support and removed Windows 32-bit support.
* Updated [Java agent compatibility requirements](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent) with Spring Batch compatibility range capped at v6.0.0.
* Updated [eBPF Kubernetes installation](/docs/ebpf/k8s-installation) with OpenTelemetry configuration changes.
* Updated [OpenTelemetry collector processing](/docs/opentelemetry/get-started/collector-processing/opentelemetry-collector-processing-intro) with updated configuration guidance.
* Updated [Transaction 360](/docs/apm/transactions/workload-performance-monitoring/introduction) with corrected introduction file reference.
* Updated [Networks configuration](/docs/new-relic-solutions/get-started/networks) with fixed dualstack endpoint permalink.
* Updated [FedRAMP compliance](/docs/security/security-privacy/compliance/certificates-standards-regulations/fedramp) with updated compliance information.
* Updated [TISAX compliance](/docs/security/security-privacy/compliance/certificates-standards-regulations/tisax) with updated compliance information.
