### Manticore HTTP client with connection reuse

The Logstash Output Plugin now uses the Manticore HTTP client, created during `register` and reused across calls, improving performance and resource efficiency. This is the production-ready release following the earlier beta.

### Changed

* **Manticore HTTP client:** Switched from the previous HTTP client to Manticore with connection reuse across calls for better throughput and reliability.
* **Fixed gzip compression:** Updated the gzip method to resolve payload unmarshalling issues in New Relic.
* **Configurable timeouts:** Added new optional parameters to configure connection timeout and socket timeout, with sensible default values.
* **Improved testing:** Updated unit and integration tests to validate log delivery to New Relic using the live API.
