### Manticore HTTP client with connection reuse

The Logstash Output Plugin now uses the Manticore HTTP client and reuses connections across calls, improving performance and resource efficiency. This is the production-ready release following the earlier beta.

### Changed

* **Manticore HTTP client:** Switched from the previous HTTP client to Manticore with connection reuse across calls for better throughput and reliability.
* **Configurable timeouts:** Added new optional parameters to configure connection timeout and socket timeout, with sensible default values.
* **Reduced logging noise:** Changed several info-level log statements to debug level to reduce noise in production environments.
