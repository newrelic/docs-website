With this release, [core users](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type/#user-type-capabilities) can manage users and manage billing. They can:

* Perform user management tasks if they have the [**Authentication domain manager** standard role](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts#roles).
* Perform billing administrative tasks if they have the [**Billing user** standard role](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts#roles).
