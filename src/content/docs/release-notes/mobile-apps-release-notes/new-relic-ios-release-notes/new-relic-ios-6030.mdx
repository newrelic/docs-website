New SwiftUI views for traces, errors and occurrences. Also added a log table visualization in dashboards.

### New features

* New Log table visualization that can be selected in the query editor, or displayed in dashboards.
* New SwiftUI views for APM transaction trace, error traces, slow query trace.
* New SwiftUI views for Browser js error traces.
* New SwiftUI views for Mobile crash, handled exception and http error occurrences.
* New SwiftUI views for Infrastructure.
