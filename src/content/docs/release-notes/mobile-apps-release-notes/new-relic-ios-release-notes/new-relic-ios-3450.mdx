### Notes

This version contains enhanced notifications, with support for selecting unique sounds for different types of notifications, along with showing a chart image for alert events.

### New features

* Select different notification sounds for push notifications to the app.
* Shows alert event open / closed chart image in the notification.

### Bug fixes

* Fix for filtering apmApplicationIds and apmApplicationNames properly in Infrastructure.
* Fixed a crash when trying to search through plugin components.
* Fixed a crash in Radar.
