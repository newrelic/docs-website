### Notes

Introducing Dashboards on iPad. Now view all your dashboards right from the New Relic mobile app.

* iPad now has support for Dashboards
* iPad now has an improved Home page with favorites and support for searching across services, apps, hosts and monitors
* Home on iPhone now shows monitors

### Improvements

* Improved cards, and searching on iPad
* Improved network performance for large lists
* Improved network calls for filtering by tags (what used to be labels & rollups) in APM and synthetics
