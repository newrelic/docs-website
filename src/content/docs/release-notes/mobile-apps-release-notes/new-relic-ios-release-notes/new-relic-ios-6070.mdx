This release includes Errors Inbox for supported entity types.

### New features

* Adds in support for viewing an Errors Inbox of a workload or selected entities, for supported entity types. These include APM services, Open telemetry services, Browser applications, Mobile applications, and Lambda functions
* Adds in the ability to have an Ask AI widget added to your home or lock screens. Tapping on this widget will open the app direct to the Ask AI screen
* Fixes issue where some dashboard visualizations would not display properly
* Updates home screen to improve tapping on favorites
