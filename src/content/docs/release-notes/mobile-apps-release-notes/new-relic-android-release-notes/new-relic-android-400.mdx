### Notes

* This release adds support for closing an open incident for authorized users and also adds more deep link support for links shared from from New Relic One.
* This release also includes a feature flag check for plugins that will not display plugins as a menu item unless the account has a feature flag enabling plugins.

### Fixes

* Broken screens in dark mode.
* Dropped notifications.
* Dashboard filtering.
