You can now access errors inbox for Android. 

### Notes

* Support for new errors inbox screen for Android allows you to:

    * View grouped errors for entities and workloads
    * Itererate through error occurrences
    * Study the associated stack trace for an error group
    * Explore logs filtered to a specific error group
    * Plow through profiled attributes of the error group
* This release also brings enhanced notification support for lock screen
