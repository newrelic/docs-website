### Notes

Support for alert event context from [New Relic Radar](https://docs.newrelic.com/docs/radar/new-relic-radar/getting-started/introduction-radar), which is a tool for faster orientation during an alert event. Alert event context provides faster orientation during an alert event by detecting unusual behavior in your system and placing it alongside alert information.

### New features

* Radar alert event context

### Improvements

* Improved Infrastructure list calls to reduce latency and network usage
