### Features

* Added new environment variable `MINION_DOCKER_RUNNER_REGISTRY_ENDPOINT` to allow use of a Docker registry that isn't quay.io. For example, registry.hub.docker.com can be used to pull the Minion Runner image from Docker Hub.

### Fixes

* We no longer require access to quay.io
