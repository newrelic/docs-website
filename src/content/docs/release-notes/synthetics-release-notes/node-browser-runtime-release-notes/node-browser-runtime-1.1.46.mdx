### New Features
* Allowed custom headers to be passed to the cert check, which helps prevent cert checks from failing on endpoints depending on custom headers.
* Parses cookies from response headers and adds them to the redirect headers, which helps prevent cert checks from failing on endpoints depending on cookies.
### Improvements
* Proper cookie handling
