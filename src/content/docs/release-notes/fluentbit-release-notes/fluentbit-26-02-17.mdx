### Support for Fluent Bit 4.2.2

Users will now receive Fluent Bit version 4.2.2. For more details, refer to the [Fluent Bit announcements](https://fluentbit.io/announcements/).

This release includes [New Relic Fluent Bit Output Plugin 3.4.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.4.0) with critical security updates and enhanced platform compatibility.

### Added

* Support for SLES 15.6 and 15.7
* New Relic Fluent Bit Output Plugin 3.4.0 integration

### Changed

* The Infrastructure agent recommends Fluent Bit 4.2.2 packages
* [Fluent Bit Output Plugin Docker image](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) uses Fluent Bit 4.2.2 with output plugin 3.4.0
* aws-for-fluent-bit base image updated from 3.1.0 to 3.2.1

### Fixed

* GLIBC compatibility issues on RHEL8 and Amazon Linux 2 using Zig cross-compilation targeting GLIBC 2.17

### Notes

To stay up to date with the most recent fixes and enhancements, subscribe to our [Logs RSS feed](/docs/release-notes/logs-release-notes/).
