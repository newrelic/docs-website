### Support for Fluent Bit 3.2.2

Users will now receive Fluent Bit version 3.2.2, which includes fixes for known vulnerabilities. For more details, refer https://fluentbit.io/announcements/.

### Changed

* Infrastructure agent recommends Fluent Bit 3.2.2 packages.
* [Fluent Bit Output Plugin Docker image](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) uses Fluent Bit 3.2.2.
* [New Relic Logging Helm chart](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) installs the new plugin image (newrelic-logging chart version 1.25.0)
