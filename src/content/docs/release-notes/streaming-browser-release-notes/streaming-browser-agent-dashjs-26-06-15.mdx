## 🎉 Overview

This is a documentation-only release focused on providing comprehensive best practices guidance for production deployments. No code changes or bug fixes are included in this release.

## 📚 Documentation

### README.md Enhancements

- **Best Practices Guide** - Added comprehensive section covering:
  - Setting `contentTitle` attribute with proper metadata handling
  - Configuring `userId` for per-user analytics tracking
  - Adding custom attributes for deployment-specific data aggregation
  - Implementing gradual rollout strategy using feature flags (5% → 15% → 25% → 50% → 100%)

- **Custom Attributes Examples** - Practical examples for common use cases:
  - Subscription tier tracking
  - Regional performance monitoring
  - CDN provider analysis
  - Device type segmentation

- **NRQL Query Examples** - Ready-to-use queries for analyzing custom attributes

- **Reserved Keywords Notice** - Documentation of reserved attribute names to avoid conflicts with default attributes

### Production Deployment Guidance

- Detailed feature flag implementation examples
- Recommended rollout schedule with validation checkpoints
- Error handling and monitoring best practices

## 📦 Installation

```bash
npm install @newrelic/video-dash@4.1.1
```

or

```bash
yarn add @newrelic/video-dash@4.1.1
```

## ⚙️ Compatibility

- **dash.js:** v4.x and v5.x
- **New Relic:** All active accounts
- **Browser Support:** Modern browsers with ES6+ support
