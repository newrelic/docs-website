### Features
* Validates that the host use case works correctly with `nrdot-collector:1.17.0`.
* Renames the following component type identifiers to `snake_case` to resolve upstream deprecations. The `hostmetricsreceiver`, `resourcedetectionprocessor`, and `metricstransformprocessor` components now use these updated names for their `metadata.yaml` fields:
  * `hostmetrics` to `host_metrics`
  * `resourcedetection` to `resource_detection`
  * `metricstransform` to `metrics_transform`
