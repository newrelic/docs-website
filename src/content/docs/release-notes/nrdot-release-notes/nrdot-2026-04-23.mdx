## Changelog
This release marks the removal of the `nrdot-collector-k8s` and `nrdot-collector-host` distribution. The last published version of those distros is `1.12.0`. Please migrate to using `nrdot-collector` instead which is a superset in terms of functionality and acts as a drop-in replacement with the two removed distros being implemented as [use cases](https://github.com/newrelic/nrdot-collector-releases/blob/1.13.0/distributions/nrdot-collector/README.md#use-cases).
