## New 
- Upgrade agent to support AGP 8.8
- Added processId as a session attribute for application exit info to ensure better troubleshooting ability

## Bug Fixes
- Resolved an issue that logs are not sent after first app launch
- Resolved the issue where `HttpHeaders` were not being tracked when added through an interceptor.


## Support statement

We recommend updating the agent at least every three months. Find specific policies and dates for Android agent support in the [Mobile Monitoring Agents EOL Policy](/docs/mobile-monitoring/new-relic-mobile/get-started/mobile-agents-eol-policy/#android-eol).
