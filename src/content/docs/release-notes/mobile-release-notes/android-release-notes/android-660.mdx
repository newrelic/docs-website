## New in this release

* Add support for DexGuard 9
* End of life for supporting android SDK 21 (Android 5) to 23 (Android 6)

## Fixed in this release

* Fix a bug to capture response body using noticeHttpTransaction method

## Support statement:

New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [5.26.0](https://docs.newrelic.com/docs/release-notes/mobile-release-notes/android-release-notes/android-5260).

<Callout variant="important">
  Please note, support for Android SDK 21-23 has been removed in this release.
</Callout>
