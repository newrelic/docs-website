## Fixed in this release
* Fixed Issue where response are not recorded correctly For HTTP errors.
* Resolved zero bytesRecieved for mobile request shown in NR One.
* Fixed crash for Mapquest SDK since NewRelic 6.0.0 release.
* Fixed single span/orphan span Distributed Tracing issue.
* Removed geolocation support
* Resolved gradle optimization failures

## Support statement:
New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [5.23.0](https://docs.newrelic.com/docs/release-notes/mobile-release-notes/android-release-notes/android-5230).
