## New in this release

* Add support for [native crash reporting](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/android-agent-native-crash-reporting) in legacy agent crash reporting.
* Add distributed tracing context support to New Relic SDK

## Fixed in this release

* Fixed issue that kept the app's supportability metrics from being reported to New Relic
* Updated the agent to address Gradle deprecation warnings

## Support statement:

New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [5.26.0](https://docs.newrelic.com/docs/release-notes/mobile-release-notes/android-release-notes/android-5260).
