## New in this release

* Add support for passing Throwable types to [NewRelic.recordHandledException()](/docs/mobile-monitoring/new-relic-mobile-android/android-sdk-api/recordhandledexception-android-sdk-api)

## Fixed in this release

* Resolved a sampler thread leak that occurred in long-lived apps.

## Support statement:

New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [5.24.0](https://docs.newrelic.com/docs/release-notes/mobile-release-notes/android-release-notes/android-5240).

<Callout variant="important">
  Please note, support for Android SDK 21-23 will be removed in a future release.
</Callout>
