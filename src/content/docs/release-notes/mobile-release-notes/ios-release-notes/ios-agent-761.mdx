## Important

* This release bumps the required iOS version of the New Relic iOS agent to iOS 16. iOS 15 support will come back in a future version.

## New in this release

* Updated Public Preview of Mobile Session Replay to include errored session sampling and fixed issues related to Session Replay.

## Fixed in this release

* Fixed an issue that could cause distributed tracing to not be used in an edge case.
* Added stabiltity fixes for detected issues.
