A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend you update the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [infrastructure agent 1.30.0](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1300).

* Updated dependency `newrelic/nri-prometheus` to v2.18.7 in [#1738](https://github.com/newrelic/infrastructure-agent/pull/1738)
* Updated dependency `newrelic/nri-docker` to v1.8.20 [#1737](https://github.com/newrelic/infrastructure-agent/pull/1737)
* Updated Go to v1.20.8 [#1739](https://github.com/newrelic/infrastructure-agent/pull/1739)
