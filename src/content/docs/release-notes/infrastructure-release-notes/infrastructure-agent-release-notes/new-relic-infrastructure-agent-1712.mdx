A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.51.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1510/).

## Changed

* Removed 32-bit Windows builds. [#2144](https://github.com/newrelic/infrastructure-agent/pull/2144)
* Updated Fluent Bit version on Windows. [#2156](https://github.com/newrelic/infrastructure-agent/pull/2156)
* Updated dependency versions. [#2159](https://github.com/newrelic/infrastructure-agent/pull/2159)
* Fixed an issue where CPU performance counters didn't work on non-English systems. [#2154](https://github.com/newrelic/infrastructure-agent/pull/2154)
