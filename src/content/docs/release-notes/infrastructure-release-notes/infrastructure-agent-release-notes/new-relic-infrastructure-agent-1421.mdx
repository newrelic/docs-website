A new version of the infrastructure agent is available. [Update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent) regularly, at least every three months.

As of this release, the oldest supported version is [Infrastructure agent 1.26.0](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1260/).

## Changed
* Bumped nri-docker version to v1.8.2 in [#1663](https://github.com/newrelic/infrastructure-agent/pull/1663).
* Bumped Go version to 1.19.9 and docker dependencies in [#1664](https://github.com/newrelic/infrastructure-agent/pull/1664).

## Fixed
* Don't use ipv6-prefixed localhost hostnames as a fully qualified domain name. Fixed in [#1662](https://github.com/newrelic/infrastructure-agent/pull/1662).
