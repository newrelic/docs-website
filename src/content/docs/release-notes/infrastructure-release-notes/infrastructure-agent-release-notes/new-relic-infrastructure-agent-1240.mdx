## Notes

A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend you to upgrade the agent every 3 months.

## Added

* https and mTLS support added to the `infra-agent` HTTP ingest endpoint [#913](https://github.com/newrelic/infrastructure-agent/pull/913).
* Self-instrumentation of the agent via the Go APM agent (disabled by default) [#981](https://github.com/newrelic/infrastructure-agent/pull/981).
* Passing the `NRI_HOST_ID` environment variable to v4 integrations [#1021](https://github.com/newrelic/infrastructure-agent/pull/1021).

## Changed

* Deprecated the `register_enabled` feature flag, and added a new sdk parameter to prevent entity registration [#1015](https://github.com/newrelic/infrastructure-agent/pull/1015).
