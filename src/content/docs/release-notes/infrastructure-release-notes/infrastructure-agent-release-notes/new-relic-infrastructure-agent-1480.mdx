A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
We recommend you upgrade the agent regularly and at a minimum every 3 months. As
of this release, the oldest supported version is [infrastructure agent
1.33.0](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1330).

## Changed

* Update dependency newrelic/nri-docker to v1.8.24 [#1740](https://github.com/newrelic/infrastructure-agent/pull/1740)
* Remove internal Prometheus metrics endpoint [#1745](https://github.com/newrelic/infrastructure-agent/pull/1745)
* Move embedded binaries to /opt in Linux systems [#1742](https://github.com/newrelic/infrastructure-agent/pull/1742)

## Fixed

* Default NTP queries timeout [#1744](https://github.com/newrelic/infrastructure-agent/pull/1744)
