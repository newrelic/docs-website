## Notes
A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
We recommend you to upgrade the agent every 3 months.

## Added
* Expanding infrastructure agent support for new OS versions:
  * Ubuntu: 20.10 (groovy), 21.04 (Hirsute Hippo)
  * Suse / SLES: 12.5, 15.1, 15.2, 15.3
  * Oracle Linux: 7, 8
  * Debian: 11(bullseye)

## Changed
* Update docker alpine curl dependency [#737](https://github.com/newrelic/infrastructure-agent/pull/737)
* Bump docker image to alpine@3.14 and curl and ntpsec to latest [#744](https://github.com/newrelic/infrastructure-agent/pull/744)
* Update log forwarder output plugin to v1.9.2 and fluent-bit v1.8.6 [#742](https://github.com/newrelic/infrastructure-agent/pull/742)

## Fixed
* Don't panic when process command line is not found in Windows [#710](https://github.com/newrelic/infrastructure-agent/pull/710)
* Log forwarder support for sles12.5 with fluent bit v1.8.6 [#742](https://github.com/newrelic/infrastructure-agent/pull/742)
