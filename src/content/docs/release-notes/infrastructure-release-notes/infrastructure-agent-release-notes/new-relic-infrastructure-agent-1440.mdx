A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend that you update the agent regularly and at a minimum every three months. As of this release, the oldest supported version is [infrastructure agent 1.27.5](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1275/).

## Added
* Use rpmv3 signature mode for sles12.x packages [#1691](https://github.com/newrelic/infrastructure-agent/pull/1691)

## Changed
* Disable CGO for amd64 [#1689](https://github.com/newrelic/infrastructure-agent/pull/1689)
* Bump dependency newrelic/nri-prometheus to v2.18.5 [#1696](https://github.com/newrelic/infrastructure-agent/pull/1696)
* Bump dependency newrelic/nri-docker to v1.8.9 [#1695](https://github.com/newrelic/infrastructure-agent/pull/1695)

## Fixed
* Fallback to td-agent-bit when fluent-bit not present [#1694](https://github.com/newrelic/infrastructure-agent/pull/1694)
