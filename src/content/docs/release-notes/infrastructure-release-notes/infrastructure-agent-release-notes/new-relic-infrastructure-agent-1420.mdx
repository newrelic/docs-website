A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.25.0](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1250/).

## Added
* Support for `containerd` as container runtime in process monitoring
* Use `enable_wmi_proc_data` to enable process monitoring via WMI on Windows (when agent is run as non-admin or any restricted environment)
* Using fluent-bit instead of td-agent package for log forwarding

## Changed
* Update dependency newrelic/nri-flex to v1.8.3
* Update alpine docker tag to v3.18
