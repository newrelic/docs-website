A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
New Relic recommends that you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.63.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1630/).

## Changed

* update dependency newrelic/nri-winservices to 1.4.4 in [#2196](https://github.com/newrelic/infrastructure-agent/pull/2196)
* Remove all the EOL OS support in [#2176](https://github.com/newrelic/infrastructure-agent/pull/2176)
* update dependency newrelic/nri-flex to 1.17.5 in [#2197](https://github.com/newrelic/infrastructure-agent/pull/2197)
