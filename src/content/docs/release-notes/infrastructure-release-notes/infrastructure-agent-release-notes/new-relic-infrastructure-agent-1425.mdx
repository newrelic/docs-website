A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

New Relic recommends that you upgrade the agent regularly and at a minimum every three months. As of this release, the oldest supported version is [Infrastructure agent 1.27.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1271/).
