A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend that you update the agent regularly and at a minimum every three months. As of this release, the oldest supported version is [infrastructure agent 1.28.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1280/).
