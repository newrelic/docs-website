## New

A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).  

We recommend you to upgrade the agent at least, every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.12.0](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1120).

## Added

- Built-in log rotation [#1281](https://github.com/newrelic/infrastructure-agent/pull/1281). For more information, see [log rotate configuration option](https://docs.newrelic.com/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings/#log).
