A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). We recommend you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.43.2](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1432/).

## Added

* Config option to disable container decoration in ProcessSample  [#1888](https://github.com/newrelic/infrastructure-agent/pull/1888)

## Changed

* Update dependency newrelic/nri-docker to v2.0.4  [#1891](https://github.com/newrelic/infrastructure-agent/pull/1891)
* Update dependency newrelic/nri-prometheus to v2.21.4 [#1879](https://github.com/newrelic/infrastructure-agent/pull/1879)
