A new version of the agent has been released. Follow standard procedures to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend you to upgrade the agent regularly, at least every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.20.5](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1205).


## New
* Add support for reizerFS
* Extend memory metrics (MemoryKernelFree, MemoryBuffers)

## Changed
* Rename NTP config option
* Bump newrelic/nri-flex to v1.5.3
* Bump newrelic/nri-prometheus to v2.16.7

## Fixed
* Fix CVE-2022-32149
* Added support for super block metadata rw/ro
