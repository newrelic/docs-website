A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). We recommend you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.39.1](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1391/).

## Changed

* Update dependency newrelic/nri-prometheus to v2.21.0  [#1820](https://github.com/newrelic/infrastructure-agent/pull/1820).
* Update dependency newrelic/nri-docker to v2 [#1828](https://github.com/newrelic/infrastructure-agent/pull/1828).

## Fixed

* Create log files with reduced permissions [#1825](https://github.com/newrelic/infrastructure-agent/pull/1825).
