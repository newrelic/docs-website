## Notes
A new version of the agent has been released. See how to [update the Infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).  

We recommend you to upgrade the agent regularly, at least every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.24.2](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1242/).

## Added
* Allow [custom HTTP headers](https://docs.newrelic.com/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings/#http-client) for all requests [#1640](https://github.com/newrelic/infrastructure-agent/pull/1640)
* Ability to load agent and integration configurations from custom commands [#1640](https://github.com/newrelic/infrastructure-agent/pull/1640). Refer to [secrets documentation](https://docs.newrelic.com/docs/infrastructure/host-integrations/installation/secrets-management/#custom-commands) to learn more. 

## Changed

* Bump newrelic/nri-flex to v1.8.2 [#1631](https://github.com/newrelic/infrastructure-agent/pull/1631)
