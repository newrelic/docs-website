A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
We recommend you upgrade the agent regularly and at a minimum every 3 months. As
of this release, the oldest supported version is [infrastructure agent
1.33.2](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1332).

## Changed

* Update dependency newrelic/nri-docker to v1.8.27 [#1752](https://github.com/newrelic/infrastructure-agent/pull/1752)
* Update dependency newrelic/nri-flex to v1.9.2 [#1752](https://github.com/newrelic/infrastructure-agent/pull/1752)
* Update dependency newrelic/nri-prometheus to v2.19.0 [#1752](https://github.com/newrelic/infrastructure-agent/pull/1752)
