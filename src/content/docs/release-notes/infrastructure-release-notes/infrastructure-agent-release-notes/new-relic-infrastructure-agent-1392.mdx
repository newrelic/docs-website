A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

We recommend you upgrade the infrastructure agent regularly, at least every three months. As of this release, the oldest supported version is [agent version 1.23.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1233).


## Changed
* Set permissions to binary in PRIVILEGED mode [#1592](https://github.com/newrelic/infrastructure-agent/pull/1592)
* Added `MemBufferLimit` parameter to limit the amount of memory used by td-agent-bit process [#1576](https://github.com/newrelic/infrastructure-agent/pull/1576)
* Log original event length instead of the limit value [#1606](https://github.com/newrelic/infrastructure-agent/pull/1606)
* Upgrade Go SDK to 1.19 [#1613](https://github.com/newrelic/infrastructure-agent/pull/1613)
* Bump nri-flex version to v1.8.0 [#1610](https://github.com/newrelic/infrastructure-agent/pull/1610)


## Fixed
* Issue with msi installer on Windows 2012 [#1609](https://github.com/newrelic/infrastructure-agent/pull/1609)
