A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). We recommend you upgrade the agent regularly and at a minimum every 3 months. As of this release, the oldest supported version is [Infrastructure agent 1.41.0](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1410/).

## Changed

* Support Cyberark on Windows  [#1853](https://github.com/newrelic/infrastructure-agent/pull/1853)
* Update dependency `newrelic/nri-docker` to v2.0.2 [#1846](https://github.com/newrelic/infrastructure-agent/pull/1846)
* Update dependency `newrelic/nri-prometheus` to v2.21.1  [#1848](https://github.com/newrelic/infrastructure-agent/pull/1848)
* Update Gopsutil to 3.24.4 [#1849](https://github.com/newrelic/infrastructure-agent/pull/1849)
* Update nri-flex to 1.12.0 [#1855](https://github.com/newrelic/infrastructure-agent/pull/1855)
