A new version of the agent has been released. Follow standard procedures to [update the infrastructure agent](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).

New Relic recommends that you upgrade the agent regularly and at a minimum every three months. As of this release, the oldest supported version is [Infrastructure agent 1.27.2](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1272/).

## Changed
* Log negative CPU percent as a warning [#1676](https://github.com/newrelic/infrastructure-agent/pull/1676)
* Bump fluent-bit-plugin version to [1.17.1](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v1.17.0) [#1677]( https://github.com/newrelic/infrastructure-agent/pull/1677)
