### New

* Amazon CloudWatch Metric Streams is a new way to send metrics for all AWS services and custom namespaces to New Relic. Learn more from the following resources:
  * [New Relic blog post](https://newrelic.com/blog/best-practices/aws-cloudwatch-metric-streams)
  * [AWS News blog](https://aws.amazon.com/blogs/aws/cloudwatch-metric-streams-send-aws-metrics-to-partners-and-to-your-apps-in-real-time/)
  * [New Relic Product Documentation](/docs/integrations/amazon-integrations/aws-integrations-list/aws-metric-stream/)
