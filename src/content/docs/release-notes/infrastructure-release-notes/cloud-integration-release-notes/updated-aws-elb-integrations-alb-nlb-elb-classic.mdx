### New

* Added support for AWS [Network Load Balancers](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-albnlb-monitoring-integration) (NLBs)
* Added two Entity Types to New Relic: **ELB network load balancer**, **ELB network target group**
* Added more metrics for [AWS Application Load Balancers](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-albnlb-monitoring-integration) see the [documentation](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-albnlb-monitoring-integration) for the full list.

### Changes

* **Moved ALB and NLB integrations under the ELB section** to align with AWS
* Renamed ELB integration to [ELB (Classic)](https://docs.newrelic.com/docs/integrations/amazon-integrations/aws-integrations-list/aws-elb-classic-monitoring-integration)

Minor changes

* AWS RDS: All charts in the dashboard now **FACET**ed by database name
* GCP: You can now see the amount of SDK API calls we make in the **GCP Status Dashboard**
