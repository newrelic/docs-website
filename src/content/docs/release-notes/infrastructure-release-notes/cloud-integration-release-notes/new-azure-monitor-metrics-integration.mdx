## New

* **Azure Monitor metrics** integration is now on Public Preview. This new integration allows to get all metrics available from [Azure Monitor](https://learn.microsoft.com/en-us/azure/azure-monitor/essentials/metrics-supported). Please refer to the [documentation](https://docs.newrelic.com/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/) for details.
