### Enhanced Syslog License Key Handling

Users can now send the license key as part of the syslog message by prefixing it with `nrLicenseKey=<LICENSE_KEY>`.
With this enhancement, syslog integration is compatible with more platforms that don't permit modification of the syslog message format.
