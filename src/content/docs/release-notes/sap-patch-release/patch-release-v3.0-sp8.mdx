## Patch files

V3.0 SP8 has one file:
  - K-300SP08NEWR.SAR

## Procedures to implement the service pack

### Service pack installation and configuration

  - **Installation Location**: Install this service pack on the Central Monitoring System's client 000.
  - **Subsequent Configuration**: All further configuration should be performed in the Central Monitoring System's business client.
  - **Detailed Instructions**: Refer to the **Installation Guide V3.0** (Appendix VIII: Install Service Pack) or the **Admin Guide V3.0** (Section: Install Service Pack) for comprehensive installation instructions.

## Updates in this patch

BTP monitoring related updates:

  - New data provider `NR_CALM_RUM_TRACE`. This data provider will send **Real User Monitoring Trace** data to New Relic.

## Bug fixes

  - Update `NR_SAP_ABAP_WP` to fix a bug that limits **Tag Name** to only **30** characters. Now the tag name is matching the SAP standard data structure of **60** characters.
