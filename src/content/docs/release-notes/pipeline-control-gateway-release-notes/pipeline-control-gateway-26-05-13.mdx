## Pipeline Control Gateway Release Notes — v2.2.0 

### Multi fleet Support
- Manage multiple gateway fleets from a single view with a new fleet selector dropdown.                                                                                            

### Updated simplified YAML
- Cleaned up simplified config YAML by removing unnecessary fields and updating version to v2

### Improved the UI & deployment UX
- Side by side config diff preview for comparing changes before deployment in transform processor.
- Updated the deployment page with pre-populated details — a default name and description are provided, editable to help track your changes.
