### Pipeline Control Gateway Release Notes - v2.0.0

#### Updated to OpenTelemetry Collector v0.139.0

This release is based on OpenTelemetry Collector v0.139.0, bringing the latest stability improvements and features from the upstream project.

#### New processor capabilities

Added support for advanced data processing features:
- **Filter processor**: Drop data based on conditions before it leaves your network
- **Sampling processor**: Implement intelligent sampling strategies for high-volume telemetry
- **Transform processor**: Modify telemetry data attributes and structure in-flight

#### Enhanced configuration options

- **YAML configuration**: Comprehensive guidance for YAML-based gateway configuration alongside UI-based setup
- **UI guide**: Detailed walkthrough for configuring the gateway through the New Relic interface

#### Improved documentation structure

- Reorganized documentation to better clarify distinctions between cloud rules (running in New Relic's infrastructure) and gateway rules (running in your environment)
- Expanded troubleshooting resources covering gateway installation, cluster health, data reception, transmission, and visibility
