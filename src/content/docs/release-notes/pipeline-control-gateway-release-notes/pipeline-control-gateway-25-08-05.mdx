### Pipeline Control Gateway Release Notes - v1.1.0

#### Support for SQL Trace and Transaction Trace Data ####
* Functionality has been implemented to selectively drop data and attributes from SqlTrace and TransactionTrace

#### Security Bug Fixes ####

* Cross-site scripting vulnerability and HTTP Proxy bypass using IPv6 Zone IDs in
 golang.org/x/net have been fixed.
* We have upgraded to Go `1.24` and the following packages to these specified versions as part of the above fixes:
   * golang.org/x/net to v0.38.0
   * golang.org/x/sys to v0.31.0
   * golang.org/x/text to v0.23.0

#### Support for New Functions ####

The system now supports several new functions in NRQL drop rule queries that were previously unavailable, including
 `aparse()`, `floor()`, `getField()`, `hourOf()`, `numeric()`, `round()`, `string()`, `substring()`, `weekdayOf()`, `mod()`, and `dimensions()`
