## Notes

This release of the Python agent fixes an issue with urllib recursion as well as an import error caused by non-ASCII characters in comments

Install the agent using `easy_install/pip/distribute` via the [Python Package Index](https://pypi.python.org/pypi/newrelic) or download it directly from the New Relic [download site](http://download.newrelic.com/python_agent/release).

## Bug Fixes

* **Urllib3 Recursion Regression**

  Urllib3 recursion regression issue that was seen in Gevent has been fixed

* **Import Error caused by non-ASCII character in comments**

  An import error caused by non-ASCII characters has been fixed to the correct quotation formatting
