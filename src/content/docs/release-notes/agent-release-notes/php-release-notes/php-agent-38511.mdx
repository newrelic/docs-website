### Notes

* Preliminary support for [Cross Application Tracing](https://newrelic.com/docs/traces/cross-application-traces) has been added. The following external calls are supported.
  * curl_exec()
  * file_get_contents()
  * httprequest::send()
* Communication between the agent and the daemon process has been improved, fixing a rare issue that could affect data quality.
* Fixes a rare issue where too many daemon processes could be launched.
* Various bug fixes and stability improvements.
