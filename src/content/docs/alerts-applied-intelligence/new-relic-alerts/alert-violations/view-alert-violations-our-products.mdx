To view [violations](/docs/using-new-relic/welcome-new-relic/get-started/glossary#alert-violation) for a specific [entity (alert target)](/docs/using-new-relic/welcome-new-relic/get-started/glossary#alert-entity), select the **Violations** index from the product page. Then, from the product's **Violations** index, use the available tools to search, sort, or view violations for the selected entity. From here you can also view or update the policies and conditions that triggered the violations.

<img
  title="screen-apm-violations.png"
  alt="screen-apm-violations.png"
  src={accountsApmViolationsScreen}
/>

<figcaption>
  **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities > (select an app) > Events > Violations:** Here's an example of the violations index. Use the available tools to search, sort, or view violations for the selected entity, or to view or update the associated policies and conditions.
</figcaption>

<CollapserGroup>
  <Collapser
    id="apm"
    title="APM"
  >
    To view violations for a specific app or key transaction:

    * [App](/docs/apm/new-relic-apm/installation-configuration/view-apps-alert-information): Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (select an app) > Events > Violations**.
    * [Key transaction:](/docs/apm/transactions/key-transactions/view-key-transactions-alert-information) Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Key transactions > (select a key transaction) > Events > Violations**.
  </Collapser>

  <Collapser
    id="browser"
    title="Browser"
  >
    To view violations for a specific [browser app](/docs/browser/new-relic-browser/installation-configuration/view-browser-apps-alert-information): Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select a browser app) > Move views > Violations**.
  </Collapser>

  <Collapser
    id="infrastructure"
    title="Infrastructure"
  >
    To view violations for a [specific host monitored by Infrastructure:](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information) Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Settings > Alerts**.
  </Collapser>

  <Collapser
    id="mobile"
    title="Mobile"
  >
    To view violations for a [specific mobile app:](/docs/mobile-monitoring/mobile-monitoring-installation/getting-started/view-mobile-apps-alert-information) Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select a mobile app) > More views > Event explorer**.
  </Collapser>

  <Collapser
    id="synthetics"
    title="Synthetic monitoring"
  >
    To view violations for a [specific monitor:](/docs/synthetics/new-relic-synthetics/dashboards/view-monitors-alert-information) Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Synthetic monitoring >(select a monitor) > Activity > Alert violations**.
  </Collapser>
</CollapserGroup>
