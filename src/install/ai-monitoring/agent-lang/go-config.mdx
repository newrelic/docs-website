To collect AI data, you need to ensure that certain configurations are enabled when you instrument your AI-powered app. We also recommend updating other variables to improve data retention.

We recommend applying these configurations in your `newrelic.NewApplication` function call. You'll pass the AI monitoring configurations as as parameters to `NewApplication`.

<CollapserGroup>
  <Collapser
    id="config"
    title="Configure through `newrelic.Config`"
  >
    ```go
    newrelic.ConfigAIMonitoringEnabled(true)
    ConfigCustomInsightsEventsMaxSamplesStored(10000)
    ```
  </Collapser>

  <Collapser
    id="environment"
    title="Configure with environment variables"
  >
    ```bash
    NEW_RELIC_AI_MONITORING_ENABLED = TRUE
    ```

    ```bash
    AIMonitoring.Streaming.Enabled = TRUE
    ```
  </Collapser>
</CollapserGroup>
