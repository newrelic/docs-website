---
headingText: Install the agent
componentType: default
---

    1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT>.
    
    2. Search and select the <DNT>**Streaming Video & Ads**</DNT> tile to open.

    3. Give a name for your application, and click <DNT>**Create**</DNT>. 
    
    4. Once the application name is created, click <DNT>**Save and Continue**</DNT>.

    5. Select <DNT>**Mobile**</DNT>, and click <DNT>**Continue**</DNT>.

    6. Select <DNT>**iOS**</DNT> to monitor video players in your applications, and click <DNT>**Continue**</DNT>.
