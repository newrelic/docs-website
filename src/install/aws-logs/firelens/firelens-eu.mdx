componentType: default
headingText: 'Optional: Forward logs to EU or JP'
---

To forward logs from Firelens to a regional New Relic account, add an additional property to the `options` field of the `logConfiguration` object in your application containers.

For EU accounts:

```json
"endpoint": "https://log-api.eu.newrelic.com/log/v1"
```

For JP accounts:

```json
"endpoint": "https://log-api.jp.nr-data.net/log/v1"
```
