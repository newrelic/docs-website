<Callout variant="tip">
    Add our streaming agent to an application that already has our browser agent installed. If you are a first time user and want to add the video agent to a new application, see [Instrument a new browser application](/install/browser-media/?task=newapplication).
</Callout>

Select any one of the following method to monitor the agent on your browser application:

<CollapserGroup>
  <SideBySide>

    <Side>
    <Collapser
        id="javascript-snippet"
        title="Place a JavaScript snippet in frontend code"
      >
      1. Select the **Place a JavaScript snippet in frontend code** option.
      2. Click **Continue**.
      3. Install the latest release from the New Relic platform by copying the code snippet and running it in your terminal (Mac user) / Command prompt (Windows user).
      4. After the script is downloaded, include the code snippet displaying in the platform to your application to start monitoring your video player.

      <Callout variant="important">
        We recommend you to pass the player instance to the newrelic player tracker.
      </Callout>
       5. Click **See your data**.
      </Collapser>
        </Side>
</SideBySide>
</CollapserGroup>

<Callout variant="important">
    It may take a few minutes to interact with your application and generate data.
</Callout>
