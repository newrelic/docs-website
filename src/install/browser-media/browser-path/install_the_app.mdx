headingText: Install the agent
componentType: default
---

1. Access the New Relic platform and begin setting up video monitoring for your application, go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT> and navigate to the integrations marketplace.

2. Locate the streaming video monitoring solution among available integrations, search and select the <DNT>**Streaming Video & Ads**</DNT> tile to open.

3. Configure the monitoring setup for your specific New Relic account, select an <DNT>**Account**</DNT> from the dropdown and click <DNT>**Continue**</DNT>.

4. Specify that you want to monitor video performance in browser applications, select <DNT>**Browser**</DNT> from the platform selection screen, and then click <DNT>**Continue**</DNT>.

5. Ensure proper instrumentation for your specific video player technology, select the video player you want to monitor and click **Continue**.
