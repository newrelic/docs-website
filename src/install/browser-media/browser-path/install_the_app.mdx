headingText: Install the agent
componentType: default
---

    1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents**</DNT>.
    
    2. Search and select the <DNT>**Streaming Video & Ads**</DNT> tile to open.

    3. Select an account from the <DNT>**Account**</DNT> dropdown and click <DNT>**Continue**</DNT>.

    3. Select <DNT>**Browser**</DNT>, and then click <DNT>**Continue**</DNT>.

    4. Select one of the video player you want to monitor and click **Continue**.
