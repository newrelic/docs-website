<Steps>
  <Step>
    Provide a name to your application and click **Save and Continue**.
  </Step>

  <Step>
    Install the latest release from the New Relic platform by copying the code snippet and running it in your terminal (Mac user) / Command prompt (Windows user).

    <Callout variant="tip">
      Select **npm** or **yarn** to install the video agent.
    </Callout>
  </Step>

  <Step>
    After the script is downloaded, include the code snippet displaying in the platform to your application to start monitoring your video player.
  </Step>

  <Step>
    Select **Get Application Info**. This would generate the application id for the browser entity generation.
  </Step>

  <Step>
    Click **See your data**.
  </Step>
</Steps>

<Callout variant="important">
    It may take a few minutes to interact with your application and generate data.
</Callout>
