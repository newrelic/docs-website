      1. [Install the browser agent](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent) on an application you're not monitoring with New Relic.

      2. Once the browser agent is installed, you will be redirected to the **Select where to install** section. Click **Continue**.
            
      3. Select any one of the following method to monitor the agent on your browser application:

<CollapserGroup>
  <SideBySide>
    <Side>
    <Collapser
        id="npm-install"
        title="Install using npm"
      >
      <p> After you have selected the **Install using npm** option, you will be redirected to the respective New Relic Tracker agent page.</p>
    </Collapser>
    </Side>

    <Side>
    <Collapser
        id="javascript-snippet"
        title="Place a JavaScript snippet in frontend code"
      >
      1. Select the **Place a JavaScript snippet in frontend code** option.
      2. Click **Continue**.
      3. Install the latest release from the New Relic platform by copying the code snippet and running it in your terminal (Mac user) / Command prompt (Windows user).
      4. After the script is downloaded, include the code snippet displaying in the platform to your application to start monitoring your video player.

      <Callout variant="important">
        We recommend you to pass the player instance to the newrelic player tracker.
      </Callout>
       5. Click **See your data**.
      </Collapser>
        </Side>
</SideBySide>
</CollapserGroup>

    <Callout variant="important">
      It may take a few minutes to interact with your application and generate data.
    </Callout>
