1. Select the **Install using npm** and click **Continue**.
2. Provide a name to your application and click **Save and Continue**.
3. Copy and add the video agent to your respective New Relic Tracker agent page..

<Callout variant="tip">
  Select **npm** or **yarn** to install the video agent.
</Callout>

4. Select **Get Application Info**. This would generate the application id for the browser entity generation.
5. Click **See your data**.

  <SideBySide>

    <Collapser

      >
      1. Select the **Place a JavaScript snippet in frontend code** option.
      2. Click **Continue**.
      3. Install the latest release from the New Relic platform by copying the code snippet and running it in your terminal (Mac user) / Command prompt (Windows user).
      4. After the script is downloaded, include the code snippet displaying in the platform to your application to start monitoring your video player.

      <Callout variant="important">
        We recommend you to pass the player instance to the newrelic player tracker.
      </Callout>
       5. Click **See your data**.
      </Collapser>
        </Side>
</SideBySide>
</CollapserGroup>
