componentType: default
headingText: Download using MSI
---

1. Download the latest .MSI installer image for the NGINX integration [from our repository](https://download.newrelic.com/infrastructure_agent/windows/integrations/nri-nginx/nri-nginx-amd64.msi).

2. In an admin account, run the install script using an absolute path.

   ```powershell
   msiexec.exe /qn /i PATH\TO\nri-nginx-amd64.msi
   ```

3. Run the following command in a Windows command prompt or double-click the file in Explorer:

   ```powershell
   msiexec.exe /qn /i PATH\TO\nri-nginx-amd64.msi
   ```
