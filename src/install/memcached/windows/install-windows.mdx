componentType: default
headingText: Configure the Memcached integration
---

1. In the Integrations directory, `C:\Program Files\New Relic\newrelic-infra\integrations.d\`, create a copy of the sample configuration file by running:

   ```shell
   copy memcached-config.yml.sample memcached-config.yml
   ```

2. Edit the `memcached-config.yml` with your info. Here's a basic configuration example:

   ```yml
   integrations:
     - name: nri-memcached
       env:
         PORT: "11211"
         HOST: memcached_host

         # ifauthentication is enabled.
         USERNAME: cacheuser
         PASSWORD: password
       interval: 15s
       inventory_source: config/memcached
   ```
