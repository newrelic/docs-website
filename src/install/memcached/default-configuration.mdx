componentType: default
headingText: Configure the Memcached integration 
---

Edit the `Memcached3-config.yml` file to configure your integration as needed. The following example config fills in all the required options and reports all metrics the integration pulls:

```yml
integrations:
  - name: nri-memcached3
    config:
      Memcached_cluster_name: MY_CLUSTER
      Memcached_uri: Memcached://username:password@localhost:27017
      exporter_port: 9126
```
