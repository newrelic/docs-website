componentType: default
headingText: Create a workflow automation using the starter template
---

1. Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Workflow Automation**. Click **Use template** to start your workflow setup.
2. In **All Templates** section, select the **Starter template** and click **Use template**. You can use **Preview** to examine the template before proceeding.
3. Configure the workflow automation with the following steps:
    - Use [**getCurrentTime**](/docs/workflow-automation/setup-and-configuration/actions-catalog/#http.get) to retrive the current time from the HTTP endpoint.
    - Log the current time using [**LogTime**](/docs/workflow-automation/setup-and-configuration/actions-catalog/#newrelic.ingest.sendEvents).

    <Callout variant="important">
      For customization:
      - View the YAML code with **`</>`**
      - Add actions using the **`+`** icon
      - Implement conditional steps using the **`Switch`** action.
    </Callout>

    <img
      title="Image of the starter template"
      alt="Image of the starter template"
      src="/images/starter-template.webp"
    />

4. Click **Save**.
5. In the **Save workflow** dialog box:
    * Enter a name and description for your workflow automation.
    * **Configure inputs**:
      * Add a the name for each input.
      * Select the **Type** from the dropdown list.
6. Select **Create** to finalize the workflow setup and click **Run**.
7. In the **Run workflow** dialog box, enter the required timezone and click **Start**.
8. Click **View logs** to view the logs of the workflow automation. For more information about log management, see [Get started with log management](/docs/logs/get-started/get-started-log-management).
