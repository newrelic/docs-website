<Callout variant="tip">
The default listening port for **ktranslate** is port `5143 (TCP/UDP)`. To use the more common syslog port of `514`, our guided install redirects traffic into the Docker container with the flag`-p 514:5143/udp`. To bind the listener to a port above `1024`, add `-syslog.source="0.0.0.0:YOUR_PORT"` to the end of the run command instead.
</Callout>
