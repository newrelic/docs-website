---

1. Use the following command to start an Apache Flink cluster:

   ```bash
   ./bin/start-cluster.sh
   ```

2. Check for metrics on the following URLs

   Job manager metrics:

   ```
   http://YOUR_DOMAIN:9250
   ```

   Task manager metrics:

   ```
   http://YOUR_DOMAIN:9251
   ```
