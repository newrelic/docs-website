To enable the integration, ensure you meet these requirements:

* [Microsoft SQL Server 2014 to Microsoft SQL Server 2022](https://learn.microsoft.com/en-us/troubleshoot/sql/releases/download-and-install-latest-updates)
* Azure SQL Managed Instance, Azure SQL virtual machines
* TLS authentication version TLS 1.2 and above
* SQL Server user, domain user, or gMSA with user privileges for both `CONNECT` and `VIEW SERVER STATE`, and `READ` access permissions


<Callout variant="tip" title="tip">
    You can enable query-level monitoring for Windows environment. For more information, refer [Enable query-level monitoring for Microsoft SQL Server](/docs/infrastructure/host-integrations/host-integrations-list/microsoft-sql/microsoft-sql-server-query-performance/).
</Callout>
