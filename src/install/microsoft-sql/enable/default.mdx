headingText: Enable your Microsoft SQL Server 
---

To enable the Microsoft SQL Server integration, you need a user with `CONNECT`, `VIEW SERVER STATE`, and read access permissions.

Check the Microsoft documentation about Microsoft SQL Server for details on:

* [Creating logins](https://docs.microsoft.com/en-us/sql/t-sql/statements/create-login-transact-sql?view=sql-server-2017) 
* [Creating users](https://docs.microsoft.com/en-us/sql/t-sql/statements/create-user-transact-sql?view=sql-server-2017)
