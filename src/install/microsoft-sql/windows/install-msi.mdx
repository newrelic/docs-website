headingText: Download using MSI
---

1. Download the latest `.msi` installer package for the desired integration [from our repository](https://download.newrelic.com/infrastructure_agent/windows/integrations/).

2. Open `Command Prompt` or `PowerShell` as an administrator.

3. Go to the directory where you downloaded the `.msi` file.

4. Run this command:

   ```shell
   msiexec.exe /qn /i PATH\TO\nri-mssql-amd64.msi
   ```
   - Replace `nri-mssql-amd64.msi` with the actual filename if it differs, or provide the full path to the file.
   - `/qn` performs a quiet installation (no user interface).
   - `/i` indicates an install operation.
