To enable the integration, ensure you meet these requirements:

- Supported SQL Server environments:
    - [Microsoft SQL Server 2014-2022](https://learn.microsoft.com/en-us/troubleshoot/sql/releases/download-and-install-latest-updates) (on-premises)
    - Azure SQL managed instance
    - Azure SQL virtual machines
    - Azure SQL database
- TLS 1.2 or later
- A SQL Server user with the following permissions:
    - `CONNECT` server permission
    - `VIEW SERVER STATE` server permission
    - `READ` access to all databases accessed by the integration
