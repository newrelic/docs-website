To enable the integration, ensure you meet these requirements:

- Supported SQL Server environments:
    - [Microsoft SQL Server 2014-2022](https://learn.microsoft.com/en-us/troubleshoot/sql/releases/download-and-install-latest-updates) (on-premises)
    - Azure SQL managed instance
    - Azure SQL Virtual Machines
- TLS 1.2 or later
- A SQL Server user, domain user, or Group Managed Service Account (gMSA) with the following permissions:
    - `CONNECT` server permission
    - `VIEW SERVER STATE` server permission
    - `READ` access to all databases accessed by the integration

<Callout variant="tip" title="tip">
    You can enable query-level monitoring for Windows environment. For more information, refer to [Enable query-level monitoring for Microsoft SQL Server](/docs/infrastructure/host-integrations/host-integrations-list/microsoft-sql/microsoft-sql-server-query-performance/).
</Callout>
