---

Follow these steps:

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Dashboards**.
2. Click **Create a dashboard**, and then select **Browse pre-built dashboards**.
3. Search for and select **MSSQL On-Host Integration w/ Custom Queries**.
4. Click **View dashboard** to see your Microsoft SQL server data in New Relic.

For more on how to find and use your data, see how to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data). You can find a full list of metrics and their attributes at the bottom of this doc.

Microsoft SQL Server data is attached to the following [event types](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#event):

* [`MssqlDatabaseSample`](#database-metric)
* [`MssqlInstanceSample`](#instance-metric)
* [`MssqlWaitSample`](#wait-metric)
* [`MssqlCustomQuerySample`](#custom-query) (if you run custom queries)
