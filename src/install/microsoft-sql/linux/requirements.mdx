To enable the integration, ensure you meet these requirements:

* If on-host SQL server, [Microsoft SQL Server 2014 to Microsoft SQL Server 2022](https://learn.microsoft.com/en-us/troubleshoot/sql/releases/download-and-install-latest-updates) is required. You can also use Azure SQL Managed Instance for the integration.
* TLS version 1.2 or later, with TLS authentication
* SQL Server user, domain user, or gMSA with user privileges for both `CONNECT` and `VIEW SERVER STATE`, and `READ` access permissions
