Once you've finished all the steps, you can test the connection to make sure everything is working properly and view your data.

<img
  width="80%;"

  alt="Testing Kubernetes installation"

/>

<Callout variant="tip">
  Not seeing data? If you completed the steps and are still not seeing data, check out [this troubleshooting page](/docs/kubernetes-pixie/kubernetes-integration/troubleshooting/kubernetes-integration-troubleshooting-not-seeing-data/).
</Callout>

After the installation, you should see the following in the UI:

* `newrelic-nrk8s-ksm` pod.
* `newrelic-nrk8s-kubelet` pod for each node in your cluster.
* `newrelic-nrk8s-control-plane` pod for each master node in your cluster, if any.
* `newrelic-kube-state-metrics` pod, if you included KSM with our installation.
* `newrelic-nri-kube-events` pod, if you enabled Kubernetes events reporting.
* `prometheus-agent` pod, if you enabled the [Prometheus agent](/docs/infrastructure/prometheus-integrations/install-configure-prometheus-agent/install-prometheus-agent) integration.
* `newrelic-newrelic-logging` pod for each node in your cluster, if you enabled the logging integration.

<CollapserGroup>
  <Collapser
    id="add-pre-defined-dashboards"
    title="Add pre-defined dashboards"
  >
    1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > +Integrations & Agents**</DNT> and type <DNT>**Kubernetes**</DNT>.

    2. Under <DNT>**Dashboards**</DNT>, click <DNT>**Kubernetes**</DNT>.

       <img
         title="Add Kubernetes dashboards"
         alt="Add Kubernetes dashboards"
         src="/images/kubernetes_screenshot-crop_add-dashboards.webp"
       />

    3. Click <DNT>**Edit**</DNT> if you want to change the account in the popup window that opens.

    4. Click <DNT>**Setup Kubernetes**</DNT> or <DNT>**Skip this step**</DNT> if you already setup this datasource.

    5. Click <DNT>**View dashboard**</DNT>, and see your Kubernetes data in New Relic.
  </Collapser>

  <Collapser
    id="add-pre-defined-alert-policies"
    title="Add pre-defined alert policies"
  >

    For more on creating alerts, see our [alert conditions](/docs/alerts/create-alert/create-alert-condition/alert-conditions/) doc.
  </Collapser>    
</CollapserGroup>

  If you install the integration through the [New Relic Kubernetes quickstart](https://newrelic.com/instant-observability/kubernetes) you'll have pre-built resources like a dashboard, instrumentation, and 23 alerts.
</Callout>
