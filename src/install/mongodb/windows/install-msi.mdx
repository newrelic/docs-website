1. Download the latest latest [.exe installer image](https://download.newrelic.com/infrastructure_agent/windows/integrations/) from our repository.

2. In an admin account, run the install script using an absolute path.

    ```shell
    msiexec.exe /qn /i <PATH_TO_YOUR_nri-mongodb3-amd64.msi>
    ```
