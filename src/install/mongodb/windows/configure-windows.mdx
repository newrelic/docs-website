componentType: default
headingText: Configure the MongoDB integration
---

1. Go to the integrations directory, `C:\Program Files\New Relic\newrelic-infra\integrations.d\`. Run this command to create a copy of the sample configuration file:

  ```shell
    cp mongodb3-config.yml.sample mongodb3-config.yml
  ```

2. Edit the `mongodb3-config.yml` file. See the [`mongodb3-config.yml`sample files](#examples) for more complex configuration examples.
