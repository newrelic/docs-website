componentType: default
headingText: Configure the MongoDB integration
---

1. Change your directory to the integrations configuration folder. Run this command:

    ```shell
    cd /etc/newrelic-infra/integrations.d
    ```

2. Copy the sample configuration file. Run this command:

    ```shell
    sudo cp mongodb3-config.yml.sample mongodb3-config.yml
    ```

3. Edit the `mongodb3-config.yml` file. The only 2 required fields are `mongodb_cluster_name` and `mongodb_uri`. This example configuration file collects all metrics:

    ```yml
    integrations:
      - name: nri-mongodb3
        config:
          mongodb_cluster_name: my_cluster
          mongodb_uri: mongodb://username:password@localhost:27017
          exporter_port: 9126
    ```

See the [`mongodb3-config.yml`sample files](#examples) for more complex configuration examples.
