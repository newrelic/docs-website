componentType: default
headingText: Configure the MongoDB integration 
---

Edit the `mongodb3-config.yml` file to configure your integration as needed. The following example config fills in all the required options and reports all metrics the integration pulls:

```yml
integrations:
  - name: nri-mongodb3
    config:
      mongodb_cluster_name: MY_CLUSTER
      mongodb_uri: mongodb://username:password@localhost:27017
      exporter_port: 9126
```
