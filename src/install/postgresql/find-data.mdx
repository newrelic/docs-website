headingText: Find and use data
---

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents** and type **PostgreSQL**.

2. In the **Dashboards**, click **PostgreSQL**.

   <img
     title="Install the PostgreSQL dashboard"
     alt="Installing the PostgreSQL dashboard"
     src="/images/infrastructure_screenshot-full_install-postgresql-dashboard.webp"
   />

3. In the displayed window, click Edit to change the account if needed.

4. Click **Setup PostgreSQL** to setup the data source or click **Skip this step** if the data source is already setup.

5. Click **View dashboard** to see your PostgreSQL data in New Relic.

For more on how to find and use your data, see how to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
