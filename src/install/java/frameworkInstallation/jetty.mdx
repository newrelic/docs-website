### Pass your argument

To pass the `-javaagent` argument on Jetty:

<Tabs>
	<TabsBar>
		<TabsBarItem id="jetty-sh">jetty.sh</TabsBarItem>
		<TabsBarItem id="jetty-startini">start.ini</TabsBarItem>
	</TabsBar>

	<TabsPages>
		<TabsPageItem id="jetty-sh">
      If you set exec in `jetty.sh`, edit `JAVA_OPTIONS` in your `jetty.sh` script:

      ```java
      export JAVA_OPTIONS="${JAVA_OPTIONS} -javaagent:/full/path/to/newrelic.jar"
      ```

		</TabsPageItem>
		<TabsPageItem id="jetty-startini">
      If you set exec in your config file, add the agent path to your `start.ini` config file:

      ```java
      -javaagent:/full/path/to/newrelic.jar
      ```
		</TabsPageItem>
	</TabsPages>
</Tabs>
