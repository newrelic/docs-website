### Pass your argument

To pass the `-javaagent` argument on Play:

<Tabs>
	<TabsBar>
		<TabsBarItem id="play-1-2-4">Play 1.2.4</TabsBarItem>
		<TabsBarItem id="play-2-0">Play 2.0</TabsBarItem>
		<TabsBarItem id="play-2-2">Play 2.2</TabsBarItem>
		<TabsBarItem id="play-2-3-etc">Play 2.3, 2.4, and 2.5</TabsBarItem>        
	</TabsBar>

	<TabsPages>
		<TabsPageItem id="play-1-2-4">
            Append the `-javaagent` argument when running your Play application:

            ```
            play run <var>your_app_name</var> -javaagent:<var>/full/path/to/</var>newrelic.jar
            ```

		</TabsPageItem>
		<TabsPageItem id="play-2-0">
            1. Begin with an **unzipped** distribution containing the `start` script:

            ```
            play clean dist && unzip dist/*.zip
            ```
            2. Append the `-javaagent` argument when starting your Play app:

            ```
            cd <var>unzipped/folder</var>; chmod a+x start; ./start -javaagent:<var>/full/path/to/</var>newrelic.jar
            ```

		</TabsPageItem>
		<TabsPageItem id="play-2-2">
            1. Begin with an **unzipped** distribution containing the `start` script:

            ```
            play clean dist && unzip <var>target/directory</var>/universal/*.zip
            ```
            2. Append the `-J-javaagent` argument when starting your Play app:

            ```
            cd <var>unzipped/folder</var>; ./bin/<var>scriptname</var> -J-javaagent:<var>/full/path/to/</var>newrelic.jar
            ```

            <Callout variant="tip">
                To enable page load timing, see [Manual instrumentation with Play 2.2](/docs/agents/java-agent/instrumentation/page-load-timing-java#manual-play-2-2).
            </Callout>

		</TabsPageItem>
		<TabsPageItem id="play-2-3-etc">
            1. Begin with an **unzipped** distribution containing the `start` script:

            ```
            activator clean dist && unzip <var>target/directory</var>/universal/*.zip
            ```
            2. Append the `-J-javaagent` argument when starting your Play app:

            ```
            cd <var>unzipped/folder</var>; ./bin/<var>scriptname</var> -J-javaagent:<var>/full/path/to/</var>newrelic.jar
            ```
            3. If you use Typesafe Activator with Play 2.4, add this line to your `build.sbt`:

            ```
            javaOptions ++= Seq("-javaagent:<var>/full/path/to/</var>newrelic.jar")
            ```
		</TabsPageItem>        
	</TabsPages>
</Tabs>
