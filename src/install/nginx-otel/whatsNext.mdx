## NGINX Open Source metrics [#nginx-opensource]

The OpenTelemetry Collector's NGINX receiver collects metrics from the HTTP stub status module. Below are the key metrics available:

<CollapserGroup>
  <Collapser
    id="nginx-connections"
    title="Connection metrics"
  >
    <table>
      <thead>
        <tr>
          <th style={{ width: "350px" }}>
            Metric
          </th>
          <th>
            Description
          </th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>
            `nginx.connections_accepted`
          </td>
          <td>
            Total number of accepted client connections
          </td>
        </tr>
        <tr>
          <td>
            `nginx.connections_handled`
          </td>
          <td>
            Total number of handled connections. Generally, the same as accepted unless some resource limit has been reached
          </td>
        </tr>
        <tr>
          <td>
            `nginx.connections_current`
          </td>
          <td>
            Current number of nginx connections by state. States include: `active`, `reading`, `writing`, `waiting`
          </td>
        </tr>
        <tr>
          <td>
            `nginx.requests`
          </td>
          <td>
            Total number of requests made to NGINX since start
          </td>
        </tr>
      </tbody>
    </table>
  </Collapser>

</CollapserGroup>

For more details, see the [NGINX receiver documentation](https://github.com/open-telemetry/opentelemetry-collector-contrib/blob/main/receiver/nginxreceiver/documentation.md).
