The NGINX Plus OpenTelemetry integration collects metrics through the NGINX Prometheus exporter. The integration retrieves data from the [NGINX Plus HTTP API module](http://nginx.org/en/docs/http/ngx_http_api_module.html) via the Prometheus exporter.

### Metric data [#metrics]
