headingText: Check the compatibility and requirements
componentType: default
---

Make sure your host meets the following requirements before you begin:

- You run either NGINX Open Source with the HTTP stub status module or NGINX Plus with the HTTP API module enabled.
- You can connect to the host with `sudo` privileges to install packages and restart services.
- The host can reach New Relic over outbound HTTPS on port 443 and the OTLP ingest endpoint you plan to use.
- You can install and run the OpenTelemetry Collector Contrib distribution on the host.
