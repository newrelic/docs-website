---
headingText: Check the compatibility and requirements
componentType: default
---

* MySQL version 5.6 or later, or MariaDB version 10.2 or later.
* TLS version 1.2 or later, with TLS authentication.
* A New Relic account. If you don't have one, you can [sign up](https://newrelic.com/signup) for free. No credit card is required.

<Callout title="Note">
  The MySQL integration (`nri-mysql`) supports MySQL and MariaDB. For MariaDB, query-level monitoring (including query samples and execution plans) is only supported for the InnoDB storage engine.
</Callout>
