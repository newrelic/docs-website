componentType: default
headingText: Install the MySQL integration
---

1. [Download the `.msi` installer for New Relic's MySQL integration](https://download.newrelic.com/infrastructure_agent/windows/integrations/nri-mysql/nri-mysql-amd64.msi).

2. To install New Relic's MySQL integration, run the following command:
   ```shell
   msiexec.exe /qn /i $PATH_TO\nri-mysql-amd64.msi
   ```

3. In the `C:\Program Files\New Relic\newrelic-infra\integrations.d\` directory, to create a copy of the sample configuration file, run the following command:
   ```shell
   copy mysql-config.yml.sample mysql-config.yml
   ```

4. Edit the `mysql-config.yml` configuration file using one of the [configuration examples](#examples). 

5. To enable automatic MySQL error log parsing and forwarding, rename the `mysql-log-win.yml.example` file to `mysql-log-win.yml`:
   ```shell
   ren "C:\Program Files\New Relic\newrelic-infra\logging.d\mysql-log-win.yml.example" mysql-log-win.yml
   ```
