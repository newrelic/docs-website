---
headingText: Check the compatibility and requirements
componentType: default
---

* MySQL version 5.6 or later.
* TLS version 1.2 or later, with TLS authentication.
* A New Relic account. If you don't have one, you can [sign up](https://newrelic.com/signup) for free. No credit card is required.

<Callout variant="tip">
  **Backup metrics:** The backup monitoring features (`EXTENDED_BACKUP_METRICS` and `EXTENDED_BACKUP_HISTORY_METRICS`) require MySQL 5.7 or higher (MySQL 8.0+ recommended), and must be explicitly enabled in the configuration file.
</Callout>
