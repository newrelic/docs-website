headingText: Find and use data
---

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Dashboards**.

2. In the **Dashboards** page, click **+ Create a dashboard**.
3. In the **Create a dashboard** page, click **Browse pre-built dashboards**.
4. In the search bar, type **PostgreSQL** and select for **MySQL**.

   <img 
      title="Install the MySQL dashboards" 
      alt="Installing the  MySQL dashboards" 
      src="/images/infrastructure_screenshot-crop_install-mysql-dashboard.webp"
   />

5. *(Optional)* In the displayed window, click **Edit** to change the account.
6. To setup the data source, click **Setup MySQL**. If the data source is already setup, click **Skip this step**.
7. Click **View dashboard**, and see your MySQL data in New Relic.

For more on how to find and use your data, refer to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
