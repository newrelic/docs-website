headingText: Download using MSI
---

1. Download the latest .MSI installer image for the integration [from our repository](https://download.newrelic.com/infrastructure_agent/windows/integrations/).
2. In an admin account, run the install script using an absolute path.
   ```shell
   msiexec.exe /qn /i PATH\TO\nri-haproxy.msi
   ```
