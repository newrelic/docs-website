Video agent for Android, monitors the video player in your mobile app, giving you a comprehensive view of your video and ad playback performance. It works for Android apps written using Java or Kotlin.

## Prerequisites [#requirements]

Before you install the Android video agent, ensure you have:

- A New Relic [account ID](/docs/accounts/accounts-billing/account-setup/account-id).
- Reviewed the [compatibility and requirements](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements).
- Reviewed the [compatibility and requirements](/docs/streaming-video-&-ads/installation/browser/compatibility-requirements/media-agent-for-browser) for the Media Agent for browser.

## Install the Android video agent [#manual-install]

Ready to get started? Click the <DNT>**Guided install**</DNT> button. Or, if your organization is located in an [EU data center](/docs/using-new-relic/welcome-new-relic/get-started/our-eu-us-region-data-centers), click <DNT>**EU Guided install**</DNT>.
