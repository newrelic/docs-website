---

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents** and type **Cassandra**.

2. Under **Dashboards**, click **Cassandra**.

   <img
     title="Install the Cassandra dashboards"
     alt="Installing the Cassandra dashboards"
     src="/images/infrastructure_screenshot-full_cassandra-dashboard.webp"
   />

3. A popup window opens.

4. Click **Edit** if you want to change the account.

5. Click **Cassandra** or **Skip this step** if you already setup this datasource.

6. Click **View dashboard**, and see your Cassandra data in New Relic.

Cassandra data is attached to the `CassandraSample` and `CassandraColumnFamilySample` [event types](/docs/using-new-relic/data/understand-data/new-relic-data-types#events-new-relic). You can [query this data](/docs/using-new-relic/data/understand-data/query-new-relic-data) for troubleshooting purposes or to create charts and dashboards.

For more on how to find and use your data, see how to [understand integration data](/docs/infrastructure/integrations/find-use-infrastructure-integration-data). You can find a full list of metrics and their attributes at the bottom of this doc.

To recreate [nodetool status](https://docs.datastax.com/en/dse/5.1/dse-dev/datastax_enterprise/tools/nodetool/toolsStatus.html) with NRQL, run this query:

```sql
SELECT cluster.datacenter, fullHostname, cluster.name, cluster.rack, entityName, hostStatus, (db.loadBytes /10e3) AS 'kb db.loadBytes' FROM CassandraSample 
```
