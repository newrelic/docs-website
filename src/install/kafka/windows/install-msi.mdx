1. Download the latest [.exe installer image](https://download.newrelic.com/infrastructure_agent/windows/integrations/nri-kafka/) from our repository.

2. In an admin account, run the install script using an absolute path.

    ```shell
    msiexec.exe /qn /i PATH\TO\nri-kafka.msi
    ```
