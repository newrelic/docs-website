componentType: default
headingText:  Configure the Kafka monitoring integration
---

1. Go to the integrations directory, `C:\Program Files\New Relic\newrelic-infra\integrations.d\`. Run this command to create a copy of the sample configuration file:

  ```shell
    cp kafka-config.yml.sample kafka-config.yml
  ```

2. Edit the `kafka-config.yml` file by using one of the [`kafka-config.yml` sample files](#examples).
