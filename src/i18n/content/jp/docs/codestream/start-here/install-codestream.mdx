特定の IDE の CodeStream 拡張機能をインストールできます。

<CollapserGroup>
  <Collapser className="freq-link" id="vscode" title="Visual Studio Code">
    VisualStudioCode用の[CodeStream](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream)をダウンロードしてインストールします。拡張機能マーケットプレイスを介して、VisualStudioCodeに直接インストールすることもできます。
  </Collapser>

  <Collapser className="freq-link" id="visual-studio" title="Visual Studio">
    CodeStream for [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22)をダウンロードしてインストールします。 拡張機能マーケットプレイスを介して Visual Studio に直接インストールすることもできます。
  </Collapser>

  <Collapser className="freq-link" id="jetbrains" title="ジェットブレインズ">
    [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream)用のCodeStreamをダウンロードしてインストールします。JetBrainsプラグインメニューからインストールすることもできます。
  </Collapser>
</CollapserGroup>

VS Code を使用している場合、一部の CodeStream 機能には、 [Java](https://marketplace.visualstudio.com/items?itemName=redhat.java) 、 [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp) 、 [PHP](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client) 、 [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python) 、 [Go](https://marketplace.visualstudio.com/items?itemName=golang.go) 、または[Ruby](https://marketplace.visualstudio.com/items?itemName=Shopify.ruby-lsp)の言語サポート拡張機能が必要です。

## サインイン [#signin]

CodeStream 拡張機能を IDE にインストールしたら、New Relic 認証情報を使用してサインインします。
