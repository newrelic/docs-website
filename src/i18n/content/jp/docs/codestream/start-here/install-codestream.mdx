特定の IDE の CodeStream 拡張機能をインストールできます。

<CollapserGroup>
  <Collapser
    className="freq-link"
    id="vscode"
    title="Visual Studio Code"
  >
    VisualStudioCode用の[CodeStream](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream)をダウンロードしてインストールします。拡張機能マーケットプレイスを介して、VisualStudioCodeに直接インストールすることもできます。
  </Collapser>

  <Collapser
    className="freq-link"
    id="visual-studio"
    title="Visual Studio"
  >
    [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22)または[Visual Studio 2019](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs)の CodeStream をダウンロードしてインストールします。また、拡張機能マーケットプレースを介して Visual Studio に直接インストールすることもできます。
  </Collapser>

  <Collapser
    className="freq-link"
    id="jetbrains"
    title="ジェットブレインズ"
  >
    [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream)用のCodeStreamをダウンロードしてインストールします。JetBrainsプラグインメニューからインストールすることもできます。
  </Collapser>
</CollapserGroup>

## サインイン [#signin]

CodeStream 拡張機能を IDE にインストールしたら、New Relic 認証情報を使用してサインインします。
