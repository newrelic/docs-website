コメントを投稿したり問題を提起すると、チームメイトはアクティビティ フィードを介して、場合によっては電子メールで通知を受け取ります。ただし、場合によっては、Microsoft Teams にも共有したい場合があります。これにより、CodeStream にまだ参加していない人や、IDE にあまり時間を費やしていない人にリーチできるようになります。

## Microsoft Teamsへの接続 [#connect]

1. CodeStream のユーザー名メニューの下にある<DoNotTranslate>**Integrations**</DoNotTranslate>ページに移動し、 <DoNotTranslate>**Microsoft Teams**</DoNotTranslate>ボタンをクリックします。

2. 表示されたサインイン コードをコピーし、 <DoNotTranslate>**Connect to Teams**</DoNotTranslate>ボタンをクリックします。

   <img
     title="Connect to Microsoft Teams"
     alt="A screenshot showing the page to connect to Microsoft Teams."
     src={codestreamMsteamsConnect}
   />

3. Teams に移動し、CodeStream アプリの次のポップアップが表示されます。 <DoNotTranslate>**Add**</DoNotTranslate>ボタンをクリックします。 展開してチームまたはチャットに追加するオプションを選択しないでください。 「追加」をクリックするだけです。

   <img
     title="Add"
     alt="A screenshot showing the Add button."
     src={codestreamMsteamsCodestreamAppPage}
   />

4. これにより、CodeStream ボットとのプライベート チャットに移動し、Teams サイドバーに CodeStream のロゴが表示されます。 ボットのウェルカム メッセージの<DoNotTranslate>**Sign In**</DoNotTranslate>ボタンをクリックするか、このチャットに`signin`と入力して送信します。

   <img
     title="Weclome"
     alt="A screenshot showing the bot welcome message."
     src={codestreamMsteamsCodestreamBotWelcome}
   />

5. ログイン トークンをポップアップに貼り付けて、 <DoNotTranslate>**Sign-in**</DoNotTranslate>ボタンをクリックします。

   <img
     title="Token"
     alt="An image showing where to paste the token."
     src={codestreamMsteamsTokenExample}
   />

6. 共有したいチャンネルに移動し、 `@`と入力して、ポップアップで<DoNotTranslate>**Get bots**</DoNotTranslate>を選択します。

   <img
     title="Bots"
     alt="An image showing the popup that appears when typing @"
     src={codestreamMsteamsAddBotButton}
   />

7. リストから CodeStream ボットを選択し (必要に応じて検索できます)、 <DoNotTranslate>**Add**</DoNotTranslate>をクリックします。

8. チャンネルに戻り、 `connect` コマンドを使用して CodeStream ボットを @メンションします。共有したいチャンネルでこれを繰り返します。

   <img
     title="Connect bot"
     alt="An image showing the command to connect a bot."
     src={codestreamMsteamsCodestreamConnectBotMessage}
   />

9. 確認が取れれば、CodeStreamからTeamsに共有する準備が整います。

10. CodeStreamに戻ると、先ほど接続したチャンネルに共有できるようになっています。

    <img
      title="Share to channel"
      alt="An image showing the the CodeStream UI for sharing to a channel"
      src={codestreamMsteamsCodestreamSelectChannel}
    />

## Microsoft TeamsからCodeStreamに参加する [#participate]

Microsoft Teamsに共有すると、チームメイトにコードマークが通知されるだけでなく、チームメイトはそのコードマークを使って自分のIDEに直接ジャンプし、CodeStream上の会話に参加することができます。

/>

### IDEで開く [#open-ide]

<DoNotTranslate>**Open in IDE**</DoNotTranslate>をクリックすると、コードとディスカッションの両方が IDE 内で表示されます。 CodeStream Web ページが表示され、どの IDE を開くかを選択できます。 CodeStream は、次回同じリポジトリからディスカッションを表示するときに選択内容を記憶します。 次に、IDE 内の適切なソース ファイルに移動し、関連するコード ブロックまでスクロールすると、CodeStream ペインにディスカッションが表示されます。 IDE でそのリポジトリを開いていない場合は、CodeStream が自動的にソース ファイルを開きます (CodeStream がインストールされた状態で以前にそのリポジトリを開いていて、その場所がわかっている場合)。

JetBrainsのIDEを使用している場合は、CodeStreamがIDEにディープリンクできるように、 [Toolbox app](https://www.jetbrains.com/toolbox-app/) をインストールしてください。

### GitHub（またはBitbucket、GitLab）での公開 [#open-github]

コードブロックがGitHub、Bitbucket、またはGitLabでホストされているリポジトリのものである場合、このボタンはそのホスティングサービス上の対応するコードブロックに移動します。
