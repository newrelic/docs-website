CodeStream アカウント設定を使用すると、個人アカウントを変更できます。

## アカウント設定 [#account-settings]

<DoNotTranslate>**Account**</DoNotTranslate>メニューは、CodeStream ペインの上部にあるユーザー名メニューの下にあります。プロフィールを表示したり、CodeStream ユーザー名を変更したりするためのオプションがあります。 Gravatar で New Relic のメール アドレスに関連付けられたプロフィール写真がある場合は、CodeStream でも自動的に使用されることに注意してください。

## 通知 [#notifications]

[異常なパフォーマンス](/docs/codestream/observability/transactions)を示すトランザクションは、IDE でトースト通知をトリガーします。 IDE で複数のリポジトリを開いている場合や、それらのリポジトリに関連付けられた複数のサービスがある場合は、異常なパフォーマンスが発生した最初のトランザクションについてのみ通知されます。 IDE のデスクトップ通知は、VS Code または JetBrains IDE を使用している場合にのみ利用できることに注意してください。
