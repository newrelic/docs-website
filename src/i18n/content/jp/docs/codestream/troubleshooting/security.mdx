私たちは、ソース コードが最も貴重で機密性の高い資産の 1 つであることを理解しています。ほとんどの開発チームにとって、これは最も重要な成果物の文字通りの表現です。

コメントや問題を通じてコードのブロックについて議論する場合、CodeStream はコードのスニペットをデータベースに保存する必要があります。CodeStream は、さまざまな理由でそのコード ブロックをデータベースに保存します。たとえば、問題のコードがプッシュされていない場合、それをチームメイトと共有する唯一の方法はそれを保存することです。
