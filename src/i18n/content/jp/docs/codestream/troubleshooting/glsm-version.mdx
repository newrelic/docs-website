GitLab セルフマネージド インスタンスの URL の末尾に`/api/v4/version`を追加すると、結果のページにバージョン情報が表示されます。

たとえば、URL は`http://gitlab.codestream.us/api/v4/version`であり、結果は次のようになります。

```
version:	"13.6.4"
revision:  "2754dab2b1b"
```

CodeStreamのGitLab Self-managedからのマージリクエストとの統合を使用する場合は、バージョン12.10以降が必要です。
