NRDOTコレクターによって収集された包括的なメトリクスを使用して、SQL Serverデータベースのパフォーマンスをモニターします。New Relic UI機能のために、以下の7つのメトリクスが自動的に収集されます。これらは**デフォルトで有効になっており**、直接のSQL Server接続を介して任意のプラットフォームで機能します。

## 利用可能なメトリクス [#default]
