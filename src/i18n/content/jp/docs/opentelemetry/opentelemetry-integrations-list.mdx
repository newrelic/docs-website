New Relic OpenTelemetry Collectorを介して増大するオンホスト統合リストをサポートします。 これらの統合により、オープン スタンダードを使用してインフラストラクチャから高忠実度のメトリクスとログを収集できるようになります。

***

## サポートされる統合

現在、以下のインテグレーションが利用可能です。 特定の設定の詳細とサポートされているバージョンについては、以下のドキュメント リンクを使用してください。

| サービス              | ドキュメント                                                                                                                                             |
| :---------------- | :------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Elasticsearch** | [ドキュメントとセットアップ](/docs/infrastructure/host-integrations/host-integrations-list/elasticsearch-otel/elasticsearch-otel-integration-overview/)         |
| **HAProxy**       | [ドキュメントとセットアップ](/docs/opentelemetry/integrations/haproxy/overview/)                                                                                |
| **カフカ**           | [ドキュメントとセットアップ](https://docs.newrelic.com/docs/infrastructure/host-integrations/infrastructure-integrations-list/kafka-monitoring-integration/)    |
| **Nginx**         | [ドキュメントとセットアップ](https://docs.newrelic.com/docs/infrastructure/host-integrations/infrastructure-integrations-list/nginx-monitoring-integration/)    |
| **RabbitMQ**      | [ドキュメントとセットアップ](https://docs.newrelic.com/docs/infrastructure/host-integrations/infrastructure-integrations-list/rabbitmq-monitoring-integration/) |
