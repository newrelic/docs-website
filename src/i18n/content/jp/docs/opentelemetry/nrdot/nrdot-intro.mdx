New Relic Distribution of OpenTelemetry（NRDOT）とは、New RelicがサポートするOpenTelemetryプロジェクトコンポーネントのディストリビューションの集合を指します。NRDOTコンポーネントは、New Relicの製品およびサービスとシームレスに連携するように設計されており、利用可能な場合は、OpenTelemetryをNew Relicに統合するための推奨される方法です。

<Callout variant="tip">
  現在、NRDOT にはOpenTelemetry [Collector](/docs/opentelemetry/nrdot/nrdot-collector/)のビルド済みディストリビューションが含まれています。言語 API と SDK については、 [APM監視](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-intro/)の開始に関するドキュメントを参照してください。
</Callout>
