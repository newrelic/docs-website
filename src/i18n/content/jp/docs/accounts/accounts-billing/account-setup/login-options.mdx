New Relic ユーザーが New Relic 組織にログインするには、いくつかの方法があります。

* 電子メールとパスワード: デフォルトのログイン方法。ユーザーの電子メール アドレスとパスワードが必要です。
* ソーシャル ログイン: Google、Google Workplace、GitHub、Bitbucket を使用して、New Relic にサインアップしてログインできます。
* シングル サインオン: ID プロバイダーのログイン オプションについては、「 [SAML SSO および SCIM の概要」を](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim)参照してください。

ログインに問題がありますか?[ログインのトラブルシューティング](/docs/accounts/accounts-billing/account-setup/troubleshoot-new-relics-password-email-address-login-problems/)を参照してください。

ユーザーのログイン方法の管理については、 [認証ドメイン](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more)を参照してください。
