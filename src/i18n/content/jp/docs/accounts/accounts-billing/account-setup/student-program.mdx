あなたは、ハードウェアとソフトウェアのテレメトリーの収集と分析について学ぼうとしている学生ですか? これは、New Relic for Students を使用して行うことができます。 これは、データ、ユーザー、New Relic 機能への無料アクセスを提供するプログラムです。

## 利点 [#student-benefits]

資格のある学生は、 New Relic for Studentsアカウントに 2 年間無料でアクセスできます。これには以下が含まれます。

* 毎月 500GB の[データ取り込み](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/)
* 3 フル[プラットフォームのユーザー](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type/)
* 10,000[合成チェック](/docs/synthetics/synthetic-monitoring/getting-started/monitor-limits/)/月
* [Standard Edition](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/)の顧客が利用できる他のすべての機能へのアクセス
* プログラム参加者向けの毎月のニュースレター、ワークショップ、ミートアップ、イベントなどの限定コンテンツへのアクセス

## 受給資格要件 [#student-eligibility]

次のカテゴリのいずれかに該当する場合は、参加できます。

* 居住国の認定された大学またはカレッジに現在在籍している学生
* 現在テクニカル スクール、コーディング スクール、またはブートキャンプに在籍している学生
* 大学、短大、その他の専門学校には通っていないかもしれないが、オンライン学習プラットフォームを使用してソフトウェア開発について学んでいる初期のキャリアの技術者

以下の方は対象外となります。

* 大学、短大、その他の技術機関
* 営利事業
* 非営利組織 (非営利団体向けの個別のプログラムを理解するには[、「善のための観察 (O4G)」を](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/eligibility-guidelines-new-relic-nonprofit-program/)参照してください)

## サインアップ方法 [#sign-up]

GitHub Education の Student Developer Pack を通じて、New Relic for Students に加えて、他の Student Developer Pack パートナーが提供する 80 以上のリソースにアクセスできるようになります。

サインアップするには、次の手順を実行します。

1. GitHub Education にサインアップしていることを確認します。

2. [New Relic で無料アカウント](https://newrelic.com/signup)を作成します。

3. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

   にログインし、左下隅の

   <DNT>
     **Help > Resources > New Relic for Students**
   </DNT>

   をクリックします。

4. <DNT>
     **GitHub Education**
   </DNT>

   をクリックし、手順に従って OAuth を使用して GitHub Student アカウントを New Relic に接続します。

New Relic システムが GitHub からステータスを受信すると、アカウントで利用可能な追加データとユーザーが表示されます。
