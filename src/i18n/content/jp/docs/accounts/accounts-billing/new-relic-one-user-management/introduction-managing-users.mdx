このセクションのドキュメントは、 [New Relic One のユーザーモデル](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) のユーザー管理に関するものです。オリジナルのユーザーモデルのドキュメントについては、 [オリジナルのユーザー管理](/docs/accounts/original-accounts-billing/original-users-roles) をご覧ください。

ここでは、New Relic One のユーザーモデルのユーザー管理についてのドキュメントをご紹介します。

* [ユーザー権限の概念](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts): ユーザーが特定の機能や特定のアカウントへのアクセスを得る方法について学びます。
* [ユーザータイプ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type): ベーシックユーザー、コアユーザー、フルプラットフォームユーザーについて学びます。
* [一般的なユーザー管理タスク](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/): ユーザー管理UIの場所、ユーザーの追加方法、ユーザータイプの編集、ユーザーのグループへの追加、カスタムロールの作成などを学びます。
* [SAML SSO と SCIM の紹介](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim): SAML SSO と SCIM のオプションの概要です。
* [認証ドメイン設定](/docs/accounts/accounts-billing/new-relic-one-pricing-users/configure-authentication-domains):"認証ドメイン" は、そのドメイン内のユーザーが New Relic に追加される方法 (手動か SCIM プロビジョニングか)、それらのユーザーが使用する認証方法 (手動ログインか SAML SSO か)、ユーザーがユーザータイプをアップグレードする方法の管理、およびユーザーセッション設定を管理するものです。
* [ユーザー管理のチュートリアル](/docs/accounts/accounts-billing/new-relic-one-user-management/tutorial-add-new-user-groups-roles-new-relic-one-user-model): ユーザー管理タスクのより詳細なチュートリアル。
* [ユーザー管理UIの動画](/docs/accounts/accounts-billing/new-relic-one-user-management/user-mgmt-videos): ユーザー管理UIの動作を動画でご覧いただけます。

ユーザー数が課金に与える影響を理解したいですか？ [ユーザー関連の課金について](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-one-pricing-billing#user-count) をご覧ください。
