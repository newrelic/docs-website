このセクションのドキュメントは[、新しいユーザー モデル](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models)でのユーザーの管理に関するものです。オリジナルのユーザー モデルに関するドキュメントについては、[オリジナルのユーザー管理](/docs/accounts/original-accounts-billing/original-users-roles)を参照してください。

最も重要なユーザー管理ドキュメントの一部を次に示します。

* [ユーザー アクセスの仕組み](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts): ユーザーが機能やアカウントにアクセスする方法について学びます。
* [ユーザータイプ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type): ベーシックユーザー、コアユーザー、フルプラットフォームユーザーについて学びます。
* [UI と一般的なタスク](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/): ユーザー管理 UI、ユーザーの追加方法、ユーザー タイプの編集方法、グループへのユーザーの追加方法、カスタム ロールの作成方法、およびその他の一般的なタスクについて学習します。
* [シングル サインオンと SCIM プロビジョニング](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim)の概要 : SAML SSO の概要と、ID プロバイダーからユーザーを管理する方法。
* [認証ドメイン設定](/docs/accounts/accounts-billing/new-relic-one-pricing-users/configure-authentication-domains):"認証ドメイン" は、そのドメイン内のユーザーが New Relic に追加される方法 (手動か SCIM プロビジョニングか)、それらのユーザーが使用する認証方法 (手動ログインか SAML SSO か)、ユーザーがユーザータイプをアップグレードする方法の管理、およびユーザーセッション設定を管理するものです。
* [ユーザー管理チュートリアル](/docs/accounts/accounts-billing/new-relic-one-user-management/tutorial-add-new-user-groups-roles-new-relic-one-user-model): 一般的なユーザー管理タスクに関する詳細なレッスン。
* [ユーザー管理に関する](/docs/accounts/accounts-billing/new-relic-one-user-management/user-mgmt-videos)動画 : ユーザー管理 UI の動画をご覧ください。

ユーザー数が課金に与える影響を理解したいですか？ [ユーザー関連の課金について](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-one-pricing-billing#user-count) をご覧ください。
