効果的なコスト管理は組織にとって非常に重要であり、New Relic は CCU の使用状況を詳細に可視化します。ただし、予算を集計して予算内に収めるための積極的なメカニズムを実装すると、組織全体でコストを効果的に管理できるようになります。

## 組織レベルの予算制限を管理する

<Steps>
  <Step>
    ### 予算計算にアクセスする

    **[one.newrelic.com](https://one.newrelic.com/)**にアクセスします。[ユーザー メニュー](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)で、 **Administration** &gt; **Compute Budgets** をクリックします。

    <Callout title="重要">
      **Compute Budgets** \[計算予算]ランディング ページを開くと、**Organization Budget** \[組織予算が]デフォルトで表示されます。この予算は購入した SKU を反映し、毎月更新されます。これらの SKU にわたる計算消費単位 (CCU) コミットメントの明確な概要が提供され、計算使用量を効果的に管理および監視するのに役立ちます。
    </Callout>
  </Step>

  <Step>
    ### 予算制限を管理する

    **Organization Budget** \[組織の予算]セクションで、**Manage limits** \[制限の管理を]クリックします。**Edit Organization Budget** \[組織予算の編集]ページが表示されます。
  </Step>

  <Step>
    ### 月間予算制限を管理する

    1. 組織の支出 を更新するには、 **Set monthly budget limits** \[月間予算の上限を設定する]セクションに移動し、 `Org budget`で追跡する計算 CCU の量を指定します。

    <Callout variant="important">
      デフォルトでは、月々の契約義務に基づいて設定されます。
    </Callout>

    2. **Next**\[次へ]をクリックします。

    <img title="Screenshot displaying the Set monthly budget limits page with advanced and core CCU limits highlighted" alt="Screenshot displaying the Set monthly budget limits page with advanced and core CCU limits highlighted" src="/images/set-monthly-budget-limits.webp" />
  </Step>

  <Step>
    ### 通知を設定する

    1. 超過を防ぎ、コストを効果的に管理するには、 **Set up organization budget notifications** \[組織の予算通知の設定]ページに移動してください。閾値のパーセンテージを設定し、通知先を選択します。

    2. 特定の月の予算に割り当てられた CCU の特定の割合を使用したときに警告を受け取るには、 **Thresholds** \[使用済み]タブに移動します。 **+ Add a threshold** \[+ 閾値を追加]を選択し、通知を受け取る割合を入力します。

    3. 予算が閾値を超えたときに通知が確実に送信されるようにするには、 **Destinations** \[宛先]タブに移動します。 通知を受け取る人を少なくとも 1 人追加します。以下のオプションから選択できます。

       * **メール**: 予算について通知する New Relic ユーザーのメール アドレスを入力します。
       * **Webhook** : 特定のエンドポイントに通知を送信するための Webhook URL を入力します。

    4. **Finish**［完了］をクリックします。
  </Step>
</Steps>
