New Relic顧客の場合は、**Usage Summary**ページで使用状況を確認できます。 数字は概算ですが、これらのビューを使用すると、消費が請求額にどのように影響するかについて大まかな見当をつけることができます。

使用状況 UI を表示するには、 **[one.newrelic.com](https://one.newrelic.com) &amp;gt; (ユーザー名をクリック) &amp;gt; Administration &amp;gt; Usage Summary**に移動します。 このページには使用状況の概要が表示されます。

<img title="New Relic usage UI" alt="New Relic usage UI" src="/images/accounts_screenshot-full_usage-ui.webp" />

<figcaption>
  New Relic の使用 UI
</figcaption>

使用状況 UI の使用に加えて、クエリを実行して使用状況を評価することもできます。 詳細については、 [「使用状況データのクエリとアラート」](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts/)を参照してください。
