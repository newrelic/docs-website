New Relic顧客の場合は、<DNT>**Plan & usage**</DNT> ページで使用状況を確認できます。 数字は概算ですが、これらのビューを使用すると、消費が請求額にどのように影響するかについて大まかな見当をつけることができます。

使用状況 UI を表示するには、 <DNT>**one.newrelic.com**</DNT>に移動し、ユーザー メニューをクリックして、 <DNT>**Administration**</DNT>をクリックします。 <DNT>**Plan & usage UI**</DNT>ページには、使用状況の概要が次のように表示されます。

  title="New Relic usage UI"
  alt="New Relic usage UI"
  src="/images/accounts_screenshot-full_usage-ui.webp"
/>

  New Relic の使用 UI
</figcaption>

使用状況 UI の使用に加えて、クエリを実行して使用状況を評価することもできます。 詳細については、 [「使用状況データのクエリとアラート」](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts/)を参照してください。
