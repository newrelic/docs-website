請求および使用状況に関連する UI を見つけるには、 **[one.newrelic.com](https://one.newrelic.com)**にアクセスしてください。 [ユーザー メニュー](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)をクリックし、次に **Administration**\[管理]をクリックします。 **Plan & usage**\[プランと使用量] UI ページが表示され、使用量の概要が表示されます。

<img
  title="New Relic usage UI"
  alt="New Relic usage UI"
  src={accountsUsageUi}
/>

<figcaption>
  **[one.newrelic.com](https://one.newrelic.com) > User menu > Administration > Plan & usage:** **Plan & usage**\[プランと使用量] ページには、使用量に関する詳細が表示されます。そこから、使用状況に関する詳細を取得できます。
</figcaption>

## 請求はどのように行われますか? [#billing-factors]

請求要因については、 [New Relic の価格設定](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing)を参照してください。

## 使用状況を管理する [#manage-usage]

データを管理する方法については、 [「データを管理する」](/docs/data-apis/manage-data/manage-data-coming-new-relic)を参照してください。ユーザーを管理する方法については、 [「ユーザーを管理する」](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-managing-users)を参照してください。
