New Relic プラットフォーム UI の左下にはユーザー メニューがあり、組織とユーザーの管理、データの管理などに関連する機能にアクセスできます。

## アカウント設定を見つける [#find]

[one.newrelic.com](https://one.newrelic.com)から、New Relic UI の左下に移動し、自分の名前をクリックします。これにより、ユーザー メニュー オプションが開きます。ユーザー メニューのオプションの詳細については、「[ユーザー](#user-menu)メニュー」を参照してください。

## ユーザーメニューの詳細 [#user-menu]

ユーザーメニューのイメージです。画像に続いて、メニューのさまざまな部分で何ができるかを説明します。

<img
  title="image of the New Relic user menu"
  alt="image of the New Relic user menu"
  src={accountsUserMenu}
/>

<figcaption>
  New Relic ユーザーメニュー。(ユーザー メニューが異なる場合は、 [アクセス許可の要因](/docs/accounts/accounts-billing/account-structure/factors-affecting-access-features-data)または[古いユーザー モデル](/docs/accounts/original-accounts-billing/original-users-roles/overview-user-models)を使用している可能性があります。)
</figcaption>

ユーザーメニューの内容について詳しく説明します。**ユーザー メニューは変更される可能性があることに注意してください。これは 2023 年 3 月 9 日に最終更新されました。**

1. ユーザー名と電子メール、および[ユーザー タイプ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type)。
2. **ユーザー設定**: 名前、[メール設定](/docs/accounts/accounts/account-maintenance/account-email-settings/)、[タイム ゾーン](/docs/accounts/accounts-billing/general-account-settings/default-time-zone-setting)などのプロファイル設定。
3. **API キー**: [New Relic API キー](/docs/apis/intro-apis/new-relic-api-keys)を管理します。
4. **プランの管理**: [請求と使用](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing)に関連する UI に移動します。
5. **管理**: [ユーザーの管理](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks#where)、 [組織とアカウントの管理](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure)などを行います。 [この UI のビデオ ツアーを参照してください](#video-tour)。
6. **テーマ**: ライト モードとダーク モードの表示を制御します。
7. **NRQL コンソール**: [NRQL](/docs/query-your-data/explore-query-data/query-builder/nrql-console)コンソールのトグル。
8. **データの管理: データ**を[理解し、管理し、制御する](/docs/data-apis/manage-data/manage-your-data)方法を学びます。
9. **その他のユーザー**:[複数のユーザー レコード](/docs/accounts/accounts-billing/account-setup/multiple-logins-found)がある場合は、それらを切り替えることができます。

## ユーザーの追加 [#add-users]
