## ユーザー名の変更 [#user-name]

ユーザー名を変更するには

1. [ユーザー メニュー](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)から \[**ユーザー設定**] を選択します。
2. 右側で「 **フルネーム** 」をクリックします。

ユーザーが[自動ユーザー管理](/docs/accounts/accounts/automated-user-management/automated-user-provisioning-single-sign)を介して管理されている場合、UIから名前を変更することはできないことに注意してください。

## アカウント名の変更 [#account-name]

アカウント構造の詳細については、 [「組織とアカウント構造」](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure)を参照してください。

UI からアカウントを追加および名前変更するには:

1. [アクセス管理UI](https://one.newrelic.com/admin-portal/organizations/organization-detail)から、\[**アカウント**]をクリックします。

2. 名前を変更するアカウントの をクリックします。

   <Icon name="fe-more-horizontal"/>

   をクリックし、 **Edit account** \[アカウントの編集]をクリックします。

このための API の使用方法については、 [NerdGraph ドキュメントを](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph)参照してください。

アカウント名は、名前ではなく [アカウント ID](/docs/accounts/accounts-billing/account-structure/account-id)によって区別されます。このため、組織は同じ名前の複数のアカウントを持つことができます。

アカウント名を変更できない場合、次の理由が考えられます。

* 適切な [権限が](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#admin-settings)ありません
* あなたは当社のオリジナルのユーザー モデルを使用しています。その場合は、ユーザー メニューから利用できる **Account settings** \[アカウント設定] UI を使用します。
* パートナーシップ アカウント構造を持っている: その場合は[、パートナーシップ API](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-account-object)を使用します。
