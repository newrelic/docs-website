このドキュメントには、New Relicのパスワード要件に関する情報が含まれています。

## パスワードを変更する [#password]

New Relicアカウントのパスワードには期限切れがありません。ただし、ユーザーは独自のパスワードや他の個人アカウント情報をいつでも変更できます。

UI からパスワードを変更できるかどうかは、ユーザーおよび組織レベルの設定によって異なります。 UI からパスワードを変更するには: <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT>から[ユーザー メニュー](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)をクリックし、 <DNT>**User preferences**</DNT>を選択します。

パスワードは次の最小要件を満たす必要があります：

* 8～50文字
* 1文字以上（a-z、A-Z）
* 1つ以上の数字（0-9）、特殊文字、またはスペース

<Callout variant="important">
  当社の要件のほかに、パスワードの長さ、大文字または小文字の使用、数字、および特殊文字について組織のガイドラインに従ってください。
</Callout>

## パスワードをリセットする [#reset]

パスワードを忘れた場合、またはパスワードのリセットをリクエストする必要がある場合は、当社の[セルフサービスのオプション](/docs/accounts-partnerships/accounts/account-setup/troubleshoot-password-email-address-login-problems#forgot-password)を使用できます。管理者は、その他のユーザーに代わりパスワードをリセットできません。他のユーザーのパスワードをリセットする必要がある場合は、[support.newrelic.com](https://support.newrelic.com)でサポートを受けてください。

## ユーザーのパスワード管理 [#manage-passwords-other-users]

管理者は他のユーザーのパスワードをリセットまたは変更することはできませんが、古いパスワードが使用されないようにする場合はユーザーを削除できます。ユーザーは、UI から、[ 一括ユーザー管理](/docs/accounts/original-accounts-billing/original-users-roles/users-roles-original-user-model/#deleting)または[API](/docs/accounts/original-accounts-billing/original-users-roles/users-roles-original-user-model/#bulk)を通じて[ 一度に1人ずつ](/docs/apis/nerdgraph/examples/nerdgraph-manage-users/#delete-users)削除できます。
