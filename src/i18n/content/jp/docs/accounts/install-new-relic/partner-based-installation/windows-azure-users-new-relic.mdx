New Relic .NET エージェントの使用に関する詳細は、以下を参照してください。

* [.NETエージェントのAzureへのインストール](/docs/dotnet/installing-the-net-agent-on-azure)
* [Azure Preview Portal](/docs/agents/net-agent/azure-installation/azure-preview-portal)
* [Azure Webサイト](/docs/dotnet/azure-web-sites)
* [Azureクラウドサービス](/docs/dotnet/azure-cloud-services)

New Relic Node.js エージェントのインストールの詳細については、 [Node.js agent on Microsoft Azure](/docs/nodejs/nodejs-agent-on-microsoft-azure) を参照してください。
