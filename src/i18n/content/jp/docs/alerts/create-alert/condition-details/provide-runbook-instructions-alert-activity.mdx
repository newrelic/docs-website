集中[条件に](/docs/alerts/create-alert/create-alert-condition/alert-conditions)は[、ランブックの](/docs/using-new-relic/welcome-new-relic/get-started/glossary#alert-runbook)説明の URL を識別するオプションが含まれています。 これにより、状況が条件の[重大な値](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-thresholds-trigger-alert)に違反したときに開かれる頻繁イベントを処理するための情報または標準手順へのリンクが可能になります。

条件に関連するランブックのURLを編集するには

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Alerts &gt; Alert conditions (policies) &gt; (select a policy) &gt; (select a condition)**</DNT>に移動します。
2. <DNT>**Edit conditions**</DNT>ページで<DNT>**Add details**</DNT>をクリックし、 <DNT>**Runbook URL**</DNT>まで下にスクロールします。 ランブック URL の完全なパス (ランブック URL、内部 Wiki ページへの URL など) を指定します。
3. <DNT>**Enable on save**</DNT>を切り替えて<DNT>**Save**</DNT>をクリックします。
4. オプション: アラート条件から既存のランブック URL を変更または削除するには、 <DNT>**Remove**</DNT>をクリックします。

担当者が正確に何をすべきかわかるように、 <DNT>**Alert event**</DNT>詳細と電子メール通知にランブックの URL を含めます。
