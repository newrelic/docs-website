## 概要

User Sessions \[ユーザーセッション] ページには、すべてのモニター ストリーミング アプリケーションおよびプラットフォームにわたる個々の表示セッションの詳細が表示されます。 このページでは、ユーザーの視聴行動を追跡し、エラーが発生したセッションを特定し、ストリーミングインフラストラクチャ全体における再生パターンを分析できます。

### User Sessions \[ユーザーセッション]でできること

このページを使って以下のことができます。

* ビデオエージェントがインストールされているすべてのアプリケーションにおける、すべてのユーザー視聴セッションを表示します。
* ビューID、セッションID、ユーザーID、アプリケーション名などのセッションレベルの詳細を追跡します。
* タイトルおよびプラットフォームごとに表示されているコンテンツをモニターします
* エラーが発生したセッションを特定してトラブルシューティングを行う
* セッションの開始時刻と終了時刻を表示して視聴時間を分析します。
* 「IDを表示」または「セッションID」をクリックすると、詳細なセッション情報にアクセスできます。
* 特定の期間に焦点を当てるために、セッションを時間範囲でフィルタリングします。
* さまざまなデバイスやプラットフォームにおける視聴パターンを分析する
* 各セッションのコンテンツの詳細を確認してください。

### このページは誰が利用していますか？

このページは、ユーザーから報告された問題のトラブルシューティング、プラットフォーム全体にわたる個々の視聴セッションの分析、すべてのストリーミング アプリケーションにわたるユーザーの行動パターンを理解する必要がある顧客サポート チーム、品質保証エンジニア、運用チームに役立ちます。

## User Sessions \[ユーザーセッション]ページ

ストリーミング ビデオと広告の **All Platforms &gt; User Sessions** には、すべてのアプリケーションのすべてのセッションが一覧表示されます。
