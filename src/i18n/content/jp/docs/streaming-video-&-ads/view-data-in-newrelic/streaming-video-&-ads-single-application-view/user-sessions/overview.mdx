ストリーミング ビデオと広告の **Single platform &gt; User Sessions** には、特定のアプリケーション全体のすべてのセッションが一覧表示されます。

## ユーザーセッションページのUIを表示する

**User Session page** \[ユーザー セッション ページの]テーブルには、各セッションのさまざまな情報を表示するための複数の列があります。 次の表は、セッション ページ UI で使用できる各列に関する情報を示しています。

<Callout variant="important">
  セッション テーブルには、テーブル上で選択した時間フィルター内で開始されたすべてのユーザー セッションが表示されます。
</Callout>
