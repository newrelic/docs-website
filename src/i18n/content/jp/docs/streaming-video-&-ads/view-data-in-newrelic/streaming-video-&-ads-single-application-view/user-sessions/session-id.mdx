## ユーザーセッションストリームイベント

ユーザー セッション ストリーミング イベントは、セッション中に生成されるイベントです。 これらのイベントは、セッション中のユーザーの動作を追跡するために使用されます。

ユーザー セッション ページには、セッション中に生成されたイベントと、イベントに関する次のような詳細情報を示すグラフが表示されます。
