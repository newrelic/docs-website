## 概要

再生タイムラインページには、特定のエンティティに対する特定のユーザー視聴セッションに関する詳細情報が表示されます。これには、ゴールデンメトリクス、メタデータ、およびストリーミングイベントの完全なタイムラインが含まれます。このページは、ユーザーがアプリケーション内で視聴している際に何が起こったのかを正確に把握し、再生に関する問題のトラブルシューティングを行うのに役立ちます。

### 再生タイムラインでできること

このページを使って以下のことができます。

* 開始時間、バッファ時間、平均ビットレート、エラーなどのセッションレベルのゴールデン メトリクスを表示します
* 特定の視聴セッションを識別する固有のセッションIDにアクセスしてください。
* エンティティ情報、プラットフォームの詳細、およびセッションメタデータを確認します。
* モニターセッションの長さと視聴セッションの合計時間
* 再生中に発生したビットレートの変動を分析する
* PLAYER\_READYからCONTENT\_ENDまでのストリーミングイベントの完全なタイムラインを表示します。
* バッファリングイベント（CONTENT\_BUFFER\_STARTおよびCONTENT\_BUFFER\_END）を追跡します。
* セッション中の一時停止および再開イベントを特定する
* 再生中に 30 秒ごとに送信されるコンテンツのハートビート イベントを監視する
* セッション中に発生したエラーを詳細なイベント情報に基づいて診断します。

### このページは誰が利用していますか？

このページは、アプリケーション開発者のアプリ内でのトラブルシューティングの問題、特定の視聴セッションに関するユーザーの苦情を解決する顧客サポート チーム、および完全なイベント タイムラインを調べて再生の問題を調査する品質保証エンジニアに役立ちます。
