## 構文

```js
  nrSendHttpResponse(nr as Object, _url as String, msg as Object) as Void
```

## 説明

ConnectedDeviceSystem タイプの HTTP\_RESPONSE イベントを送信します。

## 戻る

何もない。

## 例

```js
  	msg = wait(5000, m.port)
	  if type(msg) = "roUrlEvent" then
		        nrSendHttpResponse(m.nr, _url, msg)
	  end if
```
