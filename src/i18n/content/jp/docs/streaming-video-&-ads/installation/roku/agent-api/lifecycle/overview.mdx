## 例

### APP\_STARTED イベントを送信

```brightscript
sub Main(aa as Object)
    ...
    nrAppStarted(m.nr, aa)
```

### SCENE\_LOADED イベントを送信

```brightscript
nrSceneLoaded(m.nr, "MyVideoScene")
```

### システムログメッセージを処理する

```brightscript
while (true)
    msg = wait(0, m.port)
    if nrProcessMessage(m.nr, msg) = false
        if type(msg) = "roPosterScreenEvent"
            if msg.isScreenClosed()
                exit while
            end if
        end if
    end if
end while
```
