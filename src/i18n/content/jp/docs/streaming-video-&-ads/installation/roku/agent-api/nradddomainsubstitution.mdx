## 構文

```js
  nrAddDomainSubstitution(nr as object, pattern as String, subs as String) as Void
```

## 説明

* ドメイン属性に一致するパターンを追加し、別の文字列に置き換えます。
  * イベントまたはメトリクスがドメイン プロパティを使用して生成されるたびに、エージェントはそれが正規表現と一致するかどうかを確認し、指定された置換を適用します。
* パターンが設定されていない場合は、URL ドメインを変更せずに使用します。
  * これは、「ドメイン」プロパティを含むすべてのイベントおよびメトリクスに適用されます。

## 例

```js
  	nrAddDomainSubstitution(nr, "^.+\.my\.domain\.com$", "mydomain.com")
```
