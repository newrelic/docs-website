## 例

### カスタム属性の設定

```js
  	nrSetCustomAttribute(m.nr, "myNum", 123, "CONTENT_START")
	  nrSetCustomAttribute(m.nr, "myString", "hello")
```

### nrSetカスタム属性リスト

```js
  	attr = {"key0":"val0", "key1":"val1"}
	  nrSetCustomAttributeList(m.nr, attr, "CONTENT_HEARTBEAT")
```
