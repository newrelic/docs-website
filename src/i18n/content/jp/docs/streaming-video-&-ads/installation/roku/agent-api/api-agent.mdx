New Relic Roku エージェントは、Roku アプリケーションのシステムとビデオ イベントを監視するように設計されています。 システム、ビデオ、エラー、広告、カスタムイベントなど、さまざまなタイプのイベントを追跡するための包括的なAPIを提供します。 このドキュメントでは、API New RelicRoku エージェントとの対話に使用できる メソッドの概要を説明します。
