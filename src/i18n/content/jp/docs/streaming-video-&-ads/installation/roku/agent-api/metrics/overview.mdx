## メトリクス手法

<table>
  <thead>
    <tr>
      <th width="30%">
        方法
      </th>

      <th>
        説明
      </th>

  </thead>

  <tbody>
    <tr>
      <td>
        [nrSendMetric](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendmetric)
      </td>

      <td>
        ゲージ メトリクスを記録します。 時間の経過とともに増加または減少する値を表します。
      </td>
    </tr>

    <tr>
      <td>
        [nrSendCountメトリック](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendcountmetric)
      </td>

      <td>
        カウント メトリクスを記録します。 時間間隔中に発生したイベントの回数を測定します。
      </td>
    </tr>

    <tr>
      <td>
        [nrSendSummaryMetric](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendsummarymetric)
      </td>

      <td>
        メトリクスの概要を記録します。 事前に集計されたデータ、または集計された個別のイベントに関する情報を報告するために使用されます。
      </td>
    </tr>
  </tbody>
</table>

## 例

### nrSendMetric

```js
  	nrSendMetric(m.nr, "test", 11.1, {"one": 1})
```

### nrSendCountメトリック

```js
  	nrSendCountMetric(m.nr, "test", 250, 1500, {"one": 1})
```

### nrSendSummaryMetric

```js
  	nrSendSummaryMetric(m.nr, "test", 2000, 5, 1000, 100, 200)
```
