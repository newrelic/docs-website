## 構文

```js
  nrSendHttpRequest(nr as Object, urlReq as Object) as Void
```

## 説明

ConnectedDeviceSystem タイプの HTTP\_REQUEST イベントを送信します。

## 戻る

何もない。

## 例

```js
  	urlReq = CreateObject("roUrlTransfer")
	  urlReq.SetUrl(_url)
```
