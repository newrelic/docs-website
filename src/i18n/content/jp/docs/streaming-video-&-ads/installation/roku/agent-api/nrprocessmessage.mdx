## 構文

```js
  nrProcessMessage(nr as Object, msg as Object) as Boolean
```

## 説明

システム ログメッセージを確認して処理し、適切なイベントを送信します。

## 戻る

msg がシステム ログメッセージの場合は True、それ以外の場合は False。

## 例

```js
  	while (true)
		      msg = wait(0, m.port)
		      if nrProcessMessage(m.nr, msg) = false
			            if type(msg) = "roPosterScreenEvent"
				                  if msg.isScreenClosed()
					                        exit while
				                  end if
			            end if
		      end if
	  end while
```
