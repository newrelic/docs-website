ビデオ エージェントをインストールするには、ガイド付きインストールに従うことをお勧めします。

## 前提条件 [#requirements]

ビデオ エージェントをインストールする前に、New Relic[アカウント ID](/docs/accounts/accounts-billing/account-setup/account-id)があることを確認してください。

## ビデオエージェントをインストールする [#installation]

ストリーミング ビデオと広告を開始するには、ストリーミング メディア アセットにビデオ エージェントをインストールします。

<Steps>
  <Step>
    <DNT>**[one.newrelic.com](https://one.newrelic.com) &amp;gt; Integrations &amp;amp; Agents &amp;gt; Streaming Video &amp;amp; Ads**</DNT>に移動します。
  </Step>

  <Step>
    ドロップダウンから操作するアカウントを選択し、 **Continue** \[続行]をクリックします。
  </Step>

  <Step>
    ## ストリーミングアプリの種類を選択 [#select-streaming-app-type]

    * [Browser](/install/browser-media) : ブラウザ アプリケーションでビデオ プレーヤーを監視します。

    * モバイル: 前述のプラットフォーム上のビデオ プレーヤーを監視します:

      * [Android](/install/android) : Androidアプリケーションでビデオ プレーヤーを監視します。
      * [iOS](/install/media-ios) : Androidアプリケーションのビデオ プレーヤーを監視します。

    * [Roku](/docs/streaming-video-&-ads/installation/roku/get-started-with-roku) : Roku アプリケーションのビデオプレーヤーを監視します。
  </Step>
</Steps>
