Streaming Video &amp; Ads Agent をインストールするには、次の手順に従ってください。

## 前提条件 [#requirements]

ビデオエージェントをインストールする前に：

* New Relic[アカウントID](/docs/accounts/accounts-billing/account-setup/account-id)をお持ちであることを確認してください。
* サポートされているプラットフォームとバージョンの[互換性要件](/docs/streaming-video-&-ads/installation/compatibility-requirements)を確認してください。

## ビデオエージェントをインストールする [#install]

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents &gt; Streaming Video &amp; Ads**</DNT>に移動します。
2. ドロップダウンメニューからアカウントを選択し、 <DNT>**Continue**</DNT>をクリックしてください。
3. プラットフォーム固有のインストレーションを続行するには、プラットフォームを選択してください:
