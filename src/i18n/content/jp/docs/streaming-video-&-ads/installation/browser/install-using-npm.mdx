JavaScript アプリケーションの npm パッケージマネージャを使用してビデオエージェントをインストールします。

## あなたが始める前に

最初に[一般的なインストール手順](/docs/streaming-video-&-ads/installation/browser/get-started-with-browser#install)を完了します。

## npmを使用してインストールします [#install-npm]

<Steps>
  <Step>
    ### アプリケーション名

    アプリケーションに名前を付けて、 <DNT>**Save and Continue**</DNT>をクリックしてください。
  </Step>

  <Step>
    ### エージェントのインストール

    以下のコードスニペットをコピーして、ターミナル（Mac）またはコマンドプロンプト（Windows）で実行することで、New Relicの最新リリースをインストールできます。

    <Callout variant="important">
      ビデオエージェントをインストールするには、 **npm**または**Yarn**を選択してください。
    </Callout>
  </Step>

  <Step>
    ### アプリケーションにコードを追加する

    スクリプトをダウンロードしたら、プラットフォームに表示されているコードスニペットをアプリケーションに組み込んで、ビデオプレーヤーの監視を開始してください。
  </Step>

  <Step>
    ### アプリケーションIDを生成する

    <DNT>**Get Application Info**</DNT>を選択してください。これにより、ブラウザエンティティ生成のためのアプリケーションIDが生成されます。
  </Step>

  <Step>
    ### 自分のデータを見る

    <DNT>**See your data**</DNT>をクリックします。
  </Step>
</Steps>

<Callout variant="important">
  アプリケーションとのやり取りやデータ生成には数分かかる場合があります。
</Callout>
