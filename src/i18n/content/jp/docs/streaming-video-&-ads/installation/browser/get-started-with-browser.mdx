ブラウザ アプリケーションでビデオ プレーヤーのパフォーマンスを監視するには、npm または JavaScript スニペットを使用してNew Relicビデオエージェントをインストールします。

## 前提条件 [#before-start]

Browser用ストリーミング ビデオ &amp; 広告エージェントをインストールする前に、アプリケーションが次の要件を満たしていることを確認してください。

* [ブラウザ監視の互換性要件](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring)
* [ストリーミングビデオと広告エージェントの互換性要件](/docs/streaming-video-&-ads/installation/compatibility-requirements)

## エージェントのインストール [#install]

<Steps>
  <Step>
    ### NewRelicにアクセスする

    <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT>に移動し、統合マーケットプレイスに移動します。
  </Step>

  <Step>
    ### ストリーミングビデオと広告を選択

    <DNT>**Streaming Video &amp; Ads**</DNT>タイルを検索して選択します。
  </Step>

  <Step>
    ### アカウントを選択

    ドロップダウンから<DNT>**Account**</DNT>を選択し、 <DNT>**Continue**</DNT>をクリックしてください。
  </Step>

  <Step>
    ### プラットフォームを選択してください

    <DNT>**Browser**</DNT>を選択し、 <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    ### ビデオプレーヤーを選択してください

    お使いのビデオプレーヤーを選択し、 <DNT>**Continue**</DNT>をクリックしてください。
  </Step>

  <Step>
    ### 設置方法を選択してください

    <DocTiles>
      <DocTile title="npmを使用してインストールします" path="/docs/streaming-video-&-ads/installation/browser/install-using-npm">
        JavaScript アプリケーションのビデオエージェントパッケージマネージャをインストールします。
      </DocTile>

      <DocTile title="フロントエンドコードにJavaScriptスニペットを配置する" path="/docs/streaming-video-&-ads/installation/browser/install-using-snippet">
        アプリケーションにJavaScriptコードスニペットを含めることで、ビデオエージェントをインストールできます。
      </DocTile>
    </DocTiles>
  </Step>
</Steps>

## 代替案：ガイド付きインストール

上記の手動手順はスキップして、ガイド付きインストーラーを使用してください。下の<DNT>**Guided install**</DNT>クリックしてください。または、組織が[EUのデータセンター](/docs/using-new-relic/welcome-new-relic/get-started/our-eu-us-region-data-centers)にある場合は、 <DNT>**EU Guided install**</DNT>をクリックしてください。

## 次は何ですか？

エージェントのインストール後：

1. ストリーミングビデオと広告ダッシュボードでアプリケーションをチェックして、 New Relicにデータが流れていることを確認します。
2. [データを確認する](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-features)：監視機能とダッシュボードを活用する
3. [Quality of Experience の設定](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): アプリケーションのパフォーマンス メトリクスをカスタマイズする
