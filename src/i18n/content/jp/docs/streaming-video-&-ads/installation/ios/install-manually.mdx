このページでは、Streaming Video &amp; Ads Agent を手動でインストールする場合のインストール手順の概要を説明します。 コード例を含む完全な説明については、[ガイド付き](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios)インストールを使用してください。

<Steps>
  <Step>
    [video-エージェント-iOS](https://github.com/newrelic/video-agent-ios)リポジトリのクローンを作成します。
  </Step>

  <Step>
    `.xcodeproj`のファイルをそれぞれXcodeで開きます。
  </Step>

  <Step>
    適切なプランを選択してください。
  </Step>

  <Step>
    ビルド（CMD+B）。
  </Step>

  <Step>
    生成された`.framework`のファイルをプロジェクトに含めてください。
  </Step>

  <Step>
    <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    ### ビデオトラッカーを初期化します

    実装に合ったエージェントタイプを選択してください。

    * **AVPlayerトラッカーのみを搭載したビデオエージェント**：ビデオ再生の追跡用
    * **AVPlayerとIMAトラッカーを搭載したビデオエージェント**：ビデオ再生と広告トラッキング用
  </Step>

  <Step>
    <DNT>**See your data**</DNT>をクリックすると、New Relicでストリーミングデータが表示されます。
  </Step>
</Steps>
