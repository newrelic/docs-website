このページでは、CocoaPods を使用して Streaming Video &amp; Ads エージェントをインストールする場合のインストール手順の概要を説明します。 コード例を含む完全な説明については、[ガイド付き](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios)インストールを使用してください。

<Steps>
  <Step>
    ### 依存関係を追加する

    Podfileにビデオエージェントのポッドを追加してください。
  </Step>

  <Step>
    ### ポッドをインストールする

    ターミナルで`pod install`実行してください。
  </Step>

  <Step>
    ### プロジェクトの構築

    **Xcode**でプロジェクトを開いてビルドしてください。
  </Step>

  <Step>
    <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    ### ビデオトラッカーを初期化します

    実装に合ったエージェントタイプを選択してください。

    * **AVPlayerトラッカーのみを搭載したビデオエージェント**：ビデオ再生の追跡用
    * **AVPlayerとIMAトラッカーを搭載したビデオエージェント**：ビデオ再生と広告トラッキング用
  </Step>

  <Step>
    <DNT>**See your data**</DNT>をクリックすると、New Relicでストリーミングデータが表示されます。
  </Step>
</Steps>
