このページでは、AAR ファイルを使用して Streaming Video &amp; Ads エージェントを手動でインストールする場合のインストール手順の概要を説明します。 コード例を含む完全なインストレーションについては、[ガイド付き](/docs/streaming-video-&-ads/installation/android/get-started-with-android)インストールを使用してください。

<Steps>
  <Step>
    [video-エージェント-androidリポジトリ](https://github.com/newrelic/video-agent-android)のクローンを作成します。
  </Step>

  <Step>
    Android Studioで開いてください。
  </Step>

  <Step>
    <DNT>**View &gt; Tool Windows &gt; Gradle**</DNT>に移動してGradleツールウィンドウを開きます。
  </Step>

  <Step>
    Gradleツールウィンドウで、 <DNT>**NRVideoProject &gt; Tasks &gt; build**</DNT>を展開し、 <DNT>**assemble**</DNT>をダブルクリックします。これによりプロジェクトがビルドされ、各モジュールのフォルダ<DNT>**build &gt; outputs &gt; aar**</DNT>内に AAR ライブラリ ファイルが生成されます。
  </Step>

  <Step>
    プロジェクトで、 <DNT>**File &gt; New &gt; New Module**</DNT>に移動し、 `.JAR/.AAR`パッケージをインポートして、 <DNT>**Next**</DNT>をクリックします。
  </Step>

  <Step>
    前の手順で生成した AAR ファイルを選択し、 <DNT>**Finish**</DNT>をクリックします。
  </Step>

  <Step>
    必要なAARファイルごとに、手順5と6を繰り返してください。

    * **NewRelicVideoCore.aar** （必須）：コアビデオエージェント機能
    * **NRExoPlayerTracker.aar** （必須）：ExoPlayerトラッキング
    * **NRIMATracker.aar** （オプション）：IMA SDKを使用した広告トラッキング
  </Step>

  <Step>
    アプリモジュールの`build.gradle`ファイルに、以下の依存関係を追加してください。

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")   // Required: Core video agent
      implementation project(":NRExoPlayerTracker")  // Required: ExoPlayer tracking
      implementation project(':NRIMATracker')        // Optional: Ad tracking with IMA
    }
    ```

    注：広告追跡機能が必要な場合のみ、 `NRIMATracker`を含めてください。
  </Step>

  <Step>
    通知バーの<DNT>**Sync Now**</DNT>クリックするか、 <DNT>**File &gt; Sync Project with Gradle Files**</DNT>に移動して、プロジェクトをGradleファイルと同期します。
  </Step>

  <Step>
    ### ビデオエージェントを初期化します

    アプリケーションの`onCreate()`メソッドまたはアクティビティで、New Relicビデオエージェントを初期化します。詳細なコード例と実装手順については、[ガイド付きインストール](/docs/streaming-video-&-ads/installation/android/get-started-with-android)を参照してください。
  </Step>

  <Step>
    ### ビデオプレーヤーのトラッキングを設定する

    ニーズに合わせてトラッカーを設定してください。

    * **ExoPlayerトラッカーのみ**：ビデオ再生監視のためにExoPlayerトラッキングを設定します。
    * **ExoPlayerとIMAトラッカー**：動画再生と広告トラッキングのために、ExoPlayerとIMAの両方のトラッキングを設定します。

    <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />

    完全な実装コード例については、[ガイド付きインストールを](/docs/streaming-video-&-ads/installation/android/get-started-with-android)参照してください。
  </Step>

  <Step>
    アプリケーションを構築してデプロイすると、New Relicでビデオ追跡データを確認できます。
  </Step>
</Steps>
