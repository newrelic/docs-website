このページでは、JitPack を使用して Streaming Video &amp; Ads エージェントをインストールする場合のインストール手順の概要を説明します。 コード例を含む完全な説明については、[ガイド付き](https://onenr.io/02R54MqY0wb)インストールを使用してください。

<Steps>
  <Step>
    ## JitPack リポジトリを追加

    JitPack リポジトリをルートの`build.gradle`ファイルに追加します。
  </Step>

  <Step>
    ## 依存関係を追加する

    アプリの`build.gradle`ファイルにビデオエージェントの依存関係を追加し、 <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    ## ビデオエージェントを設定する

    NRVideo 設定を**アプリケーション クラスまたは MainActivity の**`onCreate`メソッドに追加します。
  </Step>

  <Step>
    ## ビデオプレーヤーのトラッキングを設定する

    実装に合ったエージェントタイプを選択してください。

    * **ExoPlayerトラッカーのみを搭載したビデオエージェント**：ビデオ再生トラッキング用

    * **ExoPlayerとIMAトラッカーを搭載したビデオエージェント**：ビデオ再生と広告トラッキング用

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    <DNT>**See your data**</DNT>をクリックすると、New Relicでストリーミングデータが表示されます。
  </Step>
</Steps>
