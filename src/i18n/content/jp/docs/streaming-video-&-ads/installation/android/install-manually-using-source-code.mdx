このページでは、ソースコードを使用してストリーミング ビデオ &amp; 広告エージェントを手動でインストールする場合のインストール手順の概要を説明します。 コード例を含む完全なインストレーションについては、[ガイド付き](/docs/streaming-video-&-ads/installation/android/get-started-with-android)インストールを使用してください。

<Steps>
  <Step>
    [video-エージェント-androidリポジトリ](https://github.com/newrelic/video-agent-android)のクローンを作成します。
  </Step>

  <Step>
    Android プロジェクトで、 <DNT>**File &gt; New &gt; Import Module**</DNT>に移動します。
  </Step>

  <Step>
    ライブラリモジュールディレクトリの場所を入力し、 <DNT>**Finish**</DNT>をクリックしてください。

    <Callout variant="important">
      ライブラリモジュールディレクトリは、先ほどクローンしたリポジトリ内にあります。モジュールの例: `NewRelicVideoCore` 、 `NRExoPlayerTracker` 、 `NRIMATracker` 。
    </Callout>
  </Step>

  <Step>
    含めたいすべてのモジュールについて、手順2と3を繰り返してください。
  </Step>

  <Step>
    アプリモジュールの`build.gradle`ファイルに、以下を追加してください。

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    ### ビデオプレーヤーのトラッキングを設定する

    実装に合ったエージェントタイプを選択してください。

    * **ExoPlayerトラッカーのみを搭載したビデオエージェント**：ビデオ再生トラッキング用

    * **ExoPlayerとIMAトラッカーを搭載したビデオエージェント**：ビデオ再生と広告トラッキング用

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    <DNT>**See your data**</DNT>をクリックして更新されたコードをデプロイし、データを確認してください。
  </Step>
</Steps>
