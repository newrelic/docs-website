## モバイル機能 [#mobile-app-features]

モバイル アプリでできることの例を次に示します。

* <DNT>
    **Alerts**
  </DNT>

  : 現在のインシデントや過去のインシデントを表示し、インシデントを承認し、タップして関連エンティティのパフォーマンス データを表示します。

* <DNT>
    **Application performance monitoring (APM)**
  </DNT>

  : トランザクション、エラー、データベース、外部サービスなどにわたるリアルタイムおよび履歴データを表示します。

* <DNT>
    **Browser monitoring**
  </DNT>

  : すべてのフロントエンド Web アプリにわたってスループット、読み込み時間、エラー、AJAX リクエストなどを追跡します。

* <DNT>
    **Dashboards and NRQL**
  </DNT>

  : ダッシュボードを表示したり、NRQL でさらに詳しく調べたりできます。 カスタムNRQLクエリを作成したり、チャートをタップしてオーバーフローをタップすることで、任意のチャートのNRQLを表示することができます。

  <Icon name="fe-more-horizontal"/>

  アイコンをクリックして

  <DNT>
    **View query**
  </DNT>

  を選択します。

* <DNT>
    **Explorer**
  </DNT>

  :New Relic AWSサービスからネットワーク パフォーマンスまで、 のその他すべてを参照します。

* <DNT>
    **Infrastructure monitoring**
  </DNT>

  : クラウド、専用ホストで実行されているホスト、またはオーケストレーションされた環境で実行されているコンテナの健全性を確認します。

* <DNT>
    **Mobile monitoring**
  </DNT>

  : すべてのモバイル アプリのパフォーマンス、エラー、クラッシュなどを確認します。

* <DNT>
    **Synthetic monitoring**
  </DNT>

  : 各モニターのパフォーマンスを確認し、個々の結果を表示し、障害を追跡します。

## データプライバシー [#privacy]

当社のモバイル アプリは、認証とトラブルシューティングに必要な情報のみを記録します。

* 名字と名前を含む、New Relic アカウントに関連付けられたユーザーの電子メールアドレス (認証のみを目的とする)
* IPアドレス
* デバイスID

詳細については、当社の[モバイルデータプライバシーおよびセキュリティについてのドキュメント](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps)を参照してください。
