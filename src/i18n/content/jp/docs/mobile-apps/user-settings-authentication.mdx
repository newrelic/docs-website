New Relic モバイルアプリのアカウントを認証する方法や、モバイルデバイスにユーザーを追加したり削除したりする方法について説明します。

## ユーザー認証 [#authentication]

お客様のNew Relicアカウントによっては、New Relicモバイルアプリをインストールする際に、追加のインストールまたは認証手順が必要になる場合があります。

<table>
  <thead>
    <tr>
      <th width={200}>
        **New Relicアカウント**
      </th>

      <th>
        **追加要件**
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        新規ユーザー
      </td>

      <td>
        まだNew Relicのアカウントをお持ちでない方は、こちらをご覧ください。

        1. デスクトップの Web ブラウザから、 [New Relic のアカウントを作成する](https://newrelic.com/signup) 。

        2. [アプリケーション](/docs/site/installing-the-agent) に適切な New Relic エージェントをインストールします。

           新規アカウント設定の一環として、パスワードリセットリンクを記載したメールをお送りします。パスワードリセットリンクは、モバイルアプリの場合、20分後に失効します。
      </td>
    </tr>

    <tr>
      <td>
        既存のNew Relicユーザー
      </td>

      <td>
        追加の要件はありません。サインインすると、あなたのアプリケーション、ホスト、主要な取引が自動的に表示されます。
      </td>
    </tr>

    <tr>
      <td>
        New Relicのパートナーアカウントをお持ちの方
      </td>

      <td>
        パートナーによっては、異なる [認証プロセスを完了する必要があるかもしれません](/docs/ios/ios-with-partners) 。

        **Azure Storeユーザー：** Azure StorefrontとNew Relicの深い統合により、Azure StorefrontユーザーはNew RelicのAndroidやiOSアプリで自分のアカウントにアクセスできません。
      </td>
    </tr>

    <tr>
      <td>
        SAML-SSOが有効なアカウントを持つユーザー
      </td>

      <td>
        New Relic モバイルアプリにサインインすると、セッションは自動的に Web ブラウザにリダイレクトされます。そこからNew Relic SAML-SSOアカウントにサインインすることができます。

        モバイルデバイスで SAML-SSO アカウントを使用しているときにエラーが表示された場合は、デスクトップ Web ブラウザーで**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**にサインインできることを確認してください。

        * ない場合は、管理者に連絡してください。
        * その場合は、 [support.newrelic.com](https://support.newrelic.com) でサポートを受けてください。
      </td>
    </tr>
  </tbody>
</table>

    title="アカウントの切り替え"
  >
    自分のユーザー名に関連するアカウントを切り替えることができます。

    1. **Users** メニューから、ユーザー名を選択します。
    2. アカウント名を選択します。
  </Collapser>

  <Collapser
    id="Remove a username"
    title="ユーザー名の削除・再追加"
  >
    特定のユーザー名を本機から削除するには

    1. **Users** メニューから「 **Logout** 」を選択します。

    2. このデバイスからユーザーを削除するには、ユーザーの赤いマイナス記号を選択します <Icon style={{color: 'red'}} name="fe-minus-circle"/> アイコン。

    3. ユーザーの **Log out** アイコンを選択します。

       ユーザーを再度追加するには、そのユーザー名で再度サインインしてください。
