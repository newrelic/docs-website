DevOps とサイト信頼性エンジニアリングの成功の鍵は、平均解決時間 (MTTR) を短縮することにあります。ウィジェットは、コンピューターの前にいないときでも、これを行うのに役立ちます。

## ウィジェットを使い始める前に [#requirements]

iPhone または iPad で[iOS ウィジェット](https://support.apple.com/en-us/HT207122)を設定するには、iOS 14 以降を実行し、 [New Relic モバイル アプリ](install-newrelic-ios-mobile-app)がインストールされていることを確認してください。

ホーム画面にウィジェットを追加する例を次に示します。

1. アイコンが揺れるまで、空の領域またはウィジェットを長押しします。

2. 左上隅にある<DNT>**+**</DNT>アイコンをタップします。

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the Add (+) button"
     alt="Screenshot showing the Add (+) button"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-edit-home-screen.webp"
   />

3. <DNT>**New Relic**</DNT>ウィジェットを検索します。

4. 左または右にスワイプして 3 つのウィジェット サイズから選択し、 <DNT>**Add Widget**</DNT>をタップします。

   <Video
     type="wistia"
     id="qyk74p7j56"
     vertical
     width="290px"
   />

5. <DNT>**Done**</DNT>タップします。

## ウィジェットにグラフを追加する [#add-chart]

New Relicアプリから表示するチャートをまだ選択していない場合は、次の手順に従います。

1. ウィジェットに表示したいグラフに移動します。

2. チャートをタップして詳細ビューを表示します。

3. 右上隅の省略記号アイコン<DNT>**...**</DNT>をタップすると、さらにオプションが表示されます。

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the dropdown options in the ellipsis"
     alt="Screenshot showing the dropdown options in the ellipsis menu"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-add-widget-from-chart.webp"
   />

4. <DNT>**Add to widgets**</DNT>を選択します。

5. <DNT>**Manage widgets**</DNT>を使用してウィジェットを削除したり並べ替えたりすることもできます。

## ウィジェットのチャートを指定する [#designate-charts]

iOSウィジェットごとに表示するグラフを選択するには：

1. ウィジェットが揺れるまで押し続けます。

2. <DNT>**Select chart**</DNT>またはチャート名をタップします。

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the option to select a chart"
     alt="Screen showing the option to select a chart"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-configure-widget.webp"
   />

3. 必要に応じて、この画面からタイトルや凡例を非表示にすることができます。
