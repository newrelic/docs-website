DevOps とサイト信頼性エンジニアリングの成功の鍵は、平均解決時間 (MTTR) を短縮することにあります。ウィジェットは、コンピューターの前にいないときでも、これを行うのに役立ちます。

## ウィジェットを使い始める前に [#requirements]

iPhone または iPad で[iOS ウィジェット](https://support.apple.com/en-us/HT207122)を設定するには、iOS 14 以降を実行し、 [New Relic モバイル アプリ](install-newrelic-ios-mobile-app)がインストールされていることを確認してください。

ホーム画面にウィジェットを追加する例を次に示します。

1. アイコンが揺れるまで、空の領域またはウィジェットを長押しします。

2. 左上隅の**+**アイコンをタップします。

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the Add (+) button"
     alt="Screenshot showing the Add (+) button"
     src={iOSWidgetsEditHomeScreen}
   />

3. **New Relic**ウィジェットを検索します。

4. 左または右にスワイプして3つのウィジェットサイズから選択し、\[ウィジェットの**追加]**をタップします。

   <Video
     type="wistia"
     id="qyk74p7j56"
     vertical
     width="290px"
   />

5. \[**完了]**をタップします。

## ウィジェットにグラフを追加する [#add-chart]

New Relicアプリから表示するチャートをまだ選択していない場合は、次の手順に従います。

1. ウィジェットに表示したいグラフに移動します。

2. チャートをタップして詳細ビューを表示します。

3. その他のオプションを表示するには、右上隅にある3つのドットをタップします。

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the dropdown options in the three dots"
     alt="Screenshot showing the dropdown options in the three dots menu"
     src={mobileappsiOSWidgetsAddWidgetFromChart}
   />

4. \[**ウィジェットに追加]を**選択します。

5. ウィジェットの**管理を使用**してウィジェットを削除および再配置することもできます。

   <Video
     type="wistia"
     id="7dd2al2g8w"
     vertical
     width="290px"
   />

## ウィジェットのチャートを指定する [#designate-charts]

iOSウィジェットごとに表示するグラフを選択するには：

1. ウィジェットが揺れるまで押し続けます。

2. **\[グラフの選択]**またはグラフ名をタップします。

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the option to select a chart"
     alt="Screen showing the option to select a chart"
