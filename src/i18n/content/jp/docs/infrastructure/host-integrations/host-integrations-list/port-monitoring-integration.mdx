当社のポート監視統合は、TCP、UDP などのネットワーク ポートのステータスを監視し、データを New Relic に報告します。

開始するには、GitHubにある [Port monitoring on-host integration README](https://github.com/newrelic/nri-port-monitor) を参照してください。

<Callout variant="tip">
  この統合は、Github で [オープンソースとして公開されています。また、最新の更新情報として、変更履歴も公開されています。この統合のサポートは、 ](https://github.com/newrelic/nri-port-monitor)[Expert Services のサブスクリプションで利用できます。](https://www.newrelic.com/expertservices) 。
</Callout>
