Dockerコンテナ[統合](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations)を監視し、 [Docker API](https://docs.docker.com/reference/api/engine/version/v1.48/)を使用して Docker コンテナからNew Relicにメトリクスをレポートします。

<Callout variant="tip">
  この統合は[Github 上でオープンソース](https://github.com/newrelic/nri-docker)として公開されています。 最新のアップデートについては、変更ログもそこで確認できます。
</Callout>

<Callout variant="important">
  * この統合は LCOW ( Windows上の Linux コンテナ) をサポートしていません。 詳細については、 [Docker のドキュメント](https://docs.docker.com/desktop/windows/wsl/)を参照してください。
  * Linux の場合 - この統合はインフラストラクチャエージェントにバンドルされており、docker コンテナが検出されると自動的にアクティブになります。
  * Windowsの場合 - バージョン 2.3.0 以降、互換性が提供されていますが、インフラストラクチャに加えて手動によるインストールが必要です。 将来のリリースでは、インフラストラクチャエージェントにバンドルされる予定です。
</Callout>
