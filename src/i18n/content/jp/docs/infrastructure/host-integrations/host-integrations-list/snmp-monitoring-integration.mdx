  title="有効期限（EOL）の通知"
>
  2022年3月をもって、従来のSNMPモニタリング統合を含むいくつかの機能のサポートを終了します。この移行に向けて簡単に準備できる方法を含めた詳細については、 [Explorers Hub の記事](https://discuss.newrelic.com) をご覧ください。
</Callout>

当社の [ネットワーク・パフォーマンス・モニタリング](/docs/network-performance-monitoring/) ソリューションは、 [SNMP データ](/docs/network-performance-monitoring/setup-performance-monitoring/snmp-performance-monitoring/) と [ネットワーク・フロー](/docs/network-performance-monitoring/setup-performance-monitoring/network-flow-monitoring/) の両方を収集します。この機能により、 **Is it the network?** という質問に迅速に答えることができます。
