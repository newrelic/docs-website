Collectd [インテグレーション](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) は、 [Collectd](https://collectd.org/wiki/index.php/Main_Page)-設定されたデバイスからのメトリクスを監視し、New Relic に報告します。

はじめに、GitHubにある [Collectd On-Host Integration README](https://github.com/newrelic/nri-collectd) をご覧ください。

<Callout variant="tip">
  この統合は、Github で [オープンソースとして公開されています。また、最新の更新情報として、変更履歴も公開されています。この統合のサポートは、 ](https://github.com/newrelic/nri-collectd)[Expert Services のサブスクリプションで利用できます。](https://www.newrelic.com/expertservices) 。
</Callout>
