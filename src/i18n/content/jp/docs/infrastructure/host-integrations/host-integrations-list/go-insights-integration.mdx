開始するには、GitHubにある [Go Insights On-Host Integration README](https://github.com/newrelic/go-insights) を参照してください。

  この統合は、Github で [オープンソースとして公開されています。また、最新の更新情報として、変更履歴も公開されています。この統合のサポートは、 ](https://github.com/newrelic/go-insights)[Expert Services のサブスクリプションで利用できます。](https://www.newrelic.com/expertservices) 。
