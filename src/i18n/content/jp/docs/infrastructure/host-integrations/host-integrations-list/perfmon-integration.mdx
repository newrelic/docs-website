PerfMon インテグレーションは、 [Microsoft Performance Monitor](https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc749154(v%3dws.11)) からのクエリとイベントの結果を、New Relic が使用可能な標準出力 (stdout) に提供します。

開始するには、GitHubの [Windows PerfMon / WMI On-Host Integration README](https://github.com/newrelic/nri-perfmon) を参照してください。

<Callout variant="tip">
  この統合は、GitHub で [オープンソースとして公開されています。また、最新のアップデートについては、変更履歴をご覧いただけます。この統合のサポートは、 ](https://github.com/newrelic/nri-perfmon)[Expert Services のサブスクリプションで利用できます。](https://www.newrelic.com/expertservices) 。
</Callout>
