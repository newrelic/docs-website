Unixモニタリングインテグレーションでは、AIX、Linux、macOS、Solaris/SunOSなどのUnixベースのサーバーをシステムレベルでモニタリングし、データをNew Relicに報告することができます。

開始するには、GitHubにある [Unix Monitoring On-Host Integration README](https://github.com/newrelic/newrelic-unix-monitor) を参照してください。

<Callout variant="tip">
  この統合は、Github で [オープンソースとして公開されています。また、最新の更新情報として、変更履歴も公開されています。この統合のサポートは、 ](https://github.com/newrelic/newrelic-unix-monitor)[Expert Services のサブスクリプションで利用できます。](https://www.newrelic.com/expertservices) 。
</Callout>
