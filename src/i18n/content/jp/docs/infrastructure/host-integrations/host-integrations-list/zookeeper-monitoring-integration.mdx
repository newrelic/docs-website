Zookeeper monitoring integration は、 [Apache Zookeeper](https://zookeeper.apache.org/) ノードから報告される重要なパフォーマンスメトリクスを取得し、データを New Relic に報告します。

インストールの手順については、GitHubに掲載されている [Zookeeper On-Host Integration README](https://github.com/newrelic/nri-zookeeper) をご覧ください。

<Callout variant="tip">
  この統合は、Github で [オープンソースとして公開されています。また、最新の更新情報として、変更履歴も公開されています。この統合のサポートは、 ](https://github.com/newrelic/nri-zookeeper)[Expert Services のサブスクリプションで利用できます。](https://www.newrelic.com/expertservices) 。
</Callout>
