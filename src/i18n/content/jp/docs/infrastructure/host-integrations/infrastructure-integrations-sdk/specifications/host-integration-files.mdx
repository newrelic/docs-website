このドキュメント [では、インフラストラクチャ エージェントのオンホスト統合に](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations)使用されるファイルについて説明します。

## 統合ファイル [#overview]

インフラストラクチャ エージェントのオンホスト統合には、少なくとも次のファイルが含まれます。

* インフラストラクチャ エージェントが予期する形式で JSON データをエクスポートする、任意の言語で記述された [実行可能ファイル](/docs/infrastructure/infrastructure-integrations/custom-integrations/create-integration-executable-file)
* [設定ファイル](/docs/create-integrations/infrastructure-integrations-sdk/specifications/host-integrations-standard-configuration-format)
