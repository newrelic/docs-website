## 問題

あなたの統合はデータを報告していますが、いくつかのインベントリ属性が欠けています。

## 解決

以下のインベントリキー名は、当社のプラットフォームで内部的に使用されています。お客様が提供する統合機能が同じキー名のインベントリアイテムを送信した場合、それらはユーザーには見えません。

* `pid`
* `installed_epoch`
* `epoch`
* `time`
* `size`
* `boot_id`
* `product_uuid`
* `installed_time`
