[F5 ](https://support.f5.com/csp/knowledge-center/software/BIG-IP)[オープンソース統合](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) F5 からのパフォーマンスメトリクス（仮想サーバー、プール、ノード、メモリの統計情報を含む）を監視し、データをNew Relic Infrastructureに報告します。

要件、インストール、および構成に関する情報は、GitHub上の [F5 On-Host Integration README](https://github.com/newrelic/nri-f5) を参照してください。

<Callout variant="important">
  New Relic は、この技術のモニタリングを可能にするために、この統合をオープンソース化しました。この統合は現状のまま提供され、保証やサポートはありませんが、GitHub を通じて問題の報告やこの統合への貢献が可能です。この統合のサポートは、 [Expert Services のサブスクリプションでご利用いただけます。](https://www.newrelic.com/expertservices) 。
</Callout>
