Memcached [オープンソース・インテグレーション](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) は、 [Memcached](https://memcached.org/) からパフォーマンス・カウンターや統計情報を収集し、データを New Relic Infrastructure に報告します。

要件、インストール、および構成に関する情報は、GitHub上の [Memcached On-Host Integration README](https://github.com/newrelic/nri-memcached) を参照してください。

<Callout variant="important">
  New Relic は、この技術のモニタリングを可能にするために、この統合をオープンソース化しました。この統合は現状のまま提供され、保証やサポートはありませんが、GitHub を通じて問題の報告やこの統合への貢献が可能です。この統合のサポートは、 [Expert Services のサブスクリプションでご利用いただけます。](https://www.newrelic.com/expertservices) 。
</Callout>
