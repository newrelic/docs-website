## 問題

New Relic の [Metric API](/docs/data-ingest-apis/get-data-new-relic/metric-api/introduction-metric-api) に送信されるデータについて報告された問題の原因を特定するのに問題があります。DockerまたはKubernetes用のPrometheus OpenMetricsインテグレーションからMetric APIに送信される正確なデータを確認したい。

## 解決策

標準出力ストリームに送られている生データを表示するように、Prometheus OpenMetrics インテグレーションを設定することができます。Metric API に送信されている正確なデータを見るには、 `EMITTERS` 環境変数を `" api,stdout"` に設定してください。
