インフラストラクチャー・エージェントは、 `newrelic-infra.yml` ファイルを読み込んで、 [設定を行います。](/docs/infrastructure/install-configure-infrastructure/configuration/configure-infrastructure-agent) 。最新の設定テンプレートは，GitHub の [infra-agent リポジトリで見ることができます](https://github.com/newrelic/infrastructure-agent/blob/master/assets/examples/infrastructure/newrelic-infra-template.yml.example) 。

<Callout variant="tip">
  フォーマットの問題を避けるために、使用前にYAML設定ファイルを [linting](http://www.yamllint.com/) することをお勧めします。
</Callout>

各設定の詳細については、 [インフラストラクチャの設定](/docs/infrastructure/install-configure-manage-infrastructure/configuration/infrastructure-configuration-settings) を参照してください。

<Callout variant="important">
  [設定を変更した後は、必ずエージェント](/docs/infrastructure-start-stop-restart-check-agent-status) またはウェブサーバーを再起動してください。
</Callout>
