インフラストラクチャ エージェントは、 [構成](/docs/infrastructure/install-configure-infrastructure/configuration/configure-infrastructure-agent)のために`newrelic-infra.yml`ファイルを読み取ります。最新の構成テンプレートは、 [GitHub のインフラエージェント リポジトリに](https://github.com/newrelic/infrastructure-agent/blob/master/assets/examples/infrastructure/newrelic-infra-template.yml.example)あります。

<Callout variant="tip">
  フォーマットの問題を避けるために、使用前にYAML設定ファイルを [linting](http://www.yamllint.com/) することをお勧めします。
</Callout>

各設定の詳細については、 [インフラストラクチャの設定](/docs/infrastructure/install-configure-manage-infrastructure/configuration/infrastructure-configuration-settings) を参照してください。

<Callout variant="important">
  [設定を変更した後は、必ずエージェント](/docs/infrastructure-start-stop-restart-check-agent-status) またはウェブサーバーを再起動してください。
</Callout>
