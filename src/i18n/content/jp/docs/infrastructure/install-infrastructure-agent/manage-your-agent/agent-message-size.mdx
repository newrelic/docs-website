インフラストラクチャエージェント [バージョン1.0.989](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes) 以降、最大メッセージサイズは1MB（10^6バイト）となり、イベント、メトリクス、インベントリ統合を区別なく含めることができます。以前は、インベントリデータは3MBに制限されていましたが、最大メッセージサイズは5MBでした。
