## tarballファイルを使用してインストール [#custom-setups-tarball]

<DNT>**custom**</DNT>セットアップ シナリオでは、 支援モード または[ ](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-assisted-install-infrastructure-linux)[手動](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-manual-install-infrastructure-linux) [モードで tarball](https://download.newrelic.com/infrastructure_agent/binaries/linux/) ファイル を使用して、インフラストラクチャ エージェントをインストールできます。

これは、デフォルトのインストレーション設定を環境に合わせて調整する必要がある場合や、リポジトリに`newrelic-infra`パッケージがないディストリビューションにインフラストラクチャエージェントをインストールする必要がある場合に特に便利です。

<Callout variant="important">
  tarballファイルを使用したInfrastructureエージェントのカスタムインストールは、正式にはサポートされていません。
</Callout>
