## 問題 [#problem]

New Relic Infrastructure Agent を Windows サーバーにインストールしましたが、 UIにデータが表示されません。 <DNT>**Windows Services**</DNT>からサービスを手動で開始しようとすると、次のようなエラーが発生して失敗します。

```
Windows could not start the New Relic infrastructure agent service on Local Computer
Error 1067: The process terminated unexpectedly
```

## 解決 [#solution]

ウイルス対策製品またはセキュリティ製品が、New Relic インフラストラクチャ エージェントのサービスとしての実行を妨げている可能性があります。エージェントが問題なく動作することを検証するには、管理者としてコマンド ラインから `newrelic-infra.exe` ファイルを実行し、ホストが期待どおりにデータを報告していることを確認します。その場合は、セキュリティ チームと協力して、サービスを許可リストに追加する必要があるか、追加のパラメーターを使用して構成する必要があるかを確認してください。

<Callout variant="important">
  エージェントの場所とその変更方法については、 [「エージェント ディレクトリの構成方法」を](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-manual-install-infrastructure-agent-windows/#agent-directory)参照してください。
</Callout>
