Infrastructure関連のインテグレーションは数多くあります。この分野における当社のサービスの概要を説明します。

## クラウドインテグレーション [#cloud]

当社のクラウドインテグレーションは、クラウドプラットフォームサービスとアカウントからデータを収集します。クラウドインテグレーションのインストールプロセスはなく、Infrastructureエージェントを使用する必要はありません。New Relicアカウントをクラウドプロバイダーアカウントに接続するだけです。

## その他のインテグレーションを参照する [#browse]

Infrastructure関連のインテグレーションをさらに参照するには、[インスタントオブザーバビリティのページ](https://newrelic.com/instant-observability?category=infrastructure-and-os)を参照してください。

## 機能 [#whats-next]

インテグレーションを有効にすると、以下を実行できるようになります。

* [InfrastructureモニタリングUI](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure)におけるメトリクスと設定データの[絞り込みと分析](/docs/infrastructure/integrations/find-use-infrastructure-integration-data)。
* [データのクエリ](/docs/using-new-relic/data/understand-data/query-new-relic-data)を行い、カスタムチャートとダッシュボードを作成する。
* [アラート条件を作成](/docs/alerts/new-relic-alerts/getting-started/alerts-glossary#alert-condition)して、サービスのパフォーマンス上の問題を監視する。
* クラウドインテグレーションの場合、[データ収集の設定](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations)を設定する
