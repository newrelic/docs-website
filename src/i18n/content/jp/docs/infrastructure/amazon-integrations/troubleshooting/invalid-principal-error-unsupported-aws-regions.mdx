## 問題

[AWSをインフラストラクチャ インテグレーションに](/docs/integrations/amazon-integrations/get-started/connect-aws-services-infrastructure#connect)接続すると、<DNT>**Review**</DNT> フェーズでAWSエラー `Invalid principal in policy` が発生します。

## 解決

インストールする統合のタイプを確認します。New Relic は、中国リージョンを除くすべての[AWS リージョン](https://docs.aws.amazon.com/general/latest/gr/rande.html)にあるサービスとエンドポイントからモニタリング データを取得できます。

中国を使用しているアカウントから統合機能を構成しようとすると、このエラーが表示されます。
