## 問題

[AWS を New Relic Infrastructure integration](/docs/integrations/amazon-integrations/get-started/connect-aws-services-infrastructure#connect) に接続する際、 **Review** の段階で、AWS エラー `Invalid principal in policy` が発生します。

## 解決策

インストールしたいインテグレーションの種類を確認します。New Relic は、中国リージョンを除くすべての [AWS リージョン](https://docs.aws.amazon.com/general/latest/gr/rande.html "新しいウィンドウにリンクが開きます。") に配置されたサービスやエンドポイントから監視データを取得できます。

中国を使用しているアカウントから統合機能を構成しようとすると、このエラーが表示されます。
