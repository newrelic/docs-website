[New Relic インフラストラクチャの統合に](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)は、メトリクス データを New Relic に送信する AWS IoT Twinmaker の統合が含まれるようになりました。このドキュメントでは、統合の機能、アクティブ化方法、およびレポートできるデータについて説明します。

## 特徴

統合を使用してテレメトリ データを収集し、 [WorkMail](https://aws.amazon.com/iot-twinmaker/)から New Relic に送信します。これにより、サービスを監視し、受信データをクエリし、ダッシュボードを構築してすべてを一目で観察できるようになります。

## 統合をアクティブ化して使用する

このインテグレーションを有効にするには、 [AWSサービスをNew Relicに接続する](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)方法を参照してください。 インテグレーションのメトリクスを見つけるには、 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT>に移動して`aws.IoTTwinMaker`でフィルタリングします。
