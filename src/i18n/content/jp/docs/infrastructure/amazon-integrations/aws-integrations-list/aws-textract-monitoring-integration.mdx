[New Relic インフラストラクチャ統合](/docs/infrastructure/introduction-infra-monitoring)には、メトリクス データを New Relic に送信する<DNT>AWS Textract</DNT>統合が含まれるようになりました。

このドキュメントは、インテグレーションの機能、有効化する方法、および報告可能なデータについて説明するものです。

## 特徴 [#features]

統合を使用して、 [Textract アプリ](https://aws.amazon.com/textract/) からテレメトリ データを収集して New Relic に送信します。サービスを監視し、受信データをクエリし、ダッシュボードを構築して、すべてを一目で確認します。

## レポートデータ [#report-data]

このデータを報告するには 2 つのオプションがあります。推奨されるパスは、 [Amazon CloudWatch Metric Streams 統合](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream)をセットアップすることです。あるいは、 [ポーリングに依存する古い Amazon 統合を](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations)使用することもできます。

## データを見つけて使用する [#find-use-data]

このインテグレーションからのデータは、 `Metric`データ型に添付されます。 インテグレーションのメトリクスを見つけるには、 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT>に移動して`aws.textract`でフィルタリングします。
