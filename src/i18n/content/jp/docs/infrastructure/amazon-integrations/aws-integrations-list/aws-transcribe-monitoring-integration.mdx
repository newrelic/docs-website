[New Relic インフラストラクチャ](/docs/infrastructure/introduction-infra-monitoring)統合には、Transcribe メトリクスを New Relic に報告する[Amazon Transcribe](https://aws.amazon.com/transcribe)の統合が含まれます。

## 特徴 [#features]

この統合により、 [Amazon Transcribe](https://aws.amazon.com/transcribe/) サービスからテレメトリ データを収集して New Relic に送信します。サービスを監視し、受信データをクエリし、ダッシュボードを構築してすべてを一目で観察します。

## 統合をアクティブ化する

この統合を有効にするには、 [AWS のサービスを New Relic に接続する](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream)方法を参照してください。

## データを見つけて使用する [#find-use-data]

このインテグレーションからのデータは、 `Metric`データ型に添付されます。 UI でこのデータを検索するには、 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT>に移動して`aws.transcribe`をフィルタリングします。
