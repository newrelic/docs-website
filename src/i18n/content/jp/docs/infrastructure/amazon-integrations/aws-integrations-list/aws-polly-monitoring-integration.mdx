[New Relic Infrastructure](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)には、Polly メトリクスをNew Relicに報告する <DNT>Amazon Polly</DNT> の統合が含まれます。

## 特徴 [#features]

この統合により、テレメトリーデータを収集し、New Relic [<DNT>Amazon Polly</DNT>](https://docs.aws.amazon.com/polly)サービスから に送信します。サービスを監視し、受信データを書き、ダッシュボードを構築してすべてを一目で観察します。

## レポートデータ [#report-data]

このデータを報告するには 2 つのオプションがあります。推奨されるパスは、 [Amazon CloudWatch Metric Streams 統合](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream)をセットアップすることです。あるいは、 [ポーリングに依存する古い Amazon 統合を](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations)使用することもできます。

## データを見つけて使用する [#find-use-data]

このインテグレーションからのデータは、 `Metric`データ型に添付されます。 UI でこのデータを検索するには、 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT>に移動して`aws.polly`をフィルタリングします。
