New Relic Infrastructure統合には、AWS ネットワーク ファイアウォール メトリクスを New Relic に送信する統合が含まれるようになりました。 この文書では、統合の機能、有効化方法、およびレポートに使用できるデータについて説明します。

## 特徴 [#features]

AWS ネットワーク ファイアウォール メトリクス統合は、テレメトリー データを収集し、AWS ネットワーク ファイアウォールから New Relic に送信します。 AWS サービスを監視し、受信データをクエリし、ダッシュボードを構築してすべてを一目で観察できます。

## 統合をアクティブ化する [#activate-ingration]

この統合を有効にするには、 [Amazon CloudWatch Metric Streams 統合ドキュメントの](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)手順に従ってください。

## データを見つけて使用する [#find-data]

統合のメトリクスを見つけるには:

1. に行く

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

2. 選択する

   <DNT>
     **Metrics and events**
   </DNT>

3. `aws.networkfirewall`でフィルタリングします。

## メトリックデータ [#metrics]
