New Relic Infrastructure統合には AWS WorkSpaces Web の統合が含まれ、そのメトリクス データを New Relic に送信できるようになりました。

この文書では、統合の機能、有効化方法、およびレポートに使用できるデータについて説明します。

## 特徴

当社の統合を使用して、テレメトリーデータを収集し、 [WorkSpaces Web](https://aws.amazon.com/workspaces/web)から New Relic に送信します。 サービスを監視し、受信データをクエリし、ダッシュボードを構築してすべてを一目で観察します。

## 統合をアクティブ化する

この統合を有効にするには、 [Amazon CloudWatch Metric Streams 統合ドキュメントの](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)手順に従ってください。

## データを見つけて使用する

統合のメトリクスを見つけるには:

1. [one.newrelic.com > すべての機能](https://one.newrelic.com/all-capabilities)にアクセス

2. 選択する

   <DNT>
     **Metrics and events**
   </DNT>

3. `aws.workspacesweb`でフィルタリングします。

### アラートの作成

変更があった場合に通知するように<InlinePopover type="alerts"/>を設定できます。 たとえば、重大なエラーや致命的なエラーを関係者に通知するための集計を設定できます。

アラートの作成の詳細については、 [「アラートの概要」ドキュメントを](/docs/alerts-applied-intelligence/new-relic-alerts/learn-alerts/introduction-alerts/)参照してください。
