New Relic は Amazon Web Services (AWS) と統合し、 [AWS DataSync](https://aws.amazon.com/datasync/)メトリクスやその他のデータを New Relic に報告します。

この文書では、統合を有効にする方法と、報告されるデータについて説明します。

## 特徴 [#features]

統合を使用して、 DataSync サービスからテレメトリーデータを収集し、 に送信します。New RelicAWSサービスを監視し、受信データを書き、ダッシュボードを構築してすべてを一目で観察します。

## 統合をアクティブ化する [#activate]

この統合は、 CloudWatch Metric Streamsを通じて利用できます。

この統合を有効にするには、[ 経由で サービスを に接続する 方法を参照してください。AWSNew RelicCloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)

## データを見つけて使用する [#find-data]

インテグレーションのメトリクスを見つけるには、 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT>に移動して`aws.DataSync`でフィルタリングします。

さらに、 [one.newrelic.com](https://one.newrelic.com/all-entities)にアクセスして`datasync`を検索すると、AWS DataSync に関連付けられているエンティティを見つけることができます。
