[New Relic インフラストラクチャの統合には](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) 、メトリクスを New Relic に送信する AWS Elastic Transcoder の統合が含まれます。

## 特徴 [#features]

当社の統合を使用して、テレメトリ データを収集し、 [Amazon Elastic Transcoder サービス](https://aws.amazon.com/elastictranscoder/)から New Relic に送信します。サービスを監視し、受信データをクエリし、ダッシュボードを構築してすべてを一目で観察できます。

## 始めましょう [#get-started]

この統合を有効にするには、 [「AWS サービスを New Relic に接続する」](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup)を参照してください。

インテグレーションが設定されると、 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT>に移動して`aws.elastictranscoder`でフィルタリングすることでデータを見つけることができます。
