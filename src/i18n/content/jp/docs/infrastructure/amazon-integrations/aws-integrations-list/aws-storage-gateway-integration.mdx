New Relic Infrastructureストラクチャ統合には、AWS Storage Gateway メトリクスを New Relic に送信する統合が含まれるようになりました。 AWS Storage Gateway メトリクス統合は、AWS Storage Gateway からテレメトリーデータを収集して New Relic に送信します。 AWS サービスをモニタリングし、受信データをクエリし、ダッシュボードを構築してすべてを一目で観察できます。

## 統合をアクティブ化する [#activate-ingration]

この統合を有効にするには、 [Amazon CloudWatch Metric Streams 統合ドキュメントの](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)手順に従ってください。

## データを見つけて使用する [#find-data]

統合のメトリクスを見つけるには:

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**に移動します
2. **Metrics and events** \[メトリクスとイベントを]選択します
3. `aws.storagegateway`でフィルタリングします。

## メトリックデータ [#metrics]
