[New Relic インフラストラクチャモニタリング](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) は、 [Microsoft Azure Data Factory](https://azure.microsoft.com/en-us/services/data-factory/) のデータを New Relic にレポートするための統合を提供しています。ここでは、統合を有効にする方法と、どのようなデータを報告できるかを説明します。

## 機能

New Relic は、Azure Data Factory サービスから DataFactory と Factory の両方のデータを収集します。New Relic InfrastructureからAzure Data Factoryのデータを監視したり、アラートを出したりすることができ、カスタムクエリやカスタムチャートのダッシュボードを作成することができます。

## 統合を有効にする [#activate]

統合を有効にするには、標準的な手順に従って、 [New Relic](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) で Azure サービスを有効にしてください。

## 設定およびポーリング [#polling]

[設定オプション](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations)を使用し、ポーリングの頻度を変更し、データを絞り込むことができます。

デフォルト [ポーリング](/docs/infrastructure/microsoft-azure-integrations/getting-started/azure-integration-polling) Azure Data Factory統合の情報です。

* ポーリングの間隔。5分

## データの検索と使用 [#find-data]

統合データを見つけるには、 **[one.newrelic.com](https://one.newrelic.com) > Infrastructure> Azure** にアクセスし、 **Data Factory** の統合を選択します。
