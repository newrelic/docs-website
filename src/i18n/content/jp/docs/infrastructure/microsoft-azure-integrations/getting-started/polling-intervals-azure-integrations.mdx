New Relic の [Azure インテグレーション](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations) は、インテグレーションに固有のポーリング間隔に従って、Azure サービスにクエリを実行します。

ポーリング間隔は、統合サービスに関連する<DNT>**every**</DNT> Azure エンティティに適用されます。 たとえば、CosmosDB インスタンスが 13 個ある場合、13 個のインスタンスそれぞれが 5 分ごとにポーリングされます。

## ポーリングデータを見る [#account-status]

[Azure インテグレーションをアクティベートすると](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) 、New Relic は Azure からのデータのポーリングを開始し、そのデータに UI からアクセスできるようになります。Azure のデータに加え、他の New Relic 機能からインポートした追加データを照会することができます。また、特定の統合またはアカウント全体のダッシュボードデータを表示することもできます。

ポーリング間隔やAPIコールなど、Azureインテグレーションのデータを視覚化します。

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure**
   </DNT>

   に移動します。

2. 特定のインテグレーションのデータを表示するには、インテグレーションの行の

   <DNT>
     **Dashboards**
   </DNT>

   リンクを選択します。

## New Relicのポーリングインターバル [#aws-integrations]

ポーリングと解決の詳細については、 [Azure インテグレーション リスト](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/)セクションの特定のインテグレーションのドキュメントを参照してください。
