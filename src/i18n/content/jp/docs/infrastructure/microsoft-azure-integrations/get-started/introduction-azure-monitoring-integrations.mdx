[Microsoft Azure](https://azure.microsoft.com)のアプリケーションとサービスを監視するいくつかの方法をご紹介します。これらのサービス概要と、どちらかを使用する理由を説明します。

* AzureでNew Relic：[Azure Native New Relic Service](/docs/infrastructure/microsoft-azure-integrations/get-started/azure-native)を使用すると、Azureポータルから直接、Azure上に構築されたNew Relicをデプロイできます。このオプションでは、以下の両方の機能（およびJavaアプリ監視）と同等の機能が提供され、Azureポータルから直接設定できます。
* Azureインテグレーション：当社の[Azureインフラストラクチャ統合](/docs/infrastructure/microsoft-azure-integrations/get-started/activate-azure-integrations)は、多数のAzureサービスをサポートしています。
* .NET APM：[当社の.NET APMエージェント](/install/dotnet)をインストールすると、Azureアプリケーションを監視し、詳細なテレメトリーデータを受信できます。
