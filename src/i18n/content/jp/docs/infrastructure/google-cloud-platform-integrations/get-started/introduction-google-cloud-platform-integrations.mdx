[New Relic のインフラストラクチャ](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure) インテグレーションは、人気のある製品やサービスのパフォーマンスを監視します。New Relic の Google Cloud Platform (GCP) インテグレーションでは、いくつかの New Relic の機能で GCP のデータをモニターすることができます。

## GCPとNew Relicの接続 [#connect]

GCP データを取得するためには、標準的な手順に従って [GCP サービスを New Relic](/docs/infrastructure/gcp-integrations) に接続してください。

## あなたのGCPデータを見る [#insights]

設定プロセスに従うと、Google Cloud Platform アカウントからのデータが New Relic に直接報告されるようになります。

GCP データを表示するには:

1. <DNT>\*\*[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities)</DNT>に移動します。
2. <DNT>**Google Cloud Platform**</DNT>を検索してタイルをクリックします。

事前設定された<InlinePopover type="dashboards" />ダッシュボードと<DNT>**Events explorer**</DNT>ダッシュボードの両方で NRQL クエリを表示および再利用できます。 これにより、特定のニーズに合わせてクエリをカスタマイズできます。

イベント、ダッシュボード、その他のデータは、New Relic のすべてのサービスで利用できます。
