## オペレーティング システムとランタイム プラットフォーム [#infra-os-runtime]

エージェントは、さまざまな環境でサポートされています。サポートされているオペレーティング システム、アーキテクチャ、およびランタイム環境の最新リストについて[は、インフラストラクチャ エージェントの要件を](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/)参照してください。

インフラストラクチャ監視エージェントは、これらのオペレーティング システムを製造元のサポート終了までサポートします。エージェントは互換性があり、パッケージは EOL システムで使用できる場合がありますが、サポートは保証されていません。
