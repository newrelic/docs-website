## 問題

New RelicのPixie統合の[インストール手順](/docs/kubernetes-pixie/auto-telemetry-pixie/install-auto-telemetry-pixie)が完了し、Pixieが使用するメモリの量を減らしたいと考えています。

## 解決策

Pixie [は、テレメトリデータの収集と、このデータの短期間のクラスター内ストレージという2つの異なる目的でメモリを使用し](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#memory-usage)ます。短期記憶用に別の量のメモリを指定する場合は、インストール中に[短期記憶のメモリ制限を構成](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#reduce-data-storage)できます。
