## 問題

`cannot increase buffer: current=32000 requested=64768 max=32000`や`pod out of memory`のようなエラーが発生しています。

## 解決

Fluent ビットのバッファ サイズを増やすには、 `k8sBufferSize`属性を追加します。

```yaml
newrelic-logging:
  enabled: true
  containerSecurityContext:
    privileged: true
  fluentBit:
    k8sBufferSize: "64k"
    persistence:
      mode: "none"
```
