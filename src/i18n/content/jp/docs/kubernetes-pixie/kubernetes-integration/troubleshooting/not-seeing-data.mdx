## 問題

[New RelicKubernetesインテグレーション をインストールしましたが、 アカウントに](/install/kubernetes/) KubernetesNew Relicデータが表示されません。

## 解決

1. [`values-newrelic.yaml`](https://github.com/newrelic/helm-charts/blob/master/charts/nri-bundle/values.yaml)ファイルで`CLUSTER_NAME`と`NRI_LICENSE_KEY`が正しく構成されていることを確認してください。

2. 次のコマンドを実行して、統合のデプロイメントが成功したことを確認します。

```bash
  kubectl get pods,deployments,daemonsets -o wide -n <newrelic_namespace>
```

3. インフラエージェントとインテグレーションの[ログを確認し](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/get-logs-version/)、エラーメッセージがないことを確認してください。
