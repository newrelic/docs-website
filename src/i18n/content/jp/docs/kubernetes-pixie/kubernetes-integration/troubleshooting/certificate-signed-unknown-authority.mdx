## 問題

メッセージ`x509: certificate signed by unknown authority`が表示されます。

## 解決

署名された証明書が適切に構成されていることを確認する必要があります。これを行うには、エージェントYMLコンテナに次の環境変数を追加します。

```yml
- name: NRIA_CA_BUNDLE_DIR 
  value: YOUR_CA_BUNDLE_DIR
- name: NRIA_CA_BUNDLE_FILE  
  value: YOUR_CA_BUNDLE_NAME
```
