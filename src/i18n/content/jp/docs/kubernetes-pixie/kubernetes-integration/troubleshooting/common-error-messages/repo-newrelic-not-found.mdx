## 問題

Helm または Manifest を使用した[Kubernetesの](/install/kubernetes/)インストレーション中に、このエラー メッセージが表示されます。

```shell
  repo newrelic not found
```

## 解決

次のコマンドを実行して、Helm チャートに NewRelic リポジトリを追加します。

```shell
  helm repo add newrelic https://helm-charts.newrelic.com
```
