KubernetesOpenTelemetryメトリクスは、OpenTelemetryCollector によって収集され、New Relic New RelicKubernetesインテグレーションを使用して に送信されます。

このドキュメントでは、 Kubernetes OpenTelemetryメトリクスからNew Relic Kubernetesメトリクスまでを説明します。 これは、 Kubernetesクラスタの監視に利用できるメトリクスとプロパティを理解するのに役立ちます。
