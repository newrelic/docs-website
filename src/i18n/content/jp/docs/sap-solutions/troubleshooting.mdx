## サーバーの可用性のトラブルシューティング

`NR_SAP_INSTANCE_AVAILABILITY`および`NR_SAP_SERVER_LIST`データ プロバイダーは、 `SAPCONTROL` Web サービスへの SOAP 呼び出しに依存します。これらが機能するには、モニター サーバーの`DEFAULT.PFL`プロファイルを構成する必要があります。

1. `DEFAULT.PFL`ファイルに次の行が存在することを確認してください。

`service/protectedwebmethods = SDEFAULT`

2. この行が欠落している場合は、次の行を追加してください。

`service/protectedwebmethods = SDEFAULT -GetQueueStatistic -ABAPGetWPTable -EnqGetStatistic -GetProcessList -GetEnvironment -GetVersionInfo -GetSystemInstanceList`

3. コマンド`stopsap sapstartsrv`および`startsap sapstartsrv`を使用して SAP サービスを再起動します。

4. コマンド`sapcontrol -prot GSOAP_HTTP -nr 00 -function GetSystemInstanceList`を実行して設定が機能することを確認します。
