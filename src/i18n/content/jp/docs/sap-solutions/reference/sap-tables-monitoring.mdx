このリファレンスには、<DNT>**New Relic Monitoring for SAP Solutions**</DNT>によって監視目的で取得された<DNT>**SAP tables**</DNT>がリストされています。これらのテーブルは、<DNT>**AIF**</DNT>、<DNT>**SLT**</DNT>、<DNT>**IDOC**</DNT>、ジョブ、電子メール、トランスポート、<DNT>**RFC**</DNT>、ビジネスプロセス、および変更ドキュメントを含む、さまざまな監視領域のデータを提供します。

## 認可とテーブルアクセス

<DNT>**New Relic Monitoring for SAP**</DNT> <DNT>**RFC connections**</DNT>をセットアップする際、権限ロールはモニターされるシステム内の<DNT>**RFC user**</DNT>に対して広範なテーブル表示アクセスを提供します。カスタム<DNT>**Z-tables**</DNT>を含む追加の<DNT>**SAP tables**</DNT>を取得する際、および<DNT>**IDOC**</DNT>やメッセージペイロードデータを取得する際に追加のカスタマイズを容易にするため、アクセスを広く保つことをお勧めします。ただし、テーブルへのアクセスを制限する場合は、このリストを使用してください。

<Callout variant="important">
  カスタムデータコレクターがテーブルから直接データを取得する場合、新しいデータプロバイダーが機能するためには、セキュリティロールを更新する必要がある場合があります。
</Callout>

## このリファレンスの対象者

* <DNT>**SAP Basis**</DNT> データ収集スコープを確認するアドミニストレーター
* テーブルアクセス要件を監査するセキュリティチーム
* テーブルの使用状況を監視するデータベースアドミニストレーター
