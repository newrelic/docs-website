* モニターされる<DNT>**SIDs**</DNT>の合計が10未満で、インスタンスが50を超えない場合、専用サーバーは必要ありません。<DNT>**SIDs**</DNT>が10を超えるか、インスタンスが100を超える場合は、専用サーバーをお勧めします。
* 最大でも、それぞれ96Gのメモリと12コアのCPUを持つ2つの<DNT>**App Server**</DNT>インスタンスを備えた専用サーバーを使用する必要があります。監視システムが<DNT>**HANA database**</DNT>を使用する場合、データベースサーバーのインスタンスをアプリケーションサーバーのインスタンスから分離することをお勧めします。
