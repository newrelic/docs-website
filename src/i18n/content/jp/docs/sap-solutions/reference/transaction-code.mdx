これらのトランザクションコード（<DNT>**TCODEs**</DNT>）は、<DNT>**NRM4SAP**</DNT>を設定および保守するために使用されます。アドミニストレーターは、これらのコードに対する認可が必要です。

<Callout variant="important">
  管理/サポートロールの<DNT>**TCODE**</DNT>承認オブジェクトにパラメーター`/NEWR/*`を追加することをお勧めします。
</Callout>
