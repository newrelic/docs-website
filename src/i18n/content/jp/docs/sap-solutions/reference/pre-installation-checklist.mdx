<DNT>**New Relic Monitoring for SAP Solutions**</DNT>をインストールする前に、このチェックリストを使用して必要なすべての情報を収集してください。まず準備セクションを完了し、その後、インストレーションを始める前に準備状況チェックリストの表に記入します。

## インストレーション前の準備

### New Relicアカウントを確認する

* New Relicアカウントをお持ちであることを確認するか、新しいアカウントを作成してください。どのデータセンター地域（米国またはEU）を使用しているかに注意してください。

### SAPシステムのリストを準備する

監視する予定の各<DNT>**SAP system**</DNT>について、次の情報を提供してください：

* **システム ID とクライアント**: システムとそれに関連付けられたクライアントを識別します。

* **データベース情報**: データベースベンダー (例: <DNT>**HANA**</DNT>、<DNT>**Oracle**</DNT>) とそのバージョンを指定します。

* **ソフトウェア コンポーネントの詳細**: 次のコンポーネントの SAP リリースとパッチ レベルを含めます。

  * <DNT>**SAP\_BASIS**</DNT>
  * <DNT>**Kernel**</DNT>

### 中央SAPシステムを指定する

<DNT>**New Relic Monitoring for SAP Solutions (NRM4SAP)**</DNT>のインストレーションのセントラルシステムとして機能する<DNT>**SAP system**</DNT>を1つ選択します。

* **インストレーションの詳細**: <DNT>**ABAP component**</DNT>および設定がインストールされる<DNT>**System ID**</DNT>とクライアントを指定してください。
* **プロキシサーバー情報**: プロキシサーバーを使用する場合は、ホスト名とポートを指定します。<DNT>**New Relic One**</DNT>プラットフォームにアクセスするための関連するプロキシサーバーポリシーを含めてください。

### ネットワークアクセシビリティ

ネットワーク接続を確認し、<DNT>**API endpoint**</DNT>の設定を確認します：

* ファイアウォールがNew Relicへの<DNT>**HTTPS**</DNT>トラフィックをブロックしていないことを確認してください
* 該当する場合は、プロキシサーバーの詳細を記載します。
* ご利用の地域（米国またはEU）の<DNT>**RFC destination**</DNT>設定を確認してください

ネットワーク要件とAPIエンドポイントの詳細については、[ネットワーク要件とAPIエンドポイント](/docs/sap-solutions/reference/network-accessibility)を参照してください。
