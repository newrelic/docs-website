freshnessValidatedDate: never

---

<Steps>
  <Step>
    **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**に移動します。
  </Step>

  <Step>
    **アプリと視覚化**を選択し、 **IDOC Explorer**を検索します。
  </Step>

  <Step>
    **Manage access**\[アクセスの管理]を選択します。アプリにアクセスするためのアカウントを選択します。

    <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />
  </Step>

  <Step>
    **Update**\[更新]をクリックします。IDOC エンティティを表示すると、IDOC エクスプローラーが表示されます。
  </Step>
</Steps>
