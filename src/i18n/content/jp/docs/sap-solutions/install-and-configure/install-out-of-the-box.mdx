SAP向けのNew Relic監視には、ダッシュボード、アラートポリシー、IDOC Explorerを含む、すぐに使える監視コンポーネントのリストが含まれています。それらを見つけてインストールするには、以下の指示に従ってください。

## あなたが始める前に

これらのタスクを実行するには、アカウントのNew Relic管理者権限が必要です。

<Callout variant="important">
  <DNT>**SAP Cockpit**</DNT>ダッシュボードでは、インストレーションの前に`NR_LKUP_SYSTEM`という名前の[ルックアップテーブルを作成する](#create-lookup-table)必要があります。
</Callout>

## ダッシュボードのインストール

### ルックアップテーブルの作成 [#create-lookup-table]

<Callout variant="tip">
  <DNT>**SAP Cockpit**</DNT>ダッシュボードにのみ必要です。
</Callout>

<DNT>**SAP Cockpit**</DNT>ダッシュボードは、<DNT>**SAP systems**</DNT>とその環境に<DNT>**lookup table**</DNT>を使用します。

1. 2つの列（`SYS_ID`と`Env`）を持つ<DNT>**lookup table**</DNT>を作成し、<DNT>**CSV file**</DNT>として保存します。ルックアップファイルのサンプルは次のとおりです：

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. <DNT>**Log &gt; Lookup Tables**</DNT>に移動します。

3. <DNT>**Add a table**</DNT>をクリックします。

4. <DNT>**Add a lookup table**</DNT>サイドバーで、テーブル名を<DNT>**NR\_LKUP\_SYSTEM**</DNT>に設定し、<DNT>**CSV file**</DNT>をアップロードします。詳細については、[ルックアップテーブル](/docs/logs/ui-data/lookup-tables-ui/)を参照してください。

5. テーブルの内容を確認するには、`FROM lookup(NR_LKUP_SYSTEM) SELECT *`で<DNT>**query builder**</DNT>を使用し、<DNT>**Run**</DNT>をクリックします。

### ダッシュボードをインストールする [#install-dashboards]

1. <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> &gt; <DNT>**Integrations &amp; Agents**</DNT>に移動し、<DNT>**New Relic Monitoring for SAP Solutions**</DNT>を検索して、すべてのダッシュボードを自動的にダウンロードします。

2. または、<DNT>**Dashboards &gt; Recommended Dashboards**</DNT>に移動し、<DNT>**View all**</DNT>をクリックして、SAPを検索することもできます。詳細については、[Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/)を参照してください。

3. これらのダッシュボードをインストールします：

   * <DNT>**SAP Dashboards**</DNT>
   * <DNT>**SAP Cockpit**</DNT>
   * <DNT>**SAP Data Collection Monitoring**</DNT>
   * <DNT>**SAP Data Ingestion Estimate**</DNT>

### ダッシュボードフィルターを追加する [#add-dashboard-filter]

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards**</DNT>に移動します。

2. フィルタリングのファセット リンクを有効にします。

   * ウィジェットの右上隅にある省略記号（<DNT>**…**</DNT>）をクリックします。
   * 編集画面を開くには、<DNT>**Edit**</DNT>を選択します。
   * <DNT>**Filter the current dashboard**</DNT>スイッチを切り替えます

3. <DNT>**Facet linking**</DNT> 表、円グラフ、およびバーチャートウィジェットで有効にして、<DNT>**SAP system**</DNT>、<DNT>**instance**</DNT>、<DNT>**IDOC status**</DNT>、<DNT>**jobs**</DNT>などでフィルタリングできます

## アラートポリシーをインストールする

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**</DNT>に移動します。

2. 検索して<DNT>**New Relic Monitoring for SAP Solutions alert policy**</DNT>をクリックします。

   <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />

3. <DNT>**Alert condition**</DNT>タブでアカウントを選択し、<DNT>**alert policy**</DNT>を<DNT>**Enable**</DNT>に切り替えます。ポリシーの管理方法については、[ポリシーの管理](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/)を参照してください。

4. 戻って<DNT>**Nr4sap default alert policy**</DNT>を検索します。有効にする<DNT>**alert condition**</DNT>をオンにします。

## IDOCエクスプローラーをインストールする

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT>に移動します。

2. <DNT>**Apps &amp; Visualizations**</DNT>を選択し、 <DNT>**IDOC Explorer**</DNT>を検索します。

3. <DNT>**Manage access**</DNT>を選択します。アプリにアクセスするためのアカウントを選択します。

   <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />

4. <DNT>**Update**</DNT>をクリックします。<DNT>**IDOC entities**</DNT>を表示すると、<DNT>**IDOC Explorer**</DNT>が表示されます。
