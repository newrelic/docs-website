**NRM4SAP**の ABAP コンポーネントは SAP アドオンとしてパッケージ化されており、**クライアント 000**でトランザクション**SAINT**を使用してインストールできます。

## アドオンインストレーションファイル [#add-on-installation-file]

* **初期インストール**: アドオン ファイル`K-xxxAOIINNEWR.SAR`をローカル PC 上のフォルダーにダウンロードします。
* **以前のバージョンからアップグレード**: `K-xxxAOUINNEWR.SAP`という名前のファイルを使用します。

## SAPアドオンファイルをインストールする [#install-sap-add-on-file]

1. **クライアント 000**にログインします。これは、設定およびシステム管理タスクのデフォルトのクライアントです。
2. SAP GUI コマンド フィールドに`SAINT`と入力し、 **Enter を**クリックします。
3. **Installation Package** &gt; **Load Packages**に移動します。
4. 次のいずれかの方法を選択してください。

<CollapserGroup>
  <Collapser id="sar-archive-from-frontend" title="フロントエンドからのSARアーカイブ">
    1. 解凍されたインストレーション ファイルに移動して開きます。
    2. **圧縮ファイル`K-xxxAOIINNEWR.SAR`のコンテンツ**ポップアップで、 **「解凍」を**クリックします。

    <img title="Content of the compressed file" alt="A screenshot displaying the Content of the compressed file" src="/images/content-of-compressed-file-1.webp" width="" />

    3. **「ファイル K-xxxAOIINNEWR.SAR が正常に転送、解凍、削除されました」という**メッセージが表示されます。
  </Collapser>
</CollapserGroup>

<CollapserGroup>
  <Collapser id="sar-archive-from-application-server" title="アプリケーションサーバーからのSARアーカイブ">
    1. アプリケーション サーバー ( `/usr/sap/trans/download` ) の trans ディレクトリの下にダウンロード ディレクトリがあることを確認します。
    2. アドオン ファイルをこのディレクトリにコピーします。
  </Collapser>
</CollapserGroup>

<CollapserGroup>
  <Collapser id="eps-files-from-application-server" title="アプリケーションサーバーからのEPSファイル">
    1. まず、 `SAR`ファイルを`PAT`ファイルに解凍します。OS (この場合は Linux) 上で以下のコマンドを実行します。

    ```sql
    cd /usr/sap/trans
    cp /tmp/K-300AOIINNEWR.SAR /usr/sap/trans
    SAPCAR -xvf K-300AOIINNEWR.SAR
    ```

    2. ポップアップ**「EPS 受信トレイから OCS パッケージをアップロードしますか?」**で、 **「はい」**を選択します。OCS ファイルがエラーなくアップロードされていることを確認します。
  </Collapser>
</CollapserGroup>

5. **Start**\[スタート]をクリックします。

6. **Installable Add-on packages for your syste**m\[システムのインストール可能なアドオン パッケージ]を選択し、 **Continue**\[続行]をクリックします。

7. **「変更調整トランスポートをキューに追加しますか?」**と尋ねられたら、 **「いいえ」**をクリックします。

   <img title="Do you want to add Modification Adjustment Transports to the queue?, Click No" alt="A screenshot displaying question, Do you want to add Modification Adjustment Transports to the queue?" src="/images/modify-adjustement-transports.webp" width="700" />

<Callout variant="important">
  パッケージリストに調整トランスポートと説明が記載されている場合は、**Yes**\[はい]をクリックします。
</Callout>

8. クリック<img src="/images/green-tick-button.webp" class="inline" style={{height: "25px", width: "30px"}} alt="green check button" title="Green check button" />ボタンを押して次の画面に進みます。

<Callout variant="important">
  **Errors during Disassembling OCS Packages**\[OCS パッケージの逆アセンブル中にエラーが発生する]ページが表示された場合は、 **Ignore**\[無視]をクリックします。
</Callout>

9. パッケージがインストールされたら、 **「完了」**をクリックします。**ランタイム分析の送信**ポップアップが表示され、以下を選択できます。

   * **送信**: ローカル ブラウザー ウィンドウでランタイム分析データを含むフィードバック フォームを開きます。このフォームを使用して、SAP にデータを送信できます。
   * **表示**: 最初に実行時分析を表示し、次にフィードバック フォームをローカル ブラウザー ウィンドウに送信します。
   * **送信しない**: 実行時分析データを送信せずにダイアログを終了します。

10. 「**インストールされたコンポーネント」**タブでインストールが成功したことを確認します。

11. **クライアント000**からログアウトします。

<Callout variant="tip">
  システムからログアウトするには、SAP GUI コマンド フィールドに**/nex**と入力できます。
</Callout>
