## システムワークロードをセットアップする

1. <DNT>**All entities &gt; Workloads &gt; + Create a workload**</DNT>に移動します。詳細については、[ワークロードの作成と定義](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/)を参照してください。

2. <DNT>**workload**</DNT>の名前を入力し、ドロップダウンリストからアカウントを選択します。

3. <DNT>**Entity Type = All**</DNT>の横にある<DNT>**+**</DNT>アイコンを選択し、フィルター バーに`sap.system = {SAP System SID}`を入力します。

   <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />

4. <DNT>**+ Add this query**</DNT>を選択して、クエリを<DNT>**workload definition**</DNT>に追加します。

5. <DNT>**Create a workload**</DNT>を選択します。

新しい<DNT>**workload**</DNT>は<DNT>**All entities**</DNT>に表示され、<DNT>**SAP system**</DNT>でタグ付けされたすべてのエンティティが含まれます。

## サービスパックをインストールする

<DNT>**Service packs**</DNT> `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`という命名規則の`.SAR`ファイルとして配信されます。

1. トランザクション`SPAM`を使用して<DNT>**service pack**</DNT>をインストールします。
2. <DNT>**client 000**</DNT>から新しい<DNT>**configuration template**</DNT>を読み込みます。
3. <DNT>**Client 000**</DNT>から<DNT>**Install New Release and Patch**</DNT> &gt; <DNT>**Copy Template Tables**</DNT>に移動します。
4. <DNT>**Install New Release and Patch**</DNT> &gt; <DNT>**Generate Configuration for New Data Providers**</DNT>の手順に従って、新しい<DNT>**data provider configurations**</DNT>を生成します。
5. <DNT>**Install New Release and Patch**</DNT> &gt; <DNT>**Verify Installation Completeness**</DNT>を選択して、インストレーションを確認します。
