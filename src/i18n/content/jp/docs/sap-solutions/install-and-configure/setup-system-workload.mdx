<Steps>
  <Step>
    **All entities &gt; Workloads &gt; + Create a workload**に移動します。 [ワークロードの作成方法の詳細については、「ワークロードの作成と定義」を](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/)参照してください。
  </Step>

  <Step>
    ワークロードの名前を入力し、ドロップダウン リスト ボックスから必要なアカウントを選択します。
  </Step>

  <Step>
    **エンティティ タイプ = すべて**オプションの横にある**+**アイコンをクリックし、フィルター バーに条件`sap.system = {SAP System SID}`を入力します。

    <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />
  </Step>

  <Step>
    **+ Add this query**\[+ このクエリを追加]をクリックして、クエリをワークロード定義に追加します。
  </Step>

  <Step>
    **ワークロードを作成するには、「ワークロードの作成」を**クリックします。
  </Step>

  <Step>
    ワークロードは「**すべてのエンティティ」** UI に表示され、SAP システムでタグ付けされたすべてのエンティティが含まれます。
  </Step>
</Steps>
