このガイドでは、SAP Business Technology プラットフォーム（BTP）からProcess インテグレーション Runtimeインスタンスのサービスキーを取得する方法を説明します。SAP Process Integrationの監視を設定するには、このサービスキーが必要になります。

## 前提条件

* 適切な権限での<DNT>**BTP subaccount**</DNT>へのアクセス
* <DNT>**Space**</DNT>に関する<DNT>**BTP administrator**</DNT>からの確認と <DNT>**Grant-types**</DNT>

## Process Integration Runtimeインスタンスの作成

1. <DNT>**BTP (Business Technology Platform) subaccount**</DNT>にログインします。

2. <DNT>**BTP cockpit**</DNT>で、<DNT>**Instances and Subscriptions**</DNT>をクリックします。

3. 右上の<DNT>**Create**</DNT>をクリックします。

4. ポップアップ画面で以下を選択します。

   * <DNT>**Service**</DNT>： <DNT>**SAP Process Integration Runtime**</DNT>

   * <DNT>**Plan**</DNT>： <DNT>**API**</DNT>

   * <DNT>**Runtime Environment**</DNT>： <DNT>**Cloud Foundry**</DNT>

   * <DNT>**Space**</DNT>：担当の <DNT>**BTP administrator**</DNT>

   * <DNT>**Instance Name**</DNT>: 組織の命名規則に従ってください

     <img title="Process Integration Instance" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields" src="/images/process-integration-instance.webp" width="600" />

5. <DNT>**Next**</DNT>をクリックし、次を選択します：

   * <DNT>**Roles**</DNT>： `MonitoringDataRead`

   * <DNT>**Grant-types**</DNT>：<DNT>**Default Client Credentials**</DNT>（他のタイプが使用される場合は、<DNT>**BTP administrator**</DNT>およびセキュリティチームに確認してください）

     <img title="BTP Roles and Grant Types" alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected" src="/images/btp-roles-grant-types.webp" width="600" />

6. <DNT>**Next**</DNT>をクリックして詳細を確認します。

7. <DNT>**Create**</DNT>をクリックします。

## サービスキーを生成してダウンロードする

1. 作成されたら、<DNT>**Instances**</DNT>タブでサービスを見つけます。

   <img title="BTP Instance" alt="The Instances tab showing the created Process Integration Runtime instance" src="/images/btp-instance.webp" width="600" />

2. 新しく作成したインスタンスの行にある省略記号（`...`）をクリックし、<DNT>**Create Service Key**</DNT>を選択します。

3. <DNT>**Service Key Name**</DNT>を入力し、<DNT>**Key Type**</DNT>：`ClientId/Secret`を選択します。

   <img title="BTP New Service Key" alt="The Create Service Key dialog showing Service Key Name and Key Type fields" src="/images/btp-new-service-key.webp" width="600" />

4. <DNT>**Create**</DNT>をクリックします。

5. <DNT>**service key**</DNT>の行にある省略記号（`...`）をクリックし、<DNT>**Download**</DNT>を選択します。

   <img title="BTP Download Service Key" alt="The service key row showing the ellipsis menu with the Download option" src="/images/btp-download.webp" width="600" />

<DNT>**service key file**</DNT>は<DNT>**JSON file**</DNT>としてダウンロードされます。安全に保存してください。監視セットアップを設定するには、このファイルの認証情報が必要になります。
