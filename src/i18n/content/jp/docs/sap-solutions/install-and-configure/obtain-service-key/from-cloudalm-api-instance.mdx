このガイドでは、SAP Cloud Application Lifecycle Management（CALM）APIインスタンスからサービスキーを取得する方法について説明します。SAP Cloud ALMの監視を設定するには、このサービスキーが必要です。

## 前提条件

* 適切な権限での<DNT>**CALM subaccount**</DNT>へのアクセス
* <DNT>**BTP (Business Technology Platform) administrator**</DNT>からの確認： <DNT>**Space**</DNT>
* [Managing Scopes](https://help.sap.com/docs/cloud-alm/apis/managing-scopes) から特定された必要なスコープ

1. <DNT>**CALM (Cloud Application Lifecycle Management) subaccount**</DNT>にログインし、<DNT>**Instances and Subscriptions**</DNT>をクリックします。

2. 右上の<DNT>**Create**</DNT>をクリックします。

3. ポップアップ画面で以下を選択します。

   * <DNT>**Service**</DNT>： <DNT>**SAP Cloud ALM API**</DNT>
   * <DNT>**Plan**</DNT>： <DNT>**Standard**</DNT>
   * <DNT>**Runtime Environment**</DNT>： <DNT>**Cloud Foundry**</DNT>
   * <DNT>**Space**</DNT>：担当の <DNT>**BTP administrator**</DNT>
   * <DNT>**Instance Name**</DNT>: 組織の命名規則に従ってください

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-alm-api.webp" width="600" />

4. <DNT>**Next**</DNT>をクリックします。

5. [「スコープの管理」](https://help.sap.com/docs/cloud-alm/apis/managing-scopes)から必要なスコープを確認し、インスタンス名とともに「問題」タブに挿入します。

   <img title="Manage Scopes" alt="The parameters tab showing the required scopes configuration for the CALM instance" src="/images/manage-scope.webp" width="600" />

6. <DNT>**Next**</DNT>をクリックして詳細を確認します。

   <img title="Review Details" alt="The review screen showing the summary of the SAP Cloud ALM API instance configuration before creation" src="/images/cloud-alm-new-instance.webp" width="600" />

7. <DNT>**Create**</DNT>をクリックします。

8. 作成されたら、<DNT>**Instances**</DNT>タブでサービスを見つけます。

9. 新しく作成したインスタンスの行にある省略記号（`...`）をクリックし、<DNT>**Create Service Key**</DNT>を選択します。

10. <DNT>**Service Key Name**</DNT>を入力し、<DNT>**Key Type**</DNT>：`ClientId/Secret`を選択します。

11. <DNT>**Create**</DNT>をクリックします。

12. <DNT>**service key**</DNT>の行にある省略記号（`...`）をクリックし、<DNT>**Download**</DNT>を選択します。

<DNT>**service key file**</DNT>は<DNT>**JSON file**</DNT>としてダウンロードされます。安全に保存してください。監視セットアップを設定するには、このファイルの認証情報が必要になります。
