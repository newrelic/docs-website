---
title: すぐに使えるアラートポリシーをインストールする
metaDescription: Install the out of the box alert policies
freshnessValidatedDate: never
translationType: machine
---

<Steps>
  <Step>
    **[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**に移動します。
  </Step>

  <Step>
    **SAP Solutions アラート ポリシーのNew Relic監視を**検索してクリックします。

    <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />
  </Step>

  <Step>
    **アラート条件**タブでアカウントを選択し、アラートポリシーを**「有効」**に切り替えます。[ポリシーの](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/)管理方法については、「ポリシーの管理」を参照してください。
  </Step>

  <Step>
    戻って、 **Nr4sap デフォルト アラート ポリシー**を検索します。 有効にするアラート条件をオンに切り替えます。
  </Step>
</Steps>
