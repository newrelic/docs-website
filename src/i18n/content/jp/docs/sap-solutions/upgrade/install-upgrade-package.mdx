SAP Solutions向けNew Relic監視の以前のバージョンからアップグレードするには、現在インストールされているバージョンに一致するアップグレードパッケージを使用してください。

## 前提条件

* <DNT>**Administrator access**</DNT> に <DNT>**SAP system**</DNT>
* 現在のバージョンが特定されていること。[現在のバージョンの特定セクション](#identify-current-version)を参照してください
* 既存の設定のバックアップ
* インストールしたバージョン用の正しい<DNT>**upgrade package**</DNT>

## 現在のバージョンの特定 [#identify-current-version]

1. <DNT>**System &gt; Status**</DNT>に移動します。
2. <DNT>**NEWR**</DNT>を検索して<DNT>**product version**</DNT>を識別します。
3. <DNT>**client 000**</DNT>で、トランザクション`SAINT`を実行して<DNT>**ABAP add-on**</DNT>を更新します。

## 設定のバックアップと更新 [#backup-and-update-configuration]

<Callout variant="tip">
  アップグレード中の上書きを防ぐため、変更した<DNT>**data provider configurations**</DNT>または<DNT>**custom data providers**</DNT>を<DNT>**configuration template**</DNT>に保存してください。<DNT>**V3.0 ABAP Add-on**</DNT>がインストールされていない場合でも、<DNT>**client 000**</DNT>から新しいテンプレートをコピーする前に、テンプレートが保存されていることを確認してください。
</Callout>

1. メニュー`/n/NEWR/INSTALL`にアクセスし、<DNT>**Update or Save Configuration Template**</DNT>に移動します。

2. <DNT>**Java monitoring**</DNT>に関連する<DNT>**HDB monitoring template**</DNT>を削除するには、`/NEWR/REP_DELETE_INS_TEMPLATE`を実行します。

3. `DBACockpit`を使用して<DNT>**ABAP data connection**</DNT>を<DNT>**HANA database**</DNT>に設定します。

4. <DNT>**client 000**</DNT>から<DNT>**configuration template**</DNT>をコピーします。

5. <DNT>**monitoring system configuration**</DNT>を変更します：

   * 新しい<DNT>**DB connection**</DNT>を追加します。
   * <DNT>**Java Enterprise Connector**</DNT> (例: <DNT>**NR\_EI\_AGENT**</DNT>) への参照を削除します。
   * <DNT>**HANA\_DB**</DNT>を<DNT>**monitoring groups**</DNT>に追加します。
   * 設定を更新するには、<DNT>**data provider configurations**</DNT>を再生成してください。
