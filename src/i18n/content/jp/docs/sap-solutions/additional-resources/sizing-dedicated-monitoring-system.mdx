* モニター SID の合計が 10 未満でインスタンスが 50 を超えない場合、専用サーバーは必要ありません。 SID が 10 を超える場合、またはインスタンスが 100 を超える場合は、専用サーバーをお勧めします。
* 最大で、2 つの App Server インスタンスを持つ専用サーバー (それぞれ 96G のメモリと 12 個のコア CPU を搭載) を使用する必要があります。監視システムで HANA データベースを使用する場合は、データベース サーバー インスタンスをアプリケーション サーバー インスタンスから分離することをお勧めします。
