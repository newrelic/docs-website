## New Relicアカウントを確認する

* SAPシステム監視用のNew Relicアカウントをすでにお持ちですか？

  * **はい**: アカウントが接続されている New Relic データセンター (米国または EU) を指定します。
  * **いいえ**: いずれかのデータセンターに接続された新しいアカウントを作成します。

## SAPシステムのリストを準備する

* 監視する予定の SAP システムごとに、次の情報を入力します。

  * **システム ID とクライアント**: システムとそれに関連付けられたクライアントを識別します。

  * **データベース情報**: データベース ベンダー (HANA、 Oracle ) とそのバージョンを指定します。

  * **ソフトウェア コンポーネントの詳細**: 次のコンポーネントの SAP リリースとパッチ レベルを含めます。

    * **SAP\_BASIS**
    * **カーネル**

## 中央SAPシステムを指定する

* NRM4SAP インストレーションの中央システムとして機能する SAP システムを 1 つ選択します。

  * **インストールの詳細**: ABAP コンポーネントと設定がインストールされるシステム ID とクライアントを指定します。
  * **プロキシ サーバー情報**: プロキシ サーバーを使用する場合は、そのホスト名とポートを指定します。New Relic One プラットフォームにアクセスするための関連するプロキシ サーバー ポリシーを含めます。
