<Steps>
  <Step>
    **BTP subaccount**\[BTP サブアカウント]にログインします。
  </Step>

  <Step>
    **BTP cockpit**\[BTP コックピット]で、 **Instances and Subscriptions**\[インスタンスとサブスクリプション]をクリックします。
  </Step>

  <Step>
    右上の**Create**\[作成]をクリックします。
  </Step>

  <Step>
    ポップアップ画面で以下を選択します。

    * **サービス**: API管理、APIポータル
    * **プラン**: apiportal-apiaccess
    * **ランタイム環境**: Cloud Foundry
    * **スペース**: BTP アドミニストレーターに確認してください
    * **インスタンス名**: 組織の命名規則に従ってください
  </Step>

  <Step>
    費用に関する免責事項を確認してください。
  </Step>

  <Step>
    **Next**\[次へ]をクリックし、 **Roles**\[ロール]: `APIPortal.Administrator`を選択します。
  </Step>

  <Step>
    **Next**\[次へ]をクリックして詳細を確認します。
  </Step>

  <Step>
    **Create** \[作成]をクリックします。
  </Step>

  <Step>
    作成されたら、 **Instances**\[インスタンス]タブでサービスを見つけます。
  </Step>

  <Step>
    インスタンスの横にある省略記号 ( `...` ) をクリックし、 **Create Service Key**\[サービス キーの作成] をクリックします。
  </Step>

  <Step>
    Service Key Name \[サービス キー名] と **Credential Type**\[資格情報の種類]: binding-secret を JSON 形式で入力します。
  </Step>

  <Step>
    **Create**\[作成]をクリックし、サービス キーの横にある省略記号 ( `...` ) をクリックしてダウンロードします。
  </Step>
</Steps>
