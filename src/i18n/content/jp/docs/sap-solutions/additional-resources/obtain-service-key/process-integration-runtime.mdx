<Steps>
  <Step>
    **BTP subaccount**\[BTP サブアカウント]にログインします。
  </Step>

  <Step>
    **BTP cockpit**\[BTP コックピット]で、 **Instances and Subscriptions**\[インスタンスとサブスクリプション]をクリックします。
  </Step>

  <Step>
    右上の**Create**\[作成]をクリックします。
  </Step>

  <Step>
    ポップアップ画面で以下を選択します。

    * **サービス**: SAP プロセス統合ランタイム。
    * **プラン**: API。
    * **ランタイム環境**: Cloud Foundry。
    * **スペース**: BTP アドミニストレーターに確認してください。
    * **インスタンス名**: 組織の命名規則に従います。

    <img title="Process Integration Instance" alt="A screenshot displaying the Process Integration Instance" src="/images/process-integration-instance.webp" width="600" />
  </Step>

  <Step>
    **Next**\[次へ]をクリックし、以下を選択します。

    * **役割**: `MonitoringDataRead`
    * **付与タイプ**: デフォルトのクライアント資格情報 (他のタイプが使用されている場合は、BTP アドミニストレーターおよびセキュリティ チームに確認してください)。

    <img title="BTP Roles and Grant Types" alt="A screenshot displaying the BTP Roles and Grant Types" src="/images/btp-roles-grant-types.webp" width="600" />
  </Step>

  <Step>
    **Next**\[次へ]をクリックして詳細を確認します。
  </Step>

  <Step>
    **Create** \[作成]をクリックします。
  </Step>

  <Step>
    作成されたら、 **Instances**\[インスタンス]タブでサービスを見つけます。

    <img title="BTP Instance" alt="A screenshot displaying the BTP Instance" src="/images/btp-instance.webp" width="600" />
  </Step>

  <Step>
    インスタンスの横にある省略記号 ( `...` ) をクリックし、 **Create Service Key**\[サービス キーの作成] をクリックします。
  </Step>

  <Step>
    **Service Key Name**\[サービス キー名]と**Key Type**\[キー タイプ]を入力してください: `ClientId/Secret` 。

    <img title="BTP New Service Key" alt="A screenshot displaying the BTP New Service Key" src="/images/btp-new-service-key.webp" width="600" />
  </Step>

  <Step>
    **Create** \[作成]をクリックします。
  </Step>

  <Step>
    サービス キーの横にある省略記号 ( `...` ) をクリックし、 **Download**\[ダウンロード]をクリックします。

    <img title="BTP Download Service Key" alt="A screenshot displaying the BTP Download Service Key" src="/images/btp-download.webp" width="600" />
  </Step>
</Steps>
