<Steps>
  <Step>
    **CALM subaccount**\[CALM サブアカウント]にログインし、 **Instances and Subscriptions**\[インスタンスとサブスクリプション]をクリックします。
  </Step>

  <Step>
    右上の**Create**\[作成]をクリックします。
  </Step>

  <Step>
    ポップアップ画面で以下を選択します。

    * **サービス**: SAP Cloud ALM API
    * **プラン**：スタンダード
    * **ランタイム環境**: Cloud Foundry
    * **スペース**: BTP アドミニストレーターに確認してください
    * **インスタンス名**: 組織の命名規則に従ってください

    <img title="SAP Cloud ALM API" alt="A screenshot displaying the SAP Cloud ALM API" src="/images/sap-cloud-alm-api.webp" width="600" />
  </Step>

  <Step>
    **Next**\[次へ]をクリックします。
  </Step>

  <Step>
    [「スコープの管理」](https://help.sap.com/docs/cloud-alm/apis/managing-scopes)から必要なスコープを確認し、インスタンス名とともに「問題」タブに挿入します。

    <img title="Manage Scopes" alt="A screenshot displaying the Manage Scopes" src="/images/manage-scope.webp" width="600" />
  </Step>

  <Step>
    **Next**\[次へ]をクリックして詳細を確認します。

    <img title="Review Details" alt="A screenshot displaying the Review Details" src="/images/cloud-alm-new-instance.webp" width="600" />
  </Step>

  <Step>
    **Create** \[作成]をクリックします。
  </Step>

  <Step>
    作成されたら、 **Instances**\[インスタンス]タブでサービスを見つけます。
  </Step>

  <Step>
    インスタンスの横にある省略記号 ( `...` ) をクリックし、 **Create Service Key**\[サービス キーの作成] をクリックします。
  </Step>

  <Step>
    サービス キー名と**キー タイプ**を入力してください: `ClientId/Secret` 。
  </Step>

  <Step>
    **Create** \[作成]をクリックします。
  </Step>

  <Step>
    サービス キーの横にある省略記号 ( `...` ) をクリックし、 **Download**\[ダウンロード]をクリックします。
  </Step>
</Steps>
