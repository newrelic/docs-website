サービス パックは、命名規則`K-<Version Number>SP<SP Number>IN<Software Component>.SAR`を持つ`.SAR`ファイルとして提供されます。

<Steps>
  <Step>
    トランザクション`SPAM`を使用してサービス パックをインストールします。
  </Step>

  <Step>
    クライアント000から新しい設定テンプレートを読み込みます。
  </Step>

  <Step>
    クライアント 000 から **Install New Release and Patch** &gt; **Copy Template Tables** に移動します。
  </Step>

  <Step>
    **Install New Release and Patch** &gt; **Generate Configuration for New Data Providers** の手順に従って、新しいデータ プロバイダーの構成を生成します。
  </Step>

  <Step>
    **Install New Release and Patch**\[新しいリリースとパッチのインストール] &gt; **Verify Installation Completeness**\[インストレーションの完全性を確認]をクリックして、インストレーションを確認します。
  </Step>
</Steps>
