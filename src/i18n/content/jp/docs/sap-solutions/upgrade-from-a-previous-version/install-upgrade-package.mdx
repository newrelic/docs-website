以前のバージョンのNew Relic監視 for SAP Solutions からアップグレードするには、現在インストールされているバージョンに固有の正しいアップグレード パッケージを使用していることを確認してください。

1. **System &gt; Status**に移動します。
2. 製品バージョンを識別するには、 **NEWR**を検索してください。
3. クライアント 000 でトランザクション**SAINT**を実行して ABAP アドオンを更新します。

## 設定をバックアップして更新します。

<Callout variant="tip">
  アップグレード中に上書きされないように、変更したデータ プロバイダー設定またはカスタム データ プロバイダーを設定テンプレートに保存します。 V3.0 ABAP アドオンがインストールされていない場合でも、クライアント 000 から新しいテンプレートをコピーする前に、テンプレートが保存されていることを確認してください。
</Callout>

1. メニュー`/n/NEWR/INSTALL`にアクセスし、**設定テンプレートの更新または保存**に移動します。

2. Java 監視に関連する HDB 監視テンプレートを削除するには、 `/NEWR/REP_DELETE_INS_TEMPLATE`を実行します。

3. `DBACockpit`を使用して、HANA データベースへの ABAP データ接続を設定します。

4. クライアント000から設定テンプレートをコピーします。

5. 監視システムの設定を変更します。

   * 新しい DB 接続を追加します。
   * Java Enterprise Connector (例: NR\_EI\_AGENT) への参照を削除します。
   * 監視グループに HANA\_DB を追加します。
   * 設定を更新するには、データ プロバイダーの構成を再生成します。
