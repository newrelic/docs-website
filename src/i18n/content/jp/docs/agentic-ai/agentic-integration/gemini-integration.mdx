New Relicエージェント for Google Gemini Code Assist 拡張機能は、アプリケーションのパフォーマンス、モニターインフラストラクチャ、デプロイメントの管理を最適化するために調整されたインテリジェントなインサイトとガイダンスを提供し、すべて Gemini 内にシームレスに統合されます。

このガイドでは、Gemini Code Assist 内で New Relic エージェントをインストールして使用するプロセスについて説明します。

## Gemini Code Assist にNew Relic Agent をインストールする [#install-new-relic-agent]

### 前提条件 [#prerequisites]

始める前に、セットアップ プロセスがスムーズに進むように、次の前提条件が満たされていることを確認してください。

* IDE がシステムにインストールされています。
* Google Gemini Code Assist にアクセスできます。
* Gemini Code Assist が IDE にインストールされます。 詳細については、 [Google Gemini Code Assist のドキュメント](https://cloud.google.com/gemini/docs/codeassist/overview)を参照してください。

<Callout variant="tip" title="ヒント">
  特定のロールに対して Gemini Code Assist へのアクセスを制限する場合は、Gemini Code Assist で設定を構成します。 特定のロールまたは社員に対してNew Relic AIへのアクセスを制限するには、 New Relicのロールベース アクセス制御 (RBAC) を使用できます。 詳細については、 [New Relic ユーザー管理を](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts)参照してください。
</Callout>

**Gemini Code Assist で New Relic エージェントを有効にするには:**

1. <DNT>Google Cloud Console</DNT>で<DNT>**NewRelic**</DNT>を検索し、拡張機能を有効にします。 サインインダイアログウィンドウが開きます。

2. ログインするには、New Relic の資格情報を入力してください。 **New Relic successfully linked.** \[New Relic が正常にリンクされました。]メッセージが表示されます。

   これで、IDE の Gemini Code Assist チャット拡張機能内で New Relic エージェントを使用できるようになりました。 詳細については、 [「Gemini Code Assist での New Relic エージェントの使用」を](#communicate-with-new-relic-agent-in-gemini-code-assist)参照してください。

## Gemini Code Assist で New Relic エージェントと通信する [#communicate-with-new-relic-agent-in-gemini-code-assist]

Gemini Code Assist 内のNew Relicエージェントは、さまざまなタスクを支援し、価値のあるサイトを提供します。 New Relic エージェントと対話するには、チャット入力ボックスに`@newrelic`と入力します。

### チャットメッセージの構造 [#chat-message-structure]

New Relic エージェントと対話するには、次のようにプロンプトを入力します。

`@newrelic <your message>`

どこ：

* `@newrelic`: New Relicエージェントによるインターラクション。
* `<your message>`: 処理のためにエージェントに送信するアクション。
