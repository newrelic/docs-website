Google Gemini Code Assist 拡張機能用のNew Relicツールは、アプリケーションのパフォーマンス、モニターインフラストラクチャ、デプロイメントの管理を最適化するために調整されたインテリジェントなインサイトとガイダンスを提供し、すべて Gemini Code Assist 内にシームレスに統合されます。

このガイドでは、Gemini Code Assist 内で New Relic ツールをインストールして使用するプロセスについて説明します。

## Gemini Code AssistでNew Relicツールを有効にする [#enable-new-relic-tool]

### 前提条件 [#prerequisites]

始める前に、セットアップ プロセスがスムーズに進むように、次の前提条件が満たされていることを確認してください。

* 統合開発環境 (IDE) がコンピューターにインストールされています。
* Google Gemini Code Assist にアクセスできます。
* Gemini Code Assist が IDE にインストールされます。 詳細については、 [Google Gemini Code Assist のドキュメント](https://cloud.google.com/gemini/docs/codeassist/overview)を参照してください。

<Callout variant="tip" title="ヒント">
  特定のロールに対して Gemini Code Assist へのアクセスを制限する場合は、Gemini Code Assist で設定を構成します。 特定のロールまたは社員に対してNew Relic AIへのアクセスを制限するには、 New Relicのロールベース アクセス制御 (RBAC) を使用できます。 詳細については、 [New Relic ユーザー管理を](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts)参照してください。
</Callout>

New Relic ツールを有効にするには。

1. IDE から、「Google でサインイン」をクリックして[Gemini Code Assist](https://codeassist.google.com)にログインします。

2. 認証に成功したら、IDE に戻ります。

3. <DNT>Gemini Code Assist Chat</DNT> （チャット）で`@newrelic`と入力して Enter キーを押します。New Relicツールをすぐに有効にすることができます。

4. プロンプトでリンクをクリックして、New Relic ツールを有効にします。ブラウザ ウィンドウが開き、Gemini Code Assist 内でNew Relicツールを有効にするプロンプトが表示されます。

5. New Relic ツールを有効にして、 **Connect** \[接続]をクリックします。

6. New Relic アカウントの詳細を使用してログインします。これにより、New Relic アカウントが Gemini Code Assist に正常に接続されます。

   これで、IDE の Gemini Code Assist チャット拡張機能内で New Relic ツールを使用できるようになります。

## New Relicツールを操作する [#new-relic-tool]

Gemini Code Assist 内のNew Relicツールは、さまざまなタスクを支援し、貴重な情報を提供します。 New Relic ツールと対話するには、チャット入力ボックスに次のようにプロンプトを入力します。

`@newrelic <your message>`

ここで、 `<your message>`ツールで実行するアクションです。例えば、 `@newrelic What is apdex?`
