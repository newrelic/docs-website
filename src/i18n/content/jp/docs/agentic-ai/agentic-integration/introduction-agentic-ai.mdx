New Relic&apos;s agentic integration provides an AI-driven solution to automate and optimize observability workflows, enhancing system management and monitoring. This integration facilitates seamless operations by automating repetitive tasks through AI agents, reducing manual intervention. By integrating seamlessly with platforms such as, GitHub Copilot, Gemini Code Assist, and ServiceNow, agentic integrations transform complex workflows into seamless operations.

## 主な特徴 [#features]

* **インテリジェントな自動化:** AI を活用して複雑なワークフローを自動化および合理化し、手動でのツール切り替えの必要性を減らして生産性を向上させます。
* **監視機能の強化:**包括的なインサイトをシステムに組み込むことで、インシデントの迅速な検出、分析、解決が可能になります。
* **シームレスな統合:** GitHub Copilot などの人気のあるプラットフォームに接続して、強力な分析とインサイトを既存のワークフローに直接導入します。

## アクセスとデータ管理 [#access-data-mgmt]

When enrolling in the Agentic Integrations, be aware that your New Relic users can enable any agentic integrations made available by New Relic at that time. If you desire to limit the users who have access to the agentic integrations, you may use [custom roles and groups](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial/#roles) to refine access.

さらに、弊社のインテグレーション パートナーはクライアント アプリケーションを管理し、お客様またはお客様のユーザーがパートナーのインテグレーションを通じてNew Relicに認証されるときにユーザー アカウントを維持します。 サードパーティ パートナーは、トークンが代理で使用されるため、ユーザー設定で構成された期間、認証したユーザーと同じレベルの New Relic へのアクセスを持ちます。この配置は、シームレスな統合と機能の提供に役立ちます。 詳細については、トークンの使用方法と管理方法を詳しく説明したパートナーのドキュメントを参照してください。

## 現在の統合 [#integrations]

### GitHub コパイロット [#github-copilot]

GitHub Copilot を使用して New Relic インサイトを IDE に直接統合します。 エージェント AI を使用すると、コーディング環境を離れることなく、ログ、エラー、推奨事項にアクセスできます。 彼らがコードを開発してデプロイする際に、 オンラインサイトを提供します。

この統合により以下が提供されます。

* 自動デプロイメント チェック: New Relic のデータを活用して、デプロイメント チェックを自動的に実行します。
* リアルタイムのアラートとエラー追跡: 即時のアラートで情報を入手し、エラーが発生するとすぐに追跡します。
* デプロイメントアクションに対する確率的な推奨事項。

詳細については、 [「GitHub Copilot 用 New Relic エージェントの使用開始」を](/docs/agentic-ai/agentic-integration/github-copilot-integration)参照してください。

### ジェミニコードアシスト [#gemini-code-assist]

Gemini Code Assist を使用してNew Relic RelicInサイトを IDE に統合します。 このツールを使用すると、開発者はコーディング環境内の関連するログ、エラー情報、推奨事項にアクセスでき、開発およびデプロイメントのプロセスを知らせることができるインサイトが提供されます。

この統合により以下が提供されます。

* 自動化されたデプロイメント監視: New Relicの保有データを使用してデプロイメントを自動的に監視し、問題を効率的に特定して対処します。
* アラートとエラー追跡: 即座に通知を受け取り、エラーを効果的に追跡して、アプリケーションのパフォーマンスと安定性を維持します。
* デプロイメント アクションの推奨事項: データ分析に基づく推奨事項を活用して、コードのパフォーマンスを向上させ、意思決定をサポートします。

詳細については、 [「Gemini Code Assist 用 New Relic エージェントの使用開始」を](/docs/agentic-ai/agentic-integration/gemini-integration)参照してください。
