New Relicのエージェント統合は、特定の監視ワークフローの自動化と最適化を支援する AI 主導のソリューションを提供し、システム管理と監視を強化します。 この統合により、AI によって反復的なタスクが自動化され、手動による介入が減り、シームレスな運用が促進されます。 GitHub Copilot、Gemini Code Assist、ServiceNow などのプラットフォームとシームレスに統合することで、エージェント統合により複雑なワークフローがシームレスなオペレーションに変換されます。

## 主な特徴 [#features]

* **インテリジェントな自動化:** AI を活用して複雑なワークフローを自動化および合理化し、手動でのツール切り替えの必要性を減らし、生産性を向上させながら、人間を常に監視します。
* **監視機能の強化:**包括的なインサイトをシステムに組み込むことで、インシデントの迅速な検出、分析、解決が可能になります。
* **シームレスな統合:** GitHub Copilot などの人気のあるプラットフォームに接続して、強力な分析とインサイトを既存のワークフローに直接導入します。

## アクセスとデータ管理 [#access-data-mgmt]

当社の統合パートナーは、お客様またはお客様のユーザーがパートナーの統合を通じてNew Relicに認証されるときに、クライアント アプリケーションを管理し、ユーザーアカウントを維持します。 サードパーティ パートナーは、トークンが代理で使用されるため、ユーザー設定で構成された期間、認証したユーザーと同じレベルの New Relic へのアクセスを持ちます。この配置は、シームレスな統合と機能の提供に役立ちます。 詳細については、トークンの使用方法と管理方法を詳しく説明したパートナーのドキュメントを参照してください。

### ロールベースのアクセス制御 (RBAC)

New Relicのエージェント統合機能は、 New Relicの RBAC システム内の組織スコープの役割を介して付与されます。 ユーザーは、これらの機能を含む組織の役割が割り当てられたグループに属することでアクセス権を取得します。個人アカウントでは直接設定は必要ありません。

現在、RBAC には API インターフェースを通じてアクセスできます。API 経由で RBAC を利用することにご興味がある場合は、 [agenticai@newrelic.com](mailto:agenticai@newrelic.com)までお気軽にお問い合わせください。RBAC の設定に関する詳細なガイダンスについては、 [アカウント ユーザー管理チュートリアル](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial)を参照してください。

## 現在の統合 [#integrations]

### GitHub コパイロット [#github-copilot]

GitHub Copilot を使用して、 New Relicサイトを IDE に直接統合します。 エージェント AI を使用すると、コーディング環境を離れることなく、ログ、エラー、推奨事項にアクセスできます。 コードを開発してデプロイする際に、ウォレット インサイトが提供されます。

この統合により以下が提供されます。

* 自動デプロイメント チェック: New Relic のデータを活用して、デプロイメント チェックを自動的に実行します。
* リアルタイムのアラートとエラー追跡: 即時のアラートで情報を入手し、エラーが発生するとすぐに追跡します。
* デプロイメントアクションに対する確率的な推奨事項。

詳細については、 [「GitHub Copilot 用 New Relic エージェントの使用開始」](/docs/agentic-ai/agentic-integration/github-copilot-integration)を参照してください。

### ServiceNow Now アシスト [#now-assist]

New Relic AI Site は ServiceNow の Now Assist 内にあります。 この統合により、ファーストレスポンダーやサポートエージェントなどの運用チームは、ServiceNow ワークスペースを離れることなく、インシデントの重大度を迅速に理解し、考えられる根本原因を特定し、コラボレーションを合理化することができます。

詳細については、 [「ServiceNow Now Assist 用 New Relic エージェントの使用開始」](/docs/agentic-ai/agentic-integration/servicenow-integration)を参照してください。

### ジェミニコードアシスト [#gemini-code-assist]

<Callout title="プレビュー">
  この機能はまだ開発中ですが、ぜひお試しください。

  この機能は現在、弊社の[プレリリース ポリシー](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy)に従ってプレビュー プログラムの一部として提供されています。
</Callout>

Gemini Code Assist を使用してNew Relic RelicInサイトを IDE に統合します。 このツールを使用すると、開発者はコーディング環境内の関連するログ、エラー情報、推奨事項にアクセスでき、開発およびデプロイメントのプロセスを知らせることができるインサイトが提供されます。

この統合により以下が提供されます。

* 自動化されたデプロイメント監視: New Relicの保有データを使用してデプロイメントを自動的に監視し、問題を効率的に特定して対処します。
* アラートとエラー追跡: 即座に通知を受け取り、エラーを効果的に追跡して、アプリケーションのパフォーマンスと安定性を維持します。
* デプロイメント アクションの推奨事項: データ分析に基づく推奨事項を活用して、コードのパフォーマンスを向上させ、意思決定をサポートします。

詳細については、 [「Gemini Code Assist 用 New Relic エージェントの使用開始」を](/docs/agentic-ai/agentic-integration/gemini-integration)参照してください。
