New Relicの Agentic インテグレーションは、監視ワークフローを自動化および最適化し、システム管理と監視を強化する AI 主導のソリューションを提供します。 この統合により、AI によって反復的なタスクが自動化され、手動による介入が減り、シームレスな運用が促進されます。 GitHub Copilot、Amazon Q Business、Gemini Code Assist、ServiceNow などのプラットフォームとシームレスに統合することで、Agentic 統合は複雑なワークフローをシームレスなオペレーションに変換します。

## 主な特徴 [#features]

* **インテリジェントな自動化:** AI を活用して複雑なワークフローを自動化および合理化し、手動でのツール切り替えの必要性を減らして生産性を向上させます。
* **監視機能の強化:**包括的なインサイトをシステムに組み込むことで、インシデントの迅速な検出、分析、解決が可能になります。
* **シームレスな統合:** GitHub Copilot などの人気のあるプラットフォームに接続して、強力な分析とインサイトを既存のワークフローに直接導入します。

## アクセスとデータ管理 [#access-data-mgmt]

Agentic 統合プレビューに登録する場合、 New Relicユーザーは、その時点でNew Relicによって利用可能になっているエージェント統合を有効にできることに注意してください。 エージェント統合にアクセスできるユーザーを制限したい場合は、 [カスタムの役割とグループ](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial/#roles)を使用してアクセスを絞り込むことができます。

さらに、弊社のインテグレーション パートナーはクライアント アプリケーションを管理し、お客様またはお客様のユーザーがパートナーのインテグレーションを通じてNew Relicに認証されるときにユーザー アカウントを維持します。 サードパーティ パートナーは、トークンが代理で使用されるため、ユーザー設定で構成された期間、認証したユーザーと同じレベルの New Relic へのアクセスを持ちます。この配置は、シームレスな統合と機能の提供に役立ちます。 詳細については、トークンの使用方法と管理方法を詳しく説明したパートナーのドキュメントを参照してください。

## 現在の統合 [#integrations]

### GitHub コパイロット [#github-copilot]

GitHub Copilot を使用して New Relic インサイトを IDE に直接統合します。 エージェント AI を使用すると、コーディング環境を離れることなく、ログ、エラー、推奨事項にアクセスできます。 彼らがコードを開発してデプロイする際に、 オンラインサイトを提供します。

この統合により以下が提供されます。

* 自動デプロイメント チェック: New Relic のデータを活用して、デプロイメント チェックを自動的に実行します。
* リアルタイムのアラートとエラー追跡: 即時のアラートで情報を入手し、エラーが発生するとすぐに追跡します。
* デプロイメントアクションに対する確率的な推奨事項。

詳細については、 [「GitHub Copilot 用 New Relic エージェントの使用開始」を](/docs/agentic-ai/agentic-integration/github-copilot-integration)参照してください。

This integration offers: - Conversational AI Interface: Engage with New Relic AI through Amazon Q Business conversational interface to obtain insights and recommendations. - Targeted Alert Analysis: Isolate and scrutinize alerts specific to key services, like Checkout and Payment Processing, within a defined timeframe. - Comprehensive Alert Intelligence: Access in-depth reports on anomalies, error frequencies, and latency concerns, complemented by links to New Relic dashboards for further analysis. - Proactive Recommendations: Receive informed suggestions on the necessity of additional investigation to preemptively address potential issues. For additional details, refer \[Getting Started with New Relic agent for Amazon Q Business]\(/docs/agentic-ai/agentic-integration/amazonq-integration). \*/

### ジェミニコードアシスト [#gemini-code-assist]

Gemini Code Assist を使用してNew Relic RelicInサイトを IDE に統合します。 このツールを使用すると、開発者はコーディング環境内の関連するログ、エラー情報、推奨事項にアクセスでき、開発およびデプロイメントのプロセスを知らせることができるインサイトが提供されます。

この統合により以下が提供されます。

* 自動化されたデプロイメント監視: New Relicの保有データを使用してデプロイメントを自動的に監視し、問題を効率的に特定して対処します。
* アラートとエラー追跡: 即座に通知を受け取り、エラーを効果的に追跡して、アプリケーションのパフォーマンスと安定性を維持します。
* デプロイメント アクションの推奨事項: データ分析に基づく推奨事項を活用して、コードのパフォーマンスを向上させ、意思決定をサポートします。

詳細については、 [「Gemini Code Assist 用 New Relic エージェントの使用開始」を](/docs/agentic-ai/agentic-integration/gemini-integration)参照してください。

## ツール [#tools]

各エージェント統合には、New Relic プラットフォーム機能を使用し、インサイト、アラート、予測の監視を提供するためのさまざまなツールが付属しています。

* **[Change Intelligence Tool](/docs/agentic-ai/tools/change-intelligence-tool)**\[変更インテリジェンス ツール]:変更インテリジェンス ツールは、包括的な変更分析と予測を提供することで、潜在的なリスクを理解し、軽減するのに役立ちます。 このツールを使用すると、インサイトにシステム変更の影響を即座に反映させることができます。 これは、デプロイメントや設定の変更に関連する潜在的なリスクを評価するのに役立ちます。 詳細については、 [Change Intelligence Tool](/docs/agentic-ai/tools/change-intelligence-tool)を参照してください。

* **[Alert Intelligence Tool](/docs/agentic-ai/tools/alert-intelligence-tool)** \[一括インテリジェンス ツール]:一括インテリジェンス ツールは、インシデントを迅速に解決するための高度な一括分析を提供します。 エスカレーションを防ぐために、リアルタイムの通知と即時の推奨事項を提供します。 詳細については、 [Alert Intelligence Tool](/docs/agentic-ai/tools/alert-intelligence-tool)を参照してください。
