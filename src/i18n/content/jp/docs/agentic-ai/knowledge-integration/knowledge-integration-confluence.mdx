New Relicのナレッジ コネクタ (取得および生成) 機能と confluence の統合により、New Relic New Relic AIの応答内の confluence コンテンツを直接参照することで、トラブルシューティングおよび文書化のプロセスを強化できます。 New RelicにAPIキーと confluence 書き込みを提供すると、NRAI のすべてのエクスペリエンスでこの情報を活用する AI インサイトがすぐに得られます。

## ナレッジコネクタをConfluenceに接続する

Confluence コンテンツのインデックス作成を開始し、New Relic AI のナレッジ コネクタを活用するには、次の手順に従います。

<Steps>
  <Step>
    <DNT>[**one.newrelic.com**](https://one.newrelic.com) &amp;gt; **Integrations &amp; Agents**</DNT>に移動します。
  </Step>

  <Step>
    <DNT>**RAG Manager**</DNT>を検索します。
  </Step>

  <Step>
    <DNT>**Add Confluence**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Tools**</DNT>タブでは、使用可能なすべての RAG ツールを確認できます。

    <Callout variant="important">
      インデックスを作成したすべてのドキュメントを表示するには、「ドキュメント」タブをクリックします。
    </Callout>
  </Step>

  <Step>
    Confluence コネクタを設定するには、 <DNT>**Add Tools &gt; Confluence**</DNT>をクリックします。
  </Step>

  <Step>
    Confluence 資格情報を保存するドロップダウンから<DNT>**Account**</DNT>選択し、 <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Confluence Base URL**</DNT>フィールドに、Confluence インスタンスのベース URL を入力します。 たとえば、 <DNT>[https://your-site-company.atlassian.net/wiki](https://your-site-company.atlassian.net/wiki)</DNT>をクリックして<DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    次の情報を入力します。

    * <DNT>**Confluence Username**</DNT>: Confluence のユーザー名。
    * <DNT>**API Token**</DNT>: Confluence API トークン。
  </Step>

  <Step>
    <DNT>**Save token in secret store**</DNT>クリックし、 <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    Confluence 設定セクションで、次の情報を入力します。

    1. <DNT>**Name your cofluence documents**</DNT>、Confluence ドキュメントの一意の名前を入力します (例<DNT>**Incident Runbook**</DNT> 。

    2. 内容を絞り込むには、 <DNT>**Confluence query**</DNT>を入力します (例)。 <DNT>**space=SPACEKEY**</DNT> .

    3. **Confluence reindex interval** \[Confluence の再インデックス間隔]セクションから、Confluence コンテンツを取得して再インデックスする頻度を選択します。次のオプションから選択できます。

       * <DNT>**Monthly**</DNT>
       * <DNT>**Weekly**</DNT>
       * <DNT>**Daily**</DNT>

    4. <DNT>**Save confluece settings**</DNT>クリックし、 <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Start Indexing**</DNT>をクリックします。 新しいツールが作成され、ツール タブに追加されます。
  </Step>

  <Step>
    インデックスを作成したドキュメントを表示するには、 <DNT>**Document**</DNT>タブをクリックします。
  </Step>
</Steps>
