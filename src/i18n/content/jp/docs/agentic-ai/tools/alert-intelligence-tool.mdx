New Relic のエージェント統合内の統合インテリジェンス ツールは、パートナーとのエージェント統合を通じてアプリケーションの健全性とパフォーマンスを評価する能力を強化します。 このツールを使用すると、ユーザーは自分のアプリケーションに関連するアラートや異常について問い合わせることができ、臨時インサイトが提供され、効率的なトラブルシューティングが促進されます。 このツールは、アプリケーションの累積履歴と現在の問題に関する詳細なインサイトを提供し、発生する可能性のある問題を迅速に特定して対処できるようにし、スムーズで信頼性の高いユーザー エクスペリエンスを保証します。

New Relic プラットフォームにアクセスすると、アプリケーションに関連するすべてのデータを包括的に可視化できます。

### 統計インテリジェンス機能 [#features]

集中インテリジェンス ツールは、アプリケーションのパフォーマンスの監視と分析に役立つさまざまな機能を提供します。

* **集計レポート**: 優先パートナーのエージェント チャット エクスペリエンス内で直接、主要な集計メトリクスの包括的な概要を提供します。

  * **セキュリティ脆弱性:**監視期間内に検出された新たなセキュリティ脆弱性の特定。
  * **ログの問題:**エラー、警告、またはその他のログ関連の問題の検出。
  * **トレース情報:**詳細な分析のための詳細なトレース データ。
  * **ゴールデンシグナル:**レイテンシ、トラフィック、エラー、または飽和度の異常な値を示すアラート。
  * **閾値アラート:**事前に定義された閾値を超えたアラートを通知します。
  * **エラー報告:**包括的なエラー追跡および報告機能。
  * **直接リンク:**発見された問題をさらに調査するための New Relic への直接リンクを提供し、サイトの詳細な調査と分析を可能にします。

* **一括分析**: 一括の種類、重大度、発生した問題など、集計履歴の詳細な分析を提供します。
