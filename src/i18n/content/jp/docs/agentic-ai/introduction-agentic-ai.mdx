New Relic の Agentic インテグレーションは、監視ワークフローを自動化および最適化し、システム管理と監視を強化する AI 主導のソリューションを提供します。 この統合により、AI によって反復的なタスクが自動化され、手動による介入が減り、シームレスな運用が促進されます。 GitHub Copilot、Amazon Q、Gemini Code Assist、ServiceNow などのプラットフォームとシームレスに統合することで、Agentic 統合は複雑なワークフローをシームレスなオペレーションに変換します。

## 主な特徴 [#features]

* **インテリジェントな自動化:** AI を活用して複雑なワークフローを自動化および合理化し、手動でのツール切り替えの必要性を減らして生産性を向上させます。
* **監視機能の強化:**包括的なインサイトをシステムに組み込むことで、インシデントの迅速な検出、分析、解決が可能になります。
* **シームレスな統合:** GitHub Copilot などの人気のあるプラットフォームに接続して、強力な分析とインサイトを既存のワークフローに直接導入します。

## 現在の統合 [#integrations]

### GitHub コパイロット [#github-copilot]

GitHub Copilot を使用して New Relic インサイトを IDE に直接統合します。 エージェント AI を使用すると、コーディング環境を離れることなく、ログ、エラー、推奨事項にアクセスできます。 彼らがコードを開発してデプロイする際に、 オンラインサイトを提供します。

この統合により以下が提供されます。

* 自動化されたデプロイメント チェック。
* リアルタイムのアラートとエラー追跡。
* デプロイメントアクションに対する確率的な推奨事項。

詳細については、 [「Getting Started with New Relic Agent for GitHub Copilot」を](/docs/agentic-ai/github-copilot-integration)参照してください。

## ツール [#tools]

各エージェント統合には、New Relic プラットフォーム機能を使用し、インサイト、アラート、予測の監視を提供するためのさまざまなツールが付属しています。

* **[変更インテリジェンス ツール:](/docs/agentic-ai/change-intelligence-tool)**変更インテリジェンス ツールは、包括的な変更分析と予測を提供することで、潜在的なリスクを理解し、軽減するのに役立ちます。 このツールを使用すると、インサイトにシステム変更の影響を即座に反映させることができます。 これは、デプロイメントや設定の変更に関連する潜在的なリスクを評価するのに役立ちます。 詳細については、 [Change Intelligence Tool](/docs/agentic-ai/change-intelligence-tool)を参照してください。

* **[一括インテリジェンス ツール](/docs/agentic-ai/alert-intelligence-tool):**一括インテリジェンス ツールは、インシデントを迅速に解決するための高度な一括分析を提供します。 エスカレーションを防ぐために、リアルタイムの通知と即時の推奨事項を提供します。 詳細については、 [「積算インテリジェンス ツール」](/docs/agentic-ai/alert-intelligence-tool)を参照してください。
