パフォーマンス監視用のネットワーク データをセットアップしたら、 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Network**に移動して、デバイスが New Relic に報告しているネットワーク データを調べます。

ダッシュボードでスイッチやルーターなどをグループ化して監視できます。これにより、ネットワークの状態をすばやく集約、分離、分析できます。

## 前提条件 [#prerequisites]

* NewRelicアカウント。持っていませんか？[無料でお申し込み頂けます！](https://newrelic.com/signup)クレジットカードは必要ありません。

## ネットワークパフォーマンスデータの調査を開始します [#explore-network-performance-data]

**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Network monitoring**に移動すると、ネットワーク データを確認できます。
