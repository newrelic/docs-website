パフォーマンス監視用にネットワーク データを設定したら、 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Network**</DNT>に移動して、デバイスが New Relic に報告しているネットワーク データを調べます。

ダッシュボードでスイッチやルーターなどをグループ化して監視できます。これにより、ネットワークの状態をすばやく集約、分離、分析できます。

## 前提条件 [#prerequisites]

* NewRelicアカウント。持っていませんか？[無料でお申し込み頂けます！](https://newrelic.com/signup)クレジットカードは必要ありません。

## ネットワークパフォーマンスデータの調査を開始します [#explore-network-performance-data]

<DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Network monitoring**</DNT>にアクセスすると、ネットワーク データを確認できます。
