あなたは、SNMP 対応デバイスのネットワークに依存するオペレーション エンジニアです。あなたのチームは、ソフトウェアとインフラストラクチャの監視に New Relic の機能をすでに使用していますが、ネットワーク デバイスも監視するときが来ました。

このトラックでは、Kentik の ktranslate ライブラリを使用して、ネットワーク デバイスを自動的に検出し、データを収集し、そのデータを New Relic に送信して、ネットワーク パフォーマンスをインフラストラクチャ、アプリケーション、およびデジタル エクスペリエンスと関連付けます。次に、New Relic を使用してネットワークの状態を監視します。

## 目的

* ネットワークデバイスから SNMP データをポーリングし、New Relic に送信します
* ネットワーク フロー データをサンプリングして New Relic に送信する
* ネットワーク デバイスを監視するための New Relic ダッシュボードを作成する
* New Relic ワークロードを作成して、デバイスを論理的にグループ化し、異常検出を設定します
* 新しいデータを使用してネットワーク内の行動を理解する

## 要件

* 無料の New Relic アカウント
* 完全なプラットフォーム ユーザーまたは Nerdpack の変更ユーザー権限を持つコア ユーザー
