Pathpoint は、ほぼすべての New Relic アカウントと統合して、包括的なビジネス ジャーニーのオブザーバビリティを提供できます。開始するには、組織に関連するテレメトリ データを用意することが重要です。これには、メトリック、イベント、ログ、またはトレースを含めることができます。APM イベントとログは Pathpoint の一般的な出発点ですが、テレメトリ データの選択は完全にユーザー次第です。

Pathpoint を使い始める前に、New Relic アカウントにインストールする必要があります。

## New Relic カタログから Pathpoint をインストールする

パスポイントをインストールするには:

<Steps>
  <Step>
    [New Relic](https://one.newrelic.com) にログインするか、無料でサインアップしてください。

    <InlineSignup/>
  </Step>

  <Step>
    左側のナビゲーションから、 **+ Add Data**をクリックします。
  </Step>

  <Step>
    **Apps and Visualizations を**選択し、 `pathpoint`を検索します。
  </Step>

  <Step>
    検証済みの **パスポイント**のインストール アイコンをクリックし、 **\[このアプリを追加]**をクリックします。

    <img
      title="Install Pathpoint"
      alt="Install Pathpoint"
      src={installPathpoint}
    />
  </Step>

  Pathpoint が New Relic アカウントに追加され、 **Apps**でアクセスできるようになりました。
</Steps>

Pathpoint をインストールしたら、変更した JSON ファイルを Pathpoint にアップロードするだけで、その構成をカスタマイズできます。さらに、ステージ、ステップ、およびタッチポイントを変更し、現在の構成を JSON ファイルとして保存できます。 [パスポイントの構成について詳しくは、ドキュメント を](/docs/business-observability/configure-pathpoint/)参照してください。
