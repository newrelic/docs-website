アプリケーション カタログは、組織内のエンティティの観察と管理を強化する単一ペイン ビューを提供します。 サービス - APM 、サービス - OpenTelemetry 、 Browser 、モバイル の統合ビューにより、十分な情報に基づいた意思決定、運用の合理化、チーム間のコラボレーションの向上が促進されます。

### サポートされているエンティティ [#supported-entities]

現在の New Relic アプリケーション カタログ エクスペリエンスでは、次のエンティティがサポートされています。

* `Services - APM`
* `Services - OpenTelemetry`
* `Browser`
* `Mobile`

## アプリケーションカタログにアクセスする [#access-catalog]

**前提条件:**

* New Relic アカウントでカタログを有効にします。詳細については、 [「カタログを有効にする」](/docs/service-architecture-intelligence/catalogs/catalogs/#enable-catalogs)を参照してください。

**アプリケーション カタログ インベントリにアクセスするには:**

1. New Relic プラットフォームにログインします。

2. **左側のナビゲーション ペイン &gt; Catalogs &gt; Applications catalog**に移動します。

   /\* &lt;img title=&quot;カタログ&quot; alt=&quot;カタログ&quot; src=&quot;/images/new-relic-application-catalog.webp&quot; /&gt; \*/

### 新しいデータを追加する [#add-new-data]

アプリケーション カタログに追加データを追加するには、右上の<DNT>**+ Add data**</DNT>ボタンをクリックします。画面上の指示に従って、アプリケーション カタログにデータを追加します。New Relicがサポートするインテグレーションの詳細については、 [「インテグレーション」](https://newrelic.com/instant-observability)を参照してください。

### 制限 [#limitations]

* <DNT>Applications catalog</DNT>ページではデータの並べ替えはサポートされていません。
