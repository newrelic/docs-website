アプリケーション カタログは、組織内のエンティティの観察と管理を強化する単一ペイン ビューを提供します。 サービス - APM 、サービス - OpenTelemetry 、 Browser 、モバイル の統合ビューにより、十分な情報に基づいた意思決定、運用の合理化、チーム間のコラボレーションの向上が促進されます。

### サポートされているエンティティ [#supported-entities]

現在の New Relic アプリケーション カタログ エクスペリエンスでは、次のエンティティがサポートされています。

* `Services - APM`
* `Services - OpenTelemetry`
* `Browser`
* `Mobile`

## アプリケーションカタログにアクセスする [#access-catalog]

**前提条件:**

* New Relic アカウントでカタログを有効にします。詳細については、 [「カタログを有効にする」](/docs/service-architecture-intelligence/catalogs/catalogs/#enable-catalogs)を参照してください。

**アプリケーション カタログ インベントリにアクセスするには:**

1. New Relic プラットフォームにログインします。

2. **左側のナビゲーション ペイン &gt; Catalogs &gt; Applications catalog**に移動します。

## カタログを管理および整理する [#manage-organize]

カタログには、データを並べ替え、グループ化、フィルタリングするための柔軟なツールが用意されています。

### グループデータ [#group-data]

リストを論理的なセクションに整理するには、表の右上にある<DNT>**Group by**</DNT>ドロップダウンを使用します。エンティティは次の基準でグループ化できます。

* <DNT>**Entity type**</DNT>: APMサービスをBrowserまたはモバイル エンティティから分離します。
* <DNT>**Environment**</DNT>: 環境を区別します。たとえば、本番環境とステージング環境などです。
* <DNT>**Team**</DNT>: 責任ある所有権グループ別にサービスを表示します。
* <DNT>**Language**</DNT>: 使用するプログラミング言語ごとに整理します。
* <DNT>**AI asset**</DNT>: OpenAI や Bedrock などの検出された AI ライブラリごとにグループ化します。

### 表示される列をカスタマイズする [#customize-columns]

テーブルに表示するデータ ポイントを制御して、ノイズを減らし、特定のメトリクスに焦点を当てることができます。 <DNT>**Group by**</DNT>メニューの右側にある<DNT>**Column Selection**</DNT>アイコンを使用して、 <DNT>**Account**</DNT> 、 <DNT>**Last deployment**</DNT> 、 <DNT>**AI assets**</DNT> 、またはレスポンスタイムやスループットなどのゴールデンメトリクスなどのプロパティを表示または非表示にします。

### データをフィルタリングする [#filter-data]

上部のフィルター バーを使用してエンティティを絞り込みます。

* **クイック フィルター**: 事前定義されたボタンを使用して、エンティティ タイプ、AI アセット、またはチームでフィルターします。
* **カスタム フィルター ルール**: 特定の属性を追加するには、 <DNT>**+**</DNT>アイコンをクリックします。演算子を使用して特定の属性値をフィルタリングするルールを定義できます。たとえば、 <DNT>Language</DNT> `=` `go`のエンティティをフィルタリングして、すべての Go サービスを取得します。
* **グローバル検索**: <DNT>**Search by entity name**</DNT>バーを使用して特定のサービスを即座に検索します。

### 新しいデータを追加する [#add-new-data]

アプリケーション カタログに追加データを追加するには、右上の<DNT>**+ Add data**</DNT>ボタンをクリックします。画面上の指示に従って、アプリケーション カタログにデータを追加します。New Relicがサポートするインテグレーションの詳細については、 [「インテグレーション」](https://newrelic.com/instant-observability)を参照してください。
