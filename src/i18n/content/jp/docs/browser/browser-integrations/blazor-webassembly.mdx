Blazor WebAssembly ダッシュボードを使用すると、ゴールデン メトリックを簡単に監視し、エンド ユーザー エクスペリエンスを追跡し、経時的な合計ページ ビューを確認できます。

ブラウザ監視エージェントで構築された Blazor WebAssembly 統合により、最も重要な書き込みデータであるオールインワン プレイスを表示できる一連の既定の<InlinePopover type="dashboards"/>が提供されます。

## インフラストラクチャエージェントをインストールします [#infra]

Blazor WebAssembly を New Relic と統合するためにインフラストラクチャ エージェントは必要ありませんが、システムの完全な可観測性が必要な場合はインストールをお勧めします。インフラストラクチャ エージェントは、ホストに関するログやその他のデータを収集し、パフォーマンス メトリックをアプリケーション メトリックと共に表示します。

インフラストラクチャ エージェントをインストールするには、次の 2 つの方法があります。

* [ガイド付きインストール](https://one.newrelic.com/launcher/catalog-pack-details.launcher/?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5saXN0aW5nIn0=&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyJ9)に従って、システムにインフラストラクチャ エージェントをインストルメント化します。

## Blazor WebAssembly を New Relic と統合する [#integrate]

Blazor WebAssembly からデータを収集するには、次の手順に従います。

1. [one.newrelic.com](https://one.newrelic.com)から、 [<DNT>**Integrations & Agents**</DNT>ページ](https://one.newrelic.com/marketplace)に移動します。

2. <DNT>**Browser and mobile**</DNT>をクリックし、次に<DNT>**Browser monitoring**</DNT>をクリックします。

3. <DNT>**Select copy/paste JavaScript code**</DNT>オプションを選択します (このコードの詳細については、[ブラウザのインストールに関するドキュメント](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent)を参照してください)。

4. 残りの手順を完了して、 をクリックします。 <DNT>**Enable.**</DNT>

5. Blazor WebAssembly アプリケーションの `wwwroot` ディレクトリに移動します。

6. `index.html`ファイルを開きます。

7. body タグが終了する直前 ( `</body>` ) にブラウザー コード スニペットを貼り付けます。

8. アプリケーションを再起動します。

9. <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>に移動し、 <DNT>**Metrics & events**</DNT>をクリックします。 サンプル NRQL クエリを実行して、New Relic がデータを収集していることを確認します。

   ```sql
   SELECT average(windowLoad) FROM PageViewTiming WHERE (deviceType = 'Desktop') FACET `userAgentName` SINCE 604800 seconds AGO TIMESERIES
   ```

10. 既定のダッシュボードをインストールするには、 [Blazor WebAssembly インスタント オブザーバビリティ ページ](https://newrelic.com/instant-observability/blazor-web-assembly)に移動し、ページ上部の<DNT>**Install**</DNT>をクリックします。

11. <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>から<DNT>**Dashboards**</DNT>に移動し、Blazor WebAssembly をクリックしてダッシュボードを表示します。
