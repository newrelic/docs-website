Nuxt ダッシュボードを使用すると、アプリケーションのエラー率、初期ページ読み込み平均、ルートの変更、スループット、JavaScript エラーのあるページビューを簡単に追跡できます。Nuxt ダッシュボードは、インフラストラクチャエージェントと Vue <InlinePopover type="browser" />で構築されており、重要な情報をオールインワンで表示できます。

## インストール [#install]

<Steps>
  <Step>
    ### オプション: インフラストラクチャ エージェントをインストールする [#infra-install]

    インフラストラクチャ エージェントのインストールはオプションですが、これを行うと、アプリが実行されているホストに関する重要なメトリックが取得されます。

    これを行うには、 [インフラストラクチャ エージェントのインストール手順](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent)に従います。
  </Step>

  <Step>
    ### Nuxtブラウザ監視をインストールする [#browser-install]

    1. <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT>から[<DNT>**+ Integrations &amp; Agents**</DNT>ページ](https://one.newrelic.com/marketplace)に移動します。
    2. <DNT>**Nuxt**</DNT>を検索します。(注: Nuxt タイルはブラウザー コレクションやモバイル コレクションにはないため、検索機能を使用する必要があります。)
    3. 検索結果に表示された<DNT>**Nuxt**</DNT>タイルをクリックし、指示に従って JavaScript コード スニペットを生成します。
  </Step>

  <Step>
    ### 統合を構成する [#config]

    Nuxt アプリケーションに JavaScript スニペットを追加するには:

    1. Nuxt アプリケーションの<DNT>**`public`**</DNT>フォルダーに JavaScript ファイルを作成します。

       ```bash
       touch public/<FILENAME>.js
       ```

    2. Browserエージェント スニペットを、 `<script>`タグなしで新しい JavaScript ファイルにコピーして貼り付けます（スニペット自体のみで、定義する関数はありません）。

       ```js
       // PASTE THE GENERATED BROWSER AGENT SNIPPET HERE WITHOUT <script> TAGS
       ```

    3. `nuxt.config.js` (または`nuxt.config.ts`) ファイルを開き、 `app.head`セクションにスクリプトを追加します。

       ```js
       export default defineNuxtConfig({
         compatibilityDate: '2025-07-15',
         devtools: { enabled: true },
         app: {
           head: {
             script: [
               {
                 src: "/<FILENAME>.js"
               },
             ],
           },
         },
       })
       ```

    4. アプリケーションを再起動します。

    数分以内に、アプリケーションはメトリクスを [one.newrelic.com](https://one.newrelic.com)に送信します。
  </Step>
</Steps>

## アプリケーションを監視する [#monitor]

これで、ナビゲーション バーから<DNT>**Browser**</DNT>を選択すると、 <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT>でアプリケーションを確認できるようになります。 アプリケーションを選択すると、 <DNT>**one.newrelic.com**</DNT>プラットフォームでキャプチャされたすべてのメトリックが表示されます。 次のクエリを使用して、NRQL でデータを表示することもできます。

```sql
SELECT firstPaint
FROM PageView
```
