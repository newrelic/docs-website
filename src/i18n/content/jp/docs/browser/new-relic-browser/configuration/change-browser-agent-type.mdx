Browser Agent タイプは課金に影響しません。 これは、必要に応じて、より最小限の機能セットに切り替えるための単なる方法です。 インストールされたデフォルトの Browseragent では、すべての<InlinePopover type="browser"/>機能が有効になっています。

browserエディションの種類を変更するには、<DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > Settings > Application settings**</DNT> に移動してください。

以下に、さまざまなBrowserエージェントタイプの比較を示します。

## Browserエージェントのタイプ [#agent-types]
