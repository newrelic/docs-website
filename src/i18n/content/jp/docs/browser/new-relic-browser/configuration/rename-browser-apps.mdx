ブラウザアプリをブラウザに追加する際に使用した [配置オプション](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser) に応じて、適切な手順でブラウザアプリの名前を変更してください。

## APMで監視するアプリの名称変更 [#apm-app]

既にAPMが監視しているアプリをブラウザに追加した場合、ブラウザに表示されるアプリ名はAPM UIのアプリ名と同じです。アプリの名前を変更するには、以下のいずれかのオプションを使用します。

## JavaScriptのスニペットでアプリの名前を変更 [#javascript]

この情報は、ブラウザエージェントのJavaScriptスニペットを挿入した場合に適用されます。

* APMが監視しているアプリで、 [のコピー/ペースト方式で導入されたもの。](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#copy-paste-app)
* シングルページアプリ ([SPA](/docs/browser/single-page-app-monitoring/get-started/add-apps-single-page-app-monitoring))
* APMによって監視されていないスタンドアロンのブラウザアプリ

既存のアプリでは、JavaScriptのスニペットを直接編集することはできません。代わりに、別の名前であなたのアプリを使用するために。

1. 標準的な手順に従って、 [コピー／ペースト方式で新しいアプリを作成します。](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#copy-paste-app) 。新しい名前を付けて、JavaScriptのスニペットを生成します。
2. 新しいJavaScriptスニペットをコピーします。
3. Webページでは、元のアプリのJavaScriptを、新しいアプリ名用に生成されたJavaScriptに置き換えます。
4. スタンドアロンアプリに [複数の名前がある場合](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), [設定スクリプトのカスタムアプリID](/docs/browser/new-relic-browser/installation-configuration/copy-browser-monitoring-license-key-app-id) を新しいアプリにコピーします。

[データ保持スケジュール](/docs/accounts-partnerships/accounts/account-billing-usage/data-retention) に基づき、New Relic は旧名称で収集した過去のデータを保持し、新名称でデータを収集・集計します。

## New Relic REST APIによるアプリの名称変更（v2） [#rest-api]

この情報は、APMでも監視されているブラウザアプリにのみ適用されます。ブラウザのJavaScriptスニペットをコピー＆ペーストでアプリに挿入した場合、REST APIでアプリ名を変更することはできません。

標準的な手順に従い、REST API または [API Explorer](https://rpm.newrelic.com/api/explore/applications/update) を介して [APM アプリのエイリアスを変更します。New Relic は、ブラウザを含む New Relic UI のあらゆる場所で、アプリの表示名を自動的に変更します。](/docs/apis/rest-api-v2/application-examples-v2/change-alias-your-application-v2)
