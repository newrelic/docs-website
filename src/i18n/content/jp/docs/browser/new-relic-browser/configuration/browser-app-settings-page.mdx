ブラウザー エージェントがデータを収集する方法をカスタマイズする必要がある場合、UI を構成する方法と、API を使用してエージェント インストルメンテーションをカスタマイズする方法の 2 つがあります。

## UI構成 [#UI]

<DNT>**Application settings**</DNT>ページで、 <InlinePopover type="browser"/>セットアップに対して次の設定を行うことができます。

* [プロキシ エージェント コードと分析データ](/docs/browser/new-relic-browser/configuration/proxy-agent-requests/)
* [アラートの設定](/docs/browser/new-relic-browser/configuration/view-browser-apps-alert-information/)
* [APDEX と地域の設定を構成する](/docs/browser/new-relic-browser/configuration/browser-settings-ui-options-apdex-geography/#apdex-settings)
* [ドメインとサブドメインのフィルタリング](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains/)
* [AJAX リクエストのフィルタリング](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events/)
* [URL パターンによるデータのグループ化](/docs/browser/new-relic-browser/configuration/group-browser-metrics-urls/)
* [データの難読化](/docs/browser/new-relic-browser/configuration/obfuscate-browser-agent-data/)
* [ブラウザアプリの名称変更](/docs/browser/new-relic-browser/configuration/rename-browser-apps/)
* [ブラウザの監視を無効にする](/docs/browser/new-relic-browser/installation/disable-browser-monitoring/)
* [ブラウザ アプリを削除する](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/#remove)

<DNT>**Application settings**</DNT>ページにアクセスするには:

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**
   </DNT>

   に移動してアプリを選択します。

2. 左側のナビゲーションで、

   <DNT>
     **Settings**
   </DNT>

   まで下にスクロールし、

   <DNT>
     **Application settings**
   </DNT>

   をクリックします。

## ブラウザAPI [#apis]

また、ブラウザー エージェント インストルメンテーションをカスタマイズして、カスタム トレースとイベントをキャプチャし、ニーズに合わせて特定のエラーを報告し、最適なパフォーマンスを実現するためにデータ収集のタイミングを微調整することもできます。開始するには、 [「ブラウザー API の使用」](/docs/browser/new-relic-browser/browser-apis/using-browser-apis/)を参照してください。
