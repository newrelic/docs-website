優れた<InlinePopover type="alerts" />は、システムの変更について個人またはチームに通知するのに役立ちます。[Alerts は](/docs/alerts/new-relic-alerts-beta/getting-started/new-relic-alerts)、New Relic 製品全体にわたって単一の調整されたアラート ツールを提供します。これにより、最も関心のある[ エンティティ](/docs/alerts/overview#alert-entity) (ブラウザ アプリやキーなど) のメトリクスに焦点を当てた[ アラート](/docs/alerts/overview#alert-policy)[ ポリシー と](/docs/alerts/overview#alert-condition) 条件 を管理できます。

[1 つ以上のアラート ポリシーを設定し](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/)、[アラート条件を定義すると](/docs/alerts/create-alert/create-alert-condition/alert-conditions)、[製品エンティティ (ターゲット)](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/select-product-targets-alert-condition)のその情報をアラートUIで表示できます。個々の製品のUIでアラート ポリシー情報を表示することもできます。

## ブラウザからのポリシーと条件の表示 [#policy]

特定のブラウザー・アプリやキー・トランザクションのアラート・ポリシーや条件情報を表示する。

1. browserアプリ: <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; Settings &gt; Application settings**</DNT> に移動します。

   また

   キートランザクション: <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Key transactions &gt; (select a key transaction) &gt; Settings &gt; Alert conditions**</DNT>に移動します。

2. <DNT>**Alert conditions**</DNT>ページから、利用可能なツールを使用して、アラート条件とそれに関連付けられたポリシーを検索、並べ替え、表示、または更新します。

## ブラウザからイベントやアクティビティを表示 [#activity]

[イベント](/docs/alerts/overview#event)やその他のアクティビティに関する概要情報を表示するには:

1. browserアプリ: <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT>

   また

   キートランザクション: <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Key transactions**</DNT>に移動します。

2. インデックスから、エンティティの[色分けされた稼働状況](/docs/alerts/overview#health-status)にマウスを置くか、 <DNT>**Browser activity**</DNT>リストからリンクを選択するか、特定のブラウザ アプリまたはキーステップを選択して、追加の詳細を表示します。

たとえば、重大なアラート イベントが発生した場合:

* ブラウザのインデックスと選択されたブラウザアプリでは、健康状態の表示が赤になります。
* 各種チャートの背景色がピンクに変わります。
* ブラウザ インデックスの<DNT>**Browser activity**</DNT>リストには、該当する場合、警告 (黄色) と重大 (赤色) のイベントが表示されます。 アラートの一連イベントの詳細を確認するには、グラフ上のピンクの領域にマウスを置くか選択してください。 詳細と例については、 [「製品からのイベントの表示」を](/docs/alerts/new-relic-alerts-beta/reviewing-events/view-events-their-products)参照してください。

## ブラウザから集中イベントを表示 [#alertevents]

アラート条件に警告 (黄色) または重大 (赤色) の集中イベント用の[閾値](/docs/alerts/overview#alert-threshold)が設定されている場合、製品エンティティの色分けされた[稼働ステータスが](/docs/alerts/overview#health-status)変化して、集中イベントを示します。 ブラウザ アプリまたはブラウザのキーマーケットのページから直接イベントを表示できます。

1. Browserアプリ: <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select a browser app)**</DNT>に移動し、右側のパネルで まとめイベント を確認します。

   キートランザクション: <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Key transactions &gt; (select a key transaction) &gt; Issues &amp; Activity**</DNT>に移動し、 <DNT>**Alert events**</DNT>タブをクリックします。

2. <DNT>**Alert events**</DNT>ページから、利用可能なツールを使用して、選択したブラウザ アプリまたはキーの連続イベントを検索、並べ替え、または表示します。

3. オプション: 集中イベントをトリガーしたアラートポリシー、条件、または閾値を表示または更新するには、対応するリンクを選択します。

<Callout variant="tip">
  アラートUIから[特定の集中イベント、](/docs/alerts/alert-event-management/view-alert-event-details)または[すべての製品にわたる大量イベント](/docs/alerts/alert-event-management/view-alert-event-details)やイベントの警告および重大なアラート イベントを表示することもできます。 アラートは、Critical (赤) のまとめイベントに対してのみ[個別通知を](/docs/alerts/get-notified/destinations)送信します。
</Callout>
