<InlinePopover type="browser" />[セットアップ要件](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements)の一部として、 [Browseragent をインストールする](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent)必要があります。 次に、アプリの<DNT>**Applications settings**</DNT>ページを使用して、次のブラウザ監視オプションを更新できます。

* ブラウザのサブスクリプションレベル
* ブラウザ [Apdex T値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)
* [<DNT>**Geography**</DNT>ページ](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)で強調表示する都市、地域、または国

## Apdex設定を選択 [#apdex-settings]

によって監視されるアプリの場合は、[APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex) （エンドユーザー）と APMに別々の[ Apdex T 値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) を設定できます。browser<DNT>**Geography**</DNT>ページで強調表示する都市、地域、または国を選択することもできます。

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt;**</DNT> (アプリを選択) <DNT>**&gt; Settings &gt; Application settings**</DNT>に移動します。
2. <DNT>**Application settings**</DNT>から、ブラウザ監視専用の Apdex T 値を選択するか、デフォルトの Apdex 値をそのまま使用します。
3. <DNT>**Save application settings**</DNT>を選択します。

Apdex 設定を変更するたびに、 browserアプリの <DNT>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> ページの <DNT>**Page load timing**</DNT> チャートに黒い縦バーが表示されます。

## エンドユーザーのApdexを見る [#view-browser-apdexs]

自分のApdexスコアを見るには

* <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT>に移動します。

  また

* <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DNT>に移動します。
