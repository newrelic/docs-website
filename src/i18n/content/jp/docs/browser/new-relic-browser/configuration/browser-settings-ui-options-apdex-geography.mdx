## ApdexとGeographyの設定を選択 [#apdex-settings]

APM によっても監視されるアプリの場合は、ブラウザー (エンド ユーザー) と[ APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex)[ に対して個別の Apdex T 値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) を設定できます。**Geography** \[地理]ページで強調表示する都市、地域、または国を選択することもできます。

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >** (アプリを選択) **> Settings > Application settings**に移動します。
2. **Application settings** から、ブラウザ監視専用のApdex T値を選択するか、デフォルトのApdex値のままにします。
3. **地理** ページで強調表示するブラウザ モニタリング用に最大 5 つの国または地域を選択します。 [複数のアプリ名](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app)にデータを報告している場合は、メイン アプリとサブ アプリの両方で同じ国を選択します。
4. **Save application settings**を選択します。

Apdexの設定を変更すると、ブラウザアプリの **Page load timing** チャートに黒い縦棒が表示されます。 **[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)** ページ。

## エンドユーザーのApdexを見る [#view-browser-apdexs]

自分のApdexスコアを見るには

* **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (アプリを選択) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**に移動します。

  また

* **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (アプリを選択) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**に移動します。
