<InlinePopover type="browser"/>[セットアップ要件](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements)の一部として、 [Browseragent をインストールする](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent)必要があります。 次に、アプリの<DNT>**Applications settings**</DNT>ページを使用して、次のブラウザ監視オプションを更新できます。

* ブラウザのサブスクリプションレベル

* ブラウザ [Apdex T値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)

* <DNT>
    [**Geography**](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)
  </DNT>

  [ページ](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)で強調表示する都市、地域、または国

## ApdexとGeographyの設定を選択 [#apdex-settings]

によって監視されるアプリの場合は、[APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex) （エンドユーザー）と APMに別々の[ Apdex T 値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) を設定できます。browser<DNT>**Geography**</DNT>ページで強調表示する都市、地域、または国を選択することもできます。

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >**
   </DNT>

   (アプリを選択)

   <DNT>
     **> Settings > Application settings**
   </DNT>

   に移動します。

2. <DNT>
     **Application settings**
   </DNT>

   から、ブラウザ監視専用の Apdex T 値を選択するか、デフォルトの Apdex 値をそのまま使用します。

3. <DNT>
     **Geography**
   </DNT>

   ページで強調表示するブラウザ監視の国または地域を最大 5 つ選択します。 [複数のアプリ名](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app)にデータを報告する場合は、プライマリ アプリとセカンダリ アプリの両方で同じ国を選択します。

4. <DNT>
     **Save application settings**
   </DNT>

   を選択します。

Apdex 設定を変更するたびに、 browserアプリの <DNT>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> ページの <DNT>**Page load timing**</DNT> チャートに黒い縦バーが表示されます。

## エンドユーザーのApdexを見る [#view-browser-apdexs]

自分のApdexスコアを見るには

* <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT>に移動します。

  また

* <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DNT>に移動します。
