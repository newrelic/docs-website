ブラウザ [セットアップ要件](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements) の一環として、 [ブラウザエージェント](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent) をインストールする必要があります。その後、アプリの **Applications settings** ページを使用して、以下のブラウザ監視オプションを更新することができます。

* ブラウザのサブスクリプションレベル
* ブラウザ [Apdex T値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)
* [**地理** ページ](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)で強調表示される都市、地域、または国

## ApdexとGeographyの設定を選択 [#apdex-settings]

[](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)APM によっても監視されるアプリの場合、ブラウザー (エンド ユーザー) と [APM](/docs/apm/new-relic-apm/apdex/changing-your-apdex-settings)に対して別々のApdex T 値を設定できます。 **地理** ページで強調表示する都市、地域、または国を選択することもできます。

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser >** (アプリを選択) **> Settings > Application settings**に移動します。
2. **Application settings** から、ブラウザ監視専用のApdex T値を選択するか、デフォルトのApdex値のままにします。
3. **地理** ページで強調表示するブラウザ モニタリング用に最大 5 つの国または地域を選択します。 [複数のアプリ名](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app)にデータを報告している場合は、メイン アプリとサブ アプリの両方で同じ国を選択します。
4. **Save application settings**を選択します。

Apdexの設定を変更すると、ブラウザアプリの **Page load timing** チャートに黒い縦棒が表示されます。 **[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)** ページ。

## エンドユーザーのApdexを見る [#view-browser-apdexs]

自分のApdexスコアを見るには

* **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser > (アプリを選択) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**に移動します。

  また

* **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser > (アプリを選択) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**に移動します。
