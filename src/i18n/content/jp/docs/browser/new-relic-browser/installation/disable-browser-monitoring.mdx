browserを使用してアプリケーションを監視している場合は、アプリまたは特定のページの監視を手動で無効にすることができます。 特定のページで<InlinePopover type="browser"/>オフにする理由は次のとおりです。

* 一部のページは、モニターには関係ない場合があります。
* 特定のページだけを監視したい。
* トラブルシューティングのために、ブラウザのスクリプトを削除したい。

## 特定のページ/URLの監視を無効にする [#disable]

特定のページやURLのブラウザを無効にするには、導入方法に応じた手順に従ってください。

<CollapserGroup>
  <Collapser
    id="agent"
    title="選択されたAPMエージェントで展開される"
  >
    <InlinePopover type="apm"/>エージェントを使用して JavaScript スニペットを自動的に挿入した場合は、 APM言語エージェントのAPIまたは構成ファイルを使用して、特定のページに対してのみインジェクションを無効にすることができます。 詳細については、エージェントの指示を参照してください。

    * Go（現在は該当なし）
    * [Java](/docs/agents/java-agent/instrumentation/page-load-timing-java#disable-auto-specific)
    * [.NET](/docs/agents/net-agent/features/page-load-timing-net#disable_instrumentation_pageview)
    * [Node.js](/docs/agents/nodejs-agent/supported-features/page-load-timing-nodejs#disabling): このエージェントは JavaScript を自動的に挿入しません。特定のページのモニタリングをオフにするには、そのページからAPIコールを削除するだけです。
    * [PHP](/docs/agents/php-agent/features/new-relic-browser-php-agent#disable)
    * [Python](/docs/agents/python-agent/supported-features/page-load-timing-python#disabling_instrumentation)
    * [Ruby](/docs/agents/ruby-agent/installation-configuration/ignoring-specific-transactions#page-load-timing-rum)
  </Collapser>

  <Collapser

  </Collapser>

## ドメイン監視の無効化 [#domains]

特定のドメインやサブドメインの監視をオフにするには、ブラウザを更新してください [ドメインの条件](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains).

## ブラウザエージェントの無効化 [#disable-agent]

[コピー／ペーストによる導入方法](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent#copy-paste-app) を使用し、ブラウザアプリのデータ収集を停止したい場合は、監視したくないページからブラウザのJavaScriptスニペットを削除するだけです。 [APMエージェント](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent#select-apm-app) を使用してブラウザエージェントをインストールした場合、ブラウザ監視機能の一部を個別にオフにしたり、ブラウザを完全に無効にしたりすることができます。

1. <DNT>
     **[one.newrelic.com](https://one.newrelic.com/all-capabilities), click on Browser > (select a browser app) > Settings > Application settings**
   </DNT>

   に移動します。

2. [Pro アカウント レベルの機能](/docs/browser/new-relic-browser/getting-started/new-relic-browser)のみを無効にするには、

   <DNT>
     **Lite**
   </DNT>

   を選択します。

3. ブラウザ監視を完全にオフにするには、

   <DNT>
     **Off**
   </DNT>

   を選択します。

4. <DNT>
     **Save application settings**
   </DNT>

   を選択します。

5. 該当する場合は、

   <InlinePopover type="apm"/>

   エージェントを再起動してください。

ブラウザモニタリングがオフの場合、エージェントはモニタリングスクリプトでページを処理しません。
