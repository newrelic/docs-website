## 問題

<DNT>**Web vitals**</DNT>にデータが表示されません。

## 解決

データが欠落するいくつかの原因を次に示します。

* フィルタが狭すぎます。一部の重要な属性は、特定の Web バイタルにのみ関連しています。たとえば、 `elementID`でフィルタリングした場合、 `elementID`は LCP の測定にのみ適用されるため、データは最大コンテンツ ペイント (LCP) に対してのみ入力されます。
* ブラウザーのエージェントまたはエディションは特定のデータを収集しません。New Relic ブラウザ エージェントがどのような主要な属性を収集するかについては、[ブラウザ エージェント データ ディクショナリを](/attribute-dictionary/?dataSource=Browser+agent)参照してください。
* アプリは`PageViewTiming`データを New Relic に送信できません。Web Vitals は`PageViewTiming`データに基づいて構築されているため、何らかの理由でアプリがインジェストをコレクターに送信できない場合、データが表示されないことがあります。
