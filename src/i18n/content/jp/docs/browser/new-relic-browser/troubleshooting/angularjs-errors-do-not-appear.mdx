## 問題

AngularJS を使用しており、JavaScript エラーを New Relic に報告したいのですが、 [<DNT>**JavaScript errors**</DNT>ページ](/docs/browser/new-relic-browser/browser-pro-features/js-errors-dashboard-examining-errors-over-time)にアプリケーションのエラーが表示されません。

## 解決

AngularJS アプリケーションには、すべてのエラーをキャッチするグローバル エラー ハンドラーが含まれていることが多く、デフォルトでは、 <InlinePopover type="browser"/>キャッチされていないエラーのみを報告します。

[noticeError](/docs/browser/new-relic-browser/browser-agent-spa-api/newrelicnoticeerror-browser-agent-api) API コールを使用して、エラーを New Relic に報告します。これにより、通常のエラー処理の一環としてエラーを処理するだけでなく、New Relic の UI で確認することができます。
