## 問題

**Errors Inbox** \[エラー受信トレイ] ページにいくつかのエラーが表示されません。

## 解決

ブラウザ エージェントは、ページに表示される **未処理の** エラーを自動的に収集します。ページまたはアプリケーション内でエラーが発生していることがわかっているのに、**Errors Inbox** \[エラー受信箱] ページにエラーが表示されない場合は、エラーがページまたはアプリケーションによって処理されており、ブラウザー エージェントに反映されていないことを確認することが重要です。このようなシナリオでも、 [`noticeError` API](/docs/browser/new-relic-browser/browser-apis/noticeerror)を利用すると、処理されたエラーを観察できます。
