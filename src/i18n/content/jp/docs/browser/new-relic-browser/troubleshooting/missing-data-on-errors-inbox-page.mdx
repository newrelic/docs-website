## 問題

<DNT>**Errors Inbox**</DNT>ページにエラーが表示されません。

## 解決

Browseragent は、ページに表示された<DNT>**unhandled**</DNT>エラーを自動的に収集します。 ページまたはアプリケーション内でエラーが発生していることがわかっているのに、 <DNT>**Errors Inbox**</DNT>ページに表示されない場合は、エラーがページまたはアプリケーションによって処理され、Browser Agent にバブルアップされていないことを確認することが重要です。 このようなシナリオでは、 [`noticeError` API](/docs/browser/new-relic-browser/browser-apis/noticeerror)を利用することで、処理されたエラーを監視できます。
