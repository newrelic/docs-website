## 構文

```js
newrelic.consent(accept?: boolean)
```

ブラウザエージェントがネットワーク経由でデータを送信することへの同意を許可または拒否します。

## 要件

* Browser Lite、Pro、または Pro+SPA エージェント (v1.303.0 以降)

## 説明

ブラウザエージェントには、オプションの同意モード機能があります。 アプリ設定または NerdGraph で同意モードが有効になっている場合、ユーザーは`newrelic.consent(...)`メソッドを呼び出して、ネットワーク経由でデータを送信するかどうかの同意を許可または拒否できます。

設定と使用方法の詳細については、[同意モードに関するドキュメント](/docs/browser/new-relic-browser/configuration/consent-mode)をご覧ください。

## 使用例

* ユーザーがサイト上のモーダルに同意するまでデータの収集を待ちたい場合

## 例

### ユーザーはブラウザエージェントがデータを収集することに同意します。

```js
newrelic.consent()
```

### ユーザーがブラウザエージェントによるデータ収集の同意を拒否した

```js
newrelic.consent(false)
```
