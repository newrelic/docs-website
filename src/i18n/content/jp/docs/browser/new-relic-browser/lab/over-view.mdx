あなたは、ユーザーがお気に入りのレストランに食べ物を注文できる [React](https://reactjs.org//) アプリである Relicstaurants の開発者です。あなたのアプリはユーザーの間で非常に人気があり、ユーザーがサイトで常に最高のエクスペリエンスを得られるようにしたいと考えています。この目標を達成するには、New Relic ブラウザー監視を使用して、サイトのパフォーマンスを向上させ、必要に応じて問題をトラブルシューティングします。

このラボでは、New Relic ブラウザー モニタリングを使用して Web アプリケーションを計測し、そのメトリックを監視し、フロントエンドの問題をトラブルシューティングします。

<SideBySide>
  <Side>
    ## 目的

    * React デモ アプリケーションを起動します。
    * New Relic ブラウザ エージェントをインストールします。
    * フロントエンド エラーのトラブルシューティング。
    * ソースマップを New Relic にアップロードします。
    * フロントエンドの遅さをトラブルシューティングします。
  </Side>

  <Side>
    ## 要件

    * 無料の [New Relic アカウント](https://newrelic.com/signup?utm_source=developer-site)
    * [ノード](https://nodejs.org/en/)
    * [Python3](https://www.python.org/)
    * [ジット](https://git-scm.com/)
    * [pip3](https://pip.pypa.io/en/stable/installation/)
  </Side>
</SideBySide>
