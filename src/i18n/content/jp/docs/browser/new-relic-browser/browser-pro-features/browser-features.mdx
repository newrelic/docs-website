当社の主な<InlinePopover type="browser"/>機能をご覧ください:

* <DoNotTranslate>[Summary](/docs/browser/browser-monitoring/getting-started/browser-summary-page/)</DoNotTranslate>

  : browserのパフォーマンスの概要を表示します

* <DoNotTranslate>[Web vitals](/docs/journey-performance/guide-to-monitoring-core-web-vitals)</DoNotTranslate>

  : コアウェブバイタルに貢献するものを知る

* <DoNotTranslate>[Session replay](/docs/browser/browser-monitoring/browser-pro-features/session-replay)</DoNotTranslate>

  : ユーザーセッションのライブ再生を表示して、JavaScript エラーのトラブルシューティングを行い、UI デザインを改善します。

* <DoNotTranslate>[Page views](/docs/browser/new-relic-browser/browser-pro-features/page-views-examine-page-performance/)</DoNotTranslate>

  : ページのパフォーマンスを調べる

* <DoNotTranslate>[AJAX](/docs/browser/browser-monitoring/browser-pro-features/ajax-page-identify-time-consuming-calls/)</DoNotTranslate>

  : 時間のかかる通話を特定する

* <DoNotTranslate>[JavaScript errors](/docs/browser/new-relic-browser/browser-pro-features/javascript-errors-page-detect-analyze-errors/)</DoNotTranslate>

  : エラーを検出して分析する

* <DoNotTranslate>[Error profiles](/docs/browser/new-relic-browser/browser-pro-features/browser-error-profiles-find-error-causes/)</DoNotTranslate>

  : エラーの原因を見つける

* <DoNotTranslate>[Session traces](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle/)</DoNotTranslate>

  : ウェブページのライフサイクルを調べる

* <DoNotTranslate>[Geography](/docs/browser/new-relic-browser/browser-pro-features/geography-webpage-metrics-location/)</DoNotTranslate>

  : ユーザーの所在地別にウェブページデータを表示する

* <DoNotTranslate>[Browsers](/docs/browser/new-relic-browser/browser-pro-features/browsers-problem-patterns-type-or-platform/)</DoNotTranslate>

  : browser種類またはプラットフォーム別にウェブページデータを表示

* <DoNotTranslate>[Distributed tracing](/docs/browser/new-relic-browser/browser-pro-features/browser-data-distributed-tracing/)</DoNotTranslate>

  : バックエンドとフロントエンドのパフォーマンスを接続

* <DoNotTranslate>[Browser apps index](/docs/browser/new-relic-browser/getting-started/browser-apps-index/)</DoNotTranslate>

  : すべてのbrowserアプリのリストを表示します
