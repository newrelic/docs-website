ブラウザ アプリのパフォーマンスのログを表示するには:

<Steps>
  <Step>
    <DNT>[one.newrelic.com](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Browser**</DNT>に移動します。
  </Step>

  <Step>
    ブラウザ アプリを選択します。
  </Step>

  <Step>
    左側のメニューで、 **Logs** \[ログ]に移動します。
  </Step>
</Steps>

<img title="Browser logs" alt="Screenshot of browser logs page" src="/images/browser_logs_ui.webp" />

## 次は何ですか？

* また、セッション リプレイ (セッション リプレイ機能を使用している場合) を使用してブラウザー ログを表示したり、 Errors Inboxで JavaScript エラーと並行してブラウザ ログを表示したりすることもできます。 詳細については、セッション[リプレイのログ](#session-replay-section)と[エラー受信箱のログ](#js-errors-section)を参照してください。

* UIの詳細については、 [UIの使用」を](/docs/logs/ui-data/use-logs-ui)参照してください。
