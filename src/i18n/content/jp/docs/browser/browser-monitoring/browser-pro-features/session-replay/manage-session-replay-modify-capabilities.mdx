ユーザー管理アクセス機能を使用して、セッションリプレイと変更機能を管理します。 この機能を使用すると、ユーザーロールとアクセス権限をカスタマイズして、セッションリプレイ設定を変更できるユーザーを定義できます。

## 管理者グループからメンバー/アカウントの SR 変更アクセスを取り消す [#modify-all-admin-role]

<Steps>
  <Step>
    <DNT>**Administration &gt; Access Management &gt; Groups**</DNT>に移動します。
  </Step>

  <Step>
    省略記号**(...)**アイコンをクリックし、 <DNT>**Manage Group Access**</DNT>選択します。
  </Step>

  <Step>
    <DNT>**Members**</DNT>セクションを展開し、必要なユーザーを選択します。
  </Step>

  <Step>
    <DNT>**Remove users from group**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Account access**</DNT>セクションを展開し、省略記号**(...)**アイコンをクリックします。
  </Step>

  <Step>
    <DNT>**Delete role**</DNT>を選択します。
  </Step>
</Steps>

## セッションリプレイ読み取りアクセス用のカスタム ロールを作成する [#create-custom-role]

<Steps>
  <Step>
    <DNT>**Administration &gt; Access Management &gt; Roles**</DNT>に移動します。
  </Step>

  <Step>
    <DNT>**Add New Custom Roles**</DNT>をクリックして、カスタム ロール名を入力します。
  </Step>

  <Step>
    <DNT>**Browser**</DNT>セクションでは、次の操作を実行できます。

    * <DNT>**Session replays**</DNT>に読み取り専用アクセスを許可するには、 <DNT>**View**</DNT>チェックボックスをオンにし、 <DNT>**Modify**</DNT>チェックボックスが選択されていないことを確認します。
    * <DNT>**Session replays**</DNT>に変更アクセス権を付与するには、 <DNT>**View**</DNT>と<DNT>**Modify**</DNT>の両方のチェックボックスを選択します。

    <img title="Browser Role ID" alt="A screenshot of Browser Role ID" src="/images/browser-role-id.webp" />
  </Step>

  <Step>
    <DNT>**Save**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Groups**</DNT>タブを選択し、 <DNT>**Create new group**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Group name**</DNT>を入力して<DNT>**Create group**</DNT>をクリックします。
  </Step>

  <Step>
    <DNT>**Members**</DNT>セクションを展開し、**Select a user** \[ユーザーの選択] ドロップダウン リストからユーザーを選択し、**Add user to group** \[ユーザーをグループに追加] をクリックします。
  </Step>

  <Step>
    <DNT>**Account acess**</DNT>セクションを展開し、 <DNT>**Add account roles to this group**</DNT>フィールドで以下を選択します。

    * <DNT>**Select a role**</DNT>
    * <DNT>**Select an account**</DNT>
  </Step>

  <Step>
    <DNT>**Add role**</DNT>をクリックします。
  </Step>
</Steps>

## セッションリプレイ変更機能を付与する [#grant-sr-modify-capability]

<Steps>
  <Step>
    <DNT>**Administration &gt; Access Management &gt; Roles**</DNT>に移動します。
  </Step>

  <Step>
    <DNT>**Custom role**</DNT>セクションで、省略記号**(...)**アイコンをクリックし、 <DNT>**Manage role**</DNT>を選択します。
  </Step>

  <Step>
    <DNT>**Edit**</DNT>アイコンをクリックして、 <DNT>**Browser**</DNT>セクションに移動します。
  </Step>

  <Step>
    <DNT>**Session replays**</DNT>セクションで<DNT>**Modify**</DNT>チェックボックスを選択します。
  </Step>

  <Step>
    <DNT>**Save**</DNT>をクリックします。
  </Step>
</Steps>

<Callout variant="important">
  * **Session Replay** \[セッションリプレイ]設定には、 [すべての製品管理者](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#standard-roles)ロールと**SR 変更**ユーザーのみがアクセスできます。 既存の読み取り権限を持つユーザーは設定を表示できません\*。
  * アップグレードされた設定がアカウントとユーザーの UI に表示されるまで 5 ～ 10 分かかる場合があります。
</Callout>
