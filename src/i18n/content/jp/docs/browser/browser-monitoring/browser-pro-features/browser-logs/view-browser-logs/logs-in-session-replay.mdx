セッションリプレイ機能を使用すると、セッションリプレイのコンテキストでブラウザのログを表示できます。 この統合により、アプリケーションの機能とユーザー エクスペリエンスの包括的なビューが提供され、貴重なインサイトがユーザー インタラクションに提供されます。
