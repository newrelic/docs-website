Browserのシングルページアプリケーション(SPA)モニタリングには、特定のブラウザインタラクションのカスタムモニタリングを追加するためのAPIが含まれています。これは、動的に更新されるウィジェットなど、ルートの変更を伴わないためにNew Relicが自動的に記録しないインタラクションをモニタリングするのに便利です。また、SPA API では、監視するほど重要ではないと思われるインタラクションのデフォルト監視をオフにすることもできます。

具体的なAPIコールを含むSPA APIの詳細については、 [Browser agent and SPA API](/docs/browser/new-relic-browser/guides/guide-using-browser-spa-apis/) を参照してください。
