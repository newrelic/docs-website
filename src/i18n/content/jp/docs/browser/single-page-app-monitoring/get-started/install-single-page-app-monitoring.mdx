シングル ページ アプリ（SPA）監視には[、 <InlinePopover type="browser" />エージェントがデフォルトでインストールされ](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent/)ています。

## 要件 [#requirements]

[ここで、SPA モニタリングの互換性と要件](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring#spa/)を確認できます。

New Relic アカウントで最初のモニターアプリを設定する際には、利用規約に同意する必要があります。利用規約に同意することで、New Relic が URL からハッシュフラグメントを収集することを許可します。チェックボックスのオプションを選択するのは、1つのアカウントに対して1回だけです。

## SPAモニタリングの有効化・無効化 [#enable-spa]

[ブラウザ監視を有効にする](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#enable)と、ディストリビューティッド（分散）トレーシングを含むさまざまな最新機能にアクセスできるため、SPA監視がデフォルトで含まれます。 一部の古いエージェント インストレーションはアップグレードする必要がある場合があります。 [ブラウザエージェントのタイプ](/docs/browser/new-relic-browser/configuration/change-browser-agent-type#agent-types)について詳しくは、こちらをご覧ください。
