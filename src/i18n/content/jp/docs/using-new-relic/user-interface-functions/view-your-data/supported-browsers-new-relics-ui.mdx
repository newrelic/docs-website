New Relic の UI は、最も一般的なブラウザの最新のデスクトップ版 2 つをサポートしています。サポートされていないバージョンのブラウザを使用している場合は、UIにエラーが表示されます。

<Callout variant="tip">
  当社のブラウザー・モニタリングの要件については、 [ブラウザー・モニタリングの互換性](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-new-relic-browser) を参照してください。
</Callout>

## 対応ブラウザ [#requirements]

当社のUIは、これらのブラウザの最新のデスクトップバージョン2つに対応しています。

* Google Chromeです。Linux、macOS、Windows

* Microsoft Edge: Windows

* モジラファイヤーフォックスLinux、macOS、Windows

  * FirefoxのExtended Support Release（ESR）バージョンはサポートされていません。

* Safari: macOS

お使いのブラウザのバージョンがサポートされていない場合でも、動作する可能性があります。しかし、当サイト（およびその他のサイト）の機能を最大限に活用するためには、サポートされているバージョンにアップグレードすることをお勧めします。

## 限られたサポート

New Relic の UI は、モバイルブラウザに対して限られたサポートしか提供していません。モバイルデバイスからNew Relicにアクセスするには、当社の [Androidアプリ](/docs/mobile-apps/new-relic-android-app/features/new-relic-android-app) または [iOSアプリ](/docs/mobile-apps/new-relic-ios-app/features/new-relic-ios-app) をインストールしてください。モバイルアプリのワークフローは、電話での使用例を想定して設計されています。ウェブインターフェイスは、モバイルブラウザでは積極的にサポートされていません。
