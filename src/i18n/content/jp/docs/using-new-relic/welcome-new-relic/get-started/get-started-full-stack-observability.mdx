真のオブザーバビリティにより、デジタルシステム全体で起こっていることやその理由を、どのようなソリューションを使用していてもいつでも知ることができます。全体像を把握し、それにより、クラウドでマイクロサービスを実行しているコンテナやモバイルウェブサイトのショッピングカートのボタンなど、アプリケーションとデバイスで価値を生むことが可能になります。

当社のプラットフォームは、オブザーバビリティを提供することで、単なるモニタリングにとどまらず、お客様の環境をプロアクティブかつ予測的に改善するためのデータの洞察を提供します。

## クイックスタートを使用してNew Relicにデータを取り込む [#integrations]

[New Relic I/O](https://newrelic.com/instant-observability/)はクイックスタートの豊富なオープンソースカタログで、インテグレーション、ダッシュボード、アラート設定を即座に使用できます。Node.js、Python、Rubyなどのよく知られたテクノロジーにはフル機能のクイックスタートが搭載されていますが、他のテクノロジーではツールが混在しています。各クイックスタートは、世界中のオブザーバビリティのエキスパートが作成し、New Relicにより検証されており、ワンクリックでインストールできます。New Relic I/Oを即時オブザーバビリティのハブとして利用することで、コミュニティの専門知識を活用し、テレメトリデータのさらなる価値を引き出すことができます。

## すべての回答を1か所に [#answers]

データが[New Relic](/docs/using-new-relic/welcome-new-relic/get-started/introduction-new-relic)に入ると、当社のツールを備えたUIにより、システムを取り巻く何層もの複雑さに対処できます。これはすべて1つのプラットフォーム上にあるため、診断アプリケーションを切り替える必要はありません。

パターンの条件を指定してデータを検索し、[データプラットフォーム](/docs/data-ingest-apis/get-data-new-relic/getting-started/get-started-telemetry-data-platform)を使用してデータを発見し、または[機械学習ツール](/docs/new-relic-one/use-new-relic-one/new-relic-ai/introduction-new-relic-ai)から積極的データを取得できます。

すべてのオブザーバビリティツールは相互につながっており、[New Relic One](/docs/new-relic-one/use-new-relic-one/get-started/introduction-new-relic-one)でアクセスできます。

New Relicに報告されたデータは、[メトリクス、イベント、ログ、トレース](https://newrelic.com/platform/telemetry-data-101/)として分類できます。このデータは、当社のプラットフォームの分析および監視機能に利用されます。New Relicはデータを意味のある方法で関連付けるため、[データを詳しく調べ](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer)、[ダッシュボードを構築し](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards)、[アラートを設定](/docs/alerts)できます。

すぐに使用できるオブザーバビリティUI体験により、ソフトウェアスタック全体を一元化されたプラットフォームで容易に可視化、解析およびトラブルシュートできます。[New Relicエクスプローラー](/docs/new-relic-one/use-new-relic-one/core-concepts/entity-explorer-view-performance-across-apps-services-hosts)は、パフォーマンスの傾向と問題を簡単に検知できるように、システムにあるすべてのエンティティと接続方法を1か所に統合します。インフラストラクチャの状態をアプリケーションのパフォーマンスやエンドユーザーの行動に自動的に接続することで、ノイズをカットして有効な信号を見つけることができます。
