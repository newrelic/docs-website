## JavaScriptエラーを表示する [#javascriptError-example]

[JavaScript エラー](/docs/browser/new-relic-browser/browser-pro-features/javascript-errors-page-examine-errors-over-time) を表示するには、 [NRQL クエリ](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) を実行します。 [`JavaScriptError` イベント](/docs/insights/insights-data-sources/default-data/browser-default-events-attributes-insights#javascriptError-attributes) Browser によって報告されたものです。例えば、以下のようになります。

```
SELECT * FROM JavaScriptError
```

## SPAのサンプルデータを見る [#explore-spa-data]

Browser [SPA データ](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring) の [NRQL クエリ](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) を実行するには、 `BrowserInteraction` または `AjaxRequest` イベントを使用します。例えば、以下のようになります。

```
SELECT * FROM BrowserInteraction
```

ここでは、SPA のデータに対して行うことができる NRQL クエリの例をご紹介します。すべてのブラウザの属性を見るには、 [ブラウザのデフォルト属性](/docs/insights/insights-data-sources/default-data/browser-default-events-attributes-insights) を参照してください。

<CollapserGroup>
  <Collapser
    id="count-route-changes"
    title="ルート変更やページロードのカウント"
  >
    過去1日に行われた経路変更やページロードをカウントすること。

    ```
    SELECT count(*) FROM BrowserInteraction SINCE 1 day ago
    ```
  </Collapser>

  <Collapser
    id="avg-duration-route-change"
    title="ルート変更とページロードの平均時間"
  >
    過去1日の経路変更とページロードの平均時間を調べる。

    ```
    SELECT average(duration) FROM BrowserInteraction SINCE 1 day ago
    ```
  </Collapser>

  <Collapser
    id="slowest-route-changes"
    title="最も遅いルート変更"
  >
    過去1日の中で最も遅い経路変更の10件を見つけること。

    ```
    SELECT average(duration) FROM BrowserInteraction WHERE category = 'Route change' SINCE 1 day ago FACET browserInteractionName
    ```
  </Collapser>

  <Collapser
    id="most-popular-route-changes"
    title="最も人気のあるルート変更"
  >
    過去1日の間に行われたルート変更のうち、最も人気のあった10件を見つけること。

    ```
    SELECT count(*) FROM BrowserInteraction WHERE category = 'Route change' since 1 day ago FACET browserInteractionName
    ```
  </Collapser>

  <Collapser
    id="compare-throughput"
    title="ページロードとルート変更のスループットを比較"
  >
    直近1日のページロードと経路変更の間のスループットを1時間単位で比較する。

    ```
    SELECT count(*) FROM BrowserInteraction SINCE 1 day ago FACET category TIMESERIES 1 hour
    ```
  </Collapser>

  <Collapser
    id="compare-performance"
    title="ページロードとルート変更のパフォーマンスを比較"
  >
    直近1日のページロードと経路変更のパフォーマンスを1時間単位で比較する。

    ```
    SELECT average(duration) FROM BrowserInteraction SINCE 1 day ago FACET category TIMESERIES 1 hour
    ```
  </Collapser>

  <Collapser
    id="create-table"
    title="特定の条件でテーブルを作成"
  >
    最も遅い経路変更20件のAjaxカウントの平均値、最小値、最大値のデータテーブルを作成する。

    ```
    SELECT average(duration), average(ajaxCount), min(ajaxCount), max(ajaxCount) FROM BrowserInteraction WHERE category = 'Route change' SINCE 1 day ago FACET browserInteractionName LIMIT 20
    ```
  </Collapser>
</CollapserGroup>
