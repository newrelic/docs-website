チャートを作成した後は、データを最適に表示するために外観をカスタマイズすることができます。チャートをさまざまな形式で共有したり、新規または既存のダッシュボードにチャートを追加することができます。

## チャートの外観を変える [#change-appearance]

NRQL モードでクエリを実行するか、メトリックとイベントを使用してデータを指定しながらチャートを表示すると、New Relic はデータを分析し、データに適合する[チャート タイプ](/docs/insights/use-insights-ui/manage-dashboards/insights-chart-types)を適用します。

選択できるチャートの種類にはいくつかのオプションがあります。 グラフの種類を変更するには、現在のグラフの右側にある<DNT>**Chart type**</DNT>メニューを使用します。

## チャートのカスタマイズ [#customize-charts]

データの表示方法を最適化するために最善を尽くしていますが、他のニーズがある場合もあります。グラフの種類によっては、追加のカスタマイズ オプションを利用できます。

<Callout variant="important">
  これらのグラフでバイト単位 (KB、MB、GB、TB など) を表すために使用されるシステムは、メトリックス表現に従います。 この意味は：

  * 1 KB（キロバイト） = 1,000 バイト
  * 1 MB (メガバイト) = 1,000 KB
  * 1 GB (ギガバイト) = 1,000 MB
  * 1 TB（テラバイト） = 1,000 GB

  このメトリクス システムは、計算と表示に 10 の累乗を使用する国際単位系 (SI) に準拠しています。
</Callout>

カスタマイズ機能の詳細については、以下のセクションをお読みください。[このブログ](https://newrelic.com/blog/how-to-relic/dashboard-chart-customizations)と[コンパニオン ビデオを](https://www.youtube.com/watch?v=LA2jM_asl94)参照して、いくつかのカスタマイズ オプションのガイド付きウォークスルーを取得することもできます。

**チャートのカスタマイズオプションにアクセスするには**：

1. カスタマイズしたいウィジェットで、 <Icon name="fe-more-horizontal" />メニューで、 <DNT>**Edit**</DNT>を選択します。
2. <DNT>**Customize this visualization**</DNT>パネルで、以下のセクションで説明されているオプションを設定してください。

<CollapserGroup>
  <Collapser id="widget-link-description" title="ウィジェットのリンクと説明">
    ダッシュボードウィジェットにリンクや説明を追加することで、追加のコンテキスト情報やナビゲーションオプションを提供できます。

    **ウィジェット リンクは**ウィジェット タイトル上にクリック可能なリンクとして表示され、ユーザーが関連するダッシュボード、ドキュメント、または外部リソースに移動できるようになります。 ユーザーがダッシュボードを表示すると、ウィジェットのタイトルをクリックすることで、設定済みのリンクが開きます。

    **ウィジェットの説明は、**ユーザーがウィジェットをポイントしたときにツールチップとして表示される、ウィジェットが表示する内容に関する追加のコンテキストを提供します。

    **リンクまたは説明を追加するには**：

    1. リンクを追加するには、 <DNT>**URL**</DNT>フィールドに URL を入力してください。以下の要件にご注意ください。

       * 最大文字数：2,048文字
       * サポートされているプロトコル： `http://`と`https://`のみ
       * 無効なURLには検証エラーが表示されます。

    2. 説明を追加するには、 <DNT>**Description**</DNT>フィールドにプレーンテキストの説明を入力してください。

    3. <DNT>**Apply changes**</DNT>をクリックします。

    **リンクまたは説明を削除するには**：

    1. <DNT>**URL**</DNT>または<DNT>**Description**</DNT>フィールドをクリアしてください。
    2. <DNT>**Apply changes**</DNT>をクリックします。

    **JSONを使用してウィジェットのリンクと説明を追加、編集、または削除するには**：

    1. ダッシュボードから<Icon name="fe-more-horizontal" />メニューで、 <DNT>**Manage JSON**</DNT>を選択します。

    2. JSONエディタで、変更したいウィジェットを見つけて、 `link`フィールドと`description`フィールドを更新します。追加または編集するには、値を入力してください。削除するには、フィールドを`null`に設定してください。

       ```json
       widgets: [{
         id: "WIDGET_ID"
         link: "https://example.com"  // Enter URL or null to remove
         description: "Your widget description"  // Enter text or null to remove
       }]
       ```

    3. <DNT>**Save changes**</DNT>をクリックします。

    ウィジェット リンクが期待どおりに機能しないという問題が発生した場合は、トラブルシューティング ヘルプの[「ウィジェット リンクの問題」を](/docs/query-your-data/explore-query-data/dashboards/troubleshooting-chart-errors/#widget-link-issues)参照してください。
  </Collapser>

  <Collapser id="custom-date-time" title="日付と時刻のフォーマット">
    <DNT>**Data format**</DNT>セクションから、テーブルと看板の日付と時刻の形式をカスタマイズできます。そのままにしておくことも、必要に応じてフォーマットを変更することも可能です。以下のオプションからお選びください。

    * <DNT>**Numeric**</DNT>: タイムスタンプを数値として保持します。
    * <DNT>**Date**</DNT>: 日付を標準形式で表示します。
    * <DNT>**Duration**</DNT>: タイムスタンプと現在の瞬間の間の時間範囲を表示します。
    * <DNT>**Time to Now**</DNT>: タイムスタンプからの経過時間を示します。

    ニーズに基づいて、データ表示を設定するのに最適な形式を選択してください。
  </Collapser>

  <Collapser id="customize-y-axis" title="Y軸のカスタマイズ">
    折れ線グラフや面グラフで特定の値の範囲内のデータを表示するには、 <DNT>**Y-axis**</DNT>セクションで軸の最小値と最大値を設定します。

    カスタマイズオプションが選択されていない場合、 [ダッシュボード](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) は、0 から上の値までの完全な Y 軸にマージンを加えて自動的に表示します。

    折れ線グラフ、面グラフ、ヒストグラムの凡例を無効または有効にするには、 <DNT>**Legend**</DNT>セクションで<DNT>**Visibility**</DNT>トグルを使用します。
  </Collapser>

  <Collapser id="bar-charts-pie-charts-tables" title="棒グラフ、円グラフ、表">
    バーチャート、円グラフ、または表の場合、1 つの[ファセット](/docs/query-your-data/explore-query-data/dashboards/filter-new-relic-one-dashboards-facets)の数が 5,000 より大きい場合、 <DNT>**Other groups**</DNT>ファセットは残りのファセットを集約します。 <DNT>**Other groups**</DNT>セクションで<DNT>**Other groups**</DNT>オプションを表示または非表示にするには、 <DNT>**Visible**</DNT>トグルを使用します。
  </Collapser>

  <Collapser id="dash-color" title="色でカスタマイズ">
    面グラフ、棒グラフ、折れ線グラフ、円グラフ、積み上げ棒グラフのシリーズごとに、チャートの色をカスタマイズできます。

    シリーズごとに色をカスタマイズするには、 <DNT>**Colors**</DNT>セクションで、 <Icon name="fe-plus" />アイコンをクリックし、変更したいシリーズを選択して、希望の色を選択します。
  </Collapser>

  <Collapser id="dash-units" title="単位のカスタマイズ">
    Y 軸と各シリーズの単位をカスタマイズできます。面グラフ、折れ線グラフ、積み上げ棒グラフで利用できます。

    ユニットをカスタマイズするには、 <DNT>**Units**</DNT>セクションで、 <Icon name="fe-plus" />アイコンをクリックし、別のユニットが必要なシリーズを選択して、ユニットを選択します。
  </Collapser>

  <Collapser id="dash-null" title="Null 値の表示オプション">
    クエリ対象のイベント属性にゼロがあると、時系列グラフが壊れて見えることがあります。ただし、null 値が常に状況を正しく表しているとは限りません。

    NULL 値を表示するオプションを選択して、時系列グラフを理解しやすくすることができます。それらには以下が含まれます：

    * Null のままにする (デフォルト): Null 値はグラフにギャップとして表示され、線が切断される可能性があります。レポートのギャップを見つけることが重要な場合、または実線を接続する意味がない場合は、このオプションを使用することをお勧めします。

    * 値を削除します。ヌル ポイントはチャートに存在しません。たとえば、1 日の 1 時間あたりのデータ ポイントを表示するグラフでは、24 ポイント未満が表示されます。

    * 最後の値を保持する: Null 値は前の期間の値を保持します。たとえば、グラフが 1 時間ごとの累積売上を報告しているが、1 時間のデータが欠落している場合、欠落している時間は前の 1 時間と同じ売上数を反映します。

    * 値をゼロに変換する: 欠損データはゼロとして表示されます。このオプションは、意味のあるトランザクション、カウント、または使用法がないことを意味するデータがない場合に役立ちます。

      <img title="Null value chart options" alt="Null value chart options" src="/images/dashboards_screenshot-crop_null-values.webp" />

      <figcaption>
        チャートに NULL 値を表示するには、4 つの方法から選択できます。
      </figcaption>

    null 値の表示方法をカスタマイズするには、 <DNT>**Null values**</DNT>セクションで適用するオプションを選択してください。各シリーズで異なるオプションを使用するには、 <Icon name="fe-plus" />設定するには、 <DNT>**Null values per series**</DNT>横にあるアイコンをクリックしてください。
  </Collapser>

  <Collapser id="thresholds" title="閾値">
    閾値は、グラフや表にコンテキストを追加し、データの解釈を明確にする視覚的な手がかりです。 視覚化を向上させる方法は次のとおりです。

    * 境界線: 時系列グラフで許容される最小値と最大値を確立します。

    * 色付きの領域: 閾値間の領域を網掛けして、通常の動作範囲を視覚的に識別したり、重要なゾーンを強調表示したりします。

    * 条件付きセルの色分け: 値に基づいてテーブル セルに色コードを適用し、データの外れ値や傾向を即座に明らかにします。

      <Video type="wistia" id="ikpu24sqjr" />

    * 時系列チャートの場合:

      1. グラフを編集し、 <DNT>**Thresholds**</DNT>メニューを展開します。

      2. <DNT>**+**</DNT>を選択して閾値を追加し、 <DNT>**Name**</DNT>を入力します。

      3. <DNT>**From**</DNT>と<DNT>**To**</DNT>の値を入力します。 1 行だけ作成する場合は、両方のフィールドに同じ番号を入力します。

      4. 重大度レベルを選択します。 線または領域の色を決定します。

         <img title="Configure chart thresholds in time series charts" alt="Screenshot of how to configure thresholds in a time series chart" src="/images/charts_screenshot-full_chart-threshold-time-series.webp" />

    * 表内:

      1. グラフを編集し、 <DNT>**Thresholds**</DNT>メニューを展開します。

      2. <DNT>**+**</DNT>を選択して閾値を追加し、 <DNT>**Column**</DNT>を選択します。

      3. <DNT>**From**</DNT>と<DNT>**To**</DNT>の値を入力します。 上限を設定するには、 <DNT>**From**</DNT>フィールドを空のままにします。 下限を設定するには、 <DNT>**To**</DNT>フィールドを空のままにします。

         <img title="Configure chart thresholds in tables" alt="Screenshot of how to configure thresholds in a table" src="/images/charts_screenshot-full_chart-threshold-table.webp" />
  </Collapser>
</CollapserGroup>

## チャートの共有と表示オプション [#share-charts]

ほとんどのチャートには、チャート埋め込みオプション、チャートを画像として取得する、ダッシュボードにチャートを追加するなど、さまざまなオプションがあります。 [「ダッシュボードとグラフのインポート、エクスポート、追加」を](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data/)参照してください。 チーム内でグラフを共有するには、 [「ダッシュボードとグラフを社内で共有する」を](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-internally/#share-charts-internally)参照してください。 グラフを外部で共有するには、 [「ダッシュボードとグラフを公開して共有する」を](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#chart)参照してください。
