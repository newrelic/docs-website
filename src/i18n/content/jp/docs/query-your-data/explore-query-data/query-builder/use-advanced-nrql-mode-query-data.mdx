[クエリビルダー](https://one.newrelic.com/data-exploration/query-builder)には2つのモード（NRQLモードとPromQLスタイルモード）があります。

NRQLクエリの作成方法については、[NRQLドキュメント](/docs/insights/nrql-new-relic-query-language/using-nrql/introduction-nrql)を参照してください。次の表では、NRQLモードでクエリビルダーを使用するための追加のヒントを示します。

        複数クエリを使用するには、`TIMESERIES`とともに最初のクエリを入力して実行します。結果が出たら、**別のクエリを追加**ボタンが有効になり、別のクエリを追加できます。
