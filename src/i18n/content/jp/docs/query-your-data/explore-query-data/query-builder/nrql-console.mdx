NRQL コンソールは、実行中のコンテキストを維持しながら、 **[one.newrelic.com](https://one.newrelic.com/all-capabilities)**の任意の場所にあるデータをクエリするための簡単な方法です。サービスの監視、問題のトラブルシューティング、または単にプラットフォームの閲覧を行っている場合でも、NRQL コンソールを使用すると、データを探索して、現在のビューから離れずに、クリックするだけで表示されている内容をより深く理解できます。データのクエリを高速化しましょう!

NRQL コンソールを有効または無効にするには、 [ユーザー メニュー](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) に移動し、NRQL コンソール トグルを探します。
