## 事前構築済みのダッシュボードをインストールする方法 [#use]

事前構築済みのダッシュボードをインストールするには:

1. ダッシュボード ページから、カスタマイズする事前構築済みのダッシュボードを検索します。
2. 事前構築済みのダッシュボードをクリックして選択します。
3. データとダッシュボードを追加するアカウントを選択します。
4. **「データ ソースの設定」** セクションの指示に従います。データ ソースを既にインストルメント化している場合は、この手順を省略できます。
5. **View dashboa**\[ダッシュボードを表示] をクリックしてダッシュボードを展開し、データの視覚化を開始します。作成済みの新しいダッシュボードをカスタマイズできることに注意してください ( [「ダッシュボードを管理する](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/)」を参照)。名前と設定の変更、新しいコンテンツと新しいウィジェットの追加、チャートの改良などもできます。

## その他のテンプレートを表示する方法 [#more-templates]

ダッシュボード カタログ全体を参照して、利用可能なすべての統合のダッシュボード テンプレートを見つけます。人気のある事前構築済みダッシュボードのタイトルの横にある **View All**\[すべて表示] リンクをクリックして、使用可能なすべての事前構築済みダッシュボードを表示および検索します。を使用できます。 <Icon name="fe-chevron-left"/> と <Icon name="fe-chevron-right"/> ボタンをクリックして、より多くの事前構築済みダッシュボードを参照します。
