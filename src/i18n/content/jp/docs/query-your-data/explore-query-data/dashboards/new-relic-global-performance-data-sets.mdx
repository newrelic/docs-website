New RelicのGlobal Performanceデータセットを使用して、有用で大規模な集計されたテレメトリーデータにただちにアクセスできます。

Global Performanceデータセットのデータは、次の方にとって有用です。

* 集計された実際のデータのキュレート済みの集合を一般的にモニターしトラブルシューティングについての知見を得たい、既存のNew Relicのお客様。
* New Relicのダッシュボードやデータツールについて知り、自身のデータをNew Relicに追加する方法について情報に基づき決定を行うため実際にデータを視覚化して表示したい、新規のお客様。

このリソースには、Global Performanceデータセットについての情報やそのしくみ、動作、Global Performanceデータへのアクセスについての現在のオプションが記載されています。

## 有用なデータにアクセスしNew Relicを試してみる [#why-matters]

当社初のGlobal Performanceデータセットである[Public API Performance](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/enable-public-api-performance-dashboard)は、AWSやGoogle、New Relicのお客様が（正規に）体験できるものを含む、パブリックAPIのパフォーマンスについての、実際のリアルタイムデータを数多く提供しています。

このデータはすでにNew Relicを通じて流れているため、アカウントのアクティベーションから数秒以内にデータにアクセスし、プロセスでNew Relicのダッシュボードやクエリ機能を試してみることができます。

自身のデータソースをNew Relicに最初に接続する必要はありませんが、お客様自身のデータの追加は簡単かつ無料で行え、New Relicがお客様のビジネスニーズにどのように役立つかを理解するのに最高の方法であるため、データソースを追加することを推奨します。

## Public API Performanceダッシュボードを開始する [#enable]

Public API Performanceデータを使用して開始する一助として、当社ではPublic API Performanceダッシュボードを作成しました。このダッシュボードは、実行可能な一般的な知見と分析の両方を提供し、またダッシュボードの機能を自身のデータにどのように適用可能かの例も提供します。

新規のお客様と既存のお客様のいずれも、Public API PerformanceダッシュボードでGlobal Performanceデータを迅速かつ簡単に表示できます。このダッシュボードの使用を開始するには、[Public API Performanceダッシュボードについて詳しく調べる](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/enable-public-api-performance-dashboard)をご覧ください。
