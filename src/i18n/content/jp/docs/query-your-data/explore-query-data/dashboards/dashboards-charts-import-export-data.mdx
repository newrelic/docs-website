[ダッシュボード](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/)を使用すると、システムの状態を視覚化および追跡し、問題に取り組み、さらなるステップを計画できます。さまざまなインポートおよびエクスポートのオプションを使用して、すべての情報をチーム、顧客、またはその他の関係者と共有する方法を学びます。

## ダッシュボードを JSON としてインポートする [#import-json]

コードレベルですぐに使用できるダッシュボードを変更する場合は、ダッシュボードを JSON としてインポートできます。 たとえば、2 つの既存のダッシュボードを結合してカスタム ダッシュボードを作成する場合、ベースのダッシュボードの JSON を変更して、それを New Relic にインポートし直すことができます。

ダッシュボードを JSON としてインポートするには:

1. **[one.newrelic.com >All capabilities](https://one.newrelic.com/all-capabilities) > Dashboards**に移動します。
2. **...**をクリックして既存のダッシュボードの JSON を開き、 **Copy JSON** \[JSON のコピー]を選択します。
3. **Dashboards** \[ダッシュボード]ページに戻ります。
4. 右上隅にある**Import dashboard** \[ダッシュボードのインポート]をクリックします。
5. JSON コードを貼り付けます。
6. ダッシュボードのアカウントと権限設定を選択します。 アカウントを一度設定すると変更することはできませんが、 [権限はいつでも変更](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-settings)できます。
7. クリック **保存**.

## ダッシュボードを PDF としてエクスポートする [#export-dashboards-pdf]

ダッシュボードを PDF ファイルとしてエクスポートするには、ダッシュボードの右上隅で**...**をクリックし、 **Export dashboard as PDF** \[ダッシュボードを PDF としてエクスポート]を選択します。

## チャートをCSVとしてエクスポート [#export-charts-csv]

ダッシュボードの表グラフを CSV ファイルとしてエクスポートするには、グラフの右上隅で**\[...]**をクリックし、 **\[CSV としてエクスポート]**を選択します。

<img
  title="Export chart as CSV"
  alt="Screenshot of how to export a table chart as a CSV file"
  src={dashboardsExportCsv}
/>

CSV としてのエクスポートは強力ですが、制限があります。留意すべき点がいくつかあります:

* データは常に UTC 時間 (-7:00) でエクスポートされます
* エクスポートは、次のようなアグリゲータ関数をサポートしていません。 `average()`
* エクスポートは`COMPARE_WITH`句をサポートしていません

## ダッシュボードをチームと共有する [#share-dashboards]

どのダッシュボードからでも、以下のいずれかの方法で共有することができます。

* ダッシュボードの右上にあるリンク アイコンをクリックして、ダッシュボードのパーマリンクをコピーします。 パーマリンクは、ダッシュボードへの直接リンクを共有します。 これらのリンクは 6 か月以内に使用されないと期限切れになることに注意してください。
* ダッシュボードを**as JSON** \[JSON として]コピーし、右上の **...**ボタンをクリックして**Copy JSON to clipboard** \[JSON をクリップボードにコピー]を選択してクリップボードに追加します。

また、 [ダッシュボードを複製することもできます](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-clone) 。
