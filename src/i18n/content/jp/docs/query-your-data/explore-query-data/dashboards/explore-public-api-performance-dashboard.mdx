Public API Performanceダッシュボードは、New Relicの[Global Performanceデータセット](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/new-relic-global-performance-data-sets)がサポートするダッシュボードです。New Relicアカウントの一部として含まれる、すぐに使えるダッシュボードです。パブリックAPIのパフォーマンスについての実行可能な知見と、新規のお客様が自身のデータに追加する前にNew Relicのダッシュボード機能を試す機会の両方を提供します。

ダッシュボードでは、一般的なパブリックAPIにアクセスする際にNew Relicのお客様を匿名化してサンプリングすることで生じる実際のレイテンシを表示します。

<Callout variant="important">
  Global Performanceデータセットは、現状のままで提示されます。Global Performanceデータセットは、一連のソース全体でのサンプルの集計を表し、New Relicはデータが正しいことや完全なこと、データの信憑性の確認は行いません。このデータは、データを使用する目的について唯一の情報源としないでください。New Relicは、このデータに基づいて行われた決定について責任を負いません。Global Performanceデータセットは、データセットで表明された技術についてのNew Relicによる保証や推奨とは考えないでください。
</Callout>

## New Relicにダッシュボードを追加する [#Add-dashboard]

Public API PerformanceダッシュボードがUIにまだ表示されていない場合、簡単に追加できます。

1. [one.newrelic.com/](https://one.newrelic.com/)から、ダッシュボードを有効にします。

   * 新規のお客様：ダッシュボードはデフォルトで有効にされており、すべての新規アカウントについてお気に入りリストに追加されます。
   * 既存のお客様：ダッシュボードがまだ有効にされていない場合、アバターをクリックして**データを追加**を選択して追加できます。

2. **Public API Performance**タイルをクリックしてアカウントセレクターを開いてから、**事前構築されたダッシュボードを追加して表示**をクリックします。

3. **Public API Performance**ダッシュボードのページで、詳しく調べてください。ページのペインの角にある**...**をクリックして、チャートの拡張やクエリの表示などを行います。

## ダッシュボードを詳しく調べる [#explore-dashboard]

Public API Performanceダッシュボードを詳しく調べる方法についての提案は次のとおりです。

* チャートの角にある**...**をクリックし、**クエリを表示**を選択し、使用した[NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql)クエリを表示してチャートを作成します。
* チャートの角にある**...**をクリックし、**画像として取得**を選択して、画像としてチャートを表示またはダウンロードします。
* バーチャートから特定のドメインを選択するか、ページの上部にあるテキストフィールドをクリックしてフィルターを追加します。
* 自身のデータをすでに追加している場合、クエリをコピーし自身の利用に合わせて変更してください。

<Callout variant="important">
  Public API Performanceダッシュボードは現在、EUのお客様にはご利用いただけません。
</Callout>

<Callout variant="important">
  Public API Performanceダッシュボードは現在、アラートをサポートしていません。
</Callout>

## ダッシュボードとデータの詳細 [#explore-dashboard]

* Public Performance APIダッシュボードを駆動するGlobal Performanceデータセットの詳細については、[New Relic Global Performanceデータセット](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/new-relic-global-performance-data-sets)をご覧ください。
* New Relicダッシュボードの詳細については、[ダッシュボードの説明](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards)をご覧ください。
* お客様は、新たな[データエクスプローラー](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer)を使用して、さらに詳しくこのデータセットについて知ることもできます。
