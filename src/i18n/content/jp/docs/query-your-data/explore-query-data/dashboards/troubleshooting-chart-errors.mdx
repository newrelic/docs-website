## ウィジェットのリンクと説明に関する問題 [#widget-link-issues]

ウィジェットのリンクや説明に問題が発生している場合は、以下のよくある問題とその解決策を確認してください。

<table>
  <thead>
    <tr>
      <th style={{ width: "400px" }}>
        <DNT>
          **Problem**
        </DNT>
      </th>

      <th style={{ width: "400px" }}>
        <DNT>
          **Solution**
        </DNT>
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        ウィジェットのリンクがクリックできません
      </td>

      <td>
        URLが有効であり、サポートされているプロトコル（ `http://`または`https://` ）を使用していることを確認してください。他のプロトコルとの接続はサポートされていません。
      </td>
    </tr>

    <tr>
      <td>
        説明ツールチップが表示されない
      </td>

      <td>
        説明欄にテキストが入力されていることを確認してください。ツールチップを表示するには、ウィジェットのタイトル領域にカーソルを合わせてください。それでも表示されない場合は、ダッシュボードを更新してみてください。
      </td>
    </tr>

    <tr>
      <td>
        URL検証エラー
      </td>

      <td>
        URLが2,048文字を超えず、 `http://`または`https://`プロトコルのみを使用していることを確認してください。
      </td>
    </tr>

    <tr>
      <td>
        リンクをクリックすると404エラーが表示されます
      </td>

      <td>
        ターゲット URL が正しく、アクセス可能であることを確認してください。 外部リンクは、一般に公開されているか、ネットワーク内でアクセス可能である必要があります。
      </td>
    </tr>

    <tr>
      <td>
        ウィジェットのリンクや説明は編集できません
      </td>

      <td>
        ダッシュボードの編集権限があることを確認してください。ウィジェットの設定を変更できるのは、適切な権限を持つユーザーのみです。
      </td>
    </tr>

    <tr>
      <td>
        APIミューテーションはエラーを返さないが、変更が反映されない
      </td>

      <td>
        ダッシュボードページを更新してください。それでも変更が表示されない場合は、 NerdGraph APIエクスプローラーを使用して、IDが正しいこと、およびミューテーションが正常に完了したことを確認してください。
      </td>
    </tr>
  </tbody>
</table>
