## ダッシュボードを社内で共有する [#share-dashboards-internally]

次のいずれかのオプションを使用して、チーム内で任意のダッシュボードを共有できます。

* ダッシュボードの右上にあるリンク アイコンをクリックして、ダッシュボードのパーマリンクをコピーします。パーマリンクは、ダッシュボードへの直接リンクを共有します。

または

* 次のようにダッシュボードを<DNT>**JSON**</DNT>形式でコピーし、チーム内で共有します。

  1. <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com/dashboards)**</DNT>に移動します。
  2. クリック<Icon name="fe-more-horizontal" />ダッシュボードの右上隅にあるアイコン。
  3. <DNT>**Copy JSON to clipboard**</DNT>を選択します。
  4. JSON コードをメッセージまたはドキュメントに貼り付けて、チームと共有します。

## チャートを社内で共有する [#share-charts-internally]

ダッシュボードと同様に、次の手順に従ってチーム内でグラフを共有できます。

1. <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com//dashboards)**</DNT>に移動します。
2. 共有したいチャートが含まれているダッシュボードを開きます。
3. <DNT>**View query**</DNT>を選択してください<Icon name="fe-more-horizontal" />チャートのメニュー。 書き込みビルダーで書き込みが開きます。
4. クエリビルダーから、 <DNT>**Copy query link**</DNT>をクリックします。 リンクがクリップボードにコピーされます。
5. 電子メール、チャット、またはその他のコミュニケーション ツールを使用してチーム メンバーと共有します。
