### Fluent Bit 5.0.8のサポート

ユーザーはFluent Bitバージョン5.0.8を受け取るようになります。詳細については、[Fluent Bit のお知らせ](https://fluentbit.io/announcements/)を参照してください。

このリリースには、[New Relic Fluent Bit Outputプラグイン 3.7.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.7.0)が含まれています。

### 追加した

* New Relic Fluent Bit Outputプラグイン 3.7.0インテグレーション

### かわった

* CVE修正のため、Goのバージョンが1.26.4にアップグレードされました
* infrastructureエージェントは、Fluent Bit 5.0.8パッケージ（Linux）および5.0.7（Windows）を推奨しています
* [Fluent Bit OutputプラグインのDockerイメージ](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output)は、出力プラグイン3.7.0を備えたFluent Bit 5.0.8を使用します
* [New Relic Logging Helmチャート](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging)は新しいプラグインイメージをインストールします
* WindowsのDockerイメージはFluent Bit 5.0.7を使用します（5.0.8のWindowsイメージはアップストリームでまだ利用できません）

最新の修正と機能強化の情報を常に把握するには、[ログ RSS フィード](/docs/release-notes/logs-release-notes/)を購読してください。
