### 新しいドキュメント

* [メトリクスとイベント エクスプローラー](/eol/2025/05/metrics-and-event-explorer/)のサポート終了日が 2025 年 6 月 16 日に発表されました。
* エージェント経由で [カスタマー インタラクションを作成する手順](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/android-agent-custom-Interaction/) を追加しました。Android

### 主な変更点

* `NEW_RELIC_CONFIG_FILENAME`環境変数オーバーライドを[Node.jsエージェント設定](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration/#filename)に追加しました。
* Rubyエージェントのディストリビュー[ティッド（分散）トレーシング設定](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration/#distributed_tracing-sampler-remote_parent_sampled)に、トレース親にトレースフラグが設定されている場合にリモート親とのトランザクションサンプリング動作を管理する設定を追加しました。
* [AI モニタリングの互換性と要件](/docs/ai-monitoring/compatibility-requirements-ai-monitoring/) ページに、 エージェントのサポート対象ライブラリとして Google Gen AI SDKPython を追加しました。
* [アラート条件ページ](/docs/alerts/create-alert/create-alert-condition/alert-conditions/#disable-health-status-reporting)に、個別通知発生時のエンティティ稼働ステータスの自動変更を防止できる機能を追加しました。
* [「ログUIの使用」](/docs/logs/ui-data/use-logs-ui/#customizable-table-views)ページのログ テーブル ビューのカスタマイズに機能強化が追加されました。
* インストールまたは操作時に発生する一般的な問題を[ NRDiag を使用して](/docs/new-relic-control/agent-control/troubleshooting/#nrdiag)Agent Control トラブルシューティングする方法についての情報を追加しました。
* JSON IP 範囲リストから古い合成[モニター パブリックミニオン IP](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/#dynamic-ip-addresses)範囲を削除するため、サポート終了日を 2025 年 5 月 21 日まで延長しました。
* インフラエージェントと[オンホスト統合が Linux 環境でのみ ARM64 をサポートすることを反映するために、ARM64/Graviton および x86\_64 のエージェントのサポートに関する](/docs/apm/agents/manage-apm-agents/configuration/support-arm-graviton-x86-64/)ドキュメントを更新しました。
* Dashboards[ユーザーの削除](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/#delete-users) 時に削除される資産リストから を削除しました。
* New Relic プラットフォームの[アプリケーション応答なし (ANR) UI](/docs/mobile-monitoring/mobile-monitoring-ui/application-not-responding/#view-anr-new-relic)の新しいユーザー機能に関する情報を追加しました。
* New Relic の Observability for Good プログラムの[資格ガイドライン](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/eligibility-guidelines-new-relic-nonprofit-program/#exceptions)を更新しました。
