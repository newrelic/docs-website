### 新しいドキュメント

* Ruby エージェント バージョン 10.0.0 へのアップグレードに関する包括的なガイダンスを提供するために、 [Ruby エージェント 10.x 移行ガイド](/docs/apm/agents/ruby-agent/getting-started/migration-10x-guide)を追加しました。
* [OpenTelemetry Kafka 統合](/docs/opentelemetry/integrations/kafka/overview)ドキュメントを追加して、自己ホスト型デプロイメント、Strimzi を使用したKubernetes 、メトリクス リファレンス、データ クエリなど、 OpenTelemetryを使用した監視 Kafka のガイダンスを提供しました。
* ホストベースおよびKubernetesデプロイメントを含む、 OpenTelemetryを使用した NGINX および NGINX Plus の監視に関するガイダンスを提供する[OpenTelemetry NGINX インテグレーション](/docs/opentelemetry/nginx/nginx-otel-overview)ドキュメントを追加しました。
* 基本的なワークフロー、インテグレーション、および高度な使用例の実践的なパターンを提供する[ワークフロー オートメーションの例](/docs/workflow-automation/workflow-examples/workflow-examples)を追加しました。

### 主な変更点

* CCI 予算とコスト最適化の推奨事項のページを追加して、 [Cloud Cost Intelligence](/docs/cci/financial-intelligence/cloud-cost-insights)ドキュメントを更新しました。
* マルチスコープのロールと許可のサポートを含む、 [NerdGraph 管理グループ API](/docs/apis/nerdgraph/examples/nerdgraph-manage-groups)ドキュメントが更新されました。
* マルチレベルの予算設定のサポートにより、[予算計算](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-budgets/overview)ドキュメントが更新されました。
* 構成可能なプロセスの除外を含む強化された[.NET エージェント設定](/docs/apm/agents/net-agent/configuration/net-agent-configuration)ドキュメント。
* 精度を向上させるために[.NET エージェントの互換性要件](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements)を更新しました。
* Windowsネットワーク メトリクスの新しい`win_network_interface_v2`オプションを使用して、 [インフラストラクチャ エージェント設定の設定を](/docs/infrastructure/infrastructure-agent/configuration/infrastructure-agent-configuration-settings)更新しました。
* PHP 8.5 サポートのための[PHP エージェントの互換性要件](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements)を更新しました。
* バージョン 9.0.0 の[Java エージェントの互換性要件](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent)が更新されました。
* 起動アーティファクトを含む更新された[Javaエージェント設定ファイルの](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file)ドキュメントには設定が含まれています。
* サポートされている最新バージョンの[Node.js エージェントの互換性要件](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent)を更新しました。
* [Rubyエージェント設定](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration)ドキュメントを更新しました。
* [AI モニタリングの互換性要件](/docs/ai-monitoring/compatibility-requirements-ai-monitoring)を更新して明確化しました。
* SAML およびSSO設定の[認証ドメインの](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more)ドキュメントを更新しました。

### マイナーチェンジ
