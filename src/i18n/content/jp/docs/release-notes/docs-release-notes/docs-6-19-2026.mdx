### 新しいドキュメント

* OpenTelemetry Prometheusレシーバーを使用したnginxの監視に関する包括的なガイダンスを提供するために、[ホスト環境](/docs/opentelemetry/integrations/nginx/nginx-otel-host)および[Kubernetes](/docs/opentelemetry/integrations/nginx/nginx-otel-kubernetes)向けのPrometheusレシーバーによるnginx監視を追加しました。

### 主な変更点

* 最新のサポート対象フレームワークバージョンで[.NETエージェントの互換性要件](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements)を更新しました。

### マイナーチェンジ

* 顧客の利用資格と価格モデルの情報を明確にし、以下のドキュメントを強化しました：

  * [予測アラート](/docs/alerts/create-alert/set-thresholds/predictive-alerts)
  * [パブリックダッシュボード](/docs/query-your-data/explore-query-data/dashboards/share-dashboards-publicly)
  * [NRQL予測ドキュメント](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions)

* 最適化されたOpenTelemetry設定の例を使用して、[ECS EC2監視設定](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2)を更新しました。

* 改善された設定で[ECS Fargate監視設定](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate)を更新しました。

* csecエンドポイントの許可リストIPを更新し、[IASTトラブルシューティングドキュメント](/docs/iast/troubleshooting)を更新しました。

* New Relic AI Knowledgeインテグレーションセクションを追加し、[SREエージェントの使用に関するドキュメント](/docs/agentic-ai/sre-agent/use-sre-agent)を強化しました。

* 強化された設定例を追加し、[ブラウザのセッションリプレイのプライバシー設定](/docs/browser/browser-monitoring/browser-pro-features/session-replay/configuration/customize-privacy-settings)を更新しました。

### EOL

* [サポート終了のお知らせ: Drop Filter Rules API](/eol/2025/05/drop-rule-filter) : 移行をより適切にサポートし、混乱を最小限に抑えるため、EOLの日付を2026年8月31日に延長して更新しました。
