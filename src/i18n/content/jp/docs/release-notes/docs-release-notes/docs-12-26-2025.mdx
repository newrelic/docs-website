### 新しいドキュメント

* [GitHub Enterprise 統合](/docs/service-architecture-intelligence/github-integrations/github-enterprise-integration/)ドキュメントを追加しました。

### 主な変更点

* ScalingMode とバッチ処理のサポートにより、 [Azureログ転送が](/docs/logs/forward-logs/azure-log-forwarding/)更新されました。
* モデル コンテキスト プロトコル統合のための IAM 権限要件が改訂され、 [MCP の概要](/docs/agentic-ai/mcp/overview/)が更新されました。
* 権限と許可の要件ガイダンスを強化して、 [MCP のトラブルシューティング](/docs/agentic-ai/mcp/troubleshoot/)を更新しました。
* 強化されたエンティティ スコープの権限を含む、[ユーザー権限の](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/)ドキュメントが更新されました。
* Fluent Bit プラグインの脆弱性に対処する NR25-02[セキュリティ](/docs/security/security-privacy/information-security/security-bulletins/)速報を更新しました。

### マイナーチェンジ
