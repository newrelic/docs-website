### 主な変更点

* NRQL ドキュメント全体の左側のナビゲーションを整理して簡素化しました。

* [MongoDB](/install/mongodb/) 、 [MySQL](/install/mysql/) 、および[PostgreSQL](/install/postgresql/)のオンホストインテグレーション ドキュメントを確認して書き直して、インストールを容易にし、ドキュメントが正確で最新であることを確認しました。

* 欠落していたディメンション メトリクスを[AWSインテグレーション メトリクス](/docs/infrastructure/amazon-integrations/get-started/aws-integrations-metrics)に追加しました。

* さまざまなドキュメント サイト コンポーネントのスタイルが改善されました。

  * 読みやすさと製品の一貫性を向上させるために色を更新しました。
  * 左側のナビゲーションから「製品へ移動」コンポーネントを削除しました。
  * ドキュメントのロゴから六角形の背景色を削除しました。
  * インタラクティブなインストレーション ドキュメントに右ナビゲーション フィードバック フォームを追加しました。
  * さまざまなドキュメントからサインアップバナーを削除しました。
  * ポップオーバーのテキストサイズを大きくしました。
