### 新しいドキュメント

* [データ取り込み予算](/docs/data-apis/manage-data/data-ingest-budgets)を追加し、予算を使用してデータ取り込みコストを管理するための包括的なガイダンスを提供しました。
* 複数のKubernetesクラスタを監視するための包括的なガイダンスを提供するために[、 Kubernetesクロスクラスタ オブザーバビリティ](/docs/kubernetes-pixie/kubernetes-integration/understand-use-data/kubernetes-cross-cluster)を追加しました。

### 主な変更点

* 一般公開に向けて、大幅な改訂を加えて[Advanced Maps](/docs/service-architecture-intelligence/maps/advanced-maps)ドキュメントを更新しました。
* 一般公開に向けて、[カタログ](/docs/service-architecture-intelligence/catalogs/catalogs)ドキュメントが更新され、コンテンツと組織が更新されました。

### リリースノート

* 弊社の最新リリース情報を常に把握してください:

  * [Lambda拡張機能 v2.4.0](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.0) :

    * パフォーマンスとメモリの安全性を向上させるために、Go から Rust に完全に書き直しました。
    * New Relic APMプラットフォームに直接レポートするためのLambda関数のAPMモード サポートを追加しました。
    * 最適化された Rust コンパイルにより、コールド スタート時間が 20 ～ 40% 短縮されました。

  * [Node.jsエージェント v13.8.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-8-0) :

    * @langchain/AWS 計装のサポートを追加しました。
    * 部分粒度追跡メトリクスを追加しました。

  * [Node.jsエージェント v13.7.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-7-0) :

    * OTEL スパン リンクのサポートが追加されました。
    * 比率ベースのサンプラーを構成する機能が追加されました。
    * @koa/router\@15.0.0+ の koa 計装を更新しました。
    * OTEL hrtime の正規化を修正しました。

  * [Python v11.2.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-110200) :

    * Strands 計装のサポートを追加しました。
    * AWS Bedrock Converse Streaming のサポートが追加されました。
    * Redis メソッド hybrid\_search および msetex のサポートが追加されました。
    * ASGI ヘッダー消費の問題を修正しました。

  * [インフラストラクチャエージェント v1.71.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1713) :

    * alpine docker タグを v3.23 に更新しました。
    * goバージョンを1.25.5に更新しました。

  * [インフラストラクチャエージェント v1.71.2](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1712) :

    * CVE-2025-12972、CVE-2025-12970、CVE-2025-12978、CVE-2025-12977、CVE-2025-12969 の修正を含む Fluent Bit バージョンを更新しました。
    * 32 ビット Windows ビルドが削除されました。

  * [Kubernetesインテグレーション v3.51.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-51-0) :

    * 新しい機能と改善により統合が更新されました。
