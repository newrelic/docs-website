### 主な変更点

* [Node.js エージェントの](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)互換性レポートを更新しました。
* 最新の検証済み互換性ライブラリ バージョンで[.NET エージェントを](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/)更新しました。

### マイナーチェンジ

* Linux の[ インフラストラクチャエージェント](/docs/infrastructure/infrastructure-agent/requirements-infrastructure-agent/#os)[ をサポートするために、 Debian](/docs/infrastructure/infrastructure-agent/linux-installation/package-manager-install/) OS をバージョン 12 (「ブックワーム」) に更新しました。
* データ制限を照会するための[`accountId()`](/docs/data-apis/manage-data/query-limits/)関数が更新されました。
* 64 ビット Windows (Intel/AMD) をサポートするように[Windows MSI インストーラー](/docs/infrastructure/infrastructure-agent/windows-installation/windows-msi-installer/#install-windows)を更新しました。

### リリースノート

* [AIモニタリングUI](/docs/ai-monitoring/drop-sensitive-data/)を通じてドロップ ルールのサポート終了日として 2026 年 1 月 7 日に発表されました。

* [iOS エージェント v7.5.9](/docs/release-notes/mobile-release-notes/ios-release-notes/ios-agent-759/)

  * ディストリビューティッド（分散）トレーシングがセッションの開始時に必要なアカウント情報をすべて持っていない可能性がある問題を修正しました。
  * エージェントの起動時にクラッシュを引き起こす可能性がある問題を修正しました。

* [Pythonエージェント v10.16.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-101600/)

  * 新しい[Redis](https://pypi.org/project/redis/)関数のサポートが追加されました。
  * すべての内部パスを`os.path`から`pathlib.Path`に変換しました。
  * JSON エンコード PosixPath に関する問題を修正しました。

* [Kubernetesインテグレーション v3.44.1](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-44-1/)

  * この統合は、次のチャート バージョンに含まれています。

    * [ニューレリック・インフラストラクチャー 3.49.1](https://github.com/newrelic/nri-kubernetes/releases/tag/newrelic-infrastructure-3.49.1)
    * [nri-バンドル-6.0.10](https://github.com/newrelic/helm-charts/releases/tag/nri-bundle-6.0.10)

* [.NET エージェント v10.44.1](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-44-1/)

  * Service Bus メトリック ラッパーで例外処理が修正されました。
  * Azure Function webhost (Linux) プロセスを計装から除外しました。

* [Node.js エージェント v13.2.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-2-0/)

  * 加入者ベースの計装用に`Supportability/Features/Instrumentation/OnRequire/<pkg>`メトリクスを登録する機能が追加されました。
  * orchestrio n-js で動作するように esm ローダー フックと commonjs パッチを追加しました。
  * `@modelcontextprotocol/sdk`クライアント呼び出し`client.callTool` 、 `client.readResource` 、および`client.getPrompt`の計装を追加しました。
  * OTEL パッケージの検出に関するロジックを修正しました。
  * 無効なサブスクライバーを適切に設定してスキップするように`shimmer.setupSubscribers`を更新しました。

* [Node.js エージェント v13.2.1](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-2-1/)

  * チャネルベースの計装をトレースするための CJS/ESM 登録に`@apm-js-collab/tracing-hooks`使用するようにエージェントを更新しました。
