### 主な変更点

* 一般的な Go アプリケーション パターンの包括的なステップバイステップのインストレーション ガイドと例を含む、 [Install New Relic for Go を](/docs/apm/agents/go-agent/installation/install-new-relic-go)更新しました。
* KubernetesクラスタとLinuxホストの完全なインストールと設定ガイダンスを提供するために、[セットアップとインストールAgent Control](/docs/new-relic-control/agent-control/setup)を追加しました。
* セッション リセット機能と更新されたセッション トラッキング動作を備えた、 [setUserId](/docs/browser/new-relic-browser/browser-apis/setuserid) API ドキュメントが強化されました。

### マイナーチェンジ

* PHP エージェントの互換性[要件に関する PHP](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements)エージェントの互換性情報を追加しました。
* [SPM インストレーション](/docs/mobile-monitoring/new-relic-mobile-ios/installation/spm-installation)文書を更新し、ガイダンスを強化しました。
* 説明が改善され、 [CocoaPods](/docs/mobile-monitoring/new-relic-mobile-ios/installation/cocoapods-installation)インストレーション ドキュメントが強化されました。
* 拒否リストのワイルドカード サポートを含む[フィルター AJAX 要求イベントの](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events)ドキュメントが更新されました。
* Lambda関数監視の[APMエクスペリエンスへのアップグレードを](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/upgrade-to-apm-experience)更新しました。
