### 新しいドキュメント

* 検出されたアプリケーションの脆弱性に対するJiraチケットの自動作成と同期に関するガイダンスを提供するため、[Security RX Jiraワークフロー](/docs/vulnerability-management/workflows/jira)を追加しました。
* 自動化されたワークフローの一部としてAzure VMの電源をオフにできるように、アクションカタログに[Azure仮想マシンの停止アクション](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-vm-stop)を追加しました。
* New Relicがセッション間でVideo OverviewおよびAd Overviewダッシュボードの設定を保存および復元する方法を説明するために、[ダッシュボード設定の保存と管理](/docs/streaming-video-&-ads/view-data-in-newrelic/user-preferences)に関する詳細を追加しました。
* 送信前に正規表現ベースのルールで機密データをマスクするためのガイダンスを提供するために、[Android](/docs/streaming-video-&-ads/installation/android/obfuscation-rules)、[ブラウザ](/docs/streaming-video-&-ads/installation/browser/obfuscation-rules)、[iOS](/docs/streaming-video-&-ads/installation/ios/obfuscation-rules)、および[Roku](/docs/streaming-video-&-ads/installation/roku/obfuscation-rules)向けのStreaming Video &amp; Adsエージェントに関する難読化ルールを追加しました。

### 主な変更点

* Security RX .NETインテグレーションのベースバージョンとして「10.51.1」を設定するために、[Security RXインテグレーションの概要](/docs/vulnerability-management/getting-started/integrations/overview)と[アプリケーションの概要](/docs/vulnerability-management/applications/overview)を更新しました。
* CCIのドキュメントを更新し、[プロバイダー別の請求取り込み価格](/docs/cci/cci-pricing)、[親アカウントとサブアカウントのインテグレーションスコープに関するガイダンス](/docs/cci/overview/#choose-an-integration-scope)、および[Azureセットアップの前提条件](/docs/cci/get-started/azure-cci)を追加しました。

### マイナーチェンジ

* インストゥルメントされたApolloおよびNext.jsアプリケーションの手順を追加して、[Node.jsエージェントのインストレーションガイド](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent)を更新しました。
* 最新の互換性レポートを使用して、[Node.jsエージェントの互換性要件](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent)を更新しました。
* [Rubyエージェントの設定](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration)ドキュメントを最新の設定オプションで更新しました。
* 実行中のタスク数の修正された設定を追加して、[ECS EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2)および[ECS Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate)の監視ドキュメントを更新しました。
* コンテキストの説明を追加し、アクションカタログの[New Relicエージェントアクション](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent)を強化しました。
