### 新しいドキュメント

* 計算消費単位 (CCU) の使用量を監視および集計するメカニズムを提供する[計算予算ドキュメント](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-budgets/overview)を追加しました。

### 主な変更点

* Node.js エージェントの[互換性と要件が](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent)更新されました。

* PHP エージェント用の[Alpine Linux の互換性と要件](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements)を更新しました。

* [Pipeline Control](/docs/new-relic-control/pipeline-control/modify-agent-configuration)のエージェント固有の要件を更新しました。

* [Javaエージェント](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file#jfr-harvest-interval)の設定要件に関連するJFRを更新しました。

* New Relic の価格と請求に関する以下の価格と請求情報が更新されました。

  * [データ：請求とルール](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/data-ingest-billing)
  * [使用関連のクエリとアラート設定](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts)
  * [New Relic 利用プラン](/docs/licenses/license-information/usage-plans/new-relic-usage-plan)

### マイナーチェンジ

* .NET エージェント[の互換性と要件に関するドキュメントで SQL の問題を把握するために、.NET](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements)エージェントの言語を明確にしました。
* [Oracleデータベース監視統合](/docs/infrastructure/host-integrations/host-integrations-list/oracle-database-monitoring-integration)の言語を洗練しました。
