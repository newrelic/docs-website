### 新しいドキュメント

* フリート全体にわたる大規模なインフラストラクチャ エージェント設定を管理するための[フリート コントロールAPI](/docs/new-relic-control/fleet-control/overview/)を追加しました。
* [合成モニターをChrome](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-browser-monitors-to-chrome-latest/)最新バージョンおよびNode.js 22 にアップグレードするための合成モニターChrome最新移行ガイドを追加しました。
* 外形監視に影響を与える重大な Google Chrome use-after-free 脆弱性 (CVE-2026-5281) に対処する手順を記載した[セキュリティ情報 NR26-04](/docs/security/new-relic-security/security-bulletins/security-bulletin-nr26-04)を追加しました。
* ワークフロー オートメーションを通じて Agentic プラットフォームで構成されたエージェントを実行するための[New Relicエージェント アクション](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent)を追加しました。

### 主な変更点

* [AWS LambdaとOpenTelemetryのドキュメント](/docs/serverless-function-monitoring/aws-lambda-monitoring/opentelemetry/lambda-opentelemetry-troubleshooting/)を更新し、CloudWatchメトリクスの問題やリクエストIDの問題を網羅したOpenTelemetry Lambdaのトラブルシューティングセクションを追加しました。

### マイナーチェンジ

* v1.312.0 および v1.312.1 の互換性情報を含む[Browser Agent リリース ノート](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/)を更新しました。
* [変更追跡 (変更追跡機能) の概要](/docs/change-tracking/overview/)を更新し、変更をNew Relicプラットフォーム上の任意のエンティティに関連付けることができることを明確にしました。
* 更新されたパッケージ バージョンで[Node.jsエージェントの互換性レポートを](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)更新しました。
* `Microsoft.Azure.Cosmos` 、 `Microsoft.Data.SqlClient` 、 `MongoDB.Driver`などの複数のパッケージの互換性が確認された最新のバージョンで[.NET エージェントの互換性要件](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/)を更新しました。
* 不足している分類修正を含む[.NET エージェントのリリース ノート](/docs/release-notes/agent-release-notes/net-release-notes/)を更新しました。
* ドキュメントセクション全体のユーザーエクスペリエンスを向上させるためにナビゲーション構造を更新しました。
* `NRConditionalMaskView`ブロック ビュー機能と、マスキングとブロックに関する包括的なガイダンスを含む[、モバイル セッションリプレイのドキュメント](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/)を更新しました。
* [データ保持期間に関するドキュメント](/docs/data-apis/manage-data/manage-data-retention/)が更新され、組織を対象とした保持管理と「アカウント」タブと「組織」タブの説明が追加されました。
* [Javaエージェント拡張モジュールのドキュメント](/docs/apm/agents/java-agent/instrumentation/extension-additional-instrumentation-modules/)を更新し、非推奨となったHikariCP 2.4.0インキュベーターモジュールを削除しました。これは、当該モジュールがエージェントに組み込まれたためです。
* [ナレッジ統合ドキュメント](/docs/agentic-ai/knowledge-integration/overview/)を更新し、権限名を更新し、役割を「組織製品管理者」から「組織マネージャー」に変更しました。
* 新しいイベントタイプ`NrComputeUsage`とそれに関連する属性定義を追加して、属性辞書を更新しました。
* データ使用量予測機能と高度な計算顧客向けの使用量予測機能を追加した、更新された[データ取り込み管理ドキュメント](/docs/data-apis/manage-data/manage-data-coming-new-relic/)。
* Oracle Kubernetes Engine (OKE) サポートとAPM自動アタッチの設定ガイダンスを含む[Kubernetesインテグレーション ドキュメント](/docs/kubernetes-pixie/kubernetes-integration/get-started/kubernetes-integration-compatibility-requirements/)を更新しました。
* [JavaエージェントのJava 2セキュリティに関するドキュメント](/docs/apm/agents/java-agent/additional-installation/install-java-agent-java-2-security/)を更新し、Solr v9のサポートとセキュリティマネージャーの設定に関するガイダンスを追加しました。
* [SCIMのサポートを更新し、自動ユーザー管理に](/docs/accounts/accounts/automated-user-management/scim-support-automated-user-management/)対応しました。 `title`フィールドを使用して役職情報を取得できるようになりました。

### 新着投稿

* ウィジェットのリンク、ウィジェットの説明、テンプレート変数の可視性コントロール、およびマークダウン ウィジェットとタイトルの動的テンプレート変数を通知するための[新しいDashboards機能](/whats-new/2026/03/whats-new-3-28-in-dashboards)を追加しました。
