### 新しいドキュメント

* [自動ログ](/docs/browser/browser-monitoring/browser-pro-features/auto-logging)により、Pro および Pro+SPA ブラウザエージェントのフロントエンドの観察機能が強化され、 `WARN`レベルの`console`ログメッセージの自動収集が可能になります。

### 主な変更点

* [Node.js エージェント](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)の互換性レポートを更新しました。
* [KTranslate Docker コンテナのヘルス監視](/docs/network-performance-monitoring/advanced/ktranslate-container-health)に、Linux パッケージからローカルでログを確認するメソッドを追加しました。
* 接続性とセキュリティを向上させるために、新しい有効な IP 範囲で[ネットワーク ブロック](/docs/new-relic-solutions/get-started/networks/)を更新しました。
* New Relic が使用する[OTLP プロトコル バージョン](/docs/opentelemetry/best-practices/opentelemetry-otlp/#otlp-version-support)を更新しました。
* [APM : レポート カスタムイベントとプロパティ](/docs/data-apis/custom-data/custom-events/apm-report-custom-events-attributes/#php-att)を更新し、PHP エージェントのカスタム イベントの最大数を 1 分あたり 30,000 から 100,000 に増やす機能を追加しました。
* [NewRelic Xamarin NuGet パッケージ](/docs/mobile-monitoring/new-relic-mobile-xamarin/monitor-your-xamarin-application/)の廃止とサポート終了に関する注意事項を追加しました。

### マイナーチェンジ

* サポートされる Python バージョンを更新しました。
* [Okta ユーザー プロファイル設定](/docs/accounts/accounts/automated-user-management/okta-scimsso-application-configuration/)で指定されたプロパティ値は、大文字と小文字が区別されるようになりました。
* [SCIM API ユーザー タイプ](/docs/accounts/accounts/automated-user-management/tutorial-manage-users-groups-scim/#manage-user-type)値では大文字と小文字が区別されるようになりました。
* サポートされている PHPエージェントのバージョンを更新しました。
