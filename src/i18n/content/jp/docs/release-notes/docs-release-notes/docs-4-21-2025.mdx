### 新しいドキュメント

* [NerdGraph チュートリアル: メトリクス正規化ルールの管理では、](/docs/apis/nerdgraph/examples/nerdgraph-metric-normalization-rule) NerdGraph を使用してメトリクス正規化ルールを管理するためのガイダンスを提供します。
* [スコアカード テンプレート ライブラリには、](/docs/service-architecture-intelligence/scorecards/templates)一般的な課題に取り組むためのエンタープライズ ベストプラクティスに基づいた、エンジニアリングおよびビジネス ユース ケース向けのカスタマイズ可能なテンプレートが含まれています。
* [New Relic OTel K8s メトリクス 考察では、](/docs/kubernetes-pixie/kubernetes-integration/installation/k8s-otel-metrics) Kubernetes OpenTelemetryメトリクスがNew Relic Kubernetesメトリクスにどのようにマッピングされるかを概説します。

### 主な変更点

* [Javaエージェント設定: Config File](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file/#dt-sampler-remote-parent-sampled)に`sampler.remote_parent_sampled`追加し、上流のエンティティからの親スパンがサンプリングされたときにエージェントがサンプリング スパンをどのように処理するかを定義しました。
* [SCIM プロビジョニング](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim)を更新して、New Relic での自動ユーザー管理に<DNT>**Paid Edition**</DNT>必要であることを指定します。
* [パイプライン制御用のクラウド ルール API](/docs/new-relic-control/pipeline-control/cloud-rules-api/#drop-attributes-on-dimensional-metric-rollups)ドキュメントを更新し、ディメンション メトリクス ロールアップからプロパティを削除する方法に関する情報を追加しました。
* [アカウント変更追跡 (変更追跡機能)](/docs/accounts/accounts/account-maintenance/query-account-audit-logs-nrauditevent/#user-login)ページに、ログインおよびログアウト イベントを追跡するための書き込みを追加しました。

### マイナーチェンジ

* [.Net エージェントの互換性バージョン](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/#framework)を更新しました。
* グループ プロビジョニングの制限によりこれまでサポートされていなかった[自動ユーザー管理](/docs/accounts/accounts/automated-user-management/automated-user-provisioning-single-sign)における PingOne の制限が削除されました。
* 計算上の制限により月次または年次季節性をサポートしないという制限を[異常検出](/docs/alerts/create-alert/set-thresholds/anomaly-detection/)ページに追加しました。
* [Node.js エージェントの互換性レポート](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent)を更新しました。
* [Node.js エージェント構成](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration/#infinite-tracing-batching)ドキュメントに、無限トレースのバッチ処理と圧縮の設定を追加しました。
