### 主な変更点

* 新しいプロセッサ (フィルター、サンプリング、変換)、 UIガイド、YAML 設定の概要、データ スキーマの違いなど、v2.0.0 の大幅な改訂を加えて[Pipeline Control Gateway ドキュメント](/docs/new-relic-control/pipeline-control/gateway/overview)を更新しました。
* [Elasticsearch OTel インテグレーションの](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-install)インスト[レーションと概要](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview)を更新し、インストレーションと設定ガイダンスの包括的な改善を行いました。
* [Javaエージェント設定](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file)を更新し、ハイブリッド エージェント設定サポートに関する広範なドキュメントを追加しました。
* [Go エージェントのログ記録を構成するための包括的なガイダンスを含む、Go エージェントのログ記録ドキュメント](/docs/apm/agents/go-agent/configuration/go-agent-logging)が更新されました。
* 一貫性を向上させるために、複数のアクション ファイルにわたって用語を改訂し、ワークフロー自動化[アクション カタログ](/docs/workflow-automation/setup-and-configuration/actions-catalog/actions-catalog)を更新しました。

### マイナーチェンジ

* [メトリクス正規化ルール](/docs/new-relic-solutions/new-relic-one/ui-data/metric-normalization-rules)にマルチアプリケーションのサポートが追加されました。
* [Javaエージェント設定](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file)に`sql_metadata_comments`設定オプションを追加しました。
* 再編成された AWS EC2 コンテンツで[AWS Lambda アクションを](/docs/workflow-automation/setup-and-configuration/actions-catalog/aws/aws-lambda)更新しました。
* 用語の修正を加えて、 [Pipeline Control Gateway UI ガイド](/docs/new-relic-control/pipeline-control/gateway/ui-guide)を更新しました。
* [Nginx OTel Kubernetesインテグレーション](/docs/opentelemetry/nginx/nginx-otel-kubernetes)を更新し、設定を修正しました。
* 新しい設定オプションを追加して[Rubyエージェント設定](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration)を更新しました。
* リンクを修正して[Amazon ECS 統合](/docs/infrastructure/elastic-container-service-integration/introduction-amazon-ecs-integration)ドキュメントを更新しました。
* [Javaエージェントの互換性ドキュメント](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent)を v9.1.0 用に更新しました。
* 最新のフレームワーク サポート情報に基づいて[.NET エージェントの互換性に関するドキュメント](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements)を更新しました。
* [外形監視ステップ モニターのドキュメント](/docs/apis/nerdgraph/examples/synthetics-api/step-monitor)を更新し、サポートされているブラウザーから EDGE を削除しました。
* [Browser setUserId API](/docs/browser/new-relic-browser/browser-apis/setuserid)修正された構文で更新しました。
* モバイル アプリ ユーザー向けの MFA セットアップ手順を含む[認証ドメインと SAML SSO ドキュメントが](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more)更新されました。
* [ダッシュボードとグラフのドキュメントを共有するための前提条件](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts)が更新されました。
* [NerdGraph ドキュメントを使用してドロップ データ](/docs/data-apis/manage-data/drop-data-using-nerdgraph)を更新しました。
* パスワードと電子メールの問題に関する[ログインのトラブルシューティング ドキュメント](/docs/accounts/accounts-billing/account-setup/troubleshoot-new-relics-password-email-address-login-problems)を更新しました。
