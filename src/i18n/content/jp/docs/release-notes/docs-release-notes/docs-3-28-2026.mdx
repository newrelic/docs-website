### 主な変更点

* [Microsoft SQL Server 統合](/install/microsoft-sql)ドキュメントを更新して、次の認証方法のパフォーマンス監視 (QPM) を追加しました。

  * Windowsドメイン
  * Windowsグループ管理サービスアカウント（gMSA）

* [Elasticsearch監視をOpenTelemetryドキュメントで](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview)わかりやすく更新し、包括的なKubernetesインストレーション ドキュメントを追加しました。

* [NerdGraph APIを使用してDashboards](/docs/apis/nerdgraph/examples/nerdgraph-dashboards)更新し、テンプレート変数の表示機能を追加しました。

* [ワークフロー自動化定義スキーマ](/docs/workflow-automation/workflow-automation-apis/definition-schema)を更新しました。スキーマの大幅な更新と包括的な例が含まれています。

### マイナーチェンジ

* CCU使用量予測機能と強化されたダッシュボード機能を備えた、最新の[消費管理機能](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/consumption-management)。
* [View ping モニターの結果](/docs/synthetics/synthetic-monitoring/using-monitors/view-ping-monitor-results)を更新し、ガイダンスと例を改善しました。
* Browserの属性定義と説明を追加して属性辞書を更新しました。
