### 主な変更点

* [新しいサポート マトリックス](/docs/ebpf/requirements)、 [eBPF エージェント Linux インスト](/docs/ebpf/linux-installation)レーション、および[Kubernetesインストレーション](/docs/ebpf/k8s-installation)でクライアント削除のための設定変更を追加して、eBPF ドキュメントを更新しました。
* API アクセス制御手順を改善するために、[シークレット管理のきめ細かなアクセス制御ドキュメント](/docs/apis/nerdgraph/examples/nerdgraph-api-secret-management-service)を更新しました。
* 10.x バージョンの変更に伴い、 [Ruby エージェントの要件とサポートされるフレームワーク](/docs/apm/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks)が更新されました。
* Nuxt 3 以降のサポートと改善されたフォーマットを含む[Nuxt.jsインテグレーション ドキュメント](/docs/browser/browser-integrations/nuxt-integration/)を追加しました。

### マイナーチェンジ

* 例外インスタンスのサポートを含む[Python エージェントの notice\_error API](/docs/apm/agents/python-agent/python-agent-api/noticeerror-python-agent-api)ドキュメントが強化されました。
* [ブラウザ ログのスタート ガイドが更新され、ログ記録のドキュメント](/docs/browser/browser-monitoring/browser-pro-features/browser-logs/get-started/)が改善されました。
* [Web ページのライフサイクル探索](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle)のためのセッショントレースのドキュメントを修正しました。
* 最新の互換性レポートに従って[Node.js エージェントの互換性要件](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent)を更新しました。
* [NGINX OpenTelemetry設定](/docs/opentelemetry/nginx/nginx-otel-host)を更新し、stub\_status 設定を改善しました。

### 新機能

* [読み取り専用ユーザーのダッシュボード操作](/whats-new/2026/01/whats-new-01-19-read-only-dashboards)- ダッシュボード管理の標準読み取り専用ロール権限は、2026 年 4 月 30 日より表示のみのアクセスに制限されます。
