### 新しいドキュメント

* GrafanaダッシュボードをNew Relicダッシュボードに変換するための[GrafanaダッシュボードトランスレータREST API](/docs/infrastructure/prometheus-integrations/view-query-data/grafana-to-nr-dashboard-translation-rest-api/)を追加しました。
* Javaエージェントを使用したサーバーレス監視オプションを記載した[AWS Lambda向けJavaエージェントのアプローチ](/docs/apm/agents/java-agent/getting-started/java-agent-approaches-lambda/)を追加しました。

### 主な変更点

* Android、iOS、Browser、Roku向けの新しいプラットフォーム固有のインストレーションガイドに加え、更新されたAPIリファレンスドキュメントと改善されたナビゲーションを追加して、[Streaming Video &amp; Adsのドキュメント](/docs/streaming-video-&-ads/introduction-to-streaming-video-&-ads/)を再構築しました。
* [データ取り込み予算のドキュメント](/docs/data-apis/manage-data/data-ingest-budgets/overview/)を、マルチテナント対応のアカウント レベルおよび組織レベルの予算用の個別のページに再構成しました。

### マイナーチェンジ

* UIの変更に合わせてメトリクス名を改訂し、[Kubernetesクラスタエクスプローラーのドキュメント](/docs/kubernetes-pixie/kubernetes-integration/understand-use-data/kubernetes-cross-cluster/)を更新しました。
* [アカウントのメール設定](/docs/accounts/accounts/account-maintenance/account-email-settings/)を更新し、再構成したSSO/SAMLのメール変更手順を適切なサブセクションとして追加しました。
* ドキュメントを単一のディストリビューションモデルに簡素化するために、[NRDOTコレクターのドキュメント](/docs/opentelemetry/nrdot/nrdot-collector/)を更新しました。
* [ワークフロー自動化の概要](/docs/workflow-automation/introduction-to-workflow/)を更新し、RBAC権限情報を追加しました。
* 更新されたパッケージバージョンで[Node.jsエージェントの互換性要件](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)を更新しました。
* [Javaエージェントの互換性要件](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent/)を最新の検証済み互換バージョンで更新しました。
* [サポート対象のPromQL機能](/docs/infrastructure/prometheus-integrations/view-query-data/supported-promql-features/)を更新し、Grafanaの翻訳サポートに関する追加情報を記載しました。
* [Errors Inboxブラウザタブ](/docs/errors-inbox/browser-tab/)を更新し、`SecurityPolicyViolation`イベント処理セクションを追加しました。
* ANSI SQLでは`SELECT`の前に`FROM`句を使用できないことを明確にするために、[New Relic Lensクエリデータ](/docs/new-relic-lens/query-data/)を更新しました。
* [Confluent Cloud インテグレーション](/docs/message-queues-streaming/installation/confluent-cloud-integration/)を更新し、コンピュートプール（Flink）データテーブルを追加しました。
* データベース設定の更新に伴い、[NRDOT MSSQL](/docs/opentelemetry/database/otel-mssql/)および[Oracle DB](/docs/opentelemetry/database/otel-oracledb/)のドキュメントを更新しました。
* キーローテーションの更新を反映して、「[PHPエージェントインストレーション RPMキーローテーション](/docs/apm/agents/php-agent/installation/php-agent-installation-rpm-key-rotation/)」を更新しました。
* 追加のインストレーションガイダンスを含むように、[AWS Linux/RedHat/CentOS向けPHPエージェントのインストレーション](/docs/apm/agents/php-agent/installation/php-agent-installation-aws-linux-redhat-centos/)を更新しました。
* 箇条書きのフォーマットを修正し、[Kubernetesインテグレーションの互換性要件](/docs/kubernetes-pixie/kubernetes-integration/get-started/kubernetes-integration-compatibility-requirements/)を更新しました。
* QoEインデックス画像とコールアウトを追加して、[Streaming Video &amp; Adsアプリケーションビュー](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/application/)を更新しました。
