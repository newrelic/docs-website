### 新しいドキュメント

* AndroidおよびiOS向けのセットアップ、プライバシー設定、高度な機能、リプレイの表示、トラブルシューティングを網羅した包括的なガイドを含む、一般提供を開始した[モバイルセッションリプレイ](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/introduction/)のドキュメントスイートを追加しました。
* スクリプト化されたAPIモニターをNode.js 16からNode.js 22へ移行するための段階的な手順を提供するために、[スクリプト化されたAPIモニターをNode.js 22にアップグレードする](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-api-monitors-to-node-22/)を追加しました。

### 主な変更点

* 一般提供向けにCloud Cost Intelligenceのドキュメントを更新し、新しい[CCIの料金](/docs/cci/cci-pricing/)、[Cloud Cost Intelligenceのアラート条件](/docs/cci/configure-cci/cci-alerts/)、[AIコスト](/docs/cci/financial-intelligence/ai-costs/)ページを追加したほか、インテグレーション設定とFAQコンテンツを拡充しました。
* ナビゲーションとわかりやすさを向上させるため、[New Relic Teams](/docs/service-architecture-intelligence/teams/teams/)のドキュメントを再構成し、新しい[エンティティの所有権の構成](/docs/service-architecture-intelligence/teams/team-ownership/)ガイドを追加するとともに、[Teamsの管理](/docs/service-architecture-intelligence/teams/manage-teams/)コンテンツを更新しました。
* YAML設定の移行ガイダンスとv1.4.0で導入されたフィールドの変更を含む、[Kubernetes](/docs/ebpf/k8s-installation/)および[Linux](/docs/ebpf/linux-installation/)のインストレーション向けのeBPFエージェントのドキュメントを更新しました。
* ダッシュボードのガイダンスの拡充、新しいスクリーンショット、および改善された最適化に関するレコメンデーションを追加して、[計算使用量の最適化](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-optimizer/)を更新しました。
* [OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/)を更新し、OpenTelemetry APMの体験におけるエラー監視に関する新しいセクションを追加しました。

### マイナーチェンジ

* 最新の自動生成された互換性データで[Node.jsエージェントの互換性と要件](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)を更新しました。
* [AWS Lambdaの環境変数](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/env-variables-lambda/)に新しい環境変数`NEW_RELIC_EXTENSION_SEND_PLATFORM_LOGS`、`NEW_RELIC_EXTENSION_SEND_LOGS`、および`NEW_RELIC_LAMBDA_EXTENSION_PROXY`を追加しました。
* 2つの異なるサーバーレス計装アプローチをより明確に分離し、Java Lambda用の[階層化計装](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/instrument-your-own/)を更新しました。
* [OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/)のテーブルヘッダーのずれを修正しました。
* [Google Cloud PlatformサービスをNew Relicに接続する](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)に`iam.managed.allowedPolicyMembers`IAM権限のガイダンスを追加しました。
* [New Relic AIナレッジコネクター](/docs/agentic-ai/knowledge-integration/overview/)のあいまいなアカウント参照を修正しました。
* コンテンツを再編成し、わかりやすさを向上させて、[外形監視プライベートロケーションランタイム移行ガイド](/docs/synthetics/synthetic-monitoring/private-locations/runtime-transition-guide/)を更新しました。
* [Javaエージェント9.2.0リリースノート](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-920/)のリンク切れを修正しました。
* [NerdGraphチュートリアル: データをAWS Kinesis Firehose、Azure Event Hub、またはGCP Pub/Subにストリーミングする](/docs/apis/nerdgraph/examples/nerdgraph-streaming-export/)の誤ったAWSアカウントIDの例を修正しました。
* [New Relicのデータプライバシー](/docs/security/security-privacy/data-privacy/data-privacy-new-relic/)に画面共有と録画の同意に関する文言を追加しました。
* [NRQLリファレンス](/docs/nrql/nrql-syntax-clauses-functions/)のNRQL句の大文字表記を修正しました。
* New Relic開発者ドキュメントページ内の廃止されたone.nr URLをonenr.ioに変換しました。
