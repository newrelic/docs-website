## 更新

* ビデオ イベントに`contentBitrate`属性を追加しました。利用可能な場合は`segmentationBitrate`を使用し、必要に応じて`streamBitrate`にフォールバックします。
* 広告イベントに`adBitrate`属性を追加しました。広告ビットレートは複数の方法を使用して抽出され、bps (ビット/秒) に標準化されます。
* `timeSinceLastError`プロパティをエラー イベントに追加し、前回のエラー イベントからの経過時間をレポートします。
