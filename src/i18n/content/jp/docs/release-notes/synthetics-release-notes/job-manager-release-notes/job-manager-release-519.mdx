### 修正

* SJM の基盤となる OS および Java の脆弱性を修正しました。

* CVE-2026-33871
* コンテナ化されたプライベートロケーション向けに、デュアルスタック（IPv4とIPv6）のサポートを追加しました。外形監視 Job Manager は、完全な IPv4 互換性を維持しながら、IPv6 環境でも動作するようになりました。 サポートされているコンテナプラットフォームには、Docker、Kubernetes、Podmanが含まれます。
