### 改良点

* Chrome ブラウザをバージョン 143.0.7499.146 に更新しました。ユーザーは、ブラウザ ログに「ソフトウェア WebGL への自動フォールバックは非推奨になりました」という非推奨の警告が表示される場合があります。この変更により、重大度の高いセキュリティ脆弱性 ( [CVE-2025-14765](https://chromereleases.googleblog.com/2025/12/stable-channel-update-for-desktop_16.html) ) に対処するために、自動フォールバック機能が削除されます。
