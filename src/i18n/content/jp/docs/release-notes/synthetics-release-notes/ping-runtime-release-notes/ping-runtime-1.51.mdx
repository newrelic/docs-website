### 改良点

* ping ランタイムの logback-core の脆弱性を修正しました。
* ping ランタイムの jetty サーバーの脆弱性を修正しました。
* ping ランタイムの Ubuntu の脆弱性を修正しました。

### 提案

* 最新の Ping ランタイム バージョン (現時点では 1.51.0) を使用している場合は、環境のセキュリティと安定性を確保するために、外形監視 Job Manager をバージョン release-443 以降にアップグレードすることをお勧めします。
