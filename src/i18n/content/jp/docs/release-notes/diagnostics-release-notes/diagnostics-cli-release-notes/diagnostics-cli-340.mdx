## タスクの更新

* Agent Controlのネームスペースを指定するための新しいコマンドライン オプション`-ac-agents-namespace`が追加されました ( [#243](https://github.com/newrelic/newrelic-diagnostics-cli/pull/243) )。 次のタスクは、新しいネームスペースを使用するように更新されました。

  * K8s/リソース/構成
  * K8s/リソース/デーモンセット
  * K8s/リソース/デプロイ
  * K8s/リソース/ポッド

## 修正

* `/lib64/libc.so.6: version 'GLIBC_2.34' not found`などの GLIBC 関連エラーが解決されました ( [#244](https://github.com/newrelic/newrelic-diagnostics-cli/pull/244) )。
