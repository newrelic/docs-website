## タスクの更新

* Infra/エージェント/Version - バージョンを確認するための Github への接続に問題がある場合に、エラーではなく警告を返すように更新されました ( [#231](https://github.com/newrelic/newrelic-diagnostics-cli/pull/231) )

## 新機能

* KubernetesとAgent Controlに次のタスクを追加しました ( [#235](https://github.com/newrelic/newrelic-diagnostics-cli/pull/235) 、 [#237](https://github.com/newrelic/newrelic-diagnostics-cli/pull/237) 、 [#238](https://github.com/newrelic/newrelic-diagnostics-cli/pull/238) )

  * K8s/AgentControl/helm-controller - Agent Control helm-controller ポッド ログを収集します
  * K8s/AgentControl/source-controller - Agent Controlソースコントローラーの Pod ログを収集します
  * K8s/AgentControl/エージェント-control - Agent Agent Control -control ポッドログを収集します
  * K8s/Flux/Charts - Flux Helm Charts情報を収集します
  * K8s/Flux/リポジトリ - Flux Helmリポジトリ情報を収集します。
  * K8s/Flux/Releases - Flux Helm リリース情報を収集します
  * K8s/Resources/Config - 指定されたネームスペースの K8s configMap を YAML 形式で収集します
  * K8s/Resources/デプロイ - 指定されたネームスペースの K8s デプロイメント情報を YAML 形式で収集します
  * K8s/Resources/podd - 指定されたネームスペースの K8s ポッド情報を YAML 形式で収集します
  * K8s/Resources/Daemonset - 指定されたネームスペースの K8s デーモンセット情報を YAML 形式で収集します
  * K8s/Env/Version - kubectlクライアントとクラスタのバージョンを取得します
  * K8s/Helm/Releases - Helmリリースのリストを収集します
