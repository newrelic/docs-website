### メモ

[New Relic Radar](https://docs.newrelic.com/docs/radar/new-relic-radar/getting-started/introduction-radar)からの集中イベント コンテキストのサポート。これは、集中イベント中に迅速に方向転換するためのツールです。 まとめイベント コンテキストは、システム内の異常な動作を検出し、まとめ情報と一緒に配置することで、まとめイベント中に迅速な方向性を提供します。

### 新機能

* レーダー集中イベントコンテキスト

### 改良点

* インフラストラクチャリストの呼び出しを改善し、レイテンシとネットワーク使用量を削減

### 修正

* 詳細ビューで戻るボタンを使用するとクラッシュする
