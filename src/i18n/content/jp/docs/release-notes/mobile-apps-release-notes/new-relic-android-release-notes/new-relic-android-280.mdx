### メモ

バージョン2.8.0の素晴らしい新機能

### 新機能

* アプリケーション一括イベントに概要 KPI (主要パフォーマンス指標) と KPI チャートを追加しました。
* APM詳細ページのスコープを特定のホストまたはJVMインスタンスに設定する機能が追加されました
* 新しいアクティビティインジケーターを追加しました

### 改良点

* スパークチャートを削除する
* まとめイベントからの間隔を含む新しい時間枠
* まとめイベントからのナビゲーションフローの改善

### 修正

* プッシュ通知での多発イベントの承認を修正
