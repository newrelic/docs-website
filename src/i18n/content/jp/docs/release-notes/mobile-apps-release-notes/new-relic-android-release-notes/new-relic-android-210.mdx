### 新機能

* ユニバーサル リンク - メール、チャットなどで newrelic.com リンクをクリックすると、そのリンクがアプリで開くようになりました。

### 改良点

* バージョン2.1.0セッション トークンを更新する方法が改善され、401 および 403 エラーが減少します。
* モバイルクラッシュリスト画面で時間枠が利用可能になりました

### 修正

* モバイル クラッシュを解決または再開するためのモバイル クラッシュ解決 / 再開ボタン
* イベント一括承認ボタン
* まとめてイベント通知からのお知らせ
