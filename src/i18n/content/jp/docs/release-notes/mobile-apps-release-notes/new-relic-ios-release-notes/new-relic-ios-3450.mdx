### メモ

このバージョンには強化された通知が含まれており、さまざまな種類の通知に固有のサウンドを選択できるようになったほか、一般イベントのチャート画像も表示されます。

### 新機能

* アプリへのプッシュ通知の通知音を選択します。
* 通知内に集中イベントのオープン/クローズチャート画像を表示します。

### バグ修正

* インフラストラクチャで apmApplicationIds と apmApplicationNames を適切にフィルタリングするための修正。
* プラグイン コンポーネントを検索しようとしたときに発生するクラッシュを修正しました。
* レーダーのクラッシュを修正しました。
