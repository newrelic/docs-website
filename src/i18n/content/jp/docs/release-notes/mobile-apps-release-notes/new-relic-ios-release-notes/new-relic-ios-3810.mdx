### メモ

APM および OpenTelemetry サービス向けのコンテキストログの導入

### 新機能

* APM / OpenTelemetryサービスの詳細ビューに新しいログ メニュー項目が追加され、そのサービスに限定された過去数分以内のログ (設定されている場合) を表示できるようになりました。

### 改良点

* ウォッチアプリがお気に入りを読み込めない問題を修正しました
* APM集計イベントのKPIを表示しようとするとクラッシュする問題を修正しました
