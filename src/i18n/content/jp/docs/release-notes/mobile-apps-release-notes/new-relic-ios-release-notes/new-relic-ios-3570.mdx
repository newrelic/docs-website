### メモ

[iPhone アプリ](https://docs.newrelic.com/docs/mobile-apps/new-relic-mobile-apps/ios-app/introduction-ios-mobile-app)内からの NRQL 編集を導入します。

### 新機能

* NRQL クエリ履歴を iPhone に追加します (iPad は後で追加されます)。
* NRQL クエリを記述する機能を追加します。
* アプリ内NRQLエディターを表示するためのNRQL書き込みを目的としたリンク集中イベント。
* iPad には、 APM 、 Browser 、モバイル、およびキーの詳細画面が含まれています。
