### メモ

すべてのエンティティのエクスプローラーを備えたグローバル タブ バーが機能するようになりました。

### 新機能

* インフラストラクチャ統合をサポートするエンティティエクスプローラーページに、他の多くのタイプのエンティティとともに追加されました
* グローバルタブバーに追加され、どこからでもホーム、エクスプローラー、ダッシュボード、アラート、クエリに素早くアクセスできるようになりました。
* 統合レポートサービスのサポートを追加しました
* EUアカウント向けの多発イベントに異常サポートを追加

### 修正

* カスタムタイムピッカーを更新して、開始と終了の間に5分の間隔を設定できるようにします
* フィードバックフォームの問題を修正
