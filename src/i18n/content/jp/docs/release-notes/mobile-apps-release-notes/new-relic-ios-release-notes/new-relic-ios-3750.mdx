### メモ

合成モニターの詳細が改善され、ダッシュボードウィジェットからクエリを表示する機能が追加されました。

### 新機能

* ダッシュボードウィジェットに**View query** \[ビュークエリ]オプションを追加します
* まとめイベントからいつでもNRQL表示/実行できる機能を追加します
* アプリ内から合成モニターを実行する機能を追加します
* 合成モニターに結果リストを追加します
* ほとんどのサマリー画面でプルして更新する機能を追加します
