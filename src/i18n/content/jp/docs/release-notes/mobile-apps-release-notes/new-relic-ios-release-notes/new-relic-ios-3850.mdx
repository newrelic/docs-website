### メモ

AI集中イベントの詳細に分析タブが追加されました!

### 新機能

* AI集中イベント詳細に分析セクションを追加
* 負の数と非常に小さい数値のチャートサポートを追加しました
* ダッシュボードで設定されたテーブルにタイムスタンプを表示するためのサポートが追加されました
* アカウントリストに検索を追加しました

### 改良点

* ダイナミックフォントのサポートが向上
* チャートのツールチップを表示してから凡例項目を選択すると発生する可能性のあるクラッシュを修正しました。
* エンティティのカテゴリが少ないユーザー向けのエンティティ エクスプローラーのUI問題を修正しました
