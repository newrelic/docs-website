### 新機能

* アプリケーション一括イベントに概要 KPI (主要パフォーマンス指標) と KPI チャートを追加しました。
* プッシュ通知にまとめイベントの承認を追加しました
* アプリケーション ページを特定のホストまたは JVM インスタンスにスコープする機能が追加されました
* 非ウェブ瞬間チャートをアプリケーションに追加
* **iPad** : ブラウザ製品に追加

### 改良点

* 新しいアクティビティインジケーターを追加しました
* アプリ全体の検索を改良
* **iPhone** : 詳細ビューですべてのチャートを展開
* **iPhone** : 「途中」のピボット「go to」リンクを変更し、「戻る」ボタンで「保留」リストに移動できるようにしました。

### バグ修正

* インフラストラクチャ製品へのユニバーサルリンクを開くときに発生するクラッシュを修正しました
