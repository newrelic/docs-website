### メモ

* New Relic Radar による集中イベント コンテキストのサポート -- 集中イベント中の迅速な方向確認のためのツール。
* まとめイベント コンテキストは、システム内の異常な動作を検出し、まとめ情報と一緒に配置することで、まとめイベント中に迅速な方向性を提供します。

### 改良点

* ログインフローの改善
* エラー トレース属性を検索するときの動作が改善されました

### バグ修正

* ネットワーク障害やサーバーの問題によりログアウトされる問題を修正しました。ログアウトされる代わりに再試行できるようになりました。
