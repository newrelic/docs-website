### メモ

ユーザーが適切なアクセス制御を持っている場合、手動でイベントを閉じる機能が追加されました。

### 新機能

* iPhone と iPad の両方に新しい**Close alert event** \[閉じる集中イベント]ボタンが追加され、集中イベントとその集中イベントに関連するすべてのイベントを手動で閉じることができます。
