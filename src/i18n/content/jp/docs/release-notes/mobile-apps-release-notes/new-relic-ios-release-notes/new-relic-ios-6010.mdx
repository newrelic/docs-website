バグ修正とUIの更新

### 新機能

* トップレベルビューの大部分と APM ビューのほとんどを SwiftUI に変換します。
* 大量イベント異常チャートを追加します。
* DotNet VM の詳細ページを追加します。

### 改良点

* 時間ウィンドウを尊重しないラムダ呼び出しとエラーを修正します。
* いくつかのビルボードのフォーマットの問題を修正しました。
* APM エンティティを表示するときにクロスアカウント ログを読み込む問題を修正しました。
