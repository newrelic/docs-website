## 3.44.1

### 修理済み

* 以前のリリースの誤ったタグを修正

### セキュリティ

* Dependabotセキュリティアップデート

  * `nrgochi` インテグレーションをv5.2.4に更新
  * `nrmongo` インテグレーションをv1.17.7に更新
  * `nrmongo-v2` インテグレーションをv2.4.2に更新
  * `nrnats` インテグレーションをv2.11.15に更新

### サポートステートメント

最新バージョンのGo言語を使用しています。少なくとも、Goチームが引き続きサポートしているバージョンのGoを使用してください。Goエージェントおよびサードパーティコンポーネントのサポートされているバージョンの詳細については、[GoエージェントのEOLポリシー](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy)を参照してください。
