## メモ

* `mysql2/promise`の計装を追加しました。

  * これは以前はスタンドアロンの`@newrelic/mysql`にのみ存在していましたが、現在は 2 つの間に機能パーティを提供します。

* 未使用のネイティブ CPU メトリクス サンプラーを削除しました。 Node.js バージョン 6.1.0 以降を実行している場合、このロジックは変更されなくなりました。

* プロミス インターセプターがエラーを再度スローするのを修正しました。

* [ディスカッション フォーラムの投稿](https://support.newrelic.com/s/hubtopic/aAX8W0000008ZfxWAE/relic-solution-the-philosophy-of-naming-your-node-agent-transactions)から移植されたトランザクション命名ドキュメントを追加しました。

* mysql2 バージョンテストに`promises.tap.js`を追加しました。

* @newrelic/test-utilities を最新に更新しました。

* restify バージョン管理テスト内の未使用のテスト ファイルを削除しました。

* テスト ファイルが含まれていない場合に CI 実行が適切に失敗するように、バージョン管理されたテスト ランナーに`--strict`フラグを追加しました。
