### Pipeline Controlゲートウェイ リリースノート - v1.2.0

#### OpenTelemetry Collector を 0.109.0 から 0.131.0 にアップグレード

内部の OpenTelemetry Collector をバージョン 0.109.0 から 0.131.0 にアップグレードしました。この重要なアップデートには、多数の重要な改善とバグ修正が含まれており、より堅牢で安定したプラットフォームが提供されます。変更の完全なリストについては、 [OpenTelemetry Collector リリース ノート](https://github.com/open-telemetry/opentelemetry-collector/releases)を参照してください。
