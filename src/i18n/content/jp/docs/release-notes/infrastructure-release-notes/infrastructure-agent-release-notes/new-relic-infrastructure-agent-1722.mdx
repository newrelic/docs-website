エージェントの新しいバージョンがリリースされました。標準手順に従って[インフラストラクチャエージェントを更新します](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent)。 New Relic では、エージェントを定期的に、少なくとも 3 か月ごとにアップグレードすることを推奨しています。このリリースの時点で、サポートされている最も古いバージョンは[インフラストラクチャエージェント 1.58.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1581/)です。

## かわった

* [#2177](https://github.com/newrelic/infrastructure-agent/pull/2177)で依存関係 newrelic/nri-prometheus を v2.27.6 に更新しました
* [#2180](https://github.com/newrelic/infrastructure-agent/pull/2180)で依存関係 newrelic/nri-flex を v1.17.3 に更新しました
* [#2181](https://github.com/newrelic/infrastructure-agent/pull/2181)で依存関係 newrelic/nri-winservices を v1.4.2 に更新しました
* [#2182](https://github.com/newrelic/infrastructure-agent/pull/2182)で依存関係 newrelic/nri-docker を v2.6.7 に更新しました
* [#2183](https://github.com/newrelic/infrastructure-agent/pull/2183)で go バージョンを 1.25.6 に更新しました

## セキュリティ

* 重大な脆弱性 CVE-2025-61726 および CVE-2025-61728 に対処しました
