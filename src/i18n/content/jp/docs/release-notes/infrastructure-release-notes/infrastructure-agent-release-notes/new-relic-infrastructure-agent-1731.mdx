エージェントの新しいバージョンがリリースされました。標準的な手順に従って、[infrastructureエージェントをアップデート](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent)してください。New Relicでは、エージェントを定期的に、少なくとも3か月ごとにアップグレードすることを推奨します。このリリースより、サポートされる最も古いバージョンは[Infrastructureエージェント 1.64.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1640/)です。

### かわった

* 依存関係nri-dockerを2.6.10に更新しました、[#2220](https://github.com/newrelic/infrastructure-agent/pull/2220) で依存関係 nri-flex を v1.17.7 に、依存関係 docker を v29 に
* [#2202](https://github.com/newrelic/infrastructure-agent/pull/2202)でWindows向けinfrastructureエージェントのDockerイメージをビルドしました
* [#2213](https://github.com/newrelic/infrastructure-agent/pull/2213) でWindowsイメージリリースワークフローを更新
