## 変更点

### ドキュメント

* **READMEの刷新:** 包括的なインストレーションガイド、使用例、ベストプラクティス、設定オプション、APIリファレンス、およびサポートチャネルを含めて、READMEを再構築および拡張しました。

### 改良点

* **属性の検証:** トラッカーレベルで正確性を強制するために、カスタムアトリビュートの検証ロジックを更新しました。
* **カスタムアトリビュートの制限:** イベントごとに送信できるカスタムアトリビュートの数に強制的な制限を追加し、予期しないペイロードサイズを防ぎます。
