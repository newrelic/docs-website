## 機能

### スタンドアロンエージェント

* **ブラウザBrowserからの分離**: ビデオエージェントは、独自の**収集ロジック**を使用して独立して動作するようになりました。 この分離によりオーバーヘッドが削減され、Browseragent のデータ収集スケジュールに縛られることなくビデオ データが収集され、 New Relicに送信されるようになります。
