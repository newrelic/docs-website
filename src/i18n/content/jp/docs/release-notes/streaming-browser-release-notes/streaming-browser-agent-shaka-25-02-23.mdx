## 新機能と機能強化

### 新しいイベントタイプが導入されました

* **VideoAction** : このイベント タイプは、ビデオ再生およびインタラクションに関連する標準アクションを表します。
* **VideoErrorAction** : このイベント タイプはビデオ再生中に発生するエラーをキャプチャし、トラブルシューティングのための詳細なインサイトを提供します。
* **VideoAdAction** : ビデオ広告に関連するアクションを追跡し、より詳細なレポートを作成するための専用のイベント タイプです。
* **VideoCustomAction** : ビデオ アプリケーション内でのカスタム アクションとインタラクションのための柔軟なイベント タイプ。

## 廃止予定

* **PageAction**
  * `PageAction`イベント タイプは非推奨になりました。ユーザーは、強化された追跡機能と詳細性を提供する新しいイベント タイプの使用に移行する必要があります。

## 新しい属性が導入されました

データの追跡と分析を強化するために、一連の新しい属性を導入しました。これらの属性は、新しいイベント タイプに関する追加のコンテキストと詳細を提供します。詳細については、[データ モデルのドキュメント](https://github.com/newrelic/video-shaka-js/blob/v3.0.0/DATAMODEL.md)を参照してください。

## 影響と必要な行動

* **PageAction からの移行**: 現在`PageAction`イベントを追跡しているユーザーは、新しく導入されたイベント タイプを利用するように実装を調整する必要があります。 この変更により、ビデオインタラクション全体にわたるアクションの追跡の特異性と範囲が向上します。

* **実装の更新**: インテグレーターは、イベント追跡のためのより豊富なコンテキストと詳細を提供する、リリースで導入された新しい属性を活用するためにシステムを更新する必要があります。

* **テストと検証**: 互換性と正しいデータ転送を確保するために、新しい属性とイベント タイプの実装をテストすることをお勧めします。
