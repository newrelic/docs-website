## 変更ログ

このリリースでは[、nrdot-コレクター](https://github.com/newrelic/nrdot-collector-releases/tree/main/distributions/nrdot-collector)ディストリビューションが導入されています。 このディストリビューションは現在アルファ版ですが、最終的には`nrdort-collector-host`ディストリビューションの代わりとなる予定です。将来のディストリビューションでは、 `nrdot-collector`基本マニフェストとして使用し、必要に応じて拡張します。

### 特徴

* 機能: ホテルコンポーネントのバージョンを v0.128.0 から v0.131.0 にアップグレード (#357)
* 特技: nrdot コアディストリビューションの導入 (#359)
