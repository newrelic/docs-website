### 特徴

* ホストのユースケースが`nrdot-collector:1.17.0`で正しく機能することを検証します。
* 非推奨となったため、`hostmetrics`を`host_metrics`に、`resourcedetection`を`resource_detection`に、`metricstransform`を`metrics_transform`に置き換えました。`hostmetricsreceiver`、`resourcedetectionprocessor`、および`metricstransformprocessor`は、`metadata.yaml :: type`識別子に`snake_case`を使用するように変更したため、設定での参照方法が変更されました。
