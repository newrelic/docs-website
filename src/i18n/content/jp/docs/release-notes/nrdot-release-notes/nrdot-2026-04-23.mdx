## 変更ログ

このリリースでは、`nrdot-collector-k8s`および`nrdot-collector-host`ディストリビューションが削除されました。それらのdistrosの最後の公開バージョンは`1.12.0`です。代わりに`nrdot-collector`を使用するように移行してください。これは機能面でスーパーセットであり、ドロップインの代替として機能します。削除された2つのディストリビューションは[ユースケース](https://github.com/newrelic/nrdot-collector-releases/blob/1.13.0/distributions/nrdot-collector/README.md#use-cases)として実装されています。
