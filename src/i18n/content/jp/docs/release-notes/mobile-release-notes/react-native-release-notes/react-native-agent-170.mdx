## 改良点

* ネイティブ Android エージェントをバージョン 7.7.0 に更新しました。
* インテリジェント セッション リプレイ: データ キャプチャを最適化するために、ERROR モード (クラッシュによってトリガーされる 15 ～ 30 秒のバッファリングされた録画) と FULL モード (連続録画) が導入されました。
* セッション制御API: NewRelic.recordReplay()を追加しましたおよび NewRelic.pauseReplay()リプレイ録画状態を手動でプログラム的に制御できるようにします。
* ネットワーク バグ修正: peekBody() により、コンテンツの長さが不明なストリーミング応答 (SSE/チャンク) が無期限にブロックされる OkHttp3 の問題を修正しました。
