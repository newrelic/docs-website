## 重要

* このリリースでは、New Relic iOS エージェントの必要な iOS バージョンが iOS 16 に上がりました。iOS 15 のサポートは将来のバージョンで再開される予定です。

## このリリースの新機能

* モバイル セッション リプレイのパブリック プレビューを更新して、エラーが発生したセッションのサンプリングとセッション リプレイに関連する問題の修正を追加しました。

## このリリースで修正された点

* 特定のエッジケースでディストリビューティッド（分散）トレーシングが機能しない問題を修正しました。
* 特定された問題に対する安定性の修正を追加しました。
