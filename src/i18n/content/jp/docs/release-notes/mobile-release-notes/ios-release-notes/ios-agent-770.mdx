### このリリースの新機能

* **ブロックされたビューのセッションリプレイ:**新しいブロックされたビュー機能が追加されました。 UIKit の場合は、「nr-block」アクセシビリティ ID、新しい UIView `blockView`使用するか、SwiftUI `NRConditionalMaskView`で`blockView` =true を指定して、セッション リプレイ中のキャプチャから領域をブロックします。

* まれに発生する、過剰なログ記録による過熱問題を修正しました。
