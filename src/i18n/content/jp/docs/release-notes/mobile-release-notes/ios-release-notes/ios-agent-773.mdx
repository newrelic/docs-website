### 新機能

モバイルセッションリプレイのオフラインストレージ機能。

### このリリースで修正された点

IGuidGeneratorのタイムスタンプをシードとするLCGをarc4random\_bufに置き換えました。

RNソースマップのアップロードにEUリージョンの検出を追加しました。

MAUIスタイルのデフォルトインタラクションのswizzleに対する再帰をブロックしました。

RNソースマップのアップロードでappVersionIdをappVersionに名前変更しました。

リリース時にエージェントをフリーズさせていたEventDeserializerの無限ループを修正しました。

プログラマティックAPIを使用したアンマスクを許可しないようにデフォルトのセッションリプレイのマスキングモードを変更し、UIKitのオーバーライドAPIを修正しました。

NRMAWKWebViewNavigationDelegateのwebViewWebContentProcessDidTerminateで発生する可能性があったクラッシュを修正しました。

アプリが閉じたときにアップロードに失敗した場合、次回のリリース時にセッションリプレイのデータが確実にアップロードされるようにしました。

永続的に拒否された（400/403）クラッシュレポートを破棄しました。

オフラインストレージにTTLエビクションを追加しました（#786）。

NRMAHarvestControllerのinitializeのクラッシュを修正しました（#793）。
