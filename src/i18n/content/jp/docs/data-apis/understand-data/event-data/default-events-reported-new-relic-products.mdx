New Relic の機能は、さまざまな [タイプのデータ](/docs/using-new-relic/data/understand-data/new-relic-data-types)を報告します。データの 1 つのタイプは [イベント データ](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data)です。イベント データは New Relic チャートやその他のビューに表示され、 [クエリ](/docs/using-new-relic/data/understand-data/query-new-relic-data)にも使用できます。

New Relic によって報告されたイベントの詳細については、こちらをご覧ください。

* [APMのデフォルトイベント](/docs/insights/insights-data-sources/default-data/apm-default-events-insights)
* [ブラウザのデフォルトイベント](/docs/insights/insights-data-sources/default-data/browser-default-events-insights)
* [インフラのデフォルトイベント](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-events)
* [モバイルのデフォルトイベント](/docs/insights/insights-data-sources/default-data/mobile-default-events-insights)
* [合成デフォルトイベント](/docs/insights/insights-data-sources/default-data/synthetics-default-events-insights)
* [NrAuditEventイベント](/docs/insights/event-data-sources/default-events/query-account-audit-logs-nrauditevent/) お客様のアカウントへの変更を理解するために

他のデータについては、 [データ辞書を](/attribute-dictionary)参照してください。
