New Relic 製品は、さまざまな [タイプのデータを報告します](/docs/using-new-relic/data/understand-data/new-relic-data-types) 。報告されるデータのひとつに、 [イベントデータ](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data) があります。イベントは UI チャートやテーブルに表示されるほか、 [クエリとしても利用できます。](/docs/using-new-relic/data/understand-data/query-new-relic-data) 。利用できるデータの種類については、 [NRQL で利用できるデータ](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql#what-you-can-query) を参照してください。

New Relic製品で報告されるイベントの詳細についてはこちらをご覧ください。

* [ブラウザのデフォルトイベント](/docs/insights/insights-data-sources/default-data/browser-default-events-insights)
* [インフラのデフォルトイベント](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-events)
* [モバイルのデフォルトイベント](/docs/insights/insights-data-sources/default-data/mobile-default-events-insights)
