パブリックAPIパフォーマンスダッシュボードは、New Relicアカウントの一部として含まれる、すぐに使えるダッシュボードです。パブリックAPIのパフォーマンスについての実行可能な知見と、新規のお客様が自身のデータに追加する前に、当社のダッシュボード機能を試す機会の両方を提供します。

ダッシュボードでは、一般的なパブリックAPIにアクセスする際にNew Relicのお客様を匿名化してサンプリングすることで生じる実際のレイテンシを表示します。

<Callout variant="important">
  Global Performanceデータセットは、現状のままで提示されます。Global Performanceデータセットは、一連のソース全体でのサンプルの集計を表し、New Relicはデータが正しいことや完全なこと、データの信憑性の確認は行いません。このデータは、データを使用する目的について唯一の情報源としないでください。New Relicは、このデータに基づいて行われた決定について責任を負いません。Global Performanceデータセットは、データセットで表明された技術についてのNew Relicによる保証や推奨とは考えないでください。
</Callout>

## New Relicにダッシュボードを追加する [#Add-dashboard]

Public API Performanceダッシュボードを追加するには：

1. ダッシュボードページから、パブリックAPIパフォーマンスの事前構築済みのダッシュボードを検索します。一般的な事前構築済みダッシュボードのタイトルの横にある、**すべてを表示**リンクをクリックして検索できます。
2. ダッシュボードを選択します。
3. ドロップダウンからアカウントを選択します。
4. **ダッシュボードを表示**をクリックして、アカウントに追加します。

## ダッシュボードを詳しく調べる [#explore-dashboard]

新しい事前構築済みのダッシュボードをカスタマイズできます。名前や設定の変更、コンテンツやウィジェットの新規追加、チャートの絞り込みなどを実行できます。詳細については、[ダッシュボードの管理](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/)を参照してください。

Public API Performanceダッシュボードを詳しく調べる方法についての提案は次のとおりです。

* チャートの右隅にある

  <Icon name="fe-more-horizontal"/>

  アイコンをクリックし、**クエリを表示**を選択し、使用した[NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql)クエリを表示してチャートを作成します。

* チャートの右隅にある

  <Icon name="fe-more-horizontal"/>

  アイコンをクリックし、**画像として取得**を選択して、画像としてチャートを表示またはダウンロードします。

* バーチャートから特定のドメインを選択するか、ページの上部にあるテキストフィールドをクリックしてフィルターを追加します。

* 自身のデータをすでに追加している場合、クエリをコピーし自身の利用に合わせて変更してください。

<Callout variant="important">
  Public API Performanceダッシュボードは現在、EUのお客様にはご利用いただけません。
</Callout>

<Callout variant="important">
  パブリックAPIパフォーマンスダッシュボードは現在、アラートをサポートしていません。
</Callout>

## 報告されるデータ [#data-reported]

`Public_APICall`イベントを通じて、すべてのNew Relic組織に対して、パブリックAPIデータを利用できるようにします。このデータは取り込みにはカウントされません。そのデータの構造を確認するには、次のクエリを実行します。

```sql
FROM Public_APICall select keyset()
```
