[Distributed tracing](/docs/introduction-distributed-tracing) は、New Relic で照会可能な `Span` データを作成します。 `Span` データの詳細を知る方法をご紹介します。

* スパンデータを調べるには、 [クエリビルダー](/docs/chart-builder/use-chart-builder/get-started/introduction-chart-builder) を使用します。
* スパンデータに付けられたデフォルトの属性を確認するには、 [データ辞書](/attribute-dictionary/?event=Span) を使用します。
* これらの属性を使用した NRQL クエリのヘルプについては、以下の [サンプル クエリを参照してください](/docs/apm/distributed-tracing/ui-data/example-insights-queries-distributed-trace-data) 。
