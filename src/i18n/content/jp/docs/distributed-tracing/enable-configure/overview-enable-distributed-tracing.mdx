私たちは、分散型トレースデータを取り込むために、以下のような様々な方法を提供しています。

* [New Relicの統合（APM、AWS Lambda、ブラウザ、モバイルモニタリングを含む）](#new-relic-agents)
* [サードパーティの遠隔測定ツールのためのNew Relicの統合](#third-party-integrations)
* [New Relic Trace API](#trace-api)

## サードパーティの遠隔測定ツールのためのNew Relicの統合 [#third-party-integrations]

これらのテレメトリーツールでデータを収集している場合は、そのデータをNew Relicに送信することができます。

* [OpenTelemetry](/docs/integrations/open-source-telemetry-integrations/opentelemetry/opentelemetry-quick-start/)
* [カモン](/docs/integrations/open-source-telemetry-integrations/open-source-telemetry-integration-list/kamon-reporter)
* [AWS X-Ray](/docs/integrations/amazon-integrations/aws-integrations-list/aws-x-ray-monitoring-integration)

## トレースAPIで独自のソリューションを設定 [#trace-api]

データをエクスポートする統合機能を使わずに、テレメトリツールからNew Relicに直接データを送信することができます。

<Callout variant="tip">
  ただし、統合機能を利用する場合に比べて、手動での設定が必要な場合があります。
</Callout>
