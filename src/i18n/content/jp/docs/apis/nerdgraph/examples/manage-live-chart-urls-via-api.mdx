[NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph) のクエリとミューテーションを使って、一般にアクセス可能なライブチャートのURLをリストアップしたり、取り消したりすることができます。

そのためには

1. [GraphiQL エクスプローラー](https://api.newrelic.com/graphiql) にアクセスしてください。
2. [New Relic アカウント内に作成されたすべてのライブチャート URL](#list) をリストアップします。
3. [好きなライブチャートのURLをリボークする](#revoke) 。

<Callout variant="tip">
  パブリック・ライブ・チャートを作成するには、ドキュメント [を参照してください](/docs/new-relic-one/use-new-relic-one/ui-data/basic-ui-features/#share) 。
</Callout>

## すべてのライブチャートのURLを一覧表示 [#list]

既存のライブチャートのURLの一覧を取得するには、以下のクエリを使用します。

```
{
  actor {
    dashboard {
      liveUrls {
        liveUrls {
          title
          url
          createdAt
          type
        }
        errors {
          description
        }
      }
    }
  }
}
```

## 任意のライブチャートのURLを取り消す [#revoke]

指定したライブチャートのURLを失効させるには、以下のクエリを使用します。

```
mutation {
  dashboardWidgetRevokeLiveUrl(uuid: "<enter-the-UUID-of-the-live-chart>") {
    uuid
    errors {
      description
      type
    }
  }
}
```
