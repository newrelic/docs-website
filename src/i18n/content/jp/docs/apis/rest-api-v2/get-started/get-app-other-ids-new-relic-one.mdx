[New Relic REST API](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2)を使用する場合、一部のIDプレースホルダー（ `$APP_ID` 、 `$KEYTX_ID` 、 `$COMPONENT_ID`など）をアプリ、トランザクション、またはその他のコンポーネントのIDに置き換える必要があります。

New Relic UIを使用して、これらのIDの一部を取得できます。 [RESTAPIExplorer](/docs/apis/rest-api-v2/get-started/list-application-id-host-id-instance-id)を使用してIDのリストを取得することもできます。

## UIを使用してIDを検索する [#ui]

UI には、アプリケーションの ID とその他のコンポーネントがメタデータ パネルに含まれています。エンティティのメタデータを表示するには、エンティティ エクスプローラー リストから監視対象のエンティティをクリックし、タグ アイコンをクリックします。

以下にいくつか例を示します。

<CollapserGroup>
  <Collapser
    id="apm"
    title="APMアプリID"
  >
    New Relic UIから`$APP_ID`を見つけるには：

    1. <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DNT>に移動してアプリケーションを選択してください。

    2. アプリケーション名の横にある <Icon name="fe-tag"/>アイコンをクリックすると、メタデータパネルが表示されます。

    3. <DNT>**App ID**</DNT>を見つけます。

       この番号をRESTAPI呼び出しの`$APP_ID`に置き換えてください。
  </Collapser>

  <Collapser
    id="browser"
    title="ブラウザアプリID"
  >
    New Relic UIから`$APP_ID`を見つけるには：

    1. <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**</DNT>に移動してモニター アプリを選択します。

    2. アプリケーション名の横にある <Icon name="fe-tag"/>アイコンをクリックすると、メタデータパネルが表示されます。

    3. <DNT>**App ID**</DNT>を見つけます。

       この番号をRESTAPI呼び出しの`$APP_ID`に置き換えてください。
  </Collapser>

  <Collapser
    id="mobile"
    title="モバイルアプリID"
  >
    New Relic UIから`$MOBILE_ID`を見つけるには：

    1. <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile**</DNT>に移動してモニター アプリを選択します。

    2. アプリケーション名の横にある <Icon name="fe-tag"/>アイコンをクリックすると、メタデータパネルが表示されます。

    3. <DNT>**App ID**</DNT>を見つけます。

       この番号をRESTAPI呼び出しの`$MOBILE_ID`に置き換えてください。
  </Collapser>
</CollapserGroup>
