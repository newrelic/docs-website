## 既存ラベルの一覧表示 [#list_labels]

アカウントに存在するラベルを一覧表示するには、このコマンドを使用します。

```
curl 'https://api.newrelic.com/v2/labels.json' \
     -H "Api-Key:NEW_RELIC_USER_KEY" -i
```

### 出力を見る [#list_label_output]

このレーベルのアウトプットには

* ラベルの完全なキー名（たとえば、 `Staging:Test1` ）、およびキーのコンポーネントと名前
* 各カテゴリのラベルのアプリケーション`health_status` （赤、オレンジ、緑、灰色）とアプリケーションID
* このラベルが付いているアプリ。たとえば、ラベル`Staging:Test1`が1つのアプリケーション（4937390）と2つのホスト（3801143と5879721）に適用されています
