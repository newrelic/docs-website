## 問題

あなたのREST APIコールは、この情報の一部またはすべてを含む `301` エラーメッセージを返します。

```
curl -X GET 'http://<var>rpm.newrelic.com/v2/applications/1234567890.json</var>' \
     -H 'X-Api-Key:REDACTED' -i
HTTP/1.1 301 Moved Permanently
Status: 301 Moved Permanently
Location: http://<var>rpm.newrelic.com/v2/applications/1234567890.json</var>
...
You are being redirected https://<var>rpm.newrelic.com/v2/applications/1234567890.json</var>
```

または

```
curl -X GET 'http://<var>rpm.newrelic.com/v2/applications/1234567890.json</var>' \
     -H 'X-Api-Key:REDACTED'
You are being redirected https://<var>rpm.newrelic.com/v2/applications/1234567890.json</var>
```

## 解決策

New Relic では、APM および REST API へのすべてのトラフィックに HTTPS を義務付けています。これには、インバウンドとアウトバウンドの両方のトラフィックが含まれます。REST API 呼び出しが HTTP を使用している場合は、スクリプトやプログラムを HTTPS に変更してください。
