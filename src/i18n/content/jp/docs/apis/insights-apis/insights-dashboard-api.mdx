InsightsDashboardsAPIは使用しないでください。代わりに、GraphQLAPIである[NerdGraph](/docs/query-your-data/explore-query-data/dashboards/dashboards-api/)でNewRelicDashboardsAPIを使用してください。この移行を簡単に行う方法については、この短いYouTubeビデオ（約6:30分）をご覧ください。

## エンドオブライフのお知らせ [#eol]

Insights Dashboard APIは2021年に終了しました。

* 2021年7月28日の時点で、 `CREATE`および`UPDATE`エンドポイントは使用できません。
* 2021年8月30日の時点で、 `GET`および`DELETE`エンドポイントは使用できません。

InsightsDashboardAPIからNewRelicDashboards APIに移行するには、 [移行ガイド](/docs/new-relic-one/use-new-relic-one/core-concepts/dashboards-api-migration-insights-api-nerdgraph/)を参照してください。詳細については、 [NerdGraphダッシュボードのチュートリアル](/docs/apis/nerdgraph/examples/create-widgets-dashboards-api/)と[ExplorersHubの投稿](https://discuss.newrelic.com/t/important-insights-dashboard-api-end-of-life/149357)を参照してください。
