ニューレリックは、パートナー様がニューレリックパートナープログラムで提供される価値あるリソース（パートナー様向けセールスイネーブルメントコンテンツ、トレーニング、マーケティング資料など）に簡単にアクセスできるウェブポータルを提供しています。

パートナーの皆様で、すでにアクセスされていない場合は、パートナーセールスマネージャーにお問い合わせいただくか、 partnersales @ newrelic.com までご連絡ください。

パートナーポータルにアクセスするにはブラウザで以下のURLを入力（またはブックマーク）して、ユーザー名とパスワードでログインしてください。

```
https://partners.newrelic-external.com/English/
```
