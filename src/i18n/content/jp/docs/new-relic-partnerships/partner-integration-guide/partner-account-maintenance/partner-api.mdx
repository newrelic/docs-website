パートナーシップ API では、New Relic のパートナー様、および [カスタマーパートナーシップとして設定された New Relic アカウント](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships) において、アカウント、ユーザー、およびサブスクリプション関連の設定を管理することができます。

## 要件 [#requirements]

**パートナーシップ API の使用には、いくつかの大きな制限があります。**New Relic の担当者から使用するように特に指示されていない限り、New Relic 組織はこの API を使用しないでください。要件セクション全体を読んで、制限を理解してください。

パートナーシップ API は、2 種類の New Relic アカウントで使用できます: [パートナー](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships)(マネージド サービス プロバイダーおよび New Relic リセラー) と[カスタマー パートナーシップ](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships)(当社のパートナーシップ アカウント構造へのアクセスを許可された大規模な組織)。

パートナーシップAPIの呼び出しには、パートナーシップオーナーアカウントのREST APIキーとパートナーIDの両方による認証が必要です。

正規の New Relic パートナー (マネージド サービス プロバイダーまたは再販業者) であるアカウントには、Partnership API の使用に関する制限はありません。[顧客パートナーシップ](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships)として設定された New Relic 組織には、次の制限があります。

### お客様とのパートナーシップの制限 [#customer-restrictions]

お客様のNew Relic組織が [カスタマーパートナーシップ](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions/#partnerships) として設定されている場合、お客様の組織がより新しい [プライシングモデル](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-changes-pricing-user-model/#pricing-plans) またはより新しい [ユーザーモデル](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) である場合には、いくつかの制限があります。詳細はこちら。

* **ユーザーモデル。**New Relic 組織に[新しいユーザー モデル](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models)のユーザーがいる場合、アカウントまたはユーザーを管理するパートナーシップ API 呼び出しを使用できなくなります。代わりに使用するものの詳細は次のとおりです。

  * Partnership API[ユーザー オブジェクト](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-user-object)を使用する代わりに、 [NerdGraph を介してユーザーを管理します](/docs/apis/nerdgraph/examples/nerdgraph-manage-users)。
  * パートナーシップ API[アカウント オブジェクト](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-account-object) [または子アカウント オブジェクト](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-sub-account-object)を使用する代わりに、 [NerdGraph を介してアカウントを管理します](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph)。

* **価格モデル。**新しい[使用量ベースの料金モデルを使用](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-changes-pricing-user-model/#pricing-plans)している場合、 [元の料金モデル](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-changes-pricing-user-model/#pricing-plans)を管理するAPI呼び出しを使用することはできません。 APIの影響は次のとおりです。

  * [サブスクリプション・オブジェクトには](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-subscription-object/)
  * [NerdGraph Provisioning API](/docs/apis/nerdgraph/tutorials/provisions-your-subscriptions-nerdgraph/) を使用することはできません。

お客様の価格モデルまたはユーザーモデルを決定するには、 [価格とユーザーモデルの変更の概要](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-changes-pricing-user-model) を参照してください。

## あなたができること [#functions]

この API でできることの概要は次のとおりです。

**アカウントです。**

[要件](#requirements)を満たす一部の組織は、[アカウント オブジェクト](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-account-object/)を使用してアカウントを作成および更新できます。それを使用して次のことができます。

* 新規作成
* 公演
* アップデート
* キャンセル
* 使用方法の表示
* プライマリ管理者を設定
* サブスクリプションを設定

また、子アカウントを作成するための子アカウントオブジェクト [があります。](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-sub-account-object)

[**ユーザー。**](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-sub-account-object)

[](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-sub-account-object)[要件を満たす](#customer-restrictions)一部の組織は、[ユーザー オブジェクト](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-user-object)を次の目的で使用できます。

* アカウントへのユーザーアクセスの追加
* アカウントからのユーザーアクセスの削除

**サブスクリプション。**

[要件を満たし、](#customer-restrictions)元の製品価格モデルを使用している組織は、[サブスクリプション オブジェクト](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-subscription-object)を使用して、さまざまなサブスクリプション関連の特性を構成できます。

### パスワード要件 [#password]

アカウント作成時に渡されるパスワードには、このような条件があります。

* 8～50文字
* 文字、数字、または特殊文字のみ
* スペースを含むことはできません
* 少なくとも1つの文字を含むこと
* 少なくとも1つの数字または特殊文字を含むこと
