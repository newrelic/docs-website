パートナー様がパートナーシップ統合をカスタマイズするために利用できる雑多な設定がいくつかあります。これらの設定のほとんどは、New Relic の機能の表示を有効または無効にするものです。これらの設定は、New Relic によって設定される必要があり、Partnership Console からは設定できません。

## 設定の種類 [#other-settings]

これらの設定を変更したい場合は、 [パートナーシップのテクニカルコンタクト](/docs/accounts-partnerships/partnerships/partner-integration-guide/contacting-new-relic) にお問い合わせください。

**ダウンロードリンク**

* エージェントリンクを表示
* 設定ファイルのリンクを表示

**ウェルカムメッセージ。**

* サインアップメッセージ。カスタマイズされたウェルカムメッセージ
* バナーウェルカムメッセージの表示・非表示
* 請求書のメッセージの表示・非表示

**メールコントロール**

* デプロイメントリマインダーの送信
* トライアルメールの送信

**Billing E-mail:**

これは、請求方法が"Reseller" であり、請求書が特定の相手に向けられるべきパートナーシップのためのものです。

**特集アクセス。**

* サーバー監視
* ユーザー管理)
