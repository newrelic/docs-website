本番稼動前の最終ステップとして、インテグレーション・ウォークスルーを実施します（通常は電話会議で行います）。この電話会議の目的と典型的なトピック領域は以下の通りです。

* インテグレーションの機能を確認します。
* 統合のためのお客様のサインアップと利用のUIワークフローを理解し、文書化する。
* 相互の契約上のコミットメントが満たされていることを確認する。
* 展開のしやすさを評価する。
* 必要に応じて微調整を行うためのプランを作成します。

New Relic の展開に作業用アカウントや展開されたアプリケーションが必要な場合は、これらの要素が準備されていることを事前に確認してください。
