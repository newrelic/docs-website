このドキュメントは、 [New Relic のパートナー様向けです](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions) 。パートナーシップの統合が完了すると、New Relic の技術担当者が紹介されます。緊急性のないエスカレーションにはこれらのチャネルをご利用ください。

<Callout variant="tip">
  New Relic のパートナーでない場合は、New Relic のリソースを参照してください [助けを求めたり、サポートチケットを提出したりすることができます](/docs/accounts-partnerships/education/getting-started-new-relic/finding-help) 。
</Callout>

## New Relicサポート [#support]

パートナーアカウントのサポートを受けるには、 [support.newrelic.com](https://support.newrelic.com/) でチケットを作成してください。New Relic に直接お問い合わせいただく場合は、ビジネス開発担当者にご連絡ください。

## ドキュメント [#docs]

[のドキュメント New Relic の docs サイト](https://docs.newrelic.com) は、サポートグループが New Relic の契約者にレベル 1 サポートを提供する際の重要なリソースとなります。これらのリンクをサポートページに掲載することは、セルフヘルプを促し、サポートの労力を軽減する効果的な方法です。

New Relic のドキュメントのエントリーポイント： [docs.newrelic.com](https://docs.newrelic.com).

## オンライン・テクニカル・コミュニティ [#community]

New Relic [Explorers Hub](https://discuss.newrelic.com/categories) は、New Relic の社員、パートナー、そしてコミュニティが、New Relic に関するあらゆることについての専門知識を集めたものです。
