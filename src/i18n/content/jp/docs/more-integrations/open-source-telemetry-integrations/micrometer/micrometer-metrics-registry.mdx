[Micrometer](https://micrometer.io/) は、最も一般的な監視システムの計装クライアントの上にシンプルなファサードを提供し、ベンダーロックインなしにJVMベースのアプリケーションコードを計装することができます。

New Relic の Micrometer メトリクスレジストリは、Micrometer のテレメトリーデータを New Relic アカウントに送信します。お持ちでない方は [Sign up for free](https://newrelic.com/signup) 。クレジットカードは必要ありません。

マイクロメーターの統合に関するリソース

* [READMEをご覧ください](https://github.com/newrelic/micrometer-registry-newrelic)
* [統合スペック](https://github.com/newrelic/newrelic-exporter-specs/tree/master/micrometer)

## データを検索する

お客様のデータを探すには、 [one.newrelic.com](https://one.newrelic.com/ "新しいウィンドウにリンクが開きます。") にアクセスし、 **Your applications** > **Explorer**. **Entities** の画面から、サービスを名前で検索します。New Relic は、Micrometer Spring Boot instrumentation が提供する最も有用なメトリクスのいくつかのチャートを含む、すぐに使える概要を提供します。

すぐに使える概要を出発点として、独自のダッシュボードを構築することができます。チャートの `...` アイコンをクリックして、そのチャートをダッシュボードに追加したり、チャートの背後にあるクエリを表示して修正し、それをダッシュボードに追加することができます。

New Relic に取り込んだデータを照会する方法については、 [データの照会](/docs/using-new-relic/data/understand-data/query-new-relic-data) および [NRQL の紹介](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) を参照してください。
