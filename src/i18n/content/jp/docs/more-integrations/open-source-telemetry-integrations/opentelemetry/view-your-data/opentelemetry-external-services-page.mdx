OpenTelemetryデータを送信し、UIでサービス（エンティティ）を開いたら、\[**外部サービス**]をクリックして、Webサービス、クラウド内のリソース、その他のネットワークエンティティなどのアウトプロセスサービスへの呼び出しを表示できます。 。

このセクションに表示されるデータは、以下の条件を満たしている必要があります。

* `span.kind = client` または `producer`
* `db.system` 存在しません
* `http.url` または`net.peer.name`が存在する必要があります

詳しくは、 [外部サービス](/docs/apm/apm-ui-pages/monitoring/external-services/external-services-intro) をご覧ください。

他のUIページでのOpenTelemetryについては、 [UIの概要](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/view-your-data/opentelemetry-view-your-data)を参照してください。
