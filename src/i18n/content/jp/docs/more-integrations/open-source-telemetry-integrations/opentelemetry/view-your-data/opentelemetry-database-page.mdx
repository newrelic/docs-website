OpenTelemetryデータを送信し、UIでサービス（エンティティ）を開いたら、\[**データベース**]をクリックして、アプリケーションのデータベースとキャッシュデータを表示できます。このページには、個々のデータベーストランザクションが並べ替え可能なテーブルとして表示され、操作、スループット、および応答時間がグラフとして表示されます。

このセクションに表示されるデータは、以下の条件を満たしている必要があります。
