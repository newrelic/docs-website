OpenTelemetry をセットアップし、それを最大限に活用するのは難しい場合があります。エクスペリエンスを最適化するために、これらのベスト プラクティス ガイドを作成しました。特定の問題を解決したい場合は、 [トラブルシューティング ガイドを](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/opentelemetry-troubleshooting)参照してください。

まず、 [OpenTelemetry と New Relic のエンドツーエンド実装ガイドが](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-implementation-guide)あり、OpenTelemetry と New Relic を最適に統合するための詳細なガイダンスが提供されます。

その他のベスト プラクティス ガイドには次のものがあります。

* [アラート](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-alerts): アラートを使用して問題について自分自身に通知します。
* [属性の長さ](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-attributes): 属性名と値の制限内に収まるようにしてください。
* [バッチ処理](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-batching): バッチ処理を使用して、レート制限を回避します。
* [ブラウザ](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-browser): ブラウザ アプリケーションでライセンス キーを公開しないようにします。
* [圧縮](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-compression): ペイロードを最大化します。
* [ログ](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-logs): OpenTelemetry ツールを使用してログを New Relic に送信します。
* [メトリクス](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-metrics): OpenTelemetry メトリクスが New Relic でどのように処理されるかを調べます。
* [リソース](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-resources): テレメトリ データを生成するエンティティにリソースを使用します。
* [トレース](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-traces): トレースとスパンの要件とオプションを参照してください。
* [バージョン](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-versions): サポートされているバージョンの詳細をご覧ください。
