OpenTelemetryデータを送信し、UIでサービス（エンティティ）を開いたら、\[**メトリクスエクスプローラー**]をクリックして、（選択したOpenTelemetry言語の）メトリクスに関する情報を表示できます。また、OpenTelemetryでPrometheusエクスポーターを使用している場合は、ここでメトリックデータを表示できます。
