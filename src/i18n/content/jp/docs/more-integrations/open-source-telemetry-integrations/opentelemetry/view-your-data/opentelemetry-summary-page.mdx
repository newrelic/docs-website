OpenTelemetryデータを送信し、UIでサービス（エンティティ）を開いたら、 \[**概要**]ページを表示してサービスの概要を確認できます。エンティティに関するさまざまなゴールデンシグナルが一覧表示されます。これは、応答時間、スループット、エラー率などの主要な監視の詳細です。この情報を使用することで、さらに深く掘り下げる必要があるかどうかをすばやく判断できます。

OpenTelemetryデータをこのセクションに表示するには、次のデータが含まれていることを確認してください。

  <thead>
    <tr>
      <th style={{ width: "200px" }}>
        UIエリア
      </th>

      <th>
        属性
      </th>

  </thead>

  <tbody>
    <tr>
      <td>
        レスポンスタイム
      </td>

      <td>
        `span.kind = server` または `consumer`
      </td>
    </tr>

    <tr>
      <td>
        スループット
      </td>

      <td>
        `span.kind = server` または `consumer`
      </td>

    <tr>
      <td>
        エラー率
      </td>

      <td>
