Cloud Cost Intelligence統合機能を使用すると、 cloudプロバイダー アカウントを表示および管理できます。 この機能を使用して、特定のcloudプロバイダー アカウントの統合を一時停止または削除します。

1. <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>に移動します。
2. 左側のナビゲーション メニューから**Integrations** \[統合]タブを選択して、すべてのcloudコスト統合のリストを表示します。
3. 追加の詳細を表示するには、 **Name** \[名前] 列で、必要なcloudプロバイダー アカウントをクリックします。

<Callout variant="important">
  現在、 からのcloud AWSコストのみがサポートされています。
</Callout>

4. 必要なアカウントの横にある省略記号アイコンをクリックし、次のいずれかを選択します。

   * **Pause** \[一時停止]: インテグレーションによるデータ収集を一時的に停止します。
   * **Delete** \[削除]: アカウントからインテグレーションを完全に削除します。
