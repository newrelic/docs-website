cloudプロバイダー アカウントを表示および管理し、特定のアカウントのインテグレーションを一時停止または削除します。

<Steps>
  <Step>
    ### Cloud Cost Intelligenceにアクセス

    <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>に移動します。
  </Step>

  <Step>
    ### 統合を見る

    すべてのcloudコスト統合を表示するには、 **Integrations** \[統合]タブを選択します。
  </Step>

  <Step>
    ### アカウントを選択

    詳細については、 **Name** \[名前]列のcloudプロバイダー アカウントをクリックしてください。

    <Callout variant="important">
      現在、 AWSクラウドのコストのみがサポートされています。
    </Callout>
  </Step>

  <Step>
    ### 統合を管理する

    アカウントの横にある省略記号アイコンをクリックし、次のいずれかを選択します。

    * **Pause**- データ収集を一時的に停止します
    * **Delete**- 統合を完全に削除します

    <img title="Pause and delete integration" alt="A screenshot displaying the pause and delete integration options" src="/images/pause-delete-cci.webp" />
  </Step>
</Steps>
