予算は、1 か月から 12 か月までの柔軟な期間にわたってアカウントの支出限度額を設定します。事前定義された制限に照らしてcloud支出を監視することで、クラウド支出を管理するのに役立ちます。

<img title="Cost types highlighted on the Cloud Cost Intelligence budgets page" alt="Cost types highlighted on the Cloud Cost Intelligence budgets page" src="/images/cci_budgets.webp" />

## Set up budgets

<Steps>
  <Step>
    ### Cloud Cost Intelligenceに移動
    ### Cloud Cost Intelligenceに移動

    **All Capabilities** &gt; **Cloud Cost Intelligence**に移動します。
  </Step>

  <Step>
    ### アクセス予算

    左側のナビゲーション メニューから、 **Budgets** \[予算]タブを選択します。
  </Step>

  <Step>
    ### 予算を作成する
    ### 予算を作成する

    1. **Budgets** \[予算]ページで、 **Create a budget** \[予算の作成]をクリックします。
    2. **Create a budget** \[予算の作成]ポップアップ ダイアログ ボックスで、次の情報を入力します。

    * **Budget name:**予算のわかりやすい名前。

    * 必要なセットアップタイプを選択します:

  <Step>
    ### 確認して確認する
    ### 確認して確認する

    最終的な予算設定を確認し、 **Create** \[作成]をクリックします。

    <Callout variant="important">
      **Cost Intelligence** \[コストインテリジェンスは]、過去 3 か月間の平均計算コストを示します。 これは過去 90 日間の実際の支出に基づいて計算されます。
    </Callout>
  </Step>
</Steps>
