<DNT>**Kubernetes Costs**</DNT>ページから、Kubernetes環境に特化したインサイトにアクセスします。この機能は、New Relicのテレメトリーデータを活用して、Kubernetesクラスタに関連するコストの詳細なビューを提供します。

各アプリケーションで経時的に発生したコストと詳細なコスト分析を表示するには、<DNT>**Kubernetes Costs**</DNT>ページを使用します。

<img title="Kubernetes Cost Insights highlighted on the Cloud Cost Intelligence Kubernetes Cost Insights page" alt="Kubernetes Cost Insights highlighted on the Cloud Cost Intelligence Kubernetes Cost Insights page" src="/images/k8s-cci.webp" />

## フィルターとカスタマイズ

### クラウドプロバイダーフィルター

コストデータをcloudプロバイダーでフィルタリングするには、ページ左上のドロップダウンメニューからオプションを選択します：

* <DNT>**All**</DNT>: すべてのcloudプロバイダー
* <DNT>**AWS**</DNT>: Amazon Web Servicesのみ
* <DNT>**Azure**</DNT>: Microsoft Azureのみ

### 日付範囲セレクター

ページの右上にある日付範囲セレクターを使用して、次のことを行います：

* 事前定義された期間を選択
* カスタムの開始日と終了日を設定する
* 優先タイムゾーンでコストを表示します（デフォルトではUTCが表示されます）

### エンジニアリングコストフィルター

組織でコスト配分タグを使用している場合、<DNT>**Engineering costs**</DNT>タグでフィルタリングして、cloud環境でタグ付けされた特定のチーム、プロジェクト、またはコストセンターに分析を絞り込むことができます。

## 概要

Summary \[概要] セクションには、選択した期間の4つの主要なコストメトリクスが表示されます：

* <DNT>**Total cost**</DNT>: すべてのKubernetesクラスタの総支出と、前期と比較した変化率
* <DNT>**Average cost (Daily)**</DNT>: 日次平均支出とパーセンテージの変化
* <DNT>**Unused cost**</DNT>: 割り当てられているがrequestsを受信していないリソースのコスト、および変化率
* <DNT>**Unallocated cost**</DNT>: どのソースにも割り当てられていないクラスタリソースのコスト、および変化率

## インサイト

Insightsセクションでは、Kubernetesのコストを分析するのに役立つ2つの可視化ウィジェットを提供しています。

### コスト動向

<DNT>**Cost Trends**</DNT>ウィジェットは、経時的なKubernetesコストの変化を示す折れ線グラフを表示します。ビューオプションを使用して、<DNT>**Hourly**</DNT>、<DNT>**Daily**</DNT>、<DNT>**Weekly**</DNT>、または<DNT>**Monthly**</DNT>ビューを切り替えます。

### トップ 20

<DNT>**Top 20**</DNT>ウィジェットは、ツリーチャートの視覚化を使用して、最もコストの高い領域を表示します。デフォルトでは、コスト別の上位20件のクラスタが表示されます。次を使用してビューをカスタマイズできます：

* <DNT>**Dropdown filter**</DNT>: 表示する内容（<DNT>**Cluster name, by cost**</DNT>）を選択します
* <DNT>**Group by**</DNT>：コストデータをグループ化するディメンションを選択します
* <DNT>**View by**</DNT>: <DNT>**Tree chart**</DNT>と他のビジュアル形式を切り替える

## コストと使用状況分析

<DNT>**Cost &amp; Usage Analysis**</DNT>セクションには、リソース使用量とコストの内訳をクラスタ名ごとに示す詳細な表が表示されます：

* <DNT>**Cluster name**</DNT>: Kubernetesクラスタの名前
* <DNT>**CPU cost**</DNT>: 各クラスタのCPU使用量に属性付けられたコスト
* <DNT>**CPU usage**</DNT>: 各クラスタが消費するCPUリソースの割合
* <DNT>**Memory cost**</DNT>: メモリ使用量に関連するコスト
* <DNT>**Memory usage**</DNT>：各クラスタが使用するメモリリソースの割合
* <DNT>**Unused Cost**</DNT>: 割り当てられているが使用されていないリソースのコスト
* <DNT>**Unallocated cost**</DNT>: どのソースにも割り当てられていないクラスタリソースのコスト
* <DNT>**Total cost**</DNT>: すべてのリソース使用量を統合した、各クラスタで発生した総コスト（ソート可能）

## 未使用および未割り当てのコストを理解する

これらのコスト ディメンションは、クラスターの効率を測定するのに役立ちます。

### 未使用コスト

未使用コストは、割り当てられているものの、requestsを受け取っていないリソースを表します。

**例**: クラスターのデプロイメントに割り当てられた合計コストが 5.60 ドルで、実際に使用しているのは 4.64 ドルだけの場合、未使用コストは 0.96 ドルになります。

### 未配分コスト

未割り当てのコストは、どのソースにも割り当てられていないクラスタリソースを表します。

これらのメトリクスは、クラスタの効率を向上させ、無駄を削減する機会を特定するのに役立ちます。

<Callout variant="tip">
  任意のクラスタを選択して、その<DNT>**Cost Trends**</DNT>、<DNT>**Cost Breakdown (Total vs Used)**</DNT>、および未使用と未割り当てのコストに関する詳細情報を表示します。
</Callout>

## 利点

Kubernetes Cost Insights機能は、以下の成果をもたらします：

* <DNT>**Optimized resource usage**</DNT>：使用率の低いクラスタや過剰にコストがかかっているクラスタを可視化することで、効果的なリソース割り当ての調整が可能になります。
* <DNT>**Controlled expenses**</DNT>：Kubernetes環境内の特定のコスト要因に関するInsightsにより、より情報に基づいた財務上の意思決定が可能になります
* <DNT>**Enhanced efficiency**</DNT>: ビジネス目標に沿ったリソース管理戦略により、cloudリソースの効率的な使用が確実になります

<DNT>**Kubernetes Costs**</DNT>ページは、Kubernetesのコストをプロアクティブに管理するために必要なインサイトを提供し、戦略的な意思決定と財務の最適化をサポートします。
