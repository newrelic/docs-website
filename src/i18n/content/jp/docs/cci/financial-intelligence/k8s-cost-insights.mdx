Kubernetesコスト割り当てページから、 Kubernetes環境に固有のインサイトにアクセスします。 この機能はNew Relicのテレメトリーデータを利用して、 Kubernetesアプリケーションに関連するコストのビューを提供します。

Kubernetes コスト割り当てページを使用して、時間の経過に伴う各アプリケーションで発生したコストと詳細なコスト分析を表示します。

## 割り当ての概要

割り当て概要ウィジェットは、時間の経過とともに各アプリケーションで発生するコストを示すバーチャート視覚化を提供します。

## 詳細なコスト分析

Kubernetes コスト割り当てビューには、アプリケーション名別にリソース使用量とコストの内訳を示す詳細な表も表示されます。

* **CPU 使用率:**各アプリケーションによって消費された CPU リソースの割合を表示します。
* **CPU コスト:**各アプリケーションの CPU 使用率に対するコスト属性を表示します。
* **メモリ使用量:**各アプリケーションによって使用されるメモリ リソースをパーセンテージで一覧表示します。
* **メモリコスト:**メモリ使用量に関連するコストを反映します。
* **総コスト:**すべてのリソース使用量を統合し、各アプリケーションで発生した総コストを提供します。

## 利点

Kubernetes コスト割り当て機能を利用すると、次のことが可能になります。

* **リソース使用率の最適化:**十分に活用されていないアプリケーションやコストがかかりすぎるアプリケーションを特定し、リソースの割り当てを効果的に調整します。
* **経費の管理:** Kubernetes 環境内の特定のコスト要因を可視化し、より情報に基づいた財務上の意思決定を可能にします。
* **効率性の向上:** cloudリソースを効率的に使用するために、ビジネス目標に沿ったリソース管理戦略に重点を置きます。

Kubernetesコスト割り当てページでは、 Kubernetesコストを積極的に管理するために必要な情報が提供され、戦略的な意思決定と財務の最適化をサポートします。
