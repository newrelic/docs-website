## 概要

Kubernetesコスト割り当てページから、 Kubernetes環境に固有のインサイトにアクセスします。 この機能はNew Relicのテレメトリーデータを利用して、 Kubernetesアプリケーションに関連するコストのビューを提供します。

Use the Kubernetes Cost Allocation page to view costs incurred by each application over time and detailed cost analysis.

## 割り当ての概要

割り当て概要ウィジェットは、時間の経過とともに各アプリケーションで発生するコストを示すバーチャートを視覚化します。

## 詳細なコスト分析

Kubernetes コスト割り当てビューには、アプリケーション名別のリソース使用量とコストの内訳を示す詳細な表が表示されます。

* **CPU 使用率:**各アプリケーションによって消費された CPU リソースの割合を表示します。
* **CPU コスト:**各アプリケーションの CPU 使用率に対するコスト属性を表示します。
* **メモリ使用量:**各アプリケーションによって使用されるメモリ リソースをパーセンテージで一覧表示します。
* **メモリコスト:**メモリ使用量に関連するコストを反映します。
* **総コスト:**すべてのリソース使用量を統合し、各アプリケーションで発生した総コストを提供します。

## 未使用および未割り当てコスト

これらのコスト ディメンションは、クラスターの効率を測定するのに役立ちます。

### 未使用コスト

未使用コストとは、クラスターの割り当てられているものの、リクエストを受け取っていない量です。

**例**: クラスターのデプロイメントに割り当てられた合計コストが 5.60 ドルで、実際に使用しているのは 4.64 ドルだけの場合、未使用コストは 0.96 ドルになります。

### 未配分コスト

未割り当てコストとは、クラスターのうちどのソースにも割り当てられていないコストのことです。

これらは、 Kubernetesクラスタの効率化の手段です。

## 利点

Kubernetes コスト割り当て機能により、次の成果がもたらされます。

* **最適化されたリソース使用率:**十分に活用されていないアプリケーションやコストがかかりすぎるアプリケーションを可視化することで、効果的なリソース割り当ての調整が可能になります。
* **経費の管理:** Kubernetes環境内の特定のコスト要因に関するInsightsにより、より情報に基づいた財務上の意思決定が可能になります。
* **効率性の向上:**ビジネス目標に合わせたリソース管理戦略により、 cloudリソースを効率的に使用できます。

Kubernetesコスト割り当てページでは、 Kubernetesコストを積極的に管理するために必要な情報を提供し、戦略的な意思決定と財務の最適化をサポートします。
