Cloud Cost Intelligenceセットアップが完了したら、Summary \[概要] ページからcloud支出を把握できます。 Summary \[概要] ページには、さまざまなコスト タイプとともに、 cloudコストに関するデータを提供するさまざまな ID が表示されます。

## 主な特徴

概要ページでは次のことができます。

* **支出の表示**: 使用量メトリクスやコスト配分など、クラウド サービス、チーム、期間にわたる支出データを表示します。
* **コストの表示**: リアルタイムの見積もりと詳細な内訳にアクセスして、潜在的なコスト超過を予測します。
* **コストの最適化**: 最もコストの差異が大きいリソースとタグなしのリソースを特定します。
* **カスタマイズされたインサイトの取得**: アカウント、アプリケーション、チームなどの複数のフィルターを通じてコストを表示して、分析を調整します。

## データウィジェット

Cloud Cost Intelligence (CCI) ダッシュボードは、 cloud支出の包括的なビューを提供するさまざまなデータ ウィジェットを提供します。 各ウィジェットは、支出のさまざまな側面に特定のインサイトを提供し、コストを効果的に追跡、分析、最適化するのに役立ちます。 これらのウィジェットを調べて、財務データをより深く理解し、リソースの割り当てとコスト管理について情報に基づいた意思決定を行います。利用可能なウィジェットを詳しく見てみましょう:

## 利点

概要機能を利用すると、組織のcloud経費の管理に大きな利点が得られます。

* **強化された財務管理:** cloud支出を明確かつ包括的に把握できるため、情報に基づいた財務上の意思決定が可能になり、予算を管理しやすくなります。

* **プロアクティブなコスト管理:**リアルタイムの見積もりと詳細な内訳にアクセスすることで、潜在的なコスト超過を予測し、予算や業務に影響が出る前に対処できます。

* **戦略的なリソース割り当て:**さまざまなウィジェットのインサイトを使用して、さまざまなアカウント、アプリケーション、地域、チームにわたるリソースの配分を最適化します。 リソースの使用を戦略的なビジネス目標に合わせて調整し、効率を向上させます。

* **最終コストの最適化:**最適化の対象となる最大のコスト差異とタグなしリソースを特定します。 コスト削減の可能性が高い特定の領域に焦点を当てます。

* **カスタマイズされたインサイト:**アカウント、アプリケーション、チームなどの複数のレンズを通してコストを表示して分析を調整し、組織にとって最も重要な特定のコスト要因を深く掘り下げることができます。

これらのインサイトと表示オプションを活用することで、 cloud支出を制御および最適化し、効果的な財務管理と戦略計画を確実に行うことができます。
