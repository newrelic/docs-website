<DNT>**AI Costs**</DNT>ページでは、cloudプロバイダー全体のAIサービスのコストが可視化されます。サービス、モデル、トークンタイプ別に分類されたトークンレベルのコストを表示し、マルチクラウド環境や複数のLLMモデルにわたるAI支出の統合ビューを提供します。

## AIコストへのアクセス

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Cloud Cost Intelligence**</DNT>に移動します。
2. 左側のナビゲーションで<DNT>**AI Costs**</DNT>をクリックしてください。

## クラウドプロバイダーフィルター

AIコストデータをcloudプロバイダー別にフィルタリングするには、ページ上部のプロバイダータブからオプションを選択します：

* <DNT>**All**</DNT>: すべてのcloudプロバイダー
* <DNT>**AWS**</DNT>: Amazon Web Servicesのみ
* <DNT>**Azure**</DNT>: Microsoft Azureのみ
* <DNT>**GCP**</DNT>: Google Cloud Platformのみ

## AIの上位支出

<DNT>**AI Top Spends**</DNT>セクションでは、4つのディメンションにわたってAIコストが最も高い領域を表示します。各ウィジェットは、選択した期間の合計、前の期間との比較値、変化率、および上位3項目の内訳を表示します。

### サービス

cloud AIサービス（たとえば、AmazonBedrock、Azure AI Services、AmazonSageMaker）ごとにグループ化されたAIの総コストを表示します。これを使用して、cloudプロバイダー全体でどのサービスが最も多くの支出を発生させているかを特定します。

### モデル

LLMモデル（たとえば、Claude Opus 4.6、Claude Sonnet 4.6）別にグループ化されたAIの総コストを表示します。これを使用して、モデルのバージョンとプロバイダー間でコストを比較します。

### トークンコスト

AIの総支出をトークンタイプ別に分類して表示します：

* <DNT>**Cache Write Token**</DNT>: キャッシュにトークンを書き込むコスト
* <DNT>**Input Token**</DNT>: モデルによって処理された入力トークンのコスト
* <DNT>**Cache Read Token**</DNT>: キャッシュからのトークン読み取りコスト

これを使用して、どのトークン操作がAIコストに最も寄与しているかを理解します。

### トークン使用量

トークンタイプ別に分類されたトークンの合計使用量（10億単位）を表示します。これをトークンコストと併用して、最適化の機会を示す可能性のある、大量で低コストのパターンと少量で高コストのパターンを識別します。

<Callout variant="tip">
  異なるトークンタイプやモデル間でのトークンあたりのコスト効率を理解するために、<DNT>**Token Cost**</DNT>と<DNT>**Token Usage**</DNT>のウィジェットを比較します。
</Callout>

## コスト動向

<DNT>**Cost Trends**</DNT>セクションでは、インタラクティブなチャートとコントロールを使用して、AIコストの時間ベースの分析が提供されます。

### チャートコントロール

* <DNT>**View options**</DNT>：<DNT>**Hourly**</DNT>、<DNT>**Daily**</DNT>、<DNT>**Weekly**</DNT>または<DNT>**Monthly**</DNT>ビューを切り替えます
* <DNT>**Group by**</DNT>：コストデータをグループ化するディメンションを選択します（最大3つまで選択可能）
* <DNT>**View by**</DNT>: 視覚化フォーマットを選択します（利用できるかどうかはグループ化の選択によって異なります）

### データの理解

* <DNT>**Total cost**</DNT>: 選択期間のAIの総支出と、前の期間との変化率
* <DNT>**Average cost (Daily)**</DNT>: 前の期間と比較したパーセンテージの変化を伴う、1日あたりの平均AI支出
* <DNT>**Stacked bar chart**</DNT>: 一定期間にわたるAIコストの分布を、cloudプロバイダーごとに色分けして視覚化します
* <DNT>**Output token**</DNT>: プロンプトに応答してモデルによって生成された出力トークンのコスト

### 視覚化オプション

**単一のgroup-by選択の場合：**

* <DNT>**Bar chart**</DNT>：AIコストデータを縦棒で表示します
* <DNT>**Line chart**</DNT>: 時間の経過に伴うAIコストの傾向を示します
* <DNT>**Tree chart**</DNT>: AIコストデータの階層的な視覚化
* <DNT>**Data table**</DNT>: AI コストデータの表形式ビュー

**複数のgroup-byを選択した場合：**

* <DNT>**Heat map**</DNT>: ディメンション間の色分けされた視覚化
* <DNT>**Data table**</DNT>: AI コストデータの表形式ビュー

## フィルターとカスタマイズ

### 日付範囲セレクター

右上にある日付範囲セレクターを使用して、次の操作を行います：

* 事前定義された期間を選択
* カスタムの開始日と終了日を設定する
* 優先タイムゾーンでコストを表示します（デフォルトではUTCが表示されます）

### エンジニアリングコストフィルター

組織でコスト配分タグを使用している場合は、<DNT>**Engineering costs**</DNT>タグでフィルタリングして、AIコスト分析をcloud環境でタグ付けされた特定のチーム、プロジェクト、またはコストセンターに絞り込みます。
