<DNT>**Cost Overview**</DNT>ページ（旧称<DNT>**Cost Explorer**</DNT>）は、複数のディメンションにわたるドリルダウン機能を備えた詳細なコスト分析を提供します。

サマリー ウィジェットを使用したすべてのcloud支出の統合ビューについては、[インテリジェント サマリー](/docs/cci/financial-intelligence/intelligent-summary)ページを参照してください。

<img title="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" alt="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" src="/images/cost-overview.webp" />

## クラウドプロバイダーフィルター

コストデータをcloudプロバイダーでフィルタリングするには、ページ左上のドロップダウンメニューからオプションを選択します：

* <DNT>**All**</DNT>: すべてのcloudプロバイダー
* <DNT>**AWS**</DNT>: Amazon Web Servicesのみ
* <DNT>**Azure**</DNT>: Microsoft Azureのみ
* <DNT>**GCP**</DNT>: Google Cloud Platformのみ

## 上位の支出

<DNT>**Top Spends**</DNT>ウィジェットは、最もコストの高い領域を特定するのに役立ちます。次で分類されたコストを表示：

* <DNT>**Service**</DNT>：クラウドサービスタイプ別にグループ化されたコスト（たとえば、AmazonEC2、Fastly Edge Cloud Platform、AmazonS3）
* <DNT>**Category**</DNT>: 使用量のカテゴリ別に整理されたコスト（たとえば、計算、その他、ネットワーク）
* <DNT>**Region**</DNT>：地理的地域別のコスト（たとえば、us-west-2、Untagged、eastus）。「Untagged」は、リージョンタグのないリソースからのコストを示します。
* <DNT>**Charge type**</DNT>: 請求カテゴリ別のコスト（たとえば、Usage、SavingsPlanCoveredUsage、DiscountedUsage）

各ディメンションには次が表示されます：

* 合計コスト金額
* 前の期間と比較した変化率（選択した日付範囲に基づく）
* 上位3つの項目と個々のコスト
* <DNT>**Show summary**</DNT> 詳細ビューの展開または折りたたみを切り替えます

<Callout variant="tip">
  変化率インジケーターを使用して、コストが大幅に増加しているサービスやリージョンを迅速に特定します。赤いインジケーターは、前の期間と比較して支出が増加した領域を強調表示します。
</Callout>

## コスト動向

<DNT>**Cost Trends**</DNT>セクションでは、インタラクティブなチャートとコントロールを備えた時間ベースのコスト分析が提供されます。

### チャートコントロール

* <DNT>**View options**</DNT>：<DNT>**Hourly**</DNT>、<DNT>**Daily**</DNT>、<DNT>**Weekly**</DNT>または<DNT>**Monthly**</DNT>ビューを切り替えます
* <DNT>**Group by**</DNT>: コストデータをグループ化するディメンションを1～3個選択します
* <DNT>**View by**</DNT>：視覚化フォーマットを選択します（利用できるかどうかはgroup-byの選択によって異なります）

### データの理解

* <DNT>**Total cost**</DNT>: パーセンテージの変化とともに総支出を表示します
* <DNT>**Average cost**</DNT>: 1日あたりの平均支出とパーセンテージの変化を表示します
* <DNT>**Stacked bar chart**</DNT>: サービスごとに色分けされたセグメントを使用して、一定期間にわたるコスト分布を視覚化します

### コストデータテーブル

チャートの下に、詳細なテーブルが表示されます：

* <DNT>**Cloud provider service**</DNT>：サービス名
* <DNT>**Total Cost**</DNT>: 集計支出額（並べ替え可能）
* <DNT>**% of Spend**</DNT>: 合計支出の割合
* <DNT>**Compared Cost**</DNT>: 前の期間とのコスト比較
* <DNT>**Trend**</DNT>: 色分けされたパーセンテージ変化インジケータ（減少は緑、増加は赤）

### 視覚化オプション

**グループ化の制約：**

* **最小選択**: 1次元
* **最大選択**：3次元

**単一のgroup-by選択の場合：**

* <DNT>**Bar chart**</DNT>: コストデータを縦棒として表示します
* <DNT>**Line chart**</DNT>: 期間中のコストの傾向を示します
* <DNT>**Tree chart**</DNT>: コストデータの階層的な視覚化
* <DNT>**Data table**</DNT>: コストデータの表形式ビュー

**複数のgroup-byを選択した場合：**

* <DNT>**Heat map**</DNT>: 色分けされた視覚化
* <DNT>**Data table**</DNT>: コストデータの表形式ビュー

## ドリルダウン機能

コストデータ表の任意のサービス（たとえば、AmazonEC2）をクリックして、カテゴリ、リージョン、または請求タイプ別の詳細な内訳を表示します。もう一度クリックすると、個々のリソースコストが表示されます。

計算、ネットワーク、ストレージを含むすべてのサービス タイプでドリルダウンがサポートされています。

## フィルターとカスタマイズ

<DNT>**Cost Overview**</DNT>ページは、特定のコストセグメントに焦点を当てるのに役立つフィルタリングおよびカスタマイズオプションをサポートしています。

### 日付範囲セレクター

ページの右上にある日付範囲セレクターを使用して、次の操作を行います：

* 事前定義された期間を選択
* カスタムの開始日と終了日を設定する
* 優先タイムゾーンでコストを表示します（デフォルトではUTCが表示されます）

### エンジニアリングコストフィルター

組織でコスト配分タグを使用している場合、<DNT>**Engineering costs**</DNT>タグでフィルタリングして、cloud環境でタグ付けされた特定のチーム、プロジェクト、またはコストセンターに分析を絞り込むことができます。
