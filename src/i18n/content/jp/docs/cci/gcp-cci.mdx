Cloud Cost IntelligenceのためのGoogle Cloud Platform (GCP) 統合により、GCP cloud支出をNew Relic内で直接監視および分析できます。 この統合では、サービス アカウント ベースの認証方法を使用して、GCP の請求データと価格データに安全にアクセスします。

この統合をセットアップすると、 New Relicユーザーに代わって GCP 環境に専用のサービス アカウントを作成します。 このサービス アカウントに BigQuery データセットへの読み取り専用アクセス権を付与すると、New Relic は GCP リソースへのより広範なアクセス権を必要とせずにコストデータを取得して分析できるようになります。

## 主な特徴

* New Relic アカウントごとにサービス アカウントを自動生成
* BigQuery を介した課金データへの安全な読み取り専用アクセス
* 課金エクスポートとcloud料金エクスポートテーブルの両方へのアクセス
* 接続検証とエラー報告

## 前提条件

GCP 統合を設定する前に、次のことを確認してください。

* New Relicインフラストラクチャへの接続モニタリング - GCP プロジェクトはすでに接続されています。 [Google Cloud Platform サービスを New Relic に接続する](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)方法をご覧ください

* アクティブな GCP アカウントで課金を有効にしました。

* 次のテーブルを使用して BigQuery データセットを構成しました。

  * 課金エクスポート テーブル (GCP 課金データが含まれています)
  * クラウド料金エクスポート表（GCP 料金情報を含む）

* Google Cloud Console へのアクセスが確認されました。

## Google Cloud Platform を Cloud Cost Intelligence に接続する

<Steps>
  <Step>
    ### アカウントを選択してください

    New Relicアカウントを選択して統合を開始します。

    1. <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT>に移動します。
    2. ドロップダウンからアカウントを選択し、 **Continue** \[続行]をクリックします。
  </Step>

  <Step>
    ### GCP接続を設定する

    1. GCP 請求アカウントを New Relic に接続します。

       * **接続名**- わかりやすい名前を入力します (例: 「GCP 本番環境」)
       * **プロジェクトID** - GCPプロジェクトIDを入力してください
       * **データセット名**- BigQueryデータセット名を入力します
       * **請求テーブル名**- 請求テーブル名を入力します
       * **価格表名**- 価格表名を入力してください

    2. Click **Continue**.
  </Step>

  <Step>
    ### New Relic に GCP へのアクセスを許可する

    New Relic は、BigQuery の Cloud Billing データにアクセスするために、組織固有のサービス アカウント ID を提供します。

    1. 表示されるサービス アカウント ID をコピーします。
    2. **Google Cloud Console**を開き、サービス アカウントに BigQuery 課金テーブルに対する**BigQuery DataViewer**ロールを付与します。
    3. このページに戻り、 **Continue** \[続行]をクリックします。
  </Step>

  <Step>
    ### 接続をテストして作成する

    **Create connection** \[接続の作成]をクリックして設定を確認し、統合を確立します。
  </Step>

  <Step>
    ### データ収集を確認する

    GCP のコストデータを表示するには、[**Cost Overview** \[コストの概要\]](/docs/cci/financial-intelligence/cloud-cost-insights)に移動します。

    <Callout variant="important">
      初めてデータを処理する場合は最大 12 時間かかる場合があります。
    </Callout>
  </Step>
</Steps>
