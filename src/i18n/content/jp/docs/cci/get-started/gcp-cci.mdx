Google Cloud Platform課金をCloud Cost Intelligenceに接続して、コストを監視および最適化します。 統合では、サービス アカウント ベースの認証を使用して、GCP の請求データと価格データに安全にアクセスします。 New Relic は、BigQuery データセットへの読み取り専用アクセス権を持つ専用のサービス アカウントを作成します。

## あなたが始める前に

必要なもの:

* **New Relicインフラストラクチャに接続された GCP プロジェクトのモニタリング**- [Google Cloud PlatformサービスをNew Relicに接続する」](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)を参照

* **アクティブな GCP 請求アカウント**

* 次の**BigQuery データセット**:

  * 請求書エクスポートテーブル
  * クラウド料金表のエクスポート

* **Google Cloud Console へのアクセス**

## セットアップ手順

<Steps>
  <Step>
    ### New Relicアカウントを選択 [#select-nr-account]

    1. <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT>に移動します。
    2. アカウントを選択し、 **Continue** \[続行]をクリックします。
  </Step>

  <Step>
    ### GCP接続の詳細を提供する [#gcp-connection-details]

    1. GCP の請求およびデータセット情報を入力します。

       * **Connection name** \[接続名]- わかりやすい名前
       * **Project ID** \[プロジェクトID] - GCPプロジェクトID
       * **Dataset name** \[データセット名] - BigQueryデータセット名
       * **Billing table name** \[請求テーブル名] - 請求テーブル名
       * **Pricing table name** \[価格表名] - 価格表名

    2. Click **Continue**.
  </Step>

  <Step>
    ### アクセスを許可する [#gcp-authorize-access]

    1. 表示されるサービス アカウント ID をコピーします。
    2. **Google Cloud Console**を開き、BigQuery 課金テーブルのサービス アカウントに**BigQuery DataViewer**ロールを付与します。
    3. 戻って**Continue** \[続行]をクリックします。
  </Step>

  <Step>
    ### 接続を作成 [#gcp-create-connection]

    **Create connection** \[接続の作成]をクリックして設定を確認し、統合を確立します。
  </Step>

  <Step>
    ### 自分のデータを見る [#gcp-view-data]

    GCP コストデータを表示するには、[**Intelligent summary (インテリジェント サマリー)**](/docs/cci/financial-intelligence/intelligent-summary)にアクセスしてください。

    <Callout variant="important">
      初回の処理には最大 12 時間かかる場合があります。
    </Callout>
  </Step>
</Steps>
