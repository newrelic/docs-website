Cloud Cost Intelligence を設定するときに、いくつかの問題が発生する可能性があります。以下に、一般的な問題、考えられる原因、およびそれらを効果的に解決するための推奨されるトラブルシューティング手順を示します。

## New Relicにデータが表示されない

New Relic アカウントにコスト使用状況レポート (CUR)、リアルタイムのコスト見積もり、または Kubernetes コスト割り当てデータが表示されない場合は、次の手順に従って問題をトラブルシューティングしてください。

* **理由:**前提条件の完了にギャップがある可能性があります。

* **解決：**

  1. 必要な前提条件をすべて満たしていることを再確認してください。
  2. 設定を確認してください。
  3. それでも問題が解決しない場合は、 [Cloud Cost Intelligence サポート](mailto:cci-support@newrelic.com)にお問い合わせください。

## 接続の切断

* **理由:**不安定なネットワーク接続によりセットアップが中断される可能性があります。
* **解決策:**安定したインターネット接続を確保し、再度セットアップを試みてください。問題が解決しない場合は、ファイアウォールの潜在的な制限についてネットワーク アドミニストレーターに確認してください。

## 不適切なユーザー権限

* **理由:**ユーザーに Cloud Cost Intelligence を設定するために必要な権限がありません。
* **解決策:** AWS および New Relic アカウントの権限を確認します。セットアップを実行するための適切な権限があることを確認してください。必要に応じて、アドミニストレーターに連絡してください。

## ARN設定エラー

* **メッセージ:** 「入力した ARN では AWS アカウントへの正しいアクセスが許可されません。ARN を確認してもう一度お試しください。」
* **理由:**アクセスを許可しない不正な ARN 設定。
* **解決策:**入力した ARN を再確認し、アクセスに必要な設定と一致していることを確認します。 エラーを修正してセットアップを再試行してください。

## 外部IDによるセキュリティリスク

* **メッセージ:** 「セキュリティリスク: 入力した ARN には広範な権限があります。任意の「外部 ID」を使用してロールを引き受けることができます。AWS ロールの作成時に、提供された外部 ID「%s」を externalId として追加してください。
* **理由:**過度に許容的な外部 ID ポリシーはセキュリティ上のリスクをもたらします。
* **解決策:**提供された外部 ID が externalId として使用されるよう、AWS ロール設定を更新します。

## バケットが存在しません

* **理由:**指定された AWS S3 バケットが存在しません。
* **解決策:**バケット名が AWS アカウントのバケット名と一致していることを確認します。名前にはプレフィックス`cost-and-usage-reports`も必要です。バケットが存在しない場合は作成するか、セットアップで名前を修正します。

## バケット名エラー

* **理由:**指定された AWS S3 バケットが存在しないか、バケット名に誤字があります。
* **解決策:**バケット名が AWS アカウントのものと一致し、プレフィックス`cost-and-usage-reports`が含まれていることを確認します。名前のスペルが正しいことを確認してください。バケットが存在しない場合は、作成するか、セットアップで名前を修正します。

## 接続名が重複しています

* **メッセージ:** 「%s Cloud Cost Intelligence AWS 接続名は既に存在します。新しい接続名を入力してください。&quot;
* **理由:**入力した接続名は既に使用されています。
* **解決策:**一意の接続名を選択して再試行してください。

## 重複した支払者アカウントの設定

* **メッセージ:** 「%s AWS 支払者アカウントは、この New Relic アカウントにすでにリンクされています。」
* **理由:**同じ AWS 支払者アカウントを複数回設定しようとしています。
* **解決策:** AWS 支払人アカウントがまだリンクされていないことを確認します。必要に応じて別の支払人アカウントを使用してください。

## 追加サポート

これらのトラブルシューティング手順を実行しても問題が解決しない場合は、 [Cloud Cost Intelligence サポート](mailto:cci-support@newrelic.com)にお問い合わせください。
