[Cloudflare Logpush](https://developers.cloudflare.com/logs/get-started/enable-destinations/new-relic/) サービスを設定し、Cloudflare ダッシュボードまたは API 経由で New Relic にログを送信します。

<Callout variant="important">
  Cloudflare は、2023 年 2 月 1 日にロールアウトを開始し、2023 年 8 月 1 日に終了する、Cloudflare ログのセキュリティ関連フィールドに対する一連の更新を発表しました。詳細については、 [Cloudflare の公式発表を](https://developers.cloudflare.com/logs/reference/change-notices/2023-02-01-security-fields-updates/) 確認してください。 Cloudflare から New Relic に送信されるすべてのログは、これらの変更の影響を受けます。
</Callout>

## New Relic への Logpush を有効にする [#dashboard]

Cloudflare Logpush を有効にするには、設定の選択に基づいて Cloudflare ドキュメントの手順に従います。

* [Cloudflareダッシュボード経由で有効にする](https://developers.cloudflare.com/logs/get-started/enable-destinations/new-relic/#manage-via-the-cloudflare-dashboard)
* [API経由で有効にする](https://developers.cloudflare.com/logs/get-started/enable-destinations/new-relic/#manage-via-api)

## 次は何ですか？ [#what-next]

Cloudflareネットワークログから主要なメトリックを確認するために、事前に構築されたダッシュボードを数分で開始します。

<figcaption>
  Cloudflareネットワークログのインスタント可観測性クイックスタートには、事前に構築されたダッシュボードが含まれています。

* [Cloudflare Network Log Quick Start](https://newrelic.com/instant-observability/cloudflare/fc2bb0ac-6622-43c6-8c1f-6a4c26ab5434) in New Relic Instant の可用性に移動し、

  <DNT>
    **+ Install now**
  </DNT>

  をクリックします。

* ブログ投稿を読んで統合の詳細をご覧ください： [CloudflareLogpush統合を使用してログインサイトをNewRelicに送信するためのより高速で安価な方法](https://newrelic.com/blog/how-to-relic/cloudflare-log-integration)

* [ログ UI](/docs/logs/log-management/ui-data/use-logs-ui/)を使用して、プラットフォーム全体のログ データを調べます。

* [ログインコンテキスト](/docs/logs/enable-log-management-new-relic/configure-logs-context/configure-logs-context-apm-agents/)機能を使用してログを転送することにより、アプリケーションとプラットフォームのパフォーマンスデータの両方をより深く把握できます。

* [アラートを](/docs/alerts-applied-intelligence/new-relic-alerts/alert-conditions/create-alert-conditions/)設定します。

* [データをクエリ](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data/)し、[ダッシュボードを作成します](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/)。

## ログ転送を無効にする [#disable]

ログ転送機能を無効にするには、 [Cloudflare Logpush documentation](https://developers.cloudflare.com/logs/faq/) に記載されている標準的な手順に従ってください。New Relic では、他に何もする必要はありません。
