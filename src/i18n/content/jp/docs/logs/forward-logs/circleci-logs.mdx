CI/CDログをNewRelicに送信するように[CircleCIWebhook](https://circleci.com/docs/2.0/webhooks/)サービスを構成します。

## CircleCIでWebhookトリガーを設定する [#setup]

次の手順を実行して、ログをNewRelicに転送するCircleCIWebhookを設定します。

1. [CircleCI](https://app.circleci.com/projects)にログインします。

2. CircleCIプロジェクトの1つに移動します。

3. \[**プロジェクト設定]**をクリックします。

4. プロジェクト設定のサイドバーで、\[ **Webhook** ]をクリックします。

5. \[ **Webhookの追加]を**クリックします。

6. Webhookに名前を付けます。

7. New Relicの**ログエンドポイント**を入力してください：

   * 私たち：

     ```
     "https://log-api.newrelic.com/log/v1?Api-Key=YOUR_LICENSE_KEY"
     ```

   * EU：

     ```
     "https://log-api.eu.newrelic.com/log/v1?Api-Key=YOUR_LICENSE_KEY"
     ```

   New Relic アカウントで設定したリージョンを使用します。`YOUR_LICENSE_KEY` New Relic に置き換えます <InlinePopover type="licenseKey"/>.

8. ログプッシュをトリガーするイベントのタイプに応じて、**ワークフロー**または**ジョブ**のいずれかを選択します。

9. 受信APIまたはサードパーティサービスを設定している場合は、\[ **pingイベントのテスト**]をクリックしてテストイベントを作成します。
