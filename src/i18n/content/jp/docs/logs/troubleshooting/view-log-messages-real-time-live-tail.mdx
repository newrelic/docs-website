## 問題

新しいホストやサーバーを導入したり、アプリケーションに変更を加えたりしたとき、システムがその変更にどのように反応するかをすぐに確認したいものです。そのためには、ログメッセージをリアルタイムで確認する必要があります。

## 解決策

届いたメッセージをリアルタイムで確認するために、ライブテールログを有効にします。

1. **[one.newrelic.com> Logs](https://one.newrelic.com)** に移動します。

2. 必要なログ情報の種類に合わせてクエリを入力し、 **Query logs** をクリックします。

   ```
   Find logs where <var>annotations.restartRequested:"2020-12-23T18:00:00Z"</var>
   ```

3. **Livetail** アイコンをクリックすると、 <Icon name="fe-play-circle"/>を有効にしたり、 <Icon name="fe-pause-circle"/>を無効にしたりすることができます。
