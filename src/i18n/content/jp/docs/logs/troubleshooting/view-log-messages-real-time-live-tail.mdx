## 問題

新しいホストやサーバーを導入したり、アプリケーションに変更を加えたりしたとき、システムがその変更にどのように反応するかをすぐに確認したいものです。そのためには、ログメッセージをリアルタイムで確認する必要があります。

## 解決

届いたメッセージをリアルタイムで確認するために、ライブテールログを有効にします。

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > ログ**に移動します。

2. 必要なログ情報の種類に焦点を当てるクエリを入力し、 **Query logs** \[ログのクエリ] をクリックします。クエリの例:

   ```
   Find logs where annotations.restartRequested:"2020-12-23T18:00:00Z"
   ```

3. 右側で、 **Live tail** \[ライブテール] をクリックして有効にします。 <Icon name="fe-play-circle"/> または無効にする <Icon name="fe-pause-circle"/> それ。
