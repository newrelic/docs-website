## 問題

CloudFormation スタックを削除する場合、さまざまな理由により失敗する可能性があります。

## 解決

考えられる原因とその解決策:

* スタックによって作成された S3 バケット (Firehose エラー ログ用など) が空でない場合は、削除できません。 コンテンツをアーカイブしてから、スタックの削除を再試行してください。
