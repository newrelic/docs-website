## 問題

Lambdaインテグレーション設定を介した S3 の New Relic にログが表示されません。

## 解決

考えられる原因とその解決策:

* [CloudWatch および Lambda からのログが表示されない](/docs/logs/troubleshooting/aws-log-integration-issues/logs-not-appearing-from-cloudwatch-lambda)理由を確認します。
* S3 ファイルは、New Relic の[ログ転送制限](/docs/logs/troubleshooting/aws-log-integration-issues/log-forwarding-restrictions)に準拠していない可能性があります。
* `NewRelicServerlessLogForwarder` S3 トリガーがない可能性があります。 ネストされた CloudFormation スタックの出力を確認し、追加します。
