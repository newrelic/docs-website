このページでは、ワークフロー自動化アクション カタログで利用可能なNew Relicアクションの包括的なリファレンスを提供します。 これらのアクションにより、エージェント プラットフォームで構成されたエージェントを実行できるようになります。

## エージェントのアクション

### エージェントを実行する [#newrelic-agent-run]

エージェントティック プラットフォームで構成されたエージェントを実行します。
