Azure仮想マシン（VM）アクションを使用すると、ワークフローから直接Azure VMのライフサイクルを管理できます。

## 仮想マシンを停止する [#azure-vm-stop]

このアクションは、[Azure 計算 VM - Power Off](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines/power-off?view=rest-compute-2025-11-01&tabs=HTTP) REST APIを使用してAzure VMを停止します。

    <TabsPageItem id="azurevmstopexample">
      ```yaml
      name: azure-stop-vm
      description: 'This stops a virtual machine'
      steps:
        - name: azure_vm_stop_3
          type: action
          action: azure.vm.stop
          version: 1
          inputs:
            clientId: ${{ :secrets:azure-client-id }}
            clientSecret: ${{ :secrets:azure-client-secret }}
            tenantId: ${{ :secrets:azure-client-tenant }}
            subscriptionId: "12345678-1234-1234-1234-123456789abc"
            resourceGroupName: "my-resource-group"
            virtualMachineName: ‘my-demo-vm'
          next: end
      ```
