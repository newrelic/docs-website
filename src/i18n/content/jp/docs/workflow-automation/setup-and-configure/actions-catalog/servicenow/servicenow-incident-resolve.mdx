Table APIを使用して、ServiceNowの既存のインシデントを解決します。

認証のセットアップについては、[ServiceNow インシデントアクション](/docs/workflow-automation/setup-and-configure/actions-catalog/servicenow/servicenow-incident)を参照してください。

              ```yaml
              name: resolve-servicenow-incident
              description: to resolve a basic incident in ServiceNow
              steps:
                - type: action
                  name: resolveIncident
                  action: serviceNow.incident.resolve
                  version: 1
                  inputs:
                    domain: "your-domain"
                    apiKey: "${{ :secrets:serviceNowApiKey }}"
                    sysId: "a62bcdbd476b3610f89ff3ac416d4357"
                    closeCode: "Duplicate"
                    closeNotes: "Issue resolved by restarting the application server"
                    resolvedAt: "2024-03-11 10:30:00"
                    resolvedBy: "john.doe"
                    parameters:
                      sysparm_display_value: "true"
                  next: end
              ```
