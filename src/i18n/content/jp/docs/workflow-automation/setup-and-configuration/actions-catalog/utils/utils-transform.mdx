このページでは、ワークフロー自動化アクション カタログで使用できるユーティリティ - 変換に関する包括的なリファレンスを提供します。これらのアクションにより、データ変換ユーティリティが有効になります。

## 入力データ値をCSV形式に変換する [#utils-transform-toCSV]

これは、さまざまなタイプの入力 (JSON、マップ) を CSV 形式に変換するために使用されます。

              ```yaml
                name: nrqltocsv

                steps:
                  - name: queryForLog
                    type: action
                    action: newrelic.nrql.query
                    version: 1
                    inputs:
                      accountIds: ['${{ .workflowInputs.accountId }}']
                      query: ${{ .workflowInputs.nrql }}

                  - name: csv1
                    type: action
                    action: utils.transform.toCSV
                    version: 1
                    inputs:
                      data: ${{ .steps.queryForLog.outputs.results | tostring }}
                      selectors:
                        - name: csv
                          expression: ".csv"
              ```
