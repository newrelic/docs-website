このページでは、ワークフロー自動化アクション カタログで使用できるユーティリティ - uuid に関する包括的なリファレンスを提供します。これらのアクションにより、UUID 生成ユーティリティが有効になります。

## UUIDを生成する [#utils-uuid-generate]

RFC 準拠の V4 uuid を生成します。Temporal ワークフローの決定論的 PRNG を使用することで、ワークフロー内で安全に使用できるようになります。[ネームスペース：ワークフローを参照してください。API ](https://typescript.temporal.io/api/namespaces/workflow?&_ga=2.223881945.39829224.1686648936-167080156.1680123165#uuid4)詳細については、 「Temporal TypeScript SDK リファレンス」を参照してください 。
