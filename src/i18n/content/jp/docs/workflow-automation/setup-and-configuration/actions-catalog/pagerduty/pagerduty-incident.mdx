このページでは、ワークフロー自動化アクション カタログで使用できる PagerDuty インシデント アクションの包括的なリファレンスを提供します。これらのアクションにより、PagerDuty インシデント管理が可能になります。

## 前提条件

ワークフロー自動化で PagerDuty アクションを使用する前に、次の点を確認してください。

* 適切な権限を持つ PagerDuty アカウント。
* PagerDuty API トークン (ユーザー トークンまたはアカウント トークンのいずれか)。
* PagerDuty アカウントでインシデントを管理するために必要な権限。
* アカウント トークンの場合、関連するアクションで`from`メール アドレスを指定する必要があります。
