New Relicアクション カタログには、インフラストラクチャとインテグレーションを自動化するための 46 以上のアクションが用意されています。 cloudプロバイダー、通信ツール、 New Relicアカウントでタスクを実行するアクションをリンクして、エンドツーエンドのプロセスを調整します。

アクションは階層的な命名規則に従います。 `company.product.action`
