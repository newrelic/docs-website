## サンプルリクエスト

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## サンプル回答

クエリは次を返します:

* `schedule.id`: スケジュールの一意の識別子。
