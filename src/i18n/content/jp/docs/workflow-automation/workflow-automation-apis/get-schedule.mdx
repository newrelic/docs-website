## 例 [#example]

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## レスポンス

クエリは次を返します:

* `schedule.id`: スケジュールの一意の識別子。
