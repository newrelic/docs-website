この API は、指定されたスコープ内のすべてのワークフロー定義アウトラインのリストを取得して返します。

<Callout variant="important">
  このクエリではワークフロー定義 YAML は使用できません。ワークフロー定義 YAML は、 `GetWorkflowDefinition`クエリ`/{actor/account(:id)/workflowAutomation/workflow(:name, :version)}`を使用した場合にのみ返されます。
</Callout>

## 例

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflows(cursor: "xyz123") {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
          }
        }
      }
    }
  }
}
```

## レスポンス

クエリは次を返します:

* `workflows.results`: ワークフロー定義アウトラインの配列。各アウトラインには次の内容が含まれます。

  * `definition.name`: ワークフロー定義の名前
  * `definition.scope.id`: アカウントID
  * `definition.scope.type`: スコープタイプ
  * `definition.version`: ワークフロー定義のバージョン番号
