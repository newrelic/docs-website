## 例

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```

## レスポンス

突然変異により以下が返されます:

* `runId`: 停止されたワークフロー実行の一意の識別子。ワークフローが見つからない場合は空の`runId`を返します。
