これにより、実行中のワークフローが停止します。ワークフローが見つからない場合は、空の`runId`を返します。

## 例

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```
