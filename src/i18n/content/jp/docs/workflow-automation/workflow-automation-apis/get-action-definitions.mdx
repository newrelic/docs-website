## サンプルリクエスト

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        actionDefinitions {
          results {
            description
            inputs {
              name
              required
              secretAllowed
            }
            name
            outputs {
              name
            }
            timeout {
              action
            }
            version
          }
        }
      }
    }
  }
}
```

## サンプル回答

クエリは次を返します:

* `actionDefinitions.results`: アクション定義の配列。各定義には次の内容が含まれます。

  * `name`: アクション名（例： `aws.ec2.rebootInstances` ）
  * `description`: アクションが行うこと
  * `version`：アクションバージョン
  * `inputs`: `name` 、 `required` 、および`secretAllowed`プロパティを持つ入力の配列
  * `outputs`: `name`プロパティを持つ出力フィールドの配列
  * `timeout.action`: タイムアウト（秒）
