## 例

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```

## レスポンス

突然変異により以下が返されます:

* `definition.name`: 更新されたワークフロー定義の名前
* `definition.version`: ワークフロー定義の新しいバージョン番号
* `definition.yaml`: base64エンコードされたワークフロー定義YAML
