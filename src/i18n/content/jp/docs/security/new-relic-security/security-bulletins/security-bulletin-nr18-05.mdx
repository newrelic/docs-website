## 脆弱性情報 [#vuln-info]

New Relic の Infrastructure エージェントは、プロセスのコマンドラインを含む、実行中のプロセスに関する情報を収集します。デフォルトの設定である `strip_command_line: true` では、追加のコマンドラインパラメーターが New Relic に送信されないようになっています。一部のコマンドラインオプションでは、この設定を有効にしても、エージェントは追加のコマンドラインパラメーターを取得します。

### 緩和要因 [#factors]

* 一般的に、機密データはコマンドラインオプションには含まれません。
* Linuxシステムでは、コマンドライン引数（フラグ）にパスセパレータ要素（`/` ）を含むパスがある場合にのみ、コマンドラインパラメータが収集されます。
* Windows システムでは、コマンドラインに `/` または `-` で始まるスイッチがあり、パスセパレータ要素 (` ˶ˆ꒳ˆ˵ ) を含む場合にのみ、コマンドラインパラメータが収集されます。`
