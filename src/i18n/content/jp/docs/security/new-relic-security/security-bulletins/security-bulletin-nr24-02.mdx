<DNT>**Vulnerability Identifier:**</DNT> NR24-02

<DNT>**Priority:**</DNT> 高

## 概要

New Relic は、New Relic Salesforce Exporter を使用しているすべての顧客に対し、最近発表された[OpenSSH の脆弱なバージョン](https://nvd.nist.gov/vuln/detail/CVE-2024-6387)を排除するために New Relic がリリースしたバージョン 2.2.0 にアップデートすることを推奨しています。

## 必要なアクション

New Relic 、 New Relic Salesforce Exporter を使用している顧客に、すぐにバージョン 2.2.0 にアップデートすることを推奨しています。

New Relic は、これらの更新を支援するために次のリソースを提供しています。

* [Salesforce エクスポーターのリリースノート](https://github.com/newrelic/newrelic-salesforce-exporter/releases/tag/2.2.0)
* [ホスト上のデプロイメントの検出とアップグレード](https://github.com/newrelic/newrelic-salesforce-exporter?tab=readme-ov-file#upgrading-on-host-deployments)

顧客が New Relic Salesforce Exporter をアップグレードできない場合は、ネットワークベースの制御を通じて SSH アクセスを制限し、攻撃のリスクを最小限に抑えます。

## サポートリリースノート
