## 概要

開発者が高セキュリティモードでカスタムAPIパラメータを誤って取得することを防ぐためのPythonエージェントのセキュリティアップデートです。

**発売日：** 2018年1月9日

**脆弱性の識別子:** NR18-01

**Priority:** Medium

## 脆弱性情報 [#vuln-info]

[Python High Security Mode](/docs/agents/python-agent/configuration/python-agent-configuration#high_security) の内部監査により、エージェントがトレース API を使ってメッセージのパラメータをカスタマイズすることで、New Relic に追加のデータを送信できることが判明しました。開発者が関数トレースAPIまたはメッセージトレースAPIのいずれかを使用して追加のパラメータを渡している場合、カスタムパラメータがNew Relicに送信される可能性があります。ハイセキュリティモードが有効な場合は、これらを無効にしてください。

### 緩和要因 [#factors]

* この問題は、PythonエージェントでHigh Security Modeを有効にしているお客様で、開発者がトレースAPIに追加情報を追加している場合にのみ影響します。

## 回避策

エージェントのアップグレードができない場合は、高セキュリティモードで追加のデータを送信しないようにするために、以下のような対応が可能です。

* [ファンクショントレース API](/docs/agents/python-agent/python-agent-api/function_trace) または [メッセージトレース API](/docs/agents/python-agent/python-agent-api/message-trace) へのカスタムパラメ ータを削除します。

## セキュリティの脆弱性をNew Relicに報告 [#report]

ニューレリックは、お客様とそのデータのセキュリティを重視しています。私たちの製品やウェブサイトにセキュリティ上の脆弱性を発見したと思われる場合は、New Relic の協調的な情報開示プログラムに報告していただくことを歓迎し、大変感謝しています。詳細については、 [Reporting security vulnerabilities](/docs/security/new-relic-security/data-privacy/reporting-security-vulnerabilities) をご覧ください。
