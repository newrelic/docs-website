当社のお客様は、 [SolarWinds Security Advisory](https://www.solarwinds.com/securityadvisory) のビジネスへの影響を評価されていることと思います。当面の懸念に対処するためにNew RelicはSolarWinds Orion製品の顧客ではないため、New RelicおよびNew Relicを使用している顧客は、危険にさらされたSolarWinds製品の影響を受けません。

## New RelicとSolarWinds Orion [#statement]

New Relic Securityは、SolarWinds社の事件に関する最新の情報が発表されるのを注意深く監視しています。

先を見越した対策として、ニューレリックのセキュリティは、コードインテグリティとソフトウェアデリバリーに関するプロセスの見直しを行いました。ニューレリックは、必要に応じて適切な手順で改善を実施します。また、SolarWinds社の事件に関連する指標がないか、当社のシステムを引き続き評価していきます。

当社独自のベンダーリスク管理プログラムに基づき、ニューレリックは重要なベンダーとのコミュニケーションを開始し、そのベンダーが当社に提供するサービスのいずれかでソーラーウインズ製品を使用しているかどうかを確認しました。その回答に基づいて、ニューレリックは当社とお客様のリスクを軽減するための適切な手段を特定し、実行します。

お客様のセキュリティとプライバシーは当社の最優先事項であり、当社はソーラーウインズ社の状況を引き続き注視しています。ご質問のあるお客様は、 [support.newrelic.com/](https://support.newrelic.com/) までお問い合わせください。
