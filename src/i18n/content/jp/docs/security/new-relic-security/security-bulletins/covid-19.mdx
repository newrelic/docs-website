この文書では、ニューレリックの現在の運営計画と、お客様のために製品やサービスを稼働させ続けるための取り組みについて説明しています。

## New Relic COVID-19 事業継続のための対応と処置 [#statement]

COVID-19に関連する現在の出来事を踏まえ、当社スタッフの健康と幸福を維持するため、またニューリリックが提供するソフトウェアおよびサービスの提供に影響を与える可能性がある場合の対応についてお知らせいたします。また、当社およびお客様のビジネスへの影響を未然に防ぐための当社の取り組みについてもご報告いたします。お客様のアプリケーションやサービスのパフォーマンスを向上させるために、私たちに信頼を寄せていただいていることに大変感謝しています。

当社はSaaSをルーツとする企業であり、当社の社員はクラウドネイティブプラットフォームの運用の基礎となる技術や慣習に慣れています。私たちのサービスはインターネット上で提供され、多くの場合、お客様の管理下にある環境やシステムに依存しています。New Relic のサイトエンジニアリングチームは、アップタイムとスムーズなカスタマーエクスペリエンスの両方を実現するために、モニタリングの強化と改善に取り組んでいます。私たちは、リアルタイムのデータと深い可視性によって、製品が稼働し続け、サポートを受けられるようにしています。

## 地域社会への貢献 [#community]

食料品店、医療関係のウェブサイト、ビデオ会議ツール、オンライン教育アプリ、メディア、ストリーミングビデオプラットフォーム、エクササイズアプリなど、企業は突然、これまで以上に迅速なスケールアップを迫られています。多くの企業が、オンラインサービスやシステムに予想外のプレッシャーを感じており、中には通常のスタッフの一部しかいない企業もあります。New Relicのアプリケーションとサービスは、企業が "過去最大の日々 "を過ごすのに欠かせないものとなっています。それがNew Relicの存在理由です。

そのため、私たちはさまざまな方法で支援に乗り出しています。例えば、 [、既存の非営利製品寄付プログラムを拡大することを発表しました。](https://blog.newrelic.com/product-news/new-relic-covid-19-relief-program/) 、COVID-19の救援活動を行っているすべての組織が、90日間、ニューレリックのプラットフォームを無料で使用することができますが、危機の期間に応じて延長することができます。

## 社員とビジネスを大切にする [#business]

従業員への影響を最小限に抑えるために、ニューレリックの危機管理チームは様々なポリシーとアクションを実施しました。具体的には、リスクが大幅に軽減されたと判断されるまで、全社員の出張を停止しました。また、すべての会議や予定されていた社内会議を完全なバーチャル形式に移行しました。また、ビジネスサポートや製品の提供など、事業のあらゆる面での継続的な運営を確保するため、事業継続計画に沿って、世界各地のオフィスを一時的に閉鎖し、「在宅勤務ポリシー」を実施しています。

グローバルに事業を展開しているNew Relicでは、従業員がセキュリティを確保しながらリモートで仕事をし、ビジネスやお客様をサポートするために必要なサービスにアクセスできるよう、強固なプログラムを導入しています。このプログラムでは、VPNのキャパシティを拡大することで、エンジニアの能力を向上させ、完全なリモートワークを可能にしました。

私たちは、お客様がご自身のシステムのパフォーマンスを維持するために必要な機能を継続的に提供していくことを再確認しました。私たちは、お客様が必要なときに必要なサポートを受けられるようにしています。また、お客様と緊密に連携し、お客様がこの時期のパフォーマンス管理に備えることができるようにしています。
