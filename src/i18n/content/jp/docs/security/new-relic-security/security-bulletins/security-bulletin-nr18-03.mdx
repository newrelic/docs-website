## 脆弱性情報 [#vuln-info]

New Relic SyntheticsのPrivate Locationイメージに、CPUの脆弱性「Meltdown」に対応するパッチが適用されました。Spectre に対応するパッチは、Ubuntu からまだリリースされていません。

さらに、お客様は自分のプライベートな場所にアクセスしてインプレース・アップグレードを実行することができます。

```
sudo apt-get update
sudo apt-get dist-upgrade
```

そして、そのマシンを再起動します。

追加情報： [Ubuntu Advisory](https://wiki.ubuntu.com/SecurityTeam/KnowledgeBase/SpectreAndMeltdown)

### 緩和要因 [#factors]

* プライベートロケーションはお客様のデータセンター内に設置され、内部の信頼できるWebアプリケーションを監視します。

## 回避策

既知の回避策はありません。
