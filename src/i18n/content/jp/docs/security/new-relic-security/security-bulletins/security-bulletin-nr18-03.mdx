## 脆弱性情報 [#vuln-info]

Meltdown CPUの脆弱性から保護するために、合成モニタリングを使用したプライベートロケーションイメージがパッチで更新されました。 Spectreに対処するパッチはまだUbuntuからリリースされていません。

さらに、お客様は自分のプライベートな場所にアクセスしてインプレース・アップグレードを実行することができます。

```
sudo apt-get update
sudo apt-get dist-upgrade
```

そして、そのマシンを再起動します。

追加情報： [Ubuntu Advisory](https://wiki.ubuntu.com/SecurityTeam/KnowledgeBase/SpectreAndMeltdown)

### 緩和要因 [#factors]

* プライベートロケーションは顧客のデータセンター内で実行され、内部の信頼できる Web アプリケーションとして監視されます。

## 回避策

既知の回避策はありません。
