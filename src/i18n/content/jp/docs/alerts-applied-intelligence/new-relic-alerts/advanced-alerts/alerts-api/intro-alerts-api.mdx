New Relicのアラート機能と適用されたインテリジェンス機能をプログラムで管理するためにAPIを使用するには、次の2つのオプションがあります。

* NerdGraph（GraphQL）：これは、データのクエリと構成を行うためのNewRelicの主要で推奨されるAPIです。 NerdGraphには常に機能を追加しています。 [アラート関連のNerdGraphドキュメントを参照してください。](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/alerts-nerdgraph/nerdgraph-api-examples)
* REST API：これはNewRelicのオリジナルAPIです。このAPIの作業は最小限であるため、NerdGraphの使用をお勧めします。現在RESTAPIを使用している場合は、NerdGraphに切り替えることをお勧めします。 [アラート関連のRESTAPIドキュメントを参照してください。](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/rest-api-alerts/rest-api-calls-alerts)
