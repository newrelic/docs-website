New Relic Pathpoint は、ほぼすべてのNew Relicアカウントと統合して、包括的なビジネス プロセスの監視を提供できます。 始めるには、組織に関連するテレメトリーデータを入手することが重要です。 これには、メトリクス、イベント、ログ、またはトレースが含まれます。 APMイベントとログはPathpointの一般的な開始点ですが、テレメトリーデータの選択は完全にあなた次第です。

Pathpoint を使い始める前に、New Relic アカウントにインストールする必要があります。

## New Relic カタログから Pathpoint をインストールする [#install-catalog]

パスポイントをインストールするには:

<Steps>
  <Step>
    <DNT>**[Apps & Visualizations](https://onenr.io/0yw40Vz6Ew3)**</DNT>に移動します。
  </Step>

  <Step>
    <DNT>**Pathpoint V2**</DNT>を検索します (新しいコードベースを取得するには`V2`を指定する必要があります)。 古いバージョンは非推奨になりました。
  </Step>

  <Step>
    確認済みの<DNT>**Pathpoint V2**</DNT>のインストレーション アイコンをクリックし、次に<DNT>**Add this app**</DNT>をクリックします。

    <img
      title="Install Pathpoint"
      alt="Install Pathpoint"
      src="/images/pathpoint_v2_screenshot_install_pathpoint.webp"
    />

    Pathpoint が New Relic アカウントに追加され、 <DNT>**Apps**</DNT>からアクセスできるようになりました。
  </Step>
</Steps>

Pathpoint をインストールすると、インタラクティブな UI を使用するだけで、ステージ、ステップ、シグナルを構成できます。 JSON 設定をエクスポートおよびインポートすることもできます。これは、設定の複製やバージョン管理、およびNew Relicアカウント間での設定の移動に役立ちます。 [Pathpoint の設定について詳しくは、こちらをご覧ください](/docs/new-relic-solutions/business-observability/intro-pathpoint/)。
