これで、New Relic を使い始める準備が整いました。いくつかのオプションがあります:

* **New Relic をすばやくセットアップしたいですか?**データの流れをすばやく取得し、New Relic でできることを確認したい場合は[、クイック起動ガイド](/docs/new-relic-solutions/get-started/quick-launch-guide)を確認するか[、インストールする特定のツールを参照してください](https://newrelic.com/instant-observability)。
* **完全なスタック可観測性を解き放つ準備はできましたか?**あなたは正しい場所にいます。あなたが読んでいるガイドは、チームが構築できる強力な基盤を提供します。このガイドの手順は、長期にわたる多くのお客様にとって有益であることが証明されています。
