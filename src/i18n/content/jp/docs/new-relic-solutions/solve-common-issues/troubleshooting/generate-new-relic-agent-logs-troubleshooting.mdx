[](https://newrelic.com/instant-observability/)New Relic のソリューションの中には、エージェントを手動でインストールしなければならないものがあります。ほとんどのエージェントには、トラブルシューティング、監査、診断のための詳細なログを生成するツールが組み込まれています。

関連ドキュメント：

* 一般的なエージェントのトラブルシューティングについては、 [データが表示されない](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data) を参照してください。
* About [New Relic Diagnostics](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics): 一般的な問題を自動的に検出するユーティリティーです。
