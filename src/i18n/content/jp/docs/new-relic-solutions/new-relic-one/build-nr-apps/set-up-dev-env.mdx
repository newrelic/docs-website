カスタムアプリケーションを構築したり、オープンソース アプリの 1 つを変更したりすることに決めた場合は、いくつかの重要なツールが必要です。

* [New Relic コマンドラインインターフェース (CLI)](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==) 。
* CLI をダウンロードすると取得できる API キー。

アプリで何をしたいかに応じて、追加のセットアップと設定が必要になる場合があります。 このガイドの内容は次のとおりです。

* New Relic CLIをダウンロードしてアプリを構築または変更する
* このウェブサイトにコンテンツを投稿する

## あなたが始める前に

始める前に、まず[権限](/docs/new-relic-solutions/new-relic-one/build-nr-apps/permission-manage-apps)についてお読みになることをお勧めします。

構築を開始するには、次のものが必要です。

* [Github アカウント](https://github.com/)アカウント。 アプリの構築に必ずしも必要ではありませんが、GitHub アカウントが[あれば、オープンソース アプリをダウンロードしてカスタマイズしたり](https://opensource.newrelic.com/explore-projects)、[オープンソース プロジェクト](https://opensource.newrelic.com/contributing/)に貢献したりすることができます。
* [New Relic 開発者アカウント](https://newrelic.com/signup/?partner=Developer+Edition)。 まだお持ちでない場合は、New Relic アプリケーションを開発するための無料トライアル アカウントを取得できます。
* [.npm です](https://www.npmjs.com/get-npm)。 [Node.js](https://nodejs.org/en/download/)をインストールしている場合は、JavaScript コードを共有、再利用、更新するために使用される`npm`が既に存在します。 これは、New Relic アプリとこの Web サイトのフレームワークである React コンポーネントを操作するために必要です。

### サポートに関する注意事項

New Relic アプリケーションの構築は、他の JavaScript/React アプリケーションの構築と同じです。 当社では、構築ツール ( [CLI](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==)および[SDK ライブラリ)](/docs/new-relic-solutions/build-nr-ui/sdk-component/intro-to-sdk)に関するサポートを提供しています。 ただし、基本的な JavaScript または React コーディングに関する質問や問題についてはサポートを提供していません。

ビルドに関するよくある質問と回答については、 [New Relic での構築に関する Explorers Hub ページを](https://discuss.newrelic.com/c/build-on-new-relic)参照してください。

<Callout variant="tip">
  [New Relic VSCode 拡張機能](https://marketplace.visualstudio.com/items?itemName=new-relic.nr1)または[New Relic VSCode 拡張パック](https://marketplace.visualstudio.com/items?itemName=new-relic.new-relic-extension-pack)を使用してアプリを構築します。
</Callout>

## アプリの構築または変更の準備

<Steps>
  <Step>
    ### CLIとAPIキーをダウンロードする

    [Build New Relic applications page](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==) \[New Relic アプリケーションのビルド ページ]で、**Quick start** \[クイック スタート]手順を完了してダウンロードします。

    これらの 6 つのクイック スタート手順を実行すると、アプリの開発に使用するAPIキーと、アプリの構築と に使用するNew Relic CLI を取得できます。 クイック スタートの最後には、次の内容で構成されるプロジェクトが完成します。

    * **Nerdpack** \[オタクの集団]。 アプリケーションに必要なすべてのファイルが含まれるパッケージ。 アプリを構築するためにカスタマイズする 2 種類のファイル (Nerdlets とランチャー) が含まれています。
    * 1 つ以上の**Nerdlet**ファイル。 特定の UI ビューまたはウィンドウ。 Nerdlet は、 `index.js`ファイル、 `stylesheet` 、JSON 形式の`config`ファイルを含む React JavaScript パッケージです。 あらゆる JS 機能 (チャート、インタラクティブ フィールド、ツールチップなど) を含めることができます。
    * `launcher`ファイル: これはランチャーの基礎であり、アプリを公開した後に New Relic からアプリケーションを開くために使用されます。
  </Step>

  <Step>
    ### 構築を開始する

    コーディングの準備ができたら、Nerdpack に`cd`して開始してください。

    アプリケーションの構築について詳しく知りたい場合は、次のステップバイステップ ガイドをお試しください。

    * [「Hello, World!」アプリケーションの構築では、](/docs/new-relic-solutions/tutorials/build-hello-world-app)小さなアプリケーションを作成し、それを New Relic に公開し、アカウントをサブスクライブして他のユーザーと共有する方法を説明します。
    * [地域ごとのページビューのマップでは、](/docs/new-relic-solutions/tutorials/map-pageviews-region/)人気のオープンソース アプリの 1 つを作成する手順を説明します。 アプリにカスタム クエリを追加してテーブルに表示し、そのデータをマップに追加する方法を学びます。
  </Step>
</Steps>

## `developer.newrelic.com`サイトに貢献する

このサイトはオープンソースですので、皆様のご意見をお聞かせください。 修正方法がわかっている間違いを見つけた場合は、プルリクエストを作成してください。 改善してほしいコンテンツのギャップを見つけた場合は、GitHub の問題をお送りください。 または、共有したいガイドがある場合は、まったく新しいガイドを作成してください。 方法については、以下をお読みください。

<Steps>
  <Step>
    ### `developer-website` GitHub リポジトリをフォークする

    リポジトリをフォークすると、 `developer.newrelic.com`ファイルの独自のコピーで作業し、[ローカルでサイトを構築できる](https://github.com/newrelic/developer-website#developernewreliccom)ようになります。 また、受信したプルリクエストをより簡単に管理できるようになります。

    1. GitHub の[開発者向け Web サイト](https://github.com/newrelic/developer-website)ページに移動します。
    2. ページの右上にある**Fork** \[フォーク]ボタンを選択します。
    3. フォーク先のアカウントを選択し、フォークが作成されるまで数秒待ちます。

    [定期的に同期し](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/syncing-a-fork)て、メイン ブランチの上流への変更や追加をフォークに反映させて最新の状態に保ちます。
  </Step>

  <Step>
    ### 機能やドキュメントのリクエストをする

    1. 任意のページで、ページ上部にある**GitHub**ボタンを選択します。
    2. 必要な変更の種類を選択します。
    3. GitHub フォームに記入してください。
  </Step>

  <Step>
    ### 新しいガイドを投稿する

    プロセスを順を追って説明する[寄稿者向けガイドライン](https://github.com/newrelic/developer-website/blob/master/CONTRIBUTING.md#creating-new-pages)をご覧ください。
  </Step>
</Steps>
