2014年にリリースされたNewRelicInsightsは、カスタムクエリ、チャート、ダッシュボードを作成するための独自の方法でした。 [New Relicプラットフォーム](/docs/new-relic-one/use-new-relic-one/get-started/introduction-new-relic-one)により、データにアクセス、分析、視覚化するためのエクスペリエンスが最新化されました。 New Relicは、改善されたチャートとダッシュボードエクスペリエンスを提供し、新しいイノベーションをより迅速に提供できるプラットフォームを提供します。

この移行ガイドは、あなたが理解するのに役立ちます。

* アップグレードされたNewRelicチャート、ダッシュボード、クエリで得られる新機能と改善された機能にはどのようなものがありますか
* 改善されたNewRelicプラットフォームに簡単に移行できる理由
* 切り替え時に知っておくべきことと注意点
* 改善されたNewRelicプラットフォームを最大限に活用する方法

## 機能 [#features]

トランジションの詳細までスクロールダウンできますが、最初に、NewRelicダッシュボードがInsightsダッシュボードよりも明らかに改善されていることを示すいくつかの機能を追加しました。

### 問い合わせ能力の向上 [#queries]

改善されたプラットフォームを使用すると、次のことが可能になります。

* **同じウィジェットから多くのアカウントをクエリする機能**：新しいプラットフォームを使用すると、関連付けられているすべてのアカウントを1か所でクエリできます。
* **クエリとチャートのエクスペリエンスの向上：** New Relicのどこにいても、クエリアクセスはグローバルに利用できます。 [NewRelicでデータを参照およびクエリする](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data/)方法を学びます。
* **クエリエクスペリエンスの向上：** [`Metric`データ型](/docs/data-ingest-apis/get-data-new-relic/metric-api/view-query-you-metric-data)と[メトリックタイムスライスデータ](/docs/query-data/nrql-new-relic-query-language/nrql-query-tutorials/query-metric-timeslice-data-nrql)の両方をクエリできます。
* **簡単なカスタマイズ：** すべてのビジュアライゼーションは、クエリーにアクセスできるようになりました。NRQLクエリを変更するだけで、キュレーションされたチャートを増強することができます。

### ビジュアライゼーションの向上 [#dashboard-options]

豊富な可視化オプションを選択できるだけでなく、ダッシュボードに追加することもできます。

* **表示オプションの改善：**密集した線の多いグラフ以外の視覚化を使用して、データを理解しやすくします。私たちの新しいプラットフォームは、より良い[TVモード](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard#dash-visual-tv)も提供します。
* **ファセットリンク**: ファセット化された属性でダッシュボードをフィルタリングできるので、ダッシュボードをよりインタラクティブで使いやすくすることができます。また、 [ケースのサポートもあります](/docs/query-your-data/explore-query-data/dashboards/filter-new-relic-one-dashboards-facets/#facet-linking-cases). [詳細はこちら](/docs/query-your-data/explore-query-data/dashboards/filter-new-relic-one-dashboards-facets).
* **エリア内でより多くのチャートやウィジェットを表示：** Insightsでは、3つのチャートを表示することが制限されていました。ダッシュボード上に最大12個まで表示できるようになり、データの密度が高まり、ツールチップやチャート間のトラッキングも改善されました。
* **複数ページのダッシュボードの簡単な作成：**インサイトでは、これらを[データアプリ](/docs/dashboards/new-relic-one-dashboards/manage-your-dashboard/create-dashboards-have-multiple-pages)と呼びます。 Insightsデータアプリは、複数ページのダッシュボードとして保持されます。
* **グラフの一貫性と柔軟性：**ダッシュボードには、ウィジェット間でのファセットの色の一貫性と、パフォーマンスの高いダッシュボードの読み込み時間の短縮が含まれます。また、ダッシュボードに任意のグラフタイプを追加できます。

New Relic Insights UIは、長年にわたってユーザーに役立ってきましたが、さらに優れたエクスペリエンスを提供するときが来ました。ぜひご参加ください。

## 移行を成功させるためのステップ [#switch]

移行には、UIとモバイルアプリのエクスペリエンス（2021年4月12日）とダッシュボードAPI（2021年7月）の2つの部分があります。
