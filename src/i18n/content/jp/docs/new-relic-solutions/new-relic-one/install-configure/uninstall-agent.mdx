[](https://newrelic.com/instant-observability/)New Relic ソリューションの中には、エージェントのインストールを必要とするものがあります。エージェントがデータを報告しないようにしたい場合は、エージェントをアンインストールすることができます。New Relic を完全にアンインストールする必要がある場合は、お使いのエージェントの手順に従ってください。

問題が発生した場合は、 [support.newrelic.com](https://support.newrelic.com) をご覧ください。

## アンインストールの前に [#downgrade]

<Callout variant="important">
  有料の New Relic アカウントの場合、アンインストールの代わりに [アカウントをダウングレードする方法がありますのでご注意ください](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) より低価格または無料のティアにダウングレードしてください。
</Callout>

## APMのアンインストール [#language-agents]

APM エージェントをアンインストールしたり、スタートアップスクリプトに変更を加えたりした後は、アプリ（またはアプリが設置されているアプリサーバーやウェブサーバー）を再起動する必要があります。このようにしなければならないのは、New Relic のエージェントは通常、アプリを実行しているプロセスのメモリ内で実行されるからです。 これらのファイルを削除するだけで、 **現在実行中のアプリケーションがNew Relicにデータを送信するのを停止することはできません。**
