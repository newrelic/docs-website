[](https://newrelic.com/instant-observability/)New Relic の統合の中には、エージェントを手動でインストールする必要があるものがあります。これらのエージェントの動作をどのように設定するかは、特定のエージェントによって異なります。

## APMエージェントの設定 [#apm-config]

APM エージェントに必要な設定は、 [ライセンスキー](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) と [アプリ名](/docs/apm/new-relic-apm/installation-configuration/name-your-application) のみで、これらは [エージェントのインストール時に設定されます](/docs/agents/manage-apm-agents/installation/install-agent) 。

その他の設定オプションについては、以下を参照してください。

* [C SDK](/docs/agents/c-sdk/install-configure/c-sdk-configuration)
* [Go](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [Java](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.NET](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## ブラウザエージェントの設定 [#browser-config]

[ブラウザ監視の設定については、弊社のドキュメントをご覧ください](/docs/browser/new-relic-browser/configuration).

## インフラストラクチャエージェントの設定 [#infra-config]

インフラストラクチャ・モニタリング・エージェントに必要な唯一の設定は、 [のライセンス・キー](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) です。これは、 [エージェントをインストールする際に設定されます](/docs/agents/manage-apm-agents/installation/install-agent) 。また、 [追加の設定オプション](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent) を使用して、インフラを監視することもできます。

## モバイルエージェントの設定 [#mobile-config]

モバイルモニタリングエージェントで唯一必要な設定は、 [アプリケーショントークン](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token) です。これは、 [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android) 、 [iOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) 、 [tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/tvos/tvos-installation-configuration) のアプリにモバイルモニタリングエージェントをインストールする際に設定します。

その他の設定オプションについては、以下を参照してください。

* [Androidの構成](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [iOSの構成](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)
* [tvOSの設定](/docs/mobile-monitoring/new-relic-mobile-ios/tvos/tvos-installation-configuration)

## 他のNew Relicツールの設定 [#other]

ここに記載されていない他のツールの設定については、特定のソリューションの [ドキュメントを参照してください](https://newrelic.com/instant-observability/) 。
