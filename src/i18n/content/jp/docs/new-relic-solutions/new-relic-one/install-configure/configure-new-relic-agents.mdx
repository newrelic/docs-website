[](https://newrelic.com/instant-observability/)New Relic の統合の中には、エージェントを手動でインストールする必要があるものがあります。これらのエージェントの動作をどのように設定するかは、特定のエージェントによって異なります。

## APMエージェントの設定 [#apm-config]

<InlinePopover type="apm"/>エージェントに必要な設定は[、ライセンスキー](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key)と[アプリ名](/docs/apm/new-relic-apm/installation-configuration/name-your-application)のみです。これらは、[エージェントのインストール](/docs/agents/manage-apm-agents/installation/install-agent)時に構成されます。

その他の設定オプションについては、以下を参照してください。

* [Go](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [Java](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.NET](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## ブラウザエージェントの設定 [#browser-config]

[<InlinePopover type="browser"/>設定](/docs/browser/new-relic-browser/configuration)についてはドキュメントをご覧ください。

## インフラストラクチャエージェントの設定 [#infra-config]

インフラストラクチャエージェントに必要な設定は<InlinePopover type="licenseKey"/>のみです。 これは[エージェントをインストールする](/docs/agents/manage-apm-agents/installation/install-agent)ときに構成されます。 [追加の設定オプション](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent)を使用してインフラストラクチャを監視することもできます。

## モバイルエージェントの設定 [#mobile-config]

モバイル監視エージェントに必要な設定は、 [アプリケーション トークン](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token)のみです。 これは、 [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android)、 [iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) アプリ用のモバイル監視エージェントをインストールするときに構成されます。

その他の設定オプションについては、以下を参照してください。

* [Androidの構成](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [iOS/tvOS 構成](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)

## 他のNew Relicツールの設定 [#other]

ここに記載されていない他のツールの設定については、特定のソリューションの [ドキュメントを参照してください](https://newrelic.com/instant-observability/) 。
