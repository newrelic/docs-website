New Relicにようこそ開始しますか？ 適切な場所にいます。

## 何を監視しますか？ [#what]

監視対象をご覧になりますか？ロゴをクリックして開始してください。アカウントをお持ちでない場合 [New Relicアカウントにサインアップ](https://newrelic.com/signup)するよう求められます。サインアップすると、UIがお客様に適したインストールフローへご案内します。

<Callout variant="important">
  EUデータセンターをご利用の場合は、[ここから開始してください](https://one.eu.newrelic.com/nr1-core?state=cab9400b-31f1-fab7-cd28-db21bd312951)。
</Callout>

## 監視するインテグレーションをさらにお探しですか？ [#io]

<img
  title="Instant observability landing page"
  alt="A screenshot of the Instant Observability landing page"
  src={installNrQuick}
/>

当社の[インテグレーションカタログ](https://newrelic.com/instant-observability/)では、New Relicで監視できるすべてのものを紹介しています。それだけでなく、**無償のNew Relicアカウントにサインアップ**したら、[+インストールのクイックスタート](https://newrelic.com/signup)をクリックすると、適切なインストールフローにファネルします。
