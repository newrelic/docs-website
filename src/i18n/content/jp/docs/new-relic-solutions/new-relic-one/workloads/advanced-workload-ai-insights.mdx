<Callout variant="important">
  AI インサイトは、特定の可用性要件を伴う**パブリック プレビュー段階**にあります。
</Callout>

AI インサイトは、ワークロードの健全性、パフォーマンスの傾向、潜在的な問題に関する AI を活用した概要を提供します。

## 要件 [#requirements]

* インテリジェントなワークロードが作成
* 十分な観測データ
* **AWS インフラストラクチャのみ**(Azure では利用できません)
* **FedRAMP カスタマーは利用できません**

## インサイトの生成 [#generate]

1. インテリジェントなワークロードを開く
2. クリック <DNT>**Generate insights**</DNT>
3. AIが生成した要約を確認する

Insights は、ワークロード ページに表示されるデータのみを分析し、オンデマンドで生成されます。

## インサイトの内容 [#what-included]

**健康の概要:**

```
Your workload is healthy with 12 entities reporting normal status.
Response times are within expected ranges.
```

**パフォーマンスの傾向:**

```
Throughput increased 23% in the past hour while maintaining stable
response times.
```

**エンティティのハイライト:**

```
- payment-service: Response time +40ms (within SLA)
- user-db: Connection pool at 78%, approaching threshold
- checkout-api: Error rate below 0.1%, optimal
```

**推奨事項：**

```
Investigate database connection pool trend. While not critical,
usage may require scaling if traffic continues growing.
```

## ベストプラクティス [#best-practices]

* ワークロードを初めて表示するとき、またはアラート後にインサイトを生成する
* AI の観測結果を実際のメトリクスと照合して検証する
* インサイトを分析の代わりとしてではなく出発点として使用する
* 大きな変化の後に再生する

## 制限 [#limitations]

* 現在表示されているデータのみを分析します
* 他のワークロードやアカウントにアクセスできない
* 根本原因分析ではなく観察結果を提供する
* 複雑なパターンを誤って解釈することがある

<Callout variant="caution">
  行動を起こす前に必ず AI インサイトを実際のデータと照合してください。
</Callout>

## トラブルシューティング [#troubleshooting]

### ボタンが無効です

**理由:**

* Azure インフラストラクチャ (サポートされていません)
* FedRAMP アカウント（AI 機能は無効）
* お使いのアカウントではご利用いただけない機能です

**解決策:**アカウント チームに連絡して、インフラストラクチャと機能へのアクセスを確認してください。

### 生成に失敗する

**理由:**

* データが不十分
* 一時的な接続の問題
* 複雑なワークロードデータ

**解決策:**しばらく待ってから再試行してください。問題が解決しない場合は、サポートにお問い合わせください。
