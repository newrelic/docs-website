## `nr1 plugins:install` [#nr1-pluginsinstall]

### プラグインをインストールする [#nr1-pluginsinstall-install]

CLI にプラグインをインストールします。 npm または Git URL からプラグインをインストールできます。

プラグインをインストールするとコアプラグインが上書きされることに注意してください。

たとえば、 `hello`コマンドを持つコア プラグインがある場合、 `hello`コマンドを含むプラグインをインストールすると、コア プラグインの実装がオーバーライドされます。 これは、CLI 全体にパッチを適用して更新せずに、コア プラグインの機能を更新する場合に便利です。

### 使用方法 [#nr1-pluginsinstall-usage]

`$ nr1 plugins:install PLUGIN`

### 引数 [#nr1-pluginsinstall-arguments]

* `PLUGIN`: インストールするプラグインの名前、パス、または URL。 これは**必須**です。

### オプション [#nr1-pluginsinstall-options]

<table>
  <tbody>
    <tr>
      <td>
        `-f, --force`
      </td>

      <td>
        `yarn install --force`を実行します。 これにより、すべてのプラグインのパッケージが強制的に再ダウンロードされます。
      </td>
    </tr>

    <tr>
      <td>
        `-h, --help`
      </td>

      <td>
        CLI ヘルプを表示します。
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        出力に追加情報を追加します。
      </td>
    </tr>
  </tbody>
</table>

### 例 [#nr1-pluginsinstall-examples]

* `$ nr1 plugins:install myplugin`
* `$ nr1 plugins:install https://github.com/someuser/someplugin`
* `$ nr1 plugins:install someuser/someplugin`

### エイリアス [#nr1-pluginsinstall-aliases]

* `$ nr1 plugins:add`

## `nr1 plugins:link` [#nr1-pluginslink]

### プラグインをリンクする [#nr1-pluginslink-link]

開発用にローカル プラグインを CLI にリンクします。

プラグインをリンクすると、ユーザーがインストールしたプラグインまたはコア プラグインが上書きされることに注意してください。

たとえば、 `hello`コマンドを持つユーザーがインストールしたプラグインまたはコア プラグインがある場合、プラグインを`hello`コマンドにリンクすると、ユーザーがインストールしたプラグインまたはコア プラグインの実装がオーバーライドされます。 これは開発作業に役立ちます。

### 使用方法 [#nr1-pluginslink-usage]

`$ nr1 plugins:link PLUGIN`

### 引数 [#nr1-pluginslink-arguments]

* `PLUGIN`: リンクするプラグインの名前、パス、または URL。 これは**必須**です。

### オプション [#nr1-pluginslink-options]

<table>
  <tbody>
    <tr>
      <td>
        `-h, --help`
      </td>

      <td>
        CLI ヘルプを表示します。
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        出力に追加情報を追加します。
      </td>
    </tr>
  </tbody>
</table>

### 例 [#nr1-pluginslink-examples]

* `$ nr1 plugins:link myplugin`
* `$ nr1 plugins:link someuser/someplugin`
