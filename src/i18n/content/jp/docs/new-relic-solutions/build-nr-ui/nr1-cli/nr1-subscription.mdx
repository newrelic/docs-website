## `nr1 subscription:list` [#nr1-subscriptionlist]

### サブスクリプションを確認する [#nr1-subscriptionlist-see]

あなたのアカウントが購読しているすべての Nerdpack を一覧表示します。 アカウントは API キーにリンクされています。

### 使用方法

`$ nr1 subscription:list`

### オプション

<table>
  <tbody>
    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        使用する認証プロファイル。
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        出力に追加情報を追加します。
      </td>
    </tr>
  </tbody>
</table>

## `nr1 subscription:set` [#nr1-subscriptionset]

### Nerdpackを購読する [#nr1-subscriptionset-subscribe]

アカウントを特定の Nerdpack とチャンネルに登録します。 このコマンドは、Nerdpack UUID または特定の Nerdpack フォルダー内で実行できます。 デフォルトでは、コマンドは`package.json`の Nerdpack ID を使用し、STABLE チャネルをサブスクライブします。

1 つのアカウントで一度に購読できる Nerdpack とチャンネルは 1 つだけです。

### 使用方法 [#nr1-subscriptionset-usage]

`$ nr1 subscription:set`

### オプション [#nr1-subscriptionset-options]

<table>
  <tbody>
    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        サブスクライブする Nerdpack を指定します。 デフォルトでは、コマンドは`package.json`のものを使用します。
      </td>
    </tr>

    <tr>
      <td>
        `-c, --channel=DEV/BETA/STABLE`
      </td>

      <td>
        サブスクライブするチャネルを指定します。 \[ `default: STABLE` ]
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        使用する認証プロファイル。
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        出力に追加情報を追加します。
      </td>
    </tr>
  </tbody>
</table>

### エイリアス [#nr1-subscriptionset-aliases]

* `$ nr1 nerdpack:subscribe`

## `nr1 subscription:unset` [#nr1-subscriptionunset]

### Nerdpack の購読を解除する [#nr1-subscriptionunset-unsubs]

特定の Nerdpack からアカウントの登録を解除します。 このコマンドが Nerdpack フォルダー内で実行されると、デフォルトで`package.json`の Nerdpack ID が使用されます。

### 使用方法 [#nr1-subscriptionunset-usage]

`$ nr1 subscription:unset`

### オプション [#nr1-subscriptionunset-options]

<table>
  <tbody>
    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        サブスクライブする Nerdpack を指定します。 デフォルトでは、コマンドは`package.json`のものを使用します。
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        使用する認証プロファイル。
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        出力に追加情報を追加します。
      </td>
    </tr>
  </tbody>
</table>

### エイリアス [#nr1-subscriptionunset-aliases]

* `$ nr1 nerdpack:unsubscribe`
* `$ nr1 subscription:delete`
* `$ nr1 subscription:remove`
* `$ nr1 subscription:rm`
