表グラフを作成します。 データは、特定のアカウントに対して NRQL クエリを実行することによって取得することも、 `data`プロパティを介して渡すこともできます。

`data`形式は、それぞれ`metadata`と`data`を含む一連のオブジェクトです。 各`data`には、この時点からの値と、チャートをプロットするために使用される`x`キーと`y`キーの両方が含まれます。

### 使用方法

```js
import { TableChart } from 'nr1'
```

### 例

#### ベーシック

```js
<TableChart accountIds={[1]} query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType" />
```

#### 複数のアカウントの場合

```js
<TableChart accountIds={[1, 1067061]} query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType" />
```

#### コンテナを埋める

```js
<TableChart
  accountIds={[1]}
  query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType"
  fullWidth
  fullHeight
/>
```

#### カスタムデータ付き

```js
function render() {
  const data = [
     {
       metadata: {
         id: 'series-1',
         name: 'Serie 1',
         color: '#008c99',
         viz: 'main',
         columns: ['jobType', 'count'],
       },
       data: [
         {
           jobType: 'SIMPLE',
           count: 18,
         },
         {
           jobType: 'SCRIPT_BROWSER',
           count: 27,
         },
         {
           jobType: 'SCRIPT_API',
           count: 33,
         },
         {
           jobType: 'BROWSER',
           count: 29,
         },
         {
           jobType: 'SELENIUM_JSON',
           count: 22,
         },
         {
           jobType: 'SCRIPT',
           count: 31,
         },
       ],
     },
   ];;


  return <TableChart data={data} fullWidth />;
}
```
