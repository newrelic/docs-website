`<Popover>`コンポーネントの子要素。

ポップオーバー オーバーレイの開閉を制御します。

### 使用方法

```js
import { PopoverTrigger } from 'nr1'
```

        このコンポーネントは、子 React 要素を直接渡すか、レンダリング コールバック (子として機能) を渡すかの 2 つの方法でレンダリングできます。React 要素を直接渡す場合は、 `<TextField>` 、 `<Button>`などの互換性のある NR1 SDK コンポーネントを使用するか、独自のコンポーネントを使用できます。レンダリング コールバックを使用すると、引数`{ opened: boolean, controlled: boolean }`として次の形状のオブジェクトを受け取ります。コンポーネントを直接渡すかレンダリング コールバックを使用するかに関係なく、独自のコンポーネントをトリガーとしてレンダリングする場合は、適切に動作するために常に`focus`メソッドを公開する必要があります。
