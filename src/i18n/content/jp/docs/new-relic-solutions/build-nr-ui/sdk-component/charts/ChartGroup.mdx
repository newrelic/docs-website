さまざまな種類のグラフをグループ化して、X 軸に従って針を移動したり、対応するシリーズのツールヒントを表示したり、すべてのグラフで同時に選択されている領域をグレー表示したりするなどのアクションをまとめて実行できるようにします。

このコンポーネントは React フラグメントのように動作し、DOM に追加のタグをレンダリングしません (その中のすべてのチャートのグループ化のみを有効にします)。 つまり、このコンポーネントは他のコンポーネントとは異なり、 `className`または`style`タグを受け入れません。

### 使用方法

```js
import { ChartGroup } from 'nr1'
```

### 例

#### ベーシック

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart style={chartStyle} accountIds={[1]} query={query} />
      <LineChart style={chartStyle} accountIds={[1, 1067061]} query={query} />
      <ScatterChart style={chartStyle} accountIds={[1]} query={query} />
    </ChartGroup>
  );
}
```

#### コンテナを埋める

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
      <LineChart
        style={chartStyle}
        accountIds={[1, 1067061]}
        query={query}
        fullWidth
        fullHeight
      />
      <ScatterChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
    </ChartGroup>
  );
}
```
