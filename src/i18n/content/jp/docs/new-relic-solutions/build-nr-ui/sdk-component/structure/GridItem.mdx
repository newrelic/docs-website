`<Grid>`コンポーネントの子要素。

デフォルトでは 1 列にまたがりますが、 `columnSpan`または`columnStart`と`columnEnd`プロパティを設定することで、またがる列の数を指定できます。

### 使用方法

```js
import { GridItem } from 'nr1'
```
