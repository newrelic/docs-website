ヘッダーを構成するセルのラッパー。 ヘッダーは常に存在し、 `<Table>`要素の最初の直接の子である必要があります。

### 使用方法

```js
import { TableHeader } from 'nr1'
```

      <td>
        `<TableHeaderCell>`の形の列ヘッダーのリスト。 セルはヘッダーの直接の子である必要があります。つまり、 `<TableHeader>`と`<TableHeaderCell>`の間にノードがあってはなりません。
