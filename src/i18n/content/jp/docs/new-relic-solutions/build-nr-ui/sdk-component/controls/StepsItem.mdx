`<Steps>`コンポーネントの子要素。

### 使用方法

```js
import { StepsItem } from 'nr1'
```

### 小道具

<table>
  <tbody>
    <tr>
      <td>
        `ariaLabel` <h5>ストリング</h5>
      </td>

      <td>
        スクリーン リーダーに手順を説明するラベルを提供します。 これは、スクリーン リーダーのラベルが`label`プロパティと異なる場合にのみ必要です。
      </td>
    </tr>

    <tr>
      <td>
        `checked` <h5>ブール値</h5>
      </td>

      <td>
        `true`の場合、ステップを完了としてマークします。
      </td>
    </tr>

    <tr>
      <td>
        `children` <h5>必須</h5><h5>ノード</h5>
      </td>

      <td>
        アイテム内でレンダリングするコンテンツ。
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>ストリング</h5>
      </td>

      <td>
        コンポーネントにクラス名を追加します。配置と間隔の目的にのみ使用してください。
      </td>
    </tr>

    <tr>
      <td>
        `disabled` <h5>ブール値</h5>
      </td>

      <td>
        ユーザーがラベルをクリックしてステップを開けないようにするには、disabled を使用します。
      </td>
    </tr>

    <tr>
      <td>
        `expanded` <h5>ブール値</h5>
      </td>

      <td>
        true の場合、ステップの内容が表示されます。
      </td>
    </tr>

    <tr>
      <td>
        `label` <h5>必須</h5><h5>文字列</h5>
      </td>

      <td>
        ステップのラベル。
      </td>
    </tr>

    <tr>
      <td>
        `onClick` <h5>機能</h5>
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>物体</h5>
      </td>

      <td>
        カスタム スタイル設定用のインライン スタイル。配置と間隔設定の目的でのみ使用してください。
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>ストリング</h5>
      </td>

      <td>
        `data-test-id`属性を追加します。 ユニット テストと E2E テストでコンポーネントをターゲットにするために使用します。テスト ID を有効にするには、nerdpack ID をプレフィックスとして付け、その後にドットを付けます。たとえば、 `my-nerdpack.some-element` 。 **注:** `data-test-id` 属性はDOMから削除されているため表示されない場合があります。デバッグするには、URL に `e2e-test` クエリ引数を渡します。
      </td>
    </tr>

    <tr>
      <td>
        `value` <h5>どんな</h5>
      </td>

      <td>
        アクティブなステップを識別するために`<Steps>`によって使用されます。
      </td>
    </tr>
