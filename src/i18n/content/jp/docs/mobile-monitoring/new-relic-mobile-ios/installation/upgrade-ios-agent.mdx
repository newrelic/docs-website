最新バージョンについては、 [リリース ノート](/docs/releases/ios)を参照してください。

## Swift Package Manager で更新する [#SPM]

Swift Package Manager を使用して New Relic iOS エージェントを更新するには:

1. Xcode プロジェクト ナビゲータのパッケージ依存関係セクションで

   <DNT>
     **NewRelic**
   </DNT>

   エージェントに移動します。

2. <DNT>
     **NewRelic**
   </DNT>

   エージェントを右クリックします。

3. <DNT>
     **Update Package**
   </DNT>

   を選択します。

<Callout variant="important">
  常に [リリース ノート](/docs/releases/ios) をチェックして、最新のエージェント バージョンを取得していることを確認してください。Swift Package Manager ルールの設定方法によっては、 `Up to Next Major` または `Branch` 設定を変更して新しいバージョンを含める必要がある場合があります。
</Callout>

## ココアポッドでアップデート [#Cocoa-Pods]

Cocoa Pod を使用して New Relic iOS エージェントを更新するには:

1. <DNT>
     **Terminal**
   </DNT>

   アプリを開き、PodFile があるディレクトリに移動します。

2. 次のコマンドを実行してください。

   ```shell
   pod update NewRelicAgent
   ```

## iOS エージェントを手動で置き換える [#Manual]

New Relic iOS エージェントをダウンロードして手動でインストールするには:

1. [https://download.newrelic.com/ios_agent/](https://download.newrelic.com/ios_agent/) にアクセスしてください更新先の iOS エージェントをダウンロードします。

2. iOS エージェントがダウンロードされた場所に移動し、解凍します。

3. Xcode (CMD+1) でプロジェクト ナビゲーターを開き、 `NewRelic.xcframework`を検索します。

4. `NewRelic.xcframework`を右クリックまたは Control キーを押しながらクリックし、

   <DNT>
     **Show in Finder**
   </DNT>

   を選択します。

5. 新しくダウンロードした iOS エージェントを古いエージェントと同じフォルダーにドラッグし、古いエージェントを置き換えることを選択します。
