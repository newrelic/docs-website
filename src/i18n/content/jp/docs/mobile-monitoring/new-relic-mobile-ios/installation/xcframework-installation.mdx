iOS 監視を設定するには、[ガイド付きインストールを](https://onenr.io/0qwLv87gkj5)使用することをお勧めします。 ただし、エージェントを手動でインストールする必要がある場合は、以下の手順に従って New Relic iOS エージェント (XCFramework パッケージ内) をインストールし、XCode プロジェクトに追加します。

<Steps>
  <Step>
    [iOS エージェント リリース ノート](/docs/release-notes/mobile-release-notes/ios-release-notes)から最新の iOS エージェント (XCFramework) をダウンロードしてください。
  </Step>

  <Step>
    iOS エージェントがダウンロードされた場所に移動し、解凍します。
  </Step>

  <Step>
    Xcode プロジェクトで、解凍した XCFramework を追加します。

    1. プロジェクト ナビゲータを右クリックし、 **Add Files to `[YourProjectName]`** \[`[YourProjectName]`にファイルを追加]をクリックします。
    2. `NewRelic.xcframework`ディレクトリを選択し、 **Add** \[追加] をクリックします。
  </Step>

  <Step>
    iOS エージェントを埋め込む:

    1. プロジェクト ナビゲータで、プロジェクトとターゲットの設定を開きます。
    2. エージェントを追加するターゲットをクリックします。
    3. **General** \[全般]タブをクリックします。
    4. **Frameworks, Libraries, and Embedded Content** \[フレームワーク、ライブラリ、埋め込みコンテンツ]セクションまで下にスクロールします。
    5. `NewRelic.xcframework`がここにリストされていることを確認してください。 そうでない場合は、 **+**ボタンをクリックして追加します。
  </Step>
</Steps>
