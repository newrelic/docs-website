New Relic の iOS SDK API をインストールするには、プロジェクトのプレフィックスヘッダーを変更する必要があります。プロジェクトにプレフィックスヘッダーがない場合は、以下の手順で追加してください。

1. Xcode メニューから **File> New> File... を選択します。**

2. **iOS** テンプレートのオプションから、 **Other> PCH file** を選択します。

3. ファイル名 **&lt; target_name>-Prefix.pch** を選択し、 **Create** を選択します。

4. ターゲットの **Build settings** から、 **All** を選択して、 `Prefix Header` フィールドに以下を追加します。

   ```
   $(SRCROOT)/<target_name>-Prefix.pch
   ```
