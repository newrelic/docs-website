## 構文

```
currentSessionId; 
NewRelic.currentSessionId()
```

現在のセッションのIDを返します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

現在のセッション ID を返します。このメソッドは、単一のセッション定義と識別子に基づいて、アプリのデータ（New Relic データに限らず）のモニタリングを統合する場合に便利です。例えば、マーケティング分析やユーザー分析に同じ識別子を使用したい場合などです。

このAPIの使用方法については、カスタム属性やイベントをInsights forに送信する際のドキュメントを参照してください。

* [Objective-C](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#objc-custom-att-events)
* [Swift](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#swift-custom-att-events)

## 戻り値

現在のセッションのID文字列を返します。

## 例

### Objective-C [#obj-c]

**方法です。**

```
+ (NSString*) currentSessionId;
```

**例:**

```
NSString* sessionId = [NewRelic currentSessionId()];
```

### Swift

**方法です。**

```
NewRelic.currentSessionId() -> String
```

**例:**

```
let sessionId = NewRelic.currentSessionId()
```
