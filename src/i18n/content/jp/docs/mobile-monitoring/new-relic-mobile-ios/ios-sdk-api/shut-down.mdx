## 構文

```
+ (void)shutdown;
```

実行時に、現在のアプリケーション ライフサイクル内でエージェントをシャットダウンします。

## 要件

エージェント バージョン 7.4.4 以降。

## 説明

この API は、実行時に現在のアプリケーション ライフサイクル内でエージェントをシャットダウンする機能を提供します。エージェントがシャットダウンすると、現在のアプリケーション ライフサイクル内で再起動することはできません。

## 例

### Objective-C [#obj-c]

**方法です。**

```
+ (void) shutdown();
```

**例：**

```
[NewRelic shutdown];
```

### 迅速 [#swift]

**方法です。**

```
NewRelic.shutdown() -> Void
```

**例：**

```
NewRelic.shutdown()
```
