## 構文

```
- (void) disableFeatures:(NRMAFeatureFlags)featureFlags;
```

機能を無効にするには、このビット単位のフラグを設定します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

`disableFeatures` API を使用して、実行したくない機能を無効にすることができます。無効にできるフラグの詳細については、 [iOS エージェントの構成と機能フラグを](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)参照してください。

<Callout variant="important">
  設定を有効にするには、 `startWithApplicationToken:`を呼び出す前にこの関数を呼び出す必要があります。
</Callout>

## 例

### Objective-C [#obj-c]

**方法です。**

```
+[NewRelic disableFeatures:(NRMAFeatureFlags)flags];
```

**例：**

Objective-C の例:

```
[NewRelic disableFeatures:NRFeatureFlag_NSURLSessionInstrumentation | NRFeatureFlag_CrashReporting]; 
```

迅速な例:

```
 NewRelic.disableFeatures([ NRMAFeatureFlags.NRFeatureFlag_CrashReporting, NRMAFeatureFlags.NRFeatureFlag_WebViewInstrumentation ])
```
