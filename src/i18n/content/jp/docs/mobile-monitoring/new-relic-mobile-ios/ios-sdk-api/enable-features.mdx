## 構文

```
- (void) enableFeatures:(NRMAFeatureFlags)featureFlags;
```

機能を有効にするには、このビット単位のフラグを設定します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

New Relic iOS エージェントのほとんどの機能は、デフォルトで有効になっています。デフォルトでは有効になっていない機能を有効にする場合は、 `enableFeatures` API を使用できます。デフォルトで有効になっている機能の詳細については、 [iOS エージェントの構成と機能フラグを](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)参照してください。

<Callout variant="important">
  設定を有効にするには、 `startWithApplicationToken:`を呼び出す前にこの関数を呼び出す必要があります。
</Callout>

## 例

### Objective-C [#obj-c]

**方法です。**

```
+[NewRelic enableFeatures:(NRMAFeatureFlags)flags];
```

**例：**

Objective-C の例:

```
[NewRelic enableFeatures:NRFeatureFlag_SwiftInteractionTracing | NRFeatureFlag_FedRampEnabled];
```

迅速な例:

```
 NewRelic.enableFeatures([ NRMAFeatureFlags.NRFeatureFlag_SwiftInteractionTracing, NRMAFeatureFlags.NRFeatureFlag_FedRampEnabled ])
```
