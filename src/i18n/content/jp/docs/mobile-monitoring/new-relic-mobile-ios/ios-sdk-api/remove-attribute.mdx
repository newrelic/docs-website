## 構文

```
removeAttribute:(NSString*)name;
NewRelic.removeAttribute()
```

指定されたセッション属性を削除します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

このメソッドは、名前の文字列で指定された属性をセッションから削除します。

このAPIの使用方法については、カスタムアトリビュートとイベントの送信に関するドキュメントを参照してください。

* [Objective-C](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#objc-custom-att-events)
* [Swift](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#swift-custom-att-events)

## 戻り値

成功した場合は`true`を返し、失敗した場合は`false`を返します。

## 例

### Objective-C [#obj-c]

**方法です。**

```
+ (BOOL) removeAttribute:(NSString*)name;
```

**例：**

`NSString*`で指定されたセッション属性を削除します;たとえば、 `rate` 。

```
BOOL attributeRemoved = [NewRelic removedAttribute@"rate"];
```

### Swift

**方法です。**

```
NewRelic.removeAttribute(name: String!) -> Bool
```

**例：**

`rate`という名前のセッション属性を削除します：

```
let attributeRemoved = NewRelic.removedAttribute("rate")
```
