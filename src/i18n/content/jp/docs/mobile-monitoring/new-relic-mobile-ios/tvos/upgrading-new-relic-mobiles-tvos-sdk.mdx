tvOS エージェントのインストール、設定、アップグレードを行うには、アカウント管理者である必要があります。最新バージョンの情報は、 [リリースノート](/docs/releases/ios) を参照してください。

## コンテンツ [#qiklinks]

## tvOSフレームワークの入れ替え [#v1]

管理者の方へ新しいバージョンのtvOS SDKにアップグレードする前に、以前のバージョンのtvOSエージェントフレームワークを交換する必要があります。

![tvOSのフレームワークを交換するためのワークフローです。](./images/Mobile_tvOS_replace-framework_1.png "Mobile_tvOS_replace-framework.png")

<figcaption>
  ここでは、既存のtvOSエージェントフレームワークを削除して、新しいバージョンに置き換えられるようにするためのワークフローの例をご紹介します。
</figcaption>

1. XcodeのProject Navigator（**CMD 1** ）から、 **NewRelicAgentTVOS.framework** を検索します。
2. **NewRelicAgentTVOS.framework** を右クリックまたはcontrolクリックして、 **Show in Finder** を選択します。
3. **NewRelicAgentTVOS.framework** をゴミ箱にドラッグします。
4. Xcodeプロジェクトで、 **NewRelicAgentTVOS.framework** への参照が赤でハイライトされていることを確認します。
5. **NewRelicAgentTVOS.framework** を右クリックまたはcontrolクリックして、 **Delete** を選択し、廃止された参照をプロジェクトから削除します。
6. tvOSアプリ監視のための標準的なインストール手順に従ってください。
