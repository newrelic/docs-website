iOSエージェントは、アプリのパフォーマンスを包括的に表示します。エージェントはObjective-C、Swift、またはその両方で記述されたiOSアプリを監視します。モバイルモニタリング機能の一般概要については、[モバイルモニタリングの概要](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile)を参照してください。

## 互換性および要件 [#requirements]

iOSエージェントをインストールする前に、[互換性と要件](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements/)を確認してください。

## エージェントのインストール [#installation]

iOSエージェントをインストールするには、UIに直接表示される[ガイド付きインストール](https://onenr.io/0qwLv87gkj5)に従ってください。この方法では、Swift Package Manager（SPM）またはCocoaPodsを使用して、iOSエージェントをインストールおよび設定できます。

## インストレーションのトラブルシューティング [#troubleshooting]

iOSのインストレーションに問題がある場合、またはiOSアプリのモバイルモニタリングUIにデータが表示されない場合は、次の[トラブルシューティングソリューション](/docs/mobile-monitoring/new-relic-mobile-ios/troubleshoot/no-data-appears-ios)を試してみてください。
