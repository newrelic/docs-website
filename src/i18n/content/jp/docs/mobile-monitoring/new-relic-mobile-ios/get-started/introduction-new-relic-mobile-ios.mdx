iOS用New Relicモバイルモニタリングは、アプリのパフォーマンスを包括的に把握します。これは、Objective-C、Swift、または両方の言語で記述されたiOSアプリに動作します。モバイルモニタリング機能の一般概要については、[モバイルモニタリングの概要](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile)を参照してください。

## 互換性および要件 [#requirements]

iOSエージェントをインストールする前に、[互換性と要件](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements)を確認してください。

## エージェントのインストール [#installation]

インストレーションプロセスの一環として、New Relicは[アプリケーショントークン](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token)を自動的に生成します。これは、監視する各モバイルアプリを認証するための40文字の16進数の文字列です。

iOSエージェントをインストールするには、次の2つの方法があります。

* **[CocoaPods のインストール](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/cocoapods-installation-configuration)メソッドを使用して、iOS エージェントをインストールおよび設定することをお勧めします**。
* iOSエージェントは、[New Relicフレームワークをご使用のアプリに追加する](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/ios-installation-configuration)ことにより、手動でインストールして設定することもできます。
