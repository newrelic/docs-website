iOS用New Relic Mobileはご使用のモバイルアプリケーションを監視し、アプリケーションのパフォーマンスを包括的に把握します。これは、Objective-C、Swift、または両方の言語で記述されたiOSアプリで動作します。New Relic Mobile機能の一般概要については、「[New Relic Mobileの概要](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile)」を参照してください。

## 互換性と要件 [#requirements]

iOS用New Relic Mobileをインストールする前に、[互換性と要件](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements)を確認してください。

## エージェントのインストール [#installation]

インストレーションプロセスの一環として、New Relic Mobileは[アプリケーショントークン](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token)を自動的に生成します。これは、New Relic Mobileで監視する各モバイルアプリケーションを認証するための40文字の16進数の文字列です。

iOS用New Relic Mobileのインストール方法は2通りあります。

* New Relicは、[CocoaPodsインストレーション](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/cocoapods-installation-configuration)方法を使用して、iOS用New Relic Mobileをインストールして設定することを**推奨**します。
* iOS用New Relic Mobileは、[New Relicフレームワークをご使用のアプリに追加する](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/ios-installation-configuration)ことにより、手動でインストールして設定することもできます。

iOS用New Relic Mobileがデータをレポートしない場合、これらの[推奨するトラブルシューティング手順](/docs/mobile-monitoring/new-relic-mobile-ios/troubleshoot/no-data-appears-ios)をお試しください。

## その他のヘルプ [#more_help]

さらに支援が必要な場合は、これらのサポートと学習リソースを確認してください：

* [Explorers Hub](https://discuss.newrelic.com/)を参照して、コミュニティから支援を受け、ディスカッションに参加してください。
* [当社のサイトで回答を見つけ、サポートポータルの使用方法について学びます](/docs/using-new-relic/welcome-new-relic/get-started/find-help-use-support-portal)。
* Linux、Windows、およびmacOSのトラブルシューティングツールである[New Relic Diagnosticsを実行します](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics)。
* New Relicの[データセキュリティ](/docs/security)と[ライセンス](/docs/licenses)ドキュメントを見直してください。
