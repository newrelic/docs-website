古いバージョンのAndroidを監視する必要がある場合は、レガシーバージョンのAndroidエージェントを使用することができます。

* 4.0未満のAndroidバージョンをサポートするには、Android SDKのバージョン [5.19.1](/docs/release-notes/mobile-release-notes/android-release-notes/android-5191-0) またはそれ以前のバージョンをご利用ください。これをレガシーバージョンと呼びます。
* 特にAndroid 2.2に対応するためには、Android SDKのバージョン [4.244.0](/docs/release-notes/mobile-release-notes/android-release-notes/android-24440) またはそれ以前のバージョンを使用してください。

**推奨：** これらのレガシーバージョンのAndroidを監視する必要がない場合は、代わりにエージェントの最新バージョンをインストールまたは [アップグレードしてください](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/upgrade-new-relic-mobiles-android-sdk) 。最新バージョンを使用することで、アプリのパフォーマンス、クラッシュ、インタラクションに関する豊富なデータを得ることができます。

## 古いAndroidバージョンに対応したインストール [#installation]

以前にAndroid SDKのバージョンをインストールしたことがある場合は、 を最新バージョンにアップグレードする前に、 [の手順に従ってください。その後、Androidエージェントの標準的な ](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/upgrade-new-relic-mobiles-android-sdk#android_upgrade_1)[インストールおよび設定手順](/docs/mobile-monitoring/new-relic-mobile-android/install-configure) 。
