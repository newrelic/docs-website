## 構文

```
NewRelic.crashNow()
NewRelic.crashNow(string $message)
```

New Relicのクラッシュレポートをテストするために、java.lang.RuntimeExceptionというデモ用のランタイム例外を投げます。

## 要件

Agentのバージョン4.87.0以上。

## 説明

このメソッドでは、 `java.lang.RuntimeException` という名前のデモ用ランタイム例外をメッセージ付きで投げることができます。このクラッシュは、 [**Crash Analysis** ページ](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes) に表示されます。
