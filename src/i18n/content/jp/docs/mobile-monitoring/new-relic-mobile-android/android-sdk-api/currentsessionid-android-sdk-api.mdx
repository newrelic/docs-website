## 構文

```
NewRelic.currentSessionId()
```

現在のセッションのIDを返します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

現在のセッション ID を返します。このメソッドは、単一のセッション定義と識別子に基づいて、アプリのデータ（New Relic データに限らず）のモニタリングを統合する場合に便利です。例えば、マーケティング分析やユーザー分析に同じ識別子を使用したい場合などです。

この API の使用方法については、 [Insights へのカスタム属性とイベントの送信](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api#attributes-events-insights) を参照してください。

## 戻り値

現在のセッションのID文字列を返します。

## 例

### セッションIDを返す

```
String sessionId = NewRelic.currentSessionId();
```
