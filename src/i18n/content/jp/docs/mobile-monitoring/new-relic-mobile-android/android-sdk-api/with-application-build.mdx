## 構文

```
NewRelic.withApplicationBuild(string $buildId)
```

インタラクションに関連付けられているカスタムビルド識別子文字列を設定します。

## 要件

Agentバージョン5.1.0以上。

## 説明

この呼び出しにより、 [**クラッシュレポート** ページ](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/mobile-apps-crash-report-dashboard#details) のアプリケーションバージョンの横に表示されるカスタムビルド識別子文字列を設定することができます。 `versionCode` `AndroidManifest.xml` で定義されている文字列を使用する代わりに、 `withApplicationBuild()` メソッドを呼び出して、カスタムビルドIDを文字列として渡します。

<Callout variant="important">
  **Deprecated**: New Relic Android エージェントのバージョン 5.3.0 以降、 `withApplicationBuild()` は `NewRelic.withBuildIdentifier()` に置き換わりました。
</Callout>

本APIの一般的な使用方法については、 [Android SDK API usage guide](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) をご参照ください。

## 例

### カスタムビルド識別子の設定

カスタムビルドの識別子を設定する。

```
NewRelic.withApplicationBuild('156')
```
