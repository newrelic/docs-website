## 構文

```
NewRelic.withInteractionTracing(boolean $tracingFlag)
```

インタラクション・トレースをグローバルに無効化／有効化することができます。

## 要件

Agentバージョン5.4.0以上。

## 説明

`withInteractionTracing()` を使用して、インタラクショントレースをグローバルに有効または無効にします。インタラクショントレースは、 [**Interactions** ページ](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/interactions-page) に表示されます。New Relic エージェントの開始コールの前にこれを呼び出すようにしてください。

<Callout variant="important">
  インタラクションのトレースは、エージェントの起動後に無効になります。エージェントの開始前に実行されたアクティビティやメソッドのインタラクションは、 [インタラクションページ](/docs/mobile-monitoring-ui/interactions-dashboard) に表示されます。
</Callout>

本APIの一般的な使用方法については、 [Android SDK API usage guide](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) をご参照ください。

## 例

### インタラクションの無効化

Androidエージェントが収集するすべてのインタラクショントレースをオフにします。

```
NewRelic.withInteractionTracing(false)
```
