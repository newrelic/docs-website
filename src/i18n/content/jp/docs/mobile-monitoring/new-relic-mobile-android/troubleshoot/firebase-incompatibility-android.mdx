## 問題

NewRelicAndroidエージェントとGoogleの[FirebasePerformanceMonitoring SDK](https://firebase.google.com/docs/perf-mon)の両方がアプリに含まれている場合、競合、クラッシュ、またはエラーが表示されます。

既知のエラー：

* `java.lang.StackOverflowError`
* `java.lang.RuntimeException: An error occurred while executing doInBackground()`

## 解決

AndroidエージェントとFireBasePerfMonを同時に有効にすることはできません。この問題を回避するために、どちらか一方のみを有効にすることをお勧めします。

さらにサポートが必要な場合は、 [support.newrelic.com](https://support.newrelic.com) でサポートを受けてください。
