最新版のAndroidエージェントをご利用いただくには、 [Androidエージェントのリリースノート](/docs/release-notes/mobile-release-notes/android-release-notes) をご参照ください。Androidアプリのアップグレード方法は、現在使用しているAndroidエージェントSDKのバージョンによって異なります。

## Android SDKバージョン2または3からのアップグレード [#android_upgrade_2]

以前に Android SDK のバージョン 2 または 3 をインストールした場合: **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Mobile > (アプリを選択) > Settings > Upgrade に移動します。**

## Android SDKバージョン1へのアップグレード [#android_upgrade_1]

以前に Android SDK のバージョン 1 をインストールしている場合は、 [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > に従ってください。

        **AndroidManifest.xml ファイルを開き、 `NewRelicApplication`クラスを削除します。**

        **nrandroid-setup**を初めて実行したときに、 `NewRelicApplication`クラスが`AndroidManifest.xml`ファイルに追加されました。これは New Relic SDK から削除され、不要になりました。

        既存の`Application`サブクラスがあり、それを`NewRelicApplication`から継承するように変更した場合は、サブクラスを更新して`Application`から継承してください。

        モバイルビルドの手順を呼び出すために既存のMaven構成を変更した場合は、その変更を削除して、改訂されたビルド手順を使用してください。
