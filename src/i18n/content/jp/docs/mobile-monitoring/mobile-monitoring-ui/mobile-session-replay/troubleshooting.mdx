これは、New Relicセッションリプレイの問題をご自身で診断するための最初のステップです。

問題がここにリストされていない場合は、いつでも[New Relic サポート](https://support.newrelic.com)にお問い合わせください。

## 一部のイベントは古くなったため、タイムラインやイベントトレイルに表示されなくなっている可能性があります。 [#aged-out-events]

セッションリプレイは、ハンドルされた例外イベントよりもデータ保持期間が長くなっています。他のモバイルイベントは含まれているものの、イベントトレイル内に処理された例外イベントがない、またはエラーマーカーとして表示されていないリプレイが見つかる場合があります。それらのイベントは保持期間を過ぎており、リプレイに表示されないためです。
