<InlinePopover type="mobile"/>機能には、アプリケーションのバージョン間の採用とパフォーマンスの比較分析が含まれます。これには、次の項目による上位バージョンも含まれます。

* インタラクションタイム
* アクティブセッション
* エラー率

<DNT>**Versions**</DNT>ページには、作成日、平均メモリ、平均 CPU、1 分あたりの平均セッション数、アクティブ アプリあたりの 1 分あたりの平均リクエスト数 (RPM) で各バージョンを比較した表も含まれています。 特定のバージョンに関する追加の詳細をドリルダウンすることもできます。

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Usage > Versions:**</DNT> <DNT>**Versions**</DNT>分析には、モバイル アプリの使用状況を示す色分けされたグラフと、モバイル アプリのバージョンと、メモリ、CPU、アクティブ ユーザー、ネットワーク RPM（個々のリクエスト数）の平均をまとめた表が含まれています。
</figcaption>

<DNT>**Versions**</DNT>ページには、検出されたアプリのすべてのバージョンのリストと、過去 7 日間にアクティブだったすべてのバージョンの概要情報が表示されます。 比較分析を表示するには:

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Usage > Versions**
   </DNT>

   に移動します。

2. 期間を選択するには、New Relic メニューバーの下にある

   <DNT>
     [**time picker**](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker)
   </DNT>

   を使用します。

3. オプション:

   <DNT>
     **Sort by**
   </DNT>

   オプションを選択します。

4. 特定の

   <DNT>
     **version**
   </DNT>

   の詳細のみを表示するには、その名前を選択します。

<DNT>**Versions**</DNT>ページには、選択した期間内にアクティブであったアプリのすべてのバージョンのリストが表示されます。 詳細情報をドリルダウンするには、該当する[プラットフォーム UI 機能を](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)使用します。

## ドリルダウンの詳細表示 [#details]

<img
  title="screen versions details.png"

/>

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Usage > Versions > (selected version):**</DNT> 選択したバージョンの詳細の例を次に示します。
</figcaption>

詳細ページでは、選択されたバージョンが参照バージョン（最近のバージョンや人気のあるバージョン）や、アプリの他のバージョンの平均値と比較して、さらに詳しい情報が表示されます。時系列では、エラーレート、応答時間、アクティブセッション、メモリ使用量の比較が表示されます。

詳細ページを終了するには、 <DNT>**Close**</DNT> ( <DNT>**X**</DNT> ) ボタンを選択します。

## バージョン番号の取得方法 [#how-obtain-version]

モバイルモニタリングがバージョン番号を取得する方法は、プラットフォームによって異なります。

<CollapserGroup>
  <Collapser
    id="android-ver-num"
    title="Android"
  >
    [Android エージェント](/docs/mobile-monitoring/new-relic-mobile-android/get-started/android-installation-configuration)は、マニフェストの`android:versionName`プロパティからバージョン情報を取得します。
  </Collapser>

  <Collapser
    id="ios-ver-num"
    title="iOS"
  >
    iOS エージェントは、 `CFBundleShortVersionString`と`CFBundleVersion`の両方のプロパティを使用してアプリのバージョンを取得します。エージェントは、iOS API を介してこれらのプロパティにアクセスします。`info.plist`ファイルを読み取って取得するわけではありません。
  </Collapser>
</CollapserGroup>
