\[概要] ページには、モバイル アプリの動作スナップショットが表示されます。 すべてのグラフ、表、統計には、指定された[時間枠](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker)とアプリのバージョンのデータが表示されます。

/>

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app):**</DNT> 概要ページには、高レベルのパフォーマンス データと、特定のメトリックや問題にドリルダウンするためのランディング ベースが提供されます。
</figcaption>

## サマリーページの表示 [#summary-page]

1. [one.newrelic.com > All capabilities \[すべての機能\]](https://one.newrelic.com/all-capabilities)> Mobile \[モバイル]に移動します。
2. モバイルアプリを選択します。 デフォルトで「概要」ページがロードされます。

## (推奨) モバイル監視アラートを設定する [#alerts]

ネットワーク リクエストの遅さやクラッシュ率の増加など、モバイルの主要な問題についてアラートを作成することをお勧めします。 \[概要] ページからこれらのアラートを簡単に設定するには:

1. <DNT>**Add recommended conditions**</DNT>をクリックします。

   <img
     width="80%;"
     style={{align: "left"}}
     title="Mobile Summary page"
     alt="Screenshot of the New Relic mobile Summary page"
     src="/images/mobile_screenshot-full_mobile-alert-conditions.webp"
   />


2. 推奨されるアラート条件をいくつか選択します。

   <img
     width="70%;"
     style={{align: "left"}}
     title="Mobile recommended alert conditions"
     alt="Mobile recommended alert conditions"
     src="/images/mobile_screenshot-crop_mobile-add-alert.webp"
   />

3. <DNT>**Next**</DNT>をクリックします。

4. 通知を受け取るポリシーを選択します。 既存のポリシーを選択することも、 [新しいポリシーを作成することも](/docs/alerts-applied-intelligence/new-relic-alerts/alert-policies/create-edit-or-find-alert-policy/#alert-policy-name)できます。

5. <DNT>**Activate conditions**</DNT>をクリックします。
