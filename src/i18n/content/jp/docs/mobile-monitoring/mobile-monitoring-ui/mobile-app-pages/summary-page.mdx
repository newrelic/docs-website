## サマリーページの表示 [#summary-page]

1. \*\* [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)> Mobile に移動します。
2. モバイルアプリを選択します。 デフォルトで「概要」ページがロードされます。

## (推奨) モバイル監視アラートを設定する [#alerts]

ネットワーク リクエストの遅さやクラッシュ率の増加など、モバイルの主要な問題についてアラートを作成することをお勧めします。 \[概要] ページからこれらのアラートを簡単に設定するには:

1. **Add recommended conditions**\[推奨条件を追加]をクリックします。

   <img
     width="80%;"
     style={{align: "left"}}
     title="Mobile Summary page"
     alt="Screenshot of the New Relic mobile Summary page"
     src={mobileAlertConditions}
   />


2. 推奨されるアラート条件をいくつか選択します。

   <img
     width="70%;"
     style={{align: "left"}}
     title="Mobile recommended alert conditions"
     alt="Mobile recommended alert conditions"
     src={mobileAddAlert}
   />

3. **Next**\[次へ]をクリックします。

4. 通知を受け取るポリシーを選択します。 既存のポリシーを選択することも、 [新しいポリシーを作成することも](/docs/alerts-applied-intelligence/new-relic-alerts/alert-policies/create-edit-or-find-alert-policy/#alert-policy-name)できます。

5. **Activate conditions**\[条件を有効にする]をクリックします。
