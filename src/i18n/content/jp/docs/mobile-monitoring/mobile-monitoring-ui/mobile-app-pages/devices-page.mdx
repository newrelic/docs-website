デバイスページ<InlinePopover type="mobile"/> iPad、iPhone、iPod Touch、Android タブレットなど、モバイル アプリケーションを使用している上位のデバイスに関するパフォーマンスの詳細が表示されます。グラフでは、次の基準でデバイスが比較されます。

* インタラクションタイム
* HTTPリクエスト時間
* エラー率とネットワーク障害
* アクティブなユーザーセッション

ここから、特定のモデル（例えば、iPhone 6、6S、7）ごとに詳細を掘り下げることができます。

ユーザーのモバイルデバイスに関するパフォーマンスの詳細を表示するには、次のようにします。

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (アプリを選択) > App > Devices**に移動します。
2. モバイルアプリのバージョンを選択するか、期間を変更するには、New Relic メニュー バーの下にある **Versions** \[バージョン] メニューと [**time picker** \[時間ピッカーを\]](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) 使用します。
3. オプションです。 **\[Sort by]** と **\[Hide]&lt; \[1% throughput]** オプションを選択します。
4. 特定のデバイスに詳細を限定するには **タイプ** （例：iPad）、その名前を選択します。

## ドリルダウンの詳細表示 [#details]

詳細情報にドリルダウンするには、New Relic の標準 [ユーザー インターフェイス機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)のいずれかを使用します。加えて：

* 特定のデバイスやモデル（例えば、iPad mini、iPad Airなど）のリストを表示するには、 **タイプ** （例えば、iPad）を選択します。
* 特定のデバイスやモデルの詳細を表示するには、展開されたリストからデバイス名を選択します。
* 遅いトランザクション (可能な場合) のトレース詳細を表示するには、そのリンクを選択します。詳細については、 [「インタラクション」ページ](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/interactions-page)を参照してください。
* メインの「デバイス」ページに戻るには、 **閉じる** (**X**) ボタンを選択します。
