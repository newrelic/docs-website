最新の Android または iOS リリースでアプリがどのように動作するか知りたいですか? OS バージョン ページを使用して、さまざまなリリース (メジャー リリースとマイナー リリースを含む) 間のアプリのパフォーマンスを分析および比較し、アクティブなデバイスの数、ネットワーク障害、HTTP リクエスト時間などのメトリクスを表示します。

## OS バージョンのページを表示する [#view-os-versions-page]

1. \*\* [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)> Mobile に移動します。
2. モバイルアプリを選択します。
3. 左側のメニューで**OS versions** \[OS バージョン]をクリックします。

<img
  title="OS versions page"
  alt="Use this page to view, sort, or drill down into detailed information about the top five types of operation system versions using your mobile app."
  src={mobileOsVersionsSummary}
/>

## 特定のメトリクスにドリルダウンする [#drill-down]

インタラクション時間の増加など、データのスパイクが発生した場合は、タイムピッカーをクリックしてドラッグします。 ページが更新され、その時間セグメントが展開されるので、何が起こったのかを正確に確認できます。

<img
  title="Expand metric details"
  alt="Gif of clicking and dragging the time picker"
  src={mobileTimePickerGIF}
/>

特定の OS バージョンをクリックしてパフォーマンス メトリクスを表示します。

  title="Expand metric details for one OS"
  alt="Screenshot of metrics for one OS"

/>

## アラートの作成 [#create-alerts]

アプリのパフォーマンスを可視化したので、次は主要なパフォーマンスの変化に対するアラートを設定します。

メトリクスをクリックして、チャート メニューから**Create alert condition** \[アラート条件の作成]をクリックし、UI の手順に従います。
