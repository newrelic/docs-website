モバイル アプリのエラー率と応答時間に対して設定した[アラート条件が](/docs/mobile-apps/alert-settings)長期間にわたってしきい値を超えると、 <InlinePopover type="mobile"/>エージェントは、警告 (黄色) または重大 (赤色) の問題イベントを作成します。重大 (赤) イベントが数分間発生した場合、モバイルはアラート通知を受信するように設定されているユーザーにアラート通知を送信します。また、すべての重要なイベントが終了したときにも通知が送信されます。

閲覧するには<InlinePopover type="alerts"/>ページ: **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)> Mobile > (アプリを選択) > App > Alerts**に移動します。

## アラートページの詳細

各アラート通知には、アラート アイコンと概要情報が含まれます。該当する [UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) のいずれかを使用して、詳細情報にドリルダウンします。さらに、次のオプションのいずれかを使用します。
