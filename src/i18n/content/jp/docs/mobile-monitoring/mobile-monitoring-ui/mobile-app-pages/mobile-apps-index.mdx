ニューレリックの<InlinePopover type="mobile"/> UI には、監視対象アプリのリストとそれらに関する重要な概要情報を表示するモバイル アプリ インデックスが含まれています。

## モバイルアプリの一覧を見る [#menu]

モバイル アプリのインデックスを確認するには、 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities > Mobile applications**に移動します。[エクスプローラー](/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-explorer-view-performance-across-apps-services-hosts/) を使用して、すべての [エンティティ](/docs/new-relic-one/use-new-relic-one/core-concepts/what-entity-new-relic/)、つまりアプリケーションやホストから要素のカスタム グループに至るまで、データをレポートするものとして識別できるものすべてにアクセスします。

または、 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile**に移動します。

利用可能なモバイルアプリのインデックスには、色付きの健康状態を示すものが含まれています。

* グリーン＝ノーマル
* 黄＝警告
* 赤＝クリティカル
* グレー = データを報告していない
