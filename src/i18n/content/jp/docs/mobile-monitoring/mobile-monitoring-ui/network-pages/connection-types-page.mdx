<InlinePopover type="mobile"/>の<DNT>**Connection Types**</DNT>ページには、ユーザーのワイヤレス ネットワーク接続テクノロジと、選択した期間におけるその応答時間の影響、エラー率、およびアクティブ トラフィックを示すグラフが含まれています。 データは、2G、3G などのテクノロジー世代別にグループ化されています。接続タイプの内訳には、Wi-Fi と不明も含まれます。 ここから、GPRS、HSDPA、LTE などの特定のワイヤレス テクノロジーを詳しく調べることができます。

ユーザーのモバイルワイヤレス接続タイプを表示するには

1. <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DNT>に移動します。

   <img
     title="connection types.png"
     alt="connection types.png"
     src="/images/mobile_screenshot-full_connection-types-summary.webp"
   />

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DNT>: このページを使用して、レスポンス時間の影響、ネットワーク障害、アクティブ トラフィック別に、ユーザーのモバイル ネットワーク接続タイプに関する詳細情報を表示、並べ替え、またはドリルダウンします。
</figcaption>

2. 表示される情報（応答時間、アクティブなデバイス、ネットワーク エラーなど）を変更するには、

   <DNT>
     **Sort by**
   </DNT>

   メニューから選択します。

3. [モバイル](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) アプリのバージョンまたは期間を選択するには、 メニュー

   <DNT>
     [**Versions**](/docs/mobile-apps/versions)
   </DNT>

   バーの下にある メニューと タイムピッカーNew Relic を使用します。

4. 特定の接続タイプの詳細を表示するには、その名前を選択します。

詳細情報にドリルダウンするには、該当する [プラットフォーム UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)を使用します。
