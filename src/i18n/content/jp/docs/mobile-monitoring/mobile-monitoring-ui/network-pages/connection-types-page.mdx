**Connection Types** \[接続タイプ]ページ<InlinePopover type="mobile"/>には、ユーザーのワイヤレス ネットワーク接続テクノロジーとその応答時間への影響、エラー率、および選択した期間のアクティブなトラフィックを示すグラフが含まれています。データは、2G、3G などのテクノロジー世代ごとにグループ化されています。接続タイプの内訳には、Wifi と不明も含まれます。ここから、GPRS、HSDPA、LTE などの特定のワイヤレス テクノロジーにドリルダウンできます。

ユーザーのモバイルワイヤレス接続タイプを表示するには

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (アプリを選択) > Network > Connection types**に移動します。

   <img
     title="connection types.png"
     alt="connection types.png"
     src={mobileConnectionTypesSummary}
   />

<figcaption>
  **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (アプリを選択) > Network > Connection types**: このページを使用して、応答時間への影響、ネットワークごとにユーザーのモバイル ネットワーク接続タイプに関する詳細情報を表示、並べ替え、またはドリルダウンします。障害とアクティブなトラフィック。
</figcaption>

2. 表示される情報（応答時間、アクティブなデバイス、ネットワークエラーなど）を変更するには、 **「Sort by」** メニューから選択してください。
3. モバイル アプリのバージョンまたは期間を選択するには、New Relic メニュー バーの下にある [**Versions**](/docs/mobile-apps/versions)\[バージョン] メニューと [タイム ピッカー](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) を使用します。
4. 特定の接続タイプの詳細を表示するには、その名前を選択します。

詳細情報にドリルダウンするには、該当する [プラットフォーム UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)を使用します。
