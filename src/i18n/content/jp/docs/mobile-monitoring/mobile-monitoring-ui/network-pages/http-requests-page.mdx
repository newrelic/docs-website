<DNT>**HTTP requests**</DNT> UI ページを使用すると、モバイル アプリに関連付けられた HTTP リクエストと、それらのネットワーク呼び出しがパフォーマンスにどのように影響しているかをより深く理解できます。

## HTTPリクエストページを表示する [#find-page]

<DNT>**HTTP requests**</DNT>ページを表示するには:

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > HTTP requests**
   </DNT>

   に移動します。

2. [標準の UI 機能](/docs/data-analysis/user-interface-functions/view-your-data/standard-page-functions)を使用して、HTTP 分析グラフの傾向を調べます。

3. [グループ化、ソート、フィルタリングにより、特定のリクエストやレスポンスの属性を対象とする](#groups-sort-filter) データ。

## HTTPリクエストデータの理解 [#understand-data]

最も重要な HTTP リクエスト データを見つける場所は次のとおりです。

<CollapserGroup>
  <Collapser
    id="response-time-chart"
    title="応答時間チャート"
  >
    レスポンスタイムチャートは、アプリのネットワークコールのパフォーマンスをパーセンタイルごとに示したものです。このチャートを使用して、平均応答時間を1パーセンタイル、50パーセンタイル、99パーセンタイルと比較します。パーセンタイルによって、平均応答時間を予想以上に長くしている可能性のある異常値を除外することができます。
  </Collapser>

  <Collapser
    id="errors-network-failures-chart"
    title="HTTPエラーとネットワーク障害のチャート"
  >
    このグラフには、アプリで発生した失敗したネットワーク呼び出しが表示されます。 グラフのタイトルを選択して[<DNT>**HTTP errors**</DNT>ページ](/docs/mobile-monitoring/mobile-monitoring-ui/network-pages/network-errors-http-error-network-failure-analysis)に移動し、エラーと失敗の詳細を確認してください。
  </Collapser>

  <Collapser
    id="total-requests-chart"
    title="リクエスト総数"
  >
    最も頻繁に使用されているネットワーク要求を識別するには、 <DNT>**Total requests**</DNT>で並べ替えます。 これが役立つ理由は、最も遅いネットワーク呼び出しはめったに使用されない可能性がある一方で、より頻繁に使用される要求は、最も遅くなくても最適化する価値がある可能性があるためです。
  </Collapser>
</CollapserGroup>

非エンタープライズ<DNT>**HTTP requests**</DNT> UI ページの説明については、 [「レガシー HTTP リクエスト」](#non-enterprise-http-requests)を参照してください。

## HTTPリクエストデータの表示と共有 [#insights]

インサイトで<DNT>**HTTP requests**</DNT>チャートを表示するには:

1. <Icon name="fe-more-horizontal"/>

   を任意のチャートに選択します。

2. <DNT>
     **View query > View in Insights**
   </DNT>

   を選択します。

3. オプションです。データをダッシュボードに追加したり、 [パーマリンクを使って共有したりする](/docs/data-analysis/user-interface-functions/share-your-data/permalink) 。

4. リクエストデータを詳しく調べるには、 [`MobileRequest`](/docs/insights/nrql-new-relic-query-language/nrql-query-examples/insights-query-examples-new-relic-mobile#mobilerequest-examples)のイベントと[属性](/docs/insights/insights-data-sources/default-data/mobile-events-attributes#mobilerequest-attributes)をクエリします。

エンタープライズ レベルのサブスクリプションをお持ちでない場合は、別の<DNT>**HTTP requests**</DNT> UI ページにアクセスできます。

<CollapserGroup>
