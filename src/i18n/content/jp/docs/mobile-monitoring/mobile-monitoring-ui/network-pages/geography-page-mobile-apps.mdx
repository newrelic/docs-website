Geographyページでは、モバイルユーザーの体験を世界観として表示しています。

* 応答時間をカラーで表示
* ネットワークリクエスト（1分あたりの通話数）
* データ転送サイズ
* アクティブデバイス
* ネットワーク故障率

また、各国の詳細情報にドリルダウンすることもできます。

<Callout variant="tip">
  <DNT>**Geography**</DNT>機能は<DNT>**Map**</DNT>機能と同じではありません。 [<DNT>**Map**</DNT>ページには、](/docs/mobile-monitoring/mobile-monitoring-ui/network-pages/map-page-mobile-apps)モバイル アプリと関連サービスの関係のアーキテクチャ ビューが表示されます。
</Callout>

## Geographyページの表示 [#geo_procedures]

携帯電話の応答時間を国別に表示したり、並べ替えることができます。

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Geography**
   </DNT>

   に移動します。

2. 表示される情報（レスポンス時間、1 分あたりのリクエスト数、合計転送サイズ、アクティブ デバイス、ネットワーク障害率など）を変更するには、

   <DNT>
     **Sort by**
   </DNT>

   メニューから選択します。

3. 表示される情報の量を調整するには、

   <DNT>
     **Hide &lt; 1% throughput**
   </DNT>

   を選択します。

4. 地図上の色のついた部分にマウスオーバーするか、リスト上の国名にマウスオーバーすると、その場所の概要情報が表示されます。

詳細情報にドリルダウンするには、該当する [プラットフォーム UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)を使用します。

## ドリルダウンの詳細表示 [#geo_details]

特定の場所の詳細情報（平均応答時間、1分あたりの通話数、アクティブなデバイス、種類別のネットワーク障害など）を表示するには、地理ページの地図上でその場所を選択するか、リストで名前を選択します。
