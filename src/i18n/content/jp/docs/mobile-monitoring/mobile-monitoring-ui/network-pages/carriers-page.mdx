モバイル監視の「通信事業者」ページには、ユーザーのワイヤレス通信事業者と、その応答時間の影響、エラー率、選択した期間のアクティブセッションを示すチャートが含まれています。通信事業者としてWifiも含まれています。このページでは、特定の通信事業者に関する詳細情報をソートしたり、ドリルダウンしたりすることができます。

## キャリア」ページの表示 [#procedures]

![このページを使用して、ユーザーのモバイルキャリアに関する詳細情報を、応答時間の影響、ネットワーク障害、アクティブセッションごとに表示、ソート、またはドリルダウンします。](./images/carriers.png "携帯電話会社")

<figcaption>
  **[one.newrelic.com](https://one.newrelic.com) > モバイル> （アプリを選択）> ネットワーク> 通信事業者**: このページを使用して、ユーザーのモバイル通信事業者に関する詳細な情報を、応答時間の影響、ネットワーク障害、アクティブなセッション別に表示、ソート、またはドリルダウンします。
</figcaption>

ユーザーの携帯電話会社を表示するには

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Mobile> （アプリを選択）> Network> Carriers**.
2. 表示される情報（応答時間、アクティブなデバイス、ネットワークエラーなど）を変更するには、 **「Sort by」** メニューから選択してください。
3. 表示される情報の量を調整するには、 **Hide&lt; 1% throughput** を選択します。
4. モバイルアプリのバージョンや期間を選択するには、New Relic メニューバーの下にある [**バージョン**](/docs/mobile-apps/versions) メニューと [タイムピッカー](/docs/site/timepicker-setting-time-periods-to-view-data) を使用します。
5. 特定のキャリアの詳細を表示するには、そのキャリア名を選択します。

当社標準の [ユーザーインターフェース機能](/docs/accounts-partnerships/education/getting-started-new-relic/new-relic-user-interface) および [ページ機能](/docs/accounts-partnerships/education/getting-started-new-relic/standard-dashboard-features) を使用して、詳細な情報にドリルダウンすることができます。
