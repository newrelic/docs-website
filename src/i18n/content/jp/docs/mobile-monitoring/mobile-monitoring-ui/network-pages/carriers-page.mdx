<InlinePopover type="mobile"/>のキャリア ページには、ユーザーのワイヤレス キャリアと、選択した期間におけるそのレスポンス時間の影響、エラー率、アクティブ セッションを示すグラフが含まれています。 キャリアとしてWi-Fiが含まれています。 ここから、特定のキャリアに関する詳細情報を並べ替えたり、ドリルダウンしたりできます。

  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Carriers**</DNT>: このページを使用して、レスポンス時間の影響、ネットワーク障害、アクティブ セッション別にユーザーのモバイル キャリアに関する詳細情報を表示、並べ替え、またはドリルダウンします。
</figcaption>

ユーザーの携帯電話会社を表示するには

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Carriers**
   </DNT>

   に移動します。

2. 表示される情報（応答時間、アクティブなデバイス、ネットワーク エラーなど）を変更するには、

   <DNT>
     **Sort by**
   </DNT>

   メニューから選択します。

3. 表示される情報の量を調整するには、

   <DNT>
     **Hide &lt; 1% throughput**
   </DNT>

   を選択します。

4. [モバイル](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) アプリのバージョンまたは期間を選択するには、 メニュー

   <DNT>
     [**Versions**](/docs/mobile-apps/versions)
   </DNT>

   バーの下にある メニューと タイムピッカーNew Relic を使用します。

5. 特定のキャリアの詳細を表示するには、そのキャリア名を選択します。

詳細情報にドリルダウンするには、該当する [プラットフォーム UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)を使用します。
