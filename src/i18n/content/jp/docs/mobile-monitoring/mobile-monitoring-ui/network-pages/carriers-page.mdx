モバイル監視の「通信事業者」ページには、ユーザーのワイヤレス通信事業者と、その応答時間の影響、エラー率、選択した期間のアクティブセッションを示すチャートが含まれています。通信事業者としてWifiも含まれています。このページでは、特定の通信事業者に関する詳細情報をソートしたり、ドリルダウンしたりすることができます。

ユーザーの携帯電話会社を表示するには

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Mobile > (アプリを選択) > Network > Carriers**に移動します。
2. 表示される情報（応答時間、アクティブなデバイス、ネットワークエラーなど）を変更するには、 **「Sort by」** メニューから選択してください。
3. 表示される情報の量を調整するには、 **Hide&lt; 1% throughput** を選択します。
4. モバイル アプリのバージョンまたは期間を選択するには、New Relic メニュー バーの下にある [**Versions**](/docs/mobile-apps/versions)\[バージョン] メニューと [タイム ピッカー](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) を使用します。
5. 特定のキャリアの詳細を表示するには、そのキャリア名を選択します。

詳細情報にドリルダウンするには、該当する [プラットフォーム UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)を使用します。
