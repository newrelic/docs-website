    <TabsPageItem id="android">
      ## 構文 [#syntax]

      ### Java [#java]

      ```java
      NewRelic.pauseReplay()
      ```

      ### コトリン [#kotlin]

      ```kotlin
      NewRelic.pauseReplay()
      ```

      ## 要件

      * Android エージェント v7.7.0 以上。

      ## 説明

      `newrelic.pauseReplay()` 手動でリプレイの録画を一時停止するために呼び出すことができます。

      再生がまだ進行中でない場合、この API メソッドは効果がありません。

      モバイル セッションがまだ有効な場合は、 [`newrelic.recordReplay()`](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/record-replay) API メソッドを呼び出して一時停止した再生を再開できます。

      ## 使用例

      * ヘルスケア ページ、クレジットカード情報ページなど、特定のサイト ページまたは特定の条件下で、手動でリプレイをトリガーして録画を停止したいとします。

      ## 例 [#example]

      リプレイの録画を強制的に一時停止する例を次に示します。

      ### Java [#java]

      ```java
      public class MainActivity extends Activity {
        ...
        @Override
        public boolean onOptionsItemSelected(MenuItem item) {
          switch (item.getItemId()) {
            case R.id.pause_replay:
              NewRelic.pauseReplay();
            ...
              return true;
            default:
              return super.onOptionsItemSelected(item);
          }
        }
        ...
      }
      ```

      ### コトリン [#kotlin]

      ```kotlin
      class MainActivity : AppCompatActivity() {

        ...
        var client: OkHttpClient = OkHttpClient();

            binding.fab.setOnClickListener { view ->

                NewRelic.pauseReplay()
                }
            }
          ...
      ```
