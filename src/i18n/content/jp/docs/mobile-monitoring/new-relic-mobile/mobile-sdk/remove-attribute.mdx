    <TabsPageItem id="android">
      ## 構文 [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAttribute(string $name)
      ```

      ### コトリン [#kotlin]

      ```kotlin
      NewRelic.removeAttribute(name: String?)
      ```

      ## 説明 [#description]

      このメソッドは、名前文字列で指定された属性を削除します。すべてのカスタム属性を削除する場合は、 [「すべての属性を削除する」](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes)を参照してください。

      ## パラメーター [#parameters]

      <table>
        <thead>
          <tr>
            <th width={200}>
              パラメータ
            </th>

            <th width={200}>
              タイプ
            </th>

            <th>
              説明
            </th>
          </tr>
        </thead>

        <tbody>
          <tr>
            <td>
              `$name`
            </td>

            <td>
              `string`
            </td>

            <td>
              必須項目です。削除したい属性の名前。
            </td>
          </tr>
        </tbody>
      </table>

      ## 戻り値 [#return-values]

      正常に記録された場合は`true`を返し、記録されなかった場合は`false`を返します。

      ## 例 [#example]

      `rate`という名前の属性を削除する例を次に示します。

      ### Java [#java]

      ```java
      boolean attributeRemoved = NewRelic.removeAttribute("rate");
      ```

      ### コトリン [#kotlin]

      ```kotlin
      val attributeRemoved = NewRelic.removeAttribute("rate")
      ```
