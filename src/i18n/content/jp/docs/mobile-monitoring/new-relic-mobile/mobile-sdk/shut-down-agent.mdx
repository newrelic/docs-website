      ## 構文 [#syntax]

      ### Java [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### コトリン [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```

      ## 説明 [#description]

      この API は、実行時に現在のアプリケーション ライフサイクル内でエージェントをシャットダウンする機能を提供します。エージェントがシャットダウンすると、現在のアプリケーション ライフサイクル内で再起動することはできません。

      ## 例 [#example]

      ### Java [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### コトリン [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```
    </TabsPageItem>
