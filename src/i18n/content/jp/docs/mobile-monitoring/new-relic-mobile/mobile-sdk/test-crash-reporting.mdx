      ## 構文 [#syntax]

      ### Java [#java]

      ```java
      NewRelic.crashNow(string $message)
      ```

      ### コトリン [#kotlin]

      ```kotlin
      NewRelic.crashNow(message:String!)
      ```

      ## 説明 [#description]

      New Relicのクラッシュレポートをテストするために、java.lang.RuntimeExceptionというデモ用のランタイム例外を投げます。

      このメソッドを使用すると、 `java.lang.RuntimeException` という名前のデモ ランタイム例外をメッセージとともにスローできます。このクラッシュは [\[クラッシュ分析\] ページ](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes)に表示されます。
