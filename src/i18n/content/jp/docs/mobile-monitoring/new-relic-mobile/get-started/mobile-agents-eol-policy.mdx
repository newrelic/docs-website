## モバイル ランタイム [#mobile-runtimes]

Android ランタイムをサポートするために、New Relic は[Google-Android EOL ポリシーに](https://endoflife.date/android)従います。

iOS、iPadOS、watchOS、および tvOS ランタイムをサポートするために、New Relic は[Apple EOL ポリシー](https://support.apple.com/)に従います。
