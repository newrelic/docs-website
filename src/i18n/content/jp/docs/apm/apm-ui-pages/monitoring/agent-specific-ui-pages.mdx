[APM](/docs/apm/new-relic-apm/getting-started/welcome-new-relic-apm) には言語別のエージェントがあります。すべてのAPMエージェントは同じ一般的な機能を持っていますが、UIページや機能にはエージェントごとの違いがあります。

## APMエージェントのカスタムUIページ [#agents]

また、一部のエージェントでは、特定の機能のためにカスタムUIページを用意しています。

* C SDK: C固有のUIページがない
* Go: [**Go runtime**](/docs/agents/go-agent/features/go-runtime-ui-page) page
* Java： [**JVM**](/docs/agents/java-agent/features/jvm-metrics-page) ページと [**Instrumentation editor**](/docs/agents/java-agent/custom-instrumentation/custom-instrumentation-editor-quickly-customize-your-java-instrumentation) feature
* .NET：.NET専用のUIページがない
* Node.js: [**Node VMs**](/docs/agents/nodejs-agent/supported-features/node-vms-statistics-page) ページ
* PHP： [**WordPress**](/docs/agents/php-agent/frameworks-libraries/wordpress-specific-functionality) ページと [**Drupal**](/docs/agents/php-agent/frameworks-libraries/drupal-specific-functionality) ページ
* Python：Python固有のUIページがない
* Ruby: [**Ruby VMs**](/docs/agents/ruby-agent/features/ruby-vm-measurements) ページ
