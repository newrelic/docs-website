## アプリケーションをKubernetesにリンクする [#link-app]

Kubernetesメタデータを表示して、APMエージェントにリンクできます。 `MutatingAdmissionWebhook`は、必要な環境変数をポッドに追加するために使用されます。 APMは、挿入された環境変数を使用して、そのAPMアプリケーションに関連する特定のKubernetesデプロイメント情報を表示します。

New Relicのメタデータインジェクションは、 `newrelic-metadata-injection`ラベルを介してNewRelicのKubernetes統合のインストール中に自動的に追加されます。詳細については[、アプリケーションをKubernetesにリンクする](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-your-applications-kubernetes/)方法をご覧ください。
