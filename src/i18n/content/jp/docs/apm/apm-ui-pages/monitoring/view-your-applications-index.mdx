**[one.newrelic.com](https://one.newrelic.com) > APM** にアクセスすると、New Relic [アカウント ID](/docs/accounts-partnerships/accounts/account-setup/account-id) に関連付けられたすべての APM アプリケーションのインデックスが表示されます。このページでは、アプリケーションのインデックスが 2 分ごとに自動的に更新され、最新の [ヘルスステータス](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#health-status) 、イベント、およびパフォーマンス統計が表示されます。

<Callout variant="tip">
  ご利用の全てのアプリケーションおよびサービスに関する大まかな概要を得るには、[New Relicエクスプローラー](/docs/new-relic-one/use-new-relic-one/ui-data/new-relic-one-entity-explorer)を利用してください。
</Callout>

このページでは、その一部をご紹介します。
