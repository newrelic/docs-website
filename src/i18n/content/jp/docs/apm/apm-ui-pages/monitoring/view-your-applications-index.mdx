**[one.newrelic.com >All capabilities](https://one.newrelic.com/all-capabilities) > APM**に移動すると、New Relic [アカウント ID](/docs/accounts-partnerships/accounts/account-setup/account-id)に関連付けられたすべての APM アプリのインデックスを取得できます。このページでは、アプリケーションのインデックスが 2 分ごとに自動的に更新され、最新の [正常性ステータス](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#health-status)、イベント、パフォーマンス統計が表示されます。

<Callout variant="tip">
  すべてのアプリケーションとサービスの概要を取得するには、 [エンティティ エクスプローラーを](/docs/new-relic-one/use-new-relic-one/ui-data/new-relic-one-entity-explorer)使用します。
</Callout>

このページでは、その一部をご紹介します。
