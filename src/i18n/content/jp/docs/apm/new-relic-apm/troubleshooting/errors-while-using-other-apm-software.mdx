## 問題

他の監視ソフトウェアが当社のAPMエージェントと並行して動作している場合、当社のエージェントが正しく動作することは保証できません。場合によっては、2種類以上のAPMソフトウェアを実行することで、信頼性の低いメトリクスやエージェントのクラッシュなどの問題が発生することがあります。

具体的な内容については、特定のAPMエージェントの要件や互換性に関する資料を参照してください。

## 解決策

他の監視ソフトウェアを実行することで問題が発生している可能性がある場合、以下の推奨事項があります。

* 使用しているAPMエージェントの要件や互換性に関する文書をお読みください。
* この問題を解決するには、他のAPMソフトウェアを一時的に無効にするか、またはアンインストールして、問題が解決するかどうかを確認してください。
