APMでは、カスタムレベルの監視を行いたい場合や、独自の [アラートポリシー](/docs/apm/transactions/key-transactions/view-key-transactions-alert-information) を持つようにしたい場合に、トランザクションを [キートランザクション](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events) にすることができます。

大半のNew Relicエージェントで、あらゆる種類のトランザクションをキートランザクションにできます。PHP、Node.js、Goエージェントでキートランザクションに変更できるのは、**ウェブトランザクション**のみとなります。

## キートランザクションの作成 [#create]

キー・トランザクションの作成には2つの方法があります。

<CollapserGroup>
  <Collapser
    id="index"
    title="キー・トランザクション・インデックスから"
  >
    1. **[one.newrelic.com](https://one.newrelic.com) > More> Key transactions** [key transactions index](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events#find) をご覧ください。
    2. Select **Add more**.
  </Collapser>

  <Collapser
    id="trx_page"
    title="特定の取引ページから"
  >
    1. **[one.newrelic.com](https://one.newrelic.com) > APM> （アプリを選択）> Summary**
    2. ページの中央にある「 **Transactions** 」で、レコードを選択します。
    3. Select **Track as key transaction**.
  </Collapser>
</CollapserGroup>

## キー・トランザクションの名称変更 [#rename]

リネームの方法は以下の通りです。

1. Go to **[one.newrelic.com](https://one.newrelic.com) > More> Key transactions**.

2. インデックスから、名前を変更したいキートランザクションの歯車

   <Icon name="fe-settings"/>

   アイコンを選択します。

3. トランザクションの新しい名前を入力し、保存してください。

## Apdexの設定変更 [#apdex]

キーとなるトランザクションが [ウェブトランザクション](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#transaction) である場合、カスタム [Apdex のしきい値](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) を設定することができます（Apdex は [非ウェブトランザクション](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#non-web-transaction) には適用されません）。これは、そのトランザクションが他のトランザクションよりも高速または低速であることが予想される場合に便利です。

<Callout variant="important">
  キートランザクションのApdex設定は、そのアプリに設定されたT値を上書きします。これは例えば、あるキートランザクションが **Satisfied** と報告されても、同じアプリで同じ応答時間の別のトランザクションが **Frustrated** と報告されることを意味します。
</Callout>

キートランザクションのApdex閾値を変更するには

1. Go to **[one.newrelic.com](https://one.newrelic.com) > More> Key transactions**.

2. 編集したいトランザクションの

   <Icon name="fe-settings"/>

   アイコンを選択します。

3. **Apdex T** の見出しの下に新しい値を秒単位で入力し、保存してください。

Apdexのしきい値は、 [キー・トランザクション・トレース](/docs/apm/transactions/transaction-traces/configure-transaction-traces#key-txns) の作成方法に影響します。

## キートランザクションの削除 [#delete]

キートランザクションを削除するには

1. Go to **[one.newrelic.com](https://one.newrelic.com) > More> Key transactions**.

2. 削除したいキートランザクションの歯車

   <Icon name="fe-settings"/>

   アイコンを選択します。

3. Select **Stop tracking**.

New Relic は、削除されたキー・トランザクションに関連するアラートやイベントを自動的に削除します。
