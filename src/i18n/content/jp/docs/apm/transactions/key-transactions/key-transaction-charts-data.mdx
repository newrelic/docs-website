APMでは、すべての [キートランザクション](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events) には、独自の **Overview** ページとその他いくつかのUIページがあります。 [トランザクションのタイプ（ウェブまたは非ウェブ）](/docs/apm/transactions/intro-transactions/transactions-new-relic-apm) に応じて、異なる情報が表示されます。

## 概要ページ [#key_overview]

[key transaction index](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events#find) からトランザクションを選択すると、そのキートランザクションの **Overview** ページが表示されます。 **Overview** ページには、レスポンスタイム、 [Apdex](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) 、スループット、その他のデータのチャートが含まれています。このページは、APM の [メインの Overview ページ](/docs/apm/applications-menu/monitoring/apm-overview-page) と同様で、 [New Relic の標準的なページ機能](/docs/data-analysis/user-interface-functions/view-your-data/standard-page-functions) を備えています。

## SLAレポートページ [#key_sla]

主要なトランザクションの **SLA report** ページには、そのトランザクションの [Service Level Agreement](/docs/apm/reports/service-level-agreements/apm-sla-reports-dashboard) レポートが表示されます。SLAレポートには、スループット、応答時間、およびApadexに関する日次および週次のデータが含まれています。また、レポートをカンマ区切りの値（CSV）のスプレッドシートとしてダウンロードすることもできます。

## アラート [#key_alerts]

[キー・トランザクション・インデックス](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events#find) からトランザクションを選択すると、 [警告条件の違反を表示したり、警告ポリシーを設定するオプションも表示されます](/docs/apm/transactions/key-transactions/view-key-transactions-alert-information) 。
