APMでは、 [主要なトランザクション](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events) に対して、 [New Relic Alerts](/docs/alerts) でアラートポリシーを作成することができます。

## キートランザクションに対するアラート条件の作成 [#create-alert]

キーとなるトランザクションのアラート条件を作成するには、次のようにします。

1. [APMで監視しているエンティティのアラート条件](/docs/alerts/new-relic-alerts/defining-conditions/define-alert-conditions#create-condition) を作成する手順を説明します。
2. 条件の種類として、 **キー・トランザクション・メトリック** を選択します。

## キートランザクションのアラートステータスの表示 [#view-alerts]

主要なトランザクションの現在の状態を表示する。

1. Go to **[one.newrelic.com](https://one.newrelic.com) > More> Key transactions**.
2. トランザクションの [色分けされた健康状態を見る](/docs/alerts/new-relic-alerts/configuring-alert-policies/identify-entities-without-alert-policies) 。
3. 詳細は、健康状態にカーソルを合わせると表示されます。

アラートのポリシーや条件を作成する方法については、 [New Relic Alerts](/docs/alerts/new-relic-alerts) を参照してください。
