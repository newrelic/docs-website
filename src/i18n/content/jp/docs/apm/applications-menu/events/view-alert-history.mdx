APM の<DNT>**Alerts**</DNT>ページには、過去 2 週間のアラートの表が表示され、スループット、エラー率、履歴などの詳細情報にドリルダウンできます。 特定のアプリケーションまたはすべてのアプリケーションの集計履歴を表示できます。

## 特定のアプリに対するアラート [#alert\_procedures][#alert_procedures]

特定のアプリのアラートのリストを表示するには、 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; APM &amp; Services &gt; (select an app) &gt; Events &gt; Issues &amp; activity**</DNT>に移動します。

<DNT>**Issues &amp; Activity**</DNT>ページには、問題、まとめイベント、異常、事後分析 (存在する場合) のリストが表示されます。 New Relic の標準[UI 機能](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)のいずれかを使用して、詳細をドリルダウンします。

各アラート通知には、アラートアイコンと概要情報が含まれています。アラートページでアラート通知の詳細を表示するには、そのリンクを選択します。例えば、以下のようになります。

* 特定のアラートの詳細を表示するには、その名前を選択します。
* アラートのリストを絞り込むには、 <DNT>**Filter**</DNT>を使用します。
* すべてのアプリとホストのすべてのアラートを表示するには、 <DNT>**All alerts**</DNT>リンクを選択します。
* 該当する場合は特定の Java 仮想マシン (JVM) を選択するには、 <DNT>**JVMs**</DNT>メニューから選択します。
