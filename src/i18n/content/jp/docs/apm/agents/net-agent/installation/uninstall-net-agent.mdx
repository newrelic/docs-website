本ドキュメントでは、New Relic .NET エージェントをアンインストールする方法を説明します。エージェントを一時的に無効にする方法については、 [Disable the agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#net) を参照してください。

.NETエージェントをアンインストールするには、お使いのOSに応じた手順に従ってください。

* [ウィンドウズ](#uninstall-framework-windows)
* [Linux](#uninstall-linux)

## .NETエージェントのアンインストール（Windows） [#uninstall-framework-windows]

<Callout variant="tip">
  Requires [**Administrator** rights](/docs/agents/net-agent/getting-started/compatibility-requirements-net-framework-agent#user-rights) in your Windows admin group.
</Callout>

この手順では、MSIインストーラーで.NETエージェントをインストールしたことを前提としています。

1. 実行中のエージェントを停止するには、IISを停止してください。IISを停止しないと、エージェントをアンインストールした後に再起動が必要になる場合があります。
2. 選択 **Start> All Programs> New Relic .NET Agent> Uninstall .NET agent**.
3. 画面の指示に従って、アンインストールを完了させてください。
4. IISを再起動する。

エージェントをアンインストールした後は、エージェント関連のファイルをすべて削除するとよいでしょう。他の New Relic ファイルを削除する場合は、必ず他のパスを確認してください。たとえば、.NET エージェントをアンインストールしても、 **[newrelic.config](/docs/agents/net-agent/installation-configuration/net-agent-configuration#config-options-precedence)** のファイルは削除されません。

1. **\[コントロールパネル]>\[プログラム]>\[プログラムと機能**]から、右クリックして`New Relic .NET Agent`という名前のファイルを削除します。
2. `C:\ProgramData\New Relic\.NET Agent`のフォルダ全体を削除します。
3. `C:\Program Files\New Relic\.NET Agent`のフォルダ全体を削除します。
4. `C:\Program Files (x86)\New Relic\.NET Agent`のフォルダ全体を削除します。
5. [APMのUIからアプリ](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) を削除します。

## .NETエージェントのアンインストール（Linux） [#uninstall-linux]

Linuxの.NETエージェントをアンインストールするには、パッケージ管理ツールを使用します。

<Callout variant="important">
  .NETエージェントバージョン10.0.0以降の場合、パッケージの名前は`newrelic-dotnet-agent`です。.NETエージェントバージョン9.9.0以下の場合、パッケージの名前は`newrelic-netcore20-agent`です。このドキュメントの残りの部分では、 `newrelic-dotnet-agent`を参照します。古いエージェントバージョンがインストールされている場合は、これを古い名前に置き換えてください。
</Callout>

<Callout variant="important">
  アンインストールが完了したら、アプリケーションを再起動する必要があります。
</Callout>

<CollapserGroup>
  <Collapser
    id="apt-uninstall"
    title="aptによるアンインストール（Debian、Ubuntu、Linux Mint）"
  >
    rootとして（またはsudoを使用して）次のコマンドを実行します。

    ```
    apt-get remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="dpkg-uninstall"
    title="dpkgによるアンインストール（Debian、Ubuntu、Linux Mint）"
  >
    rootとして（またはsudoを使用して）次のコマンドを実行します。

    ```
    dpkg --remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="yum-uninstall"
    title="yumによるアンインストール（CentOS, Red Hat Enterprise Linux, Oracle Linux）"
  >
    rootとして（またはsudoを使用して）次のコマンドを実行します。

    ```
    yum remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="rpm-uninstall"
    title="rpmによるアンインストール（CentOS, Red Hat Enterprise Linux, Oracle Linux）"
  >
    rootとして（またはsudoを使用して）次のコマンドを実行します。

    ```
    sudo rpm -e newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="tarball"
    title="アンインストール tarball"
  >
    1. appフォルダからNew Relicのファイルをすべて削除します。

    2. New Relic の環境変数をアンセットする。

       ```
       CORECLR_ENABLE_PROFILING=1
       CORECLR_PROFILER={36032161-FFC0-4B61-B559-F6C5D41BAE5A}
       CORECLR_NEWRELIC_HOME=PATH_TO_INSTALL
       CORECLR_PROFILER_PATH=%CORECLR_NEWRELIC_HOME%\NewRelic.Profiler.dll
       ```
  </Collapser>
</CollapserGroup>
