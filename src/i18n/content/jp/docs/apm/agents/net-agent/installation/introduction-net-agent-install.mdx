.NETエージェントは、アプリケーションのモニタリングを開始できるように、コードを自動インストゥルメンテーションします。ランチャーを使用するか、本ドキュメントの説明に従って基本的な.NETエージェントのインストールを完了できます。.NET用APMの動作の詳細については、[.NET用New Relicの概要](/docs/agents/net-agent/getting-started/introduction-new-relic-net)を参照してください。

New Relicアカウントをまだお持ちでない場合は、[New Relicアカウントを新たに作成します](https://newrelic.com/signup)。永久無料です。

## .NETエージェントのインストール [#install-instructions]

以下のオペレーティングシステムの適切なインストール手順を参照し、ダウンロードリンクを検索してください。

* [Windows](/docs/agents/net-agent/installation/install-net-agent-windows)

  * IIS上のWindows .NETアプリケーションまたはLinux上の.NET Coreアプリケーションの場合は、[ガイド付きインストール](https://one.newrelic.com/launcher/nr1-core.home?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5ob21lLXNjcmVlbiJ9&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyIsInBhdGgiOiJhcG0iLCJsYW5ndWFnZSI6ICJkb3RuZXQiLCAicmVjaXBlTmFtZSI6ICJkb3RuZXQtYWdlbnQtaW5zdGFsbGVyIiwgImluaXRpYWxBY3Rpb25JbmRleCI6MiwiYWNjb3VudElkIjpudWxsfQ==)を使用して、APMを起動します。
  * EUサーバーの場合は、代わりに[このリンク](https://one.eu.newrelic.com/launcher/nr1-core.home?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5ob21lLXNjcmVlbiJ9&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyIsInBhdGgiOiJhcG0iLCJsYW5ndWFnZSI6ICJkb3RuZXQiLCAicmVjaXBlTmFtZSI6ICJkb3RuZXQtYWdlbnQtaW5zdGFsbGVyIiwgImluaXRpYWxBY3Rpb25JbmRleCI6MiwiYWNjb3VudElkIjpudWxsfQ==)を使用します。

* [Linux](/docs/agents/net-agent/installation/install-net-agent-linux)

[最新リリース](http://download.newrelic.com/dot_net_agent/latest_release)に.NETエージェントのすべてのバイナリとパッケージがあります。また、[過去のリリース](http://download.newrelic.com/dot_net_agent/previous_releases/)でリリース履歴も確認できます。
