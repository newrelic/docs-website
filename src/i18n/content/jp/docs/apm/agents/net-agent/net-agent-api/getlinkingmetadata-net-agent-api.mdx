## 構文

```
NewRelic.Api.Agent.NewRelic.GetLinkingMetadata();
```

トレースやエンティティのリンクに使用できるキー／バリューペアを返します。

## 要件

Agentバージョン8.19以上。

すべてのアプリタイプに対応しています。

## 説明

返されるキー／バリューペアのディクショナリーには、APM製品のトレースとエンティティをリンクするために使用される項目が含まれる。意味のある値を持つ項目のみが含まれます。例えば、分散型トレースが無効の場合、 `trace.id` は含まれません。

## 戻り値

辞書&lt;文字列, 文字列>() 返された項目には、APM 製品のトレースとエンティティを結びつけるための項目が含まれています。

## 例

```
NewRelic.Api.Agent.IAgent Agent = NewRelic.Api.Agent.NewRelic.GetAgent();
var linkingMetadata = Agent.GetLinkingMetadata();
foreach (KeyValuePair<string, string> kvp in linkingMetadata)
{
     Console.WriteLine("Key = {0}, Value = {1}", kvp.Key, kvp.Value);
}
```
