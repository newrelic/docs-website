## 構文

```cs
NewRelic.Api.Agent.NewRelic.IgnoreTransaction()
```

現在のトランザクションを計測しない。

## 要件

すべてのAgentバージョンに対応しています。

[トランザクションの中で呼び出す必要があります](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction) 。

## 説明

現在のトランザクションを無視します。

<Callout variant="tip">
  トランザクションを無視することもできます [カスタムインストルメンテーションXMLファイル](/docs/agents/net-agent/custom-instrumentation/add-detail-transactions-xml-net#blocking-instrumentation) を介して。
</Callout>

## 例

```cs
NewRelic.Api.Agent.NewRelic.IgnoreTransaction();
```
