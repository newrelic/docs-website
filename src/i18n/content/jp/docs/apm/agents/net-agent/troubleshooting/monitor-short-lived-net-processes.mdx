## 問題

実行時間が60秒未満の.NETプロセスがあり、そのプロセスのデータが報告されていません。

## 解決策

APM [ハーベストサイクル](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#harvest-cycle) は60秒です。このため、それよりも短い時間で実行されるプロセスでは、データが報告されないことがあります。この問題を解決するには、 `sendDataOnExit` と `sendDataOnExitThreshold` [service elements](/docs/agents/net-agent/configuration/net-agent-configuration#service) を [newrelic.config](/docs/agents/net-agent/configuration/net-agent-configuration#config-options-precedence) で使用します。 `sendDataOnExitThreshold` の値は、処理が完了するまでの時間よりも短く設定する必要があります。
