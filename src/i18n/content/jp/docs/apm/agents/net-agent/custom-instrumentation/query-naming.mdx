New Relic .NETエージェントは、アプリケーションのSQLクエリを計測およびグループ化します。通常、.NETエージェントは、クエリのプロパティに基づいてクエリに名前を自動的に割り当てます。または、クエリにカスタム名を割り当てることもできます。

## クエリに名前を付ける方法 [#name-query]

.NETエージェントでサポートされているすべてのSQLバージョンには、カスタム名を付けることができます。

クエリに名前を付けるには、 `/* NewRelicQueryName: myQueryName */`のような形式でクエリテキストにコメントを追加します。これをアプリのコードに追加する例を次に示します。

```cs
var connection = new MySqlConnection(mySqlConnectionString);
var command = new MySqlCommand("/* NewRelicQueryName: My query name */ SELECT date FROM dates", connection);
```
