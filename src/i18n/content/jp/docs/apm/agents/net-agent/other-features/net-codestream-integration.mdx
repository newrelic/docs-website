[New Relic CodeStream](/docs/codestream/how-use-codestream/performance-monitoring#code-level)を.NETエージェントと統合することにより、コンテキスト依存のAPMデータをIDEに直接表示します。この統合により、コードを記述およびレビューするときに、エディターでコードレベルの本番テレメトリを視覚化します。

## スタートガイド [#get-started]

まず、サポートされているIDEにNew Relic CodeStream拡張機能を[インストール](/docs/codestream/start-here/install-codestream)し、ログインします。

<Callout variant="important">
  New Relic CodeStream統合は、 [.NETエージェントバージョン9.9.0以降](/docs/release-notes/agent-release-notes/net-release-notes)で利用可能であり、デフォルトでは無効になっています。この構成を変更するには、[ドキュメント](/docs/apm/agents/net-agent/configuration/net-agent-configuration#code_level_metrics)を確認してください。
</Callout>

## エージェントの属性 [#attributes]

.NETエージェントは、アプリケーションによって生成されたスパンに次の属性を報告して添付します。

<Callout variant="important">
  場合によっては、エージェントは、アプリケーション内で呼び出され、IDE内に表示されないベースフレームワークコード（データストア、外部呼び出しなど）を直接ラップします。
</Callout>
