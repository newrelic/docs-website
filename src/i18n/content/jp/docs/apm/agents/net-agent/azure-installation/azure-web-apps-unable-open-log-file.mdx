## 問題

FTP で [Microsoft Azure Web App](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps) に接続しています。 `NewRelic.Profiler.####.log` ファイルが **LogFiles\\NewRelic** に表示されていますが、そのログファイルにアクセスしようとするとエラーメッセージが表示されます。

## 解決策

Webアプリを再起動すると、ファイルが解放され、新しいプロファイルログが作成されます。
