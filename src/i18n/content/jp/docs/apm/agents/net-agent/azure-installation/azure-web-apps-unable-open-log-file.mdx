## 問題

FTP 経由で[Microsoft Azure Web アプリ](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps)に接続しています。 <DNT>**LogFiles\\NewRelic**</DNT>に`NewRelic.Profiler.####.log`ファイルが表示されていますが、ログファイルにアクセスしようとするとエラーメッセージが表示されます。

## 解決

Webアプリを再起動すると、ファイルが解放され、新しいプロファイルログが作成されます。
