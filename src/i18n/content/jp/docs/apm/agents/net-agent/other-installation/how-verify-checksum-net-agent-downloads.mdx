New Relic の .NET エージェント インストーラーのチェックサムを確認する方法についてお読みください。現在のリリースのハッシュが必要な場合は、 [ダウンロード サイト から SHA256 ハッシュをダウンロードしてください](https://download.newrelic.com/dot_net_agent/latest_release/SHA256/)。

## Linux [#checksum-linux]

Linux の場合は、次を実行します。

```sh
sha256sum YOUR_FILENAME
```

成功すると、このような回答が得られます。

```
5092fd52e40132a41ac06c320296cd9d63196d6de0fa6ce9b74fc4de1b3f9502 YOUR_FILENAME
```

## ウィンドウズ [#checksum-windows]

Windows では、ファイル ハッシュを計算するために`CertUtil`または`Get-FileHash`のいずれかを使用することをお勧めします。

<CollapserGroup>
  <Collapser
    id="certutil"
    title="CertUtil (コマンドプロンプトまたは Powershell)"
  >
    `CertUtil` 証明書サービスの一部として利用可能であり、ファイルのハッシュを計算するために使用できます。 コマンドプロンプトまたは Powershell から実行できます。

    ハッシュを計算するには、次のように実行します。

    ```sh
    CertUtil -hashfile YOUR_FILENAME SHA256
    ```

    成功すると、このような回答が得られます。

    ```
    SHA256 hash of YOUR_FILENAME:
    5092fd52e40132a41ac06c320296cd9d63196d6de0fa6ce9b74fc4de1b3f9502
    CertUtil: -hashfile command completed successfully.
    ```
  </Collapser>

  <Collapser
    id="getfilehash"
    title="Get-FileHash (Powershell)"
  >
    `Get-FileHash`コマンドレットを使用して、ファイルのハッシュを計算できます。 Powershell バージョン 5.1 以降で利用できます。

    ハッシュを計算するには、次のように実行します。

    ```powershell
    Get-FileHash YOUR_FILENAME Algorithm SHA256 | Format-List
    ```

    成功すると、このような回答が得られます。

    ```
    Algorithm : SHA256
    Hash      : 5092FD52E40132A41AC06C320296CD9D63196D6DE0FA6CE9B74FC4DE1B3F9502
    Path      : C:\Path\To\YOUR_FILENAME
    ```
  </Collapser>
</CollapserGroup>
