* 6.x エージェント ラインは、4.5 より前のバージョンの .NET Framework で実行されているアプリケーションを使用する顧客をサポートするために維持されており、重要なセキュリティ パッチのみを受け取ります。最新の 6.x エージェント バージョンのみがサポートされています。6.x エージェントの EOL の日付は、Microsoft の .NET Framework 3.5 Service Pack 1 のサポート終了日と同じです。

## Microsoft .NET ランタイム [#microsoft-runtimes]

.NET ランタイムをサポートするために、New Relic は以下の Microsoft のライフサイクル ポリシーに従います。

* [マイクロソフト .NET フレームワーク](https://docs.microsoft.com/en-us/lifecycle/products/microsoft-net-framework)
* [Microsoft .NET および .NET コア](https://docs.microsoft.com/en-us/lifecycle/products/microsoft-net-and-net-core)
