## エンドオブライフのお知らせ [#eol]

2017 年 10 月現在、Microsoft の Azure Marketplace ポータルから New Relic アカウントを作成することはできません。代わりに、Azure ID をお持ちで、まだ New Relic アカウントをお持ちでない場合は、 [New Relic Sales](https://newrelic.com/about/contact-us) にお問い合わせください。

[詳細は、New Relic のオンラインテクニカルコミュニティの Azure 移行情報](https://discuss.newrelic.com/t/new-relic-apm-for-azure-marketplace-eol-notice/52358) をご覧ください。

## New Relic .NET エージェントのインストール [#app-services]

必要に応じて、New Relic .NET エージェントを Azure アプリにインストールする手順を続けます。

* マイクロソフトの [Web App Services](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps) in Azure Portal
* マイクロソフトの [クラウドサービス](/docs/agents/net-agent/azure-installation/install-app-azure-cloud-services) プラットフォーム

.NETエージェントのMicrosoft Azureへの対応は、 [Infrastructure monitoringのAzureとの統合とは異なります。](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations) 。Azure の統合は、別の [統合のインストール手順](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) があります。
