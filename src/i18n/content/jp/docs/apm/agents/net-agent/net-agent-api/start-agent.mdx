## 構文

```
NewRelic.Api.Agent.NewRelic.StartAgent()
```

エージェントがまだ開始されていない場合は、エージェントを開始します。通常は不要です。

## 要件

エージェントのバージョン5.0.136.0以降。

すべてのアプリタイプに対応しています。

## 説明

エージェントがまだ開始されていなければ、エージェントを開始します。 [`autoStart`](/docs/agents/net-agent/installation-configuration/net-agent-configuration#service-autoStart) を無効にしない限り、インスツルメンテッド・メソッドにぶつかるとエージェントが自動的に開始されるので、この呼び出しは通常不要です。 [`SetApplicationName()`](/docs/agents/net-agent/net-agent-api/setapplicationname-net-agent) を使用する場合は、 エージェントを起動する前に **アプリ名を設定していることを確認してください。**

\*\*<Callout variant="tip">
このメソッドは、 [`syncStartup`](/docs/agents/net-agent/installation-configuration/net-agent-configuration#service-syncStartup) または [`sendDataOnExit`](/docs/agents/net-agent/installation-configuration/net-agent-configuration#service-sendDataOnExit) を有効にしていない限り、非同期にエージェントを起動します（つまり、アプリの起動をブロックしません）。
</Callout>## 例\`\`\`
NewRelic.Api.Agent.NewRelic.StartAgent();
