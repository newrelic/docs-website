<LandingPageHero>
  <HeroContent>
    New Relicの.NETエージェントは.NETアプリを監視し、アプリのパフォーマンスをエンドツーエンドで確認できるようにします。New Relicの.NET Frameworkエージェントは、VB.NET、C#、CLIなどの.NETと互換性のあるすべての言語で動作します。.NET Coreエージェントは、互換性、スピード、拡張API機能、そしてMicrosoftの.NET Coreで利用可能なクロスプラットフォーム機能を活用します。

    エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[.NETエージェントのリリースノート](/docs/release-notes/agent-release-notes/net-release-notes)を参照ください。

    クレジットカードは不要です。すでにアカウントをお持ちの場合[ログイン](http://one.newrelic.com/)。
  </HeroContent>
