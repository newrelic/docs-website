## Ruby ランタイム [#ruby-runtimes]

Ruby ランタイムをサポートするために、New Relic は Ruby コア コミュニティによって設定された[Ruby Maintenance Branches ポリシー](https://www.ruby-lang.org/en/downloads/branches/)に従います。
