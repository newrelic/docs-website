    New Relicの言語エージェントファミリーとして初のRubyエージェントは、長年にわたって実証された安定性と継続的なリリース機能を兼ね備えています。Rubyエージェントを使用して、以下を実現できます。

    * [問題の特定とアプリケーションパフォーマンスの向上ê](/docs/agents/ruby-agent/getting-started/introduction-new-relic-ruby#monitor-performance)。

    * APM、ブラウザモニタリング、その他のNew Relic機能で精選されたダッシュボードからデータを監視します。

    * [クエリと視覚化](/docs/insights/using-insights-ui/getting-started/introduction-new-relic-insights)できるレポートデータ：

      エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[Rubyエージェントのリリースノート](/docs/release-notes/agent-release-notes/ruby-release-notes)を参照ください。
