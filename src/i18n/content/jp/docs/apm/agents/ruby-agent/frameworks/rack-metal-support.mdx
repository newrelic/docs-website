[Rack のミドルウェア](/docs/agents/ruby-agent/frameworks/rack-middlewares) は、Ruby agent version 3.9.0 以上で自動的にインストルメント化されます。

`NewRelic::Agent::Instrumentation::Rack` モジュールは非推奨です。Ruby エージェントのバージョン 3.9.0 以降にアップグレードした後は、New Relic では、このモジュールへの参照をコードから削除することを推奨します。詳細については、 [Instrumenting Metal controllers](/docs/agents/ruby-agent/frameworks/instrumenting-metal-controllers) をご覧ください。
