[Rack のミドルウェア](/docs/agents/ruby-agent/frameworks/rack-middlewares) は、Ruby agent version 3.9.0 以上で自動的にインストルメント化されます。

`NewRelic::Agent::Instrumentation::Rack`モジュールは非推奨になりました。Ruby エージェント バージョン 3.9.0 以降にアップグレードした後、New Relic はコードからこのモジュールへの参照を削除することをお勧めします。詳細については、 [「Metal コントローラーのインストルメント」](/docs/agents/ruby-agent/frameworks/instrumenting-metal-controllers)を参照してください。
