## 問題

Grapeがインストールされていないというログメッセージが表示されます。

```
INFO : Not installing New Relic supported Grape instrumentation because the third party newrelic-grape gem is present
```

## 解決

`newrelic-grape`宝石を削除します。

## 原因

New Relic は、サードパーティの`newrelic-grape` gem をサポートしていません。Ruby エージェントが`newrelic-grape` gem を検出した場合、New Relic の[組み込み Grape インストルメンテーション](/docs/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks)はインストールされません。サードパーティの gem を削除すると、サポートされている Grape インストルメンテーションを使用できるようになります。
