## 問題

Grapeがインストールされていないというログメッセージが表示されます。

```
INFO : Not installing New Relic supported Grape instrumentation because the third party newrelic-grape gem is present
```

## 解決策

`newrelic-grape` gemを削除します。

## 原因

New Relic は、サードパーティ製の `newrelic-grape` gem をサポートしていません。Ruby エージェントが `newrelic-grape` gem を検出した場合、New Relic の [ビルトイン Grape instrumentation](/docs/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) をインストールしません。サードパーティ製の gem を削除することで、サポートされている Grape instrumentation を使用することができます。
