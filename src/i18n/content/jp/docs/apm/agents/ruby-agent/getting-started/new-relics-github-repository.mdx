New Relic Ruby エージェントは現在完全にオープンソースであり、GitHub ( [newrelic-ruby-agent )](https://github.com/newrelic/newrelic-ruby-agent/tree/main)で入手できます。バージョン 6.12 (2020 年 7 月 16 日リリース) の時点では、Apache License バージョン 2.0 に基づいてライセンスされています。古いバージョンのエージェントについては、ソース コードに含まれる LICENSE ファイルを確認してください。Ruby エージェント バージョン 2.1.0 のソース コードタグを使用して GitHub で入手できます。

`main`の責任者には New Relic の最新リリースがインストールされます。ただし、エッジを`main`にプッシュする機能を予約します。`dev`ブランチには未リリースの作品が含まれています。GitHub からリリースをダウンロードする場合は、適切なタグを使用してください。通常、新しいバージョンを RubyGems に公開する前に、GitHub 経由でプレビュー バージョンをリリースします。

フォークして、お気軽[にプル リクエストを送信し](https://github.com/newrelic/newrelic-ruby-agent/pulls)、[問題を作成し](https://github.com/newrelic/newrelic-ruby-agent/issues/new/choose)、GitHub リポジトリで直接[ディスカッションを開始してください](https://github.com/newrelic/newrelic-ruby-agent/discussions)。私たちはコミュニティと協力するのが大好きです!
