本ドキュメントでは、New Relic Ruby エージェントをアンインストールする方法を説明します。エージェントを一時的に無効にする方法については、 [Disable the agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#ruby) を参照してください。

Rubyエージェントをアンインストールするには

1. `gem 'newrelic_rpm'` という行をGemfileから削除します。
2. **newrelic.yml** ファイルを **RAILS_ROOT/config/** フォルダから削除します。
3. **bundle install** を実行します。
4. アンインストールが完了したら、アプリを再起動してください。

インストールの詳細については、 [New Relic for Ruby](/docs/agents/ruby-agent/getting-started/new-relic-ruby) を参照してください。
