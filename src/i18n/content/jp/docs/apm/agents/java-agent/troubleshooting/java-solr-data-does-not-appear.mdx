## 問題

Web アプリケーションで [Solr プラットフォームを](http://lucene.apache.org/solr/) 使用していますが、Solr データが <InlinePopover type="apm"/>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (Java アプリを選択) > Monitor**からアプリの UI を選択します。

## 解決

Solr が [](/docs/agents/java-agent/getting-started/compatibility-requirements-java-agent#other)正しく設定され、データが New Relic に報告されていることを確認するために、必要に応じて以下の作業を行います。

* **JMX:** `<jmx/>`タグを使用して、 `solrconfig.xml`の Solr 構成で JMX が有効になっていることを確認してください。`solrconfig.xml`ファイルは、Solr 自体に影響するほとんどのパラメーターを含む構成ファイルです。
* **Jetty:** Jetty のユーザーであれば、 [Jetty の JMX を手動で有効にしてください](https://wiki.eclipse.org/Jetty/Tutorial/JMX#Enabling_JMXConnectorServer_for_Remote_Access).
* **Solr metrics:** JMX コンソールで Solr metrics を確認します。SolrのキャッシュとアップデートはMBeansから行われます。
