## 問題

Web アプリケーションで[Solr プラットフォーム](http://lucene.apache.org/solr/)を使用していますが、 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (select a Java app) > Monitor**</DNT>のアプリの<InlinePopover type="apm"/> UI に Solr データが表示されません。

## 解決

Solr が [](/docs/agents/java-agent/getting-started/compatibility-requirements-java-agent#other)正しく設定され、データが New Relic に報告されていることを確認するために、必要に応じて以下の作業を行います。

* <DNT>
    **JMX:**
  </DNT>

  `<jmx/>`タグを使用して、 `solrconfig.xml`の Solr 設定で JMX が有効になっていることを確認します。 `solrconfig.xml`ファイルは、Solr 自体に最も影響を与える設定ファイルです。

* <DNT>
    **Jetty:**
  </DNT>

  Jetty ユーザーの場合は、[Jetty の JMX を手動で有効にします](https://wiki.eclipse.org/Jetty/Tutorial/JMX#Enabling_JMXConnectorServer_for_Remote_Access)。

* <DNT>
    **Solr metrics:**
  </DNT>

  Solr メトリックスの JMX コンソールを確認します。 Solr キャッシュと更新は MBean から取得されます。
