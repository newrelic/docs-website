## Java ランタイム [#java-runtimes]

Java ランタイムのサポートについては、New Relic は[Oracle Premier Support Policy](https://www.oracle.com/java/technologies/java-se-support-roadmap.html)に従います。New Relic は、その日以降、EOL サポートを受ける権利を留保します。
