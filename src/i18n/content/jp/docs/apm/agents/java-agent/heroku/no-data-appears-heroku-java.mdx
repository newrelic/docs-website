## 問題

数分待っても、HerokuアプリのAPMにデータが表示されません [**まとめ** ページ](/docs/apm/applications-menu/monitoring/apm-overview-page).

## 解決策

1. このHeroku toolbeltコマンドで、Herokuのログにエラーがないか確認します。

   ```
   heroku logs --tail
   ```

2. Heroku dyno内のファイルが、 [Java on Heroku](/docs/agents/java-agent/heroku/java-agent-heroku) または [Java with Scala on Heroku](/docs/agents/java-agent/heroku/java-agent-scala-heroku) のインストール手順によるファイル修正と一致していることを確認します。このHeroku toolbeltコマンドを使用します。

   ```
   heroku run bash
   ```

3. [の手順をお試しください。 データが表示されない（Java）](/docs/agents/java-agent/troubleshooting/no-data-appears-java).
