## 問題

New Relic PHP エージェントのインストールに関する問題が引き続き発生しています。

## 解決策

[トラブルシューティングのヒント](/docs/agents/php-agent/troubleshooting) を実行しても New Relic PHP のインストールに問題がある場合は、 [support.newrelic.com](https://support.newrelic.com "新しいウィンドウで開くリンク") でサポートを受けてください。

New Relic Technical Support のエンジニアに依頼する場合は、以下の情報を提供してください。

* インストール作業はどこまで進んでいるのか

* どのインストール方法を使用しているか

* A copy your `newrelic.cfg` file (if any of them)

* **phpinfo** ページへのリンク、または **phpinfo** の出力を HTML ファイルとして保存したものです。 [を作成するには **phpinfo**](/docs/php/php-kb-103) を含む .php ウェブページを作成します。

  ```
  <?php phpinfo(); ?>
  ```

* [エージェントとデーモンのログファイル](/docs/php/php-kb-105)
