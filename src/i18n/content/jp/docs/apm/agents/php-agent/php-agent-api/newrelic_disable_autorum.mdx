## 構文

```php
newrelic_disable_autorum()
```

特定のページでの<InlinePopover type="browser"/>スニペットの自動インジェクションを無効にします。

## 要件

エージェントのバージョン [2.6.5.41](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-26541) 以上。

## 説明

このコールは、現在のトランザクションに対する [ブラウザ監視](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) エージェントの自動注入を無効にします。できるだけ早い段階で呼び出します。 [の挿入で問題が発生している場合](/docs/browser/new-relic-browser/troubleshooting/browser-javascript-injection-causes-problems-page) や、JavaScript を許可していないサードパーティのサービスにページを提供している場合（例えば [Google の加速モバイルページ](/docs/browser/new-relic-browser/troubleshooting/google-amp-validator-fails-due-3rd-party-script) など）に、このコールを使用して JavaScript を削除することができます。

<Callout variant="tip">
  代わりにAPI呼び出しを使用してブラウザ監視を <DNT>**enable**</DNT> するには、次を参照してください。

  * [`newrelic_get_browser_timing_footer()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer)
  * [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header)
</Callout>

## パラメーター

このコールはパラメータを受け取りません。

## 戻り値

New Relic トランザクション内で呼び出された場合、 `true`を返します。それ以外の場合、トランザクション外の場合 (たとえば、 `newrelic_end_transaction()`が呼び出された場合) は`null`を返します。

## 例

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_disable_autorum();
    }
    // ...
}
```
