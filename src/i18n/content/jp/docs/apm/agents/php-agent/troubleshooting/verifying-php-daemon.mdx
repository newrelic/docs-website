## 問題

New Relic の PHP エージェントとデーモンをインストールしましたが、デーモンが機能していないようです。

## 解決

デーモンが正しく動作していることを確認します。

1. このシェルコマンドを実行して、"newrelic-daemon" にマッチする実行中のプロセスをリストアップします。

   ```bash
   ps -ef | grep newrelic-daemon
   ```

2. デーモンが実行されていることを確認するには、前のコマンドの出力で、以下のような2つの実行中のデーモンプロセスを確認します。それらがリストアップされていない場合は、次のステップに進みます。

   ```console
   newrelic     1368     1  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   newrelic     1370  1368  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   ```

3. プロセスが表示されていない場合は、デーモンが動作していないので、デーモンのスタートアップモードに基づいて起動する必要があります。

   * [エージェントモード起動](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-external) （デフォルト）
   * [外部モード起動](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-autostart) (advanced)
