## 構文

```php
newrelic_capture_params([bool $enable_flag])
```

URLパラメータの取得を有効または無効にします。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

URL パラメータ (URL の`?`より後のすべて) のキャプチャを有効または無効にします。これは、 `newrelic.capture_params`構成ファイルの設定をオーバーライドします。

<Callout variant="important">
  機密情報をURLで直接渡す場合は、これを無効にしてください。
</Callout>

## 例

### URLパラメータの取得を可能にする [#enable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params();
    }
    //...
}
```

### URLパラメータのキャプチャを無効にする [#disable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params(false);
    }
    //...
}
```
