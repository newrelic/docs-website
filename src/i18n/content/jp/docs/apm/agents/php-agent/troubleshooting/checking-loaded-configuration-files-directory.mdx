## 問題

**newrelic** モジュールは、 `phpinfo()` ページに表示されますが、UIにデータが表示されず、ライセンスキーも表示されません。

## 解決策

**`newrelic.ini`** ファイルが設定ファイルから読み込まれていない可能性があります。これは、PHPが期待するディレクトリに現れていないためです。 `newrelic.ini` がどこに現れるべきかを決定するために。

1. あなたの [`phpinfo()` ページ](/docs/php/php-kb-103) で、ページの最初のセクションを見てみましょう。
2. `newrelic.ini` が **と書かれたセクションに記載されているかどうかを確認します。 Additional .ini files parsed**
3. `newrelic.ini` ファイルが **と書かれたセクションで指定されたディレクトリに存在することを確認してください。 このディレクトリに追加の.iniファイルがあるかどうかをスキャンしてください** または **追加の.iniファイルが解析されているかどうか** を確認し、Webサーバーがそれを読めるようにパーミッションが適切であるかどうかを確認してください。

`newrelic.ini` が設定ファイルの名前ではない場合、ライセンスキーやその他の設定パラメータを追加したファイルが「読み込まれた」と表示されていることを確認してください。また、そのファイルのパーミッション（アクセス権）がウェブサーバのためのものであることを確認してください。
