## 構文

```
newrelic_ignore_transaction()
```

現在のトランザクションを計測しない。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

このトランザクションのデータを生成しない。これは、既知の理由で特に遅いトランザクションがあり、トランザクションのトレースを頻繁に生成したり、サイトの平均値を歪めたりしたくない場合に便利です。

## パラメータ

このコールはパラメータを受け取りません。

## 例

```
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_ignore_transaction();
    }
    <var>...</var>
}
```
