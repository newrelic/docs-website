ここでは、New Relic PHP エージェントをアンインストールする方法について説明します。エージェントを一時的に無効にする方法については、 [Disable the agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#php) を参照してください。

[本資料では、以下のシステムにおいて、New Relic PHP エージェント](/docs/agents/php-agent/getting-started/new-relic-php) をアンインストールする方法を説明します。

* RedHatとCentOS
* UbuntuとDebian
* Tar アーカイブ (任意の [サポートされたシステム](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#operating-systems))

## エージェントをアンインストールする [#install-overview]

PHP アプリケーションから New Relic をアンインストールする際には、New Relic エージェントを削除し、関連する New Relic 設定ファイルも削除する必要があります。

PHP エージェントをシステムからアンインストールするには、以下の手順に従います。

1. システムからエージェントファイルを削除します。

   <CollapserGroup>
     <Collapser
       id="remove-redhat-centos"
       title="RedHatまたはCentOS"
     >
       `newrelic-install` スクリプトをオプションなしで実行し、スクリプトの起動時にメインメニューからアンインストールオプションを選択します。
     </Collapser>

     <Collapser
       id="remove-ubuntu-debian"
       title="UbuntuまたはDebian"
     >
       `apt-get` を使って、エージェントを削除してください。

       ```
       apt-get remove newrelic-php5
       ```
     </Collapser>
   </CollapserGroup>

2. 残っているNew Relicの設定ファイルを削除します。 `newrelic-install` または `apt-get remove` コマンドに `purge` オプションを付けて使用します。

   ```
   newrelic-install purge
   ```

   または

   ```
   apt-get remove --purge newrelic
   ```

3. パッケージマネージャーを使用している場合は、そのマネージャーに特有のファイルを削除します。

   <CollapserGroup>
     <Collapser
       id="uninstall-redhat-centos"
       title="RedHatまたはCentOS"
     >
       Using `yum`:

       ```
       sudo yum remove newrelic-php5
       ```

       使用方法 **`rpm` (32-bit)**:

       ```
       rpm -e newrelic-php5-common-<var>X.X.X.X (YOUR VERSION #)</var>-1.noarch.rpm newrelic-daemon-<var>X.X.X.X (YOUR VERSION NUMBER)</var>-1.i386.rpm newrelic-php5-<var>X.X.X.X</var>-1.i386.rpm
       ```

       使用方法 **`rpm` (64-bit)**:

       ```
       rpm -e newrelic-php5-common-<var>X.X.X.X (VERSION NUMBER)</var>-1.noarch.rpm newrelic-daemon-<var>X.X.X.X (YOUR VERSION NUMBER)</var>-1.x86_64.rpm newrelic-php5-<var>X.X.X.X</var>-1.x86_64.rpm
       ```
     </Collapser>

     <Collapser
       id="uninstall-ubuntu-debian"
       title="UbuntuまたはDebian"
     >
       使用方法 `apt-get`:

       ```
       sudo apt-get purge newrelic-php5
       ```

       使用方法 **`dpkg` (32-bit)**:

       ```
       dpkg -P newrelic-php5-common_<var>X.X.X.X (YOUR VERSION NUMBER)</var>_all.deb newrelic-daemon_<var>X.X.X.X</var>_i386.deb newrelic-php5_<var>X.X.X.X</var>_i386.deb
       ```

       使用方法 **`dpkg` (64-bit)**:

       ```
       dpkg -P newrelic-php5-common_<var>X.X.X.X (YOUR VERSION NUMBER)</var>_all.deb newrelic-daemon_<var>X.X.X.X</var>_amd64.deb newrelic-php5_<var>X.X.X.X</var>_amd64.deb
       ```
     </Collapser>
   </CollapserGroup>

4. Webサーバー（Apache、Nginx、PHP-FPMなど）を再起動する。
