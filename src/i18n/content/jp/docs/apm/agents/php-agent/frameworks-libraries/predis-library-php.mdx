New Relic は PHP agent [リリース 4.22](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-422099) で Predis ライブラリのサポートを追加しました。

エージェントは、 `Predis\Client`オブジェクトまたはパイプラインへの標準呼び出しを計測します。パイプライン経由で送信されたコマンドはすべてグループ化され、個々のコマンドではなく、 `pipeline`という名前の単一のコマンド メトリックとして扱われます。

PHP エージェントは、 `executeRaw`または`transaction`メソッド、PubSub、モニターなどへの呼び出しをサポートしていません。これらは計測されません。
