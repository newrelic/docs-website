## 構文

```php
newrelic_get_browser_timing_header([bool $include_tags])
```

HTML出力のヘッドに注入するブラウザモニタリングスニペットを返します。

## 説明

HTML 出力の`<head>`に挿入する JavaScript 文字列を返します。この呼び出しを[`newrelic_get_browser_timing_footer`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer)で使用して、手動でブラウザ モニタリングを Web ページに追加します。可能であれば、代わりに New Relic は、New Relic UI を介してブラウザー監視を有効にするか、JavaScript スニペットをコピーして貼り付けることをお勧めします。これらのオプションの使用方法については、[ブラウザの監視と PHP](/docs/agents/php-agent/features/page-load-timing-php)を参照してください。

<Callout variant="tip">
  ページ上のブラウザ スクリプトを**無効にする**[`newrelic_disable_autorum`](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum)を比較します。
</Callout>

## 戻り値

ブラウザエージェントのJavaScriptヘッダー文字列を返します。

## 例

### ウェブページの作成 [#webpage]

```php
function example(){
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_header = newrelic_get_browser_timing_header();
    }
    
    // Output to page
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_footer = newrelic_get_browser_timing_footer();
    }
}
```
