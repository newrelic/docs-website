New Relic では、 **属性** は、 [イベント](/docs/accounts-partnerships/getting-started-new-relic/glossary#event) または [トランザクション](/docs/accounts-partnerships/getting-started-new-relic/glossary#transaction) のプロパティを決定する情報を含むキー・バリュー・ペアです。これらのキー・バリュー・ペアは、アプリケーションをより深く理解したり、 [データを照会したりするのに役立ちます](/docs/query-your-data/explore-query-data/query-builder/introduction-query-builder) 。

## アトリビュートの表示と使用 [#locations]

[デフォルトのAPM属性](/docs/insights/new-relic-insights/decorating-events/apm-default-attributes-insights#transaction-event) と [Cアプリケーション用のカスタム属性](/docs/agents/manage-apm-agents/agent-data/collect-custom-attributes) の両方が表示されます。

* APM [トランザクション・トレース](/docs/apm/transactions/transaction-traces/introduction-transaction-traces) and [エラー・アナリティクス](/docs/apm/applications-menu/error-analytics/introduction-error-analytics)
* [APMイベント](/docs/insights/explore-data/attributes/apm-default-attributes-insights)

## C固有の属性 [#attributes]

<Callout variant="important">
  カスタム属性を作成する前に、NRQL で使用される New Relic の [reserved terms のリストを確認してください](/docs/insights/insights-data-sources/custom-data/insights-custom-data-requirements-limits#reserved-words) 。そうしないと、予期しない結果が発生する可能性があります。
</Callout>
