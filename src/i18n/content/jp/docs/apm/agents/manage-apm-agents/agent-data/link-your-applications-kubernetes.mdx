エージェントは、Kubernetes 関連の環境変数を収集し、それらをトランザクション イベントに追加して、Kubernetes クラスターとサービスの間のコンテキストを提供するように構成できます。特典の詳細については、 [このブログ投稿](https://blog.newrelic.com/engineering/monitoring-application-performance-in-kubernetes/)を参照してください。

エージェントの互換性やインストール方法の詳細については、 [Kubernetes documentation](/docs/integrations/kubernetes-integration/metadata-injection/kubernetes-apm-metadata-injection) をご覧ください。
