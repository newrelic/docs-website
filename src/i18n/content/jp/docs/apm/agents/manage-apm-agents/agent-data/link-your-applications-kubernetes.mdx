エージェントは、Kubernetes関連の環境変数を収集してTransactionイベントに追加し、Kubernetesクラスタとサービス間のコンテキストを提供するように設定できます。メリットの詳細については、このブログ記事 [を参照してください](https://blog.newrelic.com/engineering/monitoring-application-performance-in-kubernetes/ "新しいウィンドウにリンクが開きます。") 。

エージェントの互換性やインストール方法の詳細については、 [Kubernetes documentation](/docs/integrations/kubernetes-integration/metadata-injection/kubernetes-apm-metadata-injection) をご覧ください。
