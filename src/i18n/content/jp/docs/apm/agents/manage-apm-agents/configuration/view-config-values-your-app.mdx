## 問題

アプリケーションの設定情報を簡単に取得する方法が必要です。

## 解決策

設定ファイル自体を見たくない場合は、APMのUIで設定を見ることができます。 **[one.newrelic.com](https://one.newrelic.com)** にアクセスし、 **APM** をクリックします。自分のアプリをクリックして、 **Settings> Environment> Agent initialization** をクリックします。
