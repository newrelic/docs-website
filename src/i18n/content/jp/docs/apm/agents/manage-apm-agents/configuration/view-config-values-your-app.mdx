## 問題

アプリケーションの設定情報を簡単に取得する方法が必要です。

## 解決

設定ファイル自体を表示したくない場合は、 APM UIで設定を表示できます。<DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DNT> に移動してください。 アプリをクリックし、 <DNT>**Settings > Environment > Agent initialization**</DNT>をクリックします。
