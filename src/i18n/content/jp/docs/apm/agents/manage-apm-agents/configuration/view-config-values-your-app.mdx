## 問題

アプリケーションの設定情報を簡単に取得する方法が必要です。

## 解決

設定ファイル自体を表示したくない場合は、APM UI で設定を表示できます。 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**に移動します。アプリをクリックし、 **Settings > Environment > Agent initialization**をクリックします。
