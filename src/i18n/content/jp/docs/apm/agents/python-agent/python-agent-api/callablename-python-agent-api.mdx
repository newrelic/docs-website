## 構文

```py
newrelic.agent.callable_name(object, separator=':')
```

与えられたオブジェクトを識別する文字列名を返します。

## 説明

この呼び出しは、与えられたオブジェクトを識別する文字列名を返します。これは、 [関数トレース](/docs/agents/python-agent/python-agent-api/function_trace) をプログラムで命名する際によく使われます。

## 戻り値

与えられたオブジェクトを識別する文字列名を返します。

返される名前は `module:object_path`の形式になります。指定されたオブジェクトが関数の場合、返される名前は `module:function`の形式になります。オブジェクトがクラスの場合、形式は `module:class`になります。メンバー関数の場合、形式は `module:class.function`になります。

## 例

### 関数トレースの名称設定 [#trace-example]

`callable_name` を使用して [関数トレース](/docs/agents/python-agent/python-agent-api/function_trace)の名前を設定する例を次に示します。

```py
name = callable_name(func)

with FunctionTrace(txn, name):
    func()
```
