## 問題

New Relic UIの[サーバーサイド](/docs/agents/manage-apm-agents/configuration/server-side-agent-configuration)構成設定で <DNT>**Capture attributes**</DNT> または <DNT>**Capture parameters**</DNT> オプションを有効にする従来の動作をエミュレートします。

## 解決

リクエストパラメータをキャプチャするようにPythonエージェントの設定ファイルをカスタマイズすることで、このレガシーな動作をエミュレートすることができます。詳細については、Python attribute documentation の [Capture request parameters example](/docs/agents/python-agent/attributes/python-attribute-examples#ex_req_params) を参照してください。
