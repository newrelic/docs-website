## 構文

```
newrelic.agent.get_browser_timing_footer()
```

JavaScriptエージェントを注入するためのブラウザエージェントフッターを生成します。

## 説明

このコールは、ブラウザエージェントのJavaScriptフッターのスニペットを返します。このスニペットは、 [ヘッダーのJavaScriptスニペット](/docs/agents/python-agent/python-agent-api/get_browser_timing_header) と共に、ブラウザモニタリングを有効にするために使用されます。このコールを `get_browser_timing_header` と一緒に使用してブラウザの監視を有効にする方法については、 [手動インスツルメンテーションの説明書](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) を参照してください。

この呼び出しは、JavaScriptスニペットを注入するすべてのWebリクエストに対して行う必要があります。 \*\*\*\*グローバルスコープで呼び出してキャッシュすることはできません。

## 戻り値

ブラウザエージェントのフッターJavaScriptを返します。
