## 構文

```py
newrelic.agent.get_browser_timing_footer()
```

JavaScriptエージェントを注入するためのブラウザエージェントフッターを生成します。

## 説明

この呼び出しは、ブラウザー エージェントの JavaScript フッター スニペットを返します。このスニペットは、 [ヘッダー JavaScript スニペット](/docs/agents/python-agent/python-agent-api/get_browser_timing_header)とともに、ブラウザーの監視を有効にするために使用されます。この呼び出しを `get_browser_timing_header` で使用してブラウザ監視を有効にする方法については [、手動のインストルメンテーション手順を](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) 参照してください。

この呼び出しは、JavaScriptスニペットを挿入するすべてのWebリクエストに対して呼び出す必要があります。グローバルスコープで呼び出してキャッシュ**することはできません**。

## 戻り値

ブラウザエージェントのフッターJavaScriptを返します。
