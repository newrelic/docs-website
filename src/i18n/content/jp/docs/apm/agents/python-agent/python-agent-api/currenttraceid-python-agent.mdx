## 構文 [#syntax]

```py
    newrelic.agent.current_trace_id()
```

現在のトランザクションのトレース ID を返すか、トランザクションが存在しない場合は`None`を返します。

## 説明 [#description]

現在の[トランザクション](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction)のトレース ID を取得するには、 `current_trace_id`を使用します。

## 戻り値 [#return-values]

現在のトランザクションのトレース ID を返します。 アクティブなトランザクションがない場合は、 `None`を返します。

## 例 [#examples]

### 現在のトレース ID を取得する [#function-trace-example]

```py
    import newrelic.agent

    @newrelic.agent.background_task()
    def main():
        trace_id = newrelic.agent.current_trace_id()
```
