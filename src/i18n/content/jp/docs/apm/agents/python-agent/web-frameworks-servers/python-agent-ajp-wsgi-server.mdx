Python エージェントは、 [flup](https://pypi.python.org/pypi/flup) と組み合わせて AJP で使用することができます。Pythonエージェントのインストール手順については、 [Install the Python agent](/docs/agents/python-agent/installation-configuration/python-agent-installation) を参照してください。エージェントを AJP で使用するための特別な要件や注意点はありません。
