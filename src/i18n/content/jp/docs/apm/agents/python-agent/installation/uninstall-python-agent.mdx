このドキュメントでは、Pythonエージェントをアンインストールする方法を説明します。エージェントを一時的に無効にする方法については、 [Disable the agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#python) を参照してください。

Pythonエージェントをアンインストールするには

1. 以下のいずれかの方法でご利用ください。

   * PIPでインストールした場合は、実行してください。

     ```
     pip uninstall newrelic
     ```

   * easy_installでインストールした場合は、実行してください。

     ```
     easy_install -m newrelic
     ```

2. アンインストールが完了したら、アプリを再起動してください。

<Callout variant="tip">
  また、 [APMのUIからアプリ](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) を削除することもできます。
</Callout>
