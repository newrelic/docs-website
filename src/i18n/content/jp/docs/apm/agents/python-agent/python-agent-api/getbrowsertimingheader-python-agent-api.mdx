## 構文

```
get_browser_timing_header()
```

JavaScriptエージェントを注入するためのブラウザエージェントヘッダを生成します。

## 説明

このコールは、ブラウザエージェントのJavaScriptヘッダースニペットを返します。このヘッダースニペットを使って、 [ブラウザモニタリング](/docs/agents/python-agent/supported-features/page-load-timing-python) を有効にすることができます。これは、アプリケーションで自動計測ができない場合や、特定のページだけのブラウザデータを収集したい場合などに使用します。

これは、 [エージェントによるこのJavaScript](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#select-apm-app) の自動挿入に代わる手動によるものです。このメソッドを [`get_browser_timing_footer`](/docs/agents/python-agent/python-agent-api/get_browser_timing_footer) で使用してブラウザの監視を可能にする方法については、 [手動インスツルメンテーションの説明書](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) を参照してください。

この呼び出しは、JavaScriptスニペットを注入するすべてのWebリクエストに対して行う必要があります。 \*\*\*\*グローバルスコープで呼び出してキャッシュすることはできません。

## 戻り値

ブラウザエージェントヘッダーのJavaScriptを返します。
