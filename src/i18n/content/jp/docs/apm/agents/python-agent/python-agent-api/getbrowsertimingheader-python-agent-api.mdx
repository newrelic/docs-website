## 構文

```py
get_browser_timing_header()
```

JavaScriptエージェントを注入するためのブラウザエージェントヘッダを生成します。

## 説明

このコールは、ブラウザエージェントのJavaScriptヘッダースニペットを返します。このヘッダースニペットを使って、 [ブラウザモニタリング](/docs/agents/python-agent/supported-features/page-load-timing-python) を有効にすることができます。これは、アプリケーションで自動計測ができない場合や、特定のページだけのブラウザデータを収集したい場合などに使用します。

これは、エージェントによる [この JavaScript の自動挿入](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#select-apm-app) に代わる手動の方法です。 [このメソッドを](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) [`get_browser_timing_footer`](/docs/agents/python-agent/python-agent-api/get_browser_timing_footer) で使用してブラウザ監視を有効にする方法については、手動のインストルメンテーション手順を参照してください。

この呼び出しは、JavaScriptスニペットを挿入するすべてのWebリクエストに対して呼び出す必要があります。グローバルスコープで呼び出してキャッシュ**することはできません**。

## 戻り値

ブラウザエージェントヘッダーのJavaScriptを返します。
