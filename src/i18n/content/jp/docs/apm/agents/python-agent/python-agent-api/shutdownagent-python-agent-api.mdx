## 構文

```py
newrelic.agent.shutdown_agent(timeout=None)
```

エージェントを強制的にシャットダウンします。

## 説明

エージェントを強制的にシャットダウンします。エージェントは、データを [コレクター](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector) にアップロードする最後の試みを行います。

      <td>
        `timeout`

        _int_ or _float_
      </td>

      <td>
        任意ですが、強く推奨します。エージェントがデータのアップロードを終了するまで呼び出し側が待機する最大時間です。エージェントが終了する前にタイムアウト期間が終了した場合、エージェントはバックグラウンドでデータを報告し続けますが、呼び出し側はプロセス全体を強制的に終了させ、アップロードを停止させることができます。

        値が設定されていない場合、または `None`に設定されている場合、エージェントはデフォルトのグローバル タイムアウト値を使用します。この値は構成設定で上書きできますが、デフォルトは 2.5 秒です。
      </td>
    </tr>

</table>

## 戻り値

なし。
