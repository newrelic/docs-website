## 構文

```
newrelic.agent.ignore_transaction(flag=True)
```

現在のトランザクションを無視します。

## 説明

このコールは、現在のトランザクションを無視します。エージェントは、トランザクションからのデータを報告しません。

<Callout variant="tip">
  また、WSGI environ 辞書のWebトランザクションを無視することもできます。そのためには、WSGIサーバからWSGIアプリケーションに渡されるWSGI environ辞書の特定のリクエストに対して、 `newrelic.ignore_transaction` キーを設定します。
</Callout>

## 例

### トランザクションを無視する [#ignore-example]

New Relic One に表示させたくないトランザクションを無視するには、アプリのコードでトランザクションが生成されている場所で次のように実行します。この例では、API を使用して Flask ルートを無視する方法を示しています。

```
@app.route("/")
def process_results():
    newrelic.agent.ignore_transaction(flag=True)
    return 'hello world'
```
