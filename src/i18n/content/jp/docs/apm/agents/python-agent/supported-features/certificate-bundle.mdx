## よくあるご質問 [#faq]

### サーティフィケート・バンドルとは何ですか？ [#what-is-certificate-bundle]

証明書バンドルは、基本的にトラストルートのリストを提供します。バックエンドと通信する際に誰を信頼するかをエージェントに伝えます。

### なぜ証明書のバンドルが提供されるのですか？ [#why-certificate-bundle]

多くのお客様はdockerで実行しています。それらのイメージは、通常HTTPでの通信には使用されないため、独自の証明書バンドルを提供していないことがあります。私たちは独自のバンドルを提供することで、お客様がより簡単にエージェントをインストールできるようにしています。

### 証明書バンドルはどのように使用されますか？ [#when-to-use-certificate-bundle]

システム証明書が利用可能な場合、エージェントはデフォルトで、同梱されている証明書バンドルよりもそれらを使用するようになります。証明書バンドルは、システムから証明書が提供されない場合にのみ使用されます。
