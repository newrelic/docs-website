[FastAPI](https://fastapi.tiangolo.com/) Web フレームワークを使用するアプリに当社の Python エージェントを統合するための要件とヒントについてお読みください。一般的なインストール手順に戻るには、 [Install the Python agent](/docs/agents/python-agent/installation-configuration/python-agent-installation) を参照してください。

## 管理用スクリプトによる自動初期化 [#wrapper-script]

`python app.py` でアプリを起動し、当社のPythonエージェント **バージョン5.22.0.151** 以上を使用している場合は、推奨される [admin script integration method](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) を使用することができます。

ここでは、アドミン・スクリプト方式の例を紹介します。

```
NEW_RELIC_CONFIG_FILE=<var>path/to/</var>newrelic.ini newrelic-admin run-program [web server] path_to_app
```

## PythonエージェントAPI [#api]

[Python エージェント API](/docs/agents/python-agent/python-agent-api) を使って、任意の ASGI サーバやフレームワークを監視することができます。そのためには、 [ASGIアプリケーションのエントリーポイント](/docs/agents/python-agent/python-agent-api/asgiapplication-python-agent-api) をマークし、 [トランザクション名](/docs/agents/python-agent/python-agent-api/settransactionname-python-agent-api) を設定します。

## イベントループ診断対応 [#event-loops]

当社のPythonエージェントは、 `asyncio` イベントループ診断をサポートしています。詳しくは、 [Pythonのイベントループ診断](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics).cs)をご覧ください。
