## 構文

```py
newrelic.agent.end_of_transaction()
```

現在のトランザクションを終了します。

## 説明

この呼び出しは、現在のトランザクションを直ちに終了させ、応答タイマーを現在の時間で停止させます。トランザクションが終了すると、エージェントは追加のパフォーマンスブレークダウンメトリクスを収集しません。ただし、エージェントはPythonの例外の詳細を収集します。

この呼び出しは、ウェブトランザクションのためにストリームバックされる大きな応答があり、トランザクションの最初の部分だけが興味の対象である場合に便利です。(これは、何らかの理由で、トランザクションを [バックグラウンドタスク](/docs/agents/python-agent/python-agent-api/set_background_task) としてフラグを立てたくない場合も想定しています。)

## パラメーター

なし。

## 戻り値

なし。

## 例

### トランザクションの終了 [#end-txn-example]

トランザクションを早期に終了する。

```py
newrelic.agent.end_of_transaction()
```
