Python エージェントは、 [asyncpg](https://pypi.org/project/asyncpg/) database client module for [PostgresSQL](https://www.postgresql.org/) を使用する際のデータベーストランザクションの記録をサポートしています。この機能は、データベースクエリのタイミング、データベースクエリのSQL文、長いデータベースクエリのスタックトレースをキャプチャします。

[asyncpg](https://pypi.org/project/asyncpg/) を使用するアプリに Python エージェントを統合するための要件とヒントについてお読みください。一般的なインストール手順に戻るには、 [Install the Python agent](/docs/agents/python-agent/installation-configuration/python-agent-installation) を参照してください。

## 管理用スクリプトによる自動初期化 [#wrapper-script]

`python app.py` でアプリを起動し、当社のPythonエージェント **バージョン5.20.0.149** 以上を使用している場合は、推奨される [アドミンスクリプトの統合方法](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) 例えば、以下のようになります。

```
NEW_RELIC_CONFIG_FILE=<var>path/to/</var>newrelic.ini newrelic-admin run-program python path_to_app
```

## PythonエージェントAPIの使用 [#api]

[Python エージェント API](/docs/agents/python-agent/python-agent-api) を使って、任意の ASGI サーバやフレームワークを監視することができます。そのためには、 [ASGIアプリケーションのエントリーポイント](/docs/agents/python-agent/python-agent-api/asgiapplication-python-agent-api) をマークし、 [トランザクション名](/docs/agents/python-agent/python-agent-api/settransactionname-python-agent-api) を設定します。

## イベントループ診断対応 [#event-loops]

当社のPythonエージェントは、 `asyncio` イベントループ診断をサポートしています。詳細については、 [Python イベントループ診断](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics) を参照してください。
