Web2pyは、WebベースのPythonアプリケーションを作成するためのオープンソースのフレームワークです。PythonエージェントをWeb2pyアプリに統合して、アプリのパフォーマンスを監視することができます。一般的なインストール手順については、 [Pythonエージェントのインストール](/docs/agents/python-agent/installation-configuration/python-agent-installation) を参照してください。

## Web2pyとの統合 [#web2py-integration]

詳細については、以下のリストから使用しているWeb2pyのセットアップを選択してください。

<CollapserGroup>
  <Collapser
    id="running-with-runweb2py"
    title="Runweb2pyコマンド"
  >
    **runweb2py** コマンドを使用してウェブアプリケーションを実行した場合は、次のコマンドを使用します。

    ```
    NEW_RELIC_CONFIG_FILE=<var>/some/path/</var>newrelic.ini newrelic-admin run-program runweb2py
    ```

    `newrelic-admin` は、エージェントがアプリケーションの主要機能を監視できるように、アプリケーションのスタートアップをラップするスクリプトです。ラッパースクリプトの実行については， [ラッパースクリプトの実行](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) を参照してください．
  </Collapser>

  <Collapser
    id="running-with-mod_wsgi"
    title="Apache/mod_wsgi"
  >
    Apache/mod_wsgi を使用している場合、 **Python エージェントを初期化するために、使用している WSGI スクリプトファイルに** 手動でコードを挿入する必要があります。詳細については、 [mod_wsgi documentation](/docs/python/python-agent-and-mod_wsgi) を参照してください。
  </Collapser>

  <Collapser
    id="running-with-gunicorn"
    title="グニコーン"
  >
    エージェントとGunicornの使用に関する情報は、 [Python agent and Gunicorn documentation](/docs/agents/python-agent/hosting-mechanisms/python-agent-and-gunicorn) を参照してください。
  </Collapser>

  <Collapser
    id="running-with-uwsgi"
    title="uWSGI"
  >
    エージェントでのuWSGIの使用に関する情報は、 [Python agent and uWSGI documentation](/docs/agents/python-agent/hosting-mechanisms/python-agent-and-uwsgi) を参照してください。
  </Collapser>
</CollapserGroup>
