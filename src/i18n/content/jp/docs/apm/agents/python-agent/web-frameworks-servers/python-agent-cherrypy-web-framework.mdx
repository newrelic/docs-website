このドキュメントには、Pythonエージェントを [CherryPy WSGIサーバ](http://docs.cherrypy.org/) を使用するアプリに統合するための要件とヒントが含まれています。一般的なインストール手順については、 [Pythonエージェントのインストール](/docs/agents/python-agent/installation-configuration/python-agent-installation) を参照してください。

## Adminスクリプトのインテグレーションメソッド [#integrated-server]

`CherryPyWSGIServer` API関数を直接使用する場合は、推奨されるadminスクリプトの統合方法を使用できます。推奨されるアドミンスクリプトの方法は、一般的な [インストール手順](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) に記載されています。

## PasteDeployの使い方 [#using-paste-deploy]

アプリケーションコードにCherryPy WSGIサーバを起動するコールを手動で組み込むのではなく、以下のような構成でPasteDeployを使用している場合。

```
[server:main]
use = egg:PasteScript#cherrypy
host = 127.0.0.1
port = 8080
```

WSGIアプリケーションを起動するために必要なのは、 `paster` コマンドの実行をラップすることだけです。

```
NEW_RELIC_CONFIG_FILE=newrelic.ini newrelic-admin run-program paster serve production.ini
```
