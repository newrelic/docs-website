## 構文

```py
newrelic.agent.database_trace(sql, dbapi2_module=None, async_wrapper=None)
```

データベースへの呼び出しを計装するために使用されます。

## 説明

`database_trace` 追加のセグメントという形で、[トランザクショントレース](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction-trace)により詳細な情報を追加するために使用されます。`database_trace`で報告されたすべてのコールは、[APM データベースページ](/docs/apm/applications-menu/monitoring/databases-slow-queries-page)に表示されます。`database_trace`は、データベースへの呼び出しを計測する関数のデコレータとして使用できる`DatabaseTraceWrapper`の[partial](https://docs.python.org/3.12/library/functools.html#functools.partial)を返します。

エージェントバージョン2.102.0.85以降では、ジェネレータおよびコルーチンで`database_trace`デコレータを使用できます。

アプリケーションでデコレーターを使用できない場合は、これらの他の呼び出しフォーマットのいずれかを使用することができます。

* <DNT>**The context manager**</DNT>: コンテキスト マネージャー フォームは`DatabaseTrace`です。
* <DNT>**The wrapper**</DNT>: ラッパーフォームは`DatabaseTraceWrapper`です。 デコレータを使用せずにラップされた関数を返すために使用できます。
* <DNT>**The path-based wrapper**</DNT>: パスベースのラッパー形式は`wrap_database_trace`です。 これは、モンキーパッチングを通じて、指定されたオブジェクトに`DatabaseTraceWrapper`を適用します。 これは、デコレータと同じ引数に加えて、追加の`module`と`object_path`引数を受け取ります。

これらの異なるコールフォーマットの用途についての説明は、 [Different call formats](/docs/python-agent-api-different-call-forms) を参照してください。通話例については、 [例](#examples) をご覧ください。

## 戻り値

`database_trace` `DatabaseTraceWrapper()`パーシャルを返します。

## 例

### `database_trace` [#trace\_example][#trace_example]

`database_trace`デコレーターの使用例:

```py
import newrelic.agent

@newrelic.agent.database_trace(sql="SELECT * FROM users WHERE id = 1")
def get_user(id):
    ...
```

ネイティブ コルーチンで`database_trace`デコレータを使用する例:

```py
import newrelic.agent

@newrelic.agent.database_trace(sql="SELECT * FROM users WHERE id = 1")
async def get_user(id):
    ...
```

### コンテクストマネージャーの例 [#context-mgr-example]

`DatabaseTrace`コンテキスト マネージャーの使用例: これにより、\`custom\_action\` の実行にかかる時間のタイミングが得られます。

```py
import newrelic.agent


def complex_query(a, b, c):
    with Connection(host, port, db) as conn:
        with newrelic.agent.DatabaseTrace(
            sql="SELECT * FROM users WHERE id = 1",
        ):

            conn.custom_action()
```

### ラッパーの例 [#trace-wrap-example]

`DatabaseTraceWrapper`の使用例:

```py
import newrelic.agent

def get(id):
    ...


get = newrelic.agent.DatabaseTraceWrapper(
    get, sql="SELECT * FROM users WHERE id = 1"
)
```
