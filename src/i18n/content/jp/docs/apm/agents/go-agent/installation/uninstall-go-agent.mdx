New Relic Go エージェントをアンインストールする方法については、こちらをお読みください。Go エージェントを一時的に無効にする方法については、 [Disable the agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#go) を参照してください。

New Relic for Goをアンインストールするには

1. お好みのアンインストール方法でNew Relicパッケージをアンインストールします。
2. 再コンパイルして、アプリケーションを再起動します。

New Relic パッケージをアンインストールすると、 [New Relic アカウントからアプリケーションを削除することができます](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) 。
