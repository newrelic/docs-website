New Relic の最新機能、拡張機能、重要なセキュリティパッチを最大限に活用するには、アプリの Go エージェントを最新バージョンに更新してください。特定のエージェントのアップデートについての詳細は、 [Go エージェントのリリースノート](/docs/release-notes/agent-release-notes/go-release-notes) を参照してください。

## Goエージェントのバージョンの更新 [#update-go]

Goエージェントをアップデートするには、標準的な手順に従って、以下のプロセスをターミナルコマンドまたはスクリプトとして実行します。

1. [http://github.com/newrelic/go-agent](http://github.com/newrelic/go-agent) から、このプロセスを使用します。

   ```
   go get -u github.com/newrelic/go-agent/v3/newrelic
   ```

2. アプリケーションをコンパイル、ディプロイします。
