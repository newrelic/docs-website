[APM](/docs/apm) ユーザーインターフェースでは、 [APM用Goエージェント](/docs/agents/go-agent/get-started/new-relic-go) の特別な機能など、アプリに関するさまざまなデータを提供しています。また、 **Go runtime** ページでは、パフォーマンス問題のトラブルシューティングに役立つ重要なランタイムデータを提供しています。

## Goのランタイムデータの表示とフィルタリング [#find]

**Go runtime** ページを見るには、 [**one.newrelic.com**](http://one.newrelic.com) **> APM> (select an app)> Go runtime**.

![ランタイムページへ](./images/go-runtime.png "ランタイムページへ")
