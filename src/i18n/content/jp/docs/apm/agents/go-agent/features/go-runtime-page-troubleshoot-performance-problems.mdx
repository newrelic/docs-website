[APM](/docs/apm)ユーザー インタフェース[は、 APMの Go エージェント用](/docs/agents/go-agent/get-started/new-relic-go)の特別な 機能 を含む、アプリに関するさまざまなデータを提供します。 さらに、 <DNT>**Go runtime**</DNT>ページには、パフォーマンスの問題のトラブルシューティングに役立つ重要なランタイム データが提供されます。

## Goのランタイムデータの表示とフィルタリング [#find]

<DNT>**Go runtime**</DNT>ページを表示するには、 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (select an app) > Go runtime**</DNT>に移動してください。
