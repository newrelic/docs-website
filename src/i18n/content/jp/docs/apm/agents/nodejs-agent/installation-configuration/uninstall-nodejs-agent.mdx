New Relic Node.js エージェントをアンインストールする方法については、こちらをご覧ください。エージェントを一時的に無効にする方法については、 [Disable the agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#node) を参照してください。

New Relic Node.js エージェントをアンインストールするには、以下の手順に従います。

1. `require('newrelic');` をアプリのメインモジュールの1行目として削除します。
2. `npm uninstall newrelic -save` で npm パッケージを削除します。
3. `newrelic.js` ファイルを削除し、使用されている [環境変数](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration#methods-and-precedence) の設定を解除します。
4. アンインストール処理が終了したら、 [アプリを再起動してください。](#restart)
5. [](#restart)

[](#restart)

[インストールの詳細については、 ](#restart)[Installing and maintaining Node.js](/docs/agents/nodejs-agent/installation-configuration/installing-maintaining-nodejs) を参照してください。

<Callout variant="tip">
  また、 [アプリ](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) を New Relic から削除することもできます。
</Callout>
