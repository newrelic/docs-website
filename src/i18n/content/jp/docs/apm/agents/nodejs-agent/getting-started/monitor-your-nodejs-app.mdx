New RelicでNode.jsアプリケーションの監視をすぐに開始したいですか？このガイドを使用して、アプリケーションデータをNewRelicに送信し始めるための最適なパスを見つけてください。

始める前にもっと学びたいですか？[APM](/docs/apm/new-relic-apm/getting-started/introduction-apm)の[概要とNode.jsのNewRelicの概要](/docs/apm/agents/nodejs-agent/getting-started/introduction-new-relic-nodejs)は、開始するのに適した場所です。一般的なインストールプロセスの詳細については、 [Node.jsのインストールの概要](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent)をご覧ください。

## 何を監視したいですか？ [#what]

監視対象に応じて、Node.jsエージェントをインストールするためのパスがいくつかあります。

<CollapserGroup>
  <Collapser
    id="docker"
    title="Dockerコンテナにデプロイされたアプリ"
  >
    Dockerコンテナにエージェントをインストールするには、コマンドラインを使用して次[のセルフガイドの手順](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent-docker)に従います。

    このプロセスでは、エージェントを`package.json`に追加してから、アプリレベルでNewRelicを要求する手順を説明します。最後に、ライセンスキーとアプリ名を`docker run`に追加して、エージェントがDockerアプリで実行されるようにします。
  </Collapser>

  <Collapser
    id="server"
    title="サーバーにデプロイされたアプリ"
  >
    サーバーでホストされているアプリには、いくつかの異なるオプションがあります。

    * [ガイド付きインストール](https://one.newrelic.com/marketplace/install-data-source?state=f535d53b-5b84-cd48-80cf-61704ad02e29)では、インストールをいくつかのステップに分けています。コマンドラインとの対話が必要です。
    * [Node.jsエージェントのインストール](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent)ドキュメントでは、コマンドラインのみを使用しています。[Node.jsエージェント](https://github.com/newrelic/node-newrelic#installation)リポジトリで同じプロセスを表示できます。
    * PM2 プロセス マネージャーを使用するアプリの場合は、 [ガイド付きインストール](https://one.newrelic.com/marketplace/install-data-source?state=ba2c3ecb-defb-e99a-c4be-47d786690d2a)に従ってください。PM2 ではないプロセス マネージャーを使用している場合は、ガイド付きまたは手動のインストール オプションを選択します。

    アプリを監視する準備ができたら、アプリのコードの最初の行に`$ node -r newrelic your-program.js`を追加します。これは、アプリへのrootアクセス権がない場合に特に重要です。
  </Collapser>

  <Collapser
    id="lambda"
    title="AWSLambdaでデプロイされたアプリ"
  >
    AWS Lambda 関数のエージェントをインストールするために、 [セルフガイドの手順](https://one.newrelic.com/marketplace/install-data-source?state=9aa3d004-6f13-1477-604e-b6a2d6bb992d)で、AWS を自動的に構成するセットアップ スクリプトが提供されます。

    コマンドラインを介した手動インストールをご希望の場合は、現在[、Lambdaモニタリング用のレガシーインストルメンテーション](https://docs.newrelic.com/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy/#node)のオプションがあります。
  </Collapser>
</CollapserGroup>
