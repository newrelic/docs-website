ステップ1で、newrelic-lambda CLIを使用した場合、 [APIポーリング](/docs/infrastructure/amazon-integrations/connect/connect-aws-new-relic-infrastructure-monitoring)または[メトリックストリーム](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream)統合のいずれかがインストールされているかどうかがチェックされました。どちらも表示されない場合は、APIポーリング統合をインストールしました。

手順1でCLIを使用しなかった場合は、APIポーリングまたはメトリックストリーム統合のいずれかをインストールして構成したことを確認してください。
