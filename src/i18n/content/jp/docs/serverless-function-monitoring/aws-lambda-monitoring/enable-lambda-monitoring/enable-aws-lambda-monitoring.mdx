Lambdaのモニタリングを有効にするには、いくつかのステップがあります。

1. [AWSアカウントをリンクします。](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/account-linking)
2. [Optional: configure CloudWatch Log collection](/docs/logs/enable-log-management-new-relic/enable-log-monitoring-new-relic/aws-lambda-sending-cloudwatch-logs): ステップ1で、 `newrelic-lambda` CLIを使用した場合は、デフォルトで自動的にCloudWatch logs collectorがインストールされるので、このステップはスキップできます。
3. [関数の例をインストゥルメントします。](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-example)
4. [自分の機能を計る。](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-your-own)
5. [オプションです。アラートやカスタムイベントを設定します。](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/configure-serverless-monitoring-aws-lambda)
