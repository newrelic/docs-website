<figcaption>
  **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities > Amazon Web Services > Lambda functions**: Lambda 関数をクリックしてチャートと詳細を表示します。
</figcaption>

New Relic で Lambda データを表示するには: **[one.newrelic.com](https://one.newrelic.com/all-capabilities)**に移動し、をクリックします**。\*\***アマゾン ウェブ サービス**の下の左側のナビゲーションで、\[ **Lambda 関数] をクリックします。\*\*

UIの詳細について[は、「NewRelicプラットフォームの概要](/docs/project-wanda-early-access-future-new-relic-platform)」を参照してください。

<Callout variant="important">
  Lambdaのデータが見つからない場合。

  * Lambdaモニタリングを有効にするための [の指示に従っていることを確認してください](/docs/install-enable-new-relics-monitoring-aws-lambda) 。
  * なお、この機能は、 [インフラ監視のLambda統合](/docs/integrations/amazon-integrations/aws-integrations-list/aws-lambda-monitoring-integration) とは異なります。
</Callout>
