## 問題

AWS Secrets Manager を使用して New Relic ライセンスキーを保存することはできません。

## 解決

組織で AWS Secrets Manager の使用が許可されていない場合、New Relic Lambda Extension は`NEW_RELIC_LICENSE_KEY`環境変数を受け入れます。`newrelic-lambda integrations install`コマンドから`--disable-license-key-secret`フラグを追加します。次に、この環境変数をLambda関数設定の <InlinePopover type="licenseKey" />に設定します。
