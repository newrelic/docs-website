このドキュメントは、合成モニタのバージョン0.4.x以下を対象としています。 [合成モニターのバージョン0.5以降のドキュメントもご覧ください](/docs/synthetics/new-relic-synthetics/scripting-monitors/synthetics-scripted-browser-reference-monitor-versions-050).

## 概要

合成 [スクリプト付きブラウザ](/docs/synthetics/new-relic-synthetics/scripting-monitors/write-scripted-browsers) は、変数 `$driver` と `$browser` を通じて、 [Selenium Webdriver APIs 2.47.0](http://nr-synthetics-sw-jsdoc.s3-website-us-east-1.amazonaws.com/) へのアクセスを提供します。特に、この

* `$driver`は、`selenium-webdriver`モジュールからのすべてのエクスポートを提供します（たとえば、`ActionSequence`、`Button`、`By`、`WebElement`など）。
* `$browser` は、 `selenium-webdriver.WebDriver()` の合成風味のインスタンスです。これは、 `get()` や `findElement()` のような主な基本的な `WebDriver` API、およびいくつかの合成カスタムAPIを公開します。

本ドキュメントでは、合成スクリプト付きブラウザモニターのバージョン0.4.0以下で利用可能な機能について説明しています。最新のモニタードキュメントについては、 [モニターバージョン0.5.0+ドキュメント](/docs/synthetics/new-relic-synthetics/scripting-monitors/synthetics-scripted-browser-reference-version-050) をご覧ください。

その他の関連文書

* シンセティック・スクリプティングについては、 [Write scripted browsers](/docs/synthetics/new-relic-synthetics/scripting-monitors/write-scripted-browsers) をご覧ください。
* スクリプトの例については、 [Scripted browser examples](/docs/synthetics/new-relic-synthetics/scripting-monitors/scripted-browser-examples) をご覧ください。
* モニターのバージョンとランタイムの違いについては、 [ランタイム環境](/docs/synthetics/new-relic-synthetics/scripting-monitors/scripted-monitor-runtime-environment) を参照してください。
* スクリプト化されたブラウザの例を見たり共有したりするには、New Relic の [Explorers Hub](https://discuss.newrelic.com/c/support-products-agents/synthetics) で、 [synthetic-script](https://discuss.newrelic.com/tags/c/support-products-agents/synthetics/synthetics-script) というタグの付いたトピックをチェックしてください。

## ActionSequence：複数のアクションをリンク [#actionsequence]

アクションシーケンスを使用すると、Webサイトとの複雑なユーザーインタラクションを作成できます。

* アクションシーケンスを作成するには、[`$browser.actions()`](#browser-actions)を使用します。
* 複数のアクションを1つのシーケンスにリンクするには、それぞれのアクションの後に[`perform()`](#actionSequence-perform)を含めます。これにより、シングルアクションのシーケンスを含む個々のシーケンスが実行された後、終了されます。

利用可能なアクションの一覧を以下の表に示します。 [詳細については、WebDriver ActionSequence documentation](http://seleniumhq.github.io/selenium/docs/api/javascript/module/selenium-webdriver/lib/actions_exports_ActionSequence.html) をご覧ください。
