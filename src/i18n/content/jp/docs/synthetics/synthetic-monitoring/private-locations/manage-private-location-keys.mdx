New Relic Syntheticsの[プライベートロケーションの](/docs/synthetics/synthetic-monitoring/private-locations/private-locations-overview-monitor-internal-sites-add-new-locations)UIが更新されました。プライベートロケーション キーはシークレット ストアに移行され、インターフェイスのいくつかの領域からキーの可視性が削除されました。

## 何が変わるのか [#whats-changing]

* **リスト表示**：プライベートロケーションリストからキー列が削除されました。
* **フィルター**：キーによるプライベートロケーションのフィルターはできなくなりました。
* **メタデータ &amp; タグ**: プライベートロケーション キーがメタデータ &amp; タグ セクションに表示されなくなりました。
* **キーへのアクセス**: キーは編集画面からのみアクセス可能になり、そこで[外形監視ジョブマネージャー](/docs/synthetics/synthetic-monitoring/private-locations/install-job-manager)設定用にコピーできます。

<figcaption>
  <DNT>**[one.newrelic.com &gt; Synthetic monitoring &gt; Private locations](https://one.newrelic.com/synthetics-nerdlets/private-location-list)**</DNT>: プライベートロケーションリストにキー列が表示されなくなりました。キーは「編集」メニューからアクセスできます。
</figcaption>

## プライベートロケーションキーにアクセス [#access-key]

プライベートロケーションキーにアクセスするには：

1. <DNT>**[one.newrelic.com &gt; Synthetic monitoring &gt; Private locations](https://one.newrelic.com/synthetics-nerdlets/private-location-list)**</DNT>に移動します。
2. キーが必要なプライベートロケーションを見つけます。
3. 省略記号アイコンをクリックしてください<Icon name="fe-more-horizontal" />その場所のすぐ隣。
4. <DNT>**Edit**</DNT>を選択します。
5. 編集画面で、外形監視ジョブマネージャー設定の<DNT>**Private location key**</DNT>フィールドからキーをコピーします。

<img title="Private location edit screen with key" alt="Screenshot showing the Edit screen where you can access and copy the private location key" src="/images/synthetic_screenshot-crop_private-location-edit-page.webp" />

<figcaption>
  編集画面は、プライベートロケーションキーを表示およびコピーできる唯一の場所です。
</figcaption>

## プライベートロケーションをフィルターして検索 [#filter-locations]

キーフィルターが削除されました。次の方法でプライベート位置をフィルタリングして検索できるようになりました。

* **場所名**: プライベートロケーションに割り当てた名前で検索します。
* **ステータス**：アクティブな場所と非アクティブな場所で絞り込みます。
* **[VSE（検証済みスクリプト実行）](/docs/synthetics/synthetic-monitoring/private-locations/verified-script-execution-private-locations)** ：VSEが有効になっている場所でフィルタリングします。
