**モニターのダウンタイムをスケジュールする** 合成の [モニター](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors#types-monitors) がアラートを停止する時間を指定し、同時に [SLAレポートのメトリクスを維持する](/docs/synthetics/new-relic-synthetics/pages/synthetics-sla-report-aggregate-monitor-metrics) 。モニター・ダウンタイムは次のような場合に最適です。

* 定期的なメンテナンス
* 計画停電
* デプロイメント
* サービスの中断

<Callout variant="important">
  シンセティックモニターのダウンタイム時には、選択したモニターは予定された終了時刻まで動作を停止します。モニターを一時停止せずにアラートを一時的に無効にするには、代わりに [ミュート](/docs/synthetics/new-relic-synthetics/using-monitors/alerting-synthetics#mute) を設定してください。
</Callout>

## 定期的なモニターダウンタイムの作成 [#create-window]

<Callout variant="tip">
  モニターのダウンタイムを追加、編集、削除できるかどうかは、 [機能へのアクセスに依存します](/docs/accounts/accounts-billing/general-account-settings/factors-affecting-access-features-data) 。
</Callout>

定期的なモニター・メンテナンスや定期的な停止のために、定期的なモニター・ダウンタイムを作成します。定期的なモニター・ダウンタイムを作成するには

1. Go to [**one.newrelic.com**](http://one.newrelic.com/) **> Synthetics> Monitor downtime**.
2. Select **<Icon name="fe-plus-circle"/>Create monitor downtime**.
3. モニターのダウンタイムの名前を指定します。
4. 毎日、毎週、毎月の頻度を選択します。曜日や時間帯、ウィンドウの表示時間など、追加のスケジュールオプションを選択できます。
5. モニターダウンタイムの対象となるモニターを選択します。選択できるモニターの数に上限はありません。
6. **「Create」**を選択します。

## ワンタイムモニターのダウンタイムを作る [#one-time]

突発的なメンテナンスやサービスの中断のために、再発しない1回限りのモニター・ダウンタイムを作成します。再発しないモニター・ダウンタイムを作成するには

1. Go to [**one.newrelic.com**](http://one.newrelic.com/) **> Synthetics> Monitor downtime**.
2. Select **<Icon name="fe-plus-circle"/>Create monitor downtime**.
3. 新しいモニターのダウンタイムの名前を指定します。
4. 頻度は **once** を選択し、モニターのダウンタイムの日付と開始時間も合わせて選択します。
5. モニターダウンタイムの対象となるモニターを選択します。選択できるモニターの数に上限はありません。
6. **「Create」**を選択します。

## ダウンタイムモニターの表示 [#view]

[**one.newrelic.com**](http://one.newrelic.com/) **> Synthetics> (select your monitor)> Summary** にアクセスしてください。モニターのダウンタイムが発生すると、 [**まとめ** ページ](/docs/synthetics/new-relic-synthetics/pages/synthetics-overview-page-view-monitors-performance) **ロードタイムチャート** に黄色い縦線で表示されます。チャート内でどのモニターダウンタイムが発生したかを確認するには、黄色のラインにカーソルを合わせるとモニターダウンタイムの名前が表示されます。

詳細は **Results** と **Resources** のセクションでご覧いただけます。

## モニターダウンタイムの削除 [#delete]

<Callout variant="tip">
  ウィンドウの削除ができない場合は、 [権限を確認してください](/docs/synthetics/new-relic-synthetics/administration/synthetics-permissions-user-groups) 。
</Callout>

既存のモニターのダウンタイムを削除するには

1. Go to [**one.newrelic.com**](http://one.newrelic.com/) **> Synthetics> Monitor downtime**.
2. インデックスで削除するモニターのダウンタイムを探します。
3. エリプシス **<Icon name="fe-more-horizontal"/>**アイコンを選択して、 **削除**.

## モニターのダウンタイムの編集 [#edit]

<Callout variant="tip">
  ウィンドウの編集ができない場合は、 [権限を確認してください](/docs/synthetics/synthetic-monitoring/administration/user-roles-synthetic-monitoring/) 。
</Callout>

既存のモニターのダウンタイムを編集するには

1. [**one.newrelic.com**](http://one.newrelic.com/) **> Synthetics> モニタリングのダウンタイム。**
2. モニター・ダウンタイム・インデックスで編集するモニター・ダウンタイムを探す。
3. 編集したいダウンタイムのエリプシス **<Icon name="fe-more-horizontal"/>**アイコンを選択します。
4. モニターのダウンタイムの名前や頻度を編集します。ターゲットのリストからモニターを削除するには、 **Selected targets** のリストでモニターを探し、モニターの横にあるremove **<Icon name="fe-x-circle"/>**アイコンを選択します。
5. 編集が完了したら、 **Save** を選択します。
