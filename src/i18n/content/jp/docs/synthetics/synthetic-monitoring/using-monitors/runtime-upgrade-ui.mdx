New Relic is [ending support for older runtimes](/eol/2026/05/eol-05-18-26-node-16-chrome-134-runtimes/) in 2026. To simplify migrating synthetic monitors to the latest supported runtimes, we auto-upgraded every monitor that didn&apos;t require your action. Use the runtime upgrade tool to view the remaining monitors that still need your attention.

## 残りのモニターをアップグレードする [#upgrade-options]

注意が必要な各モニターについて、2つのパスのいずれかを選択できます。

<Tabs>
  <TabsBar>
    <TabsBarItem id="fix-manually">
      手動で修正
    </TabsBarItem>

    <TabsBarItem id="fix-with-ai">
      AIで修正
    </TabsBarItem>
  </TabsBar>

  <TabsPages>
    <TabsPageItem id="fix-manually">
      手動でモニタースクリプトを記述または編集してモニターを更新するには、次の手順に従います。

      1. <DNT>**[one.newrelic.com](https://one.newrelic.com)** &gt; **Synthetic Monitoring** &gt; **Runtime Upgrades**</DNT>に移動します。

      2. モニターを見つけ、<DNT>**Fix manually**</DNT>をクリックします。

      3. <DNT>**Fix script manually**</DNT>ウィンドウで、新しいランタイム環境用にスクリプトコードを更新します。

      4. <DNT>**Close and start validation**</DNT>をクリックします。システムが更新されたスクリプトを検証している間、モニターは<DNT>**Monitors we&apos;re validating**</DNT>テーブルに移動します。

         * 検証に失敗した場合は、モニターの横にある**⋯**をクリックし、メニューから<DNT>**Fix manually**</DNT>を選択し、スクリプトを更新して、再度<DNT>**Close and start validation**</DNT>をクリックします。

      5. 検証が成功し、ステータスが<DNT>**Fixed**</DNT>と表示されたら、<DNT>**Ready to update**</DNT>タブをクリックします。

      6. 検証済みモニターの横にあるチェックボックスをオンにして、<DNT>**Upgrade selected**</DNT>をクリックします。

      完了すると、確認通知が表示され、モニターはアップグレードの成功を確認できる<DNT>**Updated**</DNT>タブに移動します。
    </TabsPageItem>

    <TabsPageItem id="fix-with-ai">
      <Callout title="注記">
        * この機能にはAdvanced 計算が必要であり、<DNT>**FedRAMP**</DNT>アカウントでは利用できません。サポートについては、担当のNew Relicアカウントチームにお問い合わせください。
        * 本番環境にモニターをデプロイする前に、AIが生成したスクリプトを必ずテストしてください。
      </Callout>

      Use this option to let <DNT>**New Relic AI (NRAI)**</DNT> generate an updated script for you. The AI-assisted script generation is mainly geared toward fixing `timeout errors` incurred during runtime upgrades. For monitors with `script errors`, the script potentially needs other changes. You can make those manually or use <DNT>[SRE Agent](/docs/agentic-ai/sre-agent/overview/)</DNT> to debug and get insights on potential script-related updates.

      AIを使用してモニターを更新するには：

      1. <DNT>**[one.newrelic.com](https://one.newrelic.com)** &gt; **Synthetic Monitoring** &gt; **Runtime Upgrades**</DNT>に移動します。
      2. モニターを見つけ、<DNT>**Fix with AI**</DNT>をクリックします。
      3. AIが提案したスクリプトを確認します。元のコードに加えられた特定の変更を表示するには、<DNT>**Show comparison**</DNT>をクリックして左右に並べた差分表示を開きます。
      4. <DNT>**Continue to validation**</DNT>をクリックします。システムは、新しいスクリプトをテストしてエラーなしで実行されることを確認するため、一時的にモニターを検証状態にします。
      5. スクリプトが検証されたら、<DNT>**Ready to update**</DNT>タブをクリックします。
      6. 検証済みモニターの横にあるチェックボックスをオンにして、<DNT>**Upgrade selected**</DNT>をクリックします。

      完了すると、確認通知が表示され、モニターはアップグレードの成功を確認できる<DNT>**Updated**</DNT>タブに移動します。
    </TabsPageItem>
  </TabsPages>
</Tabs>

また、複数のモニターを並行して検証し、チームメイトとリンクを共有して作業を分担することもできます。すべてのスクリプトの検証が完了したら、準備が整い次第、モニターを本番環境にデプロイできます。
