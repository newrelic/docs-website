シンセティックモニターは、標準的なHTTPレスポンスコードを超える数多くの追加レスポンスコードを返します。これは、 [**リソース** ページ](/docs/synthetics/new-relic-synthetics/pages/synthetics-resources-understand-load-times) で見ることができます。これらの応答コードは、 [Chromium の応答コード](https://code.google.com/p/chromium/codesearch#chromium/src/net/base/net_error_list.h&sq=package:chromium) に基づいており、モニターの動作についてより詳細に知ることができます。
