デバイス エミュレーションは、単純なスクリプト ブラウザ モニタで使用できるオプションの設定です。この設定は、Chrome DevTools のエミュレーション機能を使用して、モバイル デバイスまたはタブレット デバイスのビューポートと機能をシミュレートします。

デバイス エミュレーションを使用すると、シミュレートされた汎用モバイル デバイスまたはタブレット デバイスを使用して、ブラウザ ベースのアプリケーションをテストできます。次の点に注意してください。

* この設定は、合成モニターの実行に使用されるネットワーク、ユーザー エージェント、またはデバイス タイプを変更しません。
* デバイス エミュレーションでは、モバイル アプリケーションをインストールしてテストすることはできません。

## 要件 [#requirements]

* Chrome 100 以上のランタイムのみ
* プライベート ロケーションでは、 [ジョブ マネージャーの](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) バージョン 225 以降と、synthetics-node-browser-runtime バージョン 1.1.98 を使用する必要があります 以上
