デバイス エミュレーションは、シンプルなブラウザ化ブラウザ モニターで使用できるオプションの設定です。 この設定は、Chrome および Firefox の DevTools のエミュレーション機能を使用して、モバイル デバイスまたはタブレット デバイスのビューポートと機能をシミュレートします。

デバイス エミュレーションを使用すると、シミュレートされた汎用モバイル デバイスまたはタブレット デバイスを使用して、ブラウザ ベースのアプリケーションをテストできます。次の点に注意してください。

* この設定は、合成モニターの実行に使用されるネットワーク、ユーザー エージェント、またはデバイス タイプを変更しません。
* デバイス エミュレーションでは、モバイル アプリケーションをインストールしてテストすることはできません。

## 要件 [#requirements]

* 非レガシー ランタイムのみ (Chrome 72 はサポートされていません)
* Chrome: プライベートロケーションでは、[ジョブ マネージャー](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide)バージョン 225 以降と外部監視 -node- browser -runtime バージョン 1.1.98 を使用する必要があります。 またはそれ以上
* Firefox: プライベートロケーションでは、[ジョブ マネージャー](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide)バージョン 400 以降と外形監視 -node- browser -runtime バージョン 3.0.0 を使用する必要があります。 またはそれ以上
