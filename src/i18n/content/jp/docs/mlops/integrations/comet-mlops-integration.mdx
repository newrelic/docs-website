[Comet](https://www.comet.ml/site/)は、フルスタックのオブザーバビリティを機械学習モデルに拡張する[モデル パフォーマンス モニタリング](/docs/alerts-applied-intelligence/mlops/get-started/intro-mlops/)プラットフォームであり、モデルのパフォーマンスに基づいて本番パフォーマンスのベースラインを確立できます。Comet 統合により、機械学習メトリクスを New Relic と統合できるため、機械学習のライフサイクル全体にわたってデータを継続的に監視できます。これにより、最適なモデル パフォーマンスを確保し、より良いビジネス成果を達成することができます。

## CometとNew Relicの統合 [#integrate-comet]

New Relic の Comet インテグレーションを使用すると、より優れた精度の機械学習モデルの構築に役立つデータやインサイトを取得できるだけでなく、チーム全体の生産性、コラボレーション、可視性を向上させることができます。

連携を設定するには、以下の手順で行います。

1. <DNT>**Log into your New Relic account**</DNT>: <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > +Add more data**</DNT>へ移動します。

2. <DNT>**Click on Comet**</DNT>: 検索バーに<DNT>**Comet**</DNT>と入力します。

3. <DNT>**Select the account ID**</DNT> Comet を統合したいと考えています。

4. <DNT>**Select or create an API key**</DNT>: Comet と統合するには、New Relic <InlinePopover type="licenseKey"/>が必要です。 <DNT>**Real time training metric**</DNT>セクションで既存の API キーを作成または選択します。

5. <DNT>**Contact Comet**</DNT>: インテグレーションを設定し、Comet でモデル生産監視ダッシュボードを表示する次の手順については、 [Comet に連絡する](mailto:sales@comet.ml)必要があります。 Comet チームがインテグレーション プロセスの各ステップを案内し、セットアップを完了させます。

   <img
     title="Model Production Monitoring dashboard"
     alt="Model Production Monitoring dashboard."
     src="/images/model-performance-monitoring_screenshot-full_comet-UI.webp"
   />

6. <DNT>**View your data**</DNT>: インテグレーションのセットアップが完了したので、Comet チームによって自動的に生成される専用[ダッシュボード](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/)で機械学習モデルのパフォーマンスの追跡を開始できます。
