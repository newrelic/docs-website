当社は、データをNew Relicに取り込み、データを1か所で分析できるようにするための、[数百のソリューション](https://newrelic.com/integrations)を提供します。この有用なデータのフローにより、問題を迅速に解決し、複雑なシステムを維持し、コードを改善し、デジタル変革を加速させるのに役立つツールを提供します。

数百のアプリケーションやフレームワーク、サービス、オペレーティングシステム、その他のテクノロジーから、データを取り込むことができます。インテグレーションによりデータをまとめ、エージェントがデータをNew Relicに送信します。ソリューションでは、インテグレーションとエージェントの両方をインストールする必要があります。場合によっては、APMエージェントなどのインテグレーションを含むエージェントだけをインストールすることもできます。

どのようなデータを取り込む必要があっても、御社の環境に合ったオプションを提供できます。自社でのソリューション作成をご希望の場合、開始のためのツールも提供します。

<Callout variant="tip">
  インテグレーションとInfrastructureモニタリング、およびその他の[オブザーバビリティプラットフォーム](https://one.newrelic.com)を使用するには、New Relicファミリーに参加してください。[サインアップ](https://newrelic.com/signup)して、わずか数秒で無料アカウントを作成できます。次に、毎月最大100GBのデータを無料で取り込みます。期間無制限です。
</Callout>

## 御社に合ったものを選択 [#choose-an-integration]

当社は広範なソリューションを提供するので、御社は環境全体で簡単にデータを収集できます。必要なのは、必要なデータを収集する当社のソリューションの1つであり、広範なデータタイプを捕捉するさまざまなオプションを選択することもできます。

[New Relic Integrations](https://newrelic.com/integrations)に移動し、御社の環境に合ったソリューションをご覧ください。ソリューションの例としては次のようなものがあります。

* **アプリケーションのパフォーマンス監視（APM）：**C、Go、Java、Node、.NET、PHP、Python、Ruby
* **モバイルアプリ：**AndroidおよびiOS
* **ブラウザ監視：**Google Chrome、Mozilla Firefox、Microsoft Internet Explorer、Apple Safari
* **ホスト監視：**LinuxおよびMicrosoft Windows
* **クラウドプラットフォーム監視：**Amazon Web Services（AWS）、Microsoft Azure、Google Cloud Platform（GCP）
* **コアインフラストラクチャサービス：**Kubernetes、NGINX、MySQLなど
* **オープンソースのテレメトリインテグレーション：**Prometheus、Micrometer、OpenTelemetryなど

## 独自のソリューションを作成 [#create]

カスタムオプションをお探しの場合、独自のソリューション作成に役立つツールがあります。

* [New Relic Flex](/docs/integrations/host-integrations/host-integrations-list/flex-integration-tool-build-your-own-integration)を使用して、インフラストラクチャモニターを使用する軽量モニターソリューションを作成します。
* [New Relic Telemetry SDK](/docs/data-ingest-apis/get-data-new-relic/new-relic-sdks/telemetry-sdks-send-custom-telemetry-data-new-relic)を使用して、メトリックスやトレースなどを送信するカスタムソリューションを構築します。
* 同僚との共有や、当社の[カタログ](/docs/new-relic-one/use-new-relic-one/build-new-relic-one/new-relic-one-build-your-own-custom-new-relic-one-application)でのオープンソースアプリケーションの編集が可能な、独自の[New Relic Oneアプリケーション](/docs/new-relic-one/use-new-relic-one/build-new-relic-one/discover-manage-applications-new-relic-one-catalog)を構築します。
