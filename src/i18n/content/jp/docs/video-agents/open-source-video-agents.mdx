New Relic には、ビデオを監視するためのさまざまなオープンソース ツールがあります。 いくつかは実験的なカテゴリに属し、いくつかはコミュニティ プロジェクトです。

* iOSとtvOS
* Android モバイルとテレビ
* クロームキャスト
* Roku（デバイスまたはテレビ）

要件、インストレーション、設定情報については、GitHub の[wiki を](https://github.com/newrelic-experimental/video-documentation/wiki)ご覧ください。
