<Callout variant="important">
  IAST は非実稼働デプロイメントのみで実行します。 IAST は、エクスプロイト ペイロードを含む HTTP リクエストを呼び出してテストします。 データの破損とライブコードへのエクスプロイトの導入を避けるために、本番環境以外の環境ではシミュレートされたデータのみを使用して IAST を使用する必要があります。 さらに、実稼働前に新しいコードに対して IAST を実行すると、本番稼働前に潜在的な脆弱性を発見できます。
</Callout>

New Relic IAST をインストールするには:

<Steps>
  <Step>
    <DNT>**[one.newrelic.com](https://one.newrelic.com) > All capabilities > IAST**</DNT>に移動し、 <DNT>**Set up IAST with applications**</DNT>をクリックします。

    <img
      title="Set up IAST with applications"
      alt="Set up IAST with applications"
      src="/images/iast_screenshot-full_install.webp"
    />
  </Step>

  <Step>
    IAST が本番環境で実行されていないことを確認し、 <DNT>**Continue**</DNT>をクリックします。
  </Step>

  <Step>
    アプリケーションの言語を選択し、手順を完了します。

    <img
      title="Install New Relic IAST"
      alt="Install New Relic IAST"
      src="/images/iast_screenshot-crop_install.webp"
    />
  </Step>

  <Step>
    `newrelic.yml`構成ファイルが次のように更新されていることを確認してください。

    <CollapserGroup>
      <Collapser
        id="config-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> 設定ファイル</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
        ```
      </Collapser>

      <Collapser
        id="config-eu-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> EU 用の設定ファイル</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
            validator_service_url: wss://csec.eu01.nr-data.net
        ```
      </Collapser>

      <Collapser
        id="config-file-fed-example"
        title={<><InlineCode>newrelic.yml</InlineCode> Fed ユーザー用の設定ファイル</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
            validator_service_url: wss://csec-gov.nr-data.net
        ```
      </Collapser>
    </CollapserGroup>
  </Step>

  <Step>
    すべての手順を完了したら、アプリケーションを再起動し、アプリケーションの API に対してトラフィックを生成します。
  </Step>

  <Step>
    <DNT>**See your data**</DNT>をクリックすると、テストされたアプリケーションの概要が表示されます。
  </Step>
</Steps>
