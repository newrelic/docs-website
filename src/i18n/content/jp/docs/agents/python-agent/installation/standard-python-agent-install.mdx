## インストール手順 [#install]

<Callout variant="important">
  以下の手順は、[Pythonエージェントが自動的に計測するフレームワーク上のアプリケーション](/docs/agents/python-agent/getting-started/instrumented-python-packages)向けとなります。Web以外のPythonスクリプト、カスタムバックグラウンドタスク、もしくは非自動インストゥルメンテーションフレームワーク上のアプリケーションを監視するには、[Web以外のスクリプト／関数を監視する](/docs/agents/python-agent/supported-features/python-background-tasks)を参照してください。

  Dockerを使用している場合は、**各コンテナ内**にエージェントをインストールします。
</Callout>

New Relic Pythonエージェントを[自動インストゥルメンテーションフレームワーク上の](/docs/agents/python-agent/getting-started/instrumented-python-packages)アプリケーションにインストールするには：

1. New Relic[ライセンスキー](/docs/subscriptions/license-key)を用意しておきます（New Relicアカウントを保有していない場合は、[サインアップして作成](https://newrelic.com/signup)します）。
2. 以下のように**pip**を実行して、**newrelic**Pythonパッケージをインストールします。

   ```
   pip install newrelic
   ```
3. 以下のコマンドを実行して、エージェント設定ファイルを生成します。

   ```
   newrelic-admin generate-config <var>YOUR_LICENSE_KEY</var> newrelic.ini
   ```
4. ウェブアプリケーションまたはWSGIサーバーの起動を制御する場合は、通常の起動コマンドオプションの前に以下のNew Relic管理者スクリプトコマンドを使用します。

   ```
   NEW_RELIC_CONFIG_FILE=newrelic.ini newrelic-admin run-program <var>YOUR_COMMAND_OPTIONS</var>
   ```

   この方法で管理者スクリプトを実行できない場合は、[高度なインストール手順](/docs/agents/python-agent/installation-configuration/python-agent-installation)を読んで、より詳しいコンテキストおよびその他のインストールオプションを確認します。
5. PythonウェブアプリケーションまたはWSGIサーバーを起動または再起動します。5分間待機してからNew Relicアカウントのデータを確認します。

さらに詳しいインストール手順および有用なコンテキストについては、[Pythonエージェントのインストレーション](/docs/agents/python-agent/installation-configuration/python-agent-installation)をご覧ください。
