    # Pythonエージェント

    Python用New RelicはPythonアプリケーションを監視し、[パフォーマンスの問題](#monitor-performance)の特定と解決を支援します。また、パフォーマンスモニタリングを拡張して、[ビジネスデータ](#business-data)を収集・分析することで、顧客体験を改善し、データに基づくビジネス上の意思決定を行うことができます。[カスタムインストゥルメンテーションとAPI](/docs/agents/python-agent/custom-instrumentation/python-custom-instrumentation)の柔軟なオプションを備えたPython向けNew Relicには、アプリケーションに必要なデータをカスタマイズする多くの構築ブロックが用意されています。

    エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[Pythonエージェントのリリースノート](/docs/release-notes/agent-release-notes/python-release-notes)を参照ください。
  </HeroContent>

  ![Python_landing-page.png](./images/Python_landing-page.png "Python_landing-page.png")
