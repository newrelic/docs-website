    * [APM](/docs/apm/new-relic-apm/getting-started/introduction-new-relic-apm)、[ブラウザモニタリング](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser)、その他のNew Relic機能で精選されたダッシュボードからNode.jsデータを監視します。

    * [データのクエリと視覚化を行う](/docs/insights/using-insights-ui/getting-started/introduction-new-relic-insights)：

      エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[Node.jsエージェントのリリースノート](/docs/release-notes/agent-release-notes/nodejs-release-notes)を参照ください。
