New RelicのGoエージェントは、Go言語アプリケーションとマイクロサービスを監視して、以下を支援します。

* パフォーマンス問題の特定と解決。
* 顧客体験の向上とデータに基づくビジネス上の意思決定。
* [New Relic APM](/docs/apm/new-relic-apm/getting-started/introduction-new-relic-apm)、[New Relic Infrastructure](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure)、その他の[New Relic製品](https://newrelic.com/products)の詳細なダッシュボードでデータを監視。
* [New Relic Insight](/docs/insights/using-insights-ui/getting-started/introduction-new-relic-insights)を使用して、データをクエリ、可視化、そして共有。

エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[Goエージェントのリリースノート](/docs/release-notes/agent-release-notes/go-release-notes)を参照ください。

![apm-go-agent-app.png](./images/apm-go-agent-app.png "apm-go-agent-app.png")

**[one.newrelic.com](https://one.newrelic.com/apm) > APM > （アプリケーションを選択）：** APM **Overview**ページには、Goアプリのパフォーマンスが一目でわかるチャートや表に加え、さらにNew Relic Browserのエンドユーザーパフォーマンス、New Relic Infrastructureで監視されるホスト、New Relic Insightを使用したデータ分析などへのリンクが表示されます。
