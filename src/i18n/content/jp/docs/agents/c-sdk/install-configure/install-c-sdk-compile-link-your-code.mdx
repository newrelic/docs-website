## C SDKをコードに追加する [#get-new-relic]

* ウェブトランザクション、トランザクションイベント、非ウェブトランザクション
* セグメント（より詳細なレベルのタイミング）
* 属性
* エラー

次に、アプリケーションをC SDK静的ライブラリに対しコンパイルしリンクします。
