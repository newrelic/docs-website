.NETエージェントは、アプリケーションのモニタリングを開始できるように、コードを自動インストゥルメンテーションします。ランチャーを使用するか、本ドキュメントの説明に従って基本的な.NETエージェントのインストールを完了できます。.NET用APMの動作の詳細については、[.NET用New Relicの概要](/docs/agents/net-agent/getting-started/introduction-new-relic-net)を参照してください。

<Callout variant="tip">
  .NETまたはその他のエージェント、およびその他の[オブザーバビリティプラットフォーム](https://one.newrelic.com)を使用するには、New Relicファミリーに参加してください。[サインアップ](https://newrelic.com/signup)すると、わずか数秒で無料アカウントを作成できます。次に、毎月最大で100GBのデータを無料で取り込みます。期間無制限です。
</Callout>
