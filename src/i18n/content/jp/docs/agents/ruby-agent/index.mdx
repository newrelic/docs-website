<LandingPageHero>
  <HeroContent>
    New Relicの言語エージェントファミリーとして初のRubyエージェントは、長年にわたって実証された安定性と継続的なリリース機能を兼ね備えています。Rubyエージェントを使用して、以下を実現できます。

    * [問題の特定とアプリケーションパフォーマンスの向上ê](/docs/agents/ruby-agent/getting-started/introduction-new-relic-ruby#monitor-performance)。
    * [New Relic APM](/docs/apm/new-relic-apm/getting-started/introduction-new-relic-apm)、[New Relic Browser](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser)、その他の[New Relic製品](https://newrelic.com/products)の詳細なダッシュボードでデータを監視。
    * [New Relic Insight](/docs/insights/using-insights-ui/getting-started/introduction-new-relic-insights)を使用して、データをクエリ、可視化、そして共有。

    エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[Rubyエージェントのリリースノート](/docs/release-notes/agent-release-notes/ruby-release-notes)を参照ください。
  </HeroContent>

  ![New Relic APMのRubyアプリの概要](./images/ruby-apm-app_0.png "New Relic APMのRubyアプリの概要")

  <figcaption>
    **[rpm.newrelic.com/apm](https://rpm.newrelic.com/apm) > （アプリケーションを選択） > Overview**：APM **Overview**ページには、Rubyアプリのパフォーマンスが一目でわかるチャートや表に加え、さらにNew Relic Browserのエンドユーザーパフォーマンス、New Relic Infrastructureで監視されるホスト、New Relic Insightを使用したデータ分析などへのリンクが表示されます。
  </figcaption>
</LandingPageHero>
