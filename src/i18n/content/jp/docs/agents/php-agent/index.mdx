    # PHPエージェント

    New RelicのPHPエージェントはアプリケーションを監視し、[パフォーマンス問題の特定と解決](/docs/agents/php-agent/getting-started/introduction-new-relic-php#monitor-performance)を支援します。さらに、エージェントのパフォーマンス監視を拡張して[ビジネスデータの収集や分析](/docs/agents/php-agent/getting-started/introduction-new-relic-php#business-data)を行い、顧客体験の向上や、ビジネスにおけるデータ駆動型の意思決定を促すのに役立ちます。

    エージェントを最新の状態に保ち、最新機能へのアクセスを確保するために、[PHPエージェントのリリースノート](/docs/release-notes/agent-release-notes/php-release-notes)を参照ください。
  </HeroContent>

  ![php-apm-app-overview.png](./images/php-apm-app-overview_0.png "php-apm-app-overview.png")

  <figcaption>
    **[one.newrelic.com](https://one.newrelic.com/apm) > APM > （アプリケーションを選択）：** PHPエージェントをインストールすると、お使いのアプリケーションのパフォーマンス概要が表示されます。
