New Relic [エージェント](/docs/using-new-relic/welcome-new-relic/get-started/glossary#agent) とは、ホストやアプリケーションにインストールして、パフォーマンスデータを New Relic に送信するソフトウェアのことです。製品やコーディング言語ごとに異なるエージェントを使用します。

このドキュメントでは、私たちの最も人気のあるツールへのリンクがあります。当社のソリューションの完全なリストについては、 [New Relic Instant Observability](https://newrelic.com/instant-observability/) をご覧ください。
