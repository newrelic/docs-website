このドキュメントでは、 <DNT>Fleet Control</DNT>のインストールまたは操作中に発生する一般的な問題をトラブルシューティングする手順について説明します。

## トラブルシューティング デプロイメント with デプロイメント イベント

<DNT>Fleet Control</DNT> デプロイメント中に発生するすべてのイベントを詳細に表示できる<DNT>deployment events</DNT>タブを提供します。これは、デプロイメントの問題をトラブルシューティングし、根本原因を理解するための主要なツールです。

### デプロイメントイベントへのアクセス

デプロイメント イベント タブにアクセスするには、次の 2 つの方法があります。

**デプロイメント履歴から:**

1. 特定の艦隊内の<DNT>deployments</DNT>タブに移動します。
2. 調査するデプロイメントの行をクリックします。
3. デプロイメントの概要ビューで、 <DNT>Deployment Events</DNT>タブをクリックします。

**新しいデプロイメントを作成した後:**

1. デプロイメントを開始すると、進行状況ビューが完了し、デプロイメントの概要ビューに移行します。
2. この概要ビューで、 <DNT>Deployment Events</DNT>タブをクリックすると、トラブルシューティングまたは検証のためのイベントがすぐに表示されます。

### デプロイメントイベントテーブルについて

<DNT>deployment events</DNT>テーブルには、デプロイメント中に実行された各アクションのアトミックなイベント レベルのデータが表示されます。表には次の列が含まれます。

* <DNT>**Timestamp:**</DNT> イベントが発生した日時。
* <DNT>**Status:**</DNT> イベントのステータス (例: <DNT>Completed, Config Applied, Applying</DNT>)。
* <DNT>**Managed entity name:**</DNT> イベントが発生したクラスターまたはホストの名前。
* <DNT>**Agent name:**</DNT> イベントに関係する特定のエージェント。
* <DNT>**Agent type:**</DNT> エージェントのタイプ (例: <DNT>New Relic Infrastructure, Fluent Bit</DNT>)。
* <DNT>**Event:**</DNT> 発生したイベントの種類。
* <DNT>**Details:**</DNT> イベントに関する追加情報。多くの場合、失敗したイベントのエラー メッセージが含まれます。
* <DNT>**Duration:**</DNT> イベントが完了するまでにかかった時間。

### イベントのフィルタリングと検索

<DNT>deployment events</DNT>タブでは、イベントを絞り込むためのいくつかの方法が提供されています。

* <DNT>**Search:**</DNT> 検索バーを使用して、キーワードで特定のイベントを検索します。
* <DNT>**Filter by entities:**</DNT> 特定の管理対象エンティティのイベントを表示するようにフィルターします。これは、大規模なフリート (たとえば、100 個のクラスター) があり、1 つまたは少数のエンティティのみが失敗した場合に特に役立ちます。
* <DNT>**Filter by status:**</DNT> 成功したイベントや問題のあるイベントに焦点を当てるには、イベントをステータス (たとえば、 <DNT>Completed, Config Applied, Applying</DNT>) でフィルタリングします。

### NRQL による高度なトラブルシューティング

より高度なトラブルシューティングを行うには、デプロイメント イベント テーブルにデータを入力する基礎となる NRQL クエリにアクセスできます。

1. <DNT>Deployment Events</DNT>タブで、イベント テーブル ヘッダーの右側にあるボタンを見つけます。
2. このボタンをクリックすると、 <DNT>Data Explorer</DNT>に NRQL クエリ タブが開きます。
3. クエリにはデプロイメント ID が事前に入力され、さまざまな ID やメタデータなどのより詳細な情報を含む追加の列が表示されます。
4. 問題をさらに調査する必要がある場合は、このクエリを変更できます。
5. テーブルビューと JSON 形式ビューの切り替えなど、 <DNT>Data Explorer&apos;s</DNT>標準機能を使用します。

<DNT>Data Explorer</DNT>の使用に関する詳細については、[データ エクスプローラーの紹介を](/docs/query-your-data/explore-query-data/query-builder/introduction-new-data-explorer/)ご覧ください。
