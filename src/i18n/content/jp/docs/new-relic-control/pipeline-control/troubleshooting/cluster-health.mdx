## 問題: リソース枯渇

クラスターの CPU またはメモリ リソースが使い果たされました。

## 解決

* Kubernetes UI を使用してポッド イベントと保留中のジョブを表示し、リソースの制約を特定します。
* ノード プールのサイズを増やすか、Pod のリソース制限 (CPU およびメモリ) requestsを調整します。
* アクティブなノードの数に対するcloudプロバイダーの制限を確認し、必要に応じて設定を調整します。
