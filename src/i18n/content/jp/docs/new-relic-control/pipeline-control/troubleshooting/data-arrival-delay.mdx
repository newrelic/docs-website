/\* New Relicですべてのデータを受信しているが、予想されるサービスレベル契約 ( SLA ) を超えて到着している場合は、このガイドが問題の診断と解決に役立ちます。 発生した特定の問題に応じて、以下の手順に従ってください: \*/

## 問題: SLA を超えてデータが到着する

予想された時間枠内にデータが受信されていません。

## 解決

* レイテンシ メトリクスをチェックして、データ送信の遅延を特定します。
* ゲートウェイ ポッドの最小数を増やすと、処理能力が向上し、レイテンシが短縮されます。
