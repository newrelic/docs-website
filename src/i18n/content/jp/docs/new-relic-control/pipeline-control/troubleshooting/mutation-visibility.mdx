/\* ドラフトルールの保存、ルールの削除、設定の変更などのアクションを実行した後、ゲートウェイミューテーションが更新に表示されない場合は、このガイドが問題の診断と解決に役立ちます。発生した特定の問題に応じて、以下の手順に従ってください: \*/

## 問題: API 競合状態

複数のユーザーがゲートウェイの更新 (作成、更新、削除のアクション) のrequests同時に送信し、変更が失われます。

## 解決

* [New Relic サポート](https://support.newrelic.com)にお問い合わせください。
* 競合状態により失われたアクションをやり直します。
