/\* ゲートウェイ ルールを変更したが、それがルール リストに表示されない場合は、このガイドが一般的な問題の診断と解決に役立ちます。発生した特定の問題に応じて、以下の手順に従ってください: \*/

## 問題: 未展開の変更

ルール リストにはアクティブな状態のみが表示され、変更がまだ適用されていないことに気づいていません。

## 解決

* デプロイに対する保留中のゲートウェイ更新については、更新ページを確認してください。
* 変更がルール リストに反映されるように、必ず変更をデプロイしてください。

## 問題: API 競合状態

複数のユーザーが同時にゲートウェイの更新requests送信するため、変更が失われます。

## 解決

* [New Relic サポート](https://support.newrelic.com)にお問い合わせください。
* 競合状態により失われたアクションをやり直します。
