/\* Pipeline Control UIにゲートウェイの MELT データが表示されない場合は、このガイドが一般的な問題の診断と解決に役立ちます。 発生した特定の問題に応じて、以下の手順に従ってください: \*/

## 問題: アカウントドロップダウンの問題

**All accounts** \[すべてのアカウント]または Pipeline Control UI で選択されたゲートウェイ ライセンス キーに関連付けられたアカウントがありません。

## 解決

* Pipeline Control UIのアカウント ドロップダウンをチェックして、正しいアカウントが選択されていることを確認します。
* 関連する MELT データにアクセスするには、**All accounts** \[すべてのアカウント]またはゲートウェイ ライセンスキーに関連付けられた特定のアカウントを表示していることを確認してください。
