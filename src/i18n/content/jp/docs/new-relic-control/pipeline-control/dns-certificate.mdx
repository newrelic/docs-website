ネットワーク内の New Relic APM エージェントとゲートウェイ間の信頼性の高い通信を容易にするために、ゲートウェイ クラスターのプライベート DNS ホスト名を設定するには、次の手順に従います。

1. [AWS Route 53](https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/hosted-zones-private.html)を使用して DNS ホスト名を作成します。
2. [ACM](https://docs.aws.amazon.com/acm/latest/userguide/acm-public-certificates.html)経由でゲートウェイ ホスト名の証明書を取得します。
