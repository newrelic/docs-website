title: 受信箱の上限に達したエラー
metaDescription: Daily limit on number of unique error groups exceeded
translationType: machine
---

限られたアカウントでは、Errors inboxを使おうとすると以下のような警告が表示されることがあります。

### 問題 [#problem]

```
You’ve reached the daily limit of unique error groups on account xxx.  This limit will be reset at 00:00 UTC.
```

### 解決策 [#solution]

エラーグループの数を減らすには、エラークラスまたはメッセージから一意の識別子を削除してみてください。[詳細はこちらをご覧ください](https://docs.newrelic.com/docs/errors-inbox/errors-inbox/#similar-events)

* * *

### 問題 [#problem]

```
You’ve reached the daily maximum unique timeseries (cardinality) limit on account xxx.  This limit will be reset at 00:00 UTC.
```

この上限に達すると、エラーズの受信箱は更新されなくなり、エラーが停滞したり減少したりするようになります。

### 解決策 [#solution]

この問題を解決するために、 [仕事の範囲を狭める](/docs/new-relic-one/use-new-relic-one/workloads/use-workloads/) 。
