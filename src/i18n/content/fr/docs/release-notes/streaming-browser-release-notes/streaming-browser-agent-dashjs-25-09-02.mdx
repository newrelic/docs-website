## Fonctionnalité

### Agent autonome

* **Découplé de l&apos;agent Browser**: l&apos;agent vidéo fonctionne désormais indépendamment avec sa propre **logique de collecte**. Cette séparation réduit la surcharge et garantit que les données vidéo sont collectées et envoyées à New Relic sans être liées au cycle de collecte de données de l&apos;agent Browser.
