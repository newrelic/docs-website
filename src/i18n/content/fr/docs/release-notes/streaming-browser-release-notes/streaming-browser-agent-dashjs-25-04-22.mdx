Cette version du tracker New Relic Dash.js comprend d&apos;importantes mises à jour de compatibilité et une modification du rapport d&apos;erreurs.

## Principaux changements

* **Compatibilité avec le lecteur Dash.js**: Cette version a été testée et confirmée comme étant entièrement compatible avec la dernière version du lecteur Dash.js, garantissant une intégration fluide et un suivi précis.

* **Mise à jour du rapport d&apos;erreurs**: l&apos;attribut `errorName`, précédemment utilisé pour identifier la description de l&apos;erreur, est obsolète. L&apos;attribut `errorMessage` est désormais la norme pour transmettre des descriptions d&apos;erreur détaillées. Veuillez mettre à jour tous les systèmes de monitoring ou de logging qui s&apos;appuient sur les informations d&apos;erreur afin d&apos;utiliser l&apos;attribut `errorMessage` à l&apos;avenir.

## Notes de mise à niveau

Les utilisateurs qui s&apos;appuient sur l&apos;attribut `errorName` pour l&apos;analyse des erreurs devraient prévoir de migrer vers l&apos;attribut `errorMessage` pour assurer la continuité des fonctionnalités et l&apos;accès aux informations détaillées sur les erreurs.
