## Ce qui a changé

### Documentation

* **Refonte du README :** README restructuré et développé avec des guides d’installation complets, des exemples d’utilisation, des bonnes pratiques, des options de configuration, une référence d’API et des canaux d’assistance.

### Améliorations

* **Validation des attributs :** mise à jour de la logique de validation des attributs personnalisés afin de garantir l’exactitude au niveau du tracker.
* **Limite des attributs personnalisés :** ajout d’une limite stricte sur le nombre d’attributs personnalisés pouvant être envoyés par événement, évitant ainsi des tailles de charge inattendues.

## installation

```bash
npm install @newrelic/video-html5@4.1.1
```
