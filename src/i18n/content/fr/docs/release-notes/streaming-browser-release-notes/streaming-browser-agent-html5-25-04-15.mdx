## Nouvelles fonctionnalités et améliorations

* Nouveaux types d&apos;événements introduits

  * **VideoAction**: Ce type d&apos;événement représente les actions standard liées à la lecture et à l&apos;interaction vidéo.
  * **VideoErrorAction**: ce type d&apos;événement capture les erreurs qui se produisent pendant la lecture vidéo, fournissant des informations détaillées pour le dépannage.
  * **VideoAdAction**: type d&apos;événement dédié au suivi des actions liées aux publicités vidéo, garantissant une meilleure granularité et un meilleur reporting.
  * **VideoCustomAction**: un type d&apos;événement flexible pour les actions personnalisées et l&apos;interaction au sein des applications vidéo.

## Dépréciations

* `PageAction`
  * Le type d’événement `PageAction` est obsolète. L&apos;utilisateur doit passer à l&apos;utilisation de types d&apos;événements plus récents qui offrent des capacités de suivi et une spécificité améliorées.

## Nouvel attribut introduit

* Nous avons introduit un ensemble de nouveaux attributs pour améliorer le suivi des données et l&apos;analyse. Ces attributs fournissent un contexte et des détails supplémentaires pour les nouveaux types d&apos;événements ; reportez-vous à la [documentation sur le modèle de données](https://github.com/newrelic/video-html5-js/blob/v3.0.0/DATAMODEL.md) pour plus de détails.

## Impact &amp; actions requises

* **Transition depuis PageAction**: l&apos;utilisateur qui suit actuellement l&apos;événement `PageAction` doit adapter ses implémentations pour utiliser les types d&apos;événements nouvellement introduits. Ce changement offre une spécificité et une portée améliorées dans le suivi des actions lors des interactions vidéo.

* **Mise à jour des implémentations**: Les intégrateurs doivent mettre à jour leurs systèmes pour exploiter les nouveaux attributs introduits avec la version, qui fournissent un contexte et des détails plus riches pour le suivi des événements.

* **Tests et validation**: Nous recommandons de tester l&apos;implémentation de nouveaux attributs et types d&apos;événements pour garantir la compatibilité et la transmission correcte des données.
