### Prise en charge de Fluent Bit 4.2.2

Les utilisateurs recevront désormais la version 4.2.2 de Fluent Bit. Pour plus de détails, consultez les [annonces Fluent Bit](https://fluentbit.io/announcements/).

Cette version inclut [New Relic Fluent Bit Output Plugin 3.4.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.4.0) avec des mises à jour de sécurité critiques et une compatibilité de plateforme améliorée.

### Ajouté

* Prise en charge de SLES 15.6 et 15.7
* Plugin de sortie New Relic Fluent Bit 3.4.0 intégration

### Modifié

* L&apos;agent d&apos;infrastructure recommande les paquets Fluent Bit 4.2.2
* [L&apos;image Docker du plugin de sortie Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) utilise Fluent Bit 4.2.2 avec le plugin de sortie 3.4.0
* Image de base aws-for-fluent-bit mise à jour depuis la version 3.1.0 vers 3.2.1

### Fixé

* Problèmes de compatibilité GLIBC sur RHEL8 et Amazon Linux 2 avec la compilation croisée Zig ciblant GLIBC 2.17

### Remarques

Pour rester au courant des correctifs et améliorations les plus récents, abonnez-vous à notre [flux RSS de Logs](/docs/release-notes/logs-release-notes/).
