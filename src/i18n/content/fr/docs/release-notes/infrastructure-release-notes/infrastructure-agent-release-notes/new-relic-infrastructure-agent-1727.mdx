Une nouvelle version de l&apos;agent a été publiée. Suivez les procédures standard pour [mettre à jour l&apos;agent Infrastructure](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent).
