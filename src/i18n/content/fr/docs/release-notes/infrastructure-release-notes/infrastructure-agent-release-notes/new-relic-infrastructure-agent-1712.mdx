Une nouvelle version de l&apos;agent a été mise en service. Suivez les procédures standard pour [mettre à jour l&apos;agent d&apos;infrastructure](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recommande de mettre à jour l&apos;agent régulièrement, et au minimum tous les 3 mois. Au moment de cette sortie, la version la plus ancienne prise en charge est [l&apos;agent d&apos;infrastructure 1.51.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1510/).
