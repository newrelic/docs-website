Une nouvelle version de l&apos;agent a été publiée. Suivez les procédures standard pour [mettre à jour l&apos;agent Infrastructure](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recommande de mettre à jour l&apos;agent régulièrement et au minimum tous les 3 mois. À partir de cette version, la plus ancienne version prise en charge est [l&apos;agent Infrastructure 1.63.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1630/).

## Sécurité

* Mise à jour de `google.golang.org/grpc` vers la v1.79.3 pour corriger la [CVE-2026-33186](https://github.com/advisories/GHSA-p77j-4mvh-x3m3).
* Mise à jour de Go vers la v1.25.8 pour corriger les [CVE-2026-25679](https://pkg.go.dev/vuln/list) et [CVE-2026-27142](https://pkg.go.dev/vuln/list).
