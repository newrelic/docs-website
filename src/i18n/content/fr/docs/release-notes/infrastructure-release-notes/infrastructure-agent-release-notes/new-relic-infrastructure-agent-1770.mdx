Une nouvelle version de l&apos;agent est sortie. Suivez les procédures standard pour [mettre à jour l’agent d’infrastructure](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recommande de mettre à jour l&apos;agent régulièrement et au minimum tous les 3 mois. À partir de cette version, la plus ancienne version prise en charge est [l&apos;agent Infrastructure 1.65.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1651/).
