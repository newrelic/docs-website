Une nouvelle version de l&apos;agent est sortie. Suivez les procédures standard pour [mettre à jour l’agent d’infrastructure](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recommande de mettre à jour l&apos;agent régulièrement et au minimum tous les 3 mois. À partir de cette sortie, la plus ancienne version prise en charge est [l&apos;agent infrastructure 1.64.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1640/).

## Débogage

* Correctif de sécurité (Windows) : résolution d&apos;une vulnérabilité liée à un chemin de recherche non mis entre guillemets (BinaryPathName) qui pouvait déclencher des avertissements EDR ou bloquer l&apos;installation. Recommandé pour tous les utilisateurs Windows sur les versions 1.73.x ou 1.74.x. [#2230](https://github.com/newrelic/infrastructure-agent/pull/2230)
