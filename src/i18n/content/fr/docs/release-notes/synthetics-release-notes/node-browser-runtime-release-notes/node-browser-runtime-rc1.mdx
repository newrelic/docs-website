### Améliorations

* Le navigateur Chrome a été mis à jour vers la version 143.0.7499.146. Les utilisateurs peuvent observer un avertissement de dépréciation dans les logs du navigateur indiquant « La bascule automatique vers WebGL logiciel a été dépréciée. » Ce changement supprime la fonctionnalité de bascule automatique pour remédier à une vulnérabilité de sécurité de haute gravité ([CVE-2025-14765](https://chromereleases.googleblog.com/2025/12/stable-channel-update-for-desktop_16.html)).
