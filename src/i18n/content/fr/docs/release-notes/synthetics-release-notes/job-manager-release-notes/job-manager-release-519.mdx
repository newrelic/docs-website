### Corrections

* Correction des vulnérabilités de l&apos;OS sous-jacent et de Java dans SJM.

* Ajout de la prise en charge de la double pile (IPv4 et IPv6) pour les emplacements privés conteneurisés. Les plateformes de conteneurs prises en charge incluent Docker, Kubernetes et Podman.
