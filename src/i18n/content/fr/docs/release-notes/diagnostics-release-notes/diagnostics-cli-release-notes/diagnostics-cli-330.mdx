## Mises à jour des tâches

* Infra/Agent/Version - Mise à jour pour renvoyer un avertissement au lieu d&apos;une erreur en cas de problème de connexion à GitHub pour vérifier la version ([#231](https://github.com/newrelic/newrelic-diagnostics-cli/pull/231))

## Nouvelle fonctionnalité

* Nous avons ajouté les tâches suivantes pour Kubernetes et Agent Control ([#235](https://github.com/newrelic/newrelic-diagnostics-cli/pull/235), [#237](https://github.com/newrelic/newrelic-diagnostics-cli/pull/237), [#238](https://github.com/newrelic/newrelic-diagnostics-cli/pull/238))

  * K8s/AgentControl/helm-controller - Collecte les logs du pod helm-controller d&apos;Agent Control
  * K8s/AgentControl/source-controller - Collecte les logs pod du source-controller d&apos;Agent Control
  * K8s/AgentControl/agent-control - Collecte les logs pod agent-control d&apos;Agent Control
  * K8s/Flux/Charts - Collecte les informations des cartes Flux Helm
  * K8s/Flux/référentiel - Collecte les informations du référentiel Flux Helm
  * K8s/Flux/sortie - Collecte les informations de sortie du Flux Helm
  * K8s/Resources/Config - Collecte les configMaps K8s pour l&apos;espace de nommage donné au format YAML
  * K8s/Resources/déployer - Collecte les informations déployées par K8 pour l&apos;espace de nommage donné au format YAML
  * K8s/Resources/pod - Collecte les informations du pod K8s pour l&apos;espace de nommage donné au format YAML
  * K8s/Resources/Daemonset - Collecte les informations des daemonsets K8s pour l&apos;espace de nommage donné au format YAML
  * K8s/Env/Version - Récupère la version du client kubectl et du cluster
  * K8s/Helm/sortie - Collecte la liste des sorties de barre
