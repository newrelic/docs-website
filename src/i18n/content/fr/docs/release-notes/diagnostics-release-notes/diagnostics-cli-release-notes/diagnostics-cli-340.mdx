## Mises à jour des tâches

* Une nouvelle option de ligne de commande `-ac-agents-namespace` a été ajoutée pour spécifier un espace de nommage pour les agents de Agent Control ([#243](https://github.com/newrelic/newrelic-diagnostics-cli/pull/243)). Les tâches suivantes ont été mises à jour pour utiliser le nouvel espace de nommage :

  * K8s/Resources/Config
  * K8s/Resources/Daemonset
  * K8s/Resources/Deploy
  * K8s/Resources/Pods

## Corrections

* Les erreurs liées à GLIBC telles que `/lib64/libc.so.6: version 'GLIBC_2.34' not found` ont été résolues ([#244](https://github.com/newrelic/newrelic-diagnostics-cli/pull/244)).
