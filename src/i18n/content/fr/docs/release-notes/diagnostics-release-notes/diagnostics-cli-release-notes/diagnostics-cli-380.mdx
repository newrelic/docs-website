## Nouvelles fonctionnalités

* La CLI Diagnostics prend désormais en charge la région de données du Japon de New Relic. Si votre compte se trouve dans la région du Japon, spécifiez l’indicateur `region` lors de l’exécution de la CLI ([#263](https://github.com/newrelic/newrelic-diagnostics-cli/pull/263)) :

  ```bash
  ./nrdiag -region JP
  ```

  L’indicateur `region` accepte `US`, `EU` ou `JP` (insensible à la casse) et sa valeur par défaut est `US`.

## Mises à jour des dépendances

* Mise à jour de `golang.org/x/net` de 0.33.0 à 0.55.0 ([#264](https://github.com/newrelic/newrelic-diagnostics-cli/pull/264)).
