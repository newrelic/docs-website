### Notes de sortie de gateway de Pipeline Control - v1.1.0

#### Prise en charge des données de trace SQL et de trace de transaction

* Implémentation d&apos;une nouvelle fonctionnalité permettant de supprimer de manière sélective les données et les attributs des événements `SqlTrace` et `TransactionTrace` à l&apos;aide de règles de suppression.

#### Mises à jour de sécurité

* Des vulnérabilités ont été corrigées dans le package `golang.org/x/net`, notamment un problème de script intersite et un contournement du proxy HTTP lié aux ID de zone IPv6.

* Mise à niveau vers Go `1.24` et les dépendances suivantes sont mises à jour pour améliorer la sécurité et intégrer les correctifs récents :

  * `golang.org/x/net` à `v0.38.0`
  * `golang.org/x/sys` à `v0.31.0`
  * `golang.org/x/text` à `v0.23.0`

#### Prise en charge de nouvelles fonctionnalités

Ajout de la prise en charge de plusieurs nouvelles fonctions dans la requête de règle de suppression NRQL. Vous pouvez désormais utiliser les fonctions suivantes :

* `aparse()`
* `floor()`
* `getField()`
* `hourOf()`
* `numeric()`
* `round()`
* `string()`
* `substring()`
* `weekdayOf()`
* `mod()`
* `dimensions()`
