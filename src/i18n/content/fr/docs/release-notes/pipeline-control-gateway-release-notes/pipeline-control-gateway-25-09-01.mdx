### Notes de sortie de Pipeline Control Gateway - v1.2.0

#### Mise à niveau d&apos;OpenTelemetry Collector de 0.109.0 à 0.131.0

Nous avons mis à niveau le Collector interne OpenTelemetry de la version 0.109.0 à la version 0.131.0. Cette mise à jour importante comprend de nombreuses améliorations clés et débogages, offrant une plateforme plus robuste et plus stable. Pour une liste complète des modifications, veuillez vous référer aux [notes de sortie OpenTelemetry Collector ](https://github.com/open-telemetry/opentelemetry-collector/releases).
