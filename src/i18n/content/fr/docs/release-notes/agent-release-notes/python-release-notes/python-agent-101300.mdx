## Remarques

Cette sortie de l&apos;agent Python ajoute la prise en charge du protocole de contexte de modèle (MCP) et corrige la logique d&apos;importation pour `importlib.metadata` et `pkg_resources`.

Installez l&apos;agent à l&apos;aide de `easy_install/pip/distribute` via l&apos; [index du packagePython ](https://pypi.python.org/pypi/newrelic)ou téléchargez-le directement depuis le [site de téléchargement de New Relic](https://download.newrelic.com/python_agent/release).

## Nouvelles fonctionnalités

* Ajout de la prise en charge du protocole de contexte de modèle (MCP)

  * Ajoute la prise en charge des appels MCP dans [mcp](https://pypi.org/project/mcp/) et [fastmcp](https://pypi.org/project/fastmcp/). L&apos;agent va désormais instrument automatiquement les primitives MCP de base suivantes :

    * Outils (via `call_tool`)
    * Ressources (via `read_resource`)
    * Prompts (via `get_prompt`)

## Débogage

* Corriger la logique d&apos;importation pour `importlib.metadata` et `pkg_resources`

  * Met à jour la logique autour de l&apos;importation de `importlib.metadata` pour tenter également d&apos;utiliser le rétroportage `importlib_metadata` avant de revenir à `pkg_resources.*`. Cela supprime également l&apos;utilisation des conditionnels `sys.version` et corrige les problèmes où Python 3.9 ne pouvait utiliser que `pkg_resources` malgré la disponibilité de `importlib.metadata` .
