## Remarques

Cette version de l&apos;agent Python ajoute la prise en charge du temps jusqu&apos;au premier token et corrige une signature de fonction `add_url_rule` [Flask](https://pypi.org/project/Flask/) incompatible.

Installez l&apos;agent en utilisant `easy_install/pip/distribute` via l&apos; [index des packagesPython ](https://pypi.python.org/pypi/newrelic), en utilisant `conda` via l&apos; [index des packages Conda-Forge](https://anaconda.org/conda-forge/newrelic), ou téléchargez-le directement depuis le [site de téléchargement de New Relic](https://download.newrelic.com/python_agent/release).

## Nouvelles fonctionnalités

* Ajout du temps jusqu&apos;au premier token pour les flux de réponse LLM :

  * L&apos;agent enregistre un nouvel attribut,`time_to_first_token` sur les événements `LlmChatCompletionSummary` pour les flux de réponse [OpenAI](https://pypi.org/project/openai/) et [Bedrock](https://docs.aws.amazon.com/bedrock/). Cet attribut mesure la durée entre la requête initiale et le premier token reçu dans la réponse.

## Corrections de bugs

* Résolution de l&apos;incompatibilité de signature de la fonction `add_url_rule` de [Flask](https://pypi.org/project/Flask/)

  * Mise à jour de l&apos;instrumentation de [Flask](https://pypi.org/project/Flask/) pour aligner la signature de sa fonction `add_url_rule` avec l&apos;implémentation native de Flask. Ce correctif empêche les plantages potentiels de l&apos;application causés par la précédente incompatibilité de signature.
