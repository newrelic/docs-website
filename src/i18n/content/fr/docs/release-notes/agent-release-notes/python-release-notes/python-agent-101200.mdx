## Remarques

Cette sortie de l&apos;agent Python ajoute la prise en charge de [Redis](https://pypi.org/project/redis/) v6.0.0 et corrige les conflits avec le package [opentelemetry-proto](https://pypi.org/project/opentelemetry-proto/).

Installez l&apos;agent à l&apos;aide de `easy_install/pip/distribute` via l&apos; [index du packagePython ](https://pypi.python.org/pypi/newrelic)ou téléchargez-le directement depuis le [site de téléchargement de New Relic](https://download.newrelic.com/python_agent/release).

## Nouvelles fonctionnalités

* Ajout de la prise en charge de Redis v6.0.0

  * Ajoute la prise en charge de toutes les méthodes client introduites dans [Redis](https://pypi.org/project/redis/) v6.0.0.

## Débogage

* Corriger les conflits avec le package opentelemetry-proto

  * Corrige un problème où la copie interne de l&apos;agent d&apos;[opentelemetry-proto](https://pypi.org/project/opentelemetry-proto/) entrerait en conflit avec tout autre package s&apos;appuyant sur opentelemetry-proto (comme ChromaDB). Cela devrait désormais être traité comme un package séparé par OpenTelemetry et éviter tout conflit.
