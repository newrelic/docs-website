## Remarques

Cette version de l&apos;agent Python ajoute la prise en charge des agents [LangGraph](https://pypi.org/project/langgraph/).

Installez l&apos;agent en utilisant `easy_install/pip/distribute` via l&apos; [index des packagesPython ](https://pypi.python.org/pypi/newrelic), en utilisant `conda` via l&apos; [index des packages Conda-Forge](https://anaconda.org/conda-forge/newrelic), ou téléchargez-le directement depuis le [site de téléchargement de New Relic](https://download.newrelic.com/python_agent/release).

## Caractéristiques

* Ajouter la prise en charge des agents [LangGraph](https://pypi.org/project/langgraph/)

  * Ajoute une instrumentation pour les invocations d&apos;agents LangGraph via `create_agent()` dans LangChain. Cela inclut les appels de l&apos;agent vers `invoke()`, `ainvoke()`, `stream()` et `astream()`.
