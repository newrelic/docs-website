## 3.39.0

### Ajouté

* Ajout de l&apos;intégration nrfiber pour le framework go fiber
  * Merci au membre de la communauté @MitulShah1 pour sa contribution à cette solution
* Exemple nrslog mis à jour avec une nouvelle API
  * Merci au membre de la communauté @frankywahl pour sa contribution à cette solution
* Ajouter une fonction de filtrage de chemin facultative au middleware nrgin
  * Merci au membre de la communauté @frknikiz pour sa contribution à cette solution
* Ajout de ConfigDatastoreKeysEnabled à l&apos;intégration nrreddis permettant de signaler les noms des clés ainsi que les opérations de la datastore
* Suite de tests d&apos;action GitHub : passage d&apos;une couche ARM émulée à des couches ARM natives
* Ajout de la prise en charge des objets temporels pour les valeurs d&apos;attribut pour nrslog

### Fixé

* Correction de l&apos;emplacement des métadonnées des liens dans le message de log

### Déclaration de soutien

Nous utilisons la dernière version du langage Go. Au minimum, vous ne devriez pas utiliser de version de Go plus ancienne que celle prise en charge par l&apos;équipe Go elle-même. Consultez la [politique EOL de l&apos;agent Go](https://docs.newrelic.com/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) pour plus de détails sur les versions prises en charge de l&apos;agent Go et des composants tiers.
