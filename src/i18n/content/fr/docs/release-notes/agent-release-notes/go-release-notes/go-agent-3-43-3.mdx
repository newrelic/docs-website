## 3.43.3

### Fonctionnalité

* EXPÉRIMENTAL : ajout d&apos;une option de configuration `ConfigAppLogDecoratingWithinMessage` pour permettre au décorateur de log local (Local Log Decorator) d&apos;apparaître dans un champ de message. Ceci ne peut être appliqué qu&apos;à l&apos;intégration `logcontext-v2/nrlogrus`. L&apos;intégration `logcontext-v2/nrlogrus` sera mise à jour à la v1.1.4.

### Sécurité

* Mises à jour de sécurité Dependabot

  * `nrawsbedrock` (l&apos;intégration passera à la v1.1.1)
  * `nrawssdk-v2` (l&apos;intégration sera mise à jour à la v1.2.7)
  * `nrpgx5` (l&apos;intégration passera à la v1.3.4)\`

### Déclaration de soutien

Nous utilisons la dernière version du langage Go. À tout le moins, vous ne devriez pas utiliser une version de Go antérieure à celle prise en charge par l&apos;équipe Go elle-même. Consultez la [politique de fin de vie de l&apos;agent Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) pour plus de détails sur les versions prises en charge de l&apos;agent Go et des composants tiers.
