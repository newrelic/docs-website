## 3.44.0

### Ajouté

  \* Ajout de la prise en charge du SDK Anthropic `nranthropic`   \* Ajout de la prise en charge de GocqlX `nrgocqlx`

### Fixé

  \* Exportation des types de tests internes via `integrationsupport` pour débloquer les consommateurs de tests externes     \* Merci au membre de la communauté @nitinprajwal pour sa contribution à cette solution   \* Correction d&apos;un bug dans sqlparse pour prendre en charge les instructions `INTO` facultatives dans les intégrations sql   \* Correction d&apos;un bug dans `nrnats` qui causait des tests instables

### Sécurité

  \* Mises à jour de sécurité Dependabot     \* `nrfiber`

### Déclaration de soutien

Nous utilisons la dernière version du langage Go. Au minimum, vous ne devriez pas utiliser de version de Go plus ancienne que celle prise en charge par l&apos;équipe Go elle-même. Consultez la [politique EOL de l&apos;agent Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) pour plus de détails sur les versions prises en charge de l&apos;agent Go et des composants tiers.
