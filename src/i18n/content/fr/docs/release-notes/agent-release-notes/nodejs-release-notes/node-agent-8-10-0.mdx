## Remarques

* Ajout de l&apos;instrumentation pour `mysql2/promise`.

  * Cela n&apos;existait auparavant que dans notre `@newrelic/mysql` autonome, mais assure désormais la parité des fonctionnalités entre les deux.

* Suppression de l&apos;échantillonneur de métriques CPU natif inutilisé. Cette logique n&apos;était plus exécutée lors de l&apos;exécution de versions de Node.js supérieures à 6.1.0.

* Correction de l&apos;intercepteur de promesses pour empêcher la relance d&apos;erreurs.

* Ajout de la documentation sur le nommage des transactions issue d&apos;une [publication sur le forum de discussion](https://support.newrelic.com/s/hubtopic/aAX8W0000008ZfxWAE/relic-solution-the-philosophy-of-naming-your-node-agent-transactions).

* Ajout de `promises.tap.js` aux tests versionnés de mysql2.

* Mise à jour de @newrelic/test-utilities vers la dernière version.

* Suppression du fichier de test inutilisé dans les tests versionnés de restify.

* Ajout de l&apos;indicateur `--strict` au lanceur de tests versionné pour faire échouer correctement les exécutions CI lorsque les fichiers de test ne sont pas inclus.
