Les utilisateurs recevront désormais la version 4.2.0 de Fluent Bit. Pour plus de détails, consultez [https://fluentbit.io/announcements/](https://fluentbit.io/announcements/).

* L&apos;[image Docker du plugin de sortie Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) utilise Fluent Bit 4.2.0.
* Le [graphique New Relic Logging Helm](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) installe la nouvelle image de plugin (version 1.33.0 du graphique newrelic-logging)
