### Prise en charge de Fluent Bit 5.0.8

Les utilisateurs recevront désormais la version 5.0.8 de Fluent Bit. Pour plus de détails, consultez les [annonces Fluent Bit](https://fluentbit.io/announcements/).

Cette version inclut [New Relic Fluent Bit Output Plugin 3.7.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.7.0).

### Ajouté

* Plug-in de sortie New Relic Fluent Bit 3.7.0 intégration

### Modifié

* La version de Go est mise à niveau vers 1.26.4 pour les correctifs CVE
* L&apos;agent d&apos;infrastructure recommande les packages Fluent Bit 5.0.8 (Linux) et 5.0.7 (Windows)
* L&apos;[image Docker du plug-in de sortie Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) utilise Fluent Bit 5.0.8 avec le plug-in de sortie 3.7.0
* [Le chart Helm New Relic Logging](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) installe la nouvelle image du plug-in
* L&apos;image Docker Windows utilise Fluent Bit 5.0.7 (l&apos;image Windows 5.0.8 n&apos;est pas encore disponible en amont)

### Remarques

Pour rester au courant des correctifs et améliorations les plus récents, abonnez-vous à notre [flux RSS de Logs](/docs/release-notes/logs-release-notes/).
