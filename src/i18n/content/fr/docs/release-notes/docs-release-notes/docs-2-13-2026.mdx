### Changements majeurs

* Mise à jour de la [documentation de Pipeline Control Gateway](/docs/new-relic-control/pipeline-control/gateway/overview) avec des révisions approfondies pour la v2.0.0, incluant de nouveaux processeurs (filtre, échantillonnage, transformation), un guide de l&apos;interface utilisateur, une vue d&apos;ensemble de la configuration YAML et les différences de schéma de données.
* Mise à jour de [l&apos;installation de l&apos;intégration OTel pour Elasticsearch](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-install) et de [la vue d&apos;ensemble](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) avec des améliorations complètes pour les instructions d&apos;installation et de configuration.
* Mise à jour de la [configuration de l&apos;agent Java](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file) avec une documentation complète pour la prise en charge de la configuration de l&apos;agent hybride.
* Mise à jour de la [documentation sur le logging de l&apos;agent Go](/docs/apm/agents/go-agent/configuration/go-agent-logging) avec des instructions complètes pour configurer le logging dans l&apos;agent Go.
* Mise à jour du [catalogue d&apos;actions](/docs/workflow-automation/setup-and-configuration/actions-catalog/actions-catalog) d&apos;automatisation de workflow avec une terminologie révisée dans plusieurs fichiers d&apos;actions pour une meilleure cohérence.
