### Nouveaux documents

* Ajout des [workflows Jira Security RX](/docs/vulnerability-management/workflows/jira) pour fournir des conseils sur la création et la synchronisation automatiques des tickets Jira pour les vulnérabilités d&apos;application détectées.
* Ajout d&apos;une [action d&apos;arrêt de machine virtuelle Azure](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-vm-stop) au catalogue d&apos;actions afin que vous puissiez éteindre des machines virtuelles Azure dans le cadre de vos workflows automatisés.
* Ajout de détails sur [l&apos;enregistrement et la gestion des préférences de dashboard](/docs/streaming-video-&-ads/view-data-in-newrelic/user-preferences) pour expliquer comment New Relic enregistre et restaure les configurations de dashboard Video Overview et Ad Overview d&apos;une session à l&apos;autre.
* Ajout de règles d&apos;obfuscation concernant l&apos;agent Streaming vidéo et publicitaire pour [Android](/docs/streaming-video-&-ads/installation/android/obfuscation-rules), [navigateur](/docs/streaming-video-&-ads/installation/browser/obfuscation-rules), [iOS](/docs/streaming-video-&-ads/installation/ios/obfuscation-rules) et [Roku](/docs/streaming-video-&-ads/installation/roku/obfuscation-rules) afin de fournir des conseils sur la façon de masquer les données sensibles avec des règles basées sur des expressions régulières avant la transmission.
