### Nouveaux documents

* Ajout du [guide de migration de l&apos;agent Ruby 10.x](/docs/apm/agents/ruby-agent/getting-started/migration-10x-guide) pour fournir des conseils complets sur la mise à niveau vers la version 10.0.0 de l&apos;agent Ruby.
* Documentation de l&apos;intégration [OpenTelemetry Kafka](/docs/opentelemetry/integrations/kafka/overview) ajoutée pour fournir des conseils sur le monitoring de Kafka avec OpenTelemetry, y compris les déploiements auto-hébergés, Kubernetes avec Strimzi, la référence des métriques et l&apos;interrogation des données.
* Ajout de la documentation [OpenTelemetry NGINX integration](/docs/opentelemetry/nginx/nginx-otel-overview) pour fournir des conseils sur le monitoring de NGINX et NGINX Plus avec OpenTelemetry, y compris les déploiements basés sur l&apos;hôte et Kubernetes.
* Des [exemples d&apos;automatisation de workflow](/docs/workflow-automation/workflow-examples/workflow-examples) ont été ajoutés pour fournir des modèles pratiques pour les workflows de base, les intégrations et les cas d&apos;utilisation avancés.

* Mise à jour des [exigences de compatibilité du monitoring de l&apos;IA](/docs/ai-monitoring/compatibility-requirements-ai-monitoring) pour plus de clarté.
