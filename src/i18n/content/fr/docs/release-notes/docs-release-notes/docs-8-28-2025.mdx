* [Agent Ruby v9.21.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-9-21-0)

  * En mode Serverless APM, les noms de transaction incluent désormais le nom de la source de l&apos;événement comme préfixe. Cette amélioration améliore l&apos;identification et la traçabilité transaction dans l&apos;interface utilisateur de New Relic en indiquant clairement la source de l&apos;événement déclencheur.
  * Annulation des modifications apportées à la logique de suivi des threads pour l&apos;initialisation de la plage. Cette modification restaure le comportement précédent de suivi du thread où les étendues commencent, résolvant ainsi les problèmes découlant de la logique mise à jour.
