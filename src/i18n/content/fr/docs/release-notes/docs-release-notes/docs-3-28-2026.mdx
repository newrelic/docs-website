### Changements majeurs [#major-changes]

* Mise à jour de la documentation de l&apos;[intégration Microsoft SQL Server](/install/microsoft-sql) afin d&apos;ajouter le monitoring des performances des requêtes (QPM) pour les méthodes d&apos;authentification suivantes :

  * Domaine Windows
  * Compte de service administré de groupe Windows (gMSA)

* Mise à jour de la documentation [Monitoring d&apos;Elasticsearch avec OpenTelemetry](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) pour plus de clarté et ajout d&apos;une documentation complète d&apos;installation Kubernetes.

* Mise à jour de [Dashboards avec l&apos;API NerdGraph](/docs/apis/nerdgraph/examples/nerdgraph-dashboards) avec les fonctionnalités de visibilité des variables de template.

* Mise à jour du [schéma de définition de l&apos;automatisation des workflows](/docs/workflow-automation/workflow-automation-apis/definition-schema) avec d&apos;importantes mises à jour du schéma et des exemples complets.
