### Changements majeurs

* Mise à jour de [Installer New Relic pour Go](/docs/apm/agents/go-agent/installation/install-new-relic-go) avec un guide d&apos;installation complet, étape par étape, et des exemples pour les modèles d&apos;applications Go courants.
