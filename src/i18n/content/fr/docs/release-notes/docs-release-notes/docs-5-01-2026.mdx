### Nouveaux documents

* Ajout de la suite de documentation [Mobile Session Replay](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/introduction/), désormais disponible de manière générale, avec des guides complets couvrant la mise en place, la configuration de la confidentialité, les fonctionnalités avancées, la consultation des rediffusions et le dépannage pour Android et iOS.
* Ajout de [Mettre à niveau les scripts pour monitorer les API vers Node.js 22](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-api-monitors-to-node-22/) pour fournir des instructions de migration étape par étape afin de faire passer les scripts pour monitorer les API de Node.js 16 à Node.js 22.

### Changements majeurs

* Mise à jour de la documentation Intelligence Coûts du cloud pour la disponibilité générale, y compris les nouvelles pages [Tarification CCI](/docs/cci/cci-pricing/), [Conditions d’alerte pour Intelligence Coûts du cloud](/docs/cci/configure-cci/cci-alerts/) et [Coûts de l’IA](/docs/cci/financial-intelligence/ai-costs/), ainsi qu’un contenu élargi sur la configuration des intégrations et la FAQ.

* Mise à jour de la documentation de l&apos;agent eBPF pour les installations [Kubernetes](/docs/ebpf/k8s-installation/) et [Linux](/docs/ebpf/linux-installation/) avec des conseils de migration de la configuration YAML et les modifications des champs introduites dans la v1.4.0.
* [Optimisez l&apos;utilisation de votre compute](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-optimizer/) mis à jour avec des conseils élargis sur le dashboard, de nouvelles captures d&apos;écran et des recommandations d&apos;optimisation améliorées.
* Mise à jour d&apos;[OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/) avec une nouvelle section couvrant le monitoring des erreurs dans l&apos;expérience OpenTelemetry APM.
