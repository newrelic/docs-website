### Changements majeurs

* Ajout de `sampler.remote_parent_sampled` dans [la configuration de l&apos;agent Java : fichier de configuration](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file/#dt-sampler-remote-parent-sampled) pour définir comment l&amp;apos;agent doit gérer les plages d&amp;apos;échantillonnage lorsque leur parent de plage d&amp;apos;une entité en amont a été échantillonné.
* Mettez à jour le [provisionnement SCIM](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim) pour spécifier qu&amp;apos;il nécessite désormais un <DNT>**Paid Edition**</DNT> pour la gestion automatisée des utilisateurs dans New Relic.
* Mise à jour du document [API Cloud Rule](/docs/new-relic-control/pipeline-control/cloud-rules-api/#drop-attributes-on-dimensional-metric-rollups) pour le contrôle pipeline afin d&amp;apos;inclure des informations sur la manière de supprimer un attribut des cumuls de métriques dimensionnelles.
* Ajout d&apos;une requête pour suivre les événements de connexion et de déconnexion sur la page [suivi des changements des comptes](/docs/accounts/accounts/account-maintenance/query-account-audit-logs-nrauditevent/#user-login).

### Modifications mineures

* Mise à jour de la [version .Net compatibilité d&apos;agent](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/#framework).

* Ajout d&apos;une limitation dans la page [de détection d&apos;anomalie](/docs/alerts/create-alert/set-thresholds/anomaly-detection/) spécifiant aucune prise en charge de la saisonnalité mensuelle ou annuelle en raison de limites de calcul.

* Ajout de paramètres de traitement par lots et de compression pour un tracing infini au document [de configuration de l&apos;agent Node.js](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration/#infinite-tracing-batching).
