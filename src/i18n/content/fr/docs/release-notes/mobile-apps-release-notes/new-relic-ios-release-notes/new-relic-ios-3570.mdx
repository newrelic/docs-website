* Ajoute l&apos;historique des requêtes NRQL à l&apos;iPhone (l&apos;iPad suivra plus tard).
* Ajoute la possibilité d&apos;écrire des requêtes NRQL.

* Inclut plus d&apos;écrans de détails sur l&apos;iPad pour APM, Browser, Mobile et les transactions clés.
