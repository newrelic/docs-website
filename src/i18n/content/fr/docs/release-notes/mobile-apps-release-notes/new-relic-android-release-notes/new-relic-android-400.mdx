### Remarques

* Cette version ajoute la prise en charge de la fermeture d&apos;un événement d&apos;alerte ouvert pour les utilisateurs autorisés et étend la prise en charge des liens profonds pour les liens partagés depuis New Relic One.
* Cette version inclut également une vérification de feature flag pour les plugins, qui n&apos;affichera pas les plugins en tant qu&apos;élément de menu à moins que le compte ne dispose d&apos;un feature flag activant les plugins.
