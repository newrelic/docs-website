## Nouvelle fonctionnalité

### Analyse de la chronologie des erreurs publicitaires

* Ajout d&apos;un suivi complet des modèles de récupération d&apos;erreurs publicitaires pour améliorer les analyses de publicité vidéo.

## Ce qui a changé

### Nouvel attribut de chronométrage

* `timeSinceLastAdError`: Suit le temps écoulé depuis la dernière erreur publicitaire
  * Disponible uniquement pour le type d&apos;événement `VideoAdAction`
* Fournit des informations sur les schémas d&apos;erreurs publicitaires et les délais de rétablissement
* Aide à optimiser les stratégies de gestion des erreurs publicitaires

## Détails de l&apos;implémentation

* Fonction `nrTrackRAF()` améliorée avec suivi de l&apos;horodatage des erreurs
* Ajout d&apos;une logique d&apos;attribut conditionnel dans la fonction `nrAddRAFAttributes()`
* Mise en place d&apos;une gestion robuste des états d&apos;erreur pour un chronométrage précis
