## Débogage

* Correction d&apos;un problème où le `NewRelicNavigationObserver` plantait lors de la gestion d&apos;itinéraires sans paramètre clé, tels que les feuilles inférieures et les boîtes de dialogue modales.
* Échecs de suivi de navigation résolus pour les itinéraires qui ne s&apos;étendent pas sur `PageRoute`.

## Améliorations

* Sécurité de type améliorée dans `NewRelicNavigationObserver` en supprimant tous les types dynamiques.
* Prise en charge ajoutée pour des types d&apos;itinéraires supplémentaires, notamment `CupertinoPage` et `NoTransitionPage` à partir de `go_router`.
* refactorisation du code de l&apos;observateur de navigation pour une meilleure maintenabilité et lisibilité.
