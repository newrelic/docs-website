  À partir de la version 7.5.4 du SDK iOS, un changement a été implémenté qui provoque le signalement d&apos;un événement d&apos;exception gérée précédemment supprimé.
</Callout>

## Corrigé dans cette sortie

* L&apos;agent est construit à l&apos;aide de Xcode 16.3. Cela devrait corriger toutes les erreurs liées à la couverture du code lors de l&apos;utilisation de l&apos;agent iOS New Relic avec Xcode 16.3.
* Correction d&apos;un problème qui pouvait entraîner une définition incorrecte du compte de suivi distribué.
* Correction d&apos;un problème qui pouvait provoquer le blocage de l&apos;agent lors de la définition de l&apos;attribut de session.
