<Callout variant="important">
  À partir de la version 7.5.4 du SDK iOS, un changement a été implémenté qui provoque le signalement d&apos;un événement d&apos;exception gérée précédemment supprimé.
</Callout>

## Corrigé dans cette sortie

* Correction d&apos;un crash pouvant survenir lors de l&apos;enregistrement des exceptions gérées.
* Correction de divers plantages pouvant survenir lors de l&apos;utilisation de l&apos;agent iOS New Relic.
* L&apos;agent iOS New Relic inclut désormais les attributs Session dans les logs Mobile.
* Correction d&apos;un problème qui pouvait survenir lors de l&apos;envoi de plusieurs exceptions gérées à New Relic. L&apos;agent iOS New Relic empêchera désormais la perte de données lors de l&apos;envoi de plusieurs exceptions gérées.

## Déclaration de soutien

* À partir de la version 7.4.1 de l&apos;agent iOS, l&apos;agent iOS consolidera les agents XCFramework et tvOS précédemment séparés en un seul agent iOS.
* À ce jour, la version la plus ancienne prise en charge de [XCFramework](/docs/release-notes/mobile-release-notes/ios-release-notes/xcframework-agent-736) est la 7.3.6.
