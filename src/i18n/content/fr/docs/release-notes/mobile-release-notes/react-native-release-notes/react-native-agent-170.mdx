## Améliorations

* Mise à jour de l&apos;agent Android natif vers la version 7.7.0.
* Session Replay intelligent : introduction du mode ERROR (enregistrement en mémoire tampon de 15 à 30 s déclenché par des plantages) et du mode FULL (enregistrement continu) pour une capture de données optimisée.
* APIs de contrôle de session : Ajout de NewRelic.recordReplay() et NewRelic.pauseReplay() pour permettre un contrôle manuel et programmatique des états d&apos;enregistrement des replays.
* Correction de bug réseau : correction d&apos;un problème OkHttp3 où peekBody() provoquait un blocage indéfini sur les réponses en streaming (SSE/chunked) avec des longueurs de contenu inconnues.
