## Débogage

* Correction d&apos;un problème où les traces d&apos;appels pour les erreurs Unity ne s&apos;affichaient pas correctement.
* Correction d&apos;un problème qui empêchait l&apos;enregistrement de la métrique récapitulative lorsque l&apos;application revenait au premier plan après son lancement initial.
* Correction d&apos;un problème où un JSON non valide dans les entrées log pouvait provoquer des erreurs.

## Déclaration de soutien

Nous vous recommandons de mettre à jour l&apos;agent au moins tous les trois mois. Recherchez des politiques et des dates spécifiques pour le support des agents Android dans la [politique de fin de vie des agents des applications mobiles de surveillance](/docs/mobile-monitoring/new-relic-mobile/get-started/mobile-agents-eol-policy/#android-eol).
