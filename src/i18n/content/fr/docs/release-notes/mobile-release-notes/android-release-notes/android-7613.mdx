* Correction du problème ANR causé par setUserId

Recherchez des politiques et des dates spécifiques pour le support des agents Android dans la [politique de fin de vie des agents des applications mobiles de surveillance](/docs/mobile-monitoring/new-relic-mobile/get-started/mobile-agents-eol-policy/#android-eol).
