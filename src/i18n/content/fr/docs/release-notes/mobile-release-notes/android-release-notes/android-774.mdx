## Débogage

* Correction d&apos;un problème OutOfMemory causé par l&apos;accumulation de données de charge dans SharedPreferences. Les charges mises en cache sont désormais persistées sur le disque et plafonnées pour empêcher une croissance illimitée sur les appareils ayant une faible connectivité.

* Résolution d&apos;un ANR causé par des appels IPC Binder répétés vers ConnectivityManager lors de chaque vérification d&apos;accessibilité. L&apos;état du réseau est désormais suivi de manière asynchrone via un rappel réseau enregistré.

Déclaration de soutien

Nous recommandons de mettre à jour l&apos;agent au moins tous les trois mois. Trouvez les politiques et dates spécifiques pour la prise en charge de l&apos;agent Android dans la Politique de fin de vie des agents de monitoring des applications mobiles.
