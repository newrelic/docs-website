## Log des modifications

Cette sortie présente la distribution [nrdot-collecteur ](https://github.com/newrelic/nrdot-collector-releases/tree/main/distributions/nrdot-collector). Cette distribution est actuellement en version alpha mais servira éventuellement de remplacement à la distribution `nrdort-collector-host`. Les futures distributions utiliseront `nrdot-collector` comme manifeste de base et le développeront selon les besoins.
