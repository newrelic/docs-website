## Log des modifications

Premières notes de version dédiées pour le cas d&apos;utilisation [« monitoring d&apos;Hôte »](https://github.com/newrelic/nrdot-collector-releases/blob/main/distributions/nrdot-collector/README.md#use-cases) qui se limitent aux modifications pertinentes dans la configuration ou `nrdot-collector`. Le [collecteur nrdot](https://docs.newrelic.com/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01/) continuera d&apos;inclure la configuration nécessaire pour prendre en charge ce cas d&apos;utilisation. Cependant, avec `nrdot-collector` prenant en charge de plus en plus de cas d&apos;utilisation, nous voulons fournir des notes de sortie supplémentaires plus facilement consommables pour quelqu&apos;un utilisant NRDOT dans un but spécifique comme le monitoring d&apos;Hôte.
