## v1.293.0

### Caractéristiques

#### Ajouter un message interne « tâches longues »

Ajoutez un message interne indiquant si une fonction encapsulée est qualifiée de « tâche longue ». Ceci sera utilisé plus tard dans la fonctionnalité de l&apos;agent pour améliorer les comportements futurs.

#### Émettre SM sur RUM réponse horodatage invalide

Fournit une métrique de supportabilité à la réception d&apos;un horodatage invalide dans la réponse à l&apos;appel RUM.

### Débogage

#### Problème d&apos;adresse empêchant la désactivation des traces distribuées

Correction d&apos;un bug où l&apos;agent était capable de capturer des traces distribuées, même lorsque `distributed_tracing: false` était fourni dans la configuration.
