## v1.295.0

### Caractéristiques

#### Ajouter une détection de cause d&apos;erreur

Détecter et signaler l&apos;attribut de cause sur l&apos;événement JavaScriptError capturé. Si la cause est une instance d&apos;une erreur, la trace des appels sera capturée, sinon le contenu stringifié de la cause sera signalé. Cela sera interrogeable dans l&apos;événement JavaScriptError sous l&apos;attribut `cause`.

### Débogage

#### Ajouter des avertissements de mise à jour de singe

Ajout d&apos;une détection et d&apos;avertissements lorsque les variables globales attendues ont été corrigées avant que l&apos;agent ne soit exécuté sur la page. Cela peut entraîner des comportements inattendus et l’objectif est d’informer les clients de ce comportement.

#### Calculer le premier et le dernier horodatage à partir des données brutes

Assurez-vous que l&apos;agent calcule manuellement le premier et le dernier horodatage d&apos;une charge SessionReplay avant la récolte pour aider à améliorer la cohérence sur le lecteur d&apos;interface utilisateur.
