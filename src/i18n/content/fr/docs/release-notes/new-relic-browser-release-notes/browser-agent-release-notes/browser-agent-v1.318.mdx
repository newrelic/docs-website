## v1.318.0

### Caractéristiques

#### Ajout de la configuration d&apos;initialisation websocket

Ajoute le bloc `web_sockets` à la configuration d&apos;initialisation NREUM. Définir `web_sockets.enabled` sur `true` active la fonctionnalité des websockets. La fonctionnalité des websockets est désactivée par défaut.

#### Détection des Core Web Vitals MFE

Ajouter des méthodes de détection internes pour FCP, INP, CLS et LCP lorsqu&apos;ils sont générés par un MFE

#### Améliorer le démarrage de l&apos;agent

Fournir une meilleure prise en charge des scénarios où le démarrage de l&apos;agent peut être interrompu avant que l&apos;appel RUM ne se termine avec succès et que les modes de session de fonctionnalité ne soient enregistrés dans le stockage local (le cas échéant).
