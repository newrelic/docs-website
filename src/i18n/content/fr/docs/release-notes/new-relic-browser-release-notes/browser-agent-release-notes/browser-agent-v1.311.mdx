## v1.311.0

### Caractéristiques

#### Suppression de agentIdentifier

Suppression des références à portée par identifiant d&apos;agent pour simplifier l&apos;architecture de l&apos;agent. Toutes les références sont désormais stockées directement au sein de la référence de l&apos;agent, éliminant ainsi la nécessité de combiner les références de l&apos;agent avec des objets à portée via des clés de recherche.

#### Attribuer l&apos;URL de la page d&apos;origine à tous les nœuds PageViewTiming

Extension de l&apos;attribution `pageUrl` à toutes les mesures de temps, en s&apos;appuyant sur les travaux précédents pour le LCP. Les événements `PageViewTiming` rapportent désormais l&apos;URL de navigation complète d&apos;origine pour tous les types de nœuds, garantissant la cohérence des données pour les SPA même si l&apos;URL change avant la récolte.

#### améliorer la logique de timing MFE

Amélioration de la capacité de l&apos;agent à identifier le script MFE correct appelant l&apos;API register. Cette amélioration se traduit par des informations de timing plus précises pour les environnements MFE.

### Débogage

#### Imposer des identifiants de chaîne pour les nouveaux enregistrements MFE

Résolution d&apos;un problème où des identifiants de type non-chaîne provoquaient des erreurs lors de la synthèse d&apos;entités MFE. L&apos;agent impose désormais des identifiants de type chaîne et émet un avertissement si un type de données différent est fourni.

#### Le replaceur circulaire a supprimé les objets réutilisés des collectes

Correction d&apos;un problème où la méthode interne stringify supprimait les pointeurs partagés dans les charges utiles de collecte, entraînant parfois l&apos;absence d&apos;attributs `params` dans les collectes d&apos;erreurs.

#### Mettre à jour la RegExp pour éviter un problème de build webpack non minifié

Le caractère `$` a été remplacé par son équivalent hexadécimal dans les expressions régulières. Cela empêche Webpack de remplacer incorrectement le caractère lors du processus de build, ce qui provoquait auparavant des échecs de l&apos;agent dans les builds non minifiés.
