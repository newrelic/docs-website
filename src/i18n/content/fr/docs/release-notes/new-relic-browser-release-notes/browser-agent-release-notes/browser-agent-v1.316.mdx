## v1.316.0

### Caractéristiques

#### Utiliser des clés avec espace de noms d’application pour la session

L&apos;agent de navigateur attribue désormais un espace de nommage aux informations de session par application de navigateur, en utilisant la combinaison de `licenseKey` et `applicationID` fournie au démarrage. Cela signifie que différentes applications partageant la même origine et le même navigateur `localStorage` ne partageront plus la même session, ce qui était une collision acceptée mais pas idéale, provoquant auparavant le débordement des modes de fonctionnalité de l&apos;une sur l&apos;autre pendant la durée d&apos;une session.
