## v1.291.0

### Caractéristiques

#### Créer l&apos;API de mesure

Ajoute l&apos;API de mesure à l&apos;objet global `newrelic`. Cette API permet de générer de nouvelles données BrowserPerformance même si la détection automatique des marques et des mesures n&apos;est pas activée.

### Débogage

#### Correction de la course entre la fin de session et l&apos;abandon de la fonctionnalité

Corrigez un problème où la fonctionnalité entrait en compétition avec le gestionnaire de session lorsqu&apos;une session se termine. Cela nécessitait une vérification des horaires de session pour correspondre à une session donnée et pouvait créer un état dans lequel une nouvelle session créée à partir d&apos;une session préexistante déclencherait une relecture partielle ou aucune relecture, et empêcherait l&apos;agent d&apos;envoyer des données de trace avec une relecture donnée.

#### Récoltez immédiatement la première charge utile de trace de session

Pour une meilleure cohérence, un ancien comportement a été implémenté où la première charge utile ST est récoltée immédiatement lors du chargement de la page. Cela permet de garantir que les cas où les rediffusions sont récoltées mais où les traces ne se produisent pas moins fréquemment.

New Relic vous recommande de mettre à niveau l&apos;agent régulièrement pour vous assurer de bénéficier des dernières fonctionnalités et avantages en termes de performances. Veuillez consulter ce [dashboard d&apos;état](https://newrelic.github.io/newrelic-browser-agent-release/) pour plus d&apos;informations.

Pour les appareils mobiles, la version 1.291.0 a été créée et testée pour Android OS 16 et iOS Safari 17-18.1.
