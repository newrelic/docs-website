## Problème

Vous voyez ce message d’erreur lors de [l’installation de votre intégration Kubernetes](/install/kubernetes/) avec Helm ou Manifest.

```shell
  repo newrelic not found
```

## Solution

Ajoutez le référentiel newrelic (repository) à votre charte Helm en exécutant cette commande :

```shell
  helm repo add newrelic https://helm-charts.newrelic.com
```
