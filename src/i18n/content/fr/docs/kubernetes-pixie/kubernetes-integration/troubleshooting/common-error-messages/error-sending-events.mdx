## Problème

L&apos;agent ne peut pas se connecter aux serveurs New Relic et vous voyez une erreur comme celle-ci dans le log du conteneur `agent` ou `forwarder` :

```shell
2018-04-09T18:16:35.497195185Z time="2018-04-09T18:16:35Z" level=error 
msg="metric sender can't process 1 times" error="Error sending events: 
Post https://api.newrelic.com/metrics/events/bulk: 
net/http: request canceled (Client.Timeout exceeded while awaiting headers)"
```

## Solution

Selon la nature exacte de l’erreur, le message dans le log peut différer. Pour résoudre ce problème, consultez la [documentation des réseaux New Relic](/docs/new-relic-solutions/get-started/networks/#infrastructure) et la page GitHub [de dépannage des problèmes de réseau de l&apos;agent New Relic Infrastructure](https://github.com/newrelic/infrastructure-agent/blob/master/docs/network_troubleshooting.md?) .
