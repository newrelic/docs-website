## Problème

Vous obtenez cette erreur de sortie lorsque vous suivez l&apos;installation guidée.

```shell
  Unable to connect to the server: dial tcp [7777:777:7777:7777:77::77]:443: i/o timeout
```

## Solution

Cela indique que vous rencontrez un problème de connexion réseau entre le client Kubernetes et le serveur API Kubernetes. Assurez-vous que votre client Kubernetes peut se connecter à votre serveur API Kubernetes avant d&apos;exécuter à nouveau l&apos;installation guidée.
