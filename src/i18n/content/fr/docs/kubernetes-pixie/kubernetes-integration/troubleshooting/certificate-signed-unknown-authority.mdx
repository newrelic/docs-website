## Problème

Vous recevez le message `x509: certificate signed by unknown authority`.

## Solution

Vous devez vous assurer que vos certificats signés sont correctement configurés. Pour ce faire, vous pouvez ajouter les variables d’environnement suivantes dans le conteneur YML de l’agent :

```yml
- name: NRIA_CA_BUNDLE_DIR 
  value: YOUR_CA_BUNDLE_DIR
- name: NRIA_CA_BUNDLE_FILE  
  value: YOUR_CA_BUNDLE_NAME
```
