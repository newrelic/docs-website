## Problème

Vous avez installé l&apos; [intégration New Relic Kubernetes](/install/kubernetes/), mais vous ne voyez aucune donnée Kubernetes dans votre compte New Relic.

## Solution

1. Assurez-vous d&apos;avoir correctement configuré les paramètres `CLUSTER_NAME` et `NRI_LICENSE_KEY` dans le fichier [`values-newrelic.yaml`](https://github.com/newrelic/helm-charts/blob/master/charts/nri-bundle/values.yaml) .

2. Confirmez le déploiement réussi de l’intégration en exécutant ces commandes :

```bash
  kubectl get pods,deployments,daemonsets -o wide -n <newrelic_namespace>
```

3. [Vérifiez le log](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/get-logs-version/) de l&apos;agent infrastructure et de l&apos;intégration et assurez-vous qu&apos;il n&apos;y a pas de message d&apos;erreur.
