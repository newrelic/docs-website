## Problème

Vous obtenez des erreurs telles que `cannot increase buffer: current=32000 requested=64768 max=32000` ou `pod out of memory`.

## Solution

Ajoutez l&apos;attribut `k8sBufferSize` pour augmenter la taille de la mémoire tampon du bit fluide.

```yaml
newrelic-logging:
  enabled: true
  containerSecurityContext:
    privileged: true
  fluentBit:
    k8sBufferSize: "64k"
    persistence:
      mode: "none"
```
