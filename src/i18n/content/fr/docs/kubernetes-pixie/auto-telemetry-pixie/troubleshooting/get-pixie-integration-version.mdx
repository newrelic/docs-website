Pour déterminer quelle version de l’intégration New Relic Pixie est installée sur votre cluster, procédez comme suit :

1. Allez à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>.

2. Sélectionnez <DNT>**Query your data**</DNT>.

3. Copiez la requête suivante et cliquez sur <DNT>**RUN**</DNT>.

   ```sql
   FROM K8sContainerSample SELECT latest(containerImage) WHERE containerName LIKE '%newrelic-pixie%' FACET clusterName
   ```

Cette requête génère une liste de noms de cluster avec leurs versions d&apos;intégration pour le compte sélectionné :
