Pour vérifier l’état d’un cluster sur lequel Pixie est installé, exécutez le [script de diagnostic](https://github.com/newrelic-experimental/k8s-diag-utilities/tree/main/pixie-diag).

Ce script bash simple exécute les commandes Kubernetes standard dans l&apos;espace de nommage dans lequel Pixie est installé. Le résultat de ce script peut être utilisé pour effectuer une enquête plus approfondie par vous-même ou avec le support New Relic.
