## Dashboard Pixie pré-construit [#pixie-dashboards]

Pour installer un dashboard prédéfini pour les données Pixie :

1. Accédez à la page [Dashboards](https://one.newrelic.com/dashboards) New Relic.
2. Sélectionnez le bouton <DNT>**Browse pre-built dashboards**</DNT> .
3. Recherchez « Pixie » et choisissez l’une des options disponibles.
4. Sélectionnez le bouton <DNT>**Install this quickstart**</DNT> .

## Dashboard Pixie personnalisé [#custom-dashboards]

Le widget de visualisation dans le dashboard Pixie prédéfini est rempli de données provenant de requêtes NRQL . Sélectionnez dashboard widgetle menu à trois points de n’importe quel pour afficher la requête fournissant les données. Pour savoir comment écrire votre propre requête pour les données Pixie persistantes, consultez la page [Interrogation des données Pixie](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/query-pixie-data) .

Découvrez plus d&apos;informations sur la façon de personnaliser un dashboard prédéfini ou de créer votre propre dashboard personnalisé [ici](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).
