Utilisez l&apos; [entité](/docs/new-relic-solutions/new-relic-one/core-concepts/what-entity-new-relic/) Pixie pour voir les métriques dorées et un échantillon de requests individuelles de corps entier envoyées à un service spécifique utilisant un protocole spécifique. Les entités Pixie sont alimentées par des données qui ont été automatiquement tracées par Pixie. Sélectionnez un protocole ci-dessous pour en savoir plus :

* [DNS](#dns)
* [MySQL](#mysql)
* [PostgreSQL](#postgres)
* [Redis](#redis)
* [Kafka](#kafka)
* [AMQP](#amqp)

Vous pouvez trouver l&apos;entité Pixie sous la section <DNT>**Other entities**</DNT> de la page <DNT>**All entities**</DNT> :

<img title="Pixie entities show up under the All Entities section" alt="Pixie entities show up under the All Entities section" src="/images/pixie_screenshot-full_entities-list.webp" />

<figcaption>
  L&apos;entité Pixie apparaît sous la section <DNT>**All Entities**</DNT> .
</figcaption>

<Callout variant="important">
  Si l&apos;une de ces entités Pixie vous manque, vérifiez [votre Personnaliser Pixie Ingest](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/overview/#pixie-configure-storage) configuration. Vous avez peut-être désactivé l&apos;ingestion pour un certain protocole.
</Callout>

## DNS [#dns]

Sélectionnez un service dans la liste des entités <DNT>**DNS - Pixie**</DNT> pour afficher les graphiques chronologiques du temps de réponse, du taux d&apos;erreur et du débit pour toutes requests DNS envoyées à ce service. Inspectez le tableau des requêtes DNS échantillonnées pour voir l’intégralité requests et réponses DNS reçues par ce service.

requests DNS tracées automatiquement par Pixie peuvent être utilisées pour :

* Déterminez rapidement quels services effectuent requests DNS.
* Obtenez des informations de haut niveau sur le temps de réponse et le débit DNS.
* Observez les déséquilibres de débit entre les serveurs DNS.

<img title="The <DNT>**DNS - Pixie**</DNT> entity shows golden metrics and full-body DNS request and response pairs traced by Pixie" alt="The DNS Pixie entity shows golden metrics and full-body DNS request and response pairs traced by Pixie" src="/images/pixie_screenshot-full_dns_entity.webp" />

<figcaption>
  Recherchez l&apos;entité <DNT>**DNS - Pixie**</DNT> dans la section <DNT>**All Entities**</DNT> .
</figcaption>
