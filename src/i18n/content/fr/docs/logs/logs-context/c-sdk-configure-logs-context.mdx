<Callout variant="important" title="AVIS DE FIN DE VIE">
  À partir d&apos;avril 2022, nous ne prenons pas en charge la fonctionnalité C SDK. Pour plus de détails, consultez notre [publication sur le forum d&apos;assistance](https://discuss.newrelic.com/t/q1-bulk-eol-announcement-fy23/181744).
</Callout>

Les logs en contexte ne sont pas disponibles pour le moniteur d&apos;applications du SDK C. Au lieu de cela, vous pouvez créer votre propre bibliothèque d&apos;extensions de logging en utilisant l&apos;[API de Log](/docs/logs/log-management/log-api/introduction-log-api/). Assurez-vous que le tracing distribué est activé sur votre application, afin de pouvoir comprendre rapidement ce qui arrive aux requests lorsqu&apos;elles transitent par les services en amont et en aval.

Pour activer le tracing distribué pour le moniteur d&apos;applications par le SDK C, [installez](/docs/distributed-tracing/enable-configure/quick-start/) ou [mettez à jour](/docs/agents/c-sdk/install-configure/update-your-c-sdk-library/) vers la dernière version du SDK C. Le tracing distribué nécessite [C SDK version 1.1.0 ou supérieure](/docs/release-notes/agent-release-notes/c-sdk-release-notes/).
