## Problème

les logs n&apos;apparaissent pas dans New Relic pour une configuration d&apos;intégration CloudWatch via Firehose.

## Solution

Raisons possibles et leurs solutions :

* La clé API peut être incorrecte. Accédez à <DNT>**Amazon Data Firehose &gt; Firehose streams &gt; your-stream &gt; Edit destination settings**</DNT> et mettez à jour la clé API avec la clé de licence d’ingestion correcte.
* Le point de terminaison fourni peut être erroné. Vérifiez le point de terminaison.
* Vérifiez le log S3 ou le log des erreurs CloudWatch pour tout problème.
