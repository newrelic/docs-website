## Problème

les logs n&apos;apparaissent pas dans New Relic pour une configuration d&apos;intégration CloudWatch via Firehose.

## Solution

Raisons possibles et leurs solutions :

* Pour l&apos;intégration S3 via Lambda ou CloudWatch via Lambda , la variable d&apos;environnement `CUSTOM_META_DATA` peut ne pas être définie ou ne pas avoir de valeur JSON valide. Vérifiez-le.
* Pour CloudWatch via Firehose, les paires valeur-clé peuvent ne pas être définies correctement. Assurez-vous que vos paires valeur clé sont présentes sous <DNT>**Parameters**</DNT>.
* Vos noms de clés peuvent chevaucher des mots-clés interdits.
