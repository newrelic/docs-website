## Problème

Lors de la suppression d&apos;une stack CloudFormation, celle-ci peut échouer pour diverses raisons.

## Solution

Raison possible et sa solution :

* Si les buckets S3 créés par la stack (par exemple, pour le log des erreurs Firehose) ne sont pas vides, ils ne peuvent pas être supprimés. archivez le contenu puis réessayez de supprimer la stack.
