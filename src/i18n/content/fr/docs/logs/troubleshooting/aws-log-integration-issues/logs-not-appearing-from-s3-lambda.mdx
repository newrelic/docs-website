## Problème

les logs n&apos;apparaissent pas dans New Relic pour une configuration d&apos;intégration S3 via Lambda .

## Solution

Raisons possibles et leurs solutions :

* Vérifiez les raisons possibles pour lesquelles [les logs n&apos;apparaissent pas dans CloudWatch via Lambda](/docs/logs/troubleshooting/aws-log-integration-issues/logs-not-appearing-from-cloudwatch-lambda).
* Les fichiers S3 peuvent ne pas respecter les [restrictions de transfert de logs](/docs/logs/troubleshooting/aws-log-integration-issues/log-forwarding-restrictions) dans New Relic.
* Le `NewRelicServerlessLogForwarder` n&apos;a peut-être pas le déclencheur S3. Vérifiez la sortie de la pile CloudFormation imbriquée et ajoutez-la.
