Le dashboard des performances de l'API publique est un dashboard prêt à l'emploi inclus dans votre compte New Relic. Il fournit à la fois des informations générales exploitables détaillées sur les performances des API publiques et une opportunité pour les nouveaux clients de tester nos capacités de dashboard avant d'ajouter leurs propres données.

Le dashboard fonctionne en affichant la latence réelle subie par un échantillon anonymisé de clients New Relic lors de l'accès aux API publiques populaires.

<Callout variant="important">
  Les données sur les performances mondiales sont présentées telles quelles. L'ensemble de données sur les performances mondiales représente un agrégat d'échantillons provenant de diverses sources, et New Relic ne fait aucun effort pour tenter de confirmer l'exactitude, l'exhaustivité ou la véracité des données. Ces données ne doivent pas être considérées comme la seule source d'information à quelque fin que ce soit, et New Relic n'est pas responsable des décisions prises sur la base de ces données. L'ensemble de données sur les performances mondiales ne doit pas être considéré comme une approbation ou une recommandation par New Relic des technologies représentées dans l'ensemble de données.
</Callout>

## Ajouter le dashboard dans New Relic [#Add-dashboard]

Pour ajouter le dashboard des performances de l’API publique :

1. Depuis la page du dashboard, recherchez le dashboard prédéfini Performances de l’API publique. Vous pouvez cliquer sur le lien <DNT>**View All**</DNT> à côté du titre du dashboard prédéfini populaire pour le rechercher.
2. Sélectionnez le dashboard.
3. Sélectionnez un compte dans la liste déroulante.
4. Cliquez sur <DNT>**View dashboard**</DNT> pour l&amp;apos;ajouter à votre compte.

## Explorez le dashboard [#explore-dashboard]

Vous pouvez personnaliser votre nouveau dashboard prédéfini. Vous pouvez modifier le nom et les paramètres, ajouter du nouveau contenu et un nouveau widget, affiner les graphiques et bien plus encore ! Consultez [Gérer votre dashboard](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/) pour plus d&amp;apos;informations.

Vous trouverez ci-dessous quelques suggestions sur la façon d’explorer le dashboard des performances de l’API publique.

* Cliquez sur le <Icon name="fe-more-horizontal" /> dans le coin droit de l&amp;apos;un des graphiques et sélectionnez <DNT>**View query**</DNT> pour afficher la requête [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) utilisée pour créer le graphique.
* Cliquez sur le <Icon name="fe-more-horizontal" /> dans le coin droit de l&amp;apos;un des graphiques et sélectionnez <DNT>**Get as image**</DNT> pour afficher ou télécharger n&amp;apos;importe quel graphique sous forme d&amp;apos;image.
* Sélectionnez un domaine spécifique à partir du graphique à barres ou ajoutez un filtre en cliquant sur le champ de texte en haut de la page.
* Si vous avez déjà ajouté vos propres données, essayez de copier les requêtes et de les modifier pour votre propre usage.

<Callout variant="important">
  Le dashboard des performances de l'API publique n'est actuellement pas disponible pour les clients de l'UE.
</Callout>

<Callout variant="important">
  Le dashboard des performances de l'API publique ne prend actuellement pas en charge les alertes.
</Callout>

## Données rapportées [#data-reported]

Nous mettons les données API publiques à la disposition de toutes les organisations New Relic via l'événement `Public_APICall` . Ces données ne comptent pas dans votre ingestion. Si vous souhaitez voir la structure de ces données, exécutez cette requête :

```sql
FROM Public_APICall select keyset()
```
