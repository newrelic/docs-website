Nos SDK de télémétrie sont un ensemble open source de bibliothèque client API qui envoient des données à la plateforme New Relic. Sous le capot, ces SDK s&apos;appuient sur nos principales API d&apos;ingestion de données : l&apos;API métrique, l&apos;API trace, l&apos;API log et l&apos;API événement.

Si nos [solutions pré-construites](https://newrelic.com/instant-observability) ne répondent pas à vos besoins, nos SDK de télémétrie sont un moyen de créer une solution de télémétrie personnalisée (voir d&apos;autres [solutions de reporting de données personnalisées](/docs/data-apis/custom-data/intro-custom-data)).

## Exigences et compatibilité [#requirements]

Les SDK de télémétrie utilisent notre API métrique, notre API événement, notre API log et notre API trace, qui nécessitent toutes un <InlinePopover type="licenseKey" />, vous aurez donc besoin d&apos;une clé de licence pour le compte auquel vous souhaitez envoyer des données.

<Callout variant="tip">
  New Relic a contribué au SDK de télémétrie à la communauté open source sous une licence Apache 2.0.
</Callout>
