Par défaut, les produits New Relic signalent une variété de données utilisées dans nos graphiques UI et qui sont disponibles pour [l&apos;interrogation](/docs/using-new-relic/data/understand-data/query-new-relic-data). Nos produits ne transmettront pas d’informations sensibles sans être explicitement instrumentés pour le faire.

## Événement et attribut par défaut [#default-events]

Nos produits signalent un ensemble d&apos; [événements et d&apos;attributs par défaut](/docs/insights/insights-data-sources/default-data/insights-default-data-other-new-relic-products). Nous n&apos;enverrons jamais de paramètres de requête ou tout autre attribut qui ne sont pas dans l&apos;ensemble par défaut, à moins que quelqu&apos;un ne l&apos;ait explicitement activé via configuration.

## Ajuster les données rapportées [#default-security]

Lors de l&apos;évaluation des paramètres de sécurité d&apos;un produit New Relic, examinez l&apos; [événement et l&apos;attribut par défaut](/docs/insights/insights-data-sources/default-data/insights-default-data-other-new-relic-products). L&apos;attribut par défaut ne contient pas de données sensibles. En général, il s’agit simplement des données nécessaires à monitoring efficace des performances. Nos produits n&apos;envoient pas d&apos;autres données, sauf si vous modifiez les paramètres de sécurité par défaut.

Selon vos besoins, l&apos;une ou les deux situations suivantes peuvent s&apos;appliquer :

* Si la liste par défaut contient des données qui vous préoccupent, vous pouvez désactiver la collecte de ces attributs. Pour savoir comment modifier cela, consultez la documentation du produit que vous utilisez.
* Si vous devez envoyer des attributs non signalés par défaut, vous pouvez [activer ces attributs pour qu&apos;ils soient signalés](/docs/insights/insights-data-sources/custom-data/send-custom-data-insights). Dans ce cas, n&apos;utilisez pas [le mode haute sécurité](/docs/accounts-partnerships/accounts/security/high-security): cela désactivera la possibilité de collecter des attributs personnalisés.
