Les fonctionnalités de New Relic signalent différents [types de données](/docs/using-new-relic/data/understand-data/new-relic-data-types). Un type de données est [celui des données d’événement](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data). Les données d&apos;événement sont affichées dans les graphiques New Relic et d&apos;autres vues, et sont également mises à disposition pour [l&apos;interrogation](/docs/using-new-relic/data/understand-data/query-new-relic-data).

Apprenez-en plus sur l&apos;événement rapporté par New Relic :

* [Événement par défaut de l&apos;APM](/docs/insights/insights-data-sources/default-data/apm-default-events-insights)
* [Événement par défaut Browser](/docs/insights/insights-data-sources/default-data/browser-default-events-insights)
* [événement de défaut d&apos;infrastructure](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-events)
* [Événement par défaut mobile](/docs/insights/insights-data-sources/default-data/mobile-default-events-insights)
* [Synthétique défaut événement](/docs/insights/insights-data-sources/default-data/synthetics-default-events-insights)
* [NrAuditEvent événement](/docs/insights/event-data-sources/default-events/query-account-audit-logs-nrauditevent/) pour comprendre les changements sur votre compte

Pour en savoir plus sur d&apos;autres données, consultez [notre dictionnaire de données](/attribute-dictionary).
