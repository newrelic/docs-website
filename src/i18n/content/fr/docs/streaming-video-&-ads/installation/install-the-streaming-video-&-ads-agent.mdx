To install the video agent, we recommend following our guided installation process.

## Prérequis [#requirements]

Before installing the video agent, ensure you have a New Relic [account ID](/docs/accounts/accounts-billing/account-setup/account-id).

## Installer l&apos;agent vidéo [#installation]

To get started with Streaming Video &amp; Ads, install our video agent on your streaming media assets.

<Steps>
  <Step>
    Accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &amp;gt; Integrations &amp;amp; Agents &amp;gt; Streaming Video &amp;amp; Ads**</DNT>.
  </Step>

  <Step>
    Select the account you want to work with from the dropdown menu and click **Continue**.
  </Step>

  <Step>
    ## Sélectionnez le type d&apos;application de streaming [#select-streaming-app-type]

    * [Browser](/install/browser-media): monitorez les lecteurs vidéo dans vos applications de navigateur.

    * Mobile: Monitor video players on the following platforms:

      * [Android](/install/android): monitorez les lecteurs vidéo dans vos applications Android .
      * [iOS](/install/media-ios): Monitor video players in your iOS applications.

    * [Roku](/docs/streaming-video-&-ads/installation/roku/get-started-with-roku): monitorez les lecteurs vidéo dans vos applications Roku.
  </Step>
</Steps>
