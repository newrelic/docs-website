Suivez ces étapes pour installer l&apos;agent Streaming Video &amp; Ads.

## Prérequis [#requirements]

Avant d&apos;installer l&apos;agent vidéo :

* Assurez-vous d&apos;avoir un [ID de compte](/docs/accounts/accounts-billing/account-setup/account-id)New Relic
* Consultez les [exigences de compatibilité](/docs/streaming-video-&-ads/installation/compatibility-requirements) pour les plateformes et versions prises en charge.

## Installer l&apos;agent vidéo [#install]

1. Accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents &gt; Streaming Video &amp; Ads**</DNT>.
2. Sélectionnez votre compte dans le menu déroulant et cliquez sur <DNT>**Continue**</DNT>.
3. Choisissez votre plateforme pour continuer l&apos;installation spécifique à la plateforme :
