Pour installer l&apos;agent vidéo, nous vous recommandons de suivre notre processus d&apos;installation guidée.

## Prérequis [#requirements]

Avant d&apos;installer l&apos;agent vidéo, assurez-vous de disposer d&apos;un [identifiant de compte](/docs/accounts/accounts-billing/account-setup/account-id) New Relic.

## Installer l&apos;agent vidéo [#installation]

Pour commencer à utiliser Streaming vidéo et publicitaire, installez notre agent vidéo sur vos ressources multimédias en streaming.

<Steps>
  <Step>
    Accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents &gt; Streaming Video &amp; Ads**</DNT>.
  </Step>

  <Step>
    Sélectionnez le compte avec lequel vous souhaitez travailler dans le menu déroulant et cliquez sur **Continue**.
  </Step>

  <Step>
    ## Sélectionnez le type d&apos;application de streaming [#select-streaming-app-type]

    * [Browser](/install/browser-media): monitorez les lecteurs vidéo dans vos applications de navigateur.

    * Mobile : monitorez les lecteurs vidéo sur les plateformes suivantes :

      * [Android](/install/android): monitorez les lecteurs vidéo dans vos applications Android .
      * [iOS](/install/media-ios): monitorez les lecteurs vidéo dans vos applications iOS.

    * [Roku](/docs/streaming-video-&-ads/installation/roku/get-started-with-roku): monitorez les lecteurs vidéo dans vos applications Roku.
  </Step>
</Steps>
