Cette page fournit un aperçu des étapes d&apos;installation lors de l&apos;utilisation de CocoaPods pour installer l&apos;agent Streaming Video &amp; Ads. Pour l&apos;installation complète avec des exemples de code, utilisez l&apos;[installation guidée](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios).

<Steps>
  <Step>
    ### Ajouter des dépendances

    Ajoutez les pods de l&apos;agent vidéo à votre Podfile.
  </Step>

  <Step>
    ### Installer les pods

    Exécutez `pod install` dans votre terminal.
  </Step>

  <Step>
    ### Construire le projet

    Ouvrez votre projet dans **Xcode** et compilez-le.
  </Step>

  <Step>
    Cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Initialiser le tracker vidéo

    Sélectionnez le type d&apos;agent qui correspond à votre implémentation :

    * **Agent vidéo avec le tracker AVPlayer uniquement**: Pour le suivi de la lecture vidéo
    * **Agent vidéo avec les trackers AVPlayer et IMA**: Pour la lecture vidéo et le suivi publicitaire
  </Step>

  <Step>
    Cliquez sur <DNT>**See your data**</DNT> pour afficher vos données de streaming dans New Relic.
  </Step>
</Steps>
