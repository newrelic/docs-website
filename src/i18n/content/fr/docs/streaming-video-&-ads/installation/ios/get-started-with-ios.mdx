L&apos;agent vidéo pour iOS vous offre une vue complète de vos performances de lecture vidéo. L&apos;agent monitore les applications iOS écrites en Objective-C, Swift, ou les deux.

## Prérequis

Avant d&apos;installer l&apos;agent Streaming Video &amp; Ads pour iOS, assurez-vous d&apos;avoir :

* Un [ID de compte](/docs/accounts/accounts-billing/account-setup/account-id)New Relic
* Passé en revue les [exigences de compatibilité de l&apos;application mobile iOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements)
* Passé en revue les [exigences de compatibilité de l&apos;agent Streaming Video &amp; Ads](/docs/streaming-video-&-ads/installation/compatibility-requirements)

Vous pouvez installer l&apos;agent Streaming Video &amp; Ads à l&apos;aide de l&apos;une des deux méthodes :

* **CocoaPods**: Installer automatiquement à l&apos;aide du gestionnaire de dépendances CocoaPods
* **Installation manuelle**: compilez et incluez manuellement les fichiers du framework dans votre projet

## Installer l&apos;agent

<Steps>
  <Step>
    ### Accéder à New Relic

    Allez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT>.
  </Step>

  <Step>
    ### Sélectionner Vidéo en streaming &amp; Publicités

    Recherchez et sélectionnez la tuile <DNT>**Streaming Video &amp; Ads**</DNT>.
  </Step>

  <Step>
    ### Nommez votre application

    Donnez un nom à votre application et cliquez sur <DNT>**Create**</DNT>.

    Une fois le nom de l&apos;application créé, cliquez sur <DNT>**Save and Continue**</DNT>.
  </Step>

  <Step>
    ### Sélectionner la plateforme mobile

    Sélectionnez <DNT>**Mobile**</DNT> et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Sélectionnez iOS

    Sélectionnez <DNT>**iOS**</DNT> pour monitorer les lecteurs vidéo dans vos applications et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Choisissez votre méthode d&apos;installation

    Sélectionnez la méthode d&apos;installation qui s&apos;applique à votre application :

    <DocTiles>
      <DocTile title="Installer automatiquement avec CocoaPods" path="/docs/streaming-video-&-ads/installation/ios/install-automatically-using-cocoapods">
        Installez automatiquement l&apos;agent vidéo à l&apos;aide de CocoaPods.
      </DocTile>

      <DocTile title="Installer manuellement" path="/docs/streaming-video-&-ads/installation/ios/install-manually">
        Installez manuellement l&apos;agent vidéo.
      </DocTile>
    </DocTiles>
  </Step>
</Steps>

## Quelle est la prochaine étape ?

Après l&apos;installation de l&apos;agent :

1. Vérifiez que les données circulent dans New Relic en recherchant votre application dans le tableau de bord Streaming Video &amp; Ads
2. [Consultez vos données](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-features): explorez les fonctionnalités de monitoring et les tableaux de bord
3. [Configurer les seuils de qualité d&apos;expérience](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): personnalisez les métriques de performance pour votre application
