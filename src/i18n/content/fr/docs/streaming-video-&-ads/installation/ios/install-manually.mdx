Cette page fournit un aperçu des étapes d&apos;installation lors de l&apos;installation manuelle de l&apos;agent Streaming Video &amp; Ads. Pour l&apos;installation complète avec des exemples de code, utilisez l&apos;[installation guidée](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios).

<Steps>
  <Step>
    Clonez le dépôt [video-agent-iOS](https://github.com/newrelic/video-agent-ios).
  </Step>

  <Step>
    Ouvrez chacun des `.xcodeproj` fichiers avec Xcode.
  </Step>

  <Step>
    Sélectionnez le schéma approprié.
  </Step>

  <Step>
    Générer (CMD+B).
  </Step>

  <Step>
    Incluez les fichiers `.framework` générés dans votre projet.
  </Step>

  <Step>
    Cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Initialiser le tracker vidéo

    Sélectionnez le type d&apos;agent qui correspond à votre implémentation :

    * **Agent vidéo avec le tracker AVPlayer uniquement**: Pour le suivi de la lecture vidéo
    * **Agent vidéo avec les trackers AVPlayer et IMA**: Pour la lecture vidéo et le suivi publicitaire
  </Step>

  <Step>
    Cliquez sur <DNT>**See your data**</DNT> pour afficher vos données de streaming dans New Relic.
  </Step>
</Steps>
