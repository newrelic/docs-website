Pour monitorer les performances du lecteur vidéo dans votre application de navigateur, installez l&apos;agent vidéo New Relic à l&apos;aide de npm ou d&apos;un extrait de code JavaScript.

## Prérequis [#before-start]

Avant d&apos;installer l&apos;agent Streaming Video &amp; Ads pour navigateur, vérifiez que votre application répond à ces exigences :

* [Exigences de compatibilité du monitoring de navigateur](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring)
* [Exigences de compatibilité de l&apos;Agent Streaming Video &amp; Ads](/docs/streaming-video-&-ads/installation/compatibility-requirements)

## Installer l&apos;agent [#install]

<Steps>
  <Step>
    ### Accéder à New Relic

    Accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT> et rendez-vous sur la marketplace des intégrations.
  </Step>

  <Step>
    ### Sélectionner Vidéo en streaming &amp; Publicités

    Recherchez et sélectionnez la tuile <DNT>**Streaming Video &amp; Ads**</DNT>.
  </Step>

  <Step>
    ### Sélectionnez un compte

    Sélectionnez un <DNT>**Account**</DNT> dans le menu déroulant et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Sélectionner la plateforme

    Sélectionnez <DNT>**Browser**</DNT> et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Sélectionner le lecteur vidéo

    Sélectionnez votre lecteur vidéo et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Choisir la méthode d&apos;installation

    <DocTiles>
      <DocTile title="Installer avec npm" path="/docs/streaming-video-&-ads/installation/browser/install-using-npm">
        Installez le gestionnaire de paquets de l&apos;agent vidéo pour les applications JavaScript.
      </DocTile>

      <DocTile title="Placer un extrait de code JavaScript dans le code frontend" path="/docs/streaming-video-&-ads/installation/browser/install-using-snippet">
        Installez l&apos;agent vidéo en incluant un extrait de code JavaScript dans votre application.
      </DocTile>
    </DocTiles>
  </Step>
</Steps>

## Quelle est la prochaine étape ?

Après l&apos;installation de l&apos;agent :

1. Vérifiez que les données circulent dans New Relic en recherchant votre application dans le tableau de bord Streaming Video &amp; Ads
2. [Consultez vos données](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-features): explorez les fonctionnalités de monitoring et les tableaux de bord
3. [Configurer les seuils de qualité d&apos;expérience](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): personnalisez les métriques de performance pour votre application
