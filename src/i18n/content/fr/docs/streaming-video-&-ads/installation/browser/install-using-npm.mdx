Installez l&apos;agent vidéo à l&apos;aide du gestionnaire de paquets npm pour les applications JavaScript.

## Avant de commencer

Effectuez d&apos;abord les [étapes d&apos;installation communes](/docs/streaming-video-&-ads/installation/browser/get-started-with-browser#install).

## Installer avec npm [#install-npm]

<Steps>
  <Step>
    ### Nommez votre application

    Donnez un nom à votre application et cliquez sur <DNT>**Save and Continue**</DNT>.
  </Step>

  <Step>
    ### Installer l&apos;agent

    Installez la dernière version de New Relic en copiant l&apos;extrait de code et en l&apos;exécutant dans votre terminal (Mac) ou votre invite de commandes (Windows).

    <Callout variant="important">
      Sélectionnez **npm** ou **Yarn** pour installer l&apos;agent vidéo.
    </Callout>
  </Step>

  <Step>
    ### Ajoutez du code à votre application

    Une fois le script téléchargé, incluez l&apos;extrait de code affiché sur la plateforme dans votre application pour commencer à monitorer votre lecteur vidéo.
  </Step>

  <Step>
    ### Générer l&apos;ID d&apos;application

    Sélectionnez <DNT>**Get Application Info**</DNT>. Cela générera l&apos;ID de l&apos;application pour la génération de l&apos;entité navigateur.
  </Step>

  <Step>
    ### Visualisez vos données

    Cliquez sur <DNT>**See your data**</DNT>.
  </Step>
</Steps>

<Callout variant="important">
  Cela peut prendre quelques minutes pour interagir avec votre application et générer des données.
</Callout>
