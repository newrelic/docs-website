<table>

      <td>
        [nrAddDomainSubstitution](/docs/streaming-video-&-ads/installation/roku/agent-api/domain-management/nradddomainsubstitution)

      <td>
        Ajoutez un motif de correspondance pour l&apos;attribut de domaine et remplacez-le par une autre chaîne de caractères. Chaque fois qu&apos;un événement ou une métrique est généré avec un attribut de domaine, l&apos;agent vérifie s&apos;il correspond à une regex et applique la substitution spécifiée. Si aucun modèle n&apos;est défini, il utilisera le domaine de l&apos;URL tel quel. Il s&apos;applique à tous les événements et métriques contenant l&apos;attribut &quot;domain&quot;.
      </td>
    </tr>

    <tr>

        [nrDelDomainSubstitution](/docs/streaming-video-&-ads/installation/roku/agent-api/domain-management/nrdeldomainsubstitution)
      </td>

      <td>
        Supprimer un modèle correspondant créé avec `nrAddDomainSubstitution`.
      </td>
    </tr>
  </tbody>

## Exemples

### nrAddDomainSubstitution

```js
  	nrAddDomainSubstitution(nr, "^.+\.my\.domain\.com$", "mydomain.com")
```

### nrDelDomainSubstitution

```js
  	nrDelDomainSubstitution(nr, "^.+\.my\.domain\.com$")
```
