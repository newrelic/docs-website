      <td>
        Enregistrer une jauge métrique. Représente une valeur qui peut augmenter ou diminuer avec le temps.
      </td>
    </tr>

    <tr>
      <td>
        [nrSendCountMetrique](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendcountmetric)
      </td>

      <td>
        Enregistrer un compte métrique. Mesure le nombre d&apos;occurrences d&apos;un événement pendant un intervalle de temps.
      </td>
    </tr>

    <tr>
      <td>
        [nrSendSummaryMetric](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendsummarymetric)
      </td>

      <td>
        Enregistrer une métrique récapitulative. Utilisé pour signaler des données pré-agrégées ou des informations sur des événements discrets agrégés.
      </td>
    </tr>

## Exemples

### nrSendMetric

```js
  	nrSendMetric(m.nr, "test", 11.1, {"one": 1})
```

### nrSendCountMetrique

```js
  	nrSendCountMetric(m.nr, "test", 250, 1500, {"one": 1})
```

### nrSendSummaryMetric

```js
  	nrSendSummaryMetric(m.nr, "test", 2000, 5, 1000, 100, 200)
```
