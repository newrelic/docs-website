## Syntaxe

```js
  nrUpdateConfig(nr as Object, config as Object) as Void
```

## Description

Met à jour la configuration, comme l&apos;URL du proxy réseau.

Rien.

## Exemple

```js

	  nrUpdateConfig(m.nr, config)
```
