## Exemples

### Envoyer l&apos;événement APP\_STARTED

```brightscript
sub Main(aa as Object)
    ...
    nrAppStarted(m.nr, aa)
```

### Envoyer l&apos;événement SCENE\_LOADED

```brightscript
nrSceneLoaded(m.nr, "MyVideoScene")
```

### Traiter les messages de log système

```brightscript
while (true)
    msg = wait(0, m.port)
    if nrProcessMessage(m.nr, msg) = false
        if type(msg) = "roPosterScreenEvent"
            if msg.isScreenClosed()
                exit while
            end if
        end if
    end if
end while
```
