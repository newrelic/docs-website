## Syntaxe

```js
  function nrSendMetric(nr as Object, name as String, value as dynamic, attr = invalid as Object) as Void
```

## Description

Enregistrer une jauge métrique. Représente une valeur qui peut augmenter ou diminuer avec le temps.
