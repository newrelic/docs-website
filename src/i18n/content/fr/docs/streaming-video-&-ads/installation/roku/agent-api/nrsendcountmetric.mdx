## Syntaxe

```js
  function nrSendCountMetric(nr as Object, name as String, value as dynamic, interval as Integer, attr = invalid as Object) as Void
```

## Description

Enregistrer un compte métrique. Mesure le nombre d&apos;occurrences d&apos;un événement pendant un intervalle de temps.
