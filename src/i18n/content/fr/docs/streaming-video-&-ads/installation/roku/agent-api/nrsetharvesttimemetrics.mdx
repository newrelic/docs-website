## Syntaxe

```js
  nrSetHarvestTimeMetrics(nr as Object, time as Integer) as Void
```

## Description

Définissez l&apos;heure de récolte du log, l&apos;heure à laquelle le log est mis en mémoire tampon avant d&apos;être envoyé à New Relic. La valeur minimale est 60.
