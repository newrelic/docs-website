## Syntaxe

```js
  nrSetCustomAttributeList(nr as Object, attr as Object, actionName = "" as String) as Void
```

## Description

Définissez une liste d&apos;attributs personnalisés à inclure dans l&apos;événement.

  <thead>
    <tr>
      <th width="25%">

  <tbody>

        `nr`
      </td>

      <td>

      </td>
    </tr>

    <tr>
      <td>
        `attr`
      </td>

      <td>
        liste d&apos;attributs, en tant qu&apos;éventail associatif.
      </td>
    </tr>

    <tr>
      <td>
        `actionName`
      </td>

      <td>
        (facultatif) Action où l&apos;attribut sera inclus. Par défaut toutes les actions.
      </td>
    </tr>
  </tbody>
</table>

## Retour

Rien.

## Exemple

```js
  	attr = {"key0":"val0", "key1":"val1"}
	  nrSetCustomAttributeList(m.nr, attr, "CONTENT_HEARTBEAT")
```
