## Syntaxe

```js
  nrSetHarvestTime(nr as Object, time as Integer) as Void
```

## Description

Définissez l&apos;heure de récolte, l&apos;heure à laquelle les échantillons sont mis en mémoire tampon avant d&apos;être envoyés à New Relic pour l&apos;événement et le log. La valeur minimale est 60.
