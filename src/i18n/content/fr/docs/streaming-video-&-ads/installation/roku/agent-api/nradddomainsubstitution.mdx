## Syntaxe

```js
  nrAddDomainSubstitution(nr as object, pattern as String, subs as String) as Void
```

## Description

* Ajoutez un modèle correspondant pour l’attribut de domaine et remplacez-le par une autre chaîne.
  * Chaque fois qu&apos;un événement ou une métrique est généré avec un attribut de domaine, l&apos;agent vérifiera s&apos;il correspond à une expression régulière et appliquera la substitution spécifiée.
* Si aucun modèle n&apos;est défini, le domaine URL sera utilisé sans changement.
  * Elle s&apos;applique à tous les événements et métriques contenant l&apos;attribut &quot;domaine&quot;.

## Exemple
