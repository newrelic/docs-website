## Guide de démarrage rapide

### 1. Initialisez l&apos;agent

Commencez par initialiser l&apos;agent New Relic dans votre application Roku :

```brightscript
sub Main(aa as Object)
    screen = CreateObject("roSGScreen")
    m.port = CreateObject("roMessagePort")
    screen.setMessagePort(m.port)
    scene = screen.CreateScene("VideoScene")
    screen.show()

    m.nr = NewRelic("ACCOUNT ID", "API KEY")
```

### 2. Démarrer le logging système

Activer le monitoring des événements système :

```brightscript
m.syslog = NewRelicSystemStart(m.port)
```

### 3. Suivre les sessions vidéo

Monitorer la lecture vidéo :

```brightscript
NewRelicVideoStop(m.nr)
```

### 4. Envoyer des événements personnalisés

Suivre les événements applicatifs :

```brightscript
nrSendCustomEvent(m.nr, "MyEvent", "MY_ACTION")
attr = {"key0":"val0", "key1":"val1"}
nrSendCustomEvent(m.nr, "MyEvent", "MY_ACTION", attr)
```

### 5. Ajouter des attributs personnalisés

Ajoutez des métadonnées à vos événements :

```brightscript
nrSetCustomAttribute(m.nr, "myNum", 123, "CONTENT_START")
nrSetCustomAttribute(m.nr, "myString", "hello")
```

### 6. Définir l&apos;identifiant utilisateur

Associer des événements à des utilisateurs spécifiques :

```brightscript
nrSetUserId(m.nr, "TEST_USER")
```
