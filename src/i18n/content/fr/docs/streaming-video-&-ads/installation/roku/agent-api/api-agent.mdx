L&apos;agent New Relic Roku est conçu pour monitorer les événements système et vidéo dans les applications Roku. Il fournit une API complète pour suivre différents types d&apos;événements, notamment le système, la vidéo, les erreurs, les publicités et les événements personnalisés. Ce document décrit les méthodes API disponibles pour interagir avec l&apos;agent New Relic Roku.
