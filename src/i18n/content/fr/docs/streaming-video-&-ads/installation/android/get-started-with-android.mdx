L&apos;agent vidéo pour Android monitore le lecteur vidéo de votre application mobile, vous offrant une vue complète des performances de lecture de vos vidéos et publicités. Cela fonctionne pour les applications Android écrites en Java ou Kotlin.

## Prérequis

Avant d&apos;installer l&apos;agent Streaming Video &amp; Ads pour Android, assurez-vous de disposer de :

* Un [ID de compte](/docs/accounts/accounts-billing/account-setup/account-id)New Relic
* Avoir passé en revue les [exigences de compatibilité de l&apos;application mobile Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements)
* Passé en revue les [exigences de compatibilité de l&apos;agent Streaming Video &amp; Ads](/docs/streaming-video-&-ads/installation/compatibility-requirements)

Déterminez quel chemin d&apos;installation s&apos;applique à vous :

* **Nouvelle application mobile**: Vous ajoutez New Relic à une application pour la première fois. Vous devrez d&apos;abord installer l&apos;[agent Android New Relic](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android).
* **Application monitorée existante**: l&apos;agent Android New Relic est déjà installé sur votre application et celle-ci envoie des données à New Relic.

## Installer l&apos;agent

1. Allez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT>.
2. Recherchez et sélectionnez la tuile <DNT>**Streaming Video &amp; Ads**</DNT>.
3. Sélectionnez <DNT>**Mobile**</DNT> et cliquez sur <DNT>**Continue**</DNT>.
4. Sélectionnez <DNT>**Android**</DNT> pour monitorer les lecteurs vidéo dans vos applications Android et cliquez sur <DNT>**Continue**</DNT>.
5. Donnez un nom à votre application et cliquez sur <DNT>**Create**</DNT>.
6. Une fois le nom de l&apos;application créé, cliquez sur <DNT>**Save and continue**</DNT>.
7. Sélectionnez la méthode d&apos;instrumentation qui s&apos;applique à votre application :

## Quelle est la prochaine étape ?

Après l&apos;installation de l&apos;agent :

1. Vérifiez que les données circulent dans New Relic en recherchant votre application dans le tableau de bord Streaming Video &amp; Ads
2. [Consultez vos données](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-features): explorez les fonctionnalités de monitoring et les tableaux de bord
3. [Configurer les seuils de qualité d&apos;expérience](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): personnalisez les métriques de performance pour votre application
