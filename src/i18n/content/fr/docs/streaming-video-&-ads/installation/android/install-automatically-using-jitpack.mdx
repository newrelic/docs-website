Cette page fournit une vue d&apos;ensemble des étapes d&apos;installation lors de l&apos;utilisation de JitPack pour installer l&apos;agent Streaming Video &amp; Ads. Pour l&apos;installation complète avec des exemples de code, utilisez l&apos;[installation guidée](https://onenr.io/02R54MqY0wb).

<Steps>
  <Step>
    ## Ajouter le dépôt JitPack

    Ajoutez le dépôt JitPack à votre fichier `build.gradle` racine.
  </Step>

  <Step>
    ## Ajouter des dépendances

    Ajoutez les dépendances de l&apos;agent vidéo au fichier `build.gradle` de votre application et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ## Configurer l&apos;agent vidéo

    Ajoutez la configuration NRVideo dans la méthode `onCreate` de votre **classe Application ou MainActivity**.
  </Step>

  <Step>
    ## Configurer le suivi du lecteur vidéo

    Sélectionnez le type d&apos;agent qui correspond à votre implémentation :

    * **Agent vidéo avec tracker ExoPlayer uniquement**: Pour le suivi de la lecture vidéo

    * **Agent vidéo avec ExoPlayer et trackers IMA**: Pour la lecture vidéo et le suivi publicitaire

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    Cliquez sur <DNT>**See your data**</DNT> pour afficher vos données de streaming dans New Relic.
  </Step>
</Steps>
