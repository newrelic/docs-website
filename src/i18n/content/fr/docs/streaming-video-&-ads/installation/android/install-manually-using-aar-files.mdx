Cette page présente une vue d&apos;ensemble des étapes d&apos;installation lors de l&apos;utilisation de fichiers AAR pour installer manuellement l&apos;agent Streaming Video &amp; Ads. Pour l&apos;installation complète avec des exemples de code, utilisez l&apos;[installation guidée](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clonez le dépôt [video-agent-android](https://github.com/newrelic/video-agent-android).
  </Step>

  <Step>
    Ouvrez-le avec Android Studio.
  </Step>

  <Step>
    Accédez à <DNT>**View &gt; Tool Windows &gt; Gradle**</DNT> pour ouvrir la fenêtre d&apos;outil Gradle.
  </Step>

  <Step>
    Dans la fenêtre d&apos;outil Gradle, développez <DNT>**NRVideoProject &gt; Tasks &gt; build**</DNT> et double-cliquez sur <DNT>**assemble**</DNT>. Cela compilera le projet et générera les fichiers de bibliothèque AAR dans le dossier de chaque module <DNT>**build &gt; outputs &gt; aar**</DNT>.
  </Step>

  <Step>
    Dans votre projet, accédez à <DNT>**File &gt; New &gt; New Module**</DNT>, importez le package `.JAR/.AAR` et cliquez sur <DNT>**Next**</DNT>.
  </Step>

  <Step>
    Sélectionnez les fichiers AAR que vous avez générés à l&apos;étape précédente et cliquez sur <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    Répétez les étapes 5 et 6 pour chaque fichier AAR dont vous avez besoin :

    * **NewRelicVideoCore.aar** (Requis) : Fonctionnalité de base de l&apos;agent vidéo
    * **NRExoPlayerTracker.aar** (Requis) : suivi d&apos;ExoPlayer
    * **NRIMATracker.aar** (Facultatif) : suivi publicitaire avec le SDK IMA
  </Step>

  <Step>
    Dans le fichier `build.gradle` de votre module d&apos;application, ajoutez les dépendances suivantes :

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")   // Required: Core video agent
      implementation project(":NRExoPlayerTracker")  // Required: ExoPlayer tracking
      implementation project(':NRIMATracker')        // Optional: Ad tracking with IMA
    }
    ```

    Remarque : incluez `NRIMATracker` uniquement si vous avez besoin de la fonctionnalité de suivi publicitaire.
  </Step>

  <Step>
    Synchronisez votre projet avec les fichiers Gradle en cliquant sur <DNT>**Sync Now**</DNT> dans la barre de notifications, ou accédez à <DNT>**File &gt; Sync Project with Gradle Files**</DNT>.
  </Step>

  <Step>
    ### Initialiser l&apos;agent vidéo

    Dans la méthode ou l&apos;activité `onCreate()` de votre application, initialisez l&apos;agent vidéo New Relic. Pour obtenir des exemples de code détaillés et des instructions d&apos;implémentation, consultez l&apos;[installation guidée](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    ### Configurer le suivi du lecteur vidéo

    Configurez le tracker en fonction de vos besoins :

    * **Tracker ExoPlayer uniquement**: configurez le suivi ExoPlayer pour le monitoring de la lecture vidéo
    * **Trackers ExoPlayer et IMA**: configurez le suivi ExoPlayer et IMA pour la lecture vidéo et le suivi publicitaire.

    <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />

    Pour des exemples complets de code d&apos;implémentation, consultez l&apos;[installation guidée](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    Compilez et déployez votre application pour voir les données de suivi vidéo dans New Relic.
  </Step>
</Steps>
