Cette page fournit une vue d&apos;ensemble des étapes d&apos;installation lors de l&apos;utilisation du code source pour installer manuellement l&apos;agent Streaming Video &amp; Ads. Pour l&apos;installation complète avec des exemples de code, utilisez l&apos;[installation guidée](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clonez le dépôt [video-agent-android](https://github.com/newrelic/video-agent-android).
  </Step>

  <Step>
    Dans votre projet Android, accédez à <DNT>**File &gt; New &gt; Import Module**</DNT>.
  </Step>

  <Step>
    Saisissez l&apos;emplacement du répertoire du module de bibliothèque et cliquez sur <DNT>**Finish**</DNT>.

    <Callout variant="important">
      Le répertoire du module de bibliothèque se trouve dans le dépôt que vous venez de cloner. Exemples de modules : `NewRelicVideoCore`, `NRExoPlayerTracker`, `NRIMATracker`.
    </Callout>
  </Step>

  <Step>
    Répétez les étapes 2 et 3 pour tous les modules que vous souhaitez inclure.
  </Step>

  <Step>
    Dans le fichier `build.gradle` du module de votre application, ajoutez ce qui suit :

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    Cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Configurer le suivi du lecteur vidéo

    Sélectionnez le type d&apos;agent qui correspond à votre implémentation :

    * **Agent vidéo avec tracker ExoPlayer uniquement**: Pour le suivi de la lecture vidéo

    * **Agent vidéo avec ExoPlayer et trackers IMA**: Pour la lecture vidéo et le suivi publicitaire

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    Cliquez sur <DNT>**See your data**</DNT> pour déployer votre code mis à jour et voir les données.
  </Step>
</Steps>
