Vous pouvez avoir des centaines d&apos;applications différentes diffusant du contenu vidéo sur différentes plateformes et avoir besoin monitoring la lecture de vidéos et de publicités dans toutes les applications. Pour vous aider à gérer cette complexité, New Relic vous permet de visualiser les données télémétriques de lecture de vidéos et de publicités à partir d&apos;une application individuelle ou de toutes les applications sur différentes plateformes :

* [Vue toutes plateformes](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/overview): Visualisez les données télémétriques de lecture des vidéos et des publicités de toutes les applications où l&amp;apos;agent vidéo a été instrumenté.
* [Vue individuelle de la plateforme](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/overview): Visualisez les données télémétriques de lecture des vidéos et des publicités à partir d&amp;apos;une application/plateforme spécifique sur laquelle l&amp;apos;agent vidéo a été installé.
