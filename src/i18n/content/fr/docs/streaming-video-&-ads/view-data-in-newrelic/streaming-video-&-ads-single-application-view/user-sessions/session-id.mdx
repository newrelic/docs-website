## utilisateur session stream événement

Les événements streamings de session utilisateur sont les événements générés pendant la session. Ces événements permettent de suivre le comportement de l&apos;utilisateur au cours de la session.

Un graphique est affiché sur la page de session de l&apos;utilisateur qui montre les événements générés pendant la session et des informations détaillées sur l&apos;événement telles que :
