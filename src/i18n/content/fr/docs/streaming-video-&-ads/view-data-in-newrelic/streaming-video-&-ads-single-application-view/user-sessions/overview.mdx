## Présentation

La page User Sessions fournit une vue détaillée des sessions de consultation individuelles pour une application monitorée spécifique. Cette page vous aide à suivre le comportement de visionnage des utilisateurs, à identifier les sessions avec des erreurs et à analyser les schémas de lecture au sein d&apos;une seule application ou plateforme de streaming.

### Ce que vous pouvez faire avec User Sessions

Utilisez cette page pour :

* Afficher toutes les sessions de visionnage des utilisateurs pour une application spécifique sur laquelle l&apos;agent vidéo est installé
* Suivez les détails au niveau de la session, y compris l&apos;ID de vue, l&apos;ID de session et l&apos;ID utilisateur
* Monitorer le contenu consulté par titre
* Identifier les sessions avec des erreurs pour le dépannage
* Affichez les heures de début et de fin de session pour analyser la durée de visionnage
* Accédez aux informations détaillées de la session en cliquant sur l&apos;ID de vue ou l&apos;ID de session.
* Filtrer les sessions par plage de temps pour se concentrer sur des périodes spécifiques
* Analyser les habitudes de visionnage pour l&apos;appareil et la plateforme spécifiques
* Examiner les détails du contenu pour chaque session

### Qui utilise cette page

Cette page est utile pour les développeurs d&apos;applications, les équipes de support client et les ingénieurs en assurance qualité qui doivent résoudre les problèmes signalés par les utilisateurs pour une application spécifique, analyser les sessions de visionnage individuelles et comprendre les modèles de comportement des utilisateurs au sein de cette plateforme.

## Page User Sessions

La **Single platform &gt; User Sessions** pour le streaming vidéo et les publicités répertorie toutes les sessions d&apos;une application spécifique.
