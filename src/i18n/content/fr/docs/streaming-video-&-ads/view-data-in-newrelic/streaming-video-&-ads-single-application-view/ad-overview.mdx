La page **Single platforms &gt; Ads Overview** fournit une vue d&amp;apos;ensemble des performances des annonces d&amp;apos;une application spécifique. Cela offre un aperçu complet des mesures de performance des publicités pour des applications de monitoring spécifiques. La page d&amp;apos;aperçu des annonces comprend les informations suivantes :

## Afficher la page de présentation de l&apos;annonce [#view-the-ad-overview-page]

Pour afficher la page d&apos;aperçu de l&apos;annonce de l&apos;agent vidéo d&apos;une application spécifique :

1. Aller à <DNT>**[one.newrelic.com](https://one.newrelic.com) &amp;gt; All Capabilities &amp;gt; Streaming Video &amp;amp; Ads &amp;gt; Streaming Entities**</DNT>

2. Sélectionnez l&apos;application.

3. Cliquez sur <DNT>**Ads Overview**</DNT>.

   <img title="Streamng video & ads Ads Overview" alt="A screenshot about using streaming video & ads" src="/images/ads-overview.webp" />

## Utilisation de la fonctionnalité de filtrage [#using-the-filter-functionality]

Par défaut, la page **Ads Overview** affiche les données de performances de lecture des annonces sur des applications d&amp;apos;entité spécifiques sur lesquelles l&amp;apos;agent vidéo a été instrumenté. À l&amp;apos;aide de la barre de filtre, vous pouvez rechercher n&amp;apos;importe quel attribut ou valeur, par exemple, vous pouvez filtrer les données par **titre de contenu, ID de l&apos;appareil, ID de l&apos;utilisateur, plateforme**, etc.

## Dimensions KPI

Explorez et analysez le trafic en profondeur en utilisant plusieurs paramètres, notamment la plateforme et le nom du lecteur vidéo. Obtenez des informations détaillées sur les habitudes de visionnage des clients, telles que le pourcentage de clients qui regardent un contenu spécifique, la résolution à laquelle les vidéos sont visionnées et des informations détaillées sur la version du système d&apos;exploitation de l&apos;appareil.

* **Plateforme**: Le type d&amp;apos;appareil, par exemple, Roku, Browser, Android ou appareil iOS.

* **Version OS**: La version du système d&amp;apos;exploitation de l&amp;apos;appareil. Si vous utilisez un appareil Android ou iOS, la version du système d&amp;apos;exploitation sera affichée.

* **Nom du lecteur vidéo**: Le nom du lecteur vidéo utilisé pour lire la vidéo.

* **FAI**: Le fournisseur de services Internet (FAI) utilisé par le téléspectateur.

* **Région**: La situation géographique du spectateur.

* **Résolution de l&apos;annonce**: la résolution de l&amp;apos;annonce diffusée sur l&amp;apos;appareil du spectateur.

## Ajouter une entité de streaming

Par défaut, la page **Ads Overview** affiche toutes les entités associées au compte. Cependant, si vous souhaitez créer une nouvelle entité de streaming :

1. Cliquez sur **+ Add streaming entity**, dans le coin supérieur droit de la page.
2. Sélectionnez le type d’application que vous souhaitez monitorer. Voir [Installer l&apos;agent multimédia](/docs/streaming-video-&-ads/installation/install-the-streaming-video-&-ads-agent/#installation)
