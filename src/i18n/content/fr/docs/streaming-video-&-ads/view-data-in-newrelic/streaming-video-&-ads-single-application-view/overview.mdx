## Présentation

La vue Individual Platform fournit des capacités détaillées de monitoring et d&apos;analyse pour une application de streaming spécifique. Cette vue vous permet d&apos;analyser en profondeur les performances spécifiques à l&apos;application, de résoudre les problèmes isolés sur une seule plateforme et de monitorer l&apos;expérience de streaming complète pour cette application.

### Ce que vous pouvez faire avec la vue Individual Platform

Utilisez cette vue pour :

* Monitorer les métriques de performance des vidéos et des annonces pour une application spécifique où l&apos;agent vidéo est installé
* Analyser les métriques de qualité vidéo et d&apos;engagement spécifiques à l&apos;application
* Afficher les performances détaillées des annonces pour l&apos;application spécifique
* Suivre les sessions utilisateur dans l&apos;application
* Résoudre les problèmes isolés sur une seule plateforme ou application
* Monitorer les dépendances et les relations des services pour l&apos;application
* Créer des conditions d&apos;alerte spécifiques à l&apos;application
* Accédez à la page Summary pour une vue consolidée des performances de l&apos;application.

### Qui utilise cette vue

Cette vue est utile pour les développeurs d&apos;applications, les chefs de produit et les équipes de support qui doivent monitorer et dépanner une application de streaming spécifique, analyser les problèmes de performances au sein de cette plateforme et s&apos;assurer que l&apos;application répond aux normes de qualité.

### Quand utiliser la vue Individual Platform vs la vue All Platform

Utilisez la **vue Individual Platform** lorsque vous avez besoin de :

* Analyse approfondie des performances d&apos;une application spécifique
* Résoudre les problèmes isolés sur une seule plateforme
* Monitorer les métriques et configurations spécifiques à l&apos;application
* Analyser les sessions utilisateur dans une application spécifique
* Afficher les cartes des services et les dépendances de l&apos;application
* Créer des alertes spécifiques à l&apos;application

Utilisez la **vue All Platform** lorsque vous avez besoin de :

* Comparer les performances entre plusieurs applications
* Identifiez les tendances affectant l&apos;ensemble de votre infrastructure de streaming.
* Monitorer la santé globale du streaming sur toutes les plateformes
* Examiner les problèmes pouvant affecter plusieurs applications

## Fonctionnalités de la vue Individual Platform

Découvrez nos fonctionnalités clés pour la vue individuelle de la plateforme New Relic Streaming Video &amp; Ads.

* [Résumé](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/media-summary-page): Affichez le résumé d&apos;une application/plateforme spécifique sur laquelle l&apos;agent vidéo a été installé.
* [Aperçu de l&apos;annonce](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/ad-overview): Affichez l&apos;aperçu de l&apos;annonce à partir d&apos;une application/plateforme spécifique sur laquelle l&apos;agent vidéo a été installé.
* [Sessions utilisateur](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/user-sessions/overview): Affichez les sessions d&apos;utilisateurs à partir d&apos;une application/plateforme spécifique sur laquelle l&apos;agent vidéo a été installé.
* [Cartes des services](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/service-maps): consultez les cartes des services pour les médias en streaming et les publicités afin de comprendre les dépendances.
* [condition d&apos;alerte](/docs/tutorial-create-alerts/create-an-alert): Afficher la condition d&apos;alerte pour une application/plateforme spécifique sur laquelle l&apos;agent vidéo a été installé.

## Quelle est la prochaine étape ?

Étendez votre monitoring avec des vues multiplateformes et des fonctionnalités supplémentaires :

* [Vue All Platform](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/overview): comparez les performances sur toutes les applications où l&apos;agent vidéo est installé
* [Entités de streaming](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/streaming-entities): visualisez et comparez les KPI sur toutes les applications de streaming
* [Paramètres de l&apos;application](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): configurer les métriques de qualité d&apos;expérience et les seuils spécifiques à l&apos;application
* [Mettre à jour l&apos;agent média](/docs/streaming-video-&-ads/update-the-media-agent): maintenez votre agent à jour avec les dernières fonctionnalités
