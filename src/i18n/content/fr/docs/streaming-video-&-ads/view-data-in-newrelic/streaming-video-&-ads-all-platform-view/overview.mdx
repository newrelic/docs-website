La vue All Platform fournit un emplacement centralisé pour monitorer les performances de lecture des vidéos et des publicités sur l&apos;ensemble de vos applications et plateformes de streaming. Cette vue vous aide à comparer les performances entre différentes applications, à identifier les tendances sur l&apos;ensemble de votre infrastructure de streaming et à repérer rapidement les problèmes affectant plusieurs plateformes.

### Ce que vous pouvez faire avec la vue All Platform

Utilisez cette vue pour :

* Monitorer les performances de streaming sur toutes les applications où l&apos;agent vidéo a été instrumenté
* Comparez les KPI vidéo et publicitaires sur différentes applications et plateformes
* Identifier les problèmes à l&apos;échelle de la plateforme affectant plusieurs applications
* Analysez les tendances de performance sur l&apos;ensemble de votre infrastructure de streaming
* Afficher toutes les entités de streaming actives en un seul endroit
* Monitorer les sessions utilisateur sur toutes les applications et plateformes
* Accédez aux métriques de vue d&apos;ensemble des vidéos et des publicités agrégées sur toutes les plateformes

### Qui utilise cette vue

Cette vue est utile pour les équipes d&apos;exploitation de streaming, les responsables de l&apos;ingénierie et les dirigeants qui ont besoin d&apos;une vue d&apos;ensemble des performances de streaming à l&apos;échelle de toute l&apos;organisation, peuvent comparer les performances entre différentes applications ou plateformes, et doivent identifier les problèmes à l&apos;échelle de l&apos;infrastructure.

### Quand utiliser la vue All Platform vs la vue Individual Platform

Utilisez la **vue All Platform** lorsque vous avez besoin de :

* Comparer les performances entre plusieurs applications
* Identifiez les tendances affectant l&apos;ensemble de votre infrastructure de streaming.
* Monitorer la santé globale du streaming sur toutes les plateformes
* Examiner les problèmes pouvant affecter plusieurs applications

Utilisez la **vue Individual Platform** lorsque vous avez besoin de :

* Analyse approfondie des performances d&apos;une application spécifique
* Résoudre les problèmes isolés sur une seule plateforme
* Monitorer les métriques et configurations spécifiques à l&apos;application
* Analyser les sessions utilisateur dans une application spécifique
