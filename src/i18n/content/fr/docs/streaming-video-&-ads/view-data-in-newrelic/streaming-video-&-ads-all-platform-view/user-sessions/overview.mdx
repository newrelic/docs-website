La section **All Platforms &gt; User Sessions** pour le streaming vidéo et les publicités répertorie toutes les sessions sur toutes les applications.

## Afficher l'UI de la page Sessions utilisateur

Le tableau de la page **User Session** comporte plusieurs colonnes pour afficher diverses informations pour chaque session. Le tableau suivant fournit les informations sur chaque colonne disponible dans l&amp;apos;UI de la page Session :

<Callout variant="important">
  Le tableau des sessions affiche toutes les sessions utilisateur qui ont démarré dans le filtre temporel sélectionné au-dessus du tableau.
