## Présentation

La page User Sessions fournit une vue détaillée des sessions de visionnage individuelles sur toutes vos applications et plateformes de streaming monitorées. Cette page vous aide à suivre le comportement de visionnage des utilisateurs, à identifier les sessions contenant des erreurs et à analyser les schémas de lecture sur l&apos;ensemble de votre infrastructure de streaming.

### Ce que vous pouvez faire avec User Sessions

Utilisez cette page pour :

* Afficher toutes les sessions de visionnage utilisateur sur toutes les applications où l&apos;agent vidéo est installé
* Suivez les détails au niveau de la session, y compris l&apos;ID de vue, l&apos;ID de session, l&apos;ID utilisateur et le nom de l&apos;application.
* Monitorer le contenu visionné par titre et par plateforme
* Identifier les sessions avec des erreurs pour le dépannage
* Affichez les heures de début et de fin de session pour analyser la durée de visionnage
* Accédez aux informations détaillées de la session en cliquant sur l&apos;ID de vue ou l&apos;ID de session.
* Filtrer les sessions par plage de temps pour se concentrer sur des périodes spécifiques
* Analyser les habitudes de visionnage sur différents appareils et plateformes
* Examiner les détails du contenu pour chaque session

### Qui utilise cette page

Cette page est utile pour les équipes de support client, les ingénieurs en assurance qualité et les équipes d&apos;exploitation qui doivent résoudre les problèmes signalés par les utilisateurs, analyser les sessions de visionnage individuelles sur différentes plateformes et comprendre les modèles de comportement des utilisateurs sur l&apos;ensemble des applications de streaming.

## Page User Sessions

La section **All Platforms &gt; User Sessions** pour le streaming vidéo et les publicités répertorie toutes les sessions sur toutes les applications.
