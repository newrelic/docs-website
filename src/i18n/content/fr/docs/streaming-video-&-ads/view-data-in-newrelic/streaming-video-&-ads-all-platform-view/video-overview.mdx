## UI de la page de présentation de la vidéo

La page **All Platforms &gt; Video Overview** fournit une vue d&apos;ensemble des performances vidéo sur différentes plateformes. Ceci offre un aperçu complet des mesures de performances vidéo pour toutes les applications de monitoring. La page de présentation de la vidéo comprend les informations suivantes :

<img title="overview of the media summary page" alt="A gif depicting an overview of the media summary page" src="/images/video-overview-page.gif" />

Pour afficher la page de présentation vidéo de l&apos;agent vidéo sur toutes les plateformes, accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; All Capabilities &gt; Streaming Video &amp; Ads &gt; Video Overview**</DNT>.

## Utilisation de la fonctionnalité de filtrage [#using-the-filter-functionality]

Par défaut, la page **Video Overview** affiche les données de performances de lecture vidéo sur toutes les applications d&apos;entité sur lesquelles l&apos;agent vidéo a été instrumenté. À l&apos;aide de la barre de filtre, vous pouvez rechercher n&apos;importe quel attribut ou valeur, par exemple, vous pouvez filtrer les données par titre de contenu, ID d&apos;appareil, ID d&apos;utilisateur, plateforme, etc.

## Dimensions KPI

Explorez et analysez le trafic en profondeur en utilisant plusieurs paramètres, notamment la plateforme et le nom du lecteur vidéo. Obtenez des informations détaillées sur les habitudes de visionnage des clients, telles que le pourcentage de clients qui regardent un contenu spécifique, la résolution à laquelle les vidéos sont visionnées et des informations détaillées sur la version du système d&apos;exploitation de l&apos;appareil.

* **Plateforme**: Le type d&apos;appareil, par exemple, Roku, Browser, Android ou appareil iOS.

* **Version OS**: La version du système d&apos;exploitation de l&apos;appareil. Si vous utilisez un appareil Android ou iOS, la version du système d&apos;exploitation sera affichée.

* **Nom du lecteur vidéo**: Le nom du lecteur vidéo utilisé pour lire la vidéo.

* **FAI**: Le fournisseur de services Internet (FAI) utilisé par le téléspectateur.

* **Région**: La situation géographique du spectateur.

* **Résolution vidéo**: La résolution de la vidéo lue par le spectateur.

## Ajouter une entité de streaming

Par défaut, la page **Streaming Entities** affiche toutes les entités associées au compte. Cependant, si vous souhaitez créer une nouvelle entité de streaming :

1. Cliquez sur **+ Add streaming entity**, dans le coin supérieur droit de la page.
2. Sélectionnez le type d’application que vous souhaitez monitorer. Voir [Installer l&apos;agent vidéo](/docs/streaming-video-&-ads/installation/install-the-streaming-video-&-ads-agent/#installation)
