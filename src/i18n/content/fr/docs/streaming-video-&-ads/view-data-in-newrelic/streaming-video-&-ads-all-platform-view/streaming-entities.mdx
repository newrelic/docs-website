La page Streaming Entities fournit une vue centralisée de toutes les applications sur lesquelles l&apos;agent vidéo New Relic est installé. Cette page vous aide à monitorer et comparer les performances de lecture des vidéos et des publicités sur toutes vos applications et plateformes de streaming.

### Ce que vous pouvez faire avec les entités de streaming

Utilisez cette page pour :

* Afficher toutes les applications où l&apos;agent vidéo a été instrumenté dans un seul tableau
* Comparez les KPI vidéo et publicitaires sur différentes applications et plateformes
* Identifier les applications présentant un taux de rebuffering élevé, des échecs de démarrage ou des échecs de lecture
* Évaluez les métriques de qualité vidéo telles que le débit moyen et le temps de démarrage sur toutes les entités
* Cliquez sur n&apos;importe quelle application pour examiner les données détaillées de lecture des vidéos et des publicités.
* Créez de nouvelles entités de streaming directement depuis cette page

### Qui utilise cette page

Cette page est utile pour les équipes gérant plusieurs applications de streaming sur différentes plateformes, qui ont besoin d&apos;évaluer rapidement les performances sur l&apos;ensemble de leur infrastructure de streaming et d&apos;identifier les applications qui nécessitent une attention particulière.

## Page des entités de streaming

La page Entité Streaming affiche toutes les applications avec l&apos;agent vidéo New Relic installé. Dans la vue tableau, vous trouverez un résumé des indicateurs clés de performance de la vidéo et de la publicité. Vous pouvez cliquer sur n’importe quelle application pour examiner et analyser ses données de lecture vidéo et publicitaire.

## Ajouter une entité de streaming

Si vous souhaitez créer une nouvelle entité de streaming :

1. Cliquez sur **+ Add streaming entity**, dans le coin supérieur droit de la page.
2. Sélectionnez le type d’application que vous souhaitez monitorer. Voir [Installer l&apos;agent vidéo](/docs/streaming-video-&-ads/installation/install-the-streaming-video-&-ads-agent/#installation)

## Quelle est la prochaine étape ?

* Afficher [la présentation vidéo](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/video-overview) sur toutes les applications/plateformes.
* Afficher [l&apos;aperçu des annonces](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/ads-overview) sur toutes les applications/plateformes.
* Visualisez [les sessions utilisateur](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/user-sessions/overview) sur toutes les applications/plateformes.
