La page Entité Streaming affiche toutes les applications avec l&apos;agent vidéo New Relic installé. Dans la vue tableau, vous trouverez un résumé des indicateurs clés de performance de la vidéo et de la publicité. Vous pouvez cliquer sur n’importe quelle application pour examiner et analyser ses données de lecture vidéo et publicitaire.

## UIla page de présentation de l&apos;entité de streaming

## Ajouter une entité de streaming

Si vous souhaitez créer une nouvelle entité de streaming :

1. Cliquez sur **+ Add streaming entity**, dans le coin supérieur droit de la page.
2. Sélectionnez le type d’application que vous souhaitez monitorer. Voir [Installer l&apos;agent vidéo](/docs/streaming-video-&-ads/installation/install-the-streaming-video-&-ads-agent/#installation)

## Quelle est la prochaine étape ?

* Afficher [la présentation vidéo](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/video-overview) sur toutes les applications/plateformes.
* Afficher [l&apos;aperçu des annonces](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/ads-overview) sur toutes les applications/plateformes.
* Visualisez [les sessions utilisateur](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/user-sessions/overview) sur toutes les applications/plateformes.
