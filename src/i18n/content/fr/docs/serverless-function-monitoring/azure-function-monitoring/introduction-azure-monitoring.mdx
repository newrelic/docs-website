L’instrumentation de New Relic pour Azure Functions offre une visibilité approfondie sur le fonctionnement interne de vos fonctions. En ajoutant l&apos;agent New Relic à votre fonction, chaque invocation est monitorée par New Relic. Cela vous permet de suivre la durée des invocations de fonction, de monitorer les erreurs, de voir le tracing distribué et d&apos;enregistrer les détails sur l&apos;événement source et les réponses de votre fonction.

<Callout variant="important">
  Pour Azure Functions, l&apos;agent signale des données telles que les traces distribuées et les logs, qui sont disponibles sous « Services - APM » en tant qu&apos;entité APM. Cependant, une entité distincte (fournie par monitoring d&apos;infrastructure) pour Azure Functions existe toujours.
</Callout>
