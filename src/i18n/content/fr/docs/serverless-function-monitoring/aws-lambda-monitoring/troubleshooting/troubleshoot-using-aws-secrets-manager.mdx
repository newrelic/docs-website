## Problème

Vous ne pouvez pas utiliser AWS Secrets Manager pour stocker votre clé de licence New Relic.

## Solution

Si votre organisation n&apos;autorise pas l&apos;utilisation d&apos;AWS Secrets Manager, l&apos;extension New Relic Lambda accepte une variable d&apos;environnement `NEW_RELIC_LICENSE_KEY` . Ajoutez l’indicateur `--disable-license-key-secret` de la commande `newrelic-lambda integrations install`. Définissez ensuite cette variable d’environnement sur votre <InlinePopover type="licenseKey" />dans la configuration de votre fonction Lambda.
