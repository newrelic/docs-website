## Problème

Vous ne pouvez pas ajouter plusieurs comptes AWS avec leurs régions respectives à l&apos;extension New Relic Lambda.

## Solution

La CLI `newrelic-lambda` doit être exécutée une fois par région, avec le paramètre `--aws-region`. Utilisez le même nom de compte lié et l’outil détectera que le lien de compte a déjà été créé. Le secret <InlinePopover type="licenseKey" />doit être créé dans chaque région.

De même, plusieurs comptes AWS peuvent être liés à un compte New Relic. Donnez à chaque compte un nom de compte lié différent. L&apos;argument `--aws-profile` de l&apos;outil CLI sélectionnera le profil nommé. L&apos;outil utilise la même configuration que l&apos;AWS CLI.
