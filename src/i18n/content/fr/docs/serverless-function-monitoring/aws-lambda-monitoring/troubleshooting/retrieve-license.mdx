## Problème

Vous ne pouvez pas récupérer la clé de licence New Relic `AccessDeniedException`.

## Solution

Votre code lambda nécessite le rôle d’exécution qui dispose de l’autorisation de lire AWS Secrets Manager. Si vous trouvez un log comme celui-ci, ajoutez l’autorisation appropriée à la politique du rôle d’exécution. Dans nos exemples, consultez le fichier `template.yaml` pour voir un moyen simple d’accorder cette autorisation.

```bash
[output] Failed to retrieve license key AccessDeniedException: User: <ARN> is not authorized to perform: secretsmanager:GetSecretValue on resource: <ARN>
```
