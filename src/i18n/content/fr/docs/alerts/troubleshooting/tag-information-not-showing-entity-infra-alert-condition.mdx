## Problème

[Les balises de description incident personnalisées](/docs/alerts-applied-intelligence/new-relic-alerts/alert-conditions/alert-custom-incident-descriptions#example) ne sont pas remplacées par les valeurs de l’événement.

## Solution

Les balises ne sont disponibles que pour les événements indexés par New Relic.

Le moyen le plus simple de savoir si un événement est indexé est d'exécuter la requête `SELECT nr.entityType FROM ${EventSampleName}`, en remplaçant `${EventSampleName}` par le nom de l&amp;apos;événement en question.

S'il y a des valeurs dans la colonne `nr.entity Type` du résultat, alors l&amp;apos;événement est indexé. Sinon ce n&amp;apos;est pas le cas.

## Cause

Il n'est pas possible d'ajouter une description à un échantillon qui n'est pas indexé dans New Relic.
