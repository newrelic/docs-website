## Problème

Les [balises de description d&apos;événement d&apos;alerte personnalisée](/docs/alerts/create-alert/condition-details/custom-alert-event-descriptions#example) ne sont pas remplacées par les valeurs de l&apos;événement.

## Solution

Les balises ne sont disponibles que pour les événements indexés par New Relic.

Le moyen le plus simple de savoir si un événement est indexé est d&apos;exécuter la requête `SELECT nr.entityType FROM ${EventSampleName}`, en remplaçant `${EventSampleName}` par le nom de l&apos;événement en question.

S&apos;il y a des valeurs dans la colonne `nr.entity Type` du résultat, alors l&apos;événement est indexé. Sinon ce n&apos;est pas le cas.

## Cause

Il n&apos;est pas possible d&apos;ajouter une description à un échantillon qui n&apos;est pas indexé dans New Relic.
