Intégrez New Relic à vos équipes Microsoft pour recevoir des alertes directement dans vos canaux Teams. Cette intégration vous aide à rester informé de l’état de vos applications et de votre infrastructure.

**Prérequis :**

* Un compte Microsoft Teams actif.
* Installez l’ [application New Relic pour Microsoft Teams](https://appsource.microsoft.com/en-us/product/Office365/WA200008155) dans votre environnement Microsoft Teams. Pour plus d’informations, reportez-vous à [Ajouter New Relic pour Microsoft Teams](#add-new-relic-for-microsoft-teams).

## Ajouter New Relic pour Microsoft Teams [#add-new-relic-for-microsoft-team]

Pour recevoir des alertes dans vos canaux Teams, installez l’application New Relic dans votre environnement Microsoft Teams.

Pour ajouter l’application New Relic pour Microsoft Teams :

1. Dans Microsoft Teams, cliquez sur **Apps**.
2. Recherchez l’application **New Relic pour Microsoft Teams** et cliquez sur **Add**.
3. Sélectionnez une chaîne pour recevoir une notification. Un message de réussite et un code de sécurité seront publiés dans le canal.

## Configurer l&apos;intégration de Microsoft Teams dans New Relic [#configure-microsoft-teams-integration-in-new-relic]

Configurez l’intégration au sein de la plateforme New Relic pour la lier à vos équipes Microsoft. Cela implique la configuration d’une destination pour [les alertes.](/docs/alerts/overview/)

Pour configurer l’intégration de Microsoft Teams dans New Relic :

1. Accédez à **[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Alerts &amp;gt; Enrich and Notify &amp;gt; Destinations &amp;gt; Microsoft Teams**.
2. Donnez un nom à l’intégration et saisissez le code de sécurité depuis la [configuration de l’application Teams](#add-new-relic-for-microsoft-teams).
3. Cliquez sur **Save destination**. Un message de réussite apparaît après avoir validé le code de sécurité.

### Configurer une alerte pour Microsoft Teams [#set-up-an-alert-for-microsoft-teams]

Configurez les alertes à envoyer au canal en fonction de la condition d&apos;alerte. Pour en savoir plus sur les alertes, reportez-vous à [alertes.](/docs/alerts/overview/)

Pour ajouter une alerte :

1. Accédez à **[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Alerts &amp;gt; Enrich and Notify &amp;gt; Notification Workflow**.

2. Pour créer un nouveau workflow, cliquez sur **Add a workflow** ou sélectionnez un workflow existant à modifier.

3. Fournissez les informations requises.

4. Dans la section **Add channel** , cliquez sur **Microsoft Teams**:

   1. Sélectionnez la destination dans la liste déroulante Destination **Microsoft Teams** .
   2. Choisissez l’équipe et la chaîne dans les menus déroulants respectifs. Assurez-vous que le nom du canal correspond à celui sur lequel l’application New Relic est ajoutée.
   3. Pour vérifier la réception de la notification dans votre canal Teams, cliquez sur **Send test notification**.
   4. Cliquez sur **Save message**.

5. Cliquez sur **Activate Workflow**.
