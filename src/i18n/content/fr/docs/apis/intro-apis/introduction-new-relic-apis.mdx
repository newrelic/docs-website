New Relic propose une variété d'API et de SDK que vous pouvez utiliser pour :

* Envoyer des données à New Relic.
* Récupérer les données de New Relic.
* Afficher et configurer les paramètres.

Vous recherchez une clé API? Voir [la clé API](/docs/apis/intro-apis/new-relic-api-keys).

## API pour l'ingestion de données [#data-type-apis]

Nos API d'ingestion de données font partie de nos [nombreuses solutions de reporting de données](https://newrelic.com/instant-observability). Notre API peut être utilisée directement, mais elle constitue également la voie d&amp;apos;ingestion sous-jacente pour bon nombre de nos outils de reporting de données. Si vous commencez tout juste à signaler des données à New Relic, nous vous recommandons de commencer par [Installer New Relic](/docs/new-relic-solutions/new-relic-one/install-configure/install-new-relic).

## NerdGraph (GraphQL) [#graphql]

[NerdGraph](/docs/apis/graphql-api/getting-started/introduction-new-relic-graphql-api) est l&amp;apos;API que nous recommandons pour interroger les données New Relic , interroger les informations de compte et effectuer une gamme de configurations de fonctionnalités. Pour savoir ce que vous pouvez faire, consultez [les tutoriels NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph/#tutorials).

NerdGraph est notre toute nouvelle API et constitue notre tentative de rassembler en un seul endroit certaines de nos anciennes API. Notez qu'il existe encore certaines fonctionnalités que vous pouvez faire avec [APIREST](#rest-api) qui ne peuvent pas encore être faites avec NerdGraph, et c&amp;apos;est pourquoi certaines organisations New Relic utilisent toujours l&amp;apos;API REST.

## REST API [#rest-api]

Notre [API REST](/docs/apis/rest-api-v2) est notre ancienne API pour l&amp;apos;interrogation et la configuration, que [NerdGraph](#graphql) est en train de remplacer. Nous effectuons un travail minimal et continu sur l&amp;apos;API REST. L&amp;apos;API REST dispose encore de certaines capacités configuration que NerdGraph n&amp;apos;a pas encore, mais votre premier choix devrait être d&amp;apos;utiliser NerdGraph car c&amp;apos;est là que nous concentrerons notre attention et ajouterons des fonctionnalités.

## API par fonctionnalité [#product-apis]

Les outils et fonctionnalités New Relic , comme APM, monitoring d'infrastructure, <InlinePopover type="browser" />et les alertes, sont souvent utilisés ensemble et peuvent parfois se chevaucher en termes de fonctionnalités. C&amp;apos;est pourquoi plusieurs API peuvent être pertinentes pour chaque domaine. Certaines fonctionnalités API dépendront de votre [accès aux fonctionnalités et aux données](/docs/accounts/accounts-billing/general-account-settings/factors-affecting-access-features-data).
