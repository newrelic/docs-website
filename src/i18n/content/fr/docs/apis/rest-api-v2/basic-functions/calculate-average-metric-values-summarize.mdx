Par défaut, l&apos;appel de New Relic RESTAPI renvoieune série de valeurs de données métriques basées sur des [restrictions de plage de temps](/docs/apm/apis/requirements/extracting-metric-data#time). Pour obtenir la moyenne de ces valeurs, incluez `&summarize=true` dans votre appel d&apos;API. Par exemple:

```sh
curl -X GET "https://api.newrelic.com/v2/applications/${APP_ID}/metrics/data.json" \
     -H "Api-Key:$API_KEY" -i \
     -d 'names[]=Agent/MetricsReported/count&from=2014-08-11T14:42:00%2B08:00&to=2014-08-11T15:12:00%2B08:00&summarize=true'
```

Si vous avez un [compte régional UE](/docs/using-new-relic/welcome-new-relic/getting-started/introduction-eu-region-data-center), l&apos;URL ci-dessus commence par `https://api.eu.newrelic.com`.

<Callout variant="tip">
  Pour éviter de résumer les données, omettez `summarize` dans votre appel d&apos;API. Vous n&apos;avez pas besoin de spécifier `&summarize=false`.
</Callout>
