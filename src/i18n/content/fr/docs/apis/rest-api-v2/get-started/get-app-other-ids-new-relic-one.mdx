Lorsque vous utilisez l&apos;[New Relic API REST , vous devez remplacer certains ID d&apos;espace réservé](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2) `$APP_ID` `$KEYTX_ID``$COMPONENT_ID`(comme,,, etc.) par l&apos;ID d&apos;une application, d&apos;une transaction ou d&apos;un autre composant.

Vous pouvez utiliser l&apos;UI de New Relic pour obtenir certains de ces identifiants. Vous pouvez également obtenir une liste d&apos;ID à l&apos;aide de le [REST API Explorer](/docs/apis/rest-api-v2/get-started/list-application-id-host-id-instance-id).

## Rechercher des identifiants à l&apos;aide de l&apos;UI [#ui]

L&apos;UI comprend les identifiants de l&apos;application et d&apos;autres composants dans un panneau métadonnée. Pour voir les métadonnées d&apos;une entité, cliquez sur une entité monitorée dans la liste de l&apos;explorateur d&apos;entités, puis cliquez sur l&apos;icône tag .

Voici quelques exemples :

<CollapserGroup>
  <Collapser id="apm" title="APM : ID d'application">
    Pour localiser le à `$APP_ID` partir de l&apos;UI de New Relic :

    1. Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; APM**</DNT> et sélectionnez une application.

    2. Cliquez sur le <Icon name="fe-info" /> **<DNT>Metadata</DNT>** icône à côté du nom de l&apos;application pour ouvrir le panneau de métadonnées.

    3. Localisez le <DNT>**App ID**</DNT>.

       Remplacez ce numéro par le `$APP_ID` dans votre appel d&apos;API REST.
  </Collapser>

  <Collapser id="browser" title="ID de l'application du navigateur">
    Pour localiser le à `$APP_ID` partir de l&apos;UI de New Relic :

    1. Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT> et sélectionnez une application de monitoring.

    2. Cliquez sur le <Icon name="fe-info" /> **<DNT>Metadata</DNT>** icône à côté du nom de l&apos;application pour ouvrir le panneau de métadonnées.

    3. Localisez le <DNT>**App ID**</DNT>.

       Remplacez ce numéro par le `$APP_ID` dans votre appel d&apos;API REST.
  </Collapser>

  <Collapser id="mobile" title="ID de l'application mobile">
    Pour localiser le à `$MOBILE_ID` partir de l&apos;UI de New Relic :

    1. Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile**</DNT> et sélectionnez une application de monitoring.

    2. Cliquez sur le <Icon name="fe-info" /> **<DNT>Metadata</DNT>** icône à côté du nom de l&apos;application pour ouvrir le panneau de métadonnées.

    3. Localisez le <DNT>**App ID**</DNT>.

       Remplacez ce numéro par le `$MOBILE_ID` dans votre appel d&apos;API REST.
  </Collapser>
</CollapserGroup>
