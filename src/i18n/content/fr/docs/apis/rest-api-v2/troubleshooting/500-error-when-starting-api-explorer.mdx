## Problème

Une erreur `500` apparaît lors du démarrage de l’explorateur d’API.

## Solution

Pour corriger cela, assurez-vous que vous disposez d&apos;un compte par défaut :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; ([user menu](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)) &amp;gt; User preferences**</DNT>.
2. Dans la liste déroulante <DNT>**Default account**</DNT> , sélectionnez votre choix, puis enregistrez.

## Cause

Cela se produit lorsque vous n&apos;avez pas de compte New Relic par défaut actuel ou valide associé à votre connexion. Cela empêche l&apos;Explorateur d&apos;afficher la clé API à votre disposition.
