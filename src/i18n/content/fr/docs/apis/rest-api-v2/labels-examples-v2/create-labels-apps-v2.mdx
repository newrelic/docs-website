## Lister les étiquettes existantes [#list\_labels][#list_labels]

Pour lister les libellés qui existent dans votre compte, utilisez cette commande :

```sh
curl 'https://api.newrelic.com/v2/labels.json' \
     -H "Api-Key:NEW_RELIC_USER_KEY" -i
```

### Afficher la sortie [#list\_label\_output][#list_label_output]

La sortie pour l&apos;étiquette comprend :

* Le nom de clé complet pour l&apos;étiquette (par exemple, `Staging:Test1`), ainsi que le composant et le nom de la clé
* L&apos;application `health_status` (rouge, orange, vert, gris) et l&amp;apos;identifiant d&amp;apos;application pour le label dans chaque catégorie
* Applications portant cette étiquette ; par exemple, l&apos;étiquette `Staging:Test1` a été appliquée à une application (4937390) et à deux hôtes (3801143 et 5879721)

La sortie d&apos;une liste d&apos;étiquettes ressemblera à ceci :

```json

      "category": "Production",

        "red": [],
        "gray": []
      },
      "links": {
        "applications": [],
      }
    },
    {
      "key": "Staging:Test1",
      "category": "Staging",
      "name": "Test1",
```
