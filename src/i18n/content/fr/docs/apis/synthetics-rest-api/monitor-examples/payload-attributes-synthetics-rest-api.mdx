## Point de terminaison du moniteur spécifique [#specific-monitor]

Lorsque vous faites un appel d&apos;API REST pour un moniteur spécifique, incluez le `monitor_uuid` dans le point de terminaison. Le `monitor_uuid` est le GUID qui fait partie de l&apos;URL. Par exemple, un [moniteur Synthétique](/docs/synthetics/new-relic-synthetics/pages/synthetics-overview-page-view-monitors-performance) sélectionné a cette URL :

```
https://synthetics.newrelic.com/accounts/nnnn/monitors/ab123-c456d-e78-90123-f45g
```

Le `monitor_uuid` est la valeur qui suit `/monitors/`.
