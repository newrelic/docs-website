Browserdes applications monopages (SPA) monitoring du inclut une API pour ajouter monitoring personnalisée de l'interaction spécifique du navigateur. Cela est utile pour monitoring les interactions que New Relic n'enregistre pas automatiquement car elles n'entraînent pas de modifications d'itinéraire, comme un widget mis à jour dynamiquement. L'API SPA vous permet également de désactiver monitoring par défaut pour les interactions que vous ne considérez pas comme suffisamment importantes pour être monitorées.

Pour plus d'informations sur l'API SPA, y compris l'appel d'API spécifique, consultez l' [agent Browser et l'API SPA](/docs/browser/new-relic-browser/guides/guide-using-browser-spa-apis/).
