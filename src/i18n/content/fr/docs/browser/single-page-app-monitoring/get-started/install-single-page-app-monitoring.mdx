Le monitoring des applications à page unique (SPA) est fournie avec l&apos; [installation par défaut de l&apos;agent <InlinePopover type="browser" />](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent/).

## Exigences [#requirements]

Vous pouvez consulter [la compatibilité et les exigences pour le monitoring SPA ici](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring#spa/).

Lorsque vous configurez votre première application de monitoring dans un compte New Relic, vous devez accepter les conditions d&apos;utilisation. En acceptant les conditions, vous autorisez New Relic à collecter des fragments de hacher à partir d&apos;URL. Il vous suffit de sélectionner l&apos;option case à cocher une seule fois pour un compte.

## Activer ou désactiver monitoringSPA [#enable-spa]

Lorsque vous [activez monitoring des navigateurs](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#enable), monitoring SPA est incluse par défaut car elle donne accès à une gamme de nos fonctionnalités les plus récentes, y compris le tracing distribué. Certaines installations d&apos;agents plus anciennes peuvent nécessiter une mise à niveau. En savoir plus sur [les types d’agents de navigateur](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#agent-types).
