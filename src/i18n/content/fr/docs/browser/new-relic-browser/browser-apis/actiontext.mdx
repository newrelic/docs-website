## Syntaxe

```js
newrelic.interaction().actionText(string $value)
```

Définit la valeur de texte de l&apos;élément HTML sur lequel on a cliqué pour démarrer une interaction avec le navigateur.

## Exigences

* Agent navigateur Pro+SPA (v1099 ou supérieur)

* Si vous utilisez npm pour installer l&apos;agent du navigateur, vous devez activer la fonctionnalité `spa` lors de l&amp;apos;instanciation de la classe `BrowserAgent` . Dans l&amp;apos;éventail `features` , ajoutez ce qui suit :

  ```js
  import { Spa } from '@newrelic/browser-agent/features/spa';

  const options = {
    info: { ... },
    loader_config: { ... },
    init: { ... },
    features: [
      Spa
    ]
  }
  ```

  Pour plus d&apos;informations, consultez la [documentation d&apos;installation du navigateur npm](https://www.npmjs.com/package/@newrelic/browser-agent#new-relic-browser-agent).

## Description

Cette méthode monitoring SPA définit la valeur de texte de l&apos;élément HTML sur lequel on a cliqué pour démarrer une interaction avec le navigateur. La valeur sera exposée comme l&apos;attribut [`actionText`](/docs/insights/explore-data/attributes/browser-default-attributes-insights#interaction-actionText) dans l&amp;apos;événement `BrowserInteraction` .

REMARQUE : l&apos;agent tente automatiquement de déterminer la valeur de texte de l&apos;élément HTML sur lequel on a cliqué. Ceci s&apos;applique uniquement aux éléments `<a>`, `<button>` et `<input>` . L&amp;apos;API `actionText` peut être utilisée dans les cas où cela n&amp;apos;est pas suffisant ou précis.

<Callout variant="tip">
  Cet appel d&apos;API s&apos;applique aux données dans [les pages vues SPA](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring) dans le navigateur et au type d&amp;apos;événement [`BrowserInteraction`](/docs/insights/insights-data-sources/default-data/browser-default-events-attributes-insights#browserinteraction-attributes) . Pour définir un nom personnalisé pour les vues de page standard et le type d&amp;apos;événement <DNT>**`PageView`**</DNT> , voir [`setPageViewName`](/docs/browser/new-relic-browser/browser-agent-spa-api/set-pageview-name). Il est recommandé d&amp;apos;utiliser les deux appels ensemble.
</Callout>

## Valeurs de retour

Cette méthode renvoie le même objet API créé par `interaction()`.

## Exemples

```js
document.getElementById('subscribe').addEventListener('submit', () => {
    newrelic.interaction().actionText('Create Subscription');
    createSubscription();
});
```
