## Syntaxe

```js
newrelic.pauseReplay()
```

API Browser utilisée pour forcer une relecture pour mettre en pause l&apos;enregistrement.

## Exigences

* navigateur Pro ou agent Pro+SPA (v1.240.0 ou supérieur)

## Description

`newrelic.pauseReplay()` peut être appelé pour forcer manuellement une relecture à mettre en pause l&apos;enregistrement.

Si une relecture n&apos;est pas déjà en cours, cette méthode API n&apos;aura aucun effet.

Si la session du navigateur est toujours valide, vous pouvez reprendre une relecture interrompue en appelant la méthode API [`newrelic.recordReplay()`](/docs/browser/new-relic-browser/browser-apis/recordReplay).

## Cas d&apos;utilisation

* Vous souhaitez déclencher manuellement une rediffusion pour arrêter l&apos;enregistrement sur certaines pages du site ou dans certaines conditions, comme une page de santé, une page d&apos;informations de carte de crédit, etc.

## Exemples

### Forcer une rediffusion pour mettre en pause l&apos;enregistrement

```js
newrelic.pauseReplay()
```

### Reprise d&apos;une rediffusion en pause

```js
  newrelic.pauseReplay()
  newrelic.recordReplay()
```
