## Syntaxe

```js
newrelic.consent(accept?: boolean)
```

Autorise ou refuse le consentement à l&apos;envoi de données par l&apos;agent du browser sur le réseau.

## Exigences

* Agent Browser Lite, Pro ou Pro+SPA (v1.303.0 ou supérieur)

## Description

L&apos;agent du browser dispose d&apos;une fonctionnalité de mode de consentement optionnelle. Lorsque le mode de consentement est activé via les paramètres de l&apos;application ou avec NerdGraph, l&apos;utilisateur peut appeler la méthode `newrelic.consent(...)` pour autoriser ou refuser le consentement quant à l&apos;envoi ou non de données sur le réseau.

Pour plus d&apos;informations sur la configuration et l&apos;utilisation, consultez notre [documentation sur le mode de consentement](/docs/browser/new-relic-browser/configuration/consent-mode).

## Cas d&apos;utilisation

* Vous souhaitez attendre de récolter des données jusqu&apos;à ce qu&apos;un utilisateur ait donné son consentement avec une fenêtre modale sur votre site

## Exemples

### L&apos;utilisateur consent à ce que l&apos;agent du browser collecte des données

```js
newrelic.consent()
```

### Un utilisateur refuse de consentir à ce que l&apos;agent du browser collecte des données

```js
newrelic.consent(false)
```
