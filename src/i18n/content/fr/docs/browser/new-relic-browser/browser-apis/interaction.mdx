## Syntaxe

```js
newrelic.interaction()
```

Renvoie un nouvel objet API lié à l&apos;interaction SPA actuelle.

## Exigences

* Agent navigateur Pro+SPA (v963 ou supérieur)

* Si vous utilisez npm pour installer l&apos;agent du navigateur, vous devez activer la fonctionnalité `spa` lors de l&amp;apos;instanciation de la classe `BrowserAgent` . Dans l&amp;apos;éventail `features` , ajoutez ce qui suit :

  ```js
  import { Spa } from '@newrelic/browser-agent/features/spa';

  const options = {
    info: { ... },
    loader_config: { ... },
    init: { ... },
    features: [
      Spa
    ]
  }
  ```

  Pour plus d&apos;informations, consultez la [documentation d&apos;installation du navigateur npm](https://www.npmjs.com/package/@newrelic/browser-agent#new-relic-browser-agent).

## Description

L&apos;appel [de monitoring SPA](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring) `interaction()` renvoie un nouvel objet API lié à l&amp;apos;interaction actuelle.

* <DNT>**New interaction:**</DNT> Si l&amp;apos;appel d&amp;apos;API est effectué alors que New Relic ne monitoring pas actuellement une interaction, une nouvelle interaction est créée.
* <DNT>**New object:**</DNT> Si l&amp;apos;appel d&amp;apos;API est à nouveau effectué au sein de la même interaction, un nouvel objet référençant l&amp;apos;interaction courante est créé.

## Paramètres

Les paramètres dépendent de l&apos;interaction SPA spécifique appelée API.

## Valeurs de retour

Cette méthode renvoie un objet API lié à un [événement`BrowserInteraction` ](/docs/insights/explore-data/attributes/browser-default-attributes-insights#browserinteraction-attributes)spécifique. Chaque fois que cette méthode est appelée pour le même `BrowserInteraction`, un nouvel objet est créé, mais il fait toujours référence à la même interaction.

## Exemples

Les méthodes API SPA peuvent être utilisées sur `newrelic.interaction()`. Les méthodes peuvent également être utilisées sur un handle que vous attribuez à une variable. Par exemple:

```js
myInteraction = newrelic.interaction();
myInteraction.save();
```

Le handle nommé peut être enregistré et utilisé en dehors d&apos;une interaction, mais les méthodes n&apos;auront aucun effet une fois l&apos;interaction terminée.
