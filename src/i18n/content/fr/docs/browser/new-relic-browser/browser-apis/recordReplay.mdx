## Syntaxe

```js
newrelic.recordReplay()
```

API Browser utilisée pour forcer une relecture pour démarrer l&apos;enregistrement.

## Exigences

* navigateur Pro ou agent Pro+SPA (v1.240.0 ou supérieur)

## Description

`newrelic.recordReplay()` peut être appelé pour forcer manuellement une relecture pour commencer l&apos;enregistrement. Vous devez répondre aux exigences suivantes :

* Le compte est autorisé à enregistrer des rediffusions
* Le suivi de session est activé
* la trace de session est activée
* navigateur mutation observer global est présent dans la version actuelle du navigateur utilisé
* La fonctionnalité Session Replay est soit normalement importée, soit définie sur `autoStart: false` et a déjà été « démarrée »

Si une relecture est déjà en cours ou si tous les critères ci-dessus ne sont pas remplis, cette méthode API n&apos;aura aucun effet.

Pour plus d&apos;informations sur la mise en pause et la reprise d&apos;une rediffusion, reportez-vous à [`newrelic.pauseReplay()`](/docs/browser/new-relic-browser/browser-apis/pauseReplay)

## Cas d&apos;utilisation

* Vous souhaitez déclencher manuellement une relecture pour enregistrer sur certaines pages du site ou dans certaines conditions, quels que soient les paramètres d&apos;échantillonnage.

## Exemples

### Forcer une rediffusion pour démarrer l&apos;enregistrement

```js
  newrelic.recordReplay()
```
