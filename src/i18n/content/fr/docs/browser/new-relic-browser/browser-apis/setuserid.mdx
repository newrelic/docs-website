## Syntaxe

```js
newrelic.setUserId(value: string|null)
```

Ajoute une chaîne d&apos;identifiant définie par l&apos;utilisateur à l&apos;événement ultérieur sur la page.

## Exigences

* Agent navigateur Lite, Pro ou Pro+SPA (v1.230.0 ou supérieur)

* Si vous utilisez npm pour installer l&apos;agent du navigateur, vous devez activer au moins une fonctionnalité lors de l&apos;instanciation de la classe `BrowserAgent` . Par exemple, ajoutez ce qui suit dans l’éventail`features` :

  ```js
  import { Metrics } from '@newrelic/browser-agent/features/metrics'

  const options = {
    info: { ... },
    loader_config: { ... },
    init: { ... },
    features: [
      Metrics
    ]
  }
  ```

  Pour plus d&apos;informations, consultez la [documentation d&apos;installation du navigateur npm](https://www.npmjs.com/package/@newrelic/browser-agent#new-relic-browser-agent).

## Description

Lors de l&apos;exécution de cette fonction avec une valeur valide, l&apos;agent du navigateur enregistre la valeur comme attribut `enduser.id` avec tous les événements jusqu&amp;apos;à ce que l&amp;apos;attribut soit désactivé manuellement. L&amp;apos;identifiant sera stocké dans le navigateur, afin que les visites ultérieures de pages de la *même* origine l&amp;apos;attachent à l&amp;apos;événement <DNT>**within a session**</DNT> span. Notez que cette fonctionnalité peut fluctuer en fonction des paramètres de confidentialité du navigateur de l&amp;apos;utilisateur final. Si cette fonction est appelée avec un `value = null`, tout ID utilisateur existant sera supprimé de <DNT>**both**</DNT> l&amp;apos;événement de la page courante et du stockage.

L&apos;ID sera attaché à l&apos;événement JavaScriptError notamment pour l&apos;utilisation [Errors Inbox](/docs/errors-inbox/errors-inbox/) . Si vous utilisez [monitoringSPA](/docs/browser/single-page-app-monitoring/get-started/welcome-single-page-app-monitoring) avec une version d&amp;apos;agent compatible, l&amp;apos;ID utilisateur sera également inclus dans l&amp;apos;événement [`newrelic.interaction`](/docs/browser/new-relic-browser/browser-agent-apis/browser-spa-api-newrelicinteraction) .

## Exemples

### Marquer le « début de session » d&apos;un utilisateur final

```js
newrelic.setUserId('user-1234-v1.0')
```

### Arrêt de l&apos;événement de l&apos;attribut à l&apos;utilisateur actuel

```js
newrelic.setUserId(null)
```
