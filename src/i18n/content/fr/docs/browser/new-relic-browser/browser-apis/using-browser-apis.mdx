Les API Browser vous permettent de personnaliser et d&apos;étendre votre <InlinePopover type="browser" />. Pour commencer à utiliser les API, assurez-vous que l&apos;[agent du navigateur est présent sur chaque page que vous souhaitez monitorer](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent).

Vous trouverez ci-dessous des cas d’utilisation pour chaque méthode d’API de navigateur.
