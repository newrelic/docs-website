Si vous devez personnaliser la manière dont l&apos;agent du navigateur collecte les données, vous disposez de deux méthodes : configurer l&apos;UI ou utiliser des API pour personnaliser l&apos;instrumentation de l&apos;agent.

## Configuration UI [#UI]

Sur la page <DNT>**Application settings**</DNT> , vous pouvez effectuer les configurations suivantes pour votre configuration <InlinePopover type="browser" /> :

* [Code de l&apos;agent proxy et données d&apos;analyse](/docs/browser/new-relic-browser/configuration/proxy-agent-requests/)
* [Configurer des alertes](/docs/browser/new-relic-browser/configuration/view-browser-apps-alert-information/)
* [Configurer les paramètres APDEX et géographiques](/docs/browser/new-relic-browser/configuration/browser-settings-ui-options-apdex-geography/#apdex-settings)
* [Filtrer le domaine et le sous-domaine](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains/)
* [Filtrer requestsAJAX](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events/)
* [Regrouper les données par modèles d&apos;URL](/docs/browser/new-relic-browser/configuration/group-browser-metrics-urls/)
* [Masquer les données](/docs/browser/new-relic-browser/configuration/obfuscate-browser-agent-data/)
* [Renommer les applications du navigateur](/docs/browser/new-relic-browser/configuration/rename-browser-apps/)
* [Désactiver monitoring des navigateurs](/docs/browser/new-relic-browser/installation/disable-browser-monitoring/)
* [Supprimer une application de navigateur](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/#remove)

Pour accéder à la page <DNT>**Application settings**</DNT> :

1. Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT> et sélectionnez une application.
2. Dans la navigation de gauche, faites défiler jusqu&apos;à <DNT>**Settings**</DNT> et cliquez sur <DNT>**Application settings**</DNT>.

## API de Browser [#apis]

Vous pouvez également personnaliser instrumentation de votre navigateur d&apos;agent pour capturer des traces et des événements personnalisés, signaler des erreurs spécifiques adaptées à vos besoins et affiner le timing de collecte des données pour des performances optimales. Consultez [Utilisation des API de navigateur](/docs/browser/new-relic-browser/browser-apis/using-browser-apis/) pour commencer.
