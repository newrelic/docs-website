Dans le cadre des[exigences de configuration](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements) <InlinePopover type="browser" />, vous devez [installer l&apos;agent de navigateur](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). Vous pouvez ensuite utiliser la page <DNT>**Applications settings**</DNT> de votre application pour mettre à jour les options monitoring des navigateurs pour :

* navigateur niveau d&apos;abonnement
* [Valeurs Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)du navigateur
* Villes, régions ou pays à mettre en évidence sur la [page<DNT>**Geography**</DNT>](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)

## Sélectionnez les paramètres Apdex [#apdex-settings]

Pour les applications qui sont également monitorées par APM, vous pouvez définir [des valeurs Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) distinctes pour le navigateur (utilisateur final) et pour [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex). Vous pouvez également sélectionner des villes, des régions ou des pays à mettre en évidence sur la page <DNT>**Geography**</DNT> .

1. Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt;**</DNT> (sélectionnez une application) <DNT>**&gt; Settings &gt; Application settings**</DNT>.
2. À partir de <DNT>**Application settings**</DNT>, sélectionnez une valeur Apdex T spécifiquement destinée à monitoring des navigateurs, ou laissez la valeur Apdex par défaut.
3. Sélectionnez <DNT>**Save application settings**</DNT>.

Chaque fois que vous modifiez les paramètres Apdex, une barre verticale noire apparaît sur le graphique <DNT>**Page load timing**</DNT> sur la page <DNT>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> de votre application de navigateur.

## Afficher l&apos;Apdex de l&apos;utilisateur final [#view-browser-apdexs]

Pour consulter votre score Apdex :

* Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT>.

* Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DNT>.
