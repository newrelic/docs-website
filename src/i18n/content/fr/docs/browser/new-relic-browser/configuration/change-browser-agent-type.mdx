Le type d'agent de navigateur n'a aucun impact sur votre facturation. C'est simplement un moyen de passer à un ensemble de fonctionnalités plus minimal, si vous le souhaitez. L'agent de navigateur par défaut installé est livré avec toutes les fonctionnalités <InlinePopover type="browser" />activées.

Pour modifier le type d'édition de votre navigateur, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Browser &amp;gt; (select an app) &amp;gt; Settings &amp;gt; Application settings**</DNT>.

Voici quelques comparaisons entre les différents types d’agents de navigateur :

## Types d'agents de navigateur [#agent-types]

Nous avons trois types d'agents de navigateur : Lite, Pro et Pro+SPA.

<table>
  <thead>
    <tr>
      <th style={{ width: "150px" }}>
        Type d'agent de navigateur
      </th>

      <th>
        Comparaison
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Pro+SPA
      </td>

      <td>
        Il s'agit de l'agent installé par défaut lorsque vous activez monitoring des navigateurs. Il vous donne accès à toutes les fonctionnalités monitoring des navigateurs.

        Pro+SPA n'est pas utile uniquement pour les applications d'une seule page. Après l'installation, vous pouvez rétrograder vers les types d'agent inférieurs si vous ne souhaitez pas ou n'avez pas besoin monitoring SPA.
      </td>
    </tr>

    <tr>
      <td>
        Pro
      </td>

      <td>
        Cette version de l'agent ne dispose pas de la [fonctionnalité conçue pour monitoringSPA](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring).
      </td>
    </tr>

    <tr>
      <td>
        Lite
      </td>

      <td>
        Cette version de l'agent rapporte certaines informations de base sur le temps de chargement des pages et sur l'utilisateur du navigateur, mais manque de nombreuses fonctionnalités monitoring des navigateurs plus avancées et manque également de fonctionnalités monitoring SPA.
      </td>
    </tr>
  </tbody>
</table>
