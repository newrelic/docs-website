Le type d&apos;agent de navigateur n&apos;a aucun impact sur votre facturation. C&apos;est simplement un moyen de passer à un ensemble de fonctionnalités plus minimal, si vous le souhaitez. L&apos;agent de navigateur par défaut installé est livré avec toutes les fonctionnalités <InlinePopover type="browser" />activées.

Pour modifier le type d&apos;édition de votre navigateur, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; Settings &gt; Application settings**</DNT>.

Voici quelques comparaisons entre les différents types d’agents de navigateur :

## Types d&apos;agents de navigateur [#agent-types]

Nous avons trois types d&apos;agents de navigateur : Lite, Pro et Pro+SPA.

<table>
  <thead>
    <tr>
      <th style={{ width: "150px" }}>
        Type d&apos;agent de navigateur
      </th>

      <th>
        Comparaison
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Pro+SPA
      </td>

      <td>
        Il s&apos;agit de l&apos;agent installé par défaut lorsque vous activez monitoring des navigateurs. Il vous donne accès à toutes les fonctionnalités monitoring des navigateurs.

        Pro+SPA n&apos;est pas utile uniquement pour les applications d&apos;une seule page. Après l&apos;installation, vous pouvez rétrograder vers les types d&apos;agent inférieurs si vous ne souhaitez pas ou n&apos;avez pas besoin monitoring SPA.
      </td>
    </tr>

    <tr>
      <td>
        Pro
      </td>

      <td>
        Cette version de l&apos;agent ne dispose pas de la [fonctionnalité conçue pour monitoringSPA](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring).
      </td>
    </tr>

    <tr>
      <td>
        Lite
      </td>

      <td>
        Cette version de l&apos;agent rapporte certaines informations de base sur le temps de chargement des pages et sur l&apos;utilisateur du navigateur, mais manque de nombreuses fonctionnalités monitoring des navigateurs plus avancées et manque également de fonctionnalités monitoring SPA.
      </td>
    </tr>
  </tbody>
</table>
