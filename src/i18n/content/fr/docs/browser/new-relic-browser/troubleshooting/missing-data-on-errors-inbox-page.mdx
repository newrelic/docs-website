## Problème

Je ne vois pas certaines erreurs dans la page <DNT>**Errors Inbox**</DNT> .

## Solution

L&apos;agent du navigateur collecte automatiquement les erreurs <DNT>**unhandled**</DNT> au fur et à mesure qu&amp;apos;elles remontent vers la page. Si vous savez que des erreurs se produisent dans votre page ou application, mais que vous ne les voyez pas dans la page <DNT>**Errors Inbox**</DNT>, il est important de vérifier que les erreurs sont gérées par votre page ou application et ne remontent pas jusqu&amp;apos;à l&amp;apos;agent du navigateur. Dans de tels scénarios, les erreurs traitées peuvent toujours être observées en utilisant l&amp;apos; [API`noticeError` ](/docs/browser/new-relic-browser/browser-apis/noticeerror).
