<Callout variant="course" title="laboratoire">
  Cette procédure fait partie d&apos;un laboratoire qui vous apprend à dépanner votre application Web avec New Relic <InlinePopover type="browser" />.

  Chaque procédure du laboratoire s&apos;appuie sur la précédente, assurez-vous donc d&apos;avoir terminé la dernière procédure, [Configurer votre environnement de laboratoire](/docs/browser/new-relic-browser/lab/set-up-env/), avant de commencer celle-ci.
</Callout>

Votre application React est désormais opérationnelle dans le navigateur. Vous souhaitez vous assurer que vos utilisateurs bénéficient toujours de la meilleure expérience sur votre site. Pour cela, vous avez besoin d&apos;informations détaillées sur l&apos;expérience de votre utilisateur, comme son temps de chargement de page.

Pour atteindre cet objectif, vous devez installer notre agent de navigateur.

## Installer l&apos;agent du navigateur

<Steps>
  <Step>
    Accédez à [New Relic](https://one.newrelic.com/) et connectez-vous avec votre compte.
  </Step>

  <Step>
    Sur le côté droit de la barre de navigation supérieure, cliquez sur <DNT>**Integrations &amp; Agents**</DNT>.

    <img title="Integrations & Agents button in the UI" alt="Screenshot of the Integrations & Agents button in the UI" src="/images/browser-lab-screenshot-add-data.webp" />
  </Step>

  <Step>
    Faites défiler jusqu’à <DNT>**Browser &amp; mobile**</DNT> et sélectionnez <DNT>**Browser monitoring**</DNT>.

    <img title="Arrow pointing to new relic browser" alt="Arrow pointing to new relic browser" src="/images/browser-lab-screenshot-newrelic-browser.webp" />

    L&apos;UI vous guidera tout au long de l&apos;installation de notre agent de navigateur.
  </Step>

  <Step>
    Sélectionnez <DNT>**Copy/paste JavaScript code**</DNT> comme méthode de déploiement.

    <img title="Select deployment method" alt="Select deployment method" src="/images/browser-lab-screenshot-select-deployment-method.webp" />
  </Step>

  <Step>
    Faites défiler jusqu&apos;à <DNT>**Name your app**</DNT>. Nommez votre application <DNT>**Relicstaurants**</DNT> et cliquez sur <DNT>**Enable**</DNT>.

    <img title="Enable browser agent for your app" alt="Enable browser agent for your app" src="/images/browser-lab-screenshot-enable-browser.webp" />

    Ceci vous donne un snippet de code JavaScript pour permettre monitoring des navigateurs. Copiez-le dans votre presse-papiers.

    <img title="javascript code snippet to enable browser agent" alt="javascript code snippet to enable browser agent" src="/images/browser-lab-screenshot-browser-code-snippet.webp" />
  </Step>

  <Step>
    Ouvrez votre application dans l’IDE de votre choix.

    Dans le fichier *public/index.html* de votre application, collez le snippet JavaScript copié à l&amp;apos;intérieur de `<head>`.

    ```html lineHighlight=22-23 fileName=public/index.html
    <!DOCTYPE html>
    <html lang="en">

    <head>
      <meta charset="utf-8" />
      <link rel="icon" href="%PUBLIC_URL%/favicon.ico" />
      <meta name="viewport" content="width=device-width, initial-scale=1" />
      <meta name="theme-color" content="#000000" />
      <meta name="description" content="Web site created using create-react-app" />
      <link rel="preconnect" href="https://fonts.googleapis.com">
      <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
      <link
        href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;1,100;1,300;1,400;1,500&display=swap"
        rel="stylesheet">
      <link rel="apple-touch-icon" href="%PUBLIC_URL%/logo192.png" />
      <link rel="manifest" href="%PUBLIC_URL%/manifest.json" />

      <title>Relicstaurants</title>
        <!--Replace these lines with your 
          browser monitoring code snippet -->
    </head>

    <body>
      <noscript>You need to enable JavaScript to run this app.</noscript>
      <div id="root"></div>
    </body>

    </html>

    ```

    Votre application est désormais instrumentée avec notre agent de navigateur.
  </Step>
</Steps>

## Redémarrez votre application

Maintenant que vous avez instrumenté votre application, il est temps de redémarrer votre serveur local.

```bash
npm run build
npm run newstart
```

Redémarrez également votre générateur de charge.

```bash
python3 simulator.py
```

<Callout variant="important">
  Assurez-vous que vous exécutez ces commandes dans les bonnes fenêtres de terminal. Si vous n&apos;avez plus ces fenêtres, suivez les étapes de la [procédure d&apos;installation](/docs/browser/new-relic-browser/lab/set-up-env).
</Callout>

## Visualisez vos données

Votre application envoie maintenant les données du navigateur à New Relic. Afficher ces données dans New Relic, sous <DNT>**Browser**</DNT>.

<Steps>
  <Step>
    Accédez à [New Relic](https://one.newrelic.com/) et connectez-vous avec votre compte.
  </Step>

  <Step>
    Dans la barre de navigation de gauche, accédez à <DNT>**Browser**</DNT> et sélectionnez <DNT>**Relicstaurants**</DNT>.

    <img title="view relicstraunts" alt="view relicstaruants" src="/images/browser-lab-screenshot-view-relicstaurants.webp" />

    Ici, vous voyez <DNT>**Page views with JavaScript errors**</DNT>, <DNT>**Core web vitals**</DNT>, <DNT>**User time on the site**</DNT>, <DNT>**Initial page load and route changes**</DNT> et d&amp;apos;autres données de performances de votre application.

    <img title="view relicstraunts browser app" alt="view relicstaruants browser app" src="/images/browser-lab-screenshot-relicstaurants-browser-app.webp" />
  </Step>
</Steps>

Vous avez instrumenté votre application pour envoyer des données de navigateur à New Relic à l&apos;aide de notre agent de navigateur. Vous voyez également vos données de performances dans New Relic. Ensuite, vous utilisez ces données pour résoudre les erreurs d’interface de votre site.
