<Callout variant="course" title="laboratoire">
  Cette procédure fait partie d’un laboratoire qui vous apprend à dépanner votre application Web avec New Relic Browser. Si vous ne l’avez pas déjà fait, consultez l’ [introduction du laboratoire](/docs/browser/new-relic-browser/lab/over-view/).
</Callout>

Avant de pouvoir parcourir le laboratoire proprement dit, vous devez lancer votre application React. Ici, vous :

* Lancez votre application React
* Envoyez du trafic vers votre application avec un simple générateur de charge

<Steps>
  <Step>
    Ouvrez une fenêtre de terminal et clonez le référentiel de laboratoire.

    ```bash
    git clone https://github.com/newrelic-experimental/relicstaurants.git
    ```
  </Step>

  <Step>
    Accédez au répertoire racine de votre application et passez au répertoire lab.

    ```bash
    cd relicstaurants
    git switch browser-pro-lab-material
    ```
  </Step>

  <Step>
    Ensuite, installez dépendance et exécutez l’application.

    ```bash
    npm install
    npm run build
    npm run newstart
    ```

    Cela ouvre votre application Relicstaurants dans votre navigateur.

    <img title="Relicstraunts homepage" alt="Relicstraunts homepage" src="/images/browser-lab-screenshot-homepage.webp" />

    Saisissez votre adresse de livraison et recherchez des restaurants pour commencer.

    <img title="Nearby restaurant list" alt="Nearby restaurant list" src="/images/browser-lab-screenshot-restaurant-list.webp" />

    Ici, vous voyez une liste de restaurants où vous pouvez commander de la nourriture.
  </Step>

  <Step>
    Choisissez un restaurant.

    <img title="Choose a restaurant" alt="Choose a restaurant" src="/images/browser-lab-screenshot-choose-restaurant.webp" />
  </Step>

  <Step>
    Sélectionnez un ou deux articles et cliquez sur le panier.

    <img title="select food" alt="select food" src="/images/browser-lab-screenshot-select-food.webp" />
  </Step>

  <Step>
    Cliquez sur <DNT>**PAY**</DNT>.

    <img title="checkout" alt="checkout" src="/images/browser-lab-screenshot-checkout.webp" />
  </Step>

  <Step>
    Saisissez les informations de fausse carte suivantes et cliquez sur <DNT>**Finish payment**</DNT> pour passer votre commande.

    <img title="place order with fake card" alt="place order with fake card" src="/images/browser-lab-screenshot-place-order.webp" />

    Votre commande a été passée avec succès.

    <img title="Purchase completed" alt="Purchase completed" src="/images/browser-lab-screenshot-thank-you.webp" />

    Ensuite, vous utilisez un simulateur pour générer plus de trafic vers votre application.
  </Step>

  <Step>
    Dans une autre fenêtre de terminal, accédez au répertoire racine de votre application et exécutez le générateur de charge.

    ```bash
    # Navigate to the root directiory of your simulator
    cd relicstaurants/simulator
    # Switch to lab branch
    git switch browser-pro-lab-material
    [output]
    # Install the simulator's dependencies
    pip3 install -r requirements.txt
    [output]
    # Run the simulator
    python3 simulator.py
    [output] ====== WebDriver manager ======
    [output] Current google-chrome version is 99.0.4844
    [output] Get LATEST chromedriver version for 99.0.4844 google-chrome
    ```

    <Callout variant="important">
      Ce générateur de charge suppose que Google Chrome est installé sur votre ordinateur. Si vous utilisez un autre navigateur, ignorez cette étape et générez le trafic manuellement ou [installez Google Chrome](https://www.google.com/chrome/downloads/).
    </Callout>
  </Step>
</Steps>

Maintenant que vous savez comment exécuter votre application, il est temps de l&apos;instrumenter. Dans les fenêtres du terminal qui exécutent votre application et votre simulateur, appuyez sur `<CTRL-C>` pour les arrêter. Une fois votre application fermée, vous pourrez mettre à jour votre code pour introduire des outils monitoring .
