Avec l&apos;aide du dashboard Dojo de New Relic, vous pouvez suivre les principaux éléments essentiels du Web de votre application, le temps que vos utilisateurs passent sur votre site et les mesures critiques des performances du navigateur, telles que le chargement initial de la page et les pages contenant des erreurs Javascript. Conçu avec notre <InlinePopover type="browser" />agent, le dashboard Dojo vous permet de visualiser toutes vos données les plus importantes en un seul endroit.

## Installer l&apos;agent d&apos;infrastructure (facultatif) [#infra-install]

Bien que notre intégration Dojo ne soit pas construite avec notre agent d&apos;infrastructure, nous vous recommandons de l&apos;installer si vous souhaitez obtenir des données de performances sur vos hôtes. Vous pouvez installer l&apos;agent d&apos;infrastructure de deux manières différentes :

* Notre [guide d&apos;installation](https://one.newrelic.com/nr1-core?state=4f81feab-35f7-e97e-9903-52510f8542bd) est un outil CLI qui inspecte votre système et installe l&apos;agent d&apos;infrastructure aux côtés de l&apos;agent de monitoring d&apos;application qui fonctionne le mieux pour votre système. Pour en savoir plus sur le fonctionnement de notre guide d&apos;installation, consultez notre [aperçu du guide d&apos;installation](/docs/infrastructure/host-integrations/installation/new-relic-guided-install-overview).
* Si vous préférez installer notre agent d&apos;infrastructure manuellement, vous pouvez suivre un tutoriel d&apos;installation manuelle pour [Linux](/docs/infrastructure/install-infrastructure-agent/linux-installation/install-infrastructure-monitoring-agent-linux), [Windows](/docs/infrastructure/install-infrastructure-agent/windows-installation/install-infrastructure-monitoring-agent-windows/) ou [macOS](/docs/infrastructure/install-infrastructure-agent/macos-installation/install-infrastructure-monitoring-agent-macos/).

## Installez l’agent de navigateurs [#browser-install]

1. Installez notre agent de navigateur en accédant à notre [lanceur d’installation](https://one.newrelic.com/marketplace?account=3510613&state=79352429-fa28-4c25-1123-30c4b23ff83c) et en cliquant sur <DNT>**Browser monitoring**</DNT>.
2. Sélectionnez votre compte et suivez les instructions dans l&apos;UI.
3. Installez le dashboard Dojo pré-construit sur la [page d&apos;observation instantanée de Dojo](https://newrelic.com/instant-observability/dojo)

## Quelle est la prochaine étape ? [#whats-next]

Avec l&apos;agent navigateur installé et instrumenté avec votre application, vous pouvez alors visualiser vos données brutes dans [Métriques &amp; événement](/docs/query-your-data/explore-query-data/browse-data/introduction-data-explorer/). La valeur par défaut <InlinePopover type="dashboards" />transforme les données brutes en graphiques et diagrammes qui fournissent une vue d&apos;ensemble de l&apos;état de santé de votre système. Pour installer notre dashboard par défaut, rendez-vous sur notre [page d&apos;observabilité instantanée](https://newrelic.com/instant-observability/)
