## Installer l&apos; agent d&apos;infrastructure [#infra]

Vous n&apos;avez pas besoin de l&apos;agent d&apos;infrastructure pour intégrer Blazor WebAssembly à New Relic, mais nous recommandons son installation si vous souhaitez une observabilité complète de votre système. L&apos;agent infrastructure collecte les logs et autres données sur vos hôtes, affichant vos mesures de performances aux côtés de vos mesures d&apos;application.

Il existe deux méthodes pour installer l&apos;agent d&apos;infrastructure :

* Suivez notre [guide d&apos;installation](https://one.newrelic.com/launcher/catalog-pack-details.launcher/?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5saXN0aW5nIn0=&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyJ9) pour instrumenter votre système avec l&apos;agent infrastructure .

## Intégrer Blazor WebAssembly avec New Relic [#integrate]

Pour collecter des données à partir de Blazor WebAssembly, suivez les procédures ci-dessous :

1. De [one.newrelic.com](https://one.newrelic.com), aller à la [page<DNT>**Integrations &amp; Agents**</DNT> ](https://one.newrelic.com/marketplace).

2. Cliquez sur <DNT>**Browser and mobile**</DNT>, puis sur <DNT>**Browser monitoring**</DNT>.

3. Sélectionnez l’option <DNT>**Select copy/paste JavaScript code**</DNT> (pour en savoir plus sur ce code, consultez [notre documentation d’installation du navigateur](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent)).

4. Complétez le reste des étapes et cliquez sur <DNT>**Enable.**</DNT>

5. Accédez au répertoire `wwwroot` de votre application Blazor WebAssembly.

6. Ouvrez le fichier `index.html` .

7. Collez notre snippet de code de navigateur juste avant la fin de la tag body (`</body>`).

8. Redémarrez votre application.

9. Allez à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>, puis cliquez sur <DNT>**Metrics &amp; events**</DNT>. Exécutez un exemple de requête NRQL pour confirmer que New Relic collecte des données :

   ```sql
   SELECT average(windowLoad) FROM PageViewTiming WHERE (deviceType = 'Desktop') FACET `userAgentName` SINCE 604800 seconds AGO TIMESERIES
   ```

10. Installez le dashboard par défaut en accédant à la [page d&apos;observation instantanée de Blazor WebAssembly](https://newrelic.com/instant-observability/blazor-web-assembly), puis en cliquant sur <DNT>**Install**</DNT> en haut de la page.

11. Accédez à <DNT>**Dashboards**</DNT> depuis <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT> et cliquez sur Blazor WebAssembly pour afficher votre dashboard.
