Notre intégration Flutter monitore les performances de votre application Web Flutter , vous aidant à diagnostiquer les problèmes de votre application et à optimiser votre code. Notre intégration Flutter utilise notre <InlinePopover type="browser" />agent et vous offre un dashboard prédéfini avec les mesures les plus importantes de votre application Flutter (comme les erreurs, les plantages d&apos;application, le lancement d&apos;applications, les installations d&apos;applications, les erreurs HTTP et les pannes de réseau).

## Installation [#install]

### Étape 1 (facultative) : installer l&apos;agent d&apos;infrastructure [#infra-install]

L&apos;installation de l&apos;agent infrastructure est facultative, mais cela vous permettra d&apos;obtenir des mesures importantes sur l&apos;hôte sur lequel votre application s&apos;exécute.

Pour ce faire, suivez [les étapes d’installation de l’agent d’infrastructure](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent) pour l’hôte contenant votre application Flutter.

### Étape 2 : installer l’agent du navigateur [#browser-install]

Notre intégration Flutter s&apos;appuie sur notre agent de navigateur pour fonctionner. Pour installer notre agent de navigateur :

1. De [one.newrelic.com](https://one.newrelic.com), aller à la [page<DNT>**Integrations &amp; Agents**</DNT> ](https://one.newrelic.com/marketplace).
2. Cliquez sur <DNT>**Browser and mobile**</DNT>, puis sur <DNT>**Browser monitoring**</DNT>.
3. Sélectionnez l’option <DNT>**Select copy/paste JavaScript code**</DNT> (pour en savoir plus sur ce code, consultez [notre documentation d’installation du navigateur](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent)).
4. Complétez le reste des étapes et cliquez sur <DNT>**Enable.**</DNT>
5. Accédez au répertoire `www/root` de votre application Web Flutter.
6. Ouvrez le fichier `index.html` .
7. Collez notre snippet de code de navigateur juste avant la fin de la tag head (`</head>`).
8. Redémarrez votre application.

Attendez cinq minutes, puis procédez à la recherche de vos données dans New Relic.

## Trouvez vos données [#find-data]

Pour obtenir votre Flutter dashboard:

1. De [one.newrelic.com](https://one.newrelic.com), aller à la [page<DNT>**Integrations &amp; Agents**</DNT> ](https://one.newrelic.com/marketplace).
2. Cliquez sur <DNT>**Dashboards**</DNT>.
3. Dans la barre de recherche, tapez `flutter web`.
4. Le Flutter dashboard devrait apparaître. Cliquez dessus pour l&apos;installer.

Votre Flutter dashboard est considéré comme un dashboard personnalisé et peut être trouvé dans <DNT>**Dashboards**</DNT> UI l&apos;. Pour obtenir des documents sur l&apos;utilisation et la modification du dashboard, consultez [notre documentation dashboard ](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).

Cette intégration rapporte les données sous la forme de notre [monitoring de navigateurs événement](/docs/data-apis/understand-data/event-data/events-reported-browser-monitoring). Si vous avez installé l&apos;agent d&apos;infrastructure, vous recevrez également [des données d&apos;infrastructure](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-monitoring-data/#infrastructure-events).

Voici un exemple de requête NRQL vérifiant le temps de chargement moyen des fenêtres pour les différents navigateurs utilisés par l&apos;utilisateur sur les ordinateurs de bureau :

```sql
SELECT average(windowLoad) FROM PageViewTiming 
WHERE (deviceType = 'Desktop') FACET `userAgentName` 
SINCE 604800 seconds AGO TIMESERIES
```
