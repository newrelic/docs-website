## Détecter automatiquement le log du navigateur [#automatic-log]

Le logging automatique collecte les messages de log émis par le navigateur `console` pour vous aider à maximiser l&apos;observabilité de vos applications frontales.

Les logs du navigateur sont suivis par défaut au niveau `WARN` pour les agents <DNT>Pro</DNT> et <DNT>Pro+SPA</DNT>, mais ne sont pas disponibles pour l&apos;agent Lite browser. Nous vous recommandons de confirmer d&apos;abord que vous utilisez l&apos;agent browser <DNT>**Pro**</DNT> ou <DNT>**Pro + SPA**</DNT>.

<Callout variant="important">
  Certains paramètres, tels que l&apos;activation ou la désactivation de la collecte des logs du navigateur et l&apos;ajustement des taux d&apos;échantillonnage ou de la verbosité des logs, sont des configurations côté serveur. Une fois que vous avez ajouté l&apos;extrait de l&apos;agent à vos pages, vous n&apos;avez pas besoin de redéployer le script pour modifier ces paramètres. Cependant, les configurations côté serveur n&apos;ont aucun effet si les fonctionnalités ne sont pas incluses dans le script déployé. Ceci s&apos;applique à la version `1.259.0` ou ultérieure de l&apos;agent browser.
</Callout>

## Activer la collecte automatique des logs (Pour les applications de navigateur existantes) [#enable-automatic-log]

Améliorez l&apos;observabilité et les capacités de débogage de votre application en suivant ces instructions de configuration simples.

<Steps>
  <Step>
    Accédez à **[one.newrelic.com](https://one.newrelic.com) &gt; All Capabilities &gt; Browser**.
  </Step>

  <Step>
    Sélectionnez votre application de navigateur.
  </Step>

  <Step>
    Dans le menu de gauche, cliquez sur **Paramètres de l’application**.
  </Step>

  <Step>
    Assurez-vous que l&apos;agent browser <DNT>**Pro**</DNT> ou <DNT>**Pro + SPA**</DNT> est sélectionné.
  </Step>

  <Step>
    Basculez **Automatically collect console logs** sur **ON** pour collecter les messages envoyés à la console du navigateur. <br />Basculez **Collect logs manually** sur **ON** pour collecter les logs envoyés par programme via `newrelic.log()` ou `newrelic.wrapLogger()`.
  </Step>

  <Step>
    Modifiez le taux d&apos;échantillonnage pour spécifier le pourcentage de sessions utilisateur pour collecter les logs. Vous pouvez saisir une valeur comprise entre 0 et 100.

    <Callout variant="important">
      Le taux d&apos;échantillonnage par défaut pour le logging du navigateur est défini sur 100 % des sessions utilisateur. Vous pouvez définir ce taux d&apos;échantillonnage de session entre `0%` et `100%`. Par exemple, définir le taux d&apos;échantillonnage de session sur `50%` entraînera la collecte d&apos;événements de log à partir d&apos;environ la moitié des sessions utilisateur sélectionnées aléatoirement.
    </Callout>
  </Step>

  <Step>
    Sélectionnez le niveau de verbosité des logs du navigateur souhaité. Le niveau de verbosité par défaut est défini sur **WARN**. Les niveaux parmi lesquels vous pouvez choisir :

    * `ERROR` - collecter les logs avec le niveau `ERROR`
    * `WARN` - collecter les logs avec le niveau `WARN` ou `ERROR`
    * `INFO` - collecter les logs avec le niveau `INFO`, `WARN` ou `ERROR`
    * `DEBUG` - collecter les logs avec le niveau `DEBUG`, `INFO`, `WARN` ou `ERROR`
    * `TRACE` - collecter les logs avec le niveau `TRACE`, `DEBUG`, `INFO`, `WARN` ou `ERROR`

    <br />

    <Callout variant="important">
      Les données transmises via les méthodes de la console peuvent passer par la sérialisation et [obfuscation](/docs/browser/new-relic-browser/configuration/obfuscate-browser-agent-data). Selon la taille et la fréquence, cela peut avoir un impact négatif sur les performances des applications ainsi que sur les coûts des données. En général, il n&apos;est PAS recommandé de transmettre de gros objets ou de grandes quantités de données dans les méthodes de la console.
    </Callout>
  </Step>
</Steps>

## Désactiver le log du navigateur [#disable-browser-logs]

Pour désactiver le log du navigateur pour une application de navigateur existante :

<Steps>
  <Step>
    Accédez à **[one.newrelic.com](https://one.newrelic.com) &gt; All Capabilities &gt; Browser**.
  </Step>

  <Step>
    Sélectionnez votre application de navigateur.
  </Step>

  <Step>
    Dans le menu de gauche, cliquez sur **Paramètres de l’application**.
  </Step>

  <Step>
    Basculez **Automatically collect console logs** et **Collect logs manually** sur **OFF**.
  </Step>
</Steps>

## Consommation de données

Les logs suivent les mêmes tarifs de consommation que les autres octets de votre navigateur. La quantité d&apos;octets produits dépend du nombre et de la longueur des messages.

La fonctionnalité de logging automatique élimine le besoin d&apos;appeler les API de navigateur `newrelic.log` ou `newrelic.wrapLogger`, sauf lors de l&apos;ajout d&apos;un attribut personnalisé à l&apos;événement de logging.
