Certaines fonctionnalités de New Relic nécessitent votre clé de licence monitoring des navigateurs ou votre identifiant d&apos;application New Relic. Le collecteur New Relic lit cette clé pour déterminer à quel compte appartiennent les données entrantes.

Si vous utilisez la [méthode copier/coller](/docs/browser/new-relic-browser/installation-configuration/adding-apps-new-relic-browser#copy-paste-app) pour ajouter une application à <InlinePopover type="browser" />, [la page <DNT>**Settings**</DNT> du navigateur](/docs/browser/new-relic-browser/installation-configuration/browser-settings-ui-options-browser-monitoring) de votre application affichera le snippet JavaScript généré. Le bas du JavaScript généré inclut votre navigateur `licenseKey` (également appelé `browser_monitoring_key`) et `applicationID`.

* La monitoring des navigateurs `licenseKey` (`browser_monitoring_key`) est une clé unique qui est liée à, mais n&apos;est pas identique à, votre <InlinePopover type="licenseKey" />principal. La clé du navigateur ne peut pas être utilisée pour déterminer la clé de licence de votre compte.
* Le navigateur `applicationID` est le code d&apos;identification unique de chaque application dans monitoring des navigateurs.

Pour protéger la sécurité de vos `browser_monitoring_key` et `applicationID`, vous pouvez [monitorer ou bloquer votre domaine](/docs/browser/new-relic-browser/installation-configuration/monitor-or-block-specific-domains).

## Récupérer la clé de licence du navigateur et l&apos;ID de l&apos;application à partir de l&apos;UI [#procedures]

Cette procédure s&apos;applique <DNT>**only**</DNT> lorsque vous utilisez la méthode de déploiement copier/coller. Si vous utilisez la [méthode de déploiement de l&apos;application APM](/docs/browser/new-relic-browser/installation-configuration/adding-apps-new-relic-browser), votre page <DNT>**Settings**</DNT> n&apos;affichera pas le snippet JavaScript.

Pour obtenir uniquement les informations monitoring des navigateurs, clé de licence et identifiant d&apos;application :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; Settings**</DNT>.
2. Sélectionnez et copiez le JavaScript généré et collez-le dans un fichier texte.
3. Supprimez tout dans le fichier texte, à l&apos;exception des informations monitoring des navigateurs <InlinePopover type="licenseKey" />et d&apos;ID d&apos;application (juste au-dessus de la tag de fin `</script>`), puis enregistrez.
