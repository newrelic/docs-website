Votre utilisation de Pipeline Control est facturable conformément à votre Bon de commande. Pipeline Control est vendu dans le cadre du SKU Advanced Compute, un module complémentaire qui s&apos;ajoute aux modèles de tarification de base (tarification par utilisateur ou Core Compute). Advanced Compute regroupe des fonctionnalités avancées et les facture via des aCCU (Advanced Compute Consumption Units).

## Comment les coûts sont calculés

Les coûts associés à Pipeline Control sont déterminés par les facteurs suivants :

### Mesure du gateway

Gateway s&apos;exécute sur votre infrastructure et le comptage est simple :

**Ce qui est mesuré**:

* **Volume de données reçu** aux points de terminaison du gateway (mesuré en octets non compressés)
* Facturé **indépendamment du traitement appliqué** ou si les données sont rejetées
* Comptabilisé **à l&apos;entrée** avant tout filtrage/rejet

### Mesure des règles Cloud

Les règles cloud s&apos;exécutent sur l&apos;infrastructure de New Relic et sont donc facturées à un tarif supérieur à celui du gateway.

**Ce qui est comptabilisé** (pour les règles de suppression) :

* **Go rejetés**: Volume de données supprimé par vos règles
* **BSE (milliards d&apos;événements scannés)**: Événements évalués lors de l&apos;exécution des règles
* Les deux sont des postes distincts qui entraînent chacun des frais
* **Les BSE représentent généralement une fraction beaucoup plus faible** des frais globaux liés aux règles cloud par rapport aux Go rejetés pour la grande majorité des clients

**Comptage BSE**:

* Analyse uniquement les données correspondant au `SELECT` de niveau racine dans la règle NRQL
* Exemple : `DELETE FROM Log WHERE level = 'INFO'` ne comptabilise les BSE que pour les données dans `Log`
* Des requêtes plus sélectives analysent moins d&apos;événements

### Autres coûts de contrôle des pipelines

**Core Compute**: lors de la consultation de certaines pages de l&apos;interface de contrôle du pipeline qui génèrent des requêtes NRDB, mesuré en Core CCU.

**Ingestion de données**: toutes les données non abandonnées par le Contrôle de pipeline et finalement stockées dans NRDB, mesurées en Go ingérés.

Pour les clients Compute à la demande, les produits Core Compute et Advanced Compute sont mis à disposition sous la forme du produit Compute, mesuré en CCU.

Pour obtenir des détails précis sur la tarification et comprendre quel modèle est le plus rentable pour votre cas d&apos;utilisation, veuillez contacter notre équipe commerciale ou vous référer à votre Commande.
