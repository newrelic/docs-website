Votre utilisation de Contrôle de pipeline est facturable conformément à votre Commande. Les coûts associés à cette fonctionnalité sont déterminés par les facteurs suivants, selon le modèle de tarification associé à votre compte :

**Advanced Compute** (mesuré en CCU avancé) :

* Go non compressés reçus par le gateway (suite à la configuration des agents ou des frais de télémétrie à envoyer via le gateway), indépendamment du fait que les règles du gateway ont un effet sur les données
* Les Go supprimés par les règles du cloud
* événements (y compris métriques, événement, journal et trace) analysés par les règles cloud

**Core Compute**: lors de la visualisation de certaines pages dans l&apos;interface utilisateur de Contrôle de pipeline qui génèrent une requête NRDB, mesurée dans Core CCU.

**Ingestion de données**: toutes les données non abandonnées par le Contrôle de pipeline et finalement stockées dans NRDB, mesurées en Go ingérés.

Pour les clients Compute à la demande, les produits Core Compute et Advanced Compute sont mis à disposition sous la forme du produit Compute, mesuré en CCU.

Pour plus de détails sur les prix, veuillez contacter notre équipe commerciale ou vous référer à votre commande.
