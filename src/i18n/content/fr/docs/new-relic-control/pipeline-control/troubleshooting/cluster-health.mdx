## Problème : Épuisement des ressources

Le cluster a épuisé ses ressources CPU ou mémoire.

## Solution

* Utilisez l’interface utilisateur Kubernetes pour afficher les événements pod et les tâches en attente afin d’identifier les contraintes de ressources.
* Augmentez la taille du pool de nœuds ou ajustez les limites de ressources (CPU et mémoire) requests pour le pod.
* Vérifiez les limites du fournisseur cloud pour le nombre de nœuds actifs et ajustez la configuration selon les besoins.
