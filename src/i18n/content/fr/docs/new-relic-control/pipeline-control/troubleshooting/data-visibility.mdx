/\* Si vous ne voyez aucune donnée MELT pour le gateway dans l&apos;interface utilisateur de Contrôle de pipeline, ce guide vous aidera à diagnostiquer et à résoudre les problèmes courants. Suivez les étapes ci-dessous en fonction du problème spécifique que vous rencontrez : \*/

## Problème : problèmes liés au menu déroulant de compte

Vous n&apos;avez pas sélectionné **All accounts** ou le compte associé à la clé de licence de gateway dans l&apos;interface utilisateur Contrôle de pipeline.

## Solution

* Vérifiez le menu déroulant de compte dans l&apos;interface utilisateur de Contrôle de pipeline pour vous assurer que le bon compte est sélectionné.
* Assurez-vous de consulter **All accounts** ou le compte spécifique associé à la clé de licence de gateway pour accéder aux données MELT pertinentes.
