/\* Si vous recevez toutes les données dans New Relic mais qu&apos;elles arrivent au-delà du contrat de niveau de service (SLA) attendu, ce guide vous aidera à diagnostiquer et à résoudre le problème. Suivez les étapes ci-dessous en fonction du problème spécifique que vous rencontrez : \*/

## Problème : données arrivant au-delà du SLA

Les données ne sont pas reçues dans le délai prévu.

## Solution

* Vérifiez les mesures de latence pour identifier les retards dans la transmission des données.
* Augmentez le nombre minimum de pods de gateway pour améliorer la capacité de traitement et réduire la latence.
