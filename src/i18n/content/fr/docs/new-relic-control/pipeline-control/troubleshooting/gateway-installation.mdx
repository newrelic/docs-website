/\* Si vous essayez d'installer le gateway pour la première fois et rencontrez des problèmes, ce guide vous aidera à diagnostiquer et à résoudre les problèmes courants. Suivez les étapes ci-dessous en fonction du problème spécifique que vous rencontrez : \*/

## Problème : capacités utilisateur insuffisantes

Vous ne disposez pas des autorisations nécessaires associées à l'administrateur de produit d'organisation et au gestionnaire d'organisation.

## Solution

* L'utilisateur verra un message d'erreur indiquant : « Vous ne disposez pas des capacités requises au niveau de l'organisation pour configurer l'authentification de l'agent. »
* Demandez aux administrateurs de votre compte de se voir attribuer un rôle incluant les fonctionnalités nécessaires à la création d'identité système.
* Reportez-vous à [la documentation sur les autorisations utilisateur](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/) pour obtenir des conseils.

## Problème : version obsolète du graphique Helm

Le script de ligne de commande pour installer le graphique Helm échoue car votre version locale de Helm est obsolète.

## Solution

* Mettez à jour votre installation Helm locale vers la dernière version pour garantir la compatibilité avec le script d'installation.
* Recherchez le message d'erreur indiquant la nécessité d'une mise à jour Helm et suivez les instructions pour la mise à niveau.
