Il s’agit d’un glossaire des termes qui apparaissent dans les contrats relatifs à notre [tarification originale basée sur les produits](/docs/accounts/original-accounts-billing/product-pricing/product-based-pricing/). Pour connaître les conditions de notre modèle de tarification basé sur l&apos;utilisation, consultez [Définitions de la tarification basée sur l&apos;utilisation](/docs/licenses/license-information/product-definitions/new-relic-one-pricing-definitions).

Les services New Relic capitalisés qui apparaissent dans les contrats pour notre tarification basée sur les produits d&apos;origine sont tels que décrits dans ce site de documentation pour ce service.

<CollapserGroup>
  <Collapser id="app" title="Application">
    Une **application** désigne le logiciel d&apos;application conçu pour fonctionner sur les smartphones, les tablettes et autres appareils mobiles. Chaque application comprend un nombre prédéfini d&apos;utilisateurs.
  </Collapser>

  <Collapser id="app-transaction" title="Transaction d'application">
    Une **transaction d&apos;application** est la tentative d&apos;une application APM de traiter une demande Web ou non Web. Dans New Relic APM, celles-ci se manifestent sous la forme `throughput TIMESERIES` ou individuellement sous la forme d&apos;événements dans le type d&apos;événement `Transaction`.
  </Collapser>

  <Collapser id="lambda-event" title="Événement AWS Lambda">
    Un **événement AWS Lambda** désigne la ligne de données collectée à partir de la fonction AWS Lambda du client par l&apos;agent New Relic ou envoyée depuis un service externe vers la plateforme New Relic. Il se compose des types d&apos;événements `AwsLambdaInvocation`, `AwsLambdaInvocationError` ou personnalisés.
  </Collapser>

  <Collapser id="check" title="Vérifier">
    Une **vérification** désigne l&apos;instance unique d&apos;un moniteur synthétique exécuté dans le réseau de monitoring de New Relic et signalant le temps de réponse et si la vérification a réussi ou échoué.
  </Collapser>

  <Collapser id="compute-unit" title="unité de calcul">
    Une **unité de calcul** désigne la mesure des ressources associées à une unité de calcul sur un hôte physique ou virtuel.
  </Collapser>

  <Collapser id="dpm" title="Points de données par minute">
    **Les points de données par minute (DPM)** font référence au débit par minute auquel les valeurs métriques individuelles sont envoyées à l&apos;API New Relic Metric Ingest. À des fins de facturation, les points de données par minute sont calculés comme une valeur moyenne mensuelle en additionnant les points de données ingérés au cours d&apos;une période de 30 jours et en divisant par le nombre de minutes de cette période (43 200).
  </Collapser>

  <Collapser id="event" title="Événement">
    Un **événement** désigne la ligne de données collectée à partir de l&apos;application du client par l&apos;agent New Relic ou envoyée depuis un service externe dans notre base de données.
  </Collapser>

  <Collapser id="host" title="Hôte">
    Un **hôte** désigne l&apos;ordinateur physique ou l&apos;instance de machine virtuelle exécutant une seule copie d&apos;un système d&apos;exploitation. L&apos;utilisation de l&apos;hôte est suivie mensuellement en additionnant les heures pendant lesquelles chaque hôte du compte est connecté à New Relic et en divisant par 750. Un hôte est compté s&apos;il est connecté à un moment au cours d&apos;une heure.
  </Collapser>

  <Collapser id="incident-event" title="Événement incident">
    Un **événement incident** est un événement d&apos;alerte (ouvert, fermé, etc.) créé par un moteur d&apos;alerte qui est envoyé à la plateforme d&apos;alertes New Relic (non unique) pour la déduplication, la détection de battement, la suppression intelligente, l&apos;enrichissement et la corrélation.
  </Collapser>

  <Collapser id="page-view" title="Affichage de la page">
    Une **vue de page** signifie le chargement complet de la page (déclenchant un événement `onLoad` ) ou un changement d&apos;URL enregistré (changement d&apos;état).
  </Collapser>

  <Collapser id="gb-daily" title="Par Go par jour">
    **Par Go par jour** représente une moyenne quotidienne des données de log envoyées à New Relic sur une période de 30 jours.
  </Collapser>

  <Collapser id="span" title="Span">
    Une **étendue** représente un résumé d&apos;opération collecté à partir de l&apos;application du client via l&apos;agent New Relic APM ou l&apos;agent New Relic Serverless pour AWS Lambda, ou envoyé à partir d&apos;autres outils de traçage vers l&apos;API New Relic Trace.
  </Collapser>

  <Collapser id="user" title="Utilisateur">
    Un **utilisateur** désigne la personne qui se connecte à votre application à partir d&apos;un seul appareil. Chaque appareil unique est considéré comme un utilisateur unique.
  </Collapser>
</CollapserGroup>
