Nous aimons les logiciels open source et utilisons les éléments suivants dans les plugins de transfert de logs qui peuvent être utilisés avec nos [fonctionnalités<InlinePopover type="logs" />dans New Relic](/docs/logs/get-started/get-started-log-management/). Merci à la communauté open source d&apos;avoir créé ces excellents outils ! Certains d&apos;entre eux sont répertoriés sous plusieurs licences logicielles, et dans ce cas, nous avons répertorié la licence que nous avons choisi d&apos;utiliser.

Pour obtenir la liste des licences utilisées pour la gestion des logs dans New Relic, consultez [Licences de logs](/docs/licenses/product-or-service-licenses/new-relic-logs/logs-licenses/).
