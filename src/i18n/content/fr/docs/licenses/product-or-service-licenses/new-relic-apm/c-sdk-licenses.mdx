<Callout variant="important" title="Avis de fin de vie">
  À partir d&apos;avril 2022, nous ne prenons plus en charge la fonctionnalité C SDK. Pour plus de détails, consultez notre [publication sur le forum d&apos;assistance](https://discuss.newrelic.com/t/q1-bulk-eol-announcement-fy23/181744).
</Callout>

De plus, le SDK C utilise le code source de bibliothèques tierces, qui sont sous licence selon leurs propres conditions. Ce document répertorie ces autres bibliothèques tierces et leurs licences.

<Callout variant="important">
  New Relic a contribué au SDK C à la communauté open source sous une [licence Apache 2.0](https://github.com/newrelic/c-sdk/blob/master/LICENSE).
</Callout>
