Nous aimons les logiciels open source, et Auto-telemetry with Pixie est l&apos;intégration de New Relic avec [Community Cloud for Pixie](https://pixielabs.ai/), une implémentation New Relic du logiciel open source magique Pixie de la Cloud Native Computing Foundation. Merci à la communauté open source d&apos;avoir créé ces excellents outils !
