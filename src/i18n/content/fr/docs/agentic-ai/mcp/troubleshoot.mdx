## Le flux OAuth ne démarre pas

* Assurez-vous que votre navigateur peut accéder à `localhost`.
* Vérifiez les paramètres du pare-feu.
* Vérifiez que vos outils peuvent accéder aux URL `OAuth`.

## La clé API ne fonctionne pas

* Vérifiez le format de la clé (commence par `NRAK-`).
* Vérifiez les autorisations clés dans New Relic.
* Vérifiez que la clé n&apos;a pas expiré.

## Serveur MCP non détecté

* Redémarrez complètement l&apos;application.
* Vérifier la validité de la syntaxe `JSON`.
* Vérifiez les chemins d&apos;accès aux fichiers et les autorisations.

## Problèmes de proxy mcp-remote

* Assurez-vous que `Node.js` est installé.
* Vérifiez votre connexion internet.
* Vérifiez que npx peut accéder au package `mcp-remote`.

## Messages d’erreur

* **Connexion refusée**: Vérifiez l&apos;URL et la connectivité réseau de vos outils.
* **Authentification requise**: Vérifiez que la clé API est toujours active et qu&apos;elle a été saisie correctement ou terminez le flux `OAuth`.
* **Client invalide**: problème de configuration `OAuth`, essayez le proxy `mcp-remote`.
* **`Time.UnmarshalJSON`**: Utiliser `mcp-remote` comme proxy pour `OAuth`.
