New Relic AI Knowledge prend en charge l&apos;intégration en 1 clic avec des sources de données telles que Confluence.

## Configurations des connecteurs

Pour commencer à indexer votre contenu Confluence et bénéficier du connecteur de connaissances avec New Relic AI, suivez les étapes mentionnées :

<Steps>
  <Step>
    Accédez à <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **Integrations &amp; Agents**</DNT>.
  </Step>

  <Step>
    Rechercher <DNT>**NRAI Knowledge Connectors**</DNT>.
  </Step>

  <Step>
    Sélectionnez <DNT>**Confluence NRAI Knowledge Connector**</DNT>.
  </Step>

  <Step>
    Saisissez les détails du connecteur tels que :

    <table>
      <thead>
        <tr>
          <th>
            Nom du champ
          </th>

          <th>
            Description
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            Connector name
          </td>

          <td>
            Saisissez un nom unique pour votre connecteur, par exemple <DNT>**Demo connector**</DNT>.
          </td>
        </tr>

        <tr>
          <td>
            Connector description
          </td>

          <td>
            Saisissez une description pour votre connecteur.
          </td>
        </tr>

        <tr>
          <td>
            Knowledge category
          </td>

          <td>
            Choisissez la catégorie de connaissances dans la liste déroulante, cela aidera New Relic AI à rechercher des informations au bon endroit. Vous pouvez choisir parmi les options suivantes :

            * <DNT>**Runbook**</DNT>
            * <DNT>**Retrodoc**</DNT>
          </td>
        </tr>
      </tbody>
    </table>
  </Step>

  <Step>
    Cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Sur la page <DNT>**Data source authentication**</DNT>, sélectionnez la méthode d’authentification requise. Vous pouvez choisir parmi les options suivantes :

    * <DNT>**Basic authentication**</DNT>
    * <DNT>**OAuth 2.0 authentication**</DNT>
  </Step>

  <Step>
    Sur la page Configurer la source de données, saisissez les informations requises et définissez les documents qui doivent être récupérés à quelle fréquence. Cliquez sur <DNT>**Create**</DNT>.
  </Step>

  <Step>
    Une fois configuration réussie, vous verrez l&apos;état de votre connecteur sur la page d&apos;aperçu du connecteur.

    <table>
      <thead>
        <tr>
          <th>
            Nom du champ
          </th>

          <th>
            Description
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            Statut
          </td>

          <td>
            Vous indique si la source de données est disponible pour New Relic AI
          </td>
        </tr>

        <tr>
          <td>
            Last Synced
          </td>

          <td>
            Indique quand les données ont été mises à jour pour la dernière fois
          </td>
        </tr>

        <tr>
          <td>
            Syncing
          </td>

          <td>
            Si cette option est désactivée, aucune nouvelle mise à jour de données ne se produira. Les données existantes resteront inchangées.
          </td>
        </tr>

        <tr>
          <td>
            Delete Connector
          </td>

          <td>
            La suppression d&apos;un connecteur supprimera toutes les données indexées.
          </td>
        </tr>
      </tbody>
    </table>
  </Step>
</Steps>
