L&apos;intégration de la fonctionnalité Knowledge connecteur (Récupérer et Générer) de New Relic avec Confluence vous aide à améliorer vos processus de dépannage et de documentation en référençant directement le contenu de Confluence dans les réponses de New Relic AI. Fournissez à New Relic une API clé et une requête de confluence et vous obtenez immédiatement des informations IA détaillées exploitant ces informations avec toutes vos expériences NRAI.

## Connectez le connecteur de connaissances avec confluence

Pour commencer à indexer votre contenu Confluence et bénéficier du connecteur de connaissances avec New Relic AI, suivez les étapes mentionnées :

<Steps>
  <Step>
    Accédez à <DNT>[**one.newrelic.com**](https://one.newrelic.com) &amp;gt; **Integrations &amp; Agents**</DNT>.
  </Step>

  <Step>
    Rechercher <DNT>**RAG Manager**</DNT>.
  </Step>

  <Step>
    Cliquez sur <DNT>**Add Confluence**</DNT>.
  </Step>

  <Step>
    Dans l&apos;onglet <DNT>**Tools**</DNT>, vous pourrez voir tous les outils RAG disponibles.

    <Callout variant="important">
      Cliquez sur l’onglet Documents pour voir tous les documents que vous avez indexés.
    </Callout>
  </Step>

  <Step>
    Cliquez sur <DNT>**Add Tools &gt; Confluence**</DNT> pour configurer le connecteur de confluence.
  </Step>

  <Step>
    Sélectionnez le <DNT>**Account**</DNT> dans la liste déroulante pour lequel les informations d’identification de confluence doivent être enregistrées et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Dans le champ <DNT>**Confluence Base URL**</DNT>, saisissez l’URL de base de votre instance de confluence. Par exemple, <DNT>[https://your-site-company.atlassian.net/wiki](https://your-site-company.atlassian.net/wiki)</DNT> et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Saisissez les informations suivantes :

    * <DNT>**Confluence Username**</DNT>:Votre nom d&apos;utilisateur Confluence.
    * <DNT>**API Token**</DNT>: Votre jeton API de confluence.
  </Step>

  <Step>
    Cliquez sur <DNT>**Save token in secret store**</DNT> et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Dans la section Paramètres de Confluence, saisissez les informations suivantes :

    1. <DNT>**Name your cofluence documents**</DNT>, entrez un nom unique pour vos documents de confluence, exemple <DNT>**Incident Runbook**</DNT>.

    2. Saisissez un <DNT>**Confluence query**</DNT> pour affiner le contenu, exemple. <DNT>**space=SPACEKEY**</DNT>.

    3. Choisissez la fréquence à laquelle vous souhaitez que le contenu de Confluence soit récupéré et réindexé dans la section **Confluence reindex interval**. Vous pouvez choisir parmi les options suivantes :

       * <DNT>**Monthly**</DNT>
       * <DNT>**Weekly**</DNT>
       * <DNT>**Daily**</DNT>

    4. Cliquez sur <DNT>**Save confluece settings**</DNT> et cliquez sur <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Cliquez sur <DNT>**Start Indexing**</DNT>. Le nouvel outil est créé et ajouté sous l’onglet Outils.
  </Step>

  <Step>
    Cliquez sur l’onglet <DNT>**Document**</DNT> pour voir les documents que vous avez indexés.
  </Step>
</Steps>
