New Relic&apos;s agentic integration provides an AI-driven solution to help automate and optimize specific observability workflows, enhancing system management and monitoring. This integration facilitates seamless operations by automating repetitive tasks through AI agents, reducing manual intervention. By integrating seamlessly with platforms such as GitHub Copilot, Gemini Code Assist, and ServiceNow, agentic integrations transform complex workflows into seamless operations.

## Fonctionnalité clé [#features]

* **Intelligent automation:** Leverage AI to automate and streamline complex workflows, reducing the need for manual tool-switching and enhancing productivity, while keeping a human in the loop.
* **Observabilité améliorée :** obtenez des informations complètes et détaillées sur votre système, permettant une détection, une analyse et une résolution plus rapides des incidents.
* **Intégration transparente :** connectez-vous à des plateformes populaires telles que GitHub Copilot pour apporter des analyses puissantes et des informations détaillées directement dans vos workflows existants.

## Accès et gestion des données [#access-data-mgmt]

Nos partenaires d&apos;intégration géreront les applications clientes et maintiendront le jeton utilisateur lorsque vous ou l&apos;un de vos utilisateurs vous authentifierez auprès de New Relic via l&apos;intégration du partenaire. Le partenaire tiers aura le même niveau d&apos;accès à New Relic que l&apos;utilisateur qui s&apos;est authentifié pendant la durée configurée dans les paramètres de l&apos;utilisateur, car les jetons sont utilisés en son nom. Cette disposition permet une intégration et une fonctionnalité transparentes. Pour plus d&apos;informations, vous pouvez vous référer à la documentation du partenaire qui détaille ses pratiques d&apos;utilisation et de gestion des jetons.

### Contrôle d&apos;accès basé sur les rôles (RBAC)

New Relic&apos;s agentic integration capabilities are granted via organization-scoped roles within New Relic&apos;s RBAC system. Users gain access by belonging to a group with an assigned organizational role that includes these capabilities. Individual accounts do not require direct configuration.

Currently, RBAC can be accessed through API interfaces. If you are interested in utilizing RBAC via API, feel free to contact us at [agenticai@newrelic.com](mailto:agenticai@newrelic.com). For detailed guidance on setting up RBAC, refer to our [account user management tutorial](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial).

## Intégration actuelle [#integrations]

### Copilote GitHub [#github-copilot]

Integrate New Relic insights directly into your IDE with GitHub Copilot. You can use Agent AI to access logs, errors, and recommendations without leaving your coding environment. It provides real-time insights as you develop and deploy the code.

Cette intégration offre :

* Vérifications de déploiement automatisées : effectuez des vérifications de déploiement automatiquement, en exploitant les données de New Relic.
* Alertes en temps réel et suivi des erreurs : restez informé grâce à des alertes immédiates et suivez les erreurs au fur et à mesure qu&apos;elles se produisent.
* Recommandations probabilistes pour les actions de déploiement.

For more information, refer to [Getting Started with New Relic agent for GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration).

### ServiceNow Now Assist [#now-assist]

New Relic AI insights are now right inside ServiceNow&apos;s Now Assist. This integration helps your operations teams, such as first responders and support agents, quickly understand incident severity, identify probable root causes, and streamline collaboration, all without leaving their ServiceNow workspace.

Pour plus d&apos;informations, reportez-vous à [Prise en main de l&apos;agent New Relic pour ServiceNow Now Assist](/docs/agentic-ai/agentic-integration/servicenow-integration).

### Assistance au code Gemini [#gemini-code-assist]

Intégrez les informations détaillées de New Relic dans votre IDE avec Gemini Code Assist. Cet outil permet aux développeurs d&apos;accéder aux logs pertinents, aux informations d&apos;erreur et aux recommandations au sein de l&apos;environnement de codage, offrant des informations détaillées qui peuvent éclairer le processus de développement et de déploiement.

Cette intégration offre :

* Monitoring automatisé du déploiement : monitorez automatiquement le déploiement à l&apos;aide des données temps réel de New Relic pour identifier et résoudre les problèmes efficacement.
* Alertes et suivi des erreurs : Recevez une notification immédiate et suivez efficacement les erreurs pour maintenir les performances des applications et la stabilité.
* Recommandations pour les actions de déploiement : utilisez des recommandations basées sur l’analyse des données pour améliorer les performances du code et soutenir la prise de décision.

Pour plus d&apos;informations, reportez-vous [à Prise en main de l&apos;agent New Relic pour Gemini Code Assist](/docs/agentic-ai/agentic-integration/gemini-integration).
