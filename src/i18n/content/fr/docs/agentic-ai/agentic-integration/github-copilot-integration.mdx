L&apos;extension New Relic agent pour GitHub Copilot fournit des informations détaillées et des conseils intelligents adaptés pour optimiser les performances des application , monitorer infrastructure et gérer le déploiement, le tout parfaitement intégré à GitHub Copilot.

Ce guide vous guidera tout au long du processus d&apos;installation et d&apos;utilisation de l&apos;agent New Relic dans l&apos;extension GitHub Copilot.

## Installer l&apos;agent New Relic dans GitHub Copilot [#install-new-relic-agent]

### Prérequis [#prerequisites]

Avant de commencer, assurez-vous que les conditions préalables suivantes sont remplies pour garantir un processus de configuration fluide :

* Un IDE est installé sur votre système.
* Vous avez accès à [GitHub Copilot](https://github.com/features/copilot).
* L&apos;extension GitHub Copilot est installée dans votre IDE. Pour plus d&apos;informations, reportez-vous à [la documentation GitHub Copilot](https://docs.github.com/en/copilot/managing-copilot/configure-personal-settings/installing-the-github-copilot-extension-in-your-environment).
* La [stratégie d’extension GitHub Copilot](https://docs.github.com/en/enterprise-cloud@latest/copilot/managing-copilot/managing-github-copilot-in-your-organization/managing-policies-for-copilot-in-your-organization#setting-a-policy-for-github-copilot-extensions-in-your-organization) est activée pour votre organisation.

<Callout variant="tip" title="conseil">
  Si vous souhaitez restreindre l&apos;accès à GitHub Copilot pour certains rôles, configurez les paramètres sur GitHub. Pour restreindre l&apos;accès à New Relic AI pour des rôles ou des collaborateurs spécifiques, vous pouvez utiliser le contrôle d&apos;accès basé sur les rôles (RBAC) de New Relic. Pour plus d&apos;informations, reportez-vous [à la gestion des utilisateursNew Relic ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

**Pour installer l&apos;agent New Relic dans GitHub Copilot :**

1. Sur GitHub MarketPlace, accédez à [l&apos;agentNew Relic ](https://github.com/marketplace/newrelic-copilot-extension).
2. Cliquez sur **Add &gt; Install it for free**.

Une fois l’agent installé, vous pouvez l’utiliser dans l’extension GitHub Chat dans votre IDE. Pour plus d&apos;informations, reportez-vous [à l&apos;utilisation de l&apos;agent New Relic dans Copilot](#communicate-with-new-relic-agent-in-gitHub-copilot).

## Communiquez avec l&apos;agent New Relic dans GitHub Copilot [#communicate-with-new-relic-agent-in-gitHub-copilot]

L&apos;agent New Relic au sein de Github Copilot peut vous aider dans une variété de tâches et offrir des informations précieuses détaillées. Pour interagir avec l&apos;agent New Relic, dans la zone de saisie de discussion du copilote, saisissez `@newrelic`.

### Connectez-vous à l&apos;agent New Relic [#sign-in]

Lorsque vous lancez une conversation pour la première fois avec un agent New Relic , vous serez invité à vous connecter. Suivez les instructions pour terminer le processus de connexion.

### Structure du message de discussion [#chat-message-structure]

Pour interagir avec l&apos;agent New Relic , entrez votre prompt comme suit :

`@newrelic <your message>`

Où:

* `@newrelic`:interaction avec l&apos;agent New Relic .
* `<your message>`:Action à soumettre à l&apos;agent pour traitement.
