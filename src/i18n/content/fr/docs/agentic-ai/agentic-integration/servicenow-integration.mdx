L&apos;intégration de New Relic AI avec Now Assist de ServiceNow aidera les utilisateurs de ServiceNow, tels que les opérateurs, les premiers intervenants et les agents de support, à :

* Évaluer la gravité d&apos;un incident en comprenant son impact sur les autres services et utilisateurs
* Identifier les théories des causes profondes probables d&apos;un incident
* Faciliter l&apos;escalade vers les collaborateurs pour atténuer le problème

Ce guide vous guidera tout au long du processus d&apos;installation et d&apos;utilisation de New Relic AI dans Now Assist de ServiceNow.

## Installation [#setup]

Pour utiliser l&apos;extension New Relic AI avec ServiceNow, vous aurez besoin de :

* Une [clé API utilisateur](/docs/apis/intro-apis/new-relic-api-keys/#overview-keys) New Relic. Si un utilisateur New Relic est supprimé dans New Relic, ses clés utilisateur sont également désactivées et ne fonctionneront plus. Pour garantir une intégration sécurisée et ininterrompue, veuillez prendre note des recommandations suivantes :

  * Utilisez un compte `service user` pour éviter toute interruption lorsqu&apos;un compte personnel ou individuel est résilié.
  * Bien que `user keys` donne accès à plusieurs comptes, ils sont liés à un seul compte spécifique : le compte à partir duquel ils ont été créés. Si un compte est supprimé, toutes les clés utilisateur associées à ce compte deviendront invalides. Assurez-vous donc que le compte pour lequel vous créez la clé est durable.
  * Vérifiez que l’utilisateur pour lequel la clé utilisateur est utilisée possède l’autorisation Lecture seule.

  <Callout variant="tip" title="Note">
    Veuillez noter que la clé utilisateur fournie permettra à chaque utilisateur de votre compte ServiceNow d&apos;accéder à cette intégration.
  </Callout>

  * Pour configurer New Relic AI dans Now Assist, suivez les étapes décrites [ici.](https://www.servicenow.com/docs/csh?topicname=now-assist-itom-use-aia.html&version=latest)

## Interagissez avec l&apos; New Relic AI grâce à Now Assist [#new-relic-tool]

Dans ServiceNow :

1. Accédez à <DNT>**All &gt; Event Management &gt; Express List**</DNT>.

2. Sélectionnez une alerte en vous assurant que la source est New Relic.

3. Cliquez sur l’icône <DNT>Now Assist</DNT> pour ouvrir le panneau.

4. Demandez maintenant une assistance pour une question sur l&apos;alerte.

5. Saisissez l&apos;une des invites prises en charge, vous pouvez également mélanger et assortir :

   * Quelle est la gravité de cette alerte ?
   * Quelles entités sont impactées ?
   * Quelles sont les théories des causes probables ?
   * Qui dois-je faire intervenir pour l’enquête ?
   * Y a-t-il des déploiements récents ?
   * Quel est l’impact de l’alerte, les théories des causes probables et qui doit être impliqué dans l’enquête ?

   <Callout variant="tip" title="conseil">
     Votre conversation dans le panneau Now Assist est spécifique à l&apos;alerte que vous avez sélectionnée à l&apos;étape 2. Pour poser une question sur une autre alerte, ouvrez cette alerte et sélectionnez l&apos;icône Now Assist pour démarrer une nouvelle conversation.
   </Callout>
