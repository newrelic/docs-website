L&apos;extension New Relic Tool pour Google Gemini Code Assist fournit des informations détaillées et des conseils intelligents adaptés pour optimiser les performances des applications, surveiller infrastructure et gérer le déploiement, le tout parfaitement intégré à Gemini Code Assist.

Ce guide vous guidera tout au long du processus d&apos;installation et d&apos;utilisation de l&apos;outil New Relic dans Gemini Code Assist.

<Callout variant="tip" title="conseil">
  En cas de problème avec l&apos;intégration, contactez [agenticai@newrelic.com](mailto:agenticai@newrelic.com).
</Callout>

## Activer l&apos;outil New Relic dans Gemini Code Assist [#enable-new-relic-tool]

### Prérequis [#prerequisites]

Avant de commencer, assurez-vous que les conditions préalables suivantes sont remplies pour garantir un processus de configuration fluide :

* Un environnement de développement intégré (IDE) est installé sur votre ordinateur.
* Vous avez accès à Google Gemini Code Assist.
* Gemini Code Assist est installé dans votre IDE. Pour plus d&apos;informations, consultez [la documentation de Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="conseil">
  Si vous souhaitez restreindre l&apos;accès à Gemini Code Assist pour certains rôles, configurez les paramètres sur Gemini Code Assist. Pour restreindre l&apos;accès à New Relic AI pour des rôles ou des collaborateurs spécifiques, vous pouvez utiliser le contrôle d&apos;accès basé sur les rôles (RBAC) de New Relic. Pour plus d&apos;informations, reportez-vous [à la gestion des utilisateurs New Relic](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

Pour activer l&apos;outil de New Relic :

1. Depuis votre IDE, log vous à [Gemini Code Assist](https://codeassist.google.com) en cliquant sur Sign in with Google.

2. Une fois authentifié avec succès, revenez à votre IDE.

3. Dans le <DNT>Gemini Code Assist Chat</DNT> (chat), entrez `@newrelic` et appuyez sur Entrée. Vous serez invité à activer l’outil New Relic.

4. Dans le prompt, cliquez sur le lien pour activer l’outil New Relic. Une fenêtre de navigateur s&apos;ouvre dans laquelle vous serez invité à activer l&apos;outil New Relic dans Gemini Code Assist.

5. Activez l’outil New Relic, puis cliquez sur **Connect**.

6. Connectez-vous en utilisant les détails de votre compte New Relic. Cela connecte avec succès votre compte New Relic à Gemini Code Assist.

   Vous pouvez désormais utiliser l’outil New Relic dans l’extension de chat Gemini Code Assist dans votre IDE.

## Interagir avec l&apos;outil New Relic [#new-relic-tool]

L&apos;outil New Relic de Gemini Code Assist peut vous aider dans une variété de tâches et offrir des informations précieuses détaillées. Pour interagir avec l&apos;outil New Relic, dans la zone de saisie du chat, saisissez votre prompt comme suit :

`@newrelic <your message>`

Où, `<your message>` est l’action que vous souhaitez que l’outil exécute. Par exemple, `@newrelic What is apdex?`
