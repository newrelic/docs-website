## Installer l&apos;agent New Relic dans Gemini Code Assist [#install-new-relic-agent]

### Prérequis [#prerequisites]

Avant de commencer, assurez-vous que les conditions préalables suivantes sont remplies pour garantir un processus de configuration fluide :

* Un IDE est installé sur votre système.
* Vous avez accès à Google Gemini Code Assist.
* Gemini Code Assist est installé dans votre IDE. Pour plus d&apos;informations, consultez [la documentation de Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="conseil">
  Si vous souhaitez restreindre l&apos;accès à Gemini Code Assist pour certains rôles, configurez les paramètres sur Gemini Code Assist. Pour restreindre l&apos;accès à New Relic AI pour des rôles ou des collaborateurs spécifiques, vous pouvez utiliser le contrôle d&apos;accès basé sur les rôles (RBAC) de New Relic. Pour plus d&apos;informations, reportez-vous [à la gestion des utilisateurs New Relic](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

**Pour activer l&apos;agent New Relic dans Gemini Code Assist :**

1. Sur votre <DNT>Google Cloud Console</DNT>, recherchez <DNT>**NewRelic**</DNT> et activez l&apos;extension. Une fenêtre de dialogue de connexion s&apos;ouvre.

2. Fournissez vos identifiants New Relic pour vous connecter. Le message **New Relic successfully linked.** s&apos;affiche.

   Vous pouvez désormais utiliser l’agent New Relic dans l’extension de chat Gemini Code Assist dans votre IDE. Pour plus d&apos;informations, reportez-vous [à l&apos;utilisation de l&apos;agent New Relic dans Gemini Code Assist](#communicate-with-new-relic-agent-in-gemini-code-assist).

## Communiquez avec l&apos;agent New Relic dans Gemini Code Assist [#communicate-with-new-relic-agent-in-gemini-code-assist]

L&apos;agent New Relic au sein de Gemini Code Assist peut vous aider dans une variété de tâches et offrir des informations précieuses détaillées. Pour interagir avec l&apos;agent New Relic, dans la zone de saisie du chat, saisissez `@newrelic`.

### Structure du message de discussion [#chat-message-structure]

Pour interagir avec l&apos;agent New Relic, entrez votre prompt comme suit :

`@newrelic <your message>`

Où:

* `@newrelic`:interaction avec l&apos;agent New Relic.
* `<your message>`:Action à soumettre à l&apos;agent pour traitement.
