Une fois que vous avez [ajouté des facettes à un niveau de service](/docs/service-level-management/facet-slm/configure), vous pouvez afficher et analyser les données de conformité et de budget d’erreur réparties par ces facettes pour identifier où se produisent les problèmes de performances.

## Afficher les résultats à facettes [#view-faceted-results]

Pour afficher la conformité et le budget d&apos;erreur pour les valeurs d&apos;attribut individuelles et leurs combinaisons :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Service Levels**</DNT>.
2. Sélectionnez le niveau de service.
3. Utilisez la liste déroulante <DNT>**View by group**</DNT> et le panneau <DNT>**Group by**</DNT> pour analyser les attributs que vous avez configurés pour le regroupement.

## Exemple [#example]

Lorsque vous regroupez par `environment` et `region`, la liste déroulante <DNT>**View by group**</DNT> et le panneau <DNT>**Group by**</DNT> affichent les données de performance pour chaque valeur d&apos;attribut individuelle (tous les environnements, toutes les régions) ainsi que chaque combinaison d&apos;environnement et de région :

| Valeur de l&apos;attribut | Conformité | Budget d&apos;erreur restant |
| ------------------------- | ---------- | ---------------------------- |
| production                | 99,05 %    | 48,5 %                       |
| simulation                | 99,50 %    | 60%                          |
| États-Unis-Est-1          | 99,70 %    | 72,5 %                       |
| États-Unis-Ouest-1        | 98,20 %    | 12 %                         |
| production, us-east-1     | 99,9%      | 85%                          |
| production, us-west-1     | 98,2 %     | 12 %                         |
| simulation, us-east-1     | 99,5 %     | 60%                          |

## Analysez vos données [#interactive-analysis]

Utilisez la vue à facettes pour filtrer et comparer les données de conformité sur différents segments.

<img title="Faceted service level UI" alt="Screenshot showing the faceted service level analysis interface with group filters and compliance data" src="/images/slm_screenshot-full_faceted-view.webp" />

* **Pour afficher les données d&apos;un segment spécifique**: sélectionnez une facette dans la liste déroulante en haut de la page. Les graphiques principaux se mettent à jour pour afficher uniquement les données de ce segment, et un graphique <DNT>**Group SLO**</DNT> apparaît, montrant les performances du groupe sélectionné par rapport à la conformité globale.
* **Pour comparer les attributs individuels et les combinaisons**: lorsque vous avez sélectionné deux facettes, utilisez la vue individuelle pour voir chaque attribut séparément, ou la vue combinée pour voir les combinaisons d&apos;attributs.
* **Pour revenir à la vue d&apos;ensemble**: sélectionnez <DNT>**None**</DNT> dans la liste déroulante.

## Alertes à facettes [#faceted-alerting]

[Vous pouvez créer des alertes](/docs/service-level-management/alerts-slm/) basées sur des facettes spécifiques pour vous assurer d’être notifié lorsqu’un segment particulier est défaillant.

**Types d&apos;alertes pris en charge :**

* Alertes de conformité SLI
* Alertes de consommation rapide
* Alertes à combustion lente

**Non pris en charge :**

Alertes de budget d&apos;erreur restant (celles-ci restent disponibles uniquement au niveau agrégé)
