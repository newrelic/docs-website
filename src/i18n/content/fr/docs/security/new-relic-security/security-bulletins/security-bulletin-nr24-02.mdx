<DNT>**Vulnerability Identifier:**</DNT> NR24-02

<DNT>**Priority:**</DNT> Haut

## Résumé

New Relic conseille à tous les clients utilisant l&apos;exportateur Salesforce New Relic de mettre à jour vers la version 2.2.0, que New Relic a sorti pour éliminer une [version vulnérable récemment annoncée d&apos;OpenSSH](https://nvd.nist.gov/vuln/detail/CVE-2024-6387).

## Action requise

New Relic recommande aux clients qui utilisent l&apos;exportateur Salesforce New Relic de mettre immédiatement à jour vers la version 2.2.0.

New Relic a fourni les ressources suivantes pour vous aider avec ces mises à jour :

* [Salesforce exportateur sortie Notes](https://github.com/newrelic/newrelic-salesforce-exporter/releases/tag/2.2.0)
* [Détection et mise à niveau des déploiements sur hôte](https://github.com/newrelic/newrelic-salesforce-exporter?tab=readme-ov-file#upgrading-on-host-deployments)

Si les clients ne parviennent pas à mettre à niveau leur exportateur Salesforce New Relic, limitez l’accès SSH via des contrôles basés sur le réseau pour minimiser les risques d’attaque.
