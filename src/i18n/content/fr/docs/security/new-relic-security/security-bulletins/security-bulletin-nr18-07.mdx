## Résumé

Une mise à jour de sécurité pour les agents Java, Python et .NET corrige un problème où l&apos;agent peut signaler les résultats de la requête de base de données à New Relic ou réémettre une instruction SQL.

<DNT>**Release date:**</DNT> 7 mars 2018

<DNT>**Vulnerability identifier:**</DNT> NR18-07

<DNT>**Priority:**</DNT> Haut

## informations sur les vulnérabilités [#vuln-info]

Les agents New Relic exécutent des plans explicatifs pour [le tracing lent des transactions](/docs/apm/applications-menu/monitoring/viewing-slow-query-details) et les requêtes SQL lentes. Les versions précédentes des agents exécutaient un plan d&amp;apos;explication sur la requête SQL en préfixant la requête avec `explain`. Cela peut entraîner un problème lorsqu&amp;apos;il existe plusieurs instructions séparées par des points-virgules dans une seule requête. La première instruction de la chaîne renvoie son plan d&amp;apos;explication, mais toute instruction ultérieure peut s&amp;apos;exécuter comme une instruction SQL générale. Selon la langue, la bibliothèque et la base de données, l&amp;apos;agent peut renvoyer les résultats des instructions supplémentaires à New Relic. Il est également possible que les instructions supplémentaires exécutent une commande `INSERT` ou `UPDATE` supplémentaire. Avec cette mise à jour de sécurité, les agents New Relic n&amp;apos;exécuteront plus de plans explicatifs sur toute requête contenant un point-virgule comme séparateur d&amp;apos;instructions.

### Facteurs atténuants [#factors]

* De nombreuses bibliothèques SQL et frameworks de langage empêchent diverses formes d&apos;exécution de plusieurs instructions avec `explain`.
* Expliquez que les plans sont désactivés pour les versions plus récentes de l&apos;agent .NET.

## Solutions de contournement

Désactiver les plans `explain` avec le traceur de transaction dans la configuration de l&amp;apos;agent :

* [`transaction_tracer`](/docs/agents/java-agent/configuration/java-agent-configuration-config-file#h2-Transaction-tracer) pour Java
* [`transactiontracer`](/docs/agents/net-agent/configuration/net-agent-configuration#transaction_tracer) pour .NET
* [`transaction_tracer`](/docs/agents/python-agent/configuration/python-agent-configuration#txn-tracer-settings) pour Python

New Relic s&apos;engage à assurer la sécurité de ses clients et de leurs données.
