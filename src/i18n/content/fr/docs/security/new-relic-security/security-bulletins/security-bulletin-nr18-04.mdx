## informations sur les vulnérabilités [#vuln-info]

La collection d&apos;erreurs .NET de New Relic capture des informations sur les exceptions non détectées et les envoie à New Relic. Si le mode Haute sécurité est activé, l&apos;agent ne doit pas capturer les détails du message d&apos;erreur. Cette sortie corrige l&apos;agent pour la trace d&apos;erreur et l&apos;événement d&apos;erreur afin que le message d&apos;erreur ne soit pas envoyé à New Relic lorsque le mode haute sécurité est activé.

### Facteurs atténuants [#factors]

* Il s&apos;agit d&apos;une fonctionnalité spéciale du [mode Haute sécurité](/docs/agents/net-agent/configuration/net-agent-configuration#high_security_mode).

## Solutions de contournement

Désactivez [la collecte d’erreurs](/docs/agents/net-agent/configuration/net-agent-configuration#error_collector) dans la configuration de l’agent ou [ignorez les exceptions spécifiques](/docs/agents/net-agent/configuration/net-agent-configuration#error-ignoreErrors) dans la configuration.

## Signaler les vulnérabilités de sécurité à New Relic [#report]

New Relic s&apos;engage à assurer la sécurité de ses clients et de leurs données. Si vous pensez avoir découvert une faille de sécurité dans l&apos;un de nos produits ou sites Web, nous vous invitons à la signaler au programme de divulgation coordonnée de New Relic. Pour plus d&apos;informations, consultez [Signalement des vulnérabilités de sécurité](/docs/security/new-relic-security/data-privacy/reporting-security-vulnerabilities).
