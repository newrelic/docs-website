## Résumé

Une mise à jour de sécurité pour l’agent d’infrastructure Windows corrige un problème où l’agent peut exécuter des binaires privilégiés dans le chemin système.

<DNT>**Release date:**</DNT> 8 octobre 2018

<DNT>**Vulnerability identifier:**</DNT> NR18-11

<DNT>**Priority:**</DNT> Moyen

## informations sur les vulnérabilités [#vuln-info]

L&apos;agent d&apos;infrastructure Windows New Relic peut exécuter des commandes système dans le cadre de son fonctionnement normal. Dans les versions affectées de l’agent d’infrastructure Windows, un utilisateur malveillant disposant de l’autorisation de modifier le chemin d’accès système pourrait potentiellement abuser de cette fonctionnalité pour exécuter des binaires arbitraires avec des privilèges élevés.

### Facteurs atténuants [#factors]

* Cette vulnérabilité n&apos;existe que sur le système Windows .
* Cette vulnérabilité nécessite un accès existant au système et l&apos;autorisation de modifier le PATH du système.
