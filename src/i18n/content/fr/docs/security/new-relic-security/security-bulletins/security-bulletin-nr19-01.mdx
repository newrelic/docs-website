## Résumé

Une mise à jour de sécurité pour l&apos;agent .NET corrige un problème où les chaînes de requête peuvent être capturées lors de l&apos;utilisation de l&apos;instrumentation OpenRasta.

<DNT>**Release date:**</DNT> 9 janvier 2019

<DNT>**Vulnerability identifier:**</DNT> NR19-01

<DNT>**Priority:**</DNT> Moyen

## informations sur les vulnérabilités [#vuln-info]

Lors de l&apos;utilisation de l&apos;instrumentation OpenRasta, l&apos;URL complète peut être capturée sur requests instrumentées. Cela peut entraîner la collecte de chaînes de requête pouvant contenir des informations sensibles

### Facteurs atténuants [#factors]

Cette vulnérabilité n&apos;existe que lors de l&apos;utilisation de l&apos;instrumentation OpenRasta.

## Solutions de contournement

* [Mise à jour vers le dernier agent .NET New Relic.](/docs/agents/net-agent/installation/update-net-agent)
* Désactiver l&apos;instrumentation OpenRasta.

## Signaler les vulnérabilités de sécurité à New Relic [#report]

New Relic s&apos;engage à assurer la sécurité de ses clients et de leurs données. Si vous pensez avoir découvert une faille de sécurité dans l&apos;un de nos produits ou sites Web, nous vous invitons à la signaler au programme de divulgation coordonnée de New Relic. Pour plus d&apos;informations, consultez [Signalement des vulnérabilités de sécurité](/docs/security/new-relic-security/data-privacy/reporting-security-vulnerabilities).
