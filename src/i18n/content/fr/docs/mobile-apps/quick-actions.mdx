Pour rester au courant du monde en évolution rapide de l&apos;ingénierie et de l&apos;observabilité, utilisez des actions rapides pour accéder rapidement aux fonctionnalités communes de New Relic, directement sur l&apos;écran d&apos;accueil.

Les actions rapides affichent jusqu&apos;à quatre de vos entités préférées. Si vous avez plus de quatre favoris, vous verrez ceux les plus récemment utilisés. Si vous n&apos;avez pas configuré de favoris, l&apos;écran affiche des liens vers les fonctionnalités les plus utilisées (APM, alertes, synthétiques, dashboard).

Pour utiliser cette fonctionnalité, appuyez simplement longuement sur l&apos;icône de l&apos;application New Relic sur votre page d&apos;accueil pour voir vos raccourcis. Voici quelques exemples :
