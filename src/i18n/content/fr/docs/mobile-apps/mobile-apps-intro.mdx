L&apos;application mobile New Relic vous permet d&apos;accéder à vos données où que vous soyez. Vous pouvez afficher un dashboard organisé à partir de toutes les fonctionnalités de New Relic, de l&apos;APM à infrastructure en passant par le navigateur, et accéder à n&apos;importe quel dashboard que vous avez créé. Si vous avez besoin d&apos;aller plus loin, vous pouvez interroger vos données directement avec NRQL ou rechercher votre log. Et vous pouvez recevoir <InlinePopover type="alerts" />directement dans l&apos;application afin de pouvoir commencer à enquêter immédiatement.

## Capacités mobiles [#mobile-app-features]

Voici un échantillon de ce que vous pouvez faire avec notre application mobile :

* <DNT>**Alerts**</DNT>: Consultez les événements d&apos;alerte actuels et passés, acquittez les événements d&apos;alerte et appuyez pour voir les données de performance des entités associées.
* <DNT>**Application performance monitoring (APM)**</DNT>: Consultez le temps réel et les données historiques sur les transactions, les erreurs, la base de données, les services externes, etc.
* <DNT>**Browser monitoring**</DNT>:Suivez le débit, les temps de chargement, les erreurs, requests AJAX et bien plus encore sur toutes vos applications Web frontales.
* <DNT>**Dashboards and NRQL**</DNT>: Affichez votre dashboard ou approfondissez vos connaissances avec NRQL. Vous pouvez créer une requête NRQL personnalisée ou afficher le NRQL de n&apos;importe quel graphique en appuyant sur le graphique, puis en appuyant sur le débordement <Icon name="fe-more-horizontal" /> icône et en sélectionnant <DNT>**View query**</DNT>.
* <DNT>**Explorer**</DNT>: Parcourez tout le reste dans New Relic, des services AWS aux performances du réseau.
* <DNT>**Infrastructure monitoring**</DNT>: Consultez l&apos;état de santé de vos hôtes exécutés dans le cloud, sur des hôtes dédiés ou des conteneurs exécutés dans des environnements orchestrés.
* <DNT>**Mobile monitoring**</DNT>:Consultez les performances, les erreurs, les plantages et bien plus encore sur toutes vos applications mobiles.
* <DNT>**Synthetic monitoring**</DNT>:Consultez les performances de chaque moniteur, visualisez les résultats individuels et suivez les échecs.

## Confidentialité des données [#privacy]

Nos applications mobiles enregistrent uniquement les informations nécessaires pour aider à l&apos;authentification et au dépannage :

* Adresse e-mail de l&apos;utilisateur associée à votre compte New Relic, y compris le prénom et le nom (à des fins d&apos;authentification uniquement)
* Adresse IP
* ID de l&apos;appareil

Pour plus d&apos;informations, consultez notre [documentation sur la confidentialité et la sécurité des données mobiles](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps).
