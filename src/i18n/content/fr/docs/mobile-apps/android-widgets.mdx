## Ajouter un widget Android [#add-widget]

Complétez les éléments suivants pour ajouter un widget :

1. Appuyez et maintenez une zone vide jusqu&apos;à ce qu&apos;une fenêtre modale apparaisse.
2. Appuyez sur <DNT>**Widgets**</DNT>.
3. Recherchez le widget New Relic.
4. Appuyez longuement sur le widget pour le déplacer vers l&apos;emplacement souhaité sur votre appareil.
5. Sélectionnez le compte et la plage horaire pendant laquelle vous souhaitez compter les problèmes actifs.
6. Appuyez longuement et ajustez la taille selon vos besoins.

Le widget est ajouté à votre écran d&apos;accueil avec le compte que vous consultez actuellement et la durée est définie pour durer 30 minutes.

## Ajouter un graphique à un widget [#add-chart]

Si vous n&apos;avez pas encore sélectionné le graphique que vous souhaitez afficher à partir de l&apos;application New Relic, suivez ces étapes :

1. Accédez au graphique que vous souhaitez voir dans le widget.
2. Appuyez sur <DNT>**+ Create Widget**</DNT>.
3. Appuyez sur le bouton <DNT>**Add to home screen**</DNT> qui apparaît en bas.
4. Appuyez longuement et ajustez la taille selon vos besoins.
5. Appuyez sur le widget de l&apos;écran d&apos;accueil pour ouvrir l&apos;application New Relic dans votre liste <DNT>**Issue**</DNT> .
