L&apos;application Android New Relic vous permet d&apos;accéder à vos données où que vous soyez. Vous pouvez afficher un dashboard organisé à partir de toutes les fonctionnalités de New Relic, de l&apos;APM à infrastructure en passant par le navigateur, et accéder à n&apos;importe quel dashboard que vous avez créé. Si vous avez besoin d&apos;aller plus loin, vous pouvez interroger vos données directement avec NRQL ou rechercher dans votre log. Et vous pouvez recevoir <InlinePopover type="alerts" />directement sur l&apos;application afin de pouvoir commencer à enquêter immédiatement.

## Confidentialité des données [#privacy]

Nos applications mobiles enregistrent uniquement les informations nécessaires pour aider à l&apos;authentification et au dépannage :

* Adresse e-mail de l&apos;utilisateur associée à votre compte New Relic, y compris le prénom et le nom (à des fins d&apos;authentification uniquement)
* Adresse IP
* ID de l&apos;appareil

Pour plus d&apos;informations, consultez notre [documentation sur la confidentialité et la sécurité des données mobiles](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps).
