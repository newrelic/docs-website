Ceci explique comment authentifier votre compte d&apos;application mobile New Relic et comment ajouter ou supprimer des utilisateurs de votre appareil mobile.

## authentification utilisateur [#authentication]

Selon votre compte New Relic, des étapes d&apos;installation ou d&apos;authentification supplémentaires peuvent être requises lorsque vous installez l&apos;application mobile New Relic.

<table>
  <thead>
    <tr>
      <th width={200}>
        <DNT>
          **New Relic account**
        </DNT>
      </th>

      <th>
        <DNT>
          **Additional requirements**
        </DNT>
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Nouvel utilisateur
      </td>

      <td>
        Si vous n&apos;avez pas encore de compte New Relic :

        1. Depuis votre navigateur Web de bureau, [créez un compte New Relic](https://newrelic.com/signup).

        2. [Installez votre application](/docs/site/installing-the-agent) avec l&amp;apos;agent New Relic approprié.

           Dans le cadre de la configuration d&apos;un nouveau compte, vous recevrez un e-mail contenant un lien de réinitialisation du mot de passe. Le lien de réinitialisation du mot de passe expire après 20 minutes pour les applications mobiles.
      </td>
    </tr>

    <tr>
      <td>
        New Relic utilisateur existant
      </td>

      <td>
        Aucune exigence supplémentaire ; vos applications, hôtes et clés de transaction apparaissent automatiquement après votre connexion.
      </td>
    </tr>

    <tr>
      <td>
        utilisateur avec les comptes partenaires New Relic
      </td>

      <td>
        Selon le partenaire, vous devrez peut-être effectuer un [processus d&apos;authentification](/docs/ios/ios-with-partners) différent.

        <DNT>**Azure Store users:**</DNT> En raison de l’intégration profonde entre Azure Storefront et New Relic, les utilisateurs d’ Azure Storefront ne peuvent pas accéder à leurs comptes sur les applications Android ou iOS New Relic.
      </td>
    </tr>

    <tr>
      <td>
        utilisateur avec des comptes compatibles SAML-SSO
      </td>

      <td>
        Lorsque vous vous connectez à l&apos;application mobile New Relic, votre session est automatiquement redirigée vers votre navigateur Web. À partir de là, vous pouvez vous connecter à votre compte New Relic SAML-SSO.

        Si vous constatez des erreurs lors de l&apos;utilisation de comptes SAML-SSO sur votre appareil mobile, vérifiez que vous pouvez vous connecter à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT> avec un navigateur Web de bureau.

        * Si non, contactez votre administrateur.
        * Si oui, obtenez de l&apos;aide sur [support.newrelic.com](https://support.newrelic.com).
      </td>
    </tr>
  </tbody>
</table>

  <Collapser id="switch-accounts" title="Basculer entre les comptes">
    Pour basculer entre les comptes associés à votre nom d&apos;utilisateur :

    1. Dans le menu <DNT>**Users**</DNT> , sélectionnez le nom d’utilisateur.
    2. Sélectionnez le nom du compte.

  <Collapser id="Remove a username" title="Supprimer ou rajouter un nom d'utilisateur">
    Pour supprimer un nom d’utilisateur spécifique de cet appareil :

    1. Dans le menu <DNT>**Users**</DNT> , sélectionnez <DNT>**Logout**</DNT>.

    2. Pour supprimer un utilisateur de cet appareil, sélectionnez le signe moins rouge de l&apos;utilisateur. <Icon style={{color: 'red'}} name="fe-minus-circle" /> icône.

    3. Sélectionnez l’icône <DNT>**Log out**</DNT> de l’utilisateur.

       Pour ajouter à nouveau un utilisateur, connectez-vous à nouveau avec ce nom d&apos;utilisateur.
  </Collapser>
