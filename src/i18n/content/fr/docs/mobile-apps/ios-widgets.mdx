La clé du succès en DevOps et en ingénierie de fiabilité du site réside dans la réduction de votre Délai moyen de résolution (MTTR) (MTTR). Le widget peut vous aider à faire cela, même lorsque vous n&apos;êtes pas devant votre ordinateur.

## Avant de commencer à utiliser le widget [#requirements]

Pour configurer [le widget iOS](https://support.apple.com/en-us/HT207122) sur votre iPhone ou iPad, assurez-vous que vous utilisez iOS 14 ou une version ultérieure et que l&apos; [application New Relic Mobile](install-newrelic-ios-mobile-app) est installée.

Voici un exemple d’ajout d’un widget à l’écran d’accueil :

1. Appuyez et maintenez une zone vide ou un widget jusqu&apos;à ce que les icônes bougent.

2. Appuyez sur l’icône <DNT>**+**</DNT> dans le coin supérieur gauche.

   <img style={{ maxWidth: '290px' }} title="Screenshot showing the Add (+) button" alt="Screenshot showing the Add (+) button" src="/images/mobile-apps_screenshot-full_iOS-widgets-edit-home-screen.webp" />

3. Recherchez le widget <DNT>**New Relic**</DNT> .

4. Faites glisser votre doigt vers la gauche ou la droite pour sélectionner parmi trois tailles de widget, puis appuyez sur <DNT>**Add Widget**</DNT>.

   <Video type="wistia" id="qyk74p7j56" vertical width="290px" />

5. Appuyez sur <DNT>**Done**</DNT>.

## Ajouter un graphique à un widget [#add-chart]

Si vous n&apos;avez pas encore sélectionné le graphique que vous souhaitez afficher à partir de l&apos;application New Relic, suivez ces étapes :

1. Accédez au graphique que vous souhaitez voir dans le widget.

2. Appuyez sur le graphique pour afficher la vue détaillée.

3. Appuyez sur l’icône représentant des points de suspension <DNT>**...**</DNT> dans le coin supérieur droit pour afficher plus d’options.

   <img style={{ maxWidth: '290px' }} title="Screenshot showing the dropdown options in the ellipsis" alt="Screenshot showing the dropdown options in the ellipsis menu" src="/images/mobile-apps_screenshot-full_iOS-widgets-add-widget-from-chart.webp" />

4. Sélectionnez <DNT>**Add to widgets**</DNT>.

5. Vous pouvez également supprimer et réorganiser le widget avec <DNT>**Manage widgets**</DNT>.

   <Video type="wistia" id="7dd2al2g8w" vertical width="290px" />

## Désigner des graphiques pour le widget [#designate-charts]

Pour sélectionner le graphique à afficher pour chacun des widgets iOS :

1. Appuyez et maintenez le widget jusqu&apos;à ce qu&apos;il bouge.

2. Appuyez sur <DNT>**Select chart**</DNT> ou sur le nom du graphique.

   <img style={{ maxWidth: '290px' }} title="Screenshot showing the option to select a chart" alt="Screen showing the option to select a chart" src="/images/mobile-apps_screenshot-full_iOS-widgets-configure-widget.webp" />

3. Si vous préférez, vous pouvez masquer le titre et/ou la légende de cet écran.
