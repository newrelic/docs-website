Qu&apos;il s&apos;agisse d&apos;une requête ad hoc ou d&apos;une requête que vous enregistrez via [des fichiers NRQL ](#nrql-files), la possibilité d&apos;exécuter une requête directement depuis votre IDE vous donne un accès puissant à toutes les données de performances dont New Relic dispose sur vos services. Si vous n&apos;êtes pas déjà familier avec NRQL, vous pouvez en apprendre davantage sur [le langage de requête New Relic](/docs/nrql/get-started/introduction-nrql-new-relics-query-language).

Cliquez sur l&apos;icône &quot;demandez vos données&quot; dans la navigation globale de CodeStream pour accéder au générateur de requêtes.

<img title="Query your data." alt="A screenshot of the query builder." src="/images/codestream_screenshot-crop_query-builder.webp" />

En fonction des données que vous avez spécifiées, le générateur de requêtes sélectionne le type de graphique qui affiche vos résultats le plus efficacement. Cependant, vous pouvez utiliser la liste déroulante au-dessus des résultats pour choisir parmi d&apos;autres types de graphiques disponibles afin de présenter vos données dans le format que vous souhaitez utiliser. Vous pouvez également exporter les données au format CSV ou JSON, ce qui ouvrira un nouvel onglet d&apos;éditeur et affichera les données dans le format sélectionné.

## Fichiers NRQL [#nrql-files]

La liste déroulante au-dessus du générateur de requêtes, à droite, vous permet de sélectionner la requête que vous avez récemment exécutée. Une autre façon de sauvegarder la requête est d&apos;ajouter un fichier avec une extension `.nrql` à votre référentiel. Vous pouvez enregistrer votre requête dans ce fichier et même partager ces requêtes avec vos coéquipiers en validant et en poussant ce fichier. Cliquez simplement sur le lien « Exécuter » dans CodeLens au-dessus de chaque requête pour exécuter la requête.
