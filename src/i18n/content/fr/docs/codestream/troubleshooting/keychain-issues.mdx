Si vous devez vous connecter à New Relic CodeStream à chaque redémarrage de votre IDE JetBrains ou ouverture d&apos;un nouveau projet, il est probable que l&apos;IDE n&apos;enregistre pas et/ou ne récupère pas le jeton d&apos;authentification de votre coffre-fort de mots de passe local.

[Cet article JetBrains](https://www.jetbrains.com/help/idea/reference-ide-settings-password-safe.html) fournit des détails sur la manière dont vous pouvez vérifier qu&amp;apos;il est correctement configuré.

Par exemple, si vous êtes sur un Mac, l&apos;option <DNT>**in native Keychain**</DNT> est celle que vous souhaiterez utiliser, comme indiqué dans la capture d&amp;apos;écran ci-dessous.
