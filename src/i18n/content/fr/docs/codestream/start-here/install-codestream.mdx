Vous pouvez installer l&apos;extension CodeStream pour votre IDE spécifique.

<CollapserGroup>
  <Collapser className="freq-link" id="vscode" title="Code de Visual Studio">
    Téléchargez et installez CodeStream pour [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream). Vous pouvez également l&apos;installer directement dans Visual Studio Code via la place de marché des extensions.
  </Collapser>

  <Collapser className="freq-link" id="visual-studio" title="Visual Studio">
    Téléchargez et installez CodeStream pour [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22). Vous pouvez également l&apos;installer directement dans Visual Studio via la place de marché des extensions.
  </Collapser>

  <Collapser className="freq-link" id="jetbrains" title="JetBrains">
    Téléchargez et installez CodeStream pour [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream). Vous pouvez également l&apos;installer à partir du menu des plugins JetBrains.
  </Collapser>
</CollapserGroup>

Si vous utilisez VS Code, certaines fonctionnalités CodeStream nécessitent une extension de prise en charge linguistique pour [Java](https://marketplace.visualstudio.com/items?itemName=redhat.java), [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp), [PHP](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client), [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python), [Go](https://marketplace.visualstudio.com/items?itemName=golang.go) ou [Ruby](https://marketplace.visualstudio.com/items?itemName=Shopify.ruby-lsp).

## Se connecter [#signin]

Une fois l’extension CodeStream installée dans votre IDE, connectez-vous avec vos informations d’identification New Relic.
