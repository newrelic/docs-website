New Relic CodeStream est une extension d&apos;IDE qui vous permet de vous décaler vers la gauche en faisant de la performance du code une partie des premières étapes du processus de développement.

* Découvrez comment les services construits à partir de votre code fonctionnent, avec des informations sur les indicateurs clés, les erreurs, les anomalies de transaction, les SLO et plus encore.
* Investiguez les erreurs plus rapidement en parcourant la trace de la pile et en collaborant avec vos coéquipiers.
* Recherchez les logs depuis votre IDE pour enquêter sur les problèmes sans changer de contexte.
* Exécutez des requêtes NRQL, et enregistrez et partagez des requêtes avec vos coéquipiers via des fichiers `.nrql`.
* Consultez les données de performance pour les méthodes individuelles dans le code, directement dans l&apos;éditeur.

## Investigation des erreurs

Découvrez les erreurs qui se produisent dans le code dont vous êtes responsable. Parcourez rapidement la trace de la pile, CodeStream vous emmenant automatiquement vers le fichier et le numéro de ligne corrects, puis amenez les coéquipiers concernés dans la discussion afin que vous puissiez résoudre le problème le plus rapidement possible.

## Recherche de logs

Si vous devez enquêter sur un problème en recherchant des logs, pourquoi changer de contexte hors de votre IDE pour faire ce travail sur le Web ? Recherchez les logs directement depuis votre IDE. Vous pouvez même lancer une recherche à partir de lignes de logs individuelles dans votre code.

## Métriques au niveau du code

Consultez les données de performance, au niveau de la méthode, directement dans votre éditeur. La vue toujours active garantit que les problèmes de performance sont pris en compte dans le travail en cours. Vous pouvez même consulter les indicateurs de vos environnements inférieurs et anticiper les problèmes de performance avant qu&apos;ils n&apos;atteignent la production.

## Monitoring des performances

Obtenez une vue dans l&apos;IDE de la façon dont les services construits à partir de votre code fonctionnent, dans n&apos;importe quel environnement. Les indicateurs clés, les objectifs de niveau de service et les services associés vous donnent une vue d&apos;ensemble en un coup d&apos;œil.
