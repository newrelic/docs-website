Infinite Tracing est une solution cloud entièrement gérée qui peut analyser 100 % de vos données trace et choisir les données les plus exploitables, vous permettant d&apos;enquêter et de résoudre les problèmes rapidement.

Ce document s&apos;applique uniquement à la fonctionnalité Infinite Tracing. Si vous recherchez notre solution de tracing distribuée standard, consultez nos [options de configuration](/docs/distributed-tracing/enable-configure/quick-start).

## Qu&apos;est-ce que l&apos;Infinite Tracing ? [#understand-infinite-tracing]

Infinite Tracing vous permet d&apos;envoyer toutes vos données de trace à notre service cloud et de vous fier à Infinite Tracing pour prendre des décisions d&apos;échantillonnage. Vous pouvez configurer Infinite Tracing de différentes manières pour garantir qu&apos;il conserve les données de trace que vous devez voir.

Contrairement à nos options de tracing distribué standard, Infinite Tracing peut traiter plus de données trace . Il utilise [un échantillonnage en fin de workflow](/docs/understand-dependencies/distributed-tracing/get-started/how-new-relic-distributed-tracing-works#sampling) supérieur (échantillonnage après la collecte des données), par opposition à l&apos;échantillonnage en début de workflow utilisé par notre fonctionnalité de tracing standard.

Ressources pour en savoir plus sur Infinite Tracing :

* [Détails techniques sur l&apos;échantillonnage et l&apos;architecture](/docs/understand-dependencies/distributed-tracing/get-started/how-new-relic-distributed-tracing-works#sampling)

## Exigences [#requirements]

Les exigences diffèrent selon votre [modèle de tarification](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-pricing-models):

* Modèle de tarification basé sur l&apos;utilisation : nécessite Pro ou édition Enterprise.
* Modèle de tarification d&apos;origine : nécessite l&apos;aide de New Relic pour l&apos;activer pour votre organisation. Pour toute question, contactez votre représentant de compte New Relic.

La possibilité de configurer cette fonctionnalité est régie par [les autorisations liées au tracing infini](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions#infinite-tracing).

## Activer Infinite Tracing [#enable-infinite-tracing]

Lorsque vous activez Infinite Tracing, vous devez idéalement l&apos;activer pour tous les services associés. Si vous avez activé un mélange d&apos;Infinite Tracing et de nos solutions de tracing standard, les traces présenteront [des problèmes de conflit de configuration](/docs/understand-dependencies/distributed-tracing/troubleshooting/infinite-tracing-trace-configuration-conflicts).

Les instructions pour configurer Infinite Tracing se trouvent dans les documents spécifiques de nos solutions. Pour commencer, consultez notre [guide de démarrage rapide](/docs/distributed-tracing/enable-configure/quick-start).

## Configurer le Infinite Tracing [#configure]

Après avoir activé Infinite Tracing, vous pouvez le configurer de différentes manières pour vous assurer qu&apos;il conserve les données souhaitées. Voir [Configurer](/docs/configure-trace-observer#configure).
