La CLI New Relic permet l&apos;intégration de New Relic dans vos workflows existants. Qu&apos;il s&apos;agisse de récupérer des données depuis votre ordinateur portable lors du dépannage d&apos;un problème ou d&apos;ajouter New Relic à votre pipeline CI/CD.

## Commandes CLI de New Relic [#nr-cli-commands]

Recherchez les détails de la documentation de la commande CLI New Relic sur GitHub.

### Options [#nr-cli-commands-options]

```bash copyable=false
[output]   -a, --accountId int    the account ID to use. Can be overridden by setting NEW_RELIC_ACCOUNT_ID
[output]       --debug            debug level logging
[output]       --format string    output text format [JSON, Text, YAML] (default "JSON")
[output]   -h, --help             help for newrelic
[output]       --plain            output compact text
[output]       --profile string   the authentication profile to use
[output]       --trace            trace level logging
```

### Commandes [#commands]

* [newrelic apm](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_apm.md). Interagissez avec New Relic APM.

* [achèvement de newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_completion.md). Génère des fonctions de complétion de shell.

* [configuration de newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_config.md). Gérer la configuration de la CLI New Relic.

* [décodage newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_decode.md). Utilisez la commande decode url pour imprimer les informations cryptées dans l&apos;URL.

* [documentation de newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_documentation.md). Générer la documentation CLI.

* [entité newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_entity.md). Interagissez avec l&apos;entité New Relic.

* [nouveau nerdgraph de relic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_nerdgraph.md). Exécutez GraphQL requests vers l&apos;API NerdGraph.

* [stockage de nerd newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_nerdstorage.md). Lisez, écrivez et supprimez des documents et des collections NerdStorage.

* [newrelic nrql](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_nrql.md). Commandes pour interagir avec la base de données New Relic.

* [profil newrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_profile.md). Gérer les profils d&apos;authentification pour cet outil.

* [newrelic version](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_version.md). Afficher la version de la CLI New Relic.

* [workloadnewrelic](https://github.com/newrelic/newrelic-cli/blob/master/docs/cli/newrelic_workload.md). Interagissez avec la charge de travail de New Relic.
