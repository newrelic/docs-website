Si une application est organisée, elle est plus lisible et plus compréhensible. Dans votre application de test A/B, vous avez beaucoup de graphiques, mais aucune organisation. Dans les prochaines leçons, vous utiliserez des composants d&apos;interface utilisateur pour apporter une certaine organisation à votre application et fournir de nouvelles fonctionnalités. Tout d’abord, vous utiliserez un composant `Grid` pour organiser vos graphiques afin qu’ils correspondent à la mise en page de votre guide de conception. Deuxièmement, vous ajouterez des titres pour clarifier ce que représente chaque graphique. Troisièmement, vous ajouterez des descriptions pour vos versions de conception A et B en haut de votre application. Enfin, vous créerez une section pour terminer votre test A/B.

Dans la leçon suivante, vous organisez vos graphiques pour qu’ils ressemblent à ceux de votre guide de conception.
