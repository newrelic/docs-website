Imaginez que vous avez développé un site Web et que vous l&apos;avez instrumenté avec monitoring des navigateurs de New Relic. L&apos;ensemble de fonctionnalités de base de New Relic fournit de nombreuses informations, mais vous voulez quelque chose de personnalisé. Vous souhaitez décider si un nouveau design pour votre site permettra de mieux engager votre utilisateur. En d’autres termes, vous souhaitez tester A/B une conception et prendre une décision basée sur les données.

Pour cette tâche, vous pouvez créer une application New Relic, en utilisant React, le kit de développement logiciel New Relic One et la puissance illimitée des technologies Web modernes.

Les applications New Relic sont construites avec l&apos;une des bibliothèques JavaScript les plus populaires du développement Web : React. Parce que vous avez la liberté d&apos;écrire du code React, vous pouvez personnaliser la logique de votre application, concevoir vos propres composants ou profiter de l&apos;abondance de la bibliothèque de composants open source . Ainsi, pour votre application de test A/B, si vous souhaitez écrire une logique personnalisée pour terminer le test en fonction des résultats, vous pouvez le faire.

Le kit de développement logiciel (SDK) New Relic One vous permet de créer, servir, publier et déployer des applications sur la plateforme New Relic. Il fournit également une multitude de composants React pour la collecte de données, la présentation d&apos;informations, la gestion des interactions utilisateur, etc. Vous utilisez des composants tels que `Button` et `Dropdown` pour créer une expérience interactive qui ressemble et fonctionne de manière native à New Relic. Vous utilisez les composants `Table` et `Chart` pour afficher les données de votre compte New Relic ou d&apos;ailleurs. Lors de la création de votre application de test A/B, vous utiliserez le composant `NrqlQuery` du SDK pour récupérer les données du navigateur à partir de votre compte.

Avec du code React personnalisé, des composants SDK et le vaste monde de la bibliothèque open source , vous pouvez créer votre application de test A/B dans New Relic. Mais avant d’en créer un pour vous-même, vous devriez peut-être consulter le catalogue d’applications pour voir si quelqu’un vous a devancé ! Si le catalogue disposait déjà d&apos;une application pour cela, vous pourriez l&apos;ajouter à votre compte en quelques clics, un autre avantage de la création d&apos;applications dans New Relic.

Tout au long de ce cours, vous allez créer une application New Relic concrète pour exécuter et gérer les tests A/B. Vous visualiserez les données du navigateur pour vos conceptions concurrentes, verrez les données historiques des tests passés et choisirez même une conception gagnante et terminerez le test, le tout à partir de votre application New Relic ! Mais avant de vous lancer dans la création de graphiques et requests http, vous devez apprendre de quoi sont faites les applications New Relic.
