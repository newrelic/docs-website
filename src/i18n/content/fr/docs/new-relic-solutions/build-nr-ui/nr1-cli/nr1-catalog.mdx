## `nr1 catalog:info` [#nr1-cataloginfo]

### Obtenir les détails du catalogue [#nr1-cataloginfo-details]

Affiche les informations sur votre application qui sont affichées dans le catalogue.

Si exécuté dans un dossier Nerdpack spécifique, les informations de ce Nerdpack seront affichées. Si vous ne souhaitez pas obtenir d&apos;informations de votre Nerdpack local, utilisez l&apos;option `--nerdpack-id=NERDPACK_ID` pour effectuer une requête à partir d&apos;un Nerdpack spécifique.

### Usage [#nr1-cataloginfo-usage]

`$ nr1 catalog:info`

### Options [#nr1-cataloginfo-options]

<table>
  <tbody>
    <tr>
      <td>
        `-f, --field=FIELD`
      </td>

      <td>
        Spécifie le champ à partir duquel vous souhaitez obtenir des informations.
      </td>
    </tr>

    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        Spécifie de quel Nerdpack obtenir les informations.
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        Le profil d’authentification que vous souhaitez utiliser.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        Ajoute des informations supplémentaires à la sortie.
      </td>
    </tr>
  </tbody>
</table>

## `nr1 catalog:submit` [#nr1-catalogsubmit]

### Envoyer des informations au catalogue [#nr1-catalogsubmit-send]

Rassemble les informations que vous ajoutez au répertoire du catalogue pour votre application et les enregistre dans le catalogue.

Cette commande doit être exécutée sur un dossier Nerdpack. La commande recherchera des fichiers spécifiques en utilisant des noms de convention.

### Usage [#nr1-catalogsubmit-usage]

`$ nr1 catalog:submit`

### Options [#nr1-catalogsubmit-options]

<table>
  <tbody>
    <tr>
      <td>
        `-P, --skip-screenshots`
      </td>

      <td>
        Ignore le téléchargement des ressources de capture d&apos;écran.
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        Le profil d’authentification que vous souhaitez utiliser.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        Ajoute des informations supplémentaires à la sortie.
      </td>
    </tr>
  </tbody>
</table>
