Les info-bulles sont utilisées pour exposer des informations supplémentaires qui ne sont pas facilement disponibles dans l&apos;interface utilisateur. Vous pouvez les utiliser pour expliquer ce qu&apos;est un élément d&apos;interface utilisateur, expliquer le but d&apos;une action ou fournir une définition d&apos;un mot ou d&apos;une phrase.

### Usage

```jsx
import { Tooltip } from 'nr1'
```

### Exemples

#### Basique

```jsx
<Tooltip text="Search across all your accounts">
  <Button>Search</Button>
</Tooltip>
```

#### Avec placementType

```jsx
<Tooltip
  text="Explore your metrics, events and logs"
  placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
>
  <Button>Query your data</Button>
</Tooltip>
```

#### Avec un lien d&apos;informations supplémentaires

```jsx
function render() {
  const additionalInfoLink = {
    label: 'See the docs',
    to: 'https://docs.newrelic.com',
  };

  return (
    <p>
      Welcome to
      <Tooltip
        text="Go to docs for more information"
        placementType={Tooltip.PLACEMENT_TYPE.BOTTOM}
        additionalInfoLink={additionalInfoLink}
      >
        New Relic
      </Tooltip>
    </p>
  );
}
```
