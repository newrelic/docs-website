Affiche une cellule de tableau de lignes montrant un utilisateur particulier. Cette cellule prend l&apos;objet utilisateur comme enfant, ce qui est à son tour compatible avec l&apos;objet utilisateur NerdGraph (nécessitant, au minimum, les champs `name` et `gravatar` ).

### Usage

```jsx
import { UserTableRowCell } from 'nr1'
```

### Exemples

#### Exemple 1

```jsx
function render() {
  const items = [
    {
      name: "John Doe",
      gravatar:
        "https://secure.gravatar.com/avatar/79988df068cb4d952284a3395412dc4f",
    },
  ];

  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>User</TableHeaderCell>
      </TableHeader>

      {({ item }) => (
        <TableRow>
          <UserTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```
