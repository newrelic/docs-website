Une tag littérale de modèle JavaScript qui analyse les chaînes de requête GraphQL dans des documents GraphQL qui peuvent être utilisés pour effectuer une requête NerdGraph.

### Usage

```js
import { ngql } from 'nr1'
```

### Exemples

#### Exemple 1

```js
import { ngql } from 'nr1';


const query = ngql`
  {
    actor {
      user {
        id
        email
        name
      }
    }
  }
`;


function render() {
  return (
    <NerdGraphQuery query={query}>
      {({ data }) => JSON.stringify(data, null, 2)}
    </NerdGraphQuery>
  );
}
```
