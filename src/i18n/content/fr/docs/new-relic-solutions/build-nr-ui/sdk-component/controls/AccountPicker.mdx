Récupère tous les comptes et permet la sélection de comptes via la liste déroulante et la recherche. Pour capturer le compte actuellement sélectionné, vous pouvez utiliser l&apos;écouteur `onChange`.

Notez que la plateforme contient déjà son propre sélecteur de compte, qui se trouve dans l&apos;en-tête de celle-ci. Celui-ci peut être configuré pour être affiché à l&apos;aide de l&apos;API `nerdlet.setConfig` (en passant `{ accountPicker: true }`), et sa valeur peut être lue à l&apos;aide de `<PlatformStateContext.Consumer>`. Pour une référence complète, veuillez consulter ces API.

### Usage

```jsx
import { AccountPicker } from 'nr1'
```

### Exemples

#### Basique

```jsx
class Example extends React.Component {
  constructor() {
    super(...arguments);

    this.state = { accountId: null };

    this.onChangeAccount = this.onChangeAccount.bind(this);
  }

  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);

    this.setState({ accountId: value });
  }

  render() {
    return (
      <AccountPicker
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```

#### Avec étiquette en ligne

```jsx
class Example extends React.Component {
  constructor() {
    super(...arguments);

    this.state = { accountId: null };

    this.onChangeAccount = this.onChangeAccount.bind(this);
  }

  onChangeAccount(_, value) {
    alert(`Selected account: ${value}`);

    this.setState({ accountId: value });
  }

  render() {
    return (
      <AccountPicker
        label="Account"
        labelInline
        value={this.state.accountId}
        onChange={this.onChangeAccount}
      />
    );
  }
}
```
