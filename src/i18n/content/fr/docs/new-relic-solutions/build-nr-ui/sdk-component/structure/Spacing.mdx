Fournit un espacement à tout composant enveloppé. L&apos;espacement est fourni sous la forme d&apos;un uplet pouvant contenir zéro, une, deux, trois ou quatre valeurs. Chacune de ces valeurs provient de la propriété statique TYPE.

La structure de l&apos;uplet est très similaire au fonctionnement des propriétés `margin` ou `padding` : spécifier une valeur les réplique sur les quatre côtés, deux sur `top`/`bottom` et `left`/`right`, etc. Si vous souhaitez omettre une valeur spécifique tout en transmettant d&apos;autres valeurs, vous pouvez utiliser `OMIT` comme espace réservé. L&apos;espace sur le côté (ou les côtés) où il est utilisé sera laissé intact.

D’autres composants du SDK ont des capacités d’espacement intégrées. Si vous souhaitez reproduire la même capacité sur votre propre composant, vous pouvez utiliser `Spacing.add` et transmettre le composant ; le composant renvoyé par la méthode contiendra l&apos;accessoire `type` ainsi que les statiques.

### Usage

```js
import { Spacing } from 'nr1'
```

        Propriété d&apos;espacement. L&apos;espacement est défini comme un uplet de zéro à quatre valeurs, qui suivent les mêmes conventions que les propriétés CSS comme `margin` ou `padding`. Pour omettre une valeur, utilisez `TYPE.OMIT`.

        <OptionReference array>
          Spacing.TYPE.EXTRA\_LARGE, Spacing.TYPE.LARGE, Spacing.TYPE.MEDIUM, Spacing.TYPE.NONE, Spacing.TYPE.OMIT, Spacing.TYPE.SMALL,
        </OptionReference>

        Basique

        ```jsx
        <Spacing type={[Spacing.TYPE.MEDIUM]}>
          <Button>Spaced button</Button>
        </Spacing>;            
        ```

        Omettre les marges

        ```jsx
        <Spacing type={[Spacing.TYPE.OMIT, Spacing.TYPE.LARGE]}>
          <Button>Spaced button (only left and right)</Button>
        </Spacing>; 
        ```
      </td>
