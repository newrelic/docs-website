Élément enfant du composant `<Tabs>` .

Il crée un onglet où le titre de l&apos;onglet est défini par la propriété `label` et le contenu de l&apos;onglet est défini par la propriété `children` .

### Usage

```js
import { TabsItem } from 'nr1'
```
