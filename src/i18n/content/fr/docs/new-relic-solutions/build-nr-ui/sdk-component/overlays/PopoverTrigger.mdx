Élément enfant du composant `<Popover>` .

Contrôle l&apos;ouverture/fermeture de la superposition Popover.

### Usage

```js
import { PopoverTrigger } from 'nr1'
```

### Accessoires

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>OBLIGATOIRE</h5> <h5>Nœud|fonction</h5>
      </td>

      <td>
        Ce composant peut s&apos;afficher de deux manières, en passant directement un élément React enfant ou en passant un rappel de rendu (fonction en tant qu&apos;enfants). Lorsque vous passez directement un élément React, vous pouvez soit utiliser certains des composants SDK NR1 compatibles comme `<TextField>`, `<Button>`... ou utiliser votre propre composant. Lorsque vous utilisez un rappel de rendu, vous recevrez un objet de la forme suivante comme argument `{ opened: boolean, controlled: boolean }`. Peu importe si vous passez des composants directement ou si vous utilisez un rappel de rendu, lors du rendu de votre propre composant en tant que déclencheur, il doit toujours exposer une méthode `focus` afin de fonctionner correctement.
      </td>
