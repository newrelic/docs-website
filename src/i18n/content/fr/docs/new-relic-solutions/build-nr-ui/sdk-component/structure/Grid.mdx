Grid est un système de mise en page bidimensionnel, ce qui signifie qu&apos;il peut gérer à la fois les colonnes et les lignes, ce qui vous aide à concevoir une interface utilisateur basée sur une grille, contrairement à `<Stack>` qui est en grande partie un système unidimensionnel.

### Usage

```js
import { Grid } from 'nr1'
```
