La notification Toast donne à l&apos;utilisateur une confirmation immédiate de la réussite d&apos;une tâche ou d&apos;un échec critique lors de la tentative d&apos;achèvement d&apos;une tâche.

Vous pouvez personnaliser le toast en passant des actions personnalisées qui s&apos;afficheront sous forme de boutons cliquables à côté du texte.

### Usage

```js
import { Toast } from 'nr1'
```

### Exemples

#### Notification toast de base

```js
function render() {
  const onClick = () =>
    Toast.showToast({
      title: 'Feedback received',
      description: 'We will contact you soon.',
      actions: [
        {
          label: 'Go back',
          onClick: () => console.log('Go back'),
        },
      ],
      type: Toast.TYPE.NORMAL,
    });


  return <Button onClick={onClick}>Show toast</Button>;
}
```

#### Avec un lien d&apos;informations supplémentaires

```js
function render() {
  const onClick = () =>
    Toast.showToast({
      title: 'TV Mode activated',
      description: 'Press ESC to exit',
      additionalInfoLink: {
        label: 'Learn more about TV Mode',
        to: 'https://discuss.newrelic.com/t/tv-mode-in-new-relic-one/92963',
      },
      type: Toast.TYPE.NORMAL,
    });


  return <Button onClick={onClick}>Show toast</Button>;
}
```

#### Notification toast critique

```js
function render() {
  const onClick = () =>
    Toast.showToast({
      title: 'Service unavailable',
      description: 'Your data could not be retrieved',
      actions: [
        {
          label: 'Retry',
          onClick: () => console.log('Retry'),
        },
      ],
      type: Toast.TYPE.CRITICAL,
    });


  return <Button onClick={onClick}>Update</Button>;
}
```
