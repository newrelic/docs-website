Wrapper pour les cellules composant une seule ligne.

### Usage

```js
import { TableRow } from 'nr1'
```

### Accessoires

<table>
  <tbody>
    <tr>
      <td>
        `actions` <h5>forme\[]</h5>
      </td>

      <td>
        Définit un ensemble d&apos;actions qui apparaîtront lorsque l&apos;utilisateur survolera la ligne. Les actions ont un texte obligatoire et un rappel onClick (qui sera appelé avec l&apos;élément et l&apos;index de la ligne à laquelle elles appartiennent), et elles acceptent également une propriété désactivée.

        <h3>forme</h3>

        `disabled` <h5>booléen</h5>

        `label` <h5>OBLIGATOIRE</h5><h5>chaîne</h5> Utilisez-la pour décrire l&apos;action de la ligne.

        `onClick` <h5>OBLIGATOIRE</h5><h5>fonction</h5> Rappel déclenché à chaque fois que l&apos;utilisateur clique sur l&apos;action.

        <FunctionDefinition returnValue={[]} arguments={[{"name":"event","type":"React.MouseEvent","description":""}]} />

        `testId` <h5>chaîne</h5> Ajoute un attribut `data-test-id` . Utilisez-le pour cibler le composant dans les tests unitaires et E2E. Pour qu&apos;un identifiant de test soit valide, préfixez-le avec votre identifiant nerdpack, suivi d&apos;un point. Par exemple, `my-nerdpack.some-element`. **Remarque :** vous ne verrez peut-être pas l&apos;attribut `data-test-id` car ils sont supprimés du DOM. Pour les déboguer, transmettez un paramètre de requête `e2e-test` à l&apos;URL.

        `type` <h5>énum.</h5> Type d&apos;action pour chacun des éléments de la liste déroulante des actions. Ce n&apos;est pas obligatoire, sa valeur par défaut est `TableRow.ACTION_TYPE.NORMAL`.

        * Normal — action générique non critique.

        * Destructeur — à utiliser lorsque vous avez une action destructrice comme supprimer ou retirer, que vous souhaitez que l&apos;utilisateur interrompe et considère avant de terminer.

          <OptionReference>
            TableRow\.ACTION\_TYPE.DESTRUCTIVE, TableRow\.ACTION\_TYPE.NORMAL,
          </OptionReference>

        ```js
        const actions = [
          {
            label: 'Duplicate',
            type: TableRow.ACTION_TYPE.NORMAL,
          },
          {
            label: 'Delete',
            type: TableRow.ACTION_TYPE.DESTRUCTIVE,
          },
        ];
        ```

        ```js
        const actions = [
          {
            label: 'Action 1',
            disabled: true,
            onClick: (evt, { item, index }) => {
              alert(`Action 1:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
          {
            label: 'Action 2',
            onClick: (evt, { item, index }) => {
              alert(`Action 2:Item: ${index}${JSON.stringify(item, null, 2)}`);
            },
          },
        ];
        ```
      </td>
    </tr>

      </td>
