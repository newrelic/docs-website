Affiche une cellule de tableau de lignes affichant le titre d&apos;une entité particulière. Cette cellule prend l&apos;objet entité comme enfant, ce qui est à son tour compatible avec l&apos;objet entité NerdGraph (nécessitant, au minimum, les champs `name` et `reporting` ). Si vous pouvez être alerté, vous devez également demander `alertSeverity`.

### Usage

```jsx
import { EntityTitleTableRowCell } from 'nr1'
```

### Exemples

#### Basique

```jsx
function render() {
  const items = [
    {
      name: "Login service",
      guid: "MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx",
      alertSeverity: "CRITICAL",
      reporting: true,
    },
  ];

  return (
    <Table items={items}>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>

      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} />
        </TableRow>
      )}
    </Table>
  );
}
```

#### Avec une valeur ajoutée

```jsx
function render() {
  const items = [
    {
      name: "Login service",
      guid: "MTIzNDU2fEZPT3xCQVJ8OTg3NjU0MzIx",
      alertSeverity: "CRITICAL",
      reporting: true,
    },
  ];

  return (
    <Table items={items} multivalue>
      <TableHeader>
        <TableHeaderCell>Entity</TableHeaderCell>
      </TableHeader>

      {({ item }) => (
        <TableRow>
          <EntityTitleTableRowCell value={item} additionalValue={item.guid} />
        </TableRow>
      )}
    </Table>
  );
}
```
