Les onglets sont utilisés pour aider à regrouper divers contenus connexes dans des sections distinctes. Ces sections peuvent être toutes ou contenues dans une partie de l&apos;écran. Nous vous recommandons d&apos;utiliser un ensemble d&apos;onglets par écran et de ne l&apos;utiliser que pour différents types de contenu associés. Par exemple, lorsque vous disposez de données différentes sur une seule entité, vous pouvez regrouper les données dans différentes sections.

NE PAS séparer une action, comme créer, des autres sections de contenu dans les onglets. L&apos;action doit être déclenchée par une icône ou un bouton et ouvrir une page modale ou empilée.

### Usage

```js
import { Tabs } from 'nr1'
```

### Exemples

#### Basique

```jsx
<Tabs defaultValue="tab-3">
  <TabsItem value="tab-1" label="Tab 1">
    Tab 1 content
  </TabsItem>
  <TabsItem value="tab-2" label="Another tab with a longer name">
    Tab 2 content
  </TabsItem>
  <TabsItem value="tab-3" label="A tab open by default">
    Tab 3 content
  </TabsItem>
</Tabs>
```

#### Rendre rappel

```jsx
function render() {
  const items = new Array(10000).fill().map((_, i) => ({
    value: i,
    content: `Tab ${i} content...`,
    label: `Item ${i}`,
  }));

  return (
    <div className="nr1-Example--tabs">
      <Tabs items={items}>
        {({ item }) => (
          <TabsItem value={item.value} label={item.label}>
            {item.content}
          </TabsItem>
        )}
      </Tabs>
    </div>
  );
}
```
