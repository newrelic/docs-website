Wrapper pour les cellules composant l&apos;en-tête. L&apos;en-tête doit toujours être présent et être le premier enfant immédiat de l&apos;élément `<Table>` .

### Usage

```js
import { TableHeader } from 'nr1'
```

  <tbody>
    <tr>
      <td>

      <td>
        Liste des en-têtes de colonnes, sous la forme de `<TableHeaderCell>`s. Les cellules doivent être des enfants directs de l&apos;en-tête, c&apos;est-à-dire qu&apos;il ne doit y avoir aucun nœud entre `<TableHeader>` et `<TableHeaderCell>` .
