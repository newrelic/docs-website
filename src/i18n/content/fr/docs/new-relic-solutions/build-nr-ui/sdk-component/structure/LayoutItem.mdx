Élément enfant du composant `<Layout>` .

Par défaut, le type principal est utilisé pour couvrir toute la largeur de la zone de mise en page. La division à gauche et la division à droite fournissent une zone gauche/droite fixe. La mise en page ne prend en charge qu&apos;un seul type de chaque.

### Usage

```js
import { LayoutItem } from 'nr1'
```
