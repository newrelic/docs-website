Crée un histogramme. Les données peuvent être obtenues en effectuant une requête NRQL sur un compte particulier ou elles peuvent être transmises via la propriété `data`.

Le format `data` est une série d&apos;objets, chacun contenant `metadata` et `data`. Chaque `data` contient à la fois les valeurs de ce point et les clés `x` et `y` utilisées pour tracer le graphique.

Si vous souhaitez afficher un histogramme avec plusieurs séries, pensez à utiliser `HeatmapChart`.

### Usage

```jsx
import { HistogramChart } from 'nr1'
```

### Exemples

#### Basique

```jsx
<HistogramChart
  accountIds={[1]}
  query="SELECT histogram(duration, width:10, buckets:20) FROM PageView SINCE 1 month ago"
/>
```

#### Avec plusieurs comptes

```jsx
<HistogramChart
  accountIds={[1, 1067061]}
  query="SELECT histogram(duration, width:10, buckets:20) FROM PageView SINCE 1 month ago"
/>
```

#### Remplir le conteneur

```jsx
<HistogramChart
  accountIds={[1]}
  query="SELECT histogram(duration, width:10, buckets:20) FROM PageView SINCE 1 month ago"
  fullWidth
  fullHeight
/>
```
