Certaines [solutions New Relic](https://newrelic.com/instant-observability/) nécessitent l&apos;installation manuelle d&apos;un agent. La plupart des agents incluent des outils intégrés pour générer des logs détaillés pour le dépannage, l&apos;audit et les diagnostics.

Documents associés :

* Pour le dépannage général de l&apos;agent, voir [Ne pas voir les données](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data).
* Découvrez [New Relic Diagnostics](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics): un utilitaire qui détecte automatiquement les problèmes courants.
