L&apos;interface de ligne de commande de diagnostic (`nrdiag`) est un utilitaire qui détecte automatiquement les problèmes courants avec les produits New Relic. Si l’interface de ligne de commande de diagnostic détecte un problème, elle suggère des étapes de dépannage. L&apos;interface de ligne de commande de diagnostic peut télécharger automatiquement les données de dépannage sur un compte New Relic.

L&apos;interface de ligne de commande de diagnostic est open source et se trouve dans [GitHub](https://github.com/newrelic/newrelic-diagnostics-cli).

Pour des étapes de dépannage supplémentaires pour votre agent, consultez [Ne pas voir les données](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data).

## Compatibilité

L&apos;interface de ligne de commande de diagnostic est disponible pour Linux, macOS et Windows. Il peut détecter les problèmes de configuration courants pour :

* <DNT>**APM**</DNT>:Disponible pour tous les <InlinePopover type="apm" />agents. Pour l&apos;agent Go, seules les vérifications de connectivité de base sont disponibles.
* <DNT>**Browser monitoring**</DNT>: Détection pour l&apos;agent <InlinePopover type="browser" />.
* <DNT>**Infrastructure monitoring**</DNT>: Agents Linux et Windows
* <DNT>**Mobile agents**</DNT>: iOS et Android
* <DNT>**Synthetic monitoring**</DNT>: Minions privés conteneurisés (CPM)
* <DNT>**Kubernetes**</DNT>: Kubernetes, contrôle des agents, Helm et Flux

L&apos;interface de ligne de commande de diagnostic ne nécessite pas les autorisations `superuser` ou `admin` pour s&apos;exécuter, bien que nous recommandons ces autorisations pour certaines vérifications. Il renverra une erreur s&apos;il n&apos;a pas les autorisations nécessaires pour lire les fichiers qu&apos;il analyse.

## Démarrer

Pour utiliser l&apos;interface de ligne de commande de diagnostic :

1. Exécutez l’interface de ligne de commande de diagnostic, y compris [les suites de tâches](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/run-diagnostics-cli-nrdiag) et [les options de ligne de commande](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) selon vos besoins.
2. Fournissez l&apos;indicateur [`-attach` ou `-api-key`](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) pour télécharger les résultats sur votre compte New Relic.
3. Facultatif : Validez [les paramètres de votre fichier de configuration](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/validate-config-file-settings-nrdiag).
4. Interpréter la [sortie](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output).

Comme tout autre outil New Relic, le service CLI de diagnostic est conçu pour protéger la confidentialité de vos données et de celles de vos clients. Pour des informations détaillées, consultez notre documentation [sur les licences et la sécurité de Diagnostics CLI](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/diagnostics-cli-licensing-security) .
