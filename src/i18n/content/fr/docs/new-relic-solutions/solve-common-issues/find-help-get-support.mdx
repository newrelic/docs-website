## Accédez aux ressources d&apos;assistance sur notre plateforme [#nr-support]

Toutes les ressources [d&apos;assistance intégrées au produit](https://one.newrelic.com/redirect/support) sont accessibles lors de la navigation sur notre plateforme : sélectionnez simplement la petite icône <DNT>**?**</DNT> dans le menu de navigation en haut à droite.

La vue d&apos;assistance intégrée au produit offre une visibilité sur les nouveautés de New Relic, notre documentation et notre hub Explorers. La barre latérale d&apos;assistance offre un accès rapide à tous nos outils d&apos;assistance.

<img title="New Relic's in-product support" alt="In-product support" src="/images/solutions_screenshot-crop_product-support.webp" />

## Posez votre question dans Explorers Hub, notre forum d&apos;assistance gratuit [#nr-forum]

[Le New Relic&apos;s Explorers Hub](https://discuss.newrelic.com/) est notre forum gratuit pour tous les utilisateurs. Les utilisateurs et collaborateurs de New Relic s&apos;engagent chaque jour dans des conversations pour dépanner et résoudre les problèmes, trouver des solutions de contournement et discuter de nouvelles fonctionnalités.

## Exécutez l&apos;outil de diagnostic New Relic [#diagnostics]

[New Relic Diagnostics](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics) est notre outil de diagnostic automatisé pour Linux, Windows et Mac. S&apos;il détecte un problème avec l&apos;un de nos agents, il suggère des solutions et enregistre des logs de dépannage que vous pouvez joindre au ticket.

## Trouvez des réponses dans New Relic Docs et New Relic University [#find-answers]

[Le site de documentation](https://docs.newrelic.com) de New Relic contient des conseils utiles d&apos;installation, de configuration et de dépannage. Depuis la [page principale](http://docs.newrelic.com), sélectionnez parmi les catégories et sujets fréquemment utilisés, comme [les notes de sortie](/docs/release-notes). Ou [recherchez à partir de n&apos;importe quelle page](http://docs.newrelic.com/search).

Pour une bibliothèque de vidéos supplémentaires, de webinaires et d&apos;autres informations sur l&apos;utilisation des fonctionnalités de New Relic, visitez [New Relic University](https://learn.newrelic.com/) et [newrelic.com/resources](https://newrelic.com/resources/videos).

## Contribuez à notre documentation [#docs-contribution]

Sur le côté droit de chaque [document public,](https://docs.newrelic.com) vous trouverez un widget de commentaires. Utilisez-le pour nous indiquer si une page est utile, créer un problème ou modifier la page.

<img src="/images/solutions_screenshot-crop_docs-feedback.webp" style={{ width: '300px' }} alt="New Relic Docs - Feedback" title="New Relic Docs - Feedback" />

Notre documentation est open source et disponible sur [GitHub](https://github.com/newrelic/docs-website), et nous vous encourageons à y contribuer ! Nous veillons vraiment à ce que nos documents soient utiles, complets et précis. Pour modifier une page, cliquez sur le bouton <DNT>**Edit page**</DNT> dans n&apos;importe quel document pour créer une demande de tirage avec la modification que vous jugez nécessaire. Nous ne voulons pas que vous vous souciez du style. Lorsque vous modifiez un fichier, les rédacteurs techniques de notre équipe l&apos;examinent pour en vérifier le style, la grammaire et le formatage.

### Vous ne trouvez pas ce que vous cherchez ? Déposer une demande de documentation [#file-doc-issue]

Si vous ne trouvez pas de réponse dans la documentation, vous pouvez déposer un problème pour nous demander de l&apos;aide. Lorsque vous trouvez des endroits où la documentation pourrait être améliorée, faites-le nous savoir également ! Pour ce faire, cliquez sur le bouton <DNT>**Create issue**</DNT> dans n’importe quel document et nous examinerons votre problème pour trouver une solution.

## Déposer un dossier sur la plateforme [#file-ticket]

Si vous ne trouvez pas ce que vous cherchez et que votre niveau d&apos;abonnement inclut le support technique, déposez un ticket de support directement sur la plateforme :

1. Si vous faites partie d&apos;une organisation avec plusieurs comptes, sélectionnez votre compte dans la liste déroulante supérieure.

2. Au-dessus de votre nom en bas à gauche, cliquez sur [<DNT>Help &gt; Help &amp; Support</DNT>](https://one.newrelic.com/redirect/support).

   <img src="/images/solutions_screenshot-crop_help-button.webp" style={{ width: '300px' }} alt="Help button" title="Help button" />

3. Cliquez sur **Create a Support Case**.

   <img title="Screenshot showing the tile to create a ticket" alt="Screenshot showing the tile to create a ticket" src="/images/solutions_screenshot-crop_create-ticket.webp" style={{ width: '300px' }} />

4. Fournissez autant de détails que possible. Incluez l&apos;URL, le cas échéant, ou sélectionnez <DNT>**Attach file**</DNT> pour inclure un fichier log, un fichier de diagnostic New Relic, une capture d&apos;écran ou toute autre pièce jointe utile.

5. Cliquez sur <DNT>**Submit**</DNT>.

<Callout variant="important">
  La possibilité de créer une demande d&apos;assistance dépend du [plan d&apos;assistance](/docs/licenses/license-information/general-usage-licenses/global-technical-support-offerings/) de votre compte.
</Callout>
