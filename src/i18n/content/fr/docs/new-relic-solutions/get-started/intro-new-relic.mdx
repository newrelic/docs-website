Bienvenue sur la plateforme d&apos;observabilité New Relic ! Notre plateforme vous offre une visibilité approfondie sur votre système. Il vous aide à résoudre des problèmes dont vous n’aviez peut-être pas conscience, quelle que soit la technologie que vous utilisez. Vous pouvez voir comment New Relic fonctionne [ici](#how-it-works) ou [commencer](#get-started) maintenant si vous souhaitez vous lancer directement.

New Relic propose de nombreuses façons d&apos;augmenter la visibilité. Cela vous permet de monitorer, de dépanner et d&apos;améliorer les performances numériques. Voici un bref aperçu de nos capacités et fonctionnalités les plus populaires.

# Comment ça marche [#how-it-works]

La première étape de l&apos;observabilité nécessite de collecter de nombreuses données de performances, appelées données télémétriques. Notre plateforme d&apos;observabilité collecte les données de vos services grâce à des agents. Un agent est un fichier ou quelques lignes de code qui collecte des données sur votre service et les envoie à notre plateforme pour analyse. L&apos;instrumentation est le processus qui consiste à rendre quelque chose mesurable. Les agents s&apos;intègrent à un service dans le cadre de ce processus. Vous pouvez instrumenter un réservoir d’essence en installant un compteur qui mesure le degré de remplissage ou de vidage de votre réservoir. Vous monitorez votre réservoir ; lorsqu&apos;il est presque vide, vous savez qu&apos;il est temps de le remplir à nouveau.

New Relic fait quelque chose de similaire. Un agent est un fichier ou un morceau de code que vous attachez à un service particulier lors d&apos;une procédure d&apos;installation. À la manière d&apos;un compteur de gaz, nos agents suivent les performances du service en monitoring des mesures telles que le temps de réponse et le taux d&apos;erreur. En analysant ces données, vous pouvez obtenir des informations détaillées sur le fonctionnement de votre service et identifier les domaines à améliorer. Un agent est d&apos;abord installé dans votre environnement avec une configuration par défaut. Le processus d’installation, de configuration et d’instrumentation peut varier. Cela dépend du service auprès duquel vous souhaitez collecter des données.

Voici les agents de New Relic :

* **Agent APM**: pour les applications côté serveur

* **Agent Browser**: Pour les applications de navigateur

* **Agents d&apos;infrastructure**: Pour hôtes et intégration sur hôte

* **Agent mobile**: Pour les applications mobiles

Outre ces agents, nous prenons également en charge l&apos;intégration open source. Ils rapportent des données provenant de normes ouvertes pour l&apos;instrumentation. Ces normes incluent OpenTelemetry et des outils comme DropWizard, Prometheus, et bien d&apos;autres.

<Steps>
  <Step>
    ## Inscrivez-vous à New Relic si vous ne l&apos;avez pas déjà fait

    Prêt à commencer ? Si vous ne l&apos;avez pas déjà fait, [créez un compte New Relic](https://newrelic.com/signup). C&amp;apos;est gratuit, pour toujours.

    L&apos;inscription ne prend que quelques minutes. Une fois que vous avez un compte, vous pouvez commencer monitoring vos applications.
  </Step>

  <Step>
    ## Ajoutez vos données

    Ajoutez vos données à New Relic pour commencer à les analyser et à les comprendre. Notre guide d&apos;installation détectera votre environnement afin que vous n&apos;ayez plus qu&apos;à sélectionner le système que vous souhaitez monitorer.

    Cliquez sur le bouton <DNT>**Guided install**</DNT> . Ou, si votre organisation est située dans un [data centerde l’UE](/docs/using-new-relic/welcome-new-relic/get-started/our-eu-us-region-data-centers), cliquez sur <DNT>**EU Guided install**</DNT>.

    ## Interrogez vos données

Utilisez les dashboards pour fournir un contexte et une visibilité sur ce que fait votre système en temps réel.
  </Step>

  <Step>

Utilisez des alertes pour définir un seuil et recevoir une notification des changements de performances sur votre système. Cela vous aidera à anticiper les problèmes de performances pour éviter les impacts négatifs sur votre utilisateur. Gardez votre équipe concentrée et productive grâce à une gestion intelligente incident .
  </Step>
</Steps>
