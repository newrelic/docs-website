L&apos;interface utilisateur de New Relic prend en charge les deux versions de bureau les plus récentes des navigateurs les plus courants. Si vous utilisez une version de navigateur non prise en charge, vous verrez une erreur dans l&apos;interface utilisateur.

<Callout variant="tip">
  Pour connaître les exigences de notre <InlinePopover type="browser" />, voir [Compatibilité pour monitoring des navigateurs](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-new-relic-browser).
</Callout>

## Navigateurs pris en charge [#requirements]

Notre interface utilisateur prend en charge les deux versions de bureau les plus récentes pour ces navigateurs :

* Google Chrome : Linux, macOS, Windows
* Microsoft Edge : Windows
* Mozilla Firefox : Linux, macOS, Windows
  * Les versions Extended Support sortie (ESR) de Firefox ne sont pas prises en charge.
* Safari : macOS

Si la version de votre navigateur n&apos;est pas prise en charge, cela peut quand même fonctionner. Cependant, pour obtenir les meilleures fonctionnalités de notre site (et d&apos;autres), nous vous encourageons à passer à une version prise en charge.

## Support limité

Notre interface utilisateur offre une prise en charge limitée des navigateurs mobiles. Pour accéder à New Relic depuis un appareil mobile, installez notre [application Android](/docs/mobile-apps/new-relic-android-app/features/new-relic-android-app) ou [iOS](/docs/mobile-apps/new-relic-ios-app/features/new-relic-ios-app). Les workflows de nos applications mobiles sont spécifiquement conçus pour les cas d&apos;utilisation du téléphone. L&apos;interface Web n&apos;est pas activement prise en charge sur les navigateurs mobiles.
