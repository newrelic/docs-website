Certaines [intégrations New Relic](https://newrelic.com/instant-observability/) nécessitent l&apos;installation manuelle d&apos;un agent. La manière dont le comportement de ces agents est configuré dépend de l’agent spécifique.

## Configurer les agents APM [#apm-config]

Les seuls paramètres requis pour les agents <InlinePopover type="apm" />sont votre [clé de licence](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) et [le nom de l&apos;application](/docs/apm/new-relic-apm/installation-configuration/name-your-application), qui sont configurés lorsque vous [installez l&apos;agent](/docs/agents/manage-apm-agents/installation/install-agent).

Pour des options de configuration supplémentaires, voir :

* [Go](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [Java](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.NET](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## Configurer l’agent du navigateur [#browser-config]

Consultez notre documentation pour [la configuration<InlinePopover type="browser" />](/docs/browser/new-relic-browser/configuration).

## Configurer l’agent d’infrastructure [#infra-config]

Le seul paramètre requis pour notre agent d&apos;infrastructure est votre <InlinePopover type="licenseKey" />. Ceci est configuré lorsque vous [installez l&apos;agent](/docs/agents/manage-apm-agents/installation/install-agent). Vous pouvez également utiliser [des options configuration supplémentaires](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent) pour monitorer votre infrastructure.

## Configurer l&apos;agent mobile [#mobile-config]

Le seul paramètre requis pour la monitoring des applications mobiles agents est le[jeton d&apos;application](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token). Ceci est configuré lorsque vous installez l&apos;agent monitoring des applications mobiles pour vos applications [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android), [iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) .

Pour des options de configuration supplémentaires, voir :

* [Android configuration](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [configuration iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)

## Configurer d’autres outils New Relic [#other]

Pour configurer d&apos;autres outils non répertoriés ici, consultez la [documentation d&apos;une solution spécifique](https://newrelic.com/instant-observability/).
