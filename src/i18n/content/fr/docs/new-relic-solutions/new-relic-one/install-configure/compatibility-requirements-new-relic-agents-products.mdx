Un [agent](/docs/using-new-relic/welcome-new-relic/get-started/glossary#agent) New Relic est un logiciel que vous installez sur un hôte ou dans une application qui envoie des données de performances à New Relic. Nous utilisons différents agents pour différents produits et langages de codage.

Dans ce document, nous avons des liens vers certains de nos outils les plus populaires. Pour une liste complète de nos solutions, voir [New Relic Instant Observability](https://newrelic.com/instant-observability/).
