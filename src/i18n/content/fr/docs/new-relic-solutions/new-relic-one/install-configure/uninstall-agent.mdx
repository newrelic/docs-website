Certaines [solutions New Relic](https://newrelic.com/instant-observability/) nécessitent l&apos;installation d&apos;un agent. Vous pouvez désinstaller l&apos;agent si vous souhaitez qu&apos;il cesse de signaler des données. Si vous devez désinstaller complètement New Relic, suivez les procédures correspondant à votre agent.

Si vous rencontrez des problèmes, consultez [support.newrelic.com](https://support.newrelic.com).

## Avant de désinstaller [#downgrade]

<Callout variant="important">
  Pour les organisations payantes de New Relic, une alternative à la désinstallation est de [rétrograder](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) vers une édition inférieure.
</Callout>

## Désinstaller APM [#language-agents]

Après avoir désinstallé <InlinePopover type="apm" />agents ou modifié votre script de démarrage, vous devez redémarrer votre application (ou le serveur d&apos;applications ou le serveur Web sur lequel elle se trouve). Vous devez le faire car les agents New Relic s’exécutent généralement dans la mémoire du processus exécutant l’application. La simple suppression de ces fichiers **n’empêchera pas** les applications en cours d’exécution d’envoyer des données à New Relic.

* [Go](/docs/agents/go-agent/installation-configuration/uninstall-go-agent)
* [Java](/docs/agents/java-agent/installation/uninstall-java-agent)
* [.NET](/docs/agents/net-agent/installation-configuration/uninstall-net-agent)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/uninstall-nodejs-agent)
* [PHP](/docs/agents/php-agent/advanced-installation/uninstalling-php-agent)
* [Python](/docs/agents/python-agent/installation-configuration/uninstall-python-agent)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/uninstall-ruby-agent)
