La barre de filtre d&apos;entité vous permet de filtrer jusqu&apos;à l&apos;[entité](/docs/new-relic-solutions/new-relic-one/core-concepts/what-entity-new-relic) de monitoring que vous souhaitez examiner et vous permet d&apos;enregistrer des filtres pour une utilisation ultérieure.

<img title="filterbar.png" alt="filterbar.png" src="/images/platform_screenshot-crop_filtering-entities.webp" />

## Trouver la barre de filtre [#find]

Le filtre d&apos;entité est situé en haut de nombreuses pages de l&apos;interface utilisateur de New Relic. Il est visible dans la vue <DNT>**All entities**</DNT> , qui est la première chose que vous voyez lorsque vous [chargez New Relic](https://one.newrelic.com), et il est visible dans notre interface utilisateur APM, notre interface utilisateur <InlinePopover type="browser" />, notre interface utilisateur d&apos;infrastructure, et plus encore.

## Conseils d&apos;utilisation de la barre de filtre [#tips]

Quelques conseils pour utiliser la barre de filtre :

* Saisissez un <DNT>**string**</DNT> de caractères et/ou de chiffres (par exemple, un environnement) pour rechercher toute entité possédant cette chaîne dans son nom ou son ID. Lors de la saisie, l&apos;interface utilisateur suggère des éléments qui coïncident avec la chaîne que vous saisissez, vous pouvez donc en sélectionner un dans la liste déroulante. Vous pouvez également filtrer par le nom du <DNT>**entity**</DNT>, du <DNT>**entity type**</DNT>, <DNT>**account ID**</DNT>, <DNT>**environment**</DNT> ou du [<DNT>**tag**</DNT>](/docs/new-relic-one/use-new-relic-one/core-concepts/tagging-use-tags-organize-group-what-you-monitor). Les paramètres de sélection, une fois créés, ont un fond bleu.
* Si vous filtrez à l’aide d’une chaîne, le filtre aura l’opérateur `=` .
* Si vous filtrez en utilisant le menu déroulant, une fois le premier élément de l&apos;élément recherché (clé tag ou attribut) saisi, vous devez sélectionner un opérateur (`=`, `!=`, `LIKE`, `NOT LIKE`, `IN`, `NOT IN`) pour votre filtre.
* Une fois l&apos;opérateur sélectionné, complétez le filtre en sélectionnant la valeur pour compléter l&apos;élément de recherche.

Pour ajouter plus de filtres, vous devez d’abord sélectionner un opérateur, `AND` ou `OR`. Conseils d&apos;utilisation :

* Utilisez `AND` pour indiquer que vous souhaitez restreindre la sélection en supprimant l&apos;entité de la liste des résultats. Vous pouvez également utiliser l&apos;opérateur `AND` pour ajouter des conditions qui doivent être remplies dans la liste des résultats, par exemple, <DNT>**entityType = Services `AND` location = APAC**</DNT>.
  * Notez que <DNT>**entityType = Services `AND` entityType = Hosts**</DNT> ne renvoie aucun résultat, car l&apos;entité ne peut avoir qu&apos;un seul type et aucune entité ne correspondrait à cette condition.
* Utilisez `OR` pour ajouter plus d’entité à la sélection. Par exemple, le filtre <DNT>**entityType = Services `OR` entityType = Hosts**</DNT> renvoie chaque entité à laquelle vous avez accès et qui est de type `Services` ou `Hosts`.

Si votre filtre est très complexe, essayez d&apos;utiliser [le générateur de requêtes NRQL](https://one.newrelic.com/data-exploration/query-builder). Pour plus d&apos;informations sur les requêtes NRQL , voir [Comment utiliser NRQL](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-how-nrql-works).
