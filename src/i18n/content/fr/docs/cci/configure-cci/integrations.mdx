La fonctionnalité Intelligence Coûts du cloud intégration vous permet de visualiser et de gérer vos comptes de fournisseur cloud. Utilisez cette fonctionnalité pour suspendre ou supprimer l’intégration d’un compte de fournisseur cloud spécifique.

1. Accédez à <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>.
2. Dans le menu de navigation de gauche, sélectionnez l’onglet **Integrations** pour afficher une liste de toutes les intégrations de coûts cloud.
3. Pour afficher des détails supplémentaires, sous la colonne **Name**, cliquez sur le compte du fournisseur de cloud requis.

<Callout variant="important">
  Actuellement, seuls les coûts cloud d&apos;AWS sont pris en charge.
</Callout>

4. À côté du compte requis, cliquez sur l&apos;icône représentant des points de suspension et choisissez l&apos;une des options suivantes :

   * **Pause**: Arrêtez temporairement la collecte de données par l&apos;intégration.
   * **Delete**: supprimez définitivement l&apos;intégration de votre compte.
