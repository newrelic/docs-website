<Callout title="Aperçu">
  Nous travaillons toujours sur cette fonctionnalité, mais nous aimerions que vous l&apos;essayiez !

  Cette fonctionnalité est actuellement fournie dans le cadre d&apos;un programme d&apos;aperçu conformément à nos [politiques de pré-sortie](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Affichez et gérez vos comptes de fournisseurs cloud, et mettez en pause ou supprimez les intégrations pour des comptes spécifiques.

<Steps>
  <Step>
    ### Accéder à Cloud Cost Intelligence

    Allez à <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>.
  </Step>

  <Step>
    ### Afficher les intégrations

    Sélectionnez l&apos;onglet **Integrations** pour afficher toutes les intégrations de coûts cloud.
  </Step>

  <Step>
    ### Sélectionner un compte

    Cliquez sur un compte de fournisseur cloud sous la colonne **Name** pour plus de détails.

    <Callout variant="important">
      Actuellement, seuls les coûts cloud AWS sont pris en charge.
    </Callout>
  </Step>

  <Step>
    ### Gérer l&apos;intégration

    Cliquez sur l&apos;icône en forme de points de suspension à côté d&apos;un compte et choisissez :

    * **Pause** - Arrêter temporairement la collecte de données
    * **Delete** - Supprimer définitivement l&apos;intégration

    <img title="Pause and delete integration" alt="A screenshot displaying the pause and delete integration options" src="/images/pause-delete-cci.webp" />
  </Step>
</Steps>
