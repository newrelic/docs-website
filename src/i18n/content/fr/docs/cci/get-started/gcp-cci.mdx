<Callout title="Aperçu">
  Nous travaillons toujours sur cette fonctionnalité, mais nous aimerions que vous l&apos;essayiez !

  Cette fonctionnalité est actuellement fournie dans le cadre d&apos;un programme d&apos;aperçu conformément à nos [politiques de pré-sortie](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Connectez votre facturation Google Cloud Platform à Cloud Cost Intelligence pour monitorer et optimiser les coûts. L&apos;intégration utilise l&apos;autorisation basée sur un compte de service pour accéder en toute sécurité à vos données de facturation et de tarification GCP. New Relic crée un compte de service dédié avec un accès en lecture seule à vos ensembles de données BigQuery.

## Avant de commencer

Vous avez besoin de :

* **Projet GCP connecté au monitoring de l&apos;infrastructure New Relic** - Voir [Connecter les services Google Cloud Platform à New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)

* **Compte de facturation GCP actif**

* **Ensemble de données BigQuery** avec :

  * Table d&apos;exportation de la facturation
  * Table d&apos;exportation des prix du cloud

* **Accès à la console Google Cloud**

## Instructions de configuration

<Steps>
  <Step>
    ### Sélectionnez le compte New Relic [#select-nr-account]

    1. Allez à <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT>.
    2. Sélectionnez votre compte et cliquez sur **Continue**.
  </Step>

  <Step>
    ### Fournir les détails de connexion GCP [#gcp-connection-details]

    1. Saisissez vos informations de facturation et d&apos;ensemble de données GCP :

       * **Connection name** - Nom descriptif
       * **Project ID** - Votre ID de projet GCP
       * **Dataset name** - Nom de votre ensemble de données BigQuery
       * **Billing table name** - Nom de votre table de facturation
       * **Pricing table name** - Nom de votre table de tarification

    2. Cliquez sur **Continue**.
  </Step>

  <Step>
    ### Autoriser l&apos;accès [#gcp-authorize-access]

    1. Copiez l’ID de compte de service affiché.
    2. Ouvrez la **Google Cloud Console** et attribuez le rôle **BigQuery DataViewer** au compte de service pour vos tables de facturation BigQuery.
    3. Retournez et cliquez sur **Continue**.
  </Step>

  <Step>
    ### Créer une connexion [#gcp-create-connection]

    Cliquez sur **Create connection** pour vérifier la configuration et établir l&apos;intégration.
  </Step>

  <Step>
    ### Visualisez vos données [#gcp-view-data]

    Accédez à [**Intelligent summary (Résumé intelligent)**](/docs/cci/financial-intelligence/intelligent-summary) pour afficher vos données de coût GCP.

    <Callout variant="important">
      Le premier traitement peut prendre jusqu&apos;à 12 heures.
    </Callout>
  </Step>
</Steps>
