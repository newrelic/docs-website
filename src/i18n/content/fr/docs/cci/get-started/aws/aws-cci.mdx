Configurez votre environnement AWS pour <DNT>**Cloud Cost Intelligence**</DNT> en configurant la facturation consolidée, les rapports sur les coûts et l&apos;utilisation (CUR), les rôles et stratégies Identity and Access Management (IAM) ainsi que les régions AWS nécessaires.

## Prérequis

Assurez-vous de disposer des éléments suivants avant de commencer l&apos;une ou l&apos;autre des méthodes de configuration :

### Requis pour toutes les configurations

* <DNT>**AWS account with consolidated billing**</DNT>: [Configurer la facturation consolidée pour AWS Organizations](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/consolidated-billing.html). Ceci est nécessaire pour agréger les données de coûts de tous les comptes de votre organisation, permettant une analyse complète des coûts sur l&apos;ensemble de votre infrastructure AWS.
* <DNT>**IAM Privileges**</DNT>: un accès administrateur au compte AWS cible est requis pour déployer des ressources, créer des buckets S3, des stratégies IAM et des rôles IAM.

### Requis pour la configuration automatisée uniquement

* <DNT>**New Relic API Key**</DNT>: générez une clé API valide pour la création de la stack. Nous recommandons de créer une clé dédiée pour l&apos;intégration <DNT>**CCI**</DNT>. Générez ceci dans l&apos;UI de New Relic : [menu utilisateur &gt; API Keys](https://onenr.io/07wklp68lRL).

## Choisissez votre méthode de configuration

<DNT>**Cloud Cost Intelligence**</DNT> propose deux méthodes pour configurer l&apos;intégration AWS :

* <DNT>[**Automate AWS with CloudFormation (Recommended)**](/docs/cci/get-started/aws/automated-setup-cloudformation)</DNT>: déployez rapidement une stack CloudFormation préconfigurée qui provisionne automatiquement toutes les ressources AWS requises. Idéal pour une mise en route plus rapide avec une configuration manuelle minimale.
* <DNT>[**Manually integrate your AWS account**](/docs/cci/get-started/aws/manual-integrate-aws-account)</DNT>: configurez manuellement chaque ressource AWS pour un contrôle total sur le processus de configuration. Idéal pour les organisations ayant des exigences spécifiques en matière de conformité ou de personnalisation.
