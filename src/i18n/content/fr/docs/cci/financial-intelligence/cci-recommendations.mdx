Cloud Cost Intelligence fournit des recommandations d&apos;optimisation des coûts provenant de deux sources pour vous aider à identifier et à agir sur les opportunités de votre infrastructure AWS. Il analyse vos modèles d&apos;utilisation des ressources à l&apos;aide d&apos;AWS Trusted Advisor et de notre moteur Native Intelligence propriétaire.

<Callout variant="important">
  Les recommandations d&apos;optimisation des coûts sont actuellement disponibles pour AWS.
</Callout>

Utilisez les recommandations pour :

* Identifier les ressources surdimensionnées et sous-utilisées et les opportunités d&apos;engagement
* Affichez les économies potentielles avec des projections mensuelles et annuelles
* Suivre et hiérarchiser les actions d&apos;optimisation (<DNT>**acknowledge**</DNT>, <DNT>**dismiss**</DNT> ou <DNT>**mark as pending**</DNT>)

## Sources de recommandations

### AWS Trusted Advisor

AWS Trusted Advisor fournit des recommandations d&apos;optimisation des coûts directement depuis AWS.

#### Prérequis

Pour utiliser les recommandations de <DNT>**Trusted Advisor**</DNT> avec Intelligence Coûts du cloud, configurez l&apos;accès aux informations détaillées sur les coûts d&apos;AWS Trusted Advisor dans votre compte AWS.

#### Comment naviguer

Une fois configurées, les recommandations de <DNT>**Trusted Advisor**</DNT> apparaissent sur la page <DNT>**Cloud Cost Intelligence Recommendations**</DNT>, aux côtés d&apos;autres recommandations.

<Callout variant="important">
  Intelligence Coûts du cloud vérifie l&apos;état de <DNT>**Trusted Advisor**</DNT> lors de l&apos;intégration initiale du compte AWS. Lorsque <DNT>**Trusted Advisor**</DNT> est inactif, Intelligence Coûts du cloud ignore ce composant d&apos;intégration.

  Si <DNT>**Trusted Advisor**</DNT> est activé ultérieurement, Intelligence Coûts du cloud identifiera automatiquement la mise à jour et synchronisera les données dans les sept jours suivant l&apos;activation.
</Callout>

### Moteur d&apos;optimisation natif CCI

Le moteur d&apos;optimisation natif de CCI analyse l&apos;utilisation des ressources et l&apos;historique d&apos;utilisation pour identifier les opportunités de dimensionnement correct des instances et de réduction des coûts grâce aux engagements AWS Savings Plans.

#### Comment ça marche

Le moteur analyse les données d&apos;utilisation dans deux domaines :

* <DNT>**Right-sizing**</DNT>: vérifie l&apos;utilisation du processeur et de la mémoire, compare les types d&apos;instance actuels aux besoins de performance et identifie les instances qui peuvent être réduites sans affecter les performances.
* <DNT>**Savings plans**</DNT>: examine les modèles d&apos;utilisation pour trouver des workloads cohérents et calcule les économies potentielles des engagements des plans d&apos;économies AWS.

#### Quelles recommandations il fournit

Le moteur fournit des recommandations pour réduire les coûts :

* <DNT>**Instance type changes**</DNT>: modifications spécifiques de l&apos;instance (par exemple, <DNT>**Migrate from im4gn.16xlarge to t2.micro**</DNT>) avec la configuration actuelle, l&apos;utilisation du CPU et de la mémoire, et le nombre d&apos;instances affectées
* <DNT>**Savings plan commitments**</DNT>: recommandations pour s&apos;engager sur des niveaux d&apos;utilisation constants pour une tarification réduite
* <DNT>**Savings projections**</DNT>: économies de coûts mensuelles et annuelles pour chaque recommandation

## Afficher les recommandations

Pour accéder à vos recommandations d&apos;optimisation des coûts :

1. Aller à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Cloud Cost Intelligence**</DNT>
2. Cliquez sur <DNT>**Recommendations**</DNT> dans la navigation de gauche.
