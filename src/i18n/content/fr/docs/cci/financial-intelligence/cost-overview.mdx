La page Aperçu des coûts (anciennement appelée Explorateur de coûts) fournit une analyse détaillée des coûts avec une fonctionnalité d&apos;exploration en profondeur sur plusieurs dimensions.

Pour une vue consolidée de toutes vos dépenses cloud avec des widgets de synthèse, consultez la page [Résumé intelligent](/docs/cci/financial-intelligence/intelligent-summary).

## Fonctionnalité d&apos;exploration

Cloud Cost Intelligence fournit une capacité d&apos;exploration pour afficher les détails granulaires des coûts.

### Comment ça marche

Vous pouvez examiner les coûts selon différentes dimensions. Lorsque vous cliquez sur un service (par exemple, EC2), vous pouvez approfondir un service particulier, puis, au sein de celui-ci, une ressource particulière pour examiner les détails granulaires de cette ressource.

Tous les types de services prennent en charge l&apos;exploration en profondeur, y compris le calcul, le réseau et le stockage.

## Caractéristiques

* **Filtrable**: toutes les données sont filtrables et vous pouvez afficher et masquer des éléments
* **Vues multiples**: vous pouvez modifier les vues pour voir les données dans différents formats
* **Options de regroupement**: vous pouvez avoir plusieurs options de regroupement. Par exemple, vous pouvez afficher par dimension de service cloud, puis la modifier en type de facturation et en option de paiement.
* **Vue de tableau croisé dynamique**: Lors de l&apos;utilisation de plusieurs sélections de regroupement, les données se reflètent sous forme de vue de tableau croisé dynamique pour analyser les coûts granulaires au sein d&apos;une seule vue
* **Télécharger**: vous pouvez télécharger les données de coût pour les analyser selon différentes dimensions

## Contraintes de regroupement

* **Sélection minimale**: 1 dimension
* **Sélection maximale**: 3 dimensions

<Callout variant="important">
  Lorsque vous avez plusieurs sélections de regroupement, le graphique à barres, le graphique en courbes et le graphique en arborescence ne sont pas applicables. Seuls le tableau de données et la carte thermique sont disponibles.
</Callout>
