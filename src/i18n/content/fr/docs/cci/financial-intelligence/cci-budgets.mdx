<Callout title="Aperçu">
  Nous travaillons toujours sur cette fonctionnalité, mais nous aimerions que vous l&apos;essayiez !

  Cette fonctionnalité est actuellement fournie dans le cadre d&apos;un programme d&apos;aperçu conformément à nos [politiques de pré-sortie](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Un budget fixe une limite de dépenses pour un compte sur une période flexible allant d&apos;un à douze mois. Il vous aide à contrôler vos dépenses cloud en les monitoring à cette limite prédéfinie.

<img title="Cost types highlighted on the Cloud Cost Intelligence budgets page" alt="Cost types highlighted on the Cloud Cost Intelligence budgets page" src="/images/cci_budgets.webp" />

## Set up budgets

<Steps>
  <Step>
    ### Accédez à Intelligence Coûts du cloud

    Accédez à **All Capabilities** &gt; **Cloud Cost Intelligence**.
  </Step>

  <Step>
    ### Access budgets

    Dans le menu de navigation de gauche, sélectionnez l&apos;onglet **Budgets**.
  </Step>

  <Step>
    ### Établir des budgets

    1. Sur la page **Budgets**, cliquez sur **Create a budget**.
    2. Dans la boîte de dialogue contextuelle **Create a budget**, saisissez les informations suivantes :

    * **Budget name:** Un nom descriptif pour votre budget.

    * Sélectionnez le type de configuration requis :
