L&apos;intégration de Google Cloud Platform (GCP) pour Intelligence Coûts du cloud vous permet de monitorer et d&apos;analyser vos dépenses cloud GCP directement dans New Relic. Cette intégration utilise une méthode d&apos;autorisation basée sur un compte de service pour accéder en toute sécurité à vos données de facturation et de tarification GCP.

Lorsque vous configurez cette intégration, New Relic crée un compte de service dédié dans son environnement GCP en votre nom. Vous accordez à ce compte de service un accès en lecture seule à vos ensembles de données BigQuery, permettant à New Relic de récupérer et d&apos;analyser vos données de coût sans nécessiter un accès plus large à vos ressources GCP.

## Fonctionnalité clé

* Génération automatisée de comptes de service par compte New Relic
* Accès sécurisé en lecture seule aux données de facturation via BigQuery
* Accès aux tables d&apos;exportation de facturation et d&apos;exportation de tarification cloud
* Validation de connexion et rapport d&apos;erreurs

## Prérequis

Avant de configurer l’intégration GCP, assurez-vous d’avoir :

* Connexion à l&apos;infrastructure de monitoring New Relic - Un projet GCP déjà connecté. Voir [Connecter les services Google Cloud Platform à New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)

* Facturation activée sur un compte GCP actif.

* J&apos;ai configuré un ensemble de données BigQuery avec ces tables :

  * Tableau d&apos;exportation de facturation (contient vos données de facturation GCP)
  * Tableau d&apos;exportation des tarifs Cloud (contient les informations sur les tarifs GCP)

* Accès vérifié à Google Cloud Console.

## Connecter Google Cloud Platform à Intelligence Coûts du cloud

<Steps>
  <Step>
    ### Sélectionnez votre compte

    Commencez l&apos;intégration en choisissant votre compte New Relic.

    1. Allez à <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT>.
    2. Sélectionnez votre compte dans la liste déroulante et cliquez sur **Continue**.
  </Step>

  <Step>
    ### Configurer la connexion GCP

    1. Connectez votre compte de facturation GCP à New Relic :

       * **Nom de la connexion** : saisissez un nom descriptif (par exemple, &quot;GCP environnement de production&quot;)
       * **ID du projet** - Saisissez votre ID de projet GCP
       * **Nom du jeu de données** - Saisissez le nom de votre jeu de données BigQuery
       * **Nom de la table de facturation** - Indiquez le nom de votre table de facturation
       * **Nom du tableau de tarification** - Indiquez le nom de votre tableau de tarification

    2. Cliquez sur **Continue**.
  </Step>

  <Step>
    ### Accorder à New Relic l&apos;accès à GCP

    New Relic fournit un ID de compte de service spécifique à votre organisation pour accéder à vos données de facturation Cloud dans BigQuery.

    1. Copiez l’ID de compte de service affiché.
    2. Ouvrez **Google Cloud Console** et attribuez le rôle **BigQuery DataViewer** du compte de service à vos tables de facturation BigQuery.
    3. Revenez à cette page et cliquez sur **Continue**.
  </Step>

  <Step>
    ### Tester et créer une connexion

    Cliquez sur **Create connection** pour vérifier la configuration et établir l’intégration.
  </Step>

  <Step>
    ### Vérifier la collecte de données

    Accédez à [**Cost Overview**](/docs/cci/financial-intelligence/cloud-cost-insights) pour afficher vos données de coût GCP.

    <Callout variant="important">
      Le traitement de vos données pour la première fois peut prendre jusqu&apos;à 12 heures.
    </Callout>
  </Step>
</Steps>
