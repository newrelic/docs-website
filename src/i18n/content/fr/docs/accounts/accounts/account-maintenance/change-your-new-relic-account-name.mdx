## Changer le nom d&apos;utilisateur [#user-name]

Pour changer votre nom d&apos;utilisateur :

1. Dans le [menu utilisateur](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings), sélectionnez <DNT>**User preferences**</DNT>.
2. Sur le côté droit, cliquez sur <DNT>**Full name**</DNT>.

Notez que si un utilisateur est géré via [la gestion automatisée des utilisateurs](/docs/accounts/accounts/automated-user-management/automated-user-provisioning-single-sign), il ne peut pas modifier son nom depuis l&apos;UI.

## Changer le nom du compte [#account-name]

Pour plus de détails sur la structure du compte, voir [organisation et structure du compte](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure).

Pour ajouter et renommer des comptes depuis l&apos;UI:

1. Depuis l’ [UI de gestion des accès](https://one.newrelic.com/admin-portal/organizations/organization-detail), cliquez sur <DNT>**Accounts**</DNT>.
2. Pour le compte que vous souhaitez renommer, cliquez sur <Icon name="fe-more-horizontal" /> puis cliquez sur <DNT>**Edit account**</DNT>.

Pour savoir comment utiliser l&apos;API à cet effet, consultez [notre documentation NerdGraph](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph).

Les noms de compte sont différenciés par [l&apos;ID de compte](/docs/accounts/accounts-billing/account-structure/account-id) et non par le nom. Pour cette raison, une organisation peut avoir plusieurs comptes portant le même nom.

Si vous ne pouvez pas modifier le nom d&apos;un compte, cela peut être dû au fait que :

* Vous n&apos;avez pas [les autorisations](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#admin-settings)appropriées
* Vous êtes sur notre modèle utilisateur d&apos;origine : dans ce cas, vous utiliseriez l&apos;UI <DNT>**Account settings**</DNT>, disponible depuis le menu utilisateur
* Vous avez une structure de compte de partenariat : dans ce cas, utilisez [l&apos;API de partenariat](/docs/new-relic-partnerships/partnerships/partner-api/partnership-api-account-object)
