Si vous vous êtes inscrit à New Relic via l&apos;un des partenaires de New Relic, vous devrez peut-être suivre des procédures d&apos;authentification unique (SSO) supplémentaires pour log et commencer à utiliser New Relic. Des étapes spéciales peuvent également être nécessaires pour terminer le processus de déploiement. Ce document fournit les procédures de connexion de base pour les comptes créés via les partenaires New Relic, avec des liens vers des ressources supplémentaires, le cas échéant.

## Créer un compte New Relic (tous les partenaires) [#partner-account]

Si vous avez reçu une offre spéciale d&apos;un partenaire New Relic, suivez directement le lien du partenaire pour créer un compte New Relic.

## Connectez-vous et lancez New Relic (la plupart des partenaires) [#partner-users]

Vous recevrez un e-mail de confirmation contenant un lien pour confirmer votre compte, vous connecter à New Relic et terminer le processus de déploiement.

## Connectez-vous et lancez New Relic (partenaires spéciaux) [#partner-users]

Ces partenaires nécessitent des procédures de connexion et de déploiement spécifiques pour que vous puissiez utiliser votre compte New Relic.

<Callout variant="tip">
  Si vous avez créé un compte New Relic via un partenaire qui n&apos;est pas répertorié ici, [suivez les procédures standard](#partner-users) pour vous connecter et déployer New Relic.
</Callout>
