Rackspace propose un plugin pour le produit [Cloud Load Balancer](http://www.rackspace.com/cloud/load-balancing/ "Le lien s'ouvre dans une nouvelle fenêtre") . Le plugin vous permet de voir le trafic HTTP par rapport au trafic HTTPS, de définir facilement <InlinePopover type="alerts" />sur votre seuil prédéfini et de vérifier périodiquement l&apos;état des nœuds associés à votre équilibreur de charge pour vous assurer qu&apos;ils répondent correctement.

## Prérequis [#prereqs]

Assurez-vous que les composants suivants sont installés sur votre système :

* <DNT>Ruby</DNT> (version 1.8.7 ou supérieure)
* <DNT>Rubygems</DNT> (version 1.3.7 ou supérieure)
* <DNT>Ruby</DNT> gemme de regroupement
* Un [compte New Relic](http://newrelic.com/signup "Le lien s'ouvre dans une nouvelle fenêtre")
* Équilibreur de charge Rackspace avec logging activé

## Installation du plugin [#install]

Si vous avez des équilibreurs de charge dans plusieurs régions, configurez un plug-in distinct pour chaque région. Utilisez toujours des caractères minuscules pour le nom de la région dans votre fichier yml. Pour plus d&apos;informations sur l&apos;analyse YAML, consultez [http://yaml-online-parser.appspot.com/](http://yaml-online-parser.appspot.com/ "Le lien s'ouvre dans une nouvelle fenêtre").

```shell
wget https://github.com/newrelic-platform/newrelic_rackspace_load_balancers_plugin/archive/1.0.0.zip
```

```shell
bundle install -binstubs
  ./bin/newrelic_rs --sample-config
```

1. Téléchargez le fichier zip depuis GitHub :
2. Extrayez le contenu dans le répertoire que vous souhaitez utiliser.
3. Exécutez les commandes suivantes à partir de ce répertoire pour obtenir la configuration du gem :
4. Modifiez le `config/newrelic_plugin.yml` avec votre [clé de licence](/docs/subscriptions/license-key) New Relic, votre nom d&apos;utilisateur Rackspace et votre clé API, ainsi que la région où se trouve votre équilibreur de charge. Assurez-vous d’utiliser des caractères minuscules pour le nom de la région.
5. Démarrer l&apos;agent : `./bin/newrelic_rs`.
6. Vérifiez que la sortie indique `gathering xx statistics`.

## Affichage de l&apos;équilibreur de charge [#view]

Équilibreur de charge cloud Rackspace

Une fois le plugin installé avec succès, vous pouvez le sélectionner dans New Relic :

1. Connectez-vous à New Relic à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>.
2. Depuis l’ New Relic UI, sélectionnez l’icône Rackspace Cloud Load Balancer.

À partir de là, vous pouvez sélectionner un ou plusieurs équilibreurs de charge, afficher le trafic, configurer des alertes, etc.
