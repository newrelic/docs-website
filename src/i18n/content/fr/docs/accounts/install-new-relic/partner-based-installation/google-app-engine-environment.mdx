[Google App Engine](https://cloud.google.com/appengine/) (GAE) vous aide à créer des applications Web et mobiles sur une plateforme cloud ouverte, en utilisant les langages, le framework, [les environnements d&apos;exécution personnalisés](https://cloud.google.com/appengine/docs/flexible/custom-runtimes/), les images Docker et la bibliothèque tierce de votre choix. L&apos;environnement App Engine met automatiquement à l&apos;échelle votre application tout en équilibrant le processeur et la mémoire de vos ressources.

Vous pouvez utiliser New Relic pour monitorer les applications qui résident dans un environnement flexible Google App Engine via l&apos;[agent , et vous APM](#gae-agent) [pouvez afficher les données sur vos services Google App Engine via l&apos;intégration Google App Engine](#gae-integration) pour notre monitoring deinfrastructure.

## agent APM installation pour GAE Flexible [#gae-agent]

En tant que [partenaire Google Cloud Platform (GCP)](https://newrelic.com/partner/gcp), New Relic prend en charge l&apos;[environnement flexible Google App Engine](https://cloud.google.com/appengine/docs/flexible/) afin que vous puissiez déployer sur une image Docker avec un environnement d&apos;exécution personnalisé.

Certains agents vous permettent de déployer une image Docker standard à l&apos;aide d&apos;[ installation flexible GAE avec support « natif »](https://cloud.google.com/appengine/docs/flexible/). Avec le support « natif », vous fournissez le code de votre application et un `app.yaml` et Google App Engine déployé sur une image Docker prédéfinie. Cette configuration présente certaines limitations.

Pour monitorer votre application qui réside dans un environnement flexible Google App Engine, suivez les instructions installation de l&apos;agent <InlinePopover type="apm" />que vous utilisez :

## intégration d&apos;infrastructure pour Google App Engine [#gae-integration]

Notre [monitoring deinfrastructure](/docs/infrastructure) offre une [intégration](/docs/infrastructure/introduction-infra-monitoring) qui rapporte des données sur vos services Google App Engine. Avec l&apos;[intégration de Google App Engine](/docs/gcp-gcp_appengine-integration), vous pouvez afficher le dashboard et les alertes, les données de requête métrique et explorer les données d&apos;inventaire.
