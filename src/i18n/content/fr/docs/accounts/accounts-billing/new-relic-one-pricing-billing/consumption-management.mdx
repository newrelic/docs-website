La page <DNT>**Consumption Management**</DNT> indique exactement comment votre organisation utilise ses unités de consommation de calcul (CCU). Ce guide vous montre comment suivre, analyser et gérer votre utilisation des CCU.

<Callout variant="tip">
  <DNT>**Consumption Management**</DNT> est disponible pour tous les clients disposant de [plans Advanced ou Core Compute](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing) dans leur compte New Relic.
</Callout>

## Accéder à \[Consumption Management] [#access-consumption-management]

Pour accéder à <DNT>**Consumption Management**</DNT>:

1. Allez dans **[New Relic Administration](https://one.newrelic.com/admin-portal)**
2. Select <DNT>**Consumption Management**</DNT>

<img title="Consumption Management dashboard" alt="Image of the Consumption Management dashboard" src="/images/consumption-management-dashboard.webp" />

## Ventilez votre utilisation [#break-down-your-usage]

Utilisez le menu déroulant <DNT>**See usage for**</DNT> pour afficher différents types d&apos;utilisation des ressources de calcul :

* <DNT>**Core Compute**</DNT>: Consultez l&apos;utilisation du calcul de base de votre organisation.
* <DNT>**Advanced Compute Bundle**</DNT>: Consultez l&apos;utilisation du calcul avancé de votre organisation.

Par défaut, la page affiche les données des 30 derniers jours. Vous pouvez passer à 60 ou 90 jours à l&apos;aide du sélecteur de temps situé dans le coin supérieur droit.

Utilisez les commandes du graphique dynamique pour cibler des entités spécifiques : sélectionnez une catégorie (Comptes, Utilisateurs ou Capacités) dans le menu déroulant, recherchez des entités spécifiques ou appliquez des filtres pour affiner votre vue.

<Callout variant="tip">
  Bien que vous puissiez regrouper les données du graphique dynamique par intervalles quotidiens, hebdomadaires ou mensuels, lors de l&apos;ajout à un dashboard, sélectionnez le regroupement <DNT>**Auto**</DNT> pour permettre au système d&apos;optimiser l&apos;affichage en fonction de la plage temporelle du dashboard.
</Callout>

Les graphiques en anneau affichent le nombre et le pourcentage de CCU générés par les capacités, les utilisateurs et les comptes pour le mois sélectionné. Les filtres n&apos;affectent pas ces graphiques.

## Basculer entre les vues CCU facturables et non facturables [#switch-between-billable-and-non-billable-ccu-views]

Par défaut, la page affiche les données de CCU facturables. Pour afficher les données CCU non facturables, activez le bouton <DNT>**Non-billable CCU**</DNT>.

## Gérer et partager vos données d&apos;utilisation [#manage-and-share-your-usage-data]

Depuis le menu <Icon name="fe-more-horizontal" /> de n&apos;importe quel graphique, vous pouvez :

* Ajoutez le graphique à vos [dashboards](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/) pour faciliter le monitoring.
* [Configurez une alerte](/docs/tutorial-create-alerts/create-an-alert/) pour vous avertir des changements significatifs d&apos;utilisation.
* [Partagez une vue en temps réel](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#chart) de vos données avec les parties prenantes.
* Enregistrez le graphique sous forme d&apos;image pour des présentations ou des rapports.
* Affichez la requête sous-jacente qui alimente le graphique, que vous pouvez utiliser pour vos propres rapports personnalisés ou pour une analyse plus approfondie.

## Prévoyez votre utilisation des CCU

La page <DNT>**Consumption Management**</DNT> inclut des fonctionnalités prédictives qui vous aident à prévoir l&apos;utilisation future des ressources de calcul de votre organisation.

Pour prévoir l&apos;utilisation :

1. Accédez au <DNT>**Daily view**</DNT> du graphique de séries temporelles en haut de la page
2. Depuis le menu du graphique <Icon name="fe-more-horizontal" />, sélectionnez <DNT>**Predict Usage**</DNT>
3. Consultez les prévisions d&apos;utilisation des CCU pour 1 semaine à la fois dans le graphique et le tableau

Pour supprimer la prévision, sélectionnez <DNT>**Remove prediction**</DNT> dans le menu du graphique.

<Callout variant="important">
  Les prévisions sont uniquement disponibles pour les clients <DNT>**Advanced Compute**</DNT> et uniquement dans le <DNT>**Daily view**</DNT> du <DNT>**Consumption Management time series chart**</DNT>.
</Callout>
