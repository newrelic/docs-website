Si vous êtes client New Relic , vous pouvez voir votre utilisation sur notre page **Usage Summary** . Bien que les chiffres soient une approximation, vous pouvez utiliser ces vues pour avoir une idée générale de la manière dont votre consommation pourrait affecter votre facture.

Pour afficher l&apos;UI d&apos;utilisation, accédez à **[one.newrelic.com](https://one.newrelic.com) &amp;gt; (cliquez sur votre nom d&amp;apos;utilisateur) &amp;gt; Administration &amp;gt; Usage Summary**. Cette page montre un aperçu de votre utilisation :

<img title="New Relic usage UI" alt="New Relic usage UI" src="/images/accounts_screenshot-full_usage-ui.webp" />

<figcaption>
  L&apos;UI d&apos;utilisation de New Relic
</figcaption>

En plus d&apos;utiliser l&apos;UI d&apos;utilisation, vous pouvez également exécuter une requête pour évaluer votre utilisation. Pour plus de détails, voir [requête et alerte sur les données d&apos;utilisation](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts/).
