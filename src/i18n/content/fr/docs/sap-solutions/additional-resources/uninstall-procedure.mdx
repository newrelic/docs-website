Pour désinstaller le composant ABAP du système central :

1. connectez-vous au **client 000**.

2. Tapez le code de transaction **SAINT**.

3. Cliquez sur l’onglet **Uninstallable Components**.

4. Sélectionnez le composant **NEWR** et confirmez la version correcte à désinstaller.

5. Cliquez sur **Start**.

6. Cliquez sur la coche verte sur les écrans suivants.

7. Cliquez sur **Finish** lorsque le processus est terminé.

8. Vérifiez que le composant **NEWR** n’apparaît plus dans l’onglet **Installed Components**.
