Les Service Packs sont livrés sous forme de fichiers `.SAR` avec la convention de nommage `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`.

<Steps>
  <Step>
    Installez le service pack à l’aide de la transaction `SPAM`.
  </Step>

  <Step>
    Chargez le nouveau modèle de configuration à partir du client 000.
  </Step>

  <Step>
    Accédez à **Install New Release and Patch** &gt; **Copy Template Tables** à partir du client 000.
  </Step>

  <Step>
    Générez la nouvelle configuration du fournisseur de données en suivant les étapes décrites dans **Install New Release and Patch** &gt; **Generate Configuration for New Data Providers**.
  </Step>

  <Step>
    Cliquez sur **Install New Release and Patch** &gt; **Verify Installation Completeness** pour vérifier l&apos;installation.
  </Step>
</Steps>
