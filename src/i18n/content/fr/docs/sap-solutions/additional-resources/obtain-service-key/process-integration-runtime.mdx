<Steps>
  <Step>
    Connectez-vous au **BTP subaccount**.
  </Step>

  <Step>
    Dans le **BTP cockpit**, cliquez sur **Instances and Subscriptions**.
  </Step>

  <Step>
    Cliquez sur **Create** en haut à droite.
  </Step>

  <Step>
    Dans l’écran contextuel, sélectionnez :

    * **Service**: SAP Process intégration Runtime.
    * **Plan**: API.
    * **Environnement d&apos;exécution**: Cloud Foundry.
    * **Espace**: Vérifiez auprès de votre administrateur BTP.
    * **Nom de l&apos;instance**: suivez la convention de dénomination de votre organisation.

    <img title="Process Integration Instance" alt="A screenshot displaying the Process Integration Instance" src="/images/process-integration-instance.webp" width="600" />
  </Step>

  <Step>
    Cliquez sur **Next**, puis sélectionnez :

    * **Rôles**: `MonitoringDataRead`
    * **Types d&apos;accès**: informations d’identification client par défaut (confirmez auprès de votre administrateur BTP et de votre équipe de sécurité si d’autres types sont utilisés).

    <img title="BTP Roles and Grant Types" alt="A screenshot displaying the BTP Roles and Grant Types" src="/images/btp-roles-grant-types.webp" width="600" />
  </Step>

  <Step>
    Cliquez sur **Next** pour consulter les détails.
  </Step>

  <Step>
    Cliquez sur **Create**.
  </Step>

  <Step>
    Une fois créé, localisez le service sous l’onglet **Instances**.

    <img title="BTP Instance" alt="A screenshot displaying the BTP Instance" src="/images/btp-instance.webp" width="600" />
  </Step>

  <Step>
    Cliquez sur les points de suspension (`...`) à côté de votre instance, puis cliquez sur **Create Service Key**.
  </Step>

  <Step>
    Saisissez un **Service Key Name** et un **Key Type**: `ClientId/Secret`.

    <img title="BTP New Service Key" alt="A screenshot displaying the BTP New Service Key" src="/images/btp-new-service-key.webp" width="600" />
  </Step>

  <Step>
    Cliquez sur **Create**.
  </Step>

  <Step>
    Cliquez sur les points de suspension (`...`) à côté de votre clé de service et cliquez sur **Download**.

    <img title="BTP Download Service Key" alt="A screenshot displaying the BTP Download Service Key" src="/images/btp-download.webp" width="600" />
  </Step>
</Steps>
