<Steps>
  <Step>
    Connectez-vous au **BTP subaccount**.
  </Step>

  <Step>
    Dans le **BTP cockpit**, cliquez sur **Instances and Subscriptions**.
  </Step>

  <Step>
    Cliquez sur **Create** en haut à droite.
  </Step>

  <Step>
    Dans l’écran contextuel, sélectionnez :

    * **Service**: Gestion des API, Portail API
    * **Plan**: apiportal-apiaccess
    * **Environnement d&apos;exécution**: Cloud Foundry
    * **Espace**: Vérifiez auprès de votre administrateur BTP
    * **Nom de l&apos;instance**: suivez la convention de dénomination de votre organisation
  </Step>

  <Step>
    Consultez l&apos;avertissement relatif aux coûts.
  </Step>

  <Step>
    Cliquez sur **Next**, puis sélectionnez **Roles**: `APIPortal.Administrator`.
  </Step>

  <Step>
    Cliquez sur **Next** pour consulter les détails.
  </Step>

  <Step>
    Cliquez sur **Create**.
  </Step>

  <Step>
    Une fois créé, localisez le service sous l’onglet **Instances**.
  </Step>

  <Step>
    Cliquez sur les points de suspension (`...`) à côté de votre instance, puis cliquez sur **Create Service Key**.
  </Step>

  <Step>
    Saisissez un Service Key Name et un **Credential Type**: binding-secret au format JSON.
  </Step>

  <Step>
    Cliquez sur **Create**, puis téléchargez en cliquant sur les points de suspension (`...`) à côté de votre clé de service.
  </Step>
</Steps>
