* Si le nombre total de SID du moniteur est inférieur à 10 et que l&apos;instance ne dépasse pas 50, un serveur dédié n&apos;est pas nécessaire. Si les SID sont supérieurs à 10 ou si les instances sont supérieures à 100, un serveur dédié est recommandé.
* Au maximum, un serveur dédié avec deux instances App Server disposant chacune de 96 Go de mémoire et 12 processeurs Core doivent être utilisés. Si le système monitoring utilise une base de données HANA, nous vous recommandons de séparer l&apos;instance du serveur de base de données de l&apos;instance du serveur d&apos;applications.
