Les fournisseurs de données sont des composants qui collectent et transmettent des données de monitoring de votre <DNT>**SAP systems**</DNT> à New Relic. Cette référence technique liste tous les fournisseurs de données disponibles dans <DNT>**New Relic Monitoring for SAP Solutions**</DNT> (<DNT>**NRM4SAP**</DNT>). Utilisez ceci lors de la configuration de groupes de monitoring spécifiques. Les fournisseurs de données sont organisés par groupes de monitoring, que vous sélectionnez lors de la [configuration des systèmes monitorés](/docs/sap-solutions/install-and-configure/configure-new-relic-abap-add-on-package/#connect-systems).

## À qui s&apos;adresse cette référence

Utilisez cette référence lorsque :

* Configuration des groupes de monitoring pour <DNT>**SAP systems**</DNT>
* Dépannage des problèmes de collecte de données
* Comprendre quelles données sont collectées par chaque groupe de monitoring
* Personnalisation de la configuration des fournisseurs de données

<Callout variant="tip">
  Ces tableaux contiennent des spécifications techniques détaillées. La plupart des utilisateurs n&apos;ont besoin de consulter que les colonnes **DP\_NAME** et **OUTPUT\_TYPE**.

  Les tableaux de cette référence peuvent être défilés horizontalement pour afficher toutes les colonnes. Les fournisseurs de données marqués avec « S » dans la colonne STATUS sont suspendus par défaut.
</Callout>

## Comprendre les attributs du fournisseur de données

Chaque fournisseur de données possède les attributs techniques suivants :

* **MANDT**: Numéro de client (généralement 001)
* **DP\_NAME**: Identifiant unique du fournisseur de données
* **OUTPUT\_TYPE**: type de données télémétriques (METRIC, EVENT, LOG, TRACE, CONFIG, BTP-LOG)
* **COMMUNICATOR**: <DNT>**ABAP class**</DNT> responsable de la coordination de la collecte de données (pour le dépannage avancé)
* **DATAPROVIDER**: <DNT>**ABAP class**</DNT> qui récupère les données (pour le dépannage avancé)
* **FINISHER**: <DNT>**ABAP class**</DNT> qui traite et transmet les données (pour le dépannage avancé)
* **STATUS**: indicateur d&apos;état (S = Suspendu, vide = Actif)
* **DP\_GROUP**: Affectation du groupe de monitoring
* **METRIC\_TYPE**: pour les métriques, indique le type d&apos;agrégation (GAUGE, COUNT, SUMMARY)
* **PERIOD**: Valeur numérique de la fréquence de collecte
* **PERIOD\_UNIT**: Unité de temps pour la collecte (MIN, HRS, DAY)
* **SKIP\_RT**: Indicateur d&apos;exécution à ignorer (X = Oui)
* **RESERVED**: Réservé pour une utilisation future
