<Steps>
  <Step>
    Accédez à **All entities &gt; Workloads &gt; + Create a workload**. Consultez [Créer et définir une charge de travail](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/) pour en savoir plus sur la création d&apos;une workload.
  </Step>

  <Step>
    Saisissez le nom de la workload et sélectionnez le compte requis dans la liste déroulante.
  </Step>

  <Step>
    Cliquez sur l’icône **+** à côté de l’ option **Entity Type = All** et entrez la condition `sap.system = {SAP System SID}` dans la barre de filtre.

    <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />
  </Step>

  <Step>
    Cliquez sur **+ Add this query** pour ajouter la requête à la définition workload.
  </Step>

  <Step>
    Cliquez sur **Create a workload** pour créer la workload.
  </Step>

  <Step>
    La workload sera visible sur l&apos;interface utilisateur **All entities** et comprend toutes les balises d&apos;entité avec le système SAP.
  </Step>
</Steps>
