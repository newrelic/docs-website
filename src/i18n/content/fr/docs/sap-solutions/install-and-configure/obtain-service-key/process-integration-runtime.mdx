Ce guide vous montre comment obtenir une clé de service à partir de la plateforme SAP Business Technology Platform (BTP) pour l&apos;instance Process intégration Runtime. Vous aurez besoin de cette clé de service pour configurer le monitoring de l&apos;intégration de processus SAP.

## Prérequis

* Accès à <DNT>**BTP subaccount**</DNT> avec les permissions appropriées
* Confirmation de votre <DNT>**BTP administrator**</DNT> concernant <DNT>**Space**</DNT> et <DNT>**Grant-types**</DNT>

## Créez l&apos;instance Process Integration Runtime

1. Connectez-vous au <DNT>**BTP (Business Technology Platform) subaccount**</DNT>.

2. Dans le <DNT>**BTP cockpit**</DNT>, cliquez sur <DNT>**Instances and Subscriptions**</DNT>.

3. Cliquez sur <DNT>**Create**</DNT> en haut à droite.

4. Dans l’écran contextuel, sélectionnez :

   * <DNT>**Service**</DNT>: <DNT>**SAP Process Integration Runtime**</DNT>

   * <DNT>**Plan**</DNT>: <DNT>**API**</DNT>

   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>

   * <DNT>**Space**</DNT>: vérifiez auprès de votre <DNT>**BTP administrator**</DNT>

   * <DNT>**Instance Name**</DNT>: suivez la convention de nommage de votre organisation

     <img title="Process Integration Instance" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields" src="/images/process-integration-instance.webp" width="600" />

5. Cliquez sur <DNT>**Next**</DNT>, puis sélectionnez :

   * <DNT>**Roles**</DNT>: `MonitoringDataRead`

   * <DNT>**Grant-types**</DNT>: <DNT>**Default Client Credentials**</DNT> (confirmez auprès de votre <DNT>**BTP administrator**</DNT> et de votre équipe de sécurité si d’autres types sont utilisés)

     <img title="BTP Roles and Grant Types" alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected" src="/images/btp-roles-grant-types.webp" width="600" />

6. Cliquez sur <DNT>**Next**</DNT> pour consulter les détails.

7. Cliquez sur <DNT>**Create**</DNT>.

## Générer et télécharger la clé de service

1. Une fois créé, localisez le service sous l&apos;onglet <DNT>**Instances**</DNT>.

   <img title="BTP Instance" alt="The Instances tab showing the created Process Integration Runtime instance" src="/images/btp-instance.webp" width="600" />

2. Cliquez sur les points de suspension (`...`) dans la ligne de votre instance nouvellement créée, puis sélectionnez <DNT>**Create Service Key**</DNT>.

3. Saisissez un <DNT>**Service Key Name**</DNT> et sélectionnez <DNT>**Key Type**</DNT>: `ClientId/Secret`.

   <img title="BTP New Service Key" alt="The Create Service Key dialog showing Service Key Name and Key Type fields" src="/images/btp-new-service-key.webp" width="600" />

4. Cliquez sur <DNT>**Create**</DNT>.

5. Cliquez sur les points de suspension (`...`) dans la ligne de votre <DNT>**service key**</DNT>, puis sélectionnez <DNT>**Download**</DNT>.

   <img title="BTP Download Service Key" alt="The service key row showing the ellipsis menu with the Download option" src="/images/btp-download.webp" width="600" />

Le <DNT>**service key file**</DNT> se télécharge sous forme de <DNT>**JSON file**</DNT>. Conservez-le en toute sécurité – vous aurez besoin des identifiants de ce fichier pour configurer votre système de monitoring.
