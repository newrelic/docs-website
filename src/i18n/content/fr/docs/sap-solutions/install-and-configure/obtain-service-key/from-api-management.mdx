1. Connectez-vous au <DNT>**BTP subaccount**</DNT>.

2. Dans le <DNT>**BTP cockpit**</DNT>, cliquez sur <DNT>**Instances and Subscriptions**</DNT>.

3. Cliquez sur <DNT>**Create**</DNT> en haut à droite.

4. Dans l’écran contextuel, sélectionnez :

   * <DNT>**Service**</DNT>: <DNT>**API Management, API Portal**</DNT>
   * <DNT>**Plan**</DNT>: <DNT>**apiportal-apiaccess**</DNT>
   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>
   * <DNT>**Space**</DNT>: vérifiez auprès de votre <DNT>**BTP administrator**</DNT>
   * <DNT>**Instance Name**</DNT>: suivez la convention de nommage de votre organisation

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-management-api.webp" width="600" />

5. Consultez l&apos;avertissement relatif aux coûts.

6. Cliquez sur <DNT>**Next**</DNT>, puis sélectionnez <DNT>**Roles**</DNT>: `APIPortal.Administrator`.

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-roles-api.webp" width="600" />

7. Cliquez sur <DNT>**Next**</DNT> pour consulter les détails.

8. Cliquez sur <DNT>**Create**</DNT>.

9. Une fois créé, localisez le service sous l&apos;onglet <DNT>**Instances**</DNT>.

10. Cliquez sur les points de suspension (`...`) à côté de votre instance, puis cliquez sur <DNT>**Create Service Key**</DNT>.

11. Saisissez un <DNT>**Service Key Name**</DNT> et <DNT>**Credential Type**</DNT>: `binding-secret` dans <DNT>**JSON format**</DNT>.

<img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-servicekey-api.webp" width="600" />

12. Cliquez sur <DNT>**Create**</DNT>, puis téléchargez en cliquant sur les points de suspension (`...`) à côté de votre <DNT>**service key**</DNT>.
