<Steps>
  <Step>
    Accédez à **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**.
  </Step>

  <Step>
    Sélectionnez les **apps &amp; visualizations**, puis recherchez **IDOC Explorer**.
  </Step>

  <Step>
    Sélectionnez **Manage access**. Choisissez le compte pour accéder à l&apos;application.

    <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />
  </Step>

  <Step>
    Cliquez sur **Update**. L&apos;explorateur IDOC sera visible lors de la visualisation de l&apos;entité IDOC.
  </Step>
</Steps>
