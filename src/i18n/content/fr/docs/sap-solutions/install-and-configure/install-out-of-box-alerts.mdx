---
title: Installez la règle d'alerte prête à l'emploi
metaDescription: Install the out of the box alert policies
freshnessValidatedDate: never
translationType: machine
---

<Steps>
  <Step>
    Accédez à **[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**.
  </Step>

  <Step>
    Recherchez et cliquez sur **New Relic Monitoring for SAP Solutions alert policy**.

    <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />
  </Step>

  <Step>
    Dans l&apos;onglet **Alert condition**, sélectionnez le compte et basculez la règle d&apos;alerte sur **Enable**. Consultez [Gérer vos politiques](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) pour savoir comment gérer les politiques.
  </Step>

  <Step>
    Revenez en arrière et recherchez la **Nr4sap default alert policy**. Activez la condition d&apos;alerte que vous souhaitez activer.
  </Step>
</Steps>
