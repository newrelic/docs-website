Le composant ABAP de <DNT>**NRM4SAP**</DNT> est fourni sous forme de package complémentaire SAP et peut être installé à l&apos;aide de la transaction `SAINT` sur <DNT>**client 000**</DNT>.

## Fichier d&apos;installation du module complémentaire

* <DNT>**Initial installation**</DNT>: téléchargez le fichier du module complémentaire `K-xxxAOIINNEWR.SAR` dans un dossier sur votre PC local
* <DNT>**Upgrade from previous version**</DNT>: utilisez le fichier nommé `K-xxxAOUINNEWR.SAP`

## Installer le fichier complémentaire SAP

1. Connectez-vous à <DNT>**client 000**</DNT>
2. Tapez `SAINT` dans le champ de commande <DNT>**SAP GUI**</DNT>
3. Accédez à <DNT>**Installation Package**</DNT> &gt; <DNT>**Load Packages**</DNT>
4. Choisissez la méthode appropriée selon l&apos;emplacement de votre fichier d&apos;installation :

<DNT>**Which method should you use?**</DNT>

* <DNT>**SAR archive from frontend**</DNT> (Le plus courant) : le fichier d&apos;installation est sur votre ordinateur local
* <DNT>**SAR archive from application server**</DNT>: le fichier SAR a été pré-téléchargé sur `/usr/sap/trans/download/`
* <DNT>**EPS files from application server**</DNT>: vous devez d&apos;abord extraire le fichier SAR en fichiers PAT sur le serveur

      <img title="Content of the compressed file" alt="A screenshot displaying the Content of the compressed file" src="/images/content-of-compressed-file-1.webp" width="80%" />

    </TabsPageItem>

    <TabsPageItem id="sar-archive-from-application-server">
      1. Assurez-vous qu&apos;il y a un <DNT>**download directory**</DNT> sous le <DNT>**trans directory**</DNT> sur le serveur d&apos;application (`/usr/sap/trans/download`)
      2. Copiez le <DNT>**add-on file**</DNT> dans ce répertoire
    </TabsPageItem>

    <TabsPageItem id="eps-files-from-application-server">
      1. Extraire le <DNT>**SAR file**</DNT> dans <DNT>**PAT files**</DNT>. Exécutez les commandes suivantes :

      ```bash
      cd /usr/sap/trans
      cp /tmp/K-xxxAOIINNEWR.SAR /usr/sap/trans
      SAPCAR -xvf K-xxxAOIINNEWR.SAR
      ```

      2. Dans la fenêtre contextuelle <DNT>**Do you want to upload OCS Packages from the EPS inbox?**</DNT>, sélectionnez <DNT>**Yes**</DNT>
      3. Assurez-vous que le <DNT>**OCS file**</DNT> est téléchargé sans erreur
    </TabsPageItem>
  </TabsPages>
</Tabs>

## Terminer l&apos;installation

1. Cliquez sur <DNT>**Start**</DNT>, sélectionnez <DNT>**Installable Add-on packages for your system**</DNT> et cliquez <DNT>**Continue**</DNT>

2. Lorsqu&apos;on vous demande <DNT>**Do you want to add Modification Adjustment Transports to the queue?**</DNT>:

   * Cliquez sur <DNT>**No**</DNT> pour l&apos;installation standard (par défaut)
   * Cliquez sur <DNT>**Yes**</DNT> uniquement si votre package inclut des transports de réglage avec des instructions spécifiques

   <img title="Do you want to add Modification Adjustment Transports to the queue?" alt="A screenshot displaying question about Modification Adjustment Transports" src="/images/modify-adjustement-transports.webp" width="80%" />

3. Cliquez sur le bouton avec la coche verte pour continuer.

   <Callout variant="important">
     Si la page <DNT>**Errors during Disassembling OCS Packages**</DNT> s&apos;affiche, cliquez sur <DNT>**Ignore**</DNT>. Cette erreur est attendue pour certains types de packages et n&apos;affectera pas l&apos;installation.
   </Callout>

4. Une fois le package installé, cliquez <DNT>**Finish**</DNT>

5. La fenêtre contextuelle <DNT>**Sending the runtime analysis**</DNT> apparaît. Choisissez l&apos;un des éléments suivants :

   * <DNT>**Send**</DNT>: ouvre un formulaire de commentaires avec le <DNT>**runtime analysis data**</DNT> dans une fenêtre de navigateur locale
   * <DNT>**Display**</DNT>: affiche d&apos;abord le <DNT>**runtime analysis**</DNT>, puis ouvre le formulaire de commentaires
   * <DNT>**Do not send**</DNT>: ferme la boîte de dialogue sans envoyer <DNT>**runtime analysis data**</DNT>

6. Vérifiez l&apos;installation dans l&apos;onglet <DNT>**Installed Components**</DNT>. Le <DNT>**New Relic add-on**</DNT> devrait être listé avec un statut indiquant une installation réussie

7. Se déconnecter de <DNT>**Client 000**</DNT>
