Le composant ABAP de **NRM4SAP** est fourni sous forme de module complémentaire SAP et peut être installé à l&apos;aide de transaction **SAINT** sur le **client 000**.

## Fichier d&apos;installation du module complémentaire [#add-on-installation-file]

* **Installation initiale**: téléchargez le fichier complémentaire `K-xxxAOIINNEWR.SAR` dans un dossier sur votre PC local.
* **Mise à niveau à partir de la version précédente**: utilisez le fichier nommé `K-xxxAOUINNEWR.SAP`.

## Installer le fichier complémentaire SAP [#install-sap-add-on-file]

1. connectez-vous au **client 000**, qui est le client par défaut pour les tâches configuration et d&apos;administration système.
2. Tapez `SAINT` dans le champ de commande de l&apos;interface graphique SAP et cliquez sur **Enter**.
3. Accédez à **Installation Package** &gt; **Load Packages**.
4. Choisissez l’une des méthodes suivantes :

<CollapserGroup>
  <Collapser id="sar-archive-from-frontend" title="Archives SAR du frontend">
    1. Naviguez et ouvrez le fichier d’installation décompressé.
    2. Dans la fenêtre contextuelle **Content of the compressed file `K-xxxAOIINNEWR.SAR`**, cliquez sur **Decompress**.

    <img title="Content of the compressed file" alt="A screenshot displaying the Content of the compressed file" src="/images/content-of-compressed-file-1.webp" width="" />

    3. Le message **File K-xxxAOIINNEWR.SAR sucessfully transferred, decompressed, and deleted** s&apos;affiche.
  </Collapser>
</CollapserGroup>

<CollapserGroup>
  <Collapser id="sar-archive-from-application-server" title="Archive SAR du serveur d'applications">
    1. Assurez-vous qu&apos;il existe un répertoire de téléchargement sous le répertoire trans sur le serveur d&apos;applications (`/usr/sap/trans/download`).
    2. Copiez le fichier complémentaire dans ce répertoire.
  </Collapser>
</CollapserGroup>

<CollapserGroup>
  <Collapser id="eps-files-from-application-server" title="Fichiers EPS du serveur d'applications">
    1. Décompressez d’abord le fichier `SAR` dans le fichier `PAT`. Exécutez la commande ci-dessous sur le système d&apos;exploitation (Linux dans ce cas) :

    ```sql
    cd /usr/sap/trans
    cp /tmp/K-300AOIINNEWR.SAR /usr/sap/trans
    SAPCAR -xvf K-300AOIINNEWR.SAR
    ```

    2. Dans la fenêtre contextuelle, **Do you want to upload OCS Packages from the EPS inbox?**, sélectionnez **Yes**. Assurez-vous que le fichier OCS est téléchargé sans erreur.
  </Collapser>
</CollapserGroup>

5. Cliquez sur **Start**.

6. Sélectionnez le **Installable Add-on packages for your system** et cliquez sur **Continue**.

7. Lorsqu&apos;on vous demande **Do you want to add Modification Adjustment Transports to the queue?**, cliquez sur **No**.

   <img title="Do you want to add Modification Adjustment Transports to the queue?, Click No" alt="A screenshot displaying question, Do you want to add Modification Adjustment Transports to the queue?" src="/images/modify-adjustement-transports.webp" width="700" />

<Callout variant="important">
  Cliquez sur **Yes** si des transports de réglage sont fournis dans la liste d&apos;emballage avec des instructions.
</Callout>

8. Cliquez sur le<img src="/images/green-tick-button.webp" class="inline" style={{height: "25px", width: "30px"}} alt="green check button" title="Green check button" />bouton pour passer à l&apos;écran suivant.

<Callout variant="important">
  Si la page **Errors during Disassembling OCS Packages** s&apos;affiche, cliquez sur **Ignore**.
</Callout>

9. Une fois le package installé, cliquez sur **Finish**. La fenêtre contextuelle **Sending the runtime analysis** apparaît ; vous pouvez choisir :

   * **Send**: ouvre un formulaire de commentaires avec les données d&apos;analyse d&apos;exécution dans une fenêtre de navigateur locale. Vous pouvez utiliser ce formulaire pour envoyer des données à SAP.
   * **Display**: affiche d&apos;abord l&apos;analyse d&apos;exécution, puis envoie le formulaire de commentaires dans une fenêtre de navigateur locale.
   * **Do not send**: quitte la boîte de dialogue sans envoyer les données d&apos;analyse d&apos;exécution.

10. Assurez-vous que l’installation est réussie dans l’onglet **Installed Components**.

11. Déconnexion du **client 000**.

<Callout variant="tip">
  Vous pouvez taper **/nex** dans le champ de commande de l&apos;interface graphique SAP pour log du système.
</Callout>
