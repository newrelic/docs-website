Pour effectuer une mise à niveau à partir d&apos;une version précédente de New Relic Monitoring for SAP Solutions, assurez-vous d&apos;utiliser le package de mise à niveau approprié spécifique à votre version installée actuelle.

1. Accédez à **System &gt; Status**.
2. Recherchez **NEWR** pour identifier la version du produit.
3. Dans le client 000, exécutez la transaction **SAINT** pour mettre à jour le module complémentaire ABAP.

## Sauvegarde et mise à jour de la configuration.

<Callout variant="tip">
  Enregistrez toute configuration de fournisseur de données modifiée ou tout fournisseur de données personnalisé dans un modèle configuration pour éviter tout écrasement pendant la mise à niveau. Assurez-vous que les modèles sont enregistrés avant de copier le nouveau modèle à partir du client 000, même si le module complémentaire ABAP V3.0 n&apos;est pas installé.
</Callout>

1. Accédez au menu `/n/NEWR/INSTALL` et accédez à **Update or Save Configuration Template**.

2. Exécutez `/NEWR/REP_DELETE_INS_TEMPLATE` pour supprimer le modèle monitoring HDB lié à monitoring Java.

3. Configurez la connexion de données ABAP à la base de données HANA à l&apos;aide de `DBACockpit`.

4. Copiez le modèle de configuration du client 000.

5. Modifier la configuration du système monitoring :

   * Ajoutez la nouvelle connexion DB.
   * Supprimez les références au connecteur Java Enterprise (par exemple, NR\_EI\_AGENT).
   * Ajoutez HANA\_DB aux groupes monitoring.
   * Régénérez la configuration du fournisseur de données pour mettre à jour les paramètres.
