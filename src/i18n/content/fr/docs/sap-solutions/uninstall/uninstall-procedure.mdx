Ce guide vous montre comment désinstaller le <DNT>**New Relic ABAP component (NEWR)**</DNT> de votre système central SAP. Vous devrez peut-être le faire lors de la mise hors service du monitoring ou avant de mettre à niveau vers une nouvelle version.

<Callout variant="caution">
  La désinstallation du <DNT>**New Relic ABAP component**</DNT> arrêtera toute collecte de données de monitoring et supprimera le composant de votre système SAP. Les données historiques dans New Relic seront conservées, mais aucune nouvelle donnée ne sera collectée. Cette action ne peut pas être annulée depuis le processus de désinstallation.
</Callout>

## Prérequis

* Accès administrateur à <DNT>**SAP system**</DNT>
* Accès à <DNT>**client 000**</DNT>
* Sauvegarde de toute configuration personnalisée (si nécessaire pour une réinstallation future)

## Désinstaller le composant ABAP

Pour désinstaller le composant ABAP du système central :

1. Connectez-vous à <DNT>**client 000**</DNT>.
2. Entrez le code de transaction `SAINT`.
3. Cliquez sur l’onglet <DNT>**Uninstallable Components**</DNT>.

<img title="ABAP component on central system" alt="The SAINT transaction showing the NEWR component in the Uninstallable Components list" src="/images/ABAP-component-on-cms.webp" width="700" />

4. Sélectionnez le composant <DNT>**NEWR**</DNT> et vérifiez que le numéro de version correspond à votre version installée.
5. Cliquez sur <DNT>**Start**</DNT>.
6. Cliquez sur la coche verte pour confirmer sur chaque écran suivant.
7. Cliquez sur <DNT>**Finish**</DNT> lorsque le processus est terminé.
8. Vérifiez que le composant <DNT>**NEWR**</DNT> n’apparaît plus dans l’onglet <DNT>**Installed Components**</DNT>.

## Vérifier la désinstallation

Une fois le processus de désinstallation terminé :

* Le composant <DNT>**NEWR**</DNT> ne devrait plus apparaître dans l&apos;onglet <DNT>**Installed Components**</DNT>
* Les tâches de collecte de données dans la transaction `SM37` ne s&apos;exécuteront plus
* New Relic cessera de recevoir des données de monitoring de ce système.

**Remarque**: les données de monitoring historiques dans New Relic resteront accessibles. Seul le composant de collecte de données est supprimé du système SAP.
