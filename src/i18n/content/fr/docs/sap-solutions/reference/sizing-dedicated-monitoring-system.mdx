* Si le nombre total de <DNT>**SIDs**</DNT> monitorés est inférieur à 10 et que les instances ne dépassent pas 50, un serveur dédié n&apos;est pas nécessaire. Si les <DNT>**SIDs**</DNT> dépassent 10 ou si les instances dépassent 100, un serveur dédié est recommandé.
* Au maximum, un serveur dédié avec deux instances <DNT>**App Server**</DNT> ayant chacune 96 Go de mémoire et 12 cœurs de processeur devrait être utilisé. Si le système de monitoring utilise un <DNT>**HANA database**</DNT>, nous recommandons de séparer les instances de serveur de base de données des instances de serveur d&apos;applications.
