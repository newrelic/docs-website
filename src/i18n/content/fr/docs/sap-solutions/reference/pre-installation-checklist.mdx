Utilisez cette liste de contrôle pour collecter toutes les informations nécessaires avant l&apos;installation de <DNT>**New Relic Monitoring for SAP Solutions**</DNT>. Complétez d&apos;abord les sections de préparation, puis remplissez les tableaux de la liste de contrôle de préparation avant de commencer l&apos;installation.

## Préparation de pré-installation

### Vérifiez votre compte New Relic

* Confirmez que vous avez un compte New Relic, ou créez un nouveau compte. Notez quelle région de data center il utilise (États-Unis ou UE).

### Préparez votre liste de systèmes SAP

Pour chaque <DNT>**SAP system**</DNT> que vous prévoyez de monitorer, fournissez les informations suivantes :

* **ID système et clients**: Identifiez le système et ses clients associés.

* **Informations sur la base de données**: spécifiez le fournisseur de la base de données (par exemple, <DNT>**HANA**</DNT>, <DNT>**Oracle**</DNT>) et sa version.

* **Détails des composants logiciels**: Inclut la sortie SAP et le niveau de correctif pour les composants suivants :

  * <DNT>**SAP\_BASIS**</DNT>
  * <DNT>**Kernel**</DNT>

### Désigner un système SAP central

Choisissez un <DNT>**SAP system**</DNT> pour servir de système central pour l&apos;installation de <DNT>**New Relic Monitoring for SAP Solutions (NRM4SAP)**</DNT>.

* **Détails de l&apos;installation**: indiquez le <DNT>**System ID**</DNT> et le client où le <DNT>**ABAP component**</DNT> et la configuration seront installés.
* **Informations sur le serveur proxy**: si un serveur proxy est utilisé, spécifiez son nom d&apos;hôte et son port. Incluez toutes les politiques de serveur proxy pertinentes pour accéder à la plateforme <DNT>**New Relic One**</DNT>.

### Accessibilité du réseau

Vérifiez la connectivité réseau et la configuration de <DNT>**API endpoint**</DNT> :

* Assurez-vous que votre pare-feu ne bloque pas le trafic <DNT>**HTTPS**</DNT> vers New Relic
* Documenter les détails du serveur proxy le cas échéant
* Passez en revue les configurations <DNT>**RFC destination**</DNT> pour votre région (États-Unis ou UE)

Pour connaître les exigences réseau complètes et les détails des points de terminaison d&apos;API, consultez [Exigences réseau et points de terminaison d&apos;API](/docs/sap-solutions/reference/network-accessibility).
