Lorsque vous accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; APM**</DNT>, vous obtenez un index de toutes les applications APM associées à votre [ID de compte](/docs/accounts-partnerships/accounts/account-setup/account-id) New Relic. La page actualise automatiquement l&apos;index des applications toutes les deux minutes avec les statistiques les plus récentes [sur l&apos;état de santé](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#health-status), l&apos;événement et les performances.

<Callout variant="tip">
  Pour obtenir un aperçu de haut niveau de toutes vos applications et services, utilisez notre [explorateur d&apos;entités](/docs/new-relic-one/use-new-relic-one/ui-data/new-relic-one-entity-explorer).
</Callout>
