Utilisez [New Relic Lookout](/docs/new-relic-solutions/new-relic-one/core-concepts/new-relic-lookout-monitor-your-estate-glance/) pour détecter les écarts dans les trois signaux dorés (temps de réponse, taux d&apos;erreur et débit) de votre entité de service APM et analyser le comportement et les causes de ces écarts.

## Comment ça marche ?

Sur la page <DNT>**APM summary**</DNT> , Lookout compare les valeurs des signaux dorés au cours des cinq dernières minutes avec ses valeurs au cours de l&apos;heure précédente. Si Lookout détecte une tendance significative à la hausse ou à la baisse dans le signal, un bouton <DNT>**Diagnose**</DNT> apparaît sur la discussion récapitulative APM correspondante.

<img title="Embedded Lookout diagnose button" alt="A screenshot of the Lookout diagnose button embedded within in APM Web transactions time chart." src="/images/embedded-lookout-diagnose-button.webp" />

<figcaption>
  Le bouton Lookout <DNT>**Diagnose**</DNT> intégré dans un graphique temporel Web de transaction APM
</figcaption>

Lorsque vous cliquez sur le bouton, un outil de diagnostic Lookout s&apos;ouvre et vous pouvez l&apos;utiliser pour explorer les détails de vos signaux dorés.
