## Syntaxe

```php
newrelic_set_user_id(string $user_id)
```

Associez un identifiant utilisateur unique à la transaction en cours.

## Exigences

Version de l&apos;agent 10.12 ou supérieure.

## Description

Cette API permet à une application d&apos;attribuer un identifiant utilisateur unique à la transaction en cours.

Cela permettra à un utilisateur d&apos;obtenir des informations détaillées sur chaque utilisateur final, comme l&apos;impact d&apos;un événement d&apos;erreur.

## Valeurs de retour

Cette fonction renverra `true` si l&apos;ID utilisateur a été ajouté avec succès, sinon `false`.

## Exemples

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $user_id = "YOUR UNIQUE USER ID";
        newrelic_set_user_id($user_id);
    }
}
```
