## Syntaxe

```php
newrelic_get_browser_timing_header([bool $include_tags])
```

Renvoie unsnippet <InlinePopover type="browser" />à injecter dans l&apos;en-tête de votre sortie HTML.

## Description

Renvoie une chaîne JavaScript à injecter dans le `<head>` de votre sortie HTML. Utilisez cet appel avec [`newrelic_get_browser_timing_footer`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer) pour ajouter manuellement monitoring des navigateurs à une page Web. Si possible, New Relic recommande plutôt d&apos;activer monitoring des navigateurs via l&apos; de New Relic UI ou de copier/coller snippet l&apos; JavaScript. Pour obtenir des instructions sur l&apos;utilisation de ces options, consultez [monitoring des navigateurs et PHP](/docs/agents/php-agent/features/page-load-timing-php).

<Callout variant="tip">
  Comparez [`newrelic_disable_autorum`](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum), qui <DNT>**disables**</DNT> le script du navigateur sur une page.
</Callout>

## Valeurs de retour

Renvoie la chaîne d&apos;en-tête JavaScript de l&apos; agent du navigateur.

## Exemples

### Instrumenter une page Web [#webpage]

```php
function example(){
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_header = newrelic_get_browser_timing_header();
    }
    
    // Output to page
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_footer = newrelic_get_browser_timing_footer();
    }
}
```
