## Syntaxe

```php
newrelic_get_linking_metadata()
```

Renvoie une collection de métadonnées nécessaires pour lier des données à une trace ou à une entité.

## Exigences

Nécessite l'agent PHP [version 9.3 ou supérieure](/docs/release-notes/agent-release-notes/php-release-notes).

Doit être appelé à l'intérieur d'une [transaction](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction).

## Description

Cet appel renvoie une carte opaque de paires valeur-clé qui peuvent être utilisées pour corréler cette application à d'autres données dans le New Relic backend.

## Exemples

### Créer une charge utile de liaison

Obtient les données de liaison et les encode au format JSON pour les exporter vers l'outillage :

```php
if (extension_loaded('newrelic')) {
    $linking_md = newrelic_get_linking_metadata();
    $payload = json_encode($linking_md);
}
```
