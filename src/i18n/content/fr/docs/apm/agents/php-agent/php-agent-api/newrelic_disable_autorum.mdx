## Syntaxe

```php
newrelic_disable_autorum()
```

Désactiver l&apos;injection automatique du snippet <InlinePopover type="browser" />sur des pages particulières.

## Exigences

Version de l&apos;agent [2.6.5.41](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-26541) ou supérieure.

## Description

Cet appel désactive automatique injection de l&apos; des [monitoring navigateurs](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) agent pour la transaction en cours. Appelez le plus tôt possible. Vous pouvez utiliser cet appel pour supprimer le JavaScript si l&apos; [insertion pose des problèmes](/docs/browser/new-relic-browser/troubleshooting/browser-javascript-injection-causes-problems-page) ou si vous diffusez des pages vers des services tiers qui n&apos;autorisent pas JavaScript (par exemple, [les pages mobiles accélérées de Google](/docs/browser/new-relic-browser/troubleshooting/google-amp-validator-fails-due-3rd-party-script)).

<Callout variant="tip">
  Pour <DNT>**enable**</DNT> monitoring les navigateurs avec un appel d&apos;API, voir :

  * [`newrelic_get_browser_timing_footer()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer)
  * [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header)
</Callout>

## Paramètres

Cet appel n&apos;accepte aucun paramètre.

## Valeurs de retour

Renvoie `true` si elle est appelée dans une transaction New Relic. Sinon, renvoie `null` si en dehors d&apos;une transaction (par exemple, si `newrelic_end_transaction()` a été appelé).

## Exemples

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_disable_autorum();
    }
    // ...
}
```
