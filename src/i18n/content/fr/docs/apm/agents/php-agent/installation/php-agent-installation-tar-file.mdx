Vous pouvez installer l&apos;agent PHP de New Relic de à partir de fichiers tar compressés sur n&apos;importe quel système d&apos;exploitation pris en charge sur architecture n&apos;importe quelle de processeur prise en charge. Cependant, pour RedHat ou CentOS sur x86\_64, il est plus courant d&apos;utiliser le [package RPM](/docs/agents/php-agent/installation/php-agent-installation-aws-linux-redhat-centos/). Pour Ubuntu ou Debian sur x86\_64, il est plus courant d&amp;apos;utiliser le [package Debian](/docs/agents/php-agent/installation/php-agent-installation-ubuntu-debian).

## Télécharger la distribution tar [#download]

Contrairement à d’autres options d’installation, les archives tar ne nécessitent aucune configuration de référentiel spéciale. Il vous suffit de télécharger l’archive et de suivre ces instructions.

Téléchargez le fichier de distribution tar approprié à partir de <DNT>**[download.newrelic.com/php\_agent/release/](https://download.newrelic.com/php_agent/release/)**</DNT>:

* Pour Linux avec GNU libc (la plupart des distributions Linux), téléchargez `newrelic-php5-X.X.X.X-linux.tar.gz`.
* Pour Linux avec musl libc (Alpine Linux), téléchargez `newrelic-php5-X.X.X.X-linux-musl.tar.gz`.

<Callout variant="tip">
  Le nom du package pour l&apos;agent PHP est `newrelic-php5`. Bien que le nom du package fasse référence à PHP 5, ce package fonctionne pour toutes [les versions PHP prises en charge](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#php-release).
</Callout>

## Installer ou mettre à jour [#install]

Avec l&apos;installation du fichier tar, les étapes pour la première installation et pour la mise à jour sont les mêmes. Remplacez XXXX par la version correcte de New Relic. Pour installer ou mettre à jour l&apos;agent :

1. Téléchargez le fichier tar approprié à partir du [site Web de New Relic](https://download.newrelic.com/php_agent/release/) et enregistrez-le sur un disque local dans un emplacement pratique. N&amp;apos;utilisez pas le dossier `/tmp` .

2. Décompresser et extraire l’archive :

   ```bash
   gzip -dc newrelic-php5-X.X.X.X-OS.tar.gz | tar xf -
   ```

3. Accédez au répertoire nouvellement créé :

   ```bash
   cd newrelic-php5-X.X.X.X-OS
   ```

4. Exécutez le script d’installation :

   ```bash
   ./newrelic-install
   ```

5. Modifiez le [nom de l’application](/docs/site/naming-your-application) par défaut par un nom significatif.

6. Redémarrez votre serveur Web (Apache, Nginx, PHP-FPM, etc.).

7. Attendez quelques minutes que votre application envoie des données, puis [vérifiez les performances de votre application dans l&apos; UI](/docs/apm/applications-menu/monitoring/apm-overview-page).
