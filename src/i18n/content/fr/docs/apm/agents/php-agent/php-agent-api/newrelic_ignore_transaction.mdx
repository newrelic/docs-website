## Syntaxe

```php
newrelic_ignore_transaction()
```

N&apos;instrumentez pas la transaction en cours.

## Exigences

Compatible avec toutes les versions d&apos;agent.

## Description

Ne pas générer de données pour cette transaction. Cela est utile lorsque vous avez des transactions qui sont particulièrement lentes pour des raisons connues et que vous ne souhaitez pas qu&apos;elles génèrent fréquemment des traces de transaction ou qu&apos;elles faussent les moyennes de votre site.

## Paramètres

Cet appel n&apos;accepte aucun paramètre.

## Exemples

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_ignore_transaction();
    }
    ...
}
```
