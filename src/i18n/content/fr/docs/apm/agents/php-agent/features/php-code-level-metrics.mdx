Les métriques au niveau du code permettent aux développeurs utilisant l&apos;[extension New Relic CodeStream](/docs/codestream/observability/code-level-metrics) de voir les données APM affichées contextuellement dans leur IDE, aux côtés des méthodes individuelles dans le code. Cela permet aux développeurs d’être plus proactifs dans la résolution des problèmes de performances lorsqu’ils écrivent et révisent le code.

## Bien démarrer

Tout d’abord, [installez](/docs/codestream/start-here/install-codestream) l’extension New Relic CodeStream dans l’IDE pris en charge de votre choix et log -vous.

<Callout variant="important">
  Les métriques au niveau du code sont disponibles dans l&apos;agent PHP version 10.4.0 ou supérieure, et activées par défaut pour les versions 10.6.0 et supérieures. Pour modifier cette configuration, consultez notre [documentation](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-code-level-metrics).
</Callout>

## attribut de l&apos;agent

L&apos; agent PHP signale et attache l&apos;attribut suivant aux spans produits par votre application:

  <thead>
    <tr>
      <th>

        </DNT>
      </th>

      <th>
        <DNT>

      <th>

        </DNT>

  </thead>

  <tbody>
    <tr>
      <td>
        `code.function`
      </td>

      <td>
        Le nom de la fonction instrumentée
      </td>

    <tr>
      <td>
        `code.filepath`
      </td>

      <td>

      </td>

      <td>
        /symfony/symfony\_app/src/Controller/ProductController.php
      </td>
    </tr>

    <tr>
      <td>
        `code.lineno`

      <td>

      </td>

      <td>
        16
      </td>
    </tr>

    <tr>
      <td>
        `code.namespace`
      </td>

      <td>
        L&apos;espace de nommage (nom de classe/module) dans lequel `code.function` est défini
      </td>

      <td>
        App\Controller\ProductController
      </td>
    </tr>
  </tbody>
</table>

<Callout variant="important">
  Les métriques au niveau du code ne sont prises en charge que pour la version PHP 7.0 ou supérieure.
</Callout>
