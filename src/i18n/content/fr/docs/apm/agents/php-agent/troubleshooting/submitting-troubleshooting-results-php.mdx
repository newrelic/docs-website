## Problème

Vous continuez à rencontrer des problèmes lors de l&apos;installation de l&apos;agent PHP New Relic.

## Solution

Si vous continuez à rencontrer des problèmes avec votre installation New Relic PHP après avoir suivi les [conseils de dépannage](/docs/apm/agents/php-agent/troubleshooting/no-data-appears-php), obtenez de l&apos;aide sur [support.newrelic.com](https://support.newrelic.com).

Lorsque vous travaillez avec un ingénieur du support technique New Relic, fournissez les informations suivantes :

* Jusqu&apos;où avez-vous progressé dans le processus d&apos;installation ?

* Quelle méthode d&apos;installation utilisez-vous

* Une copie de votre fichier `newrelic.cfg` (le cas échéant)

* Un lien vers votre page <DNT>**phpinfo**</DNT> ou la sortie de <DNT>**phpinfo**</DNT> enregistrée sous forme de fichier HTML. Pour [créer <DNT>**phpinfo**</DNT>](/docs/php/php-kb-103), créez un .php page Web contenant uniquement :

  ```php
  <?php phpinfo(); ?>
  ```

* Votre [agent et daemon fichier de log](/docs/php/php-kb-105)
