## Syntaxe

```php
newrelic_capture_params([bool $enable_flag])
```

Activer ou désactiver la capture du paramètre URL.

## Exigences

Compatible avec toutes les versions d&apos;agent.

## Description

Activer ou désactiver la capture du paramètre URL (tout ce qui suit le `?` dans l&apos;URL). Ceci remplace le paramètre du fichier de configuration `newrelic.capture_params` .

<Callout variant="important">
  Si vous transmettez des informations sensibles directement dans l&apos;URL, gardez cette option désactivée.
</Callout>

## Exemples

### Activer la capture du paramètre URL [#enable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params();
    }
    //...
}
```

### Désactiver la capture du paramètre URL [#disable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params(false);
    }
    //...
}
```
