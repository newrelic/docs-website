New Relic a ajouté le support de la bibliothèque Predis dans agent PHP [sortie 4.22](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-422099).

L&apos;instrument agent standard appelle un objet `Predis\Client` ou un pipeline. Toutes les commandes envoyées via un pipeline seront regroupées et traitées comme une seule métrique de commande nommée `pipeline` plutôt que comme des commandes individuelles.

L&apos; agent PHP ne prend pas en charge les appels aux méthodes `executeRaw` ou `transaction`, PubSub, monitorer ou similaires. Ceux-ci ne seront pas instrumentés.
