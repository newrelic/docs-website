Afin d&apos;installer New Relic via un système comme Puppet ou Chef, le [`newrelic-install`](/docs/agents/php-agent/advanced-installation/using-newrelic-install-script) script peut être exécuté en <DNT>**silent**</DNT> mode et contrôlé via un certain nombre de variables d&amp;apos;environnement. Ce document fournit des informations pour définir manuellement les variables d&amp;apos;environnement afin de permettre au [script d&apos;installation PHP](/docs/agents/php-agent/advanced-installation/using-newrelic-install-script) de s&amp;apos;exécuter en mode silencieux.

## Choisir un programme d&apos;installation [#install-choose]

Deux options sont disponibles lors de l&apos;utilisation de New Relic via système comme Puppet ou Chef:

* Utilisez Puppet pour installer les archives en utilisant le gestionnaire de paquets natif (<DNT>**rpm**</DNT> ou <DNT>**dpkg**</DNT>).
* Distribuez le contenu de la distribution tarball et placez les fichiers à l&apos;emplacement sélectionné.

Si vous installez via le package système, le script d&apos;installation package place les fichiers. Les systèmes basés sur Ubuntu sont une exception, où le script de post-installation créera des liens symboliques pour l&apos;extension New Relic si vous avez installé le PHP standard. Cependant, il ne tente pas d&apos;exécuter `newrelic-install`, car cela est laissé à votre script Puppet ou Chef .

Afin d&apos;installer New Relic via un système comme Puppet ou Chef, le [`newrelic-install`](/docs/agents/php-agent/advanced-installation/using-newrelic-install-script) script peut être exécuté en <DNT>**silent**</DNT> mode et contrôlé via un certain nombre de variables d&amp;apos;environnement.

## Définir les variables d&apos;environnement [#install-environment]

Il s&apos;agit de la liste des variables d&apos;environnement que vous pouvez définir avant d&apos;appeler [`newrelic-install`](/docs/agents/php-agent/advanced-installation/using-newrelic-install-script) pour contrôler son comportement. En plus d&amp;apos;utiliser ces variables d&amp;apos;environnement, assurez-vous de définir également votre [clé de licence](/docs/agents/php-agent/configuration/php-agent-configuration#inivar-license) et [le nom de votre application](/docs/agents/php-agent/configuration/php-agent-configuration#inivar-appname) dans votre fichier `newrelic.ini` lors de l&amp;apos;installation.

<Callout variant="important">
  `NR_INSTALL_SILENT` détermine si l&amp;apos;installation s&amp;apos;exécute en mode silencieux. Une fois défini, toutes les informations du script doivent être fournies via ces variables d&amp;apos;environnement.
</Callout>
