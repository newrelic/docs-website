## Problème

Vous avez installé l&apos;agent et le daemon de PHP de New Relic, mais le daemon ne semble pas fonctionner.

## Solution

Vérifiez que le daemon fonctionne correctement :

1. Exécutez cette commande shell pour répertorier les processus en cours d&apos;exécution qui correspondent à « newrelic-daemon » :

   ```bash
   ps -ef | grep newrelic-daemon
   ```

2. Vérifiez que le daemon est en cours d’exécution en vérifiant la sortie de la commande précédente pour deux processus daemon en cours d’exécution qui ressemblent à ceci. S’ils ne sont pas répertoriés, passez à l’étape suivante.

   ```console
   newrelic     1368     1  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   newrelic     1370  1368  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   ```

3. Si aucun processus n&apos;est répertorié, votre daemon n&apos;est pas en cours d&apos;exécution et vous devez démarrer le daemon en fonction du mode de démarrage dans lequel il se trouve :

   * [Démarrage du mode agent](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-external) (par défaut)
   * [Démarrage en mode externe](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-autostart) (avancé)
