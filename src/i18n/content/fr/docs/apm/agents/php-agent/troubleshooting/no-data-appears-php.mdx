## Problème

Après avoir [installé l&apos;agent PHP de New Relic](/docs/agents/php-agent/installation/php-agent-installation-overview), généré du trafic et attendu au moins cinq minutes, aucune donnée n&apos;apparaît dans votre New Relic UI.

## Solution

Si aucune donnée n&apos;apparaît après avoir généré du trafic vers votre application et attendu au moins cinq minutes, utilisez [New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics) pour détecter automatiquement les problèmes courants et suggérer un dépannage. Si cela ne résout pas le problème, essayez ce qui suit :

      </td>

        Serveur Web

        Redémarrez votre serveur Web (Apache, Nginx, PHP-FPM, etc.) et attendez quelques minutes que les données apparaissent.
      </td>
    </tr>

      <td>

      </td>

    <tr>
      <td>

    </tr>

    <tr>
      <td>
        fichier de log

        2. Sinon, [réinstallez l&apos;agent](/docs/agents/php-agent/installation/php-agent-installation-overview).
        3. Vérifiez les autorisations du fichier de log pour vous assurer que le propriétaire log est le même que l&apos;utilisateur New Relic .
