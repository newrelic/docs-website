Si les procédures de dépannage de l&apos;agent Java(comme [Aucune donnée n&apos;apparaît](/docs/apm/agents/java-agent/troubleshooting/no-data-appears-java) et d&amp;apos;autres documents dans cette catégorie) n&amp;apos;ont pas résolu votre problème, utilisez le [fichier de log](#logfiles) pour résoudre le problème. Si vous avez besoin d&amp;apos;aide supplémentaire, contactez [support.newrelic.com](https://support.newrelic.com).
