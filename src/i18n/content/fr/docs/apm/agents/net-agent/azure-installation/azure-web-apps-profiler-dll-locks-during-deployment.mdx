## Problème

Lorsque vous exécutez une nouvelle version de l&apos;agent .NET de New Relic (`NewRelic.Profiler.dll`) pour votre [application web Microsoft Azure](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps), un message d&amp;apos;erreur vous empêche de continuer jusqu&amp;apos;à ce que vous arrêtiez ou redémarriez le processus.

## Solution

Normalement, le processus s&apos;arrête, puis les ressources sont écrasées par les ressources que New Relic regroupe dans la racine du site (située à `D:\Home\site\wwwroot\newrelic`) lors d&amp;apos;un déploiement. Pour résoudre le problème, essayez l’une de ces solutions.

<CollapserGroup>
  <Collapser id="stop-dll-deploy" title="Arrêter la sortie du fichier">
    Cette solution est disponible pour votre application Web Azure si vous essayez de déployer une nouvelle version de l&apos;agent .NET sur une version existante :

    1. Arrêtez votre application Web pour sortir le fichier `.dll` .
    2. Autoriser la mise à jour à se poursuivre.
    3. Une fois le lever terminé, redémarrez l&apos;instance.
  </Collapser>

  <Collapser id="site-extension" title="Utilisez l'extension de site New Relic">
    Vous pouvez installer l’agent .NET de New Relic dans votre application Web Azure à l’aide de l’ [extension de site New Relic](https://www.siteextensions.net/packages/NewRelic.Azure.WebSites/). (Vous ne devriez avoir besoin d&amp;apos;installer l&amp;apos;extension qu&amp;apos;une seule fois.)

    1. Accédez à <DNT>**[https://yoursite.scm.azurewebsites.net/](https://yoursite.scm.azurewebsites.net/)**</DNT> et sélectionnez <DNT>**Site Extensions**</DNT>.

    2. À partir du <DNT>**Gallery**</DNT>, sélectionnez l&amp;apos;ajout <Icon name="fe-plus" /> icône, puis sélectionnez <DNT>**New Relic**</DNT>.

    3. Suivez les instructions pour [ajouter manuellement les clés de configuration](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps) à votre application Web et utilisez l’option qui correspond le mieux à vos besoins.

       Pour mettre à jour l’extension de site : dans le [portail Azure](https://portal.azure.com), sélectionnez <DNT>**WebApps &gt; (select your site) &gt; All Settings &gt; Extensions &gt; New Relic**</DNT>.
  </Collapser>
</CollapserGroup>
