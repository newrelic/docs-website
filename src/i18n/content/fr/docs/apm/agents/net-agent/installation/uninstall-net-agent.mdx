Ce document explique comment désinstaller l&apos;agent .NET de New Relic. Pour obtenir des instructions sur la façon de désactiver temporairement l&apos;agent, voir [Désactiver l&apos;agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#net).

Pour désinstaller l&apos;agent.NET, suivez la procédure qui correspond à votre système d&apos;exploitation :

* [Windows](#uninstall-framework-windows)
* [Linux](#uninstall-linux)

## Désinstaller l&apos;agent .NET (Windows) [#uninstall-framework-windows]

<Callout variant="tip">
  Nécessite [des droits<DNT>**Administrator**</DNT> ](/docs/agents/net-agent/getting-started/compatibility-requirements-net-framework-agent#user-rights)dans votre groupe d’administrateurs Windows.
</Callout>

Cette procédure suppose que vous avez installé l&apos;agent .NET avec le programme d&apos;installation MSI :

1. Pour arrêter un agent en cours d’exécution, arrêtez IIS. Si vous n’arrêtez pas IIS, vous devrez peut-être redémarrer après la désinstallation de l’agent.
2. Sélectionnez <DNT>**Start &gt; All Programs &gt; New Relic .NET Agent &gt; Uninstall the .NET agent**</DNT>.
3. Suivez l’invite pour terminer la désinstallation.
4. Redémarrez IIS.

Après avoir désinstallé l&apos;agent, vous souhaiterez peut-être supprimer tous les fichiers liés à l&apos;agent. Assurez-vous de vérifier les autres chemins pour supprimer tous les autres fichiers New Relic. Par exemple, la désinstallation de l’agent .NET ne supprime pas votre fichier [`newrelic.config`](/docs/agents/net-agent/installation-configuration/net-agent-configuration#config-options-precedence) .

1. À partir de <DNT>**Control Panel &gt; Programs &gt; Programs and Features**</DNT>, faites un clic droit et supprimez tous les fichiers nommés `New Relic .NET Agent`.
2. Supprimez l’intégralité du dossier à `C:\ProgramData\New Relic\.NET Agent`.
3. Supprimez l’intégralité du dossier à `C:\Program Files\New Relic\.NET Agent`.
4. Supprimez l’intégralité du dossier à `C:\Program Files (x86)\New Relic\.NET Agent`.
5. [Supprimez votre application](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) de l&apos;UI APM.

## Désinstaller l&apos;agent .NET (Linux) [#uninstall-linux]

Pour désinstaller l’ agent .NET sous Linux, utilisez votre outil de gestion package .

<Callout variant="important">
  Pour les versions d’agent .NET 10.0.0 ou supérieures, le nom du package est `newrelic-dotnet-agent`. Pour les versions d’agent .NET 9.9.0 ou inférieures, le nom du package est `newrelic-netcore20-agent`. Le reste de ce document fera référence à `newrelic-dotnet-agent`; remplacez-le par l&apos;ancien nom si vous avez installé une ancienne version de l&apos;agent.
</Callout>

<Callout variant="important">
  Vous devez redémarrer votre application une fois la désinstallation terminée.
</Callout>

<CollapserGroup>
  <Collapser id="apt-uninstall" title="Désinstaller avec apt (Debian, Ubuntu et Linux Mint)">
    Exécutez la commande suivante en tant que root (ou en utilisant `sudo`) :

    ```sh
    apt-get remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser id="dpkg-uninstall" title="Désinstaller avec dpkg (Debian, Ubuntu et Linux Mint)">
    Exécutez la commande suivante en tant que root (ou en utilisant `sudo`) :

    ```sh
    dpkg --remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser id="yum-uninstall" title="Désinstaller avec yum (CentOS, Red Hat Enterprise Linux, Oracle Linux)">
    Exécutez la commande suivante en tant que root (ou en utilisant `sudo`) :

    ```sh
    yum remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser id="rpm-uninstall" title="Désinstaller avec rpm (CentOS, Red Hat Enterprise Linux, Oracle Linux)">
    Exécutez la commande suivante en tant que root (ou en utilisant `sudo`) :

    ```sh
    sudo rpm -e newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser id="tarball" title="Désinstaller tarball">
    1. Supprimez tous les fichiers New Relic de votre dossier d&apos;application.

    2. Annuler la définition des variables d’environnement New Relic :

       ```ini
       CORECLR_ENABLE_PROFILING=1
       CORECLR_PROFILER={36032161-FFC0-4B61-B559-F6C5D41BAE5A}
       CORECLR_NEWRELIC_HOME=PATH_TO_INSTALL
       CORECLR_PROFILER_PATH=%CORECLR_NEWRELIC_HOME%\NewRelic.Profiler.dll
       ```
  </Collapser>
</CollapserGroup>
