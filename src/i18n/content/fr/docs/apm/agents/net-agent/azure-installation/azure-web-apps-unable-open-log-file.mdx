## Problème

Vous vous connectez via FTP à votre [application Web Microsoft Azure](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps). Vous pouvez voir un fichier `NewRelic.Profiler.####.log` dans <DNT>**LogFiles\NewRelic**</DNT>, mais lorsque vous essayez d&apos;accéder au fichier de log, un message d&apos;erreur apparaît.

## Solution

Redémarrez votre application Web pour extraire le fichier et créer un nouveau log de profil.
