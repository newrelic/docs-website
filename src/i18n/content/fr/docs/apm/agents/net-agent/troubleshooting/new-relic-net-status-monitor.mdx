Le moniteur d&apos;état est un outil de diagnostic et d&apos;état obsolète pour l&apos;agent .NET de New Relic. Il est conçu pour :

* Vous donne une visibilité sur l’état actuel de l’agent .NET exécuté sur votre serveur.
* Vous aide à résoudre les problèmes d&apos;installation courants.
* Fournissez un lien rapide vers chacune de vos applications instrumentées sur newrelic.com.
* Fournit un accès rapide au fichier New Relic Logs pour chaque application instrumentée.

Pour obtenir de l’aide supplémentaire, consultez la documentation de dépannage de l’agent .NET de New Relic.

## Afficher l&apos;état des applications de monitoring [#viewing]

<Callout variant="important">
  À partir de la sortie 6.20 de l&apos;agent .NET, le moniteur d&apos;état .NET de New Relic n&apos;est plus disponible. Pour [les versions d&apos;agent .NET 6.20 ou supérieures](/docs/release-notes/agent-release-notes/net-release-notes), utilisez [New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics).
</Callout>

Le moniteur d&apos;état fournit une visibilité sur les applications actuellement instrumentées par l&apos;agent .NET et sur l&apos;état de chaque application. Vous ne verrez aucune donnée sur une application à moins qu&apos;il y ait eu du trafic sur celle-ci récemment. Afin de préserver les ressources de votre système, le moniteur d&apos;état ne s&apos;actualise pas automatiquement.

Pour lancer le moniteur d&apos;état :

1. Depuis le menu <DNT>**Start**</DNT> ou l’écran <DNT>**Start**</DNT> de Windows, recherchez `Status monitor`.

   OU

2. Cliquez avec le bouton droit sur l’icône New Relic dans la zone d’état de la barre des tâches Windows (barre d’état système), puis sélectionnez <DNT>**Monitored applications**</DNT>.

3. Pour voir si une application est instrumentée, assurez-vous que toutes ses cases à cocher sont vertes.

4. Après avoir sélectionné une application spécifique, sélectionnez l&apos;une des options disponibles pour afficher l&apos;application dans New Relic, afficher le fichier log, [réparer l&apos;installation](#repairing) si des erreurs sont détectées, ou actualiser, puis afficher le dernier état de vos processus de monitoring et de votre application.

## Réparation et installation [#repairing]

L&apos;installation de l&apos;agent .NET de New Relic est robuste et simple à utiliser. Cependant, il existe certains scénarios dans lesquels l&apos;installation précédente peut entrer en conflit avec la dernière version. Pour résoudre les problèmes courants, réparez votre installation :

1. Dans la liste des applications du moniteur d’état, sélectionnez une application, puis sélectionnez <DNT>**Repair installation**</DNT>.

   OU

   Dans la barre d’état système Windows, sélectionnez l’icône New Relic, puis sélectionnez <DNT>**Tools &gt; Repair installation**</DNT>.

2. Une fois la réparation de l&apos;installation terminée : à partir de la ligne de commande, effectuez une opération `IISRESET`.
