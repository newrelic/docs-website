## Syntaxe [#syntax]

```py
    newrelic.agent.current_trace_id()
```

Renvoie l&apos;ID de trace de la transaction en cours ou `None` si aucune transaction n&apos;existe.

## Description [#description]

Utilisez `current_trace_id` pour récupérer l’ID de trace de la [transaction](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction) en cours.

## Valeurs de retour [#return-values]

Renvoie l&apos;ID de trace de la transaction en cours. Renvoie `None` s&apos;il n&apos;y a pas de transaction active.

## Exemples [#examples]

### Obtenir l&apos;ID de trace actuel [#function-trace-example]

```py
    import newrelic.agent

    @newrelic.agent.background_task()
    def main():
        trace_id = newrelic.agent.current_trace_id()
```
