## Syntaxe

```py
newrelic.agent.application(name=None)
```

Renvoie un objet correspondant à une agent application de monitoring ; requis par certains appels d&apos;Python API deagent .

## Description

Cet appel renvoie un objet correspondant à une agent application-monitorer. Si aucune valeur `name` n&apos;est définie, le nom [défini via la configuration de l&apos;agent Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration#app_name) est utilisé.

L&apos;objet application renvoyé ne présente aucune API publique, mais est requis par un autre appel d&apos;API.

<Callout variant="important">
  Cet appel renvoie uniquement l&apos;objet d&apos;application. Il ne tentera pas d&apos;enregistrer l&apos;application auprès du collecteur de données si cela n&apos;a pas déjà eu lieu.
</Callout>

## Valeurs de retour

Renvoie un objet d&apos;application. L&apos;objet lui-même ne présente aucune API publique, mais est requis par un autre appel d&apos;API.

## Exemples

### Générer et utiliser un objet d&apos;application [#app-object-example]

Pour générer un objet d’application, utilisez :

```py
import newrelic.agent

app = newrelic.agent.application()
```

Certains autres appels d’API d’agent Python prennent l’objet d’application comme paramètre. Pour un exemple, voir l’appel [`notice_error`](/docs/agents/python-agent/python-agent-api/noticeerror-python-agent-api/) .
