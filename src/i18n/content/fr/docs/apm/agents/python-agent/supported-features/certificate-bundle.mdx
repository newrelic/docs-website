## Questions fréquemment posées [#faq]

### Qu&apos;est-ce qu&apos;un ensemble de certificats ? [#what-is-certificate-bundle]

Un ensemble de certificats fournit essentiellement une liste de racines de confiance. Il indique à l&apos;agent à qui faire confiance lors de la communication avec le backend.

### Pourquoi le pack de certificats est-il fourni ? [#why-certificate-bundle]

De nombreux clients fonctionnent dans Docker. Parfois, ces images ne fournissent pas leurs propres ensembles de certificats, car les images ne sont généralement pas utilisées pour communiquer via HTTP. Nous fournissons notre propre pack afin que les clients puissent installer l&apos;agent plus facilement.

### Quand le pack de certificats est-il utilisé ? [#when-to-use-certificate-bundle]

Lorsque des certificats système sont disponibles, l&apos;agent utilisera par défaut ceux-ci plutôt que le pack de certificats inclus. Le groupe de certificats n&apos;est utilisé que lorsqu&apos;aucun certificat n&apos;est fourni par le système.
