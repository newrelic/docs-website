## Syntaxe

```py
newrelic.agent.disable_browser_autorum(flag=True)
```

Désactive <InlinePopover type="browser" />pour une requête Web.

## Description

Cet appel désactive [monitoring des navigateurs](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) pour la page/requête pour laquelle il est appelé. Cet appel affecte uniquement la requête Web actuelle et doit être appelé avant le point où l&apos;en-tête/pied de page JavaScript de l&apos; agent de navigateur est injecté. Pour en savoir plus sur la manière dont l&apos; agent de navigateur JavaScript est inséré, voir [Installer monitoring des navigateurs](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser).

<Callout variant="tip">
  Vous pouvez également désactiver de navigateur injection en définissant la `newrelic.disable_browser_autorum` clé pour la demande spécifique dans le WSGI dictionnaire d&apos;environnement transmis par le WSGI serveur à l&apos;application WSGI monitorée.
</Callout>

## Exemples

### Désactiver monitoring des navigateurs [#example]

Dans cet exemple, vous avez mis à niveau l&apos;agent et vous avez remarqué que le JavaScript N qui est automatiquement injecté sur la page HTML est également injecté dans chaque iFrame chargé sur ces pages. Cela a entraîné une augmentation substantielle du nombre de pages vues.
