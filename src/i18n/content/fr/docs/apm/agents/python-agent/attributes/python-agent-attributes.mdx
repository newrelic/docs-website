[Les attributs](/docs/features/agent-attributes) sont des paires valeur clé contenant des informations qui déterminent les propriétés d&apos;un événement ou d&apos;une transaction. Nous permettons à l&apos;utilisateur de personnaliser exactement quel attribut sera envoyé à chacune de ces destinations :

* [trace de transaction](/docs/apm/transactions/transaction-traces/transaction-traces)
* [erreurs de trace](/docs/apm/applications-menu/events/viewing-apm-errors-error-traces)
* [Événement de transaction](/docs/insights/new-relic-insights/decorating-events/apm-default-attributes-insights)
* [Pages vues](/docs/insights/new-relic-insights/decorating-events/browser-default-attributes-insights)

Ces paramètres d’attribut s’appliquent à l’agent Python version 2.56.0.42 ou supérieure. Pour profiter de toutes les fonctionnalités disponibles, [mettez à jour votre Python agent](/docs/agents/python-agent/installation-configuration/upgrade-python-agent) [vers](/docs/release-notes/agent-release-notes/python-release-notes) la dernière version .

## Ajouter un attribut personnalisé [#adding\_custom\_attributes][#adding_custom_attributes]

Pour capturer des attributs personnalisés supplémentaires à partir de votre application, utilisez `newrelic.agent.add_custom_attribute()`.

Pour une référence complète, voir [Collecte d&apos;attributs personnalisés](/docs/apm/other-features/attributes/collecting-custom-attributes#python-att).

Valeurs par défaut :

* trace de transaction : Activé
* Collecteur d&apos;erreurs (trace des erreurs) : Activé
* Événement de transaction : Activé
* Pages vues (monitoring des navigateurs) : Désactivé

<Callout variant="important">
  Avant de créer un attribut personnalisé, consultez notre liste de [termes réservés](/docs/insights/event-data-sources/custom-events/data-requirements-limits-custom-event-data/#reserved-words). Dans le cas contraire, des résultats inattendus pourraient survenir.
</Callout>

* Collecteur d&apos;erreurs (trace des erreurs) : Activé
* Événement de transaction : Indisponible
* Pages vues (monitoring des navigateurs) : Indisponible

## Mettez à jour votre fichier de configuration [#upgrading]

Mettez à niveau votre fichier de configuration <DNT>**newrelic.ini**</DNT> lorsque vous effectuez une mise à niveau vers l&apos;agent Python <DNT>**2.56.0.42**</DNT> ou supérieur. Pour plus d&apos;informations sur les paramètres configuration obsolètes, voir [Activation et désactivation de l&apos;attribut](/docs/agents/python-agent/attributes/enabling-disabling-attributes-python#deprecated).
