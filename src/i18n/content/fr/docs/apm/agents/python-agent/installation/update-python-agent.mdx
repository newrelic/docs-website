Pour profiter pleinement de nos dernières fonctionnalités, améliorations et correctifs de sécurité importants, nous vous recommandons de mettre à jour Python agent votre vers la dernière version. Pour plus d&apos;informations sur agent les mises à jour spécifiques, reportez-vous aux [Python agent notes de sortie](/docs/release-notes/agent-release-notes/python-release-notes).

## Mise à jour avec Python package l&apos;index (PyPi) [#package-index]

Le package de l&apos;agent Python est répertorié sur PyPi à l&apos; [adresse http://pypi.python.org/pypi/newrelic](http://pypi.python.org/pypi/newrelic "Le lien s'ouvre dans une nouvelle fenêtre"). Pour mettre à jour avec un package spécifique, suivez les instructions correspondantes :

        ```sh
        pip install --upgrade newrelic
        ```


## Télécharger le package manuellement [#manual-download]

Pour télécharger manuellement le package Python New Relic :

1. Supprimez le package New Relic actuel.
2. Téléchargez le nouveau package, puis suivez les procédures [d&apos;installation de l&apos;agent Python](/docs/agents/python-agent/installation-configuration/python-agent-installation#download-site) .

<Callout variant="tip">
  Nous vous recommandons d&apos;utiliser `pip` plutôt que d&apos;utiliser l&apos;installation manuelle, car `pip` supprimera correctement les anciennes versions lors de la mise à jour du logiciel de l&apos;agent.
</Callout>
