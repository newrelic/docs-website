## Problème

Vous souhaitez émuler le legacy comportement consistant à activer les <DNT>**Capture attributes**</DNT> <DNT>**Capture parameters**</DNT> options ou dans les paramètres UI de [configuration côté serveur de](/docs/agents/manage-apm-agents/configuration/server-side-agent-configuration) New Relic.

## Solution

Vous pouvez émuler ce legacy comportement en personnalisant votre Python configuration fichier d’agent pour capturer les paramètres de requête. Pour plus de détails, consultez l’ [exemple de paramètres de demande de capture](/docs/agents/python-agent/attributes/python-attribute-examples#ex_req_params) dans la documentation des attributs Python .
