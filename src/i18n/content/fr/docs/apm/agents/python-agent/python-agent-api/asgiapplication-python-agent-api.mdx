## Syntaxe

```py
newrelic.agent.asgi_application(application=None, name=None, group=None, framework=None)
```

Monitorer les transactions Web en marquant le point d&apos;entrée application ASGI.

## Description

`asgi_application` est un décorateur Python utilisé pour surveiller les transactions Web en instrumentant le point d&amp;apos;entrée application ASGI. L&amp;apos;agent Python prend automatiquement [en charge la plupart des frameworks et serveurs](/docs/agents/python-agent/getting-started/compatibility-requirements-python-agent) qui utilisent ASGI. Si votre framework ou votre serveur Web n&amp;apos;est pas pris en charge, vous devrez peut-être utiliser cette API dans le cadre du [processus d&apos;intégration avancé](/docs/agents/python-agent/installation-configuration/python-agent-integration#manual-integration).

Si vous ne pouvez pas utiliser le décorateur `asgi_application`, vous pouvez utiliser l&amp;apos;un de ces autres formats d&amp;apos;appel :

* <DNT>**The wrapper:**</DNT> L&amp;apos;appel wrapper est `ASGIApplicationWrapper`. Vous pouvez utiliser le wrapper à plusieurs endroits pour des composants d&amp;apos;application ASGI distincts qui peuvent être empilés. Dans ce cas, le premier wrapper rencontré marque le début de la transaction et l&amp;apos;agent détermine l&amp;apos;application cible en fonction de ce premier wrapper (et ignore les suivants).
* <DNT>**The path-based wrapper:**</DNT> L&amp;apos;appel wrapper basé sur le chemin est `wrap_asgi_application`. Utilisez ceci si vous ne pouvez pas référencer l’objet ASGI en tant que variable dans la portée de l’instrumentation. Cela prend le même paramètre que le décorateur avec les paramètres `module` et `object_path` supplémentaires.

Pour une explication des raisons d&apos;utiliser le décorateur par rapport aux wrappers, voir [Différents formats d&apos;appel](/docs/python-agent-api-different-call-forms).

## Exemples

### Exemple d&apos;application asgi [#decorator-example]

Un exemple de décorateur appelé sans le paramètre facultatif `application` :

```py
@newrelic.agent.asgi_application()
async def application(scope, receive, send):
    await send(
        {
            "type": "http.response.start",
            "status": 200,
            "headers": [(b"Content-type", b"text/plain")]
        }
    )
    await send(
        {
            "type": "http.response.body",
            "body": b"Hello World!"
        }
    )
```

### Exemple d&apos;ASGIApplicationWrapper [#wrapper-example]

Un exemple d&apos;utilisation du `ASGIApplicationWrapper`, qui peut être nécessaire si le décorateur `asgi_application` ne fonctionne pas :

```py
import os
os.environ.setdefault("DJANGO_SETTINGS_MODULE", "mysite.settings")
from django.core.asgi import get_asgi_application
application = get_asgi_application()

application = newrelic.agent.ASGIApplicationWrapper(application)
```

### Exemple d&apos;application wrap\_asgi [#path-based-example]

Un exemple d&apos;utilisation du wrapper basé sur le chemin :

```py
import newrelic.agent
newrelic.agent.initialize('newrelic.ini')

class Application:
   def __init__(self, scope):
      self.scope = scope

   async def __call__(self, receive, send):
      await send({"type": "http.response.start", "status": 200})
      await send({"type": "http.response.body", "body": b"Hello World!"})

newrelic.agent.wrap_asgi_application(__name__, 'Application')
```
