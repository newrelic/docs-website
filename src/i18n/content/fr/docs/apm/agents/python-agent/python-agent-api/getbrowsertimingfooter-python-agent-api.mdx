Génère un pied de page d&apos;agent <InlinePopover type="browser" />pour injecter l&apos;agent JavaScript.

## Description [#description]

Cet appel renvoie de agent pied de page JavaScript de l&apos; du navigateur snippet qui, avec [l&apos; JavaScript snippet](/docs/agents/python-agent/python-agent-api/get_browser_timing_header) d&apos;en-tête, est utilisé pour activer monitoring des navigateurs. Pour savoir comment activer monitoring des navigateurs à l&apos;aide de cet appel avec `get_browser_timing_header`, voir [instrumenter manuellement monitoring des navigateurs](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).

Cet appel doit être appelé pour chaque requête Web où vous souhaitez injecter le snippet JavaScript. Il ne doit pas (<DNT>**cannot**</DNT>) être appelé au niveau global et mis en cache.

## Valeurs de retour [#return-values]

Renvoie le pied de page de l&apos; agent de navigateur JavaScript.
