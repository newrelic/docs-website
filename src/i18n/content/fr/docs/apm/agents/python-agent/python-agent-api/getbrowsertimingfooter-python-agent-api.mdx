  À partir de la version 9.4.0+ de l'agent Python, cette API est obsolète et renvoie toujours une chaîne vide. La fonctionnalité précédente de `get_browser_timing_footer()` est désormais incluse dans `get_browser_timing_header()`.

  Pour plus d'informations sur l'utilisation correcte, voir [instrumenter manuellement monitoring des navigateurs](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).

Génère un pied de page d'agent <InlinePopover type="browser" />pour injecter l&amp;apos;agent JavaScript.

## Description [#description]

Cet appel renvoie de agent pied de page JavaScript de l' du navigateur snippet qui, avec [l' JavaScript snippet](/docs/agents/python-agent/python-agent-api/get_browser_timing_header) d&amp;apos;en-tête, est utilisé pour activer monitoring des navigateurs. Pour savoir comment activer monitoring des navigateurs à l&amp;apos;aide de cet appel avec `get_browser_timing_header`, voir [instrumenter manuellement monitoring des navigateurs](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).

Cet appel doit être appelé pour chaque requête Web où vous souhaitez injecter le snippet JavaScript. Il ne doit pas (<DNT>**cannot**</DNT>) être appelé au niveau global et mis en cache.

## Valeurs de retour [#return-values]

Renvoie le pied de page de l' agent de navigateur JavaScript.
