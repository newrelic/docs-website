## Syntaxe

```py
newrelic.agent.callable_name(object, separator=':')
```

Renvoie un nom de chaîne identifiant l&apos;objet fourni.

## Description

Cet appel renvoie un nom de chaîne identifiant l&apos;objet fourni. Ceci est souvent utilisé pour nommer par programmation [la fonction trace](/docs/agents/python-agent/python-agent-api/function_trace).

## Valeurs de retour

Renvoie un nom de chaîne identifiant l&apos;objet fourni.

Le nom renvoyé sera sous la forme `module:object_path`. Si l&amp;apos;objet fourni est une fonction, le nom renvoyé sera sous la forme `module:function`. Si l&amp;apos;objet est une classe, le nom sera sous la forme `module:class`. S&amp;apos;il s&amp;apos;agit d&amp;apos;une fonction membre, la forme serait `module:class.function`.

## Exemples

### Définir le nom de la fonction trace [#trace-example]

Voici un exemple d&apos;utilisation de `callable_name` pour définir le nom d&amp;apos;une [trace de fonction](/docs/agents/python-agent/python-agent-api/function_trace):

```py
name = callable_name(func)

with FunctionTrace(txn, name):
    func()
```
