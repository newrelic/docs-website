## Syntaxe

```py
newrelic.agent.wrap_mlmodel(model, name=None, version=None, feature_names=None, label_names=None, metadata=None)
```

Permet l&apos;instrumentation manuelle des modèles d&apos;apprentissage machine.

## Exigences

Python agent version 9.1.0 ou supérieur.

## Description

Cela permet l’instrumentation manuelle des modèles d’apprentissage machine.

## Exemples

### Envelopper le modèle d&apos;apprentissage machine [#wrap-ml-model]

Un exemple d&apos;instrumentation d&apos;un modèle d&apos;apprentissage machine personnalisé :

```py
def wrap_ml_example():
    x_train = [[0, 0], [1, 1]]
    y_train = [0, 1]
    x_test = [[1.0, 2.0]]

    model = CustomTestModel().fit(x_train, y_train)
    wrap_mlmodel(
        model,
        name="MyCustomModel",
        version="1.2.3",
        feature=["feature0", "feature1"],
        label=["label0"],
        metadata={"metadata1": "value1", "metadata2": "value2"},
    )

    labels = model.predict(x_test)

    return model
```
