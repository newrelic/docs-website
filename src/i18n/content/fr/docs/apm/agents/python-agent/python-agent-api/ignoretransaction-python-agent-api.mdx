## Syntaxe

```py
newrelic.agent.ignore_transaction(flag=True)
```

Ignore la transaction en cours.

## Description

Cet appel ignore la transaction en cours. L&apos;agent ne rapporte aucune donnée de la transaction.

<Callout variant="tip">
  Vous pouvez également ignorer un Web de transaction dans le dictionnaire d&apos;environnement WSGI . Pour ce faire, définissez la clé `newrelic.ignore_transaction` pour la demande spécifique dans le dictionnaire d’environnement WSGI transmis par le serveur WSGI à votre application WSGI.
</Callout>

Aucun.

## Exemples

### Ignorer une transaction [#ignore-example]

Pour ignorer une transaction que vous ne souhaitez pas afficher dans l&apos; UI, exécutez la commande suivante lorsqu&apos;une transaction est générée dans le code de votre application. Cet exemple illustre l&apos;utilisation de l&apos;API pour ignorer une route Flask :

```py
@app.route("/")
def process_results():
    newrelic.agent.ignore_transaction(flag=True)
    return 'hello world'
```
