## Problème

Après avoir installé l&apos;agent Python , généré du trafic et attendu cinq minutes, vos données n&apos;apparaissent pas dans l&apos;UI de New Relic.

## Solution

Voici quelques conseils pour le dépannage des données manquantes, classés par ordre approximatif de probabilité qu&apos;elles soient à l&apos;origine du problème :

1. Assurez-vous d’avoir installé l’agent dans le même environnement que votre application. Si vous utilisez un environnement virtuel, installez-le là.
2. Assurez-vous que votre agent est à jour. Si vous l&apos;avez téléchargé depuis le site New Relic plutôt que d&apos;utiliser <DNT>**pip**</DNT> ou <DNT>**easy\_install**</DNT>, comparez votre numéro de version à la [sortie actuelle](https://download.newrelic.com/python_agent/release/).
3. Utilisez [New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics) pour essayer d’identifier automatiquement le problème.
4. Assurez-vous que les données peuvent accéder à l&apos;UI New Relic en exécutant le [test de l&apos;agentPython ](/docs/python/testing-the-python-agent).
5. Assurez-vous de respecter la [compatibilité et les exigences](/docs/agents/python-agent/getting-started/compatibility-requirements-python-agent). Vérifiez les exigences particulières pour le [framework Web ou le serveur Web](/docs/agents/python-agent/hosting-mechanisms) que vous utilisez.
6. Vérifiez le fichier de configuration généré avec la commande `newrelic-admin generate-config` pour vous assurer qu&amp;apos;il contient le bon <InlinePopover type="licenseKey" />.
7. [Générer un log détaillé du dépannage](/docs/agents/python-agent/troubleshooting/generating-logs-troubleshooting). Vérifiez ensuite le fichier de log pour le message d&amp;apos;erreur. Si le fichier de log est manquant, assurez-vous que vous disposez des autorisations pour écrire à cet emplacement.

De plus, si aucune donnée n&apos;apparaît et que vous voyez des messages d&apos;avertissement concernant l&apos;activation ou la réactivation de votre application, suivez les [procédures de dépannage d&apos;activation](/docs/agents/python-agent/troubleshooting/activate-application-warning-python).
