À partir de l&apos;agent Python New Relic v9.1.0, New Relic permet désormais monitoring les modèles d’apprentissage machine. Ces modèles ML de monitoring peuvent être trouvés dans l&apos;APM dans la section Modèles.

## Installation

<Callout variant="tip">
  Les métriques du modèle ML sont disponibles pour l&apos;agent Python version 9.1.0 et supérieurs, mais sont désactivés par défaut. Pour modifier cette configuration, consultez notre [documentation](/docs/apm/agents/python-agent/configuration/python-agent-configuration#ml-settings).
</Callout>

Les paramètres ML peuvent être trouvés [ici](/docs/apm/agents/python-agent/configuration/python-agent-configuration/#ml-settings).

Pour modifier la taille de récolte ML par défaut de 100 000 toutes les 5 secondes, définissez `event_harvest_config.harvest_limits.ml_event_data` dans votre fichier `newrelic.ini` sur la valeur souhaitée ou définissez la variable d&apos;environnement `NEW_RELIC_ML_INSIGHTS_EVENTS_MAX_SAMPLES_STORED` sur la valeur souhaitée :

```bash
export NEW_RELIC_ML_INSIGHTS_EVENTS_MAX_SAMPLES_STORED=${desired_value}
```

## API d&apos;apprentissage machine [#ml-apis]

Deux nouvelles API existent pour personnaliser l&apos;expérience d&apos;instrumentation ML :

* Enregistrer un événement ML personnalisé : [record\_ml\_event](/docs/agents/python-agent/python-agent-api/recordmlevent-python-agent-api)
* Instrumenter manuellement un modèle ML : [wrap\_mlmodel](/docs/agents/python-agent/python-agent-api/wrapmlmodel-python-agent-api)

## Assurer la confidentialité des données [#data-privacy]

<Callout variant="caution">
  Vous contrôlez les données log envoyées à New Relic, alors assurez-vous de suivre les consignes de sécurité de votre organisation pour masquer, obscurcir ou empêcher l&apos;envoi d&apos;informations personnelles identifiables (PII), d&apos;informations de santé protégées (PHI) ou de toute autre donnée sensible.
</Callout>

Vous pouvez également activer ou désactiver l&apos;envoi de valeurs d&apos;inférence brutes en fonction des paramètres de confidentialité souhaités [ici](/docs/apm/agents/python-agent/configuration/python-agent-configuration/#inference-events-value-enabled).
