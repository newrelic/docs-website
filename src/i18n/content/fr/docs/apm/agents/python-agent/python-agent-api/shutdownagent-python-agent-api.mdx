## Syntaxe

```py
newrelic.agent.shutdown_agent(timeout=None)
```

Arrête l'agent de force.

## Description

Arrêter l'agent de force. L'agent fait une dernière tentative pour télécharger les données vers le [collecteur](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector).

        `timeout`

        *int* ou *virgule flottante*
      </td>

      <td>
        Facultatif, mais fortement recommandé. Durée maximale pendant laquelle l'appelant attend que l'agent termine le téléchargement des données. Si le délai d'attente se termine avant que l'agent ne termine, l'agent continue de signaler les données en arrière-plan, mais l'appelant peut arrêter de force l'ensemble du processus et arrêter le téléchargement.

        Si aucune valeur n'est définie ou définie sur `None`, l&amp;apos;agent utilise la valeur de délai d&amp;apos;expiration globale par défaut, qui peut être remplacée dans les paramètres de configuration mais dont la valeur par défaut est de 2,5 secondes.
      </td>
    </tr>

## Valeurs de retour

Aucun.

## Exemples

### Avec un délai d'attente de trois secondes [#timeout-example]

```py
newrelic.agent.shutdown_agent(timeout=3)
```
