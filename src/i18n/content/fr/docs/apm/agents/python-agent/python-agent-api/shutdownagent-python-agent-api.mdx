## Syntaxe

```py
newrelic.agent.shutdown_agent(timeout=None)
```

Arrête l&apos;agent de force.

## Description

Arrêter l&apos;agent de force. L&apos;agent fait une dernière tentative pour télécharger les données vers le [collecteur](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector).

      <td>
        `timeout`

        *int* ou *virgule flottante*
      </td>

      <td>
        Facultatif, mais fortement recommandé. Durée maximale pendant laquelle l&apos;appelant attend que l&apos;agent termine le téléchargement des données. Si le délai d&apos;attente se termine avant que l&apos;agent ne termine, l&apos;agent continue de signaler les données en arrière-plan, mais l&apos;appelant peut arrêter de force l&apos;ensemble du processus et arrêter le téléchargement.

        Si aucune valeur n&apos;est définie ou définie sur `None`, l&amp;apos;agent utilise la valeur de délai d&amp;apos;expiration globale par défaut, qui peut être remplacée dans les paramètres de configuration mais dont la valeur par défaut est de 2,5 secondes.
      </td>
    </tr>
  </tbody>
</table>

## Valeurs de retour

Aucun.

## Exemples

### Avec un délai d&apos;attente de trois secondes [#timeout-example]

```py
newrelic.agent.shutdown_agent(timeout=3)
```
