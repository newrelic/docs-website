Ce document explique comment désinstaller l&apos;agent Python. Pour obtenir des instructions sur la façon de désactiver temporairement l&apos;agent, voir [Désactiver l&apos;agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#python).

Pour désinstaller votre agent Python :

1. Utilisez l’une de ces méthodes :

   * Si vous avez installé avec PIP, exécutez :

     ```sh
     pip uninstall newrelic
     ```

   * Si vous avez installé avec `easy_install`, exécutez :

     ```sh
     easy_install -m newrelic
     ```

2. Une fois le processus de désinstallation terminé, redémarrez votre application.

<Callout variant="tip">
  Vous souhaiterez peut-être également [supprimer votre application](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) de l&apos;UI APM.
</Callout>
