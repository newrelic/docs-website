## Syntaxe

```py
newrelic.agent.end_of_transaction()
```

Termine la transaction en cours.

## Description

Cet appel met fin immédiatement à la transaction en cours et arrête le minuteur de réponse à l&apos;heure actuelle. Une fois la transaction terminée, l&apos; agent ne collecte aucune mesure de répartition des performances supplémentaire. Cependant, l&apos;agent collecte toujours les détails des exceptions Python.

Cet appel est utile lorsque vous avez une réponse volumineuse qui est renvoyée vers le Web de transaction et que seule la partie initiale de la transaction vous intéresse. (Cela suppose également que, pour une raison quelconque, vous ne souhaitez pas marquer la transaction comme une [tâche d&apos;arrière-plan](/docs/agents/python-agent/python-agent-api/set_background_task).)

## Paramètres

Aucun.

## Valeurs de retour

Aucun.

### Terminer une transaction [#end-txn-example]

newrelic.agent.end_of_transaction()
