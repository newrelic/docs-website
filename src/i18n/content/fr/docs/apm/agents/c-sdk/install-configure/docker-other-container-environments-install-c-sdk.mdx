Vous pouvez installer le [SDKNew Relic C](/docs/agents/c-sdk/get-started/introduction-c-sdk) sur un conteneur Docker (ou autre conteneur) pour monitorer une ou plusieurs de vos applications C.

## Installer le C SDK dans un environnement de conteneur [#install-container]

Exigences:

* Répondre aux [exigences du C SDK](/docs/agents/c-sdk/get-started/c-sdk-compatibility-requirements)
* C SDK version 1.2 ou supérieure

<Callout variant="caution">
  Les données transmises de l&apos;agent au démon ne sont **pas** cryptées. Nous vous recommandons d&amp;apos;utiliser uniquement une connexion réseau privée entre l&amp;apos;agent et le démon. Cela s&amp;apos;applique uniquement lorsque l&amp;apos;agent et le démon s&amp;apos;exécutent sur des hôtes différents.
</Callout>

Pour installer le C SDK pour un environnement de conteneur, nous vous recommandons d&apos;installer le [démon du C SDK](/docs/agents/c-sdk/get-started/introduction-c-sdk#architecture) sur un conteneur Docker distinct :

1. Installez le démon en clonant le [référentiel C SDK](https://github.com/newrelic/c-sdk) et en créant le démon. Le moyen le plus efficace pour cela est d&amp;apos;utiliser la commande `make daemon`. Consultez le [fichier README.md](https://github.com/newrelic/c-sdk/blob/master/README.md#building-the-c-sdk) pour plus de détails.

   Si vous utilisez Docker, vous pouvez utiliser l&apos;[image du démon C SDK sur Dockerhub](https://hub.docker.com/r/newrelic/c-daemon).

2. Démarrez le démon en utilisant les arguments `--address` et `--watchdog-foreground` . L&amp;apos;argument `--address` est utilisé pour définir un port sur lequel le démon accepte les connexions. L&amp;apos;argument `--watchdog-foreground` garantit que le démon s&amp;apos;exécute au premier plan.

Ensuite, utilisez le C SDK dans votre application conteneurisée :

1. Suivez les étapes pour [ajouter le C SDK à votre code](/docs/agents/c-sdk/install-configure/install-c-sdk-compile-link-your-code).
2. Dirigez le C SDK vers le démon en ajoutant un [appel`newrelic_init` ](https://newrelic.github.io/c-sdk/libnewrelic_8h.html#a03f07e64a51b5f0cd51caa4f28c8b6c1)et en passant un argument `address` valide. La valeur de cet argument doit être `HOST:PORT`, où `HOST` est le nom ou l&amp;apos;adresse IP de l&amp;apos;hôte sur lequel le démon s&amp;apos;exécute et `PORT` est le numéro de port sur lequel le démon écoute.
