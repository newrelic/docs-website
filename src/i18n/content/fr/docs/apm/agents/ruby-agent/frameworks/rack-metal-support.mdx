[Les middlewaresRack ](/docs/agents/ruby-agent/frameworks/rack-middlewares)sont automatiquement instrumentés avec l&apos;agent Ruby version 3.9.0 ou supérieure.

Le module `NewRelic::Agent::Instrumentation::Rack` est obsolète. Après la mise à niveau vers la version 3.9.0 ou supérieure de l&apos;agent Ruby, New Relic vous recommande de supprimer toutes les références à ce module de votre code. Pour plus d&apos;informations, voir [Contrôleurs métalliques instrumentés](/docs/agents/ruby-agent/frameworks/instrumenting-metal-controllers).
