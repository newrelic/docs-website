## Problème

Vous recevez ce message log que Grape ne s&apos;installe pas :

```
INFO : Not installing New Relic supported Grape instrumentation because the third party newrelic-grape gem is present
```

## Solution

Retirez la gem `newrelic-grape` .

## Cause

New Relic ne prend pas en charge la gem tierce `newrelic-grape` . Si l&amp;apos;agent Ruby détecte la gem `newrelic-grape` , il n&amp;apos;installe pas [l&apos;instrumentation Grape intégrée](/docs/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) de New Relic. La suppression du joyau tiers vous permet d&amp;apos;utiliser l&amp;apos;instrumentation Grape prise en charge.
