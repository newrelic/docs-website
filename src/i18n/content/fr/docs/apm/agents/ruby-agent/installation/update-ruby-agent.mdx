Pour profiter pleinement des dernières fonctionnalités, améliorations et correctifs de sécurité importants de New Relic, nous vous recommandons de mettre à jour votre agent Ruby vers la dernière version. Pour plus d&apos;informations sur les mises à jour spécifiques des agents, reportez-vous aux [notes de sortie de l&apos;agentRuby ](/docs/release-notes/agent-release-notes/ruby-release-notes).

## Vérifiez vos versions d’agent Ruby [#determine-versions]

Pour déterminer quelles versions d’agent Ruby vos applications utilisent actuellement :

1. Dans New Relic, dans le [menu utilisateur](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings), sélectionnez <DNT>**Account settings &gt; Connected agents**</DNT>.
2. Facultatif : Trier par <DNT>**Oldest agent version**</DNT>.

## Mettre à jour l&apos;agent [#Updating\_the\_Agent][#Updating_the_Agent]

Pour mettre à jour l’agent Ruby :

* Utilisation de Bundler : exécutez cette commande :

  ```sh
  bundle update newrelic_rpm
  ```

* Ne pas utiliser Bundler : exécutez cette commande :

  ```sh
  gem update newrelic_rpm
  ```
