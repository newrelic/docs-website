L&apos;agent Ruby New Relic est désormais entièrement open source et disponible sur GitHub à l&apos;adresse [newrelic-ruby-agent](https://github.com/newrelic/newrelic-ruby-agent/tree/main). À partir de la version 6.12 (sortie le 16 juillet 2020), il est sous licence Apache, version 2.0. Pour les versions d&amp;apos;agent plus anciennes, vérifiez le fichier LICENSE inclus avec le code source. Le code source des versions 2.1.0 de l&amp;apos;agent Ruby sont disponibles sur GitHub en utilisant le tag.

Le chef de `main` aura la dernière sortie de New Relic. Nous nous réservons cependant la possibilité de pousser un bord vers le `main`. La branche `dev` inclut des travaux inédits. Si vous téléchargez une sortie depuis GitHub, utilisez la tag appropriée . Nous sortons généralement les versions d&amp;apos;aperçu via GitHub avant de publier une nouvelle version sur RubyGems.

N&apos;hésitez pas à [nous envoyer des demandes de tirage](https://github.com/newrelic/newrelic-ruby-agent/pulls), [à créer des tickets](https://github.com/newrelic/newrelic-ruby-agent/issues/new/choose) et [à lancer des discussions](https://github.com/newrelic/newrelic-ruby-agent/discussions) directement dans le référentiel GitHub (référentiel). Nous aimons travailler avec la communauté !
