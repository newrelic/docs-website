Ce document explique comment désinstaller l'agent Ruby New Relic. Pour obtenir des instructions sur la façon de désactiver temporairement l'agent, voir [Désactiver l'agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#ruby).

Pour désinstaller l’agent Ruby :

1. Supprimez la ligne `gem 'newrelic_rpm'` de votre Gemfile.
2. Supprimez le fichier <DNT>**newrelic.yml**</DNT> de votre dossier <DNT>**APP\_ROOT/config/**</DNT> .
3. Exécutez <DNT>**bundle install**</DNT>.
4. Une fois le processus de désinstallation terminé, redémarrez votre application.

Pour plus d'informations sur l'installation, voir [New Relic pour Ruby](/docs/agents/ruby-agent/getting-started/new-relic-ruby).
