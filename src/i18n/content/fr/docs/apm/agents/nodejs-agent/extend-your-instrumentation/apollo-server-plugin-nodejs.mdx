Le plugin New Relic Apollo Server instrumente votre application Apollo Server pour donner une visibilité sur votre charge GraphQL . Cela vous aide à découvrir et à diagnostiquer la cause de votre [requêteGraphQL ](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph)lente. La version d&apos;Apollo Server prise en charge est 2.14 ou ultérieure.

Notre plugin enregistre les temps globaux des requêtes et utilise [le traçage distribué](/docs/understand-dependencies/distributed-tracing/get-started/how-new-relic-distributed-tracing-works) pour découvrir le problème d&apos;itinéraire. Utilisez cette instrumentation pour voir si le problème provient de la résolution d&apos;une donnée demandée ou s&apos;il provient d&apos;un travail effectué sur d&apos;autres services ou bases de données.

## Compatibilité

Le plugin New Relic fonctionne avec les modules Apollo Server suivants :

* `@apollo/server`
* `@apollo/gateway`
* `@apollo/subgraph`
* `apollo-server` (&gt;= 2.14)
* `apollo-server-express`
* `apollo-server-hapi`
* `apollo-server-koa`
* `apollo-server-fastify`
* `apollo-server-lambda`

D&apos;autres plugins peuvent fonctionner, en fonction de leur implémentation sous-jacente, mais ils n&apos;ont pas été vérifiés.
