Lisez la suite pour savoir comment désinstaller l’agent New Relic Node.js. Pour obtenir des instructions sur la façon de désactiver temporairement l&apos;agent, voir [Désactiver l&apos;agent](/docs/agents/manage-apm-agents/installation/disable-apm-agent#node).

Pour désinstaller l’agent New Relic Node.js :

1. Supprimez `require('newrelic');` comme première ligne du module principal de l&apos;application.
2. Supprimez le package npm avec `npm uninstall newrelic --save`.
3. Supprimez le fichier `newrelic.js` et annulez toutes [les variables d’environnement](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration#methods-and-precedence) utilisées.
4. Une fois le processus de désinstallation terminé, [redémarrez](#restart) votre application.

Pour plus d&apos;informations sur l&apos;installation, consultez [Installation et maintenance de Node.js](/docs/agents/nodejs-agent/installation-configuration/installing-maintaining-nodejs).

<Callout variant="tip">
  Vous souhaiterez peut-être également [supprimer votre application](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) de New Relic.
</Callout>
