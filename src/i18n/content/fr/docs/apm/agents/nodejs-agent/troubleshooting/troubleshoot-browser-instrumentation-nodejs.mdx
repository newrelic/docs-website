## Problème

Si vous rencontrez des problèmes avec [l'application du navigateur instrumenté](/docs/nodejs/page-load-timing-in-nodejs) via votre New Relic Node.js agent, suivez les [procédures de <InlinePopover type="browser" />dépannage standard](/docs/new-relic-browser/troubleshooting-page-view-monitoring). Voici quelques conseils supplémentaires pour Node.js.

## Solution

Les codes d'erreur apparaîtront automatiquement dans la source du code du site et dans le Node.js agent log de votre Recherchez [`NREUM`](/docs/new-relic-browser/troubleshooting-page-view-monitoring#javascript) pour trouver ces codes.
