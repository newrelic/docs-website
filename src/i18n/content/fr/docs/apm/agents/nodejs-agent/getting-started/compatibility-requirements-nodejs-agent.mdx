Notre agent Node.js inclut une instrumentation intégrée des frameworks Node.js, des serveurs d&apos;applications, des bases de données et des systèmes de mise en file d&apos;attente de messages les plus populaires. Pour les frameworks et les bibliothèques qui ne sont pas instrumentés par défaut, vous pouvez étendre l&apos;agent avec notre [API d&apos;agent Node.js](https://newrelic.github.io/node-newrelic/API.html).

Notre Node.js agent est disponible publiquement sur le [référentiel Node Package Manager (npm)](https://npmjs.org/package/newrelic) ainsi que sur [GitHub](https://github.com/newrelic/node-newrelic). Prêt à tester l&amp;apos;agent Node.js de New Relic ? [Créez un compte New Relic](https://newrelic.com/signup)!

## Conditions requises pour installer l&apos;agent

Avant [d&apos;installer l&apos;agent](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent), vérifiez que votre système répond à sa configuration minimale requise. Pour de meilleures performances, utilisez la dernière version de support actif à long terme (LTS) de Node.js.

<CollapserGroup>
  <Collapser id="operating systems" title="Système d'exploitation compatible">
    L&apos;agent Node.js est compatible avec le système d&apos;exploitation suivant :

    * Linux
    * Système d&apos;exploitation SmartOS
    * macOS 10.7 et versions ultérieures
    * Windows Server 2008 et versions ultérieures
  </Collapser>

  <Collapser id="host" title="Services d'hébergement">
    Il est également compatible avec ces services d&apos;hébergement :

    * [Heroku](/docs/nodejs/nodejs-agent-on-heroku)
  </Collapser>

  <Collapser id="security" title="Exigences de sécurité">
