Pour profiter pleinement des dernières fonctionnalités, améliorations et correctifs de sécurité importants de New Relic, mettez à jour l&apos;agent Go de votre application vers la dernière version. Pour plus d&apos;informations sur les mises à jour spécifiques agent, reportez-vous aux [notes de sortie d&apos;agent Go](/docs/release-notes/agent-release-notes/go-release-notes).

## Mettez à jour votre version d&apos;agent Go [#update-go]

Pour mettre à jour l&apos;agent Go, suivez vos procédures standard pour exécuter le processus suivant en tant que commande de terminal ou en tant que script.

1. Depuis [http://github.com/newrelic/go-agent](http://github.com/newrelic/go-agent), utilisez ce processus :

   ```bash
   go get -u github.com/newrelic/go-agent/v3/newrelic
   ```

2. Compilez et diffusez votre application.
