Si vous n&apos;en avez pas déjà un, [créez un compte New Relic](https://newrelic.com/signup). C&amp;apos;est gratuit, pour toujours. Avant d&amp;apos;[installer New Relic for Go](/docs/agents/go-agent/installation/install-new-relic-go), assurez-vous que votre système répond à ces exigences.

## Versions de Golang [#golang-release]

New Relic prend en charge [Golang 1.19 ou supérieur](https://go.dev/doc/devel/release).

## Environnements d&apos;exploitation [#operating-systems]

L&apos;agent prend en charge <DNT>Linux</DNT>, <DNT>macOS</DNT> et <DNT>Windows</DNT>. Vous pouvez également utiliser l&amp;apos;agent Go dans un [environnement flexible Google App Engine (GAE)](/docs/agents/go-agent/get-started/install-new-relic-go-agent-gae-flexible-environment).

## intégration [#frameworks]

Le package [d&apos;intégration](https://godoc.org/github.com/newrelic/go-agent/_integrations) suivant étend le de [base newrelic](https://godoc.org/github.com/newrelic/go-agent) package pour prendre en charge d&amp;apos;autres frameworks et bibliothèques. Si le framework et la base de données n&amp;apos;ont pas de d&amp;apos;intégration,package vous pouvez toujours les instrumenter à l&amp;apos;aide des primitives de package [newrelic](https://godoc.org/github.com/newrelic/go-agent). Pour plus d&amp;apos;informations sur l&amp;apos;instrumentation de votre base de données à l&amp;apos;aide de ces primitives, consultez la [documentation des segments datastore de l&apos;agent Go sur GitHub](https://github.com/newrelic/go-agent/blob/master/GUIDE.md#datastore-segments).

Le package d&apos;intégration suivant doit être importé avec le [newrelic](https://godoc.org/github.com/newrelic/go-agent/v3/newrelic) package, comme indiqué dans l&amp;apos;[exemple nrgin sur GitHub](https://github.com/newrelic/go-agent/blob/master/v3/integrations/nrgin/example/main.go).

## base de données et performances au niveau de instance

New Relic collecte [les détailsinstance pour une variété de bases de données et de pilotes de bases de données](/docs/apm/applications-menu/features/analyze-database-instance-level-performance-issues). La possibilité d&amp;apos;afficher une instance spécifique et les types d&amp;apos;informations de base de données dans <InlinePopover type="apm" />dépend de la version de votre agent .

L&apos;agent New Relic Go [version 1.4 ou supérieure](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-14) prend en charge les détails instance pour tous les pilotes de base de données.
