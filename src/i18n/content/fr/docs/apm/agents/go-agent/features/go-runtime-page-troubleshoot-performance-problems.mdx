L&apos;utilisateur de l&apos;interface [APM](/docs/apm) fournit une variété de données sur votre application, y compris une fonctionnalité spéciale pour l&apos;[agent Go pour APM](/docs/agents/go-agent/get-started/new-relic-go). De plus, la page <DNT>**Go runtime**</DNT> fournit des données d&apos;exécution importantes utiles pour le dépannage des problèmes de performances.

## Afficher et filtrer les données d&apos;exécution de Go [#find]

Pour afficher la page <DNT>**Go runtime**</DNT>, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; APM &amp;amp; services &amp;gt; (select an app) &amp;gt; Go runtime**</DNT>.
