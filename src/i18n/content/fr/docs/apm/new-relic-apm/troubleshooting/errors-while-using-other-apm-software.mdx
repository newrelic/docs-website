## Problème

Lorsqu&apos;un autre logiciel monitoring fonctionne parallèlement à l&apos;un de nos agents APM, nous ne pouvons pas garantir que nos agents fonctionneront correctement. Dans certains cas, l’exécution de plusieurs types d’ instrumentation, d’APM ou de logiciels monitoring peut entraîner des problèmes tels que des mesures peu fiables et des pannes d’agent.

Pour plus de détails, consultez la documentation sur les exigences et la compatibilité d&apos;un agent APM spécifique.

## Solution

Si vous pensez que l’exécution d’un autre logiciel monitoring pourrait être à l’origine d’un problème, voici quelques recommandations :

* Lisez la documentation sur les exigences et la compatibilité de l’agent APM que vous utilisez.
* Pour résoudre le problème, désactivez ou désinstallez temporairement d’autres logiciels APM pour voir si le problème persiste.
