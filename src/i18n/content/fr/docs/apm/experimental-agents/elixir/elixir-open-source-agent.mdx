Monitorez le comportement [d&apos;Elixir](https://github.com/newrelic/elixir_agent) avec New Relic en utilisant l&amp;apos;agent open source Elixir. Cet agent expérimental :

* Vous aide à suivre les transactions, les traces distribuées et d&apos;autres parties du comportement de votre application
* Fournit un aperçu de l&apos;activité [BEAM](http://erlang.org/faq/implementations.html#idp32950544) sous-jacente

<Callout variant="tip">
  Cet agent est sorti en open source sur [GitHub](https://github.com/newrelic/). Un log des modifications y est également disponible pour les dernières mises à jour.
</Callout>
