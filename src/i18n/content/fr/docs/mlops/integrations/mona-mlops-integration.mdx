Mona collecte et analyse les données de l&apos;ensemble de votre système d&apos;IA en production, vous permettant d&apos;évaluer la qualité du modèle avec des métriques personnalisées et prêtes à l&apos;emploi. Il est capable de détecter les dérives de données, les biais de modèle ou les problèmes d&apos;intégrité des données via des informations exploitables détaillées, même si ceux-ci ne se produisent que dans des segments spécifiques de vos données.

Mona inclut une [monitoring des performances des modèles](/docs/alerts-applied-intelligence/mlops/get-started/intro-mlops/) avec New Relic afin que vous puissiez ajouter l&apos;état de votre IA en tant que moniteur par Mona à votre vue opérationnelle centralisée sur New Relic.

## Migrer les données de Mona vers New Relic [#migrate-data]

Mona vous permet de partager automatiquement les données contextuelles traitées avec New Relic pour une consommation ultérieure dans <DNT>**[NRQL](/docs/query-your-data/nrql-new-relic-query-language/get-started/nrql-syntax-clauses-functions/)**</DNT>. Afin de recevoir des données de Mona dans New Relic, vous avez besoin de votre numéro de compte et d&apos;une clé spéciale, que vous pouvez obtenir comme suit :

1. **Connectez-vous à votre compte New Relic**: Depuis <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; +Add more data**</DNT>.
2. **Sélectionnez Mona**: Tapez <DNT>**Mona**</DNT> dans la barre de recherche.
3. **Copiez votre identifiant de compte**: sélectionnez l’identifiant de compte avec lequel vous souhaitez intégrer Mona. Cliquez sur <DNT>**continue**</DNT>. Copiez maintenant l’ID de compte à l’aide de l’icône <DNT>**copy**</DNT> .
4. **Sélectionnez New Relic dans le portail Mona **: log maintenant au portail Mona , accédez à la page de configuration et cliquez sur la classe contextuelle appropriée. Ensuite, allez dans l’onglet <DNT>**Integration**</DNT> et sélectionnez <DNT>**New Relic**</DNT>.
5. **Collez votre identifiant de compte**: Collez l’identifiant de compte que vous avez copié depuis New Relic dans le champ ID de compte dans Mona.
6. **Sélectionnez ou créez une clé API**: Revenez maintenant à New Relic et à partir de la page d&apos;intégration de New Relic, créez une <InlinePopover type="licenseKey" />traitée (ou sélectionnez une clé existante) sous <DNT>**Processed contextual data**</DNT>. Retournez voir Mona et entrez cette clé dans le champ <DNT>**Ingest license key**</DNT> .
7. **Enregistrez l&apos;intégration**: cliquez sur <DNT>**Save**</DNT> pour que vos données soient intégrées dans Mona et New Relic.

## Créer un dashboard avec les informations détaillées Mona active dans New Relic [#create-dashboards]

Mona crée automatiquement [un dashboard New Relic](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) contenant des informations actives détaillées. Pour les activer, suivez ces étapes :

1. **Ajouter une notification dans Mona**: Une fois votre intégration dans New Relic configurée, accédez à l&apos;onglet <DNT>**Notifications**</DNT> dans Mona au sein de votre projet contextuel. Cliquez sur <DNT>**Add a notification**</DNT> dans le coin supérieur droit et remplissez les champs correspondants.
2. **Obtenez l&apos;API de la clé utilisateur**: revenez à la page d&apos;intégration de New Relic et <DNT>**Select or create an API Key**</DNT> sous <DNT>**Active insights**</DNT>. Copiez cette clé.
3. **Collez la clé dans Mona**: Dans l’onglet de configuration des notifications Mona , collez la clé sous <DNT>**New Relic dashboard**</DNT>. Cliquez sur <DNT>**Save**</DNT>.

## Connectez Mona Insights aux informations sur incident de New Relic [#connect-insights]

<DNT>**[incident intelligence](/docs/alerts-applied-intelligence/applied-intelligence/incident-intelligence/get-started-incident-intelligence/#get-started)**</DNT> de New Relic vous aide à corréler votre incident et à réduire le bruit dans votre environnement. Il vous donne un aperçu de tous vos incidents, de leurs sources et des événements associés. En envoyant les informations détaillées Mona à New Relic, vous pourrez les connecter à d&apos;autres problèmes provenant d&apos;autres sources et obtenir plus de clarté et de compréhension de votre modèle.

1. **Ajouter une notification dans Mona**: Une fois votre intégration dans New Relic configurée, accédez à l&apos;onglet <DNT>**Notifications**</DNT> dans Mona au sein de votre projet contextuel. Cliquez sur <DNT>**Add a notification**</DNT> dans le coin supérieur droit et remplissez les champs correspondants.

2. **Cliquez sur la case à cocher**: Dans l’onglet de configuration des notifications Mona , cochez la case <DNT>**include New Relic incident intelligence**</DNT>.

3. **Enregistrez et recevez une notification**: cliquez sur Enregistrer en bas de la page de configuration des notifications. Désormais, la prochaine fois que Mona trouvera des informations détaillées, vous recevrez une notification dans Mona, une alerte dans New Relic, et vous pourrez consulter les informations détaillées directement dans un [dashboard New Relic](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) dédié.

<Callout variant="Sending Older Insights">
  Le service de renseignement incident de New Relic ne peut recevoir que les nouvelles informations détaillées apparues pour la première fois au cours des dernières 24 heures. Insights antérieures à cette date ne seront pas acceptées par New Relic.
</Callout>

## Affichez et explorez vos modèles Mona dans le dashboard New Relic [#view-explore]

Une fois que vous avez configuré l’intégration New Relic dans Mona, les métriques de modèle standard sont envoyées à New Relic. Consultez notre dashboard d&apos;observabilité du modèle et configurez n&apos;importe quelle métrique de modèle spécifique et politique incident pour une observabilité du modèle adaptée à votre contexte commercial.

1. **Mona Accédez dashboard** au d&apos;intégration : Une fois que vous avez configuré votre intégration et les informations détaillées, revenez à la page d&apos;intégration de New Relic et cliquez <DNT>**See your data**</DNT> sur. Cela vous redirigera vers un dashboard généré automatiquement et propulsé par Mona.

2. **Analyser de Mona dashboard**: Le initial dashboard est rempli avec votre flux de données par classe de contexte. De plus, vous pouvez consulter les informations détaillées Mona en cliquant sur le lien intitulé <DNT>**View your active insights on a New Relic dashboard**</DNT>. Il affiche les données d&apos;IA dynamiques générées automatiquement par Mona en fonction des données que vous avez initialement intégrées.

   <img title="Anayze Mona dashboards" alt="Anayze Mona dashboards." src="/images/model-performance-monitoring_screenshot-crop_mona-dashboard.webp" />

3. **Configurer une notification d&apos;alerte**: Une fois que vous avez créé des graphiques dans votre (vos) dashboard, vous pouvez être alerté sur vos données. Pour créer une [condition d’alerte](/docs/alerts-applied-intelligence/new-relic-alerts/alert-conditions/create-alert-conditions/) NRQL à partir d’un graphique, cliquez sur le menu du widget graphique <DNT>**...**</DNT> , puis cliquez sur <DNT>**Create alert condition**</DNT>. Une fois que vous avez nommé et personnalisé votre condition, vous pouvez l&apos;ajouter à une politique existante ou en créer une nouvelle.

4. **Être averti**: une fois que vous avez créé une condition <InlinePopover type="alerts" />, vous pouvez choisir comment vous souhaitez être averti. Consultez notre documentation sur [la façon de configurer le canal de notification](/docs/alerts-applied-intelligence/new-relic-alerts/alert-notifications/notification-channels-control-where-send-alerts/).

5. **Corrélez votre incident**: En plus de la notification, vous pouvez utiliser l&apos;intelligence incident pour corréler votre incident. Consultez notre documentation sur [la façon de corréler les incidents à l&apos;aide de décisions](/docs/alerts-applied-intelligence/applied-intelligence/incident-intelligence/change-applied-intelligence-correlation-logic-decisions/).

Vous avez maintenant intégré avec succès New Relic avec Mona. Les alertes Mona nouvellement créées sont désormais corrélées avec vos alertes New Relic, et vous pouvez voir les données sur les prédictions signalées les plus récentes.
