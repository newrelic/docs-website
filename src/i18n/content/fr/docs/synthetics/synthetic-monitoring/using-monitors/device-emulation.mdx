L'émulation de périphérique est un paramètre facultatif disponible pour simple et moniteur de navigateur scripté. Ce paramètre simule la fenêtre d'affichage et les capacités d'un appareil mobile ou d'une tablette à l'aide des capacités d'émulation de DevTools dans Chrome et Firefox.

L'émulation d'appareil vous permet de tester une application basée sur un navigateur à l'aide d'un appareil mobile ou d'une tablette générique simulé. Gardez à l'esprit :

* Ce paramètre ne modifie pas le réseau, l'agent utilisateur ou le type de périphérique utilisé pour exécuter votre moniteur Synthétique.
* L'émulation d'appareil ne vous permet pas d'installer et de tester des applications mobiles.

## Exigences [#requirements]

* Exécutions non héritées uniquement (pas de prise en charge Chrome 72)
* Chrome: le site privé doit utiliser la version 225 ou supérieure [du gestionnaire de tâches](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) et la version 1.1.98 de Synthetics-Node-Browser-Runtime ou supérieur
* Firefox : le site privé doit utiliser la version 400 ou supérieure [du gestionnaire de tâches](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) et la version 3.0.0 de Synthetics-Node-Browser-Runtime ou supérieur
