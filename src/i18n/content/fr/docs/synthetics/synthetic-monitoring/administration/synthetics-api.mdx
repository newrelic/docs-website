Nous avons deux options pour gérer votre moniteur New Relic Synthetics via API :

* NerdGraph (recommandé) : c&apos;est la méthode principale et recommandée. Consultez [notre tutoriel NerdGraph Synthetics](/docs/apis/nerdgraph/examples/nerdgraph-synthetics-tutorial).
* API REST : certains clients préfèrent utiliser notre ancienne API REST. Voir [l&apos;API REST de Synthetics](/docs/apis/synthetics-rest-api/monitor-examples/manage-synthetics-monitors-rest-api)

Pour un aperçu de toutes nos API disponibles, consultez [Introduction aux API](/docs/apis/getting-started/introduction-new-relic-apis).
