Les demandes de [moniteur synthétique](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors) incluent un champ d&apos;en-tête de demande `X-Abuse-Info`, afin que vous puissiez identifier le compte de monitoring synthétique New Relic et le moniteur qui l&apos;a envoyé. Utilisez ces informations pour :

* Accédez aux résultats du moniteur (si vous avez accès au compte).
* Désactivez le moniteur (si vous avez accès au compte).

## Identifier le compte ou le moniteur [#identify-monitor]

Pour identifier le compte Synthetics et l&apos;ID du moniteur qui a envoyé la demande, vérifiez le champ d&apos;en-tête de demande `X-Abuse-Info` . Par exemple:

```
X-Abuse-Info: Request sent by a synthetic monitor (/docs/synthetics/new-relic-synthetics/administration/identify-synthetics-requests-your-app) - monitor id: ${MONITOR_ID_STRING} | account id: ${ACCOUNT_NUMBER}
```

## Accéder ou désactiver le moniteur [#enable-disable]

Si vous avez accès au compte d&apos;où provient la demande du monitoring, vous pouvez afficher les résultats du monitoring ou désactiver le monitoring dans l&apos;interface utilisateur de New Relic :

1. [Identifiez](#identify-monitor) le numéro de compte et l’ID du moniteur qui a envoyé la demande.
2. Accédez à <DNT>**[one.newrelic.com &gt; Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)**</DNT> et sélectionnez un moniteur.
3. Cliquez sur <DNT>**General**</DNT> sous <DNT>**Settings**</DNT>.
4. Cliquez sur le bouton bascule <DNT>**Monitor**</DNT> pour le régler sur <DNT>**Disabled**</DNT>.
