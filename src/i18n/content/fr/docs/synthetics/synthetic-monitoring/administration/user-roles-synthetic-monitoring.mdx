Si vous êtes un [utilisateur complet de la plateforme](/docs/accounts/accounts-billing/new-relic-one-pricing-users/users-roles#user-type), vous avez théoriquement accès à toutes les fonctionnalités de New Relic, mais votre capacité à utiliser toutes ces fonctionnalités dépend du rôle qui vous est attribué au sein de votre organisation New Relic.

Pour plus de détails sur le fonctionnement des autorisations New Relic, consultez [Concepts importants de gestion des utilisateurs et des accès](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts). Pour voir les capacités granulaires liées à Synthetics, consultez [Capacités](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/#synthetics).
