## Problème

Vous disposez d&apos;un moniteur Synthétique qui contient des informations sensibles (comme des mots de passe ou des noms d&apos;utilisateur) dans une URL ou script.

## Solution

### Remplacer les URL dans le moniteur

Si vous utilisez le moniteur Synthétique qui vérifie les URL et que vous avez placé des informations d&apos;identification ou d&apos;autres informations sensibles dans les URL, vous souhaiterez peut-être modifier ces informations d&apos;identification périodiquement. Pour ce faire, modifiez votre script Synthétique pour modifier ou supprimer les informations et assurez-vous de faire pivoter les informations d&apos;identification qui ont été précédemment placées dans l&apos;URL. Pour des instructions détaillées sur l&apos;ajout et la modification du moniteur, voir [Ajouter et modifier le moniteur](/docs/synthetics/synthetic-monitoring/using-monitors/add-edit-monitors/).

### Faire pivoter les informations d&apos;identification en texte clair dans le script

Lors de la création [d&apos;un moniteur scripté](/docs/synthetics/synthetic-monitoring/scripting-monitors/introduction-scripted-browser-monitors/), vous pouvez écrire votre propre script pour les actions que le moniteur effectuera.

Si vous avez inclus des informations sensibles en texte clair dans le script, vous devez faire pivoter vos informations d&apos;identification :

1. Accédez à <DNT>**[one.newrelic.com &gt; Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)**</DNT>, puis sélectionnez le moniteur que vous souhaitez modifier.

2. Sélectionnez <DNT>**Settings &gt; Script**</DNT> pour modifier votre script de monitoring.

   <img title="Screenshot of the synthetic script editor page" alt="Screenshot of the synthetic script editor page" src="/images/synthetic_screenshot-crop_script-editor.webp" />

3. Faites pivoter ou modifiez les informations d’identification en texte clair. Dans la mesure du possible, remplacez le texte en clair par [des informations d&apos;identification sécurisées](/docs/synthetics/synthetic-monitoring/using-monitors/store-secure-credentials-scripted-browsers-api-tests/).
