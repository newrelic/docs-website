monitoring de l&apos;IA permet à l&apos;agent de reconnaître et de capturer les données de l&apos;IA. monitoring de l&apos;IA a des exigences de compatibilité de bibliothèque différentes selon le langage que vous avez utilisé pour votre application basée sur l&apos;IA.

Lorsque vous désactivez le traçage distribué ou activez le mode haute sécurité, l&apos;agent ne capturera pas les données AI.

## monitoring à grande échelle avec NVIDIA NIM [#deploy-at-scale]

monitoring de l&apos;IA peut s&apos;intégrer et collecter des données sur tous les modèles pris en charge par NVIDIA NIM. Par exemple, si vous avez créé une application d&apos;IA Python ou Node.js qui utilise Llama3, Mistralai ou l&apos;un des LLM propriétaires de NVIDIA, vous pouvez instrument ces applications avec monitoring de l&apos;IA et afficher les données de performances de vos applications.

Aucune étape supplémentaire n&apos;est nécessaire pour intégrer NVIDIA NIM : vous pouvez suivre nos [procédures manuelles d&apos;installation](/install/ai-monitoring) ou installer [directement via la plateforme New Relic](https://onenr.io/0VRVNLqavRa).
