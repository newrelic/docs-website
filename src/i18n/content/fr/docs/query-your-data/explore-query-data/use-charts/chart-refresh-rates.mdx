Le taux de rafraîchissement des graphiques New Relic correspond au nombre de fois par seconde qu&apos;un graphique est actualisé. Vous pouvez sélectionner une configuration par défaut et vous pouvez également configurer ce taux de rafraîchissement si votre compte est sur le plan tarifaire de consommation de [New Relic Compute](https://newrelic.com/pricing/compute) . Consultez [Comment fonctionnent les tarifs de New Relic](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/) pour plus d&apos;informations.

Cette configuration vous permet de décider à quelle fréquence vous souhaitez mettre à jour vos données. Cela vous donne également le contrôle de vos coûts. Consultez l’ [aperçu de nos modèles de tarification](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-pricing-models/#determine-pricing) pour en savoir plus.

## Taux de rafraîchissement des graphiques par défaut [#default-chart-refresh]

Le taux de rafraîchissement des graphiques New Relic correspond à la plage de temps de requête divisée par 60. La fréquence maximale à laquelle la requête peut être actualisée est toutes les 5 secondes.

Ces exemples vous aideront à comprendre comment fonctionne le taux de rafraîchissement par défaut du graphique :

* Une requête avec une période de 7 jours aurait un taux de rafraîchissement d&apos;une fois toutes les 2,8 heures.
* Une requête avec une plage horaire d&apos;une heure aurait un taux de rafraîchissement d&apos;une fois toutes les 1 minute.

Pour les requêtes utilisant `TIMESERIES`, la taille du bucket `TIMESERIES` est utilisée comme intervalle d&apos;actualisation. Par exemple:

* `SELECT count(*) FROM PageView TIMESERIES 5 minutes` serait rafraîchi toutes les 5 minutes.
* `SELECT count(*) FROM PageView TIMESERIES 1 hour SINCE 1 day ago` serait rafraîchi toutes les heures.

## Personnaliser les taux de rafraîchissement des graphiques [#customize-chart-refresh]

Vous pouvez personnaliser le taux de rafraîchissement de vos graphiques en utilisant l&apos;option <DNT>**Refresh rate**</DNT> . Pour configurer le taux de rafraîchissement dans un graphique, procédez comme suit :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Query your data**</DNT>.

2. Ajoutez votre requête.

3. Cliquez sur <DNT>**Run**</DNT>.

4. Cliquez sur <DNT>**Refresh rate**</DNT> et sélectionnez une fréquence.
