Avec [le générateur de requêtes](https://one.newrelic.com/data-exploration/query-builder), vous pouvez exécuter des requêtes sur vos données pour créer des graphiques personnalisés et d&apos;autres visualisations. Vous pouvez également créer [des tableaux de bord](/docs/dashboards/manage-your-dashboard/manage-your-dashboard#dash-create-chart) personnalisés contenant plusieurs graphiques.

## Pourquoi c&apos;est important

Les systèmes d’entreprise modernes sont un labyrinthe complexe d’éléments générateurs de données. Vous avez besoin d’un moyen simple et fiable pour récupérer, analyser et visualiser le flux incessant de données provenant de vos activités professionnelles quotidiennes.

Utilisez le générateur de requêtes pour :

* Accédez rapidement à vos données et créez des graphiques personnalisés pour connaître et comprendre l’état de votre infrastructure, de vos applications et d’autres services.
* Ajoutez des graphiques à votre tableau de bord pour obtenir une vue complète en temps réel de l&apos;état de votre système.
* Partagez vos cartes avec un collègue ou un utilisateur en seulement deux clics.
* Créez NRQL <InlinePopover type="alerts" />à partir de la requête que vous créez et exécutez.

## Démarrer [#get-started]

Si vous ne l&apos;avez pas déjà fait, créez votre compte New Relic gratuit ci-dessous pour commencer monitoring vos données dès aujourd&apos;hui.

Avant d&apos;utiliser le générateur de requêtes, nous vous recommandons de lire :

* [Introduction à l&apos;interrogation](/docs/telemetry-data-platform/ingest-manage-data/understand-data/introduction-querying-new-relic-data)
* [Comment explorer vos données](/docs/query-your-data/explore-query-data/explore-data/introduction-data-explorer)

Notez également que [notre API NerdGraph](/docs/apis/nerdgraph/examples/nerdgraph-nrql-tutorial) vous donne accès à une fonctionnalité de requête plus puissante. Par exemple, vous pouvez exécuter une requête sur plusieurs comptes, exécuter une requête asynchrone, etc.

Pour trouver le générateur de requêtes, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Query builder**</DNT>.

## Utiliser et partager des graphiques [#customize-chart]

Une fois que vous avez créé un graphique, vous pouvez :

* [Changer le type de graphique](/docs/chart-builder/use-chart-builder/use-charts/use-your-charts#change-appearance). En fonction des données que vous avez spécifiées, le générateur de requêtes sélectionne le type de graphique qui affiche vos résultats le plus efficacement. Cependant, vous pouvez choisir parmi d’autres [types de graphiques disponibles](/docs/insights/use-insights-ui/manage-dashboards/insights-chart-types) pour présenter vos données dans le format visuel que vous souhaitez utiliser.
* [Partagez votre graphique](/docs/chart-builder/use-chart-builder/use-charts/use-your-charts#share-charts).
* [Ajoutez votre graphique à un dashboard](/docs/chart-builder/use-chart-builder/use-charts/use-your-charts#add-to-dashboard).
* [Créez une condition d’alerte NRQL à partir de votre graphique](/docs/alerts-applied-intelligence/new-relic-alerts/alert-conditions/create-nrql-alert-conditions/#create-chart).
