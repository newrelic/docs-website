Vous pouvez utiliser [le dashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) pour visualiser et suivre l&apos;état de votre système, résoudre les problèmes et planifier les étapes ultérieures. Apprenez à partager toutes ces informations avec votre équipe, vos clients ou d’autres parties prenantes grâce à diverses options d’importation et d’exportation.

## Importer un dashboard au format JSON [#import-json]

Vous pouvez importer un dashboard au format JSON lorsque vous souhaitez modifier un dashboard prêt à l&apos;emploi au niveau du code. Par exemple, si vous souhaitez créer un dashboard personnalisé en fusionnant deux dashboards existants, vous pouvez modifier le JSON d&apos;un dashboard de base, puis le réimporter dans New Relic.

Pour importer un dashboard au format JSON :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Dashboards**</DNT>.

2. Ouvrez le JSON d’un dashboard existant en cliquant sur l’icône <DNT>**...**</DNT>, puis sélectionnez <DNT>**Copy JSON**</DNT>.

3. Retournez à la page <DNT>**Dashboards**</DNT> .

4. Dans le coin supérieur droit, cliquez sur <DNT>**Import dashboard**</DNT>.

5. Collez votre code JSON.

6. Choisissez les paramètres de compte et d&apos;autorisation pour le dashboard. Vous ne pouvez pas modifier le compte une fois que vous l&apos;avez configuré, mais vous pouvez [modifier les autorisations](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-settings) à tout moment.

7. Cliquez sur <DNT>**Save**</DNT>.

## Exporter un dashboard au format PDF [#export-dashboards-pdf]

Pour exporter votre dashboard sous forme de fichier PDF, dans le coin supérieur droit du dashboard, cliquez sur l&apos;icône <DNT>**...**</DNT> et sélectionnez <DNT>**Export dashboard as PDF**</DNT>.

<figcaption>
  Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Dashboards**</DNT> et ouvrez un dashboard existant.
</figcaption>

Tenez compte des éléments suivants lors de l’exportation d’un dashboard au format PDF :

* [Les variables de modèle](/docs/query-your-data/explore-query-data/dashboards/dashboard-template-variables/#requirements) ne sont pas prises en charge.

* [Les visualisations personnalisées](/docs/query-your-data/explore-query-data/dashboards/add-custom-visualizations-your-dashboards/) ne sont pas prises en charge.

* [Les graphiques d&apos;affichage](/docs/query-your-data/explore-query-data/use-charts/chart-types/#widget-billboard) qui utilisent la clause [`COMPARE WITH`](/docs/nrql/nrql-syntax-clauses-functions/#sel-compare) n&apos;afficheront pas les pourcentages ou ne s&apos;afficheront pas correctement avec les liens de graphiques publics, les images exportées et les fichiers PDF.

* Le widget Markdown avec de grandes images peut ne pas s&apos;afficher correctement ou ne pas être entièrement visible.

* Les dernières fonctionnalités de visualisation telles que (mais sans s&apos;y limiter) le tri des données dans les tableaux, le format des données des graphiques, les émojis et les symboles ne sont pas pris en charge et ne seront pas reflétés dans le PDF résultant.

## Exporter un graphique au format CSV [#export-charts-csv]

Pour exporter un graphique de tableau de dashboard sous forme de fichier CSV, dans le coin supérieur droit du graphique, cliquez sur <DNT>**...**</DNT> et sélectionnez <DNT>**Export as CSV**</DNT>.

<img title="Export chart as CSV" alt="Screenshot of how to export a table chart as a CSV file" src="/images/dashboards_screenshot-crop_export-csv.webp" />

Tenez compte des éléments suivants lors de l’exportation d’un graphique au format CSV :

* Les données sont toujours exportées en heure UTC
* L&apos;exportation ne prend pas en charge les fonctions d&apos;agrégation telles que `average()`
* L&apos;exportation ne prend pas en charge la clause `COMPARE_WITH`
* Dans certains scénarios, l&apos;UI peut effectuer un formatage supplémentaire des données qui ne sont pas répliquées dans le fichier CSV exporté, ce qui peut entraîner de petites divergences (comme le rendu de -1.1E-17 sous la forme 0 dans l&apos;UI mais pas dans le fichier CSV)
* Vous ne pouvez pas exporter les résultats d&apos;une requête inter-comptes.
