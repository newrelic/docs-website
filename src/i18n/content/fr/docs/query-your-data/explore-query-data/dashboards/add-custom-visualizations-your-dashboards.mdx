Vous pouvez créer vos propres visualisations et les ajouter à un dashboard. Cela vous offre une grande flexibilité quant à ce que vous affichez sur le dashboard, d&apos;un logo d&apos;entreprise à une requête personnalisée à partir de n&apos;importe quelle source de données.

Le processus de création d&apos;une visualisation est décrit dans le guide [Créer une visualisation personnalisée pour le dashboard](/docs/new-relic-solutions/build-nr-ui/custom-visualizations/build-visualization).

## Ajouter une visualisation à un dashboard

Vous pouvez ajouter une visualisation à un dashboard nouveau ou existant.

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Apps &gt; Custom visualizations**</DNT>. (Si vous ne voyez pas la tuile <DNT>**Custom visualizations**</DNT> , utilisez la recherche pour la localiser.)
2. Dans <DNT>**Custom visualizations**</DNT>, sélectionnez la visualisation que vous souhaitez ajouter à un dashboard , puis activez-la.
3. S&apos;il existe des options de configuration, remplissez-les. La visualisation sera mise à jour avec vos modifications.
4. Cliquez sur <DNT>**Add to dashboard**</DNT>, puis sélectionnez un dashboard dans la liste des tableaux de bord disponibles ou sélectionnez <DNT>**New dashboard**</DNT>. Si vous décidez de créer un nouveau dashboard, sélectionnez le compte sur lequel vous souhaitez exécuter le dashboard et donnez un nom au dashboard .
5. Cliquez sur <DNT>**Add to dashboard**</DNT>, puis cliquez sur le lien vers votre dashboard pour voir la visualisation personnalisée.

## Gérez vos visualisations dashboard

<DNT>**Deleting**</DNT>: Pour supprimer une visualisation d’un dashboard, cliquez sur le bouton à points de suspension dans le coin droit de la visualisation et cliquez sur <DNT>**Delete**</DNT>.

<DNT>**Editing**</DNT>: Pour modifier une visualisation à partir d’un dashboard, cliquez sur le bouton à points de suspension dans le coin droit de la visualisation et cliquez sur <DNT>**Edit**</DNT>.
