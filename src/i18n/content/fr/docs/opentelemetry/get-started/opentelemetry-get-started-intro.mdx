Le projet OpenTelemetry a une portée large comprenant [les spécifications API et SDK](https://opentelemetry.io/docs/specs/otel/), [l&apos;instrumentation](https://opentelemetry.io/docs/concepts/instrumentation/), le [Collector](https://opentelemetry.io/docs/collector/), [les conventions sémantiques](https://opentelemetry.io/docs/specs/semconv/), le protocole [OTLP](https://opentelemetry.io/docs/specs/otlp/) , etc. New Relic Monitoring de l&apos;IA bénéficie d&apos;un support de premier ordre pour OpenTelemetry et dispose d&apos;une grande variété de fonctionnalités de plate-forme qui facilitent l&apos;utilisation des outils OpenTelemetry .

Les pages ci-dessous se concentrent sur les modèles d&apos;intégration courants, fournissant une base pour utiliser efficacement OpenTelemetry avec New Relic. Bien que non exhaustif, il met en évidence la manière dont les utilisateurs exploitent généralement ces outils et les capacités de la plateforme New Relic qui s&apos;appuient sur eux.
