New Relic Distribution of OpenTelemetry (NRODT) fait référence à la collection de distributions prises en charge par New Relic des composants du projet OpenTelemetry. Les composants NRDOT sont conçus pour fonctionner de manière transparente avec les produits et services New Relic et, lorsqu&apos;ils sont disponibles, constituent le moyen recommandé d&apos;intégrer OpenTelemetry à New Relic.

<Callout variant="tip">
  Actuellement, NRDOT inclut des distributions pré-construites d&apos;OpenTelemetry [Collector](/docs/opentelemetry/nrdot/nrdot-collector/). Pour les API de langage et les SDK, consultez notre documentation sur la prise en main de [la surveillance APM](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-intro/).
</Callout>
