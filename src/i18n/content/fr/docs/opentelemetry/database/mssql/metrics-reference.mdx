Monitorez les performances de votre base de données SQL Server avec des métriques complètes collectées par le collecteur NRDOT. Les 7 métriques suivantes sont collectées automatiquement pour les fonctionnalités de l’UI de New Relic. Celles-ci sont **activées par défaut** et fonctionnent sur n’importe quelle plateforme via une connexion SQL Server directe.
