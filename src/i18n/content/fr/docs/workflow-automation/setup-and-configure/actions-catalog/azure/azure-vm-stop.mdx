Les actions de la machine virtuelle Azure vous permettent de gérer le cycle de vie de votre machine virtuelle Azure directement depuis vos workflows.

## Arrêter une machine virtuelle [#azure-vm-stop]

Cette action arrête une machine virtuelle Azure à l&apos;aide de l&apos;API REST [Azure calculer machine virtuelle - Power Off](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines/power-off?view=rest-compute-2025-11-01&tabs=HTTP).

    <TabsPageItem id="azurevmstopexample">
      ```yaml
      name: azure-stop-vm
      description: 'This stops a virtual machine'
      steps:
        - name: azure_vm_stop_3
          type: action
          action: azure.vm.stop
          version: 1
          inputs:
            clientId: ${{ :secrets:azure-client-id }}
            clientSecret: ${{ :secrets:azure-client-secret }}
            tenantId: ${{ :secrets:azure-client-tenant }}
            subscriptionId: "12345678-1234-1234-1234-123456789abc"
            resourceGroupName: "my-resource-group"
            virtualMachineName: ‘my-demo-vm'
          next: end
      ```
