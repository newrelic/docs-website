Cette page fournit une référence complète pour les utilitaires - uuid disponibles dans le catalogue des actions d&apos;automatisation du workflow. Ces actions vous permettent d&apos;utiliser les utilitaires de génération uuid.

## Générer un UUID [#utils-uuid-generate]

Générer un uuid V4 conforme à la RFC. Utilise le générateur de nombres pseudo-aléatoires (PRNG) déterministe de Temporal workflow, ce qui le rend sûr pour une utilisation dans un workflow. Reportez-vous à [Namespace: workflow | Temporal TypeScript SDK API Reference](https://typescript.temporal.io/api/namespaces/workflow?&_ga=2.223881945.39829224.1686648936-167080156.1680123165#uuid4) pour plus d&apos;informations.
