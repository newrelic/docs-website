<Callout title="Aperçu">
  Nous travaillons toujours sur cette fonctionnalité, mais nous aimerions que vous l&apos;essayiez !

  Cette fonctionnalité est actuellement fournie dans le cadre d&apos;un programme d&apos;aperçu conformément à nos [politiques de pré-sortie](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Cette page fournit une référence complète des actions d&apos;incident PagerDuty disponibles dans le catalogue des actions d&apos;automatisation des workflows. Ces actions vous permettent de gérer les incidents PagerDuty.

## Prérequis

Avant d&apos;utiliser les actions PagerDuty dans l&apos;automatisation workflow, assurez-vous de disposer de :

* Un compte PagerDuty disposant des autorisations appropriées.
* Un jeton d&apos;API PagerDuty (jeton utilisateur ou jeton de compte).
* Les autorisations nécessaires pour gérer les incidents dans votre compte PagerDuty.
* Pour le compte jeton, vous devrez fournir l&apos;adresse e-mail `from` dans les actions appropriées.
