Cette page fournit une référence complète des utilitaires - transform disponibles dans le catalogue d&apos;actions d&apos;automatisation des workflows. Ces actions vous permettent d&apos;utiliser des utilitaires de transformation de données.

## Convertir une valeur de données d&apos;entrée au format CSV [#utils-transform-toCSV]

Ceci est utilisé pour transformer différents types d&apos;entrée (JSON, map) en format CSV.

              ```yaml
                name: nrqltocsv

                steps:
                  - name: queryForLog
                    type: action
                    action: newrelic.nrql.query
                    version: 1
                    inputs:
                      accountIds: ['${{ .workflowInputs.accountId }}']
                      query: ${{ .workflowInputs.nrql }}

                  - name: csv1
                    type: action
                    action: utils.transform.toCSV
                    version: 1
                    inputs:
                      data: ${{ .steps.queryForLog.outputs.results | tostring }}
                      selectors:
                        - name: csv
                          expression: ".csv"
              ```
