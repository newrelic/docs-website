L&apos;action Azure API est un exécuteur universel qui vous permet d&apos;appeler n&apos;importe quelle [API REST Azure](https://learn.microsoft.com/en-us/rest/api/azure/) directement depuis vos workflows. Cette action permet une intégration flexible avec les services Azure sans nécessiter d&apos;actions dédiées pour chaque opération Azure.

## Exécuter une API Azure [#azure-execute-api]

Cette action vous permet d&apos;exécuter toute opération documentée dans la [référence de l&apos;API REST Azure](https://learn.microsoft.com/en-us/rest/api/azure/). Indiquez simplement le chemin de l&apos;API, la méthode HTTP et le corps de la requête : l&apos;action gère automatiquement l&apos;authentification et les opérations de longue durée.

      ### Exemple : GET usages réseau

      ```yaml
        - name: getNetworkUsage
          type: action
          action: azure.execute.api
          version: 1
          inputs:
            clientId: ${{ :secrets:my-client-id }}
            clientSecret: ${{ :secrets:my-client-secret }}
            tenantId: ${{ :secrets:my-tenant-id }}
            urlPath: "subscriptions/c6f367b2-d6af-41f5-a9e3-3828fbd97241/providers/Microsoft.Network/locations/westus/usages?api-version=2025-03-01"
            method: "GET"
            selectors:
              - name: firstResponseValue
                expression: ".response.value[0] | tostring"
      ```
