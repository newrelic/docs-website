## Demande d&apos;échantillon

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        actionDefinitions {
          results {
            description
            inputs {
              name
              required
              secretAllowed
            }
            name
            outputs {
              name
            }
            timeout {
              action
            }
            version
          }
        }
      }
    }
  }
}
```

## Exemple de réponse

La requête retourne :

* `actionDefinitions.results`: Un tableau de définitions d&apos;actions, chacune contenant :

  * `name`: Le nom de l&apos;action (par exemple, `aws.ec2.rebootInstances`)
  * `description`: Ce que fait l&apos;action
  * `version`: La version de l&apos;action
  * `inputs`: Tableau des paramètres d&apos;entrée avec les propriétés `name`, `required` et `secretAllowed`
  * `outputs`: Tableau de champs de sortie avec la propriété `name`
  * `timeout.action`: Délai d&apos;attente en secondes
