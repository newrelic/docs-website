## Demande d&apos;échantillon

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```

## Exemple de réponse

La mutation retourne :

* `definition.name`: Le nom de la définition de workflow mise à jour
* `definition.version`: Le nouveau numéro de version de la définition du workflow
* `definition.yaml`: La définition de workflow YAML encodée en base64
