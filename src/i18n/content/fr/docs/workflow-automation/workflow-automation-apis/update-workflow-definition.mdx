Utilisé pour mettre à jour une définition workflow existante. Elle renvoie une erreur si la définition workflow n&apos;existe pas déjà.

## Exemple

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```
