Cette API récupère et renvoie une liste de tous les plans de définition de workflow sous la portée donnée. Remarque : le fichier YAML de définition de workflow interne n&apos;est pas disponible avec cette requête. Un fichier YAML de définition de workflow n&apos;est renvoyé qu&apos;à l&apos;aide d&apos;une requête `GetWorkflowDefinition`. `/{actor/account(:id)/workflowAutomation/workflow(:name, :version)`

## Exemple

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflows(cursor: "xyz123") {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
          }
        }
      }
    }
  }
}
```
