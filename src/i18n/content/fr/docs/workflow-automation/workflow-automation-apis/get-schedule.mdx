## Demande d&apos;échantillon

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## Exemple de réponse

La requête retourne :

* `schedule.id`: L&apos;identifiant unique de la planification.
