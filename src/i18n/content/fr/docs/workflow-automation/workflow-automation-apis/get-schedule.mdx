## Exemple [#example]

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## Réponse

La requête retourne :

* `schedule.id`: L&apos;identifiant unique de la planification.
