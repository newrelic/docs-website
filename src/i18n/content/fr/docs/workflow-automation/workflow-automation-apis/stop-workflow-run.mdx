## Exemple

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```

## Réponse

La mutation retourne :

* `runId`: L&apos;identifiant unique de l&apos;exécution du workflow arrêté. Renvoie un `runId` vide si le workflow est introuvable.
