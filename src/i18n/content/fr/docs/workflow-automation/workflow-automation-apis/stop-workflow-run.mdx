Cela interrompt un workflow en cours. Elle renvoie une valeur vide `runId` si le workflow est introuvable.

## Exemple

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```
