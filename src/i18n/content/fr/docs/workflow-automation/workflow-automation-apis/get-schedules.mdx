## Demande d&apos;échantillon

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        schedules {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
            schedule {
              id
            }
          }
        }
      }
    }
  }
}
```

## Exemple de réponse

La requête retourne :

* `schedules.results`: Un tableau d&apos;objets de planification, chacun contenant :

  * `definition.name`: Le nom de la définition du workflow
  * `definition.scope.id`: L&apos;ID du compte
  * `definition.scope.type`: Le type de portée
  * `definition.version`: La version de la définition du workflow
  * `schedule.id`: L&apos;identifiant unique de la planification
