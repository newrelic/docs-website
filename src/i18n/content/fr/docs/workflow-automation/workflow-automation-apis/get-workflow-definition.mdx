## Demande d&apos;échantillon

Utilisez le [générateur de requêtes](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) pour créer et éditer un workflow. Voir [l&apos;explorateur d&apos;API NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflow(name: "my-workflow", version: 1) {
          definition {
            name
            version
            yaml
          }
        }
      }
    }
  }
}
```

## Exemple de réponse

La requête retourne :

* `definition.name`: Le nom de la définition du workflow
* `definition.version`: Le numéro de version de la définition du workflow
* `definition.yaml`: La définition de workflow YAML encodée en base64
