Utilisez `${{ .workflowInputs.variableName }}` pour passer des valeurs dynamiques au moment de l&apos;exécution dans les URL des points de terminaison et d&apos;autres champs de saisie.

**Ce que fait cette approche :**

* Accepter les valeurs d&apos;entrée dynamiques lorsque le workflow est déclenché
* Référencer les valeurs d&apos;entrée en utilisant la syntaxe des modèles tout au long des étapes du workflow
* Activez la réutilisabilité du workflow avec différents paramètres

## Exemple

```json
  {
    "inputs": [
      {
        "key": "urlParams",
        "value": "{\"filter\": \"active\"}"
      },
      {
        "key": "headers",
        "value": "{\"Api-Key\": \"your-api-key\"}"
      }
    ]
  }
```
