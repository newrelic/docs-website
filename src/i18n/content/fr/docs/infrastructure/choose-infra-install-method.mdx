L&apos;agent d&apos;infrastructure est un fichier exécutable qui collecte des données sur vos hôtes. Vous pouvez permettre à l&apos;agent d&apos;infrastructure de signaler des données provenant de [certains services tiers](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations) et de collecter également [des données de log](/docs/logs/enable-log-monitoring-new-relic/enable-log-monitoring-new-relic/forward-your-logs-using-infrastructure-agent) .

L&apos;agent d&apos;infrastructure peut s&apos;exécuter sur Windows, macOS et de nombreuses distributions Linux. Ce document décrit les différentes méthodes que vous pouvez utiliser pour installer l&apos;agent d&apos;infrastructure. Pour commencer, vous aurez besoin d&apos;un [compte New Relic](https://newrelic.com/signup) et d&apos;une clé de licence.

## Installer l&apos;agent d&apos;infrastructure manuellement [#manual-install]

Nous vous recommandons de suivre nos procédures d&apos;installation manuelle si vous souhaitez monitorer quelques hôtes, mais rien de plus complexe. En général, vous utiliserez la ligne de commande pour créer un répertoire dans lequel réside l&apos;agent, puis ajouterez les informations de votre clé de licence. Selon votre système d’exploitation, vous devrez peut-être également créer et modifier un fichier de configuration. Pour installer manuellement, commencez ici :

* [Installer sur système Linux avec un gestionnaire de paquets](/docs/infrastructure/install-infrastructure-agent/linux-installation/install-infrastructure-monitoring-agent-linux)
* [Installer sur macOS](/docs/infrastructure/install-infrastructure-agent/macos-installation/install-infrastructure-monitoring-agent-macos)
* Installer sur Windows avec [l&apos;installation MSI](/docs/infrastructure/install-infrastructure-agent/windows-installation/install-infrastructure-monitoring-agent-windows), [l&apos;installation zip](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-assisted-install-infrastructure-agent-windows) ou [l&apos;installation zip manuelle](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-manual-install-infrastructure-agent-windows)

## Utilisez l&apos;installation guidée [#guided-install]

L&apos;installation guidée est une seule commande CLI que vous pouvez exécuter pour monitorer votre instance. C&apos;est une bonne option pour les petites organisations ou pour quiconque souhaite tester New Relic. Lorsque vous exécutez la commande CLI, New Relic inspectera votre système pour trouver d&apos;autres technologies de compatibilité que vous souhaiterez peut-être monitorer. Vous recevrez également l&apos;[API d&apos;état de l&apos;infrastructure](https://github.com/newrelic/infrastructure-agent/blob/master/docs/status_api.md) par défaut.

<Callout variant="important">
  La CLI New Relic ne prend pas en charge les serveurs FedRAMP. Si vous êtes client FedRAMP, veuillez vous référer aux [instructions d&apos;installation manuelle](#manual-install).
</Callout>

## Quelle est la prochaine étape ? [#whats-next]

Après avoir installé l’agent d’infrastructure :

* Découvrez comment [configurer l&apos;agent](/docs/infrastructure/install-configure-infrastructure/configuration/configure-infrastructure-agent) ou modifier le [modèle de configuration](https://github.com/newrelic/infrastructure-agent/blob/master/assets/examples/infrastructure/newrelic-infra-template.yml.example).
* Installez [l&apos;intégration sur hôte](/docs/integrations/host-integrations/getting-started/introduction-host-integrations) (par exemple, Apache ou MySQL).
* [Activer le transfert de logà l&apos;aide de l&apos;agent infrastructure ](/docs/logs/enable-new-relic-logs/1-enable-logs/forward-your-logs-using-new-relic-infrastructure).
* Apprenez à [gérer l&apos;agent](/docs/infrastructure/install-infrastructure-agent/manage-your-agent).
* Notre agent d&apos;infrastructure est un fichier exécutable léger. Apprenez-en plus sur ce que cela signifie [ici](/docs/infrastructure/new-relic-infrastructure/getting-started/infrastructure-agent-performance-overhead)
