# étiquettes métriques [#metric-labels]

<Callout variant="important">
  Pour se conformer au [Prometheus DataModel](https://prometheus.io/docs/concepts/data_model/#metric-names-and-labels), les noms d&amp;apos;étiquettes Kubernetes sont nettoyés après avoir été récupérés pour remplacer tout caractère non pris en charge par `_`. Par exemple, l’étiquette Kubernetes `k8s.io/app` est ajoutée en tant que `k8s_io_app`. Les valeurs des étiquettes ne sont pas modifiées.
</Callout>

# Métriques générées [#generated-metrics]

<Callout variant="tip">
  Ces métriques contiennent tous les [labels métriques](#metric-labels).
</Callout>

Les métriques suivantes sont générées pour chaque cible récupérée :

<table>
  <thead>
    <tr>
      <th style={{ width: "400px" }}>
        métrique
      </th>

      <th>
        Description
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `up`
      </td>

      <td>
        La valeur sera `0` si le scraping a échoué, sinon elle sera `1`.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_duration_seconds`
      </td>

      <td>
        Durée du grattage.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_samples_scraped`
      </td>

      <td>
        Le nombre d&apos;échantillons que la cible a exposés.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_samples_post_metric_relabeling`
      </td>

      <td>
        Le nombre d&apos;échantillons restant après l&apos;application du réétiquetage métrique.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_series_added`
      </td>

      <td>
        Le nombre approximatif de nouvelles séries dans ce scrape.
      </td>
    </tr>
  </tbody>
</table>
