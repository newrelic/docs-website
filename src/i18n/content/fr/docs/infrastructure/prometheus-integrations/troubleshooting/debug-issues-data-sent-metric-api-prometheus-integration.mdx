## Problème

Vous rencontrez des difficultés pour identifier la cause des problèmes signalés avec les données envoyées à [l&apos;API métrique](/docs/data-ingest-apis/get-data-new-relic/metric-api/introduction-metric-api) de New Relic. Vous souhaitez voir les données exactes envoyées depuis votre intégration Prometheus OpenMetrics pour Docker ou Kubernetes vers l&apos;API métrique.

## Solution

Vous pouvez configurer l&apos;intégration Prometheus OpenMetrics pour afficher les données brutes envoyées au flux de sortie standard. Pour voir les données exactes envoyées à l&apos;API métrique, définissez la variable d&apos;environnement `EMITTERS` sur `"api,stdout"`.
