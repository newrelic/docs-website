L&apos;intégration de l&apos;infrastructure New Relic inclut désormais une intégration qui envoie les métriques pare-feu de votre réseau AWS à New Relic. Ce document explique la fonctionnalité de l&apos;intégration, comment l&apos;activer et quelles données sont disponibles pour le reporting.

## Caractéristiques [#features]

L&apos;intégration AWS Network pare-feu Métriques collecte et envoie des données télémétriques à New Relic depuis votre réseau AWS pare-feu. Vous pouvez monitorer vos services AWS, interroger les données entrantes et créer un dashboard pour tout observer en un coup d&apos;œil.

## Activer l&apos;intégration [#activate-ingration]

Pour activer cette intégration, suivez les instructions de notre [documentation d&apos;intégration Amazon flux de métriques CloudWatch ](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Rechercher et utiliser des données [#find-data]

Pour trouver les métriques de votre intégration :

1. Aller à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities)**</DNT>
2. Select <DNT>**Metrics and events**</DNT>
3. Filtrer par `aws.networkfirewall`.
