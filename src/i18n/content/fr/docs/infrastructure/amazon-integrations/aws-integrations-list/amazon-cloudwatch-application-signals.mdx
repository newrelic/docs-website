New Relic s&apos;intègre à Amazon Web Services (AWS)) pour monitorer [les signaux métriques de l&apos;applicationCloudWatch ](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Agent-Application_Signals.html)et d&amp;apos;autres données vers New Relic.

Ce document explique comment activer l&apos;intégration et décrit les données signalées.

## Activer l&apos;intégration [#activate]

Cette intégration est disponible via CloudWatch MetricStreams.

Pour activer cette intégration, découvrez comment [connecter les services AWS à New Relic via les flux de métriques CloudWatch](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Rechercher et utiliser des données [#find-use-data]

Pour trouver les métriques de votre intégration, accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT> et filtrez par `aws.applicationsignals`.
