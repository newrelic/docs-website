New Relic s&apos;intègre à Amazon Web Services (AWS) pour signaler vos métriques [Amazon CodeBuild](https://aws.amazon.com/codebuild/) et d&apos;autres données à New Relic.

Ce document explique comment activer l&apos;intégration et décrit les données signalées.

## Caractéristiques [#features]

Collectez et envoyez des données télémétriques à New Relic depuis vos services AWS CodeBuild en utilisant notre intégration. Monitorez vos services, interrogez les données entrantes et créez un dashboard pour tout observer en un coup d&apos;œil.

## Activer l&apos;intégration [#activate]

Cette intégration est disponible via CloudWatch MetricStreams.

Pour activer cette intégration, découvrez comment [connecter les services AWS à New Relic via les flux de métriques CloudWatch](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Rechercher et utiliser des données [#find-data]

Pour trouver les métriques de votre intégration, accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> et filtrez par `aws.codebuild`.

De plus, vous pouvez trouver l&apos;entité associée à Amazon CodeBuild en allant sur [one.newrelic.com](https://one.newrelic.com/all-entities) et en recherchant `codebuild`.
