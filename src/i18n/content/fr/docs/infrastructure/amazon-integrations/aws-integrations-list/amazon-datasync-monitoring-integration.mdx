New Relic s&apos;intègre à Amazon Web Services (AWS) pour signaler vos métriques [AWS DataSync](https://aws.amazon.com/datasync/) et d&apos;autres données à New Relic.

Ce document explique comment activer l&apos;intégration et décrit les données signalées.

## Caractéristiques [#features]

Collectez et envoyez des données télémétriques à New Relic depuis vos services AWS DataSync en utilisant notre intégration. Monitorez vos services, interrogez les données entrantes et créez un dashboard pour tout observer en un coup d&apos;œil.

## Activer l&apos;intégration [#activate]

Cette intégration est disponible via flux de métriques CloudWatch.

Pour activer cette intégration, découvrez comment [connecter les services AWS à New Relic via le flux de métriques CloudWatch](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)

## Rechercher et utiliser des données [#find-data]

Pour trouver les métriques de votre intégration, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> et filtrez par `aws.DataSync`.

De plus, vous pouvez trouver l&apos;entité associée à AWS DataSync en accédant à [one.newrelic.com](https://one.newrelic.com/all-entities) et en recherchant `datasync`.
