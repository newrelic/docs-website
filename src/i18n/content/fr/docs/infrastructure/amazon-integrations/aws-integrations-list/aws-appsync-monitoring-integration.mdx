New Relic propose une intégration pour générer des rapports sur vos données [AWS AppSync](https://aws.amazon.com/appsync/) . Ce document explique comment activer cette intégration et décrit les données qui peuvent être signalées.

## Activer l&apos;intégration [#activate]

Pour activer cette intégration, suivez les procédures standard pour [connecter les services AWS à New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informations [d&apos;interrogation](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) par défaut pour l&amp;apos;intégration AWS AppSync :

* Intervalle d&apos;interrogation de New Relic : 5 minutes
* Intervalle de données Amazon CloudWatch : 1 minute

## Rechercher et utiliser des données [#find-data]

## données métriques [#metrics]

Cette intégration collecte les données AWS AppSync pour Api.

      </td>

      <td>
        Compter
      </td>

      <td>
        Le nombre d&apos;erreurs capturées à la suite de requests non valides en raison d&apos; configuration client incorrecte. En règle générale, ces erreurs se produisent n’importe où en dehors de l’exécution de GraphQL. Par exemple, il peut s&apos;agir d&apos;une charge utile JSON incorrecte ou d&apos;une requête incorrecte dans la demande, lorsque le service est limité, ou même d&apos;une mauvaise configuration potentielle des paramètres d&apos;authentification.
      </td>
    </tr>

    <tr>
      <td>

      </td>

Par exemple, cela peut se produire lorsqu&apos;une demande de requête est lancée pour un schéma vide ou incorrect, si l&apos;ID du pool d&apos;utilisateurs Amazon Cognito ou la région AWS n&apos;est pas valide. Alternativement, cela peut également se produire si AWS AppSync rencontre un problème lors de l&apos;exécution d&apos;une requête.
      </td>
