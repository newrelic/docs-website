New Relic s&apos;intègre à Amazon Web Services (AWS) pour signaler vos métriques [Amazon WorkMail](https://aws.amazon.com/workmail/) et d&amp;apos;autres données à New Relic.

Ce document explique comment activer l&apos;intégration et décrit les données signalées.

## Caractéristiques [#features]

Collectez et envoyez des données télémétriques à New Relic depuis votre WorkMail en utilisant notre intégration. Monitorez vos services, interrogez les données entrantes et créez un dashboard pour tout observer en un coup d&apos;œil.

## Activer l&apos;intégration [#activate]

Cette intégration est disponible via CloudWatch MetricStreams (recommandé).

Pour activer cette intégration, découvrez comment [connecter les services AWS à New Relic via les flux de métriques CloudWatch](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Rechercher et utiliser des données [#find-data]

Pour trouver les métriques de votre intégration, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT> et filtrez par `aws.workmail`.
