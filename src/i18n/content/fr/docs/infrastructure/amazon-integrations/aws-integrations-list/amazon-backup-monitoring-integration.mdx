New Relic s&apos;intègre à Amazon Web Services (AWS)) pour monitorer les métriques [AWS Backup](https://aws.amazon.com/backup/) et d&apos;autres données sur New Relic. Ce document explique comment activer l&apos;intégration et décrit les données signalées.

## Caractéristiques [#features]

Collectez et envoyez des données télémétriques à New Relic depuis vos services AWS Backup en utilisant notre intégration. Vérifiez vos services, interrogez les données entrantes et créez un dashboard pour tout observer en un coup d&apos;œil.

## Activer l&apos;intégration [#activate-integration]

Cette intégration est disponible via CloudWatch MetricStreams.

Pour activer cette intégration, découvrez comment [connecter les services AWS à New Relic via les flux de métriques CloudWatch](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Rechercher et utiliser des données [#find-use-data]

Pour trouver les métriques de votre intégration, accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics &amp; Events**</DNT> et filtrez par `aws.backup`.

De plus, vous pouvez trouver l&apos;entité associée à AWS Backup en accédant à [one.newrelic.com](https://one.newrelic.com/all-entities) et en recherchant `backup`.
