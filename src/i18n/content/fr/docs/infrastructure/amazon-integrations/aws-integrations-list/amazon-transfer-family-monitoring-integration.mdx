New Relic s&apos;intègre à Amazon Web Services (AWS) pour signaler vos métriques [AWS Transfer Family](https://aws.amazon.com/aws-transfer-family/) et d&apos;autres données à New Relic.

Ce document explique comment activer l&apos;intégration et décrit les données signalées.

## Caractéristiques [#features]

Collectez et envoyez des données télémétriques à New Relic depuis votre famille AWSTransfer en utilisant notre intégration. Monitorez vos services, interrogez les données entrantes et créez un dashboard pour tout observer en un coup d&apos;œil.

## Activer l&apos;intégration [#activate-integration]

Cette intégration est disponible via flux de métriques CloudWatch.

Pour activer cette intégration, découvrez comment [connecter les services AWS à New Relic via les flux de métriques CloudWatch](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Rechercher et utiliser des données [#find-use-data]

Pour trouver les métriques de votre intégration, accédez à <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics &amp; Events**</DNT> et filtrez par `aws.transfer`.
