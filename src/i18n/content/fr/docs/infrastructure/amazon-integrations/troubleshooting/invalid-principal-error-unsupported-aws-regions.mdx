## Problème

Lors de la connexion [d&apos;AWS à une intégration d&apos;infrastructure](/docs/integrations/amazon-integrations/get-started/connect-aws-services-infrastructure#connect), pendant la phase <DNT>**Review**</DNT> , vous recevez l&apos;erreur AWS `Invalid principal in policy`.

## Solution

Vérifiez le type d’intégration que vous souhaitez installer. New Relic peut obtenir des données monitoring à partir de services et de points de terminaison situés dans toutes [les régions AWS,](https://docs.aws.amazon.com/general/latest/gr/rande.html) à l&apos;exception des régions chinoises.

Vous verrez cette erreur lorsque vous essayez de configurer l’intégration à partir d’un compte qui utilise la Chine.
