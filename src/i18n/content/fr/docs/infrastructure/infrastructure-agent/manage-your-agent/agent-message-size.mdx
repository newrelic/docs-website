Depuis infrastructure [la version 1.0.989 de](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes) l&apos;agent , la taille maximale des messages est de 1 Mo (10^6 octets) et peut inclure indistinctement des événements, des métriques et l&apos;intégration d&apos;inventaire. Auparavant, la taille maximale des messages était de 5 Mo, bien que les données d&apos;inventaire étaient limitées à 3 Mo.
