## Installer à l'aide de fichiers tarball [#custom-setups-tarball]

Pour les scénarios de configuration <DNT>**custom**</DNT> , vous pouvez installer l&amp;apos;agent d&amp;apos;infrastructure à l&amp;apos;aide [de nos fichiers tarball](https://download.newrelic.com/infrastructure_agent/binaries/linux/) en mode [assisté](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-assisted-install-infrastructure-linux) ou [manuel](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-manual-install-infrastructure-linux) .

Ceci est particulièrement utile lorsque vous devez adapter les paramètres d'installation par défaut à votre environnement ou installer l'agent infrastructure sur des distributions qui n'ont pas le package `newrelic-infra` dans leur référentiel.

<Callout variant="important">
  Notez que l'installation personnalisée de l'agent d'infrastructure à l'aide de fichiers tarball n'est pas officiellement prise en charge.
</Callout>
