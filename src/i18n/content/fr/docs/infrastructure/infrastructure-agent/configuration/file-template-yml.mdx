L'agent d'infrastructure lit le fichier `newrelic-infra.yml` pour sa [configuration](/docs/infrastructure/install-configure-infrastructure/configuration/configure-infrastructure-agent). Vous pouvez trouver le dernier modèle de configuration dans le [référentiel infra-agent sur GitHub](https://github.com/newrelic/infrastructure-agent/blob/master/assets/examples/infrastructure/newrelic-infra-template.yml.example).

<Callout variant="tip">
  Nous vous recommandons [de vérifier](http://www.yamllint.com/) les fichiers de configuration YAML avant de les utiliser pour éviter les problèmes de formatage.
</Callout>

Pour plus d'informations sur chaque paramètre, voir [Paramètres de configuration de l'infrastructure](/docs/infrastructure/install-configure-manage-infrastructure/configuration/infrastructure-configuration-settings).

<Callout variant="important">
  [Redémarrez toujours l'agent](/docs/infrastructure-start-stop-restart-check-agent-status) ou votre serveur Web après avoir modifié les paramètres.
</Callout>
