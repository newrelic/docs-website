## Cause

Certaines métriques ne sont pas présentes dans les API habituelles des fournisseurs de cloud (CloudWatch, Stackdriver, Azure Monitor) et sont plutôt extraites des API de service. Chaque fournisseur de services cloud dispose d&apos;une API de service unique qui traite les données et interagit avec le service. Par exemple, si une métrique n&apos;est pas présente dans AWS CloudWatch, New Relic récupérera la métrique à partir de l&apos;API du service AWS ECS.
