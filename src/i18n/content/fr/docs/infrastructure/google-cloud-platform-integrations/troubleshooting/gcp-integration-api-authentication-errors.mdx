## Problème

Vous souhaitez [intégrer vos services Google Cloud Platform (GCP)](/docs/introduction-google-cloud-platform-integrations) à New Relic, mais vous constatez des erreurs d&apos;authentification API.

## Solution

Pour résoudre le problème :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Infrastructure &amp;gt; GCP**</DNT>.
2. Depuis la page de votre compte GCP, sélectionnez <DNT>**Reauthorize New Relic**</DNT>.
3. Suivez les procédures standard pour [connecter vos services GCP à New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/#connect).

## Cause

Les raisons courantes de cette erreur incluent :

* L&apos;autorisation pour New Relic d&apos;utiliser les API GCP a été révoquée manuellement par l&apos;utilisateur à partir de l&apos;application de gestion de compte Google.
* L&apos;autorisation pour New Relic d&apos;utiliser les API GCP n&apos;inclut pas une ou plusieurs des autorisations ([étendues Google OAuth](https://developers.google.com/identity/protocols/googlescopes)) nécessaires pour interroger une ou plusieurs des API GCP.
* L&apos;autorisation pour New Relic d&apos;utiliser les API GCP a expiré.
