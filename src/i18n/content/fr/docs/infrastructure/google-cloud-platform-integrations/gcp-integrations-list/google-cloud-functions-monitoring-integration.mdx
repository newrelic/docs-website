Notre [intégrationinfrastructure ](/docs/infrastructure/introduction-infra-monitoring)avec [Google Cloud Platform (GCP)](https://cloud.google.com/) inclut une solution qui transmet les données [Google Cloud Functions](https://cloud.google.com/functions/) à nos produits. Ce document explique comment activer l’intégration de GCP Cloud Functions et décrit les données qui peuvent être signalées.

## Caractéristiques

Le service Google Cloud Functions permet d&apos;exécuter du code sans serveur. En utilisant l&apos;[UI de Google](https://cloud.google.com/functions/docs/quickstarts), les développeurs peuvent créer de courts morceaux de code destinés à exécuter une fonction spécifique. La fonction peut alors répondre à un événement cloud sans avoir besoin de gérer un serveur d&apos;application ou un environnement d&apos;exécution.

## Activer l&apos;intégration [#activate]

Pour activer l&apos;intégration, suivez les procédures standard pour [connecter votre service GCP à New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Fréquence des sondages [#polling]

Notre intégration requête vos services GCP selon un intervalle d&apos;interrogation, qui varie en fonction de l&apos;intégration.

* Fréquence d&apos;interrogation pour GCP Cloud Functions : cinq minutes
* Résolution : un point de données par minute

## Afficher et utiliser les données [#find-data]

Après avoir activé l&apos;intégration puis attendu quelques minutes (en fonction de la [fréquence d&apos;interrogation](#polling)), les données apparaîtront dans l&apos;UI. Pour [afficher et utiliser vos données](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), y compris les liens vers vos <InlinePopover type="dashboards" />et paramètres d&apos;alerte, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; GCP**</DNT>, puis sélectionnez une intégration.
