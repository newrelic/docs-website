L&apos;[intégration Azure ](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations)de New Relic interroge vos services Azure selon un intervalle d&amp;apos;interrogation propre à l&amp;apos;intégration.

L’intervalle d’interrogation s’applique à <DNT>**every**</DNT> entité Azure associée au service intégré. Par exemple, si vous avez treize instances CosmosDB, chacune des treize instances sera interrogée toutes les cinq minutes.

## Afficher les données du sondage [#account-status]

Une fois que vous [avez activé une intégration Azure ](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations), New Relic commence à interroger les données d’ Azure et rend les données accessibles via notre UI. Vous pouvez interroger les données Azure ainsi que des données supplémentaires importées à partir de n’importe quelle autre fonctionnalité New Relic. Vous pouvez également afficher les données dashboard pour une intégration spécifique ou pour l&amp;apos;ensemble de votre compte.

Pour les visualisations de l&apos;intervalle d&apos;interrogation, de l&apos;appel d&apos;API et d&apos;autres données pour votre intégration Azure :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Infrastructure &amp;gt; Azure**</DNT>.
2. Pour afficher les données d’une intégration spécifique : sélectionnez le lien <DNT>**Dashboards**</DNT> pour la ligne de l’intégration.

## Intervalle d&apos;interrogation de New Relic [#aws-integrations]

Pour plus de détails sur l’interrogation et la résolution, consultez la documentation d’une intégration spécifique dans la section [Liste des intégrations Azure ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/).
