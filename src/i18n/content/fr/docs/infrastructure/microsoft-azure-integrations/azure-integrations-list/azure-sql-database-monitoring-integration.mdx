[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour rapporter votre [base de données métriquesMicrosoft Azure SQL](https://azure.microsoft.com/en-us/services/sql-database/) et d&amp;apos;autres données à New Relic. Ce document explique comment activer l&amp;apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic rassemble des données Azure cloud de base de données à partir du service relationnel de bases de données entièrement géré . Azure SQL fournit une base de données unique avec son propre ensemble de ressources et des pools élastiques qui partagent un ensemble de ressources. Les deux sont associés à un serveur logique de base de données Azure SQL. Si les bases de données sont protégées par un pare-feu, elles peuvent être répliquées et restaurées à un point antérieur dans le temps.

Grâce à New Relic, vous pouvez :

* Affichez les données de la base de données Azure SQL dans un dashboard prédéfini.
* [Exécutez une requête personnalisée et visualisez les données](/docs/using-new-relic/data/understand-data/query-new-relic-data).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Pour activer l’intégration, suivez les procédures standard pour [activer votre service Azure dans New Relic](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations).

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

New Relic interroge votre service de bases de données Azure selon un intervalle [d&apos;interrogation](/docs/integrations/microsoft-azure-integrations/getting-started/azure-integration-polling) par défaut, qui varie en fonction de l&amp;apos;intégration. Fréquence d&amp;apos;interrogation pour l&amp;apos;intégration de la base de données Azure SQL :

* intervalle d&apos;interrogatoire : 5 minutes
* Résolution : 1 point de données par minute
