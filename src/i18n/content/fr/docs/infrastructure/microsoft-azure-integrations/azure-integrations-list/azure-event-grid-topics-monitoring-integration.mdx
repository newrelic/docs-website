[L'intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour rapporter vos métriques [de sujets de grille d'événementsMicrosoft Azure ](https://azure.microsoft.com/en-us/products/event-grid/)et d&amp;apos;autres données à New Relic. Ce document explique comment activer l&amp;apos;intégration et décrit les données rapportées.

## Caractéristiques

événement Grid est un courtier d'événements hautement évolutif et sans serveur que vous pouvez utiliser pour intégrer des applications à l'aide d'événement. Les événements sont livrés par événement Grid aux destinations des abonnés telles que les applications, les services Azure ou tout point de terminaison auquel événement Grid a accès au réseau. Le sujet de la grille événement fournit un point de fin où la source envoie l'événement. Un sujet est utilisé pour une collection d'événements connexes. Pour répondre à certains types d'événements, les abonnés décident à quels sujets s'abonner.

Grâce à New Relic, vous pouvez :

* Afficher les données des rubriques Azure événement Grid dans Azure dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d'alerte pour vous avertir des changements de données.

## Activer l'intégration [#activate]

Suivez la [Azure Monitor procédure d'intégration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standard pour activer votre Azure service dans de New Relic.monitoring infrastructure

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d'interrogation et filtrer les données à l'aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge vos services Azure Event Grid Topics via l'intégration Azure Monitor selon un intervalle [d'interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &amp;gt; Azure &amp;gt; (select an integration)**</DNT>.
