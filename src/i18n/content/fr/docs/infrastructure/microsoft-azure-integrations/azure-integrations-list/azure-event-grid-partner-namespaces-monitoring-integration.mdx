[L'intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour signaler votre [espace de nommage métriques de partenaireMicrosoft Azure événement Grid ](https://learn.microsoft.com/en-us/azure/event-grid/partner-events-overview-for-partners)et d&amp;apos;autres données à New Relic. Ce document explique comment activer l&amp;apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic rassemble les données métriques d' Azure Monitor pour l'espace de nommage Azure événement Grid Partner. Un espace de nommage partenaire est une ressource régionale qui dispose d'un point de fin pour publier un événement sur Azure événement Grid. L'espace de nommage partenaire contient soit des chaînes, soit des chaînes d'événements (ressource legacy ).

Grâce à New Relic, vous pouvez :

* Affichez les données de l'espace de nommage Azure événement Grid Partner dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d'alerte pour vous avertir des changements de données.

## Activer l'intégration [#activate]

Suivez la [Azure Monitor procédure d'intégration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standard pour activer votre Azure service dans de New Relic.monitoring infrastructure

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d'interrogation et filtrer les données à l'aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic requête vos services Azure événement Grid Partner espace de nommage via l’intégration Azure Monitor selon un intervalle [d’interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &amp;gt; Azure &amp;gt; (select an integration)**</DNT>.
