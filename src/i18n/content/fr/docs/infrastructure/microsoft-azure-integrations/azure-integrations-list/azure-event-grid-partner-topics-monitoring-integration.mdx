[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour rapporter vos métriques de sujets [Microsoft Azure événement Grid Partner Topics ](https://azure.microsoft.com/en-us/products/event-grid/)et d&amp;apos;autres données à New Relic. Ce document explique comment activer l&amp;apos;intégration et décrit les données rapportées.

## Caractéristiques

Une rubrique partenaire dans Event Grid représente un ou plusieurs événements publiés par des services Azure tels qu&apos;Azure Azure et Azure Event Hubs. Par exemple, une rubrique partenaire peut représenter tous les événements blob ou uniquement les événements blob créés et supprimés publiés pour un compte de stockage spécifique.

Grâce à New Relic, vous pouvez :

* Affichez les données des rubriques partenaires Azure événement Azure dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [Azure Monitor procédure d&apos;intégration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standard pour activer votre Azure service dans de New Relic.monitoring infrastructure

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge vos services Azure Event Grid Partner Topics via l&apos;intégration Azure Monitor selon un intervalle [d&apos;interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &amp;gt; Azure &amp;gt; (select an integration)**</DNT>.
