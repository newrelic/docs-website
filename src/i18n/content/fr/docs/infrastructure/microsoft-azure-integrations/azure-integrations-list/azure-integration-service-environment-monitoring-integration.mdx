[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour rapporter vos métriques [d&apos;environnement de service d&apos;intégrationMicrosoft Azure ](https://learn.microsoft.com/en-us/azure/logic-apps/ise-manage-integration-service-environment/)et d&apos;autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

Un environnement de service d’intégration est un environnement entièrement isolé et dédié à tous les besoins d’intégration à l’échelle de l’entreprise. Lorsque vous créez un nouvel environnement de service d’intégration, il est injecté dans votre réseau virtuel Azure, ce qui vous permet de déployer Logic Apps en tant que service dans votre VNET.

Grâce à New Relic, vous pouvez :

* Affichez les données de l’environnement de service d’intégration Azure dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge votre environnement de service d’intégration Azure via l’intégration Azure Monitor selon un intervalle [d’interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; Azure &gt; (select an integration)**</DNT>.
