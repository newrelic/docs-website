[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour rapporter vos [mesures de point de terminaison de streamingMicrosoft Azure Media Services](https://learn.microsoft.com/en-us/azure/media-services/latest/stream-streaming-endpoint-concept) et d&apos;autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte des données métriques d’ Azure Monitor pour le point de terminaison Azure Media Services Streaming. Un point de terminaison de streaming représente un service d&apos;origine et de packaging dynamique (juste à temps) qui peut diffuser votre contenu en direct et à la demande directement vers une application de lecteur client, en utilisant l&apos;un des protocoles de diffusion multimédia en continu courants (HLS ou DASH).

Grâce à New Relic, vous pouvez :

* Affichez les données de point de terminaison de streaming Azure Media Services dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge votre point de terminaison Azure Media Services Streaming via l&apos;intégration Azure Monitor selon un intervalle [d&apos;interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
