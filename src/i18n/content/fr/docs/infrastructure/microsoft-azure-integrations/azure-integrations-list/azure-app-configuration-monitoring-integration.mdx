[L’intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration permettant de signaler les métriques [de configuration de votre applicationMicrosoft Azure ](https://learn.microsoft.com/en-us/azure/azure-app-configuration/)et d’autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

La configuration de l&apos;application Azure est un service Azure qui permet à l&apos;utilisateur de gérer configuration dans le cloud. L&apos;utilisateur peut créer des magasins de configuration d&apos;application pour stocker les paramètres de clé de valeur et consommer les paramètres stockés à partir d&apos;applications, de pipelines de génération, de processus de sortie, microservices et d&apos;autres ressources Azure .

Grâce à New Relic, vous pouvez :

* Affichez les données de configuration de l’application Azure dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge vos services de configuration d’application Azure via l’intégration Azure Monitor selon un intervalle [d’interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
