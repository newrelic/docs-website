[L'intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration permettant de signaler vos métriques [Microsoft Azure Network Watcher](https://azure.microsoft.com/en-us/products/network-watcher/) et d&amp;apos;autres données à New Relic. Ce document explique comment activer l&amp;apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte les données de métriques d’Azure Monitor pour le service Azure Network Watcher. Azure Network Watcher fournit des outils pour monitorer, diagnostiquer, afficher les métriques et activer ou désactiver le log des ressources dans un réseau virtuel Azure . Network Watcher est conçu pour monitorer et réparer la santé du réseau des produits IaaS (infrastructure en tant que service), y compris les machines virtuelles (VM), les réseaux virtuels (VNets), les passerelles d'application, les équilibreurs de charge, etc.

Grâce à New Relic, vous pouvez :

* Affichez les données Azure Network Watcher dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d'alerte pour vous avertir des changements de données.

## Activer l'intégration [#activate]

Suivez la [Azure Monitor procédure d'intégration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standard pour activer votre Azure service dans de New Relic.monitoring infrastructure

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d'interrogation et filtrer les données à l'aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge vos services Azure Network Watcher via l’intégration d’ Azure Monitor selon un intervalle [d’interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &amp;gt; Azure &amp;gt; (select an integration)**</DNT>.
