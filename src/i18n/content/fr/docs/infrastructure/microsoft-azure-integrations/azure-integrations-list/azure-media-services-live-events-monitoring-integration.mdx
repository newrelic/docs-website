[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration pour rapporter vos métriques [d&apos;événementsMicrosoft Azure Media Services Live](https://learn.microsoft.com/en-us/azure/media-services/latest/live-event-streaming-best-practices-guide) et d&apos;autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte des données métriques d&apos; Azure Monitor pour l&apos;événement Azure Media Services Live. Azure Media Services Live événement est un service fourni par Microsoft Azure qui permet la diffusion vidéo et audio en direct pour divers événements tels que des conférences, des webinaires, des concerts et des événements sportifs. Il permet à l&apos;utilisateur de capturer, d&apos;encoder, de package et de diffuser du contenu vidéo en direct à des publics du monde entier via Internet.

Grâce à New Relic, vous pouvez :

* Affichez les données des événements Azure Media Services Live dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge votre événement Azure Media Services Live via l&apos;intégration Azure Monitor selon un intervalle [d&apos;interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
