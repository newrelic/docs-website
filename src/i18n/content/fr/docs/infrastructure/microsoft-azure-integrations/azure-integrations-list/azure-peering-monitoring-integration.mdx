[L'intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration permettant de signaler vos métriques [Microsoft Azure Peering](https://learn.microsoft.com/en-in/azure/internet-peering/) et d&amp;apos;autres données à New Relic. Ce document explique comment activer l&amp;apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte les données de métriques d’Azure Monitor pour le peering Azure. Le peering est l'interconnexion entre le réseau mondial de Microsoft (AS8075) et votre réseau dans le but d'échanger du trafic Internet depuis/vers les services en ligne Microsoft et les services Microsoft Azure. Les opérateurs ou prestataires de services peuvent demander à se connecter à Microsoft dans l'un de nos sites Edge.

Grâce à New Relic, vous pouvez :

* Affichez les données de peering Azure dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d'alerte pour vous avertir des changements de données.

## Activer l'intégration [#activate]

Suivez la [Azure Monitor procédure d'intégration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standard pour activer votre Azure service dans de New Relic.monitoring infrastructure

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d'interrogation et filtrer les données à l'aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge votre peering Azure via l’intégration Azure Monitor selon un intervalle d’ [interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Infrastructure &amp;gt; Azure &amp;gt; (select an integration)**</DNT>.
