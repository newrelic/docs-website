[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration permettant de signaler les métriques de vos [applicationsMicrosoft Azure IoT Central](https://azure.microsoft.com/en-us/products/iot-central/) et d&apos;autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte les données de métriques d’Azure Monitor pour le service Azure IoT Central Applications. Nous pouvons rapidement déployer une nouvelle application IoT Central, puis la personnaliser selon vos besoins spécifiques. Les modèles d’application dans Azure IoT Central sont un outil pour vous aider à démarrer le développement de votre solution IoT.

Grâce à New Relic, vous pouvez :

* Affichez les données des applications Azure IoT Central dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge vos services d’applications Azure IoT Central via l’intégration Azure Monitor selon un intervalle d’ [interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
