[L&apos;intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration permettant de signaler vos métriques [Microsoft Azure Media Services Video Analyzer](https://learn.microsoft.com/en-us/azure/azure-video-analyzer) et d&apos;autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte des données de métriques à partir d’Azure Monitor pour Azure Media Services Video Analyzer. Azure Video Analyzer for Media, anciennement connu sous le nom d’Azure Video Indexer, est disponible sous AMS et également en tant que service Azure distinct.

Grâce à New Relic, vous pouvez :

* Affichez les données d’ Azure Media Services Video Analyzer dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge votre analyseur vidéo Azure Media Services via l’intégration Azure Monitor selon un intervalle d’ [interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
