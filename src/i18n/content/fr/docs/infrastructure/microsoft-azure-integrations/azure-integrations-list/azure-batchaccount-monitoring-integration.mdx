[L’intégration de New Relic](/docs/infrastructure/introduction-infra-monitoring) inclut une intégration permettant de signaler les métriques de votre [compteMicrosoft Azure Batch](https://azure.microsoft.com/en-us/services/batch/) et d’autres données à New Relic. Ce document explique comment activer l&apos;intégration et décrit les données rapportées.

## Caractéristiques

New Relic collecte les données de métriques du service Azure Monitor for Batch Account, Azure Batch, pour exécuter efficacement des tâches par lots de calcul parallèle et hautes performances (HPC) à grande échelle dans Azure. Azure Batch crée et gère un pool de nœuds calculer (machines virtuelles), installe les applications que vous souhaitez exécuter et planifie les tâches à exécuter sur les nœuds.

Grâce à New Relic, vous pouvez :

* Affichez les données du compte Azure Batch dans un dashboard prédéfini.
* Exécutez [une requête personnalisée et visualisez les données](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Créez une condition d&apos;alerte pour vous avertir des changements de données.

## Activer l&apos;intégration [#activate]

Suivez la [procédure d&apos;intégration d'Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) standarde pour activer votre Azure service dans le monitoring infrastructure de New Relic.

## configuration et sondage [#polling]

Vous pouvez modifier la fréquence d&apos;interrogation et filtrer les données à l&apos;aide [des options de configuration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic interroge vos services Azure Batch via l’intégration Azure Monitor selon un intervalle d’ [interrogation](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) par défaut.

## Rechercher et utiliser des données [#find-and-use]

Pour [explorer vos données d’intégration](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), accédez à <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
