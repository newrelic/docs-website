L&apos;[intégration open source](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) Memcached collecte les compteurs de performances et les statistiques de [Memcached](https://memcached.org/) et transmet les données à New Relic.

Pour obtenir des informations sur les exigences, l&apos;installation et configuration , consultez le [READMEde l&apos;intégration Memcached sur hôte](https://github.com/newrelic/nri-memcached) sur GitHub.

<Callout variant="important">
  New Relic a ouvert le code source de cette intégration pour permettre monitoring de cette technologie. Cette intégration est fournie « TELLE QUELLE » SANS GARANTIE NI ASSISTANCE, bien que vous puissiez signaler des problèmes et contribuer à cette intégration via GitHub. La prise en charge de cette intégration est disponible avec un [abonnement Expert Services](https://www.newrelic.com/expertservices).
</Callout>
