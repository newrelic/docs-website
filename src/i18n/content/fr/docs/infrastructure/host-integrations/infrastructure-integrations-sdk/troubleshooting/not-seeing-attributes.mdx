## Problème

Votre intégration rapporte des données, mais quelques attributs d'inventaire manquent.

## Solution

Les noms de clés d'inventaire suivants sont utilisés en interne par notre plateforme. Si l'intégration fournie par les clients envoie des éléments d'inventaire avec le même nom de clé, ils resteront invisibles pour l'utilisateur :

* `pid`
* `installed_epoch`
* `epoch`
* `time`
* `size`
* `boot_id`
* `product_uuid`
* `installed_time`
