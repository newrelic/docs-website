## Problème

Votre intégration rapporte des données, mais quelques attributs d&apos;inventaire manquent.

## Solution

Les noms de clés d&apos;inventaire suivants sont utilisés en interne par notre plateforme. Si l&apos;intégration fournie par les clients envoie des éléments d&apos;inventaire avec le même nom de clé, ils resteront invisibles pour l&apos;utilisateur :

* `pid`
* `installed_epoch`
* `epoch`
* `time`
* `size`
* `boot_id`
* `product_uuid`
* `installed_time`
