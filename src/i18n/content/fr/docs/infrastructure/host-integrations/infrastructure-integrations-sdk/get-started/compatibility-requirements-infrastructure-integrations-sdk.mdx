## Version d&apos;infrastructure [#agent-version]

Le SDK d&apos;intégration d&apos;infrastructure v4 nécessite la version 1.13.0 ou supérieure de l&apos;agent d&apos;infrastructure.

Le SDK v3 nécessite la version d&apos;agent 1.0.888 ou supérieure.

Pour obtenir des informations sur la version de l&apos;agent infrastructure et les options d&apos;installation et de mise à jour, consultez les [notes de sortie de l&apos;infrastructure](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes).

Pour obtenir des notes sur les versions et les modifications du SDK, consultez le [log des modifications](#change-log).

## système d&apos;exploitation [#os]

l&apos;intégration construite avec le SDK peut être compilée pour un système d&apos;exploitation Linux ou Windows .

## Spécifications des données et des fichiers [#data-specs]

L&apos;intégration d&apos;infrastructure sur hôte peut être créée avec n&apos;importe quel langage de programmation, à condition qu&apos;il respecte les [spécifications des données et des fichiers](/docs/infrastructure/integrations-sdk/get-started/create-activate-infrastructure-integration).
