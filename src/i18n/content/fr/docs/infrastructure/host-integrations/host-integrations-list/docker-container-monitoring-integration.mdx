L&apos;[intégration](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) de Docker Container monitore et rapporte les métriques des conteneurs Docker à l&amp;apos;aide de [l&apos;API Docker](https://docs.docker.com/reference/api/engine/version/v1.48/) vers New Relic.

<Callout variant="tip">
  Cette intégration est sortie en [open source sur Github](https://github.com/newrelic/nri-docker). Un log des modifications y est également disponible pour les dernières mises à jour. La prise en charge de cette intégration est disponible avec un [abonnement Expert Services](https://www.newrelic.com/expertservices).
</Callout>

<Callout variant="important">
  * Cette intégration ne supporte pas LCOW (conteneur Linux sur Windows). Pour plus d&apos;informations, consultez la [documentation Docker](https://docs.docker.com/desktop/windows/wsl/).
  * Pour Linux - cette intégration est fournie avec l&apos;agent infrastructure et deviendra automatiquement active si des conteneurs Docker sont détectés.
  * Pour Windows - à partir de la version 2.3.0, nous fournissons la compatibilité, mais cela nécessite une installation manuelle sur l&apos;agent d&apos;infrastructure. Il sera fourni avec l&apos;agent infrastructure dans une future sortie.
</Callout>
