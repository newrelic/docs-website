Après avoir configuré vos données réseau pour monitoring des performances, accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Network**</DNT> pour explorer les données réseau que vos appareils signalent à New Relic.

Vous pouvez regrouper et monitorer vos commutateurs, routeurs, etc. dans les dashboards. Cela vous aide à regrouper, isoler et analyser rapidement la santé de votre réseau.

Pour quelques exemples de monitoring de votre réseau dans New Relic, regardez cette courte vidéo (environ 15 minutes). 3:45 minutes) :
