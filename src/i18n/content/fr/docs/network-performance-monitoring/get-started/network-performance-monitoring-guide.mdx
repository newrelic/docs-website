Vous êtes un ingénieur d’exploitation qui s’appuie sur un réseau d’appareils compatibles SNMP. Votre équipe utilise déjà la fonctionnalité de New Relic pour monitoring les logiciels et infrastructure, mais il est temps de monitorer également vos périphériques réseau.

Dans cette piste, vous utilisez la bibliothèque ktranslate de Kentik pour découvrir automatiquement vos périphériques réseau, collecter des données et envoyer ces données à New Relic afin de corréler les performances du réseau avec l&apos;infrastructure, les applications et les expériences numériques. Ensuite, vous utilisez New Relic pour monitorer la santé de votre réseau.

## Objectifs

* Interroger les données SNMP des périphériques réseau et les envoyer à New Relic
* Exemple de données de flux réseau et envoi à New Relic
* Créez un dashboard New Relic pour monitoring vos périphériques réseau
* Créez une workload New Relic pour regrouper logiquement vos appareils et configurer la détection des anomalies
* Utilisez vos nouvelles données pour comprendre les comportements au sein de votre réseau

## Exigences

* Un compte New Relic gratuit
* Un utilisateur de la plateforme complète ou un utilisateur principal avec le privilège de modification d&apos;utilisateur Nerdpack

<Callout variant="important">
  Cette piste utilise une machine virtuelle éphémère. Par conséquent, vous devez terminer l&apos;intégralité de la piste en une seule fois, sinon vous perdrez votre progression lorsque le temps expirera.
</Callout>
