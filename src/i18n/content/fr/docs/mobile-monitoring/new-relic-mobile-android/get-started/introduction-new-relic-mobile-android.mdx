[Monitoring des applications mobiles](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile) pour Android monitorez votre application mobile, vous donnant une vue complète des performances de votre application. Cela fonctionne pour les applications Android écrites en Java ou Kotlin.

## Compatibilité et exigences [#requirements]

Avant d&apos;installer l&apos;agent Android, vérifiez la [compatibilité et les exigences](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements/).

## Installer l&apos;agent [#installation]

Pour installer l&apos;agent Android, suivez notre [installation guidée](https://onenr.io/02R5GVpaxQb), située directement dans l&amp;apos;interface utilisateur. En utilisant cette méthode, vous pouvez installer et configurer l’agent Android avec Gradle et Android Studio.
