Pour vous assurer que vous disposez de la version la plus récente de l&apos;agent Android , consultez les [notes de sortie de l&apos;agent Android](/docs/release-notes/mobile-release-notes/android-release-notes). Les instructions de mise à niveau de votre application Android varient en fonction de la version du SDK Android de l&apos;agent que vous utilisez actuellement.

## Mise à niveau à partir des versions 2 ou 3 du SDK Android [#android\_upgrade\_2][#android_upgrade_2]

Si vous avez déjà installé la version 2 ou 3 du SDK Android : Accédez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Settings &gt; Upgrade.**</DNT>
