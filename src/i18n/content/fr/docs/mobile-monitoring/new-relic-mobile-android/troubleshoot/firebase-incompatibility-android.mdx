## Problème

Des conflits, des plantages ou des erreurs s&apos;afficheront lorsque l&apos;agent Android New Relic et [le SDK de monitoring des performances Firebase](https://firebase.google.com/docs/perf-mon) de Google sont inclus dans une application.

Erreurs connues :

* `java.lang.StackOverflowError`
* `java.lang.RuntimeException: An error occurred while executing doInBackground()`

## Solution

Notre agent Android et FireBase PerfMon ne peuvent pas être activés en même temps. Nous vous recommandons d&apos;activer uniquement l&apos;un ou l&apos;autre pour éviter ce problème.

Si vous avez besoin d&apos;aide supplémentaire, obtenez de l&apos;aide sur [support.newrelic.com](https://support.newrelic.com).
