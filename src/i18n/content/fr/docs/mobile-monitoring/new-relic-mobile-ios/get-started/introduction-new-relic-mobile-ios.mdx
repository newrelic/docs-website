Notre agent iOS vous offre une vue complète des performances de votre application. Les applications iOS de monitoring d&apos;agent écrites en Objective-C, Swift ou les deux. Pour un aperçu général des fonctionnalités monitoring des applications mobiles, voir [Introduction à monitoring des applications mobiles](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile).

## Compatibilité et exigences [#requirements]

Avant d&apos;installer l&apos;agent iOS, vérifiez la [compatibilité et les exigences](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements/).

## Installer l&apos;agent [#installation]

Pour installer l&apos;agent iOS, suivez notre [installation guidée](https://onenr.io/0qwLv87gkj5), située directement dans l&apos;interface utilisateur. En utilisant cette méthode, vous pouvez installer et configurer l&apos;agent iOS avec Swift gestionnaire de paquets (SPM) ou CocoaPods.

## Dépannez votre installation [#troubleshooting]

Si vous rencontrez des problèmes avec votre installation iOS, ou si vous ne voyez pas de données dans l&apos;interface utilisateur monitoring des applications mobiles pour votre application iOS, essayez ces [solutions de dépannage](/docs/mobile-monitoring/new-relic-mobile-ios/troubleshoot/no-data-appears-ios).
