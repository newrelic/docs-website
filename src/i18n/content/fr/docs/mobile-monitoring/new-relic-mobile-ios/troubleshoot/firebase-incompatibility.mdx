## Problème

Des conflits, des plantages ou des erreurs peuvent apparaître lorsque l&apos;agent iOS New Relic et d&apos;autres agents de suivi de données télémétriques tiers comme Firebase sont inclus dans une application.

Erreurs connues :

* Crash dans les fonctions instrumentées.
* Échec de la capture du log des plantages.

## Solution

Notre agent iOS et d&apos;autres agents de suivi de données télémétriques tiers comme Firebase ne doivent pas être activés en même temps. Nous vous recommandons d&apos;activer uniquement l&apos;un ou l&apos;autre pour éviter ce problème.

S&apos;il n&apos;est pas possible de supprimer l&apos;autre SDK, vous pouvez suivre certaines étapes pour éviter ces conflits.

* Démarrez notre agent avant de démarrer tout autre SDK tiers.
* Désactiver certaines fonctionnalités de notre agent. Si cette option vous convient, consultez notre [guide API](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings/) pour plus d&amp;apos;informations.

Si vous avez besoin d&apos;aide supplémentaire, obtenez de l&apos;aide sur [support.newrelic.com](https://support.newrelic.com).
