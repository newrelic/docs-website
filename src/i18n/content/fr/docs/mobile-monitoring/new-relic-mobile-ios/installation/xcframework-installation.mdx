Nous vous recommandons d&apos;utiliser notre [guide d&apos;installation](https://onenr.io/0qwLv87gkj5) pour configurer monitoring iOS. Cependant, si vous devez installer l&apos;agent manuellement, suivez les étapes ci-dessous pour installer l&apos;agent New Relic iOS (dans un package XCFramework) et l&apos;ajouter à votre projet XCode.

<Steps>
  <Step>
    Téléchargez le dernier agent iOS (XCFramework) à partir de nos [notes de sortie de l&apos;agent iOS](/docs/release-notes/mobile-release-notes/ios-release-notes).
  </Step>

  <Step>
    Accédez à l’endroit où l’agent iOS a été téléchargé et décompressez-le.
  </Step>

  <Step>
    Dans votre projet Xcode, ajoutez le XCFramework décompressé :

    1. Faites un clic droit sur votre navigateur de projet et cliquez sur **Add Files to `[YourProjectName]`**.
    2. Sélectionnez le répertoire `NewRelic.xcframework` et cliquez sur **Add**.
  </Step>

  <Step>
    Intégrer l&apos;agent iOS :

    1. Dans le navigateur de projet, ouvrez votre projet et les paramètres cibles.
    2. Cliquez sur la cible à laquelle vous souhaitez ajouter l&apos;agent.
    3. Cliquez sur l’onglet **General** .
    4. Faites défiler jusqu&apos;à la section **Frameworks, Libraries, and Embedded Content** .
    5. Assurez-vous que `NewRelic.xcframework` est répertorié ici. Si ce n’est pas le cas, cliquez sur le bouton **+** pour l’ajouter.
  </Step>
</Steps>
