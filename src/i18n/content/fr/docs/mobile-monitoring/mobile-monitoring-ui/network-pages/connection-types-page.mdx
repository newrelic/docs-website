La page <DNT>**Connection Types**</DNT> pour <InlinePopover type="mobile" />inclut des graphiques qui montrent les technologies de connexion au réseau sans fil de votre utilisateur et leur impact sur le temps de réponse, le taux d&apos;erreur et le trafic actif pour la période sélectionnée. Les données sont regroupées par génération de technologie : 2G, 3G, etc. Le Wifi et l&apos;Inconnu sont également inclus dans la répartition des types de connexion. À partir de là, vous pouvez explorer en détail des technologies sans fil spécifiques telles que GPRS, HSDPA et LTE.

Pour afficher les types de connexion sans fil mobile de votre utilisateur :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Network &gt; Connection types**</DNT>.

   <img title="connection types.png" alt="connection types.png" src="/images/mobile_screenshot-full_connection-types-summary.webp" />

<figcaption>
  <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Network &gt; Connection types**</DNT>:Utilisez cette page pour afficher, trier ou explorer des informations détaillées sur le type de connexion au réseau mobile de votre utilisateur par impact sur le temps de réponse, les pannes de réseau et le trafic actif.
</figcaption>

2. Pour modifier les informations qui s&apos;affichent (y compris le temps de réponse, les périphériques actifs ou les erreurs réseau), sélectionnez votre choix dans le menu <DNT>**Sort by**</DNT> .
3. Pour sélectionner les versions ou la période de l&apos;application mobile, utilisez le menu [<DNT>**Versions**</DNT>](/docs/mobile-apps/versions) et [le sélecteur d&apos;intervalle de temps](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) sous la barre de menu New Relic.
4. Pour afficher les détails d’un type de connexion spécifique, sélectionnez son nom.

Pour accéder à des informations détaillées, utilisez n&apos;importe quelle [fonctionnalité d&apos;interface utilisateur de plateforme](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) applicable.
