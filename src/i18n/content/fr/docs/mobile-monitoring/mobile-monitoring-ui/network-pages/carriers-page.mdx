La page Opérateurs pour <InlinePopover type="mobile" />inclut des graphiques qui montrent les opérateurs sans fil de votre utilisateur et leur impact sur le temps de réponse, le taux d&apos;erreur et les sessions actives pour la période sélectionnée. Le Wifi est inclus en tant qu&apos;opérateur. À partir de là, vous pouvez trier et explorer des informations détaillées sur des transporteurs spécifiques.

## Affichage de la page des transporteurs [#procedures]

</figcaption>

Pour afficher les opérateurs mobiles de votre utilisateur :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Network &gt; Carriers**</DNT>.
2. Pour modifier les informations qui s&apos;affichent (y compris le temps de réponse, les périphériques actifs ou les erreurs réseau), sélectionnez votre choix dans le menu <DNT>**Sort by**</DNT> .
3. Pour ajuster la quantité d’informations qui s’affichent, sélectionnez <DNT>**Hide &lt; 1% throughput**</DNT>.
4. Pour sélectionner les versions ou la période de l&apos;application mobile, utilisez le menu [<DNT>**Versions**</DNT>](/docs/mobile-apps/versions) et [le sélecteur d&apos;intervalle de temps](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) sous la barre de menu New Relic.
5. Pour afficher les détails d’un transporteur spécifique, sélectionnez son nom.

Pour accéder à des informations détaillées, utilisez n&apos;importe quelle [fonctionnalité d&apos;interface utilisateur de plateforme](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) applicable.
