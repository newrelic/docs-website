La page Géographie présente l&apos;expérience de votre utilisateur mobile sous forme de vision du monde, notamment :

* Temps de réponse codé par couleur
* requests réseau (appels par minute)
* Taille du transfert de données
* Appareils actifs
* Taux de défaillance du réseau

Vous pouvez également accéder à des informations détaillées sur chaque pays.

<Callout variant="tip">
  La fonctionnalité <DNT>**Geography**</DNT> n&apos;est pas la même que la fonctionnalité <DNT>**Map**</DNT> . La [page<DNT>**Map**</DNT> ](/docs/mobile-monitoring/mobile-monitoring-ui/network-pages/map-page-mobile-apps)montre une vue architecturale de la relation entre une application mobile et ses services associés.
</Callout>

## Affichage de la page Géographie [#geo\_procedures][#geo_procedures]

Pour afficher ou trier le temps de réponse mobile par pays :

1. Allez à <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Network &gt; Geography**</DNT>.
2. Pour modifier les informations qui s&apos;affichent (y compris les temps de réponse, requests par minute, la taille totale du transfert, les appareils actifs ou le taux de défaillance du réseau), sélectionnez votre choix dans le menu <DNT>**Sort by**</DNT>.
3. Pour ajuster la quantité d’informations qui s’affichent, sélectionnez <DNT>**Hide &lt; 1% throughput**</DNT>.
4. Pour afficher des informations récapitulatives sur un emplacement, passez la souris sur n&apos;importe quelle zone en couleur sur la carte ou sur le nom du pays dans la liste.

Pour accéder à des informations détaillées, utilisez n&apos;importe quelle [fonctionnalité d&apos;interface utilisateur de plateforme](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) applicable.

## Affichage des détails détaillés [#geo\_details][#geo_details]

Pour afficher des informations détaillées sur un emplacement spécifique (y compris le temps de réponse moyen, les appels par minute, les appareils actifs et les pannes de réseau par type), sélectionnez son emplacement sur la carte de la page Géographie ou sélectionnez son nom dans la liste.
