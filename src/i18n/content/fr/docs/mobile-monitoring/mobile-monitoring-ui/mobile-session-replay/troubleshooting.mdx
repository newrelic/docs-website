C&apos;est la première étape pour diagnostiquer par vous-même votre problème de Session Replay New Relic.

Si vous ne trouvez pas votre problème répertorié ici, vous pouvez toujours contacter [le support New Relic](https://support.newrelic.com).

## Certains événements ont expiré et peuvent ne plus apparaître dans la chronologie ou le suivi des événements [#aged-out-events]

Session Replay a une rétention des données plus longue que les événements d&apos;exception gérés. Vous pouvez trouver un replay avec d&apos;autres événements mobiles mais aucun événement d&apos;exception gérée dans le suivi des événements ou représenté comme un marqueur d&apos;erreur. C&apos;est parce que ces événements ont dépassé la rétention, ils n&apos;apparaîtront donc pas dans un replay.
