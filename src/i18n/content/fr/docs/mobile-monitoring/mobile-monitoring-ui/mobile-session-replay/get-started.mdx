Session Replay élimine les approximations lors du dépannage en fournissant une relecture vidéo de ce que les utilisateurs finaux ont vu sur leur téléphone ou leur tablette, synchronisée avec toutes les données télémétriques de cette session. Cela signifie que vous pouvez évaluer rapidement l&apos;impact réel des erreurs sur vos utilisateurs, identifier les goulots d’étranglement des performances et les points de friction de l&apos;UX, et résoudre les problèmes plus rapidement sans dépendre des rapports de bugs ou d&apos;une reproduction locale.

* [Configurer les paramètres de confidentialité](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/configure-privacy-settings): personnaliser les options de masquage et de protection des données.
