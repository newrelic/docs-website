Session Replay possède son propre ensemble d&apos;autorisations, ce qui vous permet de contrôler précisément qui peut visionner les rediffusions et de mettre à jour ses paramètres. Tout utilisateur disposant d&apos;un rôle standard peut consulter les rediffusions, mais seuls les administrateurs de produit peuvent modifier les paramètres Session Replay. Des rôles personnalisés peuvent être créés pour un contrôle plus précis des autorisations des utilisateurs.

## Aperçu des autorisations [#permission-overview]

Les autorisations des utilisateurs sont gérées dans l&apos;interface utilisateur de gestion des utilisateurs, que l&apos;on trouve dans **Administration &gt; Access management &gt; Roles**.

Pour consulter ou modifier les paramètres de Session Replay sur la page Paramètres de l&apos;application, l&apos;utilisateur a besoin à la fois des permissions Session Replay et Paramètres de l&apos;application.
