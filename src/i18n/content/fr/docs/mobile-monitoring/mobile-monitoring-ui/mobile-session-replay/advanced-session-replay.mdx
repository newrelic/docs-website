## Enregistrer manuellement les replays [#manual-recording]

Si nécessaire, vous pouvez forcer de manière programmatique le démarrage ou l&apos;arrêt de l&apos;enregistrement d&apos;un Session Replay à l&apos;aide de méthodes d&apos;API. Quelques cas d&apos;utilisation :

* Autoriser l&apos;utilisateur à accepter l&apos;enregistrement interaction . Par exemple, si vous avez une fenêtre contextuelle demandant « Autoriser l&apos;enregistrement de cette session à des fins d&apos;analyse des performances ? » et un utilisateur consent :

  * Appelez `newrelic.recordReplay()` pour lancer l&apos;enregistrement.
  * Une fois la session terminée ou le consentement retiré, utilisez `newrelic.pauseReplay()` pour arrêter la capture de données.

* Enregistrez des sessions en fonction de critères spécifiques, tels que :

  * Démarrez l&apos;enregistrement uniquement pour les sessions sur des flux spécifiques tels qu&apos;un flux de paiement de produit.
  * Arrêtez l&apos;enregistrement pour les pages sensibles telles que `/billing` ou `/health-records`.

Ces API sont sorties dans Android v7.7.0 et iOS v7.6.1 ou version ultérieure.

Consultez la documentation de l&apos;API pour plus de détails :

* [`newrelic.recordReplay`](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/record-replay)
* [`newrelic.pauseReplay`](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/pause-replay)

## Utilisez NerdGraph pour configurer les paramètres de replay [#nerdgraph]

Vous pouvez configurer les paramètres de Session Replay à l&apos;aide de l&apos;API NerdGraph au lieu de l&apos;UI. Cela est utile si vous devez effectuer des modifications en masse, comme l&apos;application de taux d&apos;échantillonnage à plusieurs entités mobiles.

Voici un exemple de mutation pour mettre à jour les paramètres de Session Replay pour une application mobile :

```graphql
mutation {
  agentApplicationSettingsUpdate(
    guid: ""
    settings: {
      mobileSettings: {
        mobileSessionReplay: {
          customMaskingRules: "[{ type: "mask", operator: "equals", name: "creditCardNumber", identifier: "class" }]"
          enabled: true
          errorSamplingRate: 100
          maskAllImages: false
          maskAllUserTouches: true
          maskApplicationText: false
          maskUserInputText: true
          mode: CUSTOM
          samplingRate: 50
        }
      }
    }
  ) {
    mobileSettings {
      mobileSessionReplay {
        customMaskingRules
        enabled
        errorSamplingRate
        maskAllImages
        maskAllUserTouches
        maskApplicationText
        maskUserInputText
        mode
        samplingRate
      }
    }
  }
}
```
