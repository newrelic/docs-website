      ## Syntaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAllAttributes ()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.removeAllAttributes()
      ```

      ## Description [#description]

      Cette méthode supprime tous les attributs d&apos;une session. Si vous souhaitez supprimer un seul attribut de session, voir [Supprimer un attribut](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes).

      ## Valeurs de retour [#return-values]

      Renvoie `true` si l&apos;enregistrement a réussi et `false` dans le cas contraire.

      ## Exemple [#example]

      ### Java [#java]

      ```java
      boolean attributesRemoved = NewRelic.removeAllAttributes();
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      val attributesRemoved = NewRelic.removeAllAttributes()
      ```
    </TabsPageItem>
