      ## Syntaxe [#syntax]

      ```kotlin

      ```

      ## Description [#description]

      Cette API offre la possibilité d&apos;arrêter l&apos;agent dans le cycle de vie actuel de l&apos;application pendant l&apos;exécution. Une fois l’agent arrêté, il ne peut pas être redémarré dans le cycle de vie actuel de l’application.

      ## Exemple [#example]

      ### Java [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```
    </TabsPageItem>
