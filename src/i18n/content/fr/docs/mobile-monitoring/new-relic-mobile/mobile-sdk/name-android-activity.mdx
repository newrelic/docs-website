## Syntaxe [#syntax]

### Java [#java]

```java
NewRelic.withApplicationToken(apptoken).withLaunchActivityName().start(this.getApplication());
```

### Kotlin [#kotlin]

```kotlin
NewRelic.withLaunchActivityName(className: String?)
```

Définissez le nom de l’activité cible pour l’heure de lancement de l’application afin de calculer le temps froid et le temps chaud.

## Description [#description]

Par défaut, l&apos;agent Android New Relic calcule le temps froid et le temps chaud lorsque l&apos;activité par défaut est chargée. Cette API vous permet de personnaliser le nom de l&apos;activité de lancement cible lors du calcul du temps à froid et du temps à chaud.

## Exemples [#examples]

### Définir le nom de l&apos;activité

Voici un exemple de définition du nom de l&apos;activité :

### Java [#java]

```java
NewRelic.withLaunchActivityName($activityName)
```

### Kotlin [#kotlin]

```kotlin
NewRelic.withApplicationToken("apptoken").withLaunchActivityName($activityName).start(applicationContext)
```
