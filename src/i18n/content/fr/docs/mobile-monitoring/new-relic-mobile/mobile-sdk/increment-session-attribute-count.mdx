      ## Syntaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.incrementAttribute(string $name[, float $value])
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.incrementAttribute(name: String!, value : Double)
      ```

      ## Description [#description]

      Incrémente le nombre d&apos;un attribut. Écrase sa valeur et son type précédents à chaque fois qu&apos;il est appelé.

      Lorsqu&apos;elle est transmise uniquement avec une valeur de nom, cette méthode incrémente le nombre de l&apos;attribut spécifié de 1. Si l&apos;attribut n&apos;existe pas, il crée l&apos;attribut avec une valeur de 1. Lorsqu&apos;une valeur flottante facultative lui est transmise, elle incrémente le nombre de l&apos;attribut spécifié de la valeur flottante.
