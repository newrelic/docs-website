      ## Syntaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.crashNow(string $message)
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.crashNow(message:String!)
      ```

      ## Description [#description]

      Lève une exception de temps d&apos;exécution de démonstration nommée java.lang.RuntimeException pour tester le rapport de plantage de New Relic.

      Avec cette méthode, vous pouvez lancer une exception de temps d&apos;exécution de démonstration nommée `java.lang.RuntimeException` avec un message. Ce crash apparaîtra dans la [page Analyse des crashs](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes).
