      ## Syntaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAttribute(string $name)
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.removeAttribute(name: String?)
      ```

      ## Description [#description]

      Cette méthode supprime un attribut spécifié par la chaîne de nom. Si vous souhaitez supprimer tous les attributs personnalisés, voir [Supprimer tous les attributs](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes).

      ## Paramètres [#parameters]

      <table>
        <thead>
          <tr>
            <th width={200}>
              paramètres
            </th>

            <th width={200}>
              Type
            </th>

            <th>
              Description
            </th>
          </tr>
        </thead>

        <tbody>
          <tr>
            <td>
              `$name`
            </td>

            <td>
              `string`
            </td>

            <td>
              Requis. Le nom de l&apos;attribut que vous souhaitez supprimer.
            </td>
          </tr>
        </tbody>
      </table>

      ## Valeurs de retour [#return-values]

      Renvoie `true` si l&apos;enregistrement a réussi et `false` dans le cas contraire.

      ## Exemple [#example]

      Voici un exemple de suppression d&apos;un attribut nommé `rate`:

      ### Java [#java]

      ```java
      boolean attributeRemoved = NewRelic.removeAttribute("rate");
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      val attributeRemoved = NewRelic.removeAttribute("rate")
      ```
