New Relicのテレメトリデータプラットフォームは、すべてのオペレーション上のデータについて真実を伝えるシングルソースで、これによりあらゆる質問をミリ秒単位で行い回答できるようになります。

世界で最も強力な、管理されたオープンで一元化されたテレメトリープラットフォームを使用して、あらゆるソースからのすべてのメトリクスやイベント、ログ、トレースを収集して詳しく調べ、アラートを送信します。オープンソースツールの自動的なインテグレーションにより簡単にセットアップが行えるため、追加のモニタリングシステムやデータストアのホストや運用、管理のコストと複雑さが減少します。テレメトリデータを1つの場所に置くことで、未知の事柄を確信を持って調査できます。

## New Relicの価値 [#value]

テレメトリデータプラットフォーム：

* スタック全体について、柔軟で拡張可能な高性能のデータプラットフォームを提供します。
* データサイロがなくなり、平均検知時間（MTTD）と平均解決時間（MTTR）が短縮され、1か所で[メトリックやイベント、ログ、トレース](/docs/using-new-relic/data/understand-data/new-relic-data-types)を収集することでツールの統合が可能になります。
* 追加のインフラストラクチャやハードウェア、専門家が設定や運用、追加システムの維持で不要な、エンタープライズグレードのSaaSプラットフォームにより、価値実現までの時間を短縮します。
* [Grafana](/docs/query-your-data/explore-query-data/query-builder/use-advanced-nrql-mode-specify-data)での[Prometheus](/docs/query-your-data/explore-query-data/dashboards/introduction-new-relic-one-dashboards)データの可視化のサポートを含む、[簡単に作成できるチャート](/docs/integrations/prometheus-integrations/install-configure/set-your-prometheus-remote-write-integration)や[ダッシュボード](/docs/integrations/grafana-integrations/get-started/grafana-support-prometheus-promql)により、オペレーション上のデータを詳しく調べることができます。
* New Relicのクラス最高のエージェントに加え、PrometheusやTelegraf、FluentD、Logstashなどのオープンソースツールからデータを取り込むときに350以上の自動インテグレーションを提供します。
* [New Relic Oneアプリケーションを構築](/docs/new-relic-one/use-new-relic-one/build-new-relic-one/new-relic-one-build-your-own-custom-new-relic-one-application)し、ビジネスKPIやカスタマーエンゲージメントなど独自のビジネスニーズとシステムのパフォーマンスを繋げることができます。

## 機能

New Relicの機能は、次のいくつかの分野にまとめられています:

* **データ**：すべてのシステムのテレメトリーデータ（[メトリクスやイベント、ログ、トレース](/docs/using-new-relic/data/understand-data/new-relic-data-types)）は1つのプラットフォームに接続され、サイロがなくなり効率が向上します。[New Relic One](https://one.newrelic.com)の**データのブラウズ**ボタンの単一の基準点から、データに簡単にアクセスできます。
* **分析**：超高速のレスポンスタイムで収集した[データのクエリ](/docs/using-new-relic/data/understand-data/query-new-relic-data)を行い、異なるデータタイプについて一般的なクエリパターンを使用して、質問が行われたらすぐに回答を取得します。
* **ダッシュボード**：ソフトウェア開発チームとITチームがアップタイムとパフォーマンスを確保し、業務を効率化し、発売までの時間を短縮できるように、[データを可視化](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards)します。
* **アラート**：お客様にとって大切なメトリクスや閾値に基づき、[リアルタイムの通知](/docs/alerts/new-relic-alerts/getting-started/new-relic-alerts-concepts-workflow)で問題を見つけ出します。
* **プログラマビリティ**：[カスタムのNew Relic Oneアプリケーションを構築](/docs/new-relic-one/use-new-relic-one/build-new-relic-one/new-relic-one-build-your-own-custom-new-relic-one-application)し、ビジネスKPIやカスタマーエンゲージメントなど独自のビジネスニーズとシステムのパフォーマンスを繋げます。

開始の方法を含むNew Relic全般の詳細については、[New Relicの概要](/docs/using-new-relic/welcome-new-relic/get-started/introduction-new-relic)をご覧ください。
