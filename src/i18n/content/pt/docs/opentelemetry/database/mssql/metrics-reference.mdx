Monitore o desempenho do seu banco de dados SQL Server com métricas abrangentes coletadas pelo coletor NRDOT. As 7 métricas a seguir são coletadas automaticamente para a funcionalidade da interface do New Relic. Elas são **habilitadas por padrão** e funcionam em qualquer plataforma via uma conexão direta com o SQL Server.
