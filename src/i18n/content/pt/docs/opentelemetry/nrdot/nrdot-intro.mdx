A New Relic Distribution do OpenTelemetry (NRODT) refere-se à coleção de distribuições suportadas pelo New Relic de componentes do projeto OpenTelemetry. Os componentes do NRDOT são projetados para funcionar perfeitamente com os produtos e serviços da New Relic e, quando disponíveis, são a maneira recomendada de integrar o OpenTelemetry com a New Relic.

<Callout variant="tip">
  Atualmente, o NRDOT inclui distribuições pré-criadas do OpenTelemetry [Collector](/docs/opentelemetry/nrdot/nrdot-collector/). Para APIs de linguagem e SDKs, consulte nossa documentação sobre como começar a usar [o monitoramento do APM](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-intro/).
</Callout>
