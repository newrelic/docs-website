Se você for um [usuário completo da plataforma](/docs/accounts/accounts-billing/new-relic-one-pricing-users/users-roles#user-type), teoricamente terá acesso total a todos os recursos do New Relic, mas se você pode usar todos esses recursos depende da função atribuída a você em sua organização New Relic.

Para detalhes sobre como funcionam as permissões do New Relic, consulte [Conceitos importantes de gerenciamento de usuários e acesso](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts). Para ver as capacidades granulares relacionadas ao Synthetics, consulte [Capacidades](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/#synthetics).
