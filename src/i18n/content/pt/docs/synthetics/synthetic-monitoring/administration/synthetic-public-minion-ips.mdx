New Relic usa minion público para executar seu [monitor Sintético](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors). Esses lacaios estão implantados em diferentes data centers ao redor do mundo e são responsáveis pelo funcionamento do seu monitor.

Por isso, certifique-se de que seu firewall permite solicitações de rede Sintético. Os endereços IP minion estão disponíveis publicamente no formato JSON, para que você possa lê-los e analisá-los facilmente.

<Callout variant="important">
  Certifique-se de que sua lista de permissões ou configuração relacionada inclua todos os intervalos nas [listas de intervalos de IP](#dynamic-ip-addresses) , pois as solicitações podem ser enviadas de qualquer um dos ambientes.
</Callout>

<DNT>**Optional steps**</DNT>:

* Você pode automatizar suas regras para sua lista de permissões com base nesses arquivos.
* Você pode identificar monitor solicitações [Sintético usando nossos](/docs/synthetics/new-relic-synthetics/administration/identify-synthetics-requests-your-app) cabeçalhos padrão ou adicionando um cabeçalho personalizado ao seu [ping ou de navegador simples monitor](/docs/synthetics/synthetic-monitoring/using-monitors/add-edit-monitors/#simple),[monitorcom](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetic-scripted-browser-reference-monitor-versions-chrome-100#structure)[ API monitor ](/docs/synthetics/synthetic-monitoring/scripting-monitors/write-synthetic-api-tests/#request-options)script ou de navegador com script .

## Endereços IP não são dados pessoais [#not-personal-data]

Os minions são implantados nos servidores e os agentes são ativados usando credenciais não pessoais. Os endereços IP associados ao minion em execução em servidores não são dados pessoais de acordo com as leis de proteção de dados e privacidade. Para mais informações, consulte a [documentação de segurança do Sintético](/docs/synthetics/new-relic-synthetics/getting-started/security-new-relic-synthetics).

## Listagens JSON diárias para endereços IP [#dynamic-ip-addresses]

Os endereços IP dos locais divulgados estão sujeitos a alterações. Se for necessária uma alteração, tentaremos notificar proativamente os clientes antes de qualquer alteração por e-mail. Você também pode verificar o [Fórum de Suporte](https://discuss.newrelic.com/) para atualizações. Os intervalos de IP listados são reservados para uso da New Relic e não podem ser usados por mais ninguém.
