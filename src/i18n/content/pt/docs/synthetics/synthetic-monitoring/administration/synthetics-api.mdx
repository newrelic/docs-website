Temos duas opções para gerenciar seu monitor New Relic Synthetics via API:

* NerdGraph (recomendado): esta é a forma principal e recomendada. Veja [nosso tutorial do NerdGraph Sintético](/docs/apis/nerdgraph/examples/nerdgraph-synthetics-tutorial).
* API REST: alguns clientes preferem usar nossa API REST mais antiga. Veja [API REST Sintético](/docs/apis/synthetics-rest-api/monitor-examples/manage-synthetics-monitors-rest-api)

Para uma visão geral de todas as nossas API disponíveis, consulte [Introdução à API](/docs/apis/getting-started/introduction-new-relic-apis).
