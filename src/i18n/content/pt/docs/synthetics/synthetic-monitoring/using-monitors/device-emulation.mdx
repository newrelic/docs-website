A emulação de dispositivo é uma configuração opcional disponível para script de browser simples e de monitoramento. Esta configuração simula a janela de visualização e os recursos de um dispositivo móvel ou tablet usando os recursos de emulação do Chrome DevTools.

A emulação de dispositivo permite testar um aplicativo baseado em browser usando um dispositivo móvel ou tablet genérico simulado. Tenha em mente:

* Esta configuração não altera a rede, o agente do usuário ou o tipo de dispositivo usado para executar o monitor Sintético.
* A emulação de dispositivo não permite instalar e testar aplicativos móveis.

## Requisitos [#requirements]

* Somente tempos de execução do Chrome 100+
* Localização privada deve utilizar [job manager](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) versão 225 ou superior e Sintético-node-browser-runtime versão 1.1.98 ou mais alto
