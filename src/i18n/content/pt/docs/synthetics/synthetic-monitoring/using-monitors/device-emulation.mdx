A emulação de dispositivo é uma configuração opcional disponível para scripts simples e de monitoramento do navegador. Esta configuração simula a janela de visualização e os recursos de um dispositivo móvel ou tablet usando os recursos de emulação do DevTools no Chrome e no Firefox.

A emulação de dispositivo permite testar um aplicativo baseado em browser usando um dispositivo móvel ou tablet genérico simulado. Tenha em mente:

* Esta configuração não altera a rede, o agente do usuário ou o tipo de dispositivo usado para executar o monitor Sintético.
* A emulação de dispositivo não permite instalar e testar aplicativos móveis.

## Requisitos [#requirements]

* Somente tempos de execução não legado (sem suporte ao Chrome 72)
* Chrome: localização privada deve usar [job manager](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) versão 225 ou superior e Sintético-node-navegador-runtime versão 1.1.98 ou superior
* Firefox: localização privada deve usar [gerenciador de tarefas](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) versão 400 ou superior e Sintético-node-navegador-runtime versão 3.0.0 ou superior
