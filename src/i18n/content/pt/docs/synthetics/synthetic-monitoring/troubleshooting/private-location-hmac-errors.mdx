## Problema

Um de seus minions privados está falhando nas verificações e relatando um destes erros:

```
HMAC MISMATCH ERROR: HMAC mismatch between saved HMAC and calculated HMAC at job run time
```

```
HMAC MISMATCH ERROR: HMAC expected but missing from Job metadata
```

Esses erros indicam um erro na [execução verificada do script](/docs/synthetics/new-relic-synthetics/private-locations/verified-script-execution-private-locations).

## Solução

Você precisará inserir a senha correta para o local onde a verificação está falhando.

Para inserir a senha:

1. Em

   <DNT>
     **[one.newrelic.com > Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)**
   </DNT>

   \> selecione o monitor com falha.

2. Vá para

   <DNT>
     **Settings > Script**
   </DNT>

   .

3. Nos campos de senha do local, envie a senha correta.

<Callout variant="tip">
  New Relic não armazena senhas de localização privada. Caso você não saiba a senha, procure a pessoa da sua equipe que configurou a localização privada. Caso não seja possível recuperar a senha, sua única opção é criar uma nova localização privada.
</Callout>
