<Callout variant="important">
  Execute o IAST apenas com implantação não produtiva. O IAST testa invocando solicitações HTTP com uma carga útil de exploração. Você deve usar o IAST em ambiente não produção apenas com dados simulados para evitar a corrupção de dados e a introdução de explorações em seu código ativo. Além disso, ao executar o IAST em um novo código em pré-produção, você detecta vulnerabilidades potenciais antes que elas entrem em operação.
</Callout>

Para instalar o New Relic IAST:

<Steps>
  <Step>
    Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) > All capabilities > IAST**</DNT> e clique em <DNT>**Set up IAST with applications**</DNT>.

    <img
      title="Set up IAST with applications"
      alt="Set up IAST with applications"
      src="/images/iast_screenshot-full_install.webp"
    />
  </Step>

  <Step>
    Confirme que o IAST não está em execução em um ambiente de produção e clique em <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Selecione o idioma do seu aplicativo e conclua as etapas.

    <img
      title="Install New Relic IAST"
      alt="Install New Relic IAST"
      src="/images/iast_screenshot-crop_install.webp"
    />
  </Step>

  <Step>
    Certifique-se de que seu arquivo de configuração `newrelic.yml` esteja atualizado da seguinte forma:

    <CollapserGroup>
      <Collapser
        id="config-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> arquivo de configuração</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
        ```
      </Collapser>

      <Collapser
        id="config-eu-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> arquivo de configuração para a UE</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
            validator_service_url: wss://csec.eu01.nr-data.net
        ```
      </Collapser>

      <Collapser
        id="config-file-fed-example"
        title={<><InlineCode>newrelic.yml</InlineCode> arquivo de configuração para usuário Fed</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
            validator_service_url: wss://csec-gov.nr-data.net
        ```
      </Collapser>
    </CollapserGroup>
  </Step>

  <Step>
    Depois de concluir todas as etapas, reinicie seu aplicativo e gere tráfego na API do aplicativo.
  </Step>

  <Step>
    Clique em <DNT>**See your data**</DNT> para ter uma visão geral do seu aplicativo testado.
  </Step>
</Steps>
