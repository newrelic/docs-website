Isso explica como autenticar sua conta do aplicativo New Relic Mobile e como adicionar ou remover usuários do seu dispositivo móvel.

## Autenticação de usuário [#authentication]

Dependendo da sua conta New Relic, etapas adicionais de instalação ou autenticação podem ser necessárias ao instalar o aplicativo New Relic Mobile .

<table>
  <thead>
    <tr>
      <th width={200}>
        <DNT>
          **New Relic account**
        </DNT>
      </th>

      <th>
        <DNT>
          **Additional requirements**
        </DNT>
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Novo usuário
      </td>

      <td>
        Se você ainda não possui uma conta New Relic:

        1. No browser do seu desktop, [crie uma conta New Relic](https://newrelic.com/signup).

        2. [Instale seu aplicativo](/docs/site/installing-the-agent) com o agente New Relic apropriado.

           Como parte da configuração da nova conta, você receberá um e-mail com um link para redefinição de senha. O link de redefinição de senha expira após 20 minutos para aplicativos móveis.
      </td>
    </tr>

    <tr>
      <td>
        Usuário existente do New Relic
      </td>

      <td>
        Sem requisitos adicionais; seu aplicativo, hosts e principal de transação aparecem automaticamente após você entrar.
      </td>
    </tr>

    <tr>
      <td>
        Usuário com contas de parceiros New Relic
      </td>

      <td>
        Dependendo do parceiro, pode ser necessário concluir um [processo de autenticação](/docs/ios/ios-with-partners) diferente.

        <DNT>**Azure Store users:**</DNT> Devido à profunda integração entre o Azure Storefront e o New Relic, o usuário do Azure Storefront não pode acessar suas contas nos aplicativos New Relic para Android ou iOS.
      </td>
    </tr>

    <tr>
      <td>
        Usuário com contas habilitadas para SAML-SSO
      </td>

      <td>
        Quando você faz login no aplicativo New Relic Mobile , sua sessão é redirecionada automaticamente para o seu browser da web. A partir daí, você pode fazer login em sua conta SAML-SSO da New Relic.

        Se você encontrar algum erro ao usar contas SAML-SSO em seu dispositivo móvel, verifique se consegue fazer login no <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT> com um browser da web para computador.

        * Se não, entre em contato com seu administrador.
        * Se sim, obtenha suporte em [support.newrelic.com](https://support.newrelic.com).
      </td>
    </tr>
  </tbody>
</table>

  <Collapser

    title="Alternar entre contas"
  >
    Para alternar entre contas associadas ao seu nome de usuário:

    1. No menu

       <DNT>
         **Users**
       </DNT>

       , selecione o nome de usuário.

    2. Selecione o nome da conta.
  </Collapser>

  <Collapser
    id="Remove a username"
    title="Remover ou adicionar novamente um nome de usuário"
  >
    Para remover um nome de usuário específico deste dispositivo:

    1. No menu <DNT>**Users**</DNT> , selecione <DNT>**Logout**</DNT>.

    2. Para remover um usuário deste dispositivo, selecione o sinal de menos vermelho do usuário <Icon style={{color: 'red'}} name="fe-minus-circle"/> ícone.

    3. Selecione o ícone <DNT>**Log out**</DNT> do usuário.

       Para adicionar um usuário novamente, faça login com esse nome de usuário novamente.
