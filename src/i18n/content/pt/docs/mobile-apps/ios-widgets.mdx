A chave para o sucesso em DevOps e engenharia de confiabilidade de site reside na redução do tempo médio de resolução (MTTR). widget pode ajudá-lo a fazer isso, mesmo quando você não está na frente do computador.

## Antes de começar a usar o widget [#requirements]

Para configurar [o widget iOS](https://support.apple.com/en-us/HT207122) em seu iPhone ou iPad, certifique-se de estar executando o iOS 14 ou superior e de ter o [aplicativo New Relic Mobile ](install-newrelic-ios-mobile-app)instalado.

Aqui está um exemplo de adição de um widget à tela inicial:

1. Toque e segure uma área ou widget vazio até que os ícones balancem.

2. Toque no ícone <DNT>**+**</DNT> no canto superior esquerdo.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the Add (+) button"
     alt="Screenshot showing the Add (+) button"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-edit-home-screen.webp"
   />

3. Procure o widget <DNT>**New Relic**</DNT> .

4. Deslize para a esquerda ou para a direita para selecionar um dos três tamanhos de widget e toque em <DNT>**Add Widget**</DNT>.

   <Video
     type="wistia"
     id="qyk74p7j56"
     vertical
     width="290px"
   />

5. Toque em <DNT>**Done**</DNT>.

## Adicionar um gráfico a um widget [#add-chart]

Se você ainda não selecionou o gráfico que deseja mostrar no aplicativo New Relic, siga estas etapas:

1. Navegue até o gráfico que você gostaria de ver no widget.

2. Toque no gráfico para mostrar a visualização detalhada.

3. Toque no ícone de reticências <DNT>**...**</DNT> no canto superior direito para mostrar mais opções.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the dropdown options in the ellipsis"
     alt="Screenshot showing the dropdown options in the ellipsis menu"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-add-widget-from-chart.webp"
   />

4. Selecione <DNT>**Add to widgets**</DNT>.

5. Você também pode remover e reorganizar o widget com <DNT>**Manage widgets**</DNT>.

   <Video
     type="wistia"

     vertical
     width="290px"

## Designar gráficos para widget [#designate-charts]

Para selecionar qual gráfico será mostrado para cada widget do iOS:

1. Pressione e segure o widget até que ele balance.

2. Toque em <DNT>**Select chart**</DNT> ou no nome do gráfico.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the option to select a chart"
     alt="Screen showing the option to select a chart"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-configure-widget.webp"
   />

3. Se preferir, você pode ocultar o título e/ou legenda desta tela.
