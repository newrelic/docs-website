Você pode decidir trocar de produtos de monitoramento de dados por vários motivos. Seja qual for o motivo, você pode experimentar o New Relic sem precisar migrar totalmente sua stack do Datadog. Fornecemos uma série de tutoriais para ajudá-lo a começar a monitorar seu sistema com o New Relic, bem como a entender como funciona a observabilidade do New Relic. Abaixo, você encontrará informações sobre como configurar o monitoramento de infraestrutura, aplicativo e log, bem como painel e <InlinePopover type="alerts"/>para ajudá-lo a realmente controlar seus dados.

O único pré-requisito é uma conta New Relic: [cadastre-se gratuitamente](https://newrelic.com/signup) e você estará pronto para começar!

## Use entidade para observabilidade full-stack [#entities]

Na New Relic, usamos o conceito de entidade para nos ajudar a dividir e organizar os dados. Essencialmente, qualquer coisa que reporte dados ou contenha dados da New Relic que possa ser identificado com um ID único é uma entidade. Usamos um poderoso sistema de mapeamento para vincular totalmente as entidades. Para você, isso significa um monitoramento mais granular e transparente de praticamente qualquer tipo de dados de praticamente qualquer fonte. Tudo isso para permitir que você veja exatamente os dados que lhe interessam em qualquer nível, sem sobrecarregar seu monitoramento com coisas que você não quer ou precisa.

## Explorando o painel predefinido [#dash]

Não seria ótimo se você pudesse ver todos os seus dados de observabilidade sem precisar configurar nada? A New Relic oferece interfaces selecionadas para oferecer exatamente isso e muito mais. A partir do momento em que você começar a ingerir dados, você terá acesso a níveis de informações altamente detalhados desde o início, bem como a uma série de recursos fáceis de usar que permitem classificar, localizar, contextualizar e investigar esses dados com apenas um clique. clique ou dois.

Esta série de tutoriais cobrirá essas interfaces selecionadas na forma de painel predefinido para infraestrutura, aplicativo e monitoramento log . Também mostraremos onde encontrar centenas de painéis pré-configurados para nosso outro recurso. Gaste menos tempo configurando suas ferramentas e mais tempo usando-as com o New Relic.
