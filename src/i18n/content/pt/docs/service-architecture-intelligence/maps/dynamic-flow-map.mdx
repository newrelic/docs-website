<DNT>**Dynamic Flow Map**</DNT> é uma ferramenta de visualização que fornece uma visualização de mapa de dados trace agregados. Ele destaca sinais de desempenho relacionados a um serviço específico, chamado nó focal. O mapa usa dados de rastreamento para exibir relacionamentos entre serviços que participam do rastreamento com o nó focal, correlacionando mudanças de desempenho entre entidades a montante e a jusante. Isso ajuda a identificar gargalos de desempenho em aplicativos e serviços, auxiliando a identificar a causa raiz.

## Por que isso importa [#why-it-matters]

<DNT>**Dynamic Flow Map**</DNT> auxilia na redução do tempo médio de resolução, permitindo visualizar e identificar anomalias de desempenho. Ao correlacionar dados de desempenho de rastreamento upstream e downstream, você pode entender como diferentes serviços interagem e afetam o desempenho uns dos outros.

Para se familiarizar com os componentes do mapa e recurso, consulte [<DNT>**Maps**</DNT>](/docs/service-architecture-intelligence/maps/advanced-maps).

<Callout variant="important">
  [<DNT>**Distributed tracing**</DNT>](/docs/apm/distributed-tracing/getting-started/introduction-distributed-tracing) é necessário para que o <DNT>**Dynamic Flow Map**</DNT> funcione e é habilitado por padrão para todos os agentes APM. Se os dados não estiverem visíveis, certifique-se de que <DNT>**Distributed tracing**</DNT> esteja ativo para o nó focal e a entidade associada e verifique se os dados de rastreamento aparecem no New Relic.
</Callout>

## Explore o Mapa de Fluxo Dinâmico [#explore-dynamic-flow-map]

Para acessar o <DNT>**Dynamic Flow Map**</DNT>:

1. Em <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT>, abra qualquer [serviço, infraestrutura ou mapa de equipe](/docs/service-architecture-intelligence/maps/advanced-maps).
2. Para abrir o <DNT>**Dynamic Flow Map**</DNT>, aponte para um nó e clique em <DNT>**View dynamic flow**</DNT>.

Você também pode acessar <DNT>**Dynamic Flow Map**</DNT> de [<DNT>**Transactions 360**</DNT>](/docs/apm/transactions/workload-performance-monitoring/transaction-workloads/). Você pode visualizar o mapa de uma transação específica mostrando os serviços envolvidos e o tempo gasto por cada serviço.

O <DNT>**Dynamic Flow Map**</DNT> mostra relacionamentos entre o nó focal e a entidade circundante. Os serviços upstream são representados com arestas apontando para o nó focal, enquanto os serviços downstream têm arestas apontando para longe dele. Você pode interagir com o mapa para visualizar tendências de desempenho, anomalias e outros detalhes da seguinte forma:

<CollapserGroup>
  <Collapser id="anomalies" title="Analisar anomalia">
    As anomalias são exibidas com bordas e pontos codificados por cores ao lado dos nós de serviço, da seguinte maneira:

    * **Borda azul**: Problema de latência entre os serviços.
    * **Borda vermelha**: Erro de sinal entre os serviços.
    * **Borda listrada em azul e vermelho**: problema de latência e erro de sinal entre os serviços.
    * **Ponto azul ao lado de um nó de serviço**: problema de latência no serviço.
    * **Ponto vermelho ao lado de um nó de serviço**: Erro de sinal no serviço.

    Se um nó tiver pontos de anomalia ao lado, mas a aresta que o conecta ao nó focal não estiver codificada por cores, isso indica que a anomalia entre os nós não está correlacionada. A partir das [tendências de desempenho](#performance-trends), você pode revisar o gráfico de séries temporais métricas para confirmar se a anomalia ocorreu em momentos diferentes para os nós.
  </Collapser>

  <Collapser id="performance-trends" title="Ver tendências de desempenho">
    Em <DNT>**Dynamic Flow Map**</DNT>, para visualizar as tendências de desempenho entre o nó focal e um nó conectado, clique no nó ao redor. As tendências de desempenho entre os nós aparecem no painel direito, mostrando os gráficos de séries temporais métricas.

    Você pode analisar as tendências de desempenho e gerenciar os nós da seguinte maneira:

    * Identifique padrões entre os nós selecionados.
    * Interaja com os gráficos de séries temporais selecionando os nós abaixo deles.
    * Visualize todos os traços entre os nós selecionados.
    * Atribua a propriedade dos nós a equipes específicas.
    * Adicione uma tag aos nós.
  </Collapser>

  <Collapser id="investigate-issues" title="Investigar problemas">
    No <DNT>**Dynamic Flow Map**</DNT>, após identificar um nó com problemas de desempenho correlacionados ao nó focal, você pode investigar o rastreamento entre eles. Para visualizar o rastreamento, no painel direito, clique em **See all traces**. Ao analisar o rastreamento, você pode identificar a causa raiz dos problemas de desempenho e tomar as medidas necessárias para resolvê-los.
  </Collapser>
</CollapserGroup>
