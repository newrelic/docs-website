O <DNT>Repository catalog</DNT> é uma ferramenta desenvolvida para centralizar e agilizar a gestão do repositório dentro de uma organização. Ao fornecer uma visão unificada de todo o repositório, elimina-se a compartimentalização entre as equipes e aprimora-se a colaboração entre elas. Ele permite que desenvolvedores, engenheiros de confiabilidade do site (SREs) e outras partes interessadas identifiquem com eficiência o repositório existente, entendam sua finalidade e determinem a propriedade ou status. Essa abordagem centralizada simplifica a descoberta, o acesso e o gerenciamento do repositório de código, resultando em maior colaboração, responsabilidade e produtividade em toda a organização.

## Recurso chave [#features]

* Descoberta centralizada de repositórios: Identifique rapidamente repositórios relevantes para serviços, projetos ou <DNT>[Teams](/docs/service-architecture-intelligence/teams/teams)</DNT>, minimizando o tempo gasto em buscas ou consultas com colegas.

* Navegação de serviços e repositórios: navegue facilmente do aplicativo ou serviços até seu repositório definidor e vice-versa, auxiliando na compreensão da dependência e infraestrutura do código.

* Propriedade e Responsabilidade: Identifique facilmente os responsáveis pela loja, apoiando a consultoria direta e aumentando a responsabilidade dentro das equipes.

* Insights do repositório: Acesse métricas detalhadas sobre a atividade do repositório, como open pull request e histórico de implantação, para identificar projetos negligenciados e otimizar a alocação de recursos.

* Gestão e Operações: Executar operações em massa, como atribuição de etiquetas ou equipes, para gerenciar o estoque de forma eficiente e manter os padrões organizacionais.

## Acesse o catálogo do repositório [#access-repository]

**Pré-requisito:**

* Ative os catálogos na sua conta New Relic. Para obter mais informações, consulte [Ativar os catálogos](/docs/service-architecture-intelligence/catalogs/catalogs/#enable-catalogs).

**Para acessar o inventário do catálogo do repositório:**

1. Entre na sua plataforma New Relic.

2. Vá para o **painel de navegação esquerdo &gt; Catalogs &gt; Repository catalog**.

### Adicionar novos dados [#add-new-data]

Para adicionar dados adicionais ao seu <DNT>Repository catalog</DNT>, clique no botão <DNT>**+ Add GitHub repositories**</DNT> no canto superior direito. Siga as instruções na tela para adicionar dados ao seu seguro. Para obter informações adicionais sobre como adicionar repositório, consulte [GitHub integração](/docs/service-architecture-intelligence/github-integration).

### Limitação [#limitations]

* A ordenação de dados não é suportada atualmente na página <DNT>Repository catalog</DNT>.
