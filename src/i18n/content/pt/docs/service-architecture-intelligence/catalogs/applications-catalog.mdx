O catálogo de aplicativos oferece uma visão em painel único para aprimorar a observabilidade e o gerenciamento da entidade em sua organização. A visão unificada das entidades Serviços - APM, Serviços - OpenTelemetry, Browser e Dispositivos Móveis facilita a tomada de decisões bem informadas, a otimização das operações e a melhoria da colaboração entre as equipes.

### Entidade apoiada [#supported-entities]

A experiência atual do catálogo de aplicativos New Relic oferece suporte às seguintes entidades:

* `Services - APM`
* `Services - OpenTelemetry`
* `Browser`
* `Mobile`

## Acesse o catálogo de aplicativos [#access-catalog]

**Pré-requisito:**

* Ative os catálogos na sua conta New Relic. Para obter mais informações, consulte [Ativar os catálogos](/docs/service-architecture-intelligence/catalogs/catalogs/#enable-catalogs).

**Para acessar o inventário do catálogo do aplicativo:**

1. Entre na sua plataforma New Relic.

2. Vá para o **painel de navegação esquerdo &gt; Catalogs &gt; Applications catalog**.

   /\* &lt;img title=&quot;Catálogo&quot; alt=&quot;Catálogo&quot; src=&quot;/images/new-relic-application-catalog.webp&quot; /&gt; \*/

## Gerencie e organize seu catálogo [#manage-organize]

O catálogo fornece ferramentas flexíveis para ordenar, agrupar e filtrar seus dados.

### Agrupar dados [#group-data]

Para organizar sua lista em seções lógicas, use o menu suspenso <DNT>**Group by**</DNT> no canto superior direito da tabela. Você pode agrupar entidades por:

* <DNT>**Entity type**</DNT>: Separe serviços de APM de entidades de Browser ou Mobile.
* <DNT>**Environment**</DNT>: Diferencie entre ambientes. Por exemplo, produção versus staging.
* <DNT>**Team**</DNT>: Visualizar serviços pelo grupo de propriedade responsável.
* <DNT>**Language**</DNT>: Organize pela linguagem de programação usada.
* <DNT>**AI asset**</DNT>: Agrupar por bibliotecas de IA detectadas, como OpenAI ou Bedrock.

### Personalizar colunas visíveis [#customize-columns]

Você pode controlar quais pontos de dados são exibidos na tabela para reduzir o ruído e focar em métricas específicas. Use o ícone <DNT>**Column Selection**</DNT> à direita do menu <DNT>**Group by**</DNT> para exibir ou ocultar atributos como <DNT>**Account**</DNT>, <DNT>**Last deployment**</DNT>, <DNT>**AI assets**</DNT> ou golden metrics, como tempo de resposta e throughput.

### Filtrar dados [#filter-data]

Use a barra de filtros na parte superior para refinar suas entidades:

* **Filtros rápidos**: Use os botões predefinidos para filtrar por tipo de entidade, ativo de IA ou equipe.
* **Regras de filtro personalizadas**: Clique no ícone <DNT>**+**</DNT> para adicionar atributos específicos. Você pode definir regras usando operadores para filtrar por um valor de atributo específico. Por exemplo, filtre entidades onde <DNT>Language</DNT> `=` `go` para obter todos os serviços Go.
* **Pesquisa global**: Use a barra <DNT>**Search by entity name**</DNT> para encontrar serviços específicos instantaneamente.

### Adicionar novos dados [#add-new-data]

Para adicionar dados adicionais ao catálogo do seu aplicativo, clique no botão <DNT>**+ Add data**</DNT> no canto superior direito. Siga as instruções na tela para adicionar dados ao catálogo do seu aplicativo. Para obter mais informações sobre integração suportada New Relic, consulte [integração](https://newrelic.com/instant-observability).
