A New Relic fornece um portal da web para que os parceiros acessem facilmente recursos valiosos fornecidos pelo Programa de Parceiros da New Relic, como conteúdo de capacitação de vendas para parceiros, treinamento, materiais de marketing, etc.

Se você é um parceiro e ainda não tem acesso a isso, entre em contato com seu gerente de vendas de parceiros ou envie um e-mail para Partnersales @ newrelic.com.

Para acessar o Portal de Parceiros: No seu Browser, digite (ou marque) a seguinte URL e log com seu nome de usuário e senha.

```
https://partners.newrelic-external.com/English/
```
