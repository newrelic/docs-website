Conecte a cobrança do Google Cloud Platform ao Cloud Cost Intelligence para monitorar e otimizar custos. A integração usa a autorização baseada em conta de serviço para acessar com segurança seus dados de cobrança e preços do GCP. A New Relic cria uma conta de serviço dedicada com acesso somente leitura aos seus conjuntos de dados do BigQuery.

## Antes de você começar

Você precisa:

* **Projeto GCP conectado ao monitoramento de infraestrutura New Relic** - Consulte [Conectar serviços do Google Cloud Platform ao New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)

* **Conta de faturamento GCP ativa**

* **Conjunto de dados do BigQuery** com:

  * Tabela de exportação de faturamento
  * Tabela de exportação de preços da nuvem

* **Acesso ao Google Cloud Console**

## Instruções de configuração

<Steps>
  <Step>
    ### Selecionar conta New Relic [#select-nr-account]

    1. Vá para <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT>.
    2. Selecione sua conta e clique em **Continue**.
  </Step>

  <Step>
    ### Fornecer detalhes da conexão GCP [#gcp-connection-details]

    1. Insira suas informações de faturamento e conjunto de dados do GCP:

       * **Connection name** - Nome descritivo
       * **Project ID** - ID do seu projeto GCP
       * **Dataset name** - Nome do seu conjunto de dados do BigQuery
       * **Billing table name** - Nome da sua tabela de faturamento
       * **Pricing table name** - Nome da sua tabela de preços

    2. Clique em **Continue**.
  </Step>

  <Step>
    ### Autorizar acesso [#gcp-authorize-access]

    1. Copie o ID da conta de serviço exibido.
    2. Abra o **Google Cloud Console** e conceda a função **BigQuery DataViewer** à conta de serviço para suas tabelas de cobrança do BigQuery.
    3. Retorne e clique em **Continue**.
  </Step>

  <Step>
    ### Criar conexão [#gcp-create-connection]

    Clique em **Create connection** para verificar a configuração e estabelecer a integração.
  </Step>

  <Step>
    ### Ver seus dados [#gcp-view-data]

    Acesse [**Intelligent summary (Resumo inteligente)**](/docs/cci/financial-intelligence/intelligent-summary) para visualizar seus dados de custo do GCP.

    <Callout variant="important">
      O processamento pela primeira vez pode levar até 12 horas.
    </Callout>
  </Step>
</Steps>
