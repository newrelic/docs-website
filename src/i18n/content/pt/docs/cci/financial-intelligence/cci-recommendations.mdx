O Cloud Cost Intelligence fornece recomendações de otimização de custos de duas fontes para ajudá-lo a identificar e agir em oportunidades em toda a sua infraestrutura AWS. Ele analisa seus padrões de uso de recursos usando o AWS Trusted Advisor e nosso Native Intelligence Engine proprietário.

<Callout variant="important">
  Recomendações de otimização de custos estão atualmente disponíveis para AWS.
</Callout>

Use as recomendações para:

* Identifique recursos superdimensionados e subutilizados e oportunidades de compromisso
* Visualize as economias potenciais com projeções mensais e anuais
* Track and prioritize optimization actions (<DNT>**acknowledge**</DNT>, <DNT>**dismiss**</DNT>, or <DNT>**mark as pending**</DNT>)

## Fontes de recomendação

### AWS Trusted Advisor

O AWS Trusted Advisor fornece recomendações de otimização de custos diretamente da AWS.

#### Pré-requisitos

To use <DNT>**Trusted Advisor**</DNT> recommendations with Cloud Cost Intelligence, set up and configure access to AWS Trusted Advisor cost insights in your AWS account.

#### Como navegar

Once configured, <DNT>**Trusted Advisor**</DNT> recommendations appear in the <DNT>**Cloud Cost Intelligence Recommendations**</DNT> page alongside other recommendations.

<Callout variant="important">
  Cloud Cost Intelligence verifies <DNT>**Trusted Advisor**</DNT> status during the initial AWS account integration. When <DNT>**Trusted Advisor**</DNT> is inactive, Cloud Cost Intelligence skips this integration component.

  If <DNT>**Trusted Advisor**</DNT> is enabled later, Cloud Cost Intelligence will automatically identify the update and synchronize data within seven days of enablement.
</Callout>

### Mecanismo de otimização nativo CCI

O mecanismo de otimização nativo da CCI analisa a utilização de recursos e o histórico de uso para identificar oportunidades de dimensionamento correto de instâncias e reduzir custos por meio de compromissos do AWS Savings Plans.

#### Como funciona

O mecanismo analisa os dados de uso em duas áreas:

* <DNT>**Right-sizing**</DNT>: Checks CPU and memory usage, compares current instance types to performance needs, and identifies instances that can be downsized without affecting performance.
* <DNT>**Savings plans**</DNT>: Reviews usage patterns to find consistent workloads and calculates potential savings from AWS Savings Plans commitments.

#### Quais recomendações ele fornece

O mecanismo fornece recomendações para reduzir custos:

* <DNT>**Instance type changes**</DNT>: Specific instance modifications (for example, <DNT>**Migrate from im4gn.16xlarge to t2.micro**</DNT>) with current configuration, CPU and memory usage, and number of affected instances
* <DNT>**Savings plan commitments**</DNT>: Recommendations to commit to consistent usage levels for discounted pricing
* <DNT>**Savings projections**</DNT>: Monthly and annual cost savings for each recommendation

## Ver recomendações

Para acessar suas recomendações de otimização de custos:

1. Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Cloud Cost Intelligence**</DNT>
2. Clique em <DNT>**Recommendations**</DNT> na navegação à esquerda.
