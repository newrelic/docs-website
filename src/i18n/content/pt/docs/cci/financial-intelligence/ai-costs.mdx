A página <DNT>**AI Costs**</DNT> oferece visibilidade de custos para serviços de IA em seus provedores de cloud. Ele exibe os custos no nível do token divididos por serviço, modelo e tipo de token, oferecendo uma visão unificada dos gastos com IA em ambientes multinuvem e múltiplos modelos LLM.

## Acessar Custos de IA

1. Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Cloud Cost Intelligence**</DNT>.
2. Clique em <DNT>**AI Costs**</DNT> na navegação à esquerda.

## Filtros de provedor de nuvem

Para filtrar os dados de custo de IA por provedor de cloud, selecione uma opção nas abas do provedor na parte superior da página:

* <DNT>**All**</DNT>: Todos os provedores de cloud
* <DNT>**AWS**</DNT>: apenas Amazon Web Services
* <DNT>**Azure**</DNT>: apenas Microsoft Azure
* <DNT>**GCP**</DNT>: apenas Google Cloud Platform

## Principais gastos com IA

A seção <DNT>**AI Top Spends**</DNT> apresenta suas áreas de maior custo de IA em quatro dimensões. Cada widget mostra o total do período selecionado, o valor de comparação do período anterior, a variação percentual e um detalhamento dos três principais itens.

### Serviços

Exibe os custos totais de IA agrupados por serviço de IA de cloud (por exemplo, AmazonBedrock, Azure AI Services, AmazonSageMaker). Use isso para identificar quais serviços geram mais gastos em seus provedores de cloud.

### Modelos

Exibe os custos totais de IA agrupados por modelo de LLM (por exemplo, Claude Opus 4.6, Claude Sonnet 4.6). Use isso para comparar o custo entre versões de modelo e provedores.

### Custo do token

Exibe os gastos totais com IA divididos por tipo de token:

* <DNT>**Cache Write Token**</DNT>: Custo para gravar tokens no cache
* <DNT>**Input Token**</DNT>: Custo dos tokens de entrada processados pelo modelo
* <DNT>**Cache Read Token**</DNT>: Custo de leitura de tokens do cache

Use isso para entender quais operações de token contribuem mais para seus custos de IA.

### Uso de token

Exibe o consumo total de tokens (em bilhões) dividido por tipo de token. Use isso em conjunto com o Custo de token para identificar padrões de alto volume e baixo custo versus padrões de baixo volume e alto custo que podem indicar oportunidades de otimização.

<Callout variant="tip">
  Compare os widgets <DNT>**Token Cost**</DNT> e <DNT>**Token Usage**</DNT> em conjunto para entender a eficiência de custo por token em diferentes tipos de tokens e modelos.
</Callout>

## Tendências de Custos

A seção <DNT>**Cost Trends**</DNT> fornece análise baseada em tempo dos seus custos de IA com gráficos e controles interativos.

### Controles do gráfico

* <DNT>**View options**</DNT>: Alternar entre as visualizações <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT> ou <DNT>**Monthly**</DNT>
* <DNT>**Group by**</DNT>: Selecione dimensões para agrupar dados de custo (até 3 seleções)
* <DNT>**View by**</DNT>: Escolha um formato de visualização (a disponibilidade depende das suas seleções de agrupamento)

### Entendendo os dados

* <DNT>**Total cost**</DNT>: gasto agregado com IA para o período selecionado com variação percentual em relação ao período anterior
* <DNT>**Average cost (Daily)**</DNT>: Média diária de gastos com IA com variação percentual versus o período anterior
* <DNT>**Stacked bar chart**</DNT>: Visualiza a distribuição de custos de IA ao longo do tempo, codificada por cores por provedor de cloud
* <DNT>**Output token**</DNT>: Custo para token de saída gerado pelo modelo em resposta ao prompt

### Opções de visualização

**Com uma única seleção de group-by:**

* <DNT>**Bar chart**</DNT>: exibe dados de custo de IA como barras verticais
* <DNT>**Line chart**</DNT>: Mostra tendências de custos de IA ao longo do tempo
* <DNT>**Tree chart**</DNT>: visualização hierárquica de dados de custo de IA
* <DNT>**Data table**</DNT>: Visualização tabular dos dados de custo de IA

**Com várias seleções de agrupamento:**

* <DNT>**Heat map**</DNT>: Visualização codificada por cores em dimensões
* <DNT>**Data table**</DNT>: Visualização tabular dos dados de custo de IA

## Filtros e personalização

### Seletor de intervalo de datas

Use o seletor de intervalo de datas no canto superior direito para:

* Escolha um período de tempo predefinido
* Defina datas de início e fim personalizadas
* Visualize os custos no seu fuso horário preferido (UTC exibido por padrão)

### Filtro de custos de engenharia

Se a sua organização usa tags de alocação de custos, filtre pela tag <DNT>**Engineering costs**</DNT> para restringir a análise de custos de IA a equipes, projetos ou centros de custo específicos marcados com tag no seu ambiente cloud.
