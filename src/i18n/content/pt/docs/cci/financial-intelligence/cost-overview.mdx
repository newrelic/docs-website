A página Visão geral de custos (anteriormente conhecida como Cost Explorer) fornece análise detalhada de custos com funcionalidade de drill-down em várias dimensões.

Para uma visão consolidada de todos os seus gastos na nuvem com widgets de resumo, consulte a página [Resumo inteligente](/docs/cci/financial-intelligence/intelligent-summary).

## Funcionalidade de drill-down

O Cloud Cost Intelligence fornece capacidade de drill-down para visualizar detalhes granulares de custos.

### Como funciona

Você pode analisar os custos em diferentes dimensões. Ao clicar em um serviço (por exemplo, EC2), você pode detalhar um serviço específico e, em seguida, dentro dele, um recurso específico para analisar os detalhes granulares desse recurso.

Todos os tipos de serviço suportam drill-down, incluindo computação, rede e armazenamento.

## Recurso

* **Filtrável**: Todos os dados são filtráveis e você pode mostrar e ocultar elementos
* **Múltiplas visualizações**: você pode alterar as visualizações para ver os dados em formatos diferentes
* **Agrupar por opções**: Você pode ter várias opções de agrupamento. Por exemplo, você pode visualizar por dimensão de serviço de nuvem e, em seguida, alterá-la para tipo de cobrança e opção de pagamento
* **Visualização de tabela dinâmica**: Ao usar várias seleções de agrupamento, os dados refletem como uma visualização de tabela dinâmica para analisar custos granulares em uma única visualização
* **Download**: Você pode baixar os dados de custo para analisá-los em diferentes dimensões

## Restrições de agrupamento

* **Seleção mínima**: 1 dimensão
* **Seleção máxima**: 3 dimensões

<Callout variant="important">
  Quando você tem várias seleções de agrupamento, gráfico de barras, gráfico de linhas e gráfico de árvore não são aplicáveis. Apenas tabela de dados e mapa de calor estão disponíveis.
</Callout>
