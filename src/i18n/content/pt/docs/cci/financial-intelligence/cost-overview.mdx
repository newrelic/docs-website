A página <DNT>**Cost Overview**</DNT> (anteriormente conhecida como <DNT>**Cost Explorer**</DNT>) fornece análise detalhada de custos com funcionalidade de drill-down em várias dimensões.

Para uma visão consolidada de todos os seus gastos na nuvem com widgets de resumo, consulte a página [Resumo inteligente](/docs/cci/financial-intelligence/intelligent-summary).

<img title="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" alt="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" src="/images/cost-overview.webp" />

## Filtros de provedor de nuvem

Para filtrar seus dados de custo por provedor de cloud, selecione uma opção no menu dropdown no canto superior esquerdo da página:

* <DNT>**All**</DNT>: Todos os provedores de cloud
* <DNT>**AWS**</DNT>: apenas Amazon Web Services
* <DNT>**Azure**</DNT>: apenas Microsoft Azure
* <DNT>**GCP**</DNT>: apenas Google Cloud Platform

## Principais Gastos

O widget <DNT>**Top Spends**</DNT> ajuda você a identificar suas áreas de maior custo. Visualize os custos divididos por:

* <DNT>**Service**</DNT>: custos agrupados por tipo de serviço de nuvem (por exemplo, AmazonEC2, plataforma Fastly Edge Cloud, AmazonS3)
* <DNT>**Category**</DNT>: custos organizados por categoria de uso (por exemplo, Computar, Outros, Rede)
* <DNT>**Region**</DNT>: custos por região geográfica (por exemplo, us-west-2, Sem tags, eastus). &quot;Sem tags&quot; indica custos de recursos sem tags regionais
* <DNT>**Charge type**</DNT>: custos por categoria de faturamento (por exemplo, Usage, SavingsPlanCoveredUsage, DiscountedUsage)

Cada dimensão exibe:

* Valor total do custo
* Variação percentual em comparação com o período anterior (com base no intervalo de datas selecionado)
* Três principais itens com seus custos individuais
* <DNT>**Show summary**</DNT> alternar para expandir ou recolher visualizações detalhadas

<Callout variant="tip">
  Use os indicadores de variação percentual para identificar rapidamente serviços ou regiões com aumentos de custo significativos. Indicadores vermelhos destacam áreas onde os gastos cresceram em comparação com o período anterior.
</Callout>

## Tendências de Custos

A seção <DNT>**Cost Trends**</DNT> fornece análise de custos baseada em tempo com gráficos e controles interativos.

### Controles do gráfico

* <DNT>**View options**</DNT>: Alternar entre as visualizações <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT> ou <DNT>**Monthly**</DNT>
* <DNT>**Group by**</DNT>: Selecione de 1 a 3 dimensões para agrupar dados de custo
* <DNT>**View by**</DNT>: Escolha o formato de visualização (a disponibilidade depende das suas seleções de agrupamento)

### Entendendo os dados

* <DNT>**Total cost**</DNT>: Exibe o gasto agregado com alteração percentual
* <DNT>**Average cost**</DNT>: Exibe o gasto médio diário com a variação percentual
* <DNT>**Stacked bar chart**</DNT>: Visualiza a distribuição de custos ao longo do tempo com segmentos codificados por cores para diferentes serviços

### Tabela de dados de custo

Abaixo do gráfico, uma tabela detalhada é exibida:

* <DNT>**Cloud provider service**</DNT>: Nome do serviço
* <DNT>**Total Cost**</DNT>: Valor do gasto agregado (ordenável)
* <DNT>**% of Spend**</DNT>: Porcentagem do gasto total
* <DNT>**Compared Cost**</DNT>: Comparação de custos com o período anterior
* <DNT>**Trend**</DNT>: Indicador de variação percentual com codificação por cores (verde para redução, vermelho para aumento)

### Opções de visualização

**Restrições de agrupamento:**

* **Seleção mínima**: 1 dimensão
* **Seleção máxima**: 3 dimensões

**Com seleção de agrupamento único:**

* <DNT>**Bar chart**</DNT>: Exibe dados de custo como barras verticais
* <DNT>**Line chart**</DNT>: Mostra tendências de custos ao longo do tempo
* <DNT>**Tree chart**</DNT>: Visualização hierárquica de dados de custo
* <DNT>**Data table**</DNT>: Visualização tabular de dados de custo

**Com várias seleções de agrupamento:**

* <DNT>**Heat map**</DNT>: Visualização codificada por cores
* <DNT>**Data table**</DNT>: Visualização tabular de dados de custo

## Funcionalidade de drill-down

Clique em qualquer serviço (por exemplo, AmazonEC2) na tabela de dados de custo para ver detalhamentos por categoria, região ou tipo de cobrança. Clique novamente para ver os custos de recursos individuais.

Todos os tipos de serviço suportam drill-down, incluindo computação, rede e armazenamento.

## Filtros e personalização

A página <DNT>**Cost Overview**</DNT> oferece suporte a opções de filtragem e personalização para ajudar você a focar em segmentos de custo específicos.

### Seletor de intervalo de datas

No canto superior direito da página, use o seletor de intervalo de datas para:

* Escolha um período de tempo predefinido
* Defina datas de início e fim personalizadas
* Visualize os custos no seu fuso horário preferido (UTC exibido por padrão)

### Filtro de custos de engenharia

Se a sua organização usa tags de alocação de custos, você pode filtrar pela tag <DNT>**Engineering costs**</DNT> para restringir a análise a equipes, projetos ou centros de custo específicos marcados com tag no seu ambiente de cloud.
