## Visão geral

Acesse insights específicos para seus ambientes Kubernetes por meio da página Alocação de custos do Kubernetes. Este recurso utiliza os dados de telemetria do New Relic para fornecer uma visão dos custos associados ao seu aplicativo Kubernetes.

Use the Kubernetes Cost Allocation page to view costs incurred by each application over time and detailed cost analysis.

## Visão geral da alocação

The Allocation Overview widget offers a bar chart visualization illustrating the costs incurred by each application over time.

## Análise detalhada de custos

The Kubernetes Cost Allocation view presents a detailed table with a breakdown of resource usage and costs by application name:

* **Uso da CPU:** exibe a porcentagem de recursos da CPU consumidos por cada aplicativo.
* **Custo da CPU:** mostra o atributo de custo para uso da CPU para cada aplicativo.
* **Uso de memória:** Lista os recursos de memória por porcentagem usada por cada aplicativo.
* **Custo de memória:** reflete o custo associado ao uso de memória.
* **Custo total:** fornece o custo total incorrido por cada aplicativo, integrando todo o uso de recursos.

## Unused and Unallocated cost

These cost dimensions help you measure cluster efficiency:

### Custo não utilizado

Custo não utilizado é quanto do cluster foi alocado, mas não está recebendo nenhuma solicitação.

**Exemplo**: Se seu custo total for de US$ 5,60 alocados para uma implantação de cluster, mas você estiver usando apenas US$ 4,64, então US$ 0,96 é o custo não utilizado.

### Custo não alocado

Custo não alocado é quanto do cluster não foi alocado a nenhuma fonte.

Estas são medidas de eficiência para o seu cluster Kubernetes.

## Benefícios

The Kubernetes Cost Allocation feature delivers these outcomes:

* **Optimized resource usage:** Visibility into underutilized or excessively costly applications enables effective resource allocation adjustments.
* **Controlled expenses:** Insights into specific cost drivers within your Kubernetes environment enable more informed financial decisions.
* **Enhanced efficiency:** Resource management strategies aligned with business objectives ensure efficient use of cloud resources.

The Kubernetes Cost Allocation page provides the insights necessary to proactively manage Kubernetes costs, supporting strategic decision-making and financial optimization.
