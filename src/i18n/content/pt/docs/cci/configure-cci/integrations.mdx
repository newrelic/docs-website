  Atualmente, esse recurso é fornecido como parte de um programa de visualização de acordo com nossas [políticas de pré-lançamento](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).

O recurso Inteligência de Custos na Nuvem integração permite visualizar e gerenciar suas contas de provedores cloud . Use este recurso para pausar ou excluir a integração de uma conta específica de provedor cloud .

1. Navegue até <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>.
2. No menu de navegação à esquerda, selecione a aba **Integrations** para exibir uma lista de todos os custos de integração cloud .
3. Para ver detalhes adicionais, na coluna **Name** , clique na conta do provedor cloud necessária.

<Callout variant="important">
  Atualmente, apenas os custos cloud da AWS são suportados.
</Callout>

4. Ao lado da conta desejada, clique no ícone de reticências e escolha uma das seguintes opções:

   * **Pause**: interrompe temporariamente a integração da coleta de dados.
   * **Delete**: remove permanentemente a integração da sua conta.
