<Callout title="visualização">
  Ainda estamos trabalhando nesse recurso, mas adoraríamos que você experimentasse!

  Atualmente, esse recurso é fornecido como parte de um programa de visualização de acordo com nossas [políticas de pré-lançamento](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Visualize e gerencie suas contas de provedores de nuvem e pause ou exclua integrações para contas específicas.

<Steps>
  <Step>
    ### Acessar o Cloud Cost Intelligence

    Vá para <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>.
  </Step>

  <Step>
    ### Visualizar integrações

    Selecione a aba **Integrations** para visualizar todas as integrações de custo da nuvem.
  </Step>

  <Step>
    ### Selecione uma conta

    Clique em uma conta de provedor de nuvem na coluna **Name** para obter detalhes.

    <Callout variant="important">
      Atualmente, apenas os custos da nuvem AWS são suportados.
    </Callout>
  </Step>

  <Step>
    ### Gerenciar a integração

    Clique no ícone de reticências ao lado de uma conta e escolha:

    * **Pause** - Interromper temporariamente a coleta de dados
    * **Delete** - Remover integração permanentemente

    <img title="Pause and delete integration" alt="A screenshot displaying the pause and delete integration options" src="/images/pause-delete-cci.webp" />
  </Step>
</Steps>
