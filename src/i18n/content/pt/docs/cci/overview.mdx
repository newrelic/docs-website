## Visão geral

O Cloud Cost Intelligence oferece visibilidade e gerenciamento de custos de nuvem em todos os seus provedores de nuvem. Ele preenche a lacuna entre a visibilidade de custos e a observabilidade da infraestrutura, oferecendo uma visão holística de seus gastos com a nuvem.

À medida que as empresas escalam seus serviços de nuvem rapidamente, gerenciar e rastrear as despesas na nuvem se torna cada vez mais complexo. O Cloud Cost Intelligence oferece insights sobre a alocação de custos, permitindo que você identifique oportunidades potenciais de economia. Ao combinar as despesas na nuvem com o uso real, você pode otimizar os orçamentos, ao mesmo tempo em que apoia as iniciativas estratégicas de crescimento, garantindo que cada dólar investido contribua para o sucesso geral da sua empresa.

## Recurso chave

### Resumo inteligente

A página [Resumo inteligente](/docs/cci/financial-intelligence/intelligent-summary) é sua página de destino principal para o Cloud Cost Intelligence. Ela exibe todos os seus gastos na nuvem em um só lugar, mostrando como os custos se movem pela sua organização, divididos por serviços, equipes e regiões. Você pode navegar na página e alterar as visualizações com base em suas necessidades.

Os principais recursos incluem:

* **Gráfico Sankey:** Navegue pelas visualizações dinâmicas do gráfico Sankey para detalhar os custos.
* **Segregação de custos:** Custos segregados por serviços, equipes e regiões.
* **Visualizações dinâmicas:** Alterne entre diferentes tipos de visualização (por exemplo, visualização de contas)
* **Divisão por categoria:** Custos de nuvem segmentados por aplicativo, serviço, equipe proprietária e região
* **Análise de tendências:** Tendências de custos mês a mês, custos médios e principais impulsionadores de custos

### Visão geral de custos

A página [Visão geral de custos](/docs/cci/financial-intelligence/cost-overview) fornece análise detalhada de custos com funcionalidade de drill-down para explorar custos em várias dimensões, do nível de serviço até recursos individuais.

### Alocação de custos do Kubernetes

Visualize os custos do Kubernetes divididos por nome do aplicativo usando dados de telemetria do New Relic.

### Estimativa de custos em tempo real

O Cloud Cost Intelligence fornece atualizações de custo por hora usando dados de telemetria e padrões de custo históricos, refletindo as alterações dentro de uma hora do provisionamento de recursos. A integração com seus dados de faturamento permite estimativas de custos com base no preço unitário médio da semana passada e nos dados de telemetria do New Relic para o período recente de 48 horas. Sem essa integração, as estimativas são padronizadas para taxas de preços unitários padrão potencialmente mais altas.

Para obter as estimativas mais precisas, integre a telemetria e mantenha pelo menos 7 dias de dados históricos. Sem esses dados, as estimativas revertem para preços sob demanda, o modelo padrão de pagamento conforme o uso usado pelos provedores de nuvem.

### Coleta de dados entre contas

O Cloud Cost Intelligence pode coletar dados de telemetria de todas as contas New Relic às quais você tem acesso. Mesmo que você use uma única conta para Cloud Cost Intelligence, os dados de todas as outras contas que você pode acessar também podem ser usados para coletar e obter insights nessa conta. Verifique se suas permissões de conta estão alinhadas com suas políticas internas de acesso a dados e configure-as através de [Configurações](/docs/cci/configure-cci/general-setting).

## Como funciona

### Coleção de dados [#fetch-costs]

Após a conclusão da configuração, o Cloud Cost Intelligence acessa automaticamente seus dados de custo e uso do seu contêiner de armazenamento de objetos designado. Nosso serviço só pode recuperar relatórios depois que eles forem exportados para seu contêiner de armazenamento. Certifique-se de que o prefixo do caminho da pasta esteja configurado corretamente durante a configuração.

Seu provedor de nuvem normalmente atualiza os dados de faturamento várias vezes ao dia. Após a configuração inicial, o Cloud Cost Intelligence recupera o relatório mais recente de seu local de armazenamento a cada hora. Os dados de custo mais recentes aparecem em nossa plataforma logo após seu provedor publicar a atualização.

* **Sua privacidade de dados**: Os Relatórios de Custo e Uso contêm dados sobre o seu uso de serviços de nuvem e seus custos, que não devem incluir dados pessoais. Certifique-se de que os dados que você envia estejam livres de informações de identificação pessoal (PII) ou informações confidenciais antes de adicionar os Relatórios de Custo e Uso ao seu armazenamento e antes de fazer o upload para o New Relic. A exclusão de dados não é suportada no momento. Recomendamos não gerar Relatórios de Custo e Uso com PII ou informações confidenciais.

* **Retenção de dados:** Os dados são retidos por até 5 anos. Após 2 anos, os dados são movidos para um armazenamento de menor custo com tempos de recuperação mais longos.

* **Visualizando acesso:** Qualquer pessoa com acesso à conta New Relic que você selecionar durante a instalação pode visualizar todos os dados no Cloud Cost Intelligence. Escolha uma conta com privilégios de acesso adequados para corresponder aos seus requisitos de RBAC e política de acesso.

### Análise de dados e relatórios

Após recuperar os dados do Relatório de Custo e Uso, o Cloud Cost Intelligence processa e analisa essas informações para fornecer insights acionáveis em seus painéis. Visualize as implicações financeiras do seu uso da nuvem, identificando os impulsionadores de custos e áreas potenciais de otimização.
