[Comet](https://www.comet.ml/site/) é uma plataforma [de monitoramento de desempenho do modelo](/docs/alerts-applied-intelligence/mlops/get-started/intro-mlops/) que estende a observabilidade full-stack para modelos de aprendizado de máquina e permite estabelecer uma linha de base de desempenho de produção com base no desempenho do modelo. A integração do Comet permite integrar métricas de machine learning com o New Relic, para que você possa monitor continuamente seus dados durante todo o ciclo de vida do machine learning. Isso ajuda a garantir o desempenho ideal do modelo e a obter melhores resultados de negócios.

## Integre o Comet com a New Relic [#integrate-comet]

Use a integração Comet da New Relic para obter dados e insights que ajudam você a construir modelos de aprendizado de máquina melhores e mais precisos, ao mesmo tempo que melhora a produtividade, a colaboração e a visibilidade de toda a sua equipe.

Para configurar a integração, siga estas etapas:

1. <DNT>**Log into your New Relic account**</DNT>: Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > +Add more data**</DNT>.

2. <DNT>**Click on Comet**</DNT>: digite <DNT>**Comet**</DNT> na barra de pesquisa.

3. <DNT>**Select the account ID**</DNT> você deseja que o Comet se integre.

4. <DNT>**Select or create an API key**</DNT>: Para integração com Comet, você precisa de uma New Relic<InlinePopover type="licenseKey"/>. Crie ou selecione uma chave de API existente na seção <DNT>**Real time training metric**</DNT> .

5. <DNT>**Contact Comet**</DNT>: Para a próxima etapa de configuração da integração e visualização do dashboard de monitoramento de produção do modelo no Comet, você precisará [entrar em contato com o Comet](mailto:sales@comet.ml). A equipe Comet orientará você em cada etapa do processo de integração e preparará tudo.

   <img

6. <DNT>**View your data**</DNT>: Agora que configurou sua integração, você pode começar a monitorar o desempenho de seus modelos de aprendizado de máquina em um [dashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) dedicado, que é gerado automaticamente pela equipe da Comet.
