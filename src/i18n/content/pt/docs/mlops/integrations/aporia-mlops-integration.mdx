Aporia é uma maneira rápida, fácil e segura para equipes de ciência de dados monitor seus modelos de aprendizado de máquina em produção. Com Aporia, as equipes podem construir seu próprio monitor personalizável em minutos para receber alertas ao vivo para detecção precoce de problemas como imprecisão de dados, preconceito inesperado, problemas de integridade de dados e degradação de desempenho. Aporia também possui uma caixa de ferramentas de investigação para permitir investigações adicionais e análises de causa raiz. [Saiba mais sobre Aporia](https://www.aporia.com).

Aporia inclui integração com New Relic para fornecer gerenciamento completo de modelo de sua infraestrutura MLOps, com <InlinePopover type="dashboards"/>personalizados dentro do New Relic que mostram investigação de inferência.

## Integre Aporia com New Relic [#integrate-aporia]

Aporia permite que você conecte <InlinePopover type="alerts"/>gerado pelo monitor da Aporia ao mecanismo de inteligência de incidentes da New Relic e aos dados de previsão para criar um dashboard de monitoramento abrangente na New Relic para seus modelos.

1. <DNT>
     **Login into Aporia's console**
   </DNT>

   : na barra de navegação à esquerda, clique em

   <DNT>
     **Integrations**
   </DNT>

   e escolha

   <DNT>
     **New Relic**
   </DNT>

   .

2. <DNT>
     **Login into your New Relic account:**
   </DNT>

   Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > +Add more data**
   </DNT>

   .

3. <DNT>
     **Click on Aporia:**
   </DNT>

   Na barra de pesquisa, digite

   <DNT>
     **Aporia**
   </DNT>

   .

4. <DNT>
     **Select your account ID:**
   </DNT>

   Selecione o ID da conta com a qual deseja que o Aporia se integre.

5. <DNT>
     **Get a New Relic API key:**
   </DNT>

   Depois de clicar no ícone Aporia, siga o primeiro passo. Em

   <DNT>
     **Prediction data & model quality alerts**
   </DNT>

   , clique em

   <DNT>
     **Select**
   </DNT>

   , para escolher uma chave existente, ou

   <DNT>
     **Create API key**
   </DNT>

   . Isso criou uma New Relic

   <InlinePopover type="licenseKey"/>

   , que é usada para relatar dados para a New Relic.

6. <DNT>
     **Paste the key in Aporia:**
   </DNT>

   Agora vá ao dashboard da Aporia e na tela de integração do New Relic, cole a chave em

   <DNT>
     **New Relic insert token**
   </DNT>

   . Depois de colado, clique em

   <DNT>
     **Save**
   </DNT>

   .

7. <DNT>
     **Verify the key:**
   </DNT>

   No dashboard do Aporia, clique no botão

   <DNT>
     **Verify token**
   </DNT>

   para verificar se a chave está funcionando corretamente. Uma marca de seleção

   <DNT>
     **Green**
   </DNT>

   ou marcas de erro

   <DNT>
     **Red**
   </DNT>

   deverão aparecer para indicar o status.

## Monitor seus modelos de aprendizado de máquina com Aporia

Agora que integrou New Relic e Aporia, você pode monitor seus dados usando [painel](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) com gráficos automatizados criados por Aporia.

1. <DNT>**Go to the integration dashboard:**</DNT> Depois de verificar seu token e confirmar que a integração está configurada corretamente, retorne ao dashboard de integração do New Relic e clique em <DNT>**See your data**</DNT>. Isso irá redirecioná-lo para um dashboard gerado automaticamente exibindo os dados relatados à Aporia no New Relic.

2. <DNT>**Analyze Aporia's dashboard.**</DNT> Dashboard da Aporia preenchido no New Relic contém 6 gráficos:

   * O gráfico

     <DNT>
       **Most active models**
     </DNT>

     exibe os diferentes modelos que relataram previsões no período selecionado.

   * O gráfico

     <DNT>
       **Most active model versions**
     </DNT>

     exibe as diferentes versões que relataram previsões no período selecionado.

   * O gráfico

     <DNT>
       **Model inferences**
     </DNT>

     exibe o número de previsões exclusivas relatadas para cada modelo e versão.

   * O gráfico

     <DNT>
       **Average numeric inferences**
     </DNT>

     exibe as previsões numéricas de valor médio relatadas para cada modelo e versão.

   * O gráfico

     <DNT>
       **Numeric inferences heatmaps**
     </DNT>

     exibe um histograma das previsões numéricas relatadas para cada modelo e versão.

   * Os gráficos

     <DNT>
       **Categorical inferences**
     </DNT>

     exibem os diferentes valores exclusivos e suas frequências de predições categóricas reportadas para cada modelo e versão.

   <img
     title="An image of the Aporia dashboard"
     alt="An image of the Aporia dashboard."
     src="/images/model-performance-monitoring_screenshot-full_aporia-dashboard.webp"
   />

3. <DNT>**Filter data:**</DNT> Clique no botão <DNT>**…**</DNT> e clique em <DNT>**Edit**</DNT>. Na barra de navegação direita, em <DNT>**User as filter**</DNT>, ative <DNT>**Filter the current dashboard**</DNT> e clique em <DNT>**Save**</DNT>.

4. <DNT>**Setup alert notifications:**</DNT> Depois de criar algum painel, você pode receber alertas sobre seus dados. Para criar uma condição do alerta NRQL a partir de um gráfico, clique no widget de gráfico e, em seguida, clique em <DNT>**Create alert condition**</DNT>. Depois de nomear e personalizar sua condição, você poderá adicioná-la a uma política existente ou criar uma nova.

   <img
     title="Set up notifications for Aporia"
     alt="Set up notifications for Aporia."
     src="/images/model-performance-monitoring_screenshot-crop_aporia-create-alert.webp"
   />

5. <DNT>**Get notified:**</DNT> Depois de criar uma condição do alerta, você pode escolher como deseja ser notificado. Veja nossa documentação sobre [como configurar o canal de notificação](/docs/alerts-applied-intelligence/new-relic-alerts/alert-notifications/notification-channels-control-where-send-alerts).

6. <DNT>**Correlate your incidents:**</DNT> Além da notificação, você pode usar a inteligência do incidente para correlacionar o seu incidente. Veja nossos documentos sobre [como correlacionar incidentes usando decisões](/docs/alerts-applied-intelligence/applied-intelligence/incident-intelligence/change-applied-intelligence-correlation-logic-decisions).
