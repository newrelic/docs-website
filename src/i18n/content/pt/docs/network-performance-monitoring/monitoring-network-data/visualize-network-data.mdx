Depois de configurar os dados de rede para monitoramento de desempenho, acesse <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Network**</DoNotTranslate> para explorar os dados de rede que seus dispositivos estão reportando à New Relic.

Você pode agrupar e monitor seus switches, roteadores, etc. no painel. Isso ajuda você a agregar, isolar e analisar rapidamente a integridade da sua rede.

## Pré-requisitos [#prerequisites]

* Uma conta New Relic . Não tem um? [Cadastre-se gratuitamente!](https://newrelic.com/signup) Não é necessário cartão de crédito.
