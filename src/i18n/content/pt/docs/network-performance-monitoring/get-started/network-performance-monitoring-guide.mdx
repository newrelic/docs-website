Você é um engenheiro de operações que conta com uma rede de dispositivos habilitados para SNMP. Sua equipe já utiliza o recurso de monitoramento de software e infraestrutura da New Relic , mas é hora de monitor também seus dispositivos de rede.

Nesta faixa, você usa a biblioteca ktranslate da Kentik para descobrir automaticamente seus dispositivos de rede, coletar dados e enviar esses dados para a New Relic para correlacionar o desempenho da rede com infraestrutura, aplicativos e experiências digitais. Então você usa New Relic para monitor a integridade da sua rede.

## Objetivos

* Pesquise dados SNMP de dispositivos de rede e envie-os para a New Relic
* Amostra de dados de fluxo de rede e envie-os para New Relic
* Crie um New Relic dashboard para monitorar seus dispositivos de rede
* Crie uma New Relic workload para agrupar logicamente seus dispositivos e configurar a detecção de anomalias
* Use seus novos dados para entender os comportamentos em sua rede

## Requisitos

* Uma conta gratuita da New Relic
* Um usuário completo da plataforma ou um usuário principal com privilégio de modificação de usuário do Nerdpack

<Callout variant="important">
  Esta faixa utiliza uma máquina virtual efêmera. Como resultado, você precisa terminar a pista inteira de uma só vez ou perderá seu progresso quando o tempo expirar.
</Callout>
