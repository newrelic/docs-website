Sabemos que o seu código-fonte é um dos seus ativos mais valiosos e confidenciais. Para a maioria das equipes de desenvolvimento, é uma representação literal do produto de trabalho mais importante.

CodeStream precisa armazenar trecho de código em nosso banco de dados quando você discute um bloco de código por meio de um comentário ou problema. CodeStream salva esse bloco de código no banco de dados por vários motivos. Por exemplo, se o código em questão não foi enviado por push, armazená-lo é a única maneira de compartilhá-lo com seus colegas de equipe.
