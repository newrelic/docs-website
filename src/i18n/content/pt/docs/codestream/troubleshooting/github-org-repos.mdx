Quando você tenta criar um problema no GitHub, se faltar repositório de alguma de sua organização GitHub é provavelmente porque no momento em que você se autenticou no GitHub você não concedeu acesso a toda sua organização.

Se você não clicou no botão <DoNotTranslate>**Grant**</DoNotTranslate> no momento da autenticação, no GitHub vá para <DoNotTranslate>**Settings > Applications**</DoNotTranslate> e clique na guia <DoNotTranslate>**Authorized OAuth Apps**</DoNotTranslate> . A partir daí, clique no aplicativo CodeStream. Na página seguinte, clique no botão <DoNotTranslate>**Grant**</DoNotTranslate> ao lado de qualquer organização que você gostaria de poder acessar no CodeStream.

Em alguns casos, você verá um botão <DoNotTranslate>**Request**</DoNotTranslate> em vez de <DoNotTranslate>**Grant**</DoNotTranslate>, o que significa que o proprietário da sua organização GitHub precisará conceder acesso a você.
