New Relic CodeStream é uma extensão de IDE que permite mudar para a esquerda, tornando o desempenho do código parte dos estágios iniciais do processo de desenvolvimento.

* Veja o desempenho dos serviços construídos a partir do seu código, com informações sobre métricas clássicas, erros, anomalias de transação, SLOs e muito mais.
* Investigue erros mais rapidamente percorrendo o stack trace e colaborando com colegas de equipe.
* Pesquise o log do seu IDE para investigar problemas sem mudar de contexto.
* Execute a consulta NRQL e salve e compartilhe a consulta com seus colegas de equipe por meio de `.nrql` arquivos.
* Veja os dados de desempenho de métodos individuais no código, diretamente no editor.

## Investigação de erros

Descubra erros que acontecem no código pelo qual você é responsável. Percorra rapidamente o stack trace, com o CodeStream levando você automaticamente ao arquivo e ao número de linha corretos e, em seguida, traga colegas de equipe relevantes para a discussão para que você possa resolver o problema o mais rápido possível.

## Pesquisa log

Se você precisa investigar um problema pesquisando o log, por que mudar o contexto do seu IDE para fazer esse trabalho na web? Log de pesquisa diretamente do seu IDE. Você pode até iniciar uma pesquisa a partir de linhas de log individuais em seu código.

## Métrica al nível do código

Veja os dados de desempenho, no nível do método, diretamente no seu editor. A visualização sempre ativa garante que os problemas de desempenho sejam levados em consideração no trabalho em questão. Você pode até ver métricas para seu ambiente de testes e evitar problemas de desempenho antes que eles cheguem à produção.
