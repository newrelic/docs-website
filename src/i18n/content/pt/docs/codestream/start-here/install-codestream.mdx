Você pode instalar a extensão CodeStream para seu IDE específico.

<CollapserGroup>
  <Collapser className="freq-link" id="vscode" title="Código do Visual Studio">
    Baixe e instale o CodeStream para [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream). Você também pode instalá-lo diretamente no Visual Studio Code por meio do mercado de extensões.
  </Collapser>

  <Collapser className="freq-link" id="visual-studio" title="Visual Studio">
    Baixe e instale o CodeStream para [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22). Você também pode instalá-lo diretamente no Visual Studio por meio do mercado de extensões.
  </Collapser>

  <Collapser className="freq-link" id="jetbrains" title="JetBrains">
    Baixe e instale CodeStream para [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream). Você também pode instalá-lo no menu do plug-in JetBrains.
  </Collapser>
</CollapserGroup>

Se você estiver usando o VS Code, alguns recursos CodeStream exigem uma extensão de suporte a linguagens como [Java](https://marketplace.visualstudio.com/items?itemName=redhat.java), [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp), [PHP](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client), [Python](https://marketplace.visualstudio.com/items?itemName=ms-python.python), [Go](https://marketplace.visualstudio.com/items?itemName=golang.go) ou [Ruby](https://marketplace.visualstudio.com/items?itemName=Shopify.ruby-lsp).

## Entrar [#signin]

Depois de instalar a extensão CodeStream em seu IDE, faça login com suas credenciais do New Relic.
