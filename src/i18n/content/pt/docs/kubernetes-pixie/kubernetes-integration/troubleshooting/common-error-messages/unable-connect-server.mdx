## Problema

Você obtém esse erro de saída ao seguir a instalação guiada.

```shell
  Unable to connect to the server: dial tcp [7777:777:7777:7777:77::77]:443: i/o timeout
```

## Solução

Isso indica que você está enfrentando um problema de conexão de rede entre o cliente Kubernetes e o servidor da API do Kubernetes. Certifique-se de que seu cliente Kubernetes possa se conectar ao seu servidor de API do Kubernetes antes de executar a instalação guiada novamente.
