## Problema

Você está recebendo a mensagem `x509: certificate signed by unknown authority`.

## Solução

Você precisa garantir que seus certificados assinados estejam configurados corretamente. Para fazer isso, você pode adicionar as seguintes variáveis de ambiente no contêiner YML do agente:

```yml
- name: NRIA_CA_BUNDLE_DIR 
  value: YOUR_CA_BUNDLE_DIR
- name: NRIA_CA_BUNDLE_FILE  
  value: YOUR_CA_BUNDLE_NAME
```
