## Problema

Você vê esta mensagem de erro durante a [instalação da integração do Kubernetes](/install/kubernetes/) com o Helm ou o Manifest.

```shell
  repo newrelic not found
```

## Solução

Adicione o repositório newrelic ao seu gráfico helm executando este comando:

```shell
  helm repo add newrelic https://helm-charts.newrelic.com
```
