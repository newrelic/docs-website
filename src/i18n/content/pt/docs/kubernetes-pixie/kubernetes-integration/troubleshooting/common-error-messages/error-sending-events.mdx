## Problema

O agente não consegue se conectar aos servidores New Relic e você vê um erro como o seguinte no log do contêiner `agent` ou `forwarder`:

```shell
2018-04-09T18:16:35.497195185Z time="2018-04-09T18:16:35Z" level=error 
msg="metric sender can't process 1 times" error="Error sending events: 
Post https://api.newrelic.com/metrics/events/bulk: 
net/http: request canceled (Client.Timeout exceeded while awaiting headers)"
```

## Solução

Dependendo da natureza exata do erro, a mensagem no log pode ser diferente. Para resolver esse problema, consulte a [documentação das redesNew Relic ](/docs/new-relic-solutions/get-started/networks/#infrastructure)e a página do GitHub [sobre resolução de problemas do agente de rede New Relic Infrastructure](https://github.com/newrelic/infrastructure-agent/blob/master/docs/network_troubleshooting.md?) .
