## Problema

Você está recebendo erros como `cannot increase buffer: current=32000 requested=64768 max=32000` ou `pod out of memory`.

## Solução

Adicione o atributo `k8sBufferSize` para aumentar o tamanho do buffer do bit fluente.

```yaml
newrelic-logging:
  enabled: true
  containerSecurityContext:
    privileged: true
  fluentBit:
    k8sBufferSize: "64k"
    persistence:
      mode: "none"
```
