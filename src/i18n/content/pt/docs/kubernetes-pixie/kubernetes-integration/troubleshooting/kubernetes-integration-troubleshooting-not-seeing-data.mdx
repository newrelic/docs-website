## Problema

Você [instalou a integração do New Relic Kubernetes](/install/kubernetes/), mas não está vendo os dados do Kubernetes em sua conta do New Relic.

## Solução

1. Confirme que você configurou `CLUSTER_NAME` e `NRI_LICENSE_KEY` corretamente. Consulte as [instruções de instalação](/docs/integrations/host-integrations/host-integrations-list/kubernetes-monitoring-integration#install) para obter mais informações.

2. Confirme se a implantação da integração foi bem-sucedida executando:

   ```
   kubectl get pods -o wide -n <NEWRELIC_NAMESPACE>
   kubectl get deployments -o wide -n <NEWRELIC_NAMESPACE>
   kubectl get daemonsets -o wide -n <NEWRELIC_NAMESPACE>
   ```

3. [Recupere o log do agente de infraestrutura e integração](/docs/integrations/kubernetes-integration/troubleshooting/get-logs-version) e certifique-se de que não haja [mensagens de erro](/docs/integrations/host-integrations/troubleshooting/kubernetes-integration-troubleshooting-error-messages).
