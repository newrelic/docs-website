Saiba mais sobre o modelo de armazenamento híbrido para integração do New Relic Pixie [aqui](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/overview). ## Painel Pixie pré-construído [#pixie-dashboards]

Para instalar um dashboard pré-construído para dados do Pixie:

1. Navegue até a New Relic página do [painel .](https://one.newrelic.com/dashboards)

2. Selecione o botão

   <DNT>
     **Browse pre-built dashboards**
   </DNT>

   .

3. Procure por “Pixie” e escolha uma das opções disponíveis.

4. Selecione o botão

   <DNT>
     **Install this quickstart**
   </DNT>

   .

## Painel Pixie personalizado [#custom-dashboards]

O widget de visualização no painel Pixie pré-construído é preenchido com dados da consulta NRQL . Selecione dashboard widgeto menu de três pontos de qualquer para visualizar a consulta que fornece os dados. Para saber como escrever sua própria consulta para dados persistentes do Pixie, consulte a página [Consultando dados do Pixie](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/query-pixie-data) .

Descubra mais informações sobre como personalizar um dashboard pré-construído ou criar seu próprio painel personalizado [aqui](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).
