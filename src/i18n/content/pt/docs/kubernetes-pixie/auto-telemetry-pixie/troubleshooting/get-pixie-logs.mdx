Para verificar a integridade de um cluster com o Pixie instalado, execute o [script de diagnóstico](https://github.com/newrelic-experimental/k8s-diag-utilities/tree/main/pixie-diag).

Este script bash simples executa comandos padrão do Kubernetes no namespace em que o Pixie está instalado. A saída deste script pode ser usada para fazer uma investigação mais aprofundada por conta própria ou com o suporte da New Relic.
