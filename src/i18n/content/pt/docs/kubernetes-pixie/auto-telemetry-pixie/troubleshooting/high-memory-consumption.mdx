## Problema

Você concluiu o [procedimento de instalação](/docs/kubernetes-pixie/auto-telemetry-pixie/install-auto-telemetry-pixie) para integração do Pixie com o New Relic e gostaria de reduzir a quantidade de memória que o Pixie usa.

## Solução

Pixie [usa memória para duas finalidades diferentes](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#memory-usage): coleta de dados de telemetria e armazenamento desses dados em cluster de curto prazo. Se desejar especificar uma quantidade diferente de memória para armazenamento de curto prazo, você poderá [configurar o limite de memória de armazenamento de curto prazo](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#reduce-data-storage) durante a instalação.
