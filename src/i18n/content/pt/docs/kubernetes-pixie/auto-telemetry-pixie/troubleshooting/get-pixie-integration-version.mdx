Para determinar qual versão da integração do New Relic Pixie está instalada em seu cluster, siga estas etapas:

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>.

2. Selecione <DNT>**Query your data**</DNT>.

3. Copie a consulta a seguir e clique em <DNT>**RUN**</DNT>.

   ```sql
   FROM K8sContainerSample SELECT latest(containerImage) WHERE containerName LIKE '%newrelic-pixie%' FACET clusterName
   ```

Esta consulta gera uma lista de nomes de cluster com suas versões de integração para a conta selecionada:
