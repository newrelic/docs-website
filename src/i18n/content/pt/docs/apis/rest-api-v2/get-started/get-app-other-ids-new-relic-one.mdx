Ao usar a [API REST New Relic](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2), você precisa substituir algum ID de espaço reservado (como `$APP_ID`, `$KEYTX_ID`, `$COMPONENT_ID`, etc.) pelo ID de um aplicativo, transação ou outro componente.

Você pode usar a interface do New Relic para obter alguns desses IDs. Você também pode obter uma lista de IDs usando o [REST API Explorer](/docs/apis/rest-api-v2/get-started/list-application-id-host-id-instance-id).

## Encontre IDs usando a interface [#ui]

A interface inclui os IDs do aplicativo e alguns outros componentes em um painel de metadados. Para ver os metadados de uma entidade, clique em uma entidade monitora na lista do explorador de entidades e clique no ícone da tag.

Aqui estão alguns exemplos:

<CollapserGroup>
  <Collapser
    id="apm"
    title="APM: ID do aplicativo"
  >
    Para localizar o `$APP_ID` na interface do New Relic:

    1. Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DNT> e selecione um aplicativo.

    2. Clique no <Icon name="fe-tag"/> ícone ao lado do nome do aplicativo para abrir o painel de metadados.

    3. Localize o <DNT>**App ID**</DNT>.

       Substitua esse número por `$APP_ID` em sua chamada de API REST.
  </Collapser>

  <Collapser
    id="browser"
    title="ID do aplicativo do browser"

    2. Clique no <Icon name="fe-tag"/> ícone ao lado do nome do aplicativo para abrir o painel de metadados.
