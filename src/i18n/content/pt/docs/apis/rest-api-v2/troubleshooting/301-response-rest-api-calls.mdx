## Problema

Sua chamada de API REST retorna uma mensagem de erro `301` que pode incluir algumas ou todas estas informações:

```sh
curl -X GET 'http://api.newrelic.com/v2/applications/1234567890.json' \
     -H 'X-Api-Key:YOUR_API_KEY' -i
[output] HTTP/1.1 301 Moved Permanently
[output] Status: 301 Moved Permanently
[output] Location: http://api.newrelic.com/v2/applications/1234567890.json
[output] ...
[output] You are being redirected https://api.newrelic.com/v2/applications/1234567890.json
```

OU

```sh
curl -X GET 'http://api.newrelic.com/v2/applications/1234567890.json' \
     -H 'X-Api-Key:YOUR_API_KEY'
[output] You are being redirected https://api.newrelic.com/v2/applications/1234567890.json
```

## Solução

A New Relic requer HTTPS para todo o tráfego para <InlinePopover type="apm" />e a API REST. Isso inclui tráfego de entrada e saída. Se sua chamada de API REST usar HTTP, altere seu script ou programa para HTTPS.
