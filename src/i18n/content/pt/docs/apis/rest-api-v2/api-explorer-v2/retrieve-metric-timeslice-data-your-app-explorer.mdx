Ao usar o REST API Explorer (v2) da New Relic para obter [dados de métrica de fração de tempo](/docs/telemetry-data-platform/ingest-manage-data/understand-data/new-relic-data-types#timeslice-data) para seu aplicativo, você precisa:

* Uma [chave de API](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key)
* O [ID do aplicativo](#app_id)
* O tipo de dados de métrica de fração de tempo disponíveis para seu aplicativo, incluindo os [nomes e valores das métricas](#view_metrics)

## Ver o ID do seu aplicativo [#app_id]

Para visualizar [o ID do seu aplicativo](/docs/apm/apis/requirements/identification-code):

1. No New Relic REST [API Explorer](https://rpm.newrelic.com/api/explore "O link abre em uma nova janela"), selecione

   <DNT>
     **Applications > GET List**
   </DNT>

   .

2. Se você não estiver conectado ao New Relic, forneça uma [chave de API](/docs/apis/rest-api-v2/requirements/rest-api-key#viewing) para seu aplicativo.

3. Opcional: Em

   <DNT>
     **Applications > List**
   </DNT>

   , preencha os valores para os filtros `name`, `ids` ou `language` .

4. Selecione

   <DNT>
     **Send Request**
   </DNT>

   .

5. Na seção

   <DNT>
     **Response**
   </DNT>

   , copie o `id` do aplicativo.

Continue com o procedimento para visualizar [os nomes das métricas](#view_metrics).

## Veja nomes de métricas para seu aplicativo [#view_metrics]

Para visualizar os dados da métrica de frações de tempo disponíveis para seu aplicativo:

1. No New Relic REST API Explorer, selecione

   <DNT>
     **Applications > GET Metric Names**
   </DNT>

   .

2. Em

   <DNT>
     **Applications > Metric Names**
   </DNT>

   , digite ou cole seu [ID do aplicativo](/docs/apm/apis/requirements/identification-code).

3. Preencha uma métrica específica ou parcial `name` ou deixe em branco para visualizar a lista de nomes e valores de métricas disponíveis.

4. Selecione

   <DNT>
     **Send Request**
   </DNT>

   .

5. Na seção

   <DNT>
     **Response**
   </DNT>

   , revise os `metrics` disponíveis do aplicativo.

Continue com o procedimento para obter [dados](#app_metric_data) [métricos](#app_metric_data) de intervalo de tempo.

## Obtenha dados de métrica de fração de tempo para seu aplicativo [#app_metric_data]

Para obter os valores métricos do seu aplicativo, copie os valores dos resultados <DNT>**Metric Names**</DNT> e cole-os nos campos <DNT>**Metric Timeslice Data**</DNT> :

1. No New Relic REST API Explorer, selecione

   <DNT>
     **Applications > GET Metric Data**
   </DNT>

   .

2. Em

   <DNT>
     **Applications > Metric Data**
   </DNT>

   , digite ou cole seu [ID do aplicativo](/docs/apm/apis/requirements/identification-code).

3. Digite ou cole um ou mais `names` (de

   <DNT>
     **GET Metric Names**
   </DNT>

   ) para seu aplicativo. Comece cada nome em uma nova linha.

4. Selecione

   <DNT>
     **Send Request**
   </DNT>

   .

5. Na seção

   <DNT>
     **Response**
   </DNT>

   , revise os `metric_data` disponíveis do aplicativo.

Após revisar a seção <DNT>**Response**</DNT> , você poderá continuar fazendo ajustes e selecionar <DNT>**Send Request**</DNT> novamente. Por exemplo:

* Para especificar um [intervalo de tempo](/docs/apm/apis/api-v2-examples/specifying-time-range-api-v2) para os dados de métrica de fração de tempo, preencha os valores `from` e `to` .
* Para criar uma [média](/docs/apm/apis/requirements/calculating-average-metric-values-summarize) dos dados durante o intervalo de tempo, defina `summarize` como `true`. Ou, para manter os dados de cada período do intervalo especificado, deixe `summarize` em branco.
