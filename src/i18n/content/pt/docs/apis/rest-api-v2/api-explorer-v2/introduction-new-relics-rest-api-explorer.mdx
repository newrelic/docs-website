A New Relic oferece [diversas APIs](/docs/apis/getting-started/introduction-new-relic-apis), incluindo a [APIREST da New Relic](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2). Este documento apresenta o [REST API Explorer](https://api.newrelic.com), que permite ao usuário administrador e àqueles que possuem a [API Key](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key) :

* Navegue pelos endpoints REST disponíveis da API.
* Interaja com a API REST dentro de uma interface do usuário (o API Explorer).
* Veja uma fonte de documentação ao vivo.
* Obtenha o comando curl para ações da API.
* Compartilhe a chamada de API configurada com colegas copiando e colando as URLs do API Explorer.

Isso ajuda você a pesquisar soluções rapidamente e testar sua chamada de API antes de adicioná-las aos seus próprios componentes de software.

As chamadas de API REST para infraestrutura <InlinePopover type="alerts"/>não estão disponíveis no API Explorer.

## Recurso

O API Explorer da New Relic inclui uma interface interativa do usuário para sua conta selecionada. A interface do API Explorer lista os tipos de chamada de API (<DoNotTranslate>**Applications**</DoNotTranslate>, <DoNotTranslate>**Users**</DoNotTranslate>, etc.) e suas funções disponíveis, como <DoNotTranslate>**GET metric data**</DoNotTranslate>, <DoNotTranslate>**PUT (update) applications**</DoNotTranslate>, <DoNotTranslate>**DELETE applications**</DoNotTranslate>, etc.

À medida que você digita valores para <DoNotTranslate>**Parameters**</DoNotTranslate>, eles aparecem automaticamente em <DoNotTranslate>**Request**</DoNotTranslate> para que você possa testar e verificar sua sintaxe antes de enviar a solicitação. A interface indica os campos obrigatórios, as descrições dos campos, seu tipo (inteiro, float, booleano, etc.) e sua localização (caminho, consulta, etc.).

Para obter informações sobre os requisitos de chave de API, consulte [Chaves de API REST](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key).

## Diferenças da versão 1 da API [#v1]

Este API Explorer se aplica apenas à [API REST do New Relic versão 2](/docs/apm/apis/requirements/new-relic-rest-api-v2-getting-started), que se concentra nos dados que entram e saem do New Relic. A versão 2 substitui a versão 1 da API REST obsoleta da New Relic.

Esteja ciente de que existem algumas diferenças entre as versões 2 e 1:

* Os nomes dos dados podem ser diferentes.
* Alguns comandos cURL para v2 são diferentes de v1.

<Callout variant="tip">
  O agente New Relic usa APIs diferentes e não é acessível por meio do API Explorer.
</Callout>
