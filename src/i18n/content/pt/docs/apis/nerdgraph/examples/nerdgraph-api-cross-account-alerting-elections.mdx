Para permitir o compartilhamento de dados entre contas dentro de uma organização, uma conta deve optar pelo recurso de &quot;alerta entre contas&quot;. Siga estas instruções sobre como usar o NerdGraph API Explorer para atualizar as configurações da conta ativando o alerta entre contas.

<Callout variant="important">
  Somente usuários com a permissão &quot;Controle de acesso de compartilhamento de dados&quot; podem atualizar eleições entre contas para compartilhamento de dados. Por padrão, essa permissão é concedida apenas ao usuário com a função &quot;Gerente da organização&quot;.
</Callout>

## Etapas para atualizar uma eleição de alerta entre contas

<Steps>
  <Step>
    Colete a lista de IDs de conta para atualizar. A API aceita até **20 IDs de conta**.
  </Step>

  <Step>
    Indique se você deseja habilitar `true` ou desabilitar `false` a eleição.
  </Step>

  <Step>
    Use o explorador da API do NerdGraph para atualizar as eleições de conta.
  </Step>
</Steps>

Aqui está um exemplo da mutação:

```graphql
mutation {
  alertsUpdateCrossAccountElections(
    accountIds: [YOUR_1ST_ACCOUNT_ID, YOUR_2ND_ACCOUNT_ID, YOUR_3RD_ACCOUNT_ID],
    electionStatus: true
  ) {
    electionStatus
  }
}
```
