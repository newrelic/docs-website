Você pode listar e revogar URLs de gráficos ao vivo acessíveis ao público usando consultas e mutações no [NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph).

Para fazer isso:

1. Acesse o [GraphiQL Explorer](https://api.newrelic.com/graphiql).
2. [Liste todos os URLs de gráficos ao vivo](#list) criados em sua conta New Relic.
3. [Revogue qualquer URL de gráfico ao vivo que você desejar](#revoke).

<Callout variant="tip">
  Para criar URLs de gráficos ativos acessíveis ao público, consulte nossa [documentação](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#chart).
</Callout>

## Listar todos os URLs de gráficos ao vivo [#list]

Use a seguinte consulta para recuperar uma lista de URLs de gráficos dinâmicos existentes:

```graphql
{
  actor {
    dashboard {
      liveUrls {
        liveUrls {
          title
          url
          createdAt
          type
        }
        errors {
          description
        }
      }
    }
  }
}
```

## Revogar qualquer URL de gráfico ao vivo [#revoke]

Use a seguinte consulta para revogar o URL do gráfico ao vivo especificado:

```graphql
mutation {
  dashboardWidgetRevokeLiveUrl(uuid: "YOUR_LIVE_CHART_UUID") {
    uuid
    errors {
      description
      type
    }
  }
}
```
