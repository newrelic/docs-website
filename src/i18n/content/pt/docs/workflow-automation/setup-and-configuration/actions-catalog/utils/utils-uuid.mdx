Esta página fornece uma referência completa para utilitários - uuid disponíveis no catálogo de ações de automação de fluxo de trabalho. Essas ações permitem que você utilize utilitários de geração de UUID.

## Gerar um UUID [#utils-uuid-generate]

Gere um UUID V4 compatível com a RFC. Utiliza o gerador de números pseudoaleatórios (PRNG) determinístico do fluxo de trabalho temporal, tornando-o seguro para uso dentro de um fluxo de trabalho. Consulte o [namespace: fluxo de trabalho | Referência API do SDK do TypeScript temporal](https://typescript.temporal.io/api/namespaces/workflow?&_ga=2.223881945.39829224.1686648936-167080156.1680123165#uuid4) para obter mais informações.
