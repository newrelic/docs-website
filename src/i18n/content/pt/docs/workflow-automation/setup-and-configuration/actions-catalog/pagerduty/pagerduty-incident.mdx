Esta página fornece uma referência completa para as ações de incidentes do PagerDuty disponíveis no catálogo de ações de automação de fluxo de trabalho. Essas ações permitem que você gerencie incidentes do PagerDuty.

## Pré-requisitos

Antes de usar ações PagerDuty na automação do fluxo de trabalho, certifique-se de ter:

* Uma conta PagerDuty com as permissões adequadas.
* Um token da API do PagerDuty (seja um token de usuário ou um token de conta).
* As permissões necessárias para gerenciar incidentes em sua conta PagerDuty.
* Para obter o token da conta, você precisará fornecer o endereço de e-mail `from` nas ações relevantes.
