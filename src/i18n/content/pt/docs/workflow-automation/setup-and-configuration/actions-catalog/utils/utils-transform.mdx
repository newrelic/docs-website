Esta página fornece uma referência completa para as ferramentas de transformação disponíveis no catálogo de ações de automação de fluxo de trabalho. Essas ações permitem que você utilize utilitários de transformação de dados.

## Transformar um valor de dados de entrada em um formato CSV [#utils-transform-toCSV]

Esta ferramenta é utilizada para transformar vários tipos de entrada (JSON, mapa) em formato CSV.

              ```yaml
                name: nrqltocsv

                steps:
                  - name: queryForLog
                    type: action
                    action: newrelic.nrql.query
                    version: 1
                    inputs:
                      accountIds: ['${{ .workflowInputs.accountId }}']
                      query: ${{ .workflowInputs.nrql }}

                  - name: csv1
                    type: action
                    action: utils.transform.toCSV
                    version: 1
                    inputs:
                      data: ${{ .steps.queryForLog.outputs.results | tostring }}
                      selectors:
                        - name: csv
                          expression: ".csv"
              ```
