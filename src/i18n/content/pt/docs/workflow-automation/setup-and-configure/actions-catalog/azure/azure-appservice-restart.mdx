As ações do Azure App Service permitem gerenciar seus aplicativos da web diretamente de seus fluxos de trabalho.

## Reiniciar um App Service [#azure-appservice-restart]

Esta ação reinicia um App Service do Azure usando a [API REST Web Apps - Restart](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/restart?view=rest-appservice-2026-03-15).

    <TabsPageItem id="azureappservicerestartexample">
      ```yaml
      name: restartAzureWebService
      description: 'This workflow restarts an Azure App Service'
      steps:
        - name: azure_appService_restart_1
          type: action
          action: azure.appService.restart
          version: 1
          inputs:
            clientId: ${{ :secrets:azure-client-id }}
            clientSecret: ${{ :secrets:azure-client-secret }}
            tenantId: ${{ :secrets:azure-client-tenant }}
            subscriptionId: "12345678-1234-1234-1234-123456789abc"
            resourceGroupName: "my-resource-group"
            appServiceName: "my-web-app"
            softRestart: true
            synchronous: true
          next: end
      ```
