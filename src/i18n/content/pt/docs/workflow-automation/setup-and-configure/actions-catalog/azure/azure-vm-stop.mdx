As ações da máquina virtual do Azure (VM) permitem que você gerencie o ciclo de vida da sua VM do Azure diretamente do seu fluxo de trabalho.

## Parar uma máquina virtual [#azure-vm-stop]

Esta ação para uma VM do Azure usando a API REST [Azure Compute VM - Power Off](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines/power-off?view=rest-compute-2025-11-01&tabs=HTTP).

    <TabsPageItem id="azurevmstopexample">
      ```yaml
      name: azure-stop-vm
      description: 'This stops a virtual machine'
      steps:
        - name: azure_vm_stop_3
          type: action
          action: azure.vm.stop
          version: 1
          inputs:
            clientId: ${{ :secrets:azure-client-id }}
            clientSecret: ${{ :secrets:azure-client-secret }}
            tenantId: ${{ :secrets:azure-client-tenant }}
            subscriptionId: "12345678-1234-1234-1234-123456789abc"
            resourceGroupName: "my-resource-group"
            virtualMachineName: ‘my-demo-vm'
          next: end
      ```
