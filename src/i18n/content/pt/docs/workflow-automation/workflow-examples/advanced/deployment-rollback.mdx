Monitore a integridade da entidade New Relic e acione notificações de reversão via AWS SQS ou webhooks HTTP quando problemas forem detectados.

**Requisitos:**

* Monitoramento de entidade New Relic
* (Opcional) Fila AWS SQS ou endpoint de webhook HTTP

**O que este fluxo de trabalho faz:**

* Monitorar a gravidade do alerta da entidade por uma duração especificada (padrão de 30 minutos)
* Verificar a integridade da entidade a cada minuto
* Registrar status não íntegro se a entidade se tornar CRITICAL ou WARNING
* Enviar notificações de reversão do AWS SQS (se configurado) com detalhes da entidade
* Enviar notificações de reversão de webhook HTTP (se configurado)
* Registrar sucesso se a entidade permanecer íntegra por toda a duração
* Parar o monitoramento se a entidade não for encontrada

**Ações principais**: `newrelic.nerdgraph.execute`, `newrelic.ingest.sendLogs`, `aws.execute.api`, `(sqs.send_message)`, `http.post`
