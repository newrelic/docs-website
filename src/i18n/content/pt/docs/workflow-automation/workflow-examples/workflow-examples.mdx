Cenários comuns de automação para Workflow Automation, progredindo de padrões básicos a avançados. Use como pontos de partida ou explore a [biblioteca de modelos](/docs/workflow-automation/create-a-workflow-automation/use-a-template) para soluções prontas para implantação.

## Modelo disponível de fluxo de trabalho

Acesse os modelos diretamente na interface do usuário do New Relic Workflow Automation:

1. Acesse **All Capabilities &gt; Workflow Automation**
2. Clique em **Create workflow**
3. Selecione **Use a template**
4. Navegar pela biblioteca de modelos e escolher um fluxo de trabalho que corresponda ao seu caso de uso

Os modelos incluem:

* Etapas e lógica de fluxo de trabalho pré-configuradas
* Exemplo de entrada (parâmetro)
* Credenciais e integração necessárias
* Documentação dos resultados esperados

Use modelos como estão ou personalize-os para atender a requisitos específicos.
