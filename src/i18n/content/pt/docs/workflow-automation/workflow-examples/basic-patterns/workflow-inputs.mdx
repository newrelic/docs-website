Use `${{ .workflowInputs.variableName }}` para passar valores dinâmicos em tempo de execução em URLs de endpoint e outros campos de entrada.

**O que essa abordagem faz:**

* Aceitar valores de entrada dinâmicos quando o fluxo de trabalho é acionado
* Faça referência aos valores de entrada usando a sintaxe de modelo em todas as etapas do fluxo de trabalho
* Habilite a reutilização do fluxo de trabalho com parâmetros diferentes

## Exemplo

```json
  {
    "inputs": [
      {
        "key": "urlParams",
        "value": "{\"filter\": \"active\"}"
      },
      {
        "key": "headers",
        "value": "{\"Api-Key\": \"your-api-key\"}"
      }
    ]
  }
```
