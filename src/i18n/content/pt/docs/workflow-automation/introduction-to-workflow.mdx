A Automação de fluxo de trabalho automatiza a resposta a incidentes e o gerenciamento de infraestrutura sem código. Crie fluxos de trabalho que respondam a alertas, integrem-se com ferramentas como AWS e Slack e executem tarefas operacionais automaticamente.

## Requisitos

Para usar Workflows, você precisa:

* Uma conta New Relic
* Plano de preços do Advanced Compute (aCCU)

<Callout variant="important">
  A Automação de fluxo de trabalho e o gerenciamento de segredos estão incluídos no Advanced Compute (aCCU). Se você não tiver certeza se possui aCCU, verifique as [configurações da sua conta](https://one.newrelic.com/admin-portal/plan-info/home) ou entre em contato com seu representante de conta.
</Callout>

## Principais benefícios

* **Resposta mais rápida a incidentes**: redimensione automaticamente instâncias do EC2, reverta implantação ou reinicie serviços quando alertas dispararem – sem necessidade de intervenção manual.
* **Conecte suas ferramentas**: Integre o New Relic com AWS, Slack, Jira, ServiceNow e qualquer API HTTP para enviar dados de alerta enriquecidos para onde sua equipe trabalha.
* **Adicionar pontos de aprovação**: Incluir supervisão humana no workflow crítico — solicitar a confirmação da equipe antes de expandir a infraestrutura de produção ou executar reversões.
* **Lide com a complexidade**: Use lógica condicional, loops e transformações de dados para automatizar seus processos operacionais exclusivos.

## Acionar fluxos de trabalho

Você pode iniciar fluxos de trabalho de três maneiras:

* **A partir de alertas**: Inicie fluxos de trabalho automaticamente quando as condições de alerta forem violadas
* **Sob demanda**: Acione manualmente o fluxo de trabalho via API quando necessário.
* **Em conformidade com o cronograma**: Execute o fluxo de trabalho automaticamente em horários específicos usando expressões cron.

Para fluxos de trabalho acionados por alertas, consulte [Enviar notificações de fluxos de trabalho](/docs/workflow-automation/setup-and-configure/create-destinations#trigger-from-alerts).
