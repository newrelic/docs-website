## Exemplo

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedules {
              results {
                definition {
                  name
                  scope {
                    id
                    type
                  }
                  version
                }
                schedule {
                  id
                }
              }
            }
          }
        }
      }
    }
```

## Resposta

A consulta retorna:

* `schedules.results`: Uma matriz de objetos de agendamento, cada um contendo:

  * `definition.name`: O nome da definição do fluxo de trabalho
  * `definition.scope.id`: O ID da conta
  * `definition.scope.type`: O tipo de escopo
  * `definition.version`: A versão da definição do fluxo de trabalho
  * `schedule.id`: O identificador exclusivo do agendamento
