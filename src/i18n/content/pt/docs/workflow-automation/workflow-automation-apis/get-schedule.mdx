## Solicitação de amostra

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## Resposta de amostra

A consulta retorna:

* `schedule.id`: O identificador exclusivo do agendamento.
