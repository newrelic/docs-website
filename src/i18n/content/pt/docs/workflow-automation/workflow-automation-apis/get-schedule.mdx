## Exemplo [#example]

```graphql
    {
      actor {
        account(id: 11933347) {
          workflowAutomation {
            schedule(id: "schedule-id") {
              id
            }
          }
        }
      }
    }
```

## Resposta

A consulta retorna:

* `schedule.id`: O identificador exclusivo do agendamento.
