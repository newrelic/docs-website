Esta API atualiza uma definição de fluxo de trabalho existente. Ela retorna um erro se a definição do fluxo de trabalho ainda não existir.

## Solicitação de amostra

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```

## Resposta de amostra

A mutação retorna:

* `definition.name`: O nome da definição do fluxo de trabalho atualizada
* `definition.version`: O novo número da versão da definição do fluxo de trabalho
* `definition.yaml`: O YAML da definição do fluxo de trabalho codificado em base64
