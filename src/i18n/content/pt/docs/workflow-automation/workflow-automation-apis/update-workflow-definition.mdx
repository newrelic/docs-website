Utilizado para atualizar uma definição de fluxo de trabalho existente. Retorna um erro se a definição do fluxo de trabalho ainda não existir.

## Exemplo

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```
