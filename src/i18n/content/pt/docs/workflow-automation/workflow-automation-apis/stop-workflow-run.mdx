## Exemplo

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```

## Resposta

A mutação retorna:

* `runId`: O identificador exclusivo da execução do fluxo de trabalho interrompido. Retorna um `runId` vazio se o fluxo de trabalho não for encontrado.
