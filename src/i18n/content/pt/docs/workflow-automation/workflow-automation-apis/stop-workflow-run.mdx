Isso interrompe um fluxo de trabalho em execução. Retorna um valor vazio `runId` se o fluxo de trabalho não for encontrado.

## Exemplo

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```
