Esta API obtém e retorna uma definição de fluxo de trabalho.

## Exemplo

Utilize o [criador de consulta](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) para criar e editar um fluxo de trabalho. Consulte [o explorador de API do NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflow(name: "my-workflow", version: 1) {
          definition {
            name
            version
            yaml
          }
        }
      }
    }
  }
}
```
