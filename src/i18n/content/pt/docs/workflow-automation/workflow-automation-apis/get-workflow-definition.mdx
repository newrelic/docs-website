## Solicitação de amostra

Utilize o [criador de consulta](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) para criar e editar um fluxo de trabalho. Consulte [o explorador de API do NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflow(name: "my-workflow", version: 1) {
          definition {
            name
            version
            yaml
          }
        }
      }
    }
  }
}
```

## Resposta de amostra

A consulta retorna:

* `definition.name`: O nome da definição do fluxo de trabalho
* `definition.version`: O número da versão da definição do fluxo de trabalho
* `definition.yaml`: O YAML da definição do fluxo de trabalho codificado em base64
