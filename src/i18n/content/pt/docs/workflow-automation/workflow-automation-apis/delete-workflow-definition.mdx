Esta API exclui todas as versões de definição de fluxo de trabalho com o nome especificado.

<Callout variant="important">
  * Para excluir uma definição, não deve haver nenhuma programação que utilize essa definição. Se a definição estiver sendo usada por um agendamento, a exclusão da definição falhará e retornará um erro.
  * Todas as execuções de fluxo de trabalho em andamento continuarão até a conclusão, mesmo que a definição seja excluída.
</Callout>

## Solicitação de amostra

```graphql

  workflowAutomationDeleteWorkflowDefinition(
    definition: {name: "my-workflow"}
    scope: {id: "12345678", type: ACCOUNT}
  ) {
    definition {
      name
    }
  }
}
```
