Esta API obtém e retorna uma lista de todos os contornos de definição de fluxo de trabalho sob o escopo fornecido.

<Callout variant="important">
  O YAML de definição de fluxo de trabalho não está disponível com esta consulta. Um YAML de definição de fluxo de trabalho só é retornado usando uma consulta `GetWorkflowDefinition` `/{actor/account(:id)/workflowAutomation/workflow(:name, :version)}`.
</Callout>

## Exemplo

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflows(cursor: "xyz123") {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
          }
        }
      }
    }
  }
}
```

## Resposta

A consulta retorna:

* `workflows.results`: Uma matriz de contornos de definição de fluxo de trabalho, cada um contendo:

  * `definition.name`: O nome da definição do fluxo de trabalho
  * `definition.scope.id`: O ID da conta
  * `definition.scope.type`: O tipo de escopo
  * `definition.version`: O número da versão da definição do fluxo de trabalho
