Retorna todas as definições de ação que podem ser usadas para compor uma definição de fluxo de trabalho.

## Solicitação de amostra

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        actionDefinitions {
          results {
            description
            inputs {
              name
              required
              secretAllowed
            }
            name
            outputs {
              name
            }
            timeout {
              action
            }
            version
          }
        }
      }
    }
  }
}
```

## Resposta de amostra

A consulta retorna:

* `actionDefinitions.results`: Uma matriz de definições de ação, cada uma contendo:

  * `name`: O nome da ação (por exemplo, `aws.ec2.rebootInstances`)
  * `description`: O que a ação faz
  * `version`: A versão da ação
  * `inputs`: Matriz de parâmetros de entrada com as propriedades `name`, `required` e `secretAllowed`
  * `outputs`: Matriz de campos de saída com a propriedade `name`
  * `timeout.action`: Tempo limite em segundos
