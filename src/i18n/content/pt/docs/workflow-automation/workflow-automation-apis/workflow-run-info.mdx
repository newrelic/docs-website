Esta API obtém e retorna o status de um fluxo de trabalho para um runId de fluxo de trabalho específico.

## Exemplo

Utilize o [criador de consulta](https://one.newrelic.com/nerdgraph-graphiql?state=03119d23-7b51-7b7a-1c99-d2cd026300e5) para criar e editar um fluxo de trabalho. Consulte [o explorador de API do NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflowRunInfo(runId: "8911c7e4-5a1f-4316-90c8-67adde72dd1c") {
          logUrl
          status
        }
      }
    }
  }
}
```
