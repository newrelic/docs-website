A integração Agentic da New Relic fornece uma solução orientada por IA para automatizar e otimizar a observabilidade do fluxo de trabalho, melhorando o gerenciamento e monitoramento do sistema. Essa integração facilita operações contínuas ao automatizar tarefas repetitivas por meio de agente de IA, reduzindo a intervenção manual. Ao integrar-se perfeitamente com plataformas como GitHub Copilot, Amazon Q, Gemini Code Assist e ServiceNow, a integração Agentic transforma fluxos de trabalho complexos em operações contínuas.

## Recurso chave [#features]

* **Automação inteligente:** aproveite a IA para automatizar e agilizar o fluxo de trabalho complexo, reduzindo a necessidade de troca manual de ferramentas e aumentando a produtividade.
* **Observabilidade aprimorada:** obtenha insights abrangentes sobre seus sistemas, permitindo detecção, análise e resolução mais rápidas de incidentes.
* **Integração perfeita:** conecte-se a plataformas populares como GitHub Copilot para trazer análises e insights poderosos diretamente para seu fluxo de trabalho existente.

## Integração atual [#integrations]

### Copiloto do GitHub [#github-copilot]

Integre os insights do New Relic diretamente ao seu IDE com o GitHub Copilot. Você pode usar o Agente AI para acessar logs, erros e recomendações sem sair do seu ambiente de codificação. Ele fornece insights reais de tempo à medida que eles desenvolvem e implantam o código.

Esta integração oferece:

* Verificações de implantação automatizadas.
* alertas de tempo real e monitoramento de erros.
* Recomendações probabilísticas para ações de implantação.

Para obter mais informações, consulte [Introdução ao New Relic agente para GitHub Copilot](/docs/agentic-ai/github-copilot-integration).

## Ferramentas [#tools]

Cada integração de agente vem com várias ferramentas para usar os recursos da plataforma New Relic e fornecer insights de observabilidade, alertas e previsões:

* **[Ferramenta de inteligência de mudança:](/docs/agentic-ai/change-intelligence-tool)** a ferramenta de inteligência de mudança ajuda você a entender e mitigar riscos potenciais, fornecendo análises e previsões abrangentes de mudanças. Você pode usar esta ferramenta para obter insights instantâneos sobre o impacto das mudanças no sistema. Ajuda a avaliar os riscos potenciais associados à implantação e modificações de configuração. Para obter mais informações, consulte [Change Intelligence Tool](/docs/agentic-ai/change-intelligence-tool).

* **[Alert Intelligence Tool](/docs/agentic-ai/alert-intelligence-tool):** A Alert Intelligence Tool oferece análise avançada de alerta para resolver incidentes rapidamente. Ele fornece notificação em tempo real e recomendações imediatas para evitar escalada. Para mais informações, consulte [Alert Intelligence Tool](/docs/agentic-ai/alert-intelligence-tool).

## Preços [#pricing]

O uso do Agentic integração será cobrado durante a visualização de acordo com seu pedido como um produto de computador avançado, independentemente do tipo de usuário. Para recursos de produtos de computação avançada, a New Relic consulta Language consulta (NRQLS) relacionada também é faturável como computação avançada. A New Relic pretende lançar o Agentic integração como um recurso de produto de computação avançada assim que estiver disponível para o público em geral.

Para mais detalhes sobre preços, entre em contato com nossa equipe de vendas ou consulte seu pedido.
