A ferramenta New Relic para a extensão Gemini Code Assist do Google fornece insights inteligentes e orientações personalizadas para otimizar o desempenho do aplicativo, monitorar a infraestrutura e gerenciar a implantação — tudo perfeitamente integrado ao Gemini Code Assist.

## Habilitar a ferramenta New Relic no Gemini Code Assist [#enable-new-relic-tool]

### Pré-requisitos [#prerequisites]

Antes de começar, certifique-se de que os seguintes pré-requisitos sejam atendidos para garantir um processo de configuração tranquilo:

* Um ambiente de desenvolvimento integrado (IDE) está instalado no seu computador.
* Você tem acesso ao Google Gemini Code Assist.
* O Gemini Code Assist está instalado no seu IDE. Para obter mais informações, consulte [a documentação do Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="dica">
  Se você quiser restringir o acesso ao Gemini Code Assist para determinadas funções, configure as configurações no Gemini Code Assist. Para restringir o acesso ao New Relic AI para funções ou funcionários específicos, você pode usar o Controle de Acesso Baseado em Função (RBAC) do New Relic. Para obter mais informações, consulte [Gerenciamento de usuáriosNew Relic ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

Para habilitar a ferramenta New Relic:

1. No seu IDE, faça login no [Gemini Code Assist](https://codeassist.google.com) clicando em Fazer login com o Google.

2. Após a autenticação bem-sucedida, retorne ao seu IDE.

3. No <DNT>Gemini Code Assist Chat</DNT> (chat), digite `@newrelic` e pressione enter. Você será solicitado a habilitar a ferramenta New Relic .

4. No prompt, clique no link para habilitar a ferramenta New Relic. Uma janela do browser será aberta, na qual você será solicitado a habilitar a ferramenta New Relic no Gemini Code Assist.

5. Habilite a ferramenta New Relic e clique em **Connect**.

6. Efetue login usando os detalhes da sua conta New Relic. Isso conecta com sucesso sua conta New Relic ao Gemini Code Assist.

   Agora, você pode usar a ferramenta New Relic dentro da extensão de bate-papo Gemini Code Assist no seu IDE.

## Interaja com a ferramenta New Relic [#new-relic-tool]

A ferramenta New Relic dentro do Gemini Code Assist pode ajudar em uma variedade de tarefas e oferecer insights valiosos. Para interagir com a ferramenta New Relic, na caixa de entrada do bate-papo, digite seu prompt da seguinte maneira:

`@newrelic <your message>`

Onde, `<your message>` é a ação que você deseja que a ferramenta execute. Por exemplo, `@newrelic What is apdex?`
