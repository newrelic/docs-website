## Instalar o agente New Relic no Gemini Code Assist [#install-new-relic-agent]

### Pré-requisitos [#prerequisites]

Antes de começar, certifique-se de que os seguintes pré-requisitos sejam atendidos para garantir um processo de configuração tranquilo:

* Um IDE está instalado no seu sistema.
* Você tem acesso ao Google Gemini Code Assist.
* O Gemini Code Assist está instalado no seu IDE. Para obter mais informações, consulte [a documentação do Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="dica">
  Se você quiser restringir o acesso ao Gemini Code Assist para determinadas funções, configure as configurações no Gemini Code Assist. Para restringir o acesso ao New Relic AI para funções ou funcionários específicos, você pode usar o Controle de Acesso Baseado em Função (RBAC) do New Relic. Para obter mais informações, consulte [Gerenciamento de usuáriosNew Relic ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

**Para habilitar o agente New Relic no Gemini Code Assist:**

1. No seu <DNT>Google Cloud Console</DNT>, procure por <DNT>**NewRelic**</DNT> e ative a extensão. Uma janela de diálogo de login é aberta.

2. Forneça sua credencial do New Relic para fazer login. O mensagem **New Relic successfully linked.** é exibida.

   Agora, você pode usar o agente New Relic dentro da extensão de bate-papo Gemini Code Assist no seu IDE. Para obter mais informações, consulte [Como usar o agente New Relic no Gemini Code Assist](#communicate-with-new-relic-agent-in-gemini-code-assist).

## Comunique-se com o agente New Relic no Gemini Code Assist [#communicate-with-new-relic-agent-in-gemini-code-assist]

O agente New Relic do Gemini Code Assist pode ajudar com uma variedade de tarefas e oferecer insights valiosos. Para interagir com o agente da New Relic, na caixa de entrada do bate-papo, digite `@newrelic`.

### Estrutura da mensagem de bate-papo [#chat-message-structure]

Para interagir com o agente New Relic, insira seu prompt da seguinte forma:

`@newrelic <your message>`

Onde:

* `@newrelic`: interação com o agente New Relic .
* `<your message>`: Ação a ser submetida ao agente para processamento.
