A extensão do agente New Relic para GitHub Copilot fornece insights inteligentes e orientações personalizadas para otimizar o desempenho do aplicativo, monitorar a infraestrutura e gerenciar a implantação – tudo perfeitamente integrado ao GitHub Copilot.

Este guia orientará você no processo de instalação e uso do agente New Relic na extensão GitHub Copilot.

## Instale o agente New Relic no GitHub Copilot [#install-new-relic-agent]

### Pré-requisitos [#prerequisites]

Antes de começar, certifique-se de que os seguintes pré-requisitos sejam atendidos para garantir um processo de configuração tranquilo:

* Um IDE está instalado no seu sistema.
* Você tem acesso ao [GitHub Copilot](https://github.com/features/copilot).
* A extensão GitHub Copilot está instalada no seu IDE. Para obter mais informações, consulte [a documentação do GitHub Copilot](https://docs.github.com/en/copilot/managing-copilot/configure-personal-settings/installing-the-github-copilot-extension-in-your-environment).
* A [política de extensão do GitHub Copilot](https://docs.github.com/en/enterprise-cloud@latest/copilot/managing-copilot/managing-github-copilot-in-your-organization/managing-policies-for-copilot-in-your-organization#setting-a-policy-for-github-copilot-extensions-in-your-organization) está habilitada para sua organização.

<Callout variant="tip" title="dica">
  Se você quiser restringir o acesso ao GitHub Copilot para determinadas funções, configure as configurações no GitHub. Para restringir o acesso ao New Relic AI para funções ou funcionários específicos, você pode usar o Controle de Acesso Baseado em Função (RBAC) do New Relic. Para obter mais informações, consulte [Gerenciamento de usuários New Relic](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

**Para instalar o agente New Relic no GitHub Copilot:**

1. No GitHub MarketPlace, acesse [New Relic agente](https://github.com/marketplace/newrelic-copilot-extension).
2. Clique em **Add &gt; Install it for free**.

Após a instalação do agente, você poderá usá-lo na extensão GitHub Chat no seu IDE. Para mais informações, consulte [o uso do agente New Relic no Copilot](#communicate-with-new-relic-agent-in-gitHub-copilot).

## Comunique-se com o agente da New Relic no GitHub Copilot [#communicate-with-new-relic-agent-in-gitHub-copilot]

O agente New Relic no Github Copilot pode ajudar com uma variedade de tarefas e oferecer insights valiosos. Para interagir com o agente da New Relic, na caixa de entrada do bate-papo do Copiloto, digite `@newrelic`.

### Entre no agente New Relic [#sign-in]

Ao iniciar uma conversa pela primeira vez com um agente da New Relic, você será solicitado a fazer login. Siga as instruções para concluir o processo de login.

### Estrutura da mensagem de bate-papo [#chat-message-structure]

Para interagir com o agente da New Relic, digite seu prompt da seguinte forma:

`@newrelic <your message>`

Onde:

* `@newrelic`: interação com o agente New Relic.
* `<your message>`: Ação a ser submetida ao agente para processamento.
