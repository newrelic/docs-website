A integração New Relic AI com o Now Assist da ServiceNow ajudará os usuários da ServiceNow — como operadores, socorristas e agentes de suporte — em:

* Avaliar a gravidade de um incidente através da compreensão do seu impacto em outros serviços e usuários
* Identificar teorias de causa raiz prováveis para um incidente
* Facilitar a escalada para os colaboradores para mitigar o problema

Este guia orientará você no processo de instalação e uso do New Relic AI no Now Assist do ServiceNow.

## Configurar [#setup]

Para usar a extensão New Relic AI com o ServiceNow, você precisará:

* Uma [chave de API de usuário](/docs/apis/intro-apis/new-relic-api-keys/#overview-keys) do New Relic. Se um usuário do New Relic for excluído do New Relic, suas chaves de usuário também serão desativadas e não funcionarão mais. Para garantir uma integração segura e ininterrupta, observe as seguintes recomendações:

  * Use uma conta `service user` para evitar interrupções quando uma conta pessoal ou individual for encerrada.
  * Embora `user keys` forneça acesso a várias contas, elas estão vinculadas a uma única conta específica: a conta da qual foram criadas. Se uma conta for excluída, todas as chaves de usuário associadas a ela se tornarão inválidas. Portanto, certifique-se de que a conta para a qual você cria a chave tenha vida útil longa.
  * Verifique se o usuário para quem a chave de usuário é usada possui a permissão Somente leitura.

  <Callout variant="tip" title="Observação">
    Esteja ciente de que a chave de usuário fornecida permitirá que todos os usuários da sua conta ServiceNow acessem esta integração.
  </Callout>

  * Para configurar o New Relic AI no Now Assist, siga as etapas descritas [aqui.](https://www.servicenow.com/docs/csh?topicname=now-assist-itom-use-aia.html&version=latest)

## Interaja com a New Relic AI através do Now Assist [#new-relic-tool]

No ServiceNow:

1. Navegue até <DNT>**All &gt; Event Management &gt; Express List**</DNT>.

2. Selecione um alerta, certificando-se de que a fonte seja New Relic.

3. Clique no ícone <DNT>Now Assist</DNT> para abrir o painel.

4. Pergunte agora Faça uma pergunta sobre o alerta.

5. Digite qualquer um dos prompts suportados, você também pode misturar e combinar:

   * Qual é a gravidade deste alerta?
   * Quais entidades são impactadas?
   * O que são teorias de causa provável?
   * Quem devo envolver na investigação?
   * Há implantação recente?
   * Qual é o impacto do alerta, teorias de causa provável e quem envolver na investigação?

   <Callout variant="tip" title="dica">
     Sua conversa no painel do Now Assist é específica para o alerta que você selecionou na Etapa 2. Para perguntar sobre um alerta diferente, abra esse alerta e selecione o ícone do Now Assist para iniciar uma nova conversa.
   </Callout>
