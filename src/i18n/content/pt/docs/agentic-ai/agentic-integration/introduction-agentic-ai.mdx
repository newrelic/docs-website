New Relic&apos;s agentic integration provides an AI-driven solution to help automate and optimize specific observability workflows, enhancing system management and monitoring. This integration facilitates seamless operations by automating repetitive tasks through AI agents, reducing manual intervention. By integrating seamlessly with platforms such as GitHub Copilot, Gemini Code Assist, and ServiceNow, agentic integrations transform complex workflows into seamless operations.

## Recurso chave [#features]

* **Intelligent automation:** Leverage AI to automate and streamline complex workflows, reducing the need for manual tool-switching and enhancing productivity, while keeping a human in the loop.
* **Observabilidade aprimorada:** obtenha insights abrangentes sobre seus sistemas, permitindo detecção, análise e resolução mais rápidas de incidentes.
* **Integração perfeita:** conecte-se a plataformas populares como GitHub Copilot para trazer análises e insights poderosos diretamente para seu fluxo de trabalho existente.

## Acesso e gerenciamento de dados [#access-data-mgmt]

Nossos parceiros de integração gerenciarão o aplicativo cliente e manterão o token do usuário quando você ou qualquer um de seus usuários se autenticar na New Relic por meio da integração do parceiro. O parceiro terceirizado terá o mesmo nível de acesso ao New Relic que o usuário que se autenticou pelo período de tempo configurado nas configurações do usuário, pois o token é usado em seu nome. Esse arranjo ajuda a fornecer integração e funcionalidade perfeitas. Para mais informações, consulte a documentação do parceiro, que detalha o uso token e as práticas de gerenciamento.

### Controle de acesso baseado em função (RBAC)

New Relic&apos;s agentic integration capabilities are granted via organization-scoped roles within New Relic&apos;s RBAC system. Users gain access by belonging to a group with an assigned organizational role that includes these capabilities. Individual accounts do not require direct configuration.

Currently, RBAC can be accessed through API interfaces. If you are interested in utilizing RBAC via API, feel free to contact us at [agenticai@newrelic.com](mailto:agenticai@newrelic.com). For detailed guidance on setting up RBAC, refer to our [account user management tutorial](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial).

## Integração atual [#integrations]

### Copiloto do GitHub [#github-copilot]

Integrate New Relic insights directly into your IDE with GitHub Copilot. You can use Agent AI to access logs, errors, and recommendations without leaving your coding environment. It provides real-time insights as you develop and deploy the code.

Esta integração oferece:

* Verificações de implantação automatizadas: realize verificações de implantação automaticamente, aproveitando os dados do New Relic.
* alertas tempo real e monitoramento de erros: Mantenha-se informado com alertas imediatos e rastreie os erros conforme eles ocorrem.
* Recomendações probabilísticas para ações de implantação.

For more information, refer to [Getting Started with New Relic agent for GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration).

### ServiceNow Now Assist [#now-assist]

New Relic AI insights are now right inside ServiceNow&apos;s Now Assist. This integration helps your operations teams, such as first responders and support agents, quickly understand incident severity, identify probable root causes, and streamline collaboration, all without leaving their ServiceNow workspace.

Para obter mais informações, consulte [Introdução ao agente New Relic para ServiceNow Now Assist](/docs/agentic-ai/agentic-integration/servicenow-integration).

Integre os insights do New Relic ao seu IDE com o Gemini Code Assist. Esta ferramenta permite que os desenvolvedores acessem logs relevantes, informações de erros e recomendações dentro do ambiente de codificação, oferecendo insights que podem informar o processo de desenvolvimento e implantação.

Esta integração oferece:

* Monitoramento de implantação automatizada: Monitore automaticamente a implantação usando dados de tempo real da New Relic para identificar e resolver problemas de forma eficiente.
* alertas e monitoramento de erros: Receba notificações imediatas e rastreie erros de forma eficaz para manter o desempenho e a estabilidade do aplicativo.
* Recomendações para ações de implantação: utilize recomendações baseadas na análise de dados para melhorar o desempenho do código e dar suporte à tomada de decisões.

Para obter mais informações, consulte [Introdução ao agente New Relic para Gemini Code Assist](/docs/agentic-ai/agentic-integration/gemini-integration).
