A integração agente da New Relic fornece uma solução orientada por IA para automatizar e otimizar a observabilidade do fluxo de trabalho, melhorando o gerenciamento e monitoramento do sistema. Essa integração facilita operações contínuas ao automatizar tarefas repetitivas por meio de agente de IA, reduzindo a intervenção manual. Ao integrar-se perfeitamente com plataformas como GitHub Copilot, Gemini Code Assist e ServiceNow, a integração agente transforma fluxos de trabalho complexos em operações contínuas.

## Recurso chave [#features]

* **Automação inteligente:** aproveite a IA para automatizar e agilizar o fluxo de trabalho complexo, reduzindo a necessidade de troca manual de ferramentas e aumentando a produtividade.
* **Observabilidade aprimorada:** obtenha insights abrangentes sobre seus sistemas, permitindo detecção, análise e resolução mais rápidas de incidentes.
* **Integração perfeita:** conecte-se a plataformas populares como GitHub Copilot para trazer análises e insights poderosos diretamente para seu fluxo de trabalho existente.

## Acesso e gerenciamento de dados [#access-data-mgmt]

Our integration partners will manage client applications and maintain user tokens when you or any of your users authenticate to New Relic through the partner&apos;s integration. The third party partner will have the same level of access to New Relic as the user who authenticated for the amount of time configured in the user settings, as the tokens are used on their behalf. This arrangement helps to provide seamless integration and functionality. For more information, you can refer to the partner&apos;s documentation which details their token usage and management practices.

## Integração atual [#integrations]

### Copiloto do GitHub [#github-copilot]

Integre os insights do New Relic diretamente ao seu IDE com o GitHub Copilot. Você pode usar o Agente AI para acessar logs, erros e recomendações sem sair do seu ambiente de codificação. Ele fornece insights reais de tempo à medida que eles desenvolvem e implantam o código.

Esta integração oferece:

* Verificações de implantação automatizadas: realize verificações de implantação automaticamente, aproveitando os dados do New Relic.
* alertas tempo real e monitoramento de erros: Mantenha-se informado com alertas imediatos e rastreie os erros conforme eles ocorrem.
* Recomendações probabilísticas para ações de implantação.

Para obter mais informações, consulte [Introdução ao agente New Relic para GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration).

### ServiceNow Now Assist [#now-assist]

New Relic AI insights agora estão dentro do Now Assist da ServiceNow. Essa integração ajuda suas equipes de operações — como socorristas e agentes de suporte — a entender rapidamente a gravidade do incidente, identificar prováveis causas raiz e otimizar a colaboração, tudo isso sem sair do espaço de trabalho do ServiceNow.

Para obter mais informações, consulte [Introdução ao agente New Relic para ServiceNow Now Assist](/docs/agentic-ai/agentic-integration/servicenow-integration).

### Gemini Code Assist [#gemini-code-assist]

Integre os insights do New Relic ao seu IDE com o Gemini Code Assist. Esta ferramenta permite que os desenvolvedores acessem logs relevantes, informações de erros e recomendações dentro do ambiente de codificação, oferecendo insights que podem informar o processo de desenvolvimento e implantação.

Esta integração oferece:

* Monitoramento de implantação automatizada: Monitore automaticamente a implantação usando dados de tempo real da New Relic para identificar e resolver problemas de forma eficiente.
* alertas e monitoramento de erros: Receba notificações imediatas e rastreie erros de forma eficaz para manter o desempenho e a estabilidade do aplicativo.
* Recomendações para ações de implantação: utilize recomendações baseadas na análise de dados para melhorar o desempenho do código e dar suporte à tomada de decisões.

Para obter mais informações, consulte [Introdução ao agente New Relic para Gemini Code Assist](/docs/agentic-ai/agentic-integration/gemini-integration).
