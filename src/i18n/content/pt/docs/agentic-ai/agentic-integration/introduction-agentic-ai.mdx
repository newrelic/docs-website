A integração Agentic da New Relic fornece uma solução orientada por IA para automatizar e otimizar a observabilidade do fluxo de trabalho, melhorando o gerenciamento e monitoramento do sistema. Essa integração facilita operações contínuas ao automatizar tarefas repetitivas por meio de agente de IA, reduzindo a intervenção manual. Ao integrar-se perfeitamente com plataformas como GitHub Copilot, Amazon Q Business, Gemini Code Assist e ServiceNow, a Agentic integração transforma o complexo fluxo de trabalho em operações contínuas.

## Recurso chave [#features]

* **Automação inteligente:** aproveite a IA para automatizar e agilizar o fluxo de trabalho complexo, reduzindo a necessidade de troca manual de ferramentas e aumentando a produtividade.
* **Observabilidade aprimorada:** obtenha insights abrangentes sobre seus sistemas, permitindo detecção, análise e resolução mais rápidas de incidentes.
* **Integração perfeita:** conecte-se a plataformas populares como GitHub Copilot para trazer análises e insights poderosos diretamente para seu fluxo de trabalho existente.

## Acesso e gerenciamento de dados [#access-data-mgmt]

Ao se inscrever na prévia de integração do Agentic, esteja ciente de que seu usuário New Relic pode habilitar qualquer integração do Agentic disponibilizada pelo New Relic naquele momento. Se desejar limitar os usuários que têm acesso à integração do agente, você pode usar [funções e grupos personalizados](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial/#roles) para refinar o acesso.

Além disso, nossos parceiros de integração gerenciarão o aplicativo cliente e manterão o token do usuário quando você ou qualquer um de seus usuários se autenticar na New Relic por meio da integração do parceiro. O parceiro terceirizado terá o mesmo nível de acesso ao New Relic que o usuário que se autenticou pelo período de tempo configurado nas configurações do usuário, pois o token é usado em seu nome. Esse arranjo ajuda a fornecer integração e funcionalidade perfeitas. Para mais informações, consulte a documentação do parceiro, que detalha o uso token e as práticas de gerenciamento.

## Integração atual [#integrations]

### Copiloto do GitHub [#github-copilot]

Integre os insights do New Relic diretamente ao seu IDE com o GitHub Copilot. Você pode usar o Agente AI para acessar logs, erros e recomendações sem sair do seu ambiente de codificação. Ele fornece insights reais de tempo à medida que eles desenvolvem e implantam o código.

Esta integração oferece:

* Verificações de implantação automatizadas: realize verificações de implantação automaticamente, aproveitando os dados do New Relic.
* alertas tempo real e monitoramento de erros: Mantenha-se informado com alertas imediatos e rastreie os erros conforme eles ocorrem.
* Recomendações probabilísticas para ações de implantação.

Para obter mais informações, consulte [Introdução ao agente New Relic para GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration).

Ao permitir que o engenheiro de suporte diagnostique e resolva problemas rapidamente, essa integração ajuda a manter a integridade dos processos críticos da empresa, incluindo checkouts do usuário. Esta integração oferece: - Interface de IA conversacional: interaja com a New Relic AI por meio da interface de conversação do Amazon Q Business para obter insights e recomendações. - destino alerta Análise: isole e analise alertas específicos para serviços importantes, como checkout e processamento de pagamentos, dentro de um período definido. - Inteligência de alerta abrangente: acesse relatórios detalhados sobre anomalias, frequências de erros e preocupações com latência, complementados por links para o painel New Relic para análises mais detalhadas. - Recomendações proativas: receba sugestões informadas sobre a necessidade de investigação adicional para abordar preventivamente possíveis problemas. Para obter detalhes adicionais, consulte \[Conceitos básicos do agente New Relic para Amazon Q Business]\(/docs/agentic-ai/agentic-integração/amazonq-integração). \*/

### Gemini Code Assist [#gemini-code-assist]

Integre os insights do New Relic ao seu IDE com o Gemini Code Assist. Esta ferramenta permite que os desenvolvedores acessem logs relevantes, informações de erros e recomendações dentro do ambiente de codificação, oferecendo insights que podem informar o processo de desenvolvimento e implantação.

Esta integração oferece:

* Monitoramento de implantação automatizada: Monitore automaticamente a implantação usando dados de tempo real da New Relic para identificar e resolver problemas de forma eficiente.
* alertas e monitoramento de erros: Receba notificações imediatas e rastreie erros de forma eficaz para manter o desempenho e a estabilidade do aplicativo.
* Recomendações para ações de implantação: utilize recomendações baseadas na análise de dados para melhorar o desempenho do código e dar suporte à tomada de decisões.

Para obter mais informações, consulte [Introdução ao agente New Relic para Gemini Code Assist](/docs/agentic-ai/agentic-integration/gemini-integration).

## Ferramentas [#tools]

Cada integração de agente vem com várias ferramentas para usar os recursos da plataforma New Relic e fornecer insights de observabilidade, alertas e previsões:

* **[Ferramenta de inteligência de mudança:](/docs/agentic-ai/tools/change-intelligence-tool)** a ferramenta de inteligência de mudança ajuda você a entender e mitigar riscos potenciais, fornecendo análises e previsões abrangentes de mudanças. Você pode usar esta ferramenta para obter insights instantâneos sobre o impacto das mudanças no sistema. Ajuda a avaliar os riscos potenciais associados à implantação e modificações de configuração. Para obter mais informações, consulte [Change Intelligence Tool](/docs/agentic-ai/tools/change-intelligence-tool).

* **[Ferramenta de inteligência alerta](/docs/agentic-ai/tools/alert-intelligence-tool):** A ferramenta de inteligência alerta oferece análise avançada de alerta para resolver incidentes rapidamente. Ele fornece notificação em tempo real e recomendações imediatas para evitar escalada. Para mais informações, consulte [Alerta Intelligence Tool](/docs/agentic-ai/tools/alert-intelligence-tool).
