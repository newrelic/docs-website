<DNT>**Vulnerability Identifier:**</DNT> NR24-02

<DNT>**Priority:**</DNT> Alto

## Resumo

A New Relic aconselha todos os clientes que usam o New Relic Salesforce Exporter a atualizar para a versão 2.2.0, que a New Relic lançou para eliminar uma [versão vulnerável do OpenSSH](https://nvd.nist.gov/vuln/detail/CVE-2024-6387) anunciada recentemente.

## Ação requerida

A New Relic recomenda que os clientes que usam o New Relic Salesforce Exporter atualizem imediatamente para a versão 2.2.0.

A New Relic forneceu os seguintes recursos para ajudar com essas atualizações:

* [Notas de versão do exportador do Salesforce](https://github.com/newrelic/newrelic-salesforce-exporter/releases/tag/2.2.0)
* [Detectando e atualizando implantação on-host](https://github.com/newrelic/newrelic-salesforce-exporter?tab=readme-ov-file#upgrading-on-host-deployments)

Se os clientes não conseguirem atualizar seu New Relic Salesforce Exporter, limite o acesso SSH por meio de controles baseados em rede para minimizar os riscos de ataque.
