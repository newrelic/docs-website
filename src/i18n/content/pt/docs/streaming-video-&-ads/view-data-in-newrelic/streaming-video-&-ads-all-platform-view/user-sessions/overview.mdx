A **All Platforms &gt; User Sessions** for Streaming Video &amp; Ads lista todas as sessões em todo o aplicativo.

## Exibir a interface do usuário da página Sessões do usuário

A tabela da página **User Session** tem várias colunas para exibir várias informações para cada sessão. A tabela a seguir fornece informações sobre cada coluna disponível na interface do usuário da página Sessão:

<Callout variant="important">
  A tabela de sessões mostra todas as sessões de usuários que começaram dentro do filtro de tempo selecionado acima da tabela.
</Callout>
