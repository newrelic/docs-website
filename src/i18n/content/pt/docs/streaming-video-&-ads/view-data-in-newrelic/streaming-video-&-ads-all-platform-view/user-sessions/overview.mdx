## Visão geral

A página User Sessions fornece uma visão detalhada das sessões de visualização individuais em todos os seus aplicativos e plataformas de streaming monitorados. Esta página ajuda você a rastrear o comportamento de visualização do usuário, identificar sessões com erros e analisar padrões de reprodução em toda a sua infraestrutura de streaming.

### O que você pode fazer com as User Sessions

Use esta página para:

* Ver todas as sessões de visualização do usuário em todas as aplicações onde o agente de vídeo está instalado.
* Acompanhe detalhes em nível de sessão, incluindo ID da View, ID da Sessão, ID do Usuário e Nome do Aplicativo
* Monitore o conteúdo sendo visualizado por título e plataforma
* Identificar sessões com erros para solução de problemas
* Veja os horários de início e término da sessão para analisar a duração da visualização.
* Acesse informações detalhadas da sessão clicando em View ID ou Session ID
* Filtre sessões por intervalo de tempo para focar em períodos específicos
* Analise padrões de visualização em diferentes dispositivos e plataformas
* Revise os detalhes do conteúdo de cada sessão

### Quem usa esta página

Esta página é útil para equipes de suporte ao cliente, engenheiros de garantia de qualidade e equipes de operações que precisam solucionar problemas relatados pelos usuários, analisar sessões de visualização individuais em várias plataformas e entender os padrões de comportamento do usuário em todos os aplicativos de streaming.

## Página User Sessions

A **All Platforms &gt; User Sessions** for Streaming Video &amp; Ads lista todas as sessões em todo o aplicativo.
