## evento de fluxo de sessão do usuário

Os streamings de eventos da sessão do usuário são os eventos que são gerados durante a sessão. Esses eventos são usados para rastrear o comportamento do usuário durante a sessão.

Um gráfico é exibido na página de sessão do usuário que mostra os eventos gerados durante a sessão e informações detalhadas sobre o evento, como:
