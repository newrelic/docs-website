## Visão geral

A página User Sessions fornece uma visão detalhada de sessões de visualização individuais para uma aplicação monitorada específica. Esta página ajuda você a acompanhar o comportamento de visualização do usuário, identificar sessões com erros e analisar padrões de reprodução em um único aplicativo ou plataforma de streaming.

### O que você pode fazer com as User Sessions

Use esta página para:

* Ver todas as sessões de visualização do usuário para uma aplicação específica onde o agente de vídeo está instalado.
* Rastreie detalhes no nível da sessão, incluindo ID da View, ID da Sessão e ID do Usuário
* Monitorar o conteúdo sendo visualizado por título
* Identificar sessões com erros para solução de problemas
* Veja os horários de início e término da sessão para analisar a duração da visualização.
* Acesse informações detalhadas da sessão clicando em View ID ou Session ID
* Filtre sessões por intervalo de tempo para focar em períodos específicos
* Analise os padrões de visualização para o dispositivo e a plataforma específicos
* Revise os detalhes do conteúdo de cada sessão

### Quem usa esta página

Esta página é útil para desenvolvedores de aplicativos, equipes de suporte ao cliente e engenheiros de garantia de qualidade que precisam solucionar problemas relatados por usuários para um aplicativo específico, analisar sessões de visualização individuais e entender os padrões de comportamento do usuário dentro dessa plataforma.

## Página User Sessions

A **Single platforms &gt; User Sessions** para streaming de vídeo e anúncios lista todas as sessões em um aplicativo específico.
