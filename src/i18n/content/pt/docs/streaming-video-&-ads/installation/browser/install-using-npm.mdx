Instale o agente de vídeo usando o gerenciador de pacotes npm para aplicações JavaScript.

## Antes de você começar

Conclua as [etapas comuns de instalação](/docs/streaming-video-&-ads/installation/browser/get-started-with-browser#install) primeiro.

## Instalar usando o npm [#install-npm]

<Steps>
  <Step>
    ### Nomeie seu aplicativo

    Forneça um nome para sua aplicação e clique em <DNT>**Save and Continue**</DNT>.
  </Step>

  <Step>
    ### Instale o agente

    Instale a versão mais recente da New Relic copiando o trecho de código e executando-o no seu terminal (Mac) ou prompt de comando (Windows).

    <Callout variant="important">
      Selecione **npm** ou **Yarn** para instalar o agente de vídeo.
    </Callout>
  </Step>

  <Step>
    ### Adicione código à sua aplicação

    Após o download do script, inclua o trecho de código exibido na plataforma na sua aplicação para começar a monitorar seu player de vídeo.
  </Step>

  <Step>
    ### Gerar ID da aplicação

    Selecione <DNT>**Get Application Info**</DNT>. Isso gerará o ID da aplicação para a geração da entidade do browser.
  </Step>

  <Step>
    ### Ver seus dados

    Clique em <DNT>**See your data**</DNT>.
  </Step>
</Steps>

<Callout variant="important">
  Pode levar alguns minutos para interagir com sua aplicação e gerar dados.
</Callout>
