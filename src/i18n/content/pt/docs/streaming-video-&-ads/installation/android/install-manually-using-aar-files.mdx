Esta página fornece uma visão geral das etapas de instalação ao usar arquivos AAR para instalar manualmente o agente de Streaming Video &amp; Ads. Para a instalação completa com exemplos de código, use a [instalação guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clone o repositório [video-agent-android](https://github.com/newrelic/video-agent-android).
  </Step>

  <Step>
    Abra-o com o Android Studio.
  </Step>

  <Step>
    Vá para <DNT>**View &gt; Tool Windows &gt; Gradle**</DNT> para abrir a janela de ferramentas do Gradle.
  </Step>

  <Step>
    Na janela de ferramentas do Gradle, expanda <DNT>**NRVideoProject &gt; Tasks &gt; build**</DNT> e clique duas vezes em <DNT>**assemble**</DNT>. Isso compilará o projeto e gerará os arquivos de biblioteca AAR dentro da pasta de cada módulo <DNT>**build &gt; outputs &gt; aar**</DNT>.
  </Step>

  <Step>
    Em seu projeto, vá para <DNT>**File &gt; New &gt; New Module**</DNT>, importe o pacote `.JAR/.AAR` e clique em <DNT>**Next**</DNT>.
  </Step>

  <Step>
    Selecione os arquivos AAR que você gerou na etapa anterior e clique em <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    Repita as etapas 5 e 6 para cada arquivo AAR que você precisar:

    * **NewRelicVideoCore.aar** (Obrigatório): Funcionalidade principal do agente de vídeo
    * **NRExoPlayerTracker.aar** (Obrigatório): Rastreamento do ExoPlayer
    * **NRIMATracker.aar** (Opcional): Rastreamento de anúncios com o IMA SDK
  </Step>

  <Step>
    No arquivo `build.gradle` do módulo do seu app, adicione as seguintes dependências:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")   // Required: Core video agent
      implementation project(":NRExoPlayerTracker")  // Required: ExoPlayer tracking
      implementation project(':NRIMATracker')        // Optional: Ad tracking with IMA
    }
    ```

    Observação: inclua `NRIMATracker` apenas se precisar da funcionalidade de rastreamento de anúncios.
  </Step>

  <Step>
    Sincronize seu projeto com os arquivos Gradle clicando em <DNT>**Sync Now**</DNT> na barra de notificações, ou vá para <DNT>**File &gt; Sync Project with Gradle Files**</DNT>.
  </Step>

  <Step>
    ### Inicialize o agente de vídeo

    No método ou atividade `onCreate()` do seu aplicativo, inicialize o agente de vídeo da New Relic. Para exemplos detalhados de código e instruções de implementação, consulte a [instalação guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    ### Configurar o rastreamento do player de vídeo

    Configure o rastreador com base nas suas necessidades:

    * **Somente rastreador ExoPlayer**: Configure o rastreamento do ExoPlayer para monitoramento de reprodução de vídeo
    * **Rastreadores do ExoPlayer e do IMA**: Configure o rastreamento do ExoPlayer e do IMA para reprodução de vídeo e rastreamento de anúncios

    <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />

    Para exemplos completos de código de implementação, consulte a [instalação guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    Compile e implante seu aplicativo para ver os dados de rastreamento de vídeo no New Relic.
  </Step>
</Steps>
