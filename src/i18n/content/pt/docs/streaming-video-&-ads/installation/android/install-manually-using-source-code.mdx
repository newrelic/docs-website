Esta página fornece uma visão geral das etapas de instalação ao usar o código-fonte para instalar manualmente o agente de Streaming Video &amp; Ads. Para a instalação completa com exemplos de código, use a [instalação guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clone o repositório [video-agent-android](https://github.com/newrelic/video-agent-android).
  </Step>

  <Step>
    No seu projeto Android, vá para <DNT>**File &gt; New &gt; Import Module**</DNT>.
  </Step>

  <Step>
    Insira o local do diretório do módulo da biblioteca e clique em <DNT>**Finish**</DNT>.

    <Callout variant="important">
      O diretório do módulo da biblioteca está localizado no repositório que você acabou de clonar. Módulos de exemplo: `NewRelicVideoCore`, `NRExoPlayerTracker`, `NRIMATracker`.
    </Callout>
  </Step>

  <Step>
    Repita os passos 2 e 3 com todos os módulos que deseja incluir.
  </Step>

  <Step>
    No arquivo `build.gradle` do módulo do seu app, adicione o seguinte:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    Clique em <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Configurar o rastreamento do player de vídeo

    Selecione o tipo de agente que corresponde à sua implementação:

    * **Agente de vídeo apenas com o rastreador ExoPlayer**: Para rastreamento de reprodução de vídeo

    * **Agente de vídeo com rastreadores ExoPlayer e IMA**: Para reprodução de vídeo e rastreamento de anúncios

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    Clique em <DNT>**See your data**</DNT> para implantar seu código atualizado e ver os dados.
  </Step>
</Steps>
