Esta página fornece uma visão geral das etapas de instalação ao usar o JitPack para instalar o agente de Streaming Video &amp; Ads. Para a instalação completa com exemplos de código, use a [instalação guiada](https://onenr.io/02R54MqY0wb).

<Steps>
  <Step>
    ## Adicionar repositório JitPack

    Adicione o repositório JitPack ao seu arquivo `build.gradle` raiz.
  </Step>

  <Step>
    ## Adicionar dependências

    Adicione as dependências do agente de vídeo ao arquivo `build.gradle` do seu aplicativo e clique em <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ## Configure o agente de vídeo

    Adicione a configuração do NRVideo no método `onCreate` da sua **classe Application ou MainActivity**.
  </Step>

  <Step>
    ## Configurar o rastreamento do player de vídeo

    Selecione o tipo de agente que corresponde à sua implementação:

    * **Agente de vídeo apenas com o rastreador ExoPlayer**: Para rastreamento de reprodução de vídeo

    * **Agente de vídeo com rastreadores ExoPlayer e IMA**: Para reprodução de vídeo e rastreamento de anúncios

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    Clique em <DNT>**See your data**</DNT> para visualizar seus dados de streaming no New Relic.
  </Step>
</Steps>
