## Métodos de gerenciamento de domínio

<table>
  <thead>
    <tr>
      <th width="30%">
        Método
      </th>

      <th>
        Descrição
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        [nrAddDomainSubstituição](/docs/streaming-video-&-ads/installation/roku/agent-api/domain-management/nradddomainsubstitution)
      </td>

      <td>
        Adicione um padrão de correspondência para o atributo de domínio e substitua-o por outra string. Toda vez que um evento ou métrica for gerado com um atributo de domínio, o agente verificará se ele corresponde a uma regex e aplicará a substituição especificada. Se nenhum padrão for definido, ele usará o domínio da URL inalterado. Isso se aplica a todos os eventos e métricas que contêm o atributo &quot;domain&quot;.
      </td>
    </tr>

    <tr>
      <td>
        [nrDelDomainSubstituição](/docs/streaming-video-&-ads/installation/roku/agent-api/domain-management/nrdeldomainsubstitution)
      </td>

      <td>
        Exclua um padrão correspondente criado com `nrAddDomainSubstitution`.
      </td>
    </tr>
  </tbody>
</table>

## Exemplos

### nrAddDomainSubstituição

```js
  	nrAddDomainSubstitution(nr, "^.+\.my\.domain\.com$", "mydomain.com")
```

### nrDelDomainSubstituição

```js
  	nrDelDomainSubstitution(nr, "^.+\.my\.domain\.com$")
```
