## Sintaxe

```js
  nrSetCustomAttributeList(nr as Object, attr as Object, actionName = "" as String) as Void
```

## Descrição

Defina uma lista de atributos personalizados a serem incluídos no evento.

      </td>

    <tr>
      <td>
        `attr`
      </td>

      <td>

      </td>
    </tr>

    <tr>
      <td>
        `actionName`
      </td>

      <td>
        (opcional) Ação onde o atributo será incluído. Definir todas as ações como padrão.
      </td>
    </tr>
  </tbody>
</table>

## Retornar

Nada.

## Exemplo

```js
  	attr = {"key0":"val0", "key1":"val1"}
	  nrSetCustomAttributeList(m.nr, attr, "CONTENT_HEARTBEAT")
```
