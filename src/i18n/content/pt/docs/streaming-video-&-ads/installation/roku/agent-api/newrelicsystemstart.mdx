## Sintaxe

```js
NewRelicSystemStart(port as Object) as Object
```

## Descrição

Inicia o logging do sistema.

## Retornar

O objeto `roSystemLog` foi criado.

## Exemplo

```js
  m.syslog = NewRelicSystemStart(m.port)
```
