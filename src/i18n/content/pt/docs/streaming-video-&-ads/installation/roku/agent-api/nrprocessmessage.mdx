## Sintaxe

```js
  nrProcessMessage(nr as Object, msg as Object) as Boolean
```

## Descrição

Verifique se há um log de mensagem do sistema, processe-o e envie o evento apropriado.

## Retornar

Verdadeiro se msg for uma mensagem do sistema para log, Falso caso contrário.

## Exemplo

```js
  	while (true)
		      msg = wait(0, m.port)
		      if nrProcessMessage(m.nr, msg) = false
			            if type(msg) = "roPosterScreenEvent"
				                  if msg.isScreenClosed()
					                        exit while
				                  end if
			            end if
		      end if
	  end while
```
