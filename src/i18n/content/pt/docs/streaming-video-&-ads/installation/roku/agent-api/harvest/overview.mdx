## Métodos de configuração do Harvest

<table>
  <thead>
    <tr>
      <th width="30%">
        Método
      </th>

      <th>
        Descrição
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        [nrSetHarvestTime](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttime)
      </td>

      <td>
        Defina o tempo de coleta, o tempo em que as amostras ficam armazenadas em buffer antes de serem enviadas ao New Relic para eventos e logs. O valor mínimo é 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrSetHarvestTimeEvents](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimeevents)
      </td>

      <td>
        Defina o tempo de coleta para o evento, o tempo em que o evento será armazenado em buffer antes de ser enviado para o New Relic. O valor mínimo é 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrSetHarvestTimeLogs](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimelogs)
      </td>

      <td>
        Defina o tempo de coleta dos logs, o tempo em que os logs ficam armazenados em buffer antes de serem enviados ao New Relic. O valor mínimo é 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrSetHarvestTimeMetrics](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimemetrics)
      </td>

      <td>
        Defina o tempo de coleta dos logs, o tempo em que os logs ficam armazenados em buffer antes de serem enviados ao New Relic. O valor mínimo é 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrForceColheita](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvest)
      </td>

      <td>
        Colha eventos e logs imediatamente. Não zera o cronômetro de colheita.
      </td>
    </tr>

    <tr>
      <td>
        [Eventos nrForceHarvest](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvestevents)
      </td>

      <td>
        Colha o evento imediatamente. Não zera o cronômetro de colheita.
      </td>
    </tr>

    <tr>
      <td>
        [nrForceHarvestLogs](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvestlogs)
      </td>

      <td>
        Colha os troncos imediatamente. Não zera o cronômetro de colheita.
      </td>
    </tr>
  </tbody>
</table>

## Exemplos

### nrSetHarvestTime

```js
  	nrSetHarvestTime(m.nr, 60)
```

### nrSetHarvestTimeEvents

```js
  	nrSetHarvestTimeEvents(m.nr, 60)
```

### nrSetHarvestTimeLogs

```js
  	nrSetHarvestTimeLogs(m.nr, 60)
```

### nrSetHarvestTimeMetrics

```js
  	nrSetHarvestTimeMetrics(m.nr, 60)
```

### nrForceColheita

```js
  	nrForceHarvest(m.nr)
```

### Eventos nrForceHarvest

```js
  	nrForceHarvestEvents(m.nr)
```

### nrForceHarvestLogs

```js
  	nrForceHarvestLogs(m.nr)
```
