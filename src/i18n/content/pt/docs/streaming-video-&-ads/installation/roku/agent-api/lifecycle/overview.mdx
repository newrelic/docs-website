## Exemplos

### Enviar evento APP\_STARTED

```brightscript
sub Main(aa as Object)
    ...
    nrAppStarted(m.nr, aa)
```

### Enviar evento SCENE\_LOADED

```brightscript
nrSceneLoaded(m.nr, "MyVideoScene")
```

### Processar mensagens de log do sistema

```brightscript
while (true)
    msg = wait(0, m.port)
    if nrProcessMessage(m.nr, msg) = false
        if type(msg) = "roPosterScreenEvent"
            if msg.isScreenClosed()
                exit while
            end if
        end if
    end if
end while
```
