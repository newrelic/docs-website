## Sintaxe

```js
  nrUpdateConfig(nr as Object, config as Object) as Void
```

## Descrição

Atualiza a configuração, como URL do proxy de rede.

## Retornar

Nada.

## Exemplo

```js
  	config = { proxyUrl: "http://example.com:8888/;" }
	  nrUpdateConfig(m.nr, config)
```
