## Sintaxe

```js
  nrAddDomainSubstitution(nr as object, pattern as String, subs as String) as Void
```

## Descrição

* Adicione um padrão correspondente para o atributo de domínio e substitua-o por outra string.
  * Toda vez que um evento ou métrica é gerado com um atributo de domínio, o agente verificará se ele corresponde a uma regex e aplicará a substituição especificada.
* Se nenhum padrão for definido, o domínio da URL será usado inalterado.
  * Aplica-se a todos os eventos e métricas que contenham o atributo &quot;domínio&quot;.

## Exemplo

```js

```
