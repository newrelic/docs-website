## Sintaxe

```js
  nrSendHttpResponse(nr as Object, _url as String, msg as Object) as Void
```

## Descrição

Envie um evento HTTP\_RESPONSE do tipo ConnectedDeviceSystem.

## Retornar

Nada.

## Exemplo

```js
  	msg = wait(5000, m.port)
```
