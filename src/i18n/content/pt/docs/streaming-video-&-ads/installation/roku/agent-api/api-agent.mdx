O agente New Relic Roku foi projetado para monitorar eventos de sistema e vídeo no aplicativo Roku. Ele fornece uma API abrangente para rastrear vários tipos de eventos, incluindo sistema, vídeo, erro, anúncio e evento personalizado. Este documento descreve os métodos API disponíveis para interagir com o agente Roku New Relic .
