Esta página fornece uma visão geral das etapas de instalação ao instalar manualmente o agente de Streaming Video &amp; Ads. Para a instalação completa com exemplos de código, use a [instalação guiada](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios).

<Steps>
  <Step>
    Clone o repositório [video-agent-iOS](https://github.com/newrelic/video-agent-ios).
  </Step>

  <Step>
    Abra cada um dos arquivos `.xcodeproj` com o Xcode.
  </Step>

  <Step>
    Selecione o esquema apropriado.
  </Step>

  <Step>
    Compilar (CMD+B).
  </Step>

  <Step>
    Inclua os arquivos `.framework` gerados no seu projeto.
  </Step>

  <Step>
    Clique em <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Inicializar rastreador de vídeo

    Selecione o tipo de agente que corresponde à sua implementação:

    * **Agente de vídeo apenas com o rastreador AVPlayer**: Para rastreamento de reprodução de vídeo
    * **Agente de vídeo com AVPlayer e rastreadores IMA**: Para reprodução de vídeo e rastreamento de anúncios
  </Step>

  <Step>
    Clique em <DNT>**See your data**</DNT> para visualizar seus dados de streaming no New Relic.
  </Step>
</Steps>
