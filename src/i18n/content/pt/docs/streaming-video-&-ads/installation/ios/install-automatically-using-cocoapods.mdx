Esta página fornece uma visão geral das etapas de instalação ao usar o CocoaPods para instalar o agente de Streaming Video &amp; Ads. Para a instalação completa com exemplos de código, use a [instalação guiada](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios).

<Steps>
  <Step>
    ### Adicionar dependências

    Adicione os pods do agente de vídeo ao seu Podfile.
  </Step>

  <Step>
    ### Instalar pods

    Execute `pod install` no seu terminal.
  </Step>

  <Step>
    ### Construir projeto

    Abra seu projeto no **Xcode** e compile-o.
  </Step>

  <Step>
    Clique em <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Inicializar rastreador de vídeo

    Selecione o tipo de agente que corresponde à sua implementação:

    * **Agente de vídeo apenas com o rastreador AVPlayer**: Para rastreamento de reprodução de vídeo
    * **Agente de vídeo com AVPlayer e rastreadores IMA**: Para reprodução de vídeo e rastreamento de anúncios
  </Step>

  <Step>
    Clique em <DNT>**See your data**</DNT> para visualizar seus dados de streaming no New Relic.
  </Step>
</Steps>
