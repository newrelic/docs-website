Continue lendo para saber como desinstalar o agente New Relic Node.js. Para obter instruções sobre como desabilitar temporariamente o agente, consulte [Desabilitar o agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#node).

Para desinstalar o agente New Relic Node.js:

1. Remova `require('newrelic');` da primeira linha do módulo principal do aplicativo.
2. Remova o pacote npm com `npm uninstall newrelic --save`.
3. Exclua o arquivo `newrelic.js` e desmarque todas as [variáveis de ambiente](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration#methods-and-precedence) que estão sendo usadas.
4. Quando o processo de desinstalação terminar, [reinicie](#restart) seu aplicativo.

Para obter mais informações sobre instalação, consulte [Instalando e mantendo o Node.js.](/docs/agents/nodejs-agent/installation-configuration/installing-maintaining-nodejs)

<Callout variant="tip">
  Você também pode querer [remover seu aplicativo](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) do New Relic.
</Callout>
