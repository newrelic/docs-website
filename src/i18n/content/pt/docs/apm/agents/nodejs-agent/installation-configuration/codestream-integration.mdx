O nível do código métrico permite que os desenvolvedores que usam a [extensão New Relic CodeStream](/docs/codestream/observability/code-level-metrics) vejam os dados APM exibidos contextualmente em seu IDE, juntamente com métodos individuais no código. Isso permite que os desenvolvedores sejam mais proativos na abordagem de problemas de desempenho à medida que escrevem e revisam o código.

## Iniciar [#get-started]

Primeiro, [instale](/docs/codestream/start-here/install-codestream) a extensão New Relic CodeStream no IDE compatível de sua escolha e login.

<Callout variant="important">
  Nível do código métrica estão disponíveis para o agente Node.js versão 9.7.5 ou superior e habilitados por padrão para a versão 10.5.0 e superior. Para alterar esta configuração, consulte nossa [documentação](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration#code-level-metrics).
</Callout>

## Atributo do agente [#attributes]

O agente Node.js relata e anexa o seguinte atributo à maioria dos spans produzidos pela [estrutura da web](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent#support) em seu aplicativo.

<Callout variant="important">
  Em alguns casos, o agente encapsulará diretamente o código framework base invocado no seu aplicativo e não será exibido no seu IDE.
</Callout>
