O plugin New Relic Apollo Server utiliza seu aplicativo Apollo Server para dar visibilidade à sua carga GraphQL. Isso ajuda você a descobrir e diagnosticar a causa de sua [consulta lenta do GraphQL](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph). A versão suportada do Apollo Server é 2.14 ou posterior.

Nosso plugin registra os tempos gerais para consulta e usa [distributed tracing](/docs/understand-dependencies/distributed-tracing/get-started/how-new-relic-distributed-tracing-works) para descobrir o problema da rota. Use esta instrumentação para ver se o problema surge da resolução de um dado solicitado ou se decorre de trabalho realizado em outros serviços ou banco de dados.

## Compatibilidade

O plug-in New Relic funciona com os seguintes módulos do Apollo Server:

* `@apollo/server`
* `@apollo/gateway`
* `@apollo/subgraph`
* `apollo-server` (>= 2,14)
* `apollo-server-express`
* `apollo-server-hapi`
* `apollo-server-koa`
* `apollo-server-fastify`
* `apollo-server-lambda`

Outros plug-ins podem funcionar, dependendo da implementação subjacente, mas não foram verificados.
