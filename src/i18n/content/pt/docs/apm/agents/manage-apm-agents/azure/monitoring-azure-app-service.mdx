O Azure App Service é uma oferta de plataforma como serviço (PaaS) totalmente gerenciada do Microsoft Azure que permite aos desenvolvedores criar, implantar e dimensionar aplicativos da web, back-end móvel e API de forma rápida e eficiente em várias linguagens de programação e estrutura.

Ao integrar o monitoramento de desempenho de aplicativos (APM) da New Relic(agente APM com o Azure App Service, você pode obter monitoramento e otimização abrangentes de seu aplicativo. New Relic APM fornece insights e dados acionáveis em tempo real para ajudá-lo a garantir que seu aplicativo tenha o melhor desempenho. A tabela a seguir fornece detalhes para todo o monitoramento do Azure App Service com o New Relic
