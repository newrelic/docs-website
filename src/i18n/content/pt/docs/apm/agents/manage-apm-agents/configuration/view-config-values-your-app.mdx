## Problema

Você precisa de uma maneira fácil de obter informações sobre as definições de configuração do seu aplicativo.

## Solução

Se não quiser visualizar seu arquivo de configuração em si, você pode visualizar as definições de configuração na interface do APM, vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DNT>. Clique no seu aplicativo e, em seguida, clique em <DNT>**Settings > Environment > Agent initialization**</DNT>.
