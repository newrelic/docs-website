## Sintaxe

```py
newrelic.agent.application(name=None)
```

Retorna o objeto correspondente a um aplicativo monitor de agente; exigido por algumas chamadas de API do agente Python.

## Descrição

Esta chamada retorna um objeto correspondente a uma aplicação agente-monitor. Se nenhum valor `name` for definido, o nome [definido por meio da configuração do agente Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration#app_name) será usado.

O objeto aplicativo retornado não apresenta nenhuma API pública, mas é requerido por alguma outra chamada de API.

<Callout variant="important">
  Esta chamada retorna apenas o objeto do aplicativo. Ele não tentará registrar o aplicativo no coletor de dados se isso ainda não tiver ocorrido.
</Callout>

## Valores de retorno

Retorna um objeto de aplicativo. O objeto em si não apresenta nenhuma API pública, mas é requerido por alguma outra chamada de API.

## Exemplos

### Gerar e usar um objeto de aplicativo [#app-object-example]

Para gerar um objeto de aplicativo, use:

```py
import newrelic.agent

app = newrelic.agent.application()
```

Algumas outras chamadas de API do agente Python usam o objeto do aplicativo como parâmetro. Por exemplo, veja a chamada [`notice_error`](/docs/agents/python-agent/python-agent-api/noticeerror-python-agent-api/) .
