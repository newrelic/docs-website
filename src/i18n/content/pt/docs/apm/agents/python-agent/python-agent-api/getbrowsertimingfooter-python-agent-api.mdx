## Sintaxe [#syntax]

```py
newrelic.agent.get_browser_timing_footer()
```

<Callout variant="important">
  Começando com o agente Python versão 9.4.0+, esta API está obsoleta e sempre retorna uma string vazia. A funcionalidade anterior de `get_browser_timing_footer()` agora está incluída em `get_browser_timing_header()`.

  Para obter mais informações sobre o uso adequado, consulte [Instrumento de monitoramento manual do browser](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).
</Callout>

Gera um rodapé de agente <InlinePopover type="browser"/>para injetar o agente JavaScript.

## Descrição [#description]

Esta chamada retorna o rodapé JavaScript do agente browser trecho que, junto com o [cabeçalho JavaScript trecho](/docs/agents/python-agent/python-agent-api/get_browser_timing_header), é utilizado para habilitar o monitoramento do Browser. Para saber como habilitar o monitoramento de Browser usando esta chamada junto com `get_browser_timing_header`, consulte [Manualmente instrumento de monitoramento de Browser](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).

Esta chamada deve ser chamada para cada solicitação da web onde você deseja injetar o trecho JavaScript. Ele <DNT>**cannot**</DNT> será chamado em escopo global e armazenado em cache.

## Valores de retorno [#return-values]

Retorna o JavaScript do rodapé do agente browser.
