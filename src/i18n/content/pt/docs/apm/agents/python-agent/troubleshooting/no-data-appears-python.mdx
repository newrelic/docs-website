## Problema

Depois de instalar o agente Python, gerar algum tráfego e esperar cinco minutos, seus dados não aparecem na interface do New Relic.

## Solução

Aqui estão algumas dicas para resolução de problemas de dados faltantes, colocadas em ordem aproximada de probabilidade de serem o problema:

1. Certifique-se de ter instalado o agente no mesmo ambiente que seu aplicativo. Se você estiver usando um ambiente virtual, instale-o lá.

2. Certifique-se de que seu agente esteja atualizado. Se você fez download do site da New Relic em vez de usar

   <DNT>
     **pip**
   </DNT>

   ou

   <DNT>
     **easy_install**
   </DNT>

   , compare o número da sua versão com a [versão atual](https://download.newrelic.com/python_agent/release/).

3. Use [o New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics) para tentar identificar automaticamente o problema.

4. Certifique-se de que os dados possam chegar à interface do New Relic executando o [teste do agente Python](/docs/python/testing-the-python-agent).

5. Certifique-se de atender à [compatibilidade e aos requisitos](/docs/agents/python-agent/getting-started/compatibility-requirements-python-agent). Verifique se há requisitos especiais para a [framework web ou servidor web](/docs/agents/python-agent/hosting-mechanisms) que você está usando.

6. Revise o arquivo de configuração que foi gerado com o comando `newrelic-admin generate-config` para certificar-se de que ele contém o

   <InlinePopover type="licenseKey"/>

   correto.

7. [Gere log detalhado para resolução de problemas](/docs/agents/python-agent/troubleshooting/generating-logs-troubleshooting). Em seguida, verifique o arquivo de log em busca de mensagem de erro. Se o arquivo de log estiver faltando, certifique-se de ter permissões para gravar nesse local.

Além disso, se nenhum dado aparecer e você vir mensagens de aviso sobre a ativação ou reativação do seu aplicativo, siga os [procedimentos de ativação e resolução de problemas](/docs/agents/python-agent/troubleshooting/activate-application-warning-python).
