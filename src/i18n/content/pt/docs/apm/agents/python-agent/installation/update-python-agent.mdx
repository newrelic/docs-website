Para aproveitar ao máximo nossos recursos mais recentes, melhorias e patches de segurança importantes, recomendamos que você atualize seu agente Python para a versão mais recente. Para obter informações adicionais sobre atualizações específicas do agente, consulte as [notas de versão do agente Python](/docs/release-notes/agent-release-notes/python-release-notes).

## Atualização com índice de pacote Python (PyPi) [#package-index]

O pacote do agente Python está listado no PyPi em [http://pypi.python.org/pypi/newrelic](http://pypi.python.org/pypi/newrelic "O link abre em uma nova janela"). Para atualizar com um pacote específico, siga as instruções correspondentes:

        ```
        pip install --upgrade newrelic
        ```

        ```
        pip install -r requirements.txt
        ```

## Baixe o pacote manualmente [#manual-download]

Para baixar manualmente o pacote New Relic Python:

1. Remova o pacote New Relic atual.
2. Baixe o novo pacote e siga os procedimentos [de instalação do agente Python](/docs/agents/python-agent/installation-configuration/python-agent-installation#download-site) .

<Callout variant="tip">
  Recomendamos usar `pip` em vez de instalação manual, porque `pip` removerá corretamente as versões antigas ao atualizar o software do agente.
</Callout>
