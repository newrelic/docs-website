## Problema

Você deseja emular o comportamento legado de ativar as opções <DNT>**Capture attributes**</DNT> ou <DNT>**Capture parameters**</DNT> nas configurações [de Configuração no lado do servidor da](/docs/agents/manage-apm-agents/configuration/server-side-agent-configuration) interface do New Relic.

## Solução

Você pode emular esse comportamento legado personalizando o arquivo de configuração do agente Python para capturar o parâmetro de solicitação. Para obter detalhes, consulte o [exemplo do parâmetro de solicitação Capture](/docs/agents/python-agent/attributes/python-attribute-examples#ex_req_params) na documentação do atributo Python.
