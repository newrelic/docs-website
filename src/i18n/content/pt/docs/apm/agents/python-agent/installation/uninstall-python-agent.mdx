Este documento explica como desinstalar o agente Python. Para obter instruções sobre como desabilitar temporariamente o agente, consulte [Desabilitar o agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#python).

Para desinstalar seu agente Python:

1. Use um destes métodos:

   * Se você instalou com PIP, execute:

     ```
     pip uninstall newrelic
     ```

   * Se você instalou com easy_install, execute:

     ```
     easy_install -m newrelic
     ```

2. Quando o processo de desinstalação terminar, reinicie seu aplicativo.

<Callout variant="tip">
  Você também pode [remover seu aplicativo](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) da interface do APM.
</Callout>
