## Sintaxe

```py
newrelic.agent.disable_browser_autorum(flag=True)
```

Desativa <InlinePopover type="browser"/>para uma solicitação da web.

## Descrição

Esta chamada desabilita [o monitoramento do Browser](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) para a página/solicitação para a qual é chamado. Esta chamada afeta apenas a solicitação da web atual e deve ser chamada antes do ponto em que o JavaScript do cabeçalho/rodapé do agente browser é injetado. Para mais informações sobre como o JavaScript do agente browser é inserido, consulte [Instalar monitoramento de browser](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser).

<Callout variant="tip">
  Você também pode desativar a injeção do browser definindo a chave `newrelic.disable_browser_autorum` para a solicitação específica no dicionário de ambiente WSGI transmitido pelo servidor WSGI para o aplicativo WSGI que está sendo monitorado.
</Callout>

## Exemplos

### Desativando monitoramento de browser [#example]

Neste exemplo, você atualizou o agente e percebeu que o N JavaScript que é injetado automaticamente na página HTML também está sendo injetado em cada iFrame carregado nessas páginas. Isso fez com que suas visualizações de página aumentassem substancialmente.

Nessas visualizações iFrame específicas, você faria esta chamada:

```py
newrelic.agent.disable_browser_autorum()
```
