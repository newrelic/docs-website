## Sintaxe

```py
newrelic.agent.end_of_transaction()
```

Termina a transação atual.

## Descrição

Esta chamada encerra a transação atual imediatamente e interrompe o temporizador de resposta no horário atual. Após o término da transação, o agente não coleta nenhuma métrica adicional de detalhamento de desempenho. No entanto, o agente ainda coleta detalhes da exceção do Python.

Essa chamada é útil quando você tem uma resposta grande que é transmitida de volta para a transação da web e somente a parte inicial da transação é de seu interesse. (Isso também pressupõe que, por qualquer motivo, você não deseja sinalizar a transação como uma [tarefa em segundo plano](/docs/agents/python-agent/python-agent-api/set_background_task).)

## Parâmetro

Nenhum.

## Valores de retorno

Nenhum.
