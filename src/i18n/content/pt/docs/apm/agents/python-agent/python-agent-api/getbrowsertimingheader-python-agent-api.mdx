## Sintaxe [#syntax]

```py
get_browser_timing_header(nonce=None)
```

Gera um cabeçalho de agente <InlinePopover type="browser"/>para injetar o agente JavaScript.

## Descrição [#description]

<Callout variant="important">
  O parâmetro nonce para suporte CSP nonced foi adicionado no agente Python versão 9.4.0.

  Nas versões 9.3.0 e anteriores do agente Python, a API [`get_browser_timing_footer`](/docs/agents/python-agent/python-agent-api/get_browser_timing_footer) também era necessária para ativar o tempo do browser. A partir da versão do agente 9.4.0 e posterior, isso não é mais necessário.
</Callout>

Esta chamada retorna o trecho de cabeçalho JavaScript do agente browser que você pode usar para ativar [o monitoramento do Browser](/docs/agents/python-agent/supported-features/page-load-timing-python). Você poderá usar isso se a instrumentação automática não for possível para seu aplicativo, se você usar uma política de segurança de conteúdo não autorizada ou se quiser coletar dados do browser apenas para páginas específicas.

Esta é uma alternativa manual à [inserção automática deste JavaScript](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#select-apm-app) pelo agente. Para saber como habilitar o monitoramento de Browser usando esta chamada junto com `get_browser_timing_footer`, consulte [Manualmente instrumento de monitoramento de Browser](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).

Esta chamada deve ser chamada para cada solicitação da web onde você deseja injetar o trecho JavaScript. Ele <DNT>**cannot**</DNT> será chamado em escopo global e armazenado em cache.
