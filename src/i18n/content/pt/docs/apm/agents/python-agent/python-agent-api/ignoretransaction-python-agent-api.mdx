## Sintaxe

```py
newrelic.agent.ignore_transaction(flag=True)
```

Ignora a transação atual.

## Descrição

Esta chamada ignora a transação atual. O agente não reporta nenhum dado da transação.

<Callout variant="tip">
  Você também pode ignorar uma transação da web no dicionário de ambiente WSGI. Para fazer isso, defina a chave `newrelic.ignore_transaction` para a solicitação específica no dicionário de ambiente WSGI transmitido pelo servidor WSGI para seu aplicativo WSGI.
</Callout>

## Exemplos

### Ignorar uma transação [#ignore-example]

Para ignorar uma transação que você não deseja que apareça na interface, você executaria o seguinte onde uma transação está sendo gerada no código do seu aplicativo. Este exemplo ilustra o uso da API para ignorar uma rota Flask:

```py
@app.route("/")
def process_results():
    newrelic.agent.ignore_transaction(flag=True)
    return 'hello world'
```
