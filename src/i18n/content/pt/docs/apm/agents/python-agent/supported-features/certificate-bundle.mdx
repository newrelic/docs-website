## Perguntas frequentes [#faq]

### O que é um pacote de certificados? [#what-is-certificate-bundle]

Um pacote de certificados fornece essencialmente uma lista de raízes de confiança. Diz ao agente em quem confiar ao se comunicar com o backend.

### Por que o pacote de certificados é fornecido? [#why-certificate-bundle]

Muitos clientes executam no docker. Às vezes, essas imagens não fornecem seus próprios pacotes de certificados porque geralmente não são usadas para comunicação por HTTP. Fornecemos nosso próprio pacote para que os clientes tenham mais facilidade na instalação do agente.

### Quando o pacote de certificados é usado? [#when-to-use-certificate-bundle]

Quando os certificados do sistema estiverem disponíveis, o agente usará por padrão aqueles no pacote de certificados incluído. O pacote de certificados só é usado quando nenhum certificado é fornecido pelo sistema.
