## Sintaxe

```py
newrelic.agent.shutdown_agent(timeout=None)
```

Desliga o agente à força.

## Descrição

Desligar o agente à força. O agente faz uma última tentativa de enviar dados para o [coletor](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector).

## Parâmetro

<table>
  <thead>
    <tr>
      <th width="25%">
        Parâmetro
      </th>

      <th>
        Descrição
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `timeout`

        _int_ ou _float_
      </td>

      <td>
        Opcional, mas altamente recomendado. O tempo máximo que o chamador espera que o agente conclua o upload dos dados. Se o período de tempo limite terminar antes do agente terminar, o agente continuará reportando dados em segundo plano, mas o chamador poderá encerrar todo o processo à força e interromper o upload.

        Se nenhum valor for definido ou definido como `None`, o agente usará o valor de tempo limite global padrão, que pode ser substituído nas definições de configuração, mas o padrão é 2,5 segundos.
      </td>
    </tr>
  </tbody>
</table>

## Valores de retorno

Nenhum.

## Exemplos

### Com um tempo limite de três segundos [#timeout-example]

```py
newrelic.agent.shutdown_agent(timeout=3)
```
