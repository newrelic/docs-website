## Sintaxe

```py
get_linking_metadata()
```

Retorna um dicionário de dados contextuais para uso na mensagem do log.

## Requisitos

Agente Python versão 5.4.0 ou superior

## Descrição

Um dicionário de pares de valores principais que fornece informações contextuais. Esta API pode ser usada para adicionar metadados do agente para registro manual.

## Valores de retorno

Retorna um dicionário.

## Exemplos

```py
import json
import newrelic.agent
log_message = {"message": "Hello World"}
context = newrelic.agent.get_linking_metadata()
log_message.update(context)
print(json.dumps(log_message))
```
