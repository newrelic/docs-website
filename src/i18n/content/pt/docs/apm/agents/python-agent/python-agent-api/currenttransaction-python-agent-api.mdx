## Sintaxe

```py
newrelic.agent.current_transaction(active_only=True)
```

Retorna um objeto correspondente à transação atual.

## Descrição

Use `current_transaction` para recuperar o objeto que representa a [transação](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction) atual. Ele retornará `None` se não houver nenhuma transação ativa.

## Valores de retorno

Retorna um objeto de transação correspondente à transação atual. Retornará `None` se não houver nenhuma transação ativa.

## Exemplos

### Obtenha a transação atual [#function-trace-example]

```py
import newrelic.agent

@newrelic.agent.background_task()
def main():
    transaction = newrelic.agent.current_transaction()
```
