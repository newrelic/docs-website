## Sintaxe

```py
newrelic.agent.memcache_trace(command, async_wrapper=None)
```

Usado para instrumentar funções de comando do memcache para funções que não são instrumentadas por padrão.

## Descrição

`memcache_trace` é um decorador para adicionar às funções de comando do memcache. Adicionar este decorator permite coletar informações adicionais de [transação](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction) (incluindo informações de trace da transação).

O decorador `memcache_trace` pode ser usado em funções com a versão do agente 2.102.0.85 ou superior. A medição de tempo desses objetos começa quando o consumo é iniciado e termina quando o objeto é esgotado ou sai do escopo.

Você pode usar o decorador em conjunto com decoradores existentes, incluindo aqueles para métodos estáticos e de classe. Ao aninhar vários decoradores, tenha o decorador `memcache_trace` como o decorador mais externo.

## Formulários de chamada alternativos [#alternate]

Para configurações onde você não pode usar o decorador, estes formulários de chamada alternativos estão disponíveis:

### O invólucro

Se você souber antecipadamente onde estão as funções específicas que deseja trace , poderá usar o decorador `memcache_trace`. No entanto, se você <DNT>**don&apos;t**</DNT> conhece todas as funções que precisam ser rastreadas (por exemplo, se elas estão sendo pesquisadas dinamicamente como parte de um sistema de roteamento), então você deve usar o `MemcacheTraceWrapper` para encapsular a função no no momento da inscrição ou no momento da ligação.

### Quebra automática baseada em caminho [#path-based]

`wrap_memcache_trace` é usado para agrupar funções fora do código em que foram declaradas. Por exemplo: você pode usar isso para o código da biblioteca do instrumento que não deseja modificar.

Para obter mais informações sobre as diferenças entre esses formatos de chamada, consulte [Diferentes formatos de chamada](/docs/python-agent-api-different-call-forms).

## Exemplos

### `memcache_trace` [#trace\_example][#trace_example]

Um exemplo de uso do decorador `memcache_trace` :

```py
import newrelic.agent

@newrelic.agent.memcache_trace("get")
def get(id):
    ...
```

### Exemplo de wrapper [#trace-wrap-example]

Um exemplo de uso do `MemcacheTraceWrapper`:

```py
import newrelic.agent

def get(id):
    ...

get = newrelic.agent.MemcacheTraceWrapper(get, command="get")
```

Se você quiser retornar dinamicamente o valor do comando, você pode fornecer o comando como uma função quando o objeto `MemcacheTraceWrapper` for criado.

```py
import newrelic.agent

def get(id):
    ...

get = newrelic.agent.MemcacheTraceWrapper(get, command=lambda id: f"get:{id}")
```
