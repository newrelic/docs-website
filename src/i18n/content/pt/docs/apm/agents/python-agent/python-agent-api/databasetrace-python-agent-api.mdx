## Sintaxe

```py
newrelic.agent.database_trace(sql, dbapi2_module=None, async_wrapper=None)
```

Usado para instrumentar chamadas para banco de dados.

## Descrição

`database_trace` é usado para adicionar mais detalhes aos seus [traces da transação](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction-trace) na forma de segmentos adicionais. Todas as chamadas relatadas com `database_trace` aparecerão na [página de banco de dados do APM](/docs/apm/applications-menu/monitoring/databases-slow-queries-page). `database_trace` retorna um [partial](https://docs.python.org/3.12/library/functools.html#functools.partial) de `DatabaseTraceWrapper` que pode ser usado como um decorador para uma função para cronometrar chamadas ao seu banco de dados.

O decorador `database_trace` pode ser usado em geradores e corrotinas com a versão do agente 2.102.0.85 ou superior.

Se você não puder usar o decorador em seu aplicativo, poderá usar um destes outros formatos de chamada:

* <DNT>**The context manager**</DNT>: O formulário do gerenciador de contexto é `DatabaseTrace`.
* <DNT>**The wrapper**</DNT>: o formulário do wrapper é `DatabaseTraceWrapper`. Pode ser usado para retornar uma função encapsulada sem o uso de um decorador.
* <DNT>**The path-based wrapper**</DNT>: o formulário wrapper baseado em caminho é `wrap_database_trace`. Isso aplica o `DatabaseTraceWrapper` a um determinado objeto por meio de monkey patching. Isso usa os mesmos parâmetros do decorador mais um parâmetro `module` e `object_path` adicional.

Para obter uma explicação sobre os usos desses diferentes formatos de chamada, consulte [Diferentes formatos de chamada](/docs/python-agent-api-different-call-forms). Consulte [Exemplos](#examples) para exemplos de chamadas.

## Valores de retorno

`database_trace` retorna uma parcial `DatabaseTraceWrapper()` .

## Exemplos

### `database_trace` [#trace\_example][#trace_example]

Um exemplo de uso do decorador `database_trace` :

```py
import newrelic.agent

@newrelic.agent.database_trace(sql="SELECT * FROM users WHERE id = 1")
def get_user(id):
    ...
```

Um exemplo de uso do decorador `database_trace` com corrotinas nativas:

```py
import newrelic.agent

@newrelic.agent.database_trace(sql="SELECT * FROM users WHERE id = 1")
async def get_user(id):
    ...
```

### Exemplo de gerenciador de contexto [#context-mgr-example]

Um exemplo de uso do gerenciador de contexto `DatabaseTrace` : isso fornecerá o tempo necessário para executar \`custom\_action\`.

```py
import newrelic.agent


def complex_query(a, b, c):
    with Connection(host, port, db) as conn:
        with newrelic.agent.DatabaseTrace(
            sql="SELECT * FROM users WHERE id = 1",
        ):

            conn.custom_action()
```

### Exemplo de wrapper [#trace-wrap-example]

Um exemplo de uso do `DatabaseTraceWrapper`:

```py
import newrelic.agent

def get(id):
    ...


get = newrelic.agent.DatabaseTraceWrapper(
    get, sql="SELECT * FROM users WHERE id = 1"
)
```
