[Os middlewares de rack](/docs/agents/ruby-agent/frameworks/rack-middlewares) são automaticamente instrumentados com o agente Ruby versão 3.9.0 ou superior.

O módulo `NewRelic::Agent::Instrumentation::Rack` está obsoleto. Após atualizar para o agente Ruby versão 3.9.0 ou superior, a New Relic recomenda que você remova quaisquer referências a este módulo do seu código. Para mais informações, veja [Controladores Metal instrumentados](/docs/agents/ruby-agent/frameworks/instrumenting-metal-controllers).
