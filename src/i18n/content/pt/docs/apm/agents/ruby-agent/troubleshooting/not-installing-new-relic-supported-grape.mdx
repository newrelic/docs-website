## Problema

Você recebe esta mensagem do log informando que o Grape não está instalando:

```
INFO : Not installing New Relic supported Grape instrumentation because the third party newrelic-grape gem is present
```

## Solução

Remova a gema `newrelic-grape` .

## Causa

A New Relic não oferece suporte à gem `newrelic-grape` de terceiros. Se o agente Ruby detectar a gema `newrelic-grape` , ele não instalará [a instrumentação Grape integrada](/docs/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) do New Relic. A remoção da gema de terceiros permite que você use a instrumentação Grape suportada.
