Para aproveitar ao máximo os recursos, melhorias e patches de segurança importantes do New Relic, recomendamos que você atualize seu agente Ruby para a versão mais recente. Para obter informações adicionais sobre atualizações específicas do agente, consulte as [notas de versão do agente Ruby](/docs/release-notes/agent-release-notes/ruby-release-notes).

## Verifique as versões do seu agente Ruby [#determine-versions]

Para determinar quais versões do agente Ruby seus aplicativos usam atualmente:

1. No New Relic, no [menu do usuário](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings), selecione

   <DNT>
     **Account settings > Connected agents**
   </DNT>

   .

2. Opcional: Classifique por

   <DNT>
     **Oldest agent version**
   </DNT>

   .

## Atualizar o agente [#Updating_the_Agent]

Para atualizar o agente Ruby:

* Usando Bundler: Execute este comando:

  ```
  bundle update newrelic_rpm
  ```

* Não usando Bundler: Execute este comando:

  ```
  gem update newrelic_rpm
  ```
