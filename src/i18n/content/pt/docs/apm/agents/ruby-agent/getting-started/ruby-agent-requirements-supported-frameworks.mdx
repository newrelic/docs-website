Antes de [instalar o agente Ruby da New Relic](/docs/agents/ruby-agent/installation-and-configuration/ruby-agent-installation), certifique-se de atender a estes requisitos de sistema operacional compatível, requisitos de segurança e estrutura suportada.

Se você ainda não tem uma, comece [criando uma conta New Relic](https://newrelic.com/signup). É grátis, para sempre.

## Sistema operacional [#os]

O agente Ruby oferece suporte a sistemas operacionais semelhantes a UNIX, como Linux, Solaris, FreeBSD e macOS.

## Requisitos de segurança [#security]

Como [medida de segurança padrão para coleta de dados](/docs/accounts-partnerships/accounts/security/data-security), o servidor do seu aplicativo deve oferecer suporte a SHA-2 (256 bits). SHA-1 não é compatível. New Relic requer TLS 1.2 ou superior para todas as conexões de entrada.

## Outros softwares APM [#other-monitoring-software]

Se a sua aplicação utiliza outro software de monitoramento do desempenho de aplicativos (APM) (APM) além do nosso agente, não podemos garantir que nosso agente funcionará corretamente e não podemos oferecer suporte técnico. Para obter mais informações, consulte [Erros ao usar outro software de monitoramento](/docs/apm/new-relic-apm/troubleshooting/errors-while-using-new-relic-apm-alongside-other-apm-software).
