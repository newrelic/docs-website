Este documento explica como desinstalar o agente New Relic Ruby. Para obter instruções sobre como desabilitar temporariamente o agente, consulte [Desabilitar o agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#ruby).

Para desinstalar o agente Ruby:

1. Exclua a linha `gem 'newrelic_rpm'` do seu Gemfile.

2. Remova o arquivo

   <DNT>
     **newrelic.yml**
   </DNT>

   da pasta

   <DNT>
     **APP_ROOT/config/**
   </DNT>

   .

3. Execute

   <DNT>
     **bundle install**
   </DNT>

   .

4. Quando o processo de desinstalação terminar, reinicie seu aplicativo.

Para obter mais informações sobre instalação, consulte [New Relic for Ruby](/docs/agents/ruby-agent/getting-started/new-relic-ruby).
