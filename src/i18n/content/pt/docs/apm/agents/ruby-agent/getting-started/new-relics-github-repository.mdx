O agente New Relic Ruby agora é totalmente open source e está disponível no GitHub em [newrelic-ruby-agente](https://github.com/newrelic/newrelic-ruby-agent/tree/main). A partir da versão 6.12 (lançada em 16 de julho de 2020), está licenciado sob a licença Apache, versão 2.0. Para versões mais antigas do agente, verifique o arquivo LICENSE incluído no código-fonte. O código-fonte para as versões 2.1.0 do agente Ruby estão disponíveis no GitHub usando tag.

O chefe de `main` terá o lançamento mais recente da New Relic. No entanto, reservamos a capacidade de empurrar uma vantagem para `main`. A ramificação `dev` inclui trabalhos não lançados. Se você baixar uma versão do GitHub, use a tag apropriada. Geralmente lançamos versões prévias via GitHub antes de publicar uma nova versão no RubyGems.

Aproveite e sinta-se à vontade para [nos enviar solicitações pull](https://github.com/newrelic/newrelic-ruby-agent/pulls), [criar problemas](https://github.com/newrelic/newrelic-ruby-agent/issues/new/choose) e [iniciar discussões](https://github.com/newrelic/newrelic-ruby-agent/discussions) diretamente no repositório GitHub. Adoramos trabalhar com a comunidade!
