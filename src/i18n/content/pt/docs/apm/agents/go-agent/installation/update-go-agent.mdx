Para aproveitar ao máximo os recursos, melhorias e patches de segurança importantes mais recentes do New Relic, atualize o agente Go do seu aplicativo para a versão mais recente. Para obter informações adicionais sobre atualizações específicas do agente, consulte as [notas de versão do agente Go](/docs/release-notes/agent-release-notes/go-release-notes).

## Atualize sua versão do agente Go [#update-go]

Para atualizar o agente Go, siga os procedimentos padrão para executar o processo a seguir como um comando de terminal ou como um script.

1. Em [http://github.com/newrelic/go-agent](http://github.com/newrelic/go-agent), use este processo:

   ```bash
   go get -u github.com/newrelic/go-agent/v3/newrelic
   ```

2. Compile e implante sua aplicação.
