A interface do usuário [do APM](/docs/apm) fornece uma variedade de dados sobre seu aplicativo, incluindo recurso especial para o [agente Go para APM](/docs/agents/go-agent/get-started/new-relic-go). Além disso, a página <DNT>**Go runtime**</DNT> fornece dados de tempo de execução importantes, úteis para resolução de problemas de desempenho.

## Visualizar e filtrar dados de tempo de execução do Go [#find]

Para visualizar a página <DNT>**Go runtime**</DNT> , vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (select an app) > Go runtime**</DNT>.
