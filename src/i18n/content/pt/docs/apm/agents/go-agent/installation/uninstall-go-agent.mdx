Continue lendo para saber como desinstalar o agente New Relic Go. Para obter instruções sobre como desabilitar temporariamente o agente Go, consulte [Desabilitar o agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#go).

Para desinstalar o New Relic for Go:

1. Desinstale o pacote New Relic usando seu método de desinstalação preferido.
2. Recompile e reinicie seu aplicativo.

Depois de desinstalar o pacote New Relic, você poderá [remover o aplicativo da sua conta New Relic](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui).
