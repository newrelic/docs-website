Para garantir que você tenha a versão mais atualizada do SDK C do New Relic para a biblioteca de códigos do seu aplicativo, verifique as [notas de versão](/docs/release-notes/agent-release-notes/c-release-notes).

## Atualize sua biblioteca de códigos C SDK [#update-production]

Para atualizar a biblioteca de códigos do seu aplicativo para a versão mais recente do C SDK da New Relic:

1. Revise as alterações da biblioteca C SDK para verificar se seu código é compatível.
2. Atualize manualmente o código do seu aplicativo conforme aplicável.
3. Compile o código C SDK usando o comando [`make`](https://github.com/newrelic/c-sdk#building) para criar a versão mais recente do `libnewrelic.a` e vinculá-la ao seu aplicativo ou serviço.
4. Se aplicável, reimplante seu aplicativo no ambiente de teste.
5. Gere tráfego para seu aplicativo e verifique se há erros no arquivo `c_sdk.log` e na saída `newrelic-daemon.log` .
6. Reimplante seu aplicativo em seu ambiente de produção.
7. Aguarde alguns minutos para que seu aplicativo envie dados para o New Relic. Em seguida, verifique o desempenho do seu aplicativo no [New Relic](/docs/apm/applications-menu/monitoring/apm-overview-page).

Se nenhum dado aparecer em alguns minutos, siga as [dicas de resolução de problemas](/docs/apm/agents/c-sdk/troubleshooting/no-data-appears-c-sdk).
