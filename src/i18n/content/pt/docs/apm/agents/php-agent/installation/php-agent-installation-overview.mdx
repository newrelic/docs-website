## Componentes PHP [#components]

Nosso [agente PHP](/docs/agents/php-agent/getting-started/new-relic-php) consiste em dois componentes básicos:

* Uma extensão PHP, que coleta dados do seu aplicativo
* Um daemon proxy local, que transmite os dados para a New Relic

Os componentes de configuração do seu sistema, incluindo o servidor web, sistema operacional, permissões do sistema, a própria instalação do PHP, gerenciador de pacote, etc., são fatores importantes durante a instalação e configuração, porque o agente faz parte disso.

Para obter mais informações sobre versões de PHP suportadas, estrutura e outros requisitos do sistema, consulte [Compatibilidade e requisitos de PHP](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements).

## Instale o agente [#install-overview]

Siga este processo geral para instalar o agente em seu sistema. O pacote de instalação do agente ou arquivo tar inclui o [script<DNT>**`newrelic-install`**</DNT> ](/docs/agents/php-agent/installation/newrelic-install-script)interativo, que automatiza algumas tarefas de instalação.

<Callout variant="tip">
  Se o seu aplicativo fica inativo com frequência, recomendamos que você gere alguns dados interagindo com o aplicativo ou site. Isso ajudará a garantir que o agente se conecte inicialmente.
</Callout>

1. Se ainda não o fez, [crie uma conta New Relic](https://newrelic.com/signup). É grátis, para sempre.

2. Certifique-se de que seu sistema atenda aos [requisitos do agente](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements), incluindo [as permissões de sistema](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#permissions) apropriadas.

3. Em seu <DNT>**Account settings**</DNT>, copie suas informações <InlinePopover type="licenseKey"/>.

4. Instale o pacote do agente ou arquivo tar em seu sistema.

   <CollapserGroup>
     <Collapser
       id="install-redhat-centos"
       title="RedHat ou CentOS"
     >
       A forma mais comum de instalar o agente no RedHat ou CentOS é com o pacote de instalação (<DNT>**.rpm**</DNT>) e o gerenciador de pacote. Você pode [solicitar ao gerenciador do pacote](/docs/agents/php-agent/installation/php-agent-installation-redhat-centos#tell-rpm) que o obtenha no repositório da New Relic ou pode [fazer download do arquivo <DNT>**.rpm**</DNT> ](/docs/agents/php-agent/installation/php-agent-installation-redhat-centos#download-rpm)diretamente do site da New Relic.

       [Instale o pacote](/docs/agents/php-agent/installation/php-agent-installation-redhat-centos) com seu gerenciador de pacote preferido (`yum` ou `rpm`) e execute o script <DNT>**newrelic-install**</DNT> .
     </Collapser>

     <Collapser
       id="intall-ubuntu-debian"
       title="Ubuntu ou Debian"
     >
       [Instale o pacote](/docs/agents/php-agent/installation/php-agent-installation-ubuntu-debian) com comandos `apt-get` ou `dpkg` . Normalmente, a execução do script <DNT>**newrelic-install**</DNT> não é necessária.
     </Collapser>

     <Collapser
       id="install-tar"
       title="Arquivo alcatrão"
     >
       O método de arquivo <DNT>**.tar**</DNT> é o método mais genérico que você pode usar em todos [os sistemas operacionais suportados](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#operating-systems). [Baixe o arquivo tar compactado](/docs/agents/php-agent/installation/php-agent-installation-tar-file#download) para o seu sistema se alguma destas situações se aplicar a você:

       * Seu sistema operacional não está listado aqui.

       * Você não está usando um gerenciador de pacote.

       * Outros métodos não tiveram sucesso.

         [Instale o arquivo tar](/docs/agents/php-agent/installation/php-agent-installation-tar-file#install) do agente em qualquer [plataforma suportada](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements).
     </Collapser>
   </CollapserGroup>

5. Altere o [nome padrão do aplicativo](/docs/site/naming-your-application) para um nome significativo.

6. Opcional: altere outras definições [de configuração](/docs/agents/php-agent/configuration/php-agent-configuration) do agente para personalizar ainda mais sua instalação.

7. Reinicie seu servidor web (Apache, Nginx, PHP-FPM, etc.).

8. Recomendação: Para ajudar a garantir que o agente PHP seja iniciado, especialmente se seu aplicativo tiver atividades pouco frequentes, gere alguns dados usando o aplicativo por alguns segundos.

9. Aguarde alguns minutos para que seu aplicativo envie dados para o New Relic.

Em seguida, [verifique o desempenho do seu aplicativo na interface do New Relic](/docs/apm/applications-menu/monitoring/apm-overview-page).

## Visualize o log do seu APM e dados de infraestrutura [#logs-context]

Você também pode reunir os dados do seu log e da aplicação para tornar a resolução de problemas mais fácil e rápida. Com [os logs contextualizados](/docs/logs/logs-context/configure-logs-context-php/), você pode ver a mensagem do log relacionada aos seus erros e rastrear diretamente na interface do seu aplicativo. Você também pode ver o logs contextualizados dos [dados da sua infraestrutura](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent/), como cluster do Kubernetes. Não há necessidade de mudar para outra página de interface.

## Instale PHP com contas de parceria [#partnership-accounts]

Os procedimentos de instalação podem ser diferentes para administradores que instalam o agente PHP por meio de uma parceria New Relic. Para mais informações, veja:

* [AWS](/docs/accounts-partnerships/partnerships/partner-based-installation/amazon-web-services-aws-users)
* [Heroku](/docs/agents/php-agent/installation/php-agent-heroku)
* [Magento](/docs/accounts-partnerships/partnerships/partner-based-installation/log-install-new-relic-partners#magento)
* [Provedores de hospedagem compartilhada](/docs/agents/php-agent/installation/install-php-agent-shared-hosting-service)
* [Outros procedimentos de instalação de parceria](/docs/accounts-partnerships/partnerships/partner-based-installation/log-install-new-relic-partners)

Nem todos os parceiros oferecem suporte ao agente PHP.
