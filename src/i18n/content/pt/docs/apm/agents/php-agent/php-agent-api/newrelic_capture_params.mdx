## Sintaxe

```php
newrelic_capture_params([bool $enable_flag])
```

Habilite ou desabilite a captura do parâmetro URL.

## Requisitos

Compatível com todas as versões do agente.

## Descrição

Habilite ou desabilite a captura do parâmetro URL (tudo após o `?` na URL). Isso substitui a configuração do arquivo de configuração `newrelic.capture_params` .

<Callout variant="important">
  Se você passar informações confidenciais diretamente na URL, mantenha esta opção desativada.
</Callout>

## Exemplos

### Habilitar captura do parâmetro URL [#enable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params();
    }
    //...
}
```

### Desabilitar captura do parâmetro URL [#disable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params(false);
    }
    //...
}
```
