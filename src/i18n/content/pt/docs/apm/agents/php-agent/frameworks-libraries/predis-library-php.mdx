New Relic adicionou suporte para a biblioteca Predis na [versão 4.22](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-422099) do agente PHP.

O agente instrumento chama padrão para um objeto `Predis\Client` ou para um pipeline. Quaisquer comandos enviados por meio de um pipeline serão agrupados e tratados como uma única métrica de comando chamada `pipeline` em vez de comandos individuais.

O agente PHP não oferece suporte a chamadas para os métodos `executeRaw` ou `transaction` , PubSub, monitor ou similares. Estes não serão instrumentados.
