A partir de março de 2026, para melhores práticas e segurança, as novas versões dos pacotes RPM do Agente PHP da New Relic serão assinadas com uma nova chave. Para continuar a operação do agente PHP sem interrupções, antes de instalar ou atualizar o agente PHP da New Relic a partir de pacotes RPM, o seguinte comando deve ser executado para aceitar e importar a nova chave pública.

`rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public`

## Verifique a chave

Após importar a chave, verifique se ela é confiável executando o seguinte comando:

`rpm -qa | grep gpg-pubkey-bd2e199c`

Se a chave for importada e confiável, o comando retornará a seguinte saída:

`gpg-pubkey-bd2e199c-6970e650`

## O que acontece se você não importar a nova chave antes de março?

Se a nova chave não for adicionada ao seu sistema, as instalações e atualizações de novas versões do Agente PHP falharão, com uma mensagem de erro como `Error: GPG check FAILED`. Quaisquer instalações fixadas em versões mais antigas do agente continuarão funcionando como antes (embora, nesses casos, importar a nova chave seja recomendado de qualquer forma).
