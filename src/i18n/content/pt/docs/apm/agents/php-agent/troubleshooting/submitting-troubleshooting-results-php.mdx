## Problema

Você continua tendo problemas com a instalação do agente PHP New Relic.

## Solução

Se você continuar tendo problemas com a instalação do New Relic PHP depois de seguir as [dicas de resolução de problemas](/docs/apm/agents/php-agent/troubleshooting/no-data-appears-php), obtenha suporte em [support.newrelic.com](https://support.newrelic.com).

Ao trabalhar com um engenheiro de suporte técnico da New Relic, forneça as seguintes informações:

* Até que ponto você progrediu no processo de instalação

* Qual método de instalação você está usando

* Uma cópia do seu arquivo `newrelic.cfg` (se houver)

* Um link para sua página <DNT>**phpinfo**</DNT> ou a saída de <DNT>**phpinfo**</DNT> salva como um arquivo HTML. Para [criar <DNT>**phpinfo**</DNT>](/docs/php/php-kb-103), crie um .php página da web contendo apenas:

  ```php
  <?php phpinfo(); ?>
  ```

* Seu [arquivo de log do agente e daemon](/docs/php/php-kb-105)
