[atributo](/docs/features/agent-attributes) são pares de valores principais contendo informações que determinam as propriedades de um evento ou transação. Esses pares de valores principais podem ajudá-lo a obter maiores insights sobre seu aplicativo e anotar os dados no New Relic. Tanto o atributo padrão quanto o [personalizado](/docs/agents/manage-apm-agents/agent-data/collect-custom-attributes) são visíveis no rastreamento de transação do APM, no rastreamento distribuído e na análise de erros; e [APM](/docs/insights/explore-data/attributes/apm-default-attributes-insights) e [evento do browser](/docs/insights/explore-data/attributes/browser-default-attributes-insights#browser-attributes-table).

Você pode personalizar exatamente qual atributo será enviado para cada um desses destinos. Continue lendo para aprender sobre o atributo do agente PHP, como ativá-lo ou desativá-lo e as regras que o agente segue para determinar qual atributo incluir ou excluir para um [destino](/docs/features/agent-attributes#destinations).

Consulte [Configurações do agente PHP (`newrelic.ini`)](/docs/php/php-agent-phpini-settings) e [Ativação e desativação do atributo](/docs/agents/php-agent/attributes/enabling-and-disabling-attributes) para obter mais informações.

## Alterar para onde os atributos são enviados [#change-attribute-destination]

<Callout variant="tip">
  Todos os destinos estão abertos à coleta de atributos por padrão no agente PHP, exceto `newrelic.browser_monitoring.attributes.enabled`.
</Callout>

Para alterar quais atributos são enviados para [destinos](/docs/apm/other-features/attributes/agent-attributes#destinations) New Relic:

1. Abra ou feche qualquer destino para coleção de atributos alterando a [propriedade de destino`.enabled` ](/docs/agents/php-agent/attributes/enabling-disabling-attributes#destination-on-off).

2. Altere a [configuração do atributo padrão](#attributes) adicionando o nome do atributo às [propriedades`.include` ou `.exclude` ](/docs/agents/php-agent/attributes/enabling-disabling-attributes#destination-attribute-specific)do destino.

   Veja [as regras de atributo](/docs/agents/php-agent/attributes/enabling-disabling-attributes#attruls) para quais configurações se substituem.

## Atualizando o agente PHP [#upgrading]

Ao atualizar para o agente PHP <DNT>**4.9**</DNT> ou superior, atualize seu arquivo de configuração. Para obter mais informações sobre propriedades obsoletas, consulte [Habilitando e desabilitando atributo](/docs/php/php-agent-enabling-and-disabling-attributes#deprecated).
