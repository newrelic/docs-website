Você pode instalar o agente PHP da New Relic a partir de arquivos tar compactados em qualquer sistema operacional compatível e em qualquer arquitetura de processador compatível. Porém, para RedHat ou CentOS em x86_64 é mais comum usar o [pacote RPM](/docs/agents/php-agent/installation/php-agent-installation-aws-linux-redhat-centos/). Para Ubuntu ou Debian em x86_64 é mais comum usar o [pacote Debian](/docs/agents/php-agent/installation/php-agent-installation-ubuntu-debian).

## Baixe a distribuição tar [#download]

Ao contrário de outras opções de instalação, os arquivos tar não requerem nenhuma configuração especial de repositório. Tudo que você precisa fazer é baixar o arquivo e seguir estas instruções.

Faça download do arquivo de distribuição tar apropriado em <DNT>**[download.newrelic.com/php_agent/release/](https://download.newrelic.com/php_agent/release/)**</DNT>:

* Para Linux com GNU libc (a maioria das distribuições Linux), faça download de `newrelic-php5-X.X.X.X-linux.tar.gz`.
* Para Linux com musl libc (Alpine Linux), faça download `newrelic-php5-X.X.X.X-linux-musl.tar.gz`.

<Callout variant="tip">
  O nome do pacote do agente PHP é `newrelic-php5`. Embora o nome do pacote faça referência ao PHP 5, este pacote funciona para todas [as versões suportadas do PHP](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#php-release).
</Callout>

## Instalar ou atualizar [#install]

Com a instalação do arquivo tar, as etapas para a primeira instalação e para a atualização são as mesmas. Substitua XXXX pela versão correta do New Relic. Para instalar ou atualizar o agente:

1. Baixe o arquivo tar apropriado no [site da New Relic](https://download.newrelic.com/php_agent/release/) e salve-o em um disco local em um local conveniente. Não use a pasta `/tmp` .

2. Descompacte e extraia o arquivo:

   ```bash
   gzip -dc newrelic-php5-X.X.X.X-OS.tar.gz | tar xf -
   ```

3. Mude para o diretório recém-criado:

   ```bash
   cd newrelic-php5-X.X.X.X-OS
   ```

4. Execute o script de instalação:

   ```bash
   ./newrelic-install
   ```

5. Altere o [nome padrão do aplicativo](/docs/site/naming-your-application) para um nome significativo.

6. Reinicie seu servidor web (Apache, Nginx, PHP-FPM, etc.).

7. Aguarde alguns minutos para que seu aplicativo envie dados e [verifique o desempenho do seu aplicativo na interface](/docs/apm/applications-menu/monitoring/apm-overview-page).
