## Problema

Você encontra problemas com a instalação do New Relic for PHP, mas não tem certeza de qual é a causa.

## Solução

O monitoramento de seus New Relic Logs pode ajudar a diagnosticar o problema. O agente PHP e o daemon do New Relic possuem seu próprio log. Para monitor o registro:

1. Verifique o conteúdo de ambos os logs:

   * <DNT>
       **[Default agent log location](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-logfile)**
     </DNT>

     : `/var/log/newrelic/php_agent.log`

   * <DNT>
       **[Default daemon log location](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-daemon-logfile)**
     </DNT>

     : `/var/log/newrelic/newrelic-daemon.log`

2. Defina o nível de log do agente como `verbosedebug` e o nível de log do daemon como `debug`. Para maiores informações:

   * [Configurando o agente](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-loglevel)

     <DNT>
       [**loglevel**](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-loglevel)
     </DNT>

     [para](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-loglevel)

     <DNT>
       [**verbosedebug**](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-loglevel)
     </DNT>

   * [Configurando o daemon](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-daemon-loglevel)

     <DNT>
       [**loglevel**](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-daemon-loglevel)
     </DNT>

     [para](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-daemon-loglevel)

     <DNT>
       [**debug**](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-daemon-loglevel)
     </DNT>

3. Reinicie seu aplicativo.

4. Colete de 5 a 10 minutos de tempo log com seu nível de registro definido com os valores apropriados.

5. Examine seu log e use as informações para diagnosticar o problema.

6. Quando terminar a resolução de problemas, redefina o nível de log. Não os mantenha em `verbosedebug` ou `debug`.

<Callout variant="caution">
  Observe o tamanho do seu arquivo de log. Dependendo do seu aplicativo, eles podem crescer muito rapidamente.
</Callout>
