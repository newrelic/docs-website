## Sintaxe

```php
newrelic_set_user_id(string $user_id)
```

Associe um ID de usuário exclusivo à transação atual.

## Requisitos

Versão do agente 10.12 ou superior.

## Descrição

Esta API permite que um aplicativo atribua um ID de usuário exclusivo à transação atual.

Isso permitirá que um usuário obtenha insights sobre o usuário final individual, como o impacto de um evento de erro.

## Valores de retorno

Esta função retornará `true` se o ID do usuário tiver sido adicionado com sucesso, caso contrário, `false`.

## Exemplos

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $user_id = "YOUR UNIQUE USER ID";
        newrelic_set_user_id($user_id);
    }
}
```
