<Callout variant="important">
  A chave de assinatura RPM do agente PHP da New Relic será rotacionada em março de 2026. Para continuar usando o agente sem interrupções, a nova chave deve ser importada em todos os sistemas instalando via rpm/yum com o seguinte comando. Para mais informações, consulte [Instalação do agente PHP: rotação de chave RPM 2026](/docs/apm/agents/php-agent/installation/php-agent-installation-rpm-key-rotation).

  `rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public`
</Callout>

Nosso agente PHP autoinstrumenta seu código para que você possa iniciar o monitoramento do aplicativo. Siga este procedimento para instalar o agente PHP da New Relic para APM usando Amazon Linux 2, RedHat ou CentOS na arquitetura x86\_64. Para instalação na arquitetura aarch64, use o método [de arquivo tar](/docs/php/php-agent-installation-tar-files) . <DNT>**Exception:**</DNT> Se você tiver uma versão anterior instalada, [atualize o agente](/docs/agents/php-agent/installation/upgrading-php-agent) .

## Instale o agente [#install]

Embora o nome do pacote do agente PHP da New Relic se refira ao PHP 5, o pacote funciona para todas [as versões PHP suportadas](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#php-release).

1. Certifique-se de ter seu <InlinePopover type="licenseKey" />acessível.

2. Adicione a nova chave RPM (Aplicável apenas a partir de março de 2026.)

   ```bash
   rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public
   ```

3. Use uma das seguintes maneiras para obter o pacote de instalação:

   <CollapserGroup>
     <Collapser
       id="tell-rpm"
       title={<>Informe ao gerenciador de pacote (<InlineCode>
         rpm
       </InlineCode>) sobre o repositório New Relic</>
       }
     >
       ```bash
       sudo rpm -Uvh http://yum.newrelic.com/pub/newrelic/el5/x86_64/newrelic-repo-5-3.noarch.rpm
       ```
     </Collapser>

     <Collapser id="download-rpm" title="Baixe o arquivo rpm do New Relic">
       * ```
         newrelic-php5-common-X.X.X.X-1.noarch.rpm
         ```
       * ```
         newrelic-daemon-X.X.X.X-1.x86_64.rpm
         ```
       * ```
         newrelic-php5-X.X.X.X-1.x86_64.rpm
         ```
     </Collapser>
   </CollapserGroup>

4. Instale o agente e o daemon usando seu gerenciador de pacote preferido:

   <CollapserGroup>
     <Collapser id="manager-yum" title="yum">
       ```bash
       sudo yum install newrelic-php5
       ```

       Na primeira vez que você instalar o New Relic para PHP, yum solicitará que você aceite a chave pública do New Relic. O ID da chave da New Relic é `548C16BF`.
     </Collapser>

     <Collapser id="manager-rpm" title="rpm">
       Substitua `X.X.X.X` pelo [número de versão mais recente do agente PHP](/docs/release-notes/agent-release-notes/php-release-notes) ao executar este comando:

       ```bash
       rpm -i newrelic-php5-common-X.X.X.X-1.noarch.rpm newrelic-daemon-X.X.X.X-1.x86_64.rpm newrelic-php5-X.X.X.X-1.x86_64.rpm
       ```
     </Collapser>

     <Collapser id="manager-tarball" title="tarball">
       Se <DNT>**yum**</DNT> e <DNT>**rpm**</DNT> não funcionarem com a configuração do seu host, [instale a partir do tarball binário](/docs/php/php-agent-installation-tar-files).
     </Collapser>
   </CollapserGroup>

5. Execute o script `newrelic-install` e siga as instruções.

   ```bash
   sudo newrelic-install install
   ```

6. Reinicie seu servidor web (Apache, NGINX, PHP-FPM, etc.).

7. Gere tráfego para seu aplicativo e aguarde alguns minutos para que ele envie dados para o New Relic. Em seguida, [verifique o desempenho do seu aplicativo na interface](/docs/apm/applications-menu/monitoring/apm-overview-page).
