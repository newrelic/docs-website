Nosso agente PHP instrumenta automaticamente seu código para que você possa iniciar o monitoramento do aplicativo. Siga este procedimento para instalar o agente PHP da New Relic para APM usando Amazon Linux 2, RedHat ou CentOS. <DoNotTranslate>**Exception:**</DoNotTranslate> Se você tiver uma versão anterior instalada, [atualize o agente](/docs/agents/php-agent/installation/upgrading-php-agent) .

## Instale o agente [#install]

Embora o nome do pacote do agente PHP da New Relic se refira ao PHP 5, o pacote funciona para todas [as versões PHP suportadas](/docs/agents/php-agent/getting-started/php-agent-compatibility-requirements#php-release).

1. Certifique-se de ter seu <InlinePopover type="licenseKey"/>acessível.

2. Use uma das seguintes maneiras para obter o pacote de instalação:

   <CollapserGroup>
     <Collapser
       id="tell-rpm"
       title={<>Informe ao gerenciador de pacote (<InlineCode>rpm</InlineCode>) sobre o repositório New Relic</>}
     >
       Para <DoNotTranslate>**32-bit**</DoNotTranslate> sistemas, execute:

       ```bash
       sudo rpm -Uvh http://yum.newrelic.com/pub/newrelic/el5/i386/newrelic-repo-5-3.noarch.rpm
       ```

       Para <DoNotTranslate>**64-bit**</DoNotTranslate> sistemas, execute:

       ```bash
       sudo rpm -Uvh http://yum.newrelic.com/pub/newrelic/el5/x86_64/newrelic-repo-5-3.noarch.rpm
       ```
     </Collapser>

     <Collapser
       id="download-rpm"
       title="Baixe o arquivo rpm do New Relic"
     >
       Para sistemas <DoNotTranslate>**32-bit**</DoNotTranslate> , baixe estes três arquivos dos [pacotes de 32 bits](https://download.newrelic.com/pub/newrelic/el5/i386/) (substituindo `X.X.X.X` pelo [número de versão mais recente do agente PHP](/docs/release-notes/agent-release-notes/php-release-notes)):

       * ```
         newrelic-php5-common-X.X.X.X-1.noarch.rpm
         ```

       * ```
         newrelic-daemon-X.X.X.X-1.i386.rpm
         ```

       * ```
           newrelic-php5-X.X.X.X-1.i386.rpm
         ```

         Para sistemas <DoNotTranslate>**64-bit**</DoNotTranslate> , faça download destes três arquivos dos [pacotes de 64 bits](https://download.newrelic.com/pub/newrelic/el5/x86_64/) (substituindo `X.X.X.X` pelo [número de versão mais recente do agente PHP](/docs/release-notes/agent-release-notes/php-release-notes)):

       * ```
         newrelic-php5-common-X.X.X.X-1.noarch.rpm
         ```

       * ```
         newrelic-daemon-X.X.X.X-1.x86_64.rpm
         ```

       * ```
         newrelic-php5-X.X.X.X-1.x86_64.rpm
         ```
     </Collapser>
   </CollapserGroup>

3. Instale o agente e o daemon usando seu gerenciador de pacote preferido:

   <CollapserGroup>
     <Collapser
       id="manager-yum"
       title="yum"
     >
       ```bash
       sudo yum install newrelic-php5
       ```

       Na primeira vez que você instalar o New Relic para PHP, yum solicitará que você aceite a chave pública do New Relic. O ID da chave da New Relic é `548C16BF`.
     </Collapser>

     <Collapser
       id="manager-32-bit-rpm"
       title="RPM de 32 bits"
     >
       Substitua `X.X.X.X` pelo [número de versão mais recente do agente PHP](/docs/release-notes/agent-release-notes/php-release-notes) ao executar este comando:

       ```bash
       rpm -i newrelic-php5-common-X.X.X.X-1.noarch.rpm newrelic-daemon-X.X.X.X-1.i386.rpm newrelic-php5-X.X.X.X-1.i386.rpm
       ```
     </Collapser>

     <Collapser
       id="manager-64-bit-rpm"
       title="RPM de 64 bits"
     >
       Substitua `X.X.X.X` pelo [número de versão mais recente do agente PHP](/docs/release-notes/agent-release-notes/php-release-notes) ao executar este comando:

       ```bash
       rpm -i newrelic-php5-common-X.X.X.X-1.noarch.rpm newrelic-daemon-X.X.X.X-1.x86_64.rpm newrelic-php5-X.X.X.X-1.x86_64.rpm
       ```
     </Collapser>

     <Collapser
       id="manager-tarball"
       title="tarball"
     >
       Se <DoNotTranslate>**yum**</DoNotTranslate> e <DoNotTranslate>**rpm**</DoNotTranslate> não funcionarem com a configuração do seu host, [instale a partir do tarball binário](/docs/php/php-agent-installation-tar-files).
     </Collapser>
   </CollapserGroup>

4. Execute o script `newrelic-install` e siga as instruções.

   ```bash
   sudo newrelic-install install
   ```

5. Reinicie seu servidor web (Apache, NGINX, PHP-FPM, etc.).

6. Gere tráfego para seu aplicativo e aguarde alguns minutos para que ele envie dados para o New Relic. Em seguida, [verifique o desempenho do seu aplicativo na interface](/docs/apm/applications-menu/monitoring/apm-overview-page).
