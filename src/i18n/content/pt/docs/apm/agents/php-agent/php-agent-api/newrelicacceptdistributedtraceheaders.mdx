## Sintaxe

```php
newrelic_accept_distributed_trace_headers (array $headers, string $transport_type)
```

Aceita uma matriz de cabeçalhos distributed trace .

## Requisitos

Requer agente PHP [versão 9.8 ou superior](/docs/release-notes/agent-release-notes/php-release-notes).

## Descrição

[distributed tracing](/docs/agents/php-agent/features/distributed-tracing-php) permite que você veja o caminho que uma solicitação percorre ao percorrer sistemas distribuídos. Quando distributed tracing estiver ativado, use `newrelic_accept_distributed_trace_headers` para aceitar uma carga de cabeçalhos. Isso inclui W3C Trace Context e os cabeçalhos distributed trace New Relic.

É possível aceitar apenas cabeçalhos W3C Trace Context e desativar o cabeçalho distributed tracing New Relic por meio da configuração INI `newrelic.distributed_tracing_exclude_newrelic_header`.

## Valores de retorno

Retorna `true` se os cabeçalhos foram aceitos com sucesso, caso contrário, retorna `false`.
