## Sintaxe

```php
newrelic_ignore_transaction()
```

Não instrumento a transação atual.

## Requisitos

Compatível com todas as versões do agente.

## Descrição

Não gere dados para esta transação. Isso é útil quando você tem transações que são particularmente lentas por motivos conhecidos e você não deseja que elas gerem frequentemente rastreamento de transação ou distorçam as médias do seu site.

## Parâmetro

Esta chamada não aceita nenhum parâmetro.

## Exemplos

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_ignore_transaction();
    }
    ...
}
```
