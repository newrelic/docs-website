## Sintaxe

```php
newrelic_get_browser_timing_header([bool $include_tags])
```

Retorna um trecho <InlinePopover type="browser"/>para injetar no cabeçalho da saída HTML.

## Descrição

Retorna uma string JavaScript para injetar no `<head>` da sua saída HTML. Use esta chamada com [`newrelic_get_browser_timing_footer`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer) para adicionar manualmente o monitoramento do browser a uma página da web. Se possível, a New Relic recomenda ativar o monitoramento do browser por meio da interface da New Relic ou copiar/colar o trecho JavaScript. Para instruções sobre como usar essas opções, consulte [Monitoramento de Browser e PHP](/docs/agents/php-agent/features/page-load-timing-php).

<Callout variant="tip">
  Compare [`newrelic_disable_autorum`](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum), que <DNT>**disables**</DNT> é o script do browser em uma página.
</Callout>

## Valores de retorno

Retorna a string do cabeçalho JavaScript do agente browser.

## Exemplos

### Instrumentar para página da web [#webpage]

```php
function example(){
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_header = newrelic_get_browser_timing_header();
    }
    
    // Output to page
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_footer = newrelic_get_browser_timing_footer();
    }
}
```
