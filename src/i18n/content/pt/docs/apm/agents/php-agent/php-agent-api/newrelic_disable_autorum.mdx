## Sintaxe

```php
newrelic_disable_autorum()
```

Desative a injeção automática do trecho <InlinePopover type="browser"/>em páginas específicas.

## Requisitos

Versão do agente [2.6.5.41](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-26541) ou superior.

## Descrição

Esta chamada desabilita a injeção automática do agente [de monitoramento de Browser](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) para a transação atual. Ligue o mais cedo possível. Você pode usar essa chamada para remover o JavaScript se a [inserção estiver causando problemas](/docs/browser/new-relic-browser/troubleshooting/browser-javascript-injection-causes-problems-page) ou se você estiver veiculando páginas para serviços de terceiros que não permitem JavaScript (por exemplo, [as páginas aceleradas para dispositivos móveis do Google](/docs/browser/new-relic-browser/troubleshooting/google-amp-validator-fails-due-3rd-party-script)).

<Callout variant="tip">
  Para <DNT>**enable**</DNT> monitoramento do browser com uma chamada de API, veja:

  * [`newrelic_get_browser_timing_footer()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer)
  * [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header)
</Callout>

## Parâmetro

Esta chamada não aceita nenhum parâmetro.

## Valores de retorno

Retorna `true` se chamado em uma transação New Relic. Caso contrário, retornará `null` se estiver fora de uma transação (por exemplo, se `newrelic_end_transaction()` tiver sido chamado).

## Exemplos

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_disable_autorum();
    }
    // ...
}
```
