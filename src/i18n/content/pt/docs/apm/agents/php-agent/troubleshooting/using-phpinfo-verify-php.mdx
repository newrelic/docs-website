## Problema

O New Relic não relata dados alguns minutos após reiniciar o servidor web após a instalação do agente PHP.

## Solução

Use `phpinfo()` para verificar se o agente foi instalado com sucesso:

1. Crie uma página `.php` simples em seu site que contenha o seguinte:

   ```php
   <?php phpinfo(); ?>
   ```

2. Navegue até esta página em um browser da web e percorra a lista alfabética de módulos para encontrar `newrelic`. Se você não vir `newrelic` nesta lista, você está usando uma versão não padrão do PHP ou não reiniciou seu servidor web (ou outro host PHP) desde a instalação do New Relic.

3. Reinicie seu servidor web para ver se `newrelic` aparece em sua lista de módulos.

4. Se `newrelic` não aparecer em sua lista de módulos, [verifique seu diretório de extensões](/docs/php/php-kb-104) para ver se você possui uma [versão não padrão do PHP](/docs/php/php-agent-installation-non-standard-php).

5. Se `newrelic` aparecer na sua lista de módulos, verifique se a entrada `license.key` mostra o primeiro e o último caractere do seu<InlinePopover type="licenseKey"/>

6. Se `license.key` não estiver aparecendo, [verifique o diretório dos arquivos de configuração carregados](/docs/agents/php-agent/troubleshooting/checking-loaded-configuration-files-directory)
