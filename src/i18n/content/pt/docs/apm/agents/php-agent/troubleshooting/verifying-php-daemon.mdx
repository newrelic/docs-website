## Problema

Você instalou o agente PHP e o daemon do New Relic, mas o daemon parece não estar funcionando.

## Solução

Verifique se o daemon está funcionando corretamente:

1. Execute este comando shell para listar os processos em execução que correspondem a "newrelic-daemon":

   ```bash
   ps -ef | grep newrelic-daemon
   ```

2. Verifique se o daemon está em execução verificando a saída do comando anterior para dois processos daemon em execução semelhantes a este. Se eles não estiverem listados, prossiga para a próxima etapa.

   ```console
   newrelic     1368     1  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   newrelic     1370  1368  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   ```

3. Se nenhum processo estiver listado, seu daemon não está em execução e você deve iniciá daemon com base no modo de inicialização em que ele está:

   * [Inicialização no modo agente](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-external) (padrão)
   * [Inicialização em modo externo](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-autostart) (avançado)
