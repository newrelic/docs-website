Este documento explica como desinstalar o agente Java New Relic. Para obter instruções sobre como desabilitar temporariamente o agente, consulte [Desabilitar o agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#java).

Para remover o agente Java:

1. Exclua a pasta `newrelic` do diretório raiz do servidor de aplicativos.
2. Remova o [argumento JVM](/docs/agents/java-agent/installation/include-java-agent-jvm-argument) usado para incluir o agente em seu script de inicialização.
3. Se aplicável, renomeie o script inicial do backup com o nome original.
4. Depois de concluir essas etapas, [reinicie](/docs/agents/manage-apm-agents/installation/uninstall-agent#restart) seu aplicativo.

<Callout variant="tip">
  Você também pode [remover seu aplicativo](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) da interface do APM.
</Callout>
