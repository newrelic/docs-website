Se os procedimentos de resolução de problemas do agente Java (como [Nenhum dado aparece](/docs/apm/agents/java-agent/troubleshooting/no-data-appears-java) e outros documentos dessa categoria) não resolveram seu problema, use o [arquivo de log](#logfiles) para solucionar o problema. Se precisar de mais assistência, entre em contato com [support.newrelic.com](https://support.newrelic.com).
