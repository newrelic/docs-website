## Problema

Se seus [dados de registro](/docs/agents/java-agent/troubleshooting/no-log-file-java) mostrarem `SEVERE:` ou `ERROR:` com `NullPointerException` problemas, pode haver problemas que estão interferindo nos relatórios.

## Solução

Antes de começar essas dicas de resolução de problemas, certifique-se de ter a [versão mais recente](/docs/agents/java-agent/installation/upgrade-java-agent) do agente Java da New Relic.

1. Use pesquisas que diferenciam maiúsculas de minúsculas do arquivo de log do agente Java para linhas com `SEVERE:` e `ERROR: NullPointerException`.
2. Se você encontrar alguma linha com essas mensagens, [reúna informações sobre resolução de problemas](/docs/agents/java-agent/troubleshooting/providing-troubleshooting-information-java) para obter suporte na New Relic.

## Causa

Agente Java da New Relic entre versões 3.0.0 e 3.2.3 log incorretamente a mensagem de erro. Por exemplo:

```
ERROR: javax/servlet/ServletOutputStream is marked as a weaved class, but no methods are matched to be weaved.
```

Isso foi totalmente errado e não é motivo de preocupação. No entanto, não use essas versões. Em vez disso, atualize para a [versão mais recente](/docs/agents/java-agent/installation/upgrade-java-agent) do agente Java da New Relic.
