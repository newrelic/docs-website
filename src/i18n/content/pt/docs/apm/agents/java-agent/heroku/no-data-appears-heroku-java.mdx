## Problema

Depois de esperar alguns minutos, os dados não aparecem na [página APM<DNT>**Summary**</DNT> ](/docs/apm/applications-menu/monitoring/apm-overview-page)do seu aplicativo Heroku.

## Solução

1. Verifique se há erros no log do Heroku com este comando do cinto de ferramentas do Heroku:

   ```
   heroku logs --tail
   ```

2. Verifique se os arquivos em seu dinamômetro Heroku correspondem às modificações de arquivo dos procedimentos de instalação [Java no Heroku](/docs/agents/java-agent/heroku/java-agent-heroku) ou [Java com Scala no Heroku](/docs/agents/java-agent/heroku/java-agent-scala-heroku) . Use este comando do cinto de ferramentas Heroku:

   ```
   heroku run bash
   ```

3. Experimente as etapas em [Nenhum dado aparece (Java)](/docs/agents/java-agent/troubleshooting/no-data-appears-java).
