## Problema

Você vê uma mensagem de erro ao iniciar o servidor de aplicativos do seu aplicativo Java. Por exemplo:

```
Error opening zip file: newrelic.jar 
Error occurred during initialization of VM 
Agent library failed to init: instrument
```

## Solução

O caminho para o arquivo `jar` do agente Java deve ser totalmente qualificado no comutador JVM `-javaagent` . Se o caminho não for especificado corretamente, erros serão gerados na inicialização do servidor de aplicativos e a JVM será encerrada.

Para corrigir esse problema, especifique o caminho completo para o arquivo `newrelic.jar` na opção `-javaagent` . Por exemplo, se o arquivo `newrelic.jar` estiver em um diretório chamado `newrelic` dentro do diretório inicial do Tomcat <DNT>**/home/tomcat**</DNT>, especifique a opção assim:

```
-javaagent:/home/tomcat/newrelic/newrelic.jar
```
