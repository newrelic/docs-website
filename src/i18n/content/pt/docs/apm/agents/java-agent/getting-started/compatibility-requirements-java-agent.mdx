[O agente Java da New Relic](/docs/agents/java-agent/getting-started/introduction-new-relic-java) inclui instrumentação integrada das partes mais populares do ecossistema Java, incluindo servidores de aplicativos, estrutura, banco de dados e sistemas de enfileiramento de mensagens. Para framework e biblioteca que não são instrumentados fora da caixa, você pode estender o agente com [instrumentação personalizada Java](/docs/agents/java-agent/custom-instrumentation/java-custom-instrumentation).

Quer experimentar o agente Java da New Relic? [Crie uma conta New Relic](https://newrelic.com/signup) gratuitamente! Não é necessário cartão de crédito.

    Como [medida de segurança padrão para coleta de dados](/docs/accounts-partnerships/accounts/security/data-security), o servidor do seu aplicativo deve oferecer suporte a SHA-2 (256 bits). SHA-1 não é compatível.

Para obter mais informações, consulte [Erros ao usar outro software de monitoramento](/docs/apm/new-relic-apm/troubleshooting/errors-while-using-new-relic-apm-alongside-other-apm-software).

## Instrumentação integrada [#auto-instrumented]

Depois de [instalar o agente Java](/docs/agents/java-agent/installation/install-java-agent), ele automaticamente utiliza muitas estruturas e bibliotecas populares. Com instrumentação automática, o agente coleta dados avançados prontos para uso, e os dados aparecerão no painel do New Relic minutos após a instalação. Mesmo que sua biblioteca não seja instrumentada automaticamente, você ainda pode coletar dados com [a instrumentação personalizada](/docs/agents/java-agent/custom-instrumentation/java-custom-instrumentation) e a [API do agente Java](/docs/agents/java-agent/api-guides/guide-using-java-agent-api).
