Leia sobre como confirmar a soma de verificação do instalador do agente .NET da New Relic. Se você precisar do hash aplicar para nossas versões atuais, [baixe o hash SHA256 aplicar em nosso site de download](https://download.newrelic.com/dot_net_agent/latest_release/SHA256/).

## Windows

No Windows, recomendamos usar CertUtil ou Get-FileHash para calcular um arquivo aplicar hash.

### CertUtil

Este utilitário está disponível como parte dos Serviços de Certificado e pode ser usado para calcular o hash de aplicação de um arquivo. Este utilitário pode ser executado no prompt de comando ou Powershell.

Para calcular o hash aplicar, execute:

`CertUtil -hashfile YOUR_FILENAME SHA256`

Se for bem-sucedido, você receberá esta resposta:

```
SHA256 hash of YOUR_FILENAME:
5092fd52e40132a41ac06c320296cd9d63196d6de0fa6ce9b74fc4de1b3f9502
CertUtil: -hashfile command completed successfully.
```

### Powershell: Get-FileHash [#getfilehash]

O cmdlet Get-FileHash está incluído no Powershell versão 5.1+ e pode ser usado para calcular o hash aplicar de um arquivo.

Para calcular o hash aplicar, execute:

`Get-FileHash YOUR_FILENAME Algorithm SHA256 | Format-List`

Se for bem-sucedido, você receberá esta resposta:

```
Algorithm : SHA256
Hash      : 5092FD52E40132A41AC06C320296CD9D63196D6DE0FA6CE9B74FC4DE1B3F9502
Path      : C:\Path\To\YOUR_FILENAME
```

## Linux

No Linux, recomendamos usar

```
sha256sum
```

para calcular um arquivo aplique hash.

### sha256sum [#she256sum]

Para calcular o hash aplicar, execute:

`sha256sum YOUR_FILENAME`

Se for bem-sucedido, você receberá esta resposta:

```
5092fd52e40132a41ac06c320296cd9d63196d6de0fa6ce9b74fc4de1b3f9502 YOUR_FILENAME
```
