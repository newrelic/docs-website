Com nosso agente .NET para monitoramento de desempenho de aplicativos (APM), você pode:

* Use [o APM](/docs/apm/new-relic-apm/getting-started/welcome-new-relic-apm) para obter uma [visão geral de alto nível do seu aplicativo](/docs/apm/applications-menu/monitoring/apm-overview-page), para ver detalhes de nível de código, como [rastreamento de transação](/docs/apm/transactions/transaction-traces/transaction-traces), [consulta ao banco de dados](/docs/apm/transactions/transaction-traces/sql-statements) e [erros](/docs/apm/applications-menu/events/view-apm-error-analytics), e para [rastrear atividades em grandes sistemas distribuídos](/docs/apm/distributed-tracing/getting-started/introduction-distributed-tracing).
* Receba notificações proativas de [alertas](/docs/alerts/alert-policies/understanding-alert-policies/alerting-new-relic) para garantir que seu aplicativo esteja funcionando perfeitamente.
* Utilize o [criador de consulta](/docs/query-your-data/explore-query-data/query-builder/use-advanced-nrql-mode-specify-data) para consultar seus dados e criar [dashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-new-relic-one-dashboards) customizado com esses dados.
* Instale [o monitoramento de infraestrutura](/docs/infrastructure/new-relic-infrastructure/installation/install-infrastructure-windows-server) para visualizar o desempenho do ambiente host do seu aplicativo.

## Suporte para .NET framework e .NET Core [#core-features]

O agente .NET da New Relic suporta [.NET Framework](/docs/agents/net-agent/getting-started/compatibility-requirements-net-framework-agent) e .[.NET Core](/docs/agents/net-agent/getting-started/net-agent-compatibility-requirements-net-core) e funciona com todas as linguagens compatíveis com .NET, como VB.NET, C# e CLI.

O suporte do agente para .NET Core aproveita a compatibilidade, a velocidade, o recurso de API expandido e os recursos de plataforma cruzada do .NET Core da Microsoft. O agente <DNT>**does not**</DNT> oferece suporte a versões do Microsoft .NET Core anteriores à 2.0.

Com o suporte da New Relic para .NET, você pode monitor seus aplicativos de forma dinâmica ou em ambiente distribuído, como:

* Imagens de VM de servidor gerenciado em nuvem
* Servidores VM no host
* Serviços de aplicativos do Microsoft Azure
* Sistemas Windows e Linux auto-hospedados
* VM AWS EC2

## Visualize o log do seu APM e dados de infraestrutura [#logs-context]

Você também pode reunir os dados do seu log e da aplicação para tornar a resolução de problemas mais fácil e rápida. Com [os logs contextualizados](/docs/logs/logs-context/net-configure-logs-context-all), você pode ver a mensagem do log relacionada aos seus erros e rastrear diretamente na interface do seu aplicativo. Você também pode ver o logs contextualizados dos [dados da sua infraestrutura](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent/), como cluster do Kubernetes. Não há necessidade de mudar para outra página de interface.

## Verifique o código-fonte [#source-code]

O agente .NET é um software de código aberto. Isso significa que você pode [navegar pelo código-fonte](https://github.com/newrelic/newrelic-dotnet-agent) e enviar melhorias, ou criar seu próprio fork e construí-lo. Para obter mais informações, consulte o [README](https://github.com/newrelic/newrelic-dotnet-agent/blob/main/README.md).
