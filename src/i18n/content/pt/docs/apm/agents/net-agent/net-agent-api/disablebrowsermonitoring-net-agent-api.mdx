## Sintaxe

```cs
NewRelic.Api.Agent.NewRelic.DisableBrowserMonitoring([boolean $override])
```

Desabilitar injeção automática de monitoramento de trecho do Browser em páginas específicas.

## Requisitos

Compatível com todas as versões do agente.

Deve ser chamado dentro de uma [transação](/docs/glossary/glossary/#transaction).

## Descrição

Adicione esta chamada para desativar a injeção <DoNotTranslate>**automatic**</DoNotTranslate> do script [<InlinePopover type="browser"/>](/docs/browser/new-relic-browser/getting-started/new-relic-browser)em páginas específicas. Você também pode adicionar uma substituição opcional para desativar a injeção manual e automática <DoNotTranslate>**both**</DoNotTranslate> . Em ambos os casos, coloque esta chamada de API o mais próximo possível do topo da visualização na qual você deseja desabilitar o browser.

<Callout variant="tip">
  Compare [`GetBrowserTimingHeader()`](/docs/agents/net-agent/net-agent-api/getbrowsertimingheader-net-agent), que <DoNotTranslate>**adds**</DoNotTranslate> é o script do browser com a página.
</Callout>

## Exemplos

### Desativar injeção automática [#automatic-only]

Este exemplo desativa apenas a injeção <DoNotTranslate>**automatic**</DoNotTranslate> do trecho:

```cs
NewRelic.Api.Agent.NewRelic.DisableBrowserMonitoring();
```

### Desativar injeção automática e manual [#disable-both]

Este exemplo desativa <DoNotTranslate>**both**</DoNotTranslate> injeção automática e manual do trecho:

```cs

```
