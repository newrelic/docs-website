## Problema

Você está se conectando via FTP ao seu [aplicativo Web do Microsoft Azure](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps). Você pode ver um arquivo `NewRelic.Profiler.####.log` em <DNT>**LogFiles\\NewRelic**</DNT>, mas quando você tenta acessar o arquivo de log, uma mensagem de erro aparece.

## Solução

Reinicie seu aplicativo Web para liberar o arquivo e criar um novo log de perfil.
