Este documento explica como desinstalar o agente New Relic .NET. Para obter instruções sobre como desabilitar temporariamente o agente, consulte [Desabilitar o agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#net).

Para desinstalar o agente .NET, siga o procedimento que corresponde ao seu sistema operacional:

* [Windows](#uninstall-framework-windows)
* [Linux](#uninstall-linux)

## Desinstale o agente .NET (Windows) [#uninstall-framework-windows]

<Callout variant="tip">
  Requer [direitos<DNT>**Administrator**</DNT> ](/docs/agents/net-agent/getting-started/compatibility-requirements-net-framework-agent#user-rights)no seu grupo de administração do Windows.
</Callout>

Este procedimento pressupõe que você instalou o agente .NET com o instalador MSI:

1. Para encerrar qualquer agente em execução, pare o IIS. Se você não parar o IIS, poderá ser necessário reinicializar após desinstalar o agente.

2. Selecione

   <DNT>
     **Start > All Programs > New Relic .NET Agent > Uninstall the .NET agent**
   </DNT>

   .

3. Siga o prompt para concluir a desinstalação.

4. Reinicie o IIS.

Após desinstalar o agente, você pode querer remover todos os arquivos relacionados ao agente. Certifique-se de verificar outros caminhos para remover quaisquer outros arquivos New Relic. Por exemplo, desinstalar o agente .NET não exclui o arquivo [`newrelic.config`](/docs/agents/net-agent/installation-configuration/net-agent-configuration#config-options-precedence) .

1. Em

   <DNT>
     **Control Panel > Programs > Programs and Features**
   </DNT>

   , clique com o botão direito e exclua todos os arquivos chamados `New Relic .NET Agent`.

2. Exclua a pasta inteira em `C:\ProgramData\New Relic\.NET Agent`.

3. Exclua a pasta inteira em `C:\Program Files\New Relic\.NET Agent`.

4. Exclua a pasta inteira em `C:\Program Files (x86)\New Relic\.NET Agent`.

5. [Remova seu aplicativo](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) da interface do APM.

## Desinstale o agente .NET (Linux) [#uninstall-linux]

Para desinstalar o agente .NET no Linux, use sua ferramenta de gerenciamento de pacotes.

<Callout variant="important">
  Para o agente .NET versões 10.0.0 ou superior, o nome do pacote é `newrelic-dotnet-agent`. Para o agente .NET versões 9.9.0 ou inferior, o nome do pacote é `newrelic-netcore20-agent`. O restante deste documento se referirá a `newrelic-dotnet-agent`; substitua-o pelo nome mais antigo se você tiver uma versão mais antiga do agente instalada.
</Callout>

<Callout variant="important">
  Você deve reiniciar seu aplicativo assim que a desinstalação for concluída.
</Callout>

<CollapserGroup>
  <Collapser
    id="apt-uninstall"
    title="Desinstale com apt (Debian, Ubuntu e Linux Mint)"
  >
    Execute o seguinte comando como root (ou usando `sudo`):

    ```sh
    apt-get remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="dpkg-uninstall"
    title="Desinstalar com dpkg (Debian, Ubuntu e Linux Mint)"
  >
    Execute o seguinte comando como root (ou usando `sudo`):

    ```sh
    dpkg --remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="yum-uninstall"
    title="Desinstale com yum (CentOS, Red Hat Enterprise Linux, Oracle Linux)"
  >
    Execute o seguinte comando como root (ou usando `sudo`):

    ```sh
    yum remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="rpm-uninstall"
    title="Desinstalar com rpm (CentOS, Red Hat Enterprise Linux, Oracle Linux)"
  >
    Execute o seguinte comando como root (ou usando `sudo`):

    ```sh
    sudo rpm -e newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="tarball"
    title="Desinstalar tarball"
  >
    1. Exclua todos os arquivos New Relic da pasta do seu aplicativo.

    2. Desative as variáveis de ambiente do New Relic:

       ```ini
       CORECLR_ENABLE_PROFILING=1
       CORECLR_PROFILER={36032161-FFC0-4B61-B559-F6C5D41BAE5A}
       CORECLR_NEWRELIC_HOME=PATH_TO_INSTALL
       CORECLR_PROFILER_PATH=%CORECLR_NEWRELIC_HOME%\NewRelic.Profiler.dll
       ```
  </Collapser>
</CollapserGroup>
