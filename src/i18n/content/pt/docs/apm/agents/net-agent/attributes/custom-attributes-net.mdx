[atributo](/docs/features/agent-attributes) são pares de valores principais contendo informações que determinam as propriedades de um evento ou transação. Você pode criar um atributo personalizado usando a [API`AddCustomAttribute` ](/docs/agents/net-agent/net-agent-api/itransaction/#addcustomattribute).

Ao adicionar valores de atributo personalizado a transação, evento personalizado, spans e erros, a API aceita um `object`. Descreve como esses valores são processados e como aparecerão no APM. Em todos os casos, os valores `NULL` não são registrados.
