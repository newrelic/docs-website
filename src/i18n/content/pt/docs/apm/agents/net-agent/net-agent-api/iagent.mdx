## Sintaxe

```cs
public interface IAgent
```

Fornece acesso a artefatos e métodos do agente, como a transação em execução no momento.

## Requisitos

Versão do agente 8.9 ou superior.

Compatível com todos os tipos de aplicativos.

## Descrição

Fornece acesso a artefatos e métodos do agente, como a transação em execução no momento. Para obter uma referência a `IAgent`, use [`GetAgent`](/docs/agents/net-agent/net-agent-api/getagent).
