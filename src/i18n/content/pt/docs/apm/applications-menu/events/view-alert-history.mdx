A página <DNT>**Alerts**</DNT> do APM mostra uma tabela de alertas das últimas duas semanas que você pode detalhar em informações detalhadas, incluindo taxas de transferência, porcentagem de taxas de erros e histórico. Você pode visualizar o histórico de alertas de um aplicativo específico ou de todos os aplicativos.

## Alertas para aplicativos específicos [#alert\_procedures][#alert_procedures]

Para visualizar uma lista de alertas para um aplicativo específico: Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; APM &amp; Services &gt; (select an app) &gt; Events &gt; Issues &amp; activity**</DNT>.

A página <DNT>**Issues &amp; Activity**</DNT> mostra uma lista de problemas, eventos de alerta, anomalias e postmortems, se houver. Use qualquer um dos [recursos de UI](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) padrão do New Relic para obter mais detalhes.

Cada notificação de alerta inclui o ícone de alerta e informações resumidas. Para visualizar detalhes sobre uma notificação de alerta na página de alertas, selecione seu link. Por exemplo:

* Para visualizar detalhes sobre um alerta específico, selecione seu nome.
* Para restringir a lista de alertas, use <DNT>**Filter**</DNT>.
* Para visualizar todos os alertas de todos os seus aplicativos e hosts, selecione o link <DNT>**All alerts**</DNT> .
* Para selecionar uma Java máquina virtual (JVM) específica, se aplicável, selecione sua opção no menu <DNT>**JVMs**</DNT> .
