monitor o comportamento do [Elixir](https://github.com/newrelic/elixir_agent) com o New Relic usando o agente Elixir código aberto. Este agente experimental:

* Ajuda a rastrear transações, rastreamento distribuído e outras partes do comportamento do seu aplicativo
* Fornece uma visão geral da atividade [BEAM](http://erlang.org/faq/implementations.html#idp32950544) subjacente

<Callout variant="tip">
  Este agente é lançado como código aberto no [GitHub](https://github.com/newrelic/). Um log de alterações também está disponível para as atualizações mais recentes.
</Callout>

## Iniciar [#get_started]

Para obter informações sobre requisitos, instalação e configuração, consulte o [README do agente Elixir de código aberto](https://github.com/newrelic/elixir_agent/blob/master/README.md) no GitHub.

Visite [o repositório Elixir da New Relic no GitHub](https://github.com/newrelic/elixir_agent) para perguntas sobre instalação, uso ou outros tópicos. Relate problemas ou bugs como um problema no [repositório GitHub](https://github.com/newrelic/elixir_agent/issues).
