## Problema

Quando outro software de monitoramento está sendo executado junto com um de nossos agentes APM, não podemos garantir que nosso agente funcionará corretamente. Em alguns casos, a execução de mais de um tipo de instrumentação, APM ou software de monitoramento pode causar problemas como métricas não confiáveis e travamentos do agente.

Para obter detalhes, consulte os requisitos e a documentação de compatibilidade de um agente APM específico.

## Solução

Se você acha que a execução de outro software de monitoramento pode estar causando um problema, aqui estão algumas recomendações:

* Leia os requisitos e a documentação de compatibilidade do agente APM que você está usando.
* Para solucionar se isso estiver causando um problema, desative ou desinstale temporariamente outro software APM para ver se o problema ainda está presente.
