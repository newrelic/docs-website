A página Operadoras de <InlinePopover type="mobile"/>inclui gráficos que mostram as operadoras sem fio do seu usuário e seu impacto no tempo de resposta, taxa de erros e sessões ativas no período selecionado. Wifi está incluído como operadora. A partir daqui você pode classificar e detalhar informações detalhadas sobre operadoras específicas.

  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Carriers**</DNT>: use esta página para visualizar, classificar ou detalhar informações detalhadas sobre as operadoras móveis do seu usuário por impacto no tempo de resposta, falhas de rede e sessões ativas.
</figcaption>

Para visualizar as operadoras móveis do seu usuário:

1. Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Carriers**
   </DNT>

   .

2. Para alterar as informações exibidas (incluindo tempo de resposta, dispositivos ativos ou erros de rede), selecione sua opção no menu

   <DNT>
     **Sort by**
   </DNT>

   .

3. Para ajustar a quantidade de informações exibidas, selecione

   <DNT>
     **Hide &lt; 1% throughput**
   </DNT>

   .

4. Para selecionar as versões do aplicativo móvel ou o período, use o menu

   <DNT>
     [**Versions**](/docs/mobile-apps/versions)
   </DNT>

   e [o seletor de hora](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) abaixo da barra de menu do New Relic.

5. Para visualizar detalhes de uma operadora específica, selecione seu nome.

Para detalhar informações detalhadas, use qualquer [recurso de interface de plataforma](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) aplicável.
