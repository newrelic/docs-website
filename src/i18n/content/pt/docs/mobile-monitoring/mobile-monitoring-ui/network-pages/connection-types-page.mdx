A página <DoNotTranslate>**Connection Types**</DoNotTranslate> para <InlinePopover type="mobile"/>inclui gráficos que mostram as tecnologias de conexão de rede sem fio do seu usuário e seu impacto no tempo de resposta, taxa de erros e tráfego ativo no período selecionado. Os dados são agrupados por geração de tecnologia: 2G, 3G, etc. Wifi e Desconhecido também estão incluídos na discriminação dos tipos de conexão. A partir daqui você pode detalhar tecnologias sem fio específicas, como GPRS, HSDPA e LTE.

Para visualizar os tipos de conexão móvel sem fio do seu usuário:

1. Vá para <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DoNotTranslate>.

   <img
     title="connection types.png"
     alt="connection types.png"
     src={mobileConnectionTypesSummary}
   />

<figcaption>
  <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DoNotTranslate>: use esta página para visualizar, classificar ou detalhar informações detalhadas sobre o tipo de conexão de rede móvel do seu usuário por impacto no tempo de resposta, falhas de rede e tráfego ativo.
</figcaption>

2. Para alterar as informações exibidas (incluindo tempo de resposta, dispositivos ativos ou erros de rede), selecione sua opção no menu

   <DoNotTranslate>**Sort by**</DoNotTranslate>

   .

3. Para selecionar as versões do aplicativo móvel ou o período, use o menu

   <DoNotTranslate>[**Versions**](/docs/mobile-apps/versions)</DoNotTranslate>

   e [o seletor de hora](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) abaixo da barra de menu do New Relic.

4. Para visualizar detalhes de um tipo de conexão específico, selecione seu nome.

Para detalhar informações detalhadas, use qualquer [recurso de interface de plataforma](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) aplicável.
