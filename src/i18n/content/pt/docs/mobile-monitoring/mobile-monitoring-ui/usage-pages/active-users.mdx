O recurso <InlinePopover type="mobile" />inclui um relatório que rastreia o número de dispositivos, sessões ou usuários executando seu aplicativo para cada dia, semana ou mês, com tendência ao longo do tempo. Para visualizar o relatório: Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Usage &gt; Active users**</DNT>.

## Detalhes dos usuários ativos [#details]

Por padrão, os gráficos e tabelas exibirão dispositivos exclusivos mensais. Use o menu suspenso na barra de filtros para alterar a janela de tempo para semanal ou diária ou para alterar a métrica para sessões ou usuário. Além disso, você pode usar a versão, a versão do sistema operacional, o grupo de dispositivos ou filtros personalizados para segmentação adicional.

Para ver o número total de dispositivos, sessões ou usuários em qualquer período de tempo, passe o mouse sobre a linha no gráfico de linhas. O valor do período de tempo atual é um valor mensal, semanal ou diário e não indica o uso completo do mês, da semana ou do dia.

A tabela de usuários ativos exibe o período de tempo escolhido, conforme especificado pelo filtro selecionado, a métrica exclusiva e uma comparação com o período de tempo anterior. Por exemplo, se `Monthly` for selecionado, a tabela mostrará os últimos 12 meses, os dados exclusivos desses meses e um aumento ou diminuição percentual em relação ao mês anterior.

Para obter informações detalhadas, clique em uma linha da tabela. Isso abrirá uma visualização empilhada com um gráfico de linhas detalhado para aquele período de tempo.

## Rastreamento de dispositivos [#tracking]

O monitoramento de Mobile não utiliza identificador de hardware para rastreamento exclusivo de instalação.

* No iOS usamos a propriedade IdentifierForVendor. (As versões 5.3.4 e inferiores usavam a biblioteca SecureUDID.)
* No Android geramos um GUID exclusivo quando o aplicativo é instalado.
