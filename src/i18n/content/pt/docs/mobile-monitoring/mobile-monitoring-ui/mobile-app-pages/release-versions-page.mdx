Curioso para saber como está o desempenho do lançamento do seu aplicativo mais recente? Use nossa página Versões de lançamento para visualizar taxas de travamento, envolvimento do usuário e métricas de desempenho geral entre diferentes versões de lançamento, que você pode usar para analisar o sucesso de seu lançamento mais recente.

  alt="Screenshot of mobile release versions UI page"
  src="/images/mobile_screenshot-full_release-versions.webp"
/>

## Veja a página Versões de lançamento [#view-release-versions-page]

1. Acesse [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile.

2. Selecione seu aplicativo móvel.

3. No menu à esquerda, clique em

   <DNT>
     **Release versions**
   </DNT>

   .

<Callout variant="tip">
  Não está vendo dados? Experimente o seguinte:

  * Ajuste o filtro de tempo para incluir dados antes do lançamento de uma nova versão.
  * Certifique-se de que seu agente móvel esteja usando o agente iOS versão 7.0.0 e superior ou o agente Android versão 6.5.0 e superior.
</Callout>

## Aprofunde-se em métricas específicas [#drill-down]

A página Versões de lançamento exibe métricas de alto nível em cada versão do aplicativo:

* <DNT>
    **Overall performance score**
  </DNT>

  : Calculado pela taxa geral de crash dos clientes e taxas livres de erros de HTTP.

* <DNT>
    **Crash free user %**
  </DNT>

  : porcentagem de usuários que não apresentam travamentos em uma versão específica.

* <DNT>
    **User adoption**
  </DNT>

  : Porcentagem de usuários que atualizaram ou instalaram uma versão específica.

Clique em <DNT>**View summary**</DNT> ou <DNT>**View details**</DNT> para expandir para uma métrica específica para uma versão do aplicativo. Com visualizações fáceis de entender e gráficos detalhados, a página de resumo da versão permite analisar os principais indicadores de desempenho e tomar decisões informadas para otimizar a experiência do usuário do seu aplicativo.

<img
  title="Details of release versions"
  alt="Screenshot of release versions detailed view"
  src="/images/mobile_screenshot-full_release-versions-details.webp"
/>

Para cada métrica listada acima, a página Versões de lançamento inclui um gráfico dinâmico de série temporal, permitindo acompanhar o desempenho da métrica ao longo do tempo. Ao observar tendências, picos ou quedas no gráfico, você pode identificar rapidamente padrões e correlações que podem informar sua tomada de decisão.

## Criar alerta [#create-alerts]

Agora que você tem visibilidade do desempenho do seu aplicativo, é hora de configurar o alerta para as principais alterações de desempenho.

Basta clicar em uma métrica e, no menu do gráfico, clicar em <DNT>**Create alert condition**</DNT> e seguir as etapas na interface.
