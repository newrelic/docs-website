Quando a [condição de alerta](/docs/mobile-apps/alert-settings) definida para a taxa de erros e o tempo de resposta do seu aplicativo mobile ultrapassar um limite por um período prolongado, o agente <InlinePopover type="mobile"/>criará um evento de problema de Aviso (amarelo) ou Crítico (vermelho). Se um evento Crítico (vermelho) estiver aberto há vários minutos, o mobile enviará uma notificação de alerta ao usuário que foi configurado para receber notificações de alerta. Ele também enviará uma notificação quando todos os eventos críticos forem fechados.
