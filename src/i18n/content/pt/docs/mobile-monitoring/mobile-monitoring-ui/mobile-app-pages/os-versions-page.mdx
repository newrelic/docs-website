Curioso para saber como está o desempenho do seu aplicativo na versão mais recente do Android ou iOS? Use nossa página de versões de sistema operacional para analisar e comparar o desempenho do aplicativo entre diferentes versões (incluindo versões principais e secundárias) e visualizar métricas como número de dispositivos ativos, falhas de rede e tempo de solicitação HTTP.

## Veja a página de versões do sistema operacional [#view-os-versions-page]

1. Acesse [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile.

2. Selecione seu aplicativo móvel.

3. No menu à esquerda, clique em

   <DNT>
     **OS versions**
   </DNT>

   .

## Aprofunde-se em métricas específicas [#drill-down]

Se você observar um pico de dados, como um aumento no tempo de interação, clique e arraste o seletor de hora. A página será atualizada e expandida nesse período para que você possa ver exatamente o que aconteceu.

<img
  title="Expand metric details"
  alt="Gif of clicking and dragging the time picker"
  src="/images/mobile_screenshot-full_Expand-metrics.gif"
/>

Clique em uma versão específica do sistema operacional para visualizar as métricas de desempenho.

  title="Expand metric details for one OS"

## Criar alerta [#create-alerts]

Agora que você tem visibilidade do desempenho do seu aplicativo, é hora de configurar o alerta para as principais alterações de desempenho.

Basta clicar em uma métrica e, no menu do gráfico, clicar em <DNT>**Create alert condition**</DNT> e seguir as etapas na interface.
