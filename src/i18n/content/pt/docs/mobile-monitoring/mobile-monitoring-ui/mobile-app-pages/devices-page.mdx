A página Dispositivos de <InlinePopover type="mobile"/>fornece detalhes de desempenho sobre os principais dispositivos que usam seu aplicativo mobile, como iPad, iPhone, iPod Touch, Android Tablet, etc., além de visualizar métricas como:

* Tempo de interação
* Hora da solicitação HTTP
* Taxa de erros e falhas de rede
* Usuários ativos sessões

## Veja a página Dispositivos [#view-devices-page]

1. Acesse [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile.

2. Selecione seu aplicativo móvel.

3. No menu à esquerda, clique em

   <DNT>
     **Devices**
   </DNT>

   .

## Aprofunde-se em métricas específicas [#drill-down]

Para detalhar informações detalhadas, use qualquer uma das [funções padrão da interface do usuário](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) do New Relic. Além disso:

* Para visualizar uma lista de dispositivos ou modelos específicos (por exemplo, iPad mini, iPad Air, etc.), selecione

  <DNT>
    **type**
  </DNT>

  (por exemplo, iPad).

* Para visualizar detalhes de um dispositivo ou modelo específico, selecione seu nome na lista expandida.

* Para visualizar os detalhes trace de uma transação lenta (se disponível), selecione seu link. Para mais informações, consulte [página de interação](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/interactions-page).

## Criar alerta [#create-alerts]

Agora que você tem visibilidade do desempenho do seu aplicativo, é hora de configurar o alerta para as principais alterações de desempenho.

Basta clicar em uma métrica e, no menu do gráfico, clicar em <DNT>**Create alert condition**</DNT> e seguir as etapas na interface.
