## Problema

Conflitos, falhas ou erros podem aparecer quando o agente iOS da New Relic e outros agentes de rastreamento de dados de telemetria de terceiros, como o Firebase, estão incluídos em um aplicativo.

Erros conhecidos:

* Crash nas funções do instrumento.
* Falha ao capturar o log de falhas.

## Solução

Nosso agente iOS e outros agentes de rastreamento de dados de telemetria de terceiros, como o Firebase, não devem ser ativados ao mesmo tempo. Recomendamos ativar apenas um ou outro para evitar esse problema.

Se não for possível remover o outro SDK, existem algumas etapas que você pode usar para evitar esses conflitos.

* Inicie nosso agente antes de iniciar qualquer outro SDK de terceiros.
* Desative algumas das funcionalidades do nosso agente. Se esta for uma opção que funciona para você, consulte nosso [guia de API](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings/) para obter mais informações.

Se precisar de ajuda adicional, obtenha suporte em [support.newrelic.com](https://support.newrelic.com).
