Recomendamos que você use nossa [instalação guiada](https://onenr.io/0qwLv87gkj5) para configurar o monitoramento do iOS. No entanto, se você precisar instalar o agente manualmente, siga as etapas abaixo para instalar o agente New Relic iOS (em um pacote XCFramework) e adicioná-lo ao seu projeto XCode.

<Steps>
  <Step>
    Baixe o agente iOS mais recente (XCFramework) em nossas [notas de versão do agente iOS](/docs/release-notes/mobile-release-notes/ios-release-notes).
  </Step>

  <Step>
    Navegue até onde o agente iOS foi baixado e descompacte-o.
  </Step>

  <Step>
    No seu projeto Xcode, adicione o XCFramework descompactado:

    1. Clique com o botão direito no navegador do projeto e clique em **Add Files to `[YourProjectName]`**.
    2. Selecione o diretório `NewRelic.xcframework` e clique em **Add**.
  </Step>

  <Step>
    Incorpore o agente iOS:

    1. No navegador de projetos, abra as configurações do projeto e do destino.
    2. Clique no destino ao qual deseja adicionar o agente.
    3. Clique na guia **General** .
    4. Role para baixo até a seção **Frameworks, Libraries, and Embedded Content** .
    5. Certifique-se de que `NewRelic.xcframework` esteja listado aqui. Se não estiver, clique no botão **+** para adicioná-lo.
  </Step>
</Steps>
