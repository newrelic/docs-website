## Atualizar com Swift gerenciador de pacote [#SPM]

Para atualizar o agente iOS do New Relic usando o gerenciador de pacote Swift:

1. Navegue até o agente

   <DNT>
     **NewRelic**
   </DNT>

   na seção Dependência do pacote do browser do projeto Xcode.

2. Clique com o botão direito no agente

   <DNT>
     **NewRelic**
   </DNT>

   .

3. Selecione

   <DNT>
     **Update Package**
   </DNT>

   .

<Callout variant="important">
  Sempre verifique as [notas de lançamento](/docs/releases/ios) para ter certeza de que está obtendo a versão mais atualizada do agente. Dependendo de como as regras do seu gerenciador de pacote Swift estão configuradas, pode ser necessário alterar as configurações `Up to Next Major` ou `Branch` para incluir a nova versão.
</Callout>

## Atualização com cápsula de cacau [#Cocoa-Pods]

Para atualizar o agente iOS do New Relic usando Cocoa pod:

1. Abra o aplicativo

   <DNT>
     **Terminal**
   </DNT>

   e navegue até o diretório com seu PodFile.

2. Execute o seguinte comando:

   ```shell
   pod update NewRelicAgent
   ```

## Substitua manualmente seu agente iOS [#Manual]

Para baixar e instalar manualmente o agente iOS da New Relic:

1. Vá para [https://download.newrelic.com/ios_agent/](https://download.newrelic.com/ios_agent/) e baixe o agente iOS para o qual deseja atualizar.

2. Navegue até onde o agente iOS foi baixado e descompacte-o.

3. Abra o Project Navigator no Xcode (CMD+1) e procure por `NewRelic.xcframework`.

4. Clique com o botão direito ou clique com a tecla Control pressionada em `NewRelic.xcframework` e selecione

   <DNT>
     **Show in Finder**
   </DNT>

   .

5. Arraste o agente iOS recém-baixado para a mesma pasta do agente antigo e selecione para substituir o antigo.
