## Problema

Integrar o agente Android do New Relic e o SDK de monitoramento Sentry pode causar conflitos, travamentos ou erros no seu aplicativo.

Isso é conhecido por causar problemas como `java.lang.StackOverflowError`.

## Solução

Você precisará impedir que o agente Android instrumente o Sentry monitoramento SDK. Para fazer isso, configure o agente New Relic para excluir o pacote Sentry da instrumentação. Veja como:

* Adicione o seguinte bloco ao seu arquivo `build.gradle` :

  ```groovy
  newrelic {
    // Don't instrument sample classes
    excludePackageInstrumentation("io.sentry.*")
  }
  ```

Esta configuração informa ao agente New Relic para ignorar instrumentado quaisquer classes dentro do pacote `io.sentry`.

Se você encontrar algum problema ou precisar de ajuda adicional, entre em contato com o suporte da New Relic em [support.newrelic.com](https://support.newrelic.com).
