Para garantir que você tenha a versão mais atual do agente Android, consulte as [notas de versão do agente Android](/docs/release-notes/mobile-release-notes/android-release-notes). As instruções para atualizar seu aplicativo Android variam de acordo com a versão do SDK do agente Android que você está usando atualmente.

## Atualizar das versões 2 ou 3 do Android SDK [#android_upgrade_2]

Se você já instalou a versão 2 ou 3 do Android SDK: Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Settings > Upgrade.**</DNT>
