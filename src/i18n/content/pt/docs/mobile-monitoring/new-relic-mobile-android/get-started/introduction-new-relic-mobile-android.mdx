[Monitoramento de Mobile](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile) para Android monitora seu aplicativo móvel, proporcionando uma visão abrangente do desempenho do seu aplicativo. Funciona para aplicativos Android escritos em Java ou Kotlin.

## Compatibilidade e requisitos [#requirements]

Antes de instalar o agente Android, revise a [compatibilidade e os requisitos](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements/).

## Instale o agente [#installation]

Para instalar o agente Android, siga nossa [instalação guiada](https://onenr.io/02R5GVpaxQb), localizada diretamente na interface. Usando este método, você pode instalar e configurar o agente Android com Gradle e Android Studio.
