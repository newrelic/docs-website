      ## Sintaxe [#syntax]

      ### Kotlin [#kotlin]

      ```kotlin

      ```

      ## Descrição [#description]

      Esta API fornece a capacidade de desligar o agente dentro do ciclo de vida atual do aplicativo durante o tempo de execução. Depois que o agente for encerrado, ele não poderá ser reiniciado no ciclo de vida atual do aplicativo.

      ## Exemplo [#example]

      ### Java [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```
    </TabsPageItem>
