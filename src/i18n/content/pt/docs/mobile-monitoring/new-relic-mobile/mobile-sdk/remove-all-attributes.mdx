      ## Sintaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAllAttributes ()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.removeAllAttributes()
      ```

      ## Descrição [#description]

      Este método remove todos os atributos de uma sessão. Se desejar remover um único atributo de sessão, consulte [Remover um atributo](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes).

      ## Valores de retorno [#return-values]

      Retorna `true` se registrado com sucesso e `false` se não for.

      ## Exemplo [#example]

      ### Java [#java]

      ```java
      boolean attributesRemoved = NewRelic.removeAllAttributes();
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      val attributesRemoved = NewRelic.removeAllAttributes()
      ```
    </TabsPageItem>
