## Sintaxe [#syntax]

### Java [#java]

```java
   NewRelic.withApplicationToken(apptoken).withLaunchActivityName().start(this.getApplication());
```

### Kotlin [#kotlin]

```kotlin
NewRelic.withLaunchActivityName(className: String?)
```

Defina o nome da atividade de destino para o horário de lançamento do aplicativo para calcular o tempo frio e o tempo quente.

## Descrição [#description]

Por padrão, o agente Android do New Relic calcula o tempo frio e o tempo quente quando a atividade padrão é carregada. Esta API permite que você personalize o nome da atividade de lançamento de destino ao calcular o tempo frio e o tempo quente.

## Exemplos [#examples]

### Definir nome da atividade

Aqui está um exemplo de configuração do nome da atividade:

### Java [#java]

```java
NewRelic.withLaunchActivityName($activityName)
```

### Kotlin [#kotlin]

```kotlin
NewRelic.withApplicationToken("apptoken").withLaunchActivityName($activityName).start(applicationContext)
```
