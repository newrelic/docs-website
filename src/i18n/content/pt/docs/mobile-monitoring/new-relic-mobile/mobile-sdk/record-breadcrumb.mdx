      ## Sintaxe [#syntax]

      ### Java [#java]

      ```java
            NewRelic.recordBreadcrumb(
                        string $Name,
                        map<string, object> $eventAttributes
                    )
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.recordBreadcrumb(
                  breadcrumbName: String?,
                  attributes: MutableMap<String?, Any?>?
              )
      ```

      ## Descrição [#description]

      Esta chamada cria e registra um evento `MobileBreadcrumb` , que pode ser consultado no NRQL e na [trilha de crash do evento](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/mobile-crash-event-trail). Trilhas móveis são úteis para análise de acidentes; eles devem ser criados para atividades de aplicativos que podem ser úteis para resolução de problemas de travamento.

      Além de qualquer atributo personalizado que você escolher, o evento também terá [sessão associada atributo](/docs/insights/explore-data/attributes/mobile-default-attributes-insights#mobile-list). Ao contrário do uso de [`setAttribute`](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/create-attribute/), adicionar atributos a um evento de trilhas os adiciona apenas a esse evento; eles não são atributos de sessão.

      Retornará `true` se o evento for registrado com sucesso ou `false` se não.

      ## Exemplos [#examples]

      Aqui está um exemplo de chamada de evento simples de trilhas:

      ### Java [#java]

      ```java
      Map attributes = new HashMap();
      attributes.put("attributeName1", "value1");
      attributes.put("attributeName1", 2);
      boolean eventRecorded = NewRelic.recordBreadcrumb("Name", attributes);
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      val map = mutableMapOf<String,Any>("attributeName1" to "value1", "attributeName1" to 2)
      val eventRecorded = NewRelic.recordBreadcrumb("Name", map)
      ```

      Aqui está um exemplo de evento de trilhas com valores do mundo real:

      ### Java [#java]

      ```java
      Map attributes = new HashMap();
      attributes.put("button", "sign-in");
      attributes.put("location", "LaunchFragment");

      NewRelic.recordBreadcrumb("user tapped sign in button", attributes);
      ```

      ```kotlin

      NewRelic.recordBreadcrumb("user tapped sign in button", attributes)
      ```

      ## Descrição [#description]

      Usar esta chamada tem o mesmo resultado que usar a [chamada`recordCustomEvent` ](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/record-custom-events/)com `MobileBreadcrumb` como tipo de evento. Para contextualizar como usar esta API, leia sobre envio de atributo personalizado e evento:

      * [Objective-C](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/mobile-sdk-api-guide#objc-custom-att-events)
