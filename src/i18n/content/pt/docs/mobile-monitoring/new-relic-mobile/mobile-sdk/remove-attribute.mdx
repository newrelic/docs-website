      ## Sintaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAttribute(string $name)
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.removeAttribute(name: String?)
      ```

      ## Descrição [#description]

      Este método remove um atributo especificado pela string de nome. Se quiser remover todos os atributos personalizados, consulte [Remover todos os atributos](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes).

      ## Parâmetro [#parameters]

      <table>
        <thead>
          <tr>
            <th width={200}>
              Parâmetro
            </th>

            <th width={200}>
              Tipo
            </th>

            <th>
              Descrição
            </th>
          </tr>
        </thead>

        <tbody>
          <tr>
            <td>
              `$name`
            </td>

            <td>
              `string`
            </td>

            <td>
              Obrigatório. O nome do atributo que você deseja remover.
            </td>
          </tr>
        </tbody>
      </table>

      ## Valores de retorno [#return-values]

      Retorna `true` se registrado com sucesso e `false` se não for.

      ## Exemplo [#example]

      Aqui está um exemplo de remoção de um atributo chamado `rate`:

      ### Java [#java]

      ```java
      boolean attributeRemoved = NewRelic.removeAttribute("rate");
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      val attributeRemoved = NewRelic.removeAttribute("rate")
      ```
