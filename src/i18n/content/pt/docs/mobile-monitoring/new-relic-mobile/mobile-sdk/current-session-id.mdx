      ## Sintaxe [#syntax]

      ### Java [#java]

      ```java
      NewRelic.currentSessionId()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.currentSessionId()
      ```

      ## Descrição [#description]

      Retorna o ID da sessão atual.

      Este método é útil para consolidar o monitoramento de dados de aplicativos (não apenas dados do New Relic) com base em uma única definição e identificador de sessão. Por exemplo, você pode querer usar o mesmo identificador para análise de marketing ou análise de usuário.

      Compatível com todas as versões do agente.

      ## Valores de retorno [#return-values]

      Retorna a string de ID da sessão atual.

      ## Exemplo [#example]

      ### Java [#java]

      ```java
      String sessionId = NewRelic.currentSessionId();
      ```

      ### Kotlin [#kotlin]

      ```kotlin
              val sessionId = NewRelic.currentSessionId()
      ```
    </TabsPageItem>
