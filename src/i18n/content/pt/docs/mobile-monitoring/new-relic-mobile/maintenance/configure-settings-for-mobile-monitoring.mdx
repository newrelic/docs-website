## Veja o token do seu aplicativo [#application-token]

Your mobile app token is a unique, 40-character hexadecimal string that authenticates your app's data to New Relic. Each mobile app you monitor has a unique token for both the US and EU data centers.

Todos os dados do seu aplicativo móvel são relatados no mesmo local na UI de monitoramento de Mobile da New Relic, independentemente do nome do aplicativo quando instalado em um dispositivo móvel. Se você estiver usando um framework híbrido como Flutter ou React Native e implantar na plataforma iOS e Android, você precisará usar token separado, um para iOS e outro para Android.

Para visualizar o token do seu aplicativo:

1. Vá para

   <DoNotTranslate>**[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**</DoNotTranslate>

   , clique em

   <DoNotTranslate>**Mobile**</DoNotTranslate>

   e selecione um aplicativo de monitor.

2. Selecione

   <DoNotTranslate>**Application**</DoNotTranslate>

   ou

   <DoNotTranslate>**Installation**</DoNotTranslate>

   .
