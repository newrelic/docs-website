## Veja o token do seu aplicativo [#application-token]

O token do seu aplicativo móvel é uma string hexadecimal exclusiva de 40 caracteres que autentica os dados do seu aplicativo no New Relic. Cada aplicativo móvel que você monitor possui um token exclusivo para o data center dos EUA e da UE.

Todos os dados do seu aplicativo móvel são relatados no mesmo local na UI de monitoramento de Mobile da New Relic, independentemente do nome do aplicativo quando instalado em um dispositivo móvel. Se você estiver usando um framework híbrido como Flutter ou React Native e implantar na plataforma iOS e Android, você precisará usar token separado, um para iOS e outro para Android.

Para visualizar o token do seu aplicativo:

1. Vá para

   <DNT>
     **[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

   , clique em

   <DNT>
     **Mobile**
   </DNT>

   e selecione um aplicativo de monitor.

2. Selecione

   <DNT>
     **Application**
   </DNT>

   ou

   <DNT>
     **Installation**
   </DNT>

   .

## Configuração no lado do servidor [#configure-server-side-settings]

A configuração do agente móvel do lado do servidor fornece um mecanismo para gerenciar e ajustar remotamente o comportamento do agente móvel implantado nos dispositivos do usuário. Essa funcionalidade permite controlar dinamicamente as configurações de monitoramento sem exigir uma nova implantação na loja de aplicativos.

A seguinte configuração pode ser modificada remotamente na página do **Application** :

* Application Not Responding (ANR) monitoramento de erros (somente Android): Permite rastrear e reportar os motivos dos erros ANR. Esta funcionalidade está habilitada por padrão e requer o agente Android 7.4.1 ou superior.

  <Callout variant="important">
    A configuração no lado do servidor sempre tem precedência sobre a configuração do agente local. Para aplicativos móveis com menos de 100.000 dispositivos ativos simultaneamente, pode levar até 10 minutos para aplicar alterações nos dispositivos finais do usuário. Para aplicativos com mais de 100.000 dispositivos ativos, isso pode levar várias horas.
  </Callout>

### Configurar a partir da interface [#ui-configuration]

Para acessar as configurações no lado do servidor para aplicativos de monitor na interface, vá para <DNT>**[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**</DNT> > <DNT>**Mobile**</DNT> (selecione seu aplicativo) > <DNT>**Settings**</DNT> > <DNT>**Application**</DNT>.
