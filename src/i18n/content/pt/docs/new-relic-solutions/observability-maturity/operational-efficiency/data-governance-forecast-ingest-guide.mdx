Nesta etapa você irá:

* Use seus dados baseline para entender as tendências de crescimento.
* Identifique e priorize os objetivos de observabilidade.
* Coordene as partes interessadas em sua organização para desenvolver o consenso de ingestão de destino.
* Publique previsões e trabalhe com a equipe de contas da NR para direcionar o destino.

## Resultado desejado [#desired-outcome]

Utilize a análise produzida na fase de referência juntamente com os objetivos de valor de observabilidade da fase de otimização para produzir estimativas futuras de ingestão para os próximos meses e até anos.
