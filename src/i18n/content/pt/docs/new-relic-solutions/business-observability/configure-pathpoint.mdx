O New Relic Pathpoint oferece um sistema de configuração flexível que pode ser personalizado para atender às necessidades exclusivas do seu negócio. Pathpoint suporta configuração a partir de uma GUI interativa. Também é possível fazer download e upload de uma configuração JSON. Isso é particularmente útil quando se deseja controlar a versão de uma configuração.

Esse nível de personalização permite que você crie uma solução de observabilidade personalizada que atenda aos requisitos específicos da sua organização.

Nesta página, você aprenderá:

* Como configurar o Pathpoint com a GUI
* Como usar o formato de intercâmbio JSON

## Usando o formato de intercâmbio JSON

É extremamente fácil e intuitivo construir uma implementação do Pathpoint usando os editores de estágios, etapas e pontos de contato mostrados na seção anterior. Às vezes será útil ou necessário armazenar a configuração em um arquivo e importá-la posteriormente, possivelmente em outra conta ou instância do Pathpoint. Para uma implementação de produção do Pathpoint, recomendamos fortemente armazenar um arquivo de sua configuração como JSON em um sistema de controle de versão como o git.

### Baixe sua configuração atual como JSON

O Pathpoint oferece flexibilidade para baixar o arquivo de configuração JSON atual diretamente para o seu computador. Este recurso é particularmente útil se você precisar fazer alterações no arquivo.

Para baixar o arquivo de configuração JSON atualmente ativo:

<Steps>
  <Step>
    Na sua conta New Relic, vá para <DoNotTranslate>**Apps > Pathpoint**</DoNotTranslate>. Clique no<img src={threeLines} class="inline" style={{height: "30px", width: "30px"}} alt="Three lines" title="Three lines"/>

    no canto superior esquerdo.
  </Step>

  <Step>
    Clique em <DoNotTranslate>**JSON configuration**</DoNotTranslate>.
  </Step>

  <Step>
    Em seguida, clique na seta para baixo.

    <img
      title="download active JSON config file"
      alt="download active JSON config file"
      src={downloadFile}
    />

    Seu arquivo de configuração JSON ativo foi baixado agora.
  </Step>
</Steps>

### Substitua sua configuração atual fazendo upload de um arquivo JSON

Para carregar um novo arquivo de configuração JSON:

<Steps>
  <Step>
    Na sua conta New Relic, navegue até <DoNotTranslate>**Apps > Pathpoint**</DoNotTranslate> e clique em<img src={threeLines} class="inline" style={{height: "30px", width: "30px"}} alt="Three lines" title="Three lines"/>

    .
  </Step>

  <Step>
    Na janela resultante, clique em <DoNotTranslate>**JSON configuration**</DoNotTranslate>.
  </Step>

  <Step>
    Isso abre uma janela de caixa de diálogo. Em <DoNotTranslate>**Description**</DoNotTranslate>, descreva seu novo arquivo de configuração JSON. Em seguida, forneça uma observação e clique em <DoNotTranslate>**From file**</DoNotTranslate> para navegar no arquivo de configuração JSON.

    <img
      title="Upload new JSON config file"
      alt="Upload new JSON config file"
      src={uploadJson}
    />
  </Step>

  <Step>
    Localize o arquivo de configuração JSON que deseja usar para atualizar seu Pathpoint e clique em <DoNotTranslate>**Open**</DoNotTranslate>.

    <img
      title="open new JSON config file"
      alt="open new JSON config file"
      src={openFile}
    />

    Finalmente, o Pathpoint é exibido com as atualizações do arquivo JSON específico que você escolheu.
  </Step>
</Steps>
