Observabilidade de Negócios é a arte de capturar, medir e gerenciar a interação ao vivo entre telemetria de TI e análise de negócios.

Freqüentemente, a estratégia de observabilidade é implementada em silos, o que significa que conectar os pontos ao negócio pode ser reativo, uma reflexão tardia e extremamente difícil. Observabilidade de Negócios consiste em alinhar ativamente iniciativas-chave ao desempenho de sistemas, aplicativos e processos.

Ao implementar práticas de Observabilidade de Negócios, a organização pode tomar decisões informadas que impactam todas as suas operações e obter ganhos financeiros significativos.

## Quer começar? [#get-started]

Se ainda não o fez, [cadastre-se para obter uma conta gratuita da New Relic](https://newrelic.com/signup) para iniciar sua jornada de Observabilidade de Negócios hoje mesmo.

## Próximos passos [#next-steps]

Oferecemos diversas soluções para ajudá-lo em sua prática de Observabilidade de Negócios. Aqui estão algumas das principais estratégias usadas por nossos clientes maiores.

### Relatório de dados personalizados [#custom-data]

Relatamos muitos dados úteis por padrão, mas para aproveitar ao máximo o uso do New Relic, você desejará relatar dados personalizados que contenham informações exclusivas e significativas para o seu negócio.

Antes de adicionar qualquer dado personalizado, você deve elaborar um plano de como ele funcionará, para garantir que será uma implementação eficiente e organizada (e não criará muitos dados "ruidosos").

Os métodos de adição de dados personalizados incluem:

* Configurando seu agente e integração existentes
* Usando API do agente
* Usando nossa de métricas, eventos,trace API registros e para trazer quaisquer dados

Para dicas sobre como adicionar dados personalizados, consulte [Dados personalizados](/docs/data-apis/custom-data/get-custom-data-from-any-source).

### Visualizações de interface personalizadas [#visualizations]

Assim como você pode personalizar os dados que traz, você pode personalizar as visualizações desses dados para corresponder à sua estrutura de negócios. Exemplos disso:

* Crie [painéis personalizados](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) que monitoram IA para resolver problemas específicos de negócios
* Crie [carga de trabalho](/docs/new-relic-solutions/new-relic-one/workloads/use-workloads), que são agrupamentos importantes para os negócios da sua entidade monitor

### Ponto de caminho [#pathpoint]
