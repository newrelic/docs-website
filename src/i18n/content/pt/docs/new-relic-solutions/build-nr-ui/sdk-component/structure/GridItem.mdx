Elemento filho do componente `<Grid>` .

Por padrão, ele abrange uma coluna, mas você pode especificar o número de colunas que ele deve abranger definindo os adereços `columnSpan` ou `columnStart` e `columnEnd` .

### Uso

```js
import { GridItem } from 'nr1'
```
