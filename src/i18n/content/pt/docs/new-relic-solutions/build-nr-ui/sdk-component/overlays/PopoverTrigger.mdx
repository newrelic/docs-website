Elemento filho do componente `<Popover>` .

Controla a abertura/fechamento da sobreposição Popover.

### Uso

```js
import { PopoverTrigger } from 'nr1'
```

### Props

        `children` Nó <h5>REQUERIDO</h5> <h5>| função</h5>
      </td>

      <td>
        Este componente pode ser renderizado de duas maneiras: passando diretamente um elemento React filho ou passando um retorno de chamada de renderização (funcionando como filho). Ao passar um elemento React diretamente, você pode usar alguns dos componentes compatíveis do SDK NR1, como `<TextField>`, `<Button>`... ou use seu próprio componente. Ao usar um retorno de chamada de renderização, você receberá um objeto com o seguinte formato como argumento `{ opened: boolean, controlled: boolean }`. Não importa se você está passando componentes diretamente ou usando um retorno de chamada de renderização, ao renderizar seu próprio componente como gatilho, ele deve sempre expor um método `focus` para funcionar corretamente.
