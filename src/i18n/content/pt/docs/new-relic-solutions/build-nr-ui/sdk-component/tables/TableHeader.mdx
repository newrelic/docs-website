Wrapper para células que compõem o cabeçalho. O cabeçalho sempre precisa estar presente e ser o primeiro filho imediato do elemento `<Table>` .

### Uso

```js
import { TableHeader } from 'nr1'
```

### Props

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>Nó</h5> <h5>OBRIGATÓRIO</h5>
      </td>

      <td>
        Lista de cabeçalhos de coluna, no formato de `<TableHeaderCell>`s. As células precisam ser filhas diretas do cabeçalho, ou seja, não deve haver nenhum nó entre `<TableHeader>` e `<TableHeaderCell>`s.
      </td>
    </tr>
