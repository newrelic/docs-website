Agrupa diferentes tipos de gráficos para que possam realizar ações em conjunto, como mover a agulha de acordo com seus eixos x, mostrar a dica de ferramenta da série correspondente ou esmaecer a área que está sendo selecionada em todos os gráficos ao mesmo tempo.

O componente atua como um fragmento React; ele não renderizará uma tag adicional no DOM (apenas permite o agrupamento de todos os gráficos dentro dele). Isso significa que este componente, diferentemente de outros, não aceita tags `className` ou `style` .

### Uso

```js
import { ChartGroup } from 'nr1'
```

### Exemplos

#### Básico

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart style={chartStyle} accountIds={[1]} query={query} />
      <LineChart style={chartStyle} accountIds={[1, 1067061]} query={query} />
      <ScatterChart style={chartStyle} accountIds={[1]} query={query} />
    </ChartGroup>
  );
}
```

#### Preencher contêiner

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
      <LineChart
        style={chartStyle}
        accountIds={[1, 1067061]}
        query={query}
        fullWidth
        fullHeight
      />
      <ScatterChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
    </ChartGroup>
  );
}
```
