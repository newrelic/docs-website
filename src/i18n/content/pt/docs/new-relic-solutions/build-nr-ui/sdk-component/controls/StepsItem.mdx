Elemento filho do componente `<Steps>` .

### Uso

```js
import { StepsItem } from 'nr1'
```

### Props

<table>

      </td>

      <td>

    </tr>

    <tr>

      <td>
        `children` <h5>Nó</h5> <h5>OBRIGATÓRIO</h5>

        `className` <h5>corda</h5>

      </td>
    </tr>

    <tr>
      <td>
        `disabled` <h5>boleano</h5>
      </td>

      <td>
        Use desativado para evitar que o usuário abra a etapa clicando em seu rótulo.
      </td>
    </tr>

    <tr>
      <td>

      </td>

      <td>

    <tr>
