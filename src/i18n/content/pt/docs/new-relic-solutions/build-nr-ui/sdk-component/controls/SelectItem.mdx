Elemento filho do componente `<Select>` .

### Uso

```js
import { SelectItem } from 'nr1'
```

### Props

        Texto a ser mostrado como opção. Observe que apenas texto é suportado como filho.
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>corda</h5>
      </td>

    </tr>

    <tr>
      <td>
        `style` <h5>objeto</h5>
      </td>

      <td>
        Estilo embutido para estilo personalizado. Deve ser usado apenas para fins de posicionamento e espaçamento.
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>corda</h5>
      </td>

      <td>
        Adiciona um atributo `data-test-id` . Use-o para destino do componente em testes unitários e E2E. Para que um ID de teste seja válido, prefixe-o com seu ID nerdpack, seguido por um ponto. Por exemplo, `my-nerdpack.some-element`. **Nota:** Você pode não ver `data-test-id` atributo pois eles são removidos do DOM, para depurá-los passe um parâmetro de consulta `e2e-test` para a URL.
      </td>
    </tr>
