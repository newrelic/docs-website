## `nr1 catalog:info` [#nr1-cataloginfo]

### Obtenha detalhes do catálogo [#nr1-cataloginfo-details]

Mostra as informações sobre seu aplicativo exibidas no catálogo.

Se executado em uma pasta específica do Nerdpack, as informações desse Nerdpack serão mostradas. Se você não quiser obter informações do seu Nerdpack local, use a opção `--nerdpack-id=NERDPACK_ID` para consultar um Nerdpack específico.

### Uso [#nr1-cataloginfo-usage]

`$ nr1 catalog:info`

### Opções [#nr1-cataloginfo-options]

<table>
  <tbody>
    <tr>
      <td>
        `-f, --field=FIELD`
      </td>

      <td>
        Especifica de qual campo você deseja informações.
      </td>
    </tr>

    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        Especifica de qual Nerdpack obter informações.
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        O perfil de autenticação que você deseja usar.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        Adiciona informações extras à saída.
      </td>
    </tr>
  </tbody>
</table>

## `nr1 catalog:submit` [#nr1-catalogsubmit]

### Enviar informações para o catálogo [#nr1-catalogsubmit-send]

Reúne as informações que você adiciona ao diretório de catálogo do seu aplicativo e as salva no catálogo.

Este comando deve ser executado em uma pasta Nerdpack. O comando procurará arquivos específicos usando nomes de convenções.

### Uso [#nr1-catalogsubmit-usage]

`$ nr1 catalog:submit`

### Opções [#nr1-catalogsubmit-options]

<table>
  <tbody>
    <tr>
      <td>
        `-P, --skip-screenshots`
      </td>

      <td>
        Ignora o upload de recursos de captura de tela.
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        O perfil de autenticação que você deseja usar.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        Adiciona informações extras à saída.
      </td>
    </tr>
  </tbody>
</table>
