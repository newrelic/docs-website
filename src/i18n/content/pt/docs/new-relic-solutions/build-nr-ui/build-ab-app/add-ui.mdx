Se um aplicativo for organizado, ele será mais legível e compreensível. Em seu aplicativo de teste A/B, você tem muitos gráficos, mas nenhuma organização. Nas próximas lições, você usará componentes da interface do usuário para trazer alguma organização ao seu aplicativo e fornecer novas funcionalidades. Primeiro, você usará um componente `Grid` para organizar seus gráficos de acordo com o layout do seu guia de design. Segundo, você adicionará títulos para esclarecer o que cada gráfico representa. Terceiro, você adicionará descrições para suas versões de design A e B na parte superior do seu aplicativo. Por fim, você criará uma seção para finalizar seu teste A/B.

Na próxima lição, você organizará seus gráficos da mesma forma que aparecem em seu guia de design.
