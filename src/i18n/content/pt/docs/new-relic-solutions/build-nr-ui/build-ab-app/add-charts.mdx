O aplicativo New Relic que você está construindo ao longo deste curso permite que os desenvolvedores testem A/B seus sites. Para executar um teste A/B bem-sucedido, os proprietários do site precisam ser capazes de analisar os resultados. Sem saber o desempenho de cada design, como os desenvolvedores determinarão qual design usar?

Seu aplicativo exibe dados de duas versões concorrentes. Com diversos gráficos e tabelas, seu usuário poderá analisar os resultados de seus testes e tomar decisões informadas sobre qual design funcionará melhor para atingir seus objetivos.

Você consultará esta maquete muitas vezes ao longo deste curso. Ele mostra quais gráficos criar, como organizá-los e que tipo de dados fornecer a eles. Nas próximas quatro seções, você aprenderá como criar todos os gráficos que seu aplicativo de teste A/B precisa para operar de maneira eficaz. Depois disso, você organizará os gráficos para corresponder ao design da maquete e fornecerá os dados New Relic coletados do aplicativo de demonstração que você criou anteriormente.
