Um dos principais elementos do SDK do New Relic One é a interface de linha de comando (CLI). Para criar um Nerdpack, você precisará instalar o SDK, configurar a CLI para funcionar com sua conta New Relic e, em seguida, utilizar o comando `create`.

# Instalar e configurar a CLI [#install]

<Steps>
  <Step>
    Vá para o início rápido [**Build on New Relic**](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==).
  </Step>

  <Step>
    Obtenha sua [licença chave de API](https://one.newrelic.com/launcher/api-keys-ui.api-keys-launcher).

    Depois de instalar a CLI, você usará essa chave para criar um perfil de usuário associado à sua conta. A CLI usa este perfil para gerenciar entidade dentro de sua conta.
  </Step>

  <Step>
    Ao instalar a CLI, leia e aceite os termos e condições dos desenvolvedores New Relic.

    Mesmo se você instalar a CLI, não poderá usá-la sem primeiro aceitar estes termos e condições.
  </Step>

  <Step>
    Escolha seu sistema operacional e clique em **Download installer**.

    Depois de instalar o SDK, você terá acesso à CLI `nr1`. Verifique isso verificando sua versão do SDK:

    ```sh
    nr1 --version
    ```

    Se você já tinha a CLI, atualize-a:

    ```sh
    nr1 update
    ```

    <Callout variant="tip">
      É importante distinguir entre a CLI `newrelic` e a CLI `nr1`. `newrelic` é para gerenciar entidade em sua conta New Relic. `nr1` é para gerenciar o aplicativo New Relic.
    </Callout>
  </Step>

  <Step>
    Copie o comando para salvar suas credenciais.

    Este comando tem um nome de perfil, sua região e sua chave de API incorporadas.

    Execute o comando em seu terminal:

    ```sh
    nr1 profiles:add --name <profile name> --api-key <User key> --region <region>
    ```

    Os perfis permitem selecionar em qual conta do New Relic você deseja executar comandos. Se você tiver várias contas, poderá visualizá-las com `profiles:list`:

    ```sh
    nr1 profiles:list
    ```

    Observe que um perfil é seu perfil padrão. Esta é a conta na qual seus comandos serão executados, a menos que você especifique outra. Para especificar um perfil para um comando específico, use a opção `--profile` :

    ```sh
    nr1 create --profile <your profile>
    ```

    Se esta for a primeira vez que você usa a CLI, o perfil que você acabou de adicionar será seu perfil padrão. Se você tiver outros perfis, será necessário definir o padrão para aquele que você gostaria de usar neste curso:

    ```sh
    nr1 profiles:default
    ```

    <Callout variant="tip">
      Se você esquecer esses comandos, poderá procurá-los no menu de ajuda dos perfis:

      ```sh
      nr1 profiles --help
      ```
    </Callout>
  </Step>
</Steps>

Agora, você pode sair do início rápido do **Build on New Relic**. Você está pronto para criar um aplicativo com a CLI do New Relic One!
