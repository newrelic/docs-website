Um [agente](/docs/using-new-relic/welcome-new-relic/get-started/glossary#agent) New Relic é um software que você instala em um host ou em um aplicativo que envia dados de desempenho para o New Relic. Usamos agentes diferentes para diferentes produtos e linguagens de codificação.

Neste documento, temos links para algumas de nossas ferramentas mais populares. Para obter uma lista completa de nossas soluções, consulte [New Relic Instant observabilidade](https://newrelic.com/instant-observability/).
