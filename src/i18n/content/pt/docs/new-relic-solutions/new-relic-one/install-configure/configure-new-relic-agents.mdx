Algumas [integrações da New Relic](https://newrelic.com/instant-observability/) exigem a instalação manual de um agente. A forma como o comportamento desses agentes é configurado depende do agente específico.

## Configurar agente APM [#apm-config]

As únicas configurações necessárias para os agentes <InlinePopover type="apm"/>são [a chave de licença](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) e [o nome do aplicativo](/docs/apm/new-relic-apm/installation-configuration/name-your-application), que são definidos quando você [instala o agente](/docs/agents/manage-apm-agents/installation/install-agent).

Para opções de configuração adicionais, consulte:

* [Go](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [Java](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.NET](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## Configurar agente browser [#browser-config]

Consulte nossa documentação para [configuração de<InlinePopover type="browser"/>](/docs/browser/new-relic-browser/configuration).

## Configurar agente de infraestrutura [#infra-config]

A única configuração necessária para nosso agente de infraestrutura é <InlinePopover type="licenseKey"/>. Isso é configurado quando você [instala o agente](/docs/agents/manage-apm-agents/installation/install-agent). Você também pode usar [opções adicionais de configuração](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent) para monitor sua infraestrutura.

## Configurar agente móvel [#mobile-config]

A única configuração necessária para o monitoramento de agente Mobile é o [token da aplicação](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token). Isso é configurado quando você instala o agente de monitoramento de Mobile para seus aplicativos [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android), [iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) .

Para opções de configuração adicionais, consulte:

* [Configuração do Android](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [Configuração iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)
