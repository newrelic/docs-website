Algumas [soluções New Relic](https://newrelic.com/instant-observability/) requerem a instalação de um agente. Você pode desinstalar o agente se quiser que ele pare de reportar dados. Se você precisar desinstalar completamente o New Relic, siga os procedimentos do seu agente.

Se você encontrar problemas, consulte [support.newrelic.com](https://support.newrelic.com).

## Antes de desinstalar [#downgrade]

<Callout variant="important">
  Para a organização pagante da New Relic, uma alternativa à desinstalação é fazer o [downgrade](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) para uma edição inferior.
</Callout>

## Desinstalar o APM [#language-agents]

Após desinstalar o agente <InlinePopover type="apm"/>ou fazer uma alteração em seu script de inicialização, você deverá reiniciar seu aplicativo (ou o servidor de aplicativo ou servidor web onde ele está localizado). Você deve fazer isso porque o agente New Relic normalmente é executado na memória do processo que executa o aplicativo. A simples remoção desses arquivos **não** impedirá que o aplicativo em execução no momento envie dados para o New Relic.

* [Go](/docs/agents/go-agent/installation-configuration/uninstall-go-agent)
* [Java](/docs/agents/java-agent/installation/uninstall-java-agent)
* [.NET](/docs/agents/net-agent/installation-configuration/uninstall-net-agent)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/uninstall-nodejs-agent)
* [PHP](/docs/agents/php-agent/advanced-installation/uninstalling-php-agent)
* [Python](/docs/agents/python-agent/installation-configuration/uninstall-python-agent)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/uninstall-ruby-agent)
