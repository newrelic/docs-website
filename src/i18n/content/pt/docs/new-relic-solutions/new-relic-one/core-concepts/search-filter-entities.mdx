A barra de filtro de entidade permite filtrar as [entidades](/docs/new-relic-solutions/new-relic-one/core-concepts/what-entity-new-relic) do monitor que você deseja examinar e salvar filtros para uso posterior.

## Encontre a barra de filtro [#find]

O filtro de entidade está localizado na parte superior de muitas páginas da interface do New Relic. Ele fica visível na visualização <DNT>**All entities**</DNT> , que é a primeira coisa que você vê ao [carregar o New Relic](https://one.newrelic.com), e fica visível em nossa interface de usuário do APM, em nossa interface de usuário <InlinePopover type="browser"/>, em nossa interface de infraestrutura e muito mais.

## Dicas para usar a barra de filtros [#tips]

Algumas dicas para usar a barra de filtros:

* Digite um

  <DNT>
    **string**
  </DNT>

  de caracteres e/ou números (por exemplo, um ambiente) para localizar qualquer entidade que tenha essa string em seu nome ou ID. Ao digitar, a interface sugere itens que coincidem com a string que você está inserindo, para que você possa selecionar um deles no dropdown. Você também pode filtrar pelo nome de

  <DNT>
    **entity**
  </DNT>

  ,

  <DNT>
    **entity type**
  </DNT>

  ,

  <DNT>
    **account ID**
  </DNT>

  ,

  <DNT>
    **environment**
  </DNT>

  ou

  <DNT>
    [**tag**](/docs/new-relic-one/use-new-relic-one/core-concepts/tagging-use-tags-organize-group-what-you-monitor)
  </DNT>

  . O parâmetro de seleção, uma vez criado, terá fundo azul.

* Se você filtrar usando uma string, o filtro terá o operador `=` .

* Se você filtrar usando o menu dropdown , uma vez inserido o primeiro elemento do item pesquisado (chave de tag ou atributo), você precisará selecionar um operador (`=`, `!=`, `LIKE`, `NOT_LIKE`, `IN`, `NOT IN`) para seu filtro.

* Uma vez selecionado o operador, complete o filtro selecionando o valor para completar o item de pesquisa.

Para adicionar mais filtros, primeiro você deve selecionar um operador, `AND` ou `OR.` Dicas sobre como usá-los:

* Use `AND` para indicar que deseja restringir a seleção removendo entidade da lista de resultados. Você também pode usar o operador AND para adicionar condições que precisam ser atendidas na lista de resultados, por exemplo,

  <DNT>
    **entityType = Services `AND` location = APAC**
  </DNT>

  .

  * Observe que

    <DNT>
      **entityType = Services `AND` entityType = Hosts**
    </DNT>

    não retorna nenhum resultado, pois as entidades só podem ter um tipo e nenhuma entidade atenderia a esta condição.

* Use `OR` para adicionar mais entidade à seleção. Por exemplo, o filtro

  <DNT>
    **entityType = Services `OR` entityType = Hosts**
  </DNT>

  retorna todas as entidades às quais você tem acesso que sejam do tipo `Services` ou `Hosts`.

Se o seu filtro for muito complexo, experimente usar [o NRQL criador de consulta](https://one.newrelic.com/data-exploration/query-builder). Para mais informações sobre consulta NRQL, veja [Como usar NRQL](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-how-nrql-works).
