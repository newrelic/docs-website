Para seu aplicativo de monitor em nossa <InlinePopover type="apm"/>, <InlinePopover type="browser"/>e monitoramento de Mobile UI, a página da IU <DNT>**Dependencies**</DNT> mostra uma lista filtrável de todos os aplicativos, serviços, banco de dados e hosts que a New Relic identifica como uma conexão para essa [entidade](/docs/new-relic-solutions/new-relic-one/core-concepts/what-entity-new-relic). Ele mostra a dependência upstream e downstream e fornece caminhos para explorá-las.

Embora [os mapas de serviço](/docs/understand-dependencies/understand-system-dependencies/service-maps/service-maps-new-relic-one) exibam um mapa dos relacionamentos de uma entidade no escopo da entidade, a página de interface <DNT>**Dependencies**</DNT> exibe seus serviços upstream e downstream em uma tabela. Ele também usa o mesmo [sistema de codificação de cores usado pelos mapas de serviço](/docs/understand-dependencies/understand-system-dependencies/service-maps/service-maps-new-relic-one#alerts) para mostrar o que está funcionando bem e o que não está.

## Requisitos

A página da UI <DNT>**Dependencies**</DNT> está disponível para monitoramento de entidade por nosso agente APM, nosso agente móvel e nosso Browser de agente.

## Ver dependência [#access]

Para visualizar a dependência de aplicativo, serviços, banco de dados e hosts conectados a uma entidade:

1. Vá para

   <DNT>
     **[one.newrelic.com](https://one.newrelic.com)**
   </DNT>

   e selecione

   <DNT>
     **APM & services**
   </DNT>

   ou

   <DNT>
     **Browser**
   </DNT>

   ou

   <DNT>
     **Mobile**
   </DNT>

   .

2. Escolha uma entidade e clique em

   <DNT>
     **Dependencies**
   </DNT>

   .

3. Para detalhar ainda mais, filtre os aplicativos, serviços, banco de dados ou hosts.

Você pode filtrar a página de dependência para visualizar coisas específicas que reportam à entidade. dependência incluem:

* <DNT>
    **Services**
  </DNT>

  : Aplicativo e serviços do monitor [APM](/docs/apm) .

* <DNT>
    **Mobile applications**
  </DNT>

  : seus [aplicativos móveis](/docs/mobile-monitoring).

* <DNT>
    **Browser applications**
  </DNT>

  : seus [aplicativos de Browser](/docs/browser) front-end.

* <DNT>
    **External services**
  </DNT>

  : [monitorização de serviços externos por APM](/docs/apm/applications-menu/monitoring/external-services-page). Os serviços externos incluem serviços fora do processo, como serviços da Web, recursos na nuvem e quaisquer outras chamadas de rede.

* <DNT>
    **Databases**
  </DNT>

  : o banco de dados e os dados de cache do seu aplicativo. banco de dados é sem agente. Por causa disso, o alerta não pode ser definido para o banco de dados, pois apenas a chamada de serviço é reportada ao New Relic.

* <DNT>
    **Hosts**
  </DNT>

  : sua [infraestrutura](/docs/infrastructure) (servidores e hosts).
