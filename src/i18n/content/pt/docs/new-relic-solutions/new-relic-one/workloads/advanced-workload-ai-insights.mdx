<Callout variant="important">
  Os insights de IA estão em **Visualização Pública** com requisitos específicos de disponibilidade.
</Callout>

Os insights de IA fornecem resumos baseados em IA da saúde da sua carga de trabalho, tendências de desempenho e possíveis problemas.

## Requisitos [#requirements]

* Carga de trabalho inteligente criada
* Dados de observabilidade suficientes
* **Apenas infraestrutura AWS** (não disponível no Azure)
* **Não disponível para clientes FedRAMP**

## Gerar insights [#generate]

1. Abra sua carga de trabalho inteligente
2. Clique <DNT>**Generate insights**</DNT>
3. Revise o resumo gerado por IA

Os Insights analisam apenas os dados visíveis na sua página de carga de trabalho e são gerados sob demanda.

## O que os insights incluem [#what-included]

**Visão geral da saúde:**

```
Your workload is healthy with 12 entities reporting normal status.
Response times are within expected ranges.
```

**Tendências de desempenho:**

```
Throughput increased 23% in the past hour while maintaining stable
response times.
```

**Destaques da entidade:**

```
- payment-service: Response time +40ms (within SLA)
- user-db: Connection pool at 78%, approaching threshold
- checkout-api: Error rate below 0.1%, optimal
```

**Recomendações:**

```
Investigate database connection pool trend. While not critical,
usage may require scaling if traffic continues growing.
```

## Práticas medidas [#best-practices]

* Gere insights ao visualizar uma carga de trabalho pela primeira vez ou após alertas
* Verifique as observações de IA em relação às métricas reais
* Use os insights como ponto de partida, não como substituto para análise
* Regenerar após alterações significativas

## Limitações [#limitations]

* Analisa apenas os dados exibidos atualmente
* Não é possível acessar outras cargas de trabalho ou contas
* Fornece observações, não análise de causa raiz
* Pode ocasionalmente interpretar incorretamente padrões complexos

<Callout variant="caution">
  Sempre verifique os insights de IA em relação aos dados reais antes de tomar uma ação.
</Callout>

## Resolução de problemas [#troubleshooting]

### Botão desabilitado

**Motivos:**

* Infraestrutura do Azure (não suportada)
* Conta FedRAMP (recursos de IA desativados)
* Recurso não disponível para sua conta

**Resolução:** Entre em contato com sua equipe de conta para verificar a infraestrutura e o acesso aos recursos.

### Falha na geração

**Motivos:**

* Dados insuficientes
* Problema temporário de conectividade
* Dados de carga de trabalho complexa

**Resolução:** Aguarde e tente novamente. Se o problema persistir, entre em contato com o suporte.
