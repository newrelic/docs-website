A carga de trabalho inteligente rastreia automaticamente todas as entidades no seu fluxo de transação usando rastreamento distribuído. Ao contrário da [carga de trabalho padrão](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads), a carga de trabalho inteligente usa rastreamento distribuído para descobrir entidades e atualiza a associação a cada 5 minutos conforme seu sistema evolui.

## Antes de você começar [#prereqs]

Certifique-se de atender aos [requisitos de cargas de trabalho inteligentes](/docs/new-relic-solutions/new-relic-one/workloads/workloads-overview#intelligent-workloads).

<Callout variant="tip">
  Se você é novo em workloads, confira [Visão geral de workloads](/docs/new-relic-solutions/new-relic-one/workloads/workloads-overview) para entender quando usar workloads Intelligent vs. Standard.
</Callout>

## Crie cargas de trabalho inteligentes [#create]

Você pode criar uma carga de trabalho inteligente de três maneiras diferentes. Escolha o método mais conveniente para o seu fluxo de trabalho:

<CollapserGroup>
  <Collapser id="create-from-recommendations" title="Criar a partir de cargas de trabalho recomendadas">
    O New Relic analisa automaticamente as transações da sua aplicação e recomenda a criação de Intelligent Workloads de alto valor. Esta é a maneira mais rápida de começar a usar o Intelligent Workloads.

    ### Ver recomendações

    Ao navegar para a página Workloads, você verá um banner na parte superior se o New Relic tiver encontrado workloads recomendados para você:

    ```
    We found [X] recommended Intelligent Workloads. We made it easy to review our recommendations and set up Intelligent Workloads quickly.
    ```

    Clique em <DNT>**Review recommendations**</DNT> para abrir o modal de recomendações.

    ### Revisar e criar a partir de recomendações

    O modal **Recommended Intelligent Workloads** exibe transações que se beneficiariam do monitoramento:

    1. **Filtrar por categoria**: Use as abas para alternar entre:

       * **Key Transactions**: Transações de alto tráfego ou críticas para o negócio
       * **Impacted Transactions**: Transações com problemas de desempenho ou erros

    2. **Ordenar recomendações**: Use o menu suspenso para ordenar por:

       * Maior taxa de erro
       * Tempo mais lento médio de resposta
       * Maior consumo de tempo

    3. **Revise os detalhes da transação**: Cada recomendação mostra:

       * **Transaction name**: A transação focal para a carga de trabalho
       * **Focal entity**: O serviço principal que processa a transação
       * **Response Time**: Latência média (ex.: &quot;0,57s&quot;)
       * **Error Rate**: Porcentagem de requisições com falha (por exemplo, &quot;0,6%&quot;)
       * **Total Duration**: Tempo de processamento combinado (ex.: &quot;77.60s&quot;)

    4. **Criar carga de trabalho**: Clique em <DNT>**Create**</DNT> ao lado de qualquer recomendação

    Ao criar a partir de uma recomendação, o New Relic preenche automaticamente a seleção de entidade e transação, levando você diretamente para a etapa **Add details**, onde você pode:

    * Nomeie sua carga de trabalho
    * Adicionar uma descrição (opcional)
    * Verifique o serviço focal e a transação
    * Conclua a configuração

    <Callout variant="tip">
      Criar a partir de recomendações é ideal quando você é novo no Intelligent Workloads ou deseja identificar rapidamente quais transações se beneficiariam mais de um monitoramento dedicado.
    </Callout>
  </Collapser>

  <Collapser id="create-workloads-page" title="Criar a partir da página Cargas de trabalho">
    A página Workload fornece um assistente de configuração guiado para ajudar você a criar uma Carga de trabalho inteligente:

    1. Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Workloads**</DNT>.
    2. Clique em <DNT>**Create a workload**</DNT>.
    3. Selecione o tipo de <DNT>**Intelligent Workload**</DNT>.
    4. Clique em <DNT>**Setup**</DNT> para iniciar o assistente.

    <img title="Select Intelligent Workload type" alt="Screenshot showing workload type selection with Standard and Intelligent options" src="/images/create_intelligent_workload.webp" />

    ### Passo 1: Selecione uma entidade [#select-entity]

    Escolha o serviço ou entidade que contém sua transação crítica:

    1. Pesquise ou navegue pela entidade pelo nome.
    2. Filtre por tipo de entidade, se necessário (serviços de APM, aplicações de navegador, etc.).
    3. Selecione a entidade que processa sua transação-chave.
    4. Clique em <DNT>**Next**</DNT>.

    <Callout variant="tip">
      Escolha uma entidade que seja central para o seu fluxo de transação. Por exemplo, ao monitorar um processo de checkout, selecione o serviço de checkout em vez de um serviço de log periférico.
    </Callout>

    ### Passo 2: Selecione sua transação [#pick-transaction]

    Selecione a transação específica que você deseja monitorar:

    1. Navegue pela lista de transações da entidade selecionada
    2. Use a pesquisa para encontrar nomes específicos de transações
    3. Selecione a transação que representa o fluxo de trabalho do seu negócio
    4. Clique <DNT>**Next**</DNT>

    A carga de trabalho rastreará todas as entidades que participam desta transação com base em rastreamentos distribuídos.

    ### Etapa 3: Adicionar detalhes [#add-details]

    Forneça informações sobre sua carga de trabalho:

    1. **Name**: Insira um nome descritivo (por exemplo, &quot;Transação de Checkout&quot;, &quot;Fluxo de Autenticação de Usuário&quot;)
    2. **Description** (opcional): Adicione contexto sobre o que esta carga de trabalho monitora e por que é importante
    3. **Account**: Selecione a conta onde a carga de trabalho será criada
    4. Clique <DNT>**Next**</DNT>

    ### Etapa 4: Visualizar e finalizar [#preview-finalize]

    Revise a configuração da sua carga de trabalho:

    1. Verifique se o **serviço Focal** está correto
    2. Confirme se a **transação Focal** corresponde ao seu fluxo de trabalho pretendido
    3. Revise o **Name** e a **Description**da carga de trabalho
    4. Clique <DNT>**Create Workload**</DNT>

    Sua carga de trabalho inteligente começará imediatamente a rastrear as entidades participantes da transação.
  </Collapser>

  <Collapser id="create-from-t360" title="Criar a partir do Transaction 360">
    Se você já estiver visualizando uma transação no Transaction 360, poderá salvá-la rapidamente como uma Carga de trabalho inteligente:

    1. Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; APM &amp; Services**</DNT>
    2. Selecione um serviço que faça parte do seu fluxo de transação crítico

    <img title="Select entity for Intelligent Workload" alt="Screenshot showing entity selection step in Intelligent Workload creation" src="/images/intelligent-workload-select-entity.webp" />

    3. Navegue até uma transação específica que deseja monitorar e clique em <DNT>**Transaction 360**</DNT>.
    4. Clique em <DNT>**Save as intelligent workload**</DNT>.
    5. Insira um nome descritivo para a carga de trabalho.
    6. Clique em <DNT>**Save**</DNT>.

    <Callout variant="tip">
      Escolha nomes de transação que representem fluxos de trabalho de negócios completos (por exemplo, &quot;checkout&quot;, &quot;user-authentication&quot;) em vez de chamadas de API individuais.
    </Callout>
  </Collapser>
</CollapserGroup>
