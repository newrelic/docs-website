Depois de [se inscrever em uma conta New Relic](https://newrelic.com/signup), você estará pronto para adicionar seus dados. Você pode fazer isso de duas maneiras: usando nossa instalação guiada ou escolhendo uma abordagem manual. Para a [instalação guiada](https://one.newrelic.com/launcher/nr1-core.explorer?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5saXN0aW5nIn0=&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyIsImFjdGl2ZUNvbXBvbmVudCI6IlZUU09FbnZpcm9ubWVudCIsInBhdGgiOiJndWlkZWQifQ==), siga as instruções passo a passo fornecidas. Se você preferir uma abordagem mais manual, explore as opções abaixo. Não tenha pressa para escolher o método que melhor se adapta às suas necessidades e preferências.

## Registro de instalação [#install-logs]

Oferecemos uma plataforma <InlinePopover type="logs"/>rápida e escalonável para que você possa conectar seu registro ao restante dos dados de telemetria e infraestrutura em um único lugar.

* Habilite o gerenciamento de logs com [log APM em contexto](/docs/logs/logs-context/get-started-logs-context/), nosso [agente de infraestrutura](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent/) ou outras [soluções de roteamento de logs](/docs/logs/forward-logs/enable-log-management-new-relic/).

* Explore [dados log relevantes em sua plataforma](/docs/logs/logs-context/get-started-logs-context/#response-time-example), incluindo erros, rastreamento distribuído, hosts e muito mais.

* Use nossa [interface de log](/docs/logs/ui-data/use-logs-ui/) para encontrar padrões significativos em seu log, atributo, padrões, registro em tempo real e consulta. Além disso, você pode adicionar condição do alerta, consultar seus dados log e criar

  <InlinePopover type="dashboards"/>

  para compartilhar com sua equipe.

## Instale o monitor Sintético [#synthetic-monitors]

[O monitoramento sintético](/docs/synthetics/synthetic-monitoring/getting-started/get-started-synthetic-monitoring/) permite que você monitor proativamente seu site ou endpoints de API para garantir que seu conteúdo esteja totalmente funcional. Envie testes de browser com script para o seu site de locais de todo o mundo para verificar se o seu conteúdo está sempre ativo, em qualquer lugar.

O monitoramento sintético não requer instalação, exceto seu recurso [minion privado conteinerizado](/docs/synthetics/synthetic-monitoring/private-locations/install-containerized-private-minions-cpms/) .
