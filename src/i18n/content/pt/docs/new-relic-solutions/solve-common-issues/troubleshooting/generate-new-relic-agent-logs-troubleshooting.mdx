Algumas [soluções New Relic](https://newrelic.com/instant-observability/) exigem a instalação manual de um agente. A maioria dos agentes inclui ferramentas integradas para gerar registros detalhados para resolução de problemas, auditoria e diagnóstico.

Documentos relacionados:

* Para resolução geral de problemas do agente, consulte [Não vendo dados](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data).
* Saiba mais sobre [o New Relic Diagnostics](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics): um utilitário que detecta automaticamente problemas comuns.
