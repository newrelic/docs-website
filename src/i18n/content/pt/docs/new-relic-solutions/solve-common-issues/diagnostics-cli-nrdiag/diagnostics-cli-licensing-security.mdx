Como qualquer outra ferramenta da New Relic, o serviço Diagnostics CLI foi projetado para proteger a privacidade dos dados para você e seus clientes. A CLI de diagnóstico inspeciona informações do sistema e artefatos relevantes do produto New Relic, como arquivos de log e de configuração. A CLI concentra-se apenas nas informações necessárias para realizar verificações de diagnóstico que avaliam a configuração e a operabilidade do produto New Relic.

Por padrão, esses dados não são transmitidos para o New Relic. Você tem a opção de enviar essas informações para sua conta New Relic por HTTPS. Para obter mais informações sobre as medidas de segurança da New Relic, consulte nossa [documentação de segurança e privacidade](/docs/security/security-privacy/data-privacy/data-privacy-new-relic) ou visite o [site de segurança da New Relic](https://newrelic.com/security).

## Contratos de licença [#diagnostics-licenses]

O uso da CLI do Diagnostics está sujeito a este [contrato de licença](/docs/licenses/license-information/other-licenses/new-relic-diagnostics-licenses), bem como aos contratos de licenciamento para software de código aberto usado pela CLI do Diagnostics.

## Dados enviados para a conta [#uploaded-account-data]

A CLI de diagnóstico permite o upload dessas informações para uma conta New Relic por HTTPS se você usar um argumento de linha de comando específico. Você será avisado antes da coleta de quaisquer arquivos que esperamos conter informações confidenciais.

Antes que os arquivos coletados [contidos em `nrdiag-output.zip` e `nrdiag-output.json`](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output) sejam carregados no New Relic, você também será avisado. Isso permite que você revise e edite qualquer informação que não queira fornecer. (Por exemplo, `nrdiag-output.zip` incluirá seu nome de usuário.) Você também tem a opção de cancelar totalmente o upload.

## Armazenamento de dados

Todos os anexos feitos usando `nrdiag` ou contendo dados `nrdiag` são armazenados pela New Relic.

## Variáveis ambientais

A CLI de Diagnóstico examina as variáveis de ambiente a seguir para executar verificações de diagnóstico. Os valores dessas variáveis são registrados localmente no arquivo `nrdiag-output.json` . Esses incluem:

* Qualquer variável de ambiente contendo `NEWRELIC` ou `NEW_RELIC`
* Qualquer variável de ambiente começando com `NRIA`
