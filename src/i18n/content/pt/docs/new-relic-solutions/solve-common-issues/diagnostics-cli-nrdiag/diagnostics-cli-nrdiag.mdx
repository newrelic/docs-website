A CLI de diagnóstico (`nrdiag`) é um utilitário que detecta automaticamente problemas comuns com produtos New Relic. Se a CLI de Diagnóstico detectar um problema, ela sugerirá etapas de resolução de problemas. A CLI do Diagnostics pode carregar automaticamente dados de resolução de problemas para uma conta New Relic.

A CLI de diagnóstico é de código aberto e está localizada no [GitHub](https://github.com/newrelic/newrelic-diagnostics-cli).

Para etapas adicionais de resolução de problemas para seu agente, consulte [Não vendo dados](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data).

## Compatibilidade

A CLI de diagnóstico está disponível para Linux, macOS e Windows. Ele pode detectar problemas comuns de configuração para:

* <DNT>**APM**</DNT>: Disponível para todos os agentes <InlinePopover type="apm" />. Para o agente Go, apenas verificações básicas de conectividade estão disponíveis.
* <DNT>**Browser monitoring**</DNT>: Detecção para <InlinePopover type="browser" />agente.
* <DNT>**Infrastructure monitoring**</DNT>: Agente Linux e Windows
* <DNT>**Mobile agents**</DNT>: iOS e Android
* <DNT>**Synthetic monitoring**</DNT>: Minion privado em contêiner (chamadas por minuto)
* <DNT>**Kubernetes**</DNT>: Kubernetes, agente Control, Helm e Flux

A CLI de diagnóstico não requer permissões `superuser` ou `admin` para ser executada, embora recomendemos essas permissões para algumas verificações. Ele retornará um erro se não tiver permissão para ler os arquivos que verifica.

## Iniciar

Para usar a CLI de diagnóstico:

1. Execute a CLI de diagnóstico, incluindo [conjuntos de tarefas](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/run-diagnostics-cli-nrdiag) e [opções de linha de comando,](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) conforme necessário.
2. Forneça o sinalizador [`-attach` ou `-api-key`](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) para enviar resultados para sua conta New Relic.
3. Opcional: valide as [configurações do arquivo de configuração](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/validate-config-file-settings-nrdiag).
4. Interprete a [saída](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output).

Como qualquer outra ferramenta da New Relic, o serviço Diagnostics CLI foi projetado para proteger a privacidade dos dados de você e de seus clientes. Para obter informações detalhadas, consulte nossa documentação [de segurança e licenciamento da CLI de diagnóstico](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/diagnostics-cli-licensing-security) .
