Este guia mostra como desinstalar o <DNT>**New Relic ABAP component (NEWR)**</DNT> do seu sistema central SAP. Você pode precisar fazer isso ao desativar o monitoramento ou antes de atualizar para uma nova versão.

<Callout variant="caution">
  A desinstalação do <DNT>**New Relic ABAP component**</DNT> interromperá toda a coleta de dados de monitoramento e removerá o componente do seu sistema SAP. Os dados históricos no New Relic serão preservados, mas nenhum dado novo será coletado. Esta ação não pode ser desfeita a partir do processo de desinstalação.
</Callout>

## Pré-requisitos

* Acesso de administrador a <DNT>**SAP system**</DNT>
* Acesso a <DNT>**client 000**</DNT>
* Backup de quaisquer configurações personalizadas (se necessário para reinstalação futura)

## Desinstale o componente ABAP

Para desinstalar o componente ABAP do sistema central:

1. Faça login em <DNT>**client 000**</DNT>.
2. Insira o código de transação `SAINT`.
3. Clique na guia <DNT>**Uninstallable Components**</DNT> .

<img title="ABAP component on central system" alt="The SAINT transaction showing the NEWR component in the Uninstallable Components list" src="/images/ABAP-component-on-cms.webp" width="700" />

4. Selecione o componente <DNT>**NEWR**</DNT> e verifique se o número da versão corresponde à sua versão instalada.
5. Clique em <DNT>**Start**</DNT>.
6. Clique na marca de seleção verde para confirmar em cada tela subsequente.
7. Clique em <DNT>**Finish**</DNT> quando o processo estiver concluído.
8. Verifique se o componente <DNT>**NEWR**</DNT> não aparece mais na guia <DNT>**Installed Components**</DNT>.

## Verificar a desinstalação

Após concluir o processo de desinstalação:

* O componente <DNT>**NEWR**</DNT> não deve mais aparecer na aba <DNT>**Installed Components**</DNT>
* Os jobs de coleta de dados na transação `SM37` não serão mais executados
* O New Relic deixará de receber dados de monitoramento deste sistema

**Nota**: Os dados históricos de monitoramento no New Relic permanecerão acessíveis. Apenas o componente de coleta de dados é removido do sistema SAP.
