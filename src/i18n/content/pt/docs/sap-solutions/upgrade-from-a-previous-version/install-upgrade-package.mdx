Para atualizar de uma versão anterior do New Relic monitoramento para soluções SAP, certifique-se de usar os pacotes de atualização corretos específicos para sua versão instalada atual.

1. Navegue até **System &gt; Status**.
2. Pesquise por **NEWR** para identificar a versão do produto.
3. No cliente 000, execute a transação **SAINT** para atualizar o complemento ABAP.

## Backup e atualização de configuração.

<Callout variant="tip">
  Salve qualquer configuração de provedor de dados modificada ou provedores de dados personalizados em um modelo de configuração para evitar substituições durante a atualização. Certifique-se de que os modelos sejam salvos antes de copiar o novo modelo do cliente 000, mesmo que o complemento ABAP V3.0 não esteja instalado.
</Callout>

1. Acesse o menu `/n/NEWR/INSTALL` e navegue até **Update or Save Configuration Template**.

2. Execute `/NEWR/REP_DELETE_INS_TEMPLATE` para remover o modelo de monitoramento HDB relacionado ao monitoramento Java.

3. Configure a conexão de dados ABAP com o banco de dados HANA usando `DBACockpit`.

4. Copie o modelo de configuração do cliente 000.

5. Modifique a configuração do sistema de monitoramento:

   * Adicione a nova conexão de banco de dados.
   * Remova referências ao Java Enterprise Connector (por exemplo, NR\_EI\_AGENT).
   * Adicione HANA\_DB aos grupos de monitoramento.
   * Gere novamente a configuração do provedor de dados para atualizar as configurações.
