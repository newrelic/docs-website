* Se o total de SIDs monitorados for menor que 10 e a instância não exceder 50, um servidor dedicado não será necessário. Se os SIDs forem maiores que 10 ou as instâncias forem maiores que 100, um servidor dedicado é recomendado.
* No máximo, um servidor dedicado com duas instâncias de App Server cada uma com 96G de memória e 12 Core CPUs deve ser usado. Se o sistema de monitoramento utilizar um banco de dados HANA, recomendamos separar a instância do servidor banco de dados da instância do servidor aplicativo.
