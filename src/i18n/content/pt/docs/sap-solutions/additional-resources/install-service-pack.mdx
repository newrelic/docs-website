Os service packs são entregues como arquivos `.SAR` com a convenção de nomenclatura `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`.

<Steps>
  <Step>
    Instale o service pack usando a transação `SPAM`.
  </Step>

  <Step>
    Carregue o novo modelo de configuração do cliente 000.
  </Step>

  <Step>
    Navegue até **Install New Release and Patch** &gt; **Copy Template Tables** do cliente 000.
  </Step>

  <Step>
    Gere a nova configuração do provedor de dados seguindo as etapas em **Install New Release and Patch** &gt; **Generate Configuration for New Data Providers**.
  </Step>

  <Step>
    Clique em **Install New Release and Patch** &gt; **Verify Installation Completeness** para verificar a instalação.
  </Step>
</Steps>
