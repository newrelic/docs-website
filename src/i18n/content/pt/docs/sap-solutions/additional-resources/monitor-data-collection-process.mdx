## Verifique o status do trabalho e o spool

1. Use a transação `SM37` para verificar o status dos trabalhos agendados. Digite `/NEWR/*` no campo **ABAP Program Name** e clique em **Execute**.
2. Preste atenção à duração da execução; um tempo anormalmente longo indica um problema em potencial.
3. Verifique se há tarefas sobrepostas e ajuste o cronograma conforme necessário. O spool de trabalho fornece o resultado para cada provedor de dados.

## Verifique o log do aplicativo

Os processos de coleta de dados são registrados no log do aplicativo (transação SLG1). Consulte o objeto de log de aplicación `/NEWR/DP`.

## Dashboard de monitoramento de coleta de dados SAP

Os dados coletados métricos também são enviados ao New Relic para automonitoramento. Use o dashboard [**SAP Data Collection Monitoring**]() pronto para uso para monitorar os processos de coleta de dados para cada provedor de dados.
