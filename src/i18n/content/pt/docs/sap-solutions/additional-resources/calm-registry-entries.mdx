Configure entradas de registro (Call IDs) para provedores de dados do CloudALM para habilitar o monitoramento de serviços do SAP Cloud ALM.

## Visão geral

Este programa utilitário fornece funcionalidade para adicionar, alterar, excluir e gerar automaticamente configurações para provedores de dados CloudALM.

* **Código da transação**: `/NEWR/GEN_DP_CALLID`
* **Nome do programa**: `/NEWR/GENERATE_CONFIG_CALLID`
* **Tabela de configuração**: `/NEWR/CALLID_CNF`

## Pré-requisitos

Certifique-se de que o provedor de dados `NR_CALM_LANDSCAPE` esteja ativo e coletando dados antes de executar este utilitário.

## Procedimento de configuração

<Steps>
  <Step>
    ## Configurar mapeamento de provedor de dados e tipo de serviço

    1. Execute a transação `/NEWR/GEN_DP_CALLID`.
    2. Clique no **DP/CALM Services Type Maintenance Link**.
    3. Selecione os tipos de serviço que você deseja monitorar.
    4. Vincule cada provedor de dados ao sistema SAP ou serviço adequado configurado no SAP Cloud ALM.
    5. Salve a configuração.

    <Callout variant="tip">
      O tipo de serviço representa sistemas ou serviços SAP habilitados no seu ambiente SAP Cloud ALM.
    </Callout>
  </Step>

  <Step>
    ## Adicionar novos registros de ID de chamada

    1. Na tela de seleção, insira o provedor de dados e o tipo de serviço que você precisa configurar.
    2. Selecione o botão de opção **Add new (Call ID) records to Registry**.
    3. Clique em **Execute**.
    4. O sistema exibe todas as instâncias de tipo de serviço configuradas no Cloud ALM, cada uma com um ID do LMS exclusivo.
    5. Selecione as instâncias que você deseja monitorar.
    6. Exclua quaisquer entradas não necessárias.
    7. Clique em **Update and save configuration**.
  </Step>

  <Step>
    ## Verifique a configuração

    1. Vá para a transação `/NEWR/REGISTRY`.
    2. Selecione o provedor de dados.
    3. Clique em **Configure** para visualizar as entradas de Call ID salvas.
    4. Clique em **Test** para verificar a configuração.

    **Resultado esperado**: o teste de conexão exibe status bem-sucedido.
  </Step>
</Steps>

## Opções de manutenção

### Atualizar entradas existentes

Se você precisar alterar ou excluir registros de Call ID existentes:

1. Execute a transação `/NEWR/GEN_DP_CALLID`.
2. Selecione **Copy/Change/Delete existing (Call ID) records in Registry**.
3. Clique em **Execute**.
4. Faça as alterações necessárias.
5. Salve a configuração.

### Limpar todas as entradas

Para remover todos os registros de Call ID (para configuração inicial ou atualizações de pacote de serviço):

1. Execute a transação `/NEWR/GEN_DP_CALLID`.
2. Selecione **Backup &amp; delete all existing (Call ID) records in Registry**.
3. Clique em **Execute**.

<Callout variant="important">
  Isso cria um backup antes da exclusão. Use esta opção apenas durante a configuração inicial ou quando instruído durante uma atualização de service pack.
</Callout>

### Atualizar definições de serviço

Se você não conseguir localizar um tipo de serviço ou ID do LMS:

1. Clique em **Retrieve new services (LMSID) setup in Cloud ALM**.
2. Isso busca os serviços mais recentes configurados na sua conta do Cloud ALM.
