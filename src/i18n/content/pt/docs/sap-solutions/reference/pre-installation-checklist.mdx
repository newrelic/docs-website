Use esta lista de verificação para coletar todas as informações necessárias antes de instalar o <DNT>**New Relic Monitoring for SAP Solutions**</DNT>. Conclua as seções de preparação primeiro, depois preencha as tabelas da lista de verificação de prontidão antes de iniciar a instalação.

## Preparação pré-instalação

### Verifique sua conta New Relic

* Confirme se você tem uma conta New Relic, ou crie uma nova conta. Observe qual região de data center ele usa (EUA ou UE).

### Prepare sua lista de sistemas SAP

Para cada <DNT>**SAP system**</DNT> que você planeja monitorar, forneça as seguintes informações:

* **ID do sistema e clientes**: identifique o sistema e seus clientes associados.

* **Informações do banco de dados**: especifique o fornecedor do banco de dados (por exemplo, <DNT>**HANA**</DNT>, <DNT>**Oracle**</DNT>) e sua versão.

* **Detalhes do componente de software**: inclua a versão do SAP e o nível de patch para os seguintes componentes:

  * <DNT>**SAP\_BASIS**</DNT>
  * <DNT>**Kernel**</DNT>

### Designar um sistema SAP central

Escolha um <DNT>**SAP system**</DNT> para servir como o sistema central para a instalação de <DNT>**New Relic Monitoring for SAP Solutions (NRM4SAP)**</DNT>.

* **Detalhes da instalação**: forneça o <DNT>**System ID**</DNT> e o cliente onde o <DNT>**ABAP component**</DNT> e a configuração serão instalados.
* **Informações do servidor proxy**: se um servidor proxy for usado, especifique seu nome do host e porta. Inclua quaisquer políticas de servidor proxy relevantes para acessar a plataforma <DNT>**New Relic One**</DNT>.

### Acessibilidade de rede

Verifique a conectividade de rede e revise a configuração de <DNT>**API endpoint**</DNT>:

* Certifique-se de que seu firewall não esteja bloqueando o tráfego <DNT>**HTTPS**</DNT> para New Relic
* Documente os detalhes do servidor proxy, se aplicável.
* Revise as configurações de <DNT>**RFC destination**</DNT> para sua região (EUA ou UE)

Para requisitos completos de rede e detalhes dos endpoints de API, consulte [Requisitos de rede e endpoints de API](/docs/sap-solutions/reference/network-accessibility).
