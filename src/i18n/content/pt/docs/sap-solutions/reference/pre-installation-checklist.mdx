Use esta lista de verificação para reunir todas as informações necessárias antes de instalar o New Relic Monitoring for SAP Solutions. Conclua as seções de preparação primeiro, depois preencha as tabelas da lista de verificação de prontidão antes de iniciar a instalação.

## Preparação pré-instalação

### Verifique sua conta New Relic

* Confirme se você tem uma conta New Relic, ou crie uma nova conta. Observe qual região de data center ele usa (EUA ou UE).

### Prepare sua lista de sistemas SAP

Para cada sistema SAP que você planeja monitorar, forneça as seguintes informações:

* **ID do sistema e clientes**: identifique o sistema e seus clientes associados.

* **Informações do banco de dados**: Especifique o fornecedor do banco de dados (por exemplo, HANA, Oracle) e sua versão.

* **Detalhes do componente de software**: inclua a versão do SAP e o nível de patch para os seguintes componentes:

  * **SAP\_BASIS**
  * **Núcleo**

### Designar um sistema SAP central

Escolha um sistema SAP para servir como o sistema central para a instalação do New Relic Monitoring for SAP Solutions (NRM4SAP).

* **Detalhes da instalação**: Forneça o ID do sistema e o cliente onde o componente ABAP e a configuração serão instalados.
* **Informações do servidor proxy**: se um servidor proxy for usado, especifique seu nome de host e porta. Inclua quaisquer políticas de servidor proxy relevantes para acessar a plataforma New Relic One.

### Acessibilidade de rede

Verifique a conectividade de rede e revise a configuração dos endpoints de API:

* Certifique-se de que seu firewall não esteja bloqueando o tráfego HTTPS para New Relic
* Documente os detalhes do servidor proxy, se aplicável.
* Revise as configurações de destino RFC para sua região (US ou EU)

Para requisitos completos de rede e detalhes dos endpoints de API, consulte [Requisitos de rede e endpoints de API](/docs/sap-solutions/reference/network-accessibility).
