## Quem deve usar esta referência

* Administradores SAP Basis configurando conexões RFC
* Administradores de rede configurando regras de firewall
* Equipes de instalação preparando requisitos de rede

## Configuração de destino RFC

Configure esses destinos RFC (Remote Function Call) no seu sistema SAP para permitir a comunicação com as APIs da New Relic. Use os endpoints para a região onde sua conta New Relic está hospedada.

**Nota**: o Connection Type &quot;G&quot; indica uma conexão HTTP para um servidor externo.

Escolha a configuração do endpoint com base na região da sua conta New Relic:

* Use **endpoints da região dos EUA** se sua conta estiver no data center dos EUA
* Use **endpoints da região da UE** se sua conta estiver no data center da UE

## Verifique a conectividade

Após configurar os destinos RFC, teste a conectividade usando o código de transação SAP `SM59`:

1. Selecione o destino RFC
2. Clique em **Connection Test**
3. Verifique a conexão bem-sucedida com os endpoints da New Relic
