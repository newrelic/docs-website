Esta referência lista <DNT>**SAP tables**</DNT> recuperados para fins de monitoramento por <DNT>**New Relic Monitoring for SAP Solutions**</DNT>. Essas tabelas fornecem dados para várias áreas de monitoramento, incluindo <DNT>**AIF**</DNT>, <DNT>**SLT**</DNT>, <DNT>**IDOC**</DNT>, jobs, e-mail, transportes, <DNT>**RFC**</DNT>, processos da empresa e documentos de alteração.

## Autorização e acesso a tabelas

<DNT>**New Relic Monitoring for SAP**</DNT> função de autorização fornece amplo acesso de exibição de tabela para o <DNT>**RFC user**</DNT> no sistema monitorado ao configurar o <DNT>**RFC connections**</DNT>. Para facilitar a personalização adicional ao recuperar <DNT>**SAP tables**</DNT> adicionais, incluindo <DNT>**Z-tables**</DNT> personalizados, bem como ao recuperar <DNT>**IDOC**</DNT> ou dados de carga de mensagem, recomendamos manter o acesso amplo. No entanto, se você deseja restringir o acesso à tabela, use esta lista para fazê-lo.

<Callout variant="important">
  Se algum coletor de dados personalizado recuperar dados diretamente de uma tabela, a função de segurança pode precisar de atualização para que o novo provedor de dados funcione.
</Callout>

## Quem deve usar esta referência

* <DNT>**SAP Basis**</DNT> administradores revisando o escopo da coleta de dados
* Equipes de segurança auditando requisitos de acesso a tabelas
* Administradores de banco de dados monitorando o uso de tabelas
