## Use o menu /NEWR/INSTALL [#menu-newr-install]

1. Efetue login no cliente de produção do sistema de monitoramento central.
2. Navegue até o <DNT>**SAP Easy Access root menu**</DNT> digitando `/n` no campo de comando e pressionando <DNT>**Enter**</DNT>.
3. Acesse o menu <DNT>**`/NEWR/INSTALL`**</DNT> digitando `/n/NEWR/INSTALL` no campo de comando.

Após instalar o add-on no sistema central de monitoramento:

* Certifique-se de que o instalador tenha autorização adequada para executar a configuração.
* Adicione o parâmetro `/NEWR/*` ao(s) objeto(s) de autorização de código no `profile/role` do instalador.

<Callout variant="tip">
  Se estiver atualizando de uma versão anterior, consulte [Atualizar de Versão Anterior](/docs/sap-solutions/upgrade/install-upgrade-package) para determinar quais etapas são necessárias.
</Callout>

## Conectar sistemas [#connect-systems]

<Callout variant="important">
  Conclua quaisquer etapas de configuração pendentes antes de prosseguir.
</Callout>

As seguintes opções estão disponíveis no menu `/NEWR/INSTALL`:

### Configurar destinos HTTP e RFC

Use esta opção para invocar a transação `SM59` e verificar suas conexões <DNT>**RFC**</DNT> e <DNT>**HTTPS**</DNT>. Veja [Criar RFC para sistemas monitorados](/docs/sap-solutions/install-and-configure/additional-configuration/#create-rfc-for-monitored-systems) e [Definir destino HTTP para New Relic cloud](/docs/sap-solutions/install-and-configure/additional-configuration/#set-http-destination-to-new-relic-cloud).

### Configurar Conexão com Banco de Dados Remoto

* Use a transação SAP `DBACOCKPIT` para verificar a configuração das conexões com <DNT>**HANA databases**</DNT> remotos.
* Certifique-se de que uma conexão de banco de dados seja criada para cada banco de dados HANA remoto que você deseja monitorar.
* Para obter instruções detalhadas, consulte [Criar conexão com o banco de dados](/docs/sap-solutions/install-and-configure/additional-configuration/#create-database-connection).

### Configurar Conexões com os Endpoints de API do NR

1. Clique em <DNT>**New Entries**</DNT>.

2. Insira os detalhes da sua conta New Relic:

   * <DNT>**New Relic Account ID**</DNT> (opcionalmente prefixado com SAP-)
   * <DNT>**API Ingest Key**</DNT>
   * <DNT>**Metric Endpoint**</DNT> (consulte [Endpoints de rede](/docs/sap-solutions/before-you-begin/#network-endpoints))
   * <DNT>**Event Endpoint**</DNT>
   * <DNT>**Log Endpoint**</DNT>
   * <DNT>**Trace Endpoint**</DNT>

3. Clique em <DNT>**Save**</DNT>.

### Copy Template Tables from Client 000

Execute esta opção para copiar a tabela de <DNT>**client 000**</DNT> clicando no ícone de execução (símbolo de relógio). Confirme se a mensagem de sucesso aparece.

### Definir os Sistemas Monitorados

1. Clique em <DNT>**New Entries**</DNT> para inserir o <DNT>**SAP System ID**</DNT> e o número do <DNT>**Client**</DNT>.
2. Em <DNT>**Data Provider Installation Information**</DNT>, especifique o <DNT>**Destination**</DNT> RFC, selecione o <DNT>**Software Component**</DNT> SAP, insira o <DNT>**Database Vendor**</DNT> e forneça o <DNT>**DB Connection Name**</DNT> (somente banco de dados HANA).
3. Clique em <DNT>**Monitoring Group**</DNT> e selecione <DNT>**Yes**</DNT> na caixa de diálogo para confirmar.
4. Escolha os grupos de monitoramento necessários e clique no ícone de gerar para concluir o processo.
5. Clique em <DNT>**BACK**</DNT> ou na tecla <DNT>**F3**</DNT> para ir para a próxima etapa.

## Gerar configuração [#generate-configurations]

### Gerar Configurações para Novos Provedores de Dados

* Use esta opção de menu para gerar provedores de dados para uma nova versão ou patch. Para uma nova instalação, todos os Provedores de Dados (DP) são gerados.
* Na primeira tela, confirme os detalhes da versão e do patch. Campos vazios indicam configuração preexistente.
* Para regenerar, selecione <DNT>**Re-generate Data Provider Configurations**</DNT> em <DNT>**Maintain Monitored Systems**</DNT> ou use <DNT>**Reset Installation Flag (For Re-install only)**</DNT> para redefinir o sinalizador de instalação e regenerar a configuração.

<img title="Maintain Monitored Systems" alt="A screenshot displaying the SAP maintain monitored systems" src="/images/maintain-monitored-systems.webp" width="500" />

### Verificar a conclusão da instalação

Clique em <DNT>**Verify Installation Completeness**</DNT> para garantir que tudo esteja configurado corretamente. Se algum item exibir um indicador vermelho, clique no botão de execução na coluna <DNT>**Action Needed**</DNT>.

## Verifique a configuração [#verify-configuration]

1. Para garantir que os provedores de dados estejam configurados corretamente, abra a pasta <DNT>**Data Provider Registry Maintenance**</DNT>.
2. Clique em <DNT>**Data Provider Registry**</DNT> ou use a transação `/n/NEWR/REGISTRY`.
3. Selecione um <DNT>**Data Provider Name**</DNT> e verifique os parâmetros de configuração clicando em <DNT>**Configure**</DNT>.
4. Teste o provedor de dados clicando em <DNT>**TestDP**</DNT>. O teste deve ser concluído com sucesso, confirmando que o provedor de dados está configurado corretamente.
