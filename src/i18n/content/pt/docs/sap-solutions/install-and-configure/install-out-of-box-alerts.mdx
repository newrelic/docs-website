---
title: Instale a política de alertas pronta para uso
metaDescription: Install the out of the box alert policies
freshnessValidatedDate: never
translationType: machine
---

<Steps>
  <Step>
    Acesse **[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**.
  </Step>

  <Step>
    Pesquise e clique em **New Relic Monitoring for SAP Solutions alert policy**.

    <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />
  </Step>

  <Step>
    Na guia **Alert condition**, selecione a conta e alterne a política de alertas para **Enable**. Consulte [Gerenciar suas políticas](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) para saber como gerenciá-las.
  </Step>

  <Step>
    Volte e pesquise **Nr4sap default alert policy**. Ative a condição do alerta que você gostaria de habilitar.
  </Step>
</Steps>
