1. Faça login no <DNT>**BTP subaccount**</DNT>.

2. Em <DNT>**BTP cockpit**</DNT>, clique em <DNT>**Instances and Subscriptions**</DNT>.

3. Clique em <DNT>**Create**</DNT> no canto superior direito.

4. Na tela pop-up, selecione:

   * <DNT>**Service**</DNT>: <DNT>**API Management, API Portal**</DNT>
   * <DNT>**Plan**</DNT>: <DNT>**apiportal-apiaccess**</DNT>
   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>
   * <DNT>**Space**</DNT>: Verifique com seu <DNT>**BTP administrator**</DNT>
   * <DNT>**Instance Name**</DNT>: Siga a convenção de nomenclatura da sua organização

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-management-api.webp" width="600" />

5. Verifique a isenção de responsabilidade de custos.

6. Clique em <DNT>**Next**</DNT> e selecione <DNT>**Roles**</DNT>: `APIPortal.Administrator`.

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-roles-api.webp" width="600" />

7. Clique em <DNT>**Next**</DNT> para revisar os detalhes.

8. Clique em <DNT>**Create**</DNT>.

9. Depois de criado, localize o serviço na aba <DNT>**Instances**</DNT>.

10. Clique nas reticências (`...`) ao lado da sua instância e clique em <DNT>**Create Service Key**</DNT>.

11. Insira um <DNT>**Service Key Name**</DNT> e <DNT>**Credential Type**</DNT>: `binding-secret` em <DNT>**JSON format**</DNT>.

<img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-servicekey-api.webp" width="600" />

12. Clique em <DNT>**Create**</DNT> e faça o download clicando nas reticências (`...`) ao lado do seu <DNT>**service key**</DNT>.
