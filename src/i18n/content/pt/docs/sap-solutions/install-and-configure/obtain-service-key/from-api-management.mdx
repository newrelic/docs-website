1. Faça login na **subconta BTP**.

2. No **BTP cockpit**, clique em **Instances and Subscriptions**.

3. Clique em **Create** no canto superior direito.

4. Na tela pop-up, selecione:

   * **Serviço**: Gerenciamento de API, Portal de API
   * **Plano**: apiportal-apiaccess
   * **Ambiente de execução**: Cloud Foundry
   * **Espaço**: Verifique com seu administrador BTP
   * **Nome da instância**: siga a convenção de nomenclatura da sua organização

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-management-api.webp" width="600" />

5. Verifique a isenção de responsabilidade de custos.

6. Clique em **Next** e selecione **Roles**: `APIPortal.Administrator`.

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-roles-api.webp" width="600" />

7. Clique em **Next** para revisar os detalhes.

8. Clique em **Create**.

9. Depois de criado, localize o serviço na aba **Instances**.

10. Clique nas reticências (`...`) ao lado da sua instância e clique em **Create Service Key**.

11. Insira um Service Key Name e um **Credential Type**: `binding-secret` no formato JSON.

<img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-servicekey-api.webp" width="600" />

12. Clique em **Create** e faça o download clicando nas reticências (`...`) ao lado da sua chave de serviço.
