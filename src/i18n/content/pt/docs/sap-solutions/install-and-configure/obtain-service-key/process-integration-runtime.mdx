Este guia mostra como obter uma chave de serviço da plataforma SAP Business Technology (BTP) para a instância do Process Integration Runtime. Você precisará desta chave de serviço para configurar o monitoramento do SAP Process Integration.

## Pré-requisitos

* Acesso à subconta BTP com as permissões apropriadas
* Confirmação do seu administrador do BTP sobre Space e Grant-types

## Crie a instância do Process Integration Runtime

1. Faça login na **subconta BTP (Business Technology Platform)**.

2. No **BTP cockpit**, clique em **Instances and Subscriptions**.

3. Clique em **Create** no canto superior direito.

4. Na tela pop-up, selecione:

   * **Serviço**: SAP Process Integration Runtime

   * **Plano**: API

   * **Ambiente de execução**: Cloud Foundry

   * **Espaço**: Verifique com seu administrador BTP

   * **Nome da instância**: siga a convenção de nomenclatura da sua organização

     <img title="Process Integration Instance" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields" src="/images/process-integration-instance.webp" width="600" />

5. Clique em **Next** e selecione:

   * **Funções**: `MonitoringDataRead`

   * **Tipos de concessão**: credenciais de cliente padrão (confirme com o administrador do BTP e a equipe de segurança se outros tipos são usados)

     <img title="BTP Roles and Grant Types" alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected" src="/images/btp-roles-grant-types.webp" width="600" />

6. Clique em **Next** para revisar os detalhes.

7. Clique em **Create**.

## Gerar e baixar a chave de serviço

1. Depois de criado, localize o serviço na aba **Instances**.

   <img title="BTP Instance" alt="The Instances tab showing the created Process Integration Runtime instance" src="/images/btp-instance.webp" width="600" />

2. Clique nas reticências (`...`) na linha da sua instância recém-criada e selecione **Create Service Key**.

3. Insira um **Service Key Name** e selecione o **Key Type**: `ClientId/Secret`.

   <img title="BTP New Service Key" alt="The Create Service Key dialog showing Service Key Name and Key Type fields" src="/images/btp-new-service-key.webp" width="600" />

4. Clique em **Create**.

5. Clique nas reticências (`...`) na linha da sua chave de serviço e selecione **Download**.

   <img title="BTP Download Service Key" alt="The service key row showing the ellipsis menu with the Download option" src="/images/btp-download.webp" width="600" />

O arquivo de chave de serviço é baixado como um arquivo JSON. Armazene-o com segurança — você precisará das credenciais deste arquivo para configurar seu ambiente de monitoramento.
