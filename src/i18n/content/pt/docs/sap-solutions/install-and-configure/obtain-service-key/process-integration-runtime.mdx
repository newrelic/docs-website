Este guia mostra como obter uma chave de serviço da plataforma SAP Business Technology (BTP) para a instância do Process Integration Runtime. Você precisará desta chave de serviço para configurar o monitoramento do SAP Process Integration.

## Pré-requisitos

* Acesso a <DNT>**BTP subaccount**</DNT> com as permissões apropriadas
* Confirmação do seu <DNT>**BTP administrator**</DNT> sobre <DNT>**Space**</DNT> e <DNT>**Grant-types**</DNT>

## Crie a instância do Process Integration Runtime

1. Faça login no <DNT>**BTP (Business Technology Platform) subaccount**</DNT>.

2. Em <DNT>**BTP cockpit**</DNT>, clique em <DNT>**Instances and Subscriptions**</DNT>.

3. Clique em <DNT>**Create**</DNT> no canto superior direito.

4. Na tela pop-up, selecione:

   * <DNT>**Service**</DNT>: <DNT>**SAP Process Integration Runtime**</DNT>

   * <DNT>**Plan**</DNT>: <DNT>**API**</DNT>

   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>

   * <DNT>**Space**</DNT>: Verifique com seu <DNT>**BTP administrator**</DNT>

   * <DNT>**Instance Name**</DNT>: Siga a convenção de nomenclatura da sua organização

     <img title="Process Integration Instance" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields" src="/images/process-integration-instance.webp" width="600" />

5. Clique em <DNT>**Next**</DNT> e selecione:

   * <DNT>**Roles**</DNT>: `MonitoringDataRead`

   * <DNT>**Grant-types**</DNT>: <DNT>**Default Client Credentials**</DNT> (confirme com seu <DNT>**BTP administrator**</DNT> e equipe de segurança se outros tipos são usados)

     <img title="BTP Roles and Grant Types" alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected" src="/images/btp-roles-grant-types.webp" width="600" />

6. Clique em <DNT>**Next**</DNT> para revisar os detalhes.

7. Clique em <DNT>**Create**</DNT>.

## Gerar e baixar a chave de serviço

1. Depois de criado, localize o serviço na aba <DNT>**Instances**</DNT>.

   <img title="BTP Instance" alt="The Instances tab showing the created Process Integration Runtime instance" src="/images/btp-instance.webp" width="600" />

2. Clique nas reticências (`...`) na linha da sua instância recém-criada e selecione <DNT>**Create Service Key**</DNT>.

3. Insira um <DNT>**Service Key Name**</DNT> e selecione <DNT>**Key Type**</DNT>: `ClientId/Secret`.

   <img title="BTP New Service Key" alt="The Create Service Key dialog showing Service Key Name and Key Type fields" src="/images/btp-new-service-key.webp" width="600" />

4. Clique em <DNT>**Create**</DNT>.

5. Clique nas reticências (`...`) na linha do seu <DNT>**service key**</DNT> e selecione <DNT>**Download**</DNT>.

   <img title="BTP Download Service Key" alt="The service key row showing the ellipsis menu with the Download option" src="/images/btp-download.webp" width="600" />

O <DNT>**service key file**</DNT> é baixado como um <DNT>**JSON file**</DNT>. Armazene-o com segurança — você precisará das credenciais deste arquivo para configurar seu ambiente de monitoramento.
