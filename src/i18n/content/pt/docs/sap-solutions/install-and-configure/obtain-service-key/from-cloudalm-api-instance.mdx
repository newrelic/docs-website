Este guia mostra como obter uma chave de serviço da instância da API do SAP Cloud Application Lifecycle Management (CALM). Você precisará desta chave de serviço para configurar o monitoramento do SAP Cloud ALM.

## Pré-requisitos

* Acesso a <DNT>**CALM subaccount**</DNT> com as permissões apropriadas
* Confirmação do seu <DNT>**BTP (Business Technology Platform) administrator**</DNT> sobre <DNT>**Space**</DNT>
* Escopos necessários identificados em [Managing Scopes](https://help.sap.com/docs/cloud-alm/apis/managing-scopes)

1. Login no <DNT>**CALM (Cloud Application Lifecycle Management) subaccount**</DNT>, em seguida, clique em <DNT>**Instances and Subscriptions**</DNT>.

2. Clique em <DNT>**Create**</DNT> no canto superior direito.

3. Na tela pop-up, selecione:

   * <DNT>**Service**</DNT>: <DNT>**SAP Cloud ALM API**</DNT>
   * <DNT>**Plan**</DNT>: <DNT>**Standard**</DNT>
   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>
   * <DNT>**Space**</DNT>: Verifique com seu <DNT>**BTP administrator**</DNT>
   * <DNT>**Instance Name**</DNT>: Siga a convenção de nomenclatura da sua organização

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-alm-api.webp" width="600" />

4. Clique em <DNT>**Next**</DNT>.

5. Revise e insira os escopos necessários em [Gerenciando escopos](https://help.sap.com/docs/cloud-alm/apis/managing-scopes) junto com o nome da instância na guia parâmetro.

   <img title="Manage Scopes" alt="The parameters tab showing the required scopes configuration for the CALM instance" src="/images/manage-scope.webp" width="600" />

6. Clique em <DNT>**Next**</DNT> para revisar os detalhes.

   <img title="Review Details" alt="The review screen showing the summary of the SAP Cloud ALM API instance configuration before creation" src="/images/cloud-alm-new-instance.webp" width="600" />

7. Clique em <DNT>**Create**</DNT>.

8. Depois de criado, localize o serviço na aba <DNT>**Instances**</DNT>.

9. Clique nas reticências (`...`) na linha da sua instância recém-criada e selecione <DNT>**Create Service Key**</DNT>.

10. Insira um <DNT>**Service Key Name**</DNT> e selecione <DNT>**Key Type**</DNT>: `ClientId/Secret`.

11. Clique em <DNT>**Create**</DNT>.

12. Clique nas reticências (`...`) na linha do seu <DNT>**service key**</DNT> e selecione <DNT>**Download**</DNT>.

O <DNT>**service key file**</DNT> é baixado como um <DNT>**JSON file**</DNT>. Armazene-o com segurança — você precisará das credenciais deste arquivo para configurar seu ambiente de monitoramento.
