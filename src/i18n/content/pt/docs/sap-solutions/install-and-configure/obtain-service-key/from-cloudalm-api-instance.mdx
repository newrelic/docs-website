Este guia mostra como obter uma chave de serviço da instância da API do SAP Cloud Application Lifecycle Management (CALM). Você precisará desta chave de serviço para configurar o monitoramento do SAP Cloud ALM.

## Pré-requisitos

* Acesso à subconta CALM com as permissões apropriadas
* Confirmação do seu administrador do BTP (Business Technology Platform) referente ao Space
* Escopos necessários identificados em [Managing Scopes](https://help.sap.com/docs/cloud-alm/apis/managing-scopes)

1. Faça login na **subconta CALM (Cloud Application Lifecycle Management)**, clique em **Instances and Subscriptions**.

2. Clique em **Create** no canto superior direito.

3. Na tela pop-up, selecione:

   * **Serviço**: SAP Cloud ALM API
   * **Plano**: Padrão
   * **Ambiente de execução**: Cloud Foundry
   * **Espaço**: Verifique com seu administrador BTP
   * **Nome da instância**: siga a convenção de nomenclatura da sua organização

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-alm-api.webp" width="600" />

4. Clique em **Next**.

5. Revise e insira os escopos necessários em [Gerenciando escopos](https://help.sap.com/docs/cloud-alm/apis/managing-scopes) junto com o nome da instância na guia parâmetro.

   <img title="Manage Scopes" alt="The parameters tab showing the required scopes configuration for the CALM instance" src="/images/manage-scope.webp" width="600" />

6. Clique em **Next** para revisar os detalhes.

   <img title="Review Details" alt="The review screen showing the summary of the SAP Cloud ALM API instance configuration before creation" src="/images/cloud-alm-new-instance.webp" width="600" />

7. Clique em **Create**.

8. Depois de criado, localize o serviço na aba **Instances**.

9. Clique nas reticências (`...`) na linha da sua instância recém-criada e selecione **Create Service Key**.

10. Insira um **Service Key Name** e selecione o **Key Type**: `ClientId/Secret`.

11. Clique em **Create**.

12. Clique nas reticências (`...`) na linha da sua chave de serviço e selecione **Download**.

O arquivo de chave de serviço é baixado como um arquivo JSON. Armazene-o com segurança — você precisará das credenciais deste arquivo para configurar seu ambiente de monitoramento.
