<Steps>
  <Step>
    Navegue até **All entities &gt; Workloads &gt; + Create a workload**. Consulte [Criar e definir carga de trabalho](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/) para saber mais sobre como criar uma workload.
  </Step>

  <Step>
    Digite o nome da workload e selecione a conta necessária na caixa de listagem suspensa.
  </Step>

  <Step>
    Clique no ícone **+** ao lado da opção **Entity Type = All** e insira a condição `sap.system = {SAP System SID}` na barra de filtro.

    <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />
  </Step>

  <Step>
    Clique em **+ Add this query** para adicionar a consulta à definição workload.
  </Step>

  <Step>
    Clique em **Create a workload** para criar a workload.
  </Step>

  <Step>
    A workload ficará visível na interface **All entities** e inclui todas as tags de entidade com o sistema SAP.
  </Step>
</Steps>
