O componente ABAP do <DNT>**NRM4SAP**</DNT> é empacotado como um complemento SAP e pode ser instalado usando a transação <DNT>**SAINT**</DNT> no <DNT>**client 000**</DNT>.

## Arquivo de instalação do complemento

* **Instalação inicial**: baixe o arquivo do add-on `K-xxxAOIINNEWR.SAR` para uma pasta no seu PC local
* **Atualizar de versão anterior**: use o arquivo chamado `K-xxxAOUINNEWR.SAP`

## Instalar arquivo complementar SAP

1. Faça login em <DNT>**client 000**</DNT>
2. Digite `SAINT` no campo de comando <DNT>**SAP GUI**</DNT>
3. Navegue até <DNT>**Installation Package**</DNT> &gt; <DNT>**Load Packages**</DNT>
4. Escolha o método apropriado com base em onde seu arquivo de instalação está localizado:

**Qual método você deve usar?**

* **Arquivo SAR do frontend** (Mais comum): o arquivo de instalação está no seu computador local
* **Arquivo SAR do servidor de aplicativo**: o arquivo SAR foi pré-carregado em `/usr/sap/trans/download/`
* **Arquivos EPS do servidor de aplicativo**: primeiro, você precisa extrair o arquivo SAR em arquivos PAT no servidor

      1. Extraia o arquivo SAR em arquivos PAT. Execute os seguintes comandos:

      ```bash
      SAPCAR -xvf K-xxxAOIINNEWR.SAR
      ```

      2. No pop-up <DNT>**Do you want to upload OCS Packages from the EPS inbox?**</DNT>, selecione <DNT>**Yes**</DNT>
      3. Certifique-se de que o arquivo OCS seja carregado sem erros
    </TabsPageItem>

## Conclua a instalação

1. Clique em <DNT>**Start**</DNT>, selecione <DNT>**Installable Add-on packages for your system**</DNT> e clique <DNT>**Continue**</DNT>

2. Quando solicitado <DNT>**Do you want to add Modification Adjustment Transports to the queue?**</DNT>:

   * Clique em <DNT>**No**</DNT> para instalação padrão (padrão)
   * Clique em <DNT>**Yes**</DNT> apenas se o seu pacote incluir transportes de ajuste com instruções específicas

   <img title="Do you want to add Modification Adjustment Transports to the queue?" alt="A screenshot displaying question about Modification Adjustment Transports" src="/images/modify-adjustement-transports.webp" width="80%" />

3. Clique no botão de check verde para prosseguir

   <Callout variant="important">
     Se a página <DNT>**Errors during Disassembling OCS Packages**</DNT> aparecer, clique em <DNT>**Ignore**</DNT>. Este erro é esperado para certos tipos de pacote e não afetará a instalação.
   </Callout>

4. Depois que o pacote estiver instalado, clique <DNT>**Finish**</DNT>

5. A janela pop-up <DNT>**Sending the runtime analysis**</DNT> é exibida. Escolha um dos seguintes:

   * <DNT>**Send**</DNT>: abre um formulário de feedback com os dados da análise de tempo de execução em uma janela do navegador local
   * <DNT>**Display**</DNT>: exibe primeiro a análise de tempo de execução, depois abre o formulário de feedback
   * <DNT>**Do not send**</DNT>: fecha a caixa de diálogo sem enviar dados de análise de tempo de execução

6. Verifique a instalação na aba <DNT>**Installed Components**</DNT>. O complemento da New Relic deve ser listado com um status indicando instalação bem-sucedida

7. Log out de <DNT>**Client 000**</DNT>
