<Steps>
  <Step>
    Acesse **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**.
  </Step>

  <Step>
    Selecione **apps &amp; visualizations** e procure por **IDOC Explorer**.
  </Step>

  <Step>
    Selecione **Manage access**. Escolha a conta para acessar o aplicativo.

    <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />
  </Step>

  <Step>
    Clique em **Update**. O IDOC Explorer ficará visível ao visualizar a entidade IDOC.
  </Step>
</Steps>
