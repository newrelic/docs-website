O monitoramento da New Relic para SAP inclui uma lista de componentes de monitoramento prontos para uso, incluindo dashboards, políticas de alertas e o IDOC Explorer. Siga as instruções abaixo para localizá-los e instalá-los.

## Antes de você começar

Para executar essas tarefas, você deve ter privilégios de administrador da New Relic na conta.

<Callout variant="important">
  Para o dashboard <DNT>**SAP Cockpit**</DNT>, você deve [criar uma tabela de consulta](#create-lookup-table) chamada `NR_LKUP_SYSTEM` antes da instalação.
</Callout>

## Instalar painel

### Criar tabela de consulta [#create-lookup-table]

<Callout variant="tip">
  Necessário apenas para o dashboard <DNT>**SAP Cockpit**</DNT>.
</Callout>

O dashboard do <DNT>**SAP Cockpit**</DNT> usa um <DNT>**lookup table**</DNT> para <DNT>**SAP systems**</DNT> e seus ambientes.

1. Crie o <DNT>**lookup table**</DNT> com duas colunas: `SYS_ID` e `Env`, e salve-o como um <DNT>**CSV file**</DNT>. Aqui está um exemplo do arquivo de consulta:

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. Vá para <DNT>**Log &gt; Lookup Tables**</DNT>.

3. Clique em <DNT>**Add a table**</DNT>.

4. Na barra lateral <DNT>**Add a lookup table**</DNT>, nomeie a tabela como <DNT>**NR\_LKUP\_SYSTEM**</DNT> e carregue o <DNT>**CSV file**</DNT>. Consulte [Tabelas de consulta](/docs/logs/ui-data/lookup-tables-ui/) para obter mais informações.

5. Para verificar o conteúdo da tabela, use o <DNT>**query builder**</DNT> com `FROM lookup(NR_LKUP_SYSTEM) SELECT *` e clique em <DNT>**Run**</DNT>.

### Instale os dashboards [#install-dashboards]

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> &gt; <DNT>**Integrations &amp; Agents**</DNT> e procure por <DNT>**New Relic Monitoring for SAP Solutions**</DNT> para baixar automaticamente todos os dashboards

2. Alternativamente, você pode ir para <DNT>**Dashboards &gt; Recommended Dashboards**</DNT>, clicar em <DNT>**View all**</DNT> e pesquisar por SAP. Consulte [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) para obter mais informações.

3. Instale estes dashboards:

   * <DNT>**SAP Dashboards**</DNT>
   * <DNT>**SAP Cockpit**</DNT>
   * <DNT>**SAP Data Collection Monitoring**</DNT>
   * <DNT>**SAP Data Ingestion Estimate**</DNT>

### Adicionar filtro dashboard [#add-dashboard-filter]

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards**</DNT>.

2. Habilitar vinculação de facetas para filtragem:

   * Clique nas reticências (<DNT>**…**</DNT>) no canto superior direito do widget
   * Selecione <DNT>**Edit**</DNT> para abrir a tela de edição
   * Alterne a opção <DNT>**Filter the current dashboard**</DNT>

3. <DNT>**Facet linking**</DNT> pode ser habilitado para tabelas, gráficos de pizza e widgets de gráfico de barras para filtrar por <DNT>**SAP system**</DNT>, <DNT>**instance**</DNT>, <DNT>**IDOC status**</DNT>, <DNT>**jobs**</DNT>, etc.

## Instalar políticas de alertas

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**</DNT>.

2. Pesquise e clique em <DNT>**New Relic Monitoring for SAP Solutions alert policy**</DNT>.

   <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />

3. Na guia <DNT>**Alert condition**</DNT>, selecione a conta e alterne o <DNT>**alert policy**</DNT> para <DNT>**Enable**</DNT>. Consulte [Gerenciar suas políticas](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) para saber como gerenciá-las.

4. Volte e pesquise por <DNT>**Nr4sap default alert policy**</DNT>. Ative o <DNT>**alert condition**</DNT> que você gostaria de habilitar.

## Instalar o IDOC Explorer

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT>.

2. Selecione <DNT>**Apps &amp; Visualizations**</DNT> e pesquise <DNT>**IDOC Explorer**</DNT>.

3. Selecione <DNT>**Manage access**</DNT>. Escolha a conta para acessar o aplicativo.

   <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />

4. Clique em <DNT>**Update**</DNT>. O <DNT>**IDOC Explorer**</DNT> ficará visível ao visualizar <DNT>**IDOC entities**</DNT>.
