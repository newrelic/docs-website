O monitoramento da New Relic para SAP inclui uma lista de componentes de monitoramento prontos para uso, incluindo dashboards, políticas de alertas e o IDOC Explorer. Siga as instruções abaixo para localizá-los e instalá-los.

## Antes de você começar

Para executar essas tarefas, você deve ter privilégios de administrador da New Relic na conta.

<Callout variant="important">
  Para o dashboard do **SAP Cockpit**, você deve [criar uma tabela de consulta](#create-lookup-table) chamada `NR_LKUP_SYSTEM` antes da instalação.
</Callout>

## Instalar painel

### Criar tabela de consulta [#create-lookup-table]

<Callout variant="tip">
  Necessário apenas para o dashboard do **SAP Cockpit**.
</Callout>

O dashboard do **SAP Cockpit** usa uma tabela de consulta para sistemas SAP e seus ambientes.

1. Crie a tabela de consulta com duas colunas: `SYS_ID` e `Env` e salve-a como um arquivo CSV. Aqui está um exemplo do arquivo de pesquisa:

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. Acesse **Log &gt; Lookup Tables**.

3. Clique em **Add a table**.

4. Na barra lateral **Add a lookup table**, nomeie a tabela como **NR\_LKUP\_SYSTEM** e carregue o arquivo CSV. Consulte [Tabelas de consulta](/docs/logs/ui-data/lookup-tables-ui/) para obter mais informações.

5. Para verificar o conteúdo da tabela, use o criador de consulta com `FROM lookup(NR_LKUP_SYSTEM) SELECT *` e clique em **Run**.

### Instale os dashboards [#install-dashboards]

1. Acesse **[one.newrelic.com](https://one.newrelic.com)** &gt; **Integrations &amp; Agents** e procure por **New Relic Monitoring for SAP Solutions** para baixar automaticamente todos os dashboards

2. Como alternativa, você pode ir para **Dashboards &gt; Recommended Dashboards**, clicar em **View all** e pesquisar por SAP. Consulte [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) para obter mais informações.

3. Instale estes dashboards:

   * **Dashboards SAP**
   * **SAP Cockpit**
   * **Monitoramento de coleta de dados SAP**
   * **Estimativa de ingestão de dados SAP**

### Adicionar filtro dashboard [#add-dashboard-filter]

1. Acesse **[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards**.

2. Habilitar vinculação de facetas para filtragem:

   * Clique nas reticências (**…**) no canto superior direito do widget
   * Selecione **Edit** para abrir a tela de edição
   * Alterne a opção **Filter the current dashboard**

3. A vinculação de facetas pode ser habilitada para tabelas, gráficos de pizza e widgets de gráfico de barras para filtrar por sistema SAP, instância, status de IDOC, jobs, etc.

## Instalar políticas de alertas

1. Acesse **[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**.

2. Pesquise e clique em **New Relic Monitoring for SAP Solutions alert policy**.

   <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />

3. Na guia **Alert condition**, selecione a conta e alterne a política de alertas para **Enable**. Consulte [Gerenciar suas políticas](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) para saber como gerenciá-las.

4. Volte e pesquise **Nr4sap default alert policy**. Ative a condição do alerta que você gostaria de habilitar.

## Instalar o IDOC Explorer

1. Acesse **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**.

2. Selecione **Apps &amp; Visualizations** e procure por **IDOC Explorer**.

3. Selecione **Manage access**. Escolha a conta para acessar o aplicativo.

   <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />

4. Clique em **Update**. O IDOC Explorer ficará visível ao visualizar a entidade IDOC.
