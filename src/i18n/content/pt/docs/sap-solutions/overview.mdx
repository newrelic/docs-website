O New Relic Monitoring for SAP Solutions captura dados de telemetria SAP (Métricas, Eventos, Logs e Traces) e os envia para a plataforma New Relic para análise e visualização.

## Sistema de monitoramento central vs. sistemas monitorados

Os sistemas SAP que fornecem dados de telemetria são conhecidos como sistemas monitorados. O sistema de monitoramento central é onde <DNT>**NRM4SAP**</DNT> está instalado — ele coleta, transforma e transmite dados de telemetria. Normalmente, você instala <DNT>**NRM4SAP**</DNT> em um único sistema SAP, como <DNT>**SAP Solution Manager (SolMon)**</DNT>. <DNT>**NRM4SAP**</DNT> utiliza fontes de dados padrão do SAP, como <DNT>**ST\_PI**</DNT>, <DNT>**CCMS**</DNT> e o banco de dados de desempenho, removendo a necessidade de instalações adicionais de agentes nos sistemas monitorados.

## Arquitetura de fluxo de dados

<DNT>**RFC**</DNT> chamadas extraem dados de telemetria SAP para o sistema central de monitoramento. O <DNT>**ABAP Enterprise Connector**</DNT> processa os dados e os transmite com segurança para o New Relic via <DNT>**HTTPS**</DNT>.

<img title="Data Flow Architecture" alt="A diagram illustrating the data flow architecture" src="/images/SAP-Private-Cloud.webp" width="80%" />

## O que está incluído

Após a instalação, você pode usar dashboards e alertas prontos para uso:

* **Visibilidade de gerenciamento**: visualize a saúde e o desempenho do sistema SAP junto com outros sistemas de TI em dashboards centralizados.

* **Monitoramento personalizado**: adapte o monitoramento às necessidades específicas do negócio, como rastrear pontos de integração em fluxos de trabalho de processamento de pedidos. Crie dashboards personalizados e visualizações usando dados de telemetria coletados.

* **Alerta proativo**: configure alertas para receber notificações imediatas de interrupções de serviço, permitindo que você mantenha a continuidade e resolva problemas prontamente.
