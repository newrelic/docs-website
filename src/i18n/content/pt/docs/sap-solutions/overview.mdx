O monitoramento New Relic para soluções SAP captura dados de telemetria SAP — métricas, eventos, logs e rastreamento (MELT) — e os envia para a plataforma New Relic One para análise e visualização.

## Sistema Central de Monitoramento vs. Sistemas Monitorados

Os sistemas SAP que fornecem dados de telemetria são conhecidos como Sistemas Monitorados. O Sistema Central é o sistema SAP onde o NRM4SAP está instalado; ele coleta, transforma e transmite dados de telemetria. Normalmente, você instala o NRM4SAP em um único sistema SAP, como o SAP Solution Manager (SolMon). Ele utiliza fontes de dados SAP padrão, como ST\_PI, CCMS e desempenho banco de dados, eliminando a necessidade de instalação de agente adicional em sistemas monitorados.

## Arquitetura de fluxo de dados

Os dados são gerenciados de forma eficiente usando chamadas RFC para extrair dados de telemetria SAP para o Sistema Central. Os dados são então processados pelo ABAP Enterprise Connector e transmitidos com segurança para a plataforma New Relic One TDP via HTTPS.

Essa arquitetura garante um tratamento de dados otimizado e melhora a visibilidade do desempenho do sistema SAP, fornecendo ao engenheiro insights críticos para monitoramento e gerenciamento eficazes.

## Caso de uso

Os sistemas SAP são vitais para grandes empresas, servindo como espinha dorsal de processos essenciais da empresa, incluindo finanças, logística, gestão de materiais, armazenagem, vendas e distribuição. Utilizando o monitoramento New Relic para soluções SAP (NRM4SAP), as empresas podem obter visibilidade abrangente dessas operações orientadas por SAP para garantir que funcionem perfeitamente.

### integração e resolução de problemas

Integrar o NRM4SAP à infraestrutura de uma organização é crucial para monitorar os sistemas SAP que dão suporte a operações comerciais importantes, como processamento de pedidos de comércio eletrônico e gerenciamento de depósito. Esta integração oferece o seguinte:

**Monitoramento abrangente**: o NRM4SAP fornece observabilidade robusta, garantindo que os sistemas SAP operem de forma eficiente e facilitando a detecção e resolução de potenciais problemas antes que eles afetem o processo da empresa.

### Pós-instalação e próximas etapas

Após a instalação, os clientes podem aproveitar imediatamente o painel e os alertas prontos para uso fornecidos pelo NRM4SAP, gerando diversas vantagens:

* **Insights de gerenciamento aprimorados**: o painel centralizado oferece visibilidade sobre a saúde e o desempenho dos sistemas SAP, juntamente com outros sistemas de TI, simplificando a supervisão do gerenciamento e reduzindo a dependência de verificações administrativas.

* **Oportunidades de monitoramento customizado**: os clientes podem adequar o monitoramento às necessidades específicas do negócio, como rastrear pontos de integração dentro do fluxo de trabalho de processamento de pedidos. dashboards personalizados e visualizações podem ser desenvolvidas usando dados de telemetria coletados.

* **Alerta proativo**: ao definir alertas cruciais, o administrador pode ser imediatamente avisado sobre interrupções de serviço, permitindo que ele mantenha a continuidade e resolva prontamente quaisquer problemas.

O NRM4SAP capacita as empresas a otimizar suas operações SAP, aumentando a visibilidade e facilitando o gerenciamento eficaz de seu ecossistema SAP.
