Para atualizar a partir de uma versão anterior do New Relic Monitoring for SAP Solutions, use o pacote de atualização que corresponde à sua versão instalada atual.

## Pré-requisitos

* <DNT>**Administrator access**</DNT> para <DNT>**SAP system**</DNT>
* Versão atual identificada, veja a [seção Identificar versão atual](#identify-current-version)
* Backup das configurações existentes
* <DNT>**upgrade package**</DNT> correto para a sua versão instalada

## Identificar versão atual [#identify-current-version]

1. Navegue até <DNT>**System &gt; Status**</DNT>.
2. Pesquise por <DNT>**NEWR**</DNT> para identificar o <DNT>**product version**</DNT>.
3. Em <DNT>**client 000**</DNT>, execute a transação `SAINT` para atualizar o <DNT>**ABAP add-on**</DNT>.

## Fazer backup e atualizar a configuração [#backup-and-update-configuration]

<Callout variant="tip">
  Salve qualquer <DNT>**data provider configurations**</DNT> ou <DNT>**custom data providers**</DNT> modificado em um <DNT>**configuration template**</DNT> para evitar a sobrescrita durante a atualização. Certifique-se de que os modelos estejam salvos antes de copiar o novo modelo de <DNT>**client 000**</DNT>, mesmo que o <DNT>**V3.0 ABAP Add-on**</DNT> não esteja instalado.
</Callout>

1. Acesse o menu `/n/NEWR/INSTALL` e navegue até <DNT>**Update or Save Configuration Template**</DNT>.

2. Execute `/NEWR/REP_DELETE_INS_TEMPLATE` para remover o <DNT>**HDB monitoring template**</DNT> relacionado a <DNT>**Java monitoring**</DNT>.

3. Configure o <DNT>**ABAP data connection**</DNT> para o <DNT>**HANA database**</DNT> usando `DBACockpit`.

4. Copie o <DNT>**configuration template**</DNT> de <DNT>**client 000**</DNT>.

5. Modificar <DNT>**monitoring system configuration**</DNT>:

   * Adicione o novo <DNT>**DB connection**</DNT>.
   * Remova as referências ao <DNT>**Java Enterprise Connector**</DNT> (por ex., <DNT>**NR\_EI\_AGENT**</DNT>).
   * Adicione <DNT>**HANA\_DB**</DNT> ao <DNT>**monitoring groups**</DNT>.
   * Gere novamente <DNT>**data provider configurations**</DNT> para atualizar as configurações.
