Você pode gerenciar suas políticas, condições e regras de silenciamento programaticamente usando nossa [API NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph) no formato GraphQL. Esta é uma alternativa poderosa para gerenciá-los no New Relic ou com nossa [API REST](/docs/alerts/rest-api-alerts/new-relic-alerts-rest-api/rest-api-calls-new-relic-alerts) mais antiga.

Aqui estão os links para os documentos do NerdGraph relacionados ao alerta:

* [Gerenciar políticas](/docs/alerts/alerts-nerdgraph/nerdgraph-examples/nerdgraph-api-alerts-policies)
* [Usar condições NRQL](/docs/alerts/alerts-nerdgraph/nerdgraph-examples/nerdgraph-api-alerts-nrql-conditions)
* [Adicione regras de silenciamento para suprimir notificações](/docs/alerts/new-relic-alerts/managing-notification-channels/muting-rules-suppress-notifications)

* [Gerenciar fluxo de trabalho](/docs/apis/nerdgraph/examples/nerdgraph-api-workflows)
* [Personalize a perda de sinal e o preenchimento de lacunas](/docs/alerts-applied-intelligence/new-relic-alerts/alerts-nerdgraph/nerdgraph-api-loss-signal-gap-filling)

A maneira mais fácil de descobrir alertas de consulta e mutações é através do NerdGraph API Explorer.
