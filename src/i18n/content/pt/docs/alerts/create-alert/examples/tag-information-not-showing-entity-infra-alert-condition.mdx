## Problema

[A tag personalizada de descrição do incidente](/docs/alerts-applied-intelligence/new-relic-alerts/alert-conditions/alert-custom-incident-descriptions#example) não está sendo substituída pelos valores do evento.

## Solução

Tag estão disponíveis apenas para eventos indexados pela New Relic.

A maneira mais fácil de saber se um evento está indexado é executar a consulta `SELECT nr.entityType FROM ${EventSampleName}`, substituindo `${EventSampleName}` pelo nome do evento em questão.

Se houver valores na coluna `nr.entity Type` do resultado, o evento será indexado. Caso contrário, não é.

## Causa

Não é possível adicionar uma descrição a qualquer amostra que não seja indexada no New Relic.
