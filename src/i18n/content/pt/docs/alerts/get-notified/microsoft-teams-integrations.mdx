Integre o New Relic ao seu Microsoft Teams para receber alertas diretamente nos seus canais do Teams. Essa integração ajuda você a se manter informado sobre a saúde do seu aplicativo e infraestrutura.

**Pré-requisitos:**

* Uma conta ativa do Microsoft Teams.
* Instale o [aplicativo New Relic para Microsoft Teams](https://appsource.microsoft.com/en-us/product/Office365/WA200008155) no seu ambiente do Microsoft Teams. Para obter mais informações, consulte [Adicionar New Relic para Microsoft Teams](#add-new-relic-for-microsoft-teams).

## Adicionar New Relic para Microsoft Teams [#add-new-relic-for-microsoft-team]

Para receber alertas nos seus canais do Teams, instale o aplicativo New Relic no seu ambiente do Microsoft Teams.

Para adicionar o aplicativo New Relic para Microsoft Teams:

1. No Microsoft Teams, clique em **Apps**.
2. Procure o aplicativo **New Relic para Microsoft Teams** e clique em **Add**.
3. Selecione um canal para receber notificações. Uma mensagem de sucesso e um código de segurança serão postados no canal.

## Configurar a integração do Microsoft Teams no New Relic [#configure-microsoft-teams-integration-in-new-relic]

Configure a integração na plataforma New Relic para vinculá-la ao seu Microsoft Teams. Isso envolve configurar um destino para [alertas.](/docs/alerts/overview/)

Para configurar a integração do Microsoft Teams no New Relic:

1. Acesse **[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Alerts &gt; Destinations &gt; Microsoft Teams.**
2. Forneça um nome para a integração e insira o código de segurança da [configuração do aplicativo Teams](#add-new-relic-for-microsoft-teams).
3. Clique em **Save destination**. Uma mensagem de sucesso aparece após a validação do código de segurança.

### Configurar um alerta para o Microsoft Teams [#set-up-an-alert-for-microsoft-teams]

Configure os alertas a serem enviados para o canal com base no alerta do. Para saber mais sobre alertas, consulte [alertas.](/docs/alerts/overview/)

Para adicionar um alerta:

1. Acesse **[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Alerts &gt; Workflows**.

2. Para criar um novo fluxo de trabalho, clique em **Add a workflow** ou selecione um fluxo de trabalho existente para editar.

3. Forneça as informações necessárias.

4. Na seção **Add channel** , clique em **Microsoft Teams**:

   1. Selecione o destino no menu suspenso Destino do **Microsoft Teams** .
   2. Escolha a equipe e o canal nos respectivos menus suspensos. Certifique-se de que o nome do canal corresponde àquele onde o aplicativo New Relic foi adicionado.
   3. Para verificar o recebimento da notificação no seu canal do Teams, clique em **Send test notification**.
   4. Clique em **Save message**.

5. Clique em **Activate Workflow**.
