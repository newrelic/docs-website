Se você for um cliente **Advanced Compute**, poderá usar **NRQL para consultar diretamente os dados de entidade e relacionamento no NRDB,** juntamente com seus dados de telemetria. Esse acesso unificado permite uma resolução de problemas mais rápida e insights mais profunda do sistema.

Este documento abrange:

* Consultando dados de entidades e relacionamentos com NRQL
* Correlação entre entidade e telemetria para uma análise abrangente.
* Exemplos práticos incluem rastrear mudanças de estado e encontrar relações entre o aplicativo e o host.

## Principais benefícios

Você busca simplificar a compreensão dos seus sistemas e solucionar problemas mais rapidamente? Centralizamos seus dados, trazendo as entidades e seus relacionamentos diretamente para o NRDB. Isso significa que você pode usar o poder do NRQL para obter uma visão completa do seu ambiente, tudo em um só lugar.

Eis como você se beneficia:

* **Acesse todos os dados da sua entidade diretamente no NRDB:** você verá que os dados da sua entidade e dos seus relacionamentos são importados e armazenados no NRDB. Isso cria um armazenamento de dados unificado, tornando-o o local central para essas informações críticas, juntamente com seus dados de telemetria.
* **Consulte com facilidade usando NRQL:** Você pode usar o poderoso e flexível NRQL com o qual já está familiarizado para consultar os dados desta entidade e seus relacionamentos. Isso simplifica seu fluxo de trabalho, pois você não precisa aprender novas linguagens de consulta nem realizar junções complexas entre diferentes armazenamentos de dados. Suas consultas se tornam mais simples e fáceis de gerenciar.
* **Obtenha insights oportunas com disponibilidade de dados 24 horas por dia:** Está em dúvida se os dados estarão disponíveis quando você precisar? Esses dados de entidade e relacionamento são retidos no NRDB por um período de 24 horas. Essa disponibilidade permite realizar análises e correlações em tempo hábil, para que você possa recuperar rapidamente as informações necessárias para entender o comportamento do sistema e identificar dependências.
* **Obtenha uma visão holística com correlação simplificada:** você pode vincular facilmente entidades, seus relacionamentos e seus dados de telemetria associados (como métricas, eventos, logs e rastreamento). Essa capacidade de correlação aprimorada significa que você obtém uma visão completa da saúde e do desempenho do seu ambiente, permitindo solucionar problemas com mais eficiência e obter insights mais rapidamente.
