A Rackspace oferece um plug-in para o produto [Cloud Load Balancer](http://www.rackspace.com/cloud/load-balancing/ "O link abre em uma nova janela") . O plug-in permite que você veja o tráfego HTTP vs. HTTPS, configure facilmente <InlinePopover type="alerts"/>para seu limite predefinido e verifique periodicamente a integridade dos nós associados ao seu balanceador de carga para garantir que eles estejam respondendo corretamente.

## Pré-requisitos [#prereqs]

Certifique-se de que seu sistema tenha estes componentes instalados:

* <DNT>Ruby</DNT> (versão 1.8.7 ou superior)
* <DNT>Rubygems</DNT> (versão 1.3.7 ou superior)
* <DNT>Ruby</DNT> gema empacotadora
* Uma [conta New Relic](http://newrelic.com/signup "O link abre em uma nova janela")
* Rackspace Load Balancer com registro ativado

## Instalando o plug-in [#install]

Se você tiver balanceadores de carga em diversas regiões, configure um plug-in separado para cada região. Sempre use caracteres minúsculos para o nome da região em seu arquivo yml. Para obter mais informações sobre a análise YAML, consulte [http://yaml-online-parser.appspot.com/](http://yaml-online-parser.appspot.com/ "O link abre em uma nova janela").

```
wget https://github.com/newrelic-platform/newrelic_rackspace_load_balancers_plugin/archive/1.0.0.zip
```

```
bundle install -binstubs
  ./bin/newrelic_rs --sample-config
```

1. Baixe o arquivo zip do GitHub:
2. Extraia o conteúdo no diretório que deseja usar.
3. Execute os seguintes comandos neste diretório para obter a configuração do gem:
4. Modifique <DNT>**config/newrelic_plugin.yml**</DNT> com sua [chave de licença](/docs/subscriptions/license-key) New Relic, nome de usuário e chave de API da Rackspace e a região onde seu balanceador de carga está localizado. Certifique-se de usar caracteres minúsculos para o nome da região.
5. Inicie o agente: `./bin/newrelic_rs`.
6. Verifique se a saída diz `gathering xx statistics`.

Após a instalação bem-sucedida do plug-in, você poderá selecioná-lo no New Relic:

1. Faça login no New Relic em <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>.
2. Na interface do New Relic, selecione o ícone Rackspace Cloud Load Balancer.

A partir daqui você pode selecionar balanceadores de carga, visualizar o tráfego, configurar alertas e muito mais.
