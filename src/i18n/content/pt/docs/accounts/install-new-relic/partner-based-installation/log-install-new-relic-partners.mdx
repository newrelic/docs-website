Se você se inscreveu no New Relic por meio de um dos parceiros da New Relic, pode ser necessário seguir procedimentos adicionais de login único (SSO) para log e começar a usar o New Relic. Também pode haver etapas especiais necessárias para concluir o processo de implantação. Este documento fornece procedimentos básicos de login para contas criadas por meio de parceiros da New Relic, com links para recursos adicionais, conforme aplicável.

## Crie uma conta New Relic (todos os parceiros) [#partner-account]

Se você recebeu uma oferta especial de um parceiro New Relic, siga o link do parceiro diretamente para criar uma conta New Relic.

## Faça login e implante New Relic (maioria dos parceiros) [#partner-users]

Você receberá uma mensagem de confirmação por e-mail com um link para confirmar sua conta, entrar no New Relic e concluir o processo de implantação.

## Cadastre-se e implante New Relic (parceiros especiais) [#partner-users]

Esses parceiros exigem procedimentos específicos de login e implantação para você usar sua conta New Relic.

<Callout variant="tip">
  Se você se inscreveu para uma conta New Relic por meio de um parceiro que não está listado aqui, [siga os procedimentos padrão](#partner-users) para entrar e implantar o New Relic.
</Callout>

### Parceiros N - Z [#n-z-partners]

    title="Pantheon"
  >
    Para criar uma conta New Relic através [do Pantheon](https://pantheon.io/features/new-relic) e entrar no New Relic:

    1. Na [página de parceria da Pantheon com a New Relic](https://pantheon.io/features/new-relic), selecione o link <DNT>**sign up**</DNT> .

    2. Siga as instruções on-line para concluir o processo de instalação.

    3. Aguarde até que o New Relic comece a coletar dados para seu aplicativo.

       Após alguns minutos, você poderá verificar o desempenho do seu aplicativo na [página <DNT>**Overview**</DNT> do APM](/docs/apm/applications-menu/monitoring/applications-overview-dashboard). Para obter mais informações, consulte [a documentação do Pantheon](https://pantheon.io/docs/new-relic/).

  <Collapser
    id="rackspace"
    title="Ferramentas de nuvem da Rackspace"
  >
    Para criar uma conta por meio do [Rackspace Cloud Tools](https://www.rackspace.com) e entrar no New Relic, siga estes procedimentos básicos. Para obter mais informações, consulte [Plug-in Rackspace Cloud Load Balancer](/docs/accounts-partnerships/accounts/install-new-relic/partner-based-installation/rackspace-cloud-load-balancer-plugin).

    1. Login em sua conta CloudTools em [cloudtools.rackspace.com/myapps](https://cloudtools.rackspace.com/myapps).

    2. Em <DNT>**Use Your Applications**</DNT>, selecione <DNT>**New Relic**</DNT>.

    3. Selecione a linguagem do seu agente (Java, .NET, PHP, Python, Ruby) e siga as instruções on-line para concluir o processo de instalação.

    4. Aguarde até que o New Relic comece a coletar dados para seu aplicativo.

       Aguarde alguns minutos e verifique o desempenho do seu aplicativo na [página <DNT>**Overview**</DNT> do APM](/docs/apm/applications-menu/monitoring/applications-overview-dashboard).
  </Collapser>

  <Collapser
    id="wordpress-w3-edge"
    title="W3 Edge e WordPress"
  >
    <Callout variant="tip">
      Para criar uma conta através do [W3 Total Cache](https://wordpress.org/plugins/w3-total-cache/) e entrar no New Relic, você deve instalar e executar o agente PHP do W3, e não apenas obter uma chave de API.
    </Callout>

    Implantar o agente requer <DNT>**administrative access**</DNT>, incluindo:

    * Acesso SSH/linha de comando

    * Acesso root ou sudo para executar o instalador do New Relic e reiniciar seu servidor web

      Se você estiver administrando um pacote econômico, de hospedagem compartilhada ou de blogger, ou se não tiver certeza de quais são as permissões de sua conta, entre em contato com seu provedor de hospedagem. Por exemplo, se o seu site WordPress estiver em servidores compartilhados, talvez você não consiga instalar o New Relic. Seu provedor deverá ser capaz de fornecer informações sobre as permissões de sua conta ou atualizá-la, se necessário.

      Para instalar o plug-in New Relic para usuário W3 Edge/Wordpress:

    1. Login em sua conta WordPress em [wordpress.org/support/bb-login.php](https://wordpress.org/support/bb-login.php).

    2. Siga [as instruções da New Relic](/docs/agents/php-agent/installation/wordpress-users-new-relic) para concluir o processo de instalação ou entre em contato com [o suporte do W3TC](http://wordpress.org/support/plugin/w3-total-cache) para obter assistência.

    3. Aguarde até que o New Relic comece a coletar dados.

       Alguns minutos após a implantação do agente, os dados de desempenho do seu aplicativo aparecerão na página [Visão geral de aplicativos](/docs/apm/applications-menu/monitoring/applications-overview-dashboard)do APM.

       <Callout variant="tip">
         [O WordPress](https://wordpress.org/plugins/rt-newrelic-browser/) também oferece um plug-in para nosso [monitoramento de Browser](/docs/agents/php-agent/installation/wordpress-users-new-relic#browser).
       </Callout>
  </Collapser>
