Para obter mais informações sobre como usar o agente .NET New Relic, consulte:

* [Instalando o agente .NET no Azure](/docs/dotnet/installing-the-net-agent-on-azure)

* [Sites do Azure](/docs/dotnet/azure-web-sites)
* [Azure serviços na nuvem](/docs/dotnet/azure-cloud-services)

Para obter mais informações sobre como instalar o agente Node.js do New Relic, consulte [Agente Node.js no Microsoft Azure](/docs/nodejs/nodejs-agent-on-microsoft-azure).
