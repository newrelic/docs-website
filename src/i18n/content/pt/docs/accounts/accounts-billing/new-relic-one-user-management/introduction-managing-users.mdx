Os documentos nesta seção são sobre como gerenciar usuários em [nosso modelo de usuário mais recente](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models), no qual quase todos os usuários do New Relic estão. Para documentos sobre nosso modelo de usuário original, consulte [Gerenciamento de usuário original](/docs/accounts/original-accounts-billing/original-users-roles).

Aqui estão alguns de nossos documentos de gerenciamento de usuários mais importantes:

* [Como funciona o acesso do usuário](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts): saiba como os usuários obtêm acesso às permissões e às contas.
* [tipo de usuário](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type): aprenda sobre usuário básico, usuário principal e usuário de plataforma completa.
* [interface e tarefas comuns](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/): aprenda sobre a interface de gerenciamento de usuários, como adicionar usuários, editar tipo de usuário, adicionar usuários a grupos, criar funções personalizadas e outras tarefas comuns.
* [Introdução ao logon único e ao provisionamento SCIM](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim): uma visão geral do SAML SSO e como gerenciar seu usuário a partir de um provedor de identidade.
* [Configurações de domínio de autenticação](/docs/accounts/accounts-billing/new-relic-one-pricing-users/configure-authentication-domains): um "domínio de autenticação" é o que governa como os usuários nesse domínio são adicionados ao New Relic (manualmente versus provisionamento SCIM), o método de autenticação usado por esses usuários (login manual versus SAML SSO), gerenciando como o usuário atualiza seu usuário tipo e configurações de sessão do usuário.
* [tutorial de gerenciamento de usuários](/docs/accounts/accounts-billing/new-relic-one-user-management/tutorial-add-new-user-groups-roles-new-relic-one-user-model): lições aprofundadas sobre algumas tarefas comuns de gerenciamento de usuários.
* [Vídeos sobre gerenciamento de usuários](/docs/accounts/accounts-billing/new-relic-one-user-management/user-mgmt-videos): assista a vídeos da interface de gerenciamento de usuários em ação.

Quer entender como a contagem de usuários afeta o faturamento? Consulte [Faturamento relacionado ao usuário](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-one-pricing-billing#user-count).
