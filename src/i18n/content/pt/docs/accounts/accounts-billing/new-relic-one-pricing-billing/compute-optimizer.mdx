A ferramenta otimizadora de computação fornece insights práticos para ajudar você a gerenciar gastos com computação e implementar controles automatizados de computação. Isso significa que você pode identificar áreas ineficientes, receber recomendações claras para otimização e, finalmente, obter economia potencial de computador, maior eficiência de consulta e uma experiência New Relic mais simplificada.

## Acesse o otimizador de computação

Você encontrará o otimizador de computação no painel de administração do New Relic.

1. Faça login na New Relic.
2. Abra suas configurações pessoais clicando no seu nome de usuário no canto inferior esquerdo da interface do New Relic. Vá para o painel de administração.
3. Clique no **compute optimizer** para acessar o dashboard.

## O dashboard do otimizador de computação

O dashboard do otimizador de computação apresenta duas métricas cruciais:

* **Uso ineficiente:** Isso fornece uma visão geral de várias áreas de computação ineficientes identificadas em sua conta.
* **Economia potencial:** esta métrica quantifica a economia estimada que você pode obter implementando as recomendações fornecidas pela ferramenta.

### Recomendações

O núcleo do otimizador da computação é a tabela de recomendações, que detalha as oportunidades de otimização identificadas:

* **Nome do componente:** identifica o item ou instância específica que tem uma oportunidade de otimização (por exemplo, o nome de uma condição de alerta específica).
* **Capacidade do produto:** identifica a capacidade mais ampla do produto New Relic associada à oportunidade de otimização. A versão inicial foca na condição do alerta.
* **Uso da CCU:** fornece uma compreensão do consumo atual da computação (uso da CCU) para cada componente listado, ajudando você a entender seu impacto atual.
* **Recomendação:** Oferece insights práticos para otimizar seu computação. Por exemplo, você pode ver uma recomendação para &quot;Adicionar uma cláusula `WHERE` fora da instrução `SELECT` &quot;, o que pode economizar uma porcentagem significativa de CCUs para uma condição de alerta específica.
* **Estimativa de economia de CCU:** quantifica a redução potencial de computação para cada recomendação individual listada na tabela. Por exemplo, se uma recomendação for otimizar **o alerta X**, esta coluna mostra as economias específicas de CCU que você pode obter apenas para **o alerta X**. Todos esses valores individuais de estimativa de economia de CCU sugeridos em cada linha são então somados para calcular a métrica de economia potencial total.

Todos insights apresentados no otimizador de computação são dinâmicos e baseados nas configurações do seu seletor de horas, permitindo que você analise dados relevantes para períodos específicos.

O menu de ações para cada recomendação oferece uma consulta otimizada sempre que aplicável, facilitando a implementação direta das alterações sugeridas.
