Para organizações com várias contas New Relic, cada uma com requisitos de CCU distintos, o Compute Budgets fornece monitoramento, visibilidade e alertas por conta para garantir que as contas permaneçam dentro de seus limites de uso definidos.

## Tipos de orçamento por estrutura do cliente

O tipo de orçamento que você pode criar depende da sua estrutura de cliente:

### Para clientes de locatário único

Você pode criar **orçamentos no nível da conta** para contas individuais ou grupos de até 5 contas dentro de sua organização.

### Para clientes multi-tenant

**Gerenciar organizações** pode criar dois tipos de orçamentos:

* **Orçamentos no nível da organização**: Orçamentos para organizações gerenciadas individuais

  * Um orçamento por organização
  * Uma organização por orçamento
  * Crie vários orçamentos no nível da organização conforme necessário

* **Orçamentos no nível da conta**: Orçamentos para contas dentro da organização gerenciadora (mesma funcionalidade que clientes de locatário único)

**Organizações gerenciadas** não podem criar orçamentos.

<Callout variant="important">
  Se você faz parte de uma organização gerenciadora, pode criar orçamentos para qualquer organização gerenciada em sua hierarquia de clientes. Use o seletor de organização para escolher a organização ao criar um orçamento de organização.
</Callout>

## Configurar orçamentos em nível de conta

<Steps>
  <Step>
    ### Abrir Orçamentos de Computação

    1. Acesse **[one.newrelic.com](https://one.newrelic.com/)**.
    2. No [menu do usuário](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings), selecione <DNT>**Administration**</DNT> &gt; <DNT>**Compute Budgets**</DNT>.
  </Step>

  <Step>
    ### Criar um orçamento

    Na seção <DNT>**Functional budgets**</DNT> , selecione <DNT>**Add a budget**</DNT>.

    <Callout variant="important">
      **Para clientes multi-tenant**, esta seção exibe:

      * <DNT>**Organization level budgets**</DNT> para organizações gerenciadas
      * <DNT>**Account level budgets**</DNT> para a organização gerenciadora

      Organizações gerenciadas não podem criar orçamentos no nível da conta.
    </Callout>
  </Step>

  <Step>
    ### Atribua seu orçamento

    1. Insira um nome para seu orçamento.

    2. Selecione as contas ou organização para seu orçamento:

       **Para orçamentos no nível da conta de locatário único ou organização gerenciadora**: Na lista suspensa, selecione e atribua até 5 contas a este orçamento.

       **Para orçamentos de nível de organização multi-tenant**: Selecione uma organização no menu suspenso. Cada orçamento no nível da organização pode conter apenas uma organização.

       <Callout variant="tip">
         Ao criar orçamentos no nível da organização, use o seletor de organização para visualizar e selecionar a organização gerenciada para seu orçamento.
       </Callout>

    3. Selecione <DNT>**Next**</DNT>.
  </Step>

  <Step>
    ### Definir limites de orçamento mensal

    1. Para definir limites de gastos para sua conta selecionada, na seção <DNT>**Set monthly budget limits**</DNT>, atribua os limites para as SKUs de computação que você adquiriu para estabelecer o orçamento da sua conta.
    2. Selecione <DNT>**Next**</DNT>.

    <img title="Screenshot displaying the Set monthly budget limits page with advanced and core CCU limits highlighted" alt="Screenshot displaying the Set monthly budget limits page with advanced and core CCU limits highlighted" src="/images/set-monthly-account-budget.webp" />
  </Step>

  <Step>
    ### Configurar notificação

    1. Para evitar excessos e gerenciar custos de forma eficaz, na página <DNT>**Set up notifications**</DNT>, defina a porcentagem do limite e escolha o destino para as notificações.

    2. Para receber um aviso quando você tiver usado uma determinada porcentagem da CCU atribuída a um orçamento para um mês específico, na guia <DNT>**Thresholds**</DNT>, selecione <DNT>**+ Add a threshold**</DNT> e insira a porcentagem na qual você deseja receber notificações.

    3. Para garantir que as notificações sejam enviadas quando o orçamento exceder o limite, na guia <DNT>**Destinations**</DNT>, adicione pelo menos uma pessoa para ser notificada. Você pode escolher entre as seguintes opções:

       * <DNT>**Email**</DNT>: Insira os endereços de e-mail dos usuários do New Relic para notificá-los sobre o orçamento.
       * <DNT>**Webhook**</DNT>: Insira a URL do webhook para enviar notificações para um endpoint específico.

    4. Selecione <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    ### Visualizar detalhamento de uso

    Para visualizar o <DNT>**Usage breakdown**</DNT>, selecione o menu de três pontos. Você pode acessar as seguintes informações conforme disponível:

    * Uma análise detalhada do uso de CCU dia após dia, adaptada aos SKUs de Computação que você possui.
    * Uso de CCU categorizado por <DNT>**Users**</DNT>, <DNT>**Accounts**</DNT> e <DNT>**Capabilities**</DNT>.
    * Detalhes sobre a CCU, incluindo <DNT>**Budget name**</DNT>, contas associadas e informações como configurações de <DNT>**Budget time range**</DNT>, <DNT>**Product**</DNT> e <DNT>**Notification**</DNT>.
  </Step>
</Steps>
