Se você for cliente New Relic , poderá ver seu uso em nossa página **Usage Summary** . Embora os números sejam uma aproximação, você pode usar essas visualizações para ter uma ideia geral sobre como seu consumo pode afetar sua conta.

Para visualizar a interface de uso, acesse **[one.newrelic.com](https://one.newrelic.com) &gt; (clique no seu nome de usuário) &gt; Administration &gt; Usage Summary**. Esta página mostra uma visão geral do seu uso:

<img title="New Relic usage UI" alt="New Relic usage UI" src="/images/accounts_screenshot-full_usage-ui.webp" />

<figcaption>
  A interface de uso do New Relic
</figcaption>

Além de usar a interface de uso, você também pode realizar uma consulta para medir seu uso. Para mais detalhes, consulte [consulta e alerta sobre dados de utilização](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts/).
