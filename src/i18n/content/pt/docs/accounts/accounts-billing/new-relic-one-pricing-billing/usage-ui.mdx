Se você é cliente da New Relic, pode ver seu uso em nossa página <DNT>**Plan & usage**</DNT> . Embora os números sejam uma aproximação, você pode usar essas visualizações para ter uma ideia geral sobre como seu consumo pode afetar sua conta.

Para visualizar a interface de uso, acesse <DNT>**one.newrelic.com**</DNT>, clique no menu do usuário e clique em <DNT>**Administration**</DNT>. A página <DNT>**Plan & usage UI**</DNT> mostra uma visão geral do seu uso como esta:

Além de usar a interface de uso, você também pode realizar uma consulta para medir seu uso. Para mais detalhes, consulte [consulta e alerta sobre dados de utilização](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts/).
