Adicionar uma conta à sua organização New Relic é uma forma de organizar e separar dados para equipes ou projetos específicos.

## Requisitos [#requirements]

Requisitos para adicionar uma conta à sua organização New Relic:

* [Edição](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing#editions)Pro ou Enterprise

* O usuário deve ser um usuário principal ou um usuário completo da plataforma com

  <DNT>
    [**Organization settings**](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#admin-settings)
  </DNT>

  .

* organizações com [contas de parceria](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions) devem usar a [API de parceria](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partner-api).

Algumas regras de nomenclatura de contas:

* As contas são identificadas pelo seu [ID de conta](/docs/accounts/accounts-billing/account-structure/account-id). Não existem regras sobre a exclusividade do nome da conta dentro de uma organização.
* O nome de uma conta pode ter no máximo 255 caracteres.

## Por que adicionar uma conta? [#why-add-accounts]

Para motivos para adicionar contas, consulte [organização e estrutura de contas](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure).

## Adicionar contas [#add-accounts]

Para adicionar uma conta da interface: na [página da interface<DNT>**Access management**</DNT> ](https://one.newrelic.com/admin-portal/organizations/organization-detail), clique <DNT>**Accounts**</DNT> e, em seguida, clique em <DNT>**Create account**</DNT>. (Se você estiver em nosso modelo de usuário original, consulte [Adicionar contas](/docs/accounts/original-accounts-billing/original-users-roles/parent-child-account-structure/#creating).)

Para adicionar uma conta via API: consulte [Gerenciar contas com NerdGraph](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph).

Para cancelar uma conta, [registre um ticket com nossa equipe de suporte técnico global](/docs/new-relic-solutions/solve-common-issues/find-help-use-support-portal/#file-ticket).
