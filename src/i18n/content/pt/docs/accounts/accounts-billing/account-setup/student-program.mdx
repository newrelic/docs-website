Você é um estudante que está tentando aprender como coletar e analisar telemetria de hardware e software? Você pode fazer isso com o New Relic for Students. Este é um programa que oferece acesso gratuito a dados, usuário e recurso New Relic.

## Benefícios [#student-benefits]

Os alunos qualificados recebem acesso gratuito a uma conta New Relic for Students por dois anos, que inclui:

* 500 GB de [ingestão de dados](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/) por mês
* 3 [Usuário da plataforma](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type/)completa
* 10.000 [verificações sintéticas](/docs/synthetics/synthetic-monitoring/getting-started/monitor-limits/) por mês
* Acesso a todos os demais recursos disponíveis para clientes [Standard Edition](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/)
* Acesso a conteúdo exclusivo, incluindo newsletter mensal, workshops, encontros e eventos para participantes do programa

## Requisitos de elegibilidade [#student-eligibility]

Você pode participar se estiver em uma destas categorias:

* Estudantes atualmente matriculados em uma universidade ou faculdade credenciada em seu país de residência
* Alunos atualmente matriculados em uma escola técnica ou de codificação ou bootcamp
* Tecnólogos em início de carreira que podem não frequentar uma universidade, faculdade ou outra escola técnica, mas que usam uma plataforma de aprendizagem on-line para aprender sobre desenvolvimento de software

Os seguintes não são elegíveis:

* Universidades, faculdades e outras instituições técnicas
* Empresas com fins lucrativos
* Sem fins lucrativos organização (consulte [observabilidade for Good (O4G)](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/eligibility-guidelines-new-relic-nonprofit-program/) para entender nosso programa separado para organizações sem fins lucrativos)

## Como se inscrever [#sign-up]

Por meio do Student Developers Pack do GitHub Education, você terá acesso ao New Relic for Students, juntamente com mais de 80 outros recursos de outros parceiros do Student Developers Pack.

Preencha o seguinte para se inscrever:

1. Verifique se você está inscrito no GitHub Education.

2. Crie uma [conta gratuita na New Relic](https://newrelic.com/signup).

3. Login em

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

   e, no canto inferior esquerdo, clique em

   <DNT>
     **Help > Resources > New Relic for Students**
   </DNT>

   .

4. Clique em

   <DNT>
     **GitHub Education**
   </DNT>

   e siga as etapas para usar o OAuth para conectar sua conta do GitHub Student ao New Relic.

Assim que o sistema New Relic receber seu status do GitHub, você verá os dados adicionais e o usuário disponíveis em sua conta.
