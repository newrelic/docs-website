Há várias maneiras de um usuário New Relic fazer log em uma organização New Relic:

* E-mail e senha: O método de login padrão, que requer um endereço de e-mail e uma senha do usuário.
* Login social: você pode usar Google, Google Workplace, GitHub e Bitbucket para se inscrever e log no New Relic.
* Logon único: para opções de login do provedor de identidade, consulte [Introdução ao SAML SSO e SCIM](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim).

Está tendo problemas para fazer login? Consulte [Resolução de problemas de login](/docs/accounts/accounts-billing/account-setup/troubleshoot-new-relics-password-email-address-login-problems/).

Para gerenciar log do seu usuário, consulte [Domínio de autenticação](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more).

## Contagem e rastreamento de usuários [#email]

Para obter detalhes sobre como os registros do usuário são rastreados, consulte [Rastreamento de usuários](/docs/accounts/accounts-billing/account-setup/multiple-logins-found#user-records).

## Alternar entre registros de usuário [#switch]

Para usuários com acesso a mais de um registro de usuário New Relic, consulte [Alternar entre usuários](/docs/accounts/accounts-billing/account-setup/multiple-logins-found).
