Nosso editor <DoNotTranslate>**Instrumentation**</DoNotTranslate> permite que o usuário do aplicativo .NET implemente a instrumentação personalizada a partir da interface do usuário do New Relic sem a necessidade de reiniciar o aplicativo instrumentado. Este editor:

* Permite que o usuário autorizado edite a instrumentação personalizada da interface do New Relic em vez dos arquivos XML locais.
* Valida em relação ao esquema de instrumentação personalizada.
* Fornece uma opção para clientes que não têm acesso ao código do aplicativo ou aos hosts nos quais seus aplicativos são executados.

Para usar o editor <DoNotTranslate>**Instrumentation**</DoNotTranslate> :

1. Vá para <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DoNotTranslate>. Encontre seu aplicativo e clique em <DoNotTranslate>**Settings > Instrumentation**</DoNotTranslate>.

2. Use o editor <DoNotTranslate>**Instrumentation**</DoNotTranslate> para fornecer XML usando a mesma sintaxe descrita para [editar arquivos de instrumentação .xml locais](/docs/agents/net-agent/custom-instrumentation/create-transactions-xml-net) e [adicionar detalhes de instrumentação.](/docs/agents/net-agent/custom-instrumentation/add-detail-transactions-xml-net) Use qualquer uma destas opções na interface:

   * Digite o código XML diretamente.

   * Copie o código e cole-o de uma área de transferência.

   * Clique em

     <DoNotTranslate>**Download Sample XML**</DoNotTranslate>

     para criar um arquivo de instrumentação de amostra como ponto de partida.

   * Carregar um .xml arquivo selecionando

     <DoNotTranslate>**Choose file**</DoNotTranslate>

     . Para obter um exemplo, consulte [Exemplo MyInstrumentation.xml](/docs/agents/net-agent/custom-instrumentation/add-detail-transactions-xml-net#sample).

3. O editor <DoNotTranslate>**Instrumentation**</DoNotTranslate> valida seu código XML enquanto você digita. Para executar a validação XML fora do editor, selecione <DoNotTranslate>**Download XSD**</DoNotTranslate> e use o arquivo XSD com sua ferramenta de validação XML favorita para diagnosticar problemas.

4. Para implantar as alterações, selecione <DoNotTranslate>**Submit**</DoNotTranslate>.

## Ver resultados de instrumentação [#view]

O agente .NET iniciará a nova instrumentação dentro de alguns [ciclos de coleta](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#harvest-cycle) (normalmente alguns minutos). A seção <DoNotTranslate>**Current instrumentation**</DoNotTranslate> da interface <DoNotTranslate>**Instrumentation**</DoNotTranslate> mostrará o status das alterações aplicadas.

O XML do editor <DoNotTranslate>**Instrumentation**</DoNotTranslate> é combinado com o XML que existe nos arquivos de instrumentação local na máquina que executa o agente. O agente procura periodicamente alterações no XML final (combinado) e as coleta dinamicamente, sem exigir a reinicialização do agente ou aplicativo.
