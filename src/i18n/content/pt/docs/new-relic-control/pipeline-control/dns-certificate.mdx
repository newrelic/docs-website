Siga estas etapas para configurar um nome de host DNS privado para o cluster de gateway para facilitar a comunicação confiável entre o agente New Relic APM e o gateway em sua rede:

1. Crie um nome DNS para o host usando [AWS Route 53](https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/hosted-zones-private.html).
2. Obtenha um certificado para o nome do host do gateway via [ACM](https://docs.aws.amazon.com/acm/latest/userguide/acm-public-certificates.html).
