/\* Se você estiver tentando instalar o gateway pela primeira vez e encontrar problemas, este guia ajudará você a diagnosticar e resolver problemas comuns. Siga os passos abaixo com base no problema específico que você encontrar: \*/

## Problema: Capacidades de usuário insuficientes

Você não tem as permissões necessárias associadas ao Org Product Admin e ao organização Manager.

## Solução

* O usuário verá uma mensagem de erro informando: “Você não possui os recursos necessários em nível de organização para configurar a autenticação do agente”.
* Contrate o(s) administrador(es) da sua conta para receber uma função que inclua os recursos necessários para a criação de identidade do sistema.
* Consulte a [documentação de permissões do usuário](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/) para obter orientação.

## Problema: Versão desatualizada do gráfico Helm

O script de linha de comando para instalar o gráfico Helm falha porque sua versão local do Helm está desatualizada.

## Solução

* Atualize sua instalação local do Helm para a versão mais recente para garantir a compatibilidade com o script de instalação.
* Procure por mensagem de erro indicando a necessidade de uma atualização Helm e siga as instruções para atualizar.
