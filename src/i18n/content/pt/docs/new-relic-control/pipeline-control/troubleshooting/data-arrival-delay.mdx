## Problema: Dados chegando além do SLA

Os dados não estão sendo recebidos dentro do prazo esperado.

## Solução

* Verifique a latência métrica para identificar atrasos na transmissão de dados.
* Aumente o número mínimo de pods de gateway para melhorar a capacidade de processamento e reduzir a latência.
