/\* Se você alterou uma regra de gateway e ela não aparece na lista de regras, este guia ajudará você a diagnosticar e resolver problemas comuns. Siga os passos abaixo com base no problema específico que você encontrar: \*/

## Problema: Mudança não implantada

Você não sabe que a lista de regras mostra apenas estados ativos e ainda não implantou a alteração.

## Solução

* Verifique a página de atualizações para atualizações de gateway pendentes para implantar.
* Certifique-se de implantar suas alterações para que elas sejam refletidas na lista de regras.

## Problema: condição de corrida da API

Vários usuários enviam requests de atualizações de gateway simultaneamente, fazendo com que as alterações sejam perdidas.

## Solução

* Entre em contato com [o suporte da New Relic](https://support.newrelic.com).
* Refaça a ação que foi perdida devido à condição de corrida.
