/\* Se você não estiver vendo nenhum dado MELT para o gateway na interface Controle de Pipeline , este guia ajudará você a diagnosticar e resolver problemas comuns. Siga os passos abaixo com base no problema específico que você encontrar: \*/

## Problema: problemas no menu dropdown da conta

Você não tem **All accounts** ou a conta associada à chave de licença do gateway selecionada na interface do usuário do Controle de Pipeline.

## Solução

* Verifique o menu dropdown da conta na interface do Controle de Pipeline para garantir que a conta correta esteja selecionada.
* Certifique-se de que você está visualizando **All accounts** ou a conta específica associada à chave de licença do gateway para acessar os dados MELT relevantes.
