## Problema: Esgotamento de recursos

O cluster esgotou seus recursos de CPU ou memória.

## Solução

* Use a interface Kubernetes para visualizar eventos pod e trabalhos pendentes para identificar restrições de recursos.
* Aumente o tamanho do pool de nós ou ajuste os limites de recursos (CPU e memória) requests para o pod.
* Verifique os limites do provedor cloud para o número ativo de nós e ajuste a configuração conforme necessário.
