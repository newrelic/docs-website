## Problema: condição de corrida da API

Vários usuários enviam requests de atualizações de gateway (ações de criação, atualização e exclusão) simultaneamente, fazendo com que as alterações sejam perdidas.

## Solução

* Entre em contato com [o suporte da New Relic](https://support.newrelic.com).
* Refaça a ação que foi perdida devido à condição de corrida.
