O uso do Controle de Pipeline é faturável de acordo com seu Pedido. Os custos associados a esse recurso são determinados pelos seguintes fatores, conforme aplicável ao modelo de preços associado à sua Conta:

**Advanced Compute** (medido em CCU avançado):

* GBs descompactados recebidos pelo gateway (como resultado da configuração do agente ou da carga de telemetria para serem enviados pelo gateway), independentemente de as regras do gateway terem efeito sobre os dados
* GBs descartados pelas regras cloud
* evento (incluindo métrica, evento, logs e trace) escaneado por regras cloud

**Core Compute**: ao visualizar determinadas páginas na interface Controle de Pipeline que geram consulta NRDB, medida no Core CCU.

**Ingestão de dados**: todos os dados não descartados pelo Controle de Pipeline e armazenados no NRDB, medidos em GB ingeridos.

Para clientes de computador sob demanda, os produtos de computador principal e avançado são disponibilizados como produto de computador, medido como CCU.

Para mais detalhes sobre preços, entre em contato com nossa equipe de vendas ou consulte seu pedido.
