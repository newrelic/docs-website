O uso do Pipeline Control é faturável de acordo com o seu Pedido. O Pipeline Control é vendido como parte do SKU Advanced Compute, um complemento aplicado aos modelos de preços base (preços baseados em usuário ou Core Compute). O Advanced Compute agrupa recursos avançados e cobra por eles usando aCCUs (Advanced Compute Consumption Units).

## Como os custos são calculados

Os custos associados ao Controle de Pipeline são determinados pelos seguintes fatores:

### Medição do gateway

O Gateway é executado na sua infraestrutura e a medição é simples:

**O que é medido**:

* **Volume de dados recebido** nos endpoints do gateway (medido em bytes não compactados)
* Cobrado **independentemente do processamento aplicado** ou se os dados são descartados
* Medido **na entrada** antes de ocorrer qualquer filtragem/descarte

### Medição de regras de nuvem

As regras de nuvem são executadas na infraestrutura da New Relic e, portanto, são cobradas com um valor superior ao do gateway.

**O que é medido** (para regras de descarte):

* **GBs descartados**: Volume de dados removido pelas suas regras
* **BSEs (bilhões de eventos verificados)**: Eventos avaliados ao executar regras
* Ambos são itens de linha separados que incorrem em cobranças
* **BSEs normalmente representam uma fração muito menor** da cobrança geral de regras de nuvem em comparação com GBs descartados para a grande maioria dos clientes

**Contagem de BSE**:

* Apenas varre dados correspondentes ao nível raiz `SELECT` na regra NRQL
* Exemplo: `DELETE FROM Log WHERE level = 'INFO'` conta apenas BSEs para dados em `Log`
* Consultas mais seletivas escaneiam menos eventos

### Outros custos do Pipeline Control

**Core Compute**: Ao visualizar certas páginas na interface de controle de pipeline que geram consultas NRDB, medido em Core CCU.

**Ingestão de dados**: todos os dados não descartados pelo Controle de Pipeline e armazenados no NRDB, medidos em GB ingeridos.

Para clientes de computador sob demanda, os produtos de computador principal e avançado são disponibilizados como produto de computador, medido como CCU.

Para obter detalhes específicos sobre preços e entender qual modelo apresenta o melhor custo-benefício para o seu caso de uso, entre em contato com nossa equipe de vendas ou consulte seu Pedido.
