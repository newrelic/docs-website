Este documento descreve as etapas para solucionar problemas comuns durante a instalação ou operação do Controle de Agentes.

## Falha de implantação

Ocorre uma falha na implantação e mais informações são necessárias para entender o erro.

## Solução

Clique no cluster com falha para acessar informações detalhadas sobre o erro. Esses insights adicionais podem ajudar a diagnosticar e resolver o problema de forma eficaz.
