Este documento descreve as etapas para solucionar problemas comuns durante a instalação ou operação do Controle de Agentes.

## Usando o Data Explorer

O New Relic Data Explorer é uma ferramenta poderosa para investigar problemas. Você pode acessá-lo clicando em **Query your data** na parte inferior da maioria das páginas da plataforma New Relic ou usando o atalho de teclado `CTRL+SHIFT+O`.

Para obter informações mais detalhadas sobre seu recurso, consulte nossa [introdução ao Data Explorer](/docs/query-your-data/explore-query-data/query-builder/introduction-new-data-explorer/).

### Investigar evento de implantação com NRQL

Para problemas de implantação complexos, você pode executar uma consulta NRQL no Data Explorer para ver os dados granulares em nível de evento para uma implantação específica. Isso permite que você veja a sequência exata de ações e quaisquer erros que ocorreram.

Para investigar uma implantação:

1. Na aba de **Deployments** de uma frota específica, clique no nome da implantação que você deseja investigar. O **ID de implantação** será listado nos detalhes do resumo. Copie este ID.
2. Abra o Explorador de Dados.
3. Certifique-se de que a conta correta esteja selecionada.
4. Cole a seguinte consulta no criador de consulta, substituindo o espaço reservado pelo ID que você copiou.
5. Execute a consulta. Para facilitar a análise, visualize os resultados como uma **Table**. A coluna `details` geralmente contém mensagens de erro esclarecedoras.

```nrql
FROM FleetDeployment SELECT timestamp, action, status, desiredCompletedStatus, agentTypes, ringName, details, agentUid, agentControlUids, fleetId, deploymentId, duration, action WHERE deploymentId = '<YOUR_DEPLOYMENT_ID>' SINCE 1 hour ago LIMIT MAX
```
