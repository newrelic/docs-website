Depois de [adicionar facetas a um nível de serviço](/docs/service-level-management/facet-slm/configure), você pode visualizar e analisar os dados de conformidade e de orçamento de erros divididos por essas facetas para identificar onde os problemas de desempenho estão ocorrendo.

## Visualizar resultados facetados [#view-faceted-results]

Para visualizar a conformidade e o orçamento de erros para valores de atributos individuais e suas combinações:

1. Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Service Levels**</DNT>.
2. Selecione o nível de serviço.
3. Use o dropdown <DNT>**View by group**</DNT> e o painel <DNT>**Group by**</DNT> para analisar os atributos que você configurou para agrupamento.

## Exemplo [#example]

Quando você agrupa por `environment` e `region`, o dropdown <DNT>**View by group**</DNT> e o painel <DNT>**Group by**</DNT> exibem dados de desempenho para cada valor de atributo individual (todos os ambientes, todas as regiões), bem como para cada combinação de ambiente e região:

| Valor do atributo     | Conformidade | Orçamento de erros restante |
| --------------------- | ------------ | --------------------------- |
| produção              | 99,05%       | 48,5%                       |
| preparação            | 99,50%       | 60%                         |
| us-east-1             | 99,70%       | 72,5%                       |
| us-west-1             | 98,20%       | 12%                         |
| produção, us-east-1   | 99,9%        | 85%                         |
| produção, us-west-1   | 98,2%        | 12%                         |
| preparação, us-east-1 | 99,5%        | 60%                         |

## Analise seus dados [#interactive-analysis]

Use a visualização facetada para filtrar e comparar dados de conformidade em diferentes segmentos.

<img title="Faceted service level UI" alt="Screenshot showing the faceted service level analysis interface with group filters and compliance data" src="/images/slm_screenshot-full_faceted-view.webp" />

* **Para visualizar dados de um segmento específico**: selecione uma faceta no dropdown na parte superior da página. Os gráficos principais são atualizados para mostrar dados apenas para esse segmento, e um gráfico <DNT>**Group SLO**</DNT> aparece mostrando o desempenho do grupo selecionado em relação à conformidade geral.
* **Para comparar atributos individuais versus combinações**: quando você tiver duas facetas selecionadas, use a visualização individual para ver cada atributo isoladamente, ou a visualização combinada para ver as combinações de atributos.
* **Para retornar à visão geral**: selecione <DNT>**None**</DNT> no dropdown.

## Alertas facetados [#faceted-alerting]

[Você pode criar alertas](/docs/service-level-management/alerts-slm/) com base em facetas específicas para garantir que você seja notificado quando um determinado segmento estiver falhando.

**Tipos de alerta suportados:**

* Alertas de conformidade de SLI
* Alertas de queima rápida
* Alertas de queima lenta

**Não suportado:**

Alertas de orçamento de erro restante (estes permanecem disponíveis apenas no nível agregado)
