Temos diversas maneiras de você monitor seus aplicativos e serviços [Microsoft Azure](https://azure.microsoft.com) . Aqui está uma visão geral dessas ofertas e por que você usaria uma em vez de outra.

* New Relic no Azure: com nosso [serviço Azure Native New Relic](/docs/infrastructure/microsoft-azure-integrations/get-started/azure-native), você pode implantar o New Relic criado no Azure, diretamente do portal do Azure. Esta opção te dá o equivalente ao recurso abaixo (e também ao monitoramento do aplicativo Java), e você pode configurá-lo diretamente do portal do Azure.
* Nossa integração Azure: Nossa [integração de infraestrutura Azure](/docs/infrastructure/microsoft-azure-integrations/get-started/activate-azure-integrations) suporta dezenas de serviços Azure.
* .NET APM: Você pode instalar [nosso agente .NET APM](/install/dotnet) para monitor seu aplicativo Azure e receber dados de telemetria detalhados.
