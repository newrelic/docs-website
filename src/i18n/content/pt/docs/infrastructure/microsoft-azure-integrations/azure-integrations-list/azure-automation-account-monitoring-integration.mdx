[A integração da New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) inclui uma integração para reportar as métricas da sua [conta de automação da Microsoft](https://learn.microsoft.com/en-us/azure/automation/) e outros dados para New Relic. Este documento explica como ativar a integração e descreve os dados reportados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o serviço Azure Automation Account. A Automação do Azure oferece automação baseada em nuvem, atualizações de sistema operacional e serviço de configuração que oferece suporte ao gerenciamento consistente em seus ambientes Azure e não-Azure.

Usando o New Relic, você pode:

* Veja os dados da conta de automação do Azure em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta os seus serviços de Automação do Azure através da integração do Azure Monitor de acordo com um intervalo [de sondagem](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
