[O monitoramento de infraestrutura New Relic ](/docs/infrastructure)fornece uma integração para [a Rede Virtual doMicrosoft Azure](https://docs.microsoft.com/en-us/azure/virtual-network/virtual-networks-overview) que reporta dados do seu serviço de Rede Virtual para a New Relic. Este documento explica como ativar esta integração e descreve os dados que podem ser capturados.

## Recurso

A integração da New Relic para Rede Virtual do Azure reporta dados métricos sobre suas redes virtuais (VNets), como pacotes descartados por segundo ou bytes encaminhados por segundo. Também coleta dados sobre o status e a configuração da sua conta.

Você pode monitor e alertar sobre os dados da sua rede virtual a partir do [New Relic](/docs/infrastructure) e pode [criar consultas personalizadas e painéis gráficos](/docs/using-new-relic/data/understand-data/query-new-relic-data).

## Ativar integração [#requirements]

Para habilitar essa integração siga os procedimentos padrão para [ativar seu serviço Azure no New Relic](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations).

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informações [de sondagem](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-polling-intervals-infrastructure-integrations) padrão para a integração da Rede Virtual:

* Intervalo de sondagem: 1 minuto para métrica; 5 minutos para inventário
* Resolução: 1 ponto de dados por minuto

## Encontre e use dados [#find-and-use]

Para encontrar seus dados de integração, acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure**</DNT> e procure a integração.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `AzureVirtualNetworksPublicIpAddressSample` .

Para obter mais informações sobre como localizar e usar dados de integração, consulte [Compreender e usar dados](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
