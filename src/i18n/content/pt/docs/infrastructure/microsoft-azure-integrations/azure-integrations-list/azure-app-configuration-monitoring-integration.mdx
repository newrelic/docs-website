[A integração da New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) inclui uma integração para reportar sua métrica [de configuração do aplicativo Microsoft Azure ](https://learn.microsoft.com/en-us/azure/azure-app-configuration/)e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados reportados.

## Recurso

Configuração do Azure App é um serviço do Azure que permite ao usuário gerenciar configuração dentro da nuvem. O usuário pode criar lojas de configuração de aplicativos para armazenar configurações de valor principal e consumir configurações armazenadas do aplicativo, construir pipelines, liberar processos, microsserviços e outros recursos do Azure.

Usando o New Relic, você pode:

* Veja os dados de configuração do aplicativo Azure em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta seus serviços de configuração de aplicativos do Azure por meio da integração do Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
