[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar suas métricas [de interface de redeMicrosoft Azure ](https://learn.microsoft.com/en-us/azure/virtual-network/virtual-network-network-interface)e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o serviço Azure Network Interface. Uma interface de rede permite que uma máquina virtual do Azure se comunique com a Internet, o Azure e recursos locais.

Usando o New Relic, você pode:

* Veja os dados da Interface Azure Netwrok no painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração do Azure Monitor](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para ativar o serviço do Azure no New Relic.

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta seus serviços de interface Netwrok por meio da integração do Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; Azure &gt; (select an integration)**</DNT>.
