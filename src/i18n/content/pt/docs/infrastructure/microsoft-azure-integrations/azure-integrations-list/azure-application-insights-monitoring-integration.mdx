[New RelicA integração da](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) inclui uma integração para relatar métricas [Microsoft Azure do aplicativo insights](https://learn.microsoft.com/en-us/azure/azure-monitor/app/app-insights-overview) e outros dados para New Relic. Este documento explica como ativar a integração e descreve os dados reportados.

## Recurso

O aplicativo insights é um recurso do Azure Monitor. É um serviço extensível de desempenho do aplicativo Management (APM) para desenvolvedores web em múltiplas plataformas. Você pode usá-lo para monitor seu aplicativo web ao vivo e ele detectará automaticamente anomalias de desempenho.

Usando o New Relic, você pode:

* Veja os dados insights do aplicativo Azure em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta os serviços insights do seu aplicativo por meio da integração Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
