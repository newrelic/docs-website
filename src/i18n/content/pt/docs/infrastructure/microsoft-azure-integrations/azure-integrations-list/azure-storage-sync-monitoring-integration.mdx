[A integração da New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) inclui uma integração para reportar sua métrica [de sincronização de armazenamento Microsoft Azure ](https://azure.microsoft.com/en-us/services/storagesync/)e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados reportados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o serviço Storage Sync, o Azure Storage Sync permite a sincronização de dados entre servidores Windows locais e o Azure Storage. Com o Azure Storage Sync, você pode centralizar seus serviços de arquivos no Azure e armazenar dados em cache em seus servidores Windows locais, disponibilizando-os para outro usuário ou aplicativo.

Usando o New Relic, você pode:

* Visualize os dados do Azure Storage Sync em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta os serviços do Azure Storage Sync de acordo com um intervalo [de sondagem](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
