[A integração da New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) inclui uma integração para relatar suas métricas [de evento Microsoft Azure Media Services Live](https://learn.microsoft.com/en-us/azure/media-services/latest/live-event-streaming-best-practices-guide) e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados reportados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o evento Azure Media Services Live. Azure Media Services Live evento é um serviço fornecido pelo Microsoft Azure que permite streaming de vídeo e áudio ao vivo para diversos eventos, como conferências, webinars, concertos e eventos esportivos. Ele permite ao usuário capturar, codificar, empacotar e entregar conteúdo de vídeo ao vivo para públicos em todo o mundo através da Internet.

Usando o New Relic, você pode:

* Visualize dados de eventos ao vivo dos Serviços de Mídia do Azure em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta seu evento Azure Media Services Live por meio da integração Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
