[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar seu [banco de dadosMicrosoft Azure para métricas PostgreSQL](https://docs.microsoft.com/en-gb/azure/postgresql/) e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

New Relic reúne dados de banco de dados do serviço Azure banco de dados para PostgreSQL, que fornece banco de dados PostgreSQL Community totalmente gerenciado e pronto para empresas como um serviço. O serviço oferece alta disponibilidade, escalabilidade elástica, backups automáticos e proteção de dados em repouso e em movimento.

Usando o New Relic, você pode:

* Visualize o banco de dados do Azure para dados PostgreSQL no painel pré-criado New Relic .
* Execute [uma consulta personalizada e visualize os dados](/docs/using-new-relic/data/understand-data/query-new-relic-data).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga os procedimentos padrão para [ativar seu serviço Azure no New Relic](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations).

## Configuração e polling [#polling]

New Relic consulta seu serviço de banco de dados do Azure de acordo com um intervalo [de pesquisa](/docs/integrations/microsoft-azure-integrations/getting-started/azure-integration-polling) padrão, que varia dependendo da integração. Para integração do banco de dados do Azure para PostgreSQL:

* Intervalo de sondagem: 5 minutos (frequência máxima de sondagem recomendada: 1 hora)
* Resolução: 1 ponto de dados por minuto

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

## Visualizar e usar dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data): Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; Azure &gt; (select an integration)**</DNT>.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o tipo de evento `AzurePostgreSqlServerSample` , com um valor de provedor de `AzurePostgreSqlServer`.
