[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar seu [fluxo de trabalho métrico do Microsoft Azure Functions ](https://azure.microsoft.com/en-us/products/logic-apps/)e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

O fluxo de trabalho Azure Functions refere-se aos Aplicativos Lógicos do Azure no tipo de plano padrão. Azure Logic Apps é uma plataforma de nuvem onde você pode criar e executar fluxo de trabalho automatizado com pouco ou nenhum código. Ao usar o designer visual e selecionar operações pré-construídas, você pode criar rapidamente um fluxo de trabalho que integra e gerencia seus aplicativos, dados, serviços e sistemas.

Usando o New Relic, você pode:

* Visualize dados de fluxo de trabalho Azure Functions em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar o serviço de fluxo de trabalho Azure Functions no monitoramento de infraestrutura do New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta seu serviço de fluxo de trabalho do Functions por meio da integração Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
