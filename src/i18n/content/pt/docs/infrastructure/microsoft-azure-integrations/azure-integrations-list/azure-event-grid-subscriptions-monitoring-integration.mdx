[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar sua [assinatura métricaMicrosoft Azure Event Grid](https://learn.microsoft.com/en-us/azure/event-grid/subscribe-through-portal) e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o serviço Azure evento Grid assinatura. A assinatura informa ao evento Grid qual evento sobre um tema você tem interesse em receber. Ao criar a assinatura, você fornece um endpoint para tratar o evento. Você pode filtrar os eventos que são enviados ao endpoint.

Usando o New Relic, você pode:

* Visualize os dados de assinatura do evento Grid do Azure em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta seus serviços de assinatura do Azure evento Grid por meio da integração Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
