[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar sua métrica [Microsoft Azure Data Lake Analytics](https://azure.microsoft.com/en-us/products/data-lake-analytics) e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o serviço Azure Data Lake Analytics. O Azure Data Lake Analytics é um serviço de análise sob demanda que simplifica big data. Em vez de implantar, configurar e ajustar hardware, você escreve consultas para transformar seus dados e extrair insights valiosos. O serviço de análise pode lidar com trabalhos de qualquer escala instantaneamente, definindo a quantidade de energia necessária. Você só paga pelo seu trabalho quando ele está em execução, o que o torna econômico.

Usando o New Relic, você pode:

* Visualize os dados do Azure Data Lake Analytics em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta o Azure Data Lake Analytics por meio da integração do Azure Monitor de acordo com um intervalo [de pesquisa](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
