[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar seu [banco de dadosMicrosoft Azure para métricas MariaDB](https://docs.microsoft.com/en-gb/azure/mariadb/) e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

New Relic coleta dados do serviço Azure banco de dados para MariaDB, que fornece um banco de dados como serviço totalmente gerenciado e pronto para empresas. MariaDB fornece desempenho previsível e escalabilidade para aplicativos usando ferramentas e plataforma de código aberto.

Usando o New Relic, você pode:

* Visualize o banco de dados do Azure para dados MariaDB em um painel pré-criado.
* Execute uma consulta personalizada e visualize os dados na [interface New Relic ](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga os procedimentos padrão para [ativar seu serviço do Azure no New Relic](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) usando a função genérica <DNT>**Reader**</DNT> .

## Configuração e polling [#polling]

New Relic consulta seu serviço de banco de dados do Azure de acordo com um intervalo [de pesquisa](/docs/integrations/microsoft-azure-integrations/getting-started/azure-integration-polling) padrão, que varia dependendo da integração. Para integração do banco de dados do Azure para PostgreSQL:

* Intervalo de sondagem: 5 minutos (frequência máxima de sondagem recomendada: 1 hora)
* Resolução: 1 ponto de dados por minuto

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

## Visualizar e usar dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data): Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; Azure &gt; (select an integration)**</DNT>.

No New Relic, os dados sobre um único banco de dados são anexados ao tipo de evento `AzureMariaDbServerSample` , com um valor de provedor de `AzureMariaDbServer`.
