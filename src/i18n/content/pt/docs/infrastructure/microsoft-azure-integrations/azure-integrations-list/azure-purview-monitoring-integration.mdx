[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para relatar sua métrica [Microsoft Azure Purview](https://azure.microsoft.com/en-us/products/purview/) e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

O Microsoft Purview fornece uma solução unificada de governança de dados para ajudar a gerenciar e governar seus dados locais, multicloud e de software como serviço (SaaS). Crie facilmente um mapa holístico e atualizado do seu cenário de dados com descoberta automatizada de dados, classificação de dados confidenciais e linhagem de dados de ponta a ponta. Permita que o consumidor de dados acesse um gerenciamento de dados valioso e confiável.

Usando o New Relic, você pode:

* Visualize os dados do Microsoft Purview em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta os seus serviços Microsoft Purview através da integração do Azure Monitor de acordo com um intervalo [de sondagem](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
