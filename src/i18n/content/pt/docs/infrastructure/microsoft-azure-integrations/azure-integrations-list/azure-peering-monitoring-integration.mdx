[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para reportar sua métrica [de peeringMicrosoft Azure ](https://learn.microsoft.com/en-in/azure/internet-peering/)e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

New Relic reúne dados métricos do Azure Monitor para o Azure Peering. Peering é a interconexão entre a rede global da Microsoft (AS8075) e sua rede com a finalidade de trocar tráfego de Internet de/para serviços online da Microsoft e serviços do Microsoft Azure. As operadoras ou provedores de serviço podem solicitar conexão com a Microsoft em qualquer um de nossos pontos de presença.

Usando o New Relic, você pode:

* Veja os dados do Azure Peering no painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta o seu Azure Peering através da integração do Azure Monitor de acordo com um intervalo [de sondagem](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; Azure &gt; (select an integration)**</DNT>.
