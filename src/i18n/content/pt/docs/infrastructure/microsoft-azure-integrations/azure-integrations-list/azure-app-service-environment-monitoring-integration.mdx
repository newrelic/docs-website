[A integração da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para relatar sua métrica [do ambiente de serviço de aplicativoMicrosoft Azure ](https://learn.microsoft.com/en-in/azure/app-service/environment)e outros dados para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

O Azure App Service Environment (ASE) é uma oferta de recurso premium do Azure App Service. Fornece uma instância de inquilino único do Serviço de Aplicações do Azure que funciona diretamente na sua própria rede virtual do Azure (VNet), proporcionando isolamento de rede e capacidades de escalabilidade melhoradas.

Usando o New Relic, você pode:

* Veja os dados dos ambientes de serviço de aplicativos do Azure em um painel pré-criado.
* Execute [uma consulta personalizada e visualize os dados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Crie condição do alerta para notificá-lo sobre alterações nos dados.

## Ativar integração [#activate]

Siga [o procedimento padrão de integração Azure Monitor ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor)para ativar seu serviço Azure no monitoramento de infraestrutura New Relic .

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

A New Relic consulta os seus serviços de Ambientes de Serviço de Aplicações através da integração do Azure Monitor de acordo com um intervalo [de sondagem](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) padrão.

## Encontre e use dados [#find-and-use]

Para [explorar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
