New RelicA [integração Azure da](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations) consulta seus serviços Azure de acordo com um intervalo de sondagem específico para a integração.

O intervalo de sondagem aplica-se a <DNT>**every**</DNT> entidade do Azure relacionada ao serviço integrado. Por exemplo, se você tiver treze instâncias do CosmosDB, cada uma das treze instâncias será pesquisada a cada cinco minutos.

## Ver dados de votação [#account-status]

Depois de [ativar uma integração do Azure](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations), New Relic inicia a pesquisa de dados do Azure e os torna acessíveis por meio de nossa interface. Você pode consultar os dados do Azure juntamente com dados adicionais importados de qualquer outro recurso New Relic . Você também pode visualizar os dados dashboard para uma integração específica ou em toda a sua conta.

Para visualizações de intervalo de sondagem, chamada de API e outros dados para sua integração no Azure:

1. Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure**
   </DNT>

   .

2. Para visualizar dados de uma integração específica: Selecione o link

   <DNT>
     **Dashboards**
   </DNT>

   da linha da integração.

## Intervalo de sondagem New Relic [#aws-integrations]
