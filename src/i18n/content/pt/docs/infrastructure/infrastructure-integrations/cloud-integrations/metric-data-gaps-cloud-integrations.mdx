## Causa

Algumas métricas não estão presentes na API usual do provedor de nuvem (CloudWatch, Stackdriver, Azure Monitor) e são buscadas na API do serviço. Cada provedor de serviço em nuvem possui uma API de serviço exclusiva que processa dados e interage com o serviço. Por exemplo, se uma métrica não estiver presente no AWS CloudWatch, o New Relic buscará a métrica na API do serviço AWS ECS.
