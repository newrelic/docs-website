Você pode instalar nosso agente [de infraestrutura](/docs/welcome-new-relic-infrastructure) usando [extensões do Microsoft Azure](https://docs.microsoft.com/en-us/azure/virtual-machines/extensions/overview).

<CollapserGroup>
  <Collapser
    id="what-is-extension"
    title="O que é uma extensão do Azure?"
  >
    As extensões da máquina virtual do Azure (VM) são pequenos aplicativos que fornecem configuração pós-implantação e tarefas de automação na VM do Azure. Por exemplo, se uma máquina virtual exigir instalação de software, proteção antivírus ou execução de um script, uma extensão de VM poderá ser usada. A CLI do Azure, o PowerShell, os modelos do Azure Resource Manager e o portal do Azure podem executar extensões de VM do Azure. As extensões podem ser agrupadas com uma nova implantação de VM ou executadas em qualquer sistema existente.
  </Collapser>
</CollapserGroup>

A New Relic usa essas extensões para facilitar a instalação do agente de infraestrutura. Isto elimina a necessidade de atualizar manualmente o nosso agente e, mais importante, dimensiona automaticamente o agente com os seus [conjuntos de escala do Azure](https://docs.microsoft.com/en-us/azure/virtual-machine-scale-sets/overview).

## Instalar infraestrutura usando extensões do Azure [#install-infra-azure-extensions]

Para instalar o agente usando a extensão Azure no [sistema operacional](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/) compatível.

1. Acesse qualquer VM do Azure, conjunto de dimensionamento do Azure ou Azure Service Fabric usando conjuntos de dimensionamento e selecione a guia <DNT>**Extensions**</DNT> .

   <img
     title="azure_ext01.png"
     alt="azure_ext01.png"
     src="/images/infrastructure_screenshot-crop_azure-extensions-tab.webp"
   />

2. Selecione a extensão <DNT>**New Relic infrastructure**</DNT> .

   <img
     title="azure_ext02.png"
     alt="azure_ext02.png"
     src="/images/infrastructure_screenshot-crop_azure-nr-infrastructure-extension.webp"
   />

3. Siga as instruções e adicione seu <InlinePopover type="licenseKey"/>.

   <img
     title="azure_ext03.png"
     alt="azure_ext03.png"
     src="/images/infrastructure_screenshot-crop_azure-nr-license-key.webp"
   />

   <Callout variant="tip">
     Se você automatizar a instalação, lembre-se de passar seu <InlinePopover type="licenseKey"/>com a variável `NR_LICENSE_KEY` . Você pode usar [a configuração por meio de variáveis de ambiente](/docs/infrastructure/install-infrastructure-agent/configuration/configure-infrastructure-agent/#precedence) (formato `NRIA_` ) para customizar qualquer configuração de agente.
   </Callout>
