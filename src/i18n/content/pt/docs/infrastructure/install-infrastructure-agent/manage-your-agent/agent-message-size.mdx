Desde [a versão 1.0.989](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes) do agente de infraestrutura, o tamanho máximo da mensagem é de 1MB (10^6 bytes), podendo incluir indistintamente integração de evento, métrica e inventário. Anteriormente, o tamanho máximo da mensagem era de 5 MB, embora os dados de inventário estivessem limitados a 3 MB.
