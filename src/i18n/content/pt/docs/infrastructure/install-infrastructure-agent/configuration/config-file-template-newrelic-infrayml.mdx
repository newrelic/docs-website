O agente de infraestrutura lê o arquivo `newrelic-infra.yml` para sua [configuração](/docs/infrastructure/install-configure-infrastructure/configuration/configure-infrastructure-agent). Você pode encontrar o modelo de configuração mais recente no [repositório infra-agente no GitHub](https://github.com/newrelic/infrastructure-agent/blob/master/assets/examples/infrastructure/newrelic-infra-template.yml.example).

<Callout variant="tip">
  Recomendamos [lintar](http://www.yamllint.com/) os arquivos de configuração YAML antes de usá-los para evitar problemas de formatação.
</Callout>

Para obter mais informações sobre cada configuração, consulte [Definições de configuração de infraestrutura](/docs/infrastructure/install-configure-manage-infrastructure/configuration/infrastructure-configuration-settings).

<Callout variant="important">
  Sempre [reinicie o agente](/docs/infrastructure-start-stop-restart-check-agent-status) ou o servidor web após alterar as configurações.
</Callout>
