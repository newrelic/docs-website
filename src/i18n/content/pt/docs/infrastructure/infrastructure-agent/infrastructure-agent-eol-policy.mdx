## Sistema operacional e plataforma de tempo de execução [#infra-os-runtime]

O agente é suportado em uma ampla variedade de ambientes. Consulte [Requisitos para o agente de infraestrutura](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/) para obter uma lista atualizada de sistemas operacionais, arquitetura e ambientes de tempo de execução suportados.

O agente de infraestrutura dá suporte a esses sistemas operacionais até o fim da vida útil de seus fabricantes. O agente pode ser compatível e pacotes podem estar disponíveis para sistemas EOL, mas o suporte não é garantido.
