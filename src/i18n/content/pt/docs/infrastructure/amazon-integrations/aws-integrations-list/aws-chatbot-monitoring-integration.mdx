[A integração da New Relic Infrastructure](/docs/infrastructure/introduction-infra-monitoring) agora inclui uma integração para AWS Chatbot, enviando seus dados métricos para a New Relic.

Este documento explica o recurso de integração, como ativá-lo e quais dados podem ser reportados.

## Recurso

Colete e envie dados de telemetria para a New Relic a partir do seu [Chatbot](https://aws.amazon.com/chatbot/) usando nossa integração. monitor seus serviços, consulte os dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração

Para habilitar essa integração, veja como [conectar os serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream).

<Callout variant="important">
  Para obter a métrica AWS Chatbot, altere sua região do Amazon CloudWatch para `US East (N. Virginia)`. Leia mais sobre isso na [documentação da AWS](https://docs.aws.amazon.com/chatbot/latest/adminguide/monitoring-cloudwatch.html).
</Callout>

## Encontre e use dados

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.chatbot`.
