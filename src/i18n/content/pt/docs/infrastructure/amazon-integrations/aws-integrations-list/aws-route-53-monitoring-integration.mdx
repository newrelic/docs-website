[A integração New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)inclui uma integração para relatar seus dados do Amazon Route 53 para New Relic. Este documento explica como ativar a integração e descreve os dados reportados.

## Recurso [#features]

[O Amazon Route 53](https://aws.amazon.com/route53/) ajuda você a registrar nomes de domínio, encaminhar o tráfego da Internet para os recursos apropriados e verificar a integridade dos seus recursos para garantir que seu aplicativo web esteja sempre funcionando.

Para verificações de integridade da AWS, o Amazon Route 53 determina se um conjunto de registros de recursos está íntegro com base em um dos seguintes critérios:

* Enviando periodicamente uma solicitação ao endpoint especificado na verificação de integridade
* Ao agregar o status de um grupo específico de verificações de integridade (verificações de integridade calculadas)
* Determinando o estado atual de um alarme do CloudWatch (métricas das verificações de integridade do CloudWatch)

Os dados relatados à New Relic incluem tempo de conexão, verificações de integridade, tempo até o primeiro byte e outras [métricas](#metrics). Os dados do Route 53 estão disponíveis em <InlinePopover type="dashboards"/>pré-construído e você também pode criar [consultas e gráficos personalizados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).

## Ativar integração [#activate]

Para habilitar essa integração, siga os procedimentos padrão para [conectar os serviços da AWS ao New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informações [de pesquisa](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) padrão para a integração do Amazon Route 53:

* Intervalo de sondagem New Relic : 5 minutos
* Intervalo de dados do Amazon CloudWatch: 1 minuto

## Encontre e use dados [#find-data]

Para encontrar seus dados de integração, acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > AWS**</DNT> e selecione um dos links de integração do Route 53.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `HealthCheckSample` , com um valor `provider` de `Route53HealthCheck`.

Para saber mais sobre como usar seus dados, consulte [Compreender os dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).

## Dados métricos [#metrics]
