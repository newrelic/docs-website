[A integração da infraestrutura da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração AWS Identity and Access Management (IAM) para relatar seus dados IAM para a New Relic. Este documento explica o recurso de integração, como ativá-lo e quais dados podem ser informados.

## Recurso [#features]

[O gerenciamento de identidade e acesso (IAM)](https://aws.amazon.com/iam/) da Amazon permite que você controle com segurança o acesso aos serviços da AWS e aos recursos do seu usuário. Usando o IAM, você pode criar e gerenciar usuários e grupos AWS e usar permissões para permitir e negar acesso aos recursos AWS .

A integração de monitoramento IAM da New Relic permite capturar o estado de políticas, usuários, grupos e funções em momentos específicos. Os dados do IAM também estão disponíveis [para análise](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure#insights).

## Ativar integração [#activate]

Para habilitar essa integração, siga os procedimentos padrão para [conectar os serviços da AWS ao New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informações [de pesquisa padrão](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) para a integração do AWS IAM:

* Intervalo de sondagem New Relic : 5 minutos

## Encontre e use dados [#find-data]

Para encontrar os dados desta integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; AWS**</DNT> e selecione um dos links de integração do IAM.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `IamAccountSummarySample` .

## Dados métricos [#metrics]

A integração IAM não captura nenhuma métrica de desempenho no sentido tradicional da palavra. Mas a integração monitor o estado das políticas de serviço IAM, usuário, grupos e funções em momentos específicos e usa isso para criar estas métricas baseadas em contagem:

* Certificados por conta
* Grupos por conta
* Dispositivos MFA por conta
* Funções por conta
* Usuário por conta
