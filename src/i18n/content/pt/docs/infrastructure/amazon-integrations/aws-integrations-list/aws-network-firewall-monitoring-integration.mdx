As integrações New Relic Infrastructure agora incluem uma integração que envia suas métricas firewall da rede AWS para a New Relic. Este documento explica o recurso da integração, como ativá-la e quais dados estão disponíveis para reporte.

## Recurso [#features]

A integração métrica firewall da rede AWS coleta e envia dados de telemetria para o New Relic a partir firewall da sua rede AWS. Você pode monitor seus serviços da AWS, consultar dados recebidos e construir um dashboard para observar tudo rapidamente.

## Ativar integração [#activate-ingration]

Para habilitar essa integração, siga as instruções em nossa [documentação de integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-data]

Para encontrar a métrica da sua integração:

1. Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

2. Selecione

   <DNT>
     **Metrics and events**
   </DNT>

3. Filtrar por `aws.networkfirewall`.
