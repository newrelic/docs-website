[A integração da New Relic Infrastructure](/docs/infrastructure/introduction-infra-monitoring) agora inclui uma integração para AWS IoT Twinmaker que envia seus dados métricos para a New Relic. Este documento explica o recurso de integração, como ativá-lo e quais dados ele pode reportar.

## Recurso

Colete e envie dados de telemetria para o New Relic a partir do seu [WorkMail](https://aws.amazon.com/iot-twinmaker/) usando nossa integração. Isso permite monitor seus serviços, consultar dados recebidos e criar um painel para observar tudo rapidamente.

## Ative e use a integração

Para habilitar essa integração, veja como [conectar os serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/). Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.IoTTwinMaker`.
