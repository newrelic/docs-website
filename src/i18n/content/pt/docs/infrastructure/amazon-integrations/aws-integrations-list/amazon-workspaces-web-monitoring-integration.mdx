A integração New Relic Infrastructure agora inclui uma integração para AWS WorkSpaces Web e pode enviar seus dados métricos para New Relic.

Este documento explica o recurso de integração, como ativá-lo e quais dados estão disponíveis para reporte.

## Recurso

Usando nossa integração, coletaremos e enviaremos dados de telemetria para o New Relic a partir do seu [WorkSpaces Web](https://aws.amazon.com/workspaces/web). monitor seus serviços, consulte os dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração

Para habilitar essa integração, siga as instruções em nossa [documentação de integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados

Para encontrar a métrica da sua integração:

1. Acesse [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)

2. Selecione

   <DNT>
     **Metrics and events**
   </DNT>

3. Filtrar por `aws.workspacesweb`.
