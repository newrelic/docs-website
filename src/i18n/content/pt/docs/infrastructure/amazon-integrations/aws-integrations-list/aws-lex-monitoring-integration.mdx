[A integração New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)inclui uma integração do Amazon Lex para relatar seus dados Lex para a New Relic. Aqui explicamos como ativar nossa integração e quais dados podem ser reportados.

Habilite a [integração do AWS CloudWatch Metric Streams ](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-metric-stream/)para monitor todas as métricas do CloudWatch de seus serviços da AWS, incluindo namespace personalizado. A integração individual não é mais nossa opção recomendada.

## Recurso [#features]

[Amazon Lex](https://aws.amazon.com/lex/) (Lex) é um serviço da AWS para construção de interfaces conversacionais em aplicativos usando voz e texto. Com a integração Lex da New Relic, você poderá monitor o número de solicitações, os canais usados para conversação do Bot e muito mais. Você poderá criar gráficos personalizados de seus dados e [criar condições de alerta](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information) com base em alterações nos dados do Lex ou na configuração.

## Ativar integração [#activate]

Para habilitar essa integração, veja como [conectar os serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream).

## Encontre e use dados [#find-data]

Para encontrar os dados dessa integração, acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > AWS**</DNT> e selecione um dos links de integração do Amazon Lex.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando a tabela <DNT>**Metric**</DNT> e `namespace='AWS/Lex'`.

Para obter mais informações sobre como localizar e usar dados de integração, consulte como [entender os dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
