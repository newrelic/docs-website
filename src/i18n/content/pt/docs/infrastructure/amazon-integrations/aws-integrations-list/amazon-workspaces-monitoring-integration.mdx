New Relic integra-se com Amazon Web Services (AWS) para relatar seus dados métricos [do Amazon Workspaces](https://aws.amazon.com/workspaces-family/) e outros dados ao New Relic.

Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso

Colete e envie dados de telemetria para o New Relic a partir dos seus WorkSpaces usando nossa integração. monitore seus serviços, consulte dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração

Essa integração está disponível por meio do CloudWatch MetricStreams (recomendado).

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.workspaces`.
