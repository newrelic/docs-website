[A integração New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)inclui uma integração para [o Amazon Transcribe](https://aws.amazon.com/transcribe) que reporta sua métrica do Transcribe para a New Relic.

## Recurso [#features]

Com essa integração, colete e envie dados de telemetria para o New Relic a partir do seu serviço [Amazon Transcribe](https://aws.amazon.com/transcribe/) . monitor seus serviços, consulte os dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração

Para habilitar essa integração, veja como [conectar os serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream).

## Encontre e use dados [#find-use-data]

Os dados desta integração são anexados ao tipo de dados `Metric` . Para encontrar esses dados na interface: Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> e filtre por `aws.transcribe`.
