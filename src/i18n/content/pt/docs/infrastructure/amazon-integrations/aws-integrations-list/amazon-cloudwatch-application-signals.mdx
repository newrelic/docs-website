New Relic integra-se com Amazon Web Services (AWS) para monitor [sinais métricos do aplicativo CloudWatch](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Agent-Application_Signals.html) e outros dados do New Relic.

Este documento explica como ativar a integração e descreve os dados relatados.

## Ative a integração [#activate]

Essa integração está disponível por meio do CloudWatch MetricStreams.

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-use-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.applicationsignals`.
