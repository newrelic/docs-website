[A integração New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)agora inclui uma integração para AWS Backup, enviando seus dados métricos para New Relic.

Este documento explica o recurso de integração, como ativá-lo e quais dados você pode reportar.

## Recurso [#features]

Colete e envie dados de telemetria para o New Relic a partir do seu [AWS Backup](https://aws.amazon.com/backup/) usando nossa integração. Verifique seus serviços, consulte os dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração [#activate-integration]

Para habilitar essa integração, veja como [conectar os serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-use-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Metrics & Events**</DNT> e filtre por `aws.backup`.

## Dados métricos [#metric-data]
