New Relic integra-se com Amazon Web Services (AWS) para monitor [AWS Backup](https://aws.amazon.com/backup/) Métrica e outros dados no New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso [#features]

Colete e envie dados de telemetria para o New Relic a partir dos seus serviços de backup da AWS usando nossa integração. Verifique seus serviços, consulte dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração [#activate-integration]

Essa integração está disponível por meio do CloudWatch MetricStreams.

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-use-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics &amp;amp; Events**</DNT> e filtre por `aws.backup`.

Além disso, você pode encontrar a entidade associada ao AWS Backup acessando [one.newrelic.com](https://one.newrelic.com/all-entities) e pesquisando por `backup`.
