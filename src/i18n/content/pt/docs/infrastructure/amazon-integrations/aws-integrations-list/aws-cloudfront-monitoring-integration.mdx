[A integração da infraestrutura da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para relatar dados de serviço do Amazon CloudFront para a New Relic. Este documento explica como ativar essa integração e descreve os dados que podem ser relatados.

## Recurso [#features]

[Amazon CloudFront](http://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/Introduction.html) é um serviço web da Amazon que acelera a distribuição do seu conteúdo web. Com a integração New Relic CloudFront, você pode rastrear dados do CloudFront no New Relic, incluindo taxas de erros, contagens de solicitações e bytes carregados/baixados. Você pode acompanhar a configuração do CloudFront e ver como as alterações na configuração afetam o desempenho. Você também pode criar consultas personalizadas de seus dados de integração do CloudFront e gráficos personalizados.

## Ativar integração [#activate]

Para habilitar essa integração, siga os procedimentos padrão para [conectar os serviços da AWS ao New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informações [de pesquisa padrão](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) para a integração do Amazon CloudFront:

* Intervalo de sondagem New Relic : 5 minutos
* Intervalo de dados do Amazon CloudWatch: [1 minuto](http://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/monitoring-using-cloudwatch.html), com atraso de até 1 minuto porque o CloudFront às vezes relata dados parciais

Se você estiver usando o Lambda@Edge para personalizar o conteúdo que o CloudFront entrega para executar a função do Lambda em locais AWS mais próximos de seus clientes, você pode ativar o filtro <DNT>**Collect Lambda@Edge data**</DNT> para obter metadados do local de execução do Lambda.

## Encontre e use dados [#find-data]

Para encontrar seus dados de integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; AWS**</DNT> e selecione um dos links de integração do CloudFront.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `LoadBalancerSample` , com um valor `provider` de `CloudFrontDistribution`.

Para saber mais sobre como usar seus dados, consulte [Compreender e usar dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
