New Relic integra-se com Amazon Web Services (AWS) para relatar suas métricas [do Amazon AppStream](https://aws.amazon.com/appstream2/) e outros dados ao New Relic.

Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso [#features]

Colete e envie dados de telemetria para o New Relic a partir dos seus serviços AppStream usando nossa integração. monitore seus serviços, consulte dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração [#activate]

Essa integração está disponível por meio do CloudWatch Metric Streams.

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados \[find-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.appstream`.

Além disso, você pode encontrar a entidade associada ao AppStream acessando [one.newrelic.com](https://one.newrelic.com/all-entities) e pesquisando por `appstream`.
