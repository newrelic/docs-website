New Relic integra-se com Amazon Web Services (AWS) para relatar suas métricas [AWS Transfer Family](https://aws.amazon.com/aws-transfer-family/) e outros dados para o New Relic.

Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso [#features]

Colete e envie dados de telemetria para o New Relic da sua família AWSTransfer usando nossa integração. monitore seus serviços, consulte dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração [#activate-integration]

Essa integração está disponível por meio do CloudWatch Metric Streams.

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-use-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics &amp; Events**</DNT> e filtre por `aws.transfer`.
