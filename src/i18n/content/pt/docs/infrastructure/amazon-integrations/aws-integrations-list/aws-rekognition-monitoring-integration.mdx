[A integração da New Relic Infrastructure](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração do Amazon Rekognition para relatar seus dados do Rekognition para a New Relic. Aqui, explicamos como ativar nossa integração e quais dados podem ser relatados.

Habilite a [integração do AWS CloudWatch Metric Streams ](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-metric-stream/)para monitor todas as métricas do CloudWatch de seus serviços da AWS, incluindo namespace personalizado. A integração individual não é mais nossa opção recomendada.

## Recurso [#features]

[Amazon Rekognition](https://aws.amazon.com/rekognition/) (Rekognition) é um serviço da AWS para automatizar e reduzir o custo de reconhecimento de imagem e análise de vídeo com aprendizado de máquina. Com a integração Rekognition da New Relic, você poderá monitor o número de solicitações, o tempo de resposta, a contagem de rostos detectados e muito mais. Você poderá criar gráficos personalizados de seus dados e [criar condições de alerta](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information) com base em alterações nos dados do Rekognition ou na configuração.

## Ativar integração [#activate]

Para habilitar essa integração, veja como [conectar os serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream).

## Encontre e use dados [#find-data]

Os dados desta integração estão vinculados ao tipo de dados métricos. Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics &amp; Events**</DNT> e filtre por `aws.rekognition` para encontrar esses dados na interface.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando a tabela <DNT>**Metric**</DNT> e `namespace='AWS/Rekognition'`.
