[A integraçãoNew Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)agora inclui uma integração para <DNT>AWS Textract</DNT>, enviando seus dados métricos para New Relic.

Este documento explica o recurso de integração, como ativá-lo e quais dados podem ser reportados.

## Recurso [#features]

Colete e envie dados de telemetria para o New Relic a partir do seu [aplicativo Textract](https://aws.amazon.com/textract/) usando nossa integração. monitor seus serviços, consulte os dados recebidos e crie um painel para observar tudo rapidamente.

## Dados do relatório [#report-data]

Temos duas opções para relatar esses dados. O caminho recomendado é configurar nossa [integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream). Como alternativa, você pode usar nossa [integração mais antiga com a Amazon, que depende de polling](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations).

## Encontre e use dados [#find-use-data]

Os dados desta integração são anexados ao tipo de dados `Metric` . Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT> e filtre por `aws.textract`.
