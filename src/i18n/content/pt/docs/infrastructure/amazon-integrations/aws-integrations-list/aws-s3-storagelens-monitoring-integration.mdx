[A integração da infraestrutura da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para relatar seus dados do Amazon S3 para a New Relic. Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso [#features]

O [Amazon Simple Storage Service](https://aws.amazon.com/s3) (Amazon S3) oferece aos desenvolvedores e equipes de TI armazenamento em nuvem seguro, durável e altamente escalável.

Com a integração de lentes de armazenamento Amazon S3 da New Relic, os dados relatados incluem tamanho do bucket S3, contagens de objetos do bucket, solicitações GET, solicitações POST e outras [métricas](#metrics). Os dados do S3 estão disponíveis em <InlinePopover type="dashboards" />pré-construídos e você também pode criar [consultas e gráficos personalizados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights). Você também pode criar condição do alerta para notificá-lo sobre alterações nos dados do S3.

## Ativar integração [#activate]

<Callout variant="important">
  As métricas de [solicitação e transferência de dados](https://docs.aws.amazon.com/AmazonS3/latest/user-guide/configure-metrics.html) são premium e pagas separadamente pela AWS. Para obter informações sobre preços do CloudWatch, consulte a documentação [de monitoramento aprimorado do S3](https://docs.aws.amazon.com/AmazonS3/latest/dev/cloudwatch-monitoring.html) da Amazon.
</Callout>

Para habilitar essa integração, siga os procedimentos padrão para [conectar os serviços da AWS ao New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## Encontre e use dados [#find-data]

Para encontrar seus dados de integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; AWS**</DNT> e selecione um dos links de integração do S3.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) no explorador métrico pesquisando por `storagelens` na barra de pesquisa.

Para saber mais sobre como usar seus dados, consulte [Compreender os dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).

## Dados métricos [#metrics]

Esta integração coleta métricas de lentes de armazenamento do Amazon S3. Para obter mais detalhes sobre essas métricas, consulte [a documentação do S3 Storage Lens da Amazon](https://docs.aws.amazon.com/AmazonS3/latest/userguide/storage_lens_basics_metrics_recommendations.html).
