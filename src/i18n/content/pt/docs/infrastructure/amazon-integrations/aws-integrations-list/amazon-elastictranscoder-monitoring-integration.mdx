[A integração New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)inclui uma integração para AWS Elastic Transcoder, que envia sua métrica para a New Relic.

## Recurso [#features]

Use nossa integração para coletar e enviar dados de telemetria para o New Relic a partir do seu [serviço Amazon Elastic Transcoder](https://aws.amazon.com/elastictranscoder/). Você pode monitor seus serviços, consultar dados recebidos e construir um painel para observar tudo rapidamente.

## Iniciar [#get-started]

Para habilitar essa integração, consulte [Conectar serviços da AWS ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup).

Depois que a integração estiver configurada, você poderá encontrar seus dados acessando <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> e filtrando por `aws.elastictranscoder`.
