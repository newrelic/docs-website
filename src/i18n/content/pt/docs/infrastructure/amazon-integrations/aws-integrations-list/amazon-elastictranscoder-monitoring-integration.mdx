New Relic integra-se com Amazon Web Services (AWS) para relatar seus dados métricos [do Amazon Elastic Transcoder](https://aws.amazon.com/elastictranscoder/) e outros dados ao New Relic.

Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso [#features]

Colete e envie dados de telemetria para o New Relic a partir dos seus serviços do Amazon Elastic Transcoder usando nossa integração. monitore seus serviços, consulte dados recebidos e crie um painel para observar tudo rapidamente.

## Iniciar [#get-started]

Essa integração está disponível por meio do CloudWatch Metric Streams.

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT> e filtre por `aws.elastictranscoder`.

Além disso, você pode encontrar a entidade associada ao Amazon Elastic Transcoder acessando [one.newrelic.com](https://one.newrelic.com/all-entities) e pesquisando por `elastictranscoder`.
