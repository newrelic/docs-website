<Callout variant="tip">
  Para usar essa integração, você deve ter um [plano de suporte](https://aws.amazon.com/premiumsupport/plans/) AWS Business ou Enterprise.
</Callout>

O AWS [Trusted Advisor](https://aws.amazon.com/premiumsupport/technology/trusted-advisor/) é uma ferramenta on-line que fornece orientação em tempo real para ajudar você a seguir as práticas recomendadas da AWS para provisionar seus recursos.

[A integração da infraestrutura da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para relatar seus dados de limites de serviço para produtos da New Relic. Este documento explica como ativar essa integração e descreve os dados que podem ser capturados.

## Ativar integração [#activate]

Para habilitar essa integração, siga os procedimentos padrão para [conectar os serviços da AWS ao New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

<Callout variant="important">
  Esta integração requer permissões de acesso adicionais não cobertas pelas políticas gerenciadas padrão. Você pode saber mais na página [de integração e políticas gerenciadas](/docs/integrations/amazon-integrations/getting-started/integrations-managed-policies#recommended) .
</Callout>

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informações [de pesquisa padrão](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) para a integração do AWS Trusted Advisor:

* Intervalo de sondagem New Relic : 1 hora

Para ter dados atualizados, a New Relic envia programaticamente solicitações de atualização para a AWS. As verificações do Trusted Advisor são [atualizadas automaticamente pela AWS semanalmente](https://aws.amazon.com/premiumsupport/faqs/#TArefresh) apenas para clientes com planos AWS Business ou Enterprise Support.

## Visualizar e usar dados [#find-data]

Para [visualizar e usar seus dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; AWS**</DNT> e selecione o link de integração do AWS Trusted Advisor.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `TrustedAdvisorSample` dentro de uma categoria de dados:

* Limites de serviço: aparece no New Relic com `provider` valor `ServiceLimits`.

## Dados métricos

Há uma categoria `Trusted Advisor` de dados: `ServiceLimits`.
