[A integração da infraestrutura da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração do Amazon EBS para relatar seus dados do EBS para a New Relic. Aqui explicamos como ativar nossa integração e quais dados podem ser reportados.

## Recurso

[O Amazon Elastic Block Store](https://aws.amazon.com/ebs/) (EBS) fornece volumes de armazenamento em nível de bloco para suas instâncias do Amazon EC2. Com a integração EBS da New Relic, você poderá monitor gravações por volume, contagens de volume, opções de configuração e muito mais. Você poderá criar gráficos personalizados de seus dados e [criar condições de alerta](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information) com base em alterações nos dados do EBS ou na configuração.

## Ativar integração [#activate]

Para habilitar essa integração, siga os procedimentos padrão para [conectar os serviços da AWS ao New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## Configuração e polling [#polling]

Você pode alterar a frequência de pesquisa e filtrar dados usando [opções de configuração](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Informações [de pesquisa](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) padrão para a integração do Amazon EBS:

* Intervalo de sondagem New Relic : 15 minutos
* Intervalo de dados do Amazon CloudWatch: [1 minuto ou 5 minutos](http://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/ebs-metricscollected.html), dependendo do tipo de volume

## Encontre e use dados [#find-data]

Para encontrar os dados desta integração, acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; AWS**</DNT> e selecione um dos links de integração do Amazon EBS.

Você pode [consultar e explorar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) usando o [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `BlockDeviceSample` , com um valor `provider` de `EbsVolume`.

Para obter mais informações sobre como localizar e usar dados de integração, consulte [Compreender os dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
