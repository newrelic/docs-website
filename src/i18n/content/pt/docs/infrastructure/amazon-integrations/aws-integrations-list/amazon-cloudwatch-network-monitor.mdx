New Relic integra-se com Amazon Web Services (AWS) para monitor [AWS Backup](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/nw-monitor-create-console.html) Métrica e outros dados no New Relic.

Este documento explica como ativar a integração e descreve os dados relatados.

## Recurso [#features]

Colete e envie dados de telemetria para New Relic a partir dos seus serviços de monitoramento de rede CloudWatch usando nossa integração. monitore seus serviços, consulte dados recebidos e crie um painel para observar tudo rapidamente.

## Ativar integração [#activate]

Essa integração está disponível por meio do CloudWatch MetricStreams.

Para habilitar essa integração, veja como [conectar serviços da AWS ao New Relic via CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-data]

Para encontrar a métrica da sua integração, acesse <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.networkmonitor`.
