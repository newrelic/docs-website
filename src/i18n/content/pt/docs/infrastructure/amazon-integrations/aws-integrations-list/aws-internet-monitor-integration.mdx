Nossa integração de AWS Internet CloudWatch relata os dados Internet do CloudWatch monitor monitor para a New Relic plataforma .

## Recurso [#features]

Com esta integração, você pode coletar e enviar dados de telemetria para o New Relic a partir do seu [monitorde Internet CloudWatch](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-InternetMonitor.html). Uma vez integrado, você pode monitor seus serviços, consultar dados recebidos, monitor logs e usar um painel pré-construído para observar tudo rapidamente.

## Obtenha dados métricos [#get-metrics-data]

Temos duas opções para relatar esses dados. O caminho recomendado é configurar nossa [integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream). Como alternativa, você pode usar nossa [integração mais antiga com a Amazon, que depende de polling](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations).

## Obtenha dados de registro [#get-logs-data]

Os dados de log podem ser relatados à New Relic usando nosso [Lambda de ingestãolog ](/docs/logs/forward-logs/aws-lambda-sending-cloudwatch-logs). Certifique-se de [adicionar gatilhos](/docs/logs/forward-logs/aws-lambda-sending-cloudwatch-logs/#create-trigger) ao Lambda e selecionar o grupo log correspondente a `internet-monitor` nos níveis `country`, `city`, `subdivision` e `metro`.

## Encontre e use dados [#find-use-data]

Os dados métricos desta integração são anexados ao tipo de dados `Metric` . Para ver seus dados: vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> e filtre por `aws.internetmonitor`.
