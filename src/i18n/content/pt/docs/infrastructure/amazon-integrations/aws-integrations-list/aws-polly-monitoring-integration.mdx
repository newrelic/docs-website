[A integração de infraestrutura da New Relic](/docs/infrastructure/introduction-infra-monitoring) inclui uma integração para <DNT>Amazon Polly</DNT> que reporta sua métrica Polly para a New Relic.

## Recurso [#features]

Com essa integração, colete e envie dados de telemetria para o New Relic a partir do seu serviço [<DNT>Amazon Polly</DNT>](https://docs.aws.amazon.com/polly). monitor seus serviços, consulte os dados recebidos e crie um dashboard para observar tudo rapidamente.

## Dados do relatório [#report-data]

Temos duas opções para relatar esses dados. O caminho recomendado é configurar nossa [integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream). Como alternativa, você pode usar nossa [integração mais antiga com a Amazon, que depende de polling](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations).

## Encontre e use dados [#find-use-data]

Os dados desta integração são anexados ao tipo de dados `Metric` . Para encontrar esses dados na interface: Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> e filtre por `aws.polly`.
