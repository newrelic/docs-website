Nossa integração NAT Gateway reporta dados do seu serviço NAT Gateway para sua conta New Relic usando AWS CloudWatch Metric Streams.

## Recurso [#features]

Com essa integração, você pode ver os dados de telemetria [do gateway NAT](https://docs.aws.amazon.com/vpc/latest/userguide/vpc-nat-gateway.html) no New Relic. Você pode monitor seus serviços, consultar dados recebidos e construir um painel para observar tudo rapidamente.

## Ativar AWS CloudWatch Metric Streams [#activate]

Para habilitar o AWS CloudWatch Metric Streams, siga [as instruções para conectar o AWS CloudWatch Metric Streams ao New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/).

<Callout variant="tip">
  Se você já tem AWS CloudWatch Metric Streams habilitado, você já deve ver a métrica no New Relic! Não há necessidade de fazer nenhuma alteração.
</Callout>

## Encontre e use dados [#find-use-data]

Os dados desta integração estão anexados ao [nosso tipo de dados `Metric` ](/docs/data-apis/understand-data/new-relic-data-types/#dimensional-metrics).

Uma maneira de encontrar seus dados: acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> e filtre por `aws.natgateway`.
