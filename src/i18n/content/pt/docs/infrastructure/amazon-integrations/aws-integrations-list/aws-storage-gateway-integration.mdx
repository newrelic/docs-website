As integrações New Relic Infrastructure agora incluem uma integração que envia sua métrica do AWS Storage Gateway para a New Relic. A integração métrica do AWS Storage Gateway coleta e envia dados de telemetria para o New Relic a partir do AWS Storage Gateway. Você pode monitor seus serviços da AWS, consultar dados recebidos e construir um painel para observar tudo rapidamente.

## Ativar integração [#activate-ingration]

Para habilitar essa integração, siga as instruções em nossa [documentação de integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados [#find-data]

Para encontrar a métrica da sua integração:

1. Acesse **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
2. Selecione **Metrics and events**
3. Filtrar por `aws.storagegateway`.
