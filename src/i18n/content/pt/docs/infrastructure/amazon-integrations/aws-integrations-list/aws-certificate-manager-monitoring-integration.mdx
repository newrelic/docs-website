New Relic Infrastructure inclui agora uma integração para AWS Certificate Manager que permite o envio dos seus dados métricos para New Relic. Este documento explica o recurso de integração, como ativá-lo e quais dados estão disponíveis para reporte.

## Recurso

Usando nossa integração, coletamos e enviamos dados de telemetria para o New Relic a partir do seu [AWS Certificate Manager](https://aws.amazon.com/certificate-manager/). Você pode monitor seus serviços, consultar dados recebidos e construir um painel para observar tudo rapidamente.

## Ativar integração

Para habilitar essa integração, siga as instruções em nossa [documentação de integração do Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Encontre e use dados

Para encontrar a métrica da sua integração:

1. Acesse [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)

2. Selecione

   <DNT>
     **Metrics and events**
   </DNT>

3. Filtrar por `aws.certificatemanager`
