## Problema

Ao conectar [a AWS a uma integração de infraestrutura](/docs/integrations/amazon-integrations/get-started/connect-aws-services-infrastructure#connect), durante a fase <DNT>**Review**</DNT> , você recebe o erro da AWS `Invalid principal in policy`.

## Solução

Verifique o tipo de integração que deseja instalar. New Relic pode obter dados de monitoramento de serviços e endpoints localizados em todas as [regiões AWS ,](https://docs.aws.amazon.com/general/latest/gr/rande.html) exceto nas regiões da China.

Você verá este erro ao tentar configurar a integração a partir de uma conta que usa a China.
