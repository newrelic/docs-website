[A desinstalação do agente de infraestrutura](/docs/infrastructure/new-relic-infrastructure/installation/uninstall-infrastructure-agent-or-integrations) não desinstala automaticamente nenhuma [integração de infraestrutura](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

Para desinstalar [o AWS Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/) ou qualquer outra integração AWS , siga as etapas abaixo.

      <td id="disable-aws-integrations">
        Desativar integração AWS CloudWatch Metric Streams
      </td>

      <td>
        Para desabilitar a integração AWS CloudWatch Metric Streams :

        1. No Console AWS , remova todos os recursos necessários que foram criados durante a configuração da integração AWS , incluindo quaisquer fluxos AWS CloudWatch, o endpoint AWS Kinesis Data Firehouse e o bucket de backup AWS S3. Se você usou o modelo AWS CloudFormation, siga as etapas para [excluir uma pilha no console do AWS CloudFormation](https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/cfn-console-delete-stack.html).

        2. Conclua a etapa 1 para cada região ativa AWS onde você conectou o AWS CloudWatch Metric Streams.

        3. Em

           <DoNotTranslate>**Infrastructure > AWS > AWS Status Dashboard**</DoNotTranslate>

           , certifique-se de que nenhuma nova atualização de métrica esteja sendo processada no New Relic.

        4. Conclua as etapas para [desconectar a conta da AWS](#unlink-aws).
      </td>
    </tr>

    <tr>
      <td id="disable-aws-integrations">
        Desative uma ou mais integração de pesquisa AWS
      </td>

      <td>
        Para desabilitar serviços enquanto mantém sua conta AWS vinculada ao New Relic:

        1. Em

           <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure**</DoNotTranslate>

           , selecione

           <DoNotTranslate>**AWS > Manage services**</DoNotTranslate>

           .

        2. Na página

           <DoNotTranslate>**Edit AWS account**</DoNotTranslate>

           , desmarque a caixa de seleção de cada serviço ativo que você deseja desativar.

        3. Salve suas alterações.
      </td>
    </tr>

    <tr>
      <td id="unlink-aws">
        Desconectar conta AWS
      </td>

      <td>
        Para desconectar completamente sua conta AWS da New Relic, você precisa desvincular sua conta AWS. Isso desativa todas as integrações New Relic associadas a essa conta AWS .

        1. Vá para <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > AWS > Manage services**</DoNotTranslate>.

        2. Na página <DoNotTranslate>**Edit AWS account**</DoNotTranslate> , selecione <DoNotTranslate>**Unlink this account**</DoNotTranslate>.

        3. Salve suas alterações.

        4. Faça login na AWS e selecione [<DoNotTranslate>**Services > IAM > Roles**</DoNotTranslate>](https://console.aws.amazon.com/iam/home#roles).

        5. Marque a caixa de seleção da função que deseja excluir e selecione <DoNotTranslate>**Role Actions > Delete Role**</DoNotTranslate>.

           Desvincular sua conta AWS desativará a relação de confiança configurada por meio de seu ARN.
      </td>
    </tr>
