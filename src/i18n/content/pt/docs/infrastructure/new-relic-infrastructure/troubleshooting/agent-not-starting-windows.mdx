## Problema [#problem]

Você instalou o agente New Relic Infrastructure em um servidor Windows e nenhum dado está sendo exibido na interface. Ao tentar iniciar o serviço manualmente a partir de <DNT>**Windows Services**</DNT> , ele falha com um erro como:

```
Windows could not start the New Relic infrastructure agent service on Local Computer
Error 1067: The process terminated unexpectedly
```

## Solução [#solution]

Um antivírus ou produto de segurança pode estar impedindo que o agente New Relic Infrastructure seja executado como um serviço. Para validar se o agente se comporta sem problemas, execute o arquivo `newrelic-infra.exe` na linha de comando como administrador e confirme se o host está relatando os dados conforme esperado. Se for esse o caso, trabalhe com sua equipe de segurança para ver se o serviço precisa ser adicionado à sua lista de permissões ou configurado com parâmetros adicionais.

<Callout variant="important">
  Para ler sobre a localização do agente e como modificá-la, consulte [como configurar o diretório do agente](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-manual-install-infrastructure-agent-windows/#agent-directory).
</Callout>
