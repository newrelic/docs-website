O Collectd [integração](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) monitora e reporta métricas de dispositivos configurados [pelo Collectd](https://collectd.org/wiki/index.php/Main_Page)para o New Relic.

Para começar, consulte o [README do Collectd integração no host](https://github.com/newrelic/nri-collectd) no GitHub.

<Callout variant="tip">
  Esta integração é lançada como [código aberto no Github](https://github.com/newrelic/nri-collectd). Um log de alterações também está disponível para as atualizações mais recentes. O suporte para esta integração está disponível com uma [assinatura do Expert Services](https://www.newrelic.com/expertservices).
</Callout>
