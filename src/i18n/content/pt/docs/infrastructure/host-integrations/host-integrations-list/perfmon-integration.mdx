A integração PerfMon fornece resultados de consultas e eventos do [monitorde desempenho da Microsoft](https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc749154(v%3dws.11)) para uma saída padrão (stdout) que pode ser consumida pelo New Relic.

Para começar, consulte o [README de integração no host do Windows PerfMon/WMI](https://github.com/newrelic/nri-perfmon) no GitHub.

<Callout variant="tip">
  Esta integração é lançada como [código aberto no GitHub](https://github.com/newrelic/nri-perfmon). Um log de alterações também está disponível para as atualizações mais recentes. O suporte para esta integração está disponível com uma [assinatura do Expert Services](https://www.newrelic.com/expertservices).
</Callout>
