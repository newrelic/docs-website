Nossa [integração](/docs/integrations/host-integrations/getting-started/introduction-host-integrations) NFS coleta e envia inventário e métricas do seu serviço NFS para nossa plataforma, onde você pode ver os dados de uso do disco para encontrar mais facilmente a origem de quaisquer problemas.

Conclua as etapas a seguir para configurar a integração NFS:

<Steps>
  <Step>
    ## Verifique a compatibilidade e os requisitos [#req]

    Nossa integração é compatível com NFS versão 3 e superior.
  </Step>

  <Step>
    ## Instalar o agente de infraestrutura

    Para usar a integração NFS, primeiro é necessário [instalar o agente de infraestrutura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent-new-relic/) no mesmo host.
  </Step>

  <Step>
    ## Instalar e ativar [#install]

    A integração NFS já está ativada e instalada por padrão com o agente de infraestrutura. Não há requisitos de instalação ou ativação, pois funciona imediatamente.
  </Step>

  <Step>
    ## Encontre e use dados [#find-and-use]

    Para encontrar seus dados de integração:

    1. Vá para

       <DNT>
         **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Third-party services**
       </DNT>

    2. Pesquise por

       <DNT>
         **NFS integration**
       </DNT>

       .

    3. Clique no dashboard NFS.

       Ao [consultar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data), os dados NFS são anexados ao [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `NFSSample` .

       Para saber mais sobre como encontrar e usar seus dados, consulte [Compreender e usar dados de integração](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
  </Step>
</Steps>

## Desativando a integração [#disable]

Defina `metrics_nfs_sample_rate` como `-1` no [arquivo de configuração](/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings/#metrics-nfs-sample-rate) do agente de infraestrutura para evitar a raspagem da métrica NFS.
