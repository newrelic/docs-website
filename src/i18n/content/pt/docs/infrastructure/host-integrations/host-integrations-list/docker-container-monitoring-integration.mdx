A Docker [integração](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) contêiner monitora e reporta métricas do docker contêiner usando [Docker API](https://docs.docker.com/reference/api/engine/version/v1.48/) para New Relic.

<Callout variant="tip">
  Esta integração é lançada como [código aberto no Github](https://github.com/newrelic/nri-docker). Um log de alterações também está disponível para as últimas atualizações. O suporte para essa integração está disponível com uma [assinatura do Expert Services](https://www.newrelic.com/expertservices).
</Callout>

<Callout variant="important">
  * Esta integração não suporta LCOW (contêiner Linux no Windows). Para mais informações, consulte a [documentação do Docker](https://docs.docker.com/desktop/windows/wsl/).
  * Para Linux - esta integração vem junto com o agente de infraestrutura, e se tornaria automaticamente ativa, caso o contêiner docker fosse detectado.
  * Para Windows - a partir da versão 2.3.0, estamos fornecendo compatibilidade, mas requer uma instalação manual sobre o agente de infraestrutura. Ele será incluído no agente de infraestrutura em versões futuras.
</Callout>
