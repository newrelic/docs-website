A [integração de código aberto](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) do Memcached coleta contadores de desempenho e estatísticas do [Memcached](https://memcached.org/) e reporta dados para o New Relic.

Para obter informações sobre requisitos, instalação e configuração, consulte o [README de integração no host do Memcached](https://github.com/newrelic/nri-memcached) no GitHub.

<Callout variant="important">
  New Relic possui código aberto esta integração para possibilitar o monitoramento desta tecnologia. Essa integração é fornecida NO ESTADO EM QUE SE ENCONTRA, SEM GARANTIA OU SUPORTE, embora você possa relatar problemas e contribuir para essa integração por meio do GitHub. O suporte para esta integração está disponível com uma [assinatura do Expert Services](https://www.newrelic.com/expertservices).
</Callout>
