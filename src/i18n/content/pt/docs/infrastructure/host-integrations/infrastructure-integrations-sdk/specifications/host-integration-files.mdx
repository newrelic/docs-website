Este documento explica os arquivos utilizados pelo nosso [agente de infraestrutura integração no host](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations).

## Arquivos de integração [#overview]

Uma integração no host do agente de infraestrutura contém pelo menos estes arquivos:

* Um [arquivo executável](/docs/infrastructure/infrastructure-integrations/custom-integrations/create-integration-executable-file), escrito em qualquer linguagem, que exporta dados JSON em um formato esperado pelo agente de infraestrutura
* Um [arquivo de configuração](/docs/create-integrations/infrastructure-integrations-sdk/specifications/host-integrations-standard-configuration-format)
