Existem várias maneiras pelas quais o New Relic permite criar sua própria integração:

* Soluções gerais de telemetria (métrica, trace): Utilize nossos [SDKs de telemetria](/docs/data-ingest-apis/get-data-new-relic/new-relic-sdks/telemetry-sdks-send-custom-telemetry-data-new-relic).
* Crie um [aplicativo New Relic ](/docs/new-relic-one/use-new-relic-one/build-new-relic-one/new-relic-one-build-your-own-custom-new-relic-one-application)personalizado que use sua própria funcionalidade de interface JavaScript.
* Se você possui New Relic Infrastructure, você pode usar nossa [ferramenta leve de integração Flex](/docs/integrations/host-integrations/host-integrations-list/flex-integration-tool-build-your-own-integration) (recomendado) ou usar nosso SDK de integração para construir uma infraestrutura completa [de integração no host](/docs/integrations/host-integrations/getting-started/introduction-host-integrations).

<Callout variant="important">
  New Relic está fazendo a transição para contar com padrões de código aberto como o Prometheus para futura integração no host. Embora este SDK seja a base dessa transição, alguns dos tutoriais e ferramentas em torno deste SDK podem não estar atualizados com os desenvolvimentos mais recentes.
</Callout>

## O que é o SDK de integração de infraestrutura? [#what-is-it]

Nosso SDK de integração de infraestrutura é o que usamos para construir nosso [agente de infraestrutura integração no host](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations), que trabalha em conjunto com nosso agente de infraestrutura para relatar dados de seus hosts ou serviços.

Você pode estudar a documentação nesta seção para ajudá-lo a entender a estrutura de arquivos e as opções de configuração para essas integração. Isso pode ser útil para resolução de problemas e personalização de sua integração.
