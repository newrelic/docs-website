## Problema

Sua integração está reportando dados, mas faltam alguns atributos de inventário.

## Solução

Os seguintes nomes de chaves de inventário são usados internamente pela nossa plataforma. Caso a integração fornecida pelo cliente envie itens de inventário com o mesmo nome de chave, eles permanecerão invisíveis para o usuário:

* `pid`
* `installed_epoch`
* `epoch`
* `time`
* `size`
* `boot_id`
* `product_uuid`
* `installed_time`
