## Versão de infraestrutura [#agent-version]

O SDK de infraestrutura integração v4 requer agente de infraestrutura versão 1.13.0 ou superior.

O SDK v3 requer a versão do agente 1.0.888 ou superior.

Para informações sobre versão do agente de infraestrutura e opções de instalação e atualizações, consulte as [notas de versão da infraestrutura](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes).

Para notas sobre versões e alterações do SDK, consulte o [log de alterações](#change-log).

## Sistema operacional [#os]

A integração construída com o SDK pode ser compilada tanto para o sistema operacional Linux quanto para o Windows.

## Especificações de dados e arquivos [#data-specs]

infraestrutura integração nenhum host pode ser criado com qualquer linguagem de programação, desde que atenda às [especificações de dados e arquivos](/docs/infrastructure/integrations-sdk/get-started/create-activate-infrastructure-integration).
