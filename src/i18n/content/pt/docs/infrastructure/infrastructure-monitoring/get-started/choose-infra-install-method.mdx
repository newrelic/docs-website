O agente de infraestrutura é um arquivo executável que coleta dados sobre seus hosts. Você pode permitir que o agente de infraestrutura relate dados de [alguns serviços de terceiros](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations) e colete [dadoslog ](/docs/logs/enable-log-monitoring-new-relic/enable-log-monitoring-new-relic/forward-your-logs-using-infrastructure-agent).

O agente de infraestrutura pode ser executado no Windows, no macOS e em muitas distribuições do Linux. Este documento descreve os diferentes métodos que podem ser usados para instalar o agente de infraestrutura. Para começar, você precisará de uma [conta New Relic](https://newrelic.com/signup) e de uma chave de licença.

## Instale o agente de infraestrutura manualmente [#manual-install]

Recomendamos seguir nossos procedimentos de instalação manual se você deseja monitor alguns hosts, mas nada mais complexo. Em geral, você usará a linha de comando para criar um diretório onde o agente reside e, em seguida, adicionará as informações da sua chave de licença. Dependendo do seu sistema operacional, pode ser necessário criar e editar também um arquivo de configuração. Para instalar manualmente, comece aqui:

* [Instale em sistemas Linux com um gerenciador de pacote](/docs/infrastructure/install-infrastructure-agent/linux-installation/install-infrastructure-monitoring-agent-linux)
* [Instalar no macOS](/docs/infrastructure/install-infrastructure-agent/macos-installation/install-infrastructure-monitoring-agent-macos)
* Instale no Windows com [instalação MSI](/docs/infrastructure/install-infrastructure-agent/windows-installation/install-infrastructure-monitoring-agent-windows), [instalação zip](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-assisted-install-infrastructure-agent-windows) ou [instalação zip manual](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-manual-install-infrastructure-agent-windows)

## Use a instalação guiada [#guided-install]

A instalação guiada é um único comando CLI que você pode executar para monitor sua instância. É uma boa opção para pequenas organizações, ou para quem quer testar New Relic. Quando você executa o comando CLI, New Relic inspecionará seu sistema para encontrar outras tecnologias de compatibilidade que você queira monitor. Você também receberá a [API de status da infraestrutura](https://github.com/newrelic/infrastructure-agent/blob/master/docs/status_api.md) por padrão.

<Callout variant="important">
  A CLI do New Relic não oferece suporte a servidores FedRAMP. Se você é cliente FedRAMP, consulte as [instruções de instalação do manual](#manual-install).
</Callout>
