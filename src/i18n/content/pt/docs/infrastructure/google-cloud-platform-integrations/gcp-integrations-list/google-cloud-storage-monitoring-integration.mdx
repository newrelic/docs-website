A New Relic oferece uma [integração](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) para relatar seus dados do Google Cloud Storage para a New Relic. Aprenda como [conectar esta integração ao monitoramento de infraestrutura](#connect) e sobre os [dados métricos](#metrics) que New Relic reporta para esta integração.

## Recurso

O Google Cloud Storage é um serviço do Google Cloud Platform que você pode usar para fornecer conteúdo de sites, armazenar dados para arquivamento e recuperação de desastres e distribuir objetos de dados por meio de download direto. Com a integração Google Cloud Storage, você pode acessar estes recursos:

* Visualize gráficos e informações sobre os dados que você está armazenando e recuperando do Google Cloud Storage.
* [Crie consultas e gráficos personalizados](/docs/using-new-relic/data/understand-data/query-new-relic-data) a partir de [dados](#metrics) capturados automaticamente.
* Defina [alertas](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information#integration) sobre seus dados Google Cloud Storage diretamente na página de integração.

## Ativar integração [#activate]

Para ativar a integração, siga os procedimentos padrão para [conectar seu serviço GCP ao New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frequência de votação [#polling]

New Relic consulta seus serviços Google Cloud Storage com base em um intervalo de sondagem de 5 minutos.

## Encontre e use dados [#find-data]

Após conectar a integração ao New Relic e aguardar alguns minutos, os dados aparecerão na interface New Relic . Para [encontrar e usar dados de integração](/docs/telemetry-data-platform/understand-data/new-relic-data-types/), incluindo seus painéis<InlinePopover type="dashboards"/>e suas configurações de alerta, vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP > Google Cloud Storage**</DNT>.

Para criar dashboard personalizado para integração, [crie uma consulta](/docs/integrations/new-relic-integrations/getting-started/use-integration-data-new-relic-insights) para o tipo de evento `GcpStorageBucketSample` com o valor do provedor `GcpStorageBucket`.

## Dados métricos [#metrics]

A integração reporta [dados métricos](/docs/telemetry-data-platform/understand-data/new-relic-data-types/#metrics) para todos os valores de `method` e `response_code`:

* `response_code`: o código de resposta das solicitações.
* `method`: o nome do método API chamado.
