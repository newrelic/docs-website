A integração da New Relic com o [Google Cloud Platform (GCP)](https://cloud.google.com/) inclui uma integração para reportar dados [do Google BigQuery](https://cloud.google.com/bigquery/) para New Relic. Este documento explica como ativar a integração do Google BigQuery e descreve os dados que podem ser relatados.

## Recurso

O Google BigQuery é um data warehouse sem infraestrutura para gerenciar. Com o BigQuery, você pode especificar o esquema de dados ou permitir que o serviço o descubra para você. Você pode explorar seus dados usando consulta SQL.

## Ativar integração [#activate]

Para ativar a integração, siga os procedimentos padrão para [conectar seu serviço GCP ao New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frequência de votação [#polling]

New Relic integração consulta serviços GCP de acordo com um intervalo de sondagem que varia dependendo da integração. A frequência de pesquisa para a integração do Google BigQuery é de cinco minutos. Para informações sobre a resolução, consulte [detalhes dos dados métricos](#metrics).

## Ativar tabela métrica [#table-metrics]

A coleta de métricas da tabela do BigQuery está desativada por padrão. Isso afeta a métrica listada em [GcpBigQueryTableSample](#table-sample). Para coletar essas métricas, ative a configuração "Ativar métricas de tabela" no menu de configuração da integração do BigQuery.

## Encontre e use dados [#find-data]

Após ativar a integração e aguardar alguns minutos (com base na [frequência da votação](#polling)), os dados aparecerão na interface do New Relic . Para [encontrar e usar seus dados](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), incluindo links para seu <InlinePopover type="dashboards"/>e configurações de alerta, acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP > (select an integration)**</DNT>.

## Dados métricos [#metrics]

Para visualizar [dados métricos](/docs/telemetry-data-platform/understand-data/new-relic-data-types/#metrics) para sua integração do Google BigQuery no New Relic, [crie uma consulta NRQL ](/docs/integrations/new-relic-integrations/getting-started/use-integration-data-new-relic-insights#nrql)para o [evento](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#event) `GcpBigQueryProjectSample`, `GcpBigQueryDatasetSample` e `GcpBigQueryTableSample` e seu atributo.
