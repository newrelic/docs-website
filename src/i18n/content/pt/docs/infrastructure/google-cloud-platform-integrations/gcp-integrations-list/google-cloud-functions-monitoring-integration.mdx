Nossa [integração de infraestrutura](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) com o [Google Cloud Platform (GCP)](https://cloud.google.com/) inclui uma que reporta dados [Google Cloud Functions](https://cloud.google.com/functions/) aos nossos produtos. Este documento explica como ativar a integração do GCP Cloud Functions e descreve os dados que podem ser relatados.

## Recurso

O serviço Google Cloud Functions permite executar código sem servidor. Usando a [interface do Google](https://cloud.google.com/functions/docs/quickstarts), os desenvolvedores podem criar pequenos trechos de código destinados a executar uma função específica. A função pode então responder a eventos na nuvem sem a necessidade de gerenciar um servidor de aplicativos ou ambiente de tempo de execução.

## Ativar integração [#activate]

Para ativar a integração, siga os procedimentos padrão para [conectar seu serviço GCP ao New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frequência de votação [#polling]

Nossa integração consulta seus serviços GCP de acordo com um intervalo de sondagem, que varia dependendo da integração.

* Frequência de pesquisa para GCP Cloud Functions: cinco minutos
* Resolução: um ponto de dados a cada minuto

## Visualizar e usar dados [#find-data]

Após ativar a integração e aguardar alguns minutos (com base na [frequência da votação](#polling)), os dados aparecerão na interface. Para [visualizar e usar seus dados](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), incluindo links para <InlinePopover type="dashboards"/>e configurações de alerta, acesse <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP**</DNT> e selecione uma integração.
