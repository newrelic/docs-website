A integração [New Relic Infrastructure](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure) monitor o desempenho de produtos e serviços populares. New Relic Google Cloud PlatformA integração do (GCP) da permite monitor seus dados do GCP em vários New Relic recursos .

## Conecte o GCP e a New Relic [#connect]

Para obter dados do GCP, siga os procedimentos padrão para [conectar seu serviço GCP ao New Relic](/docs/infrastructure/gcp-integrations).

## Ver seus dados do GCP [#insights]

Depois de seguir o processo de configuração, os dados da sua conta do Google Cloud Platform serão reportados diretamente à New Relic.

Para visualizar seus dados do GCP:

1. Vá para <DNT>\*\*[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities)</DNT>.
2. Pesquise <DNT>**Google Cloud Platform**</DNT> e clique no bloco.

Você pode visualizar e reutilizar a consulta NRQL no painel <InlinePopover type="dashboards" />pré-configurado e no painel <DNT>**Events explorer**</DNT>. Isso permite que você adapte a consulta às suas necessidades específicas.

Evento, painel e outros dados estão disponíveis para todos os serviços no New Relic.
