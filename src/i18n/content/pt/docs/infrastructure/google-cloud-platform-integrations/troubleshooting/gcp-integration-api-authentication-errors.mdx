## Problema

Você deseja [integrar seus serviços do Google Cloud Platform (GCP)](/docs/introduction-google-cloud-platform-integrations) ao New Relic, mas vê erros de autenticação de API.

## Solução

Para resolver o problema:

1. Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP**
   </DNT>

   .

2. Na página da sua conta do GCP, selecione

   <DNT>
     **Reauthorize New Relic**
   </DNT>

   .

3. Siga os procedimentos padrão para [conectar seus serviços do GCP ao New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/#connect).

## Causa

Os motivos comuns para esse erro incluem:

* A autorização para New Relic usar API GCP foi revogada manualmente pelo usuário do aplicativo de gerenciamento de contas Google.
* A autorização para New Relic usar API do GCP não inclui uma ou mais permissões ([escopos Google OAuth](https://developers.google.com/identity/protocols/googlescopes)) necessárias para consultar uma ou mais API do GCP.
* A autorização para New Relic usar API do GCP expirou.
