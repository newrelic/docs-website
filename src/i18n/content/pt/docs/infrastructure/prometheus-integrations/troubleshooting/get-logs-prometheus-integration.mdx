## Problema

Você deseja obter o log de sua integração do Prometheus OpenMetrics com New Relic para Docker ou Kubernetes.

## Solução

<img
  title="Docker icon"
  alt="Docker icon"
  src="/images/os_icon_docker.webp"
/>

<DNT>
  **Docker**
</DNT>

```
docker logs nri-prometheus
docker logs nri-prometheus 2>&1 | grep -v "level=debug"
```

<img
  title="img-integration-k8s@2x.png"
  alt="img-integration-k8s@2x.png"
  src="/images/os_icon_k8.webp"
/>

<DNT>
  **Kubernetes**
</DNT>

```
kubectl logs deploy/nri-prometheus
kubectl logs deploy/nri-prometheus | grep -v "level=debug"
```
