## Problema

Você está tendo problemas para identificar a causa dos problemas relatados com o envio de dados para New Relic [métrica API](/docs/data-ingest-apis/get-data-new-relic/metric-api/introduction-metric-api) da . Você deseja ver os dados exatos que estão sendo enviados de sua integração Prometheus OpenMetrics para Docker ou Kubernetes para a API métrica.

## Solução

Você pode configurar a integração do Prometheus OpenMetrics para mostrar os dados brutos que estão sendo enviados para o fluxo de saída padrão. Para ver os dados exatos que estão sendo enviados para a API métrica, defina a variável de ambiente `EMITTERS` como `"api,stdout"`.
