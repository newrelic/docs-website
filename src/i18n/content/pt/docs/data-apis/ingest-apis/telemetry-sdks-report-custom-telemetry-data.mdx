Nossos SDKs de telemetria são um conjunto de biblioteca de cliente de API de código aberto que envia dados para a plataforma New Relic. Nos bastidores, esses SDKs contam com nossas APIs de ingestão de dados primárias: métrica,API trace API, log API e de API evento.

Se nossas [soluções pré-construídas](https://newrelic.com/instant-observability) não atenderem às suas necessidades, nossos SDKs de telemetria são uma forma de criar uma solução de telemetria personalizada (consulte outras [soluções para relatar dados personalizados](/docs/data-apis/custom-data/intro-custom-data)).

## Requisitos e compatibilidade [#requirements]

Os SDKs de telemetria usam nossa API métrica, API de evento, API de log e API trace , que exigem um <InlinePopover type="licenseKey"/>. Portanto, você precisará de uma chave de licença para a conta para a qual deseja enviar dados.

<Callout variant="tip">
  A New Relic contribuiu com o SDK de telemetria para a comunidade de código aberto sob uma licença Apache 2.0.
</Callout>
