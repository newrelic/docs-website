Para organizações com várias contas New Relic, cada uma com requisitos de ingestão de dados distintos, os orçamentos de ingestão de dados fornecem monitoramento, visibilidade e alertas por conta para garantir que as contas permaneçam dentro de seus limites de uso definidos.

## Tipos de orçamento por estrutura do cliente

O tipo de orçamento que você pode criar depende da sua estrutura de cliente:

### Para clientes de locatário único

Você pode criar <DNT>**account level budgets**</DNT> para contas individuais ou grupos de até 5 contas dentro de sua organização.

### Para clientes multi-tenant

<DNT>**Managing organizations**</DNT> pode criar <DNT>**account level budgets**</DNT> para contas dentro da organização gerenciadora (mesma funcionalidade que clientes de locatário único).

<DNT>**Managed organizations**</DNT> não é possível criar orçamentos.

## Configurar orçamentos no nível da conta

<Steps>
  <Step>
    ### Abrir Orçamentos de Dados

    1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com/)**</DNT>.
    2. Em <DNT>[user menu](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)</DNT>, selecione <DNT>**Administration** &gt; **Data Management** &gt; **Data Budgets**</DNT>.
  </Step>

  <Step>
    ### Criar um orçamento

    Na seção <DNT>**Functional budgets**</DNT> , selecione <DNT>**+ Add functional budget**</DNT>.

    <Callout variant="important">
      <DNT>**For multi-tenant customers**</DNT>, esta seção exibe:

      * <DNT>**Organization level budgets**</DNT> para organizações gerenciadas
      * <DNT>**Account level budgets**</DNT> para a organização gerenciadora

      <DNT>**Managed organizations**</DNT> não é possível criar orçamentos no nível da conta.
    </Callout>
  </Step>

  <Step>
    ### Atribua seu orçamento

    1. Insira um nome para seu orçamento.
    2. Na lista dropdown, selecione e atribua até 5 contas a este orçamento.

    <Callout variant="tip">
      As contas que já têm orçamentos atribuídos aparecerão desabilitadas na lista de seleção.
    </Callout>

    3. Selecione <DNT>**Next**</DNT>.
  </Step>

  <Step>
    ### Definir limites de orçamento mensal

    1. Na seção <DNT>**Set monthly budget limits**</DNT>, você verá quantos GB restam no orçamento da sua organização.
    2. Insira a quantidade de GB que deseja alocar a este orçamento de nível de conta.

    <Callout variant="tip">
      Você pode alocar mais do que o orçamento da sua organização. Isso permite um planejamento flexível, mas pode resultar em excessos.
    </Callout>

    3. Selecione <DNT>**Next**</DNT>.
  </Step>

  <Step>
    ### Configurar notificação

    1. Para receber um aviso quando você tiver usado uma determinada porcentagem dos GB atribuídos a este orçamento para um mês específico, na seção <DNT>**Thresholds**</DNT>, selecione <DNT>**+ Add a threshold**</DNT> e insira a porcentagem na qual deseja receber notificação (por exemplo, 75%, 90% ou 95%).

    2. Para garantir que as notificações sejam enviadas quando o orçamento exceder o limite, na seção <DNT>**Destinations**</DNT>, adicione pelo menos um destino para receber notificações. Você pode escolher entre as seguintes opções:

       * <DNT>**Email**</DNT>: insira os endereços de e-mail dos usuários do New Relic para notificar sobre o orçamento.
       * <DNT>**Webhook**</DNT>: Insira a URL do webhook para enviar notificações para um endpoint específico.

    3. Selecione <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    ### Revisar detalhes do orçamento

    Depois de criar o orçamento, a página <DNT>**Data Budgets**</DNT> exibe as seguintes informações para cada orçamento:

    <table>
      <thead>
        <tr>
          <th>
            Nome do campo
          </th>

          <th>
            Descrição
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            <DNT>**Budget name**</DNT>
          </td>

          <td>
            O nome que você atribuiu ao orçamento.
          </td>
        </tr>

        <tr>
          <td>
            <DNT>**Status**</DNT>
          </td>

          <td>
            Status atual do orçamento.
          </td>
        </tr>

        <tr>
          <td>
            <DNT>**GB allocation**</DNT>
          </td>

          <td>
            Quantos GB de dados foram alocados para este orçamento.
          </td>
        </tr>

        <tr>
          <td>
            <DNT>**GB utilized**</DNT>
          </td>

          <td>
            Quanto dos dados alocados foi utilizado.
          </td>
        </tr>

        <tr>
          <td>
            <DNT>**Utilization percentage**</DNT>
          </td>

          <td>
            Percentual do orçamento que foi utilizado.
          </td>
        </tr>
      </tbody>
    </table>
  </Step>
</Steps>

## Gerenciar orçamentos existentes

Para gerenciar um orçamento existente:

1. Localize o orçamento na lista e selecione o menu de três pontos.

2. Escolha uma das seguintes opções:

   * <DNT>**View details**</DNT>: veja informações detalhadas sobre o orçamento, incluindo gráficos de utilização, limites, configurações de notificação, contas associadas e fontes de dados.
   * <DNT>**Edit**</DNT>: modifique as configurações de alocação de orçamento, limite ou notificação.
   * <DNT>**Delete**</DNT>: Remover o orçamento.

### Veja os detalhes do orçamento

Selecione um nome de orçamento ou escolha <DNT>**View details**</DNT> no menu de três pontos para ver:

* <DNT>**Total allocation**</DNT>: o total de GB alocado a este orçamento
* <DNT>**Utilization**</DNT>: Gráfico visual mostrando o uso ao longo do tempo com detalhamento dia após dia
* <DNT>**Thresholds**</DNT>: O limite de notificação que você configurou
* <DNT>**Notification destinations**</DNT>: onde os alertas são enviados quando o limite é excedido
* <DNT>**Associated accounts**</DNT>: As contas incluídas neste orçamento
* <DNT>**Data sources**</DNT>: Informações sobre as fontes de dados que contribuem para a utilização do orçamento, categorizadas por tipo de fonte
* <DNT>**Budget time range**</DNT>: O período de tempo coberto por este orçamento
