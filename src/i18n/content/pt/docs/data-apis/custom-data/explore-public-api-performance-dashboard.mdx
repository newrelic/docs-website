O dashboard de desempenho da API pública é um dashboard pronto para uso incluído como parte de sua conta New Relic. Ele fornece insights gerais acionáveis sobre o desempenho da API pública e uma oportunidade para novos clientes testarem nossos recursos de dashboard antes de adicionar seus próprios dados.

O dashboard funciona mostrando a latência real experimentada por uma amostra anônima de clientes da New Relic ao acessar API públicas populares.

<Callout variant="important">
  O conjunto de dados de desempenho global é apresentado como está. O conjunto de dados de Desempenho Global representa um agregado de amostras de diversas fontes, e a New Relic não faz nenhum esforço para tentar confirmar a exatidão, integridade ou veracidade dos dados. Esses dados não devem ser considerados a única fonte de informações para qualquer finalidade que você possa usá-los, e a New Relic não é responsável por decisões tomadas com base nesses dados. O conjunto de dados de desempenho global não deve ser visto como um endosso ou uma recomendação da New Relic das tecnologias representadas no conjunto de dados.
</Callout>

## Adicione o dashboard no New Relic [#Add-dashboard]

Para adicionar o dashboard de desempenho da API pública:

1. Na página do painel, procure o dashboard pré-construído de desempenho da API pública. Você pode clicar no link

   <DNT>
     **View All**
   </DNT>

   ao lado do popular título do painel predefinido para pesquisá-lo.

2. Selecione o dashboard.

3. Selecione uma conta no dropdown.

4. Clique em

   <DNT>
     **View dashboard**
   </DNT>

   para adicioná-lo à sua conta.

## Explorar o dashboard [#explore-dashboard]

Você pode personalizar seu novo dashboard pré-construído. Você pode alterar o nome e as configurações, adicionar novo conteúdo e novo widget, refinar os gráficos e muito mais! Consulte [Gerenciar seu painel](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/) para obter mais informações.

Abaixo estão algumas sugestões sobre como explorar o dashboard de desempenho da API pública.

* Clique no

  <Icon name="fe-more-horizontal"/>

  no canto direito de qualquer um dos gráficos e selecione

  <DNT>
    **View query**
  </DNT>

  para visualizar a consulta [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) usada para criar o gráfico.

* Clique no

  <Icon name="fe-more-horizontal"/>

  no canto direito de qualquer gráfico e selecione

  <DNT>
    **Get as image**
  </DNT>

  para visualizar ou fazer download de qualquer gráfico como imagem.

* Selecione um domínio específico no gráfico de barras ou adicione um filtro clicando no campo de texto na parte superior da página.

* Se você já adicionou seus próprios dados, experimente copiar a consulta e modificá-los para seu próprio uso.

<Callout variant="important">
  O dashboard de desempenho da API pública não está disponível atualmente para clientes da UE.
</Callout>

<Callout variant="important">
  O dashboard de desempenho da API pública não oferece suporte a alertas no momento.
</Callout>

## Dados relatados [#data-reported]

Disponibilizamos dados públicos da API para todas as organizações da New Relic por meio do evento `Public_APICall` . Esses dados não contam para sua ingestão. Se você quiser ver a estrutura desses dados, execute esta consulta:

```sql
FROM Public_APICall select keyset()
```
