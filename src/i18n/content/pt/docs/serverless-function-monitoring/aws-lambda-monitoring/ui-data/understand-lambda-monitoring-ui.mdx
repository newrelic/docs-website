  Se você não conseguir encontrar seus dados do Lambda:

  * Certifique-se de ter seguido as [instruções para ativar o monitoramento do Lambda](/docs/install-enable-new-relics-monitoring-aws-lambda).
  * Observe que esse recurso é diferente de nossa [integração Lambda de monitoramento de infraestrutura](/docs/integrations/amazon-integrations/aws-integrations-list/aws-lambda-monitoring-integration).
</Callout>

## Entenda os dados do gráfico [#chart-data]

Os gráficos de dados Lambda são gerados executando [uma consulta NRQL](/docs/insights/nrql-new-relic-query-language/using-nrql/introduction-nrql) de [dados de eventos](/docs/understand-your-lambda-data#data-structure) relacionados ao Lambda. Os motivos para visualizar a consulta NRQL de um gráfico incluem:

* Para entender melhor o que um gráfico está exibindo
* Para obter ideias sobre como criar uma consulta e um gráfico NRQL personalizado

Documentação relacionada:

* [Saiba como visualizar a consulta de um gráfico](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform#view-query).
* [Saiba mais sobre o armazenamento e a estrutura de dados do Lambda](/docs/understand-your-lambda-data).
