Você pode monitorar a função do Lambda AWS que usa [streaming de resposta](https://aws.amazon.com/blogs/compute/introducing-aws-lambda-response-streaming/) no aplicativo Node.js para obter insights reais sobre o desempenho e o comportamento de cada invocação. Esta solução integra-se ao [Monitoramento de IA](/docs/ai-monitoring/intro-to-ai-monitoring/) para permitir que você rastreie respostas de modelos e colete dados de observabilidade relacionados à sua carga de trabalho de IA. Usando esses recursos, você pode otimizar o desempenho do aplicativo sem servidor e manter uma supervisão abrangente dos processos orientados por IA.

Este documento ajuda você a:

* Configurar monitoramento de resposta streaming função do Lambda
* Entenda as métricas e dados disponíveis para streaming de resposta Função do Lambda
* Aprenda a usar a interface do New Relic para monitorar e observar o desempenho do seu streaming de resposta função do Lambda

## Configurar o streaming de resposta do Lambda [#setup]

### Pré-requisitos [#prerequisites]

* Verifique se a sua função do Lambda atende à nossa [compatibilidade e requisitos](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/compatibility-requirement-lambda-monitoring).
* Use a [camada lambda mais recente](https://layers.newrelic-external.com) disponível para Node.js.

### Configurar o streaming de resposta do Lambda [#configure-response-streaming]

Para habilitar o monitoramento de streaming de resposta para sua função Node.js do Lambda:

1. Instalar e configurar [o monitoramento Lambda no New Relic](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/configure-serverless-aws-monitoring).

2. Adicione as [variáveis de ambiente](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/env-variables-lambda) necessárias para a função de streaming de resposta do Node.js do Lambda.

3. Salve as alterações e implante a função.

   Após a implantação, você pode visualizar os dados de streaming de resposta do Lambda na plataforma New Relic. Para obter mais informações, consulte [Acessar os dados de streaming de resposta do Lambda](#access-ui).

## Acesse os dados de streaming de resposta do Lambda [#access-ui]

Para visualizar os dados de streaming de resposta do Lambda na plataforma New Relic:

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; left navigation pane &gt; All Capabilities &gt; Serverless Functions**</DNT>.

2. Selecione a função necessária do Lambda que usa streaming de resposta.

3. Exibir resumo, invocações, métricas do CloudWatch e distributed tracing para sua função de streaming de resposta do Lambda.

4. Na seção **More views** , clique em **AI Responses**.

   Para saber mais sobre as respostas da IA, consulte [Exibir respostas da IA](/docs/ai-monitoring/explore-ai-data/view-ai-responses).

## Métricas-chave para streaming de resposta [#key-metrics]

Na página de **Cloudwatch metrics** , monitore as seguintes métricas relacionadas ao streaming de resposta:

* **Bytes de saída transmitidos:** exibe o número médio de bytes transmitidos pela função do Lambda.
* **Streamed Outbound taxas de transferência:** Exibe a taxa média na qual os bytes são transmitidos pela função do Lambda.
