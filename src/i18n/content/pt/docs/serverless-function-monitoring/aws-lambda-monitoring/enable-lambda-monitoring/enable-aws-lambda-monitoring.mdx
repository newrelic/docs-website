Existem várias etapas para ativar o monitoramento Lambda:

1. [Vincule sua conta AWS.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/account-linking)
2. [Opcional: configure a coleta log do CloudWatch](/docs/logs/enable-log-management-new-relic/enable-log-monitoring-new-relic/aws-lambda-sending-cloudwatch-logs): se na Etapa 1 você usou a CLI `newrelic-lambda`, que instala automaticamente o coletor de logs do CloudWatch por padrão e você pode pular esta etapa.
3. [Instrumento uma função de exemplo.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-example)
4. [Instrumento suas funções.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-your-own)
5. [Opcional: Configure alerta e evento personalizado.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/configure-serverless-monitoring-aws-lambda)

Veja também [instrumentação manual herdada para Lambda](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy).
