Na Etapa 1, se você usou a CLI newrelic-lambda, ela verificou se você havia instalado a integração [API Polling](/docs/infrastructure/amazon-integrations/connect/connect-aws-new-relic-infrastructure-monitoring) ou [métrica Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream) . Se não encontrou nenhum deles, instalou a integração API Polling.

Se você não usou a CLI na Etapa 1, certifique-se de ter instalado e configurado a API Polling ou a integração do métrica Streams.
