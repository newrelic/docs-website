## Problema

Você não pode usar o AWS Secrets Manager para armazenar sua chave de licença do New Relic.

## Solução

Se sua organização não permitir o uso do AWS Secrets Manager, a extensão Lambda do New Relic aceita uma variável de ambiente `NEW_RELIC_LICENSE_KEY` . Adicione o sinalizador `--disable-license-key-secret` do comando `newrelic-lambda integrations install` . Em seguida, defina esta variável de ambiente para seu <InlinePopover type="licenseKey" />na função de configuração do Lambda .
