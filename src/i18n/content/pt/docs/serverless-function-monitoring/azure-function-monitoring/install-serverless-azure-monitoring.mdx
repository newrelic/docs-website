Você pode configurar o monitoramento do New Relic para aplicativos de funções do Azure hospedados no Linux, Windows ou como funções em contêineres. Isso é alcançado instrumentando seu aplicativo de função com o agente New Relic apropriado para seu tempo de execução, como Node.js .NET ou Python.

## Antes de você começar [#begin]

Garanta que sua Função do Azure atenda à nossa [compatibilidade e requisitos](/docs/serverless-function-monitoring/azure-function-monitoring/compatibility-requirement-azure-monitoring).

<Steps>
  <Step>
    ## Vincule sua conta do Azure ao New Relic [#link]

    Vincule sua conta Azure ao New Relic configurando a sondagem para Azure Monitor Métrica. Isso permite que você visualize essas métricas na interface New Relic. Para obter mais informações, consulte [integração Azure ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-functions-monitoring-integration/#polling).
  </Step>

  <Step>
    ## instrumento sua função Azure [#instrument]

    Dependendo do seu ambiente de implantação, selecione uma das seguintes opções para instrumentar sua Função Azure com o New Relic.

    <Tabs>
      <TabsBar>
        <TabsBarItem id="container">Função em contêiner do</TabsBarItem> <TabsBarItem id="linux">Linux</TabsBarItem> <TabsBarItem id="windows">Windows</TabsBarItem>
      </TabsBar>

      <TabsPages>
        <TabsPageItem id="linux">
          Você pode instrumentar seu aplicativo de funções Azure no Linux com New Relic para os seguintes ambientes de tempo de execução:

          * [Node.js](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#node)
          * [Python](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#python)
          * [.NET](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#net)
        </TabsPageItem>

        <TabsPageItem id="windows">
          Você pode instrumentar seu aplicativo de funções Azure no Windows com New Relic para os seguintes ambientes de tempo de execução:

          * [Node.js](/docs/serverless-function-monitoring/azure-function-monitoring/windows/#node)
          * [.NET](/docs/serverless-function-monitoring/azure-function-monitoring/windows/#net)
        </TabsPageItem>

        <TabsPageItem id="container">
          Você pode instrumentar seu aplicativo de funções Azure no Linux com New Relic para os seguintes ambientes de tempo de execução:

          * [Node.js](/docs/serverless-function-monitoring/azure-function-monitoring/container/#node)
          * [Python](/docs/serverless-function-monitoring/azure-function-monitoring/container/#python)
          * [.NET](/docs/serverless-function-monitoring/azure-function-monitoring/container/#net)
        </TabsPageItem>
      </TabsPages>
    </Tabs>
  </Step>

  <Step>
    ## Configurar variáveis de ambiente [#env-variables]

    Depois de instrumentar seu aplicativo de funções Azure com New Relic, [configure variáveis de ambiente](/docs/serverless-function-monitoring/azure-function-monitoring/env-variables-azure) para seu ambiente de execução.
  </Step>

  <Step>
    ## Reinicie suas funções Azure [#restart]

    Depois de adicionar as variáveis de ambiente, reinicie o Azure Functions para aplicar as alterações.
  </Step>
</Steps>
