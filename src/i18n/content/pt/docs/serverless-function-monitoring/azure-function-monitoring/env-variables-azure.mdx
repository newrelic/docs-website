Variáveis de ambiente são uma maneira de armazenar configurações e segredos fora do seu código. As tabelas a seguir exibem as variáveis obrigatórias e as variáveis específicas do tempo de execução que você pode definir para monitorar suas Funções Azure com o New Relic.

## Variáveis de ambiente obrigatórias

Independentemente do seu tempo de execução, você deve definir as seguintes variáveis de ambiente para monitorar suas Azure Functions com New Relic:

<table>
  <thead>
    <tr>
      <th style={{ width: "200px" }}>
        Nome da Variável
      </th>

      <th>
        Descrição
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `NEW_RELIC_ACCOUNT_ID`
      </td>

      <td>
        Especifica o ID da sua conta New Relic
      </td>
    </tr>

    <tr>
      <td>
        `NEW_RELIC_LICENSE_KEY`
      </td>

      <td>
        Especifica sua chave de ingestão do New Relic, substituindo o Secrets Manager
      </td>
    </tr>
  </tbody>
</table>
