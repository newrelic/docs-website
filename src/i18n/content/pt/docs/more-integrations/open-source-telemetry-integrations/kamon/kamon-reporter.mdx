[Kamon](https://kamon.io/) fornece um kit de ferramentas de instrumentação especializado em instrumentação automática de aplicativos Scala e Akka. Consiste em API para instrumentação métrica e de rastreamento e módulos de instrumentação automática que criam aplicativos métricos e de rastreamento distribuído.

Kamon também fornece módulos repórteres para transferir dados de telemetria para um backend de sua escolha. O repórter Kamon da New Relic está incluído no pacote Kamon e você pode configurá-lo para enviar dados de telemetria do seu aplicativo Kamon-instrumento para sua conta New Relic.

## Habilitar repórter Kamon [#install]

Para habilitar nosso repórter Kamon:

1. Se ainda não o fez, [crie uma conta New Relic](https://newrelic.com/signup). É grátis, para sempre.

2. Siga as [instruções de instalação do repórter Kamon](https://kamon.io/docs/latest/reporters/newrelic/).

3. Opcional: em vez de usar o Kamon amostragem de trace nativo, você pode ativar nosso recurso [Infinite Tracing](/docs/introduction-infinite-tracing) . Se você usar esta opção, normalmente você desejará configurar o Kamon para nos enviar todos os dados trace ([saiba mais sobre amostragem](/docs/distributed-tracing/concepts/how-new-relic-distributed-tracing-works)). Para ativar o rastreamento infinito:

   1. Na interface do New Relic, [configure um observador de rastreamento](/docs/infinite-tracing-set-trace-observer#set-up).
   2. Configure nosso repórter Kamon para enviar dados ao observador de rastreamento: defina o valor [`span-ingest-uri`](https://kamon.io/docs/latest/reporters/newrelic/) como [YOUR_TRACE_OBSERVER_URL](/docs/infinite-tracing-set-trace-observer#ui-endpoints) em seu bloco de configuração `kamon.newrelic` .

## Encontre seus dados

Para encontrar seus dados, acesse <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities**</DoNotTranslate>. Pesquise seu serviço pelo nome. Fornecemos uma visão geral pronta para uso que inclui gráficos para algumas das métricas mais úteis fornecidas pela instrumentação Kamon. Se você também estiver enviando dados distributed tracing , o recurso <DoNotTranslate>**Distributed tracing**</DoNotTranslate> estará disponível para consulta e visualização de rastreamento.

Você pode usar a visão geral pronta para uso como ponto de partida para construir seu próprio painel. Clique no ícone `...` em um gráfico para adicioná-lo a um dashboard ou visualize a consulta atrás do gráfico para modificá-lo e adicioná-lo ao seu dashboard.

Para obter mais informações sobre como consultar seus dados quando estiverem no New Relic, consulte [consultar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) e [Introdução ao NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql).
