[Dropwizard](https://www.dropwizard.io/en/latest/) é uma framework Java para o desenvolvimento de serviços da web RESTful de alto desempenho e amigáveis para operações, fornecendo instrumentação métrica pronta para uso. O repórter DropWizard da New Relic envia sua métrica do Dropwizard para sua conta New Relic.

Quer experimentar nosso repórter dropwizard? [Crie uma conta New Relic](https://newrelic.com/signup) gratuitamente! Não é necessário cartão de crédito.

Recursos para nosso repórter Dropwizard:

* [Leia-me](https://github.com/newrelic/dropwizard-metrics-newrelic)
* [Especificações de integração](https://github.com/newrelic/newrelic-exporter-specs/tree/master/dropwizard)

## Encontre seus dados

Para encontrar seus dados, acesse <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities**</DoNotTranslate>. Pesquise seu serviço pelo nome. A partir daí, você pode explorar sua métrica usando <DoNotTranslate>**Metrics and events**</DoNotTranslate> e criar um painel usando sua métrica.

Para obter mais informações sobre como consultar seus dados quando estiverem no New Relic, consulte [consultar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) e [Introdução ao NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql).
