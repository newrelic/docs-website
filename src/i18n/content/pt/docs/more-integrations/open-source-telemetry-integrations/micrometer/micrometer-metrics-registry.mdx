[O Micrometer](https://micrometer.io/) fornece uma fachada simples sobre os clientes de instrumentação para os sistemas de monitoramento mais populares, permitindo instrumentar o código do seu aplicativo baseado em JVM sem dependência de fornecedor.

O registro métrico do Micrometer da New Relic envia os dados de telemetria do Micrometer para sua conta New Relic. Não tem um? [Cadastre-se gratuitamente](https://newrelic.com/signup). Não é necessário cartão de crédito.

Recursos para nossa integração com Micrômetro:

* [Leia-me](https://github.com/newrelic/micrometer-registry-newrelic)
* [Especificações de integração](https://github.com/newrelic/newrelic-exporter-specs/tree/master/micrometer)

## Encontre seus dados

Para encontrar seus dados, acesse <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities**</DoNotTranslate>. Pesquise seu serviço pelo nome. Fornecemos uma visão geral pronta para uso que inclui gráficos para algumas das métricas mais úteis fornecidas pela instrumentação Micrometer Spring Boot.

Você pode usar a visão geral pronta para uso como ponto de partida para construir seu próprio painel. Clique no ícone `...` em um gráfico para adicioná-lo a um dashboard ou visualize a consulta atrás do gráfico para modificá-lo e adicioná-lo ao seu dashboard.

Para obter mais informações sobre como consultar seus dados quando estiverem no New Relic, consulte [consultar seus dados](/docs/using-new-relic/data/understand-data/query-new-relic-data) e [Introdução ao NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql).
