Você tem uma variedade de opções para começar a usar o OpenTelemetry e o New Relic. Você pode ir direto para o monitoramento de seus próprios aplicativos, hosts ou cluster do Kubernetes, ou pode configurar alguns aplicativos de demonstração para experimentar OpenTelemetry e New Relic. Temos uma série de instruções que ajudarão você a aproveitar o poder do OpenTelemetry para coletar dados e enviá-los para a New Relic.

## Configure seu próprio aplicativo ou serviço [#own-app]

Se você estiver familiarizado com o OpenTelemetry e pronto para trabalhar com seu próprio código, confira [Configurar seu próprio aplicativo ou serviço com OpenTelemetry](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/get-started/opentelemetry-set-up-your-app). Essas etapas ajudarão você a coletar telemetria e exportá-la para o New Relic.

## Monitor um host [#otel-infra]

Você pode monitor seus hosts usando o coletor OpenTelemetry. Consulte [coletor para monitoramento de host](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/collector/opentelemetry-collector-infra-hosts).

## Monitor Kubernetes [#otel-kubernetes]

Monitor seu cluster do Kubernetes com o coletor OpenTelemetry. Veja as instruções em [Vincular o aplicativo OpenTelemetry-instrumento ao Kubernetes](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-otel-applications-kubernetes/).

## Experimente um aplicativo de demonstração [#demo-app-java]

Em vez de ir direto para a configuração do seu próprio aplicativo, você pode ativar nosso aplicativo de demonstração com OpenTelemetry e New Relic.

Nossos tutoriais para [Java](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/get-started/opentelemetry-tutorial-java) e [Python](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/get-started/opentelemetry-tutorial-python) oferecem uma variedade de opções para experimentar o OpenTelemetry:

* Inspecione um aplicativo de pré-instrumento que você pode ativar rapidamente para enviar dados para o New Relic.
* Use um agente OpenTelemetry para instrumentar automaticamente um aplicativo de demonstração.
* Instrumentar manualmente um aplicativo de demonstração.

## Aprenda conceitos de OpenTelemetry na masterclass [#otel-masterclass]

Se você quiser aprender mais sobre os conceitos por trás do OpenTelemetry e aplicá-los com um pequeno aplicativo de demonstração, dê uma olhada em nossa [masterclass](https://developer.newrelic.com/opentelemetry-masterclass). Você pode experimentar nossos tutoriais práticos para .NET ou Python.
