Temos duas opções de uso API para gerenciar programaticamente o recurso de alerta e inteligência aplicada da New Relic:

* NerdGraph (GraphQL): esta é a API principal e recomendada da New Relic para consulta de dados e configuração. Estamos sempre adicionando funcionalidades ao NerdGraph. [Consulte os documentos do NerdGraph relacionados a alertas.](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/alerts-nerdgraph/nerdgraph-api-examples)
* API REST: esta é a API original da New Relic. Fazemos um trabalho mínimo nesta API e é por isso que recomendamos o uso do NerdGraph. Se você estiver usando atualmente a API REST, recomendamos mudar para o NerdGraph. [Consulte a documentação da API REST relacionada a alertas.](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/rest-api-alerts/rest-api-calls-alerts)
