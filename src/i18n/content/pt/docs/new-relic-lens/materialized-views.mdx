Para consultas maiores e computacionalmente custosas, você pode armazenar os resultados como <DNT>materialized views</DNT> em <DNT>Lens</DNT>. Uma <DNT>materialized view</DNT> armazena os resultados pré-computados de uma consulta, permitindo que você consulte os dados armazenados em vez de executar a consulta complexa toda vez.

## Benefícios de <DNT>materialized views</DNT> [#benefits]

<DNT>Materialized views</DNT> ajudá-lo:

* **Reduza os custos de consulta**: Evite buscar repetidamente os mesmos dados de fontes de dados externas, o que pode gerar custos com base nos dados escaneados ou transferidos.
* **Melhore o desempenho**: Consulte resultados pré-computados em vez de executar junções complexas em vários bancos de dados a cada vez.
* **Simplifique painéis**: Armazene resultados de consultas a vários bancos de dados e referencie-os de forma simples em widgets de painel.

## Pré-requisitos [#prerequisites]

Antes de começar:

* Verifique se você tem as [permissões](/docs/new-relic-lens/overview#permissions) necessárias para criar e gerenciar <DNT>materialized views</DNT>.
* Se você precisar consultar fontes de dados externas, certifique-se de que os [<DNT>connectors</DNT>](/docs/new-relic-lens/set-up-connectors) estejam configurados.

## Criar um <DNT>materialized view</DNT> [#create]

Para criar um <DNT>materialized view</DNT>:

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Administration &gt; Materialized views**</DNT>.
2. Clique em <DNT>**Create materialized view**</DNT>.
3. Insira um nome exclusivo para o <DNT>materialized view</DNT>.
4. Para consultar dados do NRDB, selecione uma conta no seletor de contas. Você pode unir dados do NRDB desta conta com fontes de dados externas.
5. Escreva sua consulta SQL usando o [formato de referência de tabela](/docs/new-relic-lens/query-data#table-format): `connection_name.schema.table`
6. Clique em <DNT>**Create**</DNT>.

O <DNT>materialized view</DNT> aparece na lista. O armazenamento dos resultados pode levar algum tempo, dependendo da complexidade da sua consulta.

## Atualizar um <DNT>materialized view</DNT> [#refresh]

<DNT>Materialized views</DNT> contêm um snapshot de dados de quando foram criados ou atualizados pela última vez. Para obter dados recentes:

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Administration &gt; Materialized views**</DNT>.
2. Encontre o <DNT>materialized view</DNT> que você deseja atualizar.
3. Clique no menu <Icon name="fe-more-horizontal" />.
4. Selecione <DNT>**Refresh**</DNT>.

A visualização executa a consulta original novamente e atualiza os resultados armazenados com os dados mais recentes.

<Callout variant="important">
  Considere seus requisitos de atualidade dos dados ao decidir com que frequência atualizar <DNT>materialized views</DNT>. Atualizar frequentemente visualizações com consultas complexas pode aumentar os custos.
</Callout>

## Consultar um <DNT>materialized view</DNT> [#query]

Você pode consultar uma <DNT>materialized view</DNT> como qualquer outra tabela no <DNT>Lens</DNT>:

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Query your data**</DNT>.

2. Ative a chave ao lado de <DNT>**Notebooks**</DNT> e clique em <DNT>**+ SQL**</DNT> para adicionar um bloco SQL.

3. Referencie seu <DNT>materialized view</DNT> usando o prefixo `mv.` seguido pelo nome da visualização:

   ```sql
   SELECT * FROM mv.your_materialized_view_name
   ```

4. Execute sua consulta.

Você pode aplicar filtros, junções ou agregações adicionais no <DNT>materialized view</DNT>:

```sql
SELECT store_id, transaction_count
FROM mv.top_stores_mv
WHERE transaction_count > 1000
ORDER BY transaction_count DESC
```

O <DNT>materialized view</DNT> atua como uma tabela contendo os resultados pré-computados, permitindo realizar análises adicionais sem executar a consulta complexa original novamente.

## Excluir um <DNT>materialized view</DNT> [#delete]

Para excluir um <DNT>materialized view</DNT> que você não precisa mais:

1. Vá para <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Administration &gt; Materialized views**</DNT>.
2. Encontre o <DNT>materialized view</DNT> que você deseja excluir.
3. Clique no menu <Icon name="fe-more-horizontal" />.
4. Selecione <DNT>**Delete**</DNT>.
5. Confirme a exclusão.

<Callout variant="caution">
  A exclusão de um <DNT>materialized view</DNT> é permanente. Quaisquer consultas ou dashboards que referenciem a visualização excluída falharão.
</Callout>

Para exemplos de consulta incluindo junções entre bancos de dados, consulte [Consultar visualizações materializadas](/docs/new-relic-lens/query-data/#query-mv).
