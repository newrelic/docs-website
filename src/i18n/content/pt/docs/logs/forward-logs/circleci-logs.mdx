Configure o serviço [de webhook CircleCI](https://circleci.com/docs/2.0/webhooks/) para enviar log CI/CD para New Relic.

## Configure um gatilho de webhook no CircleCI [#setup]

Conclua estas etapas para configurar um webhook do CircleCI para encaminhar seu log para o New Relic:

1. Login no [CircleCI](https://app.circleci.com/projects).

2. Vá para um de seus projetos do CircleCI.

3. Clique em <DNT>**Project settings**</DNT>.

4. Na barra lateral das configurações do projeto, clique em <DNT>**Webhooks**</DNT>.

5. Clique em <DNT>**Add webhook**</DNT>.

6. Dê um nome ao seu webhook.

7. Insira <DNT>**Logs endpoint**</DNT> da New Relic:

   * NÓS:

     ```
     "https://log-api.newrelic.com/log/v1?Api-Key=YOUR_LICENSE_KEY"
     ```

   * UE:

     ```
     "https://log-api.eu.newrelic.com/log/v1?Api-Key=YOUR_LICENSE_KEY"
     ```

   Use a região que você definiu em sua conta New Relic. Substitua `YOUR_LICENSE_KEY` pela sua New Relic <InlinePopover type="licenseKey"/>.

8. Selecione <DNT>**Workflow**</DNT> ou <DNT>**Job**</DNT>, dependendo do tipo de evento que você deseja acionar o logpush.

9. Se você configurou sua API de recebimento ou serviço de terceiros, clique em <DNT>**Test ping event**</DNT> para criar um evento de teste.
