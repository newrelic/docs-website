## Problema

Ao implantar um novo host ou servidor, ou fazer outras alterações no aplicativo, você deseja ver imediatamente como o seu sistema responde a essas alterações. Você precisa visualizar suas mensagens do log conforme elas chegam em tempo real.

## Solução

Para se concentrar nas mensagens conforme elas chegam em tempo real, ative o registro de cauda em tempo real:

1. Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Logs**</DNT>.

2. Insira uma consulta para focar no tipo de informação log que você precisa e clique em <DNT>**Query logs**</DNT>. Um exemplo de consulta:

   ```
   Find logs where annotations.restartRequested:"2020-12-23T18:00:00Z"
   ```

3. À direita, clique em <DNT>**Live tail**</DNT> para ativar <Icon name="fe-play-circle"/> ou desabilitar <Icon name="fe-pause-circle"/> isto.
