## Problema

Os logs não estão aparecendo no New Relic para um CloudWatch via configuração de integração do Firehose.

## Solução

Possíveis razões e suas soluções:

* Para S3 via Lambda ou CloudWatch via integração Lambda , a variável de ambiente `CUSTOM_META_DATA` pode não estar definida ou pode não ter um valor JSON válido. Verifique isso.
* Para CloudWatch via Firehose, os pares principais de valor podem não estar definidos corretamente. Certifique-se de que seus pares de valores principais estejam presentes em <DNT>**Parameters**</DNT>.
* Seus nomes principais podem se sobrepor a palavras-chave proibidas.
