## Problema

Os logs não estão aparecendo no New Relic para um CloudWatch via configuração de integração do Firehose.

## Solução

Possíveis razões e suas soluções:

* A chave de API pode estar incorreta. Navegue até <DNT>**Amazon Data Firehose &gt; Firehose streams &gt; your-stream &gt; Edit destination settings**</DNT> e atualize a chave de API com a chave de licença de ingestão correta.
* O endpoint fornecido pode estar errado. Verifique o endpoint.
* Verifique os logs do S3 ou os logs de erros do Cloudwatch para ver se há algum problema.
