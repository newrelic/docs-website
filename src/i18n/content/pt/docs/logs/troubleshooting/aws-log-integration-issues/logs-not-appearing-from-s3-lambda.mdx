## Problema

Os logs não estão aparecendo no New Relic para uma configuração de integração S3 via Lambda.

## Solução

Possíveis razões e suas soluções:

* Verifique os possíveis motivos em [Logs não aparecem do CloudWatch vai Lambda](/docs/logs/troubleshooting/aws-log-integration-issues/logs-not-appearing-from-cloudwatch-lambda).
* Os arquivos S3 podem não aderir às [restrições de encaminhamento de logs](/docs/logs/troubleshooting/aws-log-integration-issues/log-forwarding-restrictions) do New Relic.
* O `NewRelicServerlessLogForwarder` pode não ter o gatilho S3. Verifique a saída da pilha aninhada do CloudFormation e adicione-a.
