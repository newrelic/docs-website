O logs contextualizados não está disponível para aplicativos monitorados pelo C SDK. Em vez disso, você pode criar sua própria biblioteca de extensão de log usando a [log API](/docs/logs/log-management/log-api/introduction-log-api/). Certifique-se de que seu aplicativo tenha distributed tracing habilitado, para que você possa entender rapidamente o que acontece com as solicitações à medida que elas trafegam pelos serviços upstream e downstream.

Para habilitar distributed tracing para aplicativos monitorados pelo C SDK, [instale](/docs/distributed-tracing/enable-configure/quick-start/) ou [atualize](/docs/agents/c-sdk/install-configure/update-your-c-sdk-library/) para a versão mais recente do C SDK. distributed tracing requer [C SDK versão 1.1.0 ou superior](/docs/release-notes/agent-release-notes/c-sdk-release-notes/).
