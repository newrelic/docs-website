## Use core web vitals para melhorar o SEO [#core_vitals]

Use os gráficos <DoNotTranslate>**Core web vitals**</DoNotTranslate> para entender o desempenho do seu browser de acordo com [core web vitalsdo Google](https://web.dev/vitals/#core-web-vitals). Esses incluem:

* LCP: pintura com maior conteúdo, que mede o desempenho de carregamento
* INP: interação para a próxima pintura, que mede a interatividade entre a entrada do usuário e a resposta do navegador
* CLS: mudança cumulativa de layout, que mede a estabilidade visual na página do browser

Pontuações mais altas podem ajudar sua organização a economizar dinheiro e influenciar o SEO. Pontuações mais baixas podem afetar a percepção do usuário sobre o site da sua organização, o que pode resultar em clientes insatisfeitos ou perda de oportunidades de negócios.

Para saber mais sobre monitoramento core web vitals, consulte nosso [Guia de core web vitals](/docs/browser/new-relic-browser/guides/guide-to-core-web-vitals).

## Adicionar condição recomendada do alerta [#recommended-alert-conditions]
