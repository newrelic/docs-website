    Nosso <InlinePopover type="browser"/>fornece uma solução de monitoramento de usuário real (RUM). Ele mede a velocidade e o desempenho à medida que o usuário navega até o seu site através de diferentes browsers, dispositivos, sistemas operacionais e redes. Mas o monitoramento do Browser vai muito além de fornecer informações sobre o carregamento inicial da página. Use-o para medir dados completos do ciclo de vida da página e comece a obter as informações necessárias para ajudar a garantir a satisfação dos clientes.

    New Relic permite monitor os dados da atividade do navegador e otimizar o desempenho em toda a sua stack. Use o monitoramento do Browser para ajudar a garantir uma implantação bem-sucedida e solucionar rapidamente problemas visíveis aos clientes. monitor sua stack rapidamente e certifique-se de que todas as suas entidades estão operando como deveriam. Visualize a velocidade e o desempenho do aplicativo, erros de JavaScript, solicitações AJAX, log e muito mais. Gaste menos tempo tentando solucionar problemas e mais tempo entregando uma experiência digital perfeita aos clientes.

    ## Acompanhe os sinais vitais da web

    Core web vitals do Google são três métricas para medir a experiência do usuário no seu aplicativo de browser - e a New Relic pode ajudá-lo a rastreá-los! Há dois lugares para ver core web vitals no New Relic:

    * [Monitor core web vitals do Google para garantir o desempenho do seu aplicativo](/docs/browser/new-relic-browser/guides/guide-to-core-web-vitals#monitor)

    * [Use nossos sinais vitais adicionais da web para examinar outros aspectos do desempenho do seu aplicativo](/docs/browser/browser-monitoring/getting-started/browser-summary-page/)

      ## Identifique problemas do usuário final

      Instrumento qualquer tipo de dado que você precisa, como métrica, evento, log e trace, para ajudar a otimizar a experiência de carregamento da página final do seu usuário e mantê-lo satisfeito.

    * [Compare a interação real do browser do usuário e as tendências de monitoramento sintético](/docs/synthetics/synthetic-monitoring/administration/compare-page-load-performance-browser-synthetic-monitoring/)

    * [Concentre-se em localizações geográficas específicas ou tipos específicos de atividade do usuário final](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location/)

    * [Determine se os problemas estão relacionados a tipos específicos de browser, versões de browser ou tipos de dispositivos](/docs/browser/new-relic-browser/browser-pro-features/browsers-problem-patterns-type-or-platform/)
