Para visualizar os logs de desempenho de um aplicativo de browser:

<Steps>
  <Step>
    Vá para <DNT>[one.newrelic.com](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Browser**</DNT>.
  </Step>

  <Step>
    Selecione um aplicativo de browser.
  </Step>

  <Step>
    No menu à esquerda, navegue até **Logs**.
  </Step>
</Steps>

<img title="Browser logs" alt="Screenshot of browser logs page" src="/images/browser_logs_ui.webp" />

## Qual é o próximo?

* Você também pode visualizar os logs do browser em contexto com Session Replay (se estiver usando o recurso Session Replay ) ou junto com os erros de JavaScript na Errors Inbox. Consulte [Logs no Session Replay](#session-replay-section) e [Logs na Caixa de entrada de erros](#js-errors-section) para obter mais detalhes.

* Para obter mais informações sobre a interface de usuário de logs, consulte [Usar interface de usuário de logs](/docs/logs/ui-data/use-logs-ui) para obter mais detalhes.
