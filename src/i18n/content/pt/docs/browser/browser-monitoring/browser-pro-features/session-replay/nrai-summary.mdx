Você pode usar New Relic AI (NRAI) para ajudar a resumir e mapear os dados da trilha de eventos do Session Replay para que você possa entender melhor os erros e acertos da jornada do browser do seu usuário.

Para usar o NRAI para resumir os dados da trilha de evento do Session Replay, você deve ter [o NRAI habilitado](/docs/agentic-ai/new-relic-ai/#enable) para sua conta. Depois que o NRAI estiver habilitado em sua conta, execute o seguinte para exibir seu resumo:

<Steps>
  <Step>
    Vá para <DNT>**[one.newrelic.com &gt; All Capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT>.
  </Step>

  <Step>
    Selecione seu aplicativo de browser.
  </Step>

  <Step>
    No menu à esquerda, role para baixo e clique em <DNT>**Session replay**</DNT>.
  </Step>

  <Step>
    Clique no replay para ver os detalhes da sessão que você gostaria de resumir.

    <img title="Session replay" alt="screenshot of the session replay page in browser" src="/images/browser-session-replay.webp" />
  </Step>

  <Step>
    Clique em <DNT>**Summarize**</DNT>. Um resumo da sessão é gerado no painel de foco.

    <img title="Summarize session replay" alt="screenshot of the session replay page in browser" src="/images/finalcut.gif" />
  </Step>
</Steps>
