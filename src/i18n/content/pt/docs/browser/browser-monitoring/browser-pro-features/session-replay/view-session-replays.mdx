Comece a analisar os dados capturados no New Relic. Esta seção fornece instruções sobre como acessar e navegar no Session Replay, ajudando você a explorar a interação do usuário e identificar áreas para melhoria. Aprenda como aproveitar os dados do Session Replay para melhorar a experiência do usuário e solucionar problemas de forma eficaz.

## Veja o replay da sessão no New Relic [#view-data]

Você pode encontrar o Session Replay em dois lugares:

* Na página <DNT>**Session replay**</DNT> :

<Steps>
  <Step>
    Vá para <DNT>**[one.newrelic.com &gt; All Capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT>.
  </Step>

  <Step>
    Selecione seu aplicativo de browser.
  </Step>

  <Step>
    No menu à esquerda, role para baixo e clique em <DNT>**Session replay**</DNT>.
  </Step>

  <Step>
    Clique em cada replay para ver os detalhes da sessão.

    <img title="Session replay" alt="screenshot of the session replay page in browser" src="/images/browser_screenshot-full_Session-replay-page.webp" />
  </Step>
</Steps>

* Na página <DNT>**Errors inbox**</DNT> :

<Steps>
  <Step>
    No menu esquerdo do browser, clique em <DNT>**Errors**</DNT>.
  </Step>

  <Step>
    Clique nas páginas <DNT>**Triage**</DNT> e <DNT>**Group errors**</DNT> para ver os replays anexados aos erros.

    <img title="Replays in errors inbox" alt="Screenshot of replays in errors inbox" src="/images/browser_screenshot-full_errors-inbox-session-replay.webp" />
  </Step>
</Steps>
