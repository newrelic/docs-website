<Callout
  variant="course"
  title="laboratório"
>
  Este procedimento faz parte de um laboratório que ensina como solucionar problemas do seu aplicativo da web com o New Relic Browser. Se ainda não o fez, confira a [introdução do laboratório](/docs/browser/new-relic-browser/lab/over-view/).
</Callout>

Antes de percorrer o laboratório propriamente dito, você precisa ativar seu aplicativo React. Aqui vocÊ:

* Aprimore seu aplicativo React
* Envie tráfego para seu aplicativo com um gerador de carga simples

<Steps>
  <Step>
    Abra uma janela de terminal e clone o repositório do laboratório.

    ```bash
    git clone https://github.com/newrelic-experimental/relicstaurants.git
    ```
  </Step>

  <Step>
    Navegue até o diretório raiz do seu aplicativo e alterne para o diretório lab.

    ```bash
    cd relicstaurants
    git switch browser-pro-lab-material
    ```
  </Step>

  <Step>
    Em seguida, instale o dependência e execute o aplicativo.

    ```bash
    npm install
    npm run build
    npm run newstart
    ```

    Isso abre o aplicativo Relicstaurants no seu browser.

    <img
      title="Relicstraunts homepage"
      alt="Relicstraunts homepage"
      src={homepage}
    />

    Digite seu endereço de entrega e pesquise restaurantes para começar.

    <img
      title="Nearby restaurant list"
      alt="Nearby restaurant list"
      src={restaurantList}
    />

    Aqui você vê uma lista de restaurantes onde pode pedir comida.
  </Step>

  <Step>
    Escolha um restaurante.

    <img
      title="Choose a restaurant"
      alt="Choose a restaurant"
      src={chooseRestaurant}
    />
  </Step>

  <Step>
    Selecione um ou dois itens e clique no carrinho.

    <img
      title="select food"
      alt="select food"
      src={selectFood}
    />
  </Step>

  <Step>
    Clique em <DoNotTranslate>**PAY**</DoNotTranslate>.

    <img
      title="checkout"
      alt="checkout"
      src={checkout}
    />
  </Step>

  <Step>
    Insira as seguintes informações de cartão falso e clique em <DoNotTranslate>**Finish payment**</DoNotTranslate> para fazer seu pedido.

    <img
      title="place order with fake card"
      alt="place order with fake card"
      src={placeOrder}
    />

    Seu pedido foi realizado com sucesso.

    <img
      title="Purchase completed"
      alt="Purchase completed"
      src={thankYou}
    />

    A seguir, você usa um simulador para gerar mais tráfego para seu aplicativo.
  </Step>

  <Step>
    Em outra janela do terminal, navegue até o diretório raiz do seu aplicativo e execute o gerador de carga.

    ```bash
    # Navigate to the root directiory of your simulator
    cd relicstaurants/simulator
    # Switch to lab branch
    git switch browser-pro-lab-material
    [output]
    # Install the simulator's dependencies
    pip3 install -r requirements.txt
    [output]
    # Run the simulator
    python3 simulator.py
    [output] ====== WebDriver manager ======
    [output] Current google-chrome version is 99.0.4844
    [output] Get LATEST chromedriver version for 99.0.4844 google-chrome
    ```

    <Callout variant="important">
      Este gerador de carga pressupõe que você tenha o Google Chrome instalado em seu computador. Se você estiver usando um browser diferente, pule esta etapa e gere tráfego manualmente ou [instale o Google Chrome](https://www.google.com/chrome/downloads/).
    </Callout>
  </Step>
</Steps>

Agora que você sabe como executar sua aplicação, é hora de instrumentá-la. Nas janelas do terminal que estão executando seu aplicativo e simulador, pressione `<CTRL-C>` para encerrá-los. Com seu aplicativo encerrado, você poderá atualizar seu código para introduzir ferramentas de monitoramento.
