## Sintaxe

```js
newrelic.consent(accept?: boolean)
```

Permite ou rejeita o consentimento para que o agente do browser envie dados pela rede.

## Requisitos

* Agente Browser Lite, Pro ou Pro+SPA (versão 1.303.0 ou superior)

## Descrição

O agente do browser possui uma funcionalidade opcional de modo de consentimento. Quando o modo de consentimento é ativado por meio das configurações do aplicativo ou com o NerdGraph, o usuário pode chamar o método `newrelic.consent(...)` para permitir ou negar o consentimento sobre se os dados são ou não enviados pela rede.

Para obter mais informações sobre configuração e utilização, consulte nossa [documentação sobre o modo de consentimento](/docs/browser/new-relic-browser/configuration/consent-mode).

## Casos de uso

* Você deseja esperar para coletar dados até que um usuário concorde com um modal em seu site

## Exemplos

### O usuário aceita o consentimento para que o agente do browser colete dados.

```js
newrelic.consent()
```

### Um usuário rejeita o consentimento do agente do browser para coletar dados

```js
newrelic.consent(false)
```
