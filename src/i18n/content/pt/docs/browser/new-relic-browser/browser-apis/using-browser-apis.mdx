A API do browser permite personalizar e estender seu <InlinePopover type="browser"/>. Para começar a usar a API, certifique-se de que o [agente browser esteja presente em todas as páginas que você deseja monitor](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent).

Abaixo estão os casos de uso para cada método de API do browser.

## Rastreamento e evento [#api-traces]

Para monitor rastreamento e evento, use estes métodos:

      <td>
        Rastreie uma ação que a New Relic não rastreia automaticamente
      </td>

      <td>
        [`newrelic.addPageAction()`](/docs/browser/new-relic-browser/browser-apis/addpageaction)
      </td>
    </tr>

    <tr>
      <td>
        Adicionar um objeto a uma [sessão de rastreamento](/docs/browser/new-relic-browser/browser-pro-features/session-traces-exploring-webpages-life-cycle)em andamento
      </td>

      <td>
        [`newrelic.addToTrace()`](/docs/browser/new-relic-browser/browser-apis/addtotrace)
      </td>
    </tr>

    <tr>
      <td>
