## Sintaxe

```js
newrelic.pauseReplay()
```

API do Browser usada para forçar um replay a pausar a gravação.

## Requisitos

* Browser Pro ou agente Pro+SPA (v1.240.0 ou superior)

## Descrição

`newrelic.pauseReplay()` pode ser chamado para forçar manualmente um replay para pausar a gravação.

Se uma repetição ainda não estiver em andamento, este método de API não terá efeito.

Se a sessão do navegador ainda for válida, você poderá retomar uma reprodução pausada chamando o método de API [`newrelic.recordReplay()`](/docs/browser/new-relic-browser/browser-apis/recordReplay).

## Casos de uso

* Você deseja acionar manualmente uma repetição para interromper a gravação em determinadas páginas do site ou sob determinadas condições, como uma página de saúde, página de informações de cartão de crédito, etc.

## Exemplos

### Forçando um replay para pausar a gravação

```js
newrelic.pauseReplay()
```

### Retomando uma reprodução pausada

```js
  newrelic.pauseReplay()
  newrelic.recordReplay()
```
