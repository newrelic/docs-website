Para começar com [<InlinePopover type="browser"/>](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser), vá para <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT> e selecione o browser na parte superior da página. Se você já adicionou aplicativos, o índice de aplicativos do browser aparecerá automaticamente.

A página de índice do browser fornece uma lista dos aplicativos existentes que estão sendo monitorados pelo browser. É também o ponto de partida para [instalar o browser agente](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent) para aplicativos adicionais.

## Recurso de índice de aplicativos do browser [#index]

<img
  title="Browser index UI page"

<figcaption>
  O índice do aplicativo do browser mostra os nomes dos seus aplicativos, seu status de saúde e informações resumidas, com links diretos para páginas adicionais.
</figcaption>

O índice mostra uma lista de nomes de aplicativos, status de saúde codificados por cores e resumos da experiência do usuário com base no período atual selecionado no seletor de hora. Isso inclui:

* [tempo de carregamento da página](/docs/browser/new-relic-browser/dashboard-details/page-views-dashboard), [tempo de resposta do SPA](/docs/browser/single-page-app-monitoring/use-spa-data/view-spa-data-new-relic-browser) e taxas de transferência (páginas por minuto)
* [AJAX tempo de resposta](/docs/browser/new-relic-browser/dashboard-details/ajax-dashboard) e taxas de transferência (solicitações por minuto)
* [Erros de JavaScript](/docs/errors-inbox/browser-tab)

## Nome do aplicativo do browser [#app-name]

Os aplicativos que também são monitorados pelo [APM](/docs/apm/new-relic-apm/getting-started/welcome-new-relic-apm) recebem seu nome das [configurações do aplicativo no APM](/docs/apm/new-relic-apm/installation-configuration/naming-your-application). Para aplicativos monitorados por browser e não por APM, o nome deles é aquele fornecido quando você [instalou o browser](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent).

* Para visualizar a [página Resumo](/docs/browser/new-relic-browser/dashboard-details/browser-overview-dashboard) de um aplicativo de browser: No índice do aplicativo de browser, selecione seu nome.
* Para alterar o nome de um aplicativo de browser: Siga os procedimentos para [renomear seu aplicativo](/docs/browser/new-relic-browser/installation-configuration/rename-browser-apps) com base no método de implantação original.

## Status do aplicativo [#status]

O [indicador de integridade](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#health-status) codificado por cores mostra o status atual de cada aplicativo:

* <DNT>
    **Green:**
  </DNT>

  O monitoramento do browser foi habilitado e o aplicativo está bom.

* <DNT>
    **Yellow:**
  </DNT>

  Um alerta de aviso indica que o aplicativo está enfrentando alguns problemas não críticos; por exemplo, a [pontuação Apdex](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) do usuário final ou a taxa de erros ultrapassaram um limite definido para a política de alertas associada.

* <DNT>
    **Red:**
  </DNT>

  Um aviso Crítico indica que um problema aumentou com o aplicativo.

* <DNT>
    **Gray:**
  </DNT>

  Nenhum dado está sendo relatado para o aplicativo no momento.

Para um rápido resumo das condições atuais, passe o mouse sobre a barra de status colorida do aplicativo.
