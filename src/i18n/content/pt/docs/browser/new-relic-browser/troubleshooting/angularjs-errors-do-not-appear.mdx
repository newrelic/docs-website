## Problema

Você está usando AngularJS e deseja relatar erros de JavaScript ao New Relic, mas não vê os erros do seu aplicativo na [página<DNT>**JavaScript errors**</DNT> ](/docs/browser/new-relic-browser/browser-pro-features/js-errors-dashboard-examining-errors-over-time).

## Solução

O aplicativo AngularJS geralmente inclui um manipulador de erros global que captura todos os erros e, por padrão, <InlinePopover type="browser"/>relata apenas erros não detectados.

Use a chamada de API [noticeError](/docs/browser/new-relic-browser/browser-agent-spa-api/newrelicnoticeerror-browser-agent-api) para relatar os erros à New Relic. Isso permite que você os trate como parte do processamento regular de erros, mas também os veja na interface do usuário do New Relic.
