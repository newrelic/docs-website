## Problema

Não estou vendo dados em <DNT>**Web vitals**</DNT>.

## Solução

Aqui estão algumas causas de dados ausentes:

* Seus filtros são muito estreitos. Alguns atributos principais são relevantes apenas para determinados sinais vitais da web. Por exemplo, se você filtrasse por `elementID`, os dados seriam preenchidos apenas para pintura com maior conteúdo (LCP), porque `elementID` só é aplicável à medição de LCP.
* Seu browser ou edição do agente não coleta determinados dados. Para saber qual atributo chave o agente New Relic Browser coleta, consulte o [dicionário de dados do agente browser](/attribute-dictionary/?dataSource=Browser+agent).
* Seu aplicativo não pode enviar dados `PageViewTiming` para o New Relic. O Web vitals é baseado em dados `PageViewTiming` Portanto, se por algum motivo seu aplicativo não conseguir enviar a ingestão ao nosso coletor, os dados poderão não aparecer.
