## Problema

O suporte da New Relic requer detalhes adicionais de resolução de problemas sobre o que está acontecendo no navegador e você não pode fornecer um URL acessível ou o código-fonte da sua página.

## Solução

Com base no seu tipo de navegador, crie um arquivo [HAR (HTTP Archive)](https://dvcs.w3.org/hg/webperf/raw-file/tip/specs/HAR/Overview.html) para investigação e envie-o para [support.newrelic.com](https://support.newrelic.com/). O arquivo HAR contém todas as informações sobre chamadas de rede envolvidas em uma página durante o período registrado.

Ao gravar o arquivo HAR, utilize o recurso do seu site para gerar o tipo de atividade adequado. Por exemplo, se você tiver problemas com AJAX, exercite a seção da sua página que faz chamadas AJAX.

<Callout variant="important">
  Os arquivos HAR capturam todo o tráfego da rede durante o tempo em que estão sendo gravados. Esses dados de tráfego podem incluir informações privadas ou de identificação pessoal (cookies, senhas, números de cartão de crédito, etc.). Os clientes são responsáveis por quaisquer dados que forneçam à New Relic, inclusive dentro de arquivos HAR. Os arquivos HAR podem ser editados com um editor de texto. Antes de compartilhar um arquivo HAR, recomendamos remover manualmente qualquer informação confidencial.
</Callout>

<CollapserGroup>
  <Collapser
    id="chrome"
    title="Chrome"
  >
    O navegador Google Chrome vem com a capacidade de gerar um arquivo HAR. Dependendo da versão do seu navegador, os nomes dos caminhos podem variar para abrir o console.

    1. No menu do Chrome (canto superior direito), selecione

       <DoNotTranslate>**Tools > Developer tools**</DoNotTranslate>

       ou

       <DoNotTranslate>**More tools > Developer tools**</DoNotTranslate>

       conforme aplicável.

    2. No console, selecione

       <DoNotTranslate>**Network**</DoNotTranslate>

       .

    3. Selecione o ícone de registro.

    4. Se aplicável, marque a caixa de seleção

       <DoNotTranslate>**Preserve log**</DoNotTranslate>

       para salvar os dados de transação da rede.

    5. Antes de tentar reproduzir o problema, selecione o ícone limpar para remover informações desnecessárias do cabeçalho.

    6. Navegue pelo seu site para gerar o tipo de tráfego com problemas.

    7. Aguarde alguns minutos até que o navegador reporte os dados ao coletor New Relic.

    8. Salve as informações capturadas clicando com o botão direito na lista de transações na parte inferior do console e selecionando

       <DoNotTranslate>**Save as HAR with Content**</DoNotTranslate>

       .
  </Collapser>

  <Collapser title="Firefox versão 41.x e superior">
    O Firefox versão 41 ou superior vem com a capacidade de copiar a transação para um arquivo HAR.

    1. No menu Firefox (canto superior direito), selecione

       <DoNotTranslate>**Developer > Network**</DoNotTranslate>

    2. Selecione a opção

       <DoNotTranslate>**Clear**</DoNotTranslate>

       (canto inferior direito da janela dos desenvolvedores) para remover informações desnecessárias do cabeçalho.

    3. Navegue pelo seu site para gerar o tipo de tráfego com problemas.

    4. Aguarde alguns minutos até que o navegador reporte os dados ao coletor New Relic.

    5. Capture as informações clicando com o botão direito na lista de transações no console e selecionando

       <DoNotTranslate>**Copy All As HAR**</DoNotTranslate>

       para colocar os dados em seu buffer ou

       <DoNotTranslate>**Save All As HAR**</DoNotTranslate>

       para copiá-los para um arquivo.
  </Collapser>

  <Collapser title="Microsoft borda">
    1. Para abrir

       <DoNotTranslate>**Developer Tools**</DoNotTranslate>

       , pressione a tecla F12 no teclado ou localize

       <DoNotTranslate>**Developer Tools**</DoNotTranslate>

       no menu.

    2. Selecione a guia

       <DoNotTranslate>**Network**</DoNotTranslate>

       .

    3. Atualize a página para começar a capturar o tráfego entre o navegador e o servidor ou clique em um link da página afetada.

    4. Selecione

       <DoNotTranslate>**Export as HAR**</DoNotTranslate>

       seguido de

       <DoNotTranslate>**Save As...**</DoNotTranslate>

       para salvar o arquivo HAR.
  </Collapser>

  <Collapser title="Safári da Apple">
    1. Ative o menu Desenvolvedor: na barra de menus do macOS, vá para

       <DoNotTranslate>**Safari > Preferences > Advanced > Show Develop menu in menu bar**</DoNotTranslate>

       .

    2. Na barra de menu do macOS, vá para a guia

       <DoNotTranslate>**Develop > Show Web Inspector > Network**</DoNotTranslate>

       .

    3. Na guia

       <DoNotTranslate>**Network**</DoNotTranslate>

       , marque a opção

       <DoNotTranslate>**Preserve log**</DoNotTranslate>

       .

    4. Atualize a página e permita que

       <DoNotTranslate>**Safari**</DoNotTranslate>

       registre a interação navegador-site.

    5. Depois que a página for carregada, selecione

       <DoNotTranslate>**Export**</DoNotTranslate>

       no canto superior direito da janela da guia Rede.
  </Collapser>
