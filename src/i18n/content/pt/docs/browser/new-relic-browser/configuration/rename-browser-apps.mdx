Siga o procedimento apropriado para renomear seu aplicativo de browser dependendo da [opção de implantação](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser) usada para adicioná-lo ao browser.

## Renomear monitor de aplicativo por APM [#apm-app]

Se você adicionou um aplicativo ao browser que já está sendo monitorado pelo APM, o nome do aplicativo que aparece no browser será igual ao nome do aplicativo na interface <InlinePopover type="apm"/>. Para renomear o aplicativo, use qualquer uma destas opções:

## Renomear aplicativo via trecho JavaScript [#javascript]

Esta informação se aplica se você inseriu o trecho JavaScript do agente browser em:

* App monitor da APM que foi implantado pelo [método copiar/colar](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#copy-paste-app)
* Aplicativo de página única ([SPA](/docs/browser/single-page-app-monitoring/get-started/add-apps-single-page-app-monitoring))
* Qualquer aplicativo de browser independente que também não seja monitorado pelo APM

Você não pode editar diretamente o trecho JavaScript no aplicativo existente. Em vez disso, use seu aplicativo com um nome diferente:

1. Siga os procedimentos padrão para [criar um novo aplicativo usando o método copiar/colar](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#copy-paste-app). Forneça o novo nome e gere o trecho JavaScript.
2. Copie o novo trecho de JavaScript.
3. Na sua página da Web, substitua o JavaScript do seu aplicativo original pelo JavaScript gerado para o novo nome do aplicativo.
4. Se o seu aplicativo independente tiver [vários nomes](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), [copie os IDs do aplicativo personalizado](/docs/browser/new-relic-browser/installation-configuration/copy-browser-monitoring-license-key-app-id) no seu script de configuração para o seu novo aplicativo.

Com base no [cronograma de retenção de dados](/docs/accounts-partnerships/accounts/account-billing-usage/data-retention), a New Relic reterá os dados históricos coletados com o nome antigo e coletará e agregará dados com o novo nome.

## Renomear aplicativo por meio da API REST New Relic (v2) [#rest-api]

Estas informações se aplicam apenas a aplicativos de browser que também são monitorados pelo APM. Você não poderá alterar o nome do aplicativo por meio da API REST se tiver usado o método copiar/colar para inserir o trecho JavaScript do browser em seu aplicativo.

Siga os procedimentos padrão para [alterar o alias do aplicativo APM por meio da API REST](/docs/apis/rest-api-v2/application-examples-v2/change-alias-your-application-v2) ou [do API Explorer](https://rpm.newrelic.com/api/explore/applications/update). O New Relic altera automaticamente o nome de exibição do aplicativo em todos os lugares da interface do usuário do New Relic, incluindo o browser.
