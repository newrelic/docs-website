O tipo de agente browser não tem impacto no seu faturamento. É simplesmente uma maneira de mudar para um conjunto de recursos mais mínimo, se desejado. O agente browser padrão instalado vem com todos os recursos <InlinePopover type="browser" />habilitados.

Para alterar o tipo de edição do seu browser, vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; Settings &gt; Application settings**</DNT>.

Aqui estão algumas comparações entre os diferentes tipos de browsers agentes:

## Tipos de browser de agente [#agent-types]

Temos três tipos de browser agente: Lite, Pro e Pro+SPA.

<table>
  <thead>
    <tr>
      <th style={{ width: "150px" }}>
        Tipo de agente browser
      </th>

      <th>
        Comparação
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Pró+SPA
      </td>

      <td>
        Este é o agente padrão instalado quando você habilita o monitoramento do Browser. Dá acesso a todos os recursos de monitoramento do Browser.

        Pro+SPA não é útil apenas para aplicativos de página única. Após a instalação, você poderá fazer downgrade para os tipos de agente inferiores se não desejar ou precisar do monitoramento do SPA.
      </td>
    </tr>

    <tr>
      <td>
        Pró
      </td>

      <td>
        Esta versão do agente não possui o [recurso destinado ao monitoramento de SPA](/docs/browser/single-page-app-monitoring/get-started/introduction-single-page-app-monitoring).
      </td>
    </tr>

    <tr>
      <td>
        Lite
      </td>

      <td>
        Esta versão do agente relata alguns tempos básicos de carregamento da página e informações do usuário do browser, mas carece de muitos dos recursos mais avançados de monitoramento de browser e também carece do recurso de monitoramento de SPA.
      </td>
    </tr>
  </tbody>
</table>
