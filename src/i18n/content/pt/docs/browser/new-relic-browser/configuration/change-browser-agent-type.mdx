O tipo de agente browser não tem impacto no seu faturamento. É simplesmente uma maneira de mudar para um conjunto de recursos mais mínimo, se desejado. O agente browser padrão instalado vem com todos os recursos <InlinePopover type="browser"/>habilitados.

Para alterar o tipo de edição do seu browser, vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > Settings > Application settings**</DNT>.

Aqui estão algumas comparações entre os diferentes tipos de browsers agentes:

## Tipos de browser de agente [#agent-types]

Temos três tipos de browser agente: Lite, Pro e Pro+SPA.
