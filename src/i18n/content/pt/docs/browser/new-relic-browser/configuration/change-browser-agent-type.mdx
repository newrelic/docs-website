O tipo de navegador do agente não tem impacto no seu faturamento. É simplesmente uma maneira de mudar para um conjunto de recursos mais mínimo, se desejado. O agente navegador padrão instalado vem com todos os recursos <InlinePopover type="browser"/>habilitados.

Para alterar o tipo de edição do seu navegador, vá para <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > Settings > Application settings**</DoNotTranslate>.

Aqui estão algumas comparações entre os diferentes tipos de navegadores agentes:

## Tipos de navegador de agente [#agent-types]

Temos três tipos de navegador agente: Lite, Pro e Pro+SPA.
