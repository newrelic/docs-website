Como parte dos <InlinePopover type="browser"/>[requisitos de configuração](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements), você deve [instalar o agente browser](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). Depois você pode usar a página <DoNotTranslate>**Applications settings**</DoNotTranslate> do seu aplicativo para atualizar as opções de monitoramento do browser para:

* Nível de assinatura do browser

* [Valores Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)do browser

* Cidades, regiões ou países a serem destacados na [página](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)

  <DoNotTranslate>[**Geography**](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)</DoNotTranslate>

## Selecione configurações de Apdex e geografia [#apdex-settings]

Para aplicativos que também são monitorados por APM, você pode definir [valores de Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) separados para browser (usuário final) e para [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex). Você também pode selecionar cidades, regiões ou países a serem destacados na página <DoNotTranslate>**Geography**</DoNotTranslate> .

1. Vá para

   <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >**</DoNotTranslate>

   (selecione um aplicativo)

   <DoNotTranslate>**> Settings > Application settings**</DoNotTranslate>

   .

2. A partir de

   <DoNotTranslate>**Application settings**</DoNotTranslate>

   , selecione um valor de Apdex T específico para monitoramento de Browser, ou deixe o valor de Apdex padrão.

3. Selecione até cinco países ou regiões para monitoramento do browser destacar na página

   <DoNotTranslate>**Geography**</DoNotTranslate>

   . Se você estiver relatando dados para [vários nomes de aplicativos](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), selecione os mesmos países nos aplicativos primário e secundário.

4. Selecione

   <DoNotTranslate>**Save application settings**</DoNotTranslate>

   .

Sempre que você altera as configurações do Apdex, uma barra vertical preta aparece no gráfico <DoNotTranslate>**Page load timing**</DoNotTranslate> da página <DoNotTranslate>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DoNotTranslate> do seu aplicativo de browser.

## Ver Apdex do usuário final [#view-browser-apdexs]

Para visualizar sua pontuação Apdex:

* Vá para <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DoNotTranslate>.

  OU

* Vá para <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DoNotTranslate>.
