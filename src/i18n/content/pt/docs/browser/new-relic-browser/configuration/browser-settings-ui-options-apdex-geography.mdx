Como parte dos <InlinePopover type="browser"/>[requisitos de configuração](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements), você deve [instalar o agente browser](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). Depois você pode usar a página <DNT>**Applications settings**</DNT> do seu aplicativo para atualizar as opções de monitoramento do browser para:

* Nível de assinatura do browser

* [Valores Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)do browser

* Cidades, regiões ou países a serem destacados na [página](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)

  <DNT>
    [**Geography**](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)
  </DNT>

## Selecione configurações de Apdex e geografia [#apdex-settings]

Para aplicativos que também são monitorados por APM, você pode definir [valores de Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) separados para browser (usuário final) e para [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex). Você também pode selecionar cidades, regiões ou países a serem destacados na página <DNT>**Geography**</DNT> .

1. Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >**
   </DNT>

   (selecione um aplicativo)

   <DNT>
     **> Settings > Application settings**
   </DNT>

   .

2. A partir de

   <DNT>
     **Application settings**
   </DNT>

   , selecione um valor de Apdex T específico para monitoramento de Browser, ou deixe o valor de Apdex padrão.

3. Selecione até cinco países ou regiões para monitoramento do browser destacar na página

   <DNT>
     **Geography**
   </DNT>

   . Se você estiver relatando dados para [vários nomes de aplicativos](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), selecione os mesmos países nos aplicativos primário e secundário.

4. Selecione

   <DNT>
     **Save application settings**
   </DNT>

   .

Sempre que você altera as configurações do Apdex, uma barra vertical preta aparece no gráfico <DNT>**Page load timing**</DNT> da página <DNT>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> do seu aplicativo de browser.

## Ver Apdex do usuário final [#view-browser-apdexs]

Para visualizar sua pontuação Apdex:

* Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT>.

  OU

* Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DNT>.
