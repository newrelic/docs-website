Se precisar customizar como o agente browser coleta dados, você tem dois métodos: configurar a interface ou usar API para customizar a instrumentação do agente.

## Configuração de interface [#UI]

Na página <DNT>**Application settings**</DNT> , você pode fazer estas configurações em sua configuração <InlinePopover type="browser"/>:

* [Código do agente proxy e dados analíticos](/docs/browser/new-relic-browser/configuration/proxy-agent-requests/)
* [Configurar alerta](/docs/browser/new-relic-browser/configuration/view-browser-apps-alert-information/)
* [Definir configurações de APDEX e geografia](/docs/browser/new-relic-browser/configuration/browser-settings-ui-options-apdex-geography/#apdex-settings)
* [Filtrar domínio e subdomínio](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains/)
* [Filtrar solicitações AJAX](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events/)
* [Agrupar dados por padrões de URL](/docs/browser/new-relic-browser/configuration/group-browser-metrics-urls/)
* [Ofuscar dados](/docs/browser/new-relic-browser/configuration/obfuscate-browser-agent-data/)
* [Renomear aplicativos do browser](/docs/browser/new-relic-browser/configuration/rename-browser-apps/)
* [Desativar monitoramento do browser](/docs/browser/new-relic-browser/installation/disable-browser-monitoring/)
* [Excluir um aplicativo de browser](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/#remove)

Para acessar a página <DNT>**Application settings**</DNT> :

1. Vá para

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**
   </DNT>

   e selecione um aplicativo.

2. Na navegação esquerda, role para baixo até

   <DNT>
     **Settings**
   </DNT>

   e clique em

   <DNT>
     **Application settings**
   </DNT>

   .

## API do browser [#apis]

Você também pode personalizar a instrumentação de agente do seu browser para capturar rastreamentos e eventos personalizados, relatar erros específicos adaptados às suas necessidades e ajustar o tempo de coleta de dados para obter desempenho ideal. Consulte [Usando API do browser](/docs/browser/new-relic-browser/browser-apis/using-browser-apis/) para começar.
