Se precisar customizar como o navegador do agente coleta dados, você tem dois métodos: configurar a interface ou usar API para customizar a instrumentação do agente.

## Configuração de interface [#UI]

Na página <DoNotTranslate>**Application settings**</DoNotTranslate> , você pode fazer estas configurações em sua configuração <InlinePopover type="browser"/>:

* [Código do agente proxy e dados analíticos](/docs/browser/new-relic-browser/configuration/proxy-agent-requests/)
* [Configurar alerta](/docs/browser/new-relic-browser/configuration/view-browser-apps-alert-information/)
* [Definir configurações de APDEX e geografia](/docs/browser/new-relic-browser/configuration/browser-settings-ui-options-apdex-geography/#apdex-settings)
* [Filtrar domínio e subdomínio](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains/)
* [Filtrar solicitações AJAX](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events/)
* [Agrupar dados por padrões de URL](/docs/browser/new-relic-browser/configuration/group-browser-metrics-urls/)
* [Ofuscar dados](/docs/browser/new-relic-browser/configuration/obfuscate-browser-agent-data/)
* [Renomear aplicativos do navegador](/docs/browser/new-relic-browser/configuration/rename-browser-apps/)
* [Desativar monitoramento do navegador](/docs/browser/new-relic-browser/installation/disable-browser-monitoring/)
* [Excluir um aplicativo de navegador](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/#remove)

Para acessar a página <DoNotTranslate>**Application settings**</DoNotTranslate> :

1. Vá para

   <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**</DoNotTranslate>

   e selecione um aplicativo.

2. Na navegação esquerda, role para baixo até

   <DoNotTranslate>**Settings**</DoNotTranslate>

   e clique em

   <DoNotTranslate>**Application settings**</DoNotTranslate>

   .

## APIdo navegador [#apis]

Você também pode personalizar a instrumentação de agente do seu navegador para capturar rastreamentos e eventos personalizados, relatar erros específicos adaptados às suas necessidades e ajustar o tempo de coleta de dados para obter desempenho ideal. Consulte [Usando APIdo navegador](/docs/browser/new-relic-browser/browser-apis/using-browser-apis/) para começar.
