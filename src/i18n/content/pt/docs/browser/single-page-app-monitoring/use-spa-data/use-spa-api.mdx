O monitoramento de aplicativo de página única (SPA) do browser inclui uma API para adicionar monitoramento personalizado de interação específica do browser. Isso é útil para interações de monitoramento que o New Relic não registra automaticamente porque não resultam em alterações de rota, como um widget atualizado dinamicamente. A API do SPA também permite desativar o monitoramento padrão para interações que você não considera importantes o suficiente para serem monitor.

Para obter mais informações sobre a API do SPA, incluindo chamada de API específica, consulte o [agente browser e a API do SPA](/docs/browser/new-relic-browser/guides/guide-using-browser-spa-apis/).
