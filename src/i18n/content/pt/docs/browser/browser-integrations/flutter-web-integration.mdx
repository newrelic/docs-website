Nossa integração Flutter monitora o desempenho de seu aplicativo Web Flutter, ajudando você a diagnosticar problemas em seu aplicativo e otimizar seu código. Nossa integração com o Flutter usa nosso <InlinePopover type="browser"/>agente e oferece um dashboard pré-criado com as métricas mais importantes do seu aplicativo Flutter (como erros, falhas no aplicativo, lançamento do aplicativo, instalação do aplicativo, erros de HTTP e falhas de rede).

## Instalar [#install]

### Etapa 1 (opcional): Instalar o agente de infraestrutura [#infra-install]

A instalação do agente de infraestrutura é opcional, mas isso fornecerá métricas importantes sobre o host onde seu aplicativo está sendo executado.

Para fazer isso, siga [as etapas de instalação do agente de infraestrutura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent) para o host que contém seu aplicativo Flutter.

### Etapa 2: Instale o agente browser [#browser-install]

Nossa integração com o Flutter depende do nosso browser agente para funcionar. Para instalar nosso browser agente:

1. De [one.newrelic.com](https://one.newrelic.com), vá para a [página](https://one.newrelic.com/marketplace)

   <DNT>
     [**Integrations & Agents**](https://one.newrelic.com/marketplace)
   </DNT>

   [](https://one.newrelic.com/marketplace).

2. Clique em

   <DNT>
     **Browser and mobile**
   </DNT>

   e em

   <DNT>
     **Browser monitoring**
   </DNT>

   .

3. Selecione a opção

   <DNT>
     **Select copy/paste JavaScript code**
   </DNT>

   (para saber mais sobre esse código, consulte [nossos documentos de instalação do browser](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent)).

4. Conclua o restante das etapas e clique em

   <DNT>
     **Enable.**
   </DNT>

5. Acesse o diretório `www/root` do seu aplicativo Web Flutter.

6. Abra o arquivo `index.html` .

7. Cole o trecho do código do nosso browser logo antes do término da tag head (`</head>`).

8. Reinicie seu aplicativo.

Aguarde cinco minutos e prossiga para encontrar seus dados no New Relic.

## Encontre seus dados [#find-data]

Para obter seu dashboard Flutter:

1. De [one.newrelic.com](https://one.newrelic.com), vá para a [página](https://one.newrelic.com/marketplace)

   <DNT>
     [**Integrations & Agents**](https://one.newrelic.com/marketplace)
   </DNT>

   [](https://one.newrelic.com/marketplace).

2. Clique em

   <DNT>
     **Dashboards**
   </DNT>

   .

3. Na barra de pesquisa, digite `flutter web`.

4. O dashboard do Flutter deve aparecer. Clique nele para instalá-lo.

Seu dashboard Flutter é considerado um painel personalizado e pode ser encontrado na interface <DNT>**Dashboards**</DNT>. Para obter documentos sobre como usar e editar o painel, consulte [nossos documentos dashboard ](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).

Essa integração reporta dados na forma de nosso [evento de monitoramento de Browser](/docs/data-apis/understand-data/event-data/events-reported-browser-monitoring). Se você instalou o agente de infraestrutura, também receberá [dados de infraestrutura](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-monitoring-data/#infrastructure-events).

Aqui está um exemplo de consulta NRQL verificando o tempo médio de carregamento da janela para os diversos browsers usados pelo usuário em computadores desktop:

```sql
SELECT average(windowLoad) FROM PageViewTiming 
WHERE (deviceType = 'Desktop') FACET `userAgentName` 
SINCE 604800 seconds AGO TIMESERIES
```
