Com nosso dashboard Nuxt, você pode facilmente rastrear a taxa de erros do seu aplicativo, a média de carregamento inicial da página, as alterações de rota, a taxa de transferência e as visualizações de página com erros JavaScript. Criado com nosso agente de infraestrutura e nosso Vue <InlinePopover type="browser" />, o painel Nuxt permite que você visualize informações críticas em um só lugar.

## Instalar [#install]

<Steps>
  <Step>
    ### Opcional: Instale o agente de infraestrutura [#infra-install]

    A instalação do agente de infraestrutura é opcional, mas isso fornecerá métricas importantes sobre o host onde seu aplicativo está sendo executado.

    Para fazer isso, siga [as etapas de instalação do agente de infraestrutura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent).
  </Step>

  <Step>
    ### Instalar o monitoramento do navegador Nuxt [#browser-install]

    1. De <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT>, vá para a [página<DNT>**+ Integrations &amp; Agents**</DNT> ](https://one.newrelic.com/marketplace).
    2. Pesquise por <DNT>**Nuxt**</DNT>. (Observação: o bloco Nuxt não está na coleção de navegador ou celular, então você precisará usar a função de pesquisa.)
    3. Clique no bloco <DNT>**Nuxt**</DNT> quando ele aparecer nos resultados da pesquisa e siga as instruções para gerar o snippet de código JavaScript.
  </Step>

  <Step>
    ### Configurar a integração [#config]

    Para adicionar o snippet JavaScript no aplicativo Nuxt:

    1. Crie um arquivo JavaScript na pasta <DNT>**`public`**</DNT> do seu aplicativo Nuxt:

       ```bash
       touch public/<FILENAME>.js
       ```

    2. Copie e cole o snippet do agente do navegador no novo arquivo JavaScript sem as tags `<script>` (apenas o snippet em si, não há função para definir):

       ```js
       // PASTE THE GENERATED BROWSER AGENT SNIPPET HERE WITHOUT <script> TAGS
       ```

    3. Abra seu arquivo `nuxt.config.js` (ou `nuxt.config.ts`) e adicione o script à seção `app.head`:

       ```js
       export default defineNuxtConfig({
         compatibilityDate: '2025-07-15',
         devtools: { enabled: true },
         app: {
           head: {
             script: [
               {
                 src: "/<FILENAME>.js"
               },
             ],
           },
         },
       })
       ```

    4. Reinicie seu aplicativo.

    Em alguns minutos, seu aplicativo enviará métricas para [one.newrelic.com](https://one.newrelic.com).
  </Step>
</Steps>

## Monitor seu aplicativo [#monitor]

Agora você poderá ver seu aplicativo em <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> selecionando <DNT>**Browser**</DNT> na barra de navegação. Escolha sua aplicação para ver todas as métricas que são capturadas na plataforma <DNT>**one.newrelic.com**</DNT> . Você também pode visualizar os dados com NRQL usando a seguinte consulta:

```sql
SELECT firstPaint
FROM PageView
```
