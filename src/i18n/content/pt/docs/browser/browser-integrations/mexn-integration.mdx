As pilhas MEAN, MERN e MEVN são kits de desenvolvimento de software full stack de código aberto. Cada kit usa uma única base de código para desenvolver aplicativos web. A New Relic consolidou todas as três pilhas em um único início rápido. Nossa integração MExN usa o agente <InlinePopover type="browser"/>, o agente Node.js e a integração MongoDB para fornecer um dashboard predefinido com suas métricas mais importantes (como tempo de resposta, utilização de CPU, tráfego e frequências de login).

## Instalar [#install]

<Steps>
  <Step>
    ### Opcional: Instale o agente de infraestrutura [#infra-install]

    A instalação do agente de infraestrutura é opcional, mas isso fornecerá métricas importantes sobre o host onde seu aplicativo está sendo executado.

    Para fazer isso, siga [as etapas de instalação do agente de infraestrutura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent).
  </Step>

  <Step>
    ### Instale o início rápido do MongoDB

    1. Confira nossos requisitos do agente MongoDB em nossa [documentação](/docs/infrastructure/host-integrations/host-integrations-list/mongodb-monitoring-integration/) antes de instalar o agente.

    2. Abra a [instalação de início rápido do MongoDB](https://newrelic.com/instant-observability/mongodb).

    3. Clique em

       <DNT>
         **Install now**
       </DNT>

       para iniciar a instalação do agente MongoDB.
  </Step>

  <Step>
    ### Instale o agente Node.js

    1. Confira nossos requisitos de agente Node.js em nossa [documentação](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/).

    2. Abra a [instalação do início rápido do Node.js.](https://newrelic.com/instant-observability/node-js)

    3. Clique em

       <DNT>
         **Install now**
       </DNT>

       para iniciar a instalação do agente Node.js.
  </Step>

  <Step>
    ### Instale o agente do browser [#browser-install]

    1. De [one.newrelic.com](https://one.newrelic.com), vá para a [página](https://one.newrelic.com/marketplace)

       <DNT>
         [**+ Integrations & Agents**](https://one.newrelic.com/marketplace)
       </DNT>

       [](https://one.newrelic.com/marketplace).

    2. Clique em

       <DNT>
         **Browser and mobile**
       </DNT>

       e em

       <DNT>
         **Browser monitoring**
       </DNT>

       .

    3. Selecione a opção

       <DNT>
         **Select copy/paste JavaScript code**
       </DNT>

       (para saber mais sobre esse código, consulte [nossos documentos de instalação do browser](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent)).

    4. Conclua o restante das etapas e clique em

       <DNT>
         **Enable.**
       </DNT>

    5. Vá para o arquivo `index.html `do seu aplicativo React ou aplicativo Angular ou aplicativo Vue.

    6. Cole a tag trecho script do código do browser logo antes da tag body terminar (`</body>`).

    7. Reinicie seu aplicativo.

       Em alguns minutos, seu aplicativo enviará métricas para [one.newrelic.com](https://one.newrelic.com).
  </Step>
</Steps>

## Encontre seus dados [#find-data]

Para obter seu dashboard MExN:

1. De [one.newrelic.com](https://one.newrelic.com), vá para a página

   <DNT>
     **+Integrations & Agents**
   </DNT>

   .

2. Clique em

   <DNT>
     **Dashboards**
   </DNT>

   .

3. Na barra de pesquisa, digite `MExN`.

4. O dashboard MExN deve aparecer. Clique nele para instalá-lo.

Seu dashboard MExN é considerado um painel personalizado e pode ser encontrado na interface <DNT>**Dashboards**</DNT>. Para obter documentos sobre como usar e editar o painel, consulte [nossos documentos dashboard ](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).

Para obter informações sobre os dados relatados, consulte a documentação de cada uma das ferramentas instaladas:

* [Monitoramento de Evento do Browser](/docs/data-apis/understand-data/event-data/events-reported-browser-monitoring).
* [Dados de infraestrutura](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-monitoring-data/#infrastructure-events).
* [Dados do MongoDB](/docs/infrastructure/host-integrations/host-integrations-list/mongodb/mongodb-monitoring-integration-new/#find-and-use/).
* [Dados Node.js.](/docs/apm/agents/nodejs-agent/getting-started/introduction-new-relic-nodejs/)

Aqui está uma consulta NRQL verificando o tempo médio de carregamento da janela para os diversos browsers usados pelo usuário em computadores desktop:

```sql
SELECT average(windowLoad) 
FROM PageViewTiming 
WHERE (deviceType = 'Desktop') 
FACET `userAgentName` 
SINCE 604800 seconds 
AGO TIMESERIES
```
