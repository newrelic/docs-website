No Grafana, você pode configurar o New Relic como uma fonte de dados do Prometheus. Além disso, dentro do Grafana você pode consultar métricas armazenadas no New Relic usando a [linguagem de consulta PromQL](/docs/integrations/prometheus-integrations/view-query-data/supported-promql-features).

## Use o painel Grafana existente com New Relic [#Grafana-dashboards]

Ao integrar a métrica do Prometheus com o New Relic via [Remote Write](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#remote-write), [agente do Prometheus](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#agent) ou [integração OpenMetrics](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#OpenMetric) (2.0+) e configurar o New Relic como uma fonte de dados do Prometheus no Grafana, você pode usar o Grafana <InlinePopover type="dashboards"/>existente e aproveitar perfeitamente o monitoramento adicional, confiabilidade e escala que oferecemos.

## Compatibilidade e requisitos [#compat-requirements]

Antes de começar, certifique-se de ter concluído a integração da métrica do Prometheus e de estar executando uma versão recente do Grafana.

* Você deve ter o[agente](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#agent) [Remote Write](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#remote-write) , Prometheus ou a [integração OpenMetrics](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#OpenMetrics) (v2.0+) configurado antes de poder configurar fontes de dados New Relic Prometheus no Grafana.
* Você só pode configurar fontes de dados do New Relic Prometheus usando esse método no Grafana versões 6.7.0 ou mais recentes. Você precisará configurar cabeçalhos personalizados na interface, e isso não é possível em versões anteriores. Para obter detalhes, consulte [Configurar o New Relic como uma fonte de dados do Prometheus para Grafana](/docs/integrations/grafana-integrations/set-configure/configure-new-relic-prometheus-data-source-grafana).

### Suporte para PromQL

Nossa API do Prometheus emula as APIs de consulta do Prometheus. Oferecemos suporte à linguagem de consulta Prometheus (PromQL) por meio de nosso modo de consulta estilo PromQL. Fazemos o nosso melhor para traduzir automaticamente a consulta de sintaxe do PromQL para a aproximação mais próxima do NRQL.

Para obter mais informações sobre como isso funciona e as diferenças que você pode observar entre o Prometheus e o New Relic, consulte [Recurso PromQL compatível](/docs/integrations/prometheus-integrations/view-query-data/supported-promql-features).

## Obtenha fluxo de dados no Grafana [#get-data]

Para disponibilizar seus dados do New Relic no Grafana, você pode configurar uma fonte de dados Prometheus nova ou existente em apenas algumas etapas simples:

1. Na interface Grafana, adicione e [configure uma nova fonte de dados](/docs/integrations/grafana-integrations/set-configure/configure-new-relic-prometheus-data-source-grafana).
2. Salve a nova fonte de dados e comece a visualizar seus dados.
