## Importância do explorador de dados [#importance-of-data-explorer]

Você pesquisa o estado dos seus sistemas? Você precisa planejar recursos, identificar e responder a eventos de alerta ou solucionar comportamentos defeituosos? O explorador de dados facilita a identificação, busca e visualização dos dados que você procura por meio de menus visuais, sem nunca usar NRQL ou criar consultas.

Com o <DNT>**data explorer**</DNT>, você pode acessar todos os dados armazenados no banco de dados New Relic (NRDB) de forma rápida e intuitiva. Você pode então selecionar a faceta da sua consulta usando o atributo e filtrar até o valor necessário.

Outras coisas que a visualização do explorador de dados ajuda você a fazer:

* Visualize dados de diferentes perspectivas: de dados brutos a diversas visualizações que fornecem insights sobre evolução, distribuição e muito mais.
* Faça uma busca detalhada nos dados usando filtros.
* Visualize uma previsão com base na tendência dos seus dados (disponível com a visualização pública do [NRQL Predictions](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions)).
* Adicione suas pesquisas a um dashboard.
* Entenda como NRQL funciona: o Data Explorer gera automaticamente a consulta NRQL com base nas suas seleções.

## Explore seus dados [#explore-your-data]

Para acessar o Data Explorer, navegue até a barra inferior <DNT>**Query your data**</DNT> e clique no ícone <DNT>**Data Explorer**</DNT> .

### Use o explorador de dados [#use-data-explorer]

* **Defina o escopo**: selecione o tipo de dados (métrica, evento ou log), a conta e a entidade.
* **Selecione o intervalo de tempo**: Use o seletor de horas para definir o intervalo de tempo da sua consulta.
* **Crie sua pesquisa**: use os blocos à esquerda para navegar pelos dados disponíveis e construir sua pesquisa. Os blocos podem ser pesquisados e você pode usar as ações no menu de ações para criar e atualizar sua consulta.
* **Refine sua consulta**: modifique sua consulta adicionando filtros, facetas e agregações.
* **Personalize a visualização do gráfico**: atualize a visualização do seu gráfico a partir das opções de personalização disponíveis.
* **Adicionar previsão a um gráfico de linha ou área (disponível com a visualização pública do [NRQL Predictions](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions))**: Use <DNT>**Predict trend**</DNT> no menu **Options** para obter uma tendência prevista.
* **Adicionar ao dashboard**: adicione seu gráfico a um dashboard para referência futura.

### Componentes do explorador de dados [#data-explorer-components]

* **Selecione seu tipo de dados**: escolha entre métrica, evento, log e contas. Se você selecionar métrica, poderá filtrar por entidade.
* **Área de navegação de dados**: localizada à esquerda, esta área permite que você navegue pelos dados clicando nos diferentes valores exibidos na tela ou veja as possíveis ações clicando no menu de ações de 3 pontos.
* **Área de trabalho**: esta área exibe os resultados das suas seleções.
