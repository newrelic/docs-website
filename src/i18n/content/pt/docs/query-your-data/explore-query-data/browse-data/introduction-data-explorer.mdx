Navegue por todos os seus dados visualmente usando o <DNT>**Data Explorer**</DNT> sem precisar saber como escrever [consulta em formatoNRQL](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language). O Data Explorer faz parte do Console de consulta, acessível através da gaveta de consulta na parte inferior da plataforma New Relic.

Precisa realizar uma pesquisa mais detalhada? Leia como [consultar seus dados](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data/). Se ainda não o fez, crie sua conta New Relic gratuita abaixo para começar a monitorar seus dados hoje mesmo.

## Por que isso importa [#why-matters]

Você pesquisa o estado dos seus sistemas? Você precisa planejar recursos, identificar e responder a eventos de alerta ou solucionar comportamentos defeituosos? O Data Explorer facilita identificar, buscar e visualizar os dados que você procura por meio de menus visuais, sem nunca usar NRQL ou criar consultas.

Com o <DNT>**Data Explorer**</DNT> você pode acessar todos os dados armazenados no banco de dados New Relic (NRDB) de forma rápida e intuitiva. Você pode então selecionar <DNT>**Dimensions**</DNT> para facetar por dimensões específicas desses dados.

Outras coisas que o Data Explorer ajuda você a fazer:

* Veja dados de diferentes pontos de vista: desde dados brutos até diferentes visualizações que fornecem insights sobre evolução, distribuição e muito mais.
* Faça uma busca detalhada nos dados usando filtros.
* Adicione previsões a gráficos de linhas e áreas (disponível com a visualização pública do [NRQL Predictions](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions)).
* Adicione suas pesquisas a um [dashboard](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards).
* Entenda como [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) funciona: o Data Explorer mostra como as consultas são construídas.

## Tipos de dados que você pode explorar [#data-types]

No contexto de <DNT>**Data Explorer**</DNT>, `event` refere-se a todos os objetos armazenados no banco de dados New Relic (NRDB). O Data Explorer permite que você explore:

* Métrica:

  * [Métrica dimensional](/docs/data-apis/understand-data/new-relic-data-types/#dimensional-metrics) (o tipo de dados `Metric` ).
  * [métrica de fração de tempo data](/docs/data-apis/understand-data/new-relic-data-types/#timeslice-data): <InlinePopover type="apm" />agente, agente móvel e o agente <InlinePopover type="browser" />reportam este tipo de dados. Para explorar este tipo de dados, você deve escolher um monitor de entidade por um desses agentes, e então você verá essas opções.

* Eventos: o [tipo de dados do evento](/docs/data-apis/understand-data/new-relic-data-types/#events-new-relic).

* log: o tipo de dados `Log` (que você também pode explorar por meio de nossa [interface de log](/docs/logs/ui-data/use-logs-ui)).

* tabela de consulta: Dados que não estão presentes na sua conta New Relic, enviando arquivos CSV (que você também pode explorar através da nossa [interface de tabela de consulta](/docs/logs/ui-data/lookup-tables-ui)).

## Explore seus dados [#explore-data]

Para acessar o Data Explorer, clique no botão <DNT>**Query your data**</DNT> na gaveta inferior de qualquer página do New Relic e selecione a opção <DNT>**Data Explorer**</DNT>.

O Data Explorer consiste em:

* Seção de escopo na parte superior. Você pode selecionar a conta e escolher entre vários tipos de dados: `events`, `metrics`, `timeslices`, `logs` e `lookups`. Se você selecionar métrica, poderá filtrar por entidade.
* Área de navegação de dados à esquerda. Se você escolher `metrics`, as opções serão `metric` e `dimensions`. Se você escolher `events`, as opções serão `event type`, `plot` e `dimensions`.
* Área de trabalho. Esta área mostra o resultado de suas escolhas.

Para usar o Data Explorer:

1. Defina o escopo. Selecione o tipo de dados (métrica ou evento), a conta e a entidade.

2. Selecione o intervalo de tempo usando o [seletor de hora](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker).

3. Utilize os blocos à esquerda para navegar pelos dados disponíveis e construir sua pesquisa. Você só pode selecionar um elemento por bloco. Os blocos são pesquisáveis.

## Casos de uso [#uses-cases]

Veja os exemplos a seguir para saber como e quando usar o Data Explorer.

<CollapserGroup>
  <Collapser className="freq-link" id="use-case-validate-data" title="Descubra seus novos dados">
    <DNT>
      **I&apos;ve just connected new instrumentation and want to see if new data is available.**
    </DNT>

    1. Selecione a conta e o evento ou métrica que está gerando os novos dados.
    2. Use as diferentes ferramentas no Data Explorer para explorar os novos dados que se tornaram disponíveis: veja os dados brutos daquele evento ou métrica como uma tabela, modele-os como uma lista ou clique para vê-los plotados como um gráfico.
    3. Após selecionar um evento ou métrica, descubra o formato dos dados em suas dimensões. Guiado pela cardinalidade, você pode ver os diferentes pontos de vista de quaisquer dados.
    4. Encontrou algo relevante? Salve-o em um dashboard ou compartilhe-o com um colega.
  </Collapser>

  <Collapser className="freq-link" id="use-case-new-data" title="Valide seus dados">
    <DNT>
      **I changed a custom event/metric and need to check if this change has been successful.**
    </DNT>

    1. No Data Explorer, selecione a conta, o tipo de dados e o evento/métrica nos quais você fez alterações.
    2. Verifique se a entidade está reportando dados e se todos os atributo estão sendo plotados.
    3. Encontre o atributo no qual você fez alterações e verifique se a atualização foi bem-sucedida.
  </Collapser>

  <Collapser className="freq-link" id="nrql-prediction-use-case" title="Visualize a previsão em um gráfico de linha ou área (disponível com a visualização pública do NRQL Predictions)">
    <Callout title="visualização">
      Ainda estamos trabalhando nesse recurso, mas adoraríamos que você experimentasse!

      Atualmente, esse recurso é fornecido como parte de um programa de visualização de acordo com nossas [políticas de pré-lançamento](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
    </Callout>

    <DNT>
      **I want to see the future trend of a line or area chart to take proactive measures based on the predicted data.**
    </DNT>

    1. Acesse o Data Explorer clicando no botão <DNT>**Query your data**</DNT> na gaveta inferior de qualquer página do New Relic.
    2. Selecione um gráfico de linha ou área onde você deseja adicionar previsão.
    3. No menu <DNT>**Options**</DNT> , selecione <DNT>**Predict trend**</DNT>. O gráfico com a previsão abre em <DNT>**Data Explorer**</DNT>.
    4. Salve-o em um dashboard.
  </Collapser>

  <Collapser className="freq-link" id="use-case-troubleshooting" title="Resolução de problemas questões de dados">
    <DNT>
      **I know there&apos;s something off with an event/metric from an alert or dashboard. I need to know the root cause about the event/metric/attribute behavior.**
    </DNT>

    1. No Data Explorer, use os menus para selecionar o evento ou a métrica que não está se comportando como esperado e deixe o Data Explorer traçar esse gráfico.
    2. A partir daí, você pode detalhar as dimensões desses dados e filtrar pelos atributos que são relevantes.
    3. Você também pode ver esses dados de diferentes perspectivas: sua distribuição, classificação de valores ou evolução ao longo do tempo.
    4. Encontrou algo relevante? Salve-o em um dashboard ou compartilhe-o com um colega.
  </Collapser>
</CollapserGroup>
