Você pode usar [o painel](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) para visualizar e monitorar a integridade do seu sistema, resolver problemas e planejar outras etapas. Aprenda a compartilhar todas essas informações com sua equipe, clientes ou outras partes interessadas com diversas opções de importação e exportação.

## Importe um dashboard como JSON [#import-json]

Você pode importar um dashboard como JSON quando quiser modificar um dashboard pronto para uso no nível do código. Por exemplo, se quiser criar um dashboard personalizado mesclando dois dashboards existentes, você pode modificar o JSON de um dashboard base e importá-lo de volta para New Relic.

Para importar um dashboard como JSON:

1. Vá para <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Dashboards**</DNT>.

2. Abra o JSON de um dashboard existente clicando no ícone <DNT>**...**</DNT> e selecione <DNT>**Copy JSON**</DNT>.

3. Volte para a página <DNT>**Dashboards**</DNT> .

4. No canto superior direito, clique em <DNT>**Import dashboard**</DNT>.

5. Cole seu código JSON.

6. Escolha as configurações de conta e permissão para o dashboard. Você não pode alterar a conta depois de defini-la, mas pode [alterar as permissões](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-settings) a qualquer momento.

7. Clique em <DNT>**Save**</DNT>.

## Exportar um dashboard como PDF [#export-dashboards-pdf]

Para exportar seu dashboard como um arquivo PDF, no canto superior direito do dashboard, clique no ícone <DNT>**...**</DNT> e selecione <DNT>**Export dashboard as PDF**</DNT>.

### Exportar um dashboard como limitações de PDF [#export-pdf-limitations]

Tenha em mente o seguinte ao exportar um dashboard como PDF:

* [Variáveis de modelo](/docs/query-your-data/explore-query-data/dashboards/dashboard-template-variables/#requirements) não são suportadas.

* [Visualizações personalizadas](/docs/query-your-data/explore-query-data/dashboards/add-custom-visualizations-your-dashboards/) não são suportadas.

* [Os gráficos de outdoor](/docs/query-your-data/explore-query-data/use-charts/chart-types/#widget-billboard) que usam a cláusula [`COMPARE WITH`](/docs/nrql/nrql-syntax-clauses-functions/#sel-compare) não mostrarão porcentagens.

* O widget Markdown com imagens grandes pode não ser exibido corretamente ou não estar totalmente visível.

## Exportar um gráfico como CSV [#export-charts-csv]

Para exportar o gráfico da tabela de dashboard como um arquivo CSV, no canto superior direito do gráfico, clique em <DNT>**...**</DNT> e selecione <DNT>**Export as CSV**</DNT>.

<img
  title="Export chart as CSV"
  alt="Screenshot of how to export a table chart as a CSV file"
  src="/images/dashboards_screenshot-crop_export-csv.webp"
/>

Embora poderosa, a exportação como CSV tem limitações. Algumas coisas para manter em mente:

* Os dados sempre são exportados no horário UTC
* Exportar não suporta funções de agregador como `average()`
* A exportação não suporta a cláusula `COMPARE_WITH`
* Em alguns cenários, a interface pode realizar formatação adicional dos dados que não são replicados no CSV exportado, o que pode levar a pequenas discrepâncias (como -1.1E-17 renderizado como 0 na interface, mas não no CSV).
* [Os comentários](https://docs.newrelic.com/docs/nrql/nrql-syntax-clauses-functions/#comments) no NRQL podem produzir resultados inesperados.

## Compartilhe um dashboard com sua equipe [#share-dashboards]

Em qualquer dashboard, compartilhe-o com uma destas opções:

* Copie o link permanente do dashboard clicando no ícone do link no canto superior direito do dashboard. link permanente compartilhe um link direto para o dashboard. Lembre-se de que esses links expiram se não forem usados em um período de seis meses.
* Copie seu dashboard <DNT>**as JSON**</DNT> e adicione-o à área de transferência clicando no botão <DNT>**...**</DNT> no canto superior direito e selecionando <DNT>**Copy JSON to clipboard**</DNT>.

Você também pode [duplicar seu dashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-clone).
