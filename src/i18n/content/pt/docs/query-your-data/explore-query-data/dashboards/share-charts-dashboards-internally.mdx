## Compartilhe seu dashboard internamente [#share-dashboards-internally]

Você pode compartilhar qualquer dashboard com sua equipe usando qualquer uma das seguintes opções:

* Copie o link permanente do dashboard clicando no ícone de link no canto superior direito de um dashboard. link permanente compartilhe um link direto para o dashboard.

Ou

* Copie seu dashboard no formato <DNT>**JSON**</DNT> conforme a seguir e compartilhe-o com sua equipe.

  1. Vá para <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com/dashboards)**</DNT>.
  2. Clique no <Icon name="fe-more-horizontal" /> ícone no canto superior direito do dashboard.
  3. Selecione <DNT>**Copy JSON to clipboard**</DNT>.
  4. Cole o código JSON em uma mensagem ou documento para compartilhá-lo com sua equipe.

## Compartilhe seus gráficos internamente [#share-charts-internally]

Semelhante ao dashboard, você pode compartilhar gráficos com sua equipe seguindo estas etapas:

1. Vá para <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com//dashboards)**</DNT>.
2. Abra o dashboard que contém o gráfico que você deseja compartilhar.
3. Selecione <DNT>**View query**</DNT> do <Icon name="fe-more-horizontal" /> menu do gráfico. A consulta abre no criador de consulta.
4. No criador de consulta, clique em <DNT>**Copy query link**</DNT>. O link é copiado para sua área de transferência.
5. Compartilhe com os membros da sua equipe por e-mail, chat ou qualquer outra ferramenta de comunicação.
