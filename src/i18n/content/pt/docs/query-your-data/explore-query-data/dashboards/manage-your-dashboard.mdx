Acesse qualquer [painel do New Relic](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards) para criar ou gerenciar seus gráficos diretamente no menu de gráficos, personalizar o layout do seu dashboard, ajustar os modos de exibição ou exportar seus dados.

Depois de personalizar seu dashboard e construir seus gráficos, use nosso recurso de visualização avançada e ferramentas de exploração de dados para correlacionar e analisar seus dados.

## Personalize seu dashboard [#dash-manage]

Vá para <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Dashboards**</DNT> para abrir a página do painel.

<img title="Dashboards index" alt="Dashboards index" src="/images/dashboards_screenshot-full_dashboard-index.webp" />

<figcaption>
  Acesse <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Dashboards** </DNT>para abrir o índice dashboard .
</figcaption>

Clique em um dashboard para abrir seus gráficos. Uma página dashboard inclui estes recursos:

<CollapserGroup>
  <Collapser id="dash-clone" title="Duplique seu dashboard">
    Você pode duplicar seu dashboard independentemente dos seus níveis de permissão. Clique no <Icon name="fe-more-horizontal" /> ícone no canto direito e selecione <DNT>**Duplicate dashboard**</DNT>.

    O dashboard duplicado é nomeado como o dashboard original seguido da palavra “cópia”. Por exemplo, se você duplicar um dashboard denominado `This is my dashboard`, a duplicata será chamada `This is my dashboard copy`. Você pode alterar o nome, selecionar a conta associada e adicionar as permissões ao duplicá-la. Depois de clicar em <DNT>**Create**</DNT>, você verá uma mensagem pop-up na tela para acessar o dashboard duplicado. Você pode editar o nome e outras propriedades do dashboard, como permissões, a qualquer momento.

    O dashboard é copiado automaticamente e a duplicata é adicionada ao [índice](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#get-started). A duplicata tem [<DNT>**Public - Read and write**</DNT> permissões](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-permissions).
  </Collapser>

  <Collapser id="dash-edit" title="Edite seu dashboard">
    Você pode editar um dashboard se tiver [permissão](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-permissions) para fazer isso.

    Use a edição <Icon name="fe-edit-2" /> botão para:

    * Copie o link permanente do dashboard.
    * Renomeie seu dashboard. [Os nomes podem ser pesquisados](#dashboards-search), por isso recomendamos fornecer um nome significativo usando palavras que ajudarão você a localizar seu dashboard facilmente.
    * Crie novo conteúdo clicando no botão <DNT>**+ Add widget**</DNT> . Adicione um novo gráfico usando o [criador de consulta](/docs/query-your-data/explore-query-data/query-builder/introduction-query-builder/) ou adicione texto, imagens ou links usando nosso [editor Markdown](#markdown).
    * Redimensionar e reorganizar [gráficos](/docs/query-your-data/explore-query-data/use-charts/use-your-charts/). Você pode mover qualquer gráfico e colocá-lo em qualquer lugar do dashboard para que o layout definido atenda às suas preferências: coloque os gráficos mais relevantes no topo ou solte os gráficos menos usados em um canto. Você pode configurar até 12 colunas de gráficos.
  </Collapser>

  <Collapser id="dash-export" title="Exporte seu dashboard">
    Você pode exportar seu dashboard como um arquivo PDF. Clique no <Icon name="fe-more-horizontal" /> ícone no canto direito e selecione <DNT>**Export dashboard as PDF**</DNT>. Uma nova aba no seu browser será aberta com seu PDF.

    Exportar para PDF ou como imagem altera a exibição dependendo do tipo de gráfico.
  </Collapser>

  <Collapser id="dash-json" title="Importe seu dashboard como JSON">
    Você pode importar um dashboard como JSON quando quiser modificar um dashboard pronto para uso no nível do código. Por exemplo, se você deseja criar um painel personalizado mesclando dois painéis existentes, você pode modificar o JSON de um dashboard base e importá-lo de volta para o New Relic.

    Para importar um dashboard como JSON:

    1. Vá para <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com/dashboards)**</DNT>.
    2. Abra um dashboard existente.
    3. No canto superior direito, clique em <DNT>**...**</DNT> e selecione <DNT>**Copy JSON to Clipboard**</DNT> ou <DNT>**[Manage JSON](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#manage-json)**</DNT>.
    4. Volte para a página <DNT>**Dashboards**</DNT> .
    5. No canto superior direito, clique em <DNT>**Import dashboard**</DNT>.
    6. Cole seu código JSON. O novo e aprimorado editor JSON inclui verificação de erros e sintaxe para que você possa identificar erros antes de importar o dashboard.
    7. Escolha as configurações de conta e permissão para o dashboard. Você não pode alterar a conta depois de defini-la, mas pode [alterar as permissões](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-settings) a qualquer momento.
    8. Clique em <DNT>**Save**</DNT>.
  </Collapser>

  <Collapser id="manage-json" title="Gerenciar JSON do dashboard">
    A opção &quot;Gerenciar JSON&quot; permite que você atualize o painel inteiro. Para usar:

    1. Vá para <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com/dashboards)**</DNT>.
    2. Abra um dashboard existente.
    3. No canto superior direito, clique em <DNT>**...**</DNT> e selecione <DNT>**Manage JSON**</DNT>.

    Isso abre o editor JSON Editar seu dashboard :

    <img title="Edit your dashboard JSON" alt="Image of the Edit your dashboard JSON editor" src="/images/json_editor.webp" />

    Aqui, você pode copiar, baixar ou usar o editor JSON. O editor JSON incorpora verificação de erros e sintaxe para facilitar a rápida identificação de problemas. Depois de salvar suas alterações, elas serão aplicadas automaticamente ao seu dashboard. Isso só funcionará se você tiver acesso de edição ao dashboard.

    Em seguida, use **Copiar e Baixar** para criar um novo dashboard por meio da função Importar painel disponível na página do painel.

    <Callout variant="tip">
      Recomendamos que você baixe uma cópia do JSON atual como backup. Isso facilita a restauração do seu dashboard ao estado original, se necessário.
    </Callout>
  </Collapser>

  <Collapser id="dash-share" title="Compartilhe seu dashboard">
    Você pode [compartilhar seu dashboard com sua equipe interna](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-internally) por meio de um link permanente. Para compartilhar o dashboard com espectadores fora da sua organização New Relic , você precisa criar URLs dashboard ao vivo acessíveis publicamente e compartilhá-los. Isso pode ser feito usando a [interface do usuário](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#dashboard) e [o NerdGraph](/docs/apis/nerdgraph/examples/manage-live-dashboard-urls-via-api). Ao compartilhar o dashboard externamente, você pode definir um período para que o URL dashboard compartilhado permaneça ativo.

    <Callout variant="caution">Qualquer pessoa com uma URL dashboard ativa para um dashboard compartilhado externamente pode visualizar todas as informações desse dashboard. Compartilhe informações com cuidado, respeitando as políticas e procedimentos internos da sua empresa.</Callout>
  </Collapser>

  <Collapser id="dashboards-metadata" title="Ver metadados">
    Dentro de um dashboard, clique no botão &apos;metadados&apos; para acessar o widget <DNT>**metadata**</DNT>. Lá você pode ver o GUID da entidade dashboard, o ID da conta, o nome da conta e gerenciar todas as tags adicionadas ao dashboard.

    <img width="50%;" title="Dashboards: See metadata" alt="Dashboards: See metadata" src="/images/dashboards_screenshot-crop_dashboard-metadata.webp" />

  </Collapser>

  <Collapser id="dashboards-tags" title="Ver etiqueta">
    Dentro de um dashboard, clique no botão &apos;tag&apos; para acessar o widget <DNT>**tags**</DNT>. Lá você pode ver o ID da conta dashboard, o nome da conta, o e-mail da pessoa que a criou, as permissões e o ID da conta confiável.

    <img width="50%;" title="Dashboards: See tags" alt="Dashboards: See tags" src="/images/dashboards_screenshot-crop_dashboard-tags.webp" />

    No índice dashboard , existe a opção [<DNT>**See metadata &amp; tags**</DNT>](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-metadata) para ver a tag de um dashboard.
  </Collapser>

  <Collapser id="dash-settings" title="Menu de configurações">
    Clique no <Icon name="fe-more-horizontal" /> ícone no canto direito e selecione <DNT>**Settings**</DNT>.

    <img width="40%;" title="Dashboards: Settings menu" alt="Dashboards: Settings menu" src="/images/dashboards_screenshot-crop_dashboard-settings-menu.webp" />

    Esta opção permite:

    * Para alterar o nome do dashboard. [Os nomes podem ser pesquisados](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-search), por isso recomendamos dar a eles um nome significativo que ajude você a localizar seu dashboard facilmente.
    * Para modificar as [permissões dodashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-permissions).

    No menu de configurações você também pode ver quando o dashboard foi criado e a conta a que pertence. Esses valores não podem ser modificados.

Consulte [Variáveis de modelo](/docs/query-your-data/explore-query-data/dashboards/dashboard-template-variables).

  <Collapser id="dash-visual-tv" title="Modo televisão">
    Você pode ativar um modo de TV em tela cheia que otimiza o dashboard para exibição na tela da televisão.

    Existem duas maneiras de ativar o modo TV:

    * Ao visualizar um dashboard, selecione o <Icon name="fe-monitor" /> ícone no canto superior direito.

    * Adicione este parâmetro ao URL da página dashboard : `&tv-mode`

    * <DNT>**Cycle to the next page every 20 seconds**</DNT>. Se o dashboard incluir [várias páginas](#add-pages), você verá esta opção.

## Adicione novo conteúdo ao seu dashboard [#dash-create-chart]

Aqui estão algumas maneiras de adicionar novo conteúdo ao seu dashboard:

* Clique em editar <Icon name="fe-edit-2" /> botão no canto superior direito e crie um novo conteúdo clicando no botão <DNT>**Add widget**</DNT> .
* Da [métrica e evento](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer) e [criador de consulta](/docs/introduction-chart-builder).
* Use o botão <DNT>**+ Add widget**</DNT> (acessível na página principal dashboard ou no modo de edição) para acessar o [criador de consulta](/docs/chart-builder/use-chart-builder/get-started/introduction-chart-builder) ou para adicionar conteúdo (como texto, links ou imagens) usando nosso [editor Markdown](#dash-create-chart).
* [Copie um gráfico existente](#dash-manage-charts) de qualquer dashboard.

Se você tiver problemas para adicionar novo conteúdo, [verifique nossa mensagem de erro](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data/#chart-error-messages).

### Adicione conteúdo personalizado usando o editor Markdown [#markdown]

Adicione contexto e estilo ao seu dashboard com conteúdo personalizado usando nosso editor Markdown. Com esse recurso, você pode adicionar texto formatado, listas de verificação, imagens, trecho de código e muito mais. Isso torna mais fácil explicar o painel compartilhado, adicionar hiperlinks úteis, criar listas de tarefas e criar qualquer outro conteúdo rico que você precisar.

O editor Markdown contém um painel <DNT>**Markdown**</DNT> , onde você insere seu conteúdo, e um painel <DNT>**Preview**</DNT> , onde você pode visualizá-lo. Para obter mais informações sobre as opções de sintaxe do Markdown, consulte o [guia Markdown do GitHub](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax).

Você também pode editar o conteúdo existente clicando nas reticências <Icon name="fe-more-horizontal" /> ícone em qualquer widget Markdown e selecionando <DNT>**Edit**</DNT>.

O widget Markdown suporta diagramas Mermaid. Para obter mais informações, consulte [Referência de sintaxe do Mermaid](https://mermaid.js.org/intro/syntax-reference.html#diagram-syntax).

## Organize seu painel com páginas [#add-pages]

Você pode usar várias páginas para organizar os dados dashboard em diferentes visualizações. Ao adicionar mais páginas a esse dashboard, você pode acessá-las usando as guias na parte superior da interface dashboard .

<img title="Dashboard with multiple pages" alt="Dashboard with multiple pages" src="/images/dashboards_screenshot-full_multi-page-dashboard.webp" />

<figcaption>
  <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Dashboards**</DNT>: Este é um exemplo de dashboard com múltiplas páginas, representado pelas abas na parte superior do dashboard. Você pode adicionar páginas ao painel, copiar páginas existentes e arrastar e soltar as guias da página em novas posições.
</figcaption>

Você pode usar esse recurso para agrupar visualizações dashboard relacionadas. Isso é valioso quando você agrega muitos dados e gráficos relacionados a um projeto, equipe ou assunto específico.

Por exemplo, uma equipe de aplicativos móveis pode criar um dashboard focado no desempenho do aplicativo por país. A primeira página dashboard pode ser uma visão geral do desempenho em todos os países, com outras páginas focadas em países específicos.

Há outro recurso para conectar o dashboard:

* [Crie um widget contendo texto Markdown](#markdown) para adicionar links diretos a páginas de interface ou painel específicos.
* Use [a filtragem de facetas](/docs/dashboards/new-relic-one-dashboards/manage-your-dashboard/filter-new-relic-one-dashboards-facets) para criar links que vinculam e filtram automaticamente outros painéis.
* Use a pesquisa dashboard para encontrar um painel com nome semelhante. Para aproveitar isso, você pode adicionar palavras/frases específicas da equipe ou do projeto aos nomes dashboard .

### Adicionar e editar páginas em um dashboard [#add-pages-edit]

Para adicionar ou editar uma página em um dashboard:

1. Em um dashboard novo ou existente, entre no modo de edição selecionando o <Icon name="fe-edit-2" /> ícone. Dependendo das permissões do dashboard, você verá este ícone ou não.

2. Adicione uma nova página:

   * Selecione <DNT>**Add a page**</DNT> para adicionar uma página em branco.
   * Clone uma página existente clicando no dropdown ao lado do nome da página e selecionando <DNT>**Duplicate Page**</DNT>.

3. No modo de edição, você pode adicionar widget à nova página, arrastar e soltar guias de página em novos locais e realizar outras [tarefas de edição dashboard ](#dash-manage-charts).

4. Ao terminar, selecione <DNT>**Done editing**</DNT>.

## Gerencie seus gráficos e conteúdo Markdown [#dash-manage-charts]

A partir de qualquer elemento Markdown, acesse o menu no canto superior direito para editá-lo ou excluí-lo. Em qualquer gráfico, acesse o menu de ação do gráfico no canto superior direito para:

* Expanda seu gráfico para tela inteira.
* [Compartilhe seu gráfico](#dash-export) como uma imagem ou com um link.
* Somente para gráficos de tabela, exporte como um arquivo `.csv` . Você pode importar este arquivo para outros aplicativos como Microsoft Excel ou Planilhas Google para fazer análises adicionais.
* Veja a consulta.
* Copie o gráfico para qualquer dashboard.
* Abra o [console NRQL](/docs/query-your-data/explore-query-data/query-builder/nrql-console/) para ver ou editar a consulta associada ao gráfico.
* Duplique o gráfico.
* Exclua o gráfico.

## Filtre e refine seus gráficos [#dash-search]

Você pode restringir as informações exibidas usando a função de filtragem, que é uma representação visual das condições de consulta:

* Use o <DNT>**filter bar**</DNT> para selecionar os valores ou atributos que deseja ver e remova o restante dos elementos dos gráficos.
* Abra o <DNT>**advanced filter bar**</DNT> para acessar os operadores booleanos (como `=`, `!=`, `CONTAINS`, `EXCLUDES`, etc.) e adicione condições compostas e complexas para filtrar dados.

Após aplicar um filtro, seu dashboard mostrará apenas os dados associados aos elementos que você selecionou. Um pequeno contador indica o número de filtros aplicados por vez.

Para retornar à visualização padrão, clique na pequena cruz ao lado do filtro para removê-lo.

## Filtre usando a legenda do gráfico [#filter-legend]

Clique em uma legenda em qualquer gráfico com legendas para ver apenas aquela série e remova o restante da série do gráfico. Isso ajuda a isolar os dados que deseja analisar.

Para fazer o contrário, use CMD (no Mac) ou CTRL (no Windows): isso removerá a série selecionada e manterá o restante.

## Filtrar painel usando facetas [#filter-facets]

Se a consulta NRQL de um gráfico contiver uma cláusula `FACET`, você poderá usar o atributo facetado para filtrar o dashboard atual ou outro dashboard relacionado . Para obter mais detalhes, consulte [Filtrar por facetas](/docs/dashboards/new-relic-one-dashboards/manage-your-dashboard/filter-new-relic-one-dashboards-facets).

## Use o seletor de hora para ajustar as configurações de hora [#dash-time-picker]

Por padrão, cada gráfico no dashboard exibe dados para o período de tempo especificado quando foram criados no [construtor de consultas](/docs/introduction-chart-builder). No entanto, você pode usar o [seletor de tempo](/docs/new-relic-solutions/get-started/glossary/#time-picker) para alterar o intervalo de tempo dos dados exibidos e definir o mesmo intervalo para todos os gráficos. Isso é particularmente útil durante a solução de problemas de eventos de alerta, se você precisar restringir seus dados para observar o que aconteceu em um período de tempo específico.

<img width="50%;" title="Time picker on the Dashboard page" alt="Time picker on the Dashboard page" src="/images/dashboards_screenshot-crop_time-picker.webp" />

A taxa de atualização depende da duração da janela de tempo que você está visualizando. Para obter mais informações e exemplos, consulte [Intervalos de atualização do gráfico](/docs/insights/use-insights-ui/manage-dashboards/insights-chart-refresh-intervals).

Para alterar o intervalo de tempo:

* Selecione uma das opções disponíveis no menu dropdown (do último `5 minutes` ao último `6 months`).
* Personalize o intervalo de tempo com carimbo de data e hora de início e término específico usando o menu personalizado.

Se você não quiser alterar o intervalo de tempo de um gráfico ao usar o seletor de hora:

1. Clique no <Icon name="fe-more-horizontal" /> ícone e selecione <DNT>**Edit**</DNT>.
2. Abra o menu <DNT>**Dashboard Options**</DNT> na interface de personalizações.
3. Ative ou desative o botão <DNT>**Ignore time picker**</DNT> dependendo da sua necessidade.

## Exporte seus dados [#export-share]

Você pode exportar seus dados dashboard e gráfico com qualquer uma das seguintes opções:

* Exporte qualquer dashboard como um arquivo PDF clicando em <Icon name="fe-more-horizontal" /> ícone no canto direito e selecione <DNT>**Export dashboard as PDF**</DNT>.
* Compartilhe seus gráficos como uma imagem PNG ou como um link: Clique em <Icon name="fe-more-horizontal" /> ícone no canto direito do gráfico e selecione <DNT>**Get as image**</DNT>.

A exportação para PDF ou como imagem apresenta certas limitações, dependendo do tipo de gráfico. Observe que os links gerados tanto para o PDF quanto para a imagem PNG são de acesso público e não exigem login.

## Principais ferramentas visuais [#key-visual-tools]

Dashboard oferece recursos de visualização intuitivos e ferramentas para exploração avançada de dados e rápida resolução de problemas.

<CollapserGroup>
  <Collapser id="key-visual-coloring" title="Coloração consistente do gráfico">
    Para que você possa ver e correlacionar rapidamente seus dados, as facetas aplicadas a mais de um gráfico em um dashboard têm uma cor de faceta consistente em todos os gráficos.
  </Collapser>

  <Collapser id="key-visual-needle" title="Agulha correlacionada">
    Quando você passa o mouse sobre um gráfico, a agulha correlacionada se sobrepõe a todos os gráficos ou pontos de dados no dashboard ao mesmo tempo. A dica de ferramenta fornece os pontos de dados relevantes da faceta selecionada, como valores máximos e mínimos em um gráfico de linhas. Também destaca o atributo selecionado em um gráfico de pizza.
  </Collapser>

  <Collapser id="key-visual-scrubber" title="Depurador de dados">
    O controle deslizante do gráfico ajuda a selecionar um ponto de dados ou faceta em um gráfico quando o gráfico está muito lotado e as facetas estão muito próximas umas das outras.

    Passe o mouse ao longo da agulha para selecionar suavemente as facetas adjacentes e visualizar seus pontos de dados associados. Você também pode iluminar um gráfico muito preenchido desmarcando um ou mais atributos que aparecem na interface do usuário.
  </Collapser>

  <Collapser id="key-visual-zoom" title="Pincele para ampliar">
