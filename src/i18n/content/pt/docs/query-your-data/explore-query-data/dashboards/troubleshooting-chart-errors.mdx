## Problemas com links e descrições de widgets [#widget-link-issues]

Se você estiver enfrentando problemas com links ou descrições de widgets, verifique os seguintes problemas e soluções comuns.

<table>
  <thead>
    <tr>
      <th style={{ width: "400px" }}>
        <DNT>
          **Problem**
        </DNT>
      </th>

      <th style={{ width: "400px" }}>
        <DNT>
          **Solution**
        </DNT>
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        O link do widget não é clicável
      </td>

      <td>
        Verifique se a URL é válida e usa um protocolo suportado (`http://` ou `https://`). Links com outros protocolos não são suportados.
      </td>
    </tr>

    <tr>
      <td>
        A dica de ferramenta da descrição não aparece
      </td>

      <td>
        Certifique-se de que o campo de descrição contenha texto. Aponte para a área do título do widget para exibir a dica de ferramenta. Se ainda não aparecer, tente atualizar o dashboard.
      </td>
    </tr>

    <tr>
      <td>
        Erro de validação de URL
      </td>

      <td>
        Verifique se a URL não excede 2.048 caracteres e usa apenas os protocolos `http://` ou `https://`.
      </td>
    </tr>

    <tr>
      <td>
        O link leva a um erro 404
      </td>

      <td>
        Verifique se a URL de destino está correta e acessível. Links externos devem estar publicamente disponíveis ou acessíveis na sua rede.
      </td>
    </tr>

    <tr>
      <td>
        Não é possível editar o link ou a descrição do widget
      </td>

      <td>
        Confirme se você tem permissões de edição para o dashboard. Somente usuários com permissões apropriadas podem modificar as configurações do widget.
      </td>
    </tr>

    <tr>
      <td>
        A mutação da API não retorna erros, mas as alterações não aparecem
      </td>

      <td>
        Atualize a página do dashboard. Se as alterações ainda não aparecerem, verifique se o ID do widget está correto e se a mutação foi concluída com sucesso usando o explorador de API do NerdGraph.
      </td>
    </tr>
  </tbody>
</table>
