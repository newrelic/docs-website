A taxa de atualização dos gráficos New Relic é o intervalo do período da consulta dividido por 60. A frequência máxima com que a consulta pode ser atualizada é a cada 5 segundos.

Por exemplo:

* Uma consulta com intervalo de tempo de 7 dias teria uma taxa de atualização de uma vez a cada 2,8 horas.
* Uma consulta com intervalo de tempo de 1 hora teria uma taxa de atualização de uma vez a cada 1 minuto.

Para consulta usando `TIMESERIES`, o tamanho do bucket `TIMESERIES` é usado como intervalo de atualização. Por exemplo:

* `SELECT count(*) FROM PageView TIMESERIES 5 minutes` seria atualizado a cada 5 minutos.
* `SELECT count(*) FROM PageView TIMESERIES 1 hour SINCE 1 day ago` atualizaria a cada hora.
