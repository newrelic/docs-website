[New Relic](https://one.newrelic.com) é uma poderosa plataforma de observabilidade que dá acesso a todos os seus dados em todo o sistema. Hospedamos dados de telemetria enviados pela [sua entidade](/docs/new-relic-one/use-new-relic-one/core-concepts/what-entity-new-relic/), que basicamente é qualquer coisa que possamos identificar e que possua dados que você possa monitor, incluindo aplicativos, serviços, hosts, etc. O que você disser!

Embora forneçamos uma experiência pronta para uso para ver seus dados com [um painel](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) selecionado, você pode adaptar o acesso aos seus dados e personalizar a visibilidade de várias maneiras, inclusive [na interface](#query-ui) ou [via API](#query-apis).

<Callout variant="important">
  Para entender melhor seus dados armazenados no New Relic, consulte [Tipos de dados](/docs/using-new-relic/data/understand-data/new-relic-data-types).
</Callout>

## Navegue pelos seus dados na interface [#browse-data]

Nossa plataforma oferece diversas experiências que não exigem conhecimento de NRQL ou qualquer linguagem de consulta:

* Para explorar seus dados do New Relic, você pode usar [métrica e evento](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer), um navegador de dados intuitivo para explorar dados e criar visualizações. Você também pode utilizar o [criador de consulta](/docs/chart-builder/use-chart-builder/choose-data/use-advanced-nrql-mode-specify-data) para refinar sua consulta.
* Para explorar seus dados trace , você pode usar a [consultadistributed tracing ](/docs/understand-dependencies/distributed-tracing/ui-data/additional-distributed-tracing-features-new-relic-one).
* Para explorar o log, você pode usar a [interface de log](/docs/logs/ui-data/use-logs-ui/).

## Consultar dados na interface [#query-ui]

Se você estiver pronto para fazer mais do que navegar pelos dados, torne-se um ator versátil e personalize sua consulta na interface do usuário do New Relic. Use linguagens de consulta, incluindo nossa [linguagem de consulta New Relic](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language/) ou [nossa linguagem de consulta estilo PromQL](https://prometheus.io/docs/prometheus/latest/querying/basics/), para editar consultas com total flexibilidade. Por exemplo, você pode adicionar mais [cláusulas`WHERE` ](/docs/query-your-data/nrql-new-relic-query-language/get-started/nrql-syntax-clauses-functions/#sel-where), modificar o valor retornado, mudar para [outros tipos de visualizações](/docs/query-your-data/explore-query-data/use-charts/chart-types/), etc.

<Callout variant="tip">
  Você é novo na consulta de idiomas? Comece a navegar pelos dados em [métrica e evento](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer), depois recorra ao [criador de consulta](/docs/chart-builder/use-chart-builder/choose-data/use-advanced-nrql-mode-specify-data) para ver a consulta que você construiu e refine-a.
</Callout>

Existem duas maneiras de escrever sua própria consulta para recuperar dados e construir gráficos:

* [criador de consulta no modo NRQL](/docs/chart-builder/use-chart-builder/choose-data/use-advanced-nrql-mode-specify-data): Consulta usando [a linguagem de consulta New Relic](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language) (NRQL), a mesma linguagem que usamos para construir a maioria de nossas experiências de interface, e a forma mais avançada de consultar dados em New Relic.
* [criador de consulta no modo estilo PromQL](/docs/query-your-data/explore-query-data/chart-builder/use-advanced-promql-mode-specify-data): Escreva consultas usando uma consulta estilo PromQL.

Você também pode [consultar dados do seu IDE](/docs/codestream/observability/query-builder/) usando a extensão CodeStream da New Relic.

## Consultar dados via API [#query-apis]

Quando precisar de opções programáticas ou quiser desbloquear mais poder de consulta, você pode usar API para consultar seus dados do New Relic. Nossa API NerdGraph oferece opções mais poderosas não disponíveis na interface, como consulta em contas e exportação de conjuntos de dados históricos de longo prazo. Para obter mais informações, consulte [Introdução ao NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph/#tutorials).
