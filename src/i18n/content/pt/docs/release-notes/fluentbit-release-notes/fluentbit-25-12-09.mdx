### Suporte para Fluent Bit 4.2.0

O usuário agora receberá a versão 4.2.0 Fluent Bit. Para obter mais detalhes, consulte [https://fluentbit.io/announcements/](https://fluentbit.io/announcements/).

### Mudado

* O agente de infraestrutura recomenda os pacotes Fluent Bit 4.2.0.
* [A imagem Docker do plug-inFluent Bit Output](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) usa Fluent Bit 4.2.0, exceto para Windows Server 2019.
* [O gráfico New Relic Logging Helm](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) instala a nova imagem do plug-in (gráfico newrelic-logging versão 1.33.0).
