### Suporte para Fluent Bit 5.0.8

O usuário agora receberá a versão 5.0.8 do Fluent Bit. Para mais detalhes, consulte os [anúncios do Fluent Bit](https://fluentbit.io/announcements/).

Esta versão inclui o [New Relic Fluent Bit Output Plugin 3.7.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.7.0).

### Adicionado

* Plug-in de saída New Relic Fluent Bit 3.7.0 integração

### Mudado

* A versão do Go foi atualizada para 1.26.4 para correções de CVE
* O agente de infraestrutura recomenda os pacotes do Fluent Bit 5.0.8 (Linux) e 5.0.7 (Windows)
* [A imagem Docker do plug-in de saída do Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) usa o Fluent Bit 5.0.8 com o plug-in de saída 3.7.0
* O [Helm chart do New Relic Logging](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) instala a nova imagem do plug-in
* A imagem Docker do Windows usa o Fluent Bit 5.0.7 (a imagem do Windows 5.0.8 ainda não está disponível no upstream)

### Notas

Para se manter atualizado com as correções e melhorias mais recentes, assine nosso [feed RSS de Logs](/docs/release-notes/logs-release-notes/).
