Uma nova versão do agente foi lançada. Siga os procedimentos padrão para [atualizar o agente de infraestrutura](https://docs.newrelic.com/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). A New Relic recomenda que você atualize o agente regularmente e, no mínimo, a cada 3 meses. Até o momento desta versão, a versão mais antiga suportada é [o agente de infraestrutura 1.55.2](https://docs.newrelic.com/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1552/).
