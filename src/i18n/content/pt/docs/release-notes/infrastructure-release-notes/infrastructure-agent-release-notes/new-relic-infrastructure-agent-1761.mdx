Uma nova versão do agente foi lançada. Siga os procedimentos padrão para [atualizar o agente de infraestrutura](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). A New Relic recomenda que você atualize o agente regularmente e, no mínimo, a cada 3 meses. Até o momento desta versão, a versão mais antiga suportada é [o agente de infraestrutura 1.58.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1581/).

## O que mudou

#### Bugs

* Corrigido um problema em que `set-service-account` falhava para usuários locais em [#2247](https://github.com/newrelic/infrastructure-agent/pull/2247).

#### Segurança

* Containerd atualizado para a versão 1.7.32 para corrigir a vulnerabilidade CVE-2026-46680 em [#2250](https://github.com/newrelic/infrastructure-agent/pull/2250).
* Atualizado o nri-flex para a versão 1.18.4 para resolver a vulnerabilidade CVE-2026-45570 no [#2252](https://github.com/newrelic/infrastructure-agent/pull/2252).
