Uma nova versão do agente foi lançada. Siga os procedimentos padrão para [atualizar o agente de infraestrutura](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). A New Relic recomenda que você atualize o agente regularmente e, no mínimo, a cada 3 meses. A partir desta versão, a versão mais antiga suportada é o [agente de infraestrutura 1.64.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1640/).

* Dependência nri-docker atualizada para 2.6.10, dependência nri-flex para v1.17.7 e dependência docker para v29 em [#2220](https://github.com/newrelic/infrastructure-agent/pull/2220)
* Imagens Docker criadas para o agente de infraestrutura para Windows em [#2202](https://github.com/newrelic/infrastructure-agent/pull/2202)
* Fluxo de trabalho de lançamento de imagem do Windows atualizado em [#2213](https://github.com/newrelic/infrastructure-agent/pull/2213)
