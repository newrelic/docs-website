## v1.292.0

### Recurso

#### Atualizar `BrowserInteraction` e `previousUrl` definição

Atualize `BrowserInteractions` para usar `document.referrer` para valores previousUrl na interação `initialPageLoad` em vez de espelhar `previousUrl` e `targetUrl`. Isso permite melhor funcionalidade para as jornadas do usuário e fornece mais insights sobre os links das páginas.

#### Adicionar mais eventos de inspeção

Adiciona novo evento de inspeção ao agente. Esses eventos incluem o evento de drenagem, o evento do ciclo de vida da janela, como navegar, carregar, DOMContentLoaded e o evento de sessão que emite alterações no estado de sessão do agente.

### Correções de bugs

#### Corrigir valor da API `finished` `timeSinceLoad`

Corrige um problema em que a API `finished` estava criando um valor `timeSinceLoad` para dados `PageAction` como timestamp Unix em vez de um valor de tempo relativo em segundos a partir da origem da página.

## Declaração de apoio

New Relic recomenda que você atualize o agente regularmente para garantir que esteja obtendo os benefícios mais recentes de recursos e desempenho. Versões mais antigas não terão mais suporte quando chegarem [ao fim de sua vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). As datas de lançamento refletem a data de publicação original da versão do agente.

Novos lançamentos de agente do browser são disponibilizados aos clientes em pequenas etapas ao longo de um período de tempo. Por esse motivo, a data em que o lançamento fica acessível à sua conta pode não corresponder à data de publicação original. Consulte este [dashboard de status](https://newrelic.github.io/newrelic-browser-agent-release/) para obter mais informações.

De acordo com nossa [política de suporte a browsers](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), a versão v1.292.0 do agente do Browser foi criada e testada nesses browsers e intervalos de versão: Chrome 127-137, Edge 127-137, Safari 17-18 e Firefox 128-138. Para dispositivos móveis, a versão v1.292.0 foi criada e testada para Android OS 16 e iOS Safari 17-18.1.
