### Novos documentos

* Adicionado [guia de migração do agente Ruby 10.x](/docs/apm/agents/ruby-agent/getting-started/migration-10x-guide) para fornecer orientação abrangente para a atualização para a versão 10.0.0 do agente Ruby.
* Adicionada a documentação de [integração do OpenTelemetry Kafka](/docs/opentelemetry/integrations/kafka/overview) para fornecer orientação para o monitoramento do Kafka com OpenTelemetry, incluindo implantações auto-hospedadas, Kubernetes com Strimzi, referência de métricas e consulta de dados.
* Adicionada a documentação de [integração OpenTelemetry NGINX](/docs/opentelemetry/nginx/nginx-otel-overview) para fornecer orientação para monitorar NGINX e NGINX Plus com OpenTelemetry, incluindo implantações baseadas em host e Kubernetes.
* Adicionados [exemplos de automação de workflow](/docs/workflow-automation/workflow-examples/workflow-examples) para fornecer padrões práticos para workflows básicos, integrações e casos de uso avançados.
