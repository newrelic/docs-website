## Novos documentos

* Adicionada a funcionalidade da [API de tradução de dashboard do Grafana](/docs/infrastructure/prometheus-integrations/view-query-data/grafana) para dar suporte aos fluxos de trabalho de migração de dashboard.
* Lançada a **região de dados do New Relic no Japão** e atualizadas várias páginas de documentação para refletir a nova região e seus recursos em todos os produtos.

### Grandes mudanças

* **Agente SRE renomeado para New Relic Autopilot** em toda a documentação com terminologia e referências atualizadas em toda a plataforma.
* Atualizada a documentação de [Gerenciamento de vulnerabilidades](/docs/vulnerability-management/advanced/query-examples) para remover funções NRQL inválidas e corrigir padrões de consulta para melhor precisão.
* **Removida a documentação do complemento de Gerenciamento de vulnerabilidades**, pois o recurso foi integrado aos recursos principais da plataforma.
* Atualizada a [biblioteca de modelos do Scorecards](/docs/service-architecture-intelligence/scorecards/templates) com cobertura abrangente de todos os modelos disponíveis e estrutura de organização aprimorada.
* Atualizada a [matriz de compatibilidade](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) do Node.js para incluir as versões suportadas mais recentes e esclarecer a compatibilidade com várias versões do Node.js.
* Atualizado o documento do [servidor MCP do New Relic](/docs/agentic-ai/mcp/overview) para torná-lo compatível com a conformidade da HIPAA.

### Pequenas mudanças

* Adicionadas ações de [dimensionamento de um pool de agentes do AKS](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-aks-scalepool) para gerenciar seus clusters do AKS diretamente de seus fluxos de trabalho.
* Atualizada a [configuração de integração do Terraform](/docs/infrastructure/amazon-integrations/aws-integration-for-metrics/terraform).
* Aprimorada a [integração do Microsoft SQL Server](/docs/opentelemetry/database/mssql/introduction) com links de configuração atualizados e orientações de configuração aprimoradas.
* Corrigida a documentação do [endpoint de integração do HAProxy](/docs/infrastructure/host-integrations/host-integrations-list/haproxy-monitoring-integration) para maior precisão e clareza.
* Adicionado [suporte abrangente a certificados personalizados para localizações privadas](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-configuration/#custom-ca-certificates) com instruções detalhadas de instalação e opções de configuração para monitoramento seguro.
* Atualizada a [documentação do programa para estudantes](/docs/accounts/accounts-billing/account-setup/student-program) com detalhes do programa.
