### Novos documentos

* Adicionada uma página dedicada de resolução de problemas para [O instalador MSI do Windows falha com o erro 1603](/docs/infrastructure/infrastructure-troubleshooting/troubleshoot-infrastructure/windows-msi-install-fails-error-1603), para que você possa resolver falhas de instalação do agente de infraestrutura no Windows.

### Grandes mudanças

* Reestruturamos os documentos de [monitoramento do Microsoft SQL Server](/docs/opentelemetry/database/mssql/introduction) do OpenTelemetry em uma seção dedicada de várias páginas, desde a [configuração guiada](/docs/opentelemetry/database/mssql/guided) até as instruções de host e Amazon RDS, métricas, compatibilidade e resolução de problemas.
* Monitoramento atualizado do [ECS no EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) e [ECS no Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) com uma opção de Collector NRDOT junto com o Collector OpenTelemetry, para que você possa executar o sidecar com a distribuição suportada da New Relic.

### Pequenas mudanças

* Atualizada a documentação do [banco de dados Oracle](/docs/opentelemetry/database/otel-oracledb) do OpenTelemetry e de [identificação do serviço de banco de dados](/docs/opentelemetry/database/database-service-identification) com parâmetros corrigidos.
* Documentado como [exportar dashboard e gráficos por meio da API NerdGraph](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data/#export-api) como PDF ou PNG, com dimensões personalizadas, substituições de variáveis de template e filtros de entidade, e atualizadas as etapas de exportação da interface para abranger PNG.
* Documentado como [consultar conectores de conhecimento específicos](/docs/agentic-ai/knowledge-integration/overview/#query-specific-connectors) no chat do New Relic AI, usando `/{connector-name}` para direcionar a uma única fonte ou a uma pesquisa ampla em todos os conectores.
* Adicionado o tipo de gráfico de medidor à documentação de [tipos de gráficos](/docs/query-your-data/explore-query-data/use-charts/chart-types).
* Atualizados os [requisitos de compatibilidade do agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) com o relatório de compatibilidade mais recente.
* Atualizados os [requisitos de compatibilidade do agente .NET](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements) com as versões mais recentes de bibliotecas suportadas.
* Atualizados os detalhes do [suporte híbrido do agente Ruby](/docs/apm/agents/manage-apm-agents/opentelemetry-api-support), incluindo o aumento da versão mínima suportada do agente Ruby para 10.6.0, e as instruções de configuração da [integração da AWS](/docs/infrastructure/amazon-integrations/get-started/introduction-aws-integrations).
* Esclarecemos nos documentos da [interface de ingestão de dados](/docs/data-apis/manage-data/manage-data-coming-new-relic) que a visualização Auto não oferece suporte ao seletor de hora e mostra dados em intervalos de 5 minutos por até um dia, portanto, use as visualizações Daily, Weekly ou Monthly para intervalos maiores.
* Removidas as APIs de sobreposição da documentação de [navegação do SDK](/docs/new-relic-solutions/build-nr-ui/sdk-component/platform-apis/navigation).

### Publicações de novidades

* Adicionado [Exportações Aprimoradas de Dashboard e uma Nova Mutação de Snapshot de Imagem de Gráfico no NerdGraph](/whats-new/2026/06/whats-new-0626-dashboards) para anunciar duas novas mutações do NerdGraph para gerar snapshots em PDF de páginas completas de dashboard e imagens de gráficos independentes.
