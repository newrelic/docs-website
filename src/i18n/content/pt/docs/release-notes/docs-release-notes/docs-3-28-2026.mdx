### Grandes mudanças [#major-changes]

* Atualizada a documentação da [integração do Microsoft SQL Server](/install/microsoft-sql) para adicionar o Monitoramento de Desempenho de Consultas (QPM) para os seguintes métodos de autenticação:

  * Domínio do Windows
  * Conta de Serviço Gerenciada de grupo do Windows (gMSA)

* Atualizada a documentação do [Monitoramento do Elasticsearch com OpenTelemetry](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) para maior clareza e adicionada uma documentação abrangente de instalação do Kubernetes.

* Atualizado [Dashboards com a API NerdGraph](/docs/apis/nerdgraph/examples/nerdgraph-dashboards) com recursos de visibilidade de variáveis de template.

* Atualizado o [esquema de definição de automação de fluxo de trabalho](/docs/workflow-automation/workflow-automation-apis/definition-schema) com extensas atualizações no esquema e exemplos abrangentes.

### Pequenas mudanças [#minor-changes]

* Atualizado o [Gerenciamento de consumo](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/consumption-management) com recursos de previsão de uso de CCU e funcionalidades aprimoradas de dashboard.
* Atualizamos [Visualizar resultados do monitor de ping](/docs/synthetics/synthetic-monitoring/using-monitors/view-ping-monitor-results) com orientações e exemplos aprimorados.
* Dicionário de atributos atualizado com definições e descrições adicionais de atributos para o Browser.
