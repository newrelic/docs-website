### Novos documentos

* Adicionada a [referência de atributos do eBPF](/docs/ebpf/attributes-reference) para fornecer documentação abrangente para todos os atributos coletados pelo agente eBPF da New Relic.
* Adicionado [Instalar Gateway sem Flux](/docs/new-relic-control/pipeline-control/gateway/install-gateway-without-flux) para fornecer orientação sobre como implantar o Gateway do Controle de Pipeline sem automação total, permitindo que os usuários gerenciem a infraestrutura do gateway usando ferramentas do Kubernetes enquanto ainda usam a interface para a configuração do pipeline.

### Grandes mudanças

* Atualizada a [documentação do Gateway de Controle de Pipeline](/docs/new-relic-control/pipeline-control/gateway/overview) com suporte ao modo de instalação Fluxless e orientações de configuração aprimoradas.
* Documentação de [configuração do Controle do agente](/docs/new-relic-control/agent-control/setup) atualizada com instruções de instalação Fluxless para implantar o Controle do agente sem a dependência do Flux CD.
* Documentação de [monitoramento do ECS EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) e [monitoramento do ECS Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) atualizada com exemplos aprimorados de configuração do OpenTelemetry e opções de implantação.
* Assistente de instalação da integração do MySQL e MariaDB atualizado com atualizações de compatibilidade do MariaDB, novas opções de configuração e orientação aprimorada sobre métricas de desempenho de consulta.

### Pequenas mudanças

* Documentação de [interação da API do Browser](/docs/browser/new-relic-browser/browser-apis/interaction) atualizada com novas opções e descrições detalhadas dos valores de retorno para rastreamento de interação de SPA.
* Documentação de [Medições de VM do Node.js](/docs/apm/agents/nodejs-agent/extend-your-instrumentation/nodejs-vm-measurements) atualizada para esclarecer as diferenças entre os amostradores padrão e nativo.
* Atualizamos a [Interface de Gerenciamento de Usuários e Tarefas](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks) com requisitos de validação do campo de nome para nomes de usuários.
* Atualizamos a [Visão Geral da Integração de Conhecimento](/docs/agentic-ai/knowledge-integration/overview) para melhorar a navegação e esclarecer os recursos de integração da base de conhecimento.
* Documentação [Instalar o Agente Node.js](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent) atualizada com instruções de instalação simplificadas.
* Atualizada a [interface do APM do OpenTelemetry](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui) com o comportamento de fallback `OtelMessagingConsumer1_24` quando `messaging.operation` é `NULL`.
* Removido o suporte descontinuado a logs de instância do RDS dos [logs da AWS no contexto](/docs/logs/logs-context/aws-logs-in-context).
