### Novos documentos

* Adicionada a [OpenTelemetry Collector Observability](/docs/opentelemetry/collector-observability/collector-observability) para fornecer orientações abrangentes de monitoramento para o desempenho e métricas do coletor.
* Adicionado o conjunto de documentação de [Workloads inteligentes](/docs/new-relic-solutions/new-relic-one/workloads/workloads-overview) para fornecer orientação para a criação de workloads com tecnologia de IA e gerenciamento de KPI, incluindo [criação de workloads inteligentes](/docs/new-relic-solutions/new-relic-one/workloads/create-intelligent-workload), [insights de IA](/docs/new-relic-solutions/new-relic-one/workloads/advanced-workload-ai-insights), [gerenciamento de KPI](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-kpis) e [recursos de UI](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-ui).
* Adicionado o [Suporte à API OpenTelemetry](/docs/apm/agents/manage-apm-agents/opentelemetry-api-support) para fornecer orientação abrangente para instrumentação independente de fornecedor em todos os agentes do New Relic APM.

### Grandes mudanças

* A [Workflow Automation](/docs/workflow-automation/introduction-to-workflow) agora está em disponibilidade geral.
* A [Detecção de Outliers](/docs/alerts/create-alert/set-thresholds/outlier-detection) agora está disponível para o público geral.
* Adicionada documentação abrangente de [suporte ao Windows](/docs/new-relic-control/agent-control/overview) para o Agent Control, com orientações sobre instalação, configuração, monitoramento e solução de problemas.

### Pequenas mudanças

* Documentação aprimorada de [monitoramento do Kafka com OpenTelemetry](/docs/opentelemetry/integrations/kafka/self-hosted) com diagramas de arquitetura e orientações expandidas para solução de problemas.
* Atualizamos os [requisitos de compatibilidade do agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) com as faixas de versões suportadas mais recentes para AWS SDK, LangChain, OpenAI e outros pacotes.
* Atualizados os [requisitos de compatibilidade do agente Java](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent) com suporte ao WebLogic estendido para a versão 14.1.x.

### Publicações de novidades

* Adicionada a [postagem de novidades para o New Relic Lens](/whats-new/2026/02/whats-new-02-25-nr-lens/) para anunciar a consulta a fontes de dados externas sem a necessidade de ingestão.
* Adicionado [post de novidades para a GA da Automação de Workflow](/whats-new/2026/02/whats-new-02-24-workflow-automation/) para anunciar workflows de automação no-code com integrações com AWS, Azure, Slack e PagerDuty.
* Adicionada a [postagem de Novidades para Dynamic Runbooks with Notebook](/whats-new/2026/02/%20whats-new-02-25-dynamic-runbooks/) para anunciar a entrada do recurso em Public Preview.
* Adicionada a [postagem de novidades para a página inicial do New Relic](/whats-new/2026/02/whats-new-02-24-new-home-page/) para anunciar o launchpad personalizado, que oferece uma visão curada dos serviços e dados mais importantes para o seu fluxo de trabalho diário específico (Public Preview).

### Notas de versão

* Fique por dentro dos nossos últimos lançamentos:

  * [Agente Ruby v10.2.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-10-2-0):

    * Introduzido Agente Híbrido para suporte a Rastreamento OpenTelemetry com compatibilidade de API independente de fornecedor.
    * Adicionada a configuração de truncamento de backtrace do coletor de erros e a instrumentação da gem Logging.
    * Detecção aprimorada de fornecedor de nuvem com opção de processamento paralelo.

  * [Agente Java v9.1.0](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-910):

    * Adicionada implementação abrangente do Agente Híbrido com compatibilidade com a API do OpenTelemetry para rastreamento, métricas e logs.
    * Suporte aprimorado a Kotlin Coroutines e correção de problemas de propagação de rastreamento W3C.
    * Adicionadas ações de metadados do agente e correções de configuração multihost.

  * [Agente Node.js v13.14.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-14-0):

    * Adicionada ingestão automática de logs de Azure Functions Apps.
    * Implementado suporte a causas de erro e controles de atraso/duração do profiler.
    * Integração do Google Gen AI atualizada para compatibilidade com o método `embedContentInternal`.

  * [Agente Android v7.7.0](/docs/release-notes/mobile-release-notes/android-release-notes/android-770):

    * Adicionado suporte ao Android Gradle Plugin 9.
    * Session Replay móvel aprimorado com amostragem de erros e novas APIs `pauseReplay`/`recordReplay`.
    * Implementadas várias melhorias de desempenho para a funcionalidade de reprodução de sessão.

  * [Agente de infraestrutura v1.72.6](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1726):

    * Versão do fluent-bit atualizada para 4.2.2 para ambientes Windows.

  * [Agente do Browser v1.310.1](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.310.1):

    * Entrada de LCP aprimorada com valores de atribuição de fallback para melhores relatórios de Core Web Vitals.
