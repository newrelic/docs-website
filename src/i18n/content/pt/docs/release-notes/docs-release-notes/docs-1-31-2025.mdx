### Novo documento

* [O logging automático](/docs/browser/browser-monitoring/browser-pro-features/auto-logging) aprimora a observabilidade do front-end para agentes de browser Pro e Pro+SPA, permitindo a coleta automática de mensagens de log de `console` de nível `WARN`.

### Grandes mudanças

* Atualizados os relatórios de compatibilidade para [o agente Node.js.](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)
* Adicionado um método no [monitoramento de integridade do contêiner Docker do KTranslate](/docs/network-performance-monitoring/advanced/ktranslate-container-health) para verificar o log localmente do pacote Linux.
* [Blocos de rede](/docs/new-relic-solutions/get-started/networks/) atualizados com novos intervalos de IP efetivos para melhorar conectividade e segurança.
* Atualizada a [versão do protocolo OTLP](/docs/opentelemetry/best-practices/opentelemetry-otlp/#otlp-version-support) que o New Relic usa.
* [APM atualizado: Relatar evento personalizado e atributo](/docs/data-apis/custom-data/custom-events/apm-report-custom-events-attributes/#php-att) para incluir a capacidade de aumentar o número máximo de eventos personalizados no agente PHP de 30.000 para 100.000 por minuto.
* Foi adicionada uma chamada para a descontinuação e o fim do suporte para [o pacote NewRelic Xamarin NuGet](/docs/mobile-monitoring/new-relic-mobile-xamarin/monitor-your-xamarin-application/).

### Pequena mudança

* Atualizadas as versões suportadas do Python.
* Os valores de atributo especificados na [configuração do perfil de usuário do Okta](/docs/accounts/accounts/automated-user-management/okta-scimsso-application-configuration/) agora diferenciam maiúsculas de minúsculas.
* Os valores [do tipo de usuário da API SCIM](/docs/accounts/accounts/automated-user-management/tutorial-manage-users-groups-scim/#manage-user-type) agora diferenciam maiúsculas de minúsculas.
* Atualizadas as versões suportadas do agente PHP.
