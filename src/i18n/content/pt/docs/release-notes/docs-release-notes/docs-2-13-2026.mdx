### Grandes mudanças

* Atualizada a [documentação do Pipeline Control Gateway](/docs/new-relic-control/pipeline-control/gateway/overview) com revisões extensas para a v2.0.0, incluindo novos processadores (filtro, amostragem, transformação), guia da interface de usuário, visão geral da configuração YAML e diferenças de esquema de dados.
* Atualizada a [instalação da integração OTel do Elasticsearch](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-install) e a [visão geral](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) com melhorias abrangentes nas orientações de instalação e configuração.
* Atualizada a [configuração do agente Java](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file) com documentação abrangente para suporte à configuração do Agente Híbrido.
* Atualizada a [documentação de logs do agente Go](/docs/apm/agents/go-agent/configuration/go-agent-logging) com orientações abrangentes para configurar logs no agente Go.
* Atualizado o [catálogo de ações](/docs/workflow-automation/setup-and-configuration/actions-catalog/actions-catalog) de automação de workflow com terminologia revisada em vários arquivos de ação para maior consistência.

* Documentação da [integração do Amazon ECS](/docs/infrastructure/elastic-container-service-integration/introduction-amazon-ecs-integration) atualizada com correções de links.
