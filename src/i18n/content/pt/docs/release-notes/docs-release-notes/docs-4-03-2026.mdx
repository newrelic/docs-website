### Novos documentos

* Adicionada a [Fleet control API](/docs/new-relic-control/fleet-control/overview/) para gerenciar configurações do agente de infraestrutura em escala em toda a sua frota.
* Adicionado o [guia de transição do Chrome Latest para monitores sintéticos](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-browser-monitors-to-chrome-latest/) para atualizar os monitores sintéticos para a versão Chrome Latest e Node.js 22.
* Adicionado o [Boletim de segurança NR26-04](/docs/security/new-relic-security/security-bulletins/security-bulletin-nr26-04) com procedimentos para tratar a vulnerabilidade crítica de use-after-free do Google Chrome (CVE-2026-5281) que afeta o Synthetic Monitoring.
* Adicionadas [ações do New Relic Agent](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent) para executar agentes configurados na Agentic Platform através da automação de fluxo de trabalho.

### Grandes mudanças

* Atualizada a [documentação do AWS Lambda e OpenTelemetry](/docs/serverless-function-monitoring/aws-lambda-monitoring/opentelemetry/lambda-opentelemetry-troubleshooting/) com uma seção abrangente de solução de problemas do OpenTelemetry Lambda, cobrindo problemas de métricas do CloudWatch e problemas de ID de requisição.

### Pequenas mudanças

* Atualizadas as [notas de versão do agente do Browser](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/) com informações de compatibilidade para a v1.312.0 e v1.312.1.
* Atualizamos a [Visão geral do rastreamento de alterações](/docs/change-tracking/overview/) com o esclarecimento de que as alterações podem ser associadas a qualquer entidade na plataforma New Relic.
* Atualizado o [relatório de compatibilidade do agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/) com as versões atualizadas dos pacotes.
* Atualizados os [requisitos de compatibilidade do agente .NET](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) com as versões compatíveis verificadas mais recentes para vários pacotes, incluindo `Microsoft.Azure.Cosmos`, `Microsoft.Data.SqlClient`, `MongoDB.Driver` e outros.
* Atualizadas as [notas de versão do agente .NET](/docs/release-notes/agent-release-notes/net-release-notes/) com correções de categorização ausentes.
* Estrutura de navegação atualizada para melhorar a experiência do usuário nas seções da documentação.
* Atualizada a [documentação do Mobile session replay](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/) com a funcionalidade de visualização de bloqueio para `NRConditionalMaskView` e orientações abrangentes sobre mascaramento vs. bloqueio.
* Atualizada a [documentação de retenção de dados](/docs/data-apis/manage-data/manage-data-retention/) com o gerenciamento de retenção no escopo da organização e explicações sobre as guias Conta vs. Organização.
* Atualizada a [documentação dos módulos de extensão do agente Java](/docs/apm/agents/java-agent/instrumentation/extension-additional-instrumentation-modules/) com a remoção do módulo incubador HikariCP 2.4.0 descontinuado, pois agora ele está incluído no agente.
* Atualizada a [documentação de integração de conhecimento](/docs/agentic-ai/knowledge-integration/overview/) com nomes de permissão atualizados, mudando da função &quot;Org Product Admin&quot; para &quot;Organization Manager&quot;.
* Dicionário de atributos atualizado com o novo tipo de evento `NrComputeUsage` e definições de atributos associadas.
* Atualizamos a [documentação de gerenciamento de ingestão de dados](/docs/data-apis/manage-data/manage-data-coming-new-relic/) com recursos de previsão de uso de dados e funcionalidade de previsão de uso para clientes do Advanced Compute.
* A [documentação da integração do Kubernetes](/docs/kubernetes-pixie/kubernetes-integration/get-started/kubernetes-integration-compatibility-requirements/) foi atualizada com suporte ao Oracle Kubernetes Engine (OKE) e orientações de configuração para o auto-attach do APM.
* Atualizada a [documentação de segurança do Java 2 do agente Java](/docs/apm/agents/java-agent/additional-installation/install-java-agent-java-2-security/) com suporte ao Solr v9 e orientações de configuração do gerenciador de segurança.
* Atualizado o [suporte SCIM para gerenciamento automatizado de usuários](/docs/accounts/accounts/automated-user-management/scim-support-automated-user-management/) com suporte para o campo `title` para capturar informações de cargo.

### Publicações de novidades

* Adicionados [Novos recursos de Dashboards](/whats-new/2026/03/whats-new-3-28-in-dashboards) para anunciar links de widgets, descrições de widgets, controles de visibilidade de variáveis de template e variáveis de template dinâmicas em widgets de markdown e títulos.

### Notas de versão

* Fique por dentro dos nossos últimos lançamentos:

  * [Agente Node.js v13.15.0, v13.16.0, v13.17.0, v13.18.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/):

    * Stream de servidor grpc instrumentado para vincular o contexto assíncrono aos manipuladores de eventos de stream.
    * Múltiplos módulos de instrumentação foram refatorados para usar o tratamento de eventos baseado em assinantes.
    * Atualizados os assinantes do Langchain core para instrumentar corretamente os métodos de classe de ferramentas.
    * Compatibilidade aprimorada com pacotes npm e frameworks mais recentes.

  * [Agente Go v3.43.0](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-3-43-0):

    * Adicionado suporte a UnWrap para o response writer de substituição e suporte para URL de função Lambda.
    * Integração aprimorada do Opensearch/Elasticsearch com atributos adicionais de span.
    * Adicionado suporte para cláusulas `WITH` no parsing de SQL e corrigidos problemas de teste HTTP.
    * Atualizações de segurança, incluindo a atualização de versão do `nrgrpc` e patches de segurança de dependências.

  * [Agente PHP v12.6.0.34](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-12-6-0-34):

    * Adicionado suporte ao Laravel 13 e detecção de PHP Fiber para controle de instrumentação.
    * Segurança atualizada com a versão 1.26.1 do Go e chave de assinatura RPM rotacionada.
    * Corrigida a instalação do Debian/Ubuntu e melhorada a lógica de análise de SQL.

  * [Agente de infraestrutura v1.72.7, v1.72.8, v1.72.9, v1.73.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/):

    * Adicionado suporte abrangente a contas de serviço de menor privilégio no Windows.
    * Instalador aprimorado para preservar contas de serviço existentes durante atualizações.
    * Segurança aprimorada com suporte a Conta de Serviço Gerenciada de Grupo (gMSA).
    * Adicionado do logging de depuração e recursos de segurança de rollback para instalações.

  * [Aplicativo móvel New Relic Android v5.41.5](/docs/release-notes/mobile-apps-release-notes/new-relic-android-release-notes/new-relic-android-5415):

    * Correções de bugs e melhorias.

  * [Extensão do Lambda v2.4.6](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.6):

    * Adicionadas variáveis de ambiente de configuração de proxy e controle de logs.
    * Corrigida a classificação de logs JSON e melhorado o tratamento de IDs de requisição.
    * Geração aprimorada de números aleatórios em `TraceIDGenerator`.

  * [NRDOT Collector v1.12.0](/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01):

    * Atualizado o core beta do OpenTelemetry para a v0.148.0 para um desempenho aprimorado.
    * Adicionado o spanmetricsconnector aos recursos experimentais.
    * Corrigidas vulnerabilidades de segurança, incluindo CVE-2026-33186 e CVE-2026-34040.
    * Chave pública de assinatura GPG rotacionada para maior segurança.
