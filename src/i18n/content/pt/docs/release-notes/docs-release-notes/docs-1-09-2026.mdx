### Grandes mudanças

* Atualizado [Instale o New Relic para Go](/docs/apm/agents/go-agent/installation/install-new-relic-go) com um guia de instalação passo a passo abrangente e exemplos para padrões comuns de aplicativos Go.
* Adicionado [Configurar e Instalar o Agent Control](/docs/new-relic-control/agent-control/setup) para fornecer orientação completa sobre instalação e configuração para clusters Kubernetes e hosts Linux.
* Aprimorada a documentação da API [setUserId](/docs/browser/new-relic-browser/browser-apis/setuserid) com funcionalidade de redefinição de sessão e comportamento de rastreamento de sessão atualizado.

### Pequenas mudanças

* Adicionadas informações de compatibilidade do agente PHP para [requisitos de compatibilidade do agente PHP](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements).
* Atualizada a documentação de [instalação do SPM](/docs/mobile-monitoring/new-relic-mobile-ios/installation/spm-installation) com orientação aprimorada.
* Aprimorada a documentação de [instalação do CocoaPods](/docs/mobile-monitoring/new-relic-mobile-ios/installation/cocoapods-installation) com instruções aprimoradas.
* Atualizada a documentação de [Filtrar eventos de solicitação AJAX](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events) com suporte a curingas para lista de negação.
* Atualizado [Atualizar para a experiência APM](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/upgrade-to-apm-experience) para monitoramento de funções Lambda.
