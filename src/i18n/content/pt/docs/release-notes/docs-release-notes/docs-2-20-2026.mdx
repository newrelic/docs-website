### Novos documentos

* Adicionada a [instrumentação OpenTelemetry do RabbitMQ](/docs/opentelemetry/integrations/rabbitmq/overview) para fornecer orientações sobre o monitoramento do RabbitMQ com OpenTelemetry.
* Adicionado o [monitoramento do Docker com OpenTelemetry](/docs/opentelemetry/integrations/docker-monitoring/overview) para fornecer orientações sobre o monitoramento de contêineres Docker com OpenTelemetry.
* Adicionada a documentação de [Parsing No-Code](/docs/logs/ui-data/parsing) para fornecer orientações sobre o parsing de logs sem código personalizado.
* Adicionada a documentação do [Adaptive Telemetry Processor (ATP)](/docs/opentelemetry/nrdot/atp/overview) para fornecer orientações sobre a otimização da coleta de dados de telemetria.
* Adicionado o [catálogo de ações Azure Execute](/docs/workflow-automation/setup-and-configuration/actions-catalog/azure/azure-execute-api) para fornecer orientação para ações de automação de fluxo de trabalho do Azure.
* Adicionadas notas de versão de streaming de mídia para as integrações dos players [Bitmovin e Shaka](/docs/release-notes/streaming-browser-release-notes).

### Grandes mudanças

* Atualizamos as [Variáveis de template de dashboard](/docs/query-your-data/explore-query-data/dashboards/dashboard-template-variables) com exemplos para títulos de widgets e widgets de markdown.
* Atualizado [OpenTelemetry Kafka Strimzi](/docs/opentelemetry/integrations/kafka/kubernetes-strimzi) para migrar da integração OpenTelemetry para a integração Prometheus.
* Atualizada a [documentação do OpenTelemetry Kafka](/docs/opentelemetry/integrations/kafka/overview) com a configuração do coletor NRDOT.
* Documentação de alertas atualizada para substituir a terminologia &apos;incidente(s)&apos; por &apos;evento(s) de alerta&apos; para maior consistência.

### Publicações de novidades

* Adicionada a [postagem de novidades sobre No-Code Parsing](/whats-new) para anunciar a disponibilidade geral do recurso de No-Code Parsing.

### Pequenas mudanças

* Adicionados exemplos de código em Swift e Objective-C à [documentação do agente iOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios).
* Adicionada documentação de rotação de chave RPM para a [instalação do agente PHP](/docs/apm/agents/php-agent/installation/php-agent-installation-rpm-key-rotation).
* Adicionada configuração básica otimizada do NGINX OpenTelemetry à [documentação do NGINX OTel](/docs/opentelemetry/integrations/nginx/nginx-otel-overview).
* Instruções atualizadas de [vinculação de aplicações APM](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-apm-applications-kubernetes) para Kubernetes.
* Documentação de [ofertas globais de suporte técnico](/docs/licenses/license-information/general-usage-licenses/global-technical-support-offerings) atualizada.
* Atualizada a [documentação de compatibilidade do agente .NET](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements) com o suporte mais recente ao framework.
* Atualizada a [documentação de compatibilidade do agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) com o suporte aos módulos mais recentes.
