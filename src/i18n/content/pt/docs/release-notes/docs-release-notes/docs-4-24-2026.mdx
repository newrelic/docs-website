### Novos documentos

* Adicionados guias de instalação do Sintéticos com suporte a dual stack para ambientes [Docker](/docs/synthetics/synthetic-monitoring/private-locations/install-synthetics-dual-stack-docker/), [Kubernetes](/docs/synthetics/synthetic-monitoring/private-locations/install-synthetics-dual-stack-kubernetes/) e [Podman](/docs/synthetics/synthetic-monitoring/private-locations/install-synthetics-dual-stack-podman/) para suportar testes tanto em IPv4 quanto em IPv6.
* Adicionado o [guia de transição de runtime de localizações privadas de Sintéticos](/docs/synthetics/synthetic-monitoring/private-locations/runtime-transition-guide/) para migrar monitores para o Node.js 22 e Chrome 146, com estratégias de migração in-place e paralelas, consultas NRQL para monitoramento da transição e correções de monitores assistidas por IA.
* Adicionado o guia de comparação [Azure Monitor vs. Sondagem do Azure](/docs/infrastructure/amazon-integrations/get-started/azure-monitor-vs-azure-polling/) para ajudar os usuários a escolher o método de integração do Azure correto para coletar métricas do Azure no New Relic.

### Grandes mudanças

* Documentação da [Inteligência de Custos na Nuvem (CCI)](/docs/cci/overview/) atualizada de forma abrangente com fluxos de trabalho da interface revisados, gerenciamento de orçamento atualizado, nova página de visão geral de custos, insights de custos do Kubernetes, consultas NRQL para o CCI e guias de configuração para AWS, Azure e GCP.

### Pequenas mudanças

* Atualizados os [requisitos de compatibilidade do agente .NET](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) para incluir suporte ao framework Hangfire.
* Atualizada a [visão geral do eBPF](/docs/ebpf/overview/) com informações de conformidade com o FedRAMP.
* Integração [NGINX OpenTelemetry Kubernetes](/docs/infrastructure/other-infrastructure-integrations/nginx/nginx-otel-kubernetes/) atualizada com instruções de instalação do Helm para a instrumentação do NGINX OTel.
* Atualizada a [configuração do agente Python](/docs/apm/agents/python-agent/configuration/python-agent-configuration/) com configurações do Agente Híbrido para suporte à API do OpenTelemetry.
* Atualizados os [Limites da API métrica e atributos restritos](/docs/data-apis/ingest-apis/metric-api/metric-api-limits-restricted-attributes/) removendo a documentação desatualizada de limite de pontos de dados por minuto (DPM) nas páginas da API métrica, do Prometheus e de otimização de dados.
