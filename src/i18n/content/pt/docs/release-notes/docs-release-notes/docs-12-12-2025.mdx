### Notas de versão

* Fique por dentro dos nossos últimos lançamentos:

  * [Extensão Lambda v2.4.0](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.0):

    * Reescrita completa de Go para Rust para melhor desempenho e segurança de memória.
    * Adicionado suporte ao modo APM para a função Lambda, permitindo o envio de relatórios diretamente para a plataforma New Relic APM.
    * Redução do tempo de inicialização a frio em 20-40% com compilação otimizada em Rust.

  * [AgenteNode.js v13.8.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-8-0):

    * Adicionado suporte para instrumentação @langchain/AWS.
    * Adicionada a métrica de rastreamento de granularidade parcial.

  * [AgenteNode.js v13.7.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-7-0):

    * Adicionado suporte para links OTEL span.
    * Adicionada a capacidade de configurar o amostrador baseado em proporção.
    * Instrumentação koa atualizada para @koa/router\@15.0.0+.
    * Normalização corrigida do horário OTEL.

  * [AgentePython v11.2.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-110200):

    * Adicionado suporte para instrumentação Strands.
    * Adicionado suporte para AWS Bedrock Converse Streaming.
    * Adicionado suporte para os métodos hybrid\_search e msetex do Redis.
    * Problema de consumo do cabeçalho ASGI corrigido.

  * [agente de infraestrutura v1.71.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1713):

    * Atualização da tag docker Alpine para a versão 3.23.
    * Versão do Go atualizada para 1.25.5.

  * [agente de infraestrutura v1.71.2](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1712):

    * Versão atualizada do Fluent Bit com correções para as vulnerabilidades CVE-2025-12972, CVE-2025-12970, CVE-2025-12978, CVE-2025-12977 e CVE-2025-12969.
    * Foram removidas as versões de 32 bits do Windows.
    * Correção dos contadores de desempenho da CPU em sistemas que não sejam em inglês.

  * [Integração Kubernetes v3.51.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-51-0):

    * Integração atualizada com novos recursos e melhorias.
