### Novos documentos

* Adicionado o [fluxo de trabalho do Jira do Security RX](/docs/vulnerability-management/workflows/jira) para fornecer orientação sobre a criação e sincronização automática de ticket do Jira para vulnerabilidades de aplicativo detectadas.
* Adicionada uma [ação de parada de máquina virtual do Azure](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-vm-stop) ao catálogo de ações para que você possa desligar as VMs do Azure como parte de seus fluxos de trabalho automatizados.
* Adicionados detalhes sobre [como salvar e gerenciar as preferências do dashboard](/docs/streaming-video-&-ads/view-data-in-newrelic/user-preferences) para explicar como a New Relic salva e restaura a configuração do dashboard de Video Overview e Ad Overview entre as sessões.
* Adicionadas regras de ofuscação em relação ao agente de Streaming Video &amp; Ads para [Android](/docs/streaming-video-&-ads/installation/android/obfuscation-rules), [navegador](/docs/streaming-video-&-ads/installation/browser/obfuscation-rules), [iOS](/docs/streaming-video-&-ads/installation/ios/obfuscation-rules) e [Roku](/docs/streaming-video-&-ads/installation/roku/obfuscation-rules) para fornecer orientação sobre o mascaramento de dados confidenciais com regras baseadas em regex antes da transmissão.

### Grandes mudanças

* Atualizada a [visão geral da integração do Security RX](/docs/vulnerability-management/getting-started/integrations/overview) e a [visão geral do aplicativo](/docs/vulnerability-management/applications/overview) para definir &apos;10.51.1&apos; como a versão base para a integração .NET do Security RX.
* Atualizada a documentação do CCI com [preços de ingestão de faturamento por provedor](/docs/cci/cci-pricing), [orientação de escopo de integração de conta pai vs. subconta](/docs/cci/overview/#choose-an-integration-scope) e [pré-requisitos de configuração do Azure](/docs/cci/get-started/azure-cci).

### Pequenas mudanças

* Aprimorada a [ação do agente New Relic](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent) no catálogo de ações com descrições de contexto adicionadas.
