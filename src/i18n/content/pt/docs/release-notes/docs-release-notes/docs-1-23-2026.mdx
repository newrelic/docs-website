### Grandes mudanças

* Atualizou a documentação do eBPF com [nova matriz de suporte](/docs/ebpf/requirements), [instalação do agente eBPF no Linux](/docs/ebpf/linux-installation) e [instalação do Kubernetes](/docs/ebpf/k8s-installation) com alterações de configuração para remoção do cliente.
* Documentação de controle de acesso refinado para gerenciamento de segredos [foi atualizada](/docs/apis/nerdgraph/examples/nerdgraph-api-secret-management-service) para melhorar o procedimento de controles de acesso à API.
* Atualizou [requisitos do agente Ruby e frameworks suportados](/docs/apm/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) com as alterações da versão 10.x.
* Adicionada [documentação de integração Nuxt.js](/docs/browser/browser-integrations/nuxt-integration/) com suporte atualizado para Nuxt 3+ e formatação aprimorada.

### Pequenas mudanças

* Documentação aprimorada da [API notice\_error do agente Python](/docs/apm/agents/python-agent/python-agent-api/noticeerror-python-agent-api) com suporte a instâncias de exceção.
* Guia de introdução aos [logs do navegador](/docs/browser/browser-monitoring/browser-pro-features/browser-logs/get-started/) atualizado com melhorias na documentação de logging.
* Documentação de rastreamento de sessão corrigida para [exploração do ciclo de vida da página da web](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle).
* Requisitos de compatibilidade do agente [Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) atualizados com o relatório de compatibilidade mais recente.
* Configuração do [NGINX OpenTelemetry](/docs/opentelemetry/nginx/nginx-otel-host) atualizada com configuração stub\_status aprimorada.

### Novidades

* [Operações de painel para usuários somente leitura](/whats-new/2026/01/whats-new-01-19-read-only-dashboards) - As permissões de função somente leitura padrão para gerenciamento de painel serão limitadas ao acesso somente visualização a partir de 30 de abril de 2026.
