### Grandes mudanças

* [Documentação de controle do agente New Relic Control](/docs/new-relic-control/agent-control/overview) atualizada com suporte abrangente para hosts Windows, incluindo novas opções de configuração, requisitos de rede e orientações de solução de problemas.
* Documentação de [monitoramento de navegador PHP](/docs/apm/agents/php-agent/features/browser-monitoring-php-agent) aprimorada com instruções detalhadas para adicionar atributos personalizados aos dados de monitoramento de navegador.
* Atualizada a [documentação de configuração do agente Python](/docs/apm/agents/python-agent/configuration/python-agent-configuration) com exemplos de configuração de amostrador de agente híbrido.

### Pequenas mudanças

* Atualizada a [documentação da integração com o AWS CloudWatch](/install/aws-cloudwatch) para esclarecer que cada conta da AWS requer um sink por região para monitoramento multirregional.
* Documentação de [instalação do eBPF no Kubernetes](/docs/ebpf/k8s-installation) e [instalação no Linux](/docs/ebpf/linux-installation) atualizada com informações de configuração do nome do aplicativo New Relic.
* Atualizada a [integração do RabbitMQ com OpenTelemetry](/docs/opentelemetry/integrations/rabbitmq/overview) com suporte ao NRDOT Collector para ambientes Kubernetes e auto-hospedados.
* Atualizada a [documentação do modo de alta segurança](/docs/apm/agents/manage-apm-agents/configuration/high-security-mode).
* [Documentação da integração de monitoramento de portas](/docs/infrastructure/host-integrations/host-integrations-list/port-monitoring-integration) atualizada.
* [Documentação de equipes](/docs/service-architecture-intelligence/teams/teams) atualizada.
* Atualizada a [documentação de gerenciamento de equipes](/docs/service-architecture-intelligence/teams/manage-teams).
* Atualizada a [documentação de compatibilidade do agente Java](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent).
* Atualizada a [documentação de compatibilidade do agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent).

### Novidades

* Adicionado [Novidades para o suporte ao Windows do New Relic Control](/whats-new/2026/02/whats-new-23-02-windows-support/) para anunciar o suporte a hosts Windows e a seleção de agentes aprimorada no New Relic Control (Public Preview).
* Atualizado [Novidades para Observabilidade do Coletor](/whats-new/2026/02/whats-new-02-24-collector-observability/).
