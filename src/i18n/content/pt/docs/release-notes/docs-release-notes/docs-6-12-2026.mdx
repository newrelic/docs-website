### Novos documentos

* Adicionado [Configurar um Proxy para o Agente eBPF](/docs/ebpf/proxy-configuration) para fornecer orientação sobre o roteamento da telemetria do agente eBPF por meio de proxies corporativos em ambientes de rede restrita.
* Adicionadas as [licenças da Extensão Lambda (Rust)](/docs/licenses/product-or-service-licenses/new-relic-serverless/lambda-extension-rust-licenses) para documentar as licenças de terceiros usadas na Extensão Lambda (Rust) da New Relic.

### Grandes mudanças

* Atualizados os [requisitos do agente eBPF](/docs/ebpf/requirements) com um limite superior de versão do kernel, especificando suporte para as versões v5.8 a v6.14 do kernel do Linux em ambas as arquiteturas x86-64 e arm64.
* Atualizamos a [instalação do eBPF no Kubernetes](/docs/ebpf/k8s-installation) e a [instalação no Linux](/docs/ebpf/linux-installation) com documentação abrangente de suporte a proxy, esclarecendo que proxies HTTP CONNECT usando o esquema `http://` são suportados e que os dados de telemetria permanecem criptografados com TLS de ponta a ponta.
* Atualizamos os [requisitos de compatibilidade do agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) com uma nova seção Instrumentação de Agente Híbrido, documentando o suporte ao Next.js v16+ via instrumentação nativa do OpenTelemetry a partir do agente v14.1.0.
* Atualizada a [configuração do agente SRE](/docs/agentic-ai/sre-agent/sre-agent-setup) para remover o requisito do SKU de Advanced Compute, esclarecendo os pré-requisitos para configurar o agente SRE.
* Páginas de referência da [API NerdGraph de Sintéticos](/docs/apis/nerdgraph/examples/synthetics-api/overview) atualizadas para todos os tipos de monitor.

### Pequenas mudanças

* Atualizamos os [requisitos de compatibilidade do AI Monitoring](/docs/ai-monitoring/compatibility-requirements-ai-monitoring) para direcionar os leitores à [página de compatibilidade do agente Java](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent/#ai-monitoring) para obter a lista completa de bibliotecas de AI Monitoring suportadas.
* Atualizado [Monitore seu Aplicativo .NET MAUI](/docs/mobile-monitoring/new-relic-mobile-maui-dotnet/monitor-your-net-maui-application) com pequenas correções de formatação.
* Atualizamos o documento [Fazer Downgrade ou Cancelar sua Conta](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) com um aviso importante em destaque sobre os efeitos em cascata do cancelamento da conta pai nas subcontas.
* Atualizado [Migrar do Polling de API para o CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/aws-integration-for-metrics/migrate-from-api-polling-to-cwms) para esclarecer o título da seção de validação de dados.
* Adicionado o bloco [Notas de versão do agente SRE](/docs/release-notes/sre-agent-release-notes) à página de índice de notas de versão.
* Adicionado histórico de lançamentos anteriormente ausente para agentes de streaming de Video &amp; Ads em [navegador](/docs/release-notes/streaming-browser-release-notes), [mobile](/docs/release-notes/streaming-mobile-release-notes) e [outras plataformas](/docs/release-notes/streaming-others-release-notes).
