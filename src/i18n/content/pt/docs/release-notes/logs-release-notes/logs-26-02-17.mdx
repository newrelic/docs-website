### Suporte para Fluent Bit 4.2.2

Os usuários agora receberão a versão 4.2.2 do Fluent Bit. Para mais detalhes, consulte os [anúncios do Fluent Bit](https://fluentbit.io/announcements/).

Esta versão inclui o New Relic Fluent Bit Output Plugin 3.4.0 com atualizações de segurança críticas e melhorias de compatibilidade da plataforma.

### Adicionado

* Suporte para SLES 15.6 e 15.7
* Plugin de saída New Relic Fluent Bit 3.4.0 com segurança e compatibilidade aprimoradas

### Mudado

* O agente de infraestrutura recomenda os pacotes do Fluent Bit 4.2.2
* [A imagem Docker do plugin de saída do Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) usa o Fluent Bit 4.2.2 com o plugin de saída 3.4.0
* imagem base do aws-for-fluent-bit atualizada da versão 3.1.0 para 3.2.1

### Fixo

* Problemas de compatibilidade do GLIBC no RHEL8 e Amazon Linux 2 usando compilação cruzada com Zig visando o GLIBC 2.17

### Notas

Para se manter atualizado com as correções e melhorias mais recentes, assine nosso [feed RSS de Logs](/docs/release-notes/logs-release-notes/).
