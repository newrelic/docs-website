### Notas

* Esta versão adiciona suporte para fechar um evento de alerta aberto para usuários autorizados e também adiciona mais suporte a deep links para links compartilhados do New Relic One.
* Esta versão também inclui uma verificação de feature flag para plugins que não exibirá os plugins como um item de menu, a menos que a conta tenha uma feature flag habilitando os plugins.

### Correções

* Telas quebradas no modo escuro.
* Notificações descartadas.
* Filtragem de dashboard.
