### Notas

Esta versão contém notificações aprimoradas, com suporte para a seleção de sons exclusivos para diferentes tipos de notificações, além de exibir uma imagem de gráfico para eventos de alerta.

### Novo recurso

* Selecione diferentes sons de notificação para notificações push no aplicativo.
* Exibe a imagem do gráfico de evento de alerta aberto / fechado na notificação.

### Correções de bugs

* Correção para filtrar apmApplicationIds e apmApplicationNames corretamente na Infraestrutura.
* Corrigida uma falha ao tentar pesquisar em componentes de plugin.
* Corrigido um travamento no Radar.
