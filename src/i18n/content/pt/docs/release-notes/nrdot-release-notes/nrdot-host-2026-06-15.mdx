### Recurso

* Valida que o caso de uso de host funciona corretamente com `nrdot-collector:1.17.0`.
* Substituído `hostmetrics` por `host_metrics`, `resourcedetection` por `resource_detection` e `metricstransform` por `metrics_transform` devido a descontinuações. Os `hostmetricsreceiver`, `resourcedetectionprocessor` e `metricstransformprocessor` alteraram seus identificadores `metadata.yaml :: type` para usar `snake_case`, o que mudou como são referenciados na configuração.
