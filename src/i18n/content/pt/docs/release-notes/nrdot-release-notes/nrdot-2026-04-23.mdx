## Registro de alterações

Este lançamento marca a remoção da distribuição `nrdot-collector-k8s` e `nrdot-collector-host`. A última versão publicada dessas distribuições é `1.12.0`. Em vez disso, migre para usar o `nrdot-collector`, que é um superconjunto em termos de funcionalidade e atua como um substituto direto, com as duas distros removidas sendo implementadas como [casos de uso](https://github.com/newrelic/nrdot-collector-releases/blob/1.13.0/distributions/nrdot-collector/README.md#use-cases).
