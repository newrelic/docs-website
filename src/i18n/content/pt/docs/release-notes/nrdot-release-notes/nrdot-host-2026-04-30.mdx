## Registro de alterações

Primeiras notas de versão dedicadas para o caso de uso de [&apos;Monitoramento de host&apos;](https://github.com/newrelic/nrdot-collector-releases/blob/main/distributions/nrdot-collector/README.md#use-cases), que têm como escopo alterações relevantes na configuração ou `nrdot-collector`. O [coletor nrdot](https://docs.newrelic.com/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01/) continuará a incluir a configuração necessária para viabilizar este caso de uso. No entanto, com `nrdot-collector` suportando cada vez mais casos de uso, queremos fornecer notas de versão adicionais que sejam mais fáceis de consumir para alguém que usa o NRDOT para um propósito específico, como monitoramento de host.
