## Notas

* `Java` melhorias no suporte ao runtime - detecção automática e roteamento de modo adequado

* Confiabilidade aprimorada no processamento de logs - garante que todos os logs sejam devidamente etiquetados antes do envio

* Uso de recursos reduzido ao otimizar a criação de canais

* Eficiência de `log processing` aprimorada

## Fixo

* Relatório de versão do agente Java

* Estampagem de metadados de log para logs `pre-invoke`

* Mensagens de aviso desnecessárias em `APM mode`
