## Novo recurso

* A Diagnostics CLI agora oferece suporte à região de dados do Japão da New Relic. Se a sua conta estiver na região do Japão, especifique a flag `region` ao executar a CLI ([#263](https://github.com/newrelic/newrelic-diagnostics-cli/pull/263)):

  ```bash
  ./nrdiag -region JP
  ```

  A flag `region` aceita `US`, `EU` ou `JP` (sem distinção entre maiúsculas e minúsculas) e o padrão é `US`.

## Atualizações de dependência

* Atualizado `golang.org/x/net` de 0.33.0 para 0.55.0 ([#264](https://github.com/newrelic/newrelic-diagnostics-cli/pull/264)).
