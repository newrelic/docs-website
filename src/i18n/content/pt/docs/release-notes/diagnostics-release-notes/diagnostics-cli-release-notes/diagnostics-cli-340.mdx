## Atualizações de tarefas

* Uma nova opção de linha de comando `-ac-agents-namespace` foi adicionada para especificar um namespace para o agente Agent Control ([#243](https://github.com/newrelic/newrelic-diagnostics-cli/pull/243)). As seguintes tarefas foram atualizadas para usar o novo namespace:

  * K8s/Resources/Config
  * K8s/Resources/Daemonset
  * K8s/Resources/Deploy
  * K8s/Resources/Pods

## Correções

* Erros relacionados ao GLIBC, como `/lib64/libc.so.6: version 'GLIBC_2.34' not found`, foram resolvidos ([#244](https://github.com/newrelic/newrelic-diagnostics-cli/pull/244)).
