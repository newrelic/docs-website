## Atualizado

* Adicionado o atributo `contentBitrate` aos eventos de vídeo. Isso utiliza `segmentationBitrate` se disponível e recorre a `streamBitrate` quando necessário.
* Adicionado o atributo `adBitrate` aos eventos de anúncio. A taxa de bits do anúncio agora é extraída usando vários métodos e padronizada em bps (bits por segundo).
* Adicionado o atributo `timeSinceLastError` aos eventos de erro, reportando o tempo decorrido desde o evento de erro anterior.
