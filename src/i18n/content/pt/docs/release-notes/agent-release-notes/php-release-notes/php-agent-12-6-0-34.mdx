## Agente PHP da New Relic v12.6.0.34

### Novo recurso

* Laravel 13 verificado e suportado.
* A instrumentação será desativada por requisição quando o uso de [PHP Fiber](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/#fibers) for detectado. ([#1178](https://github.com/newrelic/newrelic-php-agent/pull/1178))

### Atualizações de segurança

* Atualiza a versão do Go para 1.26.1 ([#1179](https://github.com/newrelic/newrelic-php-agent/pull/1179)). Resolve o [#1173](https://github.com/newrelic/newrelic-php-agent/issues/1173).
* Atualizar as dependências golang grpc ([#1185](https://github.com/newrelic/newrelic-php-agent/pull/1185)).

### Correções de bugs

* Corrigido o nível de backoff ao tentar novamente as conexões com o daemon ([#1169](https://github.com/newrelic/newrelic-php-agent/pull/1169))
* Corrigida a instalação do newrelic.ini em mods-available no Debian/Ubuntu ([#1170](https://github.com/newrelic/newrelic-php-agent/pull/1170)). Resolve ([https://github.com/newrelic/newrelic-php-agent/issues/1168](https://github.com/newrelic/newrelic-php-agent/issues/1168)).
* Lógica de análise de SQL aprimorada ([#1164](https://github.com/newrelic/newrelic-php-agent/pull/1164))

### Declaração de apoio

* Conforme [comunicado anteriormente](https://docs.newrelic.com/docs/apm/agents/php-agent/installation/php-agent-installation-rpm-key-rotation/), para melhores práticas e segurança, a chave de assinatura do rpm foi rotacionada e a página de [instalação do rpm](https://docs.newrelic.com/docs/apm/agents/php-agent/installation/php-agent-installation-aws-linux-redhat-centos/) reflete as alterações mais recentes.

* O suporte do Agente PHP para as seguintes versões de bibliotecas &amp; frameworks será descontinuado e a New Relic recomenda que você atualize para a versão mais recente suportada pelo fornecedor:

  * O suporte ao Drupal 7 será descontinuado em todas as versões do agente PHP após 30 de setembro de 2026
  * O suporte ao Drupal 9 será descontinuado em todas as versões do Agente PHP após 30 de setembro de 2026.

* Conforme comunicado anteriormente, o suporte do PHP Agent para as seguintes versões de bibliotecas &amp; frameworks está descontinuado em todos os lançamentos do PHP Agent após 28 de fevereiro de 2026:

  * Para as seguintes versões de bibliotecas &amp; frameworks, a New Relic recomenda que você atualize para a versão mais recente suportada pelo fornecedor:

    * Symfony 4.x
    * Laravel 7.x, 9.x, 10.x
    * Slim 3.x

  * O suporte para a seguinte biblioteca e framework será descontinuado em todas as versões:

    * MediaWiki
    * Estrutura Zend
    * Lumen

* A New Relic recomenda que você atualize o agente regularmente e, no mínimo, a cada 3 meses. Para obter mais informações sobre versões de agentes suportadas e cronogramas de fim de vida útil, confira nossa [política de fim de vida útil do PHP do agenteNew Relic ](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-eol-policy/).

* A página de [compatibilidade do agente e requisitos do PHP](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/) deve ser consultada para obter as informações mais recentes sobre as versões e plataformas suportadas do PHP.

<Callout variant="important">
  **Para instalação usando uma versão ou plataforma PHP não suportada, é altamente recomendável que você desabilite os mecanismos de atualização automática para os pacotes PHP do agente.** Isso pode ser feito adicionando os pacotes do agente PHP a uma lista de exclusão para atualizações de pacotes. Ou você pode fazer o versionamento do pacote PHP do agente para uma versão do agente que suporte os recursos antigos e sem suporte. A falha em impedir atualizações pode resultar na instalação de uma versão mais recente do agente e na remoção do suporte para o recurso necessário e não suportado. Isso interromperia a coleta de dados do APM.

  Os pacotes do agente PHP afetados são:

  * newrelic-php5
  * newrelic-php5-common
  * daemon-newrelic
</Callout>
