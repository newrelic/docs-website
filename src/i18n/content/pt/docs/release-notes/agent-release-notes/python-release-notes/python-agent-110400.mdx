## Notas

Esta versão do agente Python adiciona suporte a agentes [LangGraph](https://pypi.org/project/langgraph/).

Instale o agente usando `easy_install/pip/distribute` através do [Índice de Pacotes Python](https://pypi.python.org/pypi/newrelic), usando `conda` através do [Índice de Pacotes Conda-Forge](https://anaconda.org/conda-forge/newrelic) ou baixe-o diretamente do [site de download do New Relic](https://download.newrelic.com/python_agent/release).

## Recurso

* Adicionar suporte a agentes [LangGraph](https://pypi.org/project/langgraph/)

  * Adiciona instrumentação para invocações de agente LangGraph via `create_agent()` no LangChain. Isso inclui chamadas do agente para `invoke()`, `ainvoke()`, `stream()` e `astream()`.
