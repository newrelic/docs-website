## Notas

Esta versão do agente Python adiciona informações adicionais de modelo aos eventos de LLM de cadeia do [LangChain](https://pypi.org/project/langchain) e corrige um bug na instrumentação de streaming do [LangGraph](https://pypi.org/project/langgraph).

Instale o agente usando `easy_install/pip/distribute` através do [Índice de Pacotes Python](https://pypi.python.org/pypi/newrelic), usando `conda` através do [Índice de Pacotes Conda-Forge](https://anaconda.org/conda-forge/newrelic) ou baixe-o diretamente do [site de download do New Relic](https://download.newrelic.com/python_agent/release).

## Recursos Melhorados

* Adicionar informações adicionais do modelo aos eventos de LLM da cadeia do LangChain

  * Adicione os atributos `request.model` e `response.model` aos eventos de LLM produzidos por cadeias do [LangChain](https://pypi.org/project/langchain).
  * Essas informações são coletadas com base no melhor esforço e podem não estar sempre disponíveis.

## Bugs

* Corrigir TypeError na instrumentação de streaming do LangGraph

  * Anteriormente, ao usar um subconjunto de APIs de streaming no [LangGraph](https://pypi.org/project/langgraph), o agente encontrava um `TypeError` devido ao uso de `weakref.WeakMethod` pelo LangGraph. Isso foi corrigido, e o agente agora deve funcionar corretamente com esses métodos de streaming do LangGraph.
