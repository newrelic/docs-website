## Notas

Esta versão do agente Python corrige um problema com MetricNormalizationRules, corrige um aviso de depreciação na detecção da versão do pacote e adiciona suporte para novos comandos introduzidos no [Redis v5](https://pypi.org/project/redis/).

Instale o agente usando `easy_install/pip/distribute` por meio do [Índice de Pacotes Python](https://pypi.python.org/pypi/newrelic) ou baixe-o diretamente do [site de download do New Relic](https://download.newrelic.com/python_agent/release).

## Melhorias

* **Adiciona suporte para novos comandos introduzidos no Redis v5** Adiciona suporte completo à instrumentação do [Redis v5](https://pypi.org/project/redis/).

## Correções de bugs

* **Correção de falha em MetricNormalizationRules** Corrigida uma falha que ocorria ao usar MetricNormalizationRules no lado do cliente para ignorar o Métrica (configurado por meio do NerdGraph e disponível sob um sinalizador de recurso). A métrica deve agora ser devidamente ignorada.

* **Corrigido DeprecationWarning na detecção da versão do pacote para Python 3.10+** Corrigido um DeprecationWarning emitido pela detecção da versão do pacote usada internamente para Python 3.10+.Quando o importlib.metadada estiver disponível, prefira usá-lo para evitar a verificação do atributo **de versão** obsoleta nos módulos. (Obrigado @renanivo pela contribuição!)
