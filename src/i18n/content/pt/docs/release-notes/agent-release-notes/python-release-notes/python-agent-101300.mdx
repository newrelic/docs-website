## Notas

Esta versão do agente Python adiciona suporte ao Model Context Protocol (MCP) e corrige a lógica de importação para `importlib.metadata` e `pkg_resources`.

Instale o agente usando `easy_install/pip/distribute` por meio do [Índice de Pacotes Python](https://pypi.python.org/pypi/newrelic) ou baixe-o diretamente do [site de download do New Relic](https://download.newrelic.com/python_agent/release).

## Novo recurso

* Adicionar suporte para o Model Context Protocol (MCP)

  * Adiciona suporte para chamadas MCP em [mcp](https://pypi.org/project/mcp/) e [fastmcp](https://pypi.org/project/fastmcp/). O agente agora instrumentará automaticamente as seguintes primitivas principais do MCP:

    * Ferramentas (via `call_tool`)
    * Recursos (via `read_resource`)
    * Prompt (via `get_prompt`)

## Correções de bugs

* Corrigir lógica de importação para `importlib.metadata` e `pkg_resources`

  * Atualiza a lógica em torno da importação de `importlib.metadata` para também tentar usar o backport `importlib_metadata` antes de retornar para `pkg_resources.*`. Isso também remove o uso de condicionais `sys.version` e corrige problemas em que o Python 3.9 só podia usar `pkg_resources` apesar de ter `importlib.metadata` disponível.
