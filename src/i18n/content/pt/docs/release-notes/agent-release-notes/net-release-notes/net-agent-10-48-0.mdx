### Perceber

* A chave GPG usada para assinar os pacotes RPM do agente .NET em nosso repositório YUM foi rotacionada. Atualize seus sistemas para confiar na nova chave baixando nosso [arquivo de definição de repositório atualizado](https://download.newrelic.com/dot_net_agent/yum/newrelic-dotnet-agent.repo) e colocando-o em `/etc/yum.repos.d/`, substituindo qualquer versão existente desse arquivo. ([ef5f329](https://github.com/newrelic/newrelic-dotnet-agent/commit/ef5f3296ee4580d73437243f5437d86fafcac4ce))

### Atualizando seu agente

* Siga os procedimentos padrão para [atualizar o agente .NET](/docs/agents/net-agent/installation-configuration/update-net-agent).
* Se você estiver usando um agente particularmente antigo, revise a lista de principais alterações e procedimentos para [atualizar o legado .NET agente](/docs/apm/agents/net-agent/installation/update-net-agent/#updating_older_net).

Recomendamos atualizar para a versão mais recente do agente assim que estiver disponível. Se você não puder atualizar para a versão mais recente, atualize seu agente para uma versão com no máximo 90 dias. Saiba mais sobre [como manter seu agente atualizado](/docs/new-relic-solutions/new-relic-one/install-configure/update-new-relic-agent). Consulte o New Relic [documento de política de fim de vida útil](/docs/apm/agents/net-agent/getting-started/net-agent-eol-policy) do agente .NET do para obter informações sobre lançamentos e datas de suporte do agente.
