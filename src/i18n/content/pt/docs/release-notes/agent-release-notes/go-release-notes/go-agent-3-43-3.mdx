## 3.43.3

### Recurso

* EXPERIMENTAL: adicionada uma opção de configuração `ConfigAppLogDecoratingWithinMessage` para permitir que o Local Log Decorator apareça dentro de um campo de mensagem. Isso só pode ser aplicado à integração `logcontext-v2/nrlogrus`. A integração `logcontext-v2/nrlogrus` será atualizada para a v1.1.4.

### Segurança

* Atualizações de segurança do Dependabot

  * `nrawsbedrock` (a integração será atualizada para v1.1.1)
  * `nrawssdk-v2` (a integração será atualizada para v1.2.7)
  * `nrpgx5` (a integração será atualizada para a v1.3.4)\`

### Declaração de apoio

Utilizamos a versão mais recente da linguagem Go. No mínimo, você não deve usar nenhuma versão do Go anterior àquela suportada pela própria equipe do Go. Consulte a [Política de Fim de Vida do Agente Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) para obter detalhes sobre as versões suportadas do agente Go e componentes de terceiros.
