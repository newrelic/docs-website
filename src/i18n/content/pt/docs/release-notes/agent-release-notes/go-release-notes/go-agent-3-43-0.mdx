## 3.43.0

### Adicionado

* Adicionado suporte a UnWrap para o response writer substituto
  * Obrigado ao membro da comunidade @ArturFortunato por contribuir para esta solução.
* Adicionado suporte para requisição e resposta de URL de função Lambda na integração `nrlambda`
  * Obrigado ao membro da comunidade @rittneje por contribuir para esta solução
* Adicionado atributo de Span adicional para o endpoint do Opensearch/elastic search na integração `nrawssdk`
* Adicionado suporte para cláusulas &quot;WITH&quot; no sqlparse

### Fixo

* Corrigido httptest com falha

### Segurança

* Versão do Go no CI/CD atualizada

* Versão de `nrgrpc` atualizada

* Atualizações de segurança do Dependabot

  * `logcontext/nrlogrus`
  * `logcontext/nrlogrusplugin`
  * `logcontext-v2/nrlogrus`
  * `nrfiber`

### Declaração de apoio

Usamos a versão mais recente da linguagem Go. No mínimo, você não deve usar nenhuma versão do Go mais antiga do que aquela suportada pela própria equipe do Go. Consulte a [Política de EOL do agente Go](https://docs.newrelic.com/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) para obter detalhes sobre versões com suporte do agente Go e componentes de terceiros.
