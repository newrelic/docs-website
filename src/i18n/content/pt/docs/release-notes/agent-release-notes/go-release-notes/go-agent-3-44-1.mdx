## 3.44.1

### Fixo

* Corrigida tag incorreta para a versão anterior

### Segurança

* Atualizações de segurança do Dependabot

  * `nrgochi` integração atualizada para v5.2.4
  * `nrmongo` integração atualizada para a v1.17.7
  * `nrmongo-v2` integração atualizada para a v2.4.2
  * `nrnats` integração atualizada para a v2.11.15

### Declaração de apoio

Usamos a versão mais recente da linguagem Go. No mínimo, use uma versão do Go que a equipe do Go ainda suporte. Consulte a [Política de EOL do agente Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) para obter detalhes sobre as versões compatíveis do agente Go e de componentes de terceiros.
