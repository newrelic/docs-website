## 3.44.0

### Adicionado

  \* Adicionado suporte para Anthropic SDK `nranthropic`   \* Adicionado suporte para GocqlX `nrgocqlx`

### Fixo

  \* Exportar tipos de testes internos via `integrationsupport` para desbloquear consumidores de testes externos     \* Obrigado ao membro da comunidade @nitinprajwal por contribuir para esta solução   \* Corrigido um bug no sqlparse para suportar instruções `INTO` opcionais em integrações sql   \* Corrigido um bug no `nrnats` que causava testes instáveis

### Segurança

  \* Atualizações de Segurança do Dependabot     \* `nrfiber`

### Declaração de apoio

Usamos a versão mais recente da linguagem Go. No mínimo, você não deve usar nenhuma versão do Go mais antiga do que aquela suportada pela própria equipe do Go. Consulte a [Política de EOL do agente Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) para obter detalhes sobre versões com suporte do agente Go e componentes de terceiros.
