<Callout variant="important">
  A partir da versão 7.5.4 do iOS SDK, uma alteração foi implementada que faz com que o evento Handled Exception suprimido anteriormente seja relatado.
</Callout>

## Corrigido nesta versão

* Corrigida a falha que poderia ocorrer ao gravar exceções tratadas.
* Corrigimos vários travamentos que podiam ocorrer ao usar o agente iOS do New Relic.
* O agente iOS New Relic agora inclui os atributos Session nos Logs Mobile.
* Corrigido um problema que poderia ocorrer ao enviar várias exceções tratadas para o New Relic. O agente iOS do New Relic agora evitará perda de dados ao enviar múltiplas exceções tratadas.

## Declaração de apoio

* A partir da versão 7.4.1 do agente iOS, o agente iOS consolidará o agente XCFramework e tvOS anteriormente separados em um agente iOS singular.
* A partir deste lançamento, a versão mais antiga suportada do [XCFramework](/docs/release-notes/mobile-release-notes/ios-release-notes/xcframework-agent-736) é a 7.3.6.
