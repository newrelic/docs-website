  A partir da versão 7.5.4 do iOS SDK, uma alteração foi implementada que faz com que o evento Handled Exception suprimido anteriormente seja relatado.

  Atualize para o agente iOS 7.5.8 do New Relic. Identificamos um bug no iOS SDK versão 7.5.7 que poderia causar uma falha no lançamento do agente, e uma correção está incluída no New Relic iOS agente 7.5.8 liberar.
</Callout>

## Corrigido nesta versão

* O agente foi criado usando o Xcode 16.4.
* Corrigido um problema que poderia fazer com que distributed tracing não tivesse todas as informações de conta necessárias no início de uma sessão.
