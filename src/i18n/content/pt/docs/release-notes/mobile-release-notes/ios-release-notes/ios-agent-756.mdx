  A partir da versão 7.5.4 do iOS SDK, uma alteração foi implementada que faz com que o evento Handled Exception suprimido anteriormente seja relatado.
</Callout>

## Corrigido nesta versão

* O agente foi criado usando o Xcode 16.3. Isso deve corrigir quaisquer erros relacionados à Cobertura de Código ao usar o agente iOS do New Relic com o Xcode 16.3.
* Corrigido um problema que poderia fazer com que a conta distributed tracing fosse definida incorretamente.
* Corrigido um problema que poderia fazer o agente travar ao definir o atributo da sessão.
