## Importante

* Esta versão aumenta a versão iOS necessária do agente New Relic iOS para iOS 16. O suporte ao iOS 15 retornará em uma versão futura.

## Novidades desta versão

* Atualização da visualização pública do Session Replay móvel para incluir amostragem para sessões com erros e correção de problemas relacionados ao Session Replay.

## Corrigido nesta versão

* Corrigido um problema que impedia o rastreamento distribuído de funcionar em certos casos extremos.
* Adicionadas correções de estabilidade para problemas identificados.
