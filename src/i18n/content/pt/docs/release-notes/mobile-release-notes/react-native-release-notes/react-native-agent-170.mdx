## Melhorias

* Atualizado o agente Android nativo para a versão 7.7.0.
* Session Replay Inteligente: Introduzido o modo ERROR (gravação em buffer de 15-30s acionada por falhas) e o modo FULL (gravação contínua) para captura de dados otimizada.
* APIs de Controle de Sessão: Adicionado NewRelic.recordReplay() e NewRelic.pauseReplay() para permitir controle manual e programático sobre os estados de gravação do replay.
* Correção de bug de rede: Corrigido um problema no OkHttp3 em que o peekBody() causava bloqueio indefinido em respostas de streaming (SSE/chunked) com tamanhos de conteúdo desconhecidos.
