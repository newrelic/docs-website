## Correções de bugs

* Corrigido um problema em que o rastreamento stack para erros do Unity não era exibido corretamente.
* Corrigido um problema que impedia que o resumo métrico fosse gravado quando o aplicativo retornava ao primeiro plano após seu lançamento inicial.
