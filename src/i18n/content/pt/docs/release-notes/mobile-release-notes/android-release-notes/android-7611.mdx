## Correções

* Corrigido um problema com `NullPointerException` no processo de amostragem de memória.
* Adicionada conversão segura durante a inicialização dos tempos de lançamento do aplicativo para evitar a compatibilidade com `ClassCastException` no SDK MAM do Microsoft Intune.
* Melhor proteção de retorno de chamada okhttp para evitar falhas de agente devido a exceções de código do usuário.
* Corrigido o problema de falhas duplicadas nos relatórios de falhas nativos.

## Declaração de apoio

Recomendamos atualizar o agente pelo menos a cada três meses. Encontre políticas e datas específicas para suporte ao agente Android na [Política de EOL de monitoramento de agente móvel](/docs/mobile-monitoring/new-relic-mobile/get-started/mobile-agents-eol-policy/#android-eol).
