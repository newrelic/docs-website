## Recurso

### Agente autônomo

* **Desacoplado do agente Browser**: O agente de vídeo agora opera de forma independente com sua própria lógica de coleta. Essa separação reduz a sobrecarga e garante que os dados de vídeo sejam coletados e enviados para o New Relic sem estarem vinculados ao cronograma de coleta de dados do agente Browser.
