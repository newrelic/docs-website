## Principais alterações

* **Atualização do relatório de erros**: O atributo `errorName`, anteriormente usado para identificar a descrição do erro, foi descontinuado. O atributo `errorMessage` agora é o padrão para transmitir descrições detalhadas de erro. Por favor, atualize quaisquer sistemas de monitoramento ou log que dependam de informações de erro para utilizar o atributo `errorMessage` a partir de agora.

## Notas de atualização

* Usuários que dependem do atributo `errorName` para análise de erros devem planejar migrar para o atributo `errorMessage` para garantir a continuidade da funcionalidade e o acesso a informações detalhadas sobre erros.
