### Correções

* Otimização de memória: simplificação do tratamento do caminho de resultados de jobs para reduzir a pressão de heap de grandes cargas de monitor de navegador.
* Estabilidade aprimorada: reduz a pressão da coleta de lixo (GC), aumentando a confiabilidade para locais que executam monitores de navegador pesados com arquivos HAR massivos.

Corrigimos as seguintes vulnerabilidades:

* CVE-2025-45582
* CVE-2025-69720
* CVE-2026-41991
* CVE-2026-41992
* CVE-2026-58055
* CVE-2026-9828
