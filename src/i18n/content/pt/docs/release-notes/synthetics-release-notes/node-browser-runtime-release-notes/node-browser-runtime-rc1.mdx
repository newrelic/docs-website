### Melhorias

* O navegador Chrome foi atualizado para a versão 143.0.7499.146. Os usuários podem observar um aviso de descontinuação nos logs do navegador, afirmando &quot;O fallback automático para software WebGL foi descontinuado.&quot; Essa alteração remove a funcionalidade de fallback automático para resolver uma vulnerabilidade de segurança de alta gravidade ([CVE-2025-14765](https://chromereleases.googleblog.com/2025/12/stable-channel-update-for-desktop_16.html)).
