[Google App Engine](https://cloud.google.com/appengine/) (GAE) lo ayuda a crear aplicaciones web y móviles en una plataforma de nube abierta, utilizando los idiomas, el marco, [los tiempos de ejecución personalizados](https://cloud.google.com/appengine/docs/flexible/custom-runtimes/), las imágenes docker y la biblioteca de terceros que elija. El entorno de App Engine escala automáticamente tu aplicación mientras equilibra la CPU y la memoria de tus recursos.

Puede utilizar New Relic para monitor aplicaciones que residen en un entorno flexible de Google App Engine a través del [agente APM](#gae-agent), y puede ver datos sobre sus servicios de Google App Engine a través de la [integración de Google App Engine](#gae-integration) para nuestro monitoreo de infraestructura.

## Instalación del agente APM para GAE Flexible [#gae-agent]

Como [socio de Google Cloud Platform (GCP)](https://newrelic.com/partner/gcp), New Relic admite el [entorno flexible de Google App Engine](https://cloud.google.com/appengine/docs/flexible/) para que pueda desplegar una imagen docker con un tiempo de ejecución personalizado.

Algunos agentes le permiten desplegar una imagen docker estándar utilizando [la instalación flexible GAE con soporte "nativo"](https://cloud.google.com/appengine/docs/flexible/). Con soporte "nativo", usted proporciona el código de su aplicación y un `app.yaml` y Google App Engine lo implementa en una imagen docker prediseñada. Esta configuración tiene algunas limitaciones.

Para monitor su aplicación que reside en un entorno flexible de Google App Engine, siga las instrucciones de instalación para el <InlinePopover type="apm"/>agente que utiliza:

## Integración de infraestructura para Google App Engine [#gae-integration]

Nuestro [monitoreo de infraestructura](/docs/infrastructure) ofrece una [integración](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) que reporta datos sobre sus servicios de Google App Engine. Con la [integración de Google App Engine](/docs/gcp-gcp_appengine-integration), puede ver el panel y las alertas, consultar datos métricos y explorar datos de inventario.
