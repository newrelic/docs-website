Rackspace ofrece un complemento para el producto [Cloud Load Balancer](http://www.rackspace.com/cloud/load-balancing/ "El enlace se abre en una ventana nueva.") . El complemento le permite ver el tráfico HTTP versus HTTPS, configurar fácilmente <InlinePopover type="alerts"/>en su umbral predefinido y verificar periódicamente el estado de los nodos asociados con su balanceador de carga para asegurarse de que estén respondiendo correctamente.

## Requisitos previos [#prereqs]

Asegúrese de que su sistema tenga estos componentes instalados:

* Ruby (versión 1.8.7 o superior)
* Rubygems (versión 1.3.7 o superior)
* Gema del paquete Ruby
* Una [cuentaNew Relic](http://newrelic.com/signup "El enlace se abre en una ventana nueva.")
* Balanceador de carga de Rackspace con registro habilitado

## Instalación del complemento [#install]

Si tiene equilibradores de carga en varias regiones, configure un complemento independiente para cada región. Utilice siempre caracteres en minúscula para el nombre de la región en su archivo yml. Para obtener más información sobre el análisis de YAML, consulte [http://yaml-online-parser.appspot.com/](http://yaml-online-parser.appspot.com/ "El enlace se abre en una ventana nueva.").

```
wget https://github.com/newrelic-platform/newrelic_rackspace_load_balancers_plugin/archive/1.0.0.zip
```

```
bundle install -binstubs
  ./bin/newrelic_rs --sample-config
```

1. Descargue el archivo zip de GitHub:

2. Extraiga el contenido en el directorio que desee utilizar.

3. Ejecute los siguientes comandos desde este directorio para obtener la configuración de la gema:

4. Modifique el

   <DNT>
     **config/newrelic_plugin.yml**
   </DNT>

   con su [clave de licencia](/docs/subscriptions/license-key) de New Relic, nombre de usuario de Rackspace y clave de API, y la región donde se encuentra su balanceador de carga. Asegúrese de utilizar caracteres en minúscula para el nombre de la región.

5. Inicie el agente: `./bin/newrelic_rs`.

6. Verifique que el resultado diga `gathering xx statistics`.

Después de que el complemento se instale correctamente, puede seleccionarlo en New Relic:

1. Inicie sesión en New Relic en

   <DNT>
     **[one.newrelic.com](https://one.newrelic.com/all-capabilities)**
   </DNT>

   .

2. Desde la UI de New Relic, seleccione el icono de Rackspace Cloud Load Balancer.

Desde aquí puede seleccionar balanceadores de carga, ver el tráfico, configurar alertas y más.
