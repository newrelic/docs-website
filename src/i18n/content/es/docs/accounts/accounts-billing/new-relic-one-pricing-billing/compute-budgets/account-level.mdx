Para las organizaciones con múltiples cuentas de New Relic, cada una con distintos requisitos de CCU, los Presupuestos de cómputo proporcionan monitoreo, visibilidad y alertas por cuenta para asegurar que las cuentas se mantengan dentro de sus límites de uso definidos.

## Tipos de presupuesto por estructura de cliente

El tipo de presupuestos que puede crear depende de la estructura de su cliente:

### Para clientes de un solo inquilino

Puede crear **presupuestos a nivel de cuenta** para cuentas individuales o grupos de hasta 5 cuentas dentro de su organización.

### Para clientes de múltiples inquilinos

**Las organizaciones administradoras** pueden crear dos tipos de presupuestos:

* **Presupuestos a nivel de organización**: Presupuestos para organizaciones gestionadas individuales

  * Un presupuesto por organización
  * Una organización por presupuesto
  * Cree varios presupuestos a nivel de organización según sea necesario

* **Presupuestos a nivel de cuenta**: Presupuestos para cuentas dentro de la organización administradora (la misma funcionalidad que los clientes de un solo inquilino)

Las **organizaciones gestionadas** no pueden crear presupuestos.

<Callout variant="important">
  Si forma parte de una organización de gestión, puede crear presupuestos para cualquier organización gestionada bajo la jerarquía de su cliente. Utilice el selector de organización para elegir la organización al crear un presupuesto de organización.
</Callout>

## Configurar cotizaciones a nivel de cuenta

<Steps>
  <Step>
    ### Abrir Presupuestos de cómputo

    1. Ve a **[one.newrelic.com](https://one.newrelic.com/)**.
    2. En el [menú de usuario](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings), seleccione <DNT>**Administration**</DNT> &gt; <DNT>**Compute Budgets**</DNT>.
  </Step>

  <Step>
    ### Crear un presupuesto

    En la sección <DNT>**Functional budgets**</DNT> , seleccione <DNT>**Add a budget**</DNT>.

    <Callout variant="important">
      **Para clientes de múltiples inquilinos**, esta sección muestra:

      * <DNT>**Organization level budgets**</DNT> para organizaciones administradas
      * <DNT>**Account level budgets**</DNT> para la organización de gestión

      Las organizaciones administradas no pueden crear presupuestos a nivel de cuenta.
    </Callout>
  </Step>

  <Step>
    ### Asigna tu cotización

    1. Ingrese un nombre para su presupuesto.

    2. Seleccione las cuentas o la organización para su presupuesto:

       **Para presupuestos a nivel de cuenta de organización única o de gestión**: De la lista desplegable, seleccione y asigne hasta 5 cuentas a este presupuesto.

       **Para presupuestos a nivel de organización de múltiples inquilinos**: Seleccione una organización del menú desplegable. Cada presupuesto a nivel de organización puede contener solo una organización.

       <Callout variant="tip">
         Al crear presupuestos a nivel de organización, utilice el selector de organización para ver y seleccionar la organización gestionada para su presupuesto.
       </Callout>

    3. Seleccione <DNT>**Next**</DNT>.
  </Step>

  <Step>
    ### Establecer límites de cotización mensual

    1. Para establecer los umbrales de gasto para su cuenta seleccionada, en la sección <DNT>**Set monthly budget limits**</DNT>, asigne los límites para los SKU de cómputo que ha comprado para establecer el presupuesto de su cuenta.
    2. Seleccione <DNT>**Next**</DNT>.

    <img title="Screenshot displaying the Set monthly budget limits page with advanced and core CCU limits highlighted" alt="Screenshot displaying the Set monthly budget limits page with advanced and core CCU limits highlighted" src="/images/set-monthly-account-budget.webp" />
  </Step>

  <Step>
    ### Configurar notificación

    1. Para evitar excesos y gestionar los costos de manera efectiva, en la página <DNT>**Set up notifications**</DNT>, establezca el porcentaje de umbral y elija el destino para las notificaciones.

    2. Para recibir una advertencia cuando haya utilizado un cierto porcentaje de la CCU asignada a un presupuesto para un mes específico, en la pestaña <DNT>**Thresholds**</DNT>, seleccione <DNT>**+ Add a threshold**</DNT> e ingrese el porcentaje en el que desea recibir notificaciones.

    3. Para asegurar que se envíen notificaciones cuando el presupuesto exceda el umbral, en la pestaña <DNT>**Destinations**</DNT>, agregue al menos una persona para que sea notificada. Puede elegir entre las siguientes opciones:

       * <DNT>**Email**</DNT>: Ingrese las direcciones de correo electrónico de los usuarios de New Relic para notificarles sobre el presupuesto.
       * <DNT>**Webhook**</DNT>: Ingrese la URL del webhook para enviar notificaciones a un punto final específico.

    4. Seleccione <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    ### Ver desglose de uso

    Para ver el <DNT>**Usage breakdown**</DNT>, seleccione el menú de tres puntos. Puede acceder a la siguiente información según esté disponible:

    * Un desglose detallado día a día de su gráfico de uso de CCU, adaptado a los SKU de cómputo que tiene.
    * Uso de CCU categorizado por <DNT>**Users**</DNT>, <DNT>**Accounts**</DNT> y <DNT>**Capabilities**</DNT>.
    * Detalles sobre la CCU, incluyendo el <DNT>**Budget name**</DNT>, las cuentas asociadas e información como la configuración de <DNT>**Budget time range**</DNT>, <DNT>**Product**</DNT> y <DNT>**Notification**</DNT>.
  </Step>
</Steps>
