Hay varias formas en que un usuario de New Relic puede iniciar un sesión en una organización de New Relic:

* Correo electrónico y contraseña: el método de inicio de sesión predeterminado, que requiere la dirección de correo electrónico y una contraseña del usuario.
* Inicio de sesión social: puede utilizar Google, Google Workplace, GitHub y Bitbucket para registrarse e iniciar sesión en New Relic.
* Inicio de sesión único: para conocer las opciones de inicio de sesión del proveedor de identidad, consulte [Introducción a SAML SSO y SCIM](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim).

¿Tiene problemas para iniciar sesión? Ver [Resolución de problemas de inicio de sesión](/docs/accounts/accounts-billing/account-setup/troubleshoot-new-relics-password-email-address-login-problems/).

Para administrar cómo iniciar sesión su usuario, consulte [Dominio de autenticación](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more).
