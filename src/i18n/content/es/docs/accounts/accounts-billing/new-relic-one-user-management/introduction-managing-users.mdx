Los documentos de esta sección tratan sobre la administración de usuarios en [nuestro modelo de usuario más nuevo](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models), en el que están casi todos los usuarios de New Relic. Para obtener documentos sobre nuestro modelo de usuario original, consulte [Gestión de usuarios originales](/docs/accounts/original-accounts-billing/original-users-roles).

Estos son algunos de nuestros documentos de gestión de usuarios más importantes:

* [Cómo funciona el acceso](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts): aprenda cómo los usuarios obtienen acceso a los permisos de usuario y a las cuentas.
* [tipo de usuario](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type): obtenga información sobre el usuario básico, el usuario principal y el usuario de plataforma completa.
* [UI y tareas comunes](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/): aprenda sobre la UI de administración de usuarios, cómo agregar usuarios, editar el tipo de usuario, agregar usuarios a grupos, crear roles personalizados y otras tareas comunes.
* [Introducción al inicio de sesión único y al aprovisionamiento SCIM](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim): una descripción general de SAML SSO y cómo administrar su usuario desde un proveedor de identidad.
* [Configuración del dominio de autenticación](/docs/accounts/accounts-billing/new-relic-one-pricing-users/configure-authentication-domains): un "dominio de autenticación" es lo que gobierna cómo se agregan los usuarios de ese dominio a New Relic (manualmente versus aprovisionamiento SCIM), el método de autenticación utilizado por esos usuarios (inicio de sesión manual versus SAML SSO), administrando cómo el usuario actualiza su usuario. tipo y configuración de sesión de usuario.
* [Tutorial de gestión de usuarios](/docs/accounts/accounts-billing/new-relic-one-user-management/tutorial-add-new-user-groups-roles-new-relic-one-user-model): lecciones detalladas sobre algunas tareas comunes de gestión de usuarios.
* [Vídeos sobre gestión de usuarios](/docs/accounts/accounts-billing/new-relic-one-user-management/user-mgmt-videos): vea vídeos de la UI de usuario de gestión de usuarios en acción.

¿Quiere comprender cómo el recuento de usuarios afecta la facturación? Consulte [Facturación relacionada con el usuario](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-one-pricing-billing#user-count).
