Agregar una cuenta a su organización New Relic es una forma de organizar y separar datos para equipos o proyectos específicos.

## Requisitos [#requirements]

Requisitos para agregar una cuenta a su organización New Relic:

* [Edición](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing#editions)Pro o Enterprise

* El usuario debe ser un usuario principal o un usuario de plataforma completa con

  <DNT>
    [**Organization settings**](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#admin-settings)
  </DNT>

  .

* La organización con [cuentas asociadas](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partnership-accounts-users-subscriptions) debe utilizar la [API de asociación](/docs/new-relic-partnerships/partner-integration-guide/partner-account-maintenance/partner-api).

Algunas reglas de nomenclatura de cuentas:

* Las cuentas se identifican por su [ID de cuenta](/docs/accounts/accounts-billing/account-structure/account-id). No existen reglas sobre la unicidad del nombre de cuenta dentro de una organización.
* Un nombre de cuenta puede tener un máximo de 255 caracteres.

## ¿Por qué agregar una cuenta? [#why-add-accounts]

Para conocer los motivos para agregar cuentas, consulte [organización y estructura de cuentas](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure).

## Agregar cuentas [#add-accounts]

Para agregar una cuenta desde la UI: desde la [página UI <DNT>**Access management**</DNT> ](https://one.newrelic.com/admin-portal/organizations/organization-detail), haga clic en <DNT>**Accounts**</DNT> y luego haga clic en <DNT>**Create account**</DNT>. (Si estás en nuestro modelo de usuario original, consulta [Agregar cuentas](/docs/accounts/original-accounts-billing/original-users-roles/parent-child-account-structure/#creating)).

Para agregar una cuenta a través de API: consulte [Administrar cuentas con NerdGraph](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph).

Para cancelar una cuenta, [presente un ticket a nuestro equipo de soporte técnico global](/docs/new-relic-solutions/solve-common-issues/find-help-use-support-portal/#file-ticket).
