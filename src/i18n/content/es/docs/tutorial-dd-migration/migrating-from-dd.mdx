Puede decidir cambiar de producto de monitoreo de datos por diversas razones. Cualquiera sea el motivo, puede probar New Relic usted mismo sin necesidad de migrar completamente su stack desde Datadog. Hemos proporcionado una serie de tutoriales para ayudarle a comenzar a monitorear su sistema con New Relic, así como a comprender cómo funciona la observabilidad de New Relic. A continuación, encontrará información sobre cómo configurar el monitoreo de infraestructura, aplicación y registro, así como el panel y <InlinePopover type="alerts"/>para ayudarlo a controlar realmente sus datos.

El único requisito previo es una cuenta New Relic: [¡regístrate gratis](https://newrelic.com/signup) y estarás listo para comenzar!

## Use entidad para observabilidad de todo el stack [#entities]

En New Relic, utilizamos el concepto de entidad para ayudarnos a dividir y organizar datos. Esencialmente, cualquier cosa que informe datos o contenga datos para New Relic que pueda identificarse con una identificación única es una entidad. Usamos un poderoso sistema de mapeo para vincular completamente a las entidades. Para usted, eso significa un monitoreo más granular y transparente de casi cualquier tipo de datos de casi cualquier fuente. Todo esto para permitirle ver exactamente los datos que le interesan en cualquier nivel, sin saturar su monitoreo con cosas que no quiere o no necesita.

## Explorando el panel prediseñado [#dash]

¿No sería fantástico si pudieras ver todos tus datos de observabilidad sin tener que configurar nada? New Relic ofrece interfaces seleccionadas para brindarte eso y más. Desde el momento en que comience a ingerir datos, tendrá acceso a niveles muy detallados de información desde el principio, así como a una serie de características fáciles de usar que le permitirán ordenar, encontrar, contextualizar e investigar esos datos con solo un haga clic o dos.

Esta serie de tutoriales cubrirá estas interfaces seleccionadas en forma de panel prediseñado para infraestructura, aplicaciones y monitoreo log . También le mostraremos dónde ir para encontrar cientos de paneles preconfigurados para nuestras otras características. Dedique menos tiempo a configurar sus herramientas y más tiempo a utilizarlas con New Relic.

## Probando New Relic [#migrate]

Si desea probar nuestra plataforma usted mismo, puede seguir los procedimientos a continuación para comenzar a ingerir datos y configurar la alerta y el panel. Si solo desea obtener más información sobre nuestra característica y comenzar de inmediato, consulte el documento final de la serie sobre cómo cambiar a New Relic.
