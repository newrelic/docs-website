Si es un cliente **Advanced Compute**, puede emplear **NRQL para consultar directamente datos de entidades y relaciones en NRDB** junto con sus telemetry data. Este acceso unificado permite una resolución más rápida de problemas y una información valiosa más profunda del sistema.

Este documento cubre:

* Consulta de datos de entidades y relaciones con NRQL
* Correlación de entidades con telemetría para un análisis exhaustivo
* Ejemplos prácticos como el seguimiento de cambios de estado y la búsqueda de relaciones entre la aplicación y el host.

## Beneficios clave

¿Busca simplificar la forma en que entiende sus sistemas y solucionar problemas más rápidamente? Centralizamos sus datos al incorporar entidades y sus relaciones directamente en NRDB. Esto significa que usted puede emplear el poder de NRQL para obtener una imagen completa de su entorno y lugar.

Así es como usted se beneficia:

* **Acceda a todos los datos de su entidad directamente en NRDB:** Descubrirá que los datos de su entidad y relación se incorporan y almacenan en NRDB. Esto crea un almacenamiento de datos unificado, convirtiéndolo en el lugar central para esta información crítica junto con sus telemetry data.
* **Consulta con facilidad usando NRQL:** Puedes usar el poderoso y flexible NRQL con el que ya estás familiarizado para consultar estos datos de entidades y relaciones. Esto simplifica su flujo de trabajo, ya que no necesita aprender nuevos lenguajes de consulta ni realizar uniones complejas en diferentes almacenes de datos. Su consulta será más sencilla y fácil de gestionar.
* **Obtenga información valiosa y oportuna con disponibilidad de datos las 24 horas:** ¿Se pregunta si los datos estarán allí cuando los necesite? Estos datos de entidad y relación se conservan en NRDB durante un periodo de 24 horas. Esta disponibilidad le permite realizar análisis y correlaciones oportunos, para que pueda recuperar rápidamente la información necesaria para comprender el comportamiento del sistema e identificar la dependencia.
* **Obtenga una visión holística con correlación simplificada:** puede vincular sin esfuerzo entidades, sus relaciones y sus telemetry data asociados (como métricas, eventos, logs y trazas). Esta capacidad de correlación mejorada significa que usted obtiene una imagen completa de la salud y el rendimiento de su entorno, lo que le permite solucionar problemas de manera más eficiente y obtener información valiosa más rápidamente.
