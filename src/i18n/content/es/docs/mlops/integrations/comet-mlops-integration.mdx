[Comet](https://www.comet.ml/site/) es una plataforma [de monitoreo del rendimiento de los modelos](/docs/alerts-applied-intelligence/mlops/get-started/intro-mlops/) que extiende la observabilidad de todo el stack a modelos de aprendizaje automático y permite establecer una línea de base de rendimiento de producción basada en el rendimiento del modelo. La integración de Comet le permite integrar el aprendizaje automático métrico con New Relic, para que pueda monitor continuamente sus datos durante todo el ciclo de vida del aprendizaje automático. Esto ayuda a garantizar un rendimiento óptimo del modelo y le ayuda a lograr mejores resultados comerciales.

## Integrar Comet con New Relic [#integrate-comet]

Utilice la integración de Comet de New Relic para obtener datos e Insights que le ayudarán a crear modelos de aprendizaje automático mejores y más precisos, al mismo tiempo que mejoran la productividad, la colaboración y la visibilidad en todo su equipo.

Para configurar la integración, siga estos pasos:

1. <DNT>**Log into your New Relic account**</DNT>: Vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > +Add more data**</DNT>.

2. <DNT>**Click on Comet**</DNT>: Escriba <DNT>**Comet**</DNT> en la barra de búsqueda.

3. <DNT>**Select the account ID**</DNT> quieres que Comet se integre.

4. <DNT>**Select or create an API key**</DNT>: Para integrarse con Comet, necesitas una New Relic<InlinePopover type="licenseKey"/>. Cree o seleccione una clave de API existente en la sección <DNT>**Real time training metric**</DNT> .

5. <DNT>**Contact Comet**</DNT>: Para el siguiente paso para configurar la integración y ver el dashboard de monitoreo de producción de modelos en Comet, deberá [comunicarse con Comet](mailto:sales@comet.ml). El equipo de Comet lo guiará en cada paso del proceso de integración y lo configurará todo.

   <img
     title="Model Production Monitoring dashboard"
     alt="Model Production Monitoring dashboard."
     src="/images/model-performance-monitoring_screenshot-full_comet-UI.webp"
   />

6. <DNT>**View your data**</DNT>: ahora que ha configurado su integración, puede comenzar a realizar un seguimiento del rendimiento de sus modelos de aprendizaje automático en un [dashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) dedicado, que genera automáticamente el equipo de Comet.
