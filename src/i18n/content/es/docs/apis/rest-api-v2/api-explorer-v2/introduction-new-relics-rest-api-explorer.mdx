New Relic ofrece [varias API](/docs/apis/getting-started/introduction-new-relic-apis), incluida la [APIREST de New Relic](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2). Este documento le presenta el [Explorador de API REST](https://api.newrelic.com), que permite al usuario administrador y a aquellos con la [clave de API](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key) :

* Explore los extremos REST de API disponibles.
* Interactuar con la API REST dentro de una interfaz de usuario (el Explorador de API).
* Vea una fuente de documentación en vivo.
* Obtenga comando curl para acciones API.
* Comparta la API de llamada configurada con colegas copiando y pegando las URL del Explorador de API.

Esto le ayuda a buscar rápidamente soluciones y probar su API de llamada antes de agregarlas a sus propios componentes de software.

La API de llamada REST para la infraestructura <InlinePopover type="alerts"/>no está disponible en el Explorador de API.

## Característica

El API Explorer de New Relic incluye una interfaz de usuario interactiva para su cuenta seleccionada. La UI de API Explorer enumera los tipos de API de llamada (<DNT>**Applications**</DNT>, <DNT>**Users**</DNT>, etc.) y sus funciones disponibles, como <DNT>**GET metric data**</DNT>, <DNT>**PUT (update) applications**</DNT>, <DNT>**DELETE applications**</DNT>, etc.

A medida que escribe valores para <DNT>**Parameters**</DNT>, aparecen automáticamente en <DNT>**Request**</DNT> para que pueda probar y verificar su sintaxis antes de enviar la solicitud. La UI indica los campos obligatorios, las descripciones de los campos, su tipo (entero, flotante, booleano, etc.) y su ubicación (ruta, consulta, etc.).

Para obtener información sobre los requisitos de la clave de API, consulte [REST clave de API](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key).

## Diferencias con la versión 1 de API [#v1]

Este Explorador de API se aplica solo a la [versión 2 de la API REST de New Relic](/docs/apm/apis/requirements/new-relic-rest-api-v2-getting-started), que se centra en los datos que entran y salen de New Relic. La versión 2 reemplaza la versión 1 de la API REST obsoleta de New Relic.

Tenga en cuenta que existen algunas diferencias entre la versión 2 y 1:

* Los nombres de los datos pueden ser diferentes.
* Algunos comandos curl para v2 son diferentes a v1.

<Callout variant="tip">
  El agente de New Relic utiliza diferentes API y no se puede acceder a ellos a través del Explorador API .
</Callout>
