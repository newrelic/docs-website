El Explorador de API REST (v2) de New Relic facilita probar y enviar solicitudes para cualquier extremo de API disponible. Después de seleccionar su elección de funciones para el tipo de llamada API (aplicación, navegador, etc.), la interfaz de usuario proporciona un formulario interactivo para ver los requisitos y probar los valores de sus parámetros. También proporciona una fuente en vivo de documentación sobre los valores de API.

## Clave de requisitos API [#api-key]

Antes de poder utilizar [API Explorer](https://rpm.newrelic.com/api/explore), se debe activar el acceso a API y se debe generar una [clave de API](/docs/apis/rest-api-v2/requirements/rest-api-key) para su cuenta.

<Callout variant="important">
  Recomendamos utilizar <InlinePopover type="userKey"/>, no la [clave de API REST](/docs/apis/intro-apis/new-relic-api-keys/#rest-api-key), porque la clave de usuario tiene menos restricciones.
</Callout>

Consejos:

* Si ha iniciado sesión en New Relic, cuando utilice el Explorador de API, puede elegir su [clave de API](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key) en la parte superior de la UI y esa clave aparecerá automáticamente en las secciones

  <DNT>
    **Request**
  </DNT>

  y

  <DNT>
    **Parameters**
  </DNT>

  del Explorador.

* Si no ha iniciado sesión en New Relic, puede pegar su [clave de API](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key) en el

  <DNT>
    **Parameters**
  </DNT>

  del Explorador de API.

## Acceda al Explorador de API [#accessing]

Para utilizar el Explorador de API de New Relic:

1. Vaya a

   <DNT>
     **[rpm.newrelic.com/api/explore](https://rpm.newrelic.com/api/explore)**
   </DNT>

   .

2. En la barra de menú del Explorador de API, seleccione el nombre de la cuenta para su aplicación en la lista desplegable.

3. En la barra lateral, seleccione una característica (aplicación, navegador, etc.) y una [función](/docs/apm/apis/api-explorer-v2/parts-api-explorer#functions) API disponible:

   <DNT>
     **GET**
   </DNT>

   ,

   <DNT>
     **PUT**
   </DNT>

   ,

   <DNT>
     **DELETE**
   </DNT>

   .

4. Escriba los otros valores

   <DNT>
     **Parameters**
   </DNT>

   para su llamada API. (Consulte la UI para obtener descripciones y requisitos para la versión 2).

5. Seleccione el

   <DNT>
     **format**
   </DNT>

   para su solicitud: JSON o XML.

6. Seleccione

   <DNT>
     **Send Request**
   </DNT>

   .
