El Explorador de API REST (v2) de New Relic facilita probar y enviar solicitudes para cualquier extremo de API disponible. Después de seleccionar su elección de funciones para el tipo de llamada API (aplicación, navegador, etc.), la interfaz de usuario proporciona un formulario interactivo para ver los requisitos y probar los valores de sus parámetros. También proporciona una fuente en vivo de documentación sobre los valores de API.

## Clave de requisitos API [#api-key]

Antes de poder usar [API Explorer](https://api.newrelic.com/docs), se debe activar el acceso a la API y generar una [clave de API de usuario](/docs/apis/rest-api-v2/requirements/rest-api-key) para su cuenta.

Consejos:

* Cuando use el API Explorer, puede insertar su [clave de API de usuario](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2#api_key) en la parte superior de la UI y esa clave aparecerá automáticamente en las secciones <DNT>**Request**</DNT> y <DNT>**Parameters**</DNT> del Explorer.

## Utilice el Explorador de API [#use-api-explorer]

1. Vaya a <DNT>**[API Explorer](https://api.newrelic.com/docs/)**</DNT>.
2. Desde el menú desplegable <DNT>**Servers**</DNT>, elija una URL de API basada en EE. UU. o la UE.
3. Haz clic en <DNT>**Authorize**</DNT>, ingresa tu clave de API de usuario y haz clic en <DNT>**Authorize**</DNT> de nuevo.
4. Expanda una de las funciones de API disponibles: <DNT>**GET**</DNT>, <DNT>**PUT**</DNT>, <DNT>**DELETE**</DNT>.
5. (Opcional) Agregue valores <DNT>**Parameters**</DNT> a su llamada API para filtrar la respuesta (consulte la UI para obtener descripciones y requisitos para la v2).
6. Desde el desplegable <DNT>**Media type**</DNT>, elija un formato para su solicitud: JSON o XML.
7. Haga clic en <DNT>**Try it out**</DNT>, luego <DNT>**Execute**</DNT>.
