## Listar etiquetas existentes [#list_labels]

Para enumerar las etiquetas que existen en su cuenta, use este comando:

```
curl 'https://api.newrelic.com/v2/labels.json' \
     -H "Api-Key:NEW_RELIC_USER_KEY" -i
```

### Ver el resultado [#list_label_output]

La salida de la etiqueta incluye:

* El nombre de clave completo de la etiqueta (por ejemplo, `Staging:Test1`), así como el componente y el nombre de la clave.
* La aplicación `health_status` (rojo, naranja, verde, gris) y el ID de la aplicación para la etiqueta en cada categoría
* Aplicaciones que tienen esta etiqueta; por ejemplo, la etiqueta `Staging:Test1` se ha aplicado a una aplicación (4937390) y dos hosts (3801143 y 5879721).
