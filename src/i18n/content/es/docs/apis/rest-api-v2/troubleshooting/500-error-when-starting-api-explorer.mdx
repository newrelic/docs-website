## Problema

Aparece un error `500` al iniciar API Explorer.

## Solución

Para corregir esto, asegúrese de tener una cuenta predeterminada:

1. Vaya a

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > ([user menu](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings)) > User preferences**
   </DNT>

   .

2. En el menú desplegable

   <DNT>
     **Default account**
   </DNT>

   , seleccione su elección y luego guárdela.

## Causa

Esto ocurre cuando no tiene una cuenta New Relic predeterminada actual o válida asociada con su inicio de sesión. Esto evita que el Explorador muestre la clave de API disponible para usted.
