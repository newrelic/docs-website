Cuando utilice la [API REST de New Relic](/docs/apis/rest-api-v2/getting-started/introduction-new-relic-rest-api-v2), debe reemplazar algún marcador de posición de ID (como `$APP_ID`, `$KEYTX_ID`, `$COMPONENT_ID`, etc.) con el ID de una aplicación, transacción u otro componente.

Puede utilizar la UI de New Relic para obtener algunas de estas identificaciones. También puede obtener una lista de ID utilizando [REST API Explorer](/docs/apis/rest-api-v2/get-started/list-application-id-host-id-instance-id).

## Encuentre identificaciones usando la UI [#ui]

La UI incluye los ID de la aplicación y algunos otros componentes en un panel de metadatos. Para ver los metadatos de una entidad, haga clic en una entidad monitor de la lista del explorador de entidades y haga clic en el icono de etiqueta.

Aquí hay unos ejemplos:

<CollapserGroup>
  <Collapser
    id="apm"
    title="APM: ID de aplicación"
  >
    Para localizar el `$APP_ID` desde la UI de New Relic:

    1. Vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DNT> y seleccione una aplicación.

    2. Haga clic en el <Icon name="fe-tag"/> icono junto al nombre de la aplicación para abrir el panel de metadatos.

    3. Localice el <DNT>**App ID**</DNT>.

       Sustituya este número por el `$APP_ID` en su API de llamada REST.
  </Collapser>

  <Collapser
    id="browser"
    title="ID de la aplicación browser"
  >
    Para localizar el `$APP_ID` desde la UI de New Relic:

    1. Vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**</DNT> y seleccione una aplicación de monitor.

    2. Haga clic en el <Icon name="fe-tag"/> icono junto al nombre de la aplicación para abrir el panel de metadatos.

    3. Localice el <DNT>**App ID**</DNT>.
