Este enfoque agrega un nuevo método de autenticación a [la API REST (v2) de New Relic](https://api.newrelic.com/) orientada a los clientes que permite a los socios (y clientes a escala empresarial que utilizan la estructura de asociación para administrar sus múltiples cuentas) usar sus credenciales de asociación para autorizar todas las API REST ( v2) llamadas. No ofrece ninguna funcionalidad nueva para usuarios de API que no son socios.

Al llamar a extremos en la API REST de New Relic (v2) que requieren <InlinePopover type="userKey"/>, New Relic sugiere que utilice el [método de autenticación](/docs/apis/rest-api-v2/requirements/admins-api-key-partnerships#Partners) descrito.

## Encabezados personalizados y API de llamada [#Partners]

Al realizar una llamada API, utilice los siguientes encabezados personalizados con su [clave de API de socio](/docs/accounts-partnerships/partnerships/partner-api/partner-api-reference#the-api-key), [ID de socio](/docs/accounts-partnerships/partnerships/partner-api/partner-api-reference#partner-id) y su [ID de cuenta](/docs/accounts-partnerships/accounts/account-setup/account-id):

* `X-Api-Key:YOUR_PARTNER_ACCOUNT_API_KEY`
* `NewRelic-Partner-Id:YOUR_PARTNER_ID`
* `NewRelic-Account-Id:YOUR_CUSTOMERS_ACCOUNT_ID`

Proporcionar autenticación a la API REST (v2) en este formulario le permite completar cualquier operación de API en las cuentas de sus clientes. Esto evita el requisito de cara al cliente de que algunas llamadas API requieren la clave de API de un usuario administrador.

Este método de autenticación también funciona para extremos de API que solo requieren la clave API REST. Este nuevo método de autenticación funciona para todos los extremos, no solo aquellos que requieren la clave de API del usuario administrador.

## Limitaciones conocidas

Este método de autenticación exclusivo para socios solo funcionará con la [API REST de New Relic (v2)](https://api.newrelic.com/). <DNT>**does not**</DNT> funciona con lo siguiente:

* [API de implementación](/docs/apm/new-relic-apm/maintenance/deployment-notifications#api)
* [API de infraestructura para alerta](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/rest-api-calls-new-relic-infrastructure-alerts)
* [API de socio](/docs/accounts-partnerships/partnerships/partner-api/partner-api-reference)
* [API de monitoreo sintético](/docs/apis/synthetics-rest-api/monitor-examples/manage-synthetics-monitors-rest-api)
