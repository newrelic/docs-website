Puede enumerar y revocar las URL de gráficos en vivo de acceso público mediante consultas y mutaciones en [NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph).

Para hacer esto:

1. Vaya al [explorador GraphiQL](https://api.newrelic.com/graphiql).
2. [Enumere todas las URL de gráficos en vivo](#list) creadas dentro de su cuenta New Relic.
3. [Revoca cualquier URL de gráfico en vivo que desees](#revoke).

<Callout variant="tip">
  Para crear URL de gráficos en tiempo real de acceso público, consulte nuestra [documentación](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#chart).
</Callout>

## Enumere todas las URL de gráficos en vivo [#list]

Utilice la siguiente consulta para recuperar una lista de URL de gráficos en vivo existentes:

```graphql
{
  actor {
    dashboard {
      liveUrls {
        liveUrls {
          title
          url
          createdAt
          type
        }
        errors {
          description
        }
      }
    }
  }
}
```

## Revocar cualquier URL de gráfico en vivo [#revoke]

Utilice la siguiente consulta para revocar la URL del gráfico en vivo que especifique:

```graphql
mutation {
  dashboardWidgetRevokeLiveUrl(uuid: "YOUR_LIVE_CHART_UUID") {
    uuid
    errors {
      description
      type
    }
  }
}
```
