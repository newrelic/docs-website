Para permitir el intercambio de datos entre cuentas dentro de una organización, una cuenta debe optar por la función de &quot;alerta entre cuentas&quot;. Siga estas instrucciones sobre cómo emplear el Explorador de API de NerdGraph para actualizar la configuración de la cuenta habilitando las alertas entre cuentas.

<Callout variant="important">
  Sólo el usuario con la licencia &quot;Control de acceso para compartir datos&quot; puede actualizar las elecciones entre cuentas para compartir datos. De forma predeterminada, esta licencia se otorga sólo a los usuarios con el rol de &quot;Administrador de la organización&quot;.
</Callout>

## Pasos para actualizar una elección de alerta entre cuentas

<Steps>
  <Step>
    Recopila la lista de ID de cuentas para actualizar. La API acepta hasta **20 ID de cuenta**.
  </Step>

  <Step>
    Indique si desea habilitar `true` o deshabilitar `false` la elección.
  </Step>

  <Step>
    Emplee el explorador de API de NerdGraph para actualizar las elecciones de la cuenta.
  </Step>
</Steps>

He aquí un ejemplo de la mutación:

```graphql
mutation {
  alertsUpdateCrossAccountElections(
    accountIds: [YOUR_1ST_ACCOUNT_ID, YOUR_2ND_ACCOUNT_ID, YOUR_3RD_ACCOUNT_ID],
    electionStatus: true
  ) {
    electionStatus
  }
}
```
