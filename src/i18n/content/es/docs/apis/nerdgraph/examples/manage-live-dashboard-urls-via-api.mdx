Con la [seguridad y el acceso necesarios configurados,](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts) puede crear, gestionar y revocar URL en tiempo real de acceso público dashboard usando consulta y mutaciones en [NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph). También puede hacerlo desde la [New Relic UI](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#dashboard).

<Callout variant="caution">Si un dashboard compartido no está protegido con contraseña, cualquier persona con la URL activa puede acceder y ver toda la información que contiene. Para evitar acceso no deseado, cerciorar de [crear URL protegidas con contraseña](#create-dashboard-url-with-psswd-nerdgraph) o [habilitar la protección con contraseña para las URL compartidas existentes](#enable-password-dashboard-url-nerdgraph).</Callout>

Para crear, actualizar o revocar una URL dashboard en tiempo real de acceso público desde NerdGraph, necesita el GUID del dashboard que desea compartir.

1. Vaya a <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com//dashboards)**</DNT>.
2. Abra el dashboard que desea compartir.
3. Navegue hasta <DNT>**<Icon name="fe-more-horizontal" /> &gt; Metadata**</DNT> y copie el GUID de la entidad del dashboard.

<CollapserGroup>
  <Collapser id="create-dashboard-url-nerdgraph" title="Crear una URL sin protección de contraseña">
    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya un [gráfico llamado API](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) usando la mutación `dashboardCreateLiveUrl` con los siguientes parámetros:

    * `GUID`: Ingrese el GUID de la entidad del dashboard.

    * `Options`(opcional) &gt; `ttl`: Ingrese el periodo de tiempo en segundos que la URL permanecerá activa.

      * **Valor mínimo**: 300 segundos (5 minutos)
      * **Valor máximo**: 631152000 segundos (20 años)
      * **Valor predeterminado**: 2592000 segundos (30 días)

    * `url`: Seleccione esta opción para obtener la URL dashboard en tiempo real en la respuesta.

    * Seleccione cualquier otro campo que desee en la respuesta.

    #### Mutación de muestra [#create-public-link]

    ```graphql
    mutation {
      dashboardCreateLiveUrl(
        options: { ttl: TIME_IN_SECONDS }
        guid: "YOUR_DASHBOARD_GUID"
      ) {
        url
      }
    }
    ```

    La URL dashboard en tiempo real aparece en la respuesta. Esta es una URL pública, por lo que puedes compartirla con cualquiera.

    #### Ejemplo de respuesta [#create-public-link-response]

    ```json
    {
      "data": {
        "dashboardCreateLiveUrl": {
          "url": "https://share.newrelic.com/dashboard/PUBLIC_LINK_ID"
        }
      }
    }
    ```

    El `PUBLIC_LINK_ID` es el ID del dashboard compartido. Puede usarlo para actualizar o revocar la URL dashboard compartido a través de NerdGraph.

    <Callout variant="important">
      Cualquier actualización de contenido en el dashboard de la plataforma New Relic se refleja en la página dashboard compartido.
    </Callout>
  </Collapser>

  <Collapser id="create-dashboard-url-with-psswd-nerdgraph" title="Crear una URL con protección de contraseña">
    Para evitar acceso no deseado al dashboard compartido, puede crear una URL dashboard en tiempo real con protección con contraseña.

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya un [gráfico llamado API](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) usando la mutación `dashboardCreateLiveUrl` con los siguientes parámetros:

    * `auth` &gt; `factors` &gt; `type`: Seleccione `PASSWORD` para habilitar la protección con contraseña para la URL dashboard en tiempo real.

    * `GUID`: Ingrese el GUID de la entidad del dashboard.

    * `Options`(opcional) &gt; `ttl`: Ingrese el periodo de tiempo en segundos que la URL permanecerá activa.

      * **Valor mínimo**: 300 segundos (5 minutos)
      * **Valor máximo**: 631152000 segundos (20 años)
      * **Valor predeterminado**: 2592000 segundos (30 días)

    * `auth` &gt; `factors` &gt; `password` &gt; `value`: Seleccione esto para obtener la contraseña dashboard en tiempo real en la respuesta.

    * `url`: Seleccione esta opción para obtener la URL dashboard en tiempo real en la respuesta.

    * Seleccione cualquier otro campo que desee en la respuesta.

    #### Mutación de muestra [#create-public-link-with-password]

    ```graphql
    mutation {
      dashboardCreateLiveUrl(
        guid: "YOUR_DASHBOARD_GUID"
        options: { ttl: 3600 } # Optional: Set a time-to-live for the URL
        auth: { factors: [{ type: PASSWORD }] }
      ) {
        url
        uuid
        auth {
          factors {
            type
            password {
              value # REQUEST THE PASSWORD VALUE HERE
            }
          }
        }
      }
    }
    ```

    La URL y la contraseña dashboard en tiempo real aparecen en la respuesta.

    #### Ejemplo de respuesta [#create-public-link-with-password-response]

    ```json
    {
      "data": {
        "dashboardCreateLiveUrl": {
          "auth": {
            "factors": [
              {
                "password": {
                  "value": "pfd-3ZZ46#3a4On5p%&¿"
                },
                "type": "PASSWORD"
              }
            ]
          },
          "url": "https://one.newrelic.com/dashboard/f6c730bf-fced-4890-ad6e-86ab6a6e8159",
          "uuid": "f6c730bf-fced-4890-ad6e-86ab6a6e8159"
        }
      }
    }
    ```

    El `uuid` es el ID del dashboard compartido. Puede usarlo para [actualizar](#update-dashboard-url-nerdgraph) o [revocar](revoke-dashboard-url-nerdgraph) la URL del panel compartido a través de NerdGraph. Para compartir el dashboard con usuarios externos, copie la URL y la contraseña de la respuesta. Las contraseñas no se pueden recuperar más tarde. Si la pierdes, sólo tienes que [restablecer la contraseña](#reset-password-dashboard-url-nerdgraph) y compartirla de nuevo con el usuario. Siga nuestras [mejores prácticas de gestión de contraseñas](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#password-management) para gestionar la contraseña de forma segura.
  </Collapser>

  <Collapser id="update-dashboard-url-nerdgraph" title="Actualizar la duración de expiración dashboard en tiempo real">
    Cerciorar de tener los [licencias](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/#permissions-to-custom-role) para modificar la configuración del dashboard compartido.

    Para actualizar el periodo de tiempo durante el cual la URL estará activa:

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya un [gráfico llamado API](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) usando la mutación `dashboardUpdateLiveUrl` con los siguientes parámetros:

    * `id`:Copie el ID de la URL dashboard compartido e ingréselo.

    * `options` (opcional) &gt; `ttl`: Ingrese el periodo de tiempo en segundos que la URL permanecerá activa.

      * **Valor mínimo**: `300` segundos (5 minutos)
      * **Valor máximo**: `631152000` segundos (20 años)
      * **Valor predeterminado**: `2592000` segundos (30 días)

    #### Mutación de muestra [#update-public-link-request]

    ```graphql
    mutation {
      dashboardUpdateLiveUrl(
        id: "PUBLIC_URL_ID"
        options: { ttl: TIME_IN_SECONDS }
      )
    }
    ```

    La URL aparece en la respuesta de la siguiente manera. Ahora la URL permanecerá activa durante el periodo de tiempo actualizado.

    #### Ejemplo de respuesta [#update-public-link-response]

    ```json
    {
      "data": {
        "dashboardUpdateLiveUrl": {
          "url": "https://share.newrelic.com/dashboard/PUBLIC_LINK_ID"
        }
      }
    }
    ```
  </Collapser>

  <Collapser id="enable-password-dashboard-url-nerdgraph" title="Habilitar la protección con contraseña para la URL en tiempo real">
    Cerciorar de tener los [licencias](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/#permissions-to-custom-role) para modificar la configuración del dashboard compartido. Para habilitar la protección con contraseña para la URL de un dashboard en tiempo real:

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya un [gráfico llamado API](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) usando la mutación `dashboardUpdateLiveUrlAuth` con los siguientes parámetros:

    * `auth` &gt; `factors` &gt; `type`: Seleccione `PASSWORD` para habilitar la protección con contraseña para la URL dashboard en tiempo real.
    * `id`:Copie el ID de la URL dashboard compartido e ingréselo.
    * `auth` &gt; `factors` &gt; `password` &gt; `value`: Seleccione esto para obtener la contraseña dashboard en tiempo real en la respuesta.
    * Seleccione cualquier otro campo que desee en la respuesta.

    #### Mutación de muestra [#enable-password-protection]

    ```graphql
    mutation {
      dashboardUpdateLiveUrlAuth(
        id: "YOUR_PUBLIC_LINK_UUID"
        auth: { factors: [{ type: PASSWORD }] }
      ) {
        factors {
          password {
            value # A NEW PASSWORD WILL BE GENERATED AND RETURNED
          }
          type
        }
      }
    }
    ```

    #### Ejemplo de respuesta [#enable-password-protection-response]

    La contraseña aparece en la respuesta de la siguiente manera.

    ```json
    {
      "data": {
        "dashboardUpdateLiveUrlAuth": {
          "factors": [
            {
              "password": {
                "value": "system generated password"
              },
              "type": "PASSWORD"
            }
          ]
        }
      }
    }
    ```

    Luego de habilitar la contraseña para el panel compartido, debe compartir la contraseña con el usuario. Copie y almacene la contraseña, ya que no podrá recuperarla más tarde. Si la pierdes, sólo tienes que [restablecer la contraseña](#reset-password-dashboard-url-nerdgraph) y compartirla de nuevo con el usuario. Siga nuestras [mejores prácticas de gestión de contraseñas](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#password-management) para gestionar la contraseña de forma segura.
  </Collapser>

  <Collapser id="disable-password-dashboard-url-nerdgraph" title="Deshabilitar la protección con contraseña para la URL activa">
    Cerciorar de tener los [licencias](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/#permissions-to-custom-role) para modificar la configuración del dashboard compartido. Para deshabilitar la protección con contraseña para la URL de un dashboard en tiempo real:

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya una [llamada API de gráfico](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) empleando la mutación `dashboardUpdateLiveUrlAuth` con los siguientes valores de atributo:

    * `auth` &gt; `factors` &gt; `type`: Seleccione `PASSWORD` para habilitar la protección con contraseña para la URL dashboard en tiempo real.
    * `id`:Copie el ID de la URL dashboard compartido e ingréselo.
    * `auth` &gt; `factors`: Manténgalo vacío.
    * `factors` &gt; `type`: Seleccione esto para obtener la confirmación en respuesta de que se eliminó el valor de la contraseña.

    #### Mutación de muestra [#disable-password-protection]

    ```graphql
    mutation {
      dashboardUpdateLiveUrlAuth(
        id: "YOUR_PUBLIC_LINK_UUID"
        auth: { factors: [] } # Send an empty array to remove all authentication
      ) {
        factors {
          # This will be empty
          type
        }
      }
    }
    ```

    ## Ejemplo de respuesta [#disable-password-protection-response]

    Verifique que la respuesta aparezca de la siguiente manera. Se elimina la contraseña y la URL dashboard en tiempo real ya no está protegida con contraseña.

    ```json
    {
      "data": {
        "dashboardUpdateLiveUrlAuth": {
          "factors": []
        }
      }
    }
    ```
  </Collapser>

  <Collapser id="reset-password-dashboard-url-nerdgraph" title="Restablecer la contraseña para la URL en tiempo real">
    Cerciorar de tener los [licencias](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/#permissions-to-custom-role) para modificar la configuración del dashboard compartido. Para restablecer la contraseña de una URL dashboard en tiempo real:

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya una [llamada API de gráfico](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) empleando la mutación `dashboardResetLiveUrlPassword` con los siguientes valores de atributo:

    * `id`:Copie el ID de la URL dashboard compartido e ingréselo.
    * `password`: Seleccione esta opción para obtener la nueva contraseña dashboard en tiempo real en la respuesta.

    #### Mutación de muestra [#reset-password]

    ```graphql
    mutation {
      dashboardResetLiveUrlPassword(id: "YOUR_PUBLIC_LINK_UUID") {
    ```

    ```json
          "password": "system generated password"
        }
      }
    }
    ```

    Luego de restablecer la contraseña del panel compartido, debe compartir la nueva contraseña con el usuario. Copie y almacene la contraseña, ya que no podrá recuperarla más tarde. Si lo pierdes solo tienes que resetear de nuevo. Siga nuestras [mejores prácticas de gestión de contraseñas](/docs/query-your-data/explore-query-data/dashboards/share-charts-dashboards-externally/#password-management) para gestionar la contraseña de forma segura.

  <Collapser id="revoke-dashboard-url-nerdgraph" title="Revocar una URL activa">
    En función de los [licencias](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/#permissions-to-custom-role) que tengas, puedes revocar solo las URL que creaste o todas las URL creadas por usuario en la cuenta. Para revocar una URL dashboard compartido:

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya una [llamada API de gráfico](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) empleando la mutación `dashboardRevokeLiveUrl` con el siguiente valor de atributo:

    * `id`:Copie el ID de la URL dashboard compartido e ingréselo.

    #### Mutación de muestra [#revoke-public-link]

    ```graphql
    mutation {
      dashboardRevokeLiveUrl(id: "PUBLIC_LINK_ID") {
        id
      }
    }
    ```

    #### Ejemplo de respuesta [#revoke-public-link-response]

    La URL dashboard compartido ahora está revocada y ya no será accesible. La respuesta aparece como sigue:

    ```json
    {
      "data": {
        "dashboardRevokeLiveUrl": {
          "id": "PUBLIC_LINK_ID" //the one you passed to the mutation
        }
      }
    }
    ```
  </Collapser>

  <Collapser id="list-dashboard-url-nerdgraph" title="Listar todas las URL compartidas">
    Dependiendo de los [licencias](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts/#permissions-to-custom-role) que tengas, puedes recuperar la lista de URL que creaste, o todas las URL creadas por el usuario en la cuenta. Para obtener la lista de URL dashboard compartidos:

    1. Vaya al [explorador de API de NerdGraph](https://one.newrelic.com/nerdgraph-graphiql).
    2. Construya una [llamada API de gráfico](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) empleando los siguientes detalles:

    * `actor` &gt; `dashboard` &gt; `liveUrls`: Use esto para recuperar la lista de URL dashboard compartido.
    * `filter` &gt; `type`: seleccione `DASHBOARD` para filtrar los resultados de las URL dashboard .
    * Seleccione cualquier otro campo que desee en la respuesta.

    #### Consulta de muestra [#list-public-links]

    ```graphql
    {
      actor {
        user {
          name
        }
        dashboard {
          liveUrls(filter: { type: DASHBOARD }) {
            liveUrls {
              createdAt
              type
              uuid
            }
          }
        }
      }
    }
    ```

    En respuesta aparece la lista de URL del dashboard compartido.
