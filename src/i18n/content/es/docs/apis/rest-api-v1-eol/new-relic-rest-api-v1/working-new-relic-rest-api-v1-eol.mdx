<Callout variant="important">
  La versión 1 de la API REST de New Relic ha llegado al final de su vida útil (EOL) y ya no funciona. Migre su script v1 a su equivalente NerdGraph para mantener la funcionalidad.
</Callout>

A continuación se muestran algunos ejemplos de alternativas a tareas comunes.

Para obtener información relacionada con cuentas a las que su usuario puede acceder:

```
actor {
  organization {
    accountManagement {
      managedAccounts {
        id
        name
      }
    }
  }
}
```

Ver [Administrar cuentas](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph/)

Para obtener una lista de aplicaciones a las que un usuario puede acceder, puede usarentitySearch en NerdGraph:

```
{
  actor {
    entitySearch(queryBuilder: {domain: APM, type: APPLICATION}) {
      query
      results {
        entities {
          name
          entityType
          guid
          ... on ApmApplicationEntityOutline {
            accountId
          }
        }
      }
    }
  }
}
```

Ver [entidad Buscar](/docs/apis/nerdgraph/examples/nerdgraph-entities-api-tutorial/#search-entity)
