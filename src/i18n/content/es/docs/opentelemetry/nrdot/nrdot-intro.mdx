La New Relic Distribution de OpenTelemetry (NRODT) se refiere a la colección de distribuciones de componentes del proyecto OpenTelemetry compatibles con New Relic. Los componentes de NRDOT están diseñados para funcionar sin problemas con los productos y servicios de New Relic y, cuando están disponibles, son la forma recomendada de integrar OpenTelemetry con New Relic.

<Callout variant="tip">
  Actualmente, NRDOT incluye distribuciones prediseñadas de OpenTelemetry [Collector](/docs/opentelemetry/nrdot/nrdot-collector/). Para conocer las API y SDK de idiomas, consulte nuestra documentación sobre cómo comenzar a trabajar con [el monitoreo de APM](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-intro/).
</Callout>
