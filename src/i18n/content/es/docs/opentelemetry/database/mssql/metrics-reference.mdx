Monitoree el rendimiento de su base de datos SQL Server con métricas exhaustivas recopiladas por el recolector NRDOT. Las siguientes 7 métricas se recopilan automáticamente para la funcionalidad de la UI de New Relic. Estas están **habilitadas de forma predeterminada** y funcionan en cualquier plataforma a través de una conexión directa a SQL Server.
