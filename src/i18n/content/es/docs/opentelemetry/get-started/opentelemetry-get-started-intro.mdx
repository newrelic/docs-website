El proyecto OpenTelemetry tiene un amplio alcance que incluye [especificacionesAPI y SDK](https://opentelemetry.io/docs/specs/otel/), [instrumentación](https://opentelemetry.io/docs/concepts/instrumentation/), el [recopilador](https://opentelemetry.io/docs/collector/), [convenciones semánticas](https://opentelemetry.io/docs/specs/semconv/), el protocolo [OTLP](https://opentelemetry.io/docs/specs/otlp/) y más. New Relic monitoreo de IA tendrá soporte de primera clase para OpenTelemetry y tiene una amplia variedad de capacidades de plataforma que facilitan el uso de las herramientas OpenTelemetry .

Las páginas siguientes se centran en patrones de integración comunes y proporcionan una base para emplear OpenTelemetry de manera eficaz con New Relic. Si bien no es exhaustivo, destaca cómo el usuario suele aprovechar estas herramientas y las capacidades de la plataforma New Relic creadas sobre ellas.
