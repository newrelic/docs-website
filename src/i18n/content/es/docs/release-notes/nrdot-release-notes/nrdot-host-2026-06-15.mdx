### Característica

* Valida que el caso de uso del host funcione correctamente con `nrdot-collector:1.17.0`.
* Se reemplazó `hostmetrics` con `host_metrics`, `resourcedetection` con `resource_detection` y `metricstransform` con `metrics_transform` debido a obsolescencia. Los `hostmetricsreceiver`, `resourcedetectionprocessor` y `metricstransformprocessor` cambiaron sus identificadores `metadata.yaml :: type` para usar `snake_case`, lo que cambió la forma en que se referencian en la configuración.
