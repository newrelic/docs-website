## Log de cambios

Primeras notas de la versión dedicadas al caso de uso de [&apos;Monitoreo de host&apos;](https://github.com/newrelic/nrdot-collector-releases/blob/main/distributions/nrdot-collector/README.md#use-cases), que se limitan a los cambios relevantes en la configuración o `nrdot-collector`. El [recolector nrdot](https://docs.newrelic.com/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01/) continuará incluyendo la configuración necesaria para impulsar este caso de uso. Sin embargo, con `nrdot-collector` admitiendo cada vez más casos de uso, queremos proporcionar notas de la versión adicionales que sean más fáciles de consumir para alguien que usa NRDOT con un propósito específico como el monitoreo de hosts.

* Caso de uso de host verificado para funcionar con `nrdot-collector:1.12.0`
