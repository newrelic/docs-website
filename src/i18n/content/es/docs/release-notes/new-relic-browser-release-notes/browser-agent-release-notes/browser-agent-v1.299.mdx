## versión 1.299.0

### Característica

#### Agregar métrica de compatibilidad de Flutter

Introduce una métrica de compatibilidad para rastrear el uso del agente de browser en la aplicación sitio web Flutter.

#### Agregue señales de frustración del usuario a `UserAction`

Proporcionar mejor información valiosa sobre las posibles frustraciones de los usuarios por acciones específicas del usuario a través de detecciones de clics muertos y clics erróneos.

#### Evaluar la precisión de la atribución entre características

Para respaldar el punto de referencia de mejora interna, agregue métricas de compatibilidad que evalúen la precisión de los atributos que indican el estado de un evento separado, como `hasReplay` o `hasTrace`.

### Corrección de errores

#### Corregir `recordReplay` inconsistencias de la API

Corrige un problema en el que una llamada API `recordReplay` realizada tardíamente no recolectaba datos correctamente.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para cerciorarte de obtener los últimos beneficios de características y rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de compatibilidad de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la versión 1.299.0 del agente de Browser se creó y probó con estos navegadores y rangos de versiones: Chrome 130-140, Edge 130-140, Safari 17-19 y Firefox 132-142. Para dispositivos móviles, la versión v1.299.0 fue creada y probada para Android OS 16 e iOS Safari 17-26.
