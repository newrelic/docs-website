## versión 1.297.0

### Característica

#### Prevenir cosechas tempranas cuando se está en periodo de reintento

Evita que el agente recopile datos de forma anticipada cuando tiene una tasa limitada. Si el servicio de ingesta devuelve un 429, el agente reemplazará sus datos e intentará la recolección nuevamente más tarde. Mientras eso sucede, ya no será posible recolectar datos de manera anticipada a medida que llegan nuevos datos, hasta que lo haga con éxito.

#### eliminar el atributo meta de newrelic

Eliminar el reporte del meta atributo \_\_newrelic de la carga de reproducción saliente. Esto se está llevando a cabo como una medida para ayudar a ahorrar bytes facturables para los clientes y reducir el costo de uso de las funcionalidades Session Replay .

### Corrección de errores

#### Se corrigen las excepciones para cuerpos nulos que surgen de cosechas vacías

Se realizó un cambio para garantizar que toda carga limpia garantice una salida válida. Antes, una cadena vacía que pasaba por el proceso de limpieza del cuerpo podía generar un error ya que el mecanismo devolvería el cuerpo como nulo. Ahora las cadenas vacías pasarán intactas, lo que permitirá que el procesamiento posterior se aplique con éxito.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para garantizar que obtengas las últimas características y beneficios de rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de compatibilidad de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la versión 1.297.0 del agente del Browser se creó y probó con estos navegadores y rangos de versiones: Chrome 129-139, Edge 129-139, Safari 17-19 y Firefox 131-141. Para dispositivos móviles, la versión v1.297.0 fue creada y probada para Android OS 16 e iOS Safari 17-18.1.
