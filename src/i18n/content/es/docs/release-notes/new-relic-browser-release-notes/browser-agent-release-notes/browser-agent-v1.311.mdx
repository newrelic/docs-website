## v1.311.0

### Característica

#### Eliminación de agentIdentifier

Se eliminaron las referencias de ámbito por identificador de agente para simplificar la arquitectura del agente. Todas las referencias ahora se almacenan directamente dentro de la referencia del agente, eliminando la necesidad de combinar las referencias del agente con objetos con alcance mediante claves de búsqueda.

#### Asignar la URL de la página original a todos los nodos PageViewTiming

Se amplió la atribución de `pageUrl` a todos los tiempos, basándose en el trabajo anterior para LCP. Los eventos `PageViewTiming` ahora reportan la URL de navegación dura original para todos los tipos de nodo, asegurando la consistencia de los datos para SPA incluso si la URL cambia antes de la recolección.

#### mejorar la lógica de temporización de MFE

Se mejoró la capacidad del agente para identificar el script MFE correcto que llama a la API de registro. Esta mejora da como resultado información de tiempos más precisa para entornos MFE.

### Corrección de errores

#### Exigir IDs de cadena para nuevos registros de MFE

Se resolvió un problema en el que los ID que no son cadenas causaban errores durante la síntesis de entidades MFE. El agente ahora impone IDs basados en cadenas y emite una advertencia si se proporciona un tipo de dato diferente.

#### El reemplazador circular eliminó objetos reutilizados de las recolecciones

Se solucionó un problema en el que el método interno stringify eliminaba punteros compartidos en las cargas útiles de recolección, lo que ocasionalmente causaba la falta de atributos `params` en las recolecciones de errores.

#### Actualizar RegExp para evitar un problema de compilación de webpack sin minimizar

Se reemplazó el carácter `$` por su equivalente hexadecimal en expresiones regulares. Esto evita que Webpack reemplace incorrectamente el carácter durante el proceso de compilación, lo que anteriormente causaba fallas en el agente en compilaciones no minimizadas.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para garantizar que obtengas las últimas características y beneficios de rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de soporte de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la v1.311.0 del agente Browser se creó y probó con estos navegadores y rangos de versiones: Chrome 134-144, Edge 134-144, Safari 18-26 y Firefox 137-147. Para dispositivos móviles, la v1.311.0 se compiló y probó para Android OS 16 y iOS Safari 18-26.
