## v1.310.0

### Característica

#### mejor manejo de los tiempos de MFE

Mejora la detección de recursos de script reportados por el navegador que alimentan los atributos timeToFetch y timeToBeRequested del evento MicroFrontEndTiming. Este cambio también incluye los nuevos atributos assetUrl y assetType para ilustrar mejor los activos utilizados para completar los datos de temporización.

#### Eliminación total de la antigua funcionalidad de SPA y los envoltorios relacionados

Esto concluye la eliminación completa de la antigua función de SPA que es reemplazada por la función de Soft Navigations. Desde la versión 1.307.0 del agente, el &quot;SPA&quot; predeterminado ya se había cambiado, por lo que no se esperan cambios disruptivos a partir de esta versión.

#### Capturar tiempos de MFE como eventos MicroFrontEndTiming

Captura los tiempos del ciclo de vida de las entidades registradas en preparación para un futuro producto de micro frontend

### Corrección de errores

#### Actualizar tipos de API de medidas

Corrige un problema en el que los tipos asociados con la API measure no aceptaban PerformanceMark como una entrada válida para los valores de inicio o fin.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para garantizar que obtengas las últimas características y beneficios de rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de soporte de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la versión 1.310.0 del agente Browser se creó y probó con estos navegadores y rangos de versiones: Chrome 134-144, Edge 134-144, Safari 17-26 y Firefox 137-147. Para dispositivos móviles, la v1.310.0 se compiló y probó para Android OS 16 e iOS Safari 17-26.
