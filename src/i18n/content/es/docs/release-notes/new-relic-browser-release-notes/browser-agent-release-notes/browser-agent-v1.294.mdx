## versión 1.294.0

### Característica

#### Cosecha temprana

El agente ahora podrá activar cosechas anticipadas si se cumplen ciertos criterios. Esto ayudará a garantizar que se conserve más carga y se eviten fugas de memoria. El agente iniciará una recolección temprana cuando el tamaño del búfer de característica alcance los 16 KB; de lo contrario, continuará recolectando cada 30 segundos como antes. Este nuevo comportamiento se aplicará a los siguientes eventos:

* Solicitud de Ajax
* BrowserInteraction
* BrowserPerformance
* Evento personalizado
* Log
* Acción de página
* Tiempo de visualización de página
* Repetición de sesión
* Acción del usuario

Se realizará un esfuerzo separado para permitir la recolección temprana de métricas de intervalos de tiempo, errores de javascript y rastreo de sesión que emplean otros mecanismos de almacenamiento en el agente.

### Corrección de errores

#### reportar URL anterior vacía como indefinida

El atributo `previousUrl` del evento BrowserInteraction se informó como una cadena vacía en los casos en que document.referrer está vacío. Esto provocó que la agrupación de URL aplicara la agrupación en una cadena vacía, lo que provocó que se agregaran grupos sin sentido a previousUrl. `previousUrl` ahora se informa como `undefined` para permitir que la agrupación de URL omita las URL vacías.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para garantizar que obtengas las últimas características y beneficios de rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de compatibilidad de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la versión 1.294.0 del agente de Browser se creó y probó con estos navegadores y rangos de versiones: Chrome 128-138, Edge 128-138, Safari 17-19 y Firefox 130-140. Para dispositivos móviles, la versión v1.294.0 fue creada y probada para Android OS 16 e iOS Safari 17-26.
