## versión 1.304.0

### Característica

#### Mensaje de advertencia al restablecer la sesión

Agrega un código de advertencia cuando ocurre un restablecimiento de sesión del agente para que podamos tener una mejor depuración del comportamiento de transición del navegador.

#### Reducir el tamaño de las instantáneas Session Replay

Emplee `slimDOMOption = 'all'` de forma predeterminada para reducir el tamaño de carga de las instantáneas de reproducción de sesión.

### Corrección de errores

#### Observe document.body después de que el documento sea interactivo

Soluciona el escenario donde se emitió `MutationObserver.observe()` antes de que `document.body` estuviera disponible, lo que provocaba el error `Failed to execute 'observe' on 'MutationObserver': parameter 1 is not of type 'Node'.`.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para garantizar que obtengas las últimas características y beneficios de rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de compatibilidad de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la versión 1.304.0 del agente del navegador se creó y probó con estos navegadores y rangos de versiones: Chrome 132-142, Edge 132-142, Safari 17-26 y Firefox 134-144. Para dispositivos móviles, la versión v1.304.0 fue creada y probada para Android OS 16 e iOS Safari 17-26.
