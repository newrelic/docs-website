## v1.293.0

### Característica

#### Agregar mensaje interno de &quot;tareas largas&quot;

Agregue un mensaje interno que indique si una función envuelta califica como una &quot;tarea larga&quot;. Esto se empleará más adelante en el agente característica para mejorar comportamientos futuros.

#### Emitir SM en respuesta RUM marca de tiempo no válida

Emite una métrica de compatibilidad al recibir una timestamp no válida en la respuesta de la llamada RUM.

### Corrección de errores

#### Problema de solución que impide que se deshabilite el rastreo distribuido

Se solucionó un error en el que el agente podía capturar el rastreo distribuido, incluso cuando se suministraba `distributed_tracing: false` en la configuración.

## Declaración de apoyo

New Relic recomienda que actualices el agente periódicamente para garantizar que obtengas las últimas características y beneficios de rendimiento. Las versiones anteriores ya no recibirán soporte cuando lleguen [al final de su vida útil](https://docs.newrelic.com/docs/browser/browser-monitoring/getting-started/browser-agent-eol-policy/). Las fechas de lanzamiento reflejan la fecha de publicación original de la versión del agente.

Las nuevas versiones del agente del browser se lanzan a los clientes en pequeñas etapas a lo largo de un periodo de tiempo. Debido a esto, la fecha en que el lanzamiento esté disponible en su cuenta puede no coincidir con la fecha de publicación original. Consulte este [dashboard de estado](https://newrelic.github.io/newrelic-browser-agent-release/) para obtener más información.

De acuerdo con nuestra [política de compatibilidad de navegadores](https://docs.newrelic.com/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring/#browser-types), la versión 1.293.0 del agente del navegador se creó y probó con estos navegadores y rangos de versiones: Chrome 128-138, Edge 127-137, Safari 17-19 y Firefox 129-139. Para dispositivos móviles, la versión v1.293.0 fue creada y probada para Android OS 16 e iOS Safari 17-26.
