### Soporte para Fluent Bit 5.0.8

Los usuarios ahora recibirán la versión 5.0.8 de Fluent Bit. Para obtener más detalles, consulte los [anuncios de Fluent Bit](https://fluentbit.io/announcements/).

Esta versión incluye [New Relic Fluent Bit Output Plugin 3.7.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.7.0).

### Agregado

* Plug-in de salida Fluent Bit de New Relic 3.7.0 integración

### Cambió

* La versión de Go se actualiza a 1.26.4 para correcciones de CVE
* El agente de infraestructura recomienda los paquetes de Fluent Bit 5.0.8 (Linux) y 5.0.7 (Windows)
* La [imagen de Docker del plug-in de salida de Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) utiliza Fluent Bit 5.0.8 con el plug-in de salida 3.7.0
* [El chart de Helm de New Relic Logging](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) instala la nueva imagen del plug-in
* La imagen de Docker de Windows utiliza Fluent Bit 5.0.7 (la imagen de Windows 5.0.8 aún no está disponible en el origen)
