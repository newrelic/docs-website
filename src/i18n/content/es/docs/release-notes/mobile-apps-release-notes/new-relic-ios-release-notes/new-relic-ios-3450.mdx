### Notas

Esta versión contiene notificaciones mejoradas, con soporte para seleccionar sonidos únicos para diferentes tipos de notificaciones, además de mostrar una imagen de gráfico para eventos de alerta.

### Nueva característica

* Seleccione diferentes sonidos de notificación para las notificaciones push de la aplicación.
* Muestra la imagen del gráfico de eventos de alerta abiertos / cerrados en la notificación.

### Corrección de errores

* Corrección para filtrar apmApplicationIds y apmApplicationNames correctamente en Infrastructure.
* Se corrigió una falla al intentar buscar en los componentes de plugins.
* Se corrigió un cierre inesperado en Radar.
