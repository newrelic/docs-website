### Notas

Ahora incluye una barra de pestañas global, con un explorador para todas sus entidades.

### Nueva característica

* Se agregó una página de explorador de entidades que admite integraciones de Infraestructura, junto con muchos otros tipos de entidades
* Se agregó una barra de pestañas global para permitirle acceder rápidamente a Inicio, Explorador, Dashboards, Alertas y Consultas desde cualquier lugar
* Se agregó soporte para servicios reportados por integración
* Agrega soporte para anomalías en eventos de alerta para cuentas de la UE
