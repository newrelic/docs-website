### Notas

¡Presentamos la edición de NRQL desde la [aplicación para iPhone](https://docs.newrelic.com/docs/mobile-apps/new-relic-mobile-apps/ios-app/introduction-ios-mobile-app)!

### Nueva característica

* Agrega el historial de consultas NRQL al iPhone (el iPad llegará más tarde).
* Agrega la capacidad de escribir consultas NRQL.
* Vincula eventos de alerta que apuntan a una consulta NRQL para mostrar el editor NRQL en la aplicación.
* Incluye más pantallas de detalles en el iPad para APM, Browser, Mobile y transacciones clave.
