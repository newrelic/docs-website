### Notas

Soporte para el contexto de eventos de alertas de [New Relic Radar](https://docs.newrelic.com/docs/radar/new-relic-radar/getting-started/introduction-radar), que es una herramienta para una orientación más rápida durante un evento de alerta. El contexto del evento de alerta proporciona una orientación más rápida durante un evento de alerta al detectar un comportamiento inusual en su sistema y ubicarlo junto a la información de la alerta.

### Nueva característica

* Contexto del evento de alerta de Radar

### Mejoras

* Se mejoraron las llamadas de lista de Infraestructura para reducir la latencia y el uso de la red

### Correcciones

* Fallo al usar el botón atrás en vistas de detalle
