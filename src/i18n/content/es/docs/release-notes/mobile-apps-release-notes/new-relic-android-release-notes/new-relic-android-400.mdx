### Notas

* Esta versión agrega soporte para cerrar un evento de alerta abierto para usuarios autorizados y también añade más soporte de enlaces profundos para los enlaces compartidos desde New Relic One.
* Esta versión también incluye una verificación de feature flags para los plugins que no mostrará los plugins como un elemento de menú a menos que la cuenta tenga un feature flag que habilite los plugins.

### Correcciones

* Pantallas rotas en modo oscuro.
* Notificaciones descartadas.
* Filtrado de dashboards.
