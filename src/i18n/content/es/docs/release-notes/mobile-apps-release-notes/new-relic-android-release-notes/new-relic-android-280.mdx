### Notas

Excelentes nuevas funciones para la versión 2.8.0

### Nueva característica

* Se agregaron KPI (indicadores clave de desempeño) de resumen y gráficos de KPI a los eventos de alerta de la aplicación.
* Se agregó la capacidad de limitar la página de detalles de APM a un Host o instancia de JVM específicos
* Se agregó un nuevo indicador de actividad

### Mejoras

* Eliminar minigráficos
* Nueva ventana de tiempo con tiempo de violación de eventos de alerta
* Flujo de navegación mejorado desde eventos de alerta

### Correcciones

* Se corrigió el reconocimiento de eventos de alerta en la notificación push
