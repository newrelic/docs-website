### Mejoras

* Se actualizó el navegador Chrome a la versión 143.0.7499.146. Los usuarios pueden observar una advertencia de desaprobación en los logs del navegador que indica &quot;Se ha desaprobado la reversión automática a WebGL de software.&quot; Este cambio elimina la funcionalidad de reversión automática para abordar una vulnerabilidad de seguridad de alta gravedad ([CVE-2025-14765](https://chromereleases.googleblog.com/2025/12/stable-channel-update-for-desktop_16.html)).
