### Mejoras

Se corrigieron las vulnerabilidades del sistema operativo subyacente y de Java en el runtime de Ping.

* CVE-2026-2332
* Se agregó soporte de doble pila (IPv4 e IPv6) para ubicaciones privadas contenerizadas. Synthetics Job Manager ahora funciona en entornos IPv6 manteniendo total compatibilidad con IPv4. Las plataformas de contenedores compatibles incluyen Docker, Kubernetes y Podman.
