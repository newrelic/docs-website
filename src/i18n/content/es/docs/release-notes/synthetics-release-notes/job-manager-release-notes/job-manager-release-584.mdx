### Correcciones

* Optimización de memoria: manejo simplificado de la ruta de resultados del trabajo para reducir la presión del heap de las grandes cargas del monitor del navegador.
* Estabilidad mejorada: reduce la presión de la recolección de basura (GC), lo que aumenta la confiabilidad para las ubicaciones que ejecutan monitores de Browser pesados con archivos HAR masivos.

Se corrigieron las siguientes vulnerabilidades:

* CVE-2025-45582
* CVE-2025-69720
* CVE-2026-41991
* CVE-2026-41992
* CVE-2026-58055
* CVE-2026-9828
