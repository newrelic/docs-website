Se lanzó una nueva versión del agente. Siga los procedimientos estándar para [actualizar el agente de infraestructura](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recomienda que actualices el agente periódicamente y como mínimo cada 3 meses. A partir de esta versión, la versión compatible más antigua es [agente de infraestructura 1.51.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1510/).

## Cambió

* Se eliminaron las compilaciones de Windows de 32 bits en [#2144](https://github.com/newrelic/infrastructure-agent/pull/2144)
* Versión actualizada de Fluent Bit en Windows en [#2156](https://github.com/newrelic/infrastructure-agent/pull/2156)
* Versiones de dependencia actualizadas en [#2159](https://github.com/newrelic/infrastructure-agent/pull/2159)
* Se solucionó un problema por el cual los contadores de rendimiento de la CPU no funcionaban en sistemas que no estaban en inglés en [#2154](https://github.com/newrelic/infrastructure-agent/pull/2154)
