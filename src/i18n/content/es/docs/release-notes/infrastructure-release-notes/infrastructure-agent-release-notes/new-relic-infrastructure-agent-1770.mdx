Se ha lanzado una nueva versión del agente. Siga los procedimientos estándar para [actualizar el agente de infraestructura](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recomienda que actualice el agente regularmente y, como mínimo, cada 3 meses. A partir de este lanzamiento, la versión compatible más antigua es el [agente de Infrastructure 1.65.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1651/).

## Cambió

* Se actualizó la dependencia newrelic/nri-prometheus a v2.29.3 en [#2264](https://github.com/newrelic/infrastructure-agent/pull/2264)
* Se agregó soporte para Ubuntu 26.04 LTS (Resolute Raccoon) en [#2224](https://github.com/newrelic/infrastructure-agent/pull/2224)
* Se actualizó la dependencia `newrelic/nri-flex` a la v1.18.6 en [#2266](https://github.com/newrelic/infrastructure-agent/pull/2266)
