Se lanzó una nueva versión del agente. Siga los procedimientos estándar para [actualizar el agente de infraestructura](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). A partir de esta versión, la versión compatible más antigua es [agente de infraestructura 1.58.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1581/).
