Se ha lanzado una nueva versión del agente. Siga los procedimientos estándar para [actualizar el agente de infraestructura](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent). New Relic recomienda que actualice el agente regularmente y, como mínimo, cada 3 meses. A partir de este lanzamiento, la versión compatible más antigua es el [agente de infraestructura 1.64.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1640/).

### Cambió

* Se actualizó la dependencia nri-docker a 2.6.10, dependencia nri-flex a v1.17.7 y dependencia docker a v29 en [#2220](https://github.com/newrelic/infrastructure-agent/pull/2220)
* Se crearon imágenes de Docker para el agente de infraestructura para Windows en [#2202](https://github.com/newrelic/infrastructure-agent/pull/2202)
* Se actualizó el flujo de trabajo de lanzamiento de imagen de Windows en [#2213](https://github.com/newrelic/infrastructure-agent/pull/2213)

* Aborda el [CVE-2026-34040](https://github.com/advisories/GHSA-x744-4wpc-v9h2)
