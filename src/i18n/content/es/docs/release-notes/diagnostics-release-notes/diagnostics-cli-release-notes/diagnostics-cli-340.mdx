## Actualizaciones de tareas

* Se agregó una nueva opción de línea de comando `-ac-agents-namespace` para especificar un namespace para el agente de control ([#243](https://github.com/newrelic/newrelic-diagnostics-cli/pull/243)). Se actualizaron las siguientes tareas para emplear el nuevo namespace:

  * K8s/Resources/Config
  * K8s/Resources/Daemonset
  * K8s/Resources/Deploy
  * K8s/Resources/Pods

## Correcciones

* Se resolvieron los errores relacionados con GLIBC como `/lib64/libc.so.6: version 'GLIBC_2.34' not found` ([#244](https://github.com/newrelic/newrelic-diagnostics-cli/pull/244)).
