## Nueva característica

* La CLI de Diagnostics ahora es compatible con la región de datos de Japón de New Relic. Si su cuenta está en la región de Japón, especifique el indicador `region` al ejecutar la CLI ([#263](https://github.com/newrelic/newrelic-diagnostics-cli/pull/263)):

  ```bash
  ./nrdiag -region JP
  ```

  La bandera `region` acepta `US`, `EU` o `JP` (no distingue entre mayúsculas y minúsculas) y su valor predeterminado es `US`.

## Actualizaciones de dependencias

* Se actualizó `golang.org/x/net` de 0.33.0 a 0.55.0 ([#264](https://github.com/newrelic/newrelic-diagnostics-cli/pull/264)).
