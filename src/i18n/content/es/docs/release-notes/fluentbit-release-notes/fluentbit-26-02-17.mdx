### Soporte para Fluent Bit 4.2.2

Los usuarios ahora recibirán la versión 4.2.2 de Fluent Bit. Para obtener más detalles, consulte los [anuncios de Fluent Bit](https://fluentbit.io/announcements/).

Esta versión incluye [New Relic Fluent Bit Output Plugin 3.4.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.4.0) con actualizaciones de seguridad críticas y compatibilidad de plataforma mejorada.

### Agregado

* Soporte para SLES 15.6 y 15.7
* Plugin de salida Fluent Bit de New Relic 3.4.0 integración

### Cambió

* El agente de infraestructura recomienda los paquetes de Fluent Bit 4.2.2
* La [imagen de Docker del plugin de salida de Fluent Bit](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) utiliza Fluent Bit 4.2.2 con el plugin de salida 3.4.0
* imagen base de aws-for-fluent-bit actualizada desde 3.1.0 a 3.2.1

### Fijado

* Problemas de compatibilidad de GLIBC en RHEL8 y Amazon Linux 2 usando compilación cruzada con Zig apuntando a GLIBC 2.17

### Notas

Para mantener actualizado con las más recientes correcciones y mejoras, suscribir a nuestra [fuente RSS de Logs](/docs/release-notes/logs-release-notes/).
