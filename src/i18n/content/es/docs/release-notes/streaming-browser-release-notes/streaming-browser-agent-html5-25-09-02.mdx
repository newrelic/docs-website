## Característica

### Agente independiente

* **Desacoplado del agente Browser**: El agente de video ahora opera de forma independiente con su propia lógica de recolección. Esta separación reduce la sobrecarga y garantiza que los datos de video se recopilen y envíen a New Relic sin estar vinculados a la programación de recopilación de datos del agente del navegador.
