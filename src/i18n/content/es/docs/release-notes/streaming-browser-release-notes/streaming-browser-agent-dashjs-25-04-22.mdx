Este lanzamiento del rastreador New Relic Dash.js incluye actualizaciones de compatibilidad importantes y un cambio en el reporte de errores.

## Cambios clave

* **Compatibilidad con el reproductor Dash.js**: Esta versión ha sido probada y confirmada como totalmente compatible con la última versión del reproductor Dash.js, garantizando una integración fluida y un seguimiento preciso.

* **Actualización de reporte de errores**: El atributo `errorName`, utilizado anteriormente para identificar la descripción del error, ha quedado obsoleto. El atributo `errorMessage` es ahora el estándar para transmitir descripciones detalladas de errores. Actualice cualquier sistema de monitoreo o logging que dependa de la información de errores para utilizar el atributo `errorMessage` de ahora en adelante.

## Notas de actualización

Los usuarios que dependen del atributo `errorName` para el análisis de errores deben planificar la migración al atributo `errorMessage` para garantizar la funcionalidad continua y el acceso a información detallada de los errores.
