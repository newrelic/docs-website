### Actualizando su agente

* Siga los procedimientos estándar para [actualizar el agente .NET](/docs/agents/net-agent/installation-configuration/update-net-agent).
* Si está empleando un agente particularmente antiguo, revise la lista de cambios principales y procedimientos para [actualizar el agente .NET legacy](/docs/apm/agents/net-agent/installation/update-net-agent/#updating_older_net).

Recomendamos actualizar a la última versión del agente tan pronto como esté disponible. Si no puede actualizar a la última versión, actualice su agente a una versión que no tenga más de 90 días de antigüedad. Obtenga más información sobre [cómo mantener a su agente actualizado](/docs/new-relic-solutions/new-relic-one/install-configure/update-new-relic-agent). Consulte el [documento de política de fin de vida](/docs/apm/agents/net-agent/getting-started/net-agent-eol-policy) del agente .NET de New Relic para obtener información sobre las versiones del agente y las fechas de soporte.
