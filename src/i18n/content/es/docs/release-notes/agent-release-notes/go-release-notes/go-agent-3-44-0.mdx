## 3.44.0

### Agregado

  \* Se agregó soporte para el SDK de Anthropic `nranthropic`   \* Se agregó soporte para GocqlX `nrgocqlx`

### Fijado

  \* Exportar tipos de pruebas internas a través de `integrationsupport` para desbloquear a los consumidores de pruebas externos     \* Gracias al miembro de la comunidad @nitinprajwal por contribuir a esta solución   \* Se corrigió un error en sqlparse para admitir sentencias `INTO` opcionales en las integraciones de sql   \* Se corrigió un error en `nrnats` que causaba pruebas inestables

### Seguridad

  \* Actualizaciones de seguridad de Dependabot     \* `nrfiber`

### Declaración de apoyo

Empleamos la última versión del lenguaje Go. Como mínimo, no deberías emplear ninguna versión de Go anterior a la compatible con el propio equipo de Go. Consulte la [Política de fin de vida del agente Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) para obtener detalles sobre las versiones compatibles del agente Go y los componentes de terceros.
