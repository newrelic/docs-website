## 3.43.3

### Característica

* EXPERIMENTAL: se agregó una opción de configuración `ConfigAppLogDecoratingWithinMessage` para permitir que el decorador de log local (Local Log Decorator) aparezca dentro de un campo de mensaje. Esto solo se puede aplicar a la integración `logcontext-v2/nrlogrus`. La integración `logcontext-v2/nrlogrus` se actualizará a la v1.1.4.

### Seguridad

* Actualizaciones de seguridad de Dependabot

  * `nrawsbedrock` (la integración se actualizará a v1.1.1)
  * `nrawssdk-v2` (la integración se actualizará a la v1.2.7)
  * `nrpgx5` (la integración se actualizará a v1.3.4)\`

### Declaración de apoyo

Empleamos la última versión del lenguaje Go. Como mínimo, no deberías usar ninguna versión de Go anterior a la que admite el propio equipo de Go. Consulte la [política de fin de vida útil del agente Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) para obtener detalles sobre las versiones compatibles del agente Go y los componentes de terceros.
