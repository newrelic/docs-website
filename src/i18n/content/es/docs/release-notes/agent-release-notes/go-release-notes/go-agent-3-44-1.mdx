### Declaración de apoyo

Usamos la última versión del lenguaje Go. Como mínimo, usa una versión de Go que el equipo de Go aún soporte. Consulta la [política de EOL del agente de Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) para obtener detalles sobre las versiones compatibles del agente de Go y los componentes de terceros.
