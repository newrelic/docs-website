Empleamos la última versión del lenguaje Go. Como mínimo, no deberías emplear ninguna versión de Go anterior a la compatible con el propio equipo de Go. Consulte la [Política de fin de vida del agente Go](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) para obtener detalles sobre las versiones compatibles del agente Go y los componentes de terceros.
