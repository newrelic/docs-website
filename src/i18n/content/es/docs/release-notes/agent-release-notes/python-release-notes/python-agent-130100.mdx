## Notas

Esta versión del agente de Python agrega soporte para varios comandos nuevos de [Redis](https://redis.io/docs/latest/commands/redis-8-8-commands), corrige un error en la instrumentación de [LangGraph](https://pypi.org/project/langgraph) y mejora el manejo y el logging de la configuración de los límites de recolección.

Instale el agente usando `easy_install/pip/distribute` a través del [Índice de paquetes de Python](https://pypi.python.org/pypi/newrelic), usando `conda` a través del [Índice de paquetes de Conda-Forge](https://anaconda.org/conda-forge/newrelic), o descárguelo directamente desde el [sitio de descargas de New Relic](https://download.newrelic.com/python_agent/release).

## Características mejoradas

* Agregar instrumentación para nuevos comandos de Redis v8.8.0

  * Agregue instrumentación para los siguientes [nuevos comandos de Redis](https://redis.io/docs/latest/commands/redis-8-8-commands/#array-commands) de [redis-py](https://pypi.org/project/redis/) v8.8.0:

    * Comandos varios: INCREX, XNACK
    * Comandos de matriz: ARCOUNT, ARDEL, ARDELRANGE, ARGET, ARGETRANGE, ARGREP, ARINFO, ARINSERT, ARLASTITEMS, ARLEN, ARMGET, ARMSET, ARNEXT, AROP, ARRING, ARSCAN, ARSEEK, ARSET

## Errores

* Corregir fallo en la instrumentación de LangGraph

  * Anteriormente, al usar la instrumentación de [LangGraph](https://pypi.org/project/langgraph), el agente fallaba debido a un problema con la propagación del contexto a los trabajos en AsyncBackgroundExecutor. Esto se ha corregido, y el agente ahora debería funcionar correctamente con LangGraph.

* Manejo y logging mejorados de la configuración del límite de recolección

  * El agente ahora reporta con mayor precisión la lista de métodos de recolección y límites a la pestaña Entorno en New Relic One.
  * El agente ahora tiene un logging de depuración más consistente al inicio para la configuración del límite de recolección.
