## Notas

Esta versión del agente Python agrega soporte para Google Gemini a través del SDK generativo de Google IA e instrumenta nuevos métodos de Kinesis.

Instale el agente usando `easy_install/pip/distribute` a través del [Índice de paquetes de Python](https://pypi.python.org/pypi/newrelic) o descárguelo directamente del [sitio de descarga de New Relic](https://download.newrelic.com/python_agent/release).

## Nueva característica

* Agregue compatibilidad con Google Gemini a través del SDK de IA generativa de Google

  * Agrega soporte para [google-GenAI](https://pypi.org/project/google-genai/). El agente automáticamente llamará al instrumento `embed_content` para [incrustaciones](https://ai.google.dev/gemini-api/docs/embeddings) sincrónicas y asincrónicas. El agente ahora también admite [generaciones de texto](https://ai.google.dev/gemini-api/docs/text-generation) sincrónico y asincrónico para entradas de texto en casos que no son de transmisión. Esto incluye llamadas realizadas a `generate_content` para entradas de solo texto y llamadas a `send_message` para conversaciones de múltiples turnos.

* Nuevos métodos de AWS Kinesis

  * Agrega instrumentación [botocore](https://pypi.org/project/botocore/) para admitir nuevos métodos [de AWS Kinesis](https://aws.amazon.com/kinesis/) , incluidos `tag_resource`, `untag_resource` y `list_tags_for_resource`.
