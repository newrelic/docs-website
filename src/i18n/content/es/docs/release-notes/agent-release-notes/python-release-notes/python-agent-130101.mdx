## Notas

Esta versión del agente de Python agrega información adicional del modelo a los eventos de LLM de la cadena de [LangChain](https://pypi.org/project/langchain) y soluciona un error en la instrumentación de streaming de [LangGraph](https://pypi.org/project/langgraph).

Instale el agente usando `easy_install/pip/distribute` a través del [Índice de paquetes de Python](https://pypi.python.org/pypi/newrelic), usando `conda` a través del [Índice de paquetes de Conda-Forge](https://anaconda.org/conda-forge/newrelic), o descárguelo directamente desde el [sitio de descargas de New Relic](https://download.newrelic.com/python_agent/release).

## Características mejoradas

* Agregar información adicional del modelo a los eventos de LLM de la cadena de LangChain

  * Agregue los atributos `request.model` y `response.model` a los eventos de LLM producidos por las cadenas de [LangChain](https://pypi.org/project/langchain).
  * Esta información se recopila en la medida de lo posible y puede no estar siempre disponible.

## Errores

* Corregir TypeError en la instrumentación de streaming de LangGraph

  * Anteriormente, al usar un subconjunto de API de streaming en [LangGraph](https://pypi.org/project/langgraph), el agente encontraba un `TypeError` debido al uso de `weakref.WeakMethod` por parte de LangGraph. Esto se ha corregido, y el agente ahora debería funcionar correctamente con esos métodos de streaming de LangGraph.
