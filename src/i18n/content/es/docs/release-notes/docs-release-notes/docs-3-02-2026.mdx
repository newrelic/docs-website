### Cambios importantes

* Se actualizó la [documentación de control de agentes de New Relic Control](/docs/new-relic-control/agent-control/overview) con soporte completo para hosts Windows, incluidas nuevas opciones de configuración, requisitos de red y guía de resolución de problemas.
* Se mejoró la [documentación de monitoreo de browser PHP](/docs/apm/agents/php-agent/features/browser-monitoring-php-agent) con instrucciones detalladas para agregar atributos personalizados a los datos de monitoreo de browser.
* Se actualizó la [documentación de configuración del agente Python](/docs/apm/agents/python-agent/configuration/python-agent-configuration) con ejemplos de configuración del muestreador de agente híbrido.

### Cambios menores

* Se actualizó la [documentación de la integración de AWS CloudWatch](/install/aws-cloudwatch) para aclarar que cada cuenta de AWS requiere un sink por región para el monitoreo multirregión.
* Se actualizó la documentación de la [instalación de Kubernetes con eBPF](/docs/ebpf/k8s-installation) y la [instalación en Linux](/docs/ebpf/linux-installation) con información de configuración del nombre de la aplicación de New Relic.
* Se actualizó la [integración de RabbitMQ con OpenTelemetry](/docs/opentelemetry/integrations/rabbitmq/overview) con soporte para NRDOT Collector tanto para entornos de Kubernetes como autohospedados.
* Se actualizó la [documentación del modo de alta seguridad](/docs/apm/agents/manage-apm-agents/configuration/high-security-mode).
* Se actualizó la [documentación de la integración de monitoreo de puertos](/docs/infrastructure/host-integrations/host-integrations-list/port-monitoring-integration).
* Se actualizó la [documentación de equipos](/docs/service-architecture-intelligence/teams/teams).
* Se actualizó la [documentación de gestión de equipos](/docs/service-architecture-intelligence/teams/manage-teams).
* Se actualizó la [documentación de compatibilidad del agente Java](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent).
* Se actualizó la [documentación de compatibilidad del agente de Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent).

### Novedades

* Se agregó [Novedades para el soporte de Windows en New Relic Control](/whats-new/2026/02/whats-new-23-02-windows-support/) para anunciar el soporte para hosts de Windows y la selección de agentes mejorada en New Relic Control (Vista previa pública).
* Se actualizó [Novedades para la observabilidad del recopilador](/whats-new/2026/02/whats-new-02-24-collector-observability/).
