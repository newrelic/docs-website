### Cambios importantes

* Se agregó el atributo [traza.id](/attribute-dictionary/?event=AjaxRequest&attribute=trace.id) al evento `AjaxRequest` que actúa como un identificador distintivo para requests individuales, lo que garantiza su diferenciación a través de límites entre procesos y dentro de ellos.

* Se eliminó la sección de integración de AmazonQ de la documentación de New Relic.

* Se actualizó la [cláusula PREDICT](/docs/nrql/nrql-syntax-clauses-functions/##sel-prediction) con los comportamientos predeterminados.

* Se eliminó el agente de video y todos los documentos correspondientes de la documentación de New Relic.

* Marqué el [tutorial de NerdGraph para el canal de alerta de notificación](/docs/apis/nerdgraph/examples/nerdgraph-api-alert-notification-channels/) como obsoleto.

* Se suspendió el soporte para [la cobertura de pruebas](/docs/infrastructure/infrastructure-agent/linux-installation/instrument-container/#test-coverage) de extremo a extremo (E2E) en el sistema operativo que emplea Control Group v1.

* Se agregó un nuevo estado de investigación en la [sección de alerta](/docs/alerts/alerts-ai-overview-page/).

### Cambios menores

* Se actualizó la [configuración del agente .Net](/docs/apm/agents/net-agent/configuration/net-agent-configuration/).

* Se actualizó el [archivo de configuración del agente de Python](/docs/apm/agents/python-agent/configuration/python-agent-configuration/) para incluir la &apos;variable de entorno&apos; a través de la configuración.

* Se actualizó la sección [de descargas de agentes](/docs/new-relic-solutions/get-started/networks/#agent-download) para adaptar a la configuración de IP modificada de New Relic.

* Se actualizó la compatibilidad del agente PHP y los requisitos para [Amazon DynamoDB](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/#databases).

* Se agregó Windows Server 2025 como sistema operativo compatible para el [agente de infraestructura](/docs/infrastructure/infrastructure-agent/requirements-infrastructure-agent/).

* Se actualizó el [reporte de compatibilidad del agente Node.js.](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent)
