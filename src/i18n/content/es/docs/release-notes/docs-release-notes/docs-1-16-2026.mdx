### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Agente Browser v1.308.0](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.308.0):

    * Se agregó soporte para etiquetas MFE con la API de registro.
    * Se agregó el atributo child.id para facilitar la vinculación cruzada de MFE.
    * Se agregó la API de cancelación de registro para el soporte de micro front-end.
    * Se agregó el atributo parent.type para facilitar la síntesis de relaciones.
    * Se agregó el motivo de cierre predeterminado para los eventos de WebSocket.
    * Se aplicaron valores no nulos en campos numéricos para eventos de WebSocket.
    * Se limpiaron las URL de WebSocket eliminando los fragmentos hash y los parámetros de consulta.

  * [Agente de infraestructura v1.71.4](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1714):

    * Se corrigió la visualización de cero en transmitBytesPerSecond durante el tráfico alto en Windows.

  * [Tiempo de ejecución de la API de Node Synthetics rc1.7](/docs/release-notes/synthetics-release-notes/node-api-runtime-release-notes/node-api-runtime-rc1.7):

    * Se corrigió la vulnerabilidad de seguridad CVE-2025-15284.

  * [Synthetics Node browser runtime rc1.7](/docs/release-notes/synthetics-release-notes/node-browser-runtime-release-notes/node-browser-runtime-rc1.7):

    * Se corrigió la vulnerabilidad de seguridad CVE-2025-15284.

  * [Agente Ruby v10.0.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-10-0-0):

    * Se eliminó el soporte para Ruby 2.4 y 2.5.
    * Se eliminó la funcionalidad de rastreo entre aplicaciones.
    * Se renombraron las métricas de ActiveJob para incluir los nombres de las clases de trabajo.
    * Se renombró el comando bin/newrelic a bin/newrelic\_rpm.
    * Se eliminó el comando CLI de implementaciones de newrelic.
    * Se agregó el logger como una dependencia.
    * Se agregó la opción de configuración de lista de permitidos de notificación de Active Support.
    * Se agregaron opciones de configuración de muestreo raíz.
    * Se agregaron opciones de muestreo basadas en la relación de ID de rastreo.
    * Se corrigió NoMethodError en la detección de utilización de GCP.

  * [Agente Java v9.0.0](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-900):

    * Se agregó la configuración para eliminar los archivos jar temporales del agente obsoletos al inicio.
    * Se agregó la opción de inclusión para instrumentar aplicaciones de forma selectiva.
    * Se corrigió IllegalAccessError en la instrumentación de AWS S3.
    * Se corrigió la versión de instrumentación mal etiquetada para logback-classic-1.5.20.
    * Se corrigió el mensaje de logging para el valor máximo del atributo.
    * Se corrigieron las prioridades del muestreador estilo Otel.

  * [.NET agent v10.48.0](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-48-0):

    * Se permitió a los usuarios configurar nombres de procesos que no deben instrumentarse.
    * Se aumentó la longitud máxima del atributo de intervalo db.statement a 4096 bytes.
    * Se corrigió el error de conexión del agente cuando el logging de depuración está habilitado.
    * Se restauraron las propiedades de ocurrencias mínimas/máximas en el elemento de configuración distributedTracing.

  * [Agente PHP v12.4.0.29](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-12-4-0-29):

    * Se agregó soporte para PHP 8.5.
    * Se solucionaron las advertencias de desaprobación en Symfony 7.4 y 8.0.

  * [Agente Node.js v13.9.1](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-9-1):

    * Se actualizó getHostnameSafe para invalidar la caché al asignar el host según el ID de Cloud Run de GCP.
