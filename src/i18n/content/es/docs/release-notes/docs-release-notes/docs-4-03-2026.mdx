### Nuevos documentos

* Se agregó la [API de control de flotas](/docs/new-relic-control/fleet-control/overview/) para administrar las configuraciones del agente de infraestructura a escala en toda su flota.
* Se agregó la [Guía de transición a Chrome Latest para monitores sintéticos](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-browser-monitors-to-chrome-latest/) para actualizar los monitores sintéticos a la versión Chrome Latest y Node.js 22.
* Se agregó el [Boletín de seguridad NR26-04](/docs/security/new-relic-security/security-bulletins/security-bulletin-nr26-04) con procedimientos para abordar la vulnerabilidad crítica use-after-free de Google Chrome (CVE-2026-5281) que afecta a Synthetic Monitoring.
* Se agregaron [acciones de New Relic Agent](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent) para ejecutar agentes configurados en Agentic Platform mediante la automatización de flujos de trabajo.

### Cambios importantes

* Se actualizó la [documentación de AWS Lambda y OpenTelemetry](/docs/serverless-function-monitoring/aws-lambda-monitoring/opentelemetry/lambda-opentelemetry-troubleshooting/) con una sección completa sobre la solución de problemas de OpenTelemetry Lambda que abarca problemas con las métricas de CloudWatch y los ID de solicitud.

### Cambios menores

* Se actualizaron las [notas de la versión del agente de Browser](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/) con información de compatibilidad para v1.312.0 y v1.312.1.
* Se actualizó la [Descripción general del seguimiento de cambios](/docs/change-tracking/overview/) con la aclaración de que los cambios se pueden asociar a cualquier entidad en la plataforma de New Relic.
* Se actualizó el [reporte de compatibilidad del agente de Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/) con versiones actualizadas de los paquetes.
* Se actualizaron los [requisitos de compatibilidad del agente .NET](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) con las versiones compatibles verificadas más recientes para varios paquetes, incluidos `Microsoft.Azure.Cosmos`, `Microsoft.Data.SqlClient`, `MongoDB.Driver` y otros.
* Se actualizaron las [notas de la versión del agente de .NET](/docs/release-notes/agent-release-notes/net-release-notes/) con las correcciones de categorización faltantes.
* Estructura de navegación actualizada para mejorar la experiencia del usuario en todas las secciones de la documentación.
* Se actualizó la [documentación de repetición de sesiones móviles](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/) con la funcionalidad de vista de bloqueo para `NRConditionalMaskView` y una guía completa sobre enmascaramiento frente a bloqueo.
* Se actualizó la [documentación de retención de datos](/docs/data-apis/manage-data/manage-data-retention/) con la gestión de retención a nivel de organización y las explicaciones de las pestañas Cuenta vs. Organización.
* Se actualizó la [documentación de los módulos de extensión del agente de Java](/docs/apm/agents/java-agent/instrumentation/extension-additional-instrumentation-modules/) al eliminar el módulo incubador obsoleto HikariCP 2.4.0, ya que ahora se incluye en el agente.
* Se actualizó la [documentación de integración de conocimiento](/docs/agentic-ai/knowledge-integration/overview/) con nombres de permisos actualizados, cambiando del rol &quot;Org Product Admin&quot; a &quot;Organization Manager&quot;.
* Se actualizó el diccionario de atributos con el nuevo tipo de evento `NrComputeUsage` y las definiciones de atributos asociadas.
* Se actualizó la [documentación de gestión de ingesta de datos](/docs/data-apis/manage-data/manage-data-coming-new-relic/) con capacidades de pronóstico de uso de datos y funcionalidad de predicción de uso para los clientes de Advanced Compute.
* Se actualizó la [documentación de la integración de Kubernetes](/docs/kubernetes-pixie/kubernetes-integration/get-started/kubernetes-integration-compatibility-requirements/) con soporte para Oracle Kubernetes Engine (OKE) y una guía de configuración para la conexión automática de APM.
* Se actualizó la [documentación de seguridad de Java 2 del agente de Java](/docs/apm/agents/java-agent/additional-installation/install-java-agent-java-2-security/) con soporte para Solr v9 y una guía de configuración del administrador de seguridad.
* Se actualizó el [soporte de SCIM para la administración automatizada de usuarios](/docs/accounts/accounts/automated-user-management/scim-support-automated-user-management/) con compatibilidad para el campo `title` para capturar la información del cargo.

### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Agente de Node.js v13.15.0, v13.16.0, v13.17.0, v13.18.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/):

    * Stream de servidor grpc instrumentado para vincular el contexto asíncrono a los manejadores de eventos del stream.
    * Se refactorizaron múltiples módulos de instrumentación para usar el manejo de eventos basado en suscriptores.
    * Se actualizaron los suscriptores de Langchain core para instrumentar correctamente los métodos de clase de herramientas.
    * Compatibilidad mejorada con paquetes de npm y frameworks más recientes.

  * [Agente de Go v3.43.0](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-3-43-0):

    * Se agregó soporte de UnWrap para el escritor de respuestas de reemplazo y soporte para URL de funciones Lambda.
    * Integración mejorada de Opensearch/Elasticsearch con atributos de span adicionales.
    * Se agregó soporte para cláusulas `WITH` en el análisis de SQL y se corrigieron problemas de pruebas HTTP.
    * Actualizaciones de seguridad que incluyen la actualización de la versión de `nrgrpc` y parches de seguridad de dependencias.

  * [Agente de PHP v12.6.0.34](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-12-6-0-34):

    * Se agregó soporte para Laravel 13 y detección de PHP Fiber para el control de la instrumentación.
    * Se actualizó la seguridad con la versión 1.26.1 de Go y se rotó la clave de firma de RPM.
    * Se corrigió la instalación de Debian/Ubuntu y se mejoró la lógica de análisis de SQL.

  * [Agente de infraestructura v1.72.7, v1.72.8, v1.72.9, v1.73.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/):

    * Se agregó soporte completo para cuentas de servicio con privilegios mínimos en Windows.
    * Instalador mejorado para preservar las cuentas de servicio existentes durante las actualizaciones.
    * Seguridad mejorada con soporte para cuentas de servicio administradas de grupo (gMSA).
    * Se agregaron el logging de depuración y las funciones de seguridad de reversión para las instalaciones.

  * [Aplicación móvil de New Relic para Android v5.41.5](/docs/release-notes/mobile-apps-release-notes/new-relic-android-release-notes/new-relic-android-5415):

    * Correcciones de errores y mejoras.

  * [Extensión de Lambda v2.4.6](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.6):

    * Se agregaron variables de entorno para la configuración del proxy y el control de logs.
    * Se corrigió la clasificación de logs JSON y se mejoró el manejo del ID de solicitud.
    * Generación mejorada de números aleatorios en `TraceIDGenerator`.

  * [NRDOT Collector v1.12.0](/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01):

    * Se actualizó el core beta de OpenTelemetry a v0.148.0 para mejorar el rendimiento.
    * Se agregó spanmetricsconnector a las funciones experimentales.
    * Se corrigieron vulnerabilidades de seguridad, incluidas CVE-2026-33186 y CVE-2026-34040.
    * Se rotó la clave pública de firma GPG para mejorar la seguridad.
