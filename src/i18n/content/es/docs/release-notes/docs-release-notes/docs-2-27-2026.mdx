### Nuevos documentos

* Se agregó [OpenTelemetry Collector Observability](/docs/opentelemetry/collector-observability/collector-observability) para proporcionar una guía integral de monitoreo para el rendimiento y las métricas del collector.
* Se agregó el conjunto de documentación de [workloads inteligentes](/docs/new-relic-solutions/new-relic-one/workloads/workloads-overview) para brindar orientación sobre la creación de cargas de trabajo impulsadas por IA y la gestión de KPI, lo que incluye la [creación de workloads inteligentes](/docs/new-relic-solutions/new-relic-one/workloads/create-intelligent-workload), [información de IA](/docs/new-relic-solutions/new-relic-one/workloads/advanced-workload-ai-insights), [gestión de KPI](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-kpis) y [funciones de la interfaz de usuario](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-ui).
* Se agregó [Soporte para la API de OpenTelemetry](/docs/apm/agents/manage-apm-agents/opentelemetry-api-support) para brindar una guía integral para la instrumentación independiente del proveedor en todos los agentes APM de New Relic.

### Cambios importantes

* [Workflow Automation](/docs/workflow-automation/introduction-to-workflow) ya está disponible de forma general.
* [Detección de valores atípicos](/docs/alerts/create-alert/set-thresholds/outlier-detection) ya está disponible de forma general.
* Se agregó documentación completa de [soporte para Windows](/docs/new-relic-control/agent-control/overview) para Agent Control con orientación sobre instalación, configuración, monitoreo y resolución de problemas.

### Cambios menores

* Documentación mejorada de [monitoreo de Kafka con OpenTelemetry](/docs/opentelemetry/integrations/kafka/self-hosted) con diagramas de arquitectura y guía de solución de problemas ampliada.
* Se actualizaron los [requisitos de compatibilidad del agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) con los últimos rangos de versiones compatibles para AWS SDK, LangChain, OpenAI y otros paquetes.
* Se actualizaron los [requisitos de compatibilidad del agente Java](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent) con soporte para WebLogic extendido a la versión 14.1.x.

### Publicaciones de novedades

* Se agregó una [publicación de novedades para New Relic Lens](/whats-new/2026/02/whats-new-02-25-nr-lens/) para anunciar la consulta de fuentes de datos externas sin requisitos de ingesta.
* Se agregó la [publicación de Novedades para la disponibilidad general de la Automatización de flujos de trabajo](/whats-new/2026/02/whats-new-02-24-workflow-automation/) para anunciar flujos de trabajo de automatización sin código con integraciones con AWS, Azure, Slack y PagerDuty.
* Se agregó la [publicación de Novedades para Dynamic Runbooks with Notebook](/whats-new/2026/02/%20whats-new-02-25-dynamic-runbooks/) para anunciar que la función entra en Vista previa pública.
* Se agregó la [publicación de Novedades para la página de inicio de New Relic](/whats-new/2026/02/whats-new-02-24-new-home-page/) para anunciar una plataforma de lanzamiento personalizada que ofrece una vista seleccionada de los servicios y datos más relevantes para su flujo de trabajo diario (Vista previa pública).

### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Agente Ruby v10.2.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-10-2-0):

    * Se introdujo el Agente híbrido para el soporte de rastreo de OpenTelemetry con compatibilidad de API independiente del proveedor.
    * Se agregó la configuración de truncamiento de rastreo del recolector de errores y la instrumentación de la gema Logging.
    * Detección de proveedores de nube mejorada con opción de procesamiento paralelo.

  * [Agente Java v9.1.0](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-910):

    * Se agregó una implementación integral del Agente híbrido con compatibilidad con la API de OpenTelemetry para rastreo, métricas y logs.
    * Soporte mejorado para corrutinas de Kotlin y corrección de problemas de propagación de trazas W3C.
    * Se agregaron acciones de metadatos del agente y correcciones de configuración de múltiples hosts.

  * [Agente de Node.js v13.14.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-14-0):

    * Se agregó la ingesta automática de logs desde Azure Functions Apps.
    * Se implementó soporte para causas de error y controles de retraso/duración del generador de perfiles.
    * Se actualizó la integración de Google Gen AI para la compatibilidad con el método `embedContentInternal`.

  * [Agente Android v7.7.0](/docs/release-notes/mobile-release-notes/android-release-notes/android-770):

    * Se agregó soporte para Android Gradle Plugin 9.
    * Reproducción de sesión móvil mejorada con muestreo de errores y nuevas API de `pauseReplay`/`recordReplay`.
    * Se implementaron múltiples mejoras de rendimiento para la funcionalidad de reproducción de sesiones.

  * [Agente de infraestructura v1.72.6](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1726):

    * Se actualizó la versión de fluent-bit a la 4.2.2 para entornos Windows.

  * [Agente de Browser v1.310.1](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.310.1):

    * Entrada de LCP mejorada con valores de atribución de respaldo para mejorar los reportes de Core Web Vitals.
