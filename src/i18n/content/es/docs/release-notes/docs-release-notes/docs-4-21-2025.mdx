### Cambios importantes

* Se agregó `sampler.remote_parent_sampled` en [la configuración del agente Java : Archivo de configuración](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file/#dt-sampler-remote-parent-sampled) para definir cómo el agente debe manejar los intervalos de ejemplificación cuando se muestreó su intervalo principal de una entidad ascendente.
* Actualice el [aprovisionamiento de SCIM](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim) para especificar que ahora requiere un <DNT>**Paid Edition**</DNT> para la administración automatizada de usuarios en New Relic.
* Se actualizó el documento [de API de reglas de nube](/docs/new-relic-control/pipeline-control/cloud-rules-api/#drop-attributes-on-dimensional-metric-rollups) para el control de canalización para incluir información sobre cómo eliminar atributos de los conjuntos de métricas dimensionales.
* Se agregó una consulta para rastrear eventos de inicio y cierre de sesión en la página [Seguimiento de cambios de cuenta](/docs/accounts/accounts/account-maintenance/query-account-audit-logs-nrauditevent/#user-login) .

### Cambios menores

* Se actualizó la [versión de compatibilidad del agente .Net](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/#framework).

* Se agregaron configuraciones de procesamiento por lotes y compresión para el seguimiento infinito al documento [de configuración del agente Node.js.](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration/#infinite-tracing-batching)
