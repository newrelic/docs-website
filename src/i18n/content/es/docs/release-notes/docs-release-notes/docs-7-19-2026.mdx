## Nuevos documentos

* Se agregó la funcionalidad de la [API de traducción de dashboard de Grafana](/docs/infrastructure/prometheus-integrations/view-query-data/grafana) para admitir los flujos de trabajo de migración de dashboard.
* Se lanzó la **región de datos de New Relic en Japón** y se actualizaron varias páginas de documentación para reflejar la nueva región y sus capacidades en todos los productos.

### Cambios importantes

* **Se cambió el nombre del agente de ingeniería de fiabilidad del sitio (SRE) a New Relic Autopilot** en toda la documentación con terminología y referencias actualizadas en toda la plataforma.
* Se actualizó la documentación de [Gestión de vulnerabilidades](/docs/vulnerability-management/advanced/query-examples) para eliminar las funciones NRQL no válidas y corregir los patrones de consulta para una mejor precisión.
* **Se eliminó la documentación del complemento de Gestión de vulnerabilidades**, ya que la característica se ha integrado en las capacidades principales de la plataforma.
* Se actualizó la [biblioteca de plantillas de Scorecards](/docs/service-architecture-intelligence/scorecards/templates) con una cobertura integral de todas las plantillas disponibles y una estructura de organización mejorada.
* Se actualizó la [matriz de compatibilidad](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) de Node.js para incluir las últimas versiones compatibles y aclarar la compatibilidad con varias versiones de Node.js.
* Se actualizó el documento del [servidor MCP de New Relic](/docs/agentic-ai/mcp/overview) para que cumpla con la normativa HIPAA.

### Cambios menores

* Se agregaron acciones de [escalar un grupo de agentes de AKS](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-aks-scalepool) para administrar sus clústeres de AKS directamente desde sus flujos de trabajo.
* Se actualizó la configuración de la [integración de Terraform](/docs/infrastructure/amazon-integrations/aws-integration-for-metrics/terraform).
* Se mejoró la [integración de Microsoft SQL Server](/docs/opentelemetry/database/mssql/introduction) con enlaces de configuración actualizados y una guía de configuración mejorada.
* Se corrigió la documentación del [extremo de integración de HAProxy](/docs/infrastructure/host-integrations/host-integrations-list/haproxy-monitoring-integration) para mejorar la precisión y la claridad.
* Se agregó compatibilidad integral con [certificados personalizados para ubicaciones privadas](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-configuration/#custom-ca-certificates) con instrucciones de configuración detalladas y opciones de configuración para un monitoreo seguro.
* Se actualizó la [documentación del programa para estudiantes](/docs/accounts/accounts-billing/account-setup/student-program) con los detalles del programa.
