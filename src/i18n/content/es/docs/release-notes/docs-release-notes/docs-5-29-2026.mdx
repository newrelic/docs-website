### Nuevos documentos

* Se agregó la [referencia de atributos de eBPF](/docs/ebpf/attributes-reference) para proporcionar documentación completa sobre todos los atributos recopilados por el agente eBPF de New Relic.
* Se agregó [Instalar el gateway sin Flux](/docs/new-relic-control/pipeline-control/gateway/install-gateway-without-flux) para proporcionar orientación para desplegar el gateway de Control de pipelines sin automatización completa, lo que permite a los usuarios gestionar la infraestructura del gateway utilizando herramientas de Kubernetes mientras siguen usando la UI para la configuración del Pipeline.

### Cambios importantes

* Se actualizó la [documentación del Gateway de Control de pipelines](/docs/new-relic-control/pipeline-control/gateway/overview) con soporte para el modo de instalación Fluxless y una guía de configuración mejorada.
* Se actualizó la documentación de [configuración del Control del agente](/docs/new-relic-control/agent-control/setup) con instrucciones de instalación de Fluxless para desplegar el Control del agente sin la dependencia de Flux CD.
* Se actualizó la documentación de [monitoreo de ECS EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) y [monitoreo de ECS Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) con ejemplos mejorados de configuración de OpenTelemetry y opciones de despliegue.
* Se actualizó el asistente de instalación de la integración de MySQL y MariaDB con actualizaciones de compatibilidad con MariaDB, nuevas opciones de configuración y una guía mejorada de métricas de rendimiento de consultas.

### Cambios menores

* Se actualizó la documentación de [interacción de la API de Browser](/docs/browser/new-relic-browser/browser-apis/interaction) con nuevas opciones y descripciones detalladas de los valores de retorno para el seguimiento de la interacción de SPA.
* Se actualizó la documentación de [mediciones de la máquina virtual (VM) de Node.js](/docs/apm/agents/nodejs-agent/extend-your-instrumentation/nodejs-vm-measurements) para aclarar las diferencias entre los muestreadores predeterminados y nativos.
* Se actualizó [UI y tareas de administración de usuarios](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks) con los requisitos de validación del campo de nombre para los nombres de usuario.
* Se actualizó el [Resumen de la integración de conocimientos](/docs/agentic-ai/knowledge-integration/overview) para mejorar la navegación y aclarar las características de integración de la base de conocimientos.
* Se actualizó la documentación de [Instalar el agente de Node.js](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent) con instrucciones de instalación simplificadas.
* Se actualizó la [UI de APM de OpenTelemetry](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui) con el comportamiento de respaldo de `OtelMessagingConsumer1_24` cuando `messaging.operation` es `NULL`.
* Se eliminó el soporte obsoleto para logs de la instancia de RDS de [logs de AWS en contexto](/docs/logs/logs-context/aws-logs-in-context).
