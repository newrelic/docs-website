### Cambios importantes

* Se actualizaron las pautas para [degradar o cancelar una cuenta de New Relic](/docs/accounts/accounts-billing/account-setup/downgradecancel-account/#preparation-org-delete).
* Se mejoraron los [elementos de atributos](/docs/apm/agents/net-agent/configuration/net-agent-configuration/#agent-attributes) para la configuración del agente .NET.
* Se mejoró la [configuración del conector](/docs/agentic-ai/knowledge-integration/knowledge-connector/) para los conectores de New Relic AI Knowledge.

### Cambios menores

* Se agregaron funciones de matriz a la [referencia NRQL](/docs/nrql/nrql-syntax-clauses-functions/).
* Se actualizaron las [opciones de la línea de comando para nrdiag](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag/) introduciendo la opción `-k8s-namespace STRING` para especificar el namespace al ejecutar el comando `kubectl` .

### Notas de la versión

Mantener actualizado con nuestros últimos lanzamientos:

* [Agente Python v10.10.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-101000):

  * Se agregó soporte asincrónico para Elasticsearch.
  * Se habilitó el análisis de la configuración de GraphQL.
  * Se solucionó un fallo en Kombu cuando se ejecutaba con Sentry.

* [Integración Kubernetes v3.39.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-39-0):

  * Se actualizó `newrelic-infrastructure` a la versión 3.44.0
  * Se actualizó el `nri-bundle` a la versión 5.0.122

* [Aplicación móvil para iOS v6.9.5](/docs/release-notes/mobile-apps-release-notes/new-relic-ios-release-notes/new-relic-ios-6095):

  * Equipos migrados de EXT a NGEP.
  * Se agregó soporte multi-equipo.
  * Se introdujo la funcionalidad de búsqueda de equipo en la pantalla de detalles del equipo.

* [Aplicación móvil para iOS v6.9.6](/docs/release-notes/mobile-apps-release-notes/new-relic-ios-release-notes/new-relic-ios-6096):

  * Se corrigió la funcionalidad de comentarios en la aplicación.

* [Agente de Browser v1.289.0](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.289.0/#remove-agent-config-cache-and-getterssetters):

  * Se eliminó el caché de configuración del agente y los captadores/configuradores asociados.

* [Agente Node.js v12.18.2](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-12-18-2):

  * Se actualizó `createSegment` para evitar agregar segmentos al trazado cuando se excede `max_trace_segments` .
  * Se revisó el reporte de compatibilidad.

* [Agente de Node.js v12.18.1](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-12-18-1):

  * Se agregó código defensivo para calcular el tiempo exclusivo en un segmento cuando el segmento no se puede ubicar en el árbol de la traza de la transacción.
  * Se mejoró la forma en que el propagador traza del puente OpenTelemetry asigna los encabezados `traceparent` y `tracestate` .
  * Se actualizó el reporte de compatibilidad.
