### Nuevos documentos

* Se agregó el conjunto de documentación de [Mobile Session Replay](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/introduction/), ahora disponible de forma general, con guías completas que cubren la instalación, la configuración de privacidad, las características avanzadas, la visualización de repeticiones y la resolución de problemas para Android e iOS.
* Se agregó [Actualizar los monitores de API con scripts a Node.js 22](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-api-monitors-to-node-22/) para proporcionar instrucciones de migración paso a paso para pasar los monitores de API con scripts de Node.js 16 a Node.js 22.

### Cambios importantes

* Se actualizó la documentación de Inteligencia para costos en la nube para disponibilidad general, incluidas las nuevas páginas de [Precios de CCI](/docs/cci/cci-pricing/), [Condiciones de alerta para Inteligencia para costos en la nube](/docs/cci/configure-cci/cci-alerts/) y [Costos de IA](/docs/cci/financial-intelligence/ai-costs/), además de contenido ampliado de configuración de integraciones y preguntas frecuentes.
* Se reestructuró la documentación de [New Relic Teams](/docs/service-architecture-intelligence/teams/teams/) con una nueva guía de [Configurar la propiedad de la entidad](/docs/service-architecture-intelligence/teams/team-ownership/) y contenido actualizado de [Administrar equipos](/docs/service-architecture-intelligence/teams/manage-teams/) para mejorar la navegación y la claridad.
* Se actualizó la documentación del agente eBPF para la instalación en [Kubernetes](/docs/ebpf/k8s-installation/) y [Linux](/docs/ebpf/linux-installation/) con una guía de migración de la configuración YAML y cambios en los campos introducidos en la v1.4.0.
* [Optimice el uso de Calcular (compute)](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-optimizer/) actualizado con orientación ampliada sobre el dashboard, nuevas capturas de pantalla y recomendaciones de optimización mejoradas.
* Se actualizó la [OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/) con una nueva sección sobre el monitoreo de errores en la experiencia de APM de OpenTelemetry.

### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Agente de Go v3.43.3](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-3-43-3/):

    * Se agregó la opción de configuración experimental `ConfigAppLogDecoratingWithinMessage` para permitir que el decorador de log local (Local Log Decorator) aparezca dentro de un campo de mensaje (solo se aplica a la integración `logcontext-v2/nrlogrus`).
    * Se aplicaron actualizaciones de seguridad de Dependabot para `nrawsbedrock`, `nrawssdk-v2` y `nrpgx5`.

  * [Agente Android v7.7.3](/docs/release-notes/mobile-release-notes/android-release-notes/android-773/):

    * Session Replay para Android ya está disponible de forma general.
    * Se agregó la capacidad de bloquear programáticamente vistas específicas de la captura de Session Replay.
    * El agente ahora crea automáticamente una nueva sesión después de 4 horas de uso continuo.
    * El reenvío de logs ahora coordina las decisiones de muestreo con Session Replay.
    * Se corrigieron múltiples problemas de manejo de sesiones, excepciones de stream, puntero nulo y renderizado de reproducción.

  * [Agente de React Native v1.8.2](/docs/release-notes/mobile-release-notes/react-native-release-notes/react-native-agent-182/):
