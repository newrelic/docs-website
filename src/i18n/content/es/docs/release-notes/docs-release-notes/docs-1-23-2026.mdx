### Cambios importantes

* Se actualizó la documentación de eBPF con [la nueva matriz de compatibilidad](/docs/ebpf/requirements), [la instalación del agente eBPF en Linux](/docs/ebpf/linux-installation) y [la instalación de Kubernetes](/docs/ebpf/k8s-installation) con cambios de configuración para la eliminación del cliente.
* Se actualizó la [documentación de control de acceso detallado para la gestión de secretos](/docs/apis/nerdgraph/examples/nerdgraph-api-secret-management-service) para mejorar el procedimiento de control de acceso a la API.
* Se actualizaron los [requisitos del agente Ruby y los frameworks compatibles](/docs/apm/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) con los cambios de la versión 10.x.
* Se agregó [documentación de integración de Nuxt.js](/docs/browser/browser-integrations/nuxt-integration/) con soporte actualizado para Nuxt 3+ y formato mejorado.

### Cambios menores

* Documentación mejorada de la [API notice\_error del agente Python](/docs/apm/agents/python-agent/python-agent-api/noticeerror-python-agent-api) con soporte para instancias de excepción.
* Se actualizó la [guía de introducción a los logs del navegador](/docs/browser/browser-monitoring/browser-pro-features/browser-logs/get-started/) con mejoras en la documentación de logging.
* Se corrigió la documentación del rastreo de sesión para la [exploración del ciclo de vida de la página web](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle).
* Se actualizaron los [requisitos de compatibilidad del agente Node.js](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) con el último informe de compatibilidad.
* Se actualizó la [configuración de NGINX OpenTelemetry](/docs/opentelemetry/nginx/nginx-otel-host) con una configuración stub\_status mejorada.

### Novedades

* [Operaciones de panel para usuarios de solo lectura](/whats-new/2026/01/whats-new-01-19-read-only-dashboards): los permisos de rol de solo lectura estándar para la gestión de paneles se limitarán al acceso de solo visualización a partir del 30 de abril de 2026.
