### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Extensión Lambda v2.4.0](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.0):

    * Reescritura completa de Go a Rust para mejorar el rendimiento y la seguridad de la memoria.
    * Se agregó soporte del modo APM para que la función Lambda reporte directamente a la plataforma New Relic APM.
    * Tiempos de arranque en frío reducidos entre un 20 y un 40% con la compilación de Rust optimizada.

  * [AgenteNode.js v13.8.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-8-0):

    * Se agregó soporte para la instrumentación @langchain/AWS.
    * Se agregó seguimiento de detalle parcial métrico.

  * [AgenteNode.js v13.7.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-7-0):

    * Se agregó soporte para enlaces span OTEL.
    * Se agregó la capacidad de configurar un muestreador basado en proporción.
    * Instrumentación koa actualizada para @koa/router\@15.0.0+.
    * Se corrigió la normalización del horario de OTEL.

  * [AgentePython v11.2.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-110200):

    * Se agregó soporte para la instrumentación de Strands.
    * Se agregó soporte para AWS Bedrock Converse Streaming.
    * Se agregó soporte para los métodos Redis hybrid\_search y msetex.
    * Se solucionó el problema de consumo del encabezado ASGI.

  * [agente de infraestructura v1.71.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1713):

    * Se actualizó la etiqueta de Alpine docker a v3.23.
    * Versión de go actualizada a 1.25.5.

  * [agente de infraestructura v1.71.2](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1712):

    * Versión actualizada de Fluent Bit con correcciones para CVE-2025-12972, CVE-2025-12970, CVE-2025-12978, CVE-2025-12977 y CVE-2025-12969.
    * Se eliminaron las compilaciones de Windows de 32 bits.
    * Se corrigieron los contadores de rendimiento de la CPU en sistemas que no estaban en inglés.

  * [Integración Kubernetes v3.51.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-51-0):

    * Integración actualizada con nuevas características y mejoras.
