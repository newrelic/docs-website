### Cambios importantes

* Se actualizó la documentación de la [integración de Microsoft SQL Server](/install/microsoft-sql) para agregar el Monitoreo de rendimiento de consultas (QPM) para los siguientes métodos de autenticación:

  * Dominio de Windows
  * Cuenta de servicio administrada de grupo (gMSA) de Windows

* Se actualizó la documentación de [monitoreo de Elasticsearch con OpenTelemetry](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) para mayor claridad y se agregó documentación completa de instalación de Kubernetes.

* Se actualizó [Dashboards con la API de NerdGraph](/docs/apis/nerdgraph/examples/nerdgraph-dashboards) con funciones de visibilidad de variables de plantilla.

* Se actualizó el [Esquema de definición de automatización de flujos de trabajo](/docs/workflow-automation/workflow-automation-apis/definition-schema) con extensas actualizaciones del esquema y ejemplos completos.

### Cambios menores

* Se actualizó la [Gestión del consumo](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/consumption-management) con funciones de pronóstico de uso de CCU y capacidades mejoradas de dashboard.
* Se actualizó [Ver los resultados del monitor de ping](/docs/synthetics/synthetic-monitoring/using-monitors/view-ping-monitor-results) con orientación y ejemplos mejorados.
* Se actualizó el diccionario de atributos con definiciones y descripciones de atributos adicionales para Browser.

### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Agente Browser v1.311.0](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.311.0):

    * Se eliminaron las referencias de ámbito por identificador de agente para simplificar la arquitectura del agente.
    * Se amplió la atribución de `pageUrl` a todos los nodos `PageViewTiming` para mejorar la consistencia de la SPA.
    * Lógica de tiempos de MFE mejorada para obtener información de tiempos más precisa.
    * Se exigió el uso de IDs de cadena para los nuevos registros de MFE y se corrigieron los problemas del reemplazador circular.

  * [Agente de iOS v7.7.0](/docs/release-notes/mobile-release-notes/ios-release-notes/ios-agent-770):

    * Se solucionó un problema de sobrecalentamiento causado por el logging excesivo en casos excepcionales.
    * Se resolvieron errores en los métodos `NewRelic.setMaxOfflineStorageSize` y `getAndClearFrames`.

  * [Agente de Python v12.1.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-120100):
