### Nuevos documentos

* Se agregaron los [flujos de trabajo de Jira de Security RX](/docs/vulnerability-management/workflows/jira) para proporcionar orientación sobre cómo crear y sincronizar automáticamente tickets de Jira para las vulnerabilidades de la aplicación detectadas.
* Se agregó una [acción de detención de máquina virtual de Azure](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-vm-stop) al catálogo de acciones para que pueda apagar las máquinas virtuales (VM) de Azure como parte de sus flujos de trabajo automatizados.
* Se agregaron detalles sobre [cómo guardar y administrar las preferencias del dashboard](/docs/streaming-video-&-ads/view-data-in-newrelic/user-preferences) para explicar cómo New Relic guarda y restaura la configuración del dashboard de Video Overview y Ad Overview entre sesiones.
* Se agregaron reglas de ofuscación relacionadas con el agente de Streaming Video &amp; Ads para [Android](/docs/streaming-video-&-ads/installation/android/obfuscation-rules), [navegador](/docs/streaming-video-&-ads/installation/browser/obfuscation-rules), [iOS](/docs/streaming-video-&-ads/installation/ios/obfuscation-rules) y [Roku](/docs/streaming-video-&-ads/installation/roku/obfuscation-rules) para brindar orientación sobre cómo enmascarar datos confidenciales con reglas basadas en expresiones regulares antes de la transmisión.

### Cambios importantes

* Se actualizó la [descripción general de integraciones de Security RX](/docs/vulnerability-management/getting-started/integrations/overview) y la [descripción general de aplicaciones](/docs/vulnerability-management/applications/overview) para establecer &apos;10.51.1&apos; como la versión base para la integración de .NET de Security RX.
* Se actualizó la documentación de CCI con [precios de ingesta de facturación por proveedor](/docs/cci/cci-pricing), [orientación sobre el alcance de la integración de cuenta principal frente a subcuenta](/docs/cci/overview/#choose-an-integration-scope) y [requisitos previos de configuración de Azure](/docs/cci/get-started/azure-cci).
