### Cambios importantes

* Se actualizó [Buscar y consultar datos de Kafka en New Relic](/docs/opentelemetry/integrations/kafka/find-and-query-data) con documentación mejorada y capturas de pantalla del dashboard actualizadas.
* Documentación mejorada de [Monitorear Kafka en Kubernetes (Strimzi) con OpenTelemetry](/docs/opentelemetry/integrations/kafka/kubernetes-strimzi) con ejemplos de configuración ampliados.
* Documentación mejorada de [Monitoreo de Kafka autoalojado con OpenTelemetry](/docs/opentelemetry/integrations/kafka/self-hosted) con instrucciones de configuración mejoradas y guía del explorador de entidades.

### Cambios menores

* Se actualizó la consulta start-schedule en la documentación de automatización de flujos de trabajo.
* Se actualizó el reporte de compatibilidad del agente de Node.js.
* Números de versión actualizados para los agentes de streaming de medios (Android, Browser e iOS).
