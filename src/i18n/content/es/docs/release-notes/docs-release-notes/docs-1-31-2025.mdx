### Nuevo documento

* [El registro automático](/docs/browser/browser-monitoring/browser-pro-features/auto-logging) mejora la observabilidad del frontend para los agentes de browser Pro y Pro+SPA, lo que permite la recopilación automática de mensajes de registro de `console` de nivel `WARN`.

### Cambios importantes

* Se actualizaron los reportes de compatibilidad para [el agente Node.js.](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)
* Se agregó un método en [el monitoreo del estado del contenedor Docker de KTranslate](/docs/network-performance-monitoring/advanced/ktranslate-container-health) para verificar el log localmente desde el paquete de Linux.
* [Bloques de red](/docs/new-relic-solutions/get-started/networks/) actualizados con nuevos rangos de IP efectivos para mejorar la conectividad y la seguridad.
* Se actualizó la [versión del protocolo OTLP](/docs/opentelemetry/best-practices/opentelemetry-otlp/#otlp-version-support) que usa New Relic.
* [APM actualizado: Reporte de eventos personalizados y atributos](/docs/data-apis/custom-data/custom-events/apm-report-custom-events-attributes/#php-att) para incluir la capacidad de aumentar el número máximo de eventos personalizados en el agente PHP de 30 000 a 100 000 por minuto.
* Se agregó un llamado a la desuso y finalización del soporte para [el paquete NuGet de NewRelic para Xamarin](/docs/mobile-monitoring/new-relic-mobile-xamarin/monitor-your-xamarin-application/).

### Cambio menor

* Se actualizaron las versiones de Python compatibles.
* Los valores de atributo especificados en la [configuración del perfil de usuario de Okta](/docs/accounts/accounts/automated-user-management/okta-scimsso-application-configuration/) ahora distinguen entre mayúsculas y minúsculas.
* Los valores [de tipo de usuario de la API de SCIM](/docs/accounts/accounts/automated-user-management/tutorial-manage-users-groups-scim/#manage-user-type) ahora distinguen entre mayúsculas y minúsculas.
* Se actualizaron las versiones del agente PHP compatibles.
