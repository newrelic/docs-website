### Cambios importantes

* Se actualizó [Instalar New Relic para Go](/docs/apm/agents/go-agent/installation/install-new-relic-go) con una guía de instalación completa paso a paso y ejemplos para patrones comunes de aplicaciones Go.
* Se agregó [Configurar e instalar Agent Control](/docs/new-relic-control/agent-control/setup) para proporcionar una guía completa de instalación y configuración para clústeres de Kubernetes y hosts Linux.
* Se mejoró la documentación de la API [setUserId](/docs/browser/new-relic-browser/browser-apis/setuserid) con la funcionalidad de restablecimiento de sesión y el comportamiento actualizado de seguimiento de sesión.

### Cambios menores

* Se agregó información de compatibilidad del agente PHP para [Requisitos de compatibilidad del agente PHP](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements).
* Se actualizó la documentación de [instalación de SPM](/docs/mobile-monitoring/new-relic-mobile-ios/installation/spm-installation) con una guía mejorada.
* Se mejoró la documentación de [instalación de CocoaPods](/docs/mobile-monitoring/new-relic-mobile-ios/installation/cocoapods-installation) con instrucciones mejoradas.
* Se actualizó la documentación de [Filtrar eventos de solicitud AJAX](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events) con soporte de comodines para la lista de denegación.
* Se actualizó [Actualizar a la experiencia APM](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/upgrade-to-apm-experience) para la monitorización de funciones Lambda.
