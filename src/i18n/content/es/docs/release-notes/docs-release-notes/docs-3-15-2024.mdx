### Cambios importantes

* Se limpió y simplificó la navegación izquierda en toda nuestra documentación de NRQL.

* Revisamos y reescribimos la documentación de integración en el host para [MongoDB](/install/mongodb/), [MySQL](/install/mysql/) y [PostgreSQL](/install/postgresql/) para facilitar la instalación y asegurar que la documentación sea precisa y esté actualizada.

* Se agregaron métricas dimensionales faltantes a las [métricas de integraciones de AWS](/docs/infrastructure/amazon-integrations/get-started/aws-integrations-metrics).

* Se mejoró el estilo de varios componentes del sitio de documentación:

  * Se actualizaron los colores para mejorar la legibilidad y la consistencia con el producto.
  * Se eliminó el componente &quot;Ir al producto&quot; de la navegación izquierda.
  * Se eliminó el color de fondo del hexágono de nuestro logotipo de documentación.
  * Se agregó un formulario de comentarios en la navegación derecha a nuestra documentación de instalación interactiva.
  * Se eliminaron los banners de registro de varios documentos.
  * Se aumentó el tamaño del texto del popover.
