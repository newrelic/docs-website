### Notas de la versión

* Mantener actualizado con nuestros últimos lanzamientos:

  * [Integración de Kubernetes v3.57.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-57-0):
    * Se actualizaron los componentes de integración y las dependencias.

  * [Aplicación móvil Android v5.41.1](/docs/release-notes/mobile-apps-release-notes/new-relic-android-release-notes/new-relic-android-5411):
    * Se aplicaron correcciones de errores y mejoras.

  * [Agente Android v7.7.1](/docs/release-notes/mobile-release-notes/android-release-notes/android-771):

    * Se utilizó SecureRandom() para generar identificadores únicos.
    * Se corrigió un posible ANR en el hilo principal durante el muestreo de carga pesada.
    * Se agregó el manejo de excepciones para versiones inferiores del SDK.

  * [Job Manager v512](/docs/release-notes/synthetics-release-notes/job-manager-release-notes/job-manager-release-512):
    * Se corrigieron vulnerabilidades del sistema operativo subyacente y de Java (CVE-2025-11143, CVE-2026-1605).

  * [Ping Runtime v1.63.0](/docs/release-notes/synthetics-release-notes/ping-runtime-release-notes/ping-runtime-1.63.0):
    * Se corrigieron vulnerabilidades del sistema operativo subyacente y de Java (CVE-2025-11143, CVE-2026-1605).

  * [Agente .NET v10.50.0](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-50-0):

    * Se agregó soporte de arreglos para atributos personalizados en la API del agente .NET.
    * Se agregó soporte de agente híbrido para grpc-dotnet.
    * Se agregó el monitoreo de LLM para Microsoft.Extensions.AI mediante el puente de OpenTelemetry.
    * Se corrigió la instrumentación de la conexión websocket de Blazor/SignalR.

  * [Agente Node.js v13.17.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-17-0):

    * Se agregó soporte para Continuous Profiling, incluidos los perfiladores de CPU y heap.
    * Se agregó configuración del lado del servidor para profiling.enabled.
    * Se agregaron métricas de compatibilidad para el perfilamiento.
    * Se deshabilitó el perfilado en el modo serverless y en el modo de alta seguridad.
    * Se refactorizó la instrumentación de winston para suscribirse a eventos.

  * [Job Manager v513](/docs/release-notes/synthetics-release-notes/job-manager-release-notes/job-manager-release-513):
    * Se corrigió una NullPointerException en el manejo de etiquetas de contenedor al iniciar Synthetics Job Manager con Podman.

  * [Node API runtime rc1.13](/docs/release-notes/synthetics-release-notes/node-api-runtime-release-notes/node-api-runtime-rc1.13):
    * Se corrigieron múltiples vulnerabilidades de seguridad en el sistema operativo Ubuntu, paquetes del kernel y paquetes de aplicaciones Node.js.

  * [Runtime de navegador Node rc1.13](/docs/release-notes/synthetics-release-notes/node-browser-runtime-release-notes/node-browser-runtime-rc1.13):
    * Se corrigieron múltiples vulnerabilidades de seguridad en el sistema operativo Ubuntu, paquetes del kernel y paquetes de aplicaciones Node.js.
