## Actualizado

* Se agregó el atributo `contentBitrate` a los eventos de video. Esto utiliza `segmentationBitrate` si está disponible y recurre a `streamBitrate` cuando sea necesario.
* Se agregó el atributo `adBitrate` a los eventos de anuncios. La tasa de bits del anuncio ahora se extrae utilizando múltiples métodos y se estandariza a bps (bits por segundo).
* Se agregó el atributo `timeSinceLastError` a los eventos de error, que reporta el tiempo transcurrido desde el evento de error anterior.
