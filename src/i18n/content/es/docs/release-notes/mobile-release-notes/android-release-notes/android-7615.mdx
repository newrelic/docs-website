## Correcciones

* Se cambió el intervalo predeterminado para enviar datos de eventos de 10 minutos a 1 minuto.
* Se solucionó un problema por el cual los archivos de mapeo no se enviaban en DexGuard 9 con diferentes configuraciones de sabor.
* Se corrigieron múltiples fallos relacionados con la reproducción de sesiones móviles.

## Declaración de apoyo

Recomendamos actualizar el agente al menos cada tres meses. Encuentre políticas y fechas específicas para el soporte de agentes Android en la [Política de EOL de agentes de monitoreo de celulares](/docs/mobile-monitoring/new-relic-mobile/get-started/mobile-agents-eol-policy/#android-eol).
