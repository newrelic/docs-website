## Anuncios

Session Replay para Android ha pasado de vista previa a disponibilidad general. Obtenga visibilidad completa de la experiencia móvil de su usuario con reproducciones visuales fieles de sus sesiones. Obtenga más información sobre Session Replay.

## Nueva característica

* Se agregó la capacidad de bloquear vistas específicas de forma programática para que no se capturen durante Session Replay, lo que brinda un control detallado sobre lo que se graba más allá de las reglas de enmascaramiento predeterminadas.
* El agente ahora crea automáticamente una nueva sesión después de 4 horas de uso continuo, lo que mejora el seguimiento de sesiones de larga duración.
* El reenvío de logs ahora coordina sus decisiones de muestreo con Session Replay, asegurando que se capturen los logs para las sesiones que se están reproduciendo.

## Corrección de errores

* Se resolvió un problema en el que el agente intentó reenviar logs sin una conexión de red activa, causando errores innecesarios en dispositivos sin conexión.
* Se corrigió una RuntimeException que podía ocurrir al leer o cerrar flujos de HttpsURLConnection.
* Se solucionó una excepción de puntero nulo que podía ocurrir durante el manejo de sesiones.
* Se solucionó un problema de sincronización en el que la rotación de sesión podía ocurrir antes de que se completara el ciclo de recolección, lo que podría causar pérdida de datos durante la transición.
* Se resolvió un problema en el que el ID de sesión podía ser nulo durante la rotación de la sesión, causando que se asignara un atributo incorrectamente a los datos.
* Se solucionaron varios problemas de precisión de renderizado en las capturas de Session Replay.
* Se corrigió un problema en el que la configuración de enmascaramiento local no se aplicaba correctamente a los elementos de texto durante la captura de Session Replay.

Declaración de apoyo

Recomendamos actualizar el agente al menos cada tres meses. Encuentre las políticas y fechas específicas de soporte del agente de Android en la Política de EOL de agentes de monitoreo de móviles.
