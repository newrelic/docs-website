### Nueva característica

Funcionalidad de almacenamiento sin conexión para Mobile Session Replay.

### Corregido en esta versión

Se reemplazó el LCG sembrado por timestamp en IGuidGenerator con arc4random\_buf.

Se agregó la detección de la región de la UE a la carga de RN Sourcemap.

Se bloqueó la recursión para el swizzle de interacción predeterminado del estilo MAUI.

Se cambió el nombre de appVersionId a appVersion en la carga del mapa de origen de RN.

Se corrigió el bucle infinito en EventDeserializer que congelaba el agente en el lanzamiento.

Se cambió el modo de enmascaramiento predeterminado de Session Replay para no permitir el desenmascaramiento mediante API programáticas y se corrigieron las API de anulación de UIKit.

Se corrigió el fallo que podía ocurrir en NRMAWKWebViewNavigationDelegate webViewWebContentProcessDidTerminate.

Se garantizó que los datos de Session Replay se carguen en el próximo lanzamiento si no se pudieron cargar cuando se cerró la aplicación.

Se descartaron los informes de fallos rechazados permanentemente (400/403).

Se agregó la expulsión de TTL para el almacenamiento sin conexión (#786).

Se corrigió el fallo de inicialización de NRMAHarvestController (#793).
