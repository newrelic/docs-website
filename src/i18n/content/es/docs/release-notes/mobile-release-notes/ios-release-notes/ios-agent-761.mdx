## Importante

* Esta versión actualiza la versión de iOS requerida del agente de New Relic iOS a iOS 16. La compatibilidad con iOS 15 regresará en una versión futura.

## Novedades de esta versión

* Se actualizó la vista previa pública de Session Replay móvil para incluir el muestreo de sesiones con errores y se corrigieron problemas relacionados con Session Replay.

## Corregido en esta versión

* Se corrigió un problema que impedía que el rastreo distribuido funcionara en ciertos casos extremos.
* Se agregaron correcciones de estabilidad para los problemas identificados.
