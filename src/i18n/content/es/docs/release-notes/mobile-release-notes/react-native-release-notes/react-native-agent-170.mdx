## Mejoras

* Se actualizó el agente nativo de Android a la versión 7.7.0.
* Reproducción de sesión inteligente: Se introdujo el modo ERROR (grabación en búfer de 15 a 30 s activada por fallos) y el modo FULL (grabación continua) para una captura de datos optimizada.
* APIs de control de sesión: Se agregó NewRelic.recordReplay() y NewRelic.pauseReplay() para permitir el control manual y programático de los estados de grabación de la reproducción.
* Corrección de error de red: Se corrigió un problema de OkHttp3 donde peekBody() causaba un bloqueo indefinido en respuestas de streaming (SSE/chunked) con longitudes de contenido desconocidas.
