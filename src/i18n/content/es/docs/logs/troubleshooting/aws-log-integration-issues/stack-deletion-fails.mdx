## Problema

Al eliminar una stack de CloudFormation, es posible que falle debido a varios motivos.

## Solución

Posible razón y su solución:

* Si los depósitos S3 creados por la stack (por ejemplo, para los logs de errores de Firehose) no están vacíos, no se pueden eliminar. Archivar el contenido y luego volver a intentar eliminar la stack.
