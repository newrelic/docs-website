## Problema

Los logs no aparecen en New Relic para un CloudWatch a través de la configuración de integración de Firehose.

## Solución

Posibles razones y sus soluciones:

* La clave de API puede ser incorrecta. Navegue a <DNT>**Amazon Data Firehose &gt; Firehose streams &gt; your-stream &gt; Edit destination settings**</DNT> y actualice la clave de API con la clave de licencia de ingesta correcta.
* El extremo proporcionado puede ser incorrecto. Verificar el extremo.
* Consulte los registros de S3 o los registros de errores de Cloudwatch para detectar cualquier problema.
