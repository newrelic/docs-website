Las reglas de ofuscación permiten enmascarar datos confidenciales antes de que el agente los transmita a New Relic. Cada regla es un patrón regex emparejado con una cadena de reemplazo. Las reglas se aplican a la carga JSON serializada antes de la transmisión.

## Configuración [#configuration]

Pase una matriz `obfuscate` en la configuración del agente:

```js
config: {
  obfuscate: [
    { regex: /\/user\/[^\/?"]+/, replacement: '/user/[REDACTED]' },
    { regex: /token=[^&"]+/, replacement: 'token=[MASKED]' },
  ]
}
```

## Ordenamiento de reglas [#rule-ordering]

Las reglas se aplican secuencialmente — la salida de una regla alimenta a la siguiente. Los patrones regex no válidos se omiten con una advertencia registrada a través de `Log.warn`.
