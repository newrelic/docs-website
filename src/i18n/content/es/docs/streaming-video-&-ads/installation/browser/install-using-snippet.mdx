Instale el agente de video incluyendo un fragmento de código JavaScript en su aplicación.

## Antes de que empieces

Complete primero los [pasos comunes de instalación](/docs/streaming-video-&-ads/installation/browser/get-started-with-browser#install).

## Coloca el fragmento de JavaScript en el código del frontend [#install-snippet]

<Steps>
  <Step>
    ### Nombra tu aplicación

    Proporcione un nombre a su aplicación y haga clic en <DNT>**Save and Continue**</DNT>.
  </Step>

  <Step>
    ### Instalar el agente

    Instala la última versión de New Relic copiando el fragmento de código y ejecutándolo en tu terminal (Mac) o símbolo del sistema (Windows).
  </Step>

  <Step>
    ### Agrega código a tu aplicación

    Una vez descargado el script, incluye en tu aplicación el fragmento de código que se muestra en la plataforma para comenzar a monitorear tu reproductor de video.
  </Step>

  <Step>
    ### Generar ID de aplicación

    Seleccione <DNT>**Get Application Info**</DNT>. Esto generará el ID de la aplicación para la generación de la entidad del navegador.
  </Step>

  <Step>
    ### Ver tus datos

    Haga clic en <DNT>**See your data**</DNT>.
  </Step>
</Steps>

<Callout variant="important">
  Puede tomar unos minutos interactuar con su aplicación y generar datos.
</Callout>
