Esta página proporciona una descripción general de los pasos de instalación al usar el código fuente para instalar manualmente el agente de Streaming Video &amp; Ads. Para la instalación completa con ejemplos de código, utilice la [instalación guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clona el repositorio [video-agent-android](https://github.com/newrelic/video-agent-android).
  </Step>

  <Step>
    En tu proyecto de Android, ve a <DNT>**File &gt; New &gt; Import Module**</DNT>.
  </Step>

  <Step>
    Ingrese la ubicación del directorio del módulo de biblioteca y haga clic en <DNT>**Finish**</DNT>.

    <Callout variant="important">
      El directorio del módulo de la biblioteca se encuentra en el repositorio que acaba de clonar. Módulos de ejemplo: `NewRelicVideoCore`, `NRExoPlayerTracker`, `NRIMATracker`.
    </Callout>
  </Step>

  <Step>
    Repita los pasos 2 y 3 con todos los módulos que desee incluir.
  </Step>

  <Step>
    En el archivo `build.gradle` del módulo de tu aplicación, agrega lo siguiente:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    Haga clic en <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ### Configurar el seguimiento del reproductor de video

    Seleccione el tipo de agente que coincida con su implementación:

    * **Agente de video solo con rastreador de ExoPlayer**: Para el seguimiento de la reproducción de video

    * **Agente de video con rastreadores de ExoPlayer e IMA**: Para la reproducción de video y el seguimiento de anuncios

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    Haz clic en <DNT>**See your data**</DNT> para desplegar tu código actualizado y ver los datos.
  </Step>
</Steps>
