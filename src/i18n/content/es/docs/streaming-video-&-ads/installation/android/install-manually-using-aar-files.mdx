Esta página proporciona una descripción general de los pasos de instalación al usar archivos AAR para instalar manualmente el agente de Streaming Video &amp; Ads. Para la instalación completa con ejemplos de código, utilice la [instalación guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).

<Steps>
  <Step>
    Clona el repositorio [video-agent-android](https://github.com/newrelic/video-agent-android).
  </Step>

  <Step>
    Ábrelo con Android Studio.
  </Step>

  <Step>
    Ve a <DNT>**View &gt; Tool Windows &gt; Gradle**</DNT> para abrir la ventana de herramientas de Gradle.
  </Step>

  <Step>
    En la ventana de herramientas de Gradle, despliegue <DNT>**NRVideoProject &gt; Tasks &gt; build**</DNT> y haga doble clic en <DNT>**assemble**</DNT>. Esto compilará el proyecto y generará los archivos de biblioteca AAR dentro de la carpeta de cada módulo <DNT>**build &gt; outputs &gt; aar**</DNT>.
  </Step>

  <Step>
    En su proyecto, vaya a <DNT>**File &gt; New &gt; New Module**</DNT>, importe el paquete `.JAR/.AAR` y haga clic en <DNT>**Next**</DNT>.
  </Step>

  <Step>
    Seleccione los archivos AAR que generó en el paso anterior y haga clic en <DNT>**Finish**</DNT>.
  </Step>

  <Step>
    Repita los pasos 5 y 6 para cada archivo AAR que necesite:

    * **NewRelicVideoCore.aar** (Requerido): Funcionalidad principal del agente de video
    * **NRExoPlayerTracker.aar** (Requerido): seguimiento de ExoPlayer
    * **NRIMATracker.aar** (Opcional): Seguimiento de anuncios con el SDK de IMA
  </Step>

  <Step>
    En el archivo `build.gradle` del módulo de tu aplicación, agrega las siguientes dependencias:

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")   // Required: Core video agent
      implementation project(":NRExoPlayerTracker")  // Required: ExoPlayer tracking
      implementation project(':NRIMATracker')        // Optional: Ad tracking with IMA
    }
    ```

    Nota: Solo incluye `NRIMATracker` si necesitas la funcionalidad de seguimiento de anuncios.
  </Step>

  <Step>
    Sincroniza tu proyecto con los archivos de Gradle haciendo clic en <DNT>**Sync Now**</DNT> en la barra de notificaciones, o ve a <DNT>**File &gt; Sync Project with Gradle Files**</DNT>.
  </Step>

  <Step>
    ### Inicializar el agente de video

    En el método o actividad `onCreate()` de su aplicación, inicialice el agente de video de New Relic. Para obtener ejemplos de código detallados e instrucciones de implementación, consulte la [instalación guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    ### Configurar el seguimiento del reproductor de video

    Configure el rastreador según sus necesidades:

    * **Solo rastreador de ExoPlayer**: Configure el seguimiento de ExoPlayer para el monitoreo de la reproducción de video
    * **Rastreadores de ExoPlayer e IMA**: Configura el seguimiento tanto de ExoPlayer como de IMA para la reproducción de video y el seguimiento de anuncios

    <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />

    Para obtener ejemplos completos de código de implementación, consulte la [instalación guiada](/docs/streaming-video-&-ads/installation/android/get-started-with-android).
  </Step>

  <Step>
    Compila y despliega tu aplicación para ver los datos de seguimiento de video en New Relic.
  </Step>
</Steps>
