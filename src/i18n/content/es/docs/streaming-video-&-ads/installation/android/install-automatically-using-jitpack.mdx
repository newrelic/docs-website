Esta página proporciona una descripción general de los pasos de instalación al usar JitPack para instalar el agente de Streaming Video &amp; Ads. Para la instalación completa con ejemplos de código, utilice la [instalación guiada](https://onenr.io/02R54MqY0wb).

<Steps>
  <Step>
    ## Agregar el repositorio de JitPack

    Agrega el repositorio de JitPack a tu archivo `build.gradle` raíz.
  </Step>

  <Step>
    ## Agregar dependencias

    Agrega las dependencias del agente de video al archivo `build.gradle` de tu aplicación y haz clic en <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    ## Configure el agente de video

    Agrega la configuración de NRVideo en el método `onCreate` de tu **clase Application o MainActivity**.
  </Step>

  <Step>
    ## Configurar el seguimiento del reproductor de video

    Seleccione el tipo de agente que coincida con su implementación:

    * **Agente de video solo con rastreador de ExoPlayer**: Para el seguimiento de la reproducción de video

    * **Agente de video con rastreadores de ExoPlayer e IMA**: Para la reproducción de video y el seguimiento de anuncios

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    Haz clic en <DNT>**See your data**</DNT> para ver tus datos de streaming en New Relic.
  </Step>
</Steps>
