El agente New Relic Roku está diseñado para monitorear el sistema y los eventos de video en la aplicación Roku. Proporciona una API integral para rastrear varios tipos de eventos, incluidos sistemas, videos, errores, anuncios y eventos personalizados. Este documento describe los métodos API disponibles para interactuar con el agente Roku New Relic .
