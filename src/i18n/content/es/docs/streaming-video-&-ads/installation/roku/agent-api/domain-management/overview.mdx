## Métodos de gestión de dominios

<table>
  <thead>
    <tr>
      <th width="30%">
        Método
      </th>

      <th>
        Descripción
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        [nrAddDomainSubstitution](/docs/streaming-video-&-ads/installation/roku/agent-api/domain-management/nradddomainsubstitution)
      </td>

      <td>
        Agregue un patrón de coincidencia para el atributo de dominio y sustitúyalo por otra cadena. Cada vez que se genera un evento o una métrica con un atributo de dominio, el agente verificará si coincide con una regex y aplicará la sustitución especificada. Si no se establece un patrón, utilizará el dominio de la URL sin cambios. Se aplica a todos los eventos y métricas que contienen el atributo &quot;domain&quot;.
      </td>
    </tr>

    <tr>
      <td>
        [nrDelDomainSubstitution](/docs/streaming-video-&-ads/installation/roku/agent-api/domain-management/nrdeldomainsubstitution)
      </td>

      <td>
        Eliminar un patrón coincidente creado con `nrAddDomainSubstitution`.
      </td>
    </tr>
  </tbody>
</table>

## Ejemplos

### nrAddDomainSubstitution

```js
  	nrAddDomainSubstitution(nr, "^.+\.my\.domain\.com$", "mydomain.com")
```

### nrDelDomainSubstitution

```js
  	nrDelDomainSubstitution(nr, "^.+\.my\.domain\.com$")
```
