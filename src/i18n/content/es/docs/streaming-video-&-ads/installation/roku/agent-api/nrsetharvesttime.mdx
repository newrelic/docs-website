## Sintaxis

```js
  nrSetHarvestTime(nr as Object, time as Integer) as Void
```

## Descripción

Establezca el tiempo de cosecha, el tiempo en que se almacenan en búfer las muestras antes de enviarlas a New Relic tanto para eventos como para logs. El valor mínimo es 60.
