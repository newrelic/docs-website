## Sintaxis

```js
  nrAddDomainSubstitution(nr as object, pattern as String, subs as String) as Void
```

## Descripción

* Agregue un patrón coincidente para el atributo de dominio y sustitúyalo por otra cadena.
  * Cada vez que se genera un evento o una métrica con un atributo de dominio, el agente verificará si coincide con una expresión regular y aplicará la sustitución especificada.
* Si no se establece ningún patrón, se empleará el dominio URL sin cambios.
  * Se aplica a todos los eventos y métricas que contengan el atributo &quot;dominio&quot;.

