## Sintaxis

```js
NewRelicSystemStart(port as Object) as Object
```

## Descripción

Inicia el logging del sistema.

## Devolver

El objeto `roSystemLog` creado.

## Ejemplo

```js
  m.syslog = NewRelicSystemStart(m.port)
```
