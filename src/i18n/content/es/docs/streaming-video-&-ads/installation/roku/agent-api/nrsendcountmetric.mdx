## Sintaxis

```js
  function nrSendCountMetric(nr as Object, name as String, value as dynamic, interval as Integer, attr = invalid as Object) as Void
```

## Descripción

Registra una métrica de conteo. Mide el número de ocurrencias de un evento durante un intervalo de tiempo.
