## Sintaxis

```js
  nrSendHttpResponse(nr as Object, _url as String, msg as Object) as Void
```

## Descripción

Envía un evento HTTP\_RESPONSE del tipo RokuSystem.

Nada.

## Ejemplo

```js
  	msg = wait(5000, m.port)
```
