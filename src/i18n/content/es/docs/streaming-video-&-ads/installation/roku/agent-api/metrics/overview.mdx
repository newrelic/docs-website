## Métodos de métricas

<table>
  <thead>
    <tr>
      <th width="30%">

  <tbody>
    <tr>
      <td>
        [nrEnviarMétrica](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendmetric)
      </td>

      <td>
        Grabar un medidor métrico. Representa un valor que puede aumentar o disminuir con el tiempo.
      </td>
    </tr>

    <tr>
      <td>
        [nrMetrica de conteo de envíos](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendcountmetric)
      </td>

      <td>
        Registra una métrica de conteo. Mide el número de ocurrencias de un evento durante un intervalo de tiempo.
      </td>
    </tr>

    <tr>
      <td>
        [nrEnviarResumenMétrico](/docs/streaming-video-&-ads/installation/roku/agent-api/metrics/nrsendsummarymetric)
      </td>

      <td>
        Registre una métrica de resumen. Se emplea para informar datos preagregados o información sobre eventos discretos agregados.
      </td>
    </tr>
  </tbody>
</table>

## Ejemplos

### nrEnviarMétrica

```js
  	nrSendMetric(m.nr, "test", 11.1, {"one": 1})
```

### nrMetrica de conteo de envíos

```js
  	nrSendCountMetric(m.nr, "test", 250, 1500, {"one": 1})
```

### nrEnviarResumenMétrico

```js
  	nrSendSummaryMetric(m.nr, "test", 2000, 5, 1000, 100, 200)
```
