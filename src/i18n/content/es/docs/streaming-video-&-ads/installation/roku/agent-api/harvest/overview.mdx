## Métodos de configuración de Harvest

<table>
  <thead>
    <tr>
      <th width="30%">
        Método
      </th>

      <th>
        Descripción
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        [nrEstablecerHoraDeCosecha](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttime)
      </td>

      <td>
        Establezca el tiempo de cosecha, el tiempo en que se almacenan en búfer las muestras antes de enviarlas a New Relic tanto para eventos como para logs. El valor mínimo es 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrEstablecer eventos de tiempo de cosecha](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimeevents)
      </td>

      <td>
        Establezca el tiempo de cosecha para el evento, el tiempo en que se almacenan los eventos antes de enviar a New Relic. El valor mínimo es 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrSetHarvestTimeLogs](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimelogs)
      </td>

      <td>
        Establece el tiempo de cosecha de los logs, el tiempo durante el cual los logs se almacenan en búfer antes de enviar a New Relic. El valor mínimo es 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrEstablecerMétricasDeTiempoDeCosecha](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrsetharvesttimemetrics)
      </td>

      <td>
        Establece el tiempo de cosecha de los logs, el tiempo durante el cual los logs se almacenan en búfer antes de enviar a New Relic. El valor mínimo es 60.
      </td>
    </tr>

    <tr>
      <td>
        [nrForceHarvest](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvest)
      </td>

      <td>
        Coseche eventos y logs inmediatamente. No resetear el temporizador de cosecha.
      </td>
    </tr>

    <tr>
      <td>
        [nrForceHarvestEventos](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvestevents)
      </td>

      <td>
        Realice la cosecha inmediatamente. No resetear el temporizador de cosecha.
      </td>
    </tr>

    <tr>
      <td>
        [nrForceHarvestLogs](/docs/streaming-video-&-ads/installation/roku/agent-api/harvest/nrforceharvestlogs)
      </td>

      <td>
        Coseche los troncos inmediatamente. No resetear el temporizador de cosecha.
      </td>
    </tr>
  </tbody>
</table>

## Ejemplos

### nrEstablecerHoraDeCosecha

```js
  	nrSetHarvestTime(m.nr, 60)
```

### nrEstablecer eventos de tiempo de cosecha

```js
  	nrSetHarvestTimeEvents(m.nr, 60)
```

### nrSetHarvestTimeLogs

```js
  	nrSetHarvestTimeLogs(m.nr, 60)
```

### nrEstablecerMétricasDeTiempoDeCosecha

```js
  	nrSetHarvestTimeMetrics(m.nr, 60)
```

### nrForceHarvest

```js
  	nrForceHarvest(m.nr)
```

### nrForceHarvestEventos

```js
  	nrForceHarvestEvents(m.nr)
```

### nrForceHarvestLogs

```js
  	nrForceHarvestLogs(m.nr)
```
