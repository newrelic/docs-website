## Sintaxis

```js
  nrProcessMessage(nr as Object, msg as Object) as Boolean
```

## Descripción

Comprueba si hay un mensaje de log del sistema, lo procesa y envía el evento apropiado.

## Devolver

Verdadero si msg es un mensaje de log del sistema, falso en caso contrario.

## Ejemplo

```js
  	while (true)
		      msg = wait(0, m.port)
		      if nrProcessMessage(m.nr, msg) = false
			            if type(msg) = "roPosterScreenEvent"
				                  if msg.isScreenClosed()
					                        exit while
				                  end if
			            end if
		      end if
	  end while
```
