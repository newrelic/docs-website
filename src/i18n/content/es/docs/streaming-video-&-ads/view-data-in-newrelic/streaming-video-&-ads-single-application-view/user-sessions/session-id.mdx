## evento de transmisión de sesión de usuario

Los eventos de streaming de la sesión de usuario son los eventos que se generan durante la sesión. Estos eventos se emplean para rastrear el comportamiento del usuario durante la sesión.

En la página de la sesión del usuario se muestra un gráfico que muestra los eventos que se generan durante la sesión y una información detallada sobre el evento como:
