## Descripción general

La vista Individual Platform proporciona capacidades detalladas de monitoreo y análisis para una aplicación de streaming específica. Esta vista le ayuda a analizar a fondo el rendimiento específico de la aplicación, solucionar problemas aislados en una sola plataforma y monitorear la experiencia completa de streaming para esa aplicación.

### Qué puede hacer con la vista Individual Platform

Use esta vista para:

* Monitoree las métricas de rendimiento de video y anuncios para una aplicación específica donde está instalado el agente de video
* Analice métricas de calidad de video y de interacción específicas de la aplicación
* Ver el rendimiento detallado del anuncio para la aplicación específica
* Rastrear las sesiones de usuario dentro de la aplicación
* Solucionar problemas aislados en una sola plataforma o aplicación
* Monitorear las dependencias y relaciones de los servicios para la aplicación
* Crear condiciones de alerta específicas de la aplicación
* Acceda a la página Summary para obtener una vista consolidada del rendimiento de la aplicación.

### Quién usa esta vista

Esta vista es útil para los desarrolladores de aplicaciones, gerentes de producto y equipos de soporte que necesitan monitorear y solucionar problemas de una aplicación de streaming específica, analizar problemas de rendimiento dentro de esa plataforma y garantizar que la aplicación cumpla con los estándares de calidad.

### Cuándo usar la vista de Individual Platform vs. la vista de All Platform

Utilice la **vista Individual Platform** cuando necesite:

* Análisis en profundidad del rendimiento de una aplicación específica
* Solucionar problemas aislados en una sola plataforma
* Monitorear métricas y configuraciones específicas de la aplicación
* Analizar las sesiones de usuario dentro de una aplicación específica
* Ver mapas de servicios y dependencias de la aplicación
* Crear alertas específicas de la aplicación

Utilice la **vista All Platform** cuando necesite:

* Compare el rendimiento entre múltiples aplicaciones
* Identifica las tendencias que afectan a toda tu infraestructura de streaming
* Monitorear el estado general del streaming en todas las plataformas
* Investigar problemas que puedan estar afectando a múltiples aplicaciones
