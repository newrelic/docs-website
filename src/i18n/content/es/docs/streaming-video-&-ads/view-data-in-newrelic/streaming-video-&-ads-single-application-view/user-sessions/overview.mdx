## Descripción general

La página User Sessions proporciona una vista detallada de las sesiones de visualización individuales para una aplicación monitoreada específica. Esta página le ayuda a realizar un seguimiento del comportamiento de visualización del usuario, identificar sesiones con errores y analizar patrones de reproducción dentro de una única aplicación o plataforma de streaming.

### Qué puede hacer con User Sessions

Utilice esta página para:

* Ver todas las sesiones de visualización de usuarios para una aplicación específica donde está instalado el agente de video
* Rastrea los detalles a nivel de sesión, incluidos el ID de vista, el ID de sesión y el ID de usuario
* Monitorear el contenido visualizado por título
* Identificar sesiones con errores para la resolución de problemas
* Ver las horas de inicio y fin de la sesión para analizar la duración de la visualización
* Acceda a la información detallada de la sesión haciendo clic en View ID o Session ID
* Filtrar sesiones por rango de tiempo para enfocarse en períodos específicos
* Analiza los patrones de visualización para el dispositivo y la plataforma específicos
* Revisar los detalles del contenido de cada sesión

### Quién usa esta página

Esta página es útil para desarrolladores de aplicaciones, equipos de soporte al cliente e ingenieros de control de calidad que necesitan solucionar problemas reportados por los usuarios en una aplicación específica, analizar sesiones de visualización individuales y comprender los patrones de comportamiento de los usuarios dentro de esa plataforma.

## Página User Sessions

La **Single platform &gt; User Sessions** para transmisión de videos y anuncios enumera todas las sesiones en una aplicación específica.
