## Descripción general

La página User Sessions proporciona una vista detallada de las sesiones de visualización individuales en todas sus aplicaciones y plataformas de streaming monitoreadas. Esta página le ayuda a hacer un seguimiento del comportamiento de visualización de los usuarios, identificar sesiones con errores y analizar patrones de reproducción en toda su infraestructura de streaming.

### Qué puede hacer con User Sessions

Utilice esta página para:

* Ver todas las sesiones de visualización de los usuarios en todas las aplicaciones donde está instalado el agente de video
* Realice un seguimiento de los detalles a nivel de sesión, incluyendo el ID de vista, el ID de sesión, el ID de usuario y el nombre de la aplicación
* Monitorear el contenido visualizado por título y plataforma
* Identificar sesiones con errores para la resolución de problemas
* Ver las horas de inicio y fin de la sesión para analizar la duración de la visualización
* Acceda a la información detallada de la sesión haciendo clic en View ID o Session ID
* Filtrar sesiones por rango de tiempo para enfocarse en períodos específicos
* Analiza los patrones de visualización en diferentes dispositivos y plataformas
* Revisar los detalles del contenido de cada sesión

### Quién usa esta página

Esta página es útil para los equipos de soporte al cliente, los ingenieros de control de calidad y los equipos de operaciones que necesitan solucionar problemas reportados por los usuarios, analizar sesiones de visualización individuales en múltiples plataformas y comprender los patrones de comportamiento de los usuarios en todas las aplicaciones de streaming.

## Página User Sessions

La **All Platforms &gt; User Sessions** for Streaming Video &amp; Ads enumera todas las sesiones en toda la aplicación.
