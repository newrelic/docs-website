La **All Platforms &gt; User Sessions** for Streaming Video &amp; Ads enumera todas las sesiones en toda la aplicación.

## Ver la UIde la página Sesiones del usuario

La tabla de la página **User Session** tiene varias columnas para mostrar información variada para cada sesión. La siguiente tabla proporciona información sobre cada columna disponible en la UI de la página Sesión:

<Callout variant="important">
  La tabla de sesiones muestra todas las sesiones de usuario que se iniciaron dentro del filtro de tiempo seleccionado encima de la tabla.
