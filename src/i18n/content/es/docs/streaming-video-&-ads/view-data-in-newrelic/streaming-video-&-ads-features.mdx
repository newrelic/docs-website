Es posible que tenga cientos de aplicaciones diferentes que brinden contenido de video en diferentes plataformas y requiera monitorear la reproducción de videos y anuncios en todas las aplicaciones. Para ayudar a gestionar esta complejidad, New Relic le permite ver telemetry data de reproducción de videos y anuncios de una aplicación individual o de todas las aplicaciones en varias plataformas:

* [Vista de toda la plataforma](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-all-platform-view/overview): vea telemetry data de reproducción de videos y anuncios de todas las aplicaciones donde se instrumentó el agente de video.
* [Vista de plataforma individual](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/overview): vea telemetry data de reproducción de video y anuncios de una aplicación/plataforma específica donde se instaló el agente de video.
