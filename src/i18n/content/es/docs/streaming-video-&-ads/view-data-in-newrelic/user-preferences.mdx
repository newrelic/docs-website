Cuando configuras los dashboards <DNT>**Video Overview**</DNT> o <DNT>**Ad Overview**</DNT>, New Relic guarda tu configuración en tu perfil de usuario. La próxima vez que regreses a cualquiera de los dashboards, tu configuración se restaura para que no tengas que empezar desde cero.

Las preferencias se guardan por usuario y por tipo de dashboard. Tus configuraciones de <DNT>**Video Overview**</DNT> nunca afectan tus configuraciones de <DNT>**Ad Overview**</DNT>, y viceversa.

## Cómo se guardan las preferencias [#how-saving-works]

Los cambios se escriben de forma asíncrona en segundo plano para que el dashboard siga respondiendo mientras se guarda. Para reducir las escrituras innecesarias cuando realiza ajustes rápidos, los guardados se agrupan brevemente.

Las preferencias guardadas persisten a través de:

* Actualizaciones de Browser
* Cierre de sesión e inicio de sesión
* Diferentes navegadores o dispositivos, siempre que haya iniciado sesión con la misma cuenta

## Comportamiento por defecto [#defaults]

Si no tienes preferencias guardadas para un dashboard (por ejemplo, en tu primera visita), el dashboard se abre con la configuración predeterminada del producto.

## Restablecer a los valores predeterminados [#reset-to-defaults]

Un control <DNT>**Reset to default view**</DNT> le permite borrar sus preferencias guardadas y devolver el dashboard a los valores predeterminados originales del producto.

<Callout variant="tip">
  El restablecimiento solo afecta al dashboard que estás viendo actualmente. Restablecer las preferencias de <DNT>**Video Overview**</DNT> no afecta tus preferencias de <DNT>**Ad Overview**</DNT>.
</Callout>

## Manejo de errores [#error-handling]

Si la recuperación de preferencias falla cuando se carga el dashboard, el dashboard vuelve a la configuración predeterminada del producto y muestra una notificación de error. Tu flujo de trabajo no está bloqueado.

Si falla el guardado de una preferencia, el dashboard sigue funcionando normalmente.

Si una métrica, dimensión o faceta guardada previamente ya no está disponible (por ejemplo, porque quedó obsoleta), el dashboard sustituye un valor predeterminado razonable para esa configuración, como la primera opción disponible, para que ningún widget quede vacío o roto.
