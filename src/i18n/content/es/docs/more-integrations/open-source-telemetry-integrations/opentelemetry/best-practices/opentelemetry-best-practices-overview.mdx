Configurar OpenTelemetry y aprovecharlo al máximo puede ser un desafío. Para ayudarle a optimizar su experiencia, hemos creado estas guías de mejores prácticas. Si estás intentando resolver un problema específico, consulta nuestra [guía de resolución de problemas](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/opentelemetry-troubleshooting).

Primero, tenemos una [guía de implementación de un extremo a otro para OpenTelemetry y New Relic](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-implementation-guide), que le brinda orientación detallada para garantizar que haya integrado OpenTelemetry y New Relic de manera óptima.

Otras guías de mejores prácticas incluyen:

* [alerta](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-alerts): notifíquese sobre problemas utilizando alerta.
* [Longitudes de atributos](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-attributes): Manténgase dentro de los límites para los nombres y valores de los atributos.
* [Procesamiento por lotes](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-batching): utilice el procesamiento por lotes para evitar la limitación de velocidad.
* [browser](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-browser): evite exponer la clave de licencia en la aplicación del navegador.
* [Compresión](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-compression): Maximiza tu carga.
* [log](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-logs): envía el registro a New Relic utilizando las herramientas OpenTelemetry.
* [métrica](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-metrics): Explore cómo se procesan las métricas de OpenTelemetry en New Relic.
* [Recursos](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-resources): Utilice recursos para la entidad que genera telemetry data.
* [traza](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-traces): Ver algunos requisitos y opciones para traza y tramos.
* [Versiones](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-versions): obtenga más información sobre las versiones compatibles.
