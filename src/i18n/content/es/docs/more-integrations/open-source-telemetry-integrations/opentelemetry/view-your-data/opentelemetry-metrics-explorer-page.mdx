Después de enviarnos sus datos de OpenTelemetry y abrir su servicio (entidad) en la UI, puede hacer clic en <DoNotTranslate>**Metrics explorer**</DoNotTranslate> para ver información sobre su métrica (para idiomas OpenTelemetry seleccionados). Además, si estás utilizando el exportador Prometheus con OpenTelemetry, puedes ver tus datos métricos aquí.
