Después de enviarnos sus datos de OpenTelemetry y abrir su servicio (entidad) en la UI, puede hacer clic en <DoNotTranslate>**External services**</DoNotTranslate> para ver las llamadas a servicios fuera de proceso, como servicios web, recursos en la nube y otras redes. entidades.

Para que tus datos aparezcan en esta sección, asegúrate de que tenga lo siguiente:

* `span.kind = client` o `producer`
* `db.system` no es presente
* `http.url` o `net.peer.name` debe estar presente

Para obtener más detalles, consulte [Servicios externos](/docs/apm/apm-ui-pages/monitoring/external-services/external-services-intro).

Para obtener más información sobre OpenTelemetry en otras páginas UI , consulte la [descripción general de laUI ](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/view-your-data/opentelemetry-view-your-data).
