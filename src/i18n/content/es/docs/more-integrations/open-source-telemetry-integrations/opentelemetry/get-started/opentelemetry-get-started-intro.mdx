Tiene una variedad de opciones para comenzar con OpenTelemetry y New Relic. Puede pasar directamente al monitoreo de sus propias aplicaciones, hosts o clúster de Kubernetes, o puede configurar algunas aplicaciones de demostración para experimentar con OpenTelemetry y New Relic. Tenemos una variedad de instrucciones que lo ayudarán a aprovechar el poder de OpenTelemetry para recopilar datos y enviarlos a New Relic.

## Configura tu propia aplicación o servicio [#own-app]

Si está familiarizado con OpenTelemetry y está listo para trabajar con su propio código, consulte [Configure su propia aplicación o servicio con OpenTelemetry](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/get-started/opentelemetry-set-up-your-app). Estos pasos le ayudarán a recopilar telemetría y exportarla a New Relic.

## Monitor un anfitrión [#otel-infra]

Puede monitor sus hosts utilizando el recolector OpenTelemetry. Ver [recolector para monitoreo de host](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/collector/opentelemetry-collector-infra-hosts).

## Monitor kubernetes [#otel-kubernetes]

Monitor su clúster de Kubernetes con el recolector OpenTelemetry. Consulte las instrucciones en [Vincular aplicación instrumentada de OpenTelemetry a Kubernetes](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-otel-applications-kubernetes/).

## Pruebe una aplicación de demostración [#demo-app-java]

En lugar de pasar directamente a la configuración de su propia aplicación, puede activar nuestra aplicación de demostración con OpenTelemetry y New Relic.

Nuestros tutoriales para [Java](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/get-started/opentelemetry-tutorial-java) y [Python](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/get-started/opentelemetry-tutorial-python) le ofrecen una variedad de opciones para experimentar con OpenTelemetry:

* Inspeccione una aplicación pre-instrumentada que puede activar rápidamente para enviar datos a New Relic.
* Utilice un agente de OpenTelemetry para instrumentar automáticamente una aplicación de demostración.
* Instrumentado manualmente una aplicación de demostración.

## Aprenda conceptos de OpenTelemetry en la clase magistral [#otel-masterclass]

Si desea obtener más información sobre los conceptos detrás de OpenTelemetry y aplicarlos con una pequeña aplicación de demostración, eche un vistazo a nuestra [clase magistral](https://developer.newrelic.com/opentelemetry-masterclass). Puede probar nuestros tutoriales prácticos para .NET o Python.
