La integración agéntica de New Relic proporciona una solución impulsada por IA para automatizar y optimizar la observabilidad del flujo de trabajo, mejorando la gestión y el monitoreo del sistema. Esta integración facilita operaciones sin interrupciones al automatizar tareas repetitivas a través de un agente de IA, reduciendo la intervención manual. Al integrar perfectamente con plataformas como GitHub Copilot, Gemini Code Assist y ServiceNow, la integración de Agentic transforma flujos de trabajo complejos en operaciones fluidas.

## Característica clave [#features]

* **Automatización inteligente:** aproveche la IA para automatizar y agilizar el flujo de trabajo complejo, reduciendo la necesidad de cambiar manualmente de herramientas y mejorando la productividad.
* **Observabilidad mejorada:** Obtenga información valiosa y completa en sus sistemas, lo que permite una detección, análisis y resolución de incidentes más rápidos.
* **Integración perfecta:** conectar con plataformas populares como GitHub Copilot para incorporar análisis poderosos e información valiosa directamente a su flujo de trabajo existente.

## Acceso y gestión de datos [#access-data-mgmt]

Nuestros socios de integración gestionarán la aplicación del cliente y mantendrán el token de usuario cuando usted o cualquiera de sus usuarios se autentiquen en New Relic a través de la integración del socio. El socio externo tendrá el mismo nivel de acceso a New Relic que el usuario que se autenticó durante el tiempo configurado en la configuración del usuario, ya que los tokens se emplean en su nombre. Esta disposición ayuda a proporcionar una integración y funcionalidad perfectas. Para obtener más información, puede consultar la documentación del socio que detalla sus prácticas de uso y gestión de token.

### Control de acceso basado en roles (RBAC)

Actualmente, se puede acceder a RBAC a través de interfaces API. Si está interesado en emplear RBAC a través de API, no dude en contactarnos a [agenticai\\@newrelic.com](mailto:agenticai@newrelic.com). Más adelante este año, implementaremos la funcionalidad RBAC directamente dentro de nuestra interfaz de usuario y, en ese momento, todos los clientes deberán agregar usuarios a través de la UI. Para obtener instrucciones detalladas sobre la configuración de RBAC, consulte nuestro [tutorial de administración de usuarios de cuentas](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial).

## Integración actual [#integrations]

### Copiloto de GitHub [#github-copilot]

Integre información valiosa de New Relic directamente en su IDE con GitHub Copilot. Puede emplear agente AI para acceder a logs, errores y recomendaciones sin salir de su entorno de codificación. Proporciona tiempo real información valiosa a medida que desarrollan e implementan el código.

Esta integración ofrece:

* Comprobaciones de implementación automatizadas: realice comprobaciones de implementación automáticamente, aprovechando los datos de New Relic.
* tiempo real alertas y rastreo de errores: Mantener informado con alertas inmediatas y realice un seguimiento de los errores a medida que ocurren.
* Recomendaciones probabilísticas para acciones de despliegue.

Para obtener más información, consulte [Introducción al agente de New Relic para GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration).

### ServiceNow Ahora Asiste [#now-assist]

La información valiosa New Relic AI ahora está dentro de Now Assist de ServiceNow. Esta integración ayuda a sus equipos de operaciones (como personal de primera respuesta y agentes de soporte) a comprender rápidamente la gravedad de los incidentes, identificar las posibles causas raíz y optimizar la colaboración, todo sin salir de su espacio de trabajo de ServiceNow.

Para obtener más información, consulte [Introducción al agente de New Relic para ServiceNow Now Assist](/docs/agentic-ai/agentic-integration/servicenow-integration).

### Asistencia del Código Géminis [#gemini-code-assist]

<Callout title="avance">
  Todavía estamos trabajando en esta característica, ¡pero nos encantaría que la probaras!

  Esta característica se proporciona actualmente como parte de un programa de vista previa de conformidad con nuestras [políticas de prelanzamiento](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Integre información valiosa New Relic en su IDE con Gemini Code Assist. Esta herramienta permite a los desarrolladores acceder a logs relevantes, información de errores y recomendaciones dentro del entorno de codificación, ofreciendo información valiosa que puede informar el proceso de desarrollo e implementación.

Esta integración ofrece:

* Monitoreo automatizado de implementación: monitoree automáticamente la implementación empleando datos de tiempo real de New Relic para identificar y abordar problemas de manera eficiente.
* alertas y rastreo de errores: reciba notificaciones inmediatas y rastree los errores de manera efectiva para mantener el rendimiento de la aplicación y la estabilidad.
* Recomendaciones para la implementación Acciones: Emplear recomendaciones basadas en el análisis de datos para mejorar el rendimiento del código y respaldar la toma de decisiones.

Para obtener más información, consulte [Primeros pasos con el agente New Relic para Gemini Code Assist](/docs/agentic-ai/agentic-integration/gemini-integration).
