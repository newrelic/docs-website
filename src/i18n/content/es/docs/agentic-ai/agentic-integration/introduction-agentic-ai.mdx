La integración Agentic de New Relic proporciona una solución impulsada por IA para automatizar y optimizar la observabilidad del flujo de trabajo, mejorando la gestión y el monitoreo del sistema. Esta integración facilita operaciones sin interrupciones al automatizar tareas repetitivas a través de un agente de IA, reduciendo la intervención manual. Al integrar perfectamente con plataformas como GitHub Copilot, Amazon Q Business, Gemini Code Assist y ServiceNow, la integración de Agentic transforma el flujo de trabajo complejo en operaciones fluidas.

## Característica clave [#features]

* **Automatización inteligente:** aproveche la IA para automatizar y agilizar el flujo de trabajo complejo, reduciendo la necesidad de cambiar manualmente de herramientas y mejorando la productividad.
* **Observabilidad mejorada:** Obtenga información valiosa y completa en sus sistemas, lo que permite una detección, análisis y resolución de incidentes más rápidos.
* **Integración perfecta:** conectar con plataformas populares como GitHub Copilot para incorporar análisis poderosos e información valiosa directamente a su flujo de trabajo existente.

## Integración actual [#integrations]

### Copiloto de GitHub [#github-copilot]

Integre información valiosa de New Relic directamente en su IDE con GitHub Copilot. Puede emplear agente AI para acceder a logs, errores y recomendaciones sin salir de su entorno de codificación. Proporciona tiempo real información valiosa a medida que desarrollan e implementan el código.

Esta integración ofrece:

* Comprobaciones de implementación automatizadas: realice comprobaciones de implementación automáticamente, aprovechando los datos de New Relic.
* tiempo real alertas y rastreo de errores: Mantener informado con alertas inmediatas y realice un seguimiento de los errores a medida que ocurren.
* Recomendaciones probabilísticas para acciones de despliegue.

Para obtener más información, consulte [Introducción al agente de New Relic para GitHub Copilot](/docs/agentic-ai/agentic-integration/github-copilot-integration).

### Amazon Q Negocios [#amazon-q]

La integración de Amazon Q Business con New Relic AI está diseñada para facilitar una resolución de problemas más eficiente y mejorar la observabilidad. Al permitir que el ingeniero de soporte diagnostique y resuelva problemas rápidamente, esta integración ayuda a mantener la integridad de procesos empresariales críticos, incluidos los pagos de usuarios.

Esta integración ofrece:

* Interfaz de IA conversacional: interactúe con New Relic AI a través de la interfaz conversacional de Amazon Q Business para obtener información valiosa y recomendaciones.
* Objetivo alerta Análisis: aislar y analizar alertas específicas de servicios clave, como Checkout y Procesamiento de pagos, dentro de un periodo de tiempo definido.
* Inteligencia de alerta completa: acceda a reportes detallados sobre anomalías, frecuencias de errores y problemas de latencia, complementados con enlaces al dashboard de New Relic para un análisis más profundo.
* Recomendaciones proactivas: Recibir sugerencias informadas sobre la necesidad de realizar investigaciones adicionales para abordar de forma preventiva posibles problemas.

Para obtener detalles adicionales, consulte [Primeros pasos con el agente de New Relic para Amazon Q Business](/docs/agentic-ai/agentic-integration/amazonq-integration).

### Asistencia del Código Géminis [#gemini-code-assist]

Integre información valiosa New Relic en su IDE con Gemini Code Assist. Esta herramienta permite a los desarrolladores acceder a logs relevantes, información de errores y recomendaciones dentro del entorno de codificación, ofreciendo información valiosa que puede informar el proceso de desarrollo e implementación.

Esta integración ofrece:

* Monitoreo automatizado de implementación: monitoree automáticamente la implementación empleando datos de tiempo real de New Relic para identificar y abordar problemas de manera eficiente.
* alertas y rastreo de errores: reciba notificaciones inmediatas y rastree los errores de manera efectiva para mantener el rendimiento de la aplicación y la estabilidad.
* Recomendaciones para la implementación Acciones: Emplear recomendaciones basadas en el análisis de datos para mejorar el rendimiento del código y respaldar la toma de decisiones.

Para obtener más información, consulte [Primeros pasos con el agente New Relic para Gemini Code Assist](/docs/agentic-ai/agentic-integration/gemini-integration).

## Herramientas [#tools]

Cada integración agente viene con varias herramientas para emplear las capacidades de la plataforma New Relic y brindar observabilidad, información valiosa, alertas y predicciones:

* **[Herramienta de inteligencia de cambio:](/docs/agentic-ai/tools/change-intelligence-tool)** la herramienta de inteligencia de cambio lo ayuda a comprender y mitigar los riesgos potenciales al proporcionar un análisis y pronóstico de cambios integrales. Puede emplear esta herramienta para obtener información valiosa e instantánea sobre el impacto de los cambios en el sistema. Lo ayuda a evaluar los riesgos potenciales asociados con las modificaciones de implementación y configuración. Para obtener más información, consulte [Herramienta de inteligencia de cambios](/docs/agentic-ai/tools/change-intelligence-tool).

* **[Herramienta de inteligencia de alerta](/docs/agentic-ai/tools/alert-intelligence-tool):** la herramienta de inteligencia de alerta ofrece un análisis de alerta avanzado para resolver incidentes rápidamente. Proporciona notificación en tiempo real y recomendaciones inmediatas para evitar la escalada. Para obtener más información, consulte [la herramienta de inteligencia de alerta](/docs/agentic-ai/tools/alert-intelligence-tool).
