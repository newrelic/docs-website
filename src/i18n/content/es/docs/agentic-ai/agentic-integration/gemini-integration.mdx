## Habilitar la herramienta New Relic en Gemini Code Assist [#enable-new-relic-tool]

### Requisitos previos [#prerequisites]

Antes de comenzar, cerciorar de que se cumplan los siguientes requisitos previos para garantizar un proceso de configuración sin problemas:

* Hay un entorno de desarrollo integrado (IDE) instalado en su computadora.
* Tienes acceso a Google Gemini Code Assist.
* Gemini Code Assist está instalado en su IDE. Para obtener más información, consulte [la documentación de Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="consejo">
  Si desea restringir el acceso a Gemini Code Assist para ciertos roles, configure los ajustes en Gemini Code Assist. Para restringir el acceso a New Relic AI para roles específicos o empleados, puede usar el Control de acceso basado en roles (RBAC) de New Relic. Para obtener más información, consulte [Administración de usuariosNew Relic ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

Para habilitar la herramienta New Relic:

1. Desde su IDE, inicie sesión en [Gemini Code Assist](https://codeassist.google.com) haciendo clic en Iniciar sesión con Google.

2. Luego de haber autenticado exitosamente, regrese a su IDE.

3. En el <DNT>Gemini Code Assist Chat</DNT> (chat), ingrese `@newrelic` y presione Enter. Serás el símbolo para habilitar la herramienta New Relic .

4. En la pista, haga clic en el enlace para habilitar la herramienta New Relic . Se abrirá una ventana del navegador en la que aparecerá el símbolo para habilitar la herramienta New Relic dentro de Gemini Code Assist.

5. Habilite la herramienta New Relic y luego haga clic en **Connect**.

6. Inicie sesión empleando los datos de su cuenta de New Relic. Esto conecta exitosamente su cuenta de New Relic a Gemini Code Assist.

   Ahora, puedes usar la herramienta New Relic dentro de la extensión de chat Gemini Code Assist en tu IDE.

## Interactuar con la herramienta New Relic [#new-relic-tool]

La herramienta New Relic dentro de Gemini Code Assist puede ayudar con una variedad de tareas y ofrecer información valiosa. Para interactuar con la herramienta New Relic , en el cuadro de entrada del chat, ingresa tu símbolo de la siguiente manera:

`@newrelic <your message>`

Donde, `<your message>` es la acción que desea que ejecute la herramienta. Por ejemplo, `@newrelic What is apdex?`
