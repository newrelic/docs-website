## Instalar el agente New Relic en Gemini Code Assist [#install-new-relic-agent]

### Requisitos previos [#prerequisites]

Antes de comenzar, cerciorar de que se cumplan los siguientes requisitos previos para garantizar un proceso de configuración sin problemas:

* Hay un IDE instalado en su sistema.
* Tienes acceso a Google Gemini Code Assist.
* Gemini Code Assist está instalado en su IDE. Para obtener más información, consulte [la documentación de Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="consejo">
  Si desea restringir el acceso a Gemini Code Assist para ciertos roles, configure los ajustes en Gemini Code Assist. Para restringir el acceso a New Relic AI para roles específicos o empleados, puede usar el Control de acceso basado en roles (RBAC) de New Relic. Para obtener más información, consulte [Administración de usuariosNew Relic ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

**Para habilitar el agente New Relic en Gemini Code Assist:**

1. En <DNT>Google Cloud Console</DNT>, busque <DNT>**NewRelic**</DNT> y habilite la extensión. Se abre una ventana de diálogo de inicio de sesión.

2. Proporcione sus credenciales de New Relic para iniciar sesión. El mensajea **New Relic successfully linked.** se muestra.

   Ahora, puedes usar el agente New Relic dentro de la extensión de chat Gemini Code Assist en tu IDE. Para obtener más información, consulte [el uso del agente New Relic en Gemini Code Assist](#communicate-with-new-relic-agent-in-gemini-code-assist).

## Comunicar con el agente de New Relic en Gemini Code Assist [#communicate-with-new-relic-agent-in-gemini-code-assist]

El agente New Relic de Gemini Code Assist puede ayudar con una variedad de tareas y ofrecer información valiosa. Para interactuar con el agente de New Relic, en el cuadro de entrada del chat, escribe `@newrelic`.

### Estructura del mensaje de chat [#chat-message-structure]

Para interactuar con el agente New Relic , ingresa tu símbolo de la siguiente manera:

`@newrelic <your message>`

Dónde:

* `@newrelic`: interacción con el agente New Relic .
* `<your message>`:Acción a presentar al agente para su procesamiento.
