Este documento contiene información importante sobre vulnerabilidades de seguridad que podrían afectar algunas versiones de los productos y servicios de New Relic. Los boletines de seguridad son una forma de informarle sobre estrategias de corrección de vulnerabilidades de seguridad y actualizaciones aplicables para el software afectado. Para obtener más información, consulte nuestra documentación sobre [seguridad, privacidad de datos y cumplimiento](/docs/security), o visite el [sitio web de seguridad de New Relic](https://newrelic.com/security).

## Obtener notificación de seguridad [#notifications]

Para recibir notificaciones sobre nuevos boletines, suscribir al [<Icon name="fe-rss" /> Fuente RSS](/docs/security/new-relic-security/security-bulletins/feed.xml).

## Calificaciones de vulnerabilidades de seguridad [#severity\_ratings][#severity_ratings]

En New Relic, utilizamos cuatro niveles para calificar las vulnerabilidades de seguridad.

<table>
  <thead>
    <tr>
      <th width={100}>
        Clasificación
      </th>

      <th>
        Descripción
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Crítico
      </td>

      <td>
        A vulnerabilidades en un producto o servicio de New Relic que podrían explotarse para comprometer la confidencialidad o integridad de sus datos.
      </td>
    </tr>

    <tr>
      <td>
        Alto
      </td>

      <td>
        Es probable que New Relic reciba información atípica o no intencionada, lo que podría comprometer la confidencialidad o integridad de sus datos.
      </td>
    </tr>

    <tr>
      <td>
        Medio
      </td>

      <td>
        New Relic podría recibir información atípica o no deseada, pero el riesgo de compromiso se mitiga mediante la configuración predeterminada o las prácticas de seguridad estándar.
      </td>
    </tr>

    <tr>
      <td>
        Bajo
      </td>

      <td>
        New Relic puede recibir información atípica o no intencionada, pero las vulnerabilidades serían difíciles de explotar o tendrían un impacto mínimo.
      </td>
    </tr>
  </tbody>
</table>

## Informar vulnerabilidades de seguridad a New Relic [#report]

New Relic está comprometido con la seguridad de nuestros clientes y sus datos. Si cree que ha encontrado vulnerabilidades de seguridad en uno de nuestros productos, servicios o sitios web, le invitamos y apreciamos mucho que lo informe a nuestro programa de divulgación coordinada. Para obtener más información, consulte nuestra documentación sobre cómo [informar vulnerabilidades de seguridad](/docs/security/security-privacy/information-security/report-security-vulnerabilities/).
