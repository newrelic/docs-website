<DNT>**Vulnerability Identifier:**</DNT> NR24-02

<DNT>**Priority:**</DNT> Alto

## Resumen

New Relic aconseja a todos los clientes que emplean New Relic Salesforce Exporter que actualicen a la versión 2.2.0, que New Relic lanzó para eliminar una [versión vulnerable de OpenSSH](https://nvd.nist.gov/vuln/detail/CVE-2024-6387) anunciada recientemente.

## Acción requerida

New Relic recomienda que los clientes que emplean New Relic Salesforce Exporter actualicen inmediatamente a la versión 2.2.0.

New Relic proporcionó los siguientes recursos para ayudar con estas actualizaciones:

* [Notas de la versión del exportador de Salesforce](https://github.com/newrelic/newrelic-salesforce-exporter/releases/tag/2.2.0)
* [Detección y actualización en el host implementado](https://github.com/newrelic/newrelic-salesforce-exporter?tab=readme-ov-file#upgrading-on-host-deployments)

Si los clientes no pueden actualizar su New Relic Salesforce Exporter, limite el acceso SSH a través de controles basados en la red para minimizar los riesgos de ataque.
