La acción de API de Azure es un ejecutor universal que le permite llamar a cualquier [API REST de Azure](https://learn.microsoft.com/en-us/rest/api/azure/) directamente desde sus workflows. Esta acción proporciona una forma flexible de integrarse con los servicios de Azure sin necesidad de acciones dedicadas para cada operación de Azure.

## Ejecutar una API de Azure [#azure-execute-api]

Esta acción le permite ejecutar cualquier operación documentada en la [referencia de la API REST de Azure](https://learn.microsoft.com/en-us/rest/api/azure/). Simplemente proporcione la ruta de la API, el método HTTP y el cuerpo de la solicitud: la acción maneja la autenticación y gestiona las operaciones de larga duración automáticamente.

      ### Ejemplo: GET usos de red

      ```yaml
        - name: getNetworkUsage
          type: action
          action: azure.execute.api
          version: 1
          inputs:
            clientId: ${{ :secrets:my-client-id }}
            clientSecret: ${{ :secrets:my-client-secret }}
            tenantId: ${{ :secrets:my-tenant-id }}
            urlPath: "subscriptions/c6f367b2-d6af-41f5-a9e3-3828fbd97241/providers/Microsoft.Network/locations/westus/usages?api-version=2025-03-01"
            method: "GET"
            selectors:
              - name: firstResponseValue
                expression: ".response.value[0] | tostring"
      ```
