Las acciones de Azure App Service le permiten administrar sus aplicaciones web directamente desde sus flujos de trabajo.

## Reiniciar un App Service [#azure-appservice-restart]

Esta acción reinicia un Azure App Service mediante la [API REST de Web Apps - Restart](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/restart?view=rest-appservice-2026-03-15).

    <TabsPageItem id="azureappservicerestartexample">
      ```yaml
      name: restartAzureWebService
      description: 'This workflow restarts an Azure App Service'
      steps:
        - name: azure_appService_restart_1
          type: action
          action: azure.appService.restart
          version: 1
          inputs:
            clientId: ${{ :secrets:azure-client-id }}
            clientSecret: ${{ :secrets:azure-client-secret }}
            tenantId: ${{ :secrets:azure-client-tenant }}
            subscriptionId: "12345678-1234-1234-1234-123456789abc"
            resourceGroupName: "my-resource-group"
            appServiceName: "my-web-app"
            softRestart: true
            synchronous: true
          next: end
      ```
