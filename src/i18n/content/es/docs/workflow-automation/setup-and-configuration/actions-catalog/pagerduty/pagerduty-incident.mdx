Esta página proporciona una referencia completa de las acciones de incidentes de PagerDuty disponibles en el catálogo de acciones de automatización de flujo de trabajo. Estas acciones le permiten gestionar incidentes de PagerDuty.

## Requisitos previos

Antes de emplear las acciones de PagerDuty en la automatización del flujo de trabajo, cerciorar de tener lo siguiente:

* Una cuenta de PagerDuty con las licencias adecuadas.
* Un token de la API de PagerDuty (ya sea un token de usuario o un token de cuenta).
* Las licencias necesarias para gestionar incidentes en su cuenta PagerDuty.
* Para el token de cuenta, deberá proporcionar la dirección de email `from` en las acciones correspondientes.
