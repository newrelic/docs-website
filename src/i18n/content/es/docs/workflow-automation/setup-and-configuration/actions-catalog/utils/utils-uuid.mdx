Esta página proporciona una referencia completa de las utilidades - uuid disponibles en el catálogo de acciones de automatización del flujo de trabajo. Estas acciones le permiten emplear utilidades de generación de uuid.

## Generar un UUID [#utils-uuid-generate]

Genere un uuid V4 compatible con RFC. Emplea el PRNG determinista del flujo de trabajo temporal, lo que lo hace seguro para su uso dentro de un flujo de trabajo. Consulte el [espacio de nombres: flujo de trabajo | Referencia API del SDK de TypeScript temporal](https://typescript.temporal.io/api/namespaces/workflow?&_ga=2.223881945.39829224.1686648936-167080156.1680123165#uuid4) para obtener más información.
