## Transformar un valor de datos de entrada a formato CSV [#utils-transform-toCSV]

Esto se emplea para transformar varios tipos de entrada (JSON, mapa) a un formato CSV.

              ```yaml
              name: nrqltocsv

              steps:
                - name: queryForLog
                  type: action
                  action: newrelic.nrql.query
                  version: 1
                  inputs:
                    accountIds: ['${{ .workflowInputs.accountId }}']
                    query: ${{ .workflowInputs.nrql }}

                - name: csv1
                  type: action
                  action: utils.transform.toCSV
                  version: 1
                  inputs:
                    data: ${{ .steps.queryForLog.outputs.results | tostring }}
                    selectors:
                      - name: csv
                        expression: ".csv"
              ```
