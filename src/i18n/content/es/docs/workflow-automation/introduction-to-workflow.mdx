La automatización del flujo de trabajo conecta New Relic con sus herramientas existentes para automatizar la respuesta a incidentes, la gestión de la infraestructura y las tareas operativas sin escribir código. Cree un flujo de trabajo que responda a alertas, se integre con AWS y Slack, gestione aprobaciones y ejecute una lógica de automatización compleja.

**Activar el flujo de trabajo de tres maneras:**

* **De alertas**: Inicia automáticamente el flujo de trabajo cuando se incumple la condición de alerta (ver [Enviar notificación desde flujo de trabajo](/docs/workflow-automation/setup-and-configuration/create-destinations#trigger-from-alerts))

* **Bajo demanda**: active manualmente el flujo de trabajo a través API cuando sea necesario

* **Según lo programado**: ejecuta flujo de trabajo automáticamente en momentos específicos mediante expresiones cron

  <img title="Create and manage workflows in New Relic" alt="Screenshot showing the workflow automation interface" src="/images/workflow-automation-overview-1.webp" />

## Beneficios clave

* **Respuesta a incidentes más rápida**: cambie automáticamente el tamaño de las instancias EC2, revierta el despliegue o resetear los servicios cuando se activen alertas, sin necesidad de intervención manual.
* **Conecta tus herramientas**: Integra New Relic con AWS, Slack, Jira, ServiceNow y cualquier API HTTP para enviar datos de alerta enriquecidos donde trabaja tu equipo.
* **Agregar puntos de aprobación**: Incluir supervisión humana en el workflow crítico; aplicar confirmación del equipo antes de ampliar la infraestructura de producción o ejecutar reversiones.
* **Gestiona la complejidad**: Emplea lógica condicional, bucles y transformaciones de datos para automatizar tus procesos operativos únicos.

      Implementar flujos de trabajo preconfigurados para escenarios comunes: envío de reportes a Slack, reversión de despliegues, cambio de tamaño de instancias EC2 y más.

      1. Explore [las plantillas disponibles](/docs/workflow-automation/create-a-workflow-automation/use-a-template).

      4. desplegar y probar.
    </TabsPageItem>

    <TabsPageItem id="custom">
      Crea un flujo de trabajo desde cero empleando la interfaz de arrastrar y soltar. Encadena acciones, agrega lógica condicional e intégralo con tus herramientas.

      1. Revisa el [ejemplo de redimensionamiento automático de EC2](/docs/workflow-automation/create-a-workflow-automation/create-your-own).
      2. Comprender los patrones de flujo de trabajo (interruptores, bucles, aprobaciones).
      3. Explora [las acciones](/docs/workflow-automation/setup-and-configuration/actions-catalog/actions-catalog) disponibles.
      4. Construye de forma incremental y prueba cada paso.
    </TabsPageItem>

    <TabsPageItem id="setup">
