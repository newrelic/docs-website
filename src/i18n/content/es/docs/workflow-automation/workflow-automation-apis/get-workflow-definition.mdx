Esta API obtiene y devuelve una definición de flujo de trabajo.

## Ejemplo

Emplee el [generador de consultas](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) para crear y editar un flujo de trabajo. Consulte [el explorador de la API de NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflow(name: "my-workflow", version: 1) {
          definition {
            name
            version
            yaml
          }
        }
      }
    }
  }
}
```
