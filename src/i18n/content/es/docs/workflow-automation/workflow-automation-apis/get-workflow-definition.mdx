## Solicitud de muestra

Emplee el [generador de consultas](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) para crear y editar un flujo de trabajo. Consulte [el explorador de la API de NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflow(name: "my-workflow", version: 1) {
          definition {
            name
            version
            yaml
          }
        }
      }
    }
  }
}
```

## Ejemplo de respuesta

La consulta devuelve:

* `definition.name`: El nombre de la definición del flujo de trabajo
* `definition.version`: El número de versión de la definición del flujo de trabajo
* `definition.yaml`: La definición del flujo de trabajo YAML codificada en base64
