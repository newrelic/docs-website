## Solicitud de muestra

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        schedules {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
            schedule {
              id
            }
          }
        }
      }
    }
  }
}
```

## Ejemplo de respuesta

La consulta devuelve:

* `schedules.results`: Una matriz de objetos de programación, cada uno de los cuales contiene:

  * `definition.name`: El nombre de la definición del flujo de trabajo
  * `definition.scope.id`: El ID de la cuenta
  * `definition.scope.type`: El tipo de alcance
  * `definition.version`: La versión de la definición del flujo de trabajo
  * `schedule.id`: El identificador único de la programación
