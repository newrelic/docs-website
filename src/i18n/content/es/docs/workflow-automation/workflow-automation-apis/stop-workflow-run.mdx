Esto interrumpe un flujo de trabajo en curso. Devuelve un `runId` vacío si no se encuentra el flujo de trabajo.

## Ejemplo

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```
