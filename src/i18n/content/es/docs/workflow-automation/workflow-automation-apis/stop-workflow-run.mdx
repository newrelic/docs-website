## Solicitud de muestra

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```

## Ejemplo de respuesta

La mutación devuelve:

* `runId`: El identificador único de la ejecución del flujo de trabajo detenido. Devuelve un `runId` vacío si no se encuentra el flujo de trabajo.
