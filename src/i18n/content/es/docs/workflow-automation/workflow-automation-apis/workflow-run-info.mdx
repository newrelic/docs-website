## Solicitud de muestra

Emplee el [generador de consultas](https://one.newrelic.com/nerdgraph-graphiql?state=03119d23-7b51-7b7a-1c99-d2cd026300e5) para crear y editar un flujo de trabajo. Consulte [el explorador de la API de NerdGraph](/docs/apis/nerdgraph/get-started/nerdgraph-explorer).

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflowRunInfo(runId: "8911c7e4-5a1f-4316-90c8-67adde72dd1c") {
          logUrl
          status
        }
      }
    }
  }
}
```

## Ejemplo de respuesta

La consulta devuelve:

* `logUrl`: URL para ver los logs de ejecución del flujo de trabajo
* `status`: El estado actual de la ejecución del flujo de trabajo
