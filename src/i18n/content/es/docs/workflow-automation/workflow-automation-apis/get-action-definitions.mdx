## Solicitud de muestra

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        actionDefinitions {
          results {
            description
            inputs {
              name
              required
              secretAllowed
            }
            name
            outputs {
              name
            }
            timeout {
              action
            }
            version
          }
        }
      }
    }
  }
}
```

## Ejemplo de respuesta

La consulta devuelve:

* `actionDefinitions.results`: Una matriz de definiciones de acción, cada una de las cuales contiene:

  * `name`: El nombre de la acción (por ejemplo, `aws.ec2.rebootInstances`)
  * `description`: Lo que hace la acción
  * `version`: La versión de la acción
  * `inputs`: Matriz de parámetros de entrada con las propiedades `name`, `required` y `secretAllowed`
  * `outputs`: Matriz de campos de salida con la propiedad `name`
  * `timeout.action`: Tiempo de espera en segundos
