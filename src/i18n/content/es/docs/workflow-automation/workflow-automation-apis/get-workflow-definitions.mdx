Esta API obtiene y devuelve una lista de todos los esquemas de definición de flujo de trabajo dentro del alcance dado.

<Callout variant="important">
  La definición del flujo de trabajo YAML no está disponible con esta consulta. Un YAML de definición de flujo de trabajo solo se devuelve usando una consulta `GetWorkflowDefinition` `/{actor/account(:id)/workflowAutomation/workflow(:name, :version)}`.
</Callout>

## Ejemplo

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflows(cursor: "xyz123") {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
          }
        }
      }
    }
  }
}
```

## Respuesta

La consulta devuelve:

* `workflows.results`: Una matriz de contornos de definición de flujo de trabajo, cada uno de los cuales contiene:

  * `definition.name`: El nombre de la definición del flujo de trabajo
  * `definition.scope.id`: El ID de la cuenta
  * `definition.scope.type`: El tipo de alcance
  * `definition.version`: El número de versión de la definición del flujo de trabajo
