Se emplea para actualizar una definición de flujo de trabajo existente. Devuelve un error si la definición del flujo de trabajo no existe previamente.

## Ejemplo

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```
