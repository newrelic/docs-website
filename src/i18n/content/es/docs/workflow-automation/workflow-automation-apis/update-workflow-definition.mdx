## Ejemplo

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```

## Respuesta

La mutación devuelve:

* `definition.name`: El nombre de la definición del flujo de trabajo actualizado
* `definition.version`: El nuevo número de versión de la definición del flujo de trabajo
* `definition.yaml`: La definición del flujo de trabajo YAML codificada en base64
