Escenarios de automatización comunes para la automatización del flujo de trabajo, progresando desde patrones básicos a avanzados. Úselo como punto de partida o explore la [biblioteca de plantillas](/docs/workflow-automation/create-a-workflow-automation/use-a-template) para obtener soluciones listas para implementar.

## Plantilla disponible flujo de trabajo

Acceda a las plantillas directamente en la interfaz de usuario de New Relic Workflow Automation:

1. Vaya a **All Capabilities &gt; Workflow Automation**
2. Haz clic en **Create workflow**
3. Seleccione **Use a template**
4. Explora la biblioteca de plantillas y elige un flujo de trabajo que coincida con tu caso de uso

Las plantillas incluyen:

* Pasos y lógica de flujo de trabajo preconfigurados
* Ejemplo de parámetro de entrada
* Credenciales e integración requeridas
* Documentación de los resultados esperados

Use las plantillas tal como están o personalícelas para que se ajusten a requisitos específicos.
