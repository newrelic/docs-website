Utilice `${{ .workflowInputs.variableName }}` para pasar valores dinámicos en tiempo de ejecución en las URL de los endpoints y otros campos de entrada.

**Lo que hace este enfoque:**

* Aceptar valores de entrada dinámicos cuando se activa el flujo de trabajo
* Haga referencia a los valores de entrada usando la sintaxis de plantilla en todos los pasos del flujo de trabajo
* Habilite la reutilización del flujo de trabajo con diferentes parámetros

## Ejemplo

```json
  {
    "inputs": [
      {
        "key": "urlParams",
        "value": "{\"filter\": \"active\"}"
      },
      {
        "key": "headers",
        "value": "{\"Api-Key\": \"your-api-key\"}"
      }
    ]
  }
```
