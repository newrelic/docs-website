## Problema

Obtienes este error de salida cuando sigues la instalación guiada.

```shell
  Unable to connect to the server: dial tcp [7777:777:7777:7777:77::77]:443: i/o timeout
```

## Solución

Esto indica que está experimentando un problema de conexión de red entre el cliente de Kubernetes y el servidor de API de Kubernetes. Cerciorar de que su cliente de Kubernetes pueda conectarse a su servidor de API de Kubernetes antes de ejecutar nuevamente la instalación guiada.
