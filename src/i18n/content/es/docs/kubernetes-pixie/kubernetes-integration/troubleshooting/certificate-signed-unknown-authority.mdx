## Problema

Estás recibiendo el mensaje `x509: certificate signed by unknown authority`.

## Solución

Debe asegurarse de que sus certificados firmados estén configurados correctamente. Para hacerlo, puede agregar las siguientes variables de entorno en el contenedor YML del agente:

```yml
- name: NRIA_CA_BUNDLE_DIR 
  value: YOUR_CA_BUNDLE_DIR
- name: NRIA_CA_BUNDLE_FILE  
  value: YOUR_CA_BUNDLE_NAME
```
