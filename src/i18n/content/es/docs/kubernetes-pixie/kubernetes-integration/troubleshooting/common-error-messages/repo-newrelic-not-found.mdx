## Problema

Verá este mensaje de error durante la [instalación de la integración de Kubernetes](/install/kubernetes/) con Helm o Manifest.

```shell
  repo newrelic not found
```

## Solución

Agregue el repositorio newrelic a su gráfico de Helm ejecutando este comando:

```shell
  helm repo add newrelic https://helm-charts.newrelic.com
```
