Para comprobar el estado de un clúster con Pixie instalado, ejecute el [script de diagnóstico](https://github.com/newrelic-experimental/k8s-diag-utilities/tree/main/pixie-diag).

Este sencillo script bash ejecuta comandos estándar Kubernetes en el namespace en el que está instalado Pixie. El resultado de este script se puede utilizar para realizar una investigación más profunda por su cuenta o con el soporte de New Relic.
