Para determinar qué versión de la integración de New Relic Pixie está instalada en su clúster, siga estos pasos:

1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>.

2. Seleccione <DNT>**Query your data**</DNT>.

3. Copie la siguiente consulta y haga clic en <DNT>**RUN**</DNT>.

   ```sql
   FROM K8sContainerSample SELECT latest(containerImage) WHERE containerName LIKE '%newrelic-pixie%' FACET clusterName
   ```

Esta consulta genera una lista de nombres de clústeres con sus versiones de integración para la cuenta seleccionada:
