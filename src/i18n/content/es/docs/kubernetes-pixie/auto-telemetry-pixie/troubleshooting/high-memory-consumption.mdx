## Problema

Ha completado el [procedimiento de instalación](/docs/kubernetes-pixie/auto-telemetry-pixie/install-auto-telemetry-pixie) para la integración de Pixie de New Relic y le gustaría reducir la cantidad de memoria que utiliza Pixie.

## Solución

Pixie [utiliza la memoria para dos propósitos diferentes](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#memory-usage): recopilar telemetry data y almacenar estos datos en un clúster a corto plazo. Si desea especificar una cantidad diferente de memoria para el almacenamiento a corto plazo, puede [configurar el límite de memoria de almacenamiento a corto plazo](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#reduce-data-storage) durante la instalación.
