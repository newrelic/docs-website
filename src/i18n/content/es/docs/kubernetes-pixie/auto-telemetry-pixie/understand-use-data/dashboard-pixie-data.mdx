Obtenga más información sobre el modelo de almacenamiento híbrido para la integración de New Relic Pixie [aquí](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/overview). Después de la instalación, puedes ver tus datos de Pixie con nuestra característica <InlinePopover type="dashboards"/>.

## Panel de control Pixie prediseñado [#pixie-dashboards]

Para instalar un dashboard prediseñado para datos de Pixie:

1. Navegue a la New Relic [](https://one.newrelic.com/dashboards)página del panel .

2. Seleccione el botón

   <DNT>
     **Browse pre-built dashboards**
   </DNT>

   .

3. Busque "Pixie" y elija una de las opciones disponibles.

4. Seleccione el botón

   <DNT>
     **Install this quickstart**
   </DNT>

   .

## Panel de control personalizado de Pixie [#custom-dashboards]

El widget de visualización en el panel de Pixie prediseñado se completa con datos de NRQL consulta. Seleccione dashboard widgetel menú de tres puntos de cualquier para ver la consulta que proporciona los datos. Para aprender a escribir su propia consulta para datos persistentes de Pixie, consulte la página [Consulta de datos de Pixie](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/query-pixie-data) .

Obtenga más información sobre cómo personalizar un dashboard prediseñado o crear su propio panel personalizado [aquí](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards).
