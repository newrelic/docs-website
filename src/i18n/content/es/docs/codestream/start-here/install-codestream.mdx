Puede instalar la extensión CodeStream para su IDE específico.

<CollapserGroup>
  <Collapser
    className="freq-link"
    id="vscode"
    title="Código de estudio visual"
  >
    Descargue e instale CodeStream para [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream). También puede instalarlo directamente en Visual Studio Code a través del mercado de extensiones.
  </Collapser>

  <Collapser
    className="freq-link"
    id="visual-studio"
    title="Estudio visual"
  >
    Descargue e instale CodeStream para [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22). También puede instalarlo directamente en Visual Studio a través del mercado de extensiones.
  </Collapser>

  <Collapser
    className="freq-link"
    id="jetbrains"
    title="JetBrains"
  >
    Descargue e instale CodeStream para [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream). También puede instalarlo desde el menú del complemento JetBrains.
  </Collapser>
</CollapserGroup>

## Iniciar sesión [#signin]

Una vez que tenga la extensión CodeStream instalada en su IDE, inicie sesión con sus credenciales de New Relic.
