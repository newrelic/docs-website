Dashboards son fundamentales para tu experiencia New Relic y ahora tienes acceso a todos ellos directamente desde tu IDE. Busque el ícono <DNT>**Dashboards**</DNT> en la navegación global de CodeStream para buscar cualquier dashboard o acceder a sus favoritos.

<img title="Dashboard search." alt="A screenshot displaying the ability to search for dashboards." src="/images/codestream_screenshot-full_search-dashbaords.webp" />

Para abrir un dashboard, simplemente haga clic en él. No es necesario cambiar a su navegador sitio web.
