This guide shows you how to uninstall the <DNT>**New Relic ABAP component (NEWR)**</DNT> from your SAP central system. You might need to do this when decommissioning monitoring or before upgrading to a new version.

<Callout variant="caution">
  Uninstalling the <DNT>**New Relic ABAP component**</DNT> will stop all monitoring data collection and remove the component from your SAP system. Historical data in New Relic will be preserved, but no new data will be collected. This action cannot be undone from the uninstall process.
</Callout>

## Requisitos previos

* Administrator access to <DNT>**SAP system**</DNT>
* Access to <DNT>**client 000**</DNT>
* Respaldo de cualquier configuración personalizada (si es necesario para una futura reinstalación)

## Desinstale el componente ABAP

Para desinstalar el componente ABAP del sistema central:

1. Log in to <DNT>**client 000**</DNT>.
2. Ingrese el código de transacción `SAINT`.
3. Haga clic en la pestaña <DNT>**Uninstallable Components**</DNT> .

<img title="ABAP component on central system" alt="The SAINT transaction showing the NEWR component in the Uninstallable Components list" src="/images/ABAP-component-on-cms.webp" width="700" />

4. Select the <DNT>**NEWR**</DNT> component and verify the version number matches your installed version.
5. Haga clic en <DNT>**Start**</DNT>.
6. Haga clic en la marca de verificación verde para confirmar en cada pantalla subsiguiente.
7. Click <DNT>**Finish**</DNT> when the process completes.
8. Verify that the <DNT>**NEWR**</DNT> component no longer appears on the <DNT>**Installed Components**</DNT> tab.

## Verificar la desinstalación

Después de completar el proceso de desinstalación:

* The <DNT>**NEWR**</DNT> component should no longer appear in the <DNT>**Installed Components**</DNT> tab
* Los trabajos de recopilación de datos en la transacción `SM37` ya no se ejecutarán
* New Relic dejará de recibir datos de monitoreo de este sistema

**Nota**: los datos históricos de monitoreo en New Relic permanecerán accesibles. Solo se elimina el componente de recopilación de datos del sistema SAP.
