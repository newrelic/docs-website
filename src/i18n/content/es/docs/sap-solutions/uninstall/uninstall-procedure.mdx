Esta guía le muestra cómo desinstalar el <DNT>**New Relic ABAP component (NEWR)**</DNT> de su sistema central de SAP. Es posible que necesite hacer esto al dar de baja el monitoreo o antes de actualizar a una nueva versión.

<Callout variant="caution">
  Desinstalar el <DNT>**New Relic ABAP component**</DNT> detendrá toda la recopilación de datos de monitoreo y eliminará el componente de su sistema SAP. Los datos históricos en New Relic se conservarán, pero no se recopilarán datos nuevos. Esta acción no se puede deshacer desde el proceso de desinstalación.
</Callout>

## Requisitos previos

* Acceso de administrador a <DNT>**SAP system**</DNT>
* Acceso a <DNT>**client 000**</DNT>
* Respaldo de cualquier configuración personalizada (si es necesario para una futura reinstalación)

## Desinstale el componente ABAP

Para desinstalar el componente ABAP del sistema central:

1. Inicie sesión en <DNT>**client 000**</DNT>.
2. Ingrese el código de transacción `SAINT`.
3. Haga clic en la pestaña <DNT>**Uninstallable Components**</DNT> .

<img title="ABAP component on central system" alt="The SAINT transaction showing the NEWR component in the Uninstallable Components list" src="/images/ABAP-component-on-cms.webp" width="700" />

4. Seleccione el componente <DNT>**NEWR**</DNT> y verifique que el número de versión coincida con su versión instalada.
5. Haga clic en <DNT>**Start**</DNT>.
6. Haga clic en la marca de verificación verde para confirmar en cada pantalla subsiguiente.
7. Haga clic en <DNT>**Finish**</DNT> cuando se complete el proceso.
8. Verifique que el componente <DNT>**NEWR**</DNT> ya no aparezca en la pestaña <DNT>**Installed Components**</DNT>.

## Verificar la desinstalación

Después de completar el proceso de desinstalación:

* El componente <DNT>**NEWR**</DNT> ya no debería aparecer en la pestaña <DNT>**Installed Components**</DNT>
* Los trabajos de recopilación de datos en la transacción `SM37` ya no se ejecutarán
* New Relic dejará de recibir datos de monitoreo de este sistema

**Nota**: los datos históricos de monitoreo en New Relic permanecerán accesibles. Solo se elimina el componente de recopilación de datos del sistema SAP.
