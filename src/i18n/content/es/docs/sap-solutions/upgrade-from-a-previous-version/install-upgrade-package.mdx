Para actualizar desde una versión anterior de New Relic Monitoreo para SAP Solutions, cerciorar de emplear los paquetes de actualización correctos específicos para su versión instalada actual.

1. Vaya a **System &gt; Status**.
2. Busque **NEWR** para identificar la versión del producto.
3. En el cliente 000, ejecute la transacción **SAINT** para actualizar el complemento ABAP.

## Copia de seguridad y actualización de configuración.

<Callout variant="tip">
  Almacene cualquier configuración del proveedor de datos modificada o proveedores de datos personalizados en una plantilla de configuración para evitar sobreescribir durante la actualización. Cerciorar de que las plantillas se almacenen antes de copiar la nueva plantilla del cliente 000, incluso si el complemento ABAP V3.0 no está instalado.
</Callout>

1. Acceda al menú `/n/NEWR/INSTALL` y navegue hasta **Update or Save Configuration Template**.

2. Ejecute `/NEWR/REP_DELETE_INS_TEMPLATE` para eliminar la plantilla de monitoreo HDB relacionada con el monitoreo de Java.

3. Configure la conexión de datos ABAP a la base de datos HANA empleando `DBACockpit`.

4. Copiar la plantilla de configuración del cliente 000.

5. Modificar la configuración del sistema de monitoreo:

   * Agrega la nueva conexión a la base de datos.
   * Eliminar referencias al Java Enterprise Connector (por ejemplo, NR\_EI\_AGENT).
   * Agregue HANA\_DB a los grupos de monitoreo.
   * Vuelva a generar las configuraciones del proveedor de datos para actualizar la configuración.
