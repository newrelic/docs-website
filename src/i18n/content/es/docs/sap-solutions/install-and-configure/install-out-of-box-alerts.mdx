---
title: Instale la política de alertas lista para usar
metaDescription: Install the out of the box alert policies
freshnessValidatedDate: never
translationType: machine
---

<Steps>
  <Step>
    Vaya a **[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**.
  </Step>

  <Step>
    Busque y haga clic en **New Relic Monitoring for SAP Solutions alert policy**.

    <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />
  </Step>

  <Step>
    En la pestaña **Alert condition**, seleccione la cuenta y cambie la política de alertas a **Enable**. Consulta [Gestionar tus políticas](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) para saber cómo gestionar las políticas.
  </Step>

  <Step>
    Regrese y busque la **Nr4sap default alert policy**. Active la condición de alerta que desea habilitar.
  </Step>
</Steps>
