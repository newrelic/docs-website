El componente ABAP de **NRM4SAP** está empaquetado como un complemento de SAP y se puede instalar mediante la transacción **SAINT** en el **client 000**.

## Archivo de instalación del complemento [#add-on-installation-file]

* **Instalación inicial**: Descargue el archivo complementario `K-xxxAOIINNEWR.SAR` a una carpeta en su PC local.
* **Actualización desde la versión anterior**: emplee el archivo llamado `K-xxxAOUINNEWR.SAP`.

## Instalar el archivo complementario de SAP [#install-sap-add-on-file]

1. Inicie sesión en el **client 000**, que es el cliente predeterminado para las tareas de configuración y administración del sistema.
2. Escriba `SAINT` en el campo de comando de SAP GUI y haga clic en **Enter**.
3. Vaya a **Installation Package** &gt; **Load Packages**.
4. Elija uno de los siguientes métodos:

<CollapserGroup>
  <Collapser id="sar-archive-from-frontend" title="Archivo SAR desde la interfaz">
    1. Navegue y abra el archivo de instalación descomprimido.
    2. En la ventana emergente **Content of the compressed file `K-xxxAOIINNEWR.SAR`**, haga clic en **Decompress**.

    <img title="Content of the compressed file" alt="A screenshot displaying the Content of the compressed file" src="/images/content-of-compressed-file-1.webp" width="" />

    3. Aparece el mensaje **File K-xxxAOIINNEWR.SAR sucessfully transferred, decompressed, and deleted**.
  </Collapser>
</CollapserGroup>

<CollapserGroup>
  <Collapser id="sar-archive-from-application-server" title="Archivo SAR desde el servidor de aplicaciones">
    1. Cerciorar de que haya un directorio de descarga bajo el directorio trans en el servidor de aplicaciones (`/usr/sap/trans/download`).
    2. Copie el archivo complementario a este directorio.
  </Collapser>
</CollapserGroup>

<CollapserGroup>
  <Collapser id="eps-files-from-application-server" title="Archivos EPS del servidor de aplicaciones">
    1. Primero, descomprima el archivo `SAR` en el archivo `PAT`. Ejecute el siguiente comando en el sistema operativo (Linux en este caso):

    ```sql
    cd /usr/sap/trans
    cp /tmp/K-300AOIINNEWR.SAR /usr/sap/trans
    SAPCAR -xvf K-300AOIINNEWR.SAR
    ```

    2. En la ventana emergente ¿ **Do you want to upload OCS Packages from the EPS inbox?**, seleccione **Yes**. Cerciorar de que el archivo OCS se cargue sin errores.
  </Collapser>
</CollapserGroup>

5. Haga clic en **Start**.

6. Seleccione los **Installable Add-on packages for your system** y haga clic en **Continue**.

7. Cuando se le pregunte **Do you want to add Modification Adjustment Transports to the queue?**, haga clic en **No**.

   <img title="Do you want to add Modification Adjustment Transports to the queue?, Click No" alt="A screenshot displaying question, Do you want to add Modification Adjustment Transports to the queue?" src="/images/modify-adjustement-transports.webp" width="700" />

<Callout variant="important">
  Haga clic en **Yes** si en la lista de embalaje se proporcionan transportes de ajuste con instrucciones.
</Callout>

8. Haga clic en el<img src="/images/green-tick-button.webp" class="inline" style={{height: "25px", width: "30px"}} alt="green check button" title="Green check button" />Botón para pasar a la siguiente pantalla.

<Callout variant="important">
  Si aparece la página **Errors during Disassembling OCS Packages**, haga clic en **Ignore**.
</Callout>

9. Una vez instalado el paquete, haga clic en **Finish**. Aparece la ventana emergente **Sending the runtime analysis** ; puede elegir:

   * **Send**: abre un formulario de comentarios con los datos del análisis del tiempo de ejecución en una ventana del navegador local. Puede emplear este formulario para enviar datos a SAP.
   * **Display**: muestra primero el análisis del tiempo de ejecución y luego envía el formulario de comentarios en una ventana del navegador local.
   * **Do not send**: abandona el cuadro de diálogo sin enviar los datos de análisis en tiempo de ejecución.

10. Cerciorar de que la instalación sea exitosa en la pestaña **Installed Components**.

11. Cerrar sesión del **Client 000**.

<Callout variant="tip">
  Puede escribir **/nex** en el campo de comando de SAP GUI para cerrar sesión en el sistema.
</Callout>
