## Configurar la carga de trabajo del sistema

1. Vaya a <DNT>**All entities &gt; Workloads &gt; + Create a workload**</DNT>. Para obtener más información, consulta [Crear y definir workloads](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/).

2. Ingrese un nombre para su <DNT>**workload**</DNT> y seleccione una cuenta de la lista desplegable.

3. Junto a <DNT>**Entity Type = All**</DNT>, seleccione el ícono <DNT>**+**</DNT> e ingrese `sap.system = {SAP System SID}` en la barra de filtro.

   <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />

4. Seleccione <DNT>**+ Add this query**</DNT> para agregar la consulta a su <DNT>**workload definition**</DNT>.

5. Seleccione <DNT>**Create a workload**</DNT>.

Su nuevo <DNT>**workload**</DNT> aparece en <DNT>**All entities**</DNT> e incluye todas las entidades etiquetadas con su <DNT>**SAP system**</DNT>.

## Instalar paquete de servicio

<DNT>**Service packs**</DNT> se entregan como archivos `.SAR` con la convención de nomenclatura `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`.

1. Instale el <DNT>**service pack**</DNT> usando la transacción `SPAM`.
2. Cargue el nuevo <DNT>**configuration template**</DNT> desde <DNT>**client 000**</DNT>.
3. Vaya a <DNT>**Install New Release and Patch**</DNT> &gt; <DNT>**Copy Template Tables**</DNT> desde <DNT>**Client 000**</DNT>.
4. Genere el nuevo <DNT>**data provider configurations**</DNT> siguiendo los pasos en <DNT>**Install New Release and Patch**</DNT> &gt; <DNT>**Generate Configuration for New Data Providers**</DNT>.
5. Seleccione <DNT>**Install New Release and Patch**</DNT> &gt; <DNT>**Verify Installation Completeness**</DNT> para verificar la instalación.
