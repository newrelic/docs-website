title: Configurar la carga de trabajo del sistema
metaDescription: Information on how to setup the system workload
freshnessValidatedDate: never
translationType: machine
---

<Steps>
  <Step>
    Navegar a **All entities &gt; Workloads &gt; + Create a workload**. Consulte [Crear y definir carga de trabajo](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/) para saber más sobre cómo crear una carga de trabajo.
  </Step>

  <Step>
    Ingrese el nombre de la carga de trabajo y seleccione la cuenta requerida en el cuadro de lista desplegable.
  </Step>

  <Step>
    Haga clic en el ícono **+** junto a la opción **Entity Type = All** e ingrese la condición `sap.system = {SAP System SID}` en la barra de filtro.

    <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />
  </Step>

  <Step>
    Haga clic en **+ Add this query** para agregar la consulta a la definición de carga de trabajo.
  </Step>

  <Step>
    Haga clic en **Create a workload** para crear la carga de trabajo.
  </Step>

  <Step>
    La carga de trabajo será visible en la UI de **All entities** e incluye todas las etiquetas de entidades con el sistema SAP.
  </Step>
</Steps>
