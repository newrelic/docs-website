Esta guía le muestra cómo obtener una clave de servicio de la SAP Business Technology Platform (BTP) para la instancia de Process Integration Runtime. Necesitará esta clave de servicio para configurar el monitoreo de SAP Process Integration.

## Requisitos previos

* Acceso a <DNT>**BTP subaccount**</DNT> con los permisos apropiados
* Confirmación de su <DNT>**BTP administrator**</DNT> sobre <DNT>**Space**</DNT> y <DNT>**Grant-types**</DNT>

## Cree la instancia de Process Integration Runtime

1. Inicie sesión en el <DNT>**BTP (Business Technology Platform) subaccount**</DNT>.

2. En el <DNT>**BTP cockpit**</DNT>, haga clic en <DNT>**Instances and Subscriptions**</DNT>.

3. Haga clic en <DNT>**Create**</DNT> en la parte superior derecha.

4. En la pantalla emergente, seleccione:

   * <DNT>**Service**</DNT>: <DNT>**SAP Process Integration Runtime**</DNT>

   * <DNT>**Plan**</DNT>: <DNT>**API**</DNT>

   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>

   * <DNT>**Space**</DNT>: Consulta con tu <DNT>**BTP administrator**</DNT>

   * <DNT>**Instance Name**</DNT>: siga la convención de nomenclatura de su organización

     <img title="Process Integration Instance" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields" src="/images/process-integration-instance.webp" width="600" />

5. Haga clic en <DNT>**Next**</DNT> y luego seleccione:

   * <DNT>**Roles**</DNT>: `MonitoringDataRead`

   * <DNT>**Grant-types**</DNT>: <DNT>**Default Client Credentials**</DNT> (confirma con tu <DNT>**BTP administrator**</DNT> y equipo de seguridad si se usan otros tipos)

     <img title="BTP Roles and Grant Types" alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected" src="/images/btp-roles-grant-types.webp" width="600" />

6. Haga clic en <DNT>**Next**</DNT> para revisar los detalles.

7. Haga clic en <DNT>**Create**</DNT>.

## Generar y descargar la clave de servicio

1. Una vez creado, ubique el servicio en la pestaña <DNT>**Instances**</DNT>.

   <img title="BTP Instance" alt="The Instances tab showing the created Process Integration Runtime instance" src="/images/btp-instance.webp" width="600" />

2. Haga clic en los puntos suspensivos (`...`) en la fila de su instancia recién creada y luego seleccione <DNT>**Create Service Key**</DNT>.

3. Ingrese un <DNT>**Service Key Name**</DNT> y seleccione <DNT>**Key Type**</DNT>: `ClientId/Secret`.

   <img title="BTP New Service Key" alt="The Create Service Key dialog showing Service Key Name and Key Type fields" src="/images/btp-new-service-key.webp" width="600" />

4. Haga clic en <DNT>**Create**</DNT>.

5. Haga clic en los puntos suspensivos (`...`) en la fila de su <DNT>**service key**</DNT> y luego seleccione <DNT>**Download**</DNT>.

   <img title="BTP Download Service Key" alt="The service key row showing the ellipsis menu with the Download option" src="/images/btp-download.webp" width="600" />

El <DNT>**service key file**</DNT> se descarga como un <DNT>**JSON file**</DNT>. Almacénelo de forma segura; necesitará las credenciales de este archivo para configurar su entorno de monitoreo.
