1. Log in to the <DNT>**BTP subaccount**</DNT>.

2. In the <DNT>**BTP cockpit**</DNT>, click <DNT>**Instances and Subscriptions**</DNT>.

3. Click <DNT>**Create**</DNT> at the top right.

4. En la pantalla emergente, seleccione:

   * <DNT>**Service**</DNT>: <DNT>**API Management, API Portal**</DNT>
   * <DNT>**Plan**</DNT>: <DNT>**apiportal-apiaccess**</DNT>
   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>
   * <DNT>**Space**</DNT>: Check with your <DNT>**BTP administrator**</DNT>
   * <DNT>**Instance Name**</DNT>: Follow your organization naming convention

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-management-api.webp" width="600" />

5. Consulte la exención de responsabilidad de costos.

6. Click <DNT>**Next**</DNT>, then select <DNT>**Roles**</DNT>: `APIPortal.Administrator`.

   <img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-roles-api.webp" width="600" />

7. Click <DNT>**Next**</DNT> to review the details.

8. Haga clic en <DNT>**Create**</DNT>.

9. Once created, locate the service under the <DNT>**Instances**</DNT> tab.

10. Click the ellipsis (`...`) next to your instance, then click <DNT>**Create Service Key**</DNT>.

11. Enter a <DNT>**Service Key Name**</DNT> and <DNT>**Credential Type**</DNT>: `binding-secret` in <DNT>**JSON format**</DNT>.

<img title="SAP Cloud ALM API" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields for SAP Cloud ALM API" src="/images/sap-cloud-servicekey-api.webp" width="600" />

12. Click <DNT>**Create**</DNT>, then download by clicking the ellipsis (`...`) next to your <DNT>**service key**</DNT>.
