El monitoreo de New Relic para SAP incluye una lista de componentes de monitoreo listos para usar, incluyendo dashboards, políticas de alertas y el IDOC Explorer. Siga las siguientes instrucciones para localizarlos e instalarlos.

## Antes de que empieces

Para realizar estas tareas, debe tener privilegios de administrador de New Relic en la cuenta.

<Callout variant="important">
  Para el dashboard de <DNT>**SAP Cockpit**</DNT>, debe [crear una tabla de búsquedas](#create-lookup-table) denominada `NR_LKUP_SYSTEM` antes de la instalación.
</Callout>

## Instalar el tablero

### Crear tabla de búsquedas [#create-lookup-table]

<Callout variant="tip">
  Solo se requiere para el dashboard <DNT>**SAP Cockpit**</DNT>.
</Callout>

El dashboard <DNT>**SAP Cockpit**</DNT> emplea un <DNT>**lookup table**</DNT> para <DNT>**SAP systems**</DNT> y sus entornos.

1. Cree el <DNT>**lookup table**</DNT> con dos columnas: `SYS_ID` y `Env`, y guárdelo como un <DNT>**CSV file**</DNT>. A continuación se muestra un ejemplo del archivo de búsquedas:

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. Vaya a <DNT>**Log &gt; Lookup Tables**</DNT>.

3. Haga clic en <DNT>**Add a table**</DNT>.

4. En la barra lateral <DNT>**Add a lookup table**</DNT>, nombre la tabla como <DNT>**NR\_LKUP\_SYSTEM**</DNT> y cargue el <DNT>**CSV file**</DNT>. Consulte [Tablas de búsquedas](/docs/logs/ui-data/lookup-tables-ui/) para obtener más información.

5. Para verificar el contenido de la tabla, emplee el <DNT>**query builder**</DNT> con `FROM lookup(NR_LKUP_SYSTEM) SELECT *` y luego haga clic en <DNT>**Run**</DNT>.

### Instalar los dashboards [#install-dashboards]

1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> &gt; <DNT>**Integrations &amp; Agents**</DNT> y busque <DNT>**New Relic Monitoring for SAP Solutions**</DNT> para descargar automáticamente todos los dashboard

2. Como alternativa, puede ir a <DNT>**Dashboards &gt; Recommended Dashboards**</DNT>, hacer clic en <DNT>**View all**</DNT> y buscar SAP. Consulte [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) para obtener más información.

3. Instalar estos dashboards:

   * <DNT>**SAP Dashboards**</DNT>
   * <DNT>**SAP Cockpit**</DNT>
   * <DNT>**SAP Data Collection Monitoring**</DNT>
   * <DNT>**SAP Data Ingestion Estimate**</DNT>

### Agregar filtro dashboard [#add-dashboard-filter]

1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards**</DNT>.

2. Habilitar la vinculación de facetas para el filtrado:

   * Haga clic en los puntos suspensivos (<DNT>**…**</DNT>) en la esquina superior derecha del widget
   * Seleccione <DNT>**Edit**</DNT> para abrir la pantalla de edición.
   * Active y desactive el interruptor <DNT>**Filter the current dashboard**</DNT>

3. <DNT>**Facet linking**</DNT> se puede habilitar para tablas, gráficos circulares y widget de gráfico de barras para filtrar por <DNT>**SAP system**</DNT>, <DNT>**instance**</DNT>, <DNT>**IDOC status**</DNT>, <DNT>**jobs**</DNT>, etc.

## Instalar políticas de alertas

1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**</DNT>.

2. Busque y haga clic en <DNT>**New Relic Monitoring for SAP Solutions alert policy**</DNT>.

   <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />

3. En la pestaña <DNT>**Alert condition**</DNT>, seleccione la cuenta y cambie <DNT>**alert policy**</DNT> a <DNT>**Enable**</DNT>. Consulta [Gestionar tus políticas](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) para saber cómo gestionar las políticas.

4. Regrese y busque <DNT>**Nr4sap default alert policy**</DNT>. Active el <DNT>**alert condition**</DNT> que desea habilitar.

## Instalar el Explorador de IDOC

1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT>.

2. Seleccione <DNT>**Apps &amp; Visualizations**</DNT> y busque <DNT>**IDOC Explorer**</DNT>.

3. Seleccione <DNT>**Manage access**</DNT>. Elija la cuenta para acceder a la aplicación.

   <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />

4. Haga clic en <DNT>**Update**</DNT>. El <DNT>**IDOC Explorer**</DNT> será visible al ver <DNT>**IDOC entities**</DNT>.
