## Use el menú /NEWR/INSTALL [#menu-newr-install]

1. Inicie sesión en el cliente de producción del sistema de monitoreo central.
2. Navegue al <DNT>**SAP Easy Access root menu**</DNT> escribiendo `/n` en el campo de comando y presionando <DNT>**Enter**</DNT>.
3. Acceda al menú <DNT>**`/NEWR/INSTALL`**</DNT> escribiendo `/n/NEWR/INSTALL` en el campo de comando.

Después de instalar el add-on en el sistema de monitoreo central:

* Cerciorar de que el instalador tenga la autorización adecuada para realizar la configuración.
* Agregue el parámetro <DNT>**`/NEWR/*`**</DNT> a los objetos de autorización de código en el `profile/role` del instalador.

<Callout variant="tip">
  Si actualiza desde una versión anterior, consulte [Actualizar desde una versión anterior](/docs/sap-solutions/upgrade/install-upgrade-package) para determinar qué pasos son necesarios.
</Callout>

## Conectar sistemas [#connect-systems]

<Callout variant="important">
  Complete cualquier paso de configuración pendiente antes de continuar.
</Callout>

Las siguientes opciones están disponibles en el menú `/NEWR/INSTALL`:

### Configurar destinos HTTP y RFC

Utilice esta opción para invocar la transacción <DNT>**`SM59`**</DNT> y verificar sus conexiones [RFC](/docs/sap-solutions/install-and-configure/additional-configuration/#create-rfc-for-monitored-systems) y [HTTPS](/docs/sap-solutions/install-and-configure/additional-configuration/#set-http-destination-to-new-relic-cloud).

### Configurar conexión a base de datos remota

* Use la transacción SAP <DNT>**DBACOCKPIT**</DNT> para verificar la configuración de las conexiones a bases de datos HANA remotas.
* Cerciorar de que se cree una conexión de base de datos para cada base de datos HANA remota que desee monitorear.
* Para obtener instrucciones detalladas, consulte [Crear conexión de base de datos](/docs/sap-solutions/install-and-configure/additional-configuration/#create-database-connection).

### Configurar conexiones a los extremos de la API de NR

1. Haga clic en <DNT>**New Entries**</DNT>.

2. Ingresa los detalles de tu cuenta de New Relic:

   * <DNT>**New Relic Account ID**</DNT> (opcionalmente con el prefijo SAP-)
   * <DNT>**API Ingest Key**</DNT>
   * <DNT>**Metric Endpoint**</DNT> (consulte [Extremos de red](/docs/sap-solutions/before-you-begin/#network-endpoints))
   * <DNT>**Event Endpoint**</DNT>
   * <DNT>**Log Endpoint**</DNT>
   * <DNT>**Trace Endpoint**</DNT>

3. Haga clic en <DNT>**Save**</DNT>.

### Copiar tablas de plantilla del cliente 000

Ejecute esta opción para copiar la tabla de <DNT>**client 000**</DNT> haciendo clic en el icono de ejecución (símbolo de reloj). Confirme que aparezca el mensaje de éxito.

### Definir los sistemas monitoreados

1. Haga clic en <DNT>**New Entries**</DNT> para ingresar el <DNT>**SAP System ID**</DNT> y el número de <DNT>**Client**</DNT>.
2. En <DNT>**Data Provider Installation Information**</DNT>, especifique el <DNT>**Destination**</DNT> RFC, seleccione el <DNT>**Software Component**</DNT> SAP, ingrese el <DNT>**Database Vendor**</DNT> y proporcione el <DNT>**DB Connection Name**</DNT> (solo base de datos HANA).
3. Haga clic en <DNT>**Monitoring Group**</DNT> y seleccione <DNT>**Yes**</DNT> en el cuadro de diálogo para confirmar.
4. Seleccione los grupos de monitoreo necesarios y haga clic en el ícono de generar para completar el proceso.
5. Haga clic en <DNT>**BACK**</DNT> o en la tecla <DNT>**F3**</DNT> para ir al siguiente paso.

## Generar configuración [#generate-configurations]

### Generar configuraciones para nuevos proveedores de datos

* Emplee esta opción de menú para generar proveedores de datos para una nueva versión o parche. Para una nueva instalación, se generan todos los proveedores de datos (DP).
* En la primera pantalla, confirme los detalles del lanzamiento y del parche. Los campos vacíos indican una configuración preexistente.
* Para regenerar, seleccione <DNT>**Re-generate Data Provider Configurations**</DNT> en <DNT>**Maintain Monitored Systems**</DNT> o use <DNT>**Reset Installation Flag (For Re-install only)**</DNT> para restablecer el indicador de instalación y regenerar la configuración.

<img title="Maintain Monitored Systems" alt="A screenshot displaying the SAP maintain monitored systems" src="/images/maintain-monitored-systems.webp" width="500" />

### Verificar que la instalación esté completa

Haga clic en <DNT>**Verify Installation Completeness**</DNT> para asegurarse de que todo esté configurado correctamente. Si algún elemento muestra un indicador rojo, haga clic en el botón de ejecución en la columna <DNT>**Action Needed**</DNT>.

## Verifique la configuración [#verify-configuration]

1. Para garantizar que los proveedores de datos estén configurados correctamente, abra la carpeta <DNT>**Data Provider Registry Maintenance**</DNT>.
2. Haga clic en <DNT>**Data Provider Registry**</DNT> o emplee la transacción <DNT>**`/n/NEWR/REGISTRY`**</DNT>.
3. Seleccione un <DNT>**Data Provider Name**</DNT> y verifique la configuración de parámetros haciendo clic en <DNT>**Configure**</DNT>.
4. Pruebe el proveedor de datos haciendo clic en <DNT>**TestDP**</DNT>. La prueba debería completarse con éxito, lo que confirma que el proveedor de datos está configurado correctamente.
