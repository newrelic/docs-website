<Steps>
  <Step>
    Vaya a **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**.
  </Step>

  <Step>
    Seleccione **apps &amp; visualizations** y busque **IDOC Explorer**.
  </Step>

  <Step>
    Seleccione **Manage access**. Elija la cuenta para acceder a la aplicación.

    <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />
  </Step>

  <Step>
    Haga clic en **Update**. El Explorador de IDOC será visible al visualizar la entidad IDOC.
  </Step>
</Steps>
