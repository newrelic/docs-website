## Solucionar problemas de disponibilidad del servidor

Los proveedores de datos `NR_SAP_INSTANCE_AVAILABILITY` y `NR_SAP_SERVER_LIST` dependen de llamadas SOAP a servicios sitio web `SAPCONTROL`. Para que esto funcione, se debe configurar el perfil `DEFAULT.PFL` en los servidores monitoreados.

1. Confirme que la siguiente línea existe en el archivo `DEFAULT.PFL` :

`service/protectedwebmethods = SDEFAULT`

2. Si falta esta línea, agregue el siguiente parámetro:

`service/protectedwebmethods = SDEFAULT -GetQueueStatistic -ABAPGetWPTable -EnqGetStatistic -GetProcessList -GetEnvironment -GetVersionInfo -GetSystemInstanceList`

3. Resetear el servicio SAP con los comandos `stopsap sapstartsrv` y `startsap sapstartsrv`.

4. Confirme que la configuración funciona ejecutando el comando `sapcontrol -prot GSOAP_HTTP -nr 00 -function GetSystemInstanceList`.
