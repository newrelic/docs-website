Para actualizar desde una versión anterior de New Relic Monitoring for SAP Solutions, use el paquete de actualización que coincida con su versión instalada actual.

## Requisitos previos

* <DNT>**Administrator access**</DNT> a <DNT>**SAP system**</DNT>
* Versión actual identificada, consulte la [sección Identificar versión actual](#identify-current-version)
* Respaldo de la configuración existente
* <DNT>**upgrade package**</DNT> correcto para su versión instalada

## Identificar la versión actual [#identify-current-version]

1. Navegue hasta <DNT>**System &gt; Status**</DNT>.
2. Busque <DNT>**NEWR**</DNT> para identificar el <DNT>**product version**</DNT>.
3. En <DNT>**client 000**</DNT>, ejecute la transacción `SAINT` para actualizar el <DNT>**ABAP add-on**</DNT>.

## Respaldar y actualizar la configuración [#backup-and-update-configuration]

<Callout variant="tip">
  Guarde cualquier <DNT>**data provider configurations**</DNT> o <DNT>**custom data providers**</DNT> modificado en un <DNT>**configuration template**</DNT> para evitar la sobrescritura durante la actualización. Asegúrese de que las plantillas estén guardadas antes de copiar la nueva plantilla de <DNT>**client 000**</DNT>, incluso si el <DNT>**V3.0 ABAP Add-on**</DNT> no está instalado.
</Callout>

1. Acceda al menú `/n/NEWR/INSTALL` y navegue hasta <DNT>**Update or Save Configuration Template**</DNT>.

2. Ejecute `/NEWR/REP_DELETE_INS_TEMPLATE` para eliminar el <DNT>**HDB monitoring template**</DNT> relacionado con <DNT>**Java monitoring**</DNT>.

3. Configure el <DNT>**ABAP data connection**</DNT> en el <DNT>**HANA database**</DNT> usando `DBACockpit`.

4. Copie el <DNT>**configuration template**</DNT> de <DNT>**client 000**</DNT>.

5. Modifique <DNT>**monitoring system configuration**</DNT>:

   * Agregar el nuevo <DNT>**DB connection**</DNT>.
   * Elimine las referencias a <DNT>**Java Enterprise Connector**</DNT> (por ejemplo, <DNT>**NR\_EI\_AGENT**</DNT>).
   * Agrega <DNT>**HANA\_DB**</DNT> a <DNT>**monitoring groups**</DNT>.
   * Vuelve a generar <DNT>**data provider configurations**</DNT> para actualizar la configuración.
