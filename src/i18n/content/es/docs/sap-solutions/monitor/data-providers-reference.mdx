Los proveedores de datos son componentes que recopilan y transmiten datos de monitoreo desde su <DNT>**SAP systems**</DNT> a New Relic. Esta referencia técnica enumera todos los proveedores de datos disponibles en <DNT>**New Relic Monitoring for SAP Solutions**</DNT> (<DNT>**NRM4SAP**</DNT>). Utilice esto al configurar grupos de monitoreo específicos. Los proveedores de datos se organizan por grupos de monitoreo, que usted selecciona al [configurar los sistemas monitoreados](/docs/sap-solutions/install-and-configure/configure-new-relic-abap-add-on-package/#connect-systems).

## Quién debería usar esta referencia

Utilice esta referencia cuando:

* Configuración de grupos de monitoreo para <DNT>**SAP systems**</DNT>
* Resolución de problemas de recopilación de datos
* Comprender qué datos son recopilados por cada grupo de monitoreo
* Personalización de la configuración de proveedores de datos

<Callout variant="tip">
  Estas tablas contienen especificaciones técnicas detalladas. La mayoría de los usuarios solo necesita consultar las columnas **DP\_NAME** y **OUTPUT\_TYPE**.

  Las tablas en esta referencia se pueden desplazar horizontalmente para ver todas las columnas. Los proveedores de datos marcados con &quot;S&quot; en la columna ESTADO están suspendidos de forma predeterminada.
</Callout>

## Comprensión de los atributos del proveedor de datos

Cada proveedor de datos tiene los siguientes atributos técnicos:

* **MANDT**: Número de cliente (típicamente 001)
* **DP\_NAME**: Identificador único para el proveedor de datos
* **OUTPUT\_TYPE**: tipo de telemetry data (METRIC, EVENT, LOG, TRACE, CONFIG, BTP-LOG)
* **COMMUNICATOR**: <DNT>**ABAP class**</DNT> responsable de la coordinación de la recopilación de datos (para la resolución de problemas avanzada)
* **DATAPROVIDER**: <DNT>**ABAP class**</DNT> que recupera los datos (para resolución de problemas avanzada)
* **FINISHER**: <DNT>**ABAP class**</DNT> que procesa y transmite los datos (para la resolución avanzada de problemas)
* **STATUS**: indicador de estado (S = Suspendido, en blanco = Activo)
* **DP\_GROUP**: Asignación del grupo de monitoreo
* **METRIC\_TYPE**: para métricas, indica el tipo de agregación (GAUGE, COUNT, SUMMARY)
* **PERIOD**: Valor numérico de la frecuencia de recopilación
* **PERIOD\_UNIT**: Unidad de tiempo para la recopilación (MIN, HRS, DAY)
* **SKIP\_RT**: Omitir indicador de tiempo de ejecución (X = Sí)
* **RESERVED**: Reservado para uso futuro
