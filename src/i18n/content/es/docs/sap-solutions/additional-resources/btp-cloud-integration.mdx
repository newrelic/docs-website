Configure SAP BTP Integration Suite (integración en la nube) como middleware para transmitir datos de monitoreo a New Relic cuando la conectividad directa no esté disponible.

## Descripción general

El programa collection manager proporciona una opción para enviar datos a New Relic mediante SAP Cloud Integration (SAP CPI) como middleware en lugar de transmisión directa.

| **Método**            | **Descripción**                                             |
| --------------------- | ----------------------------------------------------------- |
| Directo               | Transmisión directa desde SAP a New Relic (método estándar) |
| SAP Cloud Integration | Uso de SAP CPI como middleware                              |

## Pasos de configuración

<Steps>
  <Step>
    ## Determine el método de autorización

    Seleccione el tipo de autorización que utilizará para el iFlow:

    * <DNT>**ClientId/Secret**</DNT> (OAuth2)
    * <DNT>**Certificate**</DNT>
    * <DNT>**External Certificate**</DNT>
  </Step>

  <Step>
    ## Crear instancia de servicio y clave

    1. Create a service instance for <DNT>**SAP Process Integration Runtime**</DNT>.
    2. Assign a <DNT>**service key**</DNT> or <DNT>**certificate**</DNT> using your selected authorization method.
    3. Download the <DNT>**service key**</DNT> for later use.

    For detailed instructions, see [Obtain Service Key for <DNT>**BTP Monitoring**</DNT>](/docs/sap-solutions/install-and-configure/obtain-service-key/process-integration-runtime).
  </Step>

  <Step>
    ## Configure y despliegue iFlow

    Use the New Relic delivered <DNT>**iFlow**</DNT> as a template:

    1. Copy the template <DNT>**iFlow**</DNT>.
    2. Actualice el método de autorización para que coincida con su selección del Paso 1.
    3. Asegúrese de que los siguientes mapeos estén configurados:

    ### Encabezados permitidos

    Configure <DNT>**allowed headers**</DNT> to contain:

    ```
    API-Key|targetURL
    ```

    ### Método de autorización

    * For <DNT>**ClientId/Secret**</DNT> or <DNT>**Certificate**</DNT>: Select <DNT>**User Role**</DNT>
    * For <DNT>**External Certificate**</DNT>: Select <DNT>**Client Certificate**</DNT> and add <DNT>**root certificates**</DNT>

    ### Modificador de contenido de la solicitud

    Configure <DNT>**exchange properties**</DNT> to hold:

    * <DNT>**targeturl**</DNT>
    * <DNT>**apikey**</DNT>

    ### Script de Groovy

    Add a <DNT>**script**</DNT> to dynamically set headers:

    ```groovy
    import com.sap.gateway.ip.core.customdev.util.Message;
    def Message processData(Message message) {

      message.setHeader("API-Key", message.getProperty("apikey"));
      message.setHeader("Content-Type", "application/json");
      message.setHeader("Accept", "application/json");

      message.setHeader("CamelHttpUri", message.getProperty("targeturl"));

      def turl = message.getHeader("CamelHttpUri", String);

      def logText = "---Headers---\n";

      message.getHeaders().each { k, v -> logText += "\${k} = \${v}\n" };
      messageLogFactory.getMessageLog(message)?.addAttachmentAsString("AllHeaders", logText, "text/plain");

      messageLogFactory.getMessageLog(message)?.addAttachmentAsString("URL", turl, "text/plain");

      return message;
    }
    ```

    ### Modificador de contenido de la respuesta

    Configure the <DNT>**message body**</DNT> to return:

    ```json
    {
      "statusCode": "\${header.CamelHttpResponseCode}",
      "response": \${body}
    }
    ```

    ### Configuración de conexión HTTP

    * <DNT>**Address**</DNT>: `\${header.CamelHttpUri}`
    * <DNT>**Method**</DNT>: <DNT>**POST**</DNT>
    * <DNT>**Timeout**</DNT>: <DNT>**60000 ms**</DNT>
    * <DNT>**Request Headers**</DNT>: `*`
    * <DNT>**Response Headers**</DNT>: `*`

    4. Deploy the <DNT>**iFlow**</DNT>.
    5. Note the <DNT>**iFlow endpoint URL**</DNT>.
  </Step>

  <Step>
    ## Crear destino RFC

    1. Use transaction <DNT>**SM59**</DNT> to create a new <DNT>**RFC destination**</DNT>.

    2. Ingrese la siguiente informacion:

       * <DNT>**RFC Destination**</DNT>: A descriptive name (e.g., <DNT>**NR\_CPI\_ENDPOINT**</DNT>)
       * <DNT>**Connection Type**</DNT>: <DNT>**G**</DNT> (HTTP Connection to External Server)
       * <DNT>**Description**</DNT>: Connection to New Relic via Cloud Integration

    3. On the <DNT>**Technical Settings**</DNT> tab:

       * <DNT>**Target Host**</DNT>: The host from your <DNT>**iFlow endpoint**</DNT>
       * <DNT>**Service No.**</DNT>: <DNT>**443**</DNT>
       * <DNT>**Path Prefix**</DNT>: The path from your <DNT>**iFlow endpoint**</DNT>

    4. On the <DNT>**Logon &amp; Security**</DNT> tab:

       * Configure la autenticación según el método seleccionado:

         * For <DNT>**ClientId/Secret**</DNT>: Enter <DNT>**ClientId**</DNT> as username and <DNT>**ClientSecret**</DNT> as password
         * For <DNT>**Certificate**</DNT>: Configure the <DNT>**certificate details**</DNT>

    5. On the <DNT>**Security Options**</DNT> tab:
       * Select <DNT>**Active**</DNT> for <DNT>**SSL**</DNT>

    6. Guarde la configuración.

    7. Click <DNT>**Connection Test**</DNT> to verify the connection.
  </Step>

  <Step>
    ## Configurar el administrador de recopilación

    1. Execute transaction <DNT>**/NEWR/ONE\_COLLECTION**</DNT> or create a <DNT>**variant**</DNT> for <DNT>**/NEWR/ABAP\_COLLECTION\_MANAGERN**</DNT>.
    2. En la sección <DNT>**Data Transmission to NR**</DNT> :
       * Select <DNT>**SAP Cloud Integration**</DNT> instead of <DNT>**Direct**</DNT>
    3. In the <DNT>**iFlow Endpoint**</DNT> field:
       * Enter the <DNT>**RFC destination name**</DNT> created in Step 4 (e.g., <DNT>**NR\_CPI\_ENDPOINT**</DNT>)
    4. Save the <DNT>**variant**</DNT>.
    5. Schedule or execute the <DNT>**collection job**</DNT>.
  </Step>
</Steps>

## Verificar la transmisión de datos

1. Execute the <DNT>**collection manager job**</DNT>.

2. Check the <DNT>**iFlow monitoring**</DNT> in <DNT>**SAP Cloud Integration**</DNT>:

   * Iniciar sesión en <DNT>**SAP BTP Integration Suite**</DNT>
   * Go to <DNT>**Monitor**</DNT> &gt; <DNT>**Integrations**</DNT>
   * Verificar el procesamiento exitoso de mensajes

3. Check data arrival in <DNT>**New Relic**</DNT>:

   * Iniciar sesión en <DNT>**New Relic**</DNT>
   * Navigate to your <DNT>**SAP dashboards**</DNT>
   * Verifica que se estén recibiendo datos de monitoreo

<Callout variant="important">
  If you modify the standard <DNT>**iFlow template**</DNT>, ensure all required mappings and headers are preserved to maintain compatibility with the <DNT>**collection manager program**</DNT>.
</Callout>
