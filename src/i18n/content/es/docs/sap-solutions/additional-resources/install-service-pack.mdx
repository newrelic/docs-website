Los paquetes de servicio se entregan como archivos `.SAR` con la convención de nombres `K-<Version Number>SP<SP Number>IN<Software Component>.SAR`.

<Steps>
  <Step>
    Instale el paquete de servicio mediante la transacción `SPAM`.
  </Step>

  <Step>
    Cargue la nueva plantilla de configuración del cliente 000.
  </Step>

  <Step>
    Vaya a **Install New Release and Patch** &gt; **Copy Template Tables** del cliente 000.
  </Step>

  <Step>
    Genere la nueva configuración del proveedor de datos siguiendo los pasos en **Install New Release and Patch** &gt; **Generate Configuration for New Data Providers**.
  </Step>

  <Step>
    Haga clic en **Install New Release and Patch** &gt; **Verify Installation Completeness** para verificar la instalación.
  </Step>
</Steps>
