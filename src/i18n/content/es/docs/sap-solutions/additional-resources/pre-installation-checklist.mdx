## Verifica tu cuenta de New Relic

* ¿Ya tienes una cuenta New Relic para el monitoreo de sistemas SAP?

  * **Yes**: especifique el centro de datos de New Relic al que está conectada su cuenta (EE. UU. o UE).
  * **No**: Crea una nueva cuenta conectada a uno de los centros de datos.

## Prepare su lista de sistemas SAP

* Para cada sistema SAP que planea monitorear, proporcione la siguiente información:

  * **ID del sistema y clientes**: identifica el sistema y sus clientes asociados.

  * **Información de la base de datos**: especifique el proveedor de la base de datos (por ejemplo, HANA, Oracle) y su versión.

  * **Detalles del componente de software**: incluye la versión de SAP y el nivel de parche para los siguientes componentes:

    * **SAP\_BASIS**
    * **Núcleo**

## Designar un sistema SAP central

* Elija un sistema SAP que sirva como sistema central para la instalación de NRM4SAP.

  * **Detalles de la instalación**: proporcione el ID del sistema y el cliente donde se instalarán el componente ABAP y la configuración.
  * **Información del servidor proxy**: si se emplea un servidor proxy, especifique su nombre de host y puerto. Incluya todas las políticas de servidor proxy relevantes para acceder a la plataforma New Relic One.
