Configure las entradas de registro (Call IDs) para los proveedores de datos de CloudALM para habilitar el monitoreo de los servicios de SAP Cloud ALM.

## Descripción general

Este programa de utilidad proporciona la funcionalidad para agregar, cambiar, eliminar y autogenerar configuraciones para los proveedores de datos de CloudALM.

* <DNT>**Transaction Code**</DNT>: <DNT>**/NEWR/GEN\_DP\_CALLID**</DNT>
* <DNT>**Program Name**</DNT>: <DNT>**/NEWR/GENERATE\_CONFIG\_CALLID**</DNT>
* <DNT>**Configuration Table**</DNT>: <DNT>**/NEWR/CALLID\_CNF**</DNT>

## Requisitos previos

Asegúrese de que el proveedor de datos <DNT>**NR\_CALM\_LANDSCAPE**</DNT> esté activo y recopilando datos antes de ejecutar esta utilidad.

## Procedimiento de configuración

<Steps>
  <Step>
    ## Configurar el mapeo de proveedor de datos y tipo de servicio

    1. Ejecute la transacción `/NEWR/GEN_DP_CALLID`.
    2. Click the <DNT>**DP/CALM Services Type Maintenance Link**</DNT>.
    3. Seleccione los tipos de servicio que desea monitorear.
    4. Vincule cada proveedor de datos al sistema o servicio SAP correspondiente configurado en SAP Cloud ALM.
    5. Guarde la configuración.

    <Callout variant="tip">
      <DNT>**Service type**</DNT> represents <DNT>**SAP systems**</DNT> or <DNT>**services**</DNT> enabled in your <DNT>**SAP Cloud ALM**</DNT> environment.
    </Callout>
  </Step>

  <Step>
    ## Agregar nuevos registros de ID de llamada

    1. En la pantalla de selección, ingrese el proveedor de datos y el tipo de servicio que necesita configurar.
    2. Select the radio button <DNT>**Add new (Call ID) records to Registry**</DNT>.
    3. Haga clic en <DNT>**Execute**</DNT>.
    4. The system displays all <DNT>**service type instances**</DNT> configured in <DNT>**Cloud ALM**</DNT>, each with a unique <DNT>**LMS ID**</DNT>.
    5. Seleccione las instancias que desea monitorear.
    6. Elimine cualquier entrada que no necesite.
    7. Haga clic en <DNT>**Update and save configuration**</DNT>.
  </Step>

  <Step>
    ## Verifique la configuración

    1. Vaya a la transacción `/NEWR/REGISTRY`.
    2. Seleccione el <DNT>**data provider**</DNT>.
    3. Click <DNT>**Configure**</DNT> to view the saved <DNT>**Call ID entries**</DNT>.
    4. Click <DNT>**Test**</DNT> to verify the configuration.

    **Resultado esperado**: la prueba de conexión muestra un estado exitoso.
  </Step>
</Steps>

## Opciones de mantenimiento

### Actualizar entradas existentes

If you need to change or delete existing <DNT>**Call ID records**</DNT>:

1. Ejecute la transacción `/NEWR/GEN_DP_CALLID`.
2. Seleccione <DNT>**Copy/Change/Delete existing (Call ID) records in Registry**</DNT>.
3. Haga clic en <DNT>**Execute**</DNT>.
4. Realice los cambios necesarios.
5. Guarde la configuración.

### Borrar todas las entradas

To remove all <DNT>**Call ID records**</DNT> (for initial setup or service pack upgrades):

1. Ejecute la transacción `/NEWR/GEN_DP_CALLID`.
2. Seleccione <DNT>**Backup &amp; delete all existing (Call ID) records in Registry**</DNT>.
3. Haga clic en <DNT>**Execute**</DNT>.

<Callout variant="important">
  Esto crea un respaldo antes de la eliminación. Utilice esta opción solo durante la configuración inicial o cuando se le indique durante una actualización de service pack.
</Callout>

### Actualizar definiciones de servicios

If you cannot locate a <DNT>**service type**</DNT> or <DNT>**LMS ID**</DNT>:

1. Haga clic en <DNT>**Retrieve new services (LMSID) setup in Cloud ALM**</DNT>.
2. This fetches the latest services configured in your <DNT>**Cloud ALM account**</DNT>.
