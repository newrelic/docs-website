* Si el total de SID monitoreados es menor a 10 y las instancias no superan las 50, no es necesario un servidor dedicado. Si los SID son más de 10 o las instancias son más de 100, se recomienda un servidor dedicado.
* Como máximo, se debe emplear un servidor dedicado con dos instancias de App Server, cada una con 96 GB de memoria y 12 CPU de núcleo. Si el sistema de monitoreo emplea una base de datos HANA, recomendamos separar la instancia del servidor de base de datos de la instancia del servidor de aplicaciones.
