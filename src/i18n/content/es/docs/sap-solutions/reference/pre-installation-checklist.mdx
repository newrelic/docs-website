Use this checklist to gather all necessary information before installing <DNT>**New Relic Monitoring for SAP Solutions**</DNT>. Complete the preparation sections first, then fill in the readiness checklist tables before beginning installation.

## Preparación previa a la instalación

### Verifica tu cuenta de New Relic

* Confirma que tienes una cuenta de New Relic, o crea una cuenta nueva. Tenga en cuenta qué región del centro de datos utiliza (EE. UU. o UE).

### Prepare su lista de sistemas SAP

For each <DNT>**SAP system**</DNT> you plan to monitor, provide the following information:

* **ID del sistema y clientes**: identifica el sistema y sus clientes asociados.

* **Database information**: Specify the database vendor (e.g., <DNT>**HANA**</DNT>, <DNT>**Oracle**</DNT>) and its version.

* **Detalles del componente de software**: incluye la versión de SAP y el nivel de parche para los siguientes componentes:

  * <DNT>**SAP\_BASIS**</DNT>
  * <DNT>**Kernel**</DNT>

### Designar un sistema SAP central

Choose one <DNT>**SAP system**</DNT> to serve as the central system for <DNT>**New Relic Monitoring for SAP Solutions (NRM4SAP)**</DNT> installation.

* **Installation details**: Provide the <DNT>**System ID**</DNT> and client where the <DNT>**ABAP component**</DNT> and configuration will be installed.
* **Proxy server information**: If a proxy server is used, specify its host name and port. Include any relevant proxy server policies for accessing the <DNT>**New Relic One**</DNT> platform.

### Accesibilidad de la red

Verify network connectivity and review <DNT>**API endpoint**</DNT> configurations:

* Ensure your firewall is not blocking <DNT>**HTTPS**</DNT> traffic to New Relic
* Documente los detalles del servidor proxy si corresponde
* Review <DNT>**RFC destination**</DNT> configurations for your region (US or EU)

Para conocer los requisitos de red completos y los detalles de los extremos de API, consulte [Requisitos de red y extremos de API](/docs/sap-solutions/reference/network-accessibility).
