Utilice esta lista de verificación para recopilar toda la información necesaria antes de instalar <DNT>**New Relic Monitoring for SAP Solutions**</DNT>. Complete primero las secciones de preparación, luego complete las tablas de la lista de verificación de preparación antes de comenzar la instalación.

## Preparación previa a la instalación

### Verifica tu cuenta de New Relic

* Confirma que tienes una cuenta de New Relic, o crea una cuenta nueva. Tenga en cuenta qué región del centro de datos utiliza (EE. UU. o UE).

### Prepare su lista de sistemas SAP

Para cada <DNT>**SAP system**</DNT> que planee monitorear, proporcione la siguiente información:

* **ID del sistema y clientes**: identifica el sistema y sus clientes asociados.

* **Información de la base de datos**: especifique el proveedor de la base de datos (por ejemplo, <DNT>**HANA**</DNT>, <DNT>**Oracle**</DNT>) y su versión.

* **Detalles del componente de software**: incluye la versión de SAP y el nivel de parche para los siguientes componentes:

  * <DNT>**SAP\_BASIS**</DNT>
  * <DNT>**Kernel**</DNT>

### Designar un sistema SAP central

Elija un <DNT>**SAP system**</DNT> para que sirva como sistema central para la instalación de <DNT>**New Relic Monitoring for SAP Solutions (NRM4SAP)**</DNT>.

* **Detalles de instalación**: proporcione el <DNT>**System ID**</DNT> y el cliente donde se instalarán el <DNT>**ABAP component**</DNT> y la configuración.
* **Información del servidor proxy**: si se utiliza un servidor proxy, especifique su nombre de host y puerto. Incluya cualquier política de servidor proxy relevante para acceder a la plataforma <DNT>**New Relic One**</DNT>.

### Accesibilidad de la red

Verifique la conectividad de la red y revise la configuración de <DNT>**API endpoint**</DNT>:

* Asegúrese de que su firewall no esté bloqueando el tráfico <DNT>**HTTPS**</DNT> a New Relic
* Documente los detalles del servidor proxy si corresponde
* Revise la configuración de <DNT>**RFC destination**</DNT> para su región (EE. UU. o UE)

Para conocer los requisitos de red completos y los detalles de los extremos de API, consulte [Requisitos de red y extremos de API](/docs/sap-solutions/reference/network-accessibility).
