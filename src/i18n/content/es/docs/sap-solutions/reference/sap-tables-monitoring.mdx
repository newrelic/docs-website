Esta referencia enumera <DNT>**SAP tables**</DNT> recuperados para fines de monitoreo por <DNT>**New Relic Monitoring for SAP Solutions**</DNT>. Estas tablas proporcionan datos para varias áreas de monitoreo, incluidas <DNT>**AIF**</DNT>, <DNT>**SLT**</DNT>, <DNT>**IDOC**</DNT>, trabajos, correo electrónico, transportes, <DNT>**RFC**</DNT>, procesos empresariales y documentos de cambio.

## Autorización y acceso a tablas

<DNT>**New Relic Monitoring for SAP**</DNT> el rol de autorización proporciona un amplio acceso de visualización de tablas para el <DNT>**RFC user**</DNT> en el sistema monitoreado al configurar el <DNT>**RFC connections**</DNT>. Para facilitar la personalización adicional al recuperar <DNT>**SAP tables**</DNT> adicionales, incluidos <DNT>**Z-tables**</DNT> personalizados, así como al recuperar <DNT>**IDOC**</DNT> o datos de carga del mensaje, recomendamos mantener el acceso amplio. Sin embargo, si desea restringir el acceso a la tabla, utilice esta lista para hacerlo.

<Callout variant="important">
  Si algún recolector de datos personalizado recupera datos directamente de una tabla, el rol de seguridad puede requerir una actualización para que el nuevo proveedor de datos funcione.
</Callout>

## Quién debería usar esta referencia

* <DNT>**SAP Basis**</DNT> administradores que revisan el alcance de la recopilación de datos
* Equipos de seguridad auditando los requisitos de acceso a la tabla
* Administradores de base de datos que monitorean el uso de tablas
