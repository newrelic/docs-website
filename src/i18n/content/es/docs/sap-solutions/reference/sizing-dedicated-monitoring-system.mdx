* Si el total de <DNT>**SIDs**</DNT> monitoreados es menor a 10 y las instancias no exceden 50, no es necesario un servidor dedicado. Si los <DNT>**SIDs**</DNT> son más de 10 o las instancias son más de 100, se recomienda un servidor dedicado.
* Como máximo, se debe utilizar un servidor dedicado con dos instancias de <DNT>**App Server**</DNT>, cada una con 96G de memoria y CPU de 12 núcleos. Si el sistema de monitoreo usa un <DNT>**HANA database**</DNT>, recomendamos separar las instancias del servidor de base de datos de las instancias del servidor de aplicaciones.
