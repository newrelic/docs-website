* If the total monitored <DNT>**SIDs**</DNT> are less than 10 and the instances do not exceed 50, a dedicated server is not necessary. If the <DNT>**SIDs**</DNT> are over 10 or instances are over 100, a dedicated server is recommended.
* At most, a dedicated server with two <DNT>**App Server**</DNT> instances each has 96G of memory and 12 Core CPUs should be used. If the monitoring system uses a <DNT>**HANA database**</DNT>, we recommend separating database server instances from the application server instances.
