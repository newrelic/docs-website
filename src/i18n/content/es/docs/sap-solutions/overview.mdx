New Relic Monitoring for SAP Solutions captura telemetry data de SAP (métricas, eventos, logs y trazas) y los envía a la plataforma New Relic para su análisis y visualización.

## Sistema de monitoreo central vs. sistemas monitoreados

Los sistemas SAP que proporcionan telemetry data se conocen como sistemas monitoreados. El sistema de monitoreo central es donde está instalado <DNT>**NRM4SAP**</DNT> —recopila, transforma y transmite telemetry data. Por lo general, se instala <DNT>**NRM4SAP**</DNT> en un único sistema SAP, como <DNT>**SAP Solution Manager (SolMon)**</DNT>. <DNT>**NRM4SAP**</DNT> utiliza fuentes de datos estándar de SAP como <DNT>**ST\_PI**</DNT>, <DNT>**CCMS**</DNT> y la base de datos de rendimiento, lo que elimina la necesidad de instalaciones adicionales de agentes en los sistemas monitoreados.

## Arquitectura de flujo de datos

<DNT>**RFC**</DNT> las llamadas extraen telemetry data de SAP al sistema de monitoreo central. El <DNT>**ABAP Enterprise Connector**</DNT> procesa los datos y los transmite de forma segura a New Relic a través de <DNT>**HTTPS**</DNT>.

## Qué incluye

Después de la instalación, puede usar el dashboard listo para usar y las alertas:

* **Visibilidad de gestión**: vea la salud y el rendimiento del sistema SAP junto con otros sistemas de TI en dashboards centralizados.

* **Monitoreo personalizado**: adapte el monitoreo a las necesidades específicas del negocio, como el seguimiento de puntos de integración en los flujos de trabajo de procesamiento de pedidos. Cree dashboards personalizados y visualizaciones usando los telemetry data recopilados.

* **Alertas proactivas**: configura alertas para recibir notificaciones inmediatas de interrupciones del servicio, lo que te permite mantener la continuidad y resolver los problemas rápidamente.
