New Relic Monitoreo para Soluciones SAP captura telemetry datade SAP —métricas, eventos, logs y trazas (MELT)— y los envía a la plataforma New Relic One para su análisis y visualización.

## Sistema de monitoreo central vs. Sistemas monitoreados

Los sistemas SAP que proporcionan telemetry data se conocen como sistemas monitoreados. El Sistema Central es el sistema SAP donde está instalado NRM4SAP; recopila, transforma y transmite telemetry data. Normalmente, NRM4SAP se instala en un solo sistema SAP, como SAP Solution Manager (SolMon). Emplea fuentes de datos estándar de SAP como ST\_PI, CCMS y la base de datos de rendimiento, eliminando la necesidad de instalación de agentes adicionales en los sistemas monitoreados.

## Arquitectura de flujo de datos

Los datos se gestionan de manera eficiente mediante el uso de llamadas RFC para extraer telemetry data de SAP al sistema central. Luego, los datos se procesan a través del conector empresarial ABAP y se transmiten de forma segura a la plataforma TDP de New Relic One mediante HTTPS.

Esta arquitectura garantiza un manejo optimizado de datos y mejora la visibilidad del rendimiento del sistema SAP, proporcionando al ingeniero información crítica y valiosa para un monitoreo y gestión efectivos.

## Caso de uso

Los sistemas SAP son vitales para las grandes compañías y sirven como columna vertebral de los procesos empresariales esenciales, incluidas las finanzas, la logística, la gestión de materiales, el almacenamiento, las ventas y la distribución. Al emplear New Relic Monitoreo para Soluciones SAP (NRM4SAP), las compañías pueden lograr una visibilidad integral de estas operaciones impulsadas por SAP para garantizar que funcionen sin problemas.

### integración y resolución de problemas

La integración de NRM4SAP en la infraestructura de una organización es crucial para monitorear los sistemas SAP que respaldan operaciones comerciales clave como el procesamiento de pedidos de comercio electrónico y la gestión de almacenes. Esta integración ofrece lo siguiente:

**Monitoreo integral**: NRM4SAP proporciona una observabilidad robusta, cerciorando que los sistemas SAP funcionen de manera eficiente y facilitando la detección y resolución de problemas potenciales antes de que afecten el proceso empresarial.

### Post-instalación y próximos pasos

Luego de la instalación, los clientes pueden aprovechar de inmediato el panel de control y las alertas listas para usar que ofrece NRM4SAP, lo que genera varios beneficios:

* **Información valiosa sobre gestión mejorada**: el panel centralizado ofrece visibilidad sobre el estado y el rendimiento de los sistemas SAP junto con otros sistemas de TI, lo que agiliza la supervisión de la gestión y reduce la dependencia de los controles administrativos.

* **Oportunidades de monitoreo personalizado**: los clientes pueden adaptar el monitoreo a necesidades comerciales específicas, como el seguimiento de puntos de integración dentro del flujo de trabajo de procesamiento de pedidos. Se pueden desarrollar paneles de control personalizados y visualizaciones empleando telemetry data recopilados.

* **Alertas proactivas**: al configurar alertas cruciales, se puede advertir inmediatamente al administrador sobre interrupciones del servicio, lo que le permite mantener la continuidad y abordar rápidamente cualquier problema.

NRM4SAP permite a las compañías optimizar sus operaciones de SAP, mejorando la visibilidad y facilitando la gestión eficaz de su ecosistema SAP.
