UI de New Relic admite las dos versiones de escritorio más recientes del navegador más común. Si estás utilizando una versión browser no compatible, verás un error en la UI.

<Callout variant="tip">
  Para conocer los requisitos de nuestro <InlinePopover type="browser"/>, consulte [Compatibilidad para monitoreo de navegador](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-new-relic-browser).
</Callout>

## Navegador compatible [#requirements]

Nuestra UI admite las dos versiones de escritorio más recientes para estos navegadores:

* Google Chrome: Linux, macOS, Windows

* Borde de Microsoft: Windows

* Mozilla Firefox: Linux, macOS, Windows

  * Las versiones de versión de soporte extendido (ESR) de Firefox no son compatibles.

* Safari: MacOS

Si la versión de su browser no es compatible, es posible que aún funcione. Sin embargo, para obtener la mejor funcionalidad de nuestro sitio (y otros), le recomendamos que actualice a una versión compatible.

## Soporte limitado

Nuestra UI proporciona soporte limitado para navegadores móviles. Para acceder a New Relic desde un dispositivo móvil, instale nuestra [aplicación de Android](/docs/mobile-apps/new-relic-android-app/features/new-relic-android-app) o [iOS](/docs/mobile-apps/new-relic-ios-app/features/new-relic-ios-app). El flujo de trabajo en nuestras aplicaciones móviles está diseñado específicamente para casos de uso de teléfonos. La interfaz web no es compatible activamente con el navegador móvil.
