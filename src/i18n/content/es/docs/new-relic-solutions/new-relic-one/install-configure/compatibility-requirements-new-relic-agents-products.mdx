Un [agente](/docs/using-new-relic/welcome-new-relic/get-started/glossary#agent) de New Relic es una pieza de software que se instala en un host o en una aplicación que envía datos de rendimiento a New Relic. Usamos diferentes agentes para diferentes productos y lenguajes de codificación.

En este documento, tenemos enlaces a algunas de nuestras herramientas más populares. Para obtener una lista completa de nuestras soluciones, consulte [Observabilidad instantánea de New Relic](https://newrelic.com/instant-observability/).
