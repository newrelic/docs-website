Algunas [soluciones de New Relic](https://newrelic.com/instant-observability/) requieren la instalación de un agente. Puede desinstalar el agente si desea que deje de informar datos. Si necesita desinstalar New Relic por completo, siga los procedimientos de su agente.

Si tiene problemas, consulte [support.newrelic.com](https://support.newrelic.com).

## Antes de desinstalar [#downgrade]

<Callout variant="important">
  Para la organización New Relic de pago, una alternativa a la desinstalación es [bajar](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) a una edición inferior.
</Callout>

## Desinstalar APM [#language-agents]

Después de desinstalar <InlinePopover type="apm"/>agente o realizar un cambio en su script de inicio, debe reiniciar su aplicación (o el servidor de aplicaciones o el servidor web donde se encuentra). Debes hacer esto porque el agente New Relic normalmente se ejecuta en la memoria del proceso que ejecuta la aplicación. Simplemente eliminar esos archivos **no** impedirá que la aplicación que se está ejecutando actualmente envíe datos a New Relic.

* [Go](/docs/agents/go-agent/installation-configuration/uninstall-go-agent)
* [Java](/docs/agents/java-agent/installation/uninstall-java-agent)
* [.NET](/docs/agents/net-agent/installation-configuration/uninstall-net-agent)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/uninstall-nodejs-agent)
* [PHP](/docs/agents/php-agent/advanced-installation/uninstalling-php-agent)
* [Python](/docs/agents/python-agent/installation-configuration/uninstall-python-agent)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/uninstall-ruby-agent)
