Algunas [integraciones de New Relic](https://newrelic.com/instant-observability/) requieren la instalación manual de un agente. La forma en que se configura el comportamiento de esos agentes depende del agente específico.

## Configurar agente APM [#apm-config]

Las únicas configuraciones requeridas para los agentes <InlinePopover type="apm"/>son su [clave de licencia](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) y [el nombre de la aplicación](/docs/apm/new-relic-apm/installation-configuration/name-your-application), que se configuran cuando [instala el agente](/docs/agents/manage-apm-agents/installation/install-agent).

Para opciones de configuración adicionales, consulte:

* [Go](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [Java](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.NET](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [Node.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [Ruby](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## Configurar agente del navegador [#browser-config]

Consulte nuestra documentación para [la configuración de<InlinePopover type="browser"/>](/docs/browser/new-relic-browser/configuration).

## Configurar el agente de infraestructura [#infra-config]

La única configuración requerida para nuestro agente de infraestructura es su <InlinePopover type="licenseKey"/>. Esto se configura cuando [instala el agente](/docs/agents/manage-apm-agents/installation/install-agent). También puede emplear [opciones de configuración adicionales](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent) para monitor su infraestructura.

## Configurar agente móvil [#mobile-config]

La única configuración requerida para el monitoreo de móviles agente es el [token de aplicación](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token). Esto se configura cuando instalas el agente de monitoreo de móviles para tus aplicaciones [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android), [iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) .

Para opciones de configuración adicionales, consulte:

* [Configuración de Android](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [Configuración de iOS/TVOS](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)

Para configurar otras herramientas que no figuran aquí, consulte la [documentación de una solución específica](https://newrelic.com/instant-observability/).
