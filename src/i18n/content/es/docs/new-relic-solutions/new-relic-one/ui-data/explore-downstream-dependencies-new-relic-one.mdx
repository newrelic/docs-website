Para su aplicación de monitoreo en nuestra UI <InlinePopover type="apm"/>, <InlinePopover type="browser"/>y monitoreo de móviles, la página UI <DNT>**Dependencies**</DNT> muestra una lista filtrable de todas las aplicaciones, servicios, base de datos y hosts que New Relic identifica como una conexión. a esa [entidad](/docs/new-relic-solutions/new-relic-one/core-concepts/what-entity-new-relic). Muestra dependencia ascendente y descendente y proporciona caminos para explorarlas.

Mientras que [los mapas de servicios](/docs/understand-dependencies/understand-system-dependencies/service-maps/service-maps-new-relic-one) muestran un mapa con ámbito de entidad de las relaciones de una entidad, la página UI <DNT>**Dependencies**</DNT> muestra sus servicios ascendentes y descendentes en una tabla. También utiliza el mismo [sistema de codificación de colores que utilizan los mapas de servicios](/docs/understand-dependencies/understand-system-dependencies/service-maps/service-maps-new-relic-one#alerts) para mostrarle qué funciona bien y qué no.

## Requisitos

La página UI <DNT>**Dependencies**</DNT> está disponible para que nuestro agente APM, nuestro agente móvil y nuestro agente del navegador puedan monitorear la entidad.

## Ver dependencia [#access]

Para ver la dependencia de la aplicación, los servicios, la base de datos y los hosts conectados a una entidad:

1. Vaya a

   <DNT>
     **[one.newrelic.com](https://one.newrelic.com)**
   </DNT>

   y seleccione

   <DNT>
     **APM & services**
   </DNT>

   ,

   <DNT>
     **Browser**
   </DNT>

   o

   <DNT>
     **Mobile**
   </DNT>

   .

2. Elija una entidad y luego haga clic en

   <DNT>
     **Dependencies**
   </DNT>

   .

3. Para profundizar más, filtre las aplicaciones, servicios, base de datos o hosts.

Puede filtrar la página de dependencia para ver cosas específicas que reportan a la entidad. dependencia incluyen:

* <DNT>
    **Services**
  </DNT>

  : Aplicación y servicios de monitor [APM](/docs/apm) .

* <DNT>
    **Mobile applications**
  </DNT>

  : tus [aplicaciones móviles](/docs/mobile-monitoring).

* <DNT>
    **Browser applications**
  </DNT>

  : tus [aplicaciones de navegador](/docs/browser) frontend.

* <DNT>
    **External services**
  </DNT>

  : [monitor de servicios externos por APM](/docs/apm/applications-menu/monitoring/external-services-page). Los servicios externos incluyen servicios fuera de proceso, como servicios web, recursos en la nube y cualquier otra llamada de red.

* <DNT>
    **Databases**
  </DNT>

  : la base de datos y los datos de caché de su aplicación. base de datos son sin agente. Debido a esto, no se puede configurar alerta para la base de datos, ya que solo se informa a New Relic la llamada de servicio.

* <DNT>
    **Hosts**
  </DNT>

  : su [infraestructura](/docs/infrastructure) (servidores y hosts).
