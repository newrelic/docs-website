New Relic Pathpoint ofrece un sistema de configuración flexible que se puede personalizar para satisfacer las necesidades únicas de su negocio. Pathpoint admite la configuración desde una GUI interactiva. También es posible descargar y cargar una configuración JSON. Esto es particularmente útil cuando se desea controlar la versión de una configuración.

Este nivel de personalización le permite crear una solución de observabilidad personalizada que se ajuste a los requisitos específicos de su organización.

En esta página, aprenderá:

* Cómo configurar Pathpoint con la GUI
* Cómo utilizar el formato de intercambio JSON

## Usando el formato de intercambio JSON

Es extremadamente fácil e intuitivo construir una implementación de Pathpoint utilizando los editores de etapas, pasos y puntos de contacto que se muestran en la sección anterior. A veces será útil o necesario almacenar la configuración en un archivo e importar esa configuración más adelante, posiblemente en otra cuenta o instancia de Pathpoint. Para una implementación de Pathpoint en producción, recomendamos encarecidamente almacenar un archivo de su configuración como JSON en un sistema de control de versiones como git.

### Descargue su configuración actual como JSON

Pathpoint brinda la flexibilidad de descargar el archivo de configuración JSON actual directamente a su computadora. Esta característica es particularmente útil si necesita realizar cambios en el archivo.

Para descargar el archivo de configuración JSON actualmente activo:

<Steps>
  <Step>
    Desde su cuenta New Relic, vaya a <DoNotTranslate>**Apps > Pathpoint**</DoNotTranslate>. Clickea en el<img src={threeLines} class="inline" style={{height: "30px", width: "30px"}} alt="Three lines" title="Three lines"/>

    en la esquina superior izquierda.
  </Step>

  <Step>
    Haga clic en <DoNotTranslate>**JSON configuration**</DoNotTranslate>.
  </Step>

  <Step>
    A continuación, haga clic en la flecha hacia abajo.

    <img
      title="download active JSON config file"
      alt="download active JSON config file"
      src={downloadFile}
    />

    Su archivo de configuración JSON activo ahora está descargado.
  </Step>
</Steps>

### Reemplace su configuración actual cargando un archivo JSON

Para cargar un nuevo archivo de configuración JSON:

<Steps>
  <Step>
    Desde su cuenta New Relic, navegue hasta <DoNotTranslate>**Apps > Pathpoint**</DoNotTranslate> y haga clic en<img src={threeLines} class="inline" style={{height: "30px", width: "30px"}} alt="Three lines" title="Three lines"/>

    .
  </Step>

  <Step>
    En la ventana resultante, haga clic en <DoNotTranslate>**JSON configuration**</DoNotTranslate>.
  </Step>

  <Step>
    Esto abre una ventana de cuadro de diálogo. En <DoNotTranslate>**Description**</DoNotTranslate>, describe tu nuevo archivo de configuración JSON. A continuación, proporcione una nota y haga clic en <DoNotTranslate>**From file**</DoNotTranslate> para explorar el archivo de configuración JSON.

    <img
      title="Upload new JSON config file"
      alt="Upload new JSON config file"
      src={uploadJson}
    />
  </Step>

  <Step>
    Localice el archivo de configuración JSON que desea utilizar para actualizar su Pathpoint y haga clic en <DoNotTranslate>**Open**</DoNotTranslate>.

    <img
      title="open new JSON config file"
      alt="open new JSON config file"
      src={openFile}
    />

    Finalmente, se muestra el Pathpoint con las actualizaciones del archivo JSON particular que eligió.
  </Step>
</Steps>
