New Relic Pathpoint se puede integrar con casi cualquier cuenta New Relic para brindar observabilidad integral del proceso empresarial. Para comenzar, es importante tener telemetry data que sean relevantes para su organización. Esto puede incluir métrica, evento, log o traza. Si bien el evento y el log APM son un punto de partida común para Pathpoint, la elección de telemetry data depende totalmente de usted.

Antes de comenzar con Pathpoint, debe instalarlo en su cuenta New Relic.

## Instale Pathpoint desde el catálogo de New Relic [#install-catalog]

Para instalar Pathpoint:

<Steps>
  <Step>
    Vaya a <DNT>**[Apps & Visualizations](https://onenr.io/0yw40Vz6Ew3)**</DNT>.
  </Step>

  <Step>
    Busque <DNT>**Pathpoint V2**</DNT> (debe especificar `V2` para obtener la nueva base de código. La versión anterior quedó obsoleta).
  </Step>

  <Step>
    Haga clic en el icono de instalación del <DNT>**Pathpoint V2**</DNT> verificado y luego haga clic en <DNT>**Add this app**</DNT>.

    <img
      title="Install Pathpoint"
      alt="Install Pathpoint"
      src="/images/pathpoint_v2_screenshot_install_pathpoint.webp"
    />

    Pathpoint ahora está agregado a su cuenta de New Relic y puede acceder a él en <DNT>**Apps**</DNT>.
  </Step>
</Steps>

Una vez que instaló Pathpoint, puede configurar sus etapas, pasos y señales simplemente usando la UI interactiva. También puede exportar e importar configuraciones JSON, lo cual es útil para clonar y controlar versiones de configuraciones, así como para moverlas entre cuentas New Relic. [Obtenga más información sobre cómo configurar Pathpoint](/docs/new-relic-solutions/business-observability/intro-pathpoint/).
