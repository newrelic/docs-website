Algunas [soluciones de New Relic](https://newrelic.com/instant-observability/) requieren la instalación manual de un agente. La mayoría de los agentes incluyen herramientas integradas para generar registros detallados para resolución de problemas, auditoría y diagnóstico.

Documentos relacionados:

* Para la resolución general de problemas del agente, consulte [No ver datos](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data).
* Obtenga más información sobre [New Relic Diagnostics](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics): una utilidad que detecta automáticamente problemas comunes.
