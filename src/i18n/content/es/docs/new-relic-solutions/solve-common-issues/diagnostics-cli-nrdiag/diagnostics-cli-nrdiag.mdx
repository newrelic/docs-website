La CLI de diagnóstico (`nrdiag`) es una utilidad que detecta automáticamente problemas comunes con los productos New Relic. Si la CLI de diagnóstico detecta un problema, sugiere pasos para la resolución de problemas. La CLI de diagnóstico puede cargar automáticamente datos de resolución de problemas a una cuenta de New Relic.

La CLI de diagnóstico es de código abierto y se encuentra en [GitHub](https://github.com/newrelic/newrelic-diagnostics-cli).

Para conocer pasos adicionales de resolución de problemas para su agente, consulte [No ver datos](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data).

## Compatibilidad

La CLI de diagnóstico está disponible para Linux, macOS y Windows. Puede detectar problemas de configuración comunes para:

* <DNT>**APM**</DNT>: Disponible para todos los <InlinePopover type="apm" />agentes. Para el agente Go, solo están disponibles comprobaciones de conectividad básicas.
* <DNT>**Browser monitoring**</DNT>: Detección de <InlinePopover type="browser" />agente.
* <DNT>**Infrastructure monitoring**</DNT>: Agente de Linux y Windows
* <DNT>**Mobile agents**</DNT>: iOS y Android
* <DNT>**Synthetic monitoring**</DNT>: Minion privado en contenedor (llamadas por minuto)
* <DNT>**Kubernetes**</DNT>: Kubernetes, agente Control, Helm y Flux

La CLI de diagnóstico no requiere permisos `superuser` o `admin` para ejecutarse, aunque recomendamos esos permisos para algunas comprobaciones. Devolverá un error si no tiene permisos para leer los archivos que escanea.

## Empezar

Para utilizar la CLI de diagnóstico:

1. Ejecute la CLI de diagnóstico, incluidos [los conjuntos de tareas](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/run-diagnostics-cli-nrdiag) y [las opciones de línea de comando,](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) según sea necesario.
2. Proporcione la bandera [`-attach` o `-api-key`](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) para cargar resultados a su cuenta de New Relic.
3. Opcional: valide la [configuración del archivo de configuración](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/validate-config-file-settings-nrdiag).
4. Interpretar la [salida](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output).

Como cualquier otra herramienta de New Relic, el servicio CLI de diagnóstico está diseñado para proteger su privacidad de datos y la de sus clientes. Para obtener información detallada, consulte nuestra documentación [de seguridad y licencias de CLI de diagnóstico](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/diagnostics-cli-licensing-security) .
