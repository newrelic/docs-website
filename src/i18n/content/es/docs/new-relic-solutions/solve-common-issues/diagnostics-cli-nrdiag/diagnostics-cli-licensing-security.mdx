Como cualquier otra herramienta de New Relic, el servicio CLI de Diagnóstico está diseñado para proteger la privacidad de los datos para usted y sus clientes. La CLI de diagnóstico inspecciona la información del sistema y los artefactos relevantes del producto New Relic, como los archivos de registro y configuración. La CLI se centra únicamente en la información necesaria para realizar comprobaciones de diagnóstico que evalúan la configuración y operabilidad del producto New Relic.

Por defecto, estos datos no se transmiten a New Relic. Tiene la opción de cargar esta información en su cuenta New Relic a través de HTTPS. Para obtener más información sobre las medidas de seguridad de New Relic, consulte nuestra [documentación de seguridad y privacidad](/docs/security/security-privacy/data-privacy/data-privacy-new-relic) o visite el [sitio web de seguridad de New Relic](https://newrelic.com/security).

## Acuerdos de licencia [#diagnostics-licenses]

El uso de Diagnostics CLI está sujeto a este [acuerdo de licencia](/docs/licenses/license-information/other-licenses/new-relic-diagnostics-licenses), así como a los acuerdos de licencia para el software de código abierto utilizado por Diagnostics CLI.

## Datos cargados a la cuenta [#uploaded-account-data]

La CLI de Diagnóstico permite cargar esta información a una cuenta de New Relic a través de HTTPS si utiliza un argumento de línea de comando específico. Será símbolo antes de la recopilación de cualquier archivo que esperamos que tenga información confidencial.

Antes de que los archivos recopilados [contenidos en `nrdiag-output.zip` y `nrdiag-output.json`](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output) se carguen en New Relic, también aparecerá el símbolo. Esto le permite revisar y editar cualquier información que no desee proporcionar. (Por ejemplo, el `nrdiag-output.zip` incluirá su nombre de usuario). También tienes la opción de cancelar la carga por completo.

## Almacenamiento de datos

Todos los archivos adjuntos realizados utilizando `nrdiag` o que contienen datos `nrdiag` son almacenados por New Relic.

## Variables de entorno

La CLI de diagnóstico examina las siguientes variables de entorno para realizar comprobaciones de diagnóstico. Los valores de estas variables se registran localmente en el archivo `nrdiag-output.json` . Éstas incluyen:

* Cualquier variable de entorno que contenga `NEWRELIC` o `NEW_RELIC`
* Cualquier variable de entorno que comience con `NRIA`
