Imagine que desarrolló un sitio web y lo instrumentó con el monitoreo de Browser de New Relic. El conjunto de características principales de New Relic proporciona mucha información, pero quieres algo personalizado. Quiere decidir si un nuevo diseño para su sitio atraerá mejor a su usuario. En otras palabras, desea realizar una Prueba A/B de un diseño y tomar una decisión basada en los datos.

Para esta tarea, puede crear una aplicación New Relic empleando React, el kit de desarrollo de software New Relic One y el poder ilimitado de las tecnologías en línea modernas.

La aplicación New Relic está construida con una de las bibliotecas de JavaScript más populares del desarrollo sitio web: React. Como tienes libertad al escribir código React, puedes personalizar la lógica de tu aplicación, diseñar tus propios componentes o aprovechar la abundancia de la biblioteca de componentes de código abierto. Entonces, para su aplicación Prueba A/B, si desea escribir una lógica personalizada para finalizar la prueba según los resultados, puede hacerlo.

El kit de desarrollo de software (SDK) New Relic One le permite crear, servir, publicar y desplegar aplicaciones en la plataforma New Relic. También proporciona una gran cantidad de componentes de React para recopilar datos, presentar información, manejar la interacción del usuario y más. Emplea componentes como `Button` y `Dropdown` para crear una experiencia interactiva que se ve y se siente nativa de New Relic. Emplea los componentes `Table` y `Chart` para mostrar datos de su cuenta New Relic o de otro lugar. Al crear su aplicación Prueba A/B, usaría el componente `NrqlQuery` del SDK para obtener datos del Browser de su cuenta.

Con código React personalizado, componentes SDK y el amplio mundo de la biblioteca de código abierto, puedes crear tu aplicación Prueba A/B en New Relic. Pero antes de crear uno para usted, es posible que desee consultar el catálogo de aplicaciones para ver si alguien se le adelantó. Si el catálogo ya tenía una aplicación para eso, podías agregarla a tu cuenta con un par de clics, otro beneficio de crear aplicaciones en New Relic.

A lo largo de este curso, creará una aplicación New Relic del mundo real para ejecutar y gestionar la Prueba A/B. Visualizará los datos del Browser para los diseños de la competencia, verá datos históricos de pruebas anteriores e incluso elegirá un diseño ganador y finalizará la prueba, ¡todo desde su aplicación New Relic! Pero antes de adentrar en la creación de gráficos y realizar solicitudes http, debe aprender de qué está hecha la aplicación New Relic.
