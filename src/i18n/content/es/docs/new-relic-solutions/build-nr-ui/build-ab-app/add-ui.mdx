Si una aplicación está organizada, es más legible y más comprensible. En tu aplicación Prueba A/B tienes muchos gráficos, pero ninguna organización. En las próximas lecciones, empleará componentes de la interfaz de usuario para darle cierta organización a su aplicación y proporcionar nuevas funciones. Primero, empleará un componente `Grid` para organizar sus gráficos para que coincidan con el diseño de su guía de diseño. En segundo lugar, agregará títulos para aclarar qué representa cada gráfico. En tercer lugar, agregará descripciones para sus versiones de diseño A y B en la parte superior de su aplicación. Finalmente, crearás una sección para finalizar tu Prueba A/B.
