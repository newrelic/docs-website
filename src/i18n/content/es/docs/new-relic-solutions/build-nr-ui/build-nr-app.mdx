New Relic le ofrece muchas opciones para [incorporar datos de cualquier fuente](/docs/data-apis/custom-data/intro-custom-data) y crear [gráficos](/docs/query-your-data/explore-query-data/query-builder/introduction-query-builder) y [paneles](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) personalizados. Pero si desea ir más allá de esas opciones y crear sus propias experiencias en New Relic, le brindamos acceso a las mismas herramientas que utilizamos para crear aplicaciones en nuestra plataforma.

La aplicación New Relic es una aplicación React JavaScript que:

* Reside en la [plataforma New Relic](/docs/new-relic-one/use-new-relic-one/get-started/introduction-new-relic-one), junto con tus otros

* Visualizaciones características que ha diseñado específicamente para su organización.

* Muestre datos de cualquier fuente que desee, ya sea de una entidad monitora de New Relic o datos de otro servicio o API.

Continúe leyendo para obtener más información sobre lo que puede hacer con las aplicaciones New Relic. O, si desea avanzar y comenzar a crear sus propias aplicaciones, primero lea los [requisitos en nuestro sitio para desarrolladores](https://developer.newrelic.com/build-apps/permission-manage-apps).

## New Relic: una plataforma programable [#what-is-it]

Nos esforzamos por tener una experiencia del usuario automatizada que proporcione un valor óptimo para todos los usuarios. Pero también sabemos que algunas organizaciones tienen necesidades comerciales únicas que no se pueden satisfacer con nuestras opciones estándar de visualización de gráficos y dashboard .

Ahora, le damos control sobre los componentes fundamentales de nuestra plataforma. Utilizando las mismas herramientas que utiliza nuestro ingeniero para construir la plataforma New Relic, usted puede crear aplicaciones personalizadas que se alineen con su estructura organizacional única y sus necesidades comerciales. Si sabe cómo utilizar React, GraphQL y NRQL (nuestro lenguaje de consulta), crear una aplicación solo le llevará unos minutos.

Consulte las [guías en nuestro sitio para desarrolladores](https://developer.newrelic.com/build-apps) para obtener ayuda para crear una aplicación personalizada. O, para obtener una descripción general rápida de cómo comenzar a crear su propia aplicación, mire este breve video de YouTube (aprox. 3:15 minutos).

## Empezar [#get-started]

Con New Relic, puedes resolver cualquier desafío impulsado por datos, sin importar cuán complejo sea. Por ejemplo:

* Utilice nuestras API para [obtener datos en New Relic desde cualquier fuente](/docs/data-apis/custom-data/intro-custom-data).
* [Visualice esos datos en su aplicación personalizada](https://developer.newrelic.com/build-apps).
