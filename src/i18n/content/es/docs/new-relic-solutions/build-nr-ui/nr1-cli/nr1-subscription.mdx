## `nr1 subscription:list` [#nr1-subscriptionlist]

### Ver tu subscripción [#nr1-subscriptionlist-see]

Enumera todos los Nerdpacks a los que está suscrita su cuenta. Tu cuenta está vinculada a tu clave de API.

### Uso

`$ nr1 subscription:list`

### Opciones

<table>
  <tbody>
    <tr>
      <td>

      </td>

      <td>
        El perfil de autenticación que desea emplear.

    </tr>

    <tr>

      </td>

      </td>
    </tr>

## `nr1 subscription:set` [#nr1-subscriptionset]

### Suscríbete a un Nerdpack [#nr1-subscriptionset-subscribe]

Suscribe su cuenta a un Nerdpack y canal específico. Este comando se puede ejecutar con un UUID de Nerdpack o dentro de una carpeta de Nerdpack específica. De forma predeterminada, el comando usa el ID de Nerdpack en `package.json` y se suscribe al canal STABLE.

Solo se puede suscribir una cuenta a un Nerdpack y canal a la vez.

### Uso [#nr1-subscriptionset-usage]

`$ nr1 subscription:set`

### Opciones [#nr1-subscriptionset-options]

<table>
  <tbody>
    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        Especifica el Nerdpack al que suscribir. De forma predeterminada, el comando empleará el de `package.json`.
      </td>
    </tr>

    <tr>
      <td>
        `-c, --channel=DEV/BETA/STABLE`
      </td>

      <td>
        Especifica el canal al que suscribir. \[`default: STABLE`]
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        El perfil de autenticación que desea emplear.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        Agrega información adicional a la salida.
      </td>
    </tr>
  </tbody>
</table>

### Alias [#nr1-subscriptionset-aliases]

* `$ nr1 nerdpack:subscribe`

## `nr1 subscription:unset` [#nr1-subscriptionunset]

### Dar de baja de un Nerdpack [#nr1-subscriptionunset-unsubs]

Cancela la subscripción de tu cuenta a un Nerdpack específico. Cuando este comando se ejecuta dentro de una carpeta Nerdpack, el ID de Nerdpack de `package.json` se usa de forma predeterminada.

### Uso [#nr1-subscriptionunset-usage]

`$ nr1 subscription:unset`

### Opciones [#nr1-subscriptionunset-options]

        Especifica el Nerdpack al que suscribir. De forma predeterminada, el comando empleará el de `package.json`.
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        El perfil de autenticación que desea emplear.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        Agrega información adicional a la salida.
      </td>
    </tr>
  </tbody>
</table>

### Alias [#nr1-subscriptionunset-aliases]
