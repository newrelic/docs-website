Crea un gráfico de tabla. Los datos se pueden obtener realizando una consulta NRQL en una cuenta en individua o se pueden pasar a través de la propiedad `data` .

El formato `data` es un serial de objetos, cada uno de los cuales contiene `metadata` y `data`. Cada `data` contiene ambos valores de este punto y las claves `x` y `y` empleadas para trazar el gráfico.

### Uso

```js
import { TableChart } from 'nr1'
```

### Ejemplos

#### Básico

```js
<TableChart accountIds={[1]} query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType" />
```

#### Con varias cuentas

```js
<TableChart accountIds={[1, 1067061]} query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType" />
```

#### Llenar el contenedor

```js
<TableChart
  accountIds={[1]}
  query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType"
  fullWidth
  fullHeight
/>
```

#### Con datos personalizados

```js
function render() {
  const data = [
     {
       metadata: {
         id: 'series-1',
         name: 'Serie 1',
         color: '#008c99',
         viz: 'main',
         columns: ['jobType', 'count'],
       },
       data: [
         {
           jobType: 'SIMPLE',
           count: 18,
         },
         {
           jobType: 'SCRIPT_BROWSER',
           count: 27,
         },
         {
           jobType: 'SCRIPT_API',
           count: 33,
         },
         {
           jobType: 'BROWSER',
           count: 29,
         },
         {
           jobType: 'SELENIUM_JSON',
           count: 22,
         },
         {
           jobType: 'SCRIPT',
           count: 31,
         },
       ],
     },
   ];;


  return <TableChart data={data} fullWidth />;
}
```
