Elemento secundario del componente `<Grid>` .

De forma predeterminada, abarca una columna, pero puede especificar el número de columnas que debe abarcar configurando los accesorios `columnSpan` o `columnStart` y `columnEnd` .

### Uso

```js
import { GridItem } from 'nr1'
```
