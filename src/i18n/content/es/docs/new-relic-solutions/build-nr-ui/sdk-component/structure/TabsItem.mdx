Elemento secundario del componente `<Tabs>` .

Crea una pestaña donde el título de la pestaña está definido por la propiedad `label` y el contenido de la pestaña está definido por la propiedad `children` .

### Uso

```js
import { TabsItem } from 'nr1'
```
