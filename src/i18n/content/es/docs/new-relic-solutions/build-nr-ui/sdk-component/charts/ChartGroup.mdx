Agrupa diferentes tipos de gráficos para que puedan realizar acciones todos juntos, como mover la aguja según sus ejes x, mostrar la información sobre herramientas para el serial correspondiente o atenuar el área seleccionada en todos los gráficos al mismo tiempo.

El componente actúa como un fragmento de React; no mostrará una etiqueta adicional en el DOM (solo permite la agrupación de todos los gráficos que contiene). Esto significa que este componente, a diferencia de otros, no acepta la etiqueta `className` o `style` .

### Uso

```js
import { ChartGroup } from 'nr1'
```

### Ejemplos

#### Básico

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart style={chartStyle} accountIds={[1]} query={query} />
      <LineChart style={chartStyle} accountIds={[1, 1067061]} query={query} />
      <ScatterChart style={chartStyle} accountIds={[1]} query={query} />
    </ChartGroup>
  );
}
```

#### Llenar el contenedor

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
      <LineChart
        style={chartStyle}
        accountIds={[1, 1067061]}
        query={query}
        fullWidth
        fullHeight
      />
      <ScatterChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
    </ChartGroup>
  );
}
```
