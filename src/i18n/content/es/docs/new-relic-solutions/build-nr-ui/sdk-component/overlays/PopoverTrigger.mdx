Elemento secundario del componente `<Popover>` .

Controla la apertura/cierre de la superposición emergente.

### Uso

```js
import { PopoverTrigger } from 'nr1'
```

### Props

<table>
  <tbody>
    <tr>
      <td>
        `children` Nodo <h5>REQUERIDO</h5> <h5>|función</h5>
      </td>

      <td>
        Este componente se puede representar de dos maneras: pasando directamente un elemento secundario de React o pasando una devolución de llamada de procesamiento (funciona como elemento secundario). Al pasar un elemento React directamente, puede usar algunos de los componentes compatibles del SDK NR1, como `<TextField>`, `<Button>`... o use su propio Componente. Cuando use una devolución de llamada de renderizado, recibirá un objeto de la siguiente forma como argumento `{ opened: boolean, controlled: boolean }`. No importa si está pasando componentes directamente o usando una devolución de llamada de renderizado, Al representar su propio componente como activador, siempre debe exponer un método `focus` para que funcione correctamente.
