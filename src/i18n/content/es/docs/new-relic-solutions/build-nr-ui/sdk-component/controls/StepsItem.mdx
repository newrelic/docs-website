Elemento secundario del componente `<Steps>` .

### Uso

```js
import { StepsItem } from 'nr1'
```

### Props

<table>
  <tbody>
    <tr>
      <td>
        `ariaLabel` <h5>cadena</h5>
      </td>

      <td>
        Proporcione una etiqueta para que los lectores de pantalla describan el paso. Esto solo es necesario si la etiqueta del lector de pantalla debe ser diferente a la propiedad `label` .
      </td>
    </tr>

    <tr>
      <td>
        `checked` <h5>booleano</h5>
      </td>

      <td>
        Si es `true`, marca el paso como completado.
      </td>
    </tr>

    <tr>
      <td>
        `children` <h5>Nodo</h5> <h5>REQUERIDO</h5>
      </td>

      <td>
        Contenido para renderizar dentro del elemento.
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>cadena</h5>
      </td>

      <td>
        Agrega nombres de clase al componente. Debe usar solo con fines de posicionamiento y espaciado.
      </td>
    </tr>

    <tr>
      <td>
        `disabled` <h5>booleano</h5>
      </td>

      <td>
        Emplee deshabilitado para evitar que el usuario abra el paso haciendo clic en su etiqueta.
      </td>
    </tr>

    <tr>
      <td>
        `expanded` <h5>booleano</h5>
      </td>

      <td>
        Cuando sea verdadero, mostrará el contenido del paso.
      </td>
    </tr>

    <tr>
      <td>
        `label` <h5>cadena</h5><h5>REQUERIDA</h5>
      </td>

      <td>
        Etiqueta del paso.
      </td>
    </tr>

    <tr>
      <td>
        `onClick` <h5>función</h5>
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>objeto</h5>
      </td>

      <td>
        Estilo en línea para un estilo personalizado. Debe usar solo con fines de posicionamiento y espaciado.
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>cadena</h5>
      </td>

      <td>
        Agrega un atributo `data-test-id` . Úselo para apuntar al componente en pruebas unitarias y E2E. Para que una identificación de prueba sea válida, antepóngale su identificación de nerdpack, seguido de un punto. Por ejemplo, `my-nerdpack.some-element`. **Nota:** Es posible que no vea el atributo `data-test-id` ya que se eliminan del DOM. Para depurarlos, pase un parámetro de consulta `e2e-test` a la URL.
      </td>
    </tr>

    <tr>
      <td>
        `value` <h5>cualquier</h5>
      </td>

      <td>
        Empleado por el `<Steps>` para identificar el paso activo.
      </td>
