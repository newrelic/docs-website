Puede utilizar [el panel](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) para visualizar y realizar un seguimiento del estado de su sistema, abordar problemas y planificar pasos adicionales. Aprenda a compartir toda esa información con su equipo, clientes u otras partes interesadas con varias opciones de importación y exportación.

## Importar un dashboard como JSON [#import-json]

Puede importar un dashboard como JSON cuando desee modificar un dashboard listo para usar a nivel de código. Por ejemplo, si desea crear un panel personalizado fusionando dos paneles existentes, puede modificar el JSON de un dashboard base y luego importarlo nuevamente a New Relic.

Para importar un dashboard como JSON:

1. Vaya a

   <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Dashboards**</DoNotTranslate>

   .

2. Abra el JSON de un dashboard existente haciendo clic en

   <DoNotTranslate>**...**</DoNotTranslate>

   y luego seleccione

   <DoNotTranslate>**Copy JSON**</DoNotTranslate>

   .

3. Vuelva a la página

   <DoNotTranslate>**Dashboards**</DoNotTranslate>

   .

4. En la esquina superior derecha, haga clic en

   <DoNotTranslate>**Import dashboard**</DoNotTranslate>

   .

5. Pega tu código JSON.

6. Elija la configuración de cuenta y permisos para el dashboard. No puede cambiar la cuenta una vez que la haya configurado, pero puede [cambiar los permisos](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-settings) en cualquier momento.

7. Haga clic en

   <DoNotTranslate>**Save**</DoNotTranslate>

   .

## Exportar un dashboard como PDF [#export-dashboards-pdf]

Para exportar su dashboard como un archivo PDF, en la esquina superior derecha del dashboard, haga clic en <DoNotTranslate>**...**</DoNotTranslate> y seleccione <DoNotTranslate>**Export dashboard as PDF**</DoNotTranslate>.

## Exportar un gráfico como CSV [#export-charts-csv]

Para exportar el gráfico de tabla de dashboard como un archivo CSV, en la esquina superior derecha del gráfico, haga clic en <DoNotTranslate>**...**</DoNotTranslate> y seleccione <DoNotTranslate>**Export as CSV**</DoNotTranslate>.

<img
  title="Export chart as CSV"
  alt="Screenshot of how to export a table chart as a CSV file"
  src={dashboardsExportCsv}
/>

Si bien es poderoso, exportar como CSV tiene limitaciones. Un par de cosas a tener en cuenta:

* Los datos siempre se exportan en hora UTC (-7:00)
* Exportar no admite funciones de agregador como `average()`
* La exportación no admite la cláusula `COMPARE_WITH`

## Comparte un dashboard con tu equipo [#share-dashboards]

Desde cualquier dashboard, compártelo con una de estas opciones:

* Copie el vínculo permanente del dashboard haciendo clic en el ícono de enlace en la parte superior derecha del dashboard. vínculo permanente comparte un enlace directo al dashboard. Tenga en cuenta que estos enlaces caducan si no se utilizan en un período de seis meses.

* Copie su dashboard

  <DoNotTranslate>**as JSON**</DoNotTranslate>

  y agréguelo al portapapeles haciendo clic en el botón

  <DoNotTranslate>**...**</DoNotTranslate>

  en la parte superior derecha y seleccionando

  <DoNotTranslate>**Copy JSON to clipboard**</DoNotTranslate>

  .

También puedes [duplicar tu dashboard](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-clone).
