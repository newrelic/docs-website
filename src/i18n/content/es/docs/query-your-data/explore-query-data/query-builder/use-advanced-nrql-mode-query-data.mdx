      <td>
        Autocompletador
      </td>

      <td>
        El autocompletado del generador de consultas mostrará el evento y atributo informados en los últimos 60 minutos. Un ejemplo de esto es un proceso que se ejecuta una vez al día, como una verificación de estado del sistema estándar que comienza todas las mañanas a las 6:00 a. m. Si intentas consultar el evento a las 7:05 a.m., el evento y el atributo posterior no serán visibles en el menú desplegable de autocompletar.

        Estos eventos y atributos aún se pueden consultar escribiendo la cadena exacta.
      </td>
    </tr>

    <tr>
      <td>
        Consulta múltiple
      </td>

      <td>
        Al utilizar la cláusula [`TIMESERIES`](/docs/query-data/nrql-new-relic-query-language/getting-started/nrql-syntax-clauses-functions#sel-timeseries) puedes ejecutar y comparar hasta 10 consultas de diferentes cuentas.

        Para utilizar multiconsulta, ingresa tu primera consulta con `TIMESERIES` y ejecútala. Una vez que se representan los resultados, el botón <DNT>**Add another query**</DNT> se activa y puede agregar otra consulta.
