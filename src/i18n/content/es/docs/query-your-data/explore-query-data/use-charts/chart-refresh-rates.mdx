La frecuencia de actualización de los gráficos de New Relic es el rango del período de tiempo de la consulta dividido por 60. La frecuencia máxima con la que se puede actualizar la consulta es cada 5 segundos.

Por ejemplo:

* Una consulta con un rango de tiempo de 7 días tendría una frecuencia de actualización de una vez cada 2,8 horas.
* Una consulta con un rango de tiempo de 1 hora tendría una frecuencia de actualización de una vez cada minuto.

Para consultas que utilizan `TIMESERIES`, el tamaño del depósito `TIMESERIES` se utiliza como intervalo de actualización. Por ejemplo:

* `SELECT count(*) FROM PageView TIMESERIES 5 minutes` se actualizaría cada 5 minutos.
* `SELECT count(*) FROM PageView TIMESERIES 1 hour SINCE 1 day ago` se actualizaría cada hora.
