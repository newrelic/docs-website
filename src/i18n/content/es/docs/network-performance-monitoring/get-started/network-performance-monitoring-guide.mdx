Eres un ingeniero de operaciones que depende de una red de dispositivos habilitados para SNMP. Su equipo ya utiliza la característica de New Relic para monitoreo de software e infraestructura, pero es hora de monitor también sus dispositivos de red.

En esta pista, utiliza la biblioteca ktranslate de Kentik para descubrir automáticamente sus dispositivos de red, recopilar datos y enviarlos a New Relic para correlacionar el rendimiento de la red con la infraestructura, la aplicación y las experiencias digitales. Luego usa New Relic para monitor el estado de su red.

## Objetivos

* Sondee datos SNMP de dispositivos de red y envíelos a New Relic
* Muestra de datos de flujo de red y envíalos a New Relic
* Cree un New Relic dashboard para monitorear sus dispositivos de red
* Cree una carga de trabajo New Relic para agrupar lógicamente sus dispositivos y configurar la detección de anomalías
* Utilice sus nuevos datos para comprender los comportamientos dentro de su red

## Requisitos

* Una cuenta gratuita de New Relic
* Un usuario de plataforma completa o un usuario principal con privilegios de usuario de modificación de Nerdpack

<Callout variant="important">
  Esta pista utiliza una máquina virtual efímera. Como resultado, debes terminar toda la pista de una sola vez o perderás tu progreso cuando se acabe el tiempo.
</Callout>
