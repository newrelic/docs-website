Después de configurar los datos de su red para el monitoreo del rendimiento, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Network**</DNT> para explorar los datos de la red que sus dispositivos informan a New Relic.

Puede agrupar y monitor sus conmutadores, enrutadores, etc. en el tablero. Esto le ayuda a agregar, aislar y analizar rápidamente el estado de su red.
