La clave del éxito en DevOps y la ingeniería de confiabilidad del sitio radica en reducir el tiempo medio de resolución (MTTR). El widget puede ayudarte a hacerlo, incluso cuando no estés frente a tu computadora.

## Antes de comenzar con el widget [#requirements]

Para configurar [el widget de iOS](https://support.apple.com/en-us/HT207122) en su iPhone o iPad, asegúrese de estar ejecutando iOS 14 o superior y tener instalada la [aplicación New Relic Mobile ](install-newrelic-ios-mobile-app).

A continuación se muestra un ejemplo de cómo agregar un widget a la pantalla de inicio:

1. Toque y mantenga presionado un área o widget vacío hasta que los íconos se muevan.

2. Toca el icono <DNT>**+**</DNT> en la esquina superior izquierda.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the Add (+) button"
     alt="Screenshot showing the Add (+) button"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-edit-home-screen.webp"
   />

3. Busque el widget <DNT>**New Relic**</DNT> .

4. Desliza el dedo hacia la izquierda o hacia la derecha para seleccionar entre tres tamaños de widget y luego toca <DNT>**Add Widget**</DNT>.

   <Video
     type="wistia"
     id="qyk74p7j56"
     vertical
     width="290px"
   />

5. Toca <DNT>**Done**</DNT>.

## Agregar un gráfico a un widget [#add-chart]

Si aún no ha seleccionado el gráfico que desea mostrar en la aplicación New Relic, siga estos pasos:

1. Navegue hasta el gráfico que desee ver en el widget.

2. Toque el gráfico para mostrar la vista detallada.

3. Toque el icono de puntos suspensivos <DNT>**...**</DNT> en la esquina superior derecha para mostrar más opciones.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the dropdown options in the ellipsis"
     alt="Screenshot showing the dropdown options in the ellipsis menu"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-add-widget-from-chart.webp"
   />

4. Seleccione <DNT>**Add to widgets**</DNT>.

5. También puedes eliminar y reorganizar el widget con <DNT>**Manage widgets**</DNT>.

     type="wistia"

     width="290px"

## Designar gráficos para widget [#designate-charts]

Para seleccionar qué gráfico se muestra para cada uno de los widgets de iOS:

1. Mantenga presionado el widget hasta que se mueva.

2. Toca <DNT>**Select chart**</DNT> o el nombre del gráfico.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the option to select a chart"
     alt="Screen showing the option to select a chart"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-configure-widget.webp"
   />

3. Si lo prefieres, puedes ocultar el título y/o leyenda de esta pantalla.
