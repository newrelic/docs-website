La aplicación New Relic para Apple TV brinda fácil acceso a los datos [del panel de New Relic](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) . Esto es útil, por ejemplo, para mostrar datos críticos en sus quioscos, en sus salas de conferencias o en cualquier lugar donde desee que sus equipos o clientes vean las tendencias de los datos.

## Instalacion [#install]

Para instalar e log en la aplicación New Relic Apple TV:

1. Descarga la aplicación desde la [tienda de aplicaciones de Apple TV](https://support.apple.com/guide/tv/find-apps-atvb5a7dd799/tvos).

2. Log sesión con las credenciales [de la aplicación New Relic para iPhone](/docs/mobile-apps/new-relic-mobile-apps/ios-app/introduction-ios-mobile-app) seleccionando

   <DNT>
     **New Relic iPhone app > Accounts settings > TV icon**
   </DNT>

   e ingresando el código que se muestra en la pantalla. Alternativamente, log sesión con su nombre de usuario y contraseña de New Relic.

<Callout variant="important">
  Es posible que se apliquen [requisitos logadicionales](#account-reqs) según su tipo de cuenta. Revise las [notas de la versión de la aplicación New Relic Apple TV](/docs/release-notes/mobile-apps-release-notes/new-relic-tvos-release-notes) para asegurarse de tener la última versión.
</Callout>

## Privacidad de datos [#privacy]

Nuestras aplicaciones móviles solo registran la información necesaria para ayudar a autenticar y solucionar problemas:

* Dirección de correo electrónico del usuario asociada con su cuenta de New Relic, incluido el nombre y apellido (solo para fines de autenticación)
* Dirección IP
* ID del dispositivo

Para obtener más información, consulte nuestra [documentación sobre seguridad y privacidad de datos móviles](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps).
