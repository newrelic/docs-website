Esto explica cómo autenticar su cuenta de la aplicación New Relic Mobile y cómo agregar usuarios o eliminarlos de su dispositivo móvil.

## Autenticacion de usuario [#authentication]

Dependiendo de su cuenta de New Relic, es posible que se requieran pasos adicionales de instalación o autenticación al instalar la aplicación New Relic Mobile .

<table>
  <thead>
    <tr>
      <th width={200}>
        <DNT>
          **New Relic account**
        </DNT>
      </th>

      <th>
        <DNT>
          **Additional requirements**
        </DNT>
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Nuevo usuario
      </td>

      <td>
        Si aún no tienes una cuenta New Relic:

        1. Desde browser web de su escritorio, [cree una cuenta New Relic](https://newrelic.com/signup).

        2. [Instale su aplicación](/docs/site/installing-the-agent) con el agente New Relic apropiado.

           Como parte de la configuración de una nueva cuenta, recibirá un correo electrónico con un enlace para restablecer la contraseña. El enlace para restablecer la contraseña caduca después de 20 minutos para las aplicaciones móviles.
      </td>
    </tr>

    <tr>
      <td>
        Usuario existente de New Relic
      </td>

      <td>
        Sin requisitos adicionales; su aplicación, hosts y clave de transacción aparecen automáticamente después de iniciar sesión.
      </td>
    </tr>

    <tr>
      <td>
        Usuario con cuentas de socios de New Relic
      </td>

      <td>
        Dependiendo del socio, es posible que tengas que completar un [proceso de autenticación](/docs/ios/ios-with-partners) diferente.

        <DNT>**Azure Store users:**</DNT> Debido a la profunda integración entre Azure Storefront y New Relic, el usuario de Azure Storefront no puede acceder a sus cuentas en las aplicaciones New Relic para Android o iOS.
      </td>
    </tr>

    <tr>
      <td>
        Usuario con cuentas habilitadas para SAML-SSO
      </td>

      <td>
        Cuando inicia sesión en la aplicación New Relic Mobile , su sesión se redirige automáticamente a su browser web. Desde allí puede iniciar sesión en su cuenta SAML-SSO de New Relic.

        Si ve algún error al utilizar cuentas SAML-SSO en su dispositivo móvil, verifique que pueda iniciar sesión en <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT> con un browser web de escritorio.

        * En caso negativo, comuníquese con su administrador.
        * En caso afirmativo, obtenga asistencia en [support.newrelic.com](https://support.newrelic.com).
      </td>
    </tr>
  </tbody>
</table>

  <Collapser
    id="switch-accounts"
    title="Cambiar entre cuentas"
  >
    Para cambiar entre cuentas asociadas con su nombre de usuario:

    1. En el menú

       <DNT>
         **Users**
       </DNT>

       , seleccione el nombre de usuario.

    2. Seleccione el nombre de la cuenta.
  </Collapser>

  <Collapser
    id="Remove a username"
    title="Eliminar o volver a agregar un nombre de usuario"
  >
    Para eliminar un nombre de usuario específico de este dispositivo:

    1. En el menú <DNT>**Users**</DNT> , seleccione <DNT>**Logout**</DNT>.

    2. Para eliminar a un usuario de este dispositivo, seleccione el signo menos rojo del usuario <Icon style={{color: 'red'}} name="fe-minus-circle"/> icono.

    3. Seleccione el icono <DNT>**Log out**</DNT> del usuario.

       Para agregar un usuario nuevamente, inicie sesión con ese nombre de usuario nuevamente.
  </Collapser>
