Le recomendamos revisar [nuestros documentos sobre tipos de muestreo](/docs/distributed-tracing/concepts/how-new-relic-distributed-tracing-works) antes de comenzar para determinar si desea utilizar el rastreo distribuido estándar o [el rastreo infinito](/docs/distributed-tracing/infinite-tracing/introduction-infinite-tracing). Además, si actualmente utiliza New Relic <InlinePopover type="apm"/>agente y desea habilitar rastreo distribuido, consulte nuestra [guía de planificación](/docs/distributed-tracing/concepts/distributed-tracing-planning-guide).

¿Listo para comenzar? Si aún no lo ha hecho, cree su cuenta New Relic gratuita a continuación para comenzar a monitorear sus datos hoy.

<InlineSignup/>

Para configurar rastreo distribuido, deberás completar tres pasos generales:

<img
  title="Diagram showing the three steps to set up distributed tracing."
  alt="Diagram showing the three steps to set up distributed tracing"
  src="/images/distributed-tracing_diagram_high-level-dist-tr.webp"
/>

1. <DNT>**Identify services:**</DNT> Identifique y anote el extremo, los servicios, los idiomas y los sistemas que se utilizan para completar esta solicitud (necesitará esta información en el siguiente paso). Si tiene un diagrama de entorno como el siguiente, puede usarlo para crear una lista de servicios que manejan solicitudes:

   <img
     title="A sample environment diagram showing how you might identify services."
     alt="A sample environment diagram showing how you might identify services."
     src="/images/distributed-tracing_screenshot-full_distributed-tracing-map.webp"
   />

2. <DNT>**Instrument services:**</DNT> Instrumentado cada servicio que identifiques para que pueda enviar tus datos de traza. Algunas herramientas, como <InlinePopover type="apm"/>agente, instrumentan los servicios automáticamente, mientras que otras herramientas requieren que insertes algún código en los servicios. Haga clic en el icono a continuación para ver los pasos de instrumentación:

3. <DNT>**View traces:**</DNT> Después de instrumentar los servicios, genere algo de tráfico en su aplicación y luego vaya a la [UI de New Relic](https://one.newrelic.com/launcher/distributed-tracing.launcher) para ver los datos de su traza.
