El rastreo distribuido rastrea y observa las solicitudes de servicio a medida que fluyen a través de sistemas distribuidos.

Las solicitudes pueden pasar a través de varios servicios para completarse, y estos servicios pueden estar en una variedad de lugares: contenedor, entornos sin servidor, máquina virtual, diferentes proveedores de nube o en las instalaciones. Cuando puede ver la ruta de una solicitud completa a través de diferentes servicios, puede identificar rápidamente fallas o problemas de rendimiento.

El rastreo distribuido recopila datos a medida que las solicitudes viajan de un servicio a otro, registrando cada segmento del viaje como un lapso. Estos tramos contienen detalles importantes sobre cada segmento de la solicitud y eventualmente se combinan en una sola traza. La traza completa le brinda una imagen de toda la solicitud.
