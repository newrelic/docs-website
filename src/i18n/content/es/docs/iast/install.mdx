<Callout variant="important">
  Ejecute IAST solo con despliegue sin producción. Pruebas IAST invocando solicitudes HTTP con una carga útil de exploit. Debe utilizar IAST en un entorno de no producción solo con datos simulados para evitar la corrupción de datos y la introducción de exploits en su código en vivo. Además, cuando ejecuta IAST en código nuevo en preproducción, detecta vulnerabilidades potenciales antes de que entren en funcionamiento.
</Callout>

Para instalar New Relic IAST:

<Steps>
  <Step>
    Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com) > All capabilities > IAST**</DNT> y haga clic en <DNT>**Set up IAST with applications**</DNT>.

    <img
      title="Set up IAST with applications"
      alt="Set up IAST with applications"
      src="/images/iast_screenshot-full_install.webp"
    />
  </Step>

  <Step>
    Confirme que IAST no se esté ejecutando en un entorno de producción y haga clic en <DNT>**Continue**</DNT>.
  </Step>

  <Step>
    Seleccione el idioma de su aplicación y complete los pasos.

    <img
      title="Install New Relic IAST"
      alt="Install New Relic IAST"
      src="/images/iast_screenshot-crop_install.webp"
    />
  </Step>

  <Step>
    Asegúrese de que su archivo de configuración `newrelic.yml` esté actualizado de la siguiente manera:

    <CollapserGroup>
      <Collapser
        id="config-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> archivo de configuración</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
        ```
      </Collapser>

      <Collapser
        id="config-eu-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> archivo de configuración para la UE</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
        validator_service_url: wss://csec.eu01.nr-data.net
        ```
      </Collapser>

      <Collapser
        id="config-file-fed-example"
        title={<><InlineCode>newrelic.yml</InlineCode> archivo de configuración para el usuario Fed</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
        validator_service_url: wss://csec-gov.nr-data.net
        ```
      </Collapser>
    </CollapserGroup>
  </Step>

  <Step>
    Una vez que haya completado todos los pasos, reinicie su aplicación y genere tráfico en las API de la aplicación.
  </Step>

  <Step>
    Haga clic en <DNT>**See your data**</DNT> para ver una descripción general de su aplicación probada.
  </Step>
</Steps>
