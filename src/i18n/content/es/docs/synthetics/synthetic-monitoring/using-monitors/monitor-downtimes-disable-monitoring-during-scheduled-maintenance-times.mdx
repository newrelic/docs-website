Programe <DNT>**monitor downtimes**</DNT> para especificar las horas en las que sus [monitores](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors#types-monitors) Sintético dejan de emitir alertas y, al mismo tiempo, conserva su [informe métrico de SLA](/docs/synthetics/new-relic-synthetics/pages/synthetics-sla-report-aggregate-monitor-metrics). Los monitores de tiempo de inactividad son ideales para:

* Mantenimiento de rutina
* Cortes planificados
* Despliegue
* Interrupciones del servicio

<Callout variant="important">
  Durante el tiempo de inactividad monitor Sintético, el monitor seleccionado deja de funcionar hasta la hora de finalización programada. Para desactivar temporalmente <InlinePopover type="alerts"/>sin pausar el monitor, [silencialos](/docs/alerts-applied-intelligence/new-relic-alerts/alert-notifications/muting-rules-suppress-notifications) .
</Callout>

## Consejos monitor el tiempo de inactividad [#tips]

Si las horas de inicio y finalización del tiempo de inactividad del monitor abarcan la medianoche, el tiempo de inactividad del monitor durará hasta el día siguiente a la hora de finalización especificada. Por ejemplo, una hora de inicio a las 11:30 p. m. y una hora de finalización a las 5:00 a. m. comenzaría el día indicado y duraría hasta las 5:00 a. m. del día siguiente.

## Crear monitor recurrente de tiempo de inactividad [#create-window]

<Callout variant="tip">
  La capacidad de agregar, editar o eliminar el monitor de tiempo de inactividad depende de [su acceso a la característica](/docs/accounts/accounts-billing/general-account-settings/factors-affecting-access-features-data).
</Callout>

Cree un tiempo de inactividad de monitor recurrente para el mantenimiento de rutina del monitor o interrupciones programadas regularmente. Para crear un tiempo de inactividad recurrente del monitor:

1. Vaya a

   <DNT>
     **[one.newrelic.com > Synthetic monitoring > Monitor downtime](https://one.newrelic.com/synthetics-nerdlets/monitor-downtime-list)**
   </DNT>

   .

2. Seleccione

   <DNT>
     **<Icon name="fe-plus-circle"/> Create monitor downtime**
   </DNT>

   .

3. Especifique un nombre para el tiempo de inactividad del monitor.

4. Seleccione una frecuencia diaria, semanal o mensual. Elija opciones de programación adicionales, como el día de la semana, la hora del día y la duración de la ventana.

5. Seleccione los monitores que le gustaría incluir en el tiempo de inactividad monitor . No hay un máximo para la cantidad de monitores que se pueden seleccionar.

6. Seleccione

   <DNT>
     **Create**
   </DNT>

   .

## Crear un tiempo de inactividad del monitor por única vez [#one-time]

Para mantenimiento espontáneo o interrupciones del servicio, cree un monitor de tiempo de inactividad único que no volverá a ocurrir. Para crear un tiempo de inactividad del monitor no recurrente:

1. <DNT>
     **[one.newrelic.com > Synthetic monitoring > Monitor downtime](https://one.newrelic.com/synthetics-nerdlets/monitor-downtime-list)**
   </DNT>

2. Especifique un nombre para el nuevo tiempo de inactividad del monitor.

3. Seleccione

   <DNT>
     **once**
   </DNT>

   para la frecuencia, junto con la fecha y la hora de inicio del tiempo de inactividad del monitor.

4. Seleccione los monitores para incluir en el tiempo de inactividad monitor . No hay un máximo para la cantidad de monitores que se pueden seleccionar.

5. Seleccione

   <DNT>
     **Create**
   </DNT>

   .

## Eliminar un tiempo de inactividad del monitor [#delete]

<Callout variant="tip">
  Si no puede eliminar una ventana, verifique sus [permisos](/docs/synthetics/new-relic-synthetics/administration/synthetics-permissions-user-groups).
</Callout>

Para eliminar un tiempo de inactividad de un monitor existente:

1. Vaya a

   <DNT>
     **[one.newrelic.com > Synthetic monitoring > Monitor downtime](https://one.newrelic.com/synthetics-nerdlets/monitor-downtime-list)**
   </DNT>

   .

2. Localice el tiempo de inactividad del monitor que se eliminará en el índice.

3. Seleccione el icono de puntos suspensivos

   <DNT>
     **<Icon name="fe-more-horizontal"/>**
   </DNT>

   y luego

   <DNT>
     **Delete**
   </DNT>

   .

## Editar un tiempo de inactividad del monitor [#edit]

<Callout variant="tip">
  Si no puede editar una ventana, verifique sus [permisos](/docs/synthetics/synthetic-monitoring/administration/user-roles-synthetic-monitoring/).
</Callout>

Para editar el tiempo de inactividad de un monitor existente:

1. Vaya a

   <DNT>
     **[one.newrelic.com > Synthetic monitoring > Monitor downtime](https://one.newrelic.com/synthetics-nerdlets/monitor-downtime-list)**
   </DNT>

   .

2. Localice el tiempo de inactividad del monitor que desea editar en el índice de tiempo de inactividad del monitor.

3. Seleccione el icono de puntos suspensivos

   <DNT>
     **<Icon name="fe-more-horizontal"/>**
   </DNT>

   para el tiempo de inactividad que desea editar.

4. Edite el nombre o la frecuencia del tiempo de inactividad del monitor. Para eliminar un monitor de la lista de objetivos, ubique el monitor en la lista de

   <DNT>
     **Selected targets**
   </DNT>

   y seleccione el ícono de eliminar

   <DNT>
     **<Icon name="fe-x-circle"/>**
   </DNT>

   al lado del monitor.

5. Una vez que haya terminado de editar, seleccione

   <DNT>
     **Save**
   </DNT>

   .
