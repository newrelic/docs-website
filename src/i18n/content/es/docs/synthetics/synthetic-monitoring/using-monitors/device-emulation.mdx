La emulación de dispositivo es una configuración opcional disponible para simplificar y monitorear el script del browser. Esta configuración simula la ventana gráfica y las capacidades de un dispositivo móvil o tableta utilizando las capacidades de emulación de Chrome DevTools.

La emulación de dispositivo le permite probar una aplicación basada en browserutilizando un dispositivo móvil o tableta genérico simulado. Tenga en cuenta:

* Esta configuración no cambia la red, el agente de usuario o el tipo de dispositivo que se utiliza para ejecutar su monitor Sintético.
* La emulación de dispositivo no le permite instalar ni probar aplicaciones móviles.

## Requisitos [#requirements]

* Solo tiempos de ejecución de Chrome 100+
* La ubicación privada debe utilizar la versión 225 o superior [del administrador de trabajos](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) y la versión 1.1.98 de Sintético-node-browser-runtime. o mas alto
