La emulación de dispositivo es una configuración opcional disponible para scripts de browser simples y de monitor. Esta configuración simula la ventana gráfica y las capacidades de un dispositivo móvil o tableta empleando las capacidades de emulación de DevTools en Chrome y Firefox.

La emulación de dispositivo le permite probar una aplicación basada en browserutilizando un dispositivo móvil o tableta genérico simulado. Tenga en cuenta:

* Esta configuración no cambia la red, el agente de usuario o el tipo de dispositivo que se utiliza para ejecutar su monitor Sintético.
* La emulación de dispositivo no le permite instalar ni probar aplicaciones móviles.

## Requisitos [#requirements]

* Solo entornos de ejecución no heredados (no compatible con Chrome 72)
* Chrome: la ubicación privada debe usar la versión 225 o superior [del administrador de trabajos](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) y la versión 1.1.98 de Sintético-node-browser-runtime o superior
* Firefox: la ubicación privada debe usar la versión 400 o superior [del administrador de trabajos](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) y la versión 3.0.0 de Sintético-node-browser-runtime o superior
