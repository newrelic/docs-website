New Relic is [ending support for older runtimes](/eol/2026/05/eol-05-18-26-node-16-chrome-134-runtimes/) in 2026. To simplify migrating synthetic monitors to the latest supported runtimes, we auto-upgraded every monitor that didn&apos;t require your action. Use the runtime upgrade tool to view the remaining monitors that still need your attention.

## Actualiza tus monitores restantes [#upgrade-options]

Para cada monitor que requiera atención, puede elegir uno de dos caminos.

  <TabsPages>
    <TabsPageItem id="fix-manually">
      Para actualizar un monitor escribiendo o editando manualmente los scripts de su monitor, siga los pasos.

      1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com)** &gt; **Synthetic Monitoring** &gt; **Runtime Upgrades**</DNT>.

      2. Ubique el monitor y haga clic en <DNT>**Fix manually**</DNT>.

      3. En la ventana <DNT>**Fix script manually**</DNT>, actualice su código de script para el nuevo entorno de ejecución.

      4. Haga clic en <DNT>**Close and start validation**</DNT>. El monitor se mueve a la tabla <DNT>**Monitors we&apos;re validating**</DNT> mientras el sistema valida el script actualizado.

         * Si la validación falla, haga clic en **⋯** junto al monitor, seleccione <DNT>**Fix manually**</DNT> del menú, actualice el script y haga clic en <DNT>**Close and start validation**</DNT> de nuevo.

      5. Una vez que la validación sea exitosa y el estado se muestre como <DNT>**Fixed**</DNT>, haga clic en la pestaña <DNT>**Ready to update**</DNT>.

      6. Seleccione la casilla de verificación junto al monitor validado y haga clic en <DNT>**Upgrade selected**</DNT>.

      Una vez finalizado, aparece una notificación de confirmación y el monitor se mueve a la pestaña <DNT>**Updated**</DNT> donde puede verificar la actualización exitosa.
    </TabsPageItem>

    <TabsPageItem id="fix-with-ai">
      <Callout title="Nota">
        * Esta característica requiere Calcular avanzado y no está disponible para cuentas <DNT>**FedRAMP**</DNT>. Comuníquese con el equipo de su cuenta de New Relic para obtener soporte.
        * Siempre pruebe los scripts generados por IA antes de desplegar los monitores en producción.
      </Callout>

      Use this option to let <DNT>**New Relic AI (NRAI)**</DNT> generate an updated script for you. The AI-assisted script generation is mainly geared toward fixing `timeout errors` incurred during runtime upgrades. For monitors with `script errors`, the script potentially needs other changes. You can make those manually or use <DNT>[SRE Agent](/docs/agentic-ai/sre-agent/overview/)</DNT> to debug and get insights on potential script-related updates.

      Para actualizar un monitor usando IA:

      1. Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com)** &gt; **Synthetic Monitoring** &gt; **Runtime Upgrades**</DNT>.
      2. Ubique el monitor y haga clic en <DNT>**Fix with AI**</DNT>.
      3. Revise el script sugerido por IA. Para ver los cambios específicos realizados en su código original, haga clic en <DNT>**Show comparison**</DNT> para abrir una vista de diferencias en paralelo.
      4. Haga clic en <DNT>**Continue to validation**</DNT>. El sistema coloca temporalmente el monitor en un estado de validación para probar el nuevo script y asegurarse de que se ejecute sin errores.
      5. Una vez validado el script, haga clic en la pestaña <DNT>**Ready to update**</DNT>.
      6. Seleccione la casilla de verificación junto al monitor validado y haga clic en <DNT>**Upgrade selected**</DNT>.

      Una vez finalizado, aparece una notificación de confirmación y el monitor se mueve a la pestaña <DNT>**Updated**</DNT> donde puede verificar la actualización exitosa.
    </TabsPageItem>
  </TabsPages>
</Tabs>

También puedes validar varios monitores en paralelo y compartir los enlaces con tus compañeros de equipo para dividir el trabajo. Una vez que todos tus scripts estén validados, puedes desplegar los monitores en producción cuando estés listo.
