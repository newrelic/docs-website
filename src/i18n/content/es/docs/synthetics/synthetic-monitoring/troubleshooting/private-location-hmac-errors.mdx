## Problema

Uno de tus minion privado no pasa las comprobaciones y reporta uno de estos errores:

```
HMAC MISMATCH ERROR: HMAC mismatch between saved HMAC and calculated HMAC at job run time
```

```
HMAC MISMATCH ERROR: HMAC expected but missing from Job metadata
```

Estos errores indican un error con [la ejecución del script verificado](/docs/synthetics/new-relic-synthetics/private-locations/verified-script-execution-private-locations).

## Solución

Deberá ingresar la contraseña correcta para la ubicación desde la cual falla esta verificación.

Para ingresar la contraseña:

1. En

   <DNT>
     **[one.newrelic.com > Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)**
   </DNT>

   \> seleccione el monitor que falla.

2. Vaya a

   <DNT>
     **Settings > Script**
   </DNT>

   .

3. En los campos de contraseña de ubicación, envíe la contraseña correcta.

<Callout variant="tip">
  New Relic no almacena las contraseñas de ubicación privada. Si no conoce la contraseña, busque a la persona de su equipo que configuró la ubicación privada. Si no es posible recuperar la contraseña, tu única opción es crear una nueva ubicación privada.
</Callout>
