Si es un [usuario de plataforma completa](/docs/accounts/accounts-billing/new-relic-one-pricing-users/users-roles#user-type), en teoría tiene acceso completo a todas las características de New Relic, pero si puede usar todas esas características depende del rol que se le asigne dentro de su organización de New Relic.

Para obtener detalles sobre cómo funcionan los permisos de New Relic, consulta [Conceptos importantes de administración de usuarios y acceso](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts). Para ver las capacidades granulares relacionadas con Synthetics, consulte [Capacidades](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/#synthetics).
