Tenemos dos opciones para administrar su monitor New Relic Synthetics a través de API:

* NerdGraph (recomendado): esta es la forma principal y recomendada. Vea [nuestro tutorial NerdGraph Sintético](/docs/apis/nerdgraph/examples/nerdgraph-synthetics-tutorial).
* API REST: algunos clientes prefieren utilizar nuestra API REST anterior. Ver [API REST de Sintético](/docs/apis/synthetics-rest-api/monitor-examples/manage-synthetics-monitors-rest-api)

Para obtener una descripción general de todas nuestras API disponibles, consulte [Introducción a las API](/docs/apis/getting-started/introduction-new-relic-apis).
