New Relic utiliza minion público para ejecutar su [monitor Sintético](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors). Estos minion están desplegados en diferentes centros de datos alrededor del mundo y están a cargo de ejecutar su monitor.

Debido a esto, asegúrese de que su firewall permita solicitudes de red de Sintético. Las direcciones IP minion están disponibles públicamente en formato JSON, por lo que puedes leerlas y analizarlas fácilmente.

<Callout variant="important">
  Asegúrese de que su lista de &apos;permitidos&apos; o configuración relacionada incluya todos los rangos en las [listas de rangos de IP](#dynamic-ip-addresses) , ya que las solicitudes pueden enviarse desde cualquier entorno.
</Callout>

<DNT>**Optional steps**</DNT>:

* Puede automatizar sus reglas para su lista de &apos;permitidos&apos; basándose en estos archivos.
* Puede identificar monitor las solicitudes sintética empleando [nuestros encabezados estándar](/docs/synthetics/new-relic-synthetics/administration/identify-synthetics-requests-your-app) o agregando un encabezado personalizado a su [ping o simple browser monitor](/docs/synthetics/synthetic-monitoring/using-monitors/add-edit-monitors/#simple), [API monitor](/docs/synthetics/synthetic-monitoring/scripting-monitors/write-synthetic-api-tests/#request-options) [con scriptmonitor o de con script browser ](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetic-scripted-browser-reference-monitor-versions-chrome-100#structure).

## Las direcciones IP no son datos personales [#not-personal-data]

Los minion se implementan en los servidores y los agentes se activan utilizando credenciales no personales. Las direcciones IP asociadas con Minion que se ejecuta en servidores no son datos personales según las leyes de privacidad y protección de datos. Para más información consulta la [documentación de seguridad de Sintético](/docs/synthetics/new-relic-synthetics/getting-started/security-new-relic-synthetics).

## Listados JSON diarios para direcciones IP [#dynamic-ip-addresses]

Las direcciones IP de las ubicaciones publicadas están sujetas a cambios. Si es necesario un cambio, intentaremos notificar proactivamente a los clientes antes de cualquier cambio por correo electrónico. También puede consultar el [Foro de soporte](https://discuss.newrelic.com/) para obtener actualizaciones. Los rangos de IP enumerados están reservados para uso de New Relic y nadie más puede utilizarlos.
