New Relic proporciona un portal web para que los socios accedan fácilmente a recursos valiosos que les proporciona el Programa de socios de New Relic, como contenido de habilitación de ventas para socios, capacitación, materiales de marketing, etc.

Si es socio y aún no tiene acceso a esto, comuníquese con su gerente de ventas de socios o envíe un correo electrónico a partnersales @ newrelic.com.

Para acceder al Portal de socios: Desde su browser, escriba (o marque) la siguiente URL y luego log sesión con su nombre de usuario y contraseña.

```
https://partners.newrelic-external.com/English/
```
