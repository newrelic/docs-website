## Múltiples subasociaciones [#multiple-names]

Si su asociación hace negocios con varios nombres o en varios mercados diferentes utilizando diferentes marcas, puede marcar de manera distintiva las diferentes asociaciones. Esta funcionalidad está disponible utilizando múltiples sub-asociaciones bajo una única cuenta de propietario de asociación. Las asociaciones pueden ser distintas con varias configuraciones, que incluyen:

* Nombre
* Precio del producto
* Branding (encabezados, pies de página, etc.)
* Configuración de facturación

Después de log en su cuenta de propietario de asociación, puede seleccionar de la lista de cuentas de sub-asociación.
