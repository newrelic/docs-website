Siga estos pasos para configurar un nombre de host DNS privado para el clúster de puerta de enlace para facilitar una comunicación confiable entre el agente de New Relic APM y la puerta de enlace dentro de su red:

1. Cree un nombre de host DNS empleando [AWS Route 53](https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/hosted-zones-private.html).
2. Obtenga un certificado para el nombre de host de la puerta de enlace a través de [ACM](https://docs.aws.amazon.com/acm/latest/userguide/acm-public-certificates.html).
