## Problema: Agotamiento de recursos

El clúster agotó sus recursos de CPU o memoria.

## Solución

* Emplee la Kubernetes UI pod para ver eventos y trabajos pendientes para identificar restricciones de recursos.
* Aumente el tamaño del grupo de nodos o ajuste los límites de recursos (CPU y memoria) requests para el pod.
* Verifique los límites del proveedor cloud para la cantidad de nodos activos y ajuste la configuración según sea necesario.
