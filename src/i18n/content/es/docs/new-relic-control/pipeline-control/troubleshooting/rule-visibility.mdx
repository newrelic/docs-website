\* Si cambió una regla de puerta de enlace y no aparece en la lista de reglas, esta guía lo ayudará a diagnosticar y resolver problemas comunes. Siga los pasos a continuación según el problema específico que encuentre: \*/

## Problema: cambio no implementado

No sabes que la lista de reglas solo muestra los estados activos y aún no implementó el cambio.

## Solución

* Consulte la página de actualizaciones para ver las actualizaciones de puerta de enlace pendientes para implementar.
* Cerciorar de implementar los cambios para que se reflejen en la lista de reglas.

## Problema: condición de carrera de la API

Varios usuarios envían requests de actualizaciones de puerta de enlace simultáneamente, lo que provoca que se pierdan los cambios.

## Solución

* Comunicar con [el soporte de New Relic](https://support.newrelic.com).
* Rehacer la acción que se perdió debido a la condición de carrera.
