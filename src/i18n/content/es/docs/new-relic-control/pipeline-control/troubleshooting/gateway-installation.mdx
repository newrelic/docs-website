\* Si está intentando instalar la puerta de enlace por primera vez y encuentra problemas, esta guía lo ayudará a diagnosticar y resolver problemas comunes. Siga los pasos a continuación según el problema específico que encuentre: \*/

## Problema: Capacidades de usuario insuficientes

No tienes las licencias necesarias asociados con Org Product Admin y Organización Manager.

## Solución

* El usuario verá un mensaje de error que indica: &quot;No tiene las capacidades a nivel de organización necesarias para configurar la autenticación del agente&quot;.
* Involucre a los administradores de su cuenta para que se les otorgue un rol que incluya las capacidades necesarias para la creación de identidad del sistema.
* Consulte [la documentación de licencias de usuario](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/) para obtener orientación.

## Problema: Versión obsoleta del gráfico de Helm

El script de línea de comando para instalar el gráfico Helm falla porque su versión local de Helm no está actualizada.

## Solución

* Actualice su instalación local de Helm a la última versión para garantizar la compatibilidad con el script de instalación.
* Busque el mensaje de error que indica la necesidad de una actualización Helm y siga las instrucciones para actualizar.
