## Problema: Los datos llegan más allá del SLA

Los datos no se están recibiendo dentro del plazo previsto.

## Solución

* Comprobar la latencia métrica para identificar retrasos en la transmisión de datos.
* Aumente la cantidad mínima de pods de puerta de enlace para mejorar la capacidad de procesamiento y reducir la latencia.
