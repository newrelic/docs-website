## Problema: condición de carrera de la API

Varios usuarios envían requests de actualizaciones de puerta de enlace (acciones de creación, actualización y eliminación) simultáneamente, lo que provoca que se pierdan los cambios.

## Solución

* Comunicar con [el soporte de New Relic](https://support.newrelic.com).
* Rehacer la acción que se perdió debido a la condición de carrera.
