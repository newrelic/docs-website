Este documento describe los pasos para solucionar problemas comunes durante la instalación u operación de control de flota.

## Fallo de implementación

Se produce un error en la implementación y se necesita más información para comprender el error.

## Solución

Haga clic en el clúster fallido para acceder a información detallada sobre el error. Esta valiosa información adicional puede ayudar a diagnosticar y resolver el problema de manera efectiva.
