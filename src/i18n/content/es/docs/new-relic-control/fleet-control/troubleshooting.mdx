Este documento describe los pasos para solucionar problemas comunes durante la instalación u operación de control de flota.

## Uso del explorador de datos

El New Relic Data Explorer es una herramienta poderosa para investigar problemas. Puedes acceder a él haciendo clic en **Query your data** en la parte inferior de la mayoría de las páginas de la plataforma New Relic, o empleando el atajo de teclado `CTRL+SHIFT+O`.

Para obtener información más detallada sobre sus características, consulte nuestra [introducción al Explorador de datos](/docs/query-your-data/explore-query-data/query-builder/introduction-new-data-explorer/).

### Investigar despliegue de eventos con NRQL

Para problemas de implementación complejos, puede ejecutar una consulta NRQL en el Explorador de datos para ver los datos granulares a nivel de evento para una implementación específica. Esto le permite ver la secuencia exacta de acciones y cualquier error que ocurrió.

Para investigar un despliegue:

1. Desde la pestaña de **Deployments** de una flota específica, haga clic en el nombre del despliegue que desea investigar. El **ID del despliegue** aparecerá en los detalles del resumen. Copiar esta identificación.
2. Abra el Explorador de datos.
3. Cerciorar de que esté seleccionada la cuenta correcta.
4. Pegue la siguiente consulta en el generador de consultas, reemplazando el marcador de posición con el ID que copió.
5. Ejecute la consulta. Para facilitar el análisis, vea los resultados como una **Table**. La columna `details` a menudo contiene un mensaje de error esclarecedor.

```nrql
FROM FleetDeployment SELECT timestamp, action, status, desiredCompletedStatus, agentTypes, ringName, details, agentUid, agentControlUids, fleetId, deploymentId, duration, action WHERE deploymentId = '<YOUR_DEPLOYMENT_ID>' SINCE 1 hour ago LIMIT MAX
```
