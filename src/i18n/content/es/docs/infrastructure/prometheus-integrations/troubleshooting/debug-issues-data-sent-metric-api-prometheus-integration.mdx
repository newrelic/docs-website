## Problema

Tiene problemas para identificar la causa detrás de los problemas reportados con el envío de datos a New Relic [métrica API](/docs/data-ingest-apis/get-data-new-relic/metric-api/introduction-metric-api) de . Quiere ver los datos exactos que se envían desde su integración Prometheus OpenMetrics para docker o Kubernetes a la API métrica.

## Solución

Puede configurar la integración de Prometheus OpenMetrics para mostrar los datos sin procesar que se envían al flujo de salida estándar. Para ver los datos exactos que se envían a la API de métrica, establezca la variable de entorno `EMITTERS` en `"api,stdout"`.
