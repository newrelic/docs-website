## Versión de infraestructura [#agent-version]

El SDK de integración de infraestructura v4 requiere la versión 1.13.0 o superior del agente de infraestructura.

SDK v3 requiere la versión del agente 1.0.888 o superior.

Para obtener información sobre la versión del agente de infraestructura y opciones de instalación y actualizaciones, consulte las [notas de la versión de infraestructura](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes).

Para obtener notas sobre las versiones y los cambios del SDK, consulte el [registro de cambios](#change-log).

## Sistema operativo [#os]

La integración creada con el SDK se puede compilar para el sistema operativo Linux o Windows.

## Especificaciones de datos y archivos. [#data-specs]

La infraestructura integración en el host se puede crear con cualquier lenguaje de programación, siempre que se ajuste a las [especificaciones de datos y archivos](/docs/infrastructure/integrations-sdk/get-started/create-activate-infrastructure-integration).
