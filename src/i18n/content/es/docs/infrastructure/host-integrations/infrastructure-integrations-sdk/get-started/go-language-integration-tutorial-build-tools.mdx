El [SDK de integración de infraestructura](/docs/infrastructure/integrations-sdk/get-started/intro-infrastructure-integrations-sdk) le permite crear una integración personalizada en el host. Este documento explica las herramientas y recursos de compilación disponibles para crear una integración en el host con nuestras herramientas de lenguaje Go.

## Tutorial de integración [#tutorial]

<Callout variant="important">
  El siguiente tutorial se basa en la integración utilizando el protocolo de integración SDK v3.  
    Encuentre más información sobre el protocolo de integración v4 en el [repositorio de Github](https://github.com/newrelic/infra-integrations-sdk).
</Callout>

El [tutorial de creación de integración del lenguaje Go en GitHub](https://github.com/newrelic/infra-integrations-sdk/blob/master/docs/tutorial.md) brinda procedimientos paso a paso para crear una integración del lenguaje Go que informe datos de Redis.

El tutorial muestra cómo construir una integración usando la línea de comando de Linux, pero puedes usar las mismas técnicas para una integración de Windows con una instalación estándar de Go y PowerShell. El comando `make` no funcionará con PowerShell, pero puede usar los comandos Go que contiene como guía para desarrollar su integración.

<Callout variant="tip">
  Puede crear una integración en el host en cualquier idioma, pero Go es el idioma que New Relic utiliza para sus propias herramientas de integración y compilación. Para crear una integración en otro idioma, cumpla con las [estructuras de archivos de integración y los requisitos de salida JSON](/docs/integrations/integrations-sdk/getting-started/integration-file-structure-activation).
</Callout>

## Paquete de creación de integración de idiomas Go [#package]

El tutorial se basa en un [paquete de biblioteca de creación de integración del lenguaje New Relic Go](https://godoc.org/github.com/newrelic/infra-integrations-sdk), que proporciona un conjunto de funciones y estructuras de datos útiles de Go. El paquete le brinda herramientas que:

* Generar una estructura de integración "scaffold" con todos los campos requeridos.
* Leer valores de argumentos de línea de comando o variables de entorno.
* Genere e imprima datos JSON en `stdout`.

Para obtener información sobre formatos de archivo y especificaciones de salida JSON, consulte [Requisitos de archivo](/docs/integrations/integrations-sdk/getting-started/integration-file-structure-activation).
