Este documento explica los archivos utilizados por nuestro [agente de infraestructura integración en el host](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations).

## Archivos de integración [#overview]

Una integración en el host del agente de infraestructura contiene al menos estos archivos:

* Un [archivo ejecutable](/docs/infrastructure/infrastructure-integrations/custom-integrations/create-integration-executable-file), escrito en cualquier idioma, que exporta datos JSON en un formato esperado por el agente de infraestructura.
* Un [archivo de configuración](/docs/create-integrations/infrastructure-integrations-sdk/specifications/host-integrations-standard-configuration-format)
