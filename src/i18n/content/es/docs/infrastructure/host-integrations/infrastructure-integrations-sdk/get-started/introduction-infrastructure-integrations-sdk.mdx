Hay varias formas en que New Relic te permite crear tu propia integración:

* Soluciones generales de telemetría (métrica, traza): Utilice nuestros [SDK de telemetría](/docs/data-ingest-apis/get-data-new-relic/new-relic-sdks/telemetry-sdks-send-custom-telemetry-data-new-relic).
* Cree una [aplicación New Relic ](/docs/new-relic-one/use-new-relic-one/build-new-relic-one/new-relic-one-build-your-own-custom-new-relic-one-application)personalizada que utilice su propia funcionalidad UI de JavaScript.
* Si tiene New Relic Infrastructure, puede utilizar nuestra ligera [herramienta de integración Flex](/docs/integrations/host-integrations/host-integrations-list/flex-integration-tool-build-your-own-integration) (recomendada) o utilizar nuestro SDK de integración para crear una [integración de infraestructura completa en el host](/docs/integrations/host-integrations/getting-started/introduction-host-integrations).

<Callout variant="important">
  New Relic está haciendo la transición para depender de estándares de código abierto como Prometheus para una futura integración en el host. Aunque este SDK es la base de esa transición, es posible que algunos de los tutoriales y herramientas relacionados con este SDK no estén actualizados con los últimos desarrollos.
</Callout>

## ¿Qué es el SDK de integración de infraestructura? [#what-is-it]

Nuestro SDK de integración de infraestructura es lo que utilizamos para construir nuestro [agente de infraestructura integración en el host](/docs/infrastructure/host-integrations/installation/install-infrastructure-host-integrations), que trabaja en conjunto con nuestro agente de infraestructura para reportar datos de sus hosts o servicios.

Puede estudiar los documentos en esta sección para ayudarlo a comprender la estructura de archivos y las opciones de configuración para esas integraciones. Esto puede resultar útil para resolver problemas y personalizar su integración.
