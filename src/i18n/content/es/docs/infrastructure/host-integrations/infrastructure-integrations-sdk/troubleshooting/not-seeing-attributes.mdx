## Problema

Su integración está reportando datos, pero faltan algunos atributos de inventario.

## Solución

Nuestra plataforma utiliza internamente los siguientes nombres de claves de inventario. Si la integración proporcionada por los clientes envía artículos de inventario con el mismo nombre clave, permanecerán invisibles para el usuario:

* `pid`
* `installed_epoch`
* `epoch`
* `time`
* `size`
* `boot_id`
* `product_uuid`
* `installed_time`
