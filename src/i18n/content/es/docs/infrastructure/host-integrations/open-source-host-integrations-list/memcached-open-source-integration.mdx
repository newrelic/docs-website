La [integración de código abierto](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) de Memcached recopila contadores de rendimiento y estadísticas de [Memcached](https://memcached.org/) y envía datos a New Relic.

Para obtener información sobre requisitos, instalación y configuración, consulte el [archivo README de integración en el host de Memcached](https://github.com/newrelic/nri-memcached) en GitHub.

<Callout variant="important">
  New Relic tiene código abierto esta integración para permitir el monitoreo de esta tecnología. Esta integración se proporciona TAL CUAL SIN GARANTÍA NI SOPORTE, aunque puede informar problemas y contribuir a esta integración a través de GitHub. El soporte para esta integración está disponible con una [suscripción a Servicios Expertos](https://www.newrelic.com/expertservices).
</Callout>
