La integración PerfMon proporciona resultados de consultas y eventos desde el [monitorde rendimiento de Microsoft](https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc749154(v%3dws.11)) a una salida estándar (stdout) que es consumible por New Relic.

Para comenzar, consulte el [archivo README de integración en el host de Windows PerfMon/WMI](https://github.com/newrelic/nri-perfmon) en GitHub.

<Callout variant="tip">
  Esta integración se publica como [código abierto en GitHub](https://github.com/newrelic/nri-perfmon). También hay disponible un log de cambios para las últimas actualizaciones. El soporte para esta integración está disponible con una [suscripción a Servicios Expertos](https://www.newrelic.com/expertservices).
</Callout>
