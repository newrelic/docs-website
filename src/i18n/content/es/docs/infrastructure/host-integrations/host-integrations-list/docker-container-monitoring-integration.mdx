La integración de Docker contenedor monitorea y reporta [métricas desde Docker contenedor usando](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) [Docker API](https://docs.docker.com/reference/api/engine/version/v1.48/) a New Relic.

<Callout variant="tip">
  Esta integración se publica como [código abierto en Github](https://github.com/newrelic/nri-docker). Allí también está disponible un log de cambios con las últimas actualizaciones.
</Callout>

<Callout variant="important">
  * Esta integración no es compatible con LCOW (Linux contenedor en Windows). Para obtener más información, consulte la [documentación de Docker](https://docs.docker.com/desktop/windows/wsl/).
  * Para Linux: esta integración se incluye con el agente de infraestructura y se activará automáticamente si se detectan contenedores.
  * Para Windows: a partir de la versión 2.3.0, ofrecemos compatibilidad, pero requiere una instalación manual sobre el agente de infraestructura. Se incluirá con el agente de infraestructura en futuras versiones.
</Callout>
