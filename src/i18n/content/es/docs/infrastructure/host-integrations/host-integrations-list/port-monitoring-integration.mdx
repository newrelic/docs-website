Nuestra integración de monitoreo de puertos monitorea el estado de los puertos de red, como TCP, UDP, etc., y reporta datos de tiempo de actividad e interrupciones a New Relic.

<img title="port-dashboard" alt="Dashboard of the port monitoring integration" src="/images/infrastructure_screenshot-crop_port-monitoring-dashboard.webp" />

<Callout variant="tip">
  Esta integración se publica como [código abierto en Github](https://github.com/newrelic/nri-port-monitor). Un log de cambios también está disponible allí para las últimas actualizaciones.
</Callout>

## Instalar la integración [#install]

Siga los siguientes pasos para instalar la integración:

<Steps>
  <Step>
    ## Instalar el agente de infraestructura [#infra-install]

    Para usar la integración Port Monitoring, primero debe [instalar el agente de infraestructura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent-new-relic/) en el mismo host. Todas nuestras integraciones en el host requieren el agente de infraestructura, que ayuda a exponer y reportar métricas a New Relic.
  </Step>

  <Step>
    ## Descargar y preparar la integración [#download]

    1. Elija y descargue una versión para su plataforma desde [el repositorio de GitHub](https://github.com/newrelic/nri-port-monitor/releases). Por ejemplo:

       ```shell
       wget https://github.com/newrelic/nri-port-monitor/releases/download/1.4/nri-port-monitor_x86_64.tar.gz && tar -zxvf nri-port-monitor_x86_64.tar.gz  
       ```

    2. Copie el ejecutable `nri-port-monitor` a `/var/db/newrelic-infra/newrelic-integrations`. Copiar `port-monitor-config.yml` a `/etc/newrelic-infra/integrations.d`

       ```shell
       cp nri-port-monitor /var/db/newrelic-infra/newrelic-integrations && cp port-monitor-config.yml /etc/newrelic-infra/integrations.d
       ```

    3. Establezca permisos de ejecución para el archivo binario `nri-port-monitor` para que pueda ejecutarse correctamente:

       ```shell
       chmod +x /var/db/newrelic-infra/newrelic-integrations/nri-port-monitor
       ```
  </Step>

  <Step>
    ## Configurar la integración [#configure-step]

    1. Abra `port-monitor-config.yml` y edite la configuración con todas las instancias de puerto que desee monitorear (agregue múltiples bloques al bloque de integraciones para cada dirección/puerto a monitorear:

       ```yml
       ```

    integraciones:

    * name: nri-port-monitor interval: 60s env:

      # Tipo de red. Las redes conocidas son:

      # tcp, tcp4 (solo IPv4), tcp6 (solo IPv6),

      # udp, udp4 (solo IPv4), udp6 (solo IPv6),

      # ip, ip4 (solo IPv4), ip6 (solo IPv6),

      # unix, unixgram y unixpacket

      RED: tcp

      # Dirección a verificar en el formato host:port

      DIRECCIÓN: localhost:80

      # Tiempo de espera en segundos

      TIMEOUT: 5 etiquetas: env: producción

    ````
    </Step>

    <Step>
    ## Find your data [#find-data]

    To easily find your Port Monitoring data, download our pre-built dashboard:

    1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com/)**</DNT> and click on <DNT>**+ Integrations & Agents**</DNT>.
    2. Click on <DNT>**Dashboards**</DNT> tab.
    3. In the search box, search for <DNT>**Port Monitoring**</DNT>.
    4. Click on the Port Monitoring dashboard to install it.

    Here's an example NRQL query that returns the status of configured ports:

    ```sql
    SELECT latest(status) FROM NetworkPortSample FACET address SINCE 30 MINUTES AGO TIMESERIES
    ````
  </Step>
</Steps>
