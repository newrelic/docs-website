Nuestro monitoreo de integración de puertos monitorea el estado de los puertos de red, como TCP, UDP, etc., y reporta datos a New Relic.

Para comenzar, consulte el [archivo README de integración en el host de monitoreo de puertos](https://github.com/newrelic/nri-port-monitor) en GitHub.

<Callout variant="tip">
  Esta integración se lanza como [código abierto en Github](https://github.com/newrelic/nri-port-monitor). También hay disponible un log de cambios para las últimas actualizaciones. El soporte para esta integración está disponible con una [suscripción a Servicios Expertos](https://www.newrelic.com/expertservices).
</Callout>
