La [integración](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) de Collectd monitorea e informa métrica desde los dispositivos configurados [por Collectd](https://collectd.org/wiki/index.php/Main_Page)a New Relic.

Para comenzar, consulte el [archivo README de integración en el host recopilado](https://github.com/newrelic/nri-collectd) en GitHub.

<Callout variant="tip">
  Esta integración se lanza como [código abierto en Github](https://github.com/newrelic/nri-collectd). También hay disponible un log de cambios para las últimas actualizaciones. El soporte para esta integración está disponible con una [suscripción a Servicios Expertos](https://www.newrelic.com/expertservices).
</Callout>
