Nuestra [integración](/docs/integrations/host-integrations/getting-started/introduction-host-integrations) NFS recopila y envía inventario y métrica de su servicio NFS a nuestra plataforma, donde podrá ver los datos de uso del disco para que pueda encontrar más fácilmente el origen de cualquier problema.

Continúe leyendo para instalar la integración y ver qué datos recopilamos.

## Compatibilidad y requisitos [#req]

Nuestra integración es compatible con NFS versión 3 o 4.

Antes de instalar la integración, asegúrese de cumplir con los siguientes requisitos:

* [Instale el agente de infraestructura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent-new-relic), versión mínima 1.5.40, en el host (cliente NFS)
* Distribución de Linux [compatible con el agente de infraestructura.](/docs/infrastructure/new-relic-infrastructure/getting-started/compatibility-requirements-new-relic-infrastructure#operating-systems)

## Instalar y activar [#install]

La integración NFS ya está activada e instalada de forma predeterminada. Por lo tanto, no hay requisitos de instalación o activación, ya que funciona de inmediato. Vea [ejemplos de variables](/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings#samples-variables) y busque NFS para obtener más información.

<InstallFeedback/>

## Buscar y utilizar datos [#find-and-use]

Para encontrar sus datos de integración, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Third-party services**</DNT> y seleccione el enlace <DNT>**NFS integration**</DNT> .

Al [consultar sus datos](/docs/using-new-relic/data/understand-data/query-new-relic-data), los datos NFS se adjuntan al [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `NFSSample` .

Para obtener más información sobre cómo encontrar y utilizar sus datos, consulte [Comprender y utilizar los datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).

## Deshabilitar la integración [#disable]

Establezca `metrics_nfs_sample_rate` en `-1` en el [archivo de configuración](/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings/#metrics-nfs-sample-rate) del agente de infraestructura para evitar raspar la métrica NFS.
