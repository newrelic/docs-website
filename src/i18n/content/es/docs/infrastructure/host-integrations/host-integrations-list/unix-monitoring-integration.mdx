La integración de monitoreo de Unix permite el monitoreo a nivel de sistema de servidores basados en Unix, como AIX, Linux, macOS y Solaris/SunOS, y reporta datos a New Relic.

Para comenzar, consulte el [archivo README de integración en el host de monitoreo de Unix](https://github.com/newrelic/newrelic-unix-monitor) en GitHub.

<Callout variant="tip">
  Esta integración se lanza como [código abierto en Github](https://github.com/newrelic/newrelic-unix-monitor). También hay disponible un log de cambios para las últimas actualizaciones. El soporte para esta integración está disponible con una [suscripción a Servicios Expertos](https://www.newrelic.com/expertservices).
</Callout>
