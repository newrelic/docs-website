## Sistema operativo y plataforma runtime [#infra-os-runtime]

El agente es compatible con una amplia variedad de entornos. Consulte [Requisitos para el agente de infraestructura](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/) para obtener una lista actualizada de sistemas operativos, arquitecturas y entornos de ejecución compatibles.

El agente de infraestructura apoya a estos sistemas operativos hasta el final de la vida útil de su fabricante. El agente puede ser compatible y puede haber paquetes disponibles para sistemas EOL, pero el soporte no está garantizado.
