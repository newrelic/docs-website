## Causa

Algunas métricas no están presentes en la API habitual del proveedor de la nube (CloudWatch, Stackdriver, monitoreo de Azure) y en su lugar se obtienen de la API del servicio. Cada proveedor de servicios en la nube tiene una API de servicio única que procesa datos e interactúa con el servicio. Por ejemplo, si una métrica no está presente en AWS CloudWatch, New Relic obtendrá la métrica de la API del servicio AWS ECS.
