La integración de New Relic con [Google Cloud Platform (GCP)](https://cloud.google.com/) incluye una integración para reportar datos [de Google BigQuery](https://cloud.google.com/bigquery/) a New Relic. Este documento explica cómo activar la integración de Google BigQuery y describe los datos que se pueden reportar.

## Característica

Google BigQuery es un almacén de datos sin infraestructura que administrar. Con BigQuery puedes especificar el esquema de datos o permitir que el servicio lo descubra por ti. Puede explorar sus datos utilizando SQL consulta.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [conectar su servicio GCP a New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frecuencia de sondeo [#polling]

New Relic integración consulta los servicios de GCP según un intervalo de sonido que varía según la integración. La frecuencia de votación para la integración de Google BigQuery es de cinco minutos. Para obtener información sobre la resolución, consulte [detalles de datos métricos](#metrics).

## Activar Tabla métrica [#table-metrics]

La colección de tablas métricas de BigQuery está deshabilitada de forma predeterminada. Esto afecta la métrica enumerada en [GcpBigQueryTableSample](#table-sample). Para recopilar esas métricas, active la configuración "Habilitar tabla métrica" en el menú de configuración de la integración de BigQuery.

## Buscar y utilizar datos [#find-data]

Después de activar la integración y esperar unos minutos (según la [frecuencia de](#polling) sondeo ), los datos aparecerán en la New Relic UI. Para [buscar y utilizar sus datos](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), incluidos enlaces a su <InlinePopover type="dashboards"/>y configuración de alertas, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP > (select an integration)**</DNT>.

## Datos métricos [#metrics]

Para ver [datos métricos](/docs/telemetry-data-platform/understand-data/new-relic-data-types/#metrics) para su integración de Google BigQuery en New Relic, [cree una consulta NRQL ](/docs/integrations/new-relic-integrations/getting-started/use-integration-data-new-relic-insights#nrql)para [los eventos](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#event) `GcpBigQueryProjectSample`, `GcpBigQueryDatasetSample` y `GcpBigQueryTableSample` y su atributo.
