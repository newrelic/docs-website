Nuestra [infraestructura de integración](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) con [Google Cloud Platform (GCP)](https://cloud.google.com/) incluye una que reporta datos [Google Cloud Functions](https://cloud.google.com/functions/) a nuestros productos. Este documento explica cómo activar la integración de GCP Cloud Functions y describe los datos que se pueden reportar.

## Característica

El servicio Google Cloud Functions permite ejecutar código sin servidor. Al utilizar la [UI de Google](https://cloud.google.com/functions/docs/quickstarts), los desarrolladores pueden crear fragmentos cortos de código destinados a realizar una función específica. Luego, la función puede responder a eventos en la nube sin la necesidad de administrar un servidor de aplicaciones o un entorno de ejecución.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [conectar su servicio GCP a New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frecuencia de sondeo [#polling]

Nuestra integración consulta sus servicios de GCP según un intervalo de sondeo, que varía según la integración.

* Frecuencia de sondeo para GCP Cloud Functions: cinco minutos
* Resolución: un punto de datos cada minuto

## Ver y usar datos [#find-data]

Después de activar la integración y luego esperar unos minutos (según la [frecuencia de sondeo](#polling)), los datos aparecerán en la UI. Para [ver y utilizar sus datos](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), incluidos enlaces a su <InlinePopover type="dashboards"/>y configuración de alerta, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP**</DNT> y luego seleccione una integración.
