[La integración deNew Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) con [Google Cloud Platform (GCP)](https://cloud.google.com) incluye una integración para informar los datos [Google Cloud Spanner](https://cloud.google.com/spanner/) a New Relic. Este documento explica cómo activar la integración de GCP Cloud Spanner y describe los datos que se pueden reportar.

## Característica

El servicio Google Cloud Spanner es un servicio de base de datos relacional distribuido globalmente creado para la nube. Con la [consola Google Spanner](https://cloud.google.com/spanner/docs/quickstart-console), los desarrolladores pueden crear una instancia de Cloud Spanner, agregar esquemas, escribir y modificar datos y ejecutar consultas.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [conectar su servicio GCP a New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frecuencia de sondeo [#polling]

New Relic integración consulta tus servicios de GCP según un intervalo de sondeo, que varía según la integración. La frecuencia de las encuestas para Google Cloud Spanner es de cinco minutos. La resolución es de 1 punto de datos cada minuto.

## Buscar y utilizar datos [#find-data]

Después de activar la integración y esperar unos minutos (según la [frecuencia de](#polling) sondeo ), los datos aparecerán en la New Relic UI. Para [buscar y utilizar sus datos](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), incluidos enlaces a su <InlinePopover type="dashboards"/>y configuración de alertas, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP > (select an integration)**</DNT>.

## Datos métricos [#metrics]

Para ver [datos métricos](/docs/integrations/new-relic-integrations/getting-started/#metric) para su integración de GCP Spanner en New Relic, [cree una consulta NRQL ](/docs/integrations/new-relic-integrations/getting-started/use-integration-data-new-relic-insights#nrql)para `GcpSpannerInstanceSample` y `GcpSpannerDatabaseSample` [evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) y su atributo relacionado.
