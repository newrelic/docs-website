[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) con [Google Cloud Platform (GCP)](https://cloud.google.com/) incluye una integración para informar los datos [Google Cloud Pub/Sub](https://cloud.google.com/pubsub/) a New Relic. Este documento explica cómo activar la integración de GCP Pub/Sub y describe los datos que se informarán.

## Característica

Como parte de la Google Cloud [solución de análisis](https://cloud.google.com/solutions/stream-analytics) de flujos, el servicio Cloud Pub/Sub ingiere y entrega flujos de eventos para un rápido procesamiento y análisis de datos.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [conectar su servicio GCP a New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frecuencia de sondeo [#polling]

New Relic integración consulta tus servicios de GCP según un intervalo de sondeo, que varía según la integración. La frecuencia de votación para Google Cloud Pub/Sub es de cinco minutos. La resolución es de 1 punto de datos cada minuto.

## Buscar y utilizar datos [#find-data]

Después de activar la integración y esperar unos minutos (según la [frecuencia de](#polling) sondeo ), los datos aparecerán en la New Relic UI. Para [buscar y utilizar sus datos](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), incluidos enlaces a su <InlinePopover type="dashboards"/>y configuración de alertas, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP > (select an integration)**</DNT>.

## Datos métricos [#metrics]

Los temas de Pub/Sub se denominan entidades que representan canales de mensajes, y las suscripción se denominan entidades que representan destinos de mensajes sobre un tema en particular.

Para ver [datos métricos](/docs/telemetry-data-platform/understand-data/new-relic-data-types/#metrics) para su integración GCP Pub/Sub en New Relic, [cree una consulta NRQL ](/docs/integrations/new-relic-integrations/getting-started/use-integration-data-new-relic-insights#nrql)para `GcpPubSubTopicSample` y `GcpPubSubSubscriptionSample` [evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) y su atributo relacionado.
