New Relic ofrece una [integración](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) para informar sus datos de Google Cloud Storage a New Relic. Conozca cómo [conectar esta integración con el monitoreo de infraestructura](#connect) y sobre los [datos métricos](#metrics) que reporta New Relic para esta integración.

## Característica

Google Cloud Storage es un servicio de Google Cloud Platform que puede utilizar para ofrecer contenido de sitios web, almacenar datos para archivo y recuperación ante desastres, y distribuir objetos de datos mediante descarga directa. Con la integración Google Cloud Storage, puedes acceder a estas características:

* Vea gráficos e información sobre los datos que está almacenando y recuperando de Google Cloud Storage.
* [Cree consultas y gráficos personalizados](/docs/using-new-relic/data/understand-data/query-new-relic-data) a partir de [datos](#metrics) capturados automáticamente.
* Configure [alertas](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information#integration) sobre sus datos Google Cloud Storage directamente desde la página de integración.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [conectar su servicio GCP a New Relic](/docs/connect-google-cloud-platform-services-infrastructure).

## Frecuencia de sondeo [#polling]

New Relic consulta sus servicios de Google Cloud Storage en función de un intervalo de sondeo de 5 minutos.

## Buscar y utilizar datos [#find-data]

Después de conectar la integración a New Relic y esperar unos minutos, los datos aparecerán en la New Relic UI. Para [buscar y utilizar datos de integración](/docs/telemetry-data-platform/understand-data/new-relic-data-types/), incluidos sus tableros<InlinePopover type="dashboards"/>y su configuración de alerta, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP > Google Cloud Storage**</DNT>.

Para crear un panel personalizado para la integración, [cree una consulta](/docs/integrations/new-relic-integrations/getting-started/use-integration-data-new-relic-insights) para el `GcpStorageBucketSample` tipo de evento con el valor del proveedor `GcpStorageBucket`.

## Datos métricos [#metrics]

La integración reporta [datos métricos](/docs/telemetry-data-platform/understand-data/new-relic-data-types/#metrics) para todos los valores de `method` y `response_code`:

* `response_code`: El código de respuesta de las solicitudes.
* `method`: El nombre del método API llamado.
