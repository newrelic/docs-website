La integración de [New Relic Infrastructure](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure) monitor el rendimiento de productos y servicios populares. New Relic Google Cloud PlatformLa integración de (GCP) de le permite monitor sus datos de GCP en varias New Relic características .

## Conecte GCP y New Relic [#connect]

Para obtener datos de GCP, siga los procedimientos estándar para [conectar su servicio GCP a New Relic](/docs/infrastructure/gcp-integrations).

## Ver tus datos de GCP [#insights]

Una vez que siga el proceso de configuración, los datos de su cuenta de Google Cloud Platform se informarán directamente a New Relic.

Para ver tus datos de GCP: ve a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP**</DNT>. Para cualquiera de las integraciones enumeradas:

* Seleccione un nombre de integración para ver los datos en un dashboard preconfigurado.

  O

* Seleccione el <DNT>**Explore data**</DNT> <Icon name="fe-database"/> icono para ver los datos de GCP.

Puede ver y reutilizar la consulta NRQL tanto en el panel preconfigurado <InlinePopover type="dashboards"/>como en el <DNT>**Events explorer**</DNT>. Esto le permite adaptar la consulta a sus necesidades específicas.

El evento, el panel y otros datos están disponibles para todos los servicios en New Relic.
