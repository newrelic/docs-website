## Problema

Quiere [integrar sus servicios de Google Cloud Platform (GCP)](/docs/introduction-google-cloud-platform-integrations) con New Relic, pero ve errores de autenticación de API.

## Solución

Para resolver el problema:

1. Vaya a

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP**
   </DNT>

   .

2. En la página de tu cuenta de GCP, selecciona

   <DNT>
     **Reauthorize New Relic**
   </DNT>

   .

3. Siga los procedimientos estándar para [conectar sus servicios de GCP a New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/#connect).

## Causa

Las razones comunes para este error incluyen:

* La autorización para que New Relic utilice las API de GCP ha sido revocada manualmente por el usuario desde la aplicación de gestión de cuentas de Google.
* La autorización para que New Relic utilice API de GCP no incluye uno o más de los permisos ([alcances de Google OAuth](https://developers.google.com/identity/protocols/googlescopes)) necesarios para consultar una o más API de GCP.
* La autorización para que New Relic utilice las API de GCP ha caducado.
