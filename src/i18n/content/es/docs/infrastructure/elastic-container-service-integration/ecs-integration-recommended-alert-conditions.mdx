[La integración ECS](/docs/introduction-amazon-ecs-integration) de New Relic informa y muestra datos de rendimiento de su entorno [Amazon ECS](https://docs.aws.amazon.com/ecs/index.html) . Este documento proporciona algunas condiciones de alerta recomendadas para el monitoreo del rendimiento de ECS.

## Condición de alerta recomendada [#alerts]

A continuación se presentan algunas condiciones de alerta ECS recomendadas. Para agregar estas alertas, vaya a la [UI de alertas](http://alerts.newrelic.com/) y agregue la siguiente [condición de alertaNRQL ](/docs/alerts/new-relic-alerts/defining-conditions/create-alert-conditions-nrql-queries)a una política de alertas nueva o existente:

* Alto uso de CPU

  * NRQL: DESDE ContainerSample SELECCIONE cpuUsedCoresPercent
  * Crítico: > 90% durante 5 minutos

* Alto uso de memoria

  * NRQL: FROM ContainerSample SELECT memoriaUsoBytes / memoriaTamañoLimitBytes
  * Crítico: > 80% durante 5 minutos

* Reiniciar recuento

  * NRQL: FROM ContainerSample SELECT max(restartCount) - min(restartCount)
  * Crítico: > 5 durante 5 minutos
