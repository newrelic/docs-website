El agente de infraestructura se inicia automáticamente después de ejecutar el script de instalación. Sin embargo, hay situaciones en las que es posible que necesite reiniciar manualmente el agente (por ejemplo, después [de cambiar la configuración del agente](/docs/infrastructure-configure-your-agent)).

¿Quieres probar nuestro agente de infraestructura? ¡ [Crea una cuenta New Relic](https://newrelic.com/signup) gratis! No se requiere tarjeta de crédito.

## Linux [#linux]

Para [Linux](/docs/infrastructure/new-relic-infrastructure/installation/install-infrastructure-linux), asegúrese de utilizar el comando correcto para [su sistema de inicio](#init-system). Seleccione `start`, `stop`, `restart` o `status` según corresponda:

* <DNT>**SystemD**</DNT> (Amazon Linux 2, SLES 12, CentOS 7 o superior, Debian 8 o superior, RHEL 7 o superior, Ubuntu 15.04 o superior):

  ```shell
  sudo systemctl <start|stop|restart|status> newrelic-infra
  ```

* <DNT>**System V**</DNT> (Debian 7, SLES 11.4, RHEL 5):

  ```shell
  sudo /etc/init.d/newrelic-infra <start|stop|restart|status>
  ```

* <DNT>**Upstart**</DNT> (Amazon Linux, RHEL 6, Ubuntu 14.04 o inferior):

  ```shell
  sudo initctl <start|stop|restart|status> newrelic-infra
  ```

## Windows [#windows]

<Callout variant="important">
  Para iniciar, detener o reiniciar el agente, debe ejecutar `cmd.exe` o PowerShell como <DNT>**Administrator**</DNT>.
</Callout>

Para [Windows Server](/docs/infrastructure/new-relic-infrastructure/installation/install-infrastructure-windows-server), puede utilizar el símbolo del sistema de Windows o PowerShell.

### Símbolo del sistema (cmd.exe)

* Inicie o detenga el agente de Windows:

  ```shell
  net <start|stop> newrelic-infra
  ```

* Reinicie el agente de Windows:

  ```shell
  net stop newrelic-infra ; net start newrelic-infra
  ```

* Verifique el estado del agente de Windows:

  ```shell
  sc query "newrelic-infra" | find "STATE"
  ```

### Potencia Shell

* Inicie o detenga el agente de Windows:

  ```shell
  Stop-Service -Name "newrelic-infra"
  Start-Service -Name "newrelic-infra"
  ```

  <Callout variant="tip">
    También puedes usar `net start|stop newrelic-infra`
  </Callout>


* Reinicie el agente de Windows:

  ```shell
  Restart-Service newrelic-infra
  ```

* Verifique el estado del agente de Windows:

  ```shell
  (Get-Service newrelic-infra).Status
  ```

## Mac OS [#macOS]

* Detener o iniciar el agente:

  ```shell
  brew services stop newrelic-infra-agent
  brew services start newrelic-infra-agent
  ```

* Reinicie el agente:

  ```shell
  brew services restart newrelic-infra-agent
  ```

* Consultar estado del agente:

  ```shell
  brew services list
  ```

## Personalizar el registro del agente [#logging]

El agente de infraestructura inicia sesión en una ubicación predeterminada que depende de su plataforma. Puede personalizar esta ubicación con la configuración [`log_file`](/docs/infrastructure/install-configure-infrastructure/configuration/infrastructure-configuration-settings#log-file) . También puede [generar un registro detallado para la resolución de problemas](/docs/infrastructure/new-relic-infrastructure/troubleshooting/generating-logs-troubleshooting-infrastructure).
