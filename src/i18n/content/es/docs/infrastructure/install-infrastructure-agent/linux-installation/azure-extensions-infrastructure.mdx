Puede instalar nuestro agente [de infraestructura](/docs/welcome-new-relic-infrastructure) utilizando [las extensiones de Microsoft Azure](https://docs.microsoft.com/en-us/azure/virtual-machines/extensions/overview).

<CollapserGroup>
  <Collapser
    id="what-is-extension"
    title="¿Qué es una extensión de Azure?"
  >
    Las extensiones de máquina virtual de Azure (máquina virtual (VM)) son pequeñas aplicaciones que proporcionan configuración posterior al despliegue y tareas de automatización en Azure máquina virtual (VM). Por ejemplo, si una máquina virtual requiere la instalación de software, protección antivirus o la ejecución de un script, se puede utilizar una extensión de máquina virtual (VM). La CLI de Azure, PowerShell, las plantillas de Azure Resource Manager y Azure Portal pueden ejecutar extensiones de máquina virtual (VM) de Azure. Las extensiones pueden incluirse con una nueva máquina virtual (VM) desplegable o ejecutarse en cualquier sistema existente.
  </Collapser>
</CollapserGroup>

New Relic utiliza estas extensiones para facilitar la instalación del agente de infraestructura. Esto elimina la necesidad de actualizar manualmente nuestro agente y, lo que es más importante, escala automáticamente el agente con sus [conjuntos de escalado de Azure](https://docs.microsoft.com/en-us/azure/virtual-machine-scale-sets/overview).

## Instalar infraestructura usando extensiones de Azure [#install-infra-azure-extensions]

Instalar el agente usando la extensión de Azure en [un sistema operativo](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/) compatible.

1. Vaya a cualquier máquina virtual (VM) de Azure, conjunto de escalado de Azure o Azure Service Fabric que utilice conjuntos de escalado y seleccione la pestaña <DNT>**Extensions**</DNT> .

   <img
     title="azure_ext01.png"
     alt="azure_ext01.png"
     src="/images/infrastructure_screenshot-crop_azure-extensions-tab.webp"
   />

2. Seleccione la extensión <DNT>**New Relic infrastructure**</DNT> .

   <img
     title="azure_ext02.png"
     alt="azure_ext02.png"
     src="/images/infrastructure_screenshot-crop_azure-nr-infrastructure-extension.webp"
   />

3. Siga las instrucciones y agregue su <InlinePopover type="licenseKey"/>.

   <img
     title="azure_ext03.png"
     alt="azure_ext03.png"
     src="/images/infrastructure_screenshot-crop_azure-nr-license-key.webp"
   />

   <Callout variant="tip">
     Si automatizas la instalación, recuerda pasar tu <InlinePopover type="licenseKey"/>con la variable `NR_LICENSE_KEY` . Puede utilizar [la configuración a través de variables de entorno](/docs/infrastructure/install-infrastructure-agent/configuration/configure-infrastructure-agent/#precedence) (formato `NRIA_` ) para personalizar cualquier configuración de agente.
   </Callout>
