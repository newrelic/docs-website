El campo <DoNotTranslate>**Alert type**</DoNotTranslate> en la página <DoNotTranslate>**Settings > Alerts**</DoNotTranslate> de monitoreo de infraestructura muestra qué opciones puede seleccionar para [crear una condición de alerta de infraestructura](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information). También puedes crear condición de alerta desde cualquier gráfico de infraestructura seleccionando las elipses <Icon name="fe-more-horizontal"/> icono y luego <DoNotTranslate>**Create alert**</DoNotTranslate>.

## Ejemplos: páginas de infraestructura [#examples-infra-pages]

A continuación se muestran algunos ejemplos de cómo crear una condición de alerta dentro del contexto de la página UI de monitoreo de infraestructura que está viendo actualmente. Para crear una condición de alerta desde cualquier gráfico, seleccione las elipses <Icon name="fe-more-horizontal"/> icono y luego <DoNotTranslate>**Create alert**</DoNotTranslate>. New Relic seleccionará automáticamente el <DoNotTranslate>**Alert type**</DoNotTranslate> apropiado.

## Ejemplos: opciones de umbral [#examples-infra-pages]

Utilice el [umbral](/docs/alerts/new-relic-alerts/getting-started/alerts-glossary#alert-threshold) desplegable para el <DoNotTranslate>**Alert type**</DoNotTranslate> seleccionado para definir aún más cómo desea que se le alerten. A continuación se muestran algunos ejemplos de las opciones disponibles.

<CollapserGroup>
  <Collapser
    id="integrations"
    title="Proveedores de integración"
  >
    Con infraestructura integración, puedes [crear una condición de alerta desde tu página <DoNotTranslate>**Integrations**</DoNotTranslate> ](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information#integration). Dependiendo del tipo de proveedor seleccionado (CloudFront, DynamoDB, EBS, etc.), las opciones variarán del menú desplegable <DoNotTranslate>**Define thresholds**</DoNotTranslate> ; por ejemplo, bytes, errores, solicitudes, CPU, conexiones, memoria, registros, latencia, etc.
  </Collapser>

  <Collapser
    id="system-thresholds"
    title="CPU, disco, carga promedio, memoria, swap"
  >
    El umbral desplegable <DoNotTranslate>**System metrics**</DoNotTranslate> te permite seleccionar varios criterios de CPU, disco, carga promedio, memoria y swap métrica.
  </Collapser>

  <Collapser
    id="byte"
    title="Tamaño de byte"
  >
    El umbral <DoNotTranslate>**Network metrics**</DoNotTranslate> proporciona flexibilidad para las necesidades de su negocio. Dependiendo del tamaño de su red, puede establecer fácilmente el umbral en bytes, KB, MB, GB o TB.
  </Collapser>
</CollapserGroup>
