## Problema [#problem]

Instalaste el agente New Relic Infrastructure en un servidor Windows y no se muestran datos en la UI. Al intentar iniciar el servicio manualmente desde <DNT>**Windows Services**</DNT> , falla con un error como:

```
Windows could not start the New Relic infrastructure agent service on Local Computer
Error 1067: The process terminated unexpectedly
```

## Solución [#solution]

Es posible que un producto antivirus o de seguridad esté impidiendo que el agente New Relic Infrastructure se ejecute como un servicio. Para validar que el agente se comporte sin problemas, ejecute el archivo `newrelic-infra.exe` desde la línea de comando como administrador y confirme que el host esté informando los datos como se esperaba. Si ese es el caso, trabaje con su equipo de seguridad para ver si es necesario agregar el servicio a su lista de 'permitidos' o configurarlo con un parámetro adicional.

</Callout>
