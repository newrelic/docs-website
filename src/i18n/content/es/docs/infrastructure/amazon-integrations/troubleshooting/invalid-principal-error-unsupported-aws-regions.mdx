## Problema

Al conectar [AWS a una integración de infraestructura](/docs/integrations/amazon-integrations/get-started/connect-aws-services-infrastructure#connect), durante la fase <DNT>**Review**</DNT> , recibe el error de AWS `Invalid principal in policy`.

## Solución

Marque el tipo de integración que desea instalar. New Relic puede obtener datos de monitoreo de servicios y extremos que se encuentran en todas [las regionesAWS ](https://docs.aws.amazon.com/general/latest/gr/rande.html)excepto en las regiones de China.

Verás este error al intentar configurar la integración desde una cuenta que usa China.
