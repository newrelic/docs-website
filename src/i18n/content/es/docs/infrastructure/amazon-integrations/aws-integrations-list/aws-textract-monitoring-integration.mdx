[La integración New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)ahora incluye una integración para AWS Textract, enviando sus datos métricos a New Relic.

Este documento explica la característica de la integración, cómo activarla y qué datos se pueden reportar.

## Característica [#features]

Recopile y envíe telemetry data a New Relic desde su [aplicación Textract](https://aws.amazon.com/textract/) utilizando nuestra integración. monitor sus servicios, consulte los datos entrantes y cree un panel para observar todo de un vistazo.

## Informar datos [#report-data]

Tenemos dos opciones para reportar estos datos. La ruta recomendada es configurar nuestra [integración de Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream). Alternativamente, puede utilizar nuestra [integración anterior de Amazon que se basa en encuestas](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations).

## Buscar y utilizar datos [#find-use-data]

Los datos de esta integración se adjuntan al tipo de datos `Metric` . Para encontrar la métrica de tu integración, ve a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> y filtra por `aws.textract`.
