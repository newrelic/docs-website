Ahora puede integrar el agente de infraestructura con AWS [base de datos Migration Service](https://aws.amazon.com/dms/), permitiéndole enviar sus datos métricos a New Relic. monitor sus servicios, consulte los datos entrantes y cree un panel para observar todo de un vistazo.

Para habilitar esta integración, siga nuestras instrucciones para [conectar los servicios de AWS a New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/). Una vez habilitado, puedes encontrar la métrica de tu integración en <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> y filtrar por `aws.dms`.

## Datos métricos [#metrics]

Esta integración New Relic Infrastructure recopila los siguientes datos del Servicio de Migración de la base de datos de Amazon:

Todos los datos importados tienen una dimensión común: `ReplicationInstanceIdentifier`
