[La integración New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)ahora incluye una integración para AWS Chatbot, enviando sus datos métricos a New Relic.

Este documento explica la característica de la integración, cómo activarla y qué datos se pueden reportar.

## Característica

Recopile y envíe telemetry data a New Relic desde su [Chatbot](https://aws.amazon.com/chatbot/) utilizando nuestra integración. monitor sus servicios, consulte los datos entrantes y cree un panel para observar todo de un vistazo.

## Activar la integración

Para habilitar esta integración, consulte cómo [conectar los servicios de AWS a New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream).

<Callout variant="important">
  Para obtener AWS Chatbot métrica, cambie su región de Amazon CloudWatch a `US East (N. Virginia)`. Lea más al respecto en [la documentación de AWS](https://docs.aws.amazon.com/chatbot/latest/adminguide/monitoring-cloudwatch.html).
</Callout>

## Buscar y utilizar datos

Para encontrar la métrica de tu integración, ve a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> y filtra por `aws.chatbot`.
