Las integraciones New Relic Infrastructure ahora incluyen una integración que envía su métrica de AWS Storage Gateway a New Relic. La integración métrica de AWS Storage Gateway recopila y envía telemetry data a New Relic desde su AWS Storage Gateway. Puede monitor sus servicios de AWS, consultar los datos entrantes y crear un panel para observar todo de un vistazo.

## Activar la integración [#activate-ingration]

Para habilitar esta integración, siga las instrucciones en nuestra [documentación de integración de Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Buscar y utilizar datos [#find-data]

Para encontrar la métrica de tu integración:

1. Vaya a **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
2. Seleccione **Metrics and events**
3. Filtrar por `aws.storagegateway`.
