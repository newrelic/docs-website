[La integración New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)incluye una integración para informar sus datos de Amazon Route 53 a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica [#features]

[Amazon Route 53](https://aws.amazon.com/route53/) le ayuda a registrar nombres de dominio, dirigir el tráfico de Internet a los recursos adecuados y comprobar el estado de sus recursos para asegurarse de que su aplicación web esté siempre en funcionamiento.

Para las verificações de estado de AWS, Amazon Route 53 determina si un conjunto de registros de recursos está en buen estado según uno de los siguientes criterios:

* Enviando periódicamente una solicitud al extremo que se especifica en el control de salud
* Agregando el estado de un grupo específico de controles de salud (controles de salud calculados)
* Determinando el estado actual de una alarma de CloudWatch (verificaciones de estado de métricas de CloudWatch)

Los datos reportados a New Relic incluyen tiempo de conexión, verificações de estado, tiempo hasta el primer byte y otras [métricas](#metrics). Los datos de Route 53 están disponibles en <InlinePopover type="dashboards"/>prediseñado y también puedes crear [consultas y gráficos personalizados](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).

## Activar la integración [#activate]

Para habilitar esta integración, siga los procedimientos estándar para [conectar los servicios de AWS a New Relic](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure).

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Información [de sondeo](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) predeterminada para la integración de Amazon Route 53:

* New Relic intervalo de sondeo: 5 minutos
* Intervalo de datos de Amazon CloudWatch: 1 minuto

## Buscar y utilizar datos [#find-data]

Para encontrar sus datos de integración, vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > AWS**</DNT> y seleccione uno de los enlaces de integración de Route 53.

Puedes [consultar y explorar tus datos](/docs/using-new-relic/data/understand-data/query-new-relic-data) utilizando el [tipo de evento](/docs/data-apis/understand-data/new-relic-data-types/#event-data) `HealthCheckSample` , con un valor `provider` de `Route53HealthCheck`.

Para obtener más información sobre cómo utilizar sus datos, consulte [Comprender los datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data).
