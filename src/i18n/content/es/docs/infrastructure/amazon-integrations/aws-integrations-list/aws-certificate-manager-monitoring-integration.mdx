New Relic Infrastructure ahora incluye una integración para AWS Certificate Manager que permite enviar sus datos métricos a New Relic. Este documento explica la característica de la integración, cómo activarla y qué datos están disponibles para generar informes.

## Característica

Utilizando nuestra integración, recopilamos y enviamos telemetry data a New Relic desde su [AWS Certificate Manager](https://aws.amazon.com/certificate-manager/). Puede monitor sus servicios, consultar datos entrantes y crear un panel para observar todo de un vistazo.

## Activar la integración

Para habilitar esta integración, siga las instrucciones en nuestra [documentación de integración de Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Buscar y utilizar datos

Para encontrar la métrica de tu integración:

1. Vaya a [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)

2. Seleccionar

   <DNT>
     **Metrics and events**
   </DNT>

3. Filtrado por `aws.certificatemanager`
