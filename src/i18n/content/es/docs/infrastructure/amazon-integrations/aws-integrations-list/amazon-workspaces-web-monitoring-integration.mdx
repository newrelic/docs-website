La integración New Relic Infrastructure ahora incluye una integración para AWS WorkSpaces Web y puede enviar sus datos métricos a New Relic.

Este documento explica la característica de la integración, cómo activarla y qué datos están disponibles para generar informes.

## Característica

Utilizando nuestra integración, recopilaremos y enviaremos telemetry data a New Relic desde su [WorkSpaces Web](https://aws.amazon.com/workspaces/web). monitor sus servicios, consulte los datos entrantes y cree un panel para observar todo de un vistazo.

## Activar la integración

Para habilitar esta integración, siga las instrucciones en nuestra [documentación de integración de Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Buscar y utilizar datos

Para encontrar la métrica de tu integración:

1. Vaya a [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)

2. Seleccionar

   <DNT>
     **Metrics and events**
   </DNT>

3. Filtrar por `aws.workspacesweb`.
