[La integración New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)ahora incluye una integración para AWS API Gateway v2, enviando sus datos métricos a New Relic.

Este documento explica la característica de la integración, cómo activarla y qué datos puedes reportar.

## Característica [#features]

Recopile y envíe telemetry data a New Relic desde su [Amazon API Gateway v2](https://aws.amazon.com/api-gateway/) utilizando nuestra integración. Verifique sus servicios, consulte los datos entrantes y cree un panel para observar todo de un vistazo.

## Activar la integración [#activate-integration]

Para habilitar esta integración, consulte cómo [conectar los servicios de AWS a New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Buscar y utilizar datos [#find-use-data]

Para encontrar la métrica de tu integración, ve a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Metrics & Events**</DNT> y filtra por `aws.apigateway`.

## Datos métricos [#metric-data]

Esta integración New Relic Infrastructure recopila las siguientes métricas de Amazon API Gateway v2:

### Crear alerta [#create-alerts]

Puede configurar alertas para recibir una notificación si hay algún cambio. Por ejemplo, puede configurar una alerta para recibir una notificación sobre las partes relevantes de errores críticos o fatales.

Obtenga más información sobre cómo crear alertas [aquí](/docs/tutorial-create-alerts/create-an-alert/).
