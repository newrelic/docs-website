[La integración New Relic Infrastructure ](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations)incluye una integración para AWS DataSync, enviando sus datos métricos a New Relic.

Este documento explica la característica de la integración, cómo activarla y qué datos se pueden reportar.

## Característica

Utilice nuestra integración para recopilar y enviar telemetry data a New Relic desde su [DataSync](https://aws.amazon.com/datasync/). Puede monitor sus servicios, consultar datos entrantes y crear un panel para observar todo de un vistazo.

## Activar la integración

Para habilitar esta integración, consulte cómo [conectar los servicios de AWS a New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Buscar y utilizar datos

Para encontrar la métrica de tu integración, ve a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> y filtra por `aws.DataSync`.
