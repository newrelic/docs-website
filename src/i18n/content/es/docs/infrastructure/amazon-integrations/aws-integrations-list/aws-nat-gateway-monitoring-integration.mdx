Nuestra integración de NAT Gateway informa datos de su servicio NAT Gateway a su cuenta de New Relic mediante AWS CloudWatch Metric Streams.

## Característica [#features]

Con esta integración, puedes ver de tu [NAT Gateway](https://docs.aws.amazon.com/vpc/latest/userguide/vpc-nat-gateway.html) telemetry data en New Relic. Puede monitor sus servicios, consultar los datos entrantes y crear un panel para observar todo de un vistazo.

## Activar AWS CloudWatch Metric Streams [#activate]

Para habilitar AWS CloudWatch Metric Streams, siga [las instrucciones para conectar AWS CloudWatch Metric Streams a New Relic](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/).

<Callout variant="tip">
  Si ya tiene habilitado AWS CloudWatch Metric Streams , ¡ya debería ver la métrica en New Relic! No es necesario realizar ningún cambio.
</Callout>

## Buscar y utilizar datos [#find-use-data]

Los datos de esta integración se adjuntan a [nuestro tipo de datos `Metric` ](/docs/data-apis/understand-data/new-relic-data-types/#dimensional-metrics).

Una forma de encontrar tus datos: ve a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> y filtra por `aws.natgateway`.
