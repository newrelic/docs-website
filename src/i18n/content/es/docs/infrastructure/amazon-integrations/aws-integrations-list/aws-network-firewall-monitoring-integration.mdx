Las integraciones New Relic Infrastructure ahora incluyen una integración que envía la métrica firewall de su red AWS a New Relic. Este documento explica la característica de la integración, cómo activarla y qué datos están disponibles para generar informes.

## Característica [#features]

La integración métrica firewall de la red AWS recopila y envía telemetry data a New Relic desde su firewall de la red AWS. Puede monitor sus servicios de AWS, consultar los datos entrantes y crear un panel para observar todo de un vistazo.

## Activar la integración [#activate-ingration]

Para habilitar esta integración, siga las instrucciones en nuestra [documentación de integración de Amazon CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/).

## Buscar y utilizar datos [#find-data]

Para encontrar la métrica de tu integración:

1. Ir a

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

2. Seleccionar

   <DNT>
     **Metrics and events**
   </DNT>

3. Filtrar por `aws.networkfirewall`.
