[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) incluye una integración para informar [el espacio de nombres de su socio Microsoft Azure Event Grid ](https://learn.microsoft.com/en-us/azure/event-grid/partner-events-overview-for-partners)métrica y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos métricos del monitoreo de Azure para el espacio de nombres Azure evento Grid Partner. Un namespace de socio es un recurso regional que tiene un extremo para publicar eventos en Azure eventos Grid. El espacio de nombres de socios contiene canales o canales de eventos (recurso legacy ).

Usando New Relic, puedes:

* Vea los datos del espacio de nombres de Azure evento Grid Partner en un panel prediseñado.
* Ejecute [una consulta personalizada y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de Azure en el monitoreo de infraestructura New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta los servicios de espacio de nombres de su Azure evento Grid Partner a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
