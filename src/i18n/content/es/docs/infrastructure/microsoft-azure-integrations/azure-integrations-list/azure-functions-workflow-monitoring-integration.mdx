[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) incluye una integración para reportar su [flujo de trabajo métrico de Microsoft Azure Functions ](https://azure.microsoft.com/en-us/products/logic-apps/)y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

El flujo de trabajo de Azure Functions se refiere a Azure Logic Apps en el tipo de plan estándar. Azure Logic Apps es una plataforma en la nube donde puede crear y ejecutar flujo de trabajo automatizado con poco o ningún código. Al utilizar el diseñador visual y seleccionar entre operaciones prediseñadas, puede crear rápidamente un flujo de trabajo que integre y administre sus aplicaciones, datos, servicios y sistemas.

Usando New Relic, puedes:

* Vea los datos del flujo de trabajo Azure Functions en un panel prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de flujo de trabajo Azure Functions en el monitoreo de infraestructura de New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta su servicio de flujo de trabajo de Functions a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
