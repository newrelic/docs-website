[New Relic incluye una integración](/docs/infrastructure/introduction-infra-monitoring) para informar sus datos de Azure Key Vault a nuestra plataforma. Aquí explicamos cómo activar la integración y qué datos recopila.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [activar su servicio Azure](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations).

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Información [de sondeo](/docs/infrastructure/microsoft-azure-integrations/getting-started/azure-integration-polling) predeterminada para la integración de Azure Key Vault:

* New Relic intervalo de sondeo: 5 minutos

## Buscar y utilizar datos [#find-data]

Para encontrar sus datos de integración en New Relic, vaya a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; Azure**</DNT> y seleccione una integración.
