[La integración de New Relic](/docs/infrastructure/introduction-infra-monitoring) incluye una integración para reportar sus métricas [deMicrosoft Azure Stream Analytics](https://azure.microsoft.com/en-us/products/stream-Analytics) y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos métricos del monitoreo de Azure para Azure Stream Analytics. Azure Stream Analytics es un motor de procesamiento de transmisiones completamente administrado que está diseñado para analizar y procesar grandes volúmenes de transmisión de datos con una latencia inferior a milisegundos. Se pueden identificar patrones y relaciones en datos que se originan en una variedad de fuentes de entrada, incluidas aplicaciones, dispositivos, sensores, secuencias de clics y feeds de redes sociales.

Usando New Relic, puedes:

* Vea los datos de Azure Stream Analytics en un dashboard prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de Azure en el monitoreo de infraestructura New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta sus servicios de Azure Stream Analytics a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
