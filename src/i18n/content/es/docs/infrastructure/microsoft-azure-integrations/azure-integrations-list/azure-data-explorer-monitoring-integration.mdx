[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) incluye una integración para reportar su métrica [Microsoft Azure Data Explorer](https://azure.microsoft.com/en-us/products/data-explorer/) y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos métricos del monitoreo de Azure para el servicio Azure Data Explorer. La caja de herramientas de Azure Data Explorer le ofrece una solución integral para la ingesta, consulta, visualización y administración de datos. Puede utilizar Azure Data Explorer para recopilar, almacenar y analizar diversos datos para mejorar productos, mejorar las experiencias de los clientes, monitor dispositivos e impulsar operaciones.

Usando New Relic, puedes:

* Vea los datos de Azure Data Explorer en un panel prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de Azure en el monitoreo de infraestructura New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta sus servicios de Azure Data Explorer a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
