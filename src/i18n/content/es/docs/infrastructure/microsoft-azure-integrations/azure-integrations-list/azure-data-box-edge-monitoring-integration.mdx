[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) incluye una integración para reportar su [Microsoft Azure Data Box Edge](https://azure.microsoft.com/en-us/updates/announcing-azure-data-box-edge/) métrica y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos métricos del monitoreo de Azure para el servicio Azure Data Box Edge. Data Box Edge actúa como una puerta de enlace de almacenamiento, creando un vínculo entre su sitio y el almacenamiento de Azure. Esto hace que mover datos dentro y fuera del almacenamiento de Azure sea tan fácil como trabajar con un recurso compartido de red local. Data Box Edge proporciona una plataforma informática a través de IoT Edge, que le permite desplegar servicios de Azure y código y aplicaciones personalizados hasta el borde.

Usando New Relic, puedes:

* Vea los datos de Azure Data Box Edge en un panel prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de Azure en el monitoreo de infraestructura New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta sus servicios Azure Data Box Edge a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
