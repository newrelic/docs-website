[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) incluye una integración para reportar su [base de datos Microsoft Azure para MariaDB](https://docs.microsoft.com/en-gb/azure/mariadb/) métrica y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos de la base de datos de Azure para el servicio MariaDB, que proporciona una base de datos como servicio completamente administrada y lista para la empresa. MariaDB proporciona rendimiento predecible y escalabilidad para la aplicación utilizando herramientas y plataforma de código abierto.

Usando New Relic, puedes:

* Vea la base de datos de Azure para los datos de MariaDB en un panel prediseñado.
* Ejecute consultas personalizadas y visualice los datos en la [New Relic UI](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga los procedimientos estándar para [activar su servicio Azure en New Relic](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations), utilizando el rol genérico <DNT>**Reader**</DNT> .

## Configuración y sondeo [#polling]

New Relic consulta su servicio de base de datos de Azure según un intervalo [de sondeo](/docs/integrations/microsoft-azure-integrations/getting-started/azure-integration-polling) predeterminado, que varía según la integración. Para la base de datos de Azure para la integración de PostgreSQL:

* Intervalo de sondeo: 5 minutos (frecuencia máxima de sondeo recomendada: 1 hora)
* Resolución: 1 punto de datos por minuto

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

## Ver y usar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data): vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure > (select an integration)**</DNT>.

En New Relic, los datos sobre una única base de datos se adjuntan al tipo de evento `AzureMariaDbServerSample` , con un valor de proveedor de `AzureMariaDbServer`.
