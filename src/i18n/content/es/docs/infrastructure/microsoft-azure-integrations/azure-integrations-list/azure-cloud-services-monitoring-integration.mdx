[La integración de New Relic](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) incluye una integración para reportar sus [servicios Microsoft Azure en la nube](https://learn.microsoft.com/en-us/azure/cloud-services) métrica y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos métricos del monitoreo de Azure para los servicios de Azure en la nube. Azure servicios en la nube es un ejemplo de plataforma como servicio (PaaS). Al igual que Azure App Service, esta tecnología está diseñada para admitir aplicaciones escalables, confiables y económicas de operar. Usando New Relic, puedes:

* Vea los datos de Azure servicios en la nube en un panel prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de Azure en el monitoreo de infraestructura New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta tus servicios de Azure en la nube a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (select an integration)**</DNT>.
