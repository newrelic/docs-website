[La integración de New Relic](/docs/infrastructure/introduction-infra-monitoring) incluye una integración para reportar sus métricas [Microsoft Azure Sitio web Apps](https://azure.microsoft.com/en-us/products/app-service/) y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

Azure Web Apps proporciona una plataforma para crear una aplicación en Azure sin tener que desplegar, configurar y mantener sus propias máquinas virtuales (VM) de Azure. Puede crear aplicaciones web utilizando ASP.NET, PHP, Node.js y Python. También integran entornos de desarrollo comunes como Visual Studio y GitHub.

Usando New Relic, puedes:

* Vea los datos de Azure Web Apps en un dashboard prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio Azure Web Apps en el monitoreo de infraestructura de New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta su servicio Web Apps a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
