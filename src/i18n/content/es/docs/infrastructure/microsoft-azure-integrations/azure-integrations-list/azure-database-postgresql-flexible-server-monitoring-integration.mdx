Ofrecemos una integración en la nube para informar los datos de su servidor flexible Azure PostgreSQL a nuestra plataforma. Aquí te explicamos cómo activar la integración y qué datos recopila.

## Activar la integración [#activate]

Para habilitar la integración, siga los procedimientos estándar para [activar su servicio Azure.](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations)

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Información [de sondeo](/docs/infrastructure/microsoft-azure-integrations/getting-started/azure-integration-polling) predeterminada para la integración del servidor flexible de Azure PostgreSQL:

* New Relic intervalo de sondeo: 5 minutos

## Buscar y utilizar datos [#find-data]

Para encontrar sus datos de integración, vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/ "Link opens in a new window.") > Infrastructure > Azure**</DNT> y seleccione la integración.

Los datos se adjuntan a los siguientes [tipos de eventos](/docs/data-apis/understand-data/new-relic-data-types/#event-data):
