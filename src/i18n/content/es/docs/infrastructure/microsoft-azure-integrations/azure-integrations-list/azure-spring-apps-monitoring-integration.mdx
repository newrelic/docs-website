[La integración de New Relic](/docs/infrastructure/introduction-infra-monitoring) incluye una integración para reportar sus métricas [Microsoft Azure Spring Apps](https://azure.microsoft.com/en-us/services/spring-apps/) y otros datos a New Relic. Este documento explica cómo activar la integración y describe los datos reportados.

## Característica

New Relic recopila datos métricos del monitoreo de Azure para el servicio Azure Spring Apps. Las aplicaciones Azure Spring ejecutan computación paralela y de alto rendimiento (HPC) a gran escala de manera eficiente en Azure. Azure Spring Apps crea y administra un grupo de nodos de cálculo (máquina virtual), instala la aplicación que desea ejecutar y programa trabajos para ejecutar en los nodos.

Usando New Relic, puedes:

* Vea los datos de las aplicaciones de Azure Spring en un dashboard prediseñado.
* Ejecute [consultas personalizadas y visualice los datos](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights).
* Cree condición de alerta para notificarle sobre cambios en los datos.

## Activar la integración [#activate]

Siga [el procedimiento estándar de monitoreo de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) para activar su servicio de Azure en el monitoreo de infraestructura New Relic .

## Configuración y sondeo [#polling]

Puede cambiar la frecuencia de sondeo y filtrar datos usando [las opciones de configuración](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings).

New Relic consulta sus servicios de Azure Spring Apps a través del monitoreo de integración de Azure según un intervalo [de sondeo](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) predeterminado.

## Buscar y utilizar datos [#find-and-use]

Para [explorar sus datos de integración](/docs/infrastructure/integrations/find-use-infrastructure-integration-data), vaya a <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT>.
