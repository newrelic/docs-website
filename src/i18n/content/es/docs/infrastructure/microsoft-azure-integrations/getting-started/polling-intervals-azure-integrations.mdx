New RelicLa [integración de](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations) Azure de consulta sus servicios de Azure de acuerdo con un intervalo de sondeo específico para la integración.

El intervalo de sondeo se aplica a <DNT>**every**</DNT> entidad de Azure relacionada con el servicio integrado. Por ejemplo, si tiene trece instancias de CosmosDB, cada una de las trece instancias será sondeada cada cinco minutos.

## Ver datos de encuestas [#account-status]

Después de [activar una integración de Azure](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations), New Relic comienza a sondear datos de Azure y los hace accesibles a través de nuestra UI. Puedes consultar los datos de Azure junto con datos adicionales importados de cualquier otra característica New Relic . También puede ver los datos dashboard para una integración específica o en toda su cuenta.

Para visualizaciones de intervalo de sonido, llamada API y otros datos para su integración de Azure:

1. Vaya a

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure**
   </DNT>

   .

2. Para ver datos de una integración específica: seleccione el enlace

   <DNT>
     **Dashboards**
   </DNT>

   para la fila de la integración.

## New Relic intervalo de sondeo [#aws-integrations]

Para obtener detalles sobre el sondeo y la resolución, consulte la documentación de una integración específica en la sección [Lista de integración de Azure](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/) .
