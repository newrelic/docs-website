## Problema

No puede emplear AWS Secrets Manager para almacenar su clave de licencia de New Relic.

## Solución

Si su organización no permite el uso de AWS Secrets Manager, la extensión Lambda de New Relic acepta una variable de entorno `NEW_RELIC_LICENSE_KEY` . Agrega la bandera `--disable-license-key-secret` del comando `newrelic-lambda integrations install` . Luego, configure esta variable de entorno en <InlinePopover type="licenseKey" />en la configuración de su función Lambda.
