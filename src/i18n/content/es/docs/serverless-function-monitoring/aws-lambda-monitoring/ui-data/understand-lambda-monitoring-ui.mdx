[La monitorización serverless para AWS Lambda](/docs/introduction-new-relic-monitoring-aws-lambda) ofrece una monitorización detallada del rendimiento de su función Lambda. Este documento explica cómo:

* [Encuentre sus datos Lambda en la UI](#find-data)
* [Comprender los componentes de la UI](#ui-pages)
* [Comprenda los datos de su gráfico](#chart-data)
* [Cómo crear gráficos personalizados](#chart-data)

Para ver sus datos Lambda en New Relic: vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT>, haga clic en <DNT>**Explorer.**</DNT> En el navegador izquierdo debajo de <DNT>**Amazon Web Services**</DNT>, haga clic <DNT>**Lambda functions.**</DNT>

Para obtener más información sobre nuestra UI, consulte [Introducción a la plataforma New Relic](/docs/project-wanda-early-access-future-new-relic-platform).

<Callout variant="important">
  Si no puede encontrar sus datos Lambda:

  * Asegúrese de haber seguido las [instrucciones para habilitar el monitoreo Lambda](/docs/install-enable-new-relics-monitoring-aws-lambda).
  * Tenga en cuenta que esta característica es diferente de nuestra [integración Lambda de monitoreo de infraestructura](/docs/integrations/amazon-integrations/aws-integrations-list/aws-lambda-monitoring-integration).
</Callout>

## Comprender los datos del gráfico [#chart-data]

Los gráficos de datos Lambda se generan ejecutando [la consulta NRQL](/docs/insights/nrql-new-relic-query-language/using-nrql/introduction-nrql) de [datos de eventos](/docs/understand-your-lambda-data#data-structure) relacionados con Lambda. Los motivos para ver la consulta NRQL de un gráfico incluyen:

* Para comprender mejor lo que muestra un gráfico
* Para obtener ideas sobre cómo crear una consulta y un gráfico NRQL personalizados

Documentación relacionada:

* [Aprenda a ver la consulta de un gráfico](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform#view-query).
* [Obtenga más información sobre la estructura y el almacenamiento de datos de Lambda](/docs/understand-your-lambda-data).
