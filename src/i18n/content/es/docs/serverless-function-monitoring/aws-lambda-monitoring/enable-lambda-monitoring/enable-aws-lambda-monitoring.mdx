Hay varios pasos para habilitar el monitoreo Lambda:

1. [Vincula tu cuenta de AWS.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/account-linking)
2. [Opcional: configure la recopilación log de CloudWatch](/docs/logs/enable-log-management-new-relic/enable-log-monitoring-new-relic/aws-lambda-sending-cloudwatch-logs): si en el paso 1 utilizó la CLI `newrelic-lambda`, instala automáticamente el recolector de registros de CloudWatch de forma predeterminada y puede omitir este paso.
3. [Instrumentado una función de ejemplo.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-example)
4. [Instrumentado sus funciones.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-your-own)
5. [Opcional: Configurar alerta y evento personalizado.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/configure-serverless-monitoring-aws-lambda)

Consulte también [instrumentación manual heredada para Lambda](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy).
