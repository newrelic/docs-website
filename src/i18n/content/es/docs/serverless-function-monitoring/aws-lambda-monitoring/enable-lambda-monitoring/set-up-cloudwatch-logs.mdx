En el Paso 1, si utilizó la CLI newrelic-lambda, verificó si había instalado la integración [API Polling](/docs/infrastructure/amazon-integrations/connect/connect-aws-new-relic-infrastructure-monitoring) o [métrica Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream) . Si no vio ninguno de ellos, instaló la integración de sondeo API.

Si no utilizó la CLI en el Paso 1, asegúrese de haber instalado y configurado la integración de API Polling o métrica Streams.
