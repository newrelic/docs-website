Puede configurar el monitoreo de New Relic para aplicaciones de funciones Azure hospedadas en Linux, Windows o como funciones en contenedores. Esto se logra instrumentando su aplicación de función con el agente New Relic apropiado para su entorno de ejecución, como Node.js .NET o Python.

## Antes de que empieces [#begin]

Cerciorar de que su función de Azure cumpla con nuestros [requisitos y compatibilidad](/docs/serverless-function-monitoring/azure-function-monitoring/compatibility-requirement-azure-monitoring).

<Steps>
  <Step>
    ## Vincula tu cuenta de Azure a New Relic [#link]

    Vincule su cuenta Azure a New Relic configurando el sondeo para el monitoreo métrico de Azure . Esto le permite ver estas métricas en la New Relic UI. Para obtener más información, consulte [Integración Azure ](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-functions-monitoring-integration/#polling).
  </Step>

  <Step>
    ## Instrumentar su función Azure [#instrument]

    Según el entorno de su implementación, seleccione una de las siguientes opciones para instrumentar su función Azure con New Relic.

    <Tabs>
      <TabsBar>
        <TabsBarItem id="container">Función contenedorizada de</TabsBarItem> <TabsBarItem id="linux">Linux</TabsBarItem> <TabsBarItem id="windows">y Windows</TabsBarItem>
      </TabsBar>

      <TabsPages>
        <TabsPageItem id="linux">
          Puede instrumentar su aplicación de funciones Azure en Linux con New Relic para los siguientes entornos de ejecución:

          * [Node.js](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#node)
          * [Python](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#python)
          * [.NET](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#net)
        </TabsPageItem>

        <TabsPageItem id="windows">
          Puede instrumentar su aplicación de funciones Azure en Windows con New Relic para los siguientes entornos de ejecución:

          * [Node.js](/docs/serverless-function-monitoring/azure-function-monitoring/windows/#node)
          * [.NET](/docs/serverless-function-monitoring/azure-function-monitoring/windows/#net)
        </TabsPageItem>

        <TabsPageItem id="container">
          Puede instrumentar su aplicación de funciones Azure en Linux con New Relic para los siguientes entornos de ejecución:

          * [Node.js](/docs/serverless-function-monitoring/azure-function-monitoring/container/#node)
          * [Python](/docs/serverless-function-monitoring/azure-function-monitoring/container/#python)
          * [.NET](/docs/serverless-function-monitoring/azure-function-monitoring/container/#net)
        </TabsPageItem>
      </TabsPages>
    </Tabs>
  </Step>

  <Step>
    ## Configurar variables de entorno [#env-variables]

    Luego de instrumentar su aplicación de funciones Azure con New Relic, [configure las variables de entorno](/docs/serverless-function-monitoring/azure-function-monitoring/env-variables-azure) para su entorno de ejecución.
  </Step>

  <Step>
    ## Resetear sus funciones Azure [#restart]

    Luego de agregar las variables de entorno, resetear Azure Functions para aplicar los cambios.
  </Step>
</Steps>
