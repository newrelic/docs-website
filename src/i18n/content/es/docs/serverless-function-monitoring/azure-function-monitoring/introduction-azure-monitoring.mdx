La instrumentación de New Relic para Azure Functions ofrece una visibilidad profunda del funcionamiento interno de sus funciones. Al agregar el agente New Relic a su función, cada invocación es monitoreada por New Relic. Esto le permite rastrear la duración de las invocaciones de funciones, monitorear errores, ver el rastreo distribuido y registrar detalles sobre el evento de origen y las respuestas de su función.

<Callout variant="important">
  Para Azure Functions, el agente informa datos como el rastreo distribuido y los logs, que están disponibles en &apos;Servicios - APM&apos; como una entidad APM . Sin embargo, todavía existe una entidad separada (proporcionada por monitoreo de infraestructura) para Azure Functions .
</Callout>
