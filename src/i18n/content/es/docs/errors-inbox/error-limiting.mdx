Un número limitado de cuentas puede recibir la siguiente advertencia al intentar utilizar Errors Inbox:

### Problema [#problem]

```
You’ve reached the daily limit of unique error groups on account xxx.  This limit will be reset at 00:00 UTC.
```

### Solución [#solution]

Para reducir la cantidad de grupos de errores, intente eliminar el identificador único de la clase o mensaje de error, [más detalles aquí](/docs/errors-inbox/errors-inbox/#similar-events)

* * *

### Problema [#problem]

```
You’ve reached the daily maximum unique timeseries (cardinality) limit on account xxx.  This limit will be reset at 00:00 UTC.
```

Cuando se alcanza este límite, Errors Inbox ya no se actualizará y los errores parecerán estabilizarse o disminuir.

### Solución [#solution]

Para solucionar este problema, [reduzca el alcance de su carga de trabajo](/docs/new-relic-one/use-new-relic-one/workloads/use-workloads/).
