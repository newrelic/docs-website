## Problema

Cuando se ejecuta otro software de monitoreo junto con uno de nuestros agentes APM, no podemos garantizar que nuestro agente funcione correctamente. En algunos casos, ejecutar más de un tipo de instrumentación, APM o software de monitoreo puede causar problemas como métricas poco confiables y fallas del agente.

Para obtener detalles específicos, consulte la documentación de requisitos y compatibilidad para un agente APM específico.

## Solución

Si cree que ejecutar otro software de monitoreo podría estar causando un problema, aquí tiene algunas recomendaciones:

* Lea la documentación de requisitos y compatibilidad del agente APM que está utilizando.
* Para solucionar si esto está causando un problema, deshabilite o desinstale temporalmente otro software APM para ver si el problema aún persiste.
