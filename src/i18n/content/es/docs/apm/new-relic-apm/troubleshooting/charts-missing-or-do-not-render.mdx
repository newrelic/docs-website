## Problema

Faltan gráficos en la UI de New Relic o no se muestran correctamente.

## Solución

Las extensiones browser que modifican JavaScript, como Adblock, Ghostery o Request Maker, pueden causar problemas de visualización. Para solucionar este problema, agregue nuestro dominio New Relic a su lista de 'permitidos' o desinstale la extensión.

Además, es posible que su browser sea demasiado antiguo para mostrar gráficos de New Relic. Verifique que su browser cumpla con nuestras [versiones mínimas browser admitidas](/docs/apm/new-relic-apm/getting-started/supported-browsers).
