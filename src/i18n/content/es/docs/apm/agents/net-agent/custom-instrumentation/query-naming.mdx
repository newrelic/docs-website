Nuevo agente Relic .NET instrumentado y agrupa la consulta SQL de tu aplicación. Normalmente, el agente .NET asignará automáticamente un nombre a una consulta según las propiedades de la consulta. Alternativamente, puede asignar un nombre personalizado a una consulta.

## Cómo nombrar una consulta [#name-query]

A todas las versiones de SQL admitidas por nuestro agente .NET se les puede asignar un nombre personalizado.

Para darle un nombre a una consulta, agrega un comentario al texto de la consulta con un formato como `/* NewRelicQueryName: myQueryName */`. Aquí hay un ejemplo de cómo agregar eso al código de una aplicación:

```cs
var connection = new MySqlConnection(mySqlConnectionString);
var command = new MySqlCommand("/* NewRelicQueryName: My query name */ SELECT date FROM dates", connection);
```

## Buscar consulta nombrada [#find-named-query]
