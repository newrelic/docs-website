## Problema

Cuando implementa una nueva versión del agente .NET de New Relic (`NewRelic.Profiler.dll`) para su [aplicación web Microsoft Azure](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps), un mensaje de error le impide continuar hasta que detenga o reinicie el proceso.

## Solución

Normalmente, el proceso se detiene, luego los recursos se sobrescriben con los recursos que New Relic empaqueta en la raíz del sitio (ubicada en `D:\Home\site\wwwroot\newrelic`) durante un despliegue. Para resolver el problema, pruebe cualquiera de estas soluciones.

<CollapserGroup>
  <Collapser
    id="stop-dll-deploy"
    title="Evitar que el archivo se publique"
  >
    Esta solución está disponible para su aplicación web de Azure si intenta implementar una nueva versión del agente .NET sobre una versión existente:

    1. Detenga su aplicación web para liberar el archivo `.dll` .
    2. Permita que la actualización continúe.
    3. Una vez que se complete el despliegue, reinicie la instancia.
  </Collapser>

  <Collapser
    id="site-extension"
    title="Utilice la extensión del sitio New Relic"
  >
    Puede instalar el agente .NET de New Relic en su aplicación web de Azure utilizando la [extensión del sitio de New Relic](https://www.siteextensions.net/packages/NewRelic.Azure.WebSites/). (Sólo debería necesitar instalar la extensión una vez).

    1. Navegue hasta <DNT>**https&#x3A;//yoursite.scm.azurewebsites.net/**</DNT> y seleccione <DNT>**Site Extensions**</DNT>.

    2. Desde <DNT>**Gallery**</DNT>, seleccione el complemento <Icon name="fe-plus"/> y luego seleccione <DNT>**New Relic**</DNT>.

    3. Siga las instrucciones para [agregar manualmente las claves de configuración](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps) a su aplicación web y use la opción que mejor se adapte a sus necesidades.

       Para actualizar la extensión del sitio: desde [Azure Portal](https://portal.azure.com), seleccione <DNT>**WebApps > (select your site) > All Settings > Extensions > New Relic**</DNT>.
  </Collapser>
</CollapserGroup>
