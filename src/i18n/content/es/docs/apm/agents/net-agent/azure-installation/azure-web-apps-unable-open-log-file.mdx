## Problema

Se está conectando vía FTP a su [aplicación web Microsoft Azure](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps). Puede ver un archivo `NewRelic.Profiler.####.log` en <DNT>**LogFiles\\NewRelic**</DNT>, pero cuando intenta acceder al archivo de registro, aparece un mensaje de error.

## Solución

Reinicie su aplicación web para liberar el archivo y crear un nuevo log de perfil.
