Este documento explica cómo desinstalar el agente New Relic .NET. Para obtener instrucciones sobre cómo deshabilitar temporalmente el agente, consulte [Deshabilitar el agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#net).

Para desinstalar el agente .NET siga el procedimiento que corresponda a su sistema operativo:

* [Windows](#uninstall-framework-windows)
* [Linux](#uninstall-linux)

## Desinstalar el agente .NET (Windows) [#uninstall-framework-windows]

<Callout variant="tip">
  Requiere [<DNT>**Administrator**</DNT> derechos](/docs/agents/net-agent/getting-started/compatibility-requirements-net-framework-agent#user-rights) en su grupo de administración de Windows.
</Callout>

Este procedimiento supone que instaló el agente .NET con el instalador MSI:

1. Para cerrar cualquier agente en ejecución, detenga IIS. Si no detiene IIS, es posible que deba reiniciar después de desinstalar el agente.

2. Seleccione

   <DNT>
     **Start > All Programs > New Relic .NET Agent > Uninstall the .NET agent**
   </DNT>

   .

3. Siga el símbolo para completar la desinstalación.

4. Reinicie IIS.

Después de desinstalar el agente, es posible que desee eliminar todos los archivos relacionados con el agente. Asegúrese de verificar otras rutas para eliminar cualquier otro archivo New Relic. Por ejemplo, desinstalar el agente .NET no elimina su archivo <DNT>**[newrelic.config](/docs/agents/net-agent/installation-configuration/net-agent-configuration#config-options-precedence)**</DNT> .

1. Desde

   <DNT>
     **Control Panel > Programs > Programs and Features**
   </DNT>

   , haga clic derecho y elimine cualquier archivo llamado `New Relic .NET Agent`.

2. Elimine toda la carpeta en `C:\ProgramData\New Relic\.NET Agent`.

3. Elimine toda la carpeta en `C:\Program Files\New Relic\.NET Agent`.

4. Elimine toda la carpeta en `C:\Program Files (x86)\New Relic\.NET Agent`.

5. [Elimine su aplicación](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) de la UI de APM.

## Desinstalar el agente .NET (Linux) [#uninstall-linux]

Para desinstalar el agente .NET en Linux, utilice su herramienta de administración de paquetes.

<Callout variant="important">
  Para las versiones 10.0.0 o superiores del agente .NET, el nombre del paquete es `newrelic-dotnet-agent`. Para las versiones 9.9.0 o inferiores del agente .NET, el nombre del paquete es `newrelic-netcore20-agent`. El resto de este documento hará referencia a `newrelic-dotnet-agent`; reemplácelo con el nombre anterior si tiene instalada una versión anterior del agente.
</Callout>

<Callout variant="important">
  Debe reiniciar su aplicación una vez finalizada la desinstalación.
</Callout>

<CollapserGroup>
  <Collapser
    id="apt-uninstall"
    title="Desinstalar con apt (Debian, Ubuntu y Linux Mint)"
  >
    Ejecute el siguiente comando como root (o usando sudo):

    ```
    apt-get remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="dpkg-uninstall"
    title="Desinstalar con dpkg (Debian, Ubuntu y Linux Mint)"
  >
    Ejecute el siguiente comando como root (o usando sudo):

    ```
    dpkg --remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="yum-uninstall"
    title="Desinstalar con yum (CentOS, Red Hat Enterprise Linux, Oracle Linux)"
  >
    Ejecute el siguiente comando como root (o usando sudo):

    ```
    yum remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="rpm-uninstall"
    title="Desinstalar con rpm (CentOS, Red Hat Enterprise Linux, Oracle Linux)"
  >
    Ejecute el siguiente comando como root (o usando sudo):

    ```
    sudo rpm -e newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="tarball"
    title="Desinstalar tarball"
  >
    1. Elimina todos los archivos de New Relic de la carpeta de tu aplicación.

    2. Variables de entorno de New Relic sin configurar:

       ```
       CORECLR_ENABLE_PROFILING=1
       CORECLR_PROFILER={36032161-FFC0-4B61-B559-F6C5D41BAE5A}
       CORECLR_NEWRELIC_HOME=PATH_TO_INSTALL
       CORECLR_PROFILER_PATH=%CORECLR_NEWRELIC_HOME%\NewRelic.Profiler.dll
       ```
  </Collapser>
</CollapserGroup>
