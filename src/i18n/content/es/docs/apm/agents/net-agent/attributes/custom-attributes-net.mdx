[Los atributos](/docs/features/agent-attributes) son pares de valores principales que contienen información que determina las propiedades de un evento o transacción. Puedes crear un atributo personalizado utilizando la [API`AddCustomAttribute` ](/docs/agents/net-agent/net-agent-api/itransaction/#addcustomattribute).

Al agregar valores de atributo personalizado a transacción, evento personalizado, intervalos y errores, la API acepta un `object`. Esto describe cómo se procesan estos valores y cómo aparecerán en APM. En todos los casos, los valores `NULL` no se registran.
