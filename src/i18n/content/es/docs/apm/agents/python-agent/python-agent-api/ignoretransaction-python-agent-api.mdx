## Sintaxis

```py
newrelic.agent.ignore_transaction(flag=True)
```

Ignora la transacción actual.

## Descripción

Esta llamada ignora la transacción actual. El agente no reporta ningún dato de la transacción.

<Callout variant="tip">
  También puede ignorar una transacción web en el diccionario de entorno WSGI. Para hacerlo, configure la clave `newrelic.ignore_transaction` para la solicitud específica en el diccionario de entorno WSGI pasado por el servidor WSGI a su aplicación WSGI.
</Callout>

## Ejemplos

### Ignorar una transacción [#ignore-example]

Para ignorar una transacción que no desea que aparezca en la UI, debe ejecutar lo siguiente donde se genera una transacción en el código de su aplicación. Este ejemplo ilustra el uso de la API para ignorar una ruta de Flask:

```py
@app.route("/")
def process_results():
    newrelic.agent.ignore_transaction(flag=True)
    return 'hello world'
```
