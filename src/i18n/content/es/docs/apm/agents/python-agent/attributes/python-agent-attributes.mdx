[Los atributos](/docs/features/agent-attributes) son pares de valores principales que contienen información que determina las propiedades de un evento o transacción. Permitimos al usuario personalizar exactamente qué atributo se enviará a cada uno de estos destinos:

* [Traza de la transaccion](/docs/apm/transactions/transaction-traces/transaction-traces)
* [Errores de traza](/docs/apm/applications-menu/events/viewing-apm-errors-error-traces)
* [Evento de transacción](/docs/insights/new-relic-insights/decorating-events/apm-default-attributes-insights)
* [Vistas de página](/docs/insights/new-relic-insights/decorating-events/browser-default-attributes-insights)

Estas configuraciones de atributos se aplican a la versión 2.56.0.42 o superior del agente Python. Para aprovechar todas las características disponibles, [actualice su agente Python](/docs/agents/python-agent/installation-configuration/upgrade-python-agent) a la [última versión](/docs/release-notes/agent-release-notes/python-release-notes).

<CollapserGroup>
  <Collapser
    id="httpUrl"
    title="http.url"
  >
    La URL de la llamada saliente, sin ningún parámetro de solicitud.

    Excluido cuando:

    * Una regla que coincide con 'http.url' está presente en "transaction_segments.attributes.exclude" configuración. (para segmentos)

    * Hay una regla que coincide con 'http.url' en "span_events.attributes.exclude" ajuste de configuración. (para tramos)

      Valores predeterminados:

    * Segmentos de transacción: habilitado

    * Evento de extensión: habilitado
  </Collapser>
</CollapserGroup>

## Añadir atributo personalizado [#adding_custom_attributes]

Para capturar atributos personalizados adicionales de su aplicación, use `newrelic.agent.add_custom_attribute()`.

Para obtener una referencia completa, consulte [Recopilación de atributos personalizados](/docs/apm/other-features/attributes/collecting-custom-attributes#python-att).

Valores predeterminados:

* Traza de la transacción: Habilitado
* Recolector de errores (errores de traza): Habilitado
* Evento de transacción: Habilitado
* Vistas de página (monitoreo de navegador): Deshabilitado

<Callout variant="important">
  Antes de crear un atributo personalizado, revisa nuestra lista de [términos reservados](/docs/insights/event-data-sources/custom-events/data-requirements-limits-custom-event-data/#reserved-words). De lo contrario, podrían producirse resultados inesperados.
</Callout>

## Agregar atributo personalizado a las excepciones [#adding_error_attributes]

Al utilizar la llamada API `newrelic.agent.notice_error()` , puede pasar un diccionario de `params` que se registrará como atributo en el error de traza.

Valores predeterminados:

* Traza de la transacción: No disponible
* Recolector de errores (errores de traza): Habilitado
* Evento de transacción: No disponible
* Vistas de página (monitoreo de navegador): No disponible

## Actualice su archivo de configuración [#upgrading]

Actualice su archivo de configuración <DNT>**newrelic.ini**</DNT> cuando actualice al agente Python <DNT>**2.56.0.42**</DNT> o superior. Para obtener más información sobre los ajustes de configuración obsoletos, consulte [Habilitación y deshabilitación de atributos](/docs/agents/python-agent/attributes/enabling-disabling-attributes-python#deprecated).
