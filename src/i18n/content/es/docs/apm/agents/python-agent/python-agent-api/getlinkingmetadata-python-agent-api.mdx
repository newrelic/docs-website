## Sintaxis

```py
get_linking_metadata()
```

Devuelve un diccionario de datos contextuales para usar en mensaje de log.

## Requisitos

Agente Python versión 5.4.0 o superior

## Descripción

Un diccionario de pares de valores principales que proporciona información contextual. Esta API se puede utilizar para agregar metadatos del agente para el registro manual.

## Valores de retorno

Devuelve un diccionario.

## Ejemplos

```py
import json
import newrelic.agent
log_message = {"message": "Hello World"}
context = newrelic.agent.get_linking_metadata()
log_message.update(context)
print(json.dumps(log_message))
```
