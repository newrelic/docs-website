## Preguntas frecuentes [#faq]

### ¿Qué es un paquete de certificados? [#what-is-certificate-bundle]

Básicamente, un paquete de certificados proporciona una lista de raíces de confianza. Le dice al agente en quién confiar cuando se comunica con el backend.

### ¿Por qué se proporciona el paquete de certificados? [#why-certificate-bundle]

Muchos clientes ejecutan Docker. A veces, esas imágenes no proporcionan sus propios paquetes de certificados porque normalmente no se utilizan para comunicarse a través de HTTP. Proporcionamos nuestro propio paquete para que a los clientes les resulte más fácil instalar el agente.

### ¿Cuándo se utiliza el paquete de certificados? [#when-to-use-certificate-bundle]

Cuando los certificados del sistema estén disponibles, el agente los utilizará de forma predeterminada en el paquete de certificados incluido. El paquete de certificados solo se utiliza cuando el sistema no proporciona ningún certificado.
