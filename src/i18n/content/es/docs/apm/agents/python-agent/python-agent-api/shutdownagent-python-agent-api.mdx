## Sintaxis

```py
newrelic.agent.shutdown_agent(timeout=None)
```

Apaga por la fuerza al agente.

## Descripción

Cerrar por la fuerza al agente. El agente hace un último intento de cargar datos en el [recolector](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector).

## Parámetros

<table>
  <thead>
    <tr>
      <th width="25%">
        Parámetro
      </th>

      <th>
        Descripción
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `timeout`

        _int_ o _flotante_
      </td>

      <td>
        Opcional, pero muy recomendable. El período máximo de tiempo que la persona que llama espera a que el agente termine de cargar datos. Si el período de tiempo de espera finaliza antes de que finalice el agente, el agente continúa informando datos en segundo plano, pero la persona que llama puede interrumpir todo el proceso por la fuerza y detener la carga.

        Si no se establece ningún valor o se establece en `None`, el agente utiliza el valor de tiempo de espera global predeterminado, que se puede anular en los ajustes de configuración, pero su valor predeterminado es 2,5 segundos.
      </td>
    </tr>
  </tbody>
</table>

## Valores de retorno

Ninguno.

## Ejemplos

### Con un tiempo muerto de tres segundos [#timeout-example]

```py
newrelic.agent.shutdown_agent(timeout=3)
```
