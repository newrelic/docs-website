## Sintaxis

```py
newrelic.agent.memcache_trace(command, async_wrapper=None)
```

Se utiliza para instrumentar funciones de comando de memcache para funciones que no están instrumentadas por defecto.

## Descripción

`memcache_trace` es un decorador para agregar a las funciones de comando de memcache. Agregar este decorador le permite recopilar información adicional de la [transacción](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction) (incluida la información de la traza de la transacción).

El decorador `memcache_trace` se puede usar en funciones con la versión del agente 2.102.0.85 o superior. La medición del tiempo de estos objetos comienza cuando se inicia el consumo, y termina cuando el objeto se agota o sale del ámbito.

Puede utilizar el decorador junto con decoradores existentes, incluidos los de métodos estáticos y de clase. Al anidar varios decoradores, tenga el decorador `memcache_trace` como el decorador más externo.

## Formularios de llamada alternativos [#alternate]

Para configuraciones en las que no puede utilizar el decorador, estos formularios de llamada alternativos están disponibles:

### The wrapper

Si sabes de antemano dónde están las funciones específicas que deseas trazar, puedes usar el decorador `memcache_trace` . Sin embargo, si <DNT>**don&apos;t**</DNT> conoce todas las funciones que deben trazarse (por ejemplo, si se buscan dinámicamente como parte de un sistema de enrutamiento), entonces debe usar `MemcacheTraceWrapper` para ajustar la función en el en el momento de la inscripción o en el momento de la convocatoria.

### Envoltura basada en rutas [#path-based]

`wrap_memcache_trace` se utiliza para encapsular funciones fuera del código en el que están declaradas. Por ejemplo: podrías usar esto para instrumentar el código de la biblioteca que no deseas modificar.

Para obtener más información sobre las diferencias entre estos formatos de llamadas, consulte [Diferentes formatos de llamadas](/docs/python-agent-api-different-call-forms).

## Ejemplos

### `memcache_trace` [#trace\_example][#trace_example]

Un ejemplo de uso del decorador `memcache_trace` :

```py
import newrelic.agent

@newrelic.agent.memcache_trace("get")
def get(id):
    ...
```

### Ejemplo de contenedor [#trace-wrap-example]

Un ejemplo de uso de `MemcacheTraceWrapper`:

```py
import newrelic.agent

def get(id):
    ...

get = newrelic.agent.MemcacheTraceWrapper(get, command="get")
```

Si desea devolver dinámicamente el valor del comando, puede proporcionar el comando como una función cuando se crea el objeto `MemcacheTraceWrapper`.

```py
import newrelic.agent

def get(id):
    ...

get = newrelic.agent.MemcacheTraceWrapper(get, command=lambda id: f"get:{id}")
```
