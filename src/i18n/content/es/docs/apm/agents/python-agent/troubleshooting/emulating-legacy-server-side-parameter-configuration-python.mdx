## Problema

Desea emular el legacy comportamiento de habilitar las <DNT>**Capture attributes**</DNT> <DNT>**Capture parameters**</DNT> opciones o en los UI ajustes de [configuración del lado del servidor de la de New Relic.](/docs/agents/manage-apm-agents/configuration/server-side-agent-configuration)

## Solución

Puede emular este comportamiento legacy personalizando el archivo de configuración del agente Python para capturar el parámetro de solicitud. Para obtener más información, consulte el [ejemplo del parámetro de solicitud de captura](/docs/agents/python-agent/attributes/python-attribute-examples#ex_req_params) en la documentación del atributo de Python.
