## Sintaxis

```py
newrelic.agent.capture_request_params(flag=True)
```

Habilita la captura del parámetro de cadena de consulta de una transacción.

## Descripción

Esta llamada permite capturar el parámetro de cadena de consulta de una transacción web como [atributo](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#attribute). Esta colección está bloqueada de forma predeterminada por motivos de seguridad. Si, en cambio, desea habilitar la recopilación de parámetros de cadenas de consulta para toda su aplicación, utilice la configuración [`record_sql`](/docs/agents/python-agent/installation-configuration/python-agent-configuration#txn-tracer-sql) .

[El modo de alta seguridad](/docs/accounts-partnerships/accounts/security/high-security) anula esta llamada si está activo, en cuyo caso esta llamada no tiene efecto. Esta llamada anula otras configuraciones relevantes en la [configuración](/docs/agents/python-agent/installation-configuration/python-agent-configuration) del agente.

<Callout variant="tip">
  También puede habilitar la captura de cadenas de consulta en el diccionario de entorno WSGI. Para hacerlo, marque la clave `newrelic.capture_request_params` para la solicitud específica en el diccionario de entorno WSGI pasado por el servidor WSGI a la aplicación WSGI de destino.
</Callout>
