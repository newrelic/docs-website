## Problema

Después de instalar el agente Python, generar algo de tráfico y esperar cinco minutos, sus datos no aparecen en la UI de New Relic.

## Solución

A continuación se ofrecen algunos consejos para la resolución de problemas de datos faltantes, colocados en orden aproximado de probabilidad de que sean el problema:

1. Asegúrese de haber instalado el agente en el mismo entorno que su aplicación. Si está utilizando un entorno virtual, instálelo allí.

2. Asegúrese de que su agente esté actualizado. Si lo descargó del sitio New Relic en lugar de usar

   <DNT>
     **pip**
   </DNT>

   o

   <DNT>
     **easy_install**
   </DNT>

   , compare su número de versión con la [versión actual](https://download.newrelic.com/python_agent/release/).

3. Utilice [New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics) para intentar identificar automáticamente el problema.

4. Asegúrese de que los datos puedan llegar a la UI de New Relic ejecutando la [prueba del agente Python](/docs/python/testing-the-python-agent).

5. Asegúrese de cumplir con la [compatibilidad y los requisitos](/docs/agents/python-agent/getting-started/compatibility-requirements-python-agent). Verifique si hay requisitos especiales para el [framework web o el servidor web](/docs/agents/python-agent/hosting-mechanisms) que está utilizando.

6. Revise el archivo de configuración que se generó con el comando `newrelic-admin generate-config` para asegurarse de que contenga el

   <InlinePopover type="licenseKey"/>

   correcto.

7. [Genere un registro detallado para la resolución de problemas](/docs/agents/python-agent/troubleshooting/generating-logs-troubleshooting). Luego revise el archivo de registro para ver si hay mensajes de error. Si falta el archivo de registro, asegúrese de tener permisos para escribir en esa ubicación.

Además, si no aparecen datos y ves mensajes de advertencia sobre cómo activar o reactivar tu aplicación, sigue los [procedimientos de resolución de problemas de activación](/docs/agents/python-agent/troubleshooting/activate-application-warning-python).
