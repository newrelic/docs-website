## Sintaxis

```py
newrelic.agent.disable_browser_autorum(flag=True)
```

Deshabilita <InlinePopover type="browser"/>para una solicitud web.

## Descripción

Esta llamada deshabilita [el monitoreo del navegador](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) para la página/solicitud que se solicita. Esta llamada solo afecta la solicitud web actual y debe realizarse antes del punto donde se inyecta el JavaScript del encabezado/pie de página del navegador. Para obtener más información sobre cómo se inserta el agente del navegador JavaScript, consulte [Instalar monitoreo de navegador](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser).

<Callout variant="tip">
  También puede deshabilitar la inyección browser configurando la clave `newrelic.disable_browser_autorum` para la solicitud específica en el diccionario de entorno WSGI pasado por el servidor WSGI a la aplicación WSGI que se está monitoreando.
</Callout>
