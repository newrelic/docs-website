Genera un pie de página de agente <InlinePopover type="browser" />para inyectar el agente de JavaScript.

## Descripción [#description]

Esta llamada devuelve el fragmento de pie de página de JavaScript del agente del navegador que, junto con el [fragmento de JavaScript del encabezado](/docs/agents/python-agent/python-agent-api/get_browser_timing_header), se utiliza para habilitar el monitoreo del navegador. Para aprender cómo habilitar el monitoreo del navegador usando esta llamada junto con `get_browser_timing_header`, consulte [Instrumento manual de monitoreo del navegador](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation).

Esta llamada debe realizarse para cada solicitud web en la que desee inyectar el fragmento de JavaScript. <DNT>**cannot**</DNT> se llamará en el ámbito global y se almacenará en caché.

## Valores de retorno [#return-values]

Devuelve el JavaScript del pie de página del agente del navegador.
