Para aprovechar al máximo nuestras últimas características, mejoras y parches de seguridad importantes, le recomendamos que actualice su agente Python a la última versión. Para obtener información adicional sobre actualizaciones de agentes específicas, consulte las [notas de la versión de Python agente](/docs/release-notes/agent-release-notes/python-release-notes).

## Actualización con índice de paquetes Python (PyPi) [#package-index]

El paquete del agente Python aparece en PyPi en [http://pypi.python.org/pypi/newrelic](http://pypi.python.org/pypi/newrelic "El enlace se abre en una nueva ventana"). Para actualizar con un paquete específico, siga las instrucciones correspondientes:

      <td>
        Para actualizar el paquete `newrelic` directamente desde PyPi, ejecute:

        ```
        pip install --upgrade newrelic
        ```

        Si tiene New Relic en su archivo `requirements.txt` , puede realizar una actualización masiva de todos los paquetes necesarios ejecutando:

        ```
        pip install -r requirements.txt
        ```

          Recomendamos usar `pip` en lugar de `easy_install`, porque `pip` eliminará correctamente las versiones antiguas al actualizar el software del agente.
        </Callout>

## Descargar el paquete manualmente [#manual-download]

Para descargar manualmente el paquete New Relic Python:

1. Elimina el paquete New Relic actual.
2. Descargue el nuevo paquete y luego siga los procedimientos [de instalación del agente Python](/docs/agents/python-agent/installation-configuration/python-agent-installation#download-site) .

<Callout variant="tip">
  Recomendamos utilizar `pip` en lugar de utilizar la instalación manual, porque `pip` eliminará correctamente las versiones antiguas al actualizar el software del agente.
</Callout>
