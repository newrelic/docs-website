## Sintaxis

```py
newrelic.agent.end_of_transaction()
```

Finaliza la transacción actual.

## Descripción

Esta llamada finaliza la transacción actual inmediatamente y detiene el temporizador de respuesta en el momento actual. Una vez finalizada la transacción, el agente no recopila ninguna métrica adicional de desglose del rendimiento. Sin embargo, el agente aún recopila detalles de la excepción de Python.

Esta llamada es útil cuando tiene una gran respuesta que se transmite para la transacción web y solo la parte inicial de la transacción es de su interés. (Esto también supone que, por cualquier motivo, no desea marcar la transacción como una [tarea en segundo plano](/docs/agents/python-agent/python-agent-api/set_background_task)).

## Parámetros

Ninguno.

## Valores de retorno

Ninguno.
