## Sintaxis

```py
newrelic.agent.application(name=None)
```

Devuelve el objeto correspondiente a una aplicación de monitorización de agentes; requerido por algunas llamadas API del agente Python.

## Descripción

Esta llamada devuelve un objeto correspondiente a una aplicación de agente-monitor. Si no se establece ningún valor `name` , se utiliza el nombre [establecido mediante la configuración del agente Python](/docs/agents/python-agent/installation-configuration/python-agent-configuration#app_name) .

El objeto de aplicación devuelto no presenta ninguna API pública, pero es requerido por alguna otra API de llamada.

<Callout variant="important">
  Esta llamada solo devuelve el objeto de la aplicación. No intentará registrar la aplicación con el recolector de datos si aún no lo ha hecho.
</Callout>

## Valores de retorno

Devuelve un objeto de aplicación. El objeto en sí no presenta ninguna API pública, pero es requerido por alguna otra API de llamada.

## Ejemplos

### Generar y utilizar un objeto de aplicación. [#app-object-example]

Para generar un objeto de aplicación, utilice:

```py
import newrelic.agent

app = newrelic.agent.application()
```

Algunas otras llamadas API del agente Python toman el objeto de la aplicación como parámetro. Para ver un ejemplo, consulte la llamada [`notice_error`](/docs/agents/python-agent/python-agent-api/noticeerror-python-agent-api/) .
