[Los middlewares de rack](/docs/agents/ruby-agent/frameworks/rack-middlewares) se instrumentan automáticamente con la versión 3.9.0 o superior del agente Ruby.

El módulo `NewRelic::Agent::Instrumentation::Rack` está en desuso. Después de actualizar a la versión 3.9.0 o superior del agente Ruby, New Relic recomienda que elimine cualquier referencia a este módulo de su código. Para obtener más información, consulte [Controladores de metal instrumentado](/docs/agents/ruby-agent/frameworks/instrumenting-metal-controllers).
