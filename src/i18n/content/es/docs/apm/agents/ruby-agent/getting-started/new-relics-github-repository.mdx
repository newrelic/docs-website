El agente New Relic Ruby ahora es completamente de código abierto y está disponible en GitHub en [newrelic-ruby-agente](https://github.com/newrelic/newrelic-ruby-agent/tree/main). A partir de la versión 6.12 (lanzada el 16 de julio de 2020), tiene la licencia Apache, versión 2.0. Para versiones anteriores del agente, consulte el archivo LICENCIA incluido con el código fuente. El código fuente de las versiones 2.1.0 del agente Ruby están disponibles en GitHub usando etiqueta.

El jefe de `main` tendrá el último lanzamiento de New Relic. Sin embargo, nos reservamos la posibilidad de llevar una ventaja al `main`. La rama `dev` incluye trabajos inéditos. Si descarga una versión de GitHub, utilice la etiqueta adecuada. Normalmente publicamos versiones preliminares a través de GitHub antes de publicar una nueva versión en RubyGems.

Bifurque y no dude en [enviarnos \[la\] solicitud de extracción](https://github.com/newrelic/newrelic-ruby-agent/pulls), [crear problemas](https://github.com/newrelic/newrelic-ruby-agent/issues/new/choose) e [iniciar discusiones](https://github.com/newrelic/newrelic-ruby-agent/discussions) directamente en el repositorio de GitHub. ¡Nos encanta trabajar con la comunidad!
