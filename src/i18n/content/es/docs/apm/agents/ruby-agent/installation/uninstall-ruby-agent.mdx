Este documento explica cómo desinstalar el agente New Relic Ruby. Para obtener instrucciones sobre cómo deshabilitar temporalmente el agente, consulte [Deshabilitar el agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#ruby).

Para desinstalar el agente Ruby:

1. Elimina la línea `gem 'newrelic_rpm'` de tu Gemfile.

2. Elimine el archivo

   <DoNotTranslate>**newrelic.yml**</DoNotTranslate>

   de su carpeta

   <DoNotTranslate>**APP_ROOT/config/**</DoNotTranslate>

   .

3. Ejecute

   <DoNotTranslate>**bundle install**</DoNotTranslate>

   .

4. Cuando finalice el proceso de desinstalación, reinicie su aplicación.

Para obtener más información sobre la instalación, consulte [New Relic for Ruby](/docs/agents/ruby-agent/getting-started/new-relic-ruby).
