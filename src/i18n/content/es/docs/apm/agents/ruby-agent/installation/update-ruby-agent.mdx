Para aprovechar al máximo las últimas características, mejoras y parches de seguridad importantes de New Relic, le recomendamos que actualice su agente Ruby a la última versión. Para obtener información adicional sobre actualizaciones de agentes específicas, consulte las [notas de la versión de Ruby agente](/docs/release-notes/agent-release-notes/ruby-release-notes).

## Verifique las versiones de su agente Ruby [#determine-versions]

Para determinar qué versiones del agente Ruby utilizan actualmente sus aplicaciones:

1. En New Relic, en el [menú de usuario](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings), seleccione

   <DNT>
     **Account settings > Connected agents**
   </DNT>

   .

2. Opcional: ordenar por

   <DNT>
     **Oldest agent version**
   </DNT>

   .

## Actualizar el agente [#Updating_the_Agent]

Para actualizar el agente Ruby:

* Usando Bundler: ejecute este comando:

  ```
  bundle update newrelic_rpm
  ```

* Sin usar Bundler: ejecute este comando:

  ```
  gem update newrelic_rpm
  ```
