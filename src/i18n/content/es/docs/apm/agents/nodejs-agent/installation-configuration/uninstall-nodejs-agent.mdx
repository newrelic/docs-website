Continúe leyendo para saber cómo desinstalar el agente New Relic Node.js. Para obtener instrucciones sobre cómo deshabilitar temporalmente el agente, consulte [Deshabilitar el agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#node).

Para desinstalar el agente New Relic Node.js:

1. Elimine `require('newrelic');` como primera línea del módulo principal de la aplicación.
2. Elimine el paquete npm con `npm uninstall newrelic --save`.
3. Elimine el archivo `newrelic.js` y desactive cualquier [variable de entorno](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration#methods-and-precedence) que se esté utilizando.
4. Cuando finalice el proceso de desinstalación, [reinicie](#restart) su aplicación.

Para obtener más información sobre la instalación, consulte [Instalación y mantenimiento de Node.js.](/docs/agents/nodejs-agent/installation-configuration/installing-maintaining-nodejs)

<Callout variant="tip">
  Es posible que también quieras [eliminar tu aplicación](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) de New Relic.
</Callout>
