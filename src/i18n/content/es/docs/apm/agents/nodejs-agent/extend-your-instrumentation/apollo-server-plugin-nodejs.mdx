El complemento New Relic Apollo Server instrumentó su aplicación Apollo Server para brindar visibilidad a su carga GraphQL. Esto le ayuda a descubrir y diagnosticar la causa de su [consulta GraphQL](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph) lenta. La versión compatible del servidor Apollo es 2.14 o posterior.

Nuestro complemento registra los tiempos generales de consulta y utiliza [rastreo distribuido](/docs/understand-dependencies/distributed-tracing/get-started/how-new-relic-distributed-tracing-works) para descubrir el problema de la ruta. Utilice esta instrumentación para ver si el problema surge de la resolución de un dato solicitado o si surge del trabajo realizado en otros servicios o base de datos.

## Compatibilidad

El complemento New Relic funciona con los siguientes módulos del servidor Apollo:

* `@apollo/server`
* `@apollo/gateway`
* `@apollo/subgraph`
* `apollo-server` (>= 2,14)
* `apollo-server-express`
* `apollo-server-hapi`
* `apollo-server-koa`
* `apollo-server-fastify`
* `apollo-server-lambda`

Es posible que funcionen otros complementos, según su implementación subyacente, pero no han sido verificados.
