¿Quiere comenzar a monitorear su aplicación Node.js con New Relic de inmediato? Utilice esta guía para encontrar la mejor ruta para comenzar a enviar los datos de su aplicación a New Relic.

¿Quiere obtener más información antes de comenzar? [Introducción a APM](/docs/apm/new-relic-apm/getting-started/introduction-apm) e [Introducción a New Relic para Node.js](/docs/apm/agents/nodejs-agent/getting-started/introduction-new-relic-nodejs) son un buen punto de partida. Si desea obtener más información sobre el proceso de instalación general, consulte nuestra [descripción general de la instalación de Node.js.](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent)

## ¿Qué quieres monitor? [#what]

Tenemos algunas rutas para instalar el agente Node.js, dependiendo de lo que estés monitoreando.

<CollapserGroup>
  <Collapser
    id="docker"
    title="Aplicación desplegar en un contenedor Docker"
  >
    Para instalar el agente en un contenedor Docker, siga [estos pasos autoguiados](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent-docker) utilizando la línea de comando.

    Este proceso lo guía para agregar el agente a `package.json` y luego solicitar New Relic en el nivel de la aplicación. Al final, agregará su clave de licencia y el nombre de la aplicación a `docker run` para que el agente se ejecute con su aplicación docker .
  </Collapser>

  <Collapser
    id="server"
    title="Aplicación desplegar en un servidor"
  >
    Tenemos algunas opciones diferentes para aplicaciones alojadas en un servidor.

    * Nuestra [instalación guiada](https://one.newrelic.com/marketplace/install-data-source?state=f535d53b-5b84-cd48-80cf-61704ad02e29) divide la instalación en pasos. Requiere cierta interacción con la línea de comando.

    * Nuestro documento [Instalar el agente Node.js](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent) utiliza la línea de comando exclusivamente. Puede ver el mismo proceso en nuestro repositorio [de agente Node.js.](https://github.com/newrelic/node-newrelic#installation)

    * Para aplicaciones que utilizan el administrador de procesos PM2, siga la [instalación guiada](https://one.newrelic.com/marketplace/install-data-source?state=ba2c3ecb-defb-e99a-c4be-47d786690d2a). Si está utilizando un administrador de procesos que no es PM2, elija la opción de instalación guiada o manual.

      Cuando esté listo para monitor su aplicación, agregue `-r newrelic` al comando de inicio de su aplicación para requerir el agente antes que otros módulos, por ejemplo, `node -r newrelic your-program.js`. Esto es especialmente importante si no tienes acceso root a tu aplicación.
  </Collapser>

  <Collapser
    id="lambda"
    title="Implementación de aplicaciones con AWS Lambda"
  >
    Para instalar un agente para su función Lambda de AWS, nuestros [pasos autoguiados](https://one.newrelic.com/marketplace/install-data-source?state=9aa3d004-6f13-1477-604e-b6a2d6bb992d) proporcionan un script de configuración que configura AWS automáticamente para usted.

    Si prefiere la instalación manual a través de la línea de comando, actualmente tenemos la opción de [Instrumentación heredada para monitoreo Lambda](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy/#node).
  </Collapser>
</CollapserGroup>
