Para aprovechar al máximo las últimas características, mejoras y parches de seguridad importantes de New Relic, actualice el agente Go de su aplicación a la última versión. Para obtener información adicional sobre actualizaciones de agentes específicas, consulte las [notas de la versión de Go agente](/docs/release-notes/agent-release-notes/go-release-notes).

## Actualice su versión de agente Go [#update-go]

Para actualizar el agente Go, siga sus procedimientos estándar para ejecutar el siguiente proceso como un comando de terminal o como un script.

1. Desde [http://github.com/newrelic/go-agent](http://github.com/newrelic/go-agent), utilice este proceso:

   ```bash
   go get -u github.com/newrelic/go-agent/v3/newrelic
   ```

2. Compile y despliegue su aplicación.
