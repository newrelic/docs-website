[Los atributos](/docs/features/agent-attributes) son pares de valores principales que contienen información para eventos de transacción, eventos de error y errores de traza. Puede ajustar los destinos del atributo predeterminado y crear un atributo personalizado para el atributo recopilado por [el agente de lenguaje Go de New Relic](/docs/agents/go-agent/get-started/new-relic-go), que incluye:

* Error de traza
* Evento de transacción
* Vistas de página

Para obtener un nivel adicional de detalle de monitoreo, [cree un atributo personalizado](/docs/agents/go-agent/go-agent-attributes#custom-attributes).

## Go agente atributo [#attributes]

El agente Go recibe el siguiente [atributo predeterminado](/docs/agents/manage-apm-agents/agent-metrics/agent-attributes) de su aplicación. Puede ajustar estas configuraciones predeterminadas y [activar o desactivar el atributo](#change-attribute-destination) para ciertos destinos.

<CollapserGroup>
  <Collapser
    id="attributeHostDisplayName"
    title="host.displayName"
  >
    El nombre del servidor host bajo el cual se ejecuta el script actual, llamado con `newrelic.AttributeHostDisplayName`.

    Configuración por defecto:

    * Traza de la transacción: Habilitado

    * Recolector de errores (errores de traza): Habilitado

    * Evento de transacción: Deshabilitado

      Ejemplo de exclusión de este atributo:

      ```go
      config.Attributes.Exclude = append(config.Attributes.Exclude, newrelic.AttributeHostDisplayName)
      ```
  </Collapser>

  <Collapser
    id="httpResponseCode"
    title="httpResponseCode"

    * Traza de la transacción: Habilitado

      <Callout variant="important">

## Atributo de tramo y segmento [#span-and-segment-attributes]

Si tienes [Go agente v2.6.0 o superior](/docs/release-notes/agent-release-notes/go-release-notes), puedes configurar atributos en tramos y segmentos. El agente Go recibe el siguiente [atributo predeterminado](/docs/agents/manage-apm-agents/agent-metrics/agent-attributes) de su aplicación. Estos atributos solo se encuentran en los segmentos span evento y traza de la transacción. Puede ajustar estas configuraciones predeterminadas y [activar o desactivar el atributo](#change-attribute-destination) para ciertos destinos.

## Cambiar destino del atributo

Utilice estas opciones para cambiar los destinos de los atributos:

<CollapserGroup>
  <Collapser
    id="turn-destinations-on-off"
    title="Activar o desactivar destinos"
  >
    Para abrir o cerrar un destino completo para el atributo, establezca el indicador `.Enabled` en `true` o `false`.

    Por ejemplo, para desactivar la recopilación de errores <InlinePopover type="apm"/>: después de la [configuración](https://github.com/newrelic/go-agent/blob/master/GUIDE.md#config-and-application), agregue:

    ```go
    config.ErrorCollector.Attributes.Enabled = false
    ```
  </Collapser>

  <Collapser
    id="turn-attributes-on-off"
    title="Activar o desactivar atributo"
  >
    Para activar o desactivar un atributo específico, utilice los métodos `.Include` o `.Exclude` .

    Por ejemplo, para deshabilitar `AttributeResponseCode`: después de la [configuración](https://github.com/newrelic/go-agent/blob/master/GUIDE.md#config-and-application), agregue:

    ```go
    config.Attributes.Exclude = append(config.Attributes.Exclude, newrelic.AttributeResponseCode)
    ```
  </Collapser>
</CollapserGroup>

## Crear atributo personalizado [#custom-attributes]

Agregue un atributo personalizado usando este método en una transacción:

```go
txn.AddAttribute("key", "value")
```

La variable `txn` es la [instrumentada para la transacción Go](/docs/agents/go-agent/get-started/instrument-go-transactions#go-txn). Por ejemplo:

```go
txn.AddAttribute("product", "widget")
txn.AddAttribute("price", 19.99)
txn.AddAttribute("importantCustomer", true)
```

Configuración predeterminada para destinos de atributo personalizado:

* Recolector de errores (errores de traza): Habilitado
* Evento de transacción: Habilitado
