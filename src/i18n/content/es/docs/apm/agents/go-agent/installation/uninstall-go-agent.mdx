Continúe leyendo para saber cómo desinstalar el agente New Relic Go. Para obtener instrucciones sobre cómo deshabilitar temporalmente el agente Go, consulte [Deshabilitar el agente](/docs/agents/manage-apm-agents/installation/disable-apm-agent#go).

Para desinstalar New Relic for Go:

1. Desinstale el paquete New Relic utilizando su método de desinstalación preferido.
2. Vuelva a compilar y reiniciar su aplicación.

Una vez que desinstales el paquete New Relic, podrás [eliminar la aplicación de tu cuenta New Relic](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui).
