## Problema

Si sus [datos de registro](/docs/agents/java-agent/troubleshooting/no-log-file-java) muestran `SEVERE:` o `ERROR:` con `NullPointerException` problemas, es posible que haya problemas que interfieran con los informes.

## Solución

Antes de comenzar con estos consejos de resolución de problemas, asegúrese de tener la [versión más reciente](/docs/agents/java-agent/installation/upgrade-java-agent) del agente de Java de New Relic.

1. Utilice búsquedas que distingan entre mayúsculas y minúsculas en su archivo de registro de agente de Java para líneas con `SEVERE:` y `ERROR: NullPointerException`.
2. Si encuentra alguna línea con estos mensajes, [recopile información sobre resolución de problemas](/docs/agents/java-agent/troubleshooting/providing-troubleshooting-information-java) para obtener soporte en New Relic.

## Causa

Agente de Java de New Relic entre versiones 3.0.0 y 3.2.3 log incorrectamente el mensaje de error. Por ejemplo:

```
ERROR: javax/servlet/ServletOutputStream is marked as a weaved class, but no methods are matched to be weaved.
```

Esto fue completamente erróneo y no es motivo de preocupación. Sin embargo, no utilice estas versiones. En su lugar, actualice a la [versión más reciente](/docs/agents/java-agent/installation/upgrade-java-agent) del agente de Java de New Relic.
