## Problema

Después de esperar unos minutos, los datos no aparecen en [la página APM<DNT>**Summary**</DNT> ](/docs/apm/applications-menu/monitoring/apm-overview-page)de su aplicación Heroku.

## Solución

1. Verifique el registro de Heroku en busca de errores con este comando del cinturón de herramientas de Heroku:

   ```
   heroku logs --tail
   ```

2. Verifique que los archivos en su banco de pruebas Heroku coincidan con las modificaciones de archivos de los procedimientos de instalación de [Java en Heroku](/docs/agents/java-agent/heroku/java-agent-heroku) o [Java con Scala en Heroku](/docs/agents/java-agent/heroku/java-agent-scala-heroku) . Utilice este comando del cinturón de herramientas de Heroku:

   ```
   heroku run bash
   ```

3. Pruebe los pasos que aparecen en [No aparecen datos (Java)](/docs/agents/java-agent/troubleshooting/no-data-appears-java).
