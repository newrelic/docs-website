## Problema

Aparece un mensaje de error al iniciar el servidor de aplicaciones para su aplicación Java. Por ejemplo:

```
Error opening zip file: newrelic.jar 
Error occurred during initialization of VM 
Agent library failed to init: instrument
```

## Solución

La ruta al archivo agente de Java `jar` debe estar completamente calificada en el conmutador JVM `-javaagent` . Si la ruta no se especifica correctamente, se generarán errores al iniciar el servidor de aplicaciones y la JVM se cerrará.

Para corregir este problema, especifique la ruta completa al archivo `newrelic.jar` en el modificador `-javaagent` . Por ejemplo, si el archivo `newrelic.jar` está en un directorio llamado `newrelic` dentro del directorio principal de Tomcat <DNT>**/home/tomcat**</DNT>, especifique el modificador de esta manera:

```
-javaagent:/home/tomcat/newrelic/newrelic.jar
```
