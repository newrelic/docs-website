Si los procedimientos de resolución de problemas del agente de Java (como [No aparecen datos](/docs/apm/agents/java-agent/troubleshooting/no-data-appears-java) y otros documentos en esa categoría) no han resuelto su problema, use el [archivo de registro](#logfiles) para solucionar el problema. Si necesita más ayuda, comuníquese con [support.newrelic.com](https://support.newrelic.com).
