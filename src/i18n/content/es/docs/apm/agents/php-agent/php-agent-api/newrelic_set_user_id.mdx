## Sintaxis

```php
newrelic_set_user_id(string $user_id)
```

Asocie una ID de usuario única con la transacción actual.

## Requisitos

Versión del agente 10.12 o superior.

## Descripción

Esta API permite que una aplicación asigne una ID de usuario única a la transacción actual.

Esto permitirá al usuario obtener información valiosa sobre el usuario final individual, como el impacto de un evento de error.

## Valores de retorno

Esta función devolverá `true` si el ID de usuario se agregó correctamente; en caso contrario, `false`.

## Ejemplos

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $user_id = "YOUR UNIQUE USER ID";
        newrelic_set_user_id($user_id);
    }
}
```
