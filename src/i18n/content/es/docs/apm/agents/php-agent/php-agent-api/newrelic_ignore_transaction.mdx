## Sintaxis

```php
newrelic_ignore_transaction()
```

No instrumente la transacción actual.

## Requisitos

Compatible con todas las versiones de agente.

## Descripción

No genere datos para esta transacción. Esto es útil cuando tiene transacciones que son particularmente lentas por razones conocidas y no desea que generen con frecuencia una traza de la transacción o sesguen los promedios de su sitio.

## Parámetros

Esta llamada no acepta ningún parámetro.

## Ejemplos

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_ignore_transaction();
    }
    ...
}
```
