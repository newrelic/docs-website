## Sintaxis

```php
newrelic_get_browser_timing_footer([bool $include_tags])
```

Devuelve un fragmento <InlinePopover type="browser"/>para insertar al final de la salida HTML.

## Requisitos

Compatible con todas las versiones de agente.

## Descripción

Devuelve una cadena de JavaScript para inyectar al final de la salida HTML. Utilice esta llamada con [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header) para agregar manualmente monitoreo de navegador a una página web. Si es posible, New Relic recomienda habilitar el monitoreo del navegador a través de la UI de New Relic o copiar y pegar el fragmento de JavaScript. Para obtener instrucciones sobre el uso de estas opciones, consulte [monitoreo del navegador y el agente PHP](/docs/agents/php-agent/features/page-load-timing-php).

<Callout variant="tip">
  Compare [`newrelic_disable_autorum()`](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum), que <DNT>**disables**</DNT> es el script de monitoreo del navegador en una página.
</Callout>

## Valores de retorno

Devuelve la cadena de pie de página JavaScript del monitoreo del navegador.

## Ejemplos

### Instrumento una página web [#webpage]

```php
function example(){
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_header = newrelic_get_browser_timing_header();
    }
    
    // Output to page
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_footer = newrelic_get_browser_timing_footer();
    }
}
```
