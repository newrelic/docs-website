## Sintaxis

```php
newrelic_disable_autorum()
```

Deshabilite la inyección automática del fragmento <InlinePopover type="browser"/>en páginas específicas.

## Requisitos

Versión del agente [2.6.5.41](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-26541) o superior.

## Descripción

Esta llamada deshabilita la inyección automática del agente [de monitoreo de navegador](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) para la transacción actual. Llame lo antes posible. Puede utilizar esta llamada para eliminar JavaScript si la [inserción está causando problemas](/docs/browser/new-relic-browser/troubleshooting/browser-javascript-injection-causes-problems-page) o si está entregando páginas a servicios de terceros que no permiten JavaScript (por ejemplo, [las páginas móviles aceleradas de Google](/docs/browser/new-relic-browser/troubleshooting/google-amp-validator-fails-due-3rd-party-script)).

<Callout variant="tip">
  Para <DNT>**enable**</DNT> monitorear el navegador con una llamada API, consulte:

  * [`newrelic_get_browser_timing_footer()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer)
  * [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header)
</Callout>

## Parámetros

Esta llamada no acepta ningún parámetro.

## Valores de retorno

Devuelve `true` si se llama dentro de una transacción New Relic. De lo contrario, devuelve `null` si está fuera de una transacción (por ejemplo, si se ha llamado a `newrelic_end_transaction()` ).

## Ejemplos

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_disable_autorum();
    }
    // ...
}
```
