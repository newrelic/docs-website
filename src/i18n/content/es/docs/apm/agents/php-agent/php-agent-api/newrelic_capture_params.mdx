## Sintaxis

```php
newrelic_capture_params([bool $enable_flag])
```

Activa o desactiva la captura del parámetro URL.

## Requisitos

Compatible con todas las versiones de agente.

## Descripción

Habilita o deshabilita la captura del parámetro URL (todo lo que sigue al `?` en la URL). Esto anula la configuración del archivo de configuración `newrelic.capture_params` .

<Callout variant="important">
  Si pasa información confidencial directamente en la URL, manténgala desactivada.
</Callout>

## Ejemplos

### Habilitar captura de parámetro de URL [#enable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params();
    }
    //...
}
```

### Deshabilitar captura de parámetro de URL [#disable-params]

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_capture_params(false);
    }
    //...
}
```
