[Los atributos](/docs/features/agent-attributes) son pares de valores principales que contienen información que determina las propiedades de un evento o transacción. Estos pares de valores principales pueden ayudarlo a obtener mayor información valiosa en su aplicación y anotar los datos en New Relic. Tanto el atributo predeterminado como [el personalizado](/docs/agents/manage-apm-agents/agent-data/collect-custom-attributes) son visibles en APM traza de la transacción, rastreo distribuido y análisis de errores; y [APM](/docs/insights/explore-data/attributes/apm-default-attributes-insights) y [evento de navegador](/docs/insights/explore-data/attributes/browser-default-attributes-insights#browser-attributes-table).

Puedes personalizar exactamente qué atributo se enviará a cada uno de estos destinos. Continúe leyendo para conocer el atributo del agente PHP, cómo habilitarlo o deshabilitarlo y las reglas que sigue el agente para determinar qué atributo incluir o excluir para un [destino](/docs/features/agent-attributes#destinations).

Consulte [Configuración del agente PHP (`newrelic.ini`)](/docs/php/php-agent-phpini-settings) y [Habilitación y deshabilitación de atributos](/docs/agents/php-agent/attributes/enabling-and-disabling-attributes) para obtener más información.

## Cambiar dónde se envían los atributos [#change-attribute-destination]

<Callout variant="tip">
  Todos los destinos están abiertos a la colección de atributos de forma predeterminada en el agente PHP, excepto `newrelic.browser_monitoring.attributes.enabled`.
</Callout>

Para cambiar qué atributo se envía a [los destinos](/docs/apm/other-features/attributes/agent-attributes#destinations) de New Relic:

1. Abra o cierre cualquier destino de la colección de atributos cambiando la [propiedad de destino`.enabled` ](/docs/agents/php-agent/attributes/enabling-disabling-attributes#destination-on-off).

2. Cambie la [configuración de atributo predeterminada](#attributes) agregando el nombre del atributo a las [propiedades`.include` o `.exclude` ](/docs/agents/php-agent/attributes/enabling-disabling-attributes#destination-attribute-specific)del destino.

   Consulte [las reglas de atributos](/docs/agents/php-agent/attributes/enabling-disabling-attributes#attruls) para saber qué configuraciones se reemplazan entre sí.

## Actualizando el agente PHP [#upgrading]

Al actualizar al agente PHP <DNT>**4.9**</DNT> o superior, actualice su archivo de configuración. Para obtener más información sobre propiedades obsoletas, consulte [Habilitación y deshabilitación de atributos](/docs/php/php-agent-enabling-and-disabling-attributes#deprecated).
