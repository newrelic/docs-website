## Problema

Continúa teniendo problemas con la instalación del agente PHP de New Relic.

## Solución

Si continúa teniendo problemas con la instalación de PHP de New Relic después de seguir los [consejos de resolución de problemas](/docs/apm/agents/php-agent/troubleshooting/no-data-appears-php), obtenga soporte en [support.newrelic.com](https://support.newrelic.com).

Cuando trabaje con un ingeniero de soporte técnico de New Relic, proporcione la siguiente información:

* ¿Hasta dónde has avanzado en el proceso de instalación?

* ¿Qué método de instalación estás utilizando?

* Una copia de su archivo `newrelic.cfg` (si corresponde)

* Un enlace a su página <DNT>**phpinfo**</DNT> o el resultado de <DNT>**phpinfo**</DNT> guardado como un archivo HTML. Para [crear <DNT>**phpinfo**</DNT>](/docs/php/php-kb-103), cree un .php página web que contiene sólo:

  ```php
  <?php phpinfo(); ?>
  ```

* Su [agente y daemon archivo de registro](/docs/php/php-kb-105)
