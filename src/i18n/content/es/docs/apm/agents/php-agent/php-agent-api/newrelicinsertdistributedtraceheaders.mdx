## Sintaxis

```php
newrelic_insert_distributed_trace_headers (array $headers)
```

Inserta encabezados W3C Trace Context y encabezados distribuidos de rastreo New Relic en una matriz de encabezados salientes.

## Requisitos

Requiere [la versión 9.8 o superior](/docs/release-notes/agent-release-notes/php-release-notes) del agente PHP.

## Descripción

Utilice `newrelic_insert_distributed_trace_headers` para agregar manualmente encabezados rastreo distribuidos y una matriz de encabezados salientes.

Cuando rastreo distribuido está habilitado, `newrelic_insert_distributed_trace_headers` siempre insertará encabezados W3C Trace Context . También, de forma predeterminada, inserta el encabezado distribuido de rastreo de New Relic, pero esto se puede desactivar mediante la configuración `newrelic.distributed_tracing_exclude_newrelic_header` INI.

## Valores de retorno

`true` si algún encabezado se insertó correctamente en la matriz proporcionada; de lo contrario, devuelve `false`
