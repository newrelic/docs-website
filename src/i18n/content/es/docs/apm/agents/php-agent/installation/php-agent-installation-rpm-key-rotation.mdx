A partir de marzo de 2026, por mejores prácticas y seguridad, los nuevos lanzamientos de los paquetes RPM del agente PHP de New Relic se firmarán con una nueva clave. Para continuar con el funcionamiento ininterrumpido del agente PHP, antes de instalar o actualizar el agente PHP de New Relic desde paquetes RPM, se debe ejecutar el siguiente comando para aceptar e importar la nueva clave pública.

`rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public`

## Verificar la clave

Después de importar la clave, verifique que sea de confianza ejecutando el siguiente comando:

`rpm -qa | grep gpg-pubkey-bd2e199c`

Si la clave se importa y es de confianza, el comando devuelve la siguiente salida:

`gpg-pubkey-bd2e199c-6970e650`

## ¿Qué sucede si no importa la nueva clave antes de marzo?

Si la nueva clave no se agrega al sistema, las instalaciones y actualizaciones de las nuevas versiones del Agente PHP fallarán, con un mensaje de error como `Error: GPG check FAILED`. Cualquier instalación fijada en versiones anteriores del agente seguirá funcionando como antes (aunque en esos casos se recomienda importar la nueva clave de todos modos).
