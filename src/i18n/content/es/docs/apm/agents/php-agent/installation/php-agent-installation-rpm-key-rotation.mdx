En marzo de 2026, por mejores prácticas y seguridad, las nuevas versiones de los paquetes RPM del agente PHP de New Relic se firmaron con una nueva clave. Para continuar el funcionamiento del agente de PHP sin interrupciones, antes de instalar o actualizar el agente de PHP de New Relic desde paquetes RPM, se debe ejecutar el siguiente comando para aceptar e importar la nueva clave pública.

`rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public`

## Verificar la clave

Después de importar la clave, verifique que sea de confianza ejecutando el siguiente comando:

`rpm -qa | grep gpg-pubkey-bd2e199c`

Si la clave se importa y es de confianza, el comando devuelve la siguiente salida:

`gpg-pubkey-bd2e199c-6970e650`

## ¿Qué sucede si no ha importado la nueva clave?

Si la nueva clave no se agrega a su sistema, las instalaciones y actualizaciones de nuevas versiones del Agente de PHP fallarán, con un mensaje de error como `Error: GPG check FAILED`. Cualquier instalación fijada a versiones anteriores del agente seguirá funcionando como antes (aunque en tales casos se recomienda importar la nueva clave de todos modos).
