New Relic agregó soporte para la biblioteca Predis en [la versión 4.22](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-422099) del agente PHP.

El agente instrumenta llamadas estándar a un objeto `Predis\Client` o a un pipeline. Todos los comandos enviados a través de una canalización se agruparán y tratarán como una métrica de comando única denominada `pipeline` en lugar de comandos individuales.

El agente PHP no admite llamadas a los métodos `executeRaw` o `transaction` , PubSub, monitor o similares. Estos no estarán instrumentados.
