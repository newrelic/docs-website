## Problema

Ha instalado el agente PHP y daemon de New Relic, pero el daemon no parece estar funcionando.

## Solución

Verifique que el daemon se esté ejecutando correctamente:

1. Ejecute este comando de shell para enumerar los procesos en ejecución que coincidan con "newrelic-daemon":

   ```bash
   ps -ef | grep newrelic-daemon
   ```

2. Verifique que el daemon se esté ejecutando verificando el resultado del comando anterior para ver dos procesos daemon en ejecución que se ven así. Si no aparecen en la lista, continúe con el siguiente paso.

   ```console
   newrelic     1368     1  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   newrelic     1370  1368  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   ```

3. Si no aparece ningún proceso, su daemon no se está ejecutando y debe daemon según el modo de inicio en el que se encuentra:

   * [Inicio en modo agente](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-external) (predeterminado)
   * [Inicio en modo externo](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-autostart) (avanzado)
