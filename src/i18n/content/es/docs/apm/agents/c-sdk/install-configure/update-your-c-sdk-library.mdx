## Actualice su biblioteca de códigos C SDK [#update-production]

Para actualizar la biblioteca de códigos de su aplicación a la última versión del C SDK de New Relic:

1. Revise los cambios de la biblioteca C SDK para verificar que su código sea compatible.
2. Actualice manualmente el código de su aplicación según corresponda.
3. Compile el código C SDK con el comando [`make`](https://github.com/newrelic/c-sdk#building) para crear la última versión de `libnewrelic.a` y vincularlo a su aplicación o servicio.
4. Si corresponde, vuelva a implementar su aplicación en su entorno de prueba.
5. Genere tráfico para su aplicación, luego verifique su archivo `c_sdk.log` y la salida `newrelic-daemon.log` para detectar errores.
6. Vuelva a implementar su aplicación en su entorno de producción.
7. Espere unos minutos hasta que su aplicación envíe datos a New Relic. Luego, verifique el rendimiento de su aplicación en [New Relic](/docs/apm/applications-menu/monitoring/apm-overview-page).

Si no aparecen datos en unos minutos, sigue los [consejos de resolución de problemas](/docs/apm/agents/c-sdk/troubleshooting/no-data-appears-c-sdk).
