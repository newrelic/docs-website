## Problema

Está utilizando el agente Pro + SPA junto con el framework Next.JS y los enlaces en su página no funcionan.

Somos conscientes de una incompatibilidad entre el agente del navegador Pro+SPA y el enrutador de la aplicación Next.JS desde la versión 13.3.2-canary.2.

Este problema puede limitarse al uso de `prefetch` en `Link` componentes y/o al uso de elementos HTML estándar que contienen una devolución de llamada `onClick` que invoca la función `router.push()` .

## Solución

Este problema se resolvió con la versión Next.JS [v13.4.20-canary.5](https://github.com/vercel/next.js/releases/tag/v13.4.20-canary.5). Actualice a la versión 13.4.20-canary.5 o posterior del framework Next.JS.

## Causa

El agente del navegador opera envolviendo muchos browser globales, incluido Promise. La modificación de Promise global parece dar como resultado un bucle sin fin dentro del enrutador de la aplicación Next.JS cuando se hace clic en un enlace.
