New Relic <InlinePopover type="browser"/>ofrece monitoreo de aplicaciones de una sola página (SPA) para brindarle una visibilidad más profunda e información procesable valiosa sobre la interacción real del usuario con aplicaciones de una sola página o cualquier aplicación que utilice solicitudes AJAX.

Además de monitorear los cambios de ruta automáticamente, nuestra API SPA le permite monitor prácticamente cualquier cosa que se ejecute dentro del browser. Los desarrolladores y sus equipos pueden utilizar la API para:

* Cree aplicaciones más rápidas, con mayor capacidad de respuesta y altamente interactivas.
* Monitor el rendimiento y desempeño que el usuario real está experimentando.
* Solucionar y resolver problemas dentro del contexto de la carga de la página.
* [Consulta tus datos](/docs/using-new-relic/data/understand-data/query-new-relic-data) para ayudarte en las decisiones comerciales.
* Lleve mejores aplicaciones al mercado más rápidamente.

## Habilitar el monitoreo de SPA [#enable]

El monitoreo de SPA está habilitado de forma predeterminada para la nueva instalación del agente del navegador. La versión del agente habilitada para SPA brinda acceso a otras poderosas características de New Relic, como el rastreo distribuido. Para obtener más información, consulte [Habilitar el monitoreo del navegador](/docs/browser/new-relic-browser/installation/update-browser-agent/#agent-types).

## Compatibilidad [#compatibility]

Para obtener información de compatibilidad sobre las características relacionadas con SPA, consulte [Requisitos de SPA](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring#spa).

El monitor de datos SPA mediante monitoreo de navegador incluye:

* Datos de rendimiento y rendimiento para cargas de páginas y cambios de ruta.
* Datos de solicitud AJAX
* Actividad de JavaScript, tanto síncrona como asincrónica
* Actualizaciones dinámicas de páginas, monitorización mediante [SPA API](/docs/browser/new-relic-browser/browser-agent-spa-api)

Con estos datos, obtendrá una comprensión clara de cómo su experiencia del usuario carga la página de su aplicación y cambia la ruta, y podrá resolver el cuello de botella y solucionar errores. Para obtener más información sobre cómo New Relic maneja los datos de SPA, consulte [Comprender la recopilación de datos de SPA](/docs/browser/single-page-app-monitoring/use-spa-data/understand-spa-data-collection).
