El monitoreo de aplicación de página única (SPA) del browser incluye una API para agregar monitoreo personalizado de la interacción específica browser . Esto es útil para monitorear interacciones que New Relic no registra automáticamente porque no resultan en cambios de ruta, como un widget actualizado dinámicamente. La API SPA también le permite desactivar el monitoreo predeterminado para interacciones que no considera lo suficientemente importantes como para monitor.

Para obtener más información sobre la API de SPA, incluida la API de llamada específica, consulte el [agente del navegador y la API de SPA](/docs/browser/new-relic-browser/guides/guide-using-browser-spa-apis/).
