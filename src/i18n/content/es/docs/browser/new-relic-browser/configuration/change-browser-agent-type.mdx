El tipo de agente del navegador no tiene impacto en su facturación. Es simplemente una forma de cambiar a un conjunto de características más mínimo, si lo desea. El agente del navegador predeterminado instalado viene con todas las características <InlinePopover type="browser"/>habilitadas.

Para cambiar el tipo de edición de su browser , vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > Settings > Application settings**</DNT>.

Aquí hay algunas comparaciones entre los diferentes tipos de agente del navegador:

## Tipos de agente del navegador [#agent-types]

Disponemos de tres tipos de agente del navegador: Lite, Pro y Pro+SPA.
