Siga el procedimiento apropiado para cambiar el nombre de la aplicación de su navegador según la [opción desplegable](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser) que utilizó para agregarla al navegador.

## Cambiar el nombre del monitor de aplicaciones por APM [#apm-app]

Si agregó una aplicación al browser que APM ya está monitoreando, el nombre de la aplicación que aparece en browser es el mismo que el nombre de la aplicación en laUI <InlinePopover type="apm"/>. Para cambiar el nombre de la aplicación, utilice cualquiera de estas opciones:

Si [una o más aplicaciones comparten el mismo nombre](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), sus datos se agregarán en la UI.

## Cambiar el nombre de la aplicación mediante un fragmento de JavaScript [#javascript]

Esta información se aplica si insertó el fragmento de JavaScript del agente del navegador en su:

* Monitor de aplicación por APM que se implementó mediante el [método de copiar/pegar](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#copy-paste-app)
* Aplicación de una sola página ([SPA](/docs/browser/single-page-app-monitoring/get-started/add-apps-single-page-app-monitoring))
* Cualquier aplicación browser independiente que APM no supervise también

No puede editar directamente el fragmento de JavaScript en la aplicación existente. En su lugar, para usar tu aplicación con un nombre diferente:

1. Siga los procedimientos estándar para [crear una nueva aplicación utilizando el método de copiar y pegar](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#copy-paste-app). Proporcione el nuevo nombre y genere el fragmento de JavaScript.
2. Copie el nuevo fragmento de JavaScript.
3. En su página web, reemplace el JavaScript de su aplicación original con el JavaScript generado para el nombre de la nueva aplicación.
4. Si su aplicación independiente tiene [varios nombres](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), [copie los ID de la aplicación personalizada](/docs/browser/new-relic-browser/installation-configuration/copy-browser-monitoring-license-key-app-id) en su script de configuración a su nueva aplicación.

Según el [programa de retención de datos](/docs/accounts-partnerships/accounts/account-billing-usage/data-retention), New Relic conservará los datos históricos recopilados con el nombre anterior y recopilará y agregará datos con el nuevo nombre.

## Cambiar el nombre de la aplicación a través de la API REST de New Relic (v2) [#rest-api]

Esta información se aplica solo a las aplicaciones browser que también son monitoreadas por APM. No puede cambiar el nombre de la aplicación a través de la API REST si utilizó el método copiar/pegar para insertar el fragmento de JavaScript del browser en su aplicación.

Siga los procedimientos estándar para [cambiar el alias de la aplicación APM mediante REST API](/docs/apis/rest-api-v2/application-examples-v2/change-alias-your-application-v2) o [API Explorer](https://rpm.newrelic.com/api/explore/applications/update). New Relic cambia automáticamente el nombre para mostrar de la aplicación en todas partes de la UI de New Relic, incluido browser.
