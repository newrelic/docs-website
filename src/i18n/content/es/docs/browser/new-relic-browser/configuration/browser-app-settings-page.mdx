Si necesita personalizar cómo el agente del navegador recopila datos, tiene dos métodos: configurar la UI o utilizar API para personalizar la instrumentación del agente.

## Configuración UI [#UI]

En la página <DNT>**Application settings**</DNT> , puede realizar estas configuraciones en su configuración <InlinePopover type="browser"/>:

* [Código de agente proxy y datos analíticos](/docs/browser/new-relic-browser/configuration/proxy-agent-requests/)
* [Configurar alerta](/docs/browser/new-relic-browser/configuration/view-browser-apps-alert-information/)
* [Configurar APDEX y ajustes geográficos](/docs/browser/new-relic-browser/configuration/browser-settings-ui-options-apdex-geography/#apdex-settings)
* [Filtrar dominio y subdominio](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains/)
* [Filtrar solicitudes AJAX](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events/)
* [Agrupar datos por patrones de URL](/docs/browser/new-relic-browser/configuration/group-browser-metrics-urls/)
* [Ofuscar datos](/docs/browser/new-relic-browser/configuration/obfuscate-browser-agent-data/)
* [Cambiar el nombre de las aplicaciones browser](/docs/browser/new-relic-browser/configuration/rename-browser-apps/)
* [Deshabilitar el monitoreo del navegador](/docs/browser/new-relic-browser/installation/disable-browser-monitoring/)
* [Eliminar una aplicación browser](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/#remove)

Para llegar a la página <DNT>**Application settings**</DNT> :

1. Vaya a

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**
   </DNT>

   y seleccione una aplicación.

2. En la navegación izquierda, desplácese hacia abajo hasta

   <DNT>
     **Settings**
   </DNT>

   y haga clic en

   <DNT>
     **Application settings**
   </DNT>

   .

## API browser [#apis]

También puede personalizar la instrumentación del agente de su browser para capturar trazas y eventos personalizados, informar errores específicos adaptados a sus necesidades y ajustar el tiempo de recolección de datos para un rendimiento óptimo. Consulte [Uso de las API del navegador](/docs/browser/new-relic-browser/browser-apis/using-browser-apis/) para comenzar.
