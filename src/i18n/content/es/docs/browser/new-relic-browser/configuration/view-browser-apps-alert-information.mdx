[Alertas](/docs/alerts/new-relic-alerts-beta/getting-started/new-relic-alerts) proporciona una herramienta de alertas única y coordinada en todos los productos de New Relic. Esto le permite administrar [políticas de alertas](/docs/alerts/overview#alert-policy) y [condiciones](/docs/alerts/overview#alert-condition) que se enfocan en las métricas de las [entidades](/docs/alerts/overview#alert-entity) (como aplicaciones de navegador o transacciones clave) que más le importan.

Después de que [configure una o más políticas de alertas](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) y [defina condiciones de alerta](/docs/alerts/create-alert/create-alert-condition/alert-conditions) para sus [entidades de producto (objetivos)](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/select-product-targets-alert-condition), puede ver esa información en la UI de alertas. También puede ver información de la política de alertas en la interfaz de usuario para productos individuales.

## Ver políticas y condiciones desde browser [#policy]

Para ver la política de alertas y la información de condiciones para una aplicación browser específica o una transacción clave:

1. Aplicación browser : vaya a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; Settings &gt; Application settings**</DNT>.

   O

   Clave de transacción: Ir a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Key transactions &gt; (select a key transaction) &gt; Settings &gt; Alert conditions**</DNT>.

2. Desde la página <DNT>**Alert conditions**</DNT> , utilice las herramientas disponibles para buscar, ordenar, ver o actualizar la condición de alerta y sus políticas asociadas.

## Ver eventos y actividades desde browser [#activity]

Para ver información resumida sobre [eventos](/docs/alerts/overview#event) y otra actividad:

1. Aplicación browser : Ir a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT>

   O

   Clave de transacción: Ir a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Key transactions**</DNT>.

2. Desde el índice, pase el mouse sobre el [estado de salud codificado por colores](/docs/alerts/overview#health-status) de la entidad, seleccione un enlace de la lista de <DNT>**Browser activity**</DNT> o seleccione una aplicación de navegador específica o una transacción clave para ver detalles adicionales.

Por ejemplo, si ocurre un evento de alerta Crítica:

* El indicador de estado de salud se vuelve rojo en el índice browser y en la aplicación del browser seleccionada.
* El color de fondo de varios gráficos cambia a rosa.
* La lista <DNT>**Browser activity**</DNT> en el índice de navegadores muestra eventos de alerta de Advertencia (amarillo) y Crítico (rojo) según corresponda. Para obtener más información sobre el evento de alerta en las alertas, pase el mouse sobre cualquier área rosa en un gráfico o selecciónela. Para obtener más información y ejemplos, consulta [Ver eventos de sus productos](/docs/alerts/new-relic-alerts-beta/reviewing-events/view-events-their-products).

## Ver eventos de alerta desde el navegador [#alertevents]

Si una condición de alerta tiene [umbrales](/docs/alerts/overview#alert-threshold) configurados para eventos de alerta de Advertencia (amarillo) o Crítico (rojo), el [estado de salud](/docs/alerts/overview#health-status) codificado por colores para una entidad de producto cambiará para indicar un evento de alerta. Puede ver los eventos de alerta directamente desde la página de la aplicación de navegador o de la transacción clave en el navegador:

1. Aplicación de navegador: Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select a browser app)**</DNT> y revise los eventos de alerta en el panel derecho.

   Clave de transacción: Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Key transactions &gt; (select a key transaction) &gt; Issues &amp; Activity**</DNT> y haga clic en la pestaña <DNT>**Alert events**</DNT> .

2. Desde la página <DNT>**Alert events**</DNT>, utilice las herramientas disponibles para buscar, ordenar o ver eventos de alerta de la aplicación de navegador o transacción clave seleccionada.

3. Opcional: Para ver o actualizar las políticas de alertas, condiciones o umbrales que activaron los eventos de alerta, seleccione el enlace correspondiente.

<Callout variant="tip">
  También puede ver los eventos de alerta de Advertencia y Críticos desde la UI de alertas para [eventos de alerta específicos](/docs/alerts/alert-event-management/view-alert-event-details) o para [eventos de alerta y eventos en todos los productos](/docs/alerts/alert-event-management/view-alert-event-details). Alertas envía [notificaciones de alerta](/docs/alerts/get-notified/destinations) solo para eventos de alerta Críticos (rojos).
</Callout>
