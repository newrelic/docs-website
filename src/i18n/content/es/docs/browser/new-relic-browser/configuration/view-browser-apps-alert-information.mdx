Buena ayuda <InlinePopover type="alerts"/>para notificar a personas o equipos sobre cambios en sus sistemas. [Alerts](/docs/alerts/new-relic-alerts-beta/getting-started/new-relic-alerts) proporciona una herramienta de alerta única y coordinada para todos los productos New Relic. Esto le permite administrar [la política de alertas](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#alert-policy) y [condiciones](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#alert-condition) que se centran en la métrica de [la entidad](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#alert-entity) (como aplicaciones browser o clave de transacción) que más le interesa.

Después de [configurar una o más políticas de alertas](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/name-or-rename-alert-policy) y [definir la condición de alerta](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-alert-conditions) para [la entidad (objetivo) de su producto](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/select-product-targets-alert-condition), puede ver esa información en la UI de Alertas e IA. También puede ver información de la política de alertas en la UI para productos individuales.

## Ver políticas y condiciones desde browser [#policy]

Para ver la política de alertas y la información de condiciones para una aplicación browser específica o una transacción clave:

1. Aplicación browser : vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > Settings > Application settings**</DNT>.

   O

   Clave de transacción: Ir a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Key transactions > (select a key transaction) > Settings > Alert conditions**</DNT>.

2. Desde la página <DNT>**Alert conditions**</DNT> , utilice las herramientas disponibles para buscar, ordenar, ver o actualizar la condición de alerta y sus políticas asociadas.

## Ver eventos y actividades desde browser [#activity]

Para ver información resumida sobre [eventos](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#event) y otras actividades:

1. Aplicación browser : Ir a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser**</DNT>

   O

   Clave de transacción: Ir a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Key transactions**</DNT>.

2. Desde el índice, pase el mouse sobre el [estado de salud codificado por colores](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#health-status) de la entidad, seleccione un enlace de la lista <DNT>**Browser activity**</DNT> o seleccione una aplicación browser específica o una transacción clave para ver detalles adicionales.

Por ejemplo, si ocurre un incidente de alerta crítica:

* El indicador de estado de salud se vuelve rojo en el índice browser y en la aplicación del browser seleccionada.

* El color de fondo de varios gráficos cambia a rosa.

* La lista

  <DNT>
    **Browser activity**
  </DNT>

  en el índice del browser muestra un incidente de advertencia (amarillo) y crítico (rojo), según corresponda.

Para obtener más información sobre el incidente en alerta, pase el mouse o seleccione cualquier área rosa en un gráfico. Para obtener más información y ejemplos, consulte [Ver evento de sus productos](/docs/alerts/new-relic-alerts-beta/reviewing-events/view-events-their-products).

## Ver incidente de alerta desde browser [#incidents]

Si una condición de alerta tiene [un umbral](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#alert-threshold) configurado para incidentes de Advertencia (amarillo) o Crítico (rojo), el [estado de salud](/docs/alerts/new-relic-alerts-beta/getting-started/alerts-glossary#health-status) codificado por colores para una entidad de producto cambiará para indicar un incidente. Puede ver el incidente directamente desde la página de la aplicación browser o de la transacción clave en browser:

1. Aplicación browser : Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser > (select a browser app)**</DNT> y verifique el incidente en el panel derecho.

   Clave de transacción: Vaya a <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Key transactions > (select a key transaction) > Issues & Activity**</DNT> y haga clic en la pestaña <DNT>**Incidents**</DNT> .

2. Desde la página <DNT>**Incidents**</DNT>, utilice las herramientas disponibles para buscar, ordenar o ver incidentes para la aplicación browser o la transacción clave seleccionada.

3. Opcional: Para ver o actualizar la política de alertas, condiciones o umbral que desencadenó el incidente, seleccione el enlace correspondiente.

<Callout variant="tip">
  También puede ver las advertencias y los incidentes críticos desde la UI de alerta e IA para [incidentes específicos](/docs/alerts/new-relic-alerts-beta/reviewing-alert-incidents/explore-incident-history) o para [incidentes y eventos en todos los productos](/docs/alerts/new-relic-alerts-beta/reviewing-events/review-events-across-products). Alertas envía [notificación de alerta](/docs/alerts/new-relic-alerts-beta/managing-notification-channels/notification-channels-controlling-where-send-alerts) solo para incidentes críticos (rojo).
</Callout>
