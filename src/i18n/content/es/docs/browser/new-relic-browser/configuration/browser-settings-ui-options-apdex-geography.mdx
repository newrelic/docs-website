Como parte de los[requisitos de configuración](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements) <InlinePopover type="browser"/>, debes [instalar el agente del navegador](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent). Luego puede usar la página <DoNotTranslate>**Applications settings**</DoNotTranslate> de su aplicación para actualizar las opciones de monitoreo del navegador para:

* Nivel de suscripción browser

* browser [Valores Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)

* Ciudades, regiones o países que se resaltarán en la [página](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)

  <DoNotTranslate>[**Geography**](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)</DoNotTranslate>

## Seleccione Apdex y configuración geográfica [#apdex-settings]

Para aplicaciones que también son monitoreadas por APM, puede establecer [valores de Apdex T](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) separados para browser (usuario final) y para [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex). También puede seleccionar ciudades, regiones o países para resaltarlos en la página <DoNotTranslate>**Geography**</DoNotTranslate> .

1. Vaya a

   <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >**</DoNotTranslate>

   (seleccione una aplicación)

   <DoNotTranslate>**> Settings > Application settings**</DoNotTranslate>

   .

2. Desde

   <DoNotTranslate>**Application settings**</DoNotTranslate>

   , seleccione un valor Apdex T específicamente para monitoreo del navegador o deje el valor Apdex predeterminado.

3. Seleccione hasta cinco países o regiones para el monitoreo del navegador y resaltarlos en la página

   <DoNotTranslate>**Geography**</DoNotTranslate>

   . Si está informando datos para [varios nombres de aplicaciones](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app), seleccione los mismos países en las aplicaciones principal y secundaria.

4. Seleccione

   <DoNotTranslate>**Save application settings**</DoNotTranslate>

   .

Cada vez que cambia la configuración de Apdex, aparece una barra vertical negra en el gráfico <DoNotTranslate>**Page load timing**</DoNotTranslate> en la página <DoNotTranslate>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DoNotTranslate> de la aplicación de su browser .

## Ver Apdex del usuario final [#view-browser-apdexs]

Para ver su puntuación Apdex:

* Vaya a <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DoNotTranslate>.

  O

* Vaya a <DoNotTranslate>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DoNotTranslate>.
