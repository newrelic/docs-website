Las API browser le permiten personalizar y ampliar su <InlinePopover type="browser"/>. Para comenzar a utilizar las API, asegúrese de que el [agente del navegador esté presente en cada página que desee monitor](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent).

A continuación se muestran casos de uso para cada método API browser .

      </td>

      <td>

      </td>
    </tr>

    <tr>
      <td>

        [`newrelic.addToTrace()`](/docs/browser/new-relic-browser/browser-apis/addtotrace)
