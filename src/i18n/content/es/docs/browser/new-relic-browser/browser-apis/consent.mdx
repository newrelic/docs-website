## Sintaxis

```js
newrelic.consent(accept?: boolean)
```

Permite o rechaza el consentimiento para que el agente del browser envíe datos a través de la red.

## Requisitos

* Agente Browser Lite, Pro o Pro+SPA (v1.303.0 o superior)

## Descripción

El agente del browser tiene una funcionalidad de modo de consentimiento opcional. Cuando el modo de consentimiento está habilitado a través de la configuración de la aplicación o con NerdGraph, el usuario puede llamar al método `newrelic.consent(...)` para permitir o denegar el consentimiento sobre si se envían o no datos a través de la red.

Para obtener más información sobre la configuración y el uso, consulte nuestra [documentación sobre el modo de consentimiento](/docs/browser/new-relic-browser/configuration/consent-mode).

## Casos de uso

* Quiere esperar para recopilar datos hasta que un usuario haya dado su consentimiento con un modal en su sitio

## Ejemplos

### El usuario acepta dar su consentimiento al agente del browser para recopilar datos.

```js
newrelic.consent()
```

### Un usuario rechaza el consentimiento para que el agente del browser recopile datos.

```js
newrelic.consent(false)
```
