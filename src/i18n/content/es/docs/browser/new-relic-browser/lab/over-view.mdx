Sois los desarrolladores de Relicstaurants, una aplicación [React](https://reactjs.org//) que permite al usuario pedir comida de sus restaurantes favoritos. Su aplicación es muy popular entre el usuario y desea asegurarse de que su usuario siempre tenga la mejor experiencia con su sitio. Para lograr este objetivo, utiliza New Relic <InlinePopover type="browser"/>para lograr un mejor rendimiento de su sitio y solucionar cualquier problema según sea necesario.

En esta práctica de laboratorio, instrumentará su aplicación web con el monitoreo de navegador New Relic, monitor su métrica y solucionará problemas de interfaz.

<SideBySide>
  <Side>
    ## Objetivos

    * Inicie una aplicación de demostración de React.
    * Instale el agente New Relic Browser .
    * Solucionar errores de front-end.
    * Sube el mapa fuente a New Relic.
    * Solucionar problemas de lentitud del frontend.
  </Side>

  <Side>
    ## Requisitos

    * Una [cuenta gratuita de New Relic](https://newrelic.com/signup?utm_source=developer-site)
    * [Nodo](https://nodejs.org/en/)
    * [Python3](https://www.python.org/)
    * [Git](https://git-scm.com/)
    * [pip3](https://pip.pypa.io/en/stable/installation/)
  </Side>
</SideBySide>
