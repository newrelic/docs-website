<Callout
  variant="course"
  title="laboratorio"
>
  Este procedimiento es parte de un laboratorio que le enseña cómo solucionar problemas de su aplicación web con New Relic Browser. Si aún no lo ha hecho, consulte la [introducción al laboratorio](/docs/browser/new-relic-browser/lab/over-view/).
</Callout>

Antes de poder recorrer el laboratorio correctamente, debe activar su aplicación React. Tu aquí:

* Haga girar su aplicación React
* Envía tráfico a tu aplicación con un simple generador de carga

<Steps>
  <Step>
    Abra una ventana de terminal y clone el repositorio de laboratorio.

    ```bash
    git clone https://github.com/newrelic-experimental/relicstaurants.git
    ```
  </Step>

  <Step>
    Navegue hasta el directorio raíz de su aplicación y cambie al directorio de laboratorio.

    ```bash
    cd relicstaurants
    git switch browser-pro-lab-material
    ```
  </Step>

  <Step>
    A continuación, instale dependencia y ejecute la aplicación.

    ```bash
    npm install
    npm run build
    npm run newstart
    ```

    Esto abre su aplicación Relicstaurants en su browser.

    <img
      title="Relicstraunts homepage"
      alt="Relicstraunts homepage"
      src={homepage}
    />

    Ingrese su dirección de entrega y busque restaurantes para comenzar.

    <img
      title="Nearby restaurant list"
      alt="Nearby restaurant list"
      src={restaurantList}
    />

    Aquí verá una lista de restaurantes donde puede pedir comida.
  </Step>

  <Step>
    Elige un restaurante.

    <img
      title="Choose a restaurant"
      alt="Choose a restaurant"
      src={chooseRestaurant}
    />
  </Step>

  <Step>
    Seleccione uno o dos artículos y haga clic en el carrito.

    <img
      title="select food"
      alt="select food"
      src={selectFood}
    />
  </Step>

  <Step>
    Haga clic en <DoNotTranslate>**PAY**</DoNotTranslate>.

    <img
      title="checkout"
      alt="checkout"
      src={checkout}
    />
  </Step>

  <Step>
    Ingrese la siguiente información de tarjeta falsa y haga clic en <DoNotTranslate>**Finish payment**</DoNotTranslate> para realizar su pedido.

    <img
      title="place order with fake card"
      alt="place order with fake card"
      src={placeOrder}
    />

    Su pedido se realizó correctamente.

    <img
      title="Purchase completed"
      alt="Purchase completed"
      src={thankYou}
    />

    A continuación, utiliza un simulador para generar más tráfico a su aplicación.
  </Step>

  <Step>
    En otra ventana de terminal, navegue hasta el directorio raíz de su aplicación y ejecute el generador de carga.

    ```bash
    # Navigate to the root directiory of your simulator
    cd relicstaurants/simulator
    # Switch to lab branch
    git switch browser-pro-lab-material
    [output]
    # Install the simulator's dependencies
    pip3 install -r requirements.txt
    [output]
    # Run the simulator
    python3 simulator.py
    [output] ====== WebDriver manager ======
    [output] Current google-chrome version is 99.0.4844
    [output] Get LATEST chromedriver version for 99.0.4844 google-chrome
    ```

    <Callout variant="important">
      Este generador de carga supone que tienes Google Chrome instalado en tu computadora. Si utilizas un browser diferente, omite este paso y genera tráfico manualmente, o [instala Google Chrome](https://www.google.com/chrome/downloads/).
    </Callout>
  </Step>
</Steps>

Ahora que sabes cómo ejecutar tu aplicación, es hora de instrumentarla. En las ventanas de terminal que ejecutan su aplicación y simulador, presione `<CTRL-C>` para cerrarlos. Con su aplicación cerrada, podrá actualizar su código para introducir herramientas de monitoreo.
