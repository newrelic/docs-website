## Problema

Ves datos diferentes en <InlinePopover type="browser"/>que en otras herramientas de análisis que utilizas. Los síntomas comunes incluyen:

* El navegador registra menos [páginas vistas](/docs/browser/new-relic-browser/additional-standard-features/page-views-insights-your-sites-popularity) de lo esperado.
* [La distribución geográfica](/docs/browser/new-relic-browser/additional-standard-features/geography-webpage-performance-location) del navegador para su usuario difiere de las ubicaciones de su base de usuarios esperada.
* El navegador muestra una cantidad diferente de [sesiones activas](/docs/browser/new-relic-browser/page-load-timing-resources/session-tracking) que la que ve en otros lugares.

## Solución

Siga estos pasos de resolución de problemas para el monitoreo del navegador:

1. Verifique si el monitoreo del navegador está instalado correctamente en todas las páginas de su aplicación.
2. Si falta el fragmento de JavaScript, o si el usuario no puede enviar datos a New Relic, siga los pasos para [la resolución de problemas de monitoreo de instalación del navegador](/docs/browser/new-relic-browser/installation-configuration/troubleshooting-browser-monitoring-installation).
3. Si tiene una aplicación de estilo de página única y espera ver los cambios de ruta como vistas, considere usar [el monitoreo SPA del navegador](/docs/browser/single-page-app-monitoring/get-started/welcome-single-page-app-monitoring), que proporciona una vista integrada de las cargas iniciales de la página y los cambios de ruta.

Si el monitoreo del navegador funciona correctamente, es posible que vea resultados diferentes porque New Relic recopila y presenta información de manera diferente a otras herramientas de análisis. Para obtener más información sobre cómo recopilamos datos, consulte:

* [Proceso de sincronización de carga de página](/docs/browser/new-relic-browser/page-load-timing-resources/page-load-timing-process)
* [Instrumentación para monitoreo de navegador](/docs/browser/new-relic-browser/page-load-timing-resources/instrumentation-browser-monitoring)
* [Comprender la recopilación de datos SPA](/docs/browser/single-page-app-monitoring/use-spa-data/understand-spa-data-collection)
