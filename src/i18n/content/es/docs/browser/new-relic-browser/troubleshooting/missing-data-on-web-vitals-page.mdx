## Problema

No veo datos en <DNT>**Web vitals**</DNT>.

## Solución

Estas son algunas de las causas de la falta de datos:

* Tus filtros son demasiado estrechos. Algunos atributos clave solo son relevantes para ciertos elementos vitales de la web. Por ejemplo, si filtró por `elementID`, los datos solo se completarán para la pintura con contenido más grande (LCP) porque `elementID` solo se aplica a la medición de LCP.
* Su agente del navegador o edición no recopila ciertos datos. Para obtener información sobre qué atributo clave recopila el agente New Relic Browser , consulte el [diccionario de datos del agente del navegador](/attribute-dictionary/?dataSource=Browser+agent).
* Tu aplicación no puede enviar `PageViewTiming` datos a New Relic. Web vitals se basa en `PageViewTiming` datos, por lo que si por algún motivo su aplicación no puede enviar la ingesta a nuestro recopilador, es posible que los datos no aparezcan.
