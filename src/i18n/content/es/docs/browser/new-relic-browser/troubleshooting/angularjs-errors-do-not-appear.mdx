## Problema

Estás utilizando AngularJS y deseas informar errores de JavaScript a New Relic, pero no ves los errores de tu aplicación en la [página<DNT>**JavaScript errors**</DNT> ](/docs/browser/new-relic-browser/browser-pro-features/js-errors-dashboard-examining-errors-over-time).

## Solución

La aplicación AngularJS suele incluir un controlador de errores global que detecta todos los errores y, de forma predeterminada, <InlinePopover type="browser"/>solo informa los errores no detectados.

Utilice la llamada API [noticeError](/docs/browser/new-relic-browser/browser-agent-spa-api/newrelicnoticeerror-browser-agent-api) para informar los errores a New Relic. Esto le permite manejarlos como parte de su procesamiento habitual de errores, pero también verlos dentro de la UI de New Relic.
