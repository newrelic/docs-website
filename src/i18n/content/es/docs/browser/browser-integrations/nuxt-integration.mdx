Con nuestro dashboard de Nuxt, puede rastrear fácilmente la tasa de error de su aplicación, el promedio de carga inicial de la página, los cambios de ruta, el rendimiento y las vistas de página con errores de JavaScript. Construido con nuestro agente de infraestructura y nuestro Vue <InlinePopover type="browser" />, el panel de control de Nuxt le permite ver información crítica, todo en un solo lugar.

## Instalar [#install]

<Steps>
  <Step>
    ### Opcional: instalar el agente de infraestructura [#infra-install]

    Instalar el agente de infraestructura es opcional, pero al hacerlo obtendrá métricas importantes sobre el host donde se ejecuta su aplicación.

    Para hacer esto, siga [los pasos de instalación del agente de infraestructura](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent).
  </Step>

  <Step>
    ### Instalar la supervisión del navegador Nuxt [#browser-install]

    1. Desde <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT>, ve a la [página<DNT>**+ Integrations &amp; Agents**</DNT> ](https://one.newrelic.com/marketplace).
    2. Busca <DNT>**Nuxt**</DNT>. (Nota: El mosaico de Nuxt no está en la colección de navegador o móvil, por lo que deberás usar la función de búsqueda).
    3. Haz clic en el mosaico <DNT>**Nuxt**</DNT> cuando aparezca en los resultados de la búsqueda y sigue las instrucciones para generar el fragmento de código JavaScript.
  </Step>

  <Step>
    ### Configurar la integración [#config]

    Para agregar el fragmento de código JavaScript en la aplicación Nuxt:

    1. Cree un archivo JavaScript en la carpeta <DNT>**`public`**</DNT> de su aplicación Nuxt:

       ```bash
       touch public/<FILENAME>.js
       ```

    2. Copie y pegue el fragmento del agente del navegador en el nuevo archivo JavaScript sin las etiquetas `<script>` (solo el fragmento en sí, no hay ninguna función para definir):

       ```js
       // PASTE THE GENERATED BROWSER AGENT SNIPPET HERE WITHOUT <script> TAGS
       ```

    3. Abre tu archivo `nuxt.config.js` (o `nuxt.config.ts`) y agrega el script a la sección `app.head`:

       ```js
       export default defineNuxtConfig({
         compatibilityDate: '2025-07-15',
         devtools: { enabled: true },
         app: {
           head: {
             script: [
               {
                 src: "/<FILENAME>.js"
               },
             ],
           },
         },
       })
       ```

    4. Reinicie su aplicación.

    En un par de minutos, tu aplicación se enviará métrica a [one.newrelic.com](https://one.newrelic.com).
  </Step>
</Steps>

## Monitor su aplicación [#monitor]

Ahora debería poder ver su aplicación en <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> seleccionando <DNT>**Browser**</DNT> en la barra de navegación. Elige tu aplicación para ver todas las métricas que se capturan en la plataforma <DNT>**one.newrelic.com**</DNT> . También puede ver los datos con NRQL utilizando la siguiente consulta:

```sql
SELECT firstPaint
FROM PageView
```
