Algunas características de New Relic requieren su clave de licencia de monitoreo de navegador o su ID de New Relic de la aplicación. El recopilador de New Relic lee esta clave para determinar a qué cuenta pertenecen los datos entrantes.

Si utiliza el [método de copiar y pegar](/docs/browser/new-relic-browser/installation-configuration/adding-apps-new-relic-browser#copy-paste-app) para agregar una aplicación a <InlinePopover type="browser"/>, [la página <DNT>**Settings**</DNT> del navegador](/docs/browser/new-relic-browser/installation-configuration/browser-settings-ui-options-browser-monitoring) de su aplicación mostrará el fragmento de JavaScript generado. La parte inferior del JavaScript generado incluye su browser `licenseKey` (también conocido como `browser_monitoring_key`) y `applicationID`.

* El monitoreo del navegador `licenseKey` (`browser_monitoring_key`) es una clave única que está vinculada a su

  <InlinePopover type="licenseKey"/>

  principal, pero no es la misma. La clave browser no se puede utilizar para determinar la clave de licencia de su cuenta.

* El browser `applicationID` es el código de identificación único para cada aplicación en el monitoreo del navegador.

Para proteger la seguridad de tu `browser_monitoring_key` y `applicationID`, puedes [monitorear o bloquear tu dominio](/docs/browser/new-relic-browser/installation-configuration/monitor-or-block-specific-domains).

## Obtenga la clave de licencia browser y el ID de la aplicación desde la UI [#procedures]

Este procedimiento se aplica <DNT>**only**</DNT> cuando utiliza el método de copiar/pegar desplegable. Si utiliza el [método de implementación de la aplicación APM](/docs/browser/new-relic-browser/installation-configuration/adding-apps-new-relic-browser), su página <DNT>**Settings**</DNT> no mostrará el fragmento de JavaScript.

<img
  title="lkey_appId.png"
  alt="lkey_appId.png"
  src="/images/browser_screenshot-crop_license-app-key.webp"
/>

<figcaption>
  <DNT>**[one.newrelic.com](https://one.newrelic.com) > Add data > Browser monitoring > (choose the copy/paste install method) > (go through each installation step)**</DNT>: Al final de la configuración de la instrumentación, su clave de licencia y el ID de la aplicación se muestran en el fragmento de JavaScript.
</figcaption>

Para obtener únicamente la clave de licencia del navegador y la información de ID de la aplicación:

1. Vaya a

   <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > Settings**
   </DNT>

   .

2. Seleccione y copie el JavaScript generado y péguelo en un archivo de texto.

3. Elimine todo el contenido del archivo de texto excepto el monitoreo del navegador

   <InlinePopover type="licenseKey"/>

   y la información de ID de la aplicación (justo encima de la etiqueta final `</script>` ) y guárdelo.
