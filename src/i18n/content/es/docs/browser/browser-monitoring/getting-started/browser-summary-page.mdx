Nuestra página <InlinePopover type="browser"/><DNT>**Summary**</DNT> le ayuda a solucionar problemas con el rendimiento browser del usuario real de su aplicación. Utilice la página <DNT>**Summary**</DNT> para:

* Vaya directamente a otras browser UI páginas para examinar los problemas y cuestiones con más detalle.
* Agregue la condición de alerta recomendada.

## Utiliza los datos principales de Métricas web para mejorar el SEO [#core_vitals]

Utilice los gráficos <DNT>**Core web vitals**</DNT> para comprender el rendimiento de su browser según [las Métricas web principales de Google](https://web.dev/vitals/#core-web-vitals). Éstas incluyen:

* LCP: pintura con contenido más grande, que mide el rendimiento de carga
* INP: interacción con next paint, que mide la interactividad entre la entrada del usuario y la respuesta browser
* CLS: cambio de diseño acumulativo, que mide la estabilidad visual en la página del browser

Las puntuaciones más altas pueden ayudar a su organización a ahorrar dinero e influir en el SEO. Las puntuaciones más bajas pueden afectar las percepciones de los usuarios sobre el sitio web de su organización, lo que podría resultar en clientes insatisfechos o pérdida de oportunidades comerciales.

Para obtener más información sobre el monitoreo de Métricas web principales, consulte nuestra [Guía de Métricas web principales](/docs/browser/new-relic-browser/guides/guide-to-core-web-vitals).

## Agregar condición de alerta recomendada [#recommended-alert-conditions]

A través de la página de monitoreo de navegador <DNT>**Summary**</DNT> puede agregar condición de alerta recomendada a su entidad.

Consulte la página [de recomendaciones de condición y brechas de cobertura de alerta](/docs/alerts-applied-intelligence/applied-intelligence/alert-coverage-gaps/) si desea ver más entidades descubiertas.

Siga estos pasos para agregar la condición de alerta recomendada:

1. Abra la página monitoreo del navegador <DNT>**Summary**</DNT> .

2. Haga clic en <DNT>**Add recommended conditions**</DNT>.

   <img
     width="80%;"
     style={{align: "left"}}
     title="Browser monitoring summary"
     alt="Screenshot of the New Relic browser monitoring Summary page"
     src="/images/browser_screenshot-full_browser-alert-conditions.webp"
   />

3. Elija una serie de condiciones de alerta recomendadas.

   <img
     width="70%;"
     style={{align: "left"}}
     title="Browser monitoring recommended alert conditions"
     alt="Browser monitoring recommended alert conditions"
     src="/images/browser_screenshot-crop_browser-add-alert.webp"
   />

4. Haga clic en <DNT>**Next**</DNT>.

5. Seleccione una política para recibir notificaciones. Puedes seleccionar uno existente o [crear uno nuevo](/docs/alerts-applied-intelligence/new-relic-alerts/alert-policies/create-edit-or-find-alert-policy/#alert-policy-name).

6. Haga clic en <DNT>**Activate conditions**</DNT>.
