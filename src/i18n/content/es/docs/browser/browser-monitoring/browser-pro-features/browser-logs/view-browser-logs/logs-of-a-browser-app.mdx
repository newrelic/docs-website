Para ver los logs del rendimiento de una aplicación del browser:

<Steps>
  <Step>
    Vaya a <DNT>[one.newrelic.com](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Browser**</DNT>.
  </Step>

  <Step>
    Seleccione una aplicación browser .
  </Step>

  <Step>
    En el menú de la izquierda, navegue hasta **Logs**.
  </Step>
</Steps>

<img title="Browser logs" alt="Screenshot of browser logs page" src="/images/browser_logs_ui.webp" />

## ¿Que sigue?

* También puede ver los logs del browser en contexto con Session Replay (si está empleando la función Session Replay ) o junto con los errores de JavaScript en Errors Inbox. Consulte [Logs en Session Replay](#session-replay-section) y [Logs en la bandeja de entrada de errores](#js-errors-section) para obtener más detalles.

* Para obtener más información sobre la UI de logs, consulte [Usar UI de logs](/docs/logs/ui-data/use-logs-ui) para obtener más detalles.
