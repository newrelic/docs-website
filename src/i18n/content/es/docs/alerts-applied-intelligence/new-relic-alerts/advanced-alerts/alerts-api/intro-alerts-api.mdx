Tenemos dos opciones para usar API para administrar mediante programación la característica de alerta e inteligencia aplicada de New Relic:

* NerdGraph (GraphQL): esta es la API principal y recomendada de New Relic para consultar datos y realizar configuraciones. Siempre estamos agregando funcionalidades a NerdGraph. [Consulte los documentos de NerdGraph relacionados con las alertas.](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/alerts-nerdgraph/nerdgraph-api-examples)
* API REST: esta es la API original de New Relic. Realizamos un trabajo mínimo en esta API y es por eso que recomendamos usar NerdGraph. Si actualmente estás utilizando la API REST, te recomendamos cambiar a NerdGraph. [Consulte los documentos de la API REST relacionados con las alertas.](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/rest-api-alerts/rest-api-calls-alerts)
