<Callout title="avance">
  Todavía estamos trabajando en esta característica, ¡pero nos encantaría que la probaras!

  Esta característica se proporciona actualmente como parte de un programa de vista previa de conformidad con nuestras [políticas de prelanzamiento](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

Conecta tu facturación de Google Cloud Platform a Cloud Cost Intelligence para monitorear y optimizar costos. La integración utiliza la autorización basada en cuentas de servicio para acceder de forma segura a tus datos de facturación y precios de GCP. New Relic crea una cuenta de servicio dedicada con acceso de solo lectura a tus conjuntos de datos de BigQuery.

## Antes de que empieces

Necesitas:

* **Proyecto GCP conectado a la supervisión de infraestructura de New Relic** - Consulte [Conectar los servicios de Google Cloud Platform a New Relic](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)

* **Cuenta de facturación de GCP activa**

* **Conjunto de datos de BigQuery** con:

  * Tabla de exportación de facturación
  * Tabla de exportación de precios en la nube

* **Acceso a la consola de Google Cloud**

## Instrucciones de configuración

<Steps>
  <Step>
    ### Seleccione la cuenta de New Relic [#select-nr-account]

    1. Vaya a <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT>.
    2. Seleccione su cuenta y haga clic en **Continue**.
  </Step>

  <Step>
    ### Proporcione los detalles de conexión de GCP [#gcp-connection-details]

    1. Ingrese su información de facturación y conjunto de datos de GCP:

       * **Connection name** - Nombre descriptivo
       * **Project ID** - Tu ID de proyecto de GCP
       * **Dataset name**: el nombre de su conjunto de datos de BigQuery
       * **Billing table name** - El nombre de su tabla de facturación
       * **Pricing table name** - El nombre de su tabla de precios

    2. Haga clic en **Continue**.
  </Step>

  <Step>
    ### Autorizar acceso [#gcp-authorize-access]

    1. Copie el ID de la cuenta de servicio que se muestra.
    2. Abra la **Google Cloud Console** y otorgue el rol de **BigQuery DataViewer** a la cuenta de servicio para sus tablas de facturación de BigQuery.
    3. Regrese y haga clic en **Continue**.
  </Step>

  <Step>
    ### Crear conexión [#gcp-create-connection]

    Haga clic en **Create connection** para verificar la configuración y establecer la integración.
  </Step>

  <Step>
    ### Ver tus datos [#gcp-view-data]

    Vaya a [**Intelligent summary (Resumen inteligente)**](/docs/cci/financial-intelligence/intelligent-summary) para ver sus datos de costos de GCP.

    <Callout variant="important">
      El procesamiento por primera vez puede tardar hasta 12 horas.
    </Callout>
  </Step>
</Steps>
