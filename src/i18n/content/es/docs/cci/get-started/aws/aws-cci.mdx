Configure su entorno de AWS para <DNT>**Cloud Cost Intelligence**</DNT> mediante la configuración de la facturación consolidada, los Informes de costos y uso (CUR), los roles y políticas de Identity and Access Management (IAM) y las regiones de AWS necesarias.

## Requisitos previos

Asegúrese de tener lo siguiente antes de iniciar cualquiera de los métodos de configuración:

### Requerido para todas las configuraciones

* <DNT>**AWS account with consolidated billing**</DNT>: [Configurar la facturación consolidada para AWS Organizations](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/consolidated-billing.html). Esto es necesario para agregar datos de costos de todas las cuentas de su organización, lo que permite un análisis de costos integral en toda su infraestructura de AWS.
* <DNT>**IAM Privileges**</DNT>: Se requiere acceso administrativo a la cuenta de AWS objetivo para desplegar recursos, crear buckets de S3, políticas de IAM y roles de IAM.

### Requerido solo para la configuración automatizada

* <DNT>**New Relic API Key**</DNT>: genera una clave de API válida para la creación del stack. Recomendamos crear una clave dedicada para la integración de <DNT>**CCI**</DNT>. Genere esto en la UI de New Relic: [menú de usuario &gt; API Keys](https://onenr.io/07wklp68lRL).

## Elija su método de configuración

<DNT>**Cloud Cost Intelligence**</DNT> ofrece dos formas de configurar la integración con AWS:

* <DNT>[**Automate AWS with CloudFormation (Recommended)**](/docs/cci/get-started/aws/automated-setup-cloudformation)</DNT>: despliegue rápidamente un stack de CloudFormation preconfigurado que aprovisiona automáticamente todos los recursos de AWS necesarios. Ideal para una incorporación más rápida con una configuración manual mínima.
* <DNT>[**Manually integrate your AWS account**](/docs/cci/get-started/aws/manual-integrate-aws-account)</DNT>: Configure manualmente cada recurso de AWS para tener un control total del proceso de configuración. Ideal para organizaciones con requisitos específicos de cumplimiento o personalización.
