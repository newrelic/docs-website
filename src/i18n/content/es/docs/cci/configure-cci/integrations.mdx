<Callout title="avance">
  Todavía estamos trabajando en esta característica, ¡pero nos encantaría que la probaras!

  Esta característica se proporciona actualmente como parte de un programa de vista previa de conformidad con nuestras [políticas de prelanzamiento](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy).
</Callout>

La característica de integración Inteligencia para costos en la nube le permite ver y gestionar sus cuentas de proveedor cloud . Emplee esta función para pausar o eliminar la integración de una cuenta de proveedor cloud específica.

1. Navegue hasta <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT>.
2. Desde el menú de navegación de la izquierda, seleccione la pestaña de **Integrations** para mostrar una lista de todas las integraciones de costos cloud .
3. Para ver detalles adicionales, en la columna **Name** , haga clic en la cuenta del proveedor cloud requerida.

<Callout variant="important">
  Actualmente, solo se admiten los costos cloud de AWS.
</Callout>

4. Junto a la cuenta requerida, haga clic en el ícono de puntos suspensivos y elija una de las siguientes opciones:

   * **Pause**: detiene temporalmente la recopilación de datos de la integración.
   * **Delete**: elimina permanentemente la integración de tu cuenta.
