    <TabsPageItem id="kubernetescost">
      Ejemplo de consulta:

      ```sql
      FROM KubernetesCost SELECT * SINCE 1 week ago
      ```

      Para obtener costos precisos de Kubernetes, multiplique `cpu_weightage + memory_weightage` en sus consultas:

      ```sql
      SELECT SUM(net_unblended_cost * (cpu_weightage + memory_weightage) + network_net_unblended_cost) FROM KubernetesCost SINCE '2024-09-01' UNTIL '2024-09-10'
      ```

      Esto multiplica `net_unblended_cost` por la suma de la ponderación de CPU y memoria para asignaciones de costos precisas basadas en el uso de recursos.
