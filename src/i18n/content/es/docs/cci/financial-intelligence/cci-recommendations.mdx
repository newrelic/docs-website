Cloud Cost Intelligence proporciona recomendaciones de optimización de costos de dos fuentes para ayudarlo a identificar y actuar sobre las oportunidades en toda su infraestructura de AWS. Analiza sus patrones de uso de recursos utilizando AWS Trusted Advisor y nuestro motor de inteligencia nativa patentado.

<Callout variant="important">
  Las recomendaciones de optimización de costos están actualmente disponibles para AWS.
</Callout>

Use recommendations to:

* Identify oversized and underutilized resources and commitment opportunities
* Vea los ahorros potenciales con proyecciones mensuales y anuales
* Rastrear y priorizar las acciones de optimización (reconocer, descartar o marcar como pendiente)

## Fuentes de recomendación

### AWS Trusted Advisor

AWS Trusted Advisor proporciona recomendaciones de optimización de costos directamente desde AWS.

#### Requisitos previos

Para usar las recomendaciones de **Trusted Advisor** con Cloud Cost Intelligence, configure y configure el acceso a la información de costos de AWS Trusted Advisor en su cuenta de AWS.

#### Cómo navegar

Una vez configuradas, las recomendaciones de **Trusted Advisor** aparecen en la página de **Cloud Cost Intelligence Recommendations** junto con otras recomendaciones.

<Callout variant="important">
  Cloud Cost Intelligence verifica el estado de **Trusted Advisor** durante la integración inicial de la cuenta de AWS. Cuando **Trusted Advisor** está inactivo, Cloud Cost Intelligence omite este componente de integración.

  Si **Trusted Advisor** se habilita más tarde, Cloud Cost Intelligence identificará automáticamente la actualización y sincronizará los datos dentro de los siete días posteriores a la habilitación.
</Callout>

### Motor de optimización nativo de CCI

El motor de optimización nativo de CCI analiza la utilización de recursos y el historial de uso para identificar oportunidades de dimensionamiento correcto de las instancias y reducir los costos a través de los compromisos de los Planes de Ahorro de AWS.

#### Cómo funciona

El motor analiza los datos de uso en dos áreas:

* **Ajuste de tamaño**: Comprueba el uso de CPU y memoria, compara los tipos de instancia actuales con las necesidades de rendimiento e identifica las instancias que se pueden reducir sin afectar el rendimiento.
* **Planes de ahorro**: Revisa los patrones de uso para encontrar cargas de trabajo consistentes y calcula los ahorros potenciales de los compromisos de los Planes de Ahorro de AWS.

#### Qué recomendaciones proporciona

El motor proporciona recomendaciones para reducir los costos:

* **Instance type changes**: Specific instance modifications (e.g., Migrate from im4gn.16xlarge to t2.micro) with current configuration, CPU and memory usage, and number of affected instances
* **Savings plan commitments**: Recommendations to commit to consistent usage levels for discounted pricing
* **Savings projections**: Monthly and annual cost savings for each recommendation

## View recommendations

Para acceder a sus recomendaciones de optimización de costos:

1. Vaya a **[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Cloud Cost Intelligence**
2. Haga clic en **Recommendations** en el menú de navegación de la izquierda.
