La página <DNT>**Cost Overview**</DNT> (anteriormente conocida como <DNT>**Cost Explorer**</DNT>) proporciona un análisis detallado de costos con funcionalidad de desglose en múltiples dimensiones.

Para una vista consolidada de todos sus gastos en la nube con widgets de resumen, consulte la página [Resumen inteligente](/docs/cci/financial-intelligence/intelligent-summary).

<img title="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" alt="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" src="/images/cost-overview.webp" />

## Filtros de proveedor de la nube

Para filtrar sus datos de costos por proveedor de cloud, seleccione una opción del menú desplegable en la parte superior izquierda de la página:

* <DNT>**All**</DNT>: todos los proveedores de cloud
* <DNT>**AWS**</DNT>: solo Amazon Web Services
* <DNT>**Azure**</DNT>: solo Microsoft Azure
* <DNT>**GCP**</DNT>: solo Google Cloud Platform

## Principales gastos

El widget <DNT>**Top Spends**</DNT> lo ayuda a identificar sus áreas de mayor costo. Vea los costos desglosados por:

* <DNT>**Service**</DNT>: costos agrupados por tipo de servicio en la nube (por ejemplo, AmazonEC2, Fastly Edge Cloud Platform, AmazonS3)
* <DNT>**Category**</DNT>: Costos organizados por categoría de uso (por ejemplo, Calcular, Otros, Red)
* <DNT>**Region**</DNT>: Costos por región geográfica (por ejemplo, us-west-2, Sin etiquetas, eastus). &quot;Untagged&quot; indica costos de recursos sin etiquetas regionales
* <DNT>**Charge type**</DNT>: Costos por categoría de facturación (por ejemplo, Usage, SavingsPlanCoveredUsage, DiscountedUsage)

Cada dimensión muestra:

* Monto total del costo
* Cambio porcentual en comparación con el período anterior (basado en el rango de fechas seleccionado)
* Los tres elementos principales con sus costos individuales
* <DNT>**Show summary**</DNT> alternar para expandir o contraer vistas detalladas

<Callout variant="tip">
  Utilice los indicadores de cambio porcentual para identificar rápidamente servicios o regiones con aumentos de costos significativos. Los indicadores rojos resaltan las áreas donde el gasto ha crecido en comparación con el período anterior.
</Callout>

## Tendencias de costos

La sección <DNT>**Cost Trends**</DNT> proporciona un análisis de costos basado en el tiempo con gráficos y controles interactivos.

### Controles del gráfico

* <DNT>**View options**</DNT>: Alternar entre vistas <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT> o <DNT>**Monthly**</DNT>
* <DNT>**Group by**</DNT>: Seleccione de 1 a 3 dimensiones para agrupar datos de costos
* <DNT>**View by**</DNT>: Elija el formato de visualización (la disponibilidad depende de sus selecciones de agrupación)

### Comprender los datos

* <DNT>**Total cost**</DNT>: muestra el gasto agregado con el cambio porcentual
* <DNT>**Average cost**</DNT>: muestra el gasto promedio diario con el cambio porcentual
* <DNT>**Stacked bar chart**</DNT>: Visualiza la distribución de costos a lo largo del tiempo con segmentos codificados por colores para diferentes servicios

### Tabla de datos de costos

Debajo del gráfico, se muestra una tabla detallada:

* <DNT>**Cloud provider service**</DNT>: Nombre del servicio
* <DNT>**Total Cost**</DNT>: Monto de gasto agregado (ordenable)
* <DNT>**% of Spend**</DNT>: Porcentaje del gasto total
* <DNT>**Compared Cost**</DNT>: Comparación de costos con el período anterior
* <DNT>**Trend**</DNT>: Indicador de cambio porcentual con codificación por colores (verde para disminución, rojo para aumento)

### Opciones de visualización

**Restricciones de agrupación:**

* **Selección mínima**: 1 dimensión
* **Selección máxima**: 3 dimensiones

**Con una sola selección de agrupación:**

* <DNT>**Bar chart**</DNT>: Muestra los datos de costos como barras verticales
* <DNT>**Line chart**</DNT>: Muestra las tendencias de costos a lo largo del tiempo
* <DNT>**Tree chart**</DNT>: visualización jerárquica de datos de costos
* <DNT>**Data table**</DNT>: Vista tabular de los datos de costos

**Con múltiples selecciones de agrupación:**

* <DNT>**Heat map**</DNT>: visualización codificada por colores
* <DNT>**Data table**</DNT>: Vista tabular de los datos de costos

## Funcionalidad de desglose

Haga clic en cualquier servicio (por ejemplo, AmazonEC2) en la tabla de datos de costos para ver desgloses detallados por categoría, región o tipo de cargo. Haz clic de nuevo para ver los costos de los recursos individuales.

Todos los tipos de servicio admiten el desglose, incluidos los de cómputo, red y almacenamiento.

## Filtros y personalización

La página <DNT>**Cost Overview**</DNT> admite opciones de filtrado y personalización para ayudarle a enfocarse en segmentos de costos específicos.

### Selector de rango de fechas

En la parte superior derecha de la página, use el selector de rango de fechas para:

* Elige un período de tiempo predefinido
* Establecer fechas de inicio y finalización personalizadas
* Ver costos en su zona horaria preferida (UTC se muestra de forma predeterminada)

### Filtro de costos de ingeniería

Si su organización utiliza etiquetas de asignación de costos, puede filtrar por la etiqueta <DNT>**Engineering costs**</DNT> para limitar el análisis a equipos, proyectos o centros de costos específicos etiquetados en su entorno cloud.
