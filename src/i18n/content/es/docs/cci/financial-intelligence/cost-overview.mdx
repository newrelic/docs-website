La página de Descripción general de costos (anteriormente conocida como Explorador de costos) proporciona un análisis detallado de costos con funcionalidad de desglose en múltiples dimensiones.

For a consolidated view of all your cloud spending with summary widgets, see the [Intelligent summary](/docs/cci/financial-intelligence/intelligent-summary) page.

## Funcionalidad de desglose

Cloud Cost Intelligence proporciona capacidad de desglose para ver detalles de costos granulares.

### Cómo funciona

Puedes ver los costos en diferentes dimensiones. Cuando haces clic en un servicio (por ejemplo, EC2), puedes profundizar en un servicio en particular y, luego, dentro de él, en un recurso en particular para ver los detalles granulares de ese recurso.

Todos los tipos de servicio admiten el desglose, incluidos los de cómputo, red y almacenamiento.

## Característica

* **Filtrable**: Todos los datos son filtrables y puede mostrar y ocultar elementos
* **Vistas múltiples**: puede cambiar las vistas para ver los datos en diferentes formatos
* **Opciones de agrupación**: Puedes tener múltiples opciones de agrupación. Por ejemplo, puedes ver por dimensión de servicio en la nube, luego cambiarla a tipo de cargo y opción de pago
* **Vista de tabla dinámica**: Cuando se utilizan múltiples selecciones de agrupación, los datos se reflejan como una vista de tabla dinámica para analizar los costos granulares dentro de una sola vista
* **Descargar**: puede descargar los datos de costos para analizarlos en diferentes dimensiones

## Restricciones de agrupación

* **Selección mínima**: 1 dimensión
* **Selección máxima**: 3 dimensiones

<Callout variant="important">
  Cuando tiene múltiples selecciones de agrupación, el gráfico de barras, el gráfico de líneas y el gráfico de árbol no son aplicables. Solo están disponibles la tabla de datos y el mapa de calor.
</Callout>
