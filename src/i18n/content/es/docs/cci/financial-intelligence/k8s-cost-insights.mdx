## Descripción general

Acceda a información valiosa específica de sus entornos Kubernetes a través de la página de Asignación de costos Kubernetes. Esta función aprovecha New Relic telemetry data los de para proporcionar una vista de los costos asociados con su Kubernetes aplicación.

Use the Kubernetes Cost Allocation page to view costs incurred by each application over time and detailed cost analysis.

## Descripción general de la asignación

The Allocation Overview widget offers a bar chart visualization illustrating the costs incurred by each application over time.

## Análisis detallado de costos

The Kubernetes Cost Allocation view presents a detailed table with a breakdown of resource usage and costs by application name:

* **Uso de CPU:** muestra el porcentaje de recursos de CPU consumidos por cada aplicación.
* **Costo de CPU:** Muestra el costo atributo al uso de CPU para cada aplicación.
* **Uso de memoria:** enumera los recursos de memoria por porcentaje empleados por cada aplicación.
* **Costo de la memoria:** refleja el costo asociado con el uso de la memoria.
* **Costo total:** proporciona el costo total incurrido por cada aplicación, integrando todo el uso de recursos.

## Unused and Unallocated cost

These cost dimensions help you measure cluster efficiency:

### Costo no utilizado

El costo no utilizado es la cantidad del clúster que se ha asignado pero que no recibe ninguna solicitud.

**Ejemplo**: Si su costo total es de $5.60 asignado a una implementación de clúster, pero solo está utilizando $4.64, entonces $0.96 es el costo no utilizado.

### Costo no asignado

El costo no asignado es la cantidad del clúster que no se ha asignado a ninguna fuente.

Estas son medidas de eficiencia para su clúster de Kubernetes.

## Beneficios

The Kubernetes Cost Allocation feature delivers these outcomes:

* **Optimized resource usage:** Visibility into underutilized or excessively costly applications enables effective resource allocation adjustments.
* **Controlled expenses:** Insights into specific cost drivers within your Kubernetes environment enable more informed financial decisions.
* **Enhanced efficiency:** Resource management strategies aligned with business objectives ensure efficient use of cloud resources.

The Kubernetes Cost Allocation page provides the insights necessary to proactively manage Kubernetes costs, supporting strategic decision-making and financial optimization.
