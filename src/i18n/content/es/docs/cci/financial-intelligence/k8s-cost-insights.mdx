Access insights specific to your Kubernetes environments through the <DNT>**Kubernetes Costs**</DNT> page. This feature leverages New Relic&apos;s telemetry data to provide a detailed view of the costs associated with your Kubernetes clusters.

Use the <DNT>**Kubernetes Costs**</DNT> page to view costs incurred by each application over time and detailed cost analysis.

<img title="Kubernetes Cost Insights highlighted on the Cloud Cost Intelligence Kubernetes Cost Insights page" alt="Kubernetes Cost Insights highlighted on the Cloud Cost Intelligence Kubernetes Cost Insights page" src="/images/k8s-cci.webp" />

## Filters and customization

### Cloud provider filters

To filter your cost data by cloud provider, select an option from the dropdown menu at the top left of the page:

* <DNT>**All**</DNT>: All cloud providers
* <DNT>**AWS**</DNT>: Amazon Web Services only
* <DNT>**Azure**</DNT>: Microsoft Azure only

### Date range selector

At the top right of the page, use the date range selector to:

* Choose a predefined time period
* Set custom start and end dates
* View costs in your preferred timezone (UTC displayed by default)

### Engineering costs filter

If your organization uses cost allocation tags, you can filter by the <DNT>**Engineering costs**</DNT> tag to narrow analysis to specific teams, projects, or cost centers tagged in your cloud environment.

## Resumen

The Summary section displays four key cost metrics for your selected time period:

* <DNT>**Total cost**</DNT>: The aggregate spending for all Kubernetes clusters with percentage change compared to the previous period
* <DNT>**Average cost (Daily)**</DNT>: The daily average spending with percentage change
* <DNT>**Unused cost**</DNT>: The cost of resources that have been allocated but are not receiving any requests, with percentage change
* <DNT>**Unallocated cost**</DNT>: The cost of cluster resources that have not been allocated to any source, with percentage change

## Insights

The Insights section provides two visualization widgets to help you analyze your Kubernetes costs.

### Cost Trends

The <DNT>**Cost Trends**</DNT> widget displays a line chart showing how your Kubernetes costs change over time. Use the view options to switch between <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT>, or <DNT>**Monthly**</DNT> views.

### Top 20

The <DNT>**Top 20**</DNT> widget displays your highest cost areas using a tree chart visualization. By default, it shows the top 20 clusters by cost. You can customize the view using:

* <DNT>**Dropdown filter**</DNT>: Select what to display (<DNT>**Cluster name, by cost**</DNT>)
* <DNT>**Group by**</DNT>: Choose dimensions to group cost data
* <DNT>**View by**</DNT>: Switch between <DNT>**Tree chart**</DNT> and other visualization formats

## Cost &amp; Usage Analysis

The <DNT>**Cost &amp; Usage Analysis**</DNT> section presents a detailed table with a breakdown of resource usage and costs by cluster name:

* <DNT>**Cluster name**</DNT>: The name of your Kubernetes cluster
* <DNT>**CPU cost**</DNT>: The cost attributed to CPU usage for each cluster
* <DNT>**CPU usage**</DNT>: The percentage of CPU resources consumed by each cluster
* <DNT>**Memory cost**</DNT>: The cost associated with memory usage
* <DNT>**Memory usage**</DNT>: The memory resources by percentage used by each cluster
* <DNT>**Unused Cost**</DNT>: The cost of allocated but unused resources
* <DNT>**Unallocated cost**</DNT>: The cost of cluster resources not allocated to any source
* <DNT>**Total cost**</DNT>: The total cost incurred by each cluster, integrating all resource usage (sortable)

## Understand unused and unallocated costs

Estas dimensiones de costos le ayudan a medir la eficiencia del clúster:

### Costo no utilizado

Unused cost represents resources that have been allocated but are not receiving any requests.

**Ejemplo**: Si su costo total es de $5.60 asignado a una implementación de clúster, pero solo está utilizando $4.64, entonces $0.96 es el costo no utilizado.

### Costo no asignado

Unallocated cost represents cluster resources that have not been allocated to any source.

These metrics help you identify opportunities to improve cluster efficiency and reduce waste.

<Callout variant="tip">
  Select any cluster to view its <DNT>**Cost Trends**</DNT>, <DNT>**Cost Breakdown (Total vs Used)**</DNT>, and detailed information about unused and unallocated costs.
</Callout>

## Beneficios

The Kubernetes Cost Insights feature delivers these outcomes:

* <DNT>**Optimized resource usage**</DNT>: Visibility into underutilized or excessively costly clusters enables effective resource allocation adjustments
* <DNT>**Controlled expenses**</DNT>: Insights into specific cost drivers within your Kubernetes environment enable more informed financial decisions
* <DNT>**Enhanced efficiency**</DNT>: Resource management strategies aligned with business objectives ensure efficient use of cloud resources

The <DNT>**Kubernetes Costs**</DNT> page provides the insights necessary to proactively manage Kubernetes costs, supporting strategic decision-making and financial optimization.
