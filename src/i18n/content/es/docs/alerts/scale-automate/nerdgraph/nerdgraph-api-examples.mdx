Puede administrar sus políticas, condiciones y reglas de silenciamiento mediante programación utilizando nuestra [API NerdGraph](/docs/apis/nerdgraph/get-started/introduction-new-relic-nerdgraph) en formato GraphQL. Esta es una alternativa poderosa para administrarlos dentro de New Relic o con nuestra [API REST](/docs/alerts/rest-api-alerts/new-relic-alerts-rest-api/rest-api-calls-new-relic-alerts) anterior.
