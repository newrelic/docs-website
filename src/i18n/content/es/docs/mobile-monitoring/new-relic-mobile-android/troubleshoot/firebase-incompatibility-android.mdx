## Problema

Se mostrarán conflictos, fallas o errores cuando tanto el agente de Android New Relic como [el SDK de monitoreo de rendimiento de Firebase](https://firebase.google.com/docs/perf-mon) de Google se incluyan en una aplicación.

Errores conocidos:

* `java.lang.StackOverflowError`
* `java.lang.RuntimeException: An error occurred while executing doInBackground()`

## Solución

Nuestro agente de Android y FireBase PerfMon no se pueden habilitar al mismo tiempo. Recomendamos habilitar solo uno u otro para evitar este problema.

Si necesita ayuda adicional, obtenga soporte en [support.newrelic.com](https://support.newrelic.com).
