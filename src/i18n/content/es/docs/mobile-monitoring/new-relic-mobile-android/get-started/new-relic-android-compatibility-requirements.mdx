La capacidad <InlinePopover type="mobile" />es compatible con las tiendas de aplicaciones estándar, incluidas Google Play Store, Amazon Appstore y otras. Antes de [instalar y configurar el agente móvil](/docs/mobile-monitoring/new-relic-mobile-android/install-configure) con su aplicación de Android, siga estas pautas para conocer la compatibilidad y otros requisitos.

## Monitoreo en primer plano [#foreground]

El agente de Android solo monitorea su aplicación mientras está en primer plano. El agente no monitor los servicios en segundo plano mientras la aplicación está cerrada. Para obtener más información, consulte nuestra [documentación sobre seguridad y privacidad de datos móviles](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps).

## No se admiten pruebas unitarias [#testing]

Nuestro agente New Relic Mobile está diseñado y probado para funcionar en el ciclo de vida normal de una aplicación. No admitimos la ejecución de ningún entorno de prueba en la aplicación con el agente. Las pruebas pueden causar conflictos y comportamientos impredecibles.
