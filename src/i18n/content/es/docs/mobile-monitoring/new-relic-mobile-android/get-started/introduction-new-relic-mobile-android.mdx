[monitoreo de móviles](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile) para Android monitorea tu aplicación móvil, brindándote una vista integral del rendimiento de tu aplicación. Funciona para aplicaciones de Android escritas con Java o Kotlin.

## Compatibilidad y requisitos [#requirements]

Antes de instalar el agente de Android, revise la [compatibilidad y los requisitos](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements/).

## Instalar el agente [#installation]

Para instalar el agente de Android, siga nuestra [instalación guiada](https://onenr.io/02R5GVpaxQb), ubicada directamente en la UI. Con este método, puede instalar y configurar el agente de Android con Gradle y Android Studio.
