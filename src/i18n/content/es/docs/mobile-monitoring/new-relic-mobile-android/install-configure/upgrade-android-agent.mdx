Para asegurarse de tener la versión más actualizada del agente de Android, consulte las [notas de la versión del agente de Android](/docs/release-notes/mobile-release-notes/android-release-notes). Las instrucciones para actualizar su aplicación de Android variarán según la versión del SDK del agente de Android que esté utilizando actualmente.

## Actualización desde las versiones 2 o 3 del SDK de Android [#android_upgrade_2]

Si ha instalado previamente la versión 2 o 3 del SDK de Android: vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Settings > Upgrade.**</DNT>
