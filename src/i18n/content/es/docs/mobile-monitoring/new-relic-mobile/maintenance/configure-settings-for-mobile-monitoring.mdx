## Ver el token de su aplicación [#application-token]

El token de su aplicación móvil es una cadena hexadecimal única de 40 caracteres que autentica los datos de su aplicación en New Relic. Cada aplicación móvil que monitor tiene un token único para el centro de datos de EE. UU. y la UE.

Todos los datos de su aplicación móvil se reportan al mismo lugar en la UI de monitoreo de móviles de New Relic, independientemente del nombre de la aplicación cuando se instala en un dispositivo móvil. Si estás usando un framework híbrido como Flutter o React Native y lo implementas en plataformas iOS y Android, necesitarás usar tokens separados, uno para iOS y otro para Android.

Para ver su token de aplicación:

1. Vaya a

   <DNT>
     **[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

   , haga clic en

   <DNT>
     **Mobile**
   </DNT>

   y seleccione una aplicación de monitor.

2. Seleccione

   <DNT>
     **Application**
   </DNT>

   o

   <DNT>
     **Installation**
   </DNT>

   .

## Configuración del lado del servidor [#configure-server-side-settings]

La configuración del agente móvil del lado del servidor proporciona un mecanismo para gestionar y ajustar de forma remota el comportamiento del agente móvil desplegar en los dispositivos de los usuarios. Esta funcionalidad le permite controlar dinámicamente la configuración de monitoreo sin necesidad de desplegar una nueva tienda de aplicaciones.

La siguiente configuración se puede modificar actualmente de forma remota en la página de **Application** :

* Application Not Responding (ANR) rastreo de errores (solo Android): permite rastrear e informar los motivos detrás de los errores ANR. Esta funcionalidad está habilitada de forma predeterminada y requiere el agente Android 7.4.1 o superior.

  <Callout variant="important">
    La configuración del lado del servidor siempre tiene prioridad sobre la configuración del agente local. Para aplicaciones móviles con menos de 100.000 dispositivos activos simultáneamente, puede llevar hasta 10 minutos aplicar los cambios a los dispositivos finales del usuario. Para aplicaciones con más de 100.000 dispositivos activos, puede tardar varias horas.
  </Callout>

### Configurar desde UI [#ui-configuration]

Para acceder a la configuración del lado del servidor para aplicaciones de monitor desde la UI, vaya a <DNT>**[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**</DNT> > <DNT>**Mobile**</DNT> (seleccione su aplicación) > <DNT>**Settings**</DNT> > <DNT>**Application**</DNT>.
