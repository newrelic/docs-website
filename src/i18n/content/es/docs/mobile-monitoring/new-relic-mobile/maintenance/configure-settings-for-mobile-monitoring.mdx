## Ver el token de su aplicación [#application-token]

El token de su aplicación móvil es una cadena hexadecimal única de 40 caracteres que autentica los datos de su aplicación en New Relic. Cada aplicación móvil que monitor tiene un token único para los centros de datos de EE. UU. y de la UE.

Todos los datos de su aplicación móvil se reportan al mismo lugar en la UI de monitoreo de móviles de New Relic, independientemente del nombre de la aplicación cuando se instala en un dispositivo móvil. Si estás usando un framework híbrido como Flutter o React Native y lo implementas en plataformas iOS y Android, necesitarás usar tokens separados, uno para iOS y otro para Android.

Para ver su token de aplicación:

1. Vaya a

   <DoNotTranslate>**[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**</DoNotTranslate>

   , haga clic en

   <DoNotTranslate>**Mobile**</DoNotTranslate>

   y seleccione una aplicación de monitor.

2. Seleccione

   <DoNotTranslate>**Application**</DoNotTranslate>

   o

   <DoNotTranslate>**Installation**</DoNotTranslate>

   .
