      ## Sintaxis [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAttribute(string $name)
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.removeAttribute(name: String?)
      ```

      ## Descripción [#description]

      Este método elimina un atributo especificado por la cadena de nombre. Si desea eliminar todos los atributos personalizados, consulte [Eliminar todos los atributos](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes).

      ## Parámetros [#parameters]

          <tr>

            <th width={200}>
              Tipo
            </th>

            <th>
              Descripción
            </th>
          </tr>
        </thead>

        <tbody>
          <tr>
            <td>
              `$name`
            </td>

            <td>
              `string`
            </td>

            <td>
              Requerido. El nombre del atributo que desea eliminar.
            </td>
          </tr>
        </tbody>
      </table>

      ## Valores de retorno [#return-values]

      Devuelve `true` si se registró correctamente y `false` si no es así.

      ## Ejemplo [#example]

      A continuación se muestra un ejemplo de cómo eliminar un atributo denominado `rate`:

      ### Java [#java]

      ```java
      boolean attributeRemoved = NewRelic.removeAttribute("rate");
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      val attributeRemoved = NewRelic.removeAttribute("rate")
      ```
