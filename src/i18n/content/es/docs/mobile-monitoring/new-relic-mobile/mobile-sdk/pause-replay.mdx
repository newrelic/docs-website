      ## Sintaxis [#syntax]

      ### Java [#java]

      ```java
      NewRelic.pauseReplay()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.pauseReplay()
      ```

      ## Requisitos

      * Agente de Android v7.7.0 o superior.

      ## Descripción

      `newrelic.pauseReplay()` se puede llamar para forzar manualmente una repetición y pausar la grabación.

      Si aún no hay una repetición en curso, este método API no tendrá ningún efecto.

      Si la sesión móvil aún es válida, puede reanudar una reproducción pausada llamando al método de API [`newrelic.recordReplay()`](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/record-replay).

      ## Casos de uso

      * Desea activar manualmente una repetición para detener la grabación en ciertas páginas del sitio o bajo ciertas condiciones, como una página de atención médica, una página de información de tarjeta de crédito, etc.

      ## Ejemplo [#example]

      Este es un ejemplo de cómo forzar una reproducción para pausar la grabación.

      ### Java [#java]

      ```java
      public class MainActivity extends Activity {
        ...
        @Override
        public boolean onOptionsItemSelected(MenuItem item) {
          switch (item.getItemId()) {
            case R.id.pause_replay:
              NewRelic.pauseReplay();
            ...
              return true;
            default:
              return super.onOptionsItemSelected(item);
          }
        }
        ...
      }
      ```

      ```kotlin
      class MainActivity : AppCompatActivity() {

        ...

            binding.fab.setOnClickListener { view ->

                NewRelic.pauseReplay()
                }
            }
          ...
      ```
