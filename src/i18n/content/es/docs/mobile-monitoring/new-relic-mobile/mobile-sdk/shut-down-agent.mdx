      ## Sintaxis [#syntax]

      ### Java [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```

      ## Descripción [#description]

      Esta API proporciona la capacidad de apagar el agente dentro del ciclo de vida actual de la aplicación durante el tiempo de ejecución. Una vez que el agente se ha cerrado, no se puede reiniciar dentro del ciclo de vida actual de la aplicación.

      ## Ejemplo [#example]

      ### Java [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```
    </TabsPageItem>
