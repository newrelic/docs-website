      ## Sintaxis [#syntax]

      ### Java [#java]

      ```java
      NewRelic.removeAllAttributes ()
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.removeAllAttributes()
      ```

      ## Descripción [#description]

      Este método elimina todos los atributos de una sesión. Si desea eliminar un único atributo de sesión, consulte [Eliminar un atributo](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes).

      ## Valores de retorno [#return-values]

      Devuelve `true` si se registró correctamente y `false` si no es así.

      ## Ejemplo [#example]

      ### Java [#java]

      ```java
      boolean attributesRemoved = NewRelic.removeAllAttributes();
      ```

      ### Kotlin [#kotlin]

      ```kotlin
          val attributesRemoved = NewRelic.removeAllAttributes()
      ```
    </TabsPageItem>
