## Sintaxis [#syntax]

### Java [#java]

```java
   NewRelic.withApplicationToken(apptoken).withLaunchActivityName().start(this.getApplication());
```

### Kotlin [#kotlin]

```kotlin
NewRelic.withLaunchActivityName(className: String?)
```

Establezca el nombre de la actividad objetivo para el tiempo de lanzamiento de la aplicación para calcular el tiempo frío y el tiempo caliente.

## Descripción [#description]

De forma predeterminada, el agente de Android New Relic calcula el tiempo frío y el tiempo caliente cuando se carga la actividad predeterminada. Esta API le permite personalizar el nombre de la actividad de lanzamiento objetivo al calcular el tiempo frío y el tiempo caliente.

## Ejemplos [#examples]

### Establecer nombre de actividad

A continuación se muestra un ejemplo de configuración del nombre de la actividad:

### Java [#java]

```java
NewRelic.withLaunchActivityName($activityName)
```

### Kotlin [#kotlin]

```kotlin
NewRelic.withApplicationToken("apptoken").withLaunchActivityName($activityName).start(applicationContext)
```
