      ## Sintaxis [#syntax]

      ### Java [#java]

      ```java
      NewRelic.crashNow(string $message)
      ```

      ### Kotlin [#kotlin]

      ```kotlin
      NewRelic.crashNow(message:String!)
      ```

      ## Descripción [#description]

      Lanza una excepción en tiempo de ejecución de demostración denominada java.lang.RuntimeException para probar los informes de fallos de New Relic.

      Con este método, puede generar una excepción en tiempo de ejecución de demostración denominada `java.lang.RuntimeException` con un mensaje. Este fallo aparecerá en la [página Análisis de fallos](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes).

      ### C objetivo [#objc]

      ```objectivec
      + (void) crashNow:(NSString* __nullable)message;
      ```

      ## Descripción [#description]

      Lanza una excepción en tiempo de ejecución de demostración denominada `NewRelicDemoException` para probar los informes de fallos de New Relic.

      Con este método, puede generar una excepción en tiempo de ejecución de demostración denominada `NewRelicDemoException` con un mensaje. Este fallo aparecerá en [la página de análisis de fallos](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes).
