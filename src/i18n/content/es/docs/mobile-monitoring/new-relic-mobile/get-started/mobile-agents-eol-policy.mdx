## Tiempos de ejecución móviles [#mobile-runtimes]

Para admitir los tiempos de ejecución de Android, New Relic sigue [las políticas de EOL de Google-Android](https://endoflife.date/android).

Para admitir los tiempos de ejecución de iOS, iPadOS, watchOS y tvOS, New Relic sigue [las políticas de EOL de Apple](https://support.apple.com/).
