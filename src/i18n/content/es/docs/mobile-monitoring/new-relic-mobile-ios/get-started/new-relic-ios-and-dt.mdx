## Requisitos [#requirements]

Para utilizar rastreo distribuido, necesitas la versión del agente iOS 7.3.0 o superior. Le recomendamos utilizar el agente más reciente.

## Cómo configurar rastreo distribuido [#setup]

Para los agentes móviles que admiten esta característica, está habilitada de forma predeterminada.

Si prefieres desactivar el rastreo distribuido, consulta [la característica flag de iOS](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags/#dt).

## Habilitar seguimiento infinito [#infinite-tracing]

Si sus aplicaciones móviles tienen algunos servicios posteriores que utilizan nuestro rastreo distribuido tail-based llamado [Infinite Tracing](/docs/distributed-tracing/infinite-tracing/introduction-infinite-tracing), debe habilitar esta característica para sus aplicaciones móviles. Esto garantiza que su tramo raíz (la aplicación móvil de inicio) esté incluido en el resto de los tramos rastreados por Infinite Tracing.

Para configurar esto, consulte los [pasos de Infinite Tracing](/docs/distributed-tracing/infinite-tracing/set-trace-observer) para crear un observador de traza y seleccionar qué aplicaciones móviles (fuentes de datos) deben informar intervalos.

## Encontrar datos [#find-data]

Los tramos móviles aparecen en cualquier [UIde rastreo distribuida de New Relic](/docs/understand-dependencies/distributed-tracing/ui-data/additional-distributed-tracing-features-new-relic-one/) donde esos tramos son parte de una traza.

A continuación se ofrecen algunos consejos para buscar y consultar datos:

* Puede encontrar la traza originada por el usuario final en cualquier [UIde usuario del rastreo distribuido de New Relic](/docs/understand-dependencies/distributed-tracing/ui-data/additional-distributed-tracing-features-new-relic-one).

* En la UI usuario del rastreo distribuido, los intervalos finales del usuario se indican con el

  icono.

* Para ver el atributo de un intervalo, seleccione [un intervalo en la UI](/docs/apm/distributed-tracing/ui-data/understand-use-distributed-tracing-data#span-details).

* Los tramos se informan como [datos de tramos](/attribute-dictionary/?event=Span) y se pueden [consultar en New Relic](/docs/using-new-relic/data/understand-data/query-new-relic-data).

* Consejos de consulta:

  * Consulta por nombre configurando `mobileApp.name` en el nombre de la aplicación móvil.
  * Consulta para traza que contiene al menos una aplicación móvil con `mobileApp.name is not null`.
  * Consulta de traza que contiene al menos una aplicación backend con `appName is not null`.
  * Consulta para traza que contiene tramos móviles y backend combinando las dos condiciones anteriores.

## Resolución de problemas [#troubleshooting]

Si no ve los intervalos finales del usuario o tiene otros problemas de distribución del rastreo, consulte [resolución de problemas](/docs/apm/distributed-tracing/troubleshooting/troubleshooting-missing-trace-data).
