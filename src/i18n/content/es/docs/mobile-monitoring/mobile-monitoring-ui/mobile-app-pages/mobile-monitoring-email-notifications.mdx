<InlinePopover type="mobile" />de New Relic proporciona varios informes enviados por correo electrónico, que incluyen:

* [Notificación de lanzamiento](#version-trends) (también conocido como correos electrónicos de tendencias de versión)
* [Correos electrónicos de informes de fallos](#crash-reports)

## Reciba correos electrónicos sobre el rendimiento de la nueva versión [#version-trends]

El monitoreo de móviles de New Relic le proporciona una página [<DNT>**Release versions**</DNT>](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/release-versions-page) . También puede habilitar los correos electrónicos de tendencias de versiones (también conocidos como notificaciones de lanzamiento). Estos informes brindan una herramienta de comunicación conveniente para compartir, rastrear y revisar el progreso, la adopción y otras tendencias de su aplicación. Cada correo electrónico incluye un enlace a la UI <DNT>**Version trends**</DNT> para ver más detalles.

### Editar la configuración de correo electrónico de tendencias de versión [#version-trend-ui]

Para configurar los informes por correo electrónico que recibe para una aplicación móvil específica, vaya a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; (select an app) &gt; Settings &gt; Application**</DNT> y edite la configuración del correo electrónico de notificación de lanzamiento.

## Correos electrónicos de informes de fallos [#crash-reports]

Cuando su aplicación móvil tiene un nuevo tipo de falla, desea saber qué salió mal para poder evitar ese tipo de error en el futuro. Con los correos electrónicos de informes de fallos, se le notifica cuando se detecta un fallo único.

Para configurar si recibe informes de fallos por correo electrónico para una aplicación móvil específica, vaya a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Settings &gt; Application**</DNT> y edite la configuración del correo electrónico del informe de fallos.
