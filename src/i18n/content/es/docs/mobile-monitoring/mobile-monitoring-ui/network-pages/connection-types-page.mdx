La página <DNT>**Connection Types**</DNT> para <InlinePopover type="mobile"/>incluye gráficos que muestran las tecnologías de conexión de red inalámbrica de su usuario y su impacto en el tiempo de respuesta, tasa de errores y tráfico activo durante el período de tiempo seleccionado. Los datos se agrupan por generación de tecnología: 2G, 3G, etc. Wifi y Desconocido también se incluyen en el desglose de tipos de conexión. Desde aquí puede profundizar en tecnologías inalámbricas específicas como GPRS, HSDPA y LTE.

Para ver los tipos de conexión inalámbrica móvil de su usuario:

1. Vaya a <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DNT>.

   <img
     title="connection types.png"
     alt="connection types.png"
     src="/images/mobile_screenshot-full_connection-types-summary.webp"
   />

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DNT>: utilice esta página para ver, ordenar o profundizar en información detallada sobre el tipo de conexión de red móvil de su usuario por impacto en el tiempo de respuesta, fallas de red y tráfico activo.
</figcaption>

2. Para cambiar la información que aparece (incluido el tiempo de respuesta, los dispositivos activos o los errores de red), seleccione su elección en el menú

   <DNT>
     **Sort by**
   </DNT>

   .

3. Para seleccionar las versiones de la aplicación móvil o el período de tiempo, use el menú

   <DNT>
     [**Versions**](/docs/mobile-apps/versions)
   </DNT>

   y [el selector de tiempo](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) debajo de la barra de menú de New Relic.

4. Para ver detalles de un tipo de conexión específico, seleccione su nombre.

Para profundizar en información detallada, utilice cualquier [característica UI de la plataforma](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) aplicable.
