Session Replay elimina las conjeturas en la resolución de problemas al proporcionar una repetición en video de lo que los usuarios finales vieron en su teléfono o tablet, sincronizada con todos los telemetry data de esa sesión. Esto significa que puede evaluar rápidamente el impacto real de los errores en sus usuarios, identificar cuellos de botella de rendimiento y puntos de fricción de UX, y resolver problemas más rápido sin depender de informes de errores o reproducción local.

Explora las siguientes secciones para aprovechar al máximo Session Replay:

* [Introducción a Session Replay](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/introduction): aprenda cómo funciona Session Replay, qué datos almacena y cómo afecta su retención.
* [Configurar Session Replay](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/set-up-session-replay): verifique los requisitos, habilite Session Replay y configure las tasas de muestreo.
* [Configurar ajustes de privacidad](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/configure-privacy-settings): personalizar las opciones de enmascaramiento y protección de datos.
* [Ver Session Replay en New Relic](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/view-session-replays): acceda y analice las sesiones grabadas, incluidas las repeticiones vinculadas a errores.
* [Gestionar Session Replay y modificar las capacidades](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/manage-session-replay): controle quién puede ver las repeticiones y modificar la configuración.
* [Session Replay avanzada](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/advanced-session-replay): utilice resúmenes de IA, filtros de secuencia, API de grabación manual y configuración de NerdGraph.
* [Resolución de problemas con Session Replay](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/troubleshooting): encuentra soluciones a problemas comunes.
* [Información adicional](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/additional-information): obtenga información sobre el impacto en el rendimiento, el consumo de datos, el cumplimiento de privacidad y los framework compatibles.
