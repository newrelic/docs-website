Este es el primer paso para diagnosticar su problema de Session Replay de New Relic usted mismo.

Si no encuentra su problema en la lista aquí, siempre puede comunicarse con [el soporte de New Relic](https://support.newrelic.com).

## Algunos eventos han caducado y es posible que ya no aparezcan en la línea de tiempo o en la traza de evento [#aged-out-events]

Session Replay tiene una retención de datos más prolongada que los eventos de excepción controlada. Puede encontrar una repetición con otros eventos móviles pero sin eventos de excepción manejada en la traza de evento o representados como un marcador de error. Esto se debe a que esos eventos pasaron la retención, por lo que no aparecerán en una repetición.
