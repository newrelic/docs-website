La capacidad <InlinePopover type="mobile" />incluye un reporte que rastrea la cantidad de dispositivos, sesiones o usuarios que ejecutan su aplicación cada día, semana o mes a lo largo del tiempo. Para ver el reporte: Vaya a <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Mobile &gt; (select an app) &gt; Usage &gt; Active users**</DNT>.

## Detalles de usuarios activos [#details]

De forma predeterminada, los gráficos y las tablas mostrarán dispositivos únicos mensuales. Emplee el desplegable en la barra de filtro para cambiar la ventana de tiempo a semanal o diaria o para cambiar la métrica a sesiones o usuario. Además, puede emplear la versión, la versión del sistema operativo, el grupo de dispositivos o filtros personalizados para una mayor segmentación.

Para ver el número total de dispositivos, sesiones o usuarios para cualquier periodo de tiempo, pase el cursor sobre la línea en el gráfico de líneas. El valor del periodo de tiempo actual es un valor de mes a la fecha, de semana a la fecha o de día a la fecha y no indica el uso del mes, la semana o el día completos.

La tabla de usuarios activos muestra el periodo de tiempo elegido, según lo especificado por el filtro seleccionado, la métrica única y una comparación con el periodo de tiempo anterior. Por ejemplo, si se selecciona `Monthly` , la tabla mostrará los últimos 12 meses, los datos únicos para esos meses y un aumento o disminución porcentual respecto del mes anterior.

Para obtener información detallada, haga clic en una fila de la tabla. Esto abrirá una vista apilada con un gráfico de líneas detallado para ese periodo de tiempo.

## Seguimiento de dispositivos [#tracking]

Monitoreo de móviles no utiliza identificador de hardware para el seguimiento único de instalaciones.

* En iOS usamos la propiedad IdentifierForVendor. (Las versiones 5.3.4 e inferiores utilizaron la biblioteca SecureUDID).
* En Android generamos un GUID único cuando se instala la aplicación.
