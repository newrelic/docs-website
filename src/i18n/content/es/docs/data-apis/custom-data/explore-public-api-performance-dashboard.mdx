El dashboard de rendimiento de API pública es un dashboard listo para usar que se incluye como parte de su cuenta de New Relic. Proporciona información general procesable y valiosa sobre el rendimiento de las API públicas y una oportunidad para que los nuevos clientes prueben nuestras capacidades de paneles antes de agregar sus propios datos.

El dashboard funciona mostrando la latencia real experimentada por una muestra anónima de clientes de New Relic al acceder a API públicas populares.

<Callout variant="important">
  El conjunto de datos de Global Performance se presenta tal cual. El conjunto de datos de Global Performance representa un conjunto de muestras de una variedad de fuentes, y New Relic no hace ningún esfuerzo por intentar confirmar la exactitud, integridad o veracidad de los datos. No se debe confiar en estos datos como la única fuente de información para cualquier propósito que pueda utilizarlos, y New Relic no es responsable de las decisiones tomadas basándose en estos datos. Global Performance conjunto de datos no debe verse como un respaldo o una recomendación por parte de New Relic de las tecnologías representadas en el conjunto de datos.
</Callout>

## Agregue el dashboard en New Relic [#Add-dashboard]

Para agregar el dashboard de rendimiento de API pública:

1. Desde la página del panel, busque el dashboard prediseñado de Rendimiento de API pública. Puede hacer clic en el enlace

   <DNT>
     **View All**
   </DNT>

   junto al popular título del panel prediseñado para buscarlo.

2. Seleccione el dashboard.

3. Seleccione una cuenta del menú desplegable.

4. Haga clic en

   <DNT>
     **View dashboard**
   </DNT>

   para agregarlo a su cuenta.

## Explora el dashboard [#explore-dashboard]

Puede personalizar su nuevo dashboard prediseñado. ¡Puede cambiar el nombre y la configuración, agregar contenido nuevo y un nuevo widget, refinar los gráficos y más! Consulte [Administrar su panel](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/) para obtener más información.

A continuación se presentan algunas sugerencias sobre cómo explorar el dashboard de rendimiento de la API pública.

* Haga clic en el

  <Icon name="fe-more-horizontal"/>

  en la esquina derecha de cualquiera de los gráficos y seleccione

  <DNT>
    **View query**
  </DNT>

  para ver la consulta [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) utilizada para crear el gráfico.

* Haga clic en el

  <Icon name="fe-more-horizontal"/>

  en la esquina derecha de cualquiera de los gráficos y seleccione

  <DNT>
    **Get as image**
  </DNT>

  para ver o descargar cualquier gráfico como imagen.

* Seleccione un dominio específico del gráfico de barras o agregue un filtro haciendo clic en el campo de texto en la parte superior de la página.

* Si ya ha agregado sus propios datos, experimente copiando consulta y modificándolos para su propio uso.

<Callout variant="important">
  El dashboard de rendimiento de API pública no está disponible actualmente para los clientes de la UE.
</Callout>

<Callout variant="important">
  Actualmente, el dashboard de rendimiento de la API pública no admite alerta.
</Callout>

## Datos reportados [#data-reported]

Ponemos los datos de API públicos a disposición de todas las organizaciones de New Relic a través del evento `Public_APICall` . Estos datos no cuentan para su ingesta. Si desea ver la estructura de esos datos, ejecute esta consulta:

```sql
FROM Public_APICall select keyset()
```
