La tecnología [SAP ®](https://www.sap.com/) es fundamental para impulsar su proceso empresarial clave. Los problemas relacionados con sus sistemas SAP pueden causar ineficiencias e interrupciones cuya identificación y resolución cuestan tiempo y dinero. Esto, a su vez, puede afectar su capacidad para generar ingresos o entregar sus productos y servicios.

Con New Relic, puede monitor sus sistemas SAP con una vista centralizada de su infraestructura, aplicación y proceso empresarial. Esto ayuda a sus equipos de TI a brindar un mejor soporte al negocio con conocimiento temprano y resoluciones más rápidas.

## Requisitos [#requirements]

Para obtener observabilidad de SAP, comuníquese con su representante de cuenta de New Relic.

Para utilizar esta característica, debe ser un [usuario de plataforma completa](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type) con [capacidades relacionadas con la carga de trabajo](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/#capability-definitions).

Esta característica tiene un cargo adicional por ID del sistema SAP (SID). Un ID del sistema SAP (SID) es un identificador de tres letras que representa una instancia única de un sistema SAP. Cada sistema SAP de producción y preproducción se cuenta como un SID independiente. Una base de datos HANA independiente también se cuenta como SID. Un solo SID puede contener múltiples recursos, como múltiples instancias de servidor de aplicaciones, y aun así contar como un SID.

Para obtener observabilidad de SAP, comuníquese con su representante de cuenta de New Relic.

## ¿Por qué elegir nuestra integración SAP? [#why]

Si es una organización empresarial mediana o grande, necesita herramientas SAP confiables para administrar sus operaciones comerciales y las relaciones con los clientes. ¿Por qué elegir la integración de New Relic con SAP?

### Más fácil de incorporar

Nuestra integración certificada con SAP es fácil de instalar y probar como complemento de SAP. Además, ofrecemos visualización llave en mano de cuadros de mando, trazas y mapas, además de detección y alerta de anomalías. El monitoreo de New Relic para soluciones SAP es una herramienta completa de observabilidad centralizada para la infraestructura, aplicación y proceso empresarial de SAP.

### Aprovecha sus inversiones existentes en SAP

A diferencia de otros proveedores, New Relic no depende de conectores de terceros, que normalmente requieren la instalación de un agente de monitoreo en los servidores de producción de SAP. En cambio, el monitoreo de New Relic para SAP no es invasivo, ya que aprovecha las fuentes de datos de SAP existentes para acceder a todos telemetry data necesarios. Es por eso que los equipos de soporte de SAP adoptan nuestra solución, ya que aprovecha sus inversiones existentes en SAP.

### No se necesitan actualizaciones específicas de la versión

Algunas soluciones de SAP se aplican solo a versiones o situaciones específicas, lo que resulta en actualizaciones y coordinación frecuentes y que requieren mucho tiempo. A diferencia de esas soluciones, nuestra integración de SAP sobrevive a las actualizaciones ya que utiliza fuentes de datos de nivel inferior en la capa NetWeaver.
