Las capacidades de New Relic informan diferentes [tipos de datos](/docs/using-new-relic/data/understand-data/new-relic-data-types). Un tipo de datos son [los datos de eventos](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data). Los datos del evento se muestran en gráficos de New Relic y otras vistas, y también están disponibles para [consultas](/docs/using-new-relic/data/understand-data/query-new-relic-data).

Conoce más sobre el evento reportado por New Relic:

* [Evento predeterminado de APM](/docs/insights/insights-data-sources/default-data/apm-default-events-insights)
* [Evento predeterminado browser](/docs/insights/insights-data-sources/default-data/browser-default-events-insights)
* [Evento predeterminado de infraestructura](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-events)
* [Evento predeterminado móvil](/docs/insights/insights-data-sources/default-data/mobile-default-events-insights)
* [Evento predeterminado sintético](/docs/insights/insights-data-sources/default-data/synthetics-default-events-insights)
* [Evento NrAuditEvent](/docs/insights/event-data-sources/default-events/query-account-audit-logs-nrauditevent/) para comprender los cambios en su cuenta

Para conocer otros datos, consulte [nuestro diccionario de datos](/attribute-dictionary).
