Google Gemini Code Assist 확장용 뉴렐릭 도구는 성능 성능 최적화, 모니터 보기, 구현 관리, 배포에 맞춰진 지능형 인사이트 및 지침을 제공하며 모두 Gemini Code Assist 내에 완벽하게 통합되어 있습니다.

이 가이드에서는 Gemini Code Assist에서 뉴렐릭 도구를 설치하고 사용하는 과정을 안내합니다.

## Gemini Code Assist에서 뉴렐릭 도구 활성화 [#enable-new-relic-tool]

### 전제 조건 [#prerequisites]

시작하기 전에 원활한 설정 과정을 위해 다음 전제 조건이 충족되었는지 확인하세요.

* 컴퓨터에 통합 개발 환경(IDE)이 설치되었습니다.
* Google Gemini Code Assist에 액세스할 수 있습니다.
* Gemini Code Assist가 IDE에 설치되었습니다. 자세한 내용은 [Google Gemini Code Assist 문서를](https://cloud.google.com/gemini/docs/codeassist/overview) 참조하세요.

<Callout variant="tip" title="팁">
  특정 역할에 대해 Gemini Code Assist에 대한 액세스를 제한하려면 Gemini Code Assist에서 설정을 구성하세요. 특정 역할이나 직원에게 뉴렐릭 AI에 대한 접근을 제한하려면 뉴렐릭의 역할 기반 접근 제어(RBAC)를 사용할 수 있습니다. 자세한 내용은 [뉴렐릭 사용자 관리 를](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts) 참조하세요.
</Callout>

뉴렐릭을 도구를 활성화하려면.

1. IDE에서 Google로 로그인을 클릭하여 [Gemini Code Assist](https://codeassist.google.com) 에 로그인합니다.

2. 성공적으로 인증되면 IDE로 돌아갑니다.

3. <DNT>Gemini Code Assist Chat</DNT> (채팅)에서 `@newrelic` 입력하고 Enter를 누르세요. 뉴렐릭 도구를 활성화해야 합니다.

4. 팀원에서 링크를 클릭하여 뉴렐릭 도구를 활성화하세요. 브라우저 창이 열리면 Gemini Code Assist에서 뉴스 도구를 활성화할 수 있습니다.

5. 뉴렐릭 도구를 활성화한 다음 **Connect** \[연결을] 클릭합니다.

6. 뉴렐릭 계정 정보를 사용하여 로그인하세요. 이것으로 귀하의 뉴렐릭 계정이 Gemini Code Assist에 성공적으로 연결되었습니다.

   이제 IDE의 Gemini Code Assist 채팅 확장 프로그램에서 뉴렐릭 도구를 사용할 수 있습니다.

## 뉴렐릭 도구와 상호 작용 [#new-relic-tool]

Gemini Code Assist의 뉴렐릭 도구는 다양한 작업을 지원하고 귀중한 인사이트를 제공할 수 있습니다. 뉴렐릭 도구와 상호 작용하려면 채팅 입력란에 다음과 같이 메신저를 입력하세요.

`@newrelic <your message>`

여기서 `<your message>` 도구에서 실행하려는 작업입니다. 예를 들어, `@newrelic What is apdex?`
