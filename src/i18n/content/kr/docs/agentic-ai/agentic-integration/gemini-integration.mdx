Google Gemini Code Assist 확장용 뉴렐릭 에이전트는 성능 성능 최적화, 모니터 보기, 구현 관리, 배포에 맞춰진 지능형 인사이트 및 지침을 제공하며 모두 Gemini 내에 원활하게 통합됩니다.

## Gemini Code Assist에 뉴렐릭 에이전트 설치 [#install-new-relic-agent]

### 전제 조건 [#prerequisites]

시작하기 전에 원활한 설정 과정을 위해 다음 전제 조건이 충족되었는지 확인하세요.

* IDE가 시스템에 설치되어 있습니다.
* Google Gemini Code Assist에 액세스할 수 있습니다.
* Gemini Code Assist가 IDE에 설치되었습니다. 자세한 내용은 [Google Gemini Code Assist 문서를](https://cloud.google.com/gemini/docs/codeassist/overview) 참조하세요.

<Callout variant="tip" title="팁">
  특정 역할에 대해 Gemini Code Assist에 대한 액세스를 제한하려면 Gemini Code Assist에서 설정을 구성하세요. 특정 역할이나 직원에게 뉴렐릭 AI에 대한 접근을 제한하려면 뉴렐릭의 역할 기반 접근 제어(RBAC)를 사용할 수 있습니다. 자세한 내용은 [뉴렐릭 사용자 관리 를](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts) 참조하세요.
</Callout>

**Gemini Code Assist에서 뉴렐릭 에이전트를 활성화하려면:**

1. <DNT>Google Cloud Console</DNT> 에서 <DNT>**NewRelic**</DNT> 을 검색하고 확장 프로그램을 활성화하세요. 로그인 대화 상자 창이 열립니다.

2. 로그인하려면 NewLick 자격 증명을 제공하세요. **New Relic successfully linked.** \[뉴렐릭이 성공적으로 링크되었습니다.] 메시지가 표시됩니다.

   이제 IDE의 Gemini Code Assist 채팅 확장 프로그램 내에서 뉴렐릭 에이전트를 사용할 수 있습니다. 자세한 내용은 [Gemini Code Assist에서 뉴렐릭 에이전트 사용을](#communicate-with-new-relic-agent-in-gemini-code-assist) 참조하세요.

## Gemini Code Assist에서 뉴렐릭 에이전트와 통신 [#communicate-with-new-relic-agent-in-gemini-code-assist]

Gemini Code Assist 내의 뉴렐릭 에이전트는 다양한 작업을 지원하고 귀중한 인사이트를 제공할 수 있습니다. 뉴렐릭 에이전트와 상호작용하려면 채팅 입력 상자에 `@newrelic` 입력하세요.

### 채팅 메시지 구조 [#chat-message-structure]

뉴렐릭 에이전트와 상호작용하려면 다음과 같이 사용자를 입력하세요:

`@newrelic <your message>`

어디에:

* `@newrelic`: 뉴렐릭 에이전트와의 결합작용.
* `<your message>`: 처리를 위해 에이전트에게 제출하는 작업입니다.
