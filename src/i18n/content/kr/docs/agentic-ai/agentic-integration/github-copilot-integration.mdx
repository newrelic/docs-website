GitHub Copilot 확장용 뉴렐릭 에이전트는 지능형 인사이트 및 성능 최적화, 모니터 보기, 개체 관리, 배포에 맞춰진 지침을 제공하며 모두 GitHub Copilot 내에 완벽하게 통합되어 있습니다.

이 가이드에서는 GitHub Copilot 확장 기능 내에서 뉴렐릭 에이전트를 설치하고 사용하는 과정을 안내합니다.

## GitHub Copilot에 뉴렐릭 에이전트 설치 [#install-new-relic-agent]

### 전제 조건 [#prerequisites]

시작하기 전에 원활한 설정 과정을 위해 다음 전제 조건이 충족되었는지 확인하세요.

* IDE가 시스템에 설치되어 있습니다.
* [GitHub Copilot](https://github.com/features/copilot) 에 액세스할 수 있습니다.
* GitHub Copilot 확장 프로그램이 IDE에 설치되었습니다. 자세한 내용은 [GitHub Copilot 문서를](https://docs.github.com/en/copilot/managing-copilot/configure-personal-settings/installing-the-github-copilot-extension-in-your-environment) 참조하세요.
* 귀하의 조직에서 [GitHub Copilot 확장 정책이](https://docs.github.com/en/enterprise-cloud@latest/copilot/managing-copilot/managing-github-copilot-in-your-organization/managing-policies-for-copilot-in-your-organization#setting-a-policy-for-github-copilot-extensions-in-your-organization) 활성화되었습니다.

<Callout variant="tip" title="팁">
  특정 역할에 대해 GitHub Copilot에 대한 액세스를 제한하려면 GitHub에서 설정을 구성하세요. 특정 역할이나 직원에게 뉴렐릭 AI에 대한 접근을 제한하려면 뉴렐릭의 역할 기반 접근 제어(RBAC)를 사용할 수 있습니다. 자세한 내용은 [뉴렐릭 사용자 관리 를](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts) 참조하세요.
</Callout>

**GitHub Copilot에 뉴렐릭 에이전트를 설치하려면:**

1. GitHub MarketPlace에서 [뉴렐릭 에이전트](https://github.com/marketplace/newrelic-copilot-extension) 로 이동합니다.
2. **Add &gt; Install it for free**를 클릭하세요.

에이전트를 설치한 후에는 IDE의 GitHub Chat 확장 프로그램에서 사용할 수 있습니다. 자세한 내용은 [Copilot에서 뉴렐릭 에이전트 사용을](#communicate-with-new-relic-agent-in-gitHub-copilot) 참조하세요.

## GitHub Copilot에서 뉴렐릭 에이전트와 통신 [#communicate-with-new-relic-agent-in-gitHub-copilot]

Github Copilot 내의 뉴렐릭 에이전트는 다양한 작업을 지원하고 귀중한 인사이트를 제공할 수 있습니다. 뉴렐릭 에이전트와 상호작용하려면 Copilot 채팅 입력 상자에 `@newrelic` 입력하세요.

### 뉴렐릭 에이전트에 로그인 [#sign-in]

뉴렐릭 에이전트와 처음으로 대화를 시작하면 로그인하게 됩니다. 지시에 따라 로그인 과정을 완료하세요.

### 채팅 메시지 구조 [#chat-message-structure]

뉴렐릭 에이전트와 상호작용하려면 다음과 같이 사용자를 입력하세요:

`@newrelic <your message>`

어디에:

* `@newrelic`: 뉴렐릭 에이전트와의 결합작용.
* `<your message>`: 처리를 위해 에이전트에게 제출하는 작업입니다.
