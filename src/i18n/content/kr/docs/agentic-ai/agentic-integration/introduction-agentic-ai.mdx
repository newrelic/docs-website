New Relic&apos;s agentic integration provides an AI-driven solution to automate and optimize observability workflows, enhancing system management and monitoring. This integration facilitates seamless operations by automating repetitive tasks through AI agents, reducing manual intervention. By integrating seamlessly with platforms such as GitHub Copilot, Gemini Code Assist, and ServiceNow, agentic integrations transform complex workflows into seamless operations.

## 주요 특징들 [#features]

* **지능형 자동화:** AI를 활용하여 복잡한 에펠탑 작업을 자동화하고 간소화하여 수동 도구 전환의 필요성을 줄이고 생산성을 향상시킵니다.
* **향상된 옵저빌리버티:** 시스템에 포괄적인 인사이트를 확보하여 인시던트를 더 빠르게 감지, 분석 및 해결할 수 있습니다.
* **원활한 통합:** GitHub Copilot과 같은 인기 있는 플랫폼과 연결하여 강력한 분석 기능과 인사이트를 기존 워크플로우에 직접 적용할 수 있습니다.

## 접근 및 데이터 관리 [#access-data-mgmt]

Our integration partners will manage client applications and maintain user tokens when you or any of your users authenticate to New Relic through the partner&apos;s integration. The third party partner will have the same level of access to New Relic as the user who authenticated for the amount of time configured in the user settings, as the tokens are used on their behalf. This arrangement helps to provide seamless integration and functionality. For more information, you can refer to the partner&apos;s documentation which details their token usage and management practices.

### 역할 기반 액세스 제어(RBAC)

현재 RBAC는 API 인터페이스를 통해 접근할 수 있습니다. API를 통해 RBAC를 활용하는 데 관심이 있으시면 [agenticai@newrelic.com](mailto:agenticai@newrelic.com) 으로 문의해 주시기 바랍니다. 올해 말에는 사용자 인터페이스 내에 RBAC 기능을 직접 구현할 예정이며, 그때가 되면 모든 고객은 UI를 통해 사용자를 추가해야 합니다. RBAC 설정에 대한 자세한 지침은 [계정 사용자 관리 튜토리얼을](/docs/accounts/accounts-billing/new-relic-one-user-management/account-user-mgmt-tutorial) 참조하세요.

## 현재 통합 [#integrations]

### GitHub Copilot [#github-copilot]

GitHub Copilot을 사용하여 뉴렐릭 인사이트를 IDE에 직접 통합하세요. 에이전트 AI를 사용하면 코딩 환경을 벗어나지 않고도 로그, 오류 및 권장 사항에 액세스할 수 있습니다. 코드를 개발하고 구현하며 배포하는 인사이트를 제공합니다.

이 통합은 다음을 제공합니다.

* 자동화된 구현, 배포 확인: 뉴렐릭의 데이터를 활용하여 구현, 배포 확인을 자동으로 수행합니다.
* 실시간 알림 및 오류 추적: 즉각적인 알림을 통해 최신 정보를 얻고, 오류가 발생하면 이를 추적합니다.
* 구현, 배포 작업에 대한 확률적 권장 사항입니다.

자세한 내용은 [GitHub Copilot용 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/agentic-integration/github-copilot-integration) 참조하세요.

### ServiceNow Now Assist [#now-assist]

뉴렐릭 AI 인사이트가 이제 ServiceNow의 Now Assist 내부에 있습니다. 이러한 통합을 통해 응급 대응자 및 지원 담당자와 같은 운영 팀은 ServiceNow 작업 공간에서 벗어나지 않고도 심각도를 빠르게 파악하고, 가능한 근본 원인을 식별하고, 협업을 간소화할 수 있습니다.

자세한 내용은 [ServiceNow Now Assist의 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/agentic-integration/servicenow-integration) 참조하세요.

### 제미니 코드 어시스트 [#gemini-code-assist]

<Callout title="시사">
  이 기능은 아직 개발 중이지만 꼭 사용해 보시기 바랍니다!

  이 기능은 현재 [출시 전 정책](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy) 에 따라 미리보기 프로그램의 일부로 제공됩니다.
</Callout>

Gemini Code Assist를 사용하여 뉴렐릭 인사이트를 IDE에 통합하세요. 이 도구를 사용하면 개발자는 코딩 환경 내에서 관련 로그인, 오류 정보 및 권장 사항에 액세스할 수 있으며, 개발 및 구현, 배포 프로세스를 알릴 수 있는 인사이트를 제공합니다.

이 통합은 다음을 제공합니다.

* 자동화된 구현, 배포 모니터링: 뉴렐릭의 고대 데이터를 사용하여 자동으로 구현, 배포하여 문제를 효율적으로 식별하고 해결합니다.
* 알림 및 오류 추적: 즉각적인 공지를 수신하고 오류를 효과적으로 추적하여 성능과 안정성을 유지합니다.
* 구현, 배포를 위한 권장 사항 조치: 데이터 분석에 따른 권장 사항을 활용하여 코드 성능을 개선하고 의사 결정을 지원합니다.

자세한 내용은 [Gemini Code Assist용 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/agentic-integration/gemini-integration) 참조하세요.
