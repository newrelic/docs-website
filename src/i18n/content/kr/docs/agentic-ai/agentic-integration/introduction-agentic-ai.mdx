뉴렐릭의 Agentic 통합은 AI 기반 솔루션을 제공하여 옵저버빌리티활동을 자동화하고 최적화하여 시스템 관리 및 감시를 강화합니다. 이러한 통합은 AI 에이전트를 통해 반복적인 작업을 자동화하여 원활한 운영을 촉진하고 수동 개입을 줄입니다. GitHub Copilot, Amazon Q Business, Gemini Code Assist, ServiceNow 등의 플랫폼과 완벽하게 통합하여 Agentic은 복잡한 앨리스우를 원활한 운영으로 통합합니다.

## 주요 특징들 [#features]

* **지능형 자동화:** AI를 활용하여 복잡한 에펠탑 작업을 자동화하고 간소화하여 수동 도구 전환의 필요성을 줄이고 생산성을 향상시킵니다.
* **향상된 옵저빌리버티:** 시스템에 포괄적인 인사이트를 확보하여 인시던트를 더 빠르게 감지, 분석 및 해결할 수 있습니다.
* **원활한 통합:** GitHub Copilot과 같은 인기 있는 플랫폼과 연결하여 강력한 분석 기능과 인사이트를 기존 워크플로우에 직접 적용할 수 있습니다.

## 현재 통합 [#integrations]

### GitHub Copilot [#github-copilot]

GitHub Copilot을 사용하여 뉴렐릭 인사이트를 IDE에 직접 통합하세요. 에이전트 AI를 사용하면 코딩 환경을 벗어나지 않고도 로그, 오류 및 권장 사항에 액세스할 수 있습니다. 코드를 개발하고 구현하며 배포하는 인사이트를 제공합니다.

이 통합은 다음을 제공합니다.

* 자동화된 구현, 배포 확인: 뉴렐릭의 데이터를 활용하여 구현, 배포 확인을 자동으로 수행합니다.
* 실시간 알림 및 오류 추적: 즉각적인 알림을 통해 최신 정보를 얻고, 오류가 발생하면 이를 추적합니다.
* 구현, 배포 작업에 대한 확률적 권장 사항입니다.

자세한 내용은 [GitHub Copilot용 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/agentic-integration/github-copilot-integration) 참조하세요.

### 아마존 Q 비즈니스 [#amazon-q]

Amazon Q Business와 뉴렐릭 AI의 통합은 보다 효율적인 문제 해결, 해결 및 옵저버빌리티 개선을 촉진하도록 설계되었습니다. 지원 엔지니어가 문제를 신속하게 진단하고 해결할 수 있게 되면서 이러한 통합은 사용자 체크아웃을 포함한 중요한 비즈니스 프로세스의 무결성을 유지하는 데 도움이 됩니다.

이 통합은 다음을 제공합니다.

* 대화형 AI 인터페이스: Amazon Q Business 대화형 인터페이스를 통해 뉴럴릭 AI와 소통하여 인사이트 및 추천을 받으세요.
* 발행, 목표 공지 분석: 정의된 기간 내에 체크아웃 및 결제 처리와 같은 주요 서비스에 대한 알림을 분리하여 조사합니다.
* 종합적인 공지 인텔리전스: 이상 행위, 오류 빈도 및 지연 시간 문제에 대한 심층 보고서에 액세스하고 추가 분석을 위해 뉴롤릭 대시보드 링크로 보완됩니다.
* 선제적 권장 사항: 잠재적 문제를 사전에 해결하기 위해 추가 조사가 필요하다는 점에 대한 정보에 입각한 제안을 받습니다.

자세한 내용은 [Amazon Q Business용 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/agentic-integration/amazonq-integration) 참조하세요.

### 제미니 코드 어시스트 [#gemini-code-assist]

Gemini Code Assist를 사용하여 뉴렐릭 인사이트를 IDE에 통합하세요. 이 도구를 사용하면 개발자는 코딩 환경 내에서 관련 로그인, 오류 정보 및 권장 사항에 액세스할 수 있으며, 개발 및 구현, 배포 프로세스를 알릴 수 있는 인사이트를 제공합니다.

이 통합은 다음을 제공합니다.

* 자동화된 구현, 배포 모니터링: 뉴렐릭의 고대 데이터를 사용하여 자동으로 구현, 배포하여 문제를 효율적으로 식별하고 해결합니다.
* 알림 및 오류 추적: 즉각적인 공지를 수신하고 오류를 효과적으로 추적하여 성능과 안정성을 유지합니다.
* 구현, 배포를 위한 권장 사항 조치: 데이터 분석에 따른 권장 사항을 활용하여 코드 성능을 개선하고 의사 결정을 지원합니다.

자세한 내용은 [Gemini Code Assist용 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/agentic-integration/gemini-integration) 참조하세요.

## 도구 [#tools]

각 에이전트 통합에는 뉴렐릭 플랫폼 기능을 사용하고 옵저버빌리티 인사이트, 알림 및 예측을 제공하는 다양한 도구가 함께 제공됩니다.

* **[Change Intelligence Tool](/docs/agentic-ai/tools/change-intelligence-tool)** \[변경 인텔리전스 도구]: 변경 인텔리전스 도구는 포괄적인 변경 분석과 예측을 제공하여 잠재적 위험을 이해하고 완화하는 데 도움이 됩니다. 이 도구를 사용하면 시스템 변경의 영향을 즉시 파악할 수 있습니다. 이는 구현, 배포 및 설정된 수정과 관련된 잠재적 위험을 평가하는 데 도움이 됩니다. 자세한 내용은 [Change Intelligence Tool을](/docs/agentic-ai/tools/change-intelligence-tool) 참조하세요.

* **[Alert Intelligence Tool](/docs/agentic-ai/tools/alert-intelligence-tool)** \[공지 인텔리전스 도구]: 공지 인텔리전스 도구는 인시던트 문제를 신속하게 해결하기 위한 고급 공지 분석 기능을 제공합니다. 연장 공지와 즉각적인 추천을 제공하여 에스컬레이션을 방지합니다. 자세한 내용은 [Alert Intelligence Tool](/docs/agentic-ai/tools/alert-intelligence-tool) 참조하세요.
