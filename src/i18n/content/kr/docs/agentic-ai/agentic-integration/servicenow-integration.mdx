뉴렐릭 ServiceNow의 Now Assist와 AI 통합은 운영자, 최초 대응자, 지원 에이전트 등 ServiceNow 사용자를 지원합니다.

* 다른 서비스와 사용자에 미치는 영향을 이해하여 인시던트의 심각성을 측정합니다.
* 사건의 근본 원인 이론 식별
* 문제 완화를 위해 협력자에게 에스컬레이션을 용이하게 합니다.

이 가이드에서는 ServiceNow의 Now Assist에서 뉴렐릭 AI를 설치하고 사용하는 과정을 안내합니다.

## 설정 [#setup]

ServiceNow에서 뉴렐릭 AI 확장을 사용하려면 다음이 필요합니다.

* 뉴렐릭 [사용자 API 키](/docs/apis/intro-apis/new-relic-api-keys/#overview-keys). 뉴렐릭 사용자가 뉴렐릭에서 삭제되면 해당 사용자 키도 비활성화되어 더 이상 작동하지 않습니다. 안전하고 중단 없는 통합을 보장하려면 다음 권장 사항을 참고하세요.

  * 개인 또는 개별 계정이 종료될 때 중단을 방지하려면 `service user` 계정을 사용하세요.
  * `user keys` 여러 계정에 대한 액세스를 제공하지만 각 계정은 해당 계정이 생성된 단일 계정에 연결됩니다. 계정이 삭제되면 해당 계정과 연결된 모든 사용자 키가 무효화됩니다. 따라서 키를 생성하는 계정의 유효 기간이 장기인지 확인하세요.
  * 사용자 키가 사용된 사용자가 읽기 전용 권한을 가지고 있는지 확인하세요.

  <Callout variant="tip" title="메모">
    제공된 사용자 키를 통해 ServiceNow 계정 내의 모든 사용자가 이 정보에 액세스할 수 있습니다.
  </Callout>

  * Now Assist에서 뉴렐릭 AI를 구성하려면 [여기에 설명된 단계를 따르세요.](https://www.servicenow.com/docs/csh?topicname=now-assist-itom-use-aia.html&version=latest)

## Now Assist를 통해 뉴렐릭 AI와 상호 작용하세요 [#new-relic-tool]

ServiceNow에서:

1. <DNT>**All &gt; Event Management &gt; Express List**</DNT> 으로 이동합니다.

2. 공지를 선택하고 출처가 뉴렐릭인지 확인하세요.

3. <DNT>Now Assist</DNT> 아이콘을 클릭하여 패널을 엽니다.

4. 지금 문의하기 공지에 대한 질문을 도와주세요.

5. 지원되는 프롬프트 중 하나를 입력하세요. 원하는 대로 섞어서 사용할 수도 있습니다.

   * 이 공지의 심각도는 무엇입니까?
   * 어떤 대학이 영향을 받나요?
   * 가능성 있는 원인 이론은 무엇인가?
   * 누구를 조사에 참여시켜야 하나요?
   * 최근 구현, 배포가 있나요?
   * 공지의 영향은 무엇이고, 가능한 원인 이론은 무엇이며, 누구를 조사에 참여시켜야 합니까?

   <Callout variant="tip" title="팁">
     Now Assist 패널의 대화는 2단계에서 선택한 공지에 한해 진행됩니다. 다른 공지에 대해 질문하려면 해당 공지를 열고 Now Assist 아이콘을 선택하여 새 대화를 시작하세요.
   </Callout>
