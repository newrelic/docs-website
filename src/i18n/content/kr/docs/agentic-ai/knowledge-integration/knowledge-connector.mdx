뉴렐릭 AI Knowledge 커넥터는 Confluence와 같은 데이터 소스와의 1-클릭 통합을 지원합니다.

## 전제 조건

Confluence 커넥터가 올바르게 작동하려면 Confluence API 토큰에 다음 범위가 필요합니다.

* `read:confluence-content.all`
* `read:confluence-space.summary`
* `read:content:confluence`
* `read:content-details:confluence`

또한 URL 스키마는 다음과 같아야 합니다: [https://api.atlassian.com/ex/confluence/cloud\_id](https://api.atlassian.com/ex/confluence/cloud_id)

## 커넥터 설정

Confluence 콘텐츠 인덱싱을 시작하고 뉴렐릭 AI를 통해 지식 커넥터의 이점을 누리려면 다음 단계를 따르세요.

<Steps>
  <Step>
    <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **Integrations &amp; Agents**</DNT> 으로 이동합니다.
  </Step>

  <Step>
    <DNT>**NRAI Knowledge Connectors**</DNT> 을(를) 검색하세요.
  </Step>

  <Step>
    <DNT>**Confluence NRAI Knowledge Connector**</DNT> 선택합니다.
  </Step>

  <Step>
    다음과 같은 커넥터 세부 정보를 입력하세요.

    <table>
      <thead>
        <tr>
          <th>
            분야 명
          </th>

          <th>
            설명
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            커넥터 이름
          </td>

          <td>
            커넥터에 고유한 이름을 입력하세요(예: <DNT>**Demo connector**</DNT>.
          </td>
        </tr>

        <tr>
          <td>
            지식 카테고리
          </td>

          <td>
            드롭다운 목록에서 지식 범주를 선택하면 AI가 올바른 위치에서 정보를 찾는 데 도움이 됩니다.
          </td>
        </tr>
      </tbody>
    </table>
  </Step>

  <Step>
    <DNT>**Continue**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    <DNT>**Data source authentication**</DNT> 페이지에서 데이터 소스를 인증하는 데 필요한 정보를 입력하세요. <DNT>**Continue**</DNT> 클릭하세요.
  </Step>

  <Step>
    데이터 소스 구성 페이지에서 필요한 정보를 입력하고 어떤 문서를 어떤 빈도로 가져와야 하는지 정의합니다. <DNT>**Create**</DNT> 클릭하세요.
  </Step>

  <Step>
    설정이 성공적으로 완료되면 커넥터 개요 페이지에서 커넥터 상태를 확인할 수 있습니다.

    <table>
      <thead>
        <tr>
          <th>
            분야 명
          </th>

          <th>
            설명
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            상태
          </td>

          <td>
            데이터 소스를 뉴렐릭 AI에서 사용할 수 있는지 여부를 보여줍니다.
          </td>
        </tr>

        <tr>
          <td>
            마지막 동기화
          </td>

          <td>
            데이터가 마지막으로 업데이트된 시점을 표시합니다.
          </td>
        </tr>

        <tr>
          <td>
            동기화 중
          </td>

          <td>
            이 옵션을 끄면 새로운 데이터가 업데이트되지 않습니다. 기존 데이터는 변경되지 않습니다.
          </td>
        </tr>

        <tr>
          <td>
            커넥터 삭제
          </td>

          <td>
            커넥터를 삭제하면 인덱싱된 모든 데이터가 삭제됩니다.
          </td>
        </tr>
      </tbody>
    </table>
  </Step>
</Steps>
