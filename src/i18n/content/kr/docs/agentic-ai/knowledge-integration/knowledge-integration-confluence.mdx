뉴렐릭의 지식 커넥터(검색 및 생성) 기능을 confluence와 통합하면 뉴렐릭 AI 응답 내의 합류 콘텐츠를 직접 참조하여 문제 해결, 해결 및 문서화 프로세스를 향상하는 데 도움이 됩니다. 뉴렐릭에 API 키와 합류 쿼리를 제공하면 모든 NRAI 경험에 이 정보를 활용하여 즉시 AI 인사이트를 얻을 수 있습니다.

## 지식 커넥터를 Confluence와 연결하세요

뉴럴릭 AI를 통해 Confluence 콘텐츠 인덱싱을 시작하고 지식 커넥터의 이점을 활용하려면 다음 단계를 따르세요.

<Steps>
  <Step>
    <DNT>[**one.newrelic.com**](https://one.newrelic.com) &amp;gt; **Integrations &amp; Agents**</DNT> 으로 이동합니다.
  </Step>

  <Step>
    <DNT>**RAG Manager**</DNT> 을(를) 검색하세요.
  </Step>

  <Step>
    <DNT>**Add Confluence**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    <DNT>**Tools**</DNT> 탭에서는 사용 가능한 모든 RAG 도구를 볼 수 있습니다.

    <Callout variant="important">
      색인한 모든 문서를 보려면 문서 탭을 클릭하세요.
    </Callout>
  </Step>

  <Step>
    <DNT>**Add Tools &gt; Confluence**</DNT> 클릭하여 합류 커넥터를 설정하세요.
  </Step>

  <Step>
    드롭다운에서 Confluence 자격 증명을 저장할 <DNT>**Account**</DNT> 을 선택하고 <DNT>**Continue**</DNT> 클릭합니다.
  </Step>

  <Step>
    <DNT>**Confluence Base URL**</DNT> 필드에 Confluence 인스턴스의 기본 URL을 입력하세요. 예를 들어, <DNT>[https://your-site-company.atlassian.net/wiki](https://your-site-company.atlassian.net/wiki)</DNT> 클릭하고 <DNT>**Continue**</DNT> 클릭합니다.
  </Step>

  <Step>
    다음 정보를 입력하십시오.

    * <DNT>**Confluence Username**</DNT>: 귀하의 합류 사용자 이름입니다.
    * <DNT>**API Token**</DNT>: Confluence API 토큰입니다.
  </Step>

  <Step>
    <DNT>**Save token in secret store**</DNT> 클릭하고 <DNT>**Continue**</DNT> 클릭하세요.
  </Step>

  <Step>
    합류 설정 섹션에서 다음 정보를 입력하세요.

    1. <DNT>**Name your cofluence documents**</DNT>, Confluence 문서에 대한 고유한 이름을 입력하세요(예: <DNT>**Incident Runbook**</DNT>.

    2. <DNT>**Confluence query**</DNT> 을 입력하여 내용을 구체화하세요. 예: <DNT>**space=SPACEKEY**</DNT>.

    3. **Confluence reindex interval** \[Confluence 재색인 간격] 섹션에서 합류 콘텐츠를 검색하고 재색인하는 빈도를 선택하세요. 다음 옵션 중에서 선택할 수 있습니다.

       * <DNT>**Monthly**</DNT>
       * <DNT>**Weekly**</DNT>
       * <DNT>**Daily**</DNT>

    4. <DNT>**Save confluece settings**</DNT> 클릭하고 <DNT>**Continue**</DNT> 클릭하세요.
  </Step>

  <Step>
    <DNT>**Start Indexing**</DNT> 클릭하세요. 새로운 도구가 생성되어 도구 탭에 추가됩니다.
  </Step>

  <Step>
    <DNT>**Document**</DNT> 탭을 클릭하면 색인한 문서를 볼 수 있습니다.
  </Step>
</Steps>
