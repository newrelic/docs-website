## OAuth 흐름이 시작되지 않습니다

* 브라우저가 `localhost` 에 액세스할 수 있는지 확인하세요.
* 방화벽 설정을 확인하세요.
* 도구가 `OAuth` URL에 액세스할 수 있는지 확인하세요.

## API 키가 작동하지 않습니다

* 키 형식을 확인합니다( `NRAK-` 으로 시작).
* 뉴렐릭에서 주요 권한을 확인하세요.
* 키가 만료되지 않았는지 확인하세요.

## MCP 서버가 감지되지 않았습니다

* 애플리케이션을 완전히 다시 시작하세요.
* `JSON` 구문 유효성을 확인하세요.
* 파일 경로와 권한을 확인하세요.

## mcp-remote 프록시 문제

* `Node.js` 설치되어 있는지 확인하세요.
* 인터넷 연결을 확인하세요.
* npx가 `mcp-remote` 패키지에 액세스할 수 있는지 확인하세요.

## 오류 메시지

* **연결이 거부되었습니다**. 도구의 URL과 네트워크 연결을 확인하세요.
* **인증 필요**: API 키가 여전히 활성 상태이고 올바르게 입력되었는지 확인하거나 `OAuth` 흐름을 완료하세요.
* **잘못된 클라이언트**: `OAuth` 설정 문제, `mcp-remote` 프록시를 시도하세요.
* **`Time.UnmarshalJSON`**: `OAuth` 에 대해 `mcp-remote` 프록시를 사용합니다.
