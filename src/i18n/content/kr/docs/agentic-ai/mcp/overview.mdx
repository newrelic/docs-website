모델 컨텍스트 프로토콜(MCP)은 AI 모델이나 에이전트가 특정 시스템이나 환경에 대한 관련 정보(컨텍스트)를 수신하는 표준적인 방법입니다. 이러한 맥락은 AI가 현재 상황을 이해하고, 작업을 더 정확하게 수행하며, 더 유용한 응답을 제공하는 데 도움이 됩니다. 역사적으로 AI 에이전트를 관찰 플랫폼과 같은 다양한 데이터 소스에 연결한다는 것은 각 AI 에이전트가 각 플랫폼과 통신하기 위한 맞춤형, 종종 복잡한 통합을 구축하는 것을 의미했습니다. 이러한 단편화된 접근 방식은 상당한 개발 및 유지 관리 오버헤드로 이어지며, 이는 AI 기반 인사이트의 &quot;통합 확산&quot;의 한 형태입니다.

뉴렐릭 AI MCP 서버는 이러한 과제를 직접 해결합니다. 이 프로토콜은 뉴렐릭에서 구현되었으며, 단일 중앙 집중식 브리지 역할을 합니다. 여러 개의 개별 통합을 구축하는 대신, 지원되는 개발 도구 내의 AI 에이전트는 이제 뉴렐릭의 관찰 플랫폼에서 풍부한 컨텍스트와 강력한 도구에 연결하고 검색할 수 있는 일관된 단일 방법을 갖게 되었습니다.

AI 개발 환경 내에서 자연어를 사용해 직접 질문하고 데이터를 조사할 수 있습니다. MCP 서버는 귀하의 requests 번역하고, 관련 지표나 로그를 검색하며, 더 명확한 답변과 요약을 제공합니다.

<Callout title="규정 준수 제한 사항" variant="important">
  보안 및 규정 준수 요구 사항으로 인해 계정이나 데이터가 FedRAMP 또는 HIPAA 규정 준수 의무에 해당하는 경우 뉴렐릭 AI MCP 서버를 사용해서는 안 됩니다. FedRAMP 또는 HIPAA 규제 계정에서 이 공개 미리보기 기능을 사용하는 것은 엄격히 금지되어 있습니다.
</Callout>

## 그것이 당신에게 어떻게 도움이 되는가

MCP 서버는 엔지니어와 운영 팀에 실질적인 이점을 제공하며 다음과 같은 방식으로 귀하와 AI에 도움을 줍니다.

* **직접 통합:** 개발 환경에서 뉴렐릭의 플랫폼과 일부 AI 기능에 직접 액세스하세요. 이렇게 하면 여러 도구 사이를 전환하는 일이 없어집니다.
* **더 빨라진 문제 해결, 해결:** AI가 제공하는 인사이트를 통해 문제를 빠르게 찾아 수정하고, 오류 로그를 분석하고, 가능한 원인을 파악합니다.
* **자동화된 작업:** 공지 상태 확인, 인지던트 보고서 생성, 구현 및 배포가 시스템에 미치는 영향 분석 등의 작업을 수행합니다.
* **간단한 데이터 쿼리:** 일반 영어로 질문을 작성하면 서버가 이를 뉴렐릭 쿼리 언어(NRQL)로 변환하여 NRQL 에 익숙하지 않더라도 결과를 얻을 수 있습니다.

## 지원되는 AI 개발 도구 [#supported-tools]

뉴렐릭 MCP는 다음 AI 개발 환경과 호환됩니다.

* **Claude Code:** Claude용 명령줄 인터페이스
* **Claude Desktop:** 대화형 AI 개발을 위한 데스크톱 애플리케이션
* **VS Code:** MCP 지원이 포함된 통합 개발 환경
* **Windsurf:** 클라우드 기반 AI 개발 플랫폼
* **Gemini CLI:** Gemini용 명령줄 인터페이스

각 환경은 귀하의 선호도에 따라 고유한 이점을 제공합니다. 플랫폼별 구성 지침은 [설정 가이드를](/docs/agentic-ai/mcp/setup) 참조하세요.

## 전제 조건 [#prerequisites]

뉴렐릭 MCP를 사용하려면 다음이 필요합니다.

* API 액세스가 가능한 새로운 계정
* `Node.js` 설치됨(일부 인증 방법에 필요)
* MCP 지원 클라이언트 중 하나가 설치됨

<Callout title="그룹 멤버십 및 역할" variant="important">
  MCP 서버에 액세스하려면 다음 특정 [조직 역할](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#role-scopes) 중 하나가 할당된 그룹의 구성원이어야 합니다: `Organization Read Only`, `Organization Manager` 또는 `Organization Product Admin`. 액세스를 시도하기 전에 이 역할이 활성화되어 있는지 확인하세요. 현재 귀하의 그룹에 이러한 역할이 하나도 없는 경우 `Authentication Domain Manager` 권한이 있는 조직 관리자에게 문의하여 해당 역할을 부여받으세요.
</Callout>

### 인증 옵션

* **사용자 API 키**:

  1. 뉴렐릭에 로그인하세요.
  2. **사용자 메뉴 -&gt; API keys** 선택하세요.
  3. 새로운 사용자 키를 생성하거나 기존 키를 복사하세요.
  4. 키 형식: `NRAK-XXXXXXXXXXXXXXXXXXXXX`.

* **OAuth**: 뉴렐릭은 사전 구성된 OAuth 엔드포인트를 제공합니다.

  * 클라이언트 ID: `pUWGgnjsQ0bydqCbavTPpw==`
  * 승인 URL: `https://login.newrelic.com/login`
  * 토큰 URL: `https://mcp.newrelic.com/oauth2/token` (미국) 및 `https://mcp.eu.newrelic.com/oauth2/token` (EU)
  * 범위: `["openid"]`

## 공개 미리보기 액세스

뉴렐릭 UI 의 미리 보기 및 평가판 페이지로 이동하여 뉴렐릭 AI MCP 서버 미리 보기를 활성화합니다.

1. 뉴렐릭 UI 에서 왼쪽 하단에 있는 사용자 이름을 클릭하세요.
2. **Administration &gt; Previews &amp; Trials** 선택합니다.
3. 뉴렐릭 AI MCP 서버 미리보기를 활성화합니다.
