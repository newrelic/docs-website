뉴렐릭의 Agentic 통합은 AI 기반 솔루션을 제공하여 옵저버빌리티활동을 자동화하고 최적화하여 시스템 관리 및 감시를 강화합니다. 이러한 통합은 AI 에이전트를 통해 반복적인 작업을 자동화하여 원활한 운영을 촉진하고 수동 개입을 줄여줍니다. GitHub Copilot, Amazon Q, Gemini Code Assist, ServiceNow 등의 플랫폼과 완벽하게 통합함으로써 Agentic은 복잡한 앨리스우를 원활한 운영으로 전환합니다.

## 주요 특징들 [#features]

* **지능형 자동화:** AI를 활용하여 복잡한 에펠탑 작업을 자동화하고 간소화하여 수동 도구 전환의 필요성을 줄이고 생산성을 향상시킵니다.
* **향상된 옵저빌리버티:** 시스템에 포괄적인 인사이트를 확보하여 인시던트를 더 빠르게 감지, 분석 및 해결할 수 있습니다.
* **원활한 통합:** GitHub Copilot과 같은 인기 있는 플랫폼과 연결하여 강력한 분석 기능과 인사이트를 기존 워크플로우에 직접 적용할 수 있습니다.

## 현재 통합 [#integrations]

### GitHub Copilot [#github-copilot]

GitHub Copilot을 사용하여 뉴렐릭 인사이트를 IDE에 직접 통합하세요. 에이전트 AI를 사용하면 코딩 환경을 벗어나지 않고도 로그, 오류 및 권장 사항에 액세스할 수 있습니다. 코드를 개발하고 구현하며 배포하는 인사이트를 제공합니다.

이 통합은 다음을 제공합니다.

* 자동화된 구현, 배포 확인.
* 실시간 알림 및 오류 추적.
* 구현, 배포 작업에 대한 확률적 권장 사항입니다.

자세한 내용은 [GitHub Copilot용 뉴렐릭 에이전트 시작하기를](/docs/agentic-ai/github-copilot-integration) 참조하세요.

## 도구 [#tools]

각 에이전트 통합에는 뉴렐릭 플랫폼 기능을 사용하고 옵저버빌리티 인사이트, 알림 및 예측을 제공하는 다양한 도구가 함께 제공됩니다.

* **[변경 인텔리전스 도구:](/docs/agentic-ai/change-intelligence-tool)** 변경 인텔리전스 도구는 포괄적인 변경 분석과 예측을 제공하여 잠재적 위험을 이해하고 완화하는 데 도움이 됩니다. 이 도구를 사용하면 시스템 변경의 영향을 즉시 파악할 수 있습니다. 이는 구현, 배포 및 설정된 수정과 관련된 잠재적 위험을 평가하는 데 도움이 됩니다. 자세한 내용은 [Change Intelligence Tool을](/docs/agentic-ai/change-intelligence-tool) 참조하세요.

* **[공지 인텔리전스 도구(Knowledge Intelligence Tool](/docs/agentic-ai/alert-intelligence-tool)) :** 공지 인텔리전스 도구는 인시던트 문제를 신속하게 해결하기 위한 고급 공지 분석 기능을 제공합니다. 연장 공지와 즉각적인 추천을 제공하여 에스컬레이션을 방지합니다. 자세한 내용은 [공지 Intelligence Tool을](/docs/agentic-ai/alert-intelligence-tool) 참조하세요.

## 가격 [#pricing]

귀하의 Agentic 통합 사용은 사용자 유형에 관계없이 고급 컴퓨터 제품으로 주문한 내용에 따라 미리 보기 중에 청구될 수 있습니다. Advanced 컴퓨트 제품 기능의 경우, 관련 뉴렐릭 쿼리 언어 쿼리(NRQLS)도 Advanced 컴퓨트로 청구할 수 있습니다. 뉴렐릭은 정식 출시되면 Agentic 통합을 고급 컴퓨트 제품 기능으로 출시할 예정입니다.

가격에 대한 자세한 내용은 당사 영업팀에 문의하시거나 주문서를 참조하세요.
