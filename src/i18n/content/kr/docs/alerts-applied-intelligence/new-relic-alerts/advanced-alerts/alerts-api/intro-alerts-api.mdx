New Relic의 경고 및 적용된 인텔리전스 기능을 프로그래밍 방식으로 관리하기 위해 API를 사용하는 두 가지 옵션이 있습니다.

* NerdGraph(GraphQL): 데이터 쿼리 및 구성을 위한 New Relic의 기본 권장 API입니다. 우리는 항상 NerdGraph에 기능을 추가하고 있습니다. [경고 관련 NerdGraph 문서를 참조하십시오.](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/alerts-nerdgraph/nerdgraph-api-examples)
* REST API: New Relic의 오리지널 API입니다. 우리는 이 API에 대해 최소한의 작업을 수행하므로 NerdGraph를 사용하는 것이 좋습니다. 현재 REST API를 사용 중이라면 NerdGraph로 전환하는 것이 좋습니다. [알림 관련 REST API 문서를 참조하세요.](/docs/alerts-applied-intelligence/new-relic-alerts/advanced-alerts/rest-api-alerts/rest-api-calls-alerts)
