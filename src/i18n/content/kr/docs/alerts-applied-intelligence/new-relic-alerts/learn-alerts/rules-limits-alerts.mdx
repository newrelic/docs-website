이 문서에는 New Relic 경고 기능에 대한 몇 가지 기술 규칙 및 제한 사항이 포함되어 있습니다.

New Relic 조직에 여러 계정이 **있는 경우 해당 계정은 다른 계정의 알림 정책을 상속하지 않습니다.** 각 계정에 대해 개별적으로 알림 정책을 생성해야 합니다.

다음 규칙은 New Relic 사용자 인터페이스와 REST API(v2)에 모두 적용됩니다.
