이벤트 히트맵은 표시된 메트릭과 동일한 시간 범위 내에서 발생하는 인프라 [이벤트](/docs/infrastructure/new-relic-infrastructure/infrastructure-ui-pages/infrastructure-events-page) 의 스냅샷을 제공합니다. 히트맵의 색상이 어두울수록 해당 기간 동안 더 많은 이벤트가 발생했습니다.

히트맵을 인프라 페이지의 차트와 비교하여 생태계의 문제를 빠르게 찾아낼 수 있습니다. 예를 들어, CPU 스파이크가 크게 발생하는 경우 해당 시간 범위의 이벤트 히트맵을 클릭하여 원인이 된 이벤트를 찾을 수 있습니다. 거기에서 더 깊이 파고들어 실제 문제를 파악할 수 있습니다.
