New Relic은 시스템 모듈, 구성 파일, 메타데이터, 패키지, 서비스, 사용자 세션 등을 포함하여 호스트별 시스템 구성에 대한 자세한 정보를 수집할 수 있습니다. **인벤토리** 페이지는 각 호스트의 구성에 대한 실시간, 필터링 및 검색 가능한 보기를 제공합니다.

**인벤토리** 페이지에서는 다음을 수행할 수 있습니다.

* 버전 업데이트가 모든 호스트에 성공적으로 적용되었는지 확인하십시오.
* 호스트 간의 버전 불일치를 감사합니다.
* 보안 취약점을 수정하기 위해 업데이트가 필요한 호스트를 빠르게 식별합니다.

인벤토리 데이터를 보고 검색하려면: **[one.newrelic.com](http://one.newrelic.com) > Infrastructure > Inventory** 로 이동합니다.

## 페이지 기능

**인벤토리** 페이지 기능을 사용하여 호스트의 특정 항목에 대한 정보를 찾습니다.

<CollapserGroup>
  <Collapser
    id="filter"
    title="데이터 필터링"
  >
    [필터 세트](/docs/infrastructure-organize-your-resources-scopes) 를 사용하여 특정 기준과 일치하는 호스트만 표시합니다.
  </Collapser>

  <Collapser
    id="search"
    title="인벤토리 검색"
  >
    검색 기능을 사용하여 인벤토리 항목을 검색합니다.

    예를 들어 OpenSSL과 관련된 정보를 찾으려면 `openssl` 을(를) 검색하십시오. 검색어가 인벤토리 항목 이름과 다시 일치합니다.
  </Collapser>

  <Collapser
    id="view"
    title="인벤토리 항목 세부 정보 보기"
  >
    인벤토리 항목 세부 정보는 New Relic [인벤토리 항목 이름](#item-naming) 에 따라 상주하는 각 호스트에 대한 호스트 및 시스템 정보를 제공합니다. 다른 호스트에 동일한 항목의 다른 버전이 있는 경우 New Relic은 이를 감지하고 인벤토리 페이지에서 **변형 호스트** 레이블로 플래그를 지정하고 각 버전을 실행하는 각 호스트를 나열합니다.

    항목 세부 정보는 해당 소스에서 지시하는 속성(키/값 쌍)입니다. 특정 속성은 일반적으로 시간이 지남에 따라 안정적이지만 새로운 속성이 추가되거나 더 이상 사용되지 않을 수 있습니다. 속성은 각 인벤토리 항목의 핵심인 중요한 메타데이터를 전달합니다.

    일반적인 인벤토리 항목 속성은 다음과 같습니다.

    * 변형 호스트(호스트 이름)
    * 아키텍처
    * 설명
    * 필수적인
    * 우선순위
    * 상태
    * 버전
  </Collapser>

  <Collapser
    id="alerts"
    title="호스트의 경보 임계값 위반 보기"
  >
    하나 이상의 호스트 [경보 임계값 위반](/docs/alerts/new-relic-alerts/getting-started/alerts-glossary#alert-threshold) 을 보려면 호스트의 **Critical** 을 선택하십시오. <Icon name="fe-x-circle"/>아이콘 또는 **경고** <Icon name="fe-alert-triangle"/>상.
  </Collapser>
</CollapserGroup>

## 인벤토리 데이터 보존 [#data-retention]

인벤토리 데이터는 실시간입니다. 호스트가 보고를 중지하면 인벤토리 데이터가 최대 24시간 동안 계속 표시됩니다.

## 차트 데이터 속성 [#attributes]

**인벤토리** 페이지를 채우는 데 사용되는 속성에 대한 기술적인 설명은 [기본 인프라 속성 및 이벤트](/docs/infrastructure/new-relic-infrastructure/data-instrumentation/default-infrastructure-attributes-events#shared-attributes) 를 참조하십시오. 여기에는 [운영 체제별 일반적인 이벤트](/docs/infrastructure/new-relic-infrastructure/data-instrumentation/default-infrastructure-attributes-events#events-os) 요약이 포함됩니다.
