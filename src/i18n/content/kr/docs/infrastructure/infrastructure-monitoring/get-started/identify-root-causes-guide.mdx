당신은 분산 인프라에 배포된 사이트인 Geek's Movie Shop의 새로운 IT 엔지니어입니다. 귀하의 전임자들은 사이트의 서버를 계측하지 않았기 때문에 귀하는 그들이 어떻게 행동하는지에 대해 많이 알지 못하지만 최근 사용자 불만은 귀하의 시스템을 관찰하기 시작할 때임을 나타냅니다.

이 실습에서는 New Relic 인프라 모니터링을 사용하여 서버를 관찰하여 문제를 식별하고 완화할 수 있습니다.

## 목표

* New Relic 인프라 에이전트로 호스트 계측
* 인프라 및 관련 서비스에서 인사이트 수집
* 인프라의 근본 원인 문제 식별
