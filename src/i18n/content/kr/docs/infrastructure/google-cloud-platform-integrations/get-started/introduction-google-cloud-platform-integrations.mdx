[New Relic 인프라](/docs/infrastructure/new-relic-infrastructure/getting-started/introduction-new-relic-infrastructure) 통합은 인기 있는 제품 및 서비스의 성능을 모니터링합니다. New Relic의 Google Cloud Platform(GCP) 통합을 통해 여러 New Relic 기능에서 GCP 데이터를 모니터링할 수 있습니다.

## GCP와 New Relic 연결 [#connect]

GCP 데이터를 얻으려면 표준 절차에 [따라 GCP 서비스를 New Relic에 연결하십시오](/docs/infrastructure/gcp-integrations) .

## GCP 데이터 보기 [#insights]

구성 프로세스를 따르면 Google Cloud Platform 계정의 데이터가 New Relic에 직접 보고됩니다.

GCP 데이터를 보려면 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Infrastructure > GCP** 로 이동합니다. 나열된 통합의 경우:

* 사전 구성된 대시보드에서 데이터를 보려면 통합 이름을 선택하십시오.

  또는

* **데이터 탐색** 선택 <Icon name="fe-database"/>아이콘을 클릭하여 GCP 데이터를 봅니다.

미리 구성된 두 가지 모두에서 NRQL 쿼리를 보고 재사용할 수 있습니다. <InlinePopover type="dashboards"/>**이벤트 탐색기** 대시보드에서. 이를 통해 특정 요구 사항에 맞게 쿼리를 조정할 수 있습니다.

이벤트, 대시보드 및 기타 데이터는 New Relic의 모든 서비스에서 사용할 수 있습니다.
