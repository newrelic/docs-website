[](/docs/infrastructure/introduction-infra-monitoring)[Google Cloud Platform (GCP)](https://cloud.google.com/) 과의 파트너십에는 데이터를 당사 제품에 보고하는 것이 포함됩니다.[Google Cloud Functions](https://cloud.google.com/functions/) 이 문서에서는 GCP Cloud Functions 통합을 활성화하는 방법을 설명하고 보고할 수 있는 데이터에 대해 설명합니다.

## 특징

Google Cloud Functions 서비스를 사용하면 서버리스 방식으로 코드를 실행할 수 있습니다. [Google UI](https://cloud.google.com/functions/docs/quickstarts) 를 사용하여 개발자는 특정 기능을 수행하기 위한 짧은 코드를 만들 수 있습니다. 그러면 이 기능은 애플리케이션 서버나 런타임 환경을 관리할 필요 없이 클라우드 이벤트에 응답할 수 있습니다.

## 통합 활성화 [#activate]

통합을 활성화하려면 표준 절차에 [따라 GCP 서비스를 New Relic에 연결하십시오](/docs/connect-google-cloud-platform-services-infrastructure) .

## 폴링 빈도 [#polling]

통합은 통합에 따라 달라지는 폴링 간격에 따라 GCP 서비스를 쿼리합니다.

* GCP Cloud Functions의 폴링 빈도: 5분
* 해결: 1분에 하나의 데이터 포인트

## 데이터 보기 및 사용 [#find-data]

통합을 활성화한 후 ( [폴링 빈도](#polling) 에 따라) 몇 분 정도 기다리면 데이터가 UI에 표시됩니다. <InlinePopover type="dashboards" />링크 및 공지 설정을 포함한 [데이터를 보고 사용](/docs/infrastructure/integrations/find-use-infrastructure-integration-data) 하려면 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Infrastructure &gt; GCP**</DNT> 로 이동한 다음 통합을 선택하세요.
