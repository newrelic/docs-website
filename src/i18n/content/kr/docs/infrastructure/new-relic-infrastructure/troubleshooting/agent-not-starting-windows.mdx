## 문제 [#problem]

뉴렐릭 인프라 에이전트를 Windows 서버에 설치했는데 UI 에 데이터가 표시되지 않습니다. <DNT>**Windows Services**</DNT> 에서 서비스를 수동으로 시작하려고 하면 다음과 같은 오류와 함께 실패합니다.

```
Windows could not start the New Relic infrastructure agent service on Local Computer
Error 1067: The process terminated unexpectedly
```

## 해결책 [#solution]

바이러스 백신 또는 보안 제품으로 인해 New Relic 인프라 에이전트가 서비스로 실행되지 않을 수 있습니다. 에이전트가 문제 없이 작동하는지 확인하려면 명령줄에서 `newrelic-infra.exe` 파일을 관리자로 실행하고 호스트가 예상대로 데이터를 보고하는지 확인합니다. 이 경우 보안 팀과 협력하여 서비스를 허용 목록에 추가해야 하는지 또는 추가 매개변수로 구성해야 하는지 확인하십시오.

<Callout variant="important">
  에이전트 위치와 이를 수정하는 방법을 읽으려면 [에이전트 디렉터리 구성 방법을](/docs/infrastructure/install-infrastructure-agent/windows-installation/zip-manual-install-infrastructure-agent-windows/#agent-directory) 참조하세요.
</Callout>
