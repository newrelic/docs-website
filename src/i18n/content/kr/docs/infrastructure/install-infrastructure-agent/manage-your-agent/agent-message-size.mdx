인프라 에이전트 [버전 1.0.989](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes) 이후 최대 메시지 크기는 1MB(10^6바이트)이며 이벤트, 메트릭 및 인벤토리 통합이 불명확하게 포함될 수 있습니다. 이전에는 인벤토리 데이터가 3MB로 제한되었지만 최대 메시지 크기는 5MB였습니다.
