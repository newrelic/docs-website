## tarball 파일을 사용하여 설치 [#custom-setups-tarball]

<DNT>**custom**</DNT> 설정 시나리오의 경우 지원 모드 또는 [](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-assisted-install-infrastructure-linux)수동 [모드에서 tarball 파일을](https://download.newrelic.com/infrastructure_agent/binaries/linux/) [사용하여 인프라 에이전트를](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-manual-install-infrastructure-linux) 설치할 수 있습니다.

이는 기본 설치 설정을 환경에 맞게 조정해야 하거나 저장소에 `newrelic-infra` 패키지가 없는 배포판에 인프라 에이전트를 설치해야 할 때 특히 유용합니다.

<Callout variant="important">
  tarball 파일을 사용한 인프라 에이전트의 사용자 지정 설치는 공식적으로 지원되지 않습니다.
</Callout>
