## tarball 파일을 사용하여 설치 [#custom-setups-tarball]

For <DoNotTranslate>**custom**</DoNotTranslate> setup scenarios, you can install the infrastructure agent using [our tarball files](https://download.newrelic.com/infrastructure_agent/binaries/linux/) in [assisted](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-assisted-install-infrastructure-linux) or [manual](/docs/infrastructure/install-configure-manage-infrastructure/linux-installation/tarball-manual-install-infrastructure-linux) modes.

This is especially useful when you need to adapt the default installation settings to your environment, or to install the infrastructure agent on distributions that lack the `newrelic-infra` package in their repositories.

<Callout variant="important">
  tarball 파일을 사용한 인프라 에이전트의 사용자 지정 설치는 공식적으로 지원되지 않습니다.
</Callout>
