## 원인

일부 메트릭은 일반적인 클라우드 공급자 API(CloudWatch, Stackdriver, Azure Monitor)에 없으며 대신 서비스 API에서 가져옵니다. 각 클라우드 서비스 제공자는 데이터를 처리하고 서비스와 상호 작용하는 고유한 서비스 API를 가지고 있습니다. 예를 들어 AWS CloudWatch에 지표가 없는 경우 New Relic은 AWS ECS 서비스 API에서 지표를 가져옵니다.
