[인프라 에이전트](/docs/infrastructure/new-relic-infrastructure/installation/uninstall-infrastructure-agent-or-integrations) 를 제거해도 [인프라 통합](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure) 이 자동으로 제거되지는 않습니다.

[AWS Metric Stream](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/) 또는 기타 AWS 통합을 제거하려면 아래 단계를 따르십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "250px" }}>
        <DNT>
          **If you want to...**
        </DNT>
      </th>

      <th>
        <DNT>
          **Do this**
        </DNT>
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td id="disable-aws-integrations">
        AWS CloudWatch 지표 스트림 통합 비활성화
      </td>

      <td>
        AWS CloudWatch 지표 스트림 통합을 비활성화하려면:

        1. AWS 콘솔에서 AWS CloudWatch 스트림, AWS Kinesis Data Firehouse 엔드포인트 및 AWS S3 백업 버킷을 포함하여 AWS 통합 설정 중에 생성된 모든 필수 리소스를 제거합니다. AWS CloudFormation 템플릿을 사용한 경우 단계 [에 따라 AWS CloudFormation 콘솔에서 스택을 삭제합니다](https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/cfn-console-delete-stack.html) .

        2. AWS CloudWatch 지표 스트림을 연결한 각 활성 AWS 리전에 대해 1단계를 완료합니다.

        3. <DNT>
             **Infrastructure > AWS > AWS Status Dashboard**
           </DNT>

           에서 뉴렐릭에 새로운 지표 업데이트가 수집되지 않는지 확인하세요.

        4. [AWS 계정 연결을 해제](#unlink-aws) 하는 단계를 완료합니다.
      </td>
    </tr>

    <tr>
      <td id="disable-aws-integrations">
        하나 이상의 AWS 폴링 통합 비활성화
      </td>

      <td>
        AWS 계정을 New Relic에 연결된 상태로 유지하면서 서비스를 비활성화하려면:

        1. <DNT>
             **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure**
           </DNT>

           에서

           <DNT>
             **AWS > Manage services**
           </DNT>

           선택합니다.

        2. <DNT>
             **Edit AWS account**
           </DNT>

           페이지에서 비활성화하려는 각 활성 서비스의 확인란을 선택 취소하세요.

        3. 변경 사항을 저장합니다.
      </td>
    </tr>

    <tr>
      <td id="unlink-aws">
        AWS 계정 연결 해제
      </td>

      <td>
        AWS 계정을 New Relic에서 완전히 연결 해제하려면 AWS 계정의 연결을 해제해야 합니다. 그러면 해당 AWS 계정과 연결된 모든 New Relic 통합이 비활성화됩니다.

        1. <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > AWS > Manage services**</DNT> 으)로 이동합니다.

        2. <DNT>**Edit AWS account**</DNT> 페이지에서 <DNT>**Unlink this account**</DNT> 선택합니다.

        3. 변경 사항을 저장합니다.

        4. AWS에 로그인하고 [<DNT>**Services > IAM > Roles**</DNT>](https://console.aws.amazon.com/iam/home#roles) 선택합니다.

        5. 삭제하려는 역할의 확인란을 선택한 다음 <DNT>**Role Actions > Delete Role**</DNT> 선택합니다.

           AWS 계정 연결을 해제하면 ARN을 통해 설정된 신뢰 관계가 비활성화됩니다.
      </td>
    </tr>
  </tbody>
</table>
