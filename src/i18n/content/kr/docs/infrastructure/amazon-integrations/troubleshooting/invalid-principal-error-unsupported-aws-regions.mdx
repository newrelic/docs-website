## 문제

[AWS를 인프라 통합에](/docs/integrations/amazon-integrations/get-started/connect-aws-services-infrastructure#connect) 연결할 때 <DNT>**Review**</DNT> 단계 중에 AWS 오류 `Invalid principal in policy` 이 발생합니다.

## 해결책

설치하려는 통합 유형을 확인하십시오. New Relic은 중국 리전을 제외한 모든 [AWS 리전](https://docs.aws.amazon.com/general/latest/gr/rande.html) 에 위치한 서비스 및 엔드포인트에서 모니터링 데이터를 얻을 수 있습니다.

중국을 사용하는 계정에서 통합을 구성하려고 할 때 이 오류가 표시됩니다.
