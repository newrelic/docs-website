[New Relic 인프라 통합에는](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) 지표 데이터를 New Relic으로 전송하는 AWS DataSync 통합이 포함됩니다.

이 문서에서는 통합 기능, 활성화 방법 및 보고할 수 있는 데이터에 대해 설명합니다.

## 특징

통합을 사용하여 원격 측정 데이터를 수집하고 [DataSync](https://aws.amazon.com/datasync/)에서 New Relic으로 보냅니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰할 수 있습니다.

## 통합 활성화

이 통합을 활성화하려면 [AWS 서비스를 New Relic에 연결하는](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)방법을 참조하십시오.

## 데이터 찾기 및 사용

통합의 지표를 찾으려면 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> 으로 이동하여 `aws.DataSync` 로 필터링하세요.
