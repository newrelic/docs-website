뉴렐릭은 Amazon Web Services (AWS) 와 통합되어 [AWS Transfer Family](https://aws.amazon.com/aws-transfer-family/) Indicator 및 기타 데이터를 뉴렐릭에 보고합니다.

이 문서에서는 통합을 활성화하는 방법과 보고되는 데이터에 대해 설명합니다.

## 특징 [#features]

통합을 사용하여 AWSTransfer 제품군에서 텔리메트리 데이터를 수집하고 뉴렐릭으로 보냅니다. 서비스를 모니터링하고, 유입 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 파악하세요.

## 통합 활성화 [#activate-integration]

이 통합은 CloudWatch Metric Streams를 통해 사용할 수 있습니다.

이 통합을 활성화하려면 통해 [AWS 서비스를 뉴렐릭에 연결하는 방법을 CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 참조하세요.

## 데이터 찾기 및 사용 [#find-use-data]

통합의 지표를 찾으려면 <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics &amp; Events**</DNT> 으로 이동하여 `aws.transfer` 로 필터링하세요.

## 측정항목 데이터 [#metric-data]

이 New Relic 인프라 통합은 다음 Amazon Transfer Family 데이터를 수집합니다.
