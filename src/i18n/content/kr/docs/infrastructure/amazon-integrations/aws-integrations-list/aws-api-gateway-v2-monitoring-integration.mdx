[New Relic 인프라 통합에는](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) 이제 AWS API Gateway v2에 대한 통합이 포함되어 해당 지표 데이터를 New Relic으로 보냅니다.

이 문서에서는 통합 기능, 활성화 방법, 보고할 수 있는 데이터에 대해 설명합니다.

## 특징 [#features]

통합 기능을 사용하여 [Amazon API Gateway v2](https://aws.amazon.com/api-gateway/) 에서 텔레메트리 데이터를 수집하고 New Relic으로 보내세요. 서비스를 확인하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰하세요.

## 통합 활성화 [#activate-integration]

이 통합을 활성화하려면 [AWS 서비스를 New Relic에 연결하는](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)방법을 참조하십시오.

## 데이터 찾기 및 사용 [#find-use-data]

통합의 지표를 찾으려면 <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Metrics & Events**</DNT> 으로 이동하여 `aws.apigateway` 로 필터링하세요.

## 측정항목 데이터 [#metric-data]

이 New Relic 인프라 통합은 다음 Amazon API Gateway v2 지표를 수집합니다.
