New Relic 인프라에는 이제 지표 데이터를 New Relic으로 전송할 수 있는 AWS Certificate Manager 통합이 포함됩니다. 이 문서에서는 통합 기능, 활성화 방법, 보고에 사용할 수 있는 데이터에 대해 설명합니다.

## 특징

통합 기능을 사용하여 [AWS Certificate Manager](https://aws.amazon.com/certificate-manager/) 에서 텔메트리 데이터를 수집하고 New Relic으로 보냅니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰할 수 있습니다.

## 통합 활성화

이 통합을 활성화하려면 [Amazon CloudWatch Metric Streams 통합 문서](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 의 지침을 따르십시오.

## 데이터 찾기 및 사용

통합 지표를 찾으려면 다음 안내를 따르세요.

1. [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)으로 이동

2. 선택하다

   <DNT>
     **Metrics and events**
   </DNT>

3. 필터링 기준 `aws.certificatemanager`
