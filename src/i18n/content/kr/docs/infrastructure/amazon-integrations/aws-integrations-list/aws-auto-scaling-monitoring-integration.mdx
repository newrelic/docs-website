<Callout variant="important">
  [AWS CloudWatch Metric Streams 통합](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-metric-stream/) 을 활성화하여 사용자 지정 네임스페이스를 포함하여 AWS 서비스의 모든 CloudWatch 지표를 모니터링합니다. 개별 통합은 더 이상 권장되는 옵션이 아닙니다.
</Callout>

[AWS Auto Scaling](https://aws.amazon.com/autoscaling/) 서비스를 사용하면 Amazon EC2 인스턴스를 자동으로 시작하거나 종료할 수 있습니다. 사용자 정의 정책, 일정 및 상태 확인을 기반으로 Amazon EC2 용량을 동적으로 조정하는 데 도움이 됩니다.

[New Relic 인프라 통합](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) 에는 Auto Scaling 서비스에서 New Relic 제품으로 그룹에 대한 데이터를 보고하는 AWS Auto Scaling 통합이 포함됩니다. 이 문서에서는 이 통합을 활성화하는 방법과 보고할 수 있는 데이터를 설명합니다.

## 통합 활성화 [#activate]

이 통합을 활성화하려면:

1. [AWS 서비스를 New Relic에 연결하려면](/docs/infrastructure/infrastructure-integrations/getting-started/connect-aws-integrations-infrastructure) 표준 절차를 따르십시오.
2. AWS Auto Scaling이 샘플링된 데이터를 1분마다 CloudWatch로 보내도록 [Auto Scaling 그룹 지표를 활성화](https://docs.aws.amazon.com/autoscaling/ec2/userguide/as-instance-monitoring.html#as-group-metrics) 합니다.

콘솔을 사용하여 그룹 측정치를 활성화하려면:

1. [console.aws.amazon.com/ec2/](https://console.aws.amazon.com/ec2/) 에서 Amazon EC2 콘솔을 엽니다.
2. 탐색 창에서 **Auto Scaling Groups > (귀하의 그룹 선택) 을 선택합니다** .
3. **모니터링** 탭에서 **Auto Scaling Metrics > Enable Group Metrics Collection** 또는 **Display > Auto Scaling** 을 선택합니다.

## 구성 및 폴링 [#polling]

AWS Auto Scaling 통합에 대한 [기본 폴링](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) 정보:

* New Relic 폴링 간격: 5분
* Amazon CloudWatch 데이터 간격: 기본적으로 1분

폴링 빈도를 변경하고 데이터를 필터링하려면 [구성 옵션을](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations) 사용하십시오.

## 데이터 찾기 및 사용 [#find-data]

[통합 데이터를 보려면](/docs/infrastructure/integrations/find-use-infrastructure-integration-data) [**one.newrelic.com**](http://one.newrelic.com) **> Infrastructure > AWS** 로 이동한 다음 **AWS Auto Scaling** 통합 링크 중 하나를 선택합니다.

다음 [이벤트 유형](/docs/data-apis/understand-data/new-relic-data-types/#event-data) 을 사용 [하여 데이터를 쿼리하고 탐색할](/docs/using-new-relic/data/understand-data/query-new-relic-data) 수 있습니다.

* `AutoScalingInstanceSample`
* `AutoScalingGroupSample`
* `AutoScalingLaunchConfigurationSample`
* `AutoScalingPolicySample`
* `AutoScalingRegionLimitSample`
