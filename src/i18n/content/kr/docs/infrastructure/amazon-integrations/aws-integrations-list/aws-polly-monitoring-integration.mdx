[뉴렐릭 인프라 통합](/docs/infrastructure/introduction-infra-monitoring) 에는 Polly 지표를 뉴렐릭에 보고하는 <DNT>Amazon Polly</DNT> 에 대한 통합이 포함됩니다.

## 특징 [#features]

이 통합을 사용하면 [<DNT>Amazon Polly</DNT>](https://docs.aws.amazon.com/polly) 서비스에서 텔메트리 데이터를 수집하여 뉴렐릭으로 보낼 수 있습니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰하세요.

## 보고서 데이터 [#report-data]

이 데이터를 보고하는 데는 두 가지 옵션이 있습니다. 권장되는 경로는 [Amazon CloudWatch Metric Streams 통합을](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream)설정하는 것입니다. 또는 [폴링에 의존하는 이전 Amazon 통합을](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations)사용할 수 있습니다.

## 데이터 찾기 및 사용 [#find-use-data]

이 통합의 데이터는 `Metric` 데이터 유형에 연결됩니다. UI에서 이 데이터를 찾으려면: <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> 로 이동하여 `aws.polly`) 필터링합니다.
