뉴렐릭은 Amazon Web Services (AWS) [AWS 데이터베이스 Migration Services(DMS)](https://aws.amazon.com/dms/) 지표 및 기타 데이터를 뉴렐릭에 보고하기 위해 와 통합됩니다.

이 문서에서는 통합을 활성화하는 방법과 보고되는 데이터에 대해 설명합니다.

## 특징 [#features]

통합을 사용하여 AWS 데이터베이스 마이그레이션 서비스에서 텔레메트리 데이터를 수집하고 뉴렐릭으로 보냅니다. 서비스를 모니터링하고, 유입 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 파악하세요.

## 통합 활성화 [#activate]

이 통합은 CloudWatch MetricStreams를 통해 사용할 수 있습니다.

이 통합을 활성화하려면 통해 [AWS 서비스를 뉴렐릭에 연결하는 방법을 CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 참조하세요.

## 데이터 찾기 및 활용 \[find-data]

통합의 지표를 찾으려면 <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> 으로 이동하여 `aws.dms` 로 필터링하세요.

또한 [one.newrelic.com](https://one.newrelic.com/all-entities) 으로 이동하여 `dms` 을 검색하면 데이터베이스 마이그레이션 서비스와 관련된 엔터티를 찾을 수 있습니다.

이 New Relic 인프라 통합은 다음 Amazon Database Migration Service 데이터를 수집합니다.

가져온 모든 데이터에는 하나의 공통 측정기준이 있습니다. `ReplicationInstanceIdentifier`
