이제 인프라 에이전트를 AWS [Database Migration Service](https://aws.amazon.com/dms/) 와 통합하여 지표 데이터를 New Relic으로 보낼 수 있습니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰하세요.

이 통합을 활성화하려면 지침에 따라 [AWS 서비스를 뉴렐릭에 연결하세요](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/). 활성화되면 <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Metrics and events**</DNT> 에서 통합 아이콘을 찾고 `aws.dms` 로 필터링할 수 있습니다.

## 측정항목 데이터 [#metrics]

이 New Relic 인프라 통합은 다음 Amazon Database Migration Service 데이터를 수집합니다.

가져온 모든 데이터에는 하나의 공통 측정기준이 있습니다. `ReplicationInstanceIdentifier`
