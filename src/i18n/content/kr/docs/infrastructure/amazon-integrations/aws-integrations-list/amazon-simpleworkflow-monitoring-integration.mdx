[New Relic 인프라 통합에는](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) 지표 데이터를 New Relic으로 전송하는 AWS SimpleWorkFlow용 통합이 포함됩니다.

이 문서에서는 통합 기능, 활성화 방법 및 보고할 수 있는 데이터에 대해 설명합니다.

## 특징

통합을 사용하여 원격 측정 데이터를 수집하고 [Amazon Simple Workflow Service](https://aws.amazon.com/swf/) 에서 New Relic으로 보냅니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰할 수 있습니다.

## 시작하다

이 통합을 활성화하려면 [AWS 서비스를 New Relic에 연결하는 방법을](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 참조하십시오.

통합이 설정되면 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Metrics and events** 로 이동하고 `aws.SimpleWorkFlow` 로 필터링하여 측정항목 데이터를 찾을 수 있습니다.

## 측정항목 데이터 [#metrics]
