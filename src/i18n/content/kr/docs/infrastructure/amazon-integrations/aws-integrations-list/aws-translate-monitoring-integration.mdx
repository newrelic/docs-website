[뉴렐릭 인프라 통합에](/docs/infrastructure/introduction-infra-monitoring) 이제 AWS Translate 통합이 포함되어 해당 지표 데이터를 뉴렐릭으로 보냅니다.

이 문서에서는 통합 기능, 활성화 방법 및 보고할 수 있는 데이터에 대해 설명합니다.

## 특징

통합을 사용하여 [번역 앱](https://aws.amazon.com/translate/) 에서 원격 측정 데이터를 수집하고 New Relic으로 보냅니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한 눈에 관찰하세요.

## 보고서 데이터 [#report-data]

이 데이터를 보고하는 데는 두 가지 옵션이 있습니다. 권장되는 경로는 [Amazon CloudWatch Metric Streams 통합을](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream)설정하는 것입니다. 또는 [폴링에 의존하는 이전 Amazon 통합을](/docs/infrastructure/infrastructure-integrations/cloud-integrations/configure-polling-frequency-data-collection-cloud-integrations)사용할 수 있습니다.

* New Relic 폴링 간격: 5분
* Amazon CloudWatch 데이터 간격: 1분 또는 5분

## 데이터 찾기 및 사용

통합의 지표를 찾으려면 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> 으로 이동하여 `aws.translate` 로 필터링하세요.
