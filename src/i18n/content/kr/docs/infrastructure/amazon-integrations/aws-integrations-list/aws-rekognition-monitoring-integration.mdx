[New Relic 인프라 통합에는](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) Rekognition 데이터를 New Relic에 보고하기 위한 Amazon Rekognition 통합이 포함됩니다. 여기에서는 통합을 활성화하는 방법과 보고할 수 있는 데이터에 대해 설명합니다.

[AWS CloudWatch Metric Streams 통합](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-metric-stream/) 을 활성화하여 사용자 지정 네임스페이스를 포함하여 AWS 서비스의 모든 CloudWatch 지표를 모니터링합니다. 개별 통합은 더 이상 권장되는 옵션이 아닙니다.

## 특징 [#features]

[Amazon Rekognition](https://aws.amazon.com/rekognition/) (Rekognition)은 기계 학습을 통해 이미지 인식 및 비디오 분석을 자동화하고 비용을 낮추는 AWS 서비스입니다. New Relic의 Rekognition 통합을 통해 요청 수, 응답 시간, 감지된 얼굴 수 등을 모니터링할 수 있습니다. 데이터의 사용자 지정 차트를 생성하고 Rekognition 데이터 또는 구성의 변경 사항에 따라 [경고 조건을 생성](/docs/infrastructure/new-relic-infrastructure/infrastructure-alert-conditions/infrastructure-alerts-add-edit-or-view-host-alert-information) 할 수 있습니다.

## 통합 활성화 [#activate]

이 통합을 활성화하려면 [AWS 서비스를 New Relic에 연결하는](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream/#set-up-metric-stream)방법을 참조하십시오.

## 데이터 찾기 및 사용 [#find-data]

이 통합의 데이터는 지표 데이터 유형에 첨부됩니다. UI에서 이 데이터를 찾으려면 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Metrics & Events**</DNT> 으로 이동하고 `aws.rekognition` 로 필터링하세요.

<DNT>**Metric**</DNT> 테이블과 `namespace='AWS/Rekognition'` 사용하여 [데이터를 쿼리하고 탐색](/docs/using-new-relic/data/understand-data/query-new-relic-data) 할 수 있습니다.
