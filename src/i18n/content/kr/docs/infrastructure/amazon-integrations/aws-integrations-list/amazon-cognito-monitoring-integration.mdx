뉴렐릭은 Amazon Web Services (AWS) 와 통합되어 [Amazon Cognito](https://aws.amazon.com/cognito/) 지표 및 기타 데이터를 뉴렐릭에 보고합니다.

이 문서에서는 통합을 활성화하는 방법과 보고되는 데이터에 대해 설명합니다.

## 특징 [#features]

통합을 사용하여 Amazon Cognito 서비스에서 텔레메트리 데이터를 수집하고 뉴렐릭으로 보냅니다. 서비스를 모니터링하고, 유입 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 파악하세요.

## 통합 활성화 [#activate]

이 통합은 CloudWatch MetricStreams(권장)를 통해 사용할 수 있습니다.

이 통합을 활성화하려면 통해 [AWS 서비스를 뉴렐릭에 연결하는 방법을 CloudWatch Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 참조하세요.

### 구형 API 폴링 통합을 사용하여 활성화 [#polling]

구형 폴링 통합을 사용하여 이 통합을 활성화하려면 [AWS API 폴링 통합](/docs/infrastructure/amazon-integrations/connect/connect-aws-new-relic-infrastructure-monitoring/) 활성화 방법을 참조하십시오.

<Callout variant="tip">
  [구성 옵션](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations) 을 사용하여 폴링 빈도를 변경하고 데이터를 필터링할 수 있습니다.

  AWS Cognito 통합에 대한 기본 [폴링](/docs/infrastructure/amazon-integrations/aws-integrations-list/aws-polling-intervals-infrastructure-integrations) 정보:

  * New Relic 폴링 간격: 5분
  * AWS CloudWatch 데이터 간격: 1분
</Callout>

## 데이터 찾기 및 사용 [#find-data]

통합 데이터를 찾으려면 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT> 로 이동하여 `aws.cognito` 로 필터링하세요.
