[뉴렐릭 인프라 통합에는](/docs/infrastructure/introduction-infra-monitoring) 이제 지표 데이터를 뉴렐릭으로 보내는 AWS IoT Twinmaker용 통합이 포함됩니다. 이 문서에서는 통합 기능, 통합을 활성화하는 방법, 통합을 통해 보고할 수 있는 데이터에 대해 설명합니다.

## 특징

통합을 사용하여 [WorkMail](https://aws.amazon.com/iot-twinmaker/) 에서 원격 측정 데이터를 수집하고 New Relic으로 보냅니다. 이를 통해 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰할 수 있습니다.

## 통합 활성화 및 사용

이 통합을 활성화하려면 [AWS 서비스를 뉴렐릭에 연결하는](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 방법을 참조하세요. 통합의 지표를 찾으려면 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> 으로 이동하여 `aws.IoTTwinMaker` 로 필터링하세요.
