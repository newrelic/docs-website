[뉴렐릭 인프라 통합에는](/docs/infrastructure/introduction-infra-monitoring) 이제 AWS Elemental MediaTailor에 대한 통합이 포함되어 해당 지표 데이터를 뉴렐릭으로 보냅니다.

이 문서에서는 통합 기능, 활성화 방법 및 보고할 수 있는 데이터에 대해 설명합니다.

## 특징 [#features]

통합을 사용하여 [AWS Elemental MediaTailor](https://aws.amazon.com/mediatailor/) 에서 원격 측정 데이터를 수집하고 New Relic으로 보냅니다. 서비스를 모니터링하고, 들어오는 데이터를 쿼리하고, 대시보드를 구축하여 모든 것을 한눈에 관찰하세요.

## 통합 활성화 [#active-integration]

이 통합을 활성화하려면 [AWS 서비스를 New Relic에 연결하는](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/)방법을 참조하십시오.

## 데이터 찾기 및 사용 [#find-data]

통합의 지표를 찾으려면 <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities) &gt; Metrics and events**</DNT> 으로 이동하여 `aws.mediatailor` 로 필터링하세요.

## 측정항목 데이터 [#metrics]

이 New Relic 인프라 통합은 다음 AWS Elemental MediaTailor 데이터를 수집합니다.

가져온 모든 데이터에는 하나의 차원이 있습니다. `ConfigurationName`
