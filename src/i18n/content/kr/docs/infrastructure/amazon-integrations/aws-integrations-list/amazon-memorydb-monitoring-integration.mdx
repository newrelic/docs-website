뉴렐릭은 Amazon Web Services (AWS) Amazon 사용하여 와 CloudWatch Metric Streams 통합됩니다. CloudWatch Metric Streams 통합을 설치하면 뉴렐릭 플랫폼에서 [Amazon MemoryDB](https://aws.amazon.com/memrydb/) 지표와 데이터에 액세스할 수 있습니다.

이 문서에서는 통합을 활성화하는 방법을 설명하고, 활성화 후 보고되는 데이터 유형을 설명합니다.

## 특징 [#features]

Amazon MemoryDB 클러스터에 대한 텔레메트리 데이터를 수집하여 서비스를 모니터링하고, 수신 데이터를 조회하고, 대시보드를 구축하여 모든 것을 한눈에 관찰할 수 있습니다.

## 통합 활성화 [#activate]

이 통합은 CloudWatch MetricStreams를 통해 사용할 수 있습니다. 이 통합을 활성화하려면 [통합을 통해 AWS 서비스를 뉴렐릭에 CloudWatch Metric Streams 연결하는 방법을](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream-setup/) 알아보세요.

## 데이터 찾기 및 사용 [#find-data]

통합 데이터를 찾으려면 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Metrics and events**</DNT> 로 이동하여 `aws.memorydb` 로 필터링하세요.
