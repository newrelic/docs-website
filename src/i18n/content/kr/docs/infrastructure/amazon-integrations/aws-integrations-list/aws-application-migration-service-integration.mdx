이제 AWS [로그 Migration Service를](https://docs.aws.amazon.com/mgn/latest/ug/what-is-application-migration-service.html) 사용하여 AWS 로 마이그레이션하는 서버에 뉴렐릭 에이전트를 설치할 수 있습니다.

## 설정 [#setup]

AWS 에서 [뉴렐릭인프라 에이전트를](/docs/infrastructure/install-infrastructure-agent/linux-installation/install-infrastructure-monitoring-agent-linux/) 설치하려면 AWS 콘솔을 통해 게시후 템플릿을 만들어야 합니다. 다음 단계를 따르세요.

1. [AWS 콘솔](https://console.aws.amazon.com/mgn/home) 에서 <DNT>**Application Migration Services** </DNT>엽니다.
2. 사용 가능한 목록에서 원하는 소스 서버를 엽니다.
3. **Post-launch settings** \[출시 후 설정] 탭을 클릭합니다.
4. **Edit &gt; Install New Relic agent** 선택하고 계정 ID와 인스턴스 키를 추가하세요.

## 데이터 모니터링 [#monitor-data]

설정이 완료되면 AWS 로그는 뉴렐릭 인프라 에이전트에 의해 모니터링됩니다. 뉴렐릭에서 데이터를 찾으려면:

1. <DNT>**[one.newrelic.com](https://one.newrelic.com/) &gt; All entities**</DNT> 으)로 이동합니다.
2. **Hosts** 엔터티 섹션에서 인스턴스를 찾습니다.
