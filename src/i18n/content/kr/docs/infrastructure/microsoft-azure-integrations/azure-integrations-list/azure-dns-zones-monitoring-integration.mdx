[뉴렐릭의 통합](/docs/infrastructure/introduction-infra-monitoring) 에는 [Microsoft Azure DNS 영역](https://azure.microsoft.com/en-us/products/dns/) 지표 및 기타 데이터를 뉴렐릭에 보고하기 위한 통합이 포함되어 있습니다. 이 문서에서는 통합을 활성화하는 방법과 보고되는 데이터에 대해 설명합니다.

## 특징

New Relic은 Azure DNS 영역 서비스에 대해 Azure Monitor에서 메트릭 데이터를 수집합니다. DNS 영역은 도메인 이름에 대한 DNS 레코드를 포함하는 데이터 리소스입니다. Azure DNS를 사용하여 DNS 영역을 호스트하고 Azure의 도메인에 대한 DNS 레코드를 관리할 수 있습니다. 또한 인터넷에서 DNS 쿼리에 응답하기 위해 DNS 이름 서버를 제공합니다.

New Relic을 사용하여 다음을 수행할 수 있습니다.

* 미리 빌드된 대시보드에서 Azure DNS 데이터를 봅니다.
* [맞춤 쿼리를 실행하고 데이터를 시각화합니다](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights) .
* 데이터 변경 사항을 알리는 경고 조건을 만듭니다.

## 통합 활성화 [#activate]

표준 [Azure Monitor 통합 절차](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) 에 따라 New Relic 인프라 모니터링에서 Azure 서비스를 활성화하십시오.

## 구성 및 폴링 [#polling]

[구성 옵션을](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings)사용하여 폴링 빈도를 변경하고 데이터를 필터링할 수 있습니다.

New Relic은 기본 [폴링](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) 간격에 따라 Azure Monitor 통합을 통해 Azure DNS 영역 서비스를 쿼리합니다.

## 데이터 찾기 및 사용 [#find-and-use]

[통합 데이터를 탐색](/docs/infrastructure/integrations/find-use-infrastructure-integration-data) 하려면 <DNT>**[one.newrelic.com/infra](https://one.newrelic.com/infra) &gt; Azure &gt; (select an integration)**</DNT> 로 이동하세요.
