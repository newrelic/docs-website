[New Relic의 통합에는](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) [Microsoft Azure Bing Search](https://learn.microsoft.com/en-us/bing/search-apis/bing-web-search/) 메트릭 및 기타 데이터를 New Relic에 보고하기 위한 통합이 포함됩니다. 이 문서는 통합을 활성화하는 방법과 보고된 데이터를 설명합니다.

## 특징

New Relic은 Azure Bing Search 서비스에 대해 Azure Monitor에서 메트릭 데이터를 수집합니다. Azure Bing Search 서비스는 개발자에게 검색 및 AI 기능을 애플리케이션에 통합하기 위한 사용하기 쉬운 플랫폼을 제공합니다.

New Relic을 사용하여 다음을 수행할 수 있습니다.

* 미리 빌드된 대시보드에서 Azure Bing Search 서비스 데이터를 봅니다.
* [맞춤 쿼리를 실행하고 데이터를 시각화합니다](/docs/infrastructure/integrations-getting-started/getting-started/use-integration-data-new-relic-insights) .
* 데이터 변경 사항을 알리는 경고 조건을 만듭니다.

## 통합 활성화 [#activate]

표준 [Azure Monitor 통합 절차](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor) 에 따라 New Relic 인프라 모니터링에서 Azure 서비스를 활성화하십시오.

## 구성 및 폴링 [#polling]

[구성 옵션을](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings)사용하여 폴링 빈도를 변경하고 데이터를 필터링할 수 있습니다.

New Relic은 기본 [폴링](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-monitor/#settings) 간격에 따라 Azure Monitor 통합을 통해 Azure Bing Search 서비스를 쿼리합니다.

## 데이터 찾기 및 사용 [#find-and-use]

[통합 데이터를 탐색](/docs/infrastructure/integrations/find-use-infrastructure-integration-data)하려면 **[one.newrelic.com/infra](https://one.newrelic.com/infra) > Azure > (통합 선택)**로 이동하십시오.
