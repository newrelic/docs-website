[New Relic 인프라 모니터링](/docs/infrastructure) 은 데이터를 New Relic에 보고하는 [Microsoft Azure Redis Cache](https://azure.microsoft.com/en-us/services/cache/) 에 대한 통합을 제공합니다. 이 문서에서는 이 통합을 활성화하는 방법과 캡처할 수 있는 데이터에 대해 설명합니다.

## 특징

Azure Redis Cache에 대한 New Relic의 통합은 연결된 클라이언트 수, 사용된 GET 및 SET 명령 수, 초당 쓰기/읽기 바이트 수와 같은 Azure Redis Cache 서비스에 대한 메트릭을 보고합니다. 또한 서비스의 상태 및 구성에 대한 데이터를 수집합니다.

[인프라 UI](/docs/infrastructure) 에서 Azure Redis Cache 데이터를 모니터링하고 경고할 수 있으며 [사용자 지정 쿼리 및 차트 대시보드를 만들](/docs/using-new-relic/data/understand-data/query-new-relic-data) 수 있습니다.

## 통합 활성화 [#requirements]

이 통합을 활성화하려면 표준 절차에 따라 [New Relic에서 Azure 서비스를 활성화하세요](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) .

## 구성 및 폴링 [#polling]

[구성 옵션](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations) 을 사용하여 폴링 빈도를 변경하고 데이터를 필터링할 수 있습니다.

Azure Functions 통합에 대한 기본 [폴링](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-polling-intervals-infrastructure-integrations) 정보:

* 기본 폴링 간격: 5분
* 해상도: 분당 1 데이터 포인트

## 데이터 찾기 및 사용 [#find-and-use]

통합 데이터를 찾으려면 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure**</DNT> 으로 이동하여 Azure Redis Cache 통합을 선택하세요.

`AzureRedisCacheSample` [이벤트 유형](/docs/data-apis/understand-data/new-relic-data-types/#event-data) 을 사용 [하여 데이터를 쿼리하고 탐색할](/docs/using-new-relic/data/understand-data/query-new-relic-data) 수 있으며 서비스에서 샤드가 구성된 경우 `AzureRedisCacheShardSample` 이벤트 유형을 사용할 수 있습니다.

통합 데이터를 찾고 사용하는 방법에 대한 자세한 내용은 데이터 [이해 및 사용](/docs/infrastructure/integrations/find-use-infrastructure-integration-data) 을 참조하십시오.

## 측정항목 데이터 [#metrics]

Azure Redis Cache 통합에서 보고된 측정항목을 보려면 `AzureRedisCacheSample` 또는 `AzureRedisCacheShardSample` 이벤트 유형을 쿼리합니다. 각 메트릭과 연결된 메타데이터를 사용하여 보고되는 데이터를 [필터링하고 패싯](/docs/infrastructure/integrations-getting-started/getting-started/understand-integration-data-data-types#metric) 합니다. 자세한 메트릭 정보는 [Azure 지원 메트릭](https://docs.microsoft.com/en-us/azure/azure-monitor/platform/metrics-supported#microsoftcacheredis) 설명서를 참조하세요. 자세한 내용은 [Azure Redis Cache 설명서](https://docs.microsoft.com/en-us/azure/redis-cache/cache-how-to-monitor#available-metrics-and-reporting-intervals) 를 참조하세요.
