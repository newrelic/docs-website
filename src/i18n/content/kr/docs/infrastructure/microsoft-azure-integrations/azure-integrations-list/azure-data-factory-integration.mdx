[New Relic 인프라 모니터링](/docs/infrastructure/integrations-getting-started/getting-started/introduction-infrastructure-integrations) 은 [Microsoft Azure Data Factory](https://azure.microsoft.com/en-us/services/data-factory/) 데이터를 New Relic에 보고하기 위한 통합을 제공합니다. 여기에서는 통합을 활성화하는 방법과 보고할 수 있는 데이터에 대해 설명합니다.

## 특징

New Relic은 Azure Data Factory 서비스에서 DataFactory 및 Factory 데이터를 모두 수집합니다. New Relic에서 Azure Data Factory 데이터를 모니터링하고 경고할 수 있으며 사용자 지정 쿼리 및 사용자 지정 차트 대시보드를 만들 수 있습니다.

## 통합 활성화 [#activate]

통합을 [활성화하려면 표준 절차에 따라 New Relic에서 Azure 서비스](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) 를 활성화합니다.

## 구성 및 폴링 [#polling]

[구성 옵션](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations) 을 사용하여 폴링 빈도를 변경하고 데이터를 필터링할 수 있습니다.

Azure Data Factory 통합에 대한 기본 [폴링](/docs/infrastructure/microsoft-azure-integrations/getting-started/azure-integration-polling) 정보:

* 폴링 간격: 5분

## 데이터 찾기 및 사용 [#find-data]

통합 데이터를 찾으려면 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Infrastructure > Azure**</DNT> 으로 이동하여 <DNT>**Data Factory**</DNT> 통합을 선택하세요.
