F5 [오픈 소스 통합](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) 은 [F5](https://support.f5.com/csp/knowledge-center/software/BIG-IP) 의 성능 메트릭(가상 서버, 풀, 노드 및 메모리 통계 포함)을 모니터링하고 데이터를 New Relic Infrastructure에 보고합니다.

요구 사항, 설치 및 구성 정보는 GitHub의 [F5 온-호스트 통합 README](https://github.com/newrelic/nri-f5) 를 참조하십시오.

<Callout variant="important">
  New Relic은 이 기술의 모니터링을 가능하게 하기 위해 이 통합을 오픈 소스로 제공했습니다. 이 통합은 보증이나 지원 없이 있는 그대로 제공되지만 GitHub를 통해 문제를 보고하고 이 통합에 기여할 수 있습니다. 이 통합에 대한 지원은 [Expert Services 구독](https://www.newrelic.com/expertservices) 으로 제공됩니다.
</Callout>
