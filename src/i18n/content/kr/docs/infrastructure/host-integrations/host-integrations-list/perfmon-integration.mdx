PerfMon 통합은 [Microsoft 성능 모니터](https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc749154(v%3dws.11)) 의 쿼리 및 이벤트 결과를 New Relic에서 소비할 수 있는 표준 출력(stdout)으로 제공합니다.

시작하려면 GitHub에서 [Windows PerfMon/WMI 온-호스트 통합 README](https://github.com/newrelic/nri-perfmon) 를 참조하십시오.

<Callout variant="tip">
  이 통합은 [GitHub에서 오픈 소스](https://github.com/newrelic/nri-perfmon) 로 릴리스됩니다. 최신 업데이트에 대한 변경 로그도 제공됩니다. 이 통합에 대한 지원은 [Expert Services 구독](https://www.newrelic.com/expertservices) 으로 제공됩니다.
</Callout>
