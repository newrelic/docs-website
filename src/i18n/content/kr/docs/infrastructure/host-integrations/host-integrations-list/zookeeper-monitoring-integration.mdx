Zookeeper 모니터링 통합은 [Apache Zookeeper](https://zookeeper.apache.org/) 노드에서 보고한 중요한 성능 메트릭을 캡처하고 데이터를 New Relic에 보고합니다.

시작하고 설치 지침은 GitHub의 [Zookeeper 온-호스트 통합 README](https://github.com/newrelic/nri-zookeeper) 를 참조하십시오.

<Callout variant="tip">
  이 통합은 [Github에서 오픈 소스](https://github.com/newrelic/nri-zookeeper) 로 릴리스되었습니다. 최신 업데이트에 대한 변경 로그도 제공됩니다. 이 통합에 대한 지원은 [Expert Services 구독](https://www.newrelic.com/expertservices) 으로 제공됩니다.
</Callout>
