## 인프라 에이전트 설치 [#infra]

Blazor WebAssembly를 New Relic과 통합하는 데 인프라 에이전트가 필요하지 않지만 시스템을 완전히 관찰하려면 설치하는 것이 좋습니다. 인프라 에이전트는 호스트에 대한 로그 및 기타 데이터를 수집하여 애플리케이션 지표와 함께 성능 지표를 표시합니다.

인프라 에이전트를 설치하는 방법에는 두 가지가 있습니다.

* [가이드 설치](https://one.newrelic.com/launcher/catalog-pack-details.launcher/?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5saXN0aW5nIn0=&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyJ9) 에 따라 인프라 에이전트로 시스템을 계측하십시오.

## Blazor WebAssembly와 New Relic 통합 [#integrate]

Blazor WebAssembly에서 데이터를 수집하려면 다음 절차를 따르세요.

1. [브라우저 모니터링 에이전트](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent) 를 설치합니다.
2. 앱을 연 후 `www/root` 디렉토리로 이동합니다. 브라우저 에이전트 설치의 스크립트를 `index.html` 파일에 추가하십시오.
3. **[one.newrelic.com](https://one.newrelic.com/all-capabilities)** 으로 이동합니다. 그런 다음 **측정항목 및 이벤트** 를 클릭합니다. 샘플 NRQL 쿼리를 실행하여 New Relic이 데이터를 수집하고 있는지 확인합니다.

```sql
SELECT average(windowLoad) FROM PageViewTiming WHERE (deviceType = 'Desktop') FACET `userAgentName` SINCE 604800 seconds AGO TIMESERIES
```

4. [Blazor WebAssembly Instant Observability 페이지](https://newrelic.com/instant-observability/blazor-web-assembly) 로 이동한 다음 페이지 상단의 **설치** 를 클릭하여 기본 대시보드를 설치합니다.
5. one.newrelic.com에서 **대시보드** 로 이동하고 Blazor **[WebAssembly](https://one.newrelic.com/all-capabilities)** 를 클릭하여 대시보드를 봅니다.

NRQL 쿼리 작성 및 대시보드 생성에 대해 자세히 알아보려면 다음 문서를 확인하세요.
