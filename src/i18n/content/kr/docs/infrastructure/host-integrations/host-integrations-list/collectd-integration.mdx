Collectd [통합](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) 은 [Collectd](https://collectd.org/wiki/index.php/Main_Page) 구성 장치에서 New Relic으로 메트릭을 모니터링하고 보고합니다.

시작하려면 GitHub에서 [Collectd 온-호스트 통합 README](https://github.com/newrelic/nri-collectd) 를 참조하십시오.

<Callout variant="tip">
  이 통합은 [Github에서 오픈 소스](https://github.com/newrelic/nri-collectd) 로 릴리스되었습니다. 최신 업데이트에 대한 변경 로그도 제공됩니다. 이 통합에 대한 지원은 [Expert Services 구독](https://www.newrelic.com/expertservices) 으로 제공됩니다.
</Callout>
