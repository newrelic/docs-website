Go Insights 통합은 New Relic에서 이벤트를 쿼리하고 New Relic에 삽입합니다.

시작하려면 GitHub에서 [Go Insights 온-호스트 통합 README](https://github.com/newrelic/go-insights) 를 참조하십시오.

<Callout variant="tip">
  이 통합은 [Github에서 오픈 소스](https://github.com/newrelic/go-insights) 로 릴리스되었습니다. 최신 업데이트에 대한 변경 로그도 제공됩니다. 이 통합에 대한 지원은 [Expert Services 구독](https://www.newrelic.com/expertservices) 으로 제공됩니다.
</Callout>
