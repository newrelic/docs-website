도커 컨테이너 [통합](/docs/integrations/new-relic-integrations/getting-started/introduction-infrastructure-integrations) 모니터 및 [도커 API](https://docs.docker.com/reference/api/engine/version/v1.48/) 사용하여 도커 컨테이너에서 뉴렐릭으로 보고하는 지표입니다.

<Callout variant="tip">
  이 통합은 [Github에서 오픈 소스](https://github.com/newrelic/nri-docker) 로 공개되었습니다. 최신 업데이트에 대한 변경 로그도 여기에서 확인할 수 있습니다.
</Callout>

<Callout variant="important">
  * 이 통합은 LCOW(Windows에서 실행되는 Linux 컨테이너)를 지원하지 않습니다. 자세한 내용은 [docker 문서를](https://docs.docker.com/desktop/windows/wsl/) 참조하세요.
  * Linux의 경우 이 통합은 인프라 에이전트와 함께 제공되며, Docker 컨테이너가 감지되면 자동으로 활성화됩니다.
  * Windows의 경우 2.3.0 버전부터 호환성을 제공하지만 인프라 에이전트 위에 수동으로 설치해야 합니다. 향후 릴리스에서는 인프라 에이전트와 함께 번들로 제공될 예정입니다.
</Callout>
