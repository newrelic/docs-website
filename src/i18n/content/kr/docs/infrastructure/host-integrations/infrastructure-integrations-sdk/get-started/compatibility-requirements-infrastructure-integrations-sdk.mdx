## 인프라 버전 [#agent-version]

인프라 통합 SDK v4에는 인프라 에이전트 버전 1.13.0 이상이 필요합니다.

SDK v3에는 에이전트 버전 1.0.888 이상이 필요합니다.

인프라 에이전트 버전 정보와 설치 및 업데이트 옵션은 [인프라 릴리스 정보 를](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes) 참조하십시오.

SDK 버전 및 변경 사항에 대한 참고 사항은 [변경 로그](#change-log) 를 참조하십시오.

## 운영체제 [#os]

SDK로 구축된 통합은 Linux 또는 Windows 운영 체제용으로 컴파일할 수 있습니다.

## 데이터 및 파일 사양 [#data-specs]

호스트 기반 인프라 통합은 [데이터 및 파일 사양](/docs/infrastructure/integrations-sdk/get-started/create-activate-infrastructure-integration) 을 준수하는 한 모든 프로그래밍 언어로 생성할 수 있습니다.
