## 문제

통합에서 데이터를 보고하지만 몇 가지 인벤토리 속성이 누락되었습니다.

## 해결책

다음 인벤토리 키 이름은 당사 플랫폼에서 내부적으로 사용됩니다. 고객 제공 통합이 동일한 키 이름을 가진 인벤토리 항목을 보내는 경우 사용자에게 보이지 않는 상태로 유지됩니다.

* `pid`
* `installed_epoch`
* `epoch`
* `time`
* `size`
* `boot_id`
* `product_uuid`
* `installed_time`
