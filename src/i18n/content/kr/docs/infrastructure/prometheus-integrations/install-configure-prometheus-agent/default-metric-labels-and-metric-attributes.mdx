<Callout variant="important">
  [Prometheus DataModel을](https://prometheus.io/docs/concepts/data_model/#metric-names-and-labels) 준수하기 위해 Kubernetes 레이블 이름은 지원되지 않는 문자를 `_` 로 대체하기 위해 스크랩한 후 삭제됩니다. 예를 들어 Kubernetes 레이블 `k8s.io/app` 은 `k8s_io_app` 로 추가됩니다. 레이블 값은 수정되지 않습니다.
</Callout>

# 생성된 메트릭 [#generated-metrics]

<Callout variant="tip">
  이러한 메트릭에는 모든 [메트릭 레이블이](#metric-labels) 포함됩니다.
</Callout>

스크랩된 각 대상에 대해 다음 메트릭이 생성됩니다.

<table>
  <thead>
    <tr>
      <th style={{ width: "400px" }}>
        미터법
      </th>

      <th>
        설명
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `up`
      </td>

      <td>
        스크랩이 실패한 경우 값은 `0` 이고 그렇지 않으면 `1` 입니다.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_duration_seconds`
      </td>

      <td>
        스크랩 기간.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_samples_scraped`
      </td>

      <td>
        대상이 노출한 샘플 수입니다.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_samples_post_metric_relabeling`
      </td>

      <td>
        메트릭 재레이블이 적용된 후 남은 샘플 수입니다.
      </td>
    </tr>

    <tr>
      <td>
        `scrape_series_added`
      </td>

      <td>
        이 스크랩의 대략적인 새 시리즈 수입니다.
      </td>
    </tr>
  </tbody>
</table>
