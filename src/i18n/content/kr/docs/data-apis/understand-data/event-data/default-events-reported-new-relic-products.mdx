New Relic 제품은 다양한 [유형의 데이터를](/docs/using-new-relic/data/understand-data/new-relic-data-types) 보고합니다. 보고되는 데이터 유형 중 하나는 [이벤트 데이터](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data) 입니다. 이벤트는 UI 차트와 테이블에 표시되며 [쿼리](/docs/using-new-relic/data/understand-data/query-new-relic-data) 에도 사용할 수 있습니다. 사용 가능한 데이터 유형을 이해하려면 [NRQL을 통해 사용 가능한 데이터](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql#what-you-can-query) 를 참조하십시오.

New Relic 제품에서 보고된 이벤트에 대해 자세히 알아보십시오.

* [APM 기본 이벤트](/docs/insights/insights-data-sources/default-data/apm-default-events-insights)
* [브라우저 기본 이벤트](/docs/insights/insights-data-sources/default-data/browser-default-events-insights)
* [인프라 기본 이벤트](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-events)
* [모바일 기본 이벤트](/docs/insights/insights-data-sources/default-data/mobile-default-events-insights)
* [합성 기본 이벤트](/docs/insights/insights-data-sources/default-data/synthetics-default-events-insights)
* 계정 변경 사항을 이해하기 위한 [NrAuditEvent 이벤트](/docs/insights/event-data-sources/default-events/query-account-audit-logs-nrauditevent/)
