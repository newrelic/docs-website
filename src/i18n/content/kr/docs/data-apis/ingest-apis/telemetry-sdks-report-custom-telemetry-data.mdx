Telemetry SDK는 데이터를 New Relic 플랫폼으로 보내는 API 클라이언트 라이브러리의 오픈 소스 세트입니다. 내부적으로 이러한 SDK는 Metric API, Trace API, Log API 및 Event API와 같은 기본 데이터 수집 API에 의존합니다.

[사전 구축된 솔루션](https://newrelic.com/instant-observability) 이 귀하의 요구 사항을 충족하지 못하는 경우 당사의 원격 분석 SDK는 사용자 지정 원격 분석 솔루션을 생성하는 한 가지 방법입니다( [사용자 지정 데이터 보고를 위한 다른 솔루션](/docs/data-apis/custom-data/intro-custom-data) 참조).

## 요구 사항 및 호환성 [#requirements]

Telemetry SDK는 Metric API, Event API, Log API 및 Trace API를 사용하며 모두 <InlinePopover type="licenseKey"/>, 데이터를 보내려는 계정의 라이센스 키가 필요합니다.

<Callout variant="tip">
  New Relic은 Apache 2.0 라이선스에 따라 Telemetry SDK를 오픈 소스 커뮤니티에 기여했습니다.
</Callout>
