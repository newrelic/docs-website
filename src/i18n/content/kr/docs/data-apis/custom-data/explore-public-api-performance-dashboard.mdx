공개 API 성능 대시보드는 New Relic 계정의 일부로 포함된 즉시 사용 가능한 대시보드입니다. 공개 API의 성능에 대한 실행 가능한 일반 통찰력과 신규 고객이 자신의 데이터를 추가하기 전에 대시보드 기능을 테스트할 수 있는 기회를 모두 제공합니다.

대시보드는 인기 있는 공개 API에 액세스할 때 New Relic 고객의 익명화된 샘플링이 경험하는 실제 지연 시간을 표시하여 작동합니다.

<Callout variant="important">
  글로벌 성능 데이터 세트는 있는 그대로 표시됩니다. Global Performance 데이터 세트는 다양한 소스의 샘플 집계를 나타내며 New Relic은 데이터의 정확성, 완전성 또는 진실성을 확인하기 위해 노력하지 않습니다. 이 데이터를 사용하는 모든 목적에 대한 정보의 유일한 소스로 의존해서는 안 되며 New Relic은 이 데이터에 의존하여 내린 결정에 대해 책임을 지지 않습니다. Global Performance 데이터 세트는 데이터 세트에 포함된 기술에 대한 New Relic의 보증 또는 권장 사항으로 간주되어서는 안 됩니다.
</Callout>

## New Relic에 대시보드 추가 [#Add-dashboard]

공개 API 성능 대시보드를 추가하려면:

1. 대시보드 페이지에서 사전 구축된 공용 API 성능 대시보드를 검색하세요. 인기 있는 사전 구축 대시보드 제목 옆에 있는

   <DNT>
     **View All**
   </DNT>

   링크를 클릭하여 검색할 수 있습니다.

2. 대시보드를 선택합니다.

3. 드롭다운에서 계정을 선택합니다.

4. 계정에 추가하려면

   <DNT>
     **View dashboard**
   </DNT>

   클릭하세요.

## 대시보드 살펴보기 [#explore-dashboard]

사전 구축된 새 대시보드를 사용자 정의할 수 있습니다. 이름과 설정을 변경하고, 새 콘텐츠와 새 위젯을 추가하고, 차트를 수정하는 등의 작업을 할 수 있습니다! 자세한 내용은 [대시보드 관리를](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/) 참조하세요.

다음은 공개 API 성능 대시보드를 탐색하는 방법에 대한 몇 가지 제안입니다.

* 다음을 클릭하세요.

  <Icon name="fe-more-horizontal"/>

  차트 오른쪽 모서리에 있는 아이콘을 클릭하고

  <DNT>
    **View query**
  </DNT>

  선택하면 차트를 만드는 데 사용된 [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) 쿼리를 볼 수 있습니다.

* 다음을 클릭하세요.

  <Icon name="fe-more-horizontal"/>

  차트 오른쪽 모서리에 있는 아이콘을 클릭하고

  <DNT>
    **Get as image**
  </DNT>

  선택하면 차트를 이미지로 보거나 다운로드할 수 있습니다.

* 막대 차트에서 특정 도메인을 선택하거나 페이지 상단에 있는 텍스트 필드를 클릭하여 필터를 추가하십시오.

* 이미 고유한 데이터를 추가한 경우 쿼리를 복사하여 직접 사용하도록 수정해 보십시오.

<Callout variant="important">
  공개 API 성능 대시보드는 현재 EU 고객이 사용할 수 없습니다.
</Callout>

<Callout variant="important">
  공개 API 성능 대시보드는 현재 알림을 지원하지 않습니다.
</Callout>

## 보고된 데이터 [#data-reported]

`Public_APICall` 이벤트를 통해 모든 New Relic 조직에서 공개 API 데이터를 사용할 수 있습니다. 이 데이터는 수집에 포함되지 않습니다. 해당 데이터의 구조를 보려면 다음 쿼리를 실행하십시오.

```sql
FROM Public_APICall select keyset()
```

## 대시보드 및 데이터에 대한 추가 정보 [#explore-dashboard]

* New Relic 대시보드에 대한 자세한 내용은 [대시보드 소개를](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 참조하십시오.
* 고객은 또한 [데이터 탐색기를](/docs/query-your-data/explore-query-data/browse-data/introduction-data-explorer/) 사용하여 이 데이터 세트를 더 깊이 파고들 수 있습니다.
