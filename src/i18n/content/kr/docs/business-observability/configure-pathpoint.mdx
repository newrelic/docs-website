Pathpoint는 비즈니스의 고유한 요구 사항을 충족하도록 사용자 지정할 수 있는 유연한 구성 시스템을 제공합니다. 구성 프로세스에는 원하는 설정을 지정하는 JSON 파일 작성이 포함됩니다. 구성 파일이 로드되면 터치포인트와 해당 쿼리를 수동으로 추가로 미세 조정할 수 있습니다. 이 수준의 사용자 지정을 통해 조직의 특정 요구 사항에 맞는 맞춤형 관찰 가능성 솔루션을 만들 수 있습니다.

이 페이지에서는 다음을 배우게 됩니다.

* JSON 구성 파일이란 무엇입니까?
* 새 JSON 구성 파일 업로드
* 현재 활성 JSON 구성 파일 다운로드

### JSON 구성 파일이란 무엇입니까?

JSON(JavaScript Object Notation)은 기본 기능이 정보 교환을 허용하는 데이터 구조입니다. 이 구조를 통해 Pathpoint 구현을 용이하게 하는 각 요소와 구성 요소를 식별할 수 있습니다. 여기에는 속성, 쿼리 및 데이터 출력의 기능에 대한 이해가 포함됩니다.

위의 이미지는 각 단계가 어떻게 몇 개의 단계에 연결되고 차례로 단일 터치포인트 또는 몇 개의 터치포인트에 연결될 수 있는지 보여줍니다. 새 단계, 단계 및 접점을 프로그래밍하려면 새 JSON 구성 파일을 업로드합니다.

### 새 JSON 구성 파일 업로드

새 JSON 구성 파일을 로드하려면:

<Steps>
  <Step>
    New Relic 계정에서 **Apps** > **Pathpoint**<img src={threeLines} class="inline" style={{height: "30px", width: "30px"}} alt="Three lines" title="Three lines">로 이동하고 클릭합니다.</img>

    .

    <img
      title="Navigate to Pathpoint config"
      alt="Navigate to Pathpoint config"
      src={pathpointConfig}
    />
  </Step>

  <Step>
    **Json 구성을**클릭합니다.

    <img
      title="JSON Configurations"
      alt="JSON Configurations"
      src={jsonConfig}
    />
  </Step>

  <Step>
    그러면 대화 상자 창이 열립니다. **설명**아래에서 새 JSON 구성 파일을 설명합니다. 그런 다음 메모를 제공하고 **파일에서를** 클릭하여 JSON 구성 파일을 찾습니다.

    <img
      title="Upload new JSON config file"
      alt="Upload new JSON config file"
      src={uploadJson}
    />
  </Step>

  <Step>
    Pathpoint를 업데이트하는 데 사용할 JSON 구성 파일을 찾고 **열기를**클릭합니다.

    <img
      title="open new JSON config file"
      alt="open new JSON config file"
      src={openFile}
    />
  </Step>

  마지막으로 선택한 특정 JSON 파일의 업데이트와 함께 Pathpoint가 표시됩니다.
</Steps>

### 현재 활성 JSON 구성 파일 다운로드

Pathpoint는 현재 JSON 구성 파일을 컴퓨터에 직접 다운로드할 수 있는 유연성을 제공합니다. 이 기능은 구성 파일을 변경해야 하는 경우에 특히 유용합니다.

현재 활성 JSON 구성 파일을 다운로드하려면:

<Steps>
  <Step>
    New Relic 계정에서 **Apps** > **Pathpoint**로 이동합니다. <img src={threeLines} class="inline" style={{height: "30px", width: "30px"}} alt="Three lines" title="Three lines">여기에서</img>

    창의 왼쪽 상단 부분에.

    <img
      title="Navigate to Pathpoint config"
      alt="Navigate to Pathpoint config"
      src={pathpointConfig}
    />
  </Step>

  <Step>
    **Json 구성을**클릭합니다.

    <img
      title="JSON Configurations"
      alt="JSON Configurations"
      src={jsonConfig}
    />
  </Step>

  <Step>
    그런 다음 아래쪽 화살표를 클릭하십시오.

    <img
      title="download active JSON config file"
      alt="download active JSON config file"
      src={downloadFile}
    />
  </Step>

  활성 JSON 구성 파일이 이제 컴퓨터에 다운로드됩니다.
</Steps>
