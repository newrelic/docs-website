파트너가 파트너 관계 통합을 사용자 지정하기 위해 사용할 수 있는 여러 가지 기타 설정이 있습니다. 이러한 설정의 대부분은 New Relic 기능의 표시를 활성화하거나 비활성화합니다. 이러한 설정은 New Relic에서 설정해야 하며 Partnership Console을 통해 구성할 수 없습니다.

## 설정 유형 [#other-settings]

이러한 설정을 수정하려면 [파트너십 기술 담당자](/docs/accounts-partnerships/partnerships/partner-integration-guide/contacting-new-relic) 에게 문의하십시오.

<DNT>
  **Download links:**
</DNT>

* 에이전트 링크 표시
* 구성 파일 링크 표시

<DNT>
  **Welcome messages:**
</DNT>

* 가입 메시지: 맞춤형 환영 메시지
* 배너 환영 메시지 숨기기 또는 표시
* 인보이스 메시지 숨기기 또는 표시

<DNT>
  **Email control:**
</DNT>

* 배포 알림 보내기
* 평가판 이메일 보내기

<DNT>
  **Billing email:**
</DNT>

청구 방법이 "리셀러"이고 송장이 특정 당사자에게 전달되어야 하는 파트너십을 위한 것입니다.

<DNT>
  **Feature access:**
</DNT>

* 서버 모니터링
* 사용자 관리)
