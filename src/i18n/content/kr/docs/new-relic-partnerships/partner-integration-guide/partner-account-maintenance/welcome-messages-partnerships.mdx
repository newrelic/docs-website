## 여러 하위 파트너십 [#multiple-names]

귀하의 파트너십이 여러 이름으로 비즈니스를 하거나 다른 브랜드를 사용하는 여러 다른 시장에서 비즈니스를 수행하는 경우 다른 파트너십을 고유하게 브랜드화할 수 있습니다. 이 기능은 단일 파트너십 소유자 계정에서 여러 하위 파트너십을 사용하여 사용할 수 있습니다. 파트너십은 다음을 포함한 여러 설정으로 구별될 수 있습니다.

* 이름
* 제품 가격
* 브랜딩(머리글, 바닥글 등)
* 결제 구성

파트너쉽 소유자 계정에 로그인한 후 하위 파트너쉽 계정 목록에서 선택할 수 있습니다.
