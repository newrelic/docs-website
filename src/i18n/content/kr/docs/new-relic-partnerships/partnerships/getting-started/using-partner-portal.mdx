New Relic은 파트너 판매 지원 콘텐츠, 교육, 마케팅 자료 등 New Relic 파트너 프로그램에서 제공하는 귀중한 리소스에 파트너가 쉽게 액세스할 수 있는 웹 포털을 제공합니다.

귀하가 파트너이고 아직 이에 대한 액세스 권한이 없는 경우 파트너 영업 관리자에게 문의하거나 Partnersales @ newrelic.com으로 이메일을 보내십시오.

파트너 포털에 액세스하려면: 브라우저에서 다음 URL을 입력(또는 책갈피)한 다음 사용자 이름과 암호로 로그인합니다.

```
https://partners.newrelic-external.com/English/
```
