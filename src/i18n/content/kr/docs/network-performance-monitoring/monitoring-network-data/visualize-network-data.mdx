성능 예측을 위한 네트워크 데이터를 설정한 후 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Network**</DNT> 으로 이동하여 기기가 뉴렐릭에 보고하는 네트워크 데이터를 탐색하세요.

대시보드에서 스위치, 라우터 등을 그룹화하고 모니터링할 수 있습니다. 이를 통해 네트워크 상태를 신속하게 집계, 격리 및 분석할 수 있습니다.

New Relic에서 네트워크를 모니터링하는 몇 가지 예를 보려면 이 짧은 비디오(약 3분 45초)를 시청하세요.

## 전제 조건 [#prerequisites]

* New Relic 계정. 하나가 없습니까? [무료 가입!](https://newrelic.com/signup) 신용 카드가 필요하지 않습니다.

## 네트워크 성능 데이터 탐색 시작 [#explore-network-performance-data]

<DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Network monitoring**</DNT> 으로 이동하여 네트워크 데이터를 볼 수 있습니다.
