Set up your AWS environment for <DNT>**Cloud Cost Intelligence**</DNT> by configuring consolidated billing, Cost and Usage Reports (CUR), Identity and Access Management (IAM) roles and policies, and necessary AWS regions.

## 전제 조건

두 가지 설치 방법 중 하나를 시작하기 전에 다음 사항을 확인하십시오.

### 모든 설정에 필수입니다.

* <DNT>**AWS account with consolidated billing**</DNT>: [Set up consolidated billing for AWS Organizations](https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/consolidated-billing.html). This is required to aggregate cost data from all accounts in your organization, enabling comprehensive cost analysis across your entire AWS infrastructure.
* <DNT>**IAM Privileges**</DNT>: Administrative access to the target AWS account is required to deploy resources, create S3 buckets, IAM policies, and IAM roles.

### 자동 설정에만 필요합니다.

* <DNT>**New Relic API Key**</DNT>: Generate a valid API key for stack creation. We recommend creating a dedicated key for <DNT>**CCI**</DNT> integration. Generate this in the New Relic UI: [User menu &gt; API Keys](https://onenr.io/07wklp68lRL).

## 설치 방법을 선택하세요

<DNT>**Cloud Cost Intelligence**</DNT> offers two ways to set up AWS integration:

* <DNT>[**Automate AWS with CloudFormation (Recommended)**](/docs/cci/get-started/aws/automated-setup-cloudformation)</DNT>: Quickly deploy a pre-configured CloudFormation stack that automatically provisions all required AWS resources. Best for faster onboarding with minimal manual configuration.
* <DNT>[**Manually integrate your AWS account**](/docs/cci/get-started/aws/manual-integrate-aws-account)</DNT>: Manually configure each AWS resource for complete control over the setup process. Best for organizations with specific compliance or customization requirements.
