Use the automated AWS CloudFormation template to accelerate your <DNT>**Cloud Cost Intelligence**</DNT> setup. The pre-configured stack automatically provisions the required AWS environment, including S3 bucket configuration, CUR export settings, [Trusted Advisor settings check](/docs/cci/financial-intelligence/cci-recommendations/#aws-trusted-advisor), and necessary IAM roles and policies. Review all resource permissions in the AWS CloudFormation console before deploying.

## 비용 고려 사항

이 템플릿은 CUR 데이터 수집을 위한 S3 버킷을 자동으로 생성합니다. 기본 지역은 `us-east-2` 이지만, CloudFormation 요구 사항에 맞춰 이를 변경할 수 있습니다.

## 설치 지침

<Steps>
  <Step>
    ### 클라우드 컴퓨팅 인텔리전스에 액세스하세요

    Navigate to <DNT>[**one.newrelic.com**](https://one.newrelic.com/)</DNT> <DNT>**Integrations &amp; Agents**</DNT> <DNT>**Cloud Cost Intelligence - AWS**</DNT>.
  </Step>

  <Step>
    ### CloudFormation 자동화를 선택하세요

    <DNT>**Automate AWS with CloudFormation (Recommended)**</DNT> 선택하고 <DNT>**Continue**</DNT> 클릭합니다.
  </Step>

  <Step>
    ### AWS의 CloudFormation

    1. 연결 이름을 확인하고 뉴렐릭 데이터 지역을 선택하세요.
    2. <DNT>**Launch your CloudFormation in AWS**</DNT> 을(를) 클릭합니다.
    3. Log in to your AWS account with <DNT>**Admin role**</DNT> when prompted. The <DNT>**CloudFormation Stacks**</DNT> screen opens in the AWS console.
  </Step>

  <Step>
    ### 구성해야 합니다.

    In the <DNT>**Create Stack**</DNT> form, configure the following parameters:

    <DNT>**Pre-populated parameters (do not modify):**</DNT>

    * <DNT>**New Relic Account ID**</DNT>: Pre-populated. Do not change.
    * <DNT>**New Relic Data Region**</DNT>: Pre-populated. Do not change.
    * <DNT>**External ID**</DNT>: Pre-populated. Do not change.

    <DNT>**Required parameters:**</DNT>

    * <DNT>**NewRelicAPIKey**</DNT>: Enter your New Relic API key. Generate this in the New Relic UI: User menu &gt; API Keys.
    * <DNT>**Stack Name**</DNT>: Default value provided. Edit if needed, but ensure the name is globally unique across all AWS CloudFormation stacks. We recommend keeping the default value.
    * <DNT>**Bucket Name**</DNT>: Default value provided. Edit if needed, but ensure the name is globally unique across all S3 buckets.
    * <DNT>**Connection Name**</DNT>: Default value provided. Edit if needed for your naming conventions.

    <DNT>**Optional parameters:**</DNT>

    * <DNT>**Grant Access to Trusted Advisor**</DNT>: Defaults to <DNT>**Yes**</DNT>. When enabled, allows checking if <DNT>**Trusted Advisor**</DNT> cost insights is enabled for your account and retrieves recommendations. If you choose <DNT>**No**</DNT> or <DNT>**Trusted Advisor**</DNT> is not enabled, <DNT>**Trusted Advisor**</DNT> recommendations will not be available.

    <Callout variant="important">
      <DNT>**CCI**</DNT> Native Recommendations are independent of <DNT>**Trusted Advisor**</DNT> and will work regardless of this setting.
    </Callout>

    다른 모든 규격은 선택 사항이며 필요에 따라 구성할 수 있습니다.
  </Step>

  <Step>
    ### 스택을 생성합니다

    <DNT>**I acknowledge that AWS CloudFormation might create IAM resources**</DNT> 선택하고 <DNT>**Create stack**</DNT> 클릭합니다.
  </Step>

  <Step>
    ### 스택 생성 확인

    When the stack creation completes successfully, the status displays <DNT>**CREATE\_COMPLETE**</DNT>.
  </Step>

  <Step>
    ### CUR 보고서 생성 여부를 확인합니다.

    24시간 기다린 후 AWS 버킷에 parquet 형식의 CUR 보고서가 생성되었는지 확인하십시오.

    <Callout variant="important">
      아래 경로는 예시입니다. 실제 경로에는 서로 다른 타임스탬프, UUID 및 청구 기간이 적용됩니다.

      <DNT>**Amazon S3**</DNT> &gt; <DNT>**Buckets**</DNT> &gt; <DNT>**bucket\_name**</DNT> &gt; <DNT>**cost-and-usage-reports/**</DNT> &gt; <DNT>**bucket\_name-cci/data**</DNT> &gt; <DNT>**BILLING\_PERIOD=2025-12/**</DNT> &gt; <DNT>**2025-12-24T03:05:19.886Z-a2e23b2a-588a-4305-97fd-be712f555b75/**</DNT> &gt; <DNT>**bucket\_name-cci-00001.snappy.parquet**</DNT>
    </Callout>

    통합 설정이 완료되었습니다.
  </Step>
</Steps>

## 제한 사항

The automated <DNT>**CloudFormation**</DNT> template does not:

* 컴퓨트 또는 네트워크 리소스를 생성하거나 구성합니다.
* 추가 AWS 서비스를 활성화하거나 지원 수준을 업데이트하세요.
* 로그, 트레이스 및 기타 지표 수집 프로세스를 자동화합니다.
* 쓰기 권한 역할 또는 정책을 생성하세요.

## 구성된 정책 및 권한

자동화된 스택은 다음과 같은 액세스 수준을 구성합니다.

* <DNT>**Billing Data Access**</DNT>: Read-only permission to the specific S3 bucket created for storing Cost and Usage Reports.
* <DNT>**Service Pricing Access**</DNT>: Read-only access to the AWS Pricing API to correlate your usage with public rates.
* <DNT>**Configuration Automation**</DNT>: Limited permissions required to generate the CUR exports and verify the integration status during the setup phase.
* <DNT>**Optional Optimization Access**</DNT>: (If selected) Read-only access to AWS Trusted Advisor to surface cost-saving recommendations.
