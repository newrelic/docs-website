  이 기능은 현재 [출시 전 정책](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy) 에 따라 미리보기 프로그램의 일부로 제공됩니다.

Google Cloud Platform 청구 정보를 클라우드 비용 인텔리전스에 연결하여 비용을 모니터링하고 최적화하세요. 이 통합 기능은 서비스 계정 기반 인증을 사용하여 GCP 청구 및 가격 데이터에 안전하게 액세스합니다. 뉴렐릭은 사용자의 BigQuery 데이터 세트에 대한 읽기 전용 액세스 권한이 있는 전용 서비스 계정을 생성합니다.

## 시작하기 전에

필요한 것:

* **뉴렐릭에 연결된 GCP 프로젝트 모니터링** - [뉴렐릭에 Google Cloud Platform 서비스 연결을](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)참조하세요.

* **활성 GCP 청구 계정**

* **BigQuery 데이터셋** 포함 내용:

  * 청구서 내보내기 표
  * 클라우드 가격 내보내기 표

* **Google Cloud Console 액세스**

## 설치 지침

<Steps>
  <Step>
    ### 뉴렐릭 계정 선택 [#select-nr-account]

    1. <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT> 으)로 이동합니다.
    2. 계정을 선택하고 **Continue** \[계속을] 클릭하세요.
  </Step>

  <Step>
    ### GCP 연결 정보를 제공해 주세요. [#gcp-connection-details]

    1. GCP 청구 및 데이터 세트 정보를 입력하세요.

       * **Connection name** \[연결 이름] - 기술 이름
       * **Project ID** \[프로젝트 ID] - GCP 프로젝트 ID
       * **Dataset name** \[데이터셋 이]름 - BigQuery 데이터셋 이름
       * **Billing table name** \[청구 테이블 이름] - 청구 테이블 이름
       * **Pricing table name** \[가격표 이름] - 가격표 이름을 입력하세요

    2. **계속** 을 클릭합니다.
  </Step>

  <Step>
    ### 접근 권한 부여 [#gcp-authorize-access]

    1. 표시된 서비스 계정 ID를 복사합니다.
    2. **Google Cloud Console** 열고 BigQuery 결제 테이블에 대한 서비스 계정에 **BigQuery DataViewer** 역할을 부여하세요.
    3. 돌아가서 **Continue** \[계속을] 클릭하세요.
  </Step>

  <Step>
    ### 연결을 생성합니다 [#gcp-create-connection]

    **Create connection** \[연결 생성]을 클릭하여 구성을 확인하고 통합을 설정하십시오.
  </Step>

  <Step>
    ### 데이터 보기 [#gcp-view-data]

    GCP 비용 데이터를 보려면 [**Intelligent summary (인텔리전트 요약)**](/docs/cci/financial-intelligence/intelligent-summary) 으로 이동하세요.

    <Callout variant="important">
      최초 처리에는 최대 12시간이 소요될 수 있습니다.
    </Callout>
  </Step>
</Steps>
