클라우드 분석 인텔리전스를 설정할 때 몇 가지 문제가 발생할 수 있습니다. 다음은 일반적인 문제, 가능한 원인, 그리고 효과적으로 문제를 해결하는 데 도움이 되는 문제 진단 및 해결 단계입니다.

## 뉴렐릭에 데이터가 표시되지 않음

뉴렐릭 계정에서 비용 사용 보고서(CUR), 실시간 비용 추정 또는 Kubernetes 비용 할당 데이터가 표시되지 않는 경우, 다음 단계에 따라 문제를 해결하세요.

* **사유:** 필수 조건을 충족하는 데 차이가 있을 수 있습니다.

* **해결책:**

  1. 모든 필수 전제 조건을 충족했는지 다시 한번 확인하세요.
  2. 설정 구성을 확인하세요.
  3. 문제가 지속되면 [클라우드 컴퓨팅 인텔리전스 지원팀](mailto:cci-support@newrelic.com) 에 문의하세요.

## 연결 끊김

* **이유:** 불안정한 네트워크 연결로 인해 설정이 중단될 수 있습니다.
* **해결 방법:** 인터넷 연결이 안정적인지 확인하고 설정을 다시 시도하세요. 문제가 지속되면 네트워크 관리자에게 문의하여 방화벽 제한 사항이 있는지 확인하세요.

## 부적절한 사용자 권한

* **이유:** 사용자에게 클라우드 솔루션 인텔리전스 설정에 필요한 권한이 없습니다.
* **해결 방법:** AWS 및 뉴렐릭 계정 권한을 검토하세요. 설정을 수행하는 데 필요한 적절한 권한이 있는지 확인하세요. 필요한 경우 관리자에게 문의하세요.

## ARN 설정 오류

* **메시지:** &quot;입력한 ARN은 AWS 계정에 대한 올바른 액세스를 허용하지 않습니다. ARN을 확인하고 다시 시도하세요.&quot;
* **사유:** 액세스를 허용하지 않는 잘못된 ARN 설정입니다.
* **해결 방법:** 입력한 ARN을 다시 한 번 확인하고 액세스에 필요한 설정과 일치하는지 확인하세요. 오류를 수정하고 설정을 다시 시도하세요.

## 외부 ID로 인한 보안 위험

* **메시지:** &quot;보안 위험: 입력한 ARN에는 광범위한 권한이 있습니다. 모든 &apos;외부 ID&apos;를 사용하여 역할을 맡을 수 있습니다. AWS 역할을 생성하는 동안 제공된 외부 ID &apos;%s&apos;를 externalId로 추가하세요.&quot;
* **이유:** 지나치게 관대한 외부 ID 정책은 보안 위험을 초래합니다.
* **해결 방법:** 제공된 외부 ID가 externalId로 사용되도록 AWS 역할 설정을 업데이트합니다.

## 버킷이 존재하지 않습니다

* **사유:** 지정된 AWS S3 버킷이 존재하지 않습니다.
* **해결 방법:** 버킷 이름이 AWS 계정의 버킷 이름과 일치하는지 확인하세요. 이름에는 접두사 `cost-and-usage-reports` 도 있어야 합니다. 버킷이 없으면 새로 만들거나, 설정에서 버킷 이름을 수정하세요.

## 버킷 이름 오류

* **사유:** 지정된 AWS S3 버킷이 존재하지 않거나 버킷 이름에 오타가 있습니다.
* **해결 방법:** 버킷 이름이 AWS 계정의 이름과 일치하고 접두사 `cost-and-usage-reports` 포함하는지 확인하세요. 이름이 올바르게 입력되었는지 확인하세요. 버킷이 존재하지 않으면 새로 만들거나 설정에서 이름을 수정하세요.

## 중복된 연결 이름

* **메시지:** &quot;%s 클라우드 분석 인텔리전스 AWS 연결 이름이 이미 존재합니다. 새로운 연결 이름을 입력하세요.&quot;
* **사유:** 입력한 연결 이름이 이미 사용 중입니다.
* **해결 방법:** 고유한 연결 이름을 선택하고 다시 시도하세요.

## 중복 지급인 계정 설정

* **메시지:** &quot;%s AWS Payer 계정이 이미 이 뉴렐릭 계정에 연결되어 있습니다.&quot;
* **사유:** 동일한 AWS 지불자 계정을 두 번 이상 구성하려고 했습니다.
* **해결 방법:** AWS 지불자 계정이 이미 연결되어 있지 않은지 확인하세요. 필요한 경우 다른 납부자 계정을 사용하세요.

## 추가 지원

이러한 문제 해결, 해결 단계를 수행한 후에도 문제가 지속되는 경우, [클라우드 적외선 인텔리전스 지원팀](mailto:cci-support@newrelic.com) 에 문의하세요.
