클라우드 매트리스 인텔리전스 통합 기능을 사용하면 cloud 공급자 계정을 보고 관리할 수 있습니다. 이 기능을 사용하면 특정 cloud 공급자 계정에 대한 통합을 일시 중지하거나 삭제할 수 있습니다.

1. <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence**</DNT> 으로 이동합니다.
2. 왼쪽 탐색 메뉴에서 **Integrations** \[통합] 탭을 선택하면 모든 cloud 비용 통합 목록이 표시됩니다.
3. 추가 세부 정보를 보려면 **Name** \[이름] 열에서 필요한 cloud 공급자 계정을 클릭하세요.

<Callout variant="important">
  현재는 의 cloud 비용만 AWS 지원됩니다.
</Callout>

4. 필요한 계정 옆에 있는 줄임표 아이콘을 클릭하고 다음 중 하나를 선택하세요.

   * **Pause** \[일시 중지]: 통합에서 데이터 수집을 일시적으로 중지합니다.
   * **Delete** \[삭제]: 계정에서 통합을 영구적으로 제거합니다.
