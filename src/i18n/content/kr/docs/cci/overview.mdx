<DNT>Cloud Cost Intelligence</DNT> 모든 cloud 공급업체에 걸쳐 cloud 비용의 가시성과 관리를 제공합니다. 비용 가시성과 인프라 옵저버빌리티 간의 격차를 해소하여 cloud 지출에 대한 전체적인 시각을 제공합니다.

기업이 클라우드 서비스를 빠르게 확장함에 따라, cloud 비용을 관리하고 추적하는 것은 점점 더 복잡해집니다. <DNT>Cloud Cost Intelligence</DNT> 은(는) 비용 할당에 대한 인사이트를 제공하여 잠재적인 비용 절감 기회를 식별할 수 있도록 합니다. cloud 지출을 실제 사용량과 일치시킴으로써 전략적 성장 이니셔티브를 지원하는 동시에 예산을 최적화할 수 있으며, 투자된 모든 금액이 회사의 전반적인 성공에 기여하도록 보장할 수 있습니다.

## 시작하다

클라우드 비용 인텔리전스에는 **Advanced CCU SKU** 및 [기능 제어 관리자](/docs/accounts/accounts-billing/new-relic-one-user-management/feature-control-manager/)를 통해 관리되는 액세스 권한이 필요합니다. cloud 비용 모니터링 및 최적화를 시작하려면 아래에서 cloud 제공업체를 선택하세요:

## 주요 특징들

### 지능형 요약

[지능형 요약](/docs/cci/financial-intelligence/intelligent-summary) 페이지는 <DNT>Cloud Cost Intelligence</DNT>의 기본 랜딩 페이지입니다. 모든 cloud 지출을 한 곳에서 표시하며, 서비스, 팀 및 지역별로 분류된 비용이 조직 내에서 어떻게 이동하는지 보여줍니다. 필요에 따라 페이지를 탐색하고 보기를 변경할 수 있습니다.

주요 기능은 다음과 같습니다:

* <DNT>**Sankey chart:**</DNT> 동적 Sankey 차트 시각화를 탐색하여 비용 세부 정보를 드릴다운하세요.
* <DNT>**Cost segregation:**</DNT> 서비스, 팀, 지역별로 구분된 비용
* <DNT>**Dynamic views:**</DNT> 다양한 보기 유형(예를 들어, 계정 보기) 간에 전환
* <DNT>**Breakdown by category:**</DNT> 애플리케이션, 서비스, 소유 팀 및 리전별로 세분화된 클라우드 비용
* <DNT>**Trend analysis:**</DNT> 전월 대비 비용 추세, 평균 비용 및 주요 비용 동인

### 비용 개요

[비용 개요](/docs/cci/financial-intelligence/cost-overview) 페이지는 서비스 수준부터 개별 리소스에 이르기까지 다양한 차원에서 비용을 자세히 분석하고 살펴볼 수 있는 드릴다운 기능을 제공합니다.

### Kubernetes 비용 할당

뉴렐릭 텔리 데이터를 사용하여 계약 이름별로 분류된 Kubernetes 비용을 확인하세요.

### FinOps 에이전트

FinOps 에이전트는 CCI에 비용 중심의 대화형 인텔리전스를 추가하며 [Ask AI](/docs/agentic-ai/new-relic-ai) 채팅 인터페이스에서 사용할 수 있습니다. 다음을 위해 사용합니다:

* 비용 데이터 및 추세에 대해 질문하기
* AI가 생성한 주별 및 월 누계(MTD) 지출 요약을 확인하세요.
* 비용 이상 징후 감지

<Callout variant="important">
  FinOps 에이전트는 현재 미리 보기 상태입니다. 이를 활성화하려면 [현재 미리 보기 상태인 기능에 액세스하려면 어떻게 해야 합니까?](/docs/cci/faq#preview-access)를 참조하십시오.
</Callout>

### 교차 계정 데이터 수집

<DNT>Cloud Cost Intelligence</DNT> 접근 권한이 있는 모든 뉴렐릭 계정에서 텔레메트리 데이터를 수집할 수 있습니다. <DNT>Cloud Cost Intelligence</DNT>에 단일 계정을 사용하더라도 액세스 권한이 있는 다른 모든 계정의 데이터도 해당 계정에서 수집 및 인사이트에 사용될 수 있습니다. 계정 권한이 내부 데이터 액세스 정책에 맞는지 확인하고 [설정](/docs/cci/configure-cci/general-setting)을 통해 동일하게 구성합니다.

## 작동 원리

### 데이터 수집 [#fetch-costs]

설정이 완료되면 <DNT>Cloud Cost Intelligence</DNT> 이(가) 지정된 객체 스토리지 컨테이너에서 비용 및 사용량 데이터에 자동으로 액세스합니다. 당사 서비스는 보고서가 저장소 컨테이너로 내보내진 후에만 보고서를 가져올 수 있습니다. 설정 중에 폴더 경로 접두사가 올바르게 구성되었는지 확인합니다.

cloud 제공업체는 일반적으로 하루에 여러 번 청구 데이터를 업데이트합니다. 초기 설정 후, <DNT>Cloud Cost Intelligence</DNT> 은(는) 매시간 저장 위치에서 최신 보고서를 가져옵니다. 가장 최근의 비용 데이터는 제공업체가 업데이트를 게시한 직후 당사 플랫폼에 나타납니다.

* **청구 파일 수집:** CCI 가격은 cloud 제공업체(AWS, Azure 또는 GCP)가 저장소 컨테이너에 게시하는 청구 내보내기 파일의 크기를 기준으로 합니다. 이 파일들은 비용 분석 및 인사이트를 구축하기 위해 <DNT>Cloud Cost Intelligence</DNT> 에 가져옵니다. 자세한 내용은 [클라우드 비용 인텔리전스 요금](/docs/cci/cci-pricing)을 참조하세요.

* **데이터 프라이버시**: <DNT>Cost and Usage Reports</DNT> 에는 클라우드 서비스 사용 및 비용에 대한 데이터가 포함되어 있으며, 개인 데이터는 포함되지 않아야 합니다. 스토리지에 <DNT>Cost and Usage Reports</DNT> 을(를) 추가하기 전과 뉴렐릭에 업로드하기 전에 전송하는 데이터에 개인 식별 정보(PII) 또는 민감한 정보가 없는지 확인하십시오. 데이터 삭제는 현재 지원되지 않습니다. PII 또는 민감한 정보가 포함된 <DNT>Cost and Usage Reports</DNT> 을(를) 생성하지 않는 것이 좋습니다.

* **데이터 보존:** 데이터는 최대 3년 동안 보존됩니다. 2년 후, 데이터는 검색 시간이 더 긴 저비용 스토리지로 이동됩니다.

* **액세스 보기:** 설치 중에 선택한 뉴렐릭 계정에 액세스할 수 있는 사람은 누구나 <DNT>Cloud Cost Intelligence</DNT> 내의 모든 데이터를 볼 수 있습니다. RBAC 및 액세스 정책 요구 사항에 맞는 적절한 액세스 권한이 있는 계정을 선택하십시오.

### 데이터 분석 및 보고

Cost and Usage Report \[비용 및 사용량 보고서] 데이터를 가져온 후, <DNT>Cloud Cost Intelligence</DNT> 은(는) 이 정보를 처리하고 분석하여 대시보드에서 실행 가능한 인사이트를 제공합니다. cloud 사용의 재정적 영향을 시각화하여 비용 동인과 최적화를 위한 잠재적 영역을 파악합니다.
