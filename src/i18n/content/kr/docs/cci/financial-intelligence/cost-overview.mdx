<DNT>**Cost Overview**</DNT> 페이지(이전 명칭: <DNT>**Cost Explorer**</DNT>)는 여러 차원으로 드릴다운할 수 있는 기능과 함께 상세한 비용 분석을 제공합니다.

cloud 지출 내역을 요약 정보와 함께 통합적으로 확인하려면 [지능형 요약](/docs/cci/financial-intelligence/intelligent-summary) 페이지를 참조하세요.

<img title="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" alt="Cost types highlighted on the Cloud Cost Intelligence Cost Overview page" src="/images/cost-overview.webp" />

## 클라우드 공급자 필터

cloud 제공업체별로 비용 데이터를 필터링하려면 페이지 왼쪽 상단의 드롭다운 메뉴에서 옵션을 선택하세요:

* <DNT>**All**</DNT>: 모든 cloud 공급자
* <DNT>**AWS**</DNT>: Amazon Web Services 전용
* <DNT>**Azure**</DNT>: Microsoft Azure 전용
* <DNT>**GCP**</DNT>: Google Cloud Platform 전용

## 상위 지출

<DNT>**Top Spends**</DNT> 위젯은 비용이 가장 많이 드는 영역을 식별하는 데 도움이 됩니다. 다음 기준으로 분류된 비용 보기:

* <DNT>**Service**</DNT>: 클라우드 서비스 유형별로 그룹화된 비용(예: AmazonEC2, Fastly Edge Cloud Platform, AmazonS3)
* <DNT>**Category**</DNT>: 사용 범주별로 정리된 비용(예: 컴퓨트, 기타, 네트워크)
* <DNT>**Region**</DNT>: 지리적 지역별 비용(예: us-west-2, Untagged, eastus). &quot;태그 없음&quot;은 리전 태그가 없는 리소스의 비용을 나타냅니다.
* <DNT>**Charge type**</DNT>: 청구 카테고리별 비용(예: Usage, SavingsPlanCoveredUsage, DiscountedUsage)

각 차원은 다음을 표시합니다:

* 총 비용 금액
* 이전 기간과 비교한 백분율 변화(선택한 날짜 범위 기준)
* 개별 비용이 포함된 상위 3개 항목
* <DNT>**Show summary**</DNT> 세부 보기를 확장하거나 접으려면 전환하세요

<Callout variant="tip">
  백분율 변화 지표를 사용하여 비용이 크게 증가한 서비스 또는 리전을 빠르게 파악하세요. 빨간색 표시기는 이전 기간에 비해 지출이 증가한 영역을 강조 표시합니다.
</Callout>

## 비용 트렌드

<DNT>**Cost Trends**</DNT> 섹션은 대화형 차트 및 컨트롤을 통해 시간 기반 비용 분석을 제공합니다.

### 차트 컨트롤

* <DNT>**View options**</DNT>: <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT> 또는 <DNT>**Monthly**</DNT> 뷰 간 전환
* <DNT>**Group by**</DNT>: 비용 데이터를 그룹화할 측정기준을 1-3개 선택합니다.
* <DNT>**View by**</DNT>: 시각화 형식을 선택합니다(사용 가능 여부는 그룹화 기준 선택에 따라 다릅니다)

### 데이터 이해

* <DNT>**Total cost**</DNT>: 비율 변화와 함께 집계 지출을 표시합니다
* <DNT>**Average cost**</DNT>: 비율 변화와 함께 일일 평균 지출을 표시합니다.
* <DNT>**Stacked bar chart**</DNT>: 다양한 서비스에 대해 색상으로 구분된 세그먼트로 시간에 따른 비용 분포를 시각화합니다

### 비용 데이터 테이블

차트 아래에 자세한 표가 표시됩니다:

* <DNT>**Cloud provider service**</DNT>: 서비스 이름
* <DNT>**Total Cost**</DNT>: 집계 지출 금액(정렬 가능)
* <DNT>**% of Spend**</DNT>: 총 지출 비율
* <DNT>**Compared Cost**</DNT>: 이전 기간과의 비용 비교
* <DNT>**Trend**</DNT>: 색상으로 구분된 백분율 변화 지표(감소는 녹색, 증가는 빨간색)

### 시각화 옵션

**그룹화 제약 조건:**

* **최소 선택 항목**: 1개 차원
* **최대 선택 가능 크기**: 3차원

**단일 group-by 선택 시:**

* <DNT>**Bar chart**</DNT>: 비용 데이터를 세로 막대로 표시합니다
* <DNT>**Line chart**</DNT>: 시간 경과에 따른 비용 추세를 보여줍니다
* <DNT>**Tree chart**</DNT>: 비용 데이터의 계층적 시각화
* <DNT>**Data table**</DNT>: 비용 데이터 표 보기

**여러 그룹화 기준을 선택한 경우:**

* <DNT>**Heat map**</DNT>: 색상으로 구분된 시각화
* <DNT>**Data table**</DNT>: 비용 데이터 표 보기

## 드릴다운 기능

비용 데이터 테이블에서 서비스(예: AmazonEC2)를 클릭하여 카테고리, 리전 또는 요금 유형별 세부 분석을 확인하세요. 다시 클릭하여 개별 리소스 비용을 확인하세요.

컴퓨트, 네트워크, 스토리지 등 모든 서비스 유형에서 드릴다운 기능을 지원합니다.

## 필터 및 사용자 정의

<DNT>**Cost Overview**</DNT> 페이지는 특정 비용 세그먼트에 초점을 맞출 수 있도록 필터링 및 사용자 지정 옵션을 지원합니다.

### 날짜 범위 선택기

페이지 오른쪽 상단에서 날짜 범위 선택기를 사용하여 다음을 수행합니다:

* 사전 정의된 기간 선택
* 커스텀 시작 및 종료 날짜 설정
* 선호하는 시간대로 비용 보기(기본적으로 UTC 표시)

### 엔지니어링 비용 필터

조직에서 비용 할당 태그를 사용하는 경우, <DNT>**Engineering costs**</DNT> 태그로 필터링하여 cloud 환경에서 태그가 지정된 특정 팀, 프로젝트 또는 비용 센터로 분석 범위를 좁힐 수 있습니다.
