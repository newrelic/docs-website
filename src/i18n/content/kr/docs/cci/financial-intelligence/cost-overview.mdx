The <DNT>**Cost Overview**</DNT> page (formerly known as <DNT>**Cost Explorer**</DNT>) provides detailed cost analysis with drill-down functionality across multiple dimensions.

## Cloud provider filters

To filter your cost data by cloud provider, select an option from the dropdown menu at the top left of the page:

* <DNT>**All**</DNT>: All cloud providers
* <DNT>**AWS**</DNT>: Amazon Web Services only
* <DNT>**Azure**</DNT>: Microsoft Azure only
* <DNT>**GCP**</DNT>: Google Cloud Platform only

## Top Spends

The <DNT>**Top Spends**</DNT> widget helps you identify your highest cost areas. View costs broken down by:

* <DNT>**Service**</DNT>: Costs grouped by cloud service type (for example, AmazonEC2, Fastly Edge Cloud Platform, AmazonS3)
* <DNT>**Category**</DNT>: Costs organized by usage category (for example, Compute, Other, Network)
* <DNT>**Region**</DNT>: Costs by geographical region (for example, us-west-2, Untagged, eastus). &quot;Untagged&quot; indicates costs from resources without regional tags
* <DNT>**Charge type**</DNT>: Costs by billing category (for example, Usage, SavingsPlanCoveredUsage, DiscountedUsage)

Each dimension displays:

* Total cost amount
* Percentage change compared to the previous period (based on your selected date range)
* Top three items with their individual costs
* <DNT>**Show summary**</DNT> toggle to expand or collapse detailed views

<Callout variant="tip">
  Use the percentage change indicators to quickly identify services or regions with significant cost increases. Red indicators highlight areas where spending has grown compared to the previous period.
</Callout>

## Cost Trends

The <DNT>**Cost Trends**</DNT> section provides time-based cost analysis with interactive charts and controls.

### Chart controls

* <DNT>**View options**</DNT>: Switch between <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT>, or <DNT>**Monthly**</DNT> views
* <DNT>**Group by**</DNT>: Select 1 to 3 dimensions to group cost data
* <DNT>**View by**</DNT>: Choose visualization format (availability depends on your group-by selections)

### Understanding the data

* <DNT>**Total cost**</DNT>: Displays aggregate spending with percentage change
* <DNT>**Average cost**</DNT>: Displays daily average spending with percentage change
* <DNT>**Stacked bar chart**</DNT>: Visualizes cost distribution over time with color-coded segments for different services

### Cost data table

Below the chart, a detailed table displays:

* <DNT>**Cloud provider service**</DNT>: Service name
* <DNT>**Total Cost**</DNT>: Aggregate spending amount (sortable)
* <DNT>**% of Spend**</DNT>: Percentage of total spending
* <DNT>**Compared Cost**</DNT>: Cost comparison with previous period
* <DNT>**Trend**</DNT>: Percentage change indicator with color coding (green for decrease, red for increase)

### Visualization options

**Group-by constraints:**

* **최소 선택 항목**: 1개 차원
* **최대 선택 가능 크기**: 3차원

**With single group-by selection:**

* <DNT>**Bar chart**</DNT>: Displays cost data as vertical bars
* <DNT>**Line chart**</DNT>: Shows cost trends over time
* <DNT>**Tree chart**</DNT>: Hierarchical visualization of cost data
* <DNT>**Data table**</DNT>: Tabular view of cost data

**With multiple group-by selections:**

* <DNT>**Heat map**</DNT>: Color-coded visualization
* <DNT>**Data table**</DNT>: Tabular view of cost data

## 드릴다운 기능

Click any service (for example, AmazonEC2) in the cost data table to view detailed breakdowns by category, region, or charge type. Click again to see individual resource costs.

## Filters and customization

The <DNT>**Cost Overview**</DNT> page supports filtering and customization options to help you focus on specific cost segments.

### Date range selector

At the top-right of the page, use the date range selector to:

* Choose a predefined time period
* Set custom start and end dates
* View costs in your preferred timezone (UTC displayed by default)

### Engineering costs filter

If your organization uses cost allocation tags, you can filter by the <DNT>**Engineering costs**</DNT> tag to narrow analysis to specific teams, projects, or cost centers tagged in your cloud environment.
