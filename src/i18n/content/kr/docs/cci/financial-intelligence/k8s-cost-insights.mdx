Kubernetes 비용 할당 페이지를 통해 Kubernetes 환경에 맞는 인사이트에 액세스하세요. 이는 뉴렐릭의 텔레메트리 데이터를 활용하여 Kubernetes 와 관련된 비용에 대한 보기를 제공합니다.

Kubernetes 비용 할당 페이지를 사용하여 시간 경과에 따라 각 애플리케이션에서 발생한 비용을 보고 비용 분석을 자세히 알아보세요.

## 할당 개요

할당 개요는 시간이 지남에 따라 각 비용에 의해 발생하는 비용을 설명하는 차트 시각화를 제공합니다.

## 자세한 비용 분석

Kubernetes 비용 할당 보기는 또한 애플리케이션 이름별로 리소스 사용량과 비용을 세부적으로 나타낸 표를 제공합니다.

* **CPU 사용량:** 각 애플리케이션이 소비한 CPU 리소스의 백분율을 표시합니다.
* **CPU 비용:** 각 구성의 CPU 사용량에 대한 비용 속성을 표시합니다.
* **메모리 사용량:** 각 구성에서 사용하는 메모리 리소스를 백분율로 나열합니다.
* **메모리 비용:** 메모리 사용과 관련된 비용을 반영합니다.
* **총 비용:** 모든 리소스 사용량을 통합하여 각 애플리케이션에서 발생한 총 비용을 제공합니다.

## 혜택

Kubernetes 비용 할당 기능을 활용하면 다음을 수행할 수 있습니다.

* **리소스 사용 최적화:** 활용도가 낮거나 비용이 과도하게 많이 드는 애플리케이션을 식별하여 리소스 할당을 효과적으로 조정합니다.
* **비용 관리:** Kubernetes 환경 내의 특정 비용 요인에 대한 가시성을 확보하여 보다 정보에 입각한 재정적 결정을 내릴 수 있습니다.
* **효율성 향상:** 비즈니스 목표에 맞춰 리소스 관리 전략에 집중하여 cloud 리소스를 효율적으로 사용합니다.

Kubernetes 비용 할당 페이지는 Kubernetes 비용을 사전에 관리하는 데 필요한 인사이트를 제공하여 전략적 의사 결정과 재무 최적화를 지원합니다.
