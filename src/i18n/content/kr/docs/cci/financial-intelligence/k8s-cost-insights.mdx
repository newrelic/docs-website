## 개요

Kubernetes 비용 할당 페이지를 통해 Kubernetes 환경에 맞는 인사이트에 액세스하세요. 이는 뉴렐릭의 텔레메트리 데이터를 활용하여 Kubernetes 와 관련된 비용에 대한 보기를 제공합니다.

Kubernetes 비용 할당 페이지를 사용하여 각 애플리케이션이 시간에 따라 발생시킨 비용과 자세한 비용 분석을 확인할 수 있습니다.

## 할당 개요

할당 개요는 시간이 지남에 따라 각 예산에 의해 발생하는 비용을 보여주는 차트 시각화를 제공합니다.

## 자세한 비용 분석

Kubernetes 비용 할당 보기에서는 애플리케이션 이름별 리소스 사용량 및 비용에 대한 자세한 분석 정보를 표 형식으로 제공합니다.

* **CPU 사용량:** 각 애플리케이션이 소비한 CPU 리소스의 백분율을 표시합니다.
* **CPU 비용:** 각 구성의 CPU 사용량에 대한 비용 속성을 표시합니다.
* **메모리 사용량:** 각 구성에서 사용하는 메모리 리소스를 백분율로 나열합니다.
* **메모리 비용:** 메모리 사용과 관련된 비용을 반영합니다.
* **총 비용:** 모든 리소스 사용량을 통합하여 각 애플리케이션에서 발생한 총 비용을 제공합니다.

## 미사용 및 미할당 비용

이러한 비용 지표는 클러스터 효율성을 측정하는 데 도움이 됩니다.

### 미사용 비용

사용되지 않는 비용은 클러스터에 할당되었지만 요청이 없는 부분의 양을 나타냅니다.

**예시**: 총 비용이 $5.60이고 이를 구현, 배포에 할당했지만 실제로 사용한 비용이 $4.64에 불과한 경우 $0.96은 미사용 비용입니다.

### 미배정 비용

할당되지 않은 비용은 클러스터에서 어떤 소스에도 할당되지 않은 부분의 비율입니다.

이는 쿠버네티스 클러스터의 효율성 측정 지표입니다.

## 혜택

Kubernetes 비용 할당 기능은 다음과 같은 결과를 제공합니다.

* **최적화된 리소스 사용:** 활용도가 낮거나 비용이 과도하게 많이 드는 애플리케이션을 파악하여 효과적인 리소스 할당 조정을 할 수 있습니다.
* **비용 관리:** Kubernetes 환경 내 특정 비용 발생 요인에 대한 Insights 보다 현명한 재정적 결정을 내릴 수 있습니다.
* **향상된 효율성:** 비즈니스 목표에 맞춰 조정된 리소스 관리 전략은 cloud 리소스를 효율적으로 활용할 수 있도록 보장합니다.

Kubernetes 비용 할당 페이지는 Kubernetes 비용을 사전에 관리하고 전략적 의사 결정 및 재정 최적화를 지원하는 데 필요한 정보를 제공합니다.
