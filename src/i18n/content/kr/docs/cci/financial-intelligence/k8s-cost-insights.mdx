<DNT>**Kubernetes Costs**</DNT> 페이지를 통해 Kubernetes 환경에 특화된 인사이트에 액세스하십시오. 이 기능은 뉴렐릭의 텔레메트리 데이터를 활용하여 쿠버네티스 클러스터와 관련된 비용에 대한 상세한 뷰를 제공합니다.

<DNT>**Kubernetes Costs**</DNT> 페이지를 사용하여 시간 경과에 따라 각 애플리케이션에서 발생한 비용과 상세한 비용 분석을 확인하세요.

## 필터 및 사용자 정의

### 클라우드 공급자 필터

cloud 제공업체별로 비용 데이터를 필터링하려면 페이지 왼쪽 상단의 드롭다운 메뉴에서 옵션을 선택하세요:

* <DNT>**All**</DNT>: 모든 cloud 공급자
* <DNT>**AWS**</DNT>: Amazon Web Services 전용
* <DNT>**Azure**</DNT>: Microsoft Azure 전용

### 날짜 범위 선택기

페이지 오른쪽 상단에서 날짜 범위 선택기를 사용하여 다음을 수행합니다:

* 사전 정의된 기간 선택
* 커스텀 시작 및 종료 날짜 설정
* 선호하는 시간대로 비용 보기(기본적으로 UTC 표시)

### 엔지니어링 비용 필터

조직에서 비용 할당 태그를 사용하는 경우, <DNT>**Engineering costs**</DNT> 태그로 필터링하여 cloud 환경에서 태그가 지정된 특정 팀, 프로젝트 또는 비용 센터로 분석 범위를 좁힐 수 있습니다.

## 요약

요약 섹션에는 선택한 기간에 대한 4가지 주요 비용 메트릭이 표시됩니다:

* <DNT>**Total cost**</DNT>: 이전 기간과 비교한 백분율 변화를 포함한 모든 쿠버네티스 클러스터의 총 지출
* <DNT>**Average cost (Daily)**</DNT>: 비율 변화가 포함된 일일 평균 지출
* <DNT>**Unused cost**</DNT>: 할당되었지만 requests를 받지 않는 리소스의 비용(백분율 변화 포함)
* <DNT>**Unallocated cost**</DNT>: 어떤 소스에도 할당되지 않은 클러스터 리소스의 비용, 백분율 변화 포함

## 인사이트

Insights 섹션은 Kubernetes 비용을 분석하는 데 도움이 되는 두 개의 시각화 위젯을 제공합니다.

### 비용 트렌드

<DNT>**Cost Trends**</DNT> 위젯은 시간 경과에 따라 Kubernetes 비용이 어떻게 변하는지 보여주는 라인 차트를 표시합니다. 보기 옵션을 사용하여 <DNT>**Hourly**</DNT>, <DNT>**Daily**</DNT>, <DNT>**Weekly**</DNT> 또는 <DNT>**Monthly**</DNT> 보기 간에 전환할 수 있습니다.

### 상위 20

<DNT>**Top 20**</DNT> 위젯은 트리 차트 시각화를 사용하여 비용이 가장 높은 영역을 표시합니다. 기본적으로 비용별 상위 20개 클러스터가 표시됩니다. 다음을 사용하여 보기를 사용자 정의할 수 있습니다:

* <DNT>**Dropdown filter**</DNT>: 표시할 항목 선택(<DNT>**Cluster name, by cost**</DNT>)
* <DNT>**Group by**</DNT>: 비용 데이터를 그룹화할 측정기준을 선택하세요
* <DNT>**View by**</DNT>: <DNT>**Tree chart**</DNT> 및 다른 시각화 형식 간에 전환

## 비용 및 사용량 분석

<DNT>**Cost &amp; Usage Analysis**</DNT> 섹션에서는 클러스터 이름별 리소스 사용량 및 비용 세부 내역이 포함된 상세 테이블을 제공합니다:

* <DNT>**Cluster name**</DNT>: 쿠버네티스 클러스터 이름
* <DNT>**CPU cost**</DNT>: 각 클러스터의 CPU 사용량에 할당된 비용
* <DNT>**CPU usage**</DNT>: 각 클러스터에서 소비하는 CPU 리소스의 백분율
* <DNT>**Memory cost**</DNT>: 메모리 사용량과 관련된 비용
* <DNT>**Memory usage**</DNT>: 각 클러스터에서 사용한 메모리 리소스 백분율
* <DNT>**Unused Cost**</DNT>: 할당되었지만 사용되지 않은 리소스의 비용
* <DNT>**Unallocated cost**</DNT>: 어떤 소스에도 할당되지 않은 클러스터 리소스 비용
* <DNT>**Total cost**</DNT>: 모든 리소스 사용량을 통합하여 각 클러스터에서 발생한 총 비용(정렬 가능)

## 미사용 및 미할당 비용 이해

이러한 비용 지표는 클러스터 효율성을 측정하는 데 도움이 됩니다.

### 미사용 비용

미사용 비용은 할당되었지만 어떠한 requests도 받지 않는 리소스를 나타냅니다.

**예시**: 총 비용이 $5.60이고 이를 구현, 배포에 할당했지만 실제로 사용한 비용이 $4.64에 불과한 경우 $0.96은 미사용 비용입니다.

### 미배정 비용

할당되지 않은 비용은 어떤 소스에도 할당되지 않은 클러스터 리소스를 나타냅니다.

이러한 메트릭은 클러스터 효율성을 개선하고 낭비를 줄일 수 있는 기회를 파악하는 데 도움이 됩니다.

<Callout variant="tip">
  <DNT>**Cost Trends**</DNT>, <DNT>**Cost Breakdown (Total vs Used)**</DNT> 및 미사용 및 미할당 비용에 대한 자세한 정보를 보려면 클러스터를 선택하세요.
</Callout>

## 혜택

Kubernetes Cost Insights 기능은 다음과 같은 결과를 제공합니다:

* <DNT>**Optimized resource usage**</DNT>: 활용도가 낮거나 비용이 과도하게 많이 드는 클러스터에 대한 가시성을 통해 효과적인 리소스 할당 조정이 가능합니다.
* <DNT>**Controlled expenses**</DNT>: Kubernetes 환경 내 특정 비용 동인에 대한 Insights를 통해 더 많은 정보에 입각한 재무 결정을 내릴 수 있습니다.
* <DNT>**Enhanced efficiency**</DNT>: 비즈니스 목표에 부합하는 리소스 관리 전략은 cloud 리소스의 효율적인 사용을 보장합니다.

<DNT>**Kubernetes Costs**</DNT> 페이지는 Kubernetes 비용을 선제적으로 관리하는 데 필요한 인사이트를 제공하여 전략적 의사 결정 및 재무 최적화를 지원합니다.
