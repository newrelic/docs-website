클라우드 비용 인텔리전스는 두 가지 소스를 기반으로 비용 최적화 권장 사항을 제공하여 AWS 인프라 전반에 걸쳐 비용 절감 기회를 파악하고 실행할 수 있도록 지원합니다. 이 기능은 AWS Trusted Advisor와 AWS 고유의 Native Intelligence Engine을 사용하여 리소스 사용 패턴을 분석합니다.

<Callout variant="important">
  현재 AWS에 대한 비용 최적화 권장 사항을 이용할 수 있습니다.
</Callout>

추천 기능을 다음과 같은 용도로 사용할 수 있습니다.

* 과잉 생산되거나 활용도가 낮은 자원을 파악하고 투자 기회를 모색하십시오.
* 월별 및 연간 예상치를 통해 잠재적 절감액을 확인하세요.
* 최적화 작업의 진행 상황을 추적하고 우선순위를 지정합니다(승인, 거부 또는 보류로 표시).

## 추천 출처

### AWS 신뢰할 수 있는 고문

AWS Trusted Advisor는 AWS에서 직접 비용 최적화 권장 사항을 제공합니다.

#### 전제 조건

클라우드 비용 인텔리전스와 함께 **Trusted Advisor** 권장 사항을 사용하려면 AWS 계정에서 AWS Trusted Advisor 비용 인사이트에 대한 액세스를 설정하고 구성해야 합니다.

#### 길찾기 방법

설정이 완료되면 **Trusted Advisor**의 권장 사항이 다른 권장 사항과 함께 **Cloud Cost Intelligence Recommendations** \[클라우드 비용 인텔리전스 권장 사항] 페이지에 표시됩니다.

<Callout variant="important">
  클라우드 비용 인텔리전스는 초기 AWS 계정 통합 중에 **Trusted Advisor** 상태를 확인합니다. **Trusted Advisor**가 비활성 상태인 경우 클라우드 컴퓨팅 인텔리전스는 이 통합 구성 요소를 건너뜁니다.

  **Trusted Advisor**가 나중에 활성화되면 클라우드 데이터 동기화 기능이 자동으로 업데이트를 감지하고 활성화 후 7일 이내에 데이터를 동기화합니다.
</Callout>

### CCI 네이티브 최적화 엔진

CCI의 기본 최적화 엔진은 리소스 활용률과 사용 기록을 분석하여 인스턴스 크기를 최적화하고 AWS Savings Plans 약정을 통해 비용을 절감할 수 있는 기회를 파악합니다.

#### 작동 원리

엔진은 두 가지 영역에서 사용량 데이터를 분석합니다.

* **적정 크기 조정**: CPU 및 메모리 사용량을 확인하고, 현재 인스턴스 유형을 성능 요구 사항과 비교하여 성능에 영향을 주지 않고 크기를 줄일 수 있는 인스턴스를 식별합니다.
* **절약 계획**: 사용 패턴을 검토하여 일관된 워크로드를 찾고 AWS 절약 계획 약정을 통해 얻을 수 있는 잠재적 절감액을 계산합니다.

#### 어떤 권장 사항을 제공하는가?

이 엔진은 비용 절감을 위한 권장 사항을 제공합니다.

* **사용자 유형 변경**: 현재 설정, CPU 및 메모리 사용량, 영향을 받은 사용자 수를 포함한 특정 수정(예: im4gn.16xlarge에서 t2.micro로 마이그레이션).
* **저축 계획 약정**: 할인 가격 적용을 위해 일정한 사용량을 유지하도록 권장합니다.
* **절감액 예상치**: 각 권장 사항에 대한 월별 및 연간 비용 절감액.

## 시청 추천

비용 최적화 권장 사항을 확인하려면 다음을 참조하세요.

1. **[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Cloud Cost Intelligence**로 이동하세요.
2. 왼쪽 탐색 메뉴에서 **Recommendations** \[추천]을 클릭하세요.

### 개요 섹션

개요 섹션에서는 주요 비용 최적화 방안을 보여줍니다.

* **총 잠재력**: 보류 중인 모든 권장 사항에 따른 총 절감액
* **확인됨**: 확인된 권장 사항으로 인한 절감액
* **기각됨**: 기각된 권고안으로 인한 절감액
* **조치 없음**: 보류 중인 권고 사항으로 인한 절감액
* **연간 잠재 효과**: 예상 연간 절감액

### 요약 측정항목

요약 섹션에는 추천 횟수가 표시됩니다.

* **총계**: 이용 가능한 모든 추천 사항
* **승인됨**: 승인된 권고사항
* **보류 중**: 조치가 필요한 권고 사항
* **기각됨**: 기각된 권고 사항

### 보기 추천 상세 정보

각 추천 카드에는 다음 내용이 표시됩니다.

* **권장 제목**: 최적화 작업 (예: im4gn.16xlarge에서 t2.micro로 크기 축소)
* **출처**: 추천 소스(예: CCI 네이티브 모델, Trusted Advisor)
* **서비스 유형**: 클라우드 서비스 (예: Amazon EC2)
* **월별 절감액**: 잠재적인 월별 비용 절감액
* **생성 타임스탬프**: 생성 날짜 및 시간

추천 항목을 클릭하여 확인하세요.

* **예상 절감액**: 월별 및 연간 절감액 내역
* **클라우드 제공업체 계정**: AWS 계정 ID
* **리소스 세부 정보**: 현재 설정 및 권장 변경 사항
* **발생 횟수**: 일치하는 인스턴스 수
* **원래대로 설정**: 현재 상태로 설정
* **실제 CPU 사용률**: 현재 CPU 활용률
* **실제 메모리 사용량**: 현재 메모리 사용량

### 권고 사항에 대한 조치

각 권장 사항에 대해 다음을 수행할 수 있습니다.

* **확인**: 검토 완료 및 조치 진행 중으로 표시 (권장 사항을 실행할 계획이거나 실행 중일 때 사용)
* **해제**: 활성 목록에서 제거합니다(사용 사례 또는 비즈니스 요구 사항에 적용되지 않을 때 사용).
* **조치 없음(보류 중)**: 추후 검토를 위해 보류 상태로 유지합니다.
* **메모 추가**: 향후 참조 및 팀 간 협업을 위해 메모를 추가하고 저장하세요.

이러한 상태 정보는 비용 최적화 노력을 체계화하고 우선순위를 정하는 데 도움이 되며, 개요 및 요약 섹션에서 진행 상황을 추적할 수 있습니다.
