  <Collapser id="kubernetes-cost-attributes" title="Kubernetes 비용 이벤트 유형">
    뉴렐릭에서 **All Capabilities &gt; CloudCost Intelligence** 로 이동합니다. 하단 쿼리 콘솔 서랍을 열고 다음 NRQL 쿼리를 입력합니다.

    ```sql
    FROM KubernetesCost SELECT * SINCE 1 week ago
    ```

    정확한 Kubernetes 비용을 계산하려면 비용 쿼리에서 `cpu_weightage + memory_weightage` 시위를 곱하세요. 다음 예에서는 올바른 쿼리 형식을 보여줍니다.

    ```sql
    SELECT SUM(net_unblended_cost * (cpu_weightage + memory_weightage)) FROM KubernetesCost SINCE '2025-09-01' UNTIL '2025-09-10'
    ```

    이 쿼리는 `net_unblended_cost` 을 CPU와 메모리 가중치의 합으로 곱하여 리소스 사용량에 따라 정확한 비용 할당을 제공합니다.
