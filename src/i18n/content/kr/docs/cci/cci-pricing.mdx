<DNT>Cloud Cost Intelligence</DNT> 은(는) Advanced 컴퓨트 제품입니다. 요금은 cloud 공급자가 게시하고 CCI가 처리하는 청구 내보내기 데이터의 양을 기준으로 책정되며, 이를 청구 파일 수집이라고 합니다.

<Callout variant="tip">
  최신 요금 세부 정보를 확인하거나 볼륨 기반 옵션에 대해 논의하려면 뉴렐릭 계정 팀에 문의하세요.
</Callout>

## 청구 파일 인제스트 속도

<DNT>Cloud Cost Intelligence</DNT> 의 정가는 청구 파일 수집 **GB당 23 aCCU** 입니다.

## 청구 파일 수집이란 무엇인가요?

청구 파일 수집은 cloud 공급자(AWS, Azure 또는 GCP)가 지정된 저장소 컨테이너에 게시하는 청구 내보내기 파일의 크기를 의미합니다. <DNT>Cloud Cost Intelligence</DNT> 은(는) 비용 분석 및 인사이트를 구축하기 위해 이러한 파일들을 가져옵니다.

<Callout variant="important">
  청구 파일 수집은 메트릭, 로그 및 트레이스와 같은 스트리밍 텔레메트리 데이터에 적용되는 뉴렐릭의 표준 [데이터 수집](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/data-ingest-billing/)과는 다릅니다. 이 두 가지는 별도로 측정되고 청구됩니다.
</Callout>

뉴렐릭 요금 체계에 대한 전반적인 개요는 [뉴렐릭 요금을](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/) 참조하세요.
