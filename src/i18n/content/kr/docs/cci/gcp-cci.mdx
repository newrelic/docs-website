클라우드 컴퓨팅 인텔리전스를 위한 Google Cloud Platform (GCP) 통합을 사용하면 뉴롤릭 내에서 직접 GCP cloud 지출을 모니터링하고 분석할 수 있습니다. 이 통합은 서비스 계정 기반 승인 방법을 사용하여 GCP 청구 및 가격 데이터에 안전하게 액세스합니다.

이 통합을 설정하면 뉴렐릭은 사용자를 대신하여 GCP 환경에 전용 서비스 계정을 만듭니다. 이 서비스 계정에 BigQuery 데이터 세트에 대한 읽기 전용 액세스 권한을 부여하면 뉴렐릭은 GCP 리소스에 대한 광범위한 액세스 권한을 요구하지 않고도 비용 데이터를 검색하고 분석할 수 있습니다.

## 주요 특징들

* 뉴렐릭 계정당 자동 서비스 계정 생성
* BigQuery를 통한 청구 데이터에 대한 안전한 읽기 전용 액세스
* 청구 내보내기 및 cloud 가격 내보내기 표에 모두 액세스 가능
* 연결 검증 및 오류 보고

## 전제 조건

GCP 통합을 설정하기 전에 다음 사항을 확인하세요.

* 에 대한 연결 예측 예측 - GCP 프로젝트가 이미 연결되어 있습니다. [Google Cloud Platform 서비스를 뉴렐릭에 연결하기](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)

* 활성 GCP 계정에서 청구가 활성화되었습니다.

* 다음 테이블을 사용하여 BigQuery 데이터 세트를 구성했습니다.

  * 청구 내보내기 테이블(GCP 청구 데이터 포함)
  * 클라우드 가격 내보내기 표(GCP 가격 정보 포함)

* Google Cloud Console에 대한 액세스가 확인되었습니다.

## 클라우드 솔루션에 Google Cloud Platform 연결 인텔리전스

<Steps>
  <Step>
    ### 계정을 선택하세요

    뉴렐릭 계정을 선택하여 통합을 시작하세요.

    1. <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Cloud Cost Intelligence - Google Cloud Platform**</DNT> 으)로 이동합니다.
    2. 드롭다운 메뉴에서 계정을 선택하고 **Continue** \[계속을] 클릭하세요.
  </Step>

  <Step>
    ### GCP 연결 설정

    1. GCP 청구 계정을 뉴렐릭에 연결하세요.

       * **연결 이름** - 기술 이름을 입력합니다(예: &quot;GCP 운영 환경&quot;)
       * **프로젝트 ID** - GCP 프로젝트 ID를 입력하세요
       * **데이터 세트 이름** - BigQuery 데이터 세트 이름을 입력하세요.
       * **청구 테이블 이름** - 청구 테이블 이름을 제공하세요
       * **가격표 이름** - 가격표 이름을 입력하세요

    2. **계속** 을 클릭합니다.
  </Step>

  <Step>
    ### GCP에 대한 뉴렐릭 액세스 권한 부여

    뉴렐릭은 BigQuery 에서 Cloud Billing 데이터에 액세스할 수 있도록 조직에 고유한 서비스 계정 ID를 제공합니다.

    1. 표시된 서비스 계정 ID를 복사합니다.
    2. **Google Cloud Console**을 열고 BigQuery 청구 테이블에 **BigQuery DataViewer** 역할을 서비스 계정에 부여합니다.
    3. 이 페이지로 돌아와서 **Continue** \[계속] 을 클릭하세요.
  </Step>

  <Step>
    ### 테스트하고 연결을 생성합니다

    **Create connection** \[연결 만들기]를 클릭하여 구성을 확인하고 통합을 설정합니다.
  </Step>

  <Step>
    ### 데이터 수집 확인

    GCP 비용 데이터를 보려면 [**Cost Overview** \[비용 개요\]](/docs/cci/financial-intelligence/cloud-cost-insights) 로 이동하세요.

    <Callout variant="important">
      처음으로 데이터를 처리하는 데 최대 12시간이 걸릴 수 있습니다.
    </Callout>
  </Step>
</Steps>
