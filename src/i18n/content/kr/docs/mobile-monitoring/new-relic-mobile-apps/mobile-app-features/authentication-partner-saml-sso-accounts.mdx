<Callout variant="important">
  이 리소스는 New Relic 파트너를 위한 것입니다. 일반 New Relic 계정의 사용자 인증에 대해서는 [인증](/docs/accounts/accounts-billing/new-relic-one-user-management/user-authentication-saml-sso-scim-more) 을 참조하십시오.
</Callout>

파트너 계정 사용자는 일반적으로 SAML-SSO를 사용하여 New Relic 파트너 사이트를 통해 로그인합니다. New Relic 계정에 대해 별도의 비밀번호나 인증 정보가 없을 수 있습니다.

New Relic 모바일 앱에 처음 로그인할 때 New Relic 파트너 계정과 연결된 이메일 주소를 사용하는 경우 New Relic은 인증을 위해 확인 이메일을 보내드립니다. Android 앱 사용자에게도 알림 메시지가 표시됩니다.

<Callout variant="important">
  인증 이메일은 발송 후 [20분이 지나면 만료됩니다](#email) .
</Callout>

## 이메일 주소 확인 [#authentication]

New Relic 파트너를 통해 제공되는 SAML-SSO 계정을 사용하여 인증하려면:

1. New Relic 모바일 앱에서 파트너 계정과 연결된 이메일 주소를 입력합니다.

2. <DNT>
     **I don't have a password**
   </DNT>

   선택합니다.

3. 20분 이내에 모바일 장치에서 인증 이메일을 검색합니다.

4. 뉴렐릭에 로그인하려면 이메일에서

   <DNT>
     **Authenticate**
   </DNT>

   버튼(Android 사용자) 또는 이메일 링크(Android 또는 iOS 사용자)를 선택하세요.

New Relic 모바일 앱으로 리디렉션되고 파트너 계정에 로그인됩니다.

## 이메일 문제 해결 [#email]

다음은 몇 가지 문제 해결 팁입니다.

* 모바일 기기의 이메일 받은편지함에서 뉴렐릭이 보낸 인증 메시지를 찾을 수 없는 경우

  <DNT>
    **Spam**
  </DNT>

  폴더를 확인하세요.

* 20분 기한을 놓치셨다면 New Relic 모바일 앱에 다시 로그인한 후 링크를 선택하여 인증 이메일을 다시 보내세요.
