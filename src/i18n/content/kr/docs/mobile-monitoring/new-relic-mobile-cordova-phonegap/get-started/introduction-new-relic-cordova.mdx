iOS 및 Android용 공식 New Relic Cordova 플러그인을 사용하면 개발자가 모바일 에이전트를 [Cordova 애플리케이션](http://cordova.apache.org/) 에 쉽게 포함할 수 있습니다. JavaScript로 작성된 플러그인에는 New Relic의 기본 에이전트가 자동으로 포함되어 [모바일 모니터링 및 성능 가시성](https://newrelic.com/mobile-monitoring) 을 제공합니다.

## 특징

New Relic Cordova 플러그인은 다음을 수행합니다.

* Cordova 애플리케이션에 추가된 플랫폼을 감지하고 적절한 New Relic 모바일 에이전트( [Android](/docs/releases/android) 또는 [iOS](/docs/releases/ios) )의 최신 릴리스를 적용합니다.
* iOS 기호화 파일을 업로드하기 위한 빌드 후 스크립트를 추가합니다.
* Android Proguard 매핑 파일을 업로드합니다.
* Cordova를 통해 구축된 모바일 애플리케이션을 자동으로 계측합니다.

자세한 내용 [은 Github의 New Relic Cordova 플러그인을](https://github.com/newrelic/newrelic-cordova-plugin) 참조하십시오.

## 호환성 및 요구 사항 [#prerequisites]

New Relic Cordova 플러그인에는 다음이 필요합니다.

* 코르도바 7.x 이상
* 노드 6.0 이상
* 코르도바 CLI 도구
* 프로젝트를 위한 [Android](https://cordova.apache.org/docs/en/latest/guide/platforms/android) 또는 [iOS](https://cordova.apache.org/docs/en/latest/guide/platforms/ios/index.html) Cordova 플랫폼
* 모바일 [애플리케이션 토큰](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token)

네트워크 요청은 webview 및 기본 프레임워크에 대한 iOS 장치에만 기록됩니다. Android는 기본 네트워크 호출만 녹음합니다.

Cordova는 일반적으로 기본적으로 예외를 처리합니다. Cordova 플러그인은 Cordova가 처리하지 않는 포착되지 않은 예외(충돌)를 기록합니다.

## Cordova 플러그인 설치 [#install]

설치하기 전에 Android 또는 iOS 플랫폼을 Cordova 프로젝트에 추가하기 위한 [전제조건](#prerequisites) 을 충족했는지 확인하십시오. 그런 다음 다음 단계에 따라 Cordova 플러그인을 설치합니다.

1. 아직 계정이 없다면 [가입](https://newrelic.com/signup) 하세요(무료).

2. 해당하는 경우 iOS 및 Android에 대해 별도의 애플리케이션 토큰을 사용합니다.

3. New Relic 애플리케이션 페이지에서 구축 중인 Cordova 플랫폼에 대한 애플리케이션 토큰을 복사하십시오.

4. Cordova 프로젝트 디렉토리로 변경한 다음 Cordova 명령줄 도구를 사용하여 프로젝트에 플러그인을 추가하십시오. `--variable` 인수를 사용하여 플러그인에 애플리케이션 토큰을 전달합니다.

   ```
   # Install from github repository
   cordova plugin add https://github.com/newrelic/newrelic-cordova-plugin.git --variable IOS_APP_TOKEN="{ios-app-token}" --variable ANDROID_APP_TOKEN="{android-app-token}"
   ```

5. 앱을 배포합니다.

6. [모바일 모니터링 UI](/docs/mobile-monitoring/mobile-monitoring-ui) 에서 성능 데이터를 봅니다.

## 플러그인 업데이트 [#update]

다음 명령을 사용하여 New Relic Cordova 플러그인을 최신 릴리스 버전으로 쉽게 업데이트하십시오.

```
cordova plugin update
```
