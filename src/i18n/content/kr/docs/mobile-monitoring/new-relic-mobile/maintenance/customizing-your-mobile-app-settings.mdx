## 모니터링되는 앱의 이름 변경 [#change-name]

UI에서 앱 이름을 언제든지 변경할 수 있으며 모바일 모니터링은 모바일 앱을 구성할 때 사용된 기본 [애플리케이션 토큰](/docs/mobile-apps/viewing-your-application-token) 을 기반으로 계속해서 데이터를 수집, 집계 및 앱에 보고합니다. 즉, 애플리케이션에서 코드를 변경할 필요가 없습니다.

<Callout variant="caution">
  모바일 앱에서 보고된 데이터는 애플리케이션 토큰을 기반으로 명명된 앱에 연결됩니다. 단일 애플리케이션 토큰이 있는 모든 앱은 모바일 장치에 설치될 때 앱 이름에 관계없이 모바일 모니터링 UI에서 동일한 앱으로 보고됩니다.
</Callout>

모바일 앱의 이름을 바꾸려면:

1. [one.newrelic.com](https://one.newrelic.com/) 으로 이동하여 **모바일** 을 클릭하고 모니터링되는 앱을 선택합니다.
2. **응용 프로그램** 을 선택합니다. 새 이름을 입력하고 **설정 저장** 을 선택합니다.
