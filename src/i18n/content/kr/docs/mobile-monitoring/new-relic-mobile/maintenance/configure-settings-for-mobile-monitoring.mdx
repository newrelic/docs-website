## 애플리케이션 토큰 보기 [#application-token]

귀하의 모바일 앱의 데이터는 뉴렐릭에 귀하의 앱 데이터를 인증하는 고유한 40자 16진수 문자열입니다. 귀하가 모니터링하는 각 모바일 앱에는 미국과 EU 데이터 센터 모두에 대한 고유한 의미가 있습니다.

모바일 앱의 모든 데이터는 모바일 장치에 설치된 앱 이름에 관계없이 New Relic 모바일 모니터링 UI의 동일한 위치에 보고됩니다. Flutter 또는 React Native와 같은 하이브리드 프레임워크를 사용하고 iOS 및 Android 플랫폼 모두에 배포하는 경우 iOS용 토큰과 Android용 토큰을 하나씩 사용해야 합니다.

애플리케이션 토큰을 보려면:

1. <DoNotTranslate>**[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)**</DoNotTranslate>

   으로 이동하여

   <DoNotTranslate>**Mobile**</DoNotTranslate>

   클릭하고 모니터링되는 앱을 선택합니다.

2. <DoNotTranslate>**Application**</DoNotTranslate>

   또는

   <DoNotTranslate>**Installation**</DoNotTranslate>

   선택합니다.
