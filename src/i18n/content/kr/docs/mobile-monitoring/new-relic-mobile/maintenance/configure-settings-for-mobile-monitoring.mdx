New Relic 모바일 모니터링 UI의 **Settings** \[설정] 아래에는 New Relic이 모니터링하는 모바일 앱 변경, 알림 조건 편집 및 기타 설정 편집을 위한 옵션이 있습니다. 앱 데이터를 인증하려면 [애플리케이션 토큰을 확인하세요](/docs/mobile-monitoring/new-relic-mobile/maintenance/configure-settings-for-mobile-monitoring/#application-token).

</Callout>

## 애플리케이션 토큰 보기 [#application-token]

모바일 앱 토큰은 앱 데이터를 New Relic에 인증하는 고유한 40자의 16진수 문자열입니다. 모니터링하는 각 모바일 앱에는 미국과 EU 데이터 센터 모두에 대한 고유한 토큰이 있습니다.

모바일 앱의 모든 데이터는 모바일 장치에 설치된 앱 이름에 관계없이 New Relic 모바일 모니터링 UI의 동일한 위치에 보고됩니다. Flutter 또는 React Native와 같은 하이브리드 프레임워크를 사용하고 iOS 및 Android 플랫폼 모두에 배포하는 경우 iOS용 토큰과 Android용 토큰을 하나씩 사용해야 합니다.

애플리케이션 토큰을 보려면:

1. **[one.newrelic.com > All Capabilities](https://one.newrelic.com/all-capabilities)** 으로 이동하여 **Mobile** \[모바일을] 클릭하고 모니터링되는 앱을 선택합니다.
2. **응용 프로그램** 또는 **설치** 를 선택합니다.
