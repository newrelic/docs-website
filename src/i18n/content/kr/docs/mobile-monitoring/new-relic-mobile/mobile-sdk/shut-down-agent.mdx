      ## 통사론 [#syntax]

      ### 자바 [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```

      ## 설명 [#description]

      이 API는 런타임 동안 현재 애플리케이션 수명 주기 내에서 에이전트를 종료하는 기능을 제공합니다. 에이전트가 종료되면 현재 애플리케이션 수명 주기 내에서 다시 시작할 수 없습니다.

      ## 예시 [#example]

      ### 자바 [#java]

      ```java
      NewRelic.shutdown()
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.shutdown()
      ```
    </TabsPageItem>
