      ## 통사론 [#syntax]

      ### 자바 [#java]

      ```java
      NewRelic.removeAllAttributes ()
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.removeAllAttributes()
      ```

      ## 설명 [#description]

      이 메소드는 세션에서 모든 속성을 제거합니다. 단일 세션 속성을 제거하려면 [속성 제거를](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes) 참조하십시오.

      ## 반환 값 [#return-values]

      성공적으로 기록된 경우 `true` 을 반환하고 기록되지 않은 경우 `false` 을 반환합니다.

      ## 예시 [#example]

      ### 자바 [#java]

      ```java
      boolean attributesRemoved = NewRelic.removeAllAttributes();
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
          val attributesRemoved = NewRelic.removeAllAttributes()
      ```
    </TabsPageItem>
