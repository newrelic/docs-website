      ## 통사론 [#syntax]

      ### 자바 [#java]

      ```java
      NewRelic.crashNow(string $message)
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.crashNow(message:String!)
      ```

      ## 설명 [#description]

      New Relic 충돌 보고를 테스트하기 위해 java.lang.RuntimeException이라는 데모 런타임 예외를 발생시킵니다.

      이 메서드를 사용하면 메시지와 함께 `java.lang.RuntimeException` 라는 데모 런타임 예외를 발생시킬 수 있습니다. 이 충돌은 [충돌 분석 페이지](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes) 에 나타납니다.
