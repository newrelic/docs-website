      ## 통사론 [#syntax]

      ### 자바 [#java]

      ```java
      NewRelic.pauseReplay()
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.pauseReplay()
      ```

      ## 요구 사항

      * 안드로이드 에이전트 v7.7.0 이상.

      ## 설명

      `newrelic.pauseReplay()` 수동으로 재생을 강제로 녹음을 일시 중지하도록 호출할 수 있습니다.

      재생이 아직 진행 중이 아닌 경우 이 API 메서드는 아무런 효과가 없습니다.

      모바일 세션이 여전히 유효한 경우 [`newrelic.recordReplay()`](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/record-replay) API 메서드를 호출하여 일시 중지된 재생을 재개할 수 있습니다.

      ## 사용 사례

      * 특정 사이트 페이지나 의료 페이지, 신용 카드 정보 페이지 등과 같은 특정 조건에서 녹화를 중지하기 위해 수동으로 재생을 트리거하려고 합니다.

      ## 예시 [#example]

      다음은 재생 중에 녹화를 일시 중지하는 방법을 보여주는 예입니다.

      ### 자바 [#java]

      ```java
      public class MainActivity extends Activity {
        ...
        @Override
        public boolean onOptionsItemSelected(MenuItem item) {
          switch (item.getItemId()) {
            case R.id.pause_replay:
              NewRelic.pauseReplay();
            ...
              return true;
            default:
              return super.onOptionsItemSelected(item);
          }
        }
        ...
      }
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      class MainActivity : AppCompatActivity() {

        ...
        var client: OkHttpClient = OkHttpClient();

            binding.fab.setOnClickListener { view ->

                NewRelic.pauseReplay()
                }
            }
          ...
      ```
