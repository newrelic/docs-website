## 통사론 [#syntax]

### 자바 [#java]

```java
   NewRelic.withApplicationToken(apptoken).withLaunchActivityName().start(this.getApplication());
```

### 코틀린 [#kotlin]

```kotlin
NewRelic.withLaunchActivityName(className: String?)
```

콜드 타임과 핫 타임을 계산하기 위해 앱 실행 시간에 대한 대상 활동 이름을 설정합니다.

## 설명 [#description]

기본적으로 New Relic Android 에이전트는 기본 활동이 로드될 때 콜드 타임과 핫 타임을 계산합니다. 이 API를 사용하면 콜드 타임과 핫 타임을 계산할 때 대상 시작 활동 이름을 사용자 지정할 수 있습니다.
