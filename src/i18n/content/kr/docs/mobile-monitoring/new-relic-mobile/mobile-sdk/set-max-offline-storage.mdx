오프라인 저장을 위해 저장할 수 있는 전체 데이터의 최대 크기를 설정합니다.

기본적으로 모바일 모니터링은 최대 100MB의 오프라인 저장소를 수집할 수 있습니다. 장치가 인터넷에 연결되어 있지 않아 데이터 페이로드 전송에 실패한 경우 인터넷에 연결될 때까지 파일 시스템에 저장할 수 있습니다. 일반적인 수확 페이로드가 성공적으로 전송된 후 모든 오프라인 데이터는 뉴렐릭으로 전송되고 저장소에서 지워집니다.

이 방법은 파일 시스템에 저장할 수 있는 오프라인 저장소의 최대 크기를 제어합니다. 크기 제한에 도달하면 뉴렐릭 모바일 에이전트는 저장할 수 없는 새 페이로드를 삭제합니다. 이 방법을 사용하면 해당 오프라인 저장소의 최대 크기를 재정의할 수 있습니다.
