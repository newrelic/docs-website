      ## 통사론 [#syntax]

      ### 자바 [#java]

      ```java
      NewRelic.removeAttribute(string $name)
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.removeAttribute(name: String?)
      ```

      ## 설명 [#description]

      이 메소드는 이름 문자열로 지정된 속성을 제거합니다. 모든 사용자 정의 속성을 제거하려면 [모든 속성 제거를](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/remove-all-attributes) 참조하십시오.

        <tbody>
          <tr>
            <td>
              `$name`
            </td>

            <td>
              `string`
            </td>

            <td>
              필수의. 제거하려는 속성의 이름입니다.
            </td>
          </tr>
        </tbody>
      </table>

      ## 반환 값 [#return-values]

      성공적으로 기록된 경우 `true` 을 반환하고 기록되지 않은 경우 `false` 을 반환합니다.

      ## 예시 [#example]

      다음은 `rate` 라는 속성을 제거하는 예입니다.

      ### 자바 [#java]

      ```java
      boolean attributeRemoved = NewRelic.removeAttribute("rate");
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      val attributeRemoved = NewRelic.removeAttribute("rate")
      ```
