      ## 통사론 [#syntax]

      ### 자바 [#java]

      ```java
      NewRelic.noticeNetworkFailure(string $url, string $httpMethod, long $startTime, long $endTime, exception $exception OR enum $networkFailure)
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
      NewRelic.noticeNetworkFailure(
                  url: String?,
                  httpMethod: String?,
                  startTime: Long,
                  endTime: Long,
                  failure: NetworkFailure?,
              )
      ```

      ## 설명 [#description]

      New Relic Android SDK API는 [네트워크 요청을 추적하는](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/network-request-success/) 여러 방법을 제공합니다. 네트워크 요청이 실패하면 `noticeNetworkFailure` 로 실패에 대한 세부 정보를 기록할 수 있습니다. 대부분의 경우 이 호출을 catch 블록과 같은 예외 처리기 내부에 배치합니다.

      ## 예 [#examples]

      다음은 New Relic에 대한 감지된 네트워크 오류의 일부로 오류를 사용하는 오류 수신기의 예입니다.

      ### 자바 [#java]

      ```java
      new Response.ErrorListener() {
                                  @Override
                                  public void onErrorResponse(Error error) {
                                      NewRelic.noticeNetworkFailure(badUrl, "GET", System.nanoTime(), System.nanoTime(), NetworkFailure.exceptionToNetworkFailure(error));
                                  }
      ```

      ### 코틀린 [#kotlin]

      ```kotlin
          NewRelic.noticeNetworkFailure(badUrl, "GET", System.nanoTime(), System.nanoTime(), NetworkFailure.exceptionToNetworkFailure(error))
      ```
