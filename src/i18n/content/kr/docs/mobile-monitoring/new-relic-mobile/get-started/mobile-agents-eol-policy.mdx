## 모바일 런타임 [#mobile-runtimes]

Android 런타임 지원을 위해 New Relic은 [Google-Android EOL 정책을](https://endoflife.date/android) 따릅니다.

iOS, iPadOS, watchOS 및 tvOS 런타임 지원을 위해 New Relic은 [Apple EOL 정책을](https://support.apple.com/) 따릅니다.
