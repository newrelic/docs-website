iOS 모니터링 설정을 위해 [안내 설치를](https://onenr.io/0qwLv87gkj5) 권장합니다. 그러나 에이전트를 수동으로 설치해야 하는 경우 아래 단계에 따라 뉴렐릭 iOS 에이전트(XCFramework 패키지)를 설치하고 XCode 프로젝트에 추가하세요.

<Steps>
  <Step>
    [iOS 에이전트 릴리스 노트](/docs/release-notes/mobile-release-notes/ios-release-notes) 에서 최신 iOS 에이전트(XCFramework)를 다운로드하세요.
  </Step>

  <Step>
    iOS 에이전트가 다운로드된 위치로 이동하여 압축을 풉니다.
  </Step>

  <Step>
    Xcode 프로젝트에서 압축을 푼 XCFramework를 추가합니다.

    1. 프로젝트 탐색기를 마우스 오른쪽 버튼으로 클릭하고 **Add Files to `[YourProjectName]`** \[`[YourProjectName]`에 파일 추가를] 클릭합니다.
    2. `NewRelic.xcframework` 디렉토리를 선택하고 **Add** \[추가를] 클릭합니다.
  </Step>

  <Step>
    iOS 에이전트를 포함합니다.

    1. 프로젝트 내비게이터에서 프로젝트를 열고, 목표 설정을 엽니다.
    2. 에이전트를 추가하려는 목표, 목표를 클릭하세요.
    3. **General** \[일반] 탭을 클릭합니다.
    4. **Frameworks, Libraries, and Embedded Content** \[프레임워크, 라이브러리 및 포함된 콘텐츠] 섹션까지 아래로 스크롤합니다.
    5. 여기에 `NewRelic.xcframework` (가) 나열되어 있는지 확인하세요. 그렇지 않은 경우 **+** 버튼을 클릭하여 추가하세요.
  </Step>
</Steps>
