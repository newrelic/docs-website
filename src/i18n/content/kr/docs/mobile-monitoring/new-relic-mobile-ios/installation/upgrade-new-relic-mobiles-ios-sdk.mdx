iOS 에이전트를 설치, 구성 및 업그레이드하려면 계정 관리자여야 합니다. 최신 버전에 대한 정보는 [릴리스 정보](/docs/releases/ios) 를 참조하십시오.

## iOS 프레임워크 교체 [#upgrading]

관리자: 최신 버전의 iOS SDK로 업그레이드하기 전에 이전 버전의 iOS 에이전트 프레임워크를 교체해야 합니다.

  title="screen mobile monitoring upgrade ios.png"

/>

<figcaption>
  다음은 새 버전으로 교체할 수 있도록 기존 iOS 에이전트 프레임워크를 제거하는 워크플로의 예입니다.
</figcaption>

1. Xcode의 프로젝트 네비게이터( **CMD 1** )에서 **NewRelicAgent.framework** 를 검색합니다.
2. **NewRelicAgent.framework** 를 마우스 오른쪽 버튼으로 클릭하거나 Control-클릭 **하고 Finder에 표시를** 선택합니다.
3. **NewRelicAgent.framework** 를 휴지통으로 드래그합니다.
4. Xcode 프로젝트가 **NewRelicAgent.framework** 에 대한 참조를 빨간색으로 강조 표시하는지 확인합니다.
5. **NewRelicAgent.framework** 를 마우스 오른쪽 버튼으로 클릭하거나 Control-클릭하고 **삭제** 를 선택하여 프로젝트에서 사용되지 않는 참조를 제거합니다.
6. **[one.newrelic.com](https://one.newrelic.com) > Mobile > (앱 선택) > Settings > Installation** 에서 iOS 앱 모니터링을 위한 표준 설치 절차를 계속 진행합니다.
