최신 버전에 대한 정보는 [릴리스](/docs/releases/ios)정보를 참조하십시오.

## Swift Package Manager로 업데이트 [#SPM]

Swift Package Manager를 사용하여 New Relic iOS 에이전트를 업데이트하려면:

1. Xcode 프로젝트 탐색기의 패키지 의존성/종속성 섹션에서

   <DNT>
     **NewRelic**
   </DNT>

   에이전트로 이동합니다.

2. <DNT>
     **NewRelic**
   </DNT>

   에이전트를 마우스 오른쪽 버튼으로 클릭하세요.

3. <DNT>
     **Update Package**
   </DNT>

   선택합니다.

<Callout variant="important">
  항상 [릴리스 정보를](/docs/releases/ios) 확인하여 최신 에이전트 버전을 받고 있는지 확인하십시오. Swift Package Manager 규칙이 설정된 방식에 따라 새 버전을 포함하도록 `Up to Next Major` 또는 `Branch` 설정을 변경해야 할 수 있습니다.
</Callout>

## 코코아 포드로 업데이트 [#Cocoa-Pods]

Cocoa 포드를 사용하여 New Relic iOS 에이전트를 업데이트하려면:

1. <DNT>
     **Terminal**
   </DNT>

   앱을 열고 PodFile이 있는 디렉터리로 이동합니다.

2. 다음 명령을 실행합니다.

   ```shell
   pod update NewRelicAgent
   ```

## iOS 에이전트를 수동으로 교체 [#Manual]

New Relic iOS 에이전트를 다운로드하고 수동으로 설치하려면:

1. [https://download.newrelic.com/ios_agent/](https://download.newrelic.com/ios_agent/) 로 이동합니다. 업데이트하려는 iOS 에이전트를 다운로드합니다.

2. iOS 에이전트가 다운로드된 위치로 이동하여 압축을 풉니다.

3. Xcode(CMD+1)에서 프로젝트 네비게이터를 열고 `NewRelic.xcframework` 을 검색하세요.

4. `NewRelic.xcframework` 마우스 오른쪽 버튼으로 클릭하거나 Ctrl 키를 누른 채 클릭하고

   <DNT>
     **Show in Finder**
   </DNT>

   선택합니다.

5. 새로 다운로드한 iOS 에이전트를 이전 에이전트와 동일한 폴더로 드래그하고 이전 에이전트를 교체하도록 선택합니다.
