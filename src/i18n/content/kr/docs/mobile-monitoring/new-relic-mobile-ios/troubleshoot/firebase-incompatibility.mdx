## 문제

충돌, 비정상 종료 또는 오류는 New Relic iOS 에이전트와 Firebase와 같은 타사 원격 측정 데이터 추적 에이전트가 모두 앱 내에 포함된 경우 표시될 수 있습니다.

알려진 오류:

* 계측된 함수에서 충돌이 발생합니다.
* 충돌 로그를 캡처하지 못했습니다.

## 해결책

iOS 에이전트와 Firebase와 같은 타사 원격 측정 데이터 추적 에이전트를 동시에 사용하면 안 됩니다. 이 문제를 방지하려면 둘 중 하나만 활성화하는 것이 좋습니다.

다른 SDK를 제거하는 옵션이 아닌 경우 이러한 충돌을 방지하기 위해 사용할 수 있는 몇 가지 단계가 있습니다.

* 다른 타사 SDK를 시작하기 전에 에이전트를 시작하십시오.
* 에이전트의 일부 기능을 비활성화합니다. 이것이 작동하는 옵션인 경우 자세한 내용은 [API 가이드를](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings/) 참조하세요.

추가 도움이 필요하면 [support.newrelic.com](https://support.newrelic.com) 에서 지원을 받으십시오.
