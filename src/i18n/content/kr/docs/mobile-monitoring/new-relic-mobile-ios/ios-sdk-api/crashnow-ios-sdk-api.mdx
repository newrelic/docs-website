## 통사론

```
crashNow;
crashNow:(NSString* __nullable)message;
```

New Relic 충돌 보고를 테스트하기 위해 NewRelicDemoException이라는 데모 런타임 예외를 throw합니다.

## 요구 사항

에이전트 버전 4.83 이상이 필요합니다.

## 설명

이 메서드를 사용하면 메시지와 함께 `NewRelicDemoException` 이라는 데모 런타임 예외를 throw할 수 있습니다. 이 충돌은 [**충돌 분석** 페이지](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes) 에 나타납니다.

## 예

**방법:**

```
+ (void) crashNow:(NSString* __nullable)message;
```

### 오브젝티브-C

**예시:**

```
[NewRelic crashNow];
```

**메시지가 있는 예:**

```
[NewRelic crashNow:@"This is a test crash"];
```

### 빠른

**예시:**

```
NewRelic.crashNow()
```

**메시지가 있는 예:**

```
NewRelic.crashNow("This is a test crash")
```
