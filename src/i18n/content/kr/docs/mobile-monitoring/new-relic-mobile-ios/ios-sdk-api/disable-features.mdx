## 통사론

```
- (void) disableFeatures:(NRMAFeatureFlags)featureFlags;
```

기능을 비활성화하려면 이 비트별 플래그를 설정합니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

`disableFeatures` API를 사용하여 실행하지 않으려는 기능을 비활성화할 수 있습니다. 비활성화할 수 있는 플래그에 대한 자세한 내용은 [iOS 에이전트 구성 및 기능 플래그를](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)참조하세요.

<Callout variant="important">
  이 함수는 다음을 호출하기 직전에 호출해야 합니다.

  Objective-C: +\[NewRelic startWithApplicationToken:] 스위프트: NewRelic.start(withApplicationToken:)
</Callout>

## 예

### 오브젝티브-C [#obj-c]

**방법:**

```
+[NewRelic disableFeatures:(NRMAFeatureFlags)flags];
```

**예:**

오브젝티브-C 예시:

```
[NewRelic disableFeatures:NRFeatureFlag_NSURLSessionInstrumentation | NRFeatureFlag_CrashReporting]; 
```

신속한 예:

```
 NewRelic.disableFeatures([ NRMAFeatureFlags.NRFeatureFlag_CrashReporting, NRMAFeatureFlags.NRFeatureFlag_WebViewInstrumentation ])
```
