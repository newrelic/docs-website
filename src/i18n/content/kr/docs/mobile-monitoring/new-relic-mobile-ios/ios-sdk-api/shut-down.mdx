## 통사론

```
+ (void)shutdown;
```

런타임 동안 현재 애플리케이션 수명 주기 내에서 에이전트를 종료합니다.

## 요구 사항

에이전트 버전 7.4.4 이상.

## 설명

이 API는 런타임 동안 현재 애플리케이션 수명 주기 내에서 에이전트를 종료하는 기능을 제공합니다. 에이전트가 종료되면 현재 애플리케이션 수명 주기 내에서 다시 시작할 수 없습니다.

## 예

### 오브젝티브-C [#obj-c]

**방법:**

```
+ (void) shutdown();
```

**예시:**

```
[NewRelic shutdown];
```

### 빠른 [#swift]

**방법:**

```
NewRelic.shutdown() -> Void
```

**예시:**

```
NewRelic.shutdown()
```
