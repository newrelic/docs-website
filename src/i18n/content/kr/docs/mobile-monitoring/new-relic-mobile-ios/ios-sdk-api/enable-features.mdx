## 통사론

```
- (void) enableFeatures:(NRMAFeatureFlags)featureFlags;
```

기능을 활성화하려면 이 비트 단위 플래그를 설정하십시오.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

New Relic iOS 에이전트의 기능 대부분은 기본적으로 활성화되어 있습니다. 기본적으로 활성화되지 않은 기능을 활성화하려는 경우 `enableFeatures` API를 사용할 수 있습니다. 기본적으로 활성화되는 기능에 대한 자세한 내용은 [iOS 에이전트 구성 및 기능 플래그를](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)참조하세요.

<Callout variant="important">
  이 함수는 다음을 호출하기 직전에 호출해야 합니다.

  Objective-C: +\[NewRelic startWithApplicationToken:] 스위프트: NewRelic.start(withApplicationToken:)
</Callout>

## 예

### 오브젝티브-C [#obj-c]

**방법:**

```
+[NewRelic enableFeatures:(NRMAFeatureFlags)flags];
```

**예:**

오브젝티브-C 예시:

```
[NewRelic enableFeatures:NRFeatureFlag_SwiftInteractionTracing | NRFeatureFlag_GestureInstrumentation];
```

신속한 예:

```
 NewRelic.enableFeatures([ NRMAFeatureFlags.NRFeatureFlag_SwiftInteractionTracing, NRMAFeatureFlags.NRFeatureFlag_GestureInstrumentation ])
```
