## 통사론

```
currentSessionId; 
NewRelic.currentSessionId()
```

현재 세션의 ID를 반환합니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

현재 세션 ID를 반환합니다. 이 방법은 단일 세션 정의 및 식별자를 기반으로 앱 데이터(단순한 New Relic 데이터가 아님) 모니터링을 통합하는 데 유용합니다. 예를 들어 마케팅 분석 또는 사용자 분석에 동일한 식별자를 사용할 수 있습니다.

이 API를 사용하는 방법에 대한 컨텍스트는 사용자 정의 속성 및 이벤트 전송에 대한 문서를 참조하십시오.

* [오브젝티브-C](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#objc-custom-att-events)
* [빠른](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#swift-custom-att-events)

## 반환 값

현재 세션의 ID 문자열을 반환합니다.

## 예

### 오브젝티브-C [#obj-c]

**방법:**

```
+ (NSString*) currentSessionId;
```

**예시:**

```
NSString* sessionId = [NewRelic currentSessionId()];
```

### 빠른

**방법:**

```
NewRelic.currentSessionId() -> String
```

**예시:**

```
let sessionId = NewRelic.currentSessionId()
```
