## 통사론

```
removeAttribute:(NSString*)name;
NewRelic.removeAttribute()
```

지정된 세션 속성을 제거합니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

이 메서드는 이름 문자열로 지정된 속성을 세션에서 제거합니다.

이 API를 사용하는 방법에 대한 컨텍스트는 사용자 정의 속성 및 이벤트 전송에 대한 문서를 참조하십시오.

* [오브젝티브-C](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#objc-custom-att-events)
* [빠른](/docs/mobile-monitoring/new-relic-mobile-ios/install-configure/work-ios-sdk-api#swift-custom-att-events)

## 반환 값

성공하면 `true` , 실패하면 `false` 를 반환합니다.

## 예

### 오브젝티브-C [#obj-c]

**방법:**

```
+ (BOOL) removeAttribute:(NSString*)name;
```

**예시:**

`NSString*` 에 의해 지정된 세션 속성을 제거합니다. 예: `rate` .

```
BOOL attributeRemoved = [NewRelic removedAttribute@"rate"];
```

### 빠른

**방법:**

```
NewRelic.removeAttribute(name: String!) -> Bool
```

**예시:**

`rate` 이라는 세션 속성을 제거합니다.

```
let attributeRemoved = NewRelic.removedAttribute("rate")
```
