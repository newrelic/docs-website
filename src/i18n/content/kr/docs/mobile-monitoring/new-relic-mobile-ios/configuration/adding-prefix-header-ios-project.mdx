New Relic의 iOS SDK API를 설치하려면 프로젝트의 접두사 헤더를 수정해야 합니다. 프로젝트에 프로젝트가 없으면 다음 단계에 따라 추가하세요.

1. Xcode 메뉴에서 **File > New > File...** 을 선택합니다.

2. **iOS** 템플릿 옵션에서 **Other > PCH file** 을 선택합니다.

3. 파일 이름을 **&lt;target_name>-Prefix.pch** 로 지정한 다음 **만들기** 를 선택합니다.

4. 대상의 **빌드 설정** 에서 **모두** 를 선택한 후 `Prefix Header` 필드에 다음을 추가합니다.

   ```
   $(SRCROOT)/<target_name>-Prefix.pch
   ```
