tvOS 에이전트를 설치, 구성 및 업그레이드하려면 계정 관리자여야 합니다. 최신 버전에 대한 정보는 [릴리스 정보](/docs/releases/ios) 를 참조하십시오.

## tvOS 프레임워크 교체 [#v1]

먼저 tvOS 에이전트 프레임워크의 이전 버전을 교체해야 합니다.

1. Xcode의 프로젝트 네비게이터( **CMD 1** )에서 **NewRelicAgentTVOS.framework** 를 검색합니다.
2. **NewRelicAgentTVOS.framework** 를 마우스 오른쪽 버튼으로 클릭하거나 Control-클릭 **하고 Finder에 표시를** 선택합니다.
3. **NewRelicAgentTVOS.framework** 를 휴지통으로 드래그하십시오.
4. Xcode 프로젝트가 **NewRelicAgentTVOS.framework** 에 대한 참조를 빨간색으로 강조 표시하는지 확인합니다.
5. **NewRelicAgentTVOS.framework** 를 마우스 오른쪽 버튼으로 클릭하거나 Control-클릭하고 **삭제** 를 선택하여 프로젝트에서 사용되지 않는 참조를 제거합니다.

그런 다음 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > Settings > Installation** 의 UI에서 설치 프로세스를 계속 진행합니다.
