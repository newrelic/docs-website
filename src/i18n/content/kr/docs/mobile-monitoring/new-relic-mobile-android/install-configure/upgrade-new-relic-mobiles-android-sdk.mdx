최신 버전의 Android 에이전트가 있는지 확인하려면 [Android 에이전트 릴리스 정보](/docs/release-notes/mobile-release-notes/android-release-notes) 를 참조하세요. Android 앱 업그레이드 지침은 현재 사용 중인 Android 에이전트 SDK 버전에 따라 다릅니다.

## Android SDK 버전 2 또는 3에서 업그레이드 [#android_upgrade_2]

이전에 Android SDK 버전 2 또는 3을 설치한 경우: **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > Settings > Upgrade**로 이동합니다.

## Android SDK 버전 1 업그레이드 [#android_upgrade_1]

이전에 Android SDK 버전 1을 설치한 경우 [one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) >

        **AndroidManifest.xml 파일을 열고 `NewRelicApplication` 클래스를 제거합니다.**

        **nrandroid-setup** 을 처음 실행했을 때 `NewRelicApplication` 클래스가 `AndroidManifest.xml` 파일에 추가되었습니다. 이것은 New Relic SDK에서 제거되었으며 더 이상 필요하지 않습니다.

        기존 `Application` 하위 클래스가 있고 이를 `NewRelicApplication` 에서 상속하도록 수정한 경우 대신 `Application` 에서 상속하도록 하위 클래스를 업데이트합니다.

        모바일 빌드 단계를 호출하도록 기존 Maven 구성을 수정한 경우 해당 수정 사항을 제거하고 수정된 빌드 지침을 사용하십시오.
