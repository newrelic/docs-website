최신 버전의 Android 에이전트가 있는지 확인하려면 [Android 에이전트 릴리스 정보](/docs/release-notes/mobile-release-notes/android-release-notes) 를 참조하세요. Android 앱 업그레이드 지침은 현재 사용 중인 Android 에이전트 SDK 버전에 따라 다릅니다.

## Android SDK 버전 2 또는 3에서 업그레이드 [#android_upgrade_2]

이전에 Android SDK 버전 2 또는 3을 설치한 경우: <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Settings > Upgrade.**</DNT>
