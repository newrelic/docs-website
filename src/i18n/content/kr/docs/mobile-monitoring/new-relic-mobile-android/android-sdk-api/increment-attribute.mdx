## 통사론

```
NewRelic.incrementAttribute(string $name[, float $value])
```

속성의 개수를 늘립니다. 이전 값을 덮어쓰고 호출될 때마다 입력합니다.

## 요구 사항

에이전트 버전 5.0.0 이상.

## 설명

이름 값만 전달하면 이 메서드는 지정된 속성의 개수를 1만큼 증가시킵니다. 속성이 존재하지 않으면 값 1로 속성을 생성합니다. 선택적 float 값을 전달하면 해당 속성에 대한 개수가 증가합니다. float 값으로 지정된 속성.

이 API를 사용하는 방법에 대한 컨텍스트는 [사용자 지정 속성 및 이벤트 보내기](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api#attributes-events-insights) 를 참조하세요.

## 반환 값

성공적으로 기록된 경우 `true` 을 반환하고 기록되지 않은 경우 `false` 을 반환합니다.

## 예

### 증분 속성

지정된 속성의 개수를 1씩 증가시킵니다. 속성이 존재하지 않으면 값이 1인 속성을 생성합니다.

```
boolean incremented = NewRelic.incrementAttribute("rate");
```

### 특정 금액만큼 속성 증가

부동 소수점 값만큼 지정된 속성의 개수를 증가시킵니다.

```
boolean incremented = NewRelic.incrementAttribute("rate", 9999.99, false);
```
