## 통사론

```
NewRelic.startInteraction(string $interactionName)
```

Android 앱 코드에서 메서드를 계측하기 위한 상호 작용을 만듭니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

## 설명

Android 앱 코드의 메서드에서 `startInteraction()` 을 사용하여 New Relic 모니터링 상호작용을 생성합니다.

1. Android용 New Relic SDK API 가져오기:

   ```
   import com.newrelic.agent.android.NewRelic;
   ```

2. 앱 코드의 메서드에 대한 상호 작용 추적을 시작하고 이름을 지정합니다.

   ```
   NewRelic.startInteraction(String <var>$interactionName</var>);
   ```

이미 존재하고 이미 추적 중인 상호작용의 이름을 지정하려면 `setInteractionName()` 을(를) 참조하십시오.

New Relic Android SDK API 사용에 대한 일반적인 정보는 [사용 가이드](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) 를 참조하세요.

## 반환 값

특정 지점에서 [상호 작용을 종료하는](/docs/mobile-monitoring/new-relic-mobile-android/android-sdk-api/end-interaction) 데 사용할 수 있는 상호 작용 ID 번호를 반환합니다.

## 예

### 버튼 누름 시 상호 작용 시작

이름이 `RefreshContacts` 인 상호작용 추적을 시작하는 예:

```
public class MainActivity extends Activity {
  ...
  @Override
  public boolean onOptionsItemSelected(MenuItem item) {
    switch (item.getItemId()) {
      case R.id.action_refresh:
        NewRelic.startInteraction("RefreshContacts");
   ……
        return true;
      default:
        return super.onOptionsItemSelected(item);
    }
  }
  ...
}
```
