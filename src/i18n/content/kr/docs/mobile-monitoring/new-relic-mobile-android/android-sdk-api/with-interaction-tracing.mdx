## 통사론

```
NewRelic.withInteractionTracing(boolean $tracingFlag)
```

상호 작용 추적을 전체적으로 비활성화/활성화합니다.

## 요구 사항

에이전트 버전 5.4.0 이상.

## 설명

`withInteractionTracing()` 을 사용하여 [**상호작용** 페이지](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/interactions-page) 에 표시되는 상호작용 추적을 전역적으로 활성화하거나 비활성화합니다. New Relic 에이전트가 호출을 시작하기 전에 이것을 호출해야 합니다.

<Callout variant="important">
  에이전트가 시작된 후에는 상호 작용 추적이 비활성화됩니다. 에이전트를 시작하기 전에 실행된 모든 활동 또는 방법에 대한 상호 작용은 여전히 [상호 작용 페이지](/docs/mobile-monitoring-ui/interactions-dashboard) 에 나타날 수 있습니다.
</Callout>

이 API 사용에 대한 일반적인 정보는 [Android SDK API 사용 가이드](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) 를 참조하세요.

## 예

### 상호작용 비활성화

Android 에이전트에서 수집한 모든 상호 작용 추적을 끄려면 다음 안내를 따르세요.

```
NewRelic.withInteractionTracing(false)
```
