## 통사론

```
NewRelic.withApplicationBuild(string $buildId)
```

상호 작용과 연결된 사용자 지정 빌드 식별자 문자열을 설정합니다.

## 요구 사항

에이전트 버전 5.1.0 이상.

## 설명

이 호출을 통해 [**충돌 보고서** 페이지](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/mobile-apps-crash-report-dashboard#details) 에서 애플리케이션 버전 옆에 표시되는 사용자 정의 빌드 식별자 문자열을 설정할 수 있습니다. `AndroidManifest.xml` 에 정의된 `versionCode` 문자열을 사용하는 대신 `withApplicationBuild()` 메서드를 호출하고 사용자 지정 빌드 ID를 문자열로 전달합니다.

<Callout variant="important">
  **지원 중단됨** : New Relic Android 에이전트 버전 5.3.0부터 `withApplicationBuild()` 이(가) `NewRelic.withBuildIdentifier()` 을(를) 대체했습니다.
</Callout>

이 API 사용에 대한 일반적인 정보는 [Android SDK API 사용 가이드](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api) 를 참조하세요.

## 예

### 사용자 정의 빌드 식별자 설정

사용자 정의 빌드 식별자를 설정하려면:

```
NewRelic.withApplicationBuild('156')
```
