## 통사론

```
NewRelic.removeAllAttributes ()
```

세션에서 모든 속성을 제거합니다.

## 요구 사항

에이전트 버전 5.0.0 이상.

## 설명

세션에서 모든 속성을 제거합니다.

이 API를 사용하는 방법에 대한 컨텍스트는 [사용자 지정 속성 및 이벤트 보내기](/docs/mobile-monitoring/new-relic-mobile-android/install-configure/work-android-sdk-api#attributes-events-insights) 를 참조하세요.

## 반환 값

성공하면 true를, 실패하면 false를 반환합니다.

## 예

### 모든 속성 제거

```
boolean attributesRemoved = NewRelic.removeAllAttributes();
```
