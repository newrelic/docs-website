Android용 [모바일 모니터링](/docs/mobile-monitoring/new-relic-mobile/getting-started/introduction-new-relic-mobile) 은 모바일 앱을 모니터링하여 앱 성능에 대한 포괄적인 보기를 제공합니다. Java 또는 Kotlin을 사용하여 작성된 Android 앱에서 작동합니다.

## 호환성 및 요구 사항 [#requirements]

Android 에이전트를 설치하기 전에 [호환성 및 요구 사항을](/docs/mobile-monitoring/new-relic-mobile-android/get-started/new-relic-android-compatibility-requirements/) 검토하십시오.

## 에이전트 설치 [#installation]

Android 에이전트를 설치하려면 UI에 직접 있는 [가이드 설치 를](https://onenr.io/02R5GVpaxQb) 따르십시오. 이 방법을 사용하면 Gradle 및 Android Studio로 Android 에이전트를 설치하고 구성할 수 있습니다.
