## 문제

충돌, 충돌 또는 오류는 New Relic Android 에이전트와 Google의 [Firebase Performance Monitoring SDK](https://firebase.google.com/docs/perf-mon) 가 모두 앱에 포함될 때 표시됩니다.

알려진 오류:

* `java.lang.StackOverflowError`
* `java.lang.RuntimeException: An error occurred while executing doInBackground()`

## 해결책

Android 에이전트와 FireBase PerfMon은 동시에 활성화할 수 없습니다. 이 문제를 방지하려면 둘 중 하나만 활성화하는 것이 좋습니다.

추가 도움이 필요하면 [support.newrelic.com](https://support.newrelic.com) 에서 지원을 받으십시오.
