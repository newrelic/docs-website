## 문제

뉴렐릭 Android 에이전트와 Sentry 모니터링 SDK를 모두 통합하면 앱 내에서 충돌, 충돌 또는 오류가 발생할 수 있습니다.

이로 인해 `java.lang.StackOverflowError` 와 같은 문제가 발생하는 것으로 알려져 있습니다.

## 해결책

Sentry 모니터링 SDK에서 Android 에이전트가 감소하는 것을 방지해야 합니다. 이를 달성하려면, 감수에서 Sentry 패키지를 제외하도록 뉴렐 에이전트를 구성하세요. 방법은 다음과 같습니다.

* `build.gradle` 파일에 다음 블록을 추가합니다.

  ```
  newrelic {
  // Don't instrument sample classes
  excludePackageInstrumentation("io.sentry.*")
  }
  ```

이 설정은 뉴렐릭 에이전트에게 `io.sentry` 패키지 내의 모든 클래스를 건너뛰도록 지시합니다.

문제가 발생하거나 추가 도움이 필요하면 [support.newrelic.com](https://support.newrelic.com) 에서 뉴렐릭 지원팀에 문의하세요.
