세션 리플레이에는 자체 권한 세트가 있으므로 리플레이를 보고 설정을 업데이트할 수 있는 사람을 세밀하게 조정할 수 있습니다. 표준 역할이 있는 사용자는 다시보기를 볼 수 있지만, 모든 제품 관리자만 세션 리플레이 설정을 수정할 수 있습니다. 사용자 권한을 더 세밀하게 제어하기 위해 커스텀 역할을 생성할 수 있습니다.

## 권한 개요 [#permission-overview]

사용자 권한은 **Administration &gt; Access management &gt; Roles** 에서 찾을 수 있는 사용자 관리 UI에서 관리됩니다.

애플리케이션 설정 페이지에서 세션 리플레이 설정을 보거나 수정하려면 사용자에게 세션 리플레이 및 애플리케이션 설정 권한이 모두 필요합니다.
