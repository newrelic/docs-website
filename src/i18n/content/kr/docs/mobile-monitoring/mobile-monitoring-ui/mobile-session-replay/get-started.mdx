세션 리플레이는 최종 사용자가 휴대폰이나 태블릿에서 본 내용을 해당 세션의 모든 텔레메트리 데이터와 동기화된 비디오 리플레이로 제공하여 문제 해결 시 추측을 배제합니다. 이는 오류가 사용자에게 미치는 실제 영향을 신속하게 평가하고, 성능 병목현상과 UX 마찰 지점을 식별하며, 버그 보고서나 로컬 재현에 의존하지 않고도 문제를 더 빠르게 해결할 수 있음을 의미합니다.

세션 리플레이를 최대한 활용하려면 다음 섹션을 살펴보세요:

* [세션 리플레이 소개](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/introduction): 세션 리플레이가 어떻게 작동하는지, 어떤 데이터를 저장하는지, 그리고 데이터 보존에 어떤 영향을 미치는지 알아보세요.
* [세션 리플레이 설정](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/set-up-session-replay): 요구 사항을 확인하고, 세션 리플레이를 활성화하고, 샘플링 속도를 구성합니다.
* [개인정보 보호 설정 구성](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/configure-privacy-settings): 마스킹 및 데이터 보호 옵션을 맞춤화합니다.
* [뉴렐릭에서 세션 리플레이 보기](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/view-session-replays): 오류와 연결된 리플레이를 포함하여 기록된 세션에 액세스하고 분석합니다.
* [세션 리플레이 관리 및 권한 수정](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/manage-session-replay): 리플레이를 보고 설정을 수정할 수 있는 사용자를 제어합니다.
* [고급 세션 리플레이](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/advanced-session-replay): AI 요약, 시퀀스 필터, 수동 기록 API 및 NerdGraph 설정을 사용하세요.
* [세션 리플레이 문제 해결](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/troubleshooting): 일반적인 문제에 대한 해결 방법을 찾아보세요.
* [추가 정보](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/additional-information): 성능 영향, 데이터 소비, 개인정보 보호 규정 준수 및 지원되는 프레임워크에 대해 알아보세요.
