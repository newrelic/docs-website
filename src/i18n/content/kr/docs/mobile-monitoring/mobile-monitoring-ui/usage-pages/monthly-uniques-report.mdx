모바일 모니터링에는 지난 1년 동안 매월 앱을 실행하는 장치 수를 추적하는 막대 차트가 있는 월간 보고서가 포함됩니다. 보고서를 보려면: **[one.newrelic.com](https://one.newrelic.com) > Mobile > (앱 선택) > Usage > Monthly uniques** 으로 이동합니다.

## 월별 고유 보고서 세부정보 [#details]

특정 월의 총 고유 기기 수를 보려면 차트에서 해당 월의 막대 위로 마우스를 가져갑니다. 이번 달의 장치 수는 월간 누계 값이며 전체 월의 사용량을 나타내지 않습니다.

New Relic의 표준 [사용자 인터페이스 기능](/docs/accounts-partnerships/education/getting-started-new-relic/new-relic-user-interface) 과 [페이지 기능](/docs/accounts-partnerships/education/getting-started-new-relic/standard-dashboard-features) 을 사용하여 자세한 정보를 드릴다운하십시오.

순 사용자는 구독 수준을 계산하는 데 사용되는 [월간 사용량](/docs/accounts/new-relic-account-usage/mobile-usage/mobile-subscription-usage-ui) 을 계산하는 데 사용됩니다.

## 기기 추적 [#tracking]

모바일 모니터링은 고유한 설치 추적을 위해 하드웨어 식별자를 사용하지 않습니다.

* iOS에서는 IdentifierForVendor 속성을 사용합니다. (버전 5.3.4 이하에서는 SecureUDID 라이브러리를 사용했습니다.)
* Android에서는 애플리케이션이 설치될 때 고유한 GUID를 생성합니다.
