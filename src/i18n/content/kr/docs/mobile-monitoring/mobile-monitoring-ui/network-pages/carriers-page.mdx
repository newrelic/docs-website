모바일 모니터링을 위한 이동통신사 페이지에는 선택한 기간 동안 사용자의 무선 이동통신사와 응답 시간 영향, 오류율 및 활성 세션을 보여주는 차트가 포함되어 있습니다. Wi-Fi는 이동 통신사로 포함됩니다. 여기에서 특정 운송업체에 대한 자세한 정보를 정렬하고 드릴다운할 수 있습니다.

## 이동통신사 페이지 보기 [#procedures]

<figcaption>
  **[one.newrelic.com](https://one.newrelic.com) > Mobile > (앱 선택) > Network > Carriers** : 이 페이지를 사용하여 응답 시간 영향, 네트워크 장애 및 활성 세션별로 사용자의 이동통신사에 대한 자세한 정보를 보거나 정렬하거나 드릴다운할 수 있습니다.
</figcaption>

사용자의 이동통신사를 보려면 다음 단계를 따르세요.

1. **[one.newrelic.com](https://one.newrelic.com) > Mobile > (앱 선택) > Network > Carriers** 로 이동합니다.
2. 표시되는 정보(응답 시간, 활성 장치 또는 네트워크 오류 포함)를 변경하려면 **정렬** 기준 메뉴에서 원하는 항목을 선택합니다.
3. 표시되는 정보의 양을 조정하려면 **&lt;1% 처리량 숨기기** 를 선택합니다.
4. 모바일 앱 버전이나 기간을 선택하려면 [**버전**](/docs/mobile-apps/versions) 메뉴와 New Relic 메뉴 모음 아래의 [시간 선택기](/docs/site/timepicker-setting-time-periods-to-view-data) 를 사용하세요.
5. 특정 이동통신사에 대한 세부정보를 보려면 해당 이름을 선택하세요.

표준 [사용자 인터페이스 기능](/docs/accounts-partnerships/education/getting-started-new-relic/new-relic-user-interface) 과 [페이지 기능](/docs/accounts-partnerships/education/getting-started-new-relic/standard-dashboard-features) 을 사용하여 자세한 정보를 드릴다운하십시오.
