<InlinePopover type="mobile"/>의 이동통신사 페이지에는 사용자의 무선 이동통신사와 선택한 기간 동안의 응답 시간 영향, 오류율 및 활성 세션을 보여주는 차트가 포함되어 있습니다. Wi-Fi가 통신사로 포함되어 있습니다. 여기에서 특정 운송업체에 대한 자세한 정보를 정렬하고 드릴다운할 수 있습니다.

## 이동통신사 페이지 보기 [#procedures]

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Carriers**</DNT>: 이 페이지를 사용하여 응답 시간 영향, 네트워크 오류 및 활성 세션을 기준으로 사용자의 이동통신사에 대한 자세한 정보를 보거나 정렬하거나 드릴다운할 수 있습니다.
</figcaption>

사용자의 이동통신사를 보려면 다음 단계를 따르세요.

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Carriers**
   </DNT>

   으)로 이동합니다.

2. 표시되는 정보(응답 시간, 활성 장치, 네트워크 오류 등)를 변경하려면

   <DNT>
     **Sort by**
   </DNT>

   메뉴에서 원하는 항목을 선택하세요.

3. 표시되는 정보의 양을 조정하려면

   <DNT>
     **Hide &lt; 1% throughput**
   </DNT>

   선택합니다.

4. 모바일 앱 버전이나 기간을 선택하려면 뉴렐릭 메뉴 표시줄 아래에 있는

   <DNT>
     [**Versions**](/docs/mobile-apps/versions)
   </DNT>

   메뉴와 [타임 피커를](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) 사용하세요.

5. 특정 이동통신사에 대한 세부정보를 보려면 해당 이름을 선택하세요.

자세한 정보를 드릴다운하려면 적용 가능한 [플랫폼 UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)사용하십시오.
