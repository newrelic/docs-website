지역 페이지는 다음을 포함하여 모바일 사용자의 경험을 세계관으로 보여줍니다.

* 색상으로 구분된 응답 시간
* 네트워크 요청(분당 호출)
* 데이터 전송 크기
* 활성 장치
* 네트워크 실패율

각 국가에 대한 자세한 정보를 드릴다운할 수도 있습니다.

<Callout variant="tip">
  **지리** 기능은 **지도** 기능과 다릅니다. [**지도** 페이지](/docs/mobile-monitoring/mobile-monitoring-ui/network-pages/map-page-mobile-apps) 는 모바일 앱과 관련 서비스 간의 관계에 대한 아키텍처 보기를 보여줍니다.
</Callout>

## 지역 페이지 보기 [#geo_procedures]

국가별로 모바일 응답 시간을 보거나 정렬하려면:

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > Network > Geography** 으로 이동합니다.
2. 표시되는 정보(응답 시간, 분당 요청 수, 총 전송 크기, 활성 장치 또는 네트워크 실패율 포함)를 변경하려면 **정렬** 기준 메뉴에서 원하는 항목을 선택합니다.
3. 표시되는 정보의 양을 조정하려면 **&lt;1% 처리량 숨기기** 를 선택합니다.
4. 위치에 대한 요약 정보를 보려면 지도에서 색상으로 표시된 영역 위에 마우스를 올리거나 목록에 있는 국가 이름 위에 마우스를 놓습니다.

자세한 정보를 드릴다운하려면 적용 가능한 [플랫폼 UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)사용하십시오.

## 드릴다운 세부 정보 보기 [#geo_details]

특정 위치에 대한 자세한 정보(평균 응답 시간, 분당 호출 수, 활성 장치, 유형별 네트워크 장애 포함)를 보려면 지리 페이지의 지도에서 위치를 선택하거나 목록에서 이름을 선택합니다.
