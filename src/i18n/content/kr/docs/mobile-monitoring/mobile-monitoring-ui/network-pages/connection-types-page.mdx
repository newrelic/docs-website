모바일 모니터링을 위한 **연결 유형** 페이지에는 사용자의 무선 네트워크 연결 기술과 응답 시간 영향, 오류율 및 선택한 기간 동안의 활성 트래픽을 보여주는 차트가 포함되어 있습니다. 데이터는 2G, 3G 등 기술 세대별로 그룹화됩니다. Wifi 및 Unknown도 연결 유형 분석에 포함됩니다. 여기에서 GPRS, HSDPA 및 LTE와 같은 특정 무선 기술로 드릴다운할 수 있습니다.

사용자의 모바일 무선 연결 유형을 보려면:

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > Network > Connection types** 으로 이동합니다.
2. 표시되는 정보(응답 시간, 활성 장치 또는 네트워크 오류 포함)를 변경하려면 **정렬** 기준 메뉴에서 원하는 항목을 선택합니다.
3. 모바일 앱 버전 또는 기간을 선택하려면 [**Versions**](/docs/mobile-apps/versions)\[버전] 메뉴와 New Relic 메뉴 표시줄 아래의 [time picker](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker)\[시간 선택기를] 사용하십시오.
4. 특정 연결 유형에 대한 세부 정보를 보려면 해당 이름을 선택합니다.

자세한 정보를 드릴다운하려면 적용 가능한 [플랫폼 UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)사용하십시오.
