<InlinePopover type="mobile"/>의 <DNT>**Connection Types**</DNT> 페이지에는 사용자의 무선 네트워크 연결 기술과 응답 시간 영향, 오류율 및 선택한 기간 동안의 활성 트래픽을 보여주는 차트가 포함되어 있습니다. 데이터는 기술 세대(2G, 3G 등)별로 그룹화됩니다. Wi-Fi 및 알 수 없음도 연결 유형 분류에 포함됩니다. 여기에서 GPRS, HSDPA 및 LTE와 같은 특정 무선 기술을 자세히 살펴볼 수 있습니다.

사용자의 모바일 무선 연결 유형을 보려면:

1. <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DNT> 으)로 이동합니다.

   <img
     title="connection types.png"
     alt="connection types.png"
     src="/images/mobile_screenshot-full_connection-types-summary.webp"
   />

<figcaption>
  <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Network > Connection types**</DNT>: 이 페이지를 사용하여 응답 시간 영향, 네트워크 오류 및 활성 트래픽을 기준으로 사용자의 모바일 네트워크 연결 유형에 대한 자세한 정보를 보거나 정렬하거나 드릴다운할 수 있습니다.
</figcaption>

2. 표시되는 정보(응답 시간, 활성 장치, 네트워크 오류 등)를 변경하려면

   <DNT>
     **Sort by**
   </DNT>

   메뉴에서 원하는 항목을 선택하세요.

3. 모바일 앱 버전이나 기간을 선택하려면 뉴렐릭 메뉴 표시줄 아래에 있는

   <DNT>
     [**Versions**](/docs/mobile-apps/versions)
   </DNT>

   메뉴와 [타임 피커를](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker) 사용하세요.

4. 특정 연결 유형에 대한 세부 정보를 보려면 해당 이름을 선택합니다.

자세한 정보를 드릴다운하려면 적용 가능한 [플랫폼 UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)사용하십시오.
