모바일 모니터링을 위한 OS 버전 페이지는 iOS 및 Android와 같은 모바일 애플리케이션을 호스팅하는 상위 운영 체제 버전에 대한 성능 세부 정보를 제공합니다. 차트는 다음을 기준으로 OS 버전을 비교합니다.

* HTTP 요청 시간
* 네트워크 장애
* 분당 요청
* 활성 장치

여기에서 주 또는 부 OS 버전(예: iOS 8, iOS 7.1.1, Android 4.2.2)별로 세부정보를 드릴다운할 수 있습니다.

</figcaption>

모바일 앱 사용자의 운영 체제 버전에 대한 성능 세부 정보를 보려면 다음 단계를 따르세요.

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > App > OS versions** 으로 이동합니다.
2. 모바일 앱 버전 또는 기간을 선택하려면 UI 메뉴 표시줄 아래에 있는 **버전** 메뉴와 [**시간 선택기**](/docs/site/timepicker-setting-time-periods-to-view-data) 를 사용하세요.
3. 선택 사항: **정렬** 기준 및 **1% 미만 처리량 숨기기** 옵션을 선택합니다.
4. 버전을 포함하도록 운영 체제 목록을 확장하거나 축소하려면 운영 체제 이름(예: iOS 7)을 선택합니다.

## 드릴다운 세부 정보 보기 [#details]

자세한 정보를 드릴다운하려면 적용 가능한 [플랫폼 UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)사용하십시오. 게다가:

* 주요 OS 버전(상호작용 시간, HTTP 요청 시간, 네트워크 오류, 활성 장치, 가장 느린 추적 또는 모든 하위 버전 포함)의 부 및 포인트 릴리스에 대한 세부 정보를 보려면 목록에서 주요 OS 버전을 선택합니다.
* 특정 OS 버전에 대한 세부 정보를 보려면 확장된 OS 목록에서 이름을 선택합니다.
* 느린 트랜잭션의 추적 세부 정보를 보려면(사용 가능한 경우) 해당 링크를 선택합니다. 자세한 내용은 [상호 작용 페이지를](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/interactions-page)참조하십시오.
* 기본 OS 버전 페이지로 돌아가려면 **닫기** ( **X** ) 버튼을 선택합니다.
