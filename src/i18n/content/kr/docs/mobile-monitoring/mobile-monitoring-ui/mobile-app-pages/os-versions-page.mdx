귀하의 앱이 최신 Android 또는 iOS 릴리스에서 어떻게 작동하는지 궁금하십니까? OS 버전 페이지를 사용하여 다양한 릴리스(주 및 부 릴리스 포함) 간의 앱 성능을 분석 및 비교하고 활성 장치 수, 네트워크 오류 및 HTTP 요청 시간과 같은 지표를 확인하세요.

## OS 버전 페이지 보기 [#view-os-versions-page]

1. [one.newrelic.com > All capabilities \[모든 기능\]](https://one.newrelic.com/all-capabilities) > Mobile \[모바일] 이동합니다.

2. 모바일 앱을 선택하세요.

3. 왼쪽 메뉴에서

   <DNT>
     **OS versions**
   </DNT>

   클릭합니다.

  alt="Use this page to view, sort, or drill down into detailed information about the top five types of operation system versions using your mobile app."

## 특정 지표를 자세히 살펴보세요. [#drill-down]

상호 작용 시간 증가 등 데이터 급증이 나타나면 타임 피커를 클릭하여 드래그하세요. 페이지가 새로 고쳐지고 해당 시간 세그먼트로 확장되므로 정확히 무슨 일이 일어났는지 확인할 수 있습니다.

<img
  title="Expand metric details"
  alt="Gif of clicking and dragging the time picker"
  src="/images/mobile_screenshot-full_Expand-metrics.gif"
/>

성능 지표를 보려면 특정 OS 버전을 클릭하세요.

지표를 클릭한 다음 차트 메뉴에서 <DNT>**Create alert condition**</DNT> 을 클릭하고 UI 의 단계를 따르세요.
