모바일 앱의 오류율 및 응답 시간에 대해 설정한 [경고 조건](/docs/mobile-apps/alert-settings) 이 장기간 임계값을 초과하면 모바일 모니터링 에이전트는 경고(노란색) 또는 치명적(빨간색) 문제 이벤트를 생성합니다. 중요(빨간색) 이벤트가 몇 분 동안 열려 있으면 모바일에서 경고 알림을 받도록 설정된 사용자에게 경고 알림을 보냅니다. 또한 모든 중요 이벤트가 닫힐 때 알림을 보냅니다.

알림 페이지 보기: **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > App > Alerts** 으로 이동합니다.
