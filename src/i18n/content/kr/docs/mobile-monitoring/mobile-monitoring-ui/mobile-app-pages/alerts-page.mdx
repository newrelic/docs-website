모바일 앱의 오류율 및 응답 시간에 대해 설정한 [공지 조건](/docs/mobile-apps/alert-settings) 이 장기간 동안 레버 값을 교차하면 <InlinePopover type="mobile"/>에이전트가 경고(노란색) 또는 위험(빨간색) 문제 이벤트를 생성합니다. 중요(빨간색) 이벤트가 몇 분 동안 열려 있는 경우 모바일에서는 공지 공지를 수신하도록 설정된 사용자에게 공지 공지를 보냅니다. 또한 모든 중요한 이벤트가 종료되면 공지를 보냅니다.

<img
  title="alert conditions page "
  alt="screenshot of alert conditions page for mobile monitoring"
  src="/images/mobile_screenshot-full_alert-conditions-page.webp"
/>

공지 조건 페이지를 보려면 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (select an app) > Alert conditions**</DNT> 으로 이동하세요.

## 공지사항 페이지 상세정보

각 경고 알림에는 경고 아이콘과 요약 정보가 포함됩니다. 적용 가능한 [UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) 사용하여 자세한 정보를 드릴다운합니다. 또한 다음 옵션 중 하나를 사용하십시오.
