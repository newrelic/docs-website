모바일 앱의 오류율 및 응답 시간에 대해 설정한 [경고 조건이](/docs/mobile-apps/alert-settings) 장기간 동안 임계값을 초과하면 <InlinePopover type="mobile"/>에이전트는 경고(노란색) 또는 심각(빨간색) 문제 이벤트를 생성합니다. 중요(빨간색) 이벤트가 몇 분 동안 진행된 경우 모바일에서는 경고 알림을 받도록 설정된 사용자에게 경고 알림을 보냅니다. 또한 모든 중요 이벤트가 종료되면 알림을 보냅니다.

<img
  title="alert conditions page "
  alt="screenshot of alert conditions page for mobile monitoring"
  src={mobileAlertConditionsPage}
/>

공지 조건 페이지를 보려면 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile > (앱 선택) > Alert conditions** 으로 이동하세요.

## 공지사항 페이지 상세정보

각 경고 알림에는 경고 아이콘과 요약 정보가 포함됩니다. 적용 가능한 [UI 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) 사용하여 자세한 정보를 드릴다운합니다. 또한 다음 옵션 중 하나를 사용하십시오.
