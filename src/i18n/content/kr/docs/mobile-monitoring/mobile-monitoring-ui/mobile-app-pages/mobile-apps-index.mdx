New Relic의 모바일 모니터링 UI에는 모니터링되는 앱 목록과 이에 대한 중요한 요약 정보를 보여주는 모바일 앱 인덱스가 포함되어 있습니다.

## 모바일 앱 목록 보기 [#menu]

모바일 앱 색인을 보려면 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > All entities > Mobile applications** 으로 이동하십시오. [Explorer를](/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-explorer-view-performance-across-apps-services-hosts/) 사용하여 모든 [entities](/docs/new-relic-one/use-new-relic-one/core-concepts/what-entity-new-relic/) \[항목], 즉 애플리케이션 및 호스트에서 모든 요소의 사용자 지정 그룹에 이르기까지 보고서 데이터를 식별할 수 있는 모든 항목에 액세스합니다.

또는 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Mobile** 로 이동합니다.

사용 가능한 모바일 앱의 색인에는 다음을 나타내는 컬러 상태가 포함됩니다.

* 녹색 = 정상
* 노란색 = 경고
* 빨간색 = 치명적
* 회색 = 데이터를 보고하지 않음
