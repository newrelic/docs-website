New Relic의 모바일 모니터링 UI에는 모니터링되는 앱 목록과 이에 대한 중요한 요약 정보를 보여주는 모바일 앱 인덱스가 포함되어 있습니다.

## 모바일 앱 목록 보기 [#menu]

모바일 앱 색인을 보려면 **[one.newrelic.com](https://one.newrelic.com) > Explorer > 모바일 애플리케이션** 으로 이동하십시오. [Explorer](/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-explorer-view-performance-across-apps-services-hosts/) 를 사용하여 모든 [엔터티](/docs/new-relic-one/use-new-relic-one/core-concepts/what-entity-new-relic/) , 즉 애플리케이션 및 호스트에서 모든 요소의 사용자 지정 그룹화에 이르기까지 데이터를 보고하는 식별 가능한 모든 항목에 액세스합니다.

또는 **[one.newrelic.com](https://one.newrelic.com) > Mobile** 로 이동합니다.

사용 가능한 모바일 앱의 색인에는 다음을 나타내는 컬러 상태가 포함됩니다.

* 녹색 = 정상
* 노란색 = 경고
* 빨간색 = 치명적
* 회색 = 데이터를 보고하지 않음
