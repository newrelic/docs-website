모바일 모니터링을 위한 장치 페이지는 iPad, iPhone, iPod Touch, Android 태블릿 등과 같이 모바일 응용 프로그램을 사용하는 상위 장치에 대한 성능 세부 정보를 제공합니다. 차트는 다음을 기준으로 장치를 비교합니다.

* 상호작용 시간
* HTTP 요청 시간
* 오류율 및 네트워크 장애
* 활성 사용자 세션

여기에서 특정 모델(예: iPhone 6, 6S, 7)별로 세부 정보를 드릴다운할 수 있습니다.

사용자의 휴대기기에 대한 성능 세부정보를 보려면 다음 단계를 따르세요.

1. **[one.newrelic.com](https://one.newrelic.com) > Mobile > (앱 선택) > App > Devices** 로 이동합니다.
2. 모바일 앱 버전을 선택하거나 기간을 변경하려면 **버전** 메뉴와 New Relic 메뉴 표시줄 아래의 [**시간 선택기**](/docs/site/timepicker-setting-time-periods-to-view-data) 를 사용하세요.
3. 선택 사항: **정렬** 기준 및 **1% 미만 처리량 숨기기** 옵션을 선택합니다.
4. 특정 장치 **유형** (예: iPad)으로 세부 정보를 제한하려면 해당 이름을 선택합니다.

## 드릴다운 세부 정보 보기 [#details]

자세한 정보로 드릴다운하려면 New Relic의 표준 [사용자 인터페이스 기능](/docs/accounts-partnerships/education/getting-started-new-relic/new-relic-user-interface) 과 [페이지 기능](/docs/accounts-partnerships/education/getting-started-new-relic/standard-dashboard-features) 을 사용하여 자세한 정보로 드릴다운하십시오. 게다가:

* 특정 장치 또는 모델(예: iPad mini, iPad Air 등) 목록을 보려면 **유형** (예: iPad)을 선택합니다.
* 특정 장치 또는 모델에 대한 세부 정보를 보려면 확장된 목록에서 해당 이름을 선택합니다.
* 느린 트랜잭션(사용 가능한 경우) 추적 세부 정보를 보려면 해당 링크를 선택합니다. 자세한 내용은 [상호 작용 페이지](/docs/mobile-monitoring-ui#interactions-timeline) 를 참조하십시오.
* 기본 장치 페이지로 돌아가려면 **닫기** ( **X** ) 버튼을 선택합니다.
