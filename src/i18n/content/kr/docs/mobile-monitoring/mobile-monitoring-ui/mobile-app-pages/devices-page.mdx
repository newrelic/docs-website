<InlinePopover type="mobile"/>의 장치 페이지는 iPad, iPhone, iPod Touch, Android 태블릿 등과 같이 모바일 성능을 사용하는 상위 장치에 대한 성능 세부 정보를 제공하고 다음과 같은 지표를 보여줍니다.

* 상호작용 시간
* HTTP 요청 시간
* 오류율 및 네트워크 장애
* 활성 사용자 세션

## 장치 페이지 보기 [#view-devices-page]

1. [one.newrelic.com > All capabilities \[모든 기능\]](https://one.newrelic.com/all-capabilities) > Mobile \[모바일] 이동합니다.

2. 모바일 앱을 선택하세요.

3. 왼쪽 메뉴에서

   <DNT>
     **Devices**
   </DNT>

   클릭합니다.

## 특정 지표를 자세히 살펴보세요. [#drill-down]

자세한 정보로 드릴다운하려면 New Relic의 표준 [사용자 인터페이스 기능을](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform)사용하십시오. 게다가:

* 특정 기기 또는 모델(예: iPad mini, iPad Air 등) 목록을 보려면

  <DNT>
    **type**
  </DNT>

  (예: iPad)을 선택합니다.

* 특정 장치 또는 모델에 대한 세부 정보를 보려면 확장된 목록에서 해당 이름을 선택합니다.

* 느린 트랜잭션의 추적 세부 정보를 보려면(사용 가능한 경우) 해당 링크를 선택합니다. 자세한 내용은 [상호 작용 페이지를](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-pages/interactions-page)참조하십시오.
