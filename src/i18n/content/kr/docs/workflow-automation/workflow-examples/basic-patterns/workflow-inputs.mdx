엔드포인트 URL 및 기타 입력 필드에서 런타임에 동적 값을 전달하려면 `${{ .workflowInputs.variableName }}` 사용하십시오.

**이 접근 방식의 효과는 다음과 같습니다.**

* 터널우가 트리거될 때 동적 입력 값을 허용합니다.
* 전체 단계에서 템플릿 구문을 사용하여 참조 입력값을 사용합니다.
* 다양한 활동으로 워크플로우 재사용성을 활성화합니다.

## 예시

```json
  {
    "inputs": [
      {
        "key": "urlParams",
        "value": "{\"filter\": \"active\"}"
      },
      {
        "key": "headers",
        "value": "{\"Api-Key\": \"your-api-key\"}"
      }
    ]
  }
```
