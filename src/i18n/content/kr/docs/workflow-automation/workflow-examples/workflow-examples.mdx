자동화의 일반적인 시나리오를 기본부터 고급 패턴까지 단계별로 살펴보겠습니다. 즉시 사용 가능한 구현하다, 배포하다 솔루션에 대한 시작점으로 사용하거나 [템플릿 라이브러리를](/docs/workflow-automation/create-a-workflow-automation/use-a-template) 탐색하세요.

## 사용 가능한 템플릿 흐름

뉴렐릭플로우 자동화 UI 에서 직접 템플릿에 액세스하세요.

1. **All Capabilities &gt; Workflow Automation**로 이동
2. **Create workflow** \[워크플로 만들기]를 클릭하세요.
3. **Use a template** \[템플릿 사용을]선택하세요
4. 템플릿을 찾아보고 사용 사례에 맞는 활동을 선택하세요.

템플릿에는 다음이 포함됩니다.

* 사전 구성된 워크플로우 단계 및 로직
* 예제 입력 매개변수,
* 필수 자격 증명 및 통합
* 예상 결과 문서화

템플릿을 그대로 사용하거나 특정 요구 사항에 맞게 사용자 지정할 수 있습니다.
