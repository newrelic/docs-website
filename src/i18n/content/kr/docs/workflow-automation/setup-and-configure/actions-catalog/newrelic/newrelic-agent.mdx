이 페이지는 워크플로우 자동화 액션 카탈로그에서 사용할 수 있는 뉴렐릭 에이전트 액션에 대한 포괄적인 참조를 제공합니다. 이러한 작업을 통해 Agentic 플랫폼에 구성된 에이전트를 실행할 수 있습니다.
