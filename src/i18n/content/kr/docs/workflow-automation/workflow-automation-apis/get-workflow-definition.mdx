이 API 폴리스우 정의를 가져와 반환합니다.

## 예시

쿼리플로우를 생성하고 편집하려면 [쿼리빌더를](https://one.newrelic.com/nerdgraph-graphiql?state=c8579e50-f154-fec6-22b8-f9fdcfc1990a) 사용하세요. [NerdGraph API 탐색기를](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) 참조하세요.

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflow(name: "my-workflow", version: 1) {
          definition {
            name
            version
            yaml
          }
        }
      }
    }
  }
}
```
