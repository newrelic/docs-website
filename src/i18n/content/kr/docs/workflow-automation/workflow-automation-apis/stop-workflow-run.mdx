실행 중인 흐름을 중지합니다. 폴리스우를 찾을 수 없으면 빈 `runId` 을 반환합니다.

## 예시

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```
