## 예시

```graphql
mutation {
  workflowAutomationStopWorkflowRun(
    accountId: 11933347
    runId: "7ffbe082-8efa-4d3d-901a-a08bcf22bcd3"
  ) {
    runId
  }
}
```

## 응답

돌연변이가 다시 나타납니다:

* `runId`: 중지된 플로우 흐름의 고유한 구분입니다. 해당 항목을 찾을 수 없는 경우 빈 `runId` 을 반환합니다.
