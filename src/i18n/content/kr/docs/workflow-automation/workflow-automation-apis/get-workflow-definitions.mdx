이 API 지정된 범위 내의 모든 개요 정의 개요 목록을 가져와 반환합니다.

<Callout variant="important">
  이 쿼리에는 워크플로우 정의 YAML을 사용할 수 없습니다. A 워크플로우 정의 YAML은 `GetWorkflowDefinition` 쿼리 `/{actor/account(:id)/workflowAutomation/workflow(:name, :version)}` 를 통해서만 반환됩니다.
</Callout>

## 샘플 요청

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflows(cursor: "xyz123") {
          results {
            definition {
              name
              scope {
                id
                type
              }
              version
            }
          }
        }
      }
    }
  }
}
```

## 샘플 응답

쿼리 결과는 다음과 같습니다.

* `workflows.results`: 워크플로우 정의 개요에 대한 내용으로, 각각 다음을 포함합니다.

  * `definition.name`: 워크플로우 정의의 이름
  * `definition.scope.id`: 계정 ID
  * `definition.scope.type`: 스코프 유형
  * `definition.version`: 참조우 정의의 버전 번호
