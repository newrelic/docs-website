기존 스텔라우 정의를 업데이트하는 데 사용됩니다. 스텔라우 정의가 아직 존재하지 않으면 오류가 반환됩니다.

## 예시

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```
