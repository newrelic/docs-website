## 예시

```graphql
  mutation {
    workflowAutomationUpdateWorkflowDefinition(
      definition: {yaml: "<workflow definition YAML>"}
      scope: {id: "11608789", type: ACCOUNT}
    ) {
      definition {
        name
        version
        yaml
      }
    }
  }
```

## 응답

돌연변이가 다시 나타납니다:

* `definition.name`: 업데이트된 정의의 이름
* `definition.version`: 참조우 정의의 새 버전 번호
* `definition.yaml`: base64로 인코딩된 워크플로우 정의 YAML
