이 API 지정된 이름을 가진 모든 펠릭스우 정의 버전을 삭제합니다.

<Callout variant="important">
  * 정의를 삭제하려면 해당 정의를 사용하는 일정이 없어야 합니다. 정의가 현재 일정에 의해 사용 중인 경우 정의를 삭제하면 실패하고 그에 따라 오류가 반환됩니다.
  * 정의가 삭제되더라도 실행 중인 모든 펠로스우 실행은 완료될 때까지 계속됩니다.
</Callout>

## 샘플 요청

```graphql
mutation {
  workflowAutomationDeleteWorkflowDefinition(
    definition: {name: "my-workflow"}
    scope: {id: "12345678", type: ACCOUNT}
  ) {
    definition {
      name
    }
  }
}
```
