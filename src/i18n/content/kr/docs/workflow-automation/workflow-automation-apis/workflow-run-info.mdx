## 샘플 요청

쿼리플로우를 생성하고 편집하려면 [쿼리빌더를](https://one.newrelic.com/nerdgraph-graphiql?state=03119d23-7b51-7b7a-1c99-d2cd026300e5) 사용하세요. [NerdGraph API 탐색기를](/docs/apis/nerdgraph/get-started/nerdgraph-explorer) 참조하세요.

```graphql
{
  actor {
    account(id: 11933347) {
      workflowAutomation {
        workflowRunInfo(runId: "8911c7e4-5a1f-4316-90c8-67adde72dd1c") {
          logUrl
          status
        }
      }
    }
  }
}
```

## 샘플 응답

쿼리 결과는 다음과 같습니다.

* `logUrl`: 워크플로우 실행 로그를 볼 수 있는 URL
* `status`: 현재의 워크플로우 실행 상태
