워크플로우 자동화는 뉴럴릭을 기존 도구와 연결하여 코드를 작성하지 않고도 인시던트 대응, 집계 관리 및 운영 작업을 자동화합니다. 알림에 응답하고, AWS 및 Slack과 통합하고, 승인을 관리하고, 복잡한 자동화 로직을 실행하는 도구를 구축하세요.

**세 가지 방법으로 흐름을 유발합니다:**

* **알림에서**: 공지 조건이 위반되면 자동으로 플로우를 시작합니다. ([공지에서 공지 보내기](/docs/workflow-automation/setup-and-configuration/create-destinations#trigger-from-alerts) 참조)

* **온디맨드 방식**: 필요할 때 API를 통해 워크플로우를 수동으로 실행합니다.

* **예약 실행**: cron 표현식을 사용하여 지정된 시간에 자동으로 실행하세요.

  <img title="Create and manage workflows in New Relic" alt="Screenshot showing the workflow automation interface" src="/images/workflow-automation-overview-1.webp" />

## 주요 이점

* **더 빠른 인시던트 대응**: 알림 실행 시 EC2의 크기를 자동으로 조정하고, 구현을 롤백하거나, 배포하거나, 서비스를 다시 시작합니다. 수동 개입이 필요하지 않습니다.
* **도구 연결**: AWS, Slack, Jira, ServiceNow 및 모든 HTTP API 와 통합하여 팀이 작업하는 곳으로 풍부한 공지 데이터를 전송하세요.
* **승인 게이트 추가**: 중요한 부분에 인적 감독을 포함합니다. 프로덕션을 확장하거나 롤백을 실행하기 전에 팀 확인을 요청합니다.
* **복잡성 처리**: 조건 논리, 루프, 데이터 변환을 사용하여 고유한 운영 프로세스를 자동화합니다.
