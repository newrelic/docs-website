이 페이지는 자동화 작업 카탈로그에서 사용 가능한 HTTP POST에 대한 포괄적인 참조 정보를 제공합니다. 이러한 작업을 통해 HTTP POST 요청 작업을 수행할 수 있습니다.

## 전제 조건

블리자드 자동화에서 HTTP 작업을 사용하기 전에 다음 사항이 있는지 확인하세요.

* 부채, 목표 API 포인트 포인트 URL.
* 필요한 인증 자격 증명(API 키, 토큰 등)
* API 요청/응답 형식에 대한 이해.

<Callout variant="tip">
  HTTP 액션은 모든 헤더 값에 대해 비밀 구문을 지원하므로 API 키와 같은 민감한 데이터를 안전하게 전달할 수 있습니다. 자세한 내용은 [비밀 관리 부분을](/docs/workflow-automation/limitations-and-faq/workflow-best-practices/#secure-credentials) 참조하세요.
</Callout>

## 서버로 데이터를 전송합니다 [#http-post]

API 엔드포인트에 데이터를 전송하기 위해 HTTP POST 호출을 수행합니다.

<Callout variant="important">
  API 키 헤더와 같은 민감한 데이터를 입력에 전달해야 하는 경우, [secretsManagementCreateSecret](https://onenr.io/0KQXX6BmdQa) NerdGraph 뮤테이션을 통해 저장된 값을 사용할 수 있습니다.

  **예시**

  ```yaml
  {
    "inputs": [
      {
        "key": "headers",
        "value": "{\"Api-Key\": \"${{ :secrets:NR_API_KEY }}\"}"
      }
    ]
  }
  ```
</Callout>
