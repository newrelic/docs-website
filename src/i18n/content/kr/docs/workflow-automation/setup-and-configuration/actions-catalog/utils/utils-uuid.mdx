이 페이지는 유틸리티 - uuid가 유틸리티 자동화 작업 카탈로그에서 사용 가능한 방법에 대한 포괄적인 참조 정보를 제공합니다. 이러한 작업을 통해 UUID 생성 유틸리티를 사용할 수 있습니다.

## UUID 생성 [#utils-uuid-generate]

RFC 규격을 준수하는 V4 UUID를 생성합니다. Temporal 플로우의 결정론적 PRNG를 사용하여 플로우 플로우 내에서 안전하게 사용할 수 있습니다. [라벨스페이스: 마커플로우 | API ](https://typescript.temporal.io/api/namespaces/workflow?&_ga=2.223881945.39829224.1686648936-167080156.1680123165#uuid4)자세한 내용은 Temporal TypeScript SDK 참조를 참조하세요.
