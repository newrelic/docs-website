<Callout title="시사">
  이 기능은 아직 개발 중이지만 꼭 사용해 보시기 바랍니다!

  이 기능은 현재 [출시 전 정책](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy) 에 따라 미리보기 프로그램의 일부로 제공됩니다.
</Callout>

이 페이지는 페이지우 자동화 액션 카탈로그에서 사용할 수 있는 PagerDuty 인시던트 액션에 대한 포괄적인 참조 정보를 제공합니다. 이러한 조치를 통해 PagerDuty 인시던트 관리를 할 수 있습니다.

## 전제 조건

블리자드 자동화에서 PagerDuty 작업을 사용하기 전에 다음 사항이 있는지 확인하세요.

* 적절한 권한이 있는 PagerDuty 계정.
* PagerDuty API 토큰(사용자 토큰 또는 계정 토큰).
* PagerDuty 계정에서 인시던트를 관리하는 데 필요한 권한입니다.
* 계정 토큰의 경우 관련 작업에서 `from` 이메일 주소를 제공해야 합니다.
