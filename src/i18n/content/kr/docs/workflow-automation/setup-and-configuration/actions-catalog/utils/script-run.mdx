## 펼쳐보기.run이 할 수 있는 일

### 지원되는 파이썬 버전

* 모든 언어 기능을 갖춘 PYTHON\_3\_13 런타임

### 허용된 수입품

```yaml
  "python-dateutil",
  "simplejson",
  "re",
  "math",
  "decimal",
  "json",
  "datetime",
  "collections",
  "itertools",
  "functools",
  "operator",
  "string",
  "argparse"
```

### 데이터 처리

* JSON 데이터 구조를 파싱하고 변환합니다.
* 복잡한 문자열을 처리하고 텍스트 조작을 수행합니다.
* 출력 형식을 표, 마크다운 또는 구조화된 데이터로 지정하세요.

### 반응, Ppssing

* argparse를 사용하여 명령줄 인수로 간단한 값을 전달하세요.

## 펼쳐보기.run이 할 수 없는 일

### 수입 제한

* `base64` 보안상의 이유로 허용되지 않습니다.
* `sys` 보안상의 이유로 허용되지 않습니다.
* `os` - 시스템 작동이 제한됨
* 대부분의 타사 클래스는 끌어서 3.13 stdlib에 포함되어 있지 않습니다.

### 반응, 제한사항

* 복잡한 JSON을 분석할 때 전달할 수 없습니다. (안전하지 않은 펼쳐보기 오류 발생)
* 시위를 통해 특수 문자가 포함된 문자열을 전달할 수 없습니다.

### 네트워크/외부 액세스

* 네트워크 호출이나 외부 API 접근은 없습니다.
* 스크립트 실행 외에는 파일 시스템에 접근할 수 없습니다.
