이 페이지는 유틸리티 - 변환 기능을 포함한 자동화 작업 카탈로그에서 사용할 수 있는 다양한 기능에 대한 포괄적인 참조 정보를 제공합니다. 이러한 작업을 통해 데이터 변환 유틸리티를 사용할 수 있습니다.

## 입력 데이터 값을 CSV 형식으로 변환 [#utils-transform-toCSV]

이 함수는 다양한 유형의 입력(JSON, 맵)을 CSV 형식으로 변환하는 데 사용됩니다.

              ```yaml

                steps:
                  - name: queryForLog
                    type: action
                    action: newrelic.nrql.query
                    version: 1
                    inputs:
                      accountIds: ['${{ .workflowInputs.accountId }}']
                      query: ${{ .workflowInputs.nrql }}

                  - name: csv1
                    type: action
                    action: utils.transform.toCSV
                    version: 1
                    inputs:
                      data: ${{ .steps.queryForLog.outputs.results | tostring }}
                      selectors:
                        - name: csv
                          expression: ".csv"
              ```
