## 변경 로그

설정 또는 `nrdot-collector`의 관련 변경 사항으로 범위가 지정된 [&apos;호스트 모니터링&apos;](https://github.com/newrelic/nrdot-collector-releases/blob/main/distributions/nrdot-collector/README.md#use-cases) 사용 사례에 대한 첫 번째 전용 릴리스 노트입니다. [nrdot-collector](https://docs.newrelic.com/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01/) 는 이 사용 사례를 지원하는 데 필요한 구성을 계속해서 번들로 제공합니다. 하지만 `nrdot-collector` 이(가) 점점 더 많은 사용 사례를 지원함에 따라, 호스트 모니터링과 같은 특정 목적으로 NRDOT를 사용하는 사용자가 더 쉽게 활용할 수 있는 추가 릴리스 노트를 제공하고자 합니다.

### 특징

* 함께 작동하는 것으로 확인된 호스트 사용 사례 `nrdot-collector:1.12.0`
