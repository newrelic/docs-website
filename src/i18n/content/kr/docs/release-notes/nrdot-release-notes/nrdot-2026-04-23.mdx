## 변경 로그

이번 릴리스에서는 `nrdot-collector-k8s` 및 `nrdot-collector-host` 배포판이 제거되었습니다. 해당 배포판의 마지막으로 게시된 버전은 `1.12.0`입니다. 대신 기능 측면에서 슈퍼셋이며 제거된 두 배포판이 [사용 사례](https://github.com/newrelic/nrdot-collector-releases/blob/1.13.0/distributions/nrdot-collector/README.md#use-cases)로 구현되어 드롭인 대체품 역할을 하는 `nrdot-collector` 를 사용하도록 마이그레이션하십시오.
