### 개량

* ping 런타임에 대한 logback-core 취약점을 수정합니다.
* ping 런타임에 대한 jetty-server 취약점을 수정합니다.
* ping 런타임에 대한 Ubuntu 취약점을 수정합니다.

### 제안

* 최신 Ping Runtime 버전(현재 1.51.0)을 사용 중이시라면 환경의 보안과 안정성을 보장하기 위해 신세틱스 Job Manager를 릴리스-443 이상으로 업그레이드하는 것이 좋습니다.
