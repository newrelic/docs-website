### 개량

* ping 런타임에 대한 logback-core 취약점을 수정합니다.
* ping 런타임에 대한 jetty-server 취약점을 수정합니다.
* ping 런타임에 대한 Ubuntu 취약점을 수정합니다.

### 제안

* 현재 최신 Ping Runtime 버전 1.51.0을 사용하고 있다면, 사용자 환경의 보안과 안정성을 확보하기 위해 신세틱스 Kevin을 릴리스-443 이상 또는 최신 버전으로 업그레이드하는 것이 좋습니다.
