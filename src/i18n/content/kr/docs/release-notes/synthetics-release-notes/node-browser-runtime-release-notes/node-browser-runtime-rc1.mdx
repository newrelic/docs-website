### 개량

* Chrome 브라우저를 버전 143.0.7499.146으로 업데이트했습니다. 사용자는 브라우저 로그에서 &quot;소프트웨어 WebGL로의 자동 대체 기능이 더 이상 사용되지 않습니다.&quot;라는 경고 메시지를 확인할 수 있습니다. 이번 변경 사항은 심각도가 높은 보안 취약점([CVE-2025-14765](https://chromereleases.googleblog.com/2025/12/stable-channel-update-for-desktop_16.html))을 해결하기 위해 자동 대체 기능을 제거합니다.
