* SJM의 보안 문제를 해결하기 위해 `common-lang3` ยู่ 및 Ubuntu와 관련된 버그를 수정했습니다.

* SJM 로그에서 문제 해결, 해결에 대한 더 나은 컨텍스트를 제공하기 위해 전체 예외 세부 정보를 포함하도록 문제를 수정했습니다.
