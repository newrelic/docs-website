### 알아채다

* YUM 리포지터리에서 .NET 에이전트 RPM 패키지에 서명하는 데 사용되는 GPG 키가 교체되었습니다. [업데이트된 저장소 정의 파일을](https://download.newrelic.com/dot_net_agent/yum/newrelic-dotnet-agent.repo) 다운로드하여 `/etc/yum.repos.d/` 에 배치하고 기존 파일을 덮어쓰는 방식으로 시스템을 업데이트하여 새 키를 신뢰하도록 하십시오. ([ef5f329](https://github.com/newrelic/newrelic-dotnet-agent/commit/ef5f3296ee4580d73437243f5437d86fafcac4ce))

### 새로운 기능

* 사용자가 업로드되지 않아야 하는 프로세스 이름을 구성할 수 있도록 허용합니다. ([#3387](https://github.com/newrelic/newrelic-dotnet-agent/issues/3387)) ([8dedf32](https://github.com/newrelic/newrelic-dotnet-agent/commit/8dedf324fed3559c5de8697a9728b31cf8dacf96))
* `db.statement` 스팬 속성의 최대 길이를 4096바이트로 늘립니다([#3364](https://github.com/newrelic/newrelic-dotnet-agent/issues/3364)) ([#3365](https://github.com/newrelic/newrelic-dotnet-agent/issues/3365)) ([bbac2f4](https://github.com/newrelic/newrelic-dotnet-agent/commit/bbac2f49fc5094036b97f25689b07611fefde40a))

### 수정 사항

* 디버그 로깅이 활성화된 경우 발생하는 에이전트 연결 오류를 수정했습니다. ([#3395](https://github.com/newrelic/newrelic-dotnet-agent/issues/3395)) ([82fc1b6](https://github.com/newrelic/newrelic-dotnet-agent/commit/82fc1b63e9194e0ed2d44e2bcca3cf3eb5dfe6cb))
* `distributedTracing` 구성 요소에서 최소/최대 발생 속성을 복원합니다. ([#3381](https://github.com/newrelic/newrelic-dotnet-agent/issues/3381)) ([#3388](https://github.com/newrelic/newrelic-dotnet-agent/issues/3388)) ([2d0b89f](https://github.com/newrelic/newrelic-dotnet-agent/commit/2d0b89fb0561ad85bfe9a7909be7f83a88dc1067))

### 에이전트 업데이트

* [.NET 에이전트를 업데이트](/docs/agents/net-agent/installation-configuration/update-net-agent) 하려면 표준 절차를 따르세요.
* 특히 오래된 에이전트를 사용하는 경우 [구형 에이전트 업데이트](/docs/apm/agents/net-agent/installation/update-net-agent/#updating_older_net) 에 대한 주요 변경 사항 및 절차 목록을 검토하세요.

최신 에이전트 버전이 출시되면 즉시 업데이트하는 것이 좋습니다. 최신 버전으로 업그레이드할 수 없는 경우, 에이전트를 90일 이내 버전으로 업데이트하세요. [에이전트에게 최신 정보를 제공하는 방법](/docs/new-relic-solutions/new-relic-one/install-configure/update-new-relic-agent) 에 대해 자세히 알아보세요. 에이전트 릴리스 및 지원 날짜에 대한 자세한 내용은 뉴렐릭 .NET 에이전트 [EOL 정책 문서를](/docs/apm/agents/net-agent/getting-started/net-agent-eol-policy) 참조하세요.
