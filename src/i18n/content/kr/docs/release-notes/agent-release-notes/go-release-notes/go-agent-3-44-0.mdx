## 3.44.0

### 추가됨

  \* Anthropic SDK `nranthropic` 지원 추가   \* GocqlX 지원 추가 `nrgocqlx`

### 결정된

  \* 외부 테스트 소비자의 차단을 해제하기 위해 `integrationsupport` 을(를) 통해 내부 테스트 유형 내보내기     \* 이 솔루션에 기여해 주신 커뮤니티 멤버 @nitinprajwal 님께 감사드립니다   \* sql 통합에서 선택적 `INTO` 문을 지원하도록 sqlparse의 버그 수정   \* 불안정한 테스트를 유발하는 `nrnats` 의 버그 수정

### 보안

  \* Dependabot 보안 업데이트     \* `nrfiber`

### 지지 성명

우리는 최신 버전의 Go 언어를 사용합니다. 최소한 Go 팀에서 지원하는 것보다 오래된 Go 버전은 사용해서는 안 됩니다. 지원되는 Go 에이전트 버전과 타사 구성 요소에 대한 자세한 내용은 [Go 에이전트 EOL 정책을](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) 참조하세요.
