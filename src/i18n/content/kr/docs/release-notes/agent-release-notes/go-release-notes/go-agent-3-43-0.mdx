## 3.43.0

### 추가됨

* 대체 응답 작성기에 UnWrap 지원이 추가되었습니다.
  * 이 해결책에 기여해주신 커뮤니티 회원 @ArturFortunato 님께 감사드립니다.
* `nrlambda` 통합에 Lambda 함수 URL 요청 및 응답 지원이 추가되었습니다.
  * 이 솔루션에 기여해 주신 커뮤니티 멤버 @rittneje에게 감사드립니다.
* `nrawssdk` 통합에서 Opensearch/Elasticsearch 엔드포인트에 추가 Span 속성을 추가했습니다.
* sqlparse에 &quot;WITH&quot; 절 지원이 추가되었습니다.

### 결정된

* httptest 오류가 수정되었습니다.

### 보안

* 통합 연결/연속 배포(CI/CD) go 버전 업데이트

* `nrgrpc` 버전 업데이트됨

* Dependabot 보안 강화

  * `logcontext/nrlogrus`
  * `logcontext/nrlogrusplugin`
  * `logcontext-v2/nrlogrus`
  * `nrfiber`

### 지지 성명

우리는 최신 버전의 Go 언어를 사용합니다. 최소한 Go 팀에서 지원하는 것보다 오래된 Go 버전은 사용해서는 안 됩니다. 지원되는 Go 에이전트 버전과 타사 구성 요소에 대한 자세한 내용은 [Go 에이전트 EOL 정책을](https://docs.newrelic.com/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) 참조하세요.
