## 3.39.0

### 추가됨

* Go Fiber 프레임워크에 nrfiber 통합이 추가되었습니다.
  * 이 솔루션에 기여해 주신 커뮤니티 멤버 @MitulShah1에게 감사드립니다.
* 새로운 API로 업데이트된 nrslog 예제
  * 이 솔루션에 기여해 주신 커뮤니티 멤버 @frankywahl에게 감사드립니다.
* nrgin 미들웨어에 선택적 경로 필터링 기능 추가
  * 이 솔루션에 기여해 주신 커뮤니티 멤버 @frknikiz에게 감사드립니다.
* nrreddis 통합에 ConfigDatastoreKeysEnabled를 추가하여 데이터스토어 작업과 함께 키 이름을 보고할 수 있습니다.
* 에뮬레이트된 ARM 계층에서 네이티브 ARM 계층으로 GitHub Action Testing Suite를 전환했습니다.
* nrslog의 속성 값에 대한 시간 개체에 대한 지원이 추가되었습니다.

### 결정된

* 로그 메시지의 고정 링크 메타데이터 위치

### 지지 성명

우리는 최신 버전의 Go 언어를 사용합니다. 최소한 Go 팀에서 지원하는 것보다 오래된 Go 버전은 사용해서는 안 됩니다. 지원되는 Go 에이전트 버전과 타사 구성 요소에 대한 자세한 내용은 [Go 에이전트 EOL 정책을](https://docs.newrelic.com/docs/apm/agents/go-agent/get-started/go-agent-eol-policy/) 참조하세요.
