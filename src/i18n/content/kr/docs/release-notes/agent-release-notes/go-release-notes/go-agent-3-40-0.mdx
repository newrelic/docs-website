## 3.40.0

### 추가됨

* 지정된 거래에 대한 Apdex 점수를 무시하기 위해 `txn.IgnoreApdex()` 함수가 추가되었습니다.
* 환경 변수를 통해 사용자 정의 속성을 추가했습니다. `NEW_RELIC_APPLICATION_LOGGING_FORWARDING_CUSTOM_ATTRIBUTES`
* Connect 라이브러리에 `nrconnect` 통합을 추가했습니다. Connect는 브라우저 및 gRPC 호환 HTTP API를 구축하기 위한 간편한 라이브러리입니다.
  * 이 솔루션에 기여해 주신 커뮤니티 멤버 @castaneai에게 감사드립니다.
* mongodb-v2 라이브러리를 지원하는 `nrmongo-v2` 통합이 추가되었습니다.
* 개편된 GitHub Actions 테스트 모음
  * mongodb 및 pgx5 통합을 위한 dockerized 데이터베이스 지원이 추가되었습니다.

### 결정된

* 향상된 쿼리 파트너, nrpgx5 통합의 표현 방식

* 컨테이너 활용도를 식별하는 데 경쟁 조건이 발생하는 버그를 수정했습니다.

* DatastoreSegment에서 DynamoDB 테이블 이름과 인덱스 이름을 캡처합니다.
  * 이 솔루션에 기여해 주신 커뮤니티 멤버 @rittneje에게 감사드립니다.

* 보고된 보안 또는 기타 지원 문제로 인해 타사 라이브러리 버전이 업데이트되었습니다.

  * `github.com/gofiber/fiber/v2` `nrfiber` 통합에서 2.52.7로
  * `github.com/go-chi/chi/v5` `nrgochi` 통합에서 5.2.2로

### 지지 성명

우리는 최신 버전의 Go 언어를 사용합니다. 최소한 Go 팀에서 지원하는 것보다 오래된 Go 버전은 사용해서는 안 됩니다. 지원되는 Go 에이전트 버전과 타사 구성 요소에 대한 자세한 내용은 [Go 에이전트 EOL 정책을](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) 참조하세요.
