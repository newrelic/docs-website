## 3.44.1

### 결정된

* 이전 릴리스의 잘못된 태그 수정

### 보안

* Dependabot 보안 업데이트

  * `nrgochi` 통합이 v5.2.4로 업데이트되었습니다.
  * `nrmongo` 통합이 v1.17.7로 업데이트됨
  * `nrmongo-v2` 통합이 v2.4.2로 업데이트됨
  * `nrnats` 통합이 v2.11.15로 업데이트됨

### 지지 성명

최신 버전의 Go 언어를 사용합니다. 최소한 Go 팀에서 계속 지원하는 Go 버전을 사용하십시오. 지원되는 Go 에이전트 및 타사 구성 요소 버전에 대한 자세한 내용은 [Go 에이전트 EOL 정책](/docs/apm/agents/go-agent/get-started/go-agent-eol-policy) 을 참조하십시오.
