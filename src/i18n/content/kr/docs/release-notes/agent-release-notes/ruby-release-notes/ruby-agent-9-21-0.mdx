## v9.21.0

* **기능: Serverless APM 모드에서 이벤트 소스 이름을 트랜잭션 이름 접두사로 사용**

  이제 에이전트는 Serverless APM 모드에서 이벤트 소스 이름을 트랜잭션 이름의 접두사로 사용합니다. 이를 통해 뉴렐릭 UI 에서 해당 소스의 출처를 더 잘 식별할 수 있습니다. [PR#3245](https://github.com/newrelic/newrelic-ruby-agent/pull/3245)

* **Bugfix: 스팬이 시작되는 스레드를 추적하는 방법에 대한 변경된 논리를 되돌립니다.**

  이 변경 사항은 업데이트된 논리로 인해 발생한 문제를 해결하여 스팬이 시작되는 스레드를 추적하는 이전 동작을 복원합니다. [@david-zw-liu](https://github.com/david-zw-liu) 님, 이 문제를 알려주셔서 감사합니다. [PR#3248](https://github.com/newrelic/newrelic-ruby-agent/pull/3248)
