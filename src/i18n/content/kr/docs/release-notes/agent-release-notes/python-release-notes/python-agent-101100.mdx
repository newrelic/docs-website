## Notes

이번 끌어 당기는 에이전트 릴리스에는 Google 생성형 AI SDK 및 축소된 새로운 Kinesis 메서드를 통해 Google Gemini에 대한 지원이 추가되었습니다.

[끌어오기 패키지 색인을](https://pypi.python.org/pypi/newrelic) 통해 `easy_install/pip/distribute` 사용하여 에이전트를 설치하거나 [뉴렐릭 다운로드 사이트](https://download.newrelic.com/python_agent/release) 에서 직접 다운로드하세요.

## 새로운 기능

* Google 생성형 AI SDK를 통해 Google Gemini에 대한 지원을 추가합니다.

  * [google-생성형 AI](https://pypi.org/project/google-genai/) 에 대한 지원을 추가합니다. 에이전트는 자동으로 동기 및 부품 [임베딩을](https://ai.google.dev/gemini-api/docs/embeddings) 위해 `embed_content` 호출을 수행합니다. 이제 에이전트는 스트리밍이 아닌 경우 텍스트 입력에 대한 동기 및 비동기 [텍스트 생성을](https://ai.google.dev/gemini-api/docs/text-generation) 지원합니다. 여기에는 단일 텍스트 전용 입력을 위한 `generate_content` 에 대한 호출과 다중 턴 대화를 위한 `send_message` 에 대한 호출이 포함됩니다.

* 새로운 AWS Kinesis 메서드를 사용하세요

  * [botocore](https://pypi.org/project/botocore/) [AWS ](https://aws.amazon.com/kinesis/)`tag_resource`,, `untag_resource` 를 포함한 새로운 Kinesis 메서드를 지원하기 위해 측정 및 계측을 `list_tags_for_resource` 추가합니다.

## 지지 성명

최신 에이전트 버전이 출시되면 즉시 업데이트하는 것이 좋습니다. 최신 버전으로 업그레이드할 수 없는 경우, 에이전트를 90일 이내 버전으로 업데이트하세요. 에이전트에게 최신 정보를 제공하는 방법에 대해 [자세히](/docs/new-relic-solutions/new-relic-one/install-configure/update-new-relic-agent/) 알아보세요.

에이전트 릴리스 및 지원 날짜에 대한 정보는 뉴렐릭 끌어 당기는 에이전트 [EOL 정책을](/docs/apm/agents/python-agent/getting-started/python-agent-eol-policy/) 참조하세요.
