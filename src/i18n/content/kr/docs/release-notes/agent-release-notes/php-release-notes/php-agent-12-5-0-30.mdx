### 지원 성명

* 향후 릴리스에서는 모범 사례와 보안을 위해 RPM 서명 키가 순환됩니다. 해당 릴리스에 가까워지면 구체적인 날짜와 RPM 사용자를 위한 필요한 조치가 제공될 것입니다.

* 2026년 2월 28일 이후 모든 PHP 에이전트 릴리스에서 다음 라이브러리 및 프레임워크 버전에 대한 PHP 에이전트 지원이 중단됩니다.

  * 다음 라이브러리 &amp; 프레임워크 버전의 경우 뉴렐릭은 지원되는 최신 버전으로 업그레이드할 것을 권장합니다.

    * Symfony 4.x
    * 라라벨 7.x, 9.x, 10.x
    * Slim 3.x

  * 다음 라이브러리 및 프레임워크에 대한 지원이 모든 버전에서 중단됩니다.

    * 미디어위키
    * 젠드 프레임워크
    * 루멘

* 뉴렐릭은 에이전트를 정기적으로, 그리고 최소 3개월마다 업그레이드할 것을 권장합니다. 지원되는 에이전트 버전 및 EOL 타임라인에 대한 자세한 내용은 [뉴렐릭 PHP 에이전트 EOL 정책 을](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-eol-policy/) 확인하세요.

* 지원되는 PHP 버전 및 플랫폼에 대한 최신 정보는 [PHP 에이전트 호환성 및 요구 사항](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/) 페이지를 참조하세요.

<Callout variant="important">
  **지원되지 않는 PHP 버전이나 플랫폼을 사용하여 설치하는 경우, PHP 에이전트 패키지의 자동 업데이트 메커니즘을 비활성화하는 것이 좋습니다.** 패키지 업그레이드를 위한 제외 목록에 PHP 에이전트 패키지를 추가하면 됩니다. 또는 PHP 에이전트 패키지를 오래되고 지원되지 않는 기능을 지원하는 에이전트 버전으로 고정할 수도 있습니다. 업그레이드를 막지 못하면 최신 에이전트 릴리스가 설치되고 필수이지만 지원되지 않는 기능에 대한 지원이 중단될 수 있습니다. 이로 인해 APM 데이터 수집이 중단됩니다.

  영향을 받는 PHP 에이전트 패키지는 다음과 같습니다.

  * newrelic-php5
  * newrelic-php5-common
  * newrelic 데몬
</Callout>
