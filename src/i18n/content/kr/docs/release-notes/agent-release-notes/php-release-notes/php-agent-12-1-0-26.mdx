## 뉴렐릭 PHP 에이전트 v12.1.0.26

### 새로운 기능

* 업스트림 w3c 트레이스 전혀 결정에 따라 샘플링 제어를 추가합니다 - [#1104](https://github.com/newrelic/newrelic-php-agent/pull/1104)
  * [여기에서](https://docs.newrelic.com/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-sampler-remote-parent-sampled)새로운 ini 값을 확인하세요
* 측정을 건너뛰기 위한 설정 옵션 추가, 프레임워크 예외 처리기의 구현 - [#1113](https://github.com/newrelic/newrelic-php-agent/pull/1113)
  * [여기에서](https://docs.newrelic.com/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-ignore-framework-exception-handler)새로운 ini 값을 확인하세요

### 보안

* Golang 버전을 1.25.1로 올립니다.

### 버그 수정

* `newrelic_end_transaction(true)` 호출 시 잠재적인 memleak 문제 해결 - [#1072](https://github.com/newrelic/newrelic-php-agent/pull/1072)
* 녹음하지 않을 때는 측정을 시도하지 마세요. - [#1112](https://github.com/newrelic/newrelic-php-agent/pull/1112)

### 지지 성명

* 다음 라이브러리/프레임워크 버전에 대한 PHP 에이전트 지원은 2025년 9월 30일에 종료됩니다.

  * 워드프레스 5.9
  * 드루팔 8.x
  * 라라벨 6.x, 8.x
  * PHPUnit 8.x

* 뉴렐릭은 에이전트를 정기적으로, 그리고 최소 3개월마다 업그레이드할 것을 권장합니다. 지원되는 에이전트 버전 및 EOL 타임라인에 대한 자세한 내용은 [뉴렐릭 PHP 에이전트 EOL 정책 을](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-eol-policy/) 확인하세요.

* 지원되는 PHP 버전 및 플랫폼에 대한 최신 정보는 [PHP 에이전트 호환성 및 요구 사항](https://docs.newrelic.com/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/) 페이지를 참조하세요.

<Callout variant="important">
  **지원되지 않는 PHP 버전이나 플랫폼을 사용하여 설치하는 경우, PHP 에이전트 패키지의 자동 업데이트 메커니즘을 비활성화하는 것이 좋습니다.** 패키지 업그레이드를 위한 제외 목록에 PHP 에이전트 패키지를 추가하면 됩니다. 또는 PHP 에이전트 패키지를 오래되고 지원되지 않는 기능을 지원하는 에이전트 버전으로 고정할 수도 있습니다. 업그레이드를 막지 못하면 최신 에이전트 릴리스가 설치되고 필수이지만 지원되지 않는 기능에 대한 지원이 중단될 수 있습니다. 이로 인해 APM 데이터 수집이 중단됩니다.

  영향을 받는 PHP 에이전트 패키지는 다음과 같습니다.

  * newrelic-php5
  * newrelic-php5-common
  * newrelic 데몬
</Callout>
