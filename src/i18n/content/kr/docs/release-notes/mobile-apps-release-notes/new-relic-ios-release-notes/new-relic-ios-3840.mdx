### Notes

이제 응용 프로그램 인텔리전스 공지 이벤트와 문제를 확인하고 수동으로 종료하는 기능이 제공됩니다.

### 새로운 기능

* AI 공지 이벤트 목록에 추가되었습니다.

### 개량

* 다중 쿼리를 지원하도록 차트 기능을 개선했습니다.
* 이슈 피드 세부 정보를 개선하여 확인, 확인 취소 및 종료 기능을 지원합니다.
* 차트를 탭한 후 범례 항목을 탭할 때 발생하던 충돌 오류를 수정했습니다.
* 주요 트랜잭션, Kubernetes 컨테이너 및 Node 차트를 새로운 차트 라이브러리로 변환했습니다.
* VM 페이지를 새 차트 라이브러리로 변환했습니다.
