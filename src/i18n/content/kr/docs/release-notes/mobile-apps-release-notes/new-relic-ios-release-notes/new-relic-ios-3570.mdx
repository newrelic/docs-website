### Notes

[아이폰 앱](https://docs.newrelic.com/docs/mobile-apps/new-relic-mobile-apps/ios-app/introduction-ios-mobile-app) 에서 바로 NRQL 편집 기능을 사용해 보세요!

### 새로운 기능

* iPhone에 NRQL 쿼리 기록을 추가합니다(iPad는 추후 지원 예정).
* NRQL 쿼리를 작성할 수 있는 기능을 추가합니다.
* 공지 이벤트 링크, 인앱 NRQL 편집기를 표시하기 위해 NRQL 쿼리를 목표로 합니다.
* APM, 브라우저, 모바일 및 주요 프로세서용 iPad에 대한 자세한 화면이 포함되어 있습니다.
