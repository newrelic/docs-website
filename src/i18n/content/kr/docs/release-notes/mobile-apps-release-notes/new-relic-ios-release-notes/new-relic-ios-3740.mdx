### Notes

사용자에게 적절한 액세스 제어 권한이 있는 경우 공지 이벤트를 수동으로 닫는 기능이 추가되었습니다.

### 새로운 기능

* iPhone과 iPad 모두에 새로운 **Close alert event** \[공지 이벤트 닫기] 버튼이 있어 공지 이벤트 및 해당 공지 이벤트와 관련된 모든 회원을 수동으로 닫을 수 있습니다.
