### Notes

휴대폰에 사용자 의견 제출 페이지를 추가하여 모바일 앱에서 의견을 보낼 수 있도록 했습니다.

### 새로운 기능

* 휴대폰에 새로운 피드백 페이지가 생겼습니다.
* APM 개요에 호스트/인스턴스의 displayName을 추가합니다.

### 버그 수정

* 백그라운드에서 공지 이벤트를 확인할 때 발생하는 충돌 문제를 해결했습니다.
* 대시보드에 작은 빌보드 차트가 있을 때 발생하던 충돌 문제를 수정했습니다.
