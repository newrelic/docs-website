### 새로운 기능

* 군대 공지 이벤트에 요약 KPI(핵심 성과 지표(KPI)) 및 KPI 차트가 추가되었습니다.
* 푸시알림에 공지이벤트 확인 추가
* 애플리케이션 페이지를 특정 호스트 또는 JVM 인스턴스로 제한할 수 있는 기능이 추가되었습니다.
* 비 웹 프로세서 차트에 추가되었습니다.
* **iPad**: 브라우저 제품에 추가되었습니다.

### 개량

* 새로운 활동 표시기를 추가했습니다.
* 앱 전체의 검색 기능을 재설계했습니다.
* **iPhone**: 상세 보기에서 모든 차트를 확장했습니다.
* **iPhone**: 공지에서 피벗 &apos;이동&apos; 링크를 변경하여 뒤로 가기 버튼을 누르면 목록으로 이동하도록 했습니다.

### 버그 수정

* 인프라 제품에 대한 유니버설 링크를 열 때 발생하던 충돌 문제를 수정했습니다.
