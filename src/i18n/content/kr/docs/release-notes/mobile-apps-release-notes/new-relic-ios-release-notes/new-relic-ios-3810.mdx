### Notes

APM 및 OpenTelemetry 서비스를 위한 컨텍스트 기반 로그 소개

### 새로운 기능

* APM / OpenTelemetry 서비스 상세 보기 화면에 새로운 로그 메뉴 항목이 추가되었습니다. 이 항목을 통해 (구성된 경우) 해당 서비스에 한정하여 최근 몇 분 동안의 로그를 볼 수 있습니다.

### 개량

* 워치 앱에서 즐겨찾기가 로드되지 않던 문제를 수정했습니다.
* APM 공지 이벤트의 KPI를 보려고 할 때 발생하던 충돌 문제를 수정했습니다.
