버그 수정 및 UI 업데이트

### 새로운 기능

* 최상위 뷰와 대부분의 APM 뷰를 SwiftUI로 변환합니다.
* 공지사항 이상 행동 차트를 추가합니다.
* .Net VM 상세 페이지를 추가합니다.

### 개량

* 람다 호출 시 시간 제한을 준수하지 않는 오류 및 문제를 수정합니다.
* 광고판 서식 관련 문제를 수정했습니다.
* APM 엔티티를 볼 때 계정 간 로그가 로드되지 않던 문제를 수정했습니다.
