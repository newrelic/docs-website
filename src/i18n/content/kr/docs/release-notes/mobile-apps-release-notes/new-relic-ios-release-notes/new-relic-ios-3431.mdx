### Notes

* 공지 이벤트 중 더 빠르게 방향을 잡을 수 있는 도구인 뉴렐릭 레이더의 공지 이벤트 컨텍스트 지원.
* 공지 이벤트 컨텍스트는 시스템의 비정상적인 동작을 감지하고 이를 공지 정보와 함께 배치하여 공지 이벤트 중에 더 빠른 방향을 제공합니다.

### 개량

* 개선된 로그인 흐름
* 오류 트레이스 속성을 통해 검색할 때 동작이 개선되었습니다.

### 버그 수정

* 네트워크 오류나 서버 문제로 인해 로그아웃되는 현상을 수정했습니다. 이제 로그아웃되는 대신 다시 시도할 수 있습니다.
