### Notes

이제 모든 회의 기능에 대한 탐색기가 포함된 전역 탭 표시줄이 제공됩니다.

### 새로운 기능

* 다른 여러 유형의 부품 통합을 지원하는 부품 탐색기 페이지에 추가되었습니다.
* 어디에서든 홈, 탐색기, 대시보드, 알림 및 쿼리에 빠르게 액세스할 수 있도록 전역 탭 바가 추가되었습니다.
* 통합 보고 서비스에 대한 지원이 추가되었습니다.
* EU 계정에 대한 공지 이벤트에 이상적으로 지원을 추가합니다.

### 수정 사항

* 사용자 지정 타임 피커를 업데이트하여 시작과 종료 사이에 5분 간격을 허용합니다.
* 피드백 양식 관련 문제를 해결합니다.
