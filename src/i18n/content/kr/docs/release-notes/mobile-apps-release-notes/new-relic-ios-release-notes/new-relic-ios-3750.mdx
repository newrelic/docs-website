### Notes

향상된 합성 모니터 세부 정보와 대시보드 위젯에서 쿼리를 볼 수 있는 기능이 추가되었습니다.

### 새로운 기능

* 대시보드 위젯에 **View query** \[보기 쿼리] 옵션을 추가합니다.
* 공지 이벤트 회원에서 NRQL 항상 확인/실행할 수 있는 기능을 추가합니다.
* 앱 내에서 합성 모니터를 실행할 수 있는 기능을 추가합니다.
* 합성 모니터에 결과 목록을 추가합니다.
* 대부분의 요약 화면에서 당겨서 새로 고침하는 기능을 추가합니다.
