### 새로운 기능

* 최근 위반 사항을 보여주는 인프라 호스트 상태 팝업이 추가되었습니다.
* 구성으로 필터링하거나 APM에서 볼 수 있는 기능이 포함된 호스트 구조 팝업을 추가했습니다.
* 앱 전체의 일관성을 유지하기 위해 새로운 UI 디자인을 추가했습니다.

### 버그 수정

* 인프라 이벤트에서 드물게 발생하던 충돌 현상을 수정했습니다.
* 차트 라이브러리에서 발생하던 충돌 오류를 수정했습니다.
* 공지 이벤트 화면의 충돌 문제를 수정했습니다.
