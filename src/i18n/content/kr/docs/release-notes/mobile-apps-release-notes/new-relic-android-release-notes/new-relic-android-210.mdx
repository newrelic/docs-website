### 새로운 기능

* 유니버설 링크 - 이제 이메일, 채팅 등에서 newrelic.com 링크를 클릭하면 해당 링크가 앱에서 열립니다.

### 개량

* 버전 2.1.0 세션 토큰 갱신 방식을 개선하여 401 및 403 오류 발생 횟수를 줄일 것입니다.
* 이제 모바일 충돌 목록 화면에서 시간 범위를 확인할 수 있습니다.

### 수정 사항

* 모바일 충돌 해결/재개 버튼을 사용하여 모바일 충돌을 해결하거나 다시 실행할 수 있습니다.
* 공지이벤트 확인 버튼
* 공지사항 공지에서 확인
