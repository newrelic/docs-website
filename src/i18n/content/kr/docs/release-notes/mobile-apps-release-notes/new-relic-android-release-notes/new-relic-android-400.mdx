### Notes

* 이번 릴리스에서는 승인된 사용자를 위해 열려 있는 공지 이벤트를 닫는 기능이 추가되었으며, 뉴렐릭 원에서 공유된 링크에 대한 딥 링크 지원도 강화되었습니다.
* 이번 릴리스에는 플러그인에 대한 기능 플래그 확인 기능이 포함되어 있어, 계정에 플러그인을 활성화하는 기능 플래그가 없는 경우 플러그인이 메뉴 항목으로 표시되지 않습니다.
