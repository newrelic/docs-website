### Notes

공지 이벤트 중 더 빠른 방향을 알려주는 도구인 [뉴렐릭 Radar](https://docs.newrelic.com/docs/radar/new-relic-radar/getting-started/introduction-radar) 의 공지 이벤트 컨텍스트를 지원합니다. 공지 이벤트 컨텍스트는 시스템의 비정상적인 동작을 감지하고 이를 공지 정보와 함께 배치하여 공지 이벤트 중에 더 빠른 방향을 제공합니다.

### 새로운 기능

* 레이더 공지 이벤트 컨텍스트

### 개량

* 지연 시간 및 네트워크 사용량을 줄이기 위해 목록 호출이 개선되었습니다.

### 수정 사항

* 상세 보기에서 뒤로 가기 버튼을 사용할 때 충돌이 발생합니다.
