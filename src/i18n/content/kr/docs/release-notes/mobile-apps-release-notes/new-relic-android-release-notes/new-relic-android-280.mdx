### Notes

버전 2.8.0에 추가된 멋진 새 기능들

### 새로운 기능

* 군대 공지 이벤트에 요약 KPI(핵심 성과 지표(KPI)) 및 KPI 차트가 추가되었습니다.
* APM 상세 페이지를 특정 호스트 또는 JVM 인스턴스로 제한할 수 있는 기능이 추가되었습니다.
* 새로운 활동 표시기를 추가했습니다.

### 개량

* 스파크 차트를 제거하세요
* 공지 이벤트의 시간이 포함된 새 시간 창
* 공지 이벤트의 탐색 흐름 개선

### 수정 사항

* 푸시 공지에서 공지 이벤트 확인 문제 수정
