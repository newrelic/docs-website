### 새로운 기능

* 패싯 테이블 위젯은 열 값을 전체 대시보드에 대한 빠른 필터로 사용할 수 있습니다.

### 수정 사항

* 대시보드 필터가 제대로 초기화되지 않음
* 클래식 공지 이벤트가 올바르게 필터링되지 않습니다.
* 이전 스타일의 공지 링크가 사용자를 공지 이벤트로 안내하지 않습니다.
