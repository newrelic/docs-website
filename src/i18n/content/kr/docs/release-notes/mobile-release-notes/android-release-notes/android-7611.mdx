## 수정 사항

* 메모리 샘플링 프로세스에서 `NullPointerException` 문제가 해결되었습니다.
* Microsoft Intune MAM SDK에서 `ClassCastException` 호환성을 방지하기 위해 앱 출시 시간 초기화 중에 안전한 캐스팅을 추가했습니다.
* 사용자 코드 예외로 인한 에이전트 충돌을 방지하기 위해 okhttp 콜백 보호 기능이 개선되었습니다.
* 네이티브 크래시 보고서에서 중복된 크래시가 수정되었습니다.

## 지지 성명

최소한 3개월에 한 번씩 에이전트를 업데이트하는 것이 좋습니다. [모바일 모니터링 에이전트 EOL 정책](/docs/mobile-monitoring/new-relic-mobile/get-started/mobile-agents-eol-policy/#android-eol) 에서 Android 에이전트 지원에 대한 구체적인 정책과 날짜를 찾아보세요.
