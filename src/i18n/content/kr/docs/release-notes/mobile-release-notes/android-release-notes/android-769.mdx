## 버그 수정

* 앱이 포그라운드와 백그라운드 사이를 여러 번 전환한 후 트레이스 리스너가 제대로 제거되지 않는 메모리 누수 문제가 해결되었습니다.
* 가끔씩 데이터가 뉴렐릭에 보고되지 않는 문제를 스레드 동기화에서 해결했습니다.
* ANR 중 로그 보고와 관련된 문제를 해결하여 안정성과 성능을 개선했습니다.
* 호환성을 보장하고 충돌을 방지하기 위해 커튼 라이브러리와 관련된 중복 클래스 문제를 수정했습니다.
* 견고성을 강화하기 위해 setUserId 메서드에서 null 포인터 예외 문제를 해결했습니다.
* 장치의 네트워크 오류로 인해 요청이 실패한 경우 URL이 올바르게 보고되지 않는 Retrofit Interceptor 문제가 해결되었습니다.
