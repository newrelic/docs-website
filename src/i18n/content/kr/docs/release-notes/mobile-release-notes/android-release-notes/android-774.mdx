## 버그 수정

* SharedPreferences에 페이로드 데이터가 누적되어 발생하는 OutOfMemory 문제를 수정했습니다. 이제 캐시된 페이로드는 디스크에 저장되며, 연결 상태가 좋지 않은 기기에서 무제한으로 증가하는 것을 방지하기 위해 상한이 설정됩니다.

* 매 도달 가능성 확인 시 ConnectivityManager에 대한 반복적인 Binder IPC 호출로 인해 발생하던 ANR 문제를 해결했습니다. 네트워크 상태는 이제 등록된 네트워크 콜백을 통해 비동기적으로 추적됩니다.

지지 성명

최소 3개월마다 에이전트를 업데이트하는 것이 좋습니다. 모바일 모니터링 에이전트 EOL 정책에서 안드로이드 에이전트 지원에 대한 구체적인 정책 및 날짜를 확인하세요.
