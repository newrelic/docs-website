## 수정 사항

* 이벤트 데이터 전송의 기본 간격을 10분에서 1분으로 변경했습니다.
* DexGuard 9에서 다른 플레이버 설정으로 인해 스냅샷 파일이 전송되지 않던 문제를 수정했습니다.
* 모바일 세션 리플레이와 관련된 여러 충돌을 수정했습니다.
