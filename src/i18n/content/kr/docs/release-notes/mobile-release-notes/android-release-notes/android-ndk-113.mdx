## 고치다

* 16KB 페이지 크기에 대한 지원이 추가되었습니다.

## 지지 성명

* Android NDK 에이전트 버전 1.1.1 이상을 사용하려면 Android 에이전트 7.5.1 이상이 필요합니다.
* 안드로이드 NDK 에이전트 버전 1.1.0 이상 버전에는 Android 에이전트 7.5.0 이상이 필요합니다.
* Android NDK 에이전트 버전 1.0.3 이상을 사용하려면 Android 에이전트 6.11.0 이상이 필요합니다.
* 버전 1.0.0 - 1.0.2는 Android 에이전트 6.10.0 이하와 호환됩니다.

최소한 3개월에 한 번씩 정기적으로 에이전트를 업그레이드하는 것이 좋습니다.
