## 수정 사항

* 고유한 식별자를 생성하려면 SecureRandom() 함수를 사용하십시오.
* 고부하 샘플링 사용 시 메인 스레드에서 발생할 수 있는 ANR 문제를 해결합니다.
* 낮은 SDK 버전에 대한 예외 처리를 추가합니다.
