## 중요

* 이번 릴리스에서는 뉴렐릭 iOS 기기의 필수 iOS 버전이 iOS 16으로 상향 조정되었습니다. iOS 15 지원은 향후 버전에서 다시 제공될 예정입니다.

## 이번 릴리스의 새로운 기능

* 오류가 발생한 세션에 대한 샘플링과 세션 리플레이와 관련된 수정된 문제를 포함하도록 모바일 세션 리플레이의 공개 미리보기를 업데이트했습니다.

## 이 릴리스에서 수정됨

* 특정 예외 상황에서 분산 추적이 작동하지 않던 문제를 수정했습니다.
* 확인된 문제점에 대한 안정성 수정 사항을 추가했습니다.
