## 개량

* Native Android 에이전트를 버전 7.7.0으로 업데이트했습니다.
* 지능형 세션 녹화: 최적화된 데이터 캡처를 위해 ERROR 모드(충돌 발생 시 15-30초간 버퍼링 녹화)와 FULL 모드(연속 녹화)를 도입했습니다.
* 세션 제어 API: NewRelic.recordReplay()가 추가되었습니다. 그리고 NewRelic.pauseReplay() 재생 녹화 상태를 수동으로 프로그래밍 방식으로 제어할 수 있도록 합니다.
* 네트워킹 버그 수정: OkHttp3에서 peekBody() 함수가 콘텐츠 길이를 알 수 없는 스트리밍 응답(SSE/청크 방식)에서 무기한 차단을 일으키는 문제를 수정했습니다.
