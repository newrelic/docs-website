## 기능

### 독립형 에이전트

* **브라우저 에이전트와 분리됨**: 이제 비디오 에이전트는 자체적인 데이터 수집 로직을 사용하여 독립적으로 작동합니다. 이러한 분리는 오버헤드를 줄이고 비디오 데이터가 브라우저 운영자의 데이터 수집 일정에 얽매이지 않고 수집되어 뉴럴릭으로 전송되도록 보장합니다.
