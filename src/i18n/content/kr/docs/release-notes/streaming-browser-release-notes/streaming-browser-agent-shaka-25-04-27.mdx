이번 뉴렐릭 샤카 트래커 릴리스에는 오류 보고 방식에 대한 변경 사항이 포함되어 있습니다.

## 주요 변경 사항

* **오류 보고 업데이트**: 이전에 오류 설명을 식별하는 데 사용되었던 `errorName` 속성은 더 이상 사용되지 않습니다. `errorMessage` 속성은 이제 자세한 오류 설명을 전달하는 표준입니다. 오류 정보를 사용하는 모든 모니터링 또는 로깅 시스템은 향후 `errorMessage` 속성을 활용하도록 업데이트하십시오.

## 업그레이드 참고 사항

* 오류 분석을 위해 `errorName` 속성에 의존하는 사용자는 지속적인 기능과 자세한 오류 정보에 대한 접근을 보장하기 위해 `errorMessage` 속성으로 마이그레이션할 계획을 세워야 합니다.
