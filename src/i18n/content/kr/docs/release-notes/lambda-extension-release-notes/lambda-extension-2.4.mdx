## Notes

* Lambda가 계속 실행되도록 메인 함수의 패닉 처리 기능을 개선했습니다.

* `APM` 모드 및 `Shutdown` 로그에 대한 새로운 텔메트리 버퍼를 추가했습니다.

* `APM` 모드에서 향상된 버전 관리 지원 및 로깅 기능 제공

## 결정된

* 플랫폼 보고서 항목을 압축 대신 `json` 으로 전송하도록 업데이트했습니다.

* EU 엔드포인트에서 가져온 라이선스 키에 대한 탐지 `SSM/Secrets Manager`
