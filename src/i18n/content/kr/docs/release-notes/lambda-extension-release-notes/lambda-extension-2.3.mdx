## 결정된

* 문제를 해결하기 위해 Go std lib 버전을 `1.23.12` 으로 업데이트했습니다. `CVE-2025-47907`
* APM 모드: APM 모드에서 SM/SSM과 관련된 라이선스 키 문제가 해결되었으며 로그 접두사가 해결되었습니다.

## 새로운 기능

* 기존 접두사에 로그 레벨 접두사 `DEBUG` 및 `INFO` 을(를) 추가했습니다. `[NR_EXT]`
* APM 모드: APM 모드에 태그가 추가되었습니다.
* APM 모드: APM 연결을 위해 호스트에 Lambda ARN 추가
* APM 모드: 업데이트된 백오프 기간 `NEW_RELIC_HOST`
* APM 모드: APM 오류 이벤트 데이터 리팩터링
* APM 모드: 루비 지원 추가
