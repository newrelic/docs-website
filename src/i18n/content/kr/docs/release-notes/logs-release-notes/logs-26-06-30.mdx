### Fluent Bit 5.0.8 지원

이제 사용자는 Fluent Bit 버전 5.0.8을 받게 됩니다. 자세한 내용은 [Fluent Bit 공지 사항](https://fluentbit.io/announcements/)을 참조하세요.

이 릴리스에는 [뉴렐릭 Fluent Bit 출력 플러그인 3.7.0](https://github.com/newrelic/newrelic-fluent-bit-output/releases/tag/v3.7.0)이 포함되어 있습니다.

### 추가됨

* 뉴렐릭 Fluent Bit 출력 플러그인 3.7.0 통합

### 변경됨

* CVE 수정을 위해 Go 버전이 1.26.4로 업그레이드되었습니다.
* 인프라 에이전트는 Fluent Bit 5.0.8 패키지(Linux) 및 5.0.7(Windows)을 권장합니다.
* [Fluent Bit 출력 플러그인 도커 이미지](https://hub.docker.com/r/newrelic/newrelic-fluentbit-output) 는 출력 플러그인 3.7.0과 함께 Fluent Bit 5.0.8을 사용합니다.
* [뉴렐릭 로깅 Helm 차트](https://github.com/newrelic/helm-charts/tree/master/charts/newrelic-logging) 는 새로운 플러그인 이미지를 설치합니다.
* Windows 도커 이미지는 Fluent Bit 5.0.7을 사용합니다(5.0.8 Windows 이미지는 아직 업스트림에서 사용할 수 없음).
