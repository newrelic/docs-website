### AWS-log-ingestion 람다의 CVE-2025-53643 수정

최신 릴리스에서는 AWS-log-ingestion 람다에서 CVE-2025-53643이 수정되었습니다.

### 변경됨

* aiohttp 라이브러리를 3.13.2로 업데이트했습니다. 및 시 버전을 1.8.3으로 업데이트하고 관련 의존성/종속성을 업데이트했습니다.
