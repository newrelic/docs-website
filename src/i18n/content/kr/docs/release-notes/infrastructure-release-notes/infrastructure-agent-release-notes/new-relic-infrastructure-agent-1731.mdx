에이전트의 새 버전이 릴리스되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent)하려면 표준 절차를 따르십시오. 뉴렐릭은 에이전트를 정기적으로, 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스부터 지원되는 가장 오래된 버전은 [인프라 에이전트 1.64.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1640/)입니다.

### 변경됨

* nri-docker 의존성/종속성을 2.6.10으로 업데이트했습니다, [#2220](https://github.com/newrelic/infrastructure-agent/pull/2220)에서 nri-flex 종속성을 v1.17.7로, docker 종속성을 v29로 업데이트
* [#2202](https://github.com/newrelic/infrastructure-agent/pull/2202)에서 Windows용 인프라 에이전트의 도커 이미지를 빌드했습니다.
* [#2213](https://github.com/newrelic/infrastructure-agent/pull/2213)에서 Windows 이미지 릴리스 워크플로우 업데이트

### 보안

* [CVE-2026-34040](https://github.com/advisories/GHSA-x744-4wpc-v9h2)을 해결합니다.
