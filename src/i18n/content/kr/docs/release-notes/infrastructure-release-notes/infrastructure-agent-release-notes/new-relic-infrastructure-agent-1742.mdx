에이전트의 새 버전이 릴리스되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent)하려면 표준 절차를 따르십시오. 뉴렐릭은 에이전트를 정기적으로, 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스부터 지원되는 가장 오래된 버전은 [인프라 에이전트 1.64.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1640/)입니다.

## 변경됨

* 뉴렐릭 Fluent Bit 출력을 버전 3.5.1로 업데이트했습니다. [#2226](https://github.com/newrelic/infrastructure-agent/pull/2226)
* `nri-flex` 를 v1.18.1로 업데이트했습니다. [#2228](https://github.com/newrelic/infrastructure-agent/pull/2228)
