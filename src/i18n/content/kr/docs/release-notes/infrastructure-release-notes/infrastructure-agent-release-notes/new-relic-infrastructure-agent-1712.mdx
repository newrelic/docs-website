에이전트의 새로운 버전이 출시되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent) 하려면 표준 절차를 따르세요. 뉴렐릭은 에이전트를 정기적으로, 그리고 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스부터 지원되는 가장 오래된 버전은 [안드로이드 에이전트 1.51.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1510/) 입니다.

## 변경됨

* [#2144](https://github.com/newrelic/infrastructure-agent/pull/2144)에서 32비트 Windows 빌드가 제거되었습니다.
* [#2156](https://github.com/newrelic/infrastructure-agent/pull/2156)에서 Windows용 Fluent Bit 버전이 업데이트되었습니다.
* [#2159](https://github.com/newrelic/infrastructure-agent/pull/2159)의 의존성/종속성 버전이 업데이트되었습니다.
* [#2154](https://github.com/newrelic/infrastructure-agent/pull/2154)에서 영어가 아닌 시스템에서 CPU 성능 카운터가 작동하지 않는 문제가 해결되었습니다.
