에이전트의 새 버전이 출시되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent) 하려면 표준 절차를 따르십시오. 뉴렐릭은 에이전트를 정기적으로, 그리고 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스를 기준으로 지원되는 가장 오래된 버전은 [인프라 에이전트 1.63.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1630/) 입니다.

* [#2176](https://github.com/newrelic/infrastructure-agent/pull/2176)에서 지원이 종료된 OS 지원을 모두 제거했습니다.
