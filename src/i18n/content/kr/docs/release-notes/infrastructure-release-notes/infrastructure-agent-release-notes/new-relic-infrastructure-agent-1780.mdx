에이전트의 새 버전이 출시되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent)하려면 표준 절차를 따르십시오. 뉴렐릭은 에이전트를 정기적으로, 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스 기준으로 지원되는 가장 오래된 버전은 [인프라 에이전트 1.65.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1653/)입니다.
