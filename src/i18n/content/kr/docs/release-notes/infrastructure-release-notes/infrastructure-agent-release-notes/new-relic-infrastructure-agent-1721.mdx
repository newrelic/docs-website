에이전트의 새로운 버전이 출시되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent) 하려면 표준 절차를 따르세요. 뉴렐릭은 에이전트를 정기적으로, 그리고 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스를 기준으로 지원되는 가장 오래된 버전은 [인프라 에이전트 1.58.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1581/) 입니다.

## 추가됨

* [#2167](https://github.com/newrelic/infrastructure-agent/pull/2167)의 블루렐릭 fluent-bit 출력에 대한 HTTP 클라이언트 시간 초과(초)를 제어하는 `logging_http_client_timeout` 설정 옵션을 추가했습니다.
* [#2171](https://github.com/newrelic/infrastructure-agent/pull/2171)에서 RHEL10용 업로드 스키마를 추가했습니다.

## 변경됨

* [#2173](https://github.com/newrelic/infrastructure-agent/pull/2173)에서 `win_network_interface_v2` 플래그를 기본적으로 true로 설정합니다.
* [#2172](https://github.com/newrelic/infrastructure-agent/pull/2172)에서 pkg-tests의 최소 버전이 업데이트되었습니다.

## 보안

* 심각한 취약점(CVE-2025-15467) 및 높은 수준의 취약점(CVE-2025-69419)을 해결했습니다.
