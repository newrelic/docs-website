에이전트의 새 버전이 출시되었습니다. [인프라 에이전트를 업데이트](/docs/infrastructure/install-configure-manage-infrastructure/update-or-uninstall/update-infrastructure-agent)하려면 표준 절차를 따르십시오. 뉴렐릭은 에이전트를 정기적으로, 최소 3개월마다 업그레이드할 것을 권장합니다. 이번 릴리스부터 지원되는 가장 오래된 버전은 [인프라 에이전트 1.65.1](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1651/)입니다.

## 변경됨

* [#2264](https://github.com/newrelic/infrastructure-agent/pull/2264)에서 newrelic/nri-prometheus 종속성을 v2.29.3으로 업데이트했습니다.
* [#2224](https://github.com/newrelic/infrastructure-agent/pull/2224)에서 Ubuntu 26.04 LTS(Resolute Raccoon)에 대한 지원을 추가했습니다.
* [#2266](https://github.com/newrelic/infrastructure-agent/pull/2266)에서 `newrelic/nri-flex` 종속성을 v1.18.6으로 업데이트했습니다.
