### 요청 ID 주입을 통한 향상된 AWS Lambda 로그 처리

뉴렐릭은 기본 Go 런타임을 버전 1.24.4로 업데이트하여 Lambda 함수의 보안을 강화합니다. 최신 보안 패치가 포함되어 있습니다.

### 변경됨

* **Go 버전 업그레이드:** 뉴렐릭은 시스템의 Go 버전을 1.24.4로 업그레이드하여 보안 태세를 강화했습니다.
