### 주요 변경 사항

* 새로운 설정 옵션, 네트워크 요구 사항, 문제 해결, 해결 지침을 포함하여 포괄적인 Windows 호스트 지원을 포함하여 [뉴렐릭 Control 에이전트 제어 문서가](/docs/new-relic-control/agent-control/overview) 업데이트되었습니다.
* 브라우저 모델링 데이터에 사용자 지정 속성을 추가하는 방법에 대한 자세한 지침이 포함된 향상된 [PHP 브라우저 모델링 문서입니다](/docs/apm/agents/php-agent/features/browser-monitoring-php-agent) .
* 하이브리드 저항 샘플러 설정 예제가 포함된 [저항력 설정 문서가](/docs/apm/agents/python-agent/configuration/python-agent-configuration) 업데이트되었습니다.

### 사소한 변경 사항

* [AWS CloudWatch 통합 문서가](/install/aws-cloudwatch) 업데이트되어 다중 리전 모니터링을 위해 각 AWS 계정에 리전별로 하나의 싱크가 필요하다는 점이 명확해졌습니다.
* 뉴렐릭 앱 이름 설정 정보로 [eBPF Kubernetes 설치](/docs/ebpf/k8s-installation) 및 [Linux 설치](/docs/ebpf/linux-installation) 문서가 업데이트되었습니다.
* Kubernetes 및 자체 호스팅 환경 모두에서 NRDOT Collector를 지원하도록 [RabbitMQ OpenTelemetry 통합이](/docs/opentelemetry/integrations/rabbitmq/overview) 업데이트되었습니다.
* [고보안 모드 관련 문서가](/docs/apm/agents/manage-apm-agents/configuration/high-security-mode) 업데이트되었습니다.
* [포트 모니터링 통합 관련 문서가](/docs/infrastructure/host-integrations/host-integrations-list/port-monitoring-integration) 업데이트되었습니다.
* [팀 문서가](/docs/service-architecture-intelligence/teams/teams) 업데이트되었습니다.
* [팀 관리 문서가](/docs/service-architecture-intelligence/teams/manage-teams) 업데이트되었습니다.
* 업데이트된 [배터리 대응 문서](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent).
* [Node.js 에이전트 호환성 문서가](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 업데이트되었습니다.

### 새로워진 사항

* 뉴렐릭 컨트롤(공개 미리보기)의 Windows 호스트 지원 및 향상된 뷰어 선택 기능을 알리기 위해 뉴렐릭 [컨트롤 Windows 지원에 대한 새로운 기능이](/whats-new/2026/02/whats-new-23-02-windows-support/) 추가되었습니다.
* [Collector 옵저버빌리티의 새로운 기능이](/whats-new/2026/02/whats-new-02-24-collector-observability/) 업데이트되었습니다.
