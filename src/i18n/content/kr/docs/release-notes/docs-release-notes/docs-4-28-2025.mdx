### 주요 변경 사항

* 이벤트에 개별 에 대한 고유한 식별자 역할을 [하는 트레이스.id](/attribute-dictionary/?event=AjaxRequest&attribute=trace.id) 속성을 추가하여 프로세스 `AjaxRequest` requests 간 및 프로세스 내 경계에서 요청을 차별화합니다.

* 뉴렐릭 설명서에서 AmazonQ 통합 섹션을 제거했습니다.

* [PREDICT 절을](/docs/nrql/nrql-syntax-clauses-functions/##sel-prediction) 기본 동작으로 업데이트했습니다.

* 비디오 에이전트 및 모든 해당 문서를 뉴렐릭 문서에서 제거했습니다.

* [공지 공지 채널에 대한 NerdGraph 튜토리얼을](/docs/apis/nerdgraph/examples/nerdgraph-api-alert-notification-channels/) 더 이상 사용되지 않는 것으로 표시했습니다.

* Control Group v1을 활용한 운영 시스템에서 종단 간(E2E) [테스트 범위](/docs/infrastructure/infrastructure-agent/linux-installation/instrument-container/#test-coverage) 에 대한 지원이 중단되었습니다.

* [공지 섹션](/docs/alerts/alerts-ai-overview-page/) 에 새로운 조사 상태가 추가되었습니다.

### 사소한 변경 사항

* [.Net 에이전트 설정을](/docs/apm/agents/net-agent/configuration/net-agent-configuration/) 업데이트했습니다.

* 설정을 통해 &apos;환경 변수&apos;를 포함하도록 [Python 에이전트 구성 파일을](/docs/apm/agents/python-agent/configuration/python-agent-configuration/) 업데이트했습니다.

* [에이전트 다운로드](/docs/new-relic-solutions/get-started/networks/#agent-download) 섹션을 업데이트하여 뉴렐릭의 변경된 IP 설정을 수용했습니다.

* [Amazon DynamoDB](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/#databases) 에 대한 PHP 에이전트 호환성 및 요구 사항이 업데이트되었습니다.

* [인프라 에이전트](/docs/infrastructure/infrastructure-agent/requirements-infrastructure-agent/) 에 지원되는 OS로 Windows Server 2025를 추가했습니다.

* [Node.js 에이전트 호환성 보고서가](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 업데이트되었습니다.
