### 새로운 문서

* 현재 정식 버전으로 제공되며 안드로이드 및 iOS용 설정, 개인정보 보호 설정, 고급 기능, 리플레이 보기, 문제 진단, 해결을 다루는 포괄적인 가이드가 포함된 [모바일 세션 리플레이](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/introduction/) 문서 모음이 추가되었습니다.
* 스크립트된 API 모니터를 Node.js 16에서 Node.js 22로 이동하기 위한 단계별 마이그레이션 지침을 제공하기 위해 [스크립트된 API 모니터를 Node.js 22로 업그레이드](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-api-monitors-to-node-22/) 를 추가했습니다.

### 주요 변경 사항

* 새로운 [CCI 가격 책정](/docs/cci/cci-pricing/), [클라우드 비용 인텔리전스용 공지 조건](/docs/cci/configure-cci/cci-alerts/), [AI 비용](/docs/cci/financial-intelligence/ai-costs/) 페이지와 확장된 통합 설정 및 FAQ 콘텐츠를 포함하여 일반 공급을 위한 클라우드 비용 인텔리전스 문서가 업데이트되었습니다.
* 더 나은 탐색과 명확성을 위해 새로운 [엔티티 소유권 구성](/docs/service-architecture-intelligence/teams/team-ownership/) 가이드와 [업데이트된 Teams 관리 콘텐츠로](/docs/service-architecture-intelligence/teams/teams/) [뉴렐릭 Teams 문서를 재구성했습니다.](/docs/service-architecture-intelligence/teams/manage-teams/)
* v1.4.0에 도입된 필드 변경 사항 및 YAML 설정 마이그레이션 지침이 포함된 [Kubernetes](/docs/ebpf/k8s-installation/) 및 [Linux](/docs/ebpf/linux-installation/) 설치에 대한 업데이트된 eBPF 에이전트 문서입니다.
* 확장된 대시보드 안내, 새로운 스크린샷 및 개선된 최적화 권장 사항으로 [컴퓨트 사용량 최적화](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-optimizer/) 를 업데이트했습니다.
* OpenTelemetry APM 환경의 오류 모니터링을 다루는 새로운 섹션으로 [OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/) 를 업데이트했습니다.

### 사소한 변경 사항

* 최신 자동 생성 호환성 데이터로 [Node.js 에이전트의 호환성 및 요구 사항](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/) 을 업데이트했습니다.
* [AWS Lambda용 환경 변수](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/env-variables-lambda/)에 새 환경 변수 `NEW_RELIC_EXTENSION_SEND_PLATFORM_LOGS`, `NEW_RELIC_EXTENSION_SEND_LOGS` 및 `NEW_RELIC_LAMBDA_EXTENSION_PROXY` 를 추가했습니다.
* 두 가지 별개의 서버리스 계측 접근 방식을 더 명확하게 분리하여 자바 Lambda에 대한 [계층화된 계측](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/instrument-your-own/) 을 업데이트했습니다.
* [OpenTelemetry APM UI](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-ui/)에서 잘못 정렬된 테이블 헤더를 수정했습니다.
* [뉴렐릭에 Google Cloud Platform 서비스 연결](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/)에 `iam.managed.allowedPolicyMembers` IAM 권한 지침을 추가했습니다.
* [뉴렐릭 AI 지식 커넥터](/docs/agentic-ai/knowledge-integration/overview/)에서 모호한 계정 참조를 수정했습니다.
* 콘텐츠를 재구성하고 명확성을 개선하여 [신세틱스 프라이빗 로케이션 런타임 전환 가이드](/docs/synthetics/synthetic-monitoring/private-locations/runtime-transition-guide/) 를 업데이트했습니다.
* [자바 에이전트 9.2.0 릴리스 노트](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-920/)의 끊어진 링크를 수정했습니다.
* [NerdGraph 튜토리얼: AWS Kinesis Firehose, Azure 이벤트 허브 또는 GCP Pub/Sub로 데이터 스트리밍](/docs/apis/nerdgraph/examples/nerdgraph-streaming-export/)에서 잘못된 AWS 계정 ID 예시를 수정했습니다.
* [뉴렐릭 데이터 프라이버시](/docs/security/security-privacy/data-privacy/data-privacy-new-relic/)에 화면 공유 및 녹화 동의 문구를 추가했습니다.
* [NRQL 참조](/docs/nrql/nrql-syntax-clauses-functions/)에서 NRQL 절의 대문자 표기를 수정했습니다.
* 뉴렐릭 개발자 문서 페이지에서 더 이상 사용되지 않는 one.nr URL을 onenr.io로 변환했습니다.
