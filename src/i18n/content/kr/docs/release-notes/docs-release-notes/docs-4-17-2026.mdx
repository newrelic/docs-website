### 새로운 문서

* OpenTelemetry를 사용한 Amazon ECS 모니터링에 대한 포괄적인 지침을 제공하기 위해 [OpenTelemetry ECS 모니터링](/docs/opentelemetry/integrations/ecs-monitoring/overview/) 을 추가했습니다.

### 주요 변경 사항

* 설치, 모니터링, 설정 및 문제 진단, 해결을 위한 전용 섹션을 포함하여 향상된 조직을 위해 재구성된 [SAP 솔루션 문서입니다](/docs/sap-solutions/overview/).
* GA 출시 및 향상된 설정 지침으로 [Kubernetes Windows 노드 모니터링](/docs/kubernetes-pixie/kubernetes-integration/installation/windows/) 이 업데이트되었습니다.
* 데이터 마스킹 및 기록 제어에 대한 광범위한 설정 옵션이 포함된 업데이트된 [브라우저 세션 리플레이 개인정보 보호 설정](/docs/browser/browser-monitoring/browser-pro-features/session-replay/configuration/customize-privacy-settings/) 입니다.
* 읽기 전용 사용자가 더 이상 대시보드를 생성, 업데이트 또는 삭제할 수 없다는 점을 반영하도록 [대시보드 관리](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/) 및 [사용자 관리 개념](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/) 문서를 업데이트했습니다.

### 사소한 변경 사항

* 최신 검증된 패키지 버전으로 [.NET 에이전트 호환성 요구 사항](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) 을 업데이트하고 컨테이너 지원을 명확히 했습니다.
* [PHP 에이전트 호환성 요구 사항](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements/) 을 현재 지원되는 버전으로 업데이트하고 EOL 버전을 제거했습니다.
* 최신 설정 옵션으로 업데이트된 [루비 에이전트 설정](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration/).
* 수정된 액세스 권한 부여 명령으로 개선된 [Oracle 데이터베이스 통합](/docs/infrastructure/host-integrations/host-integrations-list/oracle-database-monitoring-integration/).
* [ATP 문제 진단, 해결](/docs/opentelemetry/nrdot/atp/troubleshooting/)을 위한 지능형 스토리지 경로 선택 문서를 추가했습니다.
* [OpenTelemetry를 사용한 Jenkins 모니터링](/docs/infrastructure/other-infrastructure-integrations/monitoring-jenkins-ot/) 문서의 들여쓰기를 수정했습니다.
* [Node.js 에이전트 호환성 요구 사항](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/) 을 최신 호환성 매트릭스로 업데이트했습니다.

### 새로운 게시물은 무엇인가요?

* Windows 노드 모니터링의 정식 출시를 알리기 위해 [Kubernetes Windows 노드 GA](/whats-new/2026/04/whats-new-4-16-windows-ga/) 를 추가했습니다.
