### 새로운 문서

* 함대 전반에 걸쳐 대규모로 군사 에이전트 설정을 관리하기 위한 [함대 제어 API](/docs/new-relic-control/fleet-control/overview/) 추가되었습니다.
* Synthetic 모니터를 Chrome 최신 버전 및 Node.js 22로 업그레이드하는 방법에 대한 [Chrome 가이드가](/docs/synthetics/synthetic-monitoring/scripting-monitors/synthetics-scripted-browser-monitors-to-chrome-latest/) 추가되었습니다.
* 신세틱 모델에 영향을 미치는 중요한 Google Chrome use-after-free 튜토리얼(CVE-2026-5281)을 해결하기 위한 절차가 포함된 [보안 공지 NR26-04를](/docs/security/new-relic-security/security-bulletins/security-bulletin-nr26-04) 추가했습니다.
* 워크플로우 자동화를 통해 Agentic 플랫폼에 구성된 에이전트를 실행하기 위한 [뉴렐릭 에이전트 액션을](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent) 추가했습니다.

### 주요 변경 사항

* [AWS Lambda 및 OpenTelemetry 문서가](/docs/serverless-function-monitoring/aws-lambda-monitoring/opentelemetry/lambda-opentelemetry-troubleshooting/) 업데이트되었으며, CloudWatch 대기 문제 및 요청 ID 문제를 다루는 OpenTelemetry Lambda 문제 진단 및 해결 섹션이 추가되었습니다.

### 사소한 변경 사항

* [브라우저 에이전트 릴리스 노트가](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/) 업데이트되었으며, v1.312.0 및 v1.312.1 버전과의 호환성 정보가 포함되어 있습니다.
* 뉴렐릭 플랫폼의 모든 엔티티에 변경 사항을 연결할 수 있다는 점을 명확히 하여 [변경 추적 개요를](/docs/change-tracking/overview/) 업데이트했습니다.
* 업데이트된 패키지 버전을 반영하여 [Node.js 에이전트 호환성 보고서를](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/) 업데이트했습니다.
* `Microsoft.Azure.Cosmos`, `Microsoft.Data.SqlClient`, `MongoDB.Driver` 및 기타 여러 패키지에 대해 최신 검증된 호환 버전으로 [.NET 에이전트 호환성 요구 사항이](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) 업데이트되었습니다.
* 누락된 분류 오류를 수정하여 [.NET 에이전트 릴리스 노트를](/docs/release-notes/agent-release-notes/net-release-notes/) 업데이트했습니다.
* 문서 섹션 전반에 걸쳐 사용자 경험을 개선하기 위해 탐색 구조가 업데이트되었습니다.
* `NRConditionalMaskView` 에 대한 블록 보기 기능과 포괄적인 마스킹 대 블로킹 지침이 포함된 업데이트된 [모바일 세션 리플레이 문서입니다](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-session-replay/).
* 조직 범위 보존 관리 및 계정과 조직 탭 설명을 포함하여 [데이터 관리 문서를](/docs/data-apis/manage-data/manage-data-retention/) 업데이트했습니다.
* 이제 에이전트에 포함되어 있으므로 더 이상 사용되지 않는 HikariCP 2.4.0 인큐베이터 모듈을 제거하여 [이온 에이전트 확장 모듈 문서를](/docs/apm/agents/java-agent/instrumentation/extension-additional-instrumentation-modules/) 업데이트했습니다.
* [지식 통합 문서가](/docs/agentic-ai/knowledge-integration/overview/) 업데이트되었으며, 권한 이름이 &quot;조직 제품 관리자&quot;에서 &quot;조직 관리자&quot;로 변경되었습니다.
* 새로운 `NrComputeUsage` 이벤트 유형과 관련 속성 정의를 포함하여 속성 사전이 업데이트되었습니다.
* 데이터 사용량 예측 기능과 고급 컴퓨트 고객을 위한 사용량 예측 기능을 포함하여 [데이터 수집 관리 문서가](/docs/data-apis/manage-data/manage-data-coming-new-relic/) 업데이트되었습니다.
* 작업Kubernetes Engine(OKE) 지원 및 APM 자동 연결을 위한 설정 지침으로 [Kubernetes 통합 문서가](/docs/kubernetes-pixie/kubernetes-integration/get-started/kubernetes-integration-compatibility-requirements/) 업데이트되었습니다.
* Solr v9 지원 및 보안 관리자 설정 지침을 포함하여 [플레이 스테이션 2 보안 문서가](/docs/apm/agents/java-agent/additional-installation/install-java-agent-java-2-security/) 업데이트되었습니다.
* 직책 정보를 입력하기 위한 `title` 필드를 지원하도록 [자동 사용자 관리를 위한 SCIM 지원이](/docs/accounts/accounts/automated-user-management/scim-support-automated-user-management/) 업데이트되었습니다.

### 새로운 게시물은 무엇인가요?

* [새로운 Dashboards 기능을](/whats-new/2026/03/whats-new-3-28-in-dashboards) 추가하여 마크다운 위젯 및 제목에서 위젯 링크, 위젯 설명, 템플릿 변수 표시 제어, 동적 템플릿 변수를 알릴 수 있습니다.

### 릴리즈 정보

* 최신 릴리스에 대한 최신 정보를 받아보세요.

  * [Node.js 에이전트 v13.15.0, v13.16.0, v13.17.0, v13.18.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/):

    * 비동기 컨텍스트를 스트림 이벤트 핸들러에 바인딩하는 grpc 서버 스트림이 생성되었습니다.
    * 구독자 기반 이벤트 처리를 사용하도록 여러 측정, 로그 모듈을 리팩터링했습니다.
    * LangChain 코어 구독자를 업데이트하여 도구 클래스 메서드를 올바르게 사용하도록 했습니다.
    * 최신 npm 패키지 및 프레임워크와의 호환성이 향상되었습니다.

  * [Go 에이전트 v3.43.0](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-3-43-0):

    * UnWrap 대체 응답 작성기 및 Lambda 함수 URL 지원이 추가되었습니다.
    * Opensearch/Elasticsearch 통합 기능이 향상되었으며, 추가 스팬 속성이 제공됩니다.
    * SQL 구문 분석에서 `WITH` 절에 대한 지원을 추가하고 HTTP 테스트 문제를 수정했습니다.
    * `nrgrpc` 버전 범프 및 의존성/종속성 보안 패치를 포함한 보안 업데이트입니다.

  * [PHP 에이전트 v12.6.0.34](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-12-6-0-34):

    * 측정, 로그 제어를 위한 라라벨 13 지원 및 PHP Fiber 감지가 추가되었습니다.
    * Go 버전 1.26.1로 보안을 강화하고 RPM 서명 키를 갱신했습니다.
    * 데비안/우분투 설치 문제를 해결하고 SQL 구문 분석 로직을 개선했습니다.

  * [인프라 에이전트 v1.72.7, v1.72.8, v1.72.9, v1.73.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/):

    * Windows에서 최소 권한 서비스 계정에 대한 포괄적인 지원이 추가되었습니다.
    * 업그레이드 중에도 기존 서비스 계정을 유지할 수 있도록 인스톨러 기능이 향상되었습니다.
    * 그룹 관리 서비스 계정(gMSA) 지원을 통해 보안이 강화되었습니다.
    * 디버그 로깅 및 설치 롤백 안전 기능을 추가했습니다.

  * [뉴렐릭 안드로이드 모바일 앱 v5.41.5](/docs/release-notes/mobile-apps-release-notes/new-relic-android-release-notes/new-relic-android-5415):

    * 버그 수정 및 개선 사항.

  * [Lambda 확장 기능 v2.4.6](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.6):

    * 프록시 설정 및 로그 제어 환경 변수를 추가했습니다.
    * JSON 로그 분류 오류를 수정하고 요청 ID 처리 방식을 개선했습니다.
    * `TraceIDGenerator` 에서 향상된 난수 생성 기능.

  * [NRDOT Collector v1.12.0](/docs/release-notes/nrdot-release-notes/nrdot-2026-04-01):

    * OpenTelemetry 베타 코어를 성능 향상을 위해 v0.148.0으로 업데이트했습니다.
    * spanmetricsconnector를 실험적 기능에 추가했습니다.
    * CVE-2026-33186 및 CVE-2026-34040을 포함한 보안 취약점을 수정했습니다.
    * 보안 강화를 위해 공개 GPG 서명 키를 주기적으로 변경합니다.
