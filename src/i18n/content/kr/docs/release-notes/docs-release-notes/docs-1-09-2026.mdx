### 주요 변경 사항

* 일반적인 Go 패턴에 대한 포괄적인 단계별 설치 가이드와 예제를 포함하여 [Go용 뉴롤릭 설치가](/docs/apm/agents/go-agent/installation/install-new-relic-go) 업데이트되었습니다.
* 쿠버네티스 클러스터 및 Linux 호스트에 대한 완전한 설치 및 설정 지침을 제공하기 위해 [설정 및 설치 에이전트 제어가](/docs/new-relic-control/agent-control/setup) 추가되었습니다.
* [setUserId](/docs/browser/new-relic-browser/browser-apis/setuserid) API 문서가 개선되었으며, 세션 재설정 기능과 업데이트된 세션 추적 동작이 포함되었습니다.

### 사소한 변경 사항

* PHP 에이전트 성능 [요구 사항에 대한 PHP 에이전트](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements) 성능 정보를 추가했습니다.
* [SPM 설치](/docs/mobile-monitoring/new-relic-mobile-ios/installation/spm-installation) 설명서가 업데이트되었으며 설치 지침이 강화되었습니다.
* 개선된 지침을 포함하여 [CocoaPods 설치](/docs/mobile-monitoring/new-relic-mobile-ios/installation/cocoapods-installation) 설명서가 업데이트되었습니다.
* 차단 목록에 와일드카드 지원을 추가하여 [필터 AJAX 요청 이벤트](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events) 문서를 업데이트했습니다.
* Lambda 함수 모니터링을 위한 [APM 환경이 업데이트](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/upgrade-to-apm-experience) 되었습니다.
