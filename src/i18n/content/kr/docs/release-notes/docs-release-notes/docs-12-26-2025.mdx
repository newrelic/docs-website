### 새로운 문서

* [GitHub Enterprise 통합 관련](/docs/service-architecture-intelligence/github-integrations/github-enterprise-integration/) 문서를 추가했습니다.

### 주요 변경 사항

* ScalingMode 및 일괄 처리를 지원하는 [Azure 로그 포워딩을](/docs/logs/forward-logs/azure-log-forwarding/) 업데이트했습니다.
* 모델 컨텍스트 프로토콜(MCP) 통합을 위한 개정된 IAM 권한 요구 사항을 포함하여 [MCP 개요가](/docs/agentic-ai/mcp/overview/) 업데이트되었습니다.
* 향상된 권한 및 부여 요구 사항 지침을 통해 [MCP 문제 해결을](/docs/agentic-ai/mcp/troubleshoot/) 업데이트했습니다.
* [사용자 권한](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/) 문서가 업데이트되었으며, 엔티티 범위 권한이 강화되었습니다.
* Fluent Bit 플러그인 취약점을 다루는 NR25-02 보안 공지를 포함하여 [보안 공지가](/docs/security/security-privacy/information-security/security-bulletins/) 업데이트되었습니다.

### 사소한 변경 사항

* 모바일 모니터링 문제 해결, 해결을 위한 [ANR(응답 없음)](/docs/mobile-monitoring/mobile-monitoring-ui/application-not-responding/) 문서를 추가했습니다.
