### 새로운 문서

* OpenTelemetry Prometheus 수신기를 사용하여 nginx를 모니터링하기 위한 포괄적인 지침을 제공하기 위해 [호스트 환경](/docs/opentelemetry/integrations/nginx/nginx-otel-host) 및 [Kubernetes](/docs/opentelemetry/integrations/nginx/nginx-otel-kubernetes) 용 Prometheus 수신기를 사용한 nginx 모니터링을 추가했습니다.

### 주요 변경 사항

* 최신 지원 프레임워크 버전으로 [.NET 에이전트 호환성 요구 사항](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements) 을 업데이트했습니다.

### 사소한 변경 사항

* 명확한 고객 자격 및 요금 모델 정보로 다음 문서를 개선했습니다:

  * [예측 알림](/docs/alerts/create-alert/set-thresholds/predictive-alerts)
  * [공개 대시보드](/docs/query-your-data/explore-query-data/dashboards/share-dashboards-publicly)
  * [NRQL 예측 문서](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions)

* 최적화된 OpenTelemetry 설정 예제로 [ECS EC2 모니터링 설정](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) 을 업데이트했습니다.

* 개선된 설정 값으로 [ECS Fargate 모니터링 설정](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) 을 업데이트했습니다.

* csec 엔드포인트에 대해 새로 고침된 허용 목록 IP로 [IAST 문제 진단, 해결 문서](/docs/iast/troubleshooting) 를 업데이트했습니다.

* 뉴렐릭 AI 지식 통합 섹션으로 [SRE 에이전트 사용 문서](/docs/agentic-ai/sre-agent/use-sre-agent) 를 개선했습니다.

* 개선된 설정 예제로 [브라우저 세션 리플레이 개인정보 보호 설정](/docs/browser/browser-monitoring/browser-pro-features/session-replay/configuration/customize-privacy-settings) 을 업데이트했습니다.

### 수명 종료
