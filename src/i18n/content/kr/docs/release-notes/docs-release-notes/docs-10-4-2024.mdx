### 새로운 문서

* [카디널리티 관리는](/docs/data-apis/ingest-apis/metric-api/cardinality-management) 조직의 카디널리티 사용량을 관리하고 검토할 수 있는 관리 도구를 제공합니다.

### 주요 변경 사항

* 새로운 스크린샷과 다양한 구현 방법을 설명하는 표를 통해 [영향을 받은 사용자를 추적하는](/docs/errors-inbox/error-users-impacted/) 지침이 개선되었습니다.
* [Kubernetes에 대한 OpenTelemetry링크가](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-otel-applications-kubernetes) 업데이트되었으며 현재 OpenTelemetry 설정 및 설정 정보로 [쿠버네티스 클러스터를 살펴보세요](/docs/kubernetes-pixie/kubernetes-integration/understand-use-data/kubernetes-cluster-explorer).
* [알림 억제 규칙을](/docs/alerts/get-notified/muting-rules-suppress-notifications) 다시 작성하여 알림 이벤트 및 관련 문제를 생성, 관리, 억제 및 검토하는 방법을 설명합니다.
* 향상된 탐색 및 검색 가능성을 위해 [Kubernetes 문제 해결, 해결](/docs/kubernetes-pixie/kubernetes-integration/troubleshooting/overview) 문서를 재구성했습니다.
* 명확성과 완성도를 높이기 위해 [OpenTelemetry](/docs/opentelemetry/opentelemetry-introduction) 문서를 개정했습니다.
* [PHP 에이전트 설정](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-vulnerability-management) 에서 데이터베이스 관리 패키지 감지를 위해 Composer 런타임 API 구성하는 지침이 추가되었습니다.

### 사소한 변경 사항

* 링크, 내보낸 이미지 및 PDF를 통해 공유할 때 빌보드 차트가 올바르게 표시되지 않을 수 있음을 알리기 위해 몇 가지 문서를 업데이트했습니다.
* Linux 명령어와 Windows 셸 명령어가 혼용되어 사용되던 몇몇 문서를 수정했습니다.
* 일부 오래된 문서를 삭제한 후, 최신 버전으로 리디렉션을 추가했습니다.
* [공지 조건을](/docs/alerts/create-alert/create-alert-condition/alert-conditions) 최신 UI 에 맞춰 업데이트했습니다.
* 데이터 사전에 새로운 [`PageViewTiming`](/attribute-dictionary/?event=PageViewTiming) 속성을 추가했습니다.
* Meraki 폴링 API 및 KTranslate에서 ICMP를 사용하기 위한 기본 설정과 관련된 네트워크 성능 모니터링 문서를 업데이트했습니다.
* [.NET 구성 설정](/docs/apm/agents/net-agent/configuration/net-agent-configuration) 에 `NewRelic.Labels` 추가했습니다.
* [오류 인박스(errors inbox)의 영향을 받은 사용자 추적](/docs/errors-inbox/error-users-impacted/#set-users) 에 PHP `newrelic_set_user_id` API 호출을 추가했습니다.
* [맞춤형 대시보드 데이터에 대한 데이터 요구 사항 및 제한 사항](/docs/data-apis/custom-data/custom-events/data-requirements-limits-custom-event-data) 에 새로운 금지된 이벤트 유형 값을 추가했습니다: `EntityRelationship`, `RelationshipUpdate` 및 `Relationship[0-9]` 문자열 접두사.
* [뉴렐릭과 테라폼 시작하기](/docs/infrastructure-as-code/terraform/terraform-intro) 에서 테라폼과 함께 CodeStream 사용하는 방법을 설명했습니다.
