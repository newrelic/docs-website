### 주요 변경 사항 [#major-changes]

* [Microsoft SQL Server 통합](/install/microsoft-sql) 설명서를 업데이트하여 다음 인증 방법에 대한 쿼리 성능 모니터링(QPM) 기능을 추가했습니다.

  * Windows 프로세서
  * Windows 그룹 관리 서비스 계정(gMSA)

* [Elasticsearch 모니터링과 OpenTelemetry](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) 관련 문서를 명확성을 위해 업데이트하고, Kubernetes 설치에 대한 포괄적인 문서를 추가했습니다.

* [NerdGraph API사용하여 템플릿 변수 가시성 기능을 포함한 업데이트된Dashboards ](/docs/apis/nerdgraph/examples/nerdgraph-dashboards)입니다.

* 광범위한 스키마 업데이트와 포괄적인 예제를 포함한 최신 [자동화 정의 스키마입니다](/docs/workflow-automation/workflow-automation-apis/definition-schema).

### 사소한 변경 사항 [#minor-changes]

* CCU 사용량 예측 기능과 향상된 대시보드 기능을 통해 [소비 관리 기능이](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/consumption-management) 업데이트되었습니다.
* [View ping 모니터 결과가](/docs/synthetics/synthetic-monitoring/using-monitors/view-ping-monitor-results) 업데이트되었으며, 개선된 안내 및 예시가 포함되어 있습니다.
* 브라우저에 대한 추가 속성 정의 및 설명을 포함하여 속성 사전을 업데이트했습니다.
