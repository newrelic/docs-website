### 새로운 문서

* 감지된 애플리케이션 취약점에 대해 Jira 지원 티켓을 자동으로 생성하고 동기화하는 방법에 대한 지침을 제공하기 위해 [시큐리티 RX Jira 워크플로우](/docs/vulnerability-management/workflows/jira) 를 추가했습니다.
* 자동화된 워크플로우의 일부로 Azure VM의 전원을 끌 수 있도록 액션 카탈로그에 [Azure 가상 머신 중지 액션](/docs/workflow-automation/setup-and-configure/actions-catalog/azure/azure-vm-stop) 을 추가했습니다.
* 뉴렐릭이 세션 전반에 걸쳐 비디오 개요 및 광고 개요 대시보드 설정을 저장하고 복원하는 방법을 설명하기 위해 [대시보드 기본 설정 저장 및 관리](/docs/streaming-video-&-ads/view-data-in-newrelic/user-preferences) 에 대한 세부 정보를 추가했습니다.
* 전송 전에 정규식 기반 룰을 사용하여 민감한 데이터를 마스킹하는 방법에 대한 지침을 제공하기 위해 [안드로이드](/docs/streaming-video-&-ads/installation/android/obfuscation-rules), [브라우저](/docs/streaming-video-&-ads/installation/browser/obfuscation-rules), [iOS](/docs/streaming-video-&-ads/installation/ios/obfuscation-rules) 및 [Roku](/docs/streaming-video-&-ads/installation/roku/obfuscation-rules) 용 스트리밍 비디오 및 광고 에이전트와 관련된 난독화 룰을 추가했습니다.

### 주요 변경 사항

* 시큐리티 RX .NET 통합의 기본 버전을 &apos;10.51.1&apos;로 설정하도록 [시큐리티 RX 통합 개요](/docs/vulnerability-management/getting-started/integrations/overview) 및 [애플리케이션 개요](/docs/vulnerability-management/applications/overview) 를 업데이트했습니다.
* [제공자별 청구 수집 가격 책정](/docs/cci/cci-pricing), [상위 계정 대 하위 계정 통합 범위 가이드](/docs/cci/overview/#choose-an-integration-scope) 및 [Azure 설정 전제 조건](/docs/cci/get-started/azure-cci)으로 CCI 문서를 업데이트했습니다.

### 사소한 변경 사항

* Apollo 및 Next.js 애플리케이션을 계측하기 위한 지침으로 [Node.js 에이전트 설치 가이드](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent) 를 업데이트했습니다.
* 최신 호환성 보고서로 [Node.js 에이전트 호환성 요구 사항](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 을 업데이트했습니다.
* 최신 설정 옵션으로 [루비 에이전트 설정](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration) 문서를 업데이트했습니다.
* 실행 중인 작업 수에 대해 수정된 설정으로 [ECS EC2](/docs/opentelemetry/integrations/ecs-monitoring/ecs-ec2) 및 [ECS Fargate](/docs/opentelemetry/integrations/ecs-monitoring/ecs-fargate) 모니터링 문서를 업데이트했습니다.
* 컨텍스트 설명을 추가하여 액션 카탈로그의 [뉴렐릭 에이전트 액션](/docs/workflow-automation/setup-and-configure/actions-catalog/newrelic/newrelic-agent) 을 개선했습니다.
