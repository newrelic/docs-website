### 새로운 문서

* [데이터 수집 예산 항목을](/docs/data-apis/manage-data/data-ingest-budgets) 추가하여 예산을 활용한 데이터 수집 비용 관리 방법에 대한 포괄적인 지침을 제공합니다.
* 여러 쿠버네티스 클러스터에 대한 포괄적인 지침을 제공하기 위해 [Kubernetes Cross-클러스터 옵저버빌리티를](/docs/kubernetes-pixie/kubernetes-integration/understand-use-data/kubernetes-cross-cluster) 추가했습니다.

### 주요 변경 사항

* [Advanced Maps](/docs/service-architecture-intelligence/maps/advanced-maps) 설명서가 대폭 수정되어 일반 사용자에게 공개되었습니다.
* 최신 콘텐츠와 구성으로 업데이트된 [카탈로그](/docs/service-architecture-intelligence/catalogs/catalogs) 문서가 일반에 공개되었습니다.

### 사소한 변경 사항

* Rust로 재작성 및 APM 모드 지원이 포함된 [Lambda 확장 프로그램 v2.4.0](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.0) 릴리스 노트를 추가했습니다.

* @langchain/AWS를 지원하는 [Node.js 에이전트 v13.8.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-8-0) 릴리스 노트를 추가했습니다.

* OTEL 스팬 링크 지원 및 비율 기반 샘플러 설정이 포함된 [Node.js 에이전트 v13.7.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-7-0) 릴리스 노트를 추가했습니다.

* Strands 및 AWS Bedrock Converse Streaming 측정, 리소스가 포함된 [그립 에이전트 v11.2.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-110200) 릴리스 노트를 추가했습니다.

* Alpine docker 태그 및 go 버전 업데이트가 포함된 [에이전트 에이전트 v1.71.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1713) 릴리스 노트가 추가되었습니다.

* 여러 CVE에 대한 Fluent Bit 보안 업데이트가 포함된 [에이전트 에이전트 v1.71.2](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1712) 릴리스 노트가 추가되었습니다.

* [Kubernetes 통합 v3.51.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-51-0) 릴리스 노트를 추가했습니다.

* [사용 쿼리 및 알림](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts) 문서에 대한 데이터 제한 값을 10e9에서 10e8로 수정했습니다.

* [Node.js 에이전트 호환성 보고서](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 문서가 업데이트되었습니다.

* [에이전트 소프트웨어 업데이트 공지](/docs/licenses/license-information/distributed-licenses/new-relic-agent-software-notice) 문서입니다.

* 지원되지 않는 운영 체제를 제거하도록 [네트워크 syslog 모니터링](/docs/network-performance-monitoring/setup-performance-monitoring/network-syslog-monitoring) 문서를 업데이트했습니다.

* [SNMP 성능 모니터링](/docs/network-performance-monitoring/setup-performance-monitoring/snmp-performance-monitoring) 문서에 대한 OS 버전 수정 사항이 반영되었습니다.

* 향상된 [데이터 예산 관리 기능의 새로운 소식입니다](/whats-new/2025/12/whats-new-12-09-data-budgets).

### 릴리즈 정보

* 최신 릴리스에 대한 최신 정보를 받아보세요.

  * [Lambda 확장 기능 v2.4.0](/docs/release-notes/lambda-extension-release-notes/lambda-extension-2.4.0):

    * Go 언어로 작성된 코드를 Rust 언어로 완전히 재작성하여 성능과 메모리 안전성을 향상시켰습니다.
    * 뉴렐릭 APM 플랫폼에 직접 보고할 수 있도록 Lambda 에 대한 APM 모드 지원이 추가되었습니다.
    * 최적화된 Rust 컴파일을 통해 콜드 스타트 시간을 20\~40% 단축했습니다.

  * [Node.js 에이전트 v13.8.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-8-0):

    * @langchain/AWS 측정, 로그에 대한 지원이 추가되었습니다.
    * 부분적인 세분성 추적 기능을 추가했습니다.

  * [Node.js 에이전트 v13.7.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-7-0):

    * OTEL 스팬 링크 지원이 추가되었습니다.
    * 비율 기반 샘플러를 구성할 수 있는 기능이 추가되었습니다.
    * @koa/router\@15.0.0+에 대한 Koa 측정, 로그를 업데이트했습니다.
    * OTEL 시간 정규화가 수정되었습니다.

  * [접착제 에이전트 v11.2.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-110200):

    * Strands 측정, 로그에 대한 지원이 추가되었습니다.
    * AWS Bedrock Converse Streaming 지원이 추가되었습니다.
    * 레디스 메서드 hybrid\_search 및 msetex에 대한 지원이 추가되었습니다.
    * ASGI 헤더 소비 문제를 수정했습니다.

  * [인프라 에이전트 v1.71.3](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1713):

    * Alpine docker 태그를 v3.23으로 업데이트했습니다.
    * Go 버전을 1.25.5로 업데이트했습니다.

  * [인프라 에이전트 v1.71.2](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1712):

    * CVE-2025-12972, CVE-2025-12970, CVE-2025-12978, CVE-2025-12977 및 CVE-2025-12969에 대한 수정 사항이 포함된 Fluent Bit 버전이 업데이트되었습니다.
    * 32비트 Windows 빌드를 제거했습니다.
    * 영어가 아닌 시스템에서 CPU 성능 카운터의 오류를 수정했습니다.

  * [Kubernetes 통합 v3.51.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-51-0):

    * 새로운 기능과 개선 사항이 포함된 업데이트된 통합 버전입니다.
