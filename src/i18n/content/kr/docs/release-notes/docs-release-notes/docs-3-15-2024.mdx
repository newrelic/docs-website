### 주요 변경 사항

* NRQL 문서 전체의 왼쪽 탐색 메뉴를 정리하고 간소화했습니다.

* [MongoDB](/install/mongodb/), [MySQL](/install/mysql/), [PostgreSQL](/install/postgresql/) 용 온호스트 통합 문서를 검토하고 재작성하여 설치를 간소화하고 문서의 정확성과 최신성을 보장했습니다.

* [AWS 통합 메트릭](/docs/infrastructure/amazon-integrations/get-started/aws-integrations-metrics) 에 누락된 차원 메트릭을 추가했습니다.

* 문서 사이트의 다양한 구성 요소 스타일이 개선되었습니다.

  * 가독성과 제품과의 일관성을 높이기 위해 색상을 업데이트했습니다.
  * 왼쪽 탐색 메뉴에서 &quot;제품 보기&quot; 요소를 제거했습니다.
  * 저희 문서 로고에서 육각형 배경색을 제거했습니다.
  * 대화형 설치 설명서에 오른쪽 탐색 메뉴에 피드백 양식을 추가했습니다.
  * 여러 문서에서 가입 배너를 제거했습니다.
  * 팝오버 텍스트 크기를 키웠습니다.

### 소규모 업데이트

* CodeStream [로그 검색](/docs/codestream/observability/log-search) 에서 주변 로그를 표시하는 방법에 대한 정보를 추가했습니다.
* Visual Studio 2019에서 CodeStream 지원이 제거되었습니다. [CodeStream을 설치하고 가입하세요](/docs/codestream/start-here/install-codestream).
* SUSE Linux Enterprise Server(SLES) 인프라 에이전트 버전 지원이 [인프라 에이전트 요구 사항](/docs/infrastructure/install-infrastructure-agent/get-started/requirements-infrastructure-agent/) 에 업데이트되었습니다.
* 업데이트된 APM PHP 에이전트 호환성 요구 사항 [PHP 에이전트 호환성 및 요구 사항](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements).
* 삭제된 섹션 목록은 CloudWatch Metric Streams를 통해 사용할 수 없습니다. [Amazon CloudWatch Metric Streams 통합 소개](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream).
* 데이터 사전에서 [`interactionDuration`](/attribute-dictionary/?attributeSearch=interactionduration) 정의가 업데이트되었습니다.
* [에이전트 API를 사용하여 API ](/docs/apm/agents/java-agent/instrumentation/instrument-browser-monitoring-java-agent-api)에이전트 에이전트 API의 플레이스홀더 텍스트를 업데이트했습니다.
* AWS Privatelink용 베어링 에이전트 엔드포인트를 추가하고 [AWS PrivateLink를 통해 뉴렐릭 데이터 보내기](/docs/data-apis/custom-data/aws-privatelink) 에서 &quot;중요&quot; 콜아웃을 보다 관련성이 높은 위치로 이동했습니다.
* [온-호스트 통합 수동으로 실행(dry-run)](/docs/infrastructure/host-integrations/troubleshooting/run-integrations-manually) 에서 디버그 로그를 사용하여 단일 통합 파일을 테스트하는 방법을 보여주는 문제 해결, 해결 예제를 추가했습니다.
* APM 에이전트 문서 몇 개에서 버튼 유형을 변경했습니다.
* 데이터 사전 [executionDuration](/attribute-dictionary/?attributeSearch=executionduration) 에 `executionDuration` 정의를 추가했습니다.
* 지원되는 Fluent Bit 버전이 2.2.2로 업데이트되었습니다 [. 인프라 에이전트를 사용하여 로그를 전달하세요](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent).
* 뉴렐릭에서 GCP를 처음 사용하는 경우 수행해야 하는 작업에 대한 &quot;중요&quot; 안내를 추가했습니다 [Google Cloud Platform 서비스를 뉴렐릭에 연결하세요](/docs/infrastructure/google-cloud-platform-integrations/get-started/connect-google-cloud-platform-services-new-relic/).
* APM 끌어 당기는 에이전트 `event_harvest_config.harvest_limits.custom_event_data` 기본값을 `3600` 끌어 당기는 에이전트 [설정](/docs/apm/agents/python-agent/configuration/python-agent-configuration/#harvest-limits-custom-event-data) 으로 업데이트했습니다.
* 끌어당김 API 백그라운드 작업 런타임 제한 [background\_task (파이썬 에이전트 API)](/docs/apm/agents/python-agent/python-agent-api/backgroundtask-python-agent-api) 와 관련된 문제 해결, 해결 섹션을 추가했습니다.
* ServiceNow 및 [표우](/docs/alerts/get-notified/alert-event-workflows) 에서 표우와 함께 JSON 또는 숫자 출력을 사용할 수 없다는 점을 명확히 했습니다.
* [모니터 코어 웹 바이탈](/docs/tutorial-improve-site-performance/guide-to-monitoring-core-web-vitals) 에서 웹 바이탈과 관련된 콜아웃을 제거했습니다.
* [Containerized 저항 미니언(CPM) 설정](/docs/synthetics/synthetic-monitoring/private-locations/containerized-private-minion-cpm-configuration/) 의 노드 모듈 및 신세틱 모니터링 관련 언어가 업데이트되었습니다.
* [에이전트 에이전트 설정](/docs/infrastructure/install-infrastructure-agent/configuration/infrastructure-agent-configuration-settings/) 에서 SDK 동기화를 지속적으로 저장하는 데 사용되는 임시 디렉터리 및 에이전트 에이전트에 대한 섹션을 추가했습니다.
* 레벨 앱 앱으로 코드렌즈를 찾기 위한 NRQL 쿼리를 추가하고 [레벨 앱](/docs/codestream/observability/code-level-metrics) 에 새로운 &quot;Coverage&quot; 섹션을 추가했습니다.
* 명확성을 높이고 스타일 가이드와의 일관성을 개선하기 위해 여러 가지 사소한 교정 작업을 진행했습니다.
