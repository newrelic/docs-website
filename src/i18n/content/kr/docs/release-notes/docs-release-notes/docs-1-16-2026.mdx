### 새로운 문서

* 루비 에이전트 버전 10.0.0으로 업그레이드하기 위한 포괄적인 지침을 제공하기 위해 [루비 에이전트 10.x 마이그레이션 가이드를](/docs/apm/agents/ruby-agent/getting-started/migration-10x-guide) 추가했습니다.
* 자체 호스팅 구현, 배포, Strimzi가 포함된 Kubernetes, 인덱스 참조 및 데이터 쿼리를 포함하여OpenTelemetry 사용하여 Kafka 모니터링에 대한 지침을 제공하기 위해 [OpenTelemetry Kafka 통합](/docs/opentelemetry/integrations/kafka/overview) 문서를 추가했습니다.
* 호스트 기반 및 Kubernetes 구현, 배포를 포함하여OpenTelemetry 사용하여 NGINX 및 NGINX Plus를 모니터링하기 위한 지침을 제공하기 위해 [OpenTelemetry NGINX 통합](/docs/opentelemetry/nginx/nginx-otel-overview) 문서를 추가했습니다.
* 기본 플로우, 통합, 고급 사용 사례에 대한 실용적인 패턴을 제공하기 위해 [워크플로우 자동화 예제를](/docs/workflow-automation/workflow-examples/workflow-examples) 추가했습니다.

### 주요 변경 사항

* [클라우드 비용 인텔리전스(](/docs/cci/financial-intelligence/cloud-cost-insights) CCI) 문서가 업데이트되었으며, CCI 예산 및 비용 최적화 권장 사항 페이지가 추가되었습니다.
* [NerdGraph 그룹 관리 API](/docs/apis/nerdgraph/examples/nerdgraph-manage-groups) 문서가 업데이트되었으며, 다중 범위 역할 및 권한 지원 기능이 추가되었습니다.
* 다단계 예산 설정 지원으로 [컴퓨트 예산](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/compute-budgets/overview) 문서를 업데이트했습니다.
* 구성 가능한 프로세스 제외 기능을 갖춘 향상된 [.NET 에이전트 설정](/docs/apm/agents/net-agent/configuration/net-agent-configuration) 문서입니다.
* 정확도 향상을 위해 [.NET 에이전트 호환성 요구 사항이](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements) 업데이트되었습니다.
* Windows 네트워크 메트릭에 대한 새로운 `win_network_interface_v2` 옵션이 추가되어 [인프라 에이전트 구성 설정이](/docs/infrastructure/infrastructure-agent/configuration/infrastructure-agent-configuration-settings) 업데이트되었습니다.
* PHP 8.5 지원을 위한 [PHP 에이전트 호환성 요구 사항이](/docs/apm/agents/php-agent/getting-started/php-agent-compatibility-requirements) 업데이트되었습니다.
* 버전 9.0.0에 대한 [저항력 요구 사항이](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent) 업데이트되었습니다.
* 시작 아티팩트가 포함된 업데이트된 [저항 설정 파일](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file) 문서에는 설정이 포함되어 있습니다.
* 최신 지원 버전에 맞춰 [Node.js 에이전트 호환성 요구 사항이](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 업데이트되었습니다.
* [루비 에이전트 설정](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration) 문서가 업데이트되었습니다.
* [AI 모니터링 호환성 요구 사항을](/docs/ai-monitoring/compatibility-requirements-ai-monitoring) 업데이트하여 명확성을 높였습니다.
* SAML 및 SSO 설정에 대한 [인증](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more) 문서가 업데이트되었습니다.

### 사소한 변경 사항

* [PHP 에이전트 예외 수집](/docs/apm/agents/php-agent/configuration/php-agent-configuration) 관련 문서가 업데이트되었습니다.
* 추가 지침과 함께 [신세틱스 작업 관리자 설치](/docs/synthetics/synthetic-monitoring/private-locations/install-job-manager) 문서가 업데이트되었습니다.
* 네임스페이스 명확성을 위해 [데이터 보존 관련](/docs/data-apis/manage-data/manage-data-retention) 문서를 업데이트했습니다.
