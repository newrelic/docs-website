### 새로운 문서

* 뉴렐릭 OTel Collector 및 Prometheus를 사용하여 독립 실행형 및 Kubernetes 환경에 대한 설치를 다루는 포괄적인 [Elasticsearch OpenTelemetry 통합 문서](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview/) 를 추가했습니다.
* Prometheus를 사용한 자체 호스팅 Kafka 모니터링을 위한 확장된 문서와 함께 새로운 [Kubernetes 자체 관리형 환경을 위한 OTel Kafka 통합 문서](/docs/opentelemetry/integrations/kafka/kubernetes-self-managed/) 를 추가했습니다.

### 주요 변경 사항

* 카탈로그 및 맵에 더 이상 추가 역할이 필요하지 않음을 반영하여 서비스 아키텍처 인텔리전스(SAI) 권한 문서를 업데이트했습니다:

  * [애플리케이션 카탈로그](/docs/service-architecture-intelligence/catalogs/applications-catalog/) — 애플리케이션 카탈로그에 액세스하는 데 추가 역할이 필요하지 않음을 명확히 했습니다.
  * [저장소 카탈로그](/docs/service-architecture-intelligence/catalogs/repository-catalog/) — 저장소 카탈로그에 액세스하는 데 추가 역할이 필요하지 않음을 명확히 했습니다.
  * [고급 맵](/docs/service-architecture-intelligence/maps/advanced-maps/) — 업데이트된 권한 안내 및 확장된 설정 세부 정보.

* 계측 및 데이터 수집에 필요한 추가 권한을 포함하여 Lambda 모니터링에 필요한 확장된 IAM 권한 세부 정보로 [AWS Lambda 호환성 및 권한 요구 사항](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/compatibility-requirement-lambda-monitoring/) 을 업데이트했습니다.

* 분산 추적 제한 사항 및 .NET Lambda 계측 동작을 수정하기 위해 [.NET 에이전트 Lambda 및 SQS 계측 문서](/docs/apm/agents/net-agent/getting-started/net-agent-approaches-lambda/) 를 업데이트했습니다.

* 변경 추적 이벤트를 설정하는 방법을 설명하는 [튜토리얼 비디오를 NerdGraph 변경 추적 문서에](/docs/change-tracking/config/nerdgraph/) 추가했습니다.

* [AWS Lambda 계층화된 계측 문서](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/introduction-lambda/)에 지원되는 런타임으로 루비 3.4를 추가했습니다.

* 커넥터 이름 지정에 필요한 `nrlens-` 접두사를 문서화하도록 [뉴렐릭 Lens AWS 커넥터 문서](/docs/new-relic-lens/set-up-connectors/) 를 업데이트했습니다.

* 예제 도메인 이름을 규정을 준수하는 플레이스홀더 값으로 대체하도록 [사용자 관리 UI 및 작업](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/) 을 업데이트했습니다.

### 사소한 변경 사항

* Microsoft.Azure.Cosmos, EnyimMemcachedCore 및 AWSSDK.DynamoDBv2를 포함한 여러 계측된 패키지에 대해 검증된 최신 호환 버전으로 [.NET 에이전트 호환성 요구 사항](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements/) 을 업데이트했습니다.
* 지원되는 [AI 모니터링](/docs/ai-monitoring/compatibility-requirements-ai-monitoring/) 라이브러리의 전체 목록을 확인하려면 Node.js 에이전트 호환성 페이지 를 참조하도록 [AI 모니터링 호환성 요구 사항 페이지를 업데이트했습니다.](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)
* [GCP 클라우드 비용 인텔리전스 문서](/docs/cci/get-started/gcp-cci/) 를 업데이트하여 설정 후 더 쉽게 액세스할 수 있도록 일반 설정 및 통합 페이지에 대한 탐색 링크를 추가했습니다.
* FedRAMP 미승인 서비스 목록에서 뉴렐릭 eAPM을 제거하도록 [FedRAMP 규정 준수 페이지](/docs/security/security-privacy/compliance/certificates-standards-regulations/fedramp/) 를 업데이트했습니다.
* [API 폴링에서 CWMS로 마이그레이션](/docs/infrastructure/amazon-integrations/aws-integration-for-metrics/migrate-from-api-polling-to-cwms/) 문서에 대한 소규모 개선.
