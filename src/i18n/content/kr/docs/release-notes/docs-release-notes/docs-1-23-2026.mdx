### 주요 변경 사항

* 클라이언트 제거를 위한 설정 변경 사항이 포함된 [새로운 지원 매트릭스](/docs/ebpf/requirements), [eBPF 에이전트 Linux 설치](/docs/ebpf/linux-installation) 및 [Kubernetes 설치](/docs/ebpf/k8s-installation) 로 eBPF 문서를 업데이트했습니다.
* API 접근 제어 절차를 개선하기 위해 [비밀 정보 관리 관련 세부적인 접근 제어 문서를](/docs/apis/nerdgraph/examples/nerdgraph-api-secret-management-service) 업데이트했습니다.
* [루비 에이전트 요구사항이 업데이트되었으며 10.x 버전 변경으로 프레임워크가 지원되었습니다](/docs/apm/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks).
* [Nuxt.js 통합 문서가](/docs/browser/browser-integrations/nuxt-integration/) 추가되었으며, Nuxt 3+ 지원이 업데이트되고 서식이 개선되었습니다.

### 사소한 변경 사항

* 예외 지원이 포함된 강화된 [끌어 당기는 에이전트 통지\_error API](/docs/apm/agents/python-agent/python-agent-api/noticeerror-python-agent-api) 문서.
* [브라우저 로그 시작 가이드가](/docs/browser/browser-monitoring/browser-pro-features/browser-logs/get-started/) 업데이트되었으며, 로깅 관련 문서가 개선되었습니다.
* [웹페이지 수명 주기 탐색](/docs/browser/browser-monitoring/browser-pro-features/session-traces-explore-webpages-life-cycle) 에 대한 세션 트레이스 문서를 수정했습니다.
* 최신 호환성 보고서에 따라 [Node.js 에이전트 호환성 요구 사항이](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 업데이트되었습니다.
* 향상된 stub\_status 설정으로 [NGINX OpenTelemetry 설정이](/docs/opentelemetry/nginx/nginx-otel-host) 업데이트되었습니다.

### 새로워진 사항

* [읽기 전용 사용자를 위한 대시보드 작업](/whats-new/2026/01/whats-new-01-19-read-only-dashboards) - 대시보드 관리를 위한 표준 읽기 전용 역할 권한은 2026년 4월 30일부터 보기 전용 액세스로 제한됩니다.
