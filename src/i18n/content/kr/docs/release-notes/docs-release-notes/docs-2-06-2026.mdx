### 주요 변경 사항

* 개선된 문서와 업데이트된 대시보드 스크린샷을 통해 [뉴렐릭의 Kafka 데이터 찾기 및 쿼리가 업데이트](/docs/opentelemetry/integrations/kafka/find-and-query-data) 되었습니다.
* 확장된 설정 예제가 포함된 [OpenTelemetry문서를 통해 Kubernetes (Strimzi)의 향상된 모니터 Kafka입니다](/docs/opentelemetry/integrations/kafka/kubernetes-strimzi).
* [OpenTelemetry를 사용하는 자체 호스팅 Kafka 모니터링](/docs/opentelemetry/integrations/kafka/self-hosted) 관련 문서가 개선되었으며, 설정 지침 및 엔티티 탐색기 안내가 강화되었습니다.

### 사소한 변경 사항

* 워크플로우 자동화 문서의 시작 일정 쿼리가 업데이트되었습니다.
* Node.js 에이전트 호환성 보고서가 업데이트되었습니다.
* 미디어 스트리밍 에이전트(안드로이드, 브라우저 및 iOS)의 버전 번호가 업데이트되었습니다.
