### 새로운 문서

* 조직 수준의 공지 대상을 소개하기 위해 [교차 계정 대상 개요를](/docs/apis/nerdgraph/examples/Cross-account-destination/overview) 추가했습니다.
* 뉴럴릭 조직 내 모든 계정에 걸쳐 공지 대상을 생성하고 관리하는 데 필요한 포괄적인 지침을 제공하기 위해 [&apos;계정 간 공지 대상 관리&apos; 기능](/docs/apis/nerdgraph/examples/Cross-account-destination/manage-cross-account-destinations) 을 추가했습니다.
* [레지스트리 로케이션 키 관리를](/docs/synthetics/synthetic-monitoring/private-locations/manage-private-location-keys) 추가하여 신세틱스에서 패드 로케이션 키 관리에 대한 지침을 제공합니다. 여기에는 메타데이터 태그를 사용하여 키 나열, 필터링 및 편집을 위한 UI 가시성 기능이 포함됩니다.

### 주요 변경 사항

* 템플릿 변수 가시성 개선과 함께 [Dashboards API사용하여 생성 위젯 및 차트를](/docs/apis/nerdgraph/examples/create-widgets-dashboards-api) 업데이트했습니다.
* [NerdGraph API사용하여Dashboards ](/docs/apis/nerdgraph/examples/nerdgraph-dashboards)업데이트하고, 다양한 템플릿 변수 가시성 기능을 추가했습니다.
* 가시성 설정 지침으로 [대시보드 템플릿 변수가](/docs/query-your-data/explore-query-data/dashboards/dashboard-template-variables) 업데이트되었습니다.
* 업데이트됨: 템플릿 변수 관리 기능이 개선되어 [대시보드 관리가](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard) 더욱 편리해졌습니다.
* [문제 해결을 업데이트하고 포괄적인 오류 해결 지침을 통해 차트 오류를 해결했습니다](/docs/query-your-data/explore-query-data/dashboards/troubleshooting-chart-errors).
* 차트 사용법에 대한 문서가 개선되었습니다. [차트를 더욱 효과적으로 활용하세요](/docs/query-your-data/explore-query-data/use-charts/use-your-charts).
* [Kubernetes Strimzi Kafka 통합이](/docs/opentelemetry/integrations/kafka/kubernetes-strimzi) 업데이트되었으며, OpenTelemetry 수집 최적화를 위한 주요 개선 사항이 포함되었습니다.
* OpenTelemetry 데이터 수집 최적화를 위한 주요 개선 사항이 포함된 [자체 호스팅 Kafka 통합 기능이](/docs/opentelemetry/integrations/kafka/self-hosted) 업데이트되었습니다.
* 사용자 정의 속성에 대한 지원을 포함하여 [.NET에서 사용자 정의 속성을](/docs/apm/agents/net-agent/attributes/custom-attributes-net) 업데이트했습니다.
* 맞춤형 속성에 대한 지원으로 [.NET 에이전트 API](/docs/apm/agents/net-agent/net-agent-api/net-agent-api) 업데이트했습니다.
* .NET SRX(보안 검토 환경) 통합 세부 정보가 포함된 업데이트된 [애플리케이션 개요입니다](/docs/vulnerability-management/applications/overview).
* .NET SRX(보안 검토 환경) 통합 세부 정보가 포함된 업데이트된 [통합 개요](/docs/vulnerability-management/getting-started/integrations/overview) 입니다.
* OpenShift 버전 4.20을 지원하도록 [OpenTelemetry 및 Kubernetes에 대한 고급 설정이](/docs/kubernetes-pixie/k8s-otel/advance-config) 업데이트되었습니다.
* OpenShift 버전 4.20을 지원하는 [OpenTelemetry와 Kubernetes 통합 기능을 설치하도록](/docs/kubernetes-pixie/k8s-otel/install) 업데이트되었습니다.
* [Kubernetes 통합 호환성 및 요구 사항이](/docs/kubernetes-pixie/kubernetes-integration/get-started/kubernetes-integration-compatibility-requirements) 업데이트되었으며, OpenShift 버전 4.20을 지원합니다.
* AppLaunch/Cold 지표 제한이 있는 [iOS 앱의 앱 설치 시간이](/docs/mobile-monitoring/new-relic-mobile-ios/configuration/app-launch-times-ios-apps) 업데이트되고 NRQL 쿼리가 수정되었습니다.
* 앱 실행/콜드 메트릭 제한이 포함된 [모바일 모니터링 구성 설정이](/docs/mobile-monitoring/new-relic-mobile/mobile-sdk/configure-settings) 업데이트되었습니다.
* [SRE 에이전트 설정이](/docs/agentic-ai/sre-agent/sre-agent-setup) 대폭 개선되었습니다.

* [Fluent Bit 릴리스 노트가](/docs/release-notes/fluentbit-release-notes/fluentbit-26-02-17) 업데이트되었으며, 지원되는 운영 체제 버전 및 Helm 차트 정보가 포함되었습니다.
