### 주요 변경 사항

* v2.0.0에 맞춰 대폭 수정된 [최신 계측 제어 게이트웨이 설명서가](/docs/new-relic-control/pipeline-control/gateway/overview) 공개되었습니다. 새로운 프로세서(필터, 샘플링, 변환), UI 가이드, YAML 설정 개요, 데이터 스키마 차이점 등이 포함되어 있습니다.
* 설치 및 설정 지침에 대한 포괄적인 개선으로 [Elasticsearch OTel 통합 설치](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-install) 및 [개요](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) 가 업데이트되었습니다.
* 하이브리드 [에이전트](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file) 설정 지원을 위한 광범위한 문서로 업데이트되었습니다.
* [Go 에이전트 로깅 구성에 대한 포괄적인 지침이 포함된 업데이트된 Go 에이전트 로깅 문서입니다](/docs/apm/agents/go-agent/configuration/go-agent-logging).
* 용어의 일관성 향상을 위해 여러 액션 파일 전반에 걸쳐 용어를 수정하여 업데이트된 자동화 [액션 카탈로그](/docs/workflow-automation/setup-and-configuration/actions-catalog/actions-catalog) 입니다.

### 사소한 변경 사항

* [메트릭 정규화 규칙](/docs/new-relic-solutions/new-relic-one/ui-data/metric-normalization-rules) 에 다중 애플리케이션 지원 기능을 추가했습니다.
* [저항력 설정](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file) 에 `sql_metadata_comments` 설정 옵션을 추가했습니다.
* AWS EC2 콘텐츠 재구성을 반영하여 [AWS Lambda 액션을](/docs/workflow-automation/setup-and-configuration/actions-catalog/aws/aws-lambda) 업데이트했습니다.
* 용어 수정으로 [파이프라인 위험 게이트웨이 UI 가이드가](/docs/new-relic-control/pipeline-control/gateway/ui-guide) 업데이트되었습니다.
* 설정 수정으로 [Nginx OTel Kubernetes 통합이](/docs/opentelemetry/nginx/nginx-otel-kubernetes) 업데이트되었습니다.
* 새로운 설정 옵션으로 [루비 에이전트 설정이](/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration) 업데이트되었습니다.
* [Amazon ECS 통합](/docs/infrastructure/elastic-container-service-integration/introduction-amazon-ecs-integration) 문서에 링크 오류를 수정하여 업데이트했습니다.
* v9.1.0에 대한 [배터리 대응 문서가](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent) 업데이트되었습니다.
* 최신 프레임워크 지원 정보를 반영하여 [.NET 에이전트 호환성 문서를](/docs/apm/agents/net-agent/getting-started/net-agent-compatibility-requirements) 업데이트했습니다.
* 업데이트된 [신세틱스 단계 모니터 문서에서](/docs/apis/nerdgraph/examples/synthetics-api/step-monitor) EDGE를 지원 브라우저 목록에서 제외했습니다.
* 수정된 구문을 적용하여 [브라우저의 setUserId API를](/docs/browser/new-relic-browser/browser-apis/setuserid) 업데이트했습니다.
* 모바일 앱 사용자를 위한 MFA 설정 지침이 포함된 업데이트된 [인증 도메인 및 SAML SSO 문서가 제공](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more) 됩니다.
* [대시보드 및 차트 문서 공유를 위한 필수 조건이](/docs/query-your-data/explore-query-data/dashboards/prerequisites-to-share-dashboards-charts) 업데이트되었습니다.
* [NerdGraph 문서를 사용하여 드롭 데이터를](/docs/data-apis/manage-data/drop-data-using-nerdgraph) 업데이트했습니다.
* [로그인 문제 해결, 비밀번호 및 이메일 문제 해결 문서가](/docs/accounts/accounts-billing/account-setup/troubleshoot-new-relics-password-email-address-login-problems) 업데이트되었습니다.
