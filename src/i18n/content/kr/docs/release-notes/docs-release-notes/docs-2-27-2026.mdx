### 새로운 문서

* 수집기 성능 및 메트릭에 대한 포괄적인 모니터링 지침을 제공하기 위해 [OpenTelemetry Collector Observability (OpenTelemetry 수집기 관찰 가능성)](/docs/opentelemetry/collector-observability/collector-observability)을 추가했습니다.
* [UI 지능형](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-ui) 워크로드, 생성, [AI 인사이트,](/docs/new-relic-solutions/new-relic-one/workloads/advanced-workload-ai-insights) [KPI](/docs/new-relic-solutions/new-relic-one/workloads/create-intelligent-workload)[관리](/docs/new-relic-solutions/new-relic-one/workloads/intelligent-workload-kpis) 및 기능 을 포함하여 AI 기반 워크로드 [생성 및 KPI 관리에 대한 지침을 제공하는 지능형 워크로드](/docs/new-relic-solutions/new-relic-one/workloads/workloads-overview) 문서 모음을 추가했습니다.
* 모든 뉴렐릭 APM 에이전트 전반에 걸쳐 공급업체 비종속 측정에 대한 포괄적인 지침을 제공하기 위해 [OpenTelemetry API 지원이](/docs/apm/agents/manage-apm-agents/opentelemetry-api-support) 추가되었습니다.

### 주요 변경 사항

* [그로우 자동화 기능이](/docs/workflow-automation/introduction-to-workflow) 이제 정식으로 출시되었습니다.
* [이상치 탐지 기능이](/docs/alerts/create-alert/set-thresholds/outlier-detection) 이제 정식으로 제공됩니다.
* 설정, 설정, 모니터링, 문제 해결, 해결 지침이 포함된 Agent Control에 대한 포괄적인 [Windows 지원 문서를](/docs/new-relic-control/agent-control/overview) 추가했습니다.

### 사소한 변경 사항

* 개선된 [OpenTelemetry Kafka 모니터링](/docs/opentelemetry/integrations/kafka/self-hosted) 문서화에는 다이어그램과 확장된 문제 해결, 처리 지침이 포함됩니다.
* AWS SDK, LangChain, OpenAI 및 기타 패키지에 대한 최신 지원 버전 범위를 반영하여 [Node.js 에이전트 호환성 요구 사항을](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 업데이트했습니다.
* 버전 14.1.x로 확장된 WebLogic 지원을 통해 [배터리 대응 요구 사항이](/docs/apm/agents/java-agent/getting-started/compatibility-requirements-java-agent) 업데이트되었습니다.

### 새로운 게시물은 무엇인가요?

* [뉴렐릭 렌즈의 새로운 기능 게시물을](/whats-new/2026/02/whats-new-02-25-nr-lens/) 추가하여 데이터 수집 없이 외부 데이터 소스를 쿼리할 수 있게 되었음을 알려드립니다.
* AWS, Azure, Slack 및 PagerDuty 통합을 통한 노코드 자동화 워크플로우를 알리기 위해 [워크플로우 자동화 GA에 대한 새로운 기능 게시물을](/whats-new/2026/02/whats-new-02-24-workflow-automation/) 추가했습니다.
* [노트북을 사용한 동적 런북의 새로운 기능 발표 게시물을](/whats-new/2026/02/%20whats-new-02-25-dynamic-runbooks/) 추가하여 해당 기능이 공개 미리 보기에 포함되었음을 알렸습니다.
* [뉴렐릭 홈페이지에 새로운 소식 게시물을](/whats-new/2026/02/whats-new-02-24-new-home-page/) 추가하여 사용자의 특정 일상에 가장 중요한 서비스와 데이터에 대한 큐레이션된 뷰를 제공하는 개인 맞춤형 런치패드를 발표했습니다(공개 미리보기).

### 릴리즈 정보

* 최신 릴리스에 대한 최신 정보를 받아보세요.

  * [루비 에이전트 v10.2.0](/docs/release-notes/agent-release-notes/ruby-release-notes/ruby-agent-10-2-0):

    * 공급업체 비종속 API 호환성을 갖춘 OpenTelemetry Tracing 지원을 위한 하이브리드 에이전트를 도입했습니다.
    * 오류 수집기 역추적 잘림 설정 및 Logging gem 측정, 로그를 추가했습니다.
    * 병렬 처리 옵션을 통해 향상된 cloud 공급업체 감지 기능을 제공합니다.

  * [자바 에이전트 v9.1.0](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-910):

    * 추적, 메트릭 및 로그를 위한 OpenTelemetry API 호환성을 갖춘 포괄적인 하이브리드 에이전트 구현이 추가되었습니다.
    * Kotlin Coroutines 지원이 향상되었고 W3C 트레이스 전파 문제가 수정되었습니다.
    * 감시 데이터 작업 및 멀티호스트 설정 수정 사항이 추가되었습니다.

  * [Node.js 에이전트 v13.14.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-13-14-0):

    * Azure Functions Apps에서 로그를 자동으로 수집하는 기능을 추가했습니다.
    * 오류 원인 지원 및 프로파일러 지연/지속 시간 제어 기능을 구현했습니다.
    * `embedContentInternal` 메서드 호환성을 위해 Google Gen AI 통합을 업데이트했습니다.

  * [안드로이드 에이전트 v7.7.0](/docs/release-notes/mobile-release-notes/android-release-notes/android-770):

    * Android Gradle 플러그인 9에 대한 지원이 추가되었습니다.
    * 오류 샘플링 및 새로운 `pauseReplay`/`recordReplay` API를 갖춘 향상된 모바일 세션 리플레이.
    * 세션 리플레이 기능에 대한 여러 성능 개선 사항을 구현했습니다.

  * [인프라 에이전트 v1.72.6](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1726):

    * Windows 환경용 fluent-bit 버전을 4.2.2로 업데이트했습니다.

  * [브라우저 에이전트 v1.310.1](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.310.1):

    * Core Web Vitals 보고 기능을 향상시키기 위해 대체 속성 값을 포함하는 LCP 항목이 추가되었습니다.
