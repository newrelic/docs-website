### 새로운 문서

* [Elasticsearch OpenTelemetry 통합 개요를](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-overview) 추가하여 OpenTelemetry를 사용하여 Elasticsearch를 모니터링하는 방법에 대한 포괄적인 지침을 제공합니다.
* [Elasticsearch OpenTelemetry 통합 설치](/docs/opentelemetry/integrations/elasticsearch/elasticsearch-otel-integration-install) 방법을 단계별로 안내하기 위해 Elasticsearch OpenTelemetry 통합 설치 섹션을 추가했습니다.
* [Elasticsearch 문제 해결, 문제 해결](/docs/opentelemetry/integrations/elasticsearch/troubleshooting) 제공을 위한 해결, Elasticsearch 모니터링을 위한 해결 지침이 추가되었습니다.
* Xcode 15.3 이상에서 iOS 에이전트 dSYM 업로드 실패에 대한 문제 해결, 해결 지침을 제공하기 위해 [자동 dSYM 업로드 실패(Xcode 15.3+)](/docs/mobile-monitoring/new-relic-mobile-ios/troubleshoot/dSYM-upload-tools) 가 추가되었습니다.

### 주요 변경 사항

* 예산, 권장 사항, 통합을 포함한 기능의 구성 및 검색 가능성을 개선하기 위해 [클라우드 인프라 인텔리전스(CCI) 문서를](/docs/cci/overview) 재구성했습니다.
* 업데이트된 [자동화 액션 카탈로그에서](/docs/workflow-automation/setup-and-configuration/actions-catalog/actions-catalog) 액션 문서가 대폭 수정되었으며, 뉴럴릭 데이터 수집 및 NRDB 검색을 위한 두 가지 새로운 액션이 추가되었습니다.
* 개선된 OpenTelemetry 지원 문서 구조와 계정 및 청구 지원 섹션을 추가하여 [글로벌 기술 지원 서비스를](/docs/licenses/license-information/general-usage-licenses/global-technical-support-offerings) 업데이트했습니다.
* 설정 예제가 간소화되고 문서 구조가 개선되어 [자체 캐시를 사용하는 방법](/docs/distributed-tracing/infinite-tracing-on-premise/bring-your-own-cache) 이 업데이트되었습니다.
* 최신 버전 변경 사항을 반영하여 [Databricks 통합 기능을](/docs/infrastructure/other-infrastructure-integrations/databricks-integration) 업데이트했습니다.

### 사소한 변경 사항

* 에이전트 에이전트 v1.72.1에 대한 새로운 설정 옵션으로 업데이트된 [에이전트 에이전트 설정](/docs/infrastructure/infrastructure-agent/configuration/infrastructure-agent-configuration-settings).
* [Node.js 에이전트 v13.11.0에 대한 Node.js 에이전트 호환성 요구 사항이](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent) 업데이트되었습니다.
* Elasticsearch 통합 정보가 포함된 향상된 [OpenTelemetry 통합 목록입니다](/docs/opentelemetry/opentelemetry-integrations-list).
* 콘텐츠 명확성을 개선하여 [대시보드 소개를](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) 업데이트했습니다.
* [VMware Tanzu 서비스 브로커 통합](/docs/infrastructure/other-infrastructure-integrations/cloudfoundry-integrations/vmware-tanzu-service-broker-integration) 기능이 수정되었습니다.
* [인증 도메인](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more) 에 MFA 설정 지침을 추가했습니다.
* iOS 에이전트 v7.6.0을 반영하기 위해 [iOS 에이전트 요구사항이](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/new-relic-ios-compatibility-requirements) 업데이트되었습니다. iOS 16 이상이 필요합니다.
* iOS 16 이상이 필요하다는 내용을 반영하여 [.NET MAUI 에이전트 설명서가](/docs/mobile-monitoring/new-relic-mobile-maui-dotnet/monitor-your-net-maui-application) 업데이트되었습니다.
