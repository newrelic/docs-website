### 새로운 문서

* 보안 취약점 공개를 위해 [보안 게시판 NR26-05](/docs/security/new-relic-security/security-bulletins/security-bulletin-nr26-05) 를 추가했습니다.
* [에이전트 컨트롤 미러 설정](/docs/new-relic-control/agent-control/mirror-setup) 문서를 추가했습니다.

### 주요 변경 사항

* [인증 도메인](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more) 문서에 MFA를 활성화하는 단계를 추가했습니다.
* UI 개편 문서로 [신세틱스 런타임 업그레이드 UI](/docs/synthetics/synthetic-monitoring/using-monitors/runtime-upgrade-ui) 를 업데이트했습니다.
* [속성 사전](/attribute-dictionary)에 새로운 브라우저 속성을 추가했습니다.

### 사소한 변경 사항

* 여러 끊어진 링크 및 서식 문제를 수정했습니다.
* [성능 보고서](/docs/apm/reports/performance-reports) 문서에 주간 보고서 메트릭에 대한 새로운 NRQL 쿼리를 추가했습니다.

### 수명 종료
