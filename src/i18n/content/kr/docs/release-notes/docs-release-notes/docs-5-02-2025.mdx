### 주요 변경 사항

* [뉴렐릭 계정 다운그레이드 또는 취소에](/docs/accounts/accounts-billing/account-setup/downgradecancel-account/#preparation-org-delete) 대한 가이드라인이 업데이트되었습니다.
* .NET 에이전트 설정에 대한 [속성 요소가](/docs/apm/agents/net-agent/configuration/net-agent-configuration/#agent-attributes) 향상되었습니다.
* 뉴렐릭 AI Knowledge 커넥터에 대한 [커넥터 설정이](/docs/agentic-ai/knowledge-integration/knowledge-connector/) 개선되었습니다.

### 사소한 변경 사항

* [NRQL 참조](/docs/nrql/nrql-syntax-clauses-functions/) 에 내장된 기능을 추가했습니다.
* `kubectl` 명령을 실행할 때 네임스페이스를 지정하기 위해 `-k8s-namespace STRING` 옵션을 도입하여 [nrdiag에 대한 명령줄 옵션을](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag/) 업데이트했습니다.

### 릴리즈 정보

최신 릴리스에 대한 최신 정보를 받아보세요.

* [끌어 당기는 에이전트 v10.10.0](/docs/release-notes/agent-release-notes/python-release-notes/python-agent-101000):

  * Elasticsearch에 비동기 지원이 추가되었습니다.
  * GraphQL 구성 설정의 구문 분석을 활성화했습니다.
  * Sentry로 Kombu를 실행할 때 발생하는 충돌을 해결했습니다.

* [Kubernetes 통합 v3.39.0](/docs/release-notes/infrastructure-release-notes/kubernetes-integration-release-notes/kubernetes-integration-3-39-0):

  * `newrelic-infrastructure` 버전 3.44.0으로 업그레이드했습니다.
  * `nri-bundle` 을 버전 5.0.122로 업데이트했습니다.

* [iOS v6.9.5용 모바일 앱](/docs/release-notes/mobile-apps-release-notes/new-relic-ios-release-notes/new-relic-ios-6095):

  * EXT에서 NGEP로 팀을 이전했습니다.
  * 다중 팀 지원이 추가되었습니다.
  * 팀 세부 정보 화면에 팀 검색 기능을 도입했습니다.

* [iOS v6.9.6용 모바일 앱](/docs/release-notes/mobile-apps-release-notes/new-relic-ios-release-notes/new-relic-ios-6096):

  * 앱 내 피드백 기능을 수정했습니다.

* [브라우저 에이전트 v1.289.0](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.289.0/#remove-agent-config-cache-and-getterssetters):

  * 에이전트 설정 캐시 및 관련 getter/setter를 제거했습니다.

* [Node.js 에이전트 v12.18.2](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-12-18-2):

  * `max_trace_segments` 초과할 때 트레이에 세그먼트를 추가하지 못하도록 `createSegment` 을 업데이트했습니다.
  * 호환성 보고서를 수정했습니다.

* [Node.js 에이전트 v12.18.1](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-12-18-1):

  * 세그먼트를 트랜잭션 트리에서 찾을 수 없는 경우 세그먼트의 독점 시간을 계산하기 위한 방어 코드가 추가되었습니다.
  * OpenTelemetry 브리지 트레이스 전파기가 `traceparent` 및 `tracestate` 헤더를 할당하는 방식을 개선했습니다.
  * 호환성 보고서를 업데이트했습니다.
