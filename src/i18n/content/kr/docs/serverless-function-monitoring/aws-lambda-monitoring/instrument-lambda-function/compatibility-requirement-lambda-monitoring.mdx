[Lambda 모니터링을 설치하기](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/configure-serverless-aws-monitoring) 전에 시스템이 다음 요구 사항을 충족하는지 확인하세요.

## 기본 요구 사항 [#requirements]

Lambda 계층을 사용하여 서버리스 모니터링을 활성화하려면 다음이 필요합니다.

* [관리자 역할](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-managing-users)이 있거나 **Infrastructure manager** \[인프라 관리자] [추가 역할](/docs/accounts/original-accounts-billing/original-users-roles/users-roles-original-user-model#add-on)이 있는 New Relic 계정.

* <InlinePopover type="userKey" />.

* IAM 리소스, 관리형 암호 및 Lambda를 생성할 수 있는 권한이 있는 AWS 계정. CloudFormation 스택 및 S3 버킷을 생성하기 위한 권한도 필요합니다.

<Callout variant="tip">
  뉴렐릭은 [AWS 통합을](/docs/infrastructure/amazon-integrations/get-started/introduction-aws-integrations/) 사용하여AWS 계정을 뉴렐릭과 통합하여 Lambda 함수를 자동으로 검색하고 모니터링할 것을 권장합니다. 이를 통해 서버리스 기능에 뉴렐릭 APM 의 모든 기능을 활용할 수 있습니다.
</Callout>

## 지원되는 런타임 [#supported-runtimes]

측정 유형에 따라 다음과 같은 런타임이 지원됩니다.

<Tabs>
  <TabsBar>
    <TabsBarItem id="1">계층화된 측정, 플러그인</TabsBarItem> <TabsBarItem id="2">SDK 기반 측정,</TabsBarItem> <TabsBarItem id="3">컨테이너화된 측정, 리소스</TabsBarItem> <TabsBarItem id="4">OpenTelemetry</TabsBarItem>
  </TabsBar>

  <TabsPages>
    <TabsPageItem id="1">
      * Node.js: `nodejs16.x`, `nodejs18.x`, `nodejs20.x`, `nodejs22.x`
      * Python: `python3.8`, `python3.9`, `python3.10`, `python3.11`, `python3.12`, `python3.13`
      * 이동: `provided.al2` , `provided.al2023`
      * 자바: `java8.al2`, `java11`, `java17`, `java21`
      * 루비: `ruby3.2`, `ruby3.3`
      * .NET: `dotnet6`, `dotnet8`
    </TabsPageItem>

    <TabsPageItem id="2">
      * 각 런타임에 대해 지원되는 뉴렐릭 에이전트 패키지를 사용하여 Lambda 함수를 로그업하세요.
    </TabsPageItem>

    <TabsPageItem id="3">
      * Node.js(OS/아키텍처: Linux, Windows, ARM, ARM 64, x86, x86-64) : `nodejs16.x`, `nodejs18.x`, `nodejs20.x`, `nodejs22.x`
      * 끌어오기 (OS/Arch: Linux, Windows, ARM, ARM 64, x86, x86-64): `python3.7`, `python3.8`, `python3.9`, `python3.10`, `python3.11`, `python3.12`, `python3.13`
      * 다음 (OS/Arch: Linux, Windows, ARM, ARM 64, x86, x86-64): `java8`, `java11`, `java17`, `java21`
      * 루비 : `ruby3.2`, `ruby3.3`
      * .NET(OS/아키텍처: Linux, Windows, ARM, ARM 64, x86, x86-64): `dotnet6`, `dotnet8`
    </TabsPageItem>

    <TabsPageItem id="4">
      * 자바: `java8.al2` 또는 `java11`
    </TabsPageItem>
  </TabsPages>
</Tabs>

## 지원 라이브러리

* 열대: 뉴렐릭 컨테이너의 계층화 및 SDK 측정, 로그는 `com.amazonaws:aws-lambda-java-events` 버전 `3.0.0` 부터 최신 버전과 호환됩니다.
