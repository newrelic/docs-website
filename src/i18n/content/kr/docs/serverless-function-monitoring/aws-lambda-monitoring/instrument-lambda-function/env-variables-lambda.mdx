환경 변수는 코드 외부에 설정과 비밀을 저장하는 방법입니다. 다음 표에서는 Lambda 함수를 NEWLick으로 모니터링하기 위해 설정할 수 있는 필수 변수와 런타임별 변수를 표시합니다.

## 필수 환경 변수

런타임에 관계없이 Lambda 함수를 New Relic으로 모니터링하려면 다음 환경 변수를 설정해야 합니다.

<table>
  <thead>
    <tr>
      <th style={{ width: "200px" }}>
        변수 이름
      </th>

      <th>
        설명
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `NEW_RELIC_ACCOUNT_ID`
      </td>

      <td>
        귀하의 뉴렐릭 계정 ID를 지정합니다
      </td>
    </tr>

    <tr>
      <td>
        `NEW_RELIC_LAMBDA_HANDLER`
      </td>

      <td>
        함수의 실제 핸들러를 찾는 데 사용되는 핸들러 래퍼를 지정합니다. Go 런타임의 경우 이 변수는 필수가 아닙니다.
      </td>
    </tr>

    <tr>
      <td>
        `NEW_RELIC_LICENSE_KEY`
      </td>

      <td>
        Secrets Manager를 재정의하여 뉴렐릭 수집 키를 지정합니다.
      </td>
    </tr>

    <tr>
      <td>
        `NEW_RELIC_APM_LAMBDA_MODE`
      </td>

      <td>
        Lambda 함수에 대한 APM 모니터링을 활성화하려면 

        `true`

         으로 설정하세요.
      </td>
    </tr>

    <tr>
      <td>
        `NEW_RELIC_TRUSTED_ACCOUNT_KEY`
      </td>

      <td>
        존재하는 경우 귀하의 뉴렐릭 계정 ID 또는 부모 ID를 지정합니다.
      </td>
    </tr>
  </tbody>
</table>
