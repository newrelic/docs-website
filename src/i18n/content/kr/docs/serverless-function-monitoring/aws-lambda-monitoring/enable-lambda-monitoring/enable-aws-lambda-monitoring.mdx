Lambda 모니터링을 활성화하는 몇 가지 단계가 있습니다.

1. [AWS 계정을 연결합니다.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/account-linking)
2. [선택 사항: CloudWatch 로그 수집 구성](/docs/logs/enable-log-management-new-relic/enable-log-monitoring-new-relic/aws-lambda-sending-cloudwatch-logs) : 1단계에서 기본적으로 CloudWatch 로그 수집기를 자동으로 설치하는 `newrelic-lambda` CLI를 사용한 경우 이 단계를 건너뛸 수 있습니다.
3. [예제 함수를 계측합니다.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-example)
4. [기능을 계측하십시오.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-your-own)
5. [선택 사항: 경고 및 사용자 지정 이벤트를 설정합니다.](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/configure-serverless-monitoring-aws-lambda)

[Lambda에 대한 레거시 수동 계측](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy) 도 참조하십시오.
