1단계에서 newrelic-lambda CLI를 사용한 경우 [API Polling](/docs/infrastructure/amazon-integrations/connect/connect-aws-new-relic-infrastructure-monitoring) 또는 [Metric Streams](/docs/infrastructure/amazon-integrations/connect/aws-metric-stream) 통합을 설치했는지 여부를 확인했습니다. 둘 중 하나가 표시되지 않으면 API 폴링 통합을 설치한 것입니다.

1단계에서 CLI를 사용하지 않은 경우 API 폴링 또는 메트릭 스트림 통합을 설치 및 구성했는지 확인하십시오.
