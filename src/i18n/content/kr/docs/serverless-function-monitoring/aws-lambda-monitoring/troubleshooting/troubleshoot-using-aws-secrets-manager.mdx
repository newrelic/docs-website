## 문제

AWS Secrets Manager를 사용하여 뉴런 클러스터 키를 저장할 수 없습니다.

## 해결책

회사에서 AWS Secrets Manager 사용을 허용하지 않는 경우 뉴럴릭 Lambda Extension은 `NEW_RELIC_LICENSE_KEY` 환경 변수를 허용합니다. `newrelic-lambda integrations install` 명령에서 `--disable-license-key-secret` 플래그를 추가합니다. 그런 다음 Lambda 함수 구성에서 이 환경 변수를 <InlinePopover type="licenseKey" />으로 설정합니다.
