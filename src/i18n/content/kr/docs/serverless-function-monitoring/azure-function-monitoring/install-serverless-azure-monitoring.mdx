You can configure New Relic monitoring for Azure Function Apps hosted on Linux, Windows, or as containerized functions. This is achieved by instrumenting your Function App with the appropriate New Relic agent for its runtime such as Node.js, .NET, or Python.

## 시작하기 전에 [#begin]

Ensure your Azure Function meets our [compatibility and requirements](/docs/serverless-function-monitoring/azure-function-monitoring/compatibility-requirement-azure-monitoring).

<Steps>
  <Step>
    ## Azure 계정을 뉴렐릭에 연결하세요 [#link]

    Link your Azure account to New Relic by configuring polling for Azure Monitor metrics. This enables you to view these metrics in the New Relic UI. For more information, refer [Azure integration](/docs/infrastructure/microsoft-azure-integrations/azure-integrations-list/azure-functions-monitoring-integration/#polling).
  </Step>

  <Step>
    ## Instrument your Azure Function [#instrument]

    Depending on your deployment environment, select one of the following options to instrument your Azure Function with the New Relic.

    <Tabs>
      <TabsBar>
        <TabsBarItem id="linux">Linux</TabsBarItem> <TabsBarItem id="windows">Windows</TabsBarItem> <TabsBarItem id="container">Containerized Function</TabsBarItem>
      </TabsBar>

      <TabsPages>
        <TabsPageItem id="linux">
          You can instrument your Azure Function App on Linux with New Relic for the following runtime environments:

          * [노드.js](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#node)
          * [파이썬](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#python)
          * [.그물](/docs/serverless-function-monitoring/azure-function-monitoring/linux/#net)
        </TabsPageItem>

        <TabsPageItem id="windows">
          You can instrument your Azure Function App on Windows with New Relic for the following runtime environments:

          * [노드.js](/docs/serverless-function-monitoring/azure-function-monitoring/windows/#node)
          * [.그물](/docs/serverless-function-monitoring/azure-function-monitoring/windows/#net)
        </TabsPageItem>

        <TabsPageItem id="container">
          You can instrument your Azure Function App on Linux with New Relic for the following runtime environments:

          * [노드.js](/docs/serverless-function-monitoring/azure-function-monitoring/container/#node)
          * [파이썬](/docs/serverless-function-monitoring/azure-function-monitoring/container/#python)
          * [.그물](/docs/serverless-function-monitoring/azure-function-monitoring/container/#net)
        </TabsPageItem>
      </TabsPages>
    </Tabs>
  </Step>

  <Step>
    ## 환경 변수 구성 [#env-variables]

    After you&apos;ve instrumented your Azure Function App with New Relic, [configure environment variables](/docs/serverless-function-monitoring/azure-function-monitoring/env-variables-azure) for its runtime environment.
  </Step>

  <Step>
    ## Azure Functions 다시 시작 [#restart]

    환경 변수를 추가한 후 Azure Functions를 다시 시작하여 변경 사항을 적용합니다.
  </Step>
</Steps>
