[귀하의 코드로 구축된 서비스에서 새로운 프로세서](/docs/codestream/observability/transactions) CodeStream 수준 이상 행동이 감지되면 에서 이메일을 통해 귀하에게 알림을 보냅니다. 이러한 공지를 관리하려면 사용자 이름 메뉴 아래 <DNT>**Notifications**</DNT> 설정으로 이동하세요. 어떤 스토어와 서비스가 알림을 발생시키거나 완전히 비활성화할지 사용자 지정할 수 있습니다.

기본적으로 CodeStream은 IDE에서 연 저장소를 자동으로 추적합니다. 또는 CodeStream 트리 보기에서 저장소 위에 마우스를 올려놓고 **Follow** \[팔로우] 아이콘을 클릭하면 저장소를 수동으로 팔로우할 수 있습니다.

<img title="Manually following a repository." alt="A screenshot of the icon for following a repository." src="/images/codestream_screenshot-crop_follow-repo.webp" />

<DNT>**Notification Settings**</DNT> 페이지로 가서 현재 팔로우하고 있는 저장소의 오른쪽에 있는 **X** 를 클릭하면 저장소를 팔로우 취소할 수 있습니다.

기본적으로 귀하는 팔로우하는 모든 서비스에서 이상 행동에 대한 공지를 받게 됩니다. 공지를 특정 서비스로 제한하려면 서비스 이름(예: &quot;prod&quot;가 포함된 이름), 계정 ID 또는 키/값 쌍(예: `environment:production`)으로 필터링할 수 있습니다.
