CodeStream 계정 설정을 사용하면 개인 계정을 변경할 수 있습니다.

## 계정 설정 [#account-settings]

<DoNotTranslate>**Account**</DoNotTranslate> 메뉴는 CodeStream 창 상단의 사용자 이름 메뉴 아래에 있으며, 프로필을 보고 CodeStream 사용자 이름을 변경할 수 있는 옵션을 찾을 수 있습니다. Gravatar의 뉴렐릭 이메일 주소와 연결된 프로필 사진이 있으면 CodeStream 에서 자동으로 사용됩니다.

## 알림 [#notifications]

[변칙적인 성능을](/docs/codestream/observability/transactions) 보이는 프로세서는 IDE에서 알림 메시지를 표시합니다. IDE에 여러 개의 로그가 열려 있고/또는 해당 로그와 연결된 여러 서비스가 있는 경우 변칙적인 성능이 있는 첫 번째 프로세서에 대해서만 알림을 받게 됩니다. IDE의 데스크톱 공지는 VS Code 또는 JetBrains IDE를 사용하는 경우에만 사용할 수 있습니다.
