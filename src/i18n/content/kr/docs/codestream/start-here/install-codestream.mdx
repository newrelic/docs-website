특정 IDE에 대한 CodeStream 확장을 설치할 수 있습니다.

<CollapserGroup>
  <Collapser className="freq-link" id="vscode" title="비주얼 스튜디오 코드">
    [Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream) 용 CodeStream을 다운로드하여 설치합니다. 확장 마켓플레이스를 통해 Visual Studio Code에 직접 설치할 수도 있습니다.
  </Collapser>

  <Collapser className="freq-link" id="visual-studio" title="비주얼 스튜디오">
    [Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=CodeStream.codestream-vs-22) 용 CodeStream을 다운로드하고 설치합니다. 확장 마켓플레이스를 통해 Visual Studio에 직접 설치할 수도 있습니다.
  </Collapser>

  <Collapser className="freq-link" id="jetbrains" title="제트브레인">
    [JetBrains](https://plugins.jetbrains.com/plugin/12206-codestream) 용 CodeStream을 다운로드하여 설치합니다. JetBrains 플러그인 메뉴에서 설치할 수도 있습니다.
  </Collapser>
</CollapserGroup>

VS Code를 사용하는 경우 일부 CodeStream 기능에는 [저녁](https://marketplace.visualstudio.com/items?itemName=redhat.java), [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp), [PHP](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client), [스타일](https://marketplace.visualstudio.com/items?itemName=ms-python.python), [Go](https://marketplace.visualstudio.com/items?itemName=golang.go) 또는 [루비에](https://marketplace.visualstudio.com/items?itemName=Shopify.ruby-lsp) 대한 언어 지원 확장이 필요합니다.
