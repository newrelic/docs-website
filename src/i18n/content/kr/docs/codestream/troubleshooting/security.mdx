우리는 소스 코드가 가장 가치 있고 민감한 자산 중 하나라는 것을 알고 있습니다. 대부분의 개발 팀의 경우 가장 중요한 작업 제품을 문자 그대로 표현한 것입니다.

귀하가 의견이나 문제를 통해 코드 블록에 대해 논의할 때 CodeStream은 데이터베이스에 코드 조각을 저장해야 합니다. CodeStream은 다양한 이유로 해당 코드 블록을 데이터베이스에 저장합니다. 예를 들어, 문제의 코드가 푸시되지 않은 경우 해당 코드를 저장하는 것이 팀원과 공유할 수 있는 유일한 방법입니다.
