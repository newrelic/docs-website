GitHub에 연결하면 CodeStream 창의 **끌어오기 요청** 섹션에 열려 있는 모든 끌어오기 요청과 **문제** 섹션에서 귀하에게 할당된 모든 문제가 표시되어야 합니다. GitHub 조직의 끌어오기 요청 또는 문제가 누락된 경우, 이는 사용자가 GitHub에 인증하고 모든 조직에 대한 액세스 권한을 부여하지 않았기 때문일 수 있습니다.

/>

인증 시 **부여** 버튼을 클릭하지 않은 경우 GitHub에서 **Settings > Applications** 으로 이동하고 **Authorized OAuth Apps** 탭을 클릭합니다. 여기에서 CodeStream 애플리케이션을 클릭합니다. 다음 페이지에서 CodeStream에서 액세스할 수 있도록 하려는 조직 옆에 있는 **Grant** 버튼을 클릭합니다.

경우에 따라 **Grant** 대신 **요청** 버튼이 표시됩니다. 이는 GitHub 조직의 소유자가 액세스 권한을 부여해야 함을 의미합니다.
