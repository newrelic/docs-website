GitLab 자체 관리형 인스턴스의 URL 끝에 `/api/v4/version` 을 추가하면 결과 페이지에 버전 정보가 표시됩니다.

예를 들어 URL은 `http://gitlab.codestream.us/api/v4/version` 이며 결과는 다음과 같습니다.

```
version:	"13.6.4"
revision:  "2754dab2b1b"
```

GitLab 자체 관리형의 병합 요청과 CodeStream의 통합을 사용하려면 버전 12.10 이상이 필요합니다.
