댓글이나 문제를 게시하면 팀원은 활동 피드를 통해 알림을 받게 되며 이메일을 통해 알림을 받을 수도 있습니다. 하지만 때로는 Microsoft Teams에도 공유하고 싶을 수도 있습니다. 이를 통해 아직 CodeStream에 가입하지 않았거나 IDE에서 많은 시간을 보내지 않는 사람들에게 다가갈 수 있습니다.

## Microsoft 팀에 연결 [#connect]

1. CodeStream의 사용자 이름 메뉴 아래에 있는 <DoNotTranslate>**Integrations**</DoNotTranslate> 페이지로 이동하여 <DoNotTranslate>**Microsoft Teams**</DoNotTranslate> 버튼을 클릭합니다.

2. 표시된 로그인 코드를 복사한 후 <DoNotTranslate>**Connect to Teams**</DoNotTranslate> 버튼을 클릭하세요.

   <img
     title="Connect to Microsoft Teams"
     alt="A screenshot showing the page to connect to Microsoft Teams."
     src={codestreamMsteamsConnect}
   />

3. CodeStream 앱에 대한 다음 팝업이 표시되는 Teams로 이동됩니다. <DoNotTranslate>**Add**</DoNotTranslate> 버튼을 클릭하세요. 참고, 확장하지 말고 팀이나 채팅에 추가할 옵션을 선택하세요. 추가를 클릭하면 됩니다.

   <img
     title="Add"
     alt="A screenshot showing the Add button."
     src={codestreamMsteamsCodestreamAppPage}
   />

4. 그러면 CodeStream 봇과의 비공개 채팅으로 이동하고 Teams 사이드바에 CodeStream 로고가 표시됩니다. 봇의 환영 메시지에서 <DoNotTranslate>**Sign In**</DoNotTranslate> 버튼을 클릭하거나 이 채팅에 `signin` 입력하고 제출하세요.

   <img
     title="Weclome"
     alt="A screenshot showing the bot welcome message."
     src={codestreamMsteamsCodestreamBotWelcome}
   />

5. 팝업에 로그인 토큰을 붙여넣은 다음 <DoNotTranslate>**Sign-in**</DoNotTranslate> 버튼을 클릭하세요.

   <img
     title="Token"
     alt="An image showing where to paste the token."
     src={codestreamMsteamsTokenExample}
   />

6. 공유하려는 채널로 이동하여 `@` 입력한 다음 팝업에서 <DoNotTranslate>**Get bots**</DoNotTranslate> 선택합니다.

   <img
     title="Bots"
     alt="An image showing the popup that appears when typing @"
     src={codestreamMsteamsAddBotButton}
   />

7. 목록에서 CodeStream 봇을 선택하고(필요한 경우 검색할 수 있음) <DoNotTranslate>**Add**</DoNotTranslate> 클릭합니다.

8. 채널로 돌아가서 `connect` 명령을 사용하여 CodeStream 봇을 @mention합니다. 공유하려는 모든 채널에서 이 작업을 반복하세요.

   <img
     title="Connect bot"
     alt="An image showing the command to connect a bot."
     src={codestreamMsteamsCodestreamConnectBotMessage}
   />

9. 확인을 받으면 CodeStream에서 Teams로 공유할 준비가 된 것입니다.

10. 이제 CodeStream으로 돌아가면 방금 연결한 채널을 공유할 수 있습니다.

## Microsoft Teams의 CodeStream에 참여 [#participate]

Microsoft Teams에 공유하면 코드마크에 대해 팀원들에게 알릴 뿐만 아니라 이를 사용하여 IDE로 직접 이동하여 CodeStream의 대화에 참여할 수 있습니다.

### IDE에서 열기 [#open-ide]

IDE 내에서 코드와 토론을 모두 보려면 <DoNotTranslate>**Open in IDE**</DoNotTranslate> 클릭하세요. 열려는 IDE를 선택할 수 있는 CodeStream 웹 페이지로 이동합니다. CodeStream은 다음에 동일한 저장소에서 토론을 볼 때 선택 사항을 기억합니다. 그런 다음 IDE의 적절한 소스 파일로 이동하고 관련 코드 블록으로 스크롤하며 CodeStream 창에 토론이 표시됩니다. IDE에서 해당 클립이 열려 있지 않은 경우 CodeStream 자동으로 소스 파일을 엽니다(귀하가 이전에 CodeStream 설치된 클립을 열었다고 가정하여 어디서 찾을 수 있는지 알 수 있음).

JetBrains IDE를 사용하는 경우 CodeStream이 IDE에 딥 링크할 수 있도록 [Toolbox 앱](https://www.jetbrains.com/toolbox-app/) 을 설치하십시오.

### GitHub(또는 Bitbucket 또는 GitLab)에서 열기 [#open-github]

코드 블록이 GitHub, Bitbucket 또는 GitLab에서 호스팅되는 리포지토리에서 가져온 경우 이 버튼을 누르면 해당 호스팅 서비스의 해당 코드 블록으로 이동합니다.
