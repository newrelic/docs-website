Dashboards 뉴렐릭 경험의 핵심이며, 이제 IDE에서 직접 모든 대시보드에 액세스할 수 있습니다. CodeStream의 글로벌 탐색에서 <DNT>**Dashboards**</DNT> 아이콘을 찾아 대시보드를 검색하거나 즐겨찾는 대시보드에 액세스하세요.

<img title="Dashboard search." alt="A screenshot displaying the ability to search for dashboards." src="/images/codestream_screenshot-full_search-dashbaords.webp" />

대시보드를 열려면 클릭하기만 하면 됩니다. 웹 브라우저로 전환할 필요가 없습니다.
