로그를 검색하기 위해 IDE와 브라우저 간 컨텍스트 전환을 통해 조사 속도를 늦출 필요가 없습니다. CodeStream은 뉴렐릭 Logs 검색 경험을 IDE에 바로 제공합니다! CodeStream 창에 나열된 서비스에 대해 <DNT>**View Logs**</DNT> 항목을 클릭하거나 CodeStream의 전역 탐색에서 <DNT>**View Logs**</DNT> 아이콘을 클릭합니다.

페이지 상단의 드롭다운 메뉴 목록은 CodeStream 창의 트리 보기에서 현재 선택된 서비스를 반영합니다. 그러나 해당 로그를 보기 위해 서비스나 부분을 자유롭게 선택할 수 있습니다.

선택한 파티션에 파티션이 있는 경우, 드롭다운 메뉴에서 특정 파티션을 선택하여 더욱 세분화하여 검색할 수 있습니다.

자세한 정보를 보려면 로그 항목 옆에 있는 캐럿을 클릭하여 항목을 확장하세요. 복사하려면 값 위로 마우스를 가져가세요.

키워드 검색(예: 오류 메시지)을 사용하여 결과 범위를 좁혔고 이제 상황에 맞는 특정 결과를 보려면 해당 항목 위에 마우스를 놓고 오른쪽에 나타나는 _주변 로그 표시_ 버튼을 클릭하세요. 그러면 해당 특정 결과가 고정되고 로그 파일에서 해당 결과의 앞뒤에 있는 로그 항목이 표시됩니다.

## 특정 로그 줄 검색 [#context-menu]

코드의 모든 로그 줄에서 바로 로그 검색으로 이동할 수도 있습니다. 검색할 문자열을 선택하고 컨텍스트 메뉴에서 <DNT>**Find in logs**</DNT> 을 선택할 수 있습니다. 텍스트를 먼저 선택하지 않고 <DNT>**Find in logs**</DNT> 선택하면 CodeStream이 커서가 있는 줄에서 문자열을 자동으로 추출합니다.
