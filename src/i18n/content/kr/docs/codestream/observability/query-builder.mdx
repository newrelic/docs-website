임시 쿼리든 [NRQL 파일을](#nrql-files) 통해 저장하는 쿼리든, IDE에서 바로 쿼리를 실행하는 기능을 사용하면 뉴렐릭이 보유한 서비스에 대한 모든 성능 데이터에 강력하게 액세스할 수 있습니다. NRQL에 아직 익숙하지 않다면 [New Relic 쿼리 언어](/docs/nrql/get-started/introduction-nrql-new-relics-query-language) 에 대해 자세히 알아볼 수 있습니다.

쿼리 빌더에 액세스하려면 CodeStream의 전역 탐색에서 "데이터 쿼리" 아이콘을 클릭하세요.

지정한 데이터를 기반으로 쿼리 빌더는 결과를 가장 효과적으로 표시하는 차트 유형을 선택합니다. 그러나 결과 위의 드롭다운 메뉴를 사용하면 사용 가능한 다른 차트 유형 중에서 선택하여 사용하려는 형식으로 데이터를 표시할 수 있습니다. 데이터를 CSV 또는 JSON 형식으로 내보낼 수도 있습니다. 그러면 새 편집기 탭이 열리고 선택한 형식으로 데이터가 표시됩니다.

## NRQL 파일 [#nrql-files]

오른쪽 쿼리 빌더 위의 드롭다운 메뉴를 사용하면 최근 실행한 쿼리 중에서 선택할 수 있습니다. 그러나 쿼리를 저장하는 또 다른 방법은 저장소에 확장자가 `.nrql` 인 파일을 추가하는 것입니다. 쿼리를 이 파일에 저장할 수 있으며 해당 파일을 커밋하고 푸시하여 팀원과 쿼리를 공유할 수도 있습니다. 쿼리를 실행하려면 각 쿼리 위의 CodeLens에서 "실행" 링크를 클릭하기만 하면 됩니다.
