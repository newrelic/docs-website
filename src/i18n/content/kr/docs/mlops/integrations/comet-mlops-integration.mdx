[Comet](https://www.comet.ml/site/) 은 전체 스택 관찰 가능성을 기계 학습 모델로 확장하고 모델 성능을 기반으로 프로덕션 성능 기준을 설정할 수 있게 해주는 [모델 성능 모니터링](/docs/alerts-applied-intelligence/mlops/get-started/intro-mlops/) 플랫폼입니다. Comet 통합을 통해 기계 학습 지표를 New Relic과 통합할 수 있으므로 전체 기계 학습 수명 주기에서 데이터를 지속적으로 모니터링할 수 있습니다. 이는 최적의 모델 성능을 보장하고 더 나은 비즈니스 결과를 달성하는 데 도움이 됩니다.

## Comet을 New Relic과 통합 [#integrate-comet]

New Relic의 Comet 통합을 사용하여 더 우수하고 정확한 머신 러닝 모델을 구축하는 데 도움이 되는 데이터와 통찰력을 얻는 동시에 팀 전체의 생산성, 협업 및 가시성을 개선하십시오.

통합을 설정하려면 다음 단계를 따르세요.

1. <DNT>**Log into your New Relic account**</DNT>: <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > +Add more data**</DNT> 로 이동합니다.

2. <DNT>**Click on Comet**</DNT>: 검색창에 <DNT>**Comet**</DNT> 입력하세요.

3. <DNT>**Select the account ID**</DNT> 당신은 Comet이 통합되기를 원합니다.

4. <DNT>**Select or create an API key**</DNT>: Comet과 통합하려면 뉴렐릭<InlinePopover type="licenseKey"/> 필요합니다. <DNT>**Real time training metric**</DNT> 섹션에서 기존 API 키를 생성하거나 선택합니다.

5. <DNT>**Contact Comet**</DNT>: 통합을 설정하고 Comet에서 모델 생산 모니터링 대시보드를 보는 다음 단계에서는 [Comet에 문의](mailto:sales@comet.ml) 해야 합니다. Comet 팀이 통합 프로세스의 각 단계를 안내하고 모든 설정을 도와드립니다.

   <img
     title="Model Production Monitoring dashboard"
     alt="Model Production Monitoring dashboard."
     src="/images/model-performance-monitoring_screenshot-full_comet-UI.webp"
   />

6. <DNT>**View your data**</DNT>: 이제 통합을 설정했으므로 Comet 팀에서 자동으로 생성하는 전용 [대시보드](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 에서 기계 학습 모델의 성능을 추적할 수 있습니다.
