[Comet](https://www.comet.ml/site/) 은 전체 스택 관찰 가능성을 기계 학습 모델로 확장하고 모델 성능을 기반으로 프로덕션 성능 기준을 설정할 수 있게 해주는 [모델 성능 모니터링](/docs/alerts-applied-intelligence/mlops/get-started/intro-mlops/) 플랫폼입니다. Comet 통합을 통해 기계 학습 지표를 New Relic과 통합할 수 있으므로 전체 기계 학습 수명 주기에서 데이터를 지속적으로 모니터링할 수 있습니다. 이는 최적의 모델 성능을 보장하고 더 나은 비즈니스 결과를 달성하는 데 도움이 됩니다.

## Comet을 New Relic과 통합 [#integrate-comet]

New Relic의 Comet 통합을 사용하여 더 우수하고 정확한 머신 러닝 모델을 구축하는 데 도움이 되는 데이터와 통찰력을 얻는 동시에 팀 전체의 생산성, 협업 및 가시성을 개선하십시오.

통합을 설정하려면 다음 단계를 따르세요.

1. **New Relic 계정에 로그인합니다**. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > +Add more data로** 으로 이동합니다.
2. **혜성을 클릭하십시오 : 검색창에** **혜성** 을 입력하십시오.
3. Comet이 통합할 **계정 ID를 선택하십시오** .
4. **API 키 선택 또는 생성**: Comet과 연동하기 위해서는 New Relic이 필요합니다.<InlinePopover type="licenseKey"/>. **실시간 교육 메트릭** 섹션에서 기존 API 키를 생성하거나 선택합니다.
5. **Comet에 연락** 하기: Comet에서 통합을 설정하고 Model Production Monitoring 대시보드를 보는 다음 단계를 위해 Comet에 [연락](mailto:sales@comet.ml) 해야 합니다. Comet 팀이 통합 프로세스의 각 단계를 안내하고 모든 설정을 할 것입니다.
