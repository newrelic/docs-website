개인 시간대 설정은 몇 가지 예외를 제외하고 New Relic UI에서 대부분의 시간 관련 설정을 제어합니다.

## 기본 시간대 변경 [#exceptions]

New Relic 계정의 기본 시간대를 변경하려면:

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities)** 으로 이동합니다.
2. [사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 를 선택한 다음 **사용자 기본 설정** 을 선택합니다.
3. 시간대를 변경하십시오.
4. 선택 사항:[자동화된 사용자](/docs/accounts/accounts/automated-user-management/automated-user-provisioning-single-sign) 관리를 통해 관리되는 사용자인 경우 ID 공급업체의 시간대를 사용하거나 재정의할 수 있습니다.

시간대를 변경하면 UI에 반영되기까지 최대 24시간이 소요될 수 있습니다.

## 시간대가 적용되지 않는 예외 [#exceptions]

일부 New Relic 기능은 **사용자 기본 설정** 시간대 설정에 의존하지 않습니다. 다음은 협정 세계시(UTC)를 사용하며 사용자 기본 설정의 영향을 받지 않습니다.

* [경고](/docs/alerts-applied-intelligence/new-relic-alerts/get-started/introduction-alerts)
* [REST API v2](/docs/apis/rest-api-v2/requirements/specifying-time-range-v2#utc)

표준 시간대가 기본 표준 시간대 설정에 의존하지 않는 다른 기능이 있을 수 있습니다.
