가격 버전 및 기타 요소에 따라 데이터 수집 감소, 에이전트 또는 통합 삭제 또는 제거, [New Relic 조직](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-account-structure) 다운그레이드 또는 취소를 위한 다양한 옵션을 제공합니다.

## 데이터 수집 줄이기 또는 중지 [#remove-tools]

다운그레이드하거나 취소하지 않고 일부 데이터를 New Relic에 보고하는 것을 중단하려면 더 적은 데이터를 보내도록 도구를 구성할 수 있습니다.

데이터 수집을 관리하는 방법은 데이터 [관리](/docs/telemetry-data-platform/get-data-new-relic/manage-data/manage-your-data) 를 참조하십시오.

에이전트 또는 통합을 제거하려면 다음과 같은 권장 절차를 따르십시오.

* [<InlinePopover type="apm"/>, <InlinePopover type="browser"/>및 <InlinePopover type="mobile"/>앱 제거](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/)
* [인프라 에이전트 제거](/docs/infrastructure/install-infrastructure-agent/update-or-uninstall/uninstall-infrastructure-agent/)
* 다른 New Relic 도구를 비활성화하는 방법은 해당 문서를 참조하십시오. [여기에서 New Relic 솔루션을 검색](https://newrelic.com/instant-observability/) 할 수 있습니다.

## 에디션 다운그레이드 [#downgrading]

사용한 만큼만 지불하는지 여부에 따라 [버전을](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/#editions)다운그레이드할 수 있는 두 가지 옵션이 있습니다.

<CollapserGroup>
  <Collapser
    id="payg-downgrade"
    title="종량제 조직"
  >
    Standard 또는 Pro [종량제](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/#usage-plans) 조직을 다운그레이드하려면:

    1. [사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 에서 <DNT>**Manage your plan**</DNT> 클릭합니다.
    2. <DNT>**Plans and pricing**</DNT> 을(를) 클릭합니다.
    3. <DNT>**Free**</DNT> 요금 옵션에서 <DNT>**Downgrade to free**</DNT> 에 마우스를 놓고 표시되는 버튼을 클릭하세요.
    4. 표시되는 창에서 하단으로 스크롤하여 취소 사유를 입력합니다.
    5. 서비스 약관에 동의한다는 상자를 클릭합니다.
    6. 딸깍 하는 소리 <DNT>**Submit.**</DNT>
  </Collapser>

  <Collapser
    id="contract-downgrade"
    title="계약이 있는 조직"
  >
    [장기 계약이](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/#usage-plans) 있는 Pro 또는 Enterprise 에디션을 가지고 있고 다운그레이드를 원하는 경우 New Relic 계정 담당자에게 문의하십시오.
  </Collapser>
</CollapserGroup>

## 계정 취소 [#cancel-account]

조직에 두 개 이상의 계정이 있고 지금 모든 계정을 폐쇄하지 않으려는 경우 조직을 유지하면서 하나 이상의 계정을 취소할 수 있습니다.

계정을 취소하는 경우:

* 비활성화되어 있지만 조직의 **Accounts > Cancelled accounts** \[계정 &amp;gt; 해지된 계정] 에 계속 표시됩니다.
* 사용자, 그룹 및 사용자 정의 역할은 그대로 유지되지만 취소된 계정에 할당된 액세스 권한 부여로 인해 계정에 대한 모든 연결이 끊어집니다. 또한 이 계정과 연결된 모든 액세스 권한을 제거해야 합니다. 그렇지 않으면 사용자가 해당 계정을 보려고 할 때 오류가 표시될 수 있습니다.
* 취소된 계정과 연결된 모든 API 키는 비활성화되며 다른 계정에 다시 할당할 수 없습니다.

### 계정을 취소할 수 있는 사람 [#who-cancel]

계정을 취소하려면 다음과 같은 사용자여야 합니다.

* 전체 플랫폼 사용자.
* 조직 [관리 설정이](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#admin-settings) 있는 그룹.

### 계정을 취소하는 방법 [#how-to-cancel]

NerdGraph API를 통해서만 계정을 취소할 수 있습니다. UI에서는 계정을 취소할 수 없습니다.

귀하가 계정을 취소할 수 있는 사용자인지 확인한 다음 NerdGraph 튜토리얼 [계정 취소 를](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph) 참조하세요.

## 조직 삭제 [#delete-org]

개별 계정을 취소하는 것 외에도 전체 퀴렐릭 조직을 삭제할 수도 있습니다. 이를 위해서는 약간의 준비가 필요하므로 아래 단계를 검토하세요.

### 조직 삭제 준비 [#preparation-org-delete]

조직을 삭제하기 전에 다음 사항을 확인하세요.

* 귀하의 조직에는 하나의 계정만 있습니다. 조직에 두 개 이상의 계정이 있는 경우:

  1. [Nerdgraph API를](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph) 사용하여 추가 계정을 취소하세요( [계정 취소](#cancel-account) 참조).
  2. 하나의 계정만 남기고 모두 취소한 후 [뉴렐릭 지원팀](/docs/using-new-relic/welcome-new-relic/get-started/find-help-use-support-portal) 에 문의하여 취소된 계정을 모두 제거하세요.

* 귀하의 조직에는 사용자가 한 명뿐입니다. 사용자가 2명 이상인 경우 [사용자 관리 UI](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/#delete-users) 또는 [Nerdgraph를](/docs/apis/nerdgraph/examples/nerdgraph-manage-users/#delete-users) 통해 사용자를 삭제할 수 있습니다.

* 귀하의 조직은 최신 가격 모델을 사용하고 있습니다.

  * 현재 어떤 모델을 사용하고 있는지 확실하지 않은 경우 이를 확인하는 방법은 [가격 책정 모델을](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-pricing-models/#determine-pricing) 참조하세요.
  * 이전 가격 모델을 사용하는 경우 [지원팀](/docs/using-new-relic/welcome-new-relic/get-started/find-help-use-support-portal) 에 문의하세요.

* 귀하의 조직은 무료 버전을 사용하고 있습니다. 즉, 유료 구독으로는 Standard, Pro, 시리즈 에디션을 이용할 수 없습니다. 활성 유료 구독이 있는 경우 이 문서의 위 [버전 다운그레이드를](#downgrading) 참조하세요.

### 조직을 삭제하는 방법 [#how-to-delete-org]

조직 삭제 [요구 사항을](#delete-org) 충족하는 경우 다음 단계를 완료하세요.

1. [사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 에서 <DNT>**Administration**</DNT> 클릭합니다.
2. <DNT>**Access management**</DNT> 을(를) 클릭합니다.
3. <DNT>**Delete organization**</DNT> 클릭합니다. 해당 옵션이 표시되지 않으면 위의 요구 사항을 충족했는지 확인하세요. 추가 도움이 필요하면 [지원팀](/docs/using-new-relic/welcome-new-relic/get-started/find-help-use-support-portal) 에 문의하세요.
