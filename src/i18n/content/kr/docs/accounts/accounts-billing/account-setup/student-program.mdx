하드웨어와 소프트웨어를 수집하고 분석하는 방법을 배우려는 학생이신가요? 학생용 뉴렐릭으로 이 작업을 수행할 수 있습니다. 이는 데이터, 사용자 및 New Relic 기능에 대한 무료 액세스를 제공하는 프로그램입니다.

## 혜택 [#student-benefits]

자격을 갖춘 학생은 다음을 포함하는 학생용 뉴렐릭 계정에 2년 동안 무료로 액세스할 수 있습니다.

* 매월 500GB의 [데이터 수집](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/)
* 3 전체 [플랫폼 사용자](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type/)
* 매월 10,000명의 [신세틱 검사](/docs/synthetics/synthetic-monitoring/getting-started/monitor-limits/)
* [Standard Edition](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/) 고객이 사용할 수 있는 기타 모든 기능에 액세스
* 프로그램 참가자를 위한 월간 뉴스레터, 워크숍, 모임, 이벤트 등 독점 콘텐츠에 액세스

## 자격 요건 [#student-eligibility]

다음 범주 중 하나에 속하면 참여할 수 있습니다.

* 현재 거주 국가의 공인된 대학에 등록되어 있는 학생
* 현재 기술 학교나 코딩 학교 또는 부트캠프에 등록되어 있는 학생
* 대학교, 칼리지 또는 기타 기술 학교에 다니지 않지만 온라인 학습 플랫폼을 사용하여 소프트웨어 개발에 대해 배우고 있는 초기 경력 기술자

다음은 해당되지 않습니다.

* 대학교, 전문대학 및 기타 기술 기관
* 영리사업
* 특수 조직(비영리 [단체를 위한 별도의 프로그램을 이해하려면 O4G(O4G)](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/eligibility-guidelines-new-relic-nonprofit-program/) 참조)

## 가입 방법 [#sign-up]

GitHub Education의 학생 개발자 팩을 통해 다른 학생 개발자 팩 파트너가 제공하는 80개 이상의 리소스와 함께 학생용 뉴렐릭에 액세스할 수 있습니다.

가입하려면 다음을 완료하세요.

1. GitHub Education에 가입했는지 확인하세요.

2. [New Relic에서 무료 계정을](https://newrelic.com/signup) 만드세요.

3. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities)**
   </DNT>

   에 로그인하고 왼쪽 하단에서

   <DNT>
     **Help > Resources > New Relic for Students**
   </DNT>

   클릭합니다.

4. <DNT>
     **GitHub Education**
   </DNT>

   클릭한 후 OAuth를 사용하여 GitHub 학생 계정을 뉴렐릭과 연결하는 단계를 따릅니다.

New Relic 시스템이 GitHub로부터 귀하의 상태를 수신하면 귀하의 계정에서 사용 가능한 추가 데이터와 사용자를 확인할 수 있습니다.
