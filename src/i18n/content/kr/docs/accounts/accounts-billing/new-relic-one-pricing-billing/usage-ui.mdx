귀하가 뉴렐릭 고객이시라면 **Usage Summary** 페이지에서 사용 현황을 확인하실 수 있습니다. 이러한 수치는 대략적인 것이지만, 이러한 견해를 통해 소비량이 청구서에 어떤 영향을 미칠지에 대한 일반적인 아이디어를 얻을 수 있습니다.

사용 UI를 보려면 **[one.newrelic.com](https://one.newrelic.com) &amp;gt; (사용자 이름 클릭) &amp;gt; Administration &amp;gt; Usage Summary** 으로 이동하세요. 이 페이지는 사용 현황에 대한 개요를 보여줍니다.

사용량 UI를 사용하는 것 외에도 쿼리를 실행하여 사용량을 평가할 수도 있습니다. 자세한 내용은 [사용량 데이터 쿼리 및 알림을](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/usage-queries-alerts/) 참조하세요.
