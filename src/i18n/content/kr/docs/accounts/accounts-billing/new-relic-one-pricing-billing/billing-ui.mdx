청구 및 사용 관련 UI를 찾으려면 [one.newrelic.com](https://one.newrelic.com) 으로 이동하십시오. 오른쪽 상단에서 [계정 드롭다운](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#account-dropdown) 을 클릭하여 다양한 계정 설정을 확인합니다. 다음과 같은 결제 관련 UI 옵션이 표시됩니다.

* **요금제 관리: 청구 관련 설정을 보고 관리하기** 위한 옵션입니다.
* **사용량** 보기: [New Relic 조직에 보고된 데이터](/docs/data-apis/manage-data/manage-data-coming-new-relic) 보기. 계정 드롭다운에 이 항목이 표시되지 않으면 **관리** 를 클릭합니다.
* **데이터 관리** : 청구 요인이 될 수 있는 데이터 수집을 관리하기 위한 옵션입니다. [데이터 관리에 대해 자세히 알아보세요.](/docs/telemetry-data-platform/manage-data/manage-data-coming-new-relic)
* **관리** : 청구, 사용 및[사용자 관리](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-managing-users) 와 관련된 UI입니다(이 UI는 [최신 사용자 모델](/docs/accounts/original-accounts-billing/original-users-roles/overview-user-models) 의 사용자에게만 표시됨).
* **계정 설정** : 구독 및 사용 및 [사용자 관리](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 와 관련된 옵션(이 UI는 [원래 사용자 모델](/docs/accounts/original-accounts-billing/original-users-roles/overview-user-models) 의 사용자에게만 표시됨).
