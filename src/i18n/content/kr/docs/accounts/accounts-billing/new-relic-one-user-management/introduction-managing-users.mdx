이 섹션의 문서는 [New Relic One 사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 의 사용자 관리에 관한 것입니다. 원래 사용자 모델 문서는 [원래 사용자 관리](/docs/accounts/original-accounts-billing/original-users-roles) 를 참조하세요.

New Relic One 사용자 모델 사용자 관리에 대한 문서는 다음과 같습니다.

* [사용자 권한 개념](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts) : 사용자가 특정 기능 및 특정 계정에 액세스하는 방법에 대해 알아봅니다.
* [사용자 유형](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type) : 기본 사용자, 핵심 사용자 및 전체 플랫폼 사용자에 대해 알아봅니다.
* [일반적인 사용자 관리 작업](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/) : 사용자 관리 UI를 찾는 위치, 사용자 추가 방법, 사용자 유형 편집, 그룹에 사용자 추가, 사용자 지정 역할 생성 등을 알아봅니다.
* [SAML SSO 및 SCIM 소개](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim): SAML SSO 및 SCIM 옵션에 대한 개요입니다.
* [인증 도메인 설정](/docs/accounts/accounts-billing/new-relic-one-pricing-users/configure-authentication-domains) : "인증 도메인"은 해당 도메인의 사용자가 New Relic에 추가되는 방법(수동으로 SCIM 프로비저닝 대), 해당 사용자가 사용하는 인증 방법(수동 로그인 대 SAML SSO), 사용자가 사용자를 업그레이드하는 방법을 관리하는 것입니다. 유형 및 사용자 세션 설정.
* [사용자 관리 자습서](/docs/accounts/accounts-billing/new-relic-one-user-management/tutorial-add-new-user-groups-roles-new-relic-one-user-model) : 사용자 관리 작업에 대한 보다 심층적인 자습서입니다.
* [사용자 관리 UI 동영상](/docs/accounts/accounts-billing/new-relic-one-user-management/user-mgmt-videos) : 사용자 관리 UI가 작동하는 동영상을 시청하세요.

사용자 수가 청구에 미치는 영향을 이해하고 싶으십니까? [사용자 관련 결제를](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-one-pricing-billing#user-count) 참조하세요.
