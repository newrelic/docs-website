다음은 가장 중요한 사용자 관리 문서 중 일부입니다.

* [사용자 액세스 작동 방식](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts): 사용자가 권한 및 계정에 액세스하는 방법을 알아보세요.
* [사용자 유형](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type) : 기본 사용자, 핵심 사용자 및 전체 플랫폼 사용자에 대해 알아봅니다.
* [UI 및 일반 작업](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-ui-and-tasks/) : 사용자 관리 UI, 사용자 추가 방법, 사용자 유형 편집, 그룹에 사용자 추가, 사용자 지정 역할 생성 및 기타 일반 작업에 대해 알아봅니다.
* [싱글 사인온 및 SCIM 프로비저닝 소개](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim) : SAML SSO 개요 및 ID 제공업체에서 사용자를 관리하는 방법.
* [인증 도메인 설정](/docs/accounts/accounts-billing/new-relic-one-pricing-users/configure-authentication-domains) : &quot;인증 도메인&quot;은 해당 도메인의 사용자가 New Relic에 추가되는 방법(수동으로 SCIM 프로비저닝 대), 해당 사용자가 사용하는 인증 방법(수동 로그인 대 SAML SSO), 사용자가 사용자를 업그레이드하는 방법을 관리하는 것입니다. 유형 및 사용자 세션 설정.
* [사용자 관리 튜토리얼](/docs/accounts/accounts-billing/new-relic-one-user-management/tutorial-add-new-user-groups-roles-new-relic-one-user-model) : 몇 가지 일반적인 사용자 관리 작업에 대한 심층 학습입니다.
* [사용자 관리](/docs/accounts/accounts-billing/new-relic-one-user-management/user-mgmt-videos) 동영상 : 사용자 관리 UI가 작동하는 동영상을 시청하세요.

사용자 수가 청구에 미치는 영향을 이해하고 싶으십니까? [사용자 관련 결제를](/docs/accounts/accounts-billing/new-relic-one-pricing-users/new-relic-one-pricing-billing#user-count) 참조하세요.
