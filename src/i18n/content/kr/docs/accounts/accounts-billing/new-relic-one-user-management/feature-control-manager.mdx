기능 제어 관리자를 사용하면 조직 관리자는 조직 내 계정에 대한 지능형 옵저버빌리티 기능을 활성화하고 관리할 수 있습니다. 이 도구는 사용 가능한 기능과 요금 청구 방식을 중앙에서 제어하여 조직 전체에서 효율적인 관리를 보장합니다.

Control Manager는 GA(General Availability)에 있는 지능형 옵저버빌리티 기능만 표시합니다. 제한적 미리 보기 또는 공개 미리 보기에서 기능을 사용하려면 <DNT>**[Previews &amp; Trials](https://one.newrelic.com/admin-portal/promotion-management/home)**</DNT> 페이지를 참조하세요.

## 사용 사례 [#use-cases]

기능 제어 관리자는 다음과 같은 도움을 줍니다.

* **롤아웃 제어**: 조직 전체가 아닌 특정 계정 전체에서 지능형 옵저버빌리 기능티 기능을 점차적으로 활성화합니다.
* **비용 관리**: 필요할 때만 기능을 활성화하고 비용을 지불하세요.
* **중앙 집중식 관리**: 각 계정을 개별적으로 설정하는 대신 단일 위치에서 기능 접근 권한을 관리하세요.
* **규정 준수 유지**: 승인된 계정만 특정 지능형 운영 체제 기능에 접근할 수 있도록 보장합니다.

## 요구 사항 및 액세스 [#requirements]

기능 제어 관리자를 사용하려면 다음이 필요합니다.

* **관리자 권한**: 계정 설정을 관리할 수 있는 [관리자 권한이](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts) 있어야 합니다.
* **지능형 옵저버빌리티 기능**: 귀하의 조직은 지능형 옵저버빌리티 기능을 구매해야 합니다.

## 기능 제어 관리자에 액세스합니다. [#access]

기능 제어 관리자에 액세스하려면 다음 단계를 따르십시오.

1. [one.newrelic.com](https://one.newrelic.com) 에서 가져옴, 왼쪽 하단에 있는 사용자 이름을 클릭하세요.
2. <DNT>**PLAN MANAGEMENT**</DNT> 섹션에서 <DNT>**Feature Control**</DNT> 선택합니다.

<DNT>**Feature Control**</DNT> 페이지에는 조직에서 사용할 수 있는 모든 지능형 옵저버빌리티 기능이 표시되어 계정 전체의 현재 활성화 상태를 보여줍니다. 특정 계정의 활성화된 기능을 확인하려면 검색 필드에 계정 ID를 입력하여 검색하세요.

<Callout variant="tip">
  대부분의 기능은 특정 계정에 대해 활성화 또는 비활성화할 수 있습니다. 하지만 <DNT>**Teams**</DNT> 과 같은 조직 범위 기능은 조직 내 모든 계정에 자동으로 적용됩니다.
</Callout>

## 특정 계정에 대한 기능을 활성화합니다. [#activate-features]

1. <DNT>**Feature Control**</DNT> 페이지에서 계정에서 활성화하려는 기능을 찾으세요.
2. 해당 기능이 이미 활성화된 경우 <DNT>**Add account**</DNT> 클릭하고, 그렇지 않으면 <DNT>**Select accounts**</DNT> 클릭하세요.
3. 계정 선택 모달에서 해당 기능을 활성화할 계정을 선택하세요.
4. <DNT>**Activate feature**</DNT> 클릭하고 후속 화면에서 작업을 확인하세요.

<Callout variant="important">
  기능 활성화 후, 해당 기능에 대한 요금은 활성화 날짜부터 부과됩니다. 기능을 활성화하기 전에 사용할 준비가 되었는지 확인하십시오.
</Callout>

## 조직 범위 기능 활성화 [#activate-org-features]

조직 내 모든 계정에서 조직 범위 기능(예: <DNT>**Teams**</DNT>)을 활성화하려면 다음 단계를 따르세요.

1. <DNT>**Feature Control**</DNT> 페이지에서 활성화하려는 기능을 찾으세요.
2. <DNT>**Activate**</DNT> 을(를) 클릭합니다.
3. 확인 창에서 청구 정보를 검토하십시오.
4. <DNT>**Activate feature**</DNT> 을(를) 클릭합니다.

<Callout variant="important">
  활성화 후, 해당 기능에 대한 요금은 조직 전체에 걸쳐 즉시 청구됩니다. 기능을 활성화하기 전에 사용할 준비가 되었는지 확인하십시오.
</Callout>

## 활성화된 기능 보기 [#view-features]

활성화 후 기능 제어 관리자에 다음이 표시됩니다.

* **활성**: 기능 기능이 현재 조직에서 활성화되어 있습니다.
* **계정 세부 정보**: 각 기능 아래에 해당 기능이 활성화된 계정을 보여주는 표가 있습니다.
* **활성화 날짜**: 각 계정에 대해 각 기능이 활성화된 날짜입니다.

이를 통해 조직 전체의 기능 사용 현황을 완벽하게 파악할 수 있습니다.

## 특정 계정에 대한 기능 비활성화 [#deactivate-features]

특정 계정에 대해 기능을 비활성화하려면 다음 단계를 따르세요.

1. <DNT>**Feature Control**</DNT> 페이지에서 비활성화하려는 활성 기능을 확장합니다.
2. 이 기능을 비활성화할 계정을 선택하세요.
3. <DNT>**Deactivate**</DNT> 클릭하고 후속 화면에서 작업을 확인하세요.

## 조직 범위 기능 비활성화 [#deactivate-org-features]

조직 범위 기능(예: <DNT>**Teams**</DNT>)의 경우 비활성화 프로세스를 진행하려면 뉴렐릭에 문의해야 합니다.

1. <DNT>**Feature Control**</DNT> 페이지에서 비활성화하려는 활성 기능을 찾으세요.
2. <DNT>**Deactivate for all accounts**</DNT> 클릭하고 확인 모달에서 비활성화를 확인하세요.

## 시간에 따른 기능 접근 권한 관리 [#manage-access]

기능 제어 관리자는 지속적인 관리 기능을 제공합니다.

* **사용량 모니터링**: 어떤 계정이 어떤 기능을 사용하고 있는지 추적합니다.
* **비용 검토**: 활성화 날짜와 계정 사용량을 확인하여 청구에 미치는 영향을 파악하세요.
* **단계적 배포 계획**: 계정 선택을 활용하여 기능 사용 범위를 점진적으로 확장하십시오.
