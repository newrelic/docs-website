일부 New Relic 절차는 계정 ID를 사용해야 합니다(예: 일부 API 호출). 귀하의 계정 ID는 당사가 New Relic 계정에 할당하는 ID 번호입니다. [New Relic 조직에는 여러 계정이 포함될 수 있습니다](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure).

다음은 New Relic 계정의 ID를 찾는 두 가지 방법입니다.

* 조직에 여러 계정이 있는 경우:

  <DNT>
    **[one.newrelic.com](https://one.newrelic.com/all-capabilities)**
  </DNT>

  에서 페이지 상단 근처의 계정 선택기 드롭다운 메뉴를 사용하여 계정 간에 전환하고 ID를 확인하세요.

* <DNT>
    **[one.newrelic.com](https://one.newrelic.com/all-capabilities)**
  </DNT>

  에서 [사용자 메뉴를](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 클릭한 후

  <DNT>
    **Administration > Access management > Accounts**
  </DNT>

  로 이동하여 계정 ID를 확인하세요.

계정 액세스 작동 방식에 대한 자세한 내용은 액세스 [에 영향을 미치는 요소](/docs/accounts/accounts-billing/account-structure/factors-affecting-access-features-data) 를 참조하십시오.
