## 사용자 이름 변경 [#user-name]

사용자 이름을 변경하려면:

1. [계정 드롭다운](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#account-dropdown) 에서 **사용자 기본 설정** 을 선택합니다.
2. 오른쪽에서 **전체 이름** 을 클릭합니다.

사용자가[자동화된 사용자 관리](/docs/accounts/accounts/automated-user-management/automated-user-provisioning-single-sign) 를 통해 관리되는 경우 UI에서 이름을 변경할 수 없습니다.

## 계정 이름 변경 [#account-name]

New Relic 조직은 하나 이상의 계정을 가질 수 있습니다. 계정 이름을 변경하는 방법은 사용 중인 [사용자 모델](/docs/accounts/original-accounts-billing/original-users-roles/overview-user-models) 에 따라 다릅니다.

<CollapserGroup>
  <Collapser
    id="new-relic-one-user-model"
    title="New Relic One 사용자 모델"
  >
    계정을 추가하고 이름을 바꾸려면 [NerdGraph API](/docs/apis/nerdgraph/examples/manage-accounts-nerdgraph) 를 사용하세요.
  </Collapser>

  <Collapser
    id="original-user-model"
    title="원래 사용자 모델"
  >
    계정 이름을 변경하려면:

    1. [계정 드롭다운](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#account-dropdown) 에서 **계정 설정** 을 선택합니다.
    2. 오른쪽에서 **이름** 필드를 편집합니다.
  </Collapser>
</CollapserGroup>
