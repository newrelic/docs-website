New Relic의 세션 구성 기능을 사용하면 사용자의 브라우저 세션이 자동으로 만료되기 전에 유휴 시간에 대한 제한을 설정할 수 있습니다.

## 요구 사항 [#requirements]

[최신 사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 을 사용 중인 경우 [세션 설정](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more/#session-mgmt) 을 참조하세요.

## 개요 [#overview]

세션 구성을 사용하면 사용자의 브라우저 세션이 자동으로 만료되기 전에 유휴 시간에 대한 제한을 설정할 수 있습니다. 시스템이 로그아웃하기 3분 전에 메시지가 나타납니다. 그런 다음 사용자는 계속하려면 다시 로그인해야 합니다. [SAML 싱글 사인온(SSO)](/docs/subscriptions/saml-service-providers) 으로 구성된 계정의 경우 사용자의 브라우저 세션이 재인증되는 빈도를 설정하는 추가 옵션을 사용할 수 있습니다.

사용자 및 제한된 사용자는 자동 시간 초과 기간을 볼 수 있지만 변경할 수는 없습니다. 시간 초과 값을 보려면 **[사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) > Account settings > Authentication > Session configuration** 으로 이동합니다.

## 세션 시간 초과 값 선택 [#procedures]

세션 시간 초과 값을 선택하는 프로세스는 SAML 및 비 SAML 구성 모두에서 동일합니다. 추가 SAML 구성 옵션은 [SAML SSO 브라우저 재인증](/docs/accounts-partnerships/accounts/account-maintenance/set-session-timeouts#saml_reauthentication) 을 참조하십시오.

SAML SSO 계정이 있는 세션 시간 초과에 대해 미리 정의된 기간을 선택하려면 계정 소유자가 SAML SSO 구성 설정에서 이전에 로그아웃 URL을 식별해야 합니다. 이것이 설정되지 않은 경우 계정 관리자는 세션 시간 초과 슬라이드 바를 볼 수 있지만 변경할 수는 없습니다.

관리자가 계정 소유자이기도 한 경우 **세션 구성** 에는 New Relic의 SAML SSO 구성으로 직접 이동하고 로그아웃 URL을 식별하는 링크가 포함됩니다. 자세한 내용은 [SSO 설정을](/docs/subscriptions/setting-up-sso#timeout_url) 참조하십시오.

[원래 사용자 모델](#requirements) 의 사용자에 대해 미리 정의된 세션 시간 초과 기간을 선택하려면 다음을 수행합니다.

1. 이동: **[사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) > Account settings > Authentication > Session configuration** .
2. 슬라이드 바를 사용하여 유휴 세션이 만료되고 자동으로 로그아웃되는 기간을 선택합니다.
3. 선택 사항: 제한된 사용자의 브라우저 세션이 만료되지 않도록 하려면 확인란 옵션을 선택합니다.
4. **변경 사항 저장** 을 선택합니다.

변경 사항은 즉시 적용됩니다.

## SAML SSO 브라우저 재인증 선택 [#saml_reauthentication]

[SAML SSO 인증 브라우저 세션](/docs/subscriptions/setting-up-sso) 이 재인증되도록 미리 정의된 기간을 선택하려면:

1. 이동: **[사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) > Account settings > Authentication > Session configuration** .
2. **SAML 재인증 시간** 슬라이드 바를 사용하여 New Relic이 브라우저 세션을 확인하는 기간을 선택합니다.
3. **변경 사항 저장** 을 선택합니다.

## SAML 시간 초과 후 리디렉션 [#saml_timeout]

SAML용으로 구성된 계정의 세션 유휴 시간 초과로 인해 로그아웃한 경우 New Relic 로그인 페이지로 이동합니다. 귀하의 계정은 SAML용으로 구성되어 있으므로 직접 New Relic에 로그인할 수 없습니다. 인증을 위해 SAML 공급자로 리디렉션하려면:

1. **이메일** 필드에 이메일 주소를 입력합니다.
2. **암호** 필드를 비워 둡니다.
3. **로그인** 버튼을 클릭합니다.

그러면 SAML 공급자로 리디렉션됩니다. 재승인되면 New Relic 웹사이트로 돌아갑니다.
