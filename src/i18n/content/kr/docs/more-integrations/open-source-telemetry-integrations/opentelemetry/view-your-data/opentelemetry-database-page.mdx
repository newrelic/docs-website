OpenTelemetry 데이터를 보내고 UI에서 서비스(엔티티)를 연 후 **데이터베이스** 를 클릭하여 애플리케이션의 데이터베이스 및 캐시 데이터를 볼 수 있습니다. 이 페이지는 개별 데이터베이스 트랜잭션을 정렬 가능한 테이블로 표시하고 작업, 처리량 및 응답 시간을 차트로 표시합니다.

데이터가 이 섹션에 나타나도록 하려면 다음이 포함되어 있는지 확인하세요.
