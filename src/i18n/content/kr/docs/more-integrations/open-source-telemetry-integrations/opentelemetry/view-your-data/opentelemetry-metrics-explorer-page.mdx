OpenTelemetry 데이터를 보내고 UI에서 서비스(엔티티)를 연 후 **메트릭 탐색기** 를 클릭하여 메트릭(선택한 OpenTelemetry 언어의 경우)에 대한 정보를 볼 수 있습니다. 또한 OpenTelemetry와 함께 Prometheus 내보내기를 사용하는 경우 여기에서 메트릭 데이터를 볼 수 있습니다.
