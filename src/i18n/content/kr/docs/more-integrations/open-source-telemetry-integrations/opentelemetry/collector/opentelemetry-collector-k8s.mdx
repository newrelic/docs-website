Kubernetes 클러스터에서 OpenTelemetry로 계측된 앱이 이미 있는 경우 OpenTelemetry Collector를 사용하여 Kubernetes에 대한 메타데이터로 현재 원격 측정을 향상할 수 있습니다. 이를 통해 Kubernetes가 앱에 미치는 영향을 확인할 수 있습니다. 예를 들어 추적 스팬은 관련된 Kubernetes 클러스터, 노드, 포드 및 컨테이너를 표시합니다.

이 작업을 수행하는 방법에 대한 자세한 내용은 [OpenTelemetry 계측 애플리케이션을 Kubernetes에 연결](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-otel-applications-kubernetes) 에서 찾을 수 있습니다.

다른 수집기 사용 사례에 대한 도움말을 찾고 있다면 [newrelic-opentelemetry-examples](https://github.com/newrelic/newrelic-opentelemetry-examples) 리포지토리를 참조하세요.
