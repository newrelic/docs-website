OpenTelemetry 데이터를 보내고 UI에서 서비스(엔티티)를 연 후 **로그** 를 클릭하여 애플리케이션에 대한 데이터를 볼 수 있습니다. New Relic에서 로그 데이터를 애플리케이션에 연결하는 방법에 대한 자세한 내용은 [OpenTelemetry 및 로깅 설명서](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/best-practices/opentelemetry-best-practices-logs) 를 참조하십시오.

데이터가 이 섹션에 나타나도록 하려면 다음이 포함되어 있는지 확인하세요.

* `service.name`
* 추적 데이터와 상관 관계를 지정하려면 로그에 `trace.id` 및 `span.id` 속성이 포함되어야 합니다.

다른 UI 페이지의 OpenTelemetry에 대해 알아보려면 [UI 개요](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/view-your-data/opentelemetry-view-your-data) 를 참조하세요.
