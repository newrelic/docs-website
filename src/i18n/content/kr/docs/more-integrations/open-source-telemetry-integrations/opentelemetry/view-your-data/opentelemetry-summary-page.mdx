OpenTelemetry 데이터를 보내고 UI에서 서비스(엔티티)를 연 후 **요약** 페이지에서 서비스 개요를 볼 수 있습니다. 응답 시간, 처리량 및 오류율과 같은 주요 모니터링 세부 정보인 엔터티에 대한 다양한 황금 신호를 나열합니다. 이 정보를 사용하여 더 깊이 파고들 필요가 있는지 빠르게 결정할 수 있습니다.

OpenTelemetry 데이터가 이 섹션에 표시되도록 하려면 다음이 포함되어 있는지 확인하세요.
