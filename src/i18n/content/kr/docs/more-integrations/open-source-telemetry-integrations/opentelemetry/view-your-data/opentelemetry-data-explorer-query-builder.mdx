선별된 보기에서 OpenTelemetry 데이터를 보는 것 외에도 사용자 지정 쿼리를 구성하여 데이터를 볼 수도 있습니다. **메트릭 및 이벤트** 와 **쿼리 빌더** 페이지는 데이터를 파헤치는 데 도움이 될 수 있습니다.

## 지표 및 이벤트 [#general-metrics-traces]

**Metrics & events** 라는 도구를 사용하면 NRQL 쿼리 언어를 직접 사용하지 않고도 메트릭, 로그 및 추적의 쿼리를 구성할 수 있습니다.

[one.newrelic.com/all-capabilities](https://one.newrelic.com/all-capabilities) 에서 **메트릭 및 이벤트** 타일을 클릭하여 이 쿼리 도구를 찾을 수 있습니다. 이를 사용하는 방법에 대한 팁은 [지표 및 이벤트 데이터 탐색기 소개를](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer) 참조하십시오.

## 쿼리 빌더 [#query-builder]

NRQL을 사용하여 자체 데이터베이스 쿼리를 작성하려는 경우 쿼리 빌더라는 도구를 사용할 수 있습니다. [one.newrelic.com/all-capabilities](https://one.newrelic.com/all-capabilities) 에서 **데이터 쿼리** 타일을 클릭하여 이 도구를 찾을 수 있습니다. NRQL로 쿼리를 구성하는 방법에 대한 자세한 내용은 [New Relic의 데이터 쿼리 소개](/docs/query-your-data/explore-query-data/explore-data/introduction-querying-new-relic-data) 및 [NRQL 소개를](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language) 참조하십시오.

다른 UI 페이지의 OpenTelemetry에 대해 알아보려면 [UI 개요](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/view-your-data/opentelemetry-view-your-data) 를 참조하세요.
