OpenTelemetry 데이터를 보내고 UI에서 서비스(엔티티)를 연 후 **외부 서비스** 를 클릭하여 웹 서비스, 클라우드의 리소스 및 기타 네트워크 엔터티와 같은 out-of-process 서비스에 대한 호출을 볼 수 있습니다. .

데이터가 이 섹션에 나타나도록 하려면 다음이 포함되어 있는지 확인하세요.

* `span.kind = client` 또는 `producer`
* `db.system` 존재하지 않는다
* `http.url` 또는 `net.peer.name` 이(가) 있어야 합니다.

자세한 내용은 [외부 서비스](/docs/apm/apm-ui-pages/monitoring/external-services/external-services-intro) 를 참조하십시오.

다른 UI 페이지의 OpenTelemetry에 대해 알아보려면 [UI 개요](/docs/more-integrations/open-source-telemetry-integrations/opentelemetry/view-your-data/opentelemetry-view-your-data) 를 참조하세요.
