[Kamon](https://kamon.io/) 은 Scala 및 Akka 애플리케이션의 자동 계측을 전문으로 하는 계측 툴킷을 제공합니다. 메트릭 및 추적 계측용 API와 애플리케이션 메트릭 및 분산 추적을 생성하는 자동 계측 모듈로 구성됩니다.

Kamon은 원격 측정 데이터를 원하는 백엔드로 전송하기 위한 리포터 모듈도 제공합니다. New Relic의 Kamon 리포터는 Kamon 번들에 포함되어 있으며 Kamon 계측 애플리케이션에서 New Relic 계정으로 원격 측정 데이터를 보내도록 구성할 수 있습니다.

## 카몬 리포터 활성화 [#install]

Kamon 리포터를 활성화하려면:

1. 아직 없는 경우 [새 리릭 계정을 작성하십시오](https://newrelic.com/signup). 무료입니다. 영원히요.

2. [Kamon 리포터 설치 지침](https://kamon.io/docs/latest/reporters/newrelic/) 을 따르십시오.

3. 선택 사항: 기본 Kamon 추적 샘플링을 사용하는 대신 [무한 추적](/docs/introduction-infinite-tracing) 기능을 활성화할 수 있습니다.이 옵션을 사용하는 경우 일반적으로 모든 추적 데이터를 보내도록 Kamon을 구성할 수 있습니다( [샘플링에 대해 자세히 알아보기](/docs/distributed-tracing/concepts/how-new-relic-distributed-tracing-works) ).무한 추적을 활성화하려면:

   1. New Relic UI에서 [추적 관찰자를 설정합니다](/docs/infinite-tracing-set-trace-observer#set-up) .
   2. 추적 관찰자에게 데이터를 보내도록 Kamon 리포터를 구성합니다. `kamon.newrelic` 구성 블록에서 [`span-ingest-uri`](https://kamon.io/docs/latest/reporters/newrelic/) 값을 [YOUR_TRACE_OBSERVER_URL](/docs/infinite-tracing-set-trace-observer#ui-endpoints) 로 설정합니다.

## 데이터 찾기

데이터를 찾으려면 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > All entities** 로 이동하십시오. 이름으로 서비스를 검색하십시오. Kamon Instrumentation에서 제공하는 가장 유용한 지표 중 일부에 대한 차트가 포함된 즉시 사용 가능한 개요를 제공합니다. 분산 추적 데이터도 보내는 경우 **Distributed tracing** \[분산 추적] 기능을 사용하여 추적을 쿼리하고 볼 수 있습니다.

기본 개요를 시작점으로 사용하여 고유한 대시보드를 구축할 수 있습니다. 차트에서 `...` 아이콘을 클릭하여 해당 차트를 대시보드에 추가하거나 차트 뒤의 쿼리를 확인하여 수정하고 대시보드에 추가합니다.

New Relic에 있는 데이터를 쿼리하는 방법에 대한 자세한 내용은 데이터 [쿼리](/docs/using-new-relic/data/understand-data/query-new-relic-data) 및 [NRQL 소개를](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) 참조하세요.
