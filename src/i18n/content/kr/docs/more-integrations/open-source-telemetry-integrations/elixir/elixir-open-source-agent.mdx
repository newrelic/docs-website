Elixir 오픈 소스 에이전트를 사용하여 New Relic으로 [Elixir](https://github.com/newrelic/elixir_agent) 동작을 모니터링합니다. 에이전트:

* 트랜잭션, 분산 추적 및 애플리케이션 동작의 기타 부분을 추적하는 데 도움이 됩니다.
* 기본 [BEAM](http://erlang.org/faq/implementations.html#idp32950544) 활동에 대한 개요를 제공합니다.

<Callout variant="tip">
  이 에이전트는 [GitHub](https://github.com/newrelic/) 에서 오픈 소스로 릴리스됩니다. 최신 업데이트에 대한 변경 로그도 제공됩니다.
</Callout>

## 시작하다 [#get_started]

요구 사항, 설치 및 구성 정보는 GitHub의 [오픈 소스 Elixir 에이전트 README](https://github.com/newrelic/elixir_agent/blob/master/README.md) 를 참조하십시오.

설치, 사용 또는 기타 주제에 대한 질문은 [GitHub에서 New Relic의 Elixir 저장소를](https://github.com/newrelic/elixir_agent) 방문하십시오. 문제 또는 버그를 [GitHub 저장소](https://github.com/newrelic/elixir_agent/issues)의 문제로 보고하세요.
