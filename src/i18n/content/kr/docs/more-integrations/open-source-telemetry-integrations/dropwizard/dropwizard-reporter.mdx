[Dropwizard](https://www.dropwizard.io/en/latest/) 는 즉시 사용할 수 있는 메트릭 계측을 제공하여 작업 친화적인 고성능 RESTful 웹 서비스를 개발하기 위한 Java 프레임워크입니다. New Relic의 DropWizard 리포터는 Dropwizard 메트릭을 New Relic 계정으로 보냅니다.

dropwizard 리포터를 사용해 보고 싶으십니까? 무료 [로 New Relic 계정을 만드](https://newrelic.com/signup) 십시오! 신용 카드가 필요하지 않습니다.

Dropwizard 기자를 위한 리소스:

* [읽어보기](https://github.com/newrelic/dropwizard-metrics-newrelic)
* [통합 사양](https://github.com/newrelic/newrelic-exporter-specs/tree/master/dropwizard)

## 데이터 찾기

데이터를 찾으려면 [one.newrelic.com](https://one.newrelic.com/ "링크가 새 창에서 열립니다.") 으로 이동하여 **Your applications** > **Explorer** 로 이동합니다. 엔터티 화면에서 이름으로 서비스를 검색합니다. 여기에서 **데이터 탐색기** 를 사용하여 메트릭을 탐색하고 메트릭을 사용하여 대시보드를 작성할 수 있습니다.

New Relic에 있는 데이터를 쿼리하는 방법에 대한 자세한 내용은 데이터 [쿼리](/docs/using-new-relic/data/understand-data/query-new-relic-data) 및 [NRQL 소개를](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) 참조하세요.
