Grafana에서 New Relic을 Prometheus 데이터 소스로 구성할 수 있습니다. 뿐만 아니라 Grafana 내에서 [PromQL 쿼리 언어](/docs/integrations/prometheus-integrations/view-query-data/supported-promql-features) 를 사용하여 New Relic에 저장된 메트릭을 쿼리할 수 있습니다.

## New Relic과 함께 기존 Grafana 대시보드 사용

[원격 쓰기](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#remote-write), [Prometheus 에이전트](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#agent) 또는 [OpenMetrics 통합](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#OpenMetric) (2.0+)을 통해 Prometheus 메트릭을 New Relic과 통합하고 Grafana에서 New Relic을 Prometheus 데이터 소스로 구성하면 기존 Grafana를 사용할 수 있습니다. <InlinePopover type="dashboards"/>우리가 제공하는 추가 모니터링, 안정성 및 규모를 원활하게 활용할 수 있습니다.

## 호환성 및 요구 사항

시작하기 전에 Prometheus 메트릭 통합을 완료하고 최신 버전의 Grafana를 실행하고 있는지 확인하십시오.

* Grafana에서 New Relic Prometheus 데이터 소스를 구성하려면 먼저 [원격 쓰기](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#remote-write) ,[Prometheus 에이전트](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#agent) 또는 [OpenMetrics 통합](/docs/integrations/prometheus-integrations/get-started/monitor-prometheus-new-relic#OpenMetrics) (v2.0+)이 설정되어 있어야 합니다.
* Grafana 버전 6.7.0 이상에서만 이 방법을 사용하여 New Relic Prometheus 데이터 소스를 구성할 수 있습니다. UI에서 사용자 정의 헤더를 구성해야 하며 이전 버전에서는 불가능합니다. 자세한 내용은 [새 유물을 Grafana용 Prometheus 데이터 원본으로 구성](/docs/integrations/grafana-integrations/set-configure/configure-new-relic-prometheus-data-source-grafana) 을 참조하십시오.

### PromQL 지원

Prometheus API는 Prometheus의 쿼리 API를 에뮬레이트합니다. PromQL 스타일 쿼리 모드를 통해 Prometheus 쿼리 언어(PromQL)를 지원합니다. PromQL 구문 쿼리를 가장 가까운 NRQL 근사값으로 자동 변환하기 위해 최선을 다합니다.

작동 방식 및 Prometheus와 New Relic 간의 차이점에 대한 자세한 내용은 [지원되는 PromQL 기능](/docs/integrations/prometheus-integrations/view-query-data/supported-promql-features) 을 참조하십시오.

## Grafana에서 데이터 흐름 가져오기

Grafana에서 New Relic 데이터를 사용할 수 있도록 하려면 몇 가지 간단한 단계로 새 Prometheus 데이터 소스 또는 기존 Prometheus 데이터 소스를 구성할 수 있습니다.

1. Grafana UI에서 [새 데이터 소스를 추가하고 구성합니다](/docs/integrations/grafana-integrations/set-configure/configure-new-relic-prometheus-data-source-grafana) .
2. 새 데이터 소스를 저장하고 데이터 보기를 시작합니다.
