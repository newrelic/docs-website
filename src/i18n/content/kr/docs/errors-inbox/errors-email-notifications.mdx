오류 받은 편지함은 이메일 알림을 사용하여 오류 그룹에 할당되었음을 알려줍니다.

## 이메일 알림 [#notifications]

오류 그룹이 사용자 계정에 할당되면 New Relic은 해당 오류 그룹의 세부 정보가 포함된 이메일을 보냅니다.

이메일 알림은 다음과 같은 경우에 트리거됩니다.

* 오류 그룹이 사용자에게 할당 및 할당 해제됩니다.
* 오류 그룹 상태(미해결, 해결, 무시)가 변경되고 오류 그룹에 할당된 사용자에게 알림이 전송됩니다.

현재 팀원들에게 이메일 공지가 발송되었습니다. 포함하려는 새 회원이 있는 경우 왼쪽 메뉴에서 <DNT>**Add user**</DNT> 클릭합니다.

### 이메일 알림 수신 거부 [#unsubscribe]

받은 편지함 이메일 알림 오류를 원하지 않는 경우 언제든지 구독을 취소할 수 있습니다. 수신하는 각 이메일 알림에는 수신 거부 옵션이 포함되어 있습니다.

<img
  title="Unassigned user email example"
  alt="A screenshot of an example email someone sees when they're unassigned from an error group. "
  src="/images/errors-inbox_screenshot-crop_unassigned-message.webp"
/>

공지 이메일 하단의 <DNT>**Unsubscribe**</DNT> 링크를 클릭하세요.

### 이메일 알림 재구독 [#resubscribe]

공지 이메일 구독을 취소하고 재구독이 필요하다고 결정한 경우 기존의 무스 인박스(errors inbox) 이메일 공지를 찾아 <DNT>**Unsubscribe preferences**</DNT> 링크를 클릭하세요.

이메일 공지 제목은 "<DNT>**\[Errors Inbox]**</DNT>"으로 시작합니다.
