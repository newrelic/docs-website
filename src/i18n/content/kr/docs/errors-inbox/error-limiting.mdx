제한된 수의 계정에서 오류 받은 편지함을 사용하려고 할 때 다음과 같은 경고를 받을 수 있습니다.

### 문제 [#problem]

```
You’ve reached the daily limit of unique error groups on account xxx.  This limit will be reset at 00:00 UTC.
```

### 해결책 [#solution]

오류 그룹 수를 줄이려면 오류 클래스 또는 메시지에서 고유 식별자를 제거해 보십시오. [자세한 내용은 여기를 참조](/docs/errors-inbox/errors-inbox/#similar-events)하세요.

* * *

### 문제 [#problem]

```
You’ve reached the daily maximum unique time series (cardinality) limit on account xxx.  This limit will be reset at 00:00 UTC.
```

이 제한에 도달하면 오류 받은 편지함이 더 이상 업데이트되지 않고 오류가 정체되거나 감소하는 것처럼 보입니다.

### 해결책 [#solution]

이 문제를 해결하려면 [워크로드 범위를 줄이십시오](/docs/new-relic-one/use-new-relic-one/workloads/use-workloads/) .
