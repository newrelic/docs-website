터미널에서 진단 CLI를 실행하면 각 작업이 완료되는 결과가 표시됩니다. `Warning` 또는 `Failure` 상태 코드가 발생하는 작업은 문제 해결 제안 및 문서에 대한 관련 링크와 함께 실행 중에 발견된 가능한 문제에 대한 추가 세부정보를 기록합니다. [-attach 또는 -API 키](/docs/new-relic-solutions/solve-common-issues/diagnostics-cli-nrdiag/pass-command-line-options-nrdiag) 플래그를 사용하면 [UI](/docs/using-new-relic/cross-product-functions/diagnostics-cli-nrdiag/interpret-nrdiag-output/#diag-in-nr1)에서 결과를 볼 수 있습니다.

## 파일 출력

진단 CLI는 세 개의 파일을 출력합니다.

<table>
  <thead>
    <tr>
      <th style={{ width: "200px" }}>
        진단 출력 파일
      </th>

      <th>
        코멘트
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `nrdiag-output.zip`
      </td>

      <td>
        하나 이상의 구성 파일과 New Relic의 기존 로그가 있는 평면 폴더 구조. 이 파일을 New Relic 계정에 첨부하십시오. 이 아카이브에는 `nrdiag-output.json` 의 사본도 포함되어 있습니다.
      </td>
    </tr>

    <tr>
      <td>
        `nrdiag-output.json`
      </td>

      <td>
        개별 테스트의 출력. 이 출력을 New Relic 계정에 첨부하십시오.
      </td>
    </tr>

    <tr>
      <td>
        `nrdiag-filelist.txt`
      </td>

      <td>
        찾은 파일 목록입니다. New Relic 계정에 첨부한 `nrdiag-output.zip` 파일에는 이 목록이 자동으로 포함됩니다. 이 `.txt` 파일을 별도로 첨부할 필요가 없습니다.
      </td>
    </tr>
  </tbody>
</table>

## 결과 상태 코드 [#status-codes]

진단 CLI는 실행 후 다음 상태 코드를 반환합니다.

<table>
  <thead>
    <tr>
      <th style={{ width: "125px" }}>
        상태 코드
      </th>

      <th>
        정의
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        `Success`
      </td>

      <td>
        감지된 문제 없이 작업이 성공적으로 실행되었습니다.
      </td>
    </tr>

    <tr>
      <td>
        `Warning`
      </td>

      <td>
        작업이 성공적으로 실행되었습니다. 그러나 가능한 문제가 감지되었습니다.
      </td>
    </tr>

    <tr>
      <td>
        `Failure`
      </td>

      <td>
        문제가 감지되어 작업이 실패했습니다.
      </td>
    </tr>

    <tr>
      <td>
        `Error`
      </td>

      <td>
        작업을 실행할 수 없습니다. 권한 문제로 인한 것일 수 있습니다.
      </td>
    </tr>

    <tr>
      <td>
        `None`
      </td>

      <td>
        작업이 감지된 환경과 관련이 없다고 판단되어 실행되지 않았습니다.
      </td>
    </tr>
  </tbody>
</table>

## New Relic의 진단 CLI 출력 [#diag-in-nr1]

<img
  title="Diagnostics CLI Output application in New Relic"
  alt="Diagnostics CLI Output application in New Relic"
  src="/images/solutions_screenshot-crop_nerdlet-diag.webp"
/>

UI에서 `nrdiag-output.json` 출력을 검사할 수 있습니다. 선택한 계정은 업로드를 위해 제공한 진단 CLI 또는 API 키로 검증된 라이센스 키와 연결됩니다.

1. [UI](https://one.newrelic.com) 오른쪽 상단에 있는 물음표를 클릭하세요.

2. <DNT>
     **Diagnostics Home**
   </DNT>

   탭을 선택합니다.

3. 진단 CLI 출력 섹션에서

   <DNT>
     **View Run Details**
   </DNT>

   클릭합니다.

4. 그러면 최근 실행이 표시됩니다. 왼쪽에서 이전 실행을 선택할 수 있습니다.

5. 각 실행에 대해 경고 및 실패를 주요 초점으로 하는 각 작업의 상태를 볼 수 있습니다. CLI가 찾은 로그 또는 스크립트 출력도 볼 수 있습니다.
