## 파일 출력

진단 CLI는 세 개의 파일을 출력합니다.

      </td>
    </tr>

    <tr>
      <td>
        `nrdiag-output.json`
      </td>

      <td>
        개별 테스트의 출력. 이 출력을 New Relic 계정에 첨부하십시오.

    </tr>

    <tr>
      <td>
        `nrdiag-filelist.txt`
      </td>

      <td>
        찾은 파일 목록입니다. New Relic 계정에 첨부한 `nrdiag-output.zip` 파일에는 이 목록이 자동으로 포함됩니다. 이 `.txt` 파일을 별도로 첨부할 필요가 없습니다.
      </td>
    </tr>
  </tbody>
</table>

## 결과 상태 코드 [#status-codes]

진단 CLI는 실행 후 다음 상태 코드를 반환합니다.

  <thead>

      <td>

## New Relic의 진단 CLI 출력 [#diag-in-nr1]

<img
  title="Diagnostics CLI Output application in New Relic"
  alt="Diagnostics CLI Output application in New Relic"
  src={solutionsNerdletDiag}
/>

UI에서 `nrdiag-output.json` 출력을 검사할 수 있습니다. 선택한 계정은 업로드를 위해 진단 CLI에서 검증한 라이선스 키와 연결되어 있습니다.

1. [UI 에서](https://one.newrelic.com) 앱을 선택한 다음 **진단 CLI 출력** 을 검색합니다. 또는 이 [**진단 CLI 출력** 영구 링크](https://onenr.io/0x0jldL3EjW) 를 클릭하여 페이지로 직접 이동합니다.
2. 선택 사항: 아이콘 옆에 있는 별표를 클릭하여 즐겨찾기에 추가합니다.
3. **진단 CLI 출력** 드롭다운에서 계정을 선택합니다.
4. 선택한 계정에 대해 이전 진단 CLI 실행 목록에서 선택합니다.
5. 각 작업에 대해 결과 목록을 검토하고 필요에 따라 필터링합니다. 버튼을 클릭하면 상태 코드별로 특정 결과가 나열됩니다.
6. 페이지 오른쪽에서 결과의 원시 JSON을 검토합니다.
