일부 [New Relic 솔루션](https://newrelic.com/instant-observability/) 은 에이전트를 수동으로 설치해야 합니다. 대부분의 에이전트에는 문제 해결, 감사 및 진단을 위한 자세한 로그를 생성하는 도구가 내장되어 있습니다.

관련 문서:

* 일반적인 에이전트 문제 해결은 [데이터가 표시되지 않음을](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data) 참조하십시오.
* [New Relic Diagnostics](/docs/using-new-relic/cross-product-functions/troubleshooting/new-relic-diagnostics) 에 대해 알아보세요. 일반적인 문제를 자동으로 감지하는 유틸리티입니다.
