## 문제

[New Relic 에이전트를 설치한](/docs/agents/manage-apm-agents/installation/install-agent) 후 UI에 데이터가 표시되지 않습니다.

## 해결책

New Relic 에이전트를 설치하고 앱에 대한 트래픽을 생성한 후 몇 분 이내에 데이터를 보기 시작해야 합니다. 데이터가 표시되지 않으면 [New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics) 유틸리티를 사용하여 일반적인 문제를 자동으로 식별할 수 있습니다. 추가 문제 해결 팁은 에이전트별 문서를 참조하세요.

### APM 에이전트

해당 문제 해결 절차를 따르십시오. <InlinePopover type="apm"/>대리인:

* [가다](/docs/agents/go-agent/troubleshooting/no-data-appears-go)
* [자바](/docs/agents/java-agent/troubleshooting/no-data-appears-java)
* [.그물](/docs/agents/net-agent/troubleshooting/no-data-appears-net)
* [노드.js](/docs/agents/nodejs-agent/troubleshooting/troubleshooting-your-nodejs-installation)
* [PHP](/docs/agents/php-agent/troubleshooting/no-data-appears-php)
* [파이썬](/docs/agents/python-agent/troubleshooting/no-data-appears-python)
* [루비](/docs/agents/ruby-agent/troubleshooting/no-data-appears-ruby)

또한 모든 APM 에이전트에 적용되는 다음 문제 해결 단계를 시도할 수 있습니다.

<CollapserGroup>
  <Collapser
    id="deleted_apps"
    title="APM에서 삭제되거나 이름이 변경된 애플리케이션"
  >
    해당 이름을 재사용하려면 앱에서 최소 1시간 동안 데이터 보고를 중지해야 합니다. 또한 새 데이터가 수락되기 전에 New Relic 수집기와 다시 연결해야 합니다(다시 시작해야 함).

    앱은 플러시되기 전에 한 시간 동안 수집기의 캐시에 남아 있습니다. 이 시간 동안에는 "삭제됨"으로 표시되므로 새 데이터가 허용되지 않습니다. 또한 에이전트가 다시 시작될 때까지 삭제된 실행 중인 앱과 데이터가 연결됩니다.

    자세한 내용은 다음을 참조하세요.

    * [애플리케이션 이름 지정](/docs/site/naming-your-application)
    * [앱에 여러 이름 사용](/docs/site/using-multiple-names-for-an-app)
  </Collapser>

  <Collapser
    id="data_collection"
    title="수집기에 연결되지 않음"
  >
    New Relic 에이전트가 [수집기](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector) 에 연결할 수 없는 경우 앱은 영향을 받지 않습니다. 데이터는 계속 수집되며 연결이 복구되는 즉시 업로드됩니다.

    네트워크가 다운되거나 수집기를 사용할 수 없는 동안 APM CPU 및 메모리 차트에서 데이터가 누락된 간격을 볼 수 있습니다. 에이전트는 계속해서 재연결을 시도하고 성공하면 UI에 데이터가 다시 표시됩니다.

    에이전트가 수집기와 통신할 수 없는 동안에는 여전히 데이터를 수집하고 있습니다. 다시 연결할 수 있게 되면 데이터를 업로드하고 누락된 세그먼트를 채우므로 애플리케이션이 다운되었는지 또는 단순히 데이터를 보고하지 않았는지에 대한 혼동이 없을 것입니다. 메모리를 절약하기 위해 해당 기간 동안 데이터가 집계 및 평균화되므로 수집기와 통신할 수 없는 기간 동안의 평평한 막대와 차트가 표시됩니다.
  </Collapser>
</CollapserGroup>
