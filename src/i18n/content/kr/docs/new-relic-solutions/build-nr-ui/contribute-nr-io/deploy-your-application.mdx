퀵스타트를 구축하기 전에 데모 서비스를 시작해야 합니다.

## 데모 서비스 가동 [#spin-demo-services]

GitHub에서 랩 저장소를 복제합니다.

```bash
    git clone https://github.com/newrelic-experimental/build-a-quickstart-lab.git
```

`flashDB` 라는 데모 디렉토리로 변경합니다.

```bash

cd build-a-quickstart-lab/flashDB

```

이 디렉터리에는 두 가지 중요한 서비스에 대한 코드가 포함되어 있습니다.

* 데이터베이스를 모방하고 CRUD(생성, 읽기, 업데이트 및 삭제) 작업을 위한 기능도 제공하는 Python 프로그램입니다.
* CRUD 작업을 수동으로 수행할 필요가 없도록 더미 데이터베이스 트래픽을 생성하는 시뮬레이터 서비스

다음 명령을 실행하여 서비스를 실행합니다.

```Python
python simulator.py
```

여기서는 데이터베이스에 대한 모의 트래픽을 생성하는 시뮬레이터를 실행합니다. 시뮬레이터가 실행되면 터미널에 다음 출력이 표시됩니다.

```bash animate
python simulator.py
[output] Writing...
[output] try_send
[output] Writing...
[output] try_send
[output] Reading...
[output] try_send
[output] Reading...
[output] try_send
[output] Writing...
[output] try_send
[output] Writing...
[output] try_send
[output] Reading...
```

다음으로, [스프레드시트를](https://github.com/newrelic/newrelic-telemetry-sdk-python) 사용하여 flashDB 서비스의 스프레드시트 데이터를 뉴렐릭으로 보냅니다.
