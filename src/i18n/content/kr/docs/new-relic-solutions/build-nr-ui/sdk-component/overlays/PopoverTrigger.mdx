`<Popover>` 구성요소의 하위 요소입니다.

팝오버 오버레이의 열기/닫기를 제어합니다.

### 용법

```js
import { PopoverTrigger } from 'nr1'
```

### 소품

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>필수</h5> <h5>노드|함수</h5>
      </td>

      <td>
        이 구성요소는 하위 React 요소를 직접 전달하거나 렌더링 콜백(하위 요소로 함수)을 전달하는 두 가지 방법으로 렌더링할 수 있습니다. React 요소를 직접 전달할 때는 `<TextField>` 과 같은 호환되는 NR1 SDK 구성요소 중 일부를 사용할 수 있습니다. , `<Button>`... 또는 자체 구성 요소를 사용합니다. 렌더 콜백을 사용하면 다음 모양의 객체를 인수 `{ opened: boolean, controlled: boolean }` 로 받게 됩니다. 구성 요소를 직접 전달하든 렌더링 콜백을 사용하든 관계없이 자체 구성요소를 트리거로 렌더링할 때 제대로 작동하려면 항상 `focus` 메서드를 노출해야 합니다.
