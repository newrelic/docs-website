테이블 차트를 생성합니다. 특정 계정에 대해 NRQL 쿼리를 수행하여 데이터를 얻거나 `data` 소품을 통해 전달할 수 있습니다.

`data` 형식은 각각 `metadata` 및 `data` 포함하는 일련의 객체입니다. 각 `data` 에는 이 지점의 값과 차트를 그리는 데 사용되는 `x` 및 `y` 키가 모두 포함되어 있습니다.

### 용법

```js
import { TableChart } from 'nr1'
```

### 예

#### 기초적인

```js
<TableChart accountIds={[1]} query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType" />
```

#### 여러 계정으로

```js
<TableChart accountIds={[1, 1067061]} query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType" />
```

#### 채우기 컨테이너

```js
<TableChart
  accountIds={[1]}
  query="SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType"
  fullWidth
  fullHeight
/>
```

#### 맞춤 데이터 포함

```js
function render() {
  const data = [
     {
       metadata: {
         id: 'series-1',
         name: 'Serie 1',
         color: '#008c99',
         viz: 'main',
         columns: ['jobType', 'count'],
       },
       data: [
         {
           jobType: 'SIMPLE',
           count: 18,
         },
         {
           jobType: 'SCRIPT_BROWSER',
           count: 27,
         },
         {
           jobType: 'SCRIPT_API',
           count: 33,
         },
         {
           jobType: 'BROWSER',
           count: 29,
         },
         {
           jobType: 'SELENIUM_JSON',
           count: 22,
         },
         {
           jobType: 'SCRIPT',
           count: 31,
         },
       ],
     },
   ];;


  return <TableChart data={data} fullWidth />;
}
```
