`<Tabs>` 구성요소의 하위 요소입니다.

탭 제목이 `label` 소품으로 정의되고 탭 내용이 `children` 소품으로 정의되는 탭을 만듭니다.

### 용법

```js
import { TabsItem } from 'nr1'
```

### 소품

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>필수</h5> <h5>노드</h5>
      </td>

      <td>
        탭을 선택할 때 표시할 콘텐츠입니다.
      </td>
    </tr>

    <tr>
      <td>
        `className` <h5>끈</h5>
      </td>

      <td>
        구성 요소에 클래스 이름을 추가합니다. 위치 지정 및 간격 지정 목적으로만 사용해야 합니다.
      </td>
    </tr>

    <tr>
      <td>
        `disabled` <h5>부울</h5>
      </td>

      <td>
        `true` 인 경우 상호작용에 탭을 사용할 수 없습니다.
      </td>
    </tr>

    <tr>
      <td>
        `label` <h5>필수</h5><h5>문자열</h5>
      </td>

      <td>
        탭의 제목입니다.
      </td>
    </tr>

    <tr>
      <td>
        `style` <h5>물체</h5>
      </td>

      <td>
        사용자 정의 스타일을 위한 인라인 스타일입니다. 탭 콘텐츠에 적용됩니다.
      </td>
    </tr>

    <tr>
      <td>
        `testId` <h5>끈</h5>
      </td>

      <td>
        `data-test-id` 속성을 추가합니다. 이를 사용하여 단위 및 E2E 테스트의 구성 요소를 목표로 삼습니다. 테스트 ID가 유효하려면 앞에 nerdpack ID를 붙이고 그 뒤에 점이 옵니다. 예를 들어 `my-nerdpack.some-element`.

        **참고:** `data-test-id` 속성이 DOM 에서 제거될 때 표시되지 않을 수 있습니다. 이를 디버그하려면 `e2e-test` 쿼리 변수를 URL에 전달해야 합니다.
      </td>
    </tr>

    <tr>
      <td>
        `value` <h5>필수</h5><h5/>
      </td>

      <td>
        탭의 고유 식별자입니다. 활성 탭을 선택할 때 사용됩니다.
      </td>
    </tr>
  </tbody>
</table>
