`<Grid>` 구성요소의 하위 요소입니다.

기본적으로 하나의 열에 걸쳐 있지만 `columnSpan` 또는 `columnStart` 및 `columnEnd` 소품을 설정하여 걸쳐야 하는 열 수를 지정할 수 있습니다.

### 용법

```js
import { GridItem } from 'nr1'
```
