GraphQL 쿼리 문자열을 NerdGraph 쿼리를 수행하는 데 사용할 수 있는 GraphQL 문서로 구문 분석하는 JavaScript 템플릿 리터럴 태그입니다.

### 용법

```js
import { ngql } from 'nr1'
```

### 예

#### 실시예 1

```js
import { ngql } from 'nr1';


const query = ngql`
  {
    actor {
      user {
        id
        email
        name
      }
    }
  }
`;


function render() {
  return (
    <NerdGraphQuery query={query}>
      {({ data }) => JSON.stringify(data, null, 2)}
    </NerdGraphQuery>
  );
}
```
