헤더를 구성하는 셀의 래퍼입니다. 헤더는 항상 존재해야 하며 `<Table>` 요소의 첫 번째 직계 하위 항목이어야 합니다.

### 용법

```js
import { TableHeader } from 'nr1'
```

### 소품

<table>
  <tbody>
    <tr>
      <td>
        `children` <h5>필수</h5> <h5>노드</h5>
      </td>

      <td>
        `<TableHeaderCell>`모양의 열 헤더 목록입니다. 셀은 헤더의 직계 하위 항목이어야 합니다. 즉, `<TableHeader>` 과 `<TableHeaderCell>`사이에 노드가 있어서는 안 됩니다.
      </td>
    </tr>
  </tbody>
</table>
