X축을 따라 바늘 이동, 해당 계열에 대한 도구 설명 표시, 모든 차트에서 동시에 선택되는 영역을 회색으로 표시 등의 작업을 함께 수행할 수 있도록 다양한 유형의 차트를 그룹화합니다.

구성 요소는 React 조각처럼 작동합니다. DOM에서 추가 태그를 렌더링하지 않습니다(내부의 모든 차트에 대한 그룹화만 활성화합니다). 즉, 이 구성요소는 다른 구성요소와 달리 `className` 또는 `style` 태그를 허용하지 않습니다.

### 용법

```js
import { ChartGroup } from 'nr1'
```

### 예

#### 기초적인

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart style={chartStyle} accountIds={[1]} query={query} />
      <LineChart style={chartStyle} accountIds={[1, 1067061]} query={query} />
      <ScatterChart style={chartStyle} accountIds={[1]} query={query} />
    </ChartGroup>
  );
}
```

#### 채우기 컨테이너

```js
function render() {
  const query =
    'SELECT count(*) FROM `Synthetics` SINCE 1 DAY AGO TIMESERIES AUTO FACET jobType';
  const chartStyle = { height: 200 };


  return (
    <ChartGroup>
      <AreaChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
      <LineChart
        style={chartStyle}
        accountIds={[1, 1067061]}
        query={query}
        fullWidth
        fullHeight
      />
      <ScatterChart
        style={chartStyle}
        accountIds={[1]}
        query={query}
        fullWidth
        fullHeight
      />
    </ChartGroup>
  );
}
```
