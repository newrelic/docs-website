`<Popover>` 구성요소의 하위 요소입니다.

팝오버 오버레이의 콘텐츠를 포함합니다.

### 용법

```js
import { PopoverBody } from 'nr1'
```

### 소품

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""}]}
        />
      </td>

        <FunctionDefinition
          returnValue={[]}
          arguments={[{"name":"event","type":"React.MouseEvent","description":""},{"name":"opened","type":"boolean","description":""}]}
        />

        <OptionReference>

    </tr>

    <tr>
      <td>
        `testId` <h5>끈</h5>
      </td>

      <td>
        `data-test-id` 속성을 추가합니다. 이를 사용하여 단위 및 E2E 테스트의 구성 요소를 목표로 삼습니다. 테스트 ID가 유효하려면 앞에 nerdpack ID를 붙이고 그 뒤에 점이 옵니다. 예를 들어 `my-nerdpack.some-element`.

        **참고:** `data-test-id` 속성이 DOM 에서 제거될 때 표시되지 않을 수 있습니다. 이를 디버그하려면 `e2e-test` 쿼리 변수를 URL에 전달해야 합니다.
