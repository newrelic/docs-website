## `nr1 subscription:list` [#nr1-subscriptionlist]

### 구독 보기 [#nr1-subscriptionlist-see]

귀하의 계정이 구독하고 있는 모든 Nerdpack을 나열합니다. 귀하의 계정은 API 키에 연결되어 있습니다.

### 용법

`$ nr1 subscription:list`

### 옵션

<table>
  <tbody>
    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        사용하려는 인증 프로필입니다.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        출력에 추가 정보를 추가합니다.
      </td>
    </tr>
  </tbody>
</table>

## `nr1 subscription:set` [#nr1-subscriptionset]

### Nerdpack을 구독하세요 [#nr1-subscriptionset-subscribe]

귀하의 계정으로 특정 Nerdpack 및 채널을 구독합니다. 이 명령은 Nerdpack UUID 또는 특정 Nerdpack 폴더 내에서 실행할 수 있습니다. 기본적으로 이 명령은 `package.json` 의 Nerdpack ID를 사용하고 STABLE 채널을 구독합니다.

계정은 한 번에 하나의 Nerdpack과 채널만 구독할 수 있습니다.

### 용법 [#nr1-subscriptionset-usage]

`$ nr1 subscription:set`

### 옵션 [#nr1-subscriptionset-options]

<table>
  <tbody>
    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        구독할 Nerdpack을 지정합니다. 기본적으로 명령은 `package.json` 에 있는 명령을 사용합니다.
      </td>
    </tr>

    <tr>
      <td>
        `-c, --channel=DEV/BETA/STABLE`
      </td>

      <td>
        구독할 채널을 지정합니다. \[`default: STABLE`]
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        사용하려는 인증 프로필입니다.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        출력에 추가 정보를 추가합니다.
      </td>
    </tr>
  </tbody>
</table>

### 별칭 [#nr1-subscriptionset-aliases]

* `$ nr1 nerdpack:subscribe`

## `nr1 subscription:unset` [#nr1-subscriptionunset]

### Nerdpack 구독 취소 [#nr1-subscriptionunset-unsubs]

특정 Nerdpack에서 귀하의 계정을 구독 취소합니다. Nerdpack 폴더 내에서 이 명령을 실행하면 기본적으로 `package.json` 의 Nerdpack ID가 사용됩니다.

### 용법 [#nr1-subscriptionunset-usage]

`$ nr1 subscription:unset`

### 옵션 [#nr1-subscriptionunset-options]

<table>
  <tbody>
    <tr>
      <td>
        `-i, --nerdpack-id=NERDPACK_ID`
      </td>

      <td>
        구독할 Nerdpack을 지정합니다. 기본적으로 명령은 `package.json` 에 있는 명령을 사용합니다.
      </td>
    </tr>

    <tr>
      <td>
        `--profile=PROFILE`
      </td>

      <td>
        사용하려는 인증 프로필입니다.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        출력에 추가 정보를 추가합니다.
      </td>
    </tr>
  </tbody>
</table>

### 별칭 [#nr1-subscriptionunset-aliases]

* `$ nr1 nerdpack:unsubscribe`
* `$ nr1 subscription:delete`
* `$ nr1 subscription:remove`
* `$ nr1 subscription:rm`
