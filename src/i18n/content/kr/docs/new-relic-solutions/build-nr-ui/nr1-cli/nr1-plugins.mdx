## `nr1 plugins:install` [#nr1-pluginsinstall]

### 플러그인 설치 [#nr1-pluginsinstall-install]

CLI에 플러그인을 설치합니다. npm 또는 Git URL에서 플러그인을 설치할 수 있습니다.

플러그인을 설치하면 핵심 플러그인이 무시된다는 점에 유의하세요.

예를 들어, `hello` 명령이 있는 핵심 플러그인이 있는 경우 `hello` 명령으로 플러그인을 설치하면 핵심 플러그인 구현이 재정의됩니다. 이는 전체 CLI를 패치하고 업데이트하지 않고 핵심 플러그인 기능을 업데이트하려는 경우에 유용합니다.

### 용법 [#nr1-pluginsinstall-usage]

`$ nr1 plugins:install PLUGIN`

### 인수 [#nr1-pluginsinstall-arguments]

* `PLUGIN`: 설치하려는 플러그인의 이름, 경로 또는 URL입니다. 이는 **필수** 입니다.

### 옵션 [#nr1-pluginsinstall-options]

<table>
  <tbody>
    <tr>
      <td>
        `-f, --force`
      </td>

      <td>
        `yarn install --force` 실행합니다. 그러면 모든 플러그인 패키지가 강제로 다시 다운로드됩니다.
      </td>
    </tr>

    <tr>
      <td>
        `-h, --help`
      </td>

      <td>
        CLI 도움말을 표시합니다.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        출력에 추가 정보를 추가합니다.
      </td>
    </tr>
  </tbody>
</table>

### 예 [#nr1-pluginsinstall-examples]

* `$ nr1 plugins:install myplugin`
* `$ nr1 plugins:install https://github.com/someuser/someplugin`
* `$ nr1 plugins:install someuser/someplugin`

### 별칭 [#nr1-pluginsinstall-aliases]

* `$ nr1 plugins:add`

## `nr1 plugins:link` [#nr1-pluginslink]

### 플러그인 연결 [#nr1-pluginslink-link]

개발을 위해 로컬 플러그인을 CLI에 연결합니다.

플러그인을 연결하면 사용자가 설치한 플러그인이나 핵심 플러그인이 무시된다는 점에 유의하세요.

예를 들어, 사용자가 설치한 플러그인이나 `hello` 명령이 있는 핵심 플러그인이 있는 경우 플러그인을 `hello` 명령과 연결하면 사용자가 설치한 플러그인 또는 핵심 플러그인 구현이 재정의됩니다. 이는 개발 작업에 유용합니다.

### 용법 [#nr1-pluginslink-usage]

`$ nr1 plugins:link PLUGIN`

### 인수 [#nr1-pluginslink-arguments]

* `PLUGIN`: 연결하려는 플러그인의 이름, 경로 또는 URL입니다. 이는 **필수** 입니다.

### 옵션 [#nr1-pluginslink-options]

<table>
  <tbody>
    <tr>
      <td>
        `-h, --help`
      </td>

      <td>
        CLI 도움말을 표시합니다.
      </td>
    </tr>

    <tr>
      <td>
        `--verbose`
      </td>

      <td>
        출력에 추가 정보를 추가합니다.
      </td>
    </tr>
  </tbody>
</table>

### 예 [#nr1-pluginslink-examples]

* `$ nr1 plugins:link myplugin`
* `$ nr1 plugins:link someuser/someplugin`
