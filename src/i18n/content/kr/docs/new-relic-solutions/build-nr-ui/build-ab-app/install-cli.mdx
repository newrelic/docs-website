뉴렐릭 SDK의 주요 요소 중 하나는 CommandJump 인터페이스(CLI)입니다. Nerdpack을 생성하려면 SDK를 설치하고, 뉴렐릭 계정과 함께 작동하도록 CLI를 구성한 다음, `create` 명령을 활용해야 합니다.

# CLI 설치 및 구성 [#install]

<Steps>
  <Step>
    [**Build on New Relic**](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==) \[New Relic 빠른 시작에서 빌드를] 빠른 시작으로 이동하세요.
  </Step>

  <Step>
    [라이센스 API 키를](https://one.newrelic.com/launcher/api-keys-ui.api-keys-launcher) 받으세요.

    CLI를 설치하면 이 키를 사용하여 계정과 연결된 사용자 프로필을 생성하게 됩니다. CLI는 이 프로필을 사용하여 계정 내의 엔터티를 관리합니다.
  </Step>

  <Step>
    CLI를 설치할 때 New Relic 개발자 이용 약관을 읽고 동의하세요.

    CLI를 설치하더라도 먼저 본 이용 약관에 동의하지 않으면 CLI를 사용할 수 없습니다.
  </Step>

  <Step>
    운영 시스템을 선택하고 **Download installer** 클릭합니다.

    SDK를 설치하고 나면 `nr1` CLI에 액세스할 수 있습니다. SDK 버전을 확인하여 이를 확인하세요.

    ```sh
    nr1 --version
    ```

    CLI가 이미 있는 경우 업데이트하십시오.

    ```sh
    nr1 update
    ```

    <Callout variant="tip">
      `newrelic` CLI와 `nr1` CLI를 구별하는 것이 중요합니다. `newrelic` (는) 귀하의 New Relic 계정에서 엔터티를 관리하기 위한 것입니다. `nr1` New Relic을 관리하기 위한 것입니다.
    </Callout>
  </Step>

  <Step>
    자격 증명을 저장하려면 명령을 복사하세요.

    이 명령에는 프로필 이름, 지역 및 API 키가 포함되어 있습니다.

    터미널에서 다음 명령을 실행하세요.

    ```sh
    nr1 profiles:add --name <profile name> --api-key <User key> --region <region>
    ```

    프로필을 사용하면 명령을 실행할 New Relic 계정을 선택할 수 있습니다. 계정이 여러 개인 경우 `profiles:list` 을 사용하여 볼 수 있습니다.

    ```sh
    nr1 profiles:list
    ```

    하나의 프로필이 기본 프로필입니다. 다른 계정을 지정하지 않는 한 이는 명령이 실행되는 계정입니다. 특정 명령에 대한 프로필을 지정하려면 `--profile` 옵션을 사용합니다.

    ```sh
    nr1 create --profile <your profile>
    ```

    CLI를 처음 사용하는 경우 방금 추가한 프로필이 기본 프로필이 됩니다. 다른 프로필이 있는 경우 이 과정에 사용하려는 프로필로 기본값을 설정해야 합니다.

    ```sh
    nr1 profiles:default
    ```

    <Callout variant="tip">
      이러한 명령을 잊어버린 경우 프로필 도움말 메뉴에서 찾아볼 수 있습니다.

      ```sh
      nr1 profiles --help
      ```
    </Callout>
  </Step>
</Steps>

이제 **Build on New Relic** 빠른 시작을 종료할 수 있습니다. 뉴렐릭 CLI를 사용하여 기능을 구축할 준비가 되었습니다!
