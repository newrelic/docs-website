레벨 0 출력 트레이스 적용 범위 스코어카드 규칙은 분리 트레이스 데이터를 생성하는 APM 업무의 비율을 측정합니다. 이 기본 지표는 전체 인력추적 구현의 폭을 평가하는 데 도움이 됩니다.

## 이 규칙이 측정하는 것

이 점수표 규칙은 사용자 환경에서 분산 트레이에 대한 스팬 데이터를 적극적으로 생성하는 APM 의 백분율을 계산합니다. 일부분은 뉴렐릭이 해당 서비스 또는 일부분에서 스팬 데이터를 감지하면 규칙을 전달합니다.

이 규칙은 특히 다음 사항에 초점을 맞춥니다.

* APM-계측된 서비스 및 구성
* 내보내는 트레이스의 존재
* 모니터 전반에 걸친 적용 범위

## 점수를 해석하는 방법

귀하의 분리 트레이스 적용 범위 점수는 적극적으로 분기 트레이스 데이터를 생성하는 APM 의 백분율을 나타냅니다. 결과를 이해하는 방법은 다음과 같습니다.

* **높은 점수(80-100%)**: APM 부분의 대부분은 포괄적인 적용 범위를 나타내는 분할 트레이스 데이터를 생성하고 있습니다.
* **중간 점수(50-79%)**: 트레이스 데이터 수집에 개선의 여지가 있는 중간 범위
* **낮은 점수(0-49%)**: 요청 흐름 가시성에 차이가 생길 수 있는 제한된 분산 추적 구현

<Callout variant="tip">
  목표는 반드시 100% 적용이 아닙니다. 비즈니스에 중요한 서비스와 서비스 간 의존성/종속성이 복잡한 서비스가 수신되도록 하는 데 중점을 둡니다.
</Callout>

## 권장 조치

다음 전략을 사용하여 반사트레이스 커버리지 점수를 높이세요.

### 1. 서비스의 중요도와 복잡성을 평가합니다.

개발 및 운영 팀과 협력하여 다음을 수행하세요.

* 중요한 사용자 흐름을 처리하는 비즈니스 크리티컬 서비스 식별
* 복잡한 서비스 간 지도 서비스 의존성/종속성
* 성능 문제가 자주 발생하는 서비스를 우선시합니다.

### 2. 분산 추적을 전략적으로 구성

영향력이 큰 분야에 노력을 집중하세요:

* 먼저 중요 서비스에 대한 분산 추적을 활성화하세요
* 알려진 컴플렉스 의존성/종속성이 있는 서비스가 반환되었는지 확인하세요.
* 요청 체인의 부모 서비스에 추적이 활성화되어 있는지 확인하세요.

### 3. 스팬 캡처 및 품질 검증

현재 구현 내용을 검토하세요.

* 전체 컨텍스트 속성으로 스팬이 캡처되고 있는지 확인하세요.
* 트레이스 데이터에 문제해결, 해결을 위한 메타데이터가 포함되어 있는지 확인하세요.
* 성능 분석을 위해 스팬에 충분한 세부 정보가 포함되어 있는지 확인하세요.

### 4. 문서화 및 추적 범위

진행 상황을 가시적으로 확인하세요.

* 어떤 중요 서비스가 통합되어 있는지 보여주는 적용 범위 보고서를 만듭니다.
* 추가 APM 에이전트 구현, 배포가 필요한 간격을 식별합니다.
* 시간 경과에 따른 적용 범위 개선 추적

### 5. 사용자 환경에 맞게 규칙을 사용자 정의하세요

귀하의 특정 요구 사항에 맞게 점수표를 맞춤 설정하세요.

* 분산 추적이 가장 중요한 그룹 서비스에 태그를 적용합니다.
* 태그가 지정된 서비스 코호트에 초점을 맞추기 위해 점수표 규칙을 수정하세요.
* 추적이 필요하지 않은 테스트 또는 개발 서비스를 제외합니다.

## 중요한 고려 사항

분산 트레이스 적용 범위 점수를 사용할 때 다음 요소를 염두에 두십시오.

**사용자 환경에 맞게 사용자 정의**: 점수표 규칙은 일반적인 지침을 제공하지만, 사용자의 특정 요구 사항은 다를 수 있습니다. 아키텍처, 비즈니스 우선순위, 운영적 제약의 맥락에서 결과를 평가합니다.

**지속적인 개선에 집중하세요**. 분산 추적 전략은 시스템 아키텍처와 함께 발전해야 합니다. 정기적으로 보장 범위를 검토하고 서비스가 추가, 수정 또는 중단됨에 따라 접근 방식을 조정하세요.

**성능과 균형 잡힌 적용 범위**: 포괄적인 추적을 통해 가시성이 향상되지만, 처리량이 높은 서비스의 성능에 미치는 영향을 고려하세요. 샘플링 속도를 조정하거나 특정 서비스에 대한 추적을 선택적으로 활성화해야 할 수도 있습니다.

분산 트레이스 적용 범위 점수를 이해하고 이러한 권장 사항을 구현하면 비즈니스 목표에 맞춰 필요한 가시성을 제공하는 보다 효과적인 옵저버빌리티 전략을 수립할 수 있습니다.
