디지털 경험은 사용자가 웹 애플리케이션과 상호 작용하는 방식, 즉 초기 페이지 로드부터 중요한 비즈니스 작업 완료까지 모든 것을 포괄합니다. 디지털 경험 최적화는 더 빠르고 안정적이며 매력적인 사용자 결합을 통해 고객 만족도, 전환률 및 비즈니스 수익에 직접적인 영향을 미칩니다.

뉴렐릭 관찰 플랫폼은 포괄적인 프런트엔드 모니터링, 성능 분석 및 사용자 여정 최적화를 통해 디지털 경험을 측정, 모니터링 및 개선하는 데 도움이 됩니다. 이 가이드에서는 사후적 오류 대응부터 사전적 경험 최적화까지 아우르는 3단계 성숙도 모델을 제시합니다.

## 당신이 배울 것

이 디지털 경험 성숙도 진행 모델을 따르면 다음과 같은 이점이 있습니다.

* 사용자에게 영향을 미치는 문제를 포착하기 위해 안정적인 프런트엔드 모니터링을 구축합니다.
* Google의 Core Web Vitals를 기반으로 성능 최적화를 구현합니다.
* 비즈니스 가치를 촉진하는 사용자 중심 서비스 수준 지표를 만듭니다.
* 고객의 디지털 여정에 대한 포괄적인 이해를 구축하세요

## 성숙도 진행 모델

성숙도 진행 모델은 조직이 반응적 문제 대응에서 사전 예방적 경험 최적화, 그리고 궁극적으로 사용자 중심의 안정성 관리에 대한 숙달로 이어지는 체계적인 여정을 안내합니다. 각 레벨은 이전 레벨을 기반으로 고객 만족도와 비즈니스 성과를 향상시키도록 설계된 특정 관행과 지표를 바탕으로 구성됩니다.

### 레벨 1: 반응형 - &quot;고장난 것을 고치세요&quot;

**목표:** 사용자에게 직접적인 영향을 미치는 프런트엔드 문제를 감지하고 해결합니다.

반응적 수준에서는 디지털 경험 모니터링을 위한 기반을 구축하고 있습니다. 사용자가 애플리케이션에서 문제를 발견하면 팀에서 즉시 알 수 있도록 오류 추적 및 알림을 설정합니다.

**주요 관행:**

* 사용자 기능을 방해하는 JavaScript 오류를 모니터링합니다.
* 사용자 경험을 저하시키는 서버 오류를 추적합니다.
* 중요한 프런트엔드 오류에 대한 경고 설정
* 사용자에게 영향을 미치는 문제를 분류하고 해결하기 위한 프로세스 구축

다음 규칙은 반응형 디지털 경험 역량을 평가합니다.

* **[JavaScript 오류율](/docs/new-relic-solutions/observability-maturity/digital-experience/l1-js-error-rate):** 브라우저 기반 JavaScript 오류를 모니터링하여 오류를 평가하고 해결하여 사용자 경험 품질을 극대화합니다.

* **[서비스 오류율](/docs/new-relic-solutions/observability-maturity/digital-experience/l1-service-error-rate):** 사용자에게 영향을 미치는 서버 측 오류를 추적하여 이를 확인하고 해결하여 최적의 사용자 경험을 유지합니다.

### 레벨 2: 사전 예방 - &quot;성능 최적화&quot;

**목표:** 성능 최적화 및 모니터링을 통해 사용자 경험을 적극적으로 개선합니다.

반응형 대응을 넘어, 성능 데이터를 사용하여 사용자 경험 품질을 파악하고 Google의 Core Web Vitals와 같은 업계 표준을 기반으로 최적화합니다. 이 수준에서는 사용자 만족에 영향을 미치기 전에 성능 문제를 예방하는 데 중점을 둡니다.

**주요 관행:**

* 모니터 빠르고 반응성 있는 경험을 보장하기 위한 Core Web Vitals
* 기능 예산 및 최적화 구현하기, 목표
* 실사용자 모니터링을 활용해 실제 사용자 경험 이해
* 중요한 사용자 여정과 전환 경로를 최적화하세요

다음 규칙은 사전 예방적 디지털 경험 최적화를 평가합니다.

* **[Core Web Vitals | LCP](/docs/new-relic-solutions/observability-maturity/digital-experience/l2-cwv-lcp):** 가장 큰 콘텐츠 페인트를 측정하여 페이지가 빠르게 로드되고 사용자에게 즉각적인 가치를 제공하도록 보장합니다.

* **[Core Web Vitals | CLS](/docs/new-relic-solutions/observability-maturity/digital-experience/l2-cwv-cls):** 누적 레이아웃 변화를 평가하여 로딩하는 동안 페이지가 시각적으로 안정적으로 유지되는지 확인합니다.

* **[Core Web Vitals | INP](/docs/new-relic-solutions/observability-maturity/digital-experience/l2-cwv-inp):** 페이지가 사용자에 의해 신속하게 반응하도록 Next Paint에 대한 모니터 블록 확장입니다.

### 레벨 3: 마스터리 - &quot;측정 가능한 비즈니스 가치 제공&quot;

**목표:** 비즈니스 목표와 고객 만족을 직접적으로 뒷받침하는 일관된 디지털 경험 품질을 달성합니다.

마스터 수준에서는 디지털 경험 실무가 안정적이고 성과가 높은 사용자 경험을 통해 명확한 비즈니스 가치를 제공합니다. 기술적 성능과 성능 성능을 연결하는 포괄적인 지표를 설정하게 됩니다.

**주요 관행:**

* 비즈니스 영향을 반영하는 사용자 경험 SLI 정의
* 디지털 경험 품질을 위한 서비스 수준 계약 유지
* 디지털 경험 개선을 통한 ROI 입증
* 경험 데이터를 활용하여 제품 및 비즈니스 의사 결정을 내리세요

다음 규칙은 디지털 경험에 대한 숙련도를 평가합니다.

* **[서비스 수준 달성](/docs/new-relic-solutions/observability-maturity/business-uptime/l3-service-level-attainment):** 사용자 환경 서비스 수준 목표(SLO)의 일관된 달성을 측정하며, 95% 이상의 성공률은 고품질 디지털 경험을 나타냅니다.

## 디지털 경험 관행

이러한 옵저버빌리티 관행은 성숙도 모델의 실행 가능한 구성 요소로, 사용자 만족도와 무선 성능의 측정 가능한 개선을 실현할 수 있습니다.

### [품질 기반(QF)](/docs/tutorial-improve-customer-experience/intro-to-customer-experience)

고객의 디지털 경험에 대한 포괄적인 모니터링과 측정을 구축하세요. 이 연습은 다음과 같은 데 도움이 됩니다.

* 모든 터치포인트에서 현재 사용자 경험 품질을 파악합니다.
* 개선을 위한 가장 큰 기회를 파악하세요
* 경험 최적화 목표에 대한 진행 상황 추적

### [깔때기 하단 분석(BotFA)](/docs/tutorial-improve-customer-experience/bottom-funnel-analysis)

중요한 사용자 여정 특성에 초점을 맞춰 전환율을 최적화합니다. 이 연습을 통해 다음을 수행할 수 있습니다.

* 사용자가 전환 프로세스에서 이탈하는 지점을 파악합니다.
* 결제, 가입 및 기타 중요한 비즈니스 흐름을 최적화합니다.
* 경험 개선을 수익에 직접 연결

### [서비스 수준 관리(SLM)](/docs/tutorial-service-level-mgmt/intro-to-slm)

비즈니스 목표에 맞춰 기술적 성능을 조정하는 사용자 환경 서비스 수준 지표를 생성합니다. 이 연습에서는 다음을 제공합니다.

* 사용자 경험 품질에 대한 논의를 위한 표준화된 언어
* 경험 최적화 노력의 명확한 목표
* 경험 개선을 통한 ROI를 보여주는 비즈니스 중심 지표

## 시작하기

디지털 경험을 개선할 준비가 되셨나요? 현재 역량에 따라 시작하는 방법은 다음과 같습니다.

### 현재 상태를 평가하세요

1. **사용자 경험 모니터링을 검토하세요.** 사용자가 오류나 성능 문제를 겪는 경우를 알고 있나요?
2. **성능 지표를 평가하세요.** 무엇이 실제로 사용자 만족도에 영향을 미치는지 측정하고 있나요?
3. **대응 역량을 확인하세요.** 귀하의 팀이 사용자에게 영향을 미치는 문제를 신속하게 식별하고 해결할 수 있나요?

### 출발점을 선택하세요

* **프런트엔드 모니터링을 처음 접하시나요?** 레벨 1 오류 추적 및 경고로 시작
* **기본 모니터링이 있나요?** 레벨 2 Core Web Vitals 최적화로 이동
* **성숙한 모니터링 관행?** 디지털 경험을 위한 레벨 3 서비스 수준 관리에 집중

### 권장 구현 순서

1. **[JavaScript 오류율](/docs/new-relic-solutions/observability-maturity/digital-experience/l1-js-error-rate)** - 프런트엔드 오류 모니터링 구축
2. **[서비스 오류율](/docs/new-relic-solutions/observability-maturity/digital-experience/l1-service-error-rate)** - 사용자에게 영향을 미치는 백앤드 오류를 추적합니다.
3. **[Core Web Vitals 모니터링](/docs/tutorial-improve-customer-experience/intro-to-customer-experience)** - 성능 최적화 구현
4. **[품질 기반 관행](/docs/tutorial-improve-customer-experience/intro-to-customer-experience)** - 포괄적인 경험 측정 구축
5. **[서비스 수준 관리](/docs/tutorial-service-level-mgmt/intro-to-slm)** - 경험을 비즈니스 성과에 연결

## 성공 측정

디지털 경험 성숙도 진행 상황을 측정하려면 다음과 같은 주요 지표를 추적하세요.

* **오류 감소:** JavaScript 및 서비스 오류율 감소
* **성능 개선:** 더 나은 Core Web Vitals 점수(LCP, CLS, INP)
* **사용자 만족도:** 전환율 증가 및 이탈률 감소
* **비즈니스 영향:** 매출 향상 지표는 경험 개선과 연관됨
* **대응 효율성:** 사용자에게 영향을 미치는 문제를 더 빠르게 감지하고 해결

## 다음 단계

뉴렐릭 관측 플랫폼은 디지털 경험 최적화에 탁월한 데 필요한 도구와 인사이트를 제공합니다. 이 성숙도 진행 모델을 따르고 권장 사례를 구현하면 다음과 같은 이점을 얻을 수 있습니다.

* 더 빠르고 안정적인 사용자 경험 제공
* 고객의 불만과 이탈을 줄이세요
* 전환율과 사업 수익을 늘리세요
* 탁월한 디지털 경험을 통해 경쟁 우위를 구축하세요

뉴렐릭의 디지털 경험 리소스와 가이드를 살펴보고 특정 사용자 요구 사항과 비즈니스 목표에 맞춰 최적화 여정을 맞춤화하세요.
