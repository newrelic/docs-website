## 필터 표시줄 찾기 [#find]

엔티티 필터는 많은 New Relic UI 페이지의 상단에 있습니다. [New Relic을 로드할](https://one.newrelic.com) 때 가장 먼저 표시되는 **All entities** \[모든] 엔티티 보기에 표시되며 APM UI, 브라우저 모니터링 UI, 인프라 UI 등에 표시됩니다.

## 필터 표시줄 사용 팁 [#tips]

필터 표시줄 사용에 대한 몇 가지 팁:

* 이름이나 ID에 이 문자열이 포함된 엔터티를 찾으려면 문자 및/또는 숫자 **문자열** (예: 환경)을 입력합니다. 입력할 때 UI는 입력하는 문자열과 일치하는 항목을 제안하므로 드롭다운에서 항목 중 하나를 선택할 수 있습니다. **엔터티** 이름, **엔터티 유형** , **계정 ID** , **환경** 또는 [**태그**](/docs/new-relic-one/use-new-relic-one/core-concepts/tagging-use-tags-organize-group-what-you-monitor) 로 필터링할 수도 있습니다. 선택 매개변수는 일단 생성되면 파란색 배경을 갖습니다.
* 문자열을 사용하여 필터링하는 경우 필터에 `=` 연산자가 있습니다.
* 드롭다운 메뉴를 사용하여 필터링하는 경우 검색된 항목의 첫 번째 요소(태그 키 또는 속성)를 입력한 후 필터에 연산자(`=`, `!=`, `LIKE`, `NOT_LIKE`, `IN`, `NOT IN`)를 선택해야 합니다.
* 연산자가 선택되면 값을 선택하여 필터를 완성하여 검색 항목을 완성합니다.

더 많은 필터를 추가하려면 먼저 연산자 `AND` 또는 `OR.` 사용 팁을 선택해야 합니다.

* `AND` 을 사용하여 결과 목록에서 항목을 제거하는 선택을 제한하려는 것을 나타냅니다. AND 연산자를 사용하여 결과 목록에서 충족해야 하는 조건을 추가할 수도 있습니다(예: **entityType = Services `AND` location = APAC** ).

  * **entityType = Services `AND` entityType = Hosts** 는 어떤 결과도 반환하지 않습니다. 엔티티는 하나의 유형만 가질 수 있고 이 조건과 일치하는 엔티티는 없기 때문입니다.

* `OR` 을 사용하여 선택 항목에 더 많은 항목을 추가합니다. 예를 들어 필터 **entityType = Services `OR` entityType = Hosts** 는 `Services` 또는 `Hosts` 유형의 액세스 권한이 있는 모든 항목을 반환합니다.

## 필터링된 보기 저장 [#save]

원하는 필터가 있고 나중에 사용하려는 경우 필터 표시줄 오른쪽에 있는 **Save as view** \[보기로 저장] 버튼을 클릭합니다.

저장된 필터를 보려면 **All entities** \[모든 항목] 보기에서 오른쪽의 **Saved views** \[저장된 보기] 섹션을 살펴보십시오.
