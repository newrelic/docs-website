맞춤형 앱을 구축하거나 오픈소스 앱 중 하나를 수정하기로 결정한 경우 몇 가지 필수 도구가 필요합니다.

* [뉴렐릭 마우스줄 인터페이스(CLI)](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==).
* CLI를 다운로드할 때 얻는 API 키입니다.

앱으로 수행하려는 작업에 따라 몇 가지 추가 설정 및 구성이 필요할 수 있습니다. 이 가이드에서는 다음 내용을 다룹니다.

* 앱을 빌드하거나 수정하기 위한 뉴렐릭 CLI 다운로드
* 이 웹사이트에 콘텐츠를 제공하세요

## 시작하기 전에

시작하기 전에 먼저 [권한에](/docs/new-relic-solutions/new-relic-one/build-nr-apps/permission-manage-apps) 대해 읽어보는 것이 좋습니다.

구축을 시작하려면 다음이 필요합니다.

* [github 계정](https://github.com/) 계정입니다. 앱 구축에 꼭 필요한 것은 아니지만 GitHub 계정을 사용하면 [오픈 소스 앱을 다운로드 및 맞춤화하고](https://opensource.newrelic.com/explore-projects) [오픈 소스 프로젝트](https://opensource.newrelic.com/contributing/) 에 기여할 수 있습니다.
* [뉴렐릭(New Relic) 개발자 계정](https://newrelic.com/signup/?partner=Developer+Edition). 아직 계정이 없다면 뉴킹릭 피규어 개발을 위한 무료평가판 계정을 얻으실 수 있습니다.
* [npm](https://www.npmjs.com/get-npm). [Node.js를](https://nodejs.org/en/download/) 설치했다면 JavaScript 코드를 공유, 재사용 및 업데이트하는 데 사용되는 `npm` 이미 있는 것입니다. 이는 뉴렐릭 앱과 이 웹사이트를 위한 프레임워크인 React 구성 요소를 사용하는 데 필요합니다.

### 지원에 대한 참고 사항

뉴킹릭 피규어를 구축하는 것은 JavaScript/React를 구축하는 것과 같습니다. 저희는 빌딩 도구([CLI](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==) 와 [SDK 라이브러리)를](/docs/new-relic-solutions/build-nr-ui/sdk-component/intro-to-sdk) 활용하는 데 도움이 되는 지원을 제공합니다. 하지만 기본적인 JavaScript나 React 코딩 질문이나 문제에 대한 지원은 제공하지 않습니다.

빌드에 대한 일반적인 질문과 답변은 [뉴렐릭 빌드에 대한 탐험가 허브 페이지를](https://discuss.newrelic.com/c/build-on-new-relic) 참조하세요.

<Callout variant="tip">
  [뉴렐릭 VSCode 확장 프로그램](https://marketplace.visualstudio.com/items?itemName=new-relic.nr1) 이나 [뉴렐릭 VSCode 확장 프로그램 팩을](https://marketplace.visualstudio.com/items?itemName=new-relic.new-relic-extension-pack) 사용하여 앱을 빌드하세요.
</Callout>

## 앱 빌드 또는 수정 준비

<Steps>
  <Step>
    ### CLI 및 API 키 다운로드

    [Build New Relic applications page](https://one.newrelic.com/launcher/developer-center.launcher?pane=eyJuZXJkbGV0SWQiOiJkZXZlbG9wZXItY2VudGVyLmRldmVsb3Blci1jZW50ZXIifQ==) \[Build 뉴킹 피규어 페이지] 에서 다운로드할 **Quick start** \[빠른 시작] 단계를 완료하세요.

    이러한 6가지 빠른 시작 단계는 앱 개발에 사용할 API 키와 앱 구축 및 구현을 위한 뉴렐릭 CLI를 제공합니다. 빠른 시작을 마치면 다음으로 구성된 프로젝트가 생성됩니다.

    * 너 **Nerdpack** \[드팩]. 애플리케이션에 필요한 모든 파일이 포함된 패키지입니다. 여기에는 앱을 빌드하기 위해 사용자 정의하는 두 가지 유형의 파일인 Nerdlets와 실행 프로그램이 포함되어 있습니다.
    * 하나 이상의 **Nerdlet** 파일. 특정 UI 보기 또는 창. Nerdlet은 `index.js` 파일, `stylesheet` 및 JSON 형식의 `config` 파일을 포함하는 React JavaScript 패키지입니다. 여기에는 모든 JS 기능(차트, 대화형 필드, 도구 설명 등)이 포함될 수 있습니다.
    * `launcher` 파일: 이는 앱을 게시한 후 뉴렐릭에서 특별을 여는 데 사용되는 런처의 기초입니다.
  </Step>

  <Step>
    ### 건축 시작

    코딩할 준비가 되었다면 Nerdpack에 `cd` 하고 시작하세요.

    애플리케이션 구축에 대해 자세히 알아보려면 다음 단계별 가이드를 시도해 보세요.

    * [&quot;Hello, World!&quot;를 빌드하세요.](/docs/new-relic-solutions/tutorials/build-hello-world-app) 멋진 전설을 만들고, 뉴렐릭에 게시하고, 계정 을 구독하여 다른 사람들과 공유하는 방법을 보여줍니다.
    * [지역별 페이지 뷰 맵은](/docs/new-relic-solutions/tutorials/map-pageviews-region/) 인기 있는 오픈 소스 앱 중 하나를 만드는 단계를 안내합니다. 앱에 사용자 정의 쿼리를 추가하고 테이블에서 본 다음 해당 데이터를 지도에 추가하는 방법을 알아봅니다.
  </Step>
</Steps>

## `developer.newrelic.com` 사이트에 참여하세요

이 사이트는 오픈 소스이므로 여러분의 의견을 듣고 싶습니다. 풀을 작성하세요. 실수를 발견했다면 고치는 방법을 알고 있습니다. 우리가 작업하길 원하는 콘텐츠 격차가 발견되면 GitHub 문제를 제출해 주세요. 또는 공유하고 싶은 가이드가 있으면 완전히 새로운 가이드를 작성하세요. 방법을 알아보려면 계속 읽어보세요.

<Steps>
  <Step>
    ### `developer-website` GitHub 리포지터리 포크

    리포지터리를 포크하면 `developer.newrelic.com` 파일의 자체 복사본으로 작업하고 [로컬에서 사이트를 구축할 수 있습니다](https://github.com/newrelic/developer-website#developernewreliccom). 또한 Incomimg Pull을 더욱 쉽게 관리할 수 있게 해줍니다.

    1. GitHub의 [개발자 웹사이트](https://github.com/newrelic/developer-website) 페이지로 이동합니다.
    2. 페이지 오른쪽 상단에 있는 **Fork** 버튼을 선택하세요.
    3. 포크하려는 계정을 선택하고 포크가 생성되는 동안 몇 초 정도 기다립니다.

    [정기적으로 동기화하여](https://docs.github.com/en/github/collaborating-with-issues-and-pull-requests/syncing-a-fork) 메인 브랜치 업스트림에 대한 변경 사항 및 추가 사항을 최신 상태로 유지하세요.
  </Step>

  <Step>
    ### 기능 또는 문서 요청하기

    1. 어느 페이지에서나 페이지 상단에 있는 **GitHub** 버튼을 선택합니다.
    2. 원하는 변경 종류를 선택하세요.
    3. GitHub 양식을 작성하세요.
  </Step>

  <Step>
    ### 새로운 가이드를 제공하세요

    프로세스를 안내하는 [기여자 지침 을](https://github.com/newrelic/developer-website/blob/master/CONTRIBUTING.md#creating-new-pages) 확인하세요.
  </Step>
</Steps>
