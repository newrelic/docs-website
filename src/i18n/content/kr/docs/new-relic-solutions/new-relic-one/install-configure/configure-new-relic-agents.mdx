일부 [New Relic 통합](https://newrelic.com/instant-observability/) 에는 에이전트를 수동으로 설치해야 합니다. 이러한 에이전트의 동작이 구성되는 방식은 특정 에이전트에 따라 다릅니다.

## APM 에이전트 구성 [#apm-config]

에이전트에 필요한 유일한 설정은 <InlinePopover type="apm"/>[에이전트를 설치할](/docs/agents/manage-apm-agents/installation/install-agent) [때 구성되는](/docs/apis/intro-apis/new-relic-api-keys/#ingest-license-key) 인스턴스 키 와 [앱 이름](/docs/apm/new-relic-apm/installation-configuration/name-your-application) 입니다.

추가 구성 옵션은 다음을 참조하십시오.

* [가다](/docs/agents/go-agent/instrumentation/go-agent-configuration)
* [자바](/docs/agents/java-agent/configuration/java-agent-configuration-config-file)
* [.그물](/docs/agents/net-agent/installation-configuration/net-agent-configuration)
* [노드.js](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration)
* [PHP](/docs/agents/php-agent/configuration/php-agent-configuration)
* [파이썬](/docs/agents/python-agent/installation-configuration/python-agent-configuration)
* [루비](/docs/agents/ruby-agent/installation-configuration/ruby-agent-configuration)

## 브라우저 에이전트 구성 [#browser-config]

[<InlinePopover type="browser"/>설정](/docs/browser/new-relic-browser/configuration) 에 대한 설명서를 참조하세요.

## 인프라 에이전트 구성 [#infra-config]

인프라 에이전트에 필요한 유일한 설정은 <InlinePopover type="licenseKey"/>입니다. 이는 [에이전트를 설치할](/docs/agents/manage-apm-agents/installation/install-agent) 때 구성됩니다. [추가 설정 옵션을](/docs/infrastructure/new-relic-infrastructure/configuration/configure-infrastructure-agent) 사용하여 어셈블리를 모니터링할 수도 있습니다.

## 모바일 에이전트 구성 [#mobile-config]

모바일 모니터링 에이전트에 필요한 유일한 설정은 [애플리케이션 토큰](/docs/mobile-monitoring/new-relic-mobile/maintenance/viewing-your-application-token)입니다. [Android](/docs/mobile-monitoring/new-relic-mobile-android/get-started/introduction-new-relic-mobile-android), [iOS/tvOS](/docs/mobile-monitoring/new-relic-mobile-ios/get-started/introduction-new-relic-mobile-ios) 앱용 모바일 모니터링 에이전트를 설치할 때 구성됩니다.

추가 구성 옵션은 다음을 참조하십시오.

* [안드로이드 구성](/docs/mobile-monitoring/new-relic-mobile-android/api-guides/android-agent-configuration-feature-flags)
* [iOS/tvOS 구성](/docs/mobile-monitoring/new-relic-mobile-ios/api-guides/ios-agent-configuration-feature-flags)

## 다른 New Relic 도구 구성 [#other]

여기에 나열되지 않은 다른 도구를 구성하려면 [특정 솔루션에 대한 설명서를](https://newrelic.com/instant-observability/) 참조하십시오.
