일부 [New Relic 솔루션](https://newrelic.com/instant-observability/) 은 에이전트 설치가 필요합니다. 데이터 보고를 중지하려면 에이전트를 제거할 수 있습니다. New Relic을 완전히 제거해야 하는 경우 에이전트의 절차를 따르십시오.

문제가 발생하면 [support.newrelic.com](https://support.newrelic.com) 을 참조하십시오.

## 제거하기 전에 [#downgrade]

<Callout variant="important">
  New Relic 계정을 지불하는 경우 제거에 대한 대체 옵션은 [계정을 더 저렴하거나 무료 등급으로 다운그레이드하는 것](/docs/accounts/accounts-billing/account-setup/downgradecancel-account) 입니다.
</Callout>

## APM 제거 [#language-agents]

APM 에이전트를 제거하거나 시작 스크립트를 변경한 후에는 앱(또는 앱이 있는 앱 서버 또는 웹 서버)을 다시 시작해야 합니다. New Relic 에이전트는 일반적으로 앱을 실행하는 프로세스의 메모리에서 실행되기 때문에 이 작업을 수행해야 합니다. 단순히 해당 파일을 제거한다고 해서 현재 실행 중인 응용 프로그램이 New Relic으로 데이터를 보내는 것을 중지 **하지 않습니다** .

* [C SDK](/docs/agents/c-sdk/install-configure/uninstall-remove-c-sdk)
* [가다](/docs/agents/go-agent/installation-configuration/uninstall-go-agent)
* [자바](/docs/agents/java-agent/installation/uninstall-java-agent)
* [.그물](/docs/agents/net-agent/installation-configuration/uninstall-net-agent)
* [노드.js](/docs/agents/nodejs-agent/installation-configuration/uninstall-nodejs-agent)
* [PHP](/docs/agents/php-agent/advanced-installation/uninstalling-php-agent)
* [파이썬](/docs/agents/python-agent/installation-configuration/uninstall-python-agent)
* [루비](/docs/agents/ruby-agent/installation-configuration/uninstall-ruby-agent)
