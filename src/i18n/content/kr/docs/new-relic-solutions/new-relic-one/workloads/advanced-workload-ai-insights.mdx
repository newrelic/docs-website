<Callout variant="important">
  AI 인사이트 기능은 특정 사용 가능 조건 하에 **공개 미리보기** 형태로 제공됩니다.
</Callout>

AI 인사이트는 워크로드의 상태, 성능 추세 및 잠재적 문제에 대한 AI 기반 요약을 제공합니다.

## 요구 사항 [#requirements]

* 지능형 워크로드 생성
* 충분한 옵저버빌리티 데이터
* **AWS 인프라에서만** 사용 가능합니다(Azure에서는 사용할 수 없음).
* **FedRAMP 인증 고객은 이용할 수 없습니다.**

## 인사이트 생성 [#generate]

1. 지능형 워크로드를 열어보세요
2. 딸깍 하는 소리 <DNT>**Generate insights**</DNT>
3. AI가 생성한 요약을 검토하세요.

Insights 검색 페이지에 표시된 데이터만 분석하며 필요에 따라 생성됩니다.

## 어떤 인사이트가 포함되나요? [#what-included]

**건강 개요:**

```
Your workload is healthy with 12 entities reporting normal status.
Response times are within expected ranges.
```

**성능 동향:**

```
Throughput increased 23% in the past hour while maintaining stable
response times.
```

**나루토 하이라이트:**

```
- payment-service: Response time +40ms (within SLA)
- user-db: Connection pool at 78%, approaching threshold
- checkout-api: Error rate below 0.1%, optimal
```

**권장 사항:**

```
Investigate database connection pool trend. While not critical,
usage may require scaling if traffic continues growing.
```

## 모범 사례 [#best-practices]

* 워크로드를 처음 보거나 알림이 발생한 후 인사이트를 생성합니다.
* AI 관찰 결과를 실제 데이터와 비교하여 검증합니다.
* 인사이트를 분석의 대체물이 아닌 출발점으로 활용하세요.
* 중대한 변화 후 재생

## 제한 사항 [#limitations]

* 현재 표시된 데이터만 분석합니다.
* 다른 워크로드 또는 계정에 액세스할 수 없습니다.
* 관찰 내용만 제공할 뿐, 근본 원인 분석은 제공하지 않습니다.
* 복잡한 패턴을 때때로 잘못 해석할 수 있습니다.

<Callout variant="caution">
  AI 인사이트에서 조치를 취하기 전에 항상 실제 데이터와 비교하여 검증하십시오.
</Callout>

## 문제점 해결 [#troubleshooting]

### 버튼 비활성화됨

**이유:**

* Azure 인프라(지원되지 않음)
* FedRAMP 계정 (AI 기능 비활성화됨)
* 해당 기능은 귀하의 계정에서 사용할 수 없습니다.

**해결 방법:** 담당 계정 팀에 문의하여 인프라 및 기능 접근 권한을 확인하십시오.

### 세대 실패

**이유:**

* 데이터가 불충분합니다
* 일시적인 연결 문제
* 복잡한 워크로드 데이터

**해결 방법:** 잠시 기다렸다가 다시 시도하십시오. 문제가 지속되면 고객 지원팀에 문의하십시오.
