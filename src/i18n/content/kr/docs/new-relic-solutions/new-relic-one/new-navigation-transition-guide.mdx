New Relic 플랫폼 UI를 더 간단하고 사용자 정의할 수 있도록 크게 변경하여 더 잘 작동하도록 했습니다.

* **더 많은 공간이 있습니다.** 상단과 측면에서 UI를 제거하여 중요한 데이터인 데이터에 집중할 수 있는 더 많은 공간을 제공합니다. 왼쪽 탐색을 축소하여 더 많은 공간을 확보할 수도 있습니다.
* **보고 싶은 것을 보십시오.** 왼쪽 탐색은 완전히 사용자 지정할 수 있으므로 가장 많이 사용하는 항목으로 더 빠르게 이동할 수 있습니다. 가장 많이 사용하는 도구, 보기 및 기능을 추가하고 원하는 만큼 자주 변경합니다.
* **왼쪽 탐색이 안정적입니다.** 기능 또는 기능 내에서 얼마나 자주 클릭하든, 왼쪽 탐색 메뉴는 UI에서 다시 돌아갈 수 있는 안정적이고 안정적인 위치입니다.
* **어둠 속에서 일하십시오.** 우리 디자이너들은 어둠 속에서 작업하는 것을 좋아하는 당신을 위해 우리의 어두운 모드를 우아한 경험으로 세련되게 만들었습니다.

## 켜기(또는 끄기) [#on]

새 탐색을 켜려면: 오른쪽 상단 모서리에 있는 사용자 메뉴를 클릭한 다음 **새 디자인** 을 **켜기** 로 토글합니다. (어떤 이유로든 이 기능을 꺼야 하는 경우 새 왼쪽 탐색 메뉴의 왼쪽 하단 모서리에 있는 사용자 메뉴를 클릭한 다음 토글을 클릭하여 **끄기** 로 전환합니다.)

사용해 주셔서 감사합니다. 새로운 UI에 대한 의견을 알려주세요!

## 이것이 당신에게 더 나은 서비스를 제공하는 방법 [#why]

<img
  title="The new home UI"
  alt="A screenshot of the new home page UI."

  style={{ maxWidth: '70%' }}

이 업데이트를 사용하면 방해가 되지 않고 더 많은 화면 공간을 생성하여 가장 중요한 시스템 상태에 집중할 수 있는 더 간단하고 안정적인 UI 환경을 갖게 됩니다.

뿐만 아니라 UI를 원하는 만큼 자주 사용자 지정할 수 있으므로 필요에 따라 변경할 수 있습니다.

## 사용하는 기술 검색 [#add-more]

<img
  title="The monitor everything UI"
  alt="A screenshot of the new monitor everything page UI."
  src={monitorEverything}
  style={{ maxWidth: '70%' }}
/>

왼쪽 탐색 메뉴에서 **+ 데이터 추가** 로 이동하여 수백 개의 New Relic 호환 데이터 소스 카탈로그, 사전 구축된 대시보드, 유용한 경고 조건, 기타 앱 및 시각화를 확인하세요. 기술을 사용하는 경우 모니터링할 수 있는 가능성이 높습니다.

뿐만 아니라 카탈로그는 대화형 설치 경로와 특정 문서를 알려줌으로써 설치 프로세스를 안내하므로 최대한 빠르고 쉽게 시작할 수 있습니다.

<img
  title="The new all entities page"
  alt="A screenshot of the new all entities page."
  src={allEntities}
  style={{ maxWidth: '70%' }}
/>

**모든 엔터티로 이동하여 모니터링되는 모든 엔터티** 를 한 곳에서 볼 수 있습니다.

여기에서 New Relic Explorer를 찾을 수 있습니다. 이를 사용하여 모니터링 중인 모든 것을 확인하고 무슨 일이 일어나고 있는지 자세히 알아보십시오.

## 사용자 메뉴를 이동했습니다. [#user-menu]

<img
  title="The new user menu"
  alt="A screenshot of the new user menu location in the bottom left corner."
  src={userMenu}
  style={{ maxWidth: '70%' }}
/>

UI의 왼쪽 하단에서 사용자 메뉴를 찾습니다. 이를 사용하여 팀원을 New Relic에 초대하고, 사용자 기본 설정을 지정하고, 다크 모드로 전환하고, API 키를 찾거나 생성하고, 기타 설정을 변경할 수 있습니다.

이 메뉴를 사용하여 이전 UI로 다시 전환할 수도 있습니다.
