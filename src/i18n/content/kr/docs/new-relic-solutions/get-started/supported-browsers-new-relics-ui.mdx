New Relic의 UI는 가장 일반적인 브라우저의 두 가지 최신 데스크톱 버전을 지원합니다. 지원되지 않는 브라우저 버전을 사용하는 경우 UI에 오류가 표시됩니다.

<Callout variant="tip">
  브라우저 모니터링에 대한 요구 사항은 브라우저 모니터링 [호환성을](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-new-relic-browser) 참조하십시오.
</Callout>

## 지원되는 브라우저 [#requirements]

우리의 UI는 다음 브라우저에 대한 두 가지 최신 데스크톱 버전을 지원합니다.

* 구글 크롬: 리눅스, 맥OS, 윈도우

* 마이크로소프트 엣지: 윈도우

* 모질라 파이어폭스: 리눅스, 맥OS, 윈도우

  * Firefox의 ESR(Extended Support Release) 버전은 지원되지 않습니다.

* 사파리: 맥OS

브라우저 버전이 지원되지 않는 경우에도 작동할 수 있습니다. 그러나 당사 사이트(및 기타 사이트)에서 최상의 기능을 얻으려면 지원되는 버전으로 업그레이드하는 것이 좋습니다.

## 제한된 지원

우리의 UI는 모바일 브라우저에 대한 제한된 지원을 제공합니다. 모바일 장치에서 New Relic에 액세스하려면 [Android 앱](/docs/mobile-apps/new-relic-android-app/features/new-relic-android-app) 또는 [iOS 앱](/docs/mobile-apps/new-relic-ios-app/features/new-relic-ios-app) 을 설치하십시오. 모바일 앱의 워크플로는 전화 사용 사례를 위해 특별히 설계되었습니다. 웹 인터페이스는 모바일 브라우저에서 적극적으로 지원되지 않습니다.
