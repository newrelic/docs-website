## 문제

새 호스트나 서버를 배포하거나 다른 응용 프로그램을 변경할 때 시스템이 이러한 변경 사항에 어떻게 응답하는지 즉시 확인하려고 합니다. 실시간으로 도착하는 로그 메시지를 확인해야 합니다.

## 해결책

실시간으로 도착하는 메시지에 집중하려면 라이브 테일 로깅을 활성화하십시오.

1. **[one.newrelic.com > 로그](https://one.newrelic.com)** 로 이동합니다.

2. 필요한 로그 정보 유형에 초점을 맞추는 쿼리를 입력한 다음 **쿼리 로그** 를 클릭합니다. 예를 들어:

   ```
   Find logs where <var>annotations.restartRequested:"2020-12-23T18:00:00Z"</var>
   ```

3. 활성화하려면 **Livetail** 아이콘을 클릭하십시오. <Icon name="fe-play-circle"/>또는 비활성화 <Icon name="fe-pause-circle"/>그것.
