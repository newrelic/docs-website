## 문제

CloudFormation 스택을 삭제할 때 다양한 이유로 실패할 수 있습니다.

## 해결책

가능한 이유와 해결책:

* 스택에 의해 생성된 S3 버킷(예: Firehose 오류 로그용)이 비어 있지 않으면 삭제할 수 없습니다. 내용을 보관한 후 스택 삭제를 다시 시도하세요.
