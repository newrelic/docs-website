## 문제

Firehose 통합 설정을 통해 CloudWatch에 대한 쿠렐릭에 로그가 표시되지 않습니다.

## 해결책

가능한 이유와 해결책:

* API 키가 올바르지 않을 수 있습니다. <DNT>**Amazon Data Firehose &gt; Firehose streams &gt; your-stream &gt; Edit destination settings**</DNT> 로 이동하여 올바른 수집 라이선스 키로 API 키를 업데이트하세요.
* 제공된 엔드포인트가 잘못되었을 수 있습니다. 돈을 확인하세요.
* S3 로그나 Cloudwatch 오류 로그를 확인하여 문제가 있는지 확인하세요.
