## 문제

Lambda 통합 설정을 통해 S3의 쿠렐릭에 로그가 표시되지 않습니다.

## 해결책

가능한 이유와 해결책:

* [CloudWatch 또는 Lambda에서 로그가 나타나지 않는](/docs/logs/troubleshooting/aws-log-integration-issues/logs-not-appearing-from-cloudwatch-lambda) 가능한 이유를 확인하세요.
* S3 파일은 뉴렐릭의 [로그인 포워딩 제한 사항](/docs/logs/troubleshooting/aws-log-integration-issues/log-forwarding-restrictions) 을 준수하지 않을 수 있습니다.
* `NewRelicServerlessLogForwarder` 에 S3 트리거가 없을 수 있습니다. 중첩된 CloudFormation 스택의 출력을 확인하고 추가합니다.
