>
  2022년 4월부터 C SDK 기능을 지원하지 않습니다. 자세한 내용은 [지원 포럼 게시물 을](https://discuss.newrelic.com/t/q1-bulk-eol-announcement-fy23/181744)참조하십시오.
</Callout>

C SDK에서 모니터링하는 앱에는 컨텍스트 로그인을 사용할 수 없습니다. 대신 [Log API](/docs/logs/log-management/log-api/introduction-log-api/) 를 사용하여 고유한 로깅 확장 라이브러리를 빌드할 수 있습니다. 앱에서 분산 추적이 활성화되어 있는지 확인하여 요청이 업스트림 및 다운스트림 서비스를 통해 이동할 때 어떤 일이 발생하는지 빠르게 이해할 수 있습니다.

C SDK에서 모니터링하는 앱에 대해 분산 추적을 활성화하려면 최신 C SDK 버전을 [설치](/docs/distributed-tracing/enable-configure/quick-start/) 하거나 [업데이트](/docs/agents/c-sdk/install-configure/update-your-c-sdk-library/) 하세요. 분산 추적에는 [C SDK 버전 1.1.0 이상](/docs/release-notes/agent-release-notes/c-sdk-release-notes/) 이 필요합니다.
