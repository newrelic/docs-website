뉴렐릭 APM 언어 에이전트를 사용하면 에이전트 전달 로그에 사용자 정의 태그(레이블)를 추가하도록 선택할 수 있습니다. 로그에 사용자 정의 태그를 사용하면 DevOps 및 플랫폼 엔지니어가 로그 데이터를 필터링, 검색 및 연관시켜 더 빠르고 효율적인 문제 처리, 향상된 성능 및 최적화된 리소스 활용을 수행할 수 있습니다.

## 애플리케이션 로그에서 사용자 정의 태그(레이블) 활성화

활성화되면 뉴렐릭 언어 에이전트가 모든 사용자 정의 태그(레이블)를 에이전트 전달 로그에 추가합니다.

지원 언어:

* [.NET](/docs/apm/agents/net-agent/configuration/net-agent-configuration) 에이전트 버전 10.4.0 이상.
* [루비](/docs.newrelic.com/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration/) 에이전트 버전 9.16.0 이상.
* 이동: 곧 출시
* [대응](/docs/apm/agents/java-agent/configuration/java-agent-configuration-config-file/) 에이전트 버전 8.19.0 이상.
* [Node.js](/docs/apm/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration): 에이전트 버전 12.8.0 이상.
* [Python](/docs/apm/agents/python-agent/configuration/python-agent-configuration) 에이전트 버전 10.3.0 이상.
* [PHP](/docs/apm/agents/php-agent/configuration/php-agent-configuration/) 에이전트 버전 11.7.0 이상.
