CI/CD 로그를 New Relic에 보내도록 [CircleCI 웹훅](https://circleci.com/docs/2.0/webhooks/) 서비스를 구성합니다.

## CircleCI에서 웹훅 트리거 설정 [#setup]

로그를 New Relic으로 전달하도록 CircleCI 웹훅을 설정하려면 다음 단계를 완료하십시오.

1. [CircleCI](https://app.circleci.com/projects) 에 로그인합니다.

2. CircleCI 프로젝트 중 하나로 이동합니다.

3. <DNT>**Project settings**</DNT> 을(를) 클릭합니다.

4. 프로젝트 설정의 사이드바에서 <DNT>**Webhooks**</DNT> 클릭합니다.

5. <DNT>**Add webhook**</DNT> 을(를) 클릭합니다.

6. 웹훅의 이름을 지정합니다.

7. 뉴렐릭의 <DNT>**Logs endpoint**</DNT>) 입력하세요.

   * 우리를:

     ```
     "https://log-api.newrelic.com/log/v1?Api-Key=YOUR_LICENSE_KEY"
     ```

   * 유럽 연합:

     ```
     "https://log-api.eu.newrelic.com/log/v1?Api-Key=YOUR_LICENSE_KEY"
     ```

   뉴렐릭 계정에 설정한 지역을 사용하세요. `YOUR_LICENSE_KEY` 뉴렐릭 <InlinePopover type="licenseKey"/>으로 바꾸세요.

8. 로그 푸시를 트리거하려는 이벤트 유형에 따라 <DNT>**Workflow**</DNT> 또는 <DNT>**Job**</DNT> 선택합니다.

9. 수신 API 또는 타사 서비스를 설정한 경우 <DNT>**Test ping event**</DNT> 클릭하여 테스트 이벤트를 만듭니다.
