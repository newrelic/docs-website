New Relic의 Android 앱용 사용자 인터페이스는 New Relic의 표준 사용자 인터페이스와 유사한 기능을 제공하며 모바일 사용자를 위한 맞춤형 세부 정보를 제공합니다. [New Relic 웹사이트](https://newrelic.com/mobility) 에서 자세히 알아보세요.

## 호환성 및 요구 사항 [#reqs]

* Android 8.0(Oreo) 이상
* 7인치 이하의 화면 크기

## 특징 [#features]

New Relic Android 앱에는 다음 기능에 대한 데이터가 포함되어 있습니다.

* 상태 맵을 포함한 실시간 및 [과거](#time) 데이터의 APM 메트릭. 거래 선택

  <Icon name="fe-repeat"/>

  아이콘을 클릭하여 자세한 거래 메트릭을 확인하거나 **개요** 차트를 클릭하여 상위 5개 거래의 요약 차트를 확인하세요. 선택

  <Icon name="fe-filter"/>

  아이콘을 클릭하여[레이블 및 범주별로](/docs/data-analysis/user-interface-functions/labels-categories-organize-your-apps-servers) 필터링합니다.

* 평균 페이지 로드 시간, Apdex, 평균 처리량 등을 포함한 브라우저 모니터링 메트릭.

* 인프라 모니터링.

* [충돌 보고서](/docs/mobile-monitoring/mobile-monitoring-ui/mobile-app-dashboards/mobile-apps-crash-dashboard) , 네트워크 오류, API 호출 및 활성 사용자 수를 포함한 모바일 모니터링.

* 장소에 관계없이 [모바일 경고](/docs/mobile-apps/new-relic-ios-app/features/ios-alerting) 를 비롯한 이벤트 알림과 [배포 알림](/docs/applications-menu/deployments-dashboard) 및 [메모](/docs/site/public-and-private-notes) .

## 데이터 프라이버시 [#privacy]

New Relic의 모바일 앱은 인증 및 문제 해결에 필요한 정보만 기록합니다.

* 이름과 성을 포함하여 New Relic 계정과 연결된 사용자의 이메일 주소(인증 목적으로만)
* IP 주소
* 장치 아이디

자세한 내용은 [모바일 데이터 개인 정보 보호 및 보안 문서](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps) 를 참조하십시오.
