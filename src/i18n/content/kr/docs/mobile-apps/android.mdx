New Relic Android 앱을 사용하면 어디에서나 데이터에 액세스할 수 있습니다. 큐레이션을 볼 수 있습니다. <InlinePopover type="dashboards"/>APM에서 인프라, 브라우저에 이르기까지 New Relic 기능 전체에서 구축한 모든 대시보드에 액세스할 수 있습니다. 더 깊이 들어가야 하는 경우 NRQL을 사용하여 데이터를 직접 쿼리하거나 로그를 검색할 수 있습니다. 또한 앱에서 바로 알림을 받을 수 있으므로 즉시 조사를 시작할 수 있습니다.

    ## New Relic 기능 [#features]

    * **경고** : 현재 및 과거 사건을 보고, 사건을 확인하고, 관련 엔터티에 대한 성능 데이터를 확인하기 위해 탭합니다.

    * **APM(애플리케이션 성능 모니터링)** : 트랜잭션, 오류, 데이터베이스, 외부 서비스 등에 대한 실시간 및 과거 데이터를 확인합니다.

    * **브라우저 모니터링** : 모든 프런트 엔드 웹 앱에서 처리량, 로드 시간, 오류, AJAX 요청 등을 추적합니다.

    * **대시보드 및 NRQL** : 대시보드를 보거나 NRQL을 사용하여 더 자세히 알아보십시오. 사용자 지정 NRQL 쿼리를 작성하거나 차트를 탭한 다음 오버플로를 탭하여 차트에 대한 NRQL을 볼 수 있습니다.

      <Icon name="fe-more-horizontal"/>

      아이콘을 클릭하고 **쿼리 보기** 를 선택합니다.

    * **탐색기** : AWS 서비스에서 네트워크 성능에 이르기까지 New Relic의 다른 모든 것을 탐색합니다.

    * **인프라 모니터링** : 클라우드, 전용 호스트 또는 오케스트레이션 환경에서 실행되는 컨테이너에서 실행되는 호스트의 상태를 확인합니다.

    * **모바일 모니터링** : 모든 모바일 앱에서 성능, 오류, 충돌 등을 확인합니다.

    * **종합 모니터링** : 각 모니터의 성능을 확인하고 개별 결과를 확인하고 장애를 추적합니다.

## 데이터 프라이버시 [#privacy]

당사의 모바일 앱은 인증 및 문제 해결에 필요한 정보만 기록합니다.

* 이름과 성을 포함하여 New Relic 계정과 연결된 사용자의 이메일 주소(인증 목적으로만)
* IP 주소
* 장치 아이디

자세한 내용은 [모바일 데이터 개인 정보 보호 및 보안 문서](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps) 를 참조하십시오.
