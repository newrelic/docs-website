뉴렐릭 Android 앱을 사용하면 어디서든 데이터에 액세스할 수 있습니다. APM , 대시보드, 브라우저에 이르기까지 뉴렐릭 기능 전체에서 선별된 대시보드를 보고 자신이 구축한 모든 대시보드에 액세스할 수 있습니다. 더 깊이 들어가야 하는 경우 NRQL을 사용하여 데이터를 직접 쿼리하거나 로그를 검색할 수 있습니다. 그리고 앱에서 직접 <InlinePopover type="alerts" />수신하여 즉시 조사를 시작할 수 있습니다.

    ## New Relic 기능 [#features]

    * <DNT>**Alerts**</DNT>: 현재 및 과거 공지 이벤트를 확인하고, 공지 이벤트를 확인하고, 탭하여 관련 기능 데이터를 확인하세요.
    * <DNT>**Application performance monitoring (APM)**</DNT>: 소니, 오류, 데이터베이스, 외부 서비스 등 전반에 걸쳐 데이터 및 이력 데이터를 확인하세요.
    * <DNT>**Browser monitoring**</DNT>: 모든 프런트엔드 웹 앱에서 처리량, 로드 시간, 오류, AJAX 요청 등을 추적합니다.
    * <DNT>**Dashboards and NRQL**</DNT>대시보드를 확인하거나 NRQL을 사용하여 더 자세히 살펴보세요. 사용자 지정 NRQL 쿼리를 작성하거나, 차트를 탭한 다음 오버플로를 탭하여 해당 차트의 NRQL을 볼 수 있습니다. <Icon name="fe-more-horizontal" /> 아이콘을 선택하고 <DNT>**View query**</DNT> 선택합니다.
    * <DNT>**Explorer**</DNT>: AWS 서비스부터 네트워크 성능까지 뉴렐릭의 모든 것을 찾아보세요.
    * <DNT>**Infrastructure monitoring**</DNT>: 클라우드, 전용 호스트 또는 오케스트레이션된 환경에서 실행되는 컨테이너에서 실행되는 호스트의 상태를 확인하세요.
    * <DNT>**Mobile monitoring**</DNT>: 모든 모바일 앱의 성능, 오류, 충돌 등을 확인하세요.
    * <DNT>**Synthetic monitoring**</DNT>: 각 모니터의 성능을 확인하고 개별 결과를 확인하며 오류를 추적합니다.
  </Side>

## 데이터 프라이버시 [#privacy]

당사의 모바일 앱은 인증 및 문제 해결에 필요한 정보만 기록합니다.

* 이름과 성을 포함하여 New Relic 계정과 연결된 사용자의 이메일 주소(인증 목적으로만)
* IP 주소
* 장치 아이디

자세한 내용은 [모바일 데이터 개인 정보 보호 및 보안 문서](/docs/mobile-monitoring/new-relic-mobile/get-started/security-mobile-apps) 를 참조하십시오.
