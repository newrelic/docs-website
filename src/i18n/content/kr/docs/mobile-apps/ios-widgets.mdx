DevOps 및 사이트 안정성 엔지니어링 성공의 열쇠는 MTTR(평균 해결 시간)을 줄이는 데 있습니다. 컴퓨터 앞에 있지 않을 때에도 위젯을 사용하면 그렇게 할 수 있습니다.

## 위젯을 시작하기 전에 [#requirements]

iPhone 또는 iPad에서 [iOS 위젯을](https://support.apple.com/en-us/HT207122) 설정하려면 iOS 14 이상을 실행 중이고 [New Relic 모바일 앱이](install-newrelic-ios-mobile-app) 설치되어 있는지 확인하세요.

1. 아이콘이 흔들릴 때까지 빈 영역이나 위젯을 길게 터치합니다.

2. 왼쪽 상단에 있는 <DNT>**+**</DNT> 아이콘을 탭하세요.

   <img
     style={{ maxWidth: '290px' }}

3. <DNT>**New Relic**</DNT> 위젯을 검색하세요.

4. 왼쪽이나 오른쪽으로 스와이프하여 세 가지 위젯 크기 중에서 선택한 다음 <DNT>**Add Widget**</DNT> 탭합니다.

## 위젯에 차트 추가 [#add-chart]

New Relic 앱에서 표시할 차트를 아직 선택하지 않았다면 다음 단계를 따르세요.

1. 위젯에서 보고 싶은 차트로 이동합니다.

2. 상세 보기를 표시하려면 차트를 탭하세요.

3. 더 많은 옵션을 보려면 오른쪽 상단에 있는 줄임표 아이콘 <DNT>**...**</DNT> 을 탭하세요.

   <img
     style={{ maxWidth: '290px' }}
     title="Screenshot showing the dropdown options in the ellipsis"
     alt="Screenshot showing the dropdown options in the ellipsis menu"
     src="/images/mobile-apps_screenshot-full_iOS-widgets-add-widget-from-chart.webp"
   />

4. <DNT>**Add to widgets**</DNT> 선택합니다.

5. <DNT>**Manage widgets**</DNT> 사용하여 위젯을 제거하고 재정렬할 수도 있습니다.

## 위젯용 차트 지정 [#designate-charts]

각 iOS 위젯에 대해 표시할 차트를 선택하려면:

1. 위젯이 흔들릴 때까지 길게 누릅니다.

2. <DNT>**Select chart**</DNT> 또는 차트 이름을 탭합니다.
