## 요구 사항 [#requirements]

[New Relic One 사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 과 [원래 사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 모두의 사용자는 모바일 푸시를 [대상](https://docs.newrelic.com/docs/alerts-applied-intelligence/notifications/notification-integrations#mobile_push) 으로 하는 워크플로를 설정할 수 있습니다.

## 알림 켜기 또는 끄기 [#enabling-alerts]

Android 또는 iOS 앱에서 New Relic 계정에 로그인하면 기기가 자동으로 사용자와 연결됩니다. [워크플로에 모바일 푸시 대상을 추가](/docs/alerts-applied-intelligence/applied-intelligence/incident-workflows/incident-workflows#add-workflow) 해야 합니다.

## 문제 인시던트 세부정보 보기 [#viewing-alerts]

알림은 기기의 잠금 화면에 자동으로 나타납니다. New Relic 앱을 시작하려면:

1. 장치에 따라 다음을 수행하십시오.

* Android 기기의 경우 알림 창에서 알림을 탭하세요.
* iOS 기기에서는 화면을 스와이프합니다.

2. New Relic 앱의 **Alerts & AI** 메뉴에서 문제를 선택하면 연결된 애플리케이션에 대한 오류 세부 정보를 볼 수 있습니다.
3. 선택적으로 **승인** 을 선택합니다.
4. 선택적으로 추가 세부 정보를 보려면 **개요** , **인시던트** 또는 **문제 로그** 를 선택합니다.

기본 메뉴의 **알림** 목록은 시간별로 정렬된 문제를 보여줍니다. 추가 필터를 적용하거나 오른쪽 상단에서 시간 창을 변경할 수 있습니다.

## 경고 설정 문제 해결 [#troubleshooting]

푸시 알림이 모바일 장치에서 작동하지 않는 경우:

1. [요구 사항](#requirements) 을 충족하는지 확인합니다.
2. [경고가 활성화](#enabling-alerts) 되어 있는지 확인합니다.
3. 모바일 장치의 [알림 설정](#notification-settings) 을 확인하여 New Relic이 푸시 알림을 보낼 수 있는지 확인하십시오.

모바일 장치에 대한 알림 설정이 올바르지만 여전히 알림을 받지 못하는 [경우 계정에서 장치를 삭제](#account-delete) 한 다음 New Relic 애플리케이션을 [제거](#uninstall-app) 하고 [다시 설치](#reinstall-app) 하십시오.

    id="account-delete"
    title="New Relic 계정에서 Android 또는 iOS 기기 삭제"
  >
    New Relic 계정에서 모바일 장치를 삭제하려면 웹 브라우저에서 공개 graphql api [api.newrelic.com/graphiql](https://api.newrelic.com/graphiql) 을 사용하세요.

    1. Actor -> mobilePushNotification -> devices를 선택하고 appVersion, deviceId 및 deviceName을 선택하여 [현재 장치](https://api.newrelic.com/graphiql?#query=query%20List%20%7B%0A%20%20actor%20%7B%0A%20%20%20%20mobilePushNotification%20%7B%0A%20%20%20%20%20%20devices%20%7B%0A%20%20%20%20%20%20%20%20appVersion%0A%20%20%20%20%20%20%20%20deviceId%0A%20%20%20%20%20%20%20%20deviceName%0A%20%20%20%20%20%20%20%20operatingSystem%0A%20%20%20%20%20%20%7D%0A%20%20%20%20%7D%0A%20%20%7D%0A%7D%0A) 를 **쿼리** 합니다. 이 쿼리를 실행하여 장치 목록을 가져옵니다.

    2. 돌연변이 -> **mobilePushNotificationRemoveDevice** 를 선택하고 위의 목록에서 deviceId를 전달하여 장치를 [제거](https://api.newrelic.com/graphiql?#query=mutation%20%7B%0A%20%20mobilePushNotificationRemoveDevice%28deviceId%3A%20%22%22%29%20%7B%0A%20%20%20%20message%0A%20%20%7D%0A%7D%0A) 하도록 변형합니다.

       또는 메뉴 -> 설정의 인앱 **설정** 옵션에서 장치를 제거할 수 있습니다.

    3. **푸시 알림 장치** 아래에서 제거하십시오. iOS의 경우 오른쪽에서 왼쪽으로 밀어 기기 **삭제** , Android의 경우 **삭제** 를 탭합니다.

       장치에서 New Relic 앱을 [다시 설치](#uninstall-app) 하는 단계를 계속하십시오.

  >
    New Relic 모바일 앱을 다시 설치하려면:

    1. Android 기기에서 [**Google Play 스토어**](https://play.google.com/store/apps/details?id=com.newrelic.rpm) 를 선택합니다.

       또는

       iOS 기기의 홈 화면에서 [**App Store** 를 선택합니다.](https://itunes.apple.com/us/app/new-relic/id594038638?mt=8)

    2. **New Relic** 을 검색하세요.

    3. 앱을 다운로드합니다.

    4. 다운로드가 완료되면 New Relic 계정으로 New Relic 모바일 앱에 로그인합니다.
