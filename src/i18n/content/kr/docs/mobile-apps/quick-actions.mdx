빠르게 변화하는 엔지니어링 및 관찰 가능성의 세계를 따라가려면 빠른 작업을 사용하여 홈 화면에서 바로 New Relic의 일반 기능에 빠르게 액세스하세요.

빠른 작업은 즐겨찾는 엔터티를 최대 4개까지 표시합니다. 즐겨찾기가 4개 이상인 경우 가장 최근에 사용한 항목이 표시됩니다. 즐겨찾기를 설정하지 않은 경우 가장 많이 사용되는 기능(APM, 경고, 종합, 대시보드)에 대한 링크가 디스플레이에 표시됩니다.

이 기능을 사용하려면 홈 페이지에서 New Relic 앱 아이콘을 길게 누르면 바로가기가 표시됩니다. 여기 몇 가지 예가 있어요.
