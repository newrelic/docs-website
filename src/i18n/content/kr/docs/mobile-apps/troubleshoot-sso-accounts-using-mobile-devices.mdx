일반적으로 New Relic 모바일 앱에 로그인하면 세션이 자동으로 웹 브라우저로 리디렉션됩니다. 여기에서 New Relic 계정에 로그인할 수 있습니다. 다음은 SAML-SSO가 활성화된 계정으로 New Relic 모바일 앱을 사용하는 데 문제가 있는 경우 문제 해결 팁입니다.

## 사용자 이름 또는 암호 없음 [#no-sso-user]

일부 SAML 공급자가 귀하의 암호를 덮어쓰거나 관리자가 이 정보를 보내지 않았기 때문에 New Relic의 사용자 이름이나 암호가 없을 수 있습니다. 이러한 상황에서:

1. 모바일 앱의

   <DNT>
     **Log in**
   </DNT>

   에서

   <DNT>
     **I don't have a password**
   </DNT>

   링크를 선택합니다.

2. 모바일 장치를 사용하여 이메일 계정을 엽니다.

3. 이메일 계정에서 20분 이내에 New Relic 인증 이메일을 검색하세요.

4. 이메일에서

   <DNT>
     **Authenticate**
   </DNT>

   버튼이나 그 아래에 있는 링크를 선택하세요.

## 로그인 후 오류 [#sso-errors]

모바일 기기로 SSO 공급자에 성공적으로 로그인한 후 오류가 표시되면 데스크톱 웹 브라우저로 <DNT>**[one.newrelic.com](https://one.newrelic.com/all-capabilities)**</DNT> 에 로그인할 수 있는지 확인하세요.

* 그렇지 않은 경우 관리자에게 문의하십시오.
* 그렇다면 [support.newrelic.com](https://support.newrelic.com) 에서 지원을 받으십시오.

## 재인증 문제 [#sso-reauth]

SAML-SSO 계정에서 재인증을 사용하는 경우 기본 계정에 로그인해야 합니다. (다른 모든 계정은 회색으로 표시됩니다.) 회색으로 표시된 계정으로 전환하려고 하면 현재 지원되지 않는다는 오류 메시지가 표시됩니다.
