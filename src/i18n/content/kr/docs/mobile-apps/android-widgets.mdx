## Android 위젯 추가 [#add-widget]

위젯을 추가하려면 다음을 완료하세요.

1. 모달이 나타날 때까지 빈 공간을 길게 터치하세요.
2. **Widgets** \[위젯 을] 탭합니다.
3. New Relic 위젯을 검색하세요.
4. 위젯을 길게 탭하여 기기의 원하는 위치로 이동하세요.
5. 활성 문제를 계산하려는 계정과 기간을 선택하세요.
6. 필요에 따라 길게 누르고 크기를 조정하세요.

위젯은 현재 보고 있는 계정으로 홈 화면에 추가되며, 시간은 30분으로 설정됩니다.

## 위젯에 차트 추가 [#add-chart]

New Relic 앱에서 표시할 차트를 아직 선택하지 않았다면 다음 단계를 따르세요.

1. 위젯에서 보고 싶은 차트로 이동합니다.
2. **+ Create Widget** \[+ 위젯 만들기를] 탭하세요.
3. 하단에 나타나는 **Add to home screen** \[홈 화면에 추가] 버튼을 누릅니다.
4. 필요에 따라 길게 누르고 크기를 조정하세요.
5. 홈 화면 위젯을 탭하여 **Issue** \[이슈] 목록에 New Relic 앱을 엽니다.
