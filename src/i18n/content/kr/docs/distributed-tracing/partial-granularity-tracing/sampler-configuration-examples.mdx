기본 샘플링 설정은 모든 서비스에서 작동하지만 최적의 설정은 아닐 수 있습니다. APM 은 샘플링 설정 옵션을 제공하여 서비스 요구 사항에 맞게 샘플링을 사용자 지정할 수 있도록 합니다. 이 페이지에서는 다양한 유형의 서비스에 대한 몇 가지 실제 사례와 이러한 요구 사항에 맞게 샘플링 설정을 수정하는 방법을 안내합니다.

## 예시 1: 상류 샘플링으로 인해 포착된 엔드포인트 분포가 불량함

기본 설정은 적응형 샘플러를 사용합니다. 이는 상위 서비스에서 샘플링된 모든 수신 트레이스가 현재 서비스에서도 샘플링된다는 것을 의미합니다. 때때로 이로 인해 특정는 많이 샘플링되는 반면 다른는 무시되고 거의 샘플링되지 않을 수 있습니다.

### 해결책 1: 상위 서비스에서 샘플링된 트레이스를 샘플링하기 위해 비율 샘플러를 사용합니다.

다음 설정은 상위 서비스에서 샘플링한 수신 트레이스를 10%의 확률로만 샘플링하여 이 문제를 해결합니다. 이는 상위 서비스에서 샘플링한 트랜스로 스팬 리저버를 채우지 않음으로써 다른 트랜스를 샘플링할 공간을 확보합니다.

```json
{
    distributed_tracing: {
        sampler: {
            remote_parent_sampled: {
                trace_ratio_id_based: {
                    ratio: 0.1
                }
            }
        }
    }
}
```

### 해결책 2: 비율 샘플러를 사용하여 상위 서비스에서 샘플링된 트레이스의 일부를 전체 해상도로 캡처하고 나머지는 부분 해상도로 캡처합니다.

상위 서비스에서 샘플링할 때 트레이에 대한 최소한의 정보를 계속 캡처해야 하는 경우 다음 설정을 대신 사용할 수 있습니다. 이 설정은 상위 서비스에서 샘플링된 모든 트래픽의 5%를 전체 세분화 수준으로 캡처하고 나머지는 부분 세분화 수준으로 캡처합니다. 상위 서비스에서 샘플링된 대부분의 데이터를 부분적인 세분성으로 캡처함으로써 스팬 저장소가 빠르게 채워지지 않고 다른 데이터를 샘플링할 공간을 확보할 수 있습니다.

```json
{
    distributed_tracing:{
        sampler: {
            remote_parent_sampled: {
                trace_ratio_id_based: {
                    ratio: 0.5
                }
            }
            partial_granularity: {
                enabled: true,
                type: "compact",  # If capturing AI data set this to "essential"
                remote_parent_sampled: always_on
            }
        }
    }
}
```

## 예시 2: 일부 엔드포인트가 거의 호출되지 않아 캡처된 엔드포인트 분포가 불량함

기본 설정은 적응형 샘플러를 사용합니다. 즉, 분당 약 10개의 트레이스를 포착한다는 의미입니다. 무작위 확률로 인해 적응형 샘플러는 더 자주 발생하는 엔드포인트 호출을 포착할 가능성이 더 높습니다. 이는 호출 빈도가 낮은 엔드포인트는 누락될 가능성이 높고 샘플링되지 않을 가능성이 크다는 것을 의미합니다.

### 해결책 1: 상위 서비스에서 샘플링된 트레이스를 샘플링하기 위해 비율 샘플러를 사용합니다.

다음 설정은 상위 서비스에서 샘플링한 수신 트레이스를 10%의 확률로만 샘플링하여 이 문제를 해결합니다. 이는 상위 서비스에서 샘플링한 트랜스로 스팬 리저버를 채우지 않음으로써 다른 트랜스를 샘플링할 공간을 확보합니다.

```json
{
    distributed_tracing: {
        sampler: {
            remote_parent_sampled: {
                trace_ratio_id_based: {
                    ratio: 0.1
                }
            }
        }
    }
}
```
