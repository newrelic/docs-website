[분산 추적](/docs/introduction-distributed-tracing) 은 New Relic에서 쿼리할 수 있는 `Span` 데이터를 생성합니다. 다음은 `Span` 데이터에 대해 자세히 알아보는 방법입니다.

* 스팬 데이터를 탐색하기 위해 [쿼리 빌더](/docs/chart-builder/use-chart-builder/get-started/introduction-chart-builder) 를 사용할 수 있습니다.
* 스팬 데이터에 첨부된 기본 속성을 보려면 [데이터 사전](/attribute-dictionary/?event=Span) 을 사용하십시오.
* 이러한 속성을 사용하는 NRQL 쿼리에 대한 도움말은 다음 [예제 쿼리](/docs/apm/distributed-tracing/ui-data/example-insights-queries-distributed-trace-data) 를 참조하세요.
