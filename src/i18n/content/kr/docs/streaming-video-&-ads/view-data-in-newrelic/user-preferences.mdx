<DNT>**Video Overview**</DNT> 또는 <DNT>**Ad Overview**</DNT> 대시보드를 구성할 때 뉴렐릭은 설정을 사용자 프로필에 저장합니다. 다음에 두 대시보드 중 하나로 돌아오면 설정이 복원되므로 처음부터 다시 시작할 필요가 없습니다.

환경설정은 사용자 및 대시보드 유형별로 저장됩니다. <DNT>**Video Overview**</DNT> 설정은 <DNT>**Ad Overview**</DNT> 설정에 영향을 미치지 않으며, 그 반대의 경우도 마찬가지입니다.

## 환경설정이 저장되는 방식 [#how-saving-works]

변경 사항은 백그라운드에서 비동기적으로 기록되므로 저장하는 동안 대시보드가 응답성을 유지합니다. 빠른 조정을 수행할 때 불필요한 쓰기를 줄이기 위해 저장이 잠시 일괄 처리됩니다.

저장된 환경설정은 다음 전체에서 유지됩니다:

* 브라우저 새로고침
* 로그아웃 및 로그인
* 동일한 계정으로 로그인되어 있는 한, 다른 브라우저 또는 기기

## 기본 동작 [#defaults]

대시보드에 저장된 기본 설정이 없는 경우(예: 첫 방문 시) 대시보드는 제품 기본 설정으로 열립니다.

## 기본값으로 재설정 [#reset-to-defaults]

<DNT>**Reset to default view**</DNT> 컨트롤을 사용하면 저장된 기본 설정을 지우고 대시보드를 원래 제품 기본값으로 되돌릴 수 있습니다.

<Callout variant="tip">
  재설정은 현재 보고 있는 대시보드에만 영향을 미칩니다. <DNT>**Video Overview**</DNT> 환경 설정을 재설정해도 <DNT>**Ad Overview**</DNT> 환경 설정에는 영향을 미치지 않습니다.
</Callout>

## 오류 처리 [#error-handling]

대시보드가 로드될 때 환경 설정 검색에 실패하면 대시보드는 제품 기본 설정으로 대체되고 오류 공지를 표시합니다. 워크플로우가 차단되지 않습니다.

환경설정 저장에 실패해도 대시보드는 계속 정상적으로 작동합니다.

이전에 저장된 메트릭, 차원 또는 패싯을 더 이상 사용할 수 없는 경우(예: 더 이상 사용되지 않기 때문에) 대시보드는 해당 설정에 대해 사용 가능한 첫 번째 옵션과 같은 합리적인 기본값으로 대체하므로 위젯이 비어 있거나 손상된 상태로 남지 않습니다.
