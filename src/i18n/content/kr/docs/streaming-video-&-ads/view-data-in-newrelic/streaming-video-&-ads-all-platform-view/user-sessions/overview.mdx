스트리밍 비디오 및 광고를 위한 **All Platforms &gt; User Sessions**은 모든 애플리케이션의 모든 세션을 나열합니다.

## 사용자 세션 페이지 UI 보기

**User Session page** \[사용자 세션 페이지] 테이블에는 각 세션에 대한 다양한 정보를 표시하는 여러 열이 있습니다. 다음 표는 세션 페이지 UI에서 사용할 수 있는 각 열에 대한 정보를 제공합니다.

<Callout variant="important">
  세션 표는 표 위에서 선택한 시간 필터 내에 시작된 모든 사용자 세션을 보여줍니다.
</Callout>
