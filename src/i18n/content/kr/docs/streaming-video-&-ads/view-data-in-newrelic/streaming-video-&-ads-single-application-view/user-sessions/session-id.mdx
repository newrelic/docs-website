## 개요

재생 타임라인 페이지는 황금 지표, 메타데이터 및 스트리밍 이벤트의 전체 타임라인을 포함하여 특정 사용자 보기 세션에 대한 자세한 정보를 제공합니다. 이 페이지는 사용자가 애플리케이션 내에서 콘텐츠를 시청하는 동안 정확히 어떤 일이 발생했는지 이해하고 재생 문제를 해결하는 데 도움이 됩니다.

### 재생 타임라인으로 할 수 있는 일

이 페이지를 사용하여 다음을 수행하십시오.

* 시작 시간, 버퍼 시간, 평균 비트 전송률 및 오류를 포함한 세션 수준의 골든 인프라를 확인하세요.
* 특정 시청 세션을 식별하는 고유 세션 ID에 액세스하세요.
* 엔티티 정보, 플랫폼 세부 정보 및 세션 메타데이터를 검토하십시오.
* 시청 세션 길이와 총 시청 시간을 모니터링하세요.
* 재생 중 발생한 비트레이트 변화를 분석합니다.
* PLAYER\_READY부터 CONTENT\_END까지의 스트리밍 이벤트 전체 타임라인을 확인하세요.
* 버퍼링 이벤트(CONTENT\_BUFFER\_START 및 CONTENT\_BUFFER\_END)를 추적합니다.
* 세션 중 일시 중지 및 재개 이벤트를 식별합니다.
* 재생 중 30초마다 전송되는 콘텐츠 하트비트 이벤트를 모니터링합니다.
* 세션 중에 발생한 오류를 자세한 이벤트 정보를 통해 진단합니다.

### 이 페이지를 누가 사용하나요?

이 페이지는 버그 개발자 문제 해결, 앱 내 문제 해결, 고객 지원 팀이 특정 보기 세션에 대한 불만 사항을 해결하고 품질 보증 엔지니어가 전체 이벤트 타임라인을 검사하여 재생 문제를 조사하는 데 유용합니다.
