## 사용자 세션 스트림 이벤트

사용자 세션 스트리밍 이벤트는 세션 중에 생성되는 이벤트입니다. 이러한 이벤트는 세션 동안 사용자의 동작을 추적하는 데 사용됩니다.

세션 중 생성된 이벤트를 보여주는 그래프와 다음과 같은 이벤트에 대한 자세한 정보가 사용자 세션 페이지에 표시됩니다.
