## 개요

User Sessions \[사용자 세션] 페이지는 특정 모니터링 대상 애플리케이션에 대한 개별 보기 세션을 자세히 보여줍니다. 이 페이지는 단일 스트리밍 애플리케이션 또는 플랫폼 내에서 사용자 시청 행동을 추적하고, 오류가 있는 세션을 식별하고, 재생 패턴을 분석하는 데 도움이 됩니다.

### User Sessions \[사용자 세션]을 활용하여 할 수 있는 일

이 페이지를 사용하여 다음을 수행하십시오.

* 비디오 에이전트가 설치된 특정 애플리케이션에 대한 모든 사용자 시청 세션을 확인합니다.
* 뷰 ID, 세션 ID, 사용자 ID를 포함한 세션 수준의 세부 정보를 추적합니다.
* 제목별로 시청 중인 콘텐츠를 모니터링하세요.
* 문제 해결, 해결을 위해 오류가 있는 세션 식별
* 시청 시간 분석을 위해 세션 시작 및 종료 시간을 확인하세요.
* &apos;ID 보기&apos; 또는 &apos;세션 ID 보기&apos;를 클릭하면 자세한 세션 정보를 확인할 수 있습니다.
* 특정 기간에 집중하려면 시간 범위를 기준으로 세션을 필터링하세요.
* 특정 기기 및 플랫폼의 시청 패턴을 분석합니다.
* 각 세션의 콘텐츠 세부 정보를 검토하세요.

### 이 페이지를 누가 사용하나요?

이 페이지는 특정 애플리케이션에 대한 사용자 보고 문제를 해결하고, 개별 시청 세션을 분석하고, 해당 플랫폼 내 사용자 행동 패턴을 파악해야 하는 애플리케이션 개발자, 고객 지원 팀 및 품질 보증 엔지니어에게 유용합니다.

## User Sessions \[사용자 세션] 페이지

스트리밍 비디오 및 광고를 위한 **Single platform &gt; User Sessions**은 특정 애플리케이션에서 발생한 모든 세션을 나열합니다.
