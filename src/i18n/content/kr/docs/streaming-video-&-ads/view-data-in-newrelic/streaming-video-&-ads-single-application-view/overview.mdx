뉴렐릭 스트리밍 비디오 및 광고 개별 플랫폼 보기에 대한 주요 기능을 확인하세요.

* [요약](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/media-summary-page): 비디오 에이전트가 설치된 특정 애플리케이션/플랫폼의 요약을 확인합니다.
* [광고 개요](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/ad-overview): 비디오 에이전트가 설치된 특정 애플리케이션/플랫폼에서 광고 개요를 확인합니다.
* [사용자 세션](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/user-sessions/overview): 비디오 에이전트가 설치된 특정 애플리케이션/플랫폼의 사용자 세션을 확인합니다.
* [공지 조건](/docs/tutorial-create-alerts/create-an-alert): 비디오 에이전트가 설치된 특정 기능/플랫폼에 대한 공지 조건을 확인하세요.
* [오류](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/error): 문제가 있는 영역을 파악하고 수정의 우선순위를 정합니다.
