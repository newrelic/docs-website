이 페이지에서는 CocoaPods를 사용하여 스트리밍 비디오 및 광고 에이전트를 설치할 때의 설치 단계에 대한 개요를 제공합니다. 코드 예제를 포함한 전체 설치를 위해서는 [안내 설치를](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios) 이용하세요.

<Steps>
  <Step>
    ### 의존성/종속성 추가

    Podfile에 에이전트 파드 비디오를 추가하세요.
  </Step>

  <Step>
    ### 패드 설치

    터미널에서 `pod install` 실행하세요.
  </Step>

  <Step>
    ### 프로젝트 구축

    **Xcode** 에서 프로젝트를 열고 빌드하세요.
  </Step>

  <Step>
    <DNT>**Continue**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    ### 비디오 트래커를 초기화합니다.

    사용 중인 구현 방식에 맞는 에이전트 유형을 선택하십시오.

    * **AVPlayer 트래커가 포함된 비디오 에이전트만 해당**: 비디오 재생 추적용
    * **AVPlayer 및 IMA 트래커를 포함하는 비디오 에이전트**: 비디오 재생 및 광고 추적용
  </Step>

  <Step>
    스트리밍 데이터를 뉴렐릭에서 보려면 <DNT>**See your data**</DNT> 클릭하세요.
  </Step>
</Steps>
