이 페이지에서는 스트리밍 비디오 및 광고 에이전트를 수동으로 설치할 때의 설치 단계를 간략하게 설명합니다. 코드 예제를 포함한 전체 설치를 위해서는 [안내 설치를](/docs/streaming-video-&-ads/installation/ios/get-started-with-ios) 이용하세요.

<Steps>
  <Step>
    [video-에이전트-iOS](https://github.com/newrelic/video-agent-ios) 를 복제하세요.
  </Step>

  <Step>
    Xcode를 사용하여 `.xcodeproj` 파일 각각을 엽니다.
  </Step>

  <Step>
    적절한 방식을 선택하십시오.
  </Step>

  <Step>
    빌드(CMD+B).
  </Step>

  <Step>
    생성된 `.framework` 파일을 프로젝트에 포함시키세요.
  </Step>

  <Step>
    <DNT>**Continue**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    ### 비디오 트래커를 초기화합니다.

    사용 중인 구현 방식에 맞는 에이전트 유형을 선택하십시오.

    * **AVPlayer 트래커가 포함된 비디오 에이전트만 해당**: 비디오 재생 추적용
    * **AVPlayer 및 IMA 트래커를 포함하는 비디오 에이전트**: 비디오 재생 및 광고 추적용
  </Step>

  <Step>
    스트리밍 데이터를 뉴렐릭에서 보려면 <DNT>**See your data**</DNT> 클릭하세요.
  </Step>
</Steps>
