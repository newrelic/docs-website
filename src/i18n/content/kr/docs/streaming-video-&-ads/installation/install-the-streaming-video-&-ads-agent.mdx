비디오 에이전트를 설치하려면 안내 설치 과정을 따르는 것이 좋습니다.

## 전제 조건 [#requirements]

비디오 에이전트를 설치하기 전에 뉴렐릭 [계정 ID가](/docs/accounts/accounts-billing/account-setup/account-id) 있는지 확인하세요.

## 비디오 에이전트 설치 [#installation]

스트리밍 비디오 및 광고를 시작하려면 스트리밍 미디어 자산에 비디오 에이전트를 설치하세요.

<Steps>
  <Step>
    <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents &gt; Streaming Video &amp; Ads**</DNT> 으로 이동합니다.
  </Step>

  <Step>
    드롭다운 메뉴에서 작업하려는 계정을 선택하고 **Continue** \[계속을] 클릭하세요.
  </Step>

  <Step>
    ## 스트리밍 앱 유형을 선택하세요 [#select-streaming-app-type]

    * [브라우저](/install/browser-media): 브라우저 애플리케이션에서 비디오 플레이어를 모니터링합니다.

    * 모바일: 다음 플랫폼에서 비디오 플레이어를 모니터링합니다.

      * [Android](/install/android): Android 안드로이드에서 비디오 플레이어를 모니터링하세요.
      * [iOS](/install/media-ios): iOS에서 비디오 플레이어를 모니터링합니다.

    * [Roku](/docs/streaming-video-&-ads/installation/roku/get-started-with-roku): Roku 애플리케이션에서 비디오 플레이어를 모니터링합니다.
  </Step>
</Steps>
