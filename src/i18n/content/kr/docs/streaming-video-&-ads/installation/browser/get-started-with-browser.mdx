브라우저에서 비디오 플레이어 성능을 모니터링하려면 npm 또는 JavaScript 스니펫을 사용하여 뉴렐릭 비디오 에이전트를 설치하세요.

## 전제 조건 [#before-start]

브라우저용 스트리밍 비디오 및 광고 에이전트를 설치하기 전에 애플리케이션이 다음 요구 사항을 충족하는지 확인하십시오.

* [브라우저 모니터링 호환성 요구 사항](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring)
* [스트리밍 비디오 및 광고 에이전트 호환성 요구 사항](/docs/streaming-video-&-ads/installation/compatibility-requirements)

## 에이전트 설치 [#install]

<Steps>
  <Step>
    ### 새로운 유물에 액세스

    <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT> 으로 이동하여 통합 마켓플레이스로 이동하세요.
  </Step>

  <Step>
    ### 스트리밍 비디오 및 광고를 선택하세요

    <DNT>**Streaming Video &amp; Ads**</DNT> 타일을 검색하고 선택하세요.
  </Step>

  <Step>
    ### 계정 선택

    드롭다운 메뉴에서 <DNT>**Account**</DNT> 을 선택하고 <DNT>**Continue**</DNT> 클릭하세요.
  </Step>

  <Step>
    ### 플랫폼을 선택하세요

    <DNT>**Browser**</DNT> 선택하고 <DNT>**Continue**</DNT> 클릭합니다.
  </Step>

  <Step>
    ### 비디오 플레이어를 선택하세요

    원하는 비디오 플레이어를 선택하고 <DNT>**Continue**</DNT> 클릭하세요.
  </Step>

  <Step>
    ### 설치 방법을 선택하세요

    <DocTiles>
      <DocTile title="npm을 사용하여 설치하세요" path="/docs/streaming-video-&-ads/installation/browser/install-using-npm">
        JavaScript용 비디오 에이전트 에이전트 매니저를 설치합니다.
      </DocTile>

      <DocTile title="프런트엔드 코드에 JavaScript 스니펫을 배치하세요" path="/docs/streaming-video-&-ads/installation/browser/install-using-snippet">
        귀하의 제작에 JavaScript 코드 스니펫을 포함시켜 비디오 에이전트를 설치하세요.
      </DocTile>
    </DocTiles>
  </Step>
</Steps>

## Alternative: 안내 설치

위의 수동 단계를 건너뛰고 안내에 따라 인스톨러를 사용하세요. 아래 <DNT>**Guided install**</DNT> 클릭하세요. 또는 귀하의 조직이 [EU 데이터센터](/docs/using-new-relic/welcome-new-relic/get-started/our-eu-us-region-data-centers) 에 있는 경우 <DNT>**EU Guided install**</DNT> 클릭하세요.

## 다음은 뭐지?

에이전트를 설치한 후:

1. 스트리밍 비디오 및 광고 대시보드에서 기능을 확인하여 데이터가 뉴렐릭으로 흐르고 있는지 확인하세요.
2. [데이터 보기](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-features): 모니터링 기능 및 대시보드 살펴보기
3. [경험 품질(Quality of Experience) 구성, 참고](/docs/streaming-video-&-ads/view-data-in-newrelic/streaming-video-&-ads-single-application-view/application): 귀하의 성능에 대한 성능 지표를 사용자 정의하세요.
