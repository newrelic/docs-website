난독화 룰을 사용하면 에이전트가 뉴렐릭으로 전송하기 전에 민감한 데이터를 마스크할 수 있습니다. 각 룰은 대체 문자열과 쌍을 이루는 정규식 패턴입니다. 전송 전에 직렬화된 JSON 페이로드에 규칙이 적용됩니다.

## 구성 [#configuration]

에이전트 설정에서 `obfuscate` 어레이를 전달합니다.

```js
config: {
  obfuscate: [
    { regex: /\/user\/[^\/?"]+/, replacement: '/user/[REDACTED]' },
    { regex: /token=[^&"]+/, replacement: 'token=[MASKED]' },
  ]
}
```

## 룰 순서 지정 [#rule-ordering]

규칙은 순차적으로 적용됩니다 ― 한 규칙의 출력이 다음 규칙으로 전달됩니다. 잘못된 정규식 패턴은 건너뛰며 `Log.warn`을(를) 통해 경고 로그가 기록됩니다.
