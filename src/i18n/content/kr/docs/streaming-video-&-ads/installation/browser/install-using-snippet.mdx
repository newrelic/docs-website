귀하의 제작에 JavaScript 코드 스니펫을 포함시켜 비디오 에이전트를 설치하세요.

## 시작하기 전에

먼저 [일반적인 설치 단계를](/docs/streaming-video-&-ads/installation/browser/get-started-with-browser#install) 완료하십시오.

## JavaScript 스니펫을 프런트엔드 코드에 배치하세요 [#install-snippet]

<Steps>
  <Step>
    ### 애플리케이션 이름 지정

    애플리케이션 이름을 입력하고 <DNT>**Save and Continue**</DNT> 클릭하세요.
  </Step>

  <Step>
    ### 에이전트 설치

    스니펫 코드를 복사하고 터미널(Mac) 또는 단일(Windows)에서 실행하여 뉴렐릭의 최신 릴리스를 설치하세요.
  </Step>

  <Step>
    ### 애플리케이션에 코드를 추가하세요

    파일을 다운로드한 후, 플랫폼에 표시된 스니펫 코드를 포함하여 비디오 플레이어 모니터링을 시작하세요.
  </Step>

  <Step>
    ### 애플리케이션 ID 생성

    <DNT>**Get Application Info**</DNT> 선택하세요. 이렇게 하면 브라우저 엔티티 생성을 위한 애플리케이션 ID가 생성됩니다.
  </Step>

  <Step>
    ### 데이터 보기

    <DNT>**See your data**</DNT> 을(를) 클릭합니다.
  </Step>
</Steps>

<Callout variant="important">
  애플리케이션과 상호 작용하고 데이터를 생성하는 데 몇 분 정도 소요될 수 있습니다.
</Callout>
