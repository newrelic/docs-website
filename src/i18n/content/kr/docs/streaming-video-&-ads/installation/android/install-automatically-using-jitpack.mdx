이 페이지에서는 JitPack을 사용하여 스트리밍 비디오 및 광고 에이전트를 설치할 때의 설치 단계에 대한 개요를 제공합니다. 코드 예제를 포함한 전체 설치를 위해서는 [안내 설치를](https://onenr.io/02R54MqY0wb) 이용하세요.

<Steps>
  <Step>
    ## JitPack 저장소를 추가합니다.

    루트 `build.gradle` 파일에 JitPack 저장소를 추가하세요.
  </Step>

  <Step>
    ## 의존성/종속성 추가

    앱의 `build.gradle` 파일에 에이전트 의존성/종속성 비디오를 추가하고 <DNT>**Continue**</DNT> 클릭합니다.
  </Step>

  <Step>
    ## 비디오 에이전트를 구성합니다.

    **기능 클래스 또는 MainActivity의** `onCreate` 메소드에 NRVideo 설정을 추가하세요.
  </Step>

  <Step>
    ## 동영상 플레이어 추적 설정

    사용 중인 구현 방식에 맞는 에이전트 유형을 선택하십시오.

    * **ExoPlayer 트래커가 포함된 비디오 에이전트(비디오 재생 추적 전용)**: 비디오 재생 추적용

    * **ExoPlayer 및 IMA 트래커를 사용하는 비디오 에이전트**: 비디오 재생 및 광고 추적용

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    스트리밍 데이터를 뉴렐릭에서 보려면 <DNT>**See your data**</DNT> 클릭하세요.
  </Step>
</Steps>
