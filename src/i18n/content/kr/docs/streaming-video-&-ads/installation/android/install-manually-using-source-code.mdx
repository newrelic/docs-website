이 페이지에서는 소스 코드를 사용하여 스트리밍 비디오 및 광고 에이전트를 수동으로 설치할 때의 설치 단계에 대한 개요를 제공합니다. 코드 예제를 포함한 전체 설치를 위해서는 [안내 설치를](/docs/streaming-video-&-ads/installation/android/get-started-with-android) 이용하세요.

<Steps>
  <Step>
    [video-에이전트-android](https://github.com/newrelic/video-agent-android) 를 복제하세요.
  </Step>

  <Step>
    Android 프로젝트에서 <DNT>**File &gt; New &gt; Import Module**</DNT> 으로 이동합니다.
  </Step>

  <Step>
    라이브러리 모듈 디렉터리의 위치를 입력하고 <DNT>**Finish**</DNT> 클릭하십시오.

    <Callout variant="important">
      라이브러리 모듈 디렉터리는 방금 복제한 리포지터리에 있습니다. 예시 모듈: `NewRelicVideoCore`, `NRExoPlayerTracker`, `NRIMATracker`.
    </Callout>
  </Step>

  <Step>
    포함시키려는 모든 모듈에 대해 2단계와 3단계를 반복하십시오.
  </Step>

  <Step>
    앱 모듈의 `build.gradle` 파일에 다음 내용을 추가하세요.

    ```groovy
    dependencies {
      ...
      implementation project(":NewRelicVideoCore")
      implementation project(":NRExoPlayerTracker")
      implementation project(':NRIMATracker')
    }
    ```
  </Step>

  <Step>
    <DNT>**Continue**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    ### 동영상 플레이어 추적 설정

    사용 중인 구현 방식에 맞는 에이전트 유형을 선택하십시오.

    * **ExoPlayer 트래커가 포함된 비디오 에이전트(비디오 재생 추적 전용)**: 비디오 재생 추적용

    * **ExoPlayer 및 IMA 트래커를 사용하는 비디오 에이전트**: 비디오 재생 및 광고 추적용

      <img title="Install automatically using jitpack" alt="A gif depicting how to install the video agent using jitpack" src="/images/install-automatically-using-jitpack.gif" />
  </Step>

  <Step>
    <DNT>**See your data**</DNT> 클릭하여 업데이트된 코드를 구현, 배포하고 데이터를 확인하세요.
  </Step>
</Steps>
