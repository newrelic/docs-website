뉴웰릭 Roku 에이전트는 Roku의 시스템 및 비디오 이벤트를 모니터링하도록 설계되었습니다. 시스템, 영상, 오류, 광고, 커스텀 대시보드 등 다양한 유형의 이벤트를 추적할 수 있는 종합 API 제공합니다. 이 문서에서는 Roku 에이전트와 상호작용하는 데 사용할 수 있는 API 메서드를 설명합니다.
