## 예

### APP\_STARTED 이벤트를 전송합니다.

```brightscript
sub Main(aa as Object)
    ...
    nrAppStarted(m.nr, aa)
```

### SCENE\_LOADED 이벤트를 전송합니다.

```brightscript
nrSceneLoaded(m.nr, "MyVideoScene")
```

### 프로세스 시스템 로그 메시지

```brightscript
while (true)
    msg = wait(0, m.port)
    if nrProcessMessage(m.nr, msg) = false
        if type(msg) = "roPosterScreenEvent"
            if msg.isScreenClosed()
                exit while
            end if
        end if
    end if
end while
```
