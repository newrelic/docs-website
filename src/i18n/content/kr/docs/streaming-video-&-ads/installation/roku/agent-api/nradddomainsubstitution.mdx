## 통사론

```js
  nrAddDomainSubstitution(nr as object, pattern as String, subs as String) as Void
```

## 설명

* 도메인 속성에 맞는 패턴을 추가하고 다른 문자열로 대체합니다.
  * 도메인 속성을 사용하여 이벤트나 메트릭이 생성될 때마다 에이전트는 정규식과 일치하는지 확인하고 지정된 대체를 적용합니다.
* 패턴을 설정하지 않으면 URL 도메인을 변경하지 않고 사용합니다.
  * 이는 &quot;도메인&quot; 속성이 포함된 모든 이벤트와 메트릭에 적용됩니다.

## 예시

```js
  	nrAddDomainSubstitution(nr, "^.+\.my\.domain\.com$", "mydomain.com")
```
