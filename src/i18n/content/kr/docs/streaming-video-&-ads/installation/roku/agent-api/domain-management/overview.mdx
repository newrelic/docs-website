        도메인 속성에 대한 일치 패턴을 추가하고 다른 문자열로 대체합니다. 도메인 속성을 포함하는 이벤트 또는 메트릭이 생성될 때마다 에이전트는 해당 속성이 정규 표현식과 일치하는지 확인하고 지정된 대체를 적용합니다. 패턴이 설정되지 않은 경우 URL 도메인은 변경되지 않고 그대로 사용됩니다. &quot;도메인&quot; 속성을 포함하는 모든 이벤트 및 지표에 적용됩니다.
      </td>

## 예

### nrAddDomainSubstitution

```js
  	nrAddDomainSubstitution(nr, "^.+\.my\.domain\.com$", "mydomain.com")
```

### nrDelDomainSubstitution

```js
  	nrDelDomainSubstitution(nr, "^.+\.my\.domain\.com$")
```
