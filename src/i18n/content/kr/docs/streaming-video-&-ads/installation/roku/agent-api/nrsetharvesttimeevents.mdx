## 통사론

```js
  nrSetHarvestTimeEvents(nr as Object, time as Integer) as Void
```

## 설명

이벤트 수집 시간, 이벤트가 뉴렐릭으로 전송되기 전에 버퍼링되는 시간을 설정합니다. 최소값은 60입니다.
