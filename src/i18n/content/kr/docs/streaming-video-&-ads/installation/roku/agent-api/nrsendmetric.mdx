## 통사론

```js
  function nrSendMetric(nr as Object, name as String, value as dynamic, attr = invalid as Object) as Void
```

## 설명

게이지 메트릭을 기록합니다. 시간에 따라 증가하거나 감소할 수 있는 값을 나타냅니다.
