[서비스 수준에 패싯을 추가](/docs/service-level-management/facet-slm/configure)하고 나면, 해당 패싯별로 분류된 컴플라이언스 및 오류 예산 데이터를 확인하고 분석하여 성능 문제가 발생하는 위치를 식별할 수 있습니다.

## 패싯된 결과 보기 [#view-faceted-results]

개별 속성 값 및 해당 조합에 대한 컴플라이언스와 오류 예산을 확인하려면:

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Service Levels**</DNT> 으)로 이동합니다.
2. 서비스 수준을 선택하십시오.
3. <DNT>**View by group**</DNT> 드롭다운 메뉴와 <DNT>**Group by**</DNT> 패널을 사용하여 그룹화를 위해 구성한 속성을 분석하십시오.

## 예시 [#example]

`environment` 및 `region`(으)로 그룹화하면 <DNT>**View by group**</DNT> 드롭다운 메뉴와 <DNT>**Group by**</DNT> 패널은 각 개별 속성 값(모든 환경, 모든 리전)뿐만 아니라 환경 및 리전의 각 조합에 대한 성능 데이터를 표시합니다:

| 속성 값            | 규정 준수  | 남은 오류 예산 |
| --------------- | ------ | -------- |
| 프로덕션            | 99.05% | 48.5%    |
| 스테이징            | 99.50% | 60%      |
| us-east-1       | 99.70% | 72.5%    |
| us-west-1       | 98.20% | 12%      |
| 프로덕션, us-east-1 | 99.9%  | 85%      |
| 프로덕션, us-west-1 | 98.2%  | 12%      |
| 스테이징, us-east-1 | 99.5%  | 60%      |

## 데이터 분석 [#interactive-analysis]

패싯 뷰를 사용하여 다양한 세그먼트 전반의 컴플라이언스 데이터를 필터링하고 비교하십시오.

<img title="Faceted service level UI" alt="Screenshot showing the faceted service level analysis interface with group filters and compliance data" src="/images/slm_screenshot-full_faceted-view.webp" />

* **특정 세그먼트에 대한 데이터를 보려면**: 페이지 상단의 드롭다운 메뉴에서 패싯을 선택하십시오. 기본 차트가 해당 세그먼트의 데이터만 표시하도록 업데이트되며, 전체 컴플라이언스 대비 선택한 그룹의 성능을 보여주는 <DNT>**Group SLO**</DNT> 그래프가 나타납니다.
* **개별 속성과 조합을 비교하려면**: 두 개의 패싯을 선택한 경우, 개별 뷰를 사용하여 각 속성을 개별적으로 확인하거나 결합된 뷰를 사용하여 속성 조합을 확인하세요.
* **전체 보기로 돌아가려면**: 드롭다운 메뉴에서 <DNT>**None**</DNT> 을(를) 선택합니다.

## 패싯 알림 [#faceted-alerting]

특정 세그먼트가 실패할 때 알림을 받을 수 있도록 특정 패싯을 기반으로 [알림을 생성할 수 있습니다](/docs/service-level-management/alerts-slm/).

**지원되는 공지 유형:**

* SLI 준수 알림
* 빠른 소진 알림
* 슬로우 번 알림

**지원되지 않음:**

잔여 에러 버짓 알림 (집계 수준에서만 계속 사용할 수 있습니다)
