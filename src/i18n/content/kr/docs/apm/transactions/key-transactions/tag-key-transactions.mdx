APM에서는 [주요 트랜잭션](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events) 에 **태그** 를 할당할 수 있습니다.

태그는 트랜잭션과 관련된 메타데이터입니다. 공통점이 있는 트랜잭션에 적용하는 데 유용합니다. 예를 들어, 결제 프로세스의 일부인 모든 주요 트랜잭션에 `checkout` 태그를 적용할 수 있습니다. 그런 다음 `checkout` 로 필터링하여 이러한 모든 거래를 한 곳에서 볼 수 있습니다.

## 태그 할당 [#tagging_assign]

키 트랜잭션에 태그를 할당하려면:

1. **[one.newrelic.com](https://one.newrelic.com) > More > Key transactions** 로 이동합니다.

2. 색인 페이지에서 태그를 선택하십시오. <Icon name="fe-tag"/>태그를 지정하려는 거래의 이름 옆에 있는 아이콘.

3. 태그 이름 상자에 태그 이름을 입력합니다. 스페이스바를 눌러 태그를 적용하세요.

   <Callout variant="tip">
     쉼표 `,` , 꺾쇠 괄호 `< >` , 괄호 `( )` 또는 대괄호 `[ ]` 를 제외한 모든 유니코드 문자를 사용하십시오.
   </Callout>

4. 태그 생성을 종료하려면 태그 이름 상자 외부를 선택하십시오.

## 여러 거래에 태그 지정 [#tagging_multiple]

인덱스에서 볼 수 있는 모든 트랜잭션에 태그를 할당할 수 있습니다.

1. **[one.newrelic.com](https://one.newrelic.com) > More > Key transactions** 로 이동합니다.
2. 태그 [할당](#tagging_assign) 의 태그 생성 지침을 따릅니다.

일괄 태그를 제거하려면 다음 단계를 따르세요.

* 대량 태그 드롭다운에서 태그 이름의 **x** 기호를 선택합니다.

## 필터 [#tagging_filtering]

태그별로 주요 거래 색인을 필터링하려면:

1. 주요 거래 색인 검색 필드에서

   <Icon name="fe-chevron-down"/>

   드롭다운 아이콘을 클릭하고 **태그** 로 선택합니다.

2. 태그 이름을 입력합니다.
