APM에서 모든 [주요 트랜잭션](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events) 에는 자체 **개요** 페이지와 기타 여러 UI 페이지가 있습니다. [거래 유형(웹 또는 비웹)](/docs/apm/transactions/intro-transactions/transactions-new-relic-apm) 에 따라 다른 정보가 나타납니다.

## 개요 페이지 [#key_overview]

[키 트랜잭션 인덱스](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events#find) 에서 트랜잭션을 선택하면 키 트랜잭션의 **개요** 페이지가 나타납니다. **개요** 페이지에는 응답 시간, [Apdex](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) , 처리량 및 기타 데이터에 대한 차트가 포함되어 있습니다. [표준 New Relic 페이지 기능](/docs/data-analysis/user-interface-functions/view-your-data/standard-page-functions) 이 있는 APM의 [기본 개요 페이지](/docs/apm/applications-menu/monitoring/apm-overview-page) 와 유사합니다.

## SLA 보고서 페이지 [#key_sla]

주요 트랜잭션에 대한 **SLA 보고서** 페이지에는 해당 트랜잭션에 대한 [서비스 수준 계약](/docs/apm/reports/service-level-agreements/apm-sla-reports-dashboard) 보고서가 표시됩니다. SLA 보고서에는 처리량, 응답 시간 및 Apdex에 대한 일별 및 주별 데이터가 포함됩니다. 보고서를 CSV(쉼표로 구분된 값) 스프레드시트로 다운로드할 수도 있습니다.

## 경고 [#key_alerts]

[주요 트랜잭션 인덱스](/docs/apm/transactions/key-transactions/key-transactions-tracking-important-transactions-or-events#find) 에서 트랜잭션을 선택하면 [경고 조건 위반을 보고 경고 정책을 설정](/docs/apm/transactions/key-transactions/view-key-transactions-alert-information) 하는 옵션도 볼 수 있습니다.
