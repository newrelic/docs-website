[APM](/docs/apm/new-relic-apm/getting-started/welcome-new-relic-apm) 에서 트랜잭션 추적 **요약** 페이지는 특정 [트랜잭션 추적](/docs/apm/transactions/transaction-traces/transaction-traces) 의 데이터 요약을 표시합니다. 이 문서에서는 사용할 수 있는 주요 섹션과 기능에 대해 설명합니다.

## 트랜잭션 추적 요약 페이지 [#summary]

트랜잭션 추적 데이터를 보려면 **[one.newrelic.com > All capabilties](https://one.newrelic.com/all-capabilities) > APM & services > (앱 선택) > Monitor > Transactions > (트랜잭션 추적 선택)** 으로 이동합니다. 트랜잭션 추적의 **Summary** \[요약] 페이지에는 다음 네 가지 주요 영역이 있습니다.

다른 기본 트랜잭션 추적 UI 페이지는 [추적 세부 정보 페이지](/docs/apm/transactions/transaction-traces/transaction-trace-details) 입니다.

## APM 및 인프라 데이터에 대한 로그 보기 [#logs-context]

또한 로그와 애플리케이션의 데이터를 함께 가져와 더 쉽고 빠르게 문제를 해결할 수 있습니다. [컨텍스트의 로그를](/docs/logs/logs-context/configure-logs-context-apm-agents/) 사용하면 앱의 UI에서 직접 오류 및 추적과 관련된 로그 메시지를 볼 수 있습니다. Kubernetes 클러스터와 같은 [인프라 데이터](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent/) 컨텍스트에서 로그를 볼 수도 있습니다. 다른 UI 페이지로 전환할 필요가 없습니다.
