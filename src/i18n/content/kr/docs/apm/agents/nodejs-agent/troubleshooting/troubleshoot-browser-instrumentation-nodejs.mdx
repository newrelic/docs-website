## 문제

뉴렐릭 Node.js 에이전트를 통해 오류를 [처리하는](/docs/nodejs/page-load-timing-in-nodejs) 데 문제가 있는 경우 [표준 <InlinePopover type="browser"/>문제 처리 절차를](/docs/new-relic-browser/troubleshooting-page-view-monitoring) 따르세요. Node.js에 대한 몇 가지 추가 팁은 다음과 같습니다.

## 해결책

오류 코드는 사이트 소스 코드와 Node.js 에이전트 로그에 자동으로 나타납니다. 이 코드를 찾으려면 [`NREUM`](/docs/new-relic-browser/troubleshooting-page-view-monitoring#javascript) 을(를) 검색하십시오.
