New Relic으로 Node.js 애플리케이션 모니터링을 바로 시작하고 싶으십니까?이 가이드를 사용하여 애플리케이션 데이터를 New Relic으로 보내기 시작하는 가장 좋은 경로를 찾으십시오.

시작하기 전에 자세히 알아보시겠습니까?[APM](/docs/apm/new-relic-apm/getting-started/introduction-apm) 소개 및 [Node.js용 New Relic 소개](/docs/apm/agents/nodejs-agent/getting-started/introduction-new-relic-nodejs) 를 시작하는 것이 좋습니다.일반 설치 프로세스에 대해 자세히 알아보려면 [Node.js 설치 개요](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent) 를 참조하세요.

## 무엇을 모니터링하시겠습니까? [#what]

모니터링 대상에 따라 Node.js 에이전트를 설치할 수 있는 몇 가지 경로가 있습니다.

<CollapserGroup>
  <Collapser
    id="docker"
    title="Docker 컨테이너에 배포된 앱"
  >
    Docker 컨테이너에 에이전트를 설치하려면 명령줄을 사용하여 [다음 자체 안내 단계](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent-docker) 를 따르세요.

    이 프로세스는 에이전트를 `package.json` 에 추가한 다음 앱 수준에서 New Relic을 요구하는 과정을 안내합니다.마지막으로 에이전트가 Docker 앱과 함께 실행되도록 라이선스 키와 앱 이름을 `docker run` 에 추가합니다.
  </Collapser>

  <Collapser
    id="server"
    title="서버에 배포된 앱"
  >
    서버에서 호스팅되는 앱에는 몇 가지 다른 옵션이 있습니다.

    * [가이드 설치](https://one.newrelic.com/marketplace/install-data-source?state=f535d53b-5b84-cd48-80cf-61704ad02e29) 는 설치를 여러 단계로 나눕니다. 명령줄과 약간의 상호 작용이 필요합니다.
    * [Node.js 에이전트 설치](/docs/apm/agents/nodejs-agent/installation-configuration/install-nodejs-agent) 문서에서는 명령줄만 사용합니다.[Node.js 에이전트](https://github.com/newrelic/node-newrelic#installation) 리포지토리에서 동일한 프로세스를 볼 수 있습니다.
    * PM2 프로세스 관리자를 사용하는 앱의 경우 [안내 설치](https://one.newrelic.com/marketplace/install-data-source?state=ba2c3ecb-defb-e99a-c4be-47d786690d2a) 를 따르십시오. PM2가 아닌 프로세스 관리자를 사용하는 경우 가이드 또는 수동 설치 옵션을 선택합니다.

    앱을 모니터링할 준비가 되면 앱 코드의 첫 번째 줄에 `$ node -r newrelic your-program.js` 을 추가합니다.이는 앱에 대한 루트 액세스 권한이 없는 경우 특히 중요합니다.
  </Collapser>

  <Collapser
    id="lambda"
    title="AWS Lambda와 함께 배포된 앱"
  >
    AWS Lambda 함수용 에이전트를 설치하기 위해 [자체 안내 단계](https://one.newrelic.com/marketplace/install-data-source?state=9aa3d004-6f13-1477-604e-b6a2d6bb992d) 에서는 자동으로 AWS를 구성하는 설정 스크립트를 제공합니다.

    명령줄을 통한 수동 설치를 선호하는 경우 현재 [Lambda 모니터링을 위한 레거시 계측](https://docs.newrelic.com/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/enable-serverless-monitoring-aws-lambda-legacy/#node) 옵션이 있습니다.
  </Collapser>
</CollapserGroup>
