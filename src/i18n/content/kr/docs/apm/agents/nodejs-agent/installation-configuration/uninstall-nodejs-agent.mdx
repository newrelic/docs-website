New Relic Node.js 에이전트를 제거하는 방법을 배우려면 계속 읽으십시오. 에이전트를 일시적으로 비활성화하는 방법에 대한 지침은 에이전트 [비활성화](/docs/agents/manage-apm-agents/installation/disable-apm-agent#node) 를 참조하십시오.

New Relic Node.js 에이전트를 제거하려면:

1. 앱 기본 모듈의 첫 번째 줄에서 `require('newrelic');` 을 제거합니다.
2. `npm uninstall newrelic --save` 으로 npm 패키지를 제거합니다.
3. `newrelic.js` 파일을 삭제하고 사용 중인 모든 [환경 변수](/docs/agents/nodejs-agent/installation-configuration/nodejs-agent-configuration#methods-and-precedence) 를 설정 해제합니다.
4. 제거 프로세스가 완료되면 앱을 [다시 시작](#restart) 합니다.

설치에 대한 자세한 내용은 [Node.js 설치 및 유지 관리](/docs/agents/nodejs-agent/installation-configuration/installing-maintaining-nodejs) 를 참조하세요.

<Callout variant="tip">
  New Relic에서 [앱을 제거](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) 할 수도 있습니다.
</Callout>
