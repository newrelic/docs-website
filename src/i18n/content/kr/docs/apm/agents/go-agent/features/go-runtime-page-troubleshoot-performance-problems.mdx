[APM](/docs/apm) 사용자 인터페이스는 APM용 [Go 에이전트](/docs/agents/go-agent/get-started/new-relic-go) 의 특수 기능을 포함하여 앱에 대한 다양한 데이터를 제공합니다. 또한 **Go 런타임** 페이지는 성능 문제를 해결하는 데 유용한 중요한 런타임 데이터를 제공합니다.

## Go 런타임 데이터 보기 및 필터링 [#find]

**Go 런타임** 페이지를 보려면 **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (앱 선택) > Go runtime** 이동합니다.
