New Relic의 최신 기능, 개선 사항 및 중요한 보안 패치를 최대한 활용하려면 앱의 Go 에이전트를 최신 버전으로 업데이트하세요. 특정 에이전트 업데이트에 대한 추가 정보는 [Go 에이전트 릴리스 정보](/docs/release-notes/agent-release-notes/go-release-notes) 를 참조하십시오.

## Go 에이전트 버전 업데이트 [#update-go]

Go 에이전트를 업데이트하려면 표준 절차에 따라 터미널 명령이나 스크립트로 다음 프로세스를 실행하십시오.

1. [http://github.com/newrelic/go-agent](http://github.com/newrelic/go-agent) 에서 다음 프로세스를 사용합니다.

   ```bash
   go get -u github.com/newrelic/go-agent/v3/newrelic
   ```

2. 애플리케이션을 컴파일하고 배포합니다.
