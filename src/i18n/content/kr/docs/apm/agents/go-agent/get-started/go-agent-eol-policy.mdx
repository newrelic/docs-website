다음은 Go 에이전트 지원에 대한 구체적인 정책 및 날짜입니다. 뉴렐릭의 전반적인 수명 종료 정책에 대한 자세한 내용은 [일반 EOL 정책](/docs/licenses/end-of-life/notification-changes-new-relic-saas-features-distributed-software/) 에 대한 설명서를 참조하세요. Go 에이전트의 각 릴리스는 릴리스 당시 Go의 최신 버전 3개만 지원합니다. 우리는 Go 팀에서 적극적으로 지원하는 Go 릴리스 버전과 가장 최근에 지원 기간이 끝난 Go 버전만 지원합니다.
