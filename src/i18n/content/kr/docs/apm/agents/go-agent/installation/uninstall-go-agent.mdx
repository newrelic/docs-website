New Relic Go 에이전트를 제거하는 방법을 배우려면 계속 읽으십시오. Go 에이전트를 일시적으로 비활성화하는 방법에 대한 지침은 에이전트 [비활성화](/docs/agents/manage-apm-agents/installation/disable-apm-agent#go) 를 참조하십시오.

Go용 New Relic을 제거하려면:

1. 원하는 제거 방법을 사용하여 New Relic 패키지를 제거합니다.
2. 애플리케이션을 다시 컴파일하고 다시 시작합니다.

New Relic 패키지를 [제거하면 New Relic 계정에서 애플리케이션을 제거](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) 할 수 있습니다.
