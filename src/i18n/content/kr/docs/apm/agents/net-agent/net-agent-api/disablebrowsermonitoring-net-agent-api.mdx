## 통사론

```cs
NewRelic.Api.Agent.NewRelic.DisableBrowserMonitoring([boolean $override])
```

특정 페이지에서 브라우저 모니터링 스니펫의 자동 삽입을 비활성화합니다.

## 요구 사항

모든 에이전트 버전과 호환됩니다.

[트랜잭션](/docs/glossary/glossary/#transaction) 내에서 호출되어야 합니다.

## 설명

특정 페이지에서 [브라우저 모니터링](/docs/browser/new-relic-browser/getting-started/new-relic-browser) 스크립트의 **자동** 삽입을 비활성화하려면 이 호출을 추가하십시오. 수동 및 자동 주입을 **모두** 비활성화하기 위해 선택적 재정의를 추가할 수도 있습니다. 두 경우 모두 이 API 호출을 브라우저를 비활성화하려는 보기의 맨 위에 최대한 가깝게 두십시오.

<Callout variant="tip">
  페이지에 브라우저 스크립트를 **추가** 하는 [`GetBrowserTimingHeader()`](/docs/agents/net-agent/net-agent-api/getbrowsertimingheader-net-agent) 를 비교하십시오.
</Callout>

## 예

### 자동 주입 비활성화 [#automatic-only]

이 예에서는 스니펫의 **자동** 삽입만 비활성화합니다.

```cs
NewRelic.Api.Agent.NewRelic.DisableBrowserMonitoring();
```

### 자동 및 수동 주입 비활성화 [#disable-both]

이 예에서는 스니펫의 자동 및 수동 삽입을 **모두** 비활성화합니다.

```cs
NewRelic.Api.Agent.NewRelic.DisableBrowserMonitoring(true);
```
