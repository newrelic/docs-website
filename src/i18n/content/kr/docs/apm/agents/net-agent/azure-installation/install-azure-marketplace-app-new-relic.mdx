## 수명 종료 알림 [#eol]

2017년 10월부터 Microsoft의 Azure Marketplace 포털을 통해 New Relic 계정을 만들 수 없습니다. 대신 Azure ID가 있지만 아직 New Relic 계정이 없는 경우 [New Relic Sales에 문의](https://newrelic.com/about/contact-us) 하세요.

자세한 내용은 New Relic의 온라인 기술 커뮤니티에서 [Azure 전환 정보](https://discuss.newrelic.com/t/new-relic-apm-for-azure-marketplace-eol-notice/52358) 를 참조하세요.

## New Relic .NET 에이전트 설치 [#app-services]

다음에 해당하는 경우 Azure 앱과 함께 New Relic .NET 에이전트를 설치하는 절차를 계속 진행합니다.

* Azure Portal의 Microsoft [Web App Services](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps)
* 마이크로소프트의 [클라우드 서비스](/docs/agents/net-agent/azure-installation/install-app-azure-cloud-services) 플랫폼

Microsoft Azure에 대한 .NET 에이전트의 지원은 [인프라 모니터링의 Azure 통합과](/docs/infrastructure/microsoft-azure-integrations/getting-started/introduction-azure-monitoring-integrations) 동일하지 않습니다. Azure 통합에는 별도의 [통합 설치 절차](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations) 가 있습니다.
