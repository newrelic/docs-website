## 문제

FTP를 통해 [Microsoft Azure Web App](/docs/agents/net-agent/azure-installation/install-app-azure-web-apps) 에 연결하고 있습니다. **LogFiles\\NewRelic** 에서 `NewRelic.Profiler.####.log` 파일을 볼 수 있지만 로그 파일에 액세스하려고 하면 오류 메시지가 나타납니다.

## 해결책

웹 앱을 다시 시작하여 파일을 해제하고 새 프로필 로그를 만듭니다.
