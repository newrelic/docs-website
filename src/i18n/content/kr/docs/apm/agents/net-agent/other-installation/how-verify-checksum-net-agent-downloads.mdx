New Relic의 .NET 에이전트 설치 프로그램에 대한 체크섬을 확인하는 방법에 대해 읽어보십시오. 현재 릴리스의 해시가 필요한 경우 [다운로드 사이트에서 SHA256 해시를 다운로드하십시오](https://download.newrelic.com/dot_net_agent/latest_release/SHA256/).

## Linux [#checksum-linux]

Linux에서는 다음을 실행합니다.

```sh
sha256sum YOUR_FILENAME
```

성공하면 다음과 같은 응답을 받게 됩니다.

```
5092fd52e40132a41ac06c320296cd9d63196d6de0fa6ce9b74fc4de1b3f9502 YOUR_FILENAME
```

## 윈도우 [#checksum-windows]

Windows에서는 파일 컴퓨트를 위해 `CertUtil` 또는 `Get-FileHash` 중 하나를 사용하는 것이 좋습니다.

<CollapserGroup>
  <Collapser
    id="certutil"
    title="CertUtil(명령 프롬프트 또는 Powershell)"
  >
    `CertUtil` 인증서 서비스의 일부로 제공되며 파일의 컴퓨트에 사용될 수 있습니다. 명령 프롬프트나 Powershell에서 실행할 수 있습니다.

    해시를 계산하려면 다음을 실행하십시오.

    ```sh
    CertUtil -hashfile YOUR_FILENAME SHA256
    ```

    성공하면 다음과 같은 응답을 받게 됩니다.

    ```
    SHA256 hash of YOUR_FILENAME:
    5092fd52e40132a41ac06c320296cd9d63196d6de0fa6ce9b74fc4de1b3f9502
    CertUtil: -hashfile command completed successfully.
    ```
  </Collapser>

  <Collapser
    id="getfilehash"
    title="Get-FileHash(Powershell)"
  >
    `Get-FileHash` cmdlet은 파일의 컴퓨트에 사용될 수 있습니다. Powershell 버전 5.1 이상에서 사용할 수 있습니다.

    해시를 계산하려면 다음을 실행하십시오.

    ```powershell
    Get-FileHash YOUR_FILENAME Algorithm SHA256 | Format-List
    ```

    성공하면 다음과 같은 응답을 받게 됩니다.

    ```
    Algorithm : SHA256
    Hash      : 5092FD52E40132A41AC06C320296CD9D63196D6DE0FA6CE9B74FC4DE1B3F9502
    Path      : C:\Path\To\YOUR_FILENAME
    ```
  </Collapser>
</CollapserGroup>
