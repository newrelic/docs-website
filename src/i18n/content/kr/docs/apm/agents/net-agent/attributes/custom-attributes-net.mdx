[속성은](/docs/features/agent-attributes) 이벤트나 트랜잭션의 속성을 결정하는 정보가 포함된 키-값 쌍입니다. [`AddCustomAttribute` API를](/docs/agents/net-agent/net-agent-api/itransaction/#addcustomattribute) 사용하여 맞춤 속성을 만들 수 있습니다.

트랜잭션, 사용자 정의 이벤트, 범위 및 오류에 사용자 정의 속성 값을 추가할 때 API는 `object` 을 허용합니다. 이것은 이러한 값이 처리되는 방법과 APM에 표시되는 방법을 설명합니다. 모든 경우에 `NULL` 값은 기록되지 않습니다.
