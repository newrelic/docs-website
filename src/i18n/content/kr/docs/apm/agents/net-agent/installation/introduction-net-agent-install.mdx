.NET 에이전트는 코드를 자동으로 계측하므로 애플리케이션 모니터링을 시작할 수 있습니다. 런처를 사용하거나 이 문서의 지침에 따라 기본 .NET 에이전트 설치를 완료할 수 있습니다. APM for .NET이 하는 일에 대해 자세히 알아보려면 .NET [용 New Relic 소개를](/docs/agents/net-agent/getting-started/introduction-new-relic-net) 참조하세요.

## .NET 에이전트 설치 [#install-instructions]

운영 체제에 대한 적절한 설치 지침을 참조하고 다운로드 링크를 찾으십시오.

* [윈도우](/docs/agents/net-agent/installation/install-net-agent-windows)

  * IIS의 Windows .NET 애플리케이션 또는 Linux의 .NET Core 애플리케이션의 경우 [설치 안내](https://one.newrelic.com/launcher/nr1-core.home?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5ob21lLXNjcmVlbiJ9&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyIsInBhdGgiOiJhcG0iLCJsYW5ndWFnZSI6ICJkb3RuZXQiLCAicmVjaXBlTmFtZSI6ICJkb3RuZXQtYWdlbnQtaW5zdGFsbGVyIiwgImluaXRpYWxBY3Rpb25JbmRleCI6MiwiYWNjb3VudElkIjpudWxsfQ==) 를 사용하여 APM을 시작하십시오.
  * EU 서버에 있는 경우 대신 [이 링크](https://one.eu.newrelic.com/launcher/nr1-core.home?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5ob21lLXNjcmVlbiJ9&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyIsInBhdGgiOiJhcG0iLCJsYW5ndWFnZSI6ICJkb3RuZXQiLCAicmVjaXBlTmFtZSI6ICJkb3RuZXQtYWdlbnQtaW5zdGFsbGVyIiwgImluaXRpYWxBY3Rpb25JbmRleCI6MiwiYWNjb3VudElkIjpudWxsfQ==) 를 사용하십시오.

* [Linux](/docs/agents/net-agent/installation/install-net-agent-linux)

[최신 릴리스](http://download.newrelic.com/dot_net_agent/latest_release) 에서 .NET 에이전트에 대한 모든 바이너리와 패키지를 찾을 수 있습니다. 이전 릴리스에서 [이전 릴리스](http://download.newrelic.com/dot_net_agent/previous_releases/) 를 볼 수도 있습니다.
