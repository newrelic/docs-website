이 문서는 New Relic .NET 에이전트를 제거하는 방법을 설명합니다. 에이전트를 일시적으로 비활성화하는 방법에 대한 지침은 에이전트 [비활성화](/docs/agents/manage-apm-agents/installation/disable-apm-agent#net) 를 참조하십시오.

.NET 에이전트를 제거하려면 운영 체제에 해당하는 절차를 따르십시오.

* [윈도우](#uninstall-framework-windows)
* [Linux](#uninstall-linux)

## .NET 에이전트 제거(Windows) [#uninstall-framework-windows]

<Callout variant="tip">
  Windows 관리자 그룹의 [**관리자** 권한](/docs/agents/net-agent/getting-started/compatibility-requirements-net-framework-agent#user-rights) 이 필요합니다.
</Callout>

이 절차에서는 MSI 설치 프로그램을 사용하여 .NET 에이전트를 설치했다고 가정합니다.

1. 실행 중인 에이전트를 종료하려면 IIS를 중지하십시오. IIS를 중지하지 않으면 에이전트를 제거한 후 재부팅해야 할 수 있습니다.
2. **Start > All Programs > New Relic .NET Agent > Uninstall the .NET agent** 선택합니다.
3. 메시지에 따라 제거를 완료합니다.
4. IIS를 다시 시작하십시오.

에이전트를 제거한 후 모든 에이전트 관련 파일을 제거할 수 있습니다. 다른 New Relic 파일을 제거하려면 다른 경로를 확인하십시오. 예를 들어 .NET 에이전트를 제거해도 **[newrelic.config](/docs/agents/net-agent/installation-configuration/net-agent-configuration#config-options-precedence)** 파일은 삭제되지 않습니다.

1. **Control Panel > Programs > Programs and Features** 에서 `New Relic .NET Agent` 이라는 파일을 마우스 오른쪽 버튼으로 클릭하고 삭제합니다.
2. `C:\ProgramData\New Relic\.NET Agent` 에서 전체 폴더를 삭제합니다.
3. `C:\Program Files\New Relic\.NET Agent` 에서 전체 폴더를 삭제합니다.
4. `C:\Program Files (x86)\New Relic\.NET Agent` 에서 전체 폴더를 삭제합니다.
5. APM UI에서 [앱을 제거합니다](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) .

## .NET 에이전트 제거(Linux) [#uninstall-linux]

Linux에서 .NET 에이전트를 제거하려면 패키지 관리 도구를 사용하십시오.

<Callout variant="important">
  .NET 에이전트 버전 10.0.0 이상의 경우 패키지 이름은 `newrelic-dotnet-agent` 입니다..NET 에이전트 버전 9.9.0 이하의 경우 패키지 이름은 `newrelic-netcore20-agent` 입니다.이 문서의 나머지 부분은 `newrelic-dotnet-agent` 를 참조합니다. 이전 에이전트 버전이 설치된 경우 이전 이름으로 바꾸십시오.
</Callout>

<Callout variant="important">
  제거가 완료되면 애플리케이션을 다시 시작해야 합니다.
</Callout>

<CollapserGroup>
  <Collapser
    id="apt-uninstall"
    title="apt로 제거(Debian, Ubuntu 및 Linux Mint)"
  >
    루트로(또는 sudo를 사용하여) 다음 명령을 실행합니다.

    ```
    apt-get remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="dpkg-uninstall"
    title="dpkg로 제거(Debian, Ubuntu 및 Linux Mint)"
  >
    루트로(또는 sudo를 사용하여) 다음 명령을 실행합니다.

    ```
    dpkg --remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="yum-uninstall"
    title="yum으로 제거(CentOS, Red Hat Enterprise Linux, Oracle Linux)"
  >
    루트로(또는 sudo를 사용하여) 다음 명령을 실행합니다.

    ```
    yum remove newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="rpm-uninstall"
    title="rpm으로 제거(CentOS, Red Hat Enterprise Linux, Oracle Linux)"
  >
    루트로(또는 sudo를 사용하여) 다음 명령을 실행합니다.

    ```
    sudo rpm -e newrelic-dotnet-agent
    ```
  </Collapser>

  <Collapser
    id="tarball"
    title="타르볼 제거"
  >
    1. 앱 폴더에서 모든 New Relic 파일을 삭제합니다.

    2. New Relic 환경 변수 설정 해제:

       ```
       CORECLR_ENABLE_PROFILING=1
       CORECLR_PROFILER={36032161-FFC0-4B61-B559-F6C5D41BAE5A}
       CORECLR_NEWRELIC_HOME=PATH_TO_INSTALL
       CORECLR_PROFILER_PATH=%CORECLR_NEWRELIC_HOME%\NewRelic.Profiler.dll
       ```
  </Collapser>
</CollapserGroup>
