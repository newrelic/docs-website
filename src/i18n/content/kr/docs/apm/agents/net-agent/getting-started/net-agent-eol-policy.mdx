* [마이크로소프트 닷넷 프레임워크](https://docs.microsoft.com/en-us/lifecycle/products/microsoft-net-framework)
* [마이크로소프트 .NET 및 .NET 코어](https://docs.microsoft.com/en-us/lifecycle/products/microsoft-net-and-net-core)
