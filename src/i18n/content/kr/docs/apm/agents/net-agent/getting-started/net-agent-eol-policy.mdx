다음은 .NET 에이전트 지원에 대한 특정 정책 및 날짜입니다. New Relic의 전반적인 단종 정책에 대한 정보는 [일반 EOL 정책](/docs/licenses/end-of-life/notification-changes-new-relic-saas-features-distributed-software/) 에 대한 문서를 참조하십시오.

* 6.x 에이전트 라인은 4.5 이전의 .NET Framework 버전에서 실행되는 애플리케이션을 사용하는 고객을 지원하기 위해 유지되며 중요한 보안 패치만 받습니다. 최신 6.x 에이전트 버전만 지원됩니다. 6.x 에이전트의 EOL 날짜는 .NET Framework 3.5 서비스 팩 1에 대한 Microsoft의 지원 종료 날짜와 동일합니다.
