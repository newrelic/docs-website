Azure App Service는 개발자가 다양한 언어 프로그래밍 및 프로그래밍 전반에 걸쳐 웹 구성, 모바일 백앤드 및 빠르고 효율적으로 구축, 구현, 배포 및 확장할 수 있도록 하는 의 완전 관리형 Platform as a Service(PaaS) 제품입니다. 프레임워크.Microsoft Azure API
