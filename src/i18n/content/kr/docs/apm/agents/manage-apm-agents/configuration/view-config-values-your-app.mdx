## 문제

애플리케이션의 구성 설정에 대한 정보를 쉽게 얻을 수 있는 방법이 필요합니다.

## 해결책

구성 파일 자체를 보고 싶지 않다면 APM UI에서 구성 설정을 **[보고 one.newrelic.com](https://one.newrelic.com)** 으로 이동한 다음 **APM** 을 클릭하면 됩니다. 앱을 클릭한 다음 **Settings > Environment > Agent initialization** 를 클릭합니다.
