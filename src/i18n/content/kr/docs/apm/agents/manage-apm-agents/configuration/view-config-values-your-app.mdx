## 문제

애플리케이션의 구성 설정에 대한 정보를 쉽게 얻을 수 있는 방법이 필요합니다.

## 해결책

구성 파일 자체를 보고 싶지 않은 경우 APM UI에서 구성 설정을 볼 수 있습니다. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM** 으로 이동합니다. 앱을 클릭한 다음 **Settings > Environment > Agent initialization** 클릭하십시오.
