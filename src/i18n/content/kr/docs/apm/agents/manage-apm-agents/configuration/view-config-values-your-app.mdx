## 문제

애플리케이션의 구성 설정에 대한 정보를 쉽게 얻을 수 있는 방법이 필요합니다.

## 해결책

설정 파일 자체를 보지 않으려면 APM UI 에서 설정 설정을 확인하고 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM**</DNT>으로 이동하세요. 앱을 클릭한 다음 <DNT>**Settings > Environment > Agent initialization**</DNT>) 클릭하세요.
