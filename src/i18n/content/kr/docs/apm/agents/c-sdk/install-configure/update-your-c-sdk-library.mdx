## C SDK 코드 라이브러리 업데이트 [#update-production]

애플리케이션의 코드 라이브러리를 New Relic의 C SDK 최신 버전으로 업데이트하려면:

1. C SDK 라이브러리 변경 사항을 검토하여 코드가 호환되는지 확인하십시오.
2. 해당하는 경우 앱의 코드를 수동으로 업데이트합니다.
3. [`make`](https://github.com/newrelic/c-sdk#building) 명령을 사용하여 C SDK 코드를 컴파일하여 `libnewrelic.a` 의 최신 버전을 만들고 이를 애플리케이션 또는 서비스에 연결합니다.
4. 해당하는 경우 테스트 환경에 애플리케이션을 다시 배포합니다.
5. 애플리케이션에 대한 트래픽을 생성한 다음 `c_sdk.log` 파일과 `newrelic-daemon.log` 출력에 오류가 있는지 확인하십시오.
6. 프로덕션 환경에 애플리케이션을 재배포하십시오.
7. 애플리케이션이 New Relic에 데이터를 보낼 때까지 몇 분 정도 기다립니다. 그런 다음 [New Relic](/docs/apm/applications-menu/monitoring/apm-overview-page) 에서 애플리케이션의 성능을 확인하십시오.

몇 분 이내에 데이터가 나타나지 않으면 [문제 해결 팁](/docs/apm/agents/c-sdk/troubleshooting/no-data-appears-c-sdk) 을 따르십시오.
