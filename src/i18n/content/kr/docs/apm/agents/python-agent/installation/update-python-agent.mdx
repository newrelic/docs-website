최신 기능, 개선 사항 및 중요한 보안 패치를 최대한 활용하려면 Python 에이전트를 최신 버전으로 업데이트하는 것이 좋습니다. 특정 에이전트 업데이트에 대한 추가 정보는 [Python 에이전트 릴리스 정보](/docs/release-notes/agent-release-notes/python-release-notes) 를 참조하십시오.

## Python 패키지 인덱스(PyPi)로 업데이트 [#package-index]

Python 에이전트 패키지는 PyPi( [http://pypi.python.org/pypi/newrelic](http://pypi.python.org/pypi/newrelic "링크가 새 창에서 열립니다.") )에 나열되어 있습니다. 특정 패키지로 업데이트하려면 해당 지침을 따르십시오.

## 수동으로 패키지 다운로드 [#manual-download]

New Relic Python 패키지를 수동으로 다운로드하려면:

1. 현재 New Relic 패키지를 제거합니다.
2. 새 패키지를 다운로드한 후 [Python 에이전트 설치](/docs/agents/python-agent/installation-configuration/python-agent-installation#download-site) 절차를 따릅니다.

<Callout variant="tip">
  에이전트 소프트웨어를 업데이트할 때 `pip` 이(가) 이전 버전을 올바르게 제거하므로 수동 설치를 사용하는 것보다 `pip` 를 사용하는 것이 좋습니다.
</Callout>

## 지원되지 않는 Python 에이전트 버전 [#eol-versions]

**수명 종료 알림:** 2015년 1월 26일부터 Python 에이전트 버전 **2.42.0** 이전의 데이터를 더 이상 허용하지 않습니다.

이러한 에이전트 버전은 데이터 수집 서비스와 통신할 때 오래된 프로토콜을 사용합니다. 또한 이러한 버전 중 다수는 민감한 데이터를 데이터 수집기에 잘못 보낼 수 있는 잠재적인 보안 문제를 포함합니다.
