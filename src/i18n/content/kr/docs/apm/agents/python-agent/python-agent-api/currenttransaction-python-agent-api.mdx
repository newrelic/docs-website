## 통사론

```py
newrelic.agent.current_transaction(active_only=True)
```

현재 트랜잭션에 해당하는 개체를 반환합니다.

## 설명

`current_transaction` 을 사용하여 현재 [트랜잭션](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction) 을 나타내는 객체를 검색합니다. 활성 트랜잭션이 없으면 `None` 을 반환합니다.

## 반환 값

현재 트랜잭션에 해당하는 트랜잭션 개체를 반환합니다. 활성 트랜잭션이 없으면 `None` 을 반환합니다.

## 예

### 현재 트랜잭션 가져오기 [#function-trace-example]

```py
import newrelic.agent

@newrelic.agent.background_task()
def main():
    transaction = newrelic.agent.current_transaction()
```
