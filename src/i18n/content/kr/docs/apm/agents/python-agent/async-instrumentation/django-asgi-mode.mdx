[Django를 ASGI와 함께 배포하는 앱과 Python 에이전트를 통합하기 위한 요구 사항 및 팁에 대해 읽어보세요.](https://docs.djangoproject.com/en/3.1/howto/deployment/asgi/) 일반 설치 지침으로 돌아가려면 [Python 에이전트 설치](/docs/agents/python-agent/installation-configuration/python-agent-installation) 로 이동하십시오.

## 관리 스크립트로 자동 초기화 [#wrapper-script]

`python app.py` 으로 앱을 시작하고 Python 에이전트 **버전 6.0.0.154** 이상을 사용하는 경우 권장되는 [관리 스크립트 통합 방법](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) 을 사용할 수 있습니다.

다음은 관리 스크립트 방법을 사용하는 예입니다.

```
NEW_RELIC_CONFIG_FILE=path/to/newrelic.ini newrelic-admin run-program [web server] path_to_app
```

## 이벤트 루프 진단 지원 [#event-loops]

Python 에이전트는 `asyncio` 이벤트 루프 진단을 지원합니다. 자세한 내용은 [Python 이벤트 루프 진단](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics) 을 참조하세요.
