## 통사론

```py
newrelic.agent.database_trace(sql, dbapi2_module=None, async_wrapper=None)
```

데이터베이스 호출을 계측하는 데 사용됩니다.

## 설명

`database_trace` 추가 세그먼트 형태로 [트랜잭션 트레이스](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#transaction-trace) 에 더 많은 세부 정보를 추가하는 데 사용됩니다. `database_trace` (으)로 보고된 모든 호출은 [APM 데이터베이스 페이지](/docs/apm/applications-menu/monitoring/databases-slow-queries-page)에 표시됩니다. `database_trace` 은(는) 데이터베이스 호출 시간을 측정하는 함수의 데코레이터로 사용할 수 있는 `DatabaseTraceWrapper` 의 [partial](https://docs.python.org/3.12/library/functools.html#functools.partial) 을 반환합니다.

`database_trace` 데코레이터는 에이전트 버전 2.102.0.85 이상에서 제너레이터 및 코루틴에 사용할 수 있습니다.

애플리케이션에서 데코레이터를 사용할 수 없는 경우 다음 다른 호출 형식 중 하나를 사용할 수 있습니다.

* <DNT>**The context manager**</DNT>: 컨텍스트 관리자 형식은 `DatabaseTrace` 입니다.
* <DNT>**The wrapper**</DNT>: 래퍼 형식은 `DatabaseTraceWrapper` 입니다. 데코레이터를 사용하지 않고 래핑된 함수를 반환하는 데 사용할 수 있습니다.
* <DNT>**The path-based wrapper**</DNT>: 경로 기반 래퍼 형식은 `wrap_database_trace` 입니다. 이는 Monkey 패치를 통해 지정된 객체에 `DatabaseTraceWrapper` 적용합니다. 이는 데코레이터와 동일한 모범 사례와 추가 `module` 및 `object_path` 모범 사례를 사용합니다.

이러한 다른 호출 형식의 사용에 대한 설명은 [다른](/docs/python-agent-api-different-call-forms) 호출 형식 을 참조하십시오. 호출 예는 [예](#examples) 를 참조하십시오.

## 반환 값

## 예

### `database_trace` [#trace\_example][#trace_example]

`database_trace` 데코레이터 사용 예:

```py
import newrelic.agent

@newrelic.agent.database_trace(sql="SELECT * FROM users WHERE id = 1")
def get_user(id):
    ...
```

네이티브 코루틴과 함께 `database_trace` 데코레이터를 사용하는 예:

```py
import newrelic.agent

@newrelic.agent.database_trace(sql="SELECT * FROM users WHERE id = 1")
async def get_user(id):
    ...
```

### 컨텍스트 관리자 예제 [#context-mgr-example]

`DatabaseTrace` 컨텍스트 관리자를 사용하는 예: \`custom\_action\`을 수행하는 데 걸리는 시간을 알려줍니다.

```py
import newrelic.agent


def complex_query(a, b, c):
    with Connection(host, port, db) as conn:
        with newrelic.agent.DatabaseTrace(
            sql="SELECT * FROM users WHERE id = 1",
        ):

            conn.custom_action()
```

### 래퍼 예 [#trace-wrap-example]

`DatabaseTraceWrapper` 사용 예:

```py
import newrelic.agent

def get(id):
    ...


get = newrelic.agent.DatabaseTraceWrapper(
    get, sql="SELECT * FROM users WHERE id = 1"
)
```
