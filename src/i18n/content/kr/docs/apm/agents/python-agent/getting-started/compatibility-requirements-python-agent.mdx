## 기본 요구 사항 [#basic]

Python 에이전트는 가장 일반적인 많은 Python 웹 프레임워크를 지원합니다. 특정 호스팅 메커니즘, 호스팅 서비스, 웹 프레임워크 또는 백엔드 서비스에 따라 추가 구성이 필요할 수 있습니다. 지원 세부 정보는 아래 표를 참조하세요.

아직 계정이 없다면 [New Relic 계정을 만드](https://newrelic.com/signup) 십시오. 영원히 무료입니다.

## 파이썬 패키지 계측 [#python-packages]

에이전트는 또한 다양한 기타 Python [패키지 및 모듈](/docs/agents/python-agent/getting-started/instrumented-python-packages) 에 대한 도구를 제공합니다. 계측에는 데이터베이스 쿼리, Memcache 요청, 외부 서비스 요청, [인스턴스 수준 세부 정보](/docs/apm/applications-menu/features/analyze-database-instance-level-performance-issues) 등이 포함됩니다. 결과는 웹 트랜잭션 성능 분석 및 느린 트랜잭션에 대한 트랜잭션 추적 내에서 APM의 사용자 인터페이스에 자동으로 나타납니다.

## 계측 패키지 버전 지원 [#package-version]

Python 에이전트는 3년 릴리스 기간 내에 계측 패키지를 적극적으로 지원하고 테스트합니다.

## 에이전트 버전 지원 [#agent-version]

New Relic은 에이전트를 정기적으로 최소 3개월마다 업그레이드할 것을 권장합니다. 에이전트 버전은 1년 릴리스 창에서 지원됩니다.

## 파이썬 버전 지원 [#python-version]

일반적으로 에이전트는 출시되고 활성화된 모든 [Python 분기를](https://devguide.python.org/versions/#versions) 지원합니다. 그러나 향후 변경 사항을 따라잡기 위해 에이전트는 이 Python 버전 지원 일정도 따릅니다. 버전 지원 정책 [은 일반적인 EOL(수명 종료) 정책을](/docs/agents/manage-apm-agents/maintenance/new-relic-agent-plugin-end-life-policy) 대체하지 않습니다.
