## 통사론 [#syntax]

```py
newrelic.agent.get_browser_timing_footer()
```

<Callout variant="caution">
  이 API는 버전 9.4.0에서 더 이상 사용되지 않으며 버전 11.0.0에서 완전히 제거되었습니다. 끌어 당기는 에이전트 버전 9.4.0부터 `get_browser_timing_footer()` 의 이전 기능이 [`get_browser_timing_header()`](/docs/apm/agents/python-agent/python-agent-api/getbrowsertimingheader-python-agent-api) 에 통합되었습니다.

  적절한 사용법에 대한 자세한 내용은 [수동으로 부하하다 브라우저 모니터링을](/docs/agents/python-agent/supported-features/browser-monitoring-python-agent#manual_instrumentation) 참조하세요.
</Callout>

JavaScript 에이전트를 삽입하기 위한 <InlinePopover type="browser" />에이전트 바닥글을 생성합니다.

## 설명 [#description]

이 호출은 [헤더 JavaScript 조각](/docs/agents/python-agent/python-agent-api/get_browser_timing_header) 과 함께 브라우저 모니터링을 활성화하는 데 사용되는 브라우저 에이전트 JavaScript 바닥글 조각을 반환합니다. `get_browser_timing_header` 와 함께 이 호출을 사용하여 브라우저 모니터링을 활성화하는 방법을 알아보려면 [브라우저 모니터링 수동 계측을](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) 참조하세요.

이 호출은 JavaScript 스니펫을 삽입하려는 모든 웹 요청에 대해 호출되어야 합니다. <DNT>**cannot**</DNT> 전역 범위에서 호출되고 캐시됩니다.

## 반환 값 [#return-values]

브라우저 에이전트 바닥글 JavaScript를 반환합니다.
