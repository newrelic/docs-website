## 통사론

```py
newrelic.agent.disable_browser_autorum(flag=True)
```

웹 요청에 대한 브라우저 모니터링을 비활성화합니다.

## 설명

이 호출은 호출된 페이지/요청에 대한 [브라우저 모니터링](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) 을 비활성화합니다. 이 호출은 현재 웹 요청에만 영향을 미치며 브라우저 에이전트 머리글/바닥글 JavaScript가 삽입되기 전에 호출되어야 합니다. 브라우저 에이전트 JavaScript를 삽입하는 방법에 대한 자세한 내용은 [브라우저 모니터링 설치](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser) 를 참조하세요.

<Callout variant="tip">
  WSGI 서버가 모니터링 중인 WSGI 애플리케이션으로 전달한 WSGI 환경 사전의 특정 요청에 대해 `newrelic.disable_browser_autorum` 키를 설정하여 브라우저 삽입을 비활성화할 수도 있습니다.
</Callout>

## 예

### 브라우저 모니터링 비활성화 [#example]

이 예에서 에이전트를 업그레이드했으며 HTML 페이지에 자동으로 삽입되는 N JavaScript가 해당 페이지에 로드되는 모든 iFrame에도 삽입되는 것을 확인했습니다. 이로 인해 페이지뷰가 크게 증가했습니다.

특정 iFrame 보기에서 다음을 호출합니다.

```py
newrelic.agent.disable_browser_autorum()
```
