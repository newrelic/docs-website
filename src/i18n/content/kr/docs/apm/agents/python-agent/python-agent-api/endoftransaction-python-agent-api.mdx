## 통사론

```py
newrelic.agent.end_of_transaction()
```

현재 트랜잭션을 종료합니다.

## 설명

이 호출은 현재 트랜잭션을 즉시 종료하고 현재 시간에 응답 타이머를 중지합니다. 트랜잭션이 종료된 후 에이전트는 추가 성능 분석 메트릭을 수집하지 않습니다. 그러나 에이전트는 여전히 Python 예외 세부 정보를 수집합니다.

이 호출은 웹 트랜잭션에 대해 다시 스트리밍되는 큰 응답이 있고 트랜잭션의 초기 부분에만 관심이 있는 경우에 유용합니다. (이는 또한 어떤 이유로든 트랜잭션을 [백그라운드 작업](/docs/agents/python-agent/python-agent-api/set_background_task) 으로 플래그 지정하고 싶지 않다고 가정합니다.)

## 매개변수

없음.

없음.
