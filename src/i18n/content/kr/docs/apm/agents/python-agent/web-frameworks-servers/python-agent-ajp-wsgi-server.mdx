Python 에이전트는 [flup](https://pypi.python.org/pypi/flup) 과 함께 AJP와 함께 사용할 수 있습니다. Python 에이전트 설치 지침은 [Python](/docs/agents/python-agent/installation-configuration/python-agent-installation) 에이전트 설치를 참조하십시오. AJP와 함께 에이전트를 사용하는 것에 대한 특별한 요구 사항이나 참고 사항은 없습니다.
