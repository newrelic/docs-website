Web2py는 웹 기반 Python 애플리케이션을 만들기 위한 오픈 소스 프레임워크입니다. Python 에이전트를 Web2py 앱과 통합하여 앱 성능을 모니터링할 수 있습니다. 일반적인 설치 절차 [는 Python 에이전트 설치](/docs/agents/python-agent/installation-configuration/python-agent-installation) 를 참조하십시오.

## Web2py와 통합 [#web2py-integration]

자세한 내용을 보려면 아래 목록에서 사용하는 Web2py 설정을 선택하십시오.

<CollapserGroup>
  <Collapser
    id="running-with-runweb2py"
    title="web2py 명령 실행"
  >
    **runweb2py** 명령을 사용하여 웹 애플리케이션을 실행하는 경우 다음 명령을 사용하십시오.

    ```
    NEW_RELIC_CONFIG_FILE=<var>/some/path/</var>newrelic.ini newrelic-admin run-program runweb2py
    ```

    `newrelic-admin` 에이전트가 애플리케이션의 주요 기능을 모니터링할 수 있도록 애플리케이션 시작을 래핑하는 스크립트입니다. 래퍼 스크립트 실행에 대한 자세한 내용은 래퍼 스크립트 [실행](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration) 을 참조하세요.
  </Collapser>

  <Collapser
    id="running-with-mod_wsgi"
    title="아파치/mod_wsgi"
  >
    Apache/mod_wsgi를 사용하는 경우 Python 에이전트를 초기화하기 위해 사용 중인 WSGI 스크립트 파일에 수동으로 코드를 삽입 **해야 합니다** . 자세한 정보는 [mod_wsgi 문서](/docs/python/python-agent-and-mod_wsgi) 를 참조하십시오.
  </Collapser>

  <Collapser
    id="running-with-gunicorn"
    title="구니콘"
  >
    에이전트와 함께 Gunicorn을 사용하는 방법에 대한 정보는 [Python 에이전트 및 Gunicorn 설명서](/docs/agents/python-agent/hosting-mechanisms/python-agent-and-gunicorn) 를 참조하십시오.
  </Collapser>

  <Collapser
    id="running-with-uwsgi"
    title="uWSGI"
  >
    에이전트와 함께 uWSGI를 사용하는 방법에 대한 정보는 [Python 에이전트 및 uWSGI 문서](/docs/agents/python-agent/hosting-mechanisms/python-agent-and-uwsgi) 를 참조하십시오.
  </Collapser>
</CollapserGroup>
