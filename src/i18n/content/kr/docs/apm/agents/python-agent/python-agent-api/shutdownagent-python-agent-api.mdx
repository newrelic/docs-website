## 통사론

```py
newrelic.agent.shutdown_agent(timeout=None)
```

에이전트를 강제로 종료합니다.

## 설명

에이전트를 강제로 종료합니다. 에이전트는 데이터를 [수집기](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#collector) 에 업로드하기 위해 마지막 시도를 합니다.

        값이 설정되지 않거나 `None` 으로 설정되면 에이전트는 구성 설정에서 재정의할 수 있지만 기본적으로 2.5초로 설정되는 기본 전역 시간 초과 값을 사용합니다.
