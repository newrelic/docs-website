## 문제

뉴렐릭 UI 의 [서버 측 설정](/docs/agents/manage-apm-agents/configuration/server-side-agent-configuration) 설정에서 <DNT>**Capture attributes**</DNT> 또는 <DNT>**Capture parameters**</DNT> 옵션을 활성화하는 구형 동작을 에뮬레이트하려고 합니다.

## 해결책

요청 매개변수를 캡처하도록 Python 에이전트 구성 파일을 사용자 지정하여 이 레거시 동작을 에뮬레이트할 수 있습니다. 자세한 내용은 Python 특성 문서에서 [요청 매개변수 캡처](/docs/agents/python-agent/attributes/python-attribute-examples#ex_req_params) 예를 참조하세요.
