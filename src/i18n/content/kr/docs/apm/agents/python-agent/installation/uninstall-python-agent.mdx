이 문서에서는 Python 에이전트를 제거하는 방법을 설명합니다. 에이전트를 일시적으로 비활성화하는 방법에 대한 지침은 에이전트 [비활성화](/docs/agents/manage-apm-agents/installation/disable-apm-agent#python) 를 참조하십시오.

Python 에이전트를 제거하려면:

1. 다음 방법 중 하나를 사용하십시오.

   * PIP로 설치한 경우 다음을 실행합니다.

     ```
     pip uninstall newrelic
     ```

   * easy_install을 사용하여 설치한 경우 다음을 실행합니다.

     ```
     easy_install -m newrelic
     ```

2. 제거 프로세스가 완료되면 앱을 다시 시작합니다.

<Callout variant="tip">
  APM UI에서 [앱을 제거](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) 할 수도 있습니다.
</Callout>
