## 통사론

```py
newrelic.agent.get_browser_timing_footer()
```

JavaScript 에이전트를 삽입하기 위한 브라우저 에이전트 바닥글을 생성합니다.

## 설명

이 호출은 [헤더 JavaScript 스니펫](/docs/agents/python-agent/python-agent-api/get_browser_timing_header) 과 함께 브라우저 모니터링을 활성화하는 데 사용되는 브라우저 에이전트 JavaScript 바닥글 스니펫을 반환합니다. `get_browser_timing_header` 과 함께 이 호출을 사용하여 브라우저 모니터링을 활성화하는 방법은 [수동 계측 지침](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) 을 참조하세요.

JavaScript 스니펫을 삽입하려는 모든 웹 요청에 대해 이 호출을 호출해야 합니다. 전역 범위에서 호출하고 캐시 **할 수 없습니다** .

## 반환 값

브라우저 에이전트 바닥글 JavaScript를 반환합니다.

## 예

### 수동 브라우저 모니터링 계측 [#browser-example]

머리글 및 바닥글 호출을 사용하는 방법에 대한 지침은 [수동으로 브라우저 모니터링 계측](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) 을 참조하세요.
