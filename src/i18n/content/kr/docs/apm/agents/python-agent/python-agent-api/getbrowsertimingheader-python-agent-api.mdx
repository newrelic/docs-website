## 통사론

```py
get_browser_timing_header()
```

JavaScript 에이전트를 삽입하기 위한 브라우저 에이전트 헤더를 생성합니다.

## 설명

이 호출은 [브라우저 모니터링](/docs/agents/python-agent/supported-features/page-load-timing-python) 을 활성화하는 데 사용할 수 있는 브라우저 에이전트 JavaScript 헤더 스니펫을 반환합니다. 애플리케이션에 대해 자동 계측이 불가능하거나 특정 페이지에 대해서만 브라우저 데이터를 수집하려는 경우 사용할 수 있습니다.

이것은 에이전트에 의한 [이 JavaScript의 자동 삽입에](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#select-apm-app) 대한 수동 대안입니다.[`get_browser_timing_footer`](/docs/agents/python-agent/python-agent-api/get_browser_timing_footer) 에서 이 메서드를 사용하여 브라우저 모니터링을 활성화하는 방법은 [수동 계측 지침](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) 을 참조하세요.

JavaScript 스니펫을 삽입하려는 모든 웹 요청에 대해 이 호출을 호출해야 합니다. 전역 범위에서 호출하고 캐시 **할 수 없습니다** .

## 반환 값

브라우저 에이전트 헤더 JavaScript를 반환합니다.

### 수동 브라우저 모니터링 계측 [#browser-example]
