## 통사론 [#syntax]

```py
get_browser_timing_header(nonce=None)
```

자바스크립트 에이전트를 삽입하기 위한 <InlinePopover type="browser"/>에이전트 헤더를 생성합니다.

## 설명 [#description]

<Callout variant="important">
  Nonced CSP 지원을 위한 Nonce 매개변수가 Python 에이전트 버전 9.4.0에 추가되었습니다.

  Python 에이전트 버전 9.3.0 이하에서는 브라우저 타이밍을 활성화하려면 [`get_browser_timing_footer`](/docs/agents/python-agent/python-agent-api/get_browser_timing_footer) API도 필요했습니다. 에이전트 버전 9.4.0 이상부터는 더 이상 필요하지 않습니다.
</Callout>

이 호출은 [브라우저 모니터링을](/docs/agents/python-agent/supported-features/page-load-timing-python) 활성화하는 데 사용할 수 있는 브라우저 에이전트 JavaScript 헤더 조각을 반환합니다. 애플리케이션에 대해 자동 계측이 불가능하거나, 논스 콘텐츠 보안 정책을 사용하거나, 특정 페이지에 대해서만 브라우저 데이터를 수집하려는 경우 이 방법을 사용할 수 있습니다.

이는 에이전트에 의한 [이 JavaScript 자동 삽입](/docs/browser/new-relic-browser/installation-configuration/add-apps-new-relic-browser#select-apm-app) 에 대한 수동 대안입니다. `get_browser_timing_footer` 와 함께 이 호출을 사용하여 브라우저 모니터링을 활성화하는 방법을 알아보려면 [브라우저 모니터링 수동 계측을](/docs/agents/python-agent/supported-features/page-load-timing-python#manual_instrumentation) 참조하세요.

이 호출은 JavaScript 스니펫을 삽입하려는 모든 웹 요청에 대해 호출되어야 합니다. <DNT>**cannot**</DNT> 전역 범위에서 호출되고 캐시됩니다.
