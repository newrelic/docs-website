## 통사론

```py
newrelic.agent.application(name=None)
```

에이전트 모니터링 앱에 해당하는 객체를 반환합니다. 일부 Python 에이전트 API 호출에 필요합니다.

## 설명

이 호출은 에이전트 모니터링 애플리케이션에 해당하는 개체를 반환합니다. `name` 값이 설정되지 않은 경우 [Python 에이전트 구성을 통해 설정된](/docs/agents/python-agent/installation-configuration/python-agent-configuration#app_name) 이름이 사용됩니다.

반환된 응용 프로그램 개체는 공개 API를 제공하지 않지만 일부 다른 API 호출에 필요합니다.

<Callout variant="important">
  이 호출은 응용 프로그램 개체만 반환합니다. 아직 발생하지 않은 경우 데이터 수집기에 애플리케이션 등록을 시도하지 않습니다.
</Callout>

## 반환 값

응용 프로그램 개체를 반환합니다. 객체 자체는 공개 API를 제공하지 않지만 일부 다른 API 호출에 필요합니다.

## 예

### 애플리케이션 객체 생성 및 사용 [#app-object-example]

응용 프로그램 개체를 생성하려면 다음을 사용합니다.

```py
import newrelic.agent

app = newrelic.agent.application()
```

일부 다른 Python 에이전트 API 호출은 애플리케이션 객체를 매개변수로 사용합니다. 예를 들어 [`notice_error`](/docs/agents/python-agent/python-agent-api/noticeerror-python-agent-api/) 호출을 참조하세요.
