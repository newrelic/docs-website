## 자주 묻는 질문 [#faq]

### 인증서 번들이란 무엇입니까? [#what-is-certificate-bundle]

인증서 번들은 기본적으로 신뢰 루트 목록을 제공합니다. 백엔드와 통신할 때 누구를 신뢰할 것인지 에이전트에게 알려줍니다.

### 인증서 번들이 제공되는 이유는 무엇입니까? [#why-certificate-bundle]

많은 고객이 도커에서 실행합니다. 이미지가 일반적으로 HTTP를 통해 통신하는 데 사용되지 않기 때문에 이러한 이미지는 자체 인증서 번들을 제공하지 않는 경우가 있습니다. 고객이 에이전트를 보다 쉽게 설치할 수 있도록 자체 번들을 제공합니다.

### 인증서 번들은 언제 사용됩니까? [#when-to-use-certificate-bundle]

시스템 인증서를 사용할 수 있는 경우 에이전트는 기본적으로 포함된 인증서 번들을 통해 인증서를 사용합니다. 인증서 번들은 시스템에서 인증서를 제공하지 않는 경우에만 사용됩니다.
