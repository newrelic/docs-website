[랙 미들웨어](/docs/agents/ruby-agent/frameworks/rack-middlewares) 는 Ruby 에이전트 버전 3.9.0 이상으로 자동으로 계측됩니다.

`NewRelic::Agent::Instrumentation::Rack` 모듈은 더 이상 사용되지 않습니다. Ruby 에이전트 버전 3.9.0 이상으로 업그레이드한 후 New Relic은 코드에서 이 모듈에 대한 참조를 제거할 것을 권장합니다. 자세한 내용은 [금속 컨트롤러 계측](/docs/agents/ruby-agent/frameworks/instrumenting-metal-controllers) 을 참조하십시오.
