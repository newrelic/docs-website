이 문서는 New Relic Ruby 에이전트를 제거하는 방법을 설명합니다. 에이전트를 일시적으로 비활성화하는 방법에 대한 지침은 에이전트 [비활성화](/docs/agents/manage-apm-agents/installation/disable-apm-agent#ruby) 를 참조하십시오.

Ruby 에이전트를 제거하려면:

1. Gemfile에서 `gem 'newrelic_rpm'` 줄을 삭제합니다.
2. **RAILS_ROOT/config/** 폴더에서 **newrelic.yml** 파일을 제거합니다.
3. **번들 설치를** 실행합니다.
4. 제거 프로세스가 완료되면 앱을 다시 시작합니다.

설치에 대한 자세한 정보는 [New Relic for Ruby를](/docs/agents/ruby-agent/getting-started/new-relic-ruby) 참조하십시오.
