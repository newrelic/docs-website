## 문제

New Relic Ruby 에이전트 버전 [6.0.0](/docs/releases/ruby) 에서는 공개적으로 지원되지 않는 많은 API가 더 이상 사용되지 않거나 삭제되었습니다. 삭제된 방법을 사용하는 경우 Ruby 에이전트 버전 6.0.0 이상으로 업그레이드하기 전에 권장되는 대체 방법으로 API를 업데이트해야 합니다. 더 이상 사용되지 않는 방법을 사용하는 경우 6.0.0 이상의 에이전트 버전과의 호환성을 보장하기 위해 최대한 빨리 API를 업데이트하는 것이 좋습니다.

## 해결책

다음 권장 대체품으로 API를 업데이트하십시오.

    id="external-start-segment"

  </Collapser>

  <Collapser

  </Collapser>

    id="transaction-start-datastore-segment"
