## 문제

Grape가 설치되지 않는다는 로그 메시지가 나타납니다.

```
INFO : Not installing New Relic supported Grape instrumentation because the third party newrelic-grape gem is present
```

## 해결책

`newrelic-grape` 보석을 제거합니다.

## 원인

New Relic은 타사 `newrelic-grape` gem을 지원하지 않습니다. Ruby 에이전트가 `newrelic-grape` gem을 감지하면 New Relic의 [내장 Grape 계측을](/docs/agents/ruby-agent/getting-started/ruby-agent-requirements-supported-frameworks) 설치하지 않습니다. 타사 gem을 제거하면 지원되는 Grape 계측을 사용할 수 있습니다.
