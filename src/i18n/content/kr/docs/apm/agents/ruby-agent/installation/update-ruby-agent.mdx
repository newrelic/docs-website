New Relic의 최신 기능, 향상된 기능 및 중요한 보안 패치를 최대한 활용하려면 Ruby 에이전트를 최신 버전으로 업데이트하는 것이 좋습니다. 특정 에이전트 업데이트에 대한 추가 정보는 [Ruby 에이전트 릴리스 정보](/docs/release-notes/agent-release-notes/ruby-release-notes) 를 참조하십시오.

## Ruby 에이전트 버전 확인 [#determine-versions]

앱이 현재 사용하는 Ruby 에이전트 버전을 확인하려면 다음을 수행하십시오.

1. New Relic의 [사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 에서 **Account settings > Connected agents** 를 선택합니다.
2. 선택 사항: **가장 오래된 에이전트 버전** 을 기준으로 정렬합니다.

## 에이전트 업데이트 [#Updating_the_Agent]

Ruby 에이전트를 업데이트하려면:

* 번들러 사용: 다음 명령을 실행합니다.

  ```
  bundle update newrelic_rpm
  ```

* 번들러를 사용하지 않음: 다음 명령을 실행합니다.

  ```
  gem update newrelic_rpm
  ```
