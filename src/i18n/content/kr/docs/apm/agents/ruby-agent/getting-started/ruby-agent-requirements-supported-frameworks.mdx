[New Relic의 Ruby 에이전트를 설치](/docs/agents/ruby-agent/installation-and-configuration/ruby-agent-installation) 하기 전에 호환되는 운영 체제, 보안 요구 사항 및 지원되는 프레임워크에 대한 이러한 요구 사항을 충족하는지 확인하십시오.

아직 계정이 없다면 먼저 [New Relic 계정을 만드세요](https://newrelic.com/signup) . 영원히 무료입니다.

## 운영체제 [#os]

Ruby 에이전트는 Linux, Solaris, FreeBSD 및 macOS와 같은 UNIX와 유사한 운영 체제를 지원합니다.

## 보안 요구 사항 [#security]

[데이터 수집을 위한 표준 보안 수단](/docs/accounts-partnerships/accounts/security/data-security)으로 앱 서버가 SHA-2(256비트)를 지원해야 합니다. SHA-1은 지원되지 않습니다. New Relic은 모든 인바운드 연결에 대해 TLS 1.2 이상이 필요합니다.

## 기타 APM 소프트웨어 [#other-monitoring-software]

귀하의 애플리케이션이 당사 에이전트 외에 다른 APM(애플리케이션 성능 모니터링) 소프트웨어를 사용하는 경우 당사는 당사 에이전트가 올바르게 작동한다고 보장할 수 없으며 기술 지원을 제공할 수 없습니다. 자세한 내용은 [다른 모니터링 소프트웨어 사용 시 오류를](/docs/apm/new-relic-apm/troubleshooting/errors-while-using-new-relic-apm-alongside-other-apm-software) 참조하십시오.
