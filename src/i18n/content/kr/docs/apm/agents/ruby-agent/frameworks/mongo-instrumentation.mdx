## 타사 및 extends_newrelic_rpm 계측 [#third_party_instrumentation]

`extends_newrelic_rpm` 및 `newrelic_mongo` gem은 모두 `mongo` 에 대한 기본 계측을 제공합니다. New Relic은 Ruby 에이전트의 내장 Mongo 지원과 함께 이러한 gem을 통해 Mongo 계측을 실행하는 것을 권장하지 않습니다. 이는 전체 데이터베이스 시간과 같은 메트릭을 두 배로 계산할 수 있습니다.

또한 `extends_newrelic_rpm` 및 `newrelic_mongo` gem은 둘 다 사용자 환경에서 gem을 완전히 제거할 수 없는 경우 비활성화하는 데 사용할 수 있는 고유한 설정을 사용합니다. 이러한 타사 보석의 경우 **newrelic.yml** 에 `disable_mongodb: false` 를 설정합니다.

## 계측 비활성화 [#Disabling_instrumentation]

에이전트에서 Mongo 계측을 비활성화하려면 **newrelic.yml** 에 다음을 추가하세요.

```
disable_mongo: true
```
