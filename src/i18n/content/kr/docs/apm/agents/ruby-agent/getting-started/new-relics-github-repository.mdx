New Relic은 GitHub의 [newrelic-ruby-agent](https://github.com/newrelic/newrelic-ruby-agent/tree/main) 에서 Ruby 저장소를 복제합니다. GitHub는 또한 제한된 rdoc 생성 문서를 호스팅합니다: [rdoc.info/github/newrelic/newrelic-ruby-agent](https://rubydoc.info/github/newrelic/newrelic-ruby-agent) .

메인 헤드는 일반적으로 New Relic의 최신 릴리스를 갖게 됩니다. 그러나 우리는 우위를 마스터에게 밀어붙일 수 있는 능력을 보유하고 있습니다. GitHub에서 릴리스를 다운로드하는 경우 적절한 태그를 사용하십시오. 우리는 일반적으로 일반 공급을 위해 태그를 지정하기 전에 릴리스의 미리 보기 버전을 작업 분기에 푸시합니다.

[support.newrelic.com](https://support.newrelic.com/) 에서 아이디어나 버그 수정 사항이 있으면 언제든지 분기하여 풀 리퀘스트를 보내주십시오(지점만 가능).
