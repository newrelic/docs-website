New Relic Ruby 에이전트는 이제 완전한 오픈 소스이며 GitHub의 [newrelic-ruby-agent](https://github.com/newrelic/newrelic-ruby-agent/tree/main) 에서 사용할 수 있습니다. 버전 6.12(2020년 7월 16일 출시)부터 Apache 라이선스 버전 2.0에 따라 라이선스가 부여됩니다. 이전 에이전트 버전의 경우 소스 코드에 포함된 LICENSE 파일을 확인하세요. Ruby 에이전트 버전 2.1.0의 소스 코드 태그를 사용하여 GitHub에서 사용할 수 있습니다.

`main` 의 책임자는 New Relic의 최신 릴리스를 갖게 됩니다. 그러나 우리는 가장자리를 `main` 로 푸시하는 기능을 보유하고 있습니다. `dev` 브랜치에는 출시되지 않은 작업이 포함되어 있습니다. GitHub에서 릴리스를 다운로드하는 경우 적절한 태그를 사용하세요. 일반적으로 RubyGems에 새 버전을 게시하기 전에 GitHub를 통해 미리보기 버전을 출시합니다.

포크를 통해 자유롭게 [끌어오기 요청을 보내고](https://github.com/newrelic/newrelic-ruby-agent/pulls), [문제를 만들고](https://github.com/newrelic/newrelic-ruby-agent/issues/new/choose), GitHub 저장소에서 바로 [토론을 시작해 보세요](https://github.com/newrelic/newrelic-ruby-agent/discussions) . 우리는 지역사회와 함께 일하는 것을 좋아합니다!
