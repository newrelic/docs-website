## 문제

[New Relic PHP 에이전트를 설치하고](/docs/agents/php-agent/installation/php-agent-installation-overview) 약간의 트래픽을 생성하고 최소 5분을 기다린 후 New Relic UI에 데이터가 표시되지 않습니다.

## 해결책

앱에 대한 트래픽을 생성하고 5분 이상 기다린 후에도 데이터가 표시되지 않으면 [New Relic Diagnostics](/docs/agents/manage-apm-agents/troubleshooting/new-relic-diagnostics) 를 사용하여 일반적인 문제를 자동으로 감지하고 문제 해결을 제안합니다. 그래도 문제가 해결되지 않으면 다음을 시도하십시오.
