## 통사론

```php
newrelic_accept_distributed_trace_headers (array $headers, string $transport_type)
```

분산 추적 헤더의 배열을 허용합니다.

## 요구 사항

PHP 에이전트 [버전 9.8 이상이](/docs/release-notes/agent-release-notes/php-release-notes) 필요합니다.

## 설명

[분산 추적을](/docs/agents/php-agent/features/distributed-tracing-php) 사용하면 요청이 분산 시스템을 통과할 때 이동하는 경로를 볼 수 있습니다. 분산 추적이 활성화된 경우 `newrelic_accept_distributed_trace_headers` 을 사용하여 헤더 페이로드를 수락합니다. 여기에는 W3C 추적 컨텍스트 및 New Relic 분산 추적 헤더가 모두 포함됩니다.

W3C 추적 컨텍스트 헤더만 허용하고 `newrelic.distributed_tracing_exclude_newrelic_header` INI 설정을 통해 New Relic Distributed Tracing 헤더를 비활성화할 수 있습니다.

## 반환 값

헤더가 성공적으로 수락되면 `true` 을 반환하고, 그렇지 않으면 `false` 을 반환합니다.
