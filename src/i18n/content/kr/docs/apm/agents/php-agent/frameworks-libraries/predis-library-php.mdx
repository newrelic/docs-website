New Relic은 PHP 에이전트 [릴리스 4.22](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-422099) 에서 Predis 라이브러리에 대한 지원을 추가했습니다.

에이전트는 `Predis\Client` 객체 또는 파이프라인에 대한 표준 호출을 계측합니다. 파이프라인을 통해 전송된 모든 명령은 개별 명령이 아니라 `pipeline` 이라는 단일 명령 측정항목으로 그룹화되고 처리됩니다.

PHP 에이전트는 `executeRaw` 또는 `transaction` 메서드, PubSub, 모니터 등의 호출을 지원하지 않습니다. 이들은 계측되지 않습니다.
