## 문제

PHP 에이전트를 설치하고 웹 서버를 다시 시작한 후 몇 분이 되었지만 뉴렐릭이 데이터를 보고하지 않습니다.

## 솔루션

`phpinfo()`를 사용하여 에이전트가 성공적으로 설치되었는지 확인합니다.

1. 웹사이트 내에 다음을 포함하는 간단한 `.php` 웹페이지를 만듭니다.

   ```php
   <?php phpinfo(); ?>
   ```

2. 웹 브라우저에서 이 페이지로 이동하고 알파벳순 모듈 목록을 스크롤하여 `newrelic`을 찾습니다. 이 목록에 `newrelic`이 표시되지 않으면 비표준 PHP 버전을 사용 중이거나 뉴렐릭을 설치한 후 웹 서버(또는 다른 PHP 호스트)를 다시 시작하지 않은 것입니다.

3. 웹 서버를 다시 시작하여 모듈 목록에 `newrelic`이 나타나는지 확인합니다.

4. 모듈 목록에 `newrelic`이 표시되지 않으면 [확장 디렉터리를 확인하여](/docs/php/php-kb-104) [비표준 버전의 PHP](/docs/php/php-agent-installation-non-standard-php)가 있지는 않은지 확인합니다.

5. 모듈 목록에 `newrelic`이 나타나면 `license.key` 항목에 모듈의 처음과 마지막 몇 문자가 표시되는지 확인합니다.<InlinePopover type="licenseKey"/>

6. `license.key`가 표시되지 않으면 [로드된 설정 파일 디렉터리를 확인합니다.](/docs/agents/php-agent/troubleshooting/checking-loaded-configuration-files-directory)
