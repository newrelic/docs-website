## 문제

**newrelic** 모듈이 `phpinfo()` 페이지에 나타나지만 UI에 데이터가 나타나지 않고 <InlinePopover type="licenseKey"/>거기에는 나타나지 않습니다.

## 해결책

**`newrelic.ini`** 파일은 PHP가 예상하는 디렉토리에 나타나지 않을 수 있으므로 구성 파일에서 읽지 않았을 수 있습니다. `newrelic.ini` 이 표시되어야 하는 위치를 결정하려면:

1. [`phpinfo()` 페이지](/docs/php/php-kb-103) 에서 페이지의 첫 번째 섹션을 보십시오.
2. `newrelic.ini` 이(가) **구문 분석된 추가 .ini 파일** 섹션에 나열되어 있는지 확인합니다.
3. `newrelic.ini` 파일이 **추가 .ini 파일** 또는 **추가 .ini 파일 구문 분석** 섹션에서 식별된 디렉토리에 있는지 확인하고 웹 서버에서 읽을 수 있는 권한이 적합한지 확인합니다.

`newrelic.ini` 이 구성 파일의 이름이 아닌 경우 라이센스 키 및 기타 구성 매개변수를 추가한 파일이 읽은 것으로 나열되는지 확인하십시오. 파일에 대한 권한이 웹 서버에 대한 액세스를 제공하는지 확인하십시오.
