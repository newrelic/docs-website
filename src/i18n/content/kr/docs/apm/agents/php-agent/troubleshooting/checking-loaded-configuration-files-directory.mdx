## 문제

<DNT>**newrelic**</DNT> 모듈이 `phpinfo()` 페이지에 표시되지만 UI에는 데이터가 표시되지 않으며 <InlinePopover type="licenseKey"/>도 표시되지 않습니다.

## 해결책

<DNT>**`newrelic.ini`**</DNT> 파일은 PHP가 예상하는 디렉토리에 나타나지 않을 수 있으므로 설정 파일에서 읽히지 않았을 수 있습니다. `newrelic.ini` 이 표시되어야 하는 위치를 결정하려면 다음을 수행하십시오.

1. [`phpinfo()` 페이지](/docs/php/php-kb-103) 에서 페이지의 첫 번째 섹션을 보십시오.

2. 라벨이 붙은 섹션에 `newrelic.ini` (가) 나열되어 있는지 확인하세요.

   <DNT>
     **Additional .ini files parsed**
   </DNT>

3. <DNT>
     **Scan this dir for addition .ini files**
   </DNT>

   또는

   <DNT>
     **Additional .ini files parsed**
   </DNT>

   섹션에 식별된 디렉토리에 `newrelic.ini` 파일이 있는지 확인하고 웹 서버에서 해당 파일을 읽을 수 있는 권한이 있는지 확인하세요.

`newrelic.ini` 이 구성 파일의 이름이 아닌 경우 라이센스 키 및 기타 구성 매개변수를 추가한 파일이 읽은 것으로 나열되는지 확인하십시오. 파일에 대한 권한이 웹 서버에 대한 액세스를 제공하는지 확인하십시오.
