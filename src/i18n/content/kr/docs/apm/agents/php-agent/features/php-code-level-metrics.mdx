코드 수준 지표를 통해 [New Relic CodeStream 확장을](/docs/codestream/observability/code-level-metrics) 사용하는 개발자는 코드의 개별 메서드와 함께 IDE에 상황에 따라 표시되는 APM 데이터를 볼 수 있습니다. 이를 통해 개발자는 코드를 작성하고 검토할 때 성능 문제를 보다 적극적으로 해결할 수 있습니다.

## 시작하기

먼저 지원되는 IDE에 New Relic CodeStream 확장을 [설치](/docs/codestream/start-here/install-codestream) 하고 로그인합니다.

<Callout variant="important">
  코드 레벨 지표는 PHP 에이전트 버전 10.4.0 이상에서 사용 가능하며 버전 10.6.0 - 11.10.0을 제외한 모든 버전에서는 기본적으로 비활성화되어 있습니다. 기본적으로 활성화되어 있습니다. 이 설정을 변경하려면 [설명서를](/docs/apm/agents/php-agent/configuration/php-agent-configuration/#inivar-code-level-metrics) 확인하세요.
</Callout>

## 에이전트 속성

PHP 에이전트는 애플리케이션에서 생성된 스팬에 다음 속성을 보고하고 첨부합니다.
