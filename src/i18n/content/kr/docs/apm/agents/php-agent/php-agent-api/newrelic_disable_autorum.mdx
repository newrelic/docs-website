## 통사론

```php
newrelic_disable_autorum()
```

특정 페이지에서 브라우저 모니터링 스니펫의 자동 삽입을 비활성화합니다.

## 요구 사항

에이전트 버전 [2.6.5.41](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-26541) 이상.

## 설명

이 호출은 현재 트랜잭션에 대한 [브라우저 모니터링](/docs/browser/new-relic-browser/getting-started/introduction-new-relic-browser) 에이전트의 자동 삽입을 비활성화합니다. 가능한 한 빨리 전화하십시오. [삽입으로 인해 문제가 발생](/docs/browser/new-relic-browser/troubleshooting/browser-javascript-injection-causes-problems-page) 하거나 JavaScript를 허용하지 않는 타사 서비스(예: [Google의 가속 모바일 페이지](/docs/browser/new-relic-browser/troubleshooting/google-amp-validator-fails-due-3rd-party-script) )에 페이지를 제공하는 경우 이 호출을 사용하여 JavaScript를 제거할 수 있습니다.

<Callout variant="tip">
  대신 API 호출로 브라우저 모니터링을 **활성화** 하려면 다음을 참조하세요.

  * [`newrelic_get_browser_timing_footer()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer)
  * [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header)
</Callout>

## 매개변수

이 호출은 매개변수를 허용하지 않습니다.

## 반환 값

New Relic 트랜잭션 내에서 호출된 경우 `true` 를 반환합니다. 그렇지 않으면 트랜잭션 외부인 경우 `null` 을 반환합니다(예: `newrelic_end_transaction()` 가 호출된 경우).

## 예

```php
function example() {
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        newrelic_disable_autorum();
    }
    // ...
}
```
