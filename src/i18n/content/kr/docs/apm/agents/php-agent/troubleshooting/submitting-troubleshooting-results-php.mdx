## 문제

New Relic PHP 에이전트를 설치하는 데 계속 문제가 있습니다.

## 해결책

[문제 해결 팁](/docs/apm/agents/php-agent/troubleshooting/no-data-appears-php) 을 따른 후에도 New Relic PHP 설치에 계속 문제가 있으면 [support.newrelic.com](https://support.newrelic.com) 에서 지원을 받으십시오.

New Relic 기술 지원 엔지니어와 협력할 때 다음 정보를 제공하십시오.

* 설치 과정을 어느 정도 진행하셨습니까?

* 사용 중인 설치 방법

* `newrelic.cfg` 파일 복사(있는 경우)

* **phpinfo** 페이지에 대한 링크 또는 HTML 파일로 저장된 **phpinfo** 의 출력. [**phpinfo** 를 생성](/docs/php/php-kb-103) 하려면 다음을 포함하는 .php 웹페이지를 생성합니다.

  ```php
  <?php phpinfo(); ?>
  ```

* [에이전트 및 데몬 로그 파일](/docs/php/php-kb-105)
