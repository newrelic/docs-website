2026년 3월부터 모범 사례 및 보안을 위해 뉴렐릭 PHP 에이전트 RPM 패키지의 새로운 릴리스가 새로운 키로 서명됩니다. PHP의 작동을 중단 없이 계속하려면 RPM 패키지에서 뉴렐릭 PHP를 설치하거나 업그레이드하기 전에 다음 명령을 실행하여 새 공개 키를 수락하고 가져와야 합니다.

`rpm --import https://download.newrelic.com/php_agent/NEWRELIC_RPM_BD2E199C.public`

## 키를 확인하세요

키를 가져온 후 다음 명령을 실행하여 키가 신뢰할 수 있는지 확인하십시오.

`rpm -qa | grep gpg-pubkey-bd2e199c`

키가 가져오기되어 신뢰할 수 있는 것으로 확인되면 명령은 다음과 같은 출력을 반환합니다.

`gpg-pubkey-bd2e199c-6970e650`

## 3월 이전에 새 키를 가져오지 않으면 어떻게 되나요?

새 키가 시스템에 추가되지 않으면 PHP 에이전트의 새 릴리스 설치 및 업그레이드가 `Error: GPG check FAILED` 과 같은 오류 메시지와 함께 실패합니다. 이전 버전의 에이전트에 고정된 설치는 이전과 마찬가지로 계속 작동합니다(단, 이러한 경우에도 새 키를 가져오는 것이 좋습니다).
