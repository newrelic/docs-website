## 문제

New Relic의 PHP 에이전트와 데몬을 설치했지만 데몬이 작동하지 않는 것 같습니다.

## 해결책

데몬이 올바르게 실행 중인지 확인하십시오.

1. 이 셸 명령을 실행하여 "newrelic-daemon"과 일치하는 실행 중인 프로세스를 나열합니다.

   ```bash
   ps -ef | grep newrelic-daemon
   ```

2. 다음과 같은 두 개의 실행 중인 데몬 프로세스에 대한 이전 명령의 출력을 확인하여 데몬이 실행 중인지 확인합니다. 목록에 없으면 다음 단계로 진행합니다.

   ```console
   newrelic     1368     1  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   newrelic     1370  1368  0 00:28 ?     00:00:00 /usr/bin/newrelic-daemon ...
   ```

3. 프로세스가 나열되지 않으면 데몬이 실행되고 있지 않으므로 시작 모드에 따라 데몬을 시작해야 합니다.

   * [에이전트 모드 시작](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-external) (기본값)
   * [외부 모드 시작](/docs/agents/php-agent/advanced-installation/starting-php-daemon-advanced#daemon-autostart) (고급)
