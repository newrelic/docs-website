## 문제

몇 분 후에 Heroku 앱의 APM [**요약** 페이지](/docs/apm/applications-menu/monitoring/apm-overview-page) 에 데이터가 나타나지 않습니다.

## 해결책

1. 이 Heroku toolbelt 명령을 사용하여 Heroku 로그에서 오류를 확인하십시오.

   ```
   heroku logs --tail
   ```

2. Heroku dyno의 파일이 [Java on Heroku](/docs/agents/java-agent/heroku/java-agent-heroku) 또는 [Java with Scala on Heroku](/docs/agents/java-agent/heroku/java-agent-scala-heroku) 설치 절차의 파일 수정 사항과 일치하는지 확인하십시오. 이 Heroku 도구 벨트 명령을 사용하십시오.

   ```
   heroku run bash
   ```

3. [데이터가 표시되지 않음(Java)](/docs/agents/java-agent/troubleshooting/no-data-appears-java) 의 단계를 시도합니다.
