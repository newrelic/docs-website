## 자바 런타임 [#java-runtimes]

Java 런타임 지원을 위해 New Relic은 [Oracle Premier Support Policy](https://www.oracle.com/java/technologies/java-se-support-roadmap.html) 를 따릅니다. New Relic은 해당 날짜 이후에 EOL 지원 권한을 보유합니다.
