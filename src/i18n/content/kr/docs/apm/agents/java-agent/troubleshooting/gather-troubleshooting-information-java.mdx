Java 에이전트의 문제 해결 절차(예 [: 데이터 없음](/docs/apm/agents/java-agent/troubleshooting/no-data-appears-java) 및 해당 범주의 다른 문서)로 문제가 해결되지 않은 경우 [로그 파일](#logfiles) 을 사용하여 문제를 해결하세요. 추가 지원이 필요하면 [support.newrelic.com](https://support.newrelic.com) 에 문의하십시오.

      <td>
        Java 에이전트에 대한 로그 파일이 표시됩니까?

        * 그렇다면 JVM 시작부터 몇 분 동안 실행되는 [`finest` 레벨 로그](/docs/agents/java-agent/troubleshooting/generate-debug-logs-troubleshooting-java) 를 포함하십시오.
        * 아니요인 경우 `newrelic.jar` 파일을 여는 JVM을 보여주는 JVM의 시작 로그를 포함합니다.
      </td>
