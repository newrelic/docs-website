이 문서에서는 New Relic Java 에이전트를 제거하는 방법을 설명합니다. 에이전트를 일시적으로 비활성화하는 방법에 대한 지침은 에이전트 [비활성화](/docs/agents/manage-apm-agents/installation/disable-apm-agent#java) 를 참조하십시오.

Java 에이전트를 제거하려면:

1. 앱 서버의 루트 디렉토리에서 `newrelic` 폴더를 삭제합니다.
2. 시작 스크립트에 에이전트를 포함하는 데 사용한 [JVM 인수](/docs/agents/java-agent/installation/include-java-agent-jvm-argument) 를 제거하십시오.
3. 해당하는 경우 백업된 시작 스크립트의 이름을 원래 이름으로 바꿉니다.
4. 이 단계를 완료한 후 앱을 [다시 시작](/docs/agents/manage-apm-agents/installation/uninstall-agent#restart) 하세요.

<Callout variant="tip">
  APM UI에서 [앱을 제거](/docs/apm/new-relic-apm/maintenance/remove-applications-from-new-relic-ui) 할 수도 있습니다.
</Callout>
