## 문제

[로그 데이터](/docs/agents/java-agent/troubleshooting/no-log-file-java) 에 `NullPointerException` 문제와 함께 `SEVERE:` 또는 `ERROR:` 이 표시되면 보고를 방해하는 문제가 있을 수 있습니다.

## 해결책

이 문제 해결 팁을 시작하기 전에 [최신 버전](/docs/agents/java-agent/installation/upgrade-java-agent) 의 New Relic Java 에이전트가 있는지 확인하십시오.

1. `SEVERE:` 및 `ERROR: NullPointerException` 가 있는 줄에 대해 Java 에이전트 로그 파일의 대소문자 구분 검색을 사용합니다.
2. 이러한 메시지가 포함된 행을 찾으면 [문제 해결 정보를 수집](/docs/agents/java-agent/troubleshooting/providing-troubleshooting-information-java) 하여 New Relic에서 지원을 받으십시오.

## 원인

버전 3.0.0과 3.2.3 사이의 New Relic의 Java 에이전트는 오류 메시지를 잘못 기록합니다. 예를 들어:

```
ERROR: javax/servlet/ServletOutputStream is marked as a weaved class, but no methods are matched to be weaved.
```

이것은 완전히 잘못된 것이며 걱정할 필요가 없습니다. 그러나 이러한 버전은 사용하지 마십시오. 대신 New Relic의 Java 에이전트 [최신 버전](/docs/agents/java-agent/installation/upgrade-java-agent) 으로 업데이트하십시오.
