## 문제

웹 애플리케이션에서 [Solr 플랫폼을](http://lucene.apache.org/solr/) 사용하고 있지만 Solr 데이터는 <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > APM & services > (select a Java app) > Monitor**</DNT> 의 앱에 대한 <InlinePopover type="apm"/>UI에 표시되지 않습니다.

## 해결책

Solr가 올바르게 [구성](/docs/agents/java-agent/getting-started/compatibility-requirements-java-agent#other) 되고 데이터가 New Relic에 보고되는지 확인하려면 해당하는 경우 다음을 수행하십시오.

* <DNT>
    **JMX:**
  </DNT>

  `<jmx/>` 태그를 사용하여 `solrconfig.xml` 의 Solr 설정에서 JMX가 활성화되어 있는지 확인하세요. `solrconfig.xml` 파일은 Solr 자체에 가장 많은 영향을 미치는 설정 파일입니다.

* <DNT>
    **Jetty:**
  </DNT>

  Jetty 사용자인 경우[Jetty용 JMX를 수동으로 활성화하세요](https://wiki.eclipse.org/Jetty/Tutorial/JMX#Enabling_JMXConnectorServer_for_Remote_Access).

* <DNT>
    **Solr metrics:**
  </DNT>

  Solr 메트릭에 대해서는 JMX 콘솔을 확인하세요. Solr 캐시와 업데이트는 MBean에서 제공됩니다.
