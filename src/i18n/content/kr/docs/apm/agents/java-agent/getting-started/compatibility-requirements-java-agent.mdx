[뉴렐릭의 Java 에이전트](/docs/agents/java-agent/getting-started/introduction-new-relic-java)에는 앱 서버, 프레임워크, 데이터베이스 및 메시지 큐 시스템 등 Java 에코시스템의 가장 보편적인 부분에 대한 계측이 내장되어 있습니다. 기본적으로 계측되지 않은 프레임워크와 라이브러리의 경우 [Java 커스텀 계측](/docs/agents/java-agent/custom-instrumentation/java-custom-instrumentation)을 사용하여 에이전트를 확장할 수 있습니다.

뉴렐릭의 Java 에이전트를 사용해 보길 원하시면, 지금 [뉴렐릭의 무료 계정을 신청](https://newrelic.com/signup)하십시오! 신용카드는 필요하지 않습니다.

## 에이전트 설치 요구 사항 [#java-requirements]

Java 에이전트를 설치하기 전에 시스템이 다음 요구 사항을 충족하는지 확인하십시오.

<CollapserGroup>
  <Collapser
    id="jvm"
    title="JVM"
  >
    Java 에이전트는 Java, Scala, Kotlin 및 Clojure를 비롯한 모든 JVM 기반 언어와 호환됩니다. 언어별 기능에 대한 계측 지원은, 아래의 [자동으로 계측된 프레임워크 및 라이브러리](#auto-instrumented) 섹션을 참조하십시오.

    지속적으로 혁신하며 고객에게 새로운 기능을 효율적으로 제공하기 위해, 이전 JVM 버전에 대한 엔지니어링 지원 및 기술 지원을 중단해야 하는 경우가 종종 있습니다. 아래 표에서 **No New Relic support(뉴렐릭 지원 안됨)**이라는 열은 이러한 상태의 에이전트 버전을 나타냅니다. 지원되지 않는 버전의 경우 이전 JVM 버전과 호환되는 에이전트 버전을 계속 사용할 수 있지만 새 기능 및 수정 사항은 이전 에이전트 버전에 포함되지 않습니다. 최신 에이전트 릴리스를 활용하려면 현재 지원되는 JVM 버전으로 업그레이드하는 것이 좋습니다.

## 내장된 계측 [#auto-instrumented]

[Java 에이전트를 설치하면](/docs/agents/java-agent/installation/install-java-agent) 널리 사용되는 많은 프레임워크 및 라이브러리가 자동으로 계측됩니다. 자동 계측을 통해 에이전트는 즉시 풍부한 데이터를 수집하고 데이터는 설치 후 몇 분 이내에 뉴렐릭 대시보드에 표시됩니다. 라이브러리가 자동으로 계측되지 않더라도 [커스텀 계측](/docs/agents/java-agent/custom-instrumentation/java-custom-instrumentation) 및 [Java 에이전트 API](/docs/agents/java-agent/api-guides/guide-using-java-agent-api)를 사용하여 데이터를 수집할 수 있습니다.

에이전트는 다음 프레임워크와 라이브러리를 자동으로 계측합니다.
