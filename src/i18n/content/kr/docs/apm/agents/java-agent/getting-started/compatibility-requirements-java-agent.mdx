[New Relic의 Java 에이전트](/docs/agents/java-agent/getting-started/introduction-new-relic-java) 에는 앱 서버, 프레임워크, 데이터베이스 및 메시지 대기열 시스템을 포함하여 Java 에코시스템에서 가장 인기 있는 부분의 내장 계측이 포함되어 있습니다. 기본적으로 계측되지 않은 프레임워크 및 라이브러리의 경우 [Java 사용자 지정 계측](/docs/agents/java-agent/custom-instrumentation/java-custom-instrumentation) 을 사용하여 에이전트를 확장할 수 있습니다.

New Relic의 Java 에이전트를 사용해 보고 싶으십니까? 무료 [로 New Relic 계정을 만드](https://newrelic.com/signup) 십시오! 신용 카드가 필요하지 않습니다.

## 에이전트 설치 요구 사항 [#java-requirements]

Java 에이전트를 설치하기 전에 시스템이 다음 요구 사항을 충족하는지 확인하십시오.

<CollapserGroup>

    title="JVM"

    Java 에이전트는 Java, Scala, Kotlin 및 Clojure를 비롯한 모든 JVM 기반 언어와 호환됩니다. 언어별 기능에 대한 계측 지원은 아래의 [자동으로 계측된 프레임워크 및 라이브러리](#auto-instrumented) 섹션을 참조하세요.

    지속적으로 혁신하고 고객에게 새로운 기능을 효율적으로 제공하기 위해 때때로 이전 JVM 버전에 대한 엔지니어링 지원 및 기술 지원을 중단해야 합니다. 아래 표에서 **No New Relic support** 라는 제목의 열은 이 상태에 있는 에이전트 버전을 나타냅니다. 지원되지 않는 버전의 경우 이전 JVM 버전과 호환되는 에이전트 버전을 계속 사용할 수 있지만 새 기능 및 수정 사항은 이러한 이전 에이전트 버전에 포함되지 않습니다. 최신 에이전트 릴리스를 활용하려면 현재 지원되는 JVM 버전으로 업그레이드하는 것이 좋습니다.

## 내장 계측 [#auto-instrumented]

[Java 에이전트를 설치하면](/docs/agents/java-agent/installation/install-java-agent) 널리 사용되는 많은 프레임워크 및 라이브러리가 자동으로 계측됩니다. 자동 계측을 통해 에이전트는 즉시 풍부한 데이터를 수집하고 데이터는 설치 후 몇 분 이내에 New Relic 대시보드에 표시됩니다. 라이브러리가 자동으로 계측되지 않더라도 [사용자 지정 계측](/docs/agents/java-agent/custom-instrumentation/java-custom-instrumentation) 및 [Java 에이전트 API](/docs/agents/java-agent/api-guides/guide-using-java-agent-api) 를 사용하여 데이터를 수집할 수 있습니다.

에이전트는 다음 프레임워크 및 라이브러리를 자동으로 계측합니다.

  >
    New Relic은 [다양한 데이터베이스 및 데이터베이스 드라이버에 대한 인스턴스 세부 정보를](/docs/apm/applications-menu/features/analyze-database-instance-level-performance-issues) 수집합니다. APM에서 특정 인스턴스 및 데이터베이스 정보 유형을 보는 기능은 New Relic 에이전트 버전에 따라 다릅니다.

    New Relic의 Java 에이전트 [버전 3.33.0 이상](/docs/release-notes/agent-release-notes/java-release-notes/java-agent-3330) 은 다음을 지원합니다.

    * [호환되는 모든 JDBC 드라이버](#JDBC)

    * Amazon DynamoDB 1.11.106 이상

    * DataStax Cassandra 드라이버 2.1.2 - 4.0.0

    * Jedis Redis 드라이버 1.4-2.10.x, 3.0.0-최신

    * MongoDB

      * 동기 클라이언트: 2.12.0에서 최신
      * Asynchronous/Reactive streams clients: 3.4.0 - 4.1.2

    * Spymemcached 2.11.0 - 2.12.x

      **예외:** `getBulk()` API 메서드 호출에 대한 인스턴스 수준 정보는 보고되지 않습니다.

      Java 에이전트는 이러한 데이터베이스 드라이버에 대한 느린 쿼리 추적 및 트랜잭션 추적에 대한 데이터베이스 이름 및 데이터베이스 서버/식별자 특성을 보고합니다. 추가 데이터 저장소에서 인스턴스 수준 정보를 요청하려면 [support.newrelic.com](https://support.newrelic.com) 에서 지원을 받으십시오.
  </Collapser>
