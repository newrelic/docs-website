**[one.newrelic.com > All capabilties](https://one.newrelic.com/all-capabilities) > APM** 으로 이동하면 New Relic [account ID](/docs/accounts-partnerships/accounts/account-setup/account-id) \[계정 ID] 와 연결된 모든 APM 앱의 색인이 표시됩니다. 페이지는 최신 [상태](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#health-status), 이벤트 및 성능 통계로 2분마다 애플리케이션 색인을 자동으로 새로 고칩니다.

<Callout variant="tip">
  모든 애플리케이션 및 서비스에 대한 높은 수준의 개요를 보려면 [엔터티 탐색기를](/docs/new-relic-one/use-new-relic-one/ui-data/new-relic-one-entity-explorer) 사용하십시오.
</Callout>

다음은 이 페이지에서 수행할 수 있는 몇 가지 작업입니다.
