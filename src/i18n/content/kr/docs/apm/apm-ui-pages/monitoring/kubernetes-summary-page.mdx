APM의 Kubernetes 페이지는 애플리케이션에 대한 다음 정보를 제공합니다.

## Kubernetes에 애플리케이션 연결 [#link-app]

Kubernetes 메타데이터를 표시하고 APM 에이전트에 연결할 수 있습니다. `MutatingAdmissionWebhook` 은 Pod에 필요한 환경 변수를 추가하는 데 사용됩니다. APM은 주입된 환경 변수를 사용하여 해당 APM 애플리케이션과 관련된 특정 Kubernetes 배포 정보를 표시합니다.

New Relic의 메타데이터 주입은 `newrelic-metadata-injection` 레이블을 통해 New Relic의 Kubernetes 통합을 설치하는 동안 자동으로 추가됩니다. 자세한 내용은 [애플리케이션을 Kubernetes에 연결하는 방법을 참조하세요](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-your-applications-kubernetes/) .
