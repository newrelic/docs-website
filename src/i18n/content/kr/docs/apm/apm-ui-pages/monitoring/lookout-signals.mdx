[New Relic Lookout](/docs/new-relic-solutions/new-relic-one/core-concepts/new-relic-lookout-monitor-your-estate-glance/) 을 사용하여 APM 서비스 엔터티의 세 가지 황금 신호(응답 시간, 오류율 및 처리량)의 편차를 감지하고 이러한 편차의 동작과 원인을 분석합니다.

## 어떻게 작동합니까?

<DNT>**APM summary**</DNT> 페이지에서 Lookout은 지난 5분 동안의 골든 시그널 값을 이전 시간 동안의 값과 비교합니다. Lookout이 신호에서 상당한 상승 또는 하락 추세를 감지하면 해당 APM 요약 채팅에 <DNT>**Diagnose**</DNT> 버튼이 나타납니다.

<img
  title="Embedded Lookout diagnose button"
  alt="A screenshot of the Lookout diagnose button embedded within in APM Web transactions time chart."
  src="/images/embedded-lookout-diagnose-button.webp"
/>

<figcaption>
  APM 웹 트랜잭션 시간 차트에 포함된 Lookout <DNT>**Diagnose**</DNT> 버튼
</figcaption>

버튼을 클릭하면 황금 신호에 대한 세부 정보를 탐색하는 데 사용할 수 있는 진단 Lookout 도구가 열립니다.
