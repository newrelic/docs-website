APM 의 <DNT>**Alerts**</DNT> 페이지에는 처리량, 오류율 비율, 기록 등 자세한 정보를 드릴다운할 수 있는 지난 2주 동안의 알림 표가 표시됩니다. 특정 제작 또는 모든 제작에 대한 공지 이력을 보실 수 있습니다.

## 특정 앱에 대한 알림 [#alert\_procedures][#alert_procedures]

특정 앱에 대한 알림 목록을 보려면: <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; APM &amp; Services &gt; (select an app) &gt; Events &gt; Issues &amp; activity**</DNT> 로 이동하세요.

<DNT>**Issues &amp; Activity**</DNT> 페이지에는 문제 목록, 공지 이벤트, 이상 작동 및 사후 분석(있는 경우)이 표시됩니다. 더 자세한 내용을 보려면 뉴렐릭의 표준 [UI 를](/docs/new-relic-solutions/new-relic-one/introduction-new-relic-platform) 사용하세요.

각 경고 알림에는 경고 아이콘과 요약 정보가 포함됩니다. 경고 페이지에서 경고 알림에 대한 세부 정보를 보려면 해당 링크를 선택합니다. 예를 들어:

* 특정 경고에 대한 세부 정보를 보려면 해당 이름을 선택합니다.
* 알림 목록의 범위를 좁히려면 <DNT>**Filter**</DNT> 를 사용하세요.
* 모든 앱과 호스트에 대한 모든 알림을 보려면 <DNT>**All alerts**</DNT> 링크를 선택하세요.
* 해당되는 경우 특정 Java 가상머신(JVM)을 선택하려면 <DNT>**JVMs**</DNT> 메뉴에서 원하는 항목을 선택하세요.
