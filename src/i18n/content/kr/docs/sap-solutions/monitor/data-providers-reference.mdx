데이터 공급자는 <DNT>**SAP systems**</DNT> 에서 모니터링 데이터를 수집하여 뉴렐릭으로 전송하는 구성 요소입니다. 이 기술 참조는 <DNT>**New Relic Monitoring for SAP Solutions**</DNT> (<DNT>**NRM4SAP**</DNT>)에서 사용 가능한 모든 데이터 제공자를 나열합니다. 특정 모니터링 그룹을 구성할 때 이를 사용하십시오. 데이터 공급자는 [모니터 시스템을 구성할 때](/docs/sap-solutions/install-and-configure/configure-new-relic-abap-add-on-package/#connect-systems) 선택하는 모니터링 그룹별로 구성됩니다.

## 이 참조 문서를 사용해야 하는 사람

다음의 경우 이 참조를 사용하세요:

* 다음에 대한 모니터링 그룹 구성 <DNT>**SAP systems**</DNT>
* 데이터 수집 관련 문제 해결
* 각 모니터링 그룹에서 수집하는 데이터 이해
* 데이터 공급자 설정 사용자 지정

<Callout variant="tip">
  이 테이블에는 상세한 기술 사양이 포함되어 있습니다. 대부분의 사용자는 **DP\_NAME** 및 **OUTPUT\_TYPE** 열만 참조하면 됩니다.

  이 참고 자료의 표는 가로로 스크롤하여 모든 열을 볼 수 있습니다. STATUS 열에 &quot;S&quot;로 표시된 데이터 제공자는 기본적으로 사용이 중단됩니다.
</Callout>

## 데이터 제공자 속성 이해하기

각 데이터 제공업체는 다음과 같은 기술적 속성을 가지고 있습니다.

* **MANDT**: 고객 번호 (일반적으로 001)
* **DP\_NAME**: 데이터 제공자를 식별하는 고유 식별자
* **OUTPUT\_TYPE**: 텔레메트리 데이터 유형(METRIC, EVENT, LOG, TRACE, CONFIG, BTP-LOG)
* **COMMUNICATOR**: 데이터 수집 조정을 담당하는 <DNT>**ABAP class**</DNT> (고급 문제 진단, 해결을 위한)
* **DATAPROVIDER**: 데이터를 검색하는 <DNT>**ABAP class**</DNT> (고급 문제 진단, 해결용)
* **FINISHER**: 데이터를 처리하고 전송하는 <DNT>**ABAP class**</DNT> (고급 문제 진단, 해결용)
* **STATUS**: 상태 표시기(S=일시 중지됨, 공백=활성)
* **DP\_GROUP**: 모니터링 그룹 할당
* **METRIC\_TYPE**: 메트릭의 경우 집계 유형(GAUGE, COUNT, SUMMARY)을 나타냅니다.
* **PERIOD**: 수집 빈도 (숫자 값)
* **PERIOD\_UNIT**: 수집 시간 단위 (분, 시간, 일)
* **SKIP\_RT**: 런타임 건너뛰기 플래그(X = 예)
* **RESERVED**: 향후 사용을 위해 예약됨
