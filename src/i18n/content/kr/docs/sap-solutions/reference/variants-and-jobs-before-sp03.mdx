## /NEWR/ABAP\_COLLECTION\_MANAGERN에 대한 변형을 만듭니다.

1. 트랜잭션 `SE38` 을 사용하여 `/NEWR/ABAP_COLLECTION_MANAGERN`을 설정합니다.
2. <DNT>**ABAP: Variants - Initial Screen**</DNT>에서 <DNT>**Program**</DNT> 필드에 `/NEWR/ABAP_COLLECTION_MANAGERN` 을(를) 입력합니다.
3. <DNT>**Variant**</DNT> 선택하고 <DNT>**Create**</DNT> 클릭합니다.
4. 값을 업데이트하려면 아래 단계를 따르세요.

      2. <DNT>**New Relic Account ID**</DNT> 이(가) 자동으로 채워지도록 하세요.

      3. <DNT>**Records per transmission**</DNT> 을 <DNT>**10000**</DNT> 로 설정합니다.

      4. 데이터 공급자 이름을 추가하려면 <DNT>**Multiple Selection (Active)**</DNT> 을 클릭하세요.

      5. <DNT>**Select Single Values**</DNT>에 다음을 추가합니다:

         * `NR.SAP.TRANSACTION`
         * `NR.SAP.TRANSACTION.COMPLETE`
         * `NR.SAP.SYSTEM.PROCESS`

      6. <DNT>**Execution Mode**</DNT> 로 <DNT>**Sequential Processing**</DNT> 선택합니다.

      7. <DNT>**DIA WP**</DNT> ≥ 30이면 <DNT>**6**</DNT>로 설정하고, 그렇지 않으면 <DNT>**3**</DNT>로 설정합니다.

      8. <DNT>**Attributes**</DNT>을(를) 클릭하고 설명을 추가한 다음 <DNT>**Save**</DNT>을(를) 클릭합니다.
    </TabsPageItem>

      2. <DNT>**New Relic Account ID**</DNT> 이(가) 자동으로 채워지도록 하세요.

      3. <DNT>**Records per transmission**</DNT> 을 <DNT>**10000**</DNT> 로 설정합니다.

      4. 데이터 공급자 이름을 추가하려면 <DNT>**Multiple Selection (Active)**</DNT> 을 클릭하세요.

      5. <DNT>**Select Single Values**</DNT>에 다음을 추가합니다:

         * `NR_SAP_IDOC`
         * `NR_SAP_IDOC_COMPLETE`

      6. <DNT>**Execution Mode**</DNT> 로 <DNT>**Sequential Processing**</DNT> 선택합니다.

      7. <DNT>**Attributes**</DNT>을(를) 클릭하고 설명을 추가한 다음 <DNT>**Save**</DNT>을(를) 클릭합니다.

    <TabsPageItem id="system-variant">

      2. <DNT>**New Relic Account ID**</DNT> 이(가) 자동으로 채워지도록 하세요.
      3. <DNT>**Records per transmission**</DNT> 을 <DNT>**10000**</DNT> 로 설정합니다.
      4. 데이터 공급자 이름을 추가하려면 <DNT>**Multiple Selection (Active)**</DNT> 을 클릭하세요.
      5. <DNT>**Exclude Single Values**</DNT>에서 <DNT>**TRACE**</DNT> 및 <DNT>**IDOC**</DNT> 공급자의 변형을 추가합니다.
      6. <DNT>**Execution Mode**</DNT> 로 <DNT>**Parallel Processing**</DNT> 선택합니다.
      7. <DNT>**DIA WP**</DNT> ≥ 30이면 <DNT>**6**</DNT>로 설정하고, 그렇지 않으면 <DNT>**3**</DNT>로 설정합니다.
      8. <DNT>**Attributes**</DNT>을(를) 클릭하고 설명을 추가한 다음 <DNT>**Save**</DNT>을(를) 클릭합니다.

## 일정 수집 작업

1. 트랜잭션 `SM36` 을 사용하여 세 개의 작업을 예약합니다.
2. 각 작업의 작업 이름을 입력하고 <DNT>**Job Class**</DNT> 을(를) <DNT>**C**</DNT>(으)로 설정합니다.
3. <DNT>**Step**</DNT> 클릭하고, 프로그램 이름 `RSBTONEJOB2` 입력하고, 각 작업에 대한 변형을 지정합니다.
4. <DNT>**Start Condition**</DNT>을(를) 클릭하고, 시작 날짜와 시간을 설정하고, <DNT>**Periodic Job**</DNT>을(를) 선택하고, <DNT>**Period Values**</DNT>을(를) 클릭한 다음 <DNT>**Other Period**</DNT>을(를) 클릭하고, 분에 <DNT>**5**</DNT> 을(를) 입력합니다.
