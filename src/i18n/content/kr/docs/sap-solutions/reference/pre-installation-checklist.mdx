뉴렐릭 모니터링 for SAP Solutions를 설치하기 전에 필요한 모든 정보를 수집하려면 이 체크리스트를 사용하세요. 설치를 시작하기 전에 먼저 준비 섹션을 완료한 다음, 준비 체크리스트 테이블을 작성하십시오.

## 설치 전 준비

### 뉴렐릭 계정을 확인하세요

* 뉴렐릭 계정이 있는지 확인하거나, 새 계정을 생성하세요. 어떤 데이터센터 지역(미국 또는 EU)을 사용하는지 확인하세요.

### SAP 시스템 목록을 준비하세요

모니터링할 각 SAP 시스템에 대해 다음 정보를 제공하세요.

* **시스템 ID 및 클라이언트**: 시스템과 연관된 클라이언트를 식별합니다.

* **데이터베이스 정보**: 데이터베이스 공급업체(예: HANA, 작업) 및 해당 버전을 지정합니다.

* **소프트웨어 구성 요소 세부 정보**: 다음 구성 요소에 대한 SAP 릴리스 및 패치 수준을 포함합니다.

  * **SAP\_BASIS**
  * **핵심**

### 중앙 SAP 시스템 지정

뉴렐릭 모니터링 for SAP Solutions(NRM4SAP) 설치를 위한 중앙 시스템 역할을 할 SAP 시스템을 하나 선택하십시오.

* **설치 세부 정보**: ABAP 구성 요소와 설정이 설치될 시스템 ID와 클라이언트를 제공합니다.
* **프록시 서버 정보**: 프록시 서버를 사용하는 경우 호스트 이름과 포트를 지정하세요. 뉴렐릭 One 플랫폼에 접속하기 위한 관련 프록시 서버 정책을 포함하세요.

### 네트워크 접근성

네트워크 연결을 확인하고 API 엔드포인트 설정을 검토하세요:

* 방화벽이 뉴렐릭에 대한 HTTPS 트래픽을 차단하지 않는지 확인하세요.
* 해당하는 경우 프록시 서버 세부 정보를 문서화하십시오.
* 지역(미국 또는 EU)에 대한 RFC 대상 설정을 검토하십시오

전체 네트워크 요구 사항 및 API 엔드포인트 세부 정보는 [네트워크 요구 사항 및 API 엔드포인트](/docs/sap-solutions/reference/network-accessibility)를 참조하세요.
