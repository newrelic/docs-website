이 참조는 뉴렐릭 모니터링 for SAP Solutions에서 모니터링 목적으로 가져온 SAP 테이블을 나열합니다. 이 테이블은 AIF, SLT, IDOC, 작업, 이메일, 전송, RFC, 비즈니스 프로세스 및 변경 문서를 포함한 다양한 모니터링 영역에 대한 데이터를 제공합니다.

## 권한 부여 및 테이블 액세스

뉴렐릭 모니터링 for SAP 권한 역할은 RFC 연결을 설정할 때 모니터 시스템의 RFC 사용자에게 광범위한 테이블 표시 액세스를 제공합니다. 커스텀 Z-테이블을 포함한 추가 SAP 테이블을 검색하거나 IDOC 또는 메시지 페이로드 데이터를 검색할 때 추가적인 사용자 정의를 용이하게 하려면 액세스를 광범위하게 유지하는 것이 좋습니다. 하지만 테이블 액세스를 제한하려면 이 목록을 사용하세요.

<Callout variant="important">
  커스텀 데이터 수집기가 테이블에서 직접 데이터를 가져오는 경우, 새 데이터 제공자가 작동하려면 보안 역할을 업데이트해야 할 수 있습니다.
</Callout>

## 이 참조 문서를 사용해야 하는 사람

* 데이터 수집 범위를 검토하는 SAP Basis 관리자
* 테이블 액세스 요구 사항을 감사하는 보안 팀
* 테이블 사용량을 모니터링하는 데이터베이스 관리자
