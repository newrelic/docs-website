* 총 모니터되는 <DNT>**SIDs**</DNT> 이(가) 10개 미만이고 인스턴스가 50개를 초과하지 않으면 전용 서버가 필요하지 않습니다. <DNT>**SIDs**</DNT> 이(가) 10을 초과하거나 인스턴스가 100을 초과하는 경우 전용 서버가 권장됩니다.
* 최대 각각 96G의 메모리와 12 코어 CPU를 갖춘 두 개의 <DNT>**App Server**</DNT> 인스턴스가 있는 전용 서버를 사용해야 합니다. 모니터링 시스템이 <DNT>**HANA database**</DNT>을(를) 사용하는 경우, 데이터베이스 서버 인스턴스를 애플리케이션 서버 인스턴스에서 분리하는 것을 권장합니다.
