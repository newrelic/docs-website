이 참조 문서는 SAP 시스템을 뉴렐릭에 연결하기 위한 네트워크 설정 세부 정보를 제공합니다. RFC 대상 및 방화벽 규칙을 구성할 때 이 사양을 사용하십시오.

## 이 참조 문서를 사용해야 하는 사람

* RFC 연결을 구성하는 SAP Basis 관리자
* 방화벽 규칙을 설정하는 네트워크 관리자
* 네트워크 요구 사항을 준비하는 설치 팀

## RFC 대상 설정

뉴렐릭 API와의 통신을 활성화하려면 SAP 시스템에서 이러한 RFC(Remote Function Call) 대상을 구성하십시오. 뉴렐릭 계정이 호스팅되는 지역의 엔드포인트를 사용하세요.

**참고**: 연결 유형 &quot;G&quot;는 외부 서버에 대한 HTTP 연결을 나타냅니다.

뉴렐릭 계정 지역에 따라 엔드포인트 설정을 선택하세요:

* 계정이 미국 데이터센터에 있는 경우 **미국 지역 엔드포인트** 를 사용합니다.
* 계정이 EU 데이터센터에 있는 경우 **EU 지역 엔드포인트** 를 사용하세요.

## 연결 확인

RFC 대상을 구성한 후 SAP 트랜잭션 코드 `SM59`을(를) 사용하여 연결을 테스트합니다:

1. RFC 대상을 선택합니다.
2. 딸깍 하는 소리 <DNT>**Connection Test**</DNT>
3. 뉴렐릭 엔드포인트에 대한 성공적인 연결 확인
