## 서버 가용성 데이터 제공자가 작동하지 않음

**증상**: `NR_SAP_INSTANCE_AVAILABILITY` 및 `NR_SAP_SERVER_LIST` 데이터 공급자가 데이터를 수집하지 못합니다.

**원인**: 모니터 서버의 `DEFAULT.PFL` 프로필이 <DNT>**SAPCONTROL**</DNT> 웹 서비스에 대해 구성되지 않았습니다.

**해결책**:

1. `DEFAULT.PFL` 파일을 열고 이 줄이 존재하는지 확인합니다:

`service/protectedwebmethods = SDEFAULT`

2. 줄이 누락된 경우 다음 파라미터를 추가합니다:

`service/protectedwebmethods = SDEFAULT -GetQueueStatistic -ABAPGetWPTable -EnqGetStatistic -GetProcessList -GetEnvironment -GetVersionInfo -GetSystemInstanceList`

3. <DNT>**SAP service**</DNT> 재시작:

```yaml
stopsap sapstartsrv
startsap sapstartsrv
```

4. 설정을 확인합니다:

```yaml
sapcontrol -prot GSOAP_HTTP -nr 00 -function GetSystemInstanceList
```

**예상 결과**: 명령이 오류 없이 시스템 인스턴스 목록을 반환합니다.
