SAP 솔루션용 뉴렐릭 모니터링의 이전 버전에서 업그레이드하려면 현재 설치된 버전에 맞는 올바른 업그레이드 패키지를 사용해야 합니다.

1. **System &gt; Status** 로 이동합니다.
2. **NEWR** 을 검색하여 제품 버전을 확인하세요.
3. 클라이언트 000에서 **SAINT** 트랜잭션을 실행하여 ABAP 추가 기능을 업데이트합니다.

## 백업 및 업데이트 설정.

<Callout variant="tip">
  업그레이드 중에 덮어쓰기를 방지하려면 수정된 데이터 공급자 설정이나 사용자 지정 데이터 공급자를 설정 템플릿에 저장하세요. V3.0 ABAP 추가 기능이 설치되지 않았더라도 클라이언트 000에서 새 템플릿을 복사하기 전에 템플릿이 저장되었는지 확인하세요.
</Callout>

1. 메뉴 `/n/NEWR/INSTALL` 에 접속하여 **업데이트 또는 저장 설정 템플릿** 으로 이동합니다.

2. VM 모니터링과 관련된 HDB 모니터링 템플릿을 제거하려면 `/NEWR/REP_DELETE_INS_TEMPLATE` 실행합니다.

3. `DBACockpit` 사용하여 HANA 데이터베이스에 대한 ABAP 데이터 연결을 설정합니다.

4. 클라이언트 000에서 설정 템플릿을 복사합니다.

5. 모델링 시스템 설정을 수정합니다.

   * 새로운 DB 연결을 추가합니다.
   * ☐ Enterprise Connector(예: NR\_EI\_AGENT)에 대한 참조를 제거합니다.
   * 모니터링 그룹에 HANA\_DB를 추가합니다.
   * 설정을 업데이트하려면 데이터 공급자 구성을 다시 생성하세요.
