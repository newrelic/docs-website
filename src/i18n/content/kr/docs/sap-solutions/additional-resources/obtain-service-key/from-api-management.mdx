<Steps>
  <Step>
    **BTP subaccount** \[BTP 하위 계정] 에 로그인하세요.
  </Step>

  <Step>
    **BTP cockpit** \[BTP 콕핏]에서 **Instances and Subscriptions** \[인스턴스 및 구독]을 클릭합니다.
  </Step>

  <Step>
    오른쪽 상단의 **Create**\[만들기]를 클릭하세요.
  </Step>

  <Step>
    팝업 화면에서 다음을 선택하세요.

    * **서비스**: API 관리, API 포털
    * **계획**: apiportal-apiaccess
    * **런타임 환경**: Cloud Foundry
    * **공간**: BTP 관리자에게 문의하세요
    * **인스턴스 이름**: 조직의 명명 규칙을 따르세요.
  </Step>

  <Step>
    비용 면책 조항을 확인하세요.
  </Step>

  <Step>
    **Next** \[다음을] 클릭한 다음 **Roles** \[역할]: `APIPortal.Administrator` 을 선택합니다.
  </Step>

  <Step>
    **Next** \[다음을] 클릭하여 세부 정보를 검토하세요.
  </Step>

  <Step>
    **Create** \[만들기 를] 클릭합니다.
  </Step>

  <Step>
    서비스를 만든 후 **Instances** \[인스턴스] 탭에서 해당 서비스를 찾으세요.
  </Step>

  <Step>
    인스턴스 옆에 있는 줄임표(`...`)를 클릭한 다음 **Create Service Key** \[서비스 키 만들기를] 클릭합니다.
  </Step>

  <Step>
    Service Key Name \[서비스 키 이름]과 **Credential Type** \[자격 증명 유형]: JSON 형식의 binding-secret을 입력합니다.
  </Step>

  <Step>
    **Create** \[만들기를] 클릭한 다음 서비스 키 옆에 있는 줄임표(`...`)를 클릭하여 다운로드하세요.
  </Step>
</Steps>
