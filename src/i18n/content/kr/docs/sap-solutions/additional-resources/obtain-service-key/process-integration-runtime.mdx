<Steps>
  <Step>
    **BTP subaccount** \[BTP 하위 계정] 에 로그인하세요.
  </Step>

  <Step>
    **BTP cockpit** \[BTP 콕핏]에서 **Instances and Subscriptions** \[인스턴스 및 구독]을 클릭합니다.
  </Step>

  <Step>
    오른쪽 상단의 **Create**\[만들기]를 클릭하세요.
  </Step>

  <Step>
    팝업 화면에서 다음을 선택하세요.

    * **서비스**: SAP 프로세스 통합 런타임.
    * **계획**: API.
    * **런타임 환경**: Cloud Foundry.
    * **공간**: BTP 관리자에게 문의하세요.
    * **인스턴스 이름**: 조직의 명명 규칙을 따르세요.

    <img title="Process Integration Instance" alt="A screenshot displaying the Process Integration Instance" src="/images/process-integration-instance.webp" width="600" />
  </Step>

  <Step>
    **Next** \[다음을] 클릭한 후 다음을 선택하세요.

    * **역할**: `MonitoringDataRead`
    * **부여 유형**: 기본 클라이언트 자격 증명(다른 유형을 사용하는 경우 BTP 관리자 및 보안 팀에 확인하세요).

    <img title="BTP Roles and Grant Types" alt="A screenshot displaying the BTP Roles and Grant Types" src="/images/btp-roles-grant-types.webp" width="600" />
  </Step>

  <Step>
    **Next** \[다음을] 클릭하여 세부 정보를 검토하세요.
  </Step>

  <Step>
    **Create** \[만들기 를] 클릭합니다.
  </Step>

  <Step>
    서비스를 만든 후 **Instances** \[인스턴스] 탭에서 해당 서비스를 찾으세요.

    <img title="BTP Instance" alt="A screenshot displaying the BTP Instance" src="/images/btp-instance.webp" width="600" />
  </Step>

  <Step>
    인스턴스 옆에 있는 줄임표(`...`)를 클릭한 다음 **Create Service Key** \[서비스 키 만들기를] 클릭합니다.
  </Step>

  <Step>
    **Service Key Name** \[서비스 키 이름] 과 **Key Type** \[키 유형을] 입력하세요: `ClientId/Secret`.

    <img title="BTP New Service Key" alt="A screenshot displaying the BTP New Service Key" src="/images/btp-new-service-key.webp" width="600" />
  </Step>

  <Step>
    **Create** \[만들기 를] 클릭합니다.
  </Step>

  <Step>
    서비스 키 옆에 있는 줄임표(`...`)를 클릭하고 **Download** \[다운로드를] 클릭합니다.

    <img title="BTP Download Service Key" alt="A screenshot displaying the BTP Download Service Key" src="/images/btp-download.webp" width="600" />
  </Step>
</Steps>
