<Steps>
  <Step>
    **CALM subaccount** \[CALM 하위 계정] 에 로그인한 다음 **Instances and Subscriptions** \[인스턴스 및 구독을] 클릭합니다.
  </Step>

  <Step>
    오른쪽 상단의 **Create**\[만들기]를 클릭하세요.
  </Step>

  <Step>
    팝업 화면에서 다음을 선택하세요.

    * **서비스**: SAP Cloud ALM API
    * **플랜**: 표준
    * **런타임 환경**: Cloud Foundry
    * **공간**: BTP 관리자에게 문의하세요
    * **인스턴스 이름**: 조직의 명명 규칙을 따르세요.

    <img title="SAP Cloud ALM API" alt="A screenshot displaying the SAP Cloud ALM API" src="/images/sap-cloud-alm-api.webp" width="600" />
  </Step>

  <Step>
    **Next**클릭합니다.
  </Step>

  <Step>
    교정 탭 아래의 교정 이름과 함께 [범위 관리](https://help.sap.com/docs/cloud-alm/apis/managing-scopes) 에서 필요한 범위를 검토하고 삽입하세요.

    <img title="Manage Scopes" alt="A screenshot displaying the Manage Scopes" src="/images/manage-scope.webp" width="600" />
  </Step>

  <Step>
    **Next** \[다음을] 클릭하여 세부 정보를 검토하세요.

    <img title="Review Details" alt="A screenshot displaying the Review Details" src="/images/cloud-alm-new-instance.webp" width="600" />
  </Step>

  <Step>
    **Create** \[만들기 를] 클릭합니다.
  </Step>

  <Step>
    서비스를 만든 후 **Instances** \[인스턴스] 탭에서 해당 서비스를 찾으세요.
  </Step>

  <Step>
    인스턴스 옆에 있는 줄임표(`...`)를 클릭한 다음 **Create Service Key** \[서비스 키 만들기를] 클릭합니다.
  </Step>

  <Step>
    서비스 키 이름과 **키 유형을** 입력하세요: `ClientId/Secret`.
  </Step>

  <Step>
    **Create** \[만들기 를] 클릭합니다.
  </Step>

  <Step>
    서비스 키 옆에 있는 줄임표(`...`)를 클릭하고 **Download** \[다운로드를] 클릭합니다.
  </Step>
</Steps>
