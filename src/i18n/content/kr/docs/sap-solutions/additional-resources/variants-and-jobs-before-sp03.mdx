## /NEWR/ABAP\_COLLECTION\_MANAGERN에 대한 변형을 만듭니다.

1. 트랜잭션 **SE38을** 사용하여 `/NEWR/ABAP_COLLECTION_MANAGERN` 설정합니다.
2. **ABAP: Variants - Initial Screen** \[ABAP: 변형 - 초기 화면] 에서 **Program** \[프로그램] 필드에 `/NEWR/ABAP_COLLECTION_MANAGERN` 입력합니다.
3. **Variant** \[변형을] 선택하고 **Create** \[만들기를] 클릭합니다.
4. 값을 업데이트하려면 아래 단계를 따르세요.

<CollapserGroup>
  <Collapser id="trace-variant" title="트레이스 변종">
    1. 변형 이름(예: **TRACE\_ONLY**) 을 입력하고 **Create** \[만들기를] 클릭합니다.

    2. **New Relic Account ID** \[뉴렐릭 계정 ID가] 자동으로 채워지도록 하세요.

    3. **전송당 레코드 수를** **10000** 으로 설정합니다.

    4. 데이터 공급자 이름을 추가하려면 **Multiple Selection (Active)** \[다중 선택(활성)을] 클릭하세요.

    5. **Select Single Values** \[단일 값 선택] 에 다음을 추가합니다.

       * `NR.SAP.TRANSACTION`
       * `NR.SAP.TRANSACTION.COMPLETE`
       * `NR.SAP.SYSTEM.PROCESS`

    6. **Execution Mode** \[실행 모드] 로 **Sequential Processing** \[순차 처리를] 선택합니다.

    7. **DIA WP** ≥ 30이면 **6** 으로 설정하고, 그렇지 않으면 **3** 으로 설정합니다.

    8. **Attributes** \[속성을] 클릭하고 설명을 추가한 다음 **Save** \[저장을] 클릭합니다.
  </Collapser>

  <Collapser id="idoc-variant" title="IDOC 변형">
    1. 변형 이름(예: **IDOC\_ONL**Y) 을 입력하고 **Create** \[만들기를] 클릭합니다.

    2. **New Relic Account ID** \[뉴렐릭 계정 ID가] 자동으로 채워지도록 하세요.

    3. **전송당 레코드 수를** **10000** 으로 설정합니다.

    4. 데이터 공급자 이름을 추가하려면 **Multiple Selection (Active)** \[다중 선택(활성)을] 클릭하세요.

    5. **Select Single Values** \[단일 값 선택] 에 다음을 추가합니다.

       * `NR_SAP_IDOC`
       * `NR_SAP_IDOC_COMPLETE`

    6. **Execution Mode** \[실행 모드] 로 **Sequential Processing** \[순차 처리를] 선택합니다.

    7. **Attributes** \[속성을] 클릭하고 설명을 추가한 다음 **Save** \[저장을] 클릭합니다.
  </Collapser>

  <Collapser id="system-variant" title="시스템 변형">
    1. 변형 이름(예: **System**) 을 입력하고 **Create** \[만들기를] 클릭합니다.
    2. **New Relic Account ID** \[뉴렐릭 계정 ID가] 자동으로 채워지도록 하세요.
    3. **전송당 레코드 수를** **10000** 으로 설정합니다.
    4. 데이터 공급자 이름을 추가하려면 **Multiple Selection (Active)** \[다중 선택(활성)을] 클릭하세요.
    5. **Exclude Single Values** \[단일 값 제외]에서 **TRACE** 및 **IDOC** 공급자의 변형을 추가합니다.
    6. **실행 모드** 로 **병렬 처리를** 선택합니다.
    7. **DIA WP** ≥ 30이면 **6** 으로 설정하고, 그렇지 않으면 **3** 으로 설정합니다.
    8. **Attributes** \[속성을] 클릭하고 설명을 추가한 다음 **Save** \[저장을] 클릭합니다.
  </Collapser>
</CollapserGroup>

## 일정 수집 작업

1. **SM36** 트랜잭션을 사용하여 세 개의 작업을 예약합니다.
2. 각 작업에 대한 작업 이름을 입력하고 **작업 클래스를** &quot;C&quot;로 설정합니다.
3. **단계를** 클릭하고 프로그램 이름 **RSBTONEJOB2를** 입력한 다음 각 작업에 대한 변형을 지정합니다.
4. **시작 조건을** 클릭하고 시작 날짜와 시간을 설정하고, **주기적 작업을** 선택하고, **기간 값을** 클릭한 다음 **기타 기간을** 클릭하고 분을 나타내는 **5를** 입력합니다.
