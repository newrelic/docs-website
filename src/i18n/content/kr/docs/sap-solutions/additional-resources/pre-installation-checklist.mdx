## 뉴렐릭 계정을 확인하세요

* 이미 SAP 시스템 모니터링용 뉴렐릭 계정을 갖고 계십니까?

  * **예**: 귀하의 계정이 연결된 뉴렐릭 데이터센터(미국 또는 EU)를 지정하세요.
  * **아니요**: 데이터센터 중 하나에 연결된 새 계정을 만드세요.

## SAP 시스템 목록을 준비하세요

* 모니터링할 각 SAP 시스템에 대해 다음 정보를 제공하세요.

  * **시스템 ID 및 클라이언트**: 시스템과 연관된 클라이언트를 식별합니다.

  * **데이터베이스 정보**: 데이터베이스 공급업체(예: HANA, 작업) 및 해당 버전을 지정합니다.

  * **소프트웨어 구성 요소 세부 정보**: 다음 구성 요소에 대한 SAP 릴리스 및 패치 수준을 포함합니다.

    * **SAP\_BASIS**
    * **핵심**

## 중앙 SAP 시스템 지정

* NRM4SAP 설치를 위한 중앙 시스템 역할을 할 SAP 시스템 하나를 선택하세요.

  * **설치 세부 정보**: ABAP 구성 요소와 설정이 설치될 시스템 ID와 클라이언트를 제공합니다.
  * **프록시 서버 정보**: 프록시 서버를 사용하는 경우 호스트 이름과 포트를 지정하세요. 뉴렐릭 One 플랫폼에 접속하기 위한 관련 프록시 서버 정책을 포함하세요.
