서비스 팩은 `K-<Version Number>SP<SP Number>IN<Software Component>.SAR` 명명 규칙을 사용하여 `.SAR` 파일로 제공됩니다.

<Steps>
  <Step>
    트랜잭션 `SPAM` 사용하여 서비스 팩을 설치하세요.
  </Step>

  <Step>
    클라이언트 000에서 새로운 설정 템플릿을 로드합니다.
  </Step>

  <Step>
    클라이언트 000에서 **Install New Release and Patch** &gt; **Copy Template Tables**로 이동합니다.
  </Step>

  <Step>
    **Install New Release and Patch** &gt; **Generate Configuration for New Data Providers**의 단계에 따라 새로운 데이터 공급자 구성을 생성합니다.
  </Step>

  <Step>
    **Install New Release and Patch** &gt; **Verify Installation Completeness** 클릭하여 설치를 확인합니다.
  </Step>
</Steps>
