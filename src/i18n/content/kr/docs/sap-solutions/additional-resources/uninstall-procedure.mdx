중앙 시스템에서 ABAP 구성 요소를 제거하려면:

1. **client 000** \[클라이언트 000] 에 로그인합니다.

2. 프로세서 코드 **SAINT 를** 입력하세요.

3. **Uninstallable Components** \[제거할 수 없는 구성 요소] 탭을 클릭합니다.

4. **NEWR** 구성 요소를 선택하고 제거할 올바른 버전을 확인하세요.

5. **Start** \[시작을] 클릭하세요.

6. 이후 화면에서는 녹색 체크 표시를 클릭하세요.

7. 프로세스가 완료되면 **Finish** \[마침을] 클릭합니다.

8. **NEWR** 구성 요소가 더 이상 **설치된 구성 요소** 탭에 나타나지 않는지 확인합니다.
