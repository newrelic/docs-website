* 모니터링되는 총 SID가 10개 미만이고 인스턴스가 50개를 초과하지 않는 경우 전용 서버가 필요하지 않습니다. SID가 10개가 넘거나 인스턴스가 100개가 넘는 경우 전용 서버를 권장합니다.
* 최대 2개의 앱 서버 인스턴스가 있는 전용 서버는 각각 96G의 메모리와 12개의 코어 CPU를 사용해야 합니다. 모니터링 시스템에서 HANA 데이터베이스를 사용하는 경우 데이터베이스 서버 인스턴스와 애플리케이션 서버 인스턴스를 분리하는 것이 좋습니다.
