SAP Cloud ALM 서비스 모니터링을 활성화하려면 CloudALM 데이터 제공자에 대한 레지스트리 항목(Call ID)을 구성합니다.

## 개요

이 유틸리티 프로그램은 CloudALM 데이터 공급자 설정을 추가, 변경, 삭제 및 자동 생성하는 기능을 제공합니다.

* **트랜잭션 코드**: `/NEWR/GEN_DP_CALLID`
* **Program Name** \[프로그램 이름]: `/NEWR/GENERATE_CONFIG_CALLID`
* **설정 표**: `/NEWR/CALLID_CNF`

## 전제 조건

이 유틸리티를 실행하기 전에 데이터 공급자 `NR_CALM_LANDSCAPE` 이(가) 활성화되어 데이터를 수집하고 있는지 확인하십시오.

## 설정 절차

<Steps>
  <Step>
    ## 데이터 공급자 및 서비스 유형 매핑 구성

    1. 트랜잭션 `/NEWR/GEN_DP_CALLID`을 실행합니다.
    2. **DP/CALM Services Type Maintenance Link**(DP/CALM 서비스 유형 유지 관리 링크)를 클릭합니다.
    3. 모니터링할 서비스 유형을 선택합니다.
    4. 각 데이터 공급자를 SAP Cloud ALM에 구성된 적절한 SAP 시스템 또는 서비스에 연결합니다.
    5. 구성을 저장합니다.

    <Callout variant="tip">
      서비스 유형은 SAP Cloud ALM 환경에서 활성화된 SAP 시스템 또는 서비스를 나타냅니다.
    </Callout>
  </Step>

  <Step>
    ## 새 Call ID 레코드 추가

    1. 선택 화면에서 구성할 데이터 공급자 및 서비스 유형을 입력합니다.
    2. **Add new (Call ID) records to Registry** \[레지스트리에 새 (Call ID) 레코드 추가] 라디오 버튼을 선택합니다.
    3. **Execute** \[실행]을 클릭합니다.
    4. 시스템은 Cloud ALM에 구성된 모든 서비스 유형 인스턴스를 표시하며, 각각 고유한 LMS ID를 가집니다.
    5. 모니터링할 인스턴스를 선택합니다.
    6. 필요하지 않은 항목은 삭제하십시오.
    7. **Update and save configuration**을 클릭합니다.
  </Step>

  <Step>
    ## 설정을 확인합니다

    1. 프로세서 `/NEWR/REGISTRY` 로 이동합니다.
    2. 데이터 공급자를 선택합니다.
    3. 저장된 Call ID 항목을 확인하려면 **Configure** \[구성] 를 클릭합니다.
    4. 설정을 확인하려면 **Test** \[테스트]를 클릭하십시오.

    **예상 결과**: 연결 테스트에 성공 상태가 표시됩니다.
  </Step>
</Steps>

## 유지 관리 옵션

### 기존 항목 업데이트

기존 Call ID 레코드를 변경하거나 삭제해야 하는 경우:

1. 트랜잭션 `/NEWR/GEN_DP_CALLID`을 실행합니다.
2. **Copy/Change/Delete existing (Call ID) records in Registry** \[레지스트리에서 기존 (Call ID) 레코드 복사/변경/삭제]를 선택합니다.
3. **Execute** \[실행]을 클릭합니다.
4. 필요한 변경 사항을 적용하십시오.
5. 구성을 저장합니다.

### 모든 항목 지우기

모든 Call ID 레코드를 제거하려면(초기 설정 또는 서비스 팩 업그레이드 시):

1. 트랜잭션 `/NEWR/GEN_DP_CALLID`을 실행합니다.
2. **Backup &amp; delete all existing (Call ID) records in Registry** \[레지스트리의 모든 기존 (Call ID) 레코드 백업 및 삭제]를 선택합니다.
3. **Execute** \[실행]을 클릭합니다.

<Callout variant="important">
  삭제 전에 백업을 생성합니다. 초기 설정 중이거나 서비스 팩 업그레이드 중 지시가 있는 경우에만 이 옵션을 사용하십시오.
</Callout>

### 서비스 정의 새로 고침

서비스 유형 또는 LMS ID를 찾을 수 없는 경우:

1. **Retrieve new services (LMSID) setup in Cloud ALM** \[Cloud ALM에 설정된 새 서비스(LMSID) 검색을] 클릭합니다.
2. Cloud ALM 계정에 설정된 최신 서비스를 가져옵니다.
