SAP Cloud ALM 서비스 모니터링을 활성화하려면 CloudALM 데이터 제공자에 대한 레지스트리 항목(Call ID)을 구성합니다.

## 개요

이 유틸리티 프로그램은 CloudALM 데이터 공급자 설정을 추가, 변경, 삭제 및 자동 생성하는 기능을 제공합니다.

* <DNT>**Transaction Code**</DNT>: <DNT>**/NEWR/GEN\_DP\_CALLID**</DNT>
* <DNT>**Program Name**</DNT>: <DNT>**/NEWR/GENERATE\_CONFIG\_CALLID**</DNT>
* <DNT>**Configuration Table**</DNT>: <DNT>**/NEWR/CALLID\_CNF**</DNT>

## 전제 조건

이 유틸리티를 실행하기 전에 데이터 공급자 <DNT>**NR\_CALM\_LANDSCAPE**</DNT> 이(가) 활성화되어 데이터를 수집하고 있는지 확인하십시오.

## 설정 절차

<Steps>
  <Step>
    ## 데이터 공급자 및 서비스 유형 매핑 구성

    1. 트랜잭션 `/NEWR/GEN_DP_CALLID`을 실행합니다.
    2. <DNT>**DP/CALM Services Type Maintenance Link**</DNT>을 클릭합니다.
    3. 모니터링할 서비스 유형을 선택합니다.
    4. 각 데이터 공급자를 SAP Cloud ALM에 구성된 적절한 SAP 시스템 또는 서비스에 연결합니다.
    5. 구성을 저장합니다.

    <Callout variant="tip">
      <DNT>**Service type**</DNT> <DNT>**SAP Cloud ALM**</DNT> 환경에서 활성화된 <DNT>**SAP systems**</DNT> 또는 <DNT>**services**</DNT> 을 나타냅니다.
    </Callout>
  </Step>

  <Step>
    ## 새 Call ID 레코드 추가

    1. 선택 화면에서 구성할 데이터 공급자 및 서비스 유형을 입력합니다.
    2. <DNT>**Add new (Call ID) records to Registry**</DNT> 라디오 버튼을 선택합니다.
    3. <DNT>**Execute**</DNT> 을(를) 클릭합니다.
    4. 시스템은 <DNT>**Cloud ALM**</DNT>에 구성된 모든 <DNT>**service type instances**</DNT> 을(를) 표시하며, 각각 고유한 <DNT>**LMS ID**</DNT>을(를) 가집니다.
    5. 모니터링할 인스턴스를 선택합니다.
    6. 필요하지 않은 항목은 삭제하십시오.
    7. <DNT>**Update and save configuration**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    ## 설정을 확인합니다

    1. 프로세서 `/NEWR/REGISTRY` 로 이동합니다.
    2. <DNT>**data provider**</DNT> 을 선택합니다.
    3. 저장된 <DNT>**Call ID entries**</DNT>을(를) 확인하려면 <DNT>**Configure**</DNT> 을(를) 클릭합니다.
    4. 설정을 확인하려면 <DNT>**Test**</DNT> 를 클릭합니다.

    **예상 결과**: 연결 테스트에 성공 상태가 표시됩니다.
  </Step>
</Steps>

## 유지 관리 옵션

### 기존 항목 업데이트

기존 <DNT>**Call ID records**</DNT>을(를) 변경하거나 삭제해야 하는 경우:

1. 트랜잭션 `/NEWR/GEN_DP_CALLID`을 실행합니다.
2. <DNT>**Copy/Change/Delete existing (Call ID) records in Registry**</DNT> 선택합니다.
3. <DNT>**Execute**</DNT> 을(를) 클릭합니다.
4. 필요한 변경 사항을 적용하십시오.
5. 구성을 저장합니다.

### 모든 항목 지우기

모든 <DNT>**Call ID records**</DNT> 을(를) 제거하려면(초기 설정 또는 서비스 팩 업그레이드 시):

1. 트랜잭션 `/NEWR/GEN_DP_CALLID`을 실행합니다.
2. <DNT>**Backup &amp; delete all existing (Call ID) records in Registry**</DNT> 선택합니다.
3. <DNT>**Execute**</DNT> 을(를) 클릭합니다.

<Callout variant="important">
  삭제 전에 백업을 생성합니다. 초기 설정 중이거나 서비스 팩 업그레이드 중 지시가 있는 경우에만 이 옵션을 사용하십시오.
</Callout>

### 서비스 정의 새로 고침

<DNT>**service type**</DNT> 또는 <DNT>**LMS ID**</DNT>을(를) 찾을 수 없는 경우:

1. <DNT>**Retrieve new services (LMSID) setup in Cloud ALM**</DNT> 을(를) 클릭합니다.
2. <DNT>**Cloud ALM account**</DNT>에 구성된 최신 서비스를 가져옵니다.
