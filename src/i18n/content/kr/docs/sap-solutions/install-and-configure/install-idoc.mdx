<Steps>
  <Step>
    **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents** 로 이동하세요.
  </Step>

  <Step>
    **앱 및 시각화를** 선택하고 **IDOC Explorer를** 검색하세요.
  </Step>

  <Step>
    **액세스 관리를** 선택하세요. 앱에 접속할 계정을 선택하세요.

    <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />
  </Step>

  <Step>
    **업데이트를** 클릭하세요. IDOC를 볼 때 IDOC 탐색기가 표시됩니다.
  </Step>
</Steps>
