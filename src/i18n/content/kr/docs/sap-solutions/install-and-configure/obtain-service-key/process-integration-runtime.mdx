이 가이드는 SAP Business Technology 플랫폼(BTP)에서 프로세스 통합 런타임 인스턴스에 대한 서비스 키를 얻는 방법을 보여줍니다. SAP Process Integration에 대한 모니터링을 구성하려면 이 서비스 키가 필요합니다.

## 전제 조건

* 적절한 권한이 있는 <DNT>**BTP subaccount**</DNT> 에 대한 액세스
* <DNT>**Space**</DNT> 에 대한 <DNT>**BTP administrator**</DNT> 의 확인 및 <DNT>**Grant-types**</DNT>

## 프로세스 통합 런타임 인스턴스 생성

1. <DNT>**BTP (Business Technology Platform) subaccount**</DNT>에 로그인합니다.

2. <DNT>**BTP cockpit**</DNT>에서 <DNT>**Instances and Subscriptions**</DNT>를 클릭합니다.

3. 오른쪽 상단의 <DNT>**Create**</DNT> 를 클릭하세요.

4. 팝업 화면에서 다음을 선택하세요.

   * <DNT>**Service**</DNT>: <DNT>**SAP Process Integration Runtime**</DNT>

   * <DNT>**Plan**</DNT>: <DNT>**API**</DNT>

   * <DNT>**Runtime Environment**</DNT>: <DNT>**Cloud Foundry**</DNT>

   * <DNT>**Space**</DNT>: 귀하의 담당자에게 확인하세요 <DNT>**BTP administrator**</DNT>

   * <DNT>**Instance Name**</DNT>: 조직 명명 규칙을 따릅니다

     <img title="Process Integration Instance" alt="The Create Instance dialog showing Service, Plan, Runtime Environment, Space, and Instance Name fields" src="/images/process-integration-instance.webp" width="600" />

5. <DNT>**Next**</DNT>을(를) 클릭한 다음 선택합니다:

   * <DNT>**Roles**</DNT>: `MonitoringDataRead`

   * <DNT>**Grant-types**</DNT>: <DNT>**Default Client Credentials**</DNT> (다른 유형이 사용되는지 <DNT>**BTP administrator**</DNT> 및 보안 팀에 확인하십시오)

     <img title="BTP Roles and Grant Types" alt="The configuration screen showing Roles and Grant-types fields with MonitoringDataRead role selected" src="/images/btp-roles-grant-types.webp" width="600" />

6. <DNT>**Next**</DNT> 을(를) 클릭하여 세부 정보를 검토하세요.

7. <DNT>**Create**</DNT> 을(를) 클릭합니다.

## 서비스 키를 생성하고 다운로드합니다.

1. 서비스를 만든 후 <DNT>**Instances**</DNT> 탭에서 해당 서비스를 찾으세요.

   <img title="BTP Instance" alt="The Instances tab showing the created Process Integration Runtime instance" src="/images/btp-instance.webp" width="600" />

2. 새로 생성된 인스턴스의 행에 있는 줄임표(`...`)를 클릭한 다음 <DNT>**Create Service Key**</DNT>을(를) 선택합니다.

3. <DNT>**Service Key Name**</DNT> 를 입력하고 <DNT>**Key Type**</DNT>: `ClientId/Secret`를 선택합니다.

   <img title="BTP New Service Key" alt="The Create Service Key dialog showing Service Key Name and Key Type fields" src="/images/btp-new-service-key.webp" width="600" />

4. <DNT>**Create**</DNT> 을(를) 클릭합니다.

5. <DNT>**service key**</DNT> 행에 있는 줄임표(`...`)를 클릭한 다음 <DNT>**Download**</DNT>을(를) 선택합니다.

   <img title="BTP Download Service Key" alt="The service key row showing the ellipsis menu with the Download option" src="/images/btp-download.webp" width="600" />

<DNT>**service key file**</DNT> 은(는) <DNT>**JSON file**</DNT>(으)로 다운로드됩니다. 안전하게 보관하십시오. 모니터링 설정을 구성하려면 이 파일의 자격 증명이 필요합니다.
