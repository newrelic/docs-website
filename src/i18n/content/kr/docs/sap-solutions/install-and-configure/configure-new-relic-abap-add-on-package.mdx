## /NEWR/INSTALL 메뉴를 사용하십시오 [#menu-newr-install]

1. 중앙 모니터링 시스템의 프로덕션 클라이언트에 로그인합니다.
2. 명령 필드에 `/n` 을(를) 입력하고 <DNT>**Enter**</DNT>을(를) 눌러 <DNT>**SAP Easy Access root menu**</DNT> (으)로 이동합니다.
3. 명령 필드에 `/n/NEWR/INSTALL` 을(를) 입력하여 <DNT>**`/NEWR/INSTALL`**</DNT> 메뉴에 액세스합니다.

중앙 모니터링 시스템에 애드온을 설치한 후:

* 설치톨러가 설정을 수행할 수 있는 적절한 권한을 갖고 있는지 확인하세요.
* 인스톨러의 `profile/role` 에 있는 코드 인증 개체에 모듈, <DNT>**`/NEWR/*`**</DNT> 을(를) 추가합니다.

<Callout variant="tip">
  이전 버전에서 업그레이드하는 경우 [이전 버전에서 업그레이드](/docs/sap-solutions/upgrade/install-upgrade-package) 를 참조하여 필요한 단계를 확인하십시오.
</Callout>

## 시스템 연결 [#connect-systems]

<Callout variant="important">
  계속 진행하기 전에 보류 중인 설정 단계를 완료하십시오.
</Callout>

`/NEWR/INSTALL` 메뉴에서 사용 가능한 옵션은 다음과 같습니다:

### HTTP 및 RFC 대상 설정

이 옵션을 사용하여 트랜잭션 <DNT>**`SM59`**</DNT> 을 호출하고 [RFC](/docs/sap-solutions/install-and-configure/additional-configuration/#create-rfc-for-monitored-systems) 및 [HTTPS](/docs/sap-solutions/install-and-configure/additional-configuration/#set-http-destination-to-new-relic-cloud) 연결을 확인합니다.

### 원격 데이터베이스 연결 설정

* SAP 트랜잭션 <DNT>**DBACOCKPIT**</DNT> 을 사용하여 원격 HANA 데이터베이스 연결 설정을 확인합니다.
* 모니터링하려는 각 원격 HANA 데이터베이스에 대해 데이터베이스 연결이 생성되었는지 확인하세요.
* 자세한 지침은 [데이터베이스 연결 생성](/docs/sap-solutions/install-and-configure/additional-configuration/#create-database-connection)을 참조하십시오.

### NR API 엔드포인트에 대한 연결 구성

1. <DNT>**New Entries**</DNT> 을(를) 클릭합니다.

2. 뉴렐릭 계정 세부 정보를 입력하세요:

   * <DNT>**New Relic Account ID**</DNT> (선택적으로 SAP- 접두어가 붙을 수 있음)
   * <DNT>**API Ingest Key**</DNT>
   * <DNT>**Metric Endpoint**</DNT> ([네트워크 엔드포인트](/docs/sap-solutions/before-you-begin/#network-endpoints) 참조)
   * <DNT>**Event Endpoint**</DNT>
   * <DNT>**Log Endpoint**</DNT>
   * <DNT>**Trace Endpoint**</DNT>

3. <DNT>**Save**</DNT> 을(를) 클릭합니다.

<img title="Configure API endpoints" alt="A screenshot displaying the New Relic API endpoint configuration" src="/images/new-entry-1.webp" width="1000" />

### 클라이언트 000에서 템플릿 테이블 복사

<DNT>**client 000**</DNT> 에서 테이블을 복사하려면 실행 아이콘(시계 기호)을 클릭하여 이 옵션을 실행합니다. 성공 메시지가 나타나는지 확인하십시오.

### 모니터링 대상 시스템 정의

1. <DNT>**New Entries**</DNT> 을 클릭하여 <DNT>**SAP System ID**</DNT> 및 <DNT>**Client**</DNT> 번호를 입력합니다.
2. <DNT>**Data Provider Installation Information**</DNT>에서 RFC <DNT>**Destination**</DNT>을(를) 지정하고, SAP <DNT>**Software Component**</DNT>을(를) 선택하고, <DNT>**Database Vendor**</DNT>을(를) 입력하고, <DNT>**DB Connection Name**</DNT> 을(를) 제공합니다(HANA 데이터베이스만 해당).
3. <DNT>**Monitoring Group**</DNT>를 클릭하고 대화 상자에서 <DNT>**Yes**</DNT> 를 선택하여 확인합니다.
4. 필요한 모니터링 그룹을 선택하고, 생성 아이콘을 클릭하여 프로세스를 완료합니다.
5. <DNT>**BACK**</DNT> 또는 <DNT>**F3**</DNT> 키를 클릭하여 다음 단계로 이동합니다.

## 설정 생성 [#generate-configurations]

### 새 데이터 공급자에 대한 설정 생성

* 이 메뉴 옵션을 사용하면 새로운 릴리스나 패치에 대한 데이터 공급자를 생성할 수 있습니다. 새로 설치하는 경우 모든 데이터 공급자(DP)가 생성됩니다.
* 첫 번째 화면에서 릴리스 및 패치 세부 정보를 확인합니다. 빈 필드는 기존 설정을 나타냅니다.
* 재생성하려면 <DNT>**Maintain Monitored Systems**</DNT> 에서 <DNT>**Re-generate Data Provider Configurations**</DNT> 을(를) 선택하거나 <DNT>**Reset Installation Flag (For Re-install only)**</DNT> 을(를) 사용하여 설치 플래그를 재설정하고 설정을 재생성하십시오.

<img title="Maintain Monitored Systems" alt="A screenshot displaying the SAP maintain monitored systems" src="/images/maintain-monitored-systems.webp" width="500" />

### 설치 완료 확인

<DNT>**Verify Installation Completeness**</DNT> 을(를) 클릭하여 모든 것이 올바르게 구성되었는지 확인하세요. 항목에 빨간색 표시가 나타나면 <DNT>**Action Needed**</DNT> 열에서 \[실행] 버튼을 클릭하세요.

## 설정 확인 [#verify-configuration]

1. 데이터 공급자가 올바르게 구성되었는지 확인하려면 <DNT>**Data Provider Registry Maintenance**</DNT> 폴더를 엽니다.
2. <DNT>**Data Provider Registry**</DNT>을(를) 클릭하거나 트랜잭션 <DNT>**`/n/NEWR/REGISTRY`**</DNT>을(를) 사용합니다.
3. <DNT>**Data Provider Name**</DNT> 을(를) 선택하고 <DNT>**Configure**</DNT>을(를) 클릭하여 설정 파라미터를 확인합니다.
4. <DNT>**TestDP**</DNT>을(를) 클릭하여 데이터 제공자를 테스트합니다. 테스트가 성공적으로 완료되어, 데이터 공급자가 올바르게 구성되었음을 확인해야 합니다.
