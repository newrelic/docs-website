SAP용 뉴렐릭 모니터링에는 대시보드, 공지 정책 및 IDOC Explorer를 포함한 기본 제공 모니터링 구성 요소 목록이 포함되어 있습니다. 찾아서 설치하려면 아래 지침을 따릅니다.

## 시작하기 전에

이러한 작업을 수행하려면 계정에 대한 뉴렐릭 관리자 권한이 있어야 합니다.

<Callout variant="important">
  <DNT>**SAP Cockpit**</DNT> 대시보드의 경우, 설치 전에 `NR_LKUP_SYSTEM` 이라는 이름의 [룩업 테이블을 생성](#create-lookup-table) 해야 합니다.
</Callout>

## 대시보드 설치

### 룩업 테이블 생성 [#create-lookup-table]

<Callout variant="tip">
  <DNT>**SAP Cockpit**</DNT> 대시보드에만 필요합니다.
</Callout>

<DNT>**SAP Cockpit**</DNT> 대시보드는 <DNT>**SAP systems**</DNT> 및 해당 환경에 대해 <DNT>**lookup table**</DNT> 을(를) 사용합니다.

1. 두 개의 열: `SYS_ID` 및 `Env`로 <DNT>**lookup table**</DNT> 을(를) 생성하고 <DNT>**CSV file**</DNT>(으)로 저장합니다. 다음은 Lookup 파일의 샘플입니다:

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. <DNT>**Log &gt; Lookup Tables**</DNT> 으)로 이동합니다.

3. <DNT>**Add a table**</DNT> 을(를) 클릭합니다.

4. <DNT>**Add a lookup table**</DNT> 사이드바에서 테이블 이름을 <DNT>**NR\_LKUP\_SYSTEM**</DNT>(으)로 지정하고 <DNT>**CSV file**</DNT>을(를) 업로드합니다. 자세한 내용은 [룩업 테이블](/docs/logs/ui-data/lookup-tables-ui/) 을 참조하세요.

5. 테이블 내용을 확인하려면 `FROM lookup(NR_LKUP_SYSTEM) SELECT *`와 함께 <DNT>**query builder**</DNT> 을 사용한 다음 <DNT>**Run**</DNT>를 클릭합니다.

### 대시보드 설치 [#install-dashboards]

1. <DNT>**[one.newrelic.com](https://one.newrelic.com)**</DNT> &gt; <DNT>**Integrations &amp; Agents**</DNT>(으)로 이동하고, <DNT>**New Relic Monitoring for SAP Solutions**</DNT> 을(를) 검색하여 모든 대시보드를 자동으로 다운로드하세요.

2. 또는 <DNT>**Dashboards &gt; Recommended Dashboards**</DNT>으로 이동하여 <DNT>**View all**</DNT>을 클릭하고 SAP를 검색할 수 있습니다. 자세한 내용은 [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 를 참조하세요.

3. 이 대시보드를 설치하세요:

   * <DNT>**SAP Dashboards**</DNT>
   * <DNT>**SAP Cockpit**</DNT>
   * <DNT>**SAP Data Collection Monitoring**</DNT>
   * <DNT>**SAP Data Ingestion Estimate**</DNT>

### 대시보드 필터 추가 [#add-dashboard-filter]

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards**</DNT> 으)로 이동합니다.

2. 필터링을 위해 패싯 링크 활성화:

   * 위젯의 오른쪽 상단 모서리에 있는 줄임표(<DNT>**…**</DNT>)를 클릭합니다.
   * 편집 화면을 열려면 <DNT>**Edit**</DNT> 을(를) 선택합니다.
   * <DNT>**Filter the current dashboard**</DNT> 스위치를 전환합니다.

3. <DNT>**Facet linking**</DNT> <DNT>**SAP system**</DNT>, <DNT>**instance**</DNT>, <DNT>**IDOC status**</DNT>, <DNT>**jobs**</DNT> 등으로 필터링하도록 테이블, 원형 차트 및 바 차트 위젯에 대해 활성화할 수 있습니다

## 공지 정책 설치

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies**</DNT> 으)로 이동합니다.

2. <DNT>**New Relic Monitoring for SAP Solutions alert policy**</DNT>을(를) 검색하고 클릭합니다.

   <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />

3. <DNT>**Alert condition**</DNT> 탭에서 계정을 선택하고 <DNT>**alert policy**</DNT> 을(를) <DNT>**Enable**</DNT>(으)로 토글합니다. 정책을 관리하는 방법을 알아보려면 [정책 관리](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) 를 참조하십시오.

4. 돌아가서 <DNT>**Nr4sap default alert policy**</DNT>를 검색합니다. 활성화하려는 <DNT>**alert condition**</DNT> 을(를) 켜기로 전환합니다.

## IDOC 탐색기 설치

1. <DNT>**[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents**</DNT> 으)로 이동합니다.

2. <DNT>**Apps &amp; Visualizations**</DNT> 선택하고 <DNT>**IDOC Explorer**</DNT> 검색합니다.

3. <DNT>**Manage access**</DNT>을 선택합니다. 앱에 액세스할 계정을 선택합니다.

   <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />

4. <DNT>**Update**</DNT>을 클릭합니다. <DNT>**IDOC entities**</DNT>을(를) 볼 때 <DNT>**IDOC Explorer**</DNT> 이(가) 표시됩니다.
