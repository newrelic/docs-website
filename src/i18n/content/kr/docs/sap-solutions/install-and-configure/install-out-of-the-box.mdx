SAP용 뉴렐릭 모니터링에는 기본 제공 구성 요소 목록이 포함되어 있습니다. 아래 지침에 따라 해당 파일을 찾아 설치하세요. 이러한 작업을 수행하려면 해당 계정에 대한 관리자 권한이 있어야 합니다.

## 대시보드 설치

1. **[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards** 로 이동합니다.
2. 모든 SAP 대시보드를 보려면 **추천 Dashboards** 섹션에서 **모두 보기를** 클릭하세요. 맞춤형 대시보드 기능에 대한 소개는 [Dashboards](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 참조하세요.

<Callout variant="important">
  **통합 및 에이전트** 로 이동하여 **뉴렐릭 모니터링 for SAP Solutions를** 검색하여 모든 대시보드를 자동으로 다운로드할 수도 있습니다.
</Callout>

3. SAP를 검색하여 다음 대시보드를 설치하세요.

   * **SAP Dashboards**
   * **SAP 콕핏**
   * **SAP 데이터 수집 모니터링**
   * **SAP 데이터 수집 추정**

<Callout variant="important">
  **SAP Cockpit** 대시보드의 경우 이름이 `NR_LKUP_SYSTEM` 인 [룩업 테이블을 생성](#create-lookup-table) 해야 합니다.
</Callout>

## 대시보드 필터 추가

1. **[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards** 로 이동합니다.

2. 필터링을 위해 패싯 링크 활성화:

   * 위젯의 오른쪽 상단에 있는 줄임표(…)를 클릭하세요.
   * **편집을** 선택하면 편집 화면이 열립니다.
   * **현재 대시보드 필터링** 스위치를 전환합니다.

3. SAP 시스템, 제외, IDOC 상태, 작업 등을 기준으로 필터링하기 위해 테이블, 원형 차트 및 바 차트에 대해 패싯 연결을 활성화할 수 있습니다.

## 룩업테이블 생성(SAP 조종석만 해당) [#create-lookup-table]

**SAP Cockpit** 대시보드는 SAP 시스템 및 해당 환경에 대한 룩업 테이블을 사용합니다.

1. `SYS_ID` 및 `Env` 두 개의 열로 룩업 테이블을 만들고 CSV 파일로 저장합니다. 다음은 조회 파일의 샘플입니다.

   * SYS\_ID 환경
   * GD1 개발자
   * SD2 개발자
   * AQ1 테스트
   * AQ2 테스트
   * SP1 제품
   * SP2 제품

2. 메뉴에서 **Log &gt; Lookup Table** 로 이동하세요.

3. **테이블 추가를** 클릭합니다.

4. [룩업 테이블 추가](/docs/logs/ui-data/lookup-tables-ui/) 사이드바에서 테이블 이름을 **NR\_LKUP\_SYSTEM** 으로 지정하고 CSV 파일을 업로드합니다.

5. 테이블 내용을 확인하려면 `FROM lookup(NR_LKUP_SYSTEM) SELECT *` 과 함께 쿼리를 사용한 다음 **실행을** 클릭합니다.
