SAP용 뉴렐릭 모니터링에는 대시보드, 공지 정책 및 IDOC Explorer를 포함한 기본 제공 모니터링 구성 요소 목록이 포함되어 있습니다. 찾아서 설치하려면 아래 지침을 따릅니다.

## 시작하기 전에

이러한 작업을 수행하려면 계정에 대한 뉴렐릭 관리자 권한이 있어야 합니다.

<Callout variant="important">
  **SAP Cockpit** 대시보드의 경우, 설치 전에 이름이 `NR_LKUP_SYSTEM` 인 [룩업 테이블을 생성](#create-lookup-table) 해야 합니다.
</Callout>

## 대시보드 설치

### 룩업 테이블 생성 [#create-lookup-table]

<Callout variant="tip">
  **SAP Cockpit** 대시보드에만 필요합니다.
</Callout>

**SAP Cockpit** 대시보드는 SAP 시스템 및 해당 환경에 대한 룩업 테이블을 사용합니다.

1. `SYS_ID` 및 `Env` 두 개의 열로 룩업 테이블을 만들고 CSV 파일로 저장합니다. 다음은 조회 파일의 샘플입니다.

   ```csv
   SYS_ID,Env
   GD1,DEV
   SD2,DEV
   AQ1,Testing
   AQ2,Testing
   SP1,Prod
   SP2,Prod
   ```

2. **Log &gt; Lookup Tables**블로 이동합니다.

3. **테이블 추가를** 클릭합니다.

4. **Add a lookup table** \[룩업 테이블 추가] 사이드바에서 테이블 이름을 **NR\_LKUP\_SYSTEM**으로 지정하고 CSV 파일을 업로드합니다. 자세한 내용은 [룩업 테이블](/docs/logs/ui-data/lookup-tables-ui/) 을 참조하세요.

5. 테이블 내용을 확인하려면 `FROM lookup(NR_LKUP_SYSTEM) SELECT *` 과 함께 쿼리를 사용한 다음 **실행을** 클릭합니다.

### 대시보드 설치 [#install-dashboards]

1. **[one.newrelic.com](https://one.newrelic.com)** &gt; **Integrations &amp; Agents**로 이동하고, **New Relic Monitoring for SAP Solutions** \[SAP 솔루션용 뉴렐릭 모니터링] 을 검색하여 모든 대시보드를 자동으로 다운로드하세요.

2. 또는 **Dashboards &gt; Recommended Dashboards** \[권장 대시보드]로 이동하여 **View all** \[모두 보기]를 클릭하고 SAP를 검색할 수 있습니다. 자세한 내용은 [Dashboards를](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 참조하세요.

3. 이 대시보드를 설치하세요:

   * **SAP Dashboards**
   * **SAP 콕핏**
   * **SAP 데이터 수집 모니터링**
   * **SAP 데이터 수집 추정**

### 대시보드 필터 추가 [#add-dashboard-filter]

1. **[one.newrelic.com](https://one.newrelic.com) &gt; Dashboards** 로 이동합니다.

2. 필터링을 위해 패싯 링크 활성화:

   * 위젯의 오른쪽 상단 모서리에 있는 줄임표(**…**)를 클릭합니다.
   * **Edit** \[편집]을 선택하여 편집 화면을 엽니다.
   * **Filter the current dashboard** \[현재 대시보드 필터링] 스위치를 전환합니다.

3. 테이블, 원형 차트 및 바 차트 위젯에 대해 패싯 연결을 활성화하여 SAP 시스템, 인스턴스, IDOC 상태, 작업 등으로 필터링할 수 있습니다.

## 공지 정책 설치

1. **[one.newrelic.com](https://one.newrelic.com) &gt; Alerts &gt; Alert Policies** 로 이동합니다.

2. **SAP Solutions 공지를** 검색하고 클릭하세요.

   <img title="New Relic Monitoring for SAP Solutions alert policy" alt="A screenshot displaying the connection setup for New Relic Monitoring for SAP Solutions alert policy" src="/images/new-relic-monitoring-for-sap-solutions-alert-policy.webp" width="" />

3. **공지 조건** 탭에서 계정을 선택하고 공지를 **활성화** 로 전환합니다. 정책을 관리하는 방법을 알아보려면 [정책 관리를](/docs/alerts/organize-alerts/create-edit-or-find-alert-policy/) 참조하세요.

4. 뒤로 돌아가서 **Nr4sap default 공지 정책**을 검색합니다. 활성화할 공지 조건의 토글을 켭니다.

## IDOC 탐색기 설치

1. **[one.newrelic.com](https://one.newrelic.com) &gt; Integrations &amp; Agents** 로 이동하세요.

2. **Apps &amp; Visualizations** \[앱 및 시각화]를 선택하고 **IDOC Explorer**를 검색합니다.

3. **액세스 관리를** 선택하세요. 앱에 접속할 계정을 선택하세요.

   <img title="IDOC Explorer" alt="A screenshot displaying the manage access settings for IDOC Explorer" src="/images/idoc-explorer.webp" />

4. **업데이트를** 클릭하세요. IDOC를 볼 때 IDOC 탐색기가 표시됩니다.
