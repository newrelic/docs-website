<Steps>
  <Step>
    **All entities &gt; Workloads &gt; + Create a workload** 로 이동하세요. [독소를 만드는 방법에 대한 자세한 내용은 독소 만들기 및 정의를](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/) 참조하세요.
  </Step>

  <Step>
    증기의 이름을 입력하고 드롭다운 목록 상자에서 필요한 계정을 선택하세요.
  </Step>

  <Step>
    **학습 유형 = 모두** 옵션 옆에 있는 **+** 아이콘을 클릭하고 필터 막대에 조건 `sap.system = {SAP System SID}` 입력하세요.

    <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />
  </Step>

  <Step>
    **+ 이 쿼리 추가를** 클릭하여 워크로드 정의에 쿼리를 추가합니다.
  </Step>

  <Step>
    **워크로드 만들기를** 클릭하여 워크로드를 만듭니다.
  </Step>

  <Step>
    워크로드는 **모든 분리** UI 에 표시되며 SAP 시스템과 함께 모든 분리 태그를 포함합니다.
  </Step>
</Steps>
