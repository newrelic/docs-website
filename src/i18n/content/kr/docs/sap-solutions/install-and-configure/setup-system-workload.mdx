## 시스템 워크로드 설정

1. **All entities &gt; Workloads &gt; + Create a workload**로 이동하세요. 자세한 내용은 [워크로드 생성 및 정의](/docs/new-relic-solutions/new-relic-one/workloads/create-workloads/)를 참조하십시오.

2. 워크로드 이름을 입력하고 드롭다운 메뉴 목록에서 계정을 선택합니다.

3. **Entity Type=All** 옆에 있는 **+** 아이콘을 선택하고 필터 표시줄에 `sap.system = {SAP System SID}` 을(를) 입력합니다.

   <img title="SAP System" alt="A screenshot displaying the SAP System filter" src="/images/sap-system.webp" />

4. 워크로드 정의에 쿼리를 추가하려면 **+ Add this query** 를 선택합니다.

5. **Create a workload** \[워크로드 생성]을 선택합니다.

새 워크로드는 **All entities** 에 표시되며 SAP 시스템으로 태그된 모든 엔티티를 포함합니다.

## 서비스 팩 설치

서비스 팩은 `K-<Version Number>SP<SP Number>IN<Software Component>.SAR` 명명 규칙을 사용하여 `.SAR` 파일로 제공됩니다.

1. 트랜잭션 `SPAM` 사용하여 서비스 팩을 설치하세요.
2. 클라이언트 000에서 새로운 설정 템플릿을 로드합니다.
3. 클라이언트 000에서 **Install New Release and Patch** &gt; **Copy Template Tables**로 이동합니다.
4. **Install New Release and Patch** &gt; **Generate Configuration for New Data Providers**의 단계에 따라 새로운 데이터 공급자 구성을 생성합니다.
5. **Install New Release and Patch** &gt; **Verify Installation Completeness** 를 선택하여 설치를 확인합니다.
