<DNT>**NRM4SAP**</DNT> 의 ABAP 구성요소는 SAP 애드온으로 패키징되어 있으며 <DNT>**client 000**</DNT>에서 트랜잭션 <DNT>**SAINT**</DNT> 을(를) 사용하여 설치할 수 있습니다.

## 애드온 설치 파일

* **초기 설치**: 로컬 PC의 폴더에 애드온 파일 `K-xxxAOIINNEWR.SAR` 을(를) 다운로드합니다.
* **이전 버전에서 업그레이드**: 다음 이름의 파일을 사용합니다 `K-xxxAOUINNEWR.SAP`

## SAP 추가 기능 파일 설치

1. -에 로그인 <DNT>**client 000**</DNT>
2. <DNT>**SAP GUI**</DNT> 명령 필드에 `SAINT` 을(를) 입력합니다.
3. 로 이동: <DNT>**Installation Package**</DNT> &gt; <DNT>**Load Packages**</DNT>
4. 설치 파일이 있는 위치에 따라 적절한 방법을 선택하십시오:

**어떤 방법을 사용해야 하나요?**

* **프런트엔드의 SAR 아카이브** (가장 일반적): 설치 파일이 로컬 컴퓨터에 있습니다.
* **애플리케이션 서버의 SAR 아카이브**: SAR 파일이 다음 위치에 사전 업로드되었습니다: `/usr/sap/trans/download/`
* **애플리케이션 서버의 EPS 파일**: 먼저 서버에서 SAR 파일을 PAT 파일로 추출해야 합니다.

## 설치를 완료합니다.

1. <DNT>**Start**</DNT>를 클릭하고 <DNT>**Installable Add-on packages for your system**</DNT>를 선택한 다음 클릭 <DNT>**Continue**</DNT>

2. <DNT>**Do you want to add Modification Adjustment Transports to the queue?**</DNT>을(를) 질문받은 경우:

   * 표준 설치(기본)를 하려면 <DNT>**No**</DNT> 을(를) 클릭합니다.
   * 패키지에 특정 지침이 있는 조정 전송이 포함된 경우에만 <DNT>**Yes**</DNT> 을(를) 클릭하세요.

   <img title="Do you want to add Modification Adjustment Transports to the queue?" alt="A screenshot displaying question about Modification Adjustment Transports" src="/images/modify-adjustement-transports.webp" width="80%" />

3. 계속하려면 녹색 체크 버튼을 클릭합니다.

   <Callout variant="important">
     <DNT>**Errors during Disassembling OCS Packages**</DNT> 페이지가 나타나면 <DNT>**Ignore**</DNT>을 클릭합니다. 이 오류는 특정 패키지 유형에서 예상되며 설치에 영향을 미치지 않습니다.
   </Callout>

4. 패키지가 설치되면, 클릭하십시오 <DNT>**Finish**</DNT>

5. <DNT>**Sending the runtime analysis**</DNT> 팝업이 표시됩니다. 다음 중 하나를 선택합니다:

   * <DNT>**Send**</DNT>: 로컬 브라우저 창에서 런타임 분석 데이터가 포함된 피드백 양식을 엽니다
   * <DNT>**Display**</DNT>: 런타임 분석을 먼저 표시한 다음 피드백 양식을 엽니다.
   * <DNT>**Do not send**</DNT>: 런타임 분석 데이터를 전송하지 않고 대화 상자를 닫습니다

6. <DNT>**Installed Components**</DNT> 탭에서 설치를 확인하세요. 뉴렐릭 애드온은 성공적인 설치를 나타내는 상태로 나열되어야 합니다.

7. 다음에서 로그아웃 <DNT>**Client 000**</DNT>
