이 가이드에서는 SAP 중앙 시스템에서 <DNT>**New Relic ABAP component (NEWR)**</DNT> 을(를) 제거하는 방법을 설명합니다. 모니터링을 폐기할 때나 새 버전으로 업그레이드하기 전에 이 작업을 수행해야 할 수 있습니다.

<Callout variant="caution">
  <DNT>**New Relic ABAP component**</DNT> 을(를) 제거하면 모든 모니터링 데이터 수집이 중지되고 SAP 시스템에서 해당 구성 요소가 제거됩니다. 뉴렐릭의 이력 데이터는 보존되지만, 새로운 데이터는 수집되지 않습니다. 이 작업은 제거 과정에서 되돌릴 수 없습니다.
</Callout>

## 전제 조건

* 에 대한 관리자 액세스 <DNT>**SAP system**</DNT>
* 액세스 <DNT>**client 000**</DNT>
* 모든 커스텀 설정 백업(향후 재설치를 위해 필요한 경우)

## ABAP 구성 요소 제거

중앙 시스템에서 ABAP 구성 요소를 제거하려면:

1. <DNT>**client 000**</DNT>에 로그인합니다.
2. 트랜잭션 코드 `SAINT`을 입력하세요.
3. <DNT>**Uninstallable Components**</DNT> 탭을 클릭합니다.

<img title="ABAP component on central system" alt="The SAINT transaction showing the NEWR component in the Uninstallable Components list" src="/images/ABAP-component-on-cms.webp" width="700" />

4. <DNT>**NEWR**</DNT> 구성 요소를 선택하고 버전 번호가 설치된 버전과 일치하는지 확인합니다.
5. <DNT>**Start**</DNT> 을(를) 클릭합니다.
6. 이후의 각 화면에서 확인하려면 녹색 체크 표시를 클릭하세요.
7. 프로세스가 완료되면 <DNT>**Finish**</DNT> 를 클릭합니다.
8. <DNT>**Installed Components**</DNT> 탭에 <DNT>**NEWR**</DNT> 구성 요소가 더 이상 나타나지 않는지 확인합니다.

## 제거 확인

제거 프로세스를 완료한 후:

* <DNT>**NEWR**</DNT> 컴포넌트가 더 이상 <DNT>**Installed Components**</DNT> 탭에 나타나지 않아야 합니다.
* 트랜잭션 `SM37` 의 데이터 수집 작업이 더 이상 실행되지 않습니다.
* 뉴렐릭은 이 시스템에서 모니터링 데이터 수신을 중단합니다.

**참고**: 뉴렐릭의 과거 모니터링 데이터는 계속 액세스할 수 있습니다. SAP 시스템에서 데이터 수집 구성 요소만 제거됩니다.
