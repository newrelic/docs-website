SAP 솔루션용 뉴렐릭 모니터링은 SAP 텔레메트리 데이터(메트릭, 이벤트, 로그 및 트레이스)를 수집하여 분석 및 시각화를 위해 뉴렐릭 플랫폼으로 전송합니다.

## 중앙 모니터링 시스템 vs. 모니터 시스템

텔레메트리 데이터를 제공하는 SAP 시스템은 모니터 시스템이라고 합니다. 중앙 모니터링 시스템은 <DNT>**NRM4SAP**</DNT> 이(가) 설치되는 곳으로, 텔레메트리 데이터를 수집, 변환 및 전송합니다. 일반적으로 <DNT>**NRM4SAP**</DNT> 을(를) <DNT>**SAP Solution Manager (SolMon)**</DNT>와 같은 단일 SAP 시스템에 설치합니다. <DNT>**NRM4SAP**</DNT> 은(는) <DNT>**ST\_PI**</DNT>, <DNT>**CCMS**</DNT> 및 성능 데이터베이스와 같은 표준 SAP 데이터 소스를 활용하므로 모니터되는 시스템에 추가 에이전트를 설치할 필요가 없습니다.

## 데이터 흐름 아키텍처

<DNT>**RFC**</DNT> 호출은 SAP 텔레메트리 데이터를 중앙 모니터링 시스템으로 가져옵니다. <DNT>**ABAP Enterprise Connector**</DNT> 은(는) 데이터를 처리하고 <DNT>**HTTPS**</DNT>을(를) 통해 뉴렐릭으로 안전하게 전송합니다.

<img title="Data Flow Architecture" alt="A diagram illustrating the data flow architecture" src="/images/SAP-Private-Cloud.webp" width="80%" />

## 포함 사항

설치 후 즉시 사용 가능한 대시보드와 알림을 사용할 수 있습니다:

* **관리 가시성**: 중앙 집중식 대시보드에서 다른 IT 시스템과 함께 SAP 시스템 상태 및 성능을 확인하세요.

* **커스텀 모니터링**: 주문 처리 워크플로우의 통합 지점 추적과 같은 특정 비즈니스 요구 사항에 맞게 모니터링을 맞춤 설정합니다. 수집된 텔레메트리 데이터를 사용하여 커스텀 대시보드 및 시각화를 구축하세요.

* **사전 예방적 알림**: 서비스 중단에 대한 즉각적인 공지를 수신하도록 알림을 설정하여, 연속성을 유지하고 문제를 신속하게 해결할 수 있습니다.
