이전 버전의 뉴렐릭 모니터링 for SAP Solutions에서 업그레이드하려면, 현재 설치된 버전과 일치하는 업그레이드 패키지를 사용하십시오.

## 전제 조건

* <DNT>**Administrator access**</DNT> 에게 <DNT>**SAP system**</DNT>
* 현재 버전이 식별되었습니다. [현재 버전 식별 섹션](#identify-current-version)을 참조하세요.
* 기존 설정 백업
* 설치된 버전에 맞는 올바른 <DNT>**upgrade package**</DNT>

## 현재 버전 확인 [#identify-current-version]

1. <DNT>**System &gt; Status**</DNT> 으로 이동합니다.
2. <DNT>**NEWR**</DNT> 을 검색하여 <DNT>**product version**</DNT>을 식별합니다.
3. <DNT>**client 000**</DNT>에서 트랜잭션 `SAINT` 을 실행하여 <DNT>**ABAP add-on**</DNT>을 업데이트합니다.

## 백업 및 설정 업데이트 [#backup-and-update-configuration]

<Callout variant="tip">
  업그레이드 중 덮어쓰기를 방지하려면 수정된 <DNT>**data provider configurations**</DNT> 또는 <DNT>**custom data providers**</DNT> 을(를) <DNT>**configuration template**</DNT> 에 저장하세요. <DNT>**V3.0 ABAP Add-on**</DNT> 이(가) 설치되어 있지 않더라도 <DNT>**client 000**</DNT>에서 새 템플릿을 복사하기 전에 템플릿이 저장되었는지 확인하세요.
</Callout>

1. `/n/NEWR/INSTALL` 메뉴에 액세스하여 <DNT>**Update or Save Configuration Template**</DNT>(으)로 이동합니다.

2. <DNT>**Java monitoring**</DNT>와 관련된 <DNT>**HDB monitoring template**</DNT> 을 제거하려면 `/NEWR/REP_DELETE_INS_TEMPLATE` 을 실행합니다.

3. `DBACockpit`를 사용하여 <DNT>**ABAP data connection**</DNT> 를 <DNT>**HANA database**</DNT> 에 설정합니다.

4. <DNT>**client 000**</DNT>에서 <DNT>**configuration template**</DNT> 을(를) 복사합니다.

5. <DNT>**monitoring system configuration**</DNT>을(를) 수정합니다:

   * 새 <DNT>**DB connection**</DNT>을(를) 추가합니다.
   * <DNT>**Java Enterprise Connector**</DNT> (예: <DNT>**NR\_EI\_AGENT**</DNT>)에 대한 참조를 제거합니다.
   * <DNT>**monitoring groups**</DNT>에 <DNT>**HANA\_DB**</DNT> 을(를) 추가합니다.
   * 설정을 업데이트하려면 <DNT>**data provider configurations**</DNT> 을(를) 다시 생성하십시오.
