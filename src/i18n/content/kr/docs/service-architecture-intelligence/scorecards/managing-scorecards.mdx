스코어카드는 정의된 기준의 준수 여부를 전체적으로 평가하는 다양한 규칙을 담은 컨테이너 역할을 합니다. 스코어카드는 규칙을 사용하여 다양한 표본과 팀의 규정 준수를 모니터링하고, 개선 영역을 식별하며, 내부 및 외부 요구 사항에 대한 일치성을 보장합니다. 이 문서에서는 뉴렐릭에서 스코어카드와 해당 규칙을 만들고 관리하는 방법에 대한 지침을 제공합니다.

## 새로운 스코어카드 만들기 [#create-scorecard]

사용 가능한 템플릿 중 하나를 사용하거나 나만의 스코어카드를 만들고 요구 사항에 맞게 규칙을 추가할 수 있습니다. 자세한 내용은 다음을 참조하세요.

* [템플릿을 사용하여 스코어카드 만들기](#create-scorecard-using-templates)
* [사용자 정의 스코어카드 만들기](#create-custom-scorecard)

<Callout variant="tip" title="팁">
  NerdGraph 뮤테이션을 사용하여 스코어카드를 만들거나 관리하세요. 자세한 내용은 [스코어카드 튜토리얼을](/docs/apis/nerdgraph/examples/nerdgraph-scorecards-tutorial) 참조하세요.
</Callout>

**필수 조건:**

* 이 스코어카드를 생성하려면 활성화된 뉴렐릭 계정 ID가 필요합니다.

### 템플릿을 사용하여 스코어카드 만들기 [#create-scorecard-using-templates]

**템플릿을 사용하여 스코어카드를 만들려면:**

1. New Relic 계정에 로그인하세요.

2. **왼쪽 탐색 창 &gt; All Capabilities &gt; Scorecards &gt; Create Scorecard** 이동합니다.

3. **Create a Scorecard** \[스코어카드 만들기] 화면에서 **Use a template** \[템플릿 사용을] 클릭합니다.

4. **Choose a template** \[템플릿 선택] 화면에서 목록에서 스코어카드 템플릿을 선택하고 **Next** \[다음을] 클릭합니다.

5. **Manage the rules** \[규칙 관리] 화면에서:

   * 필요한 규칙을 활성화합니다.
   * **Run this rule in** \[이 규칙을 실행할] 드롭다운 목록에서 이 규칙을 실행할 계정을 선택합니다.
   * **Next**클릭합니다.

6. 필요한 경우 **Review your Scorecard details** \[스코어카드 세부 정보 검토] 화면에서 스코어카드의 이름과 설명을 업데이트합니다.
   * **Confirm &amp; Create** \[확인 및 생성을] 클릭하세요.

7. 스코어카드를 만든 후 필요에 따라 추가 규칙을 추가할 수도 있습니다. 자세한 내용은 [스코어카드 규칙 만들기를](#create-scorecard-rules) 참조하세요.

### 사용자 정의 스코어카드 만들기 [#create-custom-scorecard]

**나만의 스코어카드를 만들려면:**

1. New Relic 계정에 로그인하세요.
2. **왼쪽 탐색 창 &gt; All Capabilities &gt; Scorecards &gt; Create Scorecard** 이동합니다.
3. **Create a Scorecard** \[점수표 만들기] 화면에서 **Build your own** \[직접 만들기]를 클릭합니다.
4. 이름과 설명을 입력한 후 **Create** \[만들기]를 클릭합니다.
5. 스코어카드를 만든 후에는 특정 준수 기준을 정의하기 위한 규칙을 추가하고 구성합니다. 자세한 내용은 [스코어카드 규칙 만들기를](#create-scorecard-rules) 참조하세요.

## 스코어카드 규칙 만들기 [#create-scorecard-rules]

스코어카드의 규칙은 특정 준수 기준에 따라 한국을 평가하는 데 사용됩니다. 서비스, 애플리케이션 또는 기타 리소스가 특정 요구 사항을 준수하는지 확인하는 프로세스를 자동화하는 규칙을 정의할 수 있습니다.

NRQL 쿼리를 사용하여 스코어카드에 대한 규칙을 만들 수 있습니다. 각 규칙은 여러 지식에 걸쳐 적용될 수 있으며, 각 지식이 특정 조건을 충족하는지 여부를 참 또는 거짓으로 평가하여 검증합니다.

**필수 조건:**

* 이 규칙을 생성하려면 활성화된 뉴렐릭 계정 ID가 있어야 합니다.
* 스코어카드가 생성되었습니다. 자세한 내용은 [새 스코어카드 만들기](#create-scorecard) 를 참조하세요.

<Callout variant="tip" title="팁">
  NerdGraph 뮤테이션을 사용하여 스코어카드 규칙을 만들거나 관리하세요. 자세한 내용은 [스코어카드 튜토리얼을](/docs/apis/nerdgraph/examples/nerdgraph-scorecards-tutorial) 참조하세요.
</Callout>

**규칙을 만들려면:**

1. New Relic 계정에 로그인하세요.

2. **왼쪽 탐색 창 &gt; All Capabilities &gt; Scorecards** 로 이동합니다.

3. 필요한 점수카드를 선택하세요.

4. **+ Create Rule** \[+ 규칙 만들기를] 클릭합니다.

5. 규칙을 평가할 뉴렐릭 계정을 선택하세요. 이렇게 하면 규칙이 올바른 데이터 집합에 적용되는지 확인할 수 있습니다.

6. 조건을 지정하는 NRQL 쿼리를 입력하세요. 귀하의 쿼리에는 다음 두 필드가 포함되어야 합니다.

   * 예를 들어 `entityGUID` 에 대한 `FACET` 은 `FACET id` 와 같습니다. `entityGuid`

   * `0` 또는 `1` 에 대한 점수를 평가하는 점수 열입니다. 예를 들어, `SELECT if(latest(reporting), 1, 0) as 'score'`.

     NRQL 예시:

     ```sql
     SELECT latest(if(tags.language = 'ruby', 0, 1)) as 'score' 
     FROM Entity 
     WHERE tags.environment = 'production' and type = 'APM-APPLICATION' 
     SINCE 1 day ago 
     FACET id, tags.nr.team as 'team'
     ```

7. 쿼리를 검증하려면 &apos;이 쿼리 실행&apos;을 클릭하세요. 성공적으로 실행하면 미리보기 응답이 나타납니다.

   <Callout variant="tip" title="팁">
     기존 쿼리 사용 드롭다운 목록에서 규칙에서 실행할 이전에 만든 쿼리를 선택합니다.
   </Callout>

8. **Next**클릭합니다.

9. 규칙의 이름과 설명을 입력하세요.

10. **Save and create** \[저장 및 만들기를] 클릭하세요.

## 규칙 업데이트 [#update-rule]

기존 스코어카드 규칙을 활성화하거나 비활성화할 수도 있습니다.

**규칙을 업데이트하려면:**

1. New Relic 계정에 로그인하세요.
2. **왼쪽 탐색 창 &gt; All Capabilities &gt; Scorecards** 로 이동합니다.
3. 필요한 점수카드를 선택하세요.
4. 오른쪽 상단 모서리에 있는 <DNT>more icon</DNT> 클릭하고 **Activate and deactivate rules** \[규칙 활성화 및 비활성화를] 선택합니다.
5. 필요한 규칙을 활성화하거나 비활성화하고 **Save** \[저장]을 클릭합니다.

## 스코어카드 모범 사례 [#scorecards-best-practices]

* 전략적 목표에 부합하는지 확인하기 위해 정기적으로 스코어카드를 검토합니다.
* 이력 데이터를 사용하여 추세를 파악하고 정보에 입각한 결정을 내립니다.
