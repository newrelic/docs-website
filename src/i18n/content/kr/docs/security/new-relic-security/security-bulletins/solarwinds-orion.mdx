우리는 고객이 [SolarWinds 보안 권고](https://www.solarwinds.com/securityadvisory) 가 비즈니스에 미치는 영향을 평가하고 있음을 이해합니다. 즉각적인 문제 해결: New Relic은 SolarWinds Orion 제품의 고객이 아니므로 New Relic과 New Relic을 사용하는 고객은 손상된 SolarWinds 제품의 영향을 받지 않습니다.

## New Relic과 SolarWinds Orion [#statement]

New Relic Security는 SolarWinds 사건에 대한 업데이트된 정보가 발표되는 대로 면밀히 모니터링하고 있습니다.

사전 조치로 New Relic security는 코드 무결성 및 소프트웨어 제공과 관련된 프로세스를 검토했습니다. New Relic은 필요한 경우 개선 사항을 구현하기 위해 적절한 조치를 취할 것입니다. New Relic은 또한 SolarWinds 사고와 관련된 지표에 대해 시스템을 계속 평가할 것입니다.

자체 공급업체 위험 관리 프로그램에 따라 New Relic은 주요 공급업체와 커뮤니케이션을 시작하여 그들이 우리에게 제공하는 서비스에서 SolarWinds 제품을 사용하는지 여부를 결정했습니다. 그들의 응답에 따라 New Relic은 회사와 고객에 대한 위험을 완화하기 위해 적절한 조치를 식별하고 취할 것입니다.

고객의 보안과 개인 정보 보호는 우리의 최우선 순위이며 우리는 SolarWinds 상황을 면밀히 추적하고 있습니다. 질문이 있는 고객은 [support.newrelic.com/](https://support.newrelic.com/) 으로 문의하시기 바랍니다.
