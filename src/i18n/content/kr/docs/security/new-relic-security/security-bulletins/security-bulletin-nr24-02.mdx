<DNT>**Vulnerability Identifier:**</DNT> NR24-02

<DNT>**Priority:**</DNT> 높은

## 요약

뉴렐릭은 뉴렐릭 Salesforce 내보내기를 사용하는 모든 고객에게 뉴렐릭이 최근 발표된 [OpenSSH의 취약한 버전을](https://nvd.nist.gov/vuln/detail/CVE-2024-6387) 제거하기 위해 출시한 버전 2.2.0으로 업데이트할 것을 권고합니다.

## 조치 필요

뉴렐릭은 뉴렐릭 Salesforce 내보내기를 사용하는 고객에게 즉시 버전 2.2.0으로 업데이트할 것을 권장합니다.

뉴렐릭은 이러한 업데이트를 지원하기 위해 다음 리소스를 제공했습니다.

* [Salesforce 내보내기 릴리스 노트](https://github.com/newrelic/newrelic-salesforce-exporter/releases/tag/2.2.0)
* [호스트 내 배포 감지 및 업그레이드](https://github.com/newrelic/newrelic-salesforce-exporter?tab=readme-ov-file#upgrading-on-host-deployments)

고객이 뉴웰릭 Salesforce 내보내기를 업그레이드할 수 없는 경우 네트워크 기반 제어를 통해 SSH 액세스를 제한하여 공격 위험을 최소화하세요.
