## 요약

높은 보안 모드가 활성화된 경우 오류 추적 또는 오류 이벤트 중 오류 메시지 캡처를 방지하기 위한 .NET 에이전트용 보안 업데이트입니다.

**출시 날짜:** 2018년 1월 22일

**취약점 식별자:** NR18-04

**우선순위:** 중간

## 취약점 정보 [#vuln-info]

New Relic의 .NET 오류 컬렉션은 포착되지 않은 예외에 대한 정보를 캡처하여 New Relic으로 보냅니다. 높은 보안 모드가 활성화된 경우 에이전트는 오류 메시지의 세부 정보를 캡처해서는 안 됩니다. 이 릴리스는 높은 보안 모드가 활성화된 경우 오류 메시지가 New Relic으로 전송되지 않도록 오류 추적 및 오류 이벤트에 대한 에이전트를 수정합니다.

### 완화 요인 [#factors]

* 이것은 [높은 보안 모드](/docs/agents/net-agent/configuration/net-agent-configuration#high_security_mode) 의 특별한 기능입니다.

## 해결 방법

에이전트 구성에서 [오류 수집](/docs/agents/net-agent/configuration/net-agent-configuration#error_collector) 을 비활성화하거나 구성에서 [특정 예외를 무시](/docs/agents/net-agent/configuration/net-agent-configuration#error-ignoreErrors) 하십시오.

## New Relic에 보안 취약점 보고 [#report]

New Relic은 고객과 고객 데이터의 보안을 위해 최선을 다하고 있습니다. 당사 제품이나 웹사이트 중 하나에서 보안 취약점을 발견했다고 생각되시면 New Relic의 통합 공개 프로그램에 신고해 주셔서 감사합니다. 자세한 내용은 [보안 취약점 보고](/docs/security/new-relic-security/data-privacy/reporting-security-vulnerabilities) 를 참조하십시오.
