## 취약점 정보 [#vuln-info]

합성 모니터링 기능이 있는 개인 위치 이미지가 Meltdown CPU 취약성으로부터 보호하기 위한 패치로 업데이트되었습니다. Spectre를 해결하기 위한 패치는 아직 Ubuntu에서 출시되지 않았습니다.

또한 고객은 개인 위치에 액세스하고 다음을 실행하여 현재 위치 업그레이드를 실행할 수 있습니다.

```
sudo apt-get update
sudo apt-get dist-upgrade
```

그런 다음 컴퓨터를 재부팅합니다.

추가 정보: [우분투 주의보](https://wiki.ubuntu.com/SecurityTeam/KnowledgeBase/SpectreAndMeltdown)

### 완화 요인 [#factors]

* 법칙로케이션은 고객의 데이터센터 내부에서 실행되며 내부적으로 신뢰할 수 있는 웹을 모니터링하게 됩니다.

## 해결 방법

알려진 해결 방법이 없습니다.

## New Relic에 보안 취약점 보고 [#report]

New Relic은 고객과 고객 데이터의 보안을 위해 최선을 다하고 있습니다. 당사 제품이나 웹사이트 중 하나에서 보안 취약점을 발견했다고 생각되시면 New Relic의 통합 공개 프로그램에 신고해 주셔서 감사합니다. 자세한 내용은 [보안 취약점 보고](/docs/security/new-relic-security/data-privacy/reporting-security-vulnerabilities) 를 참조하십시오.
