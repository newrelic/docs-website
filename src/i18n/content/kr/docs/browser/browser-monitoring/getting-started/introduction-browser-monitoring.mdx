    뉴렐릭의 브라우저 모니터링은 실제 사용자 모니터링(RUM) 솔루션을 제공합니다. 엔드유저가 다양한 웹 브라우저, 디바이스, 운영 체제 및 네트워크를 이용해 사이트를 탐색할 때 속도와 성능을 측정합니다. 그러나 브라우저 모니터링은 초기 페이지 로드에 대한 정보를 제공하는 것 그 이상입니다. 이를 사용해 전체 페이지 수명주기 데이터를 측정하고 고객 만족을 보장하는 데 도움이 되는 정보를 얻을 수 있습니다.

    뉴렐릭을 사용하면 브라우저 활동에서 얻어진 데이터를 모니터링하고 전체 스택에서 성능을 최적화할 수 있습니다. 브라우저 모니터링을 사용하면 성공적인 배포를 보장하고 고객이 볼 수 있는 문제를 신속하게 해결할 수 있습니다. 스택을 한눈에 모니터링하고 모든 엔터티가 정상적으로 작동하는지 확인할 수 있습니다. 애플리케이션 속도 및 성능, JavaScript 오류, AJAX 요청 등을 시각화합니다. 문제를 추적하는 데 소요되는 시간을 줄이고 고객에게 완벽한 디지털 경험을 제공하는 데 더 많은 시간을 할애할 수 있습니다.

    ## 웹 바이탈 추적

    Google의 코어 웹 바이탈은 사용자가 브라우저 앱을 경험하는 방식을 측정하는 세 가지 메트릭으로 구성되며, 뉴렐릭은 이러한 메트릭을 추적하는 데 도움을 줄 수 있습니다. 뉴렐릭에서는 두 곳에서 코어 웹 바이탈을 확인할 수 있습니다.

    * [앱의 성능을 보장하기 위한 Google 코어 웹 바이탈 모니터링](/docs/browser/new-relic-browser/guides/guide-to-core-web-vitals#monitor)
