브라우저 앱 성능 로그를 보려면:

<Steps>
  <Step>
    <DNT>[one.newrelic.com](https://one.newrelic.com) &gt; **All Capabilities** &gt; **Browser**</DNT> 으)로 이동합니다.
  </Step>

  <Step>
    브라우저 앱을 선택합니다.
  </Step>

  <Step>
    왼쪽 메뉴에서 **로그** 로 이동합니다.
  </Step>
</Steps>

<img title="Browser logs" alt="Screenshot of browser logs page" src="/images/browser_logs_ui.webp" />

## 다음은 뭐지?

* 또한 세션 리플레이(세션 리플레이 기능을 사용하는 경우)를 사용하여 컨텍스트에서 브라우저 로그를 보거나 주스 인박스(errors inbox)의 JavaScript 오류와 함께 볼 수도 있습니다. 자세한 내용은 [세션 리플레이에서 로그인을](#session-replay-section) 참조하고, [안경스 인박스(errors inbox)에서 로그인을 참조하세요](#js-errors-section) .

* 로그 UI 에 대한 자세한 내용은 [로그 UI사용을](/docs/logs/ui-data/use-logs-ui) 참조하세요.
