뉴렐릭에서 수집된 데이터 분석을 시작하세요. 이 섹션에서는 세션 리플레이 접속 및 탐색에 대한 지침을 제공하여 사용자 복합체 복합체를 탐색하고 개선이 필요한 영역을 식별하는 데 도움을 줍니다. 세션 리플레이 데이터를 활용하여 사용자 경험을 향상하고 문제를 효과적으로 해결하는 방법을 알아보세요.

## New Relic에서 세션 재생 보기 [#view-data]

세션 리플레이는 두 곳에서 찾을 수 있습니다:

* <DNT>**Session replay**</DNT> 페이지에서:

<Steps>
  <Step>
    <DNT>**[one.newrelic.com &gt; All Capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Browser**</DNT> 으)로 이동합니다.
  </Step>

  <Step>
    브라우저 앱을 선택하세요.
  </Step>

  <Step>
    왼쪽 메뉴에서 아래로 스크롤하여 <DNT>**Session replay**</DNT> 클릭합니다.
  </Step>

  <Step>
    각 재생을 클릭하면 세션 세부정보를 볼 수 있습니다.

    <img title="Session replay" alt="screenshot of the session replay page in browser" src="/images/browser_screenshot-full_Session-replay-page.webp" />
  </Step>
</Steps>

* <DNT>**Errors inbox**</DNT> 페이지에서:

<Steps>
  <Step>
    왼쪽 브라우저 메뉴에서 <DNT>**Errors**</DNT> 클릭합니다.
  </Step>

  <Step>
    <DNT>**Triage**</DNT> 및 <DNT>**Group errors**</DNT> 페이지를 클릭하면 오류에 첨부된 리플레이를 볼 수 있습니다.

    <img title="Replays in errors inbox" alt="Screenshot of replays in errors inbox" src="/images/browser_screenshot-full_errors-inbox-session-replay.webp" />
  </Step>
</Steps>
