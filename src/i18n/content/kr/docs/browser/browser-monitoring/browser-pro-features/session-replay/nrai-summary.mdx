<Callout title="시사">
  이 기능은 아직 개발 중이지만 꼭 사용해 보시기 바랍니다!

  이 기능은 현재 [사전 출시 정책](docs/new-relic-solutions/new-relic-one/core-concepts/new-relic-ai) 에 따른 미리보기 프로그램의 일부로 제공됩니다.
</Callout>

뉴렐릭 AI(NRAI)를 사용하면 세션 리플레이 이벤트 추적 데이터를 요약하고 매핑하여 사용자의 브라우저 여정의 오류와 성공을 더 잘 이해할 수 있습니다.

NRAI를 사용하여 세션 리플레이 이벤트 추적 데이터를 요약하려면 계정에 [NRAI가 활성화되어](/docs/agentic-ai/new-relic-ai/#enable) 있어야 합니다. 계정에서 NRAI가 활성화되면 다음을 수행하여 요약을 표시합니다.

<Steps>
  <Step>
    <DNT>**[one.newrelic.com &gt; All Capabilities](https://one.newrelic.com/all-capabilities) &amp;gt; Browser**</DNT> 으)로 이동합니다.
  </Step>

  <Step>
    브라우저 앱을 선택하세요.
  </Step>

  <Step>
    왼쪽 메뉴에서 아래로 스크롤하여 <DNT>**Session replay**</DNT> 클릭합니다.
  </Step>

  <Step>
    요약하고 싶은 세션의 세부 정보를 보려면 리플레이를 클릭하세요.

    <img title="Session replay" alt="screenshot of the session replay page in browser" src="/images/browser-session-replay.webp" />
  </Step>

  <Step>
    <DNT>**Summarize**</DNT> 클릭하세요. 세션 요약은 포커스 패널에 생성됩니다.

    <img title="Summarize session replay" alt="screenshot of the session replay page in browser" src="/images/finalcut.gif" />
  </Step>
</Steps>
