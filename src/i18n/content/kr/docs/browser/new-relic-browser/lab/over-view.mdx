당신은 사용자가 좋아하는 레스토랑에서 음식을 주문할 수 있는 [React](https://reactjs.org//) 앱인 Relicstaurants의 개발자입니다. 귀하의 앱은 사용자들 사이에서 정말 인기가 있으며 사용자가 항상 귀하의 사이트에서 최상의 경험을 할 수 있도록 하고 싶습니다. 이 목표를 달성하기 위해 New Relic 브라우저 모니터링을 사용하여 사이트의 성능을 향상하고 필요에 따라 문제를 해결합니다.

이 실습에서는 New Relic 브라우저 모니터링을 사용하여 웹 애플리케이션을 계측하고 메트릭을 모니터링하며 프런트엔드 문제를 해결합니다.

<SideBySide>
  <Side>
    ## 목표

    * React 데모 애플리케이션을 가동합니다.
    * New Relic 브라우저 에이전트를 설치합니다.
    * 프런트엔드 오류를 해결합니다.
    * New Relic에 소스 맵을 업로드합니다.
    * 프런트엔드 속도 저하 문제를 해결합니다.
  </Side>

  <Side>
    ## 요구 사항

    * 무료 [New Relic 계정](https://newrelic.com/signup?utm_source=developer-site)
    * [마디](https://nodejs.org/en/)
    * [파이썬3](https://www.python.org/)
    * [힘내](https://git-scm.com/)
    * [핍3](https://pip.pypa.io/en/stable/installation/)
  </Side>
</SideBySide>
