<InlinePopover type="browser" />[설정 요구 사항](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements) 의 일부로 [브라우저 에이전트를 설치](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent) 해야 합니다. 그런 다음 앱의 <DNT>**Applications settings**</DNT> 페이지를 사용하여 다음에 대한 브라우저 모니터링 옵션을 업데이트할 수 있습니다.

* 브라우저 구독 수준
* 브라우저 [Apdex T 값](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)
* [<DNT>**Geography**</DNT> 페이지](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)에서 강조표시할 도시, 지역 또는 국가

## Apdex 설정 선택 [#apdex-settings]

APM에서도 모니터링되는 앱의 경우 브라우저(최종 사용자) 및 [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex) 에 대해 [별도의 Apdex T 값을 설정할 수 있습니다.](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) <DNT>**Geography**</DNT> 페이지에서 강조표시할 도시, 지역 또는 국가를 선택할 수도 있습니다.

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt;**</DNT> (앱 선택) <DNT>**&gt; Settings &gt; Application settings**</DNT> 로 이동합니다.
2. <DNT>**Application settings**</DNT> 에서 브라우저 모니터링을 위해 특별히 Apdex T 값을 선택하거나 기본 Apdex 값을 그대로 둡니다.
3. <DNT>**Save application settings**</DNT> 선택합니다.

Apdex 설정을 변경할 때마다 브라우저 앱의 <DNT>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> 페이지에 있는 <DNT>**Page load timing**</DNT> 차트에 검은색 세로 막대가 나타납니다.

## 최종 사용자 Apdex 보기 [#view-browser-apdexs]

Apdex 점수를 보려면:

* <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> 으)로 이동합니다.

  또는

* <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DNT> 으)로 이동합니다.
