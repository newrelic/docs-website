브라우저 [설정 요구 사항](/docs/browser/new-relic-browser/welcome-new-relic-browser/compatibility-requirements) 의 일부로 [브라우저 에이전트를 설치](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent) 해야 합니다. 그런 다음 앱의 **애플리케이션 설정** 페이지를 사용하여 다음에 대한 브라우저 모니터링 옵션을 업데이트할 수 있습니다.

* 브라우저 구독 수준
* 브라우저 [Apdex T 값](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)
* [**지리** 페이지](/docs/browser/new-relic-browser/browser-pro-features/filterable-geography-webpage-metrics-location)에서 강조 표시할 도시, 지역 또는 국가

## Apdex 및 지역 설정 선택 [#apdex-settings]

[](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction)APM에서도 모니터링되는 앱의 경우 브라우저(최종 사용자) 및 [APM](/docs/apm/new-relic-apm/apdex/changing-your-apdex-settings)에 대해 별도의 Apdex T 값을 설정할 수 있습니다. **지리** 페이지에서 강조 표시할 도시, 지역 또는 국가를 선택할 수도 있습니다.

1. **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser >** (앱 선택) **> Settings > Application settings**으로 이동합니다.
2. **애플리케이션 설정** 에서 특별히 브라우저 모니터링을 위한 Apdex T 값을 선택하거나 기본 Apdex 값을 그대로 둡니다.
3. **지역** 페이지에서 강조 표시할 브라우저 모니터링에 대해 최대 5개의 국가 또는 지역을 선택합니다. [여러 앱 이름](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app)에 데이터를 보고하는 경우 기본 앱과 보조 앱 모두에서 동일한 국가를 선택합니다.
4. **애플리케이션 설정 저장을** 선택합니다.

Apdex 설정을 변경할 때마다 브라우저 앱의 **[요약](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)** 페이지에 있는 **페이지 로드 타이밍** 차트에 검은색 세로 막대가 나타납니다.

## 최종 사용자 Apdex 보기 [#view-browser-apdexs]

Apdex 점수를 보려면:

* **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser > (앱 선택) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)** 으로 이동합니다.

  또는

* **[one.newrelic.com](https://one.newrelic.com/all-capabilities) >Browser > (앱 선택) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)** 로 이동합니다.
