* 브라우저 구독 수준

## Apdex 및 지역 설정 선택 [#apdex-settings]

APM에서도 모니터링되는 앱의 경우 브라우저(최종 사용자) 및 [APM](/docs/apm/new-relic-apm/apdex/apdex-measure-user-satisfaction/#apm-apdex) 에 대해 [별도의 Apdex T 값을 설정할 수 있습니다.](/docs/apm/new-relic-apm/apdex/apdex-measuring-user-satisfaction) <DNT>**Geography**</DNT> 페이지에서 강조표시할 도시, 지역 또는 국가를 선택할 수도 있습니다.

1. <DNT>
     **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser >**
   </DNT>

   (앱 선택)

   <DNT>
     **> Settings > Application settings**
   </DNT>

   로 이동합니다.

2. <DNT>
     **Application settings**
   </DNT>

   에서 브라우저 모니터링을 위해 특별히 Apdex T 값을 선택하거나 기본 Apdex 값을 그대로 둡니다.

3. <DNT>
     **Geography**
   </DNT>

   페이지에서 강조표시할 브라우저 모니터링 국가 또는 지역을 최대 5개까지 선택하세요. [여러 앱 이름](/docs/agents/manage-apm-agents/app-naming/use-multiple-names-app) 에 데이터를 보고하는 경우 기본 앱과 보조 앱 모두에서 동일한 국가를 선택하세요.

4. <DNT>
     **Save application settings**
   </DNT>

   선택합니다.

Apdex 설정을 변경할 때마다 브라우저 앱의 <DNT>**[Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> 페이지에 있는 <DNT>**Page load timing**</DNT> 차트에 검은색 세로 막대가 나타납니다.

## 최종 사용자 Apdex 보기 [#view-browser-apdexs]

Apdex 점수를 보려면:

* <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Summary](/docs/browser/new-relic-browser/getting-started/browser-overview-website-performance-glance)**</DNT> 으)로 이동합니다.

  또는

* <DNT>**[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Browser > (select an app) > [Page views](/docs/browser/new-relic-browser/additional-standard-features/page-views-understanding-your-sites-popularity)**</DNT> 으)로 이동합니다.
