**애플리케이션 설정** 페이지에서 브라우저 모니터링 설정에 다음 구성을 지정할 수 있습니다.

* [알림 설정](/docs/browser/new-relic-browser/configuration/view-browser-apps-alert-information/)
* [APDEX 및 지리 설정 구성](/docs/browser/new-relic-browser/configuration/browser-settings-ui-options-apdex-geography/#apdex-settings)
* [도메인 및 하위 도메인 필터링](/docs/browser/new-relic-browser/configuration/monitor-or-block-specific-domains-subdomains/)
* [AJAX 요청 필터링](/docs/browser/new-relic-browser/configuration/filter-ajax-request-events/)
* [URL 패턴별로 데이터 그룹화](/docs/browser/new-relic-browser/configuration/group-browser-metrics-urls/)
* [브라우저 앱 이름 바꾸기](/docs/browser/new-relic-browser/configuration/rename-browser-apps/)
* [브라우저 모니터링 비활성화](/docs/browser/new-relic-browser/installation/disable-browser-monitoring/)
* [브라우저 앱 삭제](/docs/apm/new-relic-apm/maintenance/remove-applications-new-relic/#remove)

**애플리케이션 설정** 페이지로 이동하려면:

1. **[one.newrelic.com > All capabilties](https://one.newrelic.com/all-capabilities) > Browser** 로 이동하여 앱을 선택합니다.
2. 왼쪽 탐색에서 **설정** 까지 아래로 스크롤하고 **애플리케이션 설정을**클릭합니다.
