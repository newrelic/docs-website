브라우저 에이전트가 보고하는 데이터를 규제하는 몇 가지 구성 옵션이 있습니다. 이것은 [브라우저 모니터링을 활성화할](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent) 때의 요소이며 나중에 이러한 구성을 변경할 수도 있습니다.

앱 설정 페이지로 이동하려면:

1. [one.newrelic.com](http://one.newrelic.com) 으로 이동하여 **브라우저** 를 클릭합니다.
2. 앱을 선택하세요.
3. 왼쪽 탐색 메뉴에서 **앱 설정** 을 클릭합니다.

이에 대한 액세스 권한이 없으면 권한 때문일 수 있습니다.
