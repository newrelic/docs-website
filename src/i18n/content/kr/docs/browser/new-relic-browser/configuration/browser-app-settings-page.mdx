### 브라우저 세션 저장소 [#browser-session-storage]

브라우저 에이전트 내에서 세션 트레이스와 세션 리플레이 데이터를 수집하기 위한 제어 기능을 통합했습니다. 이제 이 설정 페이지의 <DNT>**Access browser local storage**</DNT> 토글에서 모든 세션 수준 데이터를 직접 관리할 수 있습니다.

* **브라우저 로컬 저장소 토글 액세스**: 이 통합 제어는 세션 수준 데이터(세션 ID 및 기타 세션 수준 속성 포함)가 수집되는 방식을 결정합니다.

  * **켜진 경우(기본값)**: 뉴렐릭 브라우저 에이전트는 브라우저의 영구 저장 메커니즘을 사용하여 세션 트레이스 및 세션 리플레이 데이터를 수집합니다.
  * **OFF 시**: 뉴렐릭 브라우저 에이전트는 브라우저의 영구 저장소 사용을 즉시 중단합니다. 이 작업을 수행하면 모든 세션 트레이 및 세션 리플레이 데이터가 수집되지 않으므로 브라우저의 세션 관련 정보 처리를 직접 제어하고 데이터 개인 정보 보호 및 규정 준수 요구 사항을 충족하는 능력을 향상시킬 수 있습니다.

<DNT>**Application settings**</DNT> 페이지로 이동하려면:

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser**</DNT>로 이동하여 앱을 선택합니다.
2. 왼쪽 탐색 메뉴에서 <DNT>**Settings**</DNT> 까지 아래로 스크롤하고 <DNT>**Application settings**</DNT> 클릭합니다.

## 브라우저 API [#apis]

또한 브라우저 에이전트 계측을 사용자 정의하여 사용자 지정 추적 및 이벤트를 캡처하고, 필요에 맞게 특정 오류를 보고하고, 최적의 성능을 위해 데이터 수집 타이밍을 미세 조정할 수 있습니다. 시작하려면 [브라우저 API 사용을](/docs/browser/new-relic-browser/browser-apis/using-browser-apis/) 참조하세요.
