New Relic 무료 버전으로 시작했고 모든 강력한 브라우저 모니터링 기능을 최적화할 준비가 되었으면 에이전트를 Browser+Lite에서 Browser Pro 또는 Browser+SPA로 업그레이드할 수 있습니다.

브라우저 버전 유형을 변경하려면 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Browser > (앱 선택) > Settings > Application settings**으로 이동합니다.

다음은 요구 사항에 가장 적합한 유형을 결정하는 데 도움이 되는 다양한 브라우저 에이전트 유형 간의 몇 가지 비교입니다.

## 브라우저 에이전트 유형: Lite, Pro, Pro+SPA [#agent-types]

브라우저 에이전트에는 Lite, Pro 및 Pro+SPA의 세 가지 유형이 있습니다. 에이전트 유형은 청구에 영향을 주지 않습니다.

## 가격 세부 정보

각 브라우저 에이전트 유형의 가격 책정에 대해 자세히 알아보려면 다음 문서를 확인하세요.

* [사용량 기반 가격 책정](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing) : 이 가격 책정 모델에는 청구 요소로 데이터 수집이 있습니다. 데이터 수집을 줄이려면 설치 후 더 작은 에이전트 유형으로 다운그레이드하는 것이 좋습니다.
* [원래 가격](/docs/accounts/original-accounts-billing/product-pricing/product-based-pricing) : 브라우저 모니터링 기능에 대한 액세스는 에이전트 유형이 아닌 구독 계획에 따라 결정됩니다. 즉, 기본 Pro+SPA 에이전트를 사용하지 않을 이유가 없습니다.
