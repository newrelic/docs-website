브라우저 에이전트 유형은 청구에 영향을 미치지 않습니다. 이는 원하는 경우 더 최소한의 기능 세트로 전환하는 방법일 뿐입니다. 설치된 기본 브라우저 에이전트는 모든 <InlinePopover type="browser" />기능이 활성화된 상태로 제공됩니다.

브라우저 버전 유형을 변경하려면 <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Browser &gt; (select an app) &gt; Settings &gt; Application settings**</DNT> 으로 이동하세요.

다음은 다양한 브라우저 에이전트 유형 간의 몇 가지 비교입니다.

## 브라우저 에이전트 유형 [#agent-types]

Lite, Pro 및 Pro+SPA의 세 가지 유형의 브라우저 에이전트가 있습니다.
