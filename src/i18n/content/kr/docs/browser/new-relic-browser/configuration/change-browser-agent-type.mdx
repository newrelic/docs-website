브라우저 에이전트 유형은 청구에 영향을 주지 않습니다. 원하는 경우 보다 최소한의 기능 집합으로 전환하는 방법일 뿐입니다. 설치된 기본 브라우저 에이전트는 모든 브라우저 모니터링 기능이 활성화된 상태로 제공됩니다.

브라우저 버전 유형을 변경하려면 **[one.newrelic.com > All capabilties](https://one.newrelic.com/all-capabilities) > Browser > (앱 선택) > Settings > Application settings** 으로 이동하세요.

다음은 다양한 브라우저 에이전트 유형 간의 몇 가지 비교입니다.

## 브라우저 에이전트 유형 [#agent-types]
