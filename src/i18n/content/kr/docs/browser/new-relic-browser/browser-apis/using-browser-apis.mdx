브라우저 API를 사용하면 <InlinePopover type="browser"/>맞춤설정하고 확장할 수 있습니다. API 사용을 시작하려면 [모니터링하려는 모든 페이지에 브라우저 에이전트가 있는지](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent) 확인하세요.

다음은 각 브라우저 API 메소드의 사용 사례입니다.

## 추적 및 이벤트 [#api-traces]

추적 및 이벤트를 모니터링하려면 다음 방법을 사용하십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        원하는 경우...
      </th>

      <th>
        이 API를 사용하세요
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        New Relic이 자동으로 추적하지 않는 동작 추적
      </td>

      <td>
        [`newrelic.addPageAction()`](/docs/browser/new-relic-browser/browser-apis/addpageaction)
      </td>
    </tr>

    <tr>
      <td>
        진행 중인 [세션 추적](/docs/browser/new-relic-browser/browser-pro-features/session-traces-exploring-webpages-life-cycle) 에 개체 추가
      </td>

      <td>
        [`newrelic.addToTrace()`](/docs/browser/new-relic-browser/browser-apis/addtotrace)
      </td>
    </tr>

    <tr>
      <td>
        사용자 정의 기준에 따라 페이지가 완료되는 시점을 기록합니다.
      </td>

      <td>
        [`newrelic.finished()`](/docs/browser/new-relic-browser/browser-apis/finished)
      </td>
    </tr>

    <tr>
      <td>
        페이지의 후속 이벤트에 속성 이름 및 값 추가
      </td>

      <td>
        [`newrelic.setCustomAttribute()`](/docs/browser/new-relic-browser/browser-apis/setcustomattribute)
      </td>
    </tr>

    <tr>
      <td>
        페이지의 후속 이벤트에 애플리케이션 버전 정보 추가
      </td>

      <td>
        [`newrelic.setApplicationVersion()`](/docs/browser/new-relic-browser/browser-apis/setApplicationVersion)
      </td>
    </tr>

    <tr>
      <td>
        페이지 이름을 설정하여 페이지 보기 그룹화
      </td>

      <td>
        [`newrelic.setPageViewName()`](/docs/browser/new-relic-browser/browser-apis/setpageviewname)
      </td>
    </tr>

    <tr>
      <td>
        페이지의 후속 이벤트에 사용자 ID 정보 추가
      </td>

      <td>
        [`newrelic.setUserId()`](/docs/browser/new-relic-browser/browser-apis/setuserid)
      </td>
    </tr>

    <tr>
      <td>
        단일 브라우저 로그 이벤트 캡처
      </td>

      <td>
        [`newrelic.log()`](/docs/browser/new-relic-browser/browser-apis/log)
      </td>
    </tr>

    <tr>
      <td>
        기존 로거 메소드를 통과하는 메시지를 로그 이벤트로 자동 캡처
      </td>

      <td>
        [`newrelic.wrapLogger()`](/docs/browser/new-relic-browser/browser-apis/wrapLogger)
      </td>
    </tr>
  </tbody>
</table>

## 오류 보고 [#api-errors]

애플리케이션의 오류를 보고하려면 다음 방법을 사용하십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "350" }}>
        원하는 경우...
      </th>

      <th>
        이 API를 사용하세요
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        앱의 버전 정보로 오류에 태그 지정
      </td>

      <td>
        [`newrelic.addRelease()`](/docs/browser/new-relic-browser/browser-apis/addrelease)
      </td>
    </tr>

    <tr>
      <td>
        앱 작동을 중단하지 않고 포착되거나 처리된 오류를 캡처합니다.
      </td>

      <td>
        [`newrelic.noticeError()`](/docs/browser/new-relic-browser/browser-apis/noticeerror)
      </td>
    </tr>

    <tr>
      <td>
        알려진 오류를 선택적으로 무시
      </td>

      <td>
        [`newrelic.setErrorHandler()`](/docs/browser/new-relic-browser/browser-apis/seterrorhandler)
      </td>
    </tr>
  </tbody>
</table>

## 단일 페이지 앱 추적(SPA만 해당) [#spa-api]

단일 페이지 애플리케이션을 추적하려면 다음 SPA API 메소드를 사용하십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        원하는 경우...
      </th>

      <th>
        이 API를 사용하세요
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        [SPA 상호작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 의 [`actionText`](/attribute-dictionary/?event=BrowserInteraction&attribute=actionText)(클릭한 HTML 요소의 텍스트)를 설정합니다.
      </td>

      <td>
        [`newrelic.interaction().actionText()`](/docs/browser/new-relic-browser/browser-apis/actiontext)
      </td>
    </tr>

    <tr>
      <td>
        SPA [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 의 하위 구성 요소를 별도로 추적
      </td>

      <td>
        [`newrelic.interaction().createTracer()`](/docs/browser/new-relic-browser/browser-apis/createtracer)
      </td>
    </tr>

    <tr>
      <td>
        SPA [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 종료
      </td>

      <td>
        [`newrelic.interaction().end()`](/docs/browser/new-relic-browser/browser-apis/end)
      </td>
    </tr>

    <tr>
      <td>
        현재 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 과 관련된 데이터 집계
      </td>

      <td>
        [`newrelic.interaction().getContext()`](/docs/browser/new-relic-browser/browser-apis/getcontext)
      </td>
    </tr>

    <tr>
      <td>
        [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 무시
      </td>

      <td>
        [`newrelic.interaction().ignore()`](/docs/browser/new-relic-browser/browser-apis/ignore)
      </td>
    </tr>

    <tr>
      <td>
        현재 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 에 바인딩된 새 API 개체 반환
      </td>

      <td>
        [`newrelic.interaction()`](/docs/browser/new-relic-browser/browser-apis/interaction)
      </td>
    </tr>

    <tr>
      <td>
        [상호 작용](/attribute-dictionary/?event=BrowserInteraction) 이 기록되기 전에 이름 및 무시된 속성과 같은 속성 조정
      </td>

      <td>
        [`newrelic.interaction().onEnd()`](/docs/browser/new-relic-browser/browser-apis/onend)
      </td>
    </tr>

    <tr>
      <td>
        브라우저 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 이 끝날 때 저장되는지 확인

      <td>
        [`newrelic.interaction().save()`](/docs/browser/new-relic-browser/browser-apis/save)
      </td>
    </tr>

    <tr>
      <td>
        현재 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 에 사용자 정의 속성 추가
      </td>

      <td>

      </td>
    </tr>

    <tr>
      <td>

      </td>

      <td>

      </td>
    </tr>
