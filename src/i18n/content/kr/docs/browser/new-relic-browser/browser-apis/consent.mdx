## 통사론

```js
newrelic.consent(accept?: boolean)
```

브라우저 에이전트가 네트워크를 통해 데이터를 전송하는 데 대한 동의를 허용하거나 거부합니다.

## 요구 사항

* 브라우저 Lite, Pro 또는 Pro+SPA 에이전트(v1.303.0 이상)

## 설명

브라우저 에이전트에는 선택적인 동의 모드 기능이 있습니다. 앱 설정이나 NerdGraph를 통해 동의 모드가 활성화되면 사용자는 `newrelic.consent(...)` 메서드를 호출하여 네트워크를 통해 데이터를 전송할지 여부에 대한 동의를 허용하거나 거부할 수 있습니다.

설정 및 사용에 대한 자세한 내용은 [동의 모드에 대한 문서를](/docs/browser/new-relic-browser/configuration/consent-mode) 참조하세요.

## 사용 사례

* 사용자가 사이트의 모달에 동의할 때까지 데이터 수집을 기다리고 싶습니다.

## 예

### 사용자는 브라우저 에이전트가 데이터를 수집하는 데 동의합니다.

```js
newrelic.consent()
```

### 사용자가 브라우저 에이전트의 데이터 수집에 대한 동의를 거부합니다.

```js
newrelic.consent(false)
```
