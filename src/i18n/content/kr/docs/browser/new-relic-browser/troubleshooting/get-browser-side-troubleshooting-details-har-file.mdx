## 문제

New Relic Support를 사용하려면 브라우저 측에서 일어나는 일에 대한 추가 문제 해결 세부 정보가 필요하며 페이지에 대한 액세스 가능한 URL이나 소스 코드를 제공할 수 없습니다.

## 해결책

브라우저 유형에 따라 조사를 위해 [HAR(HTTP 아카이브)](https://dvcs.w3.org/hg/webperf/raw-file/tip/specs/HAR/Overview.html) 파일을 생성하고 [support.newrelic.com](https://support.newrelic.com/) 으로 보냅니다. HAR 파일에는 기록된 기간 동안 페이지와 관련된 네트워크 호출에 대한 모든 정보가 포함됩니다.

HAR 파일을 기록할 때 사이트의 기능을 사용하여 적절한 유형의 활동을 생성합니다. 예를 들어, AJAX 문제가 있는 경우 페이지에서 AJAX 호출을 수행하는 섹션을 연습하십시오.

<Callout variant="important">
  HAR 파일은 기록되는 동안 모든 네트워크 트래픽을 캡처합니다. 이 트래픽 데이터에는 개인 정보 또는 개인 식별 정보(쿠키, 비밀번호, 신용 카드 번호 등)가 포함될 수 있습니다. 고객은 HAR 파일을 포함하여 New Relic에 제공하는 모든 데이터에 대한 책임이 있습니다. HAR 파일은 텍스트 편집기로 편집할 수 있습니다. HAR 파일을 공유하기 전에 민감한 정보를 수동으로 제거하는 것이 좋습니다.
</Callout>

<CollapserGroup>
  <Collapser
    id="chrome"
    title="크롬"
  >
    Google Chrome 브라우저에는 HAR 파일을 생성하는 기능이 있습니다. 브라우저 버전에 따라 콘솔을 여는 경로 이름이 다를 수 있습니다.

    1. Chrome 메뉴(오른쪽 상단)에서 해당하는 경우

       <DNT>
         **Tools > Developer tools**
       </DNT>

       또는

       <DNT>
         **More tools > Developer tools**
       </DNT>

       선택합니다.

    2. 콘솔에서

       <DNT>
         **Network**
       </DNT>

       선택합니다.

    3. 녹음 아이콘을 선택합니다.

    4. 해당하는 경우

       <DNT>
         **Preserve log**
       </DNT>

       확인란을 선택하여 네트워크 트랜잭션 데이터를 저장합니다.

    5. 문제를 재현하기 전에 지우기 아이콘을 선택하여 불필요한 헤더 정보를 제거하십시오.

    6. 사이트를 탐색하여 문제가 있는 트래픽 유형을 생성합니다.

    7. 브라우저가 New Relic 수집기에 데이터를 보고할 때까지 몇 분 정도 기다립니다.

    8. 콘솔 하단에 있는 프로세서 목록을 마우스 오른쪽 버튼으로 클릭하고

       <DNT>
         **Save as HAR with Content**
       </DNT>

       선택하여 캡처된 정보를 저장합니다.
  </Collapser>

  <Collapser title="Firefox 버전 41.x 이상">
    Firefox 버전 41 이상에는 트랜잭션을 HAR 파일로 복사하는 기능이 있습니다.

    1. Firefox 메뉴(오른쪽 상단)에서

       <DNT>
         **Developer > Network**
       </DNT>

    2. 불필요한 헤더 정보를 제거하려면

       <DNT>
         **Clear**
       </DNT>

       옵션(개발자 창 오른쪽 하단)을 선택하세요.

    3. 사이트를 탐색하여 문제가 있는 트래픽 유형을 생성합니다.

    4. 브라우저가 New Relic 수집기에 데이터를 보고할 때까지 몇 분 정도 기다립니다.

    5. 콘솔에서 트랜잭션 목록을 마우스 오른쪽 버튼으로 클릭하고

       <DNT>
         **Copy All As HAR**
       </DNT>

       선택하여 데이터를 버퍼에 배치하거나

       <DNT>
         **Save All As HAR**
       </DNT>

       선택하여 파일에 복사하여 정보를 캡처합니다.
  </Collapser>

  <Collapser title="마이크로소프트 엣지">
    1. <DNT>
         **Developer Tools**
       </DNT>

       열려면 키보드에서 F12 키를 누르거나 메뉴에서

       <DNT>
         **Developer Tools**
       </DNT>

       찾으세요.

    2. <DNT>
         **Network**
       </DNT>

       탭을 선택합니다.

    3. 페이지를 새로고침하여 브라우저와 서버 간의 트래픽 캡처를 시작하거나 영향을 받는 페이지의 링크를 클릭하십시오.

    4. HAR 파일을 저장하려면

       <DNT>
         **Export as HAR**
       </DNT>

       과

       <DNT>
         **Save As...**
       </DNT>

       차례로 선택하세요.
  </Collapser>

  <Collapser title="애플 사파리">
    1. 개발 메뉴 활성화: macOS 메뉴 표시줄에서

       <DNT>
         **Safari > Preferences > Advanced > Show Develop menu in menu bar**
       </DNT>

       으로 이동합니다.

    2. macOS 메뉴 표시줄에서

       <DNT>
         **Develop > Show Web Inspector > Network**
       </DNT>

       탭으로 이동합니다.

    3. <DNT>
         **Network**
       </DNT>

       탭 내에서

       <DNT>
         **Preserve log**
       </DNT>

       옵션을 확인하세요.

    4. 페이지를 새로고침하고

       <DNT>
         **Safari**
       </DNT>

       (가) 브라우저-웹사이트 합성을 기록하도록 허용합니다.

    5. 페이지가 로드되면 네트워크 탭 창의 오른쪽 상단에 있는

       <DNT>
         **Export**
       </DNT>

       선택합니다.
  </Collapser>
</CollapserGroup>
