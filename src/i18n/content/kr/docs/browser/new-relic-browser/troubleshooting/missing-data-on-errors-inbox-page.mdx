## 문제

<DNT>**Errors Inbox**</DNT> 페이지에 일부 오류가 표시되지 않습니다.

## 해결책

브라우저 에이전트는 페이지에 표시되는 <DNT>**unhandled**</DNT> 오류를 자동으로 수집합니다. 페이지나 애플리케이션 내에서 오류가 발생하고 있음을 알고 있지만 <DNT>**Errors Inbox**</DNT> 페이지에 표시되지 않는 경우 오류가 페이지나 애플리케이션에서 처리되고 있고 브라우저 에이전트에 표시되지 않는지 확인하는 것이 중요합니다. 이러한 시나리오에서는 [`noticeError` API를](/docs/browser/new-relic-browser/browser-apis/noticeerror) 활용하여 처리된 오류를 계속 관찰할 수 있습니다.
