## 문제

<DNT>**Web vitals**</DNT> 에 데이터가 표시되지 않습니다.

## 해결책

누락된 데이터의 원인은 다음과 같습니다.

* 필터가 너무 좁습니다. 일부 주요 속성은 특정 웹 바이탈에만 관련됩니다. 예를 들어 `elementID` 로 필터링한 경우 `elementID` 은 LCP 측정에만 적용할 수 있으므로 데이터는 콘텐츠가 포함된 가장 큰 페인트(LCP)에 대해서만 채워집니다.
* 브라우저 에이전트 또는 버전은 특정 데이터를 수집하지 않습니다. New Relic 브라우저 에이전트가 수집하는 주요 속성에 대해 알아보려면 [브라우저 에이전트 데이터 사전을](/attribute-dictionary/?dataSource=Browser+agent) 참조하세요.
* 앱에서 `PageViewTiming` 데이터를 New Relic으로 보낼 수 없습니다. 웹 바이탈은 `PageViewTiming` 데이터를 기반으로 하므로 어떤 이유로 앱이 수집기에 수집을 보낼 수 없는 경우 데이터가 표시되지 않을 수 있습니다.
