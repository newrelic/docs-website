## 문제

AngularJS를 사용 중이고 New Relic에 JavaScript 오류를 보고하고 싶지만 [**JavaScript 오류** 페이지](/docs/browser/new-relic-browser/browser-pro-features/js-errors-dashboard-examining-errors-over-time) 에 애플리케이션의 오류가 표시되지 않습니다.

## 해결책

AngularJS 애플리케이션에는 모든 오류를 포착하는 전역 오류 처리기가 포함되는 경우가 많으며 기본적으로 브라우저 모니터링은 포착되지 않은 오류만 보고합니다.

[NoticeError](/docs/browser/new-relic-browser/browser-agent-spa-api/newrelicnoticeerror-browser-agent-api) API 호출을 사용하여 New Relic에 오류를 보고합니다. 이를 통해 일반 오류 처리의 일부로 처리할 수 있지만 New Relic UI에서도 볼 수 있습니다.
