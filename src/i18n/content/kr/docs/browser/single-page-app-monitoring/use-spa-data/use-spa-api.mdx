브라우저의 단일 페이지 애플리케이션(SPA) 모니터링에는 특정 브라우저 상호 작용에 대한 사용자 지정 모니터링을 추가하는 API가 포함됩니다. 이것은 동적으로 업데이트되는 위젯과 같이 경로 변경을 초래하지 않기 때문에 New Relic이 자동으로 기록하지 않는 상호 작용을 모니터링하는 데 유용합니다. SPA API를 사용하면 모니터링하기에 충분히 중요하지 않다고 생각하는 상호 작용에 대한 기본 모니터링을 끌 수도 있습니다.

특정 API 호출을 포함하여 SPA API에 대한 자세한 내용은 [브라우저 에이전트 및 SPA API](/docs/browser/new-relic-browser/guides/guide-using-browser-spa-apis/) 를 참조하십시오.
