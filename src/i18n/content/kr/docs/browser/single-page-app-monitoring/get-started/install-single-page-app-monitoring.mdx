단일 페이지 앱(SPA) 모니터링은 기본 [설치 와 함께 제공됩니다. <InlinePopover type="browser"/>대리인](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent/).

## 요구 사항 [#requirements]

[여기에서 SPA 모니터링에 대한 호환성 및 요구 사항을](/docs/browser/new-relic-browser/getting-started/compatibility-requirements-browser-monitoring#spa/)검토할 수 있습니다.

New Relic 계정에서 첫 번째 모니터링 앱을 설정할 때 서비스 약관에 동의해야 합니다. 약관에 동의하면 New Relic이 URL에서 해시 조각을 수집할 수 있도록 승인하는 것입니다. 계정에 대해 확인란 옵션을 한 번만 선택하면 됩니다.

## SPA 모니터링 활성화 또는 비활성화 [#enable-spa]

[브라우저 모니터링을 활성화](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#enable)하면 분산 추적을 비롯한 다양한 최신 기능에 대한 액세스를 제공하는 SPA 모니터링이 기본적으로 포함됩니다. 일부 이전 에이전트 설치를 업그레이드해야 할 수도 있습니다. [브라우저 에이전트 유형](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#agent-types)에 대해 자세히 알아보십시오.
