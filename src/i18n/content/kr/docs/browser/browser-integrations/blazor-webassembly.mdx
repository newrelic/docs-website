## 인프라 에이전트 설치 [#infra]

Blazor WebAssembly를 New Relic과 통합하는 데 인프라 에이전트가 필요하지 않지만 시스템을 완전히 관찰하려면 설치하는 것이 좋습니다. 인프라 에이전트는 호스트에 대한 로그 및 기타 데이터를 수집하여 애플리케이션 지표와 함께 성능 지표를 표시합니다.

인프라 에이전트를 설치하는 방법에는 두 가지가 있습니다.

* [가이드 설치](https://one.newrelic.com/launcher/catalog-pack-details.launcher/?pane=eyJuZXJkbGV0SWQiOiJucjEtY29yZS5saXN0aW5nIn0=&cards%5B0%5D=eyJuZXJkbGV0SWQiOiJucjEtaW5zdGFsbC1uZXdyZWxpYy5ucjEtaW5zdGFsbC1uZXdyZWxpYyJ9) 에 따라 인프라 에이전트로 시스템을 계측하십시오.

## Blazor WebAssembly와 New Relic 통합 [#integrate]

Blazor WebAssembly에서 데이터를 수집하려면 다음 절차를 따르세요.

1. [one.newrelic.com](https://one.newrelic.com)에서, [**데이터 추가** 페이지](https://one.newrelic.com/marketplace)로 이동합니다.
2. **브라우저 및 모바일 을**클릭한 다음 **브라우저 모니터링 을**클릭합니다.
3. **JavaScript 코드 복사/붙여넣기 선택** 옵션을 선택합니다(이 코드에 대한 자세한 내용은 [브라우저 설치 문서](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent)참조).
4. 나머지 단계를 완료하고 **활성화를 클릭합니다.**
5. Blazor WebAssembly 애플리케이션의 `wwwroot` 디렉터리로 이동합니다.
6. `index.html` 파일을 엽니다.
7. 본문 태그가 끝나기 직전( `</body>` )에 브라우저 코드 스니펫을 붙여넣습니다.
8. 응용 프로그램을 다시 시작하십시오.
9. **[one.newrelic.com](https://one.newrelic.com/all-capabilities)** 으로 이동합니다. 그런 다음 **측정항목 및 이벤트** 를 클릭합니다. 샘플 NRQL 쿼리를 실행하여 New Relic이 데이터를 수집하고 있는지 확인합니다.

```sql
SELECT average(windowLoad) FROM PageViewTiming WHERE (deviceType = 'Desktop') FACET `userAgentName` SINCE 604800 seconds AGO TIMESERIES
```

10. 기본 설치

    <InlinePopover type="dashboards"/>

    [Blazor WebAssembly Instant Observability 페이지](https://newrelic.com/instant-observability/blazor-web-assembly)로 이동한 다음 페이지 상단의 **설치를** 클릭합니다.

11. one.newrelic.com에서 **대시보드** 로 이동하고 Blazor **[WebAssembly](https://one.newrelic.com/all-capabilities)** 를 클릭하여 대시보드를 봅니다.
