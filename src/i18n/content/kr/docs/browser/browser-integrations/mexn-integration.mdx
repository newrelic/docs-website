MEAN, MERN 및 MEVN 그리드는 오픈소스 전체 그리드 소프트웨어 개발 키트입니다. 각 키트는 단일 코드베이스를 사용하여 웹 애플리케이션을 개발합니다. 뉴렐릭은 세 개의 그리드를 모두 하나의 퀵스타트로 통합했습니다. MExN 통합은 <InlinePopover type="browser"/>에이전트, Node.js 에이전트 및 MongoDB 통합을 사용하여 가장 중요한 지표(예: 응답 시간, CPU 사용률, 트래픽 및 로그인 빈도)가 포함된 사전 구축된 대시보드를 제공합니다.

## 설치 [#install]

<Steps>
  <Step>
    ### 선택사항: 인프라 에이전트 설치 [#infra-install]

    인프라 에이전트 설치는 선택 사항이지만 이렇게 하면 앱이 실행 중인 호스트에 대한 중요한 메트릭을 얻을 수 있습니다.

    이를 수행하려면 [인프라 에이전트 설치 단계를](/docs/infrastructure/install-infrastructure-agent/get-started/install-infrastructure-agent)따르십시오.
  </Step>

  <Step>
    ### MongoDB 빠른 시작 설치

    1. 에이전트를 설치하기 전에 [설명서](/docs/infrastructure/host-integrations/host-integrations-list/mongodb-monitoring-integration/) 에서 MongoDB 에이전트 요구 사항을 확인하십시오.

    2. [MongoDB 빠른 시작 설치를](https://newrelic.com/instant-observability/mongodb)엽니다.

    3. MongoDB 에이전트 설치를 시작하려면

       <DNT>
         **Install now**
       </DNT>

       클릭하세요.
  </Step>

  <Step>
    ### Node.js 에이전트 설치

    1. [설명서](/docs/apm/agents/nodejs-agent/getting-started/compatibility-requirements-nodejs-agent/)에서 Node.js 에이전트 요구 사항을 확인하세요.

    2. [Node.js 빠른 시작 설치를](https://newrelic.com/instant-observability/node-js)엽니다.

    3. Node.js 에이전트 설치를 시작하려면

       <DNT>
         **Install now**
       </DNT>

       클릭하세요.
  </Step>

  <Step>
    ### 브라우저 에이전트 설치 [#browser-install]

    1. [one.newrelic.com](https://one.newrelic.com) 에서,

       <DNT>
         [**+ Integrations & Agents**](https://one.newrelic.com/marketplace)
       </DNT>

       [페이지](https://one.newrelic.com/marketplace) 로 이동하세요.

    2. <DNT>
         **Browser and mobile**
       </DNT>

       클릭한 다음

       <DNT>
         **Browser monitoring**
       </DNT>

       클릭합니다.

    3. <DNT>
         **Select copy/paste JavaScript code**
       </DNT>

       옵션을 선택합니다(이 코드에 대한 자세한 내용은 [브라우저 설치 문서](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent) 참조).

    4. 나머지 단계를 완료하고 다음을 클릭하세요.

       <DNT>
         **Enable.**
       </DNT>

    5. React 애플리케이션이나 Angular 애플리케이션 또는 Vue 애플리케이션의 `index.html `파일로 이동합니다.

    6. 본문 태그가 끝나기 직전( `</body>` )에 브라우저 코드 스니펫 스크립트 태그를 붙여넣습니다.

    7. 응용 프로그램을 다시 시작하십시오.

       몇 분 안에 애플리케이션이 메트릭을 [one.newrelic.com](https://one.newrelic.com)으로 보냅니다.
  </Step>
</Steps>

## 데이터 찾기 [#find-data]

MExN 대시보드를 가져오려면:

1. [one.newrelic.com](https://one.newrelic.com) 에서,

   <DNT>
     **+Integrations & Agents**
   </DNT>

   페이지로 이동하세요.

2. <DNT>
     **Dashboards**
   </DNT>

   을 클릭합니다.

3. 검색창에 `MExN` 를 입력합니다.

4. MExN 대시보드가 나타납니다. 그것을 클릭하여 설치하십시오.

귀하의 MExN 대시보드는 맞춤형 대시보드로 간주되며 <DNT>**Dashboards**</DNT> UI 에서 찾을 수 있습니다. 대시보드 사용 및 편집에 대한 문서는 [대시보드 문서 를](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) 참조하세요.

보고된 데이터에 대한 자세한 내용은 설치한 각 도구의 문서를 참조하세요.

* [브라우저 모니터링 이벤트](/docs/data-apis/understand-data/event-data/events-reported-browser-monitoring).
* [인프라 데이터](/docs/infrastructure/manage-your-data/data-instrumentation/default-infrastructure-monitoring-data/#infrastructure-events).
* [몽고디비 데이터](/docs/infrastructure/host-integrations/host-integrations-list/mongodb/mongodb-monitoring-integration-new/#find-and-use/).
* [Node.js 데이터](/docs/apm/agents/nodejs-agent/getting-started/introduction-new-relic-nodejs/).

다음은 데스크톱 컴퓨터에서 사용자가 사용하는 다양한 브라우저의 평균 창 로드 시간을 확인하는 NRQL 쿼리입니다.

```sql
SELECT average(windowLoad) 
FROM PageViewTiming 
WHERE (deviceType = 'Desktop') 
FACET `userAgentName` 
SINCE 604800 seconds 
AGO TIMESERIES
```
