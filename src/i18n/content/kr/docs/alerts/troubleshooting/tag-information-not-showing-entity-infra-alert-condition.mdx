## 문제

[사용자 정의 공지 이벤트 설명 태그는](/docs/alerts/create-alert/condition-details/custom-alert-event-descriptions#example) 해당 이벤트의 값으로 대체되지 않습니다.

## 해결책

태그는 New Relic에 의해 인덱싱된 이벤트에만 사용할 수 있습니다.

이벤트가 인덱싱되었는지 여부를 확인하는 가장 쉬운 방법은 `SELECT nr.entityType FROM ${EventSampleName}` 쿼리를 실행하여 `${EventSampleName}` 을 해당 이벤트의 이름으로 바꾸는 것입니다.

결과의 열 `nr.entity Type` 에 값이 있으면 이벤트가 인덱싱됩니다. 그렇지 않으면 그렇지 않습니다.

## 원인

New Relic에서 인덱싱되지 않은 샘플에는 설명을 추가할 수 없습니다.
