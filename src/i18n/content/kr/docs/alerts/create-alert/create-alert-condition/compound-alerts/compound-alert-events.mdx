복합 공지 조건의 트리거 표현식이 true로 평가되면 뉴렐릭은 복합 공지 이벤트를 엽니다. 이벤트에는 런북 URL에 대한 링크가 포함되어 있습니다. 각 구성 요소 조건에 대한 특정 임계값 위반을 보려면 개별 공지 이벤트를 클릭하세요.

## 공지 이벤트 및 이슈 연관 [#alert-events-correlation]

복합 공지 조건은 여러 관련 공지 이벤트를 생성합니다:

* <DNT>**Multiple alert events**</DNT>: 복합 공지 조건은 자체 복합 공지 이벤트를 생성합니다. 기본적으로 각 구성 요소 조건은 임계값을 위반할 때 자체 공지 이벤트를 생성합니다. 노이즈를 줄이려면 공지 조건의 설정에서 이러한 이벤트를 무음으로 설정할 수 있습니다. 구성 요소 조건의 공지 이벤트를 무음으로 설정하면 해당 조건의 태그가 복합 공지 이벤트에 나타나지 않습니다.
* <DNT>**Correlating to the same issue**</DNT>: 모든 관련 공지 이벤트(복합 및 구성 요소 모두)가 동일한 이슈로 그룹화되도록 하려면 구성 요소 조건이 복합 공지 조건과 동일한 정책을 사용하도록 설정하세요. 이렇게 하면 뉴렐릭의 이슈 연관이 이들을 함께 그룹화할 수 있습니다.
* <DNT>**Viewing related events**</DNT>: 복합 공지 이벤트에서 개별 구성 요소 공지 이벤트로 이동하여 각 조건에 대한 임계값 위반 및 메트릭을 확인할 수 있습니다.
