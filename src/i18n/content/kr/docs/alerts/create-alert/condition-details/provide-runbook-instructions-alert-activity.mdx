공지 [조건에는](/docs/alerts/create-alert/create-alert-condition/alert-conditions) [런북](/docs/using-new-relic/welcome-new-relic/get-started/glossary#alert-runbook) 지침의 URL을 지정하는 옵션이 포함됩니다. 이를 통해 상황이 조건의 [임계 범위를](/docs/alerts/new-relic-alerts-beta/configuring-alert-policies/define-thresholds-trigger-alert) 위반할 때 열린 공지 이벤트를 처리하기 위한 정보 또는 표준 절차에 연결할 수 있습니다.

</DNT>

조건과 연결된 Runbook URL을 편집하려면 다음을 수행합니다.

1. <DNT>**[one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) &gt; Alerts &gt; Alert conditions (policies) &gt; (select a policy) &gt; (select a condition)**</DNT> 으)로 이동합니다.
2. <DNT>**Edit conditions**</DNT> 페이지에서 <DNT>**Add details**</DNT> 클릭한 다음 <DNT>**Runbook URL**</DNT> 까지 아래로 스크롤합니다. 런북 URL의 전체 경로(런북 URL, 내부 위키 페이지 URL 등)를 제공하세요.
3. <DNT>**Enable on save**</DNT> 전환하고 <DNT>**Save**</DNT> 클릭합니다.
4. 선택 사항: 공지 조건에서 기존 런북 URL을 변경하거나 삭제하려면 <DNT>**Remove**</DNT> 클릭하세요.

담당자가 무엇을 해야 할지 정확히 알 수 있도록 귀하의 런북 URL을 <DNT>**Alert event**</DNT> 세부정보와 이메일 공지에 포함하겠습니다.
