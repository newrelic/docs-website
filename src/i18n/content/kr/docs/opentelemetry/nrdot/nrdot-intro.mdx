뉴렐릭 OpenTelemetry 배포판(NRODT)은 뉴렐릭에서 지원하는 OpenTelemetry 프로젝트 구성 요소 배포판 컬렉션을 말합니다. NRDOT 구성 요소는 뉴렐릭 제품 및 서비스와 원활하게 작동하도록 설계되었으며, 가능한 경우 OpenTelemetry 뉴렐릭과 통합하는 데 권장되는 방법입니다.

<Callout variant="tip">
  현재 NRDOT에는 OpenTelemetry [Collector](/docs/opentelemetry/nrdot/nrdot-collector/) 의 사전 빌드된 배포판이 포함되어 있습니다. 언어 API 및 SDK에 대해서는 [APM 모니터링을](/docs/opentelemetry/get-started/apm-monitoring/opentelemetry-apm-intro/) 시작하는 방법에 대한 문서를 참조하세요.
</Callout>
