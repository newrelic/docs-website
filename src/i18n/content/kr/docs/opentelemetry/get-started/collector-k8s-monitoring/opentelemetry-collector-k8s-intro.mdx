[OpenTelemetry Collector](https://opentelemetry.io/docs/collector/) 텔레메트리데이터를 수신, 처리, 내보내기 위한 공급업체와 독립적인 도구입니다. Collector 요구 사항과 설정은 다양하지만 다양한 수신기와 프로세서가 포함되어 있어 Kubernetes Monitoring에 널리 사용됩니다. 필요한 텔레메트리를 모두 얻으려면 각기 다른 권한 요구 사항이 있는 여러 수신기를 사용하여 다양한 Kubernetes API를 스크래핑해야 하므로 작업을 간편하게 수행할 수 있도록 후드 아래에서 수집기 [nrdot-수집기-k8s](https://github.com/newrelic/nrdot-collector-releases/tree/main/distributions/nrdot-collector-k8s) 의 사용자 지정 배포를 사용하는 헬름 차트를 만들었습니다. 시작하는 방법에 대한 자세한 내용은 [Kubernetes용 OpenTelemetry를](/docs/kubernetes-pixie/kubernetes-integration/installation/k8s-otel/) 참조하세요.
