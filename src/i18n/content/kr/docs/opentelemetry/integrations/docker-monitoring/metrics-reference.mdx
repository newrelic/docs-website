이 페이지는 OpenTelemetry [dockerstatsreceiver](https://github.com/open-telemetry/opentelemetry-collector-contrib/tree/main/receiver/dockerstatsreceiver) 에서 수집한 모든 도커 통계에 대한 포괄적인 참조 정보를 제공합니다. 이 참조 자료를 활용하여 사용 가능한 데이터를 파악하고, 사용자 지정 쿼리를 생성하고, 컨테이너화된 애플리케이션에 대한 효과적인 알림을 설정하세요.

## 완전한 참조 [#metrics-reference]
