<Callout variant="important">
  비프로덕션 구현, 배포로만 IAST를 실행합니다. IAST는 익스플로잇 페이로드로 HTTP 요청을 호출하여 테스트합니다. 데이터 손상과 라이브 코드에 대한 익스플로잇 도입을 방지하려면 시뮬레이션된 데이터만 사용하여 운영 환경에서 IAST를 사용해야 합니다. 또한 사전 프로덕션 단계에서 새 코드에 대해 IAST를 실행하면 잠재적인 취약점이 활성화되기 전에 이를 포착할 수 있습니다.
</Callout>

New Relic IAST를 설치하려면:

<Steps>
  <Step>
    <DNT>**[one.newrelic.com](https://one.newrelic.com) > All capabilities > IAST**</DNT> 으로 이동하여 <DNT>**Set up IAST with applications**</DNT> 클릭합니다.

    <img
      title="Set up IAST with applications"
      alt="Set up IAST with applications"
      src="/images/iast_screenshot-full_install.webp"
    />
  </Step>

  <Step>
    IAST가 운영 환경에서 실행되고 있지 않은지 확인하고 <DNT>**Continue**</DNT> 클릭합니다.
  </Step>

  <Step>
    애플리케이션의 언어를 선택하고 단계를 완료하세요.

    <img
      title="Install New Relic IAST"
      alt="Install New Relic IAST"
      src="/images/iast_screenshot-crop_install.webp"
    />
  </Step>

  <Step>
    `newrelic.yml` 구성 파일이 다음과 같이 업데이트되었는지 확인하세요.

    <CollapserGroup>
      <Collapser
        id="config-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> 구성 파일</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
        ```
      </Collapser>

      <Collapser
        id="config-eu-file-example"
        title={<><InlineCode>newrelic.yml</InlineCode> EU용 구성 파일</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
            validator_service_url: wss://csec.eu01.nr-data.net
        ```
      </Collapser>

      <Collapser
        id="config-file-fed-example"
        title={<><InlineCode>newrelic.yml</InlineCode> Fed 사용자를 위한 구성 파일</>}
      >
        ```yml
        security:
            enabled: true
            agent:
                enabled: true
            validator_service_url: wss://csec-gov.nr-data.net
        ```
      </Collapser>
    </CollapserGroup>
  </Step>

  <Step>
    모든 단계를 완료한 후에는 애플리케이션을 다시 시작하고 애플리케이션의 API에 대한 트래픽을 생성하세요.
  </Step>

  <Step>
    테스트된 애플리케이션의 개요를 보려면 <DNT>**See your data**</DNT> 클릭하세요.
  </Step>
</Steps>
