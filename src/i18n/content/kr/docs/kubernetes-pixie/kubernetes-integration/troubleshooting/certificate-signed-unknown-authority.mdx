## 문제 [#logs-versions]

**`x509: certificate signed by unknown authority`** 메시지가 표시됩니다.

## 해결책

서명된 인증서가 올바르게 구성되었는지 확인해야 합니다. 이렇게 하려면 매니페스트의 `DaemonSet` 부분에서 다음 변수를 사용하여 `.pem` 파일을 설정합니다.

```
- name: NRIA_CA_BUNDLE_DIR 
  value: <var>YOUR_CA_BUNDLE_DIR</var> 
- name: NRIA_CA_BUNDLE_FILE  
  value: <var>YOUR_CA_BUNDLE_NAME</var>
```

_YAML 키 경로: `spec.template.spec.containers.name.env`_
