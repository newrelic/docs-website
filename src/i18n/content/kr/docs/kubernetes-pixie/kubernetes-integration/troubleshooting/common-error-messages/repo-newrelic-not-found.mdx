## 문제

Helm 또는 Manifest로 [Kubernetes 통합을 설치하는](/install/kubernetes/) 동안 이 오류 메시지가 표시됩니다.

```shell
  repo newrelic not found
```

## 해결책

다음 명령을 실행하여 helm 차트에 newrelic 리포지터리를 추가하세요.

```shell
  helm repo add newrelic https://helm-charts.newrelic.com
```
