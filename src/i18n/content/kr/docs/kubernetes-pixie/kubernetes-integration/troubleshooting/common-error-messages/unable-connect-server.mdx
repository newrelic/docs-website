## 문제

설치 지침을 따르면 이 출력 오류가 발생합니다.

```shell
  Unable to connect to the server: dial tcp [7777:777:7777:7777:77::77]:443: i/o timeout
```

## 해결책

이는 Kubernetes 클라이언트와 Kubernetes API 서버 간에 네트워크 연결 문제가 발생했음을 나타냅니다. 설치 지침을 다시 실행하기 전에 Kubernetes 클라이언트가 Kubernetes API 서버에 연결할 수 있는지 확인하세요.
