## 문제

설치하는 동안 [뉴렐릭 dockerhub](https://hub.docker.com/u/newrelic) 와 Google의 [`registry.k8s.io`](https://github.com/kubernetes/registry.k8s.io) 에 문제가 있습니다.

## 해결책

허용 목록에 해당 도메인을 추가했는지 확인하세요. 설치 시 이 위치에서 컨테이너 이미지를 가져옵니다. [`registry.k8s.io`에 대한 연결을 테스트하여](https://kubernetes.io/blog/2023/03/10/image-registry-redirect/#how-can-i-check-if-i-am-impacted) 화이트리스트/포함에 추가할 추가 Google 레지스트리를 찾을 수 있습니다. `registry.k8s.io` 일반적으로 로컬 레지스트리 도메인으로 리디렉션됩니다. 예를 들어, 귀하의 지역에 따라 `asia-northeast1-docker.pkg.dev` 입니다.
