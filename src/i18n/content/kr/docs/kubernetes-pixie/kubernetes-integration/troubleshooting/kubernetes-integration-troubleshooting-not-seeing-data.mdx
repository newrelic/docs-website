## 문제

[뉴렐릭 Kubernetes 통합을 설치했지만](/install/kubernetes/) 뉴렐릭 계정에 Kubernetes 데이터가 표시되지 않습니다.

## 해결책

1. `CLUSTER_NAME` 및 `NRI_LICENSE_KEY` 올바르게 구성했는지 확인하세요. 자세한 내용은 [설치 지침을](/docs/integrations/host-integrations/host-integrations-list/kubernetes-monitoring-integration#install) 참조하세요.

2. 다음을 실행하여 통합 배포가 성공했는지 확인합니다.

   ```
   kubectl get pods -o wide -n <NEWRELIC_NAMESPACE>
   kubectl get deployments -o wide -n <NEWRELIC_NAMESPACE>
   kubectl get daemonsets -o wide -n <NEWRELIC_NAMESPACE>
   ```

3. [인프라 에이전트 및 통합에서 로그를 검색하고](/docs/integrations/kubernetes-integration/troubleshooting/get-logs-version) [오류 메시지](/docs/integrations/host-integrations/troubleshooting/kubernetes-integration-troubleshooting-error-messages) 가 없는지 확인합니다.
