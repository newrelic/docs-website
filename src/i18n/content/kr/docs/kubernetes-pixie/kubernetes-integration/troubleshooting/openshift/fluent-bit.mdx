## 문제

`cannot increase buffer: current=32000 requested=64768 max=32000` 또는 `pod out of memory` 과 같은 오류가 발생합니다.

## 해결책

유창한 비트의 버퍼 크기를 늘리려면 `k8sBufferSize` 속성을 추가합니다.

```yaml
newrelic-logging:
  enabled: true
  containerSecurityContext:
    privileged: true
  fluentBit:
    k8sBufferSize: "64k"
    persistence:
      mode: "none"
```
