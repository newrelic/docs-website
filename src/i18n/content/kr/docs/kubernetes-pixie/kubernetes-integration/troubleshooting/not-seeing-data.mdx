## 문제

[뉴렐릭 Kubernetes 통합을](/install/kubernetes/) 설치했지만 뉴렐릭 계정에서 Kubernetes 데이터가 표시되지 않습니다.

## 해결책

1. [`values-newrelic.yaml`](https://github.com/newrelic/helm-charts/blob/master/charts/nri-bundle/values.yaml) 파일에서 `CLUSTER_NAME` 및 `NRI_LICENSE_KEY` 변수를 올바르게 구성했는지 확인하세요.

2. 다음 명령을 실행하여 통합이 성공적으로 배포되었는지 확인하세요.

```bash
  kubectl get pods,deployments,daemonsets -o wide -n <newrelic_namespace>
```

3. 인프라 에이전트와 통합의 [로그를 확인](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/get-logs-version/) 하고 오류 메시지가 없는지 확인하세요.
