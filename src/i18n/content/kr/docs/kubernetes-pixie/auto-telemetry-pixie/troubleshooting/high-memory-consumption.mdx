## 문제

New Relic의 Pixie 통합을 위한 [설치 절차](/docs/kubernetes-pixie/auto-telemetry-pixie/install-auto-telemetry-pixie) 를 완료했으며 Pixie가 사용하는 메모리 양을 줄이고 싶습니다.

## 해결책

Pixie [는 원격 측정 데이터 수집과 이 데이터의 단기 클러스터 내 저장이라는 두 가지 목적으로 메모리를 사용합니다](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#memory-usage) .단기 저장에 대해 다른 양의 메모리를 지정하려면 설치 중에 [단기 저장 메모리 제한을 구성](/docs/kubernetes-pixie/auto-telemetry-pixie/advanced-configuration/manage-pixie-memory#reduce-data-storage) 할 수 있습니다.
