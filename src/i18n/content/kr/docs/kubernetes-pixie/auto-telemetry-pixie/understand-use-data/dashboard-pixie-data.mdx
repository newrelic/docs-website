## 사전 제작된 Pixie 대시보드

Pixie 데이터용으로 사전 제작된 대시보드를 설치하려면:

1. 새 유물 [대시보드](https://one.newrelic.com/dashboards) 페이지로 이동합니다.
2. **미리 빌드된 대시보드 찾아보기** 버튼을 선택합니다.
3. "Pixie"를 검색하고 사용 가능한 옵션 중 하나를 선택하십시오.
4. **이 빠른 시작 설치** 버튼을 선택합니다.

## 사용자 정의 Pixie 대시보드

사전 구축된 Pixie 대시보드의 시각화 위젯은 NRQL 쿼리의 데이터로 채워집니다.대시보드 위젯의 점 3개 메뉴를 선택하여 데이터를 제공하는 쿼리를 봅니다.지속형 Pixie 데이터에 대한 쿼리를 작성하는 방법을 배우려면 [Querying Pixie 데이터](/docs/kubernetes-pixie/auto-telemetry-pixie/understand-use-data/query-pixie-data) 페이지를 참조하십시오.

사전 구축된 대시보드를 사용자 지정하거나 고유한 사용자 지정 대시보드를 구축하는 방법에 대한 자세한 내용은 [여기 를 참조하십시오](https://docs.newrelic.com/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards) .
