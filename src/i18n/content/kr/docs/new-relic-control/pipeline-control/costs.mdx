귀하의 파이프라인 사용료는 귀하의 주문에 따라 청구됩니다. 이 기능과 관련된 비용은 계정과 관련된 가격 책정 모델에 적용되는 다음 요소에 따라 결정됩니다.

**고급 컴퓨팅** (고급 CCU로 측정):

* 게이트웨이 규칙이 데이터에 영향을 미치는지 여부와 관계없이 게이트웨이를 통해 전송되도록 에이전트 또는 텔레메트리 페이로드를 구성한 결과로 게이트웨이에서 수신한 압축되지 않은 GB
* cloud 규칙에 의해 삭제된 GB
* cloud 규칙으로 검색된 이벤트(기준, 이벤트, 로그 및 트레이스 포함)

**Core 컴퓨트**: Core CCU에서 측정된 NRDB 쿼리를 생성하는 파이프라인 취소 UI 의 특정 페이지를 볼 때.

**데이터 수집**: 파이프라인 위험에 의해 삭제되지 않고 최종적으로 NRDB에 저장되는 모든 데이터(수집된 GB 단위로 측정).

온디맨드 컴퓨트 고객의 경우 Core 컴퓨트 및 Advanced 컴퓨트 제품이 컴퓨트 제품으로 제공되며 CCU로 측정됩니다.

가격에 대한 자세한 내용은 당사 영업팀에 문의하시거나 주문서를 참조하세요.
