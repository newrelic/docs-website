귀하의 파이프라인 사용료는 귀하의 주문에 따라 청구됩니다. 라인 컨트롤은 기본 가격 책정 모델(사용자 기반 가격 책정 또는 Core 컴퓨트) 위에 계층화된 추가 기능인 Advanced 컴퓨트 파이프 SKU의 일부로 판매됩니다. 고급 컴퓨트 번들에는 고급 기능이 포함되어 있으며, 고급 컴퓨트 소비 단위(aCCU)를 사용하여 해당 기능에 대한 요금이 부과됩니다.

## 비용 계산 방법

파이프라인 지배력과 관련된 비용은 다음 요소에 의해 결정됩니다.

### 게이트웨이 계량

게이트웨이는 사용자의 인프라에서 실행되며, 사용량 측정은 간단합니다.

**측정 대상 항목**:

* 게이트웨이 엔드포인트에서 **수신된 데이터 용량** (압축되지 않은 바이트 단위)
* **어떤 처리 방식이 적용되었는지 또는 데이터가 삭제되었는지 여부와 관계없이** 요금이 부과됩니다.
* 필터링/차단이 발생하기 전 **유입량을 기준으로** 측정됩니다.

### 클라우드 규칙 측정

클라우드 규칙은 뉴렐릭의 인프라에서 실행되므로 게이트웨이에 추가 요금이 부과됩니다.

(드롭 규칙에 따라) **측정되는 항목은 무엇입니까** ?

* **삭제된 GB** 수: 설정한 규칙에 따라 삭제된 데이터 용량
* **BSE(수십억 개의 이벤트 스캔)**: 규칙 실행 시 이벤트가 평가됩니다.
* 두 항목 모두 별도의 항목이며 각각 비용이 발생합니다.
* 일반적으로 BSE는 대다수 고객의 경우 사용량(GB)에 비해 전체 cloud 규칙 요금에서 **훨씬 낮은 비율을 차지합니다.**

**BSE 집계**:

* NRQL 규칙에서 루트 레벨 `SELECT` 과 일치하는 데이터만 스캔합니다.
* 예시: `DELETE FROM Log WHERE level = 'INFO'` 은 데이터에 대한 BSE만 계산합니다. `Log`
* 보다 선택적인 쿼리는 더 적은 수의 이벤트를 스캔합니다.

### 기타 파이프라인 비용

**Core Compute**: NRDB 쿼리를 생성하는 파이프라인 취소 UI 의 특정 페이지를 볼 때 Core CCU에서 측정됩니다.

**데이터 수집**: 파이프라인 위험에 의해 삭제되지 않고 최종적으로 NRDB에 저장되는 모든 데이터(수집된 GB 단위로 측정).

온디맨드 컴퓨트 고객의 경우 Core 컴퓨트 및 Advanced 컴퓨트 제품이 컴퓨트 제품으로 제공되며 CCU로 측정됩니다.

구체적인 가격 정보 및 사용 사례에 가장 비용 효율적인 모델을 알아보려면 영업팀에 문의하거나 주문 내역을 참조하십시오.
