## 문제: 계정 드롭다운 메뉴 문제

파이프라인 제어 UI에서 **All accounts** \[모든 계정]을 선택하지 않았거나 게이트웨이 라이선스 키와 연결된 계정을 선택하지 않았습니다.

## 해결책

* 파이프라인 취소 UI 의 계정 드롭다운 메뉴를 확인하여 올바른 계정이 선택되었는지 확인하세요.
* 관련 MELT 데이터에 액세스하려면 **All accounts** \[모든 계정]을 보거나 게이트웨이 라이선스 키에 연결된 특정 계정을 확인하세요.
