/\* 뉴렐릭에서 모든 데이터를 수신하고 있지만 예상 서비스 수준 계약(SLA)을 초과하여 도착하는 경우, 이 가이드는 문제를 진단하고 해결하는 데 도움이 됩니다. 발생한 특정 문제에 따라 아래 단계를 따르세요: \*/

## 문제: SLA를 초과하여 데이터가 도착함

예상 기간 내에 데이터가 수신되지 않습니다.

## 해결책

* 지연 시간 지표를 확인하여 데이터 전송 지연을 확인하세요.
* 처리 용량을 향상시키고 지연 시간을 줄이기 위해 게이트웨이 파드의 최소 개수를 늘립니다.
