## 문제: API 경쟁 조건

여러 사용자가 게이트웨이 업데이트(작성, 업데이트, 삭제 작업)에 대한 requests 동시에 보내서 변경 사항이 손실됩니다.

## 해결책

* [뉴렐릭 지원팀](https://support.newrelic.com) 에 문의하세요.
* 경쟁 조건으로 인해 손실된 작업을 다시 실행합니다.
