## 문제: 사용자 역량 부족

Org Product Admin 및 Organization Manager와 관련된 필수 권한이 없습니다.

## 해결책

* 사용자에게 &quot;에이전트 인증을 설정하는 데 필요한 조직 수준 기능이 없습니다.&quot;라는 오류 메시지가 표시됩니다.
* 시스템 ID 생성에 필요한 기능이 포함된 역할을 부여받으려면 계정 관리자에게 문의하세요.
* 자세한 내용은 [사용자 권한 문서를](/docs/accounts/accounts-billing/new-relic-one-user-management/user-permissions/) 참조하세요.

## 문제: 오래된 Helm 차트 버전

Helm 차트를 설치하는 명령줄 스크립트가 실패하는 이유는 로컬 Helm 버전이 오래되었기 때문입니다.

## 해결책

* 설치 스크립트와의 호환성을 보장하기 위해 로컬 Helm 설치를 최신 버전으로 업데이트하세요.
* Helm 업데이트가 필요하다는 오류 메시지를 찾아 지침에 따라 업그레이드하세요.
