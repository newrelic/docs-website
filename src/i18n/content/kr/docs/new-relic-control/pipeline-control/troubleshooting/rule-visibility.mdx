## 문제: 배포되지 않은 변경 사항

규칙 목록에는 활성 상태만 표시되고 아직 변경 사항이 적용되지 않았다는 사실을 모르고 계십니다.

## 해결책

* 처리중이거나 배포중인 게이트웨이 업데이트에 대한 업데이트 페이지를 확인하세요.
* 변경 사항이 규칙 목록에 반영되도록 하려면 해당 변경 사항을 구현, 배포해야 합니다.

## 문제: API 경쟁 조건

여러 사용자가 게이트웨이 업데이트에 대한 requests 동시에 보내서 변경 사항이 손실됩니다.

## 해결책

* [뉴렐릭 지원팀](https://support.newrelic.com) 에 문의하세요.
* 경쟁 조건으로 인해 손실된 작업을 다시 실행합니다.
