## 문제: 리소스 고갈

클러스터가 CPU 또는 메모리 리소스를 모두 소모했습니다.

## 해결책

* Kubernetes UI 사용하여 파드이벤트 및 보류 중인 작업을 보고 리소스 제약조건을 식별합니다.
* 노드 풀 크기를 늘리거나 패드에 대한 리소스 제한(CPU 및 메모리) requests 조정합니다.
* 활성 노드 수에 대한 cloud 공급자 한도를 확인하고 필요에 따라 설정을 조정합니다.
