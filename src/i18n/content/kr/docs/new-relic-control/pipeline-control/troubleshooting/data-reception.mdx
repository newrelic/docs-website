## 문제: 텔레메트리 생산자의 잘못된 구성

텔레메트리 생산자가 잘못 구성되어 텔레메트리 데이터가 게이트웨이로 전송되지 않습니다.

## 해결책

* 설정 오류를 식별하려면 생산자 로그에 액세스하세요.
* 올바른 설정 단계는 적절한 API, 에이전트 또는 텔레메트리 생산자 문서를 참조하세요.

## 문제: 속도 제한을 받고 있습니다

텔레메트리 데이터의 속도 제한에 도달했습니다.

## 해결책

* 뉴렐릭 API 에서 수신된 429 상태 코드에 대한 게이트웨이 HTTP 클라이언트 응답 코드를 확인하세요.
* 계정에서 생성된 이벤트를 검토하여 속도 제한을 나타냅니다.
* 텔레메트리 데이터 속도 관리 및 조정에 대한 지침은 [속도 제한 문서를](https://docs.newrelic.com/search-results/?query=rate%20limiting&page=2) 참조하세요.
