뉴렐릭 APM 에이전트와 네트워크 내 게이트웨이 간의 안정적인 통신을 촉진하기 위해 게이트웨이 클러스터에 대한 개인 DNS 호스트명을 설정하려면 다음 단계를 따르세요.

1. [AWS Route 53을](https://docs.aws.amazon.com/Route53/latest/DeveloperGuide/hosted-zones-private.html) 사용하여 DNS 호스트명을 생성합니다.
2. [ACM](https://docs.aws.amazon.com/acm/latest/userguide/acm-public-certificates.html) 을 통해 게이트웨이 호스트명에 대한 인증서를 얻습니다.
