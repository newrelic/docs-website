## 문제 해결, 구현 구현, 구현을 통한 구현, 구현 이벤트

<DNT>Fleet Control</DNT> 구현, 배포 중에 발생하는 모든 이벤트에 대한 세부적인 가시성을 제공하는 <DNT>deployment events</DNT> 탭을 제공합니다. 이는 문제 해결, 구현 구현, 배포 문제 및 근본 원인 이해를 위한 기본 도구입니다.

### 구현, 배포 이벤트 액세스

구현, 배포 이벤트 탭에 액세스하는 방법에는 두 가지가 있습니다.

**구현, 배포 기록에서:**

1. 특정 함대 내의 <DNT>deployments</DNT> 탭으로 이동합니다.
2. 조사하고자 하는 구현 및 배포 행을 클릭하십시오.
3. 구현, 배포 요약 보기에서 <DNT>Deployment Events</DNT> 탭을 클릭합니다.

**새로운 구현을 생성한 후 배포:**

1. 구현, 배포를 시작한 후 진행 보기가 완료되고 구현, 배포 요약 보기로 전환됩니다.
2. 이 요약 보기에서 <DNT>Deployment Events</DNT> 탭을 클릭하면 문제 해결, 해결 또는 확인에 대한 이벤트를 즉시 확인할 수 있습니다.

### 구현, 배포 이벤트 테이블 이해

<DNT>deployment events</DNT> 테이블은 구현, 배포 중에 수행된 각 작업에 대한 원자적 이벤트 수준 데이터를 표시합니다. 표에는 다음과 같은 열이 포함되어 있습니다.

* <DNT>**Timestamp:**</DNT> 이벤트가 발생한 시간.
* <DNT>**Status:**</DNT> 이벤트의 상태(예: <DNT>Completed, Config Applied, Applying</DNT>).
* <DNT>**Managed entity name:**</DNT> 이벤트가 발생한 클러스터 또는 호스트의 이름입니다.
* <DNT>**Agent name:**</DNT> 이벤트와 관련된 특정 에이전트입니다.
* <DNT>**Agent type:**</DNT> 에이전트 유형(예: <DNT>New Relic Infrastructure, Fluent Bit</DNT>).
* <DNT>**Event:**</DNT> 발생한 이벤트 유형입니다.
* <DNT>**Details:**</DNT> 실패한 이벤트에 대한 오류를 포함하는 이벤트에 대한 추가 정보입니다.
* <DNT>**Duration:**</DNT> 이벤트가 완료되는 데 걸린 시간.

### 이벤트 필터링 및 검색

<DNT>deployment events</DNT> 탭은 이벤트를 좁히는 여러 가지 방법을 제공합니다.

* <DNT>**Search:**</DNT> 검색창을 이용하여 키워드로 특정 이벤트를 찾아보세요.
* <DNT>**Filter by entities:**</DNT> 특정 관리 대상 엔티티에 대한 이벤트를 표시하도록 필터링합니다. 이는 특히 대규모 클러스터(예: 100개의 클러스터)를 보유하고 있고 하나 또는 소수의 엔티티만 오류가 발생한 경우에 유용합니다.
* <DNT>**Filter by status:**</DNT> 이벤트 상태(예: <DNT>Completed, Config Applied, Applying</DNT>)별로 이벤트를 필터링하여 성공 또는 문제가 발생한 이벤트에 집중할 수 있습니다.

### NRQL이용한 고급 문제 해결

보다 고급 문제 해결, 해결을 위해 구현, 배포 이벤트 테이블을 채우는 기본 NRQL 쿼리에 액세스할 수 있습니다.

1. <DNT>Deployment Events</DNT> 탭에서 이벤트 테이블 헤더 오른쪽에 있는 버튼을 찾으세요.
2. <DNT>Data Explorer</DNT> 에서 NRQL 쿼리 탭을 열려면 이 버튼을 클릭하십시오.
3. 해당 쿼리에는 구현 및 배포 ID가 미리 채워져 있으며, 다양한 ID 및 배포 ID를 포함한 더 자세한 정보가 담긴 추가 열이 표시됩니다.
4. 문제를 더 자세히 조사하기 위해 필요한 경우 이 쿼리를 수정할 수 있습니다.
5. 테이블 형식 보기와 JSON 형식 보기 간 전환과 같은 <DNT>Data Explorer&apos;s</DNT> 표준 기능을 사용하세요.

<DNT>Data Explorer</DNT> 사용에 대한 자세한 내용은 [데이터 탐색기 소개를](/docs/query-your-data/explore-query-data/query-builder/introduction-new-data-explorer/) 참조하세요.
