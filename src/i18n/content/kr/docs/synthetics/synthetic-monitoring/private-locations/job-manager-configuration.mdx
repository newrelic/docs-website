구성에서 [환경 변수](#environment-variables) 를 사용하여 [합성 작업 관리자](/docs/synthetics/private-locations/install-job-manager) 를 구성하는 방법을 알아봅니다.

<Callout variant="important">
  [사용자 정의 모듈](/docs/synthetics/private-locations/containerized-private-minion-cpm-configuration/#custom-modules) , [영구 데이터 저장소](/docs/synthetics/private-locations/containerized-private-minion-cpm-configuration/#preserve-data-volume) 및 [사용자 정의 환경 변수](/docs/synthetics/private-locations/containerized-private-minion-cpm-configuration/#vars-scripted-monitors) 는 현재 합성 작업 관리자에 대해 지원되지 않습니다.
</Callout>

참고로 New Relic은 합성 작업 관리자 파일에 대한 수정 사항에 대해 책임을 지지 않습니다.

## 환경 변수 [#environment-variables]

환경 변수를 사용하면 특정 환경 및 기능 요구 사항을 충족하도록 합성 작업 관리자 구성을 미세 조정할 수 있습니다.
