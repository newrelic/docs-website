뉴렐릭 Synthetics [대전](/docs/synthetics/synthetic-monitoring/private-locations/private-locations-overview-monitor-internal-sites-add-new-locations) UI 업데이트되었습니다. 금고 키는 비밀 저장소로 이전되었으며 인터페이스의 여러 영역에서 키 표시 기능이 제거되었습니다.

## 변경 사항 [#whats-changing]

* **목록 보기**: 고립로케이션 목록에서 키 열이 제거되었습니다.
* **필터**: 더 이상 단축키로 필터링할 수 없습니다.
* **메타 데이터 &amp; 태그**: 표시 데이터 &amp; 태그 섹션에 더 이상 표시되지 않습니다.
* **키 접근**: 이제 키는 편집 화면에서만 접근할 수 있으며, 여기에서 [신세틱스 작업 관리자](/docs/synthetics/synthetic-monitoring/private-locations/install-job-manager) 설정에 맞게 키를 복사할 수 있습니다.

<img title="Private locations list view" alt="Screenshot showing the private locations list without the key column" src="/images/synthetic_screenshot-crop_private-location-list.webp" />

<figcaption>
  <DNT>**[one.newrelic.com &gt; Synthetic monitoring &gt; Private locations](https://one.newrelic.com/synthetics-nerdlets/private-location-list)**</DNT>: 패드 위치 목록에 더 이상 키 열이 표시되지 않습니다. 키는 편집 메뉴를 통해 접근할 수 있습니다.
</figcaption>

## 귀하의 스테이션 위치 키에 접속하세요 [#access-key]

귀하의 구역 위치 키에 접근하려면:

1. <DNT>**[one.newrelic.com &gt; Synthetic monitoring &gt; Private locations](https://one.newrelic.com/synthetics-nerdlets/private-location-list)**</DNT> 으)로 이동합니다.
2. 열쇠가 필요한 지역 위치를 찾으세요.
3. 점 세 개 아이콘을 클릭하세요 <Icon name="fe-more-horizontal" /> 위치 바로 옆.
4. <DNT>**Edit**</DNT> 선택합니다.
5. 편집 화면에서 신세틱스 작업 관리자 설정의 <DNT>**Private location key**</DNT> 필드에서 키를 복사하세요.

<img title="Private location edit screen with key" alt="Screenshot showing the Edit screen where you can access and copy the private location key" src="/images/synthetic_screenshot-crop_private-location-edit-page.webp" />

<figcaption>
  편집 화면은 사용자 로케이션 키를 보고 복사할 수 있는 유일한 곳입니다.
</figcaption>

## 필터링 및 검색 [#filter-locations]

키 필터가 제거되었습니다. 이제 다음과 같은 방법으로 필터링 및 검색할 수 있습니다.

* **위치명**: 고립로케이션에 지정한 이름으로 검색하세요.
* **상태**: 활성 또는 비활성 위치로 필터링합니다.
* **[VSE(검증된 스크립트 실행)](/docs/synthetics/synthetic-monitoring/private-locations/verified-script-execution-private-locations)**: VSE가 활성화된 위치로 필터링합니다.

<figcaption>
  항체 위치 키 필터가 UI 에서 제거되었습니다.
</figcaption>
