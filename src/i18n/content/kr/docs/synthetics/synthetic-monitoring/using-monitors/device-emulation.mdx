장치 에뮬레이션은 단순하고 스크립팅된 브라우저 모니터에 사용할 수 있는 선택적 설정입니다. 이 설정은 Chrome DevTools의 에뮬레이션 기능을 사용하여 모바일 또는 태블릿 장치의 뷰포트 및 기능을 시뮬레이션합니다.

장치 에뮬레이션을 사용하면 시뮬레이션된 일반 모바일 또는 태블릿 장치를 사용하여 브라우저 기반 응용 프로그램을 테스트할 수 있습니다. 명심하십시오:

* 이 설정은 합성 모니터를 실행하는 데 사용되는 네트워크, 사용자 에이전트 또는 장치 유형을 변경하지 않습니다.
* 장치 에뮬레이션에서는 모바일 응용 프로그램을 설치하고 테스트할 수 없습니다.

## 요구 사항 [#requirements]

* Chrome 100+ 런타임만
* 개인 위치는 [작업 관리자](/docs/synthetics/synthetic-monitoring/private-locations/job-manager-transition-guide) 버전 225 이상 및 synthetics-node-browser-runtime 버전 1.1.98을 사용해야 합니다. 또는 더 높게
