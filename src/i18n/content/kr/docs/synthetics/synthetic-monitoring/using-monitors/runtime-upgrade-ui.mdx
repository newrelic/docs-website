New Relic is [ending support for older runtimes](/eol/2026/05/eol-05-18-26-node-16-chrome-134-runtimes/) in 2026. To simplify migrating synthetic monitors to the latest supported runtimes, we auto-upgraded every monitor that didn&apos;t require your action. Use the runtime upgrade tool to view the remaining monitors that still need your attention.

## 남은 모니터를 업그레이드하십시오 [#upgrade-options]

주의가 필요한 각 모니터에 대해, 두 가지 경로 중 하나를 선택할 수 있습니다.

<Tabs>
  <TabsBar>
    <TabsBarItem id="fix-manually">
      수동 해결
    </TabsBarItem>

    <TabsBarItem id="fix-with-ai">
      AI로 수정
    </TabsBarItem>
  </TabsBar>

  <TabsPages>
    <TabsPageItem id="fix-manually">
      모니터 스크립트를 수동으로 작성하거나 편집하여 모니터를 업데이트하려면, 다음 단계를 따르십시오.

      1. <DNT>**[one.newrelic.com](https://one.newrelic.com)** &gt; **Synthetic Monitoring** &gt; **Runtime Upgrades**</DNT> 으)로 이동합니다.

      2. 모니터를 찾아 <DNT>**Fix manually**</DNT>을(를) 클릭합니다.

      3. <DNT>**Fix script manually**</DNT> 창에서 새 런타임 환경에 맞게 스크립트 코드를 업데이트합니다.

      4. <DNT>**Close and start validation**</DNT>을 클릭합니다. 시스템이 업데이트된 스크립트를 검증하는 동안 모니터가 <DNT>**Monitors we&apos;re validating**</DNT> 테이블로 이동합니다.

         * 검증에 실패하면 모니터 옆의 **⋯** 을 클릭하고, 메뉴에서 <DNT>**Fix manually**</DNT> 을(를) 선택하고 스크립트를 업데이트한 다음 <DNT>**Close and start validation**</DNT> 을(를) 다시 클릭합니다.

      5. 유효성 검사가 성공하고 상태가 <DNT>**Fixed**</DNT>(으)로 표시되면 <DNT>**Ready to update**</DNT> 탭을 클릭합니다.

      6. 검증된 모니터 옆의 확인란을 선택하고 <DNT>**Upgrade selected**</DNT>을(를) 클릭합니다.

      완료되면 확인 공지가 나타나고 모니터가 <DNT>**Updated**</DNT> 탭으로 이동하여 성공적인 업그레이드를 확인할 수 있습니다.
    </TabsPageItem>

    <TabsPageItem id="fix-with-ai">
      <Callout title="메모">
        * 이 기능은 Advanced 컴퓨트가 필요하며 <DNT>**FedRAMP**</DNT> 계정에서는 사용할 수 없습니다. 지원을 받으려면 뉴렐릭 계정 팀에 문의하십시오.
        * 프로덕션 환경에 모니터를 배포하기 전에 항상 AI가 생성한 스크립트를 테스트하십시오.
      </Callout>

      Use this option to let <DNT>**New Relic AI (NRAI)**</DNT> generate an updated script for you. The AI-assisted script generation is mainly geared toward fixing `timeout errors` incurred during runtime upgrades. For monitors with `script errors`, the script potentially needs other changes. You can make those manually or use <DNT>[SRE Agent](/docs/agentic-ai/sre-agent/overview/)</DNT> to debug and get insights on potential script-related updates.

      AI를 사용하여 모니터를 업데이트하려면:

      1. <DNT>**[one.newrelic.com](https://one.newrelic.com)** &gt; **Synthetic Monitoring** &gt; **Runtime Upgrades**</DNT> 으)로 이동합니다.
      2. 모니터를 찾아 <DNT>**Fix with AI**</DNT>을(를) 클릭합니다.
      3. AI가 제안한 스크립트를 검토하세요. 원본 코드에 적용된 특정 변경 사항을 보려면 <DNT>**Show comparison**</DNT> 을(를) 클릭하여 나란히 표시되는 다른 뷰를 엽니다.
      4. <DNT>**Continue to validation**</DNT>을 클릭합니다. 시스템은 새 스크립트를 테스트하고 오류 없이 실행되는지 확인하기 위해 모니터를 일시적으로 검증 상태로 둡니다.
      5. 스크립트가 검증된 후 <DNT>**Ready to update**</DNT> 탭을 클릭합니다.
      6. 검증된 모니터 옆의 확인란을 선택하고 <DNT>**Upgrade selected**</DNT>을(를) 클릭합니다.

      완료되면 확인 공지가 나타나고 모니터가 <DNT>**Updated**</DNT> 탭으로 이동하여 성공적인 업그레이드를 확인할 수 있습니다.
    </TabsPageItem>
  </TabsPages>
</Tabs>

또한 여러 모니터를 병렬로 검증하고 팀원들과 링크를 공유하여 작업을 분담할 수 있습니다. 모든 스크립트가 검증되면, 준비가 되었을 때 모니터를 프로덕션 환경에 배포할 수 있습니다.
