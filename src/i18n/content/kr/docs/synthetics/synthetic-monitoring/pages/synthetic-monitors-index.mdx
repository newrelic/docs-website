종합 모니터링 도구에서 모니터 인덱스는 New Relic 계정과 연결된 모든 모니터를 나열하고 지난 24시간 동안 각 모니터의 성능에 대한 빠른 스냅샷을 제공합니다. [**요약** 페이지](/docs/synthetics/new-relic-synthetics/dashboards/synthetics-overview-dashboard-view-monitors-performance) 를 보고 시간 경과에 따른 성능에 대한 더 깊은 통찰력을 얻으려면 개별 모니터를 선택하십시오. 또는 [목록을 필터링하여](#search-monitors) 유사한 모니터의 성능을 빠르게 비교할 수 있습니다.

## 모니터 인덱스 보기 [#viewing]

모니터의 색인(또는 목록)에 액세스하려면 **[one.newrelic.com](https://one.newrelic.com) > Explorer > Synthetic monitors** 로 이동하십시오. [Explorer](/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-explorer-view-performance-across-apps-services-hosts/) 를 사용하여 모든 [엔터티](/docs/new-relic-one/use-new-relic-one/core-concepts/what-entity-new-relic/) , 즉 합성 모니터에서 애플리케이션, 호스트 또는 요소의 사용자 지정 그룹에 이르기까지 데이터를 보고하는 식별 가능한 모든 항목에 액세스합니다.

또는 [**one.newrelic.com**](http://one.newrelic.com/) **> Synthetics** 으로 이동할 수 있습니다.

## 모니터 메트릭 이해 [#understanding]

모니터 인덱스를 사용하여 모니터에 액세스하고 모니터 성능의 빠른 스냅샷을 봅니다. 인덱스에는 다음 측정항목이 포함됩니다.

* [**경고 상태**](/docs/alerts/new-relic-alerts/configuring-alert-policies/identify-entities-without-alert-policies) : 모니터의 경고 상태를 나타냅니다.

  * 녹색: 미결 위반 없음
  * 빨간색: 심각한 위반 진행 중
  * 회색: 경고로 정의된 경고 조건 없음

* **모니터 상태** : **Mute** 또는 **Disabled** 와 같은 상태가 모니터에 적용되었음을 나타냅니다.

* **성공률** : 성공한 모니터 검사의 백분율입니다. 모든 단계를 완료하지 않는 다단계 모니터는 실패로 간주됩니다.

* **[실패한 위치](/docs/synthetics/new-relic-synthetics/using-monitors/adding-editing-monitors#setting-location)** : 주어진 기간 동안 실패한 위치의 수입니다.

* **[기간](/docs/synthetics/new-relic-synthetics/using-monitors/adding-editing-monitors#setting-frequency)** : 모니터 검사가 실행되는 빈도입니다.

* **[모니터 종류](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors#setting-type)** : 선택한 모니터 종류입니다.
