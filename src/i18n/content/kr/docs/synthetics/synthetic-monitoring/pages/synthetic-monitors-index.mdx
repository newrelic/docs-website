신세틱 모니터링 도구에서 모니터 인덱스는 종합 모니터를 나열하고 지난 24시간 동안의 각 모니터 성능에 대한 스냅샷을 제공합니다. 특정 모니터를 선택하여 [<DNT>**Summary**</DNT> 페이지를](/docs/synthetics/new-relic-synthetics/dashboards/synthetics-overview-dashboard-view-monitors-performance) 보고 해당 성능에 대해 더 자세히 알아보세요. 또는 [목록을 필터링하여](#search-monitors) 유사한 모니터의 성능을 빠르게 비교할 수 있습니다.

## 모니터 인덱스 보기 [#viewing]

합성 모니터 목록을 보려면 <DNT>**[one.newrelic.com &gt; Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)**</DNT> 으로 이동하세요.

## 모니터 메트릭 이해 [#understanding]

모니터 인덱스를 사용하여 모니터에 액세스하고 모니터 성능의 스냅샷을 봅니다. 인덱스에는 다음 메트릭이 포함됩니다.

* [<DNT>**Alert status**</DNT>](/docs/alerts/new-relic-alerts/configuring-alert-policies/identify-entities-without-alert-policies): 모니터의 알림 상태를 나타냅니다.

  * 녹색 : 미공개 공지사항 없음
  * 빨간색: 중요 알림 이벤트 진행 중
  * 회색: 경고로 정의된 경고 조건 없음

* <DNT>**Monitor status**</DNT>: <DNT>**Mute**</DNT> 또는 <DNT>**Disabled**</DNT> 과 같은 상태가 모니터에 적용되었음을 나타냅니다.

* <DNT>**Success rate**</DNT>모니터링 점검 중 성공으로 끝나는 비율. 모든 단계를 완료하지 못한 다단계 모니터링 시스템은 실패로 간주됩니다.

* <DNT>**[Locations failing](/docs/synthetics/new-relic-synthetics/using-monitors/adding-editing-monitors#setting-location)**</DNT>주어진 기간 동안 오류가 발생한 위치의 수입니다.

* <DNT>**[Period](/docs/synthetics/new-relic-synthetics/using-monitors/adding-editing-monitors#setting-frequency)**</DNT>모니터링 점검이 실행되는 빈도.

* <DNT>**[Monitor type](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors#setting-type)**</DNT>선택된 모니터 유형입니다.
