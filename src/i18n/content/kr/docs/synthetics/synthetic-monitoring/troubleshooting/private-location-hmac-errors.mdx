## 문제

개인 미니언 중 하나가 확인에 실패했으며 다음 오류 중 하나를 보고합니다.

```
HMAC MISMATCH ERROR: HMAC mismatch between saved HMAC and calculated HMAC at job run time
```

```
HMAC MISMATCH ERROR: HMAC expected but missing from Job metadata
```

이러한 오류는 [확인된 스크립트 실행](/docs/synthetics/new-relic-synthetics/private-locations/verified-script-execution-private-locations) 오류를 나타냅니다.

## 해결책

이 검사가 실패한 위치에 대한 올바른 암호를 입력해야 합니다.

비밀번호를 입력하려면:

1. **[one.newrelic.com > Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)** 에서 실패한 모니터를 선택합니다.
2. **Settings > Script** 로 이동합니다.
3. 위치 비밀번호 필드에 올바른 비밀번호를 제출하십시오.

<Callout variant="tip">
  New Relic은 개인 위치 비밀번호를 저장하지 않습니다. 암호를 모르는 경우 팀에서 개인 위치를 구성한 사람을 찾으십시오. 암호를 복구할 수 없는 경우 유일한 옵션은 새 개인 위치를 만드는 것입니다.
</Callout>
