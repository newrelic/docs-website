새 버전의 크롬을 지원하기 위해 업데이트된 런타임이 실행됩니다. 결과적으로 방화벽 허용 목록, 로봇 감지 소프트웨어 및 기타 관련 애플리케이션을 재구성하여 새 IP 주소 범위에서 네트워크 요청을 승인해야 합니다.

새 런타임 실행 후 작성된 모니터는 새 IP 주소 공간을 사용합니다. 기존 모니터는 향후 새 IP 주소 공간으로 옮겨질 예정이다. IP 주소 및 합성 모니터링에 대한 자세한 정보는 [Synthetic 모니터 public minion IPs](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/) 를 확인하십시오.

## 공용 미니이온에 대한 새 IP 주소 범위 [#us_new_ip]
