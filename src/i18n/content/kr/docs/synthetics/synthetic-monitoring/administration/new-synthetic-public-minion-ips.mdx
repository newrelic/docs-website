추가 [소스 IP 주소 범위](#us_new_ip) 는 합성의 지속적인 성장을 지원하기 위해 2022년 10월 3일에 추가됩니다. 결과적으로 방화벽의 허용 목록, 봇 탐지 소프트웨어 및 기타 관련 애플리케이션을 재구성하여 추가 IP 주소 범위의 네트워크 요청을 수락해야 할 수 있습니다. [합성 모니터 공개 미니언 IP 문서](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/) 의 IP 및 IP 범위 JSON 목록과 함께 이러한 [새 IP 범위](#us_new_ip) 를 구성에 포함해야 합니다.

## 공용 미니이온에 대한 새 IP 주소 범위 [#us_new_ip]

추가 IP 주소 범위는 2022년 10월 3일에 추가됩니다. 이 범위는 미국 및 EU 데이터 센터를 모두 포함합니다. 이는 기존 [IP 범위 JSON 목록](/docs/synthetics/synthetic-monitoring/administration/synthetic-public-minion-ips/#dynamic-ip-addresses) 에 추가됩니다. JSON 목록은 사용되기 전에 이러한 추가 IP 범위를 포함하도록 업데이트됩니다.
