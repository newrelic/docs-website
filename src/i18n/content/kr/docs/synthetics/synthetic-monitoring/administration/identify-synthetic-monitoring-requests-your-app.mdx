[합성 모니터](/docs/synthetics/new-relic-synthetics/using-monitors/add-edit-monitors) 요청에는 `X-Abuse-Info` 요청 헤더 필드가 포함되어 있으므로 New Relic 합성 모니터링 계정과 이를 보낸 모니터를 식별할 수 있습니다. 해당 정보를 사용하여:

* 모니터 결과에 액세스합니다(계정에 대한 액세스 권한이 있는 경우).
* 모니터를 비활성화합니다(계정에 대한 액세스 권한이 있는 경우).

## 계정 식별 또는 모니터링 [#identify-monitor]

요청을 보낸 합성 계정 및 모니터 ID를 식별하려면 `X-Abuse-Info` 요청 헤더 필드를 확인하십시오. 예를 들어:

```
X-Abuse-Info: Request sent by a synthetic monitor (https://docs.newrelic.com/docs/synthetics/new-relic-synthetics/administration/identify-synthetics-requests-your-app) - monitor id: ${MONITOR_ID_STRING} | account id: ${ACCOUNT_NUMBER}
```

## 모니터 액세스 또는 비활성화 [#enable-disable]

모니터 요청이 발생한 계정에 대한 액세스 권한이 있는 경우 모니터 결과를 보거나 New Relic UI에서 모니터를 비활성화할 수 있습니다.

1. 요청을 보낸 계정 번호와 모니터 ID를 [식별](#identify-monitor) 합니다.
2. **[one.newrelic.com > Synthetic monitoring](https://one.newrelic.com/synthetics-nerdlets)** 으로 이동하여 모니터를 선택합니다.
3. **설정** 에서 **일반** 을 클릭합니다.
4. **모니터** 토글을 클릭하여 **비활성화** 로 설정합니다.
