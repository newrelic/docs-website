쿼리는 필요한 정보를 찾는 데 도움이 되며 이러한 쿼리를 기반으로 하는 차트와 그래프는 데이터를 유용하게 만들어 줍니다. 차트와 대시보드는 데이터 시각화의 핵심이며, 시각적으로 매력적이고 쉽게 이해할 수 있는 데이터 스트림을 만들 수 있는 여러 옵션을 제공합니다. NRQL 쿼리가 차트 및 대시보드로 연결되는 방식을 이해하면 데이터를 최대한 활용할 수 있습니다.

NRQL 쿼리 작성을 완료하려면 마지막 단계로 차트를 생성해야 합니다. 뉴렐릭은 데이터를 시각화하는 완벽한 방법을 찾는 데 도움이 되는 [여러 옵션](/docs/query-your-data/explore-query-data/use-charts/chart-types)을 제공하며, 차트를 만든 후에는 차트로 많은 작업을 수행할 수 있습니다. NRQL 쿼리를 작성한 후 차트를 사용하여 다음을 수행할 수 있습니다.

* 차트를 [새 대시보드](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards#dashboards-create)의 시작으로 사용하여 단일 위치에서 여러 NRQL 쿼리를 모니터링할 수 있습니다.
* 기존 대시보드에 차트를 추가하여 모니터링에 다른 NRQL 쿼리를 추가[할 수 있습니다.](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#add-pages-edit)
* 퍼머링크를 복사하거나 PDF(또는 해당되는 경우 CSV)로 다운로드하여 다른 사람과 [차트를 공유](/docs/query-your-data/explore-query-data/use-charts/use-your-charts#share-charts)하여 다른 사람이 쿼리의 데이터를 볼 수 있도록 할 수 있습니다.
* 차트의 NRQL 쿼리를 수정하여 표시되는 데이터를 미세 조정할 수 있습니다.
* NRQL 쿼리로 [알림 생성하여](/docs/alerts-applied-intelligence/new-relic-alerts/get-started/your-first-nrql-condition#create-simple-alert) 문제가 발생할 때 사전에 알림을 받을 수 있습니다.

차트는 그 자체로도 유용하지만 여러 차트를 컴파일하여 대시보드를 만들면 훨씬 더 유용해집니다.

## 대시보드 [#dashboards]

<img
  title="Dashboard screenshot.png"
  alt="A screenshot example of a dashboard"
  src={dashboardsDashboardUi}
/>

<figcaption>
  원형 및 막대 차트 형식으로 데이터를 나타내는 대시보드
</figcaption>

차트는 개별 요소에 대한 보고에 적합한 반면, 대시보드는 데이터를 훨씬 더 광범위하게 볼 수 있는 가장 좋은 방법입니다. 차트가 퍼즐의 조각이라면 대시보드는 완성된 퍼즐입니다. 대시보드는 자유롭게 구성이 가능하며 이를 사용해 원하는 스트림을 모두 표시할 수 있습니다. 또한 다음을 수행할 수 있습니다.

* 쿼리 빌더를 사용하여 대시보드에 [차트 추가](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard#dash-create-chart)
* 대시보드에 고유한 [텍스트, 이미지 또는 링크](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard#markdown)(Markdown 사용) 추가
* 가장 중요한 데이터가 원하는 위치에 있는지 확인하기 위해 대시보드 내에서 차트 이동
* 이메일로 대시보드를 [공유](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data#dashboards)하거나 대시보드에 대한 영구 링크 생성
* 대시보드를 JSON으로 [가져오기](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-import)또는 [내보내기](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard#dash-json) 또는 대시보드를 PDF로 내보내기
* 대시보드를 [복제](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard#dash-clone)하여 완전히 새로운 대시보드의 기반으로 사용
* 대시보드 구성 요소를 [수집](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard#add-pages-edit)하여 새 데이터 스트림 추가, 기존 데이터 스트림 수정, 더 이상 관련이 없는 이전 데이터 스트림 제거
