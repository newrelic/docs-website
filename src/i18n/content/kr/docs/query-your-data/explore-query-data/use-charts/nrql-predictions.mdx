<DNT>**NRQL Predictions**</DNT> 뉴렐릭에서는 이력 데이터를 사용하여 미래 추세를 예측하고 인사이트 지표가 미래에 어떻게 행동할 것인지를 제공합니다. 이러한 사전 예방적 접근 방식은 추세를 시각화하고 잠재적인 문제를 예상하는 데 도움이 되며, 적절한 개입을 통해 최적의 시스템 성능을 유지할 수 있습니다. <DNT>**NRQL Predictions**</DNT> 자체는 잠재적인 문제를 식별하지 못하지만 추세를 평가하고 미래의 과제를 계획하는 데 필요한 정보를 제공합니다.

잠재적인 문제를 자동으로 식별하려면 현재 공개 미리 보기로 제공되는 <DNT>**[Predictive Alerts](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions)**</DNT> 기능에 등록하세요. <DNT>**Predictive Alerts**</DNT> <DNT>**NRQL Predictions**</DNT> 와 함께 작동하여 예상 추세가 잠재적인 문제를 나타낼 때 자동으로 알림을 트리거하여 시스템을 사전에 관리할 수 있도록 합니다.

<Callout variant="tip">
  뉴렐릭은 현재 [Holt-Winters 모델을](https://en.wikipedia.org/wiki/Exponential_smoothing#Triple_exponential_smoothing_\(Holt_Winters\)) 사용하여 예측을 수행하고 있으며, 이는 데이터의 시간별, 일별, 주별 계절적 패턴과 추세를 지원합니다.
</Callout>

## 가격 [#pricing]

<DNT>**NRQL Predictions**</DNT> 의 사용은 귀하의 계정과 관련된 가격 모델에 적용되는 주문에 따라 미리 보기 중에 청구될 수 있습니다. 이 기능의 사용과 관련된 뉴렐릭 쿼리 언어 쿼리(NRQLS)는 핵심 컴퓨트 제품으로 청구됩니다. 이 기능이 일반적으로 제공되면, 귀하의 주문에 따라 사용료가 청구됩니다.

가격에 대한 자세한 내용은 당사 영업팀에 문의하시거나 주문서를 참조하세요.

## 선 또는 영역 차트에 예측 추가 [#add-prediction]

선 또는 영역 차트에 예측을 추가하려면 다음 단계를 따르세요.

<Steps>
  <Step>
    ### 라인 또는 영역 차트에 액세스 [#access-a-chart]

    1. [one.newrelic.com &gt; All capabilities](https://one.newrelic.com/all-capabilities) 으로 이동하세요.
    2. 선형 또는 영역형 차트에 예측을 추가하려면 <DNT>**Dashboards**</DNT> 에서 선택하거나 <DNT>**Metrics and events**</DNT> 에서 엽니다.
  </Step>

  <Step>
    ### 예측 추가 [#add-prediction]

    예측을 추가하려면 <Icon name="fe-more-horizontal" /> 메뉴에서 <DNT>**Predict trend**</DNT> 선택하세요. **Data Explorer** 에서는 예측 추세가 차트에 표시되어 미래 추세를 예측합니다. 예측 범위는 자동으로 쿼리 창의 20%로 설정되고 회색으로 강조 표시됩니다. 그래프는 예측 범위 내에서 점선으로 나타납니다.

    `PREDICT` 키워드가 차트 쿼리에 추가됩니다. 귀하의 요구 사항에 맞게 [쿼리를 구체화](#customize-prediction-range) 하고 실행하여 최신 추세를 얻을 수 있습니다.

    <Callout variant="tip">
      영역형 차트에 예측을 추가하면 차트 유형이 예측이 포함된 선형 차트로 변경됩니다.
    </Callout>
  </Step>

  <Step>
    ### 대시보드에 추가 [#add-to-dashboard]

    1. 예측이 포함된 차트를 대시보드에 추가하려면 <DNT>**Add to dashboard**</DNT> 클릭하세요.
    2. 위젯 이름을 입력하세요.
    3. 위젯을 추가할 대시보드를 선택하거나 해당 대시보드에 대한 새 대시보드를 만듭니다.

    이제 선택한 대시보드에서 예측 차트를 볼 수 있습니다.
  </Step>
</Steps>

## 기본 예측 범위 [#default-prediction-range]

차트에 예측을 추가하면 `PREDICT` 함수는 기존 차트 쿼리에 다음과 같은 기본 동작을 추가합니다.

* **예측 범위**: 쿼리 창에 지정된 총 시간의 20%를 예상하여 현재 추세에 따른 단기 예측을 확인할 수 있습니다.
* **이력 데이터**: 현재 쿼리 창과 이전 두 창의 이력 데이터를 처리하여 예측을 생성합니다.
* **시간 간격**: 쿼리 창에서 예측의 데이터 포인트 간격을 일치시켜 데이터 투영의 일관성을 보장합니다.

예시:

```sql
FROM Transaction SELECT count(*) WHERE error IS TRUE TIMESERIES PREDICT
```

## 예측 범위 사용자 정의 [#customize-prediction-range]

다음과 같은 방법으로 예측을 사용자 지정할 수 있습니다. NRQL에 대해 알아보려면 [NRQL 설명서를](/docs/nrql/get-started/introduction-nrql-new-relics-query-language) 참조하세요.

<CollapserGroup>
  <Collapser id="hyperparameters" title="하이퍼파라미터와 평활화 수준을 사용하여 쿼리를 사용자 정의합니다.">
    현재 <DNT>**Holt-Winters**</DNT> 이 우리가 지원하는 유일한 예측 모델이므로 `holtwinters` 사용하여 하이퍼파라미터와 평활화 요소를 조정하여 사용 사례에 맞게 예측 모델을 미세 조정할 수 있습니다.

    다음 하이퍼파라미터를 사용하여 예측을 사용자 지정하세요.

    * **`seasonality`**: 시계열의 계절 패턴의 길이를 나타냅니다.

      * **`seasonality: <time range>`**: 지정된 시간 범위에 대한 시계열의 계절성을 나타냅니다.
      * **`seasonality: AUTO`**: 시계열의 계절성을 자동으로 감지합니다.
      * **`seasonality: NONE`**: 시계열의 계절성을 비활성화합니다.

    * **`alpha`**: 시계열의 수준 또는 평균값을 나타냅니다.

    * **`beta`**: 시계열의 추세나 기울기를 나타내며 상승 또는 하락 움직임을 나타냅니다.

    * **`gamma`**: 시계열의 계절적 또는 반복적 패턴을 나타냅니다.

    * **`phi`**: 시계열 추세의 감쇠 계수를 나타냅니다.

    **`alpha`**, **`beta`**, **`gamma`**, **`phi`** 에 대한 평활화 계수를 0～1까지 설정할 수 있습니다. 값이 높을수록 최근 데이터에 더 많은 가중치를 두고, 값이 낮을수록 이력 데이터에 더 많은 가중치를 둡니다.
  </Collapser>

  <Collapser id="prediction-range" title="예측 범위에 따라 쿼리를 사용자 정의합니다.">
    지정된 시간 범위까지 예측을 투영하여 쿼리를 사용자 정의할 수 있습니다. `BY` 키워드를 사용하여 지정된 시간 범위까지 예측을 투영합니다. 예를 들어, `PREDICT BY 3 hours`.
  </Collapser>

  <Collapser id="historical-data-range" title="지정된 시간 범위에 대한 이력 데이터를 처리하여 쿼리를 사용자 정의합니다.">
    정의된 시간 범위에서 이력 데이터를 처리하여 예측을 투영하여 쿼리를 사용자 정의할 수 있습니다. 이력 데이터를 처리하려면 `USING` 키워드를 사용하세요. 예를 들어, `PREDICT USING 1 day`.
  </Collapser>
</CollapserGroup>

예시:

```sql
FROM Transaction SELECT count(*) WHERE error IS TRUE TIMESERIES PREDICT holtwinters(beta: 0.5) BY 3 hours USING 1 day
```
