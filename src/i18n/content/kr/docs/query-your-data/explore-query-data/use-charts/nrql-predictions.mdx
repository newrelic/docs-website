<DNT>**NRQL Predictions**</DNT> 뉴렐릭에서는 지표의 이력 데이터를 사용하여 미래 추세를 예측하고 인사이트에서 지표가 미래에 어떻게 행동할지 알려줍니다. 이러한 사전 예방적 접근 방식은 추세를 시각화하고 잠재적인 문제를 예상하는 데 도움이 되며, 적절한 개입을 통해 최적의 시스템 성능을 유지할 수 있습니다. <DNT>**NRQL Predictions**</DNT> 자체는 잠재적인 문제를 식별하지 못하지만 추세를 평가하고 미래의 과제를 계획하는 데 필요한 정보를 제공합니다.

잠재적인 문제를 자동으로 식별하려면 현재 공개 미리 보기로 제공되는 <DNT>**[Predictive Alerts](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions)**</DNT> 기능에 등록하세요. <DNT>**Predictive Alerts**</DNT> <DNT>**NRQL Predictions**</DNT> 와 함께 작동하여 예상 추세가 잠재적인 문제를 나타낼 때 자동으로 알림을 트리거하여 시스템을 사전에 관리할 수 있도록 합니다.

## 가격 [#pricing]

<DNT>**NRQL Predictions**</DNT> 의 사용은 귀하의 계정과 관련된 가격 모델에 적용되는 주문에 따라 미리 보기 중에 청구될 수 있습니다. 이 기능의 사용과 관련된 뉴렐릭 쿼리 언어 쿼리(NRQLS)는 핵심 컴퓨트 제품으로 청구됩니다. 이 기능이 일반적으로 제공되면, 귀하의 주문에 따라 사용료가 청구됩니다.

가격에 대한 자세한 내용은 당사 영업팀에 문의하시거나 주문서를 참조하세요.

## 지원되는 예측 알고리즘 [#supported-algorithm]

현재 <DNT>**NRQL predictions**</DNT> 계절 및 비계절 시계열 모두에 대해 <DNT>**Holt-Winters**</DNT> 알고리즘만 지원합니다. 이는 예측 및 예측 작업에 일반적으로 사용되는 표준 알고리즘입니다. 우리는 Holt-Winters 모델에 대해 시간별, 일별, 주별 계절 길이를 지원합니다.

## 기본 동작 [#defaults]

쿼리의 `PREDICT` 절에는 다음과 같은 기본 동작이 적용됩니다.

* **계절성**: 데이터의 계절성을 자동으로 감지하고 동일한 계절 길이를 사용하여 예측 모델을 구성합니다. 계절성이 발견되지 않으면 비계절 모델을 구성합니다.
* **예측 창**: 쿼리 창에 지정된 총 시간 범위의 20%를 예상합니다.
* **학습용 이력 데이터**: 현재 쿼리 창과 이전 두 창의 이력 데이터를 사용하여 예측을 생성합니다.
* **시간 간격**: 데이터 투영의 일관성을 보장하기 위해 쿼리 창에서 시계열의 데이터 포인트 간격에 맞춥니다.

NRQL 쿼리에서 `PREDICT` 사용하는 예:

```sql
FROM Transaction SELECT count(*) WHERE error IS TRUE TIMESERIES PREDICT
```

학습 모델의 하이퍼파라미터를 수동으로 설정하고 `USING` 및 `BY` 키워드를 추가하여 예측을 사용자 정의할 수 있습니다. NRQL에 대해 알아보려면 [NRQL 설명서를](/docs/nrql/get-started/introduction-nrql-new-relics-query-language) 참조하세요.

    `USING` 키워드를 사용하여 모델을 학습하는 데 사용할 역량 데이터의 양을 지정합니다. 예를 들어, `PREDICT USING 1 day` 이전 날의 데이터와 함께 쿼리 창을 사용하여 모델을 학습합니다. 시간 범위가 `integer unit` 시간인지 확인하세요.
  </Collapser>

`PREDICT` 하이퍼파라미터, `BY` 및 `USING` 키워드와 함께 사용하는 예:

```sql
FROM Transaction SELECT count(*) WHERE error IS TRUE TIMESERIES PREDICT holtwinters(seasonality: AUTO, alpha: 0.2) BY 1 hour USING 2 hours
```

이 쿼리는 계절성을 자동 감지로 설정하고, 레벨 평활화 요소를 조정하여 이력데이터를 강조합니다. 이 기능은 1시간 뒤의 미래를 예측하고 이전 2시간의 이력 데이터와 쿼리 창 길이를 사용하여 모델을 학습합니다. 차트에는 사용자 정의된 예측이 다음과 같이 표시됩니다.
