쿼리 빌더를 사용하면 데이터 쿼리를 실행하여 사용자 지정 차트 및 기타 시각화를 생성할 수 있습니다. 여러 차트가 포함된 사용자 정의 [대시보드](/docs/dashboards/manage-your-dashboard/manage-your-dashboard#dash-create-chart) 를 작성할 수도 있습니다.

## 중요한 이유

현대 비즈니스 시스템은 데이터 생성 요소의 복잡한 미로입니다. 일상적인 작업 활동에서 흐르는 끝없는 데이터 흐름을 가져오고, 분석하고, 시각화하는 쉽고 확실한 방법이 필요합니다.

쿼리 빌더를 사용하여 다음을 수행합니다.

* 데이터에 빠르게 액세스하고 맞춤형 차트를 작성하여 인프라, 애플리케이션 및 기타 서비스의 상태를 배우고 이해할 수 있습니다.
* 대시보드에 차트를 추가하여 시스템 상태에 대한 완전한 실시간 보기를 얻으십시오.
* 단 두 번의 클릭으로 동료 또는 사용자와 차트를 공유할 수 있습니다.
* 기본 모드를 사용하여 쿼리에 익숙해진 다음 고급 모드로 전환하여 NRQL을 사용하여 차트를 세분화합니다.
* 빌드하고 실행하는 쿼리에서 NRQL 경고를 만듭니다.

## 시작하다 [#get-started]

쿼리하기 전에 쿼리 [소개](/docs/telemetry-data-platform/ingest-manage-data/understand-data/introduction-querying-new-relic-data/) 및 [데이터 탐색](/docs/query-your-data/explore-query-data/explore-data/introduction-data-explorer/) 방법을 읽을 수 있습니다.

쿼리 작성기를 찾으려면 [one.newrelic.com](https://one.newrelic.com) 으로 이동 **하여 데이터 쿼리** 를 클릭하십시오.

## 쿼리 빌더 사용: 기본 및 고급 모드 [#specify-data]

쿼리 빌더에는 다음과 같은 여러 모드가 있습니다.

* [기본 모드](/docs/chart-builder/use-chart-builder/choose-data/use-basic-mode-specify-data)
* [고급(NRQL) 모드](/docs/chart-builder/use-chart-builder/choose-data/use-advanced-nrql-mode-specify-data)
* [고급(PromQL 스타일) 모드](#chart-builder-promql)

<Callout variant="tip">
  또한 언제든지**[데이터 탐색기](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer)** 탭으로 전환하여 쿼리를 수행하지 않고도 데이터를 탐색하고 차트를 시각적으로 만들 수 있습니다.
</Callout>

    * 데이터 소스를 선택하십시오. 기본 모드에서는 이벤트를 선택한 다음 차트에 표시할 정보에 해당하는 속성을 선택하여 시작합니다.
    * 차트의 데이터를 지정(필터링)합니다. 결과 범위를 좁히고 패싯 기준으로 사용할 속성을 선택할 수 있습니다.
    * 차트에 표시할 시간 범위를 선택합니다. 메뉴에서 시간 범위를 선택하거나 사용자 지정 옵션을 사용하여 자신만의 시간 범위를 만들 수 있습니다.
  </Collapser>

다음을 위해 쿼리를 정의하는 동안 기본 모드와 고급 모드 간에 전환할 수 있습니다.

* 일부 값을 사용자 정의합니다.
* 기본 모드에서 지정한 데이터를 사용하여 NRQL 쿼리가 구성되는 방식을 알아봅니다.

<Callout variant="important">
  기본 모드에서 쿼리를 작성하고 고급 모드를 사용하여 업데이트하면 기본 모드로 돌아가서 해당 쿼리를 편집할 **수 없습니다** . 고급(NRQL) 모드에서 추가로 변경해야 합니다.
</Callout>

원하는 데이터를 지정하면 차트를 볼 수 있습니다. 입력을 변경하면 쿼리가 자동으로 기본 모드에서 실행됩니다. 고급(NRQL 또는 PromQL 스타일) 모드를 사용하는 경우 쿼리를 실행합니다.

## 차트 사용 [#customize-chart]

차트를 만든 후에는 다음을 수행할 수 있습니다.

* [차트 유형을 변경합니다](/docs/chart-builder/use-chart-builder/use-charts/use-your-charts#change-appearance) . 지정한 데이터를 기반으로 쿼리 작성기는 결과를 가장 효과적으로 표시하는 차트 유형을 선택합니다. 그러나 [사용 가능한 다른 차트 유형](/docs/insights/use-insights-ui/manage-dashboards/insights-chart-types) 중에서 선택하여 사용하려는 시각적 형식으로 데이터를 표시할 수 있습니다.
* [차트를 공유하세요](/docs/chart-builder/use-chart-builder/use-charts/use-your-charts#share-charts) .
* [대시보드에 차트를 추가합니다](/docs/chart-builder/use-chart-builder/use-charts/use-your-charts#add-to-dashboard) .
* [차트에서 NRQL 경고 조건을 만듭니다](/docs/alerts-applied-intelligence/new-relic-alerts/alert-conditions/create-nrql-alert-conditions/#create-chart) .
