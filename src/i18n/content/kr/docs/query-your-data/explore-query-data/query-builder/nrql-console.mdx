NRQL 콘솔은 현재 수행 중인 작업의 컨텍스트를 유지하면서 [New Relic](https://one.newrelic.com/) 의 어느 곳에서나 데이터를 쿼리할 수 있는 빠른 방법입니다. 서비스를 모니터링하거나, 문제를 해결하거나, 플랫폼을 탐색하는 중이든 상관없이 NRQL 콘솔을 사용하면 데이터를 탐색하여 현재 보기를 종료하지 않고도 클릭 한 번으로 보고 있는 것에 대해 더 많이 이해할 수 있습니다. 데이터 쿼리 속도를 높이십시오!

<Callout variant="tip">
  New Relic의 쿼리 언어인 NRQL이 처음이신가요? [문서에서 자세히 알아보세요!](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language/)
</Callout>

## NRQL 콘솔 사용 [#use-nrql-console]

<Callout variant="tip">
  NRQL 콘솔은 기본적으로 표시되지 않습니다. 활성화 또는 비활성화하려면 UI의 오른쪽 상단 모서리에 있는 아바타를 클릭하여 사용자 기본 설정으로 이동합니다.
</Callout>

NRQL 콘솔은 데이터 탐색을 위해 어디서나 사용할 수 있는 얇고 우아한 막대입니다. 그것을 사용하는 것은 매우 간단합니다:

1. **언제 어디서나** UI 하단에서 NRQL 콘솔을 찾으십시오. 어디에나 있다!

   * 또한 NRQL 콘솔을 사용 [하여 대시보드](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 의 모든 위젯에 있는 [**...** 차트 작업 메뉴](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-manage-charts) 에서 **쿼리 보기** 를 클릭하여 쿼리를 볼 수 있습니다.

2. 쿼리 입력: 콘솔에는 자동 완성 및 구문 강조 표시가 있습니다.

3. 쿼리를 실행하면 시각화가 표시됩니다.

   * NRQL 콘솔에서 수행하는 모든 쿼리는 [쿼리 작성기](/docs/query-your-data/explore-query-data/query-builder/introduction-query-builder/) 의 **최근 쿼리** 탭에 표시됩니다.

4. 결과를 대시보드에 저장합니다.

5. 고급 사용자 지정의 경우 쿼리 빌더에서 쿼리를 엽니다. 쿼리를 완료하고, 시각화 유형을 변경하고, y축을 구성하고, 데이터 형식을 지정하는 등의 작업을 수행할 수 있습니다.

6. 쿼리를 마치면 콘솔을 최소화하여 나중에 필요할 때를 대비하여 쿼리를 계속 유지하거나 **x** 를 눌러 지울 수 있습니다.
