NRQL 콘솔은 수행 중인 작업의 컨텍스트를 유지하면서 **[one.newrelic.com](https://one.newrelic.com/all-capabilities)** 의 어느 곳에서나 데이터를 쿼리할 수 있는 빠른 방법입니다. 서비스를 모니터링하거나, 문제를 해결하거나, 플랫폼을 탐색하는 중이든 상관없이 NRQL 콘솔을 사용하면 데이터를 탐색하여 현재 보기를 종료하지 않고도 클릭 한 번으로 보고 있는 것에 대해 더 많이 이해할 수 있습니다. 데이터 쿼리 속도를 높이십시오!

NRQL 콘솔을 활성화 또는 비활성화하려면 [사용자 메뉴](/docs/accounts/accounts-billing/general-account-settings/intro-account-settings) 로 이동하여 NRQL 콘솔 토글을 찾으십시오.
