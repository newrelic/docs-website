이제 [NRQL 형식 쿼리를](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language) 작성하는 방법을 알 필요 없이 <DNT>**query experience data explorer**</DNT> 을 사용하여 모든 데이터를 시각적으로 탐색할 수 있습니다.

좀 더 자세한 검색을 수행해야 합니까? [데이터를 쿼리하는](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data/) 방법을 읽어보세요. 아직 만들지 않으셨다면, 오늘부터 무료 뉴렐릭 계정을 만들어 데이터를 시뮬레이션해보세요.

## 데이터 탐색기의 중요성 [#importance-of-data-explorer]

시스템 상태를 조사하시나요? 자원을 계획하거나, 공지 이벤트를 파악하고 대응하거나, 잘못된 동작을 해결해야 합니까? 데이터 탐색기를 사용하면 NRQL을 사용하거나 쿼리를 작성할 필요 없이 시각적 메뉴를 통해 원하는 데이터를 쉽게 식별, 가져오고 시각화할 수 있습니다.

<DNT>**data explorer**</DNT> 사용하면 뉴렐릭 데이터베이스(NRDB)에 저장된 모든 데이터에 빠르고 직관적인 방식으로 액세스할 수 있습니다. 그런 다음 속성을 사용하여 쿼리의 측면을 선택하고 필요한 값까지 필터링할 수 있습니다.

데이터 탐색기 뷰를 사용하면 다음과 같은 작업을 수행할 수 있습니다.

* 다양한 관점에서 데이터를 확인하세요: 원시 데이터부터 진화, 분포 등에 대한 다양한 시각화 자료까지 인사이트에 대한 정보를 제공합니다.
* 필터를 사용하여 데이터를 드릴다운합니다.
* 데이터 추세에 따른 예측을 확인하세요( [NRQL 예측](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions) 공개 미리 보기에서 사용 가능).
* 대시보드에 검색 내용을 추가하세요.
* NRQL의 작동 방식을 알아보세요. 데이터 탐색기는 사용자의 선택에 따라 NRQL 쿼리를 자동으로 생성합니다.

### 탐색할 수 있는 데이터 유형 [#types-of-data]

<DNT>**Data Explorer**</DNT> 뷰를 사용하여 탐색할 수 있는 데이터 유형은 다음과 같습니다.

* [차원 지표](/docs/data-apis/understand-data/new-relic-data-types/#dimensional-metrics) (지수 데이터 유형).
* [이벤트](/docs/data-apis/understand-data/new-relic-data-types/#events-new-relic) (이벤트 데이터 유형).
* [로그](/docs/logs/ui-data/use-logs-ui) (로그 데이터 유형).
* [타임슬라이스](/docs/data-apis/understand-data/new-relic-data-types/#timeslices) (<InlinePopover type="apm" /> 에이전트, 모바일 에이전트 및 <InlinePopover type="browser" />에이전트는 이 데이터 유형을 보고합니다. 이러한 유형의 데이터를 탐색하려면 해당 에이전트 중 하나의 사진모니터를 선택해야 하며, 그러면 해당 옵션이 표시됩니다.)
* [조회](/docs/data-apis/understand-data/new-relic-data-types/#lookups) (조회 데이터 유형).

## 데이터 탐색 [#explore-your-data]

데이터 탐색기에 액세스하려면 하단 <DNT>**Query your data**</DNT> 막대로 이동하여 <DNT>**Data Explorer**</DNT> 아이콘을 클릭하세요.

### 데이터 탐색기 사용 [#use-data-explorer]

* **범위 정의**: 데이터 유형(메트릭, 이벤트 또는 로그), 계정 및 엔터티를 선택합니다.
* **시간 범위 선택**: 타임 피커를 사용하여 쿼리의 시간 범위를 정의합니다.
* **검색 구성하기**: 왼쪽 블록을 사용하여 사용 가능한 데이터를 탐색하고 검색을 구성하세요. 블록을 검색할 수 있으며, 작업 메뉴의 작업을 사용하여 쿼리를 만들고 업데이트할 수 있습니다.
* **쿼리 구체화**: 필터, 패싯, 집계를 추가하여 쿼리를 수정합니다.
* **차트 시각화 사용자 정의**: 사용 가능한 사용자 정의 옵션으로 차트 시각화를 업그레이드하세요.
* **선형 또는 영역형 차트에 예측 추가( [NRQL 예측](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions) 의 공개 미리 보기에서 사용 가능)**: **Options** \[옵션] 메뉴에서 <DNT>**Predict trend**</DNT> 사용하여 예측된 추세를 얻습니다.
* **대시보드에 추가**: 향후 참조를 위해 차트를 대시보드에 추가합니다.

### 데이터 탐색기의 구성 요소 [#data-explorer-components]

* **데이터 유형 선택**: 메트릭, 이벤트, 로그, 계정 중에서 선택하세요. 메트릭을 선택하면 엔터티별로 필터링할 수 있습니다.
* **데이터 탐색 영역**: 왼쪽에 위치한 이 영역에서는 화면에 표시된 다양한 값을 클릭하여 데이터를 탐색하거나 3개 점으로 구성된 작업 메뉴를 클릭하여 가능한 작업을 확인할 수 있습니다.
* **작업 공간**: 이 영역에는 선택 사항의 결과가 표시됩니다.
