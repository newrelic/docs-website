데이터 탐색기를 사용하면 [NRQL](/docs/query-your-data/nrql-new-relic-query-language) 지식 없이도 모든 데이터를 시각적으로 탐색할 수 있습니다. 데이터 탐색기를 사용하여 데이터에 액세스, 쿼리 및 사용자 지정하고, 시각화를 생성하고, 일관되고 선별된 경험에서 서비스 간에 연결을 만드십시오.

더 자세한 검색을 수행해야 합니까? [데이터 쿼리](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data/) 방법을 읽어보세요.

## 중요한 이유 [#why-matters]

시스템 상태를 조사합니까? 리소스를 계획하고, 사고를 식별 및 대응하거나, 잘못된 동작을 해결해야 합니까? 데이터 탐색기를 사용하면 NRQL을 사용하거나 쿼리를 작성하지 않고도 시각적 메뉴를 통해 찾고 있는 데이터를 쉽게 식별, 가져오고 시각화할 수 있습니다.

데이터 탐색기를 사용하여 다음을 수행할 수 있습니다.

* 빠르고 직관적인 방식으로 [이벤트](/docs/using-new-relic/data/understand-data/new-relic-data-types#event-data) 및 [메트릭](/docs/using-new-relic/data/understand-data/new-relic-data-types#metrics) 데이터에 액세스합니다.
* 데이터를 가시적이고 쉽게 실행할 수 있도록 하여 데이터의 차원을 활용합니다.
* 원시 데이터에서 진화, 분포 등에 대한 통찰력을 제공하는 다양한 시각화에 이르기까지 다양한 관점에서 데이터를 봅니다.
* 필터를 사용하여 데이터를 드릴다운합니다.
* 클릭 한 번으로 [대시보드](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards) 에 검색어를 추가하세요.
* [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) 작동 방식 이해: 데이터 탐색기는 사용 가능한 데이터를 탐색하는 동안 쿼리가 구축되는 방식을 보여줍니다.

데이터 탐색기에 액세스하려면 [one.newrelic.com](https://one.newrelic.com) 으로 이동한 다음

* 페이지 상단에서 **데이터 쿼리** 아이콘을 클릭합니다.

* **데이터 찾아보기** 드롭다운에서 **이벤트** 또는 **측정항목** 을 선택합니다.

데이터 탐색기는 New Relic에 있는 모든 데이터를 탐색할 수 있는 포털입니다. 데이터 탐색기 UI에서 이벤트 및 메트릭을 탐색하거나 [로그](/docs/logs/log-management/ui-data/query-syntax-logs/) 및 [추적](/docs/distributed-tracing/ui-data/understand-use-distributed-tracing-ui/) 쿼리 인터페이스를 선택하여 탐색할 수 있습니다. 탭을 클릭하여 언제든지 [쿼리 작성기](/docs/chart-builder/use-chart-builder/get-started/introduction-chart-builder) 에 액세스할 수도 있습니다.

### 데이터 탐색기 사용 [#use-explorer]

데이터 탐색기는 상단(a)의 범위 지정 섹션, 왼쪽의 데이터 탐색 영역(b), 작업 공간(c)으로 구성됩니다.

<img
  title="data_explorer_updated.png"
  alt="Data explorer view"
  src={dataExplorerUpdated}
/>

데이터 탐색기를 사용하려면:

1. 탐색 범위 정의: 탐색할 **계정** 및 **데이터 유형** (메트릭 또는 이벤트)을 선택합니다.

2. [시간 선택기](/docs/using-new-relic/user-interface-functions/view-your-data/time-picker-set-time-periods-view-data) 를 사용하여 시간 범위를 선택합니다.

3. 왼쪽에 있는 블록을 사용하여 이벤트 또는 메트릭에 대해 사용 가능한 데이터를 찾아보고 검색을 구성합니다. 블록당 하나의 요소만 선택할 수 있습니다. 블록을 검색할 수 있습니다.

작업 공간에서 다음을 볼 수 있습니다.

<CollapserGroup>
  <Collapser
    className="freq-link"
    id="workspace-query-area"
    title="쿼리 영역"
  >
    쿼리 영역은 쿼리를 다양한 구성 요소로 나눕니다. 여기에서 탐색 결과를 NRQL 쿼리로 쉽게 읽을 수 있으며, 플롯되는 정확한 데이터를 확인할 수 있습니다.

    NRQL에 익숙하지 않은 경우 이 영역을 확인하여 쿼리 작성 방법을 알아보세요.

    쿼리의 다른 부분은 다음과 같습니다.

    * \[EVENTS ONLY] `FROM` : 첫 번째 블록에서 선택된 이벤트.

    * [`SELECT`](/docs/query-data/nrql-new-relic-query-language/getting-started/nrql-syntax-clauses-functions#state-select) : 선택한 플롯 또는 측정항목입니다. 이 입력은 하나의 값과 하나의 함수만 플로팅합니다.

    * [`GROUP BY`](/docs/query-data/nrql-new-relic-query-language/getting-started/nrql-syntax-clauses-functions#sel-facet) : `FACET` 절을 나타내며 선택한 차원별로 데이터를 그룹화합니다.

    * [`LIMIT`](/docs/query-data/nrql-new-relic-query-language/getting-started/nrql-syntax-clauses-functions#sel-limit) : 보고 싶은 값의 양을 입력합니다.

    * [`WHERE`](/docs/query-data/nrql-new-relic-query-language/getting-started/nrql-syntax-clauses-functions#sel-where) : 이 필드를 사용하여 결과를 추가로 필터링합니다. 이 입력은 `n` 값을 표시합니다.

      각 항목은 **x** 를 클릭하여 쿼리에서 삭제할 수 있습니다.
  </Collapser>

  <Collapser
    className="freq-link"
    id="workspace-query-display"
    title="NRQL 쿼리 표시"
  >
    위의 필드와 시간 선택기로 [선택한 시간 범위](#explore-data) 로 구성된 전체 쿼리를 볼 수 있습니다.

    <Callout variant="tip">
      고급 검색을 수행하거나 차트를 사용자 정의해야 합니까? 데이터 탐색기에서 [쿼리 작성기](/docs/chart-builder/use-chart-builder/choose-data/use-advanced-nrql-mode-specify-data) 에 액세스하여 쿼리를 편집할 수 있습니다.
    </Callout>
  </Collapser>

  <Collapser
    className="freq-link"
    id="workspace-chart"
    title="차트 영역"
  >
    기본적으로 데이터는 **꺾은선형 차트** 에 표시됩니다. 차트 선택기를 사용하여 **영역 차트** , **파이 차트** 및 **막대 차트** 로 쉽게 변경할 수 있습니다. 결과의 **원시 데이터** 를 테이블로 또는 **JSON 형식** 으로 보도록 선택할 수도 있습니다.

    차원을 선택한 경우 차트가 다른 패싯으로 업데이트됩니다. 차트 아래에서 패싯 목록과 각 패싯 값이 있는 패싯 테이블을 볼 수 있습니다.

    패싯 테이블을 사용하여 데이터를 드릴다운합니다. 패싯을 클릭하면 필터로 적용됩니다. 테이블이 계속 표시되므로 탐색을 계속하기 위해 다른 패싯을 쉽게 선택할 수 있습니다.
  </Collapser>

  <Collapser
    className="freq-link"
    id="share-export"
    title="공유 및 내보내기"
  >
    차트를 이미지로 가져오거나 링크로 공유하거나 오른쪽 상단 모서리에 있는 **옵션** 메뉴를 사용하여 대시보드에 추가할 수 있습니다.

    URL을 복사하고 전체 탐색을 다른 New Relic 사용자와 공유할 수도 있습니다.
  </Collapser>
</CollapserGroup>

  <Collapser

    1. 새 데이터를 생성하는 계정 및 이벤트 또는 측정항목을 선택합니다.
    2. 데이터 탐색기의 다양한 도구를 사용하여 사용할 수 있게 된 새 데이터를 둘러볼 수 있습니다. 해당 이벤트 또는 메트릭의 원시 데이터를 테이블로 보고, 목록으로 모양을 지정하거나, 차트로 표시하려면 클릭하십시오. .
    3. 이벤트 또는 메트릭을 선택한 후 해당 차원에서 데이터의 형태를 찾으십시오. 카디널리티에 따라 모든 데이터의 다양한 관점을 볼 수 있습니다.
    4. 관련 항목을 찾았습니까? 대시보드에 저장하거나 동료와 공유하세요.

    1. 데이터 탐색기 탭에서 계정, 데이터 유형 및 변경한 이벤트/메트릭을 선택합니다.
    2. 엔터티가 데이터를 보고하고 모든 속성이 표시되고 있는지 확인합니다.

    1. 데이터 탐색기에서 메뉴를 사용하여 예상대로 작동하지 않는 이벤트 또는 측정항목을 선택하고 데이터 탐색기가 해당 차트를 그리도록 합니다.
    2. 여기에서 해당 데이터의 차원을 드릴다운하고 관련 속성별로 필터링할 수 있습니다.
    3. 또한 분포, 가치 순위 또는 시간 경과에 따른 진화와 같은 다양한 관점에서 데이터를 볼 수 있습니다.
    4. 관련 항목을 찾았습니까? 대시보드에 저장하거나 동료와 공유하세요.
