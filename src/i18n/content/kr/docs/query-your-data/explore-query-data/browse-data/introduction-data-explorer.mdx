[NRQL 형식 쿼리를](/docs/query-your-data/nrql-new-relic-query-language/get-started/introduction-nrql-new-relics-query-language) 작성하는 방법을 알 필요 없이 <DNT>**Data Explorer**</DNT> 사용하여 모든 데이터를 시각적으로 탐색하세요. 데이터 탐색기는 뉴렐릭 플랫폼 하단에 있는 쿼리 서랍을 통해 액세스할 수 있는 쿼리 콘솔의 일부입니다.

더 자세한 검색을 수행해야 합니까? [데이터 쿼리](/docs/query-your-data/explore-query-data/get-started/introduction-querying-new-relic-data/)방법을 읽어보십시오. 아직 하지 않았다면 아래에서 무료 New Relic 계정을 만들어 오늘 데이터 모니터링을 시작하십시오.

## 중요한 이유 [#why-matters]

당신은 시스템 상태를 조사하시나요? 리소스를 계획하고, 인지던트를 식별하고 대응하고, 잘못된 동작을 해결해야 합니까? 데이터 탐색기를 사용하면 NRQL을 사용하거나 쿼리를 작성하지 않고도 시각적 메뉴를 통해 원하는 데이터를 쉽게 식별, 가져오고 시각화할 수 있습니다.

<DNT>**Data Explorer**</DNT> 을 사용하면 뉴렐릭 데이터베이스(NRDB)에 저장된 모든 데이터에 빠르고 직관적인 방식으로 액세스할 수 있습니다. 그런 다음 <DNT>**Dimensions**</DNT> 선택하여 해당 데이터의 특정 차원에 따라 측면을 지정할 수 있습니다.

Data Explorer를 사용하면 다음과 같은 작업을 수행할 수 있습니다.

* 원시 데이터에서 진화, 분포 등에 대한 통찰력을 제공하는 다양한 시각화에 이르기까지 다양한 관점에서 데이터를 확인하십시오.
* 필터를 사용하여 데이터를 드릴다운합니다.
* 선 및 영역 차트에 예측을 추가합니다( [NRQL 예측](/docs/query-your-data/explore-query-data/use-charts/nrql-predictions) 공개 미리 보기에서 사용 가능).
* [대시보드](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards) 에 검색을 추가합니다.
* [NRQL의](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) 작동 방식을 알아보세요. 데이터 탐색기는 쿼리가 어떻게 작성되는지 보여줍니다.

## 탐색할 수 있는 데이터 유형 [#data-types]

<DNT>**Data Explorer**</DNT> 컨텍스트에서 `event` 뉴렐릭 데이터베이스(NRDB)에 저장된 모든 개체를 나타냅니다. 데이터 탐색기를 사용하면 다음을 탐색할 수 있습니다.

* 측정항목:

  * [차원 측정항목](/docs/data-apis/understand-data/new-relic-data-types/#dimensional-metrics) ( `Metric` 데이터 유형).
  * [지표슬라이스 데이터](/docs/data-apis/understand-data/new-relic-data-types/#timeslice-data): <InlinePopover type="apm" />에이전트, 모바일 에이전트 및 <InlinePopover type="browser" />에이전트가 이 데이터 유형을 보고합니다. 이러한 유형의 데이터를 탐색하려면 해당 에이전트 중 하나의 IF모니터를 선택해야 하며, 그러면 해당 옵션이 표시됩니다.

* 이벤트: [이벤트 데이터 유형](/docs/data-apis/understand-data/new-relic-data-types/#events-new-relic) .

* 로그: `Log` 데이터 유형( [로그 UI](/docs/logs/ui-data/use-logs-ui) 를 통해서도 탐색할 수 있음).

* 룩업 테이블: 뉴렐릭 계정에 없는 데이터는 CSV 파일을 업로드하여 확인할 수 있습니다( [룩업 테이블 UI](/docs/logs/ui-data/lookup-tables-ui) 통해서도 탐색할 수 있음).

## 데이터 탐색 [#explore-data]

데이터 탐색기에 액세스하려면 뉴렐릭 페이지의 아래쪽 서랍에 있는 <DNT>**Query your data**</DNT> 버튼을 클릭한 다음 <DNT>**Data Explorer**</DNT> 옵션을 선택하세요.

데이터 탐색기는 다음으로 구성됩니다.

* 상단의 범위 섹션. 계정을 선택하고 `events`, `metrics`, `timeslices`, `logs`, `lookups` 등 여러 데이터 유형 중에서 선택할 수 있습니다. 기준을 선택하시면 분류별로 필터링이 가능합니다.
* 왼쪽의 데이터 검색 영역. `metrics` 를 선택하면 옵션은 `metric` 및 `dimensions` 입니다. `events` 을 선택하면 옵션은 `event type` , `plot` 및 `dimensions` 입니다.
* 작업 공간. 이 영역에는 선택한 결과가 표시됩니다.

<img title="Data Explorer view" alt="Data Explorer view" src="/images/data-explorer.gif" />

<figcaption>
  <DNT>**Query your data**</DNT> 서랍을 통해 데이터 탐색기에 액세스합니다: 데이터 탐색기 뷰.
</figcaption>

데이터 탐색기를 사용하려면:

1. 범위를 정의합니다. 데이터 유형(지표 또는 이벤트), 계정 및 엔터티를 선택합니다.

2. [시간 선택기를](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/#dash-time-picker)사용하여 시간 범위를 선택합니다.

3. 왼쪽에 있는 블록을 사용하여 사용 가능한 데이터를 탐색하고 검색을 구성합니다. 블록당 하나의 요소만 선택할 수 있습니다. 블록을 검색할 수 있습니다.

## 사용 사례 [#uses-cases]

다음 예를 통해 데이터 탐색기를 사용하는 방법과 시기를 알아보세요.

<CollapserGroup>
  <Collapser className="freq-link" id="use-case-validate-data" title="새로운 데이터 발견">
    <DNT>
      **I&apos;ve just connected new instrumentation and want to see if new data is available.**
    </DNT>

    1. 새 데이터를 생성하는 계정 및 이벤트 또는 측정항목을 선택합니다.
    2. 데이터 탐색기의 다양한 도구를 사용하여 사용 가능한 새 데이터를 탐색하세요. 해당 이벤트 또는 지표의 원시 데이터를 표로 보거나, 목록 형태로 만들거나, 클릭하여 차트로 표시해 보세요.
    3. 이벤트 또는 메트릭을 선택한 후 해당 차원에서 데이터의 형태를 찾으십시오. 카디널리티에 따라 모든 데이터의 다양한 관점을 볼 수 있습니다.
    4. 관련 항목을 찾았습니까? 대시보드에 저장하거나 동료와 공유하세요.
  </Collapser>

  <Collapser className="freq-link" id="use-case-new-data" title="데이터 검증">
    <DNT>
      **I changed a custom event/metric and need to check if this change has been successful.**
    </DNT>

    1. 데이터 탐색기에서 변경한 계정, 데이터 유형, 이벤트/메트릭을 선택합니다.
    2. 엔터티가 데이터를 보고하고 모든 속성이 표시되고 있는지 확인합니다.
    3. 변경한 속성을 찾아 업데이트가 성공했는지 확인합니다.
  </Collapser>

  <Collapser className="freq-link" id="nrql-prediction-use-case" title="선 또는 영역 차트에서 예측 보기(NRQL 예측의 공개 미리 보기에서 사용 가능)">
    <Callout title="시사">
      이 기능은 아직 개발 중이지만 꼭 사용해 보시기 바랍니다!

      이 기능은 현재 [출시 전 정책](/docs/licenses/license-information/referenced-policies/new-relic-pre-release-policy) 에 따라 미리보기 프로그램의 일부로 제공됩니다.
    </Callout>

    <DNT>
      **I want to see the future trend of a line or area chart to take proactive measures based on the predicted data.**
    </DNT>
