[대시보드를](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/) 사용하여 시스템 상태를 시각화 및 추적하고, 문제를 해결하고, 추가 단계를 계획할 수 있습니다. 다양한 가져오기 및 내보내기 옵션을 사용하여 팀, 고객 또는 기타 이해관계자와 모든 정보를 공유하는 방법을 알아보세요.

## 대시보드 가져오기

[대시보드 색인](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#get-started) 에서 **[대시보드를 JSON으로 가져올](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-import)** 수 있습니다.

## 대시보드 내보내기

대시보드를 쉼표로 구분된 파일(CSV) 또는 PDF로 내보낼 수 있습니다.

<Tabs>
  <TabsBar>
    <TabsBarItem id="1">
      CSV 내보내기
    </TabsBarItem>

    <TabsBarItem id="2">
      PDF 내보내기
    </TabsBarItem>
  </TabsBar>

  <TabsPages>
    <TabsPageItem id="1">
      대시보드에서 차트 오른쪽 상단의 드롭다운을 클릭하고 **Export as CSV** \[CSV로 내보내기] 를 선택하여 차트를 CSV로 내보냅니다. 그러면 데이터가 CSV로 다운로드됩니다.

      CSV로 내보내는 것은 강력하지만 제한이 있습니다. 명심해야 할 몇 가지 사항:

      * 데이터는 항상 UTC 시간(-7:00)으로 내보내집니다.
      * 내보내기는 다음과 같은 집계 기능을 지원하지 않습니다. `average()`
      * 내보내기는 `COMPARE_WITH` 절을 지원하지 않습니다.
    </TabsPageItem>

    <TabsPageItem id="2">
      오른쪽 상단의 `...` 버튼을 클릭하고 **Export dashboard as PDF** \[대시보드를 PDF로 내보내기]를 선택하여 대시보드를 PDF 파일로 내보냅니다.
    </TabsPageItem>
  </TabsPages>
</Tabs>

## 대시보드 공유

모든 대시보드에서 다음 옵션 중 하나로 공유합니다.

* 대시보드 오른쪽 상단에 있는 링크 아이콘을 클릭하여 대시보드의 고유 링크를 복사하세요. 영구 링크는 대시보드에 대한 직접 링크를 공유합니다.
* 대시보드를 **JSON으로** 복사하고 오른쪽 상단의 `...` 버튼을 클릭하고 **Copy JSON to clipboard** \[JSON을 클립보드에 복사 를] 선택하여 클립보드에 추가합니다.

[대시보드를 복제](/docs/query-your-data/explore-query-data/dashboards/introduction-dashboards/#dashboards-clone) 할 수도 있습니다.

## 대시보드에 새 콘텐츠 추가 [#charts-add]

[쿼리 빌더](/docs/introduction-chart-builder) 기능을 통해 모든 대시보드에 [지표 및 이벤트](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer) 형태의 새 콘텐츠를 추가할 수 있습니다.

또한 지정된 대시보드에서 콘텐츠를 추가하려면 다음을 수행합니다.

* 수정을 클릭

  <Icon name="fe-edit-2"/>

  버튼을 클릭한 **다음 위젯 추가** 버튼을 클릭하여 새 콘텐츠를 만듭니다. [쿼리 작성기](/docs/chart-builder/use-chart-builder/get-started/introduction-chart-builder) 를 사용하여 새 차트를 추가하거나 [마크다운 편집기](#markdown) 를 사용하여 텍스트, 이미지 또는 링크를 추가합니다.

* **+ 대시보드에 추가 버튼을** 사용합니다.

모든 차트에서 오른쪽 상단 모서리에 있는 차트 작업 메뉴를 사용하여 차트를 **복사하거나 복제** 합니다.
