## 대시보드를 내부적으로 공유하세요 [#share-dashboards-internally]

다음 옵션을 사용하여 팀 내에서 모든 대시보드를 공유할 수 있습니다.

* 대시보드 오른쪽 상단에 있는 링크 아이콘을 클릭하여 대시보드의 고유 링크를 복사하세요. 영구 링크는 대시보드에 대한 직접 링크를 공유합니다.

또는

* 다음과 같이 대시보드를 <DNT>**JSON**</DNT> 형식으로 복사하여 팀 내에서 공유하세요.

  1. <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com/dashboards)**</DNT> 으)로 이동합니다.
  2. 클릭하세요 <Icon name="fe-more-horizontal" /> 대시보드 오른쪽 상단에 있는 아이콘을 클릭하세요.
  3. <DNT>**Copy JSON to clipboard**</DNT> 선택합니다.
  4. JSON 코드를 메시지나 문서에 붙여넣어 팀과 공유하세요.

## 차트를 내부적으로 공유하세요 [#share-charts-internally]

대시보드와 유사하게 다음 단계에 따라 팀 내에서 차트를 공유할 수 있습니다.

1. <DNT>**[one.newrelic.com &gt; Dashboards](https://one.newrelic.com//dashboards)**</DNT> 으)로 이동합니다.
2. 공유하려는 차트가 있는 대시보드를 엽니다.
3. <DNT>**View query**</DNT> 을 선택하세요 <Icon name="fe-more-horizontal" /> 차트의 메뉴. 쿼리가 쿼리 빌더에서 열립니다.
4. 쿼리 빌더에서 <DNT>**Copy query link**</DNT> 클릭합니다. 링크가 클립보드에 복사되었습니다.
5. 이메일, 채팅 또는 다른 커뮤니케이션 도구를 통해 팀원과 공유하세요.
