공개 API 성능 대시보드는 New Relic의 [글로벌 성능 데이터 세트](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/new-relic-global-performance-data-sets) 에서 지원하는 대시보드입니다. New Relic 계정의 일부로 포함된 즉시 사용 가능한 대시보드입니다. 공개 API의 성능에 대한 실행 가능한 일반 통찰력과 신규 고객이 자신의 데이터를 추가하기 전에 New Relic의 대시보드 기능을 테스트할 수 있는 기회를 모두 제공합니다.

대시보드는 인기 있는 공개 API에 액세스할 때 New Relic 고객의 익명화된 샘플링이 경험하는 실제 지연 시간을 표시하여 작동합니다.

<Callout variant="important">
  글로벌 성능 데이터 세트는 있는 그대로 표시됩니다. Global Performance 데이터 세트는 다양한 소스의 샘플 집계를 나타내며 New Relic은 데이터의 정확성, 완전성 또는 진실성을 확인하기 위해 노력하지 않습니다. 이 데이터를 사용하는 모든 목적에 대한 정보의 유일한 소스로 의존해서는 안 되며 New Relic은 이 데이터에 의존하여 내린 결정에 대해 책임을 지지 않습니다. Global Performance 데이터 세트는 데이터 세트에 포함된 기술에 대한 New Relic의 보증 또는 권장 사항으로 간주되어서는 안 됩니다.
</Callout>

## New Relic에 대시보드 추가 [#Add-dashboard]

공개 API 성능 대시보드가 UI에 아직 표시되지 않는 경우 쉽게 추가할 수 있습니다.

1. [one.newrelic.com/](https://one.newrelic.com/) 에서 대시보드 활성화:

   * 신규 고객: 대시보드는 기본적으로 활성화되어 있으며 모든 신규 계정의 즐겨찾기 목록에 추가됩니다.
   * 기존 고객: 대시보드가 아직 활성화되지 않은 경우 아바타를 클릭하고 **데이터 추가** 를 선택하여 대시보드를 추가할 수 있습니다.

2. **공개 API 성능** 타일을 클릭하여 계정 선택기를 연 다음 **추가를 클릭하고 사전 빌드된 대시보드를 봅니다** .

3. **공개 API 성능** 대시보드 페이지에서 탐색을 시작하십시오! 창 모서리에 있는 **...** 를 클릭하여 차트를 확장하고 쿼리를 보는 등의 작업을 수행합니다.

## 대시보드 살펴보기 [#explore-dashboard]

다음은 공개 API 성능 대시보드를 탐색하는 방법에 대한 몇 가지 제안입니다.

* 차트 모서리에서 **...** 를 클릭하고 **쿼리 보기** 를 선택하여 차트를 만드는 데 사용된 [NRQL](/docs/query-data/nrql-new-relic-query-language/getting-started/introduction-nrql) 쿼리를 봅니다.
* 차트 모서리에 있는 **...** 를 클릭하고 이미지로 가져오기를 선택하여 차트 **를 이미지** 로 보거나 다운로드합니다.
* 막대 차트에서 특정 도메인을 선택하거나 페이지 상단에 있는 텍스트 필드를 클릭하여 필터를 추가하십시오.
* 고유한 데이터를 이미 추가한 경우 쿼리를 복사하고 직접 사용하도록 수정하는 실험을 하십시오.

<Callout variant="important">
  공개 API 성능 대시보드는 현재 EU 고객이 사용할 수 없습니다.
</Callout>

<Callout variant="important">
  공개 API 성능 대시보드는 현재 경고를 지원하지 않습니다.
</Callout>

## 대시보드 및 데이터에 대한 추가 정보 [#explore-dashboard]

* Public Performance API 대시보드를 구동하는 Global Performance 데이터 세트에 대한 자세한 정보는 [New Relic Global Performance 데이터 세트](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/new-relic-global-performance-data-sets) 를 참조하십시오.
* New Relic 대시보드에 대한 자세한 내용은 [대시보드 소개](/docs/dashboards/new-relic-one-dashboards/get-started/introduction-new-relic-one-dashboards) 를 참조하십시오.
* 고객은 또한 새로운 [데이터 탐색기](/docs/query-your-data/explore-query-data/data-explorer/introduction-data-explorer) 를 사용하여 이 데이터 세트에 대해 더 자세히 알아볼 수 있습니다.
