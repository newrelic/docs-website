New Relic의 Global Performance 데이터 세트는 의미 있는 대규모 집계 원격 측정 데이터에 대한 즉각적인 액세스를 제공합니다.

글로벌 성능 데이터 세트 데이터는 다음에 유용합니다.

* 집계된 실제 데이터 모음에서 일반적인 모니터링 및 문제 해결 통찰력을 얻고자 하는 기존 New Relic 고객.
* New Relic의 대시보드 및 데이터 도구에 대한 느낌을 얻고 New Relic에 자신의 데이터를 추가하는 방법에 대한 정보에 입각한 결정을 내리기 위해 실제 데이터 시각화를 보고자 하는 신규 이민자.

이 리소스는 글로벌 성능 데이터 세트, 작동 방식, 수행하는 작업 및 글로벌 성능 데이터에 액세스하기 위한 현재 옵션에 대한 정보를 제공합니다.

Global Performance 데이터 세트는 다양한 소스의 샘플 집계를 나타내며 New Relic은 데이터의 정확성, 완전성 또는 진실성을 확인하기 위해 노력하지 않습니다. 이 데이터를 사용하는 모든 목적에 대한 정보의 유일한 소스로 의존해서는 안 되며 New Relic은 이 데이터에 의존하여 내린 결정에 대해 책임을 지지 않습니다. Global Performance 데이터 세트는 데이터 세트에 포함된 기술에 대한 New Relic의 보증 또는 권장 사항으로 간주되어서는 안 됩니다.
</Callout>

## 귀중한 데이터에 액세스하고 New Relic을 사용해보십시오. [#why-matters]

최초의 글로벌 성능 데이터 세트인 [Public API Performance](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/enable-public-api-performance-dashboard) 는 AWS, Google 등을 포함한 공개 API의 성능에 대한 방대한 실제 실시간 데이터를 New Relic 고객이 경험한 대로 제공합니다(승인된 경우).

이 데이터는 이미 New Relic을 통해 흐르고 있기 때문에 계정을 활성화한 후 몇 초 안에 액세스할 수 있으며 프로세스에서 New Relic 대시보드 및 쿼리 기능을 테스트할 수 있습니다.

자신의 데이터 소스를 New Relic에 먼저 연결할 필요는 없지만, 자신의 데이터를 추가하는 것이 쉽고 무료이며 New Relic이 비즈니스 요구 사항을 충족하는 방법을 이해하는 가장 좋은 방법이기 때문에 연결하는 것이 좋습니다.

## 공개 API 성능 대시보드 시작하기 [#enable]

공개 API 성능 데이터 사용을 시작하는 데 도움이 되도록 공개 API 성능 대시보드를 만들었습니다. 이 대시보드는 실행 가능한 일반 통찰력과 분석을 모두 제공하며 대시보드 기능을 자체 데이터에 적용할 수 있는 방법의 예도 제공합니다.

신규 및 기존 고객 모두 공개 API 성능 대시보드에서 글로벌 성능 데이터를 빠르고 쉽게 볼 수 있어야 합니다. 이 대시보드 사용을 시작하려면 [공개 API 성능 대시보드 탐색](/docs/query-your-data/new-relic-one-dashboards/global-performance-dataset/enable-public-api-performance-dashboard) 을 참조하십시오.
