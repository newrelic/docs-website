## 대시보드에 시각화 추가

새 대시보드 또는 기존 대시보드에 시각화를 추가할 수 있습니다.

1. **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Apps > Custom visualizations** 로 이동합니다. ( **Custom visualizations** \[사용자 정의 시각화] 타일이 보이지 않으면 검색을 사용하여 찾으십시오.)
2. **사용자 지정 시각화** 에서 대시보드에 추가할 시각화를 선택한 다음 활성화합니다.
3. 구성 옵션이 있는 경우 이를 채우십시오. 시각화는 변경 사항으로 업데이트됩니다.
4. **대시보드에 추가를** 클릭한 다음 사용 가능한 대시보드 목록에서 대시보드를 선택하거나 **새 대시보드** 를 선택합니다. 새 대시보드를 생성하기로 결정했다면 대시보드를 실행할 계정을 선택하고 대시보드에 이름을 지정합니다.
5. 대시보드에 **추가 를 클릭한 다음 대시보드** 에 대한 링크를 클릭하여 사용자 지정 시각화를 확인합니다.

## 대시보드 시각화 관리

**삭제** : 대시보드에서 시각화를 제거하려면 시각화의 오른쪽 모서리에 있는 생략 버튼을 클릭하고 **삭제** 를 클릭합니다.

**편집** : 대시보드에서 시각화를 편집하려면 시각화의 오른쪽 모서리에 있는 줄임표 버튼을 클릭하고 **편집** 을 클릭합니다.
