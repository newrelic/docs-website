뉴렐릭을 사용하면 NerdGraph를 사용하여 [간단한 브라우저 모니터를](/docs/synthetics/synthetic-monitoring/using-monitors/add-edit-monitors) 만들 수 있습니다. 간단한 브라우저 모니터는 웹 구성에 대한 자동화된 브라우저 기반 검사를 수행하고 전체 브라우저 환경에서 페이지를 로드하여 실제 사용자 인스턴스 합성을 시뮬레이션합니다. 이 튜토리얼에서는 NerdGraph API를 사용하여 간단한 브라우저 모니터링 도구를 자동화하여 생성하는 방법에 대한 예제를 제공합니다.

## 간단한 브라우저 모니터 만들기 [#create-simple-browser]

`syntheticsCreateSimpleBrowserMonitor` 뮤테이션을 사용하면 간단한 브라우저 모니터를 만들 수 있습니다.

### 샘플 요청

```graphql
mutation {
  syntheticsCreateSimpleBrowserMonitor(
    accountId: ACCOUNT_ID
    monitor: {
      browsers: [BROWSERS]
      devices: [DEVICES]
      locations: { public: ["LOCATION_1", "LOCATION_2"] }
      name: "YOUR_MONITOR_NAME"
      period: PERIOD
      runtime: {
        runtimeType: "RUNTIME_TYPE"
        runtimeTypeVersion: "RUNTIME_TYPE_VERSION"
        scriptLanguage: "SCRIPT_LANGUAGE"
      }
      status: STATUS
      uri: "MONITORED_URI"
      advancedOptions: {
        customHeaders: { name: "HEADER_NAME", value: "HEADER_VALUE" }
        enableScreenshotOnFailureAndScript: ENABLE_SCREENSHOT
        responseValidationText: "VALIDATION_TEXT"
        useTlsValidation: TLS_VALIDATION
      }
      apdexTarget: APDEX_TARGET
    }
  ) {
    errors {
      description
      type
    }
  }
}
```

### 샘플 응답

성공적인 응답은 오류 발생 시 `null` 반환합니다.

```json
{
  "data": {
    "syntheticsCreateSimpleBrowserMonitor": {
      "errors": null
    }
  }
}
```

모니터를 생성하는 데 문제가 있는 경우 `errors` 에는 오류가 발생한 이유를 설명하는 `description` 및 `type` 필드가 있는 객체가 포함됩니다.
