NerdGraph `dashboardCreateSnapshotUrl` 뮤테이션을 사용하여 프로그래밍 방식으로 <InlinePopover type="dashboards" />을(를) PDF 또는 PNG 파일로 내보낼 수 있습니다. 또한 [UI를 사용하여 대시보드를 내보낼](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data/#export-dashboards-pdf) 수도 있습니다.

뮤테이션에 대한 전체 문서, 사용 가능한 모든 파라미터 및 예제 ― 변수 재정의, 사용자 지정 차원 및 위젯별 차트 이미지 내보내기 포함 ― 는 [NerdGraph API를 통해 대시보드 및 차트 내보내기](/docs/query-your-data/explore-query-data/dashboards/dashboards-charts-import-export-data/#export-api)를 참조하십시오.

## 문제점 해결 [#troubleshooting]

대시보드를 내보내는 동안 다음 오류가 발생하는 경우 수행할 작업은 다음과 같습니다.

* 뉴렐릭 UI에서: <DNT>**We ran into an error while creating the PDF. Please try again**</DNT>

* 뉴렐릭 NerdGraph에서: <DNT>**Operation on dashboard entity failed with guid:YOUR\_GUID with cause: Error 504: upstream request timeout**</DNT>

내보내기가 API 최대 응답 시간을 초과하는 경우 이러한 오류가 발생할 수 있습니다. 대시보드의 위젯이 큰 시간 창을 사용하는지 확인합니다 ― 예를 들어, 한 달 전체의 데이터를 비교하는 경우입니다. 그렇다면 시간 창을 줄여보십시오.

또 다른 가능한 원인은 위젯 수가 많은 대시보드입니다. 사용하지 않는 위젯을 제거하여 오류가 해결되는지 확인하십시오.
