## 문제

API 탐색기를 시작할 때 `500` 오류가 나타납니다.

## 해결책

이 문제를 해결하려면 기본 계정이 있는지 확인하세요.

1. [**one.newrelic.com**](http://one.newrelic.com/) **> ( [계정 드롭다운](/docs/using-new-relic/welcome-new-relic/getting-started/glossary#account-dropdown) ) > User preferences** 으로 이동합니다.
2. **기본 계정** 드롭다운에서 원하는 항목을 선택한 다음 저장합니다.

## 원인

이것은 로그인과 연결된 현재 또는 유효한 기본 New Relic 계정이 없을 때 발생합니다. 이렇게 하면 Explorer가 사용 가능한 API 키를 표시하지 않습니다.
