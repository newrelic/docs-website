## 수명 종료 알림 [#eol]

Insights Dashboard API는 2021년에 수명이 다했습니다.

* 2021년 7월 28일부터 `CREATE` 및 `UPDATE` 엔드포인트를 사용할 수 없습니다.
* 2021년 8월 30일부터 `GET` 및 `DELETE` 엔드포인트를 사용할 수 없습니다.

Insights Dashboard API에서 New Relic Dashboards API로 전환하려면 [마이그레이션 가이드](/docs/new-relic-one/use-new-relic-one/core-concepts/dashboards-api-migration-insights-api-nerdgraph/) 를 참조하십시오. 자세한 내용은 [NerdGraph 대시보드 자습서](/docs/apis/nerdgraph/examples/create-widgets-dashboards-api/) 및 [Explorers Hub 게시물](https://discuss.newrelic.com/t/important-insights-dashboard-api-end-of-life/149357) 을 참조하세요.
