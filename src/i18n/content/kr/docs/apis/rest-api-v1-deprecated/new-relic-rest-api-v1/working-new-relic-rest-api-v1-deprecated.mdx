현재 New Relic은 두 가지 버전의 REST API를 지원합니다. 버전 1은 **더 이상 사용되지 않으며** 최신 v2로 대체되었습니다. 종료 날짜는 발표되지 않았습니다. 그러나 v1에 대한 추가 개발이나 수정은 없습니다.

<Callout variant="important">
  [API v2 시작하기](/docs/apm/apis/requirements/new-relic-rest-api-v2-getting-started) 및 [New Relic REST API v2 예제](/docs/apis/rest-api-v2) 를 참조하여 새 프로젝트를 시작하십시오. 또한 v1 스크립트를 해당 v2 스크립트로 마이그레이션하기 시작합니다.
</Callout>

어떤 식으로든 REST API v1을 사용하려면 [API 키](/docs/apm/apis/requirements/api-key) 가 필요합니다. 그런 다음 명령줄에서 다음을 사용할 수 있습니다.

```
curl -gH "x-api-key:REPLACE_WITH_YOUR_API_KEY" 'ENDPOINT_URL'
```

또는

```
wget -qO- --header "x-api-key:REPLACE_WITH_YOUR_API_KEY" 'ENDPOINT_URL'
```
