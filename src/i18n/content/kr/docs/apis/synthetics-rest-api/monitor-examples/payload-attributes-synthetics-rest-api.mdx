## 특정 모니터 끝점 [#specific-monitor]

특정 모니터에 대해 REST API를 호출할 때 `monitor_uuid` 을 엔드포인트의 일부로 포함합니다. `monitor_uuid` 은 URL의 일부인 GUID입니다. 예를 들어 선택한 [합성 모니터](/docs/synthetics/new-relic-synthetics/pages/synthetics-overview-page-view-monitors-performance) 의 URL은 다음과 같습니다.

```
https://synthetics.newrelic.com/accounts/nnnn/monitors/ab123-c456d-e78-90123-f45g
```

`monitor_uuid` 은 `/monitors/` 다음에 오는 값입니다.
